New 3CXPhone for conference planning iPhone

We present you the 3CXPhone for iPhone update, the new version is already available in the Apple App Store!


This version of 3CXPhone for iPhone now supports conference scheduling, now you can schedule a conference with colleagues or customers for any day or time. When the conference time comes up, the system will automatically call back to all local participants and external subscribers will receive an email with a number and a PIN for access. 3CXPhone for iPhone version 12.0.5 supports multiple languages, the language is determined based on iOS regional settings.
