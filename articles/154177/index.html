TeamLab - reboot

The new season began, almost everyone returned from vacation and joined the work. It's time to tell what updates are waiting for TeamLab in the version for SaaS 7.0, which is already available in beta mode. We call the new version a reboot not just for the sake of red word, but because we implemented significant changes in the interface, making it more unified for all modules, added new tools, and also optimized and added a number of existing functionality. Now in order.



New interface

  • The interface has become the same for all modules: the navigation bar on the left side and the content on the right.
  • Navigation inside the modules has become more transparent, including the elements of the second level. The high-level navigation module, implemented in the left pane as a folder tree, allows you to quickly navigate through the main working blocks.
  • Elements that were not very popular among users were deleted, among those that fell under the ‚Äúreduction‚Äù - a separate page with a description of projects, project history, extra tabs in the description of tasks in Projects , as well as in the CRM contact card.
  • The design became "rubber", i.e. automatically adaptable to any width of the browser window and monitors of any size.
  • There was a common button for all modules to add content "Create"




Common workspace

  • In version 7.0, the ability to add Google Drive, Dropbox and Box accounts is joined by working with SkyDrive.
  • Added ability to set permissions for files from added resources.
  • Integration of Projects and CRM has been implemented: now a specific project can be linked to a CRM contact or a new one can be created for it.
  • Integration of Projects and Documents is implemented: now you can edit files without leaving the ‚ÄúProjects‚Äù section.




Updating and optimization of functionality
