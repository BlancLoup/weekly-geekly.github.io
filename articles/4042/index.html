Microsoft music search engine and other new patents

Despite the lull by Google, in recent months there have been many interesting patents in the field of search technology. Yahoo, IBM, Microsoft and other companies have been active.

The company Yahoo (more precisely, its Overture division) patented a new system for trading at contextual advertising auctions, and also submitted an application for a data exchange system between a computer and mobile devices.

The first invention (patent 7,035,812) is a kind of "multi-element" trading system by which positions of advertisements are determined. It describes the database, which stores detailed information about each bidder, as well as the positions paid for them. The participant can independently log in and change the price for each keyword.

The system for exchanging data between a computer and mobile devices provides for a request from a mobile device, after which preliminary formatting and sending of data is performed. This generates a unique URL that can be transmitted via SMS.

Company A9 (this company belongs to Amazon) submitted an application (20060089792) for a method of sending graphic images to mobile devices (such as a map of the area or a route of a road) in response to receiving information about the location of these devices.

IBM is known to register the most patents each year. Now her attention is attracted including search technologies. The company filed several applications at once: a full-text search method in audio recordings of human speech (patent 7,039,585); a method for predicting the quality of search results and determining the complexity of various search queries (application 20060085399); interface with the use of icons that help the user navigate the search results (application 20060085395); an identification system for viewing previously visited URLs among search results (application 20060085476).
