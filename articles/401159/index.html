Microsoft has released a fresh update of Windows 10 Creators Update for the Xbox One game console

Microsoft representatives told about the features that will appear on the Xbox One game console after installing the Windows 10 Creators Update. Some Insider Program members will be the first to appreciate the changes. Other gamers will have to wait a few days to see the updated interface of Cortana voice assistant and many other improvements. The new design of the home screen Xbox One Dashboard will increase the performance of the console.

On the TV display, it will display a miniature icon of the currently running game, provide instant access to the Game Hub, and so on. The Guide interface will appear on the left after pressing the Xbox button on the controller. It allows you to quickly search for fresh games and programs, instantly go to the home page, control the volume and playback of background music. New interface supports , multitasking, opening another application on the side of the running game. In the modified Cortana helper, you can create reminders, as well as provide signals with their voice.

Settings spatial audio output will replenish with new options for creating gaming and software products that support the technology Dolby Atmos. Microsoft developers improved the screen magnifier and the Accessibility function throughout the operating system, added the Copilot controller option. Among the other features of the new update, there is a modified game mode, an improved Game Broadcasting interface, and so on.
