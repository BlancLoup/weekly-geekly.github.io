New Blogger Blogging Template Designer

I want to share the most pleasant news - the so-called Blogger Template Designer has appeared in Blogger in Draft. In short, this is a new mechanism that allows you to customize the look of your blog on Blogspot easily and quickly. At the same time, it has enough powerful functionality to create such a template, what you want to make yourself. Now you do not need to go into the code, learn a lot of hacks, html, ccs and other technical details.

This is how wonderful it all looks live:



The system is only launched in test mode in the blogger‚Äôs drafts, I remind you that you can go there using the link draft.blogger.com using your registered entry. In the future, the developers promise to improve the features and add even more templates. At the moment, the Blogger Template Designer has:
  • 15 new fully customizable templates, which are divided into 4 groups - Simple, Picture Window, Awesome Inc and Watermark.
  • 1,2,3 column layout options for each of the templates with full control of the placement and size of the columns.
  • Hundreds of images for the background from one of the leaders in the market of photobanks iStockphoto.
  • Setting fonts, backgrounds, colors for different elements of the blog
All the results of changes made in the editor are displayed on the real-time previews so that you can see what and how it looks in the blog. Once you have the look you want for the blog, click on the Apply to Blog button in the upper-most right corner of the browser window and enjoy the results.

You can read more about innovations on the Blogger in Draft website in English or in the BlogoHelp project in Russian. From myself I want to add that this update (along with some recent updates) once again confirms the status of Blogspot as the best system for creating and maintaining a free blog on the network.
