Autodafe Framework - working with models

Version 0.3 of the node.js Autodafe web application development framework has been released .

The framework has a site autodafe.ws , which is rapidly overgrown with documentation and articles. (The documentation on the site is generated automatically from the code documented by the jsdoc rules. Therefore, if you want to quickly bring up the site with the documentation of your project, you can ask me "how?".

In the new version of the framework, work with models has been significantly improved, writing classes to save data has become more convenient, and they themselves began to look more beautiful. Multiple assignment of attributes, their implicit filtering during assignment, asynchronous validation of models and the amazing use of static methods of models will undoubtedly make your life better and simpler. Work with models in Autodafe .


For those who still do not know about the framework

Opportunities


  • Architecture: MVC + Plug-ins
  • Convenient redirection of requests received via http and websockets
  • Human understandable URL
  • Mysql ORM (ActiveRecord with relationship support)
  • user management
    • Authentication and authorization session
    • User Role Management System
  • Logging to console, fs and email
  • Many other amenities out of the box
