LiveStreet 1.0 release



Today a new release of the blog-social engine LiveStreet 1.0 has been released .

New opportunities:
- New default Synio template (contest winner, author AlekseyPopov )
- A template for HTML 5 and a complete rejection of MooTools. Now only jQuery is supported.
- User Wall
- Revised user profile
- Ajax authorization and registration, including in the pop-up
- Module geo-base and on its basis the choice of place of residence in the user profile
- Search and sort among users
- Search and sort among blogs
- Ability for selected topics to add their own tags and search for topics on them
- Favorite counter for topics and comments
- Subscribe to new comments on topics

- New topic menu - discussed, popular
- The mechanism of validation of data and entities
- Toolbar (controls on the right of the screen) with the ability to add your controls
- Counting in the topics exactly how users voted
- Accounting for the number of topics in blogs
- The ability to search in personal mail among the favorites and the text of messages, view only new messages
- Console to generate a plugin template code
- Custom fields customization - type division
- Ability to leave a note about the user in his profile
- Recycling profile settings page - ajax photo upload and avatar
- Removed actions my (moved to profile) and new (moved to index)
- Paged list of friends
- User activity feed in his profile
- General site activity feed
- When a friend or blog is added, the object automatically adds user activity to the stream for further tracking.
- During registration, the user is automatically added default event types for tracking in the activity tape
- A cloud of their tags - those that the user used when writing their topics
- Various pop-up information blocks - in the list of blogs, live in personal mail
- Ability to sort the survey results
- Links to "share" in social networks at topics
- Sorting plugins by name, not by catalog
- Security Fixes
- A huge number of bug fixes and minor changes / additions

We also launched the LiveStreet API and template documentation - docs.livestreetcms.com
Changes for developers are more fully described here - livestreet.ru/blog/dev_livestreet/10694.html



By tradition, small statistics on the development of the project:
- Over 110,000 engine loads and over 250,000 module loads;
- Created more than 300 different modules, most of which are free;
- Nearly 14,000 users are registered on the community site.
- Written over 8,000 topics and 100,000 comments on them;
- GitHub - 181 watchers, 53 forks, 50 received pull requests

Thank you all so much for your support and requests, you have helped us a lot with the release of the new version!

Download the new version from the download page .

Now we are creating a mobile template for LS 1.0. If you want to support our venture, go here - livestreet.ru/blog/addons/10180.html
