Support for users using the free PVS-Studio license is now provided on the StackOverflow website.



The popularity of the free licensing option of PVS-Studio is gradually growing, and more and more free users are turning to us in support of various issues. Many questions repeat each other and, in order to reduce the volume of correspondence, from now on we will provide support to free users only through the StackOverflow website. This will make it easier for us to work with questions, and we will be able to provide links to earlier answers. In addition, a lot of questions will simply disappear, since the answer can be immediately found through a search on StackOverflow, and you will not have to wait for an answer by mail. This solution will make life easier for both us and the PVS-Studio users.

PVS-Studio is a static analyzer that detects errors and vulnerabilities in program code written in C, C ++ and C #. Works on Windows and Linux.

You can learn more about analyzer capabilities by watching the presentation (EN) on YouTube (47 minutes). Yes, this is a long presentation, but comprehensive. PVS-Studio has become a great product, and in 5 minutes it‚Äôs impossible to tell about it. This presentation is also available as a slide on the SlideShare website.

PVS-Studio is a B2B solution and is focused on development teams. We tried to create a CppCat tool for individual developers, but we could not earn money on it. However, in our opinion, it is not good to deny individual developers the possibility of using PVS-Studio. Therefore, we have created a free licensing option that can be used by these developers.

The essence of free licensing is that a special type of comments should be added to the code. If comments are present in the code, the analyzer performs the analysis without any restrictions, without requiring a license. Details are in the article " How to use PVS-Studio for free ."

We are glad that more and more people are starting to check the code of their projects using the free version of the analyzer. For us, this is a long-term advertising project, which over the years will give a return in the form of new customers.

However, at the moment the free version is only an additional headache. It leads to more support costs. Not answering questions is not pretty. However, the answers require more and more time, and more and more distract us from the main activity.

In this case, we see that the questions are often repeated. For example, they ask a lot whether it is possible to write comments a little differently, so that they fit into 70 or 80 characters. Therefore, we concluded that this and many other issues will be much easier to discuss on StackOverflow. There we will talk about the various ways to design comments and most of the users' questions related to this topic will disappear. If someone asks a clarifying question, then our answer will also be available to everyone. Thus, I think we will gradually reduce the number of questions by 90%.

And it will make life easier not only for us, but also for free users. Instead of waiting for a response from us in the mail, they are likely to be able to immediately find the answer to StackOverflow.

Therefore, we will now provide support to free users only through replies on StackOverflow.

In order for your question not to go unnoticed, please use the " pvs-studio " tag.

Our paid customers will continue to receive support privately via mail. However, if one of the clients wants to ask questions via StackOverflow, we naturally do not mind.
