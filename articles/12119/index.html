Skype on iPhone - why not


After a clear answer from iPhone developers, we have already come to terms with the idea that Skype will never be able to work on this phone. However, it was still possible.

Thanks to the SoonR Talk application, iPhone owners will be able to use Skype, albeit with some restrictions.

SoonR Talk is just a part of the SoonR software package, which allows you to remotely access your desktop computer from a mobile phone or PDA, and manage many applications, including Skype. SoonR Talk uses a web interface to work, and can work on almost any device that has an Internet browser on board. Using a secure SSL connection, SoonR Talk communicates with the SoonR server, which in turn manages the client installed on the desktop. At the same time to work SoonR enough to use a normal GPRS connection.

After launching SoonR Talk, the page will display the usual list of Skype contacts with ‚ÄúOn-line‚Äù status of the interlocutors. You can start a chat, initiate a voice call or a conference. In order to make a call, simply click on the name of the interlocutor in the list, after which the most interesting begins!

SoonR Talk sends commands to the desktop Skype client and initiates two calls. The first is on your phone at the SkypeOut tariff, the second is a regular Pc-to-PC call. After dialing both calls are combined into a conference, and you can communicate with the other person.

In this case, the cost of a call is made up of the price of one call at a SkypeOut rate and payment of the minimum traffic that SoonR uses to exchange management teams. In the case of a call to a regular phone, you will have to pay SkypeOut calls to both destinations. But such a scheme can often be cheaper than the prices of many mobile operators. However, many Skype users do well without voice calls, using only chat, and in this case SoonR Talk is not at all inferior to a regular Skype client.

It seems that today SoonR Talk is the only application that can work with Skype on the iPhone. At the same time, SoonR can be interesting not only to iPhone users, but also the owner of regular mobile phones for which there is no Skype client yet.
