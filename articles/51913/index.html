OpenOffice.org pro 3.0.1 from Infra-Resource

The Infra-Resource site finally has a long-awaited official announcement:

OpenOffice.org pro 3.0.1, created on the basis of OpenOffice.org, the GNU LGPL license, is available for free download and use.

The build is made for the Windows, GNU / Linux i386 and x86-64 platforms in .DEB, .RPM and Generic packages, FreeBSD 7.1 systems and InfraLinux.org.




1. If the installation is performed without administrator rights, the office will be installed with incomplete functionality: Java Runtime Environment and desktop integration will not be installed. Therefore it is necessary to install under the root, if there is such an opportunity.

2. To check the grammar in OpenOffice.org usually put the extension Language Tool . For OpenOffice.org 3.0.1, the Language Tool version 0.9.6 extension has been released. It is recommended to put it, as the authors promise better integration with the office, a new Russian dictionary and work on the bugs. Those who hurried to install LLC 3.0.1 with the previous version of the Language Tool installed will have to uninstall LLC 3.0.1, then install LLC 3.0.0 and remove the old Language Tool. In the new version of LLC this will not work. Then remove the LLC 3.0.0 and install the LLC 3.0.1 and Language Tool 0.9.6. After this, the grammar checker works fine in LLC 3.0.1.

Enjoy your work with the new office, friends!
