News and Releases


  • PHPNG - The developers presented a new generation PHP interpreter. It all started with the message of Dmitry Stogov in php.internals and the corresponding branch in the repository. An analysis of the interpreter‚Äôs performance in real-world applications has shown that the problem of performance lies in the methods of working with memory and organizing the storage of data structures. That is why PHPNG is an optimized Zend Engine in which new data structures are implemented. Refactoring has already allowed to increase performance by 10-30% in real applications. Probably PHPNG will form the basis of PHP 5 ++, but for now this is only the beginning and the project is developing very dynamically.
    More information about PHPNG can be heard firsthand in the author's report at the upcoming DevConf 2014 conference - ‚ÄúGive your PHP for pumping - PHPNG is a new generation engine‚Äù .
