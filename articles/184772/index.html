USB flash drive Leef Bridge with micro USB connectors

The Leef Bridge flash drive is positioned by the Chinese manufacturer as a means of exchanging files between ordinary PCs and mobile devices without using wires and (or) wireless connections. Nevertheless, the possibility of a simple expansion of memory in those devices in which SD cards are not provided looks like an important factor.

The mechanics are simple: a USB flash drive has a USB connector at the same time, thanks to which it can be plugged into a regular USB port and micro USB, due to which it can be plugged into the corresponding port of a smartphone or tablet. The key point is that both ports write data to the same memory, so that access to the files can be obtained from any device where the USB flash drive is connected at the moment.

Technically, we are talking about USB 2.0, the memory capacity is 16 or 32 GB, the operating systems are Android, Windows, Mac, the cost is $ 17.99 , however, only the USA and Canada are called as countries where delivery is possible.
