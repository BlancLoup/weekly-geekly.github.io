PiCore: 25 MB Linux distribution for Raspberry Pi



Raspberry Pi is a tiny PC that is capable of much that many representatives of the Habrasoobschestva have already proven. The other day for this miniature PC released a special Linux distribution, a very modest volume, 25 MB. The authors of the distribution called PiCore were made by the Tiny Core Linux project team. Actually, the team reworked the Tiny Core Linux distribution, sharpening it under Raspberry Pi.

The distribution includes an updated kernel, the latest Raspberry Pi firmware, and some other improvements. At the same time, the image size is only 24.7 MB.

Here is a list of improvements and improvements that appeared in this distribution:

* kernel updated to 3.14.4
* Raspberry Pi firmware updated to May 12, 2014 version
* e2fsprogs updated to 1.42.10
* util-linux updated to 2.24
* official BusyBox patches applied
* curaga's wget3 BusyBox patch applied, setting default timeout to 10s
* tce-load:
* tce-load: Use sudo when unmounting meta-extensions
* patched to enable HDMI sound at 192kbit / s rate (thanks sbp)
* patched rtl8192cu driver to disable powersaving
* enabled CMA and FIQ-FSM by default
* filetool.sh "Done" message cosmetics

In this case, the distribution includes a graphical interface. True, neither the browser nor even the file manager is here. It is assumed that the user will install all this himself, if necessary, using the terminal and the package manager.

The authors of the distribution kit assume its full load and operation from the RAM, so for ‚Äúinstallation‚Äù it is enough to record the image of the distribution kit on the memory card.

Of course, PiCore cannot be called the most advanced Linux distribution for Raspberry Pi. But, definitely, it is one of the most minimalist distributions that do not burden the system and are able to perform the basic functions needed by the user.
