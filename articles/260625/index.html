Microsoft has released a set of updates for Windows.

Microsoft has released a set of optional (non-security) updates that improve performance and increase the stability of Windows. It released 17 updates, among which KB3064209 and KB3058168 are the most interesting. KB3064209 ( June 2015 microcode update for Intel processors in Windows ) for Windows 7-8.1 updates the OS system files that are responsible for implementing the functions of its loading at the earliest stages (Hvax64.exe, Hvix64.exe, Hvloader.efi, Hvloader.exe, Hvservice.sys, Kdhvcom.dll, Mcupdate_genuineintel.dll).



Another update KB3058168 ( Windows 8.1 and Windows 8 Update ) is another improvement of Windows 8 for its subsequent upgrade to Windows 10, it delivers a new version of the Cryptxml.dll file, as well as updated versions of the configuration files required to successfully activate Windows 10 using the KMS (Key Management Service) service.
