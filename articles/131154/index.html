Jelastic Cloud PaaS - MySQL, MariaDB, PostgreSQL and MongoDB database usage statistics

At the request of our users, we post the first database usage statistics in the Jelastic platform. Statistics collected the easiest way. We calculated the number of specified databases in all user environments. More than 1000 environments were taken into account in the statistical sample.



Based on statistics, MySQL is currently leading, however, other databases are also quite popular.

Distribution of databases by region Europe and North America.



Summary table
Europe North america Total
Mysql 47% 40% 46%
MariaDB 13% sixteen% 14%
PostgreSQL 15% 15% nineteen%
MongoDB 20% 28% 21%

The results presented are not an accurate representation of the distribution in the global market. This is the distribution statistics inside the Jelastic platform.

We plan to post monthly such statistical reports. Also, very soon we will publish statistics on web application servers.
If you have not done time to contribute to these statistics, you can do it right now by creating your environment in the Jelastic platform.
