Web Standards Update for VS2010 SP1

Today, the Web Platform and Tools team announced the release of the first Visual Studio update for web standards, extending support for HTML5, CSS3 and some JavaScript APIs for all versions of Visual Studio 2010.



Download the update can be in the gallery Visual Studio .

The update adds prompts (intellisense) and validation for:
  • HTML5

    • Audio and Video and related tags,
    • new types of input forms,
    • Drag & drop
    • WAI-ARIA,
    • Microdata,
    • Schema.org support

  • CSS3

    • 2D Transforms,
    • 3D Transforms,
    • Animations,
    • Background & Borders,
    • Basic Box Model,
    • Basic UI,
    • Behavior
    • Color,
    • Flexible Box Layout,
    • Fonts
    • Generated Content for Paged Media,
    • Hyperlink Presentation
    • Line,
    • Lists
    • Marquee,
    • Media Queries,
    • Multi Column
    • Namespaces,
    • Paged media
    • Presentations Levels,
    • Ruby
    • Selectors,
    • Speech,
    • Syntax,
    • Template layout,
    • Text,
    • Transitions,
    • as well as support for prefixes for different browsers
  • JavaScript APIs

    • Geolocation API,
    • Local storage

See also Scott Hanselman's blog for more details.
