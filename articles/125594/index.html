50 more Nokia N950 are waiting for developers

Good news for developers who want to get the Nokia N950 - the device distribution program among open-source developers has been expanded and 50 more devices are waiting for applications!



To accept an application, you must show an interesting working Open Source project written in Qt or QtQuick. It should be set up and run (in QtCreator / scratchbox and emulator). The project may not be completed, but you must show that the project is working in order to allow further selection.

Applications are accepted and processed on a dedicated wiki page - instructions there:
wiki.meego.com/Community_Office/Community_device_program/Nokia/N950_Extended_Program

Good luck!
