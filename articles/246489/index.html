Contest of games with gamepad support

Dear friends! We invite you to take part in the competition of games for the Windows Store! The only condition for accepting your application to the competition is to support a gamepad, for example, such as a controller for Xbox360 or Xbox One. You can learn how to develop Windows Store applications with gamepad support from our articles: Xbox One gamepad for PC games , How to use the GamePad in the browser and in Windows applications in HTML and JavaScript? , Vibration of the XboxOne gamepad for Unity3d and the training course " Support for gamepads when developing computer games ."
We will be very happy if you are using existing practices, for example, transfer your application for a mobile device to the Windows 8.1 platform. You can develop or port a game for Windows 8.1 with the help of a large number of technologies and tools, many of which are corss-platform. Already, you can find out how to create and transfer gaming applications to our platform on a special page of the competition .

If you already have a published game for Windows 8 or Windows Phone, and the game genre allows you to add support for the gamepad, be sure to take part in the competition!

In addition to the chance to win prizes in our competition, do not forget to also apply to the Id @ Xbox program! It is quite possible your great gaming application will interest the Xbox team, especially since it will already support the gamepad.

After the results of the competition are summed up, the games selected by the jury will go to a special ‚Äúgame shelf with gamepad support‚Äù in the Windows Store section, which means that more users will find your game.

As part of the contest, on January 29, at 10 am, there will also be an online webinar, which you can connect to via this link . As part of the webinar, we will talk about how to register in the Windows application store, what steps need to be done to create an application package (using the example of a game based on Unity3d and HTML5 technologies), and how to publish this application.

You can familiarize yourself with the criteria of refereeing and contest rules at contests.techdays.ru/GamePad-Games/2014-12/conditions
