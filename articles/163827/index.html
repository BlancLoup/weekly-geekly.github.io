Live Suggest.io search: +7 to relevance algorithms



A short announcement for those who visit our pages for the first time.

Suggest.io is a service for quick organization of live search on sites of any architecture and complexity.

Main features:
  • Creating a search site does not require special skills and takes only a few minutes. Site indexing begins immediately after installing the Suggest.io code on the user's site.
  • Suggest.io responds to the input of each letter of the query and offers search results before the query is entered completely.
  • Search results remain on your site, and does not go to the page to search engines. Search results are displayed in a drop-down box below the search field.
  • Suggest.io can show not only text, but also images from the pages of the site.
  • All search elements are easy to customize for the site design on which it is installed. Interface settings design is extremely simple.



Since the publication of the last release (12.12.12), no fundamental changes have been made to the functionality of the service. Developers are engaged in further debugging of the system and elimination of the identified deficiencies.

What is done:

  • Changes in the algorithm for calculating relevance. In the top positions of the list of search results are displayed more relevant to the search query page. Pages with low relevance are not included in search results.
  • Solved a small problem with highlighting the search results. Previously, the endings of words were not highlighted. Now the words are highlighted completely, regardless of the cases and declensions.
  • Rebuilding full-text indexes. Search calls are now faster. Accordingly, the search for Suggest.io on client sites is faster.
  • All the functionality of the settings is now available in the editor, which opens on top of the user's site.
  • Fixed compatibility editor settings editor with some browsers.
  • Improved search window resizing mechanism. Now the window height can be made larger.
  • Fixed bugs. It makes no sense to dwell on them, because all of them are moloznachitelny and practically did not affect the service.


All for now.
Please note - the changes made relate exclusively to the operation of the system. Doing any reconfiguration of Suggest.io on the connected sites is not required.

Thanks to everyone who helps to improve Suggest.io.
