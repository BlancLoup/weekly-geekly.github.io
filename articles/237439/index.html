IPad versions with Wi-Fi only are tied to an Apple ID account by serial number, which is stored in NAND memory. Versions with 3G / 4G / LTE are tied to the serial number + IMEI, which is stored in the modem module.

The process for all iPad 3G consists of the following steps:

  1. We disassemble and pull out the main board of the iPad 3G
  2. We find on the board part of the Board ID, which is responsible for identifying the device.
  3. Disable the modem part there. It will be necessary to unsolder and (possibly) solder a resistor.
  4. The device goes into DFU mode, because the Board ID does not match the firmware. On the screen of the iPad 3G, it is unlikely that there will be anything (maximum apple).
  5. We connect to iTunes. It will not immediately detect your device. Then he will find as an unknown iPad. We restore the firmware from the WiFi version and it's ready. GPS will work too (bonus).
  6. Disable automatic firmware update: Settings \ iTunes Store, App Store \ Automatic downloads \ Updates. Suddenly, Apple will want to block such devices in the future.
