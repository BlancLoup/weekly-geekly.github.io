Root Transmission: we get Root rights on one Android device using another Android device



Many of us prefer to ratovat Android device immediately after its purchase. Someone wants to add functions, someone just does it all for the sake of interest. And there are a lot of software tools that help to root the Android device, I don‚Äôt want to download it. But there is one program that stands out among a number of others. This is Root Transmission, software that allows you to get Root rights on one Android device using another Adnroid device, without including a PC in this chain.

In order to use Root Transmission, you need to connect the two mentioned devices to each other with a USB cable, and press the "Root" button. True, there are certain requirements for the device, which is the main. Firstly, one of the devices should already be under the root, and secondly, it should support USB On The Go (OTG). Yes, and a USB OTG cable should be.

For many users, of course, it's easier to connect your device to a PC, and work with it like that. Root Transmission is suitable for those who do not have a laptop or PC at hand, plus this software was created to demonstrate the fact that a tablet or smartphone is still a mobile computer, and not just a phone.

Now the application supports the routing of devices such as the first generation of Amazon Kindle Fire, Verizon Samsung Galaxy S III, Samsung Galaxy Note II and several other models.
