10 Windows Phone Applications for Administrators

In the work of the administrator, alas, periodically there are situations when something went wrong, and this is something urgently needs to be corrected. The task is complicated if at this moment in time you are not behind a large computer in order to quickly see and correct everything. In such situations, you can use a smartphone, for which connecting to a remote machine via SSH or RDP is not a problem.

And even in the absence of force majeure for some completely mundane things, instead of a large desktop or laptop, you can use a mobile and lightweight smartphone.

In this post, we have described ten useful applications for Windows Phone, which in this or that situation can make life easier for system administrators.

Mobile Server Stats

$ 1.99
This is a remote Windows Server / PC monitoring tool that allows you to monitor server statistics from a mobile device in real time. For the application to work, you need to install a free server component from www.mobileserverstats.com .

The downloaded server component contains pre-compiled WMI requests for the most required statistics (state of the system, processor, disks, services, running processes, users, groups, etc.), however, you can reconfigure them and add new ones. The only disadvantage of the application is the absence in the Russian Marketplace.

The ssh client

is free
The uncomplicated name immediately reveals the essence of the application: using The SSH Client, you can connect and manage servers via the command line. The application allows you to save multiple connection profiles and supports authentication using private keys. The application supports protocols SSH1, SSH2 and Telnet.

Cool remote

is free
This is a remote desktop client that works with Windows XP / 2003 / Vista / 2008 / Windows 7. As in the case of Mobile Server Stats, a free server must be installed on remote machines for the client to work.

The application has full support for working with the keyboard, including all system combinations, and the ability to work with multi-monitor systems. To connect to a remote machine, you can use either the manual connection settings, or by choosing from the list after scanning the local network.

A useful addition will be the fact that after installing the server side of the Cool Remote to a remote machine, it will be possible to connect via a web browser.

Remotedesktop

69 rubles
Unlike Cool Remote, this application uses a RDP connection, so installation of an additional server application is not required to connect to computers on Windows XP, Vista, 7 and 2003 Server. RemoteDesktop supports network-level authentication (Network-Level Authentication, NLA), multiple screen resolutions and zooming for more comfortable work.

Petri IT Knowledgebase

is free
This is a convenient mobile client of the knowledge base www.petri.co.il , containing thousands of tutorials, tips and tutorials on Windows Server, Exchange, Active Directory, SQL Server, Windows 7, virtualization, routing, and so on.

Group policy search

is free
Group Policy Search - is the most detailed reference to Microsoft Group Policy and system registry of different versions of Windows operating systems. Using this application, you can find the necessary parameter directly on the smartphone in the system registry of the operating system or in group policies that operate on the computer. In the application, you can specify for which products to look for keys and policies.

SubCalc Subnet Calculator

is free
This is a small but convenient application that allows you to quickly calculate IP addresses and subnet masks. If you are a system administrator - pay attention to this calculator for calculating the addresses in the subnet and network masks.

Pingdom pulse

is free
If you administer web servers, you probably know the Pingdom service for assessing the availability of web servers. With Pingdom Pulse, you can see a brief overview of the availability of all your servers, including the current status and response time, view detailed statistics for the last 30 days for each server, and quickly check the availability of any http server.

The application works with both paid and free Pingdom accounts.

My Server

is free
This official application from Microsoft allows you to seamlessly connect to server resources using Windows Phone smartphones. Using the My Server application, you can manage users, devices, alerts, and access shared files in Windows Server 2012 Essentials.



Wake my pc

is free
If the network card of your computer supports the Wake-On-LAN protocol, this application will allow you to wake it up remotely for further use. All that is required of you is to add once to the application the MAC addresses of the network cards of all computers and the IP addresses of the networks in which they are located.
