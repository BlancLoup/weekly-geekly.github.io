Site frame in one click

The design company Volkside made a gift to web developers by releasing the free bookmarklet Wirify , with which you can display the wireframe of any site with one click. A useful thing to quickly evaluate some aspects of a site's design: a modular layout system, a visual hierarchy, free space, symmetry, a golden ratio, a rule of thirds, etc.



Bookmarklet you can drag yourself to bookmarks and run on any page, works with all modern browsers,



Rendering to jQuery does not always happen perfectly, sometimes small distortions occur. Exporting a frame from Wirify is not supported, but you can use plugins to save a web page as an image: Screengrab for Firefox or Screen Capture for Chrome.

If you have to deal with the work with the framework, then it makes sense to look at other tools. For example, the developers of the web service jMockups promise to roll out the import feature of the framework of any site in a couple of mouse clicks in the near future.
