Chrome and Windows 2000

The fact that Google does not support Windows 2000 in its browser strained users of this operating system.

Moreover, a significant portion of these users have relatively old equipment, where decent Chrome performance would be very useful.

So, at the moment there is already a solution to this problem. Install and run Chrome under Win2k.


First [ ? ] solution proposed by the Japanese blogger blackwingcat .

The bottom line is that Win2k has the compatibility functions of this OS with other Windows, even newer ones, such as XP or Vista. [ Who knows something interesting about this, tell in the comments or give links. ]

From the site of the Japanese blogger you need to download the file chrome2k.zip. There is a ‚Äúbatch file‚Äù in the archive, which makes changes to the registry, installing for some programs, including IE XP compatibility mode and opens the Chrome download page.

But this decision was not very popular, because On the whole, people are not familiar with Japanese, and Google Translate, which blackwingcat appealed to, did not help much.

And here Navjot Singh connected to the problem. blogger from india

In her post, she proposed her method for installing Google Chrome under Windows 2000 .

Here he is:
  1. Download Chrome Offline Installer . (Modification of the registry for compatibility with IE with XP is no longer required).
  2. Make sure that you have the Update Rollup 1 for Windows 2000 SP4 ( KB891861 ) patch installed. If not, download it and install it, otherwise you will run into an error:
  3. Download chrome2k.zip .
  4. Run chrome2k.reg, making changes to the registry. Then run the Chrome installer.

Everything should work.

To remove the "chrome" you need to remove it in the standard way through the control panel applet and execute unchrome.reg.
