Github gets its first investment



Github is making a new step forward: for the first time since its foundation in 2008, the company receives investment. The administration of the free code repository announced on Monday an agreement to finance $ 100 million from the Andreessen Horowitz foundation for improving and expanding the platform, which has already become the industry standard for managing and searching for code on the network. The fund estimated the value of the company at 750 million US dollars.

Peter Levine, partner of the Andreessen Horowitz Foundation, will join the Github team. The funds will be used to hire additional employees and expand to new platforms, such as mobile devices. Github Executive Director Tom Presto-Werner expressed hope not only to develop new functions, but also to improve old, for example, web applications for various operating systems. It is planned to make Github convenient to the masses of customers, from enthusiasts to large companies, from software developers to designers and writers.
