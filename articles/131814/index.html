IOS 5.0.1 released for developers

November 2, Apple has updated iOS 5. This update is currently available only to developers and can be obtained directly from the developer portal.



The update is available for the following devices:
  • iPad 2 (Wi-Fi)
  • iPad 2 Wi-Fi + 3G (GSM)
  • iPad 2 Wi-Fi + 3G (CDMA)
  • iPad (Wi-Fi and Wi-Fi + 3G)
  • iPhone 4S
  • iPhone 4 (GSM)
  • iPhone 4 (CDMA)
  • iPhone 3GS
  • iPod touch (4th generation)
  • iPod touch (3rd generation)

List of changes affecting the new version:
  • Battery bugs fixed
  • Multitasking gestures added to first iPad
  • Fixed errors with documents in the cloud
  • Improved voice recognition for Australian residents
  • Improved security


Official release is expected in a few weeks.
