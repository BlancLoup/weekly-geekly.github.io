New courses on Windows Server 2012 on the MVA portal

Over the past few weeks, the Microsoft Virtual Academy portal has been updated with several new free courses on Windows Server 2012 . These courses consider in more detail a number of issues raised in the technical review of the operating system.

Windows Server 2012: Server Virtualization


  • Hyper-V scalability, including a detailed discussion of hypervisor settings in NUMA systems.
  • All aspects of live migration (Live Migration), including Storage Migration, + SMB migration, migration security settings.
  • Features of importing virtual machines (VM).


Windows Server 2012: Network Infrastructure


  • Expanding the capabilities of some network components and services, namely: DHCP Failover, NIC Teaming, Quality of Service (QoS).
  • The architecture, deployment, and operation of the IP Address Management (IPAM) service.
  • Perhaps the first more or less detailed description of the technology Network Virtualization.


Windows Server 2012: Storage Systems


  • Manage disk resources using Storage Spaces.
  • Use SMB for Hyper-V and SQL Server.
  • Everything about Cluster Shared Volumes (CSV) v2, from architecture to customization and application scenarios.
  • Changes in file systems. What's new in NTFS, what ReFS is in general, features of deduplication usage.


Windows Server 2012: manageability and automation


  • An overview of the Windows Management Framework 3.0 (WMF) and standards applied in the framework. Management WMF: WS-Management, SMI-S, OData, REST, etc.
  • Application Server Manager to manage multiple servers.
