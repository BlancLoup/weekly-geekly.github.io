Ya.Subbotnik goes to Chelyabinsk

On February 25, 2012 , at 12 noon, we invite everyone to the first Y.Subbotnik this year.

It will be held at the Aurora Hotel in Chelyabinsk at the Ural Web Developer Conference 2012 (UWDC).

We will talk about technologies, solutions and practices that may be not only interesting, but also useful for developers:
  • How to use BEM! outside of Yandex (with examples of HTML + CSS layouts on PSD, writing a JS component, building a dynamic site using PHP / Django / RoR).
  • What else do designers say (new CSS properties, beautiful trees, BEM and Bootstrap).
  • Learn more about the architecture of the frontend J. Mail (reverse side of the frontend, loading data, updating pages and interacting with users)
  • Scalable JavaScript applications (how to properly lay out the architecture so that the application can evolve smoothly)
  • Where to grow web designers (the world without photoshop)
  • Integral approach in design (Internet services and the role of the designer in the project)
  • Yandex sports projects and workloads (the right architecture for projects with clear launch dates, increased load requirements and resiliency)
  • Clustering duplicates in Yandex. Pictures (which makes the search for high-quality images)

All participants of the UWDC 2012 conference (registration on the website ) will be able to take part in Ya. Subbotnik.
