Duokan. Or how to read DJVU on Kindle

As one of the first owners of the Kindle 3 in Russia, I often heard the question ‚ÄúCan DJVUs be read on the Kindle?‚Äù. In short, the answer is ‚Äúyes, you can.‚Äù But everything is not as simple as it seems. Today I will tell how this can be implemented.

Introduction


Nobody has yet been able to add support for DJVU to the main firmware. As always, Chinese craftsmen came to the rescue. They have developed an alternative firmware Duokan, which allows you to get such goodies as:
  • DJVU support
  • PDF reflow
  • trimming edges
  • work with file structure
  • more customization options
  • a few more goodies

Installation


Let's move from words to deeds. The main difficulties are associated with installation. The fact is that the main instruction is written in Chinese. If you work hard, you can find instructions in English. But now we will analyze all the steps in detail in Russian:
  1. No jailbreak required, don't worry about it.
  2. Download the archive from here and unpack
  3. Copy 3 folders (DK_Documents, DK_Pictures and DK_System) to the main Kindle directory
  4. Go to the folder ÂçáÁ∫ß ÂåÖ (do not confuse: D), then we find the necessary bin-file for your Kindle (markings are shown below) and also copy it to the main directory of the device
  5. Turn on the Kindle. Press the Menu button, go to Settings. Press Menu again, select Update Your Kindle. We are waiting until everything is updated and Kindle will turn on.
  6. We return to the familiar settings (Menu -> Settings). Click Menu and select Restart.
  7. When a picture with Chinese characters appears, press Q on the keyboard.
  8. All firmware successfully installed.

Further I will answer questions in the form of FAQ:

What with Russian fonts ?!


So far, you have to be content with such. English font is displayed normally, and in Russian (as well as German and French) there is an additional space between the letters. It's not about the font, but about the firmware (checked by changing the font).

How to fill the books?


Copy to device in folder DK_Documents. In it, you can sort the books into folders, Duokan supports the file structure. To access books from the original firmware, you need to switch to Use Kindle Lists in Yes on the 3rd settings page. Unfortunately, it is impossible to download books via email on the firmware.

Is there a browser? 3G?


Unfortunately, not yet. You can go to a couple of Chinese sites, but this is not very useful. 3G is also not supported yet.

Are custom screensavers supported?


Yes, included in the menu on the 3rd page. Pictures rush to DK_System / Savers /. Size for 6 ': 600x800, for 10': 824x1200. A huge collection of pre-made screensavers is available here .

What formats are supported?


Text:
  • Txt
  • PDF
  • Epub
  • MOBI
  • DJVU
  • ZIP
  • Doc
  • CHM (not HTML)

Graphic:
  • Bmp
  • PNG
  • Jpg / jpeg
  • Gif
  • Tiff

Audio:
  • MP3

How to download the original firmware?


Menu -> Settings -> Switch to kindle

Can I update the original firmware with the installed Duokan?


Yes, Duokan does not affect the operation of the main firmware.

This is basic firmware information. Hope it was helpful for you, thank you.

UPD

Deletion


Menu -> Settings -> Uninstall. After that, delete the DK_Documents and DK_Pictures folders if you no longer need them.
