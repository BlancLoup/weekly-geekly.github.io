Specifications


Display : 5.0 inch TFT touchscreen ( resistive )
Resolution : 800x480
Operating system : Android 2.2.1
Camera : Rear - 5 megapixel (there is reason to believe that there are 3 with upscale to 5), frontal - 2 megapixel
CPU : Cortex A9 AMlogic AML8726-M @ 800MHz ARMv7, does not support 3G, but supports USB-host (in practice, everything is completely different for now)

RAM : 512M
WiFi and FM modules
HDMI output
Internal memory : 4Gb (how much is actually available to the user - I can not say)
Dimensions : 175mm x 75mm x 20mm
Weight : 200g
Battery : 2900mAh
Battery life : 5 hours of video; 8 hours of music; 3-4 hours full load of the device.
Of course, there is an accelerometer, support for hardware decoding of FullHD video (which, in conjunction with HDMI, can be very useful), Flash 10.x (probably higher), built-in microphone, stereo speakers, as well as all the goodies of a modern 5-inch android - the tablet.
The device, surprisingly, does not have a GSM module, GPS and bluetooth, and the CPU does not support 3G. The console communicates with the PC via USB 2.0.

Externally, the system behaves responsively, the interface does not slow down. The browser supports Flash version 10.x, however, I do not exclude the possibility of updating the player from the market to version 11. There is support for OpenGL ES 2.0. The OS boots too quickly enough - in 30-40 seconds.

Software


Onboard the device is a standard Android 2.2.1. Neither the modified interface, nor alternative launchers. But the full clip emulators for almost any platform younger than 2000. All the standard for Android hardkey buttons for some reason duplicated in the status bar android.

The menu is fast and responsive, however, I personally would advise immediately changing the launcher to something more convenient. Most of the pre-installed programs are naturally emulators. I believe that the main target audience consists of emulators. They will not be particularly indignant about the resistive screen =)
