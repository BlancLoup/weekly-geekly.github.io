Online HTML5-Converter for Android / iOS



Mobile application developers have a hard time porting code for each OS, dealing with the SDK, compilers, and emulators. The creators of the cross-platform framework PhoneGap have tried to simplify the task: it positions itself as the only open source mobile framework with support for six mobile platforms. Now they have gone even further and rolled out PhoneGap Build (beta), a simple online HTML5 converter. It is enough to download the application on html / js / css - and at the output you get a ready-made binary for the Apple App Store, Android Marketplace, Palm, Symbian or BlackBerry.



You can download the application as a ZIP or index.htm file, or simply provide a link to the Git or SVN repository. In the case of Github, private repositories are also supported.



In a couple of minutes you will receive a URL with a ready-made binary. For webOS or Symbian, you can immediately submit and sell it, and for Android and iOS, you also need to get the right certificates and / or keys.

Nitobe promises that PhoneGap Build will always be free for open source projects, and for others, they can enter a paid subscription after the end of beta testing. The API is already ready , so you can conveniently integrate the converter into your system.

UPD. Due to the fact that Adobe is buying Nitobi, it is announced that PhoneGap is going under the wing of the Apache Software Foundation as an open Apache Callback project (the name may change).
