Smart task scheduling with todoist

A good silenced person has recently advised me a very convenient task scheduler, who has been searching for a long time and unsuccessfully. I share joy with habrasoobshchestvu - a detailed overview of the Todoist task scheduler from silenced (the author does not have enough karma to publish, and useful information should belong to the people :)

I really love to plan my tasks and I recommend Todoist service for many years, which I have been using for almost a year. However, many, after registering and looking around a little, leave the site and successfully forget about it. If you ask about the reason for leaving, in response you usually get something unintelligible, and then it turns out that the person simply does not know about all the possibilities of the service.
Especially for those who are too lazy to dig and look at the description on the site, I decided to devote a separate article to this service.

And so, for starters, what attracted me so much in this todo list?
  • The site is not overloaded and looks very clean;
  • Well thought out hotkey system. Manage tasks can be almost without being distracted by the mouse;
  • Quite convenient syntax when describing tasks and search queries;
  • GMail integration, a handy Sidebar for Firefox and a nice plugin for Launchy .

And so, what is the Todoist ? And it is a logo and two panels, one of which is reserved for tasks, the other for searching and a list (or tree) of projects.



So far everything seems to be clear: on the right we have a list of tasks from the project chosen on the left or what we have found by a search (top left).
Next we will look at Todoist on close to real situations. So it will be easier and probably more interesting. And so, my wife wrote to me in the gmail-box that I really needed to go to the nearest hypermarket and buy there a lot of things. And, of course, attached a huge shopping list:
  • A kilogram of potatoes;
  • Mineral water;
  • Any magazine to read;
  • Several finger batteries.

We need to organize all this and, having printed a piece of paper, go to the supermarket on the way from work. Or, alternatively, not to print anything, but to check on the spot directly through Todoist from a mobile phone or PDA. At the same time get used to the fact that we need to read with us at least once a week, preferably on Fridays.
We define that there will be two global projects on our site: ‚Äú Work ‚Äù and ‚Äú Personal ‚Äù. There will be no tasks in the projects, but there will be subprojects.
And so, we create a project - click on Add Project and see the appeared input field where we can enter a description. Confirm the creation with the Enter key, and refuse with the Esc key .
At the same time learn some useful chips:
  • To create a parent project that does not contain tasks, as well as parent tasks that do not require a check mark, you can put an asterisk (*) at the very beginning of the description;
  • With the Ctrl + Up arrow and Ctrl + Down arrow keys in edit mode, you can move from one element to another, higher or lower, respectively;
  • The key combination Ctrl + Right arrow and Ctrl + Left arrow we can control the nesting of elements. Those. in order to make an element nested in another, you need to select it and press Ctrl + Right arrow ;
  • A useful reference Reorder (the same effect is obtained with the r key) both in projects and in the task list allows us to redistribute the list with a simple muscle Drag & Drop (for those who do not understand: you can drag the elements with the mouse).

Attention! The most useful hot key in the system is ? (question mark). Use it more often until you are comfortable;)
And so, we created two global projects (yes, with an asterisk at the beginning). A curious user has already clicked on the arrow to the right of some project and chose the color you like. Then created in the project " Personal " subproject " Shopping ".
What is this zero to the left of the project name? Yes, this is the number of active tasks in the project. Now we will fix this case and add some task inside. Here, everything is absolutely the same as with the projects, but you can add a new task without the help of a mouse, simply by pressing the ‚Äúa‚Äù button (in the Latin layout). Already more or less accustomed to adding items? Well, it's time to learn something new:
  • Each task we can assign the time of its execution, for this there is a Date field.   • You can add a task from GMail if the appropriate plugin is installed. In this case, the task will take the form of a link leading directly to the box to the letter;
  • Tasks can be prioritized. Either through the menu using the arrow to the right of the task (the Priority item), or by adding a piece of the form ! P3 (3 in this case, priority) to the task description;
  • Tasks can be assigned tags (then it is quite convenient to search for them). To do this, add to the description of the task, for example @ purchase ( purchase here is the name of the tag);
