OutWiker. Open Source program for storing notes

Hi, habravchane.

The other day I posted a new version of the program for storing OutWiker notes. Once, more than a year ago, I already wrote about it on Habr√©, during this time the program was updated many times, overgrown with new features, that it makes no sense to compare the latest version of the program with its (posture) last year version.



OutWiker is a cross-platform (Windows, Linux) program for storing notes in the form of a tree. Distributed by OutWiker open source. A feature of the program is that all records are not stored in a single file, but the structure is formed from file attachments and folders. In addition, the program supports several types of pages: simple text pages, pages where HTML is used to format notes and pages in the format of a wikination (reminiscent of pmWiki).

For example, the following screenshot shows the wikitext for the first picture of the post.



For wikination, you can see its correspondence in HTML format. This feature can be used, for example, to design posts in blogs. Still, the textbook is more concise than HTML, therefore, IMHO, it is more convenient to use it for writing large texts.

If you do not want to use wikinacation, you can write notes directly in HTML format. In both formats, the buttons for inserting the most frequently used tags (for HTML pages) and wiki commands (for wikiscases) are displayed on the toolbar. There is also a type of pages without design - plain text entries, but there is nothing special to talk about them.

For each page you can set tags (tags), which are grouped in the tag cloud in a separate panel. Clicking on a tag will open a list of notes marked with this tag.



Recently, OutWiker began to support plugins, a list of which can be found here . At the time of this writing, the following plug-ins have been created: for coloring code in various languages, for exporting notes and tree branches to HTML format, for linking to LJ-users, as is customary in LJ itself, and a few more plug-ins related to page design in wikinacii. For example, the following screenshot shows the work of the Source plugin for coloring sources.



In the near future there are ideas for some more useful plugins.

It is possible to insert formulas in the TeX format in the Wikistand, insert attached pictures or their reduced thumbnails (there is a plugin in the plans for creating a gallery of attached pictures).



In addition, the Ukrainian localization of OutWiker has recently been sent.



OutWiker 1.6 introduced the ability to set page styles. The screenshots below show the same page using different styles.



As I said before, the program is cross-platform, Windows users can download the build for this OS, there is an archive with source files for Linux users, and a PPA is prepared for Ubuntu users on the program page .

Suggestions and comments are accepted. You can also leave your wishes on the site outwiker.reformal.ru , where you can also vote for the suggestions of other users.
