Visual Studio 2008 with 80% discount

Microsoft has announced a serious reduction in prices for Microsoft Visual Studio as part of the special initiative "Initiative for licensing software developers", which will be valid from July 28 to September 30. During this time, you can purchase Microsoft Visual Studio five (!) Times cheaper, that is, with an 80% discount. The program will be able to take part individual developers or small companies with a staff of programmers of no more than 10 people.

As they say, hurry, the number of licenses is limited! This is not a standard advertising slogan, but a real fact. Only 3,500 licenses will be distributed, and no more than five per individual developer or company. At the moment, only 3106 licenses remained to be issued.

The offer applies to the following software editions: Visual Studio 2008 Standard Edition Russian version, Visual Studio 2008 Professional Edition Russian version and corporate license Visual Studio 2008 Professional Edition Russian version.

As you know, the provision of large discounts on software in countries where piracy is most developed is part of the new Microsoft strategy .
