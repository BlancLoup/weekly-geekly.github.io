Remix OS for PC beta: how to organize dual-boot with Windows and start working with the new OS



On Geektimes, they repeatedly wrote about Remix OS, a desktop OS based on Android. March 1, a beta version of this operating system, which is already quite possible to deal. Generally speaking, this is a customized version of Android, which looks and works like a full-fledged operating system for a personal computer. It was first released in 2015, with minimal support for multiple devices. In January, the alpha version of Remix OS was released, the developers made it available to everyone by opening the possibility of booting the system from USB media.

The beta version has a dual-boot, with the ability to switch between two operating systems - Remix OS itself and Windows. You can install an OS on a hard disk or on an SSD, which provides a fairly high platform performance. Now the Remix OS installer for PC supports work with Windows 7 and later versions of this OS. A version for Mac should be coming soon. If the OS is tired, removing it is even easier than installing it.



Updates now come regularly; support for over-the-air updates is built in. There are other news, including the transition to the Linux kernel version 4.4 for better compatibility with newer processors, support for 1080p videos from YouTube and Netflix, work with USB drives up to 64 GB and so on.

What is not? Support Play Store. Since Remix OS is not certified by Google, there is no default access to the market. But you can add it, about this below. First, let's talk about installing the Remix OS itself.



Don't want to install Remix OS on SSD or hard disk? No problem. You can work with OSes directly from a USB flash drive, making it bootable. But still, the OS performance in this case will be lower than if you put it on an HDD or SSD.



How to install Remix OS on HDD or SSD



1. First, download the latest OS from here . There are 32-bit and 64-bit builds for the respective PCs.

2. Unpack the archive and get three files: a textbook with instructions, an installer and ISO, a disk image.

3. Run Remix OS Installation Tool.exe and check that the installation location is correct (drive letter, etc.).



4. Click Next and select the disk image that has just been loaded. It will be something like Remix OS for PC_x86_64-B2016xxxxxx.iso.

5. Start the installation and wait a few minutes until a message appears about the need to restart the PC to complete the installation.

6. Once the reboot has occurred, you can see a screen offering a choice of two systems.



7. We are waiting.

The next step will take from 15 to 30 minutes, depending on the performance of your PC. Remix OS creates its own 8 GB boot area and installs the OS into this area.

8. After everything is ready, Remix OS will boot for the first time, and you can start working.

9. If you need to return to Windows, select the shutdown icon, then the reboot option. After that, instead of Remix OS on the selection screen, click on Windows.

Can I get more than 8 GB?


The official installer Remix OS for PC takes for its needs 8 GB. This is enough to install various applications that may be needed for work or entertainment.



But if you suddenly need more space, you can add it .

How to remove Remix OS


If suddenly something went wrong, or just wanted to remove Remix OS, it can be done very quickly.

Initially, we run the installer again. It shows that the system is already installed on the PC, and offers to remove Remix OS. We agree, and after a few seconds the OS is deleted.

You can work and not with the installer, and delete everything manually. Select C: \ Remix OS \ in the address line of the file manager, and run the UninstallRemixOS.exe file . Then everything happens in automatic mode.
