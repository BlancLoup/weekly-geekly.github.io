IBM has updated data center management software

IBM has introduced a number of new technologies as part of its IBM Autonomic Computing data center automation program.

The new IBM Systems Director Active Energy Manager power monitoring and control system (the link will start on October 13) tracks server load and prevents power overspending. This technology is also part of the ecological project IBM Project Big Green and continues the tradition of the first of its kind PowerExecutive system, which appeared on the market in 2005.

Another new program, the IBM Tivoli Change and Configuration Management Database, is a universal management center for your system administrator. This is where data about servers, storage systems, network status, and installed applications flow.

In addition, IBM has updated the IBM Tivoli Usage and Accounting Manager software , which monitors processor utilization in the data center. After the update, the system does a better job of analyzing resources on the System p and System x server virtual machines.
