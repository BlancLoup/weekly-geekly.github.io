Programs auto-balance on the desktop

How do you view the balance on the bill in your phone? On Beeline you need to dial * 102 # and wait for an answer. Customary, but long. It would be fun if the balance was always shown on the screen, just a glance to check. There is such an opportunity, here is an overview of some products that display the current account balance on the desktop.

Dynamic Operator Name (Windows Mobile)

In automatic mode, it requests the balance on the account and displays it instead of the name of the GSM-operator. Balance check is performed after each call and the device wakes up.
Prog - free, written by domestic developers.
Download Dynamic Operator Name 3.5 beta 3 for WM5-6.5 . Need to install in the main memory.
Due to the features of the OEM software, the program does not work on the following devices: Samsung I900, I780; Toshiba G900.

dciBalance (Windows Mobile)

Unlike the previous program, it monitors the balance of mobile communication right on the desktop (Today), can be used in conjunction with the Dynamic Operator Name. Free, open source.
Benefits:
* Configure information entry line and dialer number
* Frequency check
* Ability to check after the conversation
* Small size
* Easy to configure
At the moment, the development of the program is terminated.
Download dciBalance 0.1.2b for WM2003-5-6.5 . Install into main memory.

MVBalance (Windows Mobile)

Designed to display on the Today screen the remaining funds on your phone.
Distinctive features, in addition to the standard set of functions:
* the ability to ‚Äúcolorize‚Äù the plugin a little for yourself (color, font, icons, text alignment, etc.)
* pronouncing the balance by voice
Download MVBalance 2.5.9
Does not work on the following devices: Samsung I900, I780; Toshiba G900.

Touch Balance (Symbian)

Analog for Symbian.
Functions:
* auto balance inquiry after making outgoing call
* auto balance inquiry after sending SMS / MMS
* auto balance request by timer
* autoload
* customizable USSD request number
* setting up the phrase displayed on the desktop (introductory word, final, update time, difference from the last update)
* test compatibility with handy shell
Compatibility: Symbian 9.1-9.4
PS: Initially, the post was about one program, but in the comments there are questions and tips for other platforms - so I decided to make a list of products with similar functionality.
