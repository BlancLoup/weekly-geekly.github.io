Preparing for a nuclear war



It would seem that it is easier to prepare for a nuclear war, or rather, to survive its consequences.

Dig a bunker, install air purification systems there, stock up on food and water ...
But the devil, as you know, in the details.

Have you thought about what you will be doing in the intervals between eating, sleeping and forays to shoot mutants?

Here!
It is necessary to read something.
Since the Internet, newspapers and magazines will end, it is necessary to stock up with a reading material in advance.
We were lucky - even 30 years ago for this purpose we would have to order a pair of trucks of books.


And now everything is much simpler.

Step 1. Wikipedia



Wikipedia is not only interesting reading, but also useful! Be sure to have on hand anyway.
For the offline version you will need to download a fresh wikipedia dump in your favorite language. No pictures, users and discussions - only the texts of articles.

For example, the Russian version (gigabyte archive bz2) is here:
dumps.wikimedia.org/ruwiki/latest/ruwiki-latest-pages-articles.xml.bz2

English - here (8 gigabytes):
download.wikimedia.org/enwiki/latest/enwiki-latest-pages-articles.xml.bz2

WikiTaxi is offered as a reading program .
The program does not require installation and is easy to use: first, using the WikiTaxi_Importer.exe utility, we convert the downloaded dumps into the internal format of the program (it looks like something like compressed SQLite). The file size is about twice the dump.



You need to select the file with the source dump, then the file that we will create, and click "Import now!". Well, wait a bit. On my Intel Celeron E1500 2.2 GHz / 2 Gb RAM computer, the conversion of an 8-GB English base took a little less than three hours.

Then we load the file in the program itself and read it.
Internal links and search work. What is especially nice is that the program works with unicode, so the search and display of the Russian wiki works without problems.


Update : on downloading pictures. Wikipedia has closed such a service due to legal considerations. However, there are programs and scripts, for example. Pulls all links to images from the XML-dump and downloads them.

Step 2. Read in Russian



The free library Flibusta, well-known in book circles, encourages its distribution. As in the case of Wikipedia, there is an archive of the library and programs for reading from the archive.

Archive can be downloaded here (167 GB):
rutracker.org/forum/viewtopic.php?t=3170771

Now there is an open registration and the ratio is not taken into account.
The archive is periodically updated, with updates made using additional files. No need to re-download everything every time.

In the same archive there is also a program for building a library and reading MyHomeLib.
Does not require installation. At startup, select the ‚Äúexisting library (inpx)‚Äù option, and specify the path to the index file flibusta_all_local.inpx, as well as to files with books.


After the process is over, you will have a large library with easy navigation. 225 thousand books should be long enough.


Step 3. Something to read



Most habrovchan know English in varying degrees, so for a change, I recommend to stock up also with fiction in English.
To do this, the collection is distributed, distributed through a single site with a ship on the logo (23 GB). In the collection of about 80,000 books and 9000 authors, for convenience, the list is distributed separately .

The user bill_g, who flooded this collection, seems to be our man - judging by his torrents , for example, " Technika molodyogzi (Techniques Youth) 822 numbers DJVU) ".

The collection is sorted by author - but, unfortunately, not as clearly as flibusta. There are duplicate texts, some of the texts are in pdf, but most are in plain text format. The most recent files are dated 2008. The collection includes almost all authors, from antiquity to the present day.
