HabraFx to take off

Greetings, Habrazhiteli!

While some on Habrahabr argue about the future of the resource, we are trying to make life easier for some people. For this purpose, I created an extension called HabraFx , with the help of which users of the Firefox 3 browser will be able to get some additional features for working with Habrahabr.


Where to get?


You can install the extension itself right now , and, to facilitate information on the extension and the extension itself, a whole site was created by Comrade CurlyBrace , for which many thanks to him!

Goals


It seems to me that the goals I set are quite obvious:
  1. Learning XUL with all application technologies (XPCOM, JavaScript, and others) to create Firefox browser extensions;
  2. Actually, the empowerment of Habrahabr through the introduction of various experimental innovations;
  3. Create a small tutorial on how to create Firefox extensions in the future (if required);
  4. Opening the extension to be able to view its source code: all you need to do is unpack your habrafx.xpi with your favorite archiver.

Current opportunities


At the moment you can see:
  1. Monitoring karma / habrasila. Monitoring the position of habrachelovek in the rating is currently excluded from the extension due to an error in the work of the Habrahabr API, which returns incredibly huge numbers. UFO will understand ;
  2. Monitoring of changes in points (rating) and the number of comments of your last topic;
  3. Personal ban (blocking) blogs;
  4. Blocking users: hides the comment branches, where you found a disagreeable user with the ability to still read this branch (if there is an irresistible urge to do this).
The extension is compatible only with the third branch of Firefox.

However, this does not mean that HabraFx will remain at the same level of development as it is now.
