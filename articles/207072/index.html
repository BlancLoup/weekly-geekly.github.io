This is the third edition of the book. Before that, MacFarland had published books on CSS with fairly large print runs, but in this book, which came out with significant changes, some of which I myself used on books.piter.com

With the help of CSS3 technology (cascading style sheets), you can create unique, modern website designs. But even the most experienced web developer may not know all the tricks of using CSS3. After reading in this book a lot of practical examples, as well as tips, you will go to a new level of website creation using HTML and CSS. You will learn how to develop web pages that work equally fast and look the same, both on PCs and smartphones and tablets.

The book covers the following topics:
  • writing HTML5, CSS3 tags that are recognized in all browsers;
  • text formatting, adding navigation pages;
  • creating tables and forms;
  • web design techniques for creating unique website design;
  • website creation for any device using responsive design.
