News and offers to developers of Windows Phone with MWC

Today a post with interesting information and no less interesting special offers appeared in the Windows Phone Developer Blog . Could not resist not to share with the community!

First, some interesting facts about the Windows Phone platform at the moment.

Significantly increased activity in the Windows Phone Store - the maximum in the entire history - c 75% increase in the number of application downloads, a 91% increase in revenue from paid applications and more than 1 billion transactions. Also, thanks to your excellent applications, the number of applications downloaded by one user increased to 55 .

The Windows Phone community is also growing rapidly. More than 40,000 developer registrations in the first 90 days since the launch of Windows Phone 8, and we already see 15,000 new applications that use the new functionality provided by the Windows Phone 8 platform. In general, a catalog has been created with more than 130,000 apps and games specifically designed for Windows Phone UI and Ecosystem. And we expect growth to continue, as the number of downloads of the Windows Phone 8 SDK has exceeded 500,000 in less than 4 months since its release on October 30, 2012.

Dev Center application


Download the application that will allow you to access your Windows Phone Dev Center Dashboard directly from your phone:

1) application downloads, failures, fees and key metrics;
2) detailed reports on the highlighted application, including translation of reviews into the selected language;
3) data on downloads and failures on Live Tile;
4) the ability to send a link to your application without having to enter the store, copy / paste, etc.

Special offers


The Box platform provides an API that gives the developer access to all the content management capabilities of Box that can be integrated into your applications. What is available on offer?

Free account with 25 GB of free space for all time (instead of 5 GB). One (1) Enterprise Seat (including: administration, full-text search and version history, accessible via the web interface and API)

Learn more about the offer: aka.ms/wpdevofferbox

Buddy is a mobile backend as a service that allows developers to build applications and games connected via the cloud. Using Buddy, a developer can add features such as messaging or chat, user account systems, support for geo-location, photo albums, Live Tiles (and much more) without the need to engage in server development. What is available on offer?

Free Buddy account with access to up to 1 million API (1MM) API per month.
The offer is valid for 6 months. After 6 months, the developer should start to pay at the standard (published) Buddy prices.

Learn more about the offer: aka.ms/wpdevofferbuddy

ExitGames Photon Cloud allows developers to create a multiplayer real-time game for Windows Phone 8, Windows 8 and other major platforms. What is available on offer?

Free Photon Cloud service (500 CCU plan) for 6 months.

Learn more about the offer: aka.ms/wpdevofferexitgames

In order to take advantage of these offers you need a unique code. Registered Windows Phone developers receive their code along with a mailing list for Windows Phone developers (Windows Phone WPDEV newsletter). If you have just registered, welcome! You can email wpdevcenteroffers@microsoft.com .

These are offers from specified companies, not from Microsoft. Additional conditions may be required in accordance with the requirements of the service provider.
