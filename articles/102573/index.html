Fedora 14 (Laughlin) Alpha released

Today, with a delay of more than a week from the original plan, Fedora 14 (Laughlin) Alpha has been released.


Those who want to try out can already download from mirrors, torrents and Jigdo.

under the cut list of changes.

Changes for users



and the most important is kde-4.5 and Qt 4.7 (was kde-4.4.2)

other changes in versions of GUI applications are relatively small, unless it is worth highlighting
  • openoffice.org-3.3 (was openoffice.org-3.2)


Changes / buns for admins and advanced users.


  • kernel-2.6.35 (was kernel-2.6.33)
  • systemd is the new default init system.
  • use of grub2 (grub-0.97 also remained, apparently for compatibility)
  • Support for Amazon EC2 cloud (the latest release that was targeted for EC2 is Fedora 8, which has not been supported for a long time)
  • python-2.7 - everything is tied to this version of a huge job, because half of gui in redhat-based is tied to python (was python-2.6.4)


Big changes for developers


updates for languages, libraries and environments:
  • perl-5.12 (was 5.10)
  • python-2.7 by default (was python-2.6.4)
  • boost-1.44 - libraries for C ++ (was boost-1.41)
  • Erlang R14 - (was R13).
  • D Programming - Added D (LDC) and D standard runtime library (Tango) compiler
  • Fedora's Eclipse stack to Helios releases (I'm not a developer, and that says little to me)
