Big Evernote iPhone Update: Offline Notebooks and More


An unexpected New Year's gift! The assistants of Santa Claus from the Apple AppStore in some magical way managed to open for users the biggest Evernote update for iPhone (version 3.2), which we have been preparing for the last months. New products are really enough: offline notebooks that can be viewed without connecting to the network, increasing the speed of work, editing notes containing formatting, more time for audio recording, etc. Well, in addition, you can now issue or extend a premium subscription directly from the application for iphone.

Download Evernote for iPhone from the AppStore >>

As usual, it is recommended to restart the phone after installing a new application.


Note caching
All notes that you create or view on your iPhone will now be cached, meaning that you can access them later without connecting to the Internet. This feature is available to all users.

Premium feature: offline notebooks



Premium subscribers can now select certain notebooks (or all) and force them into the cache: all entries in these notebooks will be available if there is no Internet connection. Click on the new tab "Sync" and select the appropriate item to configure this feature.

The larger the notebook, the more time it takes for it to be fully synchronized, but there is good news ‚Äî you only have to do it once. To speed up the process, it is better to synchronize over a WiFi connection. Please note that the application will not be able to download all your notes if you mark more notebooks than the free space on the phone allows.

[Important: If the size of notebooks for offline storage is incorrectly determined on your phone, delete the application from the phone and reinstall it.]

Registration or renewal of premium subscriptions from the application


Get a premium account to try out all the new features, or extend the existing premium subscription is now very easy - right from the new tab "Synchronization".

Dictation in a big way
In the latest version doubled - right up to 20 minutes - the possible time of voice recording.

Local search
Search now works offline, so you can search for notes among favorites, cached entries and offline notebooks. If you have an Internet connection, the application will automatically expand the search area to all notes, both on the phone and on the server.

Editing formatted text


If you start editing a note that contains formatting, pictures or attachments, you will be given the opportunity to add a new text directly to the end of the note (as before) or make a copy without formatting (in this case, the original note will be placed in the basket if you suddenly want restore it later).

New tab "Sync"


All synchronization and account management functions are now combined into one tab, where you can monitor the status of all synchronization operations in real time.
