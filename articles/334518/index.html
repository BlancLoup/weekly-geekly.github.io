"Pot, cook": 50 tools for managing development

The inefficiency of business processes, according to IDC research company, "eats" from 20 to 30% of business income. One of the key sources of low efficiency are routine tasks that could be automated. Automation can save time, money and save headaches. It is a fact. According to McKinsey, process automation can reduce costs by up to 90%. Therefore, investment in relevant instruments is more than justified.

At the corporate level, solutions that automate key company service processes in the financial, legal, and personnel sectors are popular. For example, one of such tools is the ServiceNow platform, the implementation of the functionality of which is handled by IT Guild .

However, it is important for companies to take care not only of the automation of global processes, but also of what happens at the table for each employee. Therefore, we have prepared a small selection of tools that simplify the management of development and the lives of developers.

/ Flickr / Florian Richter / CC

Technology is the key to increasing efficiency in an organization. So say 96% of executives surveyed by the independent research agency Loudhouse. According to the study, 53% of those who, in their efficiency programs, rely on technology, are more confident of success than their colleagues less open to technological tools.

Task Management


  • Trello : project management and task distribution among team members
  • Asana : another task planner focused on team projects
  • Basecamp : an online project management tool
  • Runrun.it : delegation of tasks with a detailed description and deadlines
  • Evernote : helps control personal tasks
  • TimeCamp : project management with budgeting and reporting
  • Ecamm Call Recorder : recording business calls for further accounting of tasks
  • Standup : generating progress reports
  • Confluence : to collaborate on content
  • Bitbucket : distributed version control system

Time management


  • Calendly : a tool for organizing virtual meetings
  • Freeter : collects everything you need to work on the project
  • Mighty Networks : a platform for comprehensive preparation for meetings
  • Timeneye : for time management

Work with mail


  • IFTTT + Google Drive : export files from emails
  • DragApp : management of incoming messages
  • Cleanbox : unsubscribe from all mailings in one click
  • Boomerang : will show whether the recipient has read the letter

Collect, synchronize and prepare data


  • Tiny Reminder : collecting tasks and customer files in a convenient form
  • Agenty Chrome Plugin : plugin that extracts information from any site
  • Beyond Compare : compare files and folders with simple commands
  • Syncthing : sync and backup data
  • Koala App : A cross-platform automatic compilation application.
  • Cyberduck : FTP client for easy file transfer

Prototyping


  • InVision : for team evaluation of ideas before directly creating a product
  • XAML Designer in Visual Studio : an option for fans of this development environment
  • TranslateKarate : a toolkit for quickly localizing content
  • Axure : prototyping interfaces

Code writing


  • StackEdit : a free online editor with markup
  • Eclipse : Java IDE for autocomplete, refactoring and syntax checking
  • Jet Brains Resharper : snippets and code templates
  • Source Code Generator : source code generation for any language
  • Snippets : Snippets Manager
  • Sublime Text : source editor
  • Postman : a platform to simplify API development
  • Gulp : a tool to automate time-consuming tasks during the development process
  • Atom : another source editor

Work with databases


  • DaDaBIK : simplifies database migration
  • Devart : to compare databases
  • OFFSCALE : database monitoring

Identifying errors and problems


  • Pivotal Tracker : Teamwork Optimization
  • Code Climate : automatic code analysis
  • FindBugs : find bugs in java code
  • Rollbar : real-time bug tracking
  • Monit : monitoring and automatic server maintenance
  • SonarQube : code quality check
  • Jira : bug tracking system

Testing


  • BrowserStack : ensuring compatibility with most devices
  • JUnit : an integrated unit testing environment
  • Selenium : environment for testing on various browsers and platforms
  • Rational Functional Tester : automated testing with a variety of software environments
  • CircleCi : simple and fast automated testing
  • Hurl : API Testing Tool

Feedback collection


  • Satismeter : user satisfaction rating due to feedback analysis
  • UserVoice : a tool for collecting and analyzing data from users
  • User Testing : analysis of product usage data
