Photoshop customization

Photoshop use constantly. For creative purposes - once. And for work, with the layout of sites - two.
Studied the program with interest. Found a lot of books. But only the one that changed everything can be recommended to everyone: Photoshop CS Killer Tips by Scott Kelby & Felix Nelson.
The level of professionalism increases significantly. Due to keyboard shortcuts and special techniques, the speed increases very much. But the truly comfortable PS becomes after the autograph customization, to which this book prompted me.

To begin with, I will give a screen of my program window:

As you can see, first of all it is noticeable that the number of palettes is very small. I just closed all the unnecessary (in my experience as a layout designer, 9 is enough. Although, to whom as), and arranged the rest into groups.

Another convenient tool for the psop user is setting up shortcuts. You can read a lot about standard combinations in the above book, or any other Photoshop textbook. And user settings are made in the menu Edit-> Keyboard Shortcuts (or Alt + Shift + Ctrl + K ).

Returning to the palettes, I will say that as one of the ways to control them, I came up with a key combination, which is quite convenient when switching between panels:

It is convenient when you use only a certain combination of palettes and remember their order.

Simple Ctrl + Alt + C (Canvas size) and Ctrl + Alt + I (Image size) combinations accelerate routine operations.

The following key combination: Ctrl + Alt + Shift + G for filter blur on the Gaus. Used quite often.

And one more: Ctrl + Alt + Shift + J for the Despeckle filter. To some extent, if not abused, helps get rid of JPEG image compression artifacts.

In addition, the speed of the Shift key can be important for the speed of work: you need to uncheck the Edit-> Preferences-> Use Shift Key for Tool Switch and then switching between tools will be done in a magical way when you press only one key with the desired letter.
