Mandriva Linux has been released - Mandriva 2010 Spring

As reported in the company , there is in the new version and special Russian features.

The Mandriva 2010 Spring distribution will be available in four versions:

Mandriva One is the simplest user version of Mandriva, which can be used both in the LiveCD mode (directly from the disc) and installed on the hard disk. Also available is the option to install Mandriva One on a computer that does not have a DVD-drive, with USB-flash. This version includes commercial drivers to simplify the work of novice users, and fits on a single CD,

Mandriva Free is a version of the distribution for both workstations and servers, including only free applications. Suitable for users already versed in GNU / Linux,

Mandriva Powerpack is a version for workstations and for small servers, most often used in organizations. It includes technical support, a wider set of software packages, third-party proprietary applications, the Fluendo multimedia codec set, which supports mp3, wma, wmv formats, and the Fluendo DVD player. The Russian version of the distribution kit additionally includes a number of applications important for Russian users: wine @ etersoft software for running Russian business applications under Microsoft Windows in emulation mode (such as 1C: Enterprise, Garant, Consultant, FineReader, etc.) in the GNU / Linux environment , a free medical information system, a program for recording business transactions of an individual entrepreneur, Traffpro Internet traffic accounting software and other software.

Edumandriva is a solution for educational institutions based on Mandriva, created by teachers for teachers, included in the official free software repository for schools of the Ministry of Education and Science of the Russian Federation (www.spohelp.ru).
