WordPress 3.7 will be updated automatically.

The first beta version of WordPress 3.7 was released yesterday, and now it will be updated automatically.

By default, automatic updates will only concern technical releases (for example, from 3.7 to 3.7.1) and for major releases (for example, from 3.7 to 3.8), you will have to update manually. Of course, this is all easily manipulated with the help of filters and directives in the WordPress configuration file, and WordPress itself is smart enough not to automatically update the settings made via Git or Subversion.



It is also worth noting that the beta versions of WordPress will be updated every night before the first official release, which means that testing WordPress has become even easier. You can install WordPress 3.7 beta using the official WordPress Beta Tester plugin.
