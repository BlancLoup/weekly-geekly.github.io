Backup My App - backup sites on Ruby on Rails

Hi, ruby-on-rails-developers. I work as a programmer in theories and practices and I want to tell you about Backup My App - the backup service for ror applications that we use.

They came up with it and made the guys I work with; they somehow found out that some hosting companies are able to clean the data cleanly and immediately after that they began to dream of a simple backup.

How it works

Service fee: there is a free tariff plan for just one application per gigabyte, or you need to pay $ 5 per month for 5 gigabytes, and $ 60 for 100 gigabytes. After registration you need
  • install plugin
    $ script/plugin install git://github.com/crocoduck7/backupmyapp.git
  • start the generator
    $ script/generate backupmyapp KEY=
and then send the changes to the server. After this site - code, database, files - will start to backup.

What will it give



It will give daily automatic backups with preview in the browser, loading, restoring and viewing statistics for free. I think this is a miracle.
