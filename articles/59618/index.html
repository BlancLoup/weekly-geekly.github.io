Windows 7. Burning discs

It so happened that when I installed Windows 7 (RC 7100), I did not inquire about programs for burning CD / DVDs, including creating and burning images. I hoped for Roxio Creator, which I used on both Windows XP and Windows Vista.
Accordingly, one day, at the right moment, I realized that Roxio on Win7 does not work, and, the system hangs completely and only the Reset button helps. I found a solution in the form of copying the milcore.dll library, downloaded from the Internet, to system32. Did not help. The same result.
Well, having understood that you can forget about Roxio for a while, downloaded Nero , received a trial key, installed it and surprised the result no less: the system thought for a long time, and then involuntarily rebooted. Solutions for Nero, I did not look.

I complained about the problem to my friends, I was very surprised when I learned that they were not even interested in solving this issue, but stupidly returned to the old OSes.

Digging into Google, I found a couple of programs: one for burning discs ( BurnAware Free ), the other one works fine with images ( ImgBurn ), as I later found out. I didn‚Äôt even know about these softinks. I tinkled with them for half a day and was pleased.
