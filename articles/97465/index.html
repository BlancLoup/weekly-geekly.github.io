Civ 5 will be able to import maps from Civ 4

The developers of Civilization 5 reported that they had introduced into the game a mechanism for importing maps from Civilization 4. Thus, you can start the game right now - and then import the entire map into Civ 5, including all cities, units, resources, etc.

It is interesting how this will be done technically if in Civ 4 the territory is divided into square sections, and in Civ 5 it is hexagonal. Most likely, an algorithm with a shift like this will be used.



This is a good example for other game publishers who do not allow users to freely distribute the created cards or import old ones, but instead earn money by selling their own cards by offering them for a separate price.
