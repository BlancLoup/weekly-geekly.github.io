We draw volumetric images in Inkscape

Inkscape is a relatively young but progressive general vector editor (like Adobe Illustrator and Corel DRAW). The program is cross-platform, distributed free of charge and with the source code. The main image format is SVG (scalable vector graphics, W3C standard , recommended for publishing vector drawings on the Internet).

I want to tell you about the work with the program, and at the same time how to fill a vector image with a volume using the example of creating my picture with a snowman.
