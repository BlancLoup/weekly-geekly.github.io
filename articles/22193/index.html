Google Summer of Code 2008

Tomorrow, March 24, will begin accepting applications from students and graduate students to participate in Google Summer of Code 2008 .

Google Summer of Code is a good opportunity to make some extra money in the summer working on one of the many open source products: PostgreSQL, Python, PHP, FreeBSD, and many, many others (a list of projects participating this year can be found here: http: // code .google.com / soc / ).
