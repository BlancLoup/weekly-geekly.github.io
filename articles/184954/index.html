Windows 8.1 (aka Blue) Preview available for download

Today, Microsoft has posted .iso-images of Windows 8.1 (Developer Preview edition) for everyone here (x32 / x64 versions in various languages). 8.1 is not a new OS, because MS is positioned as an update for Windows 8, which is available through the Windows Store. We would like to talk about the new features of the OS, which, in our opinion, stand out among others.



Start Button

One of the most anticipated "innovations", compared with Windows 8, is of course the appearance of the Start button.



New kernel NT 6.3!

The ntoskrnl version of the file directly hints to us that the kernel has been reworked and is now version 6.3 (NT 6.3).



The following are the ntos kernel versions for Windows 7 SP1, Windows 8, Windows 8.1 (6.1 -> 6.2 -> 6.3).



Internet Explorer 11 Browser

IE 11 only comes with Windows 8.1 so far.
