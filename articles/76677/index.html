Creating a library using Handbrake and MetaX

This review will discuss how you can create your own library. It will review the process of converting video and formatting the library using such programs as Handbrake , MetaX and Apple iTunes .

The example will show the process of converting a DVD-Rip movie to MP4 format for viewing on devices such as: Apple TV , iPhone and iPod Touch . There will also be considered a free application MetaX , which is designed to add tags to files in MP4 format and its derivatives.


At home I use an Apple MacBook 13 " with Mac OS X 10.6.2 installed. But for many Windows or Linux users, the review may also be useful, since Handbrake is a free open source application ( Open Source ) and is available for such operating systems Systems like: Mac OS X , Windows XP / Vista / Windows 7 , Ubuntu and Fedora . MetaX is also available for Windows and Mac OS X users, and Linux users can use EasyTAG .

So let's start with the most important thing , namely the process of converting video using Handbrake , which is a free application designed to convert MPEG video (including DVD-Video) to MPEG-4 video files in MP4 or MKV format.

Input format:
  • Video : Any DVD-like source: VIDEO_TS folders, DVD image or real DVD (encoded or non-encoded), PAL or NTSC.
  • Audio : AC-3, LPCM or MPEG audio.


Output format:
  • File Format: MP4 and MKV
  • Video : MPEG-4, H.264, or Theora (1 or 2 pass encoding)
  • Audio : AAC, MP3, Vorbis or AC-3 and DTS pass-through (support for encoding multiple tracks)


Additional features:
  • Chapter selection (DVD Chapters)
  • Subtitle support
  • Built-in bitrate calculator
  • Resizing pictures
  • Black and white coding


Current program version 0.9.4:
  • Mac OS (10.5 / 10.6) : 32-bit , 64-bit , Power PC
  • Windows : XP / Vista / Windows 7
  • Linux : Ubuntu 9.10 (32-bit) , Ubuntu 9.10 (64-bit) , Fedora 12 (32-bit) , Fedora 12 (64-bit)


Official website of the program: www.handbrake.fr

Handbrake Setup

In the General tab, I recommend choosing the option Use iPod / iTunes friendly (.m4v) file extension for MP4 , in the Audio tab, specify the language for audio. By default, the specified language, if present, will be the first in the list of languages.



If you use the MetaX program to add tags to MP4 movies, I recommend selecting the option Send file to MetaX in the General tab. Thus, after successful conversion, you can add a description of the film, tags, etc. in your movie.


Select coding source

To select the source of encoding, you must click on the Source button and select a DVD disc or file. In this example, I chose the MKV file format. After the program analyzes the structure of the disk / file, you will be able to specify additional options. For those who do not want to go into the details of the encoding settings, there are many ready-made templates ( Presets ). For example, for an iPhone or iPod Touch, you can select the Apple Preset -> iPhone & iPod Touch . But I would recommend setting your preferences and saving them as your reset for later use.

Video Settings:


  • Video Codec : H.264 (x264)
  • Average bitrate (kbps) : 900

The 900 kbps bit rate allows you to comfortably watch a movie not only on the iPhone / iPod Touch, but also on Apple TV and even on a MacBook 13. " .
  • 2-pass encoding : I recommend choosing the two -pass encoding option


Screen resolution settings:

To change the resolution settings, select the Picture Settings (‚åò‚áßP) item in the Window menu. In the window that appears, you can specify the size you need. If you need to create an anamorphic video, then you need to set the Anamorphic parameter to Loose . If you watch the movie not only on the iPhone / iPod Touch , but also on Apple TV, I recommend setting the width to 640 pixels. Otherwise, you can set the width to 480 pixels, as this will also save disk space.



When you click on the Preview button, you will be able to select the correct width of the screen and make sure that there are no black bars. In my example, I set the Anamorphic parameter to None , selected the Keep Aspect Ratio option and specified a width of 640 pixels.

Sound settings:

In the Audio tab, you must specify the audio tracks that will be added to our final file. In addition to the Russian track, I prefer the film to contain the original track in English. Watch movies in the original also contribute to the study of English.
  • Audio Codec : AAC (facc)
  • Mixdown : Stereo
  • Samplerate (khz) : Auto
  • Bitrate (kbps) : 160



For movies, I recommend installing Samplerate and Bitrate at 48 khz and 160 kbps respectively. For serials, it‚Äôs probably worth setting the values ‚Äã‚Äãto 44 khz and 96 kbps to save disk space.

Chapters settings:

Usually movies on DVD are divided into chapters. If you wish, you can save these chapters in the final film. To do this, in the Chapters window, select the Create chapter markers option and specify the correct title of the chapter next to each chapter in the Chapter Chapter section.



Subtitles (Subtitles):

The latest version of the program has the ability to add subtitles using SRT files. I really did not have this option in the previous version, so I had to use the iSubtitle program additionally . There are quite a few sites with a collection of subtitles in SRT format, for example:
  • http://subs.com.ru
  • http://ru.moviesubtitles.org
  • http://www.vsesubtitri.ru


You can download the SRT file to your disk and add it to a convertible movie by clicking on the Add External SRT ... button.



Well, that's all, the basic settings are made and you can begin the process of encoding. Handbrake program allows you to encode several films in turn. I usually use this feature and put on the coding of several films for the night. To add to the coding queue it is necessary, after specifying all the settings, click on the Add to Queue button. You can view and start the queue encoding process by clicking on the Show Queue button.



Saving conversion settings:

Selected settings can be saved by clicking on the Toogle Presets button. An additional block with the current templates will be added to the main program window on the right. At the bottom of the block, click on the + button. A window will appear where you can specify the name and description of your template, as well as the option to save the dimensions of the Use Picture Size resolution.



Conversion process:

You can monitor the conversion process both in the main window and in the Show Queue window or on the Mac OS X Dock itself .



Upon completion of converting the entire queue, a dialog box appears.



If you have selected the option Send file to MetaX in the program settings, then the MetaX program will automatically open.

MetaX is a free application for adding tags to MP4 files. Metadata for foreign films can be downloaded automatically from the Amazon.com catalog. In other cases, you will have to edit the tags yourself.

Current version of MetaX:
  • 2.4.6 for Mac OS X Leopard
  • 2.3.0 for Mac OS X Tiger
  • 2.0 for Windows


At the very beginning of opening a file in MetaX, a window will appear with the choice of the type of video file: TV Show , Music Video or Movie .



In this case, since we have a regular movie, we choose the Movie option. MetaX will automatically try to retrieve all the necessary tags from the network. However, it is better to add all the necessary information about the film.

For my library, I fill in the following fields:
  • Title - The title of the film in Russian and the original title of the film.
  • Artist (Show) - Artists list
  • Date - Year
  • Genre - Movie Genre
  • Short Description - A brief description of the film

Also be sure to add the cover of the film. Approximately this is how the window of the MetaX program looks like with filled information about the film.



This information can be obtained using sites such as Film Search or IMDb .

Since MetaX also supports queues, you can immediately add descriptions to several movies at once.

After you have added all the necessary information, you must click on the Write & Share button. After successfully adding the necessary tags to all the films in the queue, the end window will appear.



After that, you can change the name of the file, for example, 2009 - Moon.m4v. Now in the system it will look comfortable and beautiful.
