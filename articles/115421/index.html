Transition to C # and the .NET Framework for Java developers

Microsoft has prepared a set of materials for those java-developers who are interested in the .NET platform and the C # programming language and want to learn more.

The package under the general title Moving to C # and the .NET Framework for Java Developers includes the following materials:
  • CSharp for Java Developers pivot table;
  • CSharp for Java Developers slide show video;
  • video demonstration of Introduction to Visual CSharp Express;
  • document Introduction to building dotNet Mobile Applications for Java Developers;
  • document Introduction to building dotNet Web Applications for Java Developers;
  • video Introduction to the Common Language Runtime;
  • Introduction to Windows Azure for Java Developers;
  • Document Introduction to Windows Client Technologies for Java Developers;
  • document Introduction to Windows Technologies for Building Enterprise Applications for Java Developers;
  • Video Overview of the .NET Framework;
  • The .NET Framework for Java Developers;
  • slide show video What is the Entity Framework for Java developers.
You can download a package of documents and video materials separately on the next page on the Microsoft download site.
