The final release of Vivaldi 1.3


Hello!


Summer is a difficult period to develop. Developers and users are leaving on vacation, so fewer employees can respond to fewer messages from voluntary testers and just users. But, be that as it may, the process does not stop for a minute. And today we are pleased to present you a new stable build - Vivaldi 1.3. From the most notable innovations can be noted:


  • Customizable Themes
  • Built-in color palette
  • Configuring IP Address Processing for WebRTC

Well, now about all this and much more.



Customizable Themes



Here we continue the process of expanding the possibilities of customizing the browser interface. To date, it has become possible to choose a theme from the already existing ones, and to make their own changes in them, thus creating new ones of our own. You can change the color of the interface elements, change the contrast settings, as well as experiment with rounded corners.


IP address translation settings for WebRTC


And this setting appeared due to our constant attention to privacy issues and keeping personal user data safe. Services based on WebRTC technology are gaining popularity in the network, but sometimes this creates a problem: often these services require the translation of network addresses and thus can easily determine the real IP address of the user. Little joys for Linux users


For Linux users in the new version of the browser Vivaldi special improvements have been made. First of all, the function of unloading open pages from memory has been improved. When using this feature, the computer does not waste system resources.
