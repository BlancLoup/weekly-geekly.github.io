Soap: all-in-one router for smart home



Recently, various companies are increasingly producing smart devices that really make life easier. In addition, in our homes there are new and new wireless devices, which are difficult to control (at least the access of such a device to the Network).

Wi-Fi printers, smart TV, tablets, game consoles, not to mention PCs, tablets and smartphones. How to control all this? Recently, an interesting development has appeared, which, as far as can be judged, is quite capable of controlling access to the Network for each home (well, or work) device. This is a smart router Soap. This is not just a router, but a whole system for managing a smart home.

The developers have equipped the system with most of the popular wired and wireless interfaces, so the capabilities of the Soap are quite wide.

Soap as a four-channel 802.11ac router





One of the main functions of the device is a router. The system is four-channel, with two channels of 2.4 GHz spectrum and two 5 GHz channels. With this approach, the router provides access to the wireless network at speeds up to 2GB / s.

The system has built-in protection against viruses / malware, so, according to the developers, there is no need to install antivirus software on home / work systems separately. There are also functions of parental control. If necessary, you can remotely disable any of the devices connected to the wireless home network (for example, a child‚Äôs tablet, which now has to do homework, and not watch the next series of Fixics on the Web)

According to the developers, the system allows you to work with almost any wireless device, with any of the presented communication modules: Z wave, Zigbee, WiFi, Insteon, X10, 433 mhz, Audio, IR, BLE, as well as some others. As far as one can understand, the device can even operate a garage door, opening and closing them at the direction of the owner.

The device is controlled from a single software package Soap Automation, compatible with the Apple Home Kit.
