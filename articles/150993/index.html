Windows Server 2012 is available for download.

On September 4, Microsoft announced the global availability of Windows Server 2012. You can get Standard and Datacenter editions as part of MSDN or TechNet subscriptions, or download the 180-day trial version (also in two editions) in the Download Center.

At the same time with the server, it became possible to download:
  • Microsoft Hyper-V Server 2012 Free Hypervisor
  • The Windows Management Framework 3.0 toolkit, which includes the new PowerShell 3.0 and updated versions of WMI and WinRM. All of these components are included with Windows Server 2012, and Windows Management Framework 3.0 will allow you to install them on Windows 7 SP1, Windows Server 2008 R2 SP1 and Windows Server 2008 SP2.
