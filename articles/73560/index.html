Google introduced Android 2.0 (Eclair) - SDK

Google has released a new version of the SDK (application development package) - Android 2.0 (Eclair).

Android 2.0 is the main platform for Android gadgets, the release of phones on this platform is planned for November 2009. The release includes new features for users and developers. For platform developers, Android 2.0 is available as a downloadable component for the Android SDK. The SDK includes a fully compatible Android library and system image, as well as a set of skins for the emulator. The platform is fully compatible and does not require any third-party libraries.
UPD: Changed the picture, that would not cut the eyes.
UPD2: Added changes to the SDK.



SDK changes:
Bluetooth - on / off, connecting to a remote device using RFCOMM with support for receiving and sending data.
Account Manager - a centralized account manager for secure storage and access to tokens and passwords.
Others are a new theme system, a new wallpaper manager.

List of changes in Android 2.0:
Accounts - you can add multiple accounts to synchronize contacts.

Mail - Exchange Support

Bluetooth - bluetooth 2.1, new BT profiles - OPP, PBAP.
Media - updated graphical architecture - improved performance, hardware acceleration.
Calendar - endless scrolling in agenda mode, statuses for each person invited to an event, it is possible to invite friends to new events.
Browser - updated user interface, bookmarks with thumbnails of web pages, double-tap zoom, support for HTML5, support for client databases using SQL, support for tags in full-screen mode, Geolocation API device location information.
Camera - support for flash, digital zoom, white balance, macro, color effects.

Virtual keyboard is an advanced layout designed to increase typing speed and reduce typing errors. Multitouch support allows you to be sure that not a single click will be missed when typing with two hands. Improved the logic of the dictionary, learning the frequency of use of input words. Contact names are now automatically added to the dictionary.
Messages - search for saved SMS and MMS, the ability to automatically delete old messages upon reaching the specified time.
