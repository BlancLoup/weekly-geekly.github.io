Movable Type 4.23

Perhaps it is already a tradition that in early December a new version of Movable Type will be released. This time the changes are not so revolutionary, but their number is impressive - more than sixty. In addition, this update is security related, so all users are encouraged to upgrade.
Download Movable Type 4.23, Russian version in a zip-archive (or tar.gz-archive ).
The update fixes some errors in the work, and also affects some of the templates: those who use Pro-functionality need to update the user profile view template. By the way, Community Pack and Professional Pack are also updated, so if you use them, you need to upgrade.
A detailed list of changes is available on the MT.org website , and the changes in the Russian version remain the same, only the language file has been slightly updated.
