14 Free Web Collaboration Applications

"People are not islands in the open ocean." This statement can be successfully used especially if you are a designer or developer. Despite the existence of paid funds like Basecamp and Zimbra , you can also use freebies, which are probably not inferior to paid ones (and sometimes overtake them).

In this article, we will look at 14 free web-based collaboration applications. If you need a simple editor with the most basic capabilities or a full-featured project management application, you can find at least one application that meets your requirements.



1. Google Docs




Perhaps this is the most famous online document collaboration application. With it, you can open access to documents and spreadsheets to your colleagues or clients and work with them in real time. Since Google Docs is a Web application, its requirements are minimal and documents are stored on the server, which allows you to access them from anywhere in the Solar System (provided you have an Internet connection). If you have not yet figured out how to use this application, you are already prompted .

2. Stixy




Stixy is a flexible system for drawing and placing ads. With it, you can create an unlimited number of notes, change their settings and move around your virtual table. Among other things, there is one more useful application: commenting and joint discussion of Web pages or their parts.

3. Project2Manage




This application is a complete free project manager (like Basecamp). Anyone who wants to start a project can make an unlimited number of them. The user can set different levels of access and restrictions for each of the team members, describe their tasks, work results, etc.

4. Bubbl.us




This service is a free, web-based, shared brainstorming application. You can create beautiful mind maps with it, sharing them with other team members. After that, you can save the map, send it by mail, print it out, or paste it directly into the web page (you are given the HTML embed code). Among other things, you can simply give a link to the created map in read-only mode, for example, as I did . This service has taken pride of place among my bookmarks, as it allows you to quickly and conveniently work with mind maps.

5. Dabbleboard




Dabbleboard is a self-sufficient drawing application. You can draw interface sketches , diagrams, create diagrams , including many other different things. You can create toolkits (reusable drawing object sets) or use predefined ones.

6. Protonotes




Protonotes is a free widget for your HTML prototypes. To add it to your own page, you only need to play with JavaScript. Protonotes is very suitable for testing the usability of the interface, the criticism of design, testing of something. To be always up to date (for example, when colleagues add a new note), you can subscribe via RSS. In addition, all data can be exported, for example, to CSV and opened in applications such as Excel.

7. ProjectPier




Another online project management application, however, ProjectPier is also an OpenSource project written in PHP. Manage your tasks, team members, projects using a single interface, which in itself is very simple, easy. If you're still not sure whether to use this Web application, try to accept 10 reasons for this from the developers.

8. Twiddla




Developers are positioning their product as a free Web-site for meetings, meetings. Twiddla is an exceptional solution for this purpose. You can browse Web sites, at the same time painting them as you please, sharing files and arranging conversations with colleagues. However, there is also the possibility of audio communication (a special option for those who draw badly). This project is also suitable for those who like to criticize Web-sites, their design or usability.

9. Wetpaint




Wetpaint is a group of people collaboration application that combines the capabilities of wiki, blogs, forums and social networks. However, despite the abundance of opportunities, the program is very simple and even a person without any special technical skills can easily master it.

10. Thinkature




Free collaboration tool. You get a workspace in which you can draw, chat with colleagues, and collect content from the entire surrounding Web. The flexibility of Thinkature allows you to become the real owner of the program.

11. Spicebird




If you want to manage your project using a desktop application, you should pay attention to Spicebird - a complete tool designed to meet all the needs of a project manager. It has built-in chat, group calendar, address book and more. If you still have questions, you can watch the video demonstration of the application.

12. Mindquarry DO




This open source teamwork application is written in Java (J2EE). You can put it on your server and be sure that the data you send will not get to third parties. For additional installation information, be sure to refer to the list of requirements .

13. Vyew




Vyew is very similar to Twiddla: with it, you can hold conferences in which up to 20 people can participate at the same time. You can share your desktops, use the built-in screen capture tool. However, you can create and share diagrams with other conference participants using the DiagramVyew extension.

14. Writeboard




Written by the well-known 37 Signals group of enthusiasts, this application is an easy way to share information with colleagues. You start by creating your own workspace, invite colleagues and just start writing. Your colleagues will be able to see your changes and edits in real time. You can save and track changes, create new ones and roll back the document to old versions.
