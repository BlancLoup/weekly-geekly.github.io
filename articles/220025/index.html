Secure payment card do it yourself

To pay for goods or services via the Internet payment card is not required. It is enough to know only three groups of numbers (16 + 4 + 3):
- card number (16 digits);
- validity period: month and year (4 digits);
- secret CVV2 code (3 digits on the reverse side).

If the card falls into the hands of crooks, you can rewrite these 23 numbers in a matter of seconds. Not to mention the situation of physical loss from which no one is immune.

Therefore, I share my practice: away from sin CVV2 code from the cards Visa and Master to erase. Better scissors. Previously, of course, somewhere duplicating it. And for the convenience of paying bills on the Internet, write the key from the code with a felt-tip pen (marker) directly on the card.


Secure card (after manipulation).



Recover CVV2 code easily. We make a simple formula like "plus 4, 0, 4". We hammer it into our memory, and when the need arises, we quickly add formula numbers to the key numbers.
For example:
1) The key written by a felt-tip pen is 575.
2) Recovery formula: +4, 0, 4.
3) Calculate:
5 + 4 = 9;
7 + 0 = 7;
5 + 4 = 9.
Total: CVV2 code - 979.

So in just 5 seconds we get the coveted number, and no risk for the ‚Äúpocket‚Äù.
