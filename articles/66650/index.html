Navigation on iphone

Good day. Now I would like to present you an overview of the three navigators under the iPhone.
Sygic vs. Navigon vs. iGO



For the tests, I used the hacked versions of the program (I'm not ready to pay $ 99 + $ 119 + $ 99 for the review), so I had the following disadvantages:
0. Navigon - a program for Europe was taken, all maps were deleted, a map of Russia was downloaded
1. Sygic - a program for Australia was taken, and the maps were replaced from the same program, first published in Cydia (iDa)
2. iGo does not have a map of Russia - wrote a review of the map of Ukraine.

So, let's begin.

0. Sygic.


Before each launch, we see a warning page.

The program interface works very quickly, it is beautiful and easy. The transition to the horizontal mode is instantaneous.

Loved the keyboard.

There is a night mode

Navigator speaks Russian in a not very pleasant voice Natashi

Bottom line: I was not comfortable using the program.
Pros:
Beautiful, thoughtful interface
Good maps from Teleatlas
Rich settings
Built-in unit converter

Maybe I'm critical, but in the navigator, not only the maps are important to me, but also the convenience of working with them.
So, the cons:
A very long time thinking, if you turn the wrong way.
At any touch of the menu, the menu opens on the main screen of the map; There is no way to flip the cards forward and see the route away.
Natasha tells us about turning only 200 meters, which is naturally very small. (they say it's normal outside the city)
I did not like the voice, which voiced turns.
The program does not notify about the route change by voice.
Cost $ 100

The program is dopilena for Russia by national craftsmen, so it's too early to talk about stability. But in general, not counting not big brakes when choosing a city and street, as well as an ugly interface, the program is better than the previous analog.
