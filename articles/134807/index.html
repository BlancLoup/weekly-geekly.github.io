Meridian - Metro style music player for Vkontakte

The post is dedicated to everyone who listens to music on the social network VKontakte.

So, good day, ladies and gentlemen! Today we will look at one very interesting application for listening to music from the social network Vkontakte - Meridian.



What makes it interesting?

First, of course, by its interface.


It is made in the Metro style (do you remember Windows 8, Windows Phone 7, Zune, Metrotwit?). This makes it very convenient to use the application on both conventional and touch devices - modern smartphones and tablets. He checked himself personally - this is all good.



Secondly, its functionality.


You can listen to audio recordings from anywhere - from your favorites, from your own wall, you can listen to your favorite songs of friends, support for tracks from groups, radio, LastFM scrobbling, etc. etc. The application works closely with other services (LastFM, for example), which allows you to receive music recommendations on your favorite genre or current mood.



Another important point is accessibility.


There are 3 versions of the application.
  1. VKontakte web application . All the most advanced improvements appear first here. Written in HTML5, which allows you to use the application on all modern devices.
  2. .NET version for Windows. A good client who has a little less functionality than the web version, but at the same time has its own features - changing the color scheme, setting the song to VKontakte status, etc.
  3. Meridian for Windows Phone 7. The application is still in testing and is not yet available for download in the Marketplace. I managed to take part in testing - I can say that all the main functionality is there.

The remaining advantages


Applications are developing very quickly. Constantly adding various new features. You can read about it in the application blog . Their developers (.NET, WP7 - https://twitter.com/#!/Stealth2010 and Web - http://vkontakte.ru/hart_flamov ) are open to various improvements and fixes. Recently I asked to add one interesting radio to me - in a few minutes it was already added to the application.
