XnView 1.90.2

XnView is probably the third most recognized user after ACDSee and IrfanView to view graphic files. Supports the discovery of more than a hundred graphic, video and audio formats and converting between many of them.

Download XnView 1.90.2 (Windows version):
Full, with all plugins: XnView-win-full.exe (8.13Mb)
Standard version: XnView-win.exe (3.84Mb)
Minimum installation: XnView-win-small.exe (2.06Mb)

The latest versions have added support for reading ID3 tags from MP3 files, the print wizard interface has been reworked and bugs fixed. The program is free for non-commercial use. On the home page are available for download versions for Linux, FreeBSD and other operating systems.
