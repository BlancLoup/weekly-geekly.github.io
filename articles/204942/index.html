Ciklum Speakers' Corner "Centralized Log Management and Monitoring of Java Applications"

On December 11, the Kiev office of Ciklum invites everyone to join the open Speakers' Corne r it topic "Centralized management of logs and monitoring Java applications . "


During the presentation, you will learn the answers to many questions. What is wrong with the traditional approach to working with logs? How to make a search by logs from different machines of a distributed system convenient? How to monitor Java applications and performance monitoring nuances? How can metrics collected help you find problem areas and improve performance? How to build a homogeneous system for centralized log management and monitoring of Java applications using Logstash, ElasticSearch and Kibana?

Spreading a beam of light to work with logs and monitoring Java will be System / Middleware Architect (Ciklum, Yapital team) Eugene Heath , a cat lover, a developer with extensive experience in high-load systems and an enthusiast who believes in a bright future for JVM.
Participation in the event is free by prior registration goo.gl/JDzy9m .
When: December 11
Time: 19.00-20.00
Venue: Kiev, 12 Amosova St., BC Horizont, 2nd tower, 20th floor, Ciklum office
