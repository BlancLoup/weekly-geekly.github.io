Hypercube: makes web widgets available on desktops

Software company Mesa Dynamics has launched the trial alpha version of Amnesty Hypercube ( Mac version ; Windows version ) - its universal widget platform.

This platform allows users to import and collect widgets from the Web and use them on the desktop, rather than in a web browser. One click of the widget will be included in the Hypercube sidebar and displayed on the desktop.

A bridge between web and desktop widget apps, Hypbercube aims to provide a centralized tool for using your widgets. Like Google and Yahoo, Hypercube provides a centralized place to access widgets, but unlike search engine solutions, the Hypercube solution allows you to use widgets on various systems: now it is Windows XP and Vista (with .NET 2.0) and also Apple Mac OS X starting with version 10.3.9.
