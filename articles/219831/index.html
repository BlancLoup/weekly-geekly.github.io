Ubuntu 14.04 LTS (Trusty Tahr) is available for download!


Ubuntu 14.04 LTS (Trusty Tahr) is finally out!
Cheers, comrades!

So far, the news has not appeared on the official website officially presented, and the distribution can be downloaded here .

Of the main innovations:
  • Linux kernel 3.13
  • TRIM support for SSD is enabled by default.
  • Dash has become more responsive
  • New Lockscreen and New Screensaver for Unity
  • Updated Libreoffice 4.2.3
  • High resolution display support - HiDPI (High pixel density display)
  • Unity launcher can now be reduced to 8 pixels
  • Removed window borders
  • Improved support for Nvidia Optimus (but still not perfect)

Of the other changes announced:
  • OpenStack 2014.1 (Icehouse)
  • Puppet 3
  • Python 3.4
  • Xen 4.4
  • Ceph 0.79
  • Qemu 2.0.0
  • Open vSwitch 2.0.1
  • Libvirt 1.2.2
  • LXC 1.0
  • MAAS 1.5
  • Juju 1.18.1
  • strongSwan
  • Apache 2.4
  • PHP 5.5
  • Percona XtraDB Cluster
