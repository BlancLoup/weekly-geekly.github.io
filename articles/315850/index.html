Antidote - TOX client for iOS has finally become available



A significant security event took place a couple of weeks ago - the release of the Antidote client for Apple mobile devices took place.

How did this happen and what prospects - consider in the post.
What is interesting is the author of this application dvor (so, in fact, it is also a quality import substitution)




Tox is a protocol for secure messaging that does not compromise its advantages:

1) Encryption is always and always used.
2) End2End encryption (compared to telegrams)
3) For registration, you do not need to specify a phone number (and indeed there is no concept of registration)
4) Tox network is fully distributed and there is no central server
5) The source code of the kernel and client is open




I have been following the creation of this client for a long time - this is the first client for the TOX network under iOS. And the movement towards iOS is another platform on which Tox is available.

Functionality


Implemented the following functions:
- Chat
- Audio calls
- Video calls
- File transfer
What is not yet:
- Group chat
- Offline posts
