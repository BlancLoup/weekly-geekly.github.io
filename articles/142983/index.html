Beta Firefox 13 available


Less than a week, Firefox 12 was released and beta 13 versions are now available.

New items


The page that opens on a new tab has ceased to be empty - now you can see previews showing the most visited pages by the user, but you can also customize this page for yourself. It looks like this
The default home page now provides quick access to bookmarks, history, settings and more.



  • Smooth scrolling is now enabled by default.
  • Implemented experimental support for Map and Set objects from ECMAScript 6
  • Now FireFox can be updated with User Account Control
  • SPDY protocol is now enabled by default, which also improves the speed of the browser.
  • Added support for CSS turn unit
    • Added support for CSS property column-fill
    • Many improvements have been made for developer tools: Page Inspector, HTML panel, Style Inspector, Scratchpad and Style Editor
    • Implemented support for extended CSS3 background-position syntax
    • Pseudo-class : invalid can be applied to elements.


    The release of the full version promise on June 5th.
