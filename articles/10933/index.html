Opera Mini 4 Dimension

A couple of years ago you bought an expensive shiny phone, in which you could use only a wap-browser. Opera Mini broke these rules, gave freedom to the mobile Internet. Now the new version of Dimension destroys the existing foundations and opens a new dimension in mobile browsing.

Prehistory


Since the release of the 3rd version in the 4th, the code has been completely rewritten: it is now cleaner and faster, but it is also fast and small, but it supports even more web standards. But that's not all ...

Features:


1. Mobile web style computer
Yes, now the site looks like it would on a computer. This is facilitated by the zoom and mouse.
