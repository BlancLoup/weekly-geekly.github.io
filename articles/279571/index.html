IBM introduced the new z13s hybrid cloud mainframe.



IBM has announced a new mainframe, which will be useful primarily to companies of medium and small size. This mainframe provides an additional level of data security through encryption, without sacrificing speed or performance. The system is called IBM z13s. When using it, any company can support the operation of hybrid cloud assets with reliable protection of corporate information.

The development of IBM z13s provided two goals - high performance and data security. Regarding the latter, for any company, data protection now is of paramount importance. The fact is that the number of malicious attacks on networks of enterprises is constantly increasing, the attacks themselves become more sophisticated. At the same time, the number of operations in the network of companies today is also constantly increasing due to the use of cloud technologies and mobile devices. According to IBM , attackers are now trying to manipulate data, not steal it, which compromises the accuracy and reliability of enterprise information.

Since the encryption in the new system is at the forefront, there are quite a lot of user data protection tools. The processors are faster, and the memory size is larger than the previous models. As a result, the encryption speed is doubled when compared with average performance servers. For additional data protection, the IBM Multi-factor Authentication for z / OS (MFA) application is used, which is already available on z / OS. Here the developers added an additional level of information security. MFA is now part of the operating system, and is not supplied as an optional software product.
