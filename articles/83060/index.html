Circuit task



The following task is proposed, which seemed to me quite entertaining.

Unfortunately the thematic blog "Entertaining puzzles " is closed and that would leave comments and find out the solution
Habarusers will have to connect, but this one-time operation will not take much time.

It is required to connect a voltage source, three buttons with normally open and normally closed contacts, three lamps and a loudspeaker so that when you press the first button, the loud speaker roars and only the first lamp lights, when the second button is pushed, the second one lights the lamp, when you pressed the third button - the loud speaker roared and only the third lamp was lit. UPD with no buttons pressed, none of the lamps should be lit and the loudspeaker should not roar.



The rated voltage of all equipment is ~ 220 V. The loudspeaker will glow only if it is supplied with 220 V, the lamp will light only if it is fed with 220 V.

FAQ
What is a button with normally open and normally closed contacts?
This is a button in which the first pair of contacts is open, the second one is closed. In the depressed position, the first pair of contacts is closed, the second is open.
