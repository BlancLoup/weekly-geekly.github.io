Ruby + Qt4, a simple text editor, part 2

In the previous post ( f3ex.habrahabr.ru/blog/54673 ) I wrote how to write a simple text editor using ruby ‚Äã‚Äãand Qt4.
Continuing porting examples from Python + Qt4 to Ruby + Qt4

Part 2.

Now consider the following question: when a file is not selected or no changes are made to the text - the ‚Äúsave‚Äù button should be inactive.
The ‚Äúenabled‚Äù property in Qt Disigner is responsible for activity, or this property can be set from ruby-code using the setEnabled method
Set the b_save button's enabled property to false (uncheck the box) and regenerate the editor.rb form.





To find out if the text has changed, we will use the textChanged () signal for textEdit.

Let's change the start.rb code to the following:
