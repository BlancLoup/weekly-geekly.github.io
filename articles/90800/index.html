IronRuby 1.0 has been released!



Great news! Along with the global launch of Visual Studio 2010 and the .NET 4.0 platform, the final version of IronRuby is released - opensource-implementation of the Ruby language for the .NET platform.

IronRuby currently targets Ruby 1.8.6 support. According to the information from the site ironruby.info version IR 1.0 passes the RubySpec test for 86%. Among them: language tests for 98%, core for 90% and libraries for 75%.

Useful links:

- the official website of the project;
- IronRuby project page on codeplex;
- download IronRuby for .NET2 / .NET4;
- RubySpec test results.
