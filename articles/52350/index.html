Zend has released a beta version of its own server assembly

Zend has expanded its list of commercial products for php-developers.
Zend Server is a server assembly that includes a web server, php itself, several tools from Zend (Optimizer, Debugger, etc.), Zend Framework, monitoring tools and various DBMS.

The project is in beta, available for Windows and Linux. There is a free version of Community Edition .

At the moment, Zend Server includes:
  • Apache 2.2.8 (either uses pre-installed IIS)
  • PHP 5.2.8 and a pack of extensions for it
  • Zend Optimizer +
  • Zend Guard Loader
  • Zend debugger
  • Zend cache
  • Zend Java Bridge
  • Zend monitor
  • Zend Data & Page Cache
  • Zend Download Server
  • Oracle OCI Driver
  • Zend Framework 1.7.4
And as additional options, the installer downloads:
  • phpMyAdmin
  • IBM DB2 RTCL
  • MySQL Server
  • MS SQL Native Client
