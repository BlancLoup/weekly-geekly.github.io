Sync content blocking and fixes for * nix

A lot of time has passed since we changed something in Opera Link. In this snapshot 10.70, we present the ability to synchronize content blocking. This makes it easy to create and maintain a list of blocked items.

This snapshot also has several fixes for Linux / FreeBSD.

Known Issues

  • Existing URLs may not be synchronized: if you already have a list of blocked addresses before enabling synchronization, you may have to add the URL to the list of blocked via blocking the contents for synchronization before starting work after logging in to Opera Link.

Download

  • Windows MSI / Windows Classic
  • Mac (Universal) / Mac (Intel-only)
  • Linux / FreeBSD

List of changes


Cross Platform
  • Added URL filter synchronization (content blocking)
  • DSK-304462 (Personal panel shows its contents in one window)
Linux / FreeBSD
  • DSK-307161 (Shortcut Dialog values)
  • DSK-308602 (Popup menus background is plain color for some GTK Themes)
  • DSK-309032 (Slow UI with progress bar set to pop up at bottom and GTK Native used)
  • DSK-309213 (Widgets: Upgrade wizard, "more" leads to wrong location)
  • DSK-309799 (Geolocation memory fix)
