Updated popular multiplatform torrent client Transmission



... immediately before version 2.00. I doubt that someone has ever heard of him, but still I remind you:
Transmission is an ascetic (without sacrificing functionality) torrent client for Mac OS (Cocoa interface), Linux (Qt, GTK, console version), Solaris, and even BeOS (Zeta).

It features an intuitive interface, low resource consumption, support for modern mechanisms (DHT, PEX) and encryption protocols. There is a possibility of working as a daemon and a convenient web interface, which is used by some manufacturers of NAS and media players (Western Digital, Synology, D-Link, etc.).

This release carries a lot of interface changes, which makes it more user-friendly, and this is good news.
List of changes:

Platform-independent


  • Local Peer Discovery support for local network peer searches
  • Optimization of download requests depending on the width of the Internet channel
  • Improved algorithm for selecting peers to connect
  • Faster verification of local data (‚Äúhashing‚Äù)
  • Quick start
  • Extended support for blacklist formats
  • Using IEC units (KiB, MiB, GiB) instead of (KB, MB, GB)
  • Improved handling of 404 error

Mac version


  • Compact View is replaced by Minimal View, which takes up much less space.
  • Show Add Transfer window (torrent options) when adding magnet links
  • Resume All now ignores completed tasks.
  • Now you can copy the list of trackers directly into Create Window
  • Added Portuguese (European version) localization
  • Removed traditional Chinese (hieroglyphs) localization

GTK + version


  • New filters (tracker, public / private, etc.)
  • Compact View is replaced by Minimal View, which takes up much less space.
  • Showing a dialog with torrent options when adding magnet links
  • Now you can physically move several torrents simultaneously.
  • Settings window is now reduced on low resolution screens
  • Support for favicons on the trackers tab in the settings

Qt version


  • Showing a dialog with torrent options when adding magnet links
  • Ability to display all active trackers in the list

  • Added confirmation dialog when deleting torrents
  • The torrent settings and options dialog no longer escapes
  • Ability to set a save location for remote sessions

Daemon mode


  • Ability to select a script to run after the download is complete
  • Improved support options for each torrent when adding multiple
  • Pidfile support

  • Added option to delete .torrent files

Web interface
