The documentation on "F-BASIC" was almost completely absent, so I did not manage to master it in any way. "G-BASIC" turned out to be quite an interesting thing, because it contained a graphics editor based on patterns, as if cut from some version of "Mario" and a set of embedded sprites from the same "Mario" that could be moved and animated with using code.
