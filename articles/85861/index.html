Only 3 functions in the date_api.module file of the Date module change :

The idea is that the name of the month in the genitive case should be displayed only if it is displayed next to the day of the month. In all other cases - the nominative case.

Example:


February 23, 2010 , not February 23, 2010 .
But March 2010 , not March 2010

  • Open the Interface Translation page ( admin / build / translate / search ) and find the line:
    !month-genitive |January|February|March|April|May|June|July|August|September|October|November|December
  • Make it translated into the desired language. Note that for English, the names of the months are usually written with a capital, but we do not. Titles
