Php


  • RFC - Deprecation of fallback to root scope - Currently, if a function or constant is not found in the namespace, an attempt will be made to find an entity with that name in the global space. It is proposed to declare the functionality obsolete and throw a Notice level error: 

    Undefined function \My\NS\strlen(), assumed \strlen() Undefined constant \My\NS\PHP_VERSION, assumed \PHP_VERSION
