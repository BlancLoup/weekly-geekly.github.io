Some items in the boot menu hit me right away.

The first three points are understandable. But "anonymous web surfing" - what is it?

Sabayon is fully Gentoo compatible, but binary packages are included for user convenience. Want - rebuild, want - no. By default, the core is monolithic. And a separate item of the description made that Sabayon is not Ubuntu (with an explanation, they say, ‚Äúwe don‚Äôt want to be like any other distros, we just take the best of them‚Äù) :)

Oh, yes, here are a lot of toys! Including the very sauerbraten, which is written in the start menu.
So what is ‚Äúanonymous web browsing‚Äù? Checking this very anonymity with the help of some suitable detector (at least check.torproject.org ) proves that yes, indeed, web browsing is anonymous - the browser does not give information about the operating system and about itself. Sauerbraten is a cube-shooter on the cube engine, on a virtual machine with eight megabytes of video memory, it hardly makes sense to run it (see cubeengine.com , the chip of this engine is in a variable level geometry).
XsistenCe Mode is booting Sabayon from a LiveCD, but with a usb flash drive attached to the / home folder.
If you want to install Sabayon next to any other operating system or on a virtual machine - do not be greedy about the size of the hard disk. It requires a lot. My virtual hard disk is occupied by, for example, 10.2 GB after installation.

I chose the text setting. As an installer here, anaconda is probably used by many, so the installation is simple and friendly. During the installation, you can go to see what is being done in neighboring consoles, but they do not write anything there.

Inside


Sabayon with the GNOME desktop after installation

Immediately loaded graphic invitation, in my case - gdm. The first time I see the Linux distribution, in which so many things are by default. I wonder how they all managed to shove the 3.5 GB installation dvd-image? :)
Unfortunately, Sabayon could not find drivers for the virtual sound card offered to him, but at least he picked up the network himself.

Portage can be used as a package manager, and enthropy can be used - this is their own development, which turns Sabayon Linux from its own colored Gentoo build into a separate distribution. Enthropy is managed through the console using a text client called equo, or through the graphical client spitz. Through it, you can update the system, install and remove packages, as well as watch ads. Very comfortable.

Appearance is good.

Convenience work - great.
Overall rating is excellent.
I think I will even install Sabayon on a real car, I will play toys supplied by default :)

The developers are great!
