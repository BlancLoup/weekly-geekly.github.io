Dropbox updated interface

The developers of one of the most popular cloud storage presented a new menu design for their client.


Windows and Mac OS X versions have the same appearance.

Now the pop-up window combines the functionality of the former context menu and the ribbon of updates to your account. With this tape, you can follow the changes in your folders or in the folders of your colleagues, as well as view, share or restore files.

A few things that the developers mention in their forum :

- The new menu is currently available only in English;
- Works on Mac (OS X 10.6 Snow Leopard and above) and Windows (XP and above);
- Does not work on Linux;
- The exit button is moved to the settings window;
- There are no tooltips;
- Screen readers and other assistive devices are not supported.

Corrected mistakes:

- A bug with the tray icon in Windows 8;
- Bug with Camera Uploads in Windows 8;
- Other minor bugs.

New in version 1.5:

- New icons with support for Retina screens;
- Increased speed due to the use of Python 2.7;
- New installer for Mac;
- Support for the "Center for Notifications" in Mountain Lion;
- Download has become faster for Mac users with a large number of files;
- Fixed a delay when working with the context menu on Mac.
