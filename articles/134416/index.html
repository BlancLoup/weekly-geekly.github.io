Pocket router Sapido RB-1632



I can not share the delight of the newly purchased device.

The router, despite its size, has many functions. In addition to standard Ethernet -> Wi-Fi, it can act as a wifi client, has a USB host for connecting a modem and smartphones, can simultaneously create several Wi-Fi networks with different settings and so on. Powered by USB or from the adapter.

In wired WAN mode, everything is standard: Static IP, DHCP, PPPoE, PPTP and L2TP. It is possible to simultaneously create up to four independent Wi-Fi, for each network you can set different encryption parameters and security level. It is convenient if you need to give access to one of the guests to the Internet with WEP, while not allowing you to connect to WPA2 for LAN.



The USB port on top is for connecting a modem. Supports more than 250 models of GPRS / 3G / 4G USB modems , here is the complete list . I was always annoyed by a sticking sticking out of a laptop, hitting for all the stubs, but now I am delighted with the convenience - it is just bliss!



A router can act as a wifi proxy.

Conveniently in cases of paid access points, in airports, hotels, cafes, in the presence of filtering by MAC addresses and other restrictions. The router acts as a Wi-Fi client and distributes the Internet through its Wi-Fi network. Also, in the Wi-Fi client mode, it is convenient to connect wired devices to a wireless network. It can be a transparent WDS-bridge, connecting two networks via Wi-Fi.



Can distribute the Internet through a wired USB connection to a smartphone and simultaneously charge it.
Supports iPhone 3GS / 4, Google Android 1.5 / 2.1 / 2.2, Windows Mobile 6.0 / 6.1 / 6.5.

Instructions for setting up www.sapido.com.tw/EN/smartphone.htm .



The power input is mini-USB, in normal mode it consumes up to 0.5 A, with a modem up to 1 A. Accordingly, it can be powered from mini-USB if it provides the necessary current.

It can automatically switch to a modem connection if the cable becomes unstable (a specified number of timeouts during a specified period of time). It has all the standard security features - enable / disable access to Wi-Fi by MAC addresses, forwarding ports from the WAN to the LAN (Virtual Server), filters by MAC, IP, ports and even by keywords in the URL (the latter is for some reason incompatible with Virtual Server).
The web interface supports many languages, including Russian.

Full specifications on the manufacturer's website www.sapido.com.tw/EN/productrb1632s.htm .

On Ebay , apparently, the device sells directly to the manufacturer. Last year, the price was $ 55, now $ 34.
