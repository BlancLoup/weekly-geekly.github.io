ARM ChrUbuntu 12.04 Alpha - Porting Ubuntu to Samsung Chromebook

Proponents of Linux are currently trying to install Linux on a Chromebook, so that it works all at once. So far, to achieve the perfect work of LInux distributions on the specified netbook does not work, but there is success. A significant success can be called porting Ubuntu to a Chromebook with ARM.
