Useful to the designer: free new items to optimize design processes. 2nd Edition



Recently, I managed to find a lot of interesting mini-products and small tools / utilities that are designed to speed up your design process. In this issue you will find 14 links to various releases, the authors of which make a significant contribution to the development of the professional community of designers and developers.

#uikit


Elements UI kit


sketchapp.com/elements
Perhaps the first official UI kit from the creators of Sketch. Positioned as an iOS kit, but the elements are not native. Judging by the comments, the purpose of the manufacturer was to bring the correct set architecture. I think it will be useful to those who are focused on design systems and develop them in Sketch.

#sketch


Chart Plugin


github.com/pavelkuligin/chart
Chart plugin for Sketch. An indispensable tool if you often develop dashboards. It allows you to quickly fill content blocks with stylish graphs that are based on random data generated by the plugin. More details in the medium blog of the author.



Avataaars


www.avataaars.com
Pablo Stanley - a famous person in the western design community, released a avatar generator in the form of a plugin for Sketch. You can customize the colors, clothes, hair and get images of drawn characters in the same style. A web version of getavataaars.com is also available.



Plantapp


plantapp.io
Version Control for Sketch. The product can be used both as a plug-in to the editor and in the web version. Full data synchronization. Laconic description of the benefits can be found in the blog creators on Medium.



#webtools


Manta


github.com/hql287/Manta
"Manta" is a purely design tool for creating invoices. Stylish interface, free templates. If on duty often have to invoice - do it beautifully! And this tool, by the way, is very nicely decorated. It is visible that designers for designers did.



TypoScan


typoscan.com
A web-based tool that can scan any resource for used fonts, their dimensions and density. In addition, it scans and collects information about the palette, corner radius, border thickness, etc. This thing will make the process faster than regular inspect and will provide more structured and clear results.



Color Scheme Generator


www.donesmart.com/free-online-tools/online-color-scheme-generator-picker
Very simple color generator. For a minimum of actions on one screen, it produces 5 common color schemes from the desired color: Complementary, Triadic, Monochromatic, etc ... You get the desired result in a few clicks if there is no goal to strongly customize the colors obtained. It is bad that when you click on a color it is not copied to the buffer. Then it would be just perfect.



#stockimages


Silhouette-ac


en.silhouette-ac.com
The Japanese entered the stock images market, and immediately ready to give away 50+ thousand vectors with different silhouettes for free. API too. Available for download formats: SVG, EPS, JPEG and PNG. A lot of things on the New Year's theme, will only decorate their own.



#icons


Font Awesome 5


fontawesome.com
The 5th version of the Font Awesome icon set has been released. They need no introduction!
The archive contains 899 free icons available for download. Subjectively speaking, this set still lacks some consistency, because it seems that some of the icons are simply out of style. However, not so long ago, the developers conducted a top-notch company ($ 1M +) to raise funds for Kickstarter and 35,000+ backers cannot be mistaken. www.kickstarter.com/projects/232193852/font-awesome-5



#development


HTML5 Canvas js charts


canvasjs.com
Literally: "Beautiful HTML5 JS graphics." But besides beauty, this library: adaptive, with a simple API, supports all browsers, and indeed promises loading acceleration x10 times. The variety and choice of visualizations, by the way, is really impressive.



Hyperion iOS


github.com/willowtreeapps/Hyperion-iOS
If you want to control the correctness of compliance with the guidelines in the developed application - this plugin is for you. You can select any element in the application, check the compliance of fonts, indents and colors. Even it is possible to slow down the animation, if you doubt that the developers correctly encoded the transitive one element to another.

Principles.design


principles.design
Sooooo extensive collection of design principles in a simple and understandable presentation. If you need to devote time to philosophy or the formation of the spirit of the product - be sure to read! (eng)
