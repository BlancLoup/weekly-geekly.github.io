Top 10 Alternative Search Engines 2007


The blog on alternative search engines has published a list of 10 applicants for the title of the 3 best alternative search engines in 2007.

1. AfterVote - calls itself a social search engine. This system uses meta search for three search engines: Googel, Yahoo, Live, to present the most popular site in all systems. Sociality is achieved by the fact that any user can vote for a specific search option and suggest that this result is not true.
2. Answers.com is a search engine that allows you to find answers to a huge number of questions. The search result is not a collection of links, but articles from Wikipedia, Oxford University and other authoritative resources.
3. Dialogus - declares itself as a search engine of the new generation, which automatically finds the answers to your questions.
4. Exalead is a French search engine that, among other functions, can search on Wikipedia, blogs, forums and images.
5. GoshMe is still in beta . After testing, this search engine should have the ability to search in a huge number of search engines.
6. KartOO is a meta search engine, where search results are provided not in the form of a regular list, but in the form of a map or cloud, where each site is a URL and a site icon. The cloud also contains the keyword to which it belongs.
7. KoolTorch - groups search results. Each group is depicted as a circle of a certain color. It looks completely new, but quite difficult to get used to.
8. Omgili is a search engine designed to index forums. Using this search engine you can quickly find already published questions and get answers to them.
9. Onkosh is the Arabic web search portal. However, the search is carried out not only on Arab sites, but also on English and French with information about the Arab world.
10. Quintura is a search engine based on visual search technology that uses results from the yahoo search engine. The visual component is the location of the found keywords in the form of a cloud, which allows you to refine the request.

Quintura - of course, a good search engine, different original idea. And there would be nothing strange in the fact that a resource like AltSearchEngines.com constantly mentions Quintura, but for one thing: Quintura is the official sponsor of this resource. So it is not clear whether Quintura is really appreciated in the west, or only sponsorship is the culprit.
