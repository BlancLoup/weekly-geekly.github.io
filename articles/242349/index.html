NetApp Certification


NetworkAppliance (or NetApp for short) is one of the leaders in the field of data storage and information storage and management solutions. Certified engineers are valued by employers who have NetApp storage systems or implement these systems with their customers. The NetApp certification allows engineers to position themselves as qualified specialists in NetApp solutions and confirms that they have the skills necessary to deploy, support, and support NetApp technologies.
Certification consists of three steps:
  • NetApp Certified Storage Associate (NCSA)
  • NetApp Certified Data Administrator (NCDA)
  • NetApp Certified Implementation Engineer (NCIE)

If you look at the table with specializations:
NetApp Certified Storage Associate (NCSA) I stage
NetApp Certified Data Administrator (NCDA) Stage II
Data ONTAP 7-Mode Administrator Clustered Data ONTAP Administrator
NetApp Certified Implementation Engineer (NCIE) Stage III
Backup & Recovery SAN Implementation Engineer

You can see that on the steps DataAdministrator and ImplementationEngineer there are specializations.
Now we will consider certification in more detail.

NetApp Certified Storage Associate ( NCSA ) - for engineers with basic knowledge of NetApp and Data ONTAP storage systems who can administer and manage NetApp solutions. NCSA specialists recognize the features of both 7-Mode and Cluster Data ONTAP, and are able to perform basic administration of NetApp storage controllers running the Data ONTAP operating system on NFS and Windows (CIFS).
To obtain certification, you must pass a single exam NS0-145.
For preparation you can take the following courses:
  • Storage Fundamentals (WBT)
  • Introduction to NetApp Products (WBT)
  • Virtualization Technology Fundamentals (WBT)
  • Basic NetApp Configuration and Administration (ILT)

The first three courses are online trainings (WBT - Web-Based Training), and the last course is a two-day course (ILT - Instructor-Led Training).
For all WBT trainings, you need an account on the NetApp website - you can create it here .
The exam is taken in certified centers PearsonVUE, consists of 65 questions, which are given 90 minutes. To pass, you need to dial 80%.

NetApp Certified Data Administrator ( NCDA ) is the second step in NetApp certification, it has two specializations - Data ONTAP 7-Mode Administrator and Clustered Data ONTAP Administrator.
They differ in that NetApp has two different versions of the operating system for Data ONTAP storage systems - the ‚Äúclassic‚Äù 7-mode and the cluster Cluster-mode.
Certification is designed for storage engineers who have the skills to support, administer, and manage performance for NetApp storage controllers running Data ONTAP on NFS and Windows (CIFS) multi-protocol environments. Engineers are able to configure a controller with high availability. They also have detailed information about NetApp technologies, such as SnapMirror, SnapRestore, and SnapVault, which are used to manage and protect critical data.
To obtain the status of NCDA for Data ONTAP 7-mode, you must pass the NS0-155 exam.
For preparation you can take the following courses:
  • Data ONTAP 7-Mode Fundamentals (WTB)
  • Data ONTAP 8.1 7-Mode New Features (WTB)
  • Data ONTAP 8.1.1 7-Mode: New Features (WTB)
  • Architecture, Configuration, and Implementation of Flash Pool (WTB)
  • Data ONTAP 7-Mode Administration (ILT) - a five-day course.
  • Accelerated NCDA Boot Camp Data ONTAP 7-Mod (ILT) - a five-day course.

The exam can be taken at the VUE centers - it consists of 60 questions, which are given an hour and a half, you must dial 80% to pass.
Even for training there is an opportunity to pass the training exam NS0-154 .

To obtain the status of NCDA by Data ONTAP Cluster-mode, you must pass the NS0-156 exam.
For preparation you can take the following courses:
  • Clustered Data ONTAP Fundamentals (WBT)
  • Clustered Data ONTAP 8.1.1: New Features (WBT)
  • Clustered Data ONTAP 8.2: New Features (WBT)
  • Architecture, Configuration, and Implementation of Flash Pool (WBT)
  • Data ONTAP 8 Cabling (WTB)
  • Clustered Data ONTAP Administration (ILT) - a five-day course.

The exam can be taken at the VUE centers - it consists of 60 questions, which are given an hour and a half, you must dial 80% to pass.
You can also take a test exam .

NetApp Certified Implementation Engineer (NCIE) - the third step in certification, also consists of two specializations: NetApp Certified B & R Implementation Engineer and NetApp Certified SAN Implementation Engineer .
The first specialization is for backup and deployment engineers for NetApp recovery systems.
You must pass the NS0-510 exam in order to qualify.
Recommended trainings:
  • Operating and Troubleshooting the SnapProtect Solution (WBT)
  • NetApp Data Protection Software Administration (ILT) - a two-day training.
  • NetApp Snap Creator Framework Integration and Administration (WBT)
Requirements:
Have an NCDA certification (pass exams NS0-155 or NS0-156).
You can also take a test exam .
The exam is a test of 60 questions in the following sections, which are given a half hour time:
  • Section 1 - How NetApp Replication Technology Works
  • Section 2 - Planning and Implementing Data Protection Solutions
  • Section 3 - Setting Up Backup and Restore Software
  • Section 4 - Deploying Online Backup and Disaster Recovery Solutions
  • Section 5 - Best Practices for Data Protection, Backup and Restore

The second specialization is for SAN engineers who have an understanding of customer data storage requirements, Fiber Channel and ISCSI SAN implementation solutions, integration of various operating systems and applications, such as Microsoft Exchange, Microsoft SQL Server, DB2, and Oracle.
To obtain status, you must pass one of two exams: NS0-502: NetApp Certified Implementation Engineer ‚Äì SAN, Data ONTAP 7-Mode or NS0-504: NetApp Certified Implementation Engineer ‚Äì SAN, Clustered Data ONTAP.
Recommended trainings:
  • SAN Fundamentals on Data ONTAP (WBT)
  • SAN Design (WBT)
  • SAN Implementation - Switch Configuration (WBT)
  • NetApp Unified Connect Technical Overview and Implementation (WBT)
  • SAN Implementation (ILT) - a three-day training.
  • Architecting SAN on NetApp Storage (ILT) - a two-day training.

Requirements:
Have an NCDA certification (pass exams NS0-155 or NS0-156).
The NS0-502 exam consists of 68 questions, which are given for an hour and a half, and you must earn 80% to complete. Training exam is available here .
The NS0-504 exam consists of 65 questions, which are also given an hour and a half, a score for passing 80%.
