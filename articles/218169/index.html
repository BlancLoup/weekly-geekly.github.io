Windows XP discontinued today

Microsoft has been providing support for Windows XP for 12 years.



And from April 8, 2014, technical support for Windows XP and the release of automatic updates, which increase computer protection, cease. Also from this day on, the user will no longer be able to download Microsoft Security Essentials for Windows XP. If you continue to use Windows XP after the end of support, your computer may become vulnerable to viruses and other security threats.

Especially on this occasion, the developers released a touching video that tells what the users were able to accomplish with Windows XP. At the end of the video, Microsoft calls to upgrade from the old version of the operating system to a newer Windows 8.1.



In addition, the Microsoft website has opened a promotional section on the end of Windows XP support - happywinday.com with a funny video on the main page.



Thanks to Windows XP, but it's time to say goodbye!

The Windows XP operating system came out on October 25, 2001 and is still one of the most common operating systems in the world - it is in second place, second only to Windows 7. According to NetMarketShare data for March 2014, there are still more than a quarter of the world (27, 69%) of all computers running on Windows XP.
