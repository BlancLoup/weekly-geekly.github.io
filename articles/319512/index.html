Skype for Business plugin release and new build 3CX Client for Windows

In the new year with new victories! Today we present a unique plugin that allows you to call contacts from a Skype for Business client through a 3CX client. It's simple - right click on a contact in the Skype for Business interface and select the 3CX Call option. The call will go from the installed 3CX client through the 3CX PBX.

Our plugin allows us to solve a simple, but highly relevant problem - to make calls from a Skype for Business client via a standard SIP infrastructure. The 3CX server in this case is a simple and inexpensive ‚Äúintermediary‚Äù between Skype for Business and the available SIP gateways, phones and standard SIP trunks.



Install the 3CX Skype For Business Plugin


Prerequisites


  • 3CX V15 SP4, PRO or Enterprise Edition
  • 3CX Client for Windows V15 SP4 and higher
  • Skype For Business 2015 or 2016 (Windows)

Plug-in installation


  1. Install Skype For Business. The following version clients are supported:

  • Skype for Business 2016 (Office 365 or Office 2016)
  • Skype for Business 2015 (Office 365 or Office 2015)

  1. Install 3CX Client for Windows V15 SP4 and higher (build no lower than 60903).
  2. Download the 3CX Skype for Business Plugin and install it.
  3. Restart the Skype for Business client to activate the plugin.

Using the plugin


  1. Select the contact you want to call, right-click on it and select 3CX Call . The 3CX Client for Windows opens and makes the call.
  2. If the contact has several numbers, a list of numbers will appear in the 3CX client. You can choose the number to call.

New 3CX Client for Windows


After the release of updated clients for Android, iOS and Mac last year, we finally got to the flagship client 3CX Client for Windows.

The latest version supports the G729 voice codec for use in networks with narrow bandwidth or high latency and additional interface languages.

New client features


  • New integration with Skype for Business client
  • Interception of numbers from the Skype For Business client
  • 3CX Client does not show the number if it is not in Skype for Business
  • When calling via Skype for Business, the 3CX client runs in the background.
  • Improved synchronization of phone numbers from Google Contacts
  • Fixed auto-tuning DTMF transfer method
  • Dialog box when calling from Skype for Business
  • Definition of Skype for Business if the 3cx client has a version less than SP2
  • Added Portuguese and Czech interface languages
  • Full support for G729 codec
  • Support for TLS 1.1 and 1.2 protocols in the client
  • Improved auto-tuning process

Plug-in installation


  1. Log in to the 3CX control interface
  2. In the top menu, select Updates> 3CX Clients . You will see the client update.
  3. Select the client update for Windows and click Download Selected . 3CX downloads clients from the update server and notifies users.
  4. When the client starts or restarts, the user will receive a notification about the availability of the update, after which the update will be installed automatically.
