Creating a simple DI container using TDD

Introduction



Today I looked through a number of screencasts from Daniel Cazzulino , in which he talks about creating from scratch the simplest DI container, which could not fail to attract my attention. Below are examples of his screencasts.

Read more about IoC / DI containers here

Requirements



We need Visual Studio 2008, 15-20 minutes of free time and the desire to learn how to make your DI container.

Let's start. Funq1

Funq1.Tests



  • We try to run tests, but they do not pass, because We intentionally introduced an error in the RegisterTypeAndGetInstance () method description in line 17: Assert.IsNull (bar) to make sure that the test worked
  • In order for the test to pass, you must change this line to Assert.IsNotNull (bar);
  • Next, we are interested in, and what about the dependency in the constructor, how to register it, for this we declare the test method ResolveGetsDepedenciesInjected () and enter the following declarations:
