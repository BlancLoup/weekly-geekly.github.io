Oh, an! - You've got an App


It becomes clear that most software will be distributed through app stores. The owners of mobile devices based on iOS and Android were the first to enjoy the delights of these stores, and later Mac OS and Ubuntu users received the App Store and the Ubuntu Software Center.
Today I want to tell you about the app store from Intel - AppUp, which has been running for more than two years.
Intel AppUp is not widely known among my friends. I think, first of all, this is due to the fact that it is not installed along with the operating system, unlike MacOS and Ubuntu.

To work with the app store, users need to install the Intel AppUp center by downloading it from appup.com .
By registering and logging in, you can start shopping. For citizens of Russia, only free applications are available so far. Since the statistics show that most downloads fall on them, this should not be too upset.


In AppUp you can find games, office applications, antiviruses, various utilities and much more. See for yourself.

For this reason, I advise you to make friends with the Intel store with those of your acquaintances who can hardly be called sophisticated users. First, they will be grateful for an excellent catalog of toys, and secondly, they will less likely pester you with a request to install something new.

Downloading and purchasing applications from AppUp has several advantages: all applications are tested by Intel and are guaranteed not to contain malicious code. If the application you purchased for any reason did not meet expectations, you can return the money spent within 24 hours. Also important point is the automatic update of the acquired programs.

For developers, the Intel AppUp Developer Program is provided, and the SDK for Windows is distributed within it. It allows you to use C, C ++, Java and Microsoft .NET
The Intel store has implemented a mechanism for conducting the so-called in-app purchase. For example, in games you can release unlocking additional levels for money, or a paid upgrade of the application from the demo mode to full.
