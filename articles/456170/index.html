How to create an application for finance: 5 API to help the developer



Hello! I continue my research of the sphere of finance in terms of technology. In the past, the material we studied the existing trading terminals and their features, and today I propose to talk, in fact, about the development of financial applications.

For the new topic, I picked up five APIs of various financial services that allow you to both get a variety of data and directly trade on the stock exchange using written robots.

Yahoo finance


Perhaps one of the most famous financial services in the world. Using the free API, developers can use service data on stock quotes, price changes and movements in the market. Information is available both on a variety of stocks and on other instruments, such as currency pairs.

The API can be quite conveniently connected using the Rapid API service.



Smartcom


API for creating an application for trading on Russian exchanges. Developers can use the API as an interlayer between the exchange's trading system and their own ready-made trading system (created independently or purchased), create their own trading robot or even a whole trading terminal with a ‚Äúcustom‚Äù interface.

Access to the API is free, you just need to create an account in the system of the broker company that developed it. It is possible to debug the work of your software with the help of test access in virtual trading.



Screenshot: SMARTcom Documentation

Sync by Paybook


Service useful to developers of financial applications for foreign markets. It allows you to collect in one place data from banks, financial and utility companies, government agencies of different countries. This service is paid, access costs from $ 50. The list of services available for integration can be found here .



Currencylayer API


A useful service for obtaining information about exchange rates and the cost of 168 world currencies. Data is collected from a variety of sources, such as banking sites, commercial information services. The developers promise compatibility with applications of any type and programming languages ‚Äã‚Äãof any kind.

There is a free limited API version (250 requests per month), a more adequate one starts at $ 9.99 (10,000 requests per month). Such solutions as Lyft, Blizzard, Intel are developing their solutions based on the Currencylayer API.



Easy finance


Well-known in runet service for accounting for personal finances provides an API for third-party developers. As indicated on the project site, third-party applications for social networks, mobile applications and browser plug-ins are most popular with resource users.

Applications are tested by the service team, approved ones fall into a special marketplace and are distributed by subscription. According to the statistics of the EasyFinance administration, application developers can earn 15-50 rubles per user per month. This income consists of income from subscription to the tariffs of the service itself, integration payments of platform partners and advertising.
