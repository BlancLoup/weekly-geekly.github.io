Opera Mini 8 for Java and Blackberry phones



Today we are introducing Opera Mini 8 for Java and BlackBerry phones in a new design and with new features. What is new? Private and night modes, access to tabs from the keyboard and statistics saved.

Private mode




The mode is enabled in the settings and allows you not to save the history of navigation on the network.

Keyboard Tab Access


Now you can open a new tab by pressing the up button on the keyboard. But there are other shortcut keys:

  • Up - for a new tab
  • # 0 - to reload the page
  • # 2 - quick google search
  • # 3 - search on page
  • # 5 - access to bookmarks
  • # * - night mode
