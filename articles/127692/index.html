OpenStreetMap and Flickr license (Creative Commons BY-SA version 2.0) translated into Russian

Today, the free library Wikisource ( what is it? ) Published a Russian translation of the Creative Commons Attribution-ShareAlike 2.0 Generic license (CC-BY-SA 2.0 Generic). You are welcome:

ru.wikisource.org/wiki/Creative_Commons_Attribution-ShareAlike_2.0_Generic

The original license in English can be found here:

creativecommons.org/licenses/by-sa/2.0/legalcode

It is for this license (note this is version number two ; for Wikipedia, for example, number three is the current version of licenses, but plans to create 4 versions of licenses are announced) the OpenStreetMap database, OSM wiki and some of the photos on Flickr are distributed. Now you can better understand your rights and terms of distribution of content. And in the future, when an adapted Russian version of the 4th version of Creative Commons licenses appears, anyone can create a derivative work based on the content under CC-BY-SA 2.0 that can be released under this Russian version.

Translations of modern CC version 3 licenses can be found here .

If you find typos in the translation text, edit boldly.
