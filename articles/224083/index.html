All controllers support SAS-SATA-SSD drives. To build RAID 50 and 60, use a server with 8 SFF disks.


Lenovo ThinkServer RD540 server has a crypto processor that stores cryptographic keys to protect information, as well as a generalized name for the implementation of the specified specification - Trusted Platform Module 1.2 (TCG 1.2-compliant). Also in the picture you can see additional USB ports, and the Avocent remote control module.

As we said above, the server has 3 RJ-45 ports, of which one (Intel 82574L) is given to remote control of the iKVM server, which is possible thanks to ASPEED's fourth generation AST2300 chip. The other two network interfaces based on the Intel i350 are free to transfer data over the network and switch to the infrastructure.


The server is powered by a pair of 800watt power supply units each, with hot-swappable support when the server is running. They have a standard of 80 PLUS Certified Power Supplies.

For ease of management and server upgrades, bundled software products are provided that greatly simplify the lives of service personnel:

  2. ThinkServer EasyStartup - simplifies the first launch and setup
  3. ThinkServer EasyManage - helps manage the server
  4. ThinkServer EasyUpdate - helps to update firmware
