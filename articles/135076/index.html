Node.JS support in Windows Azure

A lot of work has already been done on porting Node to the asynchronous Windows I / O API (IOCP) and recently Node.JS support was announced in Microsoft Windows Azure cloud hosting. Access to Azure cloud services is available through an SDK published on GitHub , which can also be installed using npm:
