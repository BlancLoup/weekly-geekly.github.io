Released Yandex.Navigator and SpeechKit for Windows Phone

Today again, the good news for Windows Phone: Yandex has released its Navigator for Windows Phone :



The application is able to do everything that is necessary for the driver behind the wheel: it builds a quick route taking into account traffic jams, tells in a voice where to go, and warns about traffic events. Navigator will be useful in trips to Russia, Ukraine, Belarus and Turkey. Detailed vector maps are available for all territories.


But this is just interesting news for users of smartphones with WP, but what is really interesting for developers is the availability of speech recognition technology from Yandex , which is used in the application for entering addresses. About the technology itself, colleagues from Yandex have already written on Habr√© about a month ago.



Let me remind you that SpeechKit is a special library (API) that provides mobile application developers with access to Yandex servers for processing and analyzing speech fragments.

  • Documentation for Windows Phone (still in English): api.yandex.ru/speechkit/generated/winphone/html/index.html
  • NuGet package: www.nuget.org/packages/Yandex.SpeechKit
  • Examples: download.cdn.yandex.net/speechkit/YandexSpeechKitSamples-wp.zip
