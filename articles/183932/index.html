Windows Azure news: mobile development, multi-factor authentication, MPEG-DASH, JWT, tests and video



Dear developers and IT specialists who are interested in the topic of cloud technologies, let me introduce you to the next batch of innovations and announcements of the Windows Azure platform that have seen the light in the last two weeks.

The list of news, updates and announcements of the Windows Azure platform in this review:

  • Announcement of a major Windows Azure update for mobile application developers;
  • Announcement of the new multifactor authentication service Active Authentication;
  • Test results for 5 cloud providers offering an IaaS platform;
  • MPEG-DASH standard support in Windows Azure Media Services;
  • Global availability of JSON Web Token (JWT) Handler;
  • Records of DevCon 2013 and TechEd North America 2013 conferences.

Below you will find a more detailed description of this news and announcements.

Major update for mobile app backend developers


A few days ago, Scott Guthrie in his blog presented the next batch of innovations and updates to the Windows Azure platform. This time, innovations mainly concern mobile services and will be of interest to developers of mobile applications of any platforms that use the cloud as a backend.

Among the innovations were presented the following features:

  • Support for user APIs in server logic;
  • Integration of server code with Git version control systems;
  • Support for NPM modules Node.js in server logic;
  • .NET API library for Windows Store, Windows Phone 7.x and .NET 4.5 clients (ASP.NET, WPF);
  • Option: free SQL Database database size of 20MB;
  • Support for sending push notifications on Android.

New service multifactor authentication Active Authentication


The new Active Authentication service, introduced for the Windows Azure cloud platform, is designed to protect the user accounts and cloud services offered by Microsoft, third-party solutions, or proprietary applications and services that use the Windows Azure Active Directory cloud service as an authentication system.

Among the Microsoft applications that can benefit right now are: Office 365, Dynamics CRM Online, Windows Azure portal, and other services.

The Active Authentication service offers the ability to protect services and users with additional factors: phone calls, SMS, mobile applications, one-time secret codes.

Test results of five cloud providers offering IaaS platform


CloudSpectrator has tested five leading providers of infrastructure services in the public cloud. Among them in testing were presented platforms: Amazon EC2, Rackspace OpenStack Cloud, HP Cloud, SoftLayer CloudLayer Compute and Windows Azure.

Testing was carried out according to two parameters: the performance of the cloud service with the monitoring of the error during the testing period and the price-performance parameter. Another goal of the test was the desire to provide the market with an understanding of what is abbreviated as ECU, HPCCU or vCPU, which vendors measure computing power.

For testing, we used a set of tools for testing the performance of servers based on Unix / Linux - Unixbench .

MPEG-DASH standard support in Windows Azure Media Services


MPEG-DASH is an open standard for adaptive dynamic streaming of video on the Internet, which replaces Apple, Adobe and Microsoft formats that were previously used, including using plug-ins.

MPEG-DASH is a standard adopted by an ISO organization ( ISO / IEC 23009-1: 2012 ) in April 2013. Immediately after this event, Microsoft announced its intention to include full support for the standard on the Windows Azure platform in Media Services . Today, DASH is a potential adaptive broadcast standard for adoption by the W3C.

Today, Windows Azure developers can work with MPEG-DASH format for broadcasting content on demand (Video on Demand). With access to the public availability of live capabilities of Media Services, developers will be able to perform live broadcasts in this format.

And finally, MPEG-DASH is available on the Windows Azure Media Services platform through the dynamic packaging function, when the broadcast format is set on demand and is dynamically generated without prior conversion.

Global availability of JSON Web Token (JWT) Handler


JSON Web Token (JWT) is an open standard whose specification is handled by The Internet Engineering Task Force. The JWT standard describes a compact token format intended for use in size-limited environments, such as HTTP authorization or request parameters.

With a recent update, global availability of JWT standard support in Windows Azure was announced. Now developers can use this format as a token format when working with the Windows Azure Active Directory and the Access Control Service.

This will allow developers of different platforms to more effectively use the authentication capabilities of Windows Azure. Update support for JWT has introduced a number of other innovations, which you can find out on the following link in the official blog . One of these innovations is closer integration and improved support for the Windows Identity Foundation.
