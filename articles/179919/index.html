Welcome to the Nokia Asha Developer Bootcamp

Hello, dear habrazhiteli!
We invite you to take part in the Nokia Asha Developer Bootcamp training, which will be held in St. Petersburg on May 23-24, 2013 at the Radisson Sonya Hotel.

At our event, you can get an extended insight into the Nokia Asha user interface and work with the new SDK. We will talk about new features of the platform, as well as introduce the possibilities of porting existing applications.




The training is intended for developers familiar with Java ME and experienced in developing Java applications for Series 40 phones. The full program of the event is here .

In order to participate in the Nokia Asha Developer Bootcamp, you must send a request to the event manager by email or register yourself:
Event Manager: Anton Pogrebnoy
Email: anton.pogrebnoy@nokia.com

Hurry up, space is limited. We will be glad to see you at our training!
