Release Firefox 24 and Firefox 24 ESR

Firefox for desktops has become available for download. What's new in this version:


  • The function of closing all tabs to the right of the active tab is implemented . The functionality will be useful to those who first opens many different tabs, and then starts to view them;
  • For OS X 10.7 and above, animation effects are used when navigating, a new scrollbar style has been added (the introduction of this feature was planned in Firefox 23, but was turned off before the release);
  • In the support interface for working with social networks, the ability to open individual windows with chat rooms has been added by dragging items from the general chat window beyond the borders of the current window;
  • The new tab (‚ÄúNew Tab Page‚Äù) will now be preloaded by default , which will speed up its opening;
  • Improved accessibility support when using fixed tabs;

  • A fixed-value audio resampler from the webrtc.org capture code has been replaced with a Speex resempler. Fixed speed in pseudo-44000Hz;
  • The dataset property (HTML5 data- *) should now work faster .


Firefox 24 ESR - a release with a long support period also becomes available for download. Updates for this version of Firefox will be released every six weeks during the year and will only include the elimination of serious security and stability issues.
The Firefox 17 ESR version will be supported until December 10, 2013 - after this date, the browser will be automatically updated for all users to the Firefox 24 ESR version.
