Open Terminal Client OTC-110 - redirection of the printer to the RDP session

The fourth post about the open terminal client OTK-110, it is also an ultra-thin client.

Now the redirection of the printer connected to the OTC-110 to the terminal session of the RDP is working.
As usual, only packages available for debian for armhf from standard debian repositories were used.


After testing, there was one option:
  • OTC-110 launched CUPS
  • FreeRDP redirects printers available in CUPS

The Windows printer driver must be installed on the Windows server.

Theoretically, everything that CUPS can print on and has drivers for Windows can be redirected to a terminal session.
Including printers that require downloading firmware after powering on (if CUPS knows and is able to, more precisely, not CUPS itself, but here it doesn‚Äôt matter).

Customization

In terms of settings, they stopped at the option:
  • CUPS is configured via a browser running on OTC-110, this option does not require a separate computer with a browser.
  • In the shell of the OTK-110 parameter settings, a line of the form <Printer name in CUPS> is written: <Printer driver name on the RDP server>.

Everything is set up, everything should work and work.

Additionally made a PDF printer - sometimes it is convenient. The received documents are stored in OTC-110 RAM, i.e. when you turn off will be lost.

Firmware upgrade

In connection with the large intervention in the original Cubie distribution, it was not possible to get by just updating the settings utility. Requires complete flashing.

For a complete flashing, you need to download the firmware image, write it to the microSD card, insert it into the OTC-110 and boot. Enter settings, select About / Update firmware on NAND, the firmware from the SD card will be copied to NAND. After the end of the firmware, OTK-110 is turned off. You need to remove the SD card.

Everything can be included.

Important!
If the network uses more than one open terminal client, you must assign different MAC addresses of network adapters.
