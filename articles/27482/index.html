Firefox 3: The release took place !!!

The long-awaited final release of the Firefox 3 browser took place. Basically, the development years were spent on polishing the Gecko engine, adding a total of 14,000 updates to it. The main task of the developers was to improve the performance and sustainability of the program. Judging by the latest release candidates, the developers have completely coped with the task.

Increased stability and performance are not the only advantages of the new product. In Firefox 3, a lot of attention is paid to security, even better protecting the user from malicious software, uncontrollable scammers, evil sites and other misfortunes.

Of course, Firefox 3 has made many improvements in terms of usability. I will focus your attention on, I think, mostly. A new address bar with intelligent sorting of bookmarks and history has been introduced, a new download manager has been developed to support resuming, password management has been improved, installation of add-ons has been simplified, pages have been improved, tabs have been improved, several improvements have been made to the bookmarks system.

As for performance and stability, here, first of all, it is worth noting an improved JavaScript engine capable of processing JS applications 2 times faster than Firefox 2. Now complex AJAX sites work a whole order faster. Secondly, the work with memory has been improved, eliminating hundreds of minor problems with its leakage and an improved caching system. Improved reliability of user data storage (bookmarks, history, cookies and settings), even in case of failure, the data will not be affected.
