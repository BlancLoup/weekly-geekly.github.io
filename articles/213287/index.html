Total Commander 8.50 Final

Yesterday, after an unusually long period of open testing, Total Commander 8.50 was released . Improvements and fixes, as usual, many hundreds, here are the most significant of them (I will state in advance that everything is set up or turned off if there is no demand for it):

  • Background search (cm_SearchStandalone command or Alt + Shift + F7). I can‚Äôt even believe it - all twenty years of the program‚Äôs existence probably asked for it! Moreover, the search tool in the TC can be called separately from the TC itself by running the program with the / S = F key.
  • Editing directly on FTP (works for FS-plug-ins). Rather, this is what it looks like for the user; in fact, TC will automatically synchronize a local copy of the modified file each time it is updated.
  • TC is optimized to work on touch devices. Touching the file icon works in the same way as right-clicking (it is also considered to be a click on the LMB).
  • Added support for High DPI screens (new high resolution icons, optimized control panel sizes at high DPI), improved performance on multi-monitor configurations.
  • Background dialogues have become more informative (for example, when copying information about the number and size of the copied / copied is displayed), and it became possible to restore some operations from the background.
  • File operations have become smarter. Copying will not stop on unreadable files, the TC will first copy what it can, and only then ask "what to do about it?". In this case, if the files are not read immediately, the dialog will appear - it is assumed that the user has not yet managed to move away from the computer. Background file operations will slow down when working with the same files in the main interface. Copying got the function of checking the CRC (which, turned off in one of the beta versions because of one report about incorrect work, and then turned back, after numerous requests and rechecks that did not confirm the problem).
  • Support for chains of internal commands: on one button you can hang several actions in sequence at once. Finally realized what for ten years ago had to write separate utilities.
  • When searching for duplicates, you can use fields from content plugins. For example, you can find all the images with the same aspect ratio (without specifying the proportions themselves).
  • Native unpacking support for RAR5 and 7Z.
  • In Lister, the selected text can be searched on Google (search engines are customized).


And many other, minor and not so much improvements, you can find a complete list of which in the TC history file .
