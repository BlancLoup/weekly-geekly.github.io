Fujitsu Global Cloud Platform

Fujitsu Global Cloud Platform is an IaaS service that allows users to quickly deploy, configure and scale virtual data centers. The offer includes virtual machines located in one or several virtual network segments, a firewall that controls the interaction between the created machines, and additional services that ensure the operability of the system as a whole.

Network structure


There are three options for networking in the system: one-, two- and three-level. Each level is a separate virtual network.



All levels are interconnected through a firewall, which also acts as a gateway. The systems themselves can also be connected to a single intranet within a single subscription. And, of course, you can create one or more physical IP addresses for the system and thereby provide access to the Internet.

Firewall


A firewall is a specially configured virtual machine that acts as a firewall, gateway, and is responsible for network address translation (NAT). The flexible configuration system allows users to filter incoming and outgoing connections by the following parameters: connection direction, sender and recipient addresses, used protocols and ports. The direction of the connection refers to two networks, selected as the initial and final. The role of such networks can be the actual network segments of the user's system, as well as the Internet and intranet.

Additional services


Optionally, the user is offered load balancing services, backup of system components, setting up a VPN connection between systems and the user's local network. The full list is available on the cloud platform portal: globalcloud.de.fujitsu.com

Terms of Service

The Fujitsu cloud platform is designed primarily for the corporate user segment, so each registration on the portal is separately supported by a service agreement. This allows you to guarantee service levels and take into account specific user requirements.

Only actually used resources are paid for, which is fully consistent with the concept of cloud services. Depending on the types of resources, the pricing process may vary slightly. For example, virtual machines are paid for hours of operation, that is, the time when the server is turned off is not taken into account, while disk space is paid based on the reserved volumes, regardless of whether the disk is stuck to any server or not.

Separately, it is worth noting that for each new contract there is a trial period of 1 month. During this time, the organization can become more familiar with the capabilities of the platform and test the necessary functionality. Consultation services are also provided to customers for launching cloud infrastructure and migrating applications.

Security

Multi-level network security is provided by a complex of antivirus and antispyware software. The software used is constantly updated, which allows you to timely identify all known threats.
