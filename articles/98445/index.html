Keyboard simulators: kbTrainer unleashed

Hi Habr!

The kbTrainer keyboard simulator widely known in narrow circles has been updated.

What's new:
1. All code is rewritten, or almost all.
2. Implemented multiline mode .
3. New skins, similar to the old, but new, there is even pink.
4. HTML 5 - the groundwork for the future.
5. Everything else remains as it was, despite the changes.

Where to get:
kbTrainer Firefox addon 0.1.1 code.google.com
kbTrainer Firefox addon 0.1.1 Mozilla Addons
UPD 3: as it was said, the HTA version is buggy, so version 2.1 (instead of 2.0 which was yesterday) is now compatible with Windows 7, Server 2008.
kbTrainer 2.1 HTML Application

Under the cut three large pictures and a little description.


The release was released in two versions:
* Mozilla FireFox AddOn (0.1.1).
* HTML Application (HTA) script (2.0)

In this case, the code is 95% identical.
Despite the addition of jQuery to the code, the optimization made it possible to remain within the framework of the previous sizes: less than 200 kilobytes.

Minuses:
1. HTA version is a bit buggy. Sometimes a jQuery error pops up. What is it - I do not know.
2. There is no sound yet, but I think it will be, if you need to tell when it will be - write in the posts, there will be a post about a new release.
3. The history of the sessions is also not being conducted. But I think that will also be.
4. Work on support for FireFox 4 is underway, but for now - alas.
5. Wishes are not implemented in previous versions, too.
