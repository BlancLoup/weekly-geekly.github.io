A selection of useful materials on the development of Xamarin

We are sharing useful Xamarin development content: The Xamarin Show video on Channel 9, the basic Microsoft Virtual Academy course for beginners, a list of educational materials and the first Moscow Xamarin Meetup in Russia.



The xamarin show


In September 2016, an interesting show on Channel 9 appeared on Xamarin cross-platform development for iOS, Android, macOS and Windows. James Montemagno, a leading Xamarin specialist at Microsoft, talks about the creation of mobile applications, the SDK, the latest Xamarin news and interesting projects.

The Xamarin Show 1: Shared code for iOS, Android and Window


The episode tells about several strategies for using common code for iOS, Android and Windows.



The Xamarin Show 2: Continuous Integration


There is a myth that setting up continuous integration is an extremely difficult and time-consuming task. In order to dispel it, James invited Simina Pasat, manager of HockeyApp , to the studio . She also spoke in more detail about the approach itself and how in a few minutes using Visual Studio Team Services you can automatically launch the assembly of Xamarin applications and deliver them to devices using HockeyApp.



The Xamarin Show 3: Secrets to improve application performance with Xamarin.Forms


James talks about the basic techniques for creating an application with high performance, as well as the latest features built into Xamarin.Forms.



The Xamarin Show 4: Continuous Delivery


Josh Weber, the manager of HockeyApp, was invited to the studio, who talked about the next DevOps step after continuous integration - continuous delivery. It will also discuss how to improve the release management process in Visual Studio Team Services.



Snack Pack 1: Android Emulators


You will learn about the launch of the android application on emulators, Visual Studio and Xamarin Studio.



Xamarin for newbies


Popular course Microsoft Virtual Academy, in which you will learn how to develop applications on Xamarin. It consists of the following sections: introduction, installation, study of the project, work with emulators, debugging and deploy.



Useful materials


Xamarin University : the official training program, which includes online lessons and advice from leading experts on Xamarin, self-study materials and more than 70 courses (from guides for beginners to enterprise development). Introductory courses can be taken for free.

The official Xamarin repository on GitHub : the latest code samples, useful components and libraries, books, manuals and more from the Xamarin development team.

Xamarin.Forms sample repository : more than 50 sample projects using Xamarin.Forms on GitHub.

The repository of available components for Xamarin : all that is necessary for effective cross-platform development on Xamarin.

Xamarin Developer : official Xamarin documentation, as well as useful materials for developers, including API and Xamarin.Forms. There you can find announcements of support for new versions of iOS and Android.

Xamarin Developer: Guides : guides to quick start, features, Xamarin Studio, game development and application testing.

Introductory Guides : Beginner Content from Xamarin.
