Google‚Äôs new features for developers: game APIs, translation and testing

Today at Google's I / O conference in San Francisco, Google announced a number of new features for developers.

Firstly, this is a new API family created specifically for game developers . The platform will support cloud storage (including synchronization between various devices), achievements and leaderboards via Google+. In addition, another API implements step-by-step or real-time multiplayer. Google will give developers not only the ability to support matches by connecting devices, but also to invite other players to participate in battles through, again, Google+.

At the same time, Google Play gaming services will be supported not only in Android applications, but also for iOS and the web, which makes them truly cross-platform.

New APIs will be available for all Android devices from version 2.2 (Froyo) and you can already subscribe to them through the developer console.



Secondly, it is the translation of applications into other languages . Directly from the Developer Console, program creators can send a request to translate their applications. This should make it easier to translate applications from one language to another, increase the audience of applications and, ultimately, increase sales. Developers can place an order for translation in just a few clicks and send them to the translation of the line they need, having received them back then.

Translation is naturally carried out for a fee, since it will deal with third-party companies.

How, who and for what money will transfer all this, however, is not entirely clear. In the Russian version of Play, you can only subscribe to the beta testing program.



Finally, these are embedded beta and alpha testing services . The idea, in general, is not new - to test the functions of the application before its official release on a small group of users (this, for example, is the service TestFlight, which literally this week opened its service for Android).
