Opinion of one of the creators of CSS


Håkon Lie, who used to create CSS with Bert Bos (Bert Bos), now works as a CTO in Opera Software.

In a recent article for CNET, Håkon touched on the situation with two formats for office documents: OpenDocument format (ODF) and Office Open XML (developed in the depths of Microsoft). Consistently analyzing the situation, Håkon notes that he does not like both specifications: neither 700 pages of ODF, nor 6000 pages of OOXML. There is another way of development:

It is possible to create a new format on top of widely implemented HTML and CSS. Additional semantics (for example, formulas) can be placed inside attributes as it is done in microformats . CSS 3 allows you to implement new features (for example, callouts , footers ).


and further:

In order to show that this is possible, I and Bert Bosch published a book that was written in HTML and CSS . A significant feature of this approach is the fact that documents can be viewed in most browsers.
