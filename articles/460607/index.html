App Store with hacking tools from Offensive Security on Android



July 16, the company Offensive Security announced the creation of the Kali Net Hunter App Store with free applications related to information security.

Kali NetHunter is a penetration testing platform for Android. Recently, it has undergone many changes and today supports more than 50 Android devices (from KitKat v4.4 to Pie v9.0).

The NetHunter repository, originally designed for convenient package management only on NetHunter devices, has been made available to other devices. Of the benefits can be identified:

  • ease of use and convenience in the Google Play Store;
  • freedom, anonymity and privacy of downloadable applications.



Kali NetHunter App Store is a one-stop shop for information security applications and an alternative to Google Play for any Android device (with / without root rights, for NetHunter and other Android devices).

Thanks to the work of the F-Droid community, in particular Peter Servaylo , whose work made an invaluable contribution, the NetHunter shop now works on a modified version of the F-Droid . While installing the original F-Droid with the telemetry disabled, consent is requested before sending crash reports, here all the code intended to implement this functionality has been removed. This is done so that confidentiality cannot be broken by accident. It was also allowed to use its own applications in the store.
