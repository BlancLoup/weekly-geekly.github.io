Samsung M100S - communicator on the operating system Android 2.1



Yesterday, Samsung introduced the first communicator on the operating system Android 2.1 - Samsung SHW-M100S.





Most likely, this communicator will be distributed only in South Korea. M100s sales will start in March of this year, starting with SK Telecom mobile operator.



Samsung M100S is equipped with:
  • 3.7-inch touchscreen AMOLED WVGA display
  • 800MHz processor
  • WiFi
  • GPS
  • T-DMB TV tuner
  • 3.5mm headphone jack
  • 5 megapixel camera with support for recording video quality 720p


The communicator has software for working in social networks and the latest version of the user interface TouchWiz UI. Also Samsung M100S supports DivX playback.
