Samsung introduced 256 GB high-speed memory chips for tablets and smartphones



Soon, the memory on our mobile devices can become very much. Of course, there is always little free space, but mobile chip manufacturers are working on it. In particular, Samsung has released new capacious high-speed memory chips for tablets and smartphones. According to the company, they will be used in next-generation devices.

The new memory is based on the Universal Flash Storage 2.0 standard, and according to Samsung representatives, this suggests a twofold increase in data transfer speed compared to the SATA SSD. The read speed here is 805 MB / s, the write - up to 260 MB / s.

So far, these chips will be installed only in premium segment phones and tablets. And only now the data transfer rate for the memory of mobile devices has reached the data transfer rate via USB 3.0, so such a communication channel will now be fully occupied. This means that a 5 GB video file can be transferred in about 12 seconds. Not bad, right?



Memory cards, by the way, are also becoming faster. For example, SanDisk has released a microSD card with a data transfer rate of up to 275 MB / s. It is three times faster than the fastest microSD UHS-I card from the same company.
