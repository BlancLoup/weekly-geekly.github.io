Bad style for new comment

There is a small "defect" in the style of comments of the new Habr ...


As can be seen from the screenshot, the right edge of textarea is beyond the scope of the comment (browser: Firefox).
This "defect" is corrected by adjusting the comment_item_new class to this type:
.comment_item_new {
border:1px solid #DDDDDD;
margin-left:-10px;
margin-top:5px;
padding:10px;
position:relative;
}

However, it is also desirable to do something with width: 100% for textarea, but this is not critical.
