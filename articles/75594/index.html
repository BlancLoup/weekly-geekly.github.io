Opera Mobile 10 for Windows Mobile



November 18, 2009, Oslo, Norway
Opera Software has released the beta version of Opera Mobile 10 mobile browser for the Windows Mobile platform. Opera Mobile 10 has the capabilities of a desktop browser and allows you to search for information, use web services, communicate on social networks literally on the go.

The Opera Mobile 10 beta interface for Windows Mobile is similar to the Opera 10 desktop browser and Opera Mini 5 beta mobile interfaces. Opera Software seeks to maximize the unified look and options of all its products for all platforms. Opera Mobile 10 beta for Windows Mobile includes Express Quick Access to favorite websites, bookmarks, Password Manager and supports Opera Turbo data compression technology.

Smartphone users need a powerful mobile browser that can handle tasks as well as a desktop one, and Opera is ready to provide it. ‚Äù

10 major innovations Opera Mobile 10 for Windows Mobile
1. New interface. Opera offers a simple and intuitive interface that does not overload the screen of a mobile phone.

2. Express panel. Express panel provides quick access to your favorite web resources. Install and change sites in the Express panel, the user can at their discretion.

3. Tabs. Clicking on the link, you can open the site in a new tab and easily navigate between web pages opened in different tabs.

4. Password Manager. Now passwords can be saved on your mobile phone for quick access to secure web pages. If the user is confused by the security issue, all saved passwords can be quickly and easily deleted from the phone‚Äôs memory.

5. Support for keyboards and touch screens. Interface Opera Mobile 10 beta is optimized to work on the Internet using both telephone keyboards and touch screens without using a stylus.

6. Saving images. Photos of friends on social networks, such as VKontakte and Facebook, can be saved and viewed offline.

7. Opera Turbo. The inclusion of this technology allows you to open websites faster and reduces the amount of downloaded data, and hence the cost of mobile Internet.

8. Compatibility. Opera Mobile 10 beta is based on the core of the Opera desktop browser, which ensures compatibility of web pages based on open standards, and makes almost any site accessible from a mobile phone.

9. Easy viewing. Cropping, scaling and smooth scrolling of pages is all that is needed for comfortable work on the Web from a mobile phone.

10. It's free! The Opera Mobile 10 beta browser for the Windows Mobile platform is available free of charge at the following addresses: www.opera.com/mobile (for downloading to a computer) and m.opera.com/mobile (for downloading directly to a mobile phone).

Opera Mobile 10 beta for users from the CIS countries comes with the default search panel from Yandex.
