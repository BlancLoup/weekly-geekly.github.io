PHP: Step to Quality Code and ZendStudio (Code Assist)

One of the easiest ways to determine the quality of an application code is to look at IDE hints (for example, ZendStudio). At first glance, a simple technique carries a well-defined and rational moments.
  • Code transparency
  • following naming standards
  • proper use of visibility modifiers
  • use of OOP
  • lack of "magic" in the code
  • documenting methods and functions

Looking at the IDE tips, you can make a very reasonable conclusion about the quality of the code.
