Plex Media Center

Multimedia files like photos, videos and music can quickly absorb gigabytes of storage space. Then there is the option of viewing or listening to these files on multiple computers. You diligently copy the necessary files to your second computer, or create a shared network folder that everyone can access.
Another option - which has the added advantage of providing you with no fuss, an attractive user interface for viewing and accessing your media collection - is the Plex server.
Plex simplifies the process of sharing and managing media content on your home network and, thanks to free plugins, you can access your content from anywhere in the world via the Internet.
For 2011, Plex learned to work not only on Mac OS, but also on Windows and Linux, and even Android.

Installation is simple: install Plex Media Server on one of your computers, usually the one that will be a home server or contains most of your media content, and when it is turned on you will have access to its content through the network or even further through The Internet.
To configure select folders (more than one folder can be allocated for each type of supported media content: photos, music, movies / videos and recorded TV shows), select additional plugins to add support for online services. Server configuration is configured using a web browser: just go to "http: // localhost: 32400 / manage".

The Mac version of the Plex server also contains client software, so you simply copy the distribution package of the server and install it on a computer with Mac OS, and then use it as a client.
You can navigate the client interface from the keyboard or the Apple Remote (you can enable mouse support through Settings> System> Input Devices). One tip: after installing and configuring on a Mac client computer, go to Settings> System> Media Server and turn off always running to save system resources.

Installation on Android and iPhone applications is even easier, just download the client for the mobile device, install it, register access to the main server and that's it.

The desktop and mobile client provide a superb, easy-to-view interface for all media collections, complete with various online databases (Wikipedia, Amazon and TheMovieDB). It looks like this: you upload a movie into Plex, music, and the server itself already recognizes what content it is and displays brief information on it. He also works with music. It is possible to prohibit him from scanning files, and specify information manually.

You can configure access to the server from anywhere in the world via the Internet.
The server is installed, the ports on your computer are opened, the ports on the router are opened (if there is one). And it's all! Now you can open your favorite music collection anywhere on the planet where the Internet is.
It goes without saying that accessing multimedia via an Internet connection will be slower than on a home network, and can be expensive if you use a 3G phone connection.

Plex Media Server is free for download on Mac OS (OS X 10.5.8 or later), Windows, Linux.
