Google added Flash support in Instant Preview


Today, Google has added a new functionality to one of its search engine's tools - Instant Preview, which allows you to evaluate the site on the search results page before switching to it. Previously, if the page contained a Flash animation, then Instant Preview displayed an error message, now Flash functionality is fully supported. This feature is working so far only on those sites that Google-indexing robot managed to visit, and full support for all sites is expected in the near future. On the screenshot I made, the Instant preview displays several clips on YouTube, but one of the Ukrainian Flash sites refused to display correctly yet.
