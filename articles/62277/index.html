Microsoft Small Basic 0.5 has been released



Here are some of the changes:

  • Uninitialized variables:
    The compiler can now track and report on variables in the code that are used, but have not been initialized.

  • Added spanish localization

  • Development Environment Updates:

    * Added search function, called by Ctrl + F
    * Added display of the current row and position in the row
    * Added "Save As ..." command
    * Added ability to format the program code (called from the context menu by right-clicking in the editor)

  • And:
    * Many bug fixes
    * Smaller installation file size (3.7 MB)
    * The installer now automatically detects the presence of .Net framework 3.5 SP1


The release can be downloaded from here.
