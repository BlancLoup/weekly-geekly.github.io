Microsoft certification


Microsoft certification is designed for professionals in various fields related to information technology, as well as hardware and software certifications for compatibility with Microsoft products.

Examinations are taken at VUE centers. Office exams are taken at Certiport testing centers.

The cost of exams is usually $ 80 (60 EU), but some exams may cost less ($ 60).
The overall picture can be seen on the following poster:



But let's take a look at certification in more detail, just Microsoft has several different levels and degrees.

1) Microsoft Certified IT Professional (MCITP) - is given for passing one exam in a specific area.
2) Microsoft Certified Technology Specialist (MCTS) - is also given for taking one exam.
3) Microsoft Certified Professional Developer (MCPD) - the status is given for development certification.
4) Microsoft Certified Solutions Associate (MCSA) - the status is given for passing several exams in a specific area. This status can be obtained in several ways: Windows Server2012, Windows Server 2008, Windows 8, Windows 7, SQL Server 2012, SQL Server 2008, Office 365.



5) Microsoft Certified Solutions Developer (MCSD) - the status is given for passing several exams in various areas of development. Such as: Windows Store Apps, Web Applications, SharePoint Applications, Applications Lifecycle Management.



6) Microsoft Certified Solutions Expert (MCSE) - is a deeper level to MCSA, therefore, MCSA is required to obtain MCSE.



7) Microsoft Office Specialist (MOS) - direction of certification for MS office products.



8) Microsoft Technology Associate (MTA) is Microsoft's entry-level certification for individuals planning professional activities as a technical specialist.



9) Microsoft Certified Trainer (MCT) - certification for Microsoft instructors.
10) Microsoft Certified Database Administrator (MCDBA) - referral for database administrators to SQL.
11) Microsoft Specialist - given for certain exams.
