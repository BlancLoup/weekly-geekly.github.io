Free software will be added to the Windows 8 program directory.

Unlike the App Store, the Windows Store App Developer Agreement will allow the cataloging of software under free licenses. Recently, Microsoft has published a preliminary version of the App Developer Agreement . Clause 3.g of this document is devoted to licensing agreements.

The conditions stipulate that each developer chooses the type of license himself and provides it to Microsoft. If he did not do this, then the program establishes standard licensing terms (see Appendix A).

A proprietary license must not conflict with standard conditions, unless it is a free license of the Open Source Initiative (GPL, Apache, etc.), which is allowed to conflict with the fourth paragraph of section 3 of the standard license (prohibition of copying and distribution of the program).

Microsoft also states that free software in the directory must exactly comply with the terms of the free license, including providing source code for the programs, if agreed. Microsoft will closely monitor this.

The cost of paid applications in the Windows Store App should be in the range of $ 1.49 to $ 999.99, and Microsoft takes from 20% to 30%, depending on the accumulated sales volume.
