VirtualBox 5.0 release

Yesterday morning, the free Oracle VirtualBox cross-platform virtual machine version 5.0 was released. Improvements include: drag-and-drop files and folders, hard disk encryption on the fly, and more. A short list of improvements under the cut, you can download from the official site .

Improved Paravirtualization


Significantly improved the speed of guest Windows and Linux due to the fact that Guest Additions better modify them to work in a virtualized environment.

Improved processor virtualization


Even more CPU instructions are now available for guest operating systems, which can be used by modern programs running on these operating systems.

USB 3.0 support


Guest operating systems can now interface with USB 3.0 devices at full speed.

Two-way drag-and-drop on windows


If virtualbox is running on windows, then double-sided drag-and-drop files and folders for guest windows, linux and solaris are available!

Virtual Disk Encryption


The virtual machine disk can be encrypted in real time using the AES algorithm with a key length of 256 bits.

Detach mode


The GUI can now be closed, leaving the virtual machine running in the background. I've been waiting for this for the past five years.
