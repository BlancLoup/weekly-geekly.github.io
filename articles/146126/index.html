NVIDIA commented on Linus Torvalds' critical remarks

Original

Linux support is important for NVIDIA, and the company understands that there are people who are very keen on Linux as an open source platform, just as we are interested in providing people with the fastest video accelerators.

Recently, the question was raised that NVIDIA does not support Optimus technology for laptops for Linux. When we released it, we ensured its support only in Windows 7. Nevertheless, people from the Open Source community rallied and created the Optimus support project - Bumblebee. As a result, we recently changed the installer and documentation of NVIDIA drivers for Linux to accommodate Bumblebee, as well as, starting with the 295 series drivers, made changes that are designed to simplify interaction with Bumblebee.

We understand that many people would prefer us to release detailed documentation about the insides of our video adapters or to become more active in discussions concerning the Linux kernel, but we decided to support Linux by using as much common code as possible (with the Windows platform) rather than investing resources in developing code specific to the Linux platform. Such a ‚Äúmodus operandi‚Äù, of course, will leave some people unsatisfied, but allows us to provide the same high quality work with our equipment, regardless of the platform.

As a result:

Linux end users have the ability to get the same and simultaneous performance with Windows version of OpenGL, as well as support for all OpenGL extensions.
Under Linux, both on desktops and laptops, we support the diversity of our products, including the latest versions of GeForce, Quadro and Tesla video accelerators. We regularly update drivers for them, so only this year we have released seven driver releases.
We are actively involved in the development of the ARM architecture in the Linux kernel. For example, the kernel version 3.4 provides support for ARM systems of a new generation, which will be used in Linux distributions, as well as on Android and Google ChromeBook. For this version of the kernel, NVIDIA ranks second in the number of modified lines of code and takes 4th place in the number of all changes made.
As a result, the provision of high-quality support for all our video accelerators on many platforms for all our consumers is one of our priorities.
