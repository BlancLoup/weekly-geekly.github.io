Windows 7 and Aladdin smart cards

For a long time (since the beta version and after the release), I could not work productively with Windows 7 due to the lack of full support for eToken smart cards from the Aladdin company. Therefore, I was very pleased with the output of the drivers for this OSes.

This driver package is named eToken PKI Client 5.1 SP1 and is at the beta testing stage, which does not prevent them from performing their function perfectly.

I would also like to note that in order to fully work with the programs that communicate with the smart card through the Windows certificate store (such as openVPN, Google Chrome or IE), you must start the service, which is disabled by default in Windows 7.
