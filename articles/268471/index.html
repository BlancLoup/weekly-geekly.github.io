Microsoft will soon close support for outdated versions of Internet Explorer

Microsoft will end up supporting versions of Internet Explorer 7-10 at the beginning of next year, January 12, 2016. For these outdated versions of the web browser, no security updates will be released and their users will be left with lifelong 0day vulnerabilities. Thus, the company will leave support only for its current web browser Internet Explorer 11 and MS Edge (Windows 10). Despite the fact that IE11 appeared in Windows 8.1, users of Windows 7 can use it.



At the same time, the company is not going to abandon the users still supported by Windows Vista SP2 (IE9), as well as Windows Server 2008 SP2 (IE9) and Windows Server 2012 (IE10). Users of these operating systems will continue to receive updates for specified versions of IE.
