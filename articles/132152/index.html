Transferring Flash to iPhone / iPad



What flasher would not want to twist their sculptures on the iPhone or iPad?
A seemingly simple procedure still contains a lot of ‚Äúpitfalls‚Äù. Many people did not know the happiness of watching flash applications on an apple ... But now everything will work out, before you the most complete passage of Flash2iOS on the network!

So, we will do good on Adobe Flash CS5.5 . It is possible on younger versions, but here without guarantees.

Open the FLA file with something beautiful.

Select File -> Publish Settings ...
Switch the player to AIR for iOS



Then go to File -> AIR for iOS Settings
With the first tab, you yourself will understand, go to the tab Deployment .

If you are the holder of the Apple developer certificate, then insert your certificates, otherwise we will download the ‚Äútraining‚Äù certificates (the application will work only with jailbreak devices). Substitute 2 necessary files, enter the password 1234 .



When all settings are complete, click Publish . If you're lucky, after a couple of minutes of watching the slider, a message pops up saying that the IPA file has been successfully created.
But many people at this stage begin to have fun problems.

Under Windows pops up a message:

Error occurred during initialization of VM
Could not reserve enough space for object heap
Could not create the Java virtual machine

It is treated only with a wrapper for Java , which was written that night (download and install, it is not friendly with the Cyrillic ways, so it is advisable not to publish IPAs from My Documents).

Under OSX may appear:

Unexpected or unknown element or attribute in the
application descriptor file. ...

Here you will laugh. Every time you need to open the ‚Äúunsuccessful‚Äù XML file in a text editor and delete the entire version tag (<version> 1.0 </ version>) - because the guys forgot to write its processing!

Next is easy . Add the resulting IPA file to your iTunes library, sync with the device, find the icon on it (white if you haven't put something) and enjoy the fun show.

Unfortunately, all dances with hardware acceleration settings do not yet give a tangible visual effect. Let's hope Adobe is working on this aspect day and night.

If someone has additional difficulties with obtaining a result on the topic - let's discuss in comments, add the material in the post.

UPDATE: By the way, it is possible to wrap ready SWF files into IPA when you for some reason do not have the original FLA (for example, exporting interactive coding from InDesign). This can be done using the ADT Helper Tool .

Also, this utility does not require the entire Adobe package; it only needs the AIR SDK to work, which can be downloaded for free here - www.adobe.com/special/products/air/sdk
