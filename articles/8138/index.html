Sun launches JavaFX - open-source rival Adobe Flash and MS Silverlight

At the JavaOne conference in San Francisco, Sun announced a Java-based product line called JavaFX, which covers development in Java in various environments, be it desktop, web, or mobile devices. This is the main feature of the new variation of the Java scripting language, called JavaFX Script. JavaFX is a product line focused on opportunities in the consumer communications market. The first released product is JavaFX Mobile, a software system for mobile devices.

"JavaFX Script focuses on creating content and achieving mass distribution of Java," says Rich Green, executive vice president of software for Sun. ‚ÄúJavaFX Script is a scripting language aimed at creating dynamic content. This is a tool for creating visually efficient, high-performance, bright network products that would run on desktops using Java SE (Standard Edition), and on mobile devices using JavaFX Mobile. ‚Äù

In other words, JavaFX Script is a direct competitor to Adobe Flash and Microsoft Silverlight, and it will also be distributed open-source.

The JavaFX platform can also attract the attention of AJAX programmers. James Gosling, vice president of Sun, said that JavaFX can be used for the same purpose as AJAX, but with Sun development you will get more dynamic solutions and access to more advanced APIs. Another distinctive feature of JavaFX from AJAX is the ability to work offline.
