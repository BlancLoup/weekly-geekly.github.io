LuCI moved to github

There was a time when the expectation of reviewing your edits for this project lasted for months. Some patches were generally deprived of attention. Trac was either ignored or there was no one to follow him.
October 10, this era ended. Now this great web interface is available on GitHub and is supported by the main OpenWrt community. Pull requests are treated fairly quickly.
Congratulations to all users and interface developers on this event.
Now almost all the basic elements of OpenWrt are available on GitHub, which is good news.
Thanks for attention.

OpenWrt - A popular operating system for routers and embedded systems.
LuCI - One of the most popular web-interfaces for systems based on OpenWrt.
