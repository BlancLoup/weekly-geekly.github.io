Linux Mint 10 Julia in the pre-release stage


The Linux Mint development team has released the Linux Mint 10 Julia RC distribution kit. The developers of the operating system strive to make the distribution as user-friendly as possible, and also to provide the most complete support for third-party plug-ins and codecs. Recall that Linux Mint is the third most popular distribution after Ubuntu and Fedora.

Components


Linux Mint is based on Ubuntu 10.10 Maverick Meerkat, using the capabilities of the Linux kernel 2.6.35, X.org 7.5, and GNOME 2.32.
Interface

Significantly changed the interface in the new version. The Linux Mint 10 welcome screen allows you to immediately install additional proprietary codecs or upgrade the distribution to the full DVD version:



The start menu has also changed, including convenient categories and the most powerful search tool:





The application center has become more social-oriented, complemented by a number of cosmetic improvements.

The developers note that the release is intended for advanced users and beta testers who are able to work with a possibly unstable system, as well as those who are ready to write bug reports on the Linux Mint bug tracker . For home and corporate use, the system is not yet ready.
