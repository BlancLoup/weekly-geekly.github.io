Avast! removes calc.exe as a virus (2: 0)

Most recently, I wrote a post about how Avast! deleted the Windows program to control the sound like a virus. Correspondence with Avast! ended in nothing, because according to them, such a case was allegedly observed only by me (although at least another person with such a problem wrote off in my topic at least).

Today is the second round. I run calc.exe (windows calculator) and get a message that there is no such file. Immediately go to quarantine Avast! and, here it is, the poor fellow calc.exe.





But maybe he was infected with an evil virus and therefore completely rightly removed? But checking the file on another server online shows that the file is quite kind and safe.
