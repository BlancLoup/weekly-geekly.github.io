Frequently used terms and abbreviations


COD Critical On-Demand ( Critical On-Demand Update)


  • CU Cumulative Update (cumulative update)
  • CTP Community Technology Preview (CTP version)
  • GA General Availability (public version
  • GDR General Distribution Release (update GDR)
  • QFE Quick Fix Engineering (QFE update) (in the Windows world, this is called LDR - Limited Release Distribution - but this term has not taken root in the SQL Server team).
  • RC Release Candidate ( Candidate Version)
  • RDP Rapid Deployment Program
  • RTM Released To Manufacturing
  • RTW Release to Web
  • TAP Technology Adoption Program
  • SP Service Pack
  • SU : Security update.
  • Hotfix

The usual procedure for installing SQL Server is as follows:


  1. The stable version you need is installed (currently recommended is 2014 or 2016)
  2. Then installs the last available service packs.
  3. Installs the latest cumulative package of updates to eliminate various kinds of bugs
  4. If necessary, the necessary Hotfix is ‚Äã‚Äãinstalled to eliminate specific problems.
