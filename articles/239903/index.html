Come freebie!


A freebie is a student goddess, to which students cry out on the eve of the exam, instead of preparing for it. But despite the fact that the session is far away, Her Majesty a freebie can make herself known even now and be useful. And in this case we are talking about tools available to students for free.

Microsoft
Microsoft has a program called DreamSpark, according to which some paid products are available to students completely free of charge.
Basic set includes:
  • Windows server
  • Windows embedded
  • Visual studio professional
  • Free registration in the Windows Store and Marketplace
  • Expressions
  • SQL Server

The extended subscription in addition to the above allows you to get:
  • Windows client
  • Visual studio premium
  • Visual studio ultimate
  • Visio
  • Project
  • Onenote
  • BizTalk Server
  • SharePoint Server

You can view the offer here .
There is also a separate tariff plan for 4 years for office 365 for students .
Github
The other day GitHub announced the Student Developer Pack program. It includes:
  1. Atom - Text editor of the 21st century. In general, it is free, but is on the list.
  2. Bitnami - Install cloud applications in one click (Business 3 plan for a year, the usual cost of $ 49 per month)
  3. Crowdflower - Crowdsourcing platform (savings of $ 2,500 per month for access and $ 50)
  4. DigitalOcean - Cloud hosting ($ 100 credit per account)
  5. DNSimple - DNS service (free for 2 years, instead of $ 3 every month)
  6. GitHub - Git-repository service ($ 7 micro account)
  7. HackHands - Programming support 24/7 (credit $ 25)
  8. Namecheap - Domain Name Registrar (.me domain for free, regular cost $ 8.99 per year) and ssl certificates (free, cost $ 9 per year)
  9. Orchestrate - Search, geolocation, graph-based database and API (developer account for $ 49 as a gift)
  10. Screenhero - Total access to screenshots for teamwork (regular price $ 9.99 per month)
  11. SendGrid - Service for working with email (regular price $ 4.95 per month)
  12. Stripe - Web and mobile payments for developers (first $ 1000 without commission)
  13. Travis CI - Continuous Integration Service (usually $ 69 per month)
  14. Unreal Engine - no comment (which allows you to save $ 19 every month)

You can get acquainted with the proposals on the program here .
