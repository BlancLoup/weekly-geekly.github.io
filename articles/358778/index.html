GeekBrains opens a set of free programmer training courses.


Hello! We are opening a set of free training courses for Ruby and Frontend developers . To participate you must pass the competition. Five of the best graduates will have an internship with us and will continue to work in the office. Applications for training are accepted until June 17 inclusive. Details of the set and training program - under the cut.


Until June 17, applicants need to pass the entrance tests consisting of 5 stages: online testing using mathematical logic, passing a video course, performing an interactive programming task, performing project tasks on GitHub, and recording video interviews about themselves. For enrollment in the free course you need to score maximum points for the minimum time in the entrance test. The results of entrance examinations will be published on June 20 .


Training will begin on June 25th . Each study group is designed for 30 people.


Training Ruby-developers consists of three modules: HTML / CSS, Ruby, Ruby on Rails.


Program description:
  • HTML / CSS
    You will practice the hypertext markup language: create an online store site and start working with it.
  • Ruby
    You will learn the basics of the Ruby language and understand its philosophy. As a practice, you will write the simplest game in this language.
  • Ruby on rails
    You will learn how to use the framework and tools that are often used with Ruby on Rails in real-world web development.

Frontend developers will take five courses: HTML / CSS, HTML5 and CSS3, JavaScript. Level 1, JavaScript. Level 2, ReactJS.


Program description:
  • HTML / CSS
    You will practice the hypertext markup language: create an online store site and start working with it.
  • HTML5 and CSS3
    The course will teach how to work with non-standard fonts, create static sites and text effects. You will try out new knowledge on your project.
  • Javascript Level 1
    You will learn how to create events in the browser and respond to them. New skills you consolidate in practice - working with the game on JavaScript.
  • Javascript Level 2
    You will learn object-oriented programming, learn how to use graphical interactive applications.
  • ReactJS
    You will master one of the most popular libraries for professional development in the field of frontend - ReactJS. This is a library for building user interfaces. By the end of the course we will develop a small social network.

Training will last 3 months. We are waiting for the best graduates for a paid 3-month internship at Mail.Ru Group and GeekBrains. The rest will have the opportunity to do an internship at partner companies. We are waiting for your applications!
