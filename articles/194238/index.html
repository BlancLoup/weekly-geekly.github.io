Mobile Opera 16 for Android

Today we are updating our Opera mobile browser for Android to version 16. The list of changes is as follows:

  • Engine updated to Chromium 29
  • Improved history navigation
  • Updated download manager
  • Improved NFC connectivity
  • More stability, speed and less bugs

In addition, the mobile Opera for Android has been updated. In addition to these changes, we have updated the compression mode (Off-Road Mode). If earlier the compression mode included Opera Mini, that is, all sites were rendered on our servers and they came to you in the OBML format (read more in the article on Dev.Opera ), now Opera Turbo, a mode known to you from computer version of Opera, which is even in Yandex. Browser. This mode also compresses pages, uses the SPDY protocol for acceleration, but processes pages directly on the phone.
