Google has released a version of Chromium with support for Dart "Dartium"

Some time ago, Google introduced to the public a site for comparing syntaxes of JavaScript and Dart . Now the company has taken another step towards popularizing its new language for web development by releasing the tech preview version of Chromium, codenamed Dartium for Linux and Mac, with the integrated Dart virtual machine (Dart VM).

The presence of Dart VM will allow you to run Dart-code, bypassing the compilation stage - the programs will run faster due to the fact that you do not need to perform Dart-to-JavaScript translation.

The developers, of course, do not recommend using Dartium in their daily work, however, after the testing and debugging stages, Dart VM is planned to be included in regular Chrome releases.
