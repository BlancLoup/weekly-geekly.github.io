Nokia X, X + and XL received an update

Many are already up to date with the latest Microsoft Mobile changes regarding the Nokia X range of smartphones (based on the Android Open Source Project). At the same time, we have always taken seriously the support of current customers, and Nokia X buyers will not be an exception - updates will be released in the next 18 months.



And, not to be unfounded, we are pleased to announce the start of distribution of the update for entry-level smartphones Nokia X, X + and XL , which was launched last Monday. The update will open to users a number of improvements in the operating system and offer new features and capabilities, thereby bringing the first generation device closer to the recently released Nokia X2 smartphone.

One of the key features of the interface after the upgrade will be the App Switcher running applications manager, migrated from the news.

Since physical control in the early Nokia X smartphones differs in the number of touch buttons (in X, X +, XL one-button interface, in X2 - two-button interface), App Switcher launch is implemented through a separate menu.



In order to see running applications, users need to open the swipe control panel from top to bottom and touch the App Switcher icon. The dispatcher's menu visually displays all open applications: to go to the one you need, just touch it, to close it, click on the cross.



With the update, a number of Microsoft services will be pre-installed and integrated into the system.

These include Outlook.com email, OneDrive cloud storage and OneNote note management service (with cloud sync).



Outlook.com fully supports multiple Outlook accounts, pop-up notifications and calendar and contact synchronization.

OneNote Mobile allows you to not only work with Microsoft OneNote 2010 notebooks and notes stored in OneDrive (as you might think), but also create sketches directly on Nokia X and sync up to 500 notes.



The first generation devices also received an updated Nokia Store, already familiar to Nokia X2 users. The first thing that users will be able to mention is the most convenient search that allows you to quickly find and install applications and games (up to 75% of standard Android applications can run on Nokia X).
The Nokia Store also integrates third-party app stores, so now, without finding the desired option, the user does not need to go to other resources on his own.
