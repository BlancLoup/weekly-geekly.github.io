Angry Birds USB Slingshot



About this game did not write and did not speak only lazy. Probably, among the representatives of Habr there are those who adore this toy, and those who hate it with pure and bright hatred. However, now it is not so much about the birds, but about a unique gadget created by one craftsman for the game and own entertainment. This device ... slingshot. Yes, natural slingshot, with USB connection to a laptop / desktop.

Do you still not have one? Well, yes, there is nowhere to buy, that's for sure. But you can do. At least one of the craftsmen, of whom there are not so many, created a self-made USB slingshot that allows you to play Angry Birds not on a touchscreen display or using a mouse. Now the game has become more interesting and exciting. The game is a craftsman, his name is Simon Ford, recorded on video.

Moreover, he posted on the Web an instruction describing the algorithm for creating the same slingshot as his. The address with the instruction can be found in the video clip - it is already visible on the starting frame. So who has the right hands and some free time (well, desires, of course) - you can try. And if anyone succeeds, share with us :)
