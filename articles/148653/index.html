Tiled Map Editor: pixelart tileset editor

http://mapeditor.org/



Tiled Map Editor is a pixelart tileset editor written in C ++ (there is also a Java version of the program). This program can be used to develop games of any 2D-genre. The editor has a large set of ready tilesets, which significantly increases the speed of game development. A pair of rollers using under the cut







The program runs on Windows, Mac OS X and ubuntu.

We advise you to familiarize yourself with the similar program Pyxel Edit
