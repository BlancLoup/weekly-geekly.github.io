5 free antiviruses for Windows

Leading antivirus software developers have taken care to popularize their products: almost every one of them has a free version of the antivirus, demonstrating the basic possibility of paid
applications - detection and removal of malware.

Consider 5 programs and their main features.
  1. Built on the basis of the original products from the developer.
  2. Do not provide real-time protection.
  3. The program does not include the module for automatic updating of virus databases, so for a new scan of your computer you will need to download the antivirus again with the latest virus databases. Exception, Norton Security Scan - automatically updated with every scan.
  4. Antiviruses do not conflict with other antivirus programs and can be used as an additional tool for an already installed basic computer protection.

Every time the computer is infected, and the installed security program does not cope with the threat, you can use the following antiviruses:

1. Dr.Web CureIt!
The well-known utility from Doctor Web, which has repeatedly saved the computers of many of us from viruses, trojans, Internet worms. Read more ...
Size / OS: ~ 11 MB, Windows 98 / 2k / XP / 2003 / Vista
Update: several times a day
Download Dr.Web CureIt!



2. Kaspersky Virus Removal Tool
Free antivirus scanner built on the engine of original products from Kaspersky Lab. Detects and removes malware. Read more ...
Size / OS: ~ 24 MB, Windows 98 / 2k / XP / 2003 / Vista
Update: every two hours
Download Kaspersky Virus Removal Tool



3. NOD32 On-Demand Scanner
Antivirus scanner on demand. Free version of NOD32 antivirus, which allows you to quickly and efficiently detect and remove viruses, Internet worms, Trojans and other malware. Read more ...
Size / OS: ~ 13 MB, Windows 2000 / XP / Vista
Update: several times a day
Download NOD32 On-Demand Scanner



4. McAfee On-Demand VirusScan
Antivirus scanner on demand. The free version of McAfee VirusScan antivirus, which allows you to check and cure your computer against viruses, Internet worms, Trojans and other malicious programs. Read more ...
Size / OS: ~ 13 MB, Windows 98 / 2k / XP / 2003 / Vista
Update: Everyday
Download McAfee On-Demand VirusScan



5. Norton Security Scan
An antivirus scanner that helps determine if your computer is protected from known malware and security threats. The main disadvantage: does not remove the detected malware - offers to use a paid version of the antivirus. Read more ...
Size / OS: ~ 13 MB, Windows 98 / 2k / XP / 2003 / Vista
Update: automatic
Download Norton Security Scan
