18. Run with elevated privileges


Do you want to quickly launch an application attached to the taskbar with administrator rights? Easy: hold down Ctrl + Shift and click on the application icon (your account, of course, must have corresponding rights).

19. Repeat, please!


I have seen people who were bewildered by the following. Suppose you have already opened some application (for example, the command line) and you want to open a second copy of the same application. You do not need to go into the Start menu again. Just hold down Shift and click on the icon on the taskbar, and instead of switching to the existing window, a new copy of this application will open. The same effect can be achieved by clicking on this button with the middle mouse button .

20. Switch to specific windows


Another innovation that will appeal to experienced users allows you to switch between the windows of a single application. For example, if you have 5 messages open in Outlook, you can quickly switch between them by holding Ctrl and pressing the button on the taskbar. This is much more convenient than searching for these windows in the Alt + Tab dialog.

21. Walk through the taskbar


The combination of Win + T will transfer the focus to the taskbar, after which you can use the cursor to move between applications. Enter activates the selected application / group. Esc removes focus from the taskbar.
