In the preview of Windows 10 appeared Spartan browser



The Microsoft company said that preliminary versions of Windows 10 will be released more often, and this promise was fulfilled. On March 30, less than 2 weeks after the previous one, a new build 10049 was released with an emphasis on the Project Spartan browser. The build number increased by only six from the previous release.

Build 10049 fixes some of the bugs in Windows 10, but the biggest change is Microsoft‚Äôs new minimalist browser, codenamed Project Spartan. This is the first official release, although there used to be other ways to access it. Because of the novelty, it lacks many important functions: no history or downloading files. The main Spartan differences already available are:

  • Voice control Cortana. The voice assistant can show the weather and stock prices, but data from the calendar and flight schedules is not yet available. At the moment, Cortana is available only to users from the United States.
  • On the pages, you can leave text notes and draw with a stylus, and then share the received by e-mail or in social networking services. You can also save notes to the OneDrive cloud storage.
  • A reading mode that focuses the user's attention on the content, not the elements of the page. The function allows you to save a web page as a PDF file, materials can be collected in the Reading List.

Joe Belfiori specifically emphasized that in its current state Spartan is very far from complete. This is not a release for distribution to users, there will be many more additions and improvements in the browser, and only after that it will be made publicly available.



Spartan will become the default browser in Windows 10, it will be available on the start screen and on the taskbar. Internet Explorer 11 will remain for compatibility, but to launch its icon you will have to consciously search.

The original page rendering plan was this: when compatibility is needed, Spartan uses the old Trident engine of the Internet Exporer browser, and Internet Explorer can use the Edge engine of its new counterpart. Later this idea was abandoned , in Spartan there will be only Edge, and in Internet Explorer only Trident. Thus, the difference between browsers is more obvious. Internet Explorer becomes a compatibility tool that you need to run for, for example, special corporate sites or for tests. Spartan is a browser for everyday use, compatible with all major standards.
