Adobe Captivate feature overview

Adobe Captivate is a program for creating and editing interactive training courses, with the possibility of demonstrating software, recording video lessons, creating program simulations, developing verification tests, and so on.

The course generated on Adobe Captivate can be downloaded in different formats. In my work, I most often use the formats: * .html, * .exe and scroom 1.2 to upload courses to Moodle and other eFront platforms.

The tool does not require the installation of additional software on users' machines, which is a definite plus.

To create simulations of programs, Captivate allows you to use the right and left mouse buttons and keystrokes, which is very convenient when teaching users, without access to the "combat" stand. For demonstrations of programs recording in real time is possible. Screencasts created with Captivate take up much less space than full screen recordings.

Now the 9th version of the product has been released and the demo version can be used for 30 days, and the 6th version is not so bad and is freely available on torrent trackers. I use the licensed version 7, it is more convenient to 6, but there are not too many differences, and there are no critical ones at all.
