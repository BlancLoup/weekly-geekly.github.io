Profiling and backup



So the first programs for the Google Chrome browser began to appear, below is a small overview of the backup program, and also, importantly, data profiling, that is, creating profiles for users.

The program is called - ‚ÄúGoogle Chrome Profile Backup‚Äù.

Despite its small size (only 400 kilobytes), this is a very useful, convenient program. Any action, in principle, will fit into one click of the mouse.

Before the description of the functionality is not bad to know the program in person (that is, the interface).

Program screenshot





Description of the functional


Create New Profile - using this function, we actually create a new profile; by default, when creating a profile, a shortcut appears on the desktop to launch Google Chrome with the profile created.

Add an existing profile (Add Exiting) - you may have created several profiles manually before installing Chrome Profile Backup, now you can easily add them for maintenance by this program.

Delete profile (Delete) - the function to delete the selected profile.

Now the most important thing ...

Create Backup (Backup) - with the help of this function we create a backup, which you can later restore.

Restore backup (Restore) - this function restores the profile from the backup file. For example, you can drop a backup from your home computer to a USB flash drive and synchronize bookmarks, visited pages, etc. with your working computer.

Create Shortcup - create a shortcut to the active profile.
