What is Jelastic?

Jelastic is a new generation of hosting platforms for Java and PHP, providing the ability to run and scale Java / PHP applications without changing the code.

Connection and setup of environments are fully automated. There are no restrictions or special requirements for applications that protects the user from unexpected expenses.

Deploying Applications

The process of deploying applications consists only of the steps of downloading the archive and choosing a software solution in which to deploy this archive.



Scaling

Jelastic PaaS is the only cloud platform that can automatically scale any application, independently changing the amount of resources required by the server. When the load on the application grows, additional resources are automatically allocated to the server, and as the load decreases, the amount of resources consumed decreases.
