Kickstrap - fork Twitter Bootstrap with themes and bonuses


Kickstrap is what will happen if you connect Twitter Bootstrap , HTML5 Boilerplate , icons from IcoMoon and Font Awesome , add proven jQuery Chosen and jGrowl plugins , enable support for different color schemes and Bootswatch themes , additional CSS grid and supposed support for earlier versions of IE.

The default Twitter Bootstrap project is distinguished by a greenish default color scheme, additional javascript plugins for creating friendly Chosen drop-down lists ( Kickstrap demo ) and jGrowl, which displays messages in the browser as well as Growl in MacOSX ( Kickstrap demo ).

The HTML5 HTML5 Boilerplate template, with all the resulting crossdomain.xml settings, work with Modernizr and the build script, is taken as the HTML base.

Kickstrap perfectly styled (a variety of topics ), now there are 12 + standard green. Support for themes for the project is adapted from https://github.com/thomaspark/bootswatch , the theme itself is connected with the appropriate .less file.

The project also adds support for the OLCG (One Line CSS Grid) grid, which contains only 3 classes - oh = "One half", ot = "One third" and tt = "Two thirds", but at the same time supports nesting, different browsers and screen sizes. Demonstrating the operation of the grid all these features perfectly shows.

Instead of Glyphicons, Kickstrap uses Awesome Icons and IcoMoon.

To support IE 6, the development project of Bootstrap-IE6 is used , which was originally made for Twitter Bootstrap v2 itself.

The project is actively developing and supporting the new versions of the original Bootstrap, and on April 15, the Kickstrap 0.9 RC version is already planned.
