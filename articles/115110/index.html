How to disable the site in Chrome

Faced a very strange and very wrong, in my opinion, behavior of the Chrome browser.
Today, March 8, more than ever, various congratulatory pictures are used, which in turn are taken from a number of garbage dump sites with such pictures. For example animashka.info and others like her.
People copy the embed code from there and use it in forums and blogs.
And then the following happens: on the hosting of images, a js-trojan appears, and it falls into the black list, according to which all normal browsers block access to this site. And Chrome, and Firefox, and rightly so.
But Chrome went even further. It also blocks those pages of forums and blogs, on which image codes are simply inserted, i.e. just an img tag.


For example - inserting this image blocks the page in Chrome, if you do not remove it under the cat - the main Habr page will be blocked ...

UPD: the site has already been fixed, taken out of the emergency situation and this page is no longer blocked

I think this is an extra reinsurance. Firstly, we all know that the JS-Trojan is not transmitted through the image. Secondly, if you already block, then not the whole page, but only that picture from a suspicious site. And thirdly, other browsers do not block such pages.
