Meteor version 0.7.0 released


Meteor.js is an open (MIT) full-stack JavaScript framework (both client and server). Today version 0.7.0 was released (version 1.0 will be released at the beginning of 2014).

The main change is support for MongoDb OpLog. This enables full horizontal scaling (see my translation ). In the previous version of Meteor, in the case of several servers, data updates could arrive with a delay of up to 10 seconds - now there is no such problem.
