At the end of last week, two versions of hypervisors came out at once. This is about ESXi 6.0 Update 3 from VMware and XenServer 7.1 from Citrix.

/ photo TJStamp CC

XenServer 7.1 is the company's first product in the Long Term Service Release format, which comes with five years of support from the manufacturer and another five years of extended support.

In XenServer, Citrix introduced a new feature called the Provisioning Services Accelerator (PVS). According to the developers, it allows you to run more virtual desktops with less load on the servers. This is possible by caching desktop images in the accelerator, which eliminates the need to access the server from each device. The company presented a small video where it explained the principles of the function.



Version 7.1 also added a virtual machine conversion tool that allows you to migrate a Linux VM from vSphere to XenServer.

As for VMware, ESXi 6.0 Update 3 was released on Friday and received a client update, in which some bugs were fixed and vSphere Client functionality added. There is also support for TLS versions v1.0, v1.1 and v1.2. You can learn how to configure TLS v1.0, TLS v1.1 and TLS v1.2 from the material at the link .

Several changes related to vSAN were also introduced, which increased performance when working with all-flash and hybrid storage systems. For this, we optimized the storage space for logs and increased the performance of write operations.
