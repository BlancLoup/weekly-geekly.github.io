Quick transition to a new platform

Some time ago a new version of the e-commerce platform for 1C-Bitrix: Site Management and 1C-Bitrix: Enterprise products was released.

We have tried to simplify the migration process, but still changing the platform is a responsible business, and some users may fear making changes to their well-established, streamlined system. Especially if the site is large enough. Therefore, we decided to make this small tutorial, in which, as clearly as possible, we described the simple process of migration to the new e-commerce platform ‚Äú1C-Bitrix: Website Management‚Äù.

Install Updates


To successfully change the platform, you must first install a number of updates. As an example, take a standard online store with a default template. And let it be specifically the old template, originally created not for the new platform, but for the previous one, version 14.5.

At this test store "roll" immediately release updates, without intermediate installation of beta versions.

After the installation is completed and the page is updated, we see the following picture:



Launching the transition wizard


After installing the updates, you need to run the wizard, which will transfer the online store to a new platform. You can find a link to launch the wizard in the first green line (in the previous illustration).

In order not to pile up a lot of screenshots here, we recorded a small video showing the process of migration of the store to a new platform:



Important points to pay attention to:

  • Before running the wizard, be sure to create a full backup of your project.
  • Ask the developers leading your project to check the compatibility of their modifications with the new platform and, if possible, make corrections. The new platform was available for more than six months in the form of beta updates, and all those who are interested in compatibility have already released updates of their solutions.
  • In our example, the locations were converted during the transition. The new version of the locations came out more than six months ago, so this may not be the case in your wizard. This means that you have previously switched to a new version of the locations.
  • At the end of the transition, you will see new forms of order management. This indicates that the transition has been successfully completed.

Installing updates after the transition to the new platform


So, the transition is made, but something else needs to be done. Return to the Platform Updates section and install the available updates.
