Jelastic PaaS - usage statistics for databases, servers and JVM in November

We continue to share with you statistics on the use of the software stack on the Jelastic platform. What were the preferences of developers in the past month?

MySQL, MariaDB, PostgreSQL and MongoDB database usage statistics

Let's start with SQL databases. As before, MySQL remains the undisputed leader. PosgreSQL takes the second place of honor, followed by MariaDB. The only one among our NoSQL databases, MongoDB showed a very good result in November.



If we compare the current data with the October data, we can conclude that the number of fans of each of the bases has not changed much.
Situation in the regions (Europe and North America).



Current numbers

Europe North america Total
Mysql 45% 44% 44%
PostgreSQL 17% 12% 21%
MariaDB 18% 24% 15%
MongoDB 20% 20% 20%

Usage statistics for Tomcat, GlassFish and Jetty servers.

Tomcat bypassed the rest by a wide margin, as in the past month.


Perhaps because it is the default server. Since December, on Jelastic , the default server is randomly selected.

Distribution by region.



Compared with the previous month, when Jetty was more popular in the United States, in November, developers from different continents almost reached a consensus in this regard.

Summary table

Europe North america Total
Tomcat 85% 86% 85%
Glassfish 12% eleven% 12%
Jetty 3% four% 3%


Java 6 vs Java 7

Let's analyze the usage statistics for Java versions. It has changed significantly during the month.



The popularity of Java 7 is growing at a very fast pace, given that this version is available on Jelastic for just a few months.
The situation has changed dramatically regarding the distribution by region. If in October Java 7 was more popular with American developers than European ones, now the opposite is true.



Current numbers
Europe North america Total
Java 6 79% 84% 82%
Java 7 21% sixteen% 18%
