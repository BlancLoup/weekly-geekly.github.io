URL spoofing in IE 7: new vulnerability in Microsoft browser

Secunia announces the discovery of a new vulnerability in the recently released final version of Internet Explorer 7 from Microsoft.

A website with malicious code can be given for reliable, using a specially formed web page. A user looking at the site will see in the address bar any URL assigned by the attacker, for example, www.microsoft.com .
You can check for vulnerabilities on the test page from Secunia.

This vulnerability once again calls into question the anti-phishing protection built into IE 7. Recall that within a few hours after the official release of the browser , the first vulnerability was discovered in it.
