Cloud9 IDE update - autocompletion, co-editing, offline mode and virtual server with console

Yesterday cloud IDE Cloud9 was updated. There have been several significant innovations. The first one is co-editing. Now several people can work on the same code in real time. There is a built-in chat.



The second innovation is a full-fledged development server in the cloud. It can run scripts in Python and Ruby, and even compile programs in C. In the paid version ($ 12 per month), full-fledged shell access to the development server is available.



The third is offline mode. Works on Mac and Linux. C using a small application, you can synchronize files between the cloud and the local file system. So far, synchronization works in experimental mode. You can synchronize up to 1000 files, in the future we plan to increase this figure to 100,000.



The fourth is autocompletion. As you type, not only the continuation options are output, but also the documentation from NodeManual . While only works for JavaScript, support for other languages ‚Äã‚Äãwill be added later.



In addition to these four major innovations, there are less significant. Updated interface, improved search, Goto File function, added ability to install npm packages from the command line.
