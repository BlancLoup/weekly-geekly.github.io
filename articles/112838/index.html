XFCE 4.8 on Ubuntu 10.04 and 10.10 through PPA



Xfce 4.8 is a lightweight working environment for the Linux family of operating systems. Previously available only as source code, and a little later through PPA, but only for ubuntu 10.04 users. Now got the long-awaited PPA and under the latest version of ubuntu maverick merkat. Tackle installation description and a couple of videos about the latest version of xfce

XFCE 4.8 for Ubuntu 10.04 Lucid Lynx (32bit OS only)
sudo add-apt-repository ppa: alexx2000 / xfce
sudo apt-get update
sudo apt-get install xubuntu-desktop

XFCE 4.8 for Ubuntu 10.10 Maverick Merkat (for 32 and 64 bit OS versions)
sudo add-apt-repository ppa: koshi / xfce-4.8
sudo apt-get update
sudo apt-get install xubuntu-desktop
