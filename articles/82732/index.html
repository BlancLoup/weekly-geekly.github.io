Golden Grid CSS: PSD Template

Relatively recently met with the Golden Grid CSS grid. I liked it a little weight and using the rule of the "golden section". Golden Grid is simpler than Blueprint. I do not insist that it is more convenient, but it is perfectly suited for my tasks.

Instead of the notation for the classes span-1, span-2 ... the more intuitive g160, g320, etc. are used. In addition, a good typography, tailored specifically for the grid.

The grid is relatively new (2009), so many important details are missing. For example, a psd template. Having decided to fill such a significant gap, I created such a template, since it does not take much time. Both rulers and gray transparent rectangles are drawn. This is done so that the grid can be put on top of the main design.
