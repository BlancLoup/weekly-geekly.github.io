Top 10 Free SharePoint Apps from Microsoft Store

Applications are your assistants throughout the day - they are convenient and easy to use on your iPhones, Android smartphones, and SharePoint. Yes, you read it right - in SharePoint . The popular collaboration tool features a wide range of applications or add-ons that can help you add new features or applications to an existing SharePoint page. While more and more IT professionals prefer this program, in particular due to the fact that SharePoint can be easily customized for individual needs of companies, the development of new functions still takes time and skills. An application can be a quick and proven solution. In the Microsoft app store you can find more than 1,500 useful features and widgets. The price range ranges from free apps to apps costing a few hundred euros.


We have compiled a list of applications for SharePoint, which we believe are most useful. And more importantly, these applications are free.



10 most useful, in our opinion, free applications for SharePoint


1. Word to Wiki Online (Lite)


This application allows you to publish Word-compatible documents that are hosted in a SharePoint document library as a Wikipedia article with just a few clicks. Graphic content does not need to be uploaded in advance, since everything is placed in the SharePoint image library automatically.



2. Export to Word


... and this application is intended for opposite purposes. The Export to Word application is designed to export SharePoint data to Word documents.




3. myFAQ


Using the myFAQ application, you can integrate FAQ (Frequently Asked Questions) into SharePoint. Thus, colleagues working together on a project can quickly find answers to frequently asked questions. FAQ can be sorted by priority and discretion.



4. Dropbox Import



The name says it all: this application allows you to import files from Dropbox into a SharePoint document library.




5. Documents shared with me


There are such projects, while working on which hundreds of documents are scattered in different folders and pages. If several colleagues work on the same SharePoint platform, a document can easily get lost. Document icons that the user is currently working on can be integrated into the application's sidebar. Documents shared with me. The author can independently filter such a list or use the search string. Access to the latest documents can be obtained at any time and from any place.

6. MeetSweet!



MeetSweet app! Allows you to invite colleagues, partners, and customers to a meeting outside of the Office environment. Invitations are sent via calendar events.



7. My Locations



Using the My Locations application, you can easily find the right places on the map for SharePoint2013. For example, maps can be integrated into a team page and, at the same time, all branches of a company can be marked on them.



8. My SharePoint Sites



The SharePoint platform offers its users numerous opportunities to provide access to specific SharePoint sites. The more pages, the easier it is to get confused. With a single click, the application displays all the allowed places of the corresponding user and brings him / her to the right place.

9. YouTube Feeds



The Youtube Feeds application selects the user's (publicly available) videos and displays them as slides. It provides a wide range of configurations regarding video selection, user settings, various slide options, etc.


Want to know more?


For non-advanced users of SharePoint and users using older versions, such as SharePoint 2007 or 2010, it is useful to visit the Microsoft Channel 9 video blog, where you can learn how easy it is to work with new applications for SharePoint.
