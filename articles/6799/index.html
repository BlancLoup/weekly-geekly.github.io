CentOS 5.0 released

Announced the release of the 5th version of the operating system CentOS .

The main changes in CentOS 5 compared to CentOS 4:
Updated software versions: Apache-2.2, php-5.1.6, kernel-2.6.18, Gnome-2.16, KDE-3.5, OpenOffice.org-2.0, Evolution-2.8, Firefox-1.5, Thunderbird-1.5, MySQL-5.0, PostgreSQL-8.1.
Improved desktop support: compiz and AIGLX.
Virtualization with Xen hypervisor with Virtual Machine Manager and libvirt.
Simplified creation of user profiles: Sabayon.
