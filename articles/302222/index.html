IBM launches first highly secure cloud blockchain services

IBM Cloud based blockchain services are designed for financial institutions, public sector and healthcare organizations



As already reported, in January of this year, our company opened a blockchain laboratory. The project was launched with the support of a partner, a non-profit organization Innovate Finance. The goal of the project is to explore the additional capabilities of the blockchain technology using significant computational and analytical capabilities of the company.

Now the partners are already quite far advanced in the project, so it was decided to launch the first services based on this technology. It is worth recalling here that blockchain networks are based on the concept of decentralized control, however some cloud environments retain a vulnerability that allows for hacking of information systems and unauthorized access. The IBM team, consisting of information security experts, cryptographers, hardware experts and researchers, has created groundbreaking cloud services for robust, trusted networks based on blockchain technology.

Services of this type help create and manage blockchain networks. This is necessary to ensure reliable operation of the new class of distributed financial applications. Now developers can create digital assets and corresponding business logic, which will open up the possibility for a more secure and confidential transfer of assets between participants of the blockchain test network with limited and protected access rights. Using new services from IBM, available on Bluemix, developers can use fully integrated DevOps tools to create, deploy, run and monitor blockchain applications on the IBM Cloud. Applications of this type can also be deployed on IBM z Systems , providing an additional level of security for especially valuable data. Blockchain applications can access existing systems on Z Systems distributed servers and computers through an API, thus supporting new types of payment, transaction, procurement and business management procedures.

Among other features, we will highlight the connection of IoT data to the blockchain. Using the Watson IoT Platform, IBM allows you to use such types of information as device positioning coordinates using radio modules, readable bar codes, or device data. In the near future, IoT gadgets will be able to work with distributed blockchain logs to update and view contact data. For example, the location and temperature of the package using this technology will be automatically recorded and recorded in the blockchain. At the same time, all parties related to the package will be able to receive an update of its status.

In order to introduce developers to the new product, our company conducts the so-called IBM Garages for blockchain developers. What it is? IBM Garages , one might say, is a service station that is already open or will be implemented in major cities in the near future. IBM Global Business Services also plans to extend blockchain counseling services to clients in the financial and banking sectors, plus logistics. It is planned to start working with IBM Interactive Experience (IBM iX) in the development of implementation options for the blockchain in more than 25 IBM studios around the world (IBM Studios).

Thanks to the IBM Garage opened in April in New York, the company actively collaborates with customers such as BNY Mellon to accelerate the design and development of a unique application for securities loans using blockchain for selling and transferring assets.

Choosing a security scheme for a blockchain in the cloud

The IBM Cloud enables enterprises to quickly create secure, crack-resistant networks with thousands of users. The cloud platform will help you create and deploy a blockchain in minutes, using registered, certified and verified Docker images along with a dashboard, analytics and support.

The new cloud services have been optimized for blockchains located in the cloud network, thanks to a verifiable operating environment with comprehensive registration data that complies with technical expertise and regulatory requirements. Hacking-resistant cipher key storage and full security of the cryptographic module allow detecting and responding to unauthorized physical access attempts. In addition, IBM Cloud services allow blockchain partners to operate in a secure environment that prevents information leaks from shared storage devices and hardware.

IBM Blockchain for Hyperledger is now available on DockerHub

For organizations that need the flexibility to run blockchain environments on different cloud servers or devices, a signed and certified distribution of IBM code for the DockerHub Hyperledger project has become available. Current updates will provide new features, such as dashboards, analytic services, online chat support and exclusive networking services.

IBM Blockchain services for the Bluemix platform offer developers more flexibility

For application developers who want to instantly launch and run the blockchain environment and start building applications, IBM Blockchain beta services on the Bluemix platform will provide access to the latest version of Linux Hyperledger code with current updates. Regardless of the platform, DockerHub or IBM Cloud, IBM Blockchain supports many usage scenarios for a wide variety of tasks and industries.
