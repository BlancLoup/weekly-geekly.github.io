Created a program for web conferencing Zoho Meeting

The Zoho Suite web-office software suite now includes the Zoho Meeting web conferencing application. And let the experts say that similar programs on the Internet are a dime a dozen (NetMeeting, Adobe Acrobat Connect or WebEx), but Zoho Suite has expanded its functionality and can now manage remote PCs.



Like other similar programs, Zoho Meeting allows you to schedule meetings, stream a picture from the desktop to all computers, remotely control computer keyboards and mice, exchange instant messages, communicate in voice (VoIP will appear soon), etc. Conference participants do not need to install any programs: they can work under any operating system where there is ActiveX, Java or Flash is a bit slow during the conference. However, the conference organizer must be under Windows to install a 732 KB ActiveX component. The entire conference process can be recorded and saved to an AVI video file, so this file can be distributed to all participants immediately after the conference ends or embedded in the slide of the Zoho Show presentation.

According to the developers , the Zoho Meeting is fundamentally different from other Zoho office applications: this is their first program to broadcast in real time. In this regard, the beta version decided not to lay out in public access, and opened first to a limited circle of people.
