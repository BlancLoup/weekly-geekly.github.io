Linux Mint 15 "Olivia"

The release of the most popular at the moment Linux distribution (according to distrowatch.com), based on Ubuntu 13.04, took place.

Linux Mint 15 comes by default with MATE and Cinnamon GUIs, which are GNOME 2 and GNOME Shell, respectively.
There are also versions of the distribution with KDE and Xfce, but they are usually released a little later.


What's new:

  • kernel 3.8.0
  • gcc 4.7.3
  • glibc 2.17
  • mesalib 9.1
  • python 2.7.4
  • python3 3.3.1
  • upstart 1.8
  • xserver-xorg 7.7

  • Cinnamon 1.8
  • MATE 1.6
  • MDM 1.2
  • GIMP 2.8.4
  • LibreOffice 4.0
  • Firefox 20.0
  • Thunderbird 17.0

So, as the distribution is based on Ubuntu 13.04, then the support, respectively, will last until January 2014.

MATE 1.6

  • MateConf configuration management system replaced by GSettings;
  • In place of MateCorba and MateComponent came Dbus;
  • Added Systemd-logind support for user session management;
  • Changes in Caja file manager: improved sidebar implementation, added support for new thumbnail generation specification, added a new area for displaying thumbnails, and more;
  • The Marco window manager has an option to open new windows in the center of the screen;
  • Additional features in the Settings Management Center: added options to enable the composite mode in the Marco window manager and to activate fast switching between windows by Atl + Tab;
  • In the list of applications, by default, you can select a terminal emulator;
  • A backend has been added to the Atril document viewer for working with the XPS format;
  • An option has been added to the daemon coordinating the output of notifications to specify the screen on which notifications should be displayed;
  • Added new themes and icons.









Cinnamon 1.8

  • Presents the concept of descripts - desktop -integrated gadgets;
  • The environment includes three basic desklet: launch programs, clock and photo frame. Additionally, desks are available for download to display the xkcd comic book and ‚Äúbinary clock‚Äù ;
  • The features of the Nemo file manager have been expanded, in which such features as the toolbar and menu, two-pane mode, a menu with links for quick transition, a compact form of displaying a list of files and a side panel are saved;
  • The structure includes its own implementation of the screen saver, a feature of which is the ability to leave a message, which will be shown during the screen lock;
  • All available configuration modules are now consolidated in a single configuration center - Cinnamon Settings;
  • For add-on developers, a new API is presented for working with settings of descriptions and applets;
  • Instead of the window manager Mutter, his Muffin fork is involved;
  • In the Muffin window manager, checks for the rendering mechanism used are added;
  • Support for desktop design templates that allow the user to choose the most optimal form for the arrangement of elements.
