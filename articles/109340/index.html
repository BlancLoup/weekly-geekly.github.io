5. Stream.FM - monitoring regional radio
PotokFM is a hardware and software solution for monitoring. Broadcasting of all FM radio stations in the largest cities of the country.
Basic features:
a) Automatically monitor discrepancies on the air of one radio station in each region for compliance with the official regional grid. Consumers radio station owners, program and regional directors. As well as state structures.
b) Search for sound patterns (commercials, musical compositions) to analyze the statistics of rotations. Consumers are large advertisers, advertising agencies, and rightholders of all stripes.
Lyrics: I propose to work with the radio as a media space that can be recorded and stored, which can be freely searched for, which can be studied and analyzed.
potok.fm

A good example of a useful and well-monetized idea: radio stations need a project, musicians do the same, and listeners will not be left behind. True, the project can hardly be called a startup, there are competitors, but the value is that the idea is already working and making money.
