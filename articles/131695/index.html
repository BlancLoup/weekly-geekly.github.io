Afisha-Mir has updated the application for iOS

An updated version of the Poster World app for the iPhone and iPad is already available in the AppStore.

During the update, navigation was improved, layout was put in order, and several convenient features were added - everything to travel and share impressions with the help of Poster-Mir was simpler and more pleasant.



What specifically has been improved:

- Now you can add tips to cities directly from the application. Previously, this could be done only on the site mir.travel. For example, that the best bike rental in Berlin is on Alexander Platz, you can write as soon as you are convinced of this.

- It became easier to update the loaded guides: as soon as new places, tips or photos appear on the mir.travel website, you can add them to the application using the ‚ÄúUpdate guide‚Äù button.

- The list of nearby places can now be updated forcibly - by pulling it down, as in other iOS applications.

- From any offline guidebook maps of all cities loaded into the application are now available. For example, if the user has guides to Paris and the surrounding area - offline maps of all these cities will be available to him as a unit from any guidebook.

- In the tinctures, you can now enable the function that prohibits downloading the offline guide via the mobile Internet, so that downloading a 50 MB file will be possible only via Wi-Fi, but access to the rest of the application functionality will remain.

In addition to the version of the application for the iPhone and iPad , Poster-Mir has an application for Android devices.
