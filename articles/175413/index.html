2GIS for Windows Phone (real)

The list of platforms on which there are 2GIS, has replenished with one more. A free directory of organizations with a city map is now on Windows Phone.




The application is already in the Windows Phone store and is available for download.


2GIS for WP is:
- access to more than 200 cities of Russia, Ukraine, Kazakhstan and Italy;
- beautiful and current maps with 3D-models of attractions;
- the most complete accurate directory of organizations;
- support for both WP7 and WP8.

The main feature of the application is that it works on API 2GIS , which means that you will need an internet connection to work. Yes, this is not entirely consistent with the traditions of 2GIS, because before that all of our applications ( PC , mobile versions ) were completely offline.

As you know, the WP7 platform has a specific limitation - there is no possibility to create applications on native code. For us, this meant the impossibility of using the accumulated baggage of debugged libraries. Leaving everything from scratch was not a panacea either - the means available for development and the required labor costs did not guarantee the desired performance. Thus, an online application on API 2GIS was the solution. It meets our performance requirements and allows you to access maps and reference books for phone owners on both WP8 and WP7.

A little about API 2GIS . In fact, in all products that work on our API (for example: Flamp , NGS , Rugion ) the most up-to-date and up-to-date information, as the data is updated daily.

Anticipating the question whether the offline version of 2GIS will be released for WP. Yes, it will, but only for WP8. And it is in it that a number of functions will appear that are absent in the current version. When? There is no exact date yet. Follow the news, we will let you know when the details become known.

Install the application from the Windows Phone store and do not get lost!
