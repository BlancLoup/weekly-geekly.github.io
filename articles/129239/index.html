Allow me to introduce. Motivate Clock is one of the most young and stylish programs for keeping track of your working time.

My name is Valery Bashtovoy. I am engaged in programming and developing all sorts of useful things. I recently woke up with the thought that, like thousands of my other colleagues, I still cannot understand where working hours go from the moment we sit at a computer with a cup of aromatic strong coffee and finish the moment when we start looking for drops for eye and falling asleep above the keyboard, write the code nose.

That's when I first thought about how effectively I spend time behind the monitor screen? Is it possible to perform production tasks faster? Where exactly does my work day go? The result of this thought was the program Mobile Clock.




Then she had little to do with the software package, which is now available for download and installation on your local machines. But, nevertheless, it helped me to figure out what exactly was spent next day stupidly and motivated me to work more efficiently by reducing my time for rest. When, as a result of using the program, I managed to hand over a project to one of the customers four days ahead of schedule, I decided that it was time to put this solution in public. The application is absolutely free. I hope it will be useful for you.

To date, you can work with the program in the modes of non-design and project work.



Non-design mode. This mode is basic, in it the program works immediately after installation. Allows you to count your work and rest time using manual switching.



Project Mode In this mode, you yourself create individual projects in which you specify the entire list of necessary applications for the work. After that, the program automatically, up to a second, calculates how much time, as part of the project, was spent on rest and work.



As a result, you get full and detailed statistics of the time spent at the computer. The resulting data can be immediately exported to a PDF file and analyzed for overall efficiency.

I hope that the programmer, coder, manager, designer, copywriter, translator, photographer, engineer. Yes, in general, anyone whose work is related to hourly rates, will not regret that Motivate Clock has installed itself. And how can you regret when, in an incomprehensible way, you start to work more productively and earn more? :)
