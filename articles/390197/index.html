Characteristics of the Project Ara modular phone hit the Web



Despite the fact that the work on the Project Ara modular phone has been going on for a long time , little is still known about this project. Yes, we know that there is a basic platform to which module elements are attached, such as a display, a camera, and other components. But what are the real possibilities of the gadget?

About this little was known until now. Someone was able to check the characteristics of Project Ara using a benchmark, and posted the results on the Web. Of course, there is a possibility that all this is fake, but let's try to believe this information. So what have we got inside?



Yes, the fact that some device tested with the help of GFXBench long before the device appeared on the market can be surprising. But GFXBench had previously also included devices that had never entered the market.
