MeeGo 1.1

Yesterday it was announced the release of version 1.1 of the MeeGo platform.

It is argued that the new version provides a solid foundation for device suppliers and developers, allowing you to start creating software for various categories of devices on Intel Atom and ARMv7 architectures.

This release includes:


  • Core OS 1.1 - a consolidated common base operating system for all UX;
  • Netbook UX 1.1 - a complete set of core applications for netbooks;
  • In-Vehicle Infotainment (IVI) UX 1.1 - includes an example of the IVI main screen, a Qt 4.7 taskbar and speech recognition;
  • Handset UX 1.1 ‚Äî Primary UX for voice calls, SMS messages, web browsing, playing music and videos, viewing photos and managing connections;
  • SDK 1.1 Beta - The SDK for the MeeGo Core OS and the supported UX will be released in the coming days before the MeeGo Summit in Dublin.

MeeGo Conference and MeeGo 1.2


You can learn more about the MeeGo project and its developers at the MeeGo Conference in Dublin on November 15-17. Information about the program and registration is available at conference2010.meego.com .

The release of MeeGo 1.2 is scheduled for April 2011 and will include a Handset UX with a full range of applications, as well as support for other usage models.
