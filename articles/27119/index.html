MediaCoder - Audio-Video recoder


MediaCoder is a versatile audio-video recoder with great capabilities. Audio formats supported are MP3, Vorbis, AAC, AAC +, AAC + v2, MusePack, Speex, AMR, WMA, RealAudio, FLAC, WavPack, Monkey Audio, OptimFrog, AAC Lossless, TTA, WAV / PCM, Waveform. Video Encoders - H.264, XviD, DivX, MPEG 1/2/4, Theora, Flash Video, Dirac, 3ivx, RealVideo, Windows Media Video; Containers - AVI, MPEG / VOB, Matroska, MP4, PMP, RealMedia, ASF, Quicktime, OGM, as well as CD, DVD, VCD, SVCD, CUESheet.



Supports transcoding using custom settings, resizing an image, cropping an image, changing frame rates, changing proportions, rotating an image, applying effects to an image, cropping by time, overlaying subtitles, supporting AviSynth , encoding video from Youtube.com , supporting presets for popular phones, music players, game consoles, iron players, PDAs, flv encoding and much more. There is an opportunity to create your own plugins.
It is possible to run on Linux and Mac OS.

There are also specialized versions: iPhone / iPod Touch Edition , PSP Edition , MP3 / MP4 Player Edition ,
3GP Mobile Phone Edition , MediaCoder Audio Edition
