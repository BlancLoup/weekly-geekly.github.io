Overview of Jabber clients under Windows

When transferring friends and acquaintances with icq to jabber, I ran into the problem: which client to transfer the user to? In the transition process, it turns out that there are quite a lot of customers, and by and large users are interested in the appearance of the client. Under the cat a brief overview of jabber clients under Windows.

Immediately, I‚Äôll note that it‚Äôs difficult for one person to evaluate and find all the pros and cons of customers, so if you have something to add, or you are using a customer not on the list, feel free to write in a comment.


Name: Coccinella
url: http://thecoccinella.org/
Installation file size: ~ 6MB
Pros:
  • Voice support
  • drawing board
  • plugin support

Minuses:
  • In windows 7 I have some problems with the encoding when writing a message, maybe in the Russian version of windows there are no such problems.



Name: jabbim
url: http://www.jabbim.com/
Installation file size: ~ 16Mb
Pros:
  • add-on support
  • There is something like a web disk, but it did not work
  • Conference Bookmarks

Minuses:
  • There are bookmarks for Czech conferences that cannot be deleted.
  • and some more unnecessary tabs




Name: pidgin
url: http://pidgin.im
Installation file size: ~ 14Mb
Pros:
  • he pulls the necessary files to check the spelling.
  • Multiprotocol

Minuses:
  • lack of service overview






Name: Spark
url: http://www.igniterealtime.org/projects/spark/index.jsp
Installation file size: ~ 14mb without JRE
Pros:
  • the ability to send messages to all contacts from the roster.
  • Task sheets
  • Notebook
  • Plugin support

Cons: the impossibility of the usual transition to offline, only exit from the account.



Name: Tkabber
url: http://tkabber.jabber.ru/
Installation file size: ~ 4mb windows installer
XEP supported: 46
Pros:
  • support for a large number of xmpp extensions
  • many settings
  • plugin support
  • latex formula support
  • Conference Bookmarks
  • metacontacts

Minuses:
  • unusual interface
  • many settings




Name: exodus
url: http://code.google.com/p/exodus/
Installation file size: ~ 3 mb
Pros:
  • spellchecking
  • plugin support

Cons: the lack of Russified interface



Name: Jajc
url: http://jajc.jrudevels.org/
Installation file size: ~ 2mb
Pros:
  • Plugin support
  • not demanding on resources

Minuses: offline messages are not inserted into the general chat window, but are displayed in a separate window in a box.

Name: Ya.Online

  • inability to connect transports
  • when using 2 or more accounts, trying to add a new contact to all roasters
