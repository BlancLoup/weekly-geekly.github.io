Php


  • Short syntax for anonymous functions - Internals actively discusses the possible implementation of short lambda. Previously , the syntax with a tilde was considered and rejected on the ballot: $x ~> $x + 1; and also ^($x) => $x + $y . Currently the following options are being considered: 

     fn(params) => expr function(params) => expr (params) ==> expr (params) => expr [](params) => expr

  • Include PCS in the kernel - Offer for PHP 7.3 not yet issued to the official RFC. PCS is a module that allows you to write extensions for PHP in PHP and C at the same time. Read more about PCS . Enabling this feature in the kernel will allow you to rewrite many PHP extensions. This, in turn, will greatly simplify the code, reduce the number of bugs, and increase the number of potential mentheners.
