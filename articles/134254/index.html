Search for torrents in Unity

This is a free translation of this text . The author of the original offers to experience his creation.



If you often have to search for torrents and you use Ubuntu, then dive under the cat ...

With this add-on, searching for torrents on Pirate Bay has become much easier: open Dash, select torrent search and enter a search phrase.

This is the result of the work of David Call√© , who is also the author of the add-on for searching by cities and a special calculator for Dash .

Features


Along with the search results, the file name, its size, type and number of distributing seeders are displayed.

In addition, when searching, you can select the type of file you are looking for:



When you click on the result, the browser should open, you can also drag it to the Torrent client:



Although now only Pirate Bay is used for searching, you can add other services.

Installing in Ubuntu 11.10


To install, run the following commands: 

sudo add-apt-repository ppa:scopes-packagers/ppa sudo apt-get update && sudo apt-get install unity-scope-piratebay python-lxml
