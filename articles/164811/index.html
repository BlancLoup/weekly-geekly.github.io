Free implementation of contactless computer management interface: Enable Viacam

After all, it's great when the future comes, in small steps it overtakes us.
For me, one such step was getting to know the wonderful open source project Enable Viacam.
This project is a complete implementation of the contactless computer control interface using head movements.

The program supports emulation of mouse clicks: left, middle, right; control methods: double tap, drag-and-drop. The selection of keystrokes is implemented by an additional panel that takes up space on the screen, although it would be more logical to realize pressing through blinking with the right / left / or two eyes.

There is an implementation for both Windows and GNU / Linux.
