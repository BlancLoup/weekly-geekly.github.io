3D print support added in Windows 8.1


Yesterday at the Build 2013 conference they announced support for 3D printing in the Windows 8.1 operating system. Developed an API that programmers can use in their applications. In fact, 3D printing now looks the same as ordinary text printing. Currently, MakerBot Replicator 2 3D printers and 3D Systems Cube are supported. The price of the latter is about $ 1,300, and both of these models will soon be sold in the Microsoft Store.
