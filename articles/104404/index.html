Windows Phone 7 Release Day



Today, September 16, as expected , the final versions of development tools have been released for the Windows Phone 7 platform.

So about everything in order:

  1. here is the news about the release of the Advertising Advertising SDK for Windows Phone 7 and the Ad Exchange platform mentioned in it;
  2. Information on the controls for Windows Phone 7 has been published on the MSDN website, among them: the long-awaited Pivot and Panorama elements , Tilt Effect (!), Bing Maps Silverlight, Web Browser ;
  3. A new set of tools for Silverlight for Windows Phone Toolkit has been released , which includes new GestureService / GestureListener controls, ContextMenu, DatePicker, TimePicker, ToggleSwitch, WrapPanel . All items are released open source under the MS-PL license. In addition, several demo applications are included with the toolkit;
  4. the official Silverlight website has published a video for iPhone developers who want to start developing software for WP7;
  5. and finally, the final tools of the Windows Phone 7 SDK (direct link) have been released, the details of this can be found in this announcement .
UPD: Windows Phone 7 Training Kit for Developers - RTM Refresh is available .
