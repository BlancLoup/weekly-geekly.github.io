The gadget is based on the Qualcomm MSM8260A chipset, which includes a dual-core ARM-processor with a clock speed of 1.5 GHz and the Krait architecture, Adreno 225 video chip, a gigabyte of RAM and 16 GB of main memory. There is no memory card slot, and 16 GB is by this measure quite a bit, so HTC has entered into an agreement with Dropbox, where each user of One devices is given a couple dozen free gigabytes.

The slot for the SIM format is micro, so before buying you should take care of cutting or changing the card from the operator beforehand, and it looks like the phone is already in the hands, but it will not work.

Naturally, there are all the necessary wireless communication modules. Wi-Fi supports standard n and streaming DLNA. Bluetooth version 4.0, which does not increase the speed of data transmission, but it reduces power consumption.
