Key.com on the iPhone keyboard turns into.ru

It can be used to accelerate the input of addresses in a browser, but how often do you visit sites in the .com zone? Where as it is more often it is necessary to type .ru handles.

We have already heard calls to RiP Dev to somehow correct this injustice, but we don‚Äôt need to do anything - everything has been done and done by Apple itself.
