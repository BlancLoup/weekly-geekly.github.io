How to "pick up" your E-NUM account


1. Click on the application icon and, while it is loading, contemplate the welcome screen:


2. When you first start you need to activate the application:

  • specify your e-mail in the service E-NUM
  • enter the activation code given to you upon registration with the service
  • Click "Login to Account"


If you forgot the activation code:
  • Click the "Forgot?" link below the entry field for this code.
  • You will be asked: Do you want to receive a code in an SMS message? Agree
  • as soon as the SMS message is received, the code will be inserted into this field automatically


2. If the e-mail and code are correct, the application will prompt you to set a password. If you do not want to install - just uncheck the "checkbox" and click the button.
