The second open test of the unofficial Fallout Online has begun.



The first habratopic-announcement of the event here .

Good news - under habrakat;)


Project site - FOnline.ru

The duration of the open test: from a week to three, depending on the circumstances.

Link to download client: fonline.ru/foclient.zip
+ Mirrors for downloading the game client.

Client archive size: 34 MB

You will need files from the original Fallout 2!



Archive includes:
- FOnline.exe (executable game file);
- FOnlineDx9.exe (+ d3dx9_38.dll, the same as the previous paragraph, only the rendering is done using Direct3D9, also this version is deprived of the still not fixed bug with incorrect folding of the client in full-screen mode);
- Mapper.exe (+ Mapper_ReadMe.txt, a program for creating maps for the game, laid out for review, there are already two maps map1, map2 in the archive);
- Manual_Ru.pdf (manual for the game, in Russian and English, preferably read before the game);
- Manual_En.doc (English manual, unfortunately this is the old version, with some important notes at the end of the document)
- New music tracks;
- Animation for children (some distributions of the game do not have them);
- Other files.

If, in the game, when you connect, you get the error "Invalid version of the network protocol.", Then look at this page, there will be a patch that you will need to download and unpack to the client folder.

Client installation:

Unpack the archive in any place convenient for you.
Run FOConfig.exe. In the program, set the paths to master.dat and critter.dat (or you can simply copy these files to the unpacked client folder), also view other settings (if you have questions about the purpose of this or that option, then look at the FOnline.cfg file there are brief descriptions of them).
Copy the contents of the folder "Fallout2 \ data \ sound \ music \" from the original Fallout2 to the folder "FOnline \ data \ sound \ music \" FOnline. If you do not, then the game will not be the original music.
You can also enter the address of the game server in the archive, enter the default address in the archive, but it can change, or add new ones, the current status of the server (s) can be found in this topic: (Server)
Run FOnline.exe or FOnlineDx9.exe.
You can read the rest in the game manuals, which will be located in the folder with the client.
If you have questions about installation and launch, then ask here:
Client Installation: Install and run the client.

To give the test some meaningfulness, the following game was invented: Hold the bases.

The essence of the game is as follows:


There are three bases Sierra, Mariposa, Cathedral. At each location, on the last level, there is a terminal with which you can subordinate the base to one of the organizations (BS or Enclave). In exchange, they will be every game day (the time in the game is increased 12 times), at 12.00, bring various objects (weapons, ammunition, armor, tools ...) to the building entrance, access to them can only be received by the player who contacted with the organization. The statistics (Pip-Boy -> Status -> Statistics) reflect the following parameters: The current owner of the database, The best owner of the database (most of all held the base), Affiliation of the base (which base of which organization belongs to), Greatest preference (to whom the players more than once subordinated base). I hope this game diversifies the process of hanging out in the game.

Some useful notes:

Regarding the resolution of the game window - it can have any shape within: width 640 - 1280, height 480 - 1024, maximum limits are set due to the fit of the minimum size of the cards under it.
At the start of the game you will be given a walkie-talkie, how to use it, you can read the manual.
Do not forget about the F1 key in the game, there you can see the statistics on the traffic used and the hot keys in the game.
Remember that all items remaining in the account manager, when leaving it, are completely removed from the game. Keep things in banks at Hubbologists.
Do not forget to replenish the replication account in time, so as not to be in a special location for non-payers.
Statistics of player achievements has been added to Pip-Boy.
Music is currently loaded entirely, which slows down the download and adds an extra 30-40 megabytes of RAM, so if several seconds or a shortage of RAM are critical for you, then turn off the music (only music, sounds can be left).
