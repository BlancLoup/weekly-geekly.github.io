PHP Digest number 113 - the latest news, materials and tools (July 16 - 30, 2017)



Fresh selection with links to news and materials. In release: PHP 7.2.0 Beta 1, fresh RFC from PHP Internals, materials on asynchronous PHP, video from conferences and meetings, and much more.
Enjoy reading!


News and Releases


  • PHP 7.2.0 Beta 1 - With the first beta release, the active development phase ends, which means that the list of new features in branch 7.2 can be considered final. The next beta is expected on August 3rd. In the meantime, you can try PHP 7.2 from a prepared Docker image .
  • PhpStorm 2017.2 - Improved integration with Composer and Docker, autostart tests, and more. Video review innovations.
  • OpenAPI Specification 3.0.0 - A release specification for describing an API, formerly known as Swagger.
  • silexphp / Pimple 3.2.0 - DI-container now with full support for PSR-11 .
  • Bolt 3.3.0 - Popular CMS on symfony components.

Php internals


  • RFC: Same Site Cookie - setcookie() and other functions for working with cookies are suggested to add support for the Same-site Cookie standard.
  • RFC: Raise warnings for json_encode () and json_decode () issues - When an error occurs during a call to json_encode()/json_decode() suggested to throw an error of class E_WARNING instead of using the function json_last_error() .
  • RFC: Short Closures - A short syntax is proposed for converting Callable to Closure: 

     $writeln = {Util\writeln}; // is a simplification for $writeln = Closure::fromCallable('Util\writeln'); $writeln = {$terminal->writeln}; // instead of $writeln = Closure::fromCallable([$terminal, 'writeln']);

  • RFC: Mixed typehint - It is proposed to add a mixed typehint: 

     function foo(mixed $arg): mixed { return $arg; }
