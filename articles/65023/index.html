Google Apps: no more than 50 users on the base package

Somehow completely unnoticed was the fact that Google canceled the earlier decision to lift the restriction on the number of users on the basic package . Now on the basic package you can create no more than 50 users, which is clearly written in the Google help .
Also the registration procedure itself has been converted. Now, by default, opens the page advertising the professional package , where there is no comparative table between the two types of packages.
When you try to register with the selected base package, the corresponding message is displayed:
The basic Google Apps package is limited to 50 user accounts. Interested in additional user accounts or other advanced features? Try Google Apps Premier Edition for free.

In the Google cache, the previous version of the registration page is still preserved, with a pivot table, which already speaks about the restriction ( link , screenshot ). It can be noted that the introduction of the restriction occurred at least 2 weeks ago, however, I did not manage to find any official news about this.

Of course, 50 users are usually more than enough, but still it is very unfortunate that they imposed such a restriction, which they themselves had previously abolished.
