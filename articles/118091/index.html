QooxDoo 1.4 - the next release of the JS framework


The other day a new release of the js framework for the development of web applications QooxDoo has been released . One of the new directions for the development of the framework is the development of web applications running in mobile browsers. The large size of the compiled js, the orientation of the framework to mouse events, and the layout features made working with the web application written by QooxDoo in mobile barsers almost impossible. In version 1.4, the framework developers begin experimenting with widgets, controls and themes focused on the features of mobile platforms.
