What do you expect from Google Chrome OS?

Here's what the subscribers of the Google Operating System blog are responding to:

Integration with Google account. I'm sure Google accounts will be used to login to Chrome OS. So that you can use your login and password to enter the system anywhere. I hope that frequently used files will be stored on a local disk and synchronized with Docs as Dropbox does.

It should work offline. There must be something like account caching for offline login when you are not connected to the network.

Instant download. What I need from Google Chrome OS is the immediate loading of the system. I want this OS so that I can open my laptop, the screen is lit and showed me the browser.

She must be quick. I just want to get rid of the endless waiting for program downloads. When I turn on my computer, half the time is spent opening a browser, media player, and chat programs. I do not want to wait forever to download all the back-end programs and services.

Built-in search. I like searching on a computer in Google Desktop - this is a good way to quickly access my files, emails and web history. It must be enabled on Chrome OS. There should also be an offline cache of everything I‚Äôve viewed so that I can access the web pages even when offline.

Synchronization. Synchronization is very important these days. Not only favorites, but also passwords and status messages so that you can log in to another machine in almost the same condition as you left the other. I would like to be able to start working on the document on my desktop machine, then take my laptop with me and go on a journey, being sure that when I open the program, I will be right at the place where I stopped.

Using applications without installation. Even though the memory is cheaper than ever, mobile phones, netbooks, etc. Only will benefit in the absence of installation special for each use of software. Sites should have the ability (and possibly a standard protocol) to interact with each other, using the browser as an environment (avoiding downloading and subsequent uploading of objects), the user should be able to choose the services to which he wants to subscribe (ultimately, choosing them online the store), the OS should be able to offer new and interesting services to the user, based on information collected about his behavior with his permission.

Run Windows programs. I want to run Windows programs, and at the same time games. I want Chrome OS to be able to do everything that a regular computer can do. Then I will throw out Windows and switch to (hopefully) free Chrome OS!

Do not forget about the iron! Support a large number of equipment. I am extremely interested in WiMax support for an ARM processor.

So, people want a fast operating system that instantly loads, does not use a lot of resources, saves your data online so that you can continue your work on another computer, and at the same time synchronize them, work offline, use applications without installing them, start Windows applications and is compatible with many combinations of equipment.

And what do you expect from Google Chrome OS?
