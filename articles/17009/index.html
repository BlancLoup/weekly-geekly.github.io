Ext 2.0 Final Release

In the past, numerous beta and candidates. Now we meet the final release of one of the most powerful JS frameworks for developing web applications - Ext 2.0 . What we have with the update:

New interface elements


- grouping datasets with the ability to display a summary view for each group
- horizontal scrolling tabs
- tree view of table elements


New Documentation:



The documentation section and the API description center have been completely redesigned. Published instructions for the transition from previous versions of Ext to a new platform. New set of demonstrations and examples . You can also get acquainted with the version of the web-desktop, developed by Ext .
