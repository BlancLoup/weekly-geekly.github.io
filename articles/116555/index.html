New features for expanding Evernote for Google Chrome

Copying fragments of web pages is one of the most popular uses for Evernote. Whenever you meet an interesting article on the Internet, a study, a recipe, interior design tips, a device review, or something else, just keep it in Evernote, and this information will stay with you forever.

Right mouse button for quick copy


Google Chrome users received a number of new items in the browser context menu, which appears when you right-click. With their help, copy the necessary information in Evernote is now possible in one step. Menu options will vary depending on what you want to do:
  • Copy this page : This feature allows you to save the entire page.
  • Save selection : In this case, only the selected area of ‚Äã‚Äãthe page will be copied, including text and images.
  • Copy image : Clicking on the right mouse button on the image will send the image to Evernote.



Each of these three options will instantly create a note in your default notebook. If you want to set labels or choose another notebook, you can still use the Evernote button on the browser toolbar. Context menu options are ideal if you want to quickly add information. Don't forget that you can always organize the copied content later if you want.
  • New note : This last item in the context menu displays the window of the new, reworked Evernote note editor. Just create your note in this editor, click "Save" and you're done. Discreet copying


Copying now takes place in the background, so you can upload necessary information to Evernote without being distracted from the main activity. After you have copied something, the corresponding message appears in the browser window. And that's all. You can even leave the current web page.

Tips


The extension for Chrome allows you to view all your notes, as well as any notes with content that you have ever copied from the current site. To view the notes, click on the green tab at the bottom of the pop-up window that appears by clicking on the Evernote icon. These notes are now displayed in a new browsing mode with tooltips, which greatly simplifies quick navigation through them. To view the desired note, click on it in the list, and it will open in a separate window.
