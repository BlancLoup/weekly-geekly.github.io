ALEAR project robots create their own language



Information about cyclops Myon first appeared in 2010. These robots were created by the efforts of developers at Humbolt University, Germany. The main goal set by humanoid robots is the creation of a synthetic language, the mastering of some lexical techniques of interest to scientists all over the world. Now the project (its name - ALEAR) has finally entered the active stage, and the robots, if one can put it this way, create their own language.

According to scientists, the best way to create a synthetic language is to use the experience of our ancestors (in the sense in which scientists understand this experience). Robots invent words that denote their own actions, denote objects around and all sorts of events. It all looks like a game in which a participant, a player, and an observer interact.
