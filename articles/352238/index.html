Workspace ONE Intelligence


Workspace ONE Intelligence is a new cloud service that collects and analyzes application, network, and endpoint data. It is integrated decision-making mechanism (Decision Engine). It assesses the performance of infrastructure components (devices, networks, etc.). Then, based on the analysis, Decision Engine provides practical recommendations to IT staff.

Decision Engine can automatically make decisions based on set IB rules and corporate policies (for example, restrict access to services and applications to individual users or groups), as well as scan for device vulnerabilities on Windows 10 and automatically apply the corresponding patches.

So IT can track security incidents and respond to malicious actions before they adversely affect application and device performance. For example, experts can detect events with a high security risk, control user access, etc.

Among the features of Workspace ONE Intelligence are the following:

  • Integrated Insights: collects information about the state of the digital workspace. This data shows which systems work properly and which do not. Then, on the basis of these data, recommendations are made for optimizing resources or improving the service. In addition, Integrated Insights allows you to assess the security of the environment and the quality of customer service in real time. Also, the tool can generate reports to analyze the current state of the infrastructure and visualize this information.
  • Insights-Driven Automation: helps users automatically eliminate violations in the entire digital space and prevent unauthorized access. Decision Engine gives IT the ability to optimize routine tasks as well as use presets to manage devices.

Workspace ONE Intelligence has already been tested in 48 schools in the Corona-Norco United District (Riverside, California). According to Brian Troudy, director of networking technology and infrastructure for the district, the integration of Workspace ONE Intelligence has helped automate security tasks and improve the interaction between the platform and its users (teachers, schoolchildren and administrators).

Workspace ONE Trust Network


According to Gartner, global spending on information security will reach 93 billion in 2018. More and more companies are investing in security solutions, but they are often supplied by different organizations. Track security using several tools (antivirus, HIPS systems, WAF, etc.) helps the VMware Workspace ONE Trust Network solution. It combines the protective tools Workspace ONE with the security products of partner companies. VMware has already been joined by Carbon Black, CrowdStrike, Cylance, Lookout, McAfee, Netskope and Symantec.

Representatives of one of the partner organizations - Carbon Black - said that the ONE Trust Network integrates with products such as Cb Defense (a platform for predictive analytics and protection against cyber threats), Cb Predictive Security Cloud (cloud analytical platform) and Next-Gen Antivirus (antivirus which monitors the attack vectors and controls the security of the company's business processes).

Tom Barsi, senior vice president of business development at Carbon Black, says that the combination of Cb Predictive Security Cloud and Workspace ONE platform will allow customers to more securely protect their applications and networks from cyber threats. For this, the product was supplied with MO systems that analyze the behavior of the infrastructure components. Users can track deviations from normal systems and anticipate attacks.

Workspace ONE AirLift


This solution coexists with SCCM (System Center Configuration Manager - a product for managing IT infrastructure) and allows you to manage the life cycle of Windows 10 devices. Sharing AirLift and SCCM speeds up and simplifies software implementation, error correction, support for users working with remote the terminal.

AirLift will be able to test all users of the platform in April 2018.

One Cloud Destination


The solution allows you to run virtual desktops in the cloud or on-premise. The function also ensures the security of all applications and automatically manages their life cycle, regardless of the OS and device used.

  • Simplified work with Mac: macOS and Workspace ONE users will have access to all applications, including virtual Windows applications.
  • Enhanced security features for O365 applications: By integrating the Microsoft Graph API and Workspace ONE, IT can take advantage of DLP tools and security monitoring tools that ‚Äúblock‚Äù access to O365 when threats occur.
