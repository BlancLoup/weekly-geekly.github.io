Launch of RadStudio XE2, Delphi XE2 and other Embarcadero products.



New versions of RadStudio, Delphi and other Embarcadero products have been released.

Notable changes:
  • 64-bit compiler for Windows (8 years after the first mention);
  • Cross-platform library of visual components "FireMonkey". As I understand it, it is based on the KsDev component library of our compatriot Evgeny Kryukov purchased this year ( http://www.ksdev.com/ ). Declared support for iOS devices and Mac OS X;
  • API for working with cloud services Windows Azure, Amazon;
  • FastReport as the main report generator (news on the site FastReport ).
