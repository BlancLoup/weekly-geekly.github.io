The two-day course in Kiev "Fundamentals of the architecture of modern servers"



The training center of the MUK group of companies invites you to attend a two-day course from May 27 to May 28, where it will tell you about the development of technologies in the modern server market and provide you with practical skills in working with equipment from all leading server manufacturers. The lesson will allow any employee of your company to master the skills of a specialist in presales server solutions in two days.

Under the cut the course program.


The course includes the following practical classes:


  • analysis of the main server nodes;
  • remote server management;
  • diagnostics of the server state;
  • use of virtualization on servers.


Course Schedule:


Day 1.


1. Introduction.

  • Server location in the enterprise corporate infrastructure
  • Major manufacturers of servers, the distinctive features of their products
  • Delivery options: tower, rackmount, superdense server modules, blade systems.


2. Processor, memory and system bus of modern servers

  • Basic architectures
  • Intel x64 architecture
  • AMD x64 architecture
  • Intel ia64 architecture
  • Sparc Architecture
  • Power Architecture


3. Storage devices

  • SCSI and SAS adapters
  • Hard Drives and Logical Partitions
  • Expansion of disk space
  • Storage Basics


4. I / O controllers

  • Ethernet
  • FiberChannel adapters
  • InfiniBand adapters
  • Integrated Adapters (CNA)


5. Optional equipment

  • Out-of-band server management (iLo, MP etc)
  • Computational Accelerators


Day 2


6. Installation of equipment

  • Mounting cabinets
  • Supply system
  • Work with cable systems


7. Server Management

  • Server Management Packs (for example HP, IBM, Dell, Oracle and Fujitsu)
  • Unified infrastructure management system
  • Service support


8. Collection and analysis of information

  • Methods of collecting primary information
  • Personal display programs
  • Infrastructure management software (for example HP, IBM, Dell, Oracle and Fujitsu)
  • Finding bottlenecks and faults


9. Basics of Virtualization

  • VMWare vSphere
  • Microsoft Hyper-V
  • Redhat virtualization


10. Basics of backup and recovery

  • Backup Methods
  • Equipment for backup systems
  • Backup Management Packages
