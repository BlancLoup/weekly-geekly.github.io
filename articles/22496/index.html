Samsung copied iPhone

Samsung Electronics has released a new touchscreen phone Anycall Haptic in Korea (SCH-W420, SPH-W4200). This is almost one hundred percent clone of the iPhone with a similar design and the same functionality.
The device uses a new interface, thanks to which the user literally feels the device when pressed. For example, when you turn up the volume, the phone makes a clicking sound and vibrates as if you were really turning the wheel. Of course, all panels are allowed to customize as you like. There is even the ability to create your own menus using widgets (Widget).
In addition to the beautiful 3.2-inch screen (16: 9 aspect ratio), on which full web pages are placed, there is support for T-DMB, a 2 megapixel camera, Bluetooth 2.0, a player and a set of useful applications. The recommended retail price of the device is 700-800 thousand won (about $ 700-800).
