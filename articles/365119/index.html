Puzzlebox Orbit: opensource copter controlled by thought power



Multi-copter Puzzlebox Orbit provides control without a remote control. Instead, an EEG is used ‚Äî a helmet that captures the electrical impulses of the brain that can be removed from the scalp. These pulses can be used as signals to control various devices ‚Äî you only need to practice a little. Concentration on any thought (for example, turn to the right) allows you to realize a mental command when controlling the driver.

Similar projects already exist, but Puzzlebox Orbit differs from the others in that the developers have posted on their website software source code, drawings, 3D models and step-by-step instructions for creating a similar copter. According to the creators of the project, they will welcome the appearance of improved models of EEG-controlled copters.



The kit includes the copter itself, a control helmet, an instruction manual, and a pair of accessories.

To get started, you must install the appropriate software, and try to control the device. As mentioned above, for this you need to practice. To manage, you need to open the application itself, select the point of displacement of the copter, and concentrate. If everything goes well, the copter will start working, moving to the desired point.

The device itself is not very cheap - its price is about $ 200. But the developers propose to create their own device for everyone, a DIY project based on the Arduino. Instructions for creating a kopter is here .
