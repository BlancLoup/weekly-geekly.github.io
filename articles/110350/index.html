Microdata validator - new tool in Yandex.Webmaster

This year, a lot of launches and innovations for webmasters were devoted to the site content and various ways to emphasize this content in Yandex search results or in the corresponding search services (Yandex.Auto, Yandex.Real Estate, etc.). Data on the contents of the site can be transferred in various ways. One way is to markup the HTML code of a page with microformats.

In order to make it easier for site owners to work with the layout of pages with microformats, Yandex.Webmaster today has a new tool - the microdata validator . Using it, you can check how our search robot sees and processes the semantic markup of the page you have proposed, and also find out if there is an error in the markup code that may prevent correct data processing.

Now we support four types of microformats:
  • hCard - a format of a marking of the contact information (addresses, phones, etc.);
  • hRecipe - a format for describing recipes;
  • hReview - layout markup reviews;
  • hProduct - the layout of goods; Now used in the affiliate program "Reviews about cars" for marking information about cars.
We also support microdata - an international standard for semantic markup, which allows using attributes to describe the meaning of the information contained in the HTML code of pages. Such attributes allow robotic handlers to find and retrieve the necessary data. Currently, in Yandex, microdata is used to mark up vocabulary and encyclopedic articles, terms and definitions.
