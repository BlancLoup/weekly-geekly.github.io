AWS: RDS appeared in the Free Tier package

Hello!

Many could have missed the fact that from October 1, RDS t1.micro instances appeared in the Free Tier package on Amazon Web Services!

Now you can run MySQL, Oracle BYOL or SQL Server Express Edition for 750 hours per month absolutely free ! Also in RDS Free Tier includes:
  1. 20 GB of storage for the database
  2. 10 million I / O operations
  3. 20 GB of storage for database and backup copies
