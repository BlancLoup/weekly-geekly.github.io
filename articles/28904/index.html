Meebone is an AIR application for the Meebo chat client.



Finally it happened!

The first beta version of the AIR application for Meebo - Meebone has been released . The excellent chat client used to be accessible only from the browser, now it can be launched as a separate desktop application in any OS that supports Adobe AIR - i.e. Windows / Mac / Linux.

The version is quite stable and does not cause any special complaints. Saved all the functionality of the Meebo client - chat, voice and video communication, built-in games, etc.
