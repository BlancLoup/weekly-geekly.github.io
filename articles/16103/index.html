15 ways to create screenshots of the website

I translated here an article about creating screenshots. And not just screenshots, namely screenshots of websites. I haven't come across a Russian translation yet, so maybe someone will come in handy ...

Making screenshots of websites is probably one of the common tasks when using the Internet. Most likely, if you have a blog or you work as a browser in an online publication, you have to do it more often. In any case, there are some interesting tools for simple screenshots - some standalone, and some in the form of add-ons to the browser. Look at the alternatives to the Print Screen button. Immediately I warn you - English-language programs, but it's not difficult to deal with them.

Browser Add-ons

Save as image - exactly named (save as an image) Firefox plugin that allows you to save a page, frame, or any part of it as an image directly from Firefox.

Pearl Crescent Page Saver - saves images of web pages in png or jpg format; save the whole page or only a part of it, with the ability to automatically reduce the size of the image.

Picnik for Firefox - if you use Picnik online image editor (this is an add-on to the Firefox program), you can easily import into it a screenshot of the currently open web page.

Snissa is a simple Firefox add-on for creating webpage screenshots.

Screengrab! - addition to the browser, which allows you to save the visible part of the window, the entire page as a whole, only the selected area, or a specific frame of the web page.

ieSnapshotter - (shareware license) is an addition to the Internet Explorer browser that allows you to take screenshots and thumbnails of web pages.


Standalone programs

SnagIt is definitely one of the best screen capture software with all the options you might need. Integrating with Firefox, it can replace the standard Windows print screen function (print screen), and allows you to create profiles for various types of use. Paid - 39.95 dollars, but you can try the free version.

FastStone Capture is a powerful utility for taking screenshots, with a floating control panel. Allows you to save anything on the screen, including windows, objects, the entire screen, rectangular areas, and even areas of arbitrarily specified shape.

WebShot is a program that allows you to create screenshots and thumbnails of web pages or entire websites. Its main advantage is its own powerful command line processor, which allows you to automate the process of creating screenshots.

Paparazzi! - A tiny application for Mac OS X, again for creating screenshots.

PrintKey - replaces the standard screen printing function with an improved version with many features. This is a commercial version, but you can find earlier, unsupported free versions at this link.

I4X Screencatcher - take screenshots of the active window, the entire screen, or a selected area of ‚Äã‚Äãthe screen with this tool.

InstantShot is an enhanced free application for Mac OS X. It is located in the menu bar and gives you a myriad of options for creating a screenshot, including area selection, keyboard shortcuts, multiple screen image saving formats, image scaling and much more.

Web applications

WebShotsPro - enter the address (URL) and get a screenshot of the website of interest. Easier you can imagine. However, there may be more people willing to take a screenshot than the server can handle. So if you have already forgotten what it is to stand in line - remember :-)

Thumbalizr - allows you to create screenshots of websites, inserting their URL; the only problem is that it does not always work.
