February 28: About the main thing in corporate IT management.


Speaker : Dr.S. Zelenkov Yuri Alexandrovich, an experienced teacher, consultant and practitioner with great experience.

- how the organization creates value based on information;

- how to formulate requirements for corporate systems;
- how to create an IT service capable of implementing these requirements.
