KNetworkManager 0.7 and Kubuntu 8.04

In KNetworkManager , which comes bundled with 8.04, I didn‚Äôt like the fact that it was impossible to set fixed IP settings when connecting to wireless networks. Once we were going to play on pairs, but I could not connect to the ad-hoc network using KNetworkManager and assign a fixed address. Either this or that.
But it is possible to upgrade to KNetworkManager 0.7 , which comes with Kubuntu 8.10, without installing Kubuntu 8.10 .

With the help of your favorite text editor, add to the file
/etc/apt/sources.list
following repositories: 
 deb http://ppa.launchpad.net/network-manager/ubuntu hardy main <br>
 deb-src http://ppa.launchpad.net/network-manager/ubuntu hardy main

And updated:
sudo apt-get update

AND
sudo apt-get upgrade


Source of knowledge.

PS
I needed a reboot to make it work.
