Statistics on the use of databases, servers, Java and PHP versions in the cloud

Hello! By the numerous requests of our users and readers, we post statistics on the use of databases, application servers, as well as Java and PHP versions in the Jelastic cloud . The data was collected in a fairly simple way: we simply calculated the number of our users who used a particular software stack when creating environments on the Jelastic platform for the past month (all test accounts, of course, were excluded). Also in this article, you can track trends over the past year for databases and everything related to Java, and over the past 5 months for application servers and PHP versions (since PHP support was implemented at the end of January of this year).

Database

As you can see, the winner among the databases is quite predictable - this is MySQL (65%). With a large margin, MySQL's sister, MariaDB, takes second place, which is used by 12% of our users. The top three are closed by Postgres and Mongo - by 11% each. Well, and the outsider, our competition - CouchDB, which scored only one percent.

Java application north

Tomcat 7 leads among Java servers with a very significant gap from the rest - it has as much as 73%. It is followed by its predecessor Tomcat 6, at which 12%. Next come GlassFish and Jetty, in which 10% and 4%, respectively. TomEE is behind all this month (1%), but still may change in the near future, as this is new to our platform.
