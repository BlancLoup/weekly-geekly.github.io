Today ends support for Windows XP SP2

Today, July 13, Microsoft officially ends support for the Windows XP operating system with Service Pack 2 . This means that security fixes for this operating system will no longer be released.

All customers who use Windows XP SP2 are invited to install Service Pack 3 , which will be officially supported until April 8, 2014.

The exception is made by clients with 64-bit Windows XP, for which SP2 is the latest service pack. Their support will be carried out as well as systems with Service Pack 3. However, according to the company , such customers are less than 1%.

Details on the update and support can be found on the official page .
