Control your website with the new monitoring service from REG.RU



On August 1, REG.RU launched an open beta test of its new service, Site Monitoring, which allows you to receive data on the health of key functions of the user's site. Testing will take place within a month and will be free for all REG.RU clients. In addition, upon completion of the beta test, an additional free period of work with the service will be available to all participants.

What is the "Monitoring of the site"?

The service is a monitor launch scheduler (site / server monitoring script), each of which monitors a specific process of an Internet resource, which allows the administrator to react in time to problems occurring in the site and promptly troubleshoot problems.
Monitor operation parameters can be set both manually and using ready-made templates.
