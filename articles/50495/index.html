SharpDevelop 3.0 Release Candidate 1

01/22/2009 Release Candidate 1 of version 3.0 of the SharpDevelop development environment was released. About the environment itself can be read in my previous topic . About changes in version 3 of the dry changelog language can be found here . Well, now I want to tell from myself. The first thing that pleased me was the support for .NET Framework 3.5 (support for .NET Framework 3.0 was still in the second version of the environment). It should also be noted the addition of support for a new language - F # . I cannot tell you how interesting the feature is, because I never wrote it in this language. Added support for IronPyton 2.0 and support for projects on IL (Intermediate Language - CLR assembler), and also removed projects for Mono.

The interface has changed a little - the corners have become smoothed, and the controllers are more voluminous, in general, the overall appearance has become more pleasant, despite the absence of fundamental changes. The only downside to the minuses is the lack of a form designer for WPF.
PS Requires .NET Framework 3.5 SP1 for installation
PPS The properties window of objects has changed for the better, the description of properties has been translated into Russian
