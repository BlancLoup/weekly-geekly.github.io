If you do not want to select the code and paste it into the bookmark, you can drag the specially prepared link to the bookmarks bar. Since it is impossible to place it here, I had to create a separate page. In short, the link with the code is on this link .
When clicking on a bookmark-bookmark, buttons will appear, the meaning of which is as follows:
  • Move from the comment with the highest rating to others in descending order of rating (and vice versa).
  • From first by date to last (and vice versa).
  • From the most "controversial" to the least controversial (and vice versa).
    The more pros (cons) of a negative (positive) comment, the more controversial it is considered. With an equal value of ‚Äúrecessive‚Äù votes, a comment with a lower rating is considered more controversial.

You can also continue to click on the tab, sorting out comments from the best (according to the rating) to the worst.

Cross-browser compatibility


Naturally, Internet Explorer had objections to my idea - it is against too long links in bookmarks. You can agree with him, if you add only the code to the bookmark to load and insert the whole script into the page (from a third-party resource). These bookmarklets are usually made this way, but the script may suddenly disappear from a third-party resource or even go to the evil side ... It‚Äôs another thing when the code is entirely in your tab. In general, I just say hello to IE users.
Chrome, Safari, Firefox cope, in Opera I could not get the bookmarks bar to check.
