Reading books on mobile devices

This topic is very interesting. Lately, one has to read a lot and it is not always possible to buy or take a book in the library. On the computer, reading is not very convenient.

In this regard, the question arises. And do not start reading on something mobile?



And here immediately there are options:

1. Read on mobile
Option disappears almost immediately. The screen is usually too small - <2 ". As well as compatibility problems - you can hardly open the same djvu ...

2. Read on PDA
Not a bad option. In any case, the screen is already quite large (usually 3.5 "). Well, you can read absolutely everything. However, I, for example, have no desire to carry the PDA and mobile phone with me. I want everything to be in one.

3. Read on e-books
A huge screen, like a book (6 "). And it looks very similar to real paper. And the battery lasts a long time. It seems to be ideal. But only for people obsessed with books. For it is not cheap (even more expensive than the average PDA), functional no (you can only read and listen to music), well, the size, respectively, is not frail.
