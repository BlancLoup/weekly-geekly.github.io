Opera TV Store: a catalog of HTML5 TV applications



‚ÄúIf you are developing applications for smartphones or tablets, then Opera TV Store will be a big step forward for you,‚Äù - with these words, Opera announces its new project: a catalog of HTML5 applications from the Opera TV Store . His presentation took place yesterday at the CES conference in Las Vegas.

The site for developers has already posted a full-featured Opera TV emulator for application testing and documentation. The emulator supports all the features of the Opera Devices SDK:
  • HTML / CSS / JavaScript / DOM
  • SVG, Canvas
  • W3C Widgets
  • HTML5 <video> and <audio>
  • RAM simulation
  • Debugging in Opera Dragonfly
The idea behind the Opera TV Store is to provide access to HTML applications in an intuitive, user-friendly interface on a large HDTV screen. The interface is specially optimized for control from a standard D / U console (see video).



The application catalog is made on the Opera browser, applications can run in the background and issue notifications while watching TV programs (for example, about the arrival of new mail). Thus, Opera browser once again proves that it is the most cross-platform browser in the world. It has even been ported to the Nintendo Wii and Playstation 3 gaming consoles (via Cell Linux), and now Opera is also on the TV.

Now the Norwegian company is negotiating with OEM-manufacturers of TVs. They can be bribed by the technological excellence of the solution and the ability to run on the most minimal hardware, including the MIPS and ARM processors. The implementation of the Opera TV Store on the TV is done using the standard set of Opera Devices SDK development tools, with all its functions, including the ability to limit the maximum amount of allocated memory ( Out of Memory System ).
