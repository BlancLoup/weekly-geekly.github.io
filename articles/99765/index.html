Glippy - simple clipboard manager

Hello, dear habrazhiteli. Most recently, I switched to Ubuntu Linux and gradually get used to the abundance of good and useful software for this distribution. And today I would like to talk about a small and convenient program - Glippy.

Glippy is a simple clipboard manager for Gnome. It is very simple and convenient and that is why I chose it (or because I simply could not find another) .

So Glippy is hanging in the tray and watching the clipboard.



The user can select the desired copied fragment through the menu of the Glippy tray icon, or by pressing the key combination (Ctrl + Super + C).



In the program settings, you can add support for images, set the size of the clipboard, and also specify the key combination for the selection menu of the desired fragment.



Who likes this program - install from the repository:
sudo add-apt-repository ppa: bikooo / glippy
sudo apt-get update
sudo apt-get install glippy
