DIY Robot Vacuum Cleaner - Part 2

Since then, the robot has seriously changed. As promised, I want to share the revision.

1. Instead of a cardboard base, now a circle of 9mm plywood - this made it possible to fasten all the details of the robot well, using normal screws, rather than screeds.
