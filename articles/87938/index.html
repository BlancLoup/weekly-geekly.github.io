xbox 360 slim


A spy photo appeared on the network, which supposedly shows a motherboard for a new version of the popular Xbox 360 game console. If this image is true, the new modification can become the smallest game console in the Xbox lineup.

It should be taken into account that this image was obtained from China, and it is not possible to verify its authenticity. Nevertheless, some conclusions can be drawn when studying it. For example, judging by the above image, the CPU and GPU in this system are combined in a single chip.

Note that the current year already promises to be fruitful in terms of news from the Microsoft camp related to the gaming industry. For example, before the end of the year, a management system under the code name Project Natal, which we have repeatedly reported in the news, must be presented. In addition, the new mobile platform Windows Phone 7 Series will be compatible with games for Xbox, and even promise to create 3D games for this operating system. However, this November, the Xbox 360 console will be five years old, so it's time to think about updating.
