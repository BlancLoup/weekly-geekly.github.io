Source


Before the source code I will clarify some points:
  • N - so I mean the length of the input string. In our case, this should be a Brainfuck program.
  • The cell count starts at 0.
  • The code of the input program is placed in the cells starting from zero.
  • Cell (N + 4) - pointer to the current cell. The default value is 2 (for an input program, it is a cell with index 0). The value - 1 - for the input program is available as a cell with an index (-1).
  • The input program data begins with a cell (N + 10).
  • The data structure has the form: 1st cell - cell number, 2nd cell - value, 3rd and 4th cell - cells for storing intermediate results.

ASCII character codes used:
  • + - 43
  • - - 45
  • . - 46
  • < - 60
  • > - 62
