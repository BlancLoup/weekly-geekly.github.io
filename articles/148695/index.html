Warsow 1.0


After 7 years of development, the first major version of Warsow was released. Warsow is an atypical project with OpenSource with a number of features, and here's why:


Open-proprietary


Warsow has an open engine and proprietary resources. That is, the entire set of graphics, maps, belongs to the main development team. Perfectly balanced gameplay


The gameplay of Warsow is above all praise, it is a game from professional gamers for professional gamers. Yes, a game for zadrotov, with a long way from novice meat to invincible father. In version 1.0, a mandatory authorization finally appeared, and possibly a ranking of players. In any case, now it can be implemented quickly and easily.
The gameplay is very dynamic, it is one of the fastest FPS, very much like Quake III, but with speeds like in Quake I.

Design


Amazing for OpenSource products design, the game looks very stylish, everything has color coding, very contrasting locations and characters, excellent visibility.

Graphics


The game is developed on QFusion's own engine, based on Id tech 2. QFusion is heavily modified, and in many ways Id tech 4 fits, while still maintaining performance on old machines.

The game is absolutely free and cross-platform, feel free to load and try . If you have suggestions for developers or a desire to join the project, they are always open to dialogue, the project has an active community, the forum lives actively.
