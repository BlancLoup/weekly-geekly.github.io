Popcorn Time - update to version 0.3.3



September 17, 2014, Popcorn Time version 0.3.3 was released - open source software that provides the user with a simple and intuitive interface for streaming watching movies and TV shows using the BitTorrent protocol.
On Habr√©, Popcorn Time and attempts to close the project have been mentioned many times ( Popcorn Time: the most convenient program for viewing torrents , Popcorn Time developers disappear one by one , Popcorn Time closed due to "the threat of legal persecution and backstage fraud" Popcorn Time with GitHub ), but Popcorn Time is still alive and actively being developed. Who cares what new version 0.3.3 introduced - welcome under cat.

Major changes:
  • A large-scale cleanup of the code was carried out and the database format was changed. Because of this, it is possible to reset the database when updating. The developers promise to be more careful next time.
  • Auto-update is not supported in this release. They promise to fix it in the future.
  • The source code repository has been moved to its own server, now it is located at http://git.popcorntime.io/
  • Fixed flicker when launching the application.
  • Icons are now implemented using FontAwesome fonts (they used to be in PNG format).
  • Reconsidered settings, now they look simpler and clearer, some settings have been moved to the advanced settings section.
  • Added 3 new themes. In the future, there will be support for third-party themes.
  • New settings have been added: On top of other windows, Main page settings, Show movies on top of posters, Shade or hide watched movies.
  • Multiple user interface improvements.
  • Implemented scaling of posters on the fly (Ctrl + and Ctrl- in the mode of viewing posters).
  • Added the ability to go directly to the next unwatched episode of the series.
  • Built-in help.
  • Added the ability to select a third-party video player (VLC, XMBC, MPlayer, mpv and others) which will be used instead of the built-in one.
  • Added support for synchronization with Trakt.tv (allows you to synchronize viewed episodes, selected movies / TV shows, ratings, and more)
  • Began experimental support for Chromecast and Airplay
  • Added HTTP API to manage Popcorn Time from other applications.
