Monetization and promotion of applications for Windows Phone


On the monetization of applications told Michael Chernomordikov from Microsoft. Geo-location services


Sergey Rudenko finished the first day of the Nokia Developer Days conference with his report on geolocation services. In his speech, Sergey spoke about the current state of Nokia cards (coverage of 200 countries, 1.5 million cities and 25 million streets) and shared plans for the development of maps and impressive capabilities that application developers will get under Windows Phone 8. Sergey told about traffic jam calculation methodology and technologies for creating Nokia maps, the Nokia Maps API and the use of maps in web projects (Maps API for JS) and on various platforms used in Nokia smartphones - from Nokia Asha (Maps API for Java ME) to Nokia Belle (Qt Maps API).
