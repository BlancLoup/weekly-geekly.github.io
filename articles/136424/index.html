Simpliste: roll up sleeves


After my post last Thursday, there is no doubt that a simple and free template (someone even calls the framework) with an adaptive layout on HTML5 is of interest to potential users. The post even collected enough votes so that the UFO would allow me to invite someone to Habrahabr.

It should be recalled that Simpliste is just such a template. And now the work on its development will go at full speed.

One of the advantages of Simpliste I consider the availability of ready-made themes. Immediately, at the time of launch, there were 3 themes. Gradually, I created some more.


It is already clear that for users of the template so many topics will not be enough. Therefore, the open-sourced project Simpliste announces a set of participants who want to create themes.

I am sure that creating a topic or two is a great opportunity for designers and layout designers not only to do useful work, but to practice skills that may have been asking for attention for a long time.

Designers

Creating a minimal design is not as easy as it sounds. It requires both taste and professionalism. So when creating a theme, you can practice by spending not too much time on each individual design. Moreover, there is a layout of a standard theme , in which there are all the elements of the template. So it will only make their own changes. If you also write CSS, then the project can open up even more possibilities for you.

Design for mobile devices from you is not required, most often it is recreated without any extra effort.

Maker-ups

In essence, you don't need to typeset anything. There is an HTML page that is immutable and there is a standard theme CSS file that can be redone to recreate a new design.

Simpliste simplifies the life of the encoder by the fact that there is no need to chase the old versions of browsers. The main thing is to have access to information, and the design is created using the features of CSS3, and he has a lot of opportunities that we want to use.

For mobile devices, most often do not need to change anything. But check the behavior of the layout at different resolutions does not hurt.

To help participants

Specially created a page with the rules to be followed, there are the necessary links.

What does it give

  • Project open-sorsny. Someone your work can help
  • Practice existing skills
  • The practice of new skills, especially interesting features of new browsers.
  • Respect and respect
  • Potential opportunity to learn new colleagues, find customers
  • Opportunity to get on Habrahabr *
* For the last item. I have already mentioned that the last post brought an invite. I am ready to give it to the one who will bring the greatest benefit to the Simpliste project. But I hope that if among the active participants there are a lot of people unregistered at Habr√©, then there may still be those who want to share invites.

How do we do that

I suggest everyone who has a desire to contribute, leave in the form of comments or the fact of the desire to impose ready-made designs, or give screen shots of designs created by you, and those who can do the layout will contact the author of the design, not forgetting to start work by layout. If you want to create both design and code, then the link to the project on Github can be useful, or send topics directly to the post office.

Those who are not registered on Habrahabr, send to my mailbox either a screenshot of the design so that I can post it here, or a message with the intention to impose a particular design, with reference to a comment with a screenshot of the design.

Each author will be mentioned in the copyright file styles. The most active participants will receive thanks in the information about the project. I invite the authors of topics and layout designers to leave links to their portfolios , it is very possible that someone will be interested in your work.

What then?

  • Jack of all trades
  • The most original solution
If you have tips, ideas or the opportunity to test topics, I will be grateful for the help.
