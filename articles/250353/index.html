Webinars on new features RAD Studio XE7



Dear developers!

We are glad to inform you that we are launching a series of webinars for you.

Here is our program for the coming months:

February 18, 2015 - 12:00

New InterBase XE7 Features

During the webinar, we will look at the new Interbase XE7 features such as: performance improvements, incremental dump, transactions, performance counters, indices, 64-bit transaction identifier, Change views and others. Dmitry Kuzmenko, InterBase expert, will take part in the webinar.

March 4, 2015 - 12:00

What's new in FireDAC XE7

FireDAC, as the flagship data access technology of Embarcadero, is successfully evolving. This webinar is devoted to innovations in FireDAC XE7, in particular, new opportunities to work with BLOB values ‚Äã‚Äãand a new component of TFDBatchMove. Dmitry Arefyev, the main developer and ideologist of FireDAC, will take part in the webinar.

March 18, 2015 12:00

Building a user interface using the MultiView universal panel

During the webinar, we will look at various options for applying the new unique component TMultiView, which allows you to significantly reduce the time to design interfaces, thanks to its capabilities. Consider the behavior of the component on different platforms, the properties of the component, the use of the component in mobile applications.


April 1, 2015 12:00

Adaptation of existing VCL applications to taskbar features in Windows 7 and Windows 8

During the webinar, we will deal with the new TJumpList component for working with the list in the taskbar and using examples, consider the options for using it, and also show the features of using the TTaskBar component for working with the taskbar. Let's create some examples and discuss options for expanding the functionality of existing VCL applications using these components.
