3D printer the size of a room for $ 250

The invention of Ludwigsen, "Hangprinter", uses a system of cables attached to the walls, floor and ceiling and computer-controlled drives on a block with a printing head. The device uses the room itself as a 3D printer case.



Hangprinter is not the first wide-format 3D printing system, but it is unique. First, it is cheap. All hardware and firmware components can be purchased for about $ 250. It is also an open source project, which means that any user can improve the system and use third-party updates. Finally, Hangprinter is self-sufficient - most of the components needed to create it can be printed by him.

It all adds up to a system that Ludwigsen hopes will open up the potential of 3D printing to a large number of people. The developer believes that 3D printing should not be an expensive privilege. In his opinion, Hangprinter is not only a technical, but also a social experiment.
