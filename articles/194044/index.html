Do not miss! Today, September 17th. Webinar "Software Testing with Visual Studio and Windows Azure"



Dear software developers and testers! We invite you to the webinar , which will be held today on September 17 at 14-00 on the topic "Software Testing with Visual Studio and Windows Azure". The webinar will consist of two parts:

  • 14:00 - 15:00 Quality assurance of applications using Visual Studio tools, Alexander Yakovlev
  • 15:00 - 16:00 Effective software development and testing process in the cloud, Vladimir Yunev

Any organization or individual developer building applications needs a separate development and testing environment. Developers need pre-installed tools, and the test environment should be able to recreate the world in which the application should be deployed. For these purposes, the use of virtual machines is now widespread. As part of this webinar, we will tell you about how to get a ready-made infrastructure with the provision of virtual machines on demand for the purposes of development and testing in the Windows Azure cloud. In addition, you will learn about popular cloud usage scenarios for developing and testing software, using Windows Azure to test applications based on open source technologies, and about the great benefits you can get by using an MSDN subscription. Connect to the webinar and find out the mass of the new!

Join us today at 14-00 to our webinar on this page !

In the meantime, we invite you to the Windows Azure Development Center . Here you will find materials on how to get Windows Azure for free, materials on the development and development of cloud business.

See you at the webinar and in the Windows Azure community on Facebook !
