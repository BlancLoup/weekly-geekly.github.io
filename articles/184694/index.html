What is Wildcard SSL?

Wildcard SSL is a universal SSL certificate that allows you to send multiple hosts from one domain name to one server at a time. This eliminates the need to purchase individual SSL certificates for each of their subdomains. That is why Wildcard SSL is an ideal solution for sites of large organizations and commercial Internet projects.
