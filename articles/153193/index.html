Opera will be released on Windows Phone 8

Opera Mobile and Mini browsers were used by many smartphone owners even before the advent of mobile Safari, Chrome or IE. With the release of iOS and Android, the market position of mobile Opera, of course, has declined, but still the browser remains the number one Internet surfing program for many Android users and older systems, such as Symbian. The main strength of the company - Opera Mini browser, which works wonders with the consumption of traffic.

Opera Vadim Makeev recently confirmed that the company is interested in porting Opera Mobile and Opera Mini browsers to the Windows Phone 8 platform. This is because the WP8 SDK allows developers to use C code in their applications, which was not possible with Windows Phone 7. This allow Opera to use most of its C code, and with sufficient simplicity to transfer its products to Windows Phone 8 :

Unfortunately, we do not comment on the release dates. I can say one thing, as soon as it becomes possible to release Opera Mobile / Mini for Windows Phone (SDK in this case), we will do it right away. This is a full-fledged C SDK for development, for which we could use the existing universal code written for all platforms. There is no such SDK for the Windows Phone platform yet. As soon as it appears, we will port Opera Mobile / Mini.


On Windows Phone 8, the native browser is Internet Explorer 10, which everyone has been praising since WP7. How big the audience of Opera will be on a new platform is a good question, but somehow the function of data compression before transfer will always find fans.
