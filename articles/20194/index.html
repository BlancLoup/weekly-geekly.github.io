Firefox 3.0 Beta 3

The third beta version of Firefox 3 has been released.


Firefox is a reworked version of the browser included in the Mozilla Suite. By removing additional programs (email client, IRC client and HTML editor), the program has become much more compact. It should be noted that Firefox supports the most powerful system of plug-ins, thanks to which you can greatly extend the functionality of the browser by adding a bunch of features.

As a result: the highest quality of performance, support for all modern www-standards, user-friendly interface, stable operation, high speed - this is the list of characteristics due to which the browser has every right to be considered the best.
