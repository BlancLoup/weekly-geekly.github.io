FTP2SHARE.COM is a new file sharing standard for the network.

This review will be primarily useful to users who distribute any files on the network. It is no secret that there is a huge number of file hosting sites that allow you to upload your files to them, and publish links to them on websites or forums (the most famous are Rapidshare.com and MegaUpload.com ). But everyone probably knows the situation when your file is deleted from the hosting and the link to it is ‚Äúdead‚Äù, what to do? Just perezolit on the same hosting - there is a high probability of deleting the file again, upload to other hosts and make "mirrors" - the approach is correct, but this is your time and your traffic (which is not common for all).
FTP2SHARE.COM is a service that is created for you, mirroring your files to the most popular file hosting, the main principle is one upload = 5 links to your file.
Main features of ftp2share.com :
- File uploading via HTTP - FTP (FXP)
- Remote fill (specify the link - the service itself downloads the file and reloads)
- Mirror your file on 5 file hosting sites
- Support for working with folders
- Ability to upload files to your premium file hosting accounts
- Ability to upload only to the file hosting you specify
- A unique feature to re-fill your file if it is removed from file hosting
- Work with folders by ftp protocol
After a simplified and quick registration, you can immediately start using the service.
