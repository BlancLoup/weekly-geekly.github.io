In the second release candidate 10.62 , not a lot of changes have been made - the release 10.62 will be soon. Developers are asked to report any regressions from version 10.61. For those who are waiting for 10.70, a new interesting build is soon promised.

Note: in this thread, build numbers for different platforms are different. There is also a PowerPC build for now.
