YouTrack release 2017.1 and Hub 2017.1

Hi, Habr! Most recently, we have released a new version of the project management system - YouTrack 2017.1 and the JetBrains collaboration tools connector - Hub 2017.1 .

The hub is built into YouTrack, so many of the updates are available in both products. Therefore, we decided to write one post in which we will tell about two releases.



Let's start with YouTrack. What's new in YouTrack 2017.1?

  • Search based on time tracking.
  • The ability to attach files to cards on the Agile board.
  • Authorization in LDAP through a fixed account.
  • The ability to generate a permanent access token.
  • Improvements in managing user credentials.
  • Updated tab "Access".
  • The ability to add a reason for blocking users.
  • Improved OAuth 2.0 authentication modules.

Time Based Search


In YouTrack 2017.1, it became possible to search for tasks based on the type of work performed, the user, or the time interval. Now it is easier to search for tasks that you or your colleagues have worked on, and also to compile the necessary reports.

For example, to find the tasks you worked on last week, use the query: work author: me work date: {last week} .



The ability to attach files to cards on the Agile board


Now you can attach files to tasks directly on the board. Simply drag one or more files onto any of the cards. You can also download all files as a zip-archive, change their visibility for certain groups of users, attach private files and edit images in the card view mode on the board.



Authorization in LDAP through a fixed account


This option allows you to configure standard two-step LDAP authentication. Now you can use a special fixed account to authenticate and search for the necessary user information in LDAP.



The ability to generate a permanent access token


The new version also has the ability to use a constant access token, which simplifies the integration of YouTrack with third-party services and makes it more secure. Simply create a new token directly from your profile and use it for authentication in API calls.



Credential Management Improvements


We have added some improvements in managing user credentials. Now you can create additional credentials for yourself and combine them with an existing account. If you mistakenly delete your credentials, you can restore them directly from your profile. Also, if you accidentally created several accounts, you can now combine them into one.



Updated tab "Access"


In YouTrack 2017.1, we completely rethought the access control process. Now you can manage access rights and analyze issued rights, filter roles, projects and user groups in one window.



The ability to add a reason for blocking users


Now you can add the reason for blocking the user. This text is displayed next to the name in the list of users and is available to those who have read access to this field.



Advanced OAuth 2.0 authentication modules


In YouTrack 2017.1, we added the ability to log in using Facebook, Yandex, Microsoft Live ID, PayPal, Azure AD, Amazon, as well as any other services that support OAuth 2.0.



All of the above improvements and updates to the administrative part of YouTrack are also available in the latest version of Hub 2017.1 . Let us tell you about updates that concern only those who use the Hub as a separate service.

Quick navigation


In Hub 2017.1 appeared fast navigation, which allows you to find any categories and subsystems in seconds. Just press Ctrl + K or Cmd + K (Mac OS) to open the global navigation window, start typing the name of a category, group or any other entity where you would like to go - the Hub will tell you the answer options and immediately transfer you there.



Deleting an entity from the details page


Previously, to delete an entity, for example, a user group, you had to find it in the group table. Now the delete button is added directly to the group page. Similarly, the button is added to the pages of users, roles, authorization modules and other entities.



Scheduled Backup


In the new version of the Hub, you can set a schedule for automatic backup of data. Pre-select the configured time interval or set your own using cron expression, and the Hub will save backup copies of the data according to the specified schedule.



We also made a number of other improvements - see the full list of completed tasks included in Hub 2017.1 release. Please remember to back up your data before installing the new version.

Try Hub 2017.1 and YouTrack 2017.1 and share your impressions with us. If you are using the cloud YouTrack, then your server will be updated on a schedule .

If you have any questions during the installation or upgrade process, contact our technical support team , we will be happy to help.
