Microsoft has built Bing search engine in the context menu of Skype

Anyone who updates Skype to the latest version will see that the context menu of the Skype chat has been updated with the Bing Search item. This is an exact analogue of the old Chrome browser chips - search for selected text in Google through the context menu.



Obviously, in this way Microsoft decided to promote its not the most popular search engine with the help of a very popular messenger. Perhaps he will succeed - there is no possibility to change the search engine in the settings. And perhaps such changes - this is exactly what will bury the ingenious Scandinavian development once. This is a question, what is worse for the user - the lack of a search button or the inability to change the search engine. At one time, European officials forced Microsoft to make Internet Explorer removable - and then at least the users had a choice between different browsers. Maybe this time abroad will help us?
