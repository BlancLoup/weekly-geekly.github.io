3CX 15.5 SP1 Beta Release and Acquisition of Askozia

This week there were two important events that we are in a hurry to share with you. So, in order.


3CX v15.5 Update 1 Beta Release (SP1)


In the midst of summer vacations, we released the beta version of 3CX v15.5 Update 1 (now we call service packs), which added some interesting features for users and administrators.


Admin Features


Centralized configuration of advanced IP phone parameters


For supported IP phones the number of adjustable parameters has been significantly expanded. The administrator now does not need to understand the documentation and create your own auto-configuration template for a particular phone. All additional parameters are specified centrally from the 3CX control interface.


List of parameters:


  • ring tone selection
  • MWI setting of the phone power indicator LED - blinking when there is a voice message, a missed call, etc.
  • reduced brightness of the 3CX logo on the Yealink, Fanvil and Htek screensaver
  • individual time zone setting for each phone
  • setting the time display format: AM / PM, 12 hour or 24 hour
  • setting the default call transfer type (unaccompanied or followed)
  • enable or disable the backlight of the phone display
  • setting the screensaver on timeout of the display, which eliminates the flashing of logos in some models
  • configure the network settings VLAN IP phone
  • Setting BLF (DSS) buttons to Line or BLF. Previously, the first two buttons of most phones could only work in Line mode.

Simplified setup and maintenance


We tirelessly continue to make 3CX extremely easy to install, configure, and maintain:

  • updated all templates of SIP operators for easier and seamless connection of new trunks to the system
  • Now you can capture network traffic on the server directly from the 3CX management interface . To do this, the Wireshark utility (on Windows systems) should be installed on the 3CX server. You can either download the capture file for further study, or include it in the data package for transfer to 3CX technical support ( Create file for technical support option)
  • The built-in utility 3CX Firewall Checker can now determine whether SIL ALG is being used on your router. Be sure to disable this service, because it is he who most often leads to problems with the passage of sound
  • start backup and restore from the command line

Opportunities for users


The new web client 3CX received a number of useful improvements:

  • Now the user can dynamically monitor and manage calls directly from the main screen Employees (which was previously called People )
  • The user can customize their call forwarding rules (and exceptions to the rules) directly in the web client.
  • Incoming calls are displayed as Chrome desktop notifications, regardless of whether the web client interface is open or not
  • The user can add his close colleagues to Favorites (just mark with an asterisk), and they will be displayed in Favorites in all 3CX clients
  • Functions that were previously intended only for hotels are now available to all users:

    • can set reminder calls
    • initialize (clear personal data) extension numbers for new or temporary employees
    • change the status of other employees




See a brief overview of the new features of the 3CX web client in English.

We also added several new features that were recently requested on the forum :

  • Group Voice Mailboxes
  • Call Distribution Report
  • An important report on the average customer waiting time in the Call Queue and the average time after which the customer ‚Äúhung up‚Äù without waiting for the operator to answer
  • Maid codes for use in hotels
  • We will also add the convenient ability to copy extensions, i.e. create a new extension (user) based on an existing one. However, it is not yet implemented in Update 1.

Install Update


If you are using 3CX v15.5, Update 1 Beta is available to you as a regular system update, provided that you have agreed to test early versions of the product. Of course, such testing is undesirable to do on a working system. However, we are very interested in fully validating the new update, so we are waiting for your feedback!

You will also need to update the 3CX desktop clients:

  • 3CX Client for Windows
  • 3CX Mac Client

3CX acquires the well-known company Askozia to expand into the global market




The second interesting news is the acquisition by 3CX of the well-known German company Askozia, which produces the well-proven Asterisk-assembly AskoziaPBX. Together with the acquisition of the brand, 3CX gained more than 10,000 small client companies using AskoziaPBX worldwide, a team of experienced developers and their own know-how.

Askozia was founded in 2013 and has grown from a small student project into a serious player in the software PBX market. The 3CX manual was impressed with the success of this highly professional team. Now it is possible not only to strengthen the position of 3CX in the German market, but also to use Askozia‚Äôs own Linux telephony developments.

All current users and partners of Askozia will continue to receive the necessary support during the company consolidation period. Askozia users will be offered free licenses of 3CX, and Askozia partners will become 3CX partners.

Another advantage of buying Askozia is that 3CX has a permanent residence in Hanover, the city that hosts the Cebit exhibition and has offices of many famous technology companies. Of course, this simplifies communication with potential customers and partners.
