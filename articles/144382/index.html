Yandex launched a system for displaying facts and figures in search tips



The Yandex search engine has been showing prompts to users for a long time. This is quite a useful feature, which is implemented by almost all search services. Sometimes it is very convenient to enter only part of the request, and receive the entire request. The developers of Yandex decided to make search tips even more useful, and now you can see the numbers and facts you need in the search tips. In other words, the answer to some questions can be obtained even in the search box.

For example, if you need a code of a city, just type the corresponding request, and a hint will show the code. In the same way you can find out the geometric dimensions of known objects, the rates of some currencies and so on. The system of intent search, as it is called, is available for both conventional browsers and mobile, in the case of working with modern smartphones with touch-sensitive displays.

Answers in the search line are not given to all the questions, to the question ‚ÄúWhat is the meaning of life‚Äù, we still get a huge amount of reasoning in the main search results. But if you ask about the name of the capital of any state, the answer will be given immediately. The name of the writer can be found in the same way. In general, the answers are given to a question asked about a fact. The main thing - the question should not be too complicated, as in the above example with the meaning of life.

The developers believe that the new feature is useful to many users who can save a lot of time without entering the request completely, and not trying to find the answer among the entire issue. Especially this feature will be useful for smartphone users - for all of this is especially convenient.
