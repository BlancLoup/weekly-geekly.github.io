New versions of JBoss Tools and JBoss Developer Studio



This week, JBoss is releasing the next release of JBDS studio (3.0.0GA) and JBoss Tools plug-ins (3.1.0GA). New versions of products contain a number of bug fixes, as well as changes in functionality and the creation of a more user-friendly user interface, which is aimed at simplifying work with all supported technologies.

JBoss Tools and JBoss Developer Studio are designed to work with JBoss technologies such as JBoss AS, JBoss Seam, JBoss jBPM, Hibernate, JBoss Drools, JBoss ESB, RichFaces, JBoss Portal.

Both products are based on the Eclipse IDE and the Eclipse WTP project. This allows you to develop J2EE applications based on solutions provided by other implementers of the J2EE technology stack - Sun, Weblogic, etc.

JBoss Tools is a collection of Eclipse plug-ins available for free download. In turn, JBoss Developer Studio is a complete set of software necessary for developing and running J2EE applications.

Both products have full technical support and detailed updated documentation.

A short list of the changes made studio JBoss Developer Studio release 3.0.0GA

Contexts and Dependency Injection (CDI) - The Contexts and Dependency Injection Project Facet has been added to the JBoss Developer Studio, the Project menu includes new options (Add CDI Support and Remove CDI Support), etc.
ESB - changed ESB Content Based Router, added new functionality in ESB Configuration Editor, officially supports ESB 4.7, which is contained in JBoss SOA Platform 5.0, etc.
Hibernate - Refactoring support has been added to the Hibernate Code Generation Configurations, the functionality of the New Hibernate XML Mapping file wizard has been changed, the Hibernate Diagram Viewer, Mapping Diagram viewer, etc. has been updated.
JavaServer Faces (JSF) - improved performance of PageContextFactory, support for JSF 2.0 structural components has been made, work of the Eclipse OpenOn function, etc. has been fixed.
RichFaces - RichFaces libraries have been updated to version 3.3.2.SR1. and etc.
Seam - fixed build and validation of Seam projects using Preferences, Seam examples of Booking and DVDStore are updated to Seam 2.2 for the JBoss Enterprise Application Platform, etc.
Smooks - Smooks Reader page has been replaced by the Input page, changes have been made to its functionality, testing of Smooks projects has been simplified, etc.
