From design to development: 10 tools to help improve and optimize workflow



Only with the help of tools do not become an artisan. Sure you can say that the role of some applications can not be overestimated. This article will be mentioned 10 tools that can simplify the workflow, create a creative mood. I hope you will be able to endure for yourself something useful that optimizes your work. So, let's begin:

Atom


Atom is an open source text editor, developed by the Github team. It is also called open source editor of the twenty-first century. The editor is not difficult to use, while providing complete freedom for installations and extensions.



Free text editor for OS X, Linux, Windows with plugin support, which are written in Node.js, and embedded under Git Control. Due to cross-platform, it works well on all popular platforms. Functional interface, built-in plug-in manager, smart auto-completion with flexible settings, the ability to divide the workspace into several independent windows.



Among the advantages are addition, themeization, customization. The application was created using HTML, CSS, Javascipt, Node.js, based on a specially developed version of the Chromium browser.

You can get acquainted with the work of the editor in more detail, read detailed documentation on the site.

Dragdis




Designers - people of creativity, always in search of inspiration. DragDis tool is a muse in which to draw it. It is easy to use, the application is an online service for storing interesting pictures, videos, texts and links; You can add the content you like, just by selecting it on the page and dragging it to the panel that opens at the same time. The panel works thanks to the installed extension (Chrome, Firefox, Safari).

iTerm 2


iTerm is an alternative to the built-in OS X terminal, although you can call a couple of features: using the iTerm2 settings, you can assign various hot keys to access important functions, supports Expose, which allows you to quickly switch between open windows.



The advantage of this application is autocomplete, iTerm2 remembers all the commands that you enter and the next time you enter a command, just press ‚åò.

Muzli


Muzli is an extension for Chrome. Simply put, your home page, which will load news from resources such as Designer News, Dribbble, Behance.



This application serves as a personal assistant, a storehouse, daily loading a lot of interesting, useful and inspiring material for creative work.



Noiz.io




Many people for a creative work environment do not have enough "atmosphere". The Noizio app helps to create a creative atmosphere in your home. You can choose the perfect bekfon yourself, such as the sound of rain pounding on the roof of a house, the sound of a coffee house, a thunderstorm or sea waves.

Invision


Invision service for turning web and mobile design ideas into clickable and interactive prototypes and mockups. This is a platform for collaborative development of interface layouts.



As for prototyping, prototyping for applications and websites is a rather laborious process that requires rapid response of the entire team (developers, designers and testers), and with the InVision platform, you can get a powerful tool for developing a software product, while both the developers and the management of the company and even the customers of the finished product.



What the font


Perhaps the best service for identifying fonts from images. It's very simple: upload a picture with an example of a font, indicate whether the service recognized the letters correctly, then click on the big green button - a few of the most similar fonts are displayed, among which you can find a 100% match.



Postman


Postman - like Google Docs for API, the platform allows teams to track changes and work together, developing APIs without failures and misunderstandings. Convenient HTTP client for testing web sites, considered to be good extensions to the Chrome Web Store directory,



Postman can be purchased for free as an extension for Chrome. With it, you can send any type of HTTP requests (GET, HEAD, POST, PUT, etc.) with any number of parameters and headers.



There is a built-in query editor, with options for encoding requests, loading from a file and sending binary data.

Wappalyzer


If your work is related to web development, Wappalyzer is a necessary tool in this work, serves to analyze the technologies used by competing or popular web pages.



This plugin for Mozilla Firefox, which recognizes CMS, plugins, libraries used in the work of sites, popular open source engines (Drupal, Joomla !, TYPO3, Wordpress), popular web tools (jQuery, MooTools, osCommerce, Prototype, Google Analytics) .

Boom 2


Boom is an equalizer that improves sound on a Mac. Ideal if you like music code. The app can increase the maximum volume limit on Mac. After setting on the main screen of the application, you can adjust the maximum volume limit, adjust the equalizer and turn on the effects.



Boom 2 is a modern and unique EQ control system, using various amazing sound effects to improve the sound quality of the audio output. The application works on Yosemite, support for 64-bit architecture and Russian interface (and much more).
