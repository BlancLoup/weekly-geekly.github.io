Sharing IE6 and IE7

Many people know that when installing Internet Explorer 7 on Windows XP, the ability to use the previous version disappears. For an ordinary user, this is unlikely to be a big loss, but how about designers and web designers? Using virtual machines is of course an exit, but if the developer uses as the main OS, for example, Linux? Of course, you can run two guest operating systems at once, on one of which IE6 is installed, and on the other IE7, but then the system starts working significantly slower. There is a way out.

The utility from VelocIT, called IE7 Standalone, allows you to use IE7 without installation, and accordingly, without losing IE6. How does this miracle make work on your machine? So, do the following:

1. Download the IE7 distribution from the Microsoft website, but do not install it.

2. Download the archive with IE7 Standalone and unpack it in the folder with the distribution.

3. Run IE7 Standalone Setup.bat

After IE7 Standalone Setup.bat is finished, to launch IE7, launch IE7.bat. That's all. The method is certainly not perfect, but I think it has the right to exist.
