Windows Server 2008 R2 is available in DreamSpark

Hooray! Windows Server 2008 R2 appeared in DreamSpark .



Both English and Russian versions are available (and many others).
Important : WS2008R2 is only in 64-bit edition.

DreamSpark in brief


Just in case, let me remind you that the DreamSpark program is available for all students, graduate students and schoolchildren over 12 years old. As part of this program, you can get free professional versions of Microsoft tools for software development and design (for non-commercial purposes).

If you do not have access to DreamSpark yet, you can register here: dreamspark.ru
