XUI: a simple javascript framework for mobile web application development

The creators of the library are familiar with ready-made popular solutions (jQuery, MooTools), but they decided to write their own framework that will not be overloaded with desktop browser support code and focused on the main needs of developing for mobile applications.

Already, XUI supports mobile browsers such as WebKit, Fennec, and Opera. In the near future, support for IE Mobile and BlackBerry.
