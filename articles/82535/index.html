Microsoft transferred opensource code to Zend Framework

The recently released Zend Framework 1.10 contains the Zend_Service_WindowsAzure component, which is passed to the Microsoft community for full support of the Windows Azure cloud platform from PHP applications written in the Zend Framework.

Zend_Service_WindowsAzure allows you to use all Windows Azure services: Blob Storage, Table Storage and Queue Service.
