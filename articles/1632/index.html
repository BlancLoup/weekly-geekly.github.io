IE7 works better with feeds than Firefox 2

According to independent experts, the latest version of IE7 beta 3 handles feeds much better than the latest version of Firefox, at least with the default settings without the appropriate extensions. Here is a small comparison with screenshots .

Firstly, instead of displaying ‚Äúraw‚Äù XML, IE7 browser formats the feed on the screen in a way that it should look like in the RSS reader. It turns out quite readable version with headers in the form of hyperlinks.

Secondly, on the right side of the feed, the browser shows all the subject tags that mark the messages in this feed, so you can sort the entries by date and title, as well as filter by tags directly in the browser ( screenshot ).

The latest official version of Firefox 1.5.0.6 does not allow you to view feeds in this way ( screenshot ). Something similar appeared in the first beta of Firefox 2.0, but even there there is no thematic filtering. So at the moment, IE7 works better with feeds than Firefox 2. True, the creators of a free browser still have time to refine their product, and numerous extensions significantly expand its functionality.
