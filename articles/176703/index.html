Oracle fixes 42 Java vulnerabilities

Today, Oracle is releasing a planned set of Critical fixes for Java. Updates affect Java SE (Standart Edition) versions 7 (7u15 and below), 6 (6u43 and below), 5 (5u41 and below). 42 vulnerabilities are subject to correction, 39 of which are related to the Remote Code Execution type and can be successfully applied without actual authentication, i.e., remotely over the network, without using a login and password to access the remote system (potential drive-by installation).
