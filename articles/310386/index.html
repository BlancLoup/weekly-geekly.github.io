Improve the work of the largest media about online business with the help of a customized referral system



vc.ru - the largest media in the Russian Internet about digital business and technology. Now a personalized block on vc.ru is formed using Relap.io. We describe how we set up the work of the block and customize the algorithms.

The audience vc.ru - 1 800 000 visitors per month. Before publication, the task is to be the main source of news about Internet business and expand the audience. For this, the editors decided to use personalization on the site, so that each user would find on Vc.ru specially selected content for him.

This helps to increase the loyalty of the audience and improve the metrics, the depth of views and the time that the user spends on the site - the necessary components for the development of the media.

Now a personalized block on vc.ru is formed using Relap.io .





Tasks for Relap.io on vc.ru :


  • Customize the work of the block with content for highly specialized media about IT.
  • Increase the number of transitions on the recommendations.
  • Make it so that people stay longer on the site and find more interesting materials.

How we set up the block and customize the algorithms


Realp.io has a package of recommendation algorithms that we can apply in different proportions. The recommendation system analyzes the behavior of all users of the site and, based on this data, allows us to predict the behavior and interests of each user individually.

Limited time recommendations

The restriction shows how user interest in the material fades. On vc.ru is 30 days. Such a gap is not suitable for all media (for example, on RIA Novosti the optimal age of the article is 3 days).
