Apache Deltacloud 1.0 release

I recently wrote a post about Red Hat CloudForms , I think it would be unfair to overlook the release of the first version of the Apache Daltacloud project, the code of which lies at the heart of the CloudForms (and the whole Red Hat cloud portability strategy ).

The Apache Software Foundation (ASF) announced the release of the first version of Apache Deltacloud , a project that aims to improve the interoperability of various IaaS clouds (Infrastructure-as-a-Service). According to ASF, the project has reached a sufficient level of maturity in order to get a figure of 1.0 in the version number. In addition, a frontend appeared in Deltacloud 1.0, which represents the Amazon EC2 API for client applications.

The new EC2 API allows developers to run applications written specifically for the EC2 API on any IaaS clouds that support Deltacloud. ‚ÄúDeltacloud enables customers using the EC2 API to translate API calls into calls such as VMware vSphere or OpenStack,‚Äù said David Lutterkort and added, ‚ÄúThis is the beginning of a long way to get away from the vendor lock- in cloud APIs.

Deltacloud works as a server process that supports several different cloud APIs, so it is possible to create applications that are independent of the IaaS cloud. Deltacloud translates API calls from one cloud vendor to API calls from another that are expected to receive various cloud services. This translation is possible through the use of a set of different drivers .

The Deltacloud project has its own REST-based API and another, independent, basic (reference) API. This API was previously added as the " Cloud Infrastructure Management Interface " client interface currently being developed by the DMTF ( Distributed Management Task Force ). Deltacloud also provides developers with a way to implement their own interfaces for services, allowing internal user applications to be deployed across various cloud providers.

Initially, it was an internal Red Hat project that was designed to give users a unified approach to various IaaS clouds. But later the project was transferred to ASF and is now developing ‚Äúunder the wing‚Äù of this organization. Deltacloud 1.0 was released on June 15, you can download this release from the project site, the code is available under the Apache 2.0 license. Detailed installation instructions and release notes are also available there.
