Twitter Bootstrap 2.0 - Russian PSD style library

I present to your attention the library that I developed, which allows you to simulate the interface by simply mixing the basic elements - CSS Twitter Bootstrap v. 2.0.

Absolutely all library elements are presented in vector form and can be easily changed in scale and color. Most of the used gradients and effects are created by me in such a way that with one click only the main casting color can be changed, while other composite colors are automatically substituted in the layers.
