Mozilla Shows Places Too Much Like Windows Media Player

Mozilla published on its website a layout of Places Organizer, one of the most anticipated features in Firefox 3.

As part of the weekly status update for Firefox 3 , which is published on the wiki, a screenshot of Places Organizer was shown showing the interface that users will see with Firefox 3: either at the end of this or early next year.

Places is a single manager of bookmarks and history, which was originally planned to be included in Firefox 2, but which was eventually moved to the third version of the browser due to lack of time for Mozilla to develop it. Unlike the bookmark manager in the current browser, Places will use SQLite database, which will allow users to search among the saved pages. Also in Places will appear support for custom tags to facilitate the search.

Places Organizer uses Windows Vista style, which, as can be seen from the screenshot under the habrokat, almost completely repeats the design of Windows Media Player 11. This fact makes us think that maybe Microsoft will not miss the moment and will file a lawsuit against Mozilla - its main a competitor in the web browser market.
