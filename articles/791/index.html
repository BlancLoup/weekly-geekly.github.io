Introduction to rel-license microformat

Under Habrakat, I posted an overview description of a rel-license microformat designed to indicate for content or part of it the license under which this content is distributed (Creative Commons 2.0, Apache 2.0 license, GNU General Public License, etc.).

Description

rel-license allows you to specify a license (or licenses) related to this document. The link to the corresponding license is indicated in the href attribute of the regular hyperlink (<a href=1URUR license ‚Äûrel= <cliclicense>)

If the rel-license is located inside the hReview microformat , then the URL specified inside the rel attribute should refer to the license for the review, not the license of the entire document. This allows aggregators to find reviews that are subject to the same licensing, but placed in different sources (on different pages, websites, etc.).
