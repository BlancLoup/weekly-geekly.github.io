Bada SDK 2.0 released

Following the just-released mobile platform, Samsung Bada 2.0 has a tool for developing for it. The new SDK can be downloaded from here , and from the following innovations can be noted:

  • Multitasking. One of the main drawbacks of Bada has been eliminated (although the SDK 1.1 described some way to launch applications in the ‚Äúbackground‚Äù, but the developers honestly called it ‚Äúpseudo-multitasking‚Äù).
  • Improved support for Flash and HTML5.
  • Developers have access to the platform buffer.
  • New navigation styles Footer, Header and TabBar. Called UI 2.0, this feature is the ability to create blocks of control elements, respectively, at the bottom, at the top of the screen, and in a kind of horizontal menu.
  • Relative has been added to the possible layouts. According to the description it is difficult to imagine the details, but, apparently, the functionality is similar to RelativeLayout in Android.
  • Added support for NFC (Near Field Communication) technology, which is most often referred to in the context of using contactless payments.
  • Two new ServerChannel and ClientChannel classes have been added to exchange data between applications. The names are relatively loud, while it is also not clear the difference (or similarity?) From Intents in Android.
  • Special classes for managing Wi-Fi Direct - the ability to communicate devices without a hot spot.
  • New API to support targeted advertising. Two components are offered ImageBanner and TextBanner. The following actions are available on clicks: go to the browser at the URL of the advertiser, auto-dial the number provided by the advertiser (potential opportunity for scammers?), Go to the Samsung Apps Store app store.
