KolibriOS Digest # 2: what February brought to us

A month has passed since the release of the first issue of our digest. All this month, in spite of everything, our community has been working on KolibriOS. Unfortunately, due to unforeseen circumstances, I was not able to publish a digest on the last day of winter, and still meet KolibriOS Digest # 2.

System-wide changes:


  • The removal of the USB host controller code from the kernel to the drivers.
  • Increase the speed of work with flash drives as a result of combining gears.
  • PCnet32, i8254x, R6040 drivers are translated to PE format.
  • Fixed drawing of windows with a fixed size with a small window height.
  • INTEL HDA: correction of errors caused by accessing uninitialized memory.
  • VMware SVGA II: reduced screen refresh rate; The hardware cursor is on.
  • box_lib: Clipboard support for input fields (Editbox component).


Changes in application software:


  • ac97snd : fix for m3u playlists.
  • IRCC : the correct output of the message, if you were kicked / banned on the channel.
