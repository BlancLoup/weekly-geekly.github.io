An overview of audience building tools in mobile retargeting

The effectiveness of any retargeting campaign depends largely on the correct segmentation of the audience. Selection of segments and careful processing of data about users gives you the opportunity to work with each of them separately. This is unique, because each user sees personal banners, goods in which they are interested. This approach increases the conversion of a mobile application an unlimited number of times.

In the article, on behalf of Getloyal's automated retargeting system for mobile applications, we will look at and compare tools such as AppsFlyer Audiences and Adjust Audience Builder. If you use the services of these trackers, then additional integration is not needed.



Before proceeding to audience segmentation, some issues need to be addressed. First, decide how to choose the right audience for retargeting . Secondly, what tools to choose to quickly create a list of the audience on the parameters that interest you. Today, many advertising platforms allow you to quickly and easily work with the audience, allow you to download the necessary segments and target specific users.

Adjust Audience Builder


The tracker that allows you to work effectively with dynamic audience segments is the Adjust Audience Builder . When creating audiences, you can highlight a variety of conditions for targeting, for example: you can make segments by Ad ID and push token, by device type, country, mobile application installation date, last attribution session, event and so on. In addition, it is possible to apply different types of use of conditions, such as: include, exclude, less than, fewer / more than.

Suppose you selected ‚Äúrevenue event include 14‚Äù, then only users who have made exactly 14 orders will get to the audience.

It is worth noting that the Adjust Audience Builder automatically updates the lists of audiences. At the moment there is an opportunity to automatically push the audience on Facebook, after which the Adjust client has the opportunity to launch a campaign targeting the created audience.

Adjust also allows you to create a URL by referring to which the client can download the list of users in the form of a .CSV file. You can manually download the .csv file to any site or send a link to the retargeting partner file.

To work with partners, Adjust also provides an opportunity to split 1 audience into 2 or more, which allows you to parallelize the process of working with partners or, for example, conduct A / B testing.

Appsflyer Audiences


Audiences tool provides a functional solution for user segmentation. You can create audiences based on any events in the application, which are registered in AppsFlyer, event attributes (product type, material, category, etc.), as well as a segment of users who deleted the application (for iOS).

You can use different filters by period. For example: users who put the product in the cart in the last 7 days, but did not buy it. Or users who installed the app made a purchase but deleted the app for some reason.

The period for which the user performed or did not perform the action cannot be chosen arbitrarily, there are strictly fixed values ‚Äã‚Äã(1, 2, 7, 14, 30, 60, 90, 120, 360 days). You can choose the number of committed actions, for example, bought 5 times in the last 14 days.

Audience AppsFlyer integrated with advertising platforms: Facebook, Yahoo, Gemini, Twitter, Inmobi. In Audiences, you need to perform the setup once, after which the user data will be synchronized with the selected sources and updated automatically.
