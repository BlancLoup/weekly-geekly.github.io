Express.js project template

You can take it here: github.com/maxatwork/expressjs_template (or as a zip archive ).

What is

  • code converted to coffeescript
  • github.com/visionmedia/jade - template engine
  • learnboost.github.com/stylus - CSS framework analogous to SASS / LESS
  • github.com/TrevorBurnham/connect-assets - a resource collector a la Rails 3.1, automatically compiles coffee-script / less / stylus
  • github.com/shomeya/bootstrap-stylus - Twitter port bootstrap on Stylus
  • github.com/visionmedia/nib - utilities and mixins for Stylus, analogue of COMPASS for SASS
  • github.com/kriszyp/promised-io - I / O library using promise pattern
  • visionmedia.github.com/mocha - unit testing framework
  • github.com/visionmedia/should.js - BDD style assertions for node.js
  • github.com/visionmedia/supertest - library for checking http applications


Installation

We take out the zip-archive (so that there are no links to git), expand it to the right folder, go to it, run npm install .

Also, it is good to have it installed: 
 # grunt -  make npm install -g grunt # nodemon -    nodejs    npm install -g nodemon


Then you can perform:
  • grunt - run tests
  • grunt watch - will look for file changes and restart tests
  • nodemon app.coffee - will start the server
