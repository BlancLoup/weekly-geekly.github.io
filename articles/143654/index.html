In Peterhof with a smartphone

From June 2012, every visitor to Peterhof will be able to use a special application for excursions. The program was developed for Android and iOS, and it will be possible to download it directly "on site" - free wi-fi will soon be done throughout the complex.

On the ten proposed routes, near the park sculptures and structures, plates with QR codes will be installed, having scanned them, you can get information about the creators, the artistic context and the epoch to which this or that object belongs. Now the technology is already operating in Oranienbaum Park, this year the route for smartphones will be laid in the Lower Park.
