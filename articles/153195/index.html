Synology Introduces New DiskStation DS213air


Synology Inc has announced the launch of a new 2-disk NAS DiskStation DS213air with built-in support for Wi-Fi. The new DS213air will be a great solution for home or small office.

Synology DS213air provides a wide range of wireless configurations that will suit every user. You can connect the DS213air to the network without additional cables or use it as a hot spot to expand your existing Wi-Fi network. You can also use the DS213air as a wireless router to create a wireless environment, at no additional cost.

Wireless access point Wireless router Connection to a wireless network

Synology DS213air is equipped with Web Assistant, which helps users complete the installation of the DSM system in just a few simple steps, right from your mobile device. And thanks to the Quick Start widget, which helps users familiarize themselves with the DSM operating system already during installation, you can quickly get started.

Key Features Synology DS213air:
  • 1.6 GHz processor
  • DDR3 256 MB RAM
  • 1 port Gigabit Lan
  • Built-in Wi-Fi b / g / n
  • 2 USB 3.0 ports
  • Read speed 108.7 MB / s, write 58.66 MB / s


For more information, visit: http://www.synology.su/products/80

The recommended retail price of DS213air is 16,500 rubles.
