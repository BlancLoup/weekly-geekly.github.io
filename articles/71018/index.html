Widget "Google Translator by Dimox"

I decided to present the habrasoobshchestvu (and, more precisely, those who are active users of the Opera browser) my newly created widget called " Google Translator by Dimox ".

There has long been a similar widget with the exact same name, which I used for a long time, but which has not been updated for a long time and has a number of drawbacks that pushed me to create my own.

The Google Translator by Dimox widget translates text using the Google AJAX Language API , and the interface works using the jQuery framework, which I love so much.

When I planned to post the widget in public, I doubted whether it would be interesting to anyone. As it turned out, my doubts were in vain - it turned out to be more than claimed. In just over two months of its existence, the widget was downloaded more than 100 thousand times.
