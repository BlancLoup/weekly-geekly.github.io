CheckBot 1.0

Well, I finished the work on version 1.0. Now this is a full-fledged plugin that works like a clock and is available on the Wordpress plugin repository :)

I did a lot of work on the plugin, completely rewrote the code, improved the protection. With the current protection, the plugin works with a bang, perfectly eliminates automatic spam. They will teach the hrumer to bypass my protection - it doesn't matter, I will constantly improve it.

I really hope for adequate criticism and help tips :)

So, a little about the plugin:

CheckBot is a simple captcha plugin for wordpress. It is very easy to use (only 1 click is needed), but at the same time it is completely customizable and rather seriously protects your blog from spam.



Opportunities:

* Protection from spam. The plugin uses javascript and randomly generated identifiers and field names, which allows the plugin to filter out spam that is automatically sent out.
* Language support. Plugin supports Russian and English.
* If the user chose the wrong picture, or forgot to choose it at all, then the plugin will inform him about it, plus, just in case, a written comment will show it.
* Fully customizable appearance. You can create your own set of images that best fit the design of your blog. Or you can simply change the standard sets of pictures.
* The plugin can work immediately after installation, without additional settings.
* You can customize the output of the plugin on your own anywhere on the comments page.

You can visit the page on my blog dedicated to the plugin.
Or download the plugin from the repository of Wordpress plugins .
