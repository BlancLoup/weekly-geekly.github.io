Windows 10 will download some updates even through limited connections.



According to preliminary data, with the release of the final version of Windows 10 Creators Update, users will no longer be able to bypass the mandatory installation of updates using customizable limit connections. Moreover, the operating system can now receive critical updates not only through a custom, but also by default, limit connection, which is, for example, a mobile Internet connection.

This conclusion was reached by one of the Windows Insider program participants Richard Hay, who studied the build of the preliminary version of the Build 15058 updates for Windows 10, presented by Microsoft at the beginning of last week. During his acquaintance with the build, he discovered a difference in the wording of the Windows Update settings with the previous version.


Text in the update center settings in version 14393.953


Text in the update center settings in build 15058

Any member of the insider program can make sure of this: if you go to Windows Settings> Update and Security> Windows Update, you can see the update parameters under the header, which can now be downloaded via a limit connection.

The main difference between these two versions of Windows 10 is the second sentence, which reads as follows: "In this case, we will automatically download only those updates that are necessary to maintain normal Windows operation."

Customizable limit connections in Windows 10 is a widely used method to control the automatic download and bypass the installation of updates that are mandatory in the home version of the operating system. Now users who use a cellular connection to transfer data or any other default chargeable channel will not be able to do anything when the updates are downloaded and use up the rest of the traffic.

When Hay saw the changed description of the Update Center, I decided to find out what those updates are meant to ensure the normal operation of the operating system. He contacted Microsoft Support Center to get a more accurate picture of the types of updates that fall into this category.

The response from the press secretary did not particularly clarify the situation: "We do not plan to send large updates via limit connections, but we will use them if necessary for critical fixes in the future." No specifics about the amount of consumed traffic followed, so Hay urges to closely monitor this.

Update Windows 10 Creators Update will be released on April 11. It is possible that the text that was written for the preliminary version will change. On the other hand, the likelihood that a large final update will change dramatically shortly before the release is extremely small. After all, the work is not limited to editing the text: the developers set up the basic network capabilities to allow installing some updates via channels with charged traffic.

If we talk about changes in the nuances of setting up limit connections in Creators Update, on the network connection settings page you will be able to mark the wired connection as limit connection. At the same time, users will be able to postpone the installation of already downloaded updates for any convenient time or, using the ‚Äúsleep‚Äù mode, automatically transfer it for three days.

In addition, the new assembly will appear more significant updates. One of the most interesting is related to multitasking: with the help of the Compact Overlay function, users can switch to the ‚Äúpicture in picture‚Äù mode. With it, you can launch applications in a small window that will be displayed on top of other open tabs. It will be useful not only for lovers to watch the video, simultaneously performing any monotonous task, but also for launching, for example, instant messengers.

Another useful feature is the automatic blocking of the computer when a smartphone connected via Bluetooth is disconnected. If you leave the range of Bluetooth, after 30 seconds, the computer will go into lock mode.

In addition, there will be support for augmented and virtual reality devices, as well as integration with other 3D capabilities, such as sharing content with other creators of three-dimensional models. Microsoft has promised gamers to optimize game support in 4K resolution and launch video broadcasts. Game Mode will optimize the performance of the computer during the game by turning off unnecessary background processes and some other techniques, but more, apparently, you should not expect from this feature.

In Windows 10 Creators Update, more attention to security will also be given. Enhanced threat detection and analytics tools will be added to the OS security center.

A closer acquaintance of Windows 10 users with all the updates will take place on April 11th.
