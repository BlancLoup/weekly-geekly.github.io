0.8.x branch became stable

On September 27, Igor Sysoev published a new stable version of the web server nginx-0.8.51. Mailing list message:
The status of version 0.8.x is changed to stable.
During the development of this version, among other things,

* support of regular expressions in regular expressions,
* support for file AIO on FreeBSD and Linux,
* SSL CRL,
* SCGI and uwsgi modules.
