Build 2014. Hot news for Windows and Windows Phone developers



Just a few hours ago, the opening of the Build 2014 conference was completed, at which many interesting announcements were made for developers and users of Windows and Windows Phone.

Let me remind you, the Build 2014 conference takes place from April 2 to April 4 in San Francisco. For us, the online broadcast of events and recordings is available.

Following the opening of the conference, the main news for the developer is the ability to create universal applications that work equally well on a smartphone, tablet, laptop, and TV, using a common API and design principles.

In addition, other changes were announced that await us in the near future:
  • New versions of operating systems and products ;
  • Changes in app stores ;
  • New devices .

And now about everything and more ...

Meet new products!


Windows Phone 8.1

In a few months, we are waiting for the release of the new mobile operating system Windows Phone 8.1 , which will be available as an update for all devices on Windows Phone 8. It will become available to developers a little earlier (no deadlines have been announced yet).



Some new features of the operating system were demonstrated:
  • The ability to use two SIM-cards, if the device allows.
  • Customizable lock screen.
  • More tiles on the main screen and background pictures.
  • Notification Center - the ability to enable / disable notifications from various applications.
  • New swype-keyboard for accelerated printing with support for the Russian language.



  • Wi-Fi Sense for searching and connecting to the Web in an unfamiliar place.
  • Quiet hours - customizable schedule for receiving calls, mail, notifications from applications. It can determine the location and apply the specified settings. For example, do not receive mail when you are at home.
  • Improved calendar and weekly breakdown of data.
  • Cortana is a bing-based personal digital assistant. You can communicate with her, she knows how to perform a huge number of voice commands and is trained.



  • VPN support, S / MIME, device management using MDM and encryption for corporate users and organizations.

Windows 8.1 Update

We are also waiting for an update for the Windows 8.1 Update client operating system, which is already available for MSDN subscribers. Other users with Windows 8.1 will receive an update already on April 8, absolutely free .







The update for Windows 8.1 contains the long-awaited changes by many users:
  • Minimize and close buttons for all applications.
  • Shutdown and search buttons on the main screen.
  • Context menu for "Live Tiles".
  • Move several tiles at once with the mouse.
  • The search is now looking for more apps in the Store.
  • The app store is attached to the "Taskbar".
  • IE - Enterprise Mode for compatibility with previous versions of IE, including the ability to manage policies for which resources to include.

Also at the opening they showed and told what they are working on now - the Start menu and the ability to launch Metro UI applications in desktop mode, in a window. See and switch to them using the "Taskbar".

WinJS - open technology

Who said that Microsoft doesn't like open source?

The WinJS JavaScript framework, which is one of the ways to create Windows and Windows Phone applications, is now becoming an open source solution and is already in open access on github .

Not a joke, not April Fools.

Universal applications for Windows Phone and Windows 8.1


Now we will have a single development platform for Windows and Windows Phone.

Windows Phone will also use Windows RT, and in Visual Studio 2013, after installing the SDK, new project templates and options for converting and creating universal applications will appear.

In addition to the common core, controls and design principles, the new operating systems will include the new DirectX 12 and IE 11 with WebGL support.

Universal Windows applications can be run on the Xbox too.

App Stores


Such global changes could not affect the app stores. And here, too, a lot of good news.

As you know, now for Windows and Windows Phone store, we use a common developer account. And now we are also waiting for a single identifier for the application.

Now you can buy the application on your smartphone and not pay again for the tablet and desktop .

Also, the interface of the stores will change a little, he will learn to recommend you applications based on your preferences.

New devices


In the summer, we expect new models of smartphones from Nokia. The flagship Nokia Lumia 930 and the bright and affordable Nokia Lumia 630 and Nokia Lumia 635 .



Nokia Lumia 930 is a new, very nice-looking smartphone based on Windows Phone 8.1 with a good screen, a 20 megapixel camera, stabilization and several sound speakers around the perimeter of the case.

Nokia Lumia 630 and Nokia Lumia 635 are affordable smartphones with one or two SIM cards. They promise that these devices will cost less than $ 200.
