Opera 10 alpha

Tlko looked at the developers of the Opera and saw that they posted a new version of Opera 10.

Download links:

Windows Classic (build 1345)
Macintosh (Universal) (build 6284)
Macintosh (Intel-only) (build 6284)
UNIX (build 4205)
A complete list of changes / corrections / novelties translated based on the original post is given below.

From what is important to me - in this build they fixed a problem with fonts, which manifested itself when using the Russian encoding as the default in Windows.
Well, a whole bunch of different;)

What's new?
During this time, a bunch of corrections. Many works are still in progress, such as spell checking ...

Check for damage to the mail database
In this assembly, Opera checks the mailer's integrity database for each run. If something is wrong, it tries to recover.

Facebook posting does not work

Interface corrections
Fixed crash when searching
More feeds to read on the preview page.

post office
