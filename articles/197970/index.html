Fresh results of Techempower framework performance testing became available.

For those who are not familiar with tests of web frameworks that Techempower regularly conducts, I highly recommend doing this now by looking at the link www.techempower.com/benchmarks

So far, only the results of the 6th round have been published for public access. Since then, several new fermworks have been added to each of the platforms and the developers have been waiting for the next round for more than two months, which is about to take place.

But so far Techempower has provided only a certain preview of the results of the 7th round in order to give developers the opportunity to make recent changes or fix bugs. If you are also interested in how your framework will show itself in the 7th round, you can see the latest results here www.techempower.com/benchmarks/previews/round7

PS Almost all PHP frameworks return errors, which indicates a problem with installing PHP or MySQL on the server. We hope that it will be fixed.
