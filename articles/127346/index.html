LiveStreet 0.5 release

Today finally released a new release of the blog-social engine LiveStreet 0.5.
More than a year has passed since the last release, and many have been waiting for the LS update. We present you a new version of LS 0.5 and its new features.

New opportunities:
- Standard templates are translated into jQuery, in the current version there is still support for MooTools;
- The ORM functionality (active record) has been added to the kernel, thank you so much runawayed ;
- New type of topic - topic photoset
- New section "Ribbon", you can create a stream of content from blogs and authors
- New section "Activity", tracking the activity of users
- custom fields functionality in user profile
- Redesigned ajax handler mechanism, all responses are now in JSON
- Setting up a parser (jevix) is now in a separate config, iframe support
- Paginated output of comments using nested set
- Updated Smarty 3.0.8, TinyMCE 4.3, MooTools 1.2.5
- Autoload classes now via spl_autoload_register
- Rewritten all js code for jQuery
- Security Fixes
- A huge number of bug fixes and minor changes / additions

By tradition, small statistics on the development of the project:
- Over 80,000 downloads;
- Created more than 180 different modules, most of which are free;
- Over 12,000 users are registered on the community site, of which several hundred are active members;
- It is written more than 6500 topics and 73 000 comments to them;

I want to express my deep gratitude to everyone who helped and helps one way or another in the development of the LiveStreet project. Thanks a lot! Separately, I want to thank Lora_GT , she has been supporting us since the very birth of the project, sending huge bug reports and giving useful advice)

Download the new version from the download page .

PS Shortly after the release, we are preparing to launch a contest for a new default template LiveStreet, we invite everyone to participate! On the start and conditions of the competition will be a separate announcement.
