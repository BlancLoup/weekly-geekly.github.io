PhoneGap 1.0 allows you to write applications for seven platforms



Until now, mobile application developers have followed a fairly predictable modus operandi: first develop an application for iOS, then for Android, and then for other platforms.

Last year, Nitobi began developing the open-source project PhoneGap , which makes it easy to create applications for Android, Bada, BlackBerry, iOS, Symbian, webOS and Windows Phone 7 using HTML, CSS and JavaScript.

In addition to Nitobi, hundreds of individual developers and a team of senior software engineers at IBM are involved in the development. The project is rapidly gaining momentum, it already has about 40,000 downloads per month - a total of 600 thousand - and a constant stream of donations.

Now version PhoneGap 1.0 has been released, which focuses on access to native device APIs.
