FireDiff - an extension for FireBug that allows you to track changes in the DOM and CSS

Kevin Decker has released FireDiff , a great Firebug extension that tracks changes in the DOM and in the style sheets of an HTML page.



Firediff implements a change monitor, which writes all changes made by both FireBug and the Web application itself to CSS and DOM. This allows you to get into the details of the operation of the application, and also allows you to submit a report on the changes that were needed to debug and customize the page display.

The release, which you can find here , uses the latest features of the latest builds of the Firebug 1.4 code.
