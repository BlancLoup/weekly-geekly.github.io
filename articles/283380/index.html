Facebook has laid out its CTF platform on Github - Facebook CTF


Facebook CTF is a platform for organizing CTF competitions (capture the flag), namely, its two varieties:


  • Jeopardy is a classic CTF with a set of tasks. Access to the following can only be obtained by correctly deciding the previous one;
  • King of the Hill - a type of CTF, in which you need maximum time to keep control over the hacked system; The trick is that the system periodically regenerates the state, ‚Äúdropping‚Äù participants from the level to which they managed to climb.

Immediately the most important thing, in my opinion: no, Facebook CTF does not contain tasks in and of itself, and no, this is not some kind of framework for creating tasks. This is essentially an interface for managing the process of CTF, no more. However, a small number of ready-made assignments are also available in the FBCTF repository, but they are only for demonstration.


It is better to watch a two-minute video .



In short, what the FBCTF provides


  • registration of participants and teams
  • posting job information
  • announcement and hint system
  • checking flags, scoring, ranking table - in real time
  • cool interface

The platform runs on (expectedly) HHVM and Flow . Installation instructions are there, in the description on Github . For devel environments, VirtualBox and Vagrant are recommended, and Ubuntu 14.04 is needed for sales (but I think it should take off on other debianes as well).


By the way, FB CTF is included in the bug bounty Facebook program .
