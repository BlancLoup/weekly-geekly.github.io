Microsoft plans to make this technology part of the following Visual C ++ compiler and fully integrate into the next version of Visual Studio, codenamed Visual Studio vNext (most likely it will be called Visual Studio 2012).

Microsoft announced C ++ AMP today at the AMD Fusion Developer Summit. At the conference, Coat of Arms Sutter told those present about the work of Microsoft in creating technology for using C ++ features on the GPU.
