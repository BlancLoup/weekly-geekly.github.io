The first release on the Opera Developer channel


As we have written before , Opera moves to an accelerated release cycle consisting of three channels: Opera, Opera Next and Opera Developer. If the first two channels are already with you, then the third, the most experimental one, we held on until we had a lot of interesting things in it. So, meet: Opera Developer for Mac and Windows .

If you are developing websites or extensions, we recommend that you install this build of Opera Developer: it will be updated frequently and will become the place where new browser features and web technologies will appear, which will eventually come in stable versions of the Opera browser.

What is the difference between Opera Next and Opera Developer? Next demonstrates the features that will appear in the next stable version, and Developer is a place for experiments that will go into Next and go further, and maybe will remain experimental.

Not to be unfounded, here are some differences between the current versions of Opera Next and Opera Developer ...

  • The current version of Opera Next is 16.0.1196.29 and it works on Chromium 29, Opera Developer has version 17.0.1224.1 and runs on Chromium 30.
  • In this version of Opera Developer, we returned some features from Opera 12: tab fixing and gestures with the mouse button held down (do not forget to enable gestures in the settings).
  • Added new APIs for extensions: bookmarks, commands, address bar, webNavigation - documentation for them in work.
  • In the settings for the first time appeared the block "At startup", where you can choose what Opera will do: continue from the last place, open the start page or open a group of pages that can be specified additionally.
  • Now you can create your own search providers and set keywords for them.
