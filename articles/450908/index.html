Black list of networks for Asterisk

After installing Asterisk and outputting the gateway to the Internet, attacks immediately begin with attempts to connect and make calls. To facilitate the work of other system administrators, I post a blacklist of networks.

After the last network with which the attack was made, it was already a month of silence. Of course, over time, the attackers will use the new networks, but they will not add new ones to this list in the future. For those who do not know where this list is stored (I have a web interface on FreePBX 14.0.11):

  1. go to the web interface
  2. Connection-> Firewall
  3. Services-> Blacklist

If the list contains the network you are using, simply remove it. In the future, add to the blacklist only the address from which there will be attacks. Information on setting up a fire in an asterisk on the Internet is complete.
