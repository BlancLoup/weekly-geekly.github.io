Jolla has released the update Sailfish 1.0.7.16, and also opened the source code of Sailfish Office

Today, the company Jolla, which produces a single mobile phone, has provided a new version of its operating system - Sailfish OS . The update implemented many of the applications that can be placed on the company's tracker .
In addition, under the GPLv2 license, the Sailfish Office office document viewer was opened, based on the code base developed by the KDE community Calligra office suite, using the Sailfish Silica components for building the interface and the Poppler library for displaying PDF.



More about innovations under the cut

  • 4G support for all users (with the appropriate sim card)
  • Display notifications of new messages on Facebook with the ability to immediately send a comment or like
  • Folder support in the application list
  • Support for popular calDAV services: Yahoo !, MemoToo, Fruux, and ownCloud (experimentally)
  • Bidirectional synchronization of the address book with Google services
  • Switching between browser tabs now does not reset the position to the beginning, but remembers it
  • HTML5 fullscreen support
  • Add your own shortcuts to the drop-down menu on the lock screen
  • Support for additional bluetooth profiles: PBAP (phone book) and HID (input devices, allows you to connect a mouse and keyboard to smartphones)
  • Traditional Chinese on a virtual keyboard
  • New browser options related to privacy: data cleansing and ‚ÄúDo Not Track‚Äù mode
  • Support SIM Application Toolkit to access the service provided by telecom operators and those present on the SIM card
  • Ability to check for updates via Settings> System> Sailfish OS updates, as well as the ability to specify modes when you can check for updates (Manual / WLAN only / Always)
  • The refusal to use the store android-applications by default Yandex.Store, with the possibility of its removal from the system
