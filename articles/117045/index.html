Plugin Overview ++

Another topic with an overview of several interesting jQuery plugins.

Mobily map




Allows you to make an interface similar to google-cards, based on your own images (the ability to drag, put markers on the image, respond to clicking on them, etc.).
Learn more | View in action




easingPlugin. 30 options for easing


A useful plugin for those who independently organize animation effects using animate () and other standard animation methods. All of them have an easing parameter, which determines the dynamics of effect execution. jQuery provides only two options for this parameter - linear and swing (for uniform animation and animation with acceleration). The easingPlugin plugin provides another 30 (!) Variants of dynamic models.
