Opera 10.60 - summer release



Today came the next stable version of the Opera browser with the number 10.60. As we promised - at once for all popular platforms. Answering possible questions about changing the number in advance, I can say that, compared with the 10.5x family, several important changes have appeared in this version, which was the reason for the beginning of the numbering. I will tell about these changes a bit later, but for now a few words about other updates.

The interface has changed the display of thumbnails of pages when you hover over tabs, now when viewing these pop-up images added animation. Changes in the Express panels were also associated with sketches: the cells were adapted for widescreen displays, and removing the search field from this panel, you can use almost the entire space to display only page thumbnails.

If we talk about corrections in the code, a lot has also been done here. Especially it can be noted that finally the errors of saving and opening pages and web archives with names in Russian have been fixed, and now Opera browser should work correctly with FTP servers containing files and directories with Cyrillic symbols. Of course, the work on the bugs is not stopped (this is like a repair - you can‚Äôt finish it, you can only stop it), so send bug reports - we will always be happy.

Well, about the speed, probably, it is not necessary to mention - by itself, these indicators have become even better both in comparison with the previous branch 10.5x, and in relation to other browsers, while acting as a catch-up. In addition to speed, the own security system has been improved, protecting the user from fake sites, Trojans, viruses and other joys of network life.

Now a little about more significant updates. This version of Opera includes support for the new WebM video codec, which today is one of the most likely candidates for the default post for all video content on the network. It is still far from perfect, but the potential of the codec is decent, which allows you to look optimistically at it and our future.
