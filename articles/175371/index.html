Overview of ECMAScript 6, the next version of JavaScript

To begin with, an educational program and a few facts:
  • ECMAScript is the official standard of the JavaScript language (The JavaScript word could not be used because the word Java was a trademark of Sun) Ie. JavaScript is an implementation of the ECMAScript standard.
  • TC39 is a committee that develops the ECMAScript standard and makes decisions on incorporating features into it.
  • ECMAScript standards a lot. The most popular of them is ECMA-262.
  • ECMAScript 5 is the latest revision of ECMA-262 (approved in 2009).
  • Previous versions of the ECMA-262 standard were (I don‚Äôt mention the old ones):
    • ECMAScript 3 - supported by most browsers (approved in 1999).
    • ECMAScript 4 - not adopted due to too radical changes in the standard. Later in July 2008, in a trimmed version (but still much richer than ECMAScript 3), it turned into a new ECMAScript Harmony project.

  • ECMAScript 6 (codename ECMAScript.next) must be established by the end of 2013 .
