Turn off spelling in Google Chrome


Spell check in Google Chrome as you can see is still far from perfect, emphasizes almost everything. For those who are annoyed, a solution was found.

TO TURN OFF THE VERIFICATION INSPECTION:

1) First we close Google Chrome (!)
2) Go to the daddy, which stores Google Chrome:
In XP, this is C: \ Documents and Settings \ <Username> \ Local Settings \ Application Data \ Google
In Whist C: \ Users \ <Username> \ AppData \ Local \ Google \ Chrome
3) Go to the folder User Data \ Default
4) Open the Preferences file.
5) Insert the following line into the existing JSON:
{"Spellcheck": {"dictionary": ""}
6) Turn on Google Chrome.
