Perspective API is available for developers.

There is one very interesting news that has remained poorly lit, especially in the Russian segment of the Internet. Google's Perspective API is now available for developers. What is this another crap you ask? In short, this is an API or service that allows you to evaluate the "toxicity" of comments on the network. Yes, this is not a joke, that's all. The project partners include such news agencies as The New York Times, The Guardian, The Economist, and the Wikipedia Internet Encyclopedia.

The project looks very interesting. We all know that everyone writes on the Internet. Here and trolling, and propaganda, a variety of appeals, fakes of any scale, threat, recruitment from all banned organizations and the like crap. This is quite a serious problem these days.
