Yandex begins to struggle with re-optimized texts

Yandex officially announced the start of the fight against re-optimized texts:

Next week, a new search algorithm will start working, affecting the ranking of pages containing overly optimized texts. This algorithm analyzes the content on the indexed pages and assesses whether it is useful and convenient for site visitors. If the algorithm determines that the content was created to influence the search engine and increase its relevance, the position of this document in the issue may deteriorate. The algorithm checks all pages that the robot bypasses, so when the problem is resolved and the site page is reindexed, the site positions will be restored. However, it will take some time (up to 2-3 weeks).

The news is more positive than not. Certainly, of course, there will be some false positives - especially at first. But I really want to see content on the Internet for people, not for robots.
