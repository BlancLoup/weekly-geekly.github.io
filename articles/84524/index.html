Deploying a wifi network to school

If for this community the post is offtopic - do not kick right away. Maybe not here, but to the gamblers, networkers. So, I ask for help.

There is a task: to deploy a wifi network to the school. At the exit, the school has an Internet channel of 10 Mbps. It is necessary to make sure that in any room of the school there is normal wireless Internet. We believe that we have an average school, has 4 floors.

Important: a decent Internet should provide 25 laptops with built-in wi-fi-modules.

Option 1. We put one router from which all the laptops in the class are powered. In this case, as I understand it, to switch to another class, you need to change the location of the router.

Option 2. We put the router, 6-8 access points are bought to it, which are scattered on different floors of the school. In this case, we get a single wi-fi-network.

Or am I, in principle, wrong and this is done quite differently?
