Internet Explorer 7 released in Russian

Microsoft has released Internet Explorer 7 for Windows XP in Russian. You can download the new version on the company's website in a special section .

As they say on the Russian version of the site , Internet Explorer 7 is a new style and more features that allow you to simplify and speed up the execution of everyday tasks, such as searching, browsing websites and printing them. To help users, Microsoft introduces a tabbed browsing method in IE7. Using them, you can open several web sites in one browser window and easily switch between them. Tab functionality has been enhanced by the new Quick Tabs feature, which gives a simultaneous snapshot of all open pages on one screen at once.
