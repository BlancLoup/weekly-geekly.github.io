Expansion


I called the extension Habrahide, that's what it can do.

Hide comments

There are 3 different methods. Depending on the method chosen, comments will be hidden whose rating is:
  • less than or equal to the specified; default is 0;
  • less than or equal to the average rating of all comments;
  • less than or equal to the average rating of positive comments; A positive comment is a comment whose rating is greater than the specified one (the default is 0).
Different methods hide a different number of comments, you can choose the best for each post. The default method is set in the settings, and the extension menu allows you to change the method only for the current tab
