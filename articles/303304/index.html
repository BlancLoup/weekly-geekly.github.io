Software PBX 3CX V15 ALPHA released

We worked hard the last 8 months and already at the finish line! 3CX Phone System v15 is a new release of our system with many new features, a modern management interface and architectural improvements.



Some new features in 3CX Phone System v15


  • New management interface
  • New Dashboard for easy system monitoring
  • NGINX web server instead of IIS / Abyss servers
  • 3CX can now provide you with a FQDN name for your PBX server
  • 3CX automatically creates and maintains free, trusted security certificates Let's Encrypt for you
  • SSL Labs A + rating
  • Automatic recognition and blocking of SIP hacking utilities
  • Encryption of voice traffic through 3CX Tunnel and SBC
  • Improved setup and operation of DID lines
  • Improved integrated video conferencing system
  • Remote automatic configuration of supported IP phones through the 3CX RPS server (Remote Provisioning Server)
  • Centralized chat logging
  • Search for call history through the web interface
  • Branding (White label) 3CX Webmeeting Interface
  • Powerful script support system for universal CRM integration and call automation
  • New URLs for the updated 3CX Web Meeting video conferencing system
  • New video conferencing application 3CX WebMeeting for Android

New features of 3CX Phone System v15, not included ALPHA version


  • New 3CX Client Interface
  • New video conferencing planning interface
  • Easy configuration of FXS gateways
  • Easy configuration of base stations and DECT terminals
  • System permissions at user level (extension)
  • Group editing of extension numbers
  • Ability to run VAD (Voice Application Designer) scripts
  • New version of 3CX SBC (Session Border Controller)
  • 3CX WebMeeting app for iOS
  • Call history cannot be transferred from 3CX v12.5 directly. You must first install 3CX v14.

Clients 3CX for Mac and Windows will be automatically updated to work with 3CX v15. 3CX clients for Android and iOS v14 will continue to work with 3CX v15. Full changelog .
