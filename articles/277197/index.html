News and Releases


  • PHP 5.6.18 , 5.5.32 , 7.0.3 - Updates to current branches, in which about 40 changes have been made, including several vulnerabilities. Also in 5.6 and 7.0 added support for HTTP code 451.
  • Go! AOP Framework 1.0.0 - The first stable release of the popular AOP framework for PHP.
  • Yii 2.0.7
  • phpschool.io - A new resource for learning PHP. Provides a special console tool that will help step by step solve educational problems.


Php


  • RFC: Add PHP Engine Identifier Constant - It is proposed to add a constant PHP_ENGINE , the value of which would indicate the type of engine. For example, php - for Zend PHP, or hhvm - for HHVM.
  • RFC: Allow specifying keys in list () - The proposal was adopted by vote . Already in 7.1, the ability to specify keys in the list() construction will be implemented:
    list("name" => $this->name, "colour" => $this->colour) = $attributes;


Instruments


  • php-ds / ds - Effective data structures for PHP 7. A post with a description, benchmarks and comparison with structures from SPL.
  • doubleleft / hook - An open alternative to the dead, deceased Parse . Implemented in PHP and has a set of ready-made clients for JavaScript, C # / Unity3D, iOS / OSX, Java / Android, etc. Slides with a description.
  • BitBar PHP Formatter - PHP plugin for BitBar - allows you to display any information in Menu Bar OS X.
  • arodygin / linode-api-php - Library for working with Linode API.
  • BZoo / Event - Easy and flexible event dispatcher. Supports work with namespaces for triggers, call by pattern, priorities, interrupt queue.
  • simpleci / simpleci - Docker-based continuous integration system.
  • linkorb / libcloud-php - Unified API for working with cloud service providers. Linode, Digital Ocean hosting is supported; DNS provider GoDaddy. Sent by dali .
  • JBZoo / Image - A rethought fork of SimpleImage - a popular library for working with images. Sent by smetdenis .
  • banago / PHPloy - FTP / SFTP deployment tool with support for multiple servers, sub-modules, and rollbacks.
  • tedious / Stash - Library for caching with PSR-6 support.
  • mamuz / PhpDependency Analysis - A static analysis tool that builds a class dependency graph. Examples of graphs .
  • opencfp / opencfp - Application for receiving applications for the conference.
  • ray-di / Ray.Di - DI framework for PHP, inspired by google / guice .
  • sebsauvage / rss-bridge - RSS feed for sites that do not have one: Facebook, Instagram, Twitter, GoogleSearch and others.
  • MUlt1mate / cron-manager - Manager of cron-tasks.
  • daylerees / scientist - A library for conducting experiments in code. An alternative solution is aaronbieber / edison .
