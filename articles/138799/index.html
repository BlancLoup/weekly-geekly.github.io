DropBoh updated the android client: testers of a new function are given 3 GB each


Yesterday, the mobile client for Android of the well-known cloud storage service DropBox was updated, providing users with a new feature - automatic upload of photos and videos to their account. Materials are uploaded without any loss of quality and size changes via Wi-Fi or 3G in the folder Camera Uploads.

For testing a new feature, users of the application will receive an additional 500 MB of space when they first automatically upload a photo. If, nevertheless, the user is so pleased with the function that he decides to upload more photos and videos, this limit will increase to 3 GB. Also, as stated, in order to increase security, it is no longer possible to install the application on the SD card.

The updated version for iOS is promised very soon.
