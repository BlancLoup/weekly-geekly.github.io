Microsoft advised Skype users to change passwords

The last few weeks, Skype users have begun to contact those. support of the company and writing on the forum about receiving phishing links from their contacts in the messenger, which may indicate that they were compromised. This topic on the Skype community forum has already collected 28 pages, and complaints are received through Twitter. Links have a shortened format, which is clearly used by attackers to hide the original URL.



The company remains silent about the reasons for what is happening, but the well-known edition of The Register received a comment from Microsoft officials in which they advised all users of the service to change their account passwords.

We are talking not only about Skype accounts, but also about Microsoft accounts (Live ID), if they are used to log into the instant messenger. In this case, the danger of compromising the Live ID is much more serious, because in this case, an attacker can get access to all user data of the Microsoft account, including Windows, since the Live ID is used as the main authentication system on Windows 8+.
