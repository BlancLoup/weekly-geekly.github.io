Windows internals. The second volume is on sale!

We are pleased to announce that the second part of the 6th edition of the legendary book Windows Internals devoted to the internal structure and algorithms of the main components of the Microsoft Windows operating system has been published in our publishing house. Recall that for the first time Windows Internals was divided by the authors into two volumes. The first volume came out last year, we wrote about it here .



The sixth edition of the book Windows Internals covers the inner core of the components of Windows 7 and Windows Server 2008 R2.
This book will help you:

  • Understand how the system kernel and control mechanisms work, starting from the object manager to the registry.
  • Learn more about the internal data structure system using tools such as the kernel debugger.
  • Examine the Windows security model from the inside to see how it permits data access.
  • Understand how physical and virtual memory is managed in Windows.
  • Examine the Windows network stack from the top to the bottom, including the API protocols for drivers and network adapter drivers.
  • Understand file system access and boot problems.
  • Learn how to analyze crashes.

The first part of the book begins with two chapters in which key concepts are introduced, the tools used in the book are described, and the general architecture and components of the system are described. The next two chapters are devoted to the underlying system and control mechanisms. The first part ends with the consideration of three key components of the operating system: first, these are processes, program flows and tasks, second, security, and third, networking.

The remaining key subsystems, which include I / O, long-term storage and memory management, as well as the cache manager and file systems, are discussed in the second part. Completes the second part of the description of the processes of starting and stopping the operating system, as well as tools for analyzing an emergency dump.
