Release of Chrome fixing vulnerabilities for $ 23,500

Last week, Chrome was updated to version 59. In addition to changes in the design material side, 30 security bugs were closed. The total amount of remuneration for which exceeded $ 23k, which once again proves the efficiency of the bugs bounty.



The most serious breach was found by Zhao Qixun, also known as S0rryMybad. This vulnerability is affected by the JavaScript V8 engine (CVE-2017-5070), the expert received $ 7,500 for it. In April, two other experts told Chrome developers about the shortcomings associated with executing code outside memory (CVE-2017-5071) and omnibox spoofing addresses (CVE-2017-5071). For each of these vulnerabilities, researchers earned $ 3,000.
