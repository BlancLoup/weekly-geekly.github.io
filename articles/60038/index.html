City Guide - First navigation traffic jams for Nokia 5800 XpressMusic

MIT has released the famous City Guide navigation software for Nokia 5800 XpressMusic (running Symbian S60 ed.5.).

This is the first available navigation software for popular Nokia 5800 XpressMusic phones , which has a service of traffic jams and the most current maps of Moscow and St. Petersburg.
City Guide 2.4 for Nokia 5800 XpressMusic in functionality corresponds to the program City Guide 2.2 for Symbian S60.

At the moment, City Guide is the only manufacturer of navigation software with support for traffic jams, covering almost all platforms.

City Guide versions are available for : Windows Mobile, Windows CE, Windows XP / Vista, Java, Brew, Symbian S60 ed.3 and ed.5.

For the "lovers" of everything free, there are freeware versions from the joint project Mail.ru and Probki.net - Maps@mail.ru .
