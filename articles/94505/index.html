Fedora 13 Goddard distribution released



The main features of Fedora 13 can be found at:
http://www.fedoraproject.org/wiki/Releases/13/FeatureList





Key features and innovations of Fedora 13:
  • Automatic installation of the printer driver
  • Automatic installation of language packs
  • Reworked user account tool
  • Color management for the calibration of monitors and scanners
  • Experimental 3D support for NVIDIA video cards

Some other features of this release:
  • New way to install Fedora via the Internet
  • Authentication for users using SSSD
  • NFS updates
  • Zarafa Open Source Edition, a new open collaboration package
  • System Btrfs System Rollback
  • Improvements in SystemTap handlers
  • Python 3 stack that can be installed at the same time as an existing Python stack
  • Support for the full Java EE 6 specification in Netbeans 6.8
