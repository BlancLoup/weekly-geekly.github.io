Firefox extension: search open site

On many sites it is not easy to find the necessary information due to the fact that site creators often forget (or cannot) make the site search function. It is a fact. I have developed a simple Firefox extension that solves this problem.

The idea is simple - use search engines (Yandex, Google, etc.) to search the site. Two options are possible:
  1. Select a fragment of text on the page -> in the context menu, select "Find ... on this site"
  2. Write the search text in the standard search form -> press Ctrl + Enter or click "Search"


After that, a new search engine (which is selected by default) will open in the new tab with a list of pages of the current (open) site, on which there is a search text.


The extension has been tested and works completely with Yandex, Google, Bing, Rambler, AOL and Baidu. Working with other search engines is possible, but not guaranteed.

You can download the extension from the official Mozilla website .
