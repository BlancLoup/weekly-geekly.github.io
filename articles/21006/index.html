New social toolbar for Firefox

The first public version of the plug-in for uniting social networks has been released.

Now, going to the page of any user bestpersons.ru in any social network and clicking on the red P button, you can see:
  • All sites on which this user is registered
  • His friends
  • A single user message feed from all sites
  • Single message feed of friends
And you need this in order to simplify life and be aware of the network life of your friends, regardless of the service they use.

Go to install the extension. (after installation, restart Firefox, open View-> Toolbars-> Customize, and drag the icon bestpersons.ru to the browser panel)
