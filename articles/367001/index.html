Turris Omnia - a powerful, open and secure home router



It is hard and sad to look at modern routers, especially expensive fresh models. Buying a device for 20,000 rubles, the owners first of all advise to change the firmware to an alternative one due to the factory instability, and some have to put up with the one that is. The vast majority of manufacturers of home network devices do not put security first, fix vulnerabilities for a long time, or even remove the model from support rather quickly.

The Czech guys from CZ.NIC, which gave the world a wonderful routing daemon BIRD, which is used, including in the Moscow traffic exchange point MSK-IX, Knot DNS DNS server, promoted DNSSEC since its inception and one of the first to launch internationalized domain names, we decided that it was time to put an end to all this - you need to make a sufficiently powerful home router that would not choke on gigabit traffic, with a lot of ROM and RAM, so that it could run all the necessary software, including LXC and Docker containers with full ennymi OS, which would stick to the principles of Open Source and Open Hardware and updates automatically, without human intervention, once the developers have closed the vulnerability. Moreover, the authors support honeypots, which simulate routers with vulnerabilities, on their servers to track the latest threats.

The router is running Turris OS - OpenWRT fork. The authors promise to backport the changes to the original project.

The router is currently only possible to pre-order Indiegogo for $ 189 in the form of a full-fledged device and $ 99 in the form of a board without antennas, a power supply, Wi-Fi adapters and a cooler. The device will be available only in April 2016. At the moment, the project has collected 89% of the planned money, although it has not passed a day.
