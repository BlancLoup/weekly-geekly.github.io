SugarJS - Syntax Sugar for JavaScript



What is SugarJS?


SugarJS is an open source (MIT license) Javascript library that extends native objects with useful methods. It is designed to be an intuitive, unobtrusive tool that enhances the expressiveness of code that allows you to do more with less code and think less about the routine.

In fact, they have a very informative and simple site - where everything is already well written.
I'll just try to highlight the main points.

Briefly...

  • Extends native Javascript elements with useful, intuitive methods.
  • Easy to understand and to use.
  • The SugarJS method is not used when there is a method in the native implementation of the browser.
  • It has a testing system with a huge set of tests.
  • Expands the possibilities of working with date and time, provides their formatting in several languages ‚Äã‚Äã(there is Russian).
  • It gets along well with other frameworks and third-party code.
  • Modularity , you can include just the right packages.
  • Fully ECMAScript compliant.
  • Declared support for all major browsers, including mobile.
  • Supports Node.js.
  • It has good and convenient documentation.
