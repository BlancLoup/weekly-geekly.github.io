Andreessen Horowitz estimates half a billion Github

According to information close to the management of the Andreessen Horowitz venture investment fund, sources from the site PandoDaily.com, GitHub will soon begin the first round of financing. The exact amount and terms of the transaction are not yet known, but according to preliminary data, the company at this stage will be estimated at 500 - 800 million dollars. Until now, Github developed independently, without resorting to the help of investors. Now there are fifty-five employees.

1.6 million users and 2.8 million repositories are registered on GitHab. A paid subscription for individual users costs from $ 7 per month, while prices for the Enterprise version installed inside the corporate network start at $ 5,000 per year. Large companies spend up to a million dollars a year on Github services.

No official comments from GitHub have been received yet.
