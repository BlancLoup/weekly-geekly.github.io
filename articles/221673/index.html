Chromium Transmission Integration

Hello. Today I want to present you an extension that can integrate the popular Transmission rocking torrent into a browser based on Chromium.

Initially, it was a port of a similar extension for uTorrent. But the process of porting api was so simple that the first beta was made after 8 hours.

When I thought about whether or not to adapt the extension for Transmission, I looked at the chrome stor and, to my surprise, did not find anything like it there.
Well, someone had to do it.

The extension is a pop-up window, with current downloads, a graph of speed and a small number of buttons.

The columns in the table are able to change the size, they can be added / removed, interchanged and so on. It is possible to set a speed limit, turn on the bug, select list filters, add torrent files, magnet, url. All you need is there.

The list of files is divided into directories, in which it is convenient to go and choose what you need to download. Columns are also customized to taste.

In general, I showed you everything.

Expansion settings are simple. Of the features I want to show the ability to make a context menu that allows you to add new downloads immediately to the desired directory.

The result was a simple but irreplaceable extension.

Sources are on GitHub , and you can install the extension from the Chrome Web Store .
