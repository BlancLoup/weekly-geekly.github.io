Skype for Symbian is now real!

The developers of Skype have finally coped with the task that many owners of Symbian phones have been waiting for for so long.

From today, the native version of the Skype client for Symbian has become available for download .

Let me remind you that earlier for Symbian phones Skype offered the so-called Lite version, the possibilities of which were greatly curtailed. For example, it was possible to make a call only through a GSM network, and then not in all countries.

The new Skype for Symbian allows you to make direct voice calls directly via the Internet when you connect to Wifi or 3G.

The following phones are currently supported:

Nokia E71, Nokia E63, Nokia E66, Nokia N82, Nokia E51, Nokia N95, Nokia N95 8GB, Nokia N81, Nokia N81 8GB, Nokia E90
Symbian S60 3rd Edition Feature Pack 1 devices

Nokia N96, Nokia N85, Nokia 5320, Nokia 6210 Navigator, Nokia 6210 Classic, Nokia N78, Nokia N79
Symbian S60 3rd Edition Feature Pack 2 devices
