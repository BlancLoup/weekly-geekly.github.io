We invite to the webinars for developers to work with the platform Nokia X

Hi, Habr!
Continuing the theme ‚ÄúNokia X for Developers,‚Äù we bring to your attention a brief announcement of a series of webinars about our new platform, available tools and new features.



Webinars in Russian will be conducted by our friend, the training partner of Nokia, Mikhail Samarin, who previously did detailed guides for us with the Nokia Imaging SDK toolkit (http://habrahabr.ru/company/Nokia/blog/214363). All the details and dates can be found under the cut.

MARCH 19. NOKIA X: INTRODUCTION AND INSTRUMENTS FOR DEVELOPERS


As part of this webinar, you will find a detailed overview of the Nokia X platform. Michael will talk about its features and demonstrate how to install and configure the development environment.

You will also learn about the process of publishing your existing Android applications in the Nokia Store, as well as the Nokia APIs features: Nokia Notification Service, In-App Payment and HERE Maps.

Check in.

MARCH 25. TESTING FOR COMPATIBILITY WITH NOKIA X PLATFORM


This webinar will help to transfer your Android applications to the Nokia X platform.

You can learn about application testing for compatibility, as well as tools such as the Nokia X Analyzer and Remote Device Access - RDA. With them, you can quickly analyze and evaluate the required costs of porting existing applications and decide which APIs can be replaced with Nokia equivalents.

Check in.

9 APRIL. IN-APP PAYMENTS FOR NOKIA X PLATFORM


The following webinar will help you integrate In-App Payments into your applications for the Nokia X platform using Nokia Publish.

Check in.

APRIL 17. HERE MAPS FOR NOKIA X PLATFORM


HERE MAPS is an important feature for developers planning to launch an application on the Nokia X platform.

With HERE MAPS, functionality will be available for your applications at the level of already existing Android mapping solutions combined with the additional exclusive features of HERE services.

Check in.

APRIL 29. NOKIA NOTIFICATIONS API FOR NOKIA X PLATFORM


The Nokia Notifications API is also available on the Nokia X platform. As part of this webinar, you will learn how to integrate this service into your applications for Nokia X and port existing solutions to Nokia Notifications APIs.

Check in.

MAY 8 CREATING GAMES FOR NOKIA X PLATFORM WITH UNITY


Unity is perhaps the most popular medium for developing 2D and 3D games simultaneously for many operating systems.
In this webinar, we will talk about how to create awesome games for Nokia X devices. If you are already the creator of Unity games, learn about the specifics of this platform and what you need to consider when porting.

Check in.
