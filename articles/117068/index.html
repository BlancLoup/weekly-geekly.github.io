Microsoft Research Accelerators



Research Accelerators is a set of tools for students, graduate students, teachers and scientists. They will help in studies, academic work, the study of science, data analysis and visualization, as well as the publication of the results of the work done. Many of the tools come with open source or have open interfaces for interaction and extension. All this makes it possible to work with the most advanced developments of engineers and researchers of the Microsoft Research group.

Structure of a set of Research Accelerators



Chemistry Add-in for Word

Chemistry Add-in for Word allows you to simplify the procedure for entering chemical formulas and images into Microsoft Word documents. It makes it possible to create a separate ‚Äúchemical zone‚Äù in the document, specialized for the input and processing of chemical information (this includes special editing, rendering and data export capabilities).



Article Authoring Add-in for Word

Article Authoring Add-in for Word allows you to enter a large amount of metadata related to copyright materials when working with Word documents. Able to save documents in NLM (National Library of Medicine's) format.


Conference XP

Conference XP is free software created for video conferencing for remote learning and other collaborative remote work scenarios. Integrating high-performance audio, video and network technologies, ConferenceXP conveniently integrates a large number of people away from each other into a flexible interactive environment, ensuring high productivity of collaboration.

Creative Commons Add-in for Microsoft Office

Creative Commons Add-in for Microsoft Office allows you to add a Creative Commons license to Microsoft Office documents, which is generated on the fly using user-defined parameters.


Dryad and DryadLINQ

Dryad and DryadLINQ is a high-performance distributed data processing engine designed to simplify the implementation of distributed applications in a Windows cluster environment. DryadLINQ allows developers to create applications in managed code using a special extended version of LINQ and an additional API.


GenePattern Add-in for Word


GenePattern Add-in for Word is a biological software that makes it possible to model and analyze some processes in the field of gene analysis. The plugin allows you to embed the created object into a Microsoft Word document and interact with other GenePattern software directly from Word.

Microsoft Biology Foundation


The Microsoft Biology Foundation is a toolkit created as an extension of the Microsoft .NET platform that can help researchers and scientists in their daily work. This platform is free software and serves as an auxiliary tool for a large number of tasks often performed in bioinformatics. A number of universities and corporations use the MBF, which helps them to devote less time to routine operations and focus on core research.

NodeXL

NodeXL is a powerful and easy-to-use tool for visualizing and analyzing large networks. NodeXL uses MS Excel as a platform for graphing, analyzing and visualizing data. The tool supports a variety of plugins to import data from various social networks.


Ontology Add-in for Word

This plugin makes it easy to work with metadata in scientific and technical texts. It makes it easier to work with terms, footnotes, definitions, and other blocks of information that are not part of the main material of the article, but are often necessary for its understanding. In addition to simplifying the creation of such materials, validation tools, export to HTML, XML and NLM formats, interaction with dictionaries, working with tags, etc. are supported.


Research Information Center

The Research Information Center (RIC) is a virtual research environment created jointly by the Microsoft Research team and the British Library. Source codes of the product and installer are available on CodePlex. RIC is an add-on to Microsoft Office SharePoint Server, expanding its functionality to meet the needs of research and academic groups. With its help, a scientific project site can be quickly created and filled with calendars, task lists, wikis, blogs, personal pages of project participants and other materials.



Project Trident: A Scientific Workflow Workbench


The Trident project is designed to manage scientific workflows.
Its functions include:
  • Automate data analysis and visualization for further study
  • Creating, maintaining and cataloging experiments
  • Conducting detailed descriptions of experiments
  • Creating a library that describes typical workflows and algorithms
  • Planning workflows, including the distribution of the operating time of computer clusters
  • Coordination of the research team


Zentity

Zentity is a platform that provides modules, utilities, and services that can help you create and maintain a digital library (data warehouse). With Zentity, researchers can easily access the necessary information, analyze it and visually see all the relationships between the various structural elements. Zentity is based on technologies such as .NET 4.0, Microsoft SQL Server 2008, the Entity Framework, and LINQ. Data access in Zentity can be done through the Zentity API, Microsoft Live Lab Pivot Viewer (Silverlight application designed to visualize large data sets of arbitrary dimensions and their metadata) or Microsoft Research Visual Explorer.
