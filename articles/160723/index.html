Useful stuff for iOS developer # 2

Some time ago I read an interesting post dedicated to useful frameworks and libraries for iOS development. With your permission and consent, Coder89 presenting “Useful things for iOS developers, part two”.

"Helpers":
  • "NSXtensions" is a collection of Cocoa helpers (categories) for standard Apple Cocoa / UIKit classes, making them more convenient to use.
  • “RNBoilerplate” - creating templates with specified settings, frameworks, libraries and much more.
  • Appirater is a very useful library that reminds the user of the application that it would be nice to evaluate it in the AppStore.
  • “IVersion” is a set of classes that reminds the user that a new version of the program has been released and shows what changes have occurred in it.
  • “IHasApp” is a framework that allows you to get a list of applications installed on the user's phone.

UI components and frameworks:
  • "ODRefreshControl" - a new visual method "Pull Down To Refresh", which appeared in iOS6. This library works with earlier versions of iOS.
  • “GHSidebarNav” is a clone of the controller entered by facebook for side out navigation.
  • “KLSectionSelect” is a horizontal scroller from the 8tracks radio application.
  • GraphView is a library for easy graphing.
  • “IOSPlot” is a library for building line and pie charts.
  • “THContactPicker” is a cute picker for selecting multiple contacts at once.
  • “CHDigitInput” is a replacement for the standard UiPickerView, but intended for the selection of numbers, provides an accelerated selection especially in the case of large numbers (999999).
  • “PaperFoldMenuController” is a popular PaperFold used for maps, now for displaying UIView.
  • “REComposeViewController” - repeats the functions of SLComposeViewController, which appeared in iOS6. Helps create a visually similar controller to support social networks.
  • ClassicMap is an application that demonstrates how to overlay standard maps with maps from other sources.
  • “SDWellSegmentedControl” is a replacement for the standard UISegmentedControl repeating control from the AppStore of the application in iOS6.
  • "MGBox2" - the creation of stylish tables, grids with a minimum amount of code.
  • The HSV Color Picker is a palette for color selection.
  • BrowserTabView is a tabView controller for the browser.
  • "MTAnimatedLabel" - animated UILabel, as on the iPhone's screen.

Networking social API:
  • “HCDownload” is a download manager.
  • "MWFeedParser" - parser RSS and Atom Feed.
  • TouchXML is the replacement for the standard NSXML * classes for iOS. Designed to work with XML.
  • “EGOImageLoading” - easy and fast downloading images from the network.
  • “LASharekit” - helps in sending pictures to facebook, twitter, pinterest, email, also can save to photo album or add to clipboard.
  • "MIS-Linkedin-Share" - a set of classes for authorization and work with LinkedIn.
  • “ESSVideoShare-for-OS-X-Lion” - posting videos on YouTube, Vimeo, Facebook and Flickr. For iOS and MacOS.
  • “DEFacebookComposeViewController” - a controller for sending photos and messages on facebook that resembles that of ios6, only supporting c work with ios4 and 5.
  • “GSDropboxActivity” is an auxiliary UIActivity class for working with DropBox.

Audio, video, graphics and animation files:
  • “GPUImage” is a set of classes for creating filters for images or videos. The power of OpenGL 2.0 in action is powerful and useful.
  • “DLCImagePickerController” is a GPUImage based picker. Easy to use, contains built-in filters.
  • “OpenAL” is a sound engine for playing background music and a variety of sound effects.
  • “Waveform Audio Player” is an audio player that draws a sound wave of a track being played.
  • “MDAudioPlayerController” is an audio player resembling an iPod app.
  • "KOFiles" is a powerful file explorer built on the basis of UITableView.
  • "PTShowcaseViewController" - also allows you to view directories and files, but is made in the style of Springboard. When you release an application with this library - remove the rounding of icons (otherwise apple will not miss).
  • Accordion is a file navigation assistant that replaces UINavigationController.

Game engines:
  • Cocos2D useful tools - many useful things for Cocos2d.
  • Torque2D is an engine for creating 2d games for iOS.
  • “UDK” is a free version of Unreal Engine 3 for mobile development, 3d rendering, film making, etc. It has a very powerful optimization.
  • Unity3d is an ecosystem for creating games. Not as powerful optimization as UDK, but easy to use. Supports many platforms.
  • "GameSalad" - a program for creating 2d games that supports most platforms. Very easy, so the possibilities are not endless.

In-App Purchases:
  • “EBPurchase” is a simple framework for implementing in-app purchases in an application.

Testing:
  • "MBFaker" is a useful library ported with Ruby for generating random data.

All for now. Share your frameworks and libraries.
