Surprise from Microsoft, or who is PCBP for Russia



In the Windows 8 Apps Store for each game, you must specify the minimum age from which it can be played. Also there is an opportunity to download certificates that confirm the specified age. For some countries, such a certificate is mandatory, that is, without it, the application cannot be distributed in the specified country. More recently, Russia has been added to Korea, South Africa, Brazil, and Taiwan. Now to distribute the application in Russia, you must provide a PCBP certificate.

But how? There is no such certificate in Microsoft documentation .
