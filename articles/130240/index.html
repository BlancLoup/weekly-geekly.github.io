Hello!

The account with the domain domain_name.ru exceeds the load limit for the master plan.

In order to avoid blocking the account, it is recommended to change the backup plan to a more resource-intensive one, or limit the load on the server by optimizing the code.

About using system resources:
https: //my.firstvds.ru.manager.myftpsite.net/? domain = domain_name.ru

For technical issues please contact support@firstvds.ru, for payment issues please contact clients@firstvds.ru.

We wish you success!

- Respectfully,
Administration FirstVDS
