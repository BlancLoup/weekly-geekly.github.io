Intel Acquires HTML5 Toolkit for Mobile Application Developers


Intel is announcing the acquisition of mobile application development tools developed by the AppMobi startup team. The list of programs and components going to Intel is as follows:
  • XDK (Cross platform development kit) IDE is the first HTML5 based mobile application development environment. Includes tools for creating, debugging and building software, as well as an emulator of devices on iOS and Android;
  • PhoneGap XDK - development environment with full support for the open source framework PhoneGap;
  • GameDev XDK - HTML5 gaming application development environment. As an example, AppMobi has already released an HTML5 game called Boom Town;
  • jqMobi and jqUI, now referred to as the Intel App Framework - a framework for implementing jQuery on mobile devices;
