Java Article Contest (a Java-based article contest)

Java Article Contest




Write a super article on The Code Project to win a trip to the JavaOne conference, which will be held in San Francisco from June 2 to 5, 2009. The cost of a pass to such a conference costs about $ 1,895 USD. Also, you will be paid up to $ 1,500 USD for transportation costs!

The article should use the language and / or the Java platform. Articles will be accepted until April 30, 2009. For writing, you need to use the standard format of articles on www.codeproject.com .



At the first stage, the articles will be evaluated by The Code Project community. Further, the highest rated and popular articles will already be evaluated by a jury elected by the administration of The Code Project.

The main requirement is Language: Java and / or Platform: Java.

It is forbidden to participate in the competition to residents of Quebec, as well as the following countries: Cuba, Iran, Iraq, Libya, North Korea, Sudan and Syria. So the list of those who are allowed to participate is quite extensive.

Naturally, the article should be written in English.
