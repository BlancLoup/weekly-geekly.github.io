Hosting Control Panel - WebsitePanel

WebsitePanel - a panel for managing open source Windows hosting. Supports managing multiple servers, allows you to manage web sites, FTP accounts, databases and applications from a unified interface.

The WebsitePanel project is the development of the DotnetPanel project in a new non-commercial incarnation with open source. The full source code is available on the SourceForge website and can be used to refine the panel to the needs of a particular hosting provider and to integrate with other tools used by the hosting provider.

The WebsitePanel panel is designed for:
  • hosting providers offering windows hosting services
  • administrators of companies responsible for managing web servers
  • developers hosting websites for customers or managing servers of customers
Of the remarkable features of WebsitePanel can be noted a number of possibilities.

Support a large number of solutions of different versions

Operating systems: Windows Server 2008 R2; Windows Server 2008; Windows Server 2003 R2; Windows Server 2003;
  • Web servers : IIS 7.5; IIS 7.0; IIS 6.0; ColdFusion 7/8/9
  • FTP servers : MS FTP 7.5; MS FTP 7.0; MS FTP 6.0; Gene6 FTP Server; Serv-U FTP Server 6.x; FileZilla FTP Server
  • Mail servers : SmarterMail 2.x - 7.x; MailEnable 2.x - 4.x; Merak Mail Server 8.x - 10.x; MDaemon 9.x - 10.x; hMailServer 4.x; ArgoMail Server 1.x; Ability Mail Server 2.x
  • DBMS : Microsoft SQL Server 2008 R2; Microsoft SQL Server 2000/2005/2008; Microsoft SQL Server 7.0; MySQL Server of 4.0.x, 4.1.x and 5.x; Microsoft Access 2000
  • DNS servers : Microsoft DNS Server; Simple DNS Plus 4.x-5.x; ISC BIND DNS Server; Nettica DNS Service
  • Statistics services : SmarterStats 3.x - 6.x; AWStats Statistics
  • Group Services : Microsoft Exchange Server 2010 SP1; Microsoft Exchange Server 2007/2010; BlackBerry Enterprise Server; Windows SharePoint Services 3.0; SharePoint Foundation 2010; Office Communication Server 2007; Microsoft Dynamics CRM 4.0
Integration with web application gallery

The web application gallery allows the user to install popular web applications on the server with a few mouse clicks, configure the web server and configure the databases for the web application to work.

A list of web applications is in the Web Application Gallery itself .
