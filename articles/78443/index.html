Bookmark All Tabs - Chrome built-in feature, alternative to Session Manager with sync

Just yesterday, I wrote about a very convenient extension for Chrome - Session Manager.
In that topic, two comments were made: 1 and 2 . In this post I will tell you how to implement this, and without plug-ins.

There is such a feature in Chrome - Bookmark All Tabs (Make all open tabs tabs). This feature creates a new folder in the bookmarks panel and adds bookmarks to all open tabs in this folder. It is called in two ways:
  1. Right-click on any open tab - "Bookmark all tabs ..."
  2. Ctrl + Shift + D

After any of these options, you will have a window for creating a folder in bookmarks. Enter her name - voila! All is ready :)
