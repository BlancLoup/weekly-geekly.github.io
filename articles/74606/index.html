Alchemy. Hello from the past

What are we talking about?


Alchemy is a DOS game that aims to create a new civilization.
A world in which everything can be built from water, earth, air and fire.



The game develops memory and trains thinking.
She is inherently brilliant, I hooked myself very tightly and for the second evening I spend on her passing :)

Here you can do whatever you want.
The main task is to study the best of the emulated world and create a new civilization.

At the beginning of the game we are given the basic elements: Fire, water, air and earth.



Combining them, we move to new stages of development:



The beauty of the game lies in the fact that there are an unusually large number of such combinations and their number grows as you progress.
You can develop in different areas: chemistry, flora, fauna, physics. Develop technologies (nuclear energy, chemical industry, etc.)

Hints:

- Not all elements can be combined
- Sometimes you need to use more than one element to create a reaction.
- There are different ways to get the same items. :) Some are shorter, others are longer.
- Each open substance can be destroyed or something new can be created from it (Nuclear energy is an exception)
- To pass the game, you need to develop in all directions.
