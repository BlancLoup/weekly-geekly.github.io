Absolute Rings: Examples of Alternative Mice

The computer mouse was shown to the world in 1968, and has since undergone a number of changes. This is a convenient and practical manipulator, but it also has flaws. A number of alternative input devices aimed at replacing the mouse are aimed at combating them.



Ring shaped ultrasonic mouse


In 2006-2007, the students of the Worcester Polytechnic Institute WPI made an ultrasonic mouse. A set of sensors tracks the movement of hands in space. The ring on which the device is fixed is its battery.



Masterkit


In 2011, the Master Kit Company offered the MT6022 finger mouse to the Russian consumer. Now it can be found in the store for about five hundred rubles. But this mouse is a wired one, you have to plug it into a USB connector.
