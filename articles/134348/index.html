Silverlight 5 Released!

Microsoft released Silverlight 5. Support will be made until 2021.



The Silverlight 5 installation file weighs less than 7MB and is installed in 10 seconds. It added a lot of tasty and healthy.

New features in Silverlight 5 include hardware decoding of H.264 formats, which provides a significant performance improvement when decoding unprotected content using a GPU; in the Postscript vector printing, the quality and size of the file was improved, as well as the 3D graphics stack improved, which is used in the XNA API on Windows platforms to get low-level access to the GPU (for drawing vertex shaders) and low-level access to 3D primitives. And in Silverlight 5, the normal 'Trusted Application' model was added (enabling through registry policy and application certificates; multi-window support, access to COM, file system, P / Invoke support for existing native code that runs directly from Silverlight is supported in the browser) .
