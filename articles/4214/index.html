Today Blogus undoubtedly provides the most objective picture of the Russian blogosphere. This is, first of all, the construction and indexing of the base of blog posts available in the rating. And with this information you can solve the main stated goal of the project: to find and filter really useful and interesting content of different thematic focus. The very first steps in this direction have already been taken - this is the ability to categorize blogs using tags, as well as search prototypes for tags inside posts.
