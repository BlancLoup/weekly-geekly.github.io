Lumia SensorCore SDK out of beta testing

We and Microsoft‚Äôs blog have already talked about the Lumia SensorCore SDK and applications using this toolkit. With the use of the relatively new SDK, such popular (within the framework of the WP8.1 platform, of course) and interesting applications, such as fitness trackers MSN Health and Fitness and Fit Buddy , Trip-O-Matic cycling travel assistant, hiking, have already been created app LEKI Trek and Ski and many others.



As a development of the Nokia Motion Monitor idea (feature was implemented in the Lumia 1520), Lumia SensorCore first appeared as a public beta in June of this year. And this week, Microsoft Mobile Oy officially announced the release of the SDK version 1.0, which means the completion of beta testing of the new toolkit.

Recall that the real-time technology SensorCore, is used to collect and store information about user actions. And this is not only the measurement of the number of steps, but also more detailed statistics of user movements during the day. For example, technology understands when a person is walking or running, and also takes into account the time spent on rest.

All this data is accumulated and then transferred to applications using this technology. SensorCore technology is optimized for power consumption, so application performance will have minimal impact on battery consumption. Other distinctive aspects include respect for free memory, secure storage and quick access to information, wide adaptability for various applications.

Lumia SensorCore interacts with the built-in geolocation APIs of Windows Phone 8.1, which allows you to keep statistics on the user's stay, to distinguish home from work.

As we said earlier, the technology is available only for the new Lumia line, and more specifically for the models: Lumia 1520, 930, 630/635, 830, 730/735 with installed updates for Lumia Cyan and Lumia Denim. All documentation and sample applications can be found at this link .
