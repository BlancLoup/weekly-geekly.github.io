KOffice 2.1 - Release

After 6 months of development, a new version of KOffice 2.1 office suite has been released. This release contains many fixes. Particular attention was paid to the support of ODF. Regarding important changes in specific applications of the package:
* KWord (word processor): a new implementation of the tables, added the possibility of detailed tracking of changes in the document;
* KPlato (project manager): implemented dependency editor, view statuses and schedules;
* Karbon (vector graphics editor): a framework has appeared for developing plug-ins that implement effects for SVG, added support for importing PDF;
* Krita (raster image editor): added new brushes, emulation of pressing force with the mouse;

In addition, a component for editing formulas (Formula). By order of Nokia, import and export to MS Office formats (.doc and .ppt) was significantly improved.

This release is still positioned as a preliminary and is not recommended for everyday use, with the exception of the Krita and Karbon applications that have been declared stable.
