Linux Mint 14 Nadia



The new version of Mint comes with two graphical skins to choose from - Mate 1.4 (fork GNOME 2, supporting all its functions), and Cinnamon 1.6 (fork GNOME 3), which focuses on 3D acceleration. As you can see, the reasons for many holivars (unity) are not here.

The new release contains new versions of the same desktop (and Mate, and Cinnamon). And the latter has a much larger list of updates (mainly due to the file manager, notifications, the workspace, and so on). Updated and program manager.
