So, everything you wanted to know about Red Hat Cloud, HP Edition!

In May, Red Hat, together with HP, released a special edition of its private cloud design reference architecture - Red Hat Cloud, HP Edition. In this architecture, the cloud is built on the basis of Red Hat Cloud, HP CloudSystem and the special template HP Cloud Map. Red Hat Cloud, HP Edition includes:
  • Red Hat Enterprise Linux with Hypervisor
  • Red Hat JBoss Enterprise Middleware
  • Red Hat RHN Satellite Update Tools
  • HP CloudSystem Matrix Private Cloud Platform Powered by HP BladeSystem
  • HP Cloud Maps for JBoss Enterprise Middleware maps (these ‚Äúmaps‚Äù are ready-made templates for automating the provision of resources when deploying JBoss in the HP CloudSystem private cloud)

Unlike the standard Red Hat Cloud Foundations, the HP Edition provides the HP Matrix Operating Environment software for the provision of all physical resources (servers, storage and network equipment), which is deployed on a separate management server. The same environment is also used to automate the deployment of Linux on bare servers (in the standard Red Hat Cloud Foundations architecture, you need to install Linux on servers manually).

HP's service division provides consulting, survey, design, and training services for Red Hat Cloud, HP Edition customers.


A full description of the Red Hat Cloud, HP Edition architecture is available on the Red Hat website , and a description of the HP Cloud Map template for the JBoss Enterprise Middleware is available on the HP website .
