WordPress 3.1 Reinhardt Released


The long-awaited release WordPress! WordPress 3.1, codenamed Reinhardt, is named after jazz guitarist Django Reinhardt. Version 3.1 is available for download , or you can update it from the admin panel.

Among the novelties of this release is a redesigned workspace that allows you to get quick access to frequently used sections of the administrative part. In the top panel there are some rarely used menu items that sometimes frightened novice bloggers. You can remove this panel and return to the classic look in the settings of your profile in the admin panel by clicking on your username in the top right admin menu.

In this release, there are a few pluses for developers, including new note formats that allow you to create your own styles for different types of posts, like Tumblr. Among the new features - an archive for the contents of different types; new admin panel for network management sites (Multisite); a completely redesigned import and export system, as well as advanced management capabilities for tags, rubrics and custom fields.

With release 3.1, WordPress has become more than just a CMS, as it was before. The only thing that limits you is your imagination.
