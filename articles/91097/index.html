USBDetect 3.0: NSA Network Flash Detector

Specialists of the US National Security Agency have developed a program USBDetect 3.0 Computer Network Defense Tool , which is able to detect all USB flash drives in the local network. Information about this development was not advertised, just accidentally leaked into the budget statement of the NSA.

The tool scans the registry data of Windows machines ( PDF ) and collects information about all the connected storage media, including external hard drives, flash drives, flash cards, etc. The tool was created to close the biggest security hole in organizations, since the main security threat is its own employees.

Since the program already has the third version number, we can assume that this is not the first such development, and similar utilities are already being used in the competent authorities. According to indirect evidence (see the PDF link above), the security services learned how to remotely detect flash drives about two years ago.
