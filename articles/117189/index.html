New version of Image Composite Editor released

A new version of the Image Composite Editor (ICE) is available - a free program from Microsoft Research, which is designed to process images and create panoramas of various types based on a set of photos. The created panoramas can then be uploaded with a few clicks to the hosting of large-sized images PhotoSynth.

Some details about ICE can be found in these entries: here and here .

The new version adds new very interesting features:
  • formation of panoramas based on video files;
  • automatic correction of image vignetting defect during image composition;
  • A new blending mechanism, i.e. image adjustments at the edges during composition;
  • the ability to set the size of the disk cache and the used ICE memory;
  • perspective projection mode for wide-angle panoramas;
  • consumption of disk space is reduced by one third when creating a panorama;
  • ICE now works better with corrupted metadata in images.
The official announcement of the new version can be read here . Download the Image Composite Editor at the following links: ICE for 32 bit Windows or ICE for 64 bit Windows .
