Conjoon - Personal organizer for Zend Framework and ExtJS 2.2

Conjoon is a web-based personal organizer with a special focus on messaging. For the greatest practicality, ease of use and general perception of information, the system is built on ZendFramework and ExtJS .


For my part, I will say that I threw gmail.com , FeedReader and MS Outlook into the trash. Now only Google Chrome + Conjoon !

Specifications:
  • Mozilla FireFox > = 3.x
  • Internet Explorer > = 7
  • Safari > = 3.x
  • Google chrome
  • Opera > = 9.x

Equipment


  1. An installer that installs and, if necessary, updates your organizer to the latest version.
  2. ZendFramework and ExtJS libraries.
  3. Conjoon application itself

Slave place


  • Menu for quick access to the necessary data
  • Status bar with system messages or prompts and progress bar
  • The data is displayed in sub-tabs.
  • Bookmarks menu for quick navigation between open tabs or modules


Quick Launch


  • Quick editing of letters and contacts
  • View recent emails
  • view new items from your feed

Mail module
  • POP3 , SMTP protocol support
  • UTF-8 encoding
  • Ability to move ( drag & drop ), rename, delete, create folders:
    • Inbox
    • Spam
    • Outgoing
    • Sent
    • Deleted
    • Shopping cart

  • Ability to hide / show folders
  • Ability to hide / show email preview
  • move the viewport to the left or bottom of the window
  • Create, save, edit mail
  • Visual Text Editor
  • The ability to move mail to folders by dragging and dropping
  • Mail Labels
    • Last
    • Read / Unread
    • Spam / Non Spam
    • Answered / Forwarded

  • Automatic search for the recipient in the personal address book during letter creation
  • Smile support
  • Automatically receive mail in customized accounts
  • Support for signatures in the letter
  • Support (theoretically) endless email accounts

RSS feed module


  • ATOM / RSS feed support
  • Automatic loading of recent records
  • Read / Unread Labels

Requirements


  • Apache with mod_rewrite module and support for .htaccess files
  • PHP > = 5.2.5
  • MySQL 5. *
