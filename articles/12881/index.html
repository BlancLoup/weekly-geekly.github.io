Mobile GMaps: World maps on your mobile



The Mobile GMaps project continues to evolve. This is a free mobile phone application that displays information from services such as Google Maps , Yahoo! Maps , Windows Live Local (MSN Virtual Earth) and Ask.com Maps , as well as satellite imagery.

Mobile phones with Java J2ME are supported, as well as Windows Mobile and Palm smartphones and communicators, if Java is installed on them. MGMaps can connect to the GPS module via Bluetooth or use the built-in GPS receiver to automatically track the user's current location on the map. The next version of MGMaps is marked 1.35.00, it is still in development, version 1.33 is considered a stable version.

The new version announced such changes as:



- Spread display (horizontal mode)
- Storage of cards - the cache format has changed, now each file can contain several card elements, which improves the use of memory cards, and on some phones - RAM memory
- Storage of cards - the user can browse the MGMapsCache folder
- Support maps from OpenStreetMap
