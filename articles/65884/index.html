Today, the number of Firefox downloads will exceed a billion


So, at the time of writing the topic, the number of Firefox downloads was 999,838,863, according to the counters installed on the Spread Firefox website and the corresponding Twitter page . Every second, an average of 20 users become Firefox owners.

It is expected that today, July 31, the counter will reach a significant figure of one billion downloads.

The calculation covers all released versions of the browser, starting with the first release in 2004. If one user downloads multiple copies for different computers, they all count. If a user manually downloads an update to an existing version from a website, the meter reading also grows. But the automatic update does not affect the number of downloads.

Thus, it is far from a fact that a billion people use this web browser. Nevertheless, gaining popularity is evident by an application that was nothing more than a few years ago and nowadays directly competes with Internet Explorer, which is included by default in Windows.
