5 useful tools for mobile developers



Author: Grigory Kharchenko - Head of Mobile Development, Kokoc Mobile (Kokoc Group).

App annie

Specializes in marketing analytics for mobile applications. The service scans all products in the world's largest markets and displays the ratings of their categories by country. Therefore, any developer the next day after the publication of the application can find out what place it occupies in a particular rating. What is important, App Annie allows you to track not only your own, but also other people's applications: when the release took place, what place is in the tops, how often it is updated, how much money it brings, in which countries it is popular.



Git

Code Version Controller. Thanks to this tool, developers can look at all the changes that have been made to the code, and, if necessary, return to one of them. And since all information is stored on the server, access to it can be obtained from anywhere in the world.



Testflight

A service popular among iOS application developers. It was created by an independent team, and for some time it functioned as an independent program. In early 2014, it was bought by Apple, and now it is a native tool integrated into iTunes. TestFlight helps test iOS applications before publishing them on the App Store: to use, simply register the product and click the "invite testers" button and add their email addresses. As a result, the beta version of the application will be able to remotely see and test not only developers, but also customers.



Redmine

Application created to manage tasks. It allows you to conduct several projects, notify about their changes using RSS feeds and e-mail, track errors, take into account time costs. What is important, Redmine has a flexible role-based access system, and new users can register themselves.



Parse

A service that developers use to send Push-notifications on both iOS and Android, notifying users about product news and maintaining interaction with them. The tool allows A / B testing of Push notifications, selecting those in which content works most efficiently, analyzing the success of a push campaign, using a cloud-based data storage service.
