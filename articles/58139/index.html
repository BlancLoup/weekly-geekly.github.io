Google improves online viewing of TIFF and PowerPoint files

For several months, Google has already added a quick view of PDF files right in the browser. Now the same viewer supports TIFF files and Microsoft PowerPoint documents: thus, the user can view these files directly in the browser. There is no need to save files on your computer, there is no need to buy or install any special viewing software.



At first, the Slideshow mode was implemented only for PowerPoint files, but now this technology is used to view PDF and TIFF.



In addition, new features give the user more control over the document:
  • it can be increased or decreased;
  • select text to copy and paste;
  • "Print" the presentation in a PDF document.

In addition, the new version does not require Flash installed in the browser.

Similar to presentations, TIFF files can contain several pages, however, while standard TIFF viewers on a computer usually show only the first page. Therefore, the online viewer will show the user all the pages of the document and provide an opportunity to ‚Äúprint‚Äù TIFF, opening it in the form of a printable PDF.
