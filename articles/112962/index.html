Prefixes for Microsoft business critical applications

So, you are facing a Task. Or even Tasks. Imagine that they are the deployment of an automated decision-making system, corporate mail to several thousand addresses or the consolidation of dozens of databases in a virtual environment.

All life, all life experience rushes before your eyes. The management is already ready to hear from you about the many months of implementation and about the fact that the estimate for the project will have to be changed in the process of work.

And then you decide to go the other way: the deadline is one day. Estimates are as accurate as a surgeon's blade. Ovation, prize, rightful place on the board of honor! And here you wake up ...

In general, this is not a fantasy at all. More recently, HP, together with Microsoft, invented a series of turnkey hardware and software solutions for key business-critical applications.

The first set of consoles in this series was the HP Data Warehouse Appliance for data warehousing, which is over 500 TB in size, optimized for SQL Server 2008 Parallel Data Warehouse. It consists of HP ProLiant DL360 G7 or DL380 G7 dual-rackmount rack-mount racks housed in one or more 19-inch racks, an HP StorageWorks P2000 G3 modular disk array, and HP Networking network hardware.

In early January, the HP Business Decision Appliance for decision-making systems was also released, which is deployed within one hour (in the photo). Its hardware is based on the HP ProLiant DL360 G7 with Xeon 5650 processors, 96 GB of RAM, and 8 300 GB SAS internal hard drives. It installs Windows Server 2008 R2 Enterprise Edition with a Microsoft SQL Server 2008 R2 Enterprise Edition database and SharePoint 2010 Enterprise Edition.

And in the coming weeks, HP will begin shipping solutions for the HP E5000 Messaging System for Microsoft Exchange Server 2010, which supports between 500 and 15 thousand mailboxes. Upgrading from previous versions of Exchange to Exchange 2010
with this thing will take a matter of hours.

Later this year, HP will launch the HP Database Consolidation Appliance, which can be used to consolidate hundreds of SQL Server 2008 R2 databases in the Microsoft Hyper-V Cloud virtual environment.
