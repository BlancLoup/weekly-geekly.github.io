Joomla! 1.5 is stable!






Joomla developers introduced the first stable version of Joomla 1.5 to the community! This version is a new generation of content management system Joomla CMS. For two and a half years that have passed since the start of development of this version, the developers have done a great job. Of the most significant changes, it is worth noting a new software platform that provides ample opportunities for developers of extensions, support for UTF-8, a more flexible system of patterns and many improvements in the system interface.


What's new in Joomla 1.5?



The new version of Joomla certainly looks much tidier and, I think, more understandable for end users. For example, the developers finally removed two visual editors from the material editing dialog. Now the entire text of the material is edited in one window, and is divided in the visual editor into an announcement and the full text by inserting a separator tag. Work with images has become much better, work with lists of materials has become more pleasant (which costs only one sorting).
