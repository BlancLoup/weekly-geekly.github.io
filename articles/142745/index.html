Webinar Announcement: How and when Metro style apps for Windows 8 are executed


Metro style apps for Windows 8 have a different life cycle from their usual desktop applications. They cease to be executed, almost immediately, as soon as the user stops seeing them, they can be activated in several different ways.

How to perform tasks for which a variety of services running in the background are used for desktop applications? The system provides special mechanisms to perform certain tasks necessary for the application, when the application itself is not executed, will allow to update the tiles and display the status of the application on the lock screen. The system provides special mechanisms for messaging applications (RTC).

During the webinar, we will try to dive into the world of Metro style apps for Windows 8 and find out
how and when the Metro style app for Windows 8 is executed.
