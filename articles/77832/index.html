CMS for flash sites: FlashMoto CMS

The development of a very interesting flash site management system has recently been completed. This system is called FlashMoto CMS. I suggest to get acquainted with the main functions.

FlashMoto is all you need to manage a full-featured web site. You can add an unlimited number of pages. Using simple tools you can easily add, delete and edit pages on the site. The system supports work with all standard browsers. Advanced WYSIWYG page editor. Editing a Flash site is as easy as editing documents in Microsoft Word. FlashMoto CMS allows you to upload images, videos and SWF files.

In order to get acquainted with the main advantages of the system, you can view the demo version - www.flashmoto.com/online-demo

While the system is in English, but soon I think there will be localizations.
