GoogleTalk, part II (ICQ transports & some tips)

In the previous part, I began to talk about the attempt to migrate from one IM client to another, namely from Pidgin to GoogleTalk (the writing took place simultaneously with the study of the subject).

Well, congratulations, migration was a success!

If you remember, only a few problems remained unsolved:
  • ICQ / AIM integration, i.e. how to teach GTalk to communicate with ICQ-contacts
  • the creation of "multiline" messages How to perform a carriage return without sending a message.
  • Separately change the font size in the roster and in the conversation window

But I did not want to retreat and it was decided to crush the crutches mentioned yesterday in the form of Psi and jabber transports. Everything turned out to be not so scary, albeit a bit dreary.


Short instruction:


  1. Download Psi jabber client
  2. Register your GoogleTalk account in it (more info on how to do it )
  3. In the Psi main menu, find the Server Discovery item and click it.
  4. To drive one of the servers represented here in the Address field in the Address field (here you have to be ready to sort out a few, I earned somewhere fourth in a row, I took it at random) and enter my UIN and ICQ password
    UPD: Carefully choose your transport for ICQ, otherwise you may experience various problems:
    • if the server is normal, then in the GTalk roster the contact will look like it looks in the ICQ contact list
    • if the server is a curve, then it is not known how it will look at all (problems with encodings may appear, nicknames will be displayed incorrectly, etc.)
    • if it's completely crooked, the contact will be added and will look like UIN@icq.jabber.bla-bla.bla and you will have to manually rename the entire multi-thousand contact list, but this is not a pleasant activity.


  5. After connecting, right-click on the server and select Register
  6. Wait a bit - and voila - the Psi roster is smoothly filled with our interlock contacts. Authorize contacts and close Psi - now it's time to launch GoogleTalk
    In GTalk, affirmatively respond to the server authorization request from p.4. And you can safely begin to organize contacts, rename them more comfortably and at the same time get rid of dead souls (as I did today *))) )

Some hot keys


  • Shift + Enter - (ha ha!) Makes such a desired carriage return

  • Shift + Esc - closes the window with the current conversation
