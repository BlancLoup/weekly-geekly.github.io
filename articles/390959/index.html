Raspberry Pi 3 went on sale



Exactly four years after the start of sales of the very first version of the Raspberry Pi Model B, on the same day, February 29, the Raspberry Pi 3 Model B was released .

The cost of Raspberry Pi 3 - $ 35, exactly the same as the Raspberry Pi 2.

  • Processor: 4-core 64-bit ARM Cortex-A53 at 1.2 GHz (~ 10x Raspberry Pi 1 performance)
  • Memory: 1 GB LPDDR2-900 SDRAM
  • Built-in 802.11b / g / n and Bluetooth 4.1
  • Full compatibility with Raspberry Pi 1 and 2

Especially for Raspberry Pi 3, Broadcom has released a new SoC BCM2837. It has the same architecture as its predecessors, BCM2835 and BCM2836, so all the projects and manuals that took into account the exact hardware architecture of the Raspberry Pi remain relevant.



The form factor of the board remained unchanged, all old cases fit here, which facilitates the upgrade.

A new specially designed 64-bit processor in 32-bit mode provides an increase in performance of 50-60% compared to ARM Cortex-A7 in Raspberry Pi 2.

All connectors remained in the same places and retained their functionality, and the board can still be powered by a 5-volt microUSB adapter. 2.5 A is recommended, with the expectation of connecting voracious USB peripherals.

The location of the LED indicators has changed - their place was taken by the WiFi antenna. And instead of the buggy spring slot for SD-cards, they put a normal one.



Raspberry Pi 3 is sold today in element14 and RS Components stores, and other resellers.

The latest images of NOOBS and Raspbian are taken here .

During this year, they promise to release Raspberry Pi 3 Model A for $ 20, and Compute Module 3 will be released within a few months.
