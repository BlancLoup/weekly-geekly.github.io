Aspia: a program for collecting and displaying information about the system

I want to present to the public a program for collecting and displaying information about a system called Aspia . The program is an open analogue of the program Everest. The author seeks to catch up and overtake it in functionality.

Aspia is a free and free tool for collecting and displaying system information, including network settings, hardware, and software. The collected data can be saved in an HTML report.

The program is written by lentin , one of the ReactOS developers for the Windows and ReactOS operating systems. At the moment, the program is in development, respectively, it may have errors.


Program features

Equipment

  • DMI data
  • Central Processing Unit (CPUID)
  • Information on ATA devices, including SMART data
  • Monitor Information
  • Power supply (power supply parameters, installed batteries / batteries)
  • Installed local and network printers

Software

  • Installed programs and updates
  • System services
  • Device drivers
  • Running processes

Network

  • Network adapters
  • RAS connections
  • Shared resources (network resources)
  • Routes
  • Internet Explorer settings (including browsing history, cookie information)

operating system

  • Registration information (product key, etc.)
  • Desktop options
  • Task Scheduler (information about scheduled tasks)
  • Local Users and User Groups
  • Regional settings
  • Environment Variables
  • System folders
  • Installed Fonts
  • Event Viewer
  • Autoload applications
