Comparison of services for monitoring client sites or insurance for SEO paranoia

A typical situation, you are promoting the site and instead of the planned positions in search results you get strange results. But you did everything right? In the process of analyzing the incident, go to the client's website and discover completely different content. You cannot swear with a client, we love him, but you really want to be able to control what the client does on his website. After all, it is you who are to blame for the poor optimization results. Once again in such a situation, I decided to automate the monitoring of changes on sites and describe how I chose a tool for this.

Task


Find a program that can automate the monitoring of content changes on a third-party site. The program should be easy to use, quickly and accurately identify changes and clearly show them.

Selection criteria


Only web services participate in the selection, because I want to have access to information wherever there is Internet. Those services that do not possess any unique properties that simplify the solution of the problem are excluded from the selection. To determine the winner, each service will be evaluated according to the number of points scored - zero (bad), one (normal), two (good).

Indicators for which we will score points:
  • Easy to add site for monitoring.
  • Convenience of working with data, if the site has a lot of changes.
  • Clear notifications about changes on the site.
  • The ability to collect additional and useful information that tells about the life and condition of the site.
  • Restrictions on the number of monitored pages within one account.
  • Cost (ideally, you want a free solution or for money, if the price is adequate).
  • Support. If an error is found, I want to be sure that my problem will be solved quickly.

Selection


Shangedetection.com

The most famous service for monitoring changes on web pages launched in 1999. The project has survived a decade due to its amazing simplicity. You specify the URL of the page for which you want to track and the e-mail address to which the reports will be sent. If the service detects a change, a notification is sent to the mail. In the service itself is available visualization of the changed parts of the page. In your personal account it is possible within one account to specify multiple pages for monitoring at the same time. Often the service is very strange reveals a change on the Russian-language sites, carefully look at the screenshot.


An example of the display of changes on the page at hangetection.com

Pros:
  • Simple interface.
  • Visual information about changes on the page.
  • Free.

Minuses:
  • The address of each page of the site for monitoring must be specified manually. If you need to follow the company's website or even worse with an online store, then adding URL links to start monitoring is reminiscent of masochism and 7 circles of hell.

Site

Followthatpage.com

Similar to Shandetection.com, but there are 2 key differences:
1) Changes on the page are not shown in the interface, but come to email. The implementation is controversial, since the visibility of such letters is low, but not if you live in an email client, you might like it.
2) The service itself can offer URLs to be added to the monitoring based on the link you specified, but this is implemented very superficially, since Followthatpage simply pulls out all the addresses it receives from html.

Pros:
  • Clear interface.
  • Automatic suggestion of addresses based on the specified link.

Minuses:
  • Free of charge, you can follow only 20 pages on one account.
  • Spammed e-mail if the site changes frequently or monitors multiple pages.
  • Information about changes to the pages is not clear.

Site

SORGE.pro

After working with SORGE, a feeling of flying on a spacecraft is created. Differences:
1) The service monitors not only the changes in the content, but separately tracks changes in the site structure, meta tags, leads its direction finding and checks the status of the domain with WHOIS information.
2) This is the only service that not only offers URLs for monitoring based on the link you specified as Followthatpage.com, but has its own robot to index the specified site. It works very simply, you specify a link to the main page of the site and SORGE indexes it completely within 1-2 days, presenting the site in the form of a relevant structure. Tracking changes in the structure of the site is carried out separately, which allows you to learn about the facts, if the client himself will demolish or add several pages. At the same time, if the page is added, the service will not just find it, but will automatically begin to track further changes to the content on it. The very same indexing is easily customizable.
3) The changes in the content are not as visual as those of Shangetetection.com, however, it is also convenient to work with them. It is worth noting that any changes on the site for all types of information collected are saved in the system, which makes it possible to view the history of these changes. This is convenient when changes to the site were made not immediately, but in a long period of time.

Pros:
  • Better automation and simplicity for monitoring the whole site, rather than individual pages.
  • Own robot for indexing with the ability to customize.
  • Convenient work with large amounts of data (sorting, displaying the history of changes).
  • Collection of additional information (site structure, meta tags, direction finding, domain).
  • Detailed documentation and support in Russian.

Minuses:
  • Fully paid. There is a 14-day test period.
  • It is better to start working with the service by reading the documentation, since there are a number of not obvious points in the interface.

Site

Femtoo.com

Similar to Shangetetection.com, but remarkable for the possibility of conveniently tracking changes not on the whole page, but on a specific part of it (price, paragraph, table). Similar functionality exists in SORGE, but the implementation of Femtoo.com is much more convenient. Just as in SORGE, there is support for a change history, but it is available only on a paid account.

Pros:
  • Simple tracking of a separate part of a web page.
  • Users support.

Minuses:
  • The free version is very limited in functionality (up to 3 monitored pages).
