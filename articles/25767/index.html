The first mobile report Opera Software

As promised, publish more information from the Norwegians. This time - regarding the mobile market. It should be noted that the company makes up a similar report for the first time.

In total, the number of Opera Mini downloads is more than 45 million, while in the first quarter of 2008 alone, the number of users of this browser increased by 26% compared with the previous quarter. In March, about 12 million Opera Mini users viewed over 2.4 billion pages, about 184 pages per month per user, and the total size of the generated traffic exceeded 33 million megabytes.

Additionally, the report presents a dozen countries in which Opera Mini browser is most actively used. Russia tops this list, surpassing other countries in almost all indicators (only in the number of appeals to entertainment portals is inferior to Ukraine).
