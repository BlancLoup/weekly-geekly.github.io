Mirc 7.1 released



Good news for fans of the good old irc-client.

Mirc 7.1 released!

This is the first stable release of the new Unicode version of Mirc.

The creation of the Unicode version of Mirc took almost two years of development and testing; It took up to 150 thousand source code edits. This has been the most difficult and time-consuming update of Mirc since its creation in 1995, when it was still a non-Unicode, 16-bit, Windows 3.1 application.

Many parts of Mirc had to be updated or rewritten.

MIRC should now be faster, more stable, and more compatible with the latest versions of Windows, and should also be better able to cope with the display and input in various languages.


In addition to converting to Unicode, there are several other important changes:
  • Improved compatibility with Windows 7
  • Stable work with INI files
  • Faster display / scrolling of text
  • Speech support
  • Improved interface design for a number of functions.
  • Automatic text color correction
  • Hide / show menu bar with ALT button
  • Support for italics and other font styles
  • Optimized SSL procedures
  • SSL connection icon in the status window
  • IPv6 support
  • Improved portable client support
  • Full channel history
  • Digital signature of executable files
  • Automatic channel / private message sharing
  • Full screen when you press F11
  • UPnP support
  • Many bug fixes
  • Much more

The full history of Mirc 7.x development starting with version 7.0 can be found on the official website.

Download Mirc 7.1 from the official site.
