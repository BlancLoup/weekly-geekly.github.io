Ubuntu 10.04 Alpha 1 Exit

Yesterday, the first alpha version of the LTS release of Ubuntu 10.04 Lucid Lynx was announced on the mailing list.
One of the most significant innovations in Ubuntu 10.04 Lucid Lynx is the removal from the HAL system, and replacing it with DeviceKit in combination with udisks and upower. It was also decided not to release the release on Debian Unstable. Kernel version in Ubuntu 10.04 Lucid Lynx - 2.6.32
