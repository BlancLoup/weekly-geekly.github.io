Comparison


As I said, AdvoCam-FD3 itself costs 7,990 rubles. What else can be found in
about the same money?

One of the main competitors, in my opinion, is the Q2 CARD for 6,900 rubles. To his
unequivocal benefits include a large number of camera settings, several
options for the duration of the saved videos and USB host (you can insert a USB flash drive).
In addition, KARKAM is able to protect a file from one-touch loop erasure,
that in AdvoCam-FD3 is done only through the menu. Disadvantages: modest grade with
only one mounting option - on the windshield. Viewing angle 120 degrees against
160 for AdvoCam-FD3, the battery capacity is half as much (2 hours of autonomous video recording
against four for any AdvoCam), and the operating temperature starts from 0 degrees
Celsius (AdvoCam operates at -20 degrees of frost).

Also, AdvoCam-FD3 can be compared with DOD F900LHD (price - 8,600 rubles). Kit
the supply of the latter is really good and there is little inferior to picking
AdvoCam-FD3, although it also offers mounting only on glass. Among the obvious
advantages - motion sensor in the frame. But the body is much more modes SPORT
and CCTV is not, the battery will last only 1.5 hours of battery life. One more
a noticeable flaw - stopping shooting for about 1 second each time during
saving one file and creating another.

The general disadvantage of KARKAM and DOD - without exaggeration, a large number of fakes
in the same package and with the same name on the market, which are easy to run into,
making a purchase over the Internet. The design of the same AdvoCam, as far as I know,
unique, there are no devices for copying it yet.
