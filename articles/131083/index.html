Volkswagen Group uses IE6 as standard

Until now, some large companies retain IE6 as a corporate standard. Comes to ridiculous. On the website of the world's largest car manufacturer Volkswagen Group published an informational message that sounds like an anecdote:

The Volkswagen Group uses Microsoft Internet Explorer 6 as the standard for displaying web pages. For daily work with our web platform, we recommend that you also use Internet Explorer 6.

By installing the current version of the browser and updates from Microsoft, we guarantee you a secure connection.

For security reasons, the Volkswagen Group does not recommend using browsers such as Mozilla Firefox, Opera, Netscape, etc., because they are inferior in security.

The HackerNews forums rated the valuable advice of Volkswagen security experts and suggested that this text was written more than five years ago. It is surprising that so far no one has revised the corporate policy in the field of browsers and web development. A typical example of the sluggishness of a large corporate car, and Volkswagen is not alone here.
