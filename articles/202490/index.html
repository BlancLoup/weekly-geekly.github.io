How I went to World Usability Day

Do you still think that usability is some kind of secret function or designer's skill, multiplied by the adequacy of the manual? I was sure of this before visiting the WUD in Moscow .



The first speaker dmitrysatin dispelled this myth. It turns out that human-centered design is a process that is very clearly and modernly described in ISO and (suddenly) in GOST . This means: usability is not a phantom. This is what you can talk about, explore, improve.

For example, in the discussion on Lean UX, an amazing moment sounded for me: each user story is a hypothesis that goes through a certain verification procedure.
