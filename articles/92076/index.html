CloudMade Navigation supports maneuver restrictions

Not so long ago, CloudMade identified several priority areas, among which was navigation. It was decided to create a special project Navi Studio that would combine several smaller services and allow them to be used to create full-fledged navigation software. Navi Studio includes:
  • Navigation engine
  • Vector Stream server
  • Local Search
  • Geocoding
Work has begun to boil, and several applications using this project have already appeared. But there is no full-fledged navigation without adhering to the rules of traffic rules, and therefore this issue was also not given enough time. As a result, starting from Friday (April 23), the CloudMade navigation engine supports OSM data navigation taking into account maneuver restrictions ( turn restrictions ). Edit the map and enjoy the navigation!

Immediately after the launch, small defects were found, which are now in the editing stage, but in general, routes are built very well and give excellent results in urban environments.
