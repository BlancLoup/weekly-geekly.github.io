Mozilla is working on Firefox for Windows 8 Metro



Many manufacturers are starting to create software for the new Windows 8 Metro. Among these manufacturers and Mozilla, whose developers are working on a new version of the browser Firefox. According to the information provided by the developers, the new browser will be as functional as the regular version of Firefox, but fully support the touch interface. Actually, support of the touch interface will be the main feature of the Metro-version of Firefox.

Despite the enthusiasm of the developers, the work will be quite painstaking, so that the finished software can be seen no earlier than the second half of 2012. Actually, there is nowhere to hurry, since by the second half of this year the final version of Windows 8 will be ready. The developers are preparing two types of touch browser at once. The first version is for mobile devices, more precisely, tablets running Windows 8. The second version is for ordinary PCs with a touchscreen display.

As mentioned above, there will not be any significant differences in the functionality of the Metro version of Firefox compared to a conventional Mozilla browser. The new browser will work both in full screen mode and in 1/5 and 1/6 mode of the screen. And yes, as far as can be understood from the comments of the developers, the first version will appear for Windows 8 x86 / 64. After that the tablet version of the new browser will be released.
