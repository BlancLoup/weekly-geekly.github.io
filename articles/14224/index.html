Google servers

Here is a list of Google server names used for various company services that are returned in HTTP headers. Not all servers are assigned obvious names, and some of them may even show interesting data (for example, ctcserver is used for the not yet launched Google Call service referenced by robots.txt). Google Web Server is a modified version of the normal Apache server running on Linux.
