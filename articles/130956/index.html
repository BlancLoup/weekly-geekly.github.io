Dropbox API



Dropbox has announced the release of the API for mobile devices and web applications!

What API allows:

Safety enhancement

You will not need to enter your personal data (login and password) into third-party applications. Thanks to the API, you only need to confirm the use of your Dropbox account (via the mobile app or via the web interface).

Application folders

Adding support for applications that can only read or save in 1 folder in your Dropbox account. You can rename or move this application folder anywhere in your account, and the application will continue to work as usual.

More supported platforms

The API was developed for mobile applications. However, improved web application support has been added! It includes new tutorials, documentation, and SDKs for Python, Ruby, and Java developers.

The API is available to developers at https://www.dropbox.com/developers
