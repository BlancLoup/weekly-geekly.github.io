Release mc-4.8.2

The console file manager Midnight commander version 4.8.2 is released.

Short list of changes:

Main:
  • in the file search dialog, a tilde (~) is used to indicate the home directory in the 'Start at:' field;
  • Added command line option -X (- no-x11), designed to process Ctrl, Alt, Shift without Xserver;
  • Support for dynamic resizing panels;
  • Long file names can scroll horizontally within the panel (ScrollLeft, ScrollRight key bindings).

Editor:
  • Added highlighting of * .as files (as.syntax).


Viewer:
  • Added key binding for reverse search.


Miscellanea:
  • Simplified by mc.menu - removed LZMA | LZ and p7 changed to 7z;
  • Updated list of known browsers: gnome-moz-remote mozilla firefox konqueror opera;
  • Added environment variable MC_HOME to specify the home directory MC;
  • Produced significant code cleaning.


And also a lot of bug fixes.

Full list of changes is available in the official announcement .
