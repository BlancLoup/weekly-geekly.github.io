(Part 1) Turn the PSP into a remote control and gamepad for Windows

Are you a happy owner of a Sony PSP console? I will tell you how, at no extra cost in just a few minutes, you can also get a remote control for your computer. And after spending a little more time, you can get a great gamepad.




To implement this venture in two ways: cheap and relatively inexpensive.

In the first case , the console will work through a mini-usb cable. Obviously, it is not necessary to count on ‚Äúswitching music from the sofa‚Äù ( although, of course, you can also forward an extension cord =) ). But it will turn out quite comfortable gamepad.
For the second case, we will additionally need a Wi-Fi router ( good, every third person has one ). Thus, your console will become full, and the gamepad - even more convenient.
Note: it does not matter if there is WiFi on the computer itself. You can also connect to the router using a regular twisted pair. The main thing is that the computer is in the same network with the PSP.

I use PSP Slim & lite, a laptop with Windows 7 on board and a D-Link DI-524 router.

Step 1: Install all the necessary


We will need :
  • Playstation Portable console with custom firmware (for example, 5.03 Gen-C).
  • Windows 7 / Vista.
  • Mini-USB cable for connecting the console.
  • Wi-Fi router (for the second method)
  • PSPdisp program

Actions :
  1. We connect the PSP with a mini-usb cable.
  2. Run the PSPdisp installer.
  3. Installation is performed with the default parameters.
  4. Confirm the installation of unsigned drivers.
  5. During installation, the program will detect the console and ask for permission to copy the utility to the memory card. If this does not happen, specify the path to the folder X: / PSP / GAME manually.
  6. Finish installing PSPdisp.

Step 2: Setup


  1. Disable the PSP from the computer.
  2. Turn on the PSP.
  3. Go to the "Games - Memory Stick", run PSPdisp (should be the first in the list).
  4. The program should issue this:
  5. Click the triangle. Enter the internal ip-address of your computer (for example, 192.168.0.3).
  6. Choose a connection method. If you choose USB - insert the cable back into the PSP and the computer. If WiFi - the standard console connection dialog will appear.

For WiFi connection:

  1. In the Windows system tray, we find the PSPDisp icon, right-click. In the menu, select "Options - Use wireless LAN".
  2. In the menu, select the item "Enabled". The program will inform the internal ip-address of the computer and ask you to connect via it.

Step 3: Final Stage. Check


  • Click the right button on the PSPdisp tray. Choose "Control - Mouse emulation".

  • We check the performance of our frauds: move analog. The cursor should move. On the PSP screen should be displayed part of the image near the cursor. "Cross" emulates a right-click, "square" - the left.



In principle, any person who knows English at least a little can further configure the program. In a pinch, there is Help.
