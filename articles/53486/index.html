KeyDB: Portable Password Manager

I'm a big lover of password managers, because I always forget my passwords. The reason for my forgetfulness is the fact that I always come up with very complex passwords that will be hard to crack or solve.

One of the great features of password managers is encryption and portability. Password manager is required to encrypt, because additional security never hurts, and portability, because the password manager is good enough to take it with you everywhere. I travel a lot and use different computers, so I want my passwords to always be with me.

KeyDb is a small and simple application that includes the above features.


Installation


Not required. KeyDb is an exe file with a weight of 1.14 Mb. I would not call it portable if it were not. The small size of the program allows you to take it with you anywhere, even on your old flash drive.

Create a password file


KeyDb stores passwords in various files. You can store all passwords in one file or create several files for several users or different purposes. To get started, simply click on the ‚Äúfile‚Äù menu and create a new password file. This window will appear:



After the file is created, proceed to add passwords.

Add passwords


All you need to do is click on the plus icon in the upper left corner of the window. A window will appear in which you can enter all the password information.



The program has some default categories, but you can add as many categories as you like to better organize the storage of passwords. You can also use a password generator to generate a very strong password if you like.
You can use the URL of the pages, and KeyDb can launch your default browser to go to the desired page, so that you can easily enter your password. You can also add notes to save more detailed information about the site.

Ready application



Once you have downloaded all the passwords, your KeyDb will look like this:



You can filter and sort passwords to help find the right password.
