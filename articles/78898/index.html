New original CMS for stores Melbis Shop 6 Freeware requests reviews

The company Melbis has completed the development of the new version of the sixth version of the program for creating online stores and puts it to the public:
www.melbis.com/lab

The main tasks that we set were the following: to maintain a powerful and fast windows-interface while ensuring multi-user system with the most flexible division of rights.


The system is not yet sold, but is available in one single edition - free. We invite all webmasters to test the system and offer a bonus system for it. Of course, we are now primarily interested in your opinion about the architecture itself.

To facilitate the acquaintance with the program, we recorded video tutorials that will tell you about new conceptual points.

Highlights of Melbis Shop 6:

The software platform for creating an online store Melbis Shop 6 has a multi-user architecture and consists of a windows-program in which the store's product base is formed. The program contains a built-in storefront development environment (SDK) and allows you to add or purchase additional function modules in PHP and HTML.

The main feature of the Melbis Shop 6 software package is the use of the windows-interface, which is still popular. Its advantage is that in addition to convenience, it allows you to significantly speed up the work of filling the store with new goods, and therefore significantly reduce the costs of the owner of the online store.

The sixth version was tested on fairly large product bases, up to 100,000 products. The architecture of the program is designed so that you can work with a small, only necessary part of the data at this moment.

Understanding the requirements and business processes of large stores, we implemented the program so that there was a possibility of a clear separation of commercial and non-commercial data among employees.

An inseparable standard for any CMS to create an online store is the presence of a data caching system, which is undoubtedly present in Melbis Shop 6.

We will be very grateful to habrovchanam for the reasoned criticism, advice on improvement and opinion on development. In the end, among the usual Web CMS appeared so different, designed specifically for developers of such systems, and for store owners.
