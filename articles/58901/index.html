7-Zip 9.03 Alpha - a new version of the archiver with LZMA2 support

About the program:



The next test version of the free, domestic 7-Zip archiver number 9.03 has been released. The program uses its own algorithm for packaging in 7Z format with a high degree of compression. In addition, it can pack in ZIP, GZIP, BZIP2, TAR formats and extract from ZIP, CAB, RAR, ARJ, LZH, GZIP, BZIP2, Z, TAR, CPIO, RPM, DEB archives.

The file manager cannot boast of significant functionality and convenience, but integration into context menus is supported, so packing and extracting files does not require a large number of manipulations.

It is distributed freely, with open source codes, and its format algorithm for 7z is used in others, both commercial (PowerArchiver) and free archivers (TUGZip). 7-Zip, probably, is the only program of this kind, which is translated into a huge number of languages, including exotic ones.

The LZMA2 compression method is now supported (allows full use of multi-core system resources), XZ format archives are implemented, tools are added to extract data from partition archives with NTFS, FAT file systems, VHD images, and MBR boot sectors. Work with GZip, BZip2, LZMA, XZ and TAR archives has also been expanded. In addition, the work of the file manager has been improved, new keys for working from the command line have been added, and minor bugs have been fixed. This release fixes minor bugs, and also improved the 7z archive format update system.





Key features of 7-Zip:



Supported formats:
Completely: 7z, ZIP (except WinZip 11 versions), gzip, bzip2, tar.
Partial (unpacking and viewing only): CAB, RAR, ARJ [2], cpio, RPM, deb, (ar), Z, LZH, Compiled HTMLHelp, SPLIT, Windows Installer, NSIS Installer.
Very high compression ratio in the new 7z format due to the use of the improved Lempel-Ziv algorithm.
For ZIP and GZIP formats, the compression ratio is 2-10% higher than that of PKZip and WinZip.
Ability to create self-extracting archives for 7z format.
In the 7z format, it is possible to create multi-volume archives (currently, except for self-extracting)
The ability to encrypt archives, AES algorithm with a key length of 256 bits (for 7z format).
Integration into the shell of Microsoft Windows and Windows NT.
Plugin for the program FAR Manager.
Plugin for Total Commander and Unreal Commander.
Multilingual graphical user interface (for Windows only) with dual-window file manager functions.
Powerful command line version.
There is a version for 32 bit and 64 bit systems.
Port under * nix p7zip
