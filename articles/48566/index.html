Officially released Windows 7 Beta

Today at the CES conference Steve Ballmer announced the beta version of the Windows 7 operating system. For everyone, it will be available for download before the end of this week, happy owners of MSDN and TechNet subscriptions can download it right now! Here is a good installation guide .

Usability, performance, work with devices - there are many possibilities, you can read more about them in the Russian translation of the blog of the Windows engineers. And of course, opportunities for developers . The PDC conference site contains a whole set of hourly reports on various aspects of creating applications for Windows 7. On MSDN there is a detailed document Windows 7 for Developers . In the near future, we plan to provide more material in Russian.
