As mentioned above, when you write a long article, it infuriates that the most important socket in the editor (H, B, I, U, etc.) goes up. To rein in her, do the following:

  1. Install the Stylish extension (for Google Chrome) or any other extension that allows you to apply custom styles to any site.

  2. In the browser go to chrome: // extensions / , find there Stylish and click "Options".

  3. On the left, click "Create style", in the window on the left, score the following code (you can change, play and watch how everything changes) 

    .ochko-editor__toolbar { position:fixed; background-color:black; top:10px; margin-top: 20px; z-index: 10000; width: 780px; }

Enjoying the result: now the main editing panel is tied to the browser window and does not go up.
