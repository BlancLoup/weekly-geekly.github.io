New webOS-based Palm Pre smartphone

Palm announced a new Palm Pre smartphone running the new webOS operating system at CES 2009.


Device characteristics


Body Type : Slider
Operating system : webOS
Screen : 3.1 "resolution of 320x480 pixels
Keyboard : QWERTY
Camera : 3 megapixel with LED flash
Communications : 3G EVDO rev. A, WiFi, Bluetooth, GPS
Connectors : micro USB, 3.5 mm headphone output
Dimensions : 59.5x100.5x16.95 (WxHxD, mm)
Weight : 136 grams
Features : wireless charging, multi-touch screen with support for gesture recognition, 8 GB of internal memory for storing user data.
