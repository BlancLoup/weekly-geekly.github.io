What awaits us in the Chrome Web Store


Earlier, we already reported on the launch by Google of its own Web Store, and now there is an unprecedented activity in the Chromium application development branch. Hundreds of developers are in a hurry to prepare their web applications for the launch of the Web Store, which may take place in conjunction with the release of the much anticipated Chrome OS.

We were able to look behind the scenes and touch a few apps from the Chrome Web Store to tell you about some interesting developments that will be ready for use in the next few months. But we warn that all applications are still in their most rudimentary form, and as developers adapt the code to the HTML5 standard, we can expect significant improvements.

To run the following applications you have to download and use the latest version of Chrome 6. Instructions at the end of the article.
