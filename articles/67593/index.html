New build opera. United by choice

While Opera Unite is still being finalized to a fully operational state, the assemblies are divided into two categories: Opera 10.0 and Opera 10.10, respectively, without Unite and with it. The release of the beta version of Opera 10.10 is scheduled immediately after the release of the main version 10.0.

If you already have the opera:config#AutoUpdate|DownloadAllSnapshots option and you have already installed the previous builds, today Opera should upgrade itself to version 10.0.
For some reason this didn't work on my Linux. I guess this is only for assemblies without Yunite.

For those who want to continue to test Ynayut there is an opportunity to download the build version 10.10, for which auto-update will also work later.

Also of the major changes, it is worth noting that Opera should now work with FreeBSD 8, but so far there are no builds for 64-bit FreeBSD 7/8. Also, the guest viewing button in the photo gallery service does not work for private / private access rights.
