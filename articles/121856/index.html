Mikogo 4.0 - a utility for screen sharing, webinars, online conferences and remote access

Habr edited Andrew Donnelly from BeamYourScreen GmbH ( Mikogo.com ) with a press release to tell about the new version of the Mikogo application for screen sharing, webinars, online conferences and remote access.

I will try to state the official press release in my own words, talking about the functionality of the new version.


To yourself a webinar or Free Screen Sharing + Remote Desktop according to Mikogo version - to familiarize those who have heard about Mikogo for the first time.

So.

First of all, I am glad that the free application developed by non-Russian specialists abroad has been translated into Russian (and into many more others). This is a minus one barrier that impedes the development of the program by a wider audience.

Secondly, in this version, attention is paid to interaction and communication between program users:

  • Chat appeared. It's just a sin, using the program to share the desktop, in parallel to use Skype or ICQ to communicate in the process of sharing. But it was in the third version. After all, people connect, introduce themselves, want to ask questions, get answers.
  • Multiplayer board . Those. several users can draw, write, mark important on one board, and all connected users will see the changes.
  • "HTML Viewer" . This is a cool feature: you send an invitation to join the session with a link.
