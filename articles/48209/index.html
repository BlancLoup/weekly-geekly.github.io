Editor TopStyle Pro resurrected - meet version 4 Beta 1

Many web designers and developers most likely know that TopStyle Pro is one of the best CSS editors for Windows OS. The editor was created by Nick Bradbury, the one who created HomeSite at one time. But TopStyle was abandoned, and was not updated for several years until a different developer bought it back a month ago.
Briefly about what brought the first beta version:
  • Unicode (UTF-8 or UTF-16)
  • Live FTP Editing (editing files via FTP)
  • HTML Toolbar (like in HomeSite)
  • Code Folding for comments in CSS and HTML, PHP and ASP inserts in HTML, JavaScript functions, as well as while, try and switch
  • Bookmarks to navigate through the code
  • New HTML templates for ASP JavaScript, ASP VBScript, ASP.NET C # and ASP.NET VB
  • Updated HTML Tidy
  • Improved appearance of the editor under Windows XP and Vista
Editor's site: topstyle4.com (requires open port 8080)
The site of the previous version .
The beta version is free and will run until April 2009.
