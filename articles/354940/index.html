What is RAID on NVMe SSDs?
You can compare the incredible performance of such a system only with a RAM disk. A RAM disk is when a piece is cut from the RAM and used through special applications as a regular disk in the system. The problem with RAM disks is that when you reboot, all the data on them disappears and they still remain extremely expensive. Once this problem is solved by SSD disks, and with the proliferation of their use with the PCI Express interface, the need for RAM disks and the interest in them has almost completely disappeared.
