Tiggr: to help business analyst


Hello! I would like to promote the development of the company ( Exade l) in which I work, because I am also involved in this project.
So, Tiggr is a tool for prototyping and creating user interface layouts.
I'll start by listing the main features of the application:
  • A decent set of controls. It is worth noting that in Tiggr recently, in addition to the desktop interface elements, UI controls for mobile phones have appeared.
  • Tiggr is an online application, so access to your project can be obtained from any workstation without installing additional software - just a browser.
  • Customizability - all controls have additional settings that allow you to control the view, position, etc. component.
  • A set of containers for the formation of the layout.
  • Convenient (in my opinion) approach to working with pictures.
  • Collaboration support: several users can simultaneously work on one project, layout.
  • From static screens you can get a dynamic, clickable prototype.
  • Manage users and access to the project.


I'll tell you about the main points in more detail.

Controls


Pallet web, desktop controls looks like this:

Mobile control pallets, which is activated if you select Screen Properties> Screen type> Mobile on the property panel.

Remarkably, some of the controls are dynamic. I will explain with examples: different tabs on the Tab Panel can contain different content, while in the layout mode you can switch between tabs; the calendar opens and closes; combobox opens.
As for the properties, for example, buttons, they are quite enough for basic manipulations. I would like to focus on such properties as Fit and Round Corners. The first property is used to stretch the controls along the edges of the container, the second rounds the corners of the element.
gotiggr.com/views/assets/asset-1720

Layout


For the organization of the layout, the location of the UI elements on the page, the Tiggr developers offer a good set of containers. For containers reserved for a separate pallet. Pallet includes:
  1. Panel - the panel dialog box.
  2. Vertical, Horizontal, Flow box - a container that has elements inside it vertically, horizontally and in order limited to the right edge of the container.
  3. Grid Panel - allows you to build a table grid.
  4. Tab Panel - a panel with several tabs.

In addition to horizontal, vertical and ‚Äústreaming‚Äù positioning, absolute positioning is also provided, since as the element was ‚Äúthrown‚Äù, so it will be located irrespective of the container and other elements.

Collaboration support


So how can Tiggr business analytics? The key feature of the Tiger is a collaboration. In real time, several people can view and edit the same layout. In order to add users to the project you just need to send them an invitation.
Quite often it happens that the members of the team working on the project are geographically located in different places. Tiggr allows you to combine them in one place, not only for discussion, but also for co-editing. From my own experience I will say that it is easier to move, say, a control to a certain place than to tell 2 minutes that it should be there and there. In addition to the work of the team itself, Tiggr can serve as a presentation "platform" in the communication team-customer. Having arranged a presentation, you can negotiate on the fly, edit the layout and do not arrange an additional session to clarify that we are on the same page after the previous rally. To improve communication, Tiggr has a built-in chat, although I used Skype to communicate and clarify requirements, since voice is always faster.
