Features:


  • In each release, we pay a lot of attention to work automation tools - this is no exception to ImageCMS 4.2. In the import-export tools, besides the previously available CSV format, support for the XLS and XLSx formats has appeared. In addition, we launched the ability to import photos: this is done by indicating the full path to the source file. Also, added more documentation and slightly improved the mechanics. Automate your work, achieve the best results with ImageCMS!
  • In the previous release, we announced a change in how user accounts work. We have changed the approach to the actual implementation of the entity and other related moments, such as a user profile, etc. In general, this opened the door to the world of rationalism in the management of user data. This time the changes affected the system of ‚ÄúDifferentiation of access rights‚Äù; as a result, it has become more flexible in terms of customizing capabilities for each user, standard groups have also been added.   • At the request of the working users of the system, we updated the ‚ÄúSocial Networks‚Äù module. Now the module is fully functional for both the Shop version and Corporate # 76000236 on http://idea.imagecms.net .
  • Launched the module ‚ÄúRedirects‚Äù. The main purpose is to redirect the end user from one page to another, using the 301st or 302nd redirect code to choose from. We are sure that such functionality will find a lot of positive reviews in modern website building and will be useful for maintaining good relations with search services. The module supports setting both manual redirects and automatic ones. For example, when a product is deleted, it forms an automatic redirect to the parent category. Thus, the user will not receive a 404 page if he gets to the deleted product from the search results or saved bookmarks. For more information, we invite you to visit the module documentation wiki page .
