We charge mobile phones with brute force

The 3DNews correspondent during CES 2010 managed to capture the fascinating process of charging a mobile phone using the YoGen Universal Mobile-device Charger. We offer you a curious video demonstrating the operation of the charger:




With YoGen Universal Mobile-device Charger, you can charge GPS-navigators, mobile phones and other portable devices, including those for charging that need a mini-USB port. The set of interchangeable adapters is included in the delivery package of the charger. It should be noted that the dimensions of the device are quite compact - 55x90x23 mm, and the weight is only 98 grams, so that it can be easily used in field conditions.
