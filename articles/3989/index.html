LinuxBIOS with graphics.

LinuxBIOS project enthusiasts (a project to create a fully open BIOS under the GNU GPL license) presented a LinuxBIOS version with the KDrive graphics server, a matchbox window manager (this window manager is used on some PDA and TabletPC with GNU / Linux), BusyBox and the rxvt terminal.
This version of LinuxBIOS is written into standard 2 megabyte flash `and. It loads the command line and the graphic manager with the terminal in two seconds in 6 seconds. Thus, in 8 seconds, the minimum operating system is loaded, which can help in recovering the crashed OS on the computer.
