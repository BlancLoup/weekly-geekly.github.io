JavaScript SIP library from the authors of the standard

I already wrote a note on this topic, since that time another JavaScript SIP library became available. In the first article I mentioned it, but at that time all the information about the library included: a description of the authors and a video demonstrating the work. Now the source code is open under the MIT license, there is documentation and a forum .

It is noteworthy that this library is developed by the authors of the same standard, which describes the features of the SIP protocol when working over WebSocket. In addition, the developers indicate the following advantages:
  • lightweight, ~ 140KB
  • simple and powerful API
  • compatible with popular SIP servers
  • audio video calls; instant messages
  • user status (this function is unique, is not yet available in other libraries)
