New build Opera 10.50 (3206/8202 / -) of January 22

This is the third build this week. We are very busy stabilizing and improving version 10.50. Also, this build includes major Presto updates that contain more than one hundred bug fixes and improvements. Next week we expect to fix most regressions in version 10.50.

Who knows, maybe a new feature will appear (function?) Next week :-)

In addition to the bugfixes, this build also includes support for the TLS Renegotiation Information Extension. This update closes a major security vulnerability in TLS / SSL protocols. Detailed information is presented on the Opera's security blog .
The Presto update also includes improved support for web fonts, including fixes for font-weight and font-style support. The Scope (?) Protocol has been updated, making it possible to improve Dragonfly.

In addition to improving the stability of the engine Oper'y, was corrected several common errors in the user interface. In particular, the completion of the address bar has received some attention. The visual modeling (drawing?) Of the element was adjusted and improved, and several individual features (quirks in behavior?) Were corrected.

- copying the link still causes crash (yes, it will be fixed)
