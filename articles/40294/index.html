Aristocrat.CMS

At the moment there are many different CMS on the market, created for the development of portals, blogs, media, online stores, etc. CMS developers often give unduly a lot of attention to auxiliary functionality, leaving the issue of configuring and implementing the system on the conscience of programmers and end users - they need to, and they understand.

The developers of Aristocrat Studio have chosen a completely different approach to creating the A.CMS system. Their product is a universal constructor with which you can build a website of any complexity. The building blocks are the universal modules of the system: a catalog of materials, a photo gallery, polls, and others.



Each section can be created on the basis of one of the standard modules of the system. To fine-tune the section, you need to set the necessary parameters on its settings page. To convince you of the flexibility of A.CMS, we give a small example. The module "Catalog of materials" allows you to create a directory of articles. If you change the format of the information, then get a news feed. Experimenting with categories and collations you can create a library.



For the visual design of sections, developers suggest using blocks - an analog of widgets from WordPress. Each block receives information from its source section and in a certain way displays it on the site. In fact, only its design is registered in the block template, and the entire logic of the program is included in the code of the corresponding A.CMS module. This was made possible through the use of the Smarty template engine.

The separation of logic and design turns A.CMS into a very flexible platform for developing and implementing web resources. A.CMS fully supports the import and export of all sections and blocks of the site, which positively affects the speed of implementation of web projects developed on its basis. According to the developers, an experienced integrator can deploy any typical configuration (online store, blog, corporate site) in 1-2 hours and implement a pre-designed design.

The guys from Aristocrat Studio have tried to simplify the work not only for webmasters, but also for the staff who will work with the content of the website. For these purposes, a client application has been developed that allows working with the data of a web resource bypassing the web interface. Such a tool will be useful for content managers and copywriters. They can fully concentrate on the visual data editor.

The separation of code and its presentation turns A.CMS into a platform for developing serious web resources. Subsequent changes to the project code will not affect its design and vice versa. Developers, in turn, receive a flexible tool for cloning and implementing similar projects.

Anyone can get acquainted with the capabilities of the system on the example of a demo site or get a free monthly license. We also recommend to download the electronic guide for the implementation of typical configurations.
