Opera 9.5 alpha performance test

New Opera 9.5 alpha , which was released on Tuesday received a lot of good reviews, many of them mentioned an increase in speed and performance.

So is Opera 9.5 faster than Opera 9.23, Firefox, Safari, and Internet Explorer?

Well, thanks NonTroppo - we have the first independent test for Opera 9.5 alpha (codename Kestrel) comparing Opera with other browsers.

The performance test is aimed at testing tasks related to the new JavaScript (ECMAScript) engine and DOM.

See results .

From the final conclusion of the result:
In general, Opera Kestrel (Opera 9.5) is already named faster than Opera Merlin and, most importantly, it surpasses other tested browsers. Some results, such as full javascript raytracer, show that Opera 9.5 is more stable and more efficient in terms of memory under heavy loads. This is important because Opera uses the same kernel for different devices, from mobile phones to desktop computers. A highly efficient core is created for less resource consumption and better scaling across devices. Opera Kestrel appeared to set a new mark in performance and speed tests.

For Mac users, there is a cheering word: Opera Kestrel on Mac is close to the Windows version (and even faster in some aspects).
