Firefox 3.6 Candidate Release Now Available for Download

The Mozilla community is proud to announce the release of the Firefox 3.6 release candidate . This preview version of the browser can be downloaded for free , it will also be delivered automatically updated for all users of the beta versions of Firefox 3.6. More than 75% of the many thousands of Firefox add-ons have been updated by their authors for compatibility with Firefox 3.6. If your favorite add-on is not yet marked as compatible, then you can help the add-on author test it using the add-on compatibility report tool . This candidate release may self-renew from time to time - and over time it will become identical to the final release of Firefox 3.6.

Firefox 3.6 (based on Gecko 1.9.2 platform ) introduces several new features for users and developers:
  • Users can change the appearance of the browser by clicking once with the mouse - with the help of Personas .
  • Firefox 3.6 warns users about outdated plugins to take care of their security.
  • Changed third-party software integration method in Firefox to increase stability.
  • Improved automatic form filling provides the best selection of options from your form filling history.
  • Open video connected by the <video> element of the same name can now be displayed in full-screen mode , and also supports poster frames .
  • The WOFF font format is supported.
  • Improved javascript performance, overall browser responsiveness and initial download speed.
  • The ability of deferred execution of scripts has appeared , which accelerates the initial loading of pages.
  • HTML5 file API is supported.
  • New CSS, DOM and HTML5 web technologies are supported.
Add-on developers and web developers should read more about the many new features of Firefox 3.6 for developers at the Mozilla Developer Center. If you want to get a complete list of changes made since the Firefox 3.6 alpha release , then look at that list (it's rather big).

So, if you want to spy on the future Firefox 3.6, then please download the release candidate and try it out !
