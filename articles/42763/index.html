New blog engine - Simple But Mighty Blog

The last six months I have been developing a new blog engine called Simple But Mighty Blog.
Today a new version was published, which is not a shame to present to your attention - 0.3.6b.

For six months, the engine was developed from version 0.1 to the current - 0.3.6b.
What gives this development? First of all, it is characterized by a simple installation that a person who is far from web programming can perform. Administering a blog is as easy to do, and an unlimited number of users can implement it, just like writing to a blog. Registration of new users is carried out by the decision of administrators. Ability to connect javascript libraries by adding one line to the engine configuration file.
