The main advantage of the technique is 100% involvement of each member of the game. LSP is not about having a great time with Lego cubes (although this goes hand in hand with the process).

How does LSP stimulate the development of new knowledge? A person is often not aware of what he knows. Our memory and the process of extracting information from it are difficult. Cubes help to stimulate this process and generate new knowledge based on existing experience.

The formula for this "magic" is as follows:

Emotions govern attention> Attention controls learning> Learning controls memory.

The LSP technique involves several steps.


  1. We set out the rules of the game and the question that needs to be answered.
  2. We build individual lego-models and stories that clearly show complex and abstract concepts. In our case - team values ‚Äã‚Äãand their practical application.
  3. We present models and stories to the team.
  4. Create a landscape, i.e. we place individual models in a common space.
  5. Create links between models.
  6. We build the general integral system.
  7. We lose non-standard situations and decisions.
  8. Reflect and extract simple guidelines.

Basic LSP Rules


  1. All get the same set of Lego cubes.
  2. The individual model should reflect your personal understanding of the question posed.
  3. Cubes should be used as a metaphor, and not as a tool for real world display.
  4. If you don't know what to build, just start building. Trust your hands.
  5. The individual model belongs to the person who built it; no one can move the parts without the owner‚Äôs consent. Make sure the changes do not change the meaning of the model.
  6. Discuss the model, not the person who built it.
