Mozilla loves HTML5 more Flash

The developers of the Firefox browser have supported Apple and Microsoft in the vision of the development of web standards. Mozilla has joined companies that believe the future is HTML5.

Although Mozilla does not go as far as Apple - Flash is still supported in Firefox - Jay Sullivan, vice president of production, said that the Adobe product will never be bundled with the browser. The problem is not specifically in Adobe, but in following the web standards, in accordance with which the industry is developing and one of which is HTML5. By opensource browser we understand the user's freedom to install scripts and applications convenient for him. And, if someone already has Flash installed, Firefox may eventually automatically update it (the feature has not yet been implemented, but will be done by analogy with Google Chrome).

A week earlier, Mozilla released Firefox 3.6.4 (Windows / Linux), which has protection against the launch of the entire browser if the Flash plugin (or QuickTime, or Silverlight) hangs. A version with such features for Mac and other operating systems is expected in the near future.

They may already be in Firefox 4.0, one step away from beta and with the official release announced in November.

Mozilla, on the other hand, would be foolish to pretend that Adobe doesn't exist. Despite the fact that more and more developers are switching to HTML5, Flash is still the standard in web video. But until recently it seemed that Mozilla adheres to the same point of view and is in the camp of adherents of Adobe technology.

So now that Apple, Microsoft, and Mozilla have declared their support for HTML5, is it possible to say that the days of Adobe Flash are numbered? None of them, however, completely from this technology does not refuse. But it would be completely short-sighted in terms of the possibilities of using web animation and video.

Nevertheless, the three main browser developers consider HTML5 to be a more promising development direction than Flash. Compared to this, even an iPhone / Flash dispute can easily seem insignificant.
