Reading pages in .mht format on a smartphone via Opera Mobile 10


Small life hacking for those that often do not have time to read all the interesting things behind a large computer, and it‚Äôs too lazy to reload the link in a mobile browser, but the html version of the page is already saved on the smartphone‚Äôs memory card. For example, in the deckstop version of line 9 and 10 of the Opera browser, by default, the web page is saved as a single file with scripts and pictures in the .mht format and it is very convenient to open this file on your smartphone via the fresh Opera Mobile 10.

1. Save all the pages we need to the memory card from the computer
2. Launch Opera Mobile 10
3. In the line for entering the address, enter two back slashes \\ and confirm the transition to the page.


The opera will show the folders of the smartphone, most often the memory card is defined as drive E (on symbian smarts)






4. Now select the desired web document and download for further reading.
