Netcraft: Microsoft halved the Apache lag in a year


According to Netcraft, an Internet monitoring company, over the past year, the share of Web sites running Microsoft products has roughly halved the gap from the Open Source Web server Apache.
In the last Netcraft survey in July 2007, more than 125.6 million Web sites were involved (compared with last month, the figure rose by 3.62 million). The results of this study showed that over the past month, the proportion of active sites running Apache has fallen by -1.13 percent, reaching 49.98%. At the same time, the number of Web sites running Microsoft products increased by 0.5%, reaching 35.53%.
Thus, at the moment, Microsoft lags behind Apache at about 14.5%, which is more than half the advantage of Apache as of July 2006 (then it was 33.4%).

If this goes on, Microsoft has a real chance to seize the hosting market. It should be noted that hosting from Microsoft has the ability to host both ASP + MSSQL and PHP + MySQL sites.
