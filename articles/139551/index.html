Lightroom 4.0 Release



Today came the final version of Lightroom 4! Only two months have passed since the announcement of the beta version and already the final release. There are not so many differences from the beta version, the main thing is that now there is an opportunity to update the catalog from Lightroom 3 and fully switch to the four.

Lightroom Developers is proud to announce the release of Lightroom 4.0.

Changes from beta:

  • Redesigned geocoding is now available in the Map module.
  • Updated and improved auto tone in the main section of the Develop module
  • Increased the range of local correction of White Balance (temperature and shade)
  • Updated processing presets plus added new video presets
  • The maximum size of Blurb books is now 240 pages.
  • More than 800 errors found and fixed! Thanks to Lightroom 4 beta users!


It is also important and interesting that for the first time in the history of Lightroom, Adobe lowered product prices from $ 299 to only $ 149 for the full version (and only $ 79 for an update).

Important Release Notes



Already found a bug in Lightroom 4 in the Books module, which can cause errors if the font Courier New and Times is selected. Please view your books in PDF before sending them to production when using these fonts.

Norton Antivirus 2012 mistakenly marks video files associated with Lightroom 4 as possible threats. Please configure exclusions in Norton Antivirus 2012 scan for Lightroom 4.

McAfee Virus Scan blocks sending email from Lightroom 4. Please configure an exception in McAfee Virus Scan for Lightroom 4.

AVCHD support is limited to MTS and M2TS video files. When transferring video to editors (such as Premiere Pro), you may need to structure the AVCHD file to process the files.

Lightroom may cause an error when printing PDFs in Windows 7. Reducing the PDF resolution in the print driver may solve the problem.
