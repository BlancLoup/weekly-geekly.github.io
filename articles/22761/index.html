10 CSS frameworks and what it is

Let's briefly understand what a CSS framework is. A free translation from the English Wikipedia reads:
This is a library that is designed to create and implement a lighter and more valid code. The general meaning of this library is similar to PHP or JS libraries, which are more widely spread and known to the masses.



It sounds tempting, these frameworks relieve a lot of routine work, give ready-made solutions that can be used and facilitate the passage of validation (not always for such frameworks, validation is not always the first).

The use of these libraries (which, by the way, is much more than ten), will certainly require you to know styles, at least basic ones. Well, how to be driven, it will be necessary to spend some time on mastering the library, but as they say - it is not difficult. There would be a desire. For almost all libraries, the basic structure (or core) is the same and simple as two kopecks. These are header, footer, menu, submenu, content, etc.

I'm more than sure that if you are a serious developer, then you probably already use the library, if not ready. then samopisnuyu. And you know how much it saves nerves, time and effort on a routine to impossible work.

Below is a list of ten libraries. I do not urge you to download and test them, just in my opinion, they deserve to be known about them. At least because of the size of the community around them and because of the number of users. Everywhere there are examples, documentation, in some places there are even video tutorials :)

  2. WYMstyle: a CSS framework
  3. YAML CSS Framework
  4. YUI Grids CSS
  5. Boilerplate CSS Framework
