Announcement of the first meeting of the Java User Group EKB

Hi, Habr!

Despite the fact that in Yekaterinburg there are more than one company developing in java, there is no java community. But when something is missing, you just need to take it and do it. So, after reviewing the next meeting of JUG.ru, it finally became clear that Ekaterinburg needs a place where java developers can meet, discuss news of java technologies, talk about java internals, and just talk!

Thus was born JUG.EKB - the Java community of Yekaterinburg developers.

JUG - Java User Group, meetings JUG.EKB - a discussion of various technologies from the world of Java, the exchange of experience, as well as just interesting communication!
We plan to regularly hold meetings of JUG.EKB with lectures by Java experts on the most interesting topics.
Our site http://jugekb.ru , community in VK - http://vk.com/jugekb .

We invite all java developers to the first meeting of JUG.EKB.
