User rating by several parameters

All the largest social networks offer to install a button on the site for users to rate content. Such buttons allow us to express our opinion in one click. This is a great invention, but it also has a drawback: the user's opinion comes down to only one parameter. If your opinion is more complicated than just ‚Äúlike / dislike‚Äù, then you will need to write a comment that you don‚Äôt always want to do.

The Web Metrics project ( wmetric.com ) offers an interim solution: a user can score on several parameters. At first, the set of parameters was fixed, but thanks to your participation, dear readers, we expanded the functionality and I would like to tell about it in more detail.

First of all, I want to thank all those who participated in the discussion of the first article , helped us see the shortcomings and suggested the necessary improvements. We were convinced that the fixed set of parameters does not suit many. Therefore, the ability to compose a vote of several parameters to your taste was added.

Fortunately, the user will need to be presented only once and then, at the expense of the long-lived session, the voting procedure will take place without additional gestures.
