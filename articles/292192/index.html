MasterCard tests authorization using selfie and fingerprint

MasterCard offers new ways to identify customers when paying by card. Currently on testing are identification mechanisms using self and fingerprint. MasterCard top manager Ajay Bhalla believes that this will help fight fraud. Today, the most secure is the SecureCode system, which sends a password via SMS to confirm the payment. However, here the scammers somehow manage to intercept the message, or just steal the phone and so on.

A special mobile application has been created for the operation of the authorization system, in which you can choose as a way to identify selfies or scan fingerprints.

The developers have already signed an agreement with all the leading manufacturers of mobile devices - from Apple to BlackBerry . Cooperation with banks in the USA is also planned. But while the project is in the testing stage, in which about 500 people take part.

The experience of using fingerprint scanning in Apple, Alibaba and other companies shows that customers are ready for similar authentication methods. But for this they must be sure that these methods really provide a higher level of security.

However, some experts fear for the security of superpersonal user data, which will be sent, stored and read from MasterCard servers.

MasterCard will continue to seek and implement new methods of identification and methods of accepting payments. Experiments are underway with voice and pulse recognition. In the future, it is planned to create applications for payment not only for mobile devices, but for such gadgets as sports bracelets, Kommersant reports .
