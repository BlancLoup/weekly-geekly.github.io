HPE offers Composable Infrastructure

In early December in London, at its European conference Discover, HPE presented the first results of the Synergy project for creating IT infrastructure ‚Äúcomposable‚Äù solutions, which can be quickly (in a few minutes) deployed as traditional server applications (ERP systems) , databases, MS Exchange email servers, etc.), as well as new applications specially written for work in the cloud environment. The Synergy project is the next stage in the evolution of a converged infrastructure that has been used by HP‚Äôs corporate products division for several years in its integrated systems.
HPE Composable Infrastructure is based on three key elements:

- Flexible resource pools: computational, data storage and networked ‚Äî providing a unified infrastructure for deploying various workloads: physical, virtual, and containers;

- Software-defined logic based on the use of templates;

- Unified APIs that enable easy and convenient integration of HPE Compiled Infrastructure with any management and cloud service platform.

The concept of composable infrastructure is implemented in the HPE Synergy system, the delivery of which is scheduled for the second quarter of next year. Synergy beta testing will take place in the first quarter of 2016, and the participants of the London Discover were able to see this system in the forum's demo zone.


Racks with HPE Synergy.

The main building block of the HPE Synergy system is the 10-unit rack-mount chassis Frame 12000, in which the compute and storage modules are installed, forming a flexible pool of resources for the composable infrastructure. Four standard chassis fit into one standard server rack. Compute Module 480 G9 and 660 G9 computing modules using ninth-generation ProLiant technologies are blades with two and four half-height and full-height processor sockets (up to 12 480 modules or six 660 G9 modules can be installed in one Frame 12000 chassis). Compute modules 620 G9 and 680 G9 for two and four sockets, which occupy one and two full-height compartments in the chassis, are intended for servicing mission-critical applications.


Chassis Synergy Frame 12000 with storage module and computing modules 480, 660, 620 and 680 Compute Modules.

The Storage Module D3940, which occupies two half-height bays in the Frame 12000 chassis, houses forty 2.5-inch 12Gb SAS, 6GB SATA or SSD hot-swappable disks that are distributed between the chassis computing modules and connected to them via Synergy 12 Gb SAS Connection Module. Up to five D3940 modules can be installed in one chassis and connected to HPE Synergy 480 Gen9 calculators with up to 100 drives.


Chassis Synergy Frame 12000 c extended storage module.

For connection to Ethernet networks, incl. convergent, HPE Virtual Connect high-speed modules are used, allowing you to combine up to 5 Frame 12000 chassis into a single domain, thus eliminating the need for Top-of-the-Rack switches and greatly simplifying the management of server connections.

HPE Synergy is compatible with all modern storage systems, with deep integration and automatic resource allocation provided for the HPE 3PAR. To connect to SANs, either Virtual Connect modules or ‚Äúclassic‚Äù Fiber Channel switches are used.

Manage the HPE Synergy infrastructure with dedicated Composer servers, for which there are two special slots in the Frame 12000 chassis. One pair of Composer (main and backup modules) controls five racks (20 chassis) as a single system; the chassis are connected to each other using the Fabric Link Module, the control network is physically separated from the production networks.

Composer uses HPE OneView technology and implements the concept of software management based on templates: configures calculators, configures connections to external and embedded storage systems, performs driver and firmware updates, both through a web interface and through universal APIs based on requests from external applications and platforms.
Image Streamer module complements the capabilities of Composer and allows you to quickly create images of operating systems with the necessary applications and immediately download them to the calculators. Preparing such an image takes only 15 seconds.


Chassis Synergy Frame 12000 c extended module Composer .

Exact resource rebuilds to application needs with Composer provides more efficient use of computing power and disk capacity, reducing costs due to idling or at the limit of its hardware capabilities. HPE Synergy implements a single, universal interface for resource discovery, discovery, accounting, configuration, provision, update, and diagnostics. This API is already integrated with VMware vCenter, Microsoft System Center, Chef, Schneider Electric, OpenStack, Docker, HPE Helion and Arista, and its integration with Emerson, Eaton and F5 is planned for the future.

Implemented with HPE Synergy, a unified infrastructure simplifies the use of traditional server applications and makes it easy to deploy cloud applications and services. In this tunable infrastructure, using a single interface, you can quickly compute computing, storage and network resources into any configuration required for a specific application.
