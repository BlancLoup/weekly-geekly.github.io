Layers - great software


A long time ago I was looking for a suitable software for taking screenshots in mac os. And so it happened, there was a miracle - Layers. Layers is a program that screenshots the screen, saves to PSD and puts each object on a separate layer.

This miracle is worth 15 dollars. The demo version can be downloaded from the official website of the developer.
