Yelp, Wunderlist and GIPHY introduced new type applications

Not so long ago, we talked about the release of new types of applications from Uber, PayPal, Boomerang and Evernote. The presented applications offer new experience to Microsoft Office users and allow the companies themselves to enter the users' workspace - office applications.

Today, as part of the official announcement of the new Microsoft Office and Outlook.com, a number of new type applications from Yelp, Wunderlist and GIPHY have been announced. Consider how they look and what they give to users.

Yelp


The Yelp application allows you to insert in the letter exact information (contacts, address, place name) about the upcoming event or dinner from the service and send it in a beautiful form in the letter:



Wunderlist


The Wunderlist application allows you to add notifications and tasks from personal or public Wunderlist lists to your correspondence. Or, on the contrary, add a new task to the list of tasks based on the contents of a letter from the correspondence:



More information about the application can be found in the official announcement on the Wunderlist blog .

GIPHY


Service GIPHY offers a selection of funny gifs for all occasions. Their application allows you to search for gifs and add them to letters without leaving the mail client window on the desktop or in the browser:



Outlook.com update


As we explained earlier , office applications can be added to the work context of Word, Excel, PowerPoint, and Outlook. Due to the fact that these applications are simple HTML5 / JS-based code, applications can be run on all platforms where Microsoft Office is available - Windows, OSX, iOS, Android (in development). And, of course, applications can be launched in the browser in the web versions of office applications!



Update Outlook.com brings this feature, previously available only as a preview. Users of the web version of the Outlook.com mail service can now launch applications from the Office Store, including those presented above.

It can be your applications, applications that add benefits to your service, integrate it into the user's work and home environment. Start exploring the capabilities of the platform today through a single portal http://dev.office.com/ .
