Ruby 1.9.2 released

The long-awaited version of the Ruby interpreter for version 1.9.2 has been released

1.9.2 on points

Ruby 1.9.2 is basically compatible with 1.9.1, except for the following changes:
  • Many new methods
  • New API for Sockets (with IPv6 support)
  • Support for more string encodings
  • Random class that supports various random number generators
  • Reworked class Time. Now the problem of 2038 is not a problem.
  • Some improvements to regexp regular expressions
  • $: no longer contains the current directory.
  • dl reworked as add-on over libffi.
  • New library psych, which is a wrapper for libyaml. You can use it instead of syck.

Good news

Ruby 1.9.2 is tested at 99% RubySpec

Supported Platforms

100% Supported Platforms
  • Debian GNU / Linux 5.0 on IA32.
Best support
  • Ruby 1.9.2 works mostly well on these platforms.
  • mswin32, x64-mswin64, mingw32
  • MacOS X 10.5 (Intel) and 10.6
  • FreeBSD 6 and later (amd64, IA32)
  • Solaris 10
  • Symbian OS
Possible support
  • Other Linux Distributions
  • Other versions of MacOS X.
  • cygwin
  • AIX 5
  • Other POSIX compatible systems
  • BeOS (Haiku)
Not supported
  • Any other system

Note

$: no longer contains the current directory, so some scripts need fixes to work correctly. In any case, your application should depend as little as possible on the current directory.
