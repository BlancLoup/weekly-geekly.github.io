Free telecom operator will start work in Russia this summer

In exchange for the services of the operator will have to perform various tasks.



This summer, the Atlas communication operator will appear in Russia, whose work can be called shareware. The fact is that customers will have to pay for the communication services of this operator not with money, but with services. More precisely, you will need to perform various tasks, write "Izvestia". Already, an application of this operator with the same name has appeared in the catalogs of applications from Apple and Google.

Atlas will operate as a virtual telecom operator - it does not have its own towers, so the company will lease capacity from one and federal mobile operators. Unfortunately, who will be a partner is still unknown.

As for the operator's application, about 400 partners are connected to it. All of them give users specific tasks. For the execution of tasks the user will receive points. 1 point is equal to 1 ruble. The tasks will be simple - for example, you will need to download the application or write a positive comment on the site reviews. For the performance of each such task are awarded "atlases". The cost of the package with 1 GB of Internet traffic, 100 minutes of calls and 10 SMS will be 200 "Atlas".

At the moment, the application has downloaded about 100 thousand users throughout Russia. However, the operator will not start work until the moment in any major city the number of application downloads is 100 thousand.
