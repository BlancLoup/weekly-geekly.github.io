Chromizer - Theme for Google Chrome is easy, convenient, fast.

Modern browsers are often used programs for both work and entertainment. In this connection, not only technical characteristics are important, but also the possibility of additional configuration and, of course, design.
I often use Google Chrome and at some point I became interested in how you can change the standard look of the application.


Main options
  • Google Theme Gallery
  • CRX Theme Creator, Chromium Theme Creator programs for creating your own themes

Cons of available solutions


Google Theme Gallery
  • The choice of topics is small
  • In addition, in many themes, usability suffers, since, for example, the tabs are designed in such a way that it is difficult to read the text, it is unusual to perceive the window frame in bright and variegated colors, etc.

Programs in the spirit of CRX Theme Creator, Chromium Theme Creator
  • Programs that I managed to find - for windows (I work on a MacBook)
  • Time spent on creating a theme - start the program, specify pictures, think over the colors of panels, buttons and other elements - it‚Äôs not a fact that you get a good combination of colors from the first time, because not everyone is gifted with a sense of beauty

What do I want?


For me, most of the topic is the wallpaper of the new tab. This is the largest element, the panel I prefer not to change - to quickly navigate in a visually familiar environment. A quick change of wallpaper tabs would be a compromise between an extensive set of features, but with a lot of time to create your theme and speed, and at the same time, simplicity.

What to do?


Looking for solutions to quickly change the wallpaper of a new tab, I did not find anything and decided to write a simple tool and at the same time delve into the internal "kitchen" of Google Chrome.

What is the theme for Google Chrome?


Theme for Google Chrome is a special type of extension. It consists of a zipped archive with a special key, a manifest.json style file and theme pictures.

How it works?


Application side
  • Chromizer uploads a picture of the user to the server
  • Adds a style file to it.
  • Packs the subject
  • Provides the user with a download link.

By the user
  • The user selects a picture in png format, from which he wants to create a theme
  • Press the "Create a theme" button
  • Downloads and installs the theme in the browser.



What do we get?


With Chromizer, you can quickly create both decorative themes and themes that contain useful information: formulas, tables, a pixel grid, and so on.
