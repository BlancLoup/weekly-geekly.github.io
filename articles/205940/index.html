The release of 3CX Phone System 12 Service Pack 3

3CX Phone System 12 Service Pack 3 saw the light. The new Service Pack has taken into account the wishes and suggestions of our customers - thanks to this, 3CX Phone System has become much more convenient for both administrators and end users. The main focus was on improving and adapting the interface (now all the basic functions are available in the lower part of the application) and simplifying the forwarding settings . Also in the version of the application for Windows, support for headsets has been added. A complete list of new features and fixes is presented below.




New opportunities


  1. Color selection. In addition to the current dark gray color of the application, there was also a white color.
  2. 3CXPhone has become a bit more, it is possible to place all the function buttons at the bottom of the application
  3. Added the ability to change the voice greeting, based on your status in the PBX
  4. Call duration is now displayed in the call log.
  5. Added headset support with answer button
  6. Standard Bluetooth headsets are supported on Windows and Mac, but without the answer button
  7. Contacts can now have several numbers. Mobile and home numbers have different icons.
  8. Queue Forwarded calls to mobile numbers are visible in the presence of active calls on the user's screen
  9. Added the ability to automatically change your status to ‚ÄúUnavailable‚Äù when there is no computer activity.
  10. Added the ability to start the 3CXPhone client automatically if the site has a "tel: protocol" tag
  11. Chat is updated when the user is not available for chat.
  12. Added status in the chat, in the upper part of the chat, showing the status of the user "You are communicating with"
  13. Messages sent in "offline" mode will be delivered when you change your status to "Available"
  14. Improved contact search in the entire application
  15. Added the ability to add a company / phonebook entry from the Call Log
  16. Improved Hot Keys feature. Now just press the Hot Keys (there is no need to press CTRL + C) and the number will be copied and cleared of any characters -, spaces or () and pasted into 3CXPhone
  17. Contacts with the company name can now be added to the company‚Äôs phone book, which is useful for business contacts.
  18. Added the ability to add international code to replace +, 00 or 011 for CTI calls
  19. Added the ability to transfer / switch a call to someone else's voice mail
  20. Enter key to delete voice messages and conversation records
  21. Added the ability to drag calls in the queue
  22. Ability to control whether the presence window is running full screen or not when the application is started (depending on whether it was closed on exit or not)
  23. Added support for G722, Speex and Ilbc codecs
  24. 3CXPhone-White - Black


Fixes


  1. Fixed bug when 3CXPhone did not come to the fore when an incoming call
  2. Fixed a bug where the sound did not pass when using 3CXPhone via VPN
  3. Fixed bug when 3CXPhone resets its configuration.
  4. SRTP bug fixed - Secure RTP
  5. Fixed problem when external participants were not always visible when creating a conference.
  6. Fixed calls to queues that are sent to the mobile phones of the agent, they are now visible in active calls
  7. Fixed a bug that prevented the client from viewing remote PBXs
  8. Fixed a bug that showed 3CXPhone as unregistered in CTI mode
  9. Scheme and position of the application at startup are remembered
  10. When creating a conference, you can now choose to connect a user with one or several users at the same time.
  11. Ability to schedule a conference and invite contacts from the corporate phone book. (Those who have configured an email address)
  12. Numerous fixes in chat, call sorting, status, group chat
  13. Fixed bugs in codecs, proxy server, SIP in TCP mode, as well as issues of sound transmission for subscribers outside the network
  14. Intrusion, listening and prompting in the PRO version are performed with automatic response.
  15. Fixed bug in 3CXPhone when it is connected through the tunnel
