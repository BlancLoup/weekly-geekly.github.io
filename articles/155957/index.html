Mobile applications for iOS and Android on php

Millions of php developers will be able to develop mobile applications for iOS and Android on php. On ZendCon this morning, Zend chief executive Andi Gutmans announced that Zend Studio 10 will allow mobile applications to be made in php.
Among the new features:
  • WYSIWYG drag-and-drop mobile interface designer
  • Integration with the Apache Cordova project for access to the native phone API (work with cameras, accelerometers)
  • Integration with PhoneGap, which will allow you to publish applications in various app stores
