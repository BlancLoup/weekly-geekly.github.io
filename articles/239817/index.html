Smart charging cable Bleep backs up data at the same time as charging a smartphone



Many owners of mobile devices store their data in a single instance - only on the phone or tablet. At the same time, the data can be very valuable for the owner of the phone - for example, a successful photograph from a vacation, or the KeePass database, with access to various services.

Some users create a backup copy of data from the phone (for this there are both cloud services and local software), but still not so often. The problem can be solved by the project Bleep is a charging cable with a built-in storage device and a corresponding application for a mobile device. Now the cable fundraising campaign has already collected twice as much money as is required, so the project has already begun to be implemented.

Bleep features





The initial idea is quite simple: the owner of a phone or tablet puts the device on charge for the night, and at the same time all data from the phone is archived. At the same time, the storage capacity of Bleep is quite sufficient for most models of mobile devices: depending on the type of cable, the storage capacity is 16-64 GB.

At the same time, access to the network is not needed, so you can archive data anytime and anywhere.

Beginning of work



You need to configure creating backup only once: when you first connect the cable to the device. After that, everything will work in automatic mode.

The first thing you need to install an application for a mobile device (here you need the Internet, of course), and set the necessary copy settings, or use the default settings. The application is for both iOS and Android.



Bleep functions :
  • Charging a mobile device;
  • Automatic archiving of data from a mobile device to the internal storage Bleep;
  • Data recovery on demand;
  • Password protection;
  • Encryption of archived data;
  • The choice of one of three modes of operation: only charging, charging + backup, recovery.




When you connect the cable to an unfamiliar device, Bleep will perform the normal charging function. If the Bleep application is installed on an unfamiliar device, the application will ask the user to select one of the three operating modes only charging, charging + backup or restoring.



In this case, working with previously saved data is impossible without specifying a previously set password. The backup itself is encrypted (although the encryption standard is not specified by the developers).



How much does it cost and when will it be available?





Now, developers provide an opportunity to pre-order the device at a price of 40/60/78 US dollars for models with a 16/32/64 GB drive, respectively. True, the shipment of the first cables is scheduled for next August, so it will take a long time to wait.

In Russia, pre-order Bleep can be issued at the Medgadgets.
