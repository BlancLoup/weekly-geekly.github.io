Windows Camp - all about Windows 8 and Windows Phone 8



Today, on the 8th day of the 8th month, we open registration for the Windows Camp conference , which will be held in Moscow on September 7, 2012.
The conference is dedicated to the newest platforms from Microsoft: Windows 8 and Windows Phone 8 .

Windows 8 is a new version of the most popular operating system, which is used by more than a billion people.
Windows Phone 8 is a new version of the operating system for smartphones, which exactly a year ago started successfully in Russia and has more than one hundred thousand applications.
Inside all devices on Windows 8 and Windows Phone 8 there are app stores.

On Windows Camp, we will talk about how to use this unique opportunity to directly interact with users around the world.

On the one hand, we will talk about the platforms themselves and about the principles of development for them: starting with the design and continuing with the story about the newest tools for developing applications - Visual Studio 2012, and completing the picture with the story of monetization and the opportunities offered by application stores. On the other hand, for those who have already begun to develop and want to get acquainted with advanced development capabilities, we have prepared a parallel track.

The conference is one of the best ways to develop and communicate with colleagues and application developers who have already become popular. In the breaks between the main reports, you are waiting for short stories from partners and independent developers, as well as stories about the development of real-world applications that you use almost every day. You will have the opportunity to ask questions to community experts and, of course, to Microsoft experts. And that is not all.

The exhibition partners will allow you to get acquainted with the latest technological innovations. And at the specialized Microsoft booths, you can ask questions about integration with cloud services or software lifecycle management technologies.

The conference site is traditionally launched in the Developer Preview format and will be updated along with the approach to the event itself.
In the near future, a more detailed schedule of reports will appear - its structure is described above.
Participation in Windows Camp is free with pre-registration - confirmation will be sent later.
Hurry up to register - places run out quickly.

See you on Windows Camp!
