New name . Two versions of the Web IDE received their official names: JetBrains WebStorm and JetBrains PhpStorm. The goal of development remains the same - an ideal tool for PHP / JS developers and web designers.

Fully customizable PHP formatter - this feature has received the most votes in the tracker. Our platform supports import, export and sharing of profiles - so it will be convenient for you to switch between projects with different formatting styles. In addition, you can automatically format the code at the time of check.



PHP debugger with optimized memory usage and correct work with symlinks (which means fewer non-working breakpoints).

PHP editor with improved type deduction for fields, arrays of objects and logical expressions, parameter hinting and working HEREDOC editing.
