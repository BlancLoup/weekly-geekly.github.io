wireframe.cc - the easiest interface prototyping tool

Today, there are many web applications for convenient and quick prototyping of interfaces, but wireframe.cc is the best one I've ever tried.



Here are its main features:

  • Minimalistic interface: there is nothing superfluous on the working screen - only a small panel of buttons and a device window.
  • The templates of the browser window and mobile device (in landscape and portrait design).
  • Damn convenient way to add elements: drag-to-draw and then select the type of object.
  • Simple add comments to the interface.
  • Smart elements: wireframe.cc tries to determine by its object size that it is a header, a form or a scrollbar.
  • Convenient saving prototypes (each given a short link) and sharing on Twitter and Facebook.
  • Simple color palette.
  • Hotkeys: standard Ctrl + C / Ctrl + V, Photoshop Alt + drag to duplicate an object and double-click to edit.
