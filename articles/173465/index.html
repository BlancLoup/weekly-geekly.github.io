Contorra site redesign: now adaptive + HTML 5

Another adaptive project that was very pleasant to work on)
We set ourselves the task to simplify the existing site as much as possible.



The left block with the logo becomes fixed on the wide monitor. Scroll (scrolling) acts only on the right half of the site with the works. Thus, the contact information is always in sight of the user and, if he decides to contact us, he can easily do it.



QR-codes (Quick Response) are very popular now, so we also decided to place it on the main page.

Contorra is our first project using a new technology: canvas on HTML 5. With its help, we implemented a screensaver when loading a site. And if you go to page 404 errors , you will see the result of applying another effect - parallax.
