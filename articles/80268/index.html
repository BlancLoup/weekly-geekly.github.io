Machine Readable Semantic Dictionary

Aces tried to reduce Melchuk's descriptions to machine-readable formulas. For this, firstly, the "axiomatics" was expanded. Say, the word "pig" was considered atomic - that is, we can assume that the "pig" is an object of the semantic field of the language, and there is no need to "explain" it to a computer. Attributes can be attributed if desired. The explanation will inevitably result in a long abracadabra, covering a variety of pig aspects.
