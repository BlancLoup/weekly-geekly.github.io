k3b 2.0.0 release


Today, after a long time, the release of the program for recording k3b discs has officially been released!

From the innovations:
  • Full support for the KDE4 platform
  • You can always choose the recording speed
  • Blu-ray support
  • Large file support (> 4GB)
  • Localization improvements
  • GUI Improvements
  • Close button on tabs of different projects

So far only source codes of the project on sourceforge are available .
