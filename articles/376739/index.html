Windows Defender taught to catch Superfish from the computer

Microsoft has added to the next update of the Windows Defender database data to remove the software Superfish. SSL certificates that are used to monitor encrypted traffic are also completely removed from the system. This eliminates the appearance of unwanted advertising in the browser and the security risk for encrypted connections.

The program Defender, previously called Microsoft AntiSpyware, is designed to detect and remove spyware. It monitors suspicious activity of programs and is able to remove malicious software. The program comes with Windows Vista and Windows 7. It can also be downloaded free of charge for Windows XP and Windows Server 2003.
