OpenOffice.org v.3.0.0 Final released


The final build of the OpenOffice.org 3.0 software package is ready. Convenient and easily customizable OpenOffice.org 3.0 realizes the potential of open source software. In addition to the new database module and XML advanced features, OpenOffice.org 3.x natively supports the OASIS OpenDocument format.

The package is available in 36 languages, able to work under MS Windows, GNU / Linux, Sun Solaris, Mac OS X and a number of other platforms.

In addition, users are offered improved support for the PDF format, a new module for working with spreadsheets, tighter desktop integration, and a host of other features that use advanced XML features, such as creating, editing, and using XForms.
