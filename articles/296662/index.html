17 useful tools for SEO audit

In the tenth article of the Cossa and Kokoc Group cycle about tools useful in the work of digital specialists, we continue to talk about services that can be useful for SEO auditors.



Author: Artem Yaskov - Head of SEO Audit at Kokoc.com (Kokoc Group).

Semrush

A useful tool for working with the semantics of the site. Allows you to determine the visibility of the resource by keywords in Google, and takes into account both the Russian-language and foreign search results. The service is convenient for building reports and auditing the site. However, SEMrush has one significant drawback: it does not work with Yandex. The tool is available on a paid and free basis. For the most simple analysis of the site, you can not pay, to access more functionality you will need to register, and for the advanced version you will have to pay.



Google trends

A free service with which you can watch and analyze trends in search queries in all regions where Google search works. The tool allows you to compare the volume of requests for two or more search phrases, and also displays news related to search phrases. This information Google Trends imposes on the schedule, showing the impact of new events on search popularity.



Wizard.Sape

Text analyzer allows you to calculate the number of occurrences of keywords in the text and its volume for each keyword. However, the specialist will have to combine the results for two or more keywords. At the initial stage of use, you need to register an account and replenish your account.



Ahrefs

Service to analyze the reference mass of both their own sites and competitors' resources. Allows for a few seconds to collect the information of interest, because it contains its own database of resources, in which more than 500 million new pages are added daily. The service is paid, but there are a number of free features, and some restrictions are removed after registration.



4seo.biz

A free system that allows you to conduct a massive check of the site pages to the server response code. By applying this tool, the specialist will be able to quickly detect which of them do not work and therefore are not indexed by search engines. For verification, it is enough to enter the analyzed addresses in the field.



Recipient Donor Service

A free browser plugin for Mozilla Firefox, Google Chrome and Opera, which helps to conduct a quick site analysis. Shows the number of resource pages indexed by search engines, the subject citation index, attendance per day. Registration in the system is not required, but gives access to more complete functionality.



Firebug lite

A free browser extension that allows you to edit, customize, and view the results of changes on a web page in real time. With its help, it is convenient to study the HTML code of the page, edit HTML and CSS directly in the browser, debug JavaScript, monitor the page loading process.



JustMagic

Paid automation service for SEO professionals. The system helps to select and expand the semantic core both for new sites and for existing ones. JustMagic selects synonyms, clusters requests for tops and determines the possibility of promoting each page of the site for similar requests: for example, ‚Äúbuy a smartphone‚Äù and ‚Äúbuy smartphones‚Äù. The service has a text analyzer that allows you to determine the required number of occurrences of the query in the text, headers, or <a> tags.



XENU

Free program that simplifies the technical audit of the site. The tool checks all pages and elements, for example, pictures and scripts, for the presence of broken links, links with redirects. Thus, you can detect, correct errors and make a list of "useful" url-addresses.



VisualSEO

Free service, similar in functionality to XENU. The tool provides an opportunity to analyze website headers, check the quality of optimization of meta-data and identify problem pages with duplicate content. It can analyze up to five resources at the same time.



Screaming Frog SEO Spider

Helps to conduct an express audit of the site and identify technical errors. The tool can scan a resource by reference or check the downloaded list of url-addresses. The program is available in two versions: paid and free. In the second case, there is a restriction on scanning: no more than 500 url-addresses.



"Watercolor"

Paid module service JustMagic to analyze the texts of the site for the presence of words that are not related to its subject. It is enough to enter a request and text into special fields, the system will mark inappropriate words in red. For example, for the query ‚Äúbuy tires,‚Äù the entire phrase ‚Äúconfirmed by a century of history‚Äù is non-thematic and will be highlighted in red. The system also has a word word generator, which is useful for writing relevant texts. For the query ‚ÄúSamsung TV‚Äù, ‚ÄúWatercolor‚Äù will select 157 words (light, screen, diagonal, connection) and 33 phrases (watch movies, diagonal screens, new models).



Pingdom tools

A free service that allows element by element to estimate the speed of loading the page of the site. To do this, you must enter the address in a special form, within a few seconds the system will analyze the page and generate a detailed report.



PageSpeed ‚Äã‚ÄãInsights

Free Google service, with which you can speed up the loading of a site page on any device. The tool tests the resource and offers recommendations for improving the load time of the upper part and full page load. By the way, the service is useful for optimizing pages for mobile devices.



Yandex Wordstat Helper

Free plug-in for working with Yandex.Wordstat via Google Chrome and Mozilla Firefox browsers. It automatically generates keywords into a ready list, eliminating the need to constantly copy them into Excel or OpenOffice documents. Having found doubles, the system deletes them.



Page weight

The program for setting the linking site. Analyzes internal and external links, their anchors and increases the static weight of pages being promoted. Page Weight helps to find pages with a lot of weight, links from which are effective for internal optimization of the resource. The program has paid and free versions, the latter is limited to the analysis of 2000 pages of the site.



Google mobile friendly

Free Google service that helps to check the convenience of site pages for users of smartphones and tablets. The tool allows you to make sure that when optimizing a resource, there are no errors that can prevent Google from properly indexing it in a mobile search.
