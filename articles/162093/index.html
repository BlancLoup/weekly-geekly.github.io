EPAM IT Share # 3: Search for related images. Define Cloud Computing

Hello to subscribers of corporate habrablog EPAM Systems Ukraine!

On Thursday, December 6, the third EPAM IT Share took place in Kharkov, during which Vyacheslav Kolbasin, Alexey Tregubov lemieux66 , Yegor Sopov and Andrey Kozhokaru korjik shared their knowledge about search algorithms for similar images and told about key definitions of cloud computing.

In this small report you will find video footage of the meeting and presentation files.

Search for related images

Vyacheslav Kolbasin, the developer of the Kharkiv office, described the algorithms for solving this problem and how these algorithms can be effectively implemented on parallel computing architectures. You can also find an overview of practical problems that may require comparing and searching for similar images.

Define Cloud Computing

In the second part of EPAM IT Share, Alexey Tregubov, a cloud technology consultant, and Yegor Sopov, a technical writer, talked about the definition and standardization of the concept of Cloud Computing. Were considered 5 key features and service model Cloud Computing.
