Official release and free upgrade from Windows 7 and 8 to Windows 10

Free upgrade to Windows 10 is already available in the "Update Center" operating system for owners of a licensed copy of the operating system from Microsoft, according to ArsTechnica .



Windows 10 will not be free for everyone and not always, as mentioned earlier . You can upgrade to the new OS for free throughout the year, until July 29, 2016, which is spelled out in the official Microsoft help and only with Windows 7 or Windows 8. Microsoft plans to earn not only on future OS sales, but also on subscriptions to its products, such like offfice 365 and others.



The network has already expressed the opinion that the transition to a new OS with the help of a system update is much more comfortable than when installing the system from scratch. For the convenience of users and reducing the load on the servers, the upgrade to Windows 10 takes place in stages for different regions.

XP and Vista users will not be able to upgrade to Windows 10 through the system update. For them, the option of purchasing the OS in the Microsoft Store with the subsequent "clean" installation of the system will be provided.
