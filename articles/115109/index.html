The final version of Visual Studio 2010 Service Pack 1 and other announcements



At today's TechEd Middle East event in Dubai, S. Somasegar , senior vice president of development at Microsoft, announced today that the final version of the fix pack for Visual Studio 2010 will be available soon and made a number of announcements .

Visual studio 2010

Today, March 8, the final version of the first service pack for Visual Studio 2010 will be available for download to MSDN subscribers. All the rest will be able to freely download the service pack on March 10.

Few details about the service pack can be found here .

UPD: service pack is already available on MSDN.

TFS-Project Server Integration Feature Pack

In addition to the update package for Visual Studio 2010, MSDN subscribers will have access to a special set of integration tools for the Project Server project management server and Team Soundation Server.

Visual Studio Load Test Feature Pack

The new tool announced today is the Visual Studio Load Test Feature Pack. It is known that Visual Studio 2010 Ultimate allows for stress testing applications. With the help of the new package, Visual Studio 2010 Ultimate users will be able to emulate virtual users without having to buy additional Visual Studio Load Test Virtual User Pack 2010 licenses. For more information, see this page .

This tool will be available for download today, March 8th.

Visual Studio LightSwitch Beta 2

Visual Studio LightSwitch allows you to create business applications in a new style. Visual Studio LightSwitch Beta 1 had a high interest from the community. It was downloaded over 100,000 times. Today it was announced that Visual Studio LightSwitch Beta 2 will be released soon, a key feature of which will be integration with Windows Azure and SQL Azure cloud services with the ability to publish projects directly from Visual Studio.

The new version of Visual Studio LightSwitch will be available for several weeks.
