Keyboard Racing

I present my project to the public - online game Racing on the keyboard . This is a javascript game, a keyboard simulator, crossed with auto racing. The goal is simple - type the text faster than rivals. You can play alone, with other visitors to the site, you can create a separate game for yourself and your friends.

In fact, this is a Russian-speaking clone of a similar game, typeracer.com , but with a number of its own chips.
