Synapse is a good alternative for Alt + F2


Synapse is a very nice launcher, with some pretty interesting buns.

So, we list the features that are available with a standard set of plug-ins (yes, yes, plug-ins, you heard right):

* Running applications
* Banshee - play music in Banshee
* Execute commands (note ‚Äúsudo apt-get update‚Äù)
* Devhelp - search documentation using Devhelp
* Dictionary - finds word definitions
* Directory search - opening folders
* Gnome session - log out, shut down, restart
* Hybrid search - search for similar files
* Rhytmbox - Rhythmbox music playback
* UPower - put your computer to sleep or hibernate
* Zeitgeist - Search using Zeitgeist

There is also support for all tastes and colors.
By the way, by default, it is called by the combination Ctrl + Space.

Synapse Installation:
sudo add-apt-repository ppa: synapse-core / ppa
sudo apt-get update
sudo apt-get dist-upgrade # only needed for Lucid
sudo apt-get install synapse

Since this launcher uses Zeitgeist data, it would be nice to install it:
sudo add-apt-repository ppa: zeitgeist / ppa
sudo apt-get update
sudo apt get get

Well, for full blunders, put the extension FTS Zeitgeist

For maverick:
sudo apt-get install zeitgeist-fts-extension

For Lucid:
sudo apt-get install zeitgeist-extension-fts
