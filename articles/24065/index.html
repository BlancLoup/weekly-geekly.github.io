Microsoft introduced a single platform Live Mesh

After two years of development, the official opening of the first, while the preliminary preview (Technology Preview) of the Live Mesh platform took place - a single interface for accessing personal information from all devices owned by the user, be it laptops, desktops, smartphones, or anywhere through the web. From a technological point of view, this is a set of feeds (no matter in what format), where all information about the information, files and folders and devices of the user, as well as a small program for accessing and synchronizing the feeds are exported. As for the offline component, this approach is more like Google Gears or Mozilla Prism than Adobe AIR.



Live Mesh is an attempt by Microsoft to create a single universal platform for synchronizing all gadgets, applications and personal information of the user in the era of ubiquitous web. The developers have set a global task: to ensure the interaction of all devices and applications with each other, access to personal information archives, as well as to simplify the order of opportunities for collaboration and data exchange. To solve all these problems, a single distributed platform was designed, where SaaS principles are used for file management, file sharing and synchronization.

Registration in the system is still available only by invitation (the software works on the latest versions of Windows, support for Mac and mobile phones will be available soon).
