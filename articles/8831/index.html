Let me remind keyboard shortcuts to search for Firefox:

You can start searching using Ctrl + K ( Cmd + K for Mac users)
Select the search engine Ctrl + Up (Mac: Cmd + Up ) and Ctrl + Down (Mac: Cmd + Down ).
Open results in a new window: Alt + Enter (or you can set Firefox so that it always opens search results in a new window: type about: config in the address bar, set browser.search.openintab to true)


And for new-found Google:

J - select the next result
K - select previous result
Enter or O - open the selected result.
/ - move the cursor to the search bar
Esc - remove the cursor from the line
