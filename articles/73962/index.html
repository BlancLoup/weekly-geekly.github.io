qutIM 0.2. Release is ready!

In honor of Halloween, we want to congratulate you all and present you a new version of the qutIM messenger!

More than a year has passed since the beginning of development of version 0.2, this year we encountered many difficulties, there were problems with servers, there was a shortage of developers (and perhaps still this problem remains), there was and there is a lack (more precisely, a complete absence) of designers. From time to time we had a choice - to use common technologies or to rely on dark horses - for a long time we decided between iris, who already proved himself in such clients as Psi and Kopete, and little-known gloox, which turned out to be a more promising development.

qutIM is a young, promising and free multiprotocol client for real-time communication. Since version 0.1.1, the kernel has been completely rewritten, the ICQ protocol implementation has undergone multiple changes and has been moved to a separate plugin, support has been added to many other protocols, so now qutIM officially supports the following protocols:
  • ICQ
  • Jabber
  • MRIM
  • IRC
  • VKontakte





The release also includes such non-protocol plug-ins as:
  • HistoryManager easily and quickly converts your history from other common clients: QIP, Miranda, Pidgin, Kopete, Gajim, Psi and many others!
  • PlugMan allows you to easily install additional themes, emoticon sets, including Koloboks, which are so beloved by everyone, sound themes, and other small utilities that help to make qutIM more beautiful.
  • Yandex.Narod will upload the files to Yandex.Disk and give the other person a link to it, which is remarkable - you can send files to offline users as well. In addition, you can manage already uploaded files.



In addition, at the expense of plug-ins, kutim is integrated into various work environments:
  • KDE - kde-integration plugin
  • Windows 7 - aero theme and w7i plugin
  • Gnome - Libnotify plugin
  • MacOS X - GrowlNotification plugin



In addition to the above, many other interesting and diverse plug-ins were written, the most remarkable of them:
  • Urlpreview - displays images by reference directly in the chat window
  • TeX - allows you to exchange various formulas with tex markup
  • ImagePub - allows you to share images through various hosting
  • Mass Messaging - allows you to send messages to the entire contact list or to individual contacts / groups at once
  • And many others!



You can specify the following program changes:
  • Multiple bugfixes
  • Fixed crashes with GNUTLS, problems connecting with OpenSSL and random segfolts with SChannel
  • More ergonomic settings window
  • Transition from a binary format of history to Json


Attention! If you have not previously used the Json history plugin, then you should convert history into the new format using History Manager .




Sources: www.qutim.org/uploads/src/qutim-0.2.0.tar.bz2
