"Heart Eater" - ASUS RT-AC68U Router: 802.11ac, usb 3.0, BitTorent, TimeMachine


 By February 23 and March 8. Please yourself and loved ones to you

Preface from the author (written 5 minutes before publication)


Initially, I wanted to comprehensively test the device with beautiful graphs and tablets, but something I was so imbued with it and just started to use it, so everything is a bit messy and without facts, only emotions. He just conquered me ... Let's go!

brief information


There are many solutions on the home networking equipment market, but today I want to tell you about one really unique device: the ASUS RT-AC68U router (described on the ASUS website).

Without delay, I will give the key features of the RT-AC68U , putting it above the rest of the devices on the market:
  • newest wireless standard 802.11ac
  • connect an external hard drive via usb 3.0 interface
  • official time machine feature
  • BitTorent standard support (including the previous item)

It is the totality of these features that led me to this acquisition. Of course, I am silent about well-known functions, such as: connecting a printer or a 3G / 4G modem to a second usb2.0 port, working as a home file server, working as a rocking chair for everything.

Time machine


But this section explains why this post I took to the Apple hub. And also this is one of the reasons why I bought this router. But in fact, everything is so simple that there is almost nothing to talk about.
Initially, I smashed the hard drive using Disk Utility into 2 sections: 0.5Tb and 2.5Tb, the HSF + file system.
In the web interface, you must specify a folder for backup



And in the Time Machine settings, simply select this disk from the automatically offered list. And that's all.



Yes, do not forget to remove unnecessary folders through the parameters, I have this downloads folder.
