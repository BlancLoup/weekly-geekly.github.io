WinFS Partly Deployed in SQL Server 2008

The next version of SQL Server, codenamed Katmai, is likely to be published in June 2007 as a technology preview (CTP). The final release is expected next year. Will a new development help Microsoft to regain its market share in recent years?

Improvements in Katmai include compatibility with new data types (for example, with unstructured MS Word texts), integration with Office 2007, SharePoint Server 2007 and PerformancePoint Server 2007, support for geographic information from specialized applications, creation of reports of any complexity, support for data presentation Entity Data Model (part of ADO.NET), improved scalability.

The new storage system may have common roots with the unrepresented WinFS file system, which was promised for Windows Vista and Longhorn Server. As you know, the file system project was closed about a year ago, and some parts of WinFS are intended for use in various Microsoft products.
