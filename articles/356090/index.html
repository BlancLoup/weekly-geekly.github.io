The Ministry of Communications and Mass Media proposes that legal entities be required to transfer information about IP addresses to a new state system



The changes will affect every Russian company that has access to the Internet (that is, it uses an autonomous system number or an IP address).

The new information state system should help ensure the stability and security of the Russian part of the Internet. It will contain data on critical elements of the Russian network segment, for example, autonomous systems, IP addresses, traffic exchange points.
Probably, these data companies and will need to transfer to the system. If the bill is adopted, the exact composition of the information and the procedure for their presentation will be established by the government.
Recall that companies can not only rent or get their IP-address, but also have an autonomous system that allows you to connect to two or more providers, so as not to depend on one.
