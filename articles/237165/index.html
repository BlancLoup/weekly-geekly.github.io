Internet by sms

Good day, Habr!

While I choose a smartphone with the obligatory support for LTE and Quad HD resolution, in some regions of the world (and Russia) people have not only 3G, even the second generation data transmission works through time. And then how to sit on this in your network? There is a way: Cosmos browser allows you to read the Internet via sms. It sounds crazy, doesn‚Äôt it? But it works.




The application for Android, which is being prepared for release, works as follows: you send a URL, and in return you receive a simple text version of the page using text messages. The developers explain that the application you send the URL to the Twilio service number, and so the message reaches the service itself. The system then receives the code from the site and clears it of CSS and javascript so that a number of text messages will appear on the user's phone.

Unfortunately, the method has several limitations:
1) at a minimum you need a tariff with unlimited sms.
2) sms do not always work correctly, so some of them may be lost, and you will not get part of the page
3) a very limited version of the site will not give you too much pleasure

Despite all these disadvantages, there is one big plus: at least you can get some information, even if the Internet is not around for kilometers, but at least messages are available.

The developers promise to release the application by the end of September, but the code can already be viewed on GitHub .
