Joomla 1.5.10 [Wohmamni] - security release

Released security release - Joomla 1.5.10 [Wohmamni]. This version fixes 66 bugs that were found during testing Joomla 1.5.x (two of them are related to security). Developers strongly recommend upgrading to this version. Updated localization packages are already available for download from our website.

Among the corrected errors there are a couple of security errors. The first error is related to the presence of a potential XSS vulnerability in administrative components (com_admin, com_media, com_search), the second to the XSS vulnerability in the form (view) of the com_content component category. The developers do not report whether previous versions are subject to these errors, but in any case, it is recommended to upgrade to Joomla 1.5.10.

Among the other changes, we can note the correction of an error when searching for words with a capital letter P (see on the forum: preg_replace () and the capital Russian P ).

More information about the bug fixes can be found in the press release for version 1.5.10: Joomla 1.5.10 Security Release Now Available.
