Oracle's Java Day in SumIT

We invite Java developers to listen about JDK7 and win iPad2.
Sergey Grinev, an engineer at the Oracle Development Center in St. Petersburg, will talk about JDK7, the new release of which was released on July 28. The event will be held as part of the SumIT summer start-up marathon at Kronverksky 49, St. Petersburg State University ITMO building at 19:00.


After this, we will launch the Oracle Javathon Contest with the main prize iPad2.

Participants will need to complete one of their tasks in 23 hours.

  1. Develop your own language and write a compiler in Java byte code. The language can be very simple, such as a calculator or Brainfuck.
  2. Java has the NIO2 library. It has the ability to work with JAR and ZIP file systems. The task is to work with a TAR (or any other) archive file system.
  3. Oracle is working on a new Java client ‚Äî JavaFX, a very new technology. JavaFX 2.0 Beta is currently released, unfortunately it works only on the WIN platform. The technology is interesting because it can visualize the work of the developer.
    - Implement a simple toy on JavaFX (River Raid, PackMan, Python, AngryBirds, etc.).
    - Implement a client for one of the social networks.
    - Implement the repository visualizer.
  4. Using the OpenJDK source code to implement some interesting language feature.

In the competition are allowed both individual participation and team up to 3 people.
Registration >>

Summing up the competition Javathon will be at 19:00 on Friday. The contest participants will show their results, a technical expert from Oracle will choose two of the most worthy, and the vote will decide who will be the owner of iPad2.

After summing up, Evgeny Ovchinnikov (coordinator of the HackDay movement in St. Petersburg) will speak with a master class on Operational Management of Startups .

Register and come!
