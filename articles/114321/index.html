Car PC on Ubuntu



Many of us probably thought about assembling a dual-purpose PC card, with a touchscreen, GPS and 3G internet connection for surfing or swapping the right maps. So now we have the opportunity to buy a ready-made car PK. What is most interesting as an operating system on it is installed Ubuntu 10.10. Details under the cut

This miracle is called Navisurfer II UBU-3G , here are those. specifications:

Main features
- Size 2 DIN
- 3G HSDPA broadband connection
- modified Ubuntu 10.10 for use in cars
- built-in Bluetooth
- GPS receiver and Navit Navigation System (a couple of sites about this system)

Hardware
Processor: 1300 MHz (Intel Celeron M)
Hard Drive: 250 GB, 2.5 "anti-shock mount
RAM: 2GB DDR2
GPS receiver / GPS antenna
3G HSDPA modem

Interfaces
3 x USB 2.0
2 x PS2
1 x Mic-in
1 x Line-in
1 x Line-out
1 x Serial Port
2 x DC 12V input
2 x AV-in (display only)
1 x auto power on rear view camera

Weight and dimensions
1.9 kg
178mm x 179mm x 100mm

Display
Size 7 "16: 9 TFT
Resolution 800x480

Modem
Networks: WCDMA / GSM
Data transmission protocols: HSDPA / UMTS / EDGE / GPRS / GSM
Max. speed: 7.2Mbps per input, and 384kbps per return
SIM cards: SIM / USIM
Frequencies: WCDMA / UMTS 2100MHz, EDGE / GPRS 850/900/1800 / 1900MHz
