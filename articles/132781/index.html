Release of the openSUSE 12.1 Linux distribution

Officially announced the release of Linux-distribution openSUSE 12.1. openSUSE 12.1 is the first release after the transition to a new release numbering.
The distribution is available for 32-bit and 64-bit in four builds:
  • universal installation DVD
  • GNOME 3.2 LiveCD
  • LiveCD with KDE 4.7
  • version to install over the network.

Key changes to openSUSE 12.1:
  • GNOME updated to GNOME 3.2
  • Linux 3.1 kernel update
  • GCC compiler set updated to version 4.6.2
  • Completed the transition to the system manager systemd.
  • Oracle JDK replaced with OpenJDK package
  • Added Tumbleweed repository (rolling updates)
  • User programs updated to versions:
    • Browsers Firefox 7, Chromium 17, Opera 11.52, Rekonq 0.8, Midori 0.4.1;
    • Audio Players Amarok 2.4.3, Clementine 0.7.3, Banshee 2.2;
    • Photo managers Digikam 2.2.0, Shotwell 0.11.5, F-spot 0.8.2;
    • Office suites LibreOffice 3.4.3; KOffice 2.3.1, Calligra 2.4 beta 3;
    • Programs for working with graphics Inkscape 0.48.2, Scribus 1.4rc6, GIMP 2.6.11, Blender 2.59, Kdenlive 0.8.2, OpenShot 1.4, PiTiVi 0.15;
  • Added application Snapper, which allows using the Btrfs file system to view the differences between the current and past state of files in the system, if necessary, changes can be rolled back, for example, to return a randomly deleted or overwritten file.


Congratulations to all users of the openSUSE distribution with the next release.
