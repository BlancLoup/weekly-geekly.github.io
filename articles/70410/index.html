Another App Store, this time for netbooks on the Intel Atom

The software store boom continued today with the announcement by Intel about the launch of the beta version of the Intel Atom Developer Program project. This is a system that allows you to create application stores for netbooks on Intel Atom processors, regardless of the operating system.

Like similar programs Apple, Microsoft and Palm, Intel promises to share revenue with developers in the proportion of 70/30.

Intel is working with original equipment manufacturers (OEMs) of netbooks and other partners to create storefronts through which approved applications will be sold. It is expected that all partners of the company, such as Acer, Asus, Dell, will participate in the development of the App Store.
