KDE is dead, long live KDE

Yes exactly. K Desktop Environment has died, now KDE is not decrypted in any way and means not a desktop environment, but a community of people.
This news somehow passed by (although she was already a couple of days).

So, for 13 years of the existence of the KDE brand, it was decided to rethink.
Main idea:
KDE is no longer a software.
Or in translation:
KDE is no longer human-made software. KDE is now the people who create the software.

Theses


  • The KDE name will be used and the K Desktop Environment will be decrypted.
  • KDE will have two definitions:
    • The community that creates free software for end users
    • A brand for a tech tree (umbrella brand or umbrella brand) created by the KDE community
  • What was previously called KDE will now be divided into several brands:

KDE Community


Why did such changes actually occur in the interpretation of the brand?
In fact, what used to be a small project to create a graphical shell for Linux, has now grown to an enormous size. There are not only applications, but also connections between people, a certain well-established culture (Akademy and KDE Camp for example). Now it is not the project that unites people, but people do not allow the project to separate, they keep it as one.

Technology tree



In fact, this tree is wider than what is released every six months and is now called KDE. Separately, there are applications such as Amarok, KDevelop, KOffice and others.

Timing


All of this is planned to be implemented in 4.4 and to complete these changes fully to exit 4.5.
