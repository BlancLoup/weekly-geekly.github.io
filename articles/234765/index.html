Dropbox introduced a single tariff plan: 1TB - $ 9.99 / month, $ 99 / year



Dropbox has updated the line of tariff plans. From today only 1 PRO tariff plan is available: 1TB for $ 9.99 / month or $ 99 per year .

And also new features for PRO-users were added: setting a password for a common link, setting a validity period for a common link, setting access for shared folders and remotely erasing files from a lost device.

Dropbox is a very popular cloud file storage, however, until today its prices were not so attractive compared to its competitors. From August 27, Dropbox simplifies its tariff plans and prices, leaving only one - 1TB for $ 9.99 / month ($ 120 per year) or $ 99 / year. (saving 17%) .



Also introduced new features available only for PRO-users:


Ability to set a password for access links

Provide an extra level of security for shared links. Only users who have a password will be able to access their content.



Ability to set expiration for access links

Protect sensitive files. Set the expiration date for shared links.



Access Management in Public Folders

Choose whether users can edit or only view files in public folders.



Remote erase files from a lost device

Delete data from a lost or stolen device. Be calm - a copy of the data is securely stored in Dropbox.



All PRO account benefits are described on a special page .



If you are already a PRO-user, then you do not need to switch to a new plan. Dropbox will do everything automatically over the next few days.

Thus, in the price range, Dropbox has caught up with its competitors Google Drive, Microsoft OneDrive, and bypasses Yandex.Disk (1TB - ~ $ 250 / year). However, a significant drawback of Dropbox is still only 2 GB of space for free accounts (the above competitors have 15GB) and the absence of intermediate tariff plans.

It is also worth noting that in August, Dropbox updated its mobile applications for Android and iOS, adding search in PDF, Microsoft Word and PowerPoint documents, faster and more convenient search of files in all folders, as well as support for animated large GIF files.
