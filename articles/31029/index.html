Google Reader for Wii

Without any extra pump, on the night of May 8, Google released a special version of Google Reader for the Wii console - the updated interface allows you to fully use the control buttons from the Wiimote. In this case, a similar version for PS3 has not yet been found.



Tips
The specialized Google Reader interface allows you to use the buttons of your Wiimote, giving you the ability to easily read news and manage subscriptions:
  • up / down - scroll up / down
  • right / left - next / previous entry
  • Button 1 - show subscriptions list
  • Button 2 - show links

In viewing subscriptions:
  • up / down - previous / next news source
  • right - select current source
  • left - close
  • - / + - collapse / expand folder





You can also try a new version of the interface without a prefix by typing http://www.google.com/reader/wii in your browser. A video about Google Wiider is also available:
