Interesting projects, tools, libraries


  • pypi.info - Improved search results for PYPI packages
    The list immediately shows the githab stars / followers of the author; sorting is available.
  • Gruvi - Network Library
    Uses libuv and implements many useful functions - such as setting up and maintaining SSL connections, abstraction of asynchronous operations in synchronous style, HTTP client and server. Designed to compete at the level of design convenience with projects such as gevent, concurrence, eventlet, asyncio
  • snowjob - and let it snow in the terminal
  • dnspython - automatically add dns records about the appearance of a new application server
    Able to add to BIND information about the new application server in such a way that he will participate in the distribution of the load on the system on the principle of round robin
  • EasyAI - artificial intelligence for board games
    A library without additional dependencies that implements the strategy of playing such games as tic-tac-toe, reversi, four in a row
  • PyBuilder - analog of Apache Maven on pure python
