3CX Enterprise Edition, an extension for Chrome CallVia3CX, integration with Zendesk and improved Webmeeting infrastructure

Last week for our company was eventful. So without further ado, let's get down to business!


Presented by 3CX Enterprise Edition


3CX Enterprise Edition is the most functional 3CX edition, including all the features of Pro version, and an additional license for a backup server. In the Enterprise edition, the backup server works in the hot-swap mode and instantly comes into operation when the primary server fails. The second license is one key that is activated on both the primary and the backup server. This solution will be in demand in large organizations, where you need to guarantee the fastest possible operation of the fault tolerance system. The second license works in conjunction with the 3CX built-in fault tolerance mechanism, also used in the Pro edition. Chrome CallVia3CX extension for calls from a web page


The extension allows you to make calls from the CRM system, e-mail message or any site with two mouse clicks. When you click on the number on the page, it is automatically copied to the 3CX client dialer for Windows, after which you click again to dial. No need to copy or enter the number manually! Any web page becomes an extension of your CRM! This small improvement drastically simplifies the work of employees who, by the nature of their activities, have to make many calls.



Work with clients is significantly accelerated, and the amount of work done increases. The new plugin quickly and easily integrates communications into the workflow of your organization. It uses applications and web sites that employees have become accustomed to use. You can install the new CallVia3CX extension from the Chrome Store app store .
