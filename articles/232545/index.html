Free CryptoLocker file decryption



The malicious program CryptoLocker belongs to the category of so-called "extortionists". It encrypts files on the hard disk and requires a ransom payment before downloading the secret key from the server to decrypt the files. However, since yesterday, users can do without payment: antivirus companies FireEye and Fox-IT jointly launched the DecryptCryptoLocker service to decrypt files that CryptoLocker has encrypted for free.

As a result of the operation, FireEye and Fox-IT managed to get a large number of secret keys that CryptoLocker uses to encrypt files. On the above site, users can use these keys to ‚Äúrelease‚Äù their documents without paying remuneration to extortionists.
