Summer Deadline: Top 10 Microsoft Virtual Academy Courses

This is the last article in a series of skills upgrading skills at the Microsoft MVA Virtual Academy. The previous two can be found here and here . It includes must-have courses for trained professionals: data center virtualization using Windows Server 2012 R2 and System Center 2012 R2, access security in corporate resources, details of deploying Windows 10 in the enterprise, Linux Cluster in Azure, virtualization of Linux workloads in Windows Server 2012 R2, hybrid solutions from Microsoft experts, deep immersion in Windows Server 2016, Windows Nano Server, Office 365 API for mobile applications and customization of SharePoint Server.




1. A series of three courses ‚ÄúData Center Virtualization Using Windows Server 2012 R2 and System Center 2012 R2‚Äù

With the release of Windows Server 2012 / R2, a large number of changes have occurred in the Microsoft hypervisor. In the three courses, you will learn about the key Windows Server and System Center technologies used to deploy, manage, and maintain the physical and virtual infrastructure of the data center. Under the link in the title you will find the first course in the series.

2. A series of two courses "Corporate devices"

For access to the corporate network, company employees often use personal devices, which are often impossible to include in the domain, and therefore apply traditional management approaches and tools to them. In this series of courses, you will learn about technologies that provide secure access to corporate resources from mobile devices running Windows, Windows Phone, iOS and Android. Under the link in the title you will find the first course in the series.

3. Windows 10: deployment, management, security

The course allows you to immerse yourself in the details of deploying the new OS across the enterprise, managing with the help of MDM solutions, ensuring the security of Windows 10 devices and preventing the leakage of corporate information. You will learn about the new configuration features of Windows 10, changes in the deployment tools of the Assessment and Deployment Kit, and Upgrade in-place features.

4. Build Linux Clusters in Azure

Linux Cluster in Azure - many virtual machines running the Linux operating system, hosted on the Microsoft Azure cloud system, and configured to work together to solve a single task. In this course, you will learn how to organize the work of a cluster of fault tolerance in the cloud: setting up fault-tolerant storage based on DRBD, cluster software Corosync and Pacemaker, a fencing system for fencing. By and large, these settings are applicable to the system of traditional architecture, without using the cloud.

5. Virtualization of Linux Workloads in Windows Server 2012 R2

Windows Server 2012 R2 provides advanced features for Linux and FreeBSD virtual machines. In this course, experts will tell you why Windows Server 2012 R2 is used for Linux and FreeBSD virtualization, how Hyper-V provides extensive support for Linux and FreeBSD workloads, and what resources you should learn to dive deeper into virtualization technology.

6. Hybrid solutions from Microsoft experts [eng]

You will learn about the security and transparency of your corporate cloud. It will be about how to simplify the process of collecting data and searching for them on several sources on a computer, which will allow you to easily determine the root cause of operational problems. You will also learn how to predict the cloud load and calculate as accurately as possible the capacities required in the future.

7. Deep immersion in Windows Server 2016 [eng]

Flexible infrastructure allows you to quickly and efficiently integrate complex systems. In this course, Microsoft experts will talk about security, scalability, and improving the efficiency of using Windows Server 2016, as well as analyze the Hyper-V background and talk about using Windows Nano Server.

8. Immersion in Windows Nano Server [eng]

Windows Nano Server is a compact version of the Windows Server 2016 operating system optimized for Windows Server and Hyper-V clouds and containers. It allows reducing the number of critical security bulletins to be installed by more than 90% and reducing the number of reboots required by more than 80%. In this course you will learn how to install, deploy and manage Windows Nano Server.

9. Installation and basic setup of SharePoint Server [eng]

This course is designed for those who want to learn how to deploy and configure the multi-server architecture of SharePoint Server 2013. You will learn how to deploy SharePoint 2013 on one or more servers to create different topologies that you can use to test and implement SharePoint 2013 solutions during different phases of the deployment life cycle. .

10. Deep immersion in the Office 365 API for mobile applications [eng]

You will learn about mobile application development using Visual Studio tools for Apache Cordova, the universal Windows platform (UWP), Xamarin, using the Office 365 API. We will also discuss native Xcode for iOS and Android Studio for Android.

More courses for trained professionals can be found on the Microsoft Virtual Academy website in the Advanced and Expert sections.
