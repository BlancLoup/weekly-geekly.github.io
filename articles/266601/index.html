Settings in the tab and other little things in the weekly assembly Vivaldi 1.0.270.16

Hello!

Today we present another test build, in which you will find the opportunity to view the browser settings in a separate tab, improving the design of the page loading indicator in case of choosing a dark theme, as well as some improvements in the tab switching speed. And now about this and other changes in a little more detail.

Settings in a separate tab

By popular demand, we added the ability to work with browser settings in a separate tab. More precisely, this possibility was before, but only the most inquisitive knew it - it was enough to enter the address vivaldi in the address bar : // settings . Now we have corrected some unpleasant errors in this format for displaying the settings and added the ability to enable this type of default settings. You can do this through the menu Tools -> Settings -> Appearance -> Open Settings in the tab .


Load indicator

The changes here are cosmetic, of course, but every little thing is pleasant if they allow you to increase the comfort of working with the application. We have slightly changed the indicator design for the dark browser theme. We hope you enjoy this change.
