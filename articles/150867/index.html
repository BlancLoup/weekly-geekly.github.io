PyCharm updated to version 2.6 with class diagrams, Flask, Python 3.3 and improved CSS / Sass / LESS editor

Even being partially on the DjangoCon that is now passing in Washington (and perhaps because of this) we were able to release today a big update on the most powerful development environment for Python & Django - PyCharm 2.6 .

Since last release in April, we have done the following:
  • Added class diagrams, model diagrams for Django & Google App Engine projects and for SQLAlchemy
  • We supported the development of web applications using the Flask framework.
  • Provided support for Python 3.3 (its release is scheduled for September).
  • They made the code analyzer even smarter - false warnings are much less now.
  • Redesigned support for CSS extensions, namely Sass / LESS / SCSS.
  • Added a number of convenient little things for everyday work in the environment.

More information about the changes can be found on the site . There are also a couple of useful videos about innovations.

Download , try. A 30-day trial period is attached.

Upgrading to version 2.6 is free for all from whom the license was acquired after September 4, 2011.

In addition, it is impossible not to mention the discounts available now up to 50% for personal licenses for JetBrains products, including PyCharm. The promotion will last until September 14. Hurry up!
