The correct bonus for your new server


Hi Habr! We have a special offer: when buying a data center - a cap as a gift!

Only now when you buy an HP ProLiant server (a list of models under the cat in the spoiler) based on Intel Xeon E5-2600 v3 series processors, you will receive an additional lifetime license for the HP iLO Essentials management tool with a 1 or 3 year support period. HP ProLiant servers fully disclose their capabilities only with the tools for remote monitoring and management of HP iLO .

Read more about how to get a license as a gift, read under the cut. Also, if you have already discovered the possibilities of iLO and you liked it, write your review in the comments to this post and get a memorable set of t-shirts and baseball caps (the number of sets is limited).

In the Essentials version, the HP iLO management tool provides the most requested features for working with small and medium-sized server parks. Among the possibilities:

- Connecting virtual images of drives to the server over the network (for example, to install the operating system)
- Remote launch of the graphical server console
- Configure advanced email notifications
- Inclusion of a server in the group iLO Federation
- Server management through open REST API
- Record all events occurring on the server in the "flight recorder"
- Convenient web interface
