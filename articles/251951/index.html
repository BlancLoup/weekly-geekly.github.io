Weekly assembly Vivaldi 1.0.118.2

Hello!

New week - and a new test build number 1.0.118.2. This time, the developers have focused on tabs, working with bookmarks, and also on improving the quality of localizations in terms of the use of singular or plural numbers. So, about this and many other things in more detail - below.

In this build, we have added one of the most requested features by users ‚Äî pinning tabs. When this option is activated, the tab will move to the left edge of the tab bar, and the tab itself will be reduced to the size of the site icon. At the same time, the cross of the tab closing will naturally disappear.

Some changes have been made to the work of the tabs themselves. Now, if you turned off the display of page thumbnails when you hover the mouse over a tab, only the site title will be displayed. Similar tips will be available on grouped tabs. Also, tooltips will display additional information - for example, the number of tabs in a group or the state of the tab pinning function.

Now about the changes regarding bookmarks. We have redesigned the bookmark creation popup dialog box, unifying it with similar items in the bookmarks bar.

Another small, but hopefully handy addition to the download panel. Starting from this assembly, you can see the progress of downloading files displayed directly on the panel icon, so information about ongoing downloads will be visible even when the sidebar is minimized.

In the new build, you can try another frequently requested feature - spatial navigation. It is carried out using the Shift + arrow keys. The function is still very test, so be prepared for any surprises :)

The new assembly has replenished with new localizations, the total number of which has reached 35. It should be noted that seven languages ‚Äã‚Äãfrom this list are supported only in Vivaldi and are absent in Chromium. Today we added the following browser interface languages:
