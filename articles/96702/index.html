APS LiveStreet Package at apsstandard.org

We are pleased to announce that LiveStreet 0.4.1 has become available as an APS package on apsstandard.org . Moreover, the package received Gold Application status, i.e. successfully passed a full test.
APS packages are designed for automatic installation of one-click web applications from hosting panels produced by Parallels (Plesk) and other developers. There are more than 200 different web applications in the apsstandard.org catalog from blogs to galleries. And we are very pleased that LiveStreet was among them - this is the first domestic free CMS placed in this directory.

We express our deep gratitude to COMTET ( komtet ), which prepared, tested and certified this package.
You can download the package on its page in the directory .

Application Packaging Standard (APS) is a web application packaging format designed to simplify the implementation of the Software-as-a-Service (SaaS) business model for cloud service providers and independent software developers.
