Update Veeam Availability Suite 9.5: what you need to know

Veeam has released a new update for the Veeam Availability Suite 9.5 Update 3 solution, which allows customers to centrally manage data in virtual , physical and multi-cloud environments.

Veeam president Peter McKay (Peter McKay) called the update VAS 9.5 U3 the largest release in the company's ten-year history. According to Peter, the update will provide a high level of availability for any data and applications.

Read more about the features of the update read under the cut.


/ photo FutUndBeidl CC

The update introduced many new features, and then - one of the most significant.

Native support for Veeam Agent for Microsoft Windows 2.1 and Linux v2

Users can centrally deploy the Veeam Agent using the control panel. Veeam Agent for Microsoft Windows 2.1 includes protection for Microsoft Windows Server failover clusters, and the Veeam Agent for Linux v2 has the ability to backup to the Veeam Cloud Connect repository, the ability to combine repositories into a single target system for backup , as well as encryption on the backup source side.

Data protection from insider cyber attacks in Veeam Cloud Connect

Service providers Veeam Cloud Connect were able to protect customer data from accidental or intentional deletion of information directly from the console Veeam Backup & Replication. On the new features Veeam Backup & Replication, you can read the link .

IBM Spectrum Virtualize Integration

New solution for working with snapshots of storage will reduce the time required to restore the system. Available for any IBM Spectrum Virtualize based storage, including IBM Storwize, Lenovo Storage V series and IBM / Cisco VersaStack.

New monitoring features in Veeam ONE

The function of verifying compliance with the " sovereignty of data ", such as the requirements of the GDPR, has been added . For this, production and backup infrastructures are marked with tags. Now, when attempting to restore data that violates the sovereignty of data, the user will receive a warning. In case of confirmation of the operation, the system will conduct a control analysis.

Universal Storage Integration API

A framework that allows customers to choose partners [companies] to integrate with the product Veeam Backup & Replication.
