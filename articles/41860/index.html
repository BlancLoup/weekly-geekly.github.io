Marker for easy scrolling pages

Now in Opera there is a marker that displays the place where the edge of the screen was before scrolling the page.



Remember: To use the marker, Opera Standard skin must be turned on.
It is included in Tools> Preferences> Advanced> Browsing> Show scroll marker

That's just not clear - that it only works at the beginning or end of the page?
UPD: found . You need to go to opera: config # UserPrefs | EnableScrollMarker and set the desired value
  • 0 - off
  • 1 - include at the beginning or end of the page
  • 2 - always show when scrolling
