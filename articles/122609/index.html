The Synaptic Package Manager will be removed from the standard Ubuntu 11.10 distribution.

The advanced package manager Synaptic Package Manager has been removed from the latest builds of Ubuntu 11.10 Oneiric Ocelot.

The exclusion of the Synaptic Package Manager from the standard Ubuntu delivery was expected and fits into the previously announced strategy for eliminating tools with overlapping functionality, instead of which a single Ubuntu Software Center tool was created. Many advanced features provided by Synaptic are scheduled to be added to the Ubuntu Software Center.

Subsequently, Synaptic can be easily installed with the help of its competitor.

However, due to the fact that Ubuntu 11.10 is currently in active development, such decisions are not final and may be changed.
