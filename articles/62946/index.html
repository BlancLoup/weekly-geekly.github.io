Apogee ONE - USB audio interface for Mac users

The company Apogee has released a new budget audio interface called ONE. The audio interface connects to MAC computers with G5 or Intel processors and OS X 10.5.7 operating systems or higher through a USB interface.





Apogee ONE is capable of recording one channel with a resolution of 24 bit / 48 kHz from one of three sources - a microphone input on an XLR with the possibility of phantom power, a high-resistance instrument input on a jack, or from a built-in electret microphone of circular orientation.

The reproduction of the signal to the monitors or headphones is carried out through the same jack, setting the appropriate settings in the instrument software.



Technical characteristics of Apogee One:

24 bit / 48 kHz
USB 2.0 interface
Stereo acoustics or headphones on a mini jack (1/8 Jack)
XLR microphone input with phantom power
Built-in circular microphone
Instrumental high impedance input
Multifunctional scroll wheel
Apogee's Maestro software
Compatibility with Apple computers
Dimensions: 4 "W x 6.3" L x 1.5 "H Inch

Package Content:

Audio interface
Cable adapter with XLR microphone input and high-impedance line-in input for instruments
USB cable
Apogee Software and CD Documentation
Quick start instruction

The recommended cost of Apogee One is 10,000 rubles.
Expected to be sold in Russia in July 2009.
