Ubuntu 9.04 Third Alpha released

* Integrated support for the ext4 file system, which can now be selected during installation via the Ubuntu Installer interface.
* The user interface has a new method for displaying system notifications and state change indicators (volume change, detection of a wifi network, readiness of USB Flash for extraction, etc.). The newly received notifications line up on the screen in a queue and look ephemeral, i.e. they do not require any reaction from the user and are hidden automatically and initially positioned as a means of intermediate information (the user is invited to view the details or take some actions directly in the application that generated the notification).


* Integrated code of the last test snapshot X Server 1.6 / X.Org 7.5;
* The installer Ubiquity has the ability to encrypt the contents of individual home directories of users.
