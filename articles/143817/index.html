Connection Raspberry Pi and Arduino



The blog of the project Raspberry Pi published a note in order to debunk any speculation about the competition Raspberry Pi and Arduino. In fact, nothing like this: the developers of Raspberry Pi believe that the Arduino is a great thing, and these two platforms do not compete at all, and can even work together.

Raspberry Pi and Arduino are supposed to be used completely for different things. The Arduino uses a microcontroller, and the Raspberry Pi uses a processor. For some things, the Raspberry Pi is better (for example, it is more convenient to connect it to the TV), while the Arduino is suitable for others.

Moreover, they can work together. For example, the Arduino board can be connected to the Raspberry Pi via USB - and controlled using Python (see instructions ).

The design of the bridge between the Raspberry Pi and the Arduino is almost ready - a device called Ponte is being designed by one of the enthusiasts.



Meanwhile, Adafruit is testing the Prototyping Pi Plate Plate I / O board, which works with the Raspberry Pi and allows you to connect peripheral devices via GPIO / I2C / SPI interfaces, etc.
