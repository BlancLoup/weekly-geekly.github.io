As always, quieter than water, lower than grass - the FreeBSD 9.2 release



Of the most notable changes:

  • ZFS file system now supports TRIM for SSD
  • ZFS also adds support for lz4 compression, which works almost three times faster than the old LZJB
  • Virtio drivers are now included in the GENERIC core for amd64 and i386 architectures
  • Bsdinstall installer can now install via HTTP
  • DTrace hooks are now enabled by default in the GENERIC kernel.
  • Updated ixgbe, cxgbe and Intel e1000 drivers
  • Firewire drivers excluded from GENERIC kernels
  • VARPD adds support for VLANs.




The set of software from the basic environment has also been updated:

  • xz 5.0.4
  • awk 20121220
  • oce 4.6.95.0
  • OpenSSL 0.9.8y
  • DTrace 1.9.0
  • Sendmail 8.14.7
  • OpenSSH 6.2p2
  • less 458
