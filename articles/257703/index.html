Treacherous Wolfhound. Transformation distribution Ubuntu

Less than a month ago, a new version of Ubuntu 15.04 was released, but now plans for the development of future versions of the distribution kit are in full swing.



I will try to put together all the information regarding Ubuntu 15.10 and 16.04 LTS, which has become known:

The Ubuntu online meeting ( Ubuntu Online Summit, (UOS)), devoted to Ubuntu, was held in early May and discussed interesting topics related not only to the development of the Ubuntu 15.10 distribution, but also to the individual components of Snappy, Unity 8, as well as interaction with the community .

The latest version of Ubuntu is currently 15.04. Where some revolutionary steps were planned and executed, for example, it is the replacement of upstart with systemd by default, with the possibility of switching back to upstart.
And now there are new details regarding the future version.
  • First, her codename is Wily Werewolf (Treacherous Wolfhound).
  • Secondly, the developers will continue to refine all bills that were made at 15.04 for the release of Ubuntu with long-term support - 16.04 LTS.
  • Third, this is the transition to the Mir default display server and Unity 8 shell.
  • Fourth, this is the use of gcc 5 and python 3.

In the current version of Ubuntu, the gcc compiler version 4.9.2 comes by default and in preparation for the release of LTS, it was decided to switch to gcc 5. To do this, a lot of work needs to be done to correct more than three hundred errors that occur during compilation due to the new libstdc + library + I.

More options for switching to Snappy. This system allows you to create a monolithic image of the system and instead of updating individual packages will use the mechanism for updating the entire system, as already involved in ChromeOS. This mechanism will be implemented in the assembly of Ubuntu Desktop Next, which in the future may replace the main build system for desktops - Ubuntu Desktop.
