Home Virtualization or Atom Virtualization

1. Linksys router WRT54GL

2. The Intel Desktop Board D510MO

Virtualization:


As a virtualization platform, Vmware server 2.0 was selected and the following virtual operating systems were created:
  1. Web server with Apache (RAM 384MB)
    a. Vtiger - for billing and storage of invoices.
    b. Gallery3 - for storing photos and videos. Web interface included.
    c. Rsync - is responsible for the distribution of photos and videos to remote PCs for viewing offline. Photographs are stored in 3 copies - in the original size, in a reduced and in the form of thumbnails.
    * cwRsync clients are on Windows, they get the photo in its original size.
    * rsync - is on the Android phone, and it gets a reduced view of 640x480
    d. Subsonic - MP3 Library and Streaming on the Web
    e. OCS Inventory NG - obtains information from 20 Windows and Linux systems
    f. Glpi - Displays information from OCS Inventory

  2. Kablink iFolder (RAM 384MB)
    * Opensorce analog Dropbox, clients installed on Windows machines.
  3. EBOX (RAM 384MB)
    Now renamed to Zentyal
    a. OpenVpn - for VPN access for 10 clients
    b. Squid - you never know where there is a filter on the sites, have your proxy.

  4. Windows XP (RAM 384MB)
    HomeSeer and JV Caddx Plug-In are installed on this virtualk to control home alarms. The main functionality is arming and disarming, as well as storing the history of events.
    a. GE Security Alarm Panel . NX-10
