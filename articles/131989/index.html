Windows 3.1 on Android

There is no limit to human ingenuity.
This time, we found a way to run Windows 3.1 (and possibly even higher) on the device running the Android OS.



The system is installed original and fully working, the only big minus is quite complicated management.

And so for this we need:
  1. The device with Android version 2.2 or higher
  2. ADosBox application
  3. Windows installation files
  4. Free space on the SD card or in the device itself (20 MB ( this is a problem .. ))
  5. Straight arms


Getting started:

  1. Install aDosBox from the market
  2. Extract the Windows 3.1 installation files to the memory card in the winsetupfiles folder
  3. Create an empty folder on the memory card - Windows
  4. Run aDosBox and enter the commands:
    • mount d / mnt / sdcard / winsetupfiles
    • mount c / mnt / sdcard / windows
    • D:
    • setup

  5. The installation starts.
  6. Perform all installation steps (Habrovchane-geeks should be able to install Win 3.1)
  7. PROFIT!


The next time you start aDosBox, the console will open again, in which you need to enter the commands:
  1. mount c / mnt / sdcard / windows
  2. c:
  3. windows \ win.com


Or add these 3 lines to the end of the file /mnt/sdcard/app-data/org.hystudio.android.dosbox/.dosbox/dosbox-0.74.conf to automatically start Windows when you start aDosBox.
