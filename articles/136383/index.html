Visa certified BlackBerry Bold 9900, Curve 9360, Bold 9790 and Curve 9380 smartphones as mobile payment devices

Not long ago, the BlackBerry Bold 9900 and Curve 9360 were approved by MasterCard as the first PayPass certified SIM smartphones. Now Visa has approved not only not only them, but also the Bold 9790 and Curve 9380 for use as mobile payment devices. Another great example of using NFC technology, which makes life easier by increasing the functionality of mobile devices.

All new Visa certified devices contain the Visa payWave application on a secure SIM card and support NFC (Near Field Communication) technology, a short-range communication standard that allows a mobile phone to be used to securely transfer payment information to contactless payment terminals.

Visa played a key role in shaping global standards for mobile payments, while ensuring that they are aligned with existing technologies and security standards for chip payment cards, and can be easily integrated into existing payment ecosystems. For example, the Visa payWave mobile app is compatible with existing contactless (NFC) payment terminals already installed in retail outlets around the world, which allows Visa account holders to simply switch on the phone before the payment terminal to pay.

The Yankee Group predicts that NFC transaction volumes will grow significantly, from $ 27 million in 2010 to $ 40 billion in 2014.
