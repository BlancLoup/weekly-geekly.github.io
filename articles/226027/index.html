Ciklum .NET / JS Subbotnik in Dnepropetrovsk, June 21

Great news for anyone interested in JavaScript and .NET development! Dnepropetrovsk office of Ciklum invites everyone to visit Ciklum JavaScript / .NET Saturday on the 21st of June .

Traditionally, Ciklum Saturdays will please everyone not only with technical reports, but also with the opportunity to communicate in an informal setting with speakers and participants of the event on various topics.


At the moment, we have formed a schedule of reports, but we are sure that we will be able to surprise all the participants with something else interesting;)

So, everyone who joins us will be able to get acquainted with the following topics prepared by our speakers.

Yaroslav Sergeev (Dnepropetrovsk) - Without a right for the complete rewrite:
- Dealing with a legacy JavaScript code
- How to return to the "Test First" principle
- Heavy weapons: automating a browser; comparing screenshots
- Put it on a diet: semi-automated testing in JavaScript
- Starting a journey of a thousand miles ...
- Q & A

Vitaliy Koval (Dnepropetrovsk) - Effective Code Review:
- Introduction
- Why to review code - advantages
- Why not to review code - disadvantages
- How to make code reviews: best practices
- Q & A

Vladimir Gaevoy (Dnepropetrovsk) - CQRS and Event Sourcing way:
- Typical architecture pros & cons
- CQRS concept
- Code example (NEventStore, CommonDomain)
- CQRS pros & cons
- Results

Gennady Krol (Dnepropetrovsk) - AngularJS Fundamentals:
- What is SPA?
- MVVM / MVC / MV *. Declarative and Imperative approaches
- AngularJS fundamentals: directives, bindings, scope, controllers
- Routing in AngularJS

Date: June 21, 2014 (Saturday)
Time: 11:00 - 15:30
Location: Dnepropetrovsk, Barrikadnaya St., 15-a, building 3. 12th floor.

Participation in the Subbotnik is free, subject to mandatory registration via the link goo.gl/5WX3xd

The final schedule of the event will be sent to all registered at the specified email address.

See you on Saturday!
