About circumventing access restrictions to postal services

This post I address primarily to users from Ukraine.

He is called to receive and send mail in the case of using Yandex, Mail.ru, etc.

So, with the adoption of the already known Law in Ukraine , a problem arose for all companies that delegated their domain to Yandex and used its email hosting.

Recall that this resource is in the sanction list under the number 275 .

It must be admitted that the problem is rather severe, since Ukrainian analogs (like ukr.net, meta.ua, bigmir.net, etc.) do not provide similar services.

Yes, there is Google - but the American giant doesn‚Äôt want a lot, but $ 5 for each box, if you want to bypass the 30 GB limit, the amount becomes $ 10 . I note that all these buns from Yandex were free.

It is clear that the first solution that comes to mind is a VPN - however, in the case of mail only, additional expenses for VPN in combination with the inevitable reduction in the speed of the connection are not acceptable.

An optimal way to solve the problem with a fairly high speed was found. About him - below.

There are many different free VPN clients on the net. Each of them has restrictions that are removed in case of payment. We will look at the Psiphon client, which can be downloaded here .

A feature of this client is several modes of operation, one of which includes the creation of SOCKS-proxy, which allows you to redirect traffic from the mail program, but do not affect browser traffic.

Unfortunately, you can implement this only on some third-party clients - Thunderbird, TheBat. Windows clients built in to Windows - Live Mail and Outlook do not allow you to have excellent connection settings, and therefore it is impossible to forward traffic to them separately from the browser.

So what needs to be done?

  1. Download the program Psiphon , but do not run it.
  2. If you use built-in OS clients

    Run and import the following data into the registry:
    Windows Registry Editor Version 5.00

    [HKEY_CURRENT_USER\Software\Psiphon3]
    "SkipBrowser"=dword:00000001
    "SkipProxySettings"=dword:00000000
    "SkipAutoConnect"=dword:00000000
    "SplitTunnel"=dword:00000000
    "DisableTimeouts"=dword:00000001
    "Transport"="VPN"
    "LocalHTTPProxyPort"=dword:00000000
    "LocalSOCKSProxyPort"=dword:00003039
    "SSHParentProxyHostname"=""
    "SSHParentProxyPort"=dword:00000000
    "SSHParentProxySkip"=dword:00000000
    "SystrayMinimize"=dword:00000001
    I tried to include in it the optimal Psiphon settings for work, but note that now all traffic will go through this VPN. The speed will decrease, sometimes significantly. You have to turn the client on and off yourself if you need to work with the network ‚Äúdirectly‚Äù, for example - you are watching streaming video or downloading a torrent.

    If you use Thunderbird or TheBat
    Run and import the following data into the registry:
    Windows Registry Editor Version 5.00

    [HKEY_CURRENT_USER\SOFTWARE\Psiphon3]
    "SkipBrowser"=dword:00000001
    "SkipProxySettings"=dword:00000001
    "SkipAutoConnect"=dword:00000000
    "SplitTunnel"=dword:00000000
    "DisableTimeouts"=dword:00000001
    "Transport"="CoreTransport"
    "LocalHTTPProxyPort"=dword:00000000
    "LocalSOCKSProxyPort"=dword:00003039
    "SSHParentProxyHostname"=""
    "SSHParentProxyPort"=dword:00000000
    "SSHParentProxySkip"=dword:00000000
    "SystrayMinimize"=dword:00000001
    In this case, the system will create a local SOCKS proxy on port 12345. Thus, it remains only to connect mail clients, the rest of the traffic will go directly.
  3. Run Psiphon. You can drag the shortcut to the program on the desktop or leave the file itself on the desktop.
  4. If you use built-in OS clients

    Just start your mail client and work. If you don‚Äôt need to work with mail, close the client and disconnect the connection in Psiphon. Well and vice versa.

    If you use Thunderbird or TheBat
    In the mail client settings, specify the connection through the local SOCKS proxy on port 12345. For Thunderbird, it looks like this:

    The SOCKS4 or SOCKS5 type is not fundamentally in principle, but as I was informed by some TheBat users, SOCKS5 should be indicated in this program without authorization.
    After that, the mail will work fine. In principle, you can even add Psiphon to autoload if all your mail is on blocked resources.
