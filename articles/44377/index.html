Google learned to view Microsoft Office 2007 files


The search giant Google has begun to support the limited format of Microsoft Office Open XML in its services, including Gmail, Google Docs, and the search itself. Now you can view HTML files created in Microsoft Office 2007: documents .docx, spreadsheets .xslx and presentations .pptx. Importing files to Google Docs has not yet been implemented, so for editing purposes it remains to use third-party services to convert from one format to another: for example, online services like Zamzar and Zoho Writer, desktop packages OpenOffice 3.0 and Microsoft Office 2007, the Microsoft Office Compatibility Pack extension.
