Oracle and Microsoft Partnership: Oracle Database, WebLogic Server, Oracle Linux and Java in the Windows Azure Cloud

On June 24, 2013, Oracle and Microsoft - long-standing rivals and competitors - announced a new strategic partnership. Under the new agreements, the two companies are joining forces to support Oracle‚Äôs products in virtualized and cloud environments based on Microsoft technologies ‚Äî the Windows Azure cloud platform and the Hyper-V hypervisor.

Support


As part of the agreement, Oracle will certify, support and license its products for launch on Microsoft technologies, including the Windows Azure cloud platform.

Starting from June 24, users of Oracle products that have a license for the company's products can use them to be placed in the cloud environment of Windows Azure, thanks to the updated licensing .

Products


As part of a strategic partnership, it was announced that companies intend to prepare and provide users with ready-made, pre-configured versions of the Oracle Database and Oracle WebLogic Server for easy deployment in the Windows Azure cloud.

In addition, users will be available images of virtual machines based on Oracle Linux, ready for deployment in the cloud environment of Windows Azure.

Java


As you know, Windows Azure has supported the development and deployment of applications on the Java platform from the very first day of its existence.

As part of the new agreements, it was announced that the Microsoft partnership with Oracle would improve the platform's support for Windows Azure, add full licensing and support for the Java platform and its environment from Oracle.
