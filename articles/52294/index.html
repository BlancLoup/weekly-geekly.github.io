A selection of apps for Google Android

During the last few weeks of using the Android platform, I have tested a lot of applications. Here I will talk about several that have caused the greatest interest. For convenience, applications are divided into categories.

Connection

Connectbot
Classic SSH client that allows you to manage the server.


Server list

ConnectBot allows you to work with multiple connections simultaneously. You can switch between them with the usual movement of your finger across the display from left to right and vice versa. The history of commands and terminals is turned over in a similar way, only in the vertical direction.


Running top

At first glance, it may seem that managing a server using a miniature keyboard is almost impossible, especially the lack of the CTRL, ESC keys, F1 - F10 function keys and the TAB keys. However, the application allows you to simulate these keys using other controls. A single click on the trackball corresponds to the CTRL key, a double click on the ESC. Function keys are emulated by simultaneously pressing the left SHIFT and the corresponding number. The TAB key corresponds to the right SHIFT.


Launched Midnight Commander

The connection has been tested using WiFi, EDGE and 3G. No problems with the response speed of the server were noticed.

Remotedroid
The application allows you to control a remote computer directly from your phone. For this purpose, the server side of the application must be installed on the computer, available for download on the project page. The server part is written in Java. Communication is carried out on the UDP port. You can control both the mouse and the keyboard.



Meebo im
The next application is an instant messenger. Supported protocols are AIM, MSN, Yahoo, Google Talk, Jabber and ICQ. This client is a multi-platform application (there are implementations for Android, iPhone and as a plug-in for Firefox) that implements communication through a web service, that is, there is no speech about the native client above the gone protocols. The implementation for Android is not yet completely stable. However, this is still the only application that simultaneously supports ICQ, Jabber and MSN.


ICQ contact list

Navigation

AndNav
AndNav is the first and still the only navigation system for the Android platform using built-in GPS. The current version on the Android Market uses data from Google Maps, but Google‚Äôs licensing policy restricts the use of applications for real-time navigation. For this reason, the creators decided to release a new version based on the maps from the OpenStreetMap project . This version can be downloaded from the home page of the application.


Menu

The maps are downloaded based on the current position of the phone, and the track is calculated on the server side. In the settings it is possible to specify the application, save all downloaded maps to SD. Navigation itself works like other systems. Navigation support is also implemented using the Text-To-Speech application.


Navigation at work

When testing navigation, I stumbled upon the problem of correct localization in small towns and villages. There were no problems in big cities.

Compass
If you suddenly find yourself in a place where there is no signal, then help will come from the electronic compass built into the phone. This application implements the management of this compass. To start it must be calibrated using the trackball. Subsequently, the real direction can be viewed along the arrow, or by numbers in degrees.


Navigation with a compass

Multimedia

Meridian Video Player
This video player is different in that it can be controlled not only by means of the menu, but also by gestures of fingers directly on the display. To pause or launch a movie, just click on the surface of the screen; to rewind, drag your finger to the left or right. In the settings it is possible to specify the automatic change of the playback orientation when the orientation of the phone changes. Unlike other players, you can play files from any folder, or directly from the Internet via RTSP (Real Time Streaming Protocol). When you select a file displays useful information about the format, length and video playback capabilities.


Select file to play

Android still has significant limitations on the supported video formats. However, on long wandering around the forums, a discussion was discovered with a great script using mencoder to transcode the video.

Instruments

Watts
This application monitors the battery status, it consists of two parts. The first part runs as a service in the background and periodically writes the status to the database. The second part - reads data from the database and displays them as a graph. There is also the possibility of voice output.


Low Battery Chart

Noisealert
The application controls the sound intensity in the vicinity of the phone. When a certain threshold is exceeded, it calls to a predefined number. It is also possible to set a controlled time period. This application was created for the night control of a crying child, but you can think of other uses.


NoiseAlert at work

Entertainment

FBReaderJ
FBReaderJ - a program for reading electronic books. Supports OEB, EPUB, FB2 formats, and can read them directly from ZIP, TAR or GZIP archives. The application has a lot of potential, but a stable version has not yet been released.


Open book

Amazed
This game is a miniature labyrinth. Your task is to take the ball through different labyrinths to a specific place. The direction of movement of the ball is determined by the tilt of the phone.
