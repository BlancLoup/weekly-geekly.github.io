Mojo SDK has leaked to the Network

As you know, one of the most unique features of the Palm webOS operating system is the Mojo development environment, which is not yet officially represented. With it, developers can create applications for the smartphone and draw the interface using standard HTML5, CSS and JavaScript tools. This is the first such platform, and therefore everyone is looking forward to its appearance with a special look, just to see how everything is done there.

Two weeks ago, developers reported that the official release of the SDK can be expected by the end of the summer . For closed testing, they posted an early beta version of the distribution kit to the internal local network, but the link accidentally fell into the wrong hands - and the day before yesterday the beta version of the Mojo SDK scattered all over the Internet.
