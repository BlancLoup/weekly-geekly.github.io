Announced by Microsoft Robotics Developer Studio 4 Beta



Today announced the new version of Microsoft Robotics Developer Studio (RDS) 4 Beta - an environment for developing applications for managing robots and simulating them. The new version of the environment announced the following important innovations:
  • New simulation tools that allow you to create robots working with Kinect without the presence of hardware;
  • published platform specification for hardware. Parallax, Inc. released the first set of hardware;
  • Support for Microsoft .NET Framework 4, XNA Game Studio 4.0 and Visual Studio 2010 has been added to RDS4.
For more information, visit the official development environment website http://www.microsoft.com/robotics/ .
