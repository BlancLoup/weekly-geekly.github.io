Google is going to combine Chrome OS and Android

According to the WSJ publication, Google is going to combine its desktop OS Chrome and the OS for Android mobile devices to create a universal OS that could run on various types of devices. A new OS may be released in 2017, and it may be presented next year. Android is a very popular OS from Google, which is not the case with Chrome OS, combining the latest with Android will allow it to run on computers.



The new OS will be able to work on a PC and its users will get access to the Google Play app store. Although Chrome OS will be merged with Android, it will continue to exist as an independent OS. This approach is similar to the one used by Microsoft, making Windows 10 available for both mobile devices and computers.
