Webinar "New features of RAD Studio XE8"

Dear friends!



Today Embarcadero released RAD Studio XE8 and we would like to share with you information about its features. For this, we have scheduled a webinar, which we will hold on April 9 (Thursday) at 12.00.

The head of the Embarcadero representative office in Russia and the CIS countries Kirill Rannev will tell not only what's new in the XE8 version, but also share the company's product strategy. Product Manager, Denis Vasilyev will elaborate on the technical features of the products in more detail and demonstrate how they can be used in development.


In the webinar program:

  • new compiler for iOS 64x
  • New cross-platform user interface development features using FireUI
  • new iOS platform components
  • Beacons support
  • application usage analytics
  • GetIt - package management manager
  • Castalia - a means of improving the performance of writing code
  • new EMS and FireDAC features
