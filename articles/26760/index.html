Acer One officially

And so Acer officially announced the release of the Acer One - another ultra-laptop laptop that ranks with the Dell Mini, ASUS Eee PC and other devices in this category.

What do we see inside this ultra-portable laptop that weighs less than 1kg?


* The processor will be, as expected, Intel Atom
* The operating system will be either Linux or Windows XP Home
* Screen - 8.9 "with a resolution of 1024x600
* The amount of RAM - 512MB or 1GB
* Built-in card reader 5 in 1, which will allow you to expand the memory of the laptop using the system "Smart File Manager" from Acer
* In the case of the Linux operating system, an 8GB NAND flash disk will be used as the drive.
* In the case of the operating system Windows XP Home - 80GB hard drive
* Since laptop is still portable it will be built-in support for Wi-Fi
* Optionally will be able to connect to WiMAX or HSDPA
* Claimed battery life - 6 hours, which is possible thanks to the Intel Atom, which does not need cooling fan
* Dimensions of this miracle - 24x17cm, weight - less than 1kg
* The color of the case is blue and white, although the company promises to present two more options later this summer - pink and brown.

The price of this ultra-portable laptop is $ 399 for a model with Linux and an 8GB NAND flash drive and this model will be available on July 10th, $ 599 for a version with Windows XP and an 80GB hard drive.
