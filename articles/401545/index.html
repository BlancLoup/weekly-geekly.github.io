Chief, and I see you. Intel Unite - a tool for professional communication



Communications rule the world. It is the interaction that makes a group of people a team, at least in football, at least in the development of a software product. How to make communication of people, regardless of the physical distance between them, more convenient, natural, while costing a minimum of financial and labor costs? Intel offers its own solution to this problem. It is called Unite - a software package for professional communication.

Intel Unite consists of three components:

  • The authorization server is responsible for the security of the system as a whole and is responsible for issuing passwords for the initial connection;
  • A computer hub connected to various multimedia devices - the participants of communication connect to it;
  • Client computers with client software that establish a connection with the hub.

Sketchy and in English it looks like this


The whole system works as follows. Team members install client software in their workplaces. To connect to the hub, you will need a password generated by the security server. After that, its resources (that is, all connected media devices) become available to all participants in the conversation. Well, communication itself becomes even more accessible, at the same time it is possible to conduct both general discussion and one-on-one conversations. Naturally, it is possible to exchange files and text messages. Unite traffic is encrypted with a 256-bit key.

At first glance, it seems that the Unite hub is under a substantial load and you need to install a powerful server to cope with it. However, a noisy and cumbersome server has no place in the negotiation room, and the load can be reduced using proprietary Intel vPro technology. VPro support is a requirement for the gateway platform. But the size, on the contrary, it is better to take a smaller to save space. Intel has a computer, as if born to be a Unite hub. This, you guessed it, Intel NUC - just find an option with vPro support in the model range. Other manufacturers also offer their own versions of mini-computers for meeting rooms that are compatible with Unite - in the photo below, for example, the very nice HP Elite Slice for Meeting Rooms.


Another interesting platform for deploying Intel Unite is a video panel with an OPS ( Open Pluggable Specification ) slot and, accordingly, an OPS computer. OPS is an open standard for pairing a video output device and a computer. The OPS computer case has the shape of a parallelepiped and is equipped with an 80-pin connector for connecting to a monitor, with which it can be said to be integrated, thereby reducing the number of wires. This is what an OPS-compatible computer looks like.


A distinctive feature of Intel Unite is the ability to interface with third-party add-on modules; to create them there is a special SDK . Using plug-ins, for example, you can implement meeting management tools in a third-party product or in a cloud-based web conferencing solution to enable Intel Unite to connect to web conferencing. And equipment manufacturers for conference rooms (microphones with speakers, cameras, etc.) can use plug-ins to create a convenient interface for controlling equipment (volume, video transmission, lighting, etc.).

Currently, the third version of Intel Unite is available for download , the clients are written for iOS, MacOS and Windows. If you are in search of meeting software, try this, it may be suitable for you.
