Chrome tests the combined menu

Alex Chitu noted that in the latest test builds of Chromium and Google Chrome there was an opportunity to use the combined menu. It includes most of the options that were available in the settings menu of the page and in the tools menu.

If you are using the latest Chromium test build or Google Chrome, you can enable this menu by adding the --new-wrench-menu flag to the browser shortcut.

In the first version, for compactness, the options for cutting, copying and pasting text are combined in one item, and in another paragraph, the zoom options with the inclusion of full-screen view are similarly combined.
