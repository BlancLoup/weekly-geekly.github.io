0.5% of the Internet may not be available for Hamachi users.


Hamachi is a fairly common program for creating VPN networks.

Recently we bought a server in Hetzner, the server was issued ip 5.9.28.205. Some users started complaining about the inaccessibility of the site. After a brief study, they discovered that they had the Hamachi program installed. It is used internally by the public subnet 5.0.0.0/8. For a long time, this subnet was unallocated, but the addresses are running out, so in November 2010 it was transferred to RIPE. And more recently distributed to end users. If Hamachi is installed, then it registers the route and allows all traffic from 5.0.0.0/8 through itself, and it is impossible to go to real sites.

The problem has been known for a long time, the developers do nothing .
If you have installed hamachi - learn that the inoperability of some sites may be due to it. To pick up without demand 0.5% of the address space of ipv4 is not a joke =)
