IE8 can be removed from Windows 7

In the newly created 7048 build, it is possible to completely remove Internet Explorer 8 from the browser. This is possible for the first time since Microsoft introduced IE tightly into the system in 1997.

For a decade, the company has repeatedly stated that the browser is "an integral part of the system" and sits so deep in it that it is theoretically impossible to remove it. Now we see that this is quite feasible (albeit with two reboots). Not only iexplore.exe, but absolutely all browser components are removed from the system.



Obviously, the ability to remove IE8 from Windows 7 appeared as a result of the antitrust process , which in the European Union is being led against Microsoft by manufacturers of alternative browsers Opera and Firefox.

Build 7048 hit the Web in informal ways on Monday and is distributed through all the major torrents. The ability to remove IE8 for the first time is fixed by one of the bloggers who installed the distribution kit.
