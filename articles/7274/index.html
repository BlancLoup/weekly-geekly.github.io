Launchy 1.25

Launchy , one of the most favorite utilities on Habrahabr (confirmed here or here ), has grown to version 1.25. If you briefly describe the purpose of the program, Launchy serves to launch the programs you need as quickly as possible with just a few keys on the keyboard.

It works completely unobtrusively, when you press a custom keyboard shortcut, Alt + Space displays its window in the form of a search string in which you enter the first characters of the name of the program you need. While you are typing, Launchy literally from the second character based on the information received after scanning the Start folder, in the style of Google Suggest offers the most likely options that you can run by typing. With some skill, it saves a lot of time, eliminating the need to wade through the wilds of the Start menu with the mouse each time.

In addition to directly launching programs from the Start menu, Laucnhy using native plugins allows you to scan Bookmarks (Bookmarks), quickly search the Internet (Alt + Space -> the first letters of the search engine (go Google, etc.) -> Tab -> "search phrase "-> Enter), do calculations using the built-in calculator.
