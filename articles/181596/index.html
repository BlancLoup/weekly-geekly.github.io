CheckiNumber.ru: another result of code therapy

Hello!

Working as a manager in the software development industry, I believe that in order to successfully complete my tasks, one way or another, it is necessary to understand how the work of your guys looks. The only way to do this is to take and try what I regularly do, calling it codotherapy.

I want to present the next result of my code therapy: CheckiNumber.ru is a service that allows you to check numbers. There is a service where you can check in your thoughts, there is a service where you can check in the fact of your stay in a certain place, but there is no place to check the numbers, and sometimes it would be convenient.

For example, somewhere I heard that if you pull up 1000 times (in total, and not at one time), then as a result, the number of pull-ups per series will double. And I decided to check it out, just where to write down such things? You can use ekselki, google docks, but I decided to make a specialized tool, where after each approach to the horizontal bar you can add a figure. The result is a graph:



And you can also ask your friends to make bets with Facebook on success or failure of the plan (at the same time it will be seen whether friends believe in you or not):



Counters are not only cumulative, when the next number is added to the cumulative amount, but also rewritable, when the number with a checker simply overwrites the value of the counter. This is necessary, for example, to measure its own weight. I decided to lose weight, set a target value, a target date, and now after each check of my current weight I can follow the trend:
