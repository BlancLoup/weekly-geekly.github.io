User Interface Guide for Windows 7



Microsoft has released an official guide, which describes in detail the principles for building user applications and interfaces for Windows 7. The guide is very large - 882 pages that give advice, specific examples, recommendations are given.

The whole guide is divided into 10 large sections:

- design principles;
- controls;
- commands (hotkeys, menus, etc.);
- text and its styles;
- messages: warnings, errors, notifications;
- user interaction: keyboard, mouse, sensors, access for people with disabilities;
- window;
- visual elements: icons, layout, fonts, colors, and more;
- user experience;
- Windows environment.
