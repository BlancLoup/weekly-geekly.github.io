Postmaster.mail.ru - smart statistics

The topic this topic is devoted to is specific by definition - mailing lists. Nevertheless, we are sure that some habrazhiteli sometimes use this type of communication. How to make it effective for you and painless for recipients is written below.

Often, users consider spamming to which they had once subscribed, but this is not quite so formally. But, of course, the owners of mailboxes are not up to the terminological subtleties - and, as a rule, they call spam any spam.

The purpose of the Postmaster.mail.ru service, to which this article is devoted, is to help web projects control the flow of email correspondence to users' mailboxes.



Since the publication of the first post about the new service for senders of mass mailings, postmaster.mail.ru, we have received dozens of feedback and suggestions.

During this time, several thousand Internet projects have joined the service (including very large ones, such as Groupon, for example). We take into account your wishes and will try to implement them all.

Over the past period, quite a few changes occurred in the service:
  • Appeared section ‚ÄúLetters‚Äù. In it you can see detailed statistics on different types of letters.
  • In the detailed statistics on the domain or on a specific letter, new counters have appeared (the number of open letters, the number of deleted open letters and the number of letters that have been deleted unopened).
  • Partially redesigned interface.
  • Improved accuracy of statistics collection.

Separate statistics on the types of letters


In the new section ‚ÄúLetters‚Äù you can see detailed statistics for each type of letters. For example, you can see the statistics on the user registration letters, and, regardless of this, the statistics on the password recovery letters.



In order to start using this new feature, you need to add a special header to each letter: X-Mailru-Msgtype. As a value, you can use an arbitrary set of English letters and numbers up to 32 characters long. The header value will be used to group messages of the same type.

New statistics


Also now it is possible to view the statistics on the letters opened by users. This is unique information, since it is impossible to find out these numbers in any other way (and, as far as we know, not a single mail in the world yet allows senders to find out such numbers). On the "Details" page for the selected domain or letter you can see:
  • The number of open letters for the selected period (at the moment it is a simple counter of non-unique discoveries, that is, if the user opened the same letter 10 times a day, then 10 will be added to the counter).
  • The number of deleted letters that did not open. By this indicator you can judge how interesting the letters are to the audience.
  • The number of letters that were deleted after opening.



I want to draw your attention to the fact that these statistics are collected for the current day, i.e. If yesterday you sent, for example, 100 letters, and today not one, then the statistics for today on discoveries and deletions will not be zero, because users read your letters not instantly, but with some delay.

All this, we hope, will help senders to more thoroughly analyze the flow of letters and correct it for the better, focusing on the reaction of users.

In the near future - the implementation of FBL technology (although this is not directly related to the postmaster service), as well as a special API for accessing postmaster data.

We are pleased to read your feedback on working with postmaster.mail.ru in the comments or in a letter to v.bespalov@corp.mail.ru.

Vasily Bespalov,
Head Antispam Mail.Ru.
