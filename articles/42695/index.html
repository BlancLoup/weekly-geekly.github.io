Simple ignoring annoying blogs (implementation)

Inspired by Simple ignoring and Turn your eyes here, since you decided to write on Habr√©!
I propose a simple solution to this problem for the Firefox browser using Geasemonkey.

With the increasing popularity of Habr, you can see that the site has become very diverse, now you can find articles in very different directions.
However, sometimes you want to get posts on a narrower subject, in other words, you want to filter information that is not interesting (or not so interesting to spend time on it at the moment).

A script was written for this purpose.

How it works:


After rendering each page, a Geasemonkey script is launched. The script identifies blogs by reference and hides those added to the ‚Äúblack list‚Äù.


Installation


1) Install Greasemonkey (If not already installed)
2) Install the script Habrahabr.ru Blog Filter

Using


Ban: hover the cursor over the name of the annoying blog, click on the appeared cross.
Unzip: Tools -> GreaseMonkey -> Script commands -> Habr -> Show all blogs. Hover the cursor on the name of the blog that you want to unban, click on the appeared positive. (the same can be done via the GreaseMonkey icon in the status bar)

Side effects :)


At the moment, only one is noticed - the more banned blogs, the more likely to get a blank page (because the content is only hidden, it may turn out that the entire content of the page is hidden)
