Wapnest - free wap-exchanger of the new generation


The main purpose of Wapnest is to download files from the computer to the phone, without using a cable or Bluetooth. It's simple: choose a file, upload it to the exchanger, get the code to download and anyone with this code can pick up a file from wap.wapnest.ru .
In addition, you can upload files to a private area where only the owner can view and download them.



Create communities, send links to your files to your friends, subscribe to your favorite authors or communities, search and share with others the best files for mobile phones.


There are no restrictions on the file type, and the maximum file size is limited to 5 megabytes.

PS At the moment, the resource does not allow placing files in the .jar format in order to prevent unauthorized sending sms from the user's phone. In the near future, the functionality of downloading jar files will be added
