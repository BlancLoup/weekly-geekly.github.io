A set of useful tools for working with DNS

The ViewDNS.info site contains useful tools for site administrators - whois, traceroute, reverse IP domain search, port scanner, checking the distribution of changes in the DNS, searching for the geographical location of IP addresses and much more.



Some of the functionality is available after registration as a REST API, XML and JSON formats are supported. Basic access (web interface and up to 250 API calls per month) is free.
