Monetizing Widgets
Widgets for Russia in general is still a related topic, and their development is just beginning. (not to mention monetization). Therefore, I will tell you how things are in the west.

So for example, let's take the addthis.com widget - a useful button widget (customizable), with which users can add a site or material to various bookmarking and social services. networks. (Russian analogue: odnaknopka.ru, may the creators forgive me, but it‚Äôs done poorly, if there is a better analog - I take two!)

A great number of sites use it, for example, a large blog like techcrunch.com. That is a huge demand, but how can you get money from it. If you think about it, 3 options will be obvious:
1. To place advertising on the widget site, but in this case the costs for servers, developers, etc. are unlikely. pay off because Only site owners visit the site and not every day.
2. Make the service paid, for $ 2-5 per site, for example. The option is acceptable in principle (for the West), but there is still a service sharethis.com (the same meaning), and suddenly it will remain free?
3. Carefully place ads inside the widget. - the most optimal

Tools for widget guides
But what kind of advertising to choose for the widget, because if you have for example a widget - an audio player, or there is a polling service, a chat, do not insert stupid adsense there? Of course, you can develop your own system for displaying ads in widgets for a long time ...

Services such as clearspring.com come to the rescue . This multifunctional service provides the following features:
- counter for widgets. Inserts the code into the widget (AS2, AS3, Java) and complete statistics are maintained, who posted the widget where, how many times it was viewed and how many times it was clicked.
- there is an opportunity to make quickly facebook app, hi5 app and so on.
- it is planned to enter the advertising system for widgets, and this is salvation, this is for widgets as adsense for sites (set up an advertisement, received and stuck the code in the widget and enjoy)
- a whole lot more (studying)

In general, friends, if you do widgets \ and do not see the light in the tunnel (monetization), do not despair, everything is just ahead.
