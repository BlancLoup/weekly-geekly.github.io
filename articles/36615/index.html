Nouz Blog Engine

If you are looking for a blog engine, then today the choice is very, very wide. The most popular products offer monstrous functionality, but the number of faceless and uninteresting blogs does not diminish from this. There are micromotors that occupy only a couple of tens of kilobytes, but, on the contrary, they suffer from a lack of functions necessary for convenient operation.

I am working on an option that should be the golden mean. Nouz's blog engine is minimalistic, but not without amenities. Free and open source, PHP / MySQL.

Main characteristics at the moment:
  • Installation is done through a convenient web interface.
  • Admin panel integrated. When you are not logged in, the blog looks like this:

    And when logged in, the controls are simply added:
    "Q: 5" means that 5 requests were made to the database. Immediately, I note that requests are made only if you are logged in as an admin, otherwise the cache is used.

  • The engine is multilingual and supports UTF-8. Included are Russian and English language packs.
  • There are no ready-made changeable themes, but there is a template system. To make serious changes to the design, it is enough to delve into two files - design / styles.css and views / page_default.php. The rest of the templates are quite complex, I think about simplifying them.
  • There is a file manager with which you can conveniently upload a couple of files.
In future versions, the possibility of a convenient blog backup, antispam, and possibly even auto-update of the engine will be added.

You can always download the latest version on Nouz.org . Enjoy using!
