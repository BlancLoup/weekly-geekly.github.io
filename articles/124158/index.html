What is the site under the hood

There is another useful tool for the webmaster - UnderTheSite . He studies the HTTP headers, the code of the pages (the crawler downloads about ten pages), JavaScript and CSS to identify the characteristic traces of certain technologies (matcher's). One request - and the page immediately displays the full list of what was found. You can immediately type URLs like underthesite.com/sites/habrahabr.ru. For example:

Google : Google Webmaster Tools Verification, Google Analytics, Google Web Server, UTF-8 Unicode charset, Google Web Fonts, HTML5, Google Website Optimizer, YUI Library.

Yandex : jQuery, Open Graph, OpenID Protocol, OpenID, OpenSearch, UTF-8 Unicode charset.

Habrahabr : Google Analytics, Nginx, MooTools, swfobject, UTF-8 Unicode charset.

The author asks to add new technologies to determine : everyone can do it on their own after registration.
