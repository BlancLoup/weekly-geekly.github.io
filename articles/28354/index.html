Starting something new

A bit of theory.

Psychologists have long noticed: it is much easier for a person to get to the top if he knows that someone has already been there. Psychologically, it is easier for a person to repeat than to create something new. It is easier to copy Facebook than to write VKontakte from scratch. (And if even VKontakte appeared, then you can stamp such sites in dozens). Easier to take ready-made design and overall concept. But.
But the work of this much less will not. It is reliably known that the project implementation stage makes up more than 90% of the total project implementation time.
The life cycle of ANY significant project consists of 4 stages:
  1. Definition
  2. Planning
  3. Performance
  4. Completion
It doesn‚Äôt matter whether you go to Olympus, make a website or arrange a mega-party - you will inevitably run into each of the 4 stages in one way or another. ‚ÄúAssigning‚Äù someone else‚Äôs idea. Copying someone else's project. Walking on the tracks of others to your dream, you simplify the first two stages. But not the third, which is the most significant. Only one thing is important - psychologically, you get the confidence that IT IS POSSIBLE. If someone made it to you - it is possible to repeat. Therefore, if it is essentially simpler, then it is psychologically.

Practice.

Starting something new that no one else has done before you, apply the following psychotechnique. (It is assumed that you are familiar with the subject of the project that you are going to implement. The technique will be useful if you already have a clear idea and at least a rough idea of ‚Äã‚Äãthe ways of implementation).
  1. Sit back or lie down
  2. Take a deep breath and take a deep breath. Breathe rhythmically.
  3. Relax your face and arm muscles
  4. Imagine that your project has already been implemented in a parallel universe. The goal has already been achieved by someone and it is normal. Feel that it is NORMAL and NATURALLY. Feel fully that it is POSSIBLE.
  5. Mentally enlarge this picture to HUGE size and make the picture Brighter, if possible.
  6. Watch for details. Realize exactly how this happened. Stay in this state for about 5 minutes.
  7. Go back to this reality and just copy the project you saw - it's so easy! ;)

The main idea

Main idea: Climbing on an unattainable peak, imagine that someone has already been there before you.
