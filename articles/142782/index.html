Way2Cloud: new design, new structure, new life

On April 23, a significant event happened - we finally completed all the work related to the Way2Cloud project and released a completely new version of the portal. It is worth noting that the once very popular project in RuNet has lost its significance over time, which was due to a number of reasons. Therefore, to revive this, of course, an interesting project, the main development team began with renewed vigor. After several days of discussions and proposals, it was decided to make Way2Cloud a kind of central resource, where materials useful for the entire Russian-speaking community will be concentrated, covering the cloud computing market.

The project structure is relatively simple. All content of the resource is distributed among the so-called hubs, each of which is a peculiar way of logical grouping of publications. Today, there are 3 hubs in the system, representing the most well-known providers of cloud services:
  • Windows Azure ( http://azure.way2cloud.com )
  • Amazon Web Services ( http://aws.way2cloud.com )
  • Google App Engine ( http://gae.way2cloud.com )

It is possible that in the future this list will be updated. Currently, the most complete and informative hub is Windows Azure, but we are actively looking for authors who would agree to work on the contents of the other two hubs.
Another way to distribute content into groups are sections, we have 9 of them, this is:
  • Data storage and access ( http://www.way2cloud.com/storage )
  • Deploy, manage, and support ( http://www.way2cloud.com/management )
  • Patterns and architectural guides ( http://www.way2cloud.com/patterns )
  • Security ( http://www.way2cloud.com/security )
  • Integration of services and data ( http://www.way2cloud.com/integration )
  • Application Development ( http://www.way2cloud.com/development )
  • Monitoring and Analysis ( http://www.way2cloud.com/monitoring )
  • Migrating data and applications ( http://www.way2cloud.com/migration )
  • Scaling and performance ( http://www.way2cloud.com/performancen )

Naturally, we could not circumvent the social component of the project. In order to always stay up to date with all the events taking place on Way2Cloud, simply subscribe to a Facebook page , a Google Plus fan page, or follow Way2Cloud on Twitter .
