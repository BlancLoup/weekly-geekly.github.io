Hyper-V will be available for free.

The new version of the virtualization program Hyper-V with the Live Migration function will be distributed free of charge and it can be freely downloaded directly from the official site. This was announced yesterday by Microsoft. This is quite unexpected news, because Microsoft has already set the product price at $ 28, but now, less than a month before the release of the new version, such a change in strategy.

Obviously, in this way Microsoft is trying to increase the pressure on the main competitor of VMware, which was declared a holy war. They say that against VMware, the Redmond giant uses the same strategy that it used in its time against Netscape Communications in the browser market. Indeed, Microsoft first included Hyper-V in the standard distribution of Windows Server 2008, and now it is beginning to distribute a separate version for free.

VMware is the most famous manufacturer of virtual machines, distributing some of its software for free. By some estimates, VMware controls up to 80% of the market.
