ABC WEB-developer

A small list of the most, in my opinion, useful sites for any WEB-developer who is interested in the field of web design, layout and programming. I think everyone among this list will find for itself sites which for a long time will enter into its RSS feed.

A - A List Apart




A List Apart contains the best tips and opinions from leading industry minds.
Still:
  • Abduzeedo (design blog)
  • Arbenting (design and development blog)
  • Alltop (selection of web design links)
  • Affiliate Ranger (design and development blog)
  • Ajaxian (news and articles for web developers)

B - Build Internet!




Build Internet! this is a relatively new blog, but it has already staked itself among the best blogs about web design and development and much more.
Still:
  • Boagworld (web design blog and podcast)
  • Best Web Gallery (CSS gallery)
  • Bittbox (design blog)

C - CSS Tricks




This is the site that will be useful to anyone who deals with CSS. Contains many tricks and CSS techniques.
Still:
  • CSS3 Info ( CSS3 Blog)
  • Cats Who Code (web development blog)
  • CSS Line (CSS gallery)
  • Creattica Daily (daily web design news blog)
  • CSS Globe (CSS News)

D - Design Bump and Design Moo






And resources like Design Bump and Design Moo have a social aspect of voting, like in Digg, but they are completely focused on web design and development.
Still:
  • Design Float (similar to Design Bump and Design Moo)
  • Design-Newz (web design and development news)
  • DevSnippets (web development blog)
  • Design Jojo (featured design news and blog posts)
