Mushroom Kingdom Fusion v0.41 Beta



Game Description


Mushroom Kingdom Fusion is a free platform game based on the Super Mario Bros game series, but the creators of the game have gone much further. The game surprisingly combines the elements, the main characters and the possibilities of several games, such hits of their time as Halo, Sonic, Doom, Castlevania, Ghouls and Ghosts, Mega Man and several other games.

Trailer




Where to get the game


The game is absolutely free and is still in Beta-stage, but it has already been done very, very much. You can find it on the official forum Mushroom Kingdom Fusion .

First, download version 0.4 , install it, then download the patch 0.41 and unpack the files of the new version into the folder with the installed game.
