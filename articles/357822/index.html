What, in fact, was done at the base

Launching under Windows requires .NET Framework 4.5, under Linux - Mono 4.6. The parameters are described in detail in the readme of the project.

  • Files larger than the maximum size are automatically broken into pieces when downloaded and collected back when downloaded.
