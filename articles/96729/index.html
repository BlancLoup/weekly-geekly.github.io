Major update in the NVIDIA ForceWare driver lineup

June 15, NVIDIA has provided access to download the new diver 257.21, which just can not be missed by the attention of users of video cards based on NVIDIA.

This release is a logical continuation of the driver ForceWare 257.15 Beta, released on May 24th.

In fact, this is the first driver of the 2xx line. By tradition, among the supported GeForce 6, 7, 8, 9, 100, 200, 300, 400 and ION series.

New in version 257.21

  • Added support for NVIDIA 3D Vision technology for Blu-ray 3D.
  • Allows you to increase the performance of GeForce GTX 400 series GPUs in several PC games.
  1. Updating the PhysX system software to version 9.10.0223.
  2. Added support for OpenGL 4.0 in GeForce GTX 400 series graphics processors.
  3. Added support for CUDA Toolkit 3.1, providing significant performance gains in mathematical operations with double precision . More information on the website CUDA Zone.
  4. Added support for new extreme smoothing modes for 3-way SLI PCs, up to SLI48x AA in GeForce 200 series GPUs and up to SLI96x AA in GeForce GTX 400 series.
  5. Added support for new "Quality" mode in the NVIDIA Ambient Occlusion control panel.
  6. Added a new SLI and PhysX settings page in the NVIDIA Control Panel, providing excellent control of multi-GPU configurations.
  7. Added new feature in NVIDIA Control Panel, providing excellent CUDA GPU configuration management. Now the user can choose which GPU will provide the calculations for each CUDA application.
  8. 3D Vision users can download v257.12 3D Vision drivers here .
  9. Multiple bug fixes.

Additional Information

  • Installs the HD Audio driver version 1.0.9.1.
  • Support for new Adobe CS5 GPU-accelerated features .
  • GPU acceleration support provides smoother online HD video playback using Adobe Flash 10.1 . Details here .
  • Support for new version of software to improve the quality of video vReveal from MotionDSP, equipped with the ability to output video in HD format. Users of NVIDIA products can download the new version of vReveal with support for the output SD format here .
  • DirectCompute support in Windows 7 when working with GeForce 8 series and later graphics processors.
  • OpenCL 1.0 support in all GeForce 8 series and later GPUs.
  • OpenGL 3.3 support for GeForce 8 series and later GPUs.
  • Support for single-GPU and NVIDIA SLI technology on SLI-certified motherboards based on Intel X58 with support for DirectX 9, DirectX 10, DirectX 11 and OpenGL, including 3-way SLI, Quad SLI and SLI.
  • Allows you to overclock the GPU and monitor the temperature using NVIDIA system tools.

So, as you can see, nVidia has prepared many goodies for us.

UPD. As prompted by shipovalov , the drivers for Quadro were also updated (albeit with a delay of 2 days) to 258.49 WHQL .
