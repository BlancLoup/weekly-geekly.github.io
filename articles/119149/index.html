Yandex.Metrica opened API

Another Yandex service opens the API. Third-party developers have been able to create applications for working with reports Yandex. Metrics.

The Yandex.Metrica API is built on REST principles and uses OAuth 2.0 authentication. The program interface allows you to receive data from all reports on the site, work with existing counters and create new ones. It is possible to manage accounts, access rights, goals, filters and operations.

The API allows you to create applications for a variety of purposes. For example:
  • CMS developers, forums or blog engines can offer users built-in reports about site traffic.
  • Mobile application developers have the ability to create programs for prompt notification of the site.
  • Companies were able to integrate Metric with business applications and internal services. Managers and marketers will be happy personalized reports on the site.
  • Internet agencies can embed Metrics reports into their campaign management tools. Reports to clients can be provided in the interface of corporate applications.
Send us your examples of using the Yandex.Metrica API. We will add the most interesting solutions to the application gallery.
A detailed description of the API is available in the Developer Reference .
