Selectel Storage - Cloud Storage

What do we offer?


Selectel Storage is a cloud-based object data storage based on OpenStack Swift.

For what?


For convenient, easy and cheap storage of backup copies of files and distribution of static files (images, audio, video, js, css) for websites and applications.

Prehistory


An analysis of virtual machine use scenarios in the cloud showed that the majority of virtual machine users are not very keen on setting up complex connections from the front-end and backend, as a result of which static content returns fall on the shoulders of servers giving away dynamics. Thus, from the server side, content distribution is ‚Äúestimated‚Äù not only by outgoing traffic and storage costs, but also by write / read operations, processor time and overhead costs for RAM.

Also, many clients keep separate virtual machines for backups, which turn on for the duration of the backup and then shut down.

Since both these cases imply no targeted use of resources, we offer you Selectel Storage.

Features


Cost of

When accounting for the cost of using Selectel Storage, we applied the same calculation scheme as in Selectel Cloud, which gives users the opportunity to pay only for the resources actually used. We also refused to record incoming traffic and the user pays only for storing and downloading data from the storage.

Storage is accounted for by the total amount of stored data on the account without taking into account the replicated copies. Network consumption is calculated based on total outgoing traffic.

We managed to achieve the lowest possible prices for data storage and transmission thanks to the use of our data centers with a developed network infrastructure.

Unlimited size

Selectel Storage allows you to store any necessary amount of data, offering the optimal solution for storing backup copies of any data, personal files and various content for sites.

Own domain

We also give the opportunity to attach your domain to the container in the storage, which can be used to distribute static data, for example: static.domain.ru/img.png


Reliability

All downloaded data is immediately replicated to three independent servers. Even if two of the three servers fail, your data will not be affected and will be available, and the internal control systems of the warehouse will automatically distribute the missing copies of the data to other working servers.

Also, the user API provides the ability to control the integrity of the downloaded data by checksum.

Fast access

Access to the storage is provided by several distributing Proxy-servers, each of which has a connection to the external network at a speed of 10 Gbps.

WEB interface

Web-based interface allows you to quickly upload and download files, set permissions, bind domains and get statistics.





OpenStack Swift API

Selectel Storage provides compatibility with the generally accepted standard developed by RackSpace and NASA, with examples of implementation in many programming languages ‚Äã‚Äã(php, python, java, .Net, ruby).

Additional features in the API

  • number of downloads for each object
  • traffic statistics for the account as a whole and separately for containers



The convenience of use

  • The ability to access via FTP-protocol any available software
  • Access from iOS and Android devices (using third-party software)
  • A wide range of software for synchronizing local data with storage (the list is available in the control panel )
  • Ability to connect as a partition through FUSE
  • Ability to automate scheduled backup github.com/selectel/supload
