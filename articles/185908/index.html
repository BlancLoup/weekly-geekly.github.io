PHPShop has released a free chat for an online store.

PHPShop made Chat - a new free Win-chat application for an online store and website. Chat is a module for all PHPShop products, available in the "Modules" section for the external part of the site, as well as a separate Windows chat.exe application for the chat administrator.

Why, you ask, your chat, when the market has a large selection of similar applications?


Indeed, today there are already SaaS services for connecting chat rooms and online helpers for sites: Livetex, WebConsult, as well as large software systems that require separate resources of your server and installation skills: LiveZilla, Rustalk. Almost all of these offers are commercially based and offer their services for a fee. On the other hand, the entire history of your correspondence is stored on a foreign SaaS server of the project.

In order to help its users communicate for free with their customers , PHPShop has created its own free chat which stores all correspondence in the databases of the stores of the users themselves. The chat application is very light: the .exe component weighs about 900 kb, and the server part is 700 kb, and is installed ‚Äúwith one click‚Äù.

Design


Included are 9 templates of different colors, but you can create your own design through the template engine by editing the phpshop templates / modules / chat / templates /. CSS styles of colors are in phpshop / modules / chat / templates / skin /. To create a new style, just create a new * .css file in this folder.

Pleasant additions


Chat supports an unlimited number of users on the site, as well as the insertion of hyperlinks and emoticons, stores the entire history of your correspondence in a database, with the ability to search for stories. When a new message arrives, the administrator beeps and a pop-up notification appears. For slow inexpensive hosting, in order not to overload the server, you can specify the appropriate time interval for polling the site by chat.

How will PHPShop Chat develop?


PHPShop Chat is updated through the update of the PHPShop EasyControl utility collection. In the near future, PHPShop plans to implement file transfer via the chat interface, make invitations to users, and also tightly integrate the chat with its Stat statistics modules, Visual Visual Recycle Bin module. It is also planned to support several administrators with the redirection of chat users between them.



Technical details


The server part is standardized for the format of PHPShop modules and implemented on the PHPShop Framework in the development environment PHPShop IDE 1.1 . The interior is made in Delphi 7.0.



Conclusion


Those of our customers who have updated and actively use the chat functionality have already managed to evaluate its capabilities. PHPShop is actively developing, and not only in web technologies - now, to help its users, more than 20 free utilities for Windows OS and more than 60 free add-on modules for websites and online stores are being offered. We invite you to test PHPShop Chat and other useful options for the store.
