The SPDY protocol will speed up the Network in half

The developers at Google have just announced that they are working on a new network protocol SPDY (read as SPeeDY, that is, ‚Äúfast‚Äù), which should upgrade the HTTP protocol and significantly increase the speed of all types of connections.

SPDY allows you to halve the latency when working via HTTP. This is done through three methods:

1) query multiplexing;
2) prioritization of requests;
3) HTTP header compression.

To demonstrate all the features of SPDY, Google engineers raised a test web server and released a special version of the Chrome browser.
