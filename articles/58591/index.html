HFS is a small and convenient HTTP file server.

I ran through the search for Habra and was surprised that no one mentioned this program, although the same Dropbox was repeatedly written.

So, meet - HFS . A program that allows you to raise your own HTTP-file server in two clicks, share files and give a link to the download to a friend.

What good is this approach? The fact that you do not need to upload anything anywhere - the files are downloaded directly from you. However, it is true that it‚Äôs bad - it will work only if the computer is turned on and has access to the network (preferably not via GPRS :)

It is possible to configure access rights both by IP and through accounts. Plus - the ability to upload files (similarly - files are uploaded directly to your computer).

The web interface is quite pleasant to the eye. The program works under both Windows and Linux through Wine.

Of the minuses - it seems like external IP is required (static or dynamic), but I did not check the possibility of working without it. And so - the thing is wildly convenient, especially when the shared folder contains files that are constantly changing. Although, Dropbox fans will not appreciate it.
