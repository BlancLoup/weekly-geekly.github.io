Canadian farmers created the world's largest QR code



It seems that not only mysterious circles may appear on the farmers' fields (which in fact turn out to be not so mysterious). A family of Canadian farmers created the largest QR code in the world in their field, which was confirmed by representatives from the Guinness Book of Records. The field is located in the Canadian province of Alberta, so if you are anywhere nearby, you can visit this place.

Unfortunately, the code is not readable in the photo (I have already tried it, and with different applications). Canadians claim that everything works if you photograph their code from a bird's eye view. Actually, there is no special sense to photograph and recognize the code, since all that is encoded in the code is the hyperlink leading to the farm's website .

In this case, the code is framed by labyrinths, which, according to the plan, should attract quite a lot of tourists. In any case, the family of farmers achieved what they wanted - fame. After all, if just the patterns on the margins have long been attracted by very few people, then the QR code, and even the largest in the world, immediately aroused the genuine interest of many authors and IT publications.
