Plotr and MooTools

So far, there are few quality add-ons for the MooTools JavaScript framework, and some do not want to work with the new version (1.2) of the latter at all. One such addition is the adaptation ( link 1 , link 2 ) of the Plotr library, which allows you to create linear / pie / bar charts.

True?


Yes, while the author of this library is already creating a new one - Flotr , which, I hope, he adapts and under our framework (the author is actively showing attention to him), you need to somehow draw graphics. Therefore, I bring my version to the public. By the way, to adapt, I also had to add several functions to the Array and Hash classes, you will find them in the archive in the file lib / plotr / MooTools.js.
