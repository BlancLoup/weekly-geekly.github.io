PickMeUp is a good jQuery datepicker plugin

Problem


Starting work on the next site needed datepicker. The most famous such datepicker is in jQuery UI, but since the jQuery UI was not used in the project, I didn‚Äôt want to pull even a part of it, I began to look for a decent alternative.

The requirements are as follows:
  • Select date, multiple dates, interval
  • Easy to customize the look
  • It is desirable without any dependencies except jQuery

The requirements are quite logical, nothing more than natural.
What was my surprise when looking at a dozen two plug-ins, I did not find the right one.

For the curious, immediately a demo of what happened as a result.

The closest to the requirements was the DatePicker .
But he had several drawbacks:
  • Redundant table layout
  • Old-style picture decoration
  • Just old, not developed since 2009
  • Some annoying bugs

It was possible to solve them one-time, but it was decided to bring to mind, because not the last project, and others can be useful.

Bicycle so bicycle


The first thing I wanted to change was to remove the pictures to round the edges. But the pictures in the cells of the table. I removed the cells - almost all the functionality fell off ... But there is no going back!

Then some things were refactored, bug fixes, small features added, bug fixes again, and so on.
It took about two full days, and it was worth it.

The code is not perfect, but much simpler and clearer, and therefore easier to draw it.

What else is new compared to the original:
  • Support for specifying configuration options in data attributes
  • Global plugin configuration
  • Independence of the language of each individual calendar
  • Styles in a small scss file with configuration variables ‚Äî if you only need to change colors ‚Äî perfect
  • The layout is rubber, the size depends on the font size of the root element, it looks equally good in any font size
  • Plugin size smaller than original
  • All classes have the prefix pmu- , the root element has a pickmeup
  • Some little things
