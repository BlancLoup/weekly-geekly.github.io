6 free CDNs to speed up and improve the security of your site



Improve the two critical indicators of your site - performance and security, using the appropriate content delivery network.

It is unlikely that someone likes the low speed of loading web pages, which can cost millions due to lost sales. The facts speak for themselves:

  • Shopzilla's revenue increased 12% after the boot time was reduced from 6 seconds to 1.2 seconds;
  • Amazon estimates that delays in loading pages of just one second can lead to a loss of $ 1.6 billion in sales annually;
  • 21% of shoppers drop a shopping cart if the site is slow;
  • Google takes into account the time it takes to load a site when displaying search results , so speed does not only affect revenue. Fast download allows you to move higher in search results.

Security is equally important for the success of a site, and you should do everything possible to prevent your site from being hacked.

The lack of a proper site protection system makes it easier for hackers to do the job and damage your reputation and reputation.

Trustswave found vulnerabilities in 97% of applications tested by them, and the latest SUCURI hacking report showed that 74% of WordPress sites are infected.



Obviously, both speed and security are paramount for commercial sites. Such tasks can be handled in different ways, but one of the easiest ways is to use a CDN for speeding and a cloud service provider for protection.

Fortunately, most CDN operators offer both. Let's see what these options are and which ones are right for you.

1. Incapsula


Imperva Incapsula - one of the leaders in the field of cloud information security, allows you to increase the speed and security of any site.

Incapsula offers a free rate, perfect for a blog or personal site, with the following features:

  • global content delivery network;
  • caching, compression, size reduction, image optimization ;
  • protection from spam bots, captcha, protection from spam in the comments;
  • IPv6 support;
  • real-time statistics , uptime monitoring , weekly reports, traffic statistics.

For a commercial website or online store, you can choose the rate Business or PRO. Paid versions offer more features, including DDoS protection, firewall for web applications, load balancer, etc.

Incapsula points of presence are located in 30 data centers, which allows you to send content to your users from the nearest server.

If you are using Joomla, then you may be interested in the extension to control the speed of the site, the delivery of content through the Joomla admin panel.

Try a free or trial version and speed up your site.



2. CloudFlare


CloudFlare is one of the most advanced solutions for speed and security.

CloudFlare provides over 4 million online resources. Points of presence are located in 101 data centers .



Some of the following features offered by Cloudflare are available for free:

  • global balanced CDN;
  • content caching, instant cache flush;
  • asynchronous download Javascript (Rocket Loader);
  • IPv6;
  • HTTP / 2 and SPDY;
  • WebSocket;
  • Restful API
  • basic DDoS protection;
  • TLS 1.2, 1.3;
  • certificate issued by Cloudflare;
  • redirect to https;
  • comment spam protection; content theft protection;
  • DNSSEC ;
  • analysis of work and security threats;
  • Page rules that allow you to set parameters for specific pages.

Setting up the site takes less than five minutes. This is a very profitable option to optimize speed and improve security.

In case security has a high priority, it makes sense to think about PRO tariff or even more advanced version, in which the application firewall will be available, image optimization, optimization for mobile devices and much more.

It is worth noting the work with applications Cloudflare. It allows you to easily implement certain functions without additional installation. For example, you can use Google Analytics, Pingdom Swiftype without making changes to the code.

3. Datapath


Datapath provides integration with AWS, which increases the speed of the site and reduces the cost of traffic. Datapath refers to IaaS (infrastructure as a service) and allows you to fully manage your network, route traffic based on latency, cost, and bandwidth.

In a nutshell, Datapath optimizes traffic and reduces AWS latency.



In the free version available:

  • one elastic IP;
  • waiting time optimization;
  • traffic monitoring;
  • 500 Gb of traffic per month.

Datapath suits you if you are using AWS, want to optimize network latency and lower overall costs.

4. Photon from Jetpack


Photon does not provide a full range of CDN services, but only allows you to speed up image processing. If you use WordPress and often resort to images, then Photon can be a valuable find for you.

You can use Photon to unload a feed to speed up image loading.

Photon is activated on the Jetpack toolbar and works with sites located on wordpress.com, as well as independently placed elsewhere.

Alternatively, you can use Cloudinary to manage images and video in the cloud.

Cloudinary works on any platform, so if your site does not use Wordpress, and you need to optimize images, then pay attention.

Another tool for image processing and fast delivery is Uploadcare .

5. Netlify


Netlify is a tool for hosting static sites and submitting them via CDN. Netlify is a non-standard solution for CDN, but if you are interested in hosting static sites, then you should try.

Over 35,000 developers use their services, and more than 90,000 sites are hosted on Netlify.



The free version includes the following:

  • forwarding rules;
  • continuous deployment;
  • Let's encrypt certificate, HTTP / 2;
  • personal domain;
  • global CDN, super fast DNS;
  • rollback to the previous version;
  • pre-rendering, clearing the cache;
  • DDoS protection;
  • using git;
  • custom HTTP headers .

6. JS Deliver


Apply JSDeliver to reduce the load on the libraries of JavaScript, jQuery, fonts, CSS, and not only to make site resources load faster.



JSDelivery supplies files from Cloudflare, Keycdn, Quantil and MaxCDN . You can upload multiple files with one HTTP request. There are no traffic restrictions.

I hope that these solutions will allow you to make loading your sites faster and safer . All options are free, so try and find the one that's right for you.
