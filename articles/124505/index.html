GitHub Reflog v1.7.19

Welcome to GitHub Reflog, a chronicle of great GitHub repositories and community activities. Previous issues are available in the Reflog Archive , translations of the issues in the Archive of the Russian-language version of Reflog .

This week on GitHub we warmly accept " The League of Moveable Type " - great open fonts!

Recommended repository of the week


ryanmcgrath / wii-js
Nintendo‚Äôs most popular Wii game console has a built-in web browser, but for some reason very little development is done for it. This JavaScript project offers a stable, reliable, and documented API for Wii / HTML interaction.

Although not entirely practical for developing real-world applications, this project claims to be an excellent tool for teaching children how to program.

Wii-JS supports up to four wiimotes interactions with a page at a time. Visit the demo page (using the Wii) to see it in action.

This week's amazing repository


danlucraft / git.js
This is really amazing. A Git implementation written in pure JavaScript. Looks like a Dulwich project for Python.

Git.js is distributed under the MIT license. It includes both a console git client for node.js, and an internal browser API for managing repositories via HTTP.

Remarkable repositories

  • j2labs / brubeck : A flexible web framework for Python, powered by Mongrel2 , Eventlet, ZeroMQ and DictShield . In order to evaluate it, look at the page with examples of the project.
  • arthuredelstein / clooj : A light IDE for Clojure, written in Clojure itself. Powered by Swing. Clooj is cross-platform and can be used as a standalone application or a built-in editor.
  • jceb / vim-orgmode : This amazing plugin provides the vim community with an emacs org-mode note system. You can also access your notes on iPhone from mobileorg via WebDAV or Dropbox.
  • Khan / khan-mobile : Source code for the new Khan Academy mobile app.

Perspective repositories

  • urbancoding / jenx : Jenkins build server monitor for Mac OS X running on MacRuby. This application reports the status of all your Jenkins builds. By clicking it opens the work in the browser. He even supports Growl!
  • jesusabdullah / pyee : Port in Python of the popular Event.Emitter node.js system. Simple and straightforward.
  • memo / iSteveJobs : This useless (but fun) project replaces all the faces found in the video stream from the webcam with the face of Steve Jobs. Works on the C ++ framework openFrameworks .
  • kennethkufluk / Twitter-Matrix : This HTML5 project takes a total stream of tweets and displays it in a Matrix style. Be sure to visit the demo page to see it in action.
