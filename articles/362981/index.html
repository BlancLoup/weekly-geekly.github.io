0h h1 - small logic game

A good way to spend a Friday night is to play a new 0h h1 puzzle.

The meaning is simple: you need to fill the entire game space with squares of red and blue. It is forbidden to place in a row three squares of the same color.



A red square is created with a single click, a blue one is created with a double click.

Another rule is that each row and column should have the same number of red and blue squares.



And the last rule - the creation of identical rows is not allowed.
