Data storage with 64 TB of flash memory

The company Texas Memory System has introduced a server rack with 64 TB of high-speed flash memory on board. The device (the names did not seem to give him any name) is capable of performing up to 800 thousand I / O operations per second. The total rack capacity is 12 GB per second. The power consumption of this monster is 2.5 kW.



The rack is made up of eight RamSan-500 units, each with a speed of about 100 thousand I / O operations per second and a throughput of 1.5 GB per second. These figures are achieved by the presence of from 16 to 64 GB of DDR-memory for each unit.

The device is not only smart, but also convenient. So, the array of flash drives supports hot swapping. At the same time, according to the Texas Memory System, the integrity of data is preserved, and the restructuring of the array takes less than 15 minutes.

As a high-performance array of applications, the Texas Memory System sees work with rendering and editing video content, as well as using huge databases for processing.
