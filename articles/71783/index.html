I want. I have. I love

Is it possible to create a simple tool that allows people to find each other using an arbitrary key? Finding the answer to this question led to the appearance of the I want service . I have. I love
The principle of the service is simple. The user is asked to leave a message by selecting one of the suggested prefixes: I want , I have , I love ... Optionally, you can specify the purpose of the message by selecting one or more add-ons. The message is added to the database, and the system, at the same time, selects messages similar to what the user left. Your posts are not displayed in the search results.
The service has no thematic limitations and currently supports Russian and English.
According to the search results, the user can send a message to the author of the message he is interested in, or follow the link if it is in this message. In addition, the user can do nothing if the messages found are not interested in him).
We believe iwantihaveilove.com can be useful. Thank you for your time.
