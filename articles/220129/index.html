Jump Start April 22: Data Center Virtualization using Windows Server 2012 R2 and System Center 2012 R2. Ensuring the security and high availability of virtual machines

We bring to your attention a series of online events dedicated to data center virtualization using Windows Server 2012 R2 and System Center 2012 R2 . As part of these online events, the key Windows Server and System Center technologies used to deploy, manage, and maintain the physical and virtual data center infrastructure are discussed in detail. If you are using Hyper-V in Windows Server 2008 / R2, you will see a huge amount of changes that have occurred in the Microsoft hypervisor with the release of Windows Server 2012 / R2. If your organization uses VMware, you can evaluate the capabilities of the latest version of Hyper-V, as well as compare the technologies of Windows Server and System Center with VMware vCenter and vCloud Suite solutions.



The first event of the series took place on April 8: Data Center Virtualization using Windows Server 2012 R2 and System Center 2012 R2. Configure virtualization hosts . Tomorrow, April 22, at 12:00, see the online broadcast of the following Jump Start on the topic: Data Center Virtualization using Windows Server 2012 R2 and System Center 2012 R2. Ensure security and high availability of virtual machines .

In the second online event of the series, we will discuss clustering of virtual machines, including physical and guest clustering, dynamic optimization and optimization of power consumption. Let's talk about advanced settings of virtual machines, such as Virtual RSS, QoS for disk operations, online cloning, second-generation virtual machines, etc. You will learn about the new features of Live Migration. Finally, special attention will be paid to the backup and replication of virtual machines.
