Intel introduced a Galileo development board

Intel introduced the Galileo development board, its first product fully compatible with the Arduino platform. Recall that the Arduino is a hardware computing platform, the main components of which are a simple I / O board and the Processing / Wiring development environment. Arduino is used by device developers and educational institutions for educational purposes. Intel Galileo is based on a tiny Quark X1000 processor. This 32-bit chip has one core with the ability to execute one instruction stream; The clock frequency is up to 400 MHz. Galileo supports interfaces such as ACPI, PCI Express, 10/100 Ethernet, SD, USB 2.0, UART and RS-232. The board uses a Linux kernel operating system with Arduino libraries. You can program from computers based on Mac OS X, Windows and Linux. Galileo supplies will begin by the end of November. Intel has pledged that over the next one and a half years, 50,000 copies of the fee will be transferred to 1000 universities around the world for free.
