New snapshot 10.70: build number synchronization

A few days passed since the release of the previous snapshot 10.70, so we decided it was time to release another one.

If you look at the build numbers, you will notice that the Mac version and the Unix version have one number. We are currently working on synchronizing build numbers on all platforms, and this is the first step. The Windows version, we hope, will follow in the near future.

Note: this time there are no builds for Linux PowerPC.
