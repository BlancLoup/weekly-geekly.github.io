2GIS launches API cards

Following the 2GIS API API, we are pleased to present a beta version of the API maps. Now, any site can use for free not only accurate and current data on 1.2 million organizations in Russia, but equally high-quality maps.

The 2GIS maps cover 140 cities and over 4,000 settlements with details up to fences, kiosks, and intra quarter passages. More than 700 employees work on the relevance and completeness of the data in 2GIS, maps are updated every month (and the data are already every day ).

Functionally, the API offers developers the necessary set of methods for working with the map. We do not plan to introduce any fee for using API cards, and to get started, no key is required.

The API will be useful for both simple corporate sites for displaying offices on a map, as ITConstruct or Irkutsk Design Studio 38 did, and for more complex implementations.
