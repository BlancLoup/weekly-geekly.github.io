Firefox 3.5pre + FireBug 1.5X.0a03 = full support

The version of Firefox 3.5b4 was a nasty bug that did not allow tracking XMLHttpRequests via the Firebug console.
With the release of Firefox 3.5pre, a number of errors have been fixed, in particular, this error, which does not allow tracking asynchronous requests to the server. It is also worth noting that the new version of Firebug supports all FBTests unit tests, which were correctly executed only on Firefox 3.0.10 before.
Download Firefox 3.5pre (EN, win32) + install FireBug 1.5X.0a03 .
