"Beautiful" in Zend Studio 5.5

By default, Zend Studio 5.5 has only two color code markup color schemes. And they are pretty scary. At one time, I was quite strongly interested in how I could make my hours-long sitting in IDE more comfortable, so I began to dig hard in search of more decent color schemes.

As a result, I found about five or six color schemes, with which it became much easier to work with.



Below I offer for downloading the schemes that I found for Zend Studio, under the cut a link to the archive and a brief description of how to install them.

1. Take the archive zend-schemes.zip here . I have included the following schemes into the archive: Katulus, Komodo, RubyBlue, Navajo-Light, Evening, Experienced, and Ambients.

2. Fill these color schemes (for W7) into \ Users \ username \ ZDE \ config_5.5 \ schemes with Zend cut down.

3. Be sure to set the rights to this folder only for reading . Otherwise, for some reason, this folder is cleared after exiting Zend.

4. Run Zend, go to Tools ‚Üí Preferences ‚Üí Colors and Fonts, select one of the color schemes and enjoy.

I would be glad if it helps someone.

PS (March 22, 2010): Transferred the archive with the schemes from rapid to another hosting, now everything should work.
