New Nimbus Screen Capture with the possibility of sharing

Not so much time has passed since the last first review of our product Nimbus Screen Capture, and today we are ready to present a new version of the addon, which added a number of important changes.



Sending to Nimbus

The update is extremely important for us due to the fact that this is the first application that fully works with the new cloud service Nimbus, which can be found at nimbus.everhelper.me (be careful, the beta version!)

Quite often there is a situation when you need to quickly make a description of the screen and get a quick link to it. It is for this purpose that we added the item Send to Nimbus in the window for saving screenshots. Just need to create a new account in Nimbus (it takes less than 30 seconds), specify the folder to send with the necessary comment and voila!

The resulting link can be immediately reduced to a more or less acceptable size.
