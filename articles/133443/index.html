NodeJS became the most popular repository on Github

The NodeJS framework has come out on top in the list of the most popular Github projects by number of subscribers (watchers).
  1. node - 11 349 followers
  2. rails - 11,234
  3. bootstrap - 10,924
  4. jquery - 10 479
  5. html5-boilerplate - 8,940
  6. homebrew (package manager for OS X) - 6,641
  7. diaspora - 5,898
  8. three20 (Objective-C library for iPhone design) - 5 380
  9. devise (authentication system for Rails) - 5,249
  10. Spoon-Knife (an empty repository for training new github users, how to fork, etc.) - 4,863
In general, the rating is rather subjective. For example, in third place is Bootstrap, a set of styles and UI elements from Twitter developers, which was born just a few months ago. In the first weeks after the release, he even visited the leaders of this list.

How the numbers with Github relate to reality can be judged by the number of followers to tags on StackOverflow: NodeJS has 2200, jQuery has 19,500, C # has 22,200, PHP has 18,000 and even ASP.NET 8300 followers.
