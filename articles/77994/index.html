The bioreporter program is a Windows application that works without server media mediators.


bioreporter connects to the site via an FTP protocol that supports almost any virtual hosting tariff.
The interface and functionality is extremely simple, but this does not prevent the program from performing its functions.

The program allows you to make notes in the .wav format and transport their compressed .mp3 copies to the site.
The program is designed for mobility. It can work while on a USB drive. This allows you to record reports on any computer with a sound card, without worrying about pre-installed recording programs and audio broadcasts.
The main feature of the program is the ability to conduct online audio broadcasts live, but with a delay of 1-2 minutes for the end listener.

To play the audio stream, a special flash player is provided, which is located on the site. The player automatically detects the audio stream and connects to it, and also interrupts the broadcast after it ends.



The program is demanding to the communication channel. By default, files are encoded to mp3 with a bit rate of 128 Kbps. And in order for the broadcast to take place without artifacts, the required outgoing connection speed should be at least 256 Kbps.

In the configuration file, you can specify the compression parameters for the mp3 encoder, which will remove the load from the broadcast channel, as well as from the site, which can transmit sound to more users in time.
