Export from * .AI to * .CDR


The other day I had to solve a very interesting and relevant task of converting a project from Adobe Illustrator to CorelDraw . Under the cut a little guide about how I did it. I hope these small tricks will help someone from the habr-community.

Save to EPS

Given the simplicity of the picture (no gradients, transparency), everything can be solved very simply. When saving, select the format * .EPS in EPS Options, select the version of the file of the ancient.
Open the saved EPS file by Corel.

Aiclipboard

Converting complex illustrations is a bit more complicated and requires very small dances with a tambourine. Download the script to convert images from Adobe Illustrator to Corel Draw.
Where to download:
  • Official site
  • The version of the script for CorelDRAW 10/11/12/X3 / X4
  • Corel DESIGNER 10/12 Version

The procedure for transferring from * .ai to * .cdr is as follows:
  1. Unpack the script with any archiver in the following way: ... \ \% folder_korela% \ Draw \ GMS \
  2. We start Corel and go to Tools / Visual Basic / Play
  3. In Macros in, select AiclipBRD.gms (do not close the menu and Korel)
  4. Open the * .ai file with Illustrator, select all and press Ctrl + C.
  5. Go to Corel and in the open menu, select the item CopyPaste.PasteAIFormat, click Run.
  6. The image will be inserted. Save to * .cdr

Macro menu in CorelDraw X4
When transferring images, I did not notice the jambs with the transfer of gradients and layers with transparency.
