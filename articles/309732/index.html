In this case, the weak point is the key, which is the same for each computer where files are encrypted, but the single key is used only if, for some reason, the C & C server is unavailable to the cryptographer.

But now the authors have changed the way they infect computers. Typically, such attacks use a small Trojan downloader that downloads and launches the cryptographer.
