Firefox 20 adds support for H.264 in Windows 7



The new nightly build of Firefox 20 in the Windows 7 environment has added support for the H.264 video compression standard. About wanting to present in his browser support for this codec, in order to avoid the need to use external plug-ins like Adobe Flash, in March of this year, Mozilla technical director Brandy Ike wrote in his blog, arguing the need to support H.264 to enter the mobile platform market. The Firefox version for Android received support for the standard, and finally, the intention to introduce it in the desktop version became noticeable.
