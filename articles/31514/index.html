Fring Launches Chat / VoIP App for iPhone


As far as unofficial sources know, Fring, an Israeli young company that has gained fame for its mobile phone VoIP / Chat service, will release an iPhone version of its program tomorrow.

This is not a browser chat program like FlickIM or Mundu, but a full-featured downloadable application that provides users with access to services such as Fring, Skype, MSN, Google Talk, ICQ, SIP, Twitter, Yahoo and AIM.

This means that the installation will require a Jailbreak installer, which will adversely affect the number of users of the program.

In spite of this, Fring can be the very application that will force advanced iPhone users to use Jailbreak. However, the ability to make calls using the VoIP service directly from the iPhone itself is worth it (even taking into account the fact that the program works only on Wi-Fi networks, EDGE is not supported).

For the most part, this is a test version, the official release of the iPhone software market is planned for the summer. Pretty risky strategy. Some developers with whom I have communicated, try not to work with Jailbreak, so as not to anger Apple and that their official products will not receive banned status in the future.
Fring is a pretty successful project without an iPhone version. The number of users of the service, which is only a year old, is increasing monthly by 100,000 people. We will let you know when it finally becomes known about the release of the Fring application.

upd: Now it is possible. (Warning: in order for the application to work, you need to install Jailbreak on your iPhone). You can find more information on the Fring blog ( Fring blog ) or download the application on the Fringcubator. As you know, the tariff plans of most mobile operators provide bonus packages of minutes, therefore, subscribers will not use Fring for local calls. But Fring is very convenient for international calls and just for communication via IM (instant message English - instant message - a service that allows you to communicate via the Internet in real time).
