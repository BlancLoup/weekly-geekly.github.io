France tests dynamic CVC code on bank cards

BNP Paribas, a financial conglomerate, a leader in the global banking and financial services market, is testing new MOTION CODE technology from Obertur Technologies. The innovation is that the CVC code (three digits on the back of the card, which are used to confirm the identity of the cardholder when making an online payment) replaces the dynamic security code.

Dynamic security code created to effectively resist the card fraud in cases where hackers steal payment information directly in the process of online payment. Dynamic code changes regularly, which prevents fraudsters from reusing card data.

For testing, BNP Paribas chose OT MOTION CODE solution developed by Oberthur Technologies (one of the market leaders in digital security solutions on the Internet). On the back of the card is an electronic display showing the security code, which changes periodically. The dynamic security code enhances the protection of online payments by BNP Paribas customers, while not changing the usual shopping process.

A representative of Obertur Technologies said that introducing MOTION CODE technology, a fully integrated solution is offered, starting with the production of the card itself and ending with the installation of authentication servers.

The dynamic code not only significantly increases the security level of billing data when making online payments, but also does not require infrastructure changes, since the presented technology is compatible with the payment procedure on any commercial site. By the way, more than 90% of online payments in France are made using bank cards.

Russian banks can follow the example of their colleagues and launch a project with a dynamic CVC. A similar product is supplied to the Russian market by NovaCard , a plastic card manufacturer.
