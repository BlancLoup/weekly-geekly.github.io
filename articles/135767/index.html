Nginx overtook Microsoft IIS and came in second place on active sites

In the latest Netcraft statistics for January 2012 , a remarkable event occurred: the nginx web server finally overtook Microsoft IIS and came in second place on the Internet among active sites (that is, sites with unique content that were not made using templates).



Developer December 2011 Share January 2012 Share Change
Apache 102 005 032 58.21% 105 684 049 57.93% -0,28
nginx 20 342 324 11.61% 22,221,514 12.18% 0.57
Microsoft 21,572,870 12.31% 22,114,114 12.14% -0.17
Google 14,240,979 8.13% 14,412,926 7.90% -0.23
It is noteworthy that in this statistics all the main competitors have lost their positions, and only nginx added compared to December 2011.

Nginx is still inferior to Microsoft IIS according to general statistics, that is, without filtering by active domains. But here, too, the Russian web server was the only one among the leaders who increased its share in January 2012.
