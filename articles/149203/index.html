Start developing on the Microsoft platform today!



Developing on the Microsoft platform has become even easier with the new Application Development Center .

Microsoft platform. Many have heard this combination of words, but does everyone understand exactly what is behind these two words? To help newcomers to deal with this, a resource has been launched on the Russian MSDN website, with the help of which you can get acquainted with the main components of the platform, its development directions, find all the necessary tools and documentation on how to develop and publish your first application step by step .

The site itself contains the latest information about the latest technologies provided by Microsoft for application development.
