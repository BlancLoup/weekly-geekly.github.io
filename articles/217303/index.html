Dell PowerEdge R820 12G server overview

The Dell PowerEdge R820 is a 2U-high-speed universal rack server designed for applications that require significant compute and storage resources, as well as virtualization in medium-sized organizations and large corporations. The R820 server has 48 DIMM memory modules and up to 1.5TB of DDR3 RAM along with seven PCIe 3.0 slots, support for additional GPU accelerators, and a number of Dell Select Network Adapter network adapters for connecting to the main matrix switches without using a PCIe slot. You can increase I / O speed with four optional Express Flash PCIe SSDs that are operatively connected to the front panel for high-speed distribution of incoming data for storage.



For the first time, Express Flash drives were introduced to the 12th generation PowerEdge R720 server model, which became the mainstay of our corporate testing laboratory. The considered R820 server is not equipped with Express Flash drives, and this is the main difference for those who are looking for the best combination of high performance and operational reliability. Like its brother, the R820 is equipped with Intel Xeon E5 processors. The R820 server supports four Xeon E5-4600 family CPUs, as opposed to the two supported in the R720. Additional storage options include support for dual RAID controllers and up to 16 internal hard drives.

The storage flexibility of the R820, as well as the direct computing power of four Intel CPUs with a 2U form factor, draw attention to the R820 server in many cases. The bottom line is that the R820 has more performance with a smaller body area than conventional servers with a height of 2U. For a business that requires significant computing power, multi-user access or dozens of computers, the advantages of this server are obvious. Equipped with a quad-core processor, the Dell R820 servers will help to significantly save space, reduce power consumption and reduce the costs associated with server maintenance, which increase with increasing number of machines.

The PowerEdge R820 server is equipped with four Intel Xeon E5-4650 processors at 2.7 GHz, each with 64 GB ECC DDR3, with a total server RAM of 256 GB. In addition, the server is equipped with an iDRAC7 Enterprise management card for out-of-band remote management capabilities, allowing administrators to use, update, monitor and maintain the server without the help of agents. The server is equipped with a Perc H710 RAID card for high-level data storage. Moreover, the analogue of the servers HP - ProLiant DL560 Gen8

The Dell R820 is fully configurable on the Dell website at an initial price of $ 6,900.

Hardware

The R820 server holds up to eight processors and is currently available with a number of Intel Xeon E5-4600 processor families, including up to eight cores per processor, depending on the model. The server we are considering uses an Intel Xeon E5-4650 processor. The R820 can also use up to two 75W NVIDIA Quadro 2000 GPU accelerators with active cooling, which in turn will provide up to 1 GB of GDDR5 memory. Dell PowerEDGE R820 Server provides support for fewer accelerators than R720. Support for up to 7 PCIe network cards provides additional upgrade and configuration options.



Dell offers an R820 server with four internal memory configurations: 8 2.5-inch drives, 8 2.5-inch drives + four 2.5-inch Express Flash PCIe SSD drives, 8 + 8 2.5-inch drives, or 16 drives of 2.5 inches. The PowerEdge R820 is compatible with Express Flash drives with a capacity of 175 GB or 350 GB, which can be used as a primary storage device or for multi-level storage of CashCade data in combination with PERC controllers. In addition to 1.5 TB of UDIMM memory modules and RDIMM, the R820 server supports memory modules with reduced load for LRDIMM servers. LRDIMM memory modules use a buffer to reduce load and increase density by increasing latency. In this configuration, the server contains four 46GB 15K SAS hard drives connected to a Perc H710 RAID card.

The capabilities of the Dell Select Network Adapter include the selection of a number of integrated network adapter cards with its own daughter network adapter. R820-compatible child network cards include 1GbE and 10GbE ports, including Base-T, 10Gb Base-T and 10Gb SFP +.



The R820 server supports hot-swappable cooling fans in an N + 1 configuration, which allows the technician to replace any fan at a time. Like all 12th generation PowerEdge servers, the R820 is equipped with a Fresh Air cooling system that allows the server to operate at temperatures above 35 ¬∞ C / 95 ¬∞ F without increasing power consumption and the associated cooling costs. The 12th generation PowerEdge platform allows the user to configure server cooling using additional hardware, such as PCIe network cards, which can lead to overheating of the automatic cooling system if the sensors are not configured to detect overheating points.

The PowerEdge R820 server supports up to two redundant power supplies with automatic detection and switching, with a power of 750 W of alternating current, 1100 W of alternating current or 1100 W of direct current depending on the requirements of the user settings. The 12th generation Dell PowerEdge servers have the ability to limit the power consumption in the data center by reducing the speed of processors to a certain level. This is especially useful when using new servers in the face of restrictions on power consumption or temperature. The server in question is equipped with two 1100 W AC power sources.

Design and manufacture

The front panel of the PowerEdge R820 server is a 2.5-inch form factor with 16 connectors and an additional lockable connector. On the left is a power button and an indicator, non-maskable interrupt (NMI) for troubleshooting, connections for a VGA display, a simple LCD control panel, a vFlash SD media reader for use with the iDRAC along with two USB 2.0 ports.



The back panel of the device provides access to two on-line power supplies and corresponding indicators, network connectivity through the Select Network Adapter, and also has two USB 2.0 ports, an iDRAC7 port, a video connector and a serial connector. Quick Resource Locator is a QD-code that redirects to the material on the management and support of the server model, specified in several places of the body.



Although outwardly difficult to distinguish between the R720 and R820 servers, as soon as you open the case cover, you will immediately notice the difference. With the support of two additional processors, the R820 server is equipped with a daughter network card located above the motherboard and connected via a quick-disconnect connection. Opening the locking element, you can get the entire unit to access the processors and RAM sockets on the motherboard. The PowerEdge R820 configuration allows up to 48 DDR3 DIMM memory modules (twice the size of the R720). The model in question has 16 GB of DIMM memory, providing a total of 256 GB of RAM.

Both sliding and fixed guides are available for mounting without the use of special tools, in racks with four supports with square or round holes without threads, or in racks with four supports with threaded holes with tools. Fixed guides for rack mounting with two supports (Telco). An additional cable-mounting bracket can be mounted in the left or right part of the sliding guides without the use of special tools.
To install the R820 server on the rails without using special tools, you need to hold the server by the handles on the front and rear panels, carefully lowering the server on the rails with aligned mounting pins and the corresponding connectors. Experience has shown that this method is more convenient and faster than other guidance systems.

Control and management

The 12th generation PowerEdge servers , including the R820 and R720, are part of the OpenManage platform for managing with and without agents. At the center of the OpenManage platform configuration is the iDRAC7 with a life cycle controller. For integration with agent-based solutions, OpenManage Server Administraton (OMSA) offers direct management systems with a command-line interface or a browser-based Java GUI.



The iDRAC7 Enterprise is available in the PowerEdge R820 server along with the iDRAC7 Express, which has many features available in the enterprise version. Dell OpenManage Power Center provides power management at the level of the rack, row and data center. The R820 server has the ability to use uninterrupted virtual system idle time hypervisor using duplicate SD cards.

Conclusion

Like the PowerEdge R720 server , when using the Express Flash PCIe drive that is operably connected to the front panel of the server, the R820 server shows advantages in storing data over other servers. In addition to high-speed backplane storage capabilities, the strengths of the R820 server are the connectivity of up to 48 DIMM memory modules providing up to 1.5 TB of DDR3 memory. The ability to store large amounts of data in memory provides an increase in computational capabilities that complement the increase in the speed of access to the stored data from CashCade.
The 12th generation PowerEdge server case design proved that the server is a flexible physical platform with the ability to be mounted and the smooth operation of the system in which the elements are freely available for ease of maintenance. An impressive range of available configurations allows the server to best meet customer requirements, and seven PCIe connectors provide many options for upgrades. From the point of view of use and management, the iDRAC7 allows you to put the R820 server into operation in the easiest way.

While in most cases the computational power and density that the R820 server possesses is not required, there are times when they are necessary. From a storage point of view, CPUs are very useful when using many PCIe storage devices that will benefit from additional cores. FPGA and other external memory controllers that can use computing power to increase performance will show their capabilities in the R820.
