The long-awaited release of SeaMonkey 2.0

Despite the fact that several years ago the Mozilla Foundation abandoned the development of the Mozilla Suite in favor of Firefox and Thunderbird, the browser did not die and continues to be developed by the enthusiasts who are coordinated by Seamonkey Council. Today, more than 2 years after the official announcement about the development of SeaMonkey 2.0, the final version of this browser has been released. Compared with previous versions of the program, the changes affected all of its components: a web browser, an email client, an HTML editor and an application for developers, an IRC client. These changes take SeaMonkey to a new level and provide users with an opportunity for even more convenient work on the Internet.
Here are the main innovations made in this version:

  • SeaMonkey is built on the same platform that supports the same level of web technology as Firefox 3.5.4;
  • The default location for the profile has changed. Data from an obsolete profile, such as from SeaMonkey or even from Thunderbird, can be automatically imported (with the exception of extensions, themes and plug-ins);
  • The new add-ons manager allows you to install, update, disable and delete them. Everything is similar to what you can see in Firefox. It is possible to disable even individual plugins. All this will greatly simplify the work with additions;
  • Added restore sessions. Now, in case of a crash of SeaMonkey, all windows and browser tabs (and even the entered data in the website fields) will be restored after it is restarted or upon request. In addition, users can choose when restarting SeaMonkey whether to restore the windows and tabs they worked with during the last session;
  • Restore closed tabs;
  • Improvements in work with the history of visits, allowing you to quickly find the necessary sites by forming a query in the address bar;
  • The toolbars in the browser, news client and message composing window are now fully customizable using the context menu, including the size and display of icons, as well as the text next to them;
  • The news component now supports RSS and Atom feeds. The browser detects feeds located on web pages, previews them and provides various ways to subscribe to them. To subscribe to the news component, an account for news must already be created;
  • New accounts using IMAP will by default keep a local copy of the messages, which is part of the improvements with this protocol. Synchronization can be disabled at the account or individual folder level and limited to the most recent messages using the Sync and Store settings;
  • Support for tabs in the mail client (now you can open folders in them, individual messages and even a calendar if you have the Lightning extension installed);
  • Improvements in the Password Manager: a more convenient search, the notification panel is replaced by a dialog for remembering the password;
  • The download manager is completely redesigned. Now it is supported to resume downloading for an unlimited number of sessions working with the browser;
  • The new manager, responsible for managing the data entered in the forms of web pages, replaced what was previously. From now on, data entered into forms is automatically saved, and in the future it becomes possible to use them by providing this data in a drop-down list;
  • Improvements in the Cookie Manager, allowing you to search for them by name and site;
  • Updated theme;
  • Completely redesigned default theme for Mac OS X;
  • Simplified interface support for developers of extensions (SMILE);
