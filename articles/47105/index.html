Motion from Movable Type - a social network for everyone

Today was the announcement of the addition of Motion to Movable Type. This free add-on will allow you to socialize MT, create a full-fledged social network. This has already been written in Mashable and TechCrunch. I think, Habrakhabr users will also be interested.

When creating this add-on, the developers of Six Apart focused on the most popular features of all well-known services: Twitter, Tumblr, Pownce and FriendFeed. And because of this, a new, open-standards-compliant application was born.

Basic Motion Features

  • Microblogging - like Pownce, Tumblr or Twitter, only in a more advanced version. This is all the power of the Movable Type publication, embodied in a simple and convenient interface for "lightweight" blogging.
  • Action Streams are activity streams like FriendFeed. A feature already known to users of the Movable Type, but in a more advanced version - currently 77 services are included (a plugin with Russian-language services is on the way). This feature allows you to collect information from third-party servers about your activity on them, and then publish it in a blog.
  • Authorize commentators through OpenID, including Google and Facebook Connect profiles.
  • Full profiles - any user has the opportunity not only to show the name and photo, but also using Action Streams to show their actions on the Internet.

In other words, you and users of your site will be able to add posts, videos, audio, pictures, links, discuss it, add favorites, track interesting flows of users from other sites and add users as friends.

Types of users in Motion


Administrator
It has full control over the Movable Type, and can also customize any element of the system.
Registered Users
These are the ones who registered with Movable Type. They can maintain their own microblogging, as well as add social networks to the profile so that activity in them is displayed on the website.
Authorized Users
Any users logged in using OpenID or using other services (Google, Yahoo, Facebook). Such users can contribute to the development of the site by voting for posts or commenting on them.



Motion Uses

  • Private site to collect information about user activity and discuss this information.
  • Private microblog for easy exchange of information.
  • Public microblogging.
  • Public social network.
