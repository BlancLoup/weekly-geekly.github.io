Tests for speed: and nevertheless Opera is faster



Testing browsers for performance has long been one of the most popular analyst occupations. Now there is a very real reason for another study - the emergence of a new Google Chrome browser. The developers of this "newcomer" on the market say the best performance when working with JavaScript, but judging by these tests, everything is not so rosy. In general, the table of records looks like this:

1. Cold / hot start test - Opera 9.5 winner.



2. Launch test with eight tabs - Safari winner (Opera-9.5 with a scanty margin in second place).



3. Test CSS - Safari winner, second place - Chrome, third - Opera-9.5.



4. JavaScript test - the author of the test is not listed as a winner, Safari and Opera-9.5 have absolutely identical results in the table. Chrome on the fourth, yielding Safari, Opera and Firefox.



5. Memory usage - Firefox winner. Opera-9.6b - right behind him.



A brief conclusion: Opera is still among the leaders, slightly inferior in some parameters, but at the same time occupying a leading position in the overall standings. That, in general, is not surprising.
