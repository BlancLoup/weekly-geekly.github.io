Astra.CMS Free - free version release

We, the young development team, would like to introduce to the habrovtsami a free version of the content management system Astra.CMS Free . Its main feature is the ability to design websites almost without programming as such, only by logic at the template level. At the same time, it is possible to assemble a site with an individual structure, due to the fact that the sets of necessary structural components (sections, blocks) are created in a couple of clicks and in any quantity.


Thus, this engine allows you to significantly save the cost of programming the project and may be of interest to web studios or webmasters, whose specialization is, to a large extent, not programming, but design and integration. Knowledge of the Smarty language and the general ideology of the engine components, as a rule, is enough to assemble the necessary site in a matter of hours.

Advantages:
  • Designing an individual site structure without interfering with the source codes of scripts.
  • Ready-made set of extensions to quickly create a site structure.
  • A single module can be configured for different tasks. For example, a ‚Äúmaterial catalog‚Äù can serve both a catalog of articles, a news feed, and a catalog of goods.
  • Smarty template engine allows you to implement any design without special requirements for the layout of the original layouts.
  • The use of blocks makes it possible to compose the site pages of any available information.
  • Export the finished site into one archive file, which allows you to quickly transfer the site from the local platform to the server and vice versa.
  • The ability to import a site from a single archive file, which allows the use of standard procurement sites for even faster development.
  • Really convenient, in our opinion, administration interface.

Opportunities:
Astra.CMS Free is an open, open source product. Its capabilities are enough to create most corporate or information sites, not to mention the simple "business cards" that are created on the basic modules in a few minutes. In addition, all extensions are compatible with the full-featured version.
