Internet tablet on Android

Archos, which mainly produces media players, has announced the development of a multimedia internet tablet called the Archos IMT (Internet Media Tablet), which combines the functions of a smartphone and the proprietary multimedia interface Archos to access entertainment features and a web browser.


The Archos IMT will use a Texas Instruments OMAP 3 processor, a hard disk (or SSD, as yet unknown) of up to 500 GB. Also announced a 5-inch touchscreen display, support for 3.5G networks (including HSUPA) and HD video playback.

Well and the main thing, Archos IMT will work under control of Android OS. However, as well as those pieces of fifty devices on the same OS, the release of which is planned for 2009.
