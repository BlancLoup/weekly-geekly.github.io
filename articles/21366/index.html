Microsoft Research Collective Search Technologies

At the annual TechFest technology conference, researchers from Microsoft Research presented several new search technology projects. These are quite interesting developments, unique in their own way.

The first one is called SearchTogether and is implemented as a plug-in to the Internet Explorer browser. This plugin serves as a single search interface for a group of users who want to track each other‚Äôs search history. Grouping and authorization is carried out through the Windows Live ID passport system and the Live Messenger program. Supported search engines Windows Live Search, Yahoo and Google.

Another development CoSearch designed for collective search from a single computer. For example, from a mobile phone that is in communication with a PC, you can control the cursor, click on the link on the search page, download the search results to the mobile phone, while the PC operator at the same time can independently click on other links.



All of these technologies can be integrated into future Microsoft software products.
