New design admin WordPress.com



On the new look the guys have been working for several months. Back in April, WordPress.com users could experience a new look while it was being finalized and tested. Today, the new look of the admin panel has touched everyone, with the temporary ability to turn it off in the settings.

Flat design, enhanced contrast, Open Sans open web font and improved compatibility with mobile devices and tablets. The option is not final and work is still underway. There is a high probability that this particular design will enter the core with WordPress version 3.7 or 3.8. By the way, if you use the version of WordPress.org (and not WordPress.com), then you can connect the same look with the help of the MP6 plugin .
