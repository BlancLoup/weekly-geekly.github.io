Music from Vkontakte in Exaile

Hello!

Inspired by a recent topic about VKontakte's plugin for Amarok , I decided to write a similarity for my favorite player Exaile .




Plugin features:
  • - actually search for audio recordings
  • - remove duplicates
  • - plug-in operation without registering on the site
  • - multiple adding entries to the playlist
  • - saving selected records to a local disk in artist - track.mp3 format (in home directory)
  • UPD - select the directory for saving tracks in the plugin settings (if the specified directory does not exist, it is created)
  • UPD - load user playlist by his ID



Exaile bonuses received:
  • - pull-up disc cover
  • - rewind the track without waiting for its full download
  • - save / import playlist
  • - last.fm support and more ...

Installation:

Edit - Parameters - Plugins tab - Install plugin file - select vk_plugin.exz , then in the list of plug-ins opposite vk_exaile put a checkbox:



Plugin page on github
Thanks to realbc for testing!

+ added search by user ID (version 0.0.2)

I would be very happy if someone came in handy, was written for myself, with the goal of learning Python and the Vkontakte API.
