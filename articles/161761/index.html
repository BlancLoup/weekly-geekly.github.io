Price reduction for Windows Azure Storage

The official Windows Azure blog announced the next price reduction for Windows Azure Storage services. From December 12, Windows Azure customers who need to store information in blobs, queues, tables or disks of the Storage subsystem will be able to purchase the necessary storage at 28% cheaper .

This is another price reduction, the last time on March 8, 2012 prices were reduced by 12% . In addition to the price reduction announcement, the developers blog contains statistics that over 4 trillion objects have already been saved in Windows Azure Storage by companies and developers from around the world.

Today, developers of any platform and in any programming language through the REST API or through toolkits for .NET, Java, Node.JS, PHP, Python and mobile platforms Windows Phone, Windows 8 and iOS, which can available with source code.

Those who want to try Windows Azure can get a free trial for 90 days . Developers with MSDN subscriptions can start using Windows Azure for free . In addition, for companies that want to save a flexible system of discounts up to 32% is offered.
