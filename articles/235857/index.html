Official announcement of MEGAsync client for Linux

We are pleased to announce the release of the Linux version of our cloud storage sync client!

MEGAsync for Linux has all the same functionality as the versions for Windows and OS X, and finally offering the Linux community to take full advantage of the cloud storage with two-way MEGA encryption.

Supports builds for all major Linux distributions (Debian, Fedora, openSUSE and Ubuntu) for 32 and 64 bit platforms.
The client is also compatible with other distributions, such as Arch Linux.

MEGAsync for Linux comes with an optional integration package with the Nautilus file manager.
In addition, we plan to support more distributions and platforms, and develop extensions for other popular file managers in the near future.

At the moment, MEGA offers to use the software development kit ( SDK ), in the near future we plan to open the MEGAsync client code under an open-source license.

If you have any comments, suggestions, or you have found any shortcomings or errors, contact our specialists at linux@mega.co.nz (currently in English only).

PS From myself, as an employee of the company, I want to add that we are very happy to provide our Linux users with the client version, we have done a lot of work on optimizing and integrating the client with various distributions. We hope very much that it will be convenient for you to work with our client!
