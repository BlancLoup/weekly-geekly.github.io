TV with built-in BitTorrent client



At the Berlin Consumer Electronics Show IFA today presented "the world's first television with a built-in client BitTorrent." Directly from the remote control, you can select files, download torrents and immediately start to view.

Vestel 's TV was one of the first devices with an integrated Chrysalis system, which BitTorrent introduced in March 2011. This is a torrent client, based on the code uTorrent 3.0, but with a completely new, simplified interface. Included is a catalog of free content that can be launched with one click. Search for other torrents is redirected to Google with the automatic addition of the filetype: torrent parameter.
