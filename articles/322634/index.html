Free IP Camera Software

For 11 years of work in the video surveillance market, we had to deal with a lot of software for managing video surveillance systems, with some we worked, which we tested, which we wanted to test, but we didn‚Äôt have time to do it, we just read about it in some reviews like what you read.

As a result, as it seems to us, we have the most comprehensive list of software for video surveillance systems.

All software is divided into three groups:

  • Open source software
  • Free software with limited functionality
  • The software is free during the trial period.



Open source software


1) iSpy
The world's most popular open source video surveillance application. It is compatible with the vast majority of IP and webcams. Developers claim 2 million users worldwide. Source code is available, as well as compiled versions.

Opportunities
Rebranding - allows you to easily and customize the software for your brand. Paid and free plugins extend functionality.

Operating systems - Windows , Android
Official website: www.ispyconnect.com
Download: www.ispyconnect.com/download.aspx

2) ZoneMinder
Free open source software for organizing video surveillance. Works on any Linux distribution that supports Video4Linux

Operating systems - Linux (Ubuntu, RedHat, Debian, Gentoo), paid mobile applications on Android and iOS from the friendly project zmNinja are available .
Official site: zoneminder.com
Download: zoneminder.com/downloads

3) Veyesys Rapidvms
Rapidvms is a simple client server software from Chinese developers. Rapidvms consists of RapidStor (server) and RapidClient (client)

The source code is open for most components, some components are available only in binary form.

Operating systems - Windows, Ubuntu, Centos, macOS
Official website: www.veyesys.com
Download: github.com/veyesys/version

4) OpenALPR
Open source software for license plate recognition

Operating systems - Debian, CentOS.
Download: github.com/openalpr/openalpr

7) Axxon Next
High-performance video surveillance system of a new generation on an open software platform with an intuitive user interface from the Russian company ITV.

A software product from Skyros Corporation for building small video surveillance systems using IP and webcams.

Operating Systems: Windows
