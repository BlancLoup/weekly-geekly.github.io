Let's go back to the system unit. Minimally, it includes a processor + cooler + motherboard (a so-called platform ), RAM, HDD (SSD), a video card, an optical disk drive, a case with a power supply.
  • Recently, the choice of platform has become a simple task: if there is no $ 330 for a processor + mother, then AMD is taken, otherwise Intel. Those. The minimum Intel processor I can recommend is the Core i5-750.
