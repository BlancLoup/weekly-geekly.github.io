Platform for conducting urban quests

Hi, Habr! I would like to talk about one interesting project for conducting city quests, and also ask for your support in raising funds for the implementation of the pilot (http://planeta.ru/campaigns/6043).

So, it is planned to implement a platform for conducting urban quests using mobile phones and tablets at any time, without being tied to a specific date of the event.
To take part in the quest, it will be enough to download the application, register and select the quest of interest.

We are a team of like-minded people who have repeatedly participated in city quests and understand the subtleties of their conduct. We had an idea to create a platform for participating in quests anytime, anywhere. We are also a team of IT specialists (developers, analysts, testers and designers) who know and know how to make such systems.



Benefits
1. Carrying out quests without reference to dates, at any time.
2. Ability to create quests by users.
3. Introducing user ratings, game money and bonuses.
4. The possibility of ordering an individual quest.

Components
1. Website for users
2. A web site for volunteers and administrators
3. Mobile application for users
4. Mobile applications for volunteers and administrators

Below is a list of the functionality that we would like to implement (only a part of the pilot will be implemented).

Functionality
1. Different types of tasks:
- Text task
- Photo / video / audio task
- Logic Challenge
- The task of finding a specific code
- Search geocaches by GPS coordinates
- Tasks with augmented reality
2. Ability to create quests based on certain types of tasks by users.
3. Global user rating.
4. Holding tournaments and championships.
5. Bonus system (translated into play money):
- for the original answers
- for the speed of passage
- for the number of quests completed per day
- for inviting friends
- for the passage of a more complex alternative subquest
- high rating of user created quests
6. Using play money (you can call Gold), you can get play money in the following ways:
- purchase for real money
- getting bonuses
You can spend play money on the following services:
- use hints during the passage of the quest
- opening new quests
- the possibility of participating in tournaments and championships
7. The ability to create a series of quests on a specific topic (based on books / films / historical events), part of the quests can be free, the rest - for play money).
8. The possibility of independent creation and holding of events, for example:
- birthday party
- congratulations on any date
9. Notification of new quests for other users (you can enter tags by area, type, duration, mileage).
10. The ability to create quests "turnkey" for corporate clients.

If you have ideas for developing the platform, we will be happy to discuss your suggestions.

Link to the project in the crowdfunding system planeta.ru/campaigns/6043
