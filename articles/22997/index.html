Installing the application on iPod Touch with firmware 1.1.4

Charge your iPod Touch / iPhone completely!
Make the restoration of the firmware and Jailbreak to version 1.1.4 as written here .
Attention, all files, applications, etc. will be deleted!

To ensure that during the installation of applications there were no abnormal situations because of the screen self-locking (and with it, turning off WiFi with a break), set the player to Never in the player settings.
Training:
And so at the moment we already have a jailbroken player with firmware 1.1.4 and, accordingly, an Installer application is installed on our desktop. Run the application and install the Community Sources (in the Sources section). We are waiting for updates Sources (it all depends on the speed of your Internet). Then we put in turn:

BSD 2.0 termfix (Tweaks) - most likely already installed
SUID Lib Fix (Tweaks 1.1.3)
Term vt100 (System)
BossTool (Utilities)
BossPrefs (Utilities)

Increasing iPod Touch Space:
Exit Installer and restart iPod.
Open BossTools and select the first item, and then successively press three buttons, thus we will transfer all applications to another virtual disk of the player (we increase the free space).

Moving Mobile to Media
Download the winscp application and install it.
Start BossPrefs (do not change anything) just look at the iPod Touch IP address (SSH should be in the On position)
Run Winscp in the settings enter the IP address, root login, password alpine
Create a folder "Media" on the path "/ private / var / root /".
In Winscp (or PuTTy), open the terminal window and enter the following commands successively (Attention! ANY error at this step can lead to device malfunction, I recommend copying them from this page and pasting into PuTTy from the clipboard with the right mouse button):
  • cp -R / private / var / root / Media / private / var / mobile /
  • rm -r / private / var / root / Media
  • ln -s / private / var / mobile / Media / private / var / root / Media
  • cp -R / private / var / root / Library / private / var / mobile /
  • rm -r / private / var / root / Library
  • ln -s / private / var / mobile / Library / private / var / root / Library

With this we moved all the folders from the new folder (for firmware 1.1.4) Mobile back to the usual Media programs.

Installing beautiful:
Run the Installer, add a new Source danimator.techdocrx.com/testrepo.xml waiting for the update. Install Customize 8.3 (Danimator's Repo). Exit the Installer and run Customize. The program can run for a long time, and will crash three times, no problem (it will start 3-4 times).

And lastly the most delicious!
Installing the standard iPhone app for the iPod Touch for which our beloved Steve Jobs asks for $ 20 Mail, Maps, Stocks, Notes, Weather:
Run Installer, add new Source ipodtouchmaster.com/files/repo.xml
We put 1.1.4 App Pack (iPhone 1.1.4 Applications). Restart.
As a result, we have Apple iPod Touch with firmware 1.1.4 jailbroken + applications for which ask $ 20

We put the Russian keyboard on the iPod Touch as before.

If Safari forgets passwords (cookies) from websites, then in the terminal run the following commands
  • chown -R mobile / var / mobile / Library / Cookies /
  • chown -R mobile /var/mobile/Library/Cookies/Cookies.plist
