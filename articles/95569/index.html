Presented by Ubuntu 10.10 Alpha 1

On June 3, the first test version of Ubuntu 10.10 (Maverick Meerkat) Alpha 1 appeared in free access. From this release you should not wait for stability, because it is intended to detect errors. The release of the final version is scheduled for October 10, 2010.



Innovations:


Information provided by MaxPv
  • The GNOME desktop environment has been updated to the current version 2.31, which includes the new dconf and gsettings API.
  • KDE has been updated to the latest beta version 4.5 available.
  • It also underwent a cross-platform software development toolkit update - Qt (Qt 4.7).
  • Linux kernel 2.6.34.
