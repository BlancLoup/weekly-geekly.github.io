Determine the number of shutter actuations on the Canon 400D or how much the camera took

In the network there is a huge number of programs that promise to show the number of shutter releases on the camera. Some of them do not work at all, some of them simply do not work correctly, showing that the camera took 9 pictures in 4 years. The result is one - Canon 400D does not penetrate any of the utilities found.

Action plan
  • check the official firmware version of the camera, and update if necessary
  • Install an additional small firmware into the camera, which allows you to download a binary file from a memory card
  • we make the memory card bootable, we load our binary on it
  • Press the button sequence on the camera and voila: a cherished number!

You can find out the current firmware version of the camera in the standard menu. Before entering the menu, make sure that the manual settings mode is enabled (the wheel is on top, above the on / off button), for example, mode "P".
