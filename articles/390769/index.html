[Testing] High-frequency memory - relevant and timely assistance to the integrated graphics core

Hi, Giktayms! You can't catch a sparrow with a chaff, and it is almost impossible for an experienced piece of iron to prove the benefits of an integrated core and high-frequency RAM. Nevertheless, in 2015, a significant breakthrough in Intel graphics was made and under the cat I propose to evaluate how much the frequency of the RAM affects the performance of the GT3e graphics core (also known as Iris Pro 6200) integrated into the Intel Core i7-5775C processor.



Obviously, the RAM for the integrated graphics is video memory. It is logical to assume that the more productive the RAM, the higher will be the performance of the graphics subsystem. Why do you need it? Everything is very simple, for monoblocks, compact home solutions (like media centers or computers constantly connected to TV), dimensions, noise level and performance of all nodes without exception are important.

The fifth generation of Intel Core processors became famous not only for the fact that there were only two SKUs in the desktop segment, but also for the fact that for the first time in the desktop segment the user was offered Iris Pro 6200 Graphics - the best integrated graphics from Intel. It has long been known that AMD's ‚Äúinsertion‚Äù loves the frequency of RAM, was there a reason to check a similar argument for a competitor? Will increasing the memory frequency increase the performance of the graphics subsystem?

To test my hypotheses, the following test bench was assembled.

  • Processor - Intel Core i7-5775C
  • Motherboard - Gigabyte Z97-SOC Force
  • RAM - HyperX Savage DDR3-2133, DDR3-2400 , HyperX Fury DDR3L-1866
  • System Drive - HyperX Predator PCI-E 480 Gb
  • Graphics Card - Intel Iris Pro 6200
  • Power Supply - Seasonic X-1250 Gold
  • Operating system - Windows 10 Professional 64-bit




There are three scenarios of work: default settings - DDR3-1600, HyperX Fury DDR3L-1866, HyperX Savage DDR3-2133, HyperX Savage DDR3-2400.

The test suite of applications includes synthetic tests, 3D Mark, CatZilla, AquaMark3, Unigine Heaven and games - Resident Evil 6 and Metro: Last Light Redux.

The difference between DDR3-1600 and DDR3-2400 on average is about 7 percent, which is not a very significant value if we are talking about the performance of a discrete graphics card. In the case of a compact integrated solution, this increase will be a good help. An even more significant increase in performance will be in the case of overclocking RAM on its own. The difference in price between simple memory and high-frequency is perceptible, so I recommend keeping the balance and not chasing the numbers 3000 MHz, but below 2133 MHz, if you intend to make a fairly fast system with integrated graphics, do not fall.
