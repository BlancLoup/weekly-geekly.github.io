Github Education - Education and Github



On Habr√© there is a post habrahabr.ru/post/195054 "How to get a Micro account for GitHub student in Russia for free" from September 24, 2013 and no more mention of this interesting program from Github.
Meanwhile, on February 10, 2014, Github launched a separate website, education.github.com, and reported on its blog that they have been offering educational discounts for many years and more than 1200 educational classes and 7000 students have taken advantage of this offer at the moment.
Now they officially launched this program, made a separate website and offered the following:
  • Free Micro Subscription for Students and Teachers
  • Free Github Organizations for Classroom Use
  • 25% discount in all other cases for educational use.


And again ... how do I get a student micro-subscription?


Not again, but again. Subscription is very simple. (If anyone does not know, the point of it is that 5 private repositories become available with it). They described it is not very obvious, but there are two ways:
  1. With student card
  2. Using an email address in the domain of the educational institution

With student card

You need to send a small application to education@github.com (of course in English):
GitHub name: Your github nickname
School name: The name of the school (as far as I understand it is possible for universities, secondary specialized educational institutions, and even regular schools) also does not hurt to indicate the site address.
Graduation year: The year you graduate. It will depend on how long the subscription will be given.

Great, thanks for sending! Just apply the upgrade on your account. Enjoy!

Using an email address in the domain of the educational institution

Well, it's still easier here, just fill out the forms here education.github.com/discount_requests/new
The data set is exactly the same as with the letter, you only need to specify the email in the domain of the educational institution, and you do not need to attach the certificate.
