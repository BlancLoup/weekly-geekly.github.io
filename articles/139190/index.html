Facebook translates public pages into a new format.

March 30, 2012 all Facebook public pages will switch to a new design. But now, page administrators can preview and see how it will look, as well as try new features.


In the new design now the information is presented in the form of a chronicle. Also from the new features and features can be noted:

Page cover


In the new design, administrators of public pages will be able to place in the header a main photo (cover), which is much more than a profile photo. And the cover should be at least 399 pixels wide. Facebook claims to have an ideal cover size of 851x315 pixels. To add a cover, you need to click on the "Add cover" button.


Select and fix messages


Now the most important messages can be distinguished from other messages. To select a message you need to press the button with an asterisk. In this case, the width of the message will increase to achieve greater visibility. To pin a message at the top of the page, you must click on the button with a pencil and select the appropriate menu item.



Improved admin panel


In the admin panel made important tools and statistics. Here you can see notifications, messages, statistics, as well as ‚ÄúLike‚Äù marks.


Track your friends


Now, when the user visits the page, they will be able to see the interaction of their friends with the page at the top.

All the most important at the top


At the top of the page near the description, the administrator can place various information and objects (events, notes, videos, applications). Photos will remain in the first place, but all other positions can be changed so that to bring to the beginning the most important.
