Dune II: The Building of a Dynasty is one of the first real-time strategy computer games. The game was released far back in 1992, it required an i80286 processor, a VGA video card, 7 MB of free hard disk space and MS-DOS.

Although this game is not the first in the genre, it has established a format for real-time computer strategy games. The following elements of Dune II have become the standard of strategic games:
  • User interface including minimap and command bar.
  • Economic model (base construction, resource gathering, troop construction, technological tree).
  • Different capabilities of the warring parties.
  • World map on which the direction for the next battle is chosen.
  • "Fog of War", hiding unexplored territory.
