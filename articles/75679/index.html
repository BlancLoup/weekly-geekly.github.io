Silverlight 4 beta released


Today at the PDC09 conference a beta version of the popular Rich Internet Applications development platform, Silverlight, was demonstrated. When developing this version, the wishes of users were taken into account. You could vote for features. Now developing applications using Silverlight will be even more convenient. Many changes are aimed at improving the development of applications running in OutOfBrowser mode. The list of changes is truly impressive. In this version, the long-awaited print from the plug-in and interaction with the camera and microphone finally appeared.

Full list of changes:
  • Print support
  • Handle a right click event
  • Webcam / microphone access
  • Mouse wheel support
  • RichTextArea control
  • HTML hosting support
  • Clipboard support
  • Privilege escalation for trusted applications
  • COM interop
  • Access to local files
  • Drag'n'Drop support
  • H.264 protected content support
  • Managed Extensibility Framework (MEF)
  • Google Chrome support
  • Also a number of other changes.
