Free functional test webinar from TestCafe



Friends, I want to invite all sympathizers and people interested in testing web applications to a free webinar organized by us, dedicated to the latest release of our test framework TestCafe v15.1 . On May 26 we will tell about the main innovations and changes that appeared in the new version. By registering , you can ask questions and get an answer from the TestCafe team. Under the cut you will find details about the upcoming event.

The main topics of the webinar:


  • Automated Screen Capture
    The new version adds the ability to take screenshots of the tested page both automatically when a test drops, and at a certain arbitrary stage of its execution.

  • Improved presentation of test results. Export reports.
    We have completely redesigned the submission of test reports. To display them, a table is now used in which you can sort and group records, as well as search . Working with reports has become fast and convenient, there is support for exporting to JSON , JUnit , NUnit formats.

  • Detailed reports on failed assertions ( failed assertions )
    In the updated report tab, detailed information is now available on failed assertions ( failed assertions ). Now you can compare the result with the expected, as well as view screenshots.

  • Drop test when javascript errors occur
    An option has appeared that allows you to determine whether a test is considered to be failed if a javascript error has occurred on the page during its passage.
