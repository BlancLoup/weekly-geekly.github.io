Interactive data visualization Envision.js

Envision.js library for creating fast dynamic and interactive HTML5 data visualizations.


Opportunities:

1) Real-time visualization.
2) Timeline
3) Currency visualization (as it is on Yandex)
4) Ajax support interactively.
5) You can indulge in fractals.


Envision is based on the Flotr2 already mentioned .

Main browsers are supported. IE6-8 via FlashCanvas . Support for mobile browsers using WebKit.

Trifle, but nice.
