Launch Internet Explorer on Mac OS X

Hello! I want to share the answer to a question that I once asked myself at various conferences as soon as I got a macintosh. Unfortunately, Microsoft has long ceased to support its favorite browser for the Mac platform, so for this OS, there is only the old version of IE 5. More recent versions have not been released and it has become a problem for developers to run the latest versions of IE on a Mac. How to start Internet Explorer on a machine with Mac OS X installed,

The easiest way to do this is to download and install the VMWare Fusion distribution, and then install Windows XP on it, for which there is software such as IE Tester . This wonderful program is designed specifically for Windows and contains several versions of our favorite Internet Explorer - 5.5 / 6/7/8. Also, you can use Parallels or Bootcamp . In order to use these virtual machines, your mac should be on an Intel platform. If this is not the case, then there is another way - ies4osx , which runs under Darwine (this is a program that allows you to run software for the Windows family without starting the operating system itself).

If you know any other ways - write, I will add with pleasure.
