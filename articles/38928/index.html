Firefox 3.1 with TraceMonkey will be faster than Chrome

Of course, all Firefox users were greatly impressed by the staggering speed of the new Google Chrome browser in handling JavaScript (this speed was achieved thanks to the V8 engine). However, the Mozilla developers rushed to reassure everyone: yesterday they published the comparative results of the performance of V8 and the new TraceMonkey engine, which they will implement in Firefox since version 3.1.



Here TraceMokey overtakes the competitor by 19-28%. The SunSpider test was used in Vista operating systems on MacBook Pro and Windows XP on Mac Mini.

In addition, the developers provide a detailed benchmarking test for different scripts. In some places, TraceMonkey runs ten times slower, but in some places it has a fourfold advantage. On average, Firefox will still be faster.
