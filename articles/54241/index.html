OmniDiskSweeper, or clean the hard drive


Ends hard drive space?
Want to release a couple of tens of gigabytes?
I myself recently faced with the fact that of the 230 GB on the disk, there are about 30 free.
Approximately 50 are busy with music, as many more - Dr. House ...
How to find out what takes the rest of the place and where are these files?

In search of a solution to this problem, I came across OmniDiskSweeper , which, fortunately, recently became free. This allowed me to release about 40 GB, which the Downloads folder imperceptibly occupied. The screenshot shows that now I have 70 GB free.

Now a few words about the program itself.
OmniDiskSweeper has a simple interface, immediately after starting offers to choose the disk that you want to clean.

After some time, which the program calculates the size of files and folders, it will show them in a list, sorted by size.

Below there is just one delete button and an icon showing the location of the selected object.
The program also sees system files, including swap files.

So, if you know what can be removed without compromising the stability of the system, this will not be superfluous information.

You can download the program on the manufacturer's website.

I wish you a pleasant cleaning!
