Blender. Introduction to the node (Part 1)


It is very difficult to do without nodes in a blender to achieve interesting effects. With the help of nodes, a lot can be done. In this article, I will show a very simple and beautiful effect of replacing a gradient using the ColorRamp node.
Traffic (1.5 MB)

1. Open the blender



2. Add a Subdivine modifier to the cube on the scene.



3. Set the appropriate values



4. Add Shade Smooth to make the surface smooth.



5. Go to the settings of the materials of the object



6. Open the node window.



7. Put a tick "Use nodes" in order to enable nodes. Create a material.



8. Now create a new node and connect it to the Normal of the material in order to convert it to Dot, which will be used to change color using the ColorRamp node, since the transition of one color to another is established through the normals of the object itself.



9. Create a node ColorRamp, Mix, connect them as shown in the picture.


10. Set Node Values

12. For rendering, press F12.



13. Now with the help of the nodes we will change the rendered layer, namely we add the glow effect using the blur
