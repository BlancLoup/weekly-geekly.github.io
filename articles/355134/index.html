Rostelecom has released a browser for Russian families

Today, Rostelecom has released a browser called “ Satellite ” for Windows in beta status (for mobile platforms iOS and Android, it has already been available since the beginning of 2015). According to Sputnik product director Mikhail Kozlov, quoted by TASS, the novelty can be a competitor of Chrome and Yandex: “We are trying to make our browser more informative than Chrome and Yandex, our strength is in localization.” In addition, as noted by Mikhail Kozlov, the target audience of the browser are Russian families. The version of “Satellite” under MacOS should appear later.

The item “About the program” contains the information that Sputnik is based on the source code of the Chromium open source project.

If you try to install the "Satellite", it turns out that the browser interface is designed in the popular minimalist style. When you open a new tab, it displays tiles, which are essentially links to the services of the Sputnik portal. Apparently, it is impossible to change the number and location of the tiles (at least from the tab itself, although they are very similar to the live tiles in the Start menu in Windows 10). However, the tabs can be changed by downloading an image from a computer disk.


“Sputnik” in advance contains several well-known technologies with original Russian-language names. For example, the browser has an ad blocking technology called “Advertiser”, which works very similar to AdBlock. In this case, the “Extensions” tab in the “Satellite” Settings is empty. There is also a special mode “For children” (with a duck icon): when it is turned on, the start tab changes to the interface of the “Sputnik.Kid” portal with the hard filtering of search queries turned on. Apparently, the technology called “Stalker” is responsible for determining the dangerous content.

The amount of investment in the project "Satellite" is not called. In the nearest plans of the creators there is a unified system of authorization and synchronization, which would link together all the devices of users on which the “Satellite” is installed.
