Yahoo introduced a new analytical tool for advertisers

Yahoo, despite financial difficulties and problems in management (not so long ago, the new CEO of the company resigned after being accused of indicating false data in the resume), continues to create new services. Now the company is working on creating such services is not as active as before, but Yahoo is able to do something. So, a powerful analytical tool for advertisers, called Genome, was recently presented.

This tool helps advertisers working on the Web to get as much data as possible about their audience, which, in turn, makes advertising more targeted. Now the platform is only announced, and in commercial use Genome will be launched only in July of this year. The tool, as far as can be judged, is interesting, it allows you to collect all sorts of data about users (behavioral factors, etc.). This data is collected from Yahoo‚Äôs own resources as well as from resources owned by the company's partners.

According to the developers, the system will help marketers organize and analyze large amounts of data about users and their reaction to advertising, which will help increase the conversion rate and, consequently, increase profits.

Interestingly, Genome provides its users with proprietary data that is owned by Yahoo. Such data includes search preferences of users of corporation services, behavioral factors, and so on. It also uses data from the Interclick network, bought by Yahoo in December last year for $ 270 million.

Unfortunately, there is no more complete information about this advertising system, but the developers have promised to publish detailed data in the near future. And the service itself will start work very soon.
