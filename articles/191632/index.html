Microsoft re-releases MS13-057 and MS13-061

The company has published new versions of updates MS13-061 (v. 3) and MS13-057 (v. 3). We already wrote that MS13-061 (cross-Exchange Server 2007-2010-2013 RCE fix) was recalled after the August Patch Tuesday due to failures in Exchange Server 2013 and problems with stopping the indexing of letters on the server. This version of the update corrects errors made in the previous release. Users who already have MS13-061 installed will be prompted to update it with the latest patch version.



We wrote about MS13-057 in the July Patch Tuesday review . The update fixes the cross-platform Remote Code Execution CVE-2013-3127 (XP-7-RT) vulnerability in the Windows Media Format Runtime Library component (wmvdecod.dll). The attacker can take advantage of the vulnerability through a specially crafted media file and remotely execute its code. The latest version of the update fixes the problem of incorrect playback of some WMV-videos that appeared when using the previous version.
