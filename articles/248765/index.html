Microsoft has joined the Student Developer Pack and some other GitHub Education news.



In continuation of the previous post . Visual Studio Community 2013 and the rest of the Dreamspark products are available to students as part of the Student Developer Pack . In addition, hackpledge.org offers an hour of help, and GitHub wants to listen to teachers. Details under the cut.

Osmicotic backpack became heavier

GitHub made the first addition to the Student Developer Pack since launch: Microsoft DreamSpark from Visual Studio. You can already download Visual Studio and register to access the Microsoft program for student developers - DreamSpark.

With DreamSpark, students gain access to a wide range of Microsoft products, including software, the ability to publish apps and games for Windows and Xbox with Windows with a developer account in the store, and e-learning materials. All for free, which can not but rejoice.

Download Visual Studio and access DreamSpark using Developer Pack .
