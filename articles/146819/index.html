Plugin that converts Photoshop styles to css code

Did someone have time to get tired of this!

css3ps.com

Works as a web service, from the waiting time of the result of 30 seconds.

1) Download the plugin for the correct version of Photoshop, open via windows-> extensions-> css3ps;
2) draw a shape, apply styles, send to the server through the extension;
3) wait 30 seconds, look at the result.

Naturally, the service does not understand everything that you have done in styles.
Note that you need to draw the shape carefully.
