DEV Labs 2014. ONLINE conference for JAVA developers

We invite you to take part in the second this year conference series DEV Labs. The event will be held on April 19 and will be fully devoted to the development of the Java language. The format of the conference is one day, 4-6 presentations, from 45 minutes to 1.5 hours for each presentation. After each report, sessions are scheduled for questions. The purpose of the conference remains unchanged - we want to share our own experience in solving non-trivial tasks and hear the opinion of colleagues in the workshop.

Approved conference reports:

  • Unit testing of "untestable" code. Solution using JMockit and PowerMock tools;
  • New in Java 8. Detailed analysis of lambda expressions and Stream API;
  • How to choose libraries for a modern Java web application. Search for an effective Java component;
  • Spring is coming, spring is the way! New to the Spring Framework.


The beginning of the first report at 12:00 Moscow time.

More information about the reports and speakers can be found on the conference website . To become a member, you must register . Participation is free.

PS According to the results of the survey of the participants of the previous conference, we made some organizational changes: at the upcoming event, the reports will be grouped by the level of material complexity (the marking of the level of complexity can be found in the report description); in the middle of the day we will have a thirty minute lunch break.
