WordPress 3.4 Green Released



Released version 3.4 of the WordPress platform. The new version was named Green in honor of jazz guitarist Grant Green (to the music of which it was really nice to write this topic). As stated , version 3.4 significantly improves the theme customizer, custom headers, tweet insertions and image captions.

The most noticeable change is the theme customizer, which allows you to change the current or another theme in different ways without immediately applying these changes. In themes that support it, you can change the colors, backgrounds, and image captions.

In other places of the console you can also find some changes. For example, if you have a lot of topics, now you can view them faster all at once, without pagination. It became possible to use images from the media library for custom headers and choose the height and width of the image headers.

Now you can easily insert a tweet into a post: just insert a link to it, and it will turn into a inserted tweet. Finally, image captions have been improved, and HTML can now be used in them.

For developers, the claimed hundreds of improvements under the hood will be interesting, in particular in XML-RPC, themes API and custom headers, and a significant increase in productivity in WP_Query and the translation system. A detailed description of the changes can be found in the Codex, or you can dive into Trac yourself .
