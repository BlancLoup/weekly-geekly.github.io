Quick download images on habraeffekt

Once such a drunk talk about loaders for a habreffection went, then I thought that my homegrown programmaline could also be useful to someone.
There are 2 options:
  • cliUploader - load images from command line parameters. Each argument is an absolute or relative path (for example, you can add a shortcut to the Send context menu, or simply drag one or more pictures onto the fill label)
  • clipboardUploader - download from clipboard (for example, after pressing PrintScreen or Alt + PrintScreen)
After working out the program, the link will be copied to the clipboard, or if in the first case several images were sent, then several links separated by "\ n".
