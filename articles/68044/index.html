Groups


Groups solve two tasks:
  1. Exchange of stickers between a narrow (closed) circle of people.
  2. The separation of stickers on certain topics.
When creating a group, the user can select its name, add a logo, give a short description and select the type of group.
There are 2 types of groups available - private and public.
Any user can join a public group. To join a private group, you must obtain permission from the group owner.

A button appeared on the sticker to save it to the selected group. When you hover over it displays a list of user groups. In the case when there are more than 10 groups in the list, a field is displayed to search for the desired group, and with the help of autocomplex you will quickly find the required group.

If the group is private, only members of the group will see the stickers.
When connecting to a group, the user selects one of the types of notifications:
  • Notify immediately after group changes
  • Notify once a day
  • Once every three days
  • Do not notify at all
