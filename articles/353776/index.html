We invite you to a workshop on React Native from the developer GeekBrains


Hello! Friends, on April 20 we invite you to the Moscow office of Mail.Ru Group. GeekBrains developer Daniel Skrypnyk will talk about his experience with React Native, its features, and also teach participants to write a cross-platform instant messenger application. Program and registration - under the cut.


React Native is a frame for developing native JavaScript mobile applications for iOS and Android created on Facebook. Its advantages are in using a single code base for two platforms, widely used PLs, familiar patterns from React patterns, as well as a level of abstraction, which potentially allows you to expand the number of end platforms.


Daniil Skrypnyk has been engaged in JavaScript front end development for 3 years. He got acquainted with React Native technology two years ago and created 5 applications on it. One of them - the cryptographs - raised $ 15 million for ICO.


Workshop program:


16.30 - 17.00 The theoretical part.
We will understand:


  • How does React Native work and what are its advantages.
  • What are the differences between React Native and React.
  • What are the alternatives of React Native - their advantages and disadvantages.

17.00 - 18.30 Practical part, part 1
We will create a small application and immediately launch it on the phone. During the development process, we will learn:


  • Configure the environment and connect third-party libraries.
  • Work with routing in the application.
  • Use special tools to develop mobile applications and debug the application.
  • Create a build for Android and iOS.

18.30 - 18.45 Break


18.45 - 20.30 Practical part, part 2
This part will be devoted to the development of chat.


Beginning of the workshop: 16:30.
Address: Leningradsky prospect, 39, bld. 79, office of Mail.Ru Group.


To participate you must register here . Do not forget to bring your passport and laptop, see you soon!
