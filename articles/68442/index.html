Overview of alternative dialers for iphone (dialer's)

It's no secret that the standard iPhone dialer leaves much to be desired in the functionality. Manual dialing of a number has long since disappeared, few people remember the numbers of even the closest ones by heart, the address book ranks first in this regard. Each time, use the search to find the necessary contacts hemorrhoid, only the favorites tab is less in demand, but there is clearly not enough display of photos.

In this review, I will try to highlight all the dialers that can only be found in the Internet. If you use any other, be sure to drop the link / name of the application and I will add it to the review.

Attention, traffic! (A lot of screenshots)

Selection criteria:


1. Availability of a convenient search by contacts (Better using T9)
2. Displaying contact photos when searching / selecting
3. Support of the Russian language (Not the Russification of the program, namely the work with the Russian names of contacts)

Programs included in the review ->

Phone


Let's start with the built-in dialer.
The main advantage of the built-in dialer is that it is built-in) There is no need to swing anything, to install, to finish the file. It works and fine. Manual dialing has no complaints: you dial the number, press to call and call. There is a simple search in the address book (without displaying photos of contacts) But scrolling with Russian letters, which slightly simplifies the process of quick search
There is also a "favorite" where you can add your favorite numbers so that you can dial them quickly. There, too, the functionality is not at the level, most of the line is occupied by the inscription mobile .
In general, the functionality is low but sufficient. You can use it, but you want more. much more.


Third Party Software


Searchdial

There is a lot of similar software, we enter a name / phone number from the standard keyboard and we filter the list. The difference from the standard one is that it immediately displays the phone number and has a black diz. Verdict: to landfill.


ABContacts

The analogue of SearchDial, a bit more functionality, the developers invented a new way of using the iphone interface - clicking on the title brings up a menu from somewhere on the right, there is support for groups (both regular and auto-generated). Maybe it will be useful to someone, but definitely not to us,% username%


Callway

Analogous to SearchDial, but much more functional - there is a search through all the data of the address book entry with the help of T9, photo display, favorites, recent calls. In general, excellent software, with one minus - NO RUSSIAN LANGUAGE. Those can find a contact only by phone number.


Groups

Program from the author of the previous software, this time the emphasis is on user groups. Regular and smart (auto-generated) for example, you can select all those who have a birthday, or not photos and much more. O5 is just not happy with the complete lack of support for Russian letters. So the program remains convenient only for editing the address book (unless of course it is large)


Faces

This program is already different from others in the review. These are just good favorites for your contacts. It has a nice design, is as functional as possible for its purpose, it allows you to create screens / groups, it has a fine mode, displays photos of contacts in general, there are some advantages. Must have uniquely.


B-bot

And this program is almost no longer related to dialing, but I still want to talk about it in this review. This program allows you to do comic identikit and broadcast it to the contacts. (After all, not everyone gives themselves a normal photograph, and not always there is time and opportunity.
