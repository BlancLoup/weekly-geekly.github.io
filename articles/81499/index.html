TotalFinder - handy plugin for Finder

In addition to third-party file managers in Mac OS X, which are rather poorly integrated with the system and replace the Finder (with the exception, perhaps, of the Path Finder - it is quite far in this respect), you can use the usual Finder :)

For a couple of months already TotalFinder exists - a plug-in for a finder working through SIMBL, which allows a little to expand the capabilities of the file manager.

By the way, the author of this plugin is a person who has proven himself Visor 'om (Quake-like Terminal.app).
