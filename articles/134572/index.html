Homemade game controller for Nokia N900



A craftsman named Andrzej was so close to his mobile phone, the Nokia N900, that he decided to use it as a convenient gaming platform. Well, what is usually not enough for a comfortable game on a mobile phone? Of course, a normal game controller! And Andrzej decided to create it on his own, since he had the necessary knowledge and skills. No sooner said than done.

Andrzej created a miniature gamepad, which is great for the game, and quite convenient to connect. Connection is made via USB port. On the gamepad, there are only 8 buttons (apparently, ordinary gaming, plus Start, plus Select). The controller also includes a regular joystick with a PSP. According to him, some features of ATmega8A make it easy to use any keyboard shortcuts for any games.
