Google continues to destroy RSS

This week, namely May 8, Google has disabled the youtube RSS feeds.
In the bug tracker this problem is explained by the fact that the API is being upgraded and version 2 will be completely disabled by 04/20/2015.

In the meantime, for API version 3, there is a ticket for RSS feed support, but it is not known whether it will be executed. I personally doubt that why disconnect something in version 2 until it is ready in version 3, unless they want to get rid of it.

Also affected are Apple TV users who have lost easy access to their subscriptions.

At the moment, I know only one way to get around the problem: add feeds to all RSS feeds to your RSS reader. Feed URLs can be constructed using the Interactive YouTube API .

The ready feed URL is: 
http://gdata.youtube.com/feeds/base/videos?alt=rss&orderby=published&author={ID}

where {ID} is a user or channel identifier (for example, GTVScience or UCzWnF-3UWAGNeK5fIkBmahg )
