  1. We take the source code that we need to remember / encrypt / pack
  2. Paste the text into the Google / Yandex / Yahoo search string or any other search engine.
  3. We perform a search, and copy the resulting long link in the address bar to the clipboard
  4. go to goo.gl and insert the link
  5. click "Shorten"
