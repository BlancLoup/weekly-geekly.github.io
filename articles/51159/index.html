Ubuntu for netbooks: HP version

Hewlett-Packard has released a version of Ubuntu Linux specifically for netbooks. Namely, for its model HP Mini 1000 Mi Edition .

HP's proprietary build is based on the Ubuntu 8.04 Hardy Heron kernel and is capable of running all standard Ubuntu applications, including OpenOffice.org, Firefox, Thunderbird, Sunbird and Pidgin. Actually, all these programs are pre-installed.

What is so different about Mi Edition is the graphical interface, which has nothing to do with the standard Ubuntu interface. For example, immediately after the system is booted, such a window appears with the main labels and the Internet search string.



More screenshots under habrakat.

Perhaps the HP Mini 1000 Mi Edition assembly can be called one of the best Ubuntu distributions for netbooks, if not the best.

In the near future, HP will upload a utility to the Network to create a bootable USB flash drive from under Windows, so that any owner of a netbook will be able to install the Mi Edition.
