Watch the broadcast of Windows Camp today at 10:00 (MSK)



Dear developers!

We are pleased to invite you to the broadcast of the largest conference for developers under the Microsoft platform - Windows Camp !

When: September 10, 2015
Start of broadcast: 10:00 (MSK)
Broadcast available on the site: windowscamp.ru

About the event
Recently, on July 29, the official release of the new generation operating system, Windows 10, took place, which has already been tested by several thousand developers. The updated Windows Store and Visual Studio 2015 became available along with the new OS, opening up to developers the unique opportunities to create the latest applications for users from all over the world.


We decided to run Windows Camp and give Russian developers the opportunity to hear live the newest and most interesting from leading technology experts. At the event, you will learn everything about how to develop applications on Windows 10, about the possibilities of the new version of the most powerful development environment for Windows - Visual Studio 2015, and we will also tell how we see and make the Internet of things using Microsoft technologies.

At the conference, you are expected to:
  • deep immersion in the latest technology
  • technical reports on Microsoft key products, technologies and solutions
  • You can also ask questions to experts online and get answers to them during the broadcast.


Watch the Windows Camp broadcast and wait for a special QR code by clicking on which you can activate a unique badge! #wincamp #dvlup aka.ms/wincampru

The full conference program is available on the website .


Respectfully,
Windows Camp Team
