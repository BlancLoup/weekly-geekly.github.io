Qt SDK v1.1.3 release

We are pleased to announce an update for Qt SDK to version 1.1.3.

There are quite a few updates in this version, partly due to changes in the SDK internals, but there are also functional and qualitative changes. Here are the main ones:
  • Qt Creator 2.3: Many minor improvements. For more information, read the release announcement on the blog .
  • Qt Simulator 1.2: in addition to significant improvements to the user interface, Qt Simulator 1.2 has several new features: sensor simulation, NFC tag simulation, and gesture simulation.
  • Notifications API 1.1: QML bindings and improved performance.
  • Qt 4.7.4 for desktop application developers.
  • MeeGo 1.2 Harmattan beta: This version of Harmattan is built on the same image that was used in the June version, and does not contain new features, but it needs to be updated to continue the development of MeeGo 1.2 Harmattan applications.
  • Symbian Complementary Package update: important CODA updates to support the latest Symbian Belle devices. Please note that applications created with a beta version for Symbian Belle devices cannot yet be represented in the OviStore.
  • Updating Qt Quick components for Symbian do not contain functional changes.
If you already have the Qt SDK installed, you can upgrade to the latest version by running the Update Qt SDK from the Qt SDK application directory on your computer. If you want to get started with the Qt SDK, you can download 1.1.3 using the installers below:
  • Microsoft Windows - Online installer: 15MB - Offline installer: 1.4GB
  • Linux 32bit - Online installer: 23MB - Offline installer: 1.2GB
  • Linux 64bit - Online installer: 23MB - Offline installer: 1.2GB
  • Mac OS-X - Online installer: 12MB - Offline installer: 1.2GB
Nokia Developer also offers updated installers on their Qt SDK website .

If you have problems, please submit a bug report to http://bugreports.qt.nokia.com .
