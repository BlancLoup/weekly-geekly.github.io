Roskomnadzor blocked web services Amazon S3

The domain s3.amazonaws.com has entered the registry of blocked sites. Only one page of the 888poker game project is included in the registry, but Dom.ru blocks the entire domain, because A blocked page is entered in the registry with the HTTPS protocol with all the ensuing consequences. Although the lock was entered on the registry on May 11, for some reason, it began to operate only today. At the moment the domain is blocked in the network Dom.ru, TTK, Akado and some other providers. As a result, all S3 web services are not available, and, accordingly, the content placed in the S3 baskets.

Update : Dom.ru blocked the lock
