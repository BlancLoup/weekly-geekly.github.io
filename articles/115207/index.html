Updated user manual in the new version of nanoCAD

We are actively preparing for the release of the new version of the free base CAD platform nanoCAD. If we are interested in the fate of the project, we will be a little intrigued and for a start we will tell you what is new in the User Guide. Due to numerous requests from users, nanoCAD has significantly redesigned both the Guide itself and the reference material in the program. As a result, the size of the User Guide has increased almost 1.5 times! But the main thing is that the User Guide has become much more complete, the information and graphics have been brought in line with the new version, they have been significantly expanded and expanded.

In the new Guide, we tried to structure the information so that users could independently master nanoCAD - without having to ask a lot of standard questions. The manual will be useful both for those who are just starting to study CAD programs, and for ‚Äúadvanced‚Äù users.
