FireFox 3 beta 3

Today, the Mozilla Foundation has released the 3rd beta of the FireFox3 browser. Download the version under your favorite axis here .
You can read about the main changes (in Russian), for example, here , but I want to briefly tell you what caught my eye after installing the new version ...
  1. A tab in the extensions window allows you to search and install add-ons directly from the extension manager. Recommended extensions are offered by default, but you can search for any compatible extensions and themes by searching. Very good stuff.
  2. The authentication window on the site has changed. It has become wider due to changes in the explanatory text, which, in my opinion, has become less meaningful. Instead, enter the name of the password for "[Realm]" on [URL], it became [URL] prompts for a name and password. The site says "[Realm]" . Personally, at first, this phrase drove me into a stupor.
  3. In acid3, the third beta scored 59 points against 56 in the second.
  4. The requirement for memory, if it has changed, is not too noticeable, at least for the naked eye. However, already the second beta personally suited me in this regard.
  5. Tab Mix Plus extension stopped working - tabs are not closed by any means whatsoever. For me, this is the most critical.
