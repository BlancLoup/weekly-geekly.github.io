MoreCSS - Make css full!



MoreCSS is a small JavaScript library, weighing only 14kb , for solving everyday tasks, such as creating pop-up windows, making tabs for menus, tables and lists with a zebra-style design ... etc. With the help of this library you can do it by means of ordinary css.


Example: 
 a: active {
 target: popup;
 target-width: 640px;
 target-height: 640px;
 }

This code will allow you to open all links in a pop-up window of 640 √ó 640 px. ( Look )

Another example: 
  a, abbr {
 tooltip: title;
 tooltip-class: tooltip;
 }

This example adds a tooltip when the cursor hovers. ( Watch )

Installation: 
  <link rel = "MoreCSS" href = "path / to / the / MoreCSS / stylesheet.css" />
 <script type = "text / javascript" src = "MoreCSS.js"> </ script>
