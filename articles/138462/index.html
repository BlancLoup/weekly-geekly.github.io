Google may submit Andoid 5.0 already this quarter.



Apparently, Google is going to speed up the work on its next-generation mobile operating system, codenamed Jelly Bean. There is no official announcement yet, but some resources have already stated that this operating system may already be presented this quarter. Most likely, this OS will be referred to as Andoid 5.0, i.e. will be a full-fledged new version.

This rush is due to the fact that a competitor, represented by Microsoft with its Windows 8, also speeds up work, trying to release its own OS faster than Google. In addition, Android 4.0 Ice Cream Sandwich has not yet become widespread, so Google is trying to create another version of the mobile OS that can replace previous versions of Android on tablets and modern smartphones. True, Android 4.0 was introduced not so long ago - in October 2011. However, not everyone at Google is happy with the results - the OS is spreading too slowly.

That is, you can choose which OS to work in at the moment. You do not need to turn off or reboot the device, everything will work without rebooting. This option looks attractive, since manufacturers will be able to install two OSes on the tablets at once, including Windows 8 and Jelly Bean. Thus, Google is going to make a worthy competition to its long-standing opponent.

Manufacturers will now be able not to choose which OS to install on their devices, but to install both OSes at once, including both Windows 8 and Android 5.0. In general, an interesting decision from the "Corporation of Good".
