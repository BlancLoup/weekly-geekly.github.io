The first demo of MonoDroid - writing Mono / .NET applications for Android

Recently, the MonoDroid development team announced via twitter that the first 250 testers had access to the MonoDroid toolbox.

MonoDroid is a platform through which developers can write Mono-based applications (open implementation of .NET) for the Android mobile platform.

Below is a video that describes how to set up MonoDroid environment with integration into Visual Studio 2010 and write the first android application on Mono.



The MonoDroid toolbox will be available to developers both under Windows and under Linux and MacOS X.
