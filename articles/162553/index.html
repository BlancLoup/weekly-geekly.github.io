Support for Windows Phone 7.x will end in 2014

Good news for anyone who owns Windows Phone 7.x phones. Many thought that Microsoft would abandon Windows Phone users with old phones, but this turned out to be completely different.



Microsoft said that after the release of update 7.8, which is scheduled for early 2013, support for this version of the OS will continue for another 18 months. It's a good news. And if you count the time from the release date of the first version of Windows Phone 7 to the end of 2014, you get 4 years of support, which is an unprecedented number for the mobile platform. Even the iPhone gets updates for three years (though, there users get exactly the new OS versions, not the old ones), but if we consider that Windows Phone 7.8 supports almost all the new features of the G8, then these two numbers can be to correlate.
