Microsoft Tag Team Announces API

Microsoft Tag is a mobile application that allows you to link any object in the real world with interactive resources such as the Internet. The Microsoft Tag client application allows you to read a tag using a mobile phone camera and access additional information associated with this tag. A business can use the platform to attract additional customers, allowing users to get information about products interestingly. You can create and manage a label at http://tag.microsoft.com/ManageAds.aspx .

Microsoft Tag API allows anyone to create tags using the web, desktop or mobile phone.

Using the Microsoft Tag Web Services API, you can programmatically manage tags without using the Microsoft Tag website. Using the NET Framework, we can make programs that create, modify and delete tags.
