We wrote, we wrote ...


The possibilities of the pen are really great. It can act as a mouse, an electronic pen, an independent system of memorization of written text, a handwriting recognition system. About the last opportunity already written here .

I was also interested in how convenient it would be to autonomously save the written text and draw online, so that the interlocutors could immediately see the sketches drawn by hand. Now I will tell a little about the features of the work. The receiver is equipped with a built-in battery, 100-sheet memory and an IR receiver.
