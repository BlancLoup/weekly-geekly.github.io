Let's encrypt: start of the public beta on December 3


On the site Let's encrypt there was news that access to public beta testing will be open on December 3, 2015. For registration, you will no longer need the invite code.

"Since the launch of the closed beta on September 12, more than 11,000 certificates were issued and it became clear that the system is ready for public testing," the news said.



For reference: Let's encrypt is a certificate authority that issues X.509 certificates for TLS HTTPS, which are issued automatically for any domain for free. To do this, you only need to confirm your control over the domain by creating a special record on the DNS server or placing the code phrase in a file on the web server.
