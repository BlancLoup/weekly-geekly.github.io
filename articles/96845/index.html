New interface and functionality MobileMe is now available to all users of the service

Apple started testing a completely new MobileMe Mail interface back in May. And after the short status of Beta, the company introduced a completely updated MobileMe interface with new features, which is now open to all users of this service.

From now on, every MobileMe subscriber can enjoy new features of the service, including support for a widescreen interface, new mail rules, improved security with SSL and much more.


Apple also added 2 more new features in MobileMe , which were not available when testing the Beta version. The first is an advanced anti-spam filter. The second, more interesting is the support of external addresses. The user can log in to the mail settings and add email accounts from Gmail, Yahoo, or any other service that supports the POP protocol. After that, it will be possible to receive mail from these addresses in MobileMe , and when replying to a letter, it will automatically be sent from the mailbox to which it arrived.
