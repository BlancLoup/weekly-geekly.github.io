MODx Revolution 2.0.5 released

After just a couple of months after the release of 2.0.4, and also thanks to the growing number of developers using MODx Revolution, we have one of the most "stuffed" releases.

Form Customization has been completely rethought and redesigned to make it much easier to use. In 2.0.5, we also included Access Policy Templates, which will allow you to more easily manage user rights.

Here are the main changes in Revolution 2.0.5:
- Over 175 fixes!
- Redesigned form customization
- New access policy templates
- More robust code in relation to multiple contexts and file uploader settings.
- Many xPDO improvements and fixes
- Many UI changes, such as an improved resource tree, more phpthumb settings, improved drag & drop
- Fixed problems with APC, WinCache, eAccelerator and other caching systems
- Enhanced security measures for XSS, added additional checks for data transmission.
- Fixed problems with radio TV parameters, resourcelist TV and other rare conflicts in the work of TV parameters.

We would not be able to make such a massive release without your bug reports and questions about new features, without suggestions for improvement and the proposed updates to the language files. Please continue to help make MODx Revolution better.
