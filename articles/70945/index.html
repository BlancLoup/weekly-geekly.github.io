20 Fantastic Latest jQuery Plugins

This post contains a collection of great penultimate plugins and tutorials that can add a spark of creativity to your design.

Translation of this article


  1. FacyBox - jQuery Based Facebook / FancyBox Style Lightbox

    FacyBox - lightbox in the style of Facebook / Fancybox which can display images, divs or page contents.


  2. jQuery Infinite Carousel Plugin

    Unlike other carousel plug-ins that finish displaying pictures on the last one, this one shows pictures endlessly in a loop;)


  3. NyroModal - jQuery Modal Plugin

    NyroModal is a plugin that allows you to easily add modal windows for different content: images, galleries, frames, AJAX data, video, error handlers and DOM elements.


  4. jQTouch - jQuery Plugin For Mobile Web Development

    jqTouch plugin that allows you to easily create web applications for mobile devices with WebKit browser (iPhone, G1 and Pre). Now you can create cool-looking mobile apps using only HTML, CSS and JavaScript.


  5. AnythingSlider jQuery Plugin

    AnythingSlider is a well-pumped slider plugin that is very easy to use and configure. Adding and deleting slides is the same as adding and removing <li> tags in code.


  6. jQuery YouTube Playlist Plugin

    The YouTube Playlist Plugin allows you to list YouTube links as a simple playlist. He will automatically add the player to your page and from the links will make a playlist;)


  7. Uploadify - Multiple File Upload Plugin For jQuery

    With Uploadify, you can easily add multiple file uploads to your site. Easy to configure, even for beginners, but also gives broad settings for advanced users;)


  8. Encrypting HTML Form Data With jCryption

    jCryption is a plugin that encrypts HTML form data using JavaScript when performing POST / GET requests.


  9. Content Translation With jQuery Translation Plugin

    The SundayMorning plugin allows your users to translate the contents of your pages into 30 languages. You can translate as using the menu, as simple as words so using remote calls.


  10. Horinaja Slide Show Plugin For jQuery And Scriptaculous

    Horinaja is an easy-to-use plugin that is easily customizable "and will be very useful on the page." Powered by jQuery and Scriptaculous. One of many features: allows you to change slides using the mouse wheel.


  11. jQuery Week Calendar Plugin

    The jQuery Week Calendar plugin allows you to make a simple and flexible calendar for your application. Based on jQuery and jQuery UI, it is easy to implement and customize to fit your needs.


  12. AD Gallery - jQuery Gallery Plugin

    AD Gallery is a widely customizable plug-in slideshow and gallery. It supports several built-in effects: horizontal and vertical scrolling, dimming, reducing / enlarging and wild style (see demo).


  13. jqDock - Fish Eye Menu Powered By jQuery

    With this plugin you can easily add a fish eye menu (Like MacOS Doc) to your site.


  14. AnythingZoomer - jQuery Plugin to Zoom Anything

    AnythingZoomer makes it easy to increase everything on your page: images, text, etc.


  15. jMonthCalendar - Full Month Event Calendar Using jQuery

    jMonthCalendar is a calendar plugin that shows the entire month and allows you to add events.


  16. Put Twitter on your own jQuery Plugin

    Juitter plugin allows you to post the latest tweets on your page. Easy, fast and well customizable.


  17. GalleryView - A jQuery Content Gallery Plugin

    GalleryView gives users a flexible and attractive gallery that is easy to build and customize. He is able to display any content in the form of an animated gallery.


  18. Jcrop - The jQuery Image Cropping Plugin

    With Jcrop - easily and simply crop and add images on the page.


  19. jQuery UI Selectmenu Plugin

    jQuery UI Selectmenu plugin that duplicates and expands the functionality of native selects, and allows them to customize the appearance and behavior, icons, and create optional groups.


  20. jQuery SuperBox!

    SuperBox is a simple Lightbox for images, their groups, external pages and internal content of the page.
