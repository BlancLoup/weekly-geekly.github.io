Overview-rating providers virtual servers Windows: 2017



In March 2017, there was a need to upgrade the server hardware for MS SQL Server and the terminal server, which are necessary for the operation of the sales accounting application. The cost of the new server and storage system exceeded 700 TR, which did not fit into our budgets, so we decided to consider virtual servers for hosting our applications.

Almost immediately abandoned the dedicated server due to the high cost of implementation: equipment, licenses, services of engineers to configure. Therefore, we decided to choose a ready-made platform for the realization of our needs, this is a virtual server.
We can highlight the following benefits:

  1. Implement fault tolerance at the hypervisor level;
  2. Mainly fault tolerant data storage;
  3. Low backup cost and simple implementation for us;
  4. Expansion of resources in accordance with our needs (Pay as you grow);
  5. Availability SLA.
