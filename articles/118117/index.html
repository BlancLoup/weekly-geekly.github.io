Contest for developers Rails-ninja

Look At Me is not only a popular site about modern culture, but also the largest Ruby on Rails project in Eastern Europe.
In anticipation of the Rails Club conference to be held on April 30, Look At Me together with the community of developers RailsClub launches the Rails-ninja competition.

The task of the competition is to implement the mechanism for generating JavaScript validation of HTML forms by reflecting the declarative validation in RubyOnRails using the validates _ * _ of (* args) methods.

Requirements:
  • The application must be written using the RubyOnRails platform.
  • All third-party components should be free for commercial use (in other words - should be OpenSource).
  • The repository with the source code of the project must be located on the github.com service.
  • To demonstrate the project must be posted on the Internet by the participant independently.
  • The project should contain a brief description of the implementation algorithm.

Works are accepted until noon on April 29, Moscow time!
Summing up on April 30 at the RailsClub'Moscow conference . Winner will get iPad64.
