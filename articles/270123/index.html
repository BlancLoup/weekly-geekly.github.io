Amazon Web Services is now in the Github Students Development Pack

Yesterday, Github announced the addition of AWS Educate to a set of free tools and services for students. From now on, all students enrolled in the Github Education program will receive such tools as DB, cloud-computing, data analysis and much more.

The program provides access to educational information, including: laboratory / homework and a portal for students. Each student receives $ 35 ($ 100 for universities registered in the program) + $ 15 bonus from github, which can be spent on paying AWS bills. Total can get up to $ 115 grant.
