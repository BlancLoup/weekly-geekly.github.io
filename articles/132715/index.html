Developers updated Twitter interface



Not so long ago, the creators of the microblogging service Twitter introduced a new interface and a large number of new features for their product. Now the Twitter interface has been updated again, with the updated version of the service interface being a logical continuation of earlier work to improve user interaction and the resource itself. For example, now the activity of the user and his followers is displayed on two independent tabs.

These tabs are called username and Activity, which is quite logical. But the @mentions tab is no longer there, the developers have removed it from the interface. So, as for the first tab, the account owner will be able to see detailed information about his account with it. For example, you can track who and when became your follower, or which of your tweets were added to your favorites.

The second tab displays detailed information about subscribers.
