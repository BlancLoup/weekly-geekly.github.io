Ruby Codecademy Course

According to the numerous wishes of the workers, the educational portal Codecademy presented a course on the Ruby programming language.

The course is interactive, does not require installation of the runtime locally, the interpreter works directly in the browser. It consists of more than a hundred exercises, divided into 5 blocks. Part of the exercises of each block is introductory, and part is an example of the implementation of the simplest applications.

The most interesting course will be, in my opinion, for those who start learning Ruby not as another language, but as the first, for those who start learning programming in general. In particular, for those who are familiar only with HTML + CSS, but want to start developing applications based on the Ruby on Rails framework. The concepts of "variable", "object", etc. are introduced in the course of the presentation.

The course is in English, but simple and clear even for me with my Soviet English.

Well, actually the link to the course: www.codecademy.com/ru/tracks/ruby
