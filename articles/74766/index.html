MacOS 10.6.2 update released

A long-awaited update of MacOS X numbered 10.6.2 has become available for download. This update is recommended for all users of Mac OS X v10.6 - 10.6.1 Snow Leopard. It contains general operating system fixes that improve the stability, compatibility, and security of your Mac.

Major (most welcome) changes include:
  • Solved the problem that led to the deletion of data when using a guest account;
  • solved the problem that led to the unexpected termination of the current user session;
  • Solved the problem with the display of Exchange contacts in Spotlight search results;
  • improved stability of add-ons to the menu;
  • Clover solved a problem that arose if Hebrew was used as the main system language;
  • a shutter sound effect while saving a screen shot;
  • solved the problem with a four-finger swipe gesture;
  • Solved the problem with adding images to the contacts of the Address Book;
  • Solved the problem with connecting monitors to DVI and Mini DisplayPort adapters;
  • Solved the problem that led to the fact that after rebooting the brightness settings could not be saved;


In fact, there are still a lot of changes, and you can read about them in more detail on the Apple website .

In the Russian version of the update downloads page yet, but in the English - it is already available .
