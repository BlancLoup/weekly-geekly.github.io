Gentlemen and ladies, fellow programmers - hot news from fring: we have just released the fringAPI release, which now allows almost anyone to create and implement any Web applications for the fring community.

This means that now any of those new applications that you are developing are automatically supported by all the functionality of fring, including presence indication, conversation, chat, file transfer, automatic roaming, and all other functions (i.e., the API provides access to GUI client fring with web browser extensions and basic phone features).
Not bad for a start!

So how do you get started with all this?
