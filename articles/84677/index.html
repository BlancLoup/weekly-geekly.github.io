Plugin for Redmine

Good day to all.

At work put the project management system Redmine. It is pretty well described in Habr√©, so I will not repeat.

Its structure includes a default wiki in which you can maintain any documentation. And for convenience, the "table of contents" is built into this wiki, i.e. tree view of all wiki pages alphabetically.

With a large number of pages, the table of contents becomes, in my opinion, terribly uncomfortable.

To fix this, I wrote a small plugin for redmine, which can be downloaded from GoogleDocs

Under the cut is an illustration of the plugin.





This plugin does the following:
1. If the page has no child pages - then its name is made a link to the page.
2. If the page has child pages, then the link to it will be "=>" characters to the right of the name, and when you click on the page name, the list of children will be hidden / shown.
