It is finished: Microsoft Remote Desktop Connection Client for Mac 2 is out of beta testing

Small-scale finally finished beta testing and uploaded for download the second version of rdc for the macOS platform.

System requirements: MacOS 10.4.9 (Tiger) or Leopard.
The following versions of Windows are supported:

Windows vista


  • Windows Vista Business
  • Windows Vista Enterprise
  • Windows Vista Ultimate


Windows XP Editions



  • Windows XP Professional
  • Windows XP Media Center


Windows Server 2008 Editions



  • Windows Server 2008 Datacenter
  • Windows Server 2008 Enterprise
  • Windows Server 2008 Standard


Windows Server 2003 Editions



  • Windows Server 2003 Datacenter
  • Windows Server 2003 Enterprise
  • Windows Server 2003 Standard


Opportunities:
  • Universal binary - run on Intel and non-Intel Mac machines
  • New interface and menu with integrated commands for working with the clipboard, etc.
  • Support Remote Desktop Protocol 6.0 (Vista and ws 2k8)
  • Multiple simultaneous sessions support
  • Customization customization (keyboard shortcuts and so on)
  • Automatic reconnection at disconnection
  • Fullscreen and windowed support (with resize)
  • Print
  • Network Level Authentication support (when using Vista)
  • Widescreen support
  • Console connectivity (without gui)

Detailed information on the microsoft website

Download at download.microsoft.com
