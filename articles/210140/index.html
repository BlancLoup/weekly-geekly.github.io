These are two new lines of mobile devices, which include four models: one Samsung GALAXY NotePRO tablet with a diagonal of 12.2 inches and three variations of the GALAXY TabPRO with diagonals of 12.2, 10.1 and 8.4 inches.



Top models GALAXY NotePRO and TabPRO are equipped with the world's first WQXGA screen with a diagonal of 12.2 inches and a 16:10 aspect ratio, which provides a high resolution of 2560x1600 with more than 4 million pixels on board. The cameras built into these tablets are 8 megapixel main and 2 megapixel frontal. For performance in versions with WiFi and 3G, Exynos 5 Octa (quad-core 1.9 GHz + quad-core 1.3 GHz) is responsible, and in the LTE version - Snapdragon 800 (2.3 GHz Quad) and as much as 3 GB of RAM. There are options with 32 and 64 GB of internal flash memory and a slot for expanding the memory with microSD cards (up to 64 GB). The battery capacity is an impressive 9500 mAh. The dimensions of both mega-tablets are 295.6 x 204 x 7.95 mm. The NotePRO version with Wi-Fi support weighs 750 grams, and the 3G / LTE version weighs 753 grams. TabPRO is a little easier: 732 and 740 grams, respectively. All this stuff works under the new Android 4.4 (KitKat) mobile operating system. And what is the main difference between these two models, you ask? And it consists only in the presence of the S Pen and the corresponding software for it in the NotePRO model.

Also, new tablets will be pleased with a set of useful tools for living and working. Magazine UX's customizable interface, optimized for the big screen, features an easy-to-use dashboard with automatically updated news feeds and quick access to frequently used applications. The Multi Window function can divide the screen into 4 parts, and the new virtual keyboard provides comfortable typing due to its large size and feedback. The good old pen-on-all S Pen is also available. The tablet is able to connect to a home or work computer using the Remote PC service, gaining access to any files for remote viewing or editing.
