Accelerate JavaScript (TraceMonkey) in Firefox, without disabling extensions.

I will tell you about switching to Firefox Minefield without chopping off plug-ins, while preserving your profile. (all passwords, cache, cookies, extensions - remain unchanged)

If you just install Firefox Nightly, extensions write about incompatibility, and chop down. We get at the output bare Firefox without extensions. Below I will describe how to fix this problem.

In order, what to do :
1. Download and install Firefox Nightly Minefield by the link .
2. type in the address bar "about:config" (without a space! Space added Habraparser)
3. change javascript.options.jit.content key value to true
4. Download and install the Nightly Tester Tools 2.0.2 add-on (this extension is needed to disable the extension compatibility check)
5. reload browser
6. Enjoy increased surfing speed.

Subjectively - the speed (compared to 3.0) increased.
