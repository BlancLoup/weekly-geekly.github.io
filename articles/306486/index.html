Tools for remote access to servers running Windows

Often, users of our VPS hosting are asked to advise them which of the remote access tools can be used to connect to a virtual server, in addition to the standard connection to the remote desktop (RDP).

Currently there are a huge number of different tools that allow you to remotely control a computer or server. Each of them has its pros and cons, different functionalities, paid or free versions, as well as mobile or only desktop options.

This article will discuss the two most famous and popular applications: Radmin and TeamViewer. Their advantages and disadvantages in benchmarking with remote connection to the remote desktop (RDP) from Microsoft.

First consider the program Remote Administrator (Radmin)


Radmin was created by Famatech, which is one of the world leaders in the development of remote computer control systems. Since its founding in 1999, they have been developing some of the best solutions in their field, which are widely used to provide technical support for users, administer corporate networks, work remotely and solve many other tasks.

Distinctive features of the program is ease of use at great opportunities and not a high price.
The main advantages that allow you to use it with maximum opportunities for you are:
  • Unsurpassed speed with DirectScreenTransfer technology
  • The highest level of security standard AES
  • Hardware-based remote management with Intel AMT technology
  • Full compatibility with Windows
  • Compatible with Wine (remote access from machines running Linux)
  • Secure file sharing with the "resume" function
  • Multiple simultaneous connections support
  • Bilateral work with the clipboard with Unicode support
  • Password guess protection with a delay after five consecutive failed attempts
  • Record in the log file of the user name and DNS decryption of his address
  • Easy to use
  • A 30-day trial version is available.
  • Benefits for registered users:
    • Perpetual license
    • Full technical support by e-mail

However, the program has only a test free version with a 30-day period, after which you will need to refuse from further use or purchase it. At the same time, Famatech provides flexible licensing options. Only the server part of the program - Radmin Server is subject to licensing, the client part - Radmin Viewer is distributed free of charge and can be used on any number of computers.

The next of the most popular remote access solutions, we looked at the TeamViewer program.


TeamViewer is familiar to many of you as an application for creating conferences, team access and collaboration. In its activities, the development company specializes in technologies that use cloud resources to provide online support and real-time interaction around the world.

The program has gained great popularity and the number of downloads, according to the information on their website, has reached 1 billion!
The main advantages that have led to such high rates are as follows:
  • Multiplatform
  • High security
  • High performance
  • Hidden view of the desktop, file transfer, organization of chat and interactive conferences for 25 people
  • Video recording of work with a remote desktop, making screenshots
  • Availability of portable and mobile version
  • Wake-on-lan
  • Synchronous clipboard
  • Teamwork
  • Connect to mobile devices
  • Linux systems without a graphical user interface
  • Works from the browser
  • Terminal server support
  • Free for test and personal use.
  • Clear interface
  • Available in over 30 languages.

The disadvantages of the TeamViewer program include the need for initial registration and its high cost.

The last of the possible ways to connect to the server, consider the standard Windows program, which in the Russian localization Window is called ‚ÄúRemote Desktop Connection‚Äù.


This program is presented in all editions of Windows since version XP. It is quite simple and easy to use, even for inexperienced users who are just starting their familiarity with virtual servers.

The main advantages can be identified as follows:
  • The program does not require installation if you have all devices on Windows
  • Supports connecting external devices from your computer to the server
  • The ability to connect local drives and USB-devices
  • Uses shared clipboard. Allows you to directly copy files
  • Allows you to run an unlimited number of connections
  • Ability to adjust color quality and screen settings
  • Setting up in the client autorun of various programs when connected to the server
  • High security, authentication and authorization
  • Free use for administration purposes

There are several points to possible disadvantages:
  1. the need to buy / rent a RDS CALs license, when using a server, not only to configure and administer it, or if more than two simultaneous connections to the server are required.
  2. no possibility of creating a conference

The determining issue for many users when choosing a means of remote access will be the question of cost. Licensing terms are different for everyone, but for most of our hosting clients, the most popular task is to organize simultaneous access to the server for up to 1 to 10 people (rarely seek a large number of RDSCALs).

When choosing a tool for remote access to the VPS , in addition to the cost, you can pay attention to how actively these programs use the resources of your computer and your network.
To find out which of the programs uses your resources least of all, we conducted the corresponding load tests in the two most significant situations:
  1. normal server work
  2. Watching a 720p video on YouTube on the server to check how effectively the desktop image is transmitted (the comfort of your work largely depends on this)


The latest version of the Google Chrome browser was used to view the video on the virtual server , and the standard Windows Resource Monitor program was used to monitor the load on the client.
The obtained indicators were presented in the form of a table, which gives us the opportunity to see how the indicators have changed during normal operation and when watching a video in full screen.

The data transfer rate in the table was translated to Mbps, while in the screenshots it is shown in bytes / sec. The colors indicate indicators of the load on the equipment, the green indicator highlights the minimum indicator, the red indicator highlights the maximum indicator.
We must note that the video quality (frames per second) was disgusting in all cases: 2-3 frames per second

After testing and examining each of the programs, it is possible to draw conclusions under what conditions they are best used:

A standard remote desktop connection is perfect in cases where you do not want to install a third-party application and you do not need rich functionality, but only access at some time points to the remote desktop server to configure it.

The Radmin program allows you to use it comfortably in all situations, in combination with great functionality, low cost and minimal load on the equipment of both the server and your PC.

TeamViewer program is wise to use if you need to organize jobs for a large number of employees. And at the same time, you would like to get all its capabilities to provide online support and real-time interaction of employees, despite the heavy workload it generates.
