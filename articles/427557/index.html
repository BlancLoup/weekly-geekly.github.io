IT digest of November events (part one)

We are serving another portion of recommendations on large conferences and modest meetings in different regions of Russia. The release language is Python, the hackathons are still numerous, and interest in blockchain and big data after a short break clearly flared up with a vengeance.

PiterPy # 5

When: November 2
Where: St. Petersburg, starting, 6, lit. BUT
Conditions of participation: 20 000 rubles.

And this international conference on the language of Python opens the door for all programmers. When: November 2
Where: St. Petersburg, starting, 6, lit. BUT
Conditions of participation: 20 000 rubles.

Efficient Infrastructure Selenium

When: November 2
Where: Yaroslavl, October Lane, 3, office 207
Participation conditions: 5000 rub.

The training is intended to show web developers and test automation engineers how to effectively use Selenoid and Ggr to create a scalable browser automation infrastructure, what are the beauty of containers, and how fail-safe clusters are obtained. Experience with Selenium, Linux command line and Docker is welcome; It is also recommended to have an installed Docker and a couple of Selenium tests with you. In total, the training will last eight hours, which will be divided between the three blocks. First, students will learn about the history and essence of the Selenium project, learn how to install and launch it, and sort out the main features. The next block will be devoted to setting up a cluster of Selenium and Ggr, as well as cluster management. In conclusion, the organizers will talk about the advanced features of Selenoid.

Machine learning on BigData projects

When: November 2
Where: Yaroslavl, October Lane, 3, office 207
Participation conditions: 8000 rub.

At the same time, training will also be held in the same place for those who are familiar with the backend, Java / Scala, Python / R or C # / C ++ and wish to deepen their knowledge of BigData for future (or current) projects. The requirements for all these people are the same - basic knowledge of Linux, Java and SQL. The theoretical part of the training covers the basic concepts of machine and deep learning, libraries in Java, integration with the Python-ecosystem, work with the Spark cluster, machine learning chains, Kafka, Flink and Storm. Each module is accompanied by practice - the execution of tasks, live coding sessions and a closer acquaintance with the tools.
