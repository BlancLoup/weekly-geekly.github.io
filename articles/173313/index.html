CMS Koken for photo galleries



There is a need to create a website with a photo gallery. And he went to look for a CMS for this comfort. After some searching, I found an unpleasant trend. There are CMS for photo galleries, but not with modern navigation, without Lightbox and so on. There are beautiful HTML templates with all sorts of goodies (for example, Gamma Gallery), but without a content management system. Considering Wordpress and others for a site with photos only is too expensive. Even sad, but the search pushed one new CMS Koken. After a brief review, I realized that this is exactly what I need: a first-class admin panel with a lot of settings, and a modern display of galleries. And with a file so all the beauty turns out.

Installation and Overview


To install, you need to copy a single index.php file to the domain. When it starts, the installation window will open:



Then everything is as usual for installation. After installing and entering your login and password, you are greeted by the main site admin window at [domain or folder] / admin :



As you can see, the appearance is very similar to many image viewers, which is very convenient. In the central area we see the display area of ‚Äã‚Äãour drawings. Drag some of our image files onto it. Then the following window will appear at the top:



Our drawings are loaded. When you select an image, a panel for editing image parameters appears on the right: name, description, labels, category, image license, etc., as well as uneditable properties of the image: image resolutions, its name, and so on:



Upstairs there is the main admin panel:



These are the main admin sections:

Library - Library of our images. Here we can group them into albums, edit their properties, delete them, and so on.
Text - Even in the photogallery itself, there are most likely some text pages: about the author, contacts or something else. Here they are editing. And if you meet the word essays in the themes, then these are just these text notes.
Site - directly work with the appearance of the site, which will be shown to users. You can change themes, change pages, and so on.
Settings - a set of site parameters.
Upload - add images or video directly.

You can change the theme of the site. To do this, go to the Themes section:

Images can be grouped into albums that can be collected in a collection, you can add to favorites, there is support for working with Lightroom and much more.

Creating your own theme


To create a unique your own theme, you need to take a few steps.

Go to the folder / storage / themes / .

Copy the folder of the topic that you most liked and which is closest to what you want on the structure of the photo gallery. And call the folder somehow in its own way (without spaces).

Open the info.json file in the theme folder and change our theme data there. First, change the theme name to your unique one.

Remove the file koken.guid from the folder with the new theme. It is necessary.

And activate the theme through the Site section.

Total


Pros:
+ Chic admin panel;
+ Modern gallery display themes;
+ Very operational support from the creators;
+ Free;
+ Ability to create their own;
+ Active development and constant updates;
+ The possibility of blogging.
+ Adaptive templates

Minuses:
- While only English (but you can translate yourself, and the developers are making progress in this direction);
- Lightbox in the mode of viewing all the content you have to screw yourself;
- Lack of commenting system (but there is Vkontakte and Disqus);
- There are still small unresolved bugs, but which can be solved manually;
- If you hover over a thumbnail of an image, I would like some effects, but this is done by hand again.
- The names of image files containing Cyrillic, the engine "cleans up" UPD. eafanasov
- No image caching. UPD. xdevel
