3CX Phone System 12.5 Final Release Released

The 3CX Team is pleased to present the final release of 3CX Phone System v.12.5!

We offer a brief overview of the new features of version 12.5:


  • The call can now be returned if after the transfer if the subscriber did not answer or was busy.
  • Updated 3CXPhone Mac client - added echo cancellation, scheduled video conferencing and improved work with the address book.
  • Improved chat for 3CXPhone for Windows - it is now possible to add a video via WebRTC to a chat and create a conference in a few seconds.
  • Newly supported IP phones: Htek (Hanlong) , Fanvil X3 , as well as Polycom SoundStation and Polycom SoundPoint .
  • Video conferencing WebMeeting for up to 10 subscribers is now FREE.


An important innovation of version 12.5 was the ability to organize free calls from site pages for site visitors. Calls are received directly and immediately transferred to the manager in a particular direction or product. This service can partially replace the free number 8-800. The call is made from a specific page of the site directly to your IP PBX.

A call from the site can be initiated as a normal voice call, after which, if desired, you can add a video.
The user must have a browser with WebRTC support. Also in the plans is a free add-on plugin for CMS WordPress.
