Published webinars on Visual Studio 11, about innovations in ASP.NET and MVC4

I am pleased to announce that the video webinars that were held during the month were published on TechDays website.

The future of ASP.NET and Visual Studio 11 for web developers

http://www.techdays.ru/videos/4196.html

As part of the webinar, I will introduce you to innovations in the development environment and innovations in frameworks designed to improve the efficiency of the programmer. A new handy Page Inspector tool that integrates into Visual Studio 11 will be presented to you. to simplify development and significantly improve the efficiency of using the framework. We will look at the new ASP.NET MVC 4 features and the recipe engine that was introduced with MVC 4 and available in Visual Studio.

Introduction to new APIs available in ASP.NET MVC 4

http://www.techdays.ru/videos/4203.html

As part of the webinar, you will learn about innovations in ASP.NET MVC 4, designed to improve the efficiency of the programmer. Your attention will be presented to the new ASP.NET Web API, new opportunities for creating mobile views of sites, a new API for automatically compressing and packaging JavaScript and CSS files to increase the performance of web applications. In addition, some new features of Visual Studio 11 of the new version will be considered.

For each report, downloadable video files and a presentation file are available by reference.

In total, these webinars were attended by about 300 people. Now everyone who missed the online broadcast has the opportunity to watch the reports in the recording. I hope they will help you learn a lot.
