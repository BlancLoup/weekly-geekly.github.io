Jabber on Mail.Ru Agent for Symbian

Mail.ru announced the release of a new version of Mobile Agent ( for Java (3.5) and for Symbian (1.51)). In the Java version, support for ICQ is implemented, and the number of accounts with which simultaneous work is possible is not limited, that is, you can sit online simultaneously from several ICQ numbers.

In the version for Symbian, ICQ support was already implemented earlier, but now Jabber has appeared here. An advanced search for people on the MRA network also appeared (not only by e-mail, but also by age, address, gender, etc.). The program interface has been significantly improved - the user is now offered several themes, including themes embedded in the Symbian OS.

In addition, the branded map service from Mail.ru with maps of Moscow and St. Petersburg is now integrated into mobile messengers. Also available is the functionality to determine its location at the moment - both using GPS and LBS (data on the location of mobile network towers) for those devices that do not have a GPS module. The user can send the coordinates of his location or any other desired point on the map to the interlocutors in the Mail.Ru Agent.
