Alpha Release Flex Builder Linux Today

I just wrote about Flash 10 and the Adobe MAX Chicago-Sneak Peeks conference where Flex was demonstrated under Linux .



And the sky trembled ... The Flex Builder Linux Alpha release has already been posted on Adobe Labs . This is the first release for Linux and it includes many features of Flex Builder 3. For example, such as project creation, code highlighting, compilation, Ajax Bridge, debugging ... Naturally, not all Flex Builder 3 functionality is supported under Linux, but nonetheless. The code name Flex 3 paid off - Moxie, i.e. cheeky (though not so smooth with Flex 3, as it turned out ). This was expected, but still unexpected. Impudent, impudent, you won't say anything ...

To start working with alpha Flex Builder Linux, do the following:
1. Download Flex Builder Linux alpha
2. Run the installer (chmod + x) or use sh flexbuilder_linux_install_a1_100207.bin
3. Be sure to read the release notes.
4. Well, run Eclipse :)
