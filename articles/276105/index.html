SEO-tools - an extension for Opera (webmaster and optimizer tools)

I did it for myself, so long not to look for links to various online tools for comprehensive site analysis. As a result, it turned out beautifully and compactly, as a result - the extension " SEO-tools " appeared, today it can be downloaded in the library of Opera browser extensions (compatible with Yandex Browser ).

SEO-tools is a browser extension that provides quick access to various online SEO tools. It provides a quick transition to online tools in relation to an open site or site page. Focuses on the Russian-speaking segment of webmasters and optimizers.

In the extension, there is no usual parsing of the site puzomerok (as is customary in extensions like ‚ÄúRDS bar‚Äù), its main functionality is aimed at quick redirecting to pages with such analysis. Since there are no background operations or hidden functions, the extension in no way interferes with the operation of your browser, and its functionality is opened only by a click on the extension itself.



The current version of SEO-tools (version 1.1) includes:

  • definition of ip and country site
  • hosting provider definition
  • reputation WOT and SiteAdvisor
  • direct links to get puzomerok site
  • identification of used resources and technologies on the site
  • determining keyword density on a page
  • checking the uniqueness of the text on the current page
  • spell checker on current page
  • relevant sites
  • history of virus activity and site checking online antivirus
  • load speed test
  • adaptability test
  • view HTTP headers
  • indexing of text and images in popular search engines
  • retrieve page cache from search or internet archives
  • counting share in social networks
  • approximate resource attendance
  • site popularity in blogs, forums, Answers @ Mail, Twitter,
  • validation of HTML, CSS and links ...


The extension works in Opera browsers, starting from version 15 above (in Windows). At the request of the workers, it is quite possible to expand the functionality of the application and make extensions, including for other browsers.

PS "SEO-tools" can now be installed for the Chrome browser .
