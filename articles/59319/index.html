New version of LiveStreet 0.3



Since the release of version 0.2, more than six months have passed and today the author of the engine, Maxim Mzhelsky ( ort ), announced the release of LiveStreet 0.3. The good news is on.

New opportunities:
- Comments are published without reloading the page using Ajax, the ability to navigate through them is provided, the discussion branches can now be collapsed / expanded;
- To upload and scroll through new comments, a control element is developed that is familiar to all users of the Habr;
- Full-text search on the site using Sphinx, the author of the kruft module;
- Optional editor TinyMCE;
- Password recovery via email;
- Delete topics for site administrators and administrators of thematic blogs;
- Protection from duplicate topics;
- Ability to screen links from search engines;
- Time limit for voting for topics and comments;
- New default template, author - Alexey Popov ;
- Additional template with minimalist design, the author - deniart
- Provided engine extensibility by third-party modules by supporting hooks, importing configs, etc .;
- Support for multilanguage, all the text in a separate language file;
- Fixed a significant number of bugs, refactored php code and rewritten all javascript.

Download the new version here . The engine requires PHP version 5 or higher and MySQL with UTF-8 and InnoDB support. Sphinx must be installed on the server in order for the search to work.
