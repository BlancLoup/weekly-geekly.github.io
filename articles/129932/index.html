QR code for lost keys

Have you ever lost a keychain in a bar or on the street? Agree that in this case it is very disappointing, because the finder of the keys also wants to return them, just does not know to whom. In large stores, the entire shelves of administrators are crammed with bundles of lost keys, which is not clear where to go, and no one comes for them.

In this case, a very simple idea comes in handy: type the QR code into the keyring with keys, which is convenient for sending an SMS to the key holder.



Keychain not only carries useful information, but also looks cute.



The QR code itself contains the following command: 

sms:5555555555?body=I%20found%20your%20keys!

Generate code with this command can be in any of the generators .

Apply the code on the plastic can be in different ways: with a soldering iron, paint or laser. On the other hand, the keychain is logical to print your phone in block letters - suddenly a person does not have a smartphone with a camera. Then the keychain is covered with aesthetic varnish (transparent nail polish can be used).
