GNU / Linux Olympiad for future admins

On the website of IT Planet, a Linux Administration contest has been announced, which will begin in the fall. This is a kind of IT Olympiad for students from Russian regions who dream of finding a job in some large company. Qualifying rounds of the competition will be held in Rostov-on-Don, Krasnodar, Samara, Yekaterinburg, Tyumen, Tomsk, Khabarovsk, and the winners will receive practical training places in the partner companies of the Olympiad.

In the first round from October 21 to October 31, 2009, registered participants of this competition must pass an online test: 30 questions on the knowledge of the Linux platform. Each participant is given one attempt and 45 minutes of time.

Topics of the first round
  • GNU / Linux history and legal issues
  • GNU / Linux command line interface
  • File systems
  • IP network and its configuration
  • GNU / Linux Security Tiers
  • Linux kernel
  • Basic services and their use
  • XWindow System GUI and its configuration

Students can now see sample assignments on the official website. By the way, the linux.org.ru forum has already expressed outrage that the test tasks are posted in the RAR archive, and inside - a document in Word.Document.8 format. However, the organizers quickly corrected this absurdity and repacked it in a standard gzip.

Another odd moment is that Microsoft is among the organizers of the Olympiad. It looks very strange, but the explanation here is simple: MS is involved in the next stage of the IT Olympiad, which is dedicated to Windows Server 2008.
