In VMware, unhappy with new licensing policy MS

VMware has published an open letter accusing Microsoft of restricting the freedom of users to choose virtual software.

Using its leading position in the field of operating systems and some other applications, noted in an open letter, Microsoft actually forces users to use only those virtual machines that are developed by the corporation.

License terms ( .doc ) prohibit users from using virtual hard disks (VHD) created by third-party software when working with software from Microsoft, and the VHD proprietary format is closed. The tightening of licensing conditions has also affected operating systems: for example, low-cost versions of Windows Vista (Home Basic and Home Premium) are not allowed to run on virtual machines.

Earlier last week, it was announced that MS Virtual PC 2007 is available for free download .
