TechEd Europe 2013: The Future Comes Today

Preliminary versions of Windows Server 2012 R2, System Center 2012 R2 and SQL Server 2014 are now available. Together with Windows Azure, these products are key elements of the concept and integrated platform for Microsoft Cloud OS applications.

Microsoft has introduced preliminary versions of the new products: the Windows Server 2012 R2 server operating system, the System Center 2012 R2 management tools, and the SQL Server 2014 information management platform. This was announced by Brad Anderson, vice president of Server and Tools, Microsoft, speaking to the participants of the annual conference TechEd Europe.

All this makes it possible to constantly improve the productivity of users, regardless of which device they work with. Today, only Microsoft is helping to create a truly dynamic information environment ‚Äúfrom client to cloud‚Äù, ‚Äùsaid Brad Anderson, vice president, Server and Tools, Microsoft.

System Center 2012 R2

System Center 2012 R2 provides comprehensive management of Microsoft Cloud OS. As a continuation of the development and improvement of System Center 2012 technologies, the new Microsoft portfolio of infrastructure management products provides full management of all the features of Windows Server 2012 R2. System Center 2012 R2 is a key tool for managing hybrid infrastructures that provides, for example, a live migration of workloads from the local infrastructure to Windows Azure Virtual Machines and back.

SQL Server 2014

Like other Microsoft Cloud OS products, the next version of the Microsoft SQL 2014 data and application management platform was created in full compliance with the Cloud in Focus strategy. Along with the progressive improvements in business intelligence, SQL 2014 opens up new opportunities for managing mission-critical business applications and more in-depth analysis of information and data, including Big Data (Big Data).
