Windows Azure SDK 1.4 update released



Officially announced the release of new versions of the Windows Azure SDK 1.4 development tools and the Windows Azure Management Portal cloud management services portal.

The new version of Windows Azure SDK 1.4 includes new features for working with Windows Azure Connect (networking between instances in the cloud) and Windows Azure Content Delivery Network (CDN content delivery network):
  • Windows Azure Connect:
    • support for multiple administrative entries in the admin UI;
    • updated Client UI with improvements in notifications and diagnostics;
    • the ability to install Windows Azure Connect on the localization of Windows other than English;
  • Windows Azure CDN:
    • WindowsAzure CDN for Hosted Services: developers can use web and VM roles as sources of objects distributed via CDNs. Thus, static content sites will be automatically cached at different sites around the world: in the USA, Europe, Asia, Australia and South America for faster access and reduction of network delays when a user accesses the content;
    • added support and customization of content delivery from CDN via secure channels via HTTPS.
  • Windows Azure Management Portal has been updated, now the user interface is more responsive.
