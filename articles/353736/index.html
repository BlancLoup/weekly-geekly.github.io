Releases


BDR 3.0
The 2ndQuadrant just announced the release of BDR 3.0. Details are not even on the site. Only in group on Facebook.

pgAdmin 4 version 3.0
The developers (pgAdmin Development Team) claim that there are more than 100 innovations and fixes in the release .
Among the innovations:
  • you can sort the data directly in edit mode
  • pg_service.conf file support
  • Greenplum support

Distributions for Windows and macOS are available, as well as in the form of Python Wheel, Docker Container and in source codes. Download here . After a few days, both the RPM and DEB packages should appear.

Pgpool-II 3.7.3
About this release can be found here . Source codes and RPMs can be downloaded here .
The PostgreSQL pgpool tutorial has just appeared . h. I blog at Severalnines. In the 2nd part will be including fault-tolerant clusters.

pgmetrics 1.2
This is a new version of the command-line application that allows you to extract and visualize PostgreSQL metrics.
