Dynamic programming is an approach to problem solving, in which there is a division into subtasks that are ‚Äúsimpler‚Äù in comparison with the original one. The word "dynamic" is close in meaning to "inductive": it is assumed that the answer is known for some value k and I want to find the answer for k+1 .
