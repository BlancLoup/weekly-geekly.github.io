Firefox 14 for Android is much better



When Mozilla last showed its Firefox for the mobile OS Android, it was not a particularly good browser. There were many flaws, including both the interface and the functionality of the browser. This time, it seems, everything is different. In any case, the developers claim that the browser has become much more productive and user-friendly, so to speak.

The last statement seems to be true. In any case, after clicking on the browser icon, it loads immediately. In addition, individual pages open very quickly. At the same time scrolling of pages with a large amount of content goes smoothly, much more smoothly than in previous versions.

According to users who have already tried this browser, it surpasses the built-in browser even if they are not using the newest device (I tried Samsung Galaxy Player 3.6 with Android Gingerbread). So even the owners of not the most powerful gadgets can work normally with the new version of Firefox 14.



And yes, it's worth noting that Firefox 14 supports Flash, despite the fact that Mozilla has recently focused on supporting open web standards like HTML5. Until now, Flash is contained on a huge number of pages, so the inclusion of support for this standard is a plus for Firefox 14. Browser developers reported that they had put a lot of effort into allowing the browser to work with normal performance on all common Android OS versions.

And finally, the good news for all users of the usual versions of Firefox - Firefox Sync is included in support of the mobile version. So now there is no problem with synchronization of passwords / history of surfing / open tabs.

The interface of the mobile version of the browser has been significantly reworked, for example, the address bar is now clearly marked.
