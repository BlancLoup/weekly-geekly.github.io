Overview of modern webOS

Nowadays, there is an OS for every taste and there are no problems with distribution. But sports interest requires its own, forcing us to try something new.

In this review, I'm going to reveal all the advantages of webOS.

Careful traffic!



So, the advantages of browser OS:

+ You have enough space for files.
+ Anonymous storage
+ Ability to use the OS with downloaded files wherever there is Internet
+ Ability to upload backups
+ You can upload movies, music and watch / listen at work, the video is played by the Flash player right in the browser

But there are also disadvantages:

- You can‚Äôt take her with you to nature
- Functionality is currently limited to JS and Flash
- A fairly fast connection is required, especially if Flash-based WebOS
- Demanding of RAM
- Demanding on the processor

So, the pros and cons equally, if you have not yet stopped, proceed to the review.

iCloud



Website: icloud.com/ru

Developed since 2001, built on the basis of AJAX and supports browsers IE 6.7 and 8 versions, Firefox 3+. Google Chrome and Opera are not supported.
A nice looking theme, the possibility of changing themes. In Firefox 3.5, it eats ~ 250MB of RAM.
There is an ICQ client, a very useful feature, for example, at my work the ICQ ports are closed.
You can share access to individual folders \ files, allocated space for files - 3GB, which is enough for a couple of movies or 500 songs in mp3.



Video player for watching YouTube with playlist support



Mail client



G.ho.st



Website: g.ho.st

It has an unusual domain name. Built on Flash, supports all modern browsers.
Provides 15GB for data storage and 10GB for mail. The possibility of expanding space for files by inviting friends (1 invited friend = 5GB of space)
There is also an IM client that supports Yahoo! Messenger, MSN, AOL, Google Talk.
There is a client for communication on Twitter and Facebook, the ability to upload files to Flickr directly from under webOS.
There is an Excel spreadsheet editor and Word-style text editor, the ability to install widgets on the desktop and theme shifts \ wallpapering, sending E-mail,
but sends via your browser using your IP.
Interface nadpominaet Windows Vista, transparent taskbar, search on the desktop. Ability to select the interface language. The last state of the desktop is restored after restarting the system.

Windows4all



Website: www.windows4all.com

The project is interesting because it uses Microsoft Silverlight.
The OS is made to show the capabilities of Silverlight, you cannot create a personal account for storing data, there are games - Chess, Spider, Solitaire. There is also a little-featured paint editor, an RSS reader and a text editor. You can watch videos with help
Media Player. The theme is almost identical to Windows Vista, but without transparency.
An interesting feature is the program Virtul PC, it allows you to run various webOS from under you, I managed to go to my desktop G.ho.st



A web browser running from under your browser using your IP, an idea with a fast proxy would be more justified.

Webtop uCoz



Website: unet.com

It cannot be called a full-fledged WebOS, such as G.ho.st and iCloud, since the main purpose of this service is to manage sites registered on uCoz.
Made on AJAX, works smartly, does not require a lot of RAM. The style is similar to Windows Vista, there is a choice of 2 themes and 22 wallpaper'ov offered.
Minus, which immediately catches the eye - a pale shade of fonts, merges with the background.

eyeOS



Website: eyeos.info

Developed since 2005, the first version was published in 2007. The only WebOS reviewed that supports Drag & drop.
Received an award in 2007 - Best Web Operating System, awarded by Softpedia.
The interface resembles KDE, very high-quality animation (for example, when resizing windows, icons are funny inside the window move).
After reviewing, I consider eyeOS to be the most functional Web system.

Wiki-os



Website: www.wiki-os.org/

Powered by Silverlight, the interface works smartly. View windows copied from Windows Vista with the addition of additional buttons. Indication of the active window only on the taskbar, which is not very convenient.
An interesting feature of Wiki-OS is the development by the users themselves, you can write your own applications and edit existing ones.
Despite the simple design, the animation is beautiful. There is no disk space, except for writing programs.
From the useful there is a Sudoku game, a calculator, a picture editor, which can superimpose one picture on another.
At the moment, applications are not written as much as we would like, but the project is still young, let's hope for changes for the better.
