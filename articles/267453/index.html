Webinar: Basics of parallelizing C / C ++ programs using OpenMP


Greetings Habr!

Our FlyElephant team continues to conduct webinars and I want to invite everyone to a webinar on September 28 at 17.00, where we will look at the basics of paralleling C / C ++ programs using OpenMP, learn about the functionality of FlyElephant and learn the principles of working with the platform using examples. Let's talk about the beta testing program and the new functionality that will be available soon.

Separately, as part of the webinar, we are announcing a special open research support program that will allow projects to use the platform for free.

Register for the webinar here .

FlyElephant is a platform for scientific computing and data management. Various computing resources (multicore systems, HPC and GPU clusters), programs, patterns, algorithms, and data that users can combine to quickly and efficiently solve their computing tasks are available in a single place. Users can also collaborate on projects, track changes in the results of calculations and publish the resulting solutions as public or private cloud services.

Now the platform is launched in test mode and allows you to run computational programs written in C / C ++ with OpenMP support on servers with 1 to 32 cores and RAM up to 448 GB, as well as work with data in the storage, view and download the calculation results.
