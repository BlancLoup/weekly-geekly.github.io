Flowplayer - Free Web Video Player
FlowPlayer is a flash media player. You can use it in your HTML pages to play video files. "This is your personal YouTube."

For those who want to know 98%:
Support for a variety of media formats
FLV, SWF, MP3, MP4, H.264 video, JPG and PNG
Flexible settings
FlowPlayer has a lot of configurable settings. You can immediately start using it, and configure it as you go.
Progressive download
The simplest and fastest way to show your users a video is selected. It's like on YouTube.
Streaming video
FlowPlayer supports streaming servers such as Flash Media Server, Wowza and Red5
List of episodes
Allow your user to immediately go to a specific episode / video fragment.
Playlist
You can play up to several videos at once, as well as several at the same time, if there is a media server.
Script API Writing
Control the player using javascript.
Full screen mode
Actually full screen without full screen mode of your browser. And if you want, then with the help of the script you can make a full-screen window
