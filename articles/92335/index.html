Ubuntu "Lucid Lynx" 10.04 LTS



The release of Ubuntu "Lucid Lynx" ("clear lynx") 10.04 LTS of the popular Linux distribution has taken place!


Major changes:
  • kernel 2.6.32,
  • complete removal of HAL, which allows you to significantly reduce the boot time / shutdown and exit from sleep mode,
  • GCC version 4.4.3,
  • Python version 2.6.5,
  • NVidia's default Nouveau driver
  • GNOME 2.30;
  • Removed GIMP
  • general improvements in Ubuntu One,
  • integration of the Ubuntu One Music Store,
  • Ubuntu Software Center 2.0,
  • New graphic design ( full rebranding of ubuntu ),
  • Gwibber package added - client for social networks
  • added Pitivi video editor,
  • package added Simple Scan - simple and convenient scanning;
