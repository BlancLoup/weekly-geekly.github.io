Drupal 6.0 released

After almost a year of development, the release of the sixth version of this wonderful CMS has finally come out.
News in the official blog.
Screencast of new features.


Brief description of new features:
Interface innovations:
* Improved installation procedure
* Content creation in multiple languages
* Improved language support

New system modules:
* Modules Actions and triggers added to CMS
* OpenID support
* New interface update modules

Styling:
* Final separation of CSS from code
* New module for developing new styles
* Updated used jQuery to 1.2.3

Security:
* Alerts for system updates
* Check password complexity during registration
* Improved user privileges and access

Optimization:
* Optimized code
* Scalability options
* Better file handling
* Extended logging
* Scripting from the command line

Miscellanea:
* Improved voting module
* New features in the forum module
* Notifications about new events via e-mail
* Personalized date and time output
* Added ability to style custom signatures.
* Improved visualization of anonymous comments
