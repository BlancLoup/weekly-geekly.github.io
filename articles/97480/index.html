Stability, fixes for Unix and even more: approaching the 10.60 Final

We had a few more fixes for testing at the weekend. In this build there is a huge bunch of fixes for Unix alone, but you will notice that there are cross-platform fixes too. Including fixes for offline applications, geolocation, widgets and more.

Although there is still a lot of work, we began to approach the final release of Opera 10.60 for all platforms, including Linux and FreeBSD. So wait for new fixes, and, of course, we will follow the new regressions and serious problems.

Tell us about the program crashes and let us know if you find any new serious errors.

Warning: This is an unstable version. It is intended for testing and may contain bugs and shortcomings.
