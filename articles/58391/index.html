On-line services that create html containing the highlighted code, based on the original code

This can be useful for people involved in the generation of content of resources and those who make up this content or simply add it using a certain resource interface.
Sometimes when filling a project with content, you need to list a script or program on a page. And, of course, a desire arises that it would be properly formatted, highlighted in accordance with the syntax of the language, and even the lines were numbered. Next, I list several resources that online form the html we need based on the original code.
