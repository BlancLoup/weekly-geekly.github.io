Open source Symantec pcAnywhere sources

This morning, a distribution appeared on The Pirate Bay under the heading "Symantec pcAnywhere Leaked Source Code". The 1.27 GB archive probably contains the source code for the Symantec pcAnywhere program.



The appearance of the file was preceded by a correspondence between hackers and Symantec, in which Symantec offered them $ 50,000, if they did not upload the files and officially announced that their previous burglary message was a lie. Correspondence is now also posted on pastebin .

Two weeks ago, a Symantec spokesman acknowledged the hacking of a corporate network in 2006, which stole the source codes of several key products: Norton Antivirus Corporate Edition, Norton Internet Security, Norton SystemWorks (Norton Utilities and Norton GoBack) and pcAnywhere.
