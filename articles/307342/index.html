Kerio Control 9.1 opens up new next-generation firewall (NGFW) capabilities for SMB companies



Dear readers, we decided to correct the injustice towards our favorite product Kerio Control, the release of the major version of which was unfairly deprived of proper attention in our blog. I hope that the little that you can read below will help you learn more about the innovations that have been added during this release.

Welcome under the cat


The new release of the network security solution introduces the next generation firewall features (NGFW), simplifying deployment and improving the monitoring and notification system.

On June 14, 2016, Kerio Technologies released Kerio Control 9.1, complementing its comprehensive network security solution for small and medium-sized companies with next-generation firewall (NGFW) capabilities, including application control and secure content filtering. These functions have greatly simplified the management of network traffic.

‚ÄúWe help small and medium-sized businesses increase productivity and safety in the workplace,‚Äù said Heather Paunet, vice president of products. "Our new application control capabilities provide a more efficient overview of network performance, making it easier to limit the use of time-consuming applications, while providing access to the applications and tools needed by the enterprise."

The latest release also simplifies the upgrade and deployment of software products through automatic update features of the software version of the firewall. When a Kerio Control device is included in the network, it installs an updated version of the control program on its own and immediately and begins to protect the network, users and data.

Kerio Control 9.1 also offers the MyKerio app for iPhone and Apple Watch, which provides real-time monitoring and notifications for all connected devices. When a state changes, a notification is immediately given, which allows you to quickly fix any problems.
