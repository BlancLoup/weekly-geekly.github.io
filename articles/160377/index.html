Firefox 18 beta



Firefox 18 beta (Windows, Mac, Linux) is available for download.

Major updates:

- IonMonkey . New JavaScript JIT compiler, allowing the browser to more efficiently handle JavaScript, which will increase the speed of web applications, games and other things.
- Retina support.
- Disable unsafe content. Now, when viewing https pages containing unsafe content, it can be turned off (the function is enabled via about: config).
- Support W3C Touch.
