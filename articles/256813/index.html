Vivaldi TP3 - we do the browser together



Hello!

Today we released the third preview version of the Vivaldi browser.

A month and a half has passed since the release of the previous version of Vivaldi TP2. All this time, we constantly received suggestions and messages about errors and shortcomings from you, our Sopranos group also worked actively - voluntary third-party testers who are constantly in direct communication with the developers and actively help us improve the browser. At the same time, we carefully read your comments on blogs and forums to be aware of your browser requirements. In general, if you didn‚Äôt even notice this, we worked on the new version of Vivaldi together with you and followed your wishes.

And today's version includes everything that we managed to do during these six weeks. What exactly? Let's look together at the list:

  • Added grouping of tabs, as well as the possibility of placing them for viewing on one page vertically, horizontally or on the grid;
  • Improved Notes - both in terms of appearance and functionality. Now you can add screenshots of pages and even files;
  • Expanded support for mouse gestures, now supporting and working with links;
  • Added new shortcut keys and improved editing function;
  • Added display of unread pages for individual tabs, and for placed in groups;
  • Added option to support the native browser window, especially in demand by users of various window managers in Linux;
  • Added browser auto-update feature for Windows version;
  • The number of supported interface languages ‚Äã‚Äãhas been increased to 40;
  • Now you can import data from other browsers;
  • Added functions to control the display of images and plug-ins only on request;
  • Added loading indicator for background pages;
  • Improved work functions "Return" and "Transition";
  • Improved spatial navigation;
  • Improved page effects;
  • Added more options to the file download dialog. There are options "Save" and "Save As" in addition to the option "Open";
  • The possibilities of managing the bookmarks panel have been expanded - now you can make it much more capacious by turning on the display of only page icons;
  • Added an informant about working in full screen mode;
  • Improved browser speed and performance;
  • Fixed a large number of errors.

Here is such a report on the work done in a few weeks. Of course, this is not all that we have been working on all this time - soon we will add some new interesting features to the browser that we hope you will like. But now it will not be a Technical Preview, but a full first beta version. And this means that soon we will be able to offer you the first stable version of the browser - the long-awaited both for you and for us.
