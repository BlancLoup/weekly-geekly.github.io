Oculus Rift ( wiki ) is an affordable, innovative VR helmet, the main features of which are:
  • reasonable price ($ 300)
  • High-speed 3D accelerometer for tracking head movements
  • large viewing angle: 90 degrees horizontally, which is two times more than the competition
  • large resolution: 960x1080 for each eye (640x800 in the version for developers)

It all started with the hobby of Polmer Lackey (Palmer Luckey) virtual reality devices, self-made prototypes with screens from mobile phones and the Kickstarter campaign , which collected $ 2.437.429 of the $ 250,000 planned.
Now the project has attracted $ 16M and is supported by Valve, Epic games, Unity Tech., And John Carmack from id Software is Oculus VR CTO.
Under Oculus Rift officially ported Half-Life 2, TF2, Doom 3, Unity Engine, Unreal Engine.
