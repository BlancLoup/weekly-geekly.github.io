Ubuntu netbook remix for two times

If you want to look at the ubuntu netbook remix interface, but do not want to install a special system assembly, then just install a couple of packages.


So, go to synaptic and add a new repository:

For Ubuntu 8.10
deb ppa.launchpad.net/netbook-remix-team/ubuntu intrepid main

For Ubuntu 8.04
deb ppa.launchpad.net/netbook-remix-team/ubuntu hardy main

Next, install all the packages from the newly added repository and restart the X - ctrl + alt + backspace.
