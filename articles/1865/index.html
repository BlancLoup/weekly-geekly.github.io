Google launches source code search

New service from Google is designed exclusively for programmers. The search engine for Google Code Search source codes, the official launch of which took place yesterday, should be the best search engine on the Internet of this kind. Google programmers initially did a search engine for themselves, but then decided to open it for public use.

From the usual search on the web source search differs by the support of specific operators. For example, the regexp operator is used to search for regular expressions. In combination with the file:, package:, lang: and license: operators, it is possible to search only in files and packages that contain the specified regular expression, and also to take into account when searching a programming language and license, under which the source code is published.

Source code search results are available as a GData / XML stream , so they can be embedded on any site. Advertising in search results is missing.

At the moment, the search is carried out in Subversion and CVS databases. You can add your own sources to the Google Code Search search index via a special form .
