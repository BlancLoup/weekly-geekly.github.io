Firefox developers have published a roadmap for 2012

Translated the next six months, perhaps someone will be interested to read. As far as I understand, in the nightly assemblies many things already work.

First quarter

Compatibility Add-ons : our experience shows that most add-ons do not break in new versions of Firefox. We decided that we would allow addons to be loaded, in which compatibility with the new version of the browser is not explicitly stated.

Migrating from Google Chrome : Firefox already supports migration (export of user data) from other popular browsers. It's time to provide similar functionality for Google Chrome users.

Add-on sync : Firefox Sync technology makes it easy to switch from one computer (or device) to another. In addition to synchronizing passwords, bookmarks, and history between browser installations, users will be able to synchronize addons as well.
