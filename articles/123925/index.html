Midnight Commander 4.8.0 File Manager Release

Months of development have passed and we are pleased to introduce the new version of the console file manager Midnight Commander 4.8.0-pre1


Of the most interesting features in this version, I would highlight:
  • Relocation of settings from $ HOME / .mc to XDG_CONFIG_HOME directories in accordance with FDO Standard .
  • Recycling VFS (Virtual FS). Now this is vfsprefix1: // vfsdata / vfsdata2 instead of vfsdata / vfsdata2 # vfsprefix
  • Support for 256-color terminals (as an example, you can try the skin Xoria256)
  • processing the macro system in the editor (works the same as in FAR)
  • in the built-in editor, the possibility has been added that, on an arbitrary combination of keys, hang up a call for external utilities (for example, a compiler, etc.)
  • appeared function REDO in the built-in editor
  • there was an event system

There are a lot of changes, the full list can be seen here www.midnight-commander.org/wiki/NEWS-4.8.0-pre1
