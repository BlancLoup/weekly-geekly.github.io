Another widget builder: 2GIS mini on your site in 5 minutes

API 2GIS is famous for providing access to functions and data of the directory and 2GIS maps, which anyone can use for free in their project. Moreover, we recall that this data includes information on more than 1.6 million organizations in more than 180 cities. But far from all are able to work with API, and these not all may not want to spend their time learning this craft. And also, many may simply dislike writing code .

The new widget designer is designed for them, which allows you to easily and easily implement the basic features of 2GIS Online on any site. Therefore, we called our widget "2GIS mini".



We created 2GIS mini in order to attract new partners to our API, because such opportunities are excellent for all sorts of Internet projects. Just some of the examples:

  • The site of the new building will be able to insert a map on which the entire infrastructure around the house will be shown.
  • On the conference website you can put a map with details of the nearest hotels, cafes and restaurants.
  • An Internet project dedicated to culture can place on the site a map with a directory of theaters, cinemas and other cultural institutions.
  • And the resource where the parents of Izhevsk communicate (for example) will be the best place for a section with detailed information on all schools, kindergartens and toy stores.

So let's say I don't want to write code, I want widgets. What to do? Just watch the video below and go to the constructor to try. And if you have any questions related to API 2GIS, write to us - we will be happy to answer.
