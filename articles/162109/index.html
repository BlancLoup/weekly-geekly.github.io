MegaCloud - alternative to Dropbox

On the Internet, another business player has appeared, providing backup and data synchronization services in the cloud, with the outstanding name MegaCloud .

A new player, respectively, "buns" for ordinary users, it provides quite a lot: 8 GB of cloud storage for synchronization + 8 GB of cloud storage for backup. There is also a bonus program that allows you to get an additional 9.5 GB of space in the cloud for friends (19 GB if you are a student) and also on trifles for other services.



Naturally, there are paid accounts, but not cheap: 100 GB for 10 euros / month, 200 GB for 20 euros / month.



The service has applications for Android, iOS, Windows and Mac OS, very similar to Dropbox applications. For Windows, two applications are installed at once: a data synchronization client and a backup and backup client. Applications work without any complaints.

Client for data backup is simple and convenient. Allows you to backup data from both local and network drives. The download speed to the cloud is small, about 100 KB / s.







The sync client is very similar to the Dropbox client.





Files in the cloud floods pretty quickly (first speed 200 KB / s, but quickly grows to 500 KB / s).



Through the web interface, there is support for drag-n-drop files, sharing of both folders and files (both by links and for specific accounts), integration with Facebook and Twitter services (via web applications).
