Announcements on PDC: Azure - a new platform "in the cloud", Oslo - a new approach to application development

Steve Martin has already managed to publish a detailed post in a message about what was announced at the opening of the Microsoft PDC conference. And behind him a bit of detail told and the chief of the Developer Devision Somasegar.

In short, the Azure Services Platform is a set of services for hosting applications that allow you to scale an application with a few mouse clicks. Azure integrates a platform for publishing applications (the application is hosted in the Microsoft data center, while the application itself and operating systems are updated on servers without application downtime), Windows Live, Microsoft .NET platform, SQL Services for data storage, SharePoint Services and Dynamics CRM Services.

At the same time, developing an application to host on Azure Services is no different from developing a standard ASP.NET web application in Visual Studio. For more information, see the Azure web site section for web developers .
