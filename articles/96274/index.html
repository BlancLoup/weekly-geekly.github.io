Yes / No / Cancel - Another Look

We did the UI from scratch and, therefore, were able to give free rein to the imagination, without limiting ourselves to the existing approaches, code base and user habits. Logically, our program should have a standard window of settings from the ‚ÄúYes / No / Cancel‚Äù series, but for the task, I wanted to make it beautiful, convenient and understandable.

Firstly, we immediately abandoned the windows from the Yes / Ok series, since the settings are either applied or canceled - there is no third option.
