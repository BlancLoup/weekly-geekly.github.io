Cardapio is a great replacement for the Ubuntu main menu.

Compact, convenient and versatile menu! I tried several alternatives, their speed of work leaves much to be desired. This is not loaded with all sorts of themes, transparencies and other beauties. It is just convenient. I want to talk about Cardapio - an alternative main menu for Ubuntu.

Here's what it looks like:



Of the main advantages worth noting:
  • Integration with Google search allows you to search not only applications, but also results on the Internet.
  • The ability to attach the most needed applications at the top of the list (right click on the menu item - ‚ÄúPin this item‚Äù)
  • The ability of the menu applet to be located on any panel (vertical, horizontal) and automatically resize the icon to fit the panel
  • The ability to easily change the name of the menu and its icon
  • Ability to resize the menu as a simple window with the same size
  • Support for add-ons (I haven‚Äôt yet found anything other than those offered in the kit)




Installation



Cardapio has an official PPA:

sudo add-apt-repository ppa:tvst-hotmail/cardapio
sudo apt-get update && sudo apt-get install python-keybinder cardapio


Once installed, Cardapio will be available in the list of Gnome panel applets. To add: right-click on the panel -> Add to panel ... -> select Cardapio from the list -> Add
