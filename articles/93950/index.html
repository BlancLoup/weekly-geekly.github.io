The App Store has fully earned the gift giving functionality.

A few months ago, iTunes had an interesting menu item ‚ÄúGift This Software‚Äù - the ability to give a paid program to other users (i.e., pay for the program for them). But this functionality was in test mode and did not work for most ordinary users.

Yesterday, Apple officially connected this functionality to all users. It is very important that this works in all App Store (but only between users of the same App Store). For example, a user from the Russian App Store (ie with a Russian account, the majority in Russia) can present and accept a gift.



Until that moment, it was possible to get a paid program for free only by the promo code from the developer. But you can use the promotional code only with a US account (valid only in the US App Store).

It is difficult to overestimate this functionality. It is really useful for both ordinary users and developers.

How it works?


Everything is very simple‚Ä¶

1. Open iTunes and select the paid program that you want to give to someone.



2. Fill out the form to send a gift. You can specify several e-mail (mail) addresses separated by commas, but remember that for each recipient you will be charged the cost of the gift program.



3. All that remains to be done is to click on the ‚ÄúBuy Gift‚Äù button.



Your account will be charged for the program. An email will be delivered to the recipient with a link to the free download of the program.

As I wrote at the very beginning, the resulting link with the code can be activated in any App Store (including Russian, that is, with a Russian account, but the sender and recipient must be in the Russian App Store), and not only American, as it happens with promo codes from developers.
