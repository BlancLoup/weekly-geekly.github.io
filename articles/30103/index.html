A couple of useful wordpress plugins

At work, we often write plugins for blogs on the wordpress.org engine, most of them are protected by customers' and our own copyrights, but there are also those that I would like to share with everyone. We put them on Plugins: 1. Google maps in posts
I think that the name speaks for itself, with this plugin you can add a map to your site. There are a lot of similar plug-ins, but for some reason everybody lacks something, so they tried to put everything together.
What can :
  • Creates your own map in a post or a page.
    In the setting of the plug-in, in addition to the API Google Map, you can specify the default location (for example, your office)
    The default location is displayed on each new map.
    For each marker on the map, you can specify your own icon
    Each marker can add a name and description.
    Automatics set a marker at the address you entered
    Set an empty marker in the center of the map
    Set map dimensions, height and width
    Set the initial zoom with which the map will be displayed


    2. Buzzwords (quick help on the highlighted word)
    The plugin will allow you to add words to any post or page, which will display a hint with a link to this post / page. Those. this way you can create a ‚Äúglossary‚Äù, for example of any terms, not only with the name and link, but also a brief description.
    What can :
    • Create a hint word (hereinafter Buzzword) to an existing post / page or as an independent unit
      When creating a standalone buzzword, you can specify any link
      Buzzwords will be highlighted in a post with a link, name and description
      Buzzwords can be added either one at a time or in "bundles" using ";" as a separator
      In the administration, you can edit, delete and add new Buzzwords
      When editing a post / page you immediately see which Buzzwords are tied to it
      In the near future, another plugin gallery will be added, its example can already be seen on the main page of blog-ware.com
