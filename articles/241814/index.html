Jump Start materials published October 7: Upgrading the company's IT infrastructure using Windows Server 2012 R2

Dear Colleagues!

A Jump Start course course is now available on the Microsoft Virtual Academy portal. October 7 : Upgrading the company's IT infrastructure using Windows Server 2012 R2. In this course, you will not only learn about the new interesting features of Windows Server 2012 R2, but also find out why you need to start migrating from Windows Server 2003 and 2003 R2 now. Detailed description under the cut.





July 14, 2015 support for the server operating systems Windows Server 2003 and Windows Server 2003 R2 will end. In many companies using these operating systems, projects are being launched to upgrade the IT infrastructure and migrate to more modern systems. That is why IT professionals will be especially useful to take a look at the technological solutions embedded in the newest OS of the Windows Server family - Windows Server 2012 R2. As part of this course, the component improvements used in almost any organization are considered: Active Directory services, storage construction tools, and automation tools for administrative tasks. In addition, you can get acquainted with identity management solutions in hybrid structures, in particular with the Azure Active Directory Premium service.
The course consists of 4 modules:

1. Modernization of infrastructure. Why Windows Server 2012 R2?


As part of this module, you will learn why you need to upgrade your current infrastructure and transfer it to Windows Server 2012 R2. You will learn about the main advantages of the transition to the latest generation of Microsoft server operating systems, which include: the ability to use cloud services, the adaptation of new technologies to improve the functionality of data warehouses, system and user management tools, and much more.

2. Improvements in Active Directory


Active Directory is one of the key components of a domain network. In this report, you will learn about new features for creating a domain controller and managing an already created domain, which appeared in Windows Server 2012 R2. In addition, you will learn about the cloning and virtualization of domain controllers.

3. Modernization of storage infrastructure


The amount of data users work with is constantly increasing. Using Windows Server 2012 R2 provides new features that can help you reduce the cost of high-performance, scalable network storage. In this report, you will learn about multi-tier storage, new file server scalability, and other storage improvement options.

4. Automate with PowerShell


The Windows PowerShell Integrated Scripting Environment (ISE) can be used to create, run, and debug various commands and scripts. Windows PowerShell ISE allows you to automate and speed up everyday tasks. In this report, you will learn about the innovations in Windows PowerShell ISE, which simplify and speed up the process of writing and executing scripts.
