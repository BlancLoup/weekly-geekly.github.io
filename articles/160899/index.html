Qt 4.8.4 is here!

Hi, Habr!

Today I want to inform habrakommuniti that the Qt development team brought Qt 4.8.4 into release state.

This is another patch release (bugs were fixed, binary compatibility with Qt 4.8.x). A list of the achievements of this release under the cut.

  • 41 bugs fixed by Qt Gui
  • 13 specific files for Mac OS X
  • 8 specific files for Windows
  • 5 specific files for Linux / X11
  • 36 specific files for QNX / Blackberry
  • 22 fixes for tools

As always, more details can be found here .

There is good news for those who were waiting for support for Windows 8 and VS2012. Now Qt works quite well on computers with Windows 8 (in desktop mode) with hardware from Intel. Qt 4.8.4 builds with the new VS2012 compiler, with the exception of WebKit.

You can download the library from Qt Project or, commercial version via Digia Customer Protal . The commercial SDK may already be updated. The team is already working on an open source SDK. But wait for its release should be after the release of Qt 5.

The post in Qt Blogs is here .
