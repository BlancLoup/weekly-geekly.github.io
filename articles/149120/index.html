JetBrains News Digest July 26th to August 5th

After the long-awaited major release of ReSharper 7 was released on July 26th, a number of events occurred, which we cannot but mention at least in the digest format.

So, we list the most important of them in reverse chronological order:
  • On August 3, we learned that dotTrace Performance , a tool for detecting performance bottlenecks in .NET applications, received the Jolt Productivity Award , which is called the Oscar of the software development industry;
  • On August 2, RubyMine 4.5.1 was released with improved support for Sass / SCSS and a number of bug fixes ;
  • On August 2nd, TeamCity 7.1 EAP appeared - build 23892 ;
  • The 1st of August brought a new EAP-build IntelliJ IDEA 12 # 120.305 ;
  • July 31 saw the light dotTrace 5.2 Performance was released with support for .NET 4.5 and Windows 8 ;
  • On July 31, the new EAP-build AppCode 1.6 # 120.293 ;
  • July 30th was marked by the release of the first EAP build of PhpStorm and WebStorm version 5.0 ;
  • On July 26, MPS (an environment for creating and using domain-specific programming languages) received an update to version 2.5.1 .

In conclusion, we invite all ReSharper users to take part in the webinar (in English), which will be held tomorrow, August 7, at 18-00 Moscow time (16-00 CET). You can still manage to register by clicking on this link .
