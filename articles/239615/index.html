Share your success story and win an MSDN annual subscription!



Hello to all!

An MSDN subscription is the most convenient way to access an extensive collection of Microsoft products, such as Visual Studio , Office , Visio, Project, Windows Server, etc. In addition, MSDN subscribers get the benefits of using Microsoft Azure cloud resources for $ 150 per month.

This fall, developers have a great opportunity to get an annual MSDN subscription for free, winning it in the competition!

To do this, you just need to tell your story about creating an application in which Microsoft products were involved. Especially if you wrote for Windows Phone or Windows 8.

What does an MSDN subscription give?
  1. At your disposal is a whole collection of Microsoft products :
    • Visual Studio ;
    • Office ;
    • Visio;
    • Project;
    • Windows Server, etc.
  2. Access to MS Azure cloud platform;
  3. Technical support from Microsoft;
  4. Subscription to Office 365;
  5. Visual Studio Online and more.

To participate in the competition, post your success stories in creating applications or IT products using Microsoft products . The story can be placed in any form, but be sure to mark it with the hashtag #tproger_success_story .
