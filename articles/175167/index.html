Firefox 20 release

Firefox for desktops has become available for download. What's new in this version:


  • Redesigned download manager. Now you can view and track the boot process in the toolbar, and not in a separate window:

Download Manager in Windows 7

Mac OS X Download Manager

  • The new implementation of ‚ÄúPrivate Mode‚Äù provides ‚Äúthe ability to open a separate private window, all tabs in which will work without data being deposited in the history of visits and local caches‚Äù ( Opennet ). Also in the context menu, you can open the link in a private window;
  • Added the ability to close hung plug-ins without hanging the browser itself. When a hung plugin does not respond within 11 seconds , Firefox will offer to restart the plugin, not the browser;
  • Continuing a series of various improvements in Firefox: loading pages , loading files , shutting down ;
  • Support for the getUserMedia API for organizing access to the user's webcam and microphone (a special warning is displayed asking if the user agrees to provide access to the equipment);
  • Continuation of the implementation of new elements presented in the draft ECMAScript 6 standard: added clear () and Math.imul calls ;
  • Introduced " Developer Toolbox " - a unified interface for quick access to tools for developers in a single window, such as a web console, debugger, "Inspector" and style editor;
  • New JavaScript profiling tool;
  • Added support for dimming for canvas;
  • Various improvements for the audio and video tags: support PlaybackRate , support for the logical combination of several Ogg streams ;
  • On Windows 7 and above , support for H.264, AAC and MP3 has been added . The function is experimental, to enable it, go to about: config , find the parameter media.windows-media-foundation.enabled and switch it to true .
    The inclusion of this feature by default is expected in Firefox 21 ;
  • Fixed a bug when pressing the Esc key terminated the WebSocket and XMLHttpRequest;
  • Fixed an issue where the Unity plugin was not displayed in HiDPI mode;
