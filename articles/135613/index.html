YouTube search in Unity



Wonderful people from the Spanish blog "Ataraeo" have collected an interesting add-on for Dash, which will allow you to search for YouTube videos without opening the browser.

There are also filters to narrow searches on popular YouTube categories.

Results can be ranked by relevance, publication date, number of views and overall rating.

Installation


To install in Ubuntu 11.10, you must first add the 'Atareao Lenses' PPA repository to the application sources.

This can be done by opening the terminal and typing the following: 

sudo add-apt-repository ppa:atareao/lenses sudo apt-get update && sudo apt-get install lens-video scope-youtube

For the changes to take effect, you need to end the session and start it again.



It would be great if the video was played in a special application, and not in the browser window, but this is not bad for the first release.
