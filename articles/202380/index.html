Linux Mint 16 "Petra"

One and a half months after the release of Ubuntu 13.10, the release of a new version of the Linux Mint distribution kit, codenamed Petra, took place.



For the past 2 years, this distribution has taken the first place in the popularity rating of distributions according to distrowatch.com, and for good reason, because the distribution is focused on convenient use as a desktop station, in fact, is an alternative to Ubuntu.

The distribution comes in four editions with different working environments: MATE, Cinnamon, Xfce and KDE. As a rule, the release with the last two desktop environments is carried out after 1.5 - 2 months.

Major changes in the core and software:

  • kernel 3.11
  • Cinnamon 2.0
  • MDM 1.4
  • Firefox 24
  • Thunderbird 24
  • gcc 4.8


Cinnamon

Development of Cinnamon 2.0 lasted for 5 months, 856 commits from 28 developers were made. Many bugs have been fixed, and new features and improvements have been added.

- Play sounds when performing common actions such as closing / minimizing windows, switching desktops, etc.
These sounds are flexible, they can be turned off globally or individually, the volume level does not depend on the volume of the system sound.


- Improved user management
Added a new applet for easy user management: exit from a user session, user switching, access to parameters, etc.


- Convenient tiling window arrangement
It can be done by dragging the window to the edge of the screen or by pressing the Super_L buttons and the arrow keys.


- Improved MIME support in Nemo
- System tray supports the display of the status of copying / moving files


- Nemo has become more beautiful and faster, which can not but rejoice

The full list of changes in Cinnamon 2.0 is here .
