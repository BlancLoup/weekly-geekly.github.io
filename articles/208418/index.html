Apple iOS 7.1 Beta 3. Reboot

Apple has released the third beta version for developers, which includes, of course, bug fixes and a small set of interface changes that allow you to feel the apple operating system a little differently.

Keyboard:

The keyboard in the new version has undergone minor changes, a bold font appeared, the design of the shift and delete buttons has changed. The guys from Cupertino also played with colors, more dark and less yellowish-gray tones. In general, the keyboard has become more contrast.


iOS 7 keyboard on the left, iOS 7.1 keyboard on the right

Icons:

Well, everything is simple, the gradient has changed and everything has become darker.
