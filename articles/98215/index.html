PivotViewer for Silverlight Overview

World Cup 2010 dedicated to fans



Not so long ago, I already wrote about the release of the Pivot data browser. The other day its version for Silverlight was released - PivotViewer. In this article I will try to give a small overview of the technology.

PivotViewer is a silverlight implementation of the Pivot browser , presented by Live Labs for a long time. Watch the following video for the browser features:



Pivotviewer


The Pivot browser was written in WPF and the obvious step was the appearance of its silverlight version, with the possibility of embedding it in web pages. Pivot uses SeaDragon technology, which is used in Silverlight DeepZoom and is used to dynamically display huge amounts of visual data without delays.



In order to start working with PivotViewer y yourself on sites it would be nice to get acquainted with examples of use that have already begun to appear.

Examples of using


The most relevant example is the World Cup Pivot , which is information about teams playing in the World Cup 2010. You can study the information in various sections and include a lot of filters, apply various sorting and grouping data. Try it!







No less interesting is the following example - the site tweet pivot . On it you can visualize your followers or those you follow and view this visualization in all possible sections using the power of PivotViewer. For example, you can select all your followers from Ukraine and see them in one list.







Best of all, the site will show the following video:



Another interesting project is a site for searching wedding venues, which introduced PivotViewer to navigate its offers. It turned out very helpful!









The final example will be the on-demand video provider database , Netflix. You can sort and select movies by a variety of parameters, for each is available an annotation with useful information.







I think there are more than enough examples. PivotViewer gives sites the ability to show data in new ways, offering users a completely new experience in working with information.
