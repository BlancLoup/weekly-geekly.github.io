Waves Blockchain Platform Hosted in Microsoft Azure Cloud

Our blockchain platform for issuing cryptographic tokens and conducting crowdfunding campaigns Waves Platform is located in the Microsoft Azure Blockchain as a Service (BaaS) cloud. Integration into BaaS makes Waves available for implementing various blockchain solutions around the world.



Today, Microsoft Azure is one of the largest cloud services. The platform gives customers the ability to develop and execute applications, store data on servers located in distributed data centers, create private blockchains and create smart contracts that are executed automatically when they perform certain actions or conditions.

BaaS is an Azure-based secure environment that includes intelligent services and tools for developing, testing and running applications on the blockchain and allows users to test various blockchain ideas in practice. The platform provides a set of cloud services that developers and IT professionals can use to develop and run applications, as well as manage them through the worldwide network of Microsoft data centers.

From the point of view of classical cloud hierarchy, IaaS-PaaS-SaaS (IaaS - Infrastructure as a Service - provides cloud computing power; PaaS - Platform as a Service - provides a certain level of abstraction, provides a ready-made platform, freeing from the need to directly configure computing resources; SaaS - Software as a Service - provides the consumer with the final service and completely abstracts from the cloud infrastructure and platform) BaaS is PaaS, a platform for developing blockchain projects.

Adding Waves to Azure provides new opportunities for companies and software developers who want, for example, to create their tokens on the blockchain.

According to Konstantin Goldstein, a technical evangelist of Microsoft Russia, the expansion of the pool of platforms in BaaS forms a unique environment for developers and companies who are interested in implementing blockchain projects. In one click, BaaS will help deploy blockchain infrastructure, as well as integrate advanced technologies, such as artificial intelligence, the Internet of things and others, into these projects.

A number of blockchain solutions are already available in Microsoft Azure, including Ethereum, ChainCore, Corda, Nxt, Lisk - and now Waves. This forms a complex ecosystem of Blockchain as a Service for the implementation of various blockchain projects for any need for enterprises of any size.
