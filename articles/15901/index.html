Microsoft launched the final version of Windows Live

A set of online services and applications for Windows Live yesterday officially came out of beta testing status. All users can now access and download all available applications from the online Windows Live panel. Previously, all applications were available from different places.

Windows Live is the brand name for a set of online services and applications that Microsoft hopes will enable it to sell online advertising to compete with Google. Sources close to the company said that although Microsoft does not worry about its profitability in the short term, it is concerned about the possibilities of Google monetizing its services through advertising. Actually, according to the same sources, Microsoft went to the Web to avoid lagging behind Google in the future.

Microsoft divides Windows Live into two groups: online services (for example, Windows Live Hotmail) and applications for Windows (like Windows Live Messenger and Windows Live Photo Gallery). A full list of applications is available on the Windows Live website .

However, not all services came out of the beta stage - the online Windows Live Skydrive file storage and Windows Live Calendar calendar will be released in the final version later.

Next year, Microsoft plans to launch new Windows Live services as well as develop existing ones.
