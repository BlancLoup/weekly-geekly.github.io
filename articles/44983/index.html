Google suggests keywords for SEO

Today, Google has released a very useful tool called the Search-Based Keyword Tool , which allows the site owner to evaluate which keywords most effectively place contextual advertising. The system analyzes the content on the site and makes recommendations.

The main advantage of the novelty is that it analyzes non-existing keywords, and can offer new ones with potentially high ROI.

In addition, it is simply a very interesting tool for analyzing other sites, including competitors' sites. You can see which keywords are most relevant for any website on the Internet. for example, for Lenta.ru, the system recommends [Aeroflot], [investment projects] and [oil price]. For each result, there are links to documents containing these phrases, the number of monthly queries and the level of competition for these keywords, as well as the approximate cost of buying them through AdWords.
