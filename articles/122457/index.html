SkyDrive update and how to connect SkyDrive as a network resource

Recently, the Microsoft SkyDrive free cloud file storage web interface has received an update. This resource still offers 25 gigabytes of free storage; however, during the upgrade, the size of the individual download file has now been increased from 50 megabytes to 100 megabytes.

The web interface now uses a new view to display folders and view an image gallery.



Additionally, the following changes are announced:
  • pinning the site to the taskbar through Internet Explorer 9
  • same interface for all file types
  • new file information panel
  • new photo viewer
  • increased resource speed
  • 100 mb file limit
  • there is no more advertising, the banner is removed.
But the most interesting thing is that now there is a way to connect your SkyDrive storage as a network drive with full support for copying, pasting, deleting, previewing files. While office files are supported for writing to the repository. The remaining files can be downloaded via the web interface. Access to files uploaded through the web interface is available for all types. In order to connect SkyDrive as a network drive, follow these steps:
  1. Go to http://skydrive.live.com/ .
  2. Login with your Windows Live ID.
  3. Copy cid from URL https://skydrive.live.com/?wa=wsignin1.0#cid=1234567890abcdef
  4. Open Windows Explorer.
  5. In the left pane, right-click on Network . Select Map network drive .
  6. Enter as URL: \\ docs.live.net@SSL \ 1234567890abcdef (replace 1234567890abcdef with your identifier)
  7. Select a drive letter and click OK.
  8. You will be prompted to enter your Windows Live ID accounts. Enter them.
  9. Now your network drive is connected!
