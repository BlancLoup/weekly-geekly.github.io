Call Interception at 3CX Phone System

Intercepting (or picking up) a call in the 3CX Phone Phone System allows the user to accept a call addressed to another user. In this article we will describe how to properly configure call pickup in the system, and about the various modes of call pickup.

Call pickup works like this: when a phone is dialed from another user, you dial the dial-code set in the system (by default * 20 * ) and ‚Äúpick up‚Äù the call to yourself. If several phones are ringing at the same time and you want to intercept a call on one of them, dial * 20 * + user extension number. Dialcode is set in the 3CX Management Console section Settings> PBX> Dialcodes> Match Call .



The dialcode * 20 * (or another one you install) is conveniently used in conjunction with the BLF buttons on the IP console of the phone or 3CXPhone. You can program one of the BLF buttons for this dialcode using Custom Speed ‚Äã‚ÄãDial .





Now you just have to click the Call Pickup button and the BLF button of the user's extension number

However, in some phone models, for example, in Snom, it is not necessary to specify the interception dial code. Simply press the BLF button of the extension, which is ringing (usually at the time of a call, it flashes), and the call will be intercepted. On the other hand, this approach does not allow you to configure user restrictions on call interception, which will be discussed below.

Call Intercept Limit


If you have a large enough installation, it is recommended to identify users in groups corresponding to the departments of the organization. After that, it becomes possible to assign users the right to intercept calls only within their group, but not the entire 3CX system. To get started, turn on the Restrict call interception option only within a user group in Settings> PBX> General .



After that, add the user to the appropriate group and assign the rights to Allow to see members of the group , Allow to see the calls of members of the group and Perform operations with active calls .



If a user is a member of several groups, his interception rights will vary depending on the rights in a particular group. Users in the Default group have minimum rights, and managers have maximum rights, i.e. see all available information about the actions of all users of the 3CX system.

Call Pickup with 3CX Switchboard


Another convenient way to intercept calls is to use the 3CX Switchboard. During an incoming call, select it, right-click and select Answer .



In this case, you must have the appropriate rights in the system, as described above.
