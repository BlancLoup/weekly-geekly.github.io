Web of Trust connected Safe Search Yandex (SafeBrowsing API)

Web of Trust (WOT) has announced it has connected to Safe Search using Yandex Safe Browsing API .

Safe Search for Yandex automatically checks sites for the presence of malicious software and other security threats.

Using the Safe Browsing API of Yandex allows you to quickly receive information about the threats found and warn the owners of web sites as well as users of the browser add-on (WOT).

The combination of community assessment and automatic anti-virus scan allows you to more accurately assess the potential danger of a web resource.
