Adobe has updated Flash Player

Adobe released security update APSB15-23 for Flash Player, closing 23 vulnerabilities in it. Most of the closed vulnerabilities can be used by attackers for remote code execution in a web browser. Adobe also added new protection mechanisms against exploiting potential vulnerabilities in its player ( further hardening to mitigation to defend against vector length corruptions ).



We recommend updating your Flash Player. Browsers such as Internet Explorer 11 on Windows 8.1 / Win10 and Google Chrome, as well as MS Edge update their Flash Player versions automatically.
