VoxOx - VoIP service that combines communications

Today I want to introduce you to a new startup from the American VoIP operator TelCentris, based in San Diego.


The main idea of ‚Äã‚Äãcreating a new VoxOx service was to unite practically all the communication tools available today, such as IP telephony, video conferencing, IM, E-mail and social networks. As a result, the number of declared functions of VoxOx is really impressive:
- Incoming and outgoing voice calls
- Telephone calls worldwide at TelCentris operator rates
- Call-back calls via WEB or sms
- Video call support, including multi-user video conferencing
- Text messaging
- Support for IM networks: ICQ, MSN, Jabber, Google Talk, Yahoo, AIM, etc.
- Support for social networks: Facebook, MySpace, LinkedIn, etc.
- Support for E-mail services: Gmail, Yahoo, etc.
- Voice mail
- FAX support
- File transfer

The VoxOx client is currently available for Windows and Mac OS users, but in the future, developers promise to provide versions for Linux and for mobile use.


However, the owners of mobile devices today can use the services of VoxOx, thanks to an interactive web interface.

VoxOx provides access to its services through any mobile browser, WAP access, as well as special access for Apple iphone users.

At the moment, the service VoxOx has the status of beta. Some of the declared functions may be inactive. Errors are also possible when working with a client. As a bonus, there is a special offer, according to which each new user gets $ 1.2 for calls to landline and mobile numbers.

Unlike Skype, VoxOx software is open source and introduces the client API to everyone. Thus, the authors of the program hope to attract the developer community to write a variety of add-ons for VoxOx. Also, VoxOx developers fundamentally use only open standards, which allows to implement almost all the functions of services integrated into the program.

From myself, I note that the idea of ‚Äã‚Äãcombining communications in one service is not new, but VoxOx went further than others, and the project looks quite ambitious. Someone can write VoxOx in the contenders for the title of the next "Skype killer". But for now, the project is clearly damp, and it will be more interesting for review. What will happen next - time will tell.
