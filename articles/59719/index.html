156 run teams

I hope that it is not a secret to anyone that in Windows it is possible to execute Run commands by calling the Start-> Run dialog or using the Win + R hotkeys .
This is convenient for launching applications to which you will either otherwise not get access or get by going through folders with a high nesting level (for example, to install and remove the program you need to go to Start -> Control Panel -> Add or Remove or simply run command appwiz.cpl).
