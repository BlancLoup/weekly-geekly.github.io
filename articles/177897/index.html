Ubuntu 13.04 released

( The news is no longer a novelty , but at the request of commenting on a more detailed post about ubuntu 13.04)

Today was the release of one of the most popular Linux distributions, codenamed Raring Ringtail (impatient to which).
Recently, the official site of ubuntu.com says this.
Images for download the same, available approximately from noon.



Updates:

  • Linux Kernel 3.8.8
  • Upstart 1.8
  • Python 3.3
  • CUPS 1.6.2, cups-filters 1.0.34
  • LibreOffice 4.0
  • Unity 7


Updates in Unity 7:

  • Menu to synchronize data using the Ubuntu One service
  • New completion dialogs
  • New animations, icons and other changes
  • Added the ability to switch open windows on the mouse wheel on the Unity panel
  • Many bugs fixed, performance improved
