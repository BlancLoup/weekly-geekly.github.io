New competitions Imagine Cup - Kinect and Windows 8. Or how to exchange an idea for Kinect for Windows

To the attention of habrasters, habra aspirants and habrastershar students!

Within the Imagine Cup , two new online contests were announced - a contest for the development of Metro applications for Windows 8 , and a Kinect Challenge contest. Read on to learn how you can almost get Kinect for Windows for free to implement your idea, as well as how to learn how to program for Windows 8 and test your knowledge! Well, the prize winners, as always, are waiting for a trip to Australia in the summer of 2012 to the international finals of the competitions.





The Kinect Fun Labs Challenge Competition is a competition of applications that use the advanced Kinect sensor capabilities to track human movement and control a computer from a distance. In the first round (the deadline for submitting the application is before March 6, 2012 ), it is necessary to submit only the idea and plan of the project. The 100 best applications will pass to the second round, and they will be provided with a new Kinect for Windows sensor for development , as well as specialized development software with examples and demos. What is important to note:
  • Having come up with a good idea, you have a very good opportunity to get a Kinect for Windows sensor, worth more than $ 250 (in the US, in Russia it is not yet sold and it is not clear when it will be).
  • Teams that use Kinect in their projects in the Software Design category should also submit a project to the Kinect Challenge.
  • If you think that using Kinect in projects is difficult, then you are mistaken! The sensor itself can track the position of the human skeleton, returning you the coordinates of 20 skeletal mates. In addition, Kinect can work webcam and 3D-scanner, returning a three-dimensional picture of the scene, which he surveys, in the form of a cloud of points. More Kinect development lessons can be viewed at DreamSpark Academy .

For the Windows 8 application competition, you need to develop a modern Metro style application for the new Microsoft operating system. At the first stage (until March 6), you must successfully complete an online test, which will be available during the day on January 31, February 7, 14, 21 and 28 and March 6 . Successfully passed the test pass in the second round, which is dedicated to the actual development.

It's time to study this topic a bit, pass the tests and try to be one of the first in the new promising development direction.
