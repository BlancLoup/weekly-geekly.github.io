ESET begins testing a new version of ESET NOD32 for Microsoft Exchange Server

ESET has started beta testing a new version of the ESET NOD32 for Microsoft Exchange Server solution designed to protect mail servers from viruses, spyware and trojan software, rootkits and spam.



The new version adds the HIPS technology (Host Intrusion Prevention System), which protects against attempts of external influence on the system, and the product self-defense is improved. The program supports Greylisting (gray lists), allowing you to more accurately and thoroughly filter spam messages. Optimized methods for scanning the email database, built-in diagnostic tool ESET SysInspector, which allows you to quickly detect hidden rootkits without having to run a full scan and get all the information about the system status In addition, ESET NOD32 for Microsoft Exchange automatically monitors all processes and provides the administrator with a detailed report on the operation of the mail server.

ESET NOD32 for Microsoft Exchange supports Microsoft Exchange Server 2010 and is compatible with versions of Microsoft Exchange Server 5.5, 2000, 2003 and 2007.

You can download the beta version on the ESET website . However, please note that this is a test version of the product. It does not have full functionality, and we do not recommend installing it on PCs and systems that perform critical tasks.
