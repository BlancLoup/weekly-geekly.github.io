Rocketbook: "cloud" notepad for automatic transfer of handwritten text and drawings to the cloud



The idea of ‚Äã‚Äãautomatically transferring hand-written text to the cloud is not new. However, the implementation of this idea is not always user friendly. Most often, the developers of devices that transfer records from paper to any service (or computer hard disk) use a special sensor installed at the top of the sheet. This sensor monitors what the user writes (and often the pen needs a special one). A similar device "Digital Fountain Pen" is in the assortment of Dadzheta.

But all this kind of gadgets that I have come across so far are local, which is not very convenient in our ‚Äúcloudy‚Äù time. A couple of weeks ago, a campaign was launched on Indiegogo to raise funds for a new system that sends notes and drawings directly to the cloud, without requiring additional tools, just a special notebook and smartphone. This system has already collected 7 times more money than planned. At the time of writing, more than 136 thousand dollars were raised, instead of 20 thousand requested by developers from the community.

It looks like a regular notepad that syncs data with Dropbox, Evernote, Google Docs and other services.



How it works?



Rocketbook is a system from a notebook and a mobile application. These two components work together, allowing users to write and by hand, create hand-drawn diagrams, putting it all online in a couple of tens of seconds.

Notebook





Notebook pages are equipped with special markers, the value of which is set in the mobile application by the user. Each marker corresponds to a specific service to which the owner of the notebook wants to transfer his record.

In order to determine the service to which the notepad page should be saved, you just need to paint over the marker with the FriXion pen (included, also sold in many stores, including 120-180 rubles in Russia, for example, in Comus) .



This notebook is not the thickest, and sooner or later all clean pages (there are 100) will run out. After digitizing the records, the Rocketbook can be put in a microwave (the most usual, only clean so as not to blot the leaves), and turn it on for half a minute. After 30 seconds, all entries will disappear.



application



The program for the mobile device automatically captures the records and pictures of the user, translating them into electronic form, and saving them in predefined services.

Currently, there is software for iOS, but users of Android devices will have to wait a bit (developers will report as soon as the Android application is released).





How much is it?



Pre-order for the RocketBook can be made for $ 25, the kit includes a notebook, pen and mobile application.

Conclusion: the idea of ‚Äã‚Äãthe Rocketbook is really interesting - this is a project that can be very useful for designers, students, businessmen and representatives of all other fields of activity. The price of a notebook is slightly higher than the cost of ordinary high-quality paper notebooks, organizers. But the possibilities are much wider.

However, he does not solve one problem. If you need to translate the written text into a text file format, you will need to use OCR tools. Or use the Dadget's Digital Fountain Pen, which has the OCR included. But in the cloud data will have to upload manually.
