Microsoft Web Platform Installer Release Released

Microsoft Web Platform Installer (WPI) is a tool for quick installation of Microsoft web platform components that include Internet Information Services (IIS) 6.0 and 7.0 web server, SQL Server 2008 Express DBMS, .NET Framework 3.5 SP1, Visual Development Tool. Web Developer 2008 Express SP1 and additional platform components (path rewriting module for IIS, etc.). WPI runs on Windows XP, Vista and Windows Server 2008. The components of the web platform and WPI itself are free and allow commercial development.



The Microsoft platform for the Web is constantly evolving, the number of new technologies and components is growing very fast, the loading and installation of components can take some time, which can be usefully spent on something else, which is why the WPI tool was created that allows you to simplify the installation of the necessary components. With WPI, you can quickly set up a server or development computer that will allow an experienced developer to get started faster, and a developer just starting to learn about the Microsoft platform can get all the components necessary to learn the platform. WPI provides several options for installation:



  • Complete - all platform components
  • ASP.NET Developer - only components required for developing and running ASP.NET applications
  • Custom - the ability to select components to be installed, for example, if you plan to host PHP using SQL Server as your database, you can choose SQL Server Driver for PHP
