Wine 1.3.11 release

Wine is free software that allows users of UNIX-like systems to run Microsoft Windows applications.
What's New in Wine 1.3.11:
  • PO files are now used for translation
  • some improvements in javascript handling
  • some fixes for the Wine debugger
  • translation updates
  • other various fixes (see below).


Source code is available at the following URL:
http://ibiblio.org/pub/linux/system/emulators/wine/wine-1.3.11.tar.bz2
