Modern Code. Program Modern



Programming has written many good books. However, the professional landscape of the developer is changing so rapidly that the printed literature does not always have time on time: new software and hardware components appear, support for advanced technologies is being introduced, more and more software functions are being transferred to hardware ... how to keep track of this professional kaleidoscope, how to keep up from life? Intel offers its own solution to this problem.

The Modern Code program is a real-time learning platform that allows the developer to stay abreast of the latest developments in programming. The program consists of three parts:

  • Video tutorials , divided into short chapters for easy perception;
  • Webinars and live broadcasts from various events;
  • Recorded webinars and broadcasts .

Service started relatively recently, stay tuned. Some interesting resources are available now, we recommend, for example, to get acquainted with the video tutorial on parallel programming and optimization features for the Intel Xeon Phi co-processors .
