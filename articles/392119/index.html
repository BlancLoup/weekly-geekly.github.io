Reflecty project: a mirror as part of a smart home



The user DerVonDenBergen decided to make the mirror smart. He reasoned that there are mirrors in every house, they take up quite a lot of space, but, frankly, there are few functions of ordinary mirrors. Why not slightly improve the mirror?

After detailed planning, the project to create a smart mirror began to be implemented. At first, the videoconcept appeared, and now the first prototype, which works quite well. According to the author, the Reflecty smart mirror allows the user to control the smart home, plus provides the information the person needs on request. To track user actions, the mirror is equipped with a motion detector and a Leap Motion gesture recognition module. The effective working distance for a mirror is about 20 centimeters.









Opportunities:
  • Widgets on the "main screen", which displays information about the weather, date / time, latest news. All this is updated in real time.
  • Weather application that can be opened using the appropriate widget on the main screen. App shows current weather and forecast for several days in advance.
  • Information on public transport routes, including nearby stops.
  • News application, also opened with the widget.
