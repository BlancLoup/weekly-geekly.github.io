Embarcadero RAD studio XE6

On April 15, 2014, Embarcadero launched a new version of its IDE - RAD studio XE6.

New opportunities


Application Tethering Components

Extend the capabilities of your existing Windows applications to interact with mobile devices without the need for full migration. Implement the functionality that should be in a mobile application, by interacting with your existing Windows applications.
Ensure the interaction of your desktop applications (VCL and FMX) and mobile applications
Create mobile apps as add-ons to your VCL apps.
Use Wi-Fi and local network. Future versions will include Bluetooth support.

Components for the taskbar (Taskbar component)

Components for implementing a preview of several windows that can be selected in your applications using control buttons
Automatic or custom previews
Display progress in the taskbar buttons for your applications
Overlapping icons on taskbar buttons

Interaction with cloud services (BaaS), components for Kinvey and Parse

Interact with leading backend-as-service providers to add this functionality to your mobile applications. You get easy access to services in the clouds, which saves you from having to create and maintain your own "backend services".
Use push notifications to engage any device and platform user.
Access data and storage of objects in the clouds
User Authentication
Implement support for REST clients that are available starting from XE5
Supports the three most popular BaaS providers - Kinvey and Parse based on a set of API access components

New VCL styles

Give your application an updated look for current versions of Windows or create a unique design for them.
Includes Windows Tablet style
Supports Windows 7 and Windows 8
Full styling of your applications, including menus and window borders

VCL components for sensor operation

Sensors have become ubiquitous, and even on tablet devices running the new Windows-like Surface Pro. Now your Delphi applications can use the capabilities of position sensors, displacement and others.
Accessing device sensors from VCL applications for Windows tablets
Accelerometer, GPS and Gyro capabilities

Shopping from the app and advertising in the apps

Mobile applications make it possible to make money for their developers. Now you can monetize your mobile apps by embedding purchase opportunities from apps and ads.
Selling content, functionality, services and subscriptions on iOS and Android
Main ad network support (Google AdMob and Apple iAd)

Apps for Google Glass

With Delphi, development for various devices now extends beyond PCs, tablets and smartphones to wearable gadgets. Create Andorid Apps for Google Glass.
New custom styles to optimize application design and resolution under Google Glass
Glass Designer Template Designer

Special abilities

Make your applications more convenient to use by a large number of users, including those who use screen readers.
New special features for
JAWS support on Windows and VoiceOver on Mac OS X

Quality, performance and stability

Get the best possible development and delivery of applications with the highest level of user interaction
Over 2,000 bug fixes fixed
Improved overall application performance at run time for all platforms

C ++ Application Development for Android

Add Android as a target platform for your applications in addition to Windows, Mac OS X and iOS
Create apps for Android and iOS using C ++ based on a single source database.
Run applications on the emulator and ARM-based Android devices
Support various versions of Android, including 4.4 KitKat

C ++ package support for 64-bit Windows

Now you can completely migrate your 32-bit applications, including packages, to a 64-bit architecture.
Place components in shared libraries (DLL)

Search for definitions and search for links

Look for all links to selected items throughout the project.
For C ++ on Win64, iOS and Android

Key features and work with databases

Expansion of key product features
Improvement in the library FireDAC database, FDMemTable
Data Explorer (Database Explorer) for FireDAC
Apache support (WebBroker)
Header files for DirectX 11
DataSnap performance and updated wizards
RTL refactoring and improvements
Update FireDAC driver for Informix
Apache C ++ support
