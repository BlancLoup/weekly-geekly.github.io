Amazon attracts application developers for Fire Phone



No matter how unusual and user-friendly the new operating system is, the main thing that can attract it is the saturated application market. To solve this situation, Amazon launched the Developer Select program ‚Äî developers receive 500,000 Amazon Coins for each application they release, adapted for the company's devices.

Amazon Coins is the local currency of a company that is used for purchases in their app store. Immediately, I‚Äôll make a reservation that using it you can only buy digital products, that is, paying them for a couple of books in a hard cover will not work.

To receive the award, the developer must fulfill the following program requirements:
  • The application must be supported by the following devices : 2nd Gen Kindle Fire (Kindle Fire HD, Kindle Fire HD 8.9 ", and Kindle Fire HD 8.9" 4G), 3rd Gen Kindle Fire (Kindle Fire HD, Kindle Fire HDX and Kindle Fire HDX 8.9 ").
  • High resolution support, maximum for each of the listed devices.
  • Amazon Services Support: Applications must use Amazon In-App Purchasing for in-app purchases and the Amazon Mobile Ads API to display ads. Games must be integrated with GameCircle .
  • All applications must have their own widget using the Home API .
  • Non-game applications should use the right or left sidebar (or both).
  • Games must use the Dynamic Perspective SDK to handle custom gestures . It is worth mentioning the fact that in the case of a simple replacement of a swipe or a gyroscope with head movements, the application will not be allowed into the store.

The period of the application check is about a week. More details about the program are available here . Oh yes, the reward is given only for the first three applications from each developer.
