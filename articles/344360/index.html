Php internals


  • RFC: Explicit call-site pass-by-reference - Great offer from Nikita Popov. At the moment, the fact that a function takes an argument by reference is indicated only in the definition of the function itself: 

    function inc(&$num) { $num++; } $i = 0; inc($i); var_dump($i); // int(1)

    It is proposed to make it possible to explicitly indicate transmission by reference. Thereby warning the developer that the transferred variable will change: 

     function inc(&$num) { $num++; } $i = 0; inc(&$i); var_dump($i); // int(1)
