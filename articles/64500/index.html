Hid.im converts torrents to PNG images

Hid.im is a new service on the Web that allows users to hide .torrent files inside PNG images. This means that users can easily upload hidden torrent files to their favorite image hosting and forums, or use them as avatars on social networking sites without being censored.

Are you one of those people who always dreamed of hiding a torrent inside a picture? Do not wait any longer; on the Hid.im website, it is enough to click the mouse once to convert the torrent into an image file with the ability to decode it back later.

We have to admit that the usefulness of this service did not immediately become obvious to us when we first discovered this project. So we contacted Michael Nutt , one of the project staff, to find out what this was all about.

‚ÄúThis is an attempt to make torrents more robust,‚Äù Michael told TorrentFreak. - The difference is that there is no more need for an indexing site to store your torrent file. Many forums allow downloading images, but more than any other file types. ‚Äù

Hiding the torrent file inside the image takes place with sufficient ease. Simply select the torrent file on your hard disk - Hid.im will take care of the rest. The only limitation of the service: the torrent file size can not exceed 250KB.

Once the torrent is converted, you can easily share it through image hosting or social networking sites that do not allow downloading of .torrent files.

Recipients can decode pictures and receive the original .torrent file using Firefox add-on or a bookmarklet (i.e. javascript in bookmarks - note). The source code is completely open, and Michael Nutt told us that they are hoping for the help of other people, for the emergence of additional decoders supported by other browsers.
