The location of the system unit from the bottom, near the legs.

Wide table top

The main place extended for: 2-3 monitors, 1 keyboard, laptop (or second keyboard), should also be places for the printer and scanner.

Monitor mount should be an optional extra.

No one needs a keyboard holder.

Rubber pads are highly desirable (indicated by a light gray color: the table top and shelf are almost completely covered).

Convenience should be increased: no slide-out stand for the keyboard, built-in socket (not shown, the location is discussed), holes for wires.
