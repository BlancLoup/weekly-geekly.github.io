SurfPatrol extension for Google Chrome



SurfPatrol is an online service for checking browser security and plugins. Getting on the site www.surfpatrol.ru , the user receives information about the current state of computer security. The service informs about critical vulnerabilities found (including those for which patches are not released) and what measures need to be taken to improve the security of web surfing. Among other things, warnings about the need to install certain updates are displayed.

Service exists not the first year. During this time, a large number of inspections were carried out, extensive statistics of typical problems were collected (according to which a topic was written on Habr√© ). But we are not resting on our laurels, and the service continues to evolve, become more convenient.

The next step in this direction was the development of SurfPatrol browser extensions. Today we present to your attention the first result of this work. So - meet: SurfPatrol Chrome Extension .

What for


Often, users forget that they need to visit a special website on the Internet to check the security of their browser. The extension will remind the user about it at the right time and in a convenient, unobtrusive form. In addition, an important task is to display the current status of security of the browser and plug-ins. Of course, when detecting vulnerabilities, you need to notify the user so that he can be checked on the site and get recommendations on how to fix security problems.

Supported Platforms


Currently, Microsoft Windows and Apple OS X platforms are supported, and Linux support is planned to be added.

How it works


When the browser starts, the SurfPatrol extension checks the browser and plug-ins. If problems are detected, the extension icon turns red and the number of vulnerabilities found is displayed:



Clicking on the extension icon opens the SurfPatrol.ru website page, which lists the found security problems, links to updates (or recommendations, if there are no updates for this plugin).



If no vulnerabilities are detected, the extension icon remains blue. In addition, the plugin is rescanned periodically in the background.



Thus, the SurfPatrol extension for Google Chrome greatly simplifies the task of maintaining the level of web surfing security at a high level: the current security status is always before your eyes.

Plans


Of course, the Chrome extension is just the first swallow. In the near future, it is planned to release extensions for such popular browsers as Opera, Firefox and Safari. In addition, the issue of developing an add-in for Internet Explorer is being addressed.
