New resource about Intel technologies for game developers on Unity3D.com


The collaboration between Unity and Intel has expanded the capabilities of game developers, has made their products more productive and more attractive. Together, support was added for the Intel architecture for Android, and access was provided through Unity to new CPU instructions, multi-threading and unique Intel graphics features. Interesting and unexpected prospects are prompted by the use of Intel RealSense technology in games, the support of which is also implemented in Unity.
As part of the Intel Achievement Unlocked game developer support program, a special page is posted on Unity3D.com, which covers all aspects of the interaction between Intel and Unity. Here, game creators will be able to find the latest information about Intel technologies and their support in Unity, view the gallery of popular Android games optimized for x86, and learn about upcoming important events in the game dev world.
You can get some info on Intel & Unity right now. In the video, the developer Jacob Pennock of their company Livid Interactive explains how to use Intel RealSense directly from the C # code of the Unity game.
