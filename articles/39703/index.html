We collect free comprehensive protection Security Suite

There are many all-in-one antivirus programs containing antivirus, antispyware module, personal firewall, web link monitor, system optimization tool, etc. But, thanks to its rich functionality, Security Suite programs are the most expensive means of information security.
However, each of us can collect for free a set of programs to protect our computer. In this case, the main issue is compatibility. Consider one of the options.

Protection against viruses and spyware

For our free collection, choose AVG Anti-Virus Free 8.0 . The program is easy to install and works with Windows XP and Vista. In addition, there is no annoying ads for the paid version of the antivirus, and there is also an anti-spyware module that detects and removes spyware (Adware) and adware. Thanks to this, you don‚Äôt have to install antispyware software separately. It is worth noting that the antivirus uses the same virus databases as the paid version, which have been tested by reputable organizations, such as AV-Test.org.

Of course, the free version of AVG antivirus has several limitations: the program will successfully detect and remove rootkits before they invade the computer, however, the antivirus will not detect and block rootkits that have already penetrated your PC. In addition, AVG Anti-Virus Free has a low ability to detect new threats that are not yet included in the virus database.

There is a free ThreatFire program that does an excellent job with detecting new threats, as well as rootkits, but the current version 3.5 conflicts with AVG and will cause the system to hang. The developer of ThreatFire, PC Tools, claims to be working on a solution to this problem. For now it is necessary to refuse sharing of these programs.

When you install the antivirus, AVG will offer to install the toolbar for the browser, but you will have to abandon the LinkScanner function, which provides the security level of websites in search queries. (For this purpose, we will consider another program.)

Internet traffic control

For firewalls, there are no generally accepted methods of testing, however, according to the results of the latest traditional test of the Firewall Challenge , the Online Armor Free 2 showed a really good result. In addition, this program is easy to use, unlike many free firewalls.

After installing and checking security, right-click on the program icon in the tray and turn off Program Guard - a function that shows pop-up windows for each new program that you want to install or run, and not just for applications trying to connect to the Internet. .

The Firewall Online Armor Free 2 does not work in Windows Vista, however, users of this operating system can turn on Vista (it is disabled by default) the free Vista Firewall Control network monitor.

Web Link Control

We have already abandoned the AVG LinkScanner feature. In order to check websites for security, providing protection against spyware, junk e-mail, viruses and Internet fraud, you can use the SiteAdvisor browser plugin (Firefox, IE) from McAfee. When using the search engines Google, Yahoo! or MSN website security ratings will appear next to the search results.

System cleaning

Many Security Suite programs offer, as an additional feature, tools for system optimization. For this purpose, you can use the well-known all program CCleaner, which cleans the system from various "garbage": deletes unnecessary files and folders, cleans and corrects registry errors, deletes the history and cookies of a web browser. When installing, you can opt out of the Yahoo! panel by unchecking the corresponding checkbox. It is also necessary every time you use the program to create a backup copy of the registry, if suddenly something goes wrong. CCleaner will offer to do it yourself.
