Presentation of webOS and new HP devices at SumIT

In February this year, in San Francisco, Hewlett-Packard unveiled its new devices : HP Veer, HP Pre 3 and HP TouchPad - and also announced the release of webOS 3.0.

This summer, Hewlett-Packard will present its products in St. Petersburg at SumIT . HP will not only give a presentation, but also provide the event participants with their new devices. This means that SumIT participants will have a unique opportunity to become one of the first developers for HP Veer 4G, HP Pre3 and HP TouchPad.




HP will be represented by Igor Belousov, Ph.D., lead manager for work with educational, scientific and government institutions of Hewlett-Packard. Igor will not only talk about the company's developments and answer all the questions that have arisen, but will also become a mentor for the teams that decide to develop their projects in this direction.

Below we present a brief overview of webOS, based on the article by Igor Belousov.

HP webOS is designed primarily for mobile devices (smartphones, tablets), but also applies to other classes of devices, for example, it is already built into some models of HP printers. The platform‚Äôs features are web-based application development (JavaScript, CSS, HTML 5), multitasking, the presence of the Ares cloud application development environment, Enyo‚Äôs convenient high-level programming environment, cross-platform programming support for the most common mobile operating systems, based on Linux kernel.

The use of standard web technologies makes webOS especially attractive for web developers, minimizing the barrier to the transition to a new platform.

HP webOS is initially focused on the cloud structure, based on events such as alerts from remote sources, data and program updates, and remote synchronization.
Ares cloud platform is available as a web service that allows you to develop and debug applications completely in the browser, without the need to install additional software. Storage of projects and all data necessary for them is also carried out in the cloud.

An example of creating an Ares application using GPS navigation with a few simple steps and a minimum of programming is presented in ares.palm.com/Ares/docstemp/tutorial10.html

HP webOS contains an embedded implementation of the open WebKit package with a built-in JavaScript code interpreter V8, PDL application development libraries, and PDK tools for creating plug-ins with C / C ++ code (Plug-in Development Kit). PDK provides, among other things, the ability to include in the program three-dimensional graphics written in OpenGL ES.

Javascript programming uses Mojo high-level software environments (versions 1.x and 2.x) and Enyo (version 3.x).

Full details July 17th at SumIT in St. Petersburg!
