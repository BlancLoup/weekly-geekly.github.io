New plugin Dr.Web for Thunderbird

It was recently announced the release of a new version of the antivirus plug-in for the popular Mozilla Firefox Internet browser by Doctor Web. A distinctive feature of this plugin is the ability to install it not only directly into Fire Chanterelle, but also into the Mozilla Thunderbird email client.

When installed in the mail client, the plugin allows you to check for viruses Internet links in emails. This feature has become very relevant due to a significant increase in the growth of the amount of spam sent by e-mail, part of which contains links that carry a serious threat. For example, users of the Internet Explorer browser may well get a Trojan program Trojan.Packed.142 on their computer by clicking on the link sent to their e-mail address.

Users of Mozilla Thunderbird, who installed the Dr.Web plugin, can check for the presence of malware using the links in the received emails by downloading the contents of an Internet resource via a link to the Doctor Web company server. If a virus is detected, the program will immediately issue a warning. Thus, this plugin, provided to all Internet users for free, also contains an anti-phishing feature, since very often with the help of e-mails, attackers try to send online banking lovers to fake websites. Now, having checked the links of the letter using Dr.Web anti-virus, the user will have a real opportunity to avoid the dangerous trap created for him by the hands of cyber-criminals.
