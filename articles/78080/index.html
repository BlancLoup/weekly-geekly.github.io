Students Calculator is now open source

Students Calculator - this is my first serious application, the idea brought to the final implementation. This is what determined my current profession. It‚Äôs ridiculous to remember, I started writing it in a notebook, in a functional style, not knowing about the existence of Javadoc and even java.util, using only the source code of other j2me applications. (Of course, later on, with the growth of my skills, the program experienced 2 refactoring)

The program was quite popular among schoolchildren and students. She was even devoted to a column in one magazine). But the highest point of fame was a letter from the Chinese representative office of Motorola with the proposal to include the Students Calculator in the standard applications of their mobile phones (which, unfortunately, did not come true)

What is this post for: I have already abandoned work on the program for quite a long time. For me, this is a goal achieved and it is difficult to motivate myself to continue to fulfill the wishes of users. I also thought that with the advent of touch-phones a wave of good touch calculators would come. However, I still receive letters with questions and suggestions, a request to provide the sources ... (their authors will forgive me - I rarely answer them). So today I decided to overcome laziness, find the source code in the depths of bakups and put them into public access. Now everyone can extend the life of this project.

Offsite: calc.motofan.ru (here is the latest version 1.5.0 PR of 03/26/2006)
Actual latest version: 1.9.2 alpha from 2.4.2008
Sourceforge: sourceforge.net/projects/stdcalc/develop - source code here

The latest version 1.9.2 bugs especially not sin. The main wishes of users - support for touch screens and localization.
The wonders of fast and accurate floating point calculations are due to the excellent implementation of the Roar Lauritzsen Real class (GPL license)
The program also uses the BigInteger class, which I modified a little already. I do not know who the author. Possibly pulled from earlier versions of J2SE

For those who wish to continue working on the program, I will provide developer access to SVN. Write in a personal. Perhaps there will be a lot of you, so especially those who wish - write especially loudly)

Very useful: the thread in the forum motofan.ru - where everything went from. Development history, communication with users. Many ideas and wishes can be collected from it.

UPD: Thanks to everyone thanked. Very good nicely.
