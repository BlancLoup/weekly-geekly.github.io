MDC is now available via the web.

What we have been waiting for has happened. Beta version of the MDC web is ready to check for lice. Web MDC is a web-based multi-protocol IM client with support for storing the history of communication on the server.

  • Icq support;
  • Jabber support (+ ya.online + g.talk);
  • Mail.ru support;
  • connection by several protocols;
