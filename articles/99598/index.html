Vim 7.3 public beta released

A public beta of the next version of the esoteric text editor Vim has been released. In the new version 7.3, they are offering us undo between the launches, interfaces to lua and python3. A new feature codenamed 'conceal text' allows you to hide parts of it when coloring text. This is very useful when wiki-formatting - the metacharacters responsible for headings, selection and other pleasures of formatting, you can simply hide and see the text colored in color.
In addition to the above, added what I waited for many years - the ability to set the vertical guideline (vertical line indicating where the text width ends in 78/80 / number of characters. The command name is 'colorcolumn'.
