Yes, the new version of the OS is likely to have the same name. No major changes are foreseen, this is work on improving the current version of Android. There are rumors about the following changes:

Second curtain for ‚Äúquick settings‚Äù

An option that is shrouded in mystery. When re-pulling the notification curtain, the user should get access to the quick settings. Most likely, it will be something similar to switches in CyanogenMod or in shells from vendors, like Samsung or LG. In the lost version, on the site of these settings, there was just a stub picture.

Custom profiles for tablets

It is no secret that the blanks for the system profiles appeared in the Android code back in 2011. The new version of the OS should allow changing the device settings and the list of installed applications depending on the selected user profile. Also, we are likely to see "parental control", which will complement and make safer multi-profiling system.

Security enhancements

Google is working hard on Android security. So, recently, a malware scanner has been added to the server side of Google Play. Also, lines were found in the Google Play client that indicate that the verification of applications already installed on the device for malware will be available soon. The leaked OS version found evidence that SELinux modules will be added to the kernel, which will restrict program access to system files, minimizing the possibility of harming data. Also, the new version of the OS should please people who constantly use VPN, it will have options for using the Internet only when VPN is available.
We should also see the option to confirm the sending of messages to short numbers. This is great to limit malicious programs that send expensive SMS. Now the pests will be powerless, unless the user himself allows sending.
