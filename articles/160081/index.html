OpenStreetMap switches to Leaflet instead of OpenLayers

For a long time, the main site OpenStreetMap used the OpenLayers javascript library as a means for displaying geographic maps on the site with the possibility of enlarging, reducing and dragging it with the mouse, as well as selecting ‚Äúlayers‚Äù (maps and labels).

Yes, and all users of OpenStreetMap will now be more comfortable.
