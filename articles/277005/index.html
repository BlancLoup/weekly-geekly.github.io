Amazon has introduced a new free game engine Lumberyard

On February 9, Amazon unveiled and publicized a beta version of its 3D engine, Lumberyard . They promise that he has deep integration with Amazon Web Services and the Twitch video streaming service (also owned by Amazon). The engine is free for developers, its source code is open and available for download. Lumberyard is based on the CryEngine engine code, game writing language is C ++. While it is possible to create cross-platform applications for PC, Xbox One and PlayStation 4 on it, support for iOS and Android devices will be announced in the near future. The company wants to earn money on it through the use of AWS services by developers: when creating network games, developers can use their own servers, but if they need to use third-party servers, they must be Amazon servers.

Declared features:

Graphics : physics based shaders, dynamic global illumination, particle system editor, plant creation tool, real-time water caustic, volume fog, effects: color grading, motion blur, depth of field, HDR lens flares and others.

Working with characters : the Gepetto tool with the Mannequin add-in for creating skeletal animation. Ragdoll physics, animation layers, blend trees are supported.

Network support : the GridMate subsystem, which optimizes channel width utilization and reduces latency.

Sound : the free version of the Wwise LTX sound engine

Modular gems : a library of 16 built-in gems facilitates working with natural effects (rain, snow, tornado, lightning), multiplayer, camera, and UI.

The Lumberyard archive for installation on Windows (for Mac and Linux, it seems, there are no versions yet) is heavy, weighs about 10 GB, plus you can download three resource packs: Woodland Asset Package (for forest creation, 1.5 GB), Beach City Asset Package (characters, objects and props for the coastal city in the style of the 1950s with zombie motifs, 4.8 GB) and almost ready engine first-person shooter, ported from CryEngine GameSDK (Legacy Game Sample, 3.5 GB).
