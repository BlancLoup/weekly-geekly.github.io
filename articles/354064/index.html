PHP Digest number 129 (April 8 - 22, 2018)


Fresh selection with links to news and materials. In the release: CakePHP 3.6.0 and other releases, php-ffi extension, a portion of useful tools, and much more.
Enjoy reading!



News and Releases


  • DevConf 2018 - May 18 in Moscow will host the annual conference of professional web developers. Declared interesting reports on PHP and not only.
  • Composer v1.6.4 (with security fixes)
  • CakePHP 3.6.0 - About future releases of the framework.
  • Deployer 6.2.0
  • PASETO v1.0.0 - Stateless token, a safe alternative to JWT.

Php internals


  • PHP FFI extenesion - Dmitiriy Stogov introduced an experimental dstogov / php-ffi extension that allows calling native C functions from PHP code, as well as using data structures and C variables.

Instruments


  • xobotyi / beansclient - Client for server queues beanstald, without dependencies and with 100% coverage tests. Sent by xobotyi .
  • vimeo / psalm - Static code analyzer to search for possible errors. Alternatives: phpstan , phan . Docker image with these and other PHP code analysis tools: jakzal / phpqa .
  • matomo-org / matomo - An open version of Google Analytics implemented in PHP. Previously known as Piwik.
  • formapro / pvm - A powerful library for describing business processes, such as BPMN.
  • makasim / yadm - Efficient library for working with MongoDB in PHP. Very fast preservation and hydration. About using in symfony.
  • PeeHaa / mailgrab - Simple SMTP server for debugging letters. Implemented on Amphp asynchronous components
  • rybakit / msgpack.php - Serialization to MessagePack in pure PHP.
  • wapmorgan / UnifiedArchive - Utility for unified access to 9+ different archive formats.
