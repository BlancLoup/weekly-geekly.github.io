3CX Phone System v15 System Development Plans

Changing the internal architecture of 3CX v15 required a significant amount of work. We made the new version very flexible and extensible. Therefore, despite the fact that in the initial release the functionality of the system was not very different from v14, we will add many new interesting features in the coming months.

To better understand the future capabilities of the system, we share development plans. Note that it is not always possible to guarantee accurate release dates, because some functions require the participation of third-party companies. But we can say with confidence that we will implement the functions below in the specified time frame.

3CX Client Updates


In v15, 3CX client updates will not be affected by server service packs. This simplifies and speeds up the release of updates, relieving us and 3CX administrators of the burden of thorough peer testing. Of course, to activate additional functions in the client, you still need to install the service package on the server. In the coming weeks it is expected:

  • 3CX Client for Mac Update
  • CRM integration template release

Release Date: July - August

v15 service pack 1


  • Full localization of server interface and clients
  • Single call information display interface
  • Adding support for third-party applications via the Call Control API

Release Date: July - August

v15 service pack 2


This will be a major update with a number of new features:

  • Features of the 3CX Hotel Module will appear in PRO and Enterprise editions
  • Convenient management of extensions connected via FXS gateways and DECT bases
  • Group operations with extensions, DID lines, etc.
  • Tweaking administrative access to the 3CX management interface
  • Possibly: cloud (multi-tenant) version of 3CX v15

Release Date: September - October

v15 service pack 3


At the moment, the final list of functions is not formed, but we can say the following:

  • 20 most popular offers posted on our forum will be implemented.
  • A new report module that allows you to generate reports and immediately view them in the web interface

Date of issue: November - December

Voice Application Designer


Currently already created VAD applications cannot work on v15. We are planning a major update of the VAD utility, which will fundamentally increase the functionality of voice applications. New VAD applications will use code and logic compatible with new CRM integration modules. You will enjoy the ability to create outbound dialing scripts and work with Call Queues. You can actually create powerful integrated call centers!

Release Date: October - November
