Suggest.io: Towards the Perfect Search

Live search for your site


Site search is a problem for most Internet resources. Creating your own search takes time and requires the skills of fine tuning the search engine. If the site owner is interested, in order for his search to remain relevant, he needs to have a specialist ‚Äúat hand‚Äù who is constantly engaged in improving the relevance algorithm, index, query base, etc. All this is troublesome, supporting your own search requires serious expenses.

Suggest.io fills this gap. The new service came closer than the others to solving the problem of organizing a live search on a site with a minimum of installation time.





Easy installation


To install Suggest.io only 3 steps are required:

1. Enter the address of your site on the start page of the service www.suggest.io



On the right, in the window that appears, you will see information about the status of indexing your site.



2. Copy the javascript code from the opened window in paragraph ‚Äú2‚Äù.



3. Paste the javascript code into the templates of the pages of your site immediately before the closing tag

Next, the following happens:

- in the event that your site already has a search field, Suggest.io will determine it and use it in further work;



- in the event that there is no search field, the system will create it - the search icon will appear on your site; when you hover the cursor, it will unfold in the search field.



Installation completed. Live search for your site works!

Development
