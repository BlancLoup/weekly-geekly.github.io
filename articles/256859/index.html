Unity in Russian

Hello, dear habroobiteli!

UPD
I can make everyone happy. We are starting to translate Unity in Action: Multiplatform Game Development in C #. We will try to do everything as quickly as possible, but at the same time with high quality.


A crazy thought occurred to us, but isn't it time to translate a book on Unity into Russian? Especially now, when the full fifth version is available (and even for free).
Books in English came out a lot, but I want to choose not too old and the most interesting.
If we missed something interesting, write in the comments, we will be happy.

1. Unity in Action: Multiplatform Game Development in C #, Joe Hocking

The book will be released in late May. Addressed to readers who are familiar with programming, but for the first time face Unity.

2. Learn Unity Programming with C #, Jonathan Weinberger

This book may appear in June. It sets out the basics of C # and Unity

3. Introduction to Game Design, Prototyping, and Development: From Game to Unity and C #, Jeremy Gibson

A cool book, but it has two drawbacks: it was published in the summer of 2014 and the price (944 pages of the original).
