7 useful plugins for Sketch



Sketch is easy to work with. Many people know how to use basic functions. But not everyone works with plugins that extend the capabilities of the program. I will talk about 7 useful additions that help optimize labor costs and automate some tasks.


Small hint: most of the plugins in question are available in the Sketch Toolbox . This is a separate small program with search and the ability to quickly remove / install plugins.



1. Sketch export assets


Let's start with the export of icons. Sketch has a good export tool, but the size (1x, 2x, 3x) in it needs to be customized additionally, which takes time each time. This plugin is needed to unload icons with grouping by screen density and with the desired folder name in a couple of clicks.






2. Craft by Invision


A plugin that can synchronize with invision, insert real data from a specific web page, folder or JSON file, as well as duplicate data with loading from the source of the first block. Contains a library of its elements.





3. Runner


It's like a spotlight, but for Sketch. He searches for artboards, objects and program commands by name. Runs plugins, creates files and applies styles.





4. RenameIt


A plugin that allows you to rename several objects at the same time. These can be icons, symbols, artboards. To prevent the name from being the same for all files, you can add a static prefix (for example, the name of an object) and a dynamic prefix (for example, 1,2,3 / A, B, C).
And also there is a search function and replace characters in the names.







5. Sketch Measure


One-click developer specification generation. In addition to the specification with sizes, colors and styles of elements, there is a functionality for cutting icons if they are marked in advance as ‚Äúmake exportable‚Äù.


Demo - http://utom.design/news





6. Sketch Focus


Commenting and writing tasks directly to Sketch makes it easier to find the desired artboard and saves time to go from program to program.





7. Sketch-Quick-Presentation


Automate the creation of presentation layouts / maps of screens. You just need to select the artboards in the list, select the plugin and click generate with headers or without (headers will be taken from the name of the artboard).





I use these plugins almost every day. And they help with the organization of the correct workflow and affect the speed of work in Sketch.
If you know any other interesting plugins, write in the comments.
