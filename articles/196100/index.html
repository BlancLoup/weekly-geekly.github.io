HTML5 Flash Player Shumway added to Firefox



Flash Player Shumway , written in HTML5, today added to the Firefox codebase . Using Shumway, you can play flash files (in SWF format) without Adobe Flash Player, see demos . To view the demos do not even need to put the extension, Shumway is loaded as a JS-file.

This is very important news from the point of view of security, because the Adobe Flash Player plugin (along with Java) is one of the main sources of vulnerabilities in the browser, and it would be good to get rid of it.

Shumway is a kind of experiment, the developers wanted to check whether efficient rendering of flash animations is possible only using HTML5, without using native code. The project was developed by the community, and now received the support of Mozilla.

SWF format (Shockwave Flash) is a proprietary format for flash animation, vector graphics, video and audio. Used in games, advertising, cartoons, videos, etc. Originally developed by FutureWave Software for creating cartoons, the format was presented along with the FutureSplash Animator program in 1996. Then FutureWave Software was acquired by Macromedia, which in turn was absorbed by Adobe. The authors of the Flash technology refused to go to Adobe and left Macromedia before the takeover.
