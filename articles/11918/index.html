Firefox 3 Alpha 6 (Win, Lin, Mac)

Released release. In it, the SQLite engine has been updated to version 3.3.17, the code for managing cookies has also been improved, with the result that working with them is 5 percent faster.
Among other changes, we can note the correction of errors in the module for downloading files that dealt with working with large files. In addition, errors were fixed in Places - a system that combines bookmarks, history and other information and created for more convenient search of visited sites.

As for innovations, Alpha 6 now has the ability to add the Firefox restart button to the toolbar. Also added the option to change the font settings for each site separately.

Firefox 3 Alpha 6, codenamed Gran Paradiso, is available for Windows, Mac OS X, and Linux. Anyone who wants to try out a test release can download it here.
