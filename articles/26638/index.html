Video@Mail.Ru: Web 2.0 style redesign

Videohosting Video@Mail.Ru changed the design and expanded its functionality to better fit the modern fashion on Web 2.0. There are tags on the project, the ability to subscribe to selected RSS feeds, and Ajax technologies are also implemented. In addition, the Video@Mail.Ru main page has been completely redesigned, access to UGC and partner content has become more convenient.

The updated functionality of the project allows you to assign a set of tags to each clip. Each user can watch the tag cloud in the personal menu "My Tags". The most popular tags are located in the global menu "Tags", as well as on the main page of the portal. Now you can watch similar videos with the same theme or tags.

The quality of movie playback is improved by increasing the resolution and bit rate. Along with the traditional 4: 3 aspect ratio video, the widescreen 16: 9 ratio is now supported, which is becoming increasingly popular in inexpensive home video cameras.

The preview bar for the contents of the video album is now available on every video viewing page.

The video player interface has also been redesigned - it has become more compact, the ability to watch video in full screen mode has been added, the interface for inserting the player code into external blogs and forums has been simplified.

It is possible to subscribe to new videos of certain users with notifications both by mail and through an RSS feed.
