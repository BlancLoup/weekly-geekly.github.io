Microsoft Azure Developer Camp Russia



May 23 | 9:30 - 19:00
Microsoft Office, BP Krylatsky Hills, st. Krylatskaya 17k1, Meeting rooms Uranus, Jupiter, Saturn
Moscow, Russia


The proliferation of cloud technologies is gradually changing the way applications are developed, and new complex tasks arise for programmers. Applications are not consolidated, operations are executed concurrently and asynchronously. Containers are actually becoming the standard for upgrading applications in the cloud.


Take part in Microsoft Azure Developer Camp - a free technical event for developers to be held on May 23 in Moscow. As part of the training, you will gain knowledge of Azure capabilities and practical skills in migrating and upgrading applications and infrastructure.


During the sessions, we will have to study containers in detail, work with a large set of Azure cloud services to implement container-based solutions, and also learn various techniques for expanding the functions of containers without changing the existing code.
