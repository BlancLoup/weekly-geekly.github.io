Convenient rescue DIY

Good day to all. I decided to describe my troubles in creating a more or less functional resuscitator, which will be always at hand.

I‚Äôll make a reservation that I used to have a 16GB usb flash drive as a resuscitator, but the place was not enough.

I will describe how to do this briefly (Maloli, useful).

USB flash resuscitator



With the help of WinSetup, I formatted the flash drive by writing msdos boot files to it. Then I trimmed the boot partition and added the NTFS partition.
I added drivers and programs for dos to the boot partition (among other things, I installed ntfs4dos in order to have access to ntfs partitions from under DOS). Then I threw it on the grub4dos boot partition. I deleted the designated letter from the boot partition, in the hope that it will not be mounted in windows, which is infested with viruses more than other systems are susceptible to viruses. On the NTFS partition, the root directory was protected from writing using access rights (yes, the protection is dubious, but that's enough for me). How to do this, see my old post .

One of the advantages I want to note is that the first msdos is loaded from under which you can:

chkdsk.exe



format.com



grub4dos.exe



alladin.exe :)


And by running grub4dos, we are able to download both BartPE and ErdCommander and Acronis and NortonGHOST, etc. etc.
