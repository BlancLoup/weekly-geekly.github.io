The first alpha of the EduMandriva supplementary educational repository is Edumandriva 2009 Alpha1.

A DVD disc with a repository was developed at the Mandriva Competence Center of the Glazov State Pedagogical Institute named after V.G. Korolenko and is designed to facilitate the installation and use of educational software, as it contains a selection of software needed to conduct classes in computer science and ICT disciplines in secondary, specialized secondary and higher educational institutions, based on the wishes of teachers and testing previous versions repository

Features:
  • Lazarus in GTK2 and Qt4 build (meta-packages task-lazarus-gtk2 and task-lazarus-qt4;
  • FreeBASIC (compatible with QBASIC);
  • Scratch - programming for kids based on Logo;
  • Idol (school algorithmic language);
  • LightLang is one of the best dictionaries;
  • Phun - a physical simulator;
  • OpenProj - free analogue of MS Project;
  • iTest - student testing system;
  • ATutor - a program for the construction of distance education systems;

and other software, a complete list of which you can find on the WiQi project :

The disk is placed on the existing installation of Mandriva 2009 Free (correct work with Mandriva PowerPack 2009 in development).

The first alpha does not yet contain the installer with a graphical interface, but allows you to install all the software using the special SetupAll.run script or connect repositories to install the software through a standard package manager.

Download the iso image from here: ftp://ftp.linuxcenter.ru/pub/Mandriva/EduMandriva/iso/2009.0/EduMandriva2009_alpha1.iso

Bug Traker: http://qa.edumandriva.ru
