Technical support 3CX answers: fast free update to the latest version 3CX

If you once purchased a 3CX license, but for some reason decided not to update it, now such an outdated system should be used very carefully. The fact is that in order to activate the license and download updates, 3CX uses SSL certificates certifying, in particular, that you are actually downloading files from the developer, and not ‚Äúfake‚Äù files, for example, it is the distributors of the Petya virus.

However, the validity of these certificates both on 3CX servers and on your systems, if they have not been updated, has come to an end. The problem is further complicated by the fact that the certification authority, which at one time issued 3CX certificates, cannot extend them due to technical disagreements with Google and other market players.

To solve this problem, 3CX offered absolutely all users of previous versions of 3CX to upgrade to the latest version for free. In fact, you received a multi-year subscription for updates as a gift! Distribution of proposals was carried out within two months by e-mail addresses specified during activation of the license. All that was required to do was to go step by step with your version of the system to the last via backup.

However, all this was true until October 31, 2018. Now we must be especially careful. What happens if you still have not updated to v15.5 Update 6? Your system will continue to work as before, but you will not be able to activate the license if the activation suddenly ‚Äúflies‚Äù. The activation of 3CX binds, first of all, to the MAC address of the network card, but also to other server parameters. Therefore, in such a situation it is extremely important not to reinstall 3CX, and not to change the server configuration! Your 3CX will no longer be able to activate and will switch to the free edition mode (for the version of 3CX that you have installed). In fact, your telephony will ‚Äústop‚Äù.

However, we have tried to protect you from such a development of events! 3CX has released a special utility that automatically updates 3CX v15 and higher to v15.5 Update 6, installing all the missing service packs. That is, if you have v15.0 or v15.5 installed, you can update the system with one click, regardless of whether there is an update subscription.

Update 3CX v15 and higher on Windows


On the computer where you installed 3CX, download the utility , unzip the files and run 3CX WinUpdater.exe with administrator rights.



Depending on the number of required updates, PBX services will be restarted several times. The update process is displayed on the Windows command line. Do not close this window!



Update 3CX v15 and higher on Linux


Connect to the 3CX computer via SSH and switch to root privileges (sudo su command). Enter system update commands: 

apt-get update apt-get upgrade

If you have 3CX older than v15.5 SP1, run these commands twice, since the newer service packs are located in a different repository.



Update 3CX v14 and below with an active subscription to updates


Update the system until you reach v15.5 Update 6. The latest 3CX distribution already contains Update 6. The procedure is simple, and there is no need to activate the license key each time when switching between versions.

  • Backup your current system.
  • V11: Download V12 , V12.5 and V14 , and switch between versions using backup and restore.
  • V12: Download V12.5 and V14 , and navigate between versions in the same way.
  • V12.5: Download V14 and navigate between versions in a similar way.
  • V14: make a backup of the system and specify it (file) when installing a new V15.5 system.
  • V15: consistently update the system from the management interface to the current version v15.5 SP6.

Update 3CX v14 and below without a subscription to updates


Contact 3CX Customer Support. Here you can help go to 3CX v15.5 Update 6, but you will need to agree on a specific timeline for the transition.
