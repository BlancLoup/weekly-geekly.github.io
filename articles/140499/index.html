YouTube updated video editor



Recently, the most famous and most visited video hosting in the world introduced a new feature that allows you to edit downloaded videos in a couple of clicks. Actually, the new feature is a kind of autocorrection, for different characteristics of the video. At the same time, the service itself will prompt the user whether to edit the downloaded video (for example, the image is dark or the hands were shaking when shooting). All this can be fixed in a couple of clicks thanks to the specified update.

If the user is not satisfied with the results of editing, then everything can be rolled back to the original state. The new features of the YouTube video editor are currently available only to users working on laptops and PCs. Users who download videos from mobile devices will be able to edit their videos, but only from a PC or laptop. Below you can see several examples of videos before and after editing.

By itself, the video editor was presented for a long time. With its appearance, users of the service have the opportunity to edit their videos, including brightness, contrast, a possible reduction in video shake. In addition, you can add and all sorts of video effects, if desired, of course.

According to the developers, the video editor allows you to avoid having to buy expensive software if you need to edit some little things. But, of course, for professionals you need the appropriate software.
