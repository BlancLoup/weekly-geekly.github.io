Develop plug-ins for JetBrains products and sell them on the JetBrains Marketplace

We launched JetBrains Marketplace - a new platform on which developers can sell and license their plugins to JetBrains products. Now the authors of plug-ins will be able to devote more time to the development - they will not have to invent their licensing decisions and engage in sales. And users will be available even more convenient tools that will help work more efficiently.



JetBrains Marketplace is a new stage in the development of the Plugin Repository , in which there are now more than 4,000 tools. They help extend the functionality of JetBrains products and tailor them to the individual needs of developers. There are a variety of plug-ins: some provide support for additional languages ‚Äã‚Äãand new technologies, others serve to integrate with cloud platforms or help to remember keyboard shortcuts; You can select an interface theme and even set the progress bar as Nyan Cat. Plug-ins are available for almost all JetBrains products, including IDE based on IntelliJ, TeamCity, and .NET tools. All these plugins will remain available further.

In recent years, more and more companies and developers are choosing IntelliJ platform and other JetBrains products as the basis for developing their own tools for programmers. We created the JetBrains Marketplace , so that developers can focus on plug-ins and do not spend time searching for solutions for managing licenses and developing distribution channels. Our platform provides a turnkey solution for software licensing, billing and sales infrastructure, as well as sales and marketing support.

How platform JetBrains Marketplace will be useful for users:

  • Additional functionality provided by third-party paid plugins.
  • Simple installation of plug-ins, payment, subscription and management through the JetBrains Account .
  • Easily manage licenses through JetBrains Account , including managing all subscriptions and licenses on a private license server.
  • Coordination of licenses for plug-ins and licenses for JetBrains products (optional).
  • Various discounts (for example, for the volume), special offers for user groups, open-end projects, as well as other JetBrains support programs (will appear soon).
  • Security and protection of personal data: when you purchase a plug-in, you will not have to provide bank card data and other information to third parties.

Now new third-party plug-ins are available only for our IDE based on the IntelliJ platform, however we plan to extend support to other tools.

What already is


More than 100 vendors participated in our Marketplace Early Access program . We are pleased to present plugins that are already being sold and licensed through the JetBrains Marketplace (descriptions provided by plug-in developers):

hybris integration
by Alexander Bartash
Using the Hybris Integration plugin, you can automatically import Hybris projects into IntelliJ IDEA. This plugin allows you to enjoy all the benefits of IDE when developing for Hybris and adds even more functionality.

Iedis 2
by codesmagic
Iedis is a beautiful and handy GUI plugin for Redis (resident data structure storage used as a database, cache, and message broker). Iedis works with millions of keys without blocking the Redis server.

Java antidecompiler
from BIS Guard & Co.
Java Antidecompiler protects bytecode and Java resources from decompiling and prevents reverse engineering and plagiarism of your software. It combines obfuscation techniques, Java cryptographic algorithms, and original BIS Guard & Co. technology.

JetForcer | The Smartest Force .com IDE
from JetForcer Team
JetForcer is a powerful plugin that allows you to develop for Salesforce in IDE from JetBrains. It is ideal for developing both client (support for Lightning and Visualforce), as well as the server part (Apex and SOQL).

JFormDesigner
by FormDev Software GmbH
JFormDesigner is a professional GUI design tool for Java Swing user interfaces. First-class support for MigLayout, JGoodies FormLayout, GroupLayout (Free Design) and GridBagLayout makes it easy to create professional-level forms.

Manifold
by Manifold Systems LLC
Manifold breathes new life into Java: you can use extension methods, structure types, Java templates and type-safe reflection. The plugin provides type-safe access to other languages, such as GraphQL, JSON Schema, YAML and Javascript, directly from Java.

Orchide
by Thomas Fr√∂scher
OrchidE provides support for Ansible IT infrastructure automation. This plugin provides the advanced capabilities of the IntelliJ platform to DevOps teams that use Ansible to manage the infrastructure.

RON (Rusty Object Notation)
from Vultix
The RON (Rusty Object Notation) language plugin adds support for importing Rust types, autocompletion, validation, and more.

Wolfram Language
by Patrick Scheibe
The Wolfram Language plugin transforms products based on the IntelliJ platform into powerful development tools for the Wolfram language and Mathematica code.

ZenUML support
by ZenUML
The ZenUML plugin helps generate sequence diagrams directly from Java or ZenUML DSL methods. This plugin is useful for analyzing existing code and for modeling processes before writing code.
