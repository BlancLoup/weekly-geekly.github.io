Microsoft releases hotfix for MS14-051 update

It looks like the last patch tuesday has become a really unpleasant event for the company. Yesterday we wrote about the recommendation of MS to refuse to install the update MS14-045, which causes BSOD and interferes with the normal loading of the computer. The update was withdrawn from the update server and company website. It turns out a similar fate befell the latest cumulative update for the Internet Explorer web browser MS14-051, which closes 26 Remote Code Execution vulnerabilities in the browser. Updating can significantly reduce browser performance and cause it to hang.
