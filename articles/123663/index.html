Google Maps (Android): Loading a piece of map for offline use

The new version of Google Maps 5.7 for Android has several interesting features , but one of them, which is still in labs, was forgotten to mention: this is the ability to download a piece of map in order to use it when there is no internet connection.




In order to download a piece of the map, you must select the desired place ( Place ), click "More" and select "Download map area". Then a map of this area will be loaded in a 10-mile radius (approximately 16 km) with approximately 20 levels of map detail.

Will not load:
  1. "View from the satellite"
  2. 3D buildings
  3. directions (directions).


Those. to search for how to go somewhere, you still need access to the Internet.

Loaded pieces of the card are in the cache (in the settings you can clean up if you no longer need). In addition, after 30 days they will be automatically deleted.

Link to Google Maps in the Android Market if anyone has not yet :)
