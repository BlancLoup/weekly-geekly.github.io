What's new in PhpStorm 6.0

On March 12, a new version of PhpStorm 6.0 became available.


Major improvements


Composer support with the following functionality:
  • Creating a new project with Composer
  • Initializing Composer in an existing project
  • Manage dependencies and add packages in a convenient graphical interface
  • Autocomplete when working with Composer via command line

Now PhpStorm provides smarter work with namespaces namespaces (auto-import, auto-creation based on the class name and many others).

Added a lot of different useful nishtyakov for refactoring code (for example, you can send classes up to the parent class, and vice versa, quickly rename the namespace)

Added Rest client (now you can test ajax requests directly in the IDE))

All templates are now fully customizable, including PHPDoc.

Improved work with a remote server, you can work with it on the principle of Drag & Drop

New interface to work with the database.

Reworked HTML and CSS part, Emmet support added, now you can typeset faster)

The appearance has been significantly reworked, and a new Dartiva netivny theme (dark) has been added, and they say that for owners of retinas, new good icons have been prepared.
