Restore a broken Coreldraw file

Having passed a two-hour journey from ‚Äúwhat to do‚Äù to ‚Äúfuh‚Äù, I decided to share with the browser a method of restoring the completely dead Coreldraw file after crashing this program.

We will consider the most unpleasant situation.
So: you are working in Coreldraw, autosave and backups are on, it takes 2 or 5 hours of work, and then suddenly Corel falls. In our situation, Corel, agonizing, grind all the pace and backups. You only have your file, when you open it you see a blank sheet.

I must say, you should not download all sorts of utilities, such as CDR repair tools and the like. If they can restore the file (in my case they could not), they will not allow it to be saved without purchasing a license.

Solution # 1 (for lucky ones): If you are lucky, your name.cdr file will open with any zip-enabled archiver. If the content folder is present, and riffData.cdr is, you are saved. Create a new file, save, open as an archive. Delete the content folder in the new file and replace it with the same name from your broken file. After that, this new file is opened in the Korela. Voila, you have recovered the file.

Solution # 2 (when it‚Äôs really bad): Trying to open the Corel file with the archiver, you get a message that this is not an archive, or another error. In this case, it saves exactly the approach to the file as an archive. We take one of the archiving utilities (I used free ZIP Repair ), and edit the archive. After that, I have to perfectly work out the method from solution No. 1.

I hope the information will be useful, I returned 6 hours of work.
