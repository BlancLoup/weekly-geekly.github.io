Habratulz plugin for Firefox 3.6

I found the Firefox plugin in the article " Habratulz: now and under FF 3.5+ ". With it, in 2 clicks and without intermediate pages you can go to all the main Habr pages and insert HTML tags through the context menu, which is especially useful for comments for which there are no tags on the insert buttons website.

It should be understood that the author did not manage to launch the second half of the plug-in ‚Äî habra editor ‚Äî just what is often required for writing comment tags. Deal is reparable, done. (In FF 3.0+ it should work everywhere, but tested on FF3.6 WinXP.)

From here: habratools092.xpi [16.9 kb], version 0.92 .

I didn‚Äôt really like the need to keep the button on the toolbar and the 2-level button itself in the plugin, so I redid the menu into an almost one-level menu and added it as an alternative to the context menu. Now the button in the toolbar is not necessary, but if there is a desire, then how to install it is written in habrahabr.ru/info/help/widgets .


In short, after installing the plug-in, go to its settings ( Tools-Add-ons-Habratulz-Settings ) and write your nickname in the " Habrayuzer " field. If you want to put a button on the toolbar (where the address bar and buttons are), then the following clicks are performed: View-Toolbars-Customize-Open Your Media Center - drag the mouse to a convenient place in the toolbar. It will look something like this:
Buttons, compared to the old layout, are placed so that the mouse movements have to be made smaller - the most popular ones are higher.

If we go to the Habr pages and click with the right mouse button (better in the input field on the cursor or on the selected text), we see at the bottom of the Habrakod menu and something like this.

(click to see the whole menu)

It can be seen that, getting into a not very convenient 2-3-story menu, we have fairly convenient tag templates ‚Äî links, images, special characters, and even among them there are the same links (with a transition in the same window). Of course, the completion of insertion of templates is not perfect, but to make it so that the cursor becomes in the right places is a matter of time and desire.

Added setting - whether the context menu ‚ÄúHabrakod‚Äù always appears on any page or only on the Habr website. By default - only on the site. If you need to use habra editor and clicks on links to habr from any page and window, in Tools-Add-ons-Habratulz-Options turn on the check box " Always in the context menu ".
