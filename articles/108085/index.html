Unity based on the compiz engine is available via PPA for Ubuntu 11.04 Natty narwhal

Already, users of ubuntu 11.04 can test the unity interface implemented as a plug-in to all known compiz. Therefore, to launch this interface, you do not need to reboot into a netbook edition session, it is enough to enable the compiz plugin



What is immediately noticeable is that the launcher becomes translucent:



Of course, not everything is working smoothly, but this is only the early beta or even the alpha version of the new interface.
But this move is pleased with the fact that since unity has become a plugin, all the other compiz tags also work in it.



For example, expose!

To install, open the terminal and ...

sudo add-apt-repository ppa: unity / ppa
sudo apt-get update
sudo apt-get dist-upgrade

And if there is still no compiz manager, then ...

sudo apt-get install compizconfig-settings-manager
