Another approximation by polynomial of a function of several variables

In problems of interpolation of functions by given values ‚Äã‚Äãof a function for a given set of arguments, the formula for approximating a function by a polynomial is widely used, which at given points coincides with the values ‚Äã‚Äãof the function under study.

This type of approximation is widely used in scientific calculations.
Example: it is very expensive to calculate some function for each value of the arguments (there are many of them, let‚Äôs say N) - therefore a table of values ‚Äã‚Äãis built and, if necessary, the function‚Äôs value is obtained at a certain point - it is interpolated by a label. Of course, the initial construction of the table and the interpolation procedure (N times) must be ‚Äúcheaper‚Äù than the exact calculation of the function itself N times.


We generalize this formula to the case of a function of several variables.


The scalar product of vectors will be calculated taking into account the squares of the variances of the values ‚Äã‚Äãof the arguments Dx


  • in order to take into account the equivalence of each coordinate of the argument, we calculate Dx [k] = M2 [k] - M1 [k] ‚àó M1 [k]
  • <A; B> = SUM A [k] * B [k] / Dx [k]

Algorithm



1. For given vectors Xi, calculate the vector Dx of the squares of the variances of the argument values
2. For the vector X, we find the p nearest points X1, ..., Xp, where the distances between the points are calculated taking into account the squares of the variances of the argument values
3. Calculate the value of the polynomial P (X) = SUM Yi * P <X-Xj, Xi-Xj> / <Xi-Xj, Xi-Xj>, where the scalar product is calculated taking into account the squares of the variances of the arguments values
