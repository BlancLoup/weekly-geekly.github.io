Opera-9.5 will fully support Google Gears

According to a recent press release , Opera Software promises to add full support for Gears technology in the entire line of 9.5 browsers. This will turn the browser into a full-fledged platform for web applications that run on both desktop and mobile devices. According to Jon von Tachner, this step is a logical continuation of Opera Software‚Äôs strategy of creating not just a browser, but a platform that can fully satisfy both the requests of web application developers and the users of these programs. In addition, the arrival of Gears in the mobile version of the Opera browser opens up a new era of developing applications for mobile devices. Naturally, debugging of all applications will be done using Dragonfly.

As for Gears itself, everything is also interesting here. This is a platform-independent development by Google, which is the AJAX-API. With it, you can create web applications that can work offline. According to Wikipedia , the thing is in a special plugin that forces the browser to work with the local page cache (based on SQLite), periodically synchronizing the cache with an online source. Google Gears is distributed under the free BSD license. At the moment, Google Gears is only supported in Firefox and Internet Explorer browsers.
