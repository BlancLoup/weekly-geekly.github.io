OpenMW 0.6 released

Let me remind you that OpenMW is an attempt to create an open source cross-platform game engine that supports the data formats of the game The Elder Scrolls 3: Morrowind. On Habr√© about him already wrote when the first public version.

Since then, the project has grown, and this is what appeared in version 0.6:
  • A full-fledged GUI based on MyGUI appeared , which allows you to recreate Morrowind's GUI with all its windows and other
  • Fully completed Monster Script - a scripting language that is designed to implement game logic
  • Solved problems with playing music in windows

I would like to wish the developers good luck in bringing the project to a playable state, because Morrowind is a legend, and I really want to play it on a new, stable, open and beautiful engine :)
For more information and screenshots, please visit the project website .

By the way, the project is also interesting because it is written in the D language. It also uses the Ogre graphics engine and the Bullet physics engine.
