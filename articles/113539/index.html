Canonical has launched a list of computer components compatible with Ubuntu.

Today, Canonical has opened for public access the largest list of computer components compatible with Ubuntu, containing more than 1,300 certified components from 161 manufacturers.


Having an initial goal to help manufacturers of system units assembled (ODM - Original Design Manufacturers), the list will also be useful for those who like to assemble a home computer for themselves, and it will help a lot when choosing a new hardware for computers with Ubuntu on board.
