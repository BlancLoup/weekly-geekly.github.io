JRebel 3.6.2 released

Zeroturnaround has released JRebel 3.6.2 this week! Download it here .

Here are some of the most exciting new features:
  • Hibernate validator
    Now you can add / edit / delete Bean Validation annotations in bins as well as in parent bins (that is, composite validation is supported). You can also create and edit your annotations.
    Many users have requested this functionality , so now they should be happy.
  • JBoss Seam 2.x Improvements Some users have been waiting for a long time - adding new components to the components.xml file.
  • EJB support for JBoss 4.2 / 5.1 and Glassfish 3.x. Now you can add new @Stateful / @Stateless @Stateful to the application and immediately start using from other working beans.
  • Improve debugger integration . Now JRebel handles some complex cases better to make work with the debugger as smooth as possible.

A full list of changes in version 3.6.2 compared to 3.6.1 can be found here .
