Digsby is now in Open Source, available on GitHub

I want to share with the community good news. The best (in the humble opinion of the author of the post) multiprotocol client for instant messaging under Windows is now available in Open Source.

For those who still do not know: the program is a multi-protocol messenger, with support for ICQ, Jabber, AIM, MSN, Yahoo, and Google Talk. There is also a very convenient integration with social networks Twitter, Facebook, MySpace, LinkedIn, and the mail service Gmail from Google. However, there is a regular POP / IMAP email client.

In other words, for me, this is the best alternative to the notorious QIP, which was safely removed after the installation of its next guard.

Official site of the project: digsby.com
Project source code is available on GitHub
