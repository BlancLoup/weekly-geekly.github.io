0.5 million mobile books

books.google.com/m - this is exactly what the address of the page looks like on which all books previously available in Google Book Search are optimized for reading on small screens, for the most part this is done for iPhone and Android device owners.

18 categories, almost 1.5 million books for residents of the North American zone and about 0.5 million for all others - these are the size of the mobile library Google Books.

One of the interesting features of the archive, which can always and everywhere be carried with you, is the method of digitizing paper media - it uses Optical Character Recognition technology, which extracts text from a book in such a way that a mobile device does not see the difference between a web page and a digitized text, while dividing it into a heading, a body, and even digesting footnotes. Clicking a couple of times on any line or paragraph from the book you can see the scanned original - this is done to prevent digitization errors - if the reader sees the wrong text, then double-clicking on any line will see how this text was printed in the book.
