Windows Azure + WPF + Wikipedia = Wikipedia Explorer

Dot Net Solutions, together with a team of Microsoft evangelists from Redmond and Monochrome, have released a new version of the WPF application Wikipedia Explorer actively using Windows Azure.

What is the Wikipedia Explorer?
This is an application for visualizing links between Wikipedia articles, built on the powerful technology of Windows Presentation Foundation.

And here Windows Azure?
The original version of the application loaded the information from Wikipedia on the fly. This turned out to be a rather slow process. Now, the load can be divided into a large number of powerful servers, which significantly increases the speed. Currently 50 servers are used. The operation, which would take 6 months on one machine, takes about 4 days in Windows Azure.
