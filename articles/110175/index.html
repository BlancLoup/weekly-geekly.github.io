Obtained root on the newly released Google Nexus S

Only sales of a new device from Google-Samsung started, as root was obtained and alternative Recovery was installed:

(clickable)

If you have already become the happy owner of this phone, then here is a rutovan manual .

From my free translation of her:


1. Reboot to bootloader / fastboot mode (Volume up + Power button or by using the 'adb reboot bootloader' button)

2. We carry out:

fastboot oem unlock

3. Then, install the recovery from here:

koush.tandtgaming.com/recoveries/recovery-clockwork-herring.img

fastboot flash recovery recovery-clockwork-herring.img

4. Next, boot into Clockwork recovery and install Superuser zip:

forum.xda-developers.com/showthread.php?t=682828

5. To install the boot boot image, download it from here:

koush.tandtgaming.com//samsung/herring/rootboot.img

fastboot flash boot rootboot.img
