Big update: Evernote for Windows and Mac got links to notes, copies of notes and more.

We made a combined release of new versions of Evernote for Windows and Evernote for Mac with a number of features that our users have repeatedly requested. All features described below are available for both platforms, unless otherwise noted.

To install the latest version:
  • check for updates;
  • download Evernote for Mac from the Mac App Store or from the Evernote website ;
  • Download Evernote for Windows here .

Links to notes


Not so often a single opportunity can significantly improve the work with Evernote for all users. It is possible that at first glance it did not seem so to you, but references to notes can definitely be attributed to the number of such significant innovations. As you may have guessed, this feature will allow you to link notes, located in various places. To create a link, right-click on a note and select Copy link to note or find it in the Note menu. After that, the link will be placed on the clipboard.

You can insert this link almost anywhere - in other notes, calendars, to-do lists, other applications, etc. By clicking on this link, a note will always open. If you press it while holding the Shift key in Windows or CMD on a Mac, the note will open in a separate window. We have already added support for links to notes in the web interface, as well as Evernote versions for iOS and Android, so you can feel free to open links in these clients and get to the right place.

Links to notes also work in joint notebooks, allowing authors to compile tables of contents and manuals for navigating notes.



Create links for multiple notes
You can select several notes, click on the right mouse button and select the item Copy link to note . When inserting you will receive a list of links. This feature will be very useful when you compile a bibliography, table of contents or a list of quotes.

Link Benefits
Have you ever wanted to create a reminder for a particular note? Now you have this opportunity, just paste the link to the note into your calendar application. At the right time, a reminder will appear on the screen along with the link, and you just have to click on it to open the note.

But that's not all, we hope that links to notes will open up many new great features and ideas for our developer community. It will come at an opportune time for the developer competition now under way.

Copies of notes


Now you have the opportunity to make copies of any notes and place them in the right places. You can make a duplicate note in the current notepad or place it in another one, as well as copy the note from the shared notebook to your own account and vice versa (if allowed by the owner of the notebook).

To copy a note, right-click on it and select Copy Note (Windows) or Copy to Notepad (Mac) from the context menu.


Creating templates
The copy function opens up the possibility of creating note templates. If you already have a note with a specific recording format that is perfect, for example, for storing accounting information, you can simply copy it. In the window that appears, you can specify whether to save for a copy of the label and the date of creation / change from the original notes.

To do this:
  • in Windows: after right-clicking a note, open the Copy Note submenu and select Notepad ... ;
  • Mac: Click the right mouse button, scroll to the bottom of the list of notebooks and hover over Options .

Fragment View Mode


In this update, the mixed viewing mode has given way to more information-rich fragments. Fragments allow you to display the maximum amount of useful information about each note without opening it. If the note contains only text, then the full width of the text will be displayed on the corresponding fragment in the list. If there are images and text in the note, text with a thumbnail will be displayed on the fragment. For notes containing only images, a large thumbnail and caption will be shown. This view mode not only allows you to get more information about the contents of the notes, but also makes it easier to navigate through them.



History of viewed notes: Back / Forward buttons


Navigating your Evernote account just got easier. We added convenient Back / Forward buttons that allow you to navigate through recently viewed notes. This feature extends to viewed notes, made search queries, visited notebooks, and more.

Pressing and holding one of the arrows, similarly to the same buttons in the browser, you will see a list of items that can be accessed without bothering yourself with unnecessary clicks.

Back / Forward buttons also greatly simplify navigation through links to notes. If you are viewing the note in a separate window and clicking on such a link, the corresponding note will open in the same window. To return to the previous note, you just need to click on the Back button.

All functions associated with the notes are now moved to the new corresponding menu item. There you can copy or move the note, share it and much more.

Windows: other improvements


The following are some of the features that were implemented only in the update for Windows. Options only for Mac on the way.

New data export options
Combining the function of links to notes with the ability to export notes to HTML, we can now offer you the export of notes as separate web pages with a table of contents file. Thus, as a result of export, you can get a practically full-fledged site. To try out the new version, select the notes you want to save to HTML, right-click and go to Export Notes . Then select the option Export as multiple web pages .

Additional improvements
  • Drag and drop. Now you can drag text and images from the note to the desktop and into other applications.
  • Disable PDF preview. You can right-click on the PDF file in the note so that it appears as a regular attachment.
  • Improved handling of text styles. We improved the handling of font styles.

Mac: VoiceOver support


We have improved Evernote compatibility with Apple VoiceOver features. You can activate VoiceOver via System Preferences> Sharing.
