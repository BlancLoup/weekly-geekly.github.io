The network posted a dump Android 4.2 c Nexus 4



Already on November 13 this year, the Google Nexus 4 smartphone will go on sale. But applications, wallpapers, ringtones for this device are already available, several resources at once laid out a dump of Android 4.2 with Nexus 4. True, this dump is not from the final software version for the specified smartphone, but only a pre-release.

You can download a dump, for example, from here or from here . The dump contains ringtones, wallpapers and a couple of videos, one of which shows a toy Android in space.

It is clear that here you can find the latest system applications for the keyboard or camera. Some of the applications may not start on previous versions of Android, or run with an error. So, before trying out all this in work, it is advisable to backup your data using ClockworkMod or TWRP.

Some applications can not be simply installed, they will have to be flashed. Of course, here, too, need to be careful.

By the way, Google has not yet laid out the source code for Android 4.2 Jelly Bean, so third-party developers can use this system dump for reverse engineering.
