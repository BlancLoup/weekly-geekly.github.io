Access Ruby 1.9 documentation from OS X Dictionary and Spotlight

Priit Haamer Estonian Ruby developer has published Ruby 1.9 documentation in dictionary format for the Dictionary application. From this page you can download the Ruby dictionary . Ruby on Rails documentation is available in the same format.

Installation


Unzip the folder in ~ / Library / Dictionaries if you want to put dictionaries only for yourself. You may have to create this folder. If you want to put dictionaries for everyone, then unpack them into the folder / Library / Dictionaries.
In order for the search for these dictionaries to work from Spotlight, open the Dictionary.app settings and drag the required dictionary up because Spotlight searches only the first dictionary from the list.
