ReSharper 7 supports Visual Studio 2012 and simplifies the development of Metro applications.

A public release of ReSharper 7, a new version of the well-known extension for Microsoft Visual Studio, took place.

ReSharper 7 provides full support for Visual Studio 2012, while maintaining compatibility with versions of Visual Studio 2005, 2008 and 2010. In addition, ReSharper 7 contains extensive functionality that is important for teams developing applications for Windows 8 and WinRT - the latest operating system and runtime environment from Microsoft.

ReSharper 7 is also useful for those developers who are not going to switch to Visual Studio 2012 yet. Among the new functionality available in all supported versions of Visual Studio is a set of code inspections and automatic fixes; new refactorings, including Extract Class to simplify the code; New ways to visualize and study the code; as well as contract annotations that help adapt code quality analysis to agreements adopted in a specific development team.

The list of major innovations in ReSharper 7 is as follows:
  • Full integration with Visual Studio 2012 while maintaining support for the three previous versions of Visual Studio: 2005, 2008 and 2010.
  • Extensive feature set for developers looking to build Metro apps for Windows 8.
  • New code inspections and annotations to improve the quality control of .NET application code.
  • New automatic refactorings code, including widely demanded Extract Class refactoring to simplify the code.
  • Additional code learning opportunities associated with visualizing hierarchies of polymorphic members of CSS classes and styles.
  • New and improved code generation commands.
  • Support for working with the INotifyPropertyChanged interface, which simplifies the work of developers of Windows Forms, WPF and Silverlight applications.
  • Remove unnecessary references to assemblies across the entire solution.
  • Advanced tools for web developers, including improved support for the JavaScript language, as well as basic support for ASP.NET 4.5 and ASP.NET MVC 4.
  • Support for two additional development technologies from Microsoft: LightSwitch and SharePoint.
