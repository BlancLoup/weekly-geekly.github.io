Buildy: HTML5 sandbox for adults

Buildy is a multiplayer online game in which you can build anything, without any specific goal, just like that. This is a three-dimensional isometric world in which you can use ready-made objects from the library (there are more than four and a half thousand of them!) Or draw your own in the editor. The game has a chat, with which you can coordinate your actions with other players. In addition to Goon World III, which is common to all of the world, in which all new players fall by default, you can create your own private sites, access to which is possible only by invite.
