Published source codes F # 2.0 under license Apache 2.0

Microsoft today released the source code for the F # 2.0 compiler and a set of other tools for this language called F # PowerPack.



The new version of F # 2.0 was released in April 2010. The language specification is available at this link (.pdf, 3 Mb). In August 2010, an important update of the second version of the compiler was released.



During the PDC10 conference, a report was made on the future of the F # language. You can see it here .



Download the F # tools or get more information on the official website fsharp.net . For developers, Don Syme's WebLog on F # and Related Topics blog will be useful.



Download the source code of the compiler and the toolbox from this page on codeplex.com.
