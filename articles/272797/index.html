New free courses of virtual academy Microsoft Virtual Academy, December 2015



Today we have prepared for you a pre-holiday review of free courses of the virtual academy of Microsoft MVA . As before, new materials from Microsoft experts continue to be released in order to benefit the IT professionals and software developers.

Hit! How to teach a child to program
The long-awaited and unusual course examines a variety of Microsoft technologies (and not only) that can help you captivate your child with computers and programming. Without pretending to methodological completeness and an exhaustive story about technology, this course provides the right links and ideas, thanks to which you can spend many hours with the child at the computer, studying and creating with it.

New! Introducing Microsoft Azure Cloud for Students and Pupils
The videos are intended for students and schoolchildren who want to make the most of the cloud offerings within the DreamSpark / Imagine Access program. It tells you how to access the Azure portal, how to deploy your site based on the content management system, as well as how to start developing your sites from scratch based on the bootstrap templates in WebMatrix, and on ASP.NET in Visual Studio.

New! Office 365 Development Overview
1 billion Office users worldwide spend an average of 2 to 3 hours a day working with Office clients, with more than 400 petabytes of data being hosted by Office 365. If you are a web developer, then the Office 365 platform gives you the opportunity to offer these users have their own business solutions, automate day-to-day tasks and even create specialized solutions. You will learn how to create solutions in contextual mode directly for Office 365 using the application model. In addition, you will learn how to call the Office 365 APIs from stand-alone web apps and mobile apps to use services such as OneDrive for Business, Mail, Calendar, People, and Sites.

Hit! Microsoft Azure Basics
Are you an experienced IT specialist? Want to learn more about Microsoft Azure? Join Bob Tabor. As part of this first course, you will learn why there is a Microsoft Azure platform and what problems it solves for young companies and Fortune 500 companies. You will also learn some general concepts related to the implementation of Microsoft Azure, as well as common administrative tasks.

Hit! Express immersion in JavaScript development
If you are developing in JavaScript, this course will help you understand what awaits you in the near future. We tried to give a brief overview of new trends and modern possibilities in developing applications for JavaScript ‚Äî be it complex web solutions, Windows applications or cross-platform applications, or games. We will also look into the future a little bit and look at the possibilities of the next version of the JavaScript standard - ECMAScript 6 and touch on new areas in which JS can be applied today.

A fascinating introduction to functional programming on F #

Functional programming is a different approach to programming that is very different from the imperative we are used to, and which is now rapidly gaining popularity among programmers. F # is the most popular (according to the TIOBE rating) functional programming language that can be used in conjunction with other .NET languages ‚Äã‚Äãto significantly improve the programmer‚Äôs performance and level of happiness. In this course you will learn the F # language and the concept of functional programming, as well as, we hope, learn to get much more pleasure from the process of creating programs.

New! Windows 10: deployment, management, security
The first technology course on Windows 10 for IT professionals provides an opportunity to dive into the details of deploying a new OS across the enterprise, managing with MDM solutions, ensuring the safety of Windows 10 devices and preventing the leakage of corporate information. The course covers the new configuration features of Windows 10 (provisioning), changes in the deployment tools of the Assessment and Deployment Kit, and features of the Upgrade in-place. You will learn what benefits the inclusion of Windows 10 in the Azure Active Directory realizes, what settings are available using MDM, get acquainted with the identity management system of the new generation Microsoft Passport, get an idea of ‚Äã‚Äãthe Windows Hello biometric authentication subsystem and much more

Hit! Technical Overview of Windows Server vNext
As part of the course, independent experts talk about the most interesting technical updates and improvements to the next version of the server operating system from Microsoft. You will get acquainted with the new features of Hyper-V, learn what Windows containers and Nano Server are, how it works and in what scenarios Storage Replica is used, how you can build Windows-based storage using local disks, what's new in RDS services.

Using Docker Containers in Azure IaaS Cloud
The course covers the issues of the device management system Docker containers and its use in IaaS-type cloud systems. You may need a quick overview and comparison of various virtualization, emulation and containerization systems, such as KVM, Xen, OpenVZ, LXC and others. The course also highlights the practical applications of Docker and its main functions.

Linux Workload Management on Windows Server and System Center
Do you manage both Linux and Windows Server workloads? As a Windows Server administrator, do you also have to deal with Linux workloads? If yes, and you want to reduce the number of administrative tools used, this course is for you! Our expert, Oguz Pastirmaci, will tell you how to manage Linux workloads in System Center 2012 R2 using a single window.

At the end of our review, let me remind you that on the website of the Microsoft Virtual Academy you can find even more online courses and pick up materials on the topic that interests you.
