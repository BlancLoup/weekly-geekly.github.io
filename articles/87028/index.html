Thunderbird 3.1 Beta 1 released


A new test version of the popular email client Thunderbird has been released.

Thunderbird 3.1 is based on Gecko platform version 1.9.2 and includes the following improvements:

  • A lot of improvements for IMAP;
  • A number of fixes for Smart Folders, message filtering and attachment processing;
  • Many minor improvements to the program interface;
  • The download manager is now available through a separate menu item;
  • Improved stability and optimized memory consumption.
