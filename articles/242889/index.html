Microsoft has released EMET 5.1

EMET updated to version 5.1. The new version fixes compatibility issues of the fifth version with third-party applications, including MS Internet Explorer, Adobe Reader, Adobe Flash Player, and Mozilla Firefox. Fixed incompatibility of EAF + settings with Adobe Reader sandbox (Protected View Mode), which led to false positives when opening pdf documents for which the protected view mode had to be enabled. In addition, processes and the EMET service are now launched as 64-bit ones. Download the new version at this link .
