Php internals


  • [RFC] Allow a trailing comma in function calls - It is proposed to allow the use of a comma at the end of the function argument list: 

    var_dump( $whatIsInThere, $probablyABugInThisOne, $oneMoreToCheck, );
  • PHP 7.2 has not yet come out, and work on the following versions is already well underway. For example, in 7.3 , the optimization of sparse conditional constant propagation is implemented . This allows the interpreter to simplify the following function: 

     Class C { public $i; } function fn(int $x) { $c = new C; $c->i = 1; if ($x) { $a = [1, 2, 3]; } else { $a = [3, 2, 1]; } return $a[$c->i]; $c->i++; return $x; }

    to this: 

     function fn(int $x) { return 2; }

    We are waiting for news about JIT, which is also being worked on .
