Plotagon is his own director



Long ago, when the trees were big, I was very fascinated by 3D animation. But as time went on, new concerns appeared and this passion was abandoned. The other day I stumbled upon a very interesting project that made me remember my childhood and spend half the day in a rather useless, but very exciting experience.

I want to talk about the project plotagon . It allows you to write a script for virtual actors. Of course, this program will not help create a masterpiece of animated art, but the process itself is very exciting.

At the very beginning we choose the location where the action will take place.


Then we add actors.


It is a pity that no more than two actors can participate in one scene. After adding characters, we can choose their location on the stage. You can choose not any, but only a limited option from the list.


After that we can start writing the script. You can make the characters speak the text with the desired intonation. There is no Russian language, there is only English. I typed in Russian Latin words, it turned out rather clumsy, but funny.

The set of actions is quite limited: moving around the stage, gestures, laughter, hugs, and of course a hook from the right. The created video can be uploaded to the site of the project itself and on youtube. This is what the sick imagination can do in ten minutes.


The program is in beta testing and is free for download. Although additional scenes and actors will have to buy. The price is not so great. And while the possibilities are pretty limited, I think some will spend their time on it. The pleasure of creating your own animated film, albeit a bit odd, is worth it.
