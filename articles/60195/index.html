easyStore AH340-UA230N - Acer's affordable home server for only $ 400



Acer invades the home server market. And their first such product was easyStore AH340-UA230N with integrated Windows Home Server. The product is equipped with an Intel Atom 230 processor and 2GB DDR2 RAM. Along with the storage capacity of 1 TV, you also get three additional 3.5 "hot swap connectors with the ability to install storage media up to 7TB.

Add to this some DLNA 1.5 certificate and a $ 400 price tag - and it will prove to be a worthy competitor to the HP MediaSmart line.

According to media reports, the possibility of buying will be presented in the very near future.



Detailed specifications:
  • Intel Atom 230 processor (1.6 GHz)
  • Windows Home Server Operating System
  • 2 GB DDR2 RAM
  • 1 TB hard drive
  • Three empty 3.5-inch bays for additional hot-swappable hard drives
  • 5 USB 2.0 ports and 1 eSATA connector
  • 10/100/1000 Mbit / s Ethernet
  • Dimensions - 211x198 x180 mm


Advantages:
  • Price
  • Hot Swap baskets
  • Design

Disadvantages:
  • Exchangeable Intel Atom 230 processor
