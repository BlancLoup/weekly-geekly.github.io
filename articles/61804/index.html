Virtual device with Yii for VMware

A virtual device (VMware image) with Yii 1.0.6 and the following components is now available:
  • PHP 5.2.3 (in fast-cgi mode)
  • Lighttpd 1.4.7 (mod_rewrite and fast-cgi included)
  • Yii 1.0.6 (including examples)
  • PostgreSQL 8.2.4
  • phpPgAdmin 4.1.3
  • Linux kernel 2.6.17.7
This is a full pre-installed server. This means that the operating system and programs are already installed and ready to use. This kind of server can be opened using VMware Server or Player, you can immediately turn on and use them.

The Yii Appliance also contains network tools like ssh, dhcp, scp, etc. The virtual device contains 2 disk images: the server disk is mounted only for reading, and the data disk can be increased up to 4 gigabytes.

How to use these images

  1. Download and install free VMware Server .
  2. Download Yii 1.0.6 Virtual Appliance (16.9 MB) .
  3. Unpack.
  4. Download the image to VMware.
  5. To shut down the server, click the stop button.
The purpose of this device is only a demonstration, it can NOT be used as the final working system.
