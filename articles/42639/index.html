MDC: a beta release of multiprotocol messenger for Linux!

More recently, on Habr√©, we talked about the beta release of the new multi-protocol messenger MDC under Windows OS from the Netstream company.

We received more than a hundred reviews and suggestions from you, fixed a number of bugs and, most importantly, prepared a beta release for Linux (32/64), which we want to present to you today.




First, a little about MDC


MDC is a multi-protocol messaging client with support for Windows, Linux, MacOS X and ICQ, Mail.Agent, Jabber, AOL protocols (we will gradually expand the list of protocols). MDC version 1.0.2.0 is now available.

What has been done since the previous beta release


  • Fixed a problem with running under some versions of Windows XP.
  • Authorization under Gtalk for accounts in the domains li.ru and liveinternet.ru (in the future, the restriction on domains will be removed, which is important for those who use Google Apps).
  • Proxy server support (in the future, autodetection of settings will also be implemented).
  • Closing the tab by clicking the middle button on the tab.
  • Displays the last conversation in the message window (displays the last conversation stored on the MDC server).
  • Display in the window the history of smiles and links.
  • Jabber: shutdown processing due to logging in from another computer.
  • Login window button in the taskbar.
  • Expanded interface for working with groups.
  • Bugfix arbitrary disable ICQ.

And so on ... You can always look at the full list of additions and changes on the Wiki page of our bugtracker .

Overview of MDC Benefits


Storing the history of correspondence on the server. No need to think about where you were: at home, at work or with friends, you can always find the necessary information, having access to the Internet. I would like to immediately note that this ‚Äúopportunity‚Äù can always be turned off in the program settings.

Portability Beta MDC for Microsoft Windows (2000 / XP / Vista, 32-bit) and Linux (32/64) is available today. In the coming weeks we will release a version for Mac OS X (Intel). After a month and a half, the web version of MDC (lite-client version) will be available.

Support basic messaging protocols. Now MDC supports protocols such as ICQ, AOL, Mail.Agent, Jabber (plus Gtalk and Ya.Online). In the near future, a beta version of MDC will be released with support for the MSN protocol. Of course, with each new release we will try to add the protocols most requested by users.

Merging contacts (metacontacts). This is another useful feature that allows you to combine the contacts of the same person from different messaging systems, which allows you at least not to think about what protocol your interlocutor uses.

Auto update. MDC allows you to automatically update the program. You do not have to download and install the client every time to take advantage of new features, improvements and corrections of MDC. And in the case of a change in any protocol, you simply won't notice it. Auto-update is available from version 1.0.2.0, so if you have already downloaded MDC before, you need to download the client again using the link indicated at the end of the topic.

What is expected in the next beta release (~ in a month)


  • Mac OS X (Intel) and FreeBSD version.
  • Font settings.
  • Merzh contacts by dragging into the contact list.
  • Receive and send files.
  • Editing personal information.
