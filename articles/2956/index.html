Interfaces of applications in the style of Microsoft Office 2007 are available to developers.

FOSS Software has announced the release of the Elegant Ribbon 1.0 product - a set of .NET Windows Forms controls that allows embedding RibbonX interface into its own application, which appeared in Microsoft Office 2007 and recently licensed by Microsoft for third-party developers.

Elegant Ribbon components are written entirely in C # and can easily be built into .NET applications. Most of the necessary tasks are solved with the help of an intuitive Windows Forms Designer. Now developers can equip their own applications with attractive and functional interfaces in the style of Microsoft Office 2007.
