Microsoft began to pay developers for applications for Windows 8 and Windows Phone 8

Microsoft launched a program to encourage developers to create programs for their new operating systems. As a result, you can earn up to $ 2000 for 20 applications. Of course, there are also conventions.

Starting from March 8 to June 30 of this year, everyone has the opportunity to receive $ 100 as an amount on a virtual Visa card for 20 applications for Windows 8 and Windows Phone 8, which will be approved for placement in the store. Requirements for applications remain common - as long as they are new and have more functionality than just launching a web page. In this case, a slight alteration of the old application and a request for a new statement is punished with a ban.

In total, you can send 10 applications for Windows 8 and 10 for Windows Phone 8. The action will end either at the designated time or when a critical mass of ready-made programs is recruited, the number of which should be 10,000.
