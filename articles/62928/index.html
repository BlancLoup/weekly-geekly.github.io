Notepad ++ as a text editing tool

I think many people use or use the text editor Notepad ++ . This editor, in addition to its basic functionality, does not have a bad set of plug-ins. So in the last build, a remarkable module called NppNetNote was found.

At first glance, the functionality of this module was not obvious. The settings window was rather scant, there was no mention of this module on the official website of the editor.


As you can see, the settings are simple: host and port. The plugin can create a server on the selected port.
The most interesting begins after someone else connects to the created server. When connecting, the client creates a new tab with the same name as the file for which the server was opened. Also, the client sees the entire contents of this file. But the most pleasant thing is that this file can be simultaneously edited by two people in real time, while one will not interfere with the other (unless they write in one place), and the text typed by each is highlighted in a different color.
The work of this plugin has been tested in the workplace in the process of writing a specification, very convenient and unlike Google Docs full realtime. True, he sometimes works is not stable, but it can be forgiven because has a version of only 0.1. I think this module has a future and I hope in the future it will be more functional and stable.
