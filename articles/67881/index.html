Overview of ASP.NET and Windows Forms Components

Free components



Chart Controls for .NET Framework

After installing Chart Controls for the .NET Framework and Chart Controls Add-on for Visual Studio 2008 (in that order), a new Chart control will appear in the Toolbox.

The control has a lot of functionality and a really customizable interface. There are demo applications for ASP.NET and Windows Forms. You can read more here .

Chart Control is based on Dundas version 5.5 Chart Control. In Visual Studio 2010, this control is included.
