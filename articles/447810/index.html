Analytics for Azure DevOps Services is now publicly available.

Reports are an important opportunity for Azure DevOps users who rely on Analytics (Azure Analytics Service) to make data-based decisions.

Today, we are pleased to announce that the following Analytics features will be included in Azure DevOps Services at no extra charge. In the near future, customers will see these changes in their accounts.



Analytics features that are now available in Azure DevOps Services


  • Analytics Widgets are custom modules that display data on a dashboard and help track progress.
    1. Burndown and Burnup - monitoring the progress of work in certain areas for a certain period of time.
    2. Cycle Time and Lead Time - visualizes the product development cycle in your team.
    3. Cumulative Flow Diagram (CFD) - tracking work items as they pass through various states.
    4. Velocity - tracking how the team adds value over several sprints.
    5. Test Results Trend - tracking test trends, identifying failures and test durations for one or more pipelines (Azure Pipelines).

  • In Product Experiences - Analytics works in DevOps Azure and outside the dashboard, which displays data and analytics.

We will continue to offer Power BI Integration through Analytics Views and direct access to OData Endpoint in preview mode for all Azure DevOps Services clients. Additional details on the pricing model for Power BI and OData integration should be expected by June 2019.

Current Azure DevOps Services customers who have the Analytics extension from the Marketplace can continue to use Analytics, as before, and they do not need to perform any additional actions to get Analytics. Thus, we will condemn the expansion of Analytics from the Marketplace for hosted customers.

Azure DevOps Server 2019


For Azure DevOps Server, the Analytics feature will remain in Preview mode as an installable extension on the local marketplace and will be made public in the next major release.

Azure DevOps Analytics is the future of reporting, and we will continue to invest in new features provided by Analytics. To learn more about Analytics and the opportunities it currently provides:
