Firefox 3b2 and huge humanoid robots

Congratulations to all users of firewall with the release of the second beta of the third version. Those who are familiar with English can get acquainted with all the innovations on the official page .

What was added to the next beta?
The address entry window has undergone dramatic improvements - autocompletion with full-text search by title, address, or tags in history and bookmarks. New, expanded view of the list of autocompletion.


Smart Bookmarks is a virtual folder that allows you to view recently added items, as well as frequently visited pages.

With the help of the bookmark manager, you can search by all bookmarks, history, tags.

The download window now shows where the file was loaded from.

Integration with web applications such as mail. For example, the mailto: link can be transferred to your favorite webmail interface. (This will be possible with other protocols)

Of internal changes, there are numerous fixes for memory leaks and a new garbage collector . Developers are working on increasing the speed and reducing the use of memory, as well as its fragmentation.

You can help the project: download the latest beta, and, after wandering around the sites, send a description of the errors noticed at feedback.mozilla.org People with a sense of taste are waiting in the discussion of the new icon theme.
