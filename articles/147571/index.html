Moqups - a new service for creating layouts UI

As the developers themselves characterize the service, Moqups is "an elegant HTML5 application for creating layouts, user interface concepts, prototypes, depending on how you call it."



For Chrome, there is a separate application .

Currently more than 60 item templates are available, export to PDF and PNG, keyboard shortcuts, grouping objects, snapping to grids, and so on.

The project is under active development and is open to criticism, help and suggestions.
