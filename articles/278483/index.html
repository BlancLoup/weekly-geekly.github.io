Special folders

When I was programming in VBScript, I used ‚Äúspecial folders‚Äù, these are folders like ‚Äúmy documents‚Äù and the like, I used access through the Shell.Application object passing the constant of the desired folder to it.

the same way works in powershell 
$a = New-Object -ComObject Shell.Application Get-ChildItem $a.NameSpace(0x21).Self.Path

can be shortened to a single line if you like it more: 
 Get-ChildItem ( (New-Object -ComObject Shell.Application).Namespace(0x21).Self.Path)


And if you want clarity then: 
 $cookies = 0x21 Get-ChildItem ( (New-Object -ComObject Shell.Application).Namespace($cookies).Self.Path)
