ModularGrid - modular grid in the browser (and not only the grid)

ModularGrid is a small javascript program (about 30 KB) that facilitates the work of the HTML-maker.


Main features of ModularGrid


  • showing guides over the page
  • display modular grid over the page
  • display of a separate font grid, horizontal or vertical modular grid over the page
  • display image layout over the page
  • Resizing the browser window under the permissions specified by the coder
  • change the transparency of the modular grid and image layout



Demo


Try modulargrid in action - see project demo page .

How ModularGrid differs from analogs


  • works in all browsers
  • contains a lot of tools for HTML-layout
  • all settings are set directly in to the modulargrid code, which allows you to connect this file on all pages and use the same configuration for the entire project
  • easily configured not only by functionality (fixed and stretching layouts, various units of measurement - pixels, em'y, percentages, etc.), but also the appearance and shortcut keys
  • does not require the use of additional CSS files, images, third-party libraries

What else


Bugreport please send to the project page on Google Code , follow the updates in the same place. Also on Google Code there is a brief overview of similar tools. If you have already turned pages using a modulargrid, send links, I will gladly post!
