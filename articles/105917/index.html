Opera 10.70 (Build 9067) October 11

The developers of the opera work tirelessly and in their blog presented to the public a new assembly branch 10.70. The most delicious Web Sockets support.

Good morning from Oslo. This is the first build with updated kernel functionality, our first implementation of web sockets, and Unite and Widgets enhancements. (no UI change.) Notice the Unite and Widgets regressions.

Attention is a test build and it may not work at all.
