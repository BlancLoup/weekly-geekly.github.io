Unreal Engine 4.21 release

Epic Games has released Unreal Engine 4.21, which according to the manufacturer, gives developers greater efficiency, performance and stability for any project on any platform, and it is thanks to the latest release that users can get world-class results using the Unreal Engine.


Unreal Engine 4.21 relies on proven tools so that you can be as efficient and creative as possible by adding more mobile optimization for Android and iOS, up to 60% increase in speed when preparing content, as well as more power and flexibility in the industry-leading Niagara tools for VFX in real time. In addition, the new, ready-to-release Replication Graph plugin allows developers to create multiplayer games at a previously impossible scale, and Pixel Streaming allows you to stream interactive content directly to remote devices without any compromises related to rendering quality.


Updates in Unreal Studio 4.21 also present new features and improved performance for users in large projects, including architecture, manufacturing, product design, and other areas of professional visualization. The Unreal Studio Datasmith toolkit now includes support for Autodesk Revit and advanced translation of materials for Autodesk 3ds Max, which provides more efficient viewing and design iteration.



Major changes to the Unreal Engine 4.21 include:
  • Replication Graph plugin
  • Niagara improvements
  • Sequencer improvements
  • Pixel Streaming (early access)
  • mobile device optimization
  • accelerated Cook Times
  • Gauntlet Automation Framework (early access)
  • optimization and improvement of the animation system
