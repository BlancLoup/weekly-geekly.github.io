Project Ideas: Radio Nevmenoz

Radio Nevmenoz is an attempt to replicate Dvac's popularity in another leading modality.

Any user can upload any mp3-file to the site of Radio Nevmenoz. After automatically checking this file (say, no more than three minutes in length, not silence, not too loud), it is queued for playing by a media server like Icecast. Files on the air are separated by a beat with the name of the radio. The content of the files is not regulated.

You can vote for the files you like on the site during the entire time the file is played. Leaders vote for example at the end of each hour. After playing files are erased from the server to protect the owners.

The obvious way to monetize: while playing a file, you can send an SMS to a short number. Such a file will be played again, perhaps - immediately after playing the next file.
