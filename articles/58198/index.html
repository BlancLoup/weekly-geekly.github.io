mctop: Monitors the memcached cluster

I present to you mctop - an advanced console utility for monitoring the status of memcached-servers.
You can pick up at - code.google.com/p/mctop

mctop allows you to track indicators such as availability, memory usage, percentage of successful read operations, uptime, number of reads per second, number of records per second, number of forced key deletions to free up memory (EVICT), and more.
The list of servers and output format is configured in the mctop.conf file.
