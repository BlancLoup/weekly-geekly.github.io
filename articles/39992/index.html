Google Desktop 5.8 is much faster

Google has announced the release of a new version of Google Desktop 5.8 for Windows, which has become much faster. Actually, in this release, developers have focused exclusively on performance and optimized everything that is possible.

Memory usage after loading the program decreased by 50%. The speed of closing the program has increased five times.

Fixed system braking due to incorrect operation of third-party software (including Windows). In particular, such "brakes" occurred on the indexing of certain file types or, sometimes, right at the time of launching Google Desktop.

In addition, monitoring tools have been added to the system to identify those third-party gadgets that are exorbitantly consuming computing resources. Also added a module that prevents theft of information from one gadget to another.
