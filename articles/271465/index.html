Remarks
  • At Telegram, by default, encryption is only up to the server, and there is only trust in Durov. Secret chat need to be cut in separately.
  • WhatsApp seems to have screwed up the same encryption as in Signal, but there are no details and it seems to work only on androids
  • Signal is the most secure of convenient non-anonymous instant messengers, but there is no desktop client. Actively sawing web extension for chrome. I advise you to look closely. Low entry threshold.
  • Tox - the future is his, but for now he is very raw. And you need to generate as much Id as you have at the same time devices on the network, otherwise it will be buggy. But he only has anonymous group encrypted voice conferencing (if the client does not fly out)
  • Jabber + OTR. excellent security but high entry threshold with lots of settings
  • Viber, Skype - only if you don't mind that the organs read your conversations almost in real time.
