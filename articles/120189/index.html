On Friday, WordPress turned 8 years old. WordPress history in screenshots



This Friday, May 27, the most popular Wordpress CMS turned out to be exactly eight years old - it was on May 27, 2003 that an open-source CMS was announced with this name. What appeared as a fork of the b2 blogging platform has become an incredibly popular self-contained platform that is used by millions of people. Wordpress websites do everything from schoolchildren to the largest online media. Now on this CMS there are about 45 million sites, including Mashable. I propose to re-view the history of Wordpress in the screenshots, and remember how it all began. Personally, I started working with Wordpress, starting with version 2.3.3. All intermediate versions will not be indicated, in the reviewer only key versions and updates are shown.

Wordpress 0.7.1

The same release, announced on May 27, 2003.



Wordpress 1.0.1

This version appeared in January 2004. The update included an updated installation system, a new default theme, and several other improvements.



Wordpress 1.2

The release date of this version is May 2004. Plugin support appeared in Wordpress, which became one of the key factors that made this system so popular.



Wordpress 1.5

The next date is 2005, February. New features, like the ability to create pages, the new default theme, Kubrick. This topic has remained the default theme for 5 years, somewhere like that.



Wordpress 2.0

The gift to bloggers was made on December 31, 2005. New admin, WYSIWYG and other goodies.



Wordpress 2.1

More than a year has passed since the release of Wordpress 2.0, and a new version appeared in January 2007. Here we added autosave, tabs appeared in the editor, plus the ability to set any page as a homepage for the site.



Wordpress 2.2

May 2007 - Wordpress 2.2 release. Presents widgets, also changed the scheme of conducting work on Wordpress.



Wordpress 2.3

September 2007. New feature - support for tags and categories. This is not surprising now, but then - it was SOMETHING.



Wordpress 2.5

March 2008 - a new admin, plus a lot of additional changes and innovations. The interface for adding materials was also changed, which caused outrage of a number of users. But after a while, it all changed again.





Wordpress 2.7

December 2008 was released Wordpress 2.7. Admin has been completely changed, and already looks more like what we see now. The interface for adding content to the blog was also changed, which remained almost unchanged until Wordpress 2.9.





Wordpress 3.0

July 2010 - Wordpress 3.0, one of the largest updates of the platform, which can be called key. Many innovations and new features. The interface for adding materials to the site has become modular and changeable.





Wordpress 3.2

The release of the new version is expected in June of this year. In the beta version presented by the developers, an updated admin panel, including the addition of new materials. The developers have also officially announced the completion of support for IE 6.
