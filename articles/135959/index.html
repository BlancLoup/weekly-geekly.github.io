In February, Microsoft will launch Kinect for Windows

Microsoft has announced the date of the official launch of the Kinect game controller for Windows.



As the head of the company, Steve Ballmer, during his presentation at the CES conference in the United States, Kinect will be available for Windows on February 1, 2012. Ballmer did not disclose specific details of the novelty, but it is already known that it will be possible to control applications in Windows without using a keyboard or mouse.

The Kinect game controller for Xbox has become a real hit of recent years from Microsoft: the novelty has transferred game interaction to a radically different level, because all movements and game actions are controlled by people without unnecessary devices - only with their own body and hands.

Now Microsoft is working with more than two hundred partners to launch applications for Kinect, so it is quite possible that in a month it will be possible to manage gestures not only in games, but also in application programs ‚Äî documents or photo sorting.
