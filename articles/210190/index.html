Nimbus Notes for iOS 7 - A new cross-platform note taking application.

We did, did, and finally completed the first mobile application Nimbus Notes. For the service of the same name, screenshots, clippers and even a full-fledged web interface have already been released, but there were no mobile applications for direct work. We finally fixed this annoying gap and are ready to introduce Nimbus Notes for iOS 7.



Link - https://itunes.apple.com/us/app/nimbus-notes/id828918459?l=uk&ls=1&mt=8



The main features of the application:

Intuitive interface

Creating the application, we tried to make it as simple and understandable as possible for the user. Using the tabs, you can quickly switch notes, folders and tags. Create a new note also does not take much time.



Two types of sync

We understand that information from the application may be needed at any time, including when there is no connection to the Internet. Therefore, Nimbus Notes supports full work in offline mode. Having chosen Full Sync, the application will completely download all notes to your device, but it‚Äôs worth considering that such synchronization can take a lot of time and especially traffic. If you do not need to have all the information on the phone, then you can choose Fast Sync (it is by default) and the body of notes with attachments will be downloaded only on request. Naturally, after downloading, the note will also be available offline.

Functional editor

The editor supports the main text formatting tags, such as: bold, strike, italic, underline, as well as changing the font color. You can insert a photo into the text or draw a picture directly in the application. If you need to link a note to a specific place, you can specify its coordinates on Google Maps. It is worth noting that the editor supports editing notes with complex formatting, that is, you can edit notes saved using a clipper as well as regular ones.





Todo lists

In Nimbus Note, we decided to show off a little unusual to make the mapping of TODO tasks. Simply put, we have taken out the task list and now it is shown separately from the note and you can safely view your tasks without being distracted by the main text. There are no reminders yet, but they will be in the next version of the application.



Power Off function

Protecting personal information from prying eyes is important. Therefore, we have added the ability to put passcode on an application that will help protect your notes if the phone falls into the hands of others (or not).



Cross platform

In addition to the application for IOS, you can work with your notes through the web interface nimbus.everhelper.me or addon in Google Chrome. In the very near future versions for Android / Windows Phone, as well as for the desktop, will be ready. And this is not counting a large number of supporting applications such as clippers or screenshots, with which you can save various information into your account.

Of course, this is not all the functions that are already in the application. Also, a number of functions are not included in the first version (for example, reminders or creating audio notes), as they are still being tested. Nevertheless, we hope that you will like the app.
