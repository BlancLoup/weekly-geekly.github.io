The project OpenNet.ru failed server

The legendary Russian site OpenNet.ru, which has been covering the topic of free software for 19 years, has been malfunctioning for more than a day. For some time it was completely unavailable, and now on the main page there is an advertisement:

"Attention!
The main server is completely out of order, the backup mirror is valid. Comments and wiki are temporarily disabled.

The system is in the process of recovery. The status of solving the problem in twitter .
Questions and suggestions are accepted at maxim.chirkov@gmail.com. "

Site owner Maxim Chirkov says that now work is focused on restoring the site in working form on the mirror. After that, we will be replacing the main server, which has died. It "requires a motherboard for a 1U case with a memory and a processor in St. Petersburg."
