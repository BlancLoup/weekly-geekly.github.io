Twitter Bootstrap 2.2.1 Released



A new version of the popular Twitter framework Twitter Bootstrap has been released.

From the new:

  • Four new templates: marketing page , login form , sticky-footer and carousel
  • New media component for comments, tweets, etc.
  • Typography scaling using @baseFontSize and @baseLineHeight
  • Updated padding for some elements - inputs and buttons
  • Fixed bugs with z-index for tool tips and pop-ups.
  • Upgrade to jQuery 1.8.1
  • Added Contributing.md file
  • Installing with Bower

In the near future, the developers promise several new large updates.
