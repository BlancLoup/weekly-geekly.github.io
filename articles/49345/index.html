And once again about the project BizSpark

Once again I would like to tell you a little about the BizSpark program that we launched in Russia at the end of last year.



BizSpark is a program that provides start-up software development companies with a set of the following benefits for 3 years:
- Access to technology
- Microsoft Professional Support: 2 technical support incidents per company
- Access to MSDN Premium
- Support for BizSpark Community Partners: consulting, investments, hosting, etc.
- Ability to be selected in BizSparkDB as BizSpark Company of the Week on the Microsoft Startup Zone website

The main conditions for participation in BizSpark:
- A company must develop a software product or a hosted Internet service based on software that will be the main component of the company's business.
- The company must be private, be in business for no more than 3 years and have an annual income of no more than 500 thousand US dollars

A detailed description of the program can be found here: http://ms-start.ru/Programs/BizSpark.aspx
