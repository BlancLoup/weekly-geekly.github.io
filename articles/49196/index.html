Interactive captcha

Since today, January 15 - International CAPTHA-day, then I will offer my version.
My idea is interactive interaction with the user to understand that this is not a robot. First you need to ask the user some simple task. Characters entered from the keyboard are immediately sent to the server using AJAX and thus determine whether it is a human or not. If a word is entered in one millisecond, then this is a robot. Again, you can interactively change the task and monitor the user's reaction. How this could be in a somewhat exaggerated form can be seen in the McBernar picture, in each joke only a fraction of the joke ;-)
