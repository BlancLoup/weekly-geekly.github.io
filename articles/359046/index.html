What was the first Windows you? Windows 3.0 birthday

Hi, Habr! Today, May 22, is the birthday of Windows 3.0. 28 years ago (can you imagine?) The first version of Windows was released, which won wide popularity among users. We decided to pause for a bit and wrote an article about significant PC versions of the system. Let's start with edinichki and come to a dozen, but there are more than ten of them. More under the cut.


So let's go! Microsoft Windows was introduced by Bill Gates on November 10, 1983. Initially, Windows was just a graphical add-on program for the operating system of the 80s and 90s MS-DOS.

Windows 1.0, 1985

Firstborn Something is not black. 4 versions. 3 updates. But so far not a success. Windows 1 did not become a breakthrough for the company on the market, but it launched the family. Today she is almost 33 years old (Birthday November 20).

The second version of Windows was born in cold December 1987. And it‚Äôs really cold. December 9, the day of release, in Moscow, for example, was before -11. Windows 3.0, 1990



Img source

And here is the birthday boy. It was in Windows 3.0 that the file manager appeared. Also seriously improved program manager. And the version has received many innovations that are useful for developers.

Windows NT 3.1, 1993



Img source

The progenitor and XP, and sevens, and today's dozens - NT 3.1. It was the first time that the NT kernel was used. In addition, the NTFS file system first appeared in it.

Windows NT 3.5, 1994



Img source

An important version, although not indulging in a large number of innovations. It was seriously optimized compared to the previous version.

Windows 95, 1995



Img source

In the yard in 1995. Java year, by the way. Microsoft releases a system that is not just a modern one, but truly similar. More precisely, modern systems are similar to 95. A new interface has appeared in it, and it became possible to use more than 8 characters in file names. In addition, Plug and Play appeared, you no longer needed to install new devices on your own.

Windows NT 4.0, 1996



Img source

The latest Windows with the designation NT right in the title. The style was already like that of Windows 95. It was maintained until 2001, releasing 9 updates with bug fixes.

Windows 98, 1998



Img source

Many people call Windows 98 the ideological continuation of Windows 95. Among the innovations, it is worth mentioning the function of putting the PC into sleep mode, the function of converting the file system from FAT16 to FAT32, as well as the integration of the Windows Update site with which updates could be downloaded.

Windows 2000, 2000



Img source

Is a continuation of Windows NT 4.0. Of the innovations is to provide support for Active Directory. The system also includes a third version of ASP . In addition, the encrypted file system EFS.

Windows ME, 2000



Img source

Windows Millennium Edition. Sounds serious, right? Unfortunately, the system has been criticized. However, there was a lot of new things in it: Windows Movie Maker first appeared in it. In addition, system recovery options and Windows file protection were introduced.

Windows XP, 2001




Oh yeah. Honestly, I would love to work with XP now. Appearing in the summer of 2001, it won the hearts of many developers and users for a long time, and for a reason. From the point of view of design, smooth lines appeared in XP - the system proceeded to this as the auto industry, moving away from right angles. Now it was possible to quickly change users and remotely control the computer. Also, the command line received improvements and a lot of other things. In 2014, we stopped supporting it.

Windows Vista 2007




It has become faster than XP in many aspects, as well as safer than XP. It first appeared Bitlocker, Windows Defender and parental controls. Further updates added Windows Search 4.0 and many more minor changes and improvements.

Windows 7, 2009




Seven. It is still popular, for sure in the voting below we will see it. And she is really cool. From the new one, a multitouch appeared in it, a new Windows Media Player interface and much more. By the way, 6 unique desktop wallpapers were made for Russia. The design has also changed a bit. It makes no sense to describe it, and so everyone knows.

Windows 8, 2012




We come to an ambiguous system. It was released only 6 years ago, but for 2 years it has not been supported. The style was completely different, in the eight it was based on the principle of Metro . It is in this system that the Microsoft account, the Windows Store, the usual guide today, as well as the new task manager appeared. Missing start menu.

Windows 8.1, 2013




It appeared just a year after the release of Windows 8. Because of this, and not only there were not so many innovations. One of them is the fourth version of PowerShell. Also, the start menu is back.

Windows 10, 2015




Windows 10 is today's current version of Windows. Compared to 8.1, the following appeared in it: Start was returned fully, significant interface improvements, and a Linux kernel interface emulation was developed. In addition, added a huge number of innovations, here it makes no sense to list them. By the way, we have removed the sapper, kerchief, spider and so on. But do not worry, now everything is in one application. If you turned off the Internet (I had this in the plane), you can still escape from boredom.
