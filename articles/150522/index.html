UICloud: The largest database of user interfaces



UICloud is a database of free user interfaces with a search engine, ranking and cataloger in which everything from PSD sources is collected, to ready-made solutions for HTML, CSS or jQuery: forms, sliders, buttons, calendars, elements and full interfaces for mobile and web applications. In the project now you can find almost everything you need to facilitate the development process for designers and developers.

At the moment, there are 23586 elements in the database and almost a thousand UI-sets that include ready-made solutions in the same style. The project was created by the British studio Double-J Design project goal is to create the most extensive UI database.
