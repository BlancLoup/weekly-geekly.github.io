What does WYSIWYG mean?
In such editors, you can not directly change the source code of your documents, but its presentation, which (if possible) will be published in the final document. Thus, instead of writing blocks of code manually (as you would, for example, do this in Word or LaTeX), you control the design using components using the editor window. This means that you see something very similar to the final result, although the document or image is still being created.
