Google's HTML5 Web Designer Update



In September last year, Google released Web Designer , a free program for Windows, Mac and Linux, with which you can develop interactive HTML5 applications, websites and banners. Since then, the program has been updated several times, but yesterday's update is the most significant.

Google says that since that time, banners created in Web Designer have been shown more than 2.5 billion times. The absolute majority of the audience (72%) is outside the United States.

The updated version of the program supports collaboration with Google Drive file hosting and CodeMirror text editor.
