Mail.Ru Group launches ICQ On-Site

The investment group Mail.Ru Group, which owns ICQ, announced at today's press conference about the launch of a new service that allows embedding the messenger into the interface of any network resource. The latest release is called ICQ On-Site and is made for website owners, online stores, online publications and is positioned by developers as a tool to increase audience loyalty.

According to the statement of Alexander Gorny, the project manager of ICQ, already at the moment the owners of online sites can test the presented solution. For this, it is enough to add a script to the code of the site pages that loads the IM client in a browser window. The code for a specific site can be obtained by writing a letter to icq_partners@corp.mail.ru. The code includes the site ID.

It is expected that later ICQ On-Site will integrate with social. Networks My World and Classmates. In addition, it is planned to implement notifications about new letters in Pochte@Mail.Ru and the ability to send free SMS messages. In addition to all this, Mail.Ru Group wants to provide exclusive product assemblies that take into account the features of partner resources.
