lancelist.0fees.net - a site for finding freelance work on Twitter

main idea



To search and place remote work it is enough to use Twitter !

How did the idea


After the publication of this topic, the idea stuck in my head: All freelancers, simply acting according to certain rules, with the help of the available means, can build a self-organizing system for finding and placing remote work.

Choice of funds

The main selection criteria were:
  • Convenient job search;
  • Simple and clear to all environment;
  • Feedback system;
  • The ability to attach a description, photo, etc .;
  • View information about employers and freelancers;
  • The opportunity to make long-term contacts;

The service twitter.com was perfectly suited for this purpose.
And the rules by which freelancers will work is a tag system .
Tag Set

Wise slyly there was no need, and it was decided to take a list of categories from the site free-lance.ru .
For each category a unique list of tags was written. Example:
Web programming: #lancelist # site # web

One of the advantages of tags is that the search can be localized:
We search by tags: #lancelist # site # web - we get work from Russian-speaking users, we search by #lancelist #web #site - we get a list of jobs provided by English-speaking users. (tags in English will appear a little later on the site)

Title

With an idea in my head I'm going to my friend.
Shoveled various names, we agree on LanceList .
Next - register a domain http://lancelist.0fees.net/ , as well as a user on Twitter lancelist .

Elaboration of the idea


Posting a Job Offer on Twitter

The employer writes a tweet with the following content, example:
#lancelist # web # site need to make a business card site for a company

Pointing the #lancelist tag in tweet, the same set of tags for exact search and a short description.

Choosing the "right" tags

lancelist.0fees.net - a special table has been made on the site so that there is no confusion.

How to get to the list of freelancers?

The freelancer needs to tweet as follows:
@lancelist # web # site doing cool HTML5 portfolio sites: t.co/KHgfkg

Therefore, when users search Twitter on @lancelist and the corresponding tags, the list of freelancers will be displayed.

Briefly about the details


  • In any Twitter client - easy to search;
  • There are no particular limitations on the answers;
  • Account management is meaningless here;
  • Convenient for beginners: when searching for artists on Twitter, newcomers will be in the top;
  • Search for tweets with customer reviews will allow you to immediately evaluate whether it is worth making a deal with him or not.

Please love and respect


http://lancelist.0fees.net/

Attention! do not confuse:


#lancelist - for tweets about work!
@lancelist - for tweets with advertising yourself!
