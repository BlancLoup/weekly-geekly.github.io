Firefox 3.0 got worse

After I upgraded to 3.0, my browser performance dropped and I had to re-develop my browser habits.

These are the controls and hotkeys:
  • Missing the button "Go" in the address bar that was on the right. How can I refresh the page without re-sending data? Re-transfer focus to the address bar and press "Enter"?

  • Shift + Ctrl + T combination stopped working. How can I open a randomly closed tab? View History?

All alternatives require more action, which begins to "plow".

Over time, I'm sure there is still something.

Question to the creators of Firefox: Why was it necessary to delete interface elements and hot keys that I used and developed a habit for them?

Question to readers: Has anyone else noticed that he has to change his habits in working with Firefox 3.0?

Update 27.06.2008 :

It turned out as I expected - everything turned out to be even worse. Firefox has not yet learned how to upgrade from the old version to the new one :(

Forgive my settings - hello default settings.
