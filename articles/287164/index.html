Feedburner counter text

- Cross post from my blog
Naturally, the code is not very new and not mine, but I found only WP plugins on the network, so I decided to post it separately. The script is as simple as two kopecks, from the settings you only need to change the feed indicator in the Feedburner system.



But from the very beginning you need to do the following to go to the admin panel of your feed -> publish -> FeedCount -> Activate

Because you can calculate your Feedburner feed without a counter, and this script works with it. By the way, there you can also choose the colors of the counter, make it animated, etc.

How it works - you can see here - absolvo.ru/tmp/10/
The code itself can be seen here - absolvo.ru/tmp/10/code.html

A little from myself:

If you get rid of the default counter, stick somewhere under the numbers a reference to the page with the counter, so that in the future there will be no questions from users. Who is curious - look.

And finally, I recommend hanging the script execution on CZK in order to cache the results. By experience, I can recommend approximately 16 hours in Moscow.

I hope you find this useful. By the way, if you know the plug-ins and for other engines - share :)
