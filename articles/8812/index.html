New version of the communication visualization engine from TouchGraph

This browser has been greatly improved since we first spoke about it a year ago .

The program is a Java applet (does not recognize Cyrillic encodings) with a lot of additional settings. Separate applets are loaded to process Google results and to process Amazon results .
