Pluginator.ru - plugin service for livestreet

Pluginator.ru is a service that is able to create simple plug-ins for cms livestreet without requiring any programming or layout knowledge from the user. All the skills that will be needed are: the ability to fill out forms in the browser and attentiveness.

The site can be used not only as a tool for creating simple plug-ins, but also a project can serve as a good tool for developers to create plug-in frameworks in their designs. A significant difference from the console built into the LS is more functionality and, consequently, lower costs for further development.

All plugins created by this method can be downloaded for free directly on the site.

At the moment, the service can make two types of plug-ins: the first is a new type of topic , the second is additional fields in the user profile .

A new type of topic - this solution will allow you to create a plugin with a new type of topic, for example a news item in which you will be able to independently set the required additional fields for the plugin.

Additional fields in the profile - the actual subject in the title, this is the creation of any number of additional fields for the profile.

The project is open in the test version, you can create plugins no more than once per hour.
