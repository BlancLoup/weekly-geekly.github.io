Are you excited about the taskbar, but do you work in Windows XP?

Despite the rapid growth in popularity of Windows 7 today, a large number of users of Windows XP remain. At the same time, users of Windows XP often feel the benefits of Windows 7, but for some reason they cannot switch to a new operating system.

As you know, one of the most interesting features of Windows 7 is the taskbar (Super Bar). Therefore, enthusiasts have developed software to turn the Windows XP taskbar into the Windows 7 taskbar. So, meet - the ViGlance application allows you to make the Windows XP taskbar very similar to the one we see in Windows 7.



To demonstrate how the taskbar will look in Windows XP, it is easier to watch the video recorded by the author.

So if you still have to use Windows XP - this is no reason to be upset! Upgrade and work even more efficiently incl. on windows xp.
