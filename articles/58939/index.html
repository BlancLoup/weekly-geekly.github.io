Backup for everyone

In this article I will tell you a little about how fast and easy to make backup copies of all your data - from copying files to external media or another computer, to backup browser bookmarks.

File Backup


Naturally, the most important task of backup is backup of important files. One of the simplest solutions seemed to me to be the easy Russian-language program GoodSync (the free version is quite functional for simple tasks).

So, we make a backup. We create a task, select its type (synchronization or backup), specify the source folder and destination folder and voila! After analysis, the program is ready to synchronize folders or simply copy files from the first to the second. It is possible to create a backup schedule. Supports FTP, WebDAV, SFTP. The only drawback noted is the impossibility of backing up directly to a zip archive.

Backup in the browser


Mozilla firefox

  • FEBE Addon :
    Addon Mozilla Firefox for easy full backup (extensions, bookmarks, passwords, etc.)

    Here, too, everything is simple. Install the add-on, choose what to back up, choose where to back up and create a schedule. Everything! The program itself will create the necessary backup copies.
    Add-on settings window:
  • Bookmark Backup:
    To back up bookmarks in Mozilla Firefox, you need to select the "Bookmarks" menu item, there is "Manage Bookmarks", look for the "Import and Backup" button, select "Backup" in the list.

Opera

  • Bookmark Backup:
    To create a backup of bookmarks, go to the "Bookmarks" menu item, click on "Manage Bookmarks", then click the "File" button and click "Save As".

Chrome

  • Bookmark Backup:
    In chrome, you need to click on the "Settings" button, then select "Bookmark Manager" (see screenshot), find "Tools" in the manager, and then "Export Bookmarks".
