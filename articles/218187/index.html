Microsoft unveiled C # compiler source code

After leaving Steve Ballmer, Microsoft continues to enjoy good news: a few years later, MS Office for iPad finally came out, the source code for the WinJS JS library (Windows Library for JavaScript) was published, and even IE11 implemented decent web development tools.

And now the biggest surprise: today launched the .NET Foundation site, where "for starters" collected 24 open source projects, including the recently released .NET Compiler Platform (Roslyn) !



The Roslyn project is an open-source compiler for Visual Basic and C #, with rich APIs for code analysis. Above the API is built on many useful services. These are the same interfaces that are used in Visual Studio.
