Official IE11 release for Windows 7



Two hours ago, the official release of Internet Explorer for Windows 7 took place. This is certainly a significant event for all web developers who now have to support twenty different versions of Internet Explorer.

So, IE11 in 95 languages ‚Äã‚Äãis already available for download . Over the coming weeks, Windows 7 users will automatically upgrade to the new version of IE. The update begins today with those who have the IE11 Developer Preview and Release Preview installed.

Microsoft promises IE11 users a record page rendering speed. For example, IE11 under Windows 7 in the SunSpider test is 30% faster than any other browser, with the exception of IE10.



IE11 implements a much more efficient garbage collector and JIT compiler. Microsoft proposes to make sure of the high performance of the browser by running a test for making popcorn and comparing the result in Chrome, Firefox and IE11.

IE11 supports web standards that are still not supported in other browsers, so you can create original beautiful applications for IE. For example, the World Data demo shows what you can do with the ECMAScript Internationalization API (version 1.0) , which allows you to sort and format data according to national standards. Calculations are performed on JavaScript in the client browser, without accessing the server.



Microsoft offers website developers the use of the modern.IE toolkit for testing websites in all 20 versions of the IE browser that are currently supported. You can also download free virtual machines for testing on Windows, Linux and Mac platforms.
