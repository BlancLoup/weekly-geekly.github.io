Mouse Control

During the advent of Google Chrome, various of my friends said that the browser is cool, but they lack mouse control, as in Opera (mouse gestures). Now, if for someone it was the only reason for loyalty to Opera, you can freely migrate to the creation from Google. My friend wrote a program with which it becomes possible not only to manage gestures in Chrome, but also in all other applications for Windows.

Description from the author:
The program is designed to manage some actions.
using the mouse. It can be considered as a convenient addition.
to any browser, but it works correctly with many others.
programs (with windows explorer in particular).

The program will globally intercept mouse clicks and its
movement and will conduct an analysis. In case of coincidence with one of the figures
from the directory of the program will be executed the corresponding event.
The man wrote the program exclusively for his own needs, so the ability to customize is missing - everything is tightly packed into the program code. However, if the community has an interest, the program will be updated and supplemented.
