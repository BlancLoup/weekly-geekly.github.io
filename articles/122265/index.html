Overview of the new Nokia N9 smartphone

Hi, Habr!

At 7 am Moscow time at the event Nokia Connection 2011 , held in Singapore, we announced a new product - Nokia N9 , a smartphone on the MeeGo operating system.
Today we will try to talk about it briefly. And, for a start, we suggest watching this video:




Features





Let's start with the design and materials. Case Nokia N9 is made of polycarbonate - a material characterized by high strength and scratch resistance; it is convenient to hold it in hand, it does not slip, it has a nice matte color. ‚ÄúBlack‚Äù (black) and ‚ÄúCyan‚Äù (blue) will be available from the color range at the beginning of sales. The display of the smartphone is protected from scratches and shock glass Gorilla Glass. You should take a closer look at the screen: if on other models it was clear that there is a small ‚Äúairbag‚Äù between the display and the protective glass, then you will not see this in the Nokia N9. It feels like you are touching the image directly. Due to this implementation of the display, it seems a bit convex.



Smartphone control is implemented through several independent screens. When you unlock the phone you get to the splash screen. It displays the time, date, missed calls, incoming messages, etc. If you scroll this screen left or right, the event screen, menu, and application manager will appear in turn, as was shown in the video.



The event screen is a tape that contains all the news from RSS feeds, messages from Twitter and Facebook, weather, calls, SMS and email. Accounts in social networks, email and various photo storage services are not just applications, but part of the operating system. Nokia N900 users on OS MAEMO should be familiar with this. In order to send a letter, tweet or update the status in Facebook, there are separate convenient interfaces.



The menu screen displays phone settings, all standard and installed applications and games. From standard applications we want to pay attention to the web browser. It is based on WebKit2, the newest engine for displaying web pages, which made it very fast and easy to use.



The list of running applications can be viewed on the next screen. From here they can be closed. However, they are not sure that such a need will arise: there are enough smartphone resources to keep dozens of applications open. By its implementation, it is similar to the Symbian ^ 3 OS application manager. True, MeeGo has the ability to zoom the display using pinch-to-zoom.



The camera in the Nokia N9 with autofocus, and with the ability to select any point of focus on the screen. Works in the mode of photo and video. The on-screen QWERTY keyboard supports swype and language changes with one finger movement. Of the other nuances, we note that there are two versions of the Nokia N9, which differ in the amount of memory: 16 and 64 gigabytes.
Especially for the Nokia N9 developed several accessories . We will tell about them later. Additional information on the smartphone can be found on conversations.nokia.com and swipe.nokia.com .

Nokia MeeGo Developers Kit



Many of you have heard about the Nokia N950 smartphone. Someone even saw his images on the net. And such a smartphone really exists! It differs from the Nokia N9 sliding keyboard. And will be available only to developers. All information about this smartphone and how to get it can be found on forum.nokia.com. And for those who are not strongly associated with application development, but like to tinker with Linux in the terminal (command line), Nokia N9 will have a special mode - Developer Mode.

If you already want to try your hand at developing a new platform, then all resources and links to Qt SDK with MeeGo support can be found at www.developer.nokia.com/Devices/MeeGo . There is also a link to the Hartman SDK, which serves to work with the platform and its components at a low level.
