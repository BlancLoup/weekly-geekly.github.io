Jailbreak done, and then what?


This article is for those who have already done a jailbreak to their id or else is wondering about whether or not to do it.
After you have done a jailbreak, you have the opportunity to expand the functionality of your iPhone or iPod touch with various patches from Cydia. About them will be discussed.


So, first of all, I would advise you to install sbSettings - after installation, if you run your finger over the top status bar, a panel will appear with wi-fi, bluetooth, 3G, ssh, on / off buttons, as well as access to the brightness change slider. Using these switches, you can quickly put the device into a more economical mode of using batteries.
Also, this patch allows you to display the status bar along with the clock and date.
By the way, I recommend to put this design theme for sbSettings: narod.ru/disk/5402070001/SBMatte5.zip.html , because the default one, in my opinion, is miserable, to put it mildly. To install this theme, unzip the archive and download it here: / private / var / mobile / Library / SBSettings / Themes and then in the sbSettings settings select the theme SBMatte-bluebuttons



Along with sbSettings, Activator will also be installed, in the settings of which you will find the ability to override actions for pressing various buttons or gestures. For example, I put the ‚Äúvolume up‚Äù button for a short time while switching to the next track, and the volume ‚Äúlower‚Äù for switching to the previous track and thus controlling the music without taking the device out of my pocket.

DisableDaemons - a patch that disables the processes of assembling and sending logs to Apple, thereby saving us the battery and allows us to increase the time of using the device on one charge.

iAdKiller - the name is clear :) A patch that will block iAd ad units in all applications.

iFile is a full-fledged file manager. The good thing is that with it you can also drop files between the device and the computer via wifi and view the pictures in high resolution and zoom in more than in the standard photo viewer, well, all the other functions that are inherent to file managers.

Installous 4 is an application that allows you to simply download and install from the Internet almost any application for ios.

FreeSync is a patch that allows you to use the device in normal mode during synchronization with aytyuns.

FullScreen for Safari - adds the ability to browser full-screen mode and support gestures (one-two and even three-finger) for various actions. Gestures can customize as you like.

Lockdown Pro - a patch with which you can set a password to run certain applications

MxTube and YourTube is the first, separate application for downloading and storing YouTube videos, and the second is a patch that integrates this function into the standard YouTube application.

PlayAwake - a patch that allows you to put on the alarm any song from the library.

Safari Download Manager - the name is clear, download manager from the browser. At first you can download everything your heart desires in a browser, and view, for example, in the same iFile file manager.

StayOpened is a patch that prevents the AppStore from closing as soon as you start downloading the application. Personally, I was very annoyed by this, I want to install several applications, but I constantly had to re-open the AppStore, with this patch you will also get rid of the constant opening and closing of the AppStore application.

xGPS is an application that allows you to download Google maps for using them offline. The route of course can not be laid without an active Internet connection, but it will help to look and navigate the map. Personally, this program rescued me more than once on my iPod.

That's all the patches that I use on my device and recommend it to you. If there are any more worthy of attention, tell us.

Ps In the review, I did not affect patches that relate only to iPhones, since I do not have an iPod touch and check the functionality of these patches.
pps If you do not find a patch in the standard repositories, then most likely it can be found in the repository from iguides.ru
