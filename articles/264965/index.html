The right bonus for your new server - Part 2

Hi, Habr! You have not heard about the best offer of the outgoing summer? Then you have a chance to use it in the fall! Now, when you buy an HP ProLiant server (a list of models under the cat in the spoiler) based on Intel Xeon E5-2600 v3 series processors, you will receive an additional lifetime license for the HP iLO Essentials management tool with 1 or 3 year support period. HP ProLiant servers fully disclose their capabilities only with the tools for remote monitoring and management of HP iLO .

Read more about how to get a license as a gift, read under the cut. Also, if you have already discovered the possibilities of iLO and you liked it, write your feedback in the comments to this post and get a small souvenir (the number of sets is limited).

In the Essentials version, the HP iLO management tool provides the most requested features for working with small and medium-sized server parks. Among the possibilities:

- Connecting virtual images of drives to the server over the network (for example, to install the operating system)
- Remote launch of the graphical server console without the participation of the operating system
- Agentless monitoring in the operating system
- Configure advanced email notifications
- Inclusion of a server in the group iLO Federation
- Server management through open REST API
- Record all events occurring on the server in the "flight recorder"
- Convenient web interface

You can learn more in this technical paper .
