Application for finding work and company information (for Windows Phone 7)



Job search is a simple process for someone, difficult for someone. But to be aware of what is happening in the labor market is desirable to all. It is for these purposes that we have created a database of companies (mainly from Ukraine) for easier navigation and search for suitable vacancies.

The purpose of the application: to provide the ability to search for IT-companies, as well as information (addresses, phone numbers, vacancies, links to social networks).

The application will be distributed free of charge, but we hope for the interest of companies for the publication of recommended vacancies and offers.


If we talk about the technical part, then the base of companies is here , where for each company a complete information package is assembled. The database is currently in the process of filling and clarifying data, so if you did not find your company or found incorrect data, please report to msugvn [at] gmail.com soap or other available means. Data is sent to the application through a simple API. In addition, in fact, information about companies and vacancies, the application will be supplemented by other useful information: company news, a calendar of technical events, as well as statistics on salaries .

Overview of the main functionality of the application:

1. Start screen: you can search for a company by its name. If there is an Internet connection, then the search is performed via an external web service; otherwise, it is performed using a local database, which is based on SterlingDb.



2. Having chosen a company, you will go to its page, where there are the following sections: general information, a list of addresses and contacts, active vacancies, a list of RSS entries, twitter feed.





At the moment we are working on improving the design and usability of the application, your wishes and constructive (and even harsh) criticism is welcome.

The application is currently not yet published in the Marketplace, but will be available there very soon.

PS Company API theoretically we can make it public to create other applications.
