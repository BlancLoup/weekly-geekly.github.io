Respondu.Net guesses thoughts

Respondu (in Esperanto means ‚Äúanswer‚Äù) is rather a toy that we made with our neighbor ( raakha ) to entertain friends and ourselves.

The essence of the program is that the user is asked to guess any object, after which the computer tries to guess it by asking the user questions. If the computer is mistaken (the knowledge object does not have a hidden object), the program asks the user to name and briefly describe the hidden thing.

How it works?
Very simple: Respondu builds a tree of signs, adding user answers to it, and a series of questions asked to the user is a simple walk through this tree.

Respondu is made on the Python framework Pylons , it took us one light day to design and develop.
