5 ways to occupy the sidebar in Firefox

Few people thought that the sidebar in FireFox can be used for quite useful things. This article shows 5 ways in which you can usefully take a place in Sidebar.

Add Facebook or Google Talk Chat





If you want Facebook or Gtalk chat to be displayed in the sidebar, just make some simple movements.



On the example of Facebook, this is done like this:
1. Go to the pop-up chat page facebook.com/presence/popout.php
2. Add this page to your bookmarks.
3. Open the properties and put a tick "Display in the sidebar"
That's it, when you go through the tab - it will open in Sidebar.

Add a video player





With the YouPlayer plugin you will be able to watch videos in the sidebar and surf at the same time on the Internet. The plugin understands videos, both from Youtube and Myspace etc. The plug-in is available function with which you can postpone viewing selected videos to the desired time, and therefore there is no need to add a bookmark to the page with the video.

Add Google Reader





Very useful feature, in fact. Displaying Google Reader from the side saves time. Also, as in the example with Facebook, go to www.google.com/reader/i/ , add it to your bookmarks and add a sidebar to the properties.

Add Google Documents





Using the gDocsBar plugin, you can manage documents in Google Docs through the sidebar. The plugin supports searching for documents through the sidebar, filters, creating a new document, as well as other functions.

New tab view





If you are tired of the usual position of the tabs - use the Tree Style Tab plugin to place the tabs in the sidebar. In addition, with this plugin, finding the right page among the tabs is not so difficult. Tabs are presented in the video hierarchy, because they are much easier to manage.

In fact, there are more ways than useful you can take the sidebar. These 5 ways - only the most common methods of using the sidebar.
