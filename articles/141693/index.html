EnerGenie EG-PMS-LAN Surge Protector with LAN

Progress does not stand still - modern devices in the house are increasingly acquiring support for the Internet. Refrigerator with internet? You are welcome! Now most modern TVs have LAN support. More and more devices with an IP address appear in the house. EnerGenie wanted to excel and released an interesting device - a programmable network filter with a LAN interface. Also in the range there are models with WiFi and USB , but it looks like they are not for sale.



The device comes in a small box in which the device itself is located, 2 patch cords (straight and cross), documentation and a disk. It is strange that the ‚ÄúUser‚Äôs Guide‚Äù is written on the cover of the manual, but inside there is no Russian language. The site has a complete guide in Russian.



The external device looks larger than a regular network filter. It looks like there is a power supply for the network controller and socket control board. The power cord has a length of 1.8 m.



On the front side of the device is a power switch that turns off all sockets. In total there are 6 sockets, 4 sockets can be controlled. Near each controlled outlet there is an orange indicator and outlet number.





On the side of the device is an RJ-45 connector for connecting a network. Next to it there are 3 buttons to reset the settings, turn off the power and configure the network. The network indicator shines bright green when data is transmitted.



On the back of the device information and MAC address. Not every day you will see the MAC address of the network filter. If desired, the device can be hung on the wall: there are hidden fasteners and holes on the edges of the surge protector. The manufacturer tried to protect against parsing and use Y-screws.



After installing the software, a wizard is launched that suggests finding an outlet on the network. In the case of a successful search, click Open and get into the web interface of the outlet!



On the main page, you can manually click the toggle switches of all 4 sockets and register the device on the EnerGenie website to remotely control the device.



For each outlet, you can view tasks and create new ones.





For convenience, each outlet can be designated by name.


Network filter settings are very simple, although what is there to configure?





Also for easy setup there is a program Power Manager (only for Windows).
You can add several network filters in it and customize them. Double-clicking on the added device will open a socket in which you can see which sockets are included. If you double-click on the outlet, it turns on or off.



Consider the settings of a separate outlet. Here we can bring the socket icon to the tray, set the keys for switching on and off, allow switching on or off at system startup or shutdown.



Let's take a closer look at the 3 buttons Hardware schedule, Software schedule and Application Events.
In the Hardware schedule, a schedule is set that will be saved in the device itself and will be executed regardless of whether the Power Manager program is running. Set the time, set the switch to ON and select a repetition for each day if required. Something like this can create schedules. Software schedule does the same thing, but the schedule only works when the Power Manager program is running.
There are recommended rules for creating the correct schedule:
  • Events on the list should be in the future.
  • Duplicate entries are not allowed.
  • The number of events for the outlet should not exceed 16.
  • The maximum number of events depends on the total length of the schedule.
  • The interval between the present moment and the last entry in the schedule can not exceed 180 days.
  • Without the use of repetition, the total duration of the event cannot exceed 215 days.
  • The replay interval of an event cannot exceed 180 days.




Using Application Events, you can configure the inclusion and disconnection of sockets depending on the programs being launched or if files have been deleted or created in the folder.
For example, when you start Word Viewer, the socket with the printer is turned on, and when you exit the program, the printer turns off. You can also configure the scanner when you run the scan program.


This job allows you to monitor the folder with print jobs and turn on the printer for printing. After 2 minutes the printer turns off.





The settings are simple, easy to understand. As you can understand the device was done to save energy, but we all know how to use it. For example, restart a hung access point or server. In the end, everyone decides for himself how to use such an unusual device. The product turned out to be niche and is unlikely to become widespread.
