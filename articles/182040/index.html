Windows client release for Nekka

We are glad to announce a new release of the Windows client for synchronization with Nekaka.com . Obeying the trend of the visualization of the Internet, the post was made with the expectation that you can understand its essence without reading a single line of text (at least we tried!).

So, to the main functionality of Windows synchronization <-> nekaka.com:


we added an expanded context menu and file sharing option:


Depending on whether the recipient is a user of Nekaki or not, his request will be processed differently, but the recipient, in any case, will be able to download the file.

You can still upload files to your account in a whole set of ways (well, God be with him, with visual Internet!): Via the web interface, sending an attachment to a special e-mail, uploading it via ftp or webdav, as well as uploading ftp / http / torrent link to our server (i.e. when leaving work, put something to download, and our server on gigabit channels downloads and uploads to all your windows machines running a nekaka client).

Download the client here .

All the described functionality is available in free 10GB accounts. No advertising, as before, we have not.
We will be glad to hear the wishes, both for the client and for the project as a whole.
