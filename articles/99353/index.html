Dolphin Browser HD

And so, let's start with our consideration with this browser.
When you start the browser, we get the following window.


As you can see, there is a tabbed panel above the address bar, which allows us to work in a multi-window mode, without switching every time to the window preview function, to switch from the current to some other.



The lower toolbar is invoked by the Menu button, the left one is accessible by simply ‚Äúdragging the screen‚Äù from left to right. Navigation is convenient enough, access to bookmarks and history can be accomplished quickly, without going into a separate window, but simply by moving the visible area.


And now features specific to this browser: It is possible to install various add-ons / extensions. At this point available:
  1. Adblocker
  2. Password manager
  3. Screen cut can

And according to the authors, more additions will be available in the near future.

And the second feature is the ability to use the phone‚Äôs screen as a kind of primitive tablet to draw a specific set of images, thereby provoking browser actions (it can be convenient for people for whom hitting the right buttons (parts of the screen) is difficult).

The browser supports multitouch.

Opera mini 5.1

The next browser in question will be the recently released Opera mini 5.1 for the Android platform.
When launching the browser, we will be greeted by the already familiar Opera users (whether desktop or mini versions) with a start page with a preview grid of your start pages.


The address bar is next to the search query string. The decision is dictated, as it seems to me by the analogy with the desktop browser. Here, in all browsers studied, except Opera mini 5.1, when typing a word in the address bar, which cannot be a website by any regular expression, it is simply automatically inserted into the used search engine.



The browser supports multiple windows (with multiple sites) at the same time. And we must pay tribute - the transition between the windows is the most convenient of all that have been tested: there is no need to leave the page you are viewing, but at the same time, the list of windows does not hang on all the time before your eyes, eating the already valuable space on the screen of a mobile phone . The bottom common toolbar contains everything that you may need when viewing a page on the Internet.



A very convenient mode of working with bookmarks and history, even if you have to move away from viewed windows. Everything is organic and not very redundant.

The browser supports multitouch.

Skyfire


The next browser in our charts is SkyFire. According to the developers: the only browser to play millions of flash-video clips on the Internet (at the moment the information is not relevant and true, since special programs have been released by both htc and youtube, which, if you have a flash plug-in from Adobe, allow you to play flash -video).
When you start the browser, we will automatically load the start page.



As you can see, in this browser, the toolbar is located under (next to the album version) address bar permanently. It presents the most frequently used user while surfing the button.



The bottom toolbar is also called the Menu button, and provides the user with a set of add. features such as: the ability to share the page being viewed, through one / several of the most popular communication services, view the list of videos available on the page and view the thematic parts of the page (video / news / tweets).



The browser supports up to 8 simultaneously open windows with digital content from the global network. The ability to navigate through them (windows) exists only if you switch to a special menu function, while the browser page being viewed disappears from view. However, in fairness, it is worth noting that the function of switching between windows, mini-preview pages, make it very easy to navigate the already open tabs. Work with bookmarks and history also does not cause censures.

The browser supports multitouch.

Default Browser

Well, finally, the last in the list, but not least, is a pre-installed browser by htc.
Launching it, we automatically open the start page, which by default is the htc page.


As can be seen from the above screenshots, the browser is always in full screen mode, however, it comes out of it right there if there are any system notifications that are reflected in the phone‚Äôs task bar. And a very important part is that when browsing a page, scrolling it down a little is enough, and the address bar also disappears, which gives the user the ability to view Internet content of interest to him throughout the entire screen of the phone.



The lower toolbar is called up again by familiar Android phone users by pressing the Menu button. Not the most functional toolbar, but what to do. The browser supports multiple windows simultaneously (up to 5 windows). Navigation between open windows is very convenient, even if it is a separate function that requires you to leave the page you are viewing.



Working with bookmarks and history is standard, and not much different from the browsers discussed earlier.

The browser supports multitouch.
