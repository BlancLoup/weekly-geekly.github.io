Google Chrome in black (change skin)

The new Google browser will probably have to sit in the beta version for a long time and acquire the usual and convenient features for the ordinary user, however, the beginnings of what we will see in the future are already visible in the browser. For example, change the appearance of the browser.
Windows Vista users are surely more or less pleased with the design - a beautiful piece of glass is on top, everything is more or less nice, but Windows XP users see some very dull blue stripes in the title.
And I'm not the only one so displeased with this slightly annoying omission, there were enthusiasts who corrected this matter.

We meet , a dark version of the design of the browser from the user Manhog with the well-known service DeviantArt .

First, let's take a look at what we have by default in Windows XP, then - what happened after the change of design to black:



Want yourself the same? It is not difficult:
  • You need to download the style from its page on DeviantArt or simply by this link ;
  • Open the folder % userprofile% \ Local Settings \ Application Data \ Google \ Chrome \ Application \ 0.2.149.27 \ Themes in your favorite file manager (yes, apparently, daddy for different topics for the future);
  • Rename the standard style ( default.dll ) to something else, just in case;
  • And copy the new library default.dll from the archive.
Restart the browser and enjoy. If we are not happy, then we return the old library to its place.

The idea for the topic spied on HowToGeek.com .
