Revolution R has been renamed to Microsoft R and is available for free to developers and students.


In the nine months since Microsoft acquired Revolution Analytics, the company has released many updates for the Revolution R Open and Revolution R Enterprise, not to mention integrating R with SQL Server, PowerBI, Azure, and Cortana Analytics.

The American company Revolution Analytics is a manufacturer of software for statistical data processing. The company specializes in commercializing the R programming language and creating software solutions using it.
The main product of Revolution Analytics is Revolution R - it is the runtime language R (a programming language for statistical data processing and graphics), optimized for multi-thread computing, as well as a set of libraries for parallel processing within the framework of the ‚Äúbig data‚Äù concept.

A few days ago, Microsoft provided access to R Server products using MSDN and DreamSpark subscriptions.

The official announcement from Microsoft took place on January 12th. The company said that the Revolution R Open was renamed to Microsoft R Open and is available for free download , and the Revolution R Enterprise was renamed to Microsoft R Server.

Joseph Sirosh, corporate vice president of Microsoft Data Group, reported on the Machine Learning Blog page about the release of a new product - Microsoft R Server Developer Edition . The same edition will be staffed previously announced Microsoft Data Science Virtual Machine. This will allow R users to quickly get started on data mining and modeling in the cloud without having to set up their environment. Microsoft R Server Developer Edition includes the features of the commercial version, but is distributed free of charge.

The commercial distribution of Microsoft R Server will be integrated into SQL Server 2016 as SQL Server R Services. Prior to SQL Server 2016, Revolution R Enterprise for Windows will be delivered as a separate product.

To get started with Microsoft R Server, you can download it from one of the following links:
For all: Download Microsoft R Open from MRAN (Free.)
For students: Download Microsoft R Server from DreamSpark (Free. Microsoft account required)
For developers: Download Microsoft R Server from Visual Studio Dev Essentials (Free. Microsoft account required)
