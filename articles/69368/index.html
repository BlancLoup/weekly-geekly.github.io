Acer Ferrari One: netbooks become more interesting

Acer presented its netbook under the Ferrari brand on Thursday, but it was not as thin and powerful as we expected from the Italians.



The screen is 11.6 "with a resolution of 1366-x768 pixels, the usual Intel Atom processor is replaced with a dual-core AMD Athlon 64 X2 L310, 2 GB DDR2 800MHz memory (expandable up to 4 GB). Graphics - ATI Radeon HD 3200. Hard disk - 250GB , Acer says it can make a version of the Ferrari One with 320GB disks, and a card reader that supports SD, MultiMediaCard, Memory Stick, Memory Stick Pro and XD-Picture Card formats. There is an 802.11b / g wireless module and optionally 3G module.
