First jQuery 1.4 alpha released

According to the developers, the code passes all tests in all supported browsers, but you need "live" testing and feedback from the developers.

From the changes in the new version:
  • Seriously updated and rewrote the event live. Now you can hang it on submit, change, mouseenter, mouseleave, focus, blur.
  • Optimized methods append, prepend, find, empty, remove, addClass, removeClass, hasClass, attr, css - should work faster.

You can get the new version here - http://code.jquery.com/jquery-1.4a1.js

If you work with this wonderful library, take some time and test the new version in your applications. Personally, this is exactly what I intend to do. Find a bug - throw it at http://dev.jquery.com/ .

Original jQuery blog article .

PS Just in case, I remind you - this is the alpha version , do not use it in production .
