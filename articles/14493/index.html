The ioDrive is a PCI Express NAND flash memory expansion card that acts as a local data store. The main thing in the new product (except for installation in a PCI Express x4 slot, of course) is its performance, which is 100,000 I / O operations per second. And the read and write speed shown in testing the ioDrive operating in the HP BladeSystem c-Cl chassis was 800 Mbps and 600 Mbps, respectively. Apparently, once again we can start discussing the topic of replacing disk drives with something more advanced, fast and reliable. In this case, on ioDrive.
