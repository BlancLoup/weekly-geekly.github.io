EtherCalc: Spreadsheets on Node.js

EtherCalc is a great tool for editing spreadsheets on the web, including collaboration.



Simply drag the finished CSV or Excel XML document (.xlsx) onto the page - and it is immediately imported into EtherCalc. Or you can create a new document with one click.

EtherCalc seems more convenient and functional than Google Spreadsheets and other similar programs. In the repository comments on Github, the author explains that EtherCalc is the port on Node.js of the SocialCalc application. The program can be installed locally to work offline.

The source code is published under a free license .
