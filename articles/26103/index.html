Electric memory on wikis

This thing has already been mentioned in the Habr√©, but I am sure that it is worthy of a separate post. How to be found, I stumbled upon it quite by accident, and now I do not understand how I lived without it before. So, TiddlyWiki , please love and favor!



What is it?

TiddlyWiki is rather not a wiki, as you might think because of the title, but a blog engine, whose entries are edited using wiki markup, making it an ideal option for storing notes. Search and tags are attached.

The most important thing is that the whole engine is one html file that can be written to a flash drive and always carried with you. You can also upload it to a web server and use it as a regular blog, by the way, most of the sites about tiddlywiki itself are made.

The engine is written in javascript, and all its self-modification capabilities are used to its fullest. You can synchronize your copy with others, update the engine from the update server, install plugins. Everything that is on the tiddlywiki page can be changed directly in it. On the fly, everything changes, from the main menu to the style sheet and page layout. There are many plug-ins, for example, for recording mathematical formulas, displaying tags in the form of a cloud, or queries to the database of records on some similarity to SQL. Yes, everything else, it all works surprisingly fast.

Installation

To fix this miracle yourself, go to http://www.tiddlywiki.com/empty.html and click Save As in the browser. The saved file is open. Enjoy everything!
