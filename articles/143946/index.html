Substantially updated Evernote 4.0 for Android

Significant operating system updates provide an excellent opportunity to rethink your applications. And the release of Android Ice Cream Sandwich is no exception. Today we released Recycled Evernote 4.0 for Android. This version has received not only a new look, but also a number of features.

Download Evernote for Android

New look for Evernote for Android


Each screen of the application has been redesigned to make working with the Android client easier, faster and more convenient.

New home screen
The updated home screen is completely different from everything we did before. It combines the functions of creating notes and buttons to quickly jump to viewing notebooks, tags and notes by location (more on that later).



In addition, it is now very convenient to return to the home screen - just slide your finger to the right or click on the icon with the elephant.

Improved lists of notes and notebooks
We redesigned the appearance of the list of notes to make it easier to view the contents and search for the necessary information. In the updated list of notebooks, you can now immediately see your personal notebooks, and those to which other users have given you access.



Action panels
We also updated the action bar, which is located at the bottom of some screens. Now they give convenient access to the functions of the application, depending on which screen is open. For example, if you are viewing a note, you can go to editing, opening access, assigning tags, and other note-related features using the panel. When the screen with a list of notes is open, the action panel will allow you to create a new note, search through the notes and change the order in which they are sorted.



Convenient reading


We believe that the best interface is the one that appears only when necessary and does not sore eyes at another time. By this we were guided while working on a new approach to viewing a note. Now, when you open a note and start scrolling it, all navigation elements disappear, putting the entire screen at your disposal. If you need to edit a note, share it with others, or return to the list of notes, just tap the screen.



New navigation gestures


Horizontal scrolling has become a new convenient way to navigate the application. If you have a home screen, swipe left and you will go to the list of notes. Swipe to the right and you will be returned to the home screen.

When you have the Notebooks, Tags, or Location screen open, you can move from one of these screens to the next by scrolling.

Location


If you have enabled location on the device, Evernote assigns information about the place where it was taken when creating a new note. The new Location screen allows you to view notes by place of their creation. Click the corresponding icon in the action bar to see the notes on the map.


New settings


A noticeable innovation in the interface was the appearance of the user name in the upper left corner of the home screen. By clicking on it, you will see the available amount of memory for notes for the current month and you can go to the advanced settings.

Next to the user name is the profile icon. It indicates the type of account: a simple icon for free and a shining emblem for premium accounts. Also, this icon will warn you if the monthly limit for creating notes comes to an end.

Much more ...


As usual, visible changes are only part of the work we have done. We have also made many improvements and fixes that will make your work with Evernote even better than ever. Click on the light bulb icon to learn about all the innovations and get useful tips.
