How to enable Time Machine?


Before you force your Mac to search for an accessible copy space, you need to install the plugin.
  1. Install ‚ÄúMac Backup‚Äù (Mac Â§á‰ªΩ Âä©Êâã) through the official smartphone application.



2. Run Time Machine on Mac, and it will immediately find the disk of the router called MacBackup.


3. Using the disk utility, we divide the router's disk into two partitions (for backing up the system and for storing data)
4. Format partitions in HFS and set labels.
5. Select data for backup (you can leave only the system).
6. The first backup will be long, the subsequent ones will be incremental, i.e. only what has changed will be transmitted over the network. New data will overwrite old data.

That's all, Mac backups are configured.
