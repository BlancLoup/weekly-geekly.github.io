22 photoshop plug-ins for front-end developer

I offer the readers of "Habrakhabr" a selection of 22 add-ons (scripts) of helpers needed by the front-end developer.

Hexy




It is sad to choose layers in the layout and each time start the color palette only to get the hex value of a particular color. Hexy allows you to use the eyedropper tool: click on the color of the object and copy it directly to the clipboard.

Griddify




Griddify is small and fast. He will help you create your own grid and do a bunch of other things on the Adobe Photoshop guides.

Size marks




Size Marks is a very useful script for a web designer or front-end developer. With it, it is easy to measure the external indents of elements.

Velositey




Velositey is a full-featured open source Photoshop add-on. Designed to create boring things, so you focus on styling the site.

Random User




Random User creates "users" to fill in the layout when creating the site, thereby saving you time.

Kaku




Kaku is a simple plugin for translating layer names into different languages.

LiveShare PS




LiveShare PS allows you to conduct online broadcast demonstration of Photoshop documents.

Layercraft




The plugin exports user interface elements from layers.

Inc




Ink is a plugin that provides you with additional information about your layout, documenting your layers starting with the printing company and ending with effects with the dimensions of the layers.

Modular Grid Pattern




Modular Grid Pattern is an application for web designers to quickly create a grid in Adobe Photoshop.

Photoshop Social Shar




Photoshop Social Shar is a free plug-in for sharing your design.

Pngquant




Pngquant - ensures saving .png better than with Save for Web. The created files are 60-70% less than 24/32-bit PNG with Save for Web and, unlike 8-bit PNG, support a translucent alpha channel.

Bounce




Bounce is a free add-on that adds access to the Dribbble community in Adobe design products.

Retinize it




Retinize It allows you to prepare your design for devices with retina-display.

Group Layer Renaming (Script)
