New free courses virtual academy Microsoft Virtual Academy, February 2016



Beginning of the year! And we are happy to share with you a review of the new free courses of the Microsoft MVA Virtual Academy. Microsoft experts regularly continue to produce courses and prepare materials specifically for you to provide relevant information to developers and IT professionals.

New! Application Lifecycle Management (ALM) for start-up companies
Are you going to launch your first product as an entrepreneur? In this course, you can familiarize yourself with the basic principles of application life cycle engineering, which are necessary for the realization of your dreams. It covers version control, release management, automated testing, and more. Find out what the terms such as Git, SCRUM and TDD mean, and find out why they can provide a higher quality first release than the competition.

Hit! Introduction to ASP.NET 5
In this case-filled beginner lesson (first in the series), you will learn about data modeling, managing user requests, and connecting to a server database. Get a solid foundation for developing your knowledge and skills in working with ASP.NET and start building your own applications. Continue the course with practical examples of using ASP.NET.

Website development using Python and Django
Learn how to get started with Django in Visual Studio, how to create dynamic data-driven applications, and how to deploy applications in Microsoft Azure. Explore the object-relational mapping (ORM) system provided by Django and learn how to ensure that your page is detected by search engines. In this case, you will see how the instructors will create a website from scratch.

Introduction to using JSON in C #
Want to learn how to work with JSON in C #?
Familiarize yourself with JSON syntax and learn how to parse and create JSON files. You will also learn how to create a console application that uses a JSON file to load and save application data.

Hit! Using Node.js with Visual Studio Code
Under the exciting guidance of experts Stacey Mulcahy and Rami Sayar, learn how to create websites in Visual Studio Code using Node.js (one of the easiest and fastest ways to develop real-time applications on the Internet today). Examine the functions of this tool, the order of their settings and basic commands. Learn how to create basic REST APIs using Node.js and Express, how to connect web applications to the database, and how to debug Node.js applications using the Visual Studio Code editor.

Microsoft Azure for Node.JS developers
In recent years, Node.js has gained popularity among the new class of developers who prefer the asynchronous programming paradigm. For the Microsoft Azure cloud platform, it makes no difference whether the applications you are running are built on the basis of Node.js, ASP.NET, Java or PHP. And in this course you will get acquainted with the Microsoft Azure cloud platform in the context of Node.JS. All major aspects that a developer needs to run applications in the cloud will be covered.

Discover the secrets of LINQ
You probably have been using LINQ for a long time, but have you thought about the basic elements and their interaction? Learn about it under the guidance of our team of experts! By the end of the day, you will have a deep understanding of one of the most important modern libraries of the .NET platform.

Hit! Development of virtual and augmented reality applications on the Microsoft platform
All issues of building virtual reality applications on the Microsoft platform in one course - from the basic principles of stereo imaging and head tracking to deeper issues of highly efficient implementation of augmented reality in C ++ / DirectX.

New! Cloud backend for Android and iOS applications on Microsoft Azure
The course will help developers on the Android and iOS platform in mastering the launch of the server side of mobile applications using Microsoft Azure. The course goes beyond basic knowledge, touches upon cross-platform development issues and at a deep level examines the Azure SDK for Android and iOS platforms. The course contains a large number of demonstrations and practical recommendations.

Hit! Windows 10 on the corporate network
If in the future you are planning to migrate to Windows 10, it's time to look at the approaches and migration tools, the ability to manage devices with Windows 10, as well as some of the security technologies of the new OS. In the course, you will be able to get acquainted with the features of migration from Windows 7 and 8.1 to Windows 10, understand when and for what provisioning can be used, what tools besides group policies can be used to manage Windows 10, see what Windows Hello biometric authentication is.

Hit! Running Linux in Microsoft Azure
Do you use Linux cloud solutions? Interested in how to use Linux in the Microsoft Azure cloud? Try this course! The course includes all the information you need to get started working with virtual machines on Linux in Azure, even if you are not familiar with the cloud platform from Microsoft.

New! New features in the pre-release version of Windows Server 2016 Preview
A panel of experts will talk about a host of new automation features, as well as support for partner technology and your own investment in open source solutions. These sessions cover advanced virtualization functionality, as well as automated processes and settings that help deploy computing, storage, and network resources faster. Explore new features that reduce system downtime, learn how consistent updates can help speed up the adaptation of updates and operating systems for the Hyper-V server and the scalable file server, as well as learn about new storage replication technology. Additionally, the Nano Server technology will be presented.

Detailed view of Azure Resource Manager scripts and models
Learn how to manage resource groups with Azure Resource Manager (ARM). Explore ARM with an international team of experts who demonstrate technologies and tools that make it easy to increase or decrease the number of infrastructure components during application development, testing, or operation.

Windows Azure Pack: Database As A Service (DBaaS)
Experts will introduce you to the features of the DBaaS offer in the Azure Pack, which provides a powerful SQL Server back-end that users can use in self-service mode or programmatically using a rich set of APIs. Using the DBaaS model, application owners and developers can independently prepare and receive exactly what is required for their applications and workloads.

New! Safety in the world of clouds
Do you protect your IT assets in a public or hybrid cloud? Concerned about the rise of cyber security threats? You found the right course! Get to know the Microsoft experience that the company acquired by responding to cybersecurity incidents from customers and helping customers protect their cloud assets and manage their shared responsibility for the security of cloud services.

Build Linux Clusters in Azure
The course examines how to organize the work of a failover cluster (high availability cluster) in the cloud, what tasks are first of all necessary for this to be solved. A detailed discussion of setting up fault-tolerant storage based on DRBD, as well as cluster software Corosync and Pacemaker, designed to organize the coordinated work of the cluster nodes. In addition, using the example of the STONITH library, the setting up of the system of enclosing failed nodes (fencing) is discussed in detail. Particular attention is paid to configuring all components to work in the Azure cloud infrastructure, but for the most part the instructions are correct both for building a cluster and in the system of traditional architecture, without using the cloud!

You can find even more online courses on the Microsoft Virtual Academy website, as well as pick up materials about the technologies you are interested in.
