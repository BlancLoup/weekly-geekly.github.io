Value Substitution in Zend_Config

When in the configuration file it becomes necessary to use the substitution of values, the official documentation doesn‚Äôt have a double hint at the application, IMHO, not the most successful practice - the use of constants in INI files, for example:

[production]
includePaths.library = APPLICATION_PATH "/../library"
bootstrap.path = APPLICATION_PATH "/Bootstrap.php"

I wrote a small class that inherits and extends the functionality of Zend_Config, allowing you to use this syntax:

[production]
path.application = "${path.root}/application"
path.configs = "${path.application}/configs"
includePaths.library = "${path.root}/library"


Here is an example of its use:
//
$environment = 'production' ;

//
$rootPath = dirname(__FILE__);

//
$configPath = rootPath . '/application/configs/application.ini' ;

// ""
// read-only
$configRaw = new Zend_Config_Ini($configPath, $environment, true );

//
$configRaw->path->root = $rootPath;

// , "" ""
$config = Inf_Config_Placeholder($configRaw);

// ,
