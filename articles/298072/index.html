‚ÄúRecently, the number of sites using clickjacking technology has increased - a mechanism of fraud related to the placement of invisible elements on the site, the interaction with which the user performs without suspecting it.

A typical example of clickjacking is the creation of invisible elements on the site over buttons, forms, videos, etc. You can also use the movement of these invisible elements after the cursor on the page. Any click on the page, for example, to play an attractive video, leads to an imperceptible for the user to perform any action on an external site.
