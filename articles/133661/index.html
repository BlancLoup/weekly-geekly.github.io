Half a million participants! And there will be even more!

We reached the 500,000 registered OpenStreetMap member last night. Yes. This is a half-million army that can edit OpenStreetMap!



This project is entirely devoted to mass collaboration. Thousands of people come together on the Internet to build something great: a free map of the world. If you want to join these ranks, go to openstreetmap.org and click on the "register" link in the upper right corner. We will be happy to see you!

When you have an account on OpenStreetMap.org, you can edit the map and add objects. It's the most important. We need a lot of people, not just joined, but moving on the path of progress to the next stage, where the fun really begins. Increase your area and move to the "edit" tab to enter the map editing interface. If you have not tried it yet (we know that there are a lot of such), do it today! Contact the community if you have any questions or concerns. Let us know what is holding you back. In recent months, we have made great strides in simplifying editing a map with a stream of innovations from editors, as well as initiatives to create new documentation. But there is still a lot of work to be done on this path, so OpenStreetMap can reach the number of a million participants, and (more importantly) that all these participants can try editing!
