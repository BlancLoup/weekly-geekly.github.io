DevConf :: Go - June 17 in Skolkovo, an exclusive workshop on Go June 18 in TASS

Every year new development leaders appear on DevConf - in this years there were many interesting applications for Go , where developers are ready to tell how they use Go in production. Are you interested in Go? Join the Go community at 4Gophers.ru

The first report " Prometheus monitoring from the heart " is presented by Evgeny Pak, a developer with 10 years of experience, Lazada Senior Developer Go. He will talk about the experience of moving to Prometheus, and why they chose him, about the migration process and how the infrastructure is organized to collect metrics for Go applications. He will talk about the intricacies of calculating quantiles both on the client side (type of metrics Summary) and on the server side (type of metrics Histogram). You will also learn about the nuances that were encountered when working with Grafana, and how they were solved.

Vadim Madison will tell you how to prepare Go Microservice for production . Report on what is considered ready for production microservice. How microservice should be implemented, what functionality it should provide in order to consider the service reliable, transparent for support and monitoring. Using the Go-microservice as an example, a list of the required functionality will be given and the main approaches to the development, deployment and support of microservices will be described.

How can you write a highload application in (not?) The appropriate language and a report from Daniil Podolsky about Golang in action .

Alexey Solomonov will tell about the mailing of ~ 20K emails per minute using Go + SMTP + RabbitMQ = PostmanQ.

About audio prints for indexing all of the VKontakte music, Alexey Akulovich will tell - Backend developer VKontakte. He will talk about the technical aspects of the implementation of the now introduced system of audio prints.

Go fans will be interested in an exclusive master class from Go masters - Daniil Podolsky and Alexander Chistyakov

Writing a web application in the Golang language
Let's tell about how to make a web service in the Golang language:
  • Let's show how to work with long connections / web sockets.
  • How to organize the collection and display of statistics of the application
  • How to profile an application
  • How to organize the assembly and deploy
  • We organize load testing and see what our application is capable of.
