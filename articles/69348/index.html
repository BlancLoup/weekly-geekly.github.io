Yahoo Mail outsourced attachments

The most popular mail service in the world began cooperation with the social file storage service Drop.io and implemented it in the mailboxes of all users. Thus, now the maximum size of attachments in Yahoo Mail has been increased to 100 MB. For comparison, in Gmail it is only 25 MB.

In the program menu (Applications) there is another item "Add a large file¬ª (Attach Large Files). When you click on it, a new tab opens with a file upload form.

It should be noted that this option is not available from the letter editing menu (the limit of 10 MB is still there), but only from the program panel, as a separate application. At the same time, the service allows you to upload files without registration, unlike My Drive (ZumoDrive), which is also in the list of Yahoo Mail applications.

By the way, to use Drop.io you do not need to create a mailbox for yourself. Files up to 100 MB can be downloaded for free via the web interface or via the Firefox extension , also without registration.

One way or another, but there is a mutually beneficial cooperation: Yahoo received the necessary functionality without investing in infrastructure, and Startup Drop.io - powerful promotion and public relations, because 106 million people use Yahoo Mail services.

The file attachment utility in Yahoo Mail is 100% made on open APIs. That is, the same functionality from Drop.io you can implement anywhere. By the way, they will soon promise to make the second version of the JavaScript API, then the process of integrating their services will be even easier.
