As you can see, the names of the models now have four numbers in the name - this is done so that there is no confusion with the processors of the previous generation. The model range turned out to be quite complete and logical - the most interesting i7 series are clearly separated from the i5 by the presence of Hyper Threading technology and an increased cache size. And processors of the i3 family differ from i5 not only in a smaller number of cores, but also in the absence of Turbo Boost technology.

Probably, you also paid attention to the letters in the names of the processors, without which the lineup is very thin. So, the letters S and T speak about reduced power consumption, and K is a free multiplier.

The visual structure of the new processors:



As you can see, in addition to the graphics and computing cores, the cache memory and the memory controller, there is a so-called System Agent - there are a lot of things dumped there, for example, DDR3 memory controllers and PCI-Express 2.0, the power management model and the blocks that are responsible for the hardware level integrated GPU and for displaying the image in case of its use.



All ‚Äúnuclear‚Äù components (including the graphics processor) are interconnected by a high-speed ring bus with full access to the L3 cache, thereby increasing the overall speed of data exchange in the processor itself; what is interesting - this approach allows you to increase performance in the future, simply by increasing the number of cores added to the bus. Although now everything promises to be on top - compared to the previous generation processors, the performance of new ones is more adaptive and, according to the manufacturer, in many tasks can demonstrate a 30-50% increase in the speed of task execution!

The work of the above-mentioned technology will be compatible with two equally interesting developments, which are also present in all models of new generation processors. Two more features for which I did not find a more suitable place in the article - Intel Advanced Vector Extensions (AVX). Processor support for these commands improves the speed of applications with intensive data processing, such as audio editors and software for professional photo editing.

... and Intel Quick Sync Video - thanks to the joint work with software companies such as CyberLink, Corel and ArcSoft, the processor giant was able to increase performance when performing this task (transcoding between H.264 and MPEG-2 formats) by 17 times compared to the performance integrated graphics of the past generation.



Suppose there are processors - how to use them? That's right - along with them were also announced new chipsets (logic sets), which are representatives of the "sixtieth" series. To all appearances, only two sets are set aside for the thirsty Consumer-s, Intel H67 and Intel P67 , on which most new motherboards will be built. The H67 is capable of working with the video core integrated into the processor, while the P67 was endowed with the Performance Tuning function to overclock the processor. All processors will work in a new socket, 1155 .
