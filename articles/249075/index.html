Online service for tracking changes on sites

Habrahabr, hello!

Have you ever had a need to track the appearance of any information on the site, especially if there are no newsletters and RSS feeds !? Did you have to wait when they publish the exam results, new vacancies, tenders, update the price.

In our company, which deals with pension-insurance consulting most of the time, such a need arises constantly. At the same time, it was sad to spend time regularly visiting sites, and finding a suitable tool that could be assigned to this kind of task did not work: either it was necessary to install a separate program, or did not like the display of the result. Therefore, it was decided to create a service that would allow to track all changes on sites. And not only to track and receive notification of changes, but also to see the changes themselves directly on the site.

The result of the development was the Websvodka.ru service, which I will briefly describe here. Perhaps you can use it too.



The main functions of Websvodka.ru :
- Tracking and displaying changes on web pages.
- Controlling the addition and deletion of site pages (monitors hyperlinks on existing pages).
- Notification of changes in the results of search queries.
- Track the size of files posted on the Internet.

1. Changes on the page
After setting the page to control, you can see all the changes that occur to it. In the screenshot below, the added phrase is highlighted in green, and the deleted phrase is highlighted in red. Controlled rbc.ru page:

3. Searches

4. File sizes
You can add files to monitor and retrieve tracking results in the same way as for other types of content. A link to the file is set, and the service will inform about changes in its size, the dates of changes are recorded.

The four possibilities for tracking changes in site content reflect the main existing functionality of the service Websvodka.ru . To increase the reliability and speed of the modules, content loading, page analysis and storage of results function in parallel and are located on different servers. Empowering work continues.

We will be happy if the service helps in solving your problems!

PS How it works:
After registering on Websvodka.ru and logging in (for a simple acquaintance with the functionality, you can not register, but select the "Demo version" mode, but then your actions on the site will not be saved), go to the "Desktop" section, select the desired category , press the "Add" button and enter the address of what you need to watch. Further, Websvodka.ru regularly checks the object of interest and notifies you of its changes (you can set up e-mail notifications). Using the main functionality for free. The changes themselves can be seen by entering the desired category in the "Desktop" section.
