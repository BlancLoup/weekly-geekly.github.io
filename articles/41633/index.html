full Internet on a computer via iPhone

as everyone probably already knows, the iPhone cannot be used as a modem for a computer in an official way. No one specifically says why this can not be done, but most likely this measure was taken due to the fact that usually in contracts for the iPhone is unlimited Internet, and telecom operators do not want for cheap to give full unlimited Internet.

Previously, the problem was partially solved using the iPhone as a proxy server for the computer. As practice has shown, this is far from the best way, because many programs do not know how to work through a proxy server.

But now there is a good alternative - PdaNet . The program is absolutely free and it turns the iPhone into a WiFi router for the computer.


For obvious reasons, PdaNet was not allowed in the AppStore, so if you want to use it, you will have to do a jailbreak procedure.

The program can be installed from Cydia. You can find it either in search or in Sections => Networking

After installing the program, we create an Ad-Hoc WiFi network on the computer, then connect from iPhone to it.

Next, run the PdaNet program.

If your program displays the same it means everything is OK. Computer on the network. You can fully use the Internet.

More detailed instructions on using the program can be found here.
By the way, PdaNet works in the background, so you can minimize it and work with any other application.
