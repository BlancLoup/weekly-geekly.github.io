New versions of MobileNoter, or OneNote comes to Android


Despite the fact that Microsoft surprised everyone by launching its own OneNote application for the iPhone, we, the MobileNoter product development team, are not turning our efforts to bring OneNote to various mobile platforms.

When I talked about the development of MobileNoter on Habr√© last time , I mentioned that a version for Android was planned. A lot of time has passed, and the Android version ... appeared! And even in two guises.

The first, in general, is analogous to the long-existing MobileNoter for iOS: a computer and a mobile device are synchronized via a Wi-Fi / cloud server using a client for Windows. The latest version, Version 2.0.2, currently supports editing OneNote pages, and handwriting and audio recordings will soon be added.



The second, completely new version is called MobileNoter SE (Standalone Edition). In it, we decided to abandon the Windows client, the application works with OneNote files (* .one) directly. Due to this, files can be uploaded to the phone in any way, for example via a USB cable. And you can set up synchronization with a computer yourself, for example, using Dropbox.
