SkypeKit as an XMPP gateway

It is strange that after the advent of such an invention as SkypeKit, no one has done this yet.

Actually such a gateway allows you to exchange messages with Skype users, using an existing bunch of clients with Jabber support.

The licensed purity of such an invention remains in question, so that all references to Skype have been removed from the sources.

What is already working?


Almost all IM features are implemented (except for file transfer). For voice / video hardly anyone will take in the near future.

  • Registration through service discovery
  • Roster exchange (using XEP-0144 ), adding / deleting contacts, authorization requests, etc.
  • Exchange of presence, including statuses (aka mood text, if in skype terms)
  • Actually messaging
  • Multi-user chats (not using MUC from XEP-0045)
  • Adding users via jabber: iq: gateway (from XEP-0100 )
  • Partial user lookup (using XEP-0055 )
  • VCards (only getting ‚Äúalien‚Äù VCards for people from their contact list or from search results). You cannot change your profile.
