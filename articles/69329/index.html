Dropbox Guru (+250 Mb)

Dropbox's great tool has updated your site *, and now you can get an additional 250 MB of storage space for your files by simply completing five or six items from the starter list , thereby becoming Dropbox Guru's status:

  1. Take the introductory tour
  2. Install Dropbox on your computer
  3. Add files to Dropbox folder
  4. Install Dropbox on other computers you use.
  5. Create a shared folder for you and your friends / colleagues.
  6. Invite people to Dropbox

Thus, the maximum possible, at the moment, the amount of synchronized data in the free version is 5.3 GB (Starting 2 GB + 3 GB - maximum for the program to attract new users + 250 MB for the status of Dropbox guru)

What's new?

- Ability to select a group of files / folders through a web interface;
- Perform group operations (with long operations show a progress bar);
- Ability to download selected groups of files / folders in a single zip archive
- File search;
- Keyboard shortcuts in the style of GMail (to activate them, go to the settings and put a tick in the appropriate field, and then just press the '?' In the home folder and get a window with a description );
- Optimized version for mobile devices (non-iphone);
- Other improvements in navigation and interface.
