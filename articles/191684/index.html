BitTorrent sync is now available for iOS.



BitTorrent Sync is a synchronization protocol that is based on the Bittorrent protocol, it allows you to synchronize any amount of information between any number of devices.

In other words, this is a personal cloud with easy configuration.

Today, the company Bittorrent decided to please us Beta version Bittorrent Sync for the iOS platform.

It looks like this is the first p2p application that Apple has approved.


Opportunities:
  • Full Bittorrent Sync protocol support
  • Backup film
  • Scanning the QR code to add the secret key to the device
  • Russian language support


The application is free, you can download it from the App Store .
