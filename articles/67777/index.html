Java in Russian. Part Two: Internet Resources

In the continuation of the first part . Today we will look at Java resources on the Internet, which we divide into 4 parts: vendors, JUG, forums, news. And at the same time let's summarize.

Vendors


Sun microsystems


The main site of the Sun company contains purely advertising information, therefore we will consider the Network of Developers . So, what is there:
  • JavaFX is the hit of the season. As part of the promotion of Sun holds contests on this technology, you can also find documentation in Russian and even Wiki .
  • Academic programs - information about educational programs. These are mainly courses for teachers and study materials ‚Äî lectures written as part of grants.
  • Java SE, Java EE, Java ME, JavaCard for 2009 contain practically no updates, and in previous years there are perhaps translations of some articles.
  • The forum is pretty dead, with the exception of the section on JavaFX.
  • There are also translations of The Aquarium and The Planetarium English-language blogs dedicated to Java EE and Java news for all other platforms (SE, ME, FX, etc ...), respectively.

Ibm


The IBM website has a DeveloperWorks section, which publishes translated review articles on such new technologies as Groovy, Scala, Guice, Spring, Eclipse, as well as IBM rulers: WebSphere, Rational.

JUG (Java User Group) & OSUM (OpenSource University Meetup)


Jug

JUG is a regional community of Java programmers who meet with presentations and discussions of various Java technologies and products.
  • JUG.ru is the most famous JUG in Russia, founded in St. Petersburg, but meetings are held throughout the country.
  • JUG Voronezh - the latest announcements of meetings in 2008, organized by DataArt.
  • JUG Samara - never had a single meeting, it seems.
  • JUG Vladimir is a rather active JUG of Vladimir University.
  • JUG Omsk - rather OSUM, about them below.
  • JUG KPI - Kiev Polytechnic, previously there was a rather active JUG with a forum and a Wiki.
  • JUG Kherson - only 3 meetings, organized by DataArt.
  • JUG Ukraine - a review of JUG news in Ukraine, a list of Java companies, a forum that seems popular before.
  • JUG Belarus is a very active group, meeting with Sun, Adobe.

OSUM

Sun has created a social network based on Ning. It has groups of different universities discussing various Sun OpenSource technologies, including Java.

Forums


The only popular topic forum I would call JavaTalks . There are also subsections on Java in other communities: SQL.ru , RSDN , Vingrad , Sources and LiveJournal .

There used to be other popular Java projects containing sections for discussion: JUGA , JavaPortal , Javable . Now nobody uses them.

news


I did not find a news resource exclusively in Java. There are the aforementioned Sun blogs, sections on Habrahabr and Linux.org.ru . Probably, there are also blogs. Russian-language Java Planet is not enough.

Results


Recently, I conducted a survey on what Russian-language resources users are interested in using Java technology. Well, it's time to take stock.
  • There is practically no documentation in Russian. If you are going to seriously engage in Java - there is nothing to do without English. It is impossible to solve this problem: working with Java, you are using a bunch of different libraries from different vendors - to translate documentation on everything and to degrade it in an updated state is an unrealistic task.
  • Sun actively collaborates with universities in moving Java to the educational environment. It is not known what will happen after its prodalscheniya Oracle Corporation, which produces software for business and less invested in educational programs.
