DeaDBeeF 0.4.2 released

DeaDBeeF 0.4.2 released


Lovers of foobar2000 and deadbeef, rejoice! A new version of this beautiful player has been released. Outwardly, an unremarkable player (although I prefer such brevity) reproduces a huge number of formats, including CUE markup, which, under Linux, is supported by player units and that is terrible.
Fast, easy, hassle-free and just a pleasant player from a domestic developer.



The list of changes is huge (the reason for the envy of other developers of audio players):
  • Added translations to many languages
  • Expansion of the menu through plugins
  • Ability to format headers through launching system utilities
  • MIDI support
  • Many lossless decoders updated
  • Support for corrupted cue files
  • New icons
  • Improved support for global hotkeys
  • Creating a new playlist on a double-click from scratch in the list of playlists
  • Alt + [1-9] fast playlist switching
  • Many tools for formatting various headers
  • Accelerated "cold start"
  • Fixed bugs and accelerated the work of loading covers
  • Ability to save current position and playlist when closing
  • Scroll through playlists
  • And many other changes ...
