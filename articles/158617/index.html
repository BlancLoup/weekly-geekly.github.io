Working with PostgreSQL: tuning and scaling



Good afternoon, habrozhiteli. Much time has passed since the release of version 2 of the book on PostgreSQL - version 9.1 and 9.2 of this wonderful database has been released. A lot has also been accumulated on the practical use of this database, so I decided to release an update on the book. So, meet: "Work with PostgreSQL: adjustment and scaling", 3rd edition.

As before, the book explores PostgreSQL performance tuning, replication, and clustering issues. A list of changes can be found on the book page. Any suggestions or comments can be sent by mail (indicated on my blog) or write to github issues (or even make a pull request for corrections). Enjoy your reading!
