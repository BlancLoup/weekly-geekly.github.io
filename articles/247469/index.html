TremulaJS: Picture Streams + Momentum Engine + B√©zier Paths + Multi-Device. This library does not even want to be called a slashed word. Sortable.js




It happens when the interface requires the implementation of a pull-and-drop, but you do not want to connect massive jQuery and jQuery UI for the sake of it. And the replacement comes with a great Sortable.js, which does not depend on third-party libraries, weighs about 10 Kb, works on the native Drag'n'Drop API, has many options and a good event model, and plus is perfectly compatible with Angular, Meteor, Bootstrap etc. And he was written habravchaninin RubaXa from Mail.ru.

Flarum




Australian developer esoTalk rethought the classic understanding of the forum and created a KickStarter campaign to create Flarum. Unfortunately, the project did not raise the required amount, but the idea itself and the revolutionary interface for the forum definitely deserves attention. So Flarum went free on GitHub. The server part is written in PHP with Laravel, in the client part is Ember.

Redacted font




A very specific font that can be useful both for prototyping and in any other place where you need to schematically depict content. For example, Facebook until the content is fully loaded in posts displays content in a similar way. By the way, I once wrote about a similar Blokk Font. But Redacted is better - these are two different types with many styles.
