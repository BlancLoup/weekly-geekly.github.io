Windows programmers are starting to develop software for Linux

Two years ago, the number of Windows application developers has fallen, while the ranks of Linux developers have grown - this is now a trend.

According to analysts from Evans Data Corporation research house, compared to last year, the share of North American developers writing software for Windows from the total number fell from 74% to 64.8%.
