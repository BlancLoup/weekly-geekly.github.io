After 10 months, a new version of TrueCrypt 5.0 was born.

There are many programs for encryption and cryptography, but TrueCrypt has some interesting features, thanks to which it has been running on my two machines for almost a year. A new version 5.0 adds more different goodies ...

The latest version 4.3a was released in May 2007, and was downloaded over 1.2 million times. After 10 months, version 5.0 appears

Brief features of TrueCrypt (new items from the 5th version are marked in bold ):
  • Open source
  • encryption with all modern algorithms: AES, Serpent, Twofish, their double and triple combinations
  • Hash functions: RIPEMD-160, SHA-1, SHA-512 and Whirlpool.
  • work under Windows Vista / XP, Mac OS X , Linux
  • double the read / write speed in Windows
  • work through the command line and through the GUI in Windows and Linux environments
  • Ability to fully encrypt the Windows boot partition with pre-boot authentication
  • Replacing the LRW recording mode with a more reliable XTS
  • Ability to create containers in the form of files, partitions or disks
  • Impossibility to prove that a certain file (disk) is encrypted, there are not even any headers
  • The most delicious thing is the possibility of creating a hidden cryptocontainer inside another one that cannot be detected and / or even prove its existence even if you know the password from the main container (protection method from ‚Äúthermal cryptanalysis‚Äù :)
  • Encryption is absolutely transparent, real-time
