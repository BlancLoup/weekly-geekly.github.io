ENTrap - a small utility for Evernote

This topic may be useful to active users of Evernote, who are trying to use it a little more widely than just to keep a base of notes, in particular, to do business in the style of GTD. Project page: http://code.google.com/p/entrap/

What is it?


The first step GTD is to collect information. Just this utility and solves. Functions:
  • Allows you to enter text notes in one line
  • Called by hotkey
  • Floats on top of other windows (or does not float)
  • Uploads files and text to your Evernote using Drag'n'Drop, drop an elephant in the icon and not the text field. PDF, pictures are supported, maybe what else
  • Supports default tags and separate notebooks for text and files
  • Windows & Mac version
  • Free & open source
  • Inglish & Rush Langvisis Supported
  • Do not ask for passwords and the Internet

How it works?


The utility does NOT use the Service API available on the Evernote website, but uses the capabilities to write scripts for the Evernote client application (AppleScript for Mac and ENScript.exe for Windows). Advantages of this approach:
  • Notes jump instantly, no need to wait for sync
  • Do not need a password to synchronize and internet

There are limitations:
  • Portable version of Evernote for Windows does not have ENSCript.exe
  • Scripting capabilities, frankly, are limited, for example, you cannot import HTML
