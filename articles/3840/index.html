The format war is over, Microsoft believes

Novell has posted on its website Open XML translator , which allows you to open and save files in the Microsoft Open XML format using the open source OpenOffice application package, a free alternative to MS Office.

Office Software Manager Brian Jones noted in his blog that now users of Microsoft products have more opportunities to work with Open XML on various platforms. Despite the fact that we are not talking about spreadsheets or presentations, he believes that the format war is over: "If we really fought, now the war is over - and both sides have emerged victorious."
