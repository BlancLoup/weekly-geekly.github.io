Using CompositeUserType to connect entities in Hibernate

The post will be brief and very technical.

Task

There is a Java application that has a large number of ORM entities inside (Entity).
You need to implement the ExtendedAttributes entity, which can be attached to any other entity without further development .

Decision

CompositeUserType comes to our aid, which contains the class and id of the entity that we want to bind within. That's all the solution. And then - the code.

To reduce the post, imports, getters, setters and ‚Äúdefault‚Äù methods are cut out of classes.
