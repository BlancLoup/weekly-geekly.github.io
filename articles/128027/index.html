ICQ 7.6: in short



Not so long ago, we released another version of our desktop client, ICQ 7.6. Her independent review has already been on Habr√© ( veter - thanks for the speed and special thanks for the kind words), but I want to add a little from myself.
There were three main innovations in the version.

First, GTalk. Now you can add your Google account to ICQ and enjoy communication in two networks from one client. Authors of alternative clients may be interested in the relevant moments of the protocol.

Everything is pretty similar to how it is done with Facebook . The connection of the account is still through the HTML window, then you need to declare support for xmpp-contacts when starting the session (the package has not changed, but now implies readiness not only for Facebook, but also for Gtalk), and additional packages with contacts ([ snac-family = 0x0013, snac-type = 0x0008]) may be two, not one. To distinguish one from the other and draw the correct icons, you should use the TLV group (@ chat.facebook.com for facebook, gmail .com for gtalk). In the future, these are ordinary contacts with which you can work absolutely as usual. However, if something does not work - write to us, help.

Secondly, a new profile view window. Despite the technical simplicity of change, it is really important to us. I try not to say bad words about AOL, but I was ashamed of its old version. The new window is not a breakthrough or a revolution in user interfaces, but when viewing a profile, the profile is now really visible.



Thirdly, emoticons cones are now used by default.
