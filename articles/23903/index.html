Useful tips

If you suddenly start to jerk and brake when playing offline video, which you did not have when connecting the power supply to the power grid, go to Control Panel -> Power Options and on the very first tab set Power Schemes to Portable / Laptop mode . If, when the battery charge drops to 20-10%, jerks, brakes and hang-ups regularly begin, update BIOS to a more recent version (it is 0801 in Eldoepians, but you need at least 0906) and be happy with this handy thing .
