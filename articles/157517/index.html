Opera 12.10 Final



Today came the final release of the browser Opera 12.10 for desktops. The main focus was on expanding the functionality of the browser, but not due to the addition of new functions (although it was not done without it - see below), but due to the closer integration of extensions and the preparation of new APIs for third-party developers and web designers. In short, what and how.

Now about what's new for developers.

- Built-in support for the SPDY protocol for improved and more secure connections. Gmail and Twitter are already using SPDY in addition to the HTTP protocol;
- Fullscreen API;
- Opera now supports ICC color profiles and color management;
- Added full support for the latest WebSocket API specifications and now it is enabled by default in Opera 12.10;
- Context Menu API is the ability to integrate your services with browser context menus (also known as the ‚Äúright-click menu‚Äù);
- Resource Loader API - allows extensions to interact with the advanced features of CSS and JavaScript on web pages;
- Support for Flexbox and @supports allows web developers to better control how their website is displayed in users' browsers;
- Opera browser has received basic touch support for Windows 8 classic and pinch-to-zoom support for Windows 8. As a bonus, it also works in Windows 7;
- Improved support for MacOS devices. Opera 12.10 now supports features such as Notification Center in the latest version of Mac OS X, and also supports Retina Screen in the latest models of Mac computers. With the new version of Mac OS X, under the name of Mountain Lion, Opera 12.10 uses the system capabilities of ‚Äúsharing‚Äù content;
- Prefetch DNS - Opera performs a preliminary DNS search when placing the cursor over a link, which allows you to load pages on links faster when clicked;
