Weekly build Vivaldi 1.0.141.2

Hello!

Today we have released a new weekly build of the Vivaldi browser under the number 1.0.141.2. In it, we continued to work on the placement of tabs. In particular, it is now possible to open tabs in a single window, both vertically and horizontally. We also added the ability to place tabs tiles.

Spatial navigation began to work better and faster, in addition to optimizing this function, we decided to use indexing.

We also worked with notes and added the ability to add multiple attachments to one note.

However, we try not to stay within the Internet. In this assembly, we added the ability to view images in the local folders of your computer. You can also use the "Go" button when viewing images. Plus, the opportunity to work with a local disk using the system path in the address bar.

The changes also affected the auto-update system for the Windows platform. It has become easier and now you will not need to download the installation package - only the browser code will be updated. True, you will be able to evaluate these changes only after the next build is released :)

And one more important news. On Wednesday, we planned to release the next Technical Preview at number 3, so today's assembly acts as a testing ground for bugs and flaws, which would be a good idea to fix by the release of TP3. We are waiting for your help in this interesting case :)
