Toshiba launches hard drives with built-in data destruction



Now offers quite a lot of solutions to urgently destroy data on the hard disk (and on other storage media), if necessary. There are options that allow you to delete data even when you are on the other side of the planet, and you need to urgently destroy data - an SMS is sent, and the system destroys the entire contents of the hard disk. Toshiba is now going to release hard drives with an already built-in data destruction system that works when someone else‚Äôs computer falls into the wrong hands .

The family of hard drives MKxx61GSYS, in addition to the data destruction system, also includes a built-in encryption system, so it can be said that the device combines two systems - information protection and its destruction. As soon as the embedded security system detects that the disk is connected to an unidentified PC or another system, the data is immediately erased. In addition, there is a second scenario: you can also choose an option that makes the data inaccessible to the attacker (unfortunately, there is no detailed description of how this is implemented).

The owner of the hard disk of the MKxx61GSYS family can independently choose which data on the disk is to be protected, and what the device needs to do if it detects a ‚Äúleak‚Äù - whether to destroy the data, or just make it inaccessible. These hard drives are positioned by the company as a convenient and secure data storage system for government organizations, corporations and other associations and individuals who need to protect their information. The security system implemented in these hard drives is based on the Opal specification from the Trusted Computing Group.

The MKxx61GSYS model range includes hard drives of various sizes: from 160 to 640 gigabytes, with a SATA interface and 7200 RPM. By the way, developers use an encryption standard such as AES 256.
