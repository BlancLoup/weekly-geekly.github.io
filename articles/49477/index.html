The Chakra Project = Arch Linux + LiveCD + KDE 4

Arch Linux distribution has recently been at the peak of popularity. It is fast, constantly updated, and ... And unfortunately it is rather difficult to install. However, if you still want to see what he is capable of, then The Chakra Project is at your disposal - an Arch Live CD based on the default KDE 4 graphics environment (moreover, a slightly modified version of it ).
This distribution has a graphical installer (based on Qt 4 and rather resembles the openSUSE installer), an automatic hardware configurator, and a set of utilities for configuring the system and simplifying working with the package manager.
In addition, in the distribution The Chakra Project much attention is paid to the aesthetic side of the issue, just look at its screenshots .

Download and try the distribution here.
