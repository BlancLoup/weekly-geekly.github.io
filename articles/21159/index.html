Nokia 7500 Prism: Overview

Vobschem recently bought a new phone - Nokia 7500 Prism.
Here I want to describe the pros, cons and feelings of use.

I'll start with the sensations. The first thing I got used to (within half an hour) is a keyboard in which each button is flat and triangular. At first it was inconvenient, but now it is already used. The second thing I liked was the half-hectare (512 Mb) memory card that came with it.

Pros (IMHO):
1. This is Nokia :-)
2. In addition to 30 MB, there is also a half-hectare (512 Mb) memory card included.
3. Pre-installed Opera.
4. Thin.
5. Large enough screen.
6. Unusual design.
7. miniUSB port (I have a Bluetooth adapter, but the cable transfers the files faster).

Cons (IMHO):
1. A flat and slippery keyboard (for dry hands it is normal, but if your fingers are wet, they will slide).
2. Rounded and again slippery joystick.
3. There is no pre-installed ICQ / Jabber client (This is now also needed no less than Opera).

PS Do not judge strictly, this is my first and hopefully not the last post.
