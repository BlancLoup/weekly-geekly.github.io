Protection of information on mobile devices and mobilEcho 4.5

Many of us have an office with us all the time ‚Äî we respond to e-mail, review and edit files on mobile devices. However, convenience comes hand in hand with the risk of compromising important corporate information. To protect information, Acronis launches the product mobilEcho, the new version 4.5 of which we presented last week.

According to readwrite and Intel , 82% of companies allow employees to use their own devices. 60% of them do not have special data security policies for mobile devices . Companies implementing special security tools are often limited to MDM systems (mobile device management), which allow remotely update, configure, and clean devices.

At the same time, users are looking for simplicity and convenience in working from mobile devices, so for the exchange of corporate data they often use public services like Dropbox, SkyDrive, Box. As a result, multiple copies of confidential information files ‚Äúspread out‚Äù between services and devices are created. In addition, the risk of access to this information by intruders increases .

Moreover, if third-party applications were used to access corporate information, attackers could recover this information by taking possession of the device ‚Äî restoring temporary files, error log contents, copies of deleted files from the device‚Äôs file system log and dozens of different ways .

Acronis mobilEcho


Product mobilEcho - is a server and a mobile application for convenient access to files on mobile devices , while maintaining control over the files.



The server provides the ability to make mobile files available in network folders, in SharePoint collections and network storages, using current Active Directory access rights and additional restrictions set by the administrator. For example, an administrator can define a list of applications in which files can be opened. The history of file users is recorded in the audit log.

To communicate with the server, the mobile application can use one of five methods, depending on the infrastructure configuration of the company using mobilEcho.

  • HTTPS on port 443
  • VPN to internal network
  • Reverse proxy server with authentication certificate or a bunch of login / password
  • Proxy server Good Technology Proxy Server
  • MobileIron AppTunnel

Data transmission and data storage on mobile devices is carried out in an encrypted form, the product is FIPS 140-2 certified, which allows it to be used even by the military.

A mobile application, available at iOS and Android app stores for free, provides access to a set of directories and files available to a specific user.



Since users mainly work with documents in Microsoft Office formats and it is for them that third-party applications are used that put security at risk, in the version of mobilEcho 4.5 , there is a built-in editor for documents, spreadsheets and presentations, as well as the ability to insert annotations into PDF documents. As a result, the files do not leave the protected perimeter of the application.
