Simply Linux. Fresh release


The latest release of the Simply Linux distribution has been released. Release 4 (6.0).
Since the distribution of Simply Linux has always been focused primarily on the novice user, in the first releases of the system, increased attention was paid to the design and optimization of interfaces. In this issue, in addition to the new design, which will help to extend the summer mood, the developers have done a lot of substantial work. Behind small external changes there is a qualitatively new distribution kit, assembled on the basis of a fresh stable branch of the Sisyphus repository ‚ÄúThe Sixth Platform‚Äù. The distribution uses the kernel 3.0.3-std-def, the new version of the Xfce 4.8 environment and the most stable versions of the programs. Also fixed a large number of errors found in previous versions.

The image of the distribution is now hybrid - one for any use case. First of all, it is the work in the system from the media (CD or DVD), without installing it on the hard disk. For fast work, you need only 512 MB of RAM (although the system also works on 384 MB). Simply Linux can also be installed on the hard disk directly from Live by clicking on the icon on the desktop. The same image can be recorded for installation from Flash media.



Another major innovation is the Compiz window manager, which runs out of the box with Simply Linux. A lot of graphic 3D effects and convenient switching between desktops are easily accessible to simply Linux users.



The distribution has become much easier to open files for sharing. In the improved Tunar file manager, this can be done with a few clicks. The network manager's program for managing network connections has also been optimized: the ability to access the Internet via L2TP has been added, and support for Cyrillic Wi-Fi-points has been added. Connecting to the internet in Smply Linux has become even easier.



This release has greatly simplified the installation of clients for Skype and DropBox. Also, there was an opportunity to install the favorite ‚ÄúMint-menu‚Äù.



Simply Linux now has its own documentation , which greatly simplifies the development of the system by novice users. In addition, with the support of ALT Linux, the documentation for the Xfce window manager was translated into Russian for the first time, which is very important for users of not only Simply Linux, but also other distributions based on this working environment.



Interface design has also changed. In addition to the new wallpaper, changed the design of the installation program and download in accordance with the corporate blue and white style Simply Linux.



The Simply Linux project has big plans. Since the distribution audience has significantly expanded, in particular, there has been great interest from the business, it was decided to release an enhanced DVD version that will collect many other programs for work and professional creativity.
