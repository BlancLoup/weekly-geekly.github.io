WinRoll


The program manages the windows, and it does it so efficiently that I figured out the program without even looking at its options. Right-clicking on the title collapses the window into the title (incorrect definition of the header width in Google Chrome is noted), by the middle click displays a menu that allows you to put the window on top of all other windows, minimize, maximize, close all visible windows, and remove them to tray or do transparent.

Homepage www.palma.com.au/winroll
Homepage of the Russian version www.loonies.narod.ru/winrollr.htm
84Kb. Free (as opposed to the one I described in the previous AltDesk review), an open-source resource. Thank you donetss

Program to create desktops. The default control is not very intuitive, but it is interesting - there are four desktops, on which of them the user is located, we are informed by a painted corner of the tray icon. To switch to another desktop, you need to use the Win + combination of the required cursor key.

The program allows you to select text anywhere on the screen, including areas that usually cannot be selected using the mouse, such as data columns in a text editing program, a table in a file manager, or an error message from any program.
