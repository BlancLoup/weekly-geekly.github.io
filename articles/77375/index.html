KDE Software Compilation 4.4 Beta1 Released

New long-awaited release is now available to everyone. The developers have prepared a lot of changes. This release brings the long-awaited functionality and accumulates projects created within the framework of Google Summer of Code 2009, where more than 20 projects were presented within KDE. The next release 4.5 promises to be more focused on bugs.
Of the most delicious innovations can be identified:
  • Nepomuk Semantic Search has got a new fast backend (Virtuoso storage backend), new interfaces for searching in various applications. From this branch begins its widespread introduction into the application.
  • further development of Plasma Desktop . Updated KNetworkmanager and Plasma Netbook shell. Several new widgets are available. Now some of the widgets can be located in the system tray. A new interface for managing widgets. Plasma theme management is transferred to the system menu.
  • changes in the desktop control menu. You can also now change the background image on your desktop by simply dragging a picture onto it.
  • KWin . There are more options to control the options and animation of the Oxygen window theme. Aero Snap - basic tiling support. Grouping windows with control through tabs.
  • Pop-up animation support in the panel (using OpenGL). Kinetic scrolling.

This release is based on the Qt4.6 platform, which will positively affect the speed and memory consumption of the DE as a whole. Well, new features.
