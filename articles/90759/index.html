New Google Docs

Today, Google announced new versions of applications for working with documents, spreadsheets and diagrams. They will be available in the next few days.

Among the features:
  • Edit documents in real time (a la Google Wave)
  • Chat Sidebar
  • Field / Alignment Rulers
  • Improved work with images inside documents
  • Optimization of work with tables (increased download speed and response)
  • Ability to transfer columns
  • Simplified navigation between sheets
  • New schema creation tool ( Drawing Editor )

New editors are not working with Gears yet (planned to be introduced later on the basis of HTML5). Under habrakat - video.
