Etch-a-Sketch drawings using an Arduino-based game controller



In my childhood I had such a board, of the Soviet type. Honestly, I didn‚Äôt achieve much success in creating drawings on it. A couple of times I found real masterpieces created by Etch-a-Sketch fans on the web. Apparently, these people spend well a lot of time behind this device. Among the fans of Etch-a-Sketch is the Alpinedelta32 modder. This person was able to create a more convenient board control system using the Arduino platform.

The result was an interesting system, the output of which is the ability to control the drawing process on Etch-a-Sketch using a regular game controller with Nintendo. Those who have tried to draw on this board may agree that control through the game controller is much more convenient than control using those two guide wheels on Etch-a-Sketch.

It is clear that all this is not very practical, but as a demonstration of the capabilities of the Arduino and the capabilities of the developer itself is very interesting. How the system works can be viewed in detail in the video below.
