Google Chrome for Linux, add-ons and add-ons site

The build 4.0.221.8 has added support for add-ons. In order for them to work - no need to register the keys, as I understand it.

Already ready site with additions, much like the Mozilla analogue: www.chromeextensions.org

The extensions are managed on the chrome: // extensions / page (a space is left so that the parser does not eat).

I myself have already installed the RSS "find", AdBlock, iMacros, Smooth Scroll. Everything is working. It offends the concealment of the bottom panel. (Smooth Scroll was deleted right away)

What you liked. Disable \ enable add-ons so far without rebooting the browser. Only for RSS it was necessary to restart.
