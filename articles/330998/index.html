Fujitsu has developed virtual machine management technology that improves server utilization

Fujitsu Laboratories - a research branch of the Japanese electronics manufacturer - introduced an algorithm to increase the density of server placement by moving virtual machines around the data center. The new approach allows you to get maximum performance from power supplies, while adjusting the load.

/ Flickr / Bernat Agullo / CC

According to Fujitsu, the new algorithm improves the efficiency of using racks in the data center up to 90%, while simultaneously saving space in the computer room. The number of servers in the rack is limited by the value of their power consumption, which should be less than the capacity of the power supply. However, most servers typically run at 10‚Äì50% of their capabilities, using only a fraction of the available power supply capacity.

To increase the density of servers in the rack, you can use special software, which, to balance the load, limits the frequency of machines to reduce power consumption. However, this approach has a negative effect - performance suffers, which is unacceptable for business critical applications.

Fujitsu Laboratories have developed an algorithm for managing virtual machines, which, using the API, migrates them to dedicated backup sections, depending on the power consumption of each rack. This ensures high performance of individual servers and eliminates the possibility of exceeding the permissible level of power consumption. At the same time, virtual machines automatically migrate between clusters when the physical server stops working due to an error or during maintenance.

Fujitsu Laboratories have also developed technology that can determine the number of servers that must be installed in a rack. It does this based on statistical data (information about server load), calculating the frequency of migrations for each rack.
