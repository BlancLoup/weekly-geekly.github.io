Conference High Performance Conference 3.0

December 6, Moscow will host High Performance Conference 3.0 ( http://hpc.itmozg.ru/ ) - the third conference on high-load systems from ITmozg.

The program includes speeches by experts from Parallels, Microsoft Russia, Mamba, Jelastic Russia and an hour-long question-and-answer session.

As part of the conference, I will make a presentation on the use of the Windows Azure cloud platform for high-loaded tasks. The report will consider cases of projects already working with high loads on Windows Azure. The architecture of ensuring the fault-tolerant broadcast of the 2012 Olympic Games in London based on Windows Azure Media Services is considered.

We invite experienced developers, architects, system administrators to get together and share experiences, discuss all aspects of high-load systems.
