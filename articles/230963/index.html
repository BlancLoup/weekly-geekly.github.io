Microsoft has opened an account on Github

Microsoft has opened an account on Github , where it will publish the source code of some of its programs.

So far, little has been published: TypeScript web application development tool, which extends the capabilities of JavaScript, and sample TypeScript programs . But let's hope this is just the beginning.

The company's description on Github looks like an apology for past mistakes: "Microsoft has changed how the company is and is becoming more open in how we collaborate with others."

As you know, Microsoft has traditionally been considered the main "opponent" of free software and, in particular, Linux. In recent years, however, the situation is changing for the better. In 2010, a single Microsoft Open Source resource appeared, where the source code of Microsoft programs was published. And now the company has opened a Github account at all.

Probably, working within Github will attract more interest in free software from Microsoft. This is almost the established standard for joint development.
