Microsoft introduced the "social" version of Bing



Microsoft is trying to make its search engine popular, and it must be said that it succeeds. In any case, the market share of this search engine is growing, although not exponentially. In order to attract more users, the corporation adds new features to the search service. This time it was decided to add a search engine on Facebook.

Since the social network itself has a huge number of users, Microsoft‚Äôs search engine may well attract some of these users, taking them from other search services (read Google).

When searching for a specific query, Bing now shows Facebook user's posts in a separate column, which are relevant to the query (it is clear that the accounts of these friends are also shown). It is not yet clear how popular this function may be, but the developers have high hopes for it. By the way, not so long ago, Google complained that Facebook is a closed sector of the Network, in which a huge amount of information, and this information remains only in this closed world.

In Bing, in addition to the new social function, there are other updates. In particular, when displaying search results, the page is divided into three parts. The first is the usual search results. The second part is Snapshot, this column can help refine your search query. Well, the third column is the already mentioned social issue.

Of course, if the user is not registered on Facebook, then the new feature will not work normally. In addition, the Bing user must have an account in Windows Live to work with new functions. In this case, Bing will be able to find and acquaintances on the system Windows Live.

Experts believe that the social functions of Bing are both an advantage and a disadvantage of the search service. On the other hand, the condition of having two accounts simultaneously for Bing users can be called a disadvantage.
