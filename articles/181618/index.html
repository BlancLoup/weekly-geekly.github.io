Popularity of Javascript Framework

Interest in Javascript MV * frameworks caused their rise. Meteor , Ember , Angular , and Backbone are all really popular on Github. Measuring popularity is quite difficult, but the number of Github-followers can be a good indicator. Using data from the Github archive , you can visualize it visually (using a query on BigQuery and some other scripts).



If interested, talk about several points of inflection:

  • Backbone is the oldest project, but seems to be gaining momentum as fast as the others.
  • Angular released version 1.0.0 in June 2012
  • Meteor was announced on hackernews in April last year, followed by the first round of investment


Although this chart is a good indicator for measuring popularity, it is not accurate to judge the use of frameworks in real-world applications. Available data from the Github archive is missing data older than March 2012. There is no data for some short intervals (either due to Github errors, or from a Github archive missed).
