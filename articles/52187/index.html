Training videos and new podcasts on TechDays.ru

February 11 on the site TechDays.ru started the second wave of online seminars. In addition, more than 60 new online reports on application development and infrastructure management have been posted on this Russian outpost, Microsoft. Below are direct links to some of them.

If you do not have broadband access, then instead of videos you can download podcasts, or separate presentation files and code samples.

In the "Application Development" section:

* Overview of Windows 7 Application Development Features
* Build a custom report for management from any database in 15 minutes
* Introduction to the Azure Services Platform
* IEAK - create your own IE8
* Security in ASP.NET
* Manage IIS with PowerShell

In the section "Infrastructure Management":

* XP, Vista, Win7 - yesterday, today, tomorrow. Networking opportunities
* XP, Vista, Win7 - yesterday, today, tomorrow. Security technology
* Exchange Server - Disaster Recovery
* We write the first script and get acquainted with Windows PowerShell

Over the next month, new reports are expected.

In addition, the site has new convenient features:
* More comfortable and modern design
* Simplified registration system
* RSS feeds for reports by tracks and categories
* Video files in 800 * 600 resolution
* Short URLs on reports pages and the whole site.
