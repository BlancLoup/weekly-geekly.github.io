Finance Accounting Service Finobox.ru announces the opening of the SDK for creating applications.




What are applications?



Finance accounting in Finobox is organized in such a way that user data and services that perform certain functions on these data are logically separated.

The user connects only those applications that he needs, without cluttering up the interface with unnecessary icons.

What can be applications


Applications can be very different, implementing many tasks. For example, which applications already exist:
  • Formation of acts and accounts for entrepreneurs and LLC
  • Invoices
  • Tax returns for individual entrepreneurs and limited liability companies on a simplified taxation system
  • ...

Since the Finobox service has gone beyond Russia (there are users from almost the entire CIS), the scope of work is very extensive. For example, accounts, acts, contracts in the Ukrainian, Belarusian, Estonian specifics, the accounting of salary payments and reporting on them and much more.

Why Finobox Developer


Finobox, as a platform, offers a simple and efficient infrastructure for applications. The developer does not need to think about organizing the storage of user data, paying for the application, entering data, and so on. The developer can direct all the power of intelligence to the implementation of the business logic of the application.


After placing paid applications in Finobox, the developer gets 70% of their sales.

The main thing to start





For development under Finobox you need five things:
  • Computer to deploy SDK
  • SDK itself
  • PHP language proficiency (5.2+)
  • Understanding Application Building Rules
  • Lemonade "Buratino" (optional)

The SDK itself, the description and sample applications are located in the Developer Center . Deploying the SDK is as simple as possible. The SDK is a thing in itself that does not require configuration of paths, databases, and other things. By default, should earn immediately after unpacking the archive.

Privacy


Let us highlight a few points related to the security of end-user data. All applications, before getting to the "counter", are tested and censored. Thus, we, Finobox, guarantee the quality of third-party applications.

The infrastructure of the platform is such that the end-user data does not go beyond its limits, except in cases where the essence of the application is in the data export.

Current restrictions


So far, the development center is operating in trial mode. This means that there is no possibility to send ready-made applications for placement in a running system. All delays associated with this (agreements, regulations, exchange of requisites), we will settle within a week.

Also, while destructive actions to change the financial information of the user are not available. These functions are run-in and stress testing.
