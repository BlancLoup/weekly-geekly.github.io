Choosing a mass mailing service

In this topic, I would like to consider three services so that the reader can decide which one is more suitable for his business. Despite the fact that two of the subjects are integrated marketing systems, I will focus only on the possibility of conducting mass mailings. The candidates are the following services:

Interface, usability and security


Mailchimp




Beautiful and easy interface. Minimum graphics, maximum css. The response of the pages is extremely high even at low speed connection with the computer. Text prompts and training videos are met at every step, and it doesn‚Äôt distract from working with the service at all. Work is a pleasure. A conservative, slightly clumsy interface, which could not be more clear in nature. Of the graphics - only the logo and some icons. The service uses a secured connection. Page response time at low Internet speeds sometimes causes sadness, as well as failures with a security certificate. But I note that these problems were observed ONLY on slow connections.

Templates


Mailchimp




A large number of ready-made templates for mailings is an advantage, but I don‚Äôt undertake to say about their absolute cross-leveling (accurate mapping to all mail services / clients) because their styles are placed in the head of the document. The work of the template editor is divided into three modes: code, combined and visual editing. The classic visual editor is missing, while the built-in editor is based on its own service API. This means that for a visual work, the template that is laid out on the local computer must be optimized for the service editor. Of course, I may be mistaken, but I did not find file hosting for images that can be used in letters.

Pardot




In its toolbox, the service has a visual editing mode and a source code mode. A plain text format is also provided for the letter, which is sent along with the original imposing one, which reduces the risk of the letter getting into spam. Other services presented in the article cannot send additional plain text version. I would like to note that the visual editor has the ability to rewrite the source code of the template in its own way, which can lead to terrible consequences, but if you make these recommendations during the layout of the template, then there will be no problems.

The huge advantage of the service is that there is a toolkit for testing the template in many mail services / clients. The test takes a few minutes, which is manna from heaven for the maker-up, and man-hours saved for the employer. The service provides file hosting with a shortening of links for images.

Webasyst




While Mailchimp and Pardot allow you to save mailing templates, in Webasyst you can only send an email, after which it will be saved in the outbox. This outgoing message can later be used as a template. We can say that the mail interface of the service is just a mailbox with an address book. Perhaps for whom it seems more convenient, but I think this is a minus. You can work with the template both in the mode of visual editing and with the source code. The free version provides a file of 100 megabytes.
