In such cases, business representatives are ready to abandon the "extra" nines in uptime, but to get an opportunity to save, while retaining the same functionality. How is the cloud cloudite?

CloudLITE, like existing DataLine cloud services, runs under the control of the VMware vSphere 5.5 hypervisor.
Thanks to the built-in balancer (Distributed Resource Scheduler, DRS), the load on the cluster is optimally distributed. DRS continuously monitors the status of the cluster, and if one of the servers lacks some resource (processor, memory), then the balancer uses a live migration vMotion transfers some VMs from this server to less busy ones. This avoids the struggle for resources between virtual machines and guarantees the same high power for all CloudLITE clients.

Cluster fault tolerance is provided by VMware HA technology (high availability). It allows you to quickly monitor failures in the hardware of a virtualization cluster and, in the event of a failure of one of the servers, migrate VMs to redundant cluster servers and restart them there.

A distinctive feature of CloudLITE from other DataLine cloud services is the use of software defined storage class storage systems - VMware VSAN. This solution combines industrial-level drives with an NL SAS interface and PCIe cache caching flash cards into a single distributed storage.

The cloud is deployed on 4-socket servers Huawei FusionServer RH5885 V3 based on Intel Xeon processors. Each server has 4 high-performance 10G Ethernet interfaces onboard. High-performance Cisco Nexus 10G switches are used as network equipment.

All equipment is located in the Tier 3 NORD 3 data center in the north of Moscow ( video tour for those who wish ), which means that all elements of the engineering infrastructure are reserved (at least according to the N + 1 scheme), the premises and the surrounding area are carefully guarded.
