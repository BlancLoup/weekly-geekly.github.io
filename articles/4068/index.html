Access to personal search engine via IM-pager

American startup Kozoru launched a personal mobile search engine. Each user creates and configures his own search engine, and then gets access to it through a mobile phone or through an IM pager. The new service bears the straightforward name of BYOMS , which means ‚Äúbuild your own mobile search‚Äù, that is, ‚Äúcreate your own mobile search‚Äù.

Personal search engines themselves are not uncommon. Such services exist for a long time. For example, visitors to the Rollyo site or the Russian ‚ÄúPersonal Search Novoteka‚Äù have the opportunity to specify a list of selected websites for indexing. They then receive a unique URL to access their personal search engine and can search their favorite sites. Alternatively, you can open your search site for all to see.

BYOMS developers are developing this idea and are trying to make personal search engine more convenient. This service is extremely simple and straightforward. The user is registered, then compiles a list of resources for indexing (this can be a single website). At this stage, you can specify a number of parameters for the future search engine to work, for example, the appearance of search results, etc. Then the sites are indexed, and the user gets access to the system through the AIM pager (only this instant communication system is supported). In the AIM contact list, a new contact appears with which you can ‚Äúchat‚Äù. This is the search engine chat bot. It takes search queries and returns search results. Since many mobile phones support the work of IM pagers, you can even communicate with this AIM chat bot from a mobile phone.

Each chat bot in the contact list corresponds to one search engine. Thus, you can create several personal search engines - and they will correspond to several chat bots, for example, Music_Fan for searching music sites, Book_Resource for searching books, Find_Bazar for searching in forums and blogs. If you want, the chat bot you created can also be available to other AIM users.

This is a really interesting idea. After all, most users of ICQ are constantly active, and opening a chat window is faster and more convenient than launching a browser and downloading a website. In this way. In many cases, it will be more convenient to start a site search through an IM-pager, and then visit the desired page, than to open the site specifically for this and start a search there. In fact, you can make search engines for each site that you regularly use - and add everyone to your contact list.
