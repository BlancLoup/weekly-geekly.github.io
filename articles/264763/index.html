How Windows 10 collects user data



The information that Windows 10 collects user data is not new. Back in 2014, Microsoft published a privacy statement, from which it follows that its servers can transmit information about the programs used, the device and the networks in which they operate. This data can be combined with a user ID (Microsoft account), information about the email address, preferences, interests of the location, and much more is also collected.

The Czech edition of Aeronet.cz has published an investigation into an unnamed IT specialist who decided to track Windows 10's data collection activity. The study used the following tools: Destroy Windows 10 Spying, which blocks data transfer to Microsoft servers, PRTG Network Monitor, Windows Resource Monitor, and Wireshark. According to the researcher, Windows 10 is more like a data collection terminal than an operating system.
