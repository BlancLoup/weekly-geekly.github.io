"Beauty Salon" Corel PaintShop Pro

In just a few seconds, your photo can receive basic ‚Äúcosmetic services‚Äù in the ‚Äúsalon‚Äù from the company Corel. To do this, you will need Corel PaintShop Pro raster editor, which has a special set of tools for instantly improving the appearance of the face and figure in the photo. The set of "cosmetic services" includes: removing minor defects on the skin (acne, moles and wrinkles), whitening teeth, removing redness on the whites of the eyes, applying tan and giving a slender figure.
Tools designed for cosmetic correction of photographs are combined into one group and are accessible from the attributes panel of the Cosmetics tool (Fig. 1).


Fig. 1. Choosing the Cosmetics tool

Cosmetic tools have only two settings: Size , responsible for the size of the brush, and Intensity , which determines the degree of interaction of the tool with the original.

Elimination of defects

Defect elimination is the first cosmetic tool. It is designed to remove acne, moles and small wrinkles. To completely remove a defect, it is necessary to choose the size of the brush so that the inner circle of the pointer is slightly larger than the defect itself (Fig. 2). Deleted by click.


Fig. 2. Using the tool Eliminate Defects

You need to understand that this tool is automatic, which means that it will not always be able to accurately match the color and texture applied to the defect. Therefore, in complex cases, it should be used in conjunction with the Cloning Brush , which allows you to manually determine the source of the cloning. But in the example shown in fig. 3, the tool Remove defects quite satisfactorily coped with its task.


Fig. 3. Original image (left) and the result of its processing by the tool Eliminating Defects

Toothbrush

The second makeup tool - Toothbrush - is used for teeth whitening. In principle, there is no need to pre-select a fragment of the image with the selection tool. Simply click the mouse on any tooth - the tool will detect objects and whiten them. But if bleaching goes beyond the limits of a smile, then, of course, you should first perform a selection operation. In fig. 4 pre-selection was not carried out, the tool automatically identified and whitened teeth. The mouse click was performed on the yellowest tooth.


Fig. 4. Original image (left) and the result of its processing with the Toothbrush tool

Using this tool, remember: for whitening to be natural, just click once or twice. Otherwise the whiteness will become artificial and excessive.

Pipette

The third cosmetic instrument, the Pipette, is also intended for whitening, but not the teeth, but the whites of the eyes. To apply the effect, simply click this tool anywhere on the white of the eye. If bleaching goes beyond the boundaries of the eye, you should first select the desired fragments. The result of using this tool is shown in Fig. five.

Fig. 5. The original image (above) and the result of its processing with the Eyedropper tool.

A tan

The fourth makeup tool is used for tanning. The effect is applied by simply wiring the mouse, and there is no need to choose a color. In fig. 6 shows an example of the result of using this tool.

Fig. 6. The original image (left) and the result of its processing tool Tan

Slimness

The last tool - Slim - is designed to give a slim figure. In fact, it simply flattens the photo, which makes the image very unnatural (Fig. 7).


Fig. 7. Original image (left) and the result of its processing with the Thin Tool

Therefore, for the correction of the figure, it is better to use another tool that has much more settings and allows you to deform image fragments using several modes. This tool is called - Deforming brush (Fig. 8).


Fig. 8. Using the Warp Brush tool in the Extrusion mode.


Fig. 9. Original image (left) and the result of its processing with the Deforming brush tool.

To make the figure slim, two modes are used: Extrusion (first in the list of modes) and Reduce (third).
In the mode of displacement of mouse movement should be carried inside the object. The fragments, along which the brush passes, are shifted in the direction of the pointer movement. So that the border of the figure after the processing with the tool is smooth, it is better to use a sufficiently large brush size, for example 200-300 pixels, and the intensity value, on the contrary, should be set small, about 30-40.
Zoom out mode wrinkles out the parts of the image in the places where you drag. For accurate processing, the intensity value should be set to small, and a brush to make short strokes.
Also do not forget that there is another mode - Elimination of the deformation , which restores fragments of the image. Therefore, if you slightly overdo it with image processing, then you can cancel it, but not completely, but partially, by varying the Intensity and Step parameters. In fig. 9 shows an example of using the Warp Brush tool .
