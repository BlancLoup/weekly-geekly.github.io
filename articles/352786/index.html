How to use Azure for free (life hacking for students)

Today we will share an interesting life hack about how you can use our Azure platform for free without problems with reduced functionality. After all, everything is simple - you need to be a student.



I give the word to the author.

Hello to all. My name is Ilya, and I am one of the veterans of the Microsoft Student Partners program. I want to tell you about all the available suggestions for using Azure for students.

Azure for Students Starter


A lot of time has passed since Azure entered the market, but earlier, students had to pay the full cost of all the services used if they heard about new technologies and wanted to try them. And to help students, Microsoft in 2015 released a special offer for students. It had both a number of advantages (a subscription is created for 1-2 minutes and no credit card data is needed during creation) and its own drawbacks (from the services only a free web site and SQL database of up to 30 MB were available). Frankly, this is not even enough for the prototype of a student project. For 3 years, the number of services increased to 6, however, age restrictions of 16+ appeared.

So, Azure for Students Starter is the ideal solution for people who want to get acquainted with cloud technologies. However, the presented stack of technologies is not enough for a full development.



If this offer is of interest, then you can read it here .

Azure Free Account


More recently, a new Azure Free Account offer has appeared. It includes the use of about 30 very different Azure services, but it has some limitations. First, all services are limited in resources (750 hours of computing on virtual machines per month, 10 applications with the Azure App Service), and if their use passes beyond the limit, then the ‚Äúgolden parachute‚Äù in the form of $ 200 saves (money issued to first 30 days). Secondly, some services (the same databases and virtual machines) will be available for free for only 12 months. Also, to activate this account, you need credit card details.

In addition to limited services, there are 25 unlimited free services (for example, Azure Active Directory, Azure Service Fabric and cognitive services).
Here are some of them:

  1. 10 sites, mobile, or application APIs using the Azure App Service with 1 GB storage
  2. 30,000 Face API transactions per month and 20 per minute
  3. 750 hours Azure B1S Virtual Machine on Linux or Windows
  4. 2 million characters in the Translator Text API
  5. 100 Machine Learning modules and 10 GB storage access per hour for each experiment



As a result, Azure Free Account is an offer that is much more technological than the previous one. It gives $ 200 for the first month, but it requires credit card details. This offer is suitable for teachers who want to demonstrate the ability of Azure and will not go beyond the limits.

Azure for Students


Students can be attracted by the first offer, since there are almost no obligations in it, but it is not very rich in technology. On the other hand, the second offer is more attractive from a technological point of view, but for its activation it is necessary to enter credit card details. Moreover, the starting $ 200 is given only for 30 days.

Why am I doing this? Besides, Azure for Students is a symbiosis of the first two sentences! In it, a large stack of technologies (the same as in the Azure Free Account) is intertwined with starting $ 100 (no time limit) and the lack of credit card data. To get it, you need to have three things:

  1. Live ID (account in Windows 10)
  2. Phone (to verify your Azure account)
  3. Participation in the program Microsoft Imagine (free)



How to become a member of Microsoft Imagine?


Microsoft Imagine is a program to support students of technical (and not only) universities. It provides free software (in particular, Visual Studio and Windows Server) and access to student Azure.

In order to receive it, you must confirm your student status (mail on the university domain / Microsoft activation code / Shibboleth account). Shibboleth and university mail are not very common in Russia, and the Microsoft activation code can only be obtained at official student events at Microsoft or Microsoft Student Partners. Activation can be done in two ways: either through the portal Microsoft Imagine, or through the site . After that, you need to give contact information on the Azure portal.
