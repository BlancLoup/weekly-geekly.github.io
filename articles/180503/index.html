Update Kits for Windows XP x86 and Windows 7 x86-x64

A little note for system administrators who use Windows and are worried about timely updates. We are talking about offline updates for the most common client systems. On working computers with accessible Internet it is definitely better to use Windows Update, but there are situations when you need to update the system, and the Internet is expensive / slow / missing, or you need to update the distribution itself with the system, which will then be installed on many computers - then the set of updates is preferable to use , this will save your time.

At the moment, I have supported update sets for two systems:

Windows XP SP3 x86 Rus


The package is designed to integrate updates into the Windows XP distribution, while the files extracted from the hotfixes are integrated directly, thereby saving time and space on the CD. The version for installation on a live system additionally contains updates for all versions of WMP and IE.
