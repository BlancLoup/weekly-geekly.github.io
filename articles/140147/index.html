What is Fiddler ?
Fiddler is a Web Debugging Proxy which logs all HTTP (S) traffic between your computer and the Internet. Fiddler allows you to inspect traffic, set breakpoints, and ‚Äúfiddle‚Äù with incoming or outgoing data. It can be extended using any .NET language.

Fiddler is a proxy, including Internet Explorer, Google Chrome, Apple Safari, Mozilla Firefox, Opera, and thousands. You can also use Windows Phone, iPod / iPad, and others.

To debug applications have been written in Java, .NET, or using WinHTTP, see this page .


So Fiddler is a proxy that works with traffic between your computer and a remote server, and allows you to view and change it.
