Tor for everyone: if they block torproject

What to do if Tor was banned in your country and all links leading to the torproject project were blocked? Where to download the Tor-browser to a person who does not understand IT at all? For this case, the community developing this network offers the service GetTor Robot . This is a mail robot that, in response to a mail request, sends links to cloud file storages with a browser distribution.


Now the project is, as I understand it, in the testing stage - there is no link to it on the main page of all TOR projects, it is necessary to search through Google. Judging by the description, it is easy to use it: you send a letter to gettor@torproject.org, in the body of the letter you specify under which OS you need a browser. In response, a link to the dropbox comes, soon will also add a link to github. Download, install, use.

But in fact, everything was not so simple. Having decided to test this service, I entered my secret mailbox on protonmail (I did not openly write that I wanted to download the forbidden browser), sent a letter of request, and ... received in reply:
Robot's answer
Thank you for your request. I am you
Tor Browser.

Of options below:

windows
linux
osx

And I will send you instructions.

If you are not sure.

As long as I did not fight, all the other answers copied this one to one ‚Äî no reference came. Having a little experimented, I understood what the bug of this service was: in order for you to get a link, the body of the letter must either be empty or contain one word query. But protonmail always wraps the text of the letter in html, even if it is an empty string. GetTor doesn‚Äôt understand such wrappers - so it‚Äôs impossible to get a secret browser through a secret email!

The same applies to mail mail.ru. And only the letter with links came to gmail, the browser downloaded. The letter also provides checksums to verify the integrity of the distribution. True, the file is simply called torbrowser-install, which gives a simple version of the future blocking. Yes, and cloud storage is not always open, even github is sometimes blocked , and not only in China .

In general, an interesting idea - time will tell how much it will be in demand.
