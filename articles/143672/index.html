Easy firmware update Raspberry Pi from Hexxeh



Many gadget fans have already pre-ordered the Raspberry Pi mini computer, and some lucky ones have already received these devices. Now the situation has changed - a reliable and uncomplicated tool has appeared, as mentioned above, that allows you to make the Hexxeh Raspberry Pi firmware update simple and fast (of course, everything is relative).
