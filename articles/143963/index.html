Firefox Beta for Android: New Interface and Flash Support

Mozilla, with its Firefox, is late in the mobile revolution and is still playing catch-up with other browsers in the mobile space. However, it seems that Firefox for Android is gradually becoming a competitive player on the Google platform. Today Mozilla has released a beta version of a radically redesigned browser for Android. The Firefox beta for Android is now available on Google Play , and unlike the Google-owned Chrome browser for Android , it will work on any phone with Android 2.2 and a later version of the OS (including, of course, Ice Cream Sandwich). Although at the moment it is available only in English.

The highlight of this release is the new native IU. This update, however, also includes an improved version of what Mozilla calls " Awesome Screen ", the main page for Firefox Mobile.

In addition, the Firefox team has worked to improve browser launch and response times, as well as the graphical characteristics of the browser as a whole. Due to hardware acceleration, web applications and games should now run and work much better on Firefox for Android. The reworked user interface, of course, should also accelerate the browser.

Mozilla also promises improved text readability in this version by adjusting the font size.

The fact that the beta version now supports Flash (for now, though, only in the version for smartphones) is likely to surprise some. Although it should be noted that by default, plugins are loaded only by clicking on their area.
