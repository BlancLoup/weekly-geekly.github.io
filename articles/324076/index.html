Mitap MSK.NET Community



Hello everyone, friends!

In June 2015, the first meeting of the SPB.NET Community took place. I am sure that many of you have not only heard about such a community, but also watched excellent performances from SPB.NET meetings (and many attended meetings). The community began to develop very vigorously and enthusiasts from Moscow, having picked up a good mood, decided to create their own local community - MSK.NET .

Moscow meetings are held from August 2016 and already have a lot of fans. Many developers attend all meetings without exception. The main purpose of mitapov, of course, is the exchange of knowledge in the field of .NET development. As a subtask, you can select getting a fan, eating pizza and drinking coffee with cookies. Well, where without them!



It all started with a meeting where Nikita Tsukanov and Mikhail Shcherbakov (leader of SPB.NET) spoke.
Nikita talked about using Akka.NET , and Michael about WinDbg .

To date, 7 interesting meetings have been held with heated discussions (no one was hurt). Here is what we have already discussed (excluding the first mentioned meeting):

  • RESTFul API in your .NET application: how, why and why?
    Julia Tsisyk about creating API based on REST, existing best-practices, what should be avoided, what to look for when designing.

  • How to make Single Sign-On in a web application in 10 lines of code
    Vyacheslav Mikhailov on how to choose authentication and authorization mechanisms in the application and about the experience of using IdentityServer.

  • Error Handling in C #
    Ilya Fofanov about the advantages and disadvantages of using exceptions, about the Result monad and the philosophy of error handling in general.

  • Messaging systems on the example of MassTransit.
    Jacob Cook about systems based on asynchronous messaging.

  • Receptions optimization Desktop applications.
    Sergey Sentsov about various optimizations using Prefetech, Superfetch and others.

  • What is useful in parsing dumps for .NET developers?
    Alexander Rakhmanov about the process of collecting dumps, analyzing and correcting them, embedding this process into the application development life cycle.

  • HoloLens for non-gaming applications.
    Egor Bogatov on how to develop non-gaming 2D and 3D applications / holograms for HoloLens mixed reality glasses in C # without using Unity and without experience in game development.

  • State of the .NET Performance
    Adam Sitnick about the latest changes in .NET Core and C # 7 regarding performance.

  • Theory and practice of .NET benchmarking. (there are only slides)
    Andrey Akinshin about the intricacies and wisdom of benchmarking in his signature hardcore style with sabzh puzzlers.

  • IoC / DI on the example of Autofac.
    Ilya Efimov about IoC / DI and IDisposable specificity in this context.

  • Design and development of modular applications.
    Kirill Maurin about his own micro-library with open source code, which allows to build TRUE-modular applications

  • Automatic search for errors.
    Vladimir Koshelev on finding errors through static analysis.



Mitapas were held at CUSTIS, Yandex, Kaspersky, Microsoft, Sbertech, Odin (Ingram Micro). So, a visit to mitap is not only an opportunity to eat a liver, have fun and share experiences, but also an opportunity to see who lives in IT companies. For example, in Odin they made an interesting excursion around the office, which the developers really liked.
