This software is a real gift for diligent and careless students.



Probably, among readers of Geektimes there is no one who at least once did not hand write a mathematical equation. Well, to build graphs of equations is a common thing for any student, student or specialist. However, it takes time, which is always short, to build a graph using a pencil and ruler (and other improvised tools).

Now there is an application that saves this time. We meet Mathpix , which the developer himself calls "the first program that can solve handwritten equations". Everything works very simply: we launch the application, the phone camera is activated. The application takes pictures of the formulas written on paper, and the data is sent to the server, where it is analyzed (everything takes only a few seconds, judging by the demo video). After analysis, the screen displays the tools for working with the equation - for example, plotting.



According to Nicholas Jimenez, an application developer, in the server part, self-learning algorithms are used that become ‚Äúsmarter‚Äù with the increase in the number of program users and the number of equations analyzed.

It is possible to use software for plotting equations, solving integral, trigonometric equations. In this case, not only the result is issued, but also the intermediate stages of the decision.

Of course, Jimenez understands that the software can be used by unscrupulous students of schools and universities, but according to him, the program is designed to help students, schoolchildren and specialists in difficult situations. With the help of Mathpix, you can quickly clarify for yourself the incomprehensible moments in the process of solving equations, in the software there are training moments.
