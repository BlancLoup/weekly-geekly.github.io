Skype for mobile phones - Beta

On April 24, Skype officially announced the release of the first public beta version of the Skype client for mobile phones with Java support . Skype can work on almost 50 models of Motorola, Nokia, Samsung and Sony Ericsson phones. But in the near future, this list should be significantly expanded.



The beta version of Skype for mobile phones allows you to use the functions: chat, group chat, a list of contacts with presence status, receive calls from Skype users, receive SkypeIN calls. But to make Skype and Skype Out calls so far can only residents of seven countries: Brazil, Denmark, Estonia, Finland, Poland, Sweden and the UK.

The restriction on outgoing calls can be explained by the principle of operation of the new client, which is well known to users of the iSkoot program, which was previously offered to Skype users for mobile phones. Support of the list of contacts, chats and call initialization occurs via GPRS connection. The calls themselves are made through a local gateway, to which the subscriber‚Äôs phone is dialing via GSM. Like the principle of callback, this method is designed to make the connection reliable and of high quality, which the GPRS connection cannot provide. Naturally, the call charge is increased by the cost of the call within the cellular network. At the same time, as practice shows, Skype calls to long-distance and international destinations still remain substantially cheaper than calls at the rates of cellular network operators.

In any case, the appearance of the official Skype client for mobile phones is an important event and a big step towards Skype users. At present, the client has Beta status and is not available for all phone models. This circumstance, as well as restrictions on making an outgoing call for many countries, allow us to recommend using at the moment alternative Skype clients, such as IM + for Skype and iSkoot (with GPRS connection) or Fring , which can be used within WiFi networks.
