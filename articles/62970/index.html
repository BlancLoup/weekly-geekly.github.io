Microsoft revealed retail prices for Windows 7

Microsoft has finally opened a curtain of secrets about the pricing policy for the upcoming release of Windows 7. In addition, general information was also announced regarding the future OS release, update information, and special conditions for pre-order, which are valid only for some countries.




Windows 7 Upgrade Option Program starts today, starting from June 26! Any user who has purchased a new computer from an OEM manufacturer or dealer participating in this program with the preinstalled OS Windows Vista Home Premium, Business or Ultimate will be able to upgrade the OS to the corresponding version of Windows 7 for free or almost for free. Windows 7 Upgrade Option Program will be valid until January 31, 2010. It is also worth noting that this program is global, and operates almost worldwide.



order and purchase page from store.microsoft.com

In conclusion, I would like to note that for all loyal Windows users, Microsoft offers a special, time-limited offer, which, unfortunately, is valid only for a limited number of markets. Microsoft offers the ability to pre-order Windows 7 at a discount of over 50%. This means that, for example, in the USA it will be possible to make a preliminary order on the OS, paying only $ 49.99 for Windows 7 Home Premium, or $ 99.99 for Windows 7 Professional.
