What's new in Parallels Plesk Panel 11

Hello. A couple of days ago, the Parallels Plesk Panel 11 hosting panel came out. In Russia, as in other countries of the world, preparations for the launch began in a few months. In December of 2011, we asked habrayusers what they lack in the current Parallels Plesk Panel 10.4. Some time later, a short review of the capabilities of the 11th generation beta of Plesk was ready. Under the cut is a very short, literally point by point, description of key innovations in Parallels Plesk Panel 11.

To summarize, the bulk of the changes in the panel touched upon three things: increased productivity through the use of modern web technologies, increased security solutions for all groups of users and expansion of tools for small and medium businesses based on its presence in the network.

  • Nginx support. Now the most popular web server of the .ru domain zone can be installed from the panel automatically - the alecksey user request is heard. Nginx in the Linux version of the panel helps to significantly save server hardware resources.   • New SDK. The long-awaited opportunity to make your own changes to Plesk, so necessary in local markets. Development kits are available for both Windows Server and Linux.
  • Prokachan supports the web development platform Microsoft WebMatrix. Plesk can now install MS Webdeploy (server side of MS Webmatrix) - it turned out that not all users can do this on their own.
