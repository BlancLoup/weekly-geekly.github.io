Expression Studio 3 released

The Microsoft Expression product line has been updated today, including new versions of Expression Blend , Expression Design , Expression Web, and Expression Encoder.
Download the full set of products in one file here: http://www.microsoft.com/downloads/details.aspx?FamilyID=7e2f033b-c6b5-4565-93a5-a6048246ce28 .
Updated grocery site - http://www.microsoft.com/expression .
The updated community site is http://expression.microsoft.com .

Further links to individual products and additional details.

Expression Blend 3 + SketchFlow



What's new:
  • Prototyping with SketchFlow.
  • Import Adobe Photoshop and Illustrator files with saving layers, graphics and text elements.
  • Support for Silverlight 3 and WPF effects directly in Blend.
  • Annotations - the ability to leave comments on individual elements for effective teamwork.
  • The control panel is now a separate Assets panel, which can be left open for the duration of the work.
  • Creating templates / skins based on selected objects via the Make Into Control option.
  • State Improvements (States) - improved Visual State Manager, editing and previewing transitions.
  • Behaviors (Behaviors) - the ability to add interactivity to the program without writing code.

  • Improved data handling - using sample data to facilitate prototyping, improved data binding (Data Binding).
  • Silverlight 3 support - including the ability to add work outside the browser and cache libraries directly from the project menu.
  • Integration with TFS - now designers will learn what check in, check out and merge are.
  • Improvements in working with animations , including EasingFunction ‚Äî ready templates for changing parameters.
  • Improved support for working with fonts - including the ability to embed fonts in Silverlight and WPF projects.
  • Improvements in the workspace , including docking-functionality, improvements in work with gradients, layers, and much more in detail.
  • Code support - editing directly from Blend, IntelliSense, auto-formatting, and others. There are buns.
  • Extensibility - a mysterious Blend SDK is put together with Blend;)
Download Expression Blend 3 : http://www.microsoft.com/downloads/details.aspx?FamilyID=e82db5e2-7106-419e-80b0-65cce89f06bb

Expression Design 3



What's new:
  • Import Adobe Photoshop files - with support for layers.
  • Improvements in the workspace - now all panels have docking-functionality.
  • Silverlight 3 support - you can create graphic elements and export them for use in Silverlight applications.
  • Export effects - for Silverlight and WPF, for example, DropShadow or Guassian Blur.
Download Expression Design 3: http://www.microsoft.com/downloads/details.aspx?FamilyID=10e4a530-2c96-4aff-b00c-a154501876bc

Expression Web 3 + SuperPreview



What's new:
  • SuperPreview is a special utility for debugging the display of pages in various browsers. See also here .
  • Snapshot Preview - an updated preview of the appearance of the page directly while editing the code.
  • DeepZoom - integrated insert support for DeepZoom image collections (Silverlight and Ajax versions)
  • Silverlight - built-in support for inserting Silverlight video players and applications with the ability to select and customize skins.
  • Improved FTP support ‚Äî transfer multiple files at once (rather than one at a time), support for secure protocols.
Download Expression Web 3 : http://www.microsoft.com/downloads/details.aspx?FamilyID=0a73a3a7-3e06-4125-b3c6-f9c10387e9cc

Expression Encoder 3



What's new:
  • Screen capture - now you can record screen castes directly in the Encoder.
  • Several clips on the Timeline - now you can make videos from a variety of pieces, inserting them anywhere on the timeline.
  • Adaptive Streaming Support - for IIS Smooth Streaming.
  • New Silverlight templates - an updated set of video players, including support for H264, playlists, splitting into chapters, playing in offline mode and in a separate window.
  • Save encoding settings to Presets for reuse.
  • Extended H264 support - with many options for fine-tuning.
