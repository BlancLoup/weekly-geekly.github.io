New Life Telebreeze Player

Today we released another version of our IPTV player with a completely new trendy Flat design and with many improvements that affected both the user interface and the software part. The update took place at once on all platforms, starting with the most popular desktop and mobile Windows, Android, IOS, MacOS, and ending with Smart TV, Mag250 and Roku. I would like to say that we have invested our whole soul in it and it turned out more than worthy.



ALARM: There are a lot of pictures below, as it assumes a review on a new design.


Introduction


It all started a couple of months ago. We worked on the concept of a new design with one cool design team , eliminated the wrong options, and set about creating the foundation. The new design is not just external changes, it is a different structure and a different vision, more convenient and improved.

At this stage, the basic version is made, which, from time to time, with each new update, will be replenished with various buns that will make the player more ‚Äúsmart‚Äù, convenient and attractive for the user.



Login authorization screen: BEFORE - AFTER



List and EPG


A little about the changes themselves. First of all, with regard to the screen selection of TV channels and EPG (program guide and archive programs). Previously, it was two different screens, that is, the user had to click on the program icon to see the EPG. Now the user clicks on the channel and an EPG appears on the left of the screen, where the program that is currently in progress is highlighted in red.



Old interface, with old EPG structure and channel list



Now, under each channel, it became available to view the progress bar, which shows the duration of the program, which is now on the channel. Enough convenient thing.



Well, the last addition on this screen is access to viewing channel information and the ability to add a channel to your favorites by clicking on it, after which the channel will be displayed at the top of the list.

Categories


With the old design, the after-start window of the channel categories has gone. That is, after authorization, it was necessary to select a category, and then the channel itself. We thought it would be more convenient if this menu appears from behind the screen, and initially, after logging in, the user will see a list of all channels.



On the mobile device, this list of categories appears on the left side of the slide. Removed as well, only in the other direction. Very convenient and intuitive.



Video


Also changes affected the video display screen. And then there were several important upgrades. First, the scroll bar used to show the range of the entire archive, that is, about 24 hours. It is inconvenient, because if the user wants to waste this particular program a couple of minutes ago, then this is definitely quite difficult. Now the scroll bar shows the time range of the particular program that is currently underway. It is much more convenient to rewind and search for the necessary moment.

Secondly, the name of the channel, the name of the program that the user is watching and the time of its broadcast are shown not in fullscreen. In the future, right in this screen, the user will be able to choose any channel and program with the help of me.



As mentioned above, this is only the basic and the simplest version of the player with a new design. In the future, it will acquire new functionality, for example, video thumbnails and the ‚Äúmetro station‚Äù system when switching between programs.

This screen has only undergone external changes. Nothing more can be said yet.

Personal Area

Each user has his own personal account. Now it is implemented with a minimum of functions. In the future, it is planned to add a billing system to the player, that is, a payment system for packages, films, etc. Also, the user will be able to communicate with his friends who will use the player.



Video on Demand (VoD)

A separate topic for the IPTV operator is video on demand. We have created the ability to add various movies, videos and other content that users can buy, watch, rate, comment, and recommend to their friends.



There is a new screen, which was not in the old design. This is a screen of information about the film, which he bought and wants to see. The screen displays various information about the film, it is possible to watch a movie trailer, as well as rate this film. In the future, it is here that users will be able to leave their comments and share their opinions about any movie.



Total

Each new problem gives us even more strength and energy, because with its solution we gain invaluable experience, which we are trying to implement in the further development of our product.
