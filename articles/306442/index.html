Choosing the best online CSS compression service



Currently, there are a huge number of web tools for optimizing CSS files. Most of them have their own algorithm of work (there are, of course, duplicates). The effectiveness of the programs themselves is also significantly different. In addition, some optimizers can ‚Äúbreak‚Äù CSS code, after which styles become invalid. As practice shows, the percentage of such "useful" tools is quite high.

This article is a test report of the most popular online optimizers. The experiment was conducted on 3 CSS files. Links to each of them are given in the results table.

The main goal of the work is a comparative characteristic of services in terms of the main indicator (degree of compression). The presence of various settings, usability and other minor factors were not taken into account.
