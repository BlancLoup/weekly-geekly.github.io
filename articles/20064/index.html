Zonbu - an unusual mini-computer running Gentoo Linux

Warning: the topic is in addition to the information from this post .

An unusual computer went on sale under the name Zonbu. The novelty is a mini-PC consisting of a motherboard with an integrated graphics core, a VIA C7 processor (1.2 GHz frequency), 512 MB of memory, a 4GB flash drive and a Wi-Fi module. To get started, the user only needs to connect a monitor and keyboard with a mouse.

At first glance, everything seems quite standard: a sort of ASUS EeePC analogue, only for the desktop segment. However, not so simple.

Zonbu is interesting primarily because it uses a special service to store information. After connecting to the Internet, the computer automatically transfers information from a flash drive to the server, pre-encrypting the data. All subsequent changes to the contents of the drive are also taken into account and saved on the server. Even if Zonbu stops working, the data will not be lost. If desired, the user can access their information on the server from any other computer. Unfortunately, the service is not free: once a month you will have to fork out for an amount of about 375 rubles ($ 15).

The list of software that comes with the Zonbu package includes the Gentoo Linux operating system and about twenty programs of various types. If a two-year subscription to server services is issued separately when purchasing a computer, the price of the device will be about 2.5 thousand rubles ($ 100), if for a year it is 5 thousand rubles. Without a subscription Zonbu will cost the buyer somewhere in 7.5 thousand rubles.
