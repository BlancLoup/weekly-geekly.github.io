Project management system

Returning to the question in my blog: Advise a solution - project tracking system.

I think many will be interested to know how I decided it.
After a long search and study of the sent links I stopped at
GBTask .

The system solves the main task that has been set, namely
control over the implementation of tasks (what stage is located).

Of the minuses, I would like to note that the system captures the time of the task, and does not allow to automatically record the moments of the task status changing, which is useful in my case. To add this feature, you will need to edit the code a little bit, for which there is no time now.

In its original version, the system had a limited number of statuses, such as the task is accepted, in progress, and executed. But this script is easily improved if you need other task statuses, for example, sent for client approval, etc.

The great advantage of the script is its simplicity: after translating it into Russian, managers mastered it within a few minutes.

The script requires MySQL, written in PHP, but read and edit very easily. Many thanks to the developer: Naomi C. Martinson.
