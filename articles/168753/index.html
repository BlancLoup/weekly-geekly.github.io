Groundwork - interesting CSS / HTML5 UI framework



Plowing the expanses of "endless", I came across an interesting CSS / HTML5 UI framework - Groundwork. According to its developers, it has one of the most advanced responsive templates in the world. Thanks to the incredible flexibility of the framework, you can get sites that meet the latest requirements.

At a cursory inspection it was found: 5 ready-made responsive templates, fluid-grid mesh system, a set of components with styles (Buttons, Messages, Navigation, Tabs, Data Tables, Forms, Tooltips, Orbit).

Groundwork is located on Github with fully open source. License - Apache License.
