Free screencasts to explore Go

GoCasts.io has published a small selection of screencasts that teach Go programming. A total of three screencasts from Alan Harper have been published. It's nice that the examples chosen are not abstract, but the most practical examples from real life.

# 1 - HTTP Get
How to implement a simple HTTP request using the net / http package in the standard Go library.



# 2 - JSON Parsing
Use Go to get the JSON file from the web service and parse it.


# 3 - Parallel Processes with Buffered Channels (Concurrency with Buffered Channels)
Channel management using Go to communicate between different parts of a parallel program and as buffered channels provides a blocking function.


It will hope that the collection will be replenished and these three things will not be limited. You can leave your email and receive notifications about new screencasts.
