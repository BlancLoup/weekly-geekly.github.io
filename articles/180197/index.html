KolibriOS came out on Kickstarter to spend its Summer of Code

KolibriOS is an open-source operating system based on MenuetOS (fork occurred in 2004), requiring only 8 MB of RAM to run and is being developed by the Russian development team exclusively in assembly language. Its distribution takes one 1.44 MB-floppy and, nevertheless, includes a text editor, image viewer, browser, a set of 30+ games, various tests, system and application utilities. FAT12 / 16/32 is fully supported, in read mode - NTFS, ISO9660 and Ext2 / 3/4. Drivers for popular sound, network and graphics cards are written.

This year, project administrators have applied to participate in Google Summer of Code 2013, but their proposal was rejected. And Google wrote that they like the project, but because of the limited budget, they can not accept all of the 416 applicants. Since the list of KolibriOS innovations that were planned to be implemented at GSoC '13 is quite extensive , and has already been included in some ways in the development plans, it was decided to organize its SOC. Since the project is non-commercial, and on a donate basis a small project is still difficult to quickly raise funding, the developers have chosen Kickstarter. The campaign was launched 3 days ago, from ¬£ 5,000 so far collected over 20%. Fundraising will end on May 31.

You can download KolibriOS and decide whether to support the project on this page: kolibrios.org/ru/download.htm
Kickstarter campaign address: www.kickstarter.com/projects/kolibrios/kolibrios-help-us-hold-our-own-summer-of-code-2013
