Dropbox 0.7 - new version with support for LAN sync, as well as DropboxPortable for flash drives

Today a new stable release of Dropbox has been released - a popular file sharing service and data synchronization service. The new version has the number 0.7.79, the main innovation - support for LAN Sync. For those who are not familiar with this interesting service, I can recommend to see a visual clip (in English), or read a short description (in Russian).

What makes LAN Sync?


LAN Sync is a long- promised innovation that significantly increases the speed of synchronization within the local network. Previously, all changes were transmitted exclusively through the Dropbox server, which limited the synchronization speed to the width of the external channel. Now, the first thing Dropbox checks is whether there is a modified file on your local network, and if it exists, it downloads from there. However, this function does not exclude the need to first upload a new / modified file to the Dropbox server.

Or maybe there is something new?


Also in version 0.7, buttons were added to the Finder toolbar under OS X. In addition, a standard set of improvements was announced:
  • Optimized consumption of RAM;
  • Increased performance;
  • Improved stability.
From the above, in the client for Windows, the appetite decrease in RAM memory by about 25% is immediately apparent, which is not bad in general.
