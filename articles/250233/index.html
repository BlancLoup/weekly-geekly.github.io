EveryLang - translate, check spelling, switch and show the current keyboard layout

EveryLang is a tool with which, in any program, you can easily translate any text, check spelling, and also switch the keyboard layout on the fly and show the flag of the current input language in the mouse pointer and text cursor.

All these features are already implemented in other programs.

Translators: Translateclient , QTranslate , Dicter , etc.
Spell Checker : Spell Checker , ORFO solutions, LanguageTool , maybe there is something else.
Switching Layouts: Punto Switcher , Arum Switcher , Key Switcher, and a few small enthusiast projects.
Current Input Language Indicator: Aml Maple .

Each of them has its own characteristics, pros and cons, some are paid, some are buggy or just uncomfortable.
In general, I‚Äôm tired of keeping this whole zoo on my computer, keeping track of their updates, setting up, getting on with their not always good behavior.
And I decided to make a bicycle for myself, so that everything and everything worked well, did not irritate and please the eye.
In the end, meet my small project EveryLang .





Translator


Translation into more than 30 languages
The program uses the services of Google, Microsoft and Yandex, which help translate texts to any desired language.

The translation can be made using the main window, for this you can paste or type the text, all is cleared with the Esc button.



You can also translate text by simply selecting it in any program and pressing Ctrl twice. A window will open with the translation of the selected text, you can copy it, or replace the original text if the field is being edited.



Hot keys, translation service and main languages ‚Äã‚Äãcan be changed in the settings.



Spellchecking



The spelling check is implemented using the ORFO service, and there is no need to choose a language for checking, the system will determine it itself and produce the result. To check the spelling you need to select the necessary text and press the key combination Ctrl + F7, and if you want to check the text from the main program window, just press F7. While there is no punctuation check, maybe in the future I will fix it.



Switch Layouts



EveryLang can easily be an alternative to Punto Switcher, although there is no auto-switching, which I have never used, and according to many reviews, others also do not favor this function. In the future I will add autoswitch, at least it is interesting in terms of implementation.

Opportunities

  • Works everywhere
  • Independent of installed input languages
  • Switch by clicking on Break, double Shift or double Scroll lock
  • Double-click on Insert toggles the layout for the entire line, not just the last word.
  • Switch layouts for selected text by pressing Ctrl + ~
  • Change the register of selected text by Win + ~

To work correctly, you must close the Punto Switcher and all similar programs.



Current Input Language Indicator


The functionality is borrowed from the Aml Maple program, shows the name of the input language or a flag. There is one thing, the pointer does not work in the text cursor in browsers, you can only accept this.
Unlike Aml Maple, there is no substitution of the cursor itself, just a small window-tail always goes after it. When changing the cursor, there is a problem of displaying it on a dark background, which is observed in Aml Maple, I had the same problem, and I had to give it up. In my implementation, there are also disadvantages, but it seems to me that they are not so significant.



Enable or disable the indicator is possible from the taskbar.



Action window


You can enable a special window. After selecting the text, click on the middle mouse button and the action window will appear, where you can choose one of the actions: translate, check spelling, copy, search in the browser.



Conclusion


Installation of the program is done with ClickOnce in one click, no advertisements and checkboxes with a trick, it is updated itself. There is also a portable version.
Questions, wishes can be left in the issues tracker .
