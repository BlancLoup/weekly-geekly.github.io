Alternative .sortable - a new solution for sorting the list on jQuery

There is a mass of solutions for sorting lists, for example, nestable, sortable, tableDTD, tableSorter, and others that have a wide functionality, but at the same time - a large code.
My solution uses jQuery, has a compact code (52 lines without comments, 1.8 kb) and provides for mixing elements (relatively positioned) within the parent container along the vertical axis.
The entire train of thought is commented in JavaScript.

See jsFiddle in action (uncommented code).
