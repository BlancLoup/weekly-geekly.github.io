Qwerty Access - a database of logins to sites

I decided to find out the opinion of the community about my little creation. The idea came to mind while searching for some files on the Internet, they all lie on file hosting, but links to these files are not so easy to get - often you need to register on the site to view them.
And I decided to make a small site where you can find a login / password or add your own if you still had to register for 1 time.

What we have:
reference: Qwerty Access
Adding
You can add your username / password to the site to change the protocol you need to click on it. Everything seems simple :)


Search
The search is very simple, just enter the address or the first characters of the address. It is also possible to change the protocol.


Voting
It is possible to vote for the login / password, he came or did not come. While the choice is saved only for the session.
