RAD Studio 10 Seattle Release

Friends, congratulations to all with the release of RAD Studio 10 Seattle !



A few hours ago, Embarcadero introduced the new version of RAD Studio 10 Seattle .
RAD Studio 10 Seattle is a complete tool for the rapid development of cross-platform applications using Object Pascal and C ++.

Just want to invite you to the webinar on new opportunities, which will be held on Thursday, September 3, at 12-00 MCK. Registration

Major Delphi 10 Seattle New Features


Incredible support for Windows 10


Delphi and C ++ Builder developers will be able to quickly update their VCL and FMX applications and take full advantage of the booming interest in Windows 10. Supports Windows 10 components and native APIs and WinRT / UWP components, Windows 10 VCL interface elements . Support for Windows 10 FMX has also been updated.



New C ++ 11 Compiler Based on CLANG Architecture


In C ++ Builder 10, the world's first C ++ compiler based on CLANG for Windows and mobile platforms with RAD PME extensions appeared, providing fast development for Windows and other platforms. It supports tight integration with VCL for Windows and FMX cross-platform structures, C ++ 11 and ARC (automatic reference counting) -based memory management for C ++, and is backward compatible. The new C ++ Builder compiler makes the version of RAD Studio 10 a mandatory update for C ++ developers and new developers migrating to C ++ from other languages ‚Äã‚Äãand toolkits - Java, Objective-C, C #, Xcode, Xamarin and Visual Studio.



Doubled project size in IDE


Among the imperceptible changes for the user, we can note the redesigned architecture of the product and assembly management system, which actually doubled the supported size of projects and increased stability and performance when working with large projects, especially for several platforms. Upgrading to this version is useful for developers who have faced resource constraints and other problems in large projects.



Stability, quality and effective documentation
