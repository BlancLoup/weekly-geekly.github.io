PHPCI updated to version 1.3



I hasten to inform you that today the PHPCI continuous integration system , which I already talked about in Habr√© , has been updated to version 1.3.

Basically, this update contains bug fixes and improved UI, but there are some interesting innovations:

  • Assembly statuses now support SVG format
  • Added new Atlassian HipChat integration plugin
  • Added new XMPP notification plugin
  • Many project subsystems refactored to facilitate testing and improving code quality.
  • Fixed many bugs with SSH keys, as well as bugs in various plugins (phpunit, composer, behat, phploc, etc.)

At that moment, when the previous article on PHPCI was written, the project version was 1.1, so finally I want to briefly list the innovations that came with version 1.2:

  • Added new plugin PHPDocblockChecker to check PHPDoc comments in projects
  • Added new plugin for working with SQLite DB
  • Added section to edit user profile with the ability to change the name and password, as well as password recovery
  • The first steps have been taken (for the future) to launch PHPCI in a Docker container with different versions of PHP
  • And again - many bugs fixed
