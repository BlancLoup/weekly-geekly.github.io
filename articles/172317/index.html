MakerBot company introduced a desktop 3D scanner



On Habr√©, references to the MakerBot company appear quite often. Mostly in connection with the 3D printers that this company produces. For example, one of the models, Replicator, is even named the most advanced desktop 3D printer. Now the company announced a new device, also a desktop one. Only this is no longer a printer, but a scanner, and it really has a very small size.

The scanner has received quite logical name - Digitizer. The full model name is MakerBot Digitizer Desktop 3D Scanner. This is a really interesting device that is able to quickly get a three-dimensional model of an object, so that this model can be reproduced on the same printer Replicator. So far, the scanner is a preliminary, test version, and there may still be additions and changes in its design. Nevertheless, the device is quite workable, more than.



According to the developers, the scanner receives a three-dimensional model of the scanned object by means of two laser beams and a webcam. It is clear that the object with which the new scanner is able to cope must be of a certain size. Of course, you can plan to scan the car body in order to print it later, but this task is not for Digitizer.



In general, 3D scanners are not so few now, but the Digitizer is one of the first desktop models that are really capable of performing the stated tasks.
