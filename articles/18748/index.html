Lightwindow - functional lightbox clone


The main difference from the original is that Lightwindow allows you to view not only the pictures, but almost anything:
  • Media : Movies, Flash (with "fit" to the window size), PDF ..
  • Web pages (including, by external link)
  • Forms
  • ... any web content


At the same time, the pictures can be combined into groups, galleries, indicating:
  • Gallery titles
  • Picture Names
  • The author


And most importantly: all this can be customized at its discretion. In particular:
  • Change background blackout
  • Customize animation
  • Custom CSS

To do this, a configuration file and the ability to individually customize each "gallery".

To close the Lightwindow window , just click outside the frame of the picture or press ESC.
