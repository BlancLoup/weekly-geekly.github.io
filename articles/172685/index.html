Twitter out for Windows 8



The official Twitter client for Windows 8 has been released.
In addition to its main functions, the client supports proprietary "chips" Windows 8, such as Charms , which allow you to write tweets from other applications, as well as the notorious live tiles, allowing you to broadcast tweets directly to the start screen.
