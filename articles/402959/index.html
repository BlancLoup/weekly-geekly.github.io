Windows 10 Creators Update can be downloaded today



According to the plan presented by Microsoft, Windows 10 will be updated on April 11th. We are talking about Windows 10 Creators Update version 1703. But if we talk about the official plan. As it turned out , it is possible to upgrade today, without the need to become a member of the Windows Insider Program.

To carry out the user on the road updates can program Windows 10 Update Assistant, located at this address . After downloading the assistant, you need to install and run it. If there is a need to perform a "clean" installation, then this will require the Media Creation Tool, the program is available via the same link. With it, you can create a boot disk or USB flash drive, and then install a clean system.

The current build of Windows 10 Creators Update is 15063.0, cumulative update 1703 will be available on April 11th. A pre-release version of the main update was distributed to the members of the Windows Insider Program. If updated now, using the Update Assistant or the Media Creation Tool, then the update version will be the same as for the Windows Insider Program participants, this is 15063.13.

The digital assistant who installs the update does everything quickly and without problems. Rather, there are problems, some dozens of users complain about them, they have tried to upgrade, but still there are many more successful installations of updates than failures. And the problems are also not too critical. For example, some users complain about the suddenly returned limit of 60 frames per second in CS: GO, someone has the settings of the OS itself. Some users complain about BSOD during installation or after it. As far as you can understand, the update occurs by reinstalling the system with transferring the old settings to the updated OS.

Before updating it is recommended to make backup copies of important files. The update may take several hours. In addition, it is necessary to prepare the drivers of their devices in case they ‚Äúfly off‚Äù during the update.

In its blog on Habrahabr, Microsoft described the main advantages of the update. In particular, the company has created Windows Security Center, which is a Windows 10 security event viewer. Windows Defender has new features to help detect and respond to network attacks.



Optimized system management, including mobile applications. A new feature protects data on personal devices without requiring registration when using mobile device management solutions.

Added auto-lock, it is activated if the user moves away from the PC. There is a built-in service for streaming Beam, also added the editor of 3D models Paint 3D. Gamers can be happy - the OS now has a special game mode, when launched, all other software will behave quietly, without distracting the user with notifications and messages. In this mode, the game will receive the highest priority with the appropriate access to PC resources.
