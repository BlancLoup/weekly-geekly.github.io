International Web Camp will come to Russia as part of Windows Azure Summit



Dear developers, I hasten to share the good news. A series of international Web Camp events devoted entirely to web development on the .NET and ASP.NET platforms, is coming to Russia. As part of the Windows Azure Summit, an extremely interesting and rich track of reports on the development of modern web applications based on ASP.NET, MVC, SignalR, HTML5, jQuery, Web API and Windows Azure technologies will take place .

Web Camp reports will be read by community experts from Microsoft: Nathan Totten and Nick Harris. You could see Nathan and Nick on MSDN blogs and Channel 9 portal entries, where they often keep video records.

Schedule and topics of reports:

  • Building and hosting websites based on ASP.NET MVC 4, Nathan Totten
  • Creating HTML5 Applications with jQuery, Nathan Totten
  • Real-time communication with SignalR, Nathan Totten
  • Building Mobile Web Applications, Nathan Totten
  • Using Windows Azure cloud services in your web application, Nick Harris
  • Build service tier applications using ASP.NET Web API, Nick Harris
  • Building applications using social services in ASP.NET, Nick Harris

Big Windows Azure Summit conference will be held February 20, 2013 in Moscow . The conference is free for everyone! Alas, we can not accommodate everyone, so registration is required.

Registration is now open! Hurry up to take a seat


Below you can find information about speakers.



Nathan Totten, Microsoft Strategic Technology Expert , Windows Azure

Nathan Totten is an expert on Microsoft strategic technologies that specializes in Windows Azure, web development, and Node.js. Nathan is also known as the creator and lead developer of the Facebook C # SDK. Prior to working at Microsoft, Nathan was a leading software developer in a startup where he worked on social media applications and analytical tools. Nathan Totten has a lot of experience creating high-powered cloud applications based on Windows Azure with high availability and performance. He is also actively involved in the development of opensource projects and community projects. Nathan Totten is a regular contributor to Stackoverflow.com and opensource forums where he often answers community questions.

Blog: ntotten.com
Twitter: @ntotten
Facebook: / totten



Nick Harris, Microsoft Strategic Technology Expert , Windows Azure

Nick Harris is a Microsoft strategic technology expert specializing in Windows Azure. Prior to working at Microsoft, Nick Harris was the founder of AdGAC, a mobile advertising company that developed Windows Azure, WP7 and ASP.NET MVC software. Nine years earlier, Nick Harris worked as a consultant and lead software developer, participating in building applications for thin clients, distributed enterprise applications, and air defense systems. In his spare time, Nick Harris blogs about Windows Azure and creates a variety of technical content.

About Windows Azure Summit


Windows Azure Summit will be the largest free conference, reports will be held in parallel in four rooms. You can view the program at this link . Venue and other information you can find on the official website of the conference .

Key topics for the conference:

Windows Azure as a development platform
latest platform updates, basic Windows Azure services

Scenarios for using Windows Azure
single backend for mobile solutions, transcoding and streaming video, a backend for gaming applications, creating extensions for SharePoint Online and SharePoint 2013, integration with corporate systems.

Business in the cloud
approaches to determining the cost of SaaS services and their pricing, methods and channels of promotion, management of the quality level agreement (SLA), compliance with the requirements of the law on personal data.

Cloud and Web
web development (HTML5, jQuery, SignalR, ASP.NET MVC 4) and hosting web projects in Windows Azure infrastructure

The conference is focused on:
  • IT executives who are already using cloud technologies for their tasks or just planning such a transition or integration;
  • Developers who want to learn Windows Azure and expand their knowledge in the technologies of creating applications, cloud services and WEB-projects;
  • Architects designing reliable cloud or hybrid solutions;
  • Directors and managers interested in innovative ways of business development

See you at the conference!
