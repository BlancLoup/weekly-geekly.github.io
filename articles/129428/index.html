Development details for GPUs in Visual Studio and C ++ AMP

Increasingly popular and relevant are technologies for developing applications that use the computing resources of graphics accelerators (GPUs) at full capacity. Today, almost every computer and laptop is equipped with a graphics adapter with huge computing power that can potentially be used by applications to significantly speed up computing.

One of the latest results of Microsoft's research in computing in a heterogeneous environment (a bunch of CPU + GPU) was the announcement of the C ++ AMP platform , which allows you to fully utilize all the capabilities of modern multi-core processors and GPUs. The announcement of the platform was made at the AMD Fusion'11 conference this summer; you can read more about this in this blog post . At the same event, built-in debugging and profiling of GPU code in the next generation of Visual Studio was announced. Detailed reports on the new platform were also read at the recent BUILD conference.
