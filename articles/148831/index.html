We work on the road on the phone


We spend so much time on the road. Traffic jams, long journeys take so much time, although this time spent could be spent on work. There are times when it is inconvenient to get a netbook or a tablet in public transport, and it‚Äôs not even a matter of decay from comfort. Our faithful friend, a cell phone, will come to the rescue.

In this post I want to present a selection of programs for phones operating under S60v5.

I advise for work


1) PAMP (Personal Apache MySQL PHP) - every web developer, even a beginner, knows what denwer or lamp is, and now imagine that it is on your typewriter that barely fits in your hand. And what is surprising, this program does not eat too much memory, and the possibilities are the same as on the PC.
The problem is that when creating php files you need to save them in utf-8.



2) Drops60 - sometimes you have to send or save large files when working on a project, and this is inconvenient from a phone, but thanks to Dropbox it is possible, and this client will help you more than anyone. You just need to install it and specify the path where the folder with the file will be on your phone.



3) SIC FTP Client- The FTP client program is suitable for all Nokia S60 3rd Edition smartphones. SIC FTP is very easy to use, but it contains a powerful file manager besides many other functional features of FTP. The client will provide an opportunity to quickly, easily and securely update and maintain your website via FTP or easily upload / download files. The user interface is intuitive to use and contains Nokia style. You can switch between your and the remote file system to get a quick overview of the files on both systems.
