25 useful shortcuts for Photoshop

Hello, dear habradrug! This article lists the 25 most useful shortcut keys in Photoshop that you can use to speed up your work. Some of them are rarely mentioned, some are used quite often, and some are generally only available in Photoshop CS5. I hope that each of you will find new and useful shortcuts in this article. So let's get started!



1. Move-approximation


Command / Ctrl + Space + Click / Move the cursor to the right or left. This is the fastest way to zoom, but this shortcut works only in CS5 and only with the GPU turned on.

2. HUD Pipette tool


Select the eyedropper tool and hold down Alt + Shift + Right Click (Win) or Control + Option + Cmd + Click (Mac). A window with a color palette will appear.



3. Review the entire document.


Hold H + Click. Works perfectly with high resolution pictures. This is a quick way to fit the whole picture to the window size.

4. Brush size and stiffness


Select the Brush tool and hold down Ctrl + Alt + Right click (Win) or Control + Option + Click (Mac). Move the cursor left or right to change the size of the brush, and up or down to change the stiffness. The CS5 works with both size and stiffness. In CS4, only with size.



5. Draw straight lines


Hold Shift. Works with almost any drawing tool.

6. Moving


Hold Ctrl. This is a quick way to move an object and then return to the previous tool.

7. Zoom all documents simultaneously


Shift + Ctrl + Space (Win) or Shift + Command + Space (Mac). With this shortcut, you can zoom in or out all open documents at the same time.

8. Hide panels


Press Tab and all panels will disappear. To open them again, press Tab again.

9. Display images from Mini-Bridge in full screen.


Space + Arrows. It works the same way as in the Bridge itself.

10. Text size


Highlight text and hold down Command / Ctrl + Shift + "." or ",". Shortcut will not work if you select only the layer with the text.

11. All text in capital letters.


Command / Ctrl + Shift + K. Quick way to change the case of text.

12. The distance between the letters.


Alt / Option + Arrow Right or Left. So you can easily change the spacing between letters.

13. Line spacing


Alt / Option + Up or Down Arrow. So you can easily change the distance between the selected lines.

14. Baseline offset


Alt / Option + Shift + Up or Down Arrow. Easy way to shift the baseline of the text.



15. Fill


Command / Ctrl + Backspace is the main color. Alt / Option + Backspace - background color. Shift + Backspace - fill options. A great way to fill a selection with color or change the color of text and shape.

16. Restore action history


Alt + Click on the history panel. Instead of undoing actions, just try to restore them.

17. Fill variation and opacity


To change the transparency, use numbers, and to fill - Shift + numbers. This will change the transparency of the selected layer and drawing tools.

18. Inversion of color of selected layer


Command / Ctrl + I. This is the fastest way to invert visible parts of a layer.

19. Changing the cursor of drawing tools


Click Caps Lock. So you can choose to see the edges of the brush or not.

20. Resetting Settings


Ctrl + Alt + Shift (Win) or Command + Option + Shift (Mac). When Photoshop loads, use this shortcut to apply the default settings.

21. Switching the main and background colors


Just press X. No comments.

22. Mask layer


Alt / Option + Click on the layer mask. A good way to tidy up the layer mask after you have saved it after the selection.



23. Turning off the layer mask


Shift + Click on the layer mask. No comments.

24. Pooling selections


Alt / Option + Shift + Click with the Selection tool.

25. Merge all visible layers into one


Ctrl + Alt + Shift + E (Win) or Command + Option + Shift + E. If you want to save all layers, but you also need to merge them all into one, you can convert them to a Smart object or use this shortcut to create a duplicate .
