Running Mac OS X Leopard on ASUS Eee PC - it is possible!

So there were the first mentions of the successful installation and launch of MacOS X Leopard on the ASUS portable Eee PC subnotebook. True, this was achieved not without the use of a tambourine, a set of files and reading mantras, but still the result was impressive. A detailed instruction is provided by the blogger of the site Uneasysilence.com [eng], and here is her translation made by http://eee-pc.ru (a review of the real copy will be released soon. It is also rumored that the site will sell this megadevise in Russia).


The main problem is that the Eee PC processor supports SSE2 instructions, but there is no support for SSE3, which is required for running the MacOS X Leopard version for x86. You can read about how to learn to circumvent this in the outstanding detailed guide on LifeHacker.com .

So, first you need to download Brazil Mac Hack from osx86scene.com. This patch will help create a version of Leopard for installation on a regular x86 PC. You will need a DMG image of the Leopard system. On a Mac, you can create it in Disk Utility, with the result that you should have osx-leopard105.dmg. Next, patch the image as described below, or download the patched image already ready for use from BitTorrent networks ( this is the advice of the author of the article, and not my ). There he appears under the name ‚ÄúMac OS X 10.5 Leopard [ToH] -RC2 SSE3 / SSE2 Intel Only‚Äù.

Patches
After downloading the patch archives, unzip the contents to your desktop (there should be two files and three folders).

Open the shell script 9a581-patch.sh in your editor and in the first lines replace XXX with your current username in the poppy.

Then edit the 9a581PostPatch.sh file, changing the fourth and fifth lines as shown below:
PATCH="/Volumes/LeopardPatch/leopatch/" # path to the patched extensions
LEO="/Volumes/Leopard" # path to Leopard installation

Now you can patch the distribution system. Open the terminal, enter sudo -s , then the administrator password. Go to the desktop - cd Desktop . Everything is ready to start, make sure that there is enough free space for the operation, the more - the better. We start the process with the command ./9a581-patch.sh .

You will also need to download the patched kernel using SSE2, from here . After that, unzip to the desktop, open the terminal and execute:
sudo hdiutil attach -readwrite $path_to_patch_dvd -owners on
where $ path_to_patch_dvd is the path to the current ISO image on your desktop. The terminal will require you to enter an administrator password, enter it.

This will mount the ISO image to the system in read / write mode. Drag the mach_kernel file onto the mounted disk from the downloaded archive and replace the existing file. Then remove the mounted disk image and burn it to DVD. This will be Leopard compatible with SSE2.

After creating the disc, connect the DVD drive to the USB port of your Eee PC. In the BIOS, specify to boot from a USB drive. While the Eee PC boots, you will see a black screen prompting you to start the installation or press F8 to configure. Press F8.

You will see the boot: prompt at the command prompt, enter -v -x , press Enter. After about 15-20 minutes, the MacOS X graphical installer will start.

Next, follow the tutorial from LiveHacker.com.
Format the disk. Go to the Utilities item in the menu and select Disk Utility (by mouse or keyboard, if the mouse has not yet been defined). In the sidebar, select your hard drive, go to the Partition tab, select the ‚Äú1 partition Volume Scheme‚Äù option, call the partition the name Leopard and specify the file system ‚ÄúMac OS Extended (Journaled)‚Äù. And the last action: click Options and select ‚ÄúMaster Boot Record‚Äù.

The easiest part of the manual is installing Leopard. Just follow the instructions of the graphical installer and select the section you just created for installation. But before you click the Install button, click Customize and uncheck Additional Fonts, Language Translations and X11. These components were removed during the patch process so that the distribution can fit on the DVD.

As a result, Leopard should be installed. The author notes that the system does not work very quickly on the Eee PC due to its not very outstanding power: a slow processor and only 512MB of RAM. Therefore, he recommends using a previous version of MacOS that should work fine on an Eee PC. In BitTorrent networks, an image suitable for installation is seen as ‚ÄúMac OS X 10.4.8 [JaS AMD-Intel-SSE2-SSE3 with PPF1 & PPF2] .iso‚Äù.

Those who are afraid of spoiling something permanently in the Eee PC can be reassured. ASUS fully supports the ability to install Windows, in which the default Linux will naturally be demolished. Therefore, they made a convenient bootable system recovery DVD, with which you can roll back the system to factory settings at any time. To Eee PC was as new, in one word.

In the comments to this guide, the author is asked how different devices work there. The author promised to write on this topic in the future, but for now he noted that WiFi and sound work without problems.

We see the Eee PC becoming a hobby computer. And the opening of the possibility of installing Mac OS Leopard on it is an example.
