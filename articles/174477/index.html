Google Translate for Android translates now offline



Google Translate for Android, as you know, allows you to translate text or words spoken aloud from one language to another. And a lot of languages ‚Äã‚Äãare supported. The only problem so far has been that for the application to work, it was necessary to connect to the Web. Now the application can work without the Internet, good, the developers have created offline dictionaries of different languages .

This means that once in a country where a free access point is not so common, you can translate the words of the language of this country into your own language without any problems. Of course, you can come up with a dozen or more ways to use the application without access to the Web.



Now the application is available for free download in the Google Play directory. In order to take advantage of the new feature, you must first download the language packs you need, which will then be used by the system.

Currently there are 50 language packs that can be freely downloaded. In general, the news is quite positive, for many representatives of the hardware community, Google Translate for Android will clearly become more useful.
