HTML table-based captcha



This check is based on ordinary HTML tables. The principle is as follows:
  • 1. Sets a list of colors for letters - table cells
  • 2. Sets the list of colors for the background - table cells


Parsing such a table is quite difficult, since you can change the colors of text and cells at any time, or change the number of characters displayed.
The data entered by the user is transmitted through session variables, and not as it is implemented in many other Captcha via the invisible field of the ‚ÄúINPUT‚Äù element.


Pre-create a table in the database


Each digit separated by a semicolon is the coordinate of the painted square (table cell) with the background of letters.

Installing html-Captcha
  • 1. Copy the html-Captcha folder from the archive to the plugins directory of your site (For example: wp-content / plugins)
  • 2. From the plugin installation page in the WordPress admin panel, activate the plugin
  • 3. Enter the settings page of the plugin and set the required parameters (by default they are already set, if you are not sure, do not touch them)
  • 4. Open the comments.php file from the archive and copy all the code written in it to the clipboard.
  • 5. Paste the code you just copied into the comments.php file of your template in the place where the SUBMIT button is written
  • 6. Paste the code at the very beginning of the wp-config.php file located in the root folder of the wordpress
