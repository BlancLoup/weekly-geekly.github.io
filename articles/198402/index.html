3CX is developing a video conferencing solution based on WebRTC

3CX acquired from the Italian company e-works the proven technology of web conferencing, which will be integrated into 3CX Phone System. Compatible with 3CX Phone System, the technology should provide the user with one-piece functionality, including desktop sharing and remote support , video conferencing, and many other options available in a few clicks.



An easy-to-use, innovative solution with open standards.


The need for an affordable, open-standards web conferencing solution that integrates into the telephone system is obvious: a study by Infonetics identified the need for PBX-based web conferencing technology. Companies need an efficient and cost-effective solution, without having to learn new features that would provide a way to improve the efficiency of multimedia communications without requiring expansion of the existing infrastructure.

3CX: a revolutionary video conferencing solution using WebRTC


The 3CX server technology for web conferencing will be fully compatible with the new WebRTC standard, allowing users to participate in conferences without installing any client software. An innovative feature allows you to overcome existing problems with patents and those solutions in the field of web conferencing that are tightly tied to client programs.

WebRTC, supported by Google and other developers, is rapidly gaining momentum and setting new, revolutionary standards in communications.
