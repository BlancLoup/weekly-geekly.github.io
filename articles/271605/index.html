EOIP & DD-WRT for connecting two apartments into one network or all for Remote Play

Background:

Actually, the task is to unite house 1 and house 2 into one network with walking broadcasts and multicastes.
The goal is to make PlayStation TV connect to the PlayStation 4 (without the main acc.) And fully use Remote Play locally.

In service, we have the following iron:

House number 1:
Dlink dir615C2
House 2:
Dlink dir615E4

Preparation, solution:

Dd-wrt firmware was installed on both routers. The installation procedure is simple, there is a lot of information on this topic on the Internet.

In order for EOIP to work, it is necessary for both sides to have a ‚Äúwhite‚Äù IP address, which is purchased from the provider for 100+ rubles. If this is not possible, then you can forward a vpn-tunnel (which I mentioned in the last publication), and use EoIP on top of it.

In order for users on both sides to see each other in a networked environment, it is necessary that they belong to the same local area network. I used the following settings:

Network: 192.168.100.0/24
Home 1: The router (192.168.100.10) has a pool of 192.168.100.1-192.168.100.50.
House 2: the router (192.168.100.70) has a pool of 192.168.100.51-192.168.100.100

After we installed dd-wrt on our hardware and configured it on every DHCP, hooked up clients, and were convinced of their work, we proceed to create an EOIP tunnel.

To do this, go to the web. router interface, go to the Setup-> EoIP Tunnel tab
We set EoIP Tunnel to the Enable position;
In the Remote IP Address column we enter the external IP router (in the case of VPN, then we specify the IP in the vpn-tunnel) to which we need to connect;
Bridging to Enable ;
* Required, tunnel numbers must match on both routers.



You can check the result in the Web interface of the router by going to the tab Administration -> Commands :
In the Commands field, write ifconfig and click Run Commands . After that, we will receive raised interfaces, among which will be our oet1

Now we have an EOIP tunnel that allows us not only to see shared folders on the network, but also to send video from the phone to the TV in another apartment.
