Creation and configuration of Analyzed Result based on Analysis Template in HPE Performance Center

  • Create an Analysis Report and set it up
  • Create Analysis Template based on Analysis Reports
  • Download the Analysis Template file in Performance Center
  • Add Analysis Template to Performance Test
  • Get Analyzed Results Based on Analysis Template
