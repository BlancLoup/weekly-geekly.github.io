Firefox 3.1 Alpha 2 released

Mozilla has released the second alpha version of the popular browser Firefox 3, just a few days after the release of the new Chrome browser from Google. The code name of the version - Shiretoko, is intended mainly for developers and testers.

In the new version, a lot of snacks, including support for the video tag from the HTML 5 standard, dragging tabs between browser windows, improved integration with the Aero Glass theme for Windows Vista, and JavaScript has been accelerated.

Mozilla plans to integrate the new JS engine, named TraceMonkey, in future versions of Firefox. Although now it can be tested by following the instructions on the official website.

Download Firefox 3.1a2 from the official site.
