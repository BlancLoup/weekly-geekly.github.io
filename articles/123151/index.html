Calculate Linux 11.6

Today, the anniversary (4 years project) release of the [lorMode] binary genta [/ lorMode] of the remarkable Russian Calculate Linux distribution has been released. It would be worthwhile to issue as a topic link, but unfortunately I can not.

Details on the cut.

So, the main points are:

-Added support for selecting a network manager between OpenRC and NetworkManager.
-In the console installer added support for network settings.
-Udev integrated into initramfs to better support devices and speed up livecd loading.
-By default boot loader is Grub 2.
-Added recognition of operating systems installed on the disk.
-Added auto definition of screen resolution in splash.
-Added video driver selection menu. Additionally, nouveau and ati-drivers are supported.
-By default, free video drivers with KMS support are used.
-Nvidia is no longer a pre-installed driver and is installed when the user selects.
-Use XZ compression format livecd image.
-The transition to hybrid images with support for recording on CD / DVD and USB Flash.
-For the Calculate Scratch Server created a binary package repository.
-Added integration with the web service pastebin.calculate-linux.ru .
-In Calculate Directory Server added support for storing the history of mail correspondence.
-In the Calculate Directory Server package included Shorewall to configure the firewall.
-Added support for Portuguese and Romanian languages.
