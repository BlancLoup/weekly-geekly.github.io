New Fusion ioDrive Duo


Fusion-io announced the release of the fastest SSD hard drive - ioDrive Duo.
It has twice the capacity compared to the previous generation product and has the following characteristics:
  • PCI Express x8 or PCI Express 2.0 x4 bus
  • Reading (MB / s): 1500
  • Record (MB / s): 1400
  • Reading (IOPS): 186,000
  • Record (IOPS): 167,000
  • Delay: 50 ¬µs

The drive is available in versions 160, 320, 640 GB, will be available in April, only the price does not provide a prudent indication. I think it will not be very low.

UPD : In the second half of 2009 they promise to release a model with 1.2 TB of memory.
The device can rebuild its storage (ioMemory) from RAID0 to RAID1. Naturally, the memory size will be halved, but in the event of the collapse of some memory cells, the data can be restored with almost no problems.
Based on today's prices for the first generation ioDrive (about $ 30 per gigabyte), the price of a model with 640 GB of memory may be close to $ 20,000. Not so much, considering that this is the fastest drive available today, showing the speed of work several orders of magnitude higher than mass, and so not cheap, SSD.
