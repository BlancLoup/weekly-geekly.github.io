Lenovo ThinkServer TS140 - server available to everyone

Good day, dear habrazhiteli.
As we already know, Lenovo at the end of last year presented ThinkServer server solutions for small and corporate businesses. Among the solutions there are servers in the Tower and in the Rack package, as well as disk shelves that expand disk space with direct connection via direct-attached storage (DAS), which are already available for order in Russia. But today we will not talk about the entire model range of server solutions, but only about an entry-level server that is available to absolutely everyone - Lenovo ThinkServer TS140 .





Floor-standing servers are in demand in the small and medium-sized business sector, for those organizations where the IT infrastructure is, in principle, absent, installing a rack for equipment is impractical, and the server is bought mainly for accounting, and most often stands there. Therefore, such machines should work quietly, provide a large disk space for the general resources of the company, be easy to maintain and be reliable, and most importantly affordable.

Presentation and first emotions:


The entry-level server Lenovo ThinkServer TS140 is a productive solution for organizations that are not particularly demanding of deployed infrastructure. The legendary quality of the Think family and simple software solutions combined in an economical server solution for branch offices and small businesses.

Hardware component of the server line of ThinkServer TS140:


Lenovo Thinkserver TS140 is presented in several versions based on different processors of the Intel Xeon E3-1200 v3 family, Core i3 4000 and Pentium G3000 series, all CPUs on the LGA1150 socket. Intel C226 motherboard chipset. RAM UDIMM, DDR3 1600Mhz c ECC in dual channel mode. The maximum amount of RAM supported by the 32Gb server (4 slots). Up to 4 3.5-inch SATA or SSD hard drives can be installed, without hot-swappable drives. There is also RAID level 0, 1, 5, 10, thanks to the integrated ThinkServer RAID 100 controller with Intel RSTe 3.8 support. Integrated TPM module (security chip ST33ZP24AR28PVSP) The server has an integrated Intel HD4600 video card with three 1-VGA and 2-DysplayPort outputs, and it is also possible to install discrete video cards in a separate PCIe x16 slot. Expansion slots 1 x PCIe GEN3, 1 x PCIe GEN2, 1 x PCIe GEN2, and 1 full-length 32-bit PCI designed for any old expansion cards. Integrated Intel i217 series gigabit network interface, to increase the number of network interfaces, additional network cards can be installed.



The server is powered by one 280 watt or 450 watt power supply. The server is designed in a 4U Tower enclosure with rack-mount capability (optional). Special attention should be paid to the noise emission of the 26dbi server, which makes the server noiseless enough even to work in the library. Built-in IPMI 2.0 monitoring and management technologies (on low-end Core i3 and Pentium G models) and AMT 9.0 (only on servers with Intel Xeon CPUs) with a remote iKVM virtual machine allow you to remotely control the server. And the most important thing that I want to notice is the warranty on the Lenovo ThinkServer TS140 servers - 1 year and 3 years for some configurations.

Options kits and their features:


Lenovo TS140 server is quite client-oriented. There are ready-made equipment with built-in disk system and pre-installed Windows Server software, which greatly simplifies commissioning. Each client will be able to choose and equip their Lenovo TS140 according to their tasks and needs.



Operating recommendations:


The server is oriented as we have said above for medium and small businesses. Of the classic tasks worth noting the most common:
  • File server
  • Server 1C (both file and SQL)
  • Corporate Email Server
  • Backup server
  • CRM


A small bonus for Habr Users!


The Lenovo TS140 server works with Vmware ESXi version 5 hypervisors, provided that the build of the boot * .iso image is done manually with step-by-step loading of the required drivers. And loading the hypervisor from USB media itself. Now the most advanced equipment is being tested for stability in working with the hypervisor and virtual machines for small businesses in an environment of 10 users: 1C server, corporate mail, file server .
