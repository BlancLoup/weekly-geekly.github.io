October Ciklum .NET Saturday in Kiev

As part of the October .NET Subbotnikov series, another stellar event will take place in Kiev.
On October 20, Ciklum SkyPoint will open for the participants of the event both an unforgettable view of the city and excellent reports that will appeal to all .NET specialists.

In a programme:
1. Overview of ASP.NET Web API Framework by Alexander Beletsky
ASP.NET version 4.5 has acquired a large number of new features, including improved Web Forms, MVC support, but one of the main features the author calls the new ASP.NET Web API Framework, which allows you to create RESTful web services in .NET. The report will examine in detail the evolution of web services, the REST architecture and the main features of the Web API.

2. Continuous Deployments in Windows Azure Web Sites by Anton Vidischev
The report will discuss in detail the Azure Web Sites elements and possibilities of deployment and Cloud TFS and GIT settings.
.
3. Introduction to TypeScript by Dmitry Mindra
Everyone has already heard about the new language from Mictosoft, which will allow developing scalable services on JS. JS developers already test code with jslint, use closure compiler, use module pattern. Your attention is invited to consider TypeScript. It combines type checking and static analysis, explicit interfaces, and best functions within a single language and compiler. Consider this language in detail.

4.5 Resharper features I can't live without by Dmitro Suvorov
R # is one of the least popular tools in the .NET world. Developers who use it will talk for a long time about an endless list of useful features, but most consider it a slow, constantly falling and extremely useless tool. But having met him again with a competent introduction from the experts, you see him as completely different: quite stable and possessing a large number of useful features.
Dmitry‚Äôs report will be about why he can no longer live without R #.
5. Functional programming in C # by Sergey Kalynets
C # is considered an object-oriented programming language, but it supports many paradigms, one of which is functional programming.
In his report, Sergey will show with examples how the use of functional programming patterns can create more comfortable conditions for the developer and help the software code become cleaner and more efficient. The basic constructs of C #, LINQ from basic to advanced, parallel and asynchronous code will be considered.
