A week of free online Windows Azure courses, registration is open

Great news for developers and IT professionals interested in cloud technologies and Windows Azure. Microsoft experts, including Scott Guthrie and Scott Hanselman, provide you with the opportunity to explore cloud technologies through daily online courses throughout the week starting January 27, 2014 .



The following courses will be presented during the week:

  • January 27, Get Started with Windows Azure Today Jump Start, registration
  • January 28, Designing Applications for Windows Azure Jump Start, registration
  • January 29, Building Windows Azure Applications Jump Start, registration
  • January 30, Windows Azure IaaS Deep Dive Jump Start, registration
  • January 31, Mobile Apps to IoT: Connected Devices with Windows Azure, registration

Register for courses today and join the stream to learn about the Windows Azure cloud first hand!
