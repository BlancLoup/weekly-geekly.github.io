Interface and usage.

The first thing we see after installing the program is a nice help window that tells us about the program in brief.



As you add reminders, a list of them is created, with a runtime and an action that should occur when you hit the spot.



It is worth noting an interesting menu that includes sorting the list of actions, disabling all tasks, deleting all tasks, help and information about the developer.



On the bottom panel there is a view switch (list / map). The map shows all the points, as well as the current location of the phone where the application is installed.



To add a new point, you need to click the add button, which is located in the lower right corner.

Adding points is implemented very original - just make a long press on the screen to light vibration and just stretch the point, thereby changing the radius of its action.



Next we press the button further and get to the panel of other settings, where you can just specify the name of the point, the action and the schedule for its implementation. You can switch between the settings panel and the map view using the buttons on the left.



The interface is intuitive, so there should be no problems with the use of the application, besides, there are training tips for the user in the application.
