Obtained root rights on HTC Desire (Upd)

HTC has released a stunning device on Android, which everyone has been waiting for. But what could upset a real technogic was the lack of superuser rights in the firmware. However, Paul O'Brien, a well-known in narrow circles, the author of the Modaco project, was able to write an exploit and get root on the latest version of the official firmware 1.15. 405.4 , as evidenced by the photo on the left.

According to Paul, the same method works on HTC Legend, and will probably work on HTC Droid Incredible. O'Brien was able to get write access to the system partition, bootloader, and also radio. Recovery mode is still protected.

Upd: Detailed instructions with all the necessary files are here .

What can it give us? For example:
  • The ability to install custom firmware, fixes that appear more often than official ones.
  • Ability to install the application on the SD card using A2SD.
  • A variety of useful utilities, like AdFree, work only on "rutted" firmware.
