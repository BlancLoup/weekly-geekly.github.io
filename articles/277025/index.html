From complex to simple: we start a virtual server Windows / Linux

Today we are talking about the ‚Äú Windows / Linux virtual server ‚Äù service, recently available on CloudLITE service. We announced its launch in November , when we talked about the transition to a new billing and provisioning platform based on BILLManager, and now we are ready to share details.



What have we got?


As practice has shown, for most CloudLITE users, simplicity of service is more important than customization and functional delights: ready-made VM templates with a pre-installed OS are an order of magnitude more popular than ‚Äúindividual‚Äù assemblies, and those who want to set up a network on their own or go into the administration of cloud infrastructure with fire in the afternoon. We decided to make the face simpler and in addition to the virtual data center (IaaS) we launched the ‚ÄúVirtual Server (VDS)‚Äù service.

In order to start working with the server, you just need to select a tariff, OS template and disk size.



Immediately after activation, the server is ready to work without any extra settings. You will receive a letter with emails and server passwords and brief instructions for remote connection via RDP / SSH or via the web console in your account.



For the time being we stopped at the ready-made templates of the most popular OS among our clients:
  • Ubuntu 14.04,
  • Centos 6.7,
  • Russian and English versions of Windows Server 2008/2012.

In the future, of course, we will expand this list at the request of radio listeners.

There are five tariffs for each OS group ‚Äî not much compared to a virtual data center, but within the tariff you can flexibly control the size of the disk: the higher the tariff, the longer the interval from which you can choose.



Each server is allocated an external IP-address, which is now attached directly to the server, and not through NAT, as in a virtual data center.
From bonuses: a server license is already included in the VDS Windows package, and the ISPmanager 5 LITE panel is attached to the Linux virtual server.
Like all CloudLITE services, the Linux / Windows virtual server is based on VMware. The equipment is located in its own data center NORD 3 in the north of Moscow.

What else is interesting?


Hourly billing. In addition to daily payment, which is already available for the services of a virtual data center, hourly billing will work for VDS. What does this mean in practice? If a virtual server is not needed all the time during the day, then you can turn it off and thereby save money. In the inactive state, only the cost of the ordered RAM, disk space (they continue to store your data, even when the server is turned off :)) and licenses (in the case of Windows North) will be taken into account. But you won't have to pay for the processor. During the day, billing records the number of hours of ‚Äúwakefulness‚Äù of the server and once a day writes off money, taking into account these statistics. Information on charges can be found in the ‚ÄúExpenses‚Äù section in the Personal Account.



More detailed data on the number of hours during which the server worked, the load (%) on the processor and disk can be monitored in the Monitoring tab (this section is still in beta testing). In the future, there will be a function of SMS- and email-alerts about server unavailability and / or critical processor load.

Migration to a virtual data center . Compared with the virtual data center (IaaS) , VDS does not require additional settings and allows you to start working right away, but the capabilities in this format are slightly less. In case you feel that you run into the ceiling by functionality and want more control over resources, you can always upgrade and move your server to a virtual data center (ACC). All that is needed is to create VDC of a suitable size, choose a virtual server for transfer, and CloudLITE technical support will do the rest.



By tradition, we invite everyone to a test drive of a new service. Now you can try a virtual server with Ubuntu / Centos in configuration 1 vCPU, 1 GB RAM, 30-80 GB SSD . The kit includes the ISPmanager 5 LITE panel and DNS hosting , which we also launched at the beginning of this year, as promised :).
And as always, we will be grateful for any constructive feedback.
