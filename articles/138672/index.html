Canonical introduced Ubuntu for Android (UPD)



Canonical announced its new development Ubuntu for Android, which Mark Shuttleworth is going to present next week at Mobile World Congress and allowing to host a full desktop version of Ubuntu on a productive smartphone besides Android, which can be started by connecting the smartphone to a computer through a special docking station.

Get the phone, connect it to the monitor and get the desktop version of Ubuntu with all the usual applications running on the same device as Android ... Your next desktop can be your phone
says Shuttleworth.

Ubuntu for Android is designed to work on high-performance smartphones - this should be an ARM device with a dual-core processor (HDMI is also required). The system has a serious set of pre-installed applications ‚Äî the Chromium browser, the Thunderbird email client, the Gwibber social networking client and even the VLC video player are tightly integrated on the second system aboard the smartphone: you can get access to the Android address book, wireless capabilities such as Wi- Fi, 3G, 4G.

It is worth recalling that about a year ago, Motorola introduced a similar solution WebTop for devices based on Atrix, but so far it has not received distribution.
