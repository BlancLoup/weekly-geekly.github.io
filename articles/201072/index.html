BitTorrent Sync: speed up to 90 megabytes / s and open API

Yesterday a new version of the program for file synchronization via BitTorrent Sync 1.2 torrents was released. The developers reported some good news. Firstly, the popularity of BitTorrent Sync is growing rapidly, which guarantees the successful development of the project in the future. The number of users has exceeded 1 million people (monthly active audience), and the amount of synchronized files is 30 petabytes.

Secondly, the synchronization protocol has been optimized in the new version, so that the maximum speed in the local network has been increased to 90 MB / s. This means that a 10-gigabyte file is automatically synchronized between computers in LAN in less than two minutes.

And most importantly - finally released open API for adding synchronization through torrents to any third-party software.

To use the Sync Developer API, you need to register on the site and get the key. After that, you can add BitTorrent Sync program interfaces to your program, here is the API documentation .

Thanks to software interfaces, it will be more convenient to use BitTorrent Sync in non-standard tasks: for example, for a social network without a central server or P2P hosting sites .

BitTorrent Sync 1.2 is released in versions for all popular operating systems, it also supports the new iOS7, including through the native application for iPads. New features have been added to the iOS application: you can send and synchronize files via BitTorrent Sync and other programs, and you can also save the received files from Sync directly to all photos and videos.
