Microsoft launches Popfly - a mashup application

Microsoft has released the closed beta version of Popfly . This is a new Silverlight-based application that allows users to create mashups, widgets and other applications using a simple graphical web interface. Earlier we talked about the launch of Yahoo Pipes , a service that made it easy to create mashups and mini-applications using a drag-and-drop interface. After a few more similar projects. Microsoft has recently come to this market, and at the moment, if you believe the impressions of the author of the topic on TechCrunch, it is ahead of all its competitors.

Popfly has overtaken its competitors because it offers more options. With Popfly, you can create mashups, applications, web pages, widgets, and then combine them all into a social network (as part of the Live Spaces platform) where you can unite with other users and application creators.
