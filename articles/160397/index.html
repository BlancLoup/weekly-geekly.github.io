As I wrote above, I have a profitable project that works in almost complete autonomy, which gives me a lot of time to develop SmartProgress. Also, due to the current project, there is funding for SmartProgress - advertising, hiring freelancers (they hired a translator to create an English version). The project does not have a special financial need, so we are not actively engaged in the active search for investors as long as we manage on our own. Of course, several additional programmers would significantly speed up the development process, so we do not exclude the possibility of investing.

Technical details


Long thought on what to do the project. As a result, I decided to take the Yii framework as a basis. The main criterion for selection was the speed of work and Yii showed good results. Also behind Yii was: good documentation, active community and dynamic development of the framework.

Here are some extensions used:
Mailing - YiiMailMessage
Photo Upload - Xupload
Log in / register through social networks: HybridAuth

The site is located on a dedicated server XEON X3440 4x2.53 GHZ / 24GB RAM / 2x2000GB HDD
Nginx is used as a web server in conjunction with php-fpm
Database - MySQL
Opcode Caster - APC
There is still enough server capacity, so the issue of optimizing for high loads is not yet worth it.
