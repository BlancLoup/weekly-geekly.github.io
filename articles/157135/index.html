New modLivestreet 0.5.0-rc full release


After twenty hours of work, I publish a fresh modLivestreet package, which I hope will be a turning point in perception and will already be able to clearly demonstrate why this was all started.

What is new this time? This time the emphasis was placed on creating everything needed to complete the installation of Livestreet with just a few clicks and almost complete control of the engine from the MODX admin panel, namely:
  • Sources of Livestreet files (full section, modLivestreet plugin, uploads folder). Now you don‚Äôt have to search for where everything is long, and there you can also manage Livestreet files through the admin panel
  • Templates for MODX to display the main Livestreet, any Livestreet documents, Any MODX documents in two variations (two columns with news feeds in the right block, and one column without sidebar).
  • A couple of chunks to get into the Livestreet templating

In general, in many ways the module is already ready to use it.

Under the cut a more complete list of changes and demo video.


The latest version of the package can be downloaded here .



List of changes.

1. A hook has been added to the Livetreet plugin that adds the modLivestreet.html_head_begin chunk. 
function html_head_begin(){ return '[[$modLivestreet.html_head_begin]]'; }

The /> tag is registered in this chunk, but you can instead write a snippet of metaX or something like that.
Although you shouldn‚Äôt rush with metaX for now, because all Livestreeet pages are now capitalized for it.
For it to work correctly, MODX documents will be generated for all LS sections. Not an option. It is better to write an updated package.
metaX is the first contender for migration, as I like it very much :-)

2. Added 4 templates:
1. modLivestreet.MainPage - to display the Livestreet home page
(critical if the livestreet section is not in the root of the site)
2. modLivestreet.Default - to display all Livestreet pages
3. modLivestreet.Modx - to display the content of real MODX-pages.
Also on the page is a Livestreet feed.
4. modLivestreet.ModxNoSidebar - the same as template 2, but without the news feed.

3. Added file sources:
1. Livestreet - Fully Livestreet Website Directory
2. LivestreetUploads - Livestreet static files download directory
3. modLivestreetPlugin - plugin directory modLivestreet for Livestreet
Now these sources just add more controllability,
