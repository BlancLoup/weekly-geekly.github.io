DevDocs: all developer documentation in one place, with a fast and user-friendly interface



The site devdocs.io is a project of the French programmer Thibault Curubl. Here is collected and streamlined documentation on the most popular web technologies, frameworks and APIs, and many other development tools. DOM, HTML, JavaScript, jQuery, Node.js, PHP, Ruby, Python, Git, Angular, Backbone, CoffeScript, Less, Sass, Redis and a lot more ... Everything is designed in the same style, including fuzzy. It is possible to select only the necessary technologies for which you need to search. In general, the DevDocs interface is pleasing - nothing superfluous, everything is very clear and functional, a lot of keyboard shortcuts are available .

DevDocs is a completely open source project, so you can download it from the repository on Github and install it locally (it is written in Ruby). Although the project does not have a year yet (it was launched on June 18, 2013), it managed to acquire many additional amenities: it is available as Chrome and Firefox extensions and as plug-ins for Sublime Text and Brackets. It supports OpenSearch and integrates into the browser search bar.

You can vote for adding new sections of documentation in DevDocs on the project board in Trello.
