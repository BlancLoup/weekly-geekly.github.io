The second wave of fateful changes GitHub



At the Universe-2016 conference in San Francisco, CEO and GitHub founder Chris Vanstrat announced the largest update of the resource. In his opinion, these changes will change the life of community members for the better.

He noted that he was proud of the team that was able to realize such a number of improvements, but made it clear that everything was just beginning.

Project management


Hoping to make it easier for developers to go from idea to implementation, GitHub has launched its project management tool called Projects .


The tool allows you to build a development process so that developers do not have to once again leave GitHub or use additional applications. Project allows you to create cards with pull requests, requests for revision, error messages or notes, as well as to mark the stages of the implementation of tasks (‚ÄúIn progress‚Äù, ‚ÄúCompleted‚Äù and so on).

Long-awaited update Code Reviews


The GitHub team has penetrated even more deeply into the ideals of productive interaction and timely feedback. They believe that updating Code Reviews will help make software development more efficient. Now, in addition to the ability to comment on individual lines of code, the tool allows you to accept changes to the code, or send a request for revision.



Global integration


GitHub developers have made significant changes to the integration API and added new ways to interact with integration experts not only within the resource, but also beyond.

Among the new tools - Platform Roadmap . It is available for viewing to all members of the community and shows what kind of integration will be implemented in the near future and why.

Another innovation was the Platform Forum , which allows community members to directly contact GitHub engineers.

Two-factor authentication


Now administrators can adhere to a security policy for absolutely all accounts of the Organization.



GitHub Community Forum


The task of the new forum is to unite all the participants of GitHub, where they can communicate, share experience, learn and help each other. The launch of the forum is planned for 2017.
