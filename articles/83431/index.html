Tablet Haleron iLet Mini 7



Haleron has expanded its product range with the iLet Mini 7 model - a tablet netbook on a VIA processor. The device works under Windows CE 6.0 and costs only $ 150.

Specifications:

Processor: VIA VT8505 (ARM926EJ-S) 300 MHZ
OS: Microsoft Windows CE 6.0
Display: 7-inch, 800x480 pixels
Memory: 128 MB DDR II
Winchester: 2 GB of internal memory + support for SD / MMC cards up to 16 GB + support for external hard drives up to 250 GB
Connectivity: Ethernet: 10 / 100M, Wi-Fi, Bluetooth (optional)
Optional: 2 USB ports, GPS (optional), support for external keyboard
Dimensions: 188x114x24 mm
Battery: 2200mAh lithium (4-6 hours of battery life), optional - 3300mAh (8-10 hours of battery life)
