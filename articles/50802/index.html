And again Fugue!


I may have already buzzed all your ears about this set, but could not not mention it again. Yes, yes, fugia refreshed again. This version 1.4.3, though not the last, is still relevant for everyone. Finally, it has all the icons necessary for the design of your web application. I understand that the network has a lot of useful sets of icons, but for some reason this has affected me the most.

Let me remind you again - these icons are designed for the design of web applications, well, or ordinary programs, but not for the design of your desktops or folders. All icons are made in the mixed style of Mac OS X and Windows Vista, size 16x16px in PNG format. The author also shared DevKit with us in PSD format, thanks to which we can change the icons as we like.

The combination of icons is the possibility of this set I like the most. The author took the time and created miniature versions of almost all the icons thanks to which you can create wonderful combinations. It is good that the author did not dare to combine all the options, otherwise the set would be much more. But there are combinations in the set, that's why he is still so impressive.
