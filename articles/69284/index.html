Mouse Extender: work on bugs

Good evening!
Two days of work on the Mouse Extender were not in vain.


Added a bit of new and hopefully useful functionality:
  • Add program shortcuts to ME using the list of running programs.
  • Add exception programs to ME using the list of running programs.
  • Customize shortcuts in ME: you can now edit tooltips
  • You can disable the "Close" button.
  • Mouse Extender can be turned off globally
  • The ME window can be displayed using the tray menu. Also, the ME window is shown at the first launch.
  • Now, using the button-hairpins, you can make ME not hide
  • Added a shadow to the main window
  • Stylized tray menu
  • Now with the middle button you can not only show ME, but also hide it (it is included in the settings).

We also tried to fix as many bugs as possible.
Of those bugs that could kill:
  • Start brakes
  • Running multiple copies of the program
  • A bug with the visibility of the settings window
  • ME positioning bug near screen borders
  • Overdrive bug and click middle button


Actually, you can download the new release in the same place: http://bit.ly/meurl
Thanks for attention!

UPD: Attention! Shutdown and reboot occur without warning!
