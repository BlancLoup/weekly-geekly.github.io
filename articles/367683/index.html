IBM Watson helps you organize your daily routine.



IBM and Under Armor have teamed up to create an advanced application to help you create the right daily routine. Simply put, now the application based on the cognitive abilities of IBM Watson will tell you when to eat, sleep and play sports. The application itself is called ‚ÄúUA Record‚Äù. The IBM Watson cognitive skills program was introduced at CES 2015.

The idea is for Watson to analyze daily user activity, sleep and nutrition data. The person enters this data into the application, and the cognitive system evaluates and analyzes the information. It is also interesting that the application works not only with the data of its user, but also with information received from other users of the system.

Example: the user enters all his data, including weight and duration of sleep (5 hours). The system, after analyzing this data, tells the user that UA Record users of his age group are about 5.7 million people. Using the data of people in this group, the system showed that the lowest body mass index is observed in those people who sleep between seven and eight hours every night.

Many users will find such assistance from the program very useful. Indeed, it is one thing when doctors, nutritionists and other specialists talk about healthy lifestyles from TV screens, pages of newspapers and magazines (including online publications). And it is quite another when an impartial program makes its judgment based on the data of millions of real users.



In the future, IBM and Under Armor plan to add weather data for the Weather Channel service, recently acquired by IBM. What for? So that the user can plan trainings, walks and business meetings based on the weather forecast data.

For example, if after a couple of days the system predicts the sun, you can run on the street. And, on the contrary, if according to the forecast it rains and bad weather, it is better to go to the gym.

Next year, Under Armor will add new features to its cognitive training system, the foundation for which is IBM Watson. Among others will be added such:

Individual selection of the training program: the application, using the data of Watson's analysis for a specific user and his preferences, will help to create a customized training program.

Power system: using technologies such as Watson Visual Recognition and Watson Discovery, the service will be able to closely monitor the power of an individual user.

Weather and environment: this has already been mentioned above. Using the data of the weather forecast and other information, the program will be able to modify the user's lesson plan according to specific conditions.

By the way, IBM services can now predict not only bad weather, but also possible catastrophic events, including a volcanic eruption or an earthquake. This system is called IBM Sequoia, information about it is available in the company's blog on Geektimes.

As for the UA Record application, perhaps not all users will like the fact that the system uses personal data. But all this information is anonymized and not tied to the username.

Anyway, this application looks like a project from the future. However, this is already a reality that millions of users will soon be able to appreciate.
