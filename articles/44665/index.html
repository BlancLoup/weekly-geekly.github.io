Yandex.Maps open API

Yandex company officially announced the opening of the Yandex.Maps API service.

Now everyone can use Yandex.Maps features for their sites.

This news is particularly important in particular for Ukraine. In Russia, Google has long since made a normal map service, and as you probably know, Google Maps has open API for many years. Therefore, Russian developers for several years can fully use the cartographic capabilities when creating new services.

Now, Ukrainian developers have had the opportunity to create a mass of interesting new services, which they could only dream of earlier: from showing the map to the office to full-scale geo-information services. Social networks with finding neighbors nearby, tourist and transport routes, integrated reference databases, the location of hotels and many other interesting services will now appear in Ukraine

To date, Yandex has already created maps for 28 places in Ukraine.

Yandex also opened a separate Yandex.Maps API development club where you can ask a question and share your thoughts.
