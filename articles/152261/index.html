Adobe Edge Tools & Services



Adobe launched Adobe Edge Tools & Services , a set of services and tools designed to help developers create websites and applications.

What's inside

- Edge Animate (formerly Edge Preview). Allows you to create interactive and animated content using HTML, CSS, JavaScript.
- Edge Inspect (formerly Shadow). Preview mobile web sites.
- PhoneGap Build , packer of mobile applications.
- Edge WebFonts.
- Edge Code . Innovative code editor.
- Edge Reflow . Creating layouts using CSS.
- Typekit (commercial fonts).

Terms of Use

If you have a Creative Cloud account (of any level) - the use of most services is free, Edge Animate is free only at its first trial launch, and so - $ 15 per month or $ 499 forever.
