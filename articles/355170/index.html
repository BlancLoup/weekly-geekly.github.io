The Great Suspender - sleep mode for background tabs + unload RAM

The great suspender

It's time to somehow unload the RAM from an infinite number of browser tabs.

Briefly:


  1. Install the extension from the Chrome online store . Then the The Great Suspender icon will appear in the upper right corner:

    If you have Mozilla Firefox, you will have to use Chrome Store Foxified .
    If Opera, then Install Chrome Extensions .
  2. Open the tab. Click on the extension icon in the upper right corner and select: Suspend this tab . The tab went into sleep mode and no longer loads the RAM.

    Default: if nothing happens on the tab for an hour, it will automatically go into sleep mode.
  3. In order to ‚Äúwake up‚Äù the tab, just refresh the page.

Extension setting:


  1. General settings - Basic settings:

    • Automatically suspend tabs after - Automatically put tabs into sleep mode via .
      The default is 1 hour , but I would advise you to set 10-15 minutes .
    • Do not suspend pinned tabs - Do not put locked tabs into sleep mode .
      My advice to you is to leave a tick here.
    • Do not suspend tabs that contain unsaved form inputs - Do not put into sleep the tabs containing the data to be filled .
      Vital function. For example, you fill in the data when you buy something on the site, went away for a couple of minutes, and the tab went into sleep mode. Because of this, you will have to re-enter your details later. Therefore, leave a tick and go on.
    • Do not suspend tabs that are playing audio - Do not put tabs into sleep mode on which audio files are played .
      Definitely leave a tick.
    • Only auto-suspend if connected to the internet - Put tabs into sleep mode only if you have an internet connection .
      It will be useful if the tab has already loaded and after going into sleep mode, it will not be possible to load it again.
    • Only auto-suspend if running on battery - Put tabs into sleep mode only when there is no power connection .
      Absolutely useless function, as for me, but someone on a laptop can be useful.

      Instead of the topic there will be a screenshot of the page, but because of this, the expenditure of RAM, the load on the processor will increase. It is worth thinking twice before ticking.
