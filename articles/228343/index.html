Started the production of wireless sensors nooLite PM111 and PT111

This week, what has long been awaited by many home automation enthusiasts, the Belarusian company Nootech started the production of nooLite wireless sensors: a PM111 motion sensor and a PT111 temperature / humidity sensor .



Wireless Motion Sensor PM111


The sensor sends an activation command to the associated power units if an object is moving in its field of view, the temperature of which is above the ambient temperature. If movement is not detected for some time, the sensor sends an off command to the associated blocks. The time to shutdown of the blocks and the sensitivity of the sensor are adjusted by the knobs on its case. There is also a photosensor with which you can configure the sensor to send commands only at night.


The range of action of the thermal sensor is up to 5 m, the time to shutdown can be adjusted from 5 s to 21.8 minutes. The maximum range to the power unit is the same as that of the other nooLite consoles - up to 50 m. Power supply - from 2 AAA batteries (" little finger "). The manufacturer claims that the sensor will work for at least 1 year from one set of batteries.

The cost of the sensor - about 750 p.

Temperature and humidity sensor PT111


This sensor measures the microclimate parameters (temperature and humidity) and transmits them over the radio channel to the nooLite Ethernet gateway PR1132 . The gateway displays the received data in its web interface. The sensor can also operate in the thermostat or gyrostat mode, sending control commands to the attached power units to maintain the desired level of temperature or humidity in the room. For example, in the gyrostat mode, the sensor can turn on the exhaust fan in the bathroom if the humidity there is higher than acceptable.

Unfortunately, the PT111 sensor is not compatible with the nooLite RX1164 USB receiver . More precisely, the receiver can receive control commands intended for power units, but cannot receive commands with data. This is due to the change in the nooLite protocol that occurred after the start of production of the receiver. The Ethernet gateway contains another version of the receiver and can accept commands with data. Probably in the future there will be a new version of the receiver or, m. the possibility of updating its firmware.
The sensor can operate at temperatures from -20 to +50 ¬∞ C, the threshold value can be set from +5 to +30 ¬∞ C (in thermostat mode) and from 5 to 99% (in gyrostat mode). The remaining parameters are the same as the previous sensor.

The cost of the sensor - about 1400 p.
