This book offers a quick way to learn the principles of object-oriented programming and learn the practice of programming in C ++. The publication can be used as a training course for beginners to master C ++, as well as a handy reference for those who want to quickly find relevant information about a particular aspect of the language. The author of the book, Alex Ellline - a professional C ++ developer, the creator of the most popular resource Cprogramming.com, offers his own unique programming teaching methodology that will allow you to become a C ++ development expert as soon as possible.

The first part of this book is devoted to the initial preparation for writing programs and using the basic components of C ++. Having mastered the proposed material, you will learn to think like a programmer and create real programs that you can show to your friends (although so far only very close ones). You will not become a C ++ expert, but be well prepared for exploring the possibilities of the language, which allow you to write really useful and powerful programs. You will gain knowledge and terminology for more complex topics.

You will learn how to write programs that work with large amounts of data (read data from files and easily and efficiently process them), and learn effective programming techniques. You will learn how to write large and complex programs with a clear and clear structure and use professional tools.

This book contains numerous examples of source code that you can use when writing your own programs without any restrictions - but without any guarantees.
