Vertical block alignment in CSS

}

.b-valign-middle p {
position: relative;
top: -50%;
}

Implementation example

Let me explain a little: our container for which we indicate the height is .b-valign-middle, inside it we need to center the content of an unknown volume that we wrap in a div and p (many will not say very well from the point of view of semantics, but it's worth it ). The first wrapper is the div in order to shift the beginning of the content to the middle of the container (top: 50%). And now the most interesting thing: the second wrapper - p is needed in order to raise the content by 50% of the height of the first wrapper; for this, we set the div to position: absolute; (relative won't work) and p top: -50%, that's all! Pretty simple, isn't it?
Moreover, this method works only in IE, normal browsers do not raise p by 50% of the height of the div, because the height is not explicitly set for it, and IE was pleasantly surprised this time!
