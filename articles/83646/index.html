KDE 4.4 released

KDE 4.4 released today - Our World, Clearly



Major changes:


Plasma:

Storage devices can now automatically connect;
Desktop configuration is now possible using JavaScript scripts;
New shell Plasma Netbook;
Plasma widgets can now be added to the system tray;
New network management interface in KNetworkManager;
Kinetic scrolling;
OpenGL is used to animate pop-up windows;
Arbitrary windows can be grouped as tabs;
Added many enhancements and functionality to KRunner.



KDE Development Platform:

With the help of multitouch, you can now resize, rotate and move;
Nepomuk: New Virtuoso is much faster and more scalable;
Nepomuk now has the ability to search on an unmounted storage device;
Integration of QtWebKit into KDE (cookies, proxies, etc., but not KPart).



Basic:

Added a new history module on the sidebar in Konqueror;
Dolphin has built-in search and version control;
Gwenview: thumbnails of images on folders;
Gwenview: a new photo import tool.
