Project Logan: NVIDIA's Next-Generation Mobile Graphics



NVIDIA is constantly trying to create a new solution that will allow something that could not be achieved with previous generations of the graphics platform. This time, it seems, NVIDIA has surpassed itself. Currently, work is underway on a new type of mobile GPU, which will help to make the graphic component of games (and not only) for mobile devices, if not perfect, then close to it.

Project Logan is based on the powerful Kepler GPU, for desktop PCs and workstations. In its official blog, the company stated that Kepler is optimized in such a way that the performance remains the same, but power consumption is about a third of what we see in modern tablets (including the fourth-generation iPad).

At the output, we see an image with a lot of chips, like a new rendering, tessellation, anti-aliasing, and physical effects. The video below shows all the achievements of Project Logan. According to the developers, about the same effect a year ago could have been achieved only with the help of a thousand-dollar GeForce GTX Titan GPU.
