Red Hat, Revolution R, Elasticsearch, MariaDB, Blockchain-as-a-service and much more is available in the Azure Marketplace

Not so long ago, we talked about the opportunities that the Azure Marketplace brings to both independent software developers (ISVs) and ordinary programmers and IT professionals who develop applications and services in a cloud environment. In other materials, we described in detail and with examples how some ISVs, for example, the Jelastic company, have already been published in the Azure Marketplace.



Since then, many interesting things have happened at the Azure Marketplace, which will be discussed in this article. Developers and IT professionals have gained access to new solutions, such as Red Hat, Revolution R, Elasticsearch, MariaDb, STRATO Blockchain platform , and many other solutions. More interesting things are expected soon, but for now let's look at the hot news.

Red hat


Long-awaited news for many - Red Hat certified solutions officially supported are available on the Microsoft Azure platform! Details about the possibilities can be found on a special page .

Red Hat users, thanks to the benefits of Red Hat Cloud Access, can easily transfer their existing licenses to the Azure cloud and deploy virtual machines with company products there.

Azure Marketplace offers a simple and easy way to deploy Red Hat Enterprise Linux 6.7 and version 7.2 with a few clicks.

Revolution R


Great news for all scientists, analysts and other specialists who use the R language in their work and Revolution products was the release of the official Revolution R Enterprise version 7.4.1 server , available for quick installation through the Azure Marketplace.

The server is available for installation both on the basis of OpenLogic CentOS Linux 6.5 , and based on Windows Server 2012 R2 .

The newest version of the server is already released under the brand Microsoft R Server 8.0 and is also available in the Azure Marketplace for installation in a couple of clicks. Try it today in a cloudy environment available on request!

Elasticsearch


Platform Elasticsearch known to many. The popular solution for indexing and searching for petabytes of data is now officially available for automatic deployment from the Azure Marketplace!

Subscriptions from Elastic, the manufacturer of Elasticsearch, are offered to Azure users with a license for commercial software and enterprise-level support for the Elastic stack.

Try Elasticsearch and Kibana right now from the Azure Marketplace.

MariaDB


Another famous open source project, MariaDB , also officially announced its support for Azure, and is now available for rapid deployment of clusters through the Azure Marketplace.

The MariaDB Enterprise Cluster cluster in Microsoft Azure offers the deployment of 3 nodes and an additional MariaDB MaxScale node running on CentOS 7. MaxScale automates the configuration of load balancing and Read / Write Split, Write Connection Router and Read Connection Router services for database nodes.

Users can, with a few clicks, lift a MariaDB cluster with this official solution available in the Azure Marketplace.

Blockchain platform STRATO


The technology of blockchains today is very popular and interested in a wide range of specialists. Azure offers many solutions for blockchain technology support, including the Ethereum and STRATO platforms . You can learn more from this list of announcements .

The good news is that the Azure Marketplace offers you to deploy BlockApps STRATO ready-made solution in just a few steps!

Open projects from Bitnami


A huge number of solutions based on open source is available in the Azure Marketplace due to the publication of ready-to-use virtual machines from the company Bitnami !

Among them are ready preconfigured and configured stacks of Django , Ruby , LAMP , MEAN and others, MongoDB , Apache Solr and other databases, Memcached , Nginx , Parse Server and other tools, many CMS ( Moodle , Wordpress , Drupal , Joomla ) and others source code tools that you can add to your cloud subscription with just a couple of clicks.

Install any of the dozens of open source solutions in a couple of clicks from the Azure Marketplace.

Azure Marketplace as a sales channel


We looked at the Azure Marketplace as a powerful tool that makes it easy for developers and IT professionals to deploy solutions and organize environments based on different solutions in the Azure cloud.

On the other hand, Azure Marketplace is a powerful sales channel for independent software developers. Today dozens of new solutions from companies from many countries are added to the store every month. These solutions receive a variety of marketing support from Microsoft and additional promotion to the target audience. An example of such a promotion is the Microsoft Partner Apps channel.

With its presence in more than a hundred markets around the world, the Azure Marketplace channel is a great way for any software developer to enter the global scene and start selling themselves anywhere in the world without the hassle of legal and tax issues.
