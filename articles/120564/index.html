PaymentGate is a payment solution from Paysto that allows you to fully automate payments to any number of recipients

PaymentGate is a payment gateway that allows you to automate the transfer of money transfers by transferring the payment command from the remote Owner system to PAYSTO. Integration is carried out only through a special protected interface.

Product presentation can be viewed on www.slideshare.net

The PaymentGate service provides the ability to both directly integrate with the client system, and the new functionality with which customers can download the list of recipients directly in the Paysto user account.

The principle of operation is as follows: The client, having registered beforehand, connects to the system using a special interface for sending transfer requests or (new) loads a list of recipients with an indication of the means of payment.

Direct integration allows you to remotely submit applications in any quantity and monitor the status of any transfer at any time - this is a complete automation of the payment process, which can be used in various systems to pay agents, employees, etc.
Further, the client replenishes the balance in his Paysto user account, from which funds will be withdrawn for outgoing transfers.

Outgoing transfers can be made in Webmoney, Yandex.Money, RBC.Money, MoneyMail, bank transfers to accounts of legal entities, as well as accounts and cards of individuals, PaymentGate has been operating since December 2010. An example of direct integration is www.onlinechange.com and www.changer.ru

Learn more about the PaymentGate service at www.paysto.ru
