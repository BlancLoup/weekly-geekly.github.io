Announcements of new Microsoft technologies on the first day of MIX'11. Great review!



Yesterday, the first day of the largest developer conference held by Microsoft to showcase the latest innovations of the company in the areas of web and mobile technologies. During the plenary report, which opened the first day of the conference, Microsoft announced a number of new technologies and products, which will be discussed in this article.

Let me remind you that MIX is not just announcements of technologies, but also numerous technical reports, of which 125 will be presented at MIX'11, 47 of which will be read by invited experts and third-party speakers. Description of the MIX'11 conference topics can be found in this article .

The first day of MIX'11 brought us interesting news on the following products and technologies:
  • Internet Explorer 10, HTML5 Labs, new demos and performance tests;
  • additional tools ASP.NET MVC 3;
  • new version of Orchard CMS 1.1;
  • new features of cloud services Windows Azure;
  • announcement of the next conference for developers;
  • and ... a demonstration of Windows on the ARM processor!
Below are the details on these topics.

Some time before the conference, some things appeared on the network, about which I would like to tell separately.

First, it was officially announced that during the conference a new version of Silverlight 5 Beta will be introduced, which will add a ton of new features, for example, 3D graphics. Good news arrived some time before the start of the conference: Windows 7, released just 1.5 years ago, already overtook Windows XP in some markets: the USA, Australia, the UK and others. Records of sales that the OS installs talk about the great success of Windows 7 around the world.

Internet Explorer 10

Probably the biggest theme of the first day of the conference was the announcement and demonstration of new features in Internet Explorer 10 Platform Preview 1. Let me remind you that Platform Preview is IE issues that offer developers the opportunity to familiarize themselves with new technologies that are added to Internet Explorer. However, Platform Preview is not intended for daily use, as it lacks basic user interface functions.

First of all, what's new in IE10 PP1:
  • CSS3 Multi-column Layout ( link );
  • CSS3 Grid Layout ( link );
  • CSS3 Flexible Box Layout ( link );
  • CSS3 Gradients ( link );
  • EcmaScript5 Strict Mode;

Windows and Internet Explorer running on ARM!

During the plenary report, Stephen Sinofsky took the stage, who is responsible for developing Windows at the company and after several graphic and other demonstrations surprised the public by demonstrating the platform on which all the demonstrations were working. It turned out that Windows and Internet Explorer were demonstrated running on an ARM 7 processor with a frequency of 1 GHz.



As you can see, the work on porting Windows to the ARM architecture continues successfully.

Next Developer Conference

In the course of the plenary report, another major conference for Microsoft developers was announced in 2011. It will be held in the United States from September 13 to September 16, 2011:

ASP.NET MVC3

After demonstrating the new features of Internet Explorer 10, Scott Guthrie and Scott Hanselman took the stage and talked about web development and ASP.NET.



It was told about the update package manager Nuget 1.2 for .NET-projects and its official website. Then there was a big announcement about updating the package of tools for ASP.NET MVC 3:



Key announcements:
  • the new version of ASP.NET MVC will include the javascript-library Modernizer and the latest version of jQuery;
  • ASP.NET MVC 3 tools will get HTML5 support, project templates will contain markup based on HTMl5 semantic tags;
  • publicly presented an updated version of the Entity Framework 4.1 with support for Code First;
  • Presented and demonstrated new Visual Studio tools for scaffolding when working with MVC3.
After that, the public was shown how to work with WebMatrix, its integration with Nuget, the use of templates from the TemplateMonster.com resource and the rapid creation of a modern website.





Orchard CMS 1.1

Then we talked about the new version of the young content management system Orchard CMS. During the report, its new version Orchard 1.1 was announced and some new features were described:
  • updated administration panel interface;
  • Recipe mechanism for presetting CMS;
  • New site for work and customization of themes http://tuning.bind.pt/ ;
  • updated portal http://orchardproject.net/gallery/ with themes and Orchard extension modules;
  • Amazon integration and more


Windows Azure

At the end of the plenary report, Microsoft lifted the veil over the following technologies, which will be presented in the Windows Azure cloud service.



Innovations include the following technologies:
  • The next version of Windows Azure SDK will include the new Web Deployment Tool, which will simplify the migration, management and deployment of IIS servers, websites and applications. This tool will be integrated with Visual Studio and Web Platform Installer;
  • Presents Windows Azure AppFabric Access Control update with new features for organizing federated authentication based on different protocols;
  • within 30 days, the final version of Windows Azure AppFabric Caching will be released, which aims to accelerate projects based on Windows Azure and SQL Azure;
  • A new Windows Azure Traffic Manager cloud service has been introduced, with which Windows Azure users can flexibly manage application performance depending on geographic location;
  • A preliminary version of the new features of the Azure Content Delivery Network (CDN) for working with streaming video based on Information Services (IIS) Smooth Streaming is presented. It will allow you to stream Smooth Streaming video to Windows Azure Storage and provide unified access to it for devices working with Silverlight, as well as for iOS and Android Honeycomb.
In addition, a number of promotions and benefits have been announced for MSDN users. More is written here and in the main announcement .

We are waiting for the second day and the new information from Microsoft on technologies Silverlight 5 and Windows Phone 7. Do not miss the broadcast of the second plenary report today at 20-00 (Moscow) at http://live.visitmix.com/ .
