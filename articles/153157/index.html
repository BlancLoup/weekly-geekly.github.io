New for web designer for September 2012

I made a small selection of new useful materials for web designers for September 2012. I hope it will be useful for habrovchan.

Useful services


Easel.io is a great service for prototyping right in the browser.



Fluid UI is a similar service for prototyping mobile apps.



CSSFilterLab - visual application of CSS-filters from Adobe.



On / Off FlipSwitch - a generator of beautiful switches on the CSS.



Infogr.am is an assistant for creating beautiful infographics.



Screenqueri.es is a handy service for testing responsive design at different screen resolutions.



Golden Grid System is a convenient grid for testing responsive design.



Free buns


Jarallax is a free js library for implementing a beautiful parallax effect. An example of use is on the website of Dublgis



tiltShift.js - jquery plugin for a beautiful rollover effect.

Casper Typeface is a very high-quality free font by Michael Chereda .
