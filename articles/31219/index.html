60+ add-ins for Gmail

Adoring lists Mashable recently published 60-plus add-ins for Gmail. I waited a long time for him to be translated into Russian, but, despite the universal love for this postal service, for some reason I did not wait.
Oh, you have to do everything yourself! :-)

Desktop applications
GCount - Alerts Mac OSX users of new emails.
gDisk - Mac OSX users can turn a Gmail account into a file storage.
gFeeder - A small pribluda that allows you to understand at a glance what else you are worried about.
GetMail - Makes it easy to migrate from Hotmail to Gmail.
Gmail Drive - Another utility for turning your Gmail account into a file storage, works from Windows Explorer.
Gmail Loader - A program under Windows or Linux, helps to upload your archive of letters to your account on Gmail.
GMailFS - Gmail account is converted into a virtual disk, which is visible through Windows Explorer.
GotMailG - Mac widget that shows the number of unread emails in the Inbox.
POP3 / IMAP server for GMAIL in JAVA - Provides POP3 access to Gmail from mail programs, IMAP in development.
Send To GMail - With one click, you can send any file from your computer to your Gmail account.

Firefox Utilities
Better Gmail - Several best scripts for Greasemonkey in one plugin. Skins, saved searches, keyboard macros and all that.
Delegate to Remember The Milk - Adds a Gmail button and lets you create to-do lists.
Dragdropupload - Now attachments can simply be drag-n-drop.
GCalQuickTab - Quickly switch between Google Mail and Calendar using keyboard shortcuts.
Gmail Checker - Checks Gmail for new messages.
Gmail Manager - Allows you to switch between multiple accounts and check how much spam and mail are on them.
Gmail Notifier - Reports new emails.
Gmail RTL - Adds a button to change the direction of the letter, which is now available only in the interface in Arabic and Hebrew.
Gmail S / Mime - Allows you to send and receive encrypted emails.
Gmail Skins - Changes the appearance of the box, adds HTML tables, emoticons and all that.
Google Toolbar - The official Google toolbar for all kinds of Gmail features.
Gspace - Turns a Gmail account into a virtual disk for files.
gTalk Sidebar - Implements Google Talk in the sidebar.
GTDInBox - Utility for fans of the system Getting Things Done, embedded in Gmail, helps in organizing.
WebMail Notifier - Checks for Gmail mail, also works with other email services.

Greasemonkey scripts

Clean Gmail Print - Removes the Gmail logo when printing a letter
Folders4Gmail - If you prefer the old style of sorting letters into folders - this is for you.
Force HTTPS for GMail, GCal, and GDocs - Changes links in Google-based applications to work through HTTPS. (here I am not quite sure about the translation)
gDate2Cal 0.1 - Turns dates in Gmail into links so that they can be added to Google Calendar.
Gmail + Reader Integration - Integrates Google Reader into a Gmail account.
GMail Attachment Icons - Shows an icon that allows you to find out what type of file is in the attachment to an even undisclosed letter.
Gmail attachment reminder v2.4.1 - Recalls that you forgot to attach a file to a letter (if such intention was expressed in the letter).
Gmail Contact List - Adds a contact list next to the ‚ÄúTo‚Äù field.
Gmail Conversation Preview - Right-clicking on a conversation allows you to view its preview in a small window.
Gmail Date Search - Adds a search by date to the Gmail search form.
Gmail Delete Button - RED Color - Colors the Delete button in red to make it harder to click on it accidentally.
Gmail Emoticons! - Adds an emoticon button (emoticons) next to the "Spell Check" button.
Gmail Encrypt - Encrypts letters and creates encryption keys.
Gmail FavIcon Alerts - The Gmail icon in the tab will change color if there are unread letters in the box.
Gmail from address selector - Helps change the reply-to field depending on where you are writing. For example, sending letters to work addresses, you can select your work address in the From field.
Gmail Full Width - Removes the ad unit on the right.
Gmail Google Calendar sidebar - Adds a Google Calendar to the Gmail interface.
Gmail Lite - Removes ads, footer, stars and chats.
Gmail Macros - Create your own shortcut keys to simplify your use of Gmail.
Gmail Multi-User Login - For those who have more than one account on Gmail, this script will show a drop-down menu with all accounts.
Gmail Multiple Signatures, plus Float - Changes the signature depending on what address was selected in the From field.
Gmail - One Click Conversations - One -click access to the latest conversations with the selected recipient.
GMail Header, Navigation and Ads Switcher - You can choose whether to show the top of the interface, the navigation bar and advertisements.
Gmail - Zip Attachments and Download - Allows you to download all attachments without opening a message.
GmailTo - When you click on the link to the mailbox, the Gmail interface opens for writing a letter.
Google Air Skin - Air Skin for Gmail, Google Calendar, Google.com and Google Reader.
Label Colors - Different labels can be displayed in different colors.
Plaxo4Gmail - Syncs Gmail with Plaxo.
Saved Searches - Retains links to several recent search queries on Gmail.

Any miscellaneous
GBooks - Turns a Gmail account into a bookmark server.
gExodus - Exports your emails from other Gmail accounts in mbox format.
Gmail icon generator - Generates a small image with your email address to be placed on sites to protect against spam bots.
Gmail Music - Saves your MP3s in a Gmail inbox and plays them on your computer.
Gmail Todo - Turns Gmail into a To-Do list.
PhpGmailDrive - Retrieves a list of attachments from your account on Gmail and allows you to publish them on a website or blog so that anyone can download them.
You've Got Gmail - Plugin for Trillian to manage your Gmail account.
