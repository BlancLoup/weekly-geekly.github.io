Convenient search directly in the explorer with the Bing search provider

As many know, Windows 7 supports OpenSearch providers for native search in the windows environment, without the need to use a browser. For example, I use this provider to search for articles on MSDN.

Recently, a search provider has been published on the Bing community site, which, after being integrated into Win7, allows you to search directly in the explorer. Moreover, in windows 7 there is a preview of the elements of the conductor, which allows you to preview the search results by Bing. Thus, Win7 users can very conveniently navigate through the result of a search engine, view found links without navigating through tabs and even without a browser.
