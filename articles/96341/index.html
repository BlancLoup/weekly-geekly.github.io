Sync notes in Google Chrome

Chrome Notepad is a note-taking extension that supports the built-in Google Chrome sync.

It is important that for synchronization of notes it does not use any third-party servers / synchronization services, but the built-in synchronization option for bookmarks.

This technique, invented by George Papadakis, is described (with examples) here so that other developers can implement it in their extensions.

Note syncing is turned on by default if Google Chrome has already configured bookmark sync. Synchronization, according to the developer, takes place every 5+ minutes.



The picture below (taken from the expansion page) shows a note synchronized between several computers.



In addition to the note itself, the extension remembers and synchronizes the cursor position and scrolling.
In the settings, you can turn off the synchronization of notes (will not affect the synchronization of bookmarks), change the size of the "notepad" window and the font size.

If anyone does not know how to enable bookmark synchronization:
Settings and management> Settings> Personal stuff> Synchronization> Settings> check Bookmarks> OK. Done!
