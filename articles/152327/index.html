Free tools for VMware and Hyper-V

Today we want to tell about 5 useful freeware utilities that will be useful in the daily work of the administrator of virtual environments VMware vSphere and Microsoft Hyper-V. All these tools were developed by Veeam, whose products focus on backup, report generation and operational monitoring of virtualization systems based on VMware and Hyper-V. Today we will tell about:

  • Veeam Backup Free Edition for VMware and Hyper-V
  • Veeam ONE Free Edition for VMware and Hyper-V
  • Veeam Extended Generic Report Library for Microsoft System Center
  • Veeam Stencils for VMware and Hyper-V
  • Veeam Management Pack for VMware

So let's get started:

1. Veeam Backup Free Edition for VMware and Hyper-V

With the free version of Veeam Backup you can quickly:
  • Back up, archive or copy virtual machines without having to shut them down or pause them.
  • Recover data from virtual machine backups.
  • Migrate and manage virtual machines and files.

The product provides the following features:
  • VeeamZIP : the ability to quickly create VM backups on demand for work, archiving or transferring VMs.

  • Instant File Recovery (IFLR) : Restore individual guest OS files directly from a backup image. IFLR eliminates the need to restore the entire virtual machine if you need only a few guest files. You simply browse the guest file system directly in the backup and select the files and folders you need to restore.
  • File Manager : easy management of virtual machine and host files:
    • Overview : review of the virtual machine and host files in a familiar graphical interface, similar to Windows Explorer.
    • Copy between hosts : for example, copying virtual machines and templates between data stores.
    • Copy between host and Windows / Linux : for example, the delivery of ISO files to VMware hosts.
  • Quick Migration for VMware: Migrate a running virtual machine to any host or storage ‚Äî even if you are not using clusters or shared storage, and vMotion is not available in your vSphere edition.


2. Veeam ONE Free Edition for VMware and Hyper-V


Veeam ONE - a product for real-time round-the-clock monitoring of the virtual environment, optimizing resource use, documenting and creating reports for management. The free version of Veeam ONE provides all the same basic features as the full version, including full data collection, support for simultaneous multiple users, a dashboard, and the use of a Microsoft SQL Server database. The free version imposes no restrictions on the number of hosts and virtual machines, the number of users, or the size of the data archive. The limitations of freeware mainly relate to advanced reporting capabilities, the configuration of complex notifications, and specific dashboards.


3. Veeam Extended Generic Report Library for Microsoft System Center


The Report Library (GRL) allows you to analyze the state and performance of infrastructure objects - for both physical and virtual environments. It contains features that are not provided in the standard Microsoft System Center report library. The Veeam Extended Report Library includes reports such as:
  • Alert Statistics Report - analysis of the statistics of triggering alerts in two modes - by the rules / monitors and by objects.
  • Generic Performance Top (Bottom) N Report - a visual representation of infrastructure objects, instances, or both, for a selected performance indicator.
  • Performance Report - a visual representation of the values ‚Äã‚Äãof performance indicators in charts and tables.
  • Performance Details Report - analysis of trends with the help of detailed performance data.



4. Veeam Stencils for VMware and Hyper-V for Visio


A set of Microsoft Visio graphic elements for creating detailed diagrams for a virtual infrastructure. The kit includes the following items:
  • ESX and Hyper-V hosts
  • Data center
  • SCVMM Servers
  • local and network storage
  • logical device numbers (LUN)
  • virtual machines with their status
  • network interface cards (NIC)
  • networks, etc.

Report example:

The set of elements Visio was originally created for the product Veeam ONE, which allows, in particular, to build reports in the format of Visio diagrams.

5. Veeam Management Pack for VMware


The free package of 10 sockets Veeam Management Pack (Veeam MP) is designed for IT administrators who are already using Microsoft System Center in their work, but are not yet using the full version of the Veeam MP product. Veeam MP is specifically designed to monitor the VMware virtual environment using System Center. The solution allows you to maintain uniformity of administrative policies and rules for System Center and monitor the status of the entire infrastructure using the Operations Manager console. Veeam MP allows you to avoid the cost of acquiring, deploying, researching and supporting any additional monitoring platform and using an existing platform.
