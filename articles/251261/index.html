Weekly assembly Vivaldi 1.0.111.2

Hello!

Another week has passed and we traditionally offer to test the next test build of the Vivaldi browser (number 1.0.111.2). Work on this version went along the same lines: implementation of the functions requested by users, improvement of the existing capabilities of the browser, and correction of errors. Let's start with the new products, which today are not very many (much still in the process of implementation).
