Overview of Microsoft Azure platform updates for September

The news digest of the Microsoft Azure cloud platform contains key announcements and content for developers, IT professionals, and all professionals interested in cloud technologies and the Azure platform. Previous editions of the digest can always be found at the following link .

In this digest - interesting announcements and content. The next digest will be even more interesting, because we will soon have Connect () and TechEd , where there will be even more news.



Announcements


Not about Azure, but important: JTC 1 announced two key standards for cloud computing, in the development of which we took an active part:
  • ISO / IEC 17788, Cloud Computing - Overview and Vocabulary
  • ISO / IEC 17789, Cloud Computing - Reference Architecture

Partnership with Docker . In the same series - Mark Russinovich explains why Docker.

Partnership with Cloudera and CoreOS - more partners - more choices.

New virtual machines of level G, according to the news the most powerful, which in general is in the cloud market. From 1 to 16 vCores, 3.5 to 112 gigabytes of RAM and 800 gigabytes of SSD.

GA : SQL Azure DB , modes of operation. After the global rethinking and changes in SQL Azure DB, new modes have become publicly available.

GA: Redis Cache Service - besides the fact that Redis is a popular caching mechanism, it is fully accessible and is the recommended caching solution for Azure applications.

GA : Site Recovery - a service for disaster recovery (including high availability, replication, security, and automation of the entire process) for virtual infrastructures has been released to full availability. After the acquisition of Microsoft company InMage, engaged in activities in the related field, the functionality of their product was integrated into Site Recovery. More details .

GA: Web Sites Web Jobs - the function of performing background tasks when hosting sites in the Azure cloud is now available.

Document Explorer for DocumentDB - we just started a series of articles on DocumentDB here. From updates - Document Explorer came out, with which you can view the status of your JSON-documents in DocumentDB.



GA: public IP for virtual machines - the function of using public IP for virtual machines and the roles of cloud services has become available to the public. Now you can, for example, raise an FTP server in the cloud or watch directly over IP what is happening with the virtual machine.

GA: API Management - a service for creating and managing APIs based on cloud infrastructure has become publicly available.

GA: Mobile Services .NET - Development support for .NET for Azure Mobile Services has become publicly available.

Preview: Live Streaming, Content Protection - in Media Services you can now do Live Streaming and content protection. Protection: static, dynamic encryption, PlayReady DRM, Clear Key Advanced Encryption Standard (AES) 128-bit.
