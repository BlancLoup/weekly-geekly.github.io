PCs and servers based on the Elbrus-4C processors are on sale

The Russian company MCST announced the start of sales of personal APM computers for office use Elbrus-401 and Elbrus-4.4 servers based on 4 nuclear microprocessors Elbrus-4C, created using 65 nanometer technology. The company's website provides detailed characteristics of these 2 new products from the Russian manufacturer, and now we will briefly highlight the main ones.



Sales of Elbrus-401 will begin this summer, the approximate cost is $ 8,000. The impressive price of the system is not explained by stunning technical characteristics, they are quite ordinary (if not more), primarily because its sale is aimed at legal entities that have promising projects. Prices for the server "Elbrus-4.4" are expected to be even higher than on the PC.

ARM computer "Elbrus-401"


The automated workplace "Elbrus-401" was developed on the basis of the microprocessor "Elbrus-4C" and is intended for the equipment of automated workplaces (AWP) of operators, organization of microservers and information terminals, applications in industrial automation and in systems with increased requirements to information security. AWP "Elbrus-401" is made in the case of MiniTower.



A set of peripheral interfaces is implemented on the motherboard: Gigabit Ethernet, SATA 2.0, IDE, USB 2.0, RS-232, DVI (the integrated video card is used). The board provides a footprint and an SSD-format microSATA disk. There is a PCI-Express x16 slot (8 lines are used), which allows you to install expansion cards or a 3D video card of the AMD Radeon family (installed in a regular configuration). The IDE interface is connected to the CompactFlash card slot (used for the needs of the binary translator). The base operating system for the automated workplace "Elbrus-401" is the OS "Elbrus". It is based on the Linux kernel and supports many open source applications, including the AbiWord editor, the GNumeric spreadsheet, the Firefox browser, and the email client. There are tools for developing application software, tests for self-diagnostics of equipment. "

Server "Elbrus-4.4"


The Elbrus-4.4 server is a rack-based server based on the Elbrus-4C microprocessors. It contains four Elbrus-4 processors, one or two south bridges of the KPI. The Elbrus-4.4 server is suitable for organizing web servers, data servers, data storage systems, remote desktop servers, and for organizing high-performance clusters.



The Elbrus-4.4 server is made in a rack-mount case for installation in a standard 19 ‚Äùrack with a height of 1U. The overall server performance is 200 Gflops single precision. The server supports installation of up to 384 GB of DDR3 memory with error correction.

The base operating system for the Elbrus-4.4 server is the Elbrus OS. The Elbrus-4C processor supports hardware-software binary translation of Intel x86 and Intel x86-64 codes into codes for execution on a processor with the Elbrus architecture. Supports end-to-end execution of more than 1000 popular programs written for Intel processors, and the launch of more than 20 operating systems, including Windows XP, Linux, QNX on Elbrus processors.
