Google Code closes and invites everyone to go to GitHub

All project owners on Google Code received a letter.
Abbreviated translation of the news:
Starting today, we will stop the possibility of creating new projects on Google Code. The service will be closed in about 10 months by January 25, 2016. Listed below are links to migration tools designed to help you migrate your projects.

Terms:
March 12, 2015 - Creating new projects is disabled.
August 24, 2015 - Switch to read-only mode.
January 25, 2016 - Closing the service. It will be available to download the archive of the project including the source code, a bugtracker and a wiki. The archive will be available until the end of 2016.

Google will continue to provide Git and Gerrit hosting for projects such as Android and Chrome. We will continue to support mirrors of projects such as Eclipse, kernel.org, and others.



Next are links to the Google Code to GitHub exporter tool and stand-alone tools for migrating to Bitbucket or SourceForge.
