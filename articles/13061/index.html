Apogee Drive 350: New Swiftech Cooling System

The company Swiftech announced the release of a new liquid cooling system Apogee Drive 350. This is a processor water unit, which has a pump with a capacity of 330 liters per hour. The combination of the water block and the pump, according to the manufacturer, provides compactness and ease of installation.


Apogee Drive 350 is compatible with AMD (stones Socket 754, 939, 940, Socket AM2 and Socket F) and Intel Pentium 4, Core Duo, Core 2 Duo, Extreme (Socket 775) processor sockets. The cost of the cooling system from Swiftech, which should be on sale in about a week, will be $ 100.

Buy Apogee Drive 350 can be both separately and complete with a set of H20-120, which includes a 120-mm radiator with a built-in coolant reservoir. It is noteworthy that the combined CO H20-120 (the cost of which will be only $ 160) can replace four-component systems (water block, pump, radiator, reservoir). And for the mass user it is more than profitable, since it will be necessary to tinker with the installation and placement of CO in the case much less.
