All in one


phpDesigner 7 is both a powerful and fast PHP application development environment, and a full-featured HTML, CSS and JavaScript editor.

This IDE is extensible and customizable and carries:
  • Easy to learn and easy to use interface;
  • intellectual syntax highlighting;
  • instant syntax analyzer;
  • OOP support;
  • hints in the code;
  • code completion (code insight) on the fly;
  • code templates and snippets;
  • support for projects, frameworks;
  • intuitive code navigation and much more ..

phpDesigner 7 offers new features and improvements that will help improve, ease the programming process and make phpDesigner a more complete tool for web development for both beginners and professional programmers.

Boost your productivity with phpDesigner 7


Do more in less time. phpDesigner 7 includes many elements that save time:
  • Easy to learn and easy to use interface;
  • Intellectual syntax highlighting;
  • Completing code for PHP, HTML, CSS, JavaScript, including jQuery and other known frameworks;
  • Code tips for PHP and JavaScript;
  • "Inspector" HTML-code;
  • Intuitive process of navigating code and projects.



Intelligent syntax highlighting


Switching between PHP, HTML, CSS, JavaScript highlighting occurs automatically, depending on where the cursor is located. The nearby code is obscured, allowing you to more clearly see the text you are working with. See syntax errors as you type.
  • Syntax highlighting with dimming and without.
  • PHP, HTML and CSS syntax checking.

Program faster using hints and code completion


Increase coding speed with support for completing nested code and hints for PHP and JavaScript.
  • Completion of code and hints;
  • Support for stacked PHP objects;
  • Extended support for working with PHP classes;
  • OOP support in PHP and JavaScript.
  • Completing the code for phpDocumentor;
  • Completing code for HTML, XHTML, CSS;
  • Completion of code and tips for JavaScript;
  • "Inspector" HTML-code;
  • HTML Browser (with filters), CSS, and JavaScript.

Work faster with your favorite PHP and JavaScript frameworks.


phpDesigner 7 supports all PHP frameworks. You can instantly access any class, function, variable as you type.
Also, phpDesigner supports popular JavaScript frameworks such as jQuery, Ext JS, YUI, Dojo, MooTools, and Prototype .
  • Work with PHP frameworks;
  • Completion of code and tips for JavaScript;
  • Move to any PHP, CSS, and JavaScript "ads";



Debugging and profiling PHP scripts using Xdebug


Analyze the code step by step using breakpoints. Look for bottlenecks in your code and optimize slow parts using a profiler.
phpDesigner 7 fully supports PHP 5.3 and is already configured to work with Xdebug.
  • Debugging with breakpoints;
  • Call stack;
  • Context variables;
  • "Watches"
  • Evaluation of injections (local, global);
  • Debugging errors;
  • PHP syntax checking;
  • Profiler



Learn PHP


PHP help integration allows you to learn and explore the PHP world. Reduce your learning time.
