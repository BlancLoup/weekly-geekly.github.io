Online data analysis software on Stepik.org

Autumn 2016 will be rich: SPbAU RAS will not only restart Stepik.org for the second time, but will launch the new online retraining program , this time in collaboration with the Institute of Bioinformatics, and it will be dedicated to working with data.

The program will begin in October'16 and will run without deadlines, so you can join it at any time, and you can spend as much time on learning as the rhythm of life allows: it‚Äôs not at all necessary to sleep at night and combine difficult work with equally complex studies.



The online data analysis program is designed for specialists in any industry and provides an opportunity to focus on areas of interest to the student: some of the courses are required for all students, and from the remaining courses you can add an individual learning path.

List of courses included in the program:

Required:

  1. Data analysis
  2. Computer architecture
  3. Database
  4. Discrete Math
  5. Python programming
  6. R programming
  7. Statistics
  8. Linux

Optionally:

  • Algorithms: theory and practice
  • Additional chapters of statistics
  • Mathematical analysis
  • Machine learning
  • Neural networks
  • C ++ programming
  • Java programming

All those who successfully complete the program will receive a professional retraining diploma from SPbAU RAS. The cost of training at the program is 1 999 rubles per month and includes the support of assistants, exams and a diploma; however, all courses that are included in the online program will be or are already open for free to study.

More information about the program: data.stepik.org . Frequently asked questions: data.stepik.org/faq .
