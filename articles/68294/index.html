Mandriva Linux Community Builds

Mandriva Linux is officially available for free download only in the Free, One KDE4 and One Gnome editions. But there are also a number of assemblies that are classified as ‚Äúunofficial‚Äù and executed by various Mandriva Linux communities. Let's take a look at what builds distribution lovers can still try. At once I will say that informal assemblies may have more errors than official assemblies, but this is leveled by the resulting features.

Build Italian Mandriva Linux Community ( MIB )


  • MIB Live E17 2009.1 - International Edition is an assembly where E17 is used as the graphical environment. Image size 2.4 GB
  • MIB Live Xfce 2009.1 - build based on XFCE. A list of incoming software can be found here: mib.pianetalinux.org/2009.1/iso/docs/xfce
  • MIB Live Gnome 2009.1 - everything is clear from the title. The list of incoming software here: mib.pianetalinux.org/mib/en/mib-news/1-news/392-mib-live-gnome-20091.html

Build French community Mandriva


XFCELive2009Spring is a semi-official build, supported by the Mandriva Linux head office. Read more about it on the wiki: wiki.mandriva.com/en/XfceLive

Russian assembly


  • EduMandriva One LXDE - build with LXDE-based training software. Read more on the wiki: wiki.edumandriva.ru/index.php/EduMandriva_LXDE_ONE_2009.1_CD
  • MiniOS Fire is an assembly that is positioned to be installed on netbooks / USB drives. Also differs original installation software in the form of modules. You can read about it on the project website or here: http://mdv-club.ru/*forum/read.php?10,38007
  • MagOS - an assembly that was made in opposition to the commercial version of Mandriva Flash. Allows you to work with a USB flash drive, saving on it all the changes and additions. You can read in the appropriate forum thread: http://mdv-club.ru/*forum/list.php?10
  • Mandriva Linux 2009 LAR - Distributive from Russian Club Mandriva Linux
