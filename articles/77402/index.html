Pidgin 2.6.4

The other day version 2.6.4 of Pidgin, a multi-protocol client for instant messaging networks, was released.

Despite the "minority" of the release, it contains a number of important improvements both in shells (Pidgin and Finch) and in the core (libpurple):
  • implemented the function "hold" in the window of audio / video-communication;
  • MXit protocol added;
  • a theme editor for icons and emoticons;
  • Fixed receiving offline ICQ messages - the function did not work from version 2.6.3. Do not be surprised if after the update you will immediately receive all your offline messages in a week;

many other important changes.
