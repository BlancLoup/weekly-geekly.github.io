Integration of E1tele.com service with Microsoft Outlook

A client addressed our company with a question: is it possible to make calls from outlook contacts? While our specialists helped him deal with this task, I decided that I would write a detailed article about it. Still, our service can be used in all its diversity, especially if you figure out how easy it is.
If you have a habit of putting all the phone numbers of partners and colleagues in Microsoft Outlook contacts, this can turn you into a functional communication service that facilitates daily communication.
We present you the application E1: OUTLOOK , which allows you to make calls with a single click.
Any contact from Microsoft Outlook and other interfaces that are compatible with Windows-based applications may be called by you at the moment it comes to mind. The solution in the form of a TSP driver works on the principle of callback (Callback) and integrates easily with Windows.

The TSP driver is available for download as a w3ttapi.tsp file. To install TSP and enable the dialing feature from any applications that support Microsoft TAPI (standard Microsoft dialer, Outlook and others), you need:

1. Make sure that the Telephony service is running (from the Control Panel, Administrative Tools -> Services):



2. Copy the TSP driver (W3ttapi.tsp) to the C: \ Windows \ System32 \ directory.

3. Install the TSP driver:

- Go to control panel -> Telephone and Modem



- Select a country and enter the city code (if this has not been done before):





- Go to the Advanced / Advanced tab and add the Web3Tel TSP service provider to the list of existing service providers.



To do this, click the Add / Add button and select Web3Tel TAPI from the list:



After clicking the Add / Add button, Web3Tel TAPI will be installed:



After installing Web3Tel TSP, you can make international and long distance calls.


If you already have an account on E1tele.com , you need to configure Web3Tel TSP to use your credentials.

1. In the list of available service providers, select Web3Tel TAPI and click Configure / Configure:



2. Enter the following parameters in the appropriate fields:



- Host server Web3Tel (as shown in the example www.web3tel.com ).
- Port must be 8022
- Hashed: Yes
- User / Password - your credentials at E1tele.com
- TerminalDN is a registered terminal (mobile or SIP number) to which incoming calls will be received for the connection.
- Realm: servicedata@e1tele.com
- LineID: 0


If you want to use E1tele.com with Microsoft Outlook, you should configure it as follows:

1. In Microsoft Outlook, open the Contact (in the example used by Outlook 2007):



2. Click on the "Call" button with the phone icon:

Now you can make calls using E1tele.com .

Conclusion
TSP seamlessly integrates with Windows. This means that any application can use this TSP driver.
Now we support:
  • Multiple calls
  • Call hold
  • Drop call
  • Swap calls
