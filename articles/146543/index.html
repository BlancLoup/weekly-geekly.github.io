Summer. Windows 8. Application Excellence Labs

The previous week was rich in news from Microsoft: this is the announcement of Microsoft Surface and the announcement of Windows Phone 8 . Both of these announcements caused a lively discussion on the Habr√© and the blogosphere.

Somewhat earlier, in May, at the opening of the DevCon'12 conference , we showed the first applications from Russian developers in the Windows Store. Today I would like to please those who wish to become part of the following announcements :) We have launched a new series of Application Excellence Labs program.



So, Application Excellence Labs will be held in Moscow from July 23-27.
Attention! Remote Script Possible (Lync, Skype)

Application Excellence Labs
What is Application Excellence Labs?
  1. Free consultation from a Russian-speaking expert.
  2. Opportunity to get one of the first in the Windows Store.

What do you need to take part in Application Excellence Lab?
You must have a working Metro style app for Windows 8 that has been internally certified by Visual Studio (Windows App Certification Kit). The application is not required to fully pass all tests, however, testing is necessary.

How to get to Application Excellence Labs?
Application Excellence Labs is a very, very compact format with a very limited number of participants. Therefore, among those who want to get to the laboratory work, an additional selection is carried out:
  • Before July 16, send an email with a request to email: rush4apps@microsoft.com with the title ‚Äú[W8 AEL] Name of your application‚Äù. In the letter, specify
    • Description of the application (what tasks it solves and for what audience it is calculated)
    • Screenshots of the main screens of the application
    • Test results via WACK
    • Contact us
  • The authors of the best applications will receive an invitation to the Application Excellence Labs.

What do you get from participating in Application Excellence Labs?
  1. A report with recommendations on how to improve your application.
  2. The developers of the best applications will be able to be among the first to get into the Windows Store!

Application Excellence Labs. Format
Everyone wants the applications in the Windows Store to be really high-quality - and we want to help you, as a developer, to do just that. Laboratory works are organized for this, in which you can discuss your application with our (Russian-speaking) engineer, get expert feedback and, as a result, come to understand how to make excellent from a good application.

How are the Application Excellence Lab:
  • Testing the application in various scenarios, including on the tablet.
  • Analysis of the design and layout of the application in accordance with the metro style.
  • Analysis of the layout and use of the menu, flayout, ‚Äúrevitalizing‚Äù tiles.
  • Analysis of examples of the implementation of contracts. Search, Share, Settings.
  • Discussion of the monetization strategy of the application.
  • Answers to technical questions.
