New File button on Github

Starting today, Github has the opportunity to create files directly in the repository, without leaving the site, and edit their contents.



Clicking on the button immediately opens a new editor window, where you can copy or write text. In this way it is very convenient to add files README, LICENSE, documentation. If you click on this button in another repository, a fork is automatically created.
