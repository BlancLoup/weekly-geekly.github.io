Joint decision making in Google Wave

Today I want to share with you information about the possibilities that Google Wave provides for decision-making. And if we are talking about Google Wave, then of course we will consider exactly the collaborative decision-making.

Even without additional extensions, the team has the opportunity to communicate and jointly display a common opinion on the desired issue. There is a text editor for this, a very important distinctive feature (from mail or forums, for example) is that when users go to the wave, they see only up-to-date information, and all hypotheses, unrealized guesses, etc. do not clog up the wave (although the whole story is available for viewing in Playback mode).
Although Wave technology is still very young, but thanks to the excellent API, new extensions are being created every day, which using Wave capabilities will enable many things to be done faster and more conveniently. This is what makes Wave a very attractive decision-making tool. Consider the most interesting and close to the topic of expansion.

Mind map gadget


Install XML | Gadget xml

The gadget allows you to work together to create a MindMap ( mind map ). Mainmeps are a visual hierarchical representation of an idea, in the center is a word or phrase that describes the idea very briefly, further from it the main points branching out the idea and so on recursively branch out. It is very convenient to simultaneously edit such a mindmap - throwing ideas to each other.
