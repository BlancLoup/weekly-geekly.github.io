Silverlight 2 Beta2 features and terms

At the North American TechEd conference, Bill Gates announced the Silverlight 2 Beta2.

News number 1 - the second beta will be available until the end of this week!
News # 2 - second beta features:
  • UI Framework - improvements in animation support, error handling, accessibility, keyboard input, speed. More compatibility between Silverlight and WPF.
  • Rich Controls is a new model of Visual State Manager templates that facilitate working with controls. Introduction TabControl, text wrapping and scrolling for TextBox, as well as additions for DataGrid - automatic resizing, sorting, reordering, improving performance Most controls now live at runtime.
  • Networking - improved support for cross-domain queries and security, WebClient downloads and full-duplex communication.
  • Rich Base Class Library - improved work with streams, LINQ-to-JSON, support for ADO.NET Data Services, improved support for SOAP.
  • Deep Zoom is a new format based on XML, the introduction of MultiScaleTileSource. Improved support for notifications when zooming and moving around the image.

Also from the news from there - Internet Explorer 8 Beta2 will be in August in 20 languages.
