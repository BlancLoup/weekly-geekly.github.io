Anatomy of Blogger templates

Introduction


On the Web, as it turned out, there is not so much information on how to create your own themes for Google’s blogging platform, at least in RuNet. There are some scanty posts about how to set up a separate property of a specific widget, and only that. Therefore, in this article, I decided to consolidate knowledge on this topic.

Basic Template Elements


In general, a Blogger template is an XML file containing a description of the HTML page (CSS formatting is also included in this file). It looks like this (some details are omitted for simplicity): 

<?xml version="1.0" encoding="UTF-8" ?> <!DOCTYPE html> <html xmlns='http://www.w3.org/1999/xhtml' xmlns:b='http://www.google.com/2005/gml/b' xmlns:data='http://www.google.com/2005/gml/data' xmlns:expr='http://www.google.com/2005/gml/expr'> <head> <meta content='IE=EmulateIE7' http-equiv='X-UA-Compatible'/> <title> ... </title> <b:skin> ... </b:skin> <b:template-skin> ... </b:template-skin> </head> <body> ... </body> </html>

I will not consider the <b:skin> and <b:template-skin> blocks responsible for CSS design, since I personally use the Twitter Bootstrap framework to create templates. Now we better deal with the part that is responsible for displaying content.
