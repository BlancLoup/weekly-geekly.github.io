Opera 10 is preparing a revolution in the browser market



The beta version of the Opera 10 browser offers new functionality, new performance and a new interface. In the near future, users expect other pleasant surprises.

June 3 - Oslo, Norway - Opera Software today announced the first beta version of its browser for Opera 10 desktop platforms. Opera 10 browser offers new opportunities for browsing the Internet, a new interface, and, of course, improved performance.

The beta version of Opera 10 is already available for free download: www.opera.com/next

Opera Turbo: high speed on slow channels
If you use dial-up access channels to the Network (dial-up), the service of your ‚Äúbroadband‚Äù Internet provider leaves much to be desired, or the Wi-Fi access point is so overloaded that it does not provide an acceptable connection speed, Opera Turbo technology will come to the rescue. Thanks to the advanced data compression algorithm, Opera Turbo allows several times to increase the speed of loading sites with a slow connection. This technology gives the feeling of working with a broadband channel, even if dial-up is used in practice.

"Makeup" for the interface
In order to improve Opera's browser interface and give it a non-standard, unique look, Opera Software invited Jon Hicks, a world-renowned designer, to collaborate. ‚ÄúAt first, we planned to limit ourselves to upgrading only some elements and focus on creating graphics for new options,‚Äù says John Hicks. ‚ÄúBut over time, the new interface elements have become so different from the general style that we decided to rework the entire browser design. Upgraded tabs (tabs)
Now the height of the tab bar Opera 10 can be changed at your discretion. If you lower the lower border of the panel with the mouse, not only the titles of the web pages opened in the tabs will appear in the upper part of the screen, but also their thumbnails - thumbnails. Working with a lot of open tabs has become even easier!

Express panel - for each its own
Since Opera first introduced the Express Panel for quick access to websites (Speed ‚Äã‚ÄãDial), this option has become one of the most sought-after browsers. Now each user can customize this component at his discretion. First of all, based on the size and format of the monitor, you can set the number of sites for quick access - from 4 to 25. In addition, you can also change the background image of the Express panel, for example, set photos of family or friends as the background.

Performance and Web Standards
Even with Opera Turbo technology turned off, Opera 10 quickly downloads bulky web pages like Gmail and Facebook. The performance of Opera 10 is 40% higher than Opera 9.6. Web developers will appreciate support for web fonts, RGBA / HSLA color models and improved support for scalable SVG vector graphics.

In addition, web developers come in handy with new functionality Opera Dragonfly. The third alpha version of this toolkit allows you to edit DOM document object models and validate HTTP headers. The toolkit will be available in 36 languages.

But that's not all!
The beta version of Opera 10 has much more features. For example, the spell checker option built into the browser now supports 51 languages ‚Äã‚Äãand uses the format of the popular open source dictionary Hunspell. In addition, Opera 10 has received the function of downloading updates - it can be done either automatically or in accordance with the user's settings.

The capabilities of the Opera Mail mail client have also improved - it will fully support the HTML format. In addition, it has the function of automatically deleting letters from the POP server after a set time.

Opera browsers include a mail client, an RSS feed aggregator and a BitTorrent client, but by default you can also use external web-based email services, as well as popular RSS feeds and BitTorrent apps.
