Great-grandfather of modern computers landed where the latest computers are designed and manufactured - in Silicon Valley.

The London Science Museum built two exact replicas of the Difference Machine # 2 according to the original project of Charles Babbage. Designed by Babbage between 1847 and 1849 and designed to calculate the values ‚Äã‚Äãof polynomials, the five-ton colossus, consisting of 8,000 parts, was built by a team of engineers by 2002. Work on it took 17 years.

Fully mechanical Differential machine can add 31-bit numbers and calculate polynomials up to the seventh order.

The project of Difference Machine No. 2 included an absolutely original printing component, which, if Babbage could then realize his project, would be the first automatic typesetting machine in history. A printing device can either print results using ink or generate printing form. In doing so, it automatically formats the results into columns and uses two different font sizes.
