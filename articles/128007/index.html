Minus.com: Almost Dropbox with Social Chips

I, like many of the local inhabitants, use Dropbox in my life. True, it is convenient: I throw there books that I want to read, some non-critical documents, backing tracks of songs that I love to sing under :)

Today our employee from Odessa came to the office (he has a Motorola Milestone) and spoke about the new file sharing in his phone - Minus.com . I became interested, I saw that it was not only about Android, I dug a little deeper and decided that the site is worthy of a topic on Habr√©.



Immediately about the buns:
  • Each user 10 Gb free
  • For each invited user +1 gigabyte (upper limit - 50 gigs)
  • Any movement of any files - without any restrictions
  • Single file size is limited to 2 gigabytes.
  • Convenient tools for sharing files with other people
  • A complete set of clients: Win + Lin + Mac and Iphone + Ipad + Android (soon WP7), web client and extensions for Chrome and Firefox
Liked? Let's go further:

So, let's begin.

On the main site we see the key benefits of the service + D & D zone for a quick start:



It is very easy to register on the website - just enter your username and password (optional - email):



You can also log in using Twitter or Facebook.

Immediately after registration or authorization, you enter the Dashboard.



As we can see, the site is user friendly and quite web-danceable. When you click on the bright orange Upload button, you can hook one or more files from the screw and upload them to the site. If you upload a group of files, they automatically suggest creating a separate Gallery for this group.

With every file uploaded you can do a lot of actions. For example, you can set the rights to view files - Private or Public. A short link like http://min.us/mbmd5BijWG is automatically generated for each file, files can be discussed (the Disqus engine), and the iframe can be embedded anywhere.

There are context-dependent functions. For example, by installing an extension for Chrome or installing an application, you can quickly take screenshots and easily share them. Additionally, all images are automatically generated in the Gallery, which can also be shuffled and embedded into an external site.

And, for example, for pdf-files, a preview of the first page is automatically generated, which helps to easily navigate documents. And you can listen to audio files with the built-in audio player on html5.

Of the social chips, I would like to point out the opportunity to follow other users . There is a separate viewing feed - Feed , in which you can view files, zasarelnye friends. Files can be viewed and downloaded (unfortunately, it is not possible to add a file to your account without downloading).

Also, files can be conveniently fumbled through a post on Facebook and Twitter . You can even specify in the Settings autopost for these social networks.

While one big minus is noticeable: the service does not synchronize files from the hard disk to the cloud . Those. files can be fully worked through the website only. And it seems that this is not a drawback, but a positioning of the project. This is what distinguishes the Minus from the Dropbox for the worse in practical application.

Although the rest of the service - just a fairy tale. It is quite a useful project, I think I will actively test it.

I wish Minus.com a long life and big heaps of money. Maybe then the files from the screw will synchronize :)

PS The project has an API, so (perhaps) it will turn out to write software to synchronize local files with the cloud.

PPS In Minus.com, you can easily add files and links from Chrome through the Cloud Save extension. Recommend!
