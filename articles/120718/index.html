Flash games for dummies



The first version of the StencylWorks framework has been released - it is a fairly advanced tool for creating two-dimensional flash games on a Mac or Windows computer (iOS support will be available soon). The program can be used without knowledge of programming languages ‚Äã‚Äã(design mode), and for coders there is a separate powerful interface (code mode) through the ActionScript 3 API with the support of any third-party editor.

Creating a game, you design the game space and levels in the graphical interface, add game characters, including you can choose the appearance and behavior model from the finished catalog. Then just drag them to the playing field. Set the properties of each object in the menu. How to make a beautiful character kill, a spectacular visual effect, optimize game performance, add a hit parade of players, etc. - All these topics are discussed on the forums and in Stencylpedia.

Preview of the game is available at any stage.



The finished game can be published on any website or simply exported to SWF and do anything with it, even uploaded to specialized portals like Kongregate , which are paid to game authors.

The authors of this project have chosen an interesting business model: the framework itself is distributed free of charge, but they are going to earn money by selling virtual goods to game authors (a character store and other goods are currently in development), as well as by selling a subscription for developers, which will give additional benefits .
