Installation from SVN



How to update HabraFx is not a standard way, but through SVN?

If you suddenly need just such an opportunity (for special geeks), then you will need a convenient SVN-client (for example, standard, console) and the following guide.

  • Find the home directory of your Firefox profile, go to the extensions directory (on Linux this directory will be in ~ / .mozilla / firefox /% profile% / extensions)
  • Create a file named "{0c8ce5d0-4db6-11dd-ae16-0800200c9a66}" (without quotes). Write in it the name of the directory where you plan to store the extension (it is important to indicate the absolute path to this directory).
  • Go to the directory you specified and make svn co apo.web-zine.org/habrafx .


Everything, you can work with the version from SVN, updating it in exactly the same way.
