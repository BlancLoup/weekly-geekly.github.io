Equipment


In a beautiful small box the following components are hidden:

- Battery (Li-Ion, 3.7V, 1100mAh);

Turn on and work


The communicator is turned on with the button on the upper end and goes into a fully operational state in approximately 30 seconds. Immediately after loading, a desktop appears in front of us, on which non-standard WM applications are installed. In particular, SmartToday plug-in is immediately apparent - three tabs on which you can place any applications or settings.



With a more detailed acquaintance, you notice the inscription " Spb Home " in the lower right corner. What is it? That's right - SPB Mobile Shell 3 ;)



The one that I used for a long time on my communicator. In this case, this is the latest version - with a 3D carousel and sliding screens, on which there is almost complete freedom - if you want weather, lights, contacts, shortcuts, profiles, a clock or a player - place it all.



There were no brakes in this program even on my phone (from 200 MHz), what can we say about this configuration. This program seems to be coming from the considerations that Windows Mobile 6.1 itself implies the use of a stylus, and thanks to this software, you can only use your fingers. This shell alone is worth a lot - perhaps most users will need nothing more than it)



A little later, it turns out that there are a few more solutions from SPB Software - in particular, the finger-oriented SPB Keyboard (I'm really a Resco fan, but I was able to type without addiction), as well as a couple of online toys and an SPB TV demo software for view central channels on the communicator screen through the network. It is a pity that she is just a demo, so you can ignore it.
