Welcome to iOS-meetup SuperJob

November 30, we invite iOS developers to meetup at the Superjob office.

He will talk about how SuperJob accelerated the development of application screens, dividing them into independent blocks, how they used the IGListKit library, how they made it even better, and how they managed to speed up development and unify code style by implementing code generation.
