2 million passwords of social network users stolen

The news about password leakage has become commonplace, which is obviously a big problem for the future. Employees of Trustwave, an information security company, discovered a Dutch server where hackers stored about 2 million stolen passwords to access services such as Google, Facebook, VKontakte and Odnoklassniki. The server was discovered on November 24th.



About 326 thousand logins and passwords belong to Facebook users, about 60 thousand - Google, over 59 thousand - Yahoo and about 22 thousand - Twitter. Data for access to FTP accounts, mailboxes, etc. were also stolen.
Data was stolen using Pony keylogger.
