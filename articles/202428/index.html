Published program Cloud OS Summit. See the November 27 online broadcast of 30 reports

The final program , which consists of a plenary report and 5 parallel tracks of reports, is published on the Cloud OS * Summit conference website. In total, conference participants and online viewers will be offered more than 30 presentations on a variety of topics on Microsoft cloud products.



All reports of the conference are divided into 4 main sections:

  • Modern data center
  • Personal devices in the enterprise
  • Cloud Application Development Platform
  • Cloud infrastructure as a service - application scenarios

Within these sections you will be able to hear reports from experts from Microsoft and our partners in the entire spectrum of the company's cloud products: from Windows Server and System Center to Windows Azure, SQL Server and Visual Studio.

Conference reports will be made by experts from Microsoft as well as specially invited representatives of our partners from companies: Vision Solutions, IFD Capital, Commvault, DEPO Computers, Verter, Brightbox, VIAcode, Kaspersky Lab, and NetroxSC.

Experiencing a huge demand for attending the conference, we have already received three times more applications for participation than we could provide. In this regard, a month before the conference, we had to close the registration of new participants. For everyone who is not able to visit us personally, we want to announce the good news: on November 27, you can see all the reports without exception online on our website and on the websites of our information sponsors.

About Cloud OS Summit


Cloud OS Summit is the largest cloud conference of Microsoft in Russia. The conference program will present the latest technologies designed to build modern data centers, integrate private and public clouds, involve all mobile devices in the company's work, efficiently manage data and their analysis in the enterprise and create modern business applications using the power of a public cloud. .

By participating in the event, you will learn all about the newest versions of Microsoft products - Windows Server 2012 R 2 , System Center 2012 R 2 , SQL Server 2014 , Visual Studio 2013, and Windows Azure services. Together with experts from Microsoft and its partners, you will discuss scenarios for applying the latest products to solve the problems of your business.

Within the framework of the conference, a plenary report will be presented, which will open the event by covering all the main topics and more than twenty reports divided into sections:

  • Modern data center - scenarios for building private clouds and automating, monitoring and managing applications, extensions to the public cloud, hybrid scripts;
  • Personal devices in the enterprise - the involvement of mobile devices in the work of the enterprise, the management of personal devices and the issue of security, access to the desktop from anywhere at any time;
  • Cloud platform for developing applications - building scripts for modern applications using the latest Windows Azure cloud services and Visual Studio tools, integrating cloud resources and applications with resources in the enterprise;
  • Cloud infrastructure as a service - application scenarios - scripts for building applications based on Windows Azure public cloud infrastructure services, developing and testing in the cloud, working with data, analytics and Big Data in the cloud, the latest SQL Server 2014 application scenarios.
