The official release of Joomla 2.5 supports MS SQL Server and PostgreSQL



Despite the fact that most Joomla-based sites still work on Joomla 1.5, developers still try to make a difference. Recently, a new version of this CMS, version 2.5, has been released. The development team has introduced support for new Joomla database management systems, including Microsoft SQL Server and PostgreSQL, in the new version. Previously, only MySQL was supported.

Among other new products, Joomla finally has the opportunity to update this CMS automatically, by pressing just a few confirmation buttons. It is worth noting that in Wordpress this opportunity has appeared for quite some time. So it will now be easier for newbies to work with Joomla, and the update process, which is quite difficult for a beginner, no longer requires special knowledge. Yes, and for a professional such an opportunity is a big plus, since the time spent on updating can be significantly reduced.

The developers themselves are most proud of the support of different DBMS, considering that this opportunity will make their product even more popular. Inclusion of such support in the new version of the product is a solution caused by multiple user complaints. For example, many complained that for certain reasons they could not work with MySQL, but they wanted to get to know Joomla better. In general, the developers met the users.



The new version now has an integrated search, it was decided to make this function basic (previously, the search was a separate option). The search function, so to speak, stepped far forward in Joomla, corresponding to some commercial solutions.

Currently, on the basis of this CMS, there are about 2 million sites in the world, which is very, very good.
