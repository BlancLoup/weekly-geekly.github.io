Sotsdem user without departing from the checkout

... using Javascript.



The Mike on Ads blog post tells how simple Javascript helps to find out the user's sex of a page.

In short about the method:

- the script analyzes the frequency of user visits to sites from his history;
- from some source the script gets data about social mode of these sites;
- combining the received information together, we can conclude about which social layer the user belongs to and set the corresponding cookie for him;

The following is a practical application:

- Target this user advertising by his social characteristics.

What does all this mean?

1. Is it really not necessary to buy the Mamba base anymore, as does the Runner?
UPD:
2. In addition to gender, in the same way it is possible to find out the identity of a user to a particular social group.

PS The script works in FF3, Opera 9.5, Safari 3 on a Mac and on Windows, as well as in IE6 on Windows.
