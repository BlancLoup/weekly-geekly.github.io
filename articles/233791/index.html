Run Windows 8 on your Intel Galileo


Dear owners of the Intel Galileo microcomputer! We inform you that now you can install and run Windows 8 on it. As promised , absolutely free. For Windows to work, you need to upgrade your Galileo firmware to version 1.0.2. A detailed description of the installation process can be found here . Please note that at the moment Windows can only be run on the first version of Galileo (Intel Galileo Gen 1).
