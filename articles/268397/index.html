How VMware DRS works

Today we will talk about the profile topic - load balancing using VMware DRS (Distributed Resource Scheduler). To accomplish this task, he uses the available resources of the virtual infrastructure .


/ photo Arthur Caranta CC

This tool allows you to deploy new capacities or migrate existing ones without suspending your product or service. Its task is to automatically move virtual machines around the cluster and equalize the load on each of them.

Available to configure the rules of balancing and resource allocation between virtual machines. The application of certain rules or the provision of recommendations for the transfer in manual mode determines the monitoring.

This tool determines the standard deviation of the host load based on CPU and memory usage. Evaluation is made with a certain time interval.

How it works for us


We believe that IaaS should be accessible even for those who have not experienced IT. To do this, we optimize service services and implement special tools , which we are talking about here.

If we talk about VMware DRS, then we use it to work with our entire IT infrastructure. In addition, we use the technology of VMware High Availability , which allows you to restart virtual servers automatically. This combination allows you to achieve a high level of fault tolerance.
