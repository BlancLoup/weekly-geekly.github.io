Economical smartphone with Retina-screen type Mirasol



At the SID Display Week 2013 event held in Vancouver, Engadget journalists noticed a curious Qualcomm device with a Retina display made using Mirasol technology.

The device prototype shown has a screen resolution of 2560 x 1440 pixels, which corresponds to 577 ppi (for comparison: the iPhone 5 has the same figure of 326 ppi; Sharp recently showed IGZO displays, which Apple is going to install on new iPhones, 498 ppi).
