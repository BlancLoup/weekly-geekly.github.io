BitComet 0.84

A new version of one of the most functional BitTorrent clients has appeared. BitComet 0.84 can set speed limits, provides detailed statistics for downloaded files, allows you to preview video, supports trackerless downloads and can turn off your computer when downloading is completed.

Download BitComet 0.84: BitComet_0.84_setup.exe (5.30Mb)

The latest version carries mainly enhancements to the interface of the program: a more visual display of the downloaded parts of the file, a more precise definition of the codec for downloaded video files, monitoring of the clipboard, as well as fixes for several errors. Among the advantages of BitComet, we can mention the availability of the interface in several dozens of languages, including Russian, and the ability to download files using the good old HTTP and FTP protocols.
