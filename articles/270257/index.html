Mozilla Firefox 42 Web browser released

The Mozilla Foundation has released a new version of the Firefox 42 web browser, fixing vulnerabilities in it and improving some of its features. Several fixed vulnerabilities were present in the Network Security Services (NSS) library set, which are used by the web browser to encrypt traffic using the SSL and TLS protocols ( Mozilla Foundation Security Advisory 2015-133 ). Vulnerabilities allowed a malicious website to compromise a secure https connection and remotely inject malicious content into a running web browser process. The NSS package itself has been updated to versions 3.19.2.1, 3.19.4, 3.20.1. ESR - Firefox ESR 38 38.4 was also upgraded.
