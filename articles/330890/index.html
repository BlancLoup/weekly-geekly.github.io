GeekUniversity opens a set of students at the Faculty of Java-development



In our online university for programmers opened a new faculty. Now at GeekUniversity, students will be able to master Java-development at the Middle-level and is guaranteed to start a career immediately after training.

GeekUniversity is a joint educational project of the Mail.Ru Group and the IT-portal GeekBrains. The curriculum and special courses for the faculty are developed by Avito, Alfa Bank, MTS, Tinkoff, DeliveryClub.

During training, students create 2 independent and 2 team projects: network chat, game for mobile devices on Android, cloud storage, e-commerce site. Progress will be supervised by a mentor. After training graduates are able to:

  • program in Java and Java Enterprise Edition;
  • create web applications using the Spring Framework;
  • make up HTML, CSS and work with Bootstrap;
  • use the principles of OOP and design patterns;
  • work with GIT;
  • design architecture;
  • use design patterns and SOLID principles;
  • follow code style.




A year of study at GeekUniversity is a year of real-world Java development experience for resumes and 4 projects for a portfolio. Graduates are guaranteed further employment in partner companies. Training lasts a year, classes take place 3-4 times a week in the evening.

GeekUniversity teaches students on the basis of state license No. 038188. Graduates receive a standard certificate and a personalized electronic certificate from the GeekBrains portal and Mail.Ru Group.

We offer all applicants to get tested on Java theory. If the test is not passed, we recommend preparatory courses, where you will learn the basic structures of the language and the basics of OOP. Education at the faculty is suitable for those who have a basic knowledge of programming.
