Cross-cloud architecture and VMware Cloud Foundation, a single platform for managing SDDC

This week at VMworld 2016, the development of the company's own hybrid cloud strategy based on the innovative cross-cloud architecture of VMware Cross-Cloud Architecture was announced; This architecture enables the creation of a unified environment in which the entire cycle of application management will be carried out, physically located in different cloud environments and on different devices. To implement this strategy, the company proposed a number of software components that increase cloud model utilization efficiency: VMware Cloud Foundation (SDDC management platform for software-defined data centers), Technology Preview of Cross-Cloud Services (preview of management services for private and public clouds from various suppliers), VMware vCloud Availability (a new line of backup tools) and a new version of VMware vCloud Air Hybrid Cloud Manager (allows instant migration of applications from vSphere to vCloud Air). Today we will talk about the VMware Cloud Foundation.



Unified platform for software-defined data centers


The VMware Cloud Foundation provides the next generation hyperconvergent structure for building private clouds that for the first time combine VMware's scalable hyperconvergent software (for VMware vSphere and VMware Virtual SAN) with the world's leading virtualization platform, VMware NSX.

VMware SDDC Manager, a core component of VMware Cloud Foundation, helps customers and service providers automate the deployment and management of VMware cloud software. SDDC Manager helps you create and maintain the entire VMware cloud software stack, freeing cloud administrators from the complex and time-consuming task of installing, configuring, managing, and updating cloud infrastructure, which allows you to build a complete cloud in a matter of hours. IT departments are actively seeking solutions to simplify the infrastructure by removing barriers that today significantly impede their ability to respond quickly to business needs.
