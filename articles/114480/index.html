Into the cloud with Microsoft and Softline!



On March 4, 2011, Softline together with the Russian representative office of Microsoft will hold a conference for startups Microsoft BizSpark Camp on the Microsoft Windows Azure cloud platform in Moscow . At the event, we will discuss the benefits of a business in the cloud, as well as examine in detail the key components of Windows Azure and their use scenarios for creating applications in the SaaS model (software as a service).

During and after the conference you will have the opportunity to perform laboratory work. Conference participants will be provided with 30-day access to Windows Azure and a book on the Microsoft cloud platform. If you have any technical issues related to the implementation of your solution on the Windows Azure platform, questions about the platform itself, the solution architecture, etc., you can receive individual expert advice after the workshop. Start-up companies that want (or plan) to create solutions on the Microsoft cloud platform are invited.

Participation is free, registration is required.
