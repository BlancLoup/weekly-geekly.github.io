Mozilla revealed the mobile operating system Boot2Gecko



Last year's mention of the Mozilla Foundation that the fund intends to make efforts to create an operating system for smartphones and tablets was justified. Mozilla intends to show a working demo system at the upcoming Mobile World Congress in February, and the final product stage will be reached at the end of 2012.

Mozilla's mobile OS ‚Äî the user interface and application stack ‚Äî is completely built on the use of web technologies, and Gecko is expected to be used as the engine, as in FireFox.

In total, the creators distinguish three layers of the system: the lower Gonk is the Linux kernel, the level of hardware abstraction, the telephone stack, and other low-level system components. The middle layer is the Gecko rendering engine, supplemented with a number of APIs. And finally, the top layer of Gaia UI - the user interface - is completely implemented in HTML and JavaScript.
