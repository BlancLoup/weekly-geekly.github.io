Firefox Chat Encryption Extension

For Firefox, an extension has been developed that supports the encryption protocol OTR (Off-the-Record). This is a protocol created specifically for instant messaging networks. The library is distributed under the GNU LGPL license, and is already supported through plugins by many IM clients, including Miranda, Pidgin and Trillian. At least two IM clients have built-in support for OTR: these are Adium (Mac OS X) and Climm (Unix).

Now, chat encryption will also appear in the browser, so using the new extension you can safely talk, for example, in the built-in Facebook chat, if both people have Firefox installed with OTR.

The extension for Firefox has not yet been completed, there is no UI, it only works on Facebook. The author will be glad to any advice.

Source code: git: //gitorious.org/fireotr/fireotr.git.
