ANN: MT Browser Update Switcher v1.1 - for using multiple browsers

After my first announcement of the program, quite a long time passed. Involving the Habralyud community in testing has significantly improved this small program.

For those who have not read about this program, I will repeat a simple and clear scheme of its work:


I am glad to present you version 1.1 . Almost nothing has appeared, but the quality of the existing functionality has been significantly improved. In particular, the program now works much better under Windows XP. A simple MSI installer was also added (the portable version also remained).

For more information about the new version here: mito-team.com/ru/news/mtbs/v1-1-release

The program was made "for myself", and almost everything I needed was already in it. Therefore, its development will depend on the interest in it. It will be necessary and interesting to someone - we will slowly finish it.

Therefore, I would like to publish ideas for the further development of the program:
  • Ability to add your own browsers to the list of available (for example, with specific command line parameters, or different versions of the same browser living in different folders)
  • A systray menu that allows you to launch any of the browsers added / found or open an arbitrary URL in any of the browsers found / added
  • Ability to create rules that open the URL in multiple browsers. This can be very useful for those who need to test any page in different browsers.

If you found something interesting, in demand, or have your own ideas, wishes - write in the comments. I will read it and take it into account. Thank you all in advance!
