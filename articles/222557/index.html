No binding to the vendor! OpenShift Cartridge Standard support will appear on InfoboxCloud Jelastic platform

Last November, Jelastic and RedHat collaborated in the standardization of plug-in cartridges for platforms as a service (PaaS) were announced. As a result of this interaction, support for OpenShift cartridges on the InfoboxCloud Jelastic platform will appear . This will allow technology and service developers to provide solutions for various cloud platforms without rewriting and gain access to the widest number of users. Reducing the time it takes companies to adapt and package software for various cloud platforms will make it easier and faster to deliver technology to users and focus on solving real-world problems.

Thanks to standardization on the Jelastic platform , new technologies will appear earlier. Among them in the new release are expected:
  • Neo4j Server: scalable graph database
  • Redis: key-value storage
  • Apache Cassandra: Scalable High Availability Database
  • Jetty (8 and 9): web server with embedded HTTP server and servlet container
  • Python (2.7, 3.3, 3.4)
  • JBoss 7: RedHat Java Application Server
  • Varnish: cache


There are plans to add more than 200 new applications to Jelastic in the near future (several months).


Cartridge support will be added to Jelastic 2 - the next release of the Jelastic platform .
