Jailbreak g1



G1 lasted only about two weeks, and the developers are already paying off. G1 jabbreak! In short, what is a jailbreak (jailbreak) - Jailbreak - Change the state of the system part of the T-Mobile G1 file system so that it can be written to, and not just read. Without jailbreak, recording can be made only in the user part of the phone. Description of the process, see under the cut.

- Turn on WiFi in G1 (Settings> Wireless Controls> Wi-Fi / Settings> Wireless Control> Wi-Fi). Make sure you are connected to the network.
- Launch PTerminal (if not, download from the Android Market).
- Type cd system and press Enter.
- Type cd bin and press Enter.
- Type telnetd and press Enter. You have now enabled Telnet on your G1.
- Type netstat and press Enter. Now you see your IP address G1. You can also find it in Settings> Wireless Management> Wi-Fi> and click on the network to which you are connected. It will show your IP.
- Enable Terminal on Mac OS X, or Command Prompt in Windows. Enter telnet [your ip-address] and press Enter. So, if your IP was 192.168.1.101, you need to enter telnet 192.168.1.101 and press Enter.
- You are logged in as root. Type mount -oremount, rw / dev / block / mtdblock3 / system and press Enter.

All this opens up great opportunities. A list of directories that will open thanks to a jailbreak.
