Windows Camp // Labs - how to learn how to develop universal Windows 10 applications in one day

So long ago we had Windows Camp , where we were told about the new Windows 10 and universal applications (if you haven‚Äôt yet seen it, you can see all the reports online ). However, as it is known, it is better to see once than to hear 7 times, and it is better to touch once than to see 7 times .



We invite you together with us to learn how to develop applications on the Windows 10 platform during special lab work - Windows Camp // Labs . The event will be held on October 28 in Moscow (prior registration is required) at Digital October, participants not from Moscow can watch the broadcast ( on the event website ) and do the work at home. To participate (both at the event and online), you will need a laptop with installed software (details below).

The topics of laboratory works with a brief description are presented on the event website (section ‚Äúprogram‚Äù). They will cover a wide range of topics, ranging from the most basic (introduction to universal applications, demonstrating the work of universal applications on Raspberry Pi 2 and Windows 10 Mobile), and ending with personalized voice communication with applications and Cortana, working with pen input, using Microsoft cloud Azure and publishing in the store.

To participate in the event (both on the site and at home), you need a computer with Windows 10 and Visual Studio 2015 installed (for example, Community Edition ). It is also recommended (but not necessary) to install an emulator for Windows 10 Mobile, or to have a smartphone with Windows 10.

At the site, most laboratory work will be conducted with an instructor who will step by step demonstrate how the work is done on the screen. Those who want to do the work at their own pace will be able to do it - we have provided a separate room, created clear instructions for completing assignments, tutors will be available for consultation, ready to answer individual questions on laboratory work at any time. Those who do the work themselves, instructions and source code examples will be available online in a special repository on GitHub.

There are not so many places at the event, so hurry up to register ! This is a great opportunity to study the development on the new Windows 10 platform in just one day!
