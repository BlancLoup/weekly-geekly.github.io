Finally, a few tricks to use Jet Screenshot


  1. To change the original three colors, double-click on the color box. In the process of drawing, the color of the marker can be changed by holding Ctrl or Alt on the keyboard.
  2. You can configure the program to automatically save screenshots to the selected folder with one click. To do this, go to the settings on the Share Options tab, select the Save to File item, tick Use This Action as Default, enter the folder to save in the Save automatically to this folder field and click Save.
    Now when you click on Fast Screen Share (by default, Ctrl + PrintScr), the screenshots are stored in the selected folder without asking any questions.
  3. To open a graphic file in Jet Screenshot, select it in the explorer, press Ctrl + C, and then Copy From Clipboard or simply Ctrl + Shift + C. Supported formats are * .BMP, * .JPG, * .JPEG, * .PNG, * .GIF, * .TIFF, * .PCX, * .ICO, * .CUR, * .WMF, * .EMF, * .TGA, * .PXM. * .JP2, * .J2K.
