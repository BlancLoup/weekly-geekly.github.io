Mac OS 10.8 platform support in the new version of Oracle VM VirtualBox

A new version of Oracle VM VirtualBox has been released, in which, in addition to a number of other improvements, official support for Mac OS X 10.8 Mountain Lion has been announced.

Official press release:
www.oracle.com/us/corporate/press/1842885

Download Oracle VM VirtualBox at the link below:
www.oracle.com/technetwork/server-storage/virtualbox/downloads/index.html?ssSourceSiteId=ocomen

UPDATE: in the comments quite rightly noted that the press release refers to the Mac OS X platform, i.e. it only means that you can run the Virtial Box on Mac OS X 10.8. Mac OS X Server support was released in Virtual Box 3.2. I apologize to those who were reassured in vain.
