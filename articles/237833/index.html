Sony introduced competitor Google Glasses



The Japanese corporation Sony has recently shown to the public its own virtual reality helmet Project Morpheus. The device turned out to be quite interesting, the feedback on the device, in general, is positive.

Now Sony has introduced its own video glasses , so far a version for developers. Points were called SmartEyeglass, this device has a number of differences from Google Glass.

First of all, there is no mini-projector, which closes, albeit slightly, the user's field of view. In SmartEyeglass, both lenses are used as a display, 100% transparent in the normal state. When using glasses as a wearable gadget, the image transparency is 15%.

The resolution of the ‚Äúdisplay‚Äù is 419 * 138 pixels, the image is color, but only 8 bits, on a monochrome green background.

The developers also installed a 3MP camera, with which you can get VGA images, there are also sensors: an accelerometer, a gyroscope, a compass, an illumination detector. Glasses are built in microphones, plus communication modules 802.11n WiFi, Bluetooth 3.0.

SmartEyeglass can be connected to any mobile device based on Android 4.1 or later.

The control is unusual, compared to the same Google Glasses: a wired controller with a built-in battery, speakers, buttons and a sensor is used.

As for the ecosystem of applications, for developers presented an emulator software environment SmartEyeglass, with the ability to test their applications.
