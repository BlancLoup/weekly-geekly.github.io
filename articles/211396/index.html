Firefox 27 release

Firefox for desktops has become available for download. What's new in this version:


  • Implemented support for simultaneous launch of several services using the Social API , which allows you to receive notifications, communicate and perform other actions simultaneously on several integrated social networks ;
  • Added support for TLS 1.1 and TLS 1.2 ;
  • Added support for SPDY 3.1 protocol ;
  • Support for free editing HTML blocks has been added to the Inspector without the need to install a special add-on ( video ).
    Provided a preview of the background images and colors (next to the digital values ‚Äã‚Äãdisplayed a circle of the corresponding color). Added setting to determine the default color output format (rgb, hexadecimal, names);
  • In the style editor, debugger, inspector and Scratchpad , the Codemirror HTML code editor is used, allowing you to fully edit HTML blocks;
  • The ability to reset style sheet parameters using the all: unset element;
  • Added the ability to deobfuse javascript in the "Debugger";
  • Support for scrolling a set of fields that do not fit into the selected area (overflow in the fieldset tag);
  • For the iframe tag, the allow-popups sandbox mode directive is introduced that controls the ability to display pop-up windows;
  • Eliminated the prefix of the CSS cursor properties "-moz-grab" and "-moz-grabbing";
  • Support for ECMAScript 6 generators has been added to the SpiderMonkey engine;
  • Implemented support for the math function Math.hypot () in ECMAScript 6;
  • Dashed line support added to Canvas;
  • On Linux , the ability to render content using Azure / Skia ;
