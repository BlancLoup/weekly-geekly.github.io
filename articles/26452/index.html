For rapidshare.com, filefactory.com and Firefox users

Automatically check links for download.

We put the extension Greasemonkey
On it script Rapidshare Links Checker

After that, visiting the page containing the rapidshare.com * and filefactory.com * links in the background will check these links for download.


Even for example, if the links on the page are presented in plain text, the script will make them clickable.
It is also possible to check local text files with links, which is much more convenient than Rapidshar checker as there is no limit on the number of links. To do this, simply add file: /// * to the script
Who uses filefactory.com can remove the check files rapidshare.com and vice versa.
