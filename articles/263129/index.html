Microsoft has released an unscheduled security update for Windows MS15-078

Microsoft has released update MS15-078 , which closes the critical Remote Code Execution (RCE) vulnerability in all versions of Windows (CVE-2015-2426). For the second time this month, the Windows System Library Adobe Type Manager Library (Atmfd.dll) will be updated. Attackers can remotely execute code in the system using a specially formed OpenType font file, which can be placed on a malicious website.



Starting this month, the company has stopped supporting Windows Server 2003, so MS15-078 is addressed for Windows Vista + systems only. We recommend that users install this update as soon as possible.
