Installing CentOS 6.3 + LNMP on VirtualBox (video cheat sheet)


If you need to run a web server locally in a Windows environment, you can do this on a VirtualBox virtual machine by installing Linux and all the necessary components. But many are afraid of the complexity of such an installation. In fact, all this is done quite quickly and simply.

As a result, you will have a local VPS server with root access through putty and a working LNMP configuration (Nginx + MySql + PHP).

Who is this video for?


For those who have been communicating with Linux for a long time, setting up such a server will most likely not cause any difficulties. But if a person has never before dealt with virtual machines, or with the Linux command line, there may be some difficulties at different stages of installation.

You can, of course, all the answers and instructions found in Google, but for this you need to spend more than one hour of time. I tried to collect all that is needed in this short video tutorial.

Content


  • Installing CentOS 6.3 on VirtualBox;
  • redirection of 22 and 80 ports, for connecting to a virtual machine from Windows;
  • connecting to this virtual machine via ssh via putty;
  • installing LNMP (Nginx + MySql + PHP) using the same auto-installer;
  • firewall shutdown;
  • installation of Guest OS additions.
