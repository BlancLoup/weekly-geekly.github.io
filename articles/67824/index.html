Scrum'd - Agile style project management system.

Not so long ago, I discovered for myself a project management system based on the Scrum methodology. Descriptions of Scrum'd on Habr√© I did not find, therefore, my short description under the cut.

Scrum'd philosophy


Business applications are only useful when they do their work faster and more efficiently. Complex tools, attempts to replace human communication or change the essence of the processes, only interfere. Such applications require large resources and remain unclaimed within the organization. Scrum'd develops efficient business applications that allow you to improve the performance of your team without significant changes in your existing business.

Feature Description


The system provides 30 days of free, test use. You create a project, then add users to it, and distribute roles to them (ScrumMaster, ProductOwner, ScrumTeam). Then in the project you are invited to add the scheduled releases, and attach a sprint to each release. Next, create tasks for users and track the performance of each task. Burndown chart provides real-time display of the status of current sprints Also in the project there are sections WiKi and Attachments.
