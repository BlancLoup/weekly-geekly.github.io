What would you like to know about the NooLite lighting control system?

I think the NooLite system (it has already been written about in Habr√© ) is a very cool project: it provides previously unavailable features, has a low cost, it is very easy to connect and use. These switches are at my home and it is very convenient. But, in my opinion, few people (mostly geeks) know about the NooLite system, although it is no more difficult to connect / use it than a conventional wired switch and almost anyone can handle it.

Information about the system available on the Internet is monotonous and not very detailed: the connection process is described everywhere, but little information is available about impressions of actual use, features of work, etc. For example, no one writes whether it is comfortable to use touch switches (without tactile feedback) instead of the usual ones.

We make a site dedicated to the system NooLite. The goal is to compile and structure information about it so that anyone who wants to automate the control of lighting can learn everything in one place. Therefore, it is important for us to understand what it would be interesting for you to learn about this.

If you are thinking of putting such switches on yourself, but you lack some information about the features of the system or about using it in real conditions, write your question in the comments or in a personal message to me. We will try to answer all questions (and if necessary, contact the developers) and publish the answers here and on our website.

UPD. After reading the comments, I want to make a small addition: I have no relation to the system developers, I just like it very much.
