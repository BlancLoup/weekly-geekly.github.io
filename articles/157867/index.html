Synology¬Æ Announces DiskStation DS713 + Release

Synology Inc. introduced DiskStation DS713 +, a scalable 2-port NAS server that provides small and medium business users with highly efficient storage solutions.

The key difference of the DS713 + from the previous model is the transition to a dual-core processor with a frequency of 2.13 GHz. With Ethernet port aggregation activated, the drive reads data at a speed of 202.3 MB / s and writes data at a speed of 135.84 MB / s under a RAID 1 configuration in a Windows environment. In addition, DS713 + is optimized for work with large-scale video surveillance systems and supports simultaneous recording of 20 high-resolution channels.

The network drive is equipped with two USB 3.0 ports, which provide data transfer speed 10 times higher than USB 2.0. To ensure uninterrupted availability of stored data, the DS713 + is equipped with a hot-swap front-mounted HDD and two Gigabit Ethernet ports, which together increase the resiliency of the system.

The DS713 + has full iSCSI support and virtualization, enabling the user to build a comprehensive storage solution based on all major server platforms, including VMware vSphere 5 VAAI, Citrix XenServer and Microsoft Windows Server 2012 with Hyper-V. The storage server is seamlessly integrated into the business environment with support for LDAP, Windows AD and ACL.

Key Features of Synology DS713 +:

  • 2.13 GHz dual-core processor
  • 1 GB DDRIII RAM
  • 2 Gigabit LAN ports with Link Aggregation support
  • Two USB 3.0 ports, 1 USB 2.0 port, 1 eSATA port
  • Support 4TB HDD
  • Hot-swappable hard drive support
  • VMware VAAI, Citrix, Microsoft Hyper-V support
  • Scalable to 7 drives with Synology DX513
  • Read speed 202.3 MB / s, write speed 135.84 MB / s
  • Power consumption 30.72 W (work), 17.88 W (sleep mode)
