BitTorrent 6.0 Final

The final release of the official client for the most popular Peer 2 Peer BitTorrent network. The sixth version of the program is a merger of the corporate Bittorrent client with the recently acquired uTorrent . From the original BitTorrent program gained experience in the development of network protocols, and from uTorrent compact code and a wonderful interface. As a result, the program has become much easier, it has acquired the ability to search for local peers, several simultaneous downloads, speed limiting, fast resuming downloads, RSS-downloader, etc. In fact, the same uTorrent.

BitTorrent P2P network in the shortest possible time became the most popular in the world, having shifted KaZaA from the first place and having pressed eDonkey2000. BitTorrent is fundamentally different from its counterparts - in order to download something, you need to have a special .torrent file, collections of which can be found on the web. The network works on a non-tricky principle: in parallel to how you take a file from someone, you give it to other users at the same time. Many large developers (as a rule, open source products, such as Linux distributions) noticed BitTorrent and use this P2P network among other distribution channels of their products.

Currently only for Windows.
