According to the official blog, the next version of one of the most popular scripts for organizing personal online blogs, WordPress 2.6 Tyner, has become available for download today.
It is named after jazz pianist McCoy Tyner ( McCoy Tyner ) and contains a large number of features that make WordPress a more powerful CMS: it is possible to track changes to any message or page (some equivalent version control system within WordPress), as well as several additional improvements announced in version 2.5.

The WordPress team has prepared a short video tour of version 2.6, lasting 3 minutes 29 seconds.


Because video, most likely, will not be displayed ... then look at the off blog or my site :).

List of new features:
  1. Track changes to posts or pages.
    Now you can track who, what and when changed in the message or on the page. You can track changes through a very simple interface, similar to Wikipedia or to a version control system.

    The difference between the two versions of the message.


    Thanks to this, you can now return to an older version of the message. This is especially true for blogs with multiple editors.
  2. Become a journalist! ..
    So WordPress creators have mysteriously called the possibility of creating posts on their blog, being on any site. What this means to the end is not very clear - you probably still need to put yourself a new version (I have not updated it yet). Declared the ability to quickly insert YouTube videos, being on the YouTube website itself, while WordPress engine will parse the page and pull the video insertion code into your blog post. The same applies to posting photos from Flickr ... etc.

    Well what can I say - we must try!
  3. Accelerate Gears. Turbo speed for blogging.
    Using Gears technology from Google and the corresponding browser extensions, all frequently used JavaScript and CSS files will be cached to the user's computer.
  4. Preview topics.
    Added the ability to preview themes without activating them on the blog. This will be very useful for developers of topics and for their debugging - users will not see most of the bugs :) - just fine :)
  5. In one line:
    • Added the ability to count the number of words.
    • Added captions under the images.
    • Mass management of plugins.
    • Completely reworked image management - simplifying insertion, positioning, resizing. All this is integrated with the WYSIWYG editor.
    • Technology Drag-and-drop in the Gallery.
    • Pop-up messages about new versions of plug-ins.
    • Customize avatars by default.
    • You can download media content in full window mode.
    • Remote publishing of materials via XML-RPC and APP is now not done by default via a secure connection, but you can simply turn it on in the blog settings screen.
    • Full kernel-level SSL support.
    • Now the interface does not slow down when there are thousands of pages or categories.
    • Added the ability to place the wp-config file and the wp-content directory in an arbitrary location.
    • You can switch between the flash loader and the classic one.
    • A large number of security improvements are used, including cookies and interaction with the database.
    • More stable and faster versions of TinyMCE, jQuery and jQuery UI are used.
    • Version 2.6 fixes approximately 194 errors.



Updating from version 2.5 to 2.6 according to the developers should be painless. No plugin updates required.

Everyone is recommended to upgrade, because The 2.5 development branch will not be supported anymore .
