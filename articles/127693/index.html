Guitar Pro 6 on Ubuntu 10.10 - solving installation problems

The DEB expansion inspired confidence and hope that everything will rise by itself. However, after successfully installing the package according to dpkg, a problem was immediately discovered: the sound in the program does not work - neither MIDI nor RSE. When starting the playback of the tablature, Guitar Pro swore at the missing soundbanks (soundbanks), when RSE was turned off (by pressing F2), no problems were reported, but there was still no sound.

Lay down the legs and be satisfied with the work of the program through Wine (with periodic departures and display only the English interface) It seemed to me unacceptable, and I found a solution to the problem that I would like to share with you.

In the sound settings ( Window menu -> Sound -> Audio settings ) were found:
  • one MIDI output (MIDI Through: 0) that does not respond to the Check button;
  • sound card driver PulseAudio output.
In the general settings (F12 or Window menu -> File -> Settings ) only one item was related to sound processing:
  • ‚ÄúAllow multithreaded RSE processing‚Äù - it is better to leave it turned on in order to avoid brakes when playing multi-track tablature.
When feeding soundbanks to the program via the Window menu -> File -> Open -> [SoundBanks.gpbank file selection], an alert occurred: "Guitar Pro requires a restart to start working with the new installed sound banks", but after restarting the sound did not appear, and the whole story repeated.

So, the essence of the problem is clear: the soundbanks are not loaded either during installation or with the standard means of the installed program. So you need to connect them manually.

Finding out where Guitar Pro is:
sudo find / -name GuitarPro6

Among other things, we find:
/opt/GuitarPro6

In this directory, we see the GPBankInstaller file, which turns out to be the SoundBanks.gpbank archive unpacker . Immediately use it:
sudo /opt/GuitarPro6/GPBankInstaller [ Soundbanks.gpbank] /opt/GuitarPro6/

For example, it happened to me like this:
sudo /opt/GuitarPro6/GPBankInstaller /home/smoren/gp6/Soundbanks.gpbank /opt/GuitarPro6/

After that, feel free to open Guitar Pro 6 and enjoy the sound that appears.

PS As for the MIDI sound, the question remains: when the RSE is disabled (it is he who uses sound banks), the sound is still missing. All the necessary drivers are in the Rhythmbox * .mid files are played.

Of course, the fact that the RSE is now (unlike the 5th version) is used not only when playing, but also when typing tabs, completely replacing MIDI. However, the lack of MIDI sound is alarming, so I'm trying to find a solution to this problem. If you have ideas, I will be glad to read them in the comments.
