Xdebug toolbar for debugging PHP in Opera

Instructions for setting up the Xdebug toolbar - for those who want to debug the code without fail in the Opera (as well as for those whom life has forced).



For debugging in the Opera, I found three ways:

  3. Manually add parameters to the URL.
