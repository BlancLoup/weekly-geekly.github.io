Gmail backup

Many users (including myself) have long since transferred from their usual e-mail clients to work with mail directly online. And some customers did not use in principle. This means that they do not have a local copy of the letters and documents. So, if somewhere in the distant nooks of consciousness these users have a fear for their emails on the Gmail server (well, you know, anything can happen, because nothing is perfect), then you can protect yourself and the data using the Gmail utility Backup.

This useful feature saves backup copies of mailbox contents to Gmail. She does it by IMAP. And saves data in EML format. The program works both in Windows OS, and under Linux and Mac OS X. In the first case, the user is offered a graphical interface. And for Linux and Mac OS X - command line utility. The program allows you to back up both letters and attachments.

One of the most significant advantages of Gmail Backup is the ability to download the archive back to the Gmail server. Under a different account. So if there is a need to ‚Äúmove‚Äù to another account, then this can be done as simply and painlessly as possible.
