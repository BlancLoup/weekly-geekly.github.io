Rare clone atari 2600

Hello dear habrayuzer! Today I want to write about a very rare atari 2600 gaming console, or rather about the Chinese atari 2600 clone with built-in games. There is practically no information about her. I know that it has about 128 built-in games. The console itself, in principle, well preserved but the original joysticks did not live to this time: (
Well, actually the photo of the console itself


There are 4 levers on the front panel:
1. Turning off the console
2. Game selection
3. Select the number of players
4. Reset the game
Yes, it looks very similar to the Atari 2600, but only the cartridges do not smell it :) it's just a decorative cap


On the top side panel there are 2 slots for joysticks



Switch for choosing the difficulty: NOVICE and EXPERT (there are two of them too)



Power supply connector


And an interesting switch, it is written that this is supposedly to change the channel

but actually it is a triple switch to select cells with games
