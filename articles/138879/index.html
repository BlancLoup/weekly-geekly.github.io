Concept UI for Windows 8

Designer Sputnik8 (Phyek) on the Verge forums submitted to the public a concept design for the desktop version of Windows 8 (all screenshots are clickable).



The screenshots show different versions of Explorer, IE browser (with converted Windows.com and Bing sites), Media Center / Player and Skype. The author emphasizes that he did not strive to strictly comply with the new style of Metro, in general, his work is rather a fantasy on what Windows 8 might look like.
