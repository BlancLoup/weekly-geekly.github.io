35 useful virtualization tools



Today, the word "virtualization" is widely known, and virtualization technologies are becoming one of the main components of the IT infrastructure of organizations. By the way, today our post came out, in which we shared our experience of creating a virtual infrastructure provider using the example of 1cloud .

We decided to continue working with this topic and compile a collection of tools that should be paid attention to everyone who is related to the virtual infrastructure. Further, only a small part of the existing tools, located in a random order, will be given.

Virtualization Management

  • Migrate: a tool for migrating to a virtual environment
  • OpenFiler: creating web-based file storage
  • StarWind Virtual SAN: Virtualized, hypervisor-based storage
  • vCenter Converter: allows you to convert a physical machine under Windows and Linux into a virtual
  • RV Tools: an application that uses the VI SDK to display information about virtual machines and ESX hosts
  • Citrix Essentials: Managing XenServer and Hyper-V Host Servers for Enterprise-Level Virtualization

  • Veeam Backup and Replication: Repairing Virtualized Applications and Data from vSphere and Hyper-V Platforms
  • vSphere Mobile Watchlist: allows you to monitor virtual infrastructure remotely using a smartphone
  • ESX Deployment Appliance: simplifies the deployment of ESX servers
  • Ganeti: Google's Virtualization Cluster Management System
  • oVirt: virtualization platform with a simple web interface
  • Packer: image building tool

  • Xen: virtual machine monitor or hypervisor with para-virtualization support for x86 architecture processors
  • CloudStack: allows you to automate the deployment, configuration and maintenance of virtual infrastructure
  • Vagrant: console utility for creating and configuring a virtual development environment
  • VirtualBox: an easy and reliable virtualization tool
  • FusionSphere: Huawei's Virtualization Platform

KVM

  • KVM: Linux Virtualization Solution for x86 Processors
  • Mist.io: A complete solution to help manage your hybrid infrastructure.
  • nbsvm: creating, working and interacting with VM using simple chain commands
  • Witsbits: a service that simplifies the deployment of virtual machines on physical servers
  • UVMM: Intuitively Manage Xen and KVM-Based Virtual Machines
  • libvirt: cross-platform library for managing virtualization

  • cloonix: configure virtual network settings
  • Archipel: push notifications using XMPP
  • kimchi: HTML5 based management tool for KVM
  • SolusVM: one of the most popular control panels for commercial use.
  • Virtual Machine Manager: provides an interface for managing virtual machines using libvirt
  • vmmaestro: a small shell script capable of starting and stopping KVM-based virtual machines
  • kvm-wrapper: a set of shell scripts that simplify the management of KVM-based virtual machines

Virtual Security

  • VMware vShield Edge: product for comprehensive data center perimeter protection
  • VMware Compliance Checker Tool: security compliance checks (FISMA, PCI / DSS, etc.)
  • Catbird Secure: Security Solution for Physical and Virtual Infrastructures
  • HyTrust: a means of protecting user connections in a virtual infrastructure
  • Trend Micro: a platform for protecting physical, virtual and cloud servers
