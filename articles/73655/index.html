Focusing Tools


Lectures on IT-disciplines (and not only) are often accompanied by the presentation of the presentation and demonstration of practical actions, for example, in any development environment.
There are free tools to improve the efficiency of information transfer. Also, these tools are indispensable when recording screencasts, allowing you to draw the attention of viewers to the necessary details.


Zoomoom




Mark Russinovich utility, part of the Sysinternals Suite.
Allows:
  • enlarge the screen area at the location of the mouse cursor (Ctrl + 1, for Windows Vista and higher, LiveZoom is available: Ctrl + 4)
  • draw ‚Äúby hand‚Äù (mouse click in zoom mode or Ctrl + 2), straight lines (Shift), rectangles (Ctrl-Shift), ellipses (Tab) and arrows (Shift + Ctrl)
  • clear screen - fill with white (w) or black (k) color
  • copy enlarged / drawn image to clipboard (Ctrl-C) or save to file (Ctrl-S)
  • print the text (t in the drawing mode), unfortunately, only in Latin
Attention! The utility is addictive :)

Keyjedi




The purpose of the utility is to display the pressed key combinations in a small window. In order to convey all the delights of, for example, the editor of Visual Studio - the most it. Work requires .NET Framework.
