Homemade computers

I doubted which blog to write this message in: Cyberpunk or here. Because the message on the one hand is definitely about the cyberpunks of modernity, but on the other about the construction of computer systems. And so, I decided to make the post here, because the habrakibernks are looking more at the future, and the sites have links to resources that are useful for any system analyst, such as uIP sources.

So to the point. Wandering around the Internet in search of information on the device of logic chips, I came across a couple of resources on what can be done from a set of the widely known (in the USA) 7400 series transistor logic.

1. 4bit processor running at 3.75mhz and capable of calculating even factorial, that is true :) www.vttoth.com/vicproc.htm

2. Computer assembled from ~ 200 logical chips and SDRAM. On which the web-server works, and with which you can communicate via telnet and start various programs, such as life or guest. magic-1.org

From communicating with the number 2. you can get a huge pleasure - this is really magic. There are much more emotions than when working with the latest achievement of supercomputer engineering. Mostly, probably because you feel you can do the same.
