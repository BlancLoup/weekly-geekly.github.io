OOP and Design Patterns in Python


The course ‚ÄúOOP and Design Patterns in Python‚Äù will help you step beyond the simple use of language syntax and move from writing simple concrete classes to professional application design within an object-oriented paradigm.

For whom

The course is designed for those who confidently write simple programs in Python and know the implementation of OOP paradigms in this language: classes, objects, attributes, methods.

Duration

6-8 hours a week for 5 weeks. Each week will be devoted to a separate topic.

Course content

  1. Testing and debugging programs

    • creating base classes
    • UML notation,
    • creating class hierarchies.
  3. Design patterns (part 1)
    • design patterns,
    • creating a class decorator
    • creating an adapter for the class,
    • Observer pattern.
  4. Design patterns (part 2)
    • Chain of Responsibility implementation
    • implementation of an abstract factory,
    • parsing a yaml file
  5. The final project . As part of the final project ‚ÄúKnight in the Underground‚Äù, participants will need to implement a complex system of classes using the studied design patterns.
