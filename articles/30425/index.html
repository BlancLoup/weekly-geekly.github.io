Music library

What do you think you see, does this look like a desktop player you are used to:


I hope that it seems, because it was this goal that I set myself when I decided to create this service.



Have you ever had a situation when you want to let a friend listen to a song or listen to a song, and it's difficult to upload it or place it somewhere, would you like to just give a link where he would listen to it directly from the browser as on a youtube video? My friends and I used to do it quite often, so I devoted two months free of work to a similar site.
I am glad to present the result to the public. So I will be brief, and I will describe the main functions:
- common to all database of songs
- songs can be shared in the form of links that can be thrown off via instant messages or mail
- you can create playlists and also share them just in the form of links
- you can upload your songs through the uploader, while the id3 tags of the file are processed on the server and the song is immediately added to the common database
- if the song has lyrics, it will seem like a separate window
- multilingual
- support for skins (there is support but so far only one skin has been drawn)

A few details about the structure if anyone is interested:
- the whole interface is javascript, jQuery library
- plays of course through the flash player JW Flash Player which is hidden and controlled by javascript
- pictures for performers are taken from lastfm through an open API
- lyrics are taken from a friendly resource accords.com.ua
- link to the song looks like this:
muslib.net/#s-48080
- link to playlist like this:
muslib.net/#p-78

Currently supported browsers are FireFox and Internet Explorer.
Opera is on its way.

I will not hide the fact that I am posting an article here to learn from you your feelings and especially tips on how to improve. noticed errors. I will also be glad if you add some new songs to the collection because the base is still small.

Resource link: muslib.net
