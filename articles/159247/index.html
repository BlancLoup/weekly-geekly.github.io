New Brief: Samsung Launches New EMMC Pro Class 2000 Memory Modules

Samsung Electronics has announced the next generation of 64 GB flash memory modules using a 10nm process technology. New NAND-memory went into production at the end of last month.

The EMMC Pro Class 2000 series of embedded multimedia cards is designed for use in advanced smartphones, tablets and other mobile devices. Performance chips are 30% higher than previous EMMC Pro Class 1500, which were based on the EMMC4.5 interface.



The new high-speed module EMMC will be presented to JEDEC next year for adoption as an industry standard.

The memory has an arbitrary write speed of up to 2000 IOPS, and an arbitrary read speed is 5000 IOPS. In addition, sequential read / write speeds are 260 MB / s and 50 MB / s, respectively, which is up to 10 times faster than similar Class 10 external memory cards (read - 24 MB / s, write - 12 MB / s). Undoubtedly, such an increase in the performance of chips will have an extremely positive effect on the work of mobile gadgets in multitasking conditions.

The dimensions of the new drive were reduced by 20%: the overall dimensions of the module are 11.5x13 mm, versus the previously used form factor 12x16 mm.
