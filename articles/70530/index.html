WebsiteSpark

Microsoft has provided a new opportunity to legally use their products to young web studios or web developers without prepayment. Any company that has a staff of up to 10 people and does not develop products for itself has the opportunity to participate in the new WebsiteSpark program and get licenses for the following products:
  • 3 licenses for Visual Studio 2008 Professional Edition
  • 1 license for Expression Studio 3 (which includes Expression Blend, Sketchflow and Web)
  • 2 Expression Web 3 Licenses
  • 4 processor license for Windows Web Server 2008 R2
  • 4 processor license for SQL Server 2008 Web Edition
  • DotNetPanel control panel
After 3 years of participation in the program, you will be asked to pay only $ 100.
To get acquainted with the rules of participation in the program, as well as apply, you should visit the official website
