Qt 4.6.2 official release for Maemo released

Finally waited !!! Qt for Maemo officially supported package with a bunch of fixes.
What does it mean ???
Now you can officially develop applications using Qt. Moreover, the integration with Maemo has improved and many errors and omissions have been fixed.

Here is a video demonstration:


qt.nokia.com/products/platform/maemo - link to the official project page.

Under the cut details.


The package is available from the extra-devel repository called libqt4-maemo5-dev (this is for development).
What has changed since the last TP:

  • Dorabtali Kinetic scrolling, QTableView now scrolls smoothly.
  • Improved widgets, so it was more convenient to poke fingers.
  • Improved OpenGL.
  • QSpinBox now looks native
  • Auto-rotate is now supported via window flags, not via D-Bus integration
  • All input widgets have been improved, now they are like relatives.
  • And many many many fixes ....


You can download a number of examples from this link qt.nokia.com/doc/qt-maemo-4.6/examples-maemo5.html .


The new release is supported only in the scratchbox SDK, since MADDE is still in beta. But still, MADDE supports the new release; to install, you need to install the package in rootstrap as on the device.

All successful developments, a link to the official news page can be found here: labs.trolltech.com/blogs/2010/02/15/qt-462-for-maemo-5-released
