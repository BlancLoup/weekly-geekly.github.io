IM on VK

Not so long ago, a new version of the API for VKontakte became available.
New features really allow you to develop very powerful applications that are well integrated into the social network. You can read about these innovations here .

It so happened that after the release of the new API, I wanted to do something interesting on the basis of it, both in principle and in terms of technical implementation. And it would be just great if this something turned out to be useful.

Judge you :)

Let me introduce Mesager , the instant messaging system .



Mesager is an instant messenger for VK.

The main advantage over conventional IM (ICQ, Jabber, MSN Messenger) - the contact list is automatically compiled from your friends who installed the application.

Application features:
  • Messaging occurs in real time. Messages are displayed instantly - no page reload is required. Therefore, using Mesager is more convenient than regular VK messages.
  • The contact list is automatically compiled from your friends who have installed the application.
  • Notify users who are not online about incoming messages in the left menu (Appendices ( 1 )).
  • Message history.
Application Development Plans:
  • File sharing.
  • The possibility of authorization outside VK (for example, mobile clients).
  • Integration with Jabber.
  • Much more ...

I will be very glad to any opinion, any suggestions and wishes :)
