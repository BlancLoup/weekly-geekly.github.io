New model reader Lbook V60

Welcome to the Habrosocommunity. We bring to your attention a new model of the electronic book Lbook V60. We have long been waiting for the release of this device. Here is the official announcement , as well as the most detailed and most valuable review of the device. We will not repeat, under the habrokat explanatory note from the manufacturer illustrated with photographs.



Here is the actual device



For extremely delicate connoisseurs, we immediately want to note that in the new line of devices we use the Linux operating system with a graphical environment based on Qt. Good or bad, time will tell and operating experience. So far, only the best impressions, especially in terms of the stability and convenience of porting applications and the development of the software interface.

So it sounded the main word "interface". The device control interface is not very clear. When developing it, we wanted to achieve the following goals:
- make maximum use of graphic icons and a minimum of text, this minimizes the time spent on localizing the device (now, in the device software, you can find almost 4 dozen localization options)
- to control the interface to use the programmable function buttons F1-F5
- standardize the management interface for all device applications.

We think we should say a few words about the case cover. Remembering old troubles, this time we approached this issue very seriously. A super cover was applied to the front panel with the property of repelling grease and dirt. So, white lovers can not worry about the virginal purity of their favorite book - there are no fingerprints left - it is always white and beautiful. This is guaranteed. In combination with a contrasting Pearl screen, the white version of the book looks very attractive. The reverse side of the device is covered with a soft-touch coating - because of which the device can be pleasantly and firmly held in the hands.


The book is equipped with two high-quality micro speakers, it provides quite comfortable listening to audio books in difficult traveling conditions. A joke, of course, but who knows, maybe someone will come in handy. For example, bedtime stories for children on vacation, scroll.

You can put the 3G module. There is also a button to turn off the battery to the left, if, for example, you want to turn off the device for a long time, then to prevent the battery from discharging, this switch physically disconnects the battery contacts.

Keyboard V60 consists of 5 programmable function buttons F1-F5, located directly below the screen. Standard alphanumeric set of QWERTY and 4-button joystick. The buttons are perfectly made, reliable and convenient to use. Unfortunately, due to the lack of sufficient space under the buttons, we did not label 4 Ukrainian letters. But, with certain skills, this, we hope, will not cause great difficulties when typing in Ukrainian, these characters are in their ‚Äúusual‚Äù place.



In contrast to the ‚Äúkeyboard‚Äù books of Chinese competitors, in the V60, the keyboard is quite functional and is not a decorative design element for a greater resemblance to Kindl. We tried to make using the keyboard as logical and simple as possible. There is only one related combination of Shift + Alt for switching layouts. In all other cases, just one click of a button. From the interface of V3 shifted the use of long button presses. But here we tried not to overdo it.

The device supports RSS feeds.
PCC can be configured to load: a) headers only; b) titles and annotations; c) headlines and the whole body of the article. Very convenient - somewhere connected to WI-FI, downloaded RSS and read in the subway

There is a full built-in mail client. You can even customize spam filters on it.

Wi-Fi works as it should - WebKit-based internet browser

Thanks to the use of the database, we got practically a built-in virtual library: five types of sorting, search by book title, author, book series, display of book or file names, developed tools for working with files and folders. In combination with an external librarian program, this gives an excellent result. Metadata on the file is displayed - when it shows the name of the file (below it the name of the book, the author, etc.) in small print. This made it much easier to work with book files (especially FB2). And plus to this, you can sort files / books in alphabetical order AZ / ZA or AZ / ZA - the first button in the menu. In our plans to make a synchronization module with an external library. But, so far there is no definitive solution how to best implement it and who needs it at all.



The first button is sorting the files that were mentioned. The second button is a file search, the 3rd button with an arrow is a search inside a file (select the required one in the file list and go directly to the required page). Button to switch between the SD card and the built-in memory and settings. The program of the file manager "Bookshelf" also did not avoid the "changes". Externally, it differs little from a similar application in V3 / V5. The main external difference of the "Bookshelf" is only that the screen displays 10 positions instead of 8, as before.



The second menu. The first button to switch between books and files. The second button (open / closed eye) is to show / hide system files, select a language, select a font (which can be uploaded), adjust the screen.

There are also 5 personal user profiles. Each profile has 18 settings - from font size, screen layout and text indents, to line and line spacing.

In our opinion, we were able to make a fully functional interface. Yes, the user will have to first look at the manual to the book more than once, but after a few days, the need for this will disappear. Tested on a dozen test users and even 3 blondes. So, we ask novice users to bravely endure the first shock and believe in a happy near future. Everything will be fine. Moreover, the entire interface is designed so that all the basic functions are available at the first level, and all the ‚Äúfine‚Äù settings are removed ‚Äúin depth‚Äù and do not interfere with the work with the book for novice users.
