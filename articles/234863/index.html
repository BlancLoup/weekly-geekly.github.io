Twitter has opened analytics for all users.

Each Twitter user can now see how many people have read his tweets, how many retweets and notes have been made in his favorites, how many people have followed the published links, and how many people have visited the profile. All data is exported to CSV. The site only stores numbers for the last 28 days (but they are updated in real time).

Previously, this statistic was shown only to advertisers. Now it is available absolutely for all users at analytics.twitter.com (you should first disable AdBlock).



The Followers tab shows statistics for readers.



See also the Twitter Analytics reference page .
