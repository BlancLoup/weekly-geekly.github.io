5 online courses in Russian on private and public clouds

Last week we talked about new materials on MSDN, and today we want to please you with new courses at MVA, which will tell about the clouds (especially in the light of interest in the topic from the system administrators, inspired by this post on Habr√© ). Separately, I want to note that the courses are in Russian and are absolutely free (you only need to register on the site for viewing).

2 private cloud courses:
- Private cloud. Introductory course
- System Center Virtual Machine Manager 2012

And 3 courses in public:
- Introduction to Windows Azure
- Introduction to SQL Azure
- Security Overview of Windows Azure

Under the cat more details about the courses:

Private cloud. Introductory course
This course is intended for an initial introduction to the concept of the cloud. This course covers the ideology of building a private cloud. It describes the basic types of services, such as IaaS, PaaS, SaaS. Disclosed architecture and private cloud management information. Shows the way to migrate applications to the cloud. Discusses the types of load in the cloud, and how elasticity differs from scaling. The course consists of three modules: Introduction to Private Cloud, Private Cloud Planning, Private Cloud Management

System Center Virtual Machine Manager 2012
Not so long ago, the entire package of System Center 2012 to the Release Candidate version was updated. The course covers the ideology of using System Center Virtual Machine Manager 2012 to manage a private cloud. It tells about the basic components of building a private cloud. We consider the centralized management of services (multi-tier applications in the cloud). Introduced a private cloud testing program based on System Center products. Modules: SCVMM 2012 Overview, Using SCVMM 2012 to Configure a Private Cloud, SCVMM 2012 Modeling and Maintenance Services

Discover Windows Azure
The course will talk about the basic aspects necessary for successful work with Windows Azure. Modules: Introduction to Windows Azure, Publishing Azure Applications, Managing Windows Azure

Introduction to SQL Azure
The course tells about the main features of SQL Azure, as part of the course you will learn about step-by-step demonstration of creating SQL Azure databases and various ways to manage the created database (SQL Azure portal, SSMS, VS Server Explorer). Modules: Creating and Configuring Azure SQL Database, SQL Azure Security, Synchronizing and Migrating Local Data, Business Intelligence

Windows Azure Security Overview
This course covers the security mechanisms for Windows Azure at the physical, network, host, application, and data levels. In addition, you will gain basic knowledge of authentication and authorization mechanisms in Windows Azure. Course Modules: Windows Azure Security Overview, Authentication and Access Control, Data Security and Cryptography, Network Security

What other courses would you be interested in? And what to change / improve in the format of courses. Comments on the portal are also welcome.
