Codename Ubuntu 12.04 is Precise Pangolin



Mark Shuttleworth announced that the codename of Ubuntu 12.04 is Precise Pangolin.

By tradition, the code name of each version of Ubuntu consists of an adjective and a noun starting with one letter ‚Äî each time next in alphabetical order. This time it was the turn of the letter "p".
