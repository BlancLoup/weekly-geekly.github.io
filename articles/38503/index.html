"Honest" the murder of advertising in LJ.

I figured out how relatively honestly, comfortably and without unnecessary gestures to nail the advertisement in LJ. Immediately make a reservation - a way for Opera.

Without unnecessary gestures - because it is done by the purely built-in means of the Opera in 30 seconds.

The main and popular way to block ads is to set the display style in the user style of the magazine: none; for an advertising diva.
And in Opera there is the ability to connect custom styles. So we do the same, but on the browser side. Well, actually, we connect: go to Preferences-> Advanced-> Content-> Style options, in the field My style sheet select some css-file (usually the default is% operadir% \ styles \ user.css). We write to the file (if it already exists, we add it):

.adv { display: none;}
#ad-leaderboard {display: none;}
#ad-5linkunit {display: none;}

Everything. Advertising is no more.
