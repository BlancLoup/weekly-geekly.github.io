Mojo Share - all social buttons on CSS3 with an icon font in one place

Good day, dear habravchane. I'll start from afar ... Share buttons - this is very important. Most of us with you use either the standard buttons of a specific social network, or one of these services: the Share block from Yandex, AddThis, ShareThis, and newly created Pluso. If we consider all these methods from the point of view of customization and design, then, in my opinion, it is worthy of everything that Pluso implemented. Everything else is almost the same. Considering the main functionality (directly sharing), only Pluso is different in baganutness, where you can easily wind the counters by pressing the buttons many times. I will not compare the quality of statistics collection of these services, although Pluso is clearly not a favorite , because I am not competent in this matter. Let's return to the customization of buttons. As such, at least convenient, it is not. And if we talk about the design, the buttons are not adapted for Retina displays, which immediately catches the eye. So I decided to create a project on GitHub , which consists of a MojoSocial icon font and a MojoShare style file.



Now in the project more than 40 vector icons. All buttons are built on CSS3 using gradients from the corporate colors of each individual service. Plus a few stylistic classes that define the design of the button. And it is convenient to use it, because the file of styles on LESS.
