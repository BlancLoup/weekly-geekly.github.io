Nokia turned mobile phones into RSS readers.

Nokia has introduced the WidSets platform with a collection of Java widgets for mobile phones. Widgets automatically upload information to your phone via RSS feeds: fresh posts from blogs, photos, news, etc.

The WidSets client program gives you access to widgets, allows you to customize their appearance and tracks the amount of traffic. It is suitable for all mobile phones that support Java MIDP 2.0, including those from other manufacturers. Already, the catalog on the site contains dozens of interesting widgets. They are available completely free for all users.

Moreover, developers can easily create their own widgets using the WidSets SDK and publish them in a shared directory.

According to some experts, when creating WidSets, Nokia borrowed some good ideas from the mobile platform Opera Platform .
