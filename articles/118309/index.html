3D landscape in a browser using WebGL

The OpenWebGlobe project of the University of Applied Sciences in northwestern Switzerland released the alpha version of the OpenWebGlobe SDK for WebGL (the usual OpenWebGlobe SDK is for creating applications, and the word Web apparently indicates one of the transports to receive data).

The alpha version currently works only in Firefox 4 (a check is inserted into the code). For terrain, SRTM elevation data is used for the Switzerland leg. Sources are available on GitHub .
