VM Depot launched - a repository of virtual machine images on Linux for the Windows Azure community



I hasten to share great news for developers and IT professionals working with Windows Azure. Yesterday, the official launch of the VM Depot portal took place, which is a repository of images of virtual machines based on open operating systems (primarily Linux), ready for deployment in the Windows Azure cloud.

VM Depot is a project for the Windows Azure community launched by a team of Microsoft Open Technologies, Inc. responsible for open technology within the company. It is expected that portal content configured for different tasks of virtual machines will be created and published by developers and IT professionals.

Today, at the start of the portal, you can find more than 20 configured virtual machine images for various tasks. Below is a list of them:

  • WordPress 3.5-0 (Ubuntu 12.10) and WordPress 3.5-0-multisite (Ubuntu 12.10)
  • Drupal 7.18-0 (Ubuntu 12.10)
  • Joomla 3.0.2-0 (Ubuntu 12.10)
  • Redmine 1.4.5-1 (Ubuntu 12.10)
  • Ruby Stack 1.8.7-1 (Ubuntu 12.10) and Ruby Stack 1.9.3-4 (Ubuntu 12.10)
  • Debian wheezy
  • Mageia 3
  • Riak
  • DevPack 2013.01.08-0 (Ubuntu 12.10)
  • Redmine 2.2.0-2 (Ubuntu 12.10)
  • JRuby Stack 1.7.2-0 (Ubuntu 12.10)
  • Gitorious 2.4.4-2 (Ubuntu 12.10)
  • WebPack 1.5-0 (Ubuntu 12.10)
  • Tomcat 7.0.34-0 (Ubuntu 12.10)

  • Django Stack 1.4.3-0 (Ubuntu 12.10) and Django Stack 1.5b2-0-dev (Ubuntu 12.10)
  • LAMP Stack 5.3.19-0 (Ubuntu 12.10) and LAMP Stack 5.4.10-0 (Ubuntu 12.10)
