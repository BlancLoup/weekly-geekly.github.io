Error in the site cost Electronic Arts thousands of downloaded free games

Over the weekend, Origin , an online computer games store, Origin conducted a survey for gamers, after which, in gratitude for participating in it, the account owner received an OS3874XVC code that allowed him to download the game from the $ 20 store.

At first, some users noticed that with the help of such a code it was possible to download the game twice - through the website and through the Origin client application. A version quickly emerged that the server did not check for validity and reuse of the code, and therefore an attempt was made to download more games and more ...

As it turned out, the code was not only not unique to the user, it was also suitable for downloading many different games for free. It is curious that the site did not even need to log in with a username and password to exploit the vulnerability, and thus it can be assumed that Electronic Arts is unlikely to make any claims for those who took advantage of the error and pumped up free games for themselves .
