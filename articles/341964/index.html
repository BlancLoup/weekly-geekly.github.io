Introducing 3CX v15.5 Update 2 Beta: auto-tuning by QR code, server CRM integration and more

This week we presented the latest update 3CX V15.5 Update 2, which is designed to make installation of the server and 3CX clients even faster and more convenient - both in the local and in the cloud. The main focus of the update is on automating and speeding up routine tasks, but also added a few features long requested by users.

Please note that the update currently has Beta status, so we do not recommend installing it on responsible systems.

Consider the most important innovations in 3CX V15.5 Update 2.

Auto-tuning of mobile clients by QR code


3CX clients for Android and iOS can now be automatically configured to connect to the 3CX server by scanning the QR code. To do this, go to the PC in the 3CX web client and select Settings - Scan QR code. After that, install and run the client on your mobile device, and then select More - QR Provisioning in the lower menu. Video of the procedure is presented here (include Russian captions).



Support multi-user (shift) mode on IP phones


Many users asked us to realize the possibility of using IP phones in shifts. This is in demand in organizations where employees work for shared workplaces in several shifts, for example, in round-the-clock production, at airports, medical facilities or call centers.

Now support for shift work is integrated into the 3CX server and does not depend on the phone model, as it was before. It works in this way: the user enters into the phone a special dial-code and its extension number, and then the PIN of the voice mail. After that, the phone is automatically configured for this user, including the assignment of BLF buttons. Demo video .



Create audio and video conferencing in your scheduler


3CX now has a fairly original audio and video conferencing planning interface - when you create a conference, a pop-up window appears in the selected calendar (Google, iCalendar, Outlook Online or Office 365), in which you continue to set up alerts.

First, specify the basic parameters of the conference.



After that, click the Create Meeting button, and your calendar window will open, where you can continue to configure the notification.



For this feature to work correctly, allow pop-up windows in your browser for the URL of the 3CX server.

CRM Server Integration


In new versions of 3CX, CRM integration modules are gradually transferred from 3CX desktop clients to the server. The client interface is now the 3CX web client. Integration is configured on the server through the REST API. Any CRM system that works with the REST API can be integrated with 3CX. Currently, ready-made integrations are written for Bitrix, ConnectWise, Freshdesk, Nutshell, Salesforce, Zendesk, Office 365, and Google.



When an incoming call is received, the 3CX web client displays a short client card. When you click on the notepad icon, the corresponding CRM system with a full card opens in the browser.



Integrated SMTP server from 3CX


To simplify the life of the system administrator, we have added our own default preinstalled SMTP server 3CX. Now you do not need to create a special account for 3CX on your own server or use a Google account, etc., free boxes of which have restrictions on sending emails via SMTP.



Please note that this is a 3CX SMTP server through which notifications are sent, and not an integrated PBX SMTP service.

3CX for Linux upgraded to Debian 9


We have significantly improved the 3CX edition for Linux - we transferred it to the Debian 9 Stretch platform, almost doubled the installation speed and significantly reduced the requirements for RAM. For installations of 3CX on Debian 8, a script will be released soon, safely upgrading the operating system to Debian 9.

Please note that we do not recommend installing other applications and packages on the stock build (ISO image) 3CX for Linux. This can cause errors both when updating the OS with our script, as well as with the subsequent updating of Debian from the 3CX interface.

Improvements in server installation and administration


The update has several improvements that will certainly be appreciated by system administrators:

  • The system services management section now displays resource consumption (CPU and memory) for each 3CX service.
  • The PBX Initial Configuration Wizard is simplified and integrated with the Network Configuration Wizard.
  • The time zone, locale, and time server are now automatically configured.
  • Getting Let's Encrypt certificates is almost instant.
  • Improved description of various errors related to the activation of the 3CX license.
  • Now it is possible to download Syslog files from the management interface.
  • The Virt what command shows the type of virtualization (hypervisor) being used.
  • The new terminal window allows you to administer Linux OS directly from the 3CX interface, without connecting to the SSH console. See the video .




Network Quality Indicator


3CX clients for iOS and Android have received a network quality indicator. It is useful to assess the quality of the network from which you are making the call. If you hear artifacts during voice transmission, pay attention to it.



New features for the secretary


For the secretary of the organization, the update adds a number of very convenient and demanded functions:

  • The ability to remotely set / change the status of any user
  • Setting DND mode for users
  • Ability to initialize the extension number - its full "cleaning" and assigning to another user

Update to V15.5 Update 2


If you are using 3CX V15.5 or 15.5 SP1, go to the update section and install 15.5 SP2 BETA. After that, the system will disconnect you from the interface. Take your time - the update takes a few minutes. A sure sign of the end of the update will be a notification to the administrator's e-mail.



If you want to try the new 3CX version on 3CX hosting or Google for 1 year, use the PBX Express service .
