Opera Mini 6.1 and Opera Mobile 11.1


We have not finished discussing the new version of Opera 11.50 for desktop computers, and we have already prepared another surprise - the final release of Opera Mini 6.1 and Opera Mobile 11.1. Yes, the version numbers have not grown very noticeably, but in fact there are quite a few changes. We hope you enjoy them. We will not list all and dwell only on the most interesting.


Address bar

Now you can enter website addresses and search queries directly into the address bar of the browser. Or, if necessary, you can use the search field to access your favorite search engines.

Search tips and history

When you enter characters, you will be offered options for entering from the Google search engine and browsing history in the browser. Select the desired option so that it appears in the field.

Autocomplete address bar

When you enter a URL, domains such as .com, .org, or .net are automatically added. In addition, the browser is now learning new domains when you visit websites, such as .gov, .edu, .co.uk, or your country's domain.

Text selection function

If you want to select text, hold your finger over the desired area. A label will appear around the word, with which you can accurately select the desired passage. Then this text can be copied to the clipboard or pasted into the search bar.

Easy viewing

The option of single-column web browsing makes it much easier to read large websites on small-screen devices, freeing users from the need to constantly use scrolling.

Improved support for HTML5 and web standards

Now the browser supports CSS gradients and automatic splitting of text into columns ( Multi-column ). The updated Opera Presto 2.8.149 visualization engine contains many improvements and provides more complete support for the HTML5, CSS3, and many other standards.

Learn more about new technologies in Opera Mobile 11.1 for developers .

Improved Opera Turbo function

This improvement only applies to Opera Mobile - for owners of this version of the page will load much faster and with better quality - the display of pictures is optimized through the use of WebP technology.
