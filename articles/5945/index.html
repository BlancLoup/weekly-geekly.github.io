Boot Camp 1.2 beta released

Apple announced the beta version of the product Boot Camp 1.2, which will be included in Mac OS X Leopard. Boot Camp 1.2 will allow you to run on your Mac not only Windows XP, but also Windows Vista 32-bit. The beta version can now be downloaded from here .

To install Boot Camp you will need:
Mac OS X Tiger v10.4.6 or later. Latest firmware version 10Gb of free space. Blank CD or DVD. Windows license disc: XP Home or Professional with Service Pack 2, Windows Vista Home Basic, Home Premium, Business, or Ultimate

Major changes in the new version:
Supports Windows Vista (32-bit) Updated drivers for audio, video, modem, iSight camera, supports Apple Remote (works with iTunes and Windows Media Player) Improved support for various keyboard layouts, including Russian
