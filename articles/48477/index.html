Skype 2.8 Beta for Mac OS X

Let the version numbering of Skype for Mac and behind the version for Windows, but some features for the Mac appear earlier. Today, Skype has released a new beta of its application for Mac OS X (number 2.8), which implements two new features that are not yet available to users of the Windows version:
  • Screen sharing
  • Skype Access
Screen Sharing allows you to display the current desktop view instead of the webcam image in the video chat window. This way, you can use Skype for improvised screen castes or presentations, and in any other situation where it is easier to show something to you than to tell.


Skype Access is a feature implemented in conjunction with Boingo, an international network of over 100,000 Wi-Fi hotspots around the world, and it will be useful primarily for travelers. Skype Access will allow Skype users to use their account to authorize and pay for Internet access through Boingo access points. On the one hand, this method will simplify the connection and payment procedure (you will not need to register, enter your card details, etc.), and the other will give you the opportunity to use Boingo Wi-Fi access for several minutes without buying a watch or longer session.

In addition, the new beta version has improved the quality of sound and video, and made some less significant changes to the interface.
