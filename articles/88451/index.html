XP and Ubuntu on Apple iMac (holivar machine)

This configuration of the three operating systems is necessary for the operation of our center, in which training is planned to work in these systems. Download the rEFIt utility from refit.sourceforge.net using any available method, launch the file for installation on Mac OSX, selecting the installer rEFIt.mpkg after the file is launched in the opened window. If necessary, enter the system administrator password. This utility is required and needed to manage and synchronize the boot loaders of installed operating systems. Turn off the computer. Reboot. 6. After installing Windows, insert the MacOS 1 disk and install all the necessary drivers.
