The Little Redis Book Translation

The Little Redis Book is a free Redis book.

The book was written by Karl Seguin, with the support of Perry Neal. Karl Seguin is also the author of The Little MongoDB Book, which is often recommended for a quick start with MongoDB. The first version was about a week ago.

Today I made the first version of the translation of this book. The source code is available in the repository on GitHub. There you can also find ready PDF .

The translation was made in order to popularize Redis among Russian-speaking developers. The book is a very convenient and compact manual.

I am grateful to the following people for their help in translating the original text and reading the result:
  • Dmitry Scriptin
  • Denis Veselov
  • polinom


I ask you to send all inaccuracies and typographical errors to correct the translation.
