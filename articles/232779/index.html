Microsoft will close support for unsafe versions of Internet Explorer in 2016

IE developers announced that in early 2016, support for unsafe browser versions, which can still be used on Windows 7 and Windows 8 / 8.1, will not be available if there are no updates. Thus, the company once again pushes users to use the latest version of Internet Explorer 11 on Windows 7-8-8.1, which contains a significant amount of security features (Advanced Protected Mode, 64-bit tabs, Anti-UAF, HEASLR modules, forced ASLR ) and protection against exploits (can work in conjunction only with the latest versions of Windows).



Microsoft highlighted the so-called combination of Windows and Internet Explorer, which will continue to receive updates. For example, for the outdated up-to-date version of Windows Vista SP2 + Internet Explorer 9, updates will still be delivered, due to the fact that Vista as an OS does not contain the security features necessary for correct operation of IE10 +, i.e. the user is not may upgrade to newer browser versions. Microsoft has to support such outdated combinations, as there is a certain period of enhanced support for Vista and the software that comes with it.
