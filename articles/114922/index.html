C ++ / CLI IntelliSense in Visual Studio

The developers of the Visual C ++ Team shared their thoughts on why, with the release of Visual Studio 2010, the IntelliSense hint for C ++ / CLI projects was not included.

It will not be in the outgoing SP1 package for Visual Studio 2010 for C ++ / CLI either ‚Äî this is due to the process of rewriting the internal mechanisms for implementing Intellisense hints. In Visual Studio 2010 they decided to abandon the usual use of the old mechanism of .ncb files due to the large amount of work on transferring it from old versions of VS. And its senselessness as such due to the presence of the started development of a new version of the hint. The new version of the tooltip was not included due to its lack of completeness, including a large number of planned innovations. So, in the current version of IntelliSense, the import of metadata would not work, which would make it largely meaningless.

Thus, in the new versions of Visual Studio, we should see not only the new mechanism of the intellectual hint, but also many innovations related to it.
