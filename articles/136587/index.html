CrackMe from our company

Hi, Habr!

So the New Year holidays are over, work days begin, for two incomplete weeks we all, as far as possible, returned to the working rhythm.
It's time to start a contest!

What is CrackMe?

This is a CLI applet that demonstrates the capabilities of our two products: C3 Cloud Licensing and Stringer.
C3 Cloud Licensing solution consists of several components:
  • Security component
    Automatically (in most cases, without a single line of code) is inserted into a protected Java application
  • License server
  • Management server
    Provides an interface for creating products, defining licensed fields, managing licenses, etc.

What is a stringer you ask? This is our new product for the protection of Java-applications, announce it a little later.

How does CrackMe work?

When you first start the application, it will ask for the activation code, after entering the correct code (the code is checked on the licensing server) a file license will be generated and activated. By the way, our solution can also work with JavaCard.
After the above actions, the application will work normally.

Contest Rules

1) The contest is held within two weeks from the date of publication of the news.
2) Prize - iPad 2 16 Gb Wi-Fi + 3G
3) Publication of hacking without our written permission is prohibited. Violators of this clause are automatically dropped from the competition, and can also be held accountable under the law.
4) If you manage to get rid of our protection, follow the instructions that the application will issue.

Link to jar-file: https://licel.ru/crackme.jar
Run accordingly: $ java -jar crackme.jar (works on Windows / Linux / Mac OS)
