Hero - Firmware VillainROM 6.2 (Android 2.1)

Good day, habrazhiteli.
Recently, I wrote about the selected firmware for myself - VillainROM (based on Android 2.1).

A few days ago there was another update.

What we have at the moment:
  • Fixed problems with the camera (no more shades - neither blue nor green);
  • camera autofocus works properly in applications such as Barcode Sacnner;
  • support for long SMS messages;
  • GPS turns on and off without rebooting the phone;
  • Russification - 99%;
  • available a large number of HTC widgets and the item "Get More HTC Widgets";
  • the kernel supports overclocking, just install an overclockwidget from the market;
  • The browser supports flash, which, if necessary, can be disabled.
The only problem that has arisen is the Russification of the dialer.
But she decides to install the patch .

Above all, Navitel finally stopped sticking with me on this firmware (yes, I use it because there are no longer any normal maps of our province).

Download the new firmware by the link .

The most annoying thing is that the update requires full wipe :(

BTW, the author of the firmware is now engaged in a new version based on Froyo, so we look forward to the results of his work.
