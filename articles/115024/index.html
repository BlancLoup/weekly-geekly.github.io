Patch intrusive warning "Press Esc for ..." in Flash Player version 10.2.152.32

It has long been known (since beta 10.2) that, in the area of ‚Äã‚Äãfixing flaws, version 10.2 of the flash player brought good results. You no longer need to patch the player so that it does not roll up from the full screen when switching focus to the second monitor. Remaining needed (or desirability) patch obtrusive inscription "Press Esc for ...".

On March 4, 2011 a new version of the player appeared - 10.2.152.32 (it was 10.2.152.26 from 27. Jan. 2011). A good reason to upgrade versions of patches. Now 1 patch remains relevant, so all that we need is usually quickly, quickly posted by readers in the forum.videohelp.com/threads/304807-How-to-remove-annoying-Press-Esc-to-message-in-Flash branch -Video . At the end of the message thread, you can often see the newest versions; then they are laid out in the first post of the topic. The patch method is a bit different than what we used before: all warning messages are removed in all languages. The plug-in version has become multilingual, and now the patch is suitable for any national language in the operating system (tested in WinXP 32 bit and Windows 7 64 bit).

Now in the branch of that forum there are 2 fresh patched files for FF-Opera and IE for the new version of the player (they reacted literally in a day); not enough - for Chrome. Put here all 3 files in a single archive. (If the topic is interesting to you, do not forget to support it, otherwise other readers will not see.)

You can patch either manually according to the instructions below, or simply replace the files with the patched ones (for other systems besides Windows, it wasn‚Äôt checked - try to patch them yourself). (Former articles on this topic for outdated versions of the player: 1 , 2. )

1. Get a hex editor (a text editor that allows you to search and change file codes in hexadecimal). Examples: Cygnus Hex Editor FREE EDITION, UltraEdit, many other professional editors.

2. For IE (Internet Explorer), if you wish, download the installer at fpdownload.adobe.com/get/flashplayer/current/install_flash_player_ax.exe and run (automatically in IE it will not start downloading and the player will remain outdated).
Currently install_flash_player_ax.exe version 10.2.152.32 is being downloaded. For other browsers - download from the page get.adobe.com/ru/flashplayer or when reopening the browser.

3. Close all browsers and their auxiliary windows (in the task manager (Ctrl-Shift-Esc) you can make sure that the browser processes are closed).

4. Open files in Hex-editor:
4.1) C: \ WINDOWS \ system32 \ Macromed \ Flash \ NPSWF32.dll (for WinXP, or similar for other OS and for other ways of installation) version 10.1.102.64.
4.2) C: \ Documents and Settings \ <user name> \ Local Settings \ Application Data \ Google \ Chrome \ Application \ <browser_version_number> \ gcswf32.dll (for WinXP, or similar for other OS and for other ways of installation) version 10.1 .103.19 (yes, here the version was a bit different). If there are several versions of Google Chrome, the latest actual version is updated, the file from this folder should be opened.
4.3) For Win7, there is probably a way for the flash library used in Opera, of the approximate form% PROGRAMFILES (X86)% \ Opera \ program \ plugins \ NPSWF32.dll , where% PROGRAMFILES (X86)% is usually "c: \ Program Files (x86). " Open this file if there is one.

5. For IE. Removing protection from changes Flash10n.ocx.
5.1) Move the file C: \ WINDOWS \ system32 \ Macromed \ Flash \ Flash10n.ocx (for WinXP, or similar for other operating systems and for other ways of installation) to another location and remove the read-only attribute from the copy.
5.2) Move the Flash10n.ocx file back (without moving, the system will not allow anything to do with the file) and open it in the HEX editor.
