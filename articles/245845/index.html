Overview of App Annie features for Windows and Windows Phone developers



Good news for Windows and Windows Phone developers. App Annie, one of the largest and free data collection platforms, announced support for analytics of Windows app stores.

Availability of up-to-date information about the rating, the number of downloads and income from the application and quick access to detailed analysis of the entire store will allow developers and mobile studios to make informed business decisions. Solutions based on fresh, processed data, generated into detailed reports using the intuitive App Annie interface.

We will review the main tools for monitoring and analyzing the Windows and Windows Phone app stores provided by this platform.

Store Stats


App Annie is a recognized expert in the processing and analysis of information on leading app stores. The service you need to use to get this data is called Store Stats .

Store Stats allows you to track trends in the application markets of all popular platforms in one place, as well as monitor regular changes in app ratings.

To obtain information about the Windows and Windows Phone app stores, it is not necessary to be a registered user, but to access the extended functionality and information for previous periods, you still need to create an AppAnnie account:



In Store Stats, it is convenient to use filters to view application ratings in a store by country, category, and subcategory:



Data history for stores is available from March 2013 for Windows and March 2012 for Windows Phone.

Also for each application in App Annie collected:
  • Rating history;
  • Daily ratings;
  • Information about the application;
  • Publishers page.

Without going to the application page in the store, all interesting information about it is available inside the service:









Filter parameters for displaying data about the application of interest can also be set manually.

Analytics for Windows Phone and Windows Store


Analytics will be a useful tool for developers. This service as part of App Annie will help collect app rating data in the store and generate reports on downloads and earnings.

In the case of this tool, it is necessary to register with App Annie , which will take no more than 2 minutes, and then linking the Windows / Windows Phone developer account to the analytic platform account:



Here you need to enter the data used for authorization in the Windows / Windows Phone Development Center :



The next stage is an additional check:



In the event that an account cannot be linked for any reason, it is necessary to apply in support of App Annie privately .

When a developer account is successfully linked to the platform, the developer gains access to all the capabilities of the service for analytics of any applications associated with the selected account:

All generated reports have the ability to save to a CSV file, which is useful if necessary to share information. For the same purpose in App Annie provides a mechanism for Team Sharing.

Team Sharing makes it easy to share any information from Analytics, creating groups, inviting participants and providing them with access to data downloads and earnings, simply press the "Share" button in the upper right corner of the report:



Team Sharing will immediately provide the opportunity to select an individual user or create a group.

Conclusion


In this article, we reviewed the main possibilities of the App Annie analytic service for applications in Windows and Windows Phone stores. If you are an application developer for these platforms, the AppAnnie service analytics will help you evaluate and improve the success of your application.
