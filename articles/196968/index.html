Mozilla has released Firefox OS 1.1



Good news for those who follow the fate of Firefox OS. Mozilla has released the largest update to date for its mobile operating system. The OS itself came out this year, and now an update has appeared, which, according to the developers, removes a number of glitches, speeds up the system, plus adds support for new languages.

Among other things, the developers added to the update automatic correction of custom text, MMS support, the ability to import contacts from Gmail and Hotmail mail systems. Probably, users of this OS will find it useful and, finally, the added possibility to save photos from the Web (yes, it wasn‚Äôt before) plus video and audio materials.

The developers did some work on the OS interface by adding the search bar for the applications needed by the user at the top of the display. As for the mail application, the client for this OS, users can now save drafts of the generated letters, as well as add images, which was not there either.

It is worth recalling that at the moment Firefox-smartphones are sold only in countries such as Poland, Spain, Colombia, Venezuela. Now the company's management is working to expand the sales region of these smartphones (ZTE Open and Alcatel One Touch Fire).
