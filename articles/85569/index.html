Firefox and arbitrary selection of columns in tables

Sometimes it becomes necessary to copy the contents of only one column from a table.
But the text is allocated line by line and, accordingly, the entire table is copied from which then you must manually select the desired column.
But it turns out that in Firefox, you can simply hold down the Ctrl key and click on the desired table cells ! And then copy the selection to the clipboard.


This is such a useful thing that even after several years of using Firefox I did not know.
UPD : as suggested by eudj1n : "In the same way, you can select not only table cells, but also arbitrary text on the page"
UPD2 : I was not lazy and I tried this method in other browsers, nor in IE, Opera, Chrome I was not able to select a text column.
