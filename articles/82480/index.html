Seminar Exadel HiTech (Minsk)

On February 5, Minsk, with the support of High Technologies Park, conducts a seminar on modern Java technologies (and not only) with the support of High Technologies Park.

Topics for presentations:
ÔÄ≠
  • Computer Graphics Internals. Boris Dubrov, Exadel (50 min.)
    (The structure of modern graphics systems used to render 2D and 3D graphics. Low-level algorithms for rasterizing vector graphics, color and font management systems, implementing 3D effects. The complexity of modern graphics systems and the algorithms used in them.)
  • Seam 3.0. Maxim Kashinsky, Exadel (30 min.)
    (JSR-299 specification (Java Contexts and Dependency Injection for the Java EE platform). Weld as an implementation of the specification. Seam 3.0 as an extension of the specification. Basic concepts and capabilities. Differences between Seam 3 and Seam 2.)
  • ÔÄ≠ JavaServer (tm) Faces 2.0 (JSR-314). Ilya Shaykovsky, Nikolay Belyaevsky, Exadel (1.5 hours)
    (Updated Web Application Development Standard. Review of the implementation of the Mojarra standard from Sun Microsystems with a practical demonstration of new features.)
  • RichFaces 3.x & JSF 1.2. Ilya Shaykovsky, Exadel (1 hour)
    (The leader among popular component libraries based on JSF 1.2. Overview of using RichFaces to create applications and components.)
  • RichFaces 4.x and JSF 2.0. Nikolay Belyaevsky, Exadel (1 hour)
    (Future RichFaces: a review of planned changes to the library. Component development using the JSF 2 & RichFaces CDK (Component Development Kit) specification.)
  • JBDS overview report. Maxim Oreshkov, Exadel (30 min.)
    (JBDS and JBoss Tools. Purpose and Difference. Supported Technologies.)
  • Visual Page Editor in JBoss Tools. Yegor Radtsevich, Exadel (30 min.)
    (Supported technologies. Using settings, functions and capabilities. Designing templates for the visual editor.)
  • Hibernate Tools in JBDS. Denis Malyarevich, Exadel (30 min.)
