New Mobile Mail.Ru Agent: IM + mail + cards


Mail.Ru has updated the line of mobile applications for Java and Symbian, turning them, in fact, into a mobile "combine". Now Mobile Mail.Ru Agent includes not only support for the MRA protocol, but also ICQ, Jabber, a thin email client, mobile maps and current information about traffic jams on the roads of Moscow and St. Petersburg.
You can download updated versions from the official website ( Java | Symbian ) or directly from a mobile device - wap.mail.ru.

In addition, now Mail.Ru Agent includes the ability to work with many accounts - both under the MRA protocol and ICQ.

Let me remind you that Mail.Ru Agent still allows you to send free SMS to mobile phones.

And in general: MRA - open protocol ;)



The dialogue window has not undergone significant changes, but, perhaps, visually it became more pleasant - due to the support of skins.



Mobile maps are convenient - there is a mode of display of traffic jams. If your gadget has a GPS receiver or your cellular operator provides such data, the Agent will show you where you are. In addition, there is still such a convenient functionality - you can inform your interlocutors about your location.
