FastStone Image Viewer 4.7 released

Recently released a new version of the wonderful program FastStone Image Viewer - 4.7.

This small (takes up less than 10 megabytes on disk) and a smart program is not only a viewer, it allows you to convert images to other formats, as well as perform simple editing: resizing, cropping, color correction, retouching; removes red eye.

But as a viewer, it is very good: it starts up quickly and opens images, has a built-in magnifier, supports many image formats, including PSD and ICO, understands EXIF, can show slideshows. The built-in file browser (from the full-screen view of the image is called up simply by pressing Enter) allows you to quickly select the desired files or directories to view (the images in the selected directory are shown as thumbnails).



It also has convenient print settings, in particular, you can resize and position the image on the sheet.

Major changes to the new version: a touch interface has been added, a new file browser mode has been added with a larger preview window, a tool to reduce noise has been added, an image comparison tool has been improved.

The program is free for personal use. Commercial use requires buying a license for $ 34.95 .

Main features in several lines:
- reads formats: JPEG, BMP, GIF (including animated), PNG, TIFF, PCX, TGA, JPEG2000, PSD, EPS, WMF, CUR, ICO
- writes formats: JPEG, BMP, GIF, PNG, TIFF, PCX, TGA, JPEG2000, PDF
- support for popular digital camera formats: CRW, CR2, NEF, PEF, RAF, MRW, ORF, SRF, ARW, SR2, RW2 and DNG
- fullscreen view with zoom and drop-down toolbars
- powerful editing tools: resize, crop, rotate / mirror
- color effects: black and white, negative, RGB color correction
- comparison of images (up to 4 at a time)
- slideshow with over 150 transition effects and music support (MP3, WMA, WAV ...)
- printing with full control of the location
- image acquisition from scanner
- skins support
- dual monitor support
- touch interface support
- and much more.
