Gnome 2.26 released


Today a new stable release of the desktop environment for GNU / Linux has been released - Gnome 2.26.
The new version of the popular work environment has brought few new products, of which however it can be noted:
* The official inclusion of the Brasero CD / DVD burning software in Gnome (it is enabled by default in the popular Ubuntu distribution). From now on, the version numbers of this program correspond to the release numbering of Gnome. Future plans include tighter integration of the program into the working environment (for example, recording discs via Brasero from Rhythbox)
* Simplified file sharing . Thanks to the new Nautilus file manager plugin, accessing files via WebDAV, HTTP or Bluetooth has become even easier.
* Migrate from Microsoft Outlook to Evolution . Evolution mail client learned to import emails, contacts and tasks from Microsoft Outlook.
* Media player Totem learned to download subtitles for movies . In addition, you can share content via DLNA / UPnP protocols directly from the player.
* Volume control is carried out through PulseAudio . However, if PulseAudio is not installed, the old mixer is left.
* Empathy messenger now has the ability to transfer files (so far only in Jabber), invitations to the chat room, sound alerts, and experimental support for VoIP .
* In the Epiphany browser, the tooltip of visited sites in the address bar , which was implemented in Firefox 3, is implemented.
* The file manager has gained the ability to install programs and libraries through PackageKit to open the documents you need .
