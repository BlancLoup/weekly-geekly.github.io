Bespin is a web-based editor from Mozilla.

Bespin is a Mozilla Labs experiment that offers an open, extensible, web-based working environment for editing code that is aimed at increasing developer productivity, allowing you to enhance his experience and promote the use of open standards.





Initial prototype


As part of the announcement, we also released an early experimental prototype, which should show some of the concepts and capabilities of Bespin.

Bespin 0.1


* The initial prototype, which includes support for the simplest editing functions, such as: highlighting code, large files, undo / redo, preview files in the browser, import / export projects, etc.




Screenshots of Bespin 0.1, launched in modern, compatible with standards, browsers

All source codes related to the Bespin experiment are released as open source software under MPL .

Try


Mozilla Labs is a virtual lab where people gather online to create and experiment with Web-innovations for the common good. The experiment with Bespin is still in the embryo stage and has just begun. There are many ways to join the team and try:
  • play with the demo and let us know what you think about it;
  • report an error ;
  • take the source code and fix the bug or add new features;
  • discuss, argue on the Bespin discussion group ;
  • Join the #bespin channel on irc.mozilla.org.
