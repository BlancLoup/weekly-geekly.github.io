Google Wave Desktop Client for Mac OS X

Many have already received invitations from Google or from their friends on the new Google Wave communication service . Google Wave is designed to combine the functions of email, instant messaging, chat, forum, co-editing documents, etc.

The main client for Google Wave is a browser. However, we must admit that the more stable and faster Google Wave works with the Google Chrome browser .

At the moment, Apple Mac OS X users are most fortunate: they have already written a client called Waveboard , which implements and repeats all the functionality of the browser version of Google Wave.



A distinctive feature of the Waveboard program is integration with all notification systems of the Mac OS X operating system:
  • Update alerts using the Growl system
  • Show the number of updated waves in the status menu and on the icon located in the Dock





You no longer need to separately install on Google Gears , since it is already integrated into the Waveboard program itself .

Like browsers, Waveboard also supports hotkeys.



Current program version: Waveboard 0.11
Download link: http://d.getwaveboard.com/shine/download_waveboard.php
Program site: http://www.getwaveboard.com

The commercial version of Waveboard Pro has recently appeared, which includes the following features:
  • Single wave printing (currently this option is not available in the browser version of Google Wave)
  • Opening unread waves directly from the status menu
  • Change icon in status menu
  • Automatic reset of notifications when the window is activated
  • Save Wave Contents as Webarchive
  • Ability to hide icons in the Dock.
The cost of the commercial version of Waveboard Pro is 9 EUR .

For Apple iPhone , the Waveboard client is also available, which can be purchased on the App Store for $ 0.99 . The functionality of version 1.0 is the same as the desktop version. At the moment, the second version of the program that will support Push notification has been sent to the App Store for review.
