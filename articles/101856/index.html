Internet Explorer: 15 years of history


The journey into the history of Microsoft Internet Explorer begins on August 16, 1995, with the release of its first version. Internet Explorer has gone through very difficult stages in its development.

It competes with browsers like Firefox Mozilla, Opera and Google Chrome. In Europe, Firefox is the strongest competitor. But, be that as it may, IE is conveniently located on the top and accounts for a large share in use.



Internet Explorer 1


released August 16, 1995

Internet Explorer 1 is a graphical web browser from Microsoft.

Internet Explorer 2


released November 22, 1995

Integration with the conductor appeared

Internet Explorer 3


released August 13, 1996

Internet Explorer 3 is the first commercial browser with CSS support. In addition, support for ActiveX control, Java-applets.
The third version includes a set of deliveries of Internet services and mail. By the way, in the third version appeared the IE logo!

Internet Explorer 4


appeared in September 1997

In version 4, deepened the level of integration between the web browser and the operating system. Internet Explorer 4 adds support for group policies. Also, this version already comes with Windows 98.

Internet Explorer 5


appeared March 18, 1998

In early 2000, Internet Explorer 5 reached over 50% of the market share!
Following this, in July 2000, version 5.5 was released, with an improved preview of print pages, improved support for CSS and HTML, and an application programming interface.

Internet Explorer 6


pleased with his appearance on August 27, 2001

The sixth version included: support for DHTML and partial support for CSS1, DOM1 and SMIL 2.0. The MSXML engine has also been updated to version 3.0.
Other new features include new versions of Internet Explorer Administration Kit (IEAK), a multimedia panel, integration with Windows Messenger, automatic resizing of images, and a new interface, which was important (matching the visual style of the Moon interface in Windows XP).
Internet Explorer 6 was released a few months before Windows XP.

Internet Explorer 7


saw the light October 18, 2006

In this version, bugs were fixed to improve the browser to support web standards. There are tabs, vid preview, search management, reading feeds, support for Extended Validation and anti-Phishing Filter certificates.

Internet Explorer 8


released March 19, 2009

Only a beta version was spent a year!
Changes in version 8 allow Internet Explorer 8 to pass Acid2 test

Internet Explorer 9


Will appear September 15, 2010 (estimated date)


Exactly in a month, Microsoft is going to release the ninth version of Internet Explorer, which has been tested for the past six months. This version does not support Windows XP, because Microsoft decided to stop supporting this operating system.

IE9 will have full or almost full support for all CSS3 selectors, JavaScript performance. IE9 will present hardware accelerated rendering using Direct2D, hardware accelerated text visualization using DirectWrite, hardware video acceleration using the Media Foundation, image manipulation with the support of the Windows Imaging Component and high printing accuracy (XPS).
