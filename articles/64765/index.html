8 free Ajax / php file managers

In this small article I will tell you about the server file managers with the Ajax interface that I found on the Internet.

The order is very conditional, but I tried to arrange it out of usability and functionality.

eXtplorer


.
  • viewing the list of folders (what kind of file manager is it without it :))
  • edit, copy, create, delete files
  • search, upload and download files
  • work with archives
  • create files and folders
  • change chmod
  • ...

Demo not found

ajaXplorer


.
  • view folder list
  • edit, copy, create, delete files
  • search, upload and download files
  • backlit editing
  • bulk download (flash)
  • work with archives (ZIP)
  • create files and folders
  • change chmod
  • listening to mp3 and watching FLV
  • view images

Demo

phpLetter


Nothing extra.
.
  • view folder list
  • edit, copy, create, delete files
  • create files and folders
  • image preview

Demo

relay


.
  • view folder list
  • edit, copy, create, delete files
  • create files and folders
  • drag-n-powered powered
  • listening to mp3 and watching FLV
  • apload progress bar
  • access control

Demo

filethingiey


.
  • consists of 1 file, easily painted with CSS
  • view folder list
  • edit, copy, create, delete files
  • search, upload and download files
  • bulk load
  • work with archives (ZIP)
  • create files and folders
  • access control at the level of users and groups, and black / whitelist

Demo

FM on MooTools


.
  • view folder list
  • edit, copy, create, delete files (drag-n-drop)
  • search, upload and download files
  • listening to mp3
  • Download via FancyUpload
  • work with images (resize)
  • there is like a TinyMCE plugin
