ZF 1.6 RC2

Today came the second candidate version 1.6. At the moment, the announcement has not yet been, but you can already download it.

What is new? So far I can only assume based on a comparison of the code of the first and second candidates:
  • Added a class for working with captcha and the corresponding field for the form
  • Added FirePHP support ( FireBug extensions) for Zend_Log and Zend_Db_Profiler
  • Added new helpers for Dojo
  • Added file upload class and field for Zend_Form + validators
  • ... and a lot of changes in other classes ...
Try and wait for the release?
