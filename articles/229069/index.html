Linux Foundation introduced the first release of the OS for the car



The non-profit organization Linux Foundation, or rather, the working group from this organization, presented the first release of the Linux platform for cars. This platform was called Automotive Grade Linux (AGL), the development of AGL began in 2012 , when a special working group was formed.

According to the developers, the new platform is designed to work with all systems of cars from different manufacturers (with the need to "tweak" AGL for themselves), including dashboards, media systems, and so on.

AGL is a product that needs to be customized and customized for each automaker. In general, AGL can be called a special Linux distribution, which uses Tizen . The distribution kit already has a low-level support for automotive equipment, so that automakers need to customize the interface, add individual applications and ensure regular system updates.

The working group itself includes such well-known auto companies as Nissan, Toyota, Hyundai, Jaguar Land Rover and HARMAN, as well as major IT companies, Intel, Samsung, Fujitsu, NEC, NVIDIA and TI.



In the working prototype of the platform, control of the dashboard, climate control, multimedia system and other elements is organized. There is also the ability to control voice, connect peripheral devices (for example, a smartphone) and synchronize data on the state of each individual auto subsystem. Work with various auto systems through the Automotive Message Broker (AMB) bus



AGL is a fully open source project that is accessible to everyone.
