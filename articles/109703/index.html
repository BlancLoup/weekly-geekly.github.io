Java Mobile Agent - now with Jabber

Friends,

In August, we wrote about the appearance of support for the Jabber protocol in the Mobile version of Mail.Ru Agent for Symbian.

Now, Java Mobile Agent also has built-in support for the Jabber protocol, which means that you can constantly communicate with those who are on the VKontakte site or use such Jabber services as Facebook, G-Talk, LJ Talk, etc. d.


How to download new Mobile Agent for Java?

- Log in from your phone or smartphone on m.mail.ru

In this case, you will immediately be offered the appropriate version of the Mobile Agent, since The model and operating system of the mobile phone are determined automatically.

- Or get via SMS a link to download the version from here
