LastPass Launches New Mac App



For regular Internet users, the protection of their passwords and personal data is always relevant. Thus, many software developers began to create desktop, mobile and browser applications for storing and managing passwords. Applications like Dashlane, LastPass and 1Password and others are already beginning to be included in the list of required programs when setting up a new computer.

The main players in this market now arrange an arms race, releasing updates one after the other. For example, LastPass and its competitor DashLane, on the same day, launched a security update for their programs, which allows users to automatically reset compromised passwords on several sites at the same time.

The company LastPass , which has long established itself in the market of applications for remembering passwords, this morning introduced a new application for the Mac. Now the application is even more integrated into OS X and includes several new features, such as automatic login, a tool to check the security of your online accounts and a quick search function.


Quick Search, a new option that allows users to easily find passwords or personal information from specific sites. According to the developers, this option is the main advantage of the update, and support for hot keys makes it even more convenient for Mac OS. After finding the desired site, the user can immediately open it in your favorite browser, and LastPass will automatically enter all the login information.


The next innovation is Security Check. The tool allows LastPass users on Mac to quickly check passwords on all online accounts. If weak passwords are found, they can be easily changed right in the program.



The updated program also allows users to store logins, passwords and other important information in the so-called "storage", which allows you to have completely secure local access to it. The ability to access important information without connecting to the Internet, as it used to be through browser extensions from LastPass, can be considered a huge step forward, which makes the program a very strong competitor for DashLane and 1Password on the Mac platform.



The application will appear in the Mac App Store today. The program will have two distribution options : free (with advertising) and with a $ 12 subscription, which includes technical support and two-factor authentication.
