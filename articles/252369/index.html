Installing RSAT for Windows 10 TP

Windows 10 has long been available for upgrade, and therefore one day it was decided to upgrade. After the upgrade, there was a question about managing servers. Install RSAT may fail.

On the Microsoft RSAT website for Windows 10 TP is available, but through the search you can get only to the old version and build 9926 will not install it.

After some time, the RSAT search for Windows 10 TP build 9926 was found and successfully downloaded, but it refused to be installed immediately.

The installation is successful and pretends that everything is fine, but the update itself, which contains remote control, is not installed.

Further search has led to the fact that RSAT for Windows 10 TP still exists exclusively for the EN-US interface.

To install RSAT, download the EN-US package , then install it

Now you can download a working RSAT for Winwows 10 build 9926 , if you have not yet downloaded it, and install it.

After performing all the operations, RSAT for Windows 10 build 9926 is successfully installed and working.
