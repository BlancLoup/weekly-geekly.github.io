Google Chrome Gets Experimental Gesture Support

An experimental extension to the Google Chrome browser called Chromegestures is presented. It should be noted that compared to, for example, the gesture control system in the Opera browser, the Chromegestures technology looks no less developed and rather mature, considering that the first version was released only on July 16 of this year.


To install the Chromegestures extension, you need the Chrome for developers version (dev). In addition, before installing an extension, you must enable support for extensions by including the enable-extensions option in the browser launch command. After that, you can safely go to the download page and install the ChromeGestures.crx file.

Installing the extension and restarting the browser will give you full access to the gesture control system. For example, holding the right mouse button and moving the mouse to the left will take you back through the viewing history. Accordingly, holding the right mouse button and moving to the right - go forward. Movement down and to the right corresponds to the command to create a new tab, and if you add another upward movement to this, you will get the command to search for the selected text. If you do not understand something, help can be opened by drawing a question mark with the mouse - up, right, down, left and once again down. By moving the mouse with the right button pressed up and down opens the settings dialog where you can change the sensitivity of the browser to your movements.

You can learn more about the Chromegestures extension to the Chrome browser, get help information and download the installation files for yourself on the official project page.
