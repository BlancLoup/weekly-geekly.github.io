Adobe has released ColdFusion 8

Adobe has begun selling a new version of the ColdFusion 8 web server. This is one of the most popular in the past, web development environments, which is now at the end of its popularity , but still enjoys some respect among web programmers.

The new version of ColdFusion significantly increased the speed of the web server ( comparison in PDF ). The difference can be dozens of times. In addition, the system implements new server performance monitoring tools, new PDF generation tools, Ajax toolkit, integration with .NET and Adobe Flex, interactive debugger, special CFTHREAD tag for multithreading, CFFEED tag for working with feeds, 50 new tags for manipulating with images and many other new features .
