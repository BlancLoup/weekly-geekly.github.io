Installing Redmine on Windows 7 as a service, with sqlite3 database

This article discusses the option of installing Redmine with the SQlite3 database on Windows and running it as a service. All necessary components for installation are given in the article. For testing, a virtual machine was installed with a ‚Äúbare‚Äù Windows 7 without service packs, no additional software, and a step-by-step installation was made according to the instructions in the article.

The answer to the question: "How many resources does the established Redmine require?"
RAM: The Redmine process uses 2 MB of RAM, and the Ruby interpreter eats 170 MB of RAM.
HD: The installed ‚Äúempty‚Äù Redmine with the rest of the software takes ~ 500 MB of hard disk.

Part 2, devkit installation
1. Open the cmd console on behalf of the administrator.
2. Go to the Devkit directory:

3. Execute the Devkit initialization command:

4. Run the Devkit installation command:

2. Install sqlite3 for ruby

3. We update all the gems and confirm the updates:

4. We turn off the console.

1. In the config folder (C: \ Ruby \ redmine \ config) open the file ‚Äúdatabase.yml.example‚Äù.
