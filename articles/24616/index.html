Glare of Firefox 3 in Vista


Today we will talk about using some of the advantages of Firefox 3 ( using beta 5 ) in Vista, namely the use of the Aero glass effect on the toolbar.

I stumbled upon this remarkable extension on the Mozilla Russia forum, and from there I was taken to the MozillaZine forums .


The bottom line is:

  • Install the Glasser extension from here or from the author's post on MozillaZine
  • Install the Stylish extension
  • Download additional styles for the design of the bookmarks bar from here or, again, from the original message on MozillaZine ( GlasserExtras.zip )
  • Open Stylish and add the bookmark bar style you need:
  • We right-click on the Stylish icon:
    • Choosing "Style Management"
    • Click "Create"
    • Paste the code from the CSS file we need (I use the Vista Personal Bar file ‚Äî Windows Explorer.css )
    • Give the name to the style, save and apply
    • Done! :)
If you want to achieve greater similarity to IE7 in Vista, you can additionally install the Personal Menu or Hide Menubar to hide the main menu (I personally liked the Personal Menu more)
