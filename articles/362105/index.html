ZBOX PI320: Zotac micro PCs with 4-core Intel Atom for $ 200



The company Zotac for a long time announced the release of a new miniature PC with an interesting configuration for the user. This is a ZBOX PI320, a computer with a 4-core Intel Atom Z3735F processor with a core frequency of 1.33 GHz.

In addition, the developer installed 2 GB of RAM, as well as 32 GB of flash memory available to the user. As the OS, you can choose Windows 8.1, Ubuntu or another Linux distribution.



The device configuration also includes a wireless communication module WiFi and Bluetooth 4.0. There are also LAN, HDMI ports, three USB 2.0 ports and a headphone jack. As for the graphics card, the system is installed familiar Intel HD Graphics.

In this case, the price of the device is $ 200 (as far as you can understand, the price already installed Windows 8.1).
