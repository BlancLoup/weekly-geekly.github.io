Templater - template manager and tweaker for MantisBT



MantisBT is an open server web application for managing projects and tasks ala Redmine . In my last article I looked at the problem that was relevant for this project - the lack of templates and the gloom of the interface. Here is a link to the previous post. After all the discussions, I decided to develop the theme, and wrote a plugin for MantisBT, with which you can apply custom design templates. Actually I present to you this plugin - Templater v1.0 .

Templater is designed as a standard plugin for MantisBT and can apply custom templates to the application. Since the developers of MantisBT do not really wrap over the design and layout, in templater templates are also based on 40% on redefining css-rules and 60% on gutting DOM. Perhaps not the most beautiful option, but workable and so far the only one. In addition, it is very simple to create your own template and feed it to templater - just create several folders and files with the corresponding code.

Templater is also a small tweaker for MantisBT. With the help of the plugin, you can turn on several useful features and turn off some painful bugs . The list of useful tweaks will grow over time.

In the first Persia templater'a there are two templates: DarkBlue (a modified version of the template from the previous post) and DarkRed (based on the DarkBlue template, but has a dark red coloring).

In general, I did not spend much time on this, but I am pleased with the result. I would like to share this result with you, to hear your opinion.
Here is the link to the plugin repository . In it, you can read a more detailed README.md, which describes all the features of the Templater template and the templates themselves, learn how to install the plugin on your MantisBT.

PS So far the plugin has been tested on the mantis of versions 1.2.17, 1.2.18, 1.2.19 . For 1.3.0 beta I can not say anything.

UPD 01/31/2015. Updated version of the plugin - v1.1. In mantisbt 1.3.X versions, all tweaks now work, as well as the DarkBlue-1.3.X template added for this version
