New Year's digest of Microsoft Azure

Hello,
Continuing the tradition of summing up the results before the New Year , we release the New Year's digest in abbreviated form.
2014 was productive for Microsoft Azure - the platform entered new markets, became the leader in PaaS, IaaS and storage, according to Gartner , many new services emerged (from my favorite updates, for example, that now Azure Backup supports client versions of Windows ) . Those following the development of the platform might have noticed that parallel to the services used for corporate purposes, services began to emerge that are also oriented towards the scientific environment - which only machine learning as a service and HPC development cost. In 2015, we are still waiting for a lot of interesting things.
The rebranding of Windows Azure to Microsoft Azure has also become big news - the fact that on the Microsoft platform a huge amount of resources are used by clients from OSS, shows that the platform is not only powerful, but also open.
