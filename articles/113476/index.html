PPA GIMP Paint Studio for Ubuntu


work done in GIMP Paint Studio
GIMP Paint Studio is a set of tool installations, high-quality brushes and textures that turn GIMP into a complete paint program.

Gimp Paint Studio is a collection of brushes and related tool settings (presets) designed to accelerate repetitive tasks. GPS avoids the need to manually adjust instrument parameters each time. Many users do not know about presets at all (preset tool settings). This feature is rarely discussed, but is a very useful feature of GIMP.

You can learn more and download the latest version of GPS on their website.

Ramon Miranda (developer) calls the project‚Äôs goal to provide an adequate working environment for graphic designers and artists to start painting and feeling comfortable in GIMP from the first launch.



Installation in Ubuntustal is much easier due to the emergence of PPA GPS, GIMP pre-installation is required (by the way, GPS will only work with GIMP 2.6.x, if your version is higher, you need to downgrade the version via purge), for the time being we set this:

sudo add-apt-repository ppa: shnatsel / gimp-paint-studio
sudo apt-get update
sudo apt-get install gimp-paint-studio
