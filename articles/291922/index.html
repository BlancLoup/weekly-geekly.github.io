Google is testing cloud competitor GitHub and Bitbucket

This year, Google launched the free beta version of the Cloud Source Repositories service . This is a new service for sharing and editing code. The repository is deployed on the Google Platform cloud platform.

The direct competitors of the service are the companies GitHub and Atlassian .

Over time, the Cloud Source Repositories can become part of a single system for building and deploying applications. Given the rapid spread of cloud technologies, Google is thinking about developing its own cloud solutions and systems, following the example of Amazon Web Services and Azure cloud , reports Venturebeat.com.

The previous Google Code project could not compete with GitHub and Bitbucket (a project from Atlassian). Google Code was conceived as a service for publishing source codes, with support for Subversion , Git and Mercurial version control systems. It was launched in 2006. Then Google Code had no strong competitors. But with the advent of GitHub and Bitbucket, the situation has changed dramatically.
