Lubuntu 10.10 released

Following Ubuntu , version 10.10 and the Lubuntu distribution came out .
  • Torrent: people.ubuntu.com/~gilir/lubuntu-10.10.iso.torrent ;
  • direct link: people.ubuntu.com/~gilir/lubuntu-10.10.iso ;
  • Md5: people.ubuntu.com/~gilir/md5sum.txt .

On the Lubuntu on Habr√© already told more than once, in particular, the topic about the release of version 10.04 was quite complete.

In short, this is a project whose goal is to create a lightweight, energy-efficient distribution with low resource consumption. LXDE is used as a desktop manager.

According to the Linux Magazine test results, compared to Xubuntu and Ubuntu, the Lubuntu distribution uses almost half the memory of a typical desktop installation, and almost three times less when booting from a Live CD.

Changes in the new version





Standard Applications

Added:
  • Update-notifier, to receive notifications of available updates;
  • Xpad, to quickly create notes;
  • Ace-of-penguins, for providing some games.
Removed:
  • Parcellite, not supported by the developer, is less useful, since most of the errors with copyipe disappeared with pcmanfm 0.9.X;
  • Pyneighborhood, replaced by gvfs support in pcmanfm.
Changes:
  • LXtask replaced Xfce4-taskmanager;
  • Evince is now being used to read PDF, as ePDFview has seen serious memory leaks.

New topic

A new and fresh theme created by Rafael Laguna is available.

Installation Slideshow

When installed, a slideshow about Lubuntu is shown.

Indicator support

Lxpanel now supports Ubuntu indicator applets (disabled by default).

New metapackages
  • The lubuntu-core meta package allows you to install only basic Lubuntu packages;
  • The lubuntu-restricted-extras meta package allows you to install packages such as flash, java, and codecs for chromium.
LXDE Updates
  • LXDE is no longer HAL dependent, although HAL is still used to burn CDs;
  • new lxappareance (without Openbox integration).

Both regular updates of the LXDE and Ubuntu packages are also included in Lubuntu.
