Opera 9.5 SDK for devices

A press release from Opera Software announces the release of a new development toolkit - Opera 9.5 SDK for devices. This release allows developers of various devices to use the latest features found in Opera-9.5. A full-featured demo version of Opera 9.5 SDK for devices is available on the company's official website, which is usable for 4 calendar weeks. This toolkit is designed to work only in a Linux environment, since It is this platform that is most popular as an embedded operating system.
