Pixar announces free version of Renderman for non-commercial use.

From 10 to 14 August, a new Renderman will be shown on SIGGRAPH in Vancouver (Canada) in two versions.



One version for $ 495, which will combine (under one flexible license) functionality in itself, before different packages: RenderMan Pro Server and RenderMan Studio. It can be used in commercial projects.
And another Renderman can be downloaded for free and used for personal training, experiments, research, writing plug-ins and tools for it.
Renderman will be available only for 64-bit OSes (Mac OS 10.8 and 10.7, Windows 8, 7, Vista and Linux). Only Autodesk Maya 2013, 2013.5, 2014 and 2015 will be supported.

The official page of the announcement and a little more here .

RenderMan - software product, software package, industry standard rendering for 3D-animation. In particular, there is a standard for describing three-dimensional data for their subsequent visualization, as well as a stand-alone render, recently released under the same name.

RenderMan was created by Pixar and has been used since 1986 by many famous production houses.

In most cases, the implementation of the Renderman standard is a set of programs that are called from the command line and play a role in the rendering process. The standard distinguishes between the concepts of scene description files and geometry ‚Äî RIB files, and material description files ‚Äî SL files, or shaders. All of these files have a simple text format, described in the specification. Shader files are mini-programs in a highly simplified C language dialect.

There are connectors to most 3d-modeling programs: Autodesk Maya, Cinema 4D.

Current version as a render for Autodesk Maya, RenderMan for Maya 4.0, as Renderman Pro server 16.4, as RenderMan Studio 3.0.3.
