Python support in Power BI

We are pleased to announce that Python, a programming language widely used by statisticians, scientists and analysts, is now integrated into our free Power BI Desktop. After enabling the feature in the settings, you can use Python to clean, analyze and visualize data. More under the cut!



The first integration feature we added is the ability to run Python scripts directly in Power BI Desktop to create a data model. You can do this in the "Get Data" dialog box.



After selecting this option, you can insert your script.



Integration with the query editor allows you to clean up data using Python and use advanced methods of analyzing your data, including, for example, removing missings, prediction and clustering.

Finally, you can use Python to create new visual effects in your report. Like rendering in R, Python visual effects will be updated with data updates and cross-filtering. However, the visual itself is not yet interactive.
