Brave Browser introduces an advertising platform based on user rewards.


Starting today (24/04/2019), Brave Rewards feature is available for users of the latest version of the Brave browser. When activating Brave Rewards, users viewing ads provided by Brave Ads will receive 70% of their advertising fees as a reward.

A little about Brave


Brave is a free and open source web browser based on Chromium, announced by the Mozilla Project co-founder and JavaScript creator Brendan Eich. @ ru.wiki
The default browser has a built-in ad blocker and web tracking. The main direction of development is the development of the user reward model for the viewed content. @ en.wiki

Brave ads


Brave Ads is not a replacement for existing online advertising. These systems are separate and complementary (?). Brave Browser - by default - blocks Internet advertising and user tracking systems. The option to view ads provided by Brave Ads itself, the user can activate in the browser settings.

Opt-in


Activating Brave Ads, in the process of web surfing provided notifications about the possibility of viewing ads. Clicking on the notification, the user goes to a new browser tab with an advertising page. The user also has the ability to customize the frequency of ad notifications.

Personal Information


Brave Ads ensures the safety of personal data. A selection of advertising is determined by the user, and personal information is not transmitted anywhere.

70% of ad revenue


Users receive 70% of revenue from paying for advertising, 30% will receive Brave. User revenue is translated into Basic Attention Tokens (BAT). BAT can be used to donate to your favorite sites or favorite authors. Brave is also working on ways to convert BAT to real currency.

Total


Brave Ads offers a reciprocal benefit system that complies with the GDPR ( General Data Protection Regulations ) and prioritizes a decent assessment of the user's attention.

PS


Brave Ads, starting from today (04/24/2019), will be available in the USA, Canada, France, Germany and the UK. Expansion of the number of regions is planned in the near future. The integration of Brave Ads into mobile versions of the browser will occur in the following months.
