Transferring contacts from Microsoft Outlook to Mac OS Address book

Judging by the number of times this question is asked in various forums and the community, the question is relevant, but in most cases it is suggested to use Google Mail to transfer contacts. because I don‚Äôt have a google post, but for the sake of registering a box and fooling around with synchronization, I honestly feel lazy, in addition to various utilities for transferring contacts, I found such a simple and convenient way in 10 steps in 2 minutes.

in general, everything is done quite simply, because there is such a wonderful thing as a vCard! Unfortunately, MS Outlook does not know how to export contacts to the vCard format, so you have to fool it a bit:

1. Open Microsoft Outlook

2. Open contacts

3. Select all contacts (ctrl + a)

4. Poke the right mouse button and select "Send as a business card"

5. MS Outlook creates an email with an attachment from our vCard

6. Select all additions (ctrl + a)

7. Create a folder

8. Paste from buffer (ctrl + v)

As a result, we have a folder with our vCard from our contacts. Next, either by mail, because by file sharing we transfer these files to the Mac, open the Address Book, select File - Import - vCards in the menu ... and select the folder with our cards. Everything is perfectly imported.
