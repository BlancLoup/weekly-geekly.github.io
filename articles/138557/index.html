Updated MSDN Russian-language web development section: new design, new training courses and articles

The other day I received an update and redesign of a large Russian-language section on web development on the MSDN website.


The updated section now consists of three main sections: ASP.NET MVC , ASP.NET Web Forms and WebMatrix , which clearly define the three main directions for the development of the Microsoft web platform for developers.

With the updating of the site, new materials on the main sections were launched. We have already written about the release of the new WebMatrix guide in 16 parts. In addition to it, the updated section contains the following materials:
  • A selection of 24 ASP.NET MVC 3 training materials and a description of MVC 4 innovations;
  • a completely new large collection (66 articles) of material in Russian on ASP.NET Web Forms :
    • Part 1. Introduction to ASP.NET Web Forms (4 articles);
    • Part 2. Presentation and data management (4 articles);
    • Part 3. Data management elements in ASP.NET (11 articles);
    • Part 4. Working with data (13 articles);
    • Part 5. Routing in ASP.NET (5 articles);
    • Part 6. Security issues in ASP.NET (3 articles);
    • Part 7. Caching and performance optimization (9 articles);
    • Part 8. Internationalization (3 articles);
    • Part 9. Publishing ASP.NET web projects to the server (14 articles).
In addition to the main sections on the updated site are presented and additional:
  • IIS web server
  • Node.js
  • Javascript (jquery)
  • HTML5
  • CSS3
  • ASP.NET in Windows Azure
  • SQL Azure
  • Cloud CMS
I draw your attention that sections will be constantly updated and updated with new materials.

We hope that the submissions and the updated MSDN section will be useful to you.
