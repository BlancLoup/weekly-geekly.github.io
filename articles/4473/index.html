SearchMash: a new search interface from Google

Today, without making loud announcements, Google has opened the search site SearchMash , which some experts already call Google 2.0 . Indeed, at first glance, this site can be said that it is a searchable Ajax interface of the new generation. Obviously, here Google will test its most innovative developments in the field of interfaces, which for some reason cannot be tested on the official website.

The site SearchMash implemented a number of interesting features.

1. Instant search query set . The text appears in the search bar at the moment when the user starts typing it on the keyboard. Now you do not need to specifically place the cursor in the search box.

2. Menu for URL . Clicking on any link among the search results brings up a context menu with a list of options: open in this window; open in new window; open a saved copy; more results from this site; more similar pages. Due to the fact that these options are built into the context menu, the opportunity to visually clear the search results.

3. Image panel . Instead of contextual advertising, the right side of the screen shows the image search results for this key query (three pictures).
