SOLUTIONS FOR KYOCERA SAFETY

Let's be honest, security is a major issue in offices. A huge number of people use one device together, including to work with documents of a confidential nature. Here we describe the solutions that are used in KYOCERA devices for security.

IPsec technologies

The new single-processor controller used in some full-color MFPs of the new series provides an increased degree of data protection and the MFP itself through the use of SSL, IPv6 and IPsec. This functionality provides users with improved performance in a secure local area network.
Recall what IPsec is. It is a technology that implements the IP protocol security function, which is the basic element of the Internet infrastructure. Thanks to the appropriate use of IPsec technology in new MFPs, it is possible to avoid exposing MFPs to data integrity or theft.

Using SSL

To protect information from unauthorized access to the MFP of a new series, encryption of data transmitted over the local network is supported, as well as protection of access to information located in the MFP. Using the SSL protocol allows you to protect information when:

Thus, when using new MFPs, you can protect not only the printed data, but also the data obtained by scanning.

Network Authentication

For cases when the MFP is used as a public device of a large workgroup, the devices of the new series support the authentication function that uses the user name and password to access the MFP and the data. In this case, both local authentication and network authentication can be used.
With local authentication, usernames and passwords are stored on the MFP itself. This method is recommended when using the MFP series as a stand-alone device, as well as in small enterprises with peer-to-peer networks without dedicated servers.



Network Authentication supports Active Directory servers running the Windows operating system. It is preferable to use it in the conditions of large local networks of large enterprises that have dedicated servers in their structure.



Thus, in the case of using the network authentication function, the user does not need to remember several passwords and user names (one for working on his PC and the other for using the MFP), since the same password and username will be used to access the MFP. user uses to access his PC.

Using data encryption technology

When copying, printing, scanning and sending messages via the fax line, an array of data is formed on the hard disk of the multifunction printer, which is stored there until it is overwritten by other data during the next copying, printing, scanning and sending fax message . This is true for any MFP of any manufacturer. While the data array has not been overwritten, it is theoretically possible to access this data block from the outside in order to steal information

In order to prevent possible data leakage on the new series MFP, the option of overwriting / encrypting data on the MFP hard disk can be set. This option automatically overwrites the data on the hard disk of the MFP after the completion of each task, as well as encrypts the data as it is written to the hard disk of the MFP. In this case, decoding of the encrypted data cannot be performed by other devices, which eliminates the risk of data leakage. It should be noted that in addition to the data in the same way, the MFP address book is encrypted.

The ability to protect documents from unauthorized printing / transmission / scanning

If necessary, the option to protect printed documents from unauthorized copying, transferring or scanning can be installed on new full-color MFPs. This option allows you to protect the information of printed documents from unauthorized copying, scanning or sending along facsimile lines using special watermarks. If you attempt to unauthorized copying, scanning, or sending a secure document via fax lines on the same MFP series, the image of the first page of the copy will be hidden with a solid background, and scanning and sending the document along the fax lines will not be available.



If an unauthorized copying of a protected document is attempted on any other digital copying machine or MFP, the corresponding messages will be displayed on the received copies as a background image.
This security feature is only available with PCL XL.
