A service for scanning programs installed on a PC and downloading stable and beta versions of common software, with the help of this service you can very easily track the freshness of the software currently installed.

The service works as follows:

Download the program Update Checker



The program displays an icon on the "start" panel that changes the appearance of the icon if there is a possibility of updating, and at the same time can produce such a concise message.



After that, you can double-click on the icon and a page will open with links to download new versions, and on this page you can see which version you have installed and the place where it is installed. Works on Vista, XP, 2003, 2000, ME and 98.



I have been using the service for about 6 months all this time, it worked well and stably.
