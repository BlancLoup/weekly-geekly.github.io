FAR manager - old friend is better than new two

It will be a question of building in the recent past a fairly popular file manager FAR.
Unfortunately, a sad picture has recently been observed - this powerful FM is fading into the background, being in the merciless jaws of Total Commander.

I will describe the FPP build, which I often use myself in order to refresh the memory of the once very common FM. In addition, I hope for someone the article will be a discovery.


# 0


The topic is not advertising and does not pretend to be the ultimate truth. The choice of assembly was made solely for my own reasons (+ there are still a few of my plug-ins).

Installation


The installer can reinstall the application.


During the installation process, you can select the full (PRO) or normal (LITE) version. Differences between Lite and Pro are described here.

It should be noted that in addition to plug-ins, you can select a set of archivers (), as well as additional utilities, such as chmview and zd2t (to view the contents of the .doc file in the HEAR editor).


Plugins are grouped by functional component.

At the end there are a number of useful tips that will help in the development of the product.
I would single out:
1. If not all the default settings suit you, it is recommended
using RegBrowser plugin to find out / save .reg file of necessary settings
(use ShiftF1 in the plugin itself) and use it as a patch
2. Note the .bat files in the bundle - using
you can save / restore them to the registry settings of PAR, and plugins
3. You can apply the Far PowerPack settings without being a user from which
installation was made. To do this, you need to select all .reg files in the PlugInSettings folder.
and pressing Ctrl-G to type and execute regedit -s!.!
The same should be done with the files FarSave1.reg, FarSave2.reg.
If you do not have registry access to the HKLM branch, then you need to do the same thing, excluding
files FarSave2.reg and MultiArc.reg.



Pay attention to the checkboxes at the last stage of the installation:
Console colors (for all console applications) - we are talking about the forced installation of the color gamut for all terminal windows being created
Macros, FPP settings - clear from the name - apply recommended macros and settings for basic FAR
Plugin settings - register plugin settings

The package is made by the Nullsoft-installer, so it implies a silent installation:
FarPowerPack_setup.exe / S / D = C: \ Far

Using


1. Work with multimedia
One of the most convenient features is viewing images, listening to audio / video using a plug-in.
MultiMediaViewer


2. Using the built-in editor
Syntax highlighting


3. The power of the FAR manager is primarily in the processing of files and data in them, although with this statement it is possible to argue - for someone it is more important colorer and gadgets for programmers.
I would like to highlight the main things you can do:
- mark / compare matching files (SelectingEx, AdvCompare)
- rename a group of files by template (FileNum, rename using Ctrl-G (ren "!!" "!. .Htm"))
- rename a group of files in the editor (FRename)
- change the register of files (FileCase)
- change the end of lines UNIX / Windows format (FileCRLF)
- recoding files (FarTrans)
- transliteration / de-transliteration (Translit)
- creating image files (Mold)
- spreading files into folders (FileMove2)

in the editor:
- search and replace using reg. Expressions (Search & Replace)
- Sort file contents (SortEdit)
- insert numbering in the file (RenumberLines)
- autocompletion of words (EditCompletion)
- work with selected blocks (Blocks, BlockFormating, BlockIndent)
- drawing pseudographics (DrawLine)
- change of register inside the file (EditCaseEx)
