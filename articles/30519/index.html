Firefly - communication on the site pages



Firefly - an interesting variation of the chat, which can be activated on any page of the site. Communication takes place using the mouse and keyboard.

When you activate the chat mode, all other users in the chat can see the position of the mouse on the page at the moment. Messages written by you appear where it was in the pointer.
For a private (almost) conversation, you can move to a corner of the site and correspond there :-).

All this works by overlaying the second layer on the flash on the site page.

In this chat are available:
  • Ability to change the image (avatar)
  • Integration with Twitter (you can send messages there)
  • View message history using timeline
  • Displays the number of people in the chat


This chat can be activated on your website by inserting a short code at the end of the page:
< script language = "javascript" src = " static.onfirefly.com/ff.js " > </ script >
