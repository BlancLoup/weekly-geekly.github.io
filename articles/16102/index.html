VMware Fusion is now compatible with Leopard.

Today, VMware has announced the release of VMware Fusion 1.1, an update for its virtualization software in Macs, based on Intel processors.

Update 1.1 adds official support for the new Mac OS X 10.5 Leopard operating system, and also improves support for 3D graphics and experimental support for DirectX 9.0. The update also improved the interaction between the main and virtual operating systems (minimizing Windows applications to the Mac OS X dock and displaying Windows applications in Expos√©), and it became possible to create a virtual machine from the BootCamp with Windows Vista. Naturally, productivity has been increased.
