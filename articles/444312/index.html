Installing ReactOS from a USB flash drive

Hi, Habr!

Etsnovo we! In the news about the release of release 0.4.11, we said that thanks to the updated RAM-disk driver, ReactOS can not only be launched, but installed over the network using the PXE protocol .

However, a project volunteer under the name Illen found out that using the same new RAM disk driver, ReactOS can be installed from a regular USB flash drive. Thus, release 0.4.11 was the first release supporting the installation of ReactOS on computers without optical drives. Illen wrote detailed instructions and recorded a demo video.


Briefly about what is happening on the video:

  1. On the USB flash drive copy the installation and live images of ReactOS.
  2. A boot record is created, and the boot loader is configured to boot using the RAM method.
  3. Booting from a flash drive, running the installation image and the first stage of installation on the target computer.
  4. Booting a ReactOS live image from a flash drive and adjusting freeldr.ini (an analogue of the boot.ini file) on the target computer.
  5. Booting from the disk of the target computer and completing the installation of ReactOS

Note that in the upcoming release at number 0.4.12, item ‚Äú 4. ‚Äù will not be required, since the master branch has an improved algorithm for detecting and recalculating available disks and partitions in the system, which takes into account the possible presence of removable devices .
