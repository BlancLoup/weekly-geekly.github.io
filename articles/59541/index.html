Tasks is now in Google Calendar


Tasks, previously available as a Lab extension for Gmail, is integrated into Google Calendar. The following options are now available for Tasks:

- Tasks for which the date is specified are now automatically added to the Calendar.
- To set the deadline for a task, simply right-click in the task list.
- Change the date of the task, you can simply drag and drop.
- To mark the completion of the task, just put a check mark in front of it.
- Tasks can now be sorted by date.
