Ubuntu 11.04 Natty Narwhal Alpha 1 released

The first alpha Ubuntu 11.04 Natty Narwhal is released.

What's new?


Since this is only the first alpha, there are still many packages and features in active development. And of course, much still has time to change before the April release.

This is one of the most significant changes. Considering that work on it was started only in October, the progress achieved is quite impressive.
