Ubuntu 11.10 Oneiric Ocelot Alpha1 released


Hello to all.

Today came the first alpha Ubuntu 11.10.
Gnome 3 and all related applications are embedded in the system:
Here is the complete list:

Network Manager 0.8.9997
Evolution 3.0.2
Nautilus 3.0.2
Gedit 3.0.4, Totem 3.0.1
GNOME Control Center 3.0. 2
Firefox 5.0 Beta 2
LibreOffice 3.3.2,
Shotwell 0.9.2
Compiz 0.9.4 (the version is the same as in Natty)
Empathy 2.34.0
Banshee 2.1.0, Gwibber 3.0.0.1
Xorg Server 1.10.1
Linux kernel 2.6.39-3
Under the cut, I posted a video with an example of using the new system.



I personally fell off the desktop wallpaper and had other lightweight gaps. Alpha version should be set only for experimental purposes, if you need a working system, I recommend previous versions.
