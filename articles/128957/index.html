Meeting 1, season 2011: storing data in WP 7.5 and using Event Tracing in Windows



Colleagues, I invite you to the meeting of the Moscow developer community, which will be held on September 26 at 19:00 in the Microsoft office. This is the first meeting in the season 2011, which will talk about plans for the next year and useful technologies.

Reports


Data storage on the Windows Phone 7.5 platform . Work with SQL CE

The report is dedicated to the storage and access to data on phones running Windows Phone 7.5. Including work with isolated storage and built-in DBMS SQL CE.

Speaker: Sergey Pugachev

Use Event Tracing for Windows to improve support and debugging

Event Tracing for Windows (ETW) is a high-performance, scalable tracing system provided by the operating system. The report will discuss the use of ETW in troubleshooting, both in its applications and in the operating system. It will also cover writing your own ETW provider and trace file analyzer.

Speaker: Alexey Gusarov

How to get there?


Microsoft office is located at ul. Krylatskaya, 17, business center "Krylatsky Hills".

From the subway go special route buses. Ford buses (or similar), on buses sign "Krylatsky Hills".

Stop at the metro: Krylatskaya metro station, exit the metro head car, then exit the metro from the door to the left, go to the right. The stop will be 10 meters from the metro exit. Buses run every 10-15 minutes. Since everyone cannot fit in one bus, it is better to arrive at the Krylatskoye metro station at 18: 20-18: 30 (the meeting starts at 19:00)

After you enter the building with the Microsoft sign on the roof, turn left - this will take you to the Microsoft office. Then you will be shown how to get to the meeting venue.
