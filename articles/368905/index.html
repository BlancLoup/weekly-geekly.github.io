Windows security updates can only be downloaded using Internet Explorer

Microsoft has announced that from May 10, 2016, it will stop uploading all updates on the Download Center. Some updates from this day will be distributed through the Microsoft Update Catalog.

Unfortunately, the site Microsoft Update Catalog users waiting for an unpleasant surprise.



Updates can only be downloaded using Internet Explorer 6 or higher. Browsers Google Chrome, Mozilla Firefox and even Microsoft Edge are not supported.

Apparently, only Windows security updates move to the new address, and the rest will be available through the Download Center.

In this case, the standard update channel of the system through Windows Update in the operating system remains to work as before, the browser is not needed here. True, from the user's point of view, this is not the safest way. For example, users of Windows 7 and 8 have to make some efforts to block the installer of Windows 10, which penetrates the system using trojan methods .

In addition, individual Windows updates may violate the functionality of the system, for example, remove installed programs , change default settings, add new telemetry options (PC data collection). Therefore, experts recommend studying each update before installation by disabling automatic updating via WSUS.

Microsoft does not explain why it is moving updates from the Download Center to the Microsoft Update Catalog. According to one version , this is preparing to make Windows Update the exclusive distribution channel of updates for home users in the future, and Microsoft Update Catalog for corporate ones.
