Extensions for OpenOffice

In order to somehow fill the gap in extensions in OpenOffice , I decided to make a small selection with a brief description. A little googling, I realized that the topic is very relevant, but very poorly covered. The purpose of this article is not to describe all existing extensions, but only to tell what they are, what they can do and where to get them.

If you compare the number of extensions with what offers, for example, the same FireFox, then for OpenOffice there are many times less of them, but their number is constantly increasing.

Writer


Writer's Tools - a whole package of new features: a list of tasks (tasks), a search for the desired word (lookup tool), a built-in translation (google translate), backup with sending a copy to mail (Email Backup) or FTP (Remote Backup), a converter into different formats of a group of files (Convert To Tool), an application manager for launching programs directly from a text document (App Launcher), copying text fragments into a database (Add To Basket) and much more, including the similarity of the game ‚ÄúField of Miracles‚Äù :)

Annotation Tool - allows you to select a part of the text for giving comments. Now users cost small yellow rectangles that are easily lost in the text.

Sun Weblog Publisher - the extension allows you to publish blog entries on various blog services. It is possible to add tables, lists, links, graphics and spell checking. (Paid, $ 10).

ParaDTP is a utility that allows you to quickly work with text: change the font size, the distance between lines in the text.

Calc


Calc2DokuWiki - import tables (or parts of a table) into Dokuwiki.

Base


Sun Report Builder - create stylish reports.

Draw


CADOO.o - the creation of technical drawings.
SVG Import - An extension for importing and editing SVG graphics.

Impress


eVoice is an extension for inserting voice comments into slides. The utility allows you to record a voice and "attach" it to the selected slide. Speex format is used for voice compression.
Sun Presentation Minimizer is a utility for optimizing graphics in presentations and reducing file size.

Common to the entire OpenOffice package


Multipages is an extension that simplifies working with documents that contain hundreds or thousands of pages.
OpenOffice.org2GoogleDocs - import / export documents from / to Google Docs or Zoho.
PhotoAlbum - creating photo albums.
CodeFormatter - code highlighting for C ++, Java, StarBasic.
Tabbed windows extenstion - bookmark functionality for open documents (for example, as in Firefox).


The complete collection of extensions can be found here: Extension repository . It is worth remembering that some functionality will be already built into the next versions of the package (for example, advanced features for working with Notes2 comments). The general impression of extensions in OpenOffice I have is: the industry is in its infancy. Many extensions are under development.
