The best resources to get started with Drupal 8

2014 will be the year of Drupal 8!

Currently, the community publishes a monthly alpha version of the system. At the time of this writing (December 11, 2013 - my note ), we have Drupal 8.0-alpha6 . Although there is no specific release date, it's safe to say that the new version of Drupal will be in 2014.


Reasons to be in awe of Drupal 8

  • Symfony 2 as part of the kernel
  • Reworked the theme. Twig, HTML 5
  • Mobile Publishing Improvements
  • Improved multi-language support
  • RESTful web services
  • Configuration management
  • Migration API
  • and much more
