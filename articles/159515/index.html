Release Firefox 17 and Firefox 17 ESR

After the release of Firefox for Android, the official release was released for desktops. What's new in this version:


  • Support Notification Center in Mac OS X 10.8;
  • Vulnerable versions of plug-ins are no longer loaded by default and are activated only with the user's permission using the click-to-play function. More details can be found here and here ;
  • Made about 20 performance optimizations. Also improved the work of the New Tab (New Tab);
  • The implementation of the Social API , which allows integrating into the browser services for working with social networks (for example, Facebook );
  • Increased icons in the Smart Address Bar (Awesome Bar);
  • Added support for the " sandbox " attribute for iframe, which includes additional isolation elements for embedded iframe content;
  • At-rule support @supports, which allows you to test support for specific CSS properties and values. At the moment, support for @supports is experimental, and to activate it in about: config, you need to find the layout.css.supports-rule.enabled parameter and switch to true ;
  • Fixed a min-height / max-height bug related to box-sizing;
  • SVG support for attributes FillPaint and StrokePaint;
  • For JavaScript Maps and Sets , support for iterable has been added ;
  • Accelerated and simplified work with the Web console, debugger and development panel;
  • A new markup panel has been added to the Inspector (Ctrl + Shift + I) that simplifies DOM editing;
  • Added support for Pointer lock in web applications;
  • Fixed the scrolling of pages on sites with fixed headers.


Firefox 17 ESR - a release with a long support period also becomes available for download. Updates for this version of Firefox will be released every 6 weeks during the year and will only include addressing serious security and stability issues.
The Firefox 10 ESR version will be supported until February 19, 2013 - after this date, the browser will be automatically updated for all users of the Firefox 10 ESR version.
