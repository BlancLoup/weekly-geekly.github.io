Update 1.1.3 to correct 1.1.3 (almost live broadcast)

Just the other day I updated my phone firmware to 1.1.3. But, having read this topic, I realized that I did it in the wrong way and the programs that should be started and run from an unprivileged user are started and run by root. In the comments was a link to the correct method. I decided to try to remake everything to be correct.
