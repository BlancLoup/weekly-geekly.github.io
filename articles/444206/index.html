March 27, Moscow - JavaScript MeetUP

We are gathering the next JavaScript Meetup JavaScript next Wednesday, March 27, at 19.00 at the Corporate Innovations Hub site (Bolshaya Polyanka St., 2/10, p. 1).

In the program there will be 3 reports from our employees. Admission is free (but you need to register in advance), there will be an online broadcast.



List of reports and useful links - under the cut.

1. ReactiveX Redux // Dmitry Samokhvalov


Dmitry will talk about the Observable concept and the benefits it brings to working with asynchronous applications. Shows how to practice Observable in conjunction with Redux, and how it differs from Redux-Saga.

2. Derivation of the Action type using Typescript // Dmitry Novikov


And again about Redux and about what threatens the loss of typing in action types, when the literal type becomes primitive. In the report - on how to track this process, plus the experience of our solution to this problem using Typescript.

3. The process of continuous improvement // Oleg Luchansky


When it is worth spending time on tests, and when it is possible to do without them, why it is worth increasing competences in all areas, and not just writing code, this and much more can be discussed in the framework of this report.
