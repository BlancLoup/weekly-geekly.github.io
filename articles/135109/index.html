Image Catalyst 2.1

Hello Habr!



After several months of work on correcting errors and adding new features to the public, I am taking out a new version of Image Catalyst 2.1. In the new version a number of key changes were made.

Image Catalyst is a collection of programs for integrated optimization / compression of PNG and JPEG images without loss of quality within the same format.

PNG optimization algorithms

The new version has been completely rewritten optimization algorithms PNG. Instead of the usual three PNG optimization algorithms (Fast, Normal and Xtreme), there are only two algorithms (Advanced and Xtreme). New algorithms have nothing to do with previous algorithms, this is due to the large number of innovations in TruePNG (thanks to x128). Thanks to the innovations in TruePNG, it became possible to abandon the CryoPNG and PNGWolf applications, which significantly reduced the time to optimize images with virtually no loss in compression ratio.

JPEG optimization algorithms

In this case, the changes affected only JPEG Progressive, new features from the JPGCrush project were added, which gave a slight increase in the compression rate, while the optimization speed remained at the same level.

Full list of major changes
  • removed CryoPNG and PNGWolf;
  • project interface has been changed;
  • TruePNG 0.3.5.4 dated 12.25.2011 updated;
  • Added JType from 11/04/2011 (thanks x128);
  • the function of creating the Images.cvs file has been removed;
  • updated JHead 2.93 from 12/02/2011 (thanks GoblinNN);
  • PNG and JPEG optimization algorithms were completely rewritten;
