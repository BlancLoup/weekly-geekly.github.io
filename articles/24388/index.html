Making the installation of extensions easier

When installing extensions, Firefox has two things that are annoying - the inability to properly install an extension from a local disk and a delay of a few seconds during installation. Both problems are solved in a couple of minutes.


Install from a local disk.

1. Enter in the address bar about: config
2. We are looking for the line extensions.hideInstallButton
3. Change the value to false
4. Go to Tools -> Addons and click on the ‚ÄúInstall‚Äù button

Installation without delay

1. Enter in the address bar about: config
2. We are looking for the string security.dialog_enable_delay
3. Change the value to 0

That's all. Thanks to the Daily Gyan blog.
