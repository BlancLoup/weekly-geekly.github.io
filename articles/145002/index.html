Windows 8 Release Preview, Visual Studio 2012 RC, Windows Server 2012 RC and TFS 2012 RC



Today, Windows 8 Release Preview is available for download in 14 languages, including Russian. As Steve Sinofsky writes, RTM will follow her, so there is not much time to wait. I recall that the release of the Preview Preview was scheduled for the first week of June this year, respectively. Windows Server 2012 RC and Visual Studio 2012 RC became available a bit earlier.

If the testimonials and telemetry data for Windows 8 and Windows RT meet our expectations, then we will proceed to the final stages of the RTM process within 2 months .

Sam Release Preview (build 8400) was compiled a week ago. The rest of the time it took to create localized assemblies and preparing for distribution. Incidentally, the Chinese version, which was merged this morning, has a similar build number.

Unfortunately, there is no changelog yet. Upgrading from the Developer or Customer Preview will not work.
