Google's XSS-game

Google introduced a game of finding xss vulnerabilities in order to spread information about this most dangerous and widespread type of vulnerability. Google takes vulnerability detection so seriously that it pays up to $ 7,500 for serious xss.



But the game will show how to detect xss, this knowledge will help you to prevent errors in your applications. In total, the game has 6 levels, in case of difficulty in performing tasks, you can use the tips. By passing the game promise a cake.
