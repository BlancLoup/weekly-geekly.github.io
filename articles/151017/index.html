How to make the game special?

Ted Price (Ted Price), the executive director of Insomniac Games, told Gamasutra about the four rules that his studio adheres to when creating any game. In turn, we consider it necessary to share these principles with you.

1. You can do something alone, but better than anyone.

When you are just starting to develop a new game, you have an advantage - there are plenty of opportunities. But this is your misfortune. Where to begin? Which path to choose? Crazy ideas can be many, especially if you generate them in a fun company with a keg of beer. But if you then try to implement at least one of them, there is a chance to make a very expensive and complicated project.

2. It all starts with fun.

Defining this phenomenon is quite difficult. But when you play, its presence or absence is clearly felt. The game should be interesting and entertaining. If you yourself do not like to play, then do not pretend to be interesting, do not fool yourself. Correct, change everything that you see fit. Fan will definitely come.

However, many make a mistake, believing that if you add a story, graphics and music, then it will become more interesting to play. Nothing like this. All this is secondary and is only a supplement. The game itself must create a fan.

4. Make the game yourself

By investing yourself, your interests and personal experience in the game, you invest in the project soul. Then the game will be special, unique, which means it will show a good result.
