State expenditures: public expenditure monitoring, API and competition for developers

State expenses ( clearspending.ru ) is a new public and non-commercial project that we have been working on for the last six months in the form of a base of all state and municipal contracts.

We took the entire base of contracts with zakupki.gov.ru and, after a certain amount of effort, turned into a website with convenient search, navigation, some analytics and, most importantly, with an API.


And here we will stop in more detail. API is almost the main distinguishing feature of the site from the majority of similar ones.

More about him on the special page www.mashape.com/infoculture/clearspending-ru-as-russian-government-spending
as well as using the API example on Github'e - github.com/idalab/clearspending-examples

The main advantages:
  • contract base (9,034,993 as of February 2014)
  • complete directories of suppliers and customers participating in the state order
  • full text search in the entire database


For each contract, the supplier and the customer are available full information and links to the source.

And, the main thing.

We hold a competition among developers with a prize fund of 300 thousand rubles.
The competition has two nominations:
  • Nomination "Web project" - for all who want to make a project in the form of a website
  • Nomination "Mobile application" - for a mobile application on any platform



We accept applications until April 30.
More details can be read here - clearspending.ru/contest
