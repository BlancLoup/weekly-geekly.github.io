New free video courses on ASP.NET Web Forms and ASP.NET MVC

The other day we published excellent video courses on ASP.NET technologies on the official website http://asp.net/ .

The courses were created by Pluralsight (a great company with training courses in .NET technologies), are available completely free of charge and offer an excellent opportunity to learn (or refresh in memory) ASP.NET WebForms and ASP.NET MVC 3 web development technologies. Each course is created as a separate trainer and offers a complete training program (from basic fundamentals, working with the concept of Code First in the Entity Framework to security issues, deployment, testing).

Below are some details on two free video courses published the other day (and links to every available video):

ASP.NET MVC Training


Over the weekend, we published the final video tutorials in a completely new ASP.NET MVC 3 training course created by K Scott Allen. You can study these video materials from http://asp.net/mvc (section on the left in the Essential Videos section):



Below is a list of ten video modules available in the new training course:
  • Introduction (35 minutes)
  • Controllers (35 minutes)
  • Razor Views (45 minutes)
  • Working with Data: Entity Framework Code First (39 minutes)
  • Working with Data: Validation and Mapping (33 minutes)
  • Javascript, jQuery and Ajax (47 minutes)
  • Security (43 minutes)
  • Caching, Localization, Resources and Diagnostics (34 minutes)
  • Unit Testing (37 minutes)
  • Deployment and Configuration (37 minutes)
More information can be found on the official page at http://asp.net/mvc .

ASP.NET Web Forms Training


Coach Dan Wahlin has developed an excellent course on ASP.NET Web Forms 4.0.

You can take a free video course at http://asp.net/web-forms (section on the left in the Essential Videos section):



Below is a list of nine video modules available in the new training course:
  • Introduction to ASP.NET (64 minutes)
  • ASP.NET Web Forms Features (111 minutes)
  • Working with Data (75 minutes)
  • Entity Framework (47 minutes)
  • Master Pages (24 minutes)
  • JavaScript and Ajax (62 minutes)
  • Security (coming soon)
  • Using ASP.NET Dynamic Data (coming soon)
  • Web Deployment (coming soon)
More information can be found on the official page at http://asp.net/web-forms . The last three videos will be published this month.

More content


Watch the training centers http://asp.net/web-forms and http://asp.net/mvc in order not to miss even more educational text and video content that will be added by us in the next few months. Each of the training centers today contains a good structured set of content that will allow you to quickly learn and start using ASP.NET technologies. We will improve the content and add more in the coming weeks and months.
