Intel will open an atom netbook app store next year

Intel has released a beta SDK for application developers creating programs for netbooks based on Atom processors. The SDK is currently running under Windows and Moblin, but they promise support for Adobe AIR and Microsoft Silverligh.

Next year, a dedicated online developer store will open, where
application authors will receive up to 70% of application costs, and Intel will take the remaining amount to itself.

The release of the developer kit became part of the Atom Developer Program initiative, the main task of which is to optimize, speed up and reduce the cost of creating software for portable devices on Atom chips.

Intel said Wednesday that more than 50 million netbooks will be sold by the end of 2009.
