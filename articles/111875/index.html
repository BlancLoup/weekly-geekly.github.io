Signed final SP1 for Windows 7 and Windows Server 2008 R2

Today, Microsoft has posted on the corporate website of the corporation the final RTM build of the first service pack (SP1) for Windows 7 and Windows Server 2008 R2. The signing took place version 7601.17514.101119-1850 - it is she who becomes gold, and Microsoft plans to provide it to its partners in the near future.
Most likely, in the public domain SP1 will appear only in February. If you believe the rumors, then it can happen at the MWC 2011, but this information is questionable, because the event is devoted to mobile devices and, in particular, phones.

Microsoft has announced its plans for Windows 7 SP1 and Windows Server 2008 R2 SP1 in the summer at the Tech-Ed conference. Service Pack 1 will include traditional hotfixes, as well as a number of new virtualization technologies designed to prepare current versions of the client and server Windows OS for the era of cloud computing. This, first of all, is about the RemoteFX technology, which should provide rich multimedia capabilities when users are connected remotely.

Those who previously installed the beta version of SP1 should pay attention to the update KB976902 , distributed recently. It is designed so that you can easily uninstall them, as well as to avoid problems with the subsequent installation of the final SP1. The presence of this update in the system will be a prerequisite for installing the update package.
