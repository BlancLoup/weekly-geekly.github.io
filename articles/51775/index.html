Placemarker - Google Earth Signatures

One of the programs that we have developed and distribute for free is Placemarker.
The program is not supported by any organization and is entirely authoring and proprietary.


Placemarker is a program for generating images based on text and icon. Placemarker allows you to create special signatures in Google Earth. We enter the required caption text, add an icon, effects and select the font. The results of all operations are displayed in the preview window. Next, we select the position of the label in Google Earth and save the result as a local KMZ file. All of these features allow Placemarker to be a good tool for GE designers and theme makers.

The main features of the program that are missing in the standard Google Earth:
  • Multiline Signatures
  • Font and effect selection (shadow, contour)
  • Color gradient selection for text and effect
  • Selecting the position of the icon relative to the signature
  • Selecting the position of a point in Google Earth
  • Save as KMZ file with embedded image

One of the main features required when creating presentation cards is to create high-quality legends.

And, as a rule, it is required
  1. create a set of markers
  2. arrange them on a map
  3. align to common style
  4. select individual features based on the overall look of the picture

To solve this problem and was created Placemarker.
