Rocking files for iPhone and iPod Touch via Safari !!!

Now you can download files from the Internet through the built-in Safari browser!
To do this, you need to install a plugin for Safari - inside the archive there is an Info.plist file, which shows the mime file types processed by the plugin.



You need to download the plugin, unzip it, copy it to the player folder / System / Library / Internet Plug-Ins and restart your iPod Touch!
