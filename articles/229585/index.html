The GameTutorials website made all of its 350 game programming lessons free.



The GameTutorials website, known for its C / C ++ / Win32 / OpenGL / Direct3D / C # / Java lessons, has opened all its content for free. All lessons are tested for compatibility with Visual Studio 2013, in the very near future lessons on the Unreal Engine and the Unity Engine are expected, in addition, outdated lessons on OpenGL and DirectX will be updated (the site now describes the version of DirectX 9).

Most of the lessons affect the basics, but some of the lessons cover the creation of full-fledged small games. The lessons themselves are source code, specially equipped with a large number of detailed comments. According to the promises of the authors of the lessons, if you pass most of the lessons, you will gain a real baggage of knowledge, the acquisition of which usually takes several years.

To view and download lessons you need to register.
