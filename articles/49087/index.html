SSD RAID Level 0

OCZ Technology has unveiled new Apex SSDs , which are a Level 0 RAID array packed in a single package.


OCZ Apex with SATA II interface have the following characteristics: random access time - 0.2-0.3; reading speed - 230 MB per second, writing - 160 MB per second.

Capacity of new products - 60, 120 and 250 GB. The cost, respectively, is $ 199.99, $ 369.99 and $ 829.99.
