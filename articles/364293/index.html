The big presentation of Windows 10 will take place on January 21

What will Microsoft show?




January 21, 2015 (all day), Microsoft will hold a presentation of Windows 10 . Most likely, this event will officially present the version of Windows 10 Consumer Preview, built on the basis of the existing Technical Preview (or just show a new version of Technical Preview).

One way or another, but by this time new functions will be included in the operating system. The current Technical Preview is focused on desktop computers, but in the final version should implement the touch interface. Other changes are pending.

Experts speak on other innovations that Microsoft may announce.

It is known that Windows 10 will support devices running on the open-source protocol AllJoyn . This protocol allows devices to exchange information with each other (for example, a motion sensor transmits information to a light bulb that no one is in the room), forming networks on the Internet of Things. So you can now download the AllJoyn SDK and experiment with applications.

There is an opinion that the new OS will be distributed, including by subscription . The company's management is now discussing this way of monetization, the decision has not yet been made.

Another supposed feature is Continuum mode support, in which two-in-one devices will be able to quickly switch from one mode to another when the keyboard is connected / disconnected.

The January build may be the first to introduce Cortana integration ( voice assistant with artificial intelligence ).

For IT professionals who are planning an early transition to Windows 10 in a corporate environment, Microsoft offers to get acquainted with a deep technical analysis of the fundamental changes in the operating system that you definitely need to know in the 4-hour Windows 10 Technical Preview Fundamentals for IT Pros podcast. It is published on the Microsoft Virtual Academy portal, there are many other useful materials.

Broadcast presentation on January 21 will be held live via the Internet.

The final release of Windows 10 is scheduled for autumn 2015.
