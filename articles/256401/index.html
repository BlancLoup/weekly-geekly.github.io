Compare Visual Studio Community 2013 with Visual Studio 2013 Express. Features of the license agreement

Most recently, Microsoft has introduced a new edition of the most popular development environment - Visual Studio Community 2013. Most developers can not understand the features of the license agreement, and therefore questions arise, for example, "Can I use Visual Studio Community 2013 in commercial development?". Let's look at this issue together.

Actually, how does Microsoft itself position Visual Studio Community 2013? - This is a free, fully functional integrated development environment. What does it mean? - this means that any individual developer can create free or paid applications using the Visual Studio Community. Unfortunately, the Community edition is provided without a MSDN subscription.

Here‚Äôs how Visual Studio Community can be used in organizations :
- Visual Studio Community can use an unlimited number of users in an organization in the following cases: in classrooms, for research or participation in open source projects.
- For all other use cases: in non-corporate organizations, the Visual Studio Community can use up to 5 users. In corporate organizations (in which> 250 PCs are used or whose annual income is> 1 million US dollars), use is prohibited, except for the cases listed above (open source, research and academic audiences).

How is Visual Studio Community 2013 different from other Visual Studio releases?
- Visual Studio Community 2013 contains all the great features of Visual Studio Professional 2013, designed and optimized for individual developers, students, participants in open source projects and small groups.

Let's now consider what is the Express edition of Visual Studio 2013.
Visual Studio Express 2013 is the latest version of free developer tools that are defined based on the platforms for which the end products are created. Each tool supports Visual C ++, Visual C #, and Visual Basic.

Commercial use of Visual Studio Express :
Visual Studio Express products are available for free and can be used for commercial and production purposes in accordance with the license terms of each product. For example, you can use Express for Windows to create applications and send them for sale to the Windows Store.

Bottom line: If you like Express, you'll love Visual Studio Community 2013. With all the features of Express, application development tools for multiple Windows, Android and iOS devices, and access to thousands of extensions in one integrated development environment, Visual Studio Community 2013 is the perfect tool. for private developers, students, open source authors and small groups.
