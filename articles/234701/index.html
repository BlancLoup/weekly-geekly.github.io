Launch the 64-bit version of Chrome for Windows

In June, we started testing the 64-bit version of Chrome for Windows. Until today, it was available only to Chrome Dev and Canary users, but now everyone else can install it too! To do this, simply download the 37 version of Chrome.

This version features even faster speed, stability and security. Our research shows that the browser began to work with graphics and media content faster. For example, a VP9 decoder used to compress HD video on YouTube has improved its performance by 15%. We also measured the stability of the 64-bit version - compared to the 32-bit version, it doubled when working with standard web content. In addition, security tools, such as Partition Alloc, are much more effective in protecting vulnerabilities from attacks that rely on certain locations of software objects in memory (in particular, they prevent the bug discovered in the Pwn2Own 2014 competition).

To appreciate all the advantages of the 64-bit version, open the Chrome page and click on the link "Windows (64 bits)". If you want to update Chrome for business, download the 64-bit MSI package . Please note that in the 64-bit version of Chrome there is no NPAPI plugin. They can still be used in the 32-bit version - until we stop supporting the plugin.

We hope you enjoy the new version of Chrome. Install it and share your impressions . Thanks to your feedback, Chrome will be even faster, safer and more stable.
