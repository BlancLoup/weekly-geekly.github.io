Nokia Belle has been updated

If you own a Nokia N8, Nokia E7, Nokia C7, Nokia Oro, Nokia C6-01, Nokia X7, Nokia E6 or Nokia 500 smartphone, we have good news for you: an update to the Nokia Belle OS is available for your device. Technically, the update for most devices appeared a week ago. Today, an update to the Nokia Belle appeared for the Nokia 500, covering the entire list of devices on Symbian ^ 3.

We already wrote about the merits of the new mobile operating system Nokia Belle in the posts ‚Äú Review of OS Symbian Belle ‚Äù and ‚Äú Review of Nokia 700 ‚Äù, but here is a brief list of the main innovations:
  • Variable widgets (now they can be five different sizes);
  • Drop-down menu that displays various notifications and controls connections. It shows notifications of missed calls and unread messages, you can select a quiet mode, as well as manage connection settings and WiFi and Bluetooth connections;
  • The number of desktops increased to six;
  • The lock screen has become more informative;
  • At the bottom of the main screen, three virtual buttons appeared to open the main menu, the numeric keypad for dialing (or selecting a contact) and the settings for the main screen;
  • The main menu in Belle received a one-level navigation, that is, applications go in one list;
  • There was a function to remove programs through the menu, and not through the application manager;
  • Improved interface for most embedded applications.

How to upgrade


To install the update, you will need to run the program Nokia Suite version 3.3 or higher (you can download it here by selecting your phone from the list) and connect your smartphone on Symbian ^ 3 to your computer via USB cable, following the instructions on the screen.

There should be no difficulties with installing Nokia Belle, but if you're interested, you can watch the video tutorial for this update.
