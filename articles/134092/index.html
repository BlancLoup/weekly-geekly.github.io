IntelliJ IDEA 11


As usual at the end of the year, a Java development environment update appeared - IntelliJ IDEA 11 .

The new version is primarily characterized by a completely new, more compact user interface for all supported platforms.

Thanks to the rewritten architecture of the editor, it is possible to work efficiently even with very large files. Also accelerated the indexing of projects and launch IDE.

Much attention is paid to the functionality specific to the supported platforms, such as, for example, full-screen mode on Mac OS X, or symbolic links to Linux.

Naturally, a huge number of other features have been added and improved. - such as support for the Play framework, the latest versions of Groovy (2.0) and Grails (2.0), new technologies for Spring: Spring Testing, Spring Integration, Spring Data (JPA), etc. Added the ability to quickly create and configure projects / modules based on Gradle- projects. There is full support for editing CoffeeScript, with code analysis, auto-formatting, etc.
stackoverflow.com/questions/10185858/how-to-hide-files-outside-intellij-project-but-in-same-mercurial-repo
IntelliJ IDEA 11 is available in two versions - Commercial Ultimate and open-sourced / free Community Edition. IntelliJ IDEA Community Edition is becoming an increasingly popular development environment for Android. Version 11 adds the ability to automatically preview user interface layout files, as well as work with the latest version of Android SDK 4.0 (Ice Cream Sandwich).

For a more complete list of changes in both versions and to download IntelliJ IDEA 11, please visit jetbrains.com.
