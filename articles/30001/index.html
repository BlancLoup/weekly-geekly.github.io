Pending KDE 4.1

A few days remain until the release of KDE 4.1, one of the most anticipated open source releases of this summer. In this regard, we will try to remember what KDE 4.0.x gave us and what should appear in 4.1.

Immediately make a reservation that this is my first post on Habr√©. If something is wrong, then do not scold.

So KDE 4.0 was released in early January of this year. It was intended solely for developers, this was repeatedly mentioned on the official website of the project. The main architectural difference from the 3.x series is of course that it was written using the Qt4 library. This gave a significant increase in performance and stability. Appeared Plasma - a new means of organizing the desktop, able to display widgets. In the first releases (up to 4.0.3), Plasma worked extremely unstable. Phonon took over all the work in multimedia files (later it was included in the OpenSource version of Qt). The standard file manager was Dolphin, which, however, did not receive much of Konqueror's functionality (for example, integration of applications into the context menu). Nevertheless, KDE 4.0 has become popular among ordinary users, thanks in large part to its attractive design.

What awaits us with the arrival of KDE 4.1?
Firstly, Dolphin functionality has been significantly increased. Now that same integration of applications into the context menu has appeared. Improved support for metadata, thanks to the Nepomuk project. Now, if you open files with one click of the mouse, then you will have no problems selecting one file. When a file gets into focus, a nice "+" sign will appear in the upper corner of the icon. Finally, the much-needed ‚ÄúUp‚Äù button appeared, it is unclear why it was absent in previous builds. The contents of the folders can now be viewed as a tree, before that there was only a list and icons.

The Gwenview image viewer has undergone major changes. They mainly concern the appearance of the program. However, there are changes inside. For example, thanks to improved algorithms for working with JPEG, the download speed of large photos has significantly increased.

Plasma changes are also not spared. The Kickoff menu was heavily reworked and got a new look. Changes have undergone the quick start program Run Command. Kwin's window manager got a full set of effects and is now in no way inferior to Compiz.

Kontact reappeared, beloved by many KDE3 users. Let me remind you that Kontact is an excellent program for organizing personal information, combining a calendar, TODO-list and much more. The project has been completely rewritten to Qt4, new useful features have also been added: KTimeTracker and KJots (a tool for distributing time and creating notes)
It is unclear whether the Amarok 2 music player will be included in the release. Now the project is in Alpha 2 stage.
I am very pleased that KDE lives and is very actively developing. And the development is very correct. Developers pay great attention not only to the functional, but also to the design of their offspring (although probably one cannot exist without the other). All applications are decorated in one and very nice style.
I also remind you that KDE 4.1 will be available to Windows users. This will probably further increase the popularity of the project.
Thanks to all. After the release I will try to write what we got in reality.
