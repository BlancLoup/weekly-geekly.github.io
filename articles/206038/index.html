SteamOS available for download



Starting today, a preliminary version of SteamOS 1.0 (Alchemist) is available for download.
Valve's operating system is based on Debian (Wheezy 7.1) and is tightly integrated with the Steam service.

System requirements:
  • CPU: Intel or AMD 64-bit
  • RAM: 4GB
  • HDD: 500GB
  • GPU: NVIDIA (AMD and Intel are not supported yet)
  • Boot: UEFI
  • USB port for installation

Three installation methods:
The easiest one is to restore from an image (2.4Gb)
Complicated - use Debian Installer (960Mb)
The most difficult is to build the most from the repository

Answers to questions and a detailed description can be found in the official FAQ .
