What's new will be in Dropbox

Hello, hello.

Of course, many are aware of the existence of Votebox , part of the Dropbox site, in which users can request specific innovations in Dropbox and vote for them. I get information from the section In Progress .

The list of functional implemented now, in descending order of the number of votes:
  1. Ability to synchronize folders not only in the folder "My / Dropbox".
  2. Set read / write permissions on shared files and folders to prevent users / owners of these objects from deleting / modifying files and folders.
  3. Optimize the use of RAM. This item is unlikely to ever disappear.
  4. Protect certain folders with a password.
  5. Localization Dropbox. Japanese, German, Spanish and French are currently available for the web interface (beta). More on the forum .
  6. Business version with a large number of storage options and the ability to work multiple users with different levels of access.
  7. The ability to set a password for access to a public folder (or subfolders) so that only a password-aware person can access them. This will allow you to make a selective exchange with users not registered in Dropbox.
