Joost flash clones went into circulation

TechCrunch writes about Joost clones. The revolutionary video service has not yet emerged from the beta version, but has already become a source for copying. It is interesting that not so much the P2P technology on which Joost is based is cloned - it seems that it is not very easy to recreate it - namely, the appearance and functionality of the interface for viewing online video. How much is reasonable - time will tell.

The first of the clones is not a commercial startup, but an experimental development in which the Joost interface is implemented in flash.

Recently it became known about the Spanish clone Joost called DNAStream . It is also made on flash, and to watch TV channels you do not need to download and install any program. The player interface copies Joost almost one hundred percent, but the developers do not say anything about it and position their development as a commercial startup.
