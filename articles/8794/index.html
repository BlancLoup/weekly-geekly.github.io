New Technorati: now not only blogs

Technorati's blog search engine has been radically updated: the changes have affected both the content and the look.
Technorati is now not focused only on blogs, but also searches for multimedia data. As the CEO of Dave Sifry noted in his blog , such an update is caused by a changing audience - the service uses an increasing number of ordinary Internet users.

UPD: News already published here .

The first thing that catches your eye is the complete redesign of the main page:



In Technorati, there are no special search queries now (search by keyword, search by tags, search by blog directory), instead a single search string is made. All queries of the type technorati.com/tag/query return search results for blogs, videos, audio, photos right on the first page. Using tabs, you can limit the search results for any particular data type.

However, the ability to search only blogs saved: http://search.technorati.com .
