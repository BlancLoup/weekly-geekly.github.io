Programming for iOS. For professionals

Yesterday I wrote about programming for Android and therefore as a counter-argument I cite as an example the book " Programming for iOS. For Professionals " This is the third edition of the book with significant additions.

This book, coming out in the third edition and awarded the Jolt Productivity Award 2012 award according to Dr. Dobb's Journal, will teach you all the nuances of developing applications for iOS devices (iPhone, iPad and iPod Touch). Based on the training course of the famous educational IT project Big Nerd Ranch, this practical guide contains the key concepts of programming in iOS, many examples of code and application development exercises. The new edition covers versions of iOS 5 and Xcode 4.3.
