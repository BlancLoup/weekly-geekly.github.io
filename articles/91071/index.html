Events on HabraHabré

I propose to implement a feature that, in my opinion, will make registered habra people as happy as Schwartz on this photo.



1. The “events” link is added to the main page (for example, before “personal mail”).

2. This page contains a list of elements (for each user unique) that have the format: “topic name” (link), “comment author”, “comment text”.

3. The list is updated in three cases:
a) commented on the topic created by the user
b) someone replied to the user's comment (if the topic was not created by the user himself)
c) the topic to which the user subscribed was commented

For example, Peter comments on my topic “Fedora 13 has been released”, in which case I receive an event (“Fedora 13”, “Peter”, “Will it start under Slackware?”).

4. To subscribe to a topic, add a “follow comments” button to each topic, for example, in a line with news / date weight, etc. To unsubscribe from the topic - the same button.

5. When, say, five events appear, the “events” link turns into “events (5)”. After opening this link, it turns back into "events".

6. Right hang advertising.

This feature was implemented some time ago on news2.ru , and turned out to be very convenient. It became easier to communicate → they began to communicate more. I know that in the “settings of the site” Habrahabr can be done so that the “events” a) and b) are sent to the mail, but this is not very convenient.

There is a suspicion that the "event" page will become the most visited among the registered habra people after the main one.
