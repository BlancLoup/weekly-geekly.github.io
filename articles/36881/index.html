Movable Type 4.2 released

Today, the long-awaited release of the new version of Movable Type 4.2. In addition to the fact that the speed of the engine has been significantly increased in this release, it is also the safest of all existing releases. New features that appeared in MT 4.2 were more than once considered on the MT.ru site during the release of preliminary versions . Now we can note the main improvements and innovations:
  • Branched comments are now at the system level, no plugins are needed.
  • The anti-spam tool - TypePad AntiSpam , is included in the distribution.
  • Preview templates .
  • Improved page management templates and widgets .
  • Simple, very simple templates for quick and easy site customization.
  • Native OpenID 2.0 support.
  • Performance logging will allow you to see which publication takes the most time and needs optimization.
  • Automatic intelligent SSI (Server Side Includes) with support for PHP, ASP, JSP, and SHTML.
  • The template caching module automatically caches frequently used parts of templates (widgets, for example) for instant publishing.
  • Publish profiles allow experts to create and manage publish settings for peak work.
  • Caching information (for example, about the number of comments and trackbacks).
  • Improved database indexes and query optimization.
  • Reduced memory consumption. Basically, due to the closure of memory leaks and optimization of the code, which was divided into small parts for simple tasks.
In the near future, the FaceBook Connect plugin should be released, with the help of which FacebookBook users can log in to the Movable Type blog.


Movable type pro

Probably, this news will be for someone more important than all previous ones. Because from today there will be no more confusion between the functionality of MTOS and Commercial versions. There will be only 2 versions: MTOS and MT Pro. And the Pro-version in most cases is free . And it includes the following:
  • Community Pack (the one that used to cost about $ 10,000)
  • Professional Pack (Custom Fields and Templates Sets)
Thus, each blogger using Movable Type can create not only a blog, but also build a full-fledged social network (user profiles, ratings, shared blogs, forums, etc.). Both versions of Movable Type are distributed with open source codes. all for free for bloggers:
  • unlimited number of blogs;
  • unlimited number of authors;
  • unlimited number of community members;
  • all in unlimited quantities.
Six Apart is going to earn on those bloggers whose websites create a business using a blog (a blog store, for example). Such users are offered a business license and professional support (for MT Pro).
