Release MDC 1.0.2.1 beta

Today the next MDC release took place. At this time, saw the light of version 1.0.2.1 beta. Long and thorny was the way to it. We tried to take into account the opinions and comments expressed by our users on bugs.mdc.ru. In this release, I would like to highlight finally appeared versions for Mac OS X and FreeBSD. Users of version 1.0.2.0 win32 have the opportunity to upgrade to 1.0.2.1 using our auto-update system.




Below is a list of changes included in the release:
  1. Mac OS X version;
  2. FreeBSD version;
  3. the opportunity to download packages for Ubuntu 32 and 64;
  4. an opportunity to download packages for FreeBSD 32 and 64;
  5. bug fix update for win32 # 79 ;
  6. the mechanism of working with local and server settings has been improved;
  7. reduced MDC startup time under Linux;
  8. fixed bug with proxy settings;
  9. fixed bug with links in history window # 116 ;
  10. load history indication is now displayed in percent;
  11. the ability to choose your avatars from the collection;
  12. fixed bug in font settings;
  13. fixed # 54 popup bug;
  14. fixed bug with non-vanishing login window # 108 ;
  15. fixed bug when adding an account;
  16. password for ICQ is cut off to 8 characters # 129 ;
  17. Added an item to the tray menu, a list of contacts that opens the contact sheet window;
  18. the contact list is now shown by a single click on the tray icon
  19. Now, with the right (left with Mac OsX case) clicking on the icon of the new message, a menu appears in the tray with the list of senders of unread messages;
  20. fixed bug with visibility and behavior of windows for Linux, Mac OsX and win32 # 59 ;
  21. clicking on the tray icon with visible MDC leads to its folding (not yet for win32) # 89 .

On the way package for ArchLinux.
