W3C launches two new drafts

Today (July 23, 2009), the W3C announced the launch of work on two new drafts: the flexible box layout module and the images values module.

The flexible box layout module defines new keywords 'box' and 'inline-box' for the property 'display', which indicate that the element should be displayed in a column or row of descendant elements. Additional properties determine the order of elements (from left to right, from bottom to top, etc.) and how are the descendants and the space between them. The module is aimed primarily at implementing the same height and width of the rows of controls in the GUI.
