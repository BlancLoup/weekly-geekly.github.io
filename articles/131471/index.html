Ajenti 0.6

We welcome you again on the waves of Changelog FM, and today in our studio - Ajenti 0.6.0.



We remind you that Ajenti is a web-based dashboard for server administration .

First, I would like to thank all the 560 users who are with us despite the fact that the project was until recently in the beta stage, as well as all of our wonderful developers ;)
The main news is of course almost full support for the following systems:


FreeBSD 8.0+




Gentoo




CentOS 6



We remind you that in addition, it supports Debian Squeeze + (and Lenny, with the addition of packages from squeeze), Ubuntu 10.04+ and Arch Linux.

Some changes




New Dashboard

Now the widgets can be moved by dragging, and each user can customize the dashboard for themselves.



Monitoring

Added a plugin for monitoring system indicators and their export via HTTP.



New terminal

Terminals open in separate tabs, a list of terminals with thumbnails appeared.


Totally made more than 16000 changes from version 0.5-13.1;)

Distribution form


The installation process is described in more detail at ajenti.org
  • Debian / Ubuntu : APT repository
  • CentOS / RHEL : Yum repository
  • Gentoo : via Sunrise overlay
  • FreeBSD : Download the DIY port from the home page.
