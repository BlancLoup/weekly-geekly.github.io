Mail.Ru officially opened ICQ sources for Windows and OS X (and announced a client for Linux)


According to the news on the site , ICQ has been completely rewritten on Qt, and therefore they post full source code on GitHub .

Previously, there was an unsuccessful attempt to lay out the sources, and private keys for Sparkle (update systems for OS X) were made public.

Now the repository is cleared, and in addition, a new build for Windows is already available for download . Versions for OS X and Linux (!) Are on the way, since Qt is a cross-platform framework and almost anything can be built.

Judging by the fact that they posted exactly one commit, the development of ICQ does not move to GitHub, and the repository is needed only to upload the source code of current versions, as required by the free Qt license. Although I would like to hope that the project will somehow react to pull-requests and issues on the githaba.

Well, let's hope for the further development of the OpenSource Mail.ru direction and wait for new forks and alternative customers!

PS: unfortunately, the VoIP module has remained closed and comes in the form of static libraries.
