Microsoft has released drivers for Xbox One controllers for Windows



Game controllers for the PlayStation and Xbox consoles are very convenient, and many gamers playing games for the PC platform, I would like to use these controllers not only with consoles. However, this was not possible before (there were some unofficial connection methods, but not many gamers heard / read about it).

Now Microsoft has released special drivers for Xbox One controllers (drivers for Xbox 360 controllers are available here ). Drivers allow you to use all the functions of the controller.

To connect the controller, you just need to install the driver by connecting the device to the USB port.
