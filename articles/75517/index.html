Tasty Gadgets for Google Wave


Not so long ago, managed to get an invite to Google Wave (thanks again gxcreator ). The thing is versatile and promising, which has already been repeatedly written / mentioned, especially in terms of teamwork - support for bots and gadgets. I would like to share information about the most interesting representatives of the last group.

Mind map gadget


Google Labs Page
XML Link to Gadget

Allows wave users to co-create, edit the simplest hierarchy of data / ideas. Allows you to add icons (for example, to prioritize tasks), as well as vote for each of the items. The clear interface and a minimum of functions.

Plany


Google Code Page
XML Link to Gadget


A good functional task scheduler. It gives the opportunity to monitor the implementation of a task, indicate the person in charge, the start and end dates of the implementation, as well as the state. Using the Google Charts service, it builds 2 schedules: how many man hours are allocated for a particular task and a Gantt chart (illustration of the plan, schedule of work on a project) A useful gadget for organizing the workflow.

Code Snippet


Link to the Wave with a gadget

Syntax Highlighting. Features:
- More than 30 languages
- Ability to block a fragment (read only) to prevent unwanted editing by others
- Ability to jointly highlight lines to assist in checking the code

Napkin


Google Code Page
XML Link to Gadget

Often something is easier to explain in a drawing than in words. Napkin - Google Wave gadget written in Adobe Flex, which allows you to very quickly make small sketches in your wave.

Retro chat


Google Labs Page
XML Link to Gadget

This gadget allows you to insert a compact chat into a wave.
