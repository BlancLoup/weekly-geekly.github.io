Sony released open source analysis tool APK



Are you developing applications and looking for a powerful tool for analyzing them? Do you need a comfortable view of their architecture? Have you tried to parse the byte code of Android programs? All this is able to new virtual application analysis tool ApkAnalyser from Sony, which comes with open source.

ApkAnalyser is designed to validate and verify the operation of your application. You can unpack, install, run and check the results of the Logcat command . With ApkAnalyser, you can also decode XML, identify potential vulnerabilities in your application, and more. ApkAnalyser is completely written in Java and will require JRE 1.6 or higher; Works on Windows, Linux and Mac OS X.

Before downloading and installing it, please read the detailed instructions on the Sony Mobile developer blog, for more information on the various features, visit the APKAnalyser wiki page .

We hope this utility will be useful in your work.
