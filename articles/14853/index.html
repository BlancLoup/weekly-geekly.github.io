Apple fights Caps Lock


Every computer user knows how often the Caps Lock key tends to be pressed at the most inopportune moment. And in the new Apple keyboards, the eternal problem of Caps Lock will be solved.

On new keyboards, this key will not work if you press it accidentally. And if the user still needed to type text using Caps Lock, it should be held for a relatively long time - a few seconds. The capital letters mode is also canceled. Thus, if the user accidentally presses a key, it will not work and the keyboard will not enter upper case mode.

Apple has struggled with Caps Lock before, it could be disabled through the Keyboard & Mouse panel, but the new solution, which appeared as a built-in keyboard, is beyond praise.

It remains only to congratulate Apple on a simple but very successful solution to the sore problem. And it remains to hope that such a decision will interest all manufacturers of keyboards.
