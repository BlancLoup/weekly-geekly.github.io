New version of Code :: Blocks IDE 10.5 released


A couple of days ago there was a new version (after 2 years of development) of a fairly popular open IDE written in C ++ and wxWidgets - Code :: Blocks . The main development language is C ++. There is a plugin system for extending the environment.

Supports a large number of compilers:
  • Mingw
  • Digital Mars C / C ++
  • Microsoft Visual C ++
  • Borland C ++
  • Watcom
  • Intel C ++ compiler

Some of the interface features are:
  • Code highlighting
  • Code completion
  • Folding
  • Plugin support
  • System toolbars (You can customize the view)

Download under the desired platform in the Downloads section.
Download list:
  • Sources of the latest version
  • Svn repository

Code :: Blocks 10.5 ChangeLog
The latest version has significantly improved the interface (now the environment does not fall, with any action with it). Many minor bug fixes. Significant improvement in code auto-substitution and debug mode (The first thing that caught my eye was that you can view the contents of the vector).
