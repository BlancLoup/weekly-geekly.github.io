Evernote updates for different platforms


Feedback has been and remains the key factor determining the vector of development of our product. Our experts, including the developers themselves, constantly monitor your feedback on Evernote not only at the official service forum, but here, at Habrahabr, and at other sites as much as possible. Wishes and comments are the basis of product development, one of our main landmarks. Of course, we cannot always respond to all your requests and desires as quickly as we would like, but still, the work on improving quality goes on continuously, and we do not leave any comment without attention. Any feedback to some extent is reflected in the plan, which we implement to the best of our resources in practice.

So, over the last week we released updates of Evernote software clients for three Android, Mac OS and Windows platforms at once, in which we tried to implement some of the popular requests and wishes of users, which we read both here in Habrahabr and in our forum and others resources. We hope that the new functionality will like.

Android


The new version of the client for Android (1.4) is devoted to bug fixes and a number of nice innovations. Download the latest version, as usual, through the update or on the Android Market.

1) A new search string allows you not only to search for notes, but also quickly find tags, notepads and saved queries for later filtering notes by them:


2) Improved work with the camera: now when creating a photo note, the standard application for working with the camera is raised, which means that now you have access to all the parameters and settings - zoom, brightness, focus and other parameters that are provided by the standard application.

About offline access to notes

We know how important for Android users is the topic of local storage of notes, and finally, we can officially declare: offline access to notebooks will soon appear on Android!

Windows


The main theme of the new version of the Evernote client for Windows (3.5.4) was the improvement in performance and stability. But besides bug fixes, there are some nice innovations that users asked us about:

1) Now the list of tags can be dynamically filtered by displaying only the currently relevant tags (that is, the tags assigned to the currently displayed notes):


To select this mode, right-click on the labels panel or on the View item in the menu and select Hide Unassigned Labels. After that, the labels panel will become dynamic and will show you only those labels that are associated with the filtered notes.

By the way, the function of hiding tags in a pair with a search for your Evernote allows you to implement frequently requested search for tags. If the feature is on, just start typing the label name in the search bar, and all the labels that do not contain the keyword or are not assigned to the notes containing this word will instantly disappear. This works especially effectively if you use unique names for tags.

2) A clearer display of the geographical location of the note:


When possible, Evernote adds geographic location information to your notes. In the new version, we learned how to convert the coordinates of a note into the name of a country and a city, which is much clearer.

3) Ability to view the history of editing notes directly from the client:


We recently added the ability to store note history for premium accounts, which allows you to revert to previous versions of your notes. The new version of Evernote for Windows now allows you to open the history of editing notes without leaving the application.

4) Work with tables

The new version significantly improved the work with tables. If earlier you could set table parameters only at creation, now you can add or delete columns and columns in existing tables.

Mac


The main theme of the release of the new version of the client for Mac OS (1.9) was the acceleration of synchronization. The longer Evernote exists, the more our users' accounts become, so we are starting to pay more and more attention to optimizing the performance of large accounts.
