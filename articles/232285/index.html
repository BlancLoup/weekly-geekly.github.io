Get acquainted! Eric The Epic Elephant

It will be about the development of our Russian company A-Real Consulting LLC, namely, the Internet access control system - Internet Control Server (IKS) , with such a funny name ‚ÄúEric the epic elephant‚Äù.
IKS is a universal Internet gateway with tools for protection of a corporate network, traffic accounting, access control, mail, proxy, file server, Web, jabber and even IP telephony organization.
The core of this system is FreeBSD 8.4 with the ZFS file system.

Let's see what features are used.

  • Universal Gateway. Support for various connections: Internet, Wi-fi, 3G, PPTP, PPPoE, L2TP, works with VLAN and DMZ, as well as, it is possible to organize VPN tunnels (cPSEC PSK encryption and OpenVPN).
  • Access control. Authorization of users by IP address, MAC, name / password, AD and many others
  • Network security. Firewall, built-in antiviruses from Dr.Web, Kaspersky, ClamAV. DLP module - to protect against leakage of confidential information.
