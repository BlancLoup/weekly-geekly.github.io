Opera 17 release

Today, Opera 17 was transferred from the development branch to a stable one, and the browser was automatically updated for everyone who uses the latest Blink releases, which is reported in the team‚Äôs official blog . The release includes the following features announced back in the summer.

  • Return of fixed tabs (Pinned tabs)



  • A choice of action at startup has been added to the settings - earlier Opera has always opened the same tabs that were in the browser at shutdown, but now you can more accurately outline the behavior at startup.
  • Return of customizable search engines - we can again add any additional searches along with their aliases for the address bar. The developers mention that in the future they will also return the corresponding option to the context menu when clicking on the search form.
  • The browser now uses Chromium / Blink 30 (1599.69), and new APIs are available for extensions:
    • chrome.bookmarks. *
    • chrome.commands. *
    • chrome.webNavigation. *

  • Full list of changes and fixes

Active development of the next versions continues, and very soon Opera 18 Next will be shown to us. Promised improvements in syncronization and tabs, as well as the return of normal bookmarks. On the screenshots from the developer's blog, there is also a Quick Access Bar.
