Running Android apps on Windows, OS X, Linux on Chrome



Google has recently added Android apps to the Chrome Web Store directory, allowing you to launch a number of such apps on a Chromebook or Chromebox. Not so long ago, several developers reported that it is possible to run other Android applications, not from the Chrome Web Store catalog. However, before launching such applications need to be prepared in a special way.

With this preparation, Android applications can be launched not only on a Chromebook, but also work with them on Windows, OS X, and Linux. Moreover, applications run as native.

For processing applications before launching, ARChon is used - a modified version of Android Runtime for Chrome. ARChon allows you to run multiple Android applications at the same time, in Google Chrome browser 37 or later.

The developer who created the ARChon claims that the applications are fairly stable, although there may be problems. However, thanks to ARChon, Android applications can be run on a PC running Windows, OS X, or Linux without using emulators or virtual machines.

Project sources and details can be found on ChromeOS-APK and ARChon Custom Runtime .
