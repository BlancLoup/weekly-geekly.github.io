Release Rust 1.0 Alpha

I was surprised to find out that this event, expected by many, somehow bypassed Habr. I think that for many programmers who follow this programming language, it will be useful to know that the very release of version Rust 1.0.0 Alpha took place on January 9th . In this version, the core of the language and most of the standard library have finally been stabilized, so now you can start writing programs on Rust without fear that in a week something will break after updating the compiler.

From myself I can add that the language is really interesting and, at least, worth the attention.
