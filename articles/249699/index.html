The short answer is simple and monosyllabic: "remotely." Skype + TeamViewer. All necessary tools must be installed on the student's computer: Apache, PHP, mySQL or PostgreSQL, code editor. At the appointed time, we communicate in Skype, I connect via TeamViewer and the ‚Äúprocess has started‚Äù. As a result, all codes remain with the student. By the way, about the forums. This is also very useful for learning! But not just read, although it is also useful. Try to find the answer to the question asked by someone.
