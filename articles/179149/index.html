MikroTik - quick setup of the access point

Very often, the current "System Administrators", there is a problem with setting up 802.11 equipment, I will try to explain at an intelligible level.


In this post I will try to tell you how to set up a regular access point on MikroTIk 751,951,2011 etc 802.11 b / g / n hardware.
