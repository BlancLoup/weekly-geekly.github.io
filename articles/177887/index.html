The new BeagleBone Black is available for order, for only $ 45


From now on, the new BeagleBone Black is available for purchase, for only $ 45. BeagleBone Black is a community-supported developer platform. Allowing you to load GNU / Linux in 10 seconds and start development within 5 minutes using a single USB cable. To whom the characteristics have become interesting, I ask for a habrakat.
It is worth mentioning the support of the AM3359 processor for hardware acceleration of encryption and the presence of two PRU (Programmable Realtime Unit) modules, but I can add that it is a very useful unit. More information about the PRU modules can be found on the Texas Instruments wiki . But all this was in the previous version of BeagleBone, let's take a look at the characteristics of the new BeagleBone Black:

Device Features:

  • AM335X 1GHz ARM Cortex-A8 main processor
  • SGX530 graphics processor
  • 2x PRU 32-bit RISC CPUs
  • 512 MB DDR3L 800MHZ RAM
  • 2GB flash memory soldered on board (eMMC)
  • 2x High Speed ‚Äã‚ÄãUSB 2.0 ports: Host and Client
  • Serial port UART (RS-232) TTL, 3.3V
  • Microsd
  • HDMI
  • Ethernet
  • two 46-pin connectors


I consider the main feature of this board to be the 2GB flash memory sealed on the eMMC board and the appearance of a micro HDMI connector. It is a worthy competitor Raspberry Pi. Beautiful picture for a snack.
