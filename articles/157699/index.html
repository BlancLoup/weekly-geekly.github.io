Aichi-Event.rf - all IT events of Runet!

Today, we are launching our new project , IT-Event . RF , which is a portal covering events in the field of information technology, robotics and telecom in Russia, Ukraine and Belarus.


Now about 140 events have already been added to the site and every day we publish ~ 5-10 new ones.
To date, we have implemented almost all the planned basic functionality of the site. The current state of the portal is beta version.

This is a tab for events that are discussed on Twitter under your hashtag (or several). Through the API, we knock on twitter, get tweets on the hashtags we need, check for spam and censorship, save them in our database and then output them on the pages of events to which these hashtags are tied. Event news


In fact, this is the same as Twi-broadcast. Displays the news before the event, in a special tab "News", the period of their update 3 hours. In the news, the organizers can inform the participants about the course of the preparation of the event by means of twitter.

Tabs


In total there are 6 types of tabs:
  1. Announcement
  2. Program
  3. Speakers
  4. news
  5. Broadcast
  6. Results

The Totals tab, in turn, can consist of 4 sub-tabs:
  1. Report
  2. Reviews
  3. Gallery ( photos and video )
  4. Presentations

In the Announcements, Program, Speakers and Report tabs there is a built-in editor CKEditor, but for a slightly more advanced public it will be nice to know that in these tabs you can fully use the HTML markup language. For all events, only the Announcement tab is mandatory, all other tabs are displayed if they are NOT empty.

Add event


An event can be added by any registered user, after which it will be submitted for moderation.
If you know about an event that is not on our website, send us a link (contact@it-sobytie.ru) and we will add it ourselves.

Edit existing event


If there is already an event on the site that you are organizing, register and contact us (contact@it-sobytie.ru) so that we can give you the right to edit it.
