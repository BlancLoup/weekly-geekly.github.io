Big update f.lux program

After several years of development, the well-known f.lux program has finally been updated to version 3.9 (without beta). It implemented a large number of new features, some of which were not enough before.



Recall that f.lux is a program that changes the color profile of a monitor, depending on the time of day. At night, warm color tones are set up, which correspond to artificial lighting in the apartment, cold ones during the day, corresponding to daylight. Thus, the eyes are much less tired, because the glow of the monitor becomes more natural and corresponds to the environment.

Most importantly, in the new version, the restriction on the minimum color temperature of 3400 ¬∞ K has been removed. Now the temperature can be reduced to 2700 ¬∞ K or even less, if you so wish.

It is very convenient that support for hotkeys is implemented to quickly lower or brighten the monitor: Alt + PgDn, Alt + PgUp. Additional combination to instantly disable f.lux: Alt + End.

Movie mode appeared: in this mode, the color temperature is reduced, but not much to preserve the detail in the shadows, the natural colors in the film.

Added support for color profiles from hardware calibrators.

New mode "turn off before the morning."

The new regime of "dark room" to work in absolute darkness: the color temperature is reduced as much as possible, the screen becomes very red.

Added a world map to pinpoint your location before determining the time of dusk and dawn.

Support for notebook mode with external backlight.

Collaboration support with Philips Hue programmable lamps.
