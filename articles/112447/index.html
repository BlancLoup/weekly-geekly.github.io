VLC Media Player 1.1.6 saw the light

Two months after the release of the previous version of VLC 1.1.5 and after its more than 58 million downloads , a minor release of the excellent cross-platform free multimedia player VLC 1.1.6 took place.

First of all, I would like to introduce small innovations and important corrections concerning safety as well (in particular, a buffer overflow in the Real demultiplexer ). So, what's new in version 1.1.6:

  • Security updates for codecs and demultiplexers
  • Added support for RTP / PCM 24bit, audio / L24
  • Added support for MPC SV7 / SV8 for Windows and MacOS
  • Increased decoding speed Webm / VP8
  • Added support for Midi for MacOS
  • Important updates for most language translations
  • Added support for projectM visualization for Windows
  • Improvements in KDE-PulseAudio interaction
  • Subtitle fixes and improvements
  • Improved visualization and program interface
  • Codec updates
  • and many other various fixes

The official site can be used to install the most recent version for Windows , MacOS , iOS and other platforms .

Fresh source code also lies in sight of everyone .

Ubuntu Maverick users can indulge themselves with PPA:

sudo add-apt-repository ppa:ferramroberto/vlc
sudo apt-get update
sudo apt-get install vlc mozilla-plugin-vlc

or wait for the update of the official universe repository and update:
sudo apt-get update
sudo apt-get install vlc vlc-plugin-pulse mozilla-plugin-vlc

PS In order to play locked DVDs in Ubuntu, you need to install libdvdcss2. To do this, you can follow this instruction or add " medibuntu " to your repositories and thus install libdvdcss2.
