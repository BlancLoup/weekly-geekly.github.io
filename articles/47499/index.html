GTARadio 4 Just In Time

We, as promised, did what many asked. Namely, support for Webrama just released GTA4. Since the version for GTA3 / SA , nothing has really changed, except that GTARama now stores its cache in the user directory, not the system directory.



In general, download , put in the directory with GTAIV.exe, launch, select a genre / mood (see below), launch GTA4, sit in the car, turn on IndependentsFM and enjoy.



How to choose the mood and genre



With GTARame running, use the following keyboard shortcuts:

Mood
CTRL + F1 - Fun and quiet
CTRL + F2 - Fun
CTRL + F3 - Fun-energetic
CTRL + F4 - Calm
CTRL + F5 - Neutral
CTRL + F6 - Energetic
CTRL + F7 - Sad-calm
CTRL + F8 - Sad
CTRL + F9 - Sad-energetic

Genre
ALT + F1 - Any
ALT + F2 - Rap
ALT + F3 - Rhythm and Blues
ALT + F4 - Rock
ALT + F5 - Classic
ALT + F6 - Electronics
ALT + F7 - Alternative
ALT + F8 - Jazz
ALT + F9 - Metal

Download GTARam.
