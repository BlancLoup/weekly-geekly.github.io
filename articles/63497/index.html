NetSetMan - a program for quickly changing network settings on a laptop

Laptops are firmly established in the life of mankind. :)
When moving throughout the day, it is often necessary to change the network configuration to access the network. During the day, you can visit at home, at work, at the university, at a cafe at lunch, and in many of these places you have access to the internet, you just have to constantly change ip-schniki, dns, gateways, etc.
In order not to do this manually, I suggest that you familiarize yourself with a convenient and free program for changing settings in 2 clicks.



Are you tired of changing the network settings on your
laptop or computer every day? Tired of the same procedure?

On the move: change ...
Houses: change ...
In the office: change ...

Then NetSetMan is what you need! NetSetMan will do the job for you.

- welcomes the site of the program ( http://netsetman.com ).

The author of the program from Germany.
The program interface is translated into many languages, including Russian.
NetSetMan allows you to save six profiles and, if necessary, quickly switch them.



In the program settings you can specify the launch at system startup (even immediately minimized to tray).



It is not necessary to manually drive in the settings, it‚Äôs enough to set the necessary parameters once (in each of the places of stay) and in the program menu click ‚ÄúGet all current settings‚Äù



The program is minimized to tray, and when you hover the mouse cursor over the icon, it shows all currently used settings



Additionally, in the settings for each network adapter, you can specify the name of the computer on the network, domain, assign a workgroup, connect network drives, run scripts, and even disable the adapter if necessary.

Now you do not need to constantly change the settings by hand.
For personal use, the program is provided to the user for free. When extracting any commercial benefit, the program must be purchased ( http://www.netsetman.com/ )
