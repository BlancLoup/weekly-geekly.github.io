Microsoft introduced Windows 10



A few hours ago, Microsoft introduced a new Windows operating system, which will replace Windows 8.1. The operating system is called Windows 10.

The new OC is positioned as one, common operating system for a huge range of different devices.

Single Windows for all devices


Windows 10 will be installed on a wider range of devices compared to previous versions of OC. These devices include not only personal computers, laptops, tablets, but also smart phones, televisions, various boards, and industrial devices used for Internet of Things scenarios.

The new OC will also be a single platform for developing applications and a single channel for delivering applications through the Store .

Improved user interface


The operating system interface, on the one hand, remained familiar, on the other hand, acquired several new useful features.

Start menu

Opened by clicking on the logo in the lower left corner of the desktop. The left column of the menu contains shortcuts to the most recently launched applications, the right one contains the tiles of universal applications . The menu also contains power management buttons and an updated search function, to search not only on the local system, but also on the Internet.



Universal application windows

Now both classic desktop applications and new universal applications run in their own window on the desktop.



Improved Snap function

Work in several applications at once became easier and more convenient. Now Snap allows you to open up to 4 applications on the screen at the same time, and Windows tells you what other applications are running and how they can be placed. Windows 10 will help to more efficiently use unallocated screen space for other applications.



Many desktops

The user can create multiple desktops and place application windows on them. You can display all created tables on the screen and switch between them.



Multitasking

A new Task View button has appeared on the taskbar. When you click on it, a window of all running applications appears in the reduced view.

Special attention to corporate users and data


When creating Windows 10, special attention was paid to corporate users.
The new OS will be compatible with all traditional management systems that organizations use.

The operating system contains new features for the separation of work and personal data on devices of all types, which will allow to implement the concept of BYOD in organizations.

Try new windows


On October 1, the official Windows 10 page will be able to download the Windows 10 Technical Preview .
Do not miss the opportunity to look at the new OC with your own eyes and try it yourself.
Information about the release date of the operating system and the conditions for upgrading to Windows 10 will be announced later.

UPD: A preliminary version of Windows 10 is already available for download .
