Feedly opened API for all RSS clients

So far, the Feedly infrastructure has been working with approximately 50 RSS applications, but now the company removes all restrictions and opens the API for all developers .



Any RSS reader or mobile app can use the Feedly backend to access millions of feeds and sync via the Cloud API .

Representatives of the company say that they have worked closely with developers for half a year to design simple and convenient software interfaces, at the same time flexible enough to serve different classes of applications: readers, vertical corporate applications, dashboards, integration hubs, vertical communities. In June, Feedly opened the API , but only for individual RSS clients, the list of which was gradually expanded.

"After the public opening of the API, we expect the emergence of applications and services that we at Feedly could not even imagine." The company promises to tell about the best and most unusual applications in the corporate blog.

Cloud API is not only access to feeds, but also a powerful platform for personal work with information. After authorization, users can work with categories, tags, filters, selections, and so on. Through the API, you can access all these functions.

The user's personal preferences work regardless of which device he comes from: from an Android client, iPhone or desktop computer, and which applications he uses on each of these devices.
