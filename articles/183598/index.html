One of the computing nodes Tianhe-2

Tianhe-2 is based on Intel Xeon processors. The computer consists of 16,000 nodes, each of which contains two Xeon IvyBridge processors and three Xeon Phi processors. He has 3,120,000 cores and 1.4 petabytes of RAM. Supercomputer running Kylin Linux. The communication between the nodes is carried out using a specially developed proprietary network TH Express-2, the heart of which are thirteen 576-port switches based on ASIC chips made on 90nm technology, each of which has a throughput of 2.56 terabits per second.
