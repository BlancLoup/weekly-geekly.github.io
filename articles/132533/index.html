Yii 1.1.8 - download files with arbitrary extension using CMultiFileUpload

Good all the time of day!

Recently, when making a website on Yii, we found such a hole in uploading files to the server using CMultiFileUpload (for which special thanks to Mx21 ): you can take a file with an arbitrary extension, add something like .jpg to the end of the name, and then fill the server with CMultiFileUpload widget . The version of the framework used by us is 1.1.8.

It is treated as follows:
In the /framework/web/js/jquery.multifile.js file, you need to fix lines 222-223:
