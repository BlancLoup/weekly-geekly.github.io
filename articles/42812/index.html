Visual Design Tools

Everyone knows that the design stage is very important for successful development of a website / web service / program (underline the necessary). A competent development plan is half the entire project. It is clear that first of all a technical task is needed in which requirements, terms, etc. will be described. But visual design is even more important, namely: UML diagrams, database schemas, user interfaces, page thumbnails ...

To solve each of these problems, there are own means.
For example, user interfaces can be depicted on paper or in Photoshop. But, you see, it would be much more convenient to be able to create all of the above in one program.

Below I offer an overview of several such programs with a brief annotation. I really hope to see in the comments links to other software that allows you to perform such a comprehensive visual design.

Microsoft Visio


License: Trial
Cost: 18,000 rubles.
Website: http://office.microsoft.com/ru-ru/visio/

MS Visio 2007 is a universal tool for building diagrams, graphs, charts of any kind. It includes powerful visualization tools and is suitable for a wide range of tasks.

Open Office.org Draw


License: FreeWare
Cost: free
Website: www.openoffice.org/product/draw.html

Although OO Draw is a vector graphics editor, many use it to draw drawings, flowcharts, and other types of diagrams.

Axure RP Pro


License: Trial
Cost: $ 585
Website: http://axure.com

Axure is sharpened by the creation and rapid prototyping of web site frameworks. The interface is similar to MS Visio. Contains a very large library with a set of objects for page design and user interfaces. It allows you not only to create a graphical representation, but also to link events, link blocks with each other and much more. Extensive ability to export the finished scheme.

SmartDraw 2009


License: Trial
Cost: $ 197
Website: http://www.smartdraw.com/

Very powerful tool that allows you to create all possible types of interfaces. The interface is similar to MS Office 2007. A giant library of objects (additional elements are available on the off site), and various types of diagrams.

Gliffy.com


License: ShareWare
Cost: $ 5 per month
Website: http://www.gliffy.com/

gliffy.com is a web application written in flash. It has in its arsenal tools for creating flowcharts, structures, user interfaces, uml diagrams. Represents a means of convenient publication and joint development of documents. There are import possibilities both in raster and vector graphics. In the free version there is a limit on the maximum number of documents - 5.
