Reports available from PDC '09



Reports from the PDC conference became available to everyone. There are a lot of reports, 90 pieces, all of them are presented in HD-quality video plus optionally available video of lower quality. Presentation slides are available for many reports.

Themes of the reports cover a huge mass of technologies, both new and familiar to developers. Here are some of them:
  • Silverlight 4, WPF 4, Windows Workflow Foundation 4, Smooth Streaming
  • DirectX, DirectX 11
  • UML, DSL
  • Entity Framework, ADO.NET Data Services
  • C # / VB.NET: Futures, Garbage Collector
  • ASP.NET: Futures, MVC, AJAX Library and jQuery
  • C ++, Visual C ++ 2010
  • Software + Services Identity
  • REST Services Security
  • Windows Identity Foundation
  • Windows 7: Touch, Sensors, Ribbon, Error Reporting
  • Windows Server: 2008 R2 (I recommend the Mark Rusinovich report ), HPC Server, DryadLINQ
  • Windows Server AppFabric
  • SharePoint 2010, SharePoint Server 2010
  • BizTalk Server
  • SQL Server 2008
  • Azure: SQL Azure, Java to Azure, PHP and MySQL to Azure
  • Expression Blend, SketchFlow
  • Feeddler
  • Visual studio 2010
  • .NET Micro Framework
  • Code Contracts, Pex
  • Rx Reactive Framework
  • Dynamic Languages
  • M Language
  • Unified communications
  • Dallas
  • Powershell
  • Application Server Technologies: Present and Future
  • Perspectives on the future of programming
  • The State of Parallel Programming
Material for many, many hours. I think everyone will find a lot of new and interesting things for themselves.
