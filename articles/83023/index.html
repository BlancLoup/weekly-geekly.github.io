Re-flashing USB modem ZTE MF627. The continuation of the post Mypuka

Continuing the theme flashing USB modem ZTE MF627 .
Good day to all users of habrahabr!
So I decided to be a non-vegetable and share my experience in unlocking the device: ZTE MF627 USB modem. And at the same time get an invitation to register on the site.
After you perform a series of actions indicated by me, you can make calls, send SMS messages and have access to the Internet. (All these functions will be available from any operator).
To unlock, we need: 1. Firmware from GLOBE - MF626UpdateTool_GLOBE_PHV1.2.exe Downloading (Instructions for its use are here )
2. The program for working with the file system QPST_2.7.323 Downloading
3. File from modem file system PCUI.VN Download
4. Fresh firmware from MTS 626 627
Let's start ...
Attention!!! All actions are performed with a sim card pulled out !!!
1. Run the firmware from GLOBE, press the "DOWNLOAD" button and wait for the completion of the PROCESS OF THE FIRMWARE ...;
2. We distort the device (just in case);
3. Install the manager from GLOBE, launch it, after the drivers are installed, close it; (Details 1-3 are discussed in detail in the post ZTE MF627 USB Modem Reinsertion )
4. Install QPST_2.7.323 and launch the QPST Configuration, click the ‚ÄúAdd New Port ...‚Äù button and enter the port where we have the ‚ÄúZTE Diagnostics Interface‚Äù, for example - com6 (see which port you have on the ZTE Diagnostics Interface "can be found in the Device Manager in the" Ports (COM and LPT) section, click "OK", then select the menu item "Start Clients" => "EFS Explorer", click "OK" in the appeared window. A couple of seconds before you A window will appear with the file system of your modem, in the right window we find the file PCUI.VN and delete it, then copy the file that you downloaded from the attachment, for we cover the program.
5. Run the downloaded firmware from the MTS, press the "LOAD" button, wait for the completion of the firmware ... click "OK"
