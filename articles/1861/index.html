Google Ajax Search API updated

Google has released an updated version of the Ajax Search API.

The preliminary release of Google Ajax Search API 1.0 will replace the version with the sequence number 0.1, released in June of this year. The tool allows not only to provide a web page with a search form, but also provides output of results on the same page, without going to another site. In addition, the software product can be used to develop your own applications.

Information search is carried out through the services of Google Search, News, Maps, Video and Blog. The maximum number of returned results is limited (no more than eight) so that users cannot create full search services based on the Ajax Search API.
