The fact that modern high-tech toys is a big plus, since children can play learning the basics of electronics and programming. One of these cool toys is the robot UBTECH Alpha1 Pro. Programmable robot that boasts a set of interesting features.

The company UBTECH creates robots for several years. It also makes relatively simple designers that can be turned into a Voltron in almost anything and complex functional devices - for example, the Cruzr robot, which can help at home or in the office. The fact that the robot drives for coffee to the nearest coffee shop can not be counted on, but to find an interlocutor in it is not a problem.

What is in the box?




The complete set of the robot is minimalist - it is he himself, the charger, the USB cable and the manual (read without fail, yes!). The system is controlled with the help of an application for mobile devices - there is an option for both PCs and mobile platforms.



To get started, you just need to download the application and turn on the robot, everything is as usual. However, the setup process runs smoothly, a lot of time is not required.



Design


The robot looks menacing. It is good that he is small, so if you met him somewhere in the dark, you could be scared. Alpha1 Pro humanoid: two arms, two legs, head. There is also a ‚Äúbackpack‚Äù, similar to the astronauts' knapsack - this is the unit used to control, charge and turn on the robot.

The assembly of the robot quality. We had to overdo it in the hands of a bunch of different gadgets. This robot leaves behind the impression of a good assembly: no backlash, no gaps - nothing. There are LEDs that are particularly spectacular in the dark. The robot can reproduce sound - for this it is equipped with a speaker (it is located right in the head). Servomotors work noisily, the audibility of the sounds made by the robot is quite low.



And what can he do?


When you exit the box, the robot can do exercises, show asanas from yoga, dance or fight (it shows the skills of a fighter without casualties). The user can add to the list by programming various movements.

By the way, it's great if there are several robots. In this case, they can show dances in a group or fight. Well, if all this is not like - you can play football. A little later, the creators of the robot promise to add features - in any case, to make the dances more exciting.

Our assessment
