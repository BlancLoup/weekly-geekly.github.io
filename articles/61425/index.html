ESET NOD32 is now compatible with The Bat!

Now users of the mail client The Bat! will be able to protect your mail with antivirus software ESET NOD32. Just install the special integration module ESET NOD32 Antivirus Module for The Bat! ..

The module supports work in Microsoft Windows operating systems 2000 \ XP \ Vista and integrates with The Bat! version 1.6 and up.

To install the ESET NOD32 Antivirus Module for The Bat!, Close The Bat! Download the module from the ESET website and run the installer



When you start the mail client, its settings will display ESET NOD32 Antivirus Module



The status of the module can be checked by clicking the "Settings" button.



Now, when a user receives an infected email, the ESET NOD32 antivirus system will display a message stating that the email contains an infected object and, depending on the settings, will delete it or place it in the specified folder, for example, in quarantine.



By default, the user can configure the scan of attached files, outgoing mail, attachment files before saving to disk and checking a separate quarantine folder where all infected messages will be placed along with attachments
