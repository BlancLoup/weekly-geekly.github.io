W3C will tell how to make sites for mobile phones.

The W3C consortium announced a public web-based mobile seminar seminar on November 16 in Paris.

The Mobile Web Initiative (MWI) was launched in 2005, but little has changed since then. Users accessing the Internet using mobile devices are still inconvenient when working with websites that are not adapted for slow communication lines and small screens. The goal of the MWI team is to develop standards for creating mobileOK sites, that is, resources suitable for downloading to mobile devices.

The alternative transition to the mobile platform is a partial rejection of the universal browser, but instead the release of special client software, mobile widgets that would be installed on the phone to work with a particular Web 2.0 service.
