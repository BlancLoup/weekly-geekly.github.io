Microsoft: Start menu will return to Windows 10

In its official Windows tweet , Microsoft made it clear that in the new version of its operating system, the Start button will be returned to its place.


As you can guess from the screenshot, we are really talking about the full return of the usual button, and not the transition to the tiles, as is done in Windows 8.1.

It is worth recalling that on January 21 a special event is expected, which in the invitations sent to journalists is designated as "Windows 10: the next chapter." It is expected that a new version of Windows will be introduced, which, according to rumors, will work both on desktops and mobile devices.

The presentation will be led by Microsoft's top management: CEO Satya Nadella, as well as board members Terry Meyerson and Joe Belfiore, who are responsible for developing Windows.
