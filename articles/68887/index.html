ufirefox

ufirefox (usetools firefox) - build a web browser mozilla firefox
from usetools.net with additions
pre-configured profile in the universal automatic installer or portable version (does not require installation, can work with usb-drive).





ufirefox - a web browser for the safest, most clean and efficient work on the Internet.
suitable for a wide range of different tasks when working with web content
as well as optimally tuned and honed to achieve good performance
with high functionality without sacrificing reliability, ease of use and appearance. It has a built-in balanced selection of functional and thoughtfully pre-configured add-ons (extensions).



main functionality:
  • convenient tabs system for opening web pages, switching between them, changing the order of location, returning to closed tabs and other actions in one browser window; tabs have a contrasting and at the same time soft color backlight (colorfultabs), as well as are equipped with effective auxiliary control tools (tab buttons);
  • smart address bar with perception-enhancing highlighting and subscript output of the history of entered web addresses; includes the strengths and convenience of the address bar of the firefox3 and firefox2 web browsers;
  • everything is at hand to save web addresses locally (the bookmarks panel and menu, link pad, smart bookmarks bar) or google online bookmarks with flexible import / export / backup options (gmarks);
  • file downloads: simple download with color indication of the status of current downloads in the bottom panel (download statusbar); loading in several streams with resume and control through multi-functional download manager (downthemall!); accelerated loading of files from the most popular file web services (skipscreen); in addition - a full-fledged ftp-client (fireftp);
  • special settings, extensions and plugins for the most efficient and fast search and web surfing (google toolbar, locationbar2, dns prefetch, customizegoogle);
  • brief but useful information display in the bottom panel (external ip, flagfox, organize status bar);
  • blocking unwanted advertising web content (adblock plus, element hiding helper);
  • opening web pages in alternative view mode using the internet explorer kernel in a new tab and without leaving the firefox window for pages and web tools made ‚Äúunder ie‚Äù (ie tab);
  • clean automatic installation procedure with the correct purpose of ufirefox web browser by default, as well as high-speed and clear removal procedure *;
  • implemented full support for multi-user work with a separate original ufirefox profile (the same pre-configured profile for each user of the system, auto-deployment profile when first started, full compatibility with the terminal mode of operation) *;
  • transfer of key information of the ufirefox profile to the new profile when updating over (credentials, bookmarks), as well as the ability to quickly reset the profile to its original state or simply complete clearing of personal data from the program *;
  • and other small, but carefully thought out details, convenient for everyday use of the Internet.

In addition, ufirefox includes several preconfigured web applications (prism) * that implement a convenient and practical way of working with web services of the new generation from google:
- mail - http://gmail.com
- documents and spreadsheets, presentations - docs.google.com
- calendar planner - google.com/calendar
- maps - maps.google.com
- dictionaries - google.com/dictionary
