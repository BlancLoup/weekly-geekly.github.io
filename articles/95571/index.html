Rockbox 3.6 is here!

Just yesterday, a fresh version of the popular alternative firmware for various media players, Rockbox, followed by version three, point six. What's new?

Core


  • Added support for Packard Bell Vibe 500.
  • Now it is supported to change the hard disk in all HDD-players (except Archos). It also supports disks> 137GB and disks with a large sector size.
  • Screen skins support for FM radio.
  • Added rewind option to continue.
  • Bookmarks now include information about the step and playback speed.
  • Hotkey support
  • Bar graph on the playback screen. So far, only for iriver h1x0 / h3x0.
  • Multi-font support.

Plugins


  • New features in the text viewer plugin: global settings, individual settings for each file, header and footer, font selection, bookmarks.
  • FFT - plug-in frequency analyzer.
  • New plugin "Alarm".

Codecs


  • Support for Musepack SV8. Integration of the decoding library (version 1.3.0).
  • True Audio codec support.
  • Support for Sun Audio, Vox, Wave64 codecs.
  • Support for SMAF codec.
  • ADPCM support from Microsoft, Apple, Adobe, Yamaha, etc. in WAV / AIFF files.
  • New iMDCT library, which increased the running time when playing Ogg Vorbis, WMA, AAC, ATRAC3, Cook, AC3
  • ATRAC3 support in WAV / AT3 containers.
  • Play ATRAC3 on ARM and Coldfire.
  • ATRAC3 support for Sony OMA container.
  • Support for floating point PCM in WAV and AIFF.

Also fixed bugs with battery indication on iPods, backlighting, and more.
