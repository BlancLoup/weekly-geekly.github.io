Plans to develop 3CX and a new training video course for partners and administrators

Immediate plans for system development


Today we will talk about what new features will appear in 3CX soon. This cannot be called the official roadmap, and it is impossible to assess the real dates for the release of a particular function, but I will present the general direction of our developments. You can actually see them in the near future!


First of all, this is the release of 3CX Service Pack 5. In this update, emphasis will be placed on new capabilities for managing VoIP infrastructure. Expect news on the release of the update soon! Immediately, I note that we are gradually moving to a system of small updates, instead of the rare release of large service packages.


In addition to the improvements that are already included in SP5, we are working in the following areas:


  • The module of the hotel PBX - alarm calls are already included in SP5, but integration with real estate management systems (PMS) will be added in SP6.
  • Calls WebRTC Click2Call is a very popular feature for calls from a web site, which was in version 14. It will be added to SP6.
  • Playlists for selecting hold music, i.e. folders with music files .wav format. The system loops through files, losing them to callers. Playlists will be added in SP6.
  • The 3CX web client and web operator panel is a completely new client system running in the browser. It will have a number of new features and a new intuitive interface. Most likely, will not be presented in SP6.
  • The system for creating voice applications VAD (Voice Application Designer). We are developing a new VAD that will be integrated with the Call Queue service (call center). Management of outgoing calls will be added (in the current version, you can only work with incoming calls) and performance has been significantly improved. Most likely, VAD will not be introduced in SP6.
  • Mode shift work in clients 3CX and IP phones. For example, in any round-the-clock service or in an enterprise with a continuous production cycle. This feature is often requested, and now we are working to integrate it into the basic 3CX functionality (it was previously implemented in some IP phone templates). Most likely, will not be presented in SP6.

If we talk about the timing, the SP6 update is expected in one to two months. The remaining features, if not included in SP6, will appear in the following updates.


New basic video course 3CX


As you probably know, we pay great attention to the training of partners and administrators of 3CX. After the release of 3CX v15, we updated the training course by introducing an additional level of certification - Intermediate (Medium). Now we are conducting offline trainings in various parts of the world (including Russia, Moscow) and online webinars (including in Russian). After these trainings, partners can take a certification test and get 3CX Certified Partner status. 3CX Certified Partners can be posted on the 3CX website as recommended partners .


In addition to training and online training on V15, our methodologists prepared a series of video tutorials that you can view at a convenient time for you. This course opens with a Basic lessons cycle that will prepare you for initial certification. In the near future, Intermediate and Advanced video courses will be published for obtaining relevant certificates.


The Basic course includes 5 modules that cover the basic steps of installing and starting to use 3CX:


  • Install and configure 3CX server
  • Install and configure 3CX clients
  • Setting up desktop IP phones
  • Firewall settings
  • SIP trunk connection

We recommend complementing the viewing of lessons with the study of 3CX documentation (in Russian). Now you are ready for the exam!


  • If you are a 3CX partner, log in to the partner portal and proceed to the test.
  • If you are an end user of the 3CX system (system administrator), register here to take the test.
