RPG game in Excel workbook



An accountant from Toronto named Cary Walkin (Cary Walkin) has never been involved in the development of computer games, but has mastered Excel perfectly. That was enough to create an RPG game that runs on VBA macros, an embedded Excel programming language.

Game Arena.Xlsm version 1.0 can be downloaded here:

v1-0-arena.xlsm (works in Excel 2007 and Excel 2010 under Windows)

Key features:
  • Fascinating story with four different endings, depending on the passing game
  • Over 2000 enemies with different levels of AI
  • 39 item modifiers means more than 1000 possible combinations and properties of objects
  • 8 fights with bosses, to defeat each of them need a special tactic
  • 4 pre-prepared gaming spaces
  • 31 spells, there are many different strategies for success
  • 15 unique artifacts
  • 36 awards
  • The whole game runs from start to finish inside an Excel workbook.

The official wiki page for help on the game, there is information about basic strategies.
