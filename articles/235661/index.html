Gear VR - Samsung's Unusual Virtual Reality Helmet



Devices that provide the user with access to virtual reality are now developing very actively. For example, video helmets and video glasses are now being promoted by several companies at once, including Facebook, Sony, Google and several other, not so well-known companies.

It turns out that Samsung has views on the market for video helmets. Today, at the exhibition IFA 2014 in Berlin, this device was shown to the general public. The device name is Gear VR.

What is unusual about it? The fact that, in fact, the simulation of virtual reality is not using the platform of the device itself, but a third-party gadget. In this case, this is Samsung Note 4.



Yes, you understood correctly - a smartphone is inserted into the helmet (the connection is via micro USB) and work is already underway with Note 4. At the side of the glasses there is a special button to exit applications. The same button is also used to quickly return to the real world from the virtual world, for which the button needs to be clamped and pressed. In the helmet itself there are built-in modules of the gyroscope and accelerometer, there is nothing more than lenses and mounts.



Those who managed to test the Samsung's helmet at the exhibition claim that there were no lags and brakes when working with special applications for the Samsung Gear VR. The interface is sane, management is also made without much difficulty.
