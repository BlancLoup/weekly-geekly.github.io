Build 2014. Announced changes in Windows and Windows Phone app stores



On April 4, the Build 2014 conference ended, at which Microsoft made many different announcements. In addition to the announced mobile operating system Windows Phone 8.1 , available updates for the client Windows 8.1 and updates to Visual Studio 2013 with the ability to create universal applications, we will have another important change that will affect developers of mobile and client applications. Speech about changes in app stores.

The good news for developers is that by the summer we will have a single application store for the entire platform with an interface adapted for mobile phones and tablets, simplified registration as a developer and a general procedure for publishing applications for Windows and Phone.
