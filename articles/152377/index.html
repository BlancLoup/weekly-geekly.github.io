Step 2.

Now you can start building Qt from sources.

First of all, I suggest downloading the source code.
I took these, and maybe even a fresher suit, but I did not take the risk
Now you need to unpack this case in X: \ QtSDK \ QtSources \ 4.7.4 \
you will not have QtSources \ 4.7.4 \ folders, you need to create them



Now we need to open your favorite notepad file
X: \ QtSDK \ QtSources \ 4.7.4 \ mkspecs \ win32-g ++ \ qmake.conf

find the line containing the text: QMAKE_LFLAGS
and bring it to this kind of mind:

and save the file.


Now we need to open the patched terminal (forgive me if the thing is not so called), which can be found at home by looking at my screenshot


Now in this terminal we need to go to the directory
X: \ QtSDK \ QtSources \ 4.7.4 \

and score a team

then you have to read everything that you will be offered to read and correctly answer questions
