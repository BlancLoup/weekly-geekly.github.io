Skype 4.0 Beta 1 for Windows

Today, Skype announced the first public beta version of the Skype 4.0 client for Windows - release 4.0.0.145.

Skype 4.0 Beta 1 for Windows received a completely updated user interface, in which little was left of the usual Skype. All work with the program now takes place in a single window, including open chats, voice and video calls. In fact, Skype has become a unified communication environment that can be expanded to a full window and fully enjoy the program‚Äôs features. According to the developers, the new interface should be intuitive for any user, and is designed to help him easily deal with all the functionality of Skype. Most of the actions in the new environment can be done by dragging the mouse.



The new tool - the Welcome window - will help first of all beginner Skype users. The welcome window now allows you to quickly find your friends and invite them to Skype. Here you can quickly and conveniently set up equipment for working with Skype: incoming and outgoing audio, video parameters.

Among the announced new products there is another useful feature for working with user equipment. Now the program itself can recognize almost any accessory that has a Skype certificate. Skype 4.0 will help you quickly set up familiar equipment, thereby once again facilitating the user's work.

Skype 4.0 Beta 1 for Windows is an early beta version of the program, and is recommended for downloading and installation only for experienced users and testers.
