iPhone updated to version 2.2


Apple updated the iPhone firmware to version 2.2. New features have become available:

- StreetView in Google Maps.
- Selecting the type of route (auto / public transport / pedestrian) in Google Maps.
- Set autocorrection when typing.
- Rating the program to be deleted.
- Updated Safari (improved stability and performance of the browser).
- Updated AppStore interface.
- Updated mobile iTunes (now with podcasts).
