Training on ASP.NET MVC Framework in Moscow

On March 12, Microsoft will be hosting a free ASP.NET MVC Framework ‚ÄúDeveloping Web Applications Using ASP.NET MVC Framework‚Äù for developers at the Microsoft office in Moscow. The training is designed for developers with experience in developing web applications, but not familiar with the technology ASP.NET MVC Framework. Knowledge of ASP.NET technology is welcome but not required.

Conducts training - Microsoft Web Development Technology Expert Gaidar Magdanurov . Start of training at 10.00, ending at 18.00.

Description of the training

The purpose of the training is to talk about a new development paradigm for the ASP.NET platform - ASP.NET MVC Framework. It describes the software model that allows you to clearly separate the presentation code to create an interface from the query processing logic code and the code that works with the data source. This approach allows you to get the possibility of effective automatic testing of the application logic, which is also discussed at the training.

Short program:
  1. MVC Framework Overview
  2. Application development process using the MVC Framework
  3. Comparison of MVC and Web Forms
  4. Model and data access
  5. Controller and request processing
  6. Application presentation and interface
  7. Routing mechanisms
  8. AJAX client side
  9. Using ASP.NET infrastructure
  10. Testing MVC Applications
Each training participant will be provided with a computer.

During the training, participants will be provided with coffee and buns during the breaks. Lunch will be held in the canteen of the business center at the expense of the participants.
