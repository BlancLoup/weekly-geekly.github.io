A long time ago I was waiting for an invite to this rather interesting service:
left them my email and google inviting on the way, really wanted to
see what it is. And this invite was dropped in my inbox
at that moment when I already forgot about what fav.or.it is . In this small note I will briefly tell you about the service and how my acquaintance with photography took place.

RSS reader


fav.or.it is more than an advanced RSS reader.
The creators of the resource position it not only as a newsreader using
RSS technology, but also
as a system for discussing certain news directly on the site. In fav.or.it we can see most of the trends of WEB 2.0 fashion.
Recall that the development was used Zend Framework and JS Framework Prototype .
(Both the user side and the server side are quite fast.)
In addition, there is its own API interface.
Here there is a message rating system, and the now fashionable tag cloud, which at the touch of a button turns into a clear list.
And with the help of AJAX, without reloading the pages, you can leave a comment in the stream of news and continue reading further.

Start dating



Having passed the elementary registration (for those who have an invite :)) we get to the main page.

It should be noted that fav.or.it has an unobtrusive design and an intuitive interface.



The main page displays the most popular news that can be sorted by category.
On the left is the user sidebar, which displays your subscriptions, categories, tags, etc.
On the other side of the page are Tips and Tricks with tips on working with the service and the latest pictures.
To navigate through the records, you can use shortcuts: j ‚Äî the next post, k ‚Äî the previous post, u ‚Äî hide the panel.


And what do we do?


Commenting



For each entry there is the possibility of commenting (write a comment? In 60 seconds you can edit it). Of particular interest is the ability to download comments from
parent blog (blog from which RSS is taken),
as well as posting comments to the blog through the comment form on the site.
3 main blogging platforms are supported:
  • Wordpress
  • Blogger
  • Livejournal


Voting


Right there you can vote for the topic: positively or negatively.
In addition, there you can see a graph of voting statistics or manifestations of attention to the article.

Sharing with friends


There is the possibility of "sharing" messages.
To do this, you need to specify in the settings account information where you need to send a link to the post.
Again supported:
  • blogger
  • livejournal
  • All of us favorite twitter
  • delicious
  • wordpress
  • friendfeed


And then in the post, click share and select the place where you need to do it. Verified on your own twitter account.

Tags


In fav.or.it, two main possibilities are used to implement a convenient search: categories and tags.
For each individual post, you can assign your own tags and categories.
All tags and categories are displayed on the sidebar for each message of their own.

Import streams


The main purpose of the service is a feed reader (RSS aggregator) . Of course
it is possible to add RSS feeds.
You can add subscriptions in several simple ways:
  • Find through a special site search.
  • Specify the stream address or a whole list of addresses.
  • Download the OPML file from, for example, Google Reader ,
    or a plain text file with URL stream list. Import is very flexible. In the process, you can specify feeds that are not
    need to download and set categories.
  • Find the desired tape in the catalog through a convenient search, specifying a category and / or keyword to search.

to replace your google reader .
But this is not a reason to be upset, because in their catalog there are already so many various popular English-language feeds.
