Linux distribution and web browser for PDAs and smartphones

The Linux release of the Poky 3.0 distribution has been released. It is based on the latest developments of the Gnome Mobile and OpenEmbedded projects.

There is support for installation on the following types of PDAs and smartphones:

* Nokia N800;
* OpenMoko GTA01;
* Sharp Zaurus C700, C750, C760, C860, SL-7500;
* Sharp Zaurus Akita (SL-C1000);
* spitz (SL-C3000, SL-C3100, SL-C3200);
* Can run in QEMU emulator.

Also presented a new open mobile web browser OWB , built on the basis of the engine Apple WebKit. Packages for installing OWB on a Nokia N800 Tablet PC can be found here .
