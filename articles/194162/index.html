Free Month at CodeSchool by NewRelic

The popular application monitoring service NewRelic gives all new users a month of full access to CodeSchool materials. Register here . To get a coupon, absolutely any application needs to be closed - even Hello World is quite suitable.
CodeSchool in Brief
Code School is an educational project from Envy Labs , which includes about 30 interactive online courses in the field of IT. The main directions (paths) at the moment - Ruby / Rails, Javascript / jQuery, HTML / CSS / Design, iOS. In addition, there are additional courses that are not included in the main areas and are designated as Electives, for example, on Chrome DevTools and Git. In addition to interactive courses, the resource offers dozens of screencasts on the same subject. Normally, full access to all this beauty is available for $ 25 a month, although some courses are provided free of charge.

Detailed instructions for deploying hello_world.wsgi using python / gunicorn under the cat

Since the codeschool courses are useful primarily for beginners, and the NewRelic service is primarily for experienced developers, below I provide instructions for gaining access as painlessly as possible.

I assume that you have python and pip installed on your machine. If not - you here .

So, when registering do the following:
  • When filling out the form in step 2, select the python language
  • Create the file myapp.py (the code is taken from the official gunicorn site ): 
    def app(environ, start_response): data = "Hello, World!\n" start_response("200 OK", [ ("Content-Type", "text/plain"), ("Content-Length", str(len(data))) ]) return iter([data])
