Updating the interface of Google Images

Google has announced an update to the image search interface:



  • Images are more densely arranged, so that you can view more images at the same time, but at the same time, the thumbnails are enlarged.
  • Instant scrolling between internal pages. Now you can view up to 1000 images on one scrolled page. At the same time, the numbers of internal pages are displayed, so users will not be lost.
  • When you hover the mouse pointer over a thumbnail, a panel appears with information about the image.
  • When you click on the image, the user is redirected to the page on which it is located, and the image itself is shown against the background of this page. If this click outside the image, the user is redirected directly to the original page.
  • You can switch between internal pages with the standard Page Up and Page Down keys.

Allegedly, more than 10 billion images have now been indexed in Google.
