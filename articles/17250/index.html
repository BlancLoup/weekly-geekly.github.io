IGoogle Winter Theme

For lovers of winter landscapes in iGoogle there is a winter theme.

Looks like that:



But it is not available through the standard interface.

You can activate it in the following way:

we copy this javascript code

javascript: _dlsetp ('preview_skin = skins / winterscape.xml'); void (0);

go to the iGoogle page, paste the code directly into the address bar of the browser and press Enter. After updating the page, click "Save" in the theme selection menu. Enjoy :)
