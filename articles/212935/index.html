Mobile applications for NetApp

Anyone who is involved in the process of operating a NetApp storage system , sales, presale, end customers, engineers, the following mobile applications can be useful.

NetApp Document Search

Android
Provides access to the NetApp Public Library, which contains customer histories, analytical, technical and research and support documents, as well as video. The application is available to the general public and does not require registration data.
It will be interesting for end users, integrators and distributors representatives.

Hardware universe

Android IOS
This mobile application provides quick access to the list of compatibility of NetApp products, specifications, information about system components, adapters, disks, disk shelves, cables, software, and more. All specifications are based on compatibility information based on the latest versions of Data ONTAP. The Hardware Universe mobile application is designed as a reference resource for NetApp employees, authorized users and end customers during the equipment selection and upgrade stages as well as during operation.

NetApp Support

Android IOS
NetApp Support provides information on case studies, contact information, AutoSupport status, space efficiency, and the overall state of your storage . Service partners can update information on the implementation of service visits by answering questionnaires. The application requires a login account for the technical support site (http://support.netapp.com).
It will be interesting for end users and integrators to monitor installed storage systems .

Storage RFP Builder

Android IOS
Change the approach to purchasing storage c NetApp RFP builder that allows you to create a request for a commercial offer from a smartphone wherever you are. Answering some questions, the application will automatically create and send an RFP to your email, which saves you time. The app includes tips and helpful questions based on your tasks.
It will be interesting for both end users and integrators to compile an RFP .

NetApp Connect

Ios
Mobile users need access to corporate information everywhere and at any time from any device, whether it is personal or corporate. As a result of the introduction of BYOD technologies, IT was forced to deal with complex security implications for increasing business efficiency. NetApp Connect software provides mobile users with effective access to corporate data, while allowing IT to manage information leakage risks and maintain compliance with security policies. What provides secure access to your own equipment with corporate data, such as SharePoint servers, CIFS file resources, home directories, internal site and application portals and everything without VPN and the need to move, copy or synchronize data.

OnCommand Resource Navigator

Android
The application is designed to provide quick and mobile access to management information for the NetApp OnCommand family. OnCommand is a management software that helps to increase the efficiency of storage and service through functions that provide control, automation, analysis and integration of your shared information infrastructure. OnCommand Resource Navigator is designed to help you in practice quickly find the information you need to solve storage management problems.
