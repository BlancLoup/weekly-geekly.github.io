Avast Mobile Security for Android Review

Last summer, there was information that Avast will release a version for Android.

I admit, I was a little skeptical about this information. Because I had experience installing anti-viruses for Linux Desktop systems to protect these very systems. And did not see from this whole sense. Behavioral analyzer was not. Test rootkits were freely installed in the system, etc. In general, one disappointment. But my innate curiosity did not give me rest, and I decided to install this miracle on my tablet. The result pleased me. Especially considering that the application is free. And the most interesting is available for owners of root-access to their android-devices.
