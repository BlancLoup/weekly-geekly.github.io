SQL Server 2008 Service Pack 2 Released and some more updates.



The second service pack for SQL Server 2008 has been released .

A full list of innovations can be found here , below are some of them:
  • Supports up to 15,000 sections for tables and indexes. Previously, there was a limit to 1000 sections. A full description of the innovation can be found in this separate document (.docx);
  • Reporting Services integration with SharePoint 2010 ;
  • SQL Server 2008 R2 Application and Multi-Server Management is now compatible and can work with SQL Server 2008;
  • new administrative tool SQL Server Utility ;
  • support for Data-tier Application (DAC) applications that was previously introduced for SQL Server 2008 R2.
You can download the second service pack for SQL Server 2008 at this address .
In addition, the following tools have been updated:
  • SQL Server 2008 SP2 Reporting Services Report Builder ( link );
  • SQL Server 2008 SP2 Reporting Services Add-in for Microsoft SharePoint Technologies ( link ).
