A selection of syntax highlighting color schemes for Sublime, TextMate and Vim

Dale Reese - one of the developers of the Laravel PHP framework - laid out on GitHab a very nice selection of color schemes compatible with Sublime Text 2, Vim and TextMate. It includes mostly dark themes (21 out of 25), and, which happens not so often, almost all of them look very decent - there is plenty to choose from. Installation instructions and screenshots of each topic are in the repository .
