Webasyst Blog: a collective blogging application

Introducing the free blogging application (LGPL license) for blogging written in the Webasyst PHP framework:




The application is designed primarily for corporate use: for companies that maintain open (public) and closed (intra-corporate) blogs. The application allows you to keep all these blogs together in a single thread. The application can be interesting to bloggers who are tired of Wordpress their blog engine.

It supports all the basic functionality that is in popular engines: many blogs, comments, RSS, sitemaps, cat, frontend, backend, design editor, etc. There are some nice features, like, for example, the calendar of publications :
