Fedora 25 release

The Fedora Project is pleased to present the next release of its distro, Fedora 25. Fedora is a large community that works every day to promote free software. Fedora 25 is offered to us in three different variations, each of which serves to meet specific needs: Fedora 25 Atomic Host, Fedora 25 Server, and Fedora 25 Workstation. Fedora 25 contains a large number of fixes and improvements, along with Fedora includes the following packages:

  • Docker 1.12 for installing and running container applications
  • Node.js 6.9.1, the latest version of the popular JavaScript engine
  • Rust support, fast and stable system programming language
  • Simultaneous support for multiple versions of Python - 2.6, 2.7, 3.3, 3.4 and 3.5.

Major changes


Fedora Workstation includes such important changes as:

  • Discarding the use of X11 by default in favor of Wayland, the X11 session will remain as an option
  • Upgrading GNOME and all its programs to version 3.22. Changes can be found here.
  • MP3 support without installing additional decoders
  • Added program for recording boot images "Fedora Media Writer"
  • Now when you upgrade to the new version of GNOME, the extensions will no longer break.
  • New versions of the packages: PHP 7.0, Ruby on Rails 5.0, GHC (Haskell) 7.10, Perl 5.24, Erlang 19, Go 1.7, Glibc 2.24
  • Built-in ability to install, remove and update packages in Flatpak format

Upgrade to Fedora 25


Fedora 24 users can upgrade to the new version within 30 minutes using several commands in the terminal: 

$ sudo dnf upgrade --refresh $ sudo dnf install dnf-plugin-system-upgrade $ sudo dnf system-upgrade download --releasever=25 $ sudo dnf system-upgrade reboot
