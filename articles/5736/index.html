My discovery Web 2.0: Education, myschoolog


mySchoollog is an on-line application with the help of which students and schoolchildren can organize their hard lives, save time and knowledge, create schedules and task lists. The resource works on AJAX, you do not need to wait for the pages to reload. The service is free. As always, the most remarkable thing is that this miracle is available from any place where there is a computer and the Internet, as well as using a mobile phone with Symbian. PHP project with 15.197 lines of code, the current version is beta 0.92. Developed since December 20, 2006.

www.myschoolog.com
