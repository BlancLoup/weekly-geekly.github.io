Firefox 16 is released

The stable release of Firefox traditionally becomes available for download a little earlier than the official release, which will be held tonight .
What's new in this version:


  • The garbage collector has moved to an incremental strategy ;
  • For Mac OS X, VoiceOver is enabled by default;
  • Display statistics of memory consumption of each open tab in about: memory ;
  • Added localization in Acholi;
  • Changed the design of the browser version number in the User-Agent header, which now includes only the first two digits of the ‚Äú XY ‚Äù version instead of ‚Äú XYZ ‚Äù;
  • A web developer toolbar has been added, which has buttons for quick access to the web console, Inspector, debugger, and the Command Line (Shift + F2);
  • Added support for unprefixed rules for CSS3 Animations, Transitions, Transforms, Gradients, IndexedDB, Battery API, Vibration API and calc ().
    (All previously available moz variants are retained for backward compatibility);
  • Added support for web applications ;
  • HTML5 Microdata API support and HTML5 meter element;
  • Added support for reverse animation direction;
  • Digital signatures no longer support MD5 hash algorithm;
  • By default, support for the Opus audio codec and the third version of the SPDY protocol is included;
  • Scratchpad JavaScript editor has added support for displaying the list of recently opened files;
  • Debugger breakpoints now work when page reloads.

Important : the release of this version is the latest for Mac OS X 10.5 users (Leopard)
