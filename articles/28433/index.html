Opera-9.51 release

According to the Opera Desktop Team development team , an updated version of Opera 9.51 was released today.

List of changes:


User interface
- Changed the appearance of Opera
- Improved tab dragging function
- Fixed a problem with the list of search resources that occurs when updating over the version of Opera 9.2x
- Improved browser stability when printing or previewing.
- Added option to enable / disable navigation using mouse buttons in opera: config (User Prefs - Enable Mouse Flips)
- When caching is disabled, the text input fields remain empty.
- Saving images from pages no longer appears in the download window

Mail / News
- News feeds are displayed when a subscription starts.
- Solved the problem of unstable work when using the drop-down list to switch the display mode mail
- Added disclosure of grouped message threads when new messages are received
- Fixed error in displaying messages from one account in multiple folders

Display and operation of scripts
- Improved stability when using custom javascript
- Now style sheets are loaded when navigating in history
- The window.close () function remains working after using the context menu and closing Opera Dragonfly

Security
- Fixed function access to arbitrary data in RAM
- Fixed a bug that could be used to execute arbitrary code. Details of the error will be made public later.
- Fixed switching security state when switching from HTTP to HTTPS
- Fixed a bug with the OCSP and CRL certificate handling functions

Other
- Improved Yahoo! stability. Mail
- TinyMCE 2.1.x editor now works correctly
- Improved print chat items
- Improved chat reconnect
- The menu is working on the site deviantart.com
- Fixed unwanted end of lines in full-text editors

List of changes specific to Winodws
- Fixed a memory leak that occurred when the download window was open
- Command line parameters should now appear before the address of the opened pages (URL)

List of GNU / Linux Specific Changes
- Added Opera installation package for 64-bit GNU / Linux systems
- Fixed unauthorized closing of pages on Qt4 builds
- Improved saving of plugin configuration changes

List of Mac OSX specific changes
- Improved the function "Use the current page" in the Quick Start window
