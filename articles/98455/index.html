Microsoft has opened all source codes to Russian special services

Microsoft signed an addendum to the agreement with the Federal State Unitary Enterprise Atlas Scientific and Technical Center , which gives access to the sources of all major Microsoft programs, including the Windows 7 and Microsoft Windows Server 2008 R2 operating systems, the Microsoft Office 2010 office software package and the database management system Microsoft SQL Server data. This was reported in an interview with the newspaper Vedomosti by a source at Microsoft.

The agreement itself between Microsoft and Atlas SEC was concluded in 2002 and gave access to the sources of Windows XP, Windows 2000 and Windows Server 2000.
