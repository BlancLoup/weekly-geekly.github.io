Kharkiv Ciklum .NET Saturday for real men

Dedicated to real men and beautiful ladies on Defender of the Fatherland Day on February 23! We invite to the Kharkov office on Ciklum .NET Saturday everyone interested in the subject of .NET and Microsoft technologies.

In the Ciklum .NET Saturday program:

1. Real-time ASP.NET with SignalR by Alexander Kondofurov
(.NET developer / Project Lead, MCPD, co-organizer of the Kharkov AI Club (artificial intelligence club), employee AltexSoft).
Have you ever written multi-user real-time web applications? Would you like to learn about technologies that are already changing the paradigm of request-response and web programming that we are used to? Then you should definitely listen to SignalR.



2. MongoDB: Aggregation framework by Vitaliy Domnikova
(professional web application developer, as well as Windows Phone and Android mobile platforms, handyman).
This framework is a declarative way to work with such data. No more missing SQL :)

3. Readability: Algorithms and text mining from Alexander Krakovetsky (Author of Windows Phone 7 Rocks! And Microsoft User Group Community. Head of DevRain Solutions (developing applications for Windows Phone and Windows 8). Microsoft Regional Director, Microsoft MVP, Telerik MVP, Ph.D. in Information Technology, speaker, active member of the technical community, columnist Forbes). Let's talk about the methods of word processing and the principles of work services ala Readability.


4. Roslyn Compiler as a service from Evgeny Zharkov (5 years of experience in the commercial development of Deman video systems, IPO, augmented reality. Active speaker, Silverlight MVP. Works towards Kinect, Windows 8 platform and JavaScript. EPAM Systems employee).
What new features does the future of C # hide? Perhaps we will be able to syntactic sugar, expand previous capabilities, change the code in the process of work, and implement all this outside the program window.

5. TDD + AutoMocking: Instead of boredom, fun from Sergey Kalints (has been developing for quite a long time. Conference speaker and user groups. Engineering practice coach. In the world he leads development at CompatibL's Kiev office).

6. WinRT: integration with C / C ++ from Mikhail Galushko (Co-founder of the DevRain Solutions team developing applications for Windows Phone and Windows8. Co-author of the site Windows Phone 7 Rocks! .. Has MCPD certification .NET 3.5, 4.0).
