Facebook Connect: links to authentication guides

Interaction with social networks is a very important and urgent task for developers of web projects. As the slow flow of users to Facebook, already noted by Norwegian Forest , begins in Russia, the problem of mastering Facebook Connect is becoming increasingly urgent.

Some CMS like BuddyPress already include FBC functionality in the base distribution.

Searching through Habra, I found only the best practices for creating applications inside the FB, but there are almost no materials on using the FB authentication system.

Below I offer an overview of links for tutorials, plugins, and libraries for connecting to Facebook Connect. Unfortunately, all materials are in English. I suggest everyone to add material.


1. Facebook Resources


  • Basic Facebook Connect Connection Guide
  • FBC material on authorization on sites
  • Detailed material on the site authorization mechanism

2. Facebook Connect Plugins for Wordpress, Drupal, Joomla and other CMS


  • List of plugins for CMS

3. Resources for PHP MVC frameworks (CodeIgniter, CakePhp)


  • Facebook Connect Library for CodeIgniter by Elliot Howin
