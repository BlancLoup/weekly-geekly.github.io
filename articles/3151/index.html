Microsoft announced Windows Mobile 6

Microsoft has announced the release of the sixth version of the operating system Windows Mobile 6.

The novelty, whose interface resembles Windows Vista, has a closer (compared to previous releases) integration with the 2007 Office suite and the Outlook 2007 e-mail client. At the same time, the requirements for RAM and a processor, according to Microsoft, have grown insignificantly, Ars Technica writes.

The Windows Mobile 6.0 operating system includes the Office Communicator IM application and is also compatible with Windows Live and VoIP. The first portable devices running the new OSes will appear in the second quarter of this year.
