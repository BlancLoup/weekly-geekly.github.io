Games and the cloud. Myths and reality. We expect a heated discussion on April 12. Come virtually



A couple of months ago, colleagues came to me and said, let's tell game developers how Azure can be used to develop game backends. I knew that we had something to tell, but I was looking for someone to invite to us, so that it was not just a review story from Microsoft, but also an opinion of an industry expert who is familiar with our cloud firsthand. And so, colleagues from Game Insight rolled out an article about using Azure in a real game . And I understood, we must invite Andrei Ivashentsev ! It was not easy - Andrei is very busy and hard to find him in Moscow, but we could!

On April 12, at 11 am Moscow time, we will begin broadcasting the event .

First, Romuald Zdebsky, Game Director, Microsoft, will review the game backend development scenarios where you can use the Microsoft Azure cloud. Immediately after that, I, Roma Zdebsky and Andrei Ivashentsev will discuss, without cuts, whether you need cloud games at all, if necessary, for which scenarios, and then go through how, in principle, you can implement the game logic, storage and analytics.

Join us, I know Andrey for a long time, so I am sure it will be interesting!

Do not forget to register to view the broadcast.
