This fall, Dell introduced its new network storage solutions in Russia - the PowerVault NX3300 and NX3200. If you try to describe them in a nutshell, you get something like the following: these are high-capacity systems designed to optimize data storage and conveniently exchange files / data blocks, as well as to efficiently manage them. And all this, as mentioned above, with an eye on small business companies. Well, let's take a look in more detail.

The Dell PowerVault NX3300 is a comprehensive hardware and software solution for storing, managing, sharing and ensuring high availability of block and file data. Consists of a fault-tolerant network storage device (NAS gateway) integrated with specialized software based on the Windows Storage Server 2008 R2 platform (Nordic character, single) .



The platform itself is based on an optimized release of flexible and reliable Windows Server 2008 R2 file services, enhanced by a number of useful technologies, such as file duplication elimination and iSCSI Software Target software, helping to create unified file and data store repositories.

Why iSCSI, but not, say, FC (fiber channel)? There is no secret - because of the cost of the solution. In fact, iSCSI technology only requires an Ethernet interface or any other TCP / IP compatible network. In this case, there is no need to acquire expensive equipment and acquire special knowledge in the field of hardware, without which it is often impossible to operate FC storage networks. But back to the NX3300.

The device is made in a rack-mount 1U form factor and, depending on the needs of the company, supports building up the cluster configuration from two to sixteen devices. SIS (Single Instant Storage) technology provides simple file-level deduplication, and the File Services Resource Manager (FSRM) File Services Resource Manager provides a single interface for managing and monitoring CIFS / SMB and NFS file services, which reports on how files are used. In turn, the File Classification Infrastructure file classification infrastructure from Windows Server 2008 R2 allows administrators to automate the processes of classifying and implementing policies for the most efficient data management: set the classification validity periods, set up tasks and reports.

The advantage of the PowerVault NX3300 is the lack of binding to a specific storage family, which means variations are possible. For example, use a PowerVault NX3300 NAS gateway with an existing PowerVault MD, EqualLogic, or Compellent disk array. Do not forget about the 12th generation Dell PowerEdge server clusters based on the Windows Storage Server 2008 R2 platform, which are able to work even at very high temperatures. Our story would be incomplete if everything ended at this table. Therefore, we offer to meet with another newcomer - Dell PowerVault NX3200 .

This high density network system is designed to exchange both file and block-type data and supports internal storage systems up to 36 TB using 12 disks in a compact 2U package.



If, for some reason, 36 TB is not enough (and we remember how much RAM is ‚Äúenough for everyone‚Äù), you can increase the capacity with the help of external arrays of Dell PowerVault MD1200 and MD1220.

All file sharing resources are combined to allow users of different platforms to collaborate using the network file system NFS. When NFS is used, storage reliability is replicated across the Distributed File System (DFS), using the Volume Shadow Copy Service (VSS) to create backup copies for them, and the File Services Resource Manager (FSRM) monitors and generates reports.

Another advantage of the NX3200 is the ability to select 600 GB SAS drives or 3 TB Near-Line SAS drives. As a matter of fact, Near-Line SAS is a SATA disk, in which all external electronics are made according to the SAS standard. And this is not some kind of external snag that allows you to insert a regular SATA disk into the SAS system, but a full-fledged 2-port SAS electronics, which significantly increases the fault tolerance of the system disks in case of problems with one of the ports.

In conclusion, we mention a couple of obvious things, which are meaningless to boast about, since any self-respecting manufacturer has long been obliged to take care of them. It is about ease of use and speed of installation. The PowerVault NX3200 package includes all the components that are needed to safely and efficiently exchange information without spending time, money and resources on deploying a file server from scratch (installation wizards, as well as an unlimited number of CAL licenses for additional customers). In addition, the system can not only fully integrate into the Windows environment using the already deployed management and protection tools, but also store data of client computers using a different OS (Not Windows, ‚Äùsuggests a certain OO).
