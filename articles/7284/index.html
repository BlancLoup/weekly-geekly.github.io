Seven Tips After Installing Ubuntu 7.04

Immediately make a reservation, I have already downloaded my Brave Deer, but I have not recorded the disc yet, and have not tried to install it. Lies while the torrent is on the distribution. And this is a brief translation of Matthew Newton's advice from PC World: what to do with ubuntu, if you had Windows an hour ago. Since I have debian, and to see exactly what was called the possibility in Ubuntu 7.04 was not yet possible, I leave the names of the menu items in English. I beg you forgive me for this :) So, Matthew's advice:


1. Fix the ALT button

By default, it works as a layout modifier. Change in System - Preferences - Keyboard. ALT-F2 can be used to run programs.

2. Adjust the screen resolution

In general, System - Preferences - Screen Resolution.

(Interestingly, Feisty Fawn can define wide screens? We'll see ...)

3. Install Automatix

The easiest way to get the opportunity to watch a DVD and perform other, forbidden, but enjoyable actions. Automatics take here . For example, for i386 , this is the installation package . Run Automation from Applications - System tools - Automatix.

4. Make sure all your media files play.

Files to search in Places. There is also a link to the Windows section, where they most likely lie ...

If something is not played, we launch Automatix, install the AUD-DVD Codecs (the auto-loading codec feature will be disabled, but this step is only needed if something is really not played, it should be played immediately). Next, run Synaptics and install totem-xine.

5. Try to play with graphic effects.

The function is experimental, it does not work for everyone, but if it works, you will be pleased, and if you will be pleased ... well, then you know :)

To enable, go to System - Preferences - Desktop effects. Enable driver (here it is necessary to reboot). Again there, turn on the effect itself - Enable Desktop Effects. If the flight is normal, keep the settings (Keep settings).

Try switching between desktops (toggle buttons on the bottom right). Like? Try pressing ALT-TAB to switch between windows.

[Curious, and the mouse in the upper right corner in Ubuntu 7.04 works as usual in compiz?]

6. You can make even more effects!

In general, I omit the installation instructions for beryl instead of compiz in translation, see the original :)

7. Learn how to install and uninstall software.

Installing and uninstalling programs on Linux is much easier than on Windows. This is done in the package manager. So, two ways.

The first is the easiest. Applications - Add / Remove. And choose what you want. The choice is wide, and often the quality is on top.

Well, if you want more features, you can use the Synaptics package manager. System - Administration - Synaptics Package Manager. Here it is convenient to look for new and unknown programs.

You can write a whole article about how many great programs are in Linux, but not installed by default in Ubuntu. However, at the end of this article, I note that if you installed Ubuntu, and your WiFi card was not detected or not supported, install the Ndisgtk package. Then select System - Administration - Windows Wireless Drivers in the menu, and tell Ubuntu your WiFi card driver for Windows (it probably has it on the disk that came with the card and is a file with the .INF extension). Most likely your WiFi card will work after that.
