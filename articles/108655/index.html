Set up AirPrint on a regular USB printer for Mac OS X 10.6.5 (Video)

Yesterday, the firmware for iOS devices was numbered 4.2.1, paired with Mac OS X 10.6.5, it allows you to print to printers from iPad and iPhone.

But there is a small BUT, as conceived by Apple, this is only possible with certain special printers.

The program airprinthacktivator solves this problem at the root, and we can print on any USB printer.

The program to activate the function AirPrint for conventional USB printers download here - netputing.com/airprinthacktivator
