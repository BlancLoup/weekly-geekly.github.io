Large-scale update CMS Umbraco 8: what's new

In late February, Umbraco was updated to version 8.0. We love this CMS, because in it you can create an intuitive structure for any site, so we could not get past this news.

In the eighth version of the CMS, new features have appeared, the interface has also been redesigned - it has become nicer and friendlier. What is worth only a reminder about saving changes, which moved from the bottom of the screen to the center! Just kidding, of course, this is not the main innovation, but purely visually CMS has become much more pleasant.

Updates have been waiting for almost 5 years, and a month ago it finally appeared in the release. Let's take a look at what has changed.



In each version of Umbraco, the creators' desire for simplification was clearly visible: easier to expand, easier to edit, easier to publish, easier to use. In general, it seems that their motto is to make both developers and content editors working for Umbraco feel most comfortable and as little as possible steamed up.

This trend has continued in version 8.0.

Three new features


In the eighth version, Umbraco introduced three new features:

  1. Infinity Editing - unlimited editing.
  2. Language Variants - language options.
  3. Content Apps - content apps.

In addition, they cleared the code base and improved the performance and stability of the CMS. All this makes significant differences between the new version and the previous - seventh.

1. Unlimited editing


Thanks to this feature, you can easily navigate your tasks in the Umbraco backoffice. The function is useful for content editors and developers, as it simplifies the editing process and makes it intuitive.

For example, if you need to add an image, there is no need to go to a separate section of multimedia to load it or edit it - everything happens right on the same page.

3. Content Apps - an opportunity to improve content using deep data


In Umbraco 8, potential tools appeared for collecting, measuring and analyzing content: how popular is the publication, how many people downloaded the instructions or price list, how readable are the articles on the site, whether the keywords, title and description are true.
