Nokia Configuration Tool - Assistant to the Smart Admin

Hello!

Today we would like to talk about the Windows application Nokia Configuration Tool , designed to quickly configure various settings on one or more Nokia phones with Symbian via a USB cable or Bluetooth connection.

First of all, this application will be useful for system administrators, who, for example, need to connect all employee phones to the corporate Wi-Fi network, configure VPNs on them, add all colleagues' numbers to their address books or download and install the necessary applications on them. Due to the large number of various settings that can be made through the Nokia Configuration Tool, there are many examples of using the application.

Nokia Beta Labs has a fresh beta version of the Nokia Configuration Tool v6.3 with a number of additions, so we‚Äôll talk about it later. The list of new features now includes the automatic detection of all phones on Symbian, as well as a text description for the options in the settings of mail accounts, SIP, VoIP service, MS Exchange and others, so that administrators can quickly get the brief information necessary to configure.

An example of how this works.


The list of options to list meaningless, it is easier to look at the screenshots below. And at the same time look at the process of creating settings for your mailbox on Gmail, in order to understand how the Nokia Configuration Tool looks like in action.

In the list of settings, select E-mail accounts, add a new Gmail account.



Enter the parameters for incoming mail.



And settings for outgoing mail. Save the settings.



And then we load the settings into the phone. Everything.



Now, having configured a profile with all settings (in this case, only mail) of one phone, you can forget about the same type of routine actions for setting up mail on each phone. Simply alternately connect via USB or Bluetooth all the necessary phones to your computer, and literally in a couple of mouse clicks, transfer all the settings to them.

The Nokia Configuration Tool beta is available at Nokia Beta Labs . Full documentation on the Nokia Configuration Tool can be downloaded as pdf (1 MB).
