Proxmox 3.3 Virtualization Management System Release



For those who do not know, Proxmox is a shell that runs on top of Debian and allows you to conveniently manage kvm virtual machines and openvz containers through a web interface. Analogue Xenserver, vmware vcenter, RHEV, oracle virtualbox, etc.
Quite a lot of nice changes that I honestly expected two years ago:

  1. Finally added the firewall pve-firewall.
  2. Finally, we have done and defaulted to a non-VNC console based on html5; finally, you can forget about the terrible Java applet for VNC.
  3. Setting up a firewall network between openvz containers, finally you can build some more complex, isolated, network things based on containers.
  4. Added a mobile web interface, even with a mobile phone you can steer your virtual girls, it is very useful for those who travel a lot.
  5. There was a work with zfs in the web interface, to be honest, I thought that zfs on Linux stopped development and began to die smoothly.
  6. Implemented the possibility of two-factor authorization according to the OATH standard.
  7. A bunch of other small not so important changes about which you can read in the Roadmap .
