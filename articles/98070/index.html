Firefox Mobile (Fennec) 1.1 release for Maemo platform


Last night, Mozilla finally unveiled the final version of its mobile browser for the Maemo5 platform.

You can try Firefox Mobile (versions for mobile devices, as well as assemblies for Mac OS X, Linux, Windows) and learn more about it from the links, and you can learn about plans for Fennec 2.0 in my previous article . Under the cut - innovations and a small lyrical digression.

The most important innovations:
  • Rotate portrait mode
  • The new "smart" system of "tapov" (now you do not have to choose links and put checkboxes with the tip of the nail)
  • Improvements when working with forms (new topic of forms, autocompletion , etc.)
  • Homepage redesign
  • Automatically updated addons
  • The ability to open links in a new tab and save images using the new context menu
  • Improved Site Menu (the dialog available through there by the favicon in the URL field)
  • The ability to use the volume keys to increase / decrease
  • Save pages to PDF
  • Built-in Crash Repoter
  • The ability to add search engines to the main screen and access them through the Site Menu
  • Since the days of version 1.0, the range of add-ons has expanded. Firefox Sync and AdBlock Plus added 58 extras .
