Glass Opera 10.50 on Windows Vista / 7

One of the main innovations of Opera 10.50 for Windows is the full support for Aero effects, including the Aero Glass glass interface. However, this has not spread to all elements of the browser. Opera designers have managed to maintain a good balance between glass and classic graphics, but there are also those who want their favorite program's interface to be completely glass;)


Meet: Z1-Glass for the authorship of Z1-AV69 .

Actually Z1-Glass skin, which glazes all toolbars.
my.opera.com/community/customize/skins/info/?id=9281
Installation:
Installed automatically when downloading, after installation, you can switch between it and other skins in the browser settings.

Skin can be used with several variants of the address bar.
my.opera.com/Z1-AV69/blog/2010/02/11/3-different-addressfield-options-in-z1-glass-1-52
Installation:
1. Find the archive with the Z1-Glass skin (standard location: % appdata% \ Opera \ Opera \ skin \ ) and extract the skin.ini file from it, which will have to be edited a bit;
2. Starting from the 20th line there is an instruction in English. If everything is clear from it, we follow it and go straight to step 4. If not, read on;
3. Starting with line 35, there is a [Dropdown Addressfield Skin] section. It contains 3 different lines defining the background image of the panel. To select any option, respectively, 2 lines must be commented out, and one (necessary) left.
Options:
  • Standard:

    Tile Center = border/addressfield.png
    #Tile Center= border/addressfield_alternate.png
    #Tile Center= border/addressfield_classic.png
  • Combined:

    #Tile Center = border/addressfield.png
    Tile Center = border/addressfield_alternate.png
    #Tile Center= border/addressfield_classic.png
    For the combined version, an alternative address panel configuration is also required. The finished panel can be downloaded , and then saved in % appdata% \ Opera \ Opera \ toolbar \ and included in the settings of the opera;
  • Classical:

    #Tile Center= border/addressfield.png
    #Tile Center= border/addressfield_alternate.png
    Tile Center= border/addressfield_classic.png
4. Save changes to skin.ini and pack it back into the archive;
5. Restart the browser.
