KDE Project Announces Javascript Widget Design Contest

The KDE project announced a Plasma Javascript Jam Session . Participants are invited to create the most original and interesting plasmoid for the desktop, written using the Plasma JavaScript API . The works can be sent to the email javascriptjam@kde.org until March 31, 2010, after which the works will be posted on kde-look.org on April 2 , and the winners will be announced on April 9.

The code for all entries must be distributed under free licenses that are compatible with the main KDE codebase. It is possible to use third-party JavaScript libraries and data processing engines, however, the work itself must be fully created independently, without copying code from other plasmoids. Both individuals and development teams can participate in the competition, but only one plasmoid is accepted from each participant / team.

The criteria for the evaluation of works are:
  • 40% - the usefulness and applicability of the plasmoid for solving problems arising from users;
  • 20% - originality and uniqueness of the idea;
  • 20% - the quality of design;
  • 20% - technical skill and the use of the full range of capabilities Plasma.

The winner of the competition will receive a Nokia N900 Communicator, an attribute kit and payment for attending any of the KDE developer conferences, such as Akademy and Camp KDE. The runner-up will receive payment for a trip to the KDE developer conference and an attribute kit. The third place winner will be given a set of attributes. In addition, winners in competitions will be considered: the most stunning project, the most technologically advanced plasmoid and the most creative solution.
