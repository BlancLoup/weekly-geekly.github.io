jor1k: Linux in a web-enabled browser

What we have been waiting for has appeared.
Meet: jor1k: OpenRISC OR1K Javascript Emulator Running Linux With Network Support
  • Each virtual has a shared network with Internet access.
  • Each virtual has telnetd with root without password
  • Each virtualka has nmap;)
  • Virtualka gets IP via DHCP
  • There is support for framebuffer
The network is implemented in websockets from the server side and emulated by OpenCores ethmac from the VM side.

I have 120 MIPS with a kernel asm.js on my laptop.
More information in the author's blog .
