Firefox 3.1 Alpha is released


A new version of Fire Fox numbered 3.1, but still in the alpha testing stage. This version was given the code name Shiretoko. It is based on the new engine Gecko 1.9.1, which in turn allows you to use a number of new functions:

* Broader web standards support
* Text API for canvas elements
* Supports border-pictures
* Support javascript query selectors
* Several innovations in the address bar
* New page switch via ctrl-tab

Here is the official record of the developers: Mozilla Developers News
Here you can see a review of innovations: Arstechnica.com
