An updated version of Russian keyboard for iPhone has been released.


In the blog of Slava Karpenko , one of the developers of the Russian keyboard for the iPhone, a note appeared on the release of the updated version 1.0a2 of the Russian keyboard for the iPhone. A number of improved with respect to the first version of the keyboard appeared in it :
* Available landscape mode in Safari
* Fixed crash when exiting the program while typing and returning to it
* Autocorrection is automatically disabled if it is enabled. This is done to troubleshoot problems, because there is no implementation of Russian autocorrection yet.
If you install the Russian keyboard in your IPhone, then in SMS, Notes, Safari (besides the URL entry line) there will be a keyboard with the ability to switch to Russian.
Read more on Apple iPhone sites in Russia and on the developer blog.
