MakeMap - show yourself on the map

Hello% username%
I want to bring to your attention a new project dedicated to maps and visual creation of code for Google maps.

Some time ago, my friend wondered how to embed a Google map into your website? Having shown him the standard JavaScript code for embedding a map in my site, I heard that it is quite difficult for a non-specialist. And apparently for a person who is far from web development, this is the case. As a result, the idea was born which I submit to your,%% username%, court. Service implemented using Google Maps API + Mootools + Yourls (thanks to getnaked )

www.makemap.ru

Main features of the service:
  1. Embedding the map into your site without getting the Google Maps API code via iframe
  2. Short link to the point you specified. Put a marker, get a link, give a friend - he sees what you marked on the map.
  3. Getting a clean JavaScript Google Maps API code to embed in your site.

Service highlights:
  1. Specifying a marker with a design.
  2. Ability to select the viewing depth. Those. what depth you see on the screen and see on your site.
  3. The ability to select controls zoom.
  4. Ability to select the type of card. Again, which type of map (scheme, satellite, hybrid) you choose is current, and that‚Äôs what you‚Äôll see
  5. Map width (valid for embedding only)
  6. Map height (valid for embedding only)

Service benefits:
  1. No registration. (Perhaps it will be in the future, but not yet planned).
  2. All actions take place on one page.
  3. Embedding a map into your site does not require the receipt of a Google API Key.
    The third version of the API is used. She does not need to use the key.


PS: Example of a short link: http://y.makemap.ru/13
