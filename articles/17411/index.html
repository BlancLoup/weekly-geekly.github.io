Google Code 2007 Day in Moscow



I did not find references to this event on Habr√©, although it promises to be very interesting. I apologize in advance if it turns out that I simply do not know how to use the search ...

On Monday, a girl from Google called me at work, apologized for being so sudden, but she offered to attend the conference, which the company is already arranging on December 19 in Moscow. Where did she get the phone of the company where I work, I did not even ask - they know everything.

She sent me an e-mail in which there was a link to the company's blog, and this is what is written there:



The Internet is constantly evolving, and we at Google feel we are involved in this process. Over the past few years, much has changed in the network, especially due to the appearance of accessible APIs for working with web services. The web is becoming more open, and developers play a key role in this development. Therefore, for developers, we spend the day Google Code in Moscow, December 19.

Google Code Day is a one-day conference where Google engineers will not only present the API, but also conduct master classes and workshops. We will talk about the main APIs of Google today, as well as show what, and, most importantly, how to do it with them:

  • Open Social, for creating social applications that work with data from several major social networks
  • Geo API, for quickly creating geo-information services and web applications
  • GData API, for reading from writing data to the web, working with Google Docs and Spreadsheets via API


Not only engineers from Google Russia, but also our colleagues from Zurich, London and Mountain View will present API in Moscow. We will publish the final program of the conference a little later in the official Google Russia blog, but for now, sign up for Google Code Day December 19 in Moscow: google.ru/googlecodeday07 , participation in the conference is free.


Information taken from here: googlerussiablog.blogspot.com/2007/12/google-code-2007.html

Sign up for the conference here: google.com/googlecodeday07

Come! By the way, if you have a laptop - take it with you, it will come in handy.
