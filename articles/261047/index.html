Updated functionality on VDS.menu

Today I want to talk about updates on VDS.menu - a project for selecting virtual servers according to various criteria. Updates for the most part cosmetic, but they turned out quite a lot and they are all aimed at improving usability.





The appearance of the button for quick sorting and switching the display mode. Previously, changing the sort order was not very convenient - you had to choose a value from the drop-down list. The task was to make the possibility of sorting for the two display modes (tile and table) and different screen widths. As a result, I settled on two options: for wide and narrow displays. In wide tabular mode, the current order is highlighted in blue. The button to switch the display mode was previously hidden at the very bottom and many did not even know about it, but now you can switch quickly and easily.





Display information about the number of tariffs and hosters. Previously, the number of filtered tariffs could be recognized only by the number of pages (and then approximately), and the number of hosters remained a mystery. Now, with any search, you can see the total number of tariffs and the number of hosters that offer them.



Expansion of the list of currencies. Full list: Russian ruble, US dollar, Euro, Ukrainian hryvnia, Belarusian ruble, Kyrgyz som, Kazakhstan tenge, Swiss franc, Japanese yen. All prices are automatically updated every hour at the rate of the Central Bank. In the last update, it was possible to search for rates on the currency of the host, but now, with the current number of currencies, this is unnecessary.

Displays administration information along with other fare information. If the administration is paid, then a dollar sign is displayed, with one to five asterisks displayed in the administration tariff.



To save space in the tile version, the host‚Äôs logo is no longer displayed. The logo took quite a lot of space and did not have a functional purpose, but now the favicon of the site is visible and a link is available to go to the hoster site.

Search by age is available. I thought for a long time whether to add this parameter or not, but I decided that I could add as an experiment. The hosting age is determined by the date of registration of the domain and it is necessary to take into account the fact that this date may not correspond to the date of commencement of services.



For narrow screens, the tabular mode has changed and the content is displayed without horizontal scrolling.



In the wide tabular mode, you can now see the speed of the server channel - you just need to move the mouse to the graph indicating the amount of traffic.

When viewing information about the tariff, you can now see its value in different currencies. A button is available to select prices for 3 months, six months or a year.



Display in the hosters list the date of domain registration and the date of addition to the database. By both dates you can move the mouse and see how much time has passed since that moment.



Ability to search the list of hosters. You can search either by the name of the host, or by the name of the site (domain).

On wide screens, the tariff information page view has changed slightly. 2 blocks with the price moved to the right, and the block with the parameters has already become.



When viewing information about the tariff now you can see a list of similar in price rates. 6 tariffs are selected at a higher and lower price, the user sees 6 random ones.



Disappeared section with discounts. There was little discount and expediency in this section yet.

Unfortunately, the development of the project is not as fast as we would like, but in the next few months I plan to submit a similar project to search for virtual (shared) hosting.
