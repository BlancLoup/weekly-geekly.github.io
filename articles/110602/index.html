Firefox 4 Beta 3 for Android is released

Firefox 4 beta for Android and Maemo has been released. From the new:
  • New android menu view
  • Added the ability to upload files
  • Added ability to save pages to PDF
  • Fixed errors when opening links in other applications.
  • Fixed problems with support for on-screen and hardware keyboard
  • Full screen video
  • Copy-paste in the address bar

For developers, the following have been added: the localStorage API and orientation events on Android (this is how I understand the events related to the change in the orientation of the phone screen)

Well, the browser can now send reports when crashes (where without it). In this application, I think this feature will be in demand and will really help in improving the stability of the browser.
