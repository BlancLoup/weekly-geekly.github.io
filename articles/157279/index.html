A selection of tools for creating web interfaces in the style of Metro

I present to your attention a selection of frameworks, templates, jquery plug-ins and icons for creating interfaces in the style of Windows 8.



Frameworks and templates


METRO UI CSS is one of the most popular frameworks at the moment from the Ukrainian developer Sergey Pimenov.



BootMetro - template for Twitter Bootstrap, recently released a new version that can already be used on work projects.



metro-bootstrap is a similar development using LESS.



Metro UI template - another framework that is gaining popularity, is distinguished by quality and a large number of ready-made tiles: RSS, Twitter, weather, etc.



Droptiles.com is a Windows 8 style start page (similar to iGoogle).



PHP Image Gallery In Metro UI - Metro-style photo gallery.



jQuery plugins


jq-metro - plugin that helps to create some elements of the metro-interface: progress bar, switches, panorama, etc.



Metro JS - helps to create tiles and other elements.



The jQuery Metro UI Plugin is also the creator of the metro tile.



jQuery Mobile Metro Theme - plugin for mobile version.



Metro UI Theme Builder For jQuery UI - creating great metro elements.



TileJS (not jQuery) is a simple script for creating tiles.



Free icons


Modern UI Icons in formats .design, 48 * 48px PNG and as XAML-files.



Metro UI Dock Icon Set - in formats 512 * 512px .PNG, 256 * 256px .ICO and .SVG



Metro Icons - 130 .PNG Icons.
