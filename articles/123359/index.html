New release platform for creating online stores Synergy 0.60.0

Intro: Synergy is the cumulative Open Source extension of Spree , focused on the creation of online stores in Russia.
The project's mission is to provide all the necessary functionality specific to Russian online stores, as well as to simplify the integration of recommended extensions.

List of changes from Synergy 0.50.0 :
  • Added the ability to print instructions to the courier
  • Added payment method ‚ÄúSberbank Receipt‚Äù
  • Added ‚ÄúConfirmation‚Äù step for orders
  • Translated 500th error page and a letter with instructions for resetting the password
  • The generation of notifications in the administration panel has been improved; now they are declining by genera.
  • Updated versions of used extensions: spree_static_content, spree_editor, spree_robokassa
  • Added a few magic comments for Ruby 1.9.2, but the recommended version of Ruby is still Ruby Enterprise Edition.
  • Spree 0.50.x support removed, recommended Spree version - 0.60.stable
  • Improved appearance of the administration panel
  • Simplified installation procedure
  • Several minor bug fixes and improvements.

In addition, there was an updated version of the default theme - Synergy Default Theme 1.0.2

Also, due to numerous requests, we deployed a separate platform instance to demonstrate the capabilities of the administration panel: synergy.webbyapp.com/admin

Plans for the following releases:
  • The ability to print a sales receipt
  • Possibility of printing of post forms f.113 and f.116 for sending by cash on delivery
  • Batch loading product images
  • EMS Russian Post delivery method (with automatic cost calculation)
  • Legal support
