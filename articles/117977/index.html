MobTop - mobile site traffic analysis tool

Our team has been working in the field of mobile Internet for a long time. From the very beginning there was a need for a quality tool for analyzing the attendance of mobile sites. I think this need is not only ours. However, existing tools cannot fully satisfy this need.
Actually, that's why we decided to create Mobtop.ru - a rating of mobile sites and at the same time a system for analyzing their attendance.

I would very much like to know the opinion of the respected community about our project.
While we are in the beta stage, please do not judge strictly.

For a long time I will not paint, I will list only the main features.
We currently retrieve the following information:
1. Total attendance by days and hours (visitors, views)
2. Mobile operators by country and region.
3. Mobile devices:
- manufacturer, model
- screen resolution
- browser
- operating system
- cookies support
- ... what else to add?
4. + many small convenient buns.

It is also important to note that the Mobtop rating itself (or rather its mobile version m.mobtop.ru ) serves as a good means of exchanging traffic between sites. Mobile users are often not very experienced on the Internet. For many, the phone is the first and only way to access the network. Therefore, they are happy to explore new sites.

We hope that our project will be useful for all web developers and managers. Still, universal mobility is now one of the main global trends.
We will be grateful for any feedback, comments and suggestions. We are very interested in the development of the project.

Registration is limited for now, but we will gladly send invites to habrovans.
