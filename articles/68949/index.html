Hello to all! I want to share with Habr's audience the utility that my friend Eugene Domanov and I wrote.
The program helps not shat on the desktop and at the same time have all the shortcuts on hand.


Mouse Extender loads and hangs in the tray.
When the user clicks with the middle mouse button, ME is shown and you can add a new shortcut / folder / file or click on the already selected one and open the program you need. In addition to shortcuts to programs / folders / files, ME also has 3 system buttons - access to the Control Panel, restart and shut down the computer.

Attention! System keys work by double-clicking - in order not to accidentally restart the computer.

A brief list of Mouse Extender features:

1. Saving a list of shortcuts and folders and quick access to them
2. Access to system buttons - Control Panel, Restart and Shutdown Computer
3. Win7-style effects
4. Autoload
5. Disable Mouse Extender in certain programs. Often, people use the middle mouse button in games / browsers / editors / CAD systems.
6. Freeware and Open-source
The program is written in WPF and requires. NET Framework 3.5 SP1
