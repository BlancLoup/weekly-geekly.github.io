REALbasic 2007 Release 3 released


This news is interesting for those who are engaged in programming in the BASIC language.
12.06.2007 updated version of REALbasic. Today it is Release 3.

REALbasic is a rapid application development environment that provides the ability to transfer Windows applications created using Microsoft Visual Basic to Linux and Mac OS X operating systems. Using the same source code, these applications become cross-platform.
