Firefox 2.0 RC2: list of differences

The release of the final version of the open browser Firefox 2.0 is not far off, while the developers offer to the critics of the new "candidate". The version of Firefox 2.0 Release Candidate 2 demonstrates what the final release of Firefox 2 will be, which is just a short wait.

Compared to the previous beta version and the first release candidate, Firefox 2.0 RC2 has quite a few minor changes to the interface ( full analysis of the changes ).

Some significant innovations, including the Places interface and an improved bookmarking system, were excluded from the program for refinement, and you can see them in future versions. The changes also affected the appearance of the application and nested windows. Perhaps the latter was done under the influence of Internet Explorer 7. In addition, the browser has been protected against phishing and the extension manager has been fixed.

You can get acquainted with the detailed list of Firefox 2.0 RC2 features and download the distribution package on the Mozilla website .
