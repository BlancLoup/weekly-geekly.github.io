Google has stepped into a bright future

Google, in a press release today, announces "the first step towards a universal search model." The embodiment of this step was the updated main page - google.com .

The company said that they are moving to a universal search model, thanks to which visitors of Google.com will be able to use a comprehensive search integrated with numerous services. The model is based on the assertion that the user is interested in finding out comprehensive information about the search object. ‚ÄúOur goal is to make the search for our users as simple and clear as possible,‚Äù said Marissa Mayer, vice president of Google.

The new design of the main search page, according to the developers, should emphasize the desire for this goal. In the python search results, it is automatically suggested to use additional search services and search for this word in blogs, books, groups and program codes: in the new search results design the links to these services are displayed in a more prominent place.



At the top of the page, a menu is now displayed with links to many Google search services. (The similarity of this menu has long been familiar to GMail users.) Accordingly, thanks to this menu, you can quickly access these services and search for maps, pictures, news, patents, etc.



Google Labs continues to work on improving search services. An updated section of the Google Experimental offers to view the search results, distributed across the time line, by geographic location, etc.
