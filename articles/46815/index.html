Skype 4.0 Beta 3 for Windows

Skype has announced the release of the next beta version of the newest Skype 4.0 Beta client for Windows (release 4.0.0.176). This is the third preliminary release of Skype 4.0 Beta, which, as usual, has brought many changes and improvements. The program gradually gets rid of a lot of problems noticed by users and is filled with features available in a stable release 3.8.



For example, in this update, the usual program features such as Event History , Skype Prime service, the ability to quickly add Outlook contacts to the program became available.

Among the updates in this release you can see:
Custom Chrome - a new style for the program interface
Abuse reporting - the ability to send a report on the actions of users abusing communication and sending spam.
The personal account access panel from the program has been updated and become more informative.

But still, the main efforts were aimed at correcting the many errors of previous beta releases, among which the propensity of the program to crash when running in Windows XP SP1, as well as errors of notifications about events, was particularly highlighted.
