Cezurity's little antivirus test


Recently VKontakte began active promotion on all fronts of the new antivirus.
I decided to see the effectiveness of this Cezurity anti-virus scanner against several viruses in action.

So, what we have is:
  • Virtual machine with clean Windows XP SP3 and a pair of installed programs
  • Cezurity antivirus scanner


Now let's try to test a couple of viruses. The source will be the site vxer.org (formerly vx.netlux.org)

The testing method is as follows:
Download, unpack zip, run a computer scan. Run the executable. And, if possible, scan the computer again. After virtualka restored to its original state.

Results:
Title VirusTotal Detection Rate Identifying Cezurity
Virus.Win32.Sality 51/57 Not
Trojan.Win32.Agent 44/55 Not
Virus.Win32.Zevity 40/57 Not
Worm.Win32.PornRun 44/50 Not
Worm.Win32.Torun 44/51 Not
Trojan.Win32.Bum 45/57 Not

Actually, this is all you need to know about Cezurity antivirus.
