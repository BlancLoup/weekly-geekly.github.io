New Microsoft logo

For the first time in 25 years, Microsoft has completely changed its corporate logo. Now it is more modern and fits perfectly into the new design style of the company's products.

Mobile version of the site or responsive design?

Starting to create a modern site, everyone should make a choice for themselves, how to deal with mobile users - either to create a separate mobile version of the site, or to introduce an adaptive design. This question and tried to consider the author of the article, taking as an example the sites of US presidential candidates.

Release Bootstrap 2.1.0

Despite the fact that the project Bootstrap all year, he managed to catch the fancy of many developers and become extremely popular on the web. Meet the long-awaited update, which included fixes for bugs, new plugins and components.

Google introduced a new benchmark - Octane

A new player has appeared among the benchmarks that fans are interested in to measure the performance of browsers. In addition to the standard js tests for Google, Octane offers to drive your browser to js libraries that are actually used.

The history of the BEM from Vitaly Kharisov

Probably, there is not a single front-end developer in the post-Soviet space who would not know about BEM. However, not everyone knows how it was created, what underlies the methodology, and what technologies are best used at work. About all this from the mouth of the creator and ideologist of BEM Vitaly Kharisov.

Full size page animation with CSS

Some of the new technologies supported by the new IE10 browser are CSS 3D Transforms and CSS Animations . You can read about the syntax and possible application of these technologies as a new concept of navigation between pages in the msdn blog .
