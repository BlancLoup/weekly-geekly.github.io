Mail.Ru: Russian Developers Cup - accepting applications

Mail.ru Group launches a program for those who create custom web-, mobile services and games.
From October 01 to December 01, 2013, participants will develop prototypes in one of the following areas:
- applications based on social networking API (preferably for My World and Odnoklassniki),
- mobile applications using the functionality of any Mail.ru Group web service,
- games for social networks and mobile platforms.

Any platform and development tool can be used.
The main task of the participants is to take a fresh look at familiar industries (for example: mail, IM, search) to invent and implement new services based on them.



The best projects become residents of the IT Park Business Incubator.
Registration is until October 29.

All details here: russiandevcup.ru
