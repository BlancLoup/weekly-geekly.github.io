Anonymous introduced their own software for Twitter



Anonymous group this time decided to distinguish itself not by breaking into or any DDoS servers. No, representatives of this group presented software to track the most interesting and popular topics in the Twitter microblogging service. Thus, the user, having found a popular trend, will be able to ‚Äúget into the stream‚Äù with his own message, which will be noticed by quite a lot of people.

This software is called Universal Rapid Gamma Emitter (URGE), and so far it works only under Windows. Anonymous reported that this software is not an exploit or any other tool for hacking. This is really the usual software designed to detect popular trends on Twitter. According to the developers of URGE, they are all tired of pop themes on Twitter, and this tool will help the user find a trend that is really popular and useful for a specific user, without having to shovel dozens of trends about celebrities and their behavior.

Anonymous believe that the Twitter team itself could have released something similar, and for a long time already, since the need for such software is obvious. In fact, Twitter chooses those popular trends in which the maximum number of people will obviously communicate. The group is dissatisfied with this fact, and therefore the software, referred to above, was born.

It is clear that with the help of URGE you can share with other news from Anonymous. But in addition, this software allows you to create your own popular trends, attracting users of the microblogging service to discuss any important issues.

In general, the description given by Anonymous for its software is rather vague, and you can try out the program yourself by downloading it here .
