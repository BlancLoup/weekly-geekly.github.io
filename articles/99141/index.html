The final release of openSuSE 11.3

It remains a few hours before the final version of openSuSE 11.3.

About 8 months have passed since the release of the previous version. During this time, novell and the community have brought many innovations.

Here are some of them:

Netbook support

Netbooks are becoming more popular every day. And if earlier the installation of SuSE on eeePC was associated with a lot of problems, then with the release of version 11.3, everything changed.
This version has good compatibility with the equipment of netbooks. The KDE environment will delight owners of these devices with a mode optimized specifically for netbooks (netbook mode).



BTRFS

This file system is considered as the next step after ext4 and as a competitor of the ZFS file system from Solaris. Previously, she was already in SuSE, but only as a test, now it is stable. This was largely due to the new 2.6.34 kernel, where BtrFS support is implemented at the proper level.
To be fair, I must say that you cannot boot from the partition to btrFS so far and you need to do / boot under another FS.

Server and vitalization
As for the servers, there is also a certain set of improvements:
  • New MariaDB Package
  • MySQL Cluster Package

Virtualization is represented by the following packages:
  • Xen 4.0
  • KVM 0.12.3
  • VirtualBox-ose 3.2.6


Lxde

Previously, SuSE offered a choice of 3 work environments: KDE, GNOME, and XFCE. Now it is possible to put a lightweight LXDE.
I am not an ardent fan of this DE, but I use it on my servers running GNU / Linux Debian and will definitely try it on openSuSE servers.



Improvements in KDE and GNOME
OpenSuSE 11.3 introduces KDE version SC4.4.4.
It includes:
  • Touchpad support added
  • KTorrent now with DHT
  • KUpdateApplet now allows for an update between versions.
  • In Dolphin / Konqueror, you can view the contents of ISO images.


GNOME introduced version 2.30.1:
She is notable for the following:
  • GNOME 3.0 preview available
  • Tracker replaced beagle
  • Access from iPhone and iPad directly from Nautilus
  • Empathy's IM client now has the ability to add files for opacity by dragging and dropping.
