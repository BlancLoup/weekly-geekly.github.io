The first stable version of the JVM-language from RedHat - Ceylon

The first stable version of the JVM-language from RedHat - Ceylon - 1.0.0 has been released.

The language has static typing, type inference, mixins, algebraic types, declarative syntax for tree structures, lambdas, higher order functions and so on. I will not rewrite everything, because This is all in the quick guide on the site . Naturally, the site is in English, but because infa in Russian about this language will not appear soon, it makes no sense to know about this news to those who do not understand English, no.

The language was developed together with the Eclipse plugin, so the development environment is also ready.
