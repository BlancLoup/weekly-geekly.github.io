StarStar Number - just call or alternative to QR code

We are all familiar with the technologies of bar code data representation (Bar Code) and its more modern version - QR code (2D Bar Code type).

The first option is now used everywhere as a label and product identification.



The second one was increasingly used as a quick access to Internet resources from mobile platforms (iPhone, devices based on Android, Windows Mobile, etc.). You are photographing a QR code, a special application reads the encoded information and opens a page in the browser with the URL specified in the code (although other actions besides opening the browser can be performed). It seems cool, but still annoying to many that you need to take pictures, then perform some actions.



Most recently, I learned about the existence of an alternative way to get quick access to various information from mobile platforms - StarStar Number technology from Zoove ( www.zoove.com ).

As far as I understand from the information on the site, the technology works so far only in the USA and provides the owners of mobile phones and smartphones with the opportunity to access various information simply by dialing the short number ** CODE (similar to how you can get information about the balance * 110 * 04 # or how there) and you will open the desired page in the browser or perform some other action. Works on iPhone, Android, BlackBerry, Windows Phone, Palm, Symbian devices.

It seems that the company Zoove is the provider of these shortest numbers (StarStar Number) and interacts with telecom operators. In order to use the capabilities of the service, you must register a free number and set up a mobile campaign.

Some types of content that can be used for mobile campaigns:

  • Applications
    Downloading applications on the phone / smartphone
  • the Internet
    No need to enter a web address
  • Videos, ringtones and wallpapers
    Distribution of video, audio, photos - always and everywhere
  • Information
    Stay in touch with product information
  • Share
    Share StarStar Number in various social networks
  • Voting
    Just call to vote - no longer need to send a text keyword to a short number
  • Donations
    Receive donations
  • Subscriptions
    Newsletter Subscription
  • Coupons
    Save money with mobile coupons
  • Gift cards
    Gift cards for later use in stores
