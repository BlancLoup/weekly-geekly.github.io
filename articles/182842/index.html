Creating OLAP cube in MS SQL Server 2012

Hello.

I needed to use cubes to analyze OLAP data. I actively studied and used this technology in 2001-2002 and even certified it. But then for a long time did not touch her. And just recently I tried to use 2012 SQL Server, but it was not there: a lot of things and what appeared in the new server turned out to be completely unfamiliar to me.

After spending some time getting to know each other, I decided to share my skills with the community: I recorded a training video on how to create a cube and measurements.

The video is an introductory, perhaps some things are done incorrectly, but the example is quite working. It is assumed that the reader (viewer) is already familiar with the concepts of multidimensional cube, dimensions, measures, and what it is used for.




The example used the Northwind database from MS SQL Server 2000. An excellent database, small, simple, understandable - what you need for training. You can get it on the Microsoft website ( http://www.microsoft.com/en-us/download/details.aspx?id=23654 ). To deploy to SQL Server 2012, you need to install the package, find the database creation script file and run it in SQL Server 2012.
