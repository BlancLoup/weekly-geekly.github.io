Equil Smartpen 2: pens can also be smart



In a world where not only computers, but even refrigerators are getting smarter, more and more hybrid devices are appearing at the junction of the real and virtual worlds. Equil Smartpen 2 is a smart pen that translates an image or record created by the user on paper, into electronic form, on the display of a paired mobile device.

Interestingly, funds for the launch of mass production pens were collected by developers at Indiegogo, and 200 thousand US dollars were raised, with the required amount of 50 thousand.



Features Equil Smartpen 2





You can use the Equil Smartpen 2 pen as an ordinary recording tool on paper - inside is a completely ordinary ampoule with ink for a ballpoint pen. Replacing the ampoule is not a problem, everything is done as with a regular pen. Everything that the user draws or writes is perceived by the receiver (it must be placed in the upper part of the sheet, for which a magnetic clip is used). In order for the receiver to work correctly, you need to set the required paper size (A5, A4 and others, but the maximum size is A4).





At the beginning of work, you also need to connect a pen to a mobile device, where all data will be transmitted. To do this, you must install the appropriate application, or rather, two applications at once:

Equil Note. The application allows you to create handwritten notes, followed by structuring, saving and editing text. Digitized handwritten characters are converted to block letters, numbers and signs. The application is Russified.



Equil Sketch. The program is designed to recognize and digitize images. The application interface contains many tools that allow you to use different styles, brushes, colors, layers. The program has a Russian version.





After that, you can start to pair the pen and the mobile device.



The developers have provided the following set of features and capabilities of the Equil Smartpen 2:
  • The ability to recognize 11 languages ‚Äã‚Äã(Russian is also included in the set);
  • Layering work with graphics;
  • Fixing the level of pressing the pen on the pen, which allows copying a record or image from a sheet into the memory of a mobile device;
  • The pen works with any type of paper;
  • The form factor of the pen is normal - neither the weight nor the shape is much different from the usual ballpoint pen. Disadvantages at work does not occur;
  • The pen very accurately digitizes the image, transferring all the features of the drawing or recording;
  • The ability to transfer data to the "cloud" or e-mail;
  • Equil Smartpen 2 can be used as a stylus for some devices;
  • The autonomy of the device is up to 8 hours; when the battery is low, the quality of image digitization decreases (recharging takes place when the pen is connected via a data cable with a mobile device or PC).




Among other things, the developers have provided the pen with its own flash memory, the volume of which is enough to save up to 10 thousand pages of handwritten text. As soon as it becomes possible to synchronize with a mobile device, all this information can be transferred to a mobile device or to a PC.



Equipment





The kit includes pens:
  • Equil Smartpen 2;
  • Receiver;
  • Docking station (made in the form factor of the pencil case, here you can store all the necessary accessories for the handle);
  • Data cable;
  • Instruction;
  • Cap.




Compatibility



The pen can work with mobile devices on iOS and Android OS. In addition, the device is compatible with Mac and Windows. Communication with devices is via Bluetooth.
