Veeam Management Pack for VMware v6.5 released for Microsoft System Center: what's new?

If you are using Microsoft System Center and VMware vSphere at the same time, you most likely would like to monitor VMware through System Center, since using several monitoring tools at the same time is not very convenient. System Center can work in heterogeneous virtual environments:
  • Operations Manager can monitor virtual machines, guest OSs and applications in them.
  • Orchestrator has VMware integration packages.
  • Virtual Machine Manager can create and move, as well as perform various actions on virtual machines running on the VMware platform.

Veeam Management Pack for VMware (Veeam MP) extends the capabilities of Microsoft System Center in monitoring and managing VMware virtual infrastructure. Moreover, starting from the new version (v6.5), Veeam MP allows monitoring the backup infrastructure of Veeam Backup & Replication .

Veeam MP analyzes more than 1,000 indicators and events, turning them into convenient alerts, dashboards, and reports. While System Center provides extensive monitoring capabilities for applications and operating systems, Veeam MP allows you to monitor ‚Äúvirtual hardware‚Äù (virtualization servers of VMware vSphere), virtual machines, and all hardware data center resources (network, storage, computing power).

Last week a new version of Veeam MP (v6.5) was released, and in this post I will briefly tell you about what's new in it.



Fault tolerance of the monitoring and control process

If the vCenter fails, Veeam MP v6.5 automatically switches to operating mode through direct access to the vSphere virtualization servers. Of course, some functions may not be available. When vCenter is restored, Veeam MP automatically returns to data collection mode through vCenter.

Configuration control and event correlation

The ‚ÄúConfiguration Tracking and Alert Correlation‚Äù report allows you to see what changes have occurred in the virtual infrastructure of VMware over a period of time, and to perform a causal analysis. The report includes detailed data on all objects of the virtual environment, including clusters, data storages, network resources, hosts and virtual machines.


Monitoring backup infrastructure Veeam Backup & Replication

Now (starting with version 6.5) Veeam MP allows you to monitor the elements of the backup infrastructure of Veeam Backup & Replication: backup proxy servers, backup repositories, WAN accelerators, backup tasks, etc.


Support for the Veeam B & R topology is implemented in a similar way to VMware topology support.


In addition to real-time monitoring, the product also allows you to build forecast and analytical reports on the Backup & Replication infrastructure data. For example:
  • report ‚ÄúProtected VMs‚Äù, which allows you to see which virtual machines are backed up using Veeam B & R

  • reports ‚ÄúBackup Copy Job Overview‚Äù, ‚ÄúRestore Operator Activity‚Äù and others



The new version supports VMware vSphere 5.5, Microsoft Windows 2012 R2, and Microsoft SCOM 2012 R2.
