The first security updates in Windows 7

Yesterday, February 24, Microsoft released the first patches for its new operating system Windows 7 beta. At the same time, the built-in update system must pass the reliability test.

A total of 5 patches are presented that close security holes - correct a bug or add a new feature. I note that the operating system itself will not fundamentally change.

"These updates will allow us to test the issue, reception and installation of patches on Windows 7," - said Brandon LeBlanc.

PS Updates downloaded and installed. It all ended successfully, no changes in the work have yet been noticed.
