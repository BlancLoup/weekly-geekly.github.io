Windows Vista runs out of memory while copying files.

Incredible bug found in the operating system Windows Vista. It turns out that when copying a large number of files in one operation, the system may run out of free RAM, even on the most powerful PC with gigabytes of RAM. This bug was planned to be fixed in Service Pack 1, but the patch was never included there, so to fix the glitch you need to download a separate patch .
