AWS: New EBS speeds and performance tests

Hello!

At the end of last year, I conducted a small disk performance test on Amazon and wrote an article about it. More recently, Amazon Web Services announced new features in EBS with a guaranteed IOPS number of 4,000 . I decided to write not just news, but continue the tests with these discs and tell the community about it.


For the tests, I chose the m1.xlarge instance type with EBS optimization . With 400 gigabyte drive attached with 4000 guaranteed IOPS.

Like last time I ran tests: 
# hdparm -tT /dev/xcdX 

 # dd if=/dev/zero of=tempfile bs=5M count=1024 conv=fdatasync,notrunc # echo 3 > /proc/sys/vm/drop_caches # dd if=tempfile of=/dev/null bs=5M count=1024
