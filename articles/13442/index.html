Windows vista works worse on the network because of the music

After the release of the Windows Vista operating system, users immediately began to discuss a strange effect: lower network performance compared to WinXP. There was no explanation for this; no changes to the Windows settings and network connection settings helped. And now the first reliable hypothesis was discovered : it turns out that network performance decreases by 5-10% if the music is playing, that is, the software media player is running. Why this happens - no one can understand again, but the facts speak for themselves.

If you start to play the MP3 file, the network manufacturer immediately decreases. Dozens of users ran tests on different PC configurations and confirmed this fact. The strangest thing is that there is no performance degradation during video playback.

On Slashdot there is a discussion of this strange phenomenon . Some versions are very funny .
