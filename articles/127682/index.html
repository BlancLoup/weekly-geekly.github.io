NASA has released a browser 3D application ‚ÄúEyes on the Solar System‚Äù



Probably most habrazhiteli understood what was going on, after reading the name of the program. Indeed, NASA has released a browser-based application showing a three-dimensional model of our solar system. If you wish, you can move to any point of the model, if you are curious to see what is in a certain point in space. In addition, the application allows you to "speed up" time, watching the movement of the planets, their satellites and artificial objects created by man.

Everything you need for the application can be downloaded here . The main component that allows this application to work on a PC is the Unity Web Player, for PC or Mac. Once you have downloaded and installed all the necessary components, you can start working with the application itself, watching the movements of the planets, asteroids and the Sun.

In addition, you can use individual modules of the program, such as tracking the Juno expedition, which lasts about five years.

According to developers from NASA, this application is the first such opportunity for users when anyone can see the entire solar system in detail, plus track NASA missions in real time. The application, according to the same developers, was released to popularize science, in order to make the data and experience accumulated by NASA accessible to everyone.
