Ubuntu 14.10 (Utopic Unicorn) is available for download.



A few hours ago on the Canonical servers appeared links to download images and torrent files of the new version of the distribution. On the main page of the announcement is not yet, but the announcement is already there, you can download the image here . As always, I urge you to download .torrent and become a distribution.

There are no revolutionary changes, nothing new in the interface either. From interesting things, we can note the appearance of the ‚ÄúUbuntu Developer Tools Center‚Äù, which makes it easy to install the Android SDK and all related utilities.
