Five useful services for social integration of your site

We all programmers love to write bikes. Of course, this is an interesting lesson, it helps to develop, but sometimes there is a reason to use other people's solutions that have already been proven by time. Information about the services that add a bit of sociality to your site without any programming, I decided to collect in this article. Some have probably heard about many, some will be new. If at least one person finds something new in this post, then this text was not written in vain!



Cackle

http://cackle.ru/



Allows you to leave comments on the page, using the existing account in ... yes a lot of where. It supports Google account, Vkontakte, Facebook, Twitter, Odnoklassniki and 12 more social networks for authorization. Plus, you can leave a comment just by entering the email and captcha (when you try to leave a comment, a window will appear with the link "or login anonymously").

In principle, it is free, but if you want to save data to your database at the same time and use the advanced API (with the possibility of customization and, in fact, write your own admin panel to manage comments) - you will have to fork out for a crazy amount - 250r. per month.
