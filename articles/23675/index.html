The man in terms of IT

The man is a very well protected system. He can open the skull, but you can not get root, run the code to execute or modify files. It cannot be restarted, interrupted by any process or returned to factory settings. There is, of course, hypnosis and intoxication, but this is all doubtful.

Man is a closed system. For communication with the outside world, it has five APIs for receiving and transmitting data. There is no intelligible standardization or documentation for them. Through the API, you can send commands, but from a certain age, the security system blocks the sending command on all APIs.

Man is an intellectual system. He alone can modify his program. For modification, it uses data obtained through the API. The highest priority in the system is the process of receiving information of interest. Many other tasks can be unloaded for its sake.

The only way to interact with such a system is to submit information of interest to it. If cheloek decided to learn about the fight, let him take part in it. If a person said nonsense, support in it a rational grain. If a person cannot do something well, he will first have to do it badly.

The main thing is to trust a person and support development. What is right and what is wrong a person defines himself.
