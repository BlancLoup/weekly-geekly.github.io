256 GB SSD


Solid state SSD drives have already gained popularity, but are mostly still used in small gadgets like MIDs, cameras, netbooks, etc.
It is possible to connect SSD to PC for a long time. Several varieties of connecting drives via PCIExpress have been implemented, and on this platform it was possible to build a system with more than a terabyte of memory on board (that's just the price of such luxury is quite high). But roomy SATA SSD is not so common yet, apparently because the company Corsair decided to release just that.
Model Storage Solutions P256 can store up to 256GB of data. Filling it mainly from Samsung: flash memory Samsung MLC, Samsung Controller IC. 128 megabytes of cache memory and the ability to write / read at speeds of 200/220 megabytes per second should meet the needs of modern fans of powerful systems.
