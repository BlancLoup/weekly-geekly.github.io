City for Windows Phone 7

Something that Windows Phone 7 users have been waiting for in Russia has happened! The city directory of organizations based on 2GIS data has been published.

The City application allows you to:
  • search for organizations by name, business, telephone,
  • look for organizations located near
  • view information about organizations (contacts, schedule, branches),
  • search for buildings, streets, stops, etc.
Data is available for all cities in which there are 2GIS.

The use of 2GIS cards in the mobile application is prohibited by the API user agreement, therefore the user is offered the choice of the following:
  • Google maps
  • Nokia Maps,
  • OSM Mapnik and
  • OSM Osmarender.
