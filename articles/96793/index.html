Sync extensions in Google Chrome



Good evening, Habrahabr. It happened that all users of the Google Chrome browser waited so long. The latest version of chromium has now learned to synchronize extensions.

In order for the browser to learn this useful feature, you need to do 2 simple things.

1. Update Chrome for Windows or Linux to version 6.0.437.1, for Mac to version 6.0.437.2.
2. Add the key "--enable-sync-extensions" to the application shortcut.



To do this, click on the chrome icons with the right button, select Properties and add this value in the Target field.

All congratulations!
