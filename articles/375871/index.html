Paperwork: A Free Alternative to Evernote

The popular web service for creating and storing Evernote notes will soon have a free alternative - Paperwork . Thanks to open source, everyone can install Paperwork on their server and provide access to users via a local network or via the Internet.

Demo access is available at demo.paperwork.rocks . After registration, a demo account is created to create and edit notes. The database is deleted daily at 3:00 am (CET). For registration, you can use a random email, it is not checked.



The project was launched quite recently, so that it is still in active development (see the code on Github ). API documentation is constantly updated (it is still far from complete). To version 1.0 is still far away.
