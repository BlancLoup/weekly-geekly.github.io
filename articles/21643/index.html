Searching for objects using keywords and phrases is a standard way to search the Internet. Yes, you can optionally set filters by country, language, file type, or domain. But a properly matched search query is the key to success. The problem is that it is not always possible in principle to formulate a text query. The only way is to use content as a request.
When working with the information retrieval service for images on picollator.ru, this rule remains in force. Instead of a text query, you need to pick up a request-image. In other words, you need a fairly good photo of the person, information about which we want to find.
What kind of photo should I submit to search for optimal results? To begin with, look at the FAQ, published on the service website at www.picollator.ru/ru/faq and scroll the page to the question ‚ÄúAre there any requirements for uploaded photos?‚Äù. Carefully read the answer:
