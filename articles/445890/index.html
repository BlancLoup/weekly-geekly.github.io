PyTest Neo

One evening, contemplating endless tests and looking at running points, the thought slipped through - how similar it is to the Matrix! So the idea was born, to make visualization of tests in the form of the Matrix.



Under the cut animation and some details.



If someone is worried about readability, then only the progress bar is animated, and if there are errors, then a standard report is displayed.

To install the pytest-neo plugin, just run: 

pip install pytest-neo

The project is in the early stages of development. Linux and presumably MacOS supported. Possible bugs. For now, verbose mode is left as is. The plans in it to implement a credible animation from the Matrix.

Please write your wishes and suggestions. This will greatly help the development of the project.
