Activating TRIM SSDs on Mac OS X

Soon the official announcement of Mac OS X 10.7 Lion, one of the key functions of the new OS will be a hidden recovery partition, AirDrop functions, as well as support for the TRIM team, which increases the speed of rewriting on SSD drives.

What is the TRIM feature? The fact is that when operating, the operating system notifies the ssd disk that the data blocks are no longer used and can be cleaned by the disk itself. What does TRIM give? TRIM team optimizes ssd-disk performance and maintains its speed for a long time. Learn more about the TRIM feature on the wiki .



By default, the team only works in the latest generation of new MacBook Air and MacBook Pro with native SSD drives. However, craftsmen from groths.org have found a way to extend this support to other Mac computers with any drives. Using the TRIM Enabler utility, activating this command for SSD is reduced to pressing a single button.

This command is supported by the latest generations of MacBook Air and MacBook Pro with native ssd drive. And again, the owners of previous versions remained unsuccessful, because the previous versions of macbooks were also packaged with ssd disks, but the craftsmen from the groths.org website found a way to activate TRIM on older versions of MacBooks. The program is called TRIM Enabler . Download TRIM Enabler: groths.org/zeus/TRIMEnabler.zip .

FIX! Activates TRIM in one click. If the program gives an error, you need to restart your Mac in safe mode.
