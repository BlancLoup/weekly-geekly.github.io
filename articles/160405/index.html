ID Software continues to publish Doom III source



ID Software began to upload the source code for its game engines about a year ago. Now the repository of the company is replenished with the Doom III BFG edition engine. This is an updated version of the engine Doom III with improved graphics, sound effects, support for three-dimensional displays, virtual reality helmets. Full list of differences can be found here . The code is published under the GNU GPL license.
