Agnitum Releases Outpost Security Suite Free

On November 22, 2010, the Russian company Agnitum released a free version of its integrated solution for protecting personal computers under Windows - Outpost Security Suite Free. The package includes antivirus, firewall, proactive components (protecting the system from data leakage) and antispam ... Unlike popular Internet Security class solutions on Download.com from Comodo, the product contains antispam and has certified its antivirus, and is distinguished from Lavasoft Internet Security also the presence of a firewall and proactive protection. Download Outpost Free at free.agnitum.com/download.php

The differences between the free version and the paid version are very significant. The free version in one form or another contains all the modules: firewall, email protection, proactive protection, antivirus scanner, content filtering. At the same time, the work of almost all of these modules is very limited. In particular, in the paid version, in contrast to the free

- you can run a scheduled scan,
- firewall can detect attacks
- it is possible to block ads, specific domain names and IP addresses,
- additional proactive protection components work,
- there is a Russian-language interface.
