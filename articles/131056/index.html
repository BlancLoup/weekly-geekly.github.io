TUF SABERTOOTH X79 - New ASUS Motherboard for Socket LGA2011

ASUS has introduced a new motherboard on the Intel X79 Express chipset for socket 2011. TUF SABERTOOTH X79 supports Intel Corei7-39xx processors with 4-channel memory controller. The board features 2 full-fledged PCI-E 3.0 x16 with support for AMD CrossFireX and NVIDIA SLI.

Since the board is a logical replacement for the X58 chipset, for comparison of performance, the closest possible configurations were chosen:

1. ASUS SABERTOOTH X58, Intel Core i7-970 Processor (6 Cores, 12M Cache, 3.20 GHz, 130W), 12GB DDR3 1333

2. ASUS TUF SABERTOOTH X79, Intel Core i7-3930K Processor (6 Cores, 12M Cache, 3.20 GHz, 130W), 12GB DDR3 1333

SPECviewperf11 x64bit

The first test was performance testing with the SPECviewperf test suite. In both cases, the NVIDIA Quadro 4000 graphics card was used. The results fell short of expectations, since we already observed a similar gap when comparing the X58 and Z68 platforms


PCMark, 3DMark and WinRAR

In tests of processor performance, the Sandy Bridge-E platform showed a more significant lead. The i7-3930K processor, although equivalent to the i7-970, in the WinRAR test even outpaced the i7-980X and i7-990X processors.


CINEBench R10

In testing with the CINEBench R10, the new processor showed an undeniable superiority with an impressive margin.


CrystalMark 2004R2

In this test, you can observe the acceleration when working with memory, which provides a 4-channel controller.


In conclusion, I would like to note several features of the motherboard. Included is a fan that is installed on a remote radiator that removes heat from the VRM module. It is small, but quite quiet. The second fan cools the chipset's heatsink.
There are two SATA-III ports on the motherboard with SSD Caching support. They can be used to connect two productive solid-state disks and use them as a cache of the disk subsystem.
Well and on trifles: on the board there is a connector for outputting two USB3.0 to the front panel of the case; There are two external SATA ports on the back panel - one eSATA 6Gb / s, and the other Power eSATA 6Gb / s; The platform supports almost 3 times more memory than X58.
But in general, the platform showed predictably good results, and the new 6-core Sandy Bridge, in multichannel applications, will be a justified replacement of the first-generation i7 processors.

Testing was carried out in the laboratory DEPO Computers
