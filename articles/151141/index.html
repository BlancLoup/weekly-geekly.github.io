o Quezal (excerpt from Wikipedia)
Quezal, or Quezal, or Ketsal (Latin Pharomachrus mocinno) - the largest bird trogon-like order. The length to the end of the tail is 35 cm; the same length is two of the longest supra tail feathers, the other two are shorter. It feeds almost exclusively on okotei fruits, which it swallows whole, but occasionally eats insects, small tree frogs, lizards and snails. Inhabits the cold mountain forests from southern Mexico to Panama. In the dry season, these birds can be found at an altitude of 3,000 meters above sea level, and in the rainy season they descend from the highlands to altitudes up to 1000 m. The sacred bird is in the ancient Maya and Aztec. They considered him the god of the air and his long green tail-feathers used in religious ceremonies. However, the birds for this were never killed, but caught alive, plucked feathers from them and released.
In modern Guatemala, Kwezal is the state symbol of the country, the national symbol of freedom. It is believed that, deprived of freedom, he dies of a broken heart. Kvezal is depicted on the coat of arms of the state, the monetary unit there is called quetzal.

Unity


In Ubuntu 12.10, a new version of the Unity shell will flaunt itself - 6.8.

There is a new feature for previewing files and applications - Previews .



In the application preview, the ability to install this application has been added, while the installation is not carried out through the Ubuntu Application Center (Software Center), but with the help of aptdaemon .



Added ability to close windows in browse mode.



... and many more nice looking changes.

Dash lens


The search interface has the ability to search and filter photos by tags and EXIF ‚Äã‚Äãdata. This function works both with local images and with photos stored in external services - Flickr, Facebook and Picasa (for this, the user must configure Online Accounts).



Search on social networks is available - messages, links and pictures from social services Twitter, Facebook, Flickr and others. Combined with the Unity Previews feature, messages can be viewed as well as responded directly from Dash.

Sync menu


The sync menu is designed to manage and monitor Ubuntu One and Dropbox.





Ubuntu One is supported by default, Dropbox is still in the plans



Messaging menu


The notification menu is also not left unattended developers. This time, the display of the current status in the Gwibber, Thunderbird and Empathy programs is added.





Unity greter


Added the ability to connect to a remote desktop directly from the login menu. Three connection methods are available:
  • ICA (Citrix)
  • RDP
  • VMWare View

This feature is already built into the system and does not require the installation of additional packages. To connect to a remote table it will be enough to enter your username and password from the USSO service (Ubuntu Single Sign On). Settings for all connections are stored on Canonical servers.



By default, a set of applications is activated to ensure the work of people with disabilities;

Wayland


We will not see the graphical subsystem based on Wayland and the Weston composite server in Quantal Quetzal, however everyone can test this miracle using ppa . Inclusion in the basic delivery, as well as the fate of Wayland, will be decided at the Ubuntu Developer Summit on 13.04, which will be held from October 29 to November 1 in Copenhagen (Denmark).

Libreoffice


The free office suite developed by The Document Foundation has been updated to version 3.6.1, and has also been integrated with the HUD interface, support for the global menu has been added.

Nautilus


Due to the significantly reduced functionality of the Nautilus 3.6 file manager, Ubuntu developers decided to leave the old version of Nautilus 3.4, which was not affected by the changes. As for the further development of the situation, it is possible that Ubuntu developers will join the development of Marlin or Nemo forks, as well as the option of creating your own fork.

Server edition enhancements:


  • Apache Tomcat 7 by default;

Livecd


The developers decided to abandon the LiveCD-images and focus on the support of a single image of 800 MB. The classic separation between CD and DVD images and alternative images has sunk into oblivion. Server versions of the distribution, this change is not affected.
