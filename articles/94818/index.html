Office MFP - not just a free printer

In many companies, instead of small personal printers are the so-called multifunction devices:

In such a box weighing several tens of kilograms, a fast printer and a scanner are combined, often there is an automatic feeder on the scanner.

Usually, employees use the following features of tricky devices:
  • Printing waste paper before meetings
  • Copying documents for visa / international passport / loan
  • Copying materials before class in a foreign language
  • Printing essays / coursework / diplomas
  • Other similar ways to double entropy and / or GDP


Nothing else comes to mind? Then go under the cat!

As British scientists recently found out, the MFP can still scan and send a scan by mail instead of printing - it has a network interface (the one through which documents come to print) and software for communicating with the mail server.
True, the use is dubious. You receive a picture of the page, and all you can do is put it in the folder for the junk archive and forget it, because you can only find the document by its content using telepathy. Now, if instead of the image would get a document with the text ...

Suddenly (!) They come to the rescue ‚Üí
ABBYY Recognition Server can retrieve documents from a mailbox (Exchange or POP3) and send recognition results to a mailbox (Exchange or SMTP). The plan is this: a special mailbox is created on the company's mail server (for example, ocr@company.com). Then at any time of the day ...
  1. The employee chooses to send the result by mail in the settings of the MFP
  2. Indicates your address and that special address ocr@company.com
  3. The MFP scans the document and sends the result to the user and in the box
  4. Recognition Server retrieves a document from that mailbox.
  5. Recognition Server sends the recognized document to the user (‚Äúreply all‚Äù mode)

Everything. No need to put anything on the user's machine. You do not need to configure and update anything on the user's machine.
