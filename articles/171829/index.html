Flat UI - free set of flat web interface elements [updated]

The company Designmodo, specializing in the development of web interfaces, has published on Github a rather impressive set of today's flat interface elements - menus, buttons, sliders, switches, checkboxes, icons, and more. The flat design of interfaces is actively promoted by Microsoft, and it is becoming more and more common on the web.

One can argue about the advantages and disadvantages of such a design, but it cannot be ignored - fashion is fashion, so this set surely has a place in the collection of most web designers. UPD : Currently, access to the Flat UI repository is closed at the request of LayerVault , which believes that Designmodo has violated its copyright. Designmodo denies any borrowing and claims that the Flat UI is a completely original design.

Active discussion of closing the repository goes to Gitkhab and Hacker News . So far, the only way to get to Flat-UI is to use the Github search to find one of the many forks of the project.

UPD2: On March 20, access to the Flat UI repository was restored.
