Firefox blocked the official Skype Toolbar

The Skype Toolbar toolbar for the Firefox browser recognizes phone numbers on a web page and draws clickable buttons for calls from desktop applications to them.

Skype Toolbar is installed by default when you install Skype, and, in some cases, when you update Skype. Unfortunately, there are a dozen bugs found in this toolbar (at least four have not been closed yet). After several months of discussion, the patience of the Mozilla developers was over and now Skype Toolbar has been added to the block list , which contains various malicious extensions for the Firefox browser.



Skype Toolbar will be automatically blocked in all browsers. In this case, the ‚Äúsoft lock‚Äù mode is selected, so that users will receive a notification and the opportunity to reactivate it.

According to the explanation in the Mozilla Offblog, this statistics program is one of the most common causes of Mozilla Firefox 3.6.13 crashes (for example, last week it led to 33,000 crashes). In addition, depending on the version of Skype and the method of determining and re-rendering phone numbers, manipulations with DOM can be performed 300 times slower, which significantly slows down rendering of almost all web pages.

Thus, for all versions of Firefox, all versions of the Skype Toolbar have been added to the block list, including the current current version and the beta version.
