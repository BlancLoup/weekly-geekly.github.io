Launching the good old games on the Nokia 5800

Good day, habrasoobschestvo!

I hope this text will please the owners of Nokia 5800 and fans of good old games such as Full Throttle, Sam & Max, The Curse of Monkey Island, Broken Sword 1 ... you can see the full list here .



1. Download and install one great program, this is SCUMM VM for touchscreen (5800), a kind of emulator. Download here .
2. Downloading the game.
4. Write the game files to any folder on the memory card. (for example E: \ Games \ * game folder *)
5. Run the installed emulator, click Add game.



6. Next, select the path to the game files and click Choose . If everything is done correctly, the emulator will show the dialog box for adding a new game.



7. In the Audio section, set Speech and Subtitles (if you want both voice acting and captions). Just choose the scroll speed of the subs.



8. This is the most pleasant item in the menu - launch the game with the Start button and enjoy the good old classics that have been proven by time:



PS Important: before using, you need to turn off the automatic screen rotation in the settings of the 5800 - this is necessary (otherwise it will turn into portrait mode and vice versa when you look at the landscape)!
