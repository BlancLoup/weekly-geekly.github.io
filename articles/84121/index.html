PHP project as a local application

Many people know the Denwer project - the rapid deployment of a bunch of Apache, PHP and MySQL.
However, this kit has certain disadvantages when applied to certain tasks. In particular, there is some difficulty in launching and using the project for a regular user, for aunts and uncles, who use a computer only as much as many obvious things for them are a dark forest. The next drawback is not the performance of this kit from the CD.

Recently I found this modification - Wapache is a program that allows you to create desktop applications using web development technologies. It combines a modified version of Apache 2 with Explorer‚Äôs built-in Internet browser. Wapache also allows you to manage various browser settings, such as window size and the presence of Javascript.
