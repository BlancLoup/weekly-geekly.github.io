Using Latex in Open Office

If you very often have to use formulas when typing a document, then this post is for you.

Many use MS Word and Equation for this purpose - this is not our way.

All over the world, TeX is used for typing formulas (LaTeX, MikTeX, etc.). But for typing in TeX completely, certain skills are required. However, for OpenOffice there is one remarkable extension - OOoLatex. This is a set of macros to support LaTeX in OO. The extension allows you to use all the features LaTeX, without leaving a convenient editor environment. All formulas that you type will be ‚Äúon the fly‚Äù converted into pictures (and you can always go back and edit typed earlier).

Before installation, you need to check whether you have installed:
* OpenOffice
* LaTeX
* Ghostscript
* MSys (for Windows users)

To install the extension, you need to download it from the site and install it in OpenOffice. (Service -> Extension Management ... -> My Extensions -> Add). The OoLatex panel appears.

you must specify the shortcut keys and LaTex and Ghostscript paths installed on your system. (In my case and in another case, this is / usr / bin / - as practice has shown, this is enough). When typing formulas, you can use more subtle settings, such as font size, resolution for images, colors, etc., you can even correct the preamble (well, this is for sophisticated users).
