Phatch - batch image processing

The Phatch program is designed for batch image processing. It can be useful for those who process images in large groups according to the pattern.



The application is able to rotate, rename, change the perspective of the image, impose shadows and watermarks, and much more.

You can create a list of actions, and then apply it to all files that need to be processed.
