Opera browser turns into a network operating system

The Norwegian company Opera Software recently released the ninth version of the browser, but has already begun to prepare a new release. The main role in it will be played by widgets.

Released browser Opera 10 has not yet announced. Obviously, this will happen over the next year. Apparently, the Norwegian company has high hopes for the tenth version of the browser. Representatives of the company say that their development will compete with IE7 .

The most interesting thing is how Opera 10 should compete with IE7. According to the developers, the main advantage of the program will be widgets - additional software modules, support for which appeared in the ninth version. These small programs work relatively independently of the browser, have a separate interface. So far, the functionality of widgets is very poorly evaluated by users, but in the future the situation should change.

Opera Software is now focusing on widget development. It is planned to release special tools for developers to make it easier for them to create their own software modules based on Opera. In the future, the Opera engine can become a universal platform for web applications with the support of all recognized standards. Opera widgets will include office programs, games, and more, so in the future this platform may even become a replacement for the standard operating system.

In addition, Opera 10 will become a more universal browser and will work on more platforms: on personal computers, on mobile phones and on game consoles, including Nintendo DS. All this is done specifically to attack Internet Explorer positions from various sides. The fact is that Microsoft, in principle, cannot port its browser to such a number of different platforms due to the fact that they are its direct competitors. For example, Nintendo directly competes with the Microsoft Xbox.

The ninth version of the Opera browser was released on the market on June 20, 2006. On the first day, it was downloaded by more than 700 thousand people. To date, the number of downloads, including the mobile version of Opera Mini, has reached 25 million.
