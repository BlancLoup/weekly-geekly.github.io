NAS servers Synology XS series received the status of officially certified VMware vSphere 5 VAAI



Synology Inc. announced that its XS NAS servers received full support for VMware vSphere 5 vStorage APIs for Array Integration (VAAI) technology. One of the first to realize the full support of VAAI technology, Synology offers users advanced storage solutions with high performance and optimal use of disk space.

With full VAAI support, Synology XS Series network drives provide advanced storage organization and high-performance virtual deployment. Full Copy feature enhances data backup speeds by unloading the hypervisor and performing virtual disk duplication operations directly on Synology XS Series network drives. The implementation of support for the low-level feature of Zero Blocks allowed Synology XS Series Network Drives to simultaneously reset a large number of storage blocks, which significantly increases the speed of initialization of virtual disks. The dynamic capacity allocation function for monitoring the space used by the allocated LUNs has been optimized. Moreover, the Atomic Test & Set (ATS) function allows block blocking, rather than LUN, during updates and improves system performance while simultaneously accessing LUNs from several hypervisors.

Full-featured virtualization support allows users of Synology products to enjoy benefits such as simplicity, speed and efficiency. For more information, go to this link http://www.synology.com/dsm/virtualization_support.php?lang=rus

Synology XS series NAS servers are included in the list of VMware-compatible hardware, read more: www.vmware.com/resources/compatibility/search.php?deviceCategory=san
