Came out fring for Windows Mobile - Skype, Google Talk, MSN and SIP on PDAs and smartphones

Today, the company fringland Ltd. announced the expansion of the free mVoIP (mobile VoIP) client fring , adding support for devices running the Windows Mobile operating system. Now, the fring program can be used on almost any PDA or communicator running Windows Mobile.

fring for Windows Mobile is a full-featured analog of the already popular fring client for Symbian OS, and allows you to make free calls on fring, Skype, Google Talk, MSN networks and many SIP providers using 3G, GPRS or Wi-Fi connections. The WM client fring also supports IM (instant text messaging) and Presence functions - showing the user's online status in real time.

Let me remind you that fring gained popularity, becoming the first and almost the only opportunity to use Skype and Google Talk on mobile phones running Symbian OS. Later, the program added the ability to work with MSN service and SIP operators. In the case of Windows Mobile, this client will be a little more difficult to win a place under the sun. Skype users have long been using the "native" Skype for Pocket PC client, which has been actively developing lately. Also, manufacturers of SIP clients for PDAs and smartphones are not far behind. Well, the followers of the Jabber protocol have long learned to use Google Talk in their favorite programs. Thus, the only, but indisputable advantage of fring may be its versatility. After all, fring is a kind of combine that allows you to choose which network to use for communication at any time.
As for the choice, use specialized software, or the program "all in one", it will show the time and personal preferences of users.
