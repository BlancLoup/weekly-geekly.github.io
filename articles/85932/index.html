Ogre3D 1.7 Cthugha release!


At the end, the long-awaited release of the open source render-engine Ogre3D 1.7 Chtunga.
The main changes in this thread can be found on this page . Download source from svn .
Download precompiled SDK for different platforms here.

The new version has many great additions:
  • Fully redesigned system run demo applications. Almost all the examples rewrote under the new features.
  • The engine officially transferred to the MIT license.
  • The developers refused to use zip-archives and switched to 7zip archives. (immediately used tar.gz)
  • One of the main features is the new Terrain plugin (see the screenshots and history of changes on the forum ).
  • The new system of runtime shaders RTSS , where code generation is performed on the fly.
  • Improved compatibility with 64-bit systems.
  • A project for Visual Studio (sln) has been added for compilation under Windows.

I want to remind you that several high-level projects are written on this engine: Torchlight, MotorM4X, Zombie Driver, Venetica
