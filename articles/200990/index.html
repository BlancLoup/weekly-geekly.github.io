YouTube Center

YouTube Center is the best browser extension running on YouTube that I have seen. I can not share.
Actually, this extension solves, if not all, most of the problems that we encounter on YouTube and which have been recalled more than once in Habr√©. The expansion itself was also talked about on this site, but how good and convenient it is, which deserves a separate post.

YouTube Center features

Addition allows:
  • place the video in the center of the page ( 1 )
  • specify player size ( 2 )
  • specify video quality ( 3 )
  • disable autoplay and video buffering, including in Linux in HTML5 ( 4 )
  • disable Dynamic Adaptive Streaming over HTTP (Dash), which does not allow to move freely along the video, because buffering starts anew each time ( 5 )
  • disable ads, including ads inside the video ( 6 )
(numbers in brackets indicate links to comments on Habr√©, which talk about the desirability of such features on YouTube)

There are other small buns, like the possibility:
  • download videos (with the choice of quality, format and video title)
  • delete video offer at the end of viewing
  • specify the type of player (Flash or HTML5)
  • specify the resolution in which the clips are loaded
  • add auto light, choose its color and transparency
‚Ä¶ and much more

However, all the functions are perfectly configured, and your version may be completely different from this one. The main thing is a choice, and how to use it - you decide.

The YouTube Center extension is cross-platform. It works under Windows and Linux (verified), download statistics indicate that it is installed on both Mac OS and Android.
There are versions for Firefox, Chrome, Opera (including Opera <15>), Safari, Internet Explorer, Maxthon

Extension home page: userscripts.org/scripts/show/114002

After installing the extension on the YouTube page in the upper right corner appears the settings icon. The settings panel conveniently opens on top of the existing YouTube page and allows you to customize a lot of different parameters.

Installing YouTube Center

A few words about installing YouTube Center.

For Firefox and Opera, everything is quite simple, YouTube Center is on the add-ons page, where it is easy to find and where it is convenient to install.

Firefox: addons.mozilla.org/ru/firefox/addon/youtube-center
Opera: addons.opera.com/en/extensions/details/youtube-center/?display=en

Google chrome

In Chrome's online store, this extension has not been approved.

But even if you install them all, the result will not be as user-friendly as a single well-thought-out extension.

There are YouTube Center and disadvantages.
It is not always possible to download video as 1080, although such a resolution is specified for it, the music is downloaded in a format that involves converting it.

Video viewing is not implemented at all in countries where it is prohibited for viewing.
To view this video, you can use the YouTube Unblock site or the ProxTube extension for Google Chrome, Firefox proxtube.com/install.php , which automatically processes and allows you to view all inaccessible video.

But generally speaking, YouTube Center makes using the site more convenient and comfortable than it was intended by developers.
It would be just great for all popular sites to create such extensions. It would be even better if the functionality provided by the extension was implemented by the site developers themselves, but this is already from the realm of fantasy.
