Joomla customers mini review

Hello!
It so happened that I ran into a person who found it difficult to explain how the administrative panel (hereafter, for simplicity, the admin panel) was arranged by my beloved Joomla. The fact is that the admin panel is overloaded with unnecessary information for the manager, and even when you limit the user level there are many questions left, and this person is not used to working with web interfaces due to age.
At that moment I was thinking about writing a certain client for Joomla, but the search pointed me to a couple of projects about which I want to tell. Well, let's get down to business, gentlemen.

Blogg-x
Official website: iobinary.com
Foreign project, made using Adobe Air. Cross-platform. Stylish design, a set of functions for the manager and no more, would be great for my purposes, if the Russian language were present.

When you start the application, we are greeted by this window, which contains a list of sites for quick access.


Going to the settings (the Options button) we get to the site manager, from where that list is typed in the main menu.


Then we simply add the site, enter the settings, select the version of Joomla (yes, yes, two branches 1.0 and 1.5 are supported) and click ok


We check the settings of the application itself, choose the language (there is no Russian yet, sent the translation to the author) and color to taste.


Go back to the main menu and go to the site we added. Three options appear in front of us: Add an article, article manager and media manager.


Well, let's go in order, the interface for adding articles speaks for itself, it uses FCKeditor, nothing remarkable, but nothing superfluous. It's nice to work.


The article manager is a regular list that allows you to delete, edit and copy articles, from which the status of publication of materials is controlled.


The media manager is convenient, though also ascetic. Allows you to upload images in batches, which is very convenient. There is a preview of images, audio and video.

(the picture on the preview comes first on hand)

In general, a convenient and functional application for the manager.
Requires for its work. NET Framework 3.5

Joomlaposter
Official site: http://www.joomposter.ru/
Our Russian development, therefore the Russian interface, which made me very happy. Uses .NET Framework 2.5.

It supports not only Joomla 1.0 and Joomla 1.5, but also Joostina. There are versions for 32 bit and for 64 bit Windows.

The program meets us with a site manager similar to the previous one.

The settings of the language and the proxy also go there, which is not supported in Blogg-X

The window for editing site settings is richer here

The component key is set in the Joomla admin panel and compared with the program, something like a password for access.

But the list of materials upsets the lack of a filter. The thing is useful.


But in Joomlaposter there is an editor for sections and categories, which Blogg-X cannot boast of.

But it is made somewhat inconvenient, although this is only my opinion.

Well, the very heart - d actor.

Here is my editor, as I understand it. The interface in the tradition of the latest fashion Microsoft'a. Conveniently enough and able to clean from extra HTML trash. What can not but rejoice. Vords love to add a lot of excess to the texts written in them.
Adding pictures to an article occurs along with uploading to the site, and does not distract the user from creative work. Clicked to add a picture, chose, and the head does not hurt.
Another huge plus of the editor is that you can save articles as drafts on your computer or write them offline and then upload to the site with a pack by pressing just a couple of buttons.

I also met a client program for Android on the Internet, but due to the lack of it I did not study.

Instead of conclusion
Personally, I stopped at Joomlaposter, after all, the Russian language, the convenience of drafts and so on. However, I will follow the development of Blogg-X and help as much as I can.

Thanks for attention.
