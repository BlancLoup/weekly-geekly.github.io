Meet Evernote for iOS4

In a few days filled with tedious waiting, the new iOS4 operating system, and with it the new shiny iPhone, will be released to the masses. And just the other day, Apple approved the version of Evernote (3.3.5), compatible with iOS4. In addition to supporting new firmware, this update also improves iPad image processing and syncing for all devices. Below we briefly describe the significant innovations.

What did I stop at?


iOS4 supports fast application switching. Evernote supports this feature, so you can leave the client during the creation and viewing and search for notes, and in the process of any other actions. Then you will return to the same place where you left work with the application.

Background mode


The new firmware allows applications to continue working on certain tasks even after the user has left the program. In the case of Evernote, the following operations can be performed in the background:
  • Download note headers (i.e. service data required to display and search through notes).
  • Synchronization of new notes: the application will upload to the server all new or edited notes that are in the queue.

In order for Evernote not to shorten the battery life and not to download the Internet channel without your knowledge, the application in the background does not fully synchronize offline notebooks with downloading all the notes on the phone (a function that is available to our premium users).

Keep talking




Audio note recording will now continue after exiting the application. This is useful if you started the dictation, and it became necessary to clarify some data in another application. To end the recording, simply click on the red Evernote bar to quickly return to the program.

About this release


Please note that iOS4 features are only available on devices that support this OS. Do not worry if your device does not support the new firmware, this update also contains many other inconspicuous fixes and improvements that will make working with Evernote better for everyone.
