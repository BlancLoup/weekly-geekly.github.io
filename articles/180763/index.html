FlexPod: Cloupia Unified Infrastructure Controller (CUIC) - New Acquisition of Cisco

Somehow imperceptibly in the vastness of IT, news of the purchase by Cisco of the Cloupia company for $ 125 million passed. Cisco saw the addition of its portfolio of solutions for data centers providing cloud services with software from Cloupia. I also recall that Cloupia is the first orchestrator validated by both Cisco and NetApp to manage FlexPod architecture. What is FlexPod , can be found in the relevant article on Habr√©. For greater clarity, there will be many pictures.

CUIC


CUIC is a data center infrastructure orchestrator, capable of combining both physical equipment in a data center and virtualized environments. With it, you can orchestrate from the "one point" of the entrance. Thus, this tool is capable of providing services like: IaaS and PaaS .
Examples of such equipment:
  • Network routers and switches: Cisco Nexus 7000, 5000, 1000v.
  • Computing Systems: Cisco UCS Fabric Interconnect.
  • Various storage systems, including NetApp.
  • Virtualization: VMware, Microsoft, Citrix.


CUIC features


The possibility of "management from one place" is not all that this orchestrator can do.
Cloupia allows you to integrate into a sequence of any atomic operations, in terms of Cloupia, called "workflow". By transferring such a workflow to the user, he will be able to query the data center resources available in the pool and receive them at his disposal in a few mouse clicks in minutes. In the basic configuration, a large number of already prepared typical workflows are supplied.
You can control not only virtual environments, but also bare metal, such as switches, storage systems , servers, etc.
Another useful feature is also included monitoring of consumed resources. Here you can get detailed reports on various types of objects, such as: users using cloud provider resources (in the broad sense of the word), groups of these users, entire organizations (including user groups), storage space consumed, CPU and memory usage, network counting traffic and more.
Graph display can be flexibly configured and available in the sections on: network part, computing resources, data storage system, virtualized environments, financial expenses (up to power consumption), etc.
Well, as it became clear from the previous paragraph - in Cloupia CUIC there is a self-service portal that allows you to get statistics on the resources consumed by the user with the display on all possible sections.
In the same place, on a self-service portal, the load generated by all entities, both physical and virtual, is displayed.

In one of the tabs of the portal is available a list of possible "to order" workflow.
The process of the application can be seen as the user who ordered the service, and from the admin panel, the administrator of the orchestrator.

FlexPod


Since CUIC is the very first validated orchestrator for the FlexPod, it provides convenient tools for managing this architecture.
