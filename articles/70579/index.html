Are you well prepared for using Windows 7? Several hot keys

Just want to apologize, the article is not very technical. If you think that you already know all the hotkeys of Windows 7 that simplify working with it, then just skip this topic.

Accidentally I came across a sign with ‚Äúhot‚Äù keys associated with the new toolbar of Windows 7 - I recognized many new and useful ones for myself and I hasten to share them with you, having first collected the maximum useful combinations.


1. Opening a new instance of the application. When you have attached (not necessarily) some kind of icon to the taskbar, for example, Word, and you want to launch a new instance of the program quickly - I used to do this: right-click on the icon and there I already chose a new instance of the program. It turns out that everything can be made easier by holding down the Shift key: we hold it down and click on the icon with the mouse - we launch a new instance of the application. You can also click the mouse wheel (middle mouse button) on the icon to open a new instance of the program (by flashface )

2. Opening the program on behalf of the administrator. As in the first case, I did this before: with the right mouse on the icon, then with the right mouse, on the program's shortcut and there I chose Run As Administrator. Here, the Ctrl + Shift + mouse click on the icon comes to the rescue - we see a favorite window that we don‚Äôt mind running the program as an administrator. If the program needs to be constantly run as an administrator, this can be configured on the Compatibility tab of the program shortcut - there is the Run option there. I did this with Visual Studio, and then I was tortured that, sometimes, I didn‚Äôt have enough administrator rights to work with Internet Information Services.

3. See the old forgotten main window menu . It is difficult to explain in words what it is, I will show it with a picture:



This is the standard menu that we saw in previous versions of Windows when you right-click on the icon of a running application (in the case of a non-launched one, there will be a shortcut file menu). In order to run it, you must again hold down the Shift key and right-click . In the case of a group of running windows (instances), there will be a context menu for managing windows of this group.

4. Activate the next window from the group. In the case when we have several instances of the program running and we see the group icon, then when the Ctrl key is pressed and the left mouse button is clicked, we will activate the next window from the group in turn.

5. Run the program from the taskbar by the icon number. Clamp Windows logo key and click the number of the icon on the taskbar - a copy of the program is launched or we activate the running window (counting from 1 :) ) to launch exactly a new copy of the program - hold down the Shift key . To activate the last active window, hold down Ctrl to open the Jump List - hold down Alt .

6. Run over the taskbar icons. Another way to not use the mouse. Clamp Windows logo key and press the key T - turn the focus one by one in sequence to the taskbar icons.
