The Firefox 8 system will block third-party add-ons



Despite the fact that now the actual version of the Firefox browser is version 5.0, you can read plans for the future on the blog of the development team regarding versions of Firefox 7.0 and even 8.0. The eighth version should appear already on November 8, so it is not so long to wait. So, in version 8.0 there will be an automatic blocking feature for add-ons that are installed by third-party programs / sites. When opening the browser, the built-in security system will check it for the presence of third-party add-ons and warn the user about it.

If something suspicious is installed, the browser will ask the user to confirm the installation - you will need to tick. Those. in case of harm caused by such an addition, the user will only have to blame himself, and no one else.



Interestingly, the security system will check not only the new additions, but also analyze all that has been installed earlier. So, when updating the browser to the eighth version, all add-ons of third-party companies will be disabled. To enable them, you will need to view the list of detected add-ons, and tick each corresponding field. Otherwise, such additions will not be activated.



Well, for now we are waiting for the release of Mozilla Firefox 6.0, this version should appear tomorrow, and Mozilla Firefox 7.0, which should be released on September 27 As mentioned above, number 8 will be released on November 8.
