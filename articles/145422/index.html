New versions of Windows Azure tools released, Python support added

Today, along with the update of Windows Azure ( details on Habr√© ) and the big announcement of new services , the new Windows Azure SDK for .NET 1.7 development tools for Visual Studio 2010 and Visual Studio 2012 were presented.





Details of the announcement of the new services you can get in this document Fact Sheet (.docx).

In addition, a number of other tools are available for steel developers, including:

  • Special free edition of Visual Studio Express 2012 RC for Web with support for Windows Azure
  • Updated WebMatrix 2 RC with support for Windows Azure, projects based on Node.js and many other innovations
  • Updated Windows Azure SDK for PHP
  • Updated Windows Azure SDK for Node.js
  • A brand new set of Windows Azure SDK tools for the Python language - with the release of this set, Windows Azure officially welcomes Python as a platform for cloud services. Tools come with open source

Installing new tools is recommended using the Web Platform Installer, the central repository of all Microsoft development tools. Download all the tools at the following links:

  • Windows Azure SDK for .NET - June 2012 ( download )
  • Installing the toolkit for Visual Studio 2010 via the Web Platform Installer ( download )
  • Installing the toolkit for Visual Studio 2012 via the Web Platform Installer ( download )
  • Visual Studio Express 2012 RC for Web with Windows Azure support (in WebPI)
  • WebMatrix RC (in WebPI)
  • Windows Azure SDK for PHP (in WebPI)
  • Windows Azure SDK for Node.js (in WebPI)
  • Windows Azure SDK for Python (in WebPI)

Additionally:

  • SQL Azure Data Sync Agent Preview ( download )
  • Web Platform Installer ( x86 and x64 )

You can find out more information about all the updates, new tools and services of Windows Azure today at the special online event Meet Windows Azure , where Scott Guthrie will give a plenary report. Do not miss, there will be a lot of interesting things.
