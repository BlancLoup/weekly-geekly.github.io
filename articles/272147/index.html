Adobe says goodbye to Flash

Adobe urged web developers to stop using Flash and move to modern web standards like HTML5. In its post on the official blog, the company pointed out the importance of developing HTML5 and moving away from the Flash platform, which could be successfully replaced by a new standard. The Flash title will also be replaced in the flagship product Flash Professional CC with Animate CC. It in turn will become the main product for creating HTML5 content.



It should be noted that this is not about stopping support for Flash Player for popular web browsers, its company will continue to support. This is about Flash as a technology and about encouraging developers to switch to the modern HTML5 platform, which Adobe will now support. The Flash Player itself is known for a large number of its vulnerabilities, exploits for which are constantly used by attackers to install malware.

Adobe indicates that Flash is still running a large number of games and various web applications, so it will interact with key IT players to support the proper level of user security. In particular, it says about Facebook, which hosts a sufficient amount of Flash content. The company is not going to abandon these users to their fate. Adobe will no longer add new features to Flash Player, but will only focus on maintaining its security.
