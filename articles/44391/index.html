My experience of setting up a wl500gvp2 router as a print server

I am the happy owner of this magical device . In addition to its main functions, this router can serve as a print server and file server thanks to the presence of two USB 2.0 ports.

My hands didn't reach the file server configuration, but the print server is my old dream.

My firmware is from Oleg , version 1.9.2.7-10. The printer is a Xerox 3120 (similar to the Samsung 1510).

A small digression: in the Russian Internet there is a wonderful forum wl500g.info , led by the same Oleg, from whom I have firmware. First of all, after buying any router from asus, you should pay a visit there. A lot of interesting and useful.

Let's return to our sheep.
We connect the printer to any USB port of the router you like and look in the web interface section
Status & Log


The printer is determined, it already pleases. We try to configure the printer without a driver .

If it worked, then you can not continue to read, and if not, you can try the following:
go through telnet to the router. In my case it is
telnet 192.168.1.1

We give a command
p910nd -f -b /dev/usb/lp0

The p910nd program p910nd designed just for our case: everything that comes to tcp-port 9100, it sends to the specified device. The router is configured so that it considers that the printer is a device
/dev/usb/lp0

We will not disappoint him.

After that we put the printer driver on the machine with which we are going to print. Those. We do everything that is written in this manual , except for item 11. We select the correct printer driver.

After that, everything worked for me, and I was extremely happy about it!
