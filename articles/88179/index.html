Writing posts in Google Buzz from Jabber

This Python script is a simple jabber bot that sends messages sent to it to Google Buzz. For those who are interested in this opportunity, the details under the cut.

Personally, this opportunity was terribly lacking for me. To work from non-standard libraries, the script needs only xmpppy, if I'm not mistaken. Testing was conducted under Ubuntu 9.10, Python 2.6.4.
It is better to run the script in the console, the output is carried out by Ctrl + C.

Script itself: paste.ubuntu.com/397824

PS Practical criticism of the quality of the code is very welcome, because Python I study only 1.5 weeks.
