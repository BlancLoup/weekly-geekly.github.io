Style for Zend Studio

Already quite a long time ago I ran across the open spaces of the network to an interesting and convenient, I think, style for ZendStudio. It just so happened that now I have entered this style in the set, which must be installed. Now I wanted to share this style with the habra people.

To install the archive with the style must be unpacked in
C:\Documents and Settings\user\ZDE\config_x.x\schemes
Then in the settings of Zend Studio specify the style in
Preferences -> Colors & Fonts -> Scheme Name
