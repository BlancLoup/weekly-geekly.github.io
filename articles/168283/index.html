Evasi0n: jailbreak for all iOS 6.x devices released



The hacker team Evad3rs has just released a utility for untethered iOS 6.x firmware jailbreak for all iOS devices except Apple TV 3G. This is the first jailbreak available to users with devices running A6 (iPhone 5) and A6X (iPad 4) processors. Also for the first time, jailbreak is available for the iPad Mini, which originally comes with firmware version 6.0. Previously, the jailbreak was available for iPad 3 users (A5X processor) running iOS 5.

Evasi0n for OS X, Windows and Linux is available for download on Evasi0n

Evasi0n 1.1 fixes the weather bug. But only if you do not have AppSync installed. Those who have already jailbroken with Evasi0n 1.0 need only update the evasi0n 6.0-6.1 Untether and UIKit Tools packages in Cydia.

Before starting the jailbreak procedure, it is recommended to backup data in iTunes or iCloud. If you make a backup in iCloud, go to Settings - iCloud - Storage and copies - Storage - your device and select the data that you want to back up. It is better to turn off the unnecessary so that it does not take up too much space in the cloud.

Evad3rs also recommend making a recovery for iOS 6.1 firmware via iTunes. If you have switched to iOS 6.1 by updating the firmware via the Software Update directly on the device, it is strongly recommended that you update the firmware again, but through iTunes. This will help to avoid problems in the process of jailbreaking.

Hackers on their twitter noted that the number of active visitors to Evasi0n.com at the time of release reached 270,000 people. It seems that the jailbreak iOS 6.x waited even more than the release of the firmware itself.
