How glow.mozilla.org works



The Mozilla blog explains how SQLStream processes Firefox 4.0 download statistics from each server and issues a map of downloads on glow.mozilla.org in almost real time. The source code for the Python backend is also posted there.

A week ago, Internet Explorer 9 followers boasted that 2.35 million users downloaded their browser in the first day (see the wow message in the official Windows Blog ).

But the output of Firefox 4 showed which browser is actually the most popular in the world. Already ten hours after the official release (that is, today at 4 am Moscow time), the number of downloads of Firefox 4 exceeded IE9, and by now has already reached 5.7 million (see the counter ).
