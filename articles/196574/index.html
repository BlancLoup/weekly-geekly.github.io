We study cloud technologies with free courses of the Microsoft MVA Virtual Academy



In this review, we‚Äôll talk about the free courses of the Microsoft MVA Virtual Academy, which will help you explore cloud technologies from both infrastructure and software development.

NEW. Introduction to the development of modern applications with Windows Azure for beginners
This course was created to provide novice cloud application developers with a basic knowledge of the capabilities of the Windows Azure platform, the latest and updated services.

NEW. Security and authentication in modern applications using Window Azure
This course was created to provide developers with an overview of the possibilities of organizing the authentication and authorization of users using the Windows Azure Active Directory cloud platform.

NEW. Organizing a cloud environment for developing and testing applications
This course was created to provide developers with an overview of the possibilities for organizing the environment for developing and testing applications in the cloud.

Developing Windows 8 applications using Windows Azure Mobile Services
This course is dedicated to providing IT professionals and developers with basic knowledge of the concepts of the Windows Azure Mobile Services cloud service and the basics of developing Windows 8 applications using this service. The Windows Azure Mobile Services service provides the ability to transfer such types of loads as authentication, data storage and push notifications to the cloud backend.

Developer Intro: Storage in Windows Azure
This course provides a basic knowledge of cloud service concepts for the storage platform of Windows Azure. The storage provides storage services that provide security, scalability and high availability for long-term use. The storage service supports almost all the necessary storage functions, including storing structured and unstructured data, a NoSQL database, and a queue. This course covers how to work with all types of storage (tables, BLOBs, queues, and disks), how to work with Windows Azure storage services in a local computing emulator, and how to publish a solution on Windows Azure.

Build cloud web applications on Windows Azure Web Sites
This course was created to provide developers with basic knowledge of the concepts of the Windows Azure Web Sites cloud service and the basics of developing web applications using this service. The Windows Azure Web Sites service allows you to quickly create an application based on a template, load a ready-made application, and scale expanded web sites as required.

Windows Azure Security Overview
Learn the basics of Windows Azure security at each level of infrastructure. This course covers the security mechanisms for Windows Azure at the physical, network, host, application, and data levels. In addition, you will gain basic knowledge of authentication and authorization mechanisms in Windows Azure.

Discover Windows Azure
Cloud computing is already a reality. Running applications on servers available on the Internet can provide many benefits. Wherever they work, applications are based on a specific platform. For applications running on a local network, this platform typically includes an operating system, data storage, and other components. For applications running in the cloud, a platform is also needed. This is the main purpose of Microsoft Windows Azure. This course is dedicated to providing IT professionals with the basic knowledge necessary to successfully work with Windows Azure.

HPC clusters on demand in Windows Azure
This course is dedicated to providing IT professionals with basic knowledge of the cloud services concepts of the Windows Azure platform, which can be used to deploy supercomputer clusters that can perform various types of tasks (MPI, SOA, Excel). This course covers the basics of working with Windows Azure and examples of deploying HPC clusters and their further use using the Windows Azure Management Portal and Visual Studio 2012.

Windows Azure for system administrators
The Windows Azure course for system administrators is devoted to the updated functionality of the Windows Azure platform, namely the appearance of the functionality covering the infrastructure solutions, IaaS. In the context of the course, innovations that have appeared in general in the platform are considered, as well as special cases, examples of which are Active Directory directory services in Windows Azure and virtual machines. In conclusion, we consider the possibilities of integrating the Windows Azure public cloud and the platform for creating a private cloud, a combination of Windows Server 2012 and System Center 2012 SP1 for obtaining a hybrid cloud model. The course is designed for system administrators, students and IT enthusiasts.

System Center App Controller: Overview and Integration with Windows Azure
This course talks about modern cloud models in the field of IT infrastructure. Particular attention is paid to the hybrid cloud model, as it is considered the most likely and logical model for the development of IT infrastructure. We will also consider the main components for building a cloud model based on Microsoft solutions - System Center and Windows Azure. The combination of these components and their tight mutual integration allows you to create a hybrid cloud.
