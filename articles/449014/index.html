Promoting Ethereum DAPP

Today there is enough information on the Internet how to create DAPP applications. So you created a cool DAPP and now the question is even more difficult, how to tell potential users about it?



I want to share my own experience in promoting DAPP applications to directories. The method is good in that it is aimed at users who already know what DAPP is and how to use them, search engines will also learn about DAPP.

In the list of Ethereum DAPP catalogs, placement in all catalogs is free.

Website: dappradar.com
Additional blockchains: EOS, Tron
User opinion: no
Languages: English, Chinese, Italian
Note: The largest DAPP directory, often other DAPP directories take information from it.

Website: stateofthedapps.com
Additional blockchains: EOS, GoChain, POA, Steem, xDai,
User opinion: ratings
Languages: English, German, Chinese, Korean
Note: There is an adnroid app. Most transitions to DAPP from this directory.

Website: dapp.com
Additional blockchains: EOS, Tron, Steem, Qtum, Nebulas, Neo, ETC, TomoChain, IOST, Blockstack
User opinion: ratings
Languages: English, Chinese, Vietnamese, Japanese, Russian, Turkish
Note: Conducts an analysis of the number of subscribers in the social. networks
