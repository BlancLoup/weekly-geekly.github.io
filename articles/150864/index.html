iPhone Style Checkbox for Twitter Bootstrap

Hello, dear habravchane.

It so happened that during the development of my toy web application, I decided to make checkboxes in the iPhone style in the interface. Since the entire client part is developed on Twitter Bootstrap, I decided to look for the implementation of such checkboxes, but I did not find it. Then I decided to fork this project ,
took it as a basis, redrawn the pictures, corrected css and eventually got about this checkbox




Of course, I'm not a professional designer, but I tried to get as close as possible to the Twitter Bootstrap style. As it seems to me, I managed to do this at least four.

If you are interested, the source code and examples of use can be downloaded here . You can also use it without restrictions, modify and generally do anything with it. Moreover, if someone takes the liberty to make a normal fork on GitHub and accompany it, it will be quite good.

PS I would be grateful if someone uploads my archive to a sensible file hosting and deploys a demo somewhere. Thank you in advance.
