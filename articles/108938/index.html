A group of enthusiasts has developed a technology for surfing the web using Kinect



Everything is exactly as the title says. A group of students at the Massachusetts Institute of Technology used their own work to bring together Kinect from Microsoft, Google‚Äôs Chrome browser and their own Java scripts to create sign-surfing technology on the Web. Only gestures, and nothing but gestures - this was the main task set by the developers.

The project of surfing the web through gestures got its own name - DepthJS. Developers joke that their system will help users to feel like Obi-Wan Kenobi and Bruce Lee at the same time - such a variety of gestures can be used to work with the Internet. Gestures help to perform operations such as scrolling, clicking on links, increasing / decreasing the scale of the page when viewing, and many others.

The developers are confident that this is just the beginning of a new era, when thousands of applications using gestures will be created. Indeed, it would be more convenient to use many programs without working with a mouse, and to control them with gestures. This may seem a little awkward at first, like many new products, but then gesture technologies can firmly enter our lives. For example, such technologies can be used to work with Google TV.

You can use not only gestures, but work alternately with the keyboard and mouse we are used to, then control the PC with gestures. Experts suggest that this alternation will help to avoid diseases of the radius and other "professional" diseases of the locomotor apparatus of computer scientists. Naturally, a lot of time may pass before the implementation of all this, but there is almost no doubt that gesture technologies will really "go to the people", largely due to projects like this.
