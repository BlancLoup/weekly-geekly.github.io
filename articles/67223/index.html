New product of the JetBrains team - ‚ÄúCharisma‚Äù in EAP

The JetBrains team (known to many for the R # product) creates a new product with the code name ‚Äú Charisma ‚Äù.

Charisma is a web-based issue tracker (error tracking system). Its main functions that distinguish it from similar systems:
  • Search using the question as an alternative to standard search filters (this function is visible in the picture - a request of the form By Me Fixed - closed tasks by me ).
  • Lightweight user interface based on AJAX - allows you to work with the system faster.
  • Extended keyboard support for faster creation, editing and transition between tasks using the keyboard.
  • Ability to perform command operation on selected requests.
Charisma is developed with the help of JetBrains MPS and is already used to track errors in the JetBrains TeamCity system (there you can try their praised user interface). Charisma is already available in the Early Access Program (EAP) , because everyone can try to download it and install it (on any J2EE server, such as Apache Tomcat 5+ , Mortbay Jetty , JBoss , Caucho Resin ), the last build was July 28 at number 45 And of course, the JetBrains team will be happy to listen to all your comments and suggestions (as it is written on the EAP website). It's a pity that the prices have not been published, but I think those who will participate quite strongly in discussing and searching for errors can be JetBrains and will thank them with discounts or gifts.
