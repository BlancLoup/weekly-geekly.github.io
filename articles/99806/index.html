After installation, the plug-in icon is added to the lower right corner of the browser. The plugin has a counter that automatically counts stickers on each page.
The appearance of the plug-in is slightly different from the plug-in for Chrome * - the plug-in is closed by clicking the X button:



Using the plugin, you can create a new sticker, as well as show / hide all the stickers on the page. The parameters for displaying stickers on pages are set in Settings:



All the basic functions of creating and editing a sticker are preserved:
  • font size
  • background color
  • sticker location
  • its size, etc.
And of course the main thing - the plugin for Firefox has got a screen grabber!

Now not only Chrome users can make screenshots on pages :)



It is possible to rob as a separate fragment on the page, and the entire page.
