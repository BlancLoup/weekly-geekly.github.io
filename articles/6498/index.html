Xbox 360 acquires chat and keyboard

In the near future, it will be possible to launch the Windows Live Messenger IM program on the Xbox 360 gaming console ‚Äî a chat client the same as on personal computers and Windows smartphones. For the chat on TV, Microsoft will release a special keyboard in the summer of this year, which will be plugged into the headphone jack.



Reportedly, the console version of Windows Live Messenger will support simultaneous chat with up to six users.

The subscriber base of Windows Live Messenger is more than 260 million active accounts, and now it can grow even more at the expense of those users who do not have a computer, but there is a game console and a TV.

The addition of an IM-pager to the software package will take place as part of the Spring Update of the Xbox 360 Spring Dashboard, which is scheduled for May 7th. Together with it will be implemented and other improvements, for example, a separate window for the online store Xbox Live Marketplace, the ability to start watching partially downloaded movies and the function to automatically turn off the device after the file has finished downloading.
