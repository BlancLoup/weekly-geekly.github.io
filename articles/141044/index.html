Evernote for Windows Phone 7 Update: Notebook Sharing, Improved Editing and Search, and More

Today, we are pleased to present an Evernote update for Windows Phone 7. The new version of the application (2.2) includes a number of frequently requested features, including support for sharing notebooks and improved editing. We also worked on the convenience and reliability of work.

Sharing notebooks


Evernote for Windows Phone 7 now allows you to open access to your notebooks for individuals as well as for everyone. You can open access to the notebook in two ways:
  • tap and hold the notepad in the list, and then select Access Settings ;
  • click on the notepad, select the ellipsis in the action bar, and then access settings .

Contextual search and note creation


If, when viewing a separate notebook or tag, you make a search query, then the search will be performed only on the notes in this notepad or with the corresponding tag. The same applies to new notes that you create - they will be added to this notebook or labeled accordingly. If you view all the notes, then the search will be carried out across the entire account, and new notes will be placed in the default notebook.

Improved note editing


Editing notes in this update has become more enjoyable. The text editor now shows the entire text and allows you to edit it directly. Images and attached audio files will be displayed above the text box.
