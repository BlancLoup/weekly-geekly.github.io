Python in Europe under threat

The Python Software Foundation, a nonprofit organization, is sounding the alarm : a British company has announced itself that is trying to register the Python trademark for software, services, etc. throughout the European Union.

This company registered the domain python.co.uk 13 years ago. Previously, she just sent traffic from this domain to her sites veber.co.uk and pobox.co.uk, but now the parent company has decided that she can use the name Python in her business.

The Python Software Foundation organization needs testimony that the word Python was used before the python.co.uk domain was registered. The organization appeals to all companies in the EU that use the software in Python, asking them to send a letter that can be presented in court.

The letter should have a brief description of how the company uses Python software, recognizing that the company recognizes the Python trademark only as coming from the Python Software Foundation, and also evidence that another company will use the Python trademark in terms of software and services mislead.

The letter does not have to be voluminous: just a few paragraphs.

Sample testimony in another similar case.

In addition, other evidence is needed for the trial: any documents that were published in the territory of the European Union, in which the term Python referred specifically to the Python programming language. Scanned images, any copies or originals are accepted: they can be books, brochures, conference programs, job advertisements, magazines and other publications, leaflets.

PDF copies are accepted at psf-trademarks@python.org. Books with the word Python and other documents can be sent directly to the address of the organization.

Organization of Python Software Foundation appeals for help to all not indifferent. Let's not hurt our programming language!

The trial will cost tens of thousands of dollars, so financial donations are also welcome: http://www.python.org/psf/donations/ .
