Release Flash Player 10.2

English translation post about the release of flash player 10.2

I am very pleased to announce that Adobe Flash Player 10.2 is available for download ( debug version is available here ).

I already talked about the innovations of this release a few weeks ago . Flash Player 10.2 introduces Stage Video to provide high performance when watching videos. Also, there are very cool native mouse cursors that game developers should like. In addition, added optimization for IE9 and improved text quality for mobile devices. In the new Flash Player, you can also enjoy full-screen mode on one screen, while you continue to work on the other, very cool for people who love videos. We will also continue to work on minor improvements, like this, in future releases, to make it easier for users to work with Flash.

I wrote a couple of articles with which you can start working with Stage Video and native mouse cursors . I hope you enjoy these things. To work with 10.2, just use the latest versions of the Flex Hero SDKs, links to which can be found at opensource.adobe.com (4.5.0.18623 or 4.5.0.19786).

For more details, check out the Flash Player Developer Blog or write to other blogs where people like Brightcove talk about Stage Video .
