KDE Utilities. New site and module updates

A new representative appeared in the site collection about KDE - a site about small utilities included in the kdeutils module. Despite the fact that the kdeutils module in KDE appeared one of the first, it never had a website. Now everything has changed - utils.kde.org , here you can see a variety of information about KDE Utilities.


The kdeutils module has appeared in KDE since the project began. At the time of KDE 1.0 , it included the following programs:
  • KArm (now KTimeTracker)
  • KCalc
  • Kedit
  • Kflopy
  • KHexedit
  • Kjots
  • KLJetTools
  • KNotes
  • KTop (now KSysGuard)
  • KZip (now Ark)

Most of these programs are still part of KDE, but the development of three of them has been discontinued. Before the release of KDE 3.1, KLJetTools was removed - a very specific program for setting up HP Laserjet laser printers. KEdit and KHexEdit were removed before the release of KDE 4 and replaced by their more advanced counterparts - KWrite and Okteta, respectively.

Over time, the module certainly lost some programs, for example, KTimeTracker, KNotes and KJots are now part of the kdepim module. However, new utilities have appeared, the most recent update is the inclusion of Okteta and Printer Applet programs in the module (they will be released together with KDE 4.1). These programs are also the first among the main KDE modules, written not in C / C ++, but in Python.
So, the full list of programs from kdeutils:
  • Ark - utility for working with archives
  • KCalc - Scientific Calculator
  • KCharSelect - character table
  • kdessh - running programs on a remote host
  • KDiskFree - free disk space overview
  • KFLoppy - program for working with floppy disks
  • KGpg - GnuPG Interface
  • KTimer - run the program on a schedule
  • KDE Wallet Manager - KDE wallet management utility
  • Okteta - HEX-editor
  • Printer Applet - software for setting up and managing printers
  • SuperKaramba - Karamba version supporting Python
  • Sweeper - a tool for removing "traces of use" in the system


Each program has a page with a description, a development page (project activity, translation progress, etc.) and a contact page. Further content will constantly increase.
