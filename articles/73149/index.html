VS.Php add-on over Visual Studio for PHP


I am sure most of you, dear habraklyud, are familiar, and some actively use Microsoft Visual Studio, as well as there are people who simply do not represent their professional activities without this IDE. It is for these people that the VS.Php add-in was created!








Go!


And so, what does this add-in offer:
  • PHP4 and PHP5 support
  • Syntax Highlight PHP, Smarty, HTML, JavaScript, CSS, XML and XSLT
  • PHP, Smarty, HTML, JavaScript, CSS, XML and XSLT files templates
  • IntelliSense for PHP, Smarty, HTML, JavaScript, CSS, XML and XSLT
  • Built-in apache for debugging
  • XDebug and DBG support


The add-in is supplied with an installer so everything is simple :)

Creating a new project, we are presented with a standard window for selecting the type of project, in our case there are six of them:
  • Php 5 web project
  • Local IIS Project
  • Zend MVC project
  • Php 4 web project
  • Remote server project
  • Command Line Script project



After creation, we can admire the text editor window, so familiar and dear :)



The syntax highlighting color scheme is very similar to the scheme in NotePad ++, but you can't compare IDE with TE (text editor)
The add-in allows you to set the parameters for connecting to a remote FTP server, for automatic uploading of project files to the server after clicking the ‚ÄúPost‚Äù button -> ‚ÄúDeploy solution‚Äù



In addition to PHP files, you can edit the files * .css * .js * .html, there is naturally syntax highlighting, and in the case of CSS, the authors decided to make a convenient additional menu for navigating the style sheet items



Unfortunately, or fortunately, the developers did not consider it necessary to translate the add-in into various other languages, so we are content with what we have;)



There are two versions of this add-in: for VS 2005 and for VS 2008

VS.Php is not free and the official site is priced at $ 99.99, not weak, right?
