Material Design Icon Set by Google



In the list of the most popular Github repositories for the week, the Material Design icon set developed by Google has come out on top.

The set contains as many as 750 (!) Glyphs. These are icons that are often used in various mobile applications for media playback, communication, content editing, connection setup, etc.

Icons are designed to become standard for applications made in a unified style of "material" design, which Google introduced in Android 5.0 (see the Material Design specifications ). At the same time, they are equally well suited for Android applications, and for iOS, and for web applications.

The repository includes SVG versions of all icons for 24 and 48 px, SVG and CSS sprites, PNG versions for the web (1x, 2x) and iOS (1x, 2x, 3x), as well as high resolution versions (PNG). There is an online preview .
