Solutions for creating a social network

This text is inspired by the previous <a
href = " sevavaber.habrahabr.ru/blog/38613.html "> posts that list social networking engines, but are practically not analyzed in any way for use.

Here, I first collect all the possible engines that are found on the references, search engines and various archives. And filter them, removing obviously unsuitable for use by me. In this case, the use criteria for other people may be different, so all engines remain in the second part of the list. Total reviewed 65 projects.

Criteria for evaluation:
1. Code in OOP style
2. Mandatory use of the template engine (separation of code and layout)
3. Localization support (separation of texts and everything else)
4. Structuring (separation of modules into files and folders)
5. Open source (free or low cost)
6. implementation language (php, ruby, c #, python)
