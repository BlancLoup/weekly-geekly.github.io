A selection of SEO plugins for WordPress


I submit to your attention my selection of SEO plugins for WordPress. The collection includes plugins for page optimization, for linking and navigation, plug-ins for social networks and much more. In general, see for yourself ...

1. Plugins for page optimization


All in One SEO Pack
One of the best plugins to optimize a Wordpress blog for search engines. It has a variety of settings. Allows you to enter meta tags for each page, change the format for displaying titles, etc.
  • Automatically optimizes your headlines for search engines.
  • Generates META tags automatically
  • Helps avoid duplicate blog content on WordPress
  • You can redefine any title and set any META description and any META keywords you want.
  • You can make fine tuning for yourself

Platinum SEO Pack
More advanced version of the All in One SEO Pack. Additionally able:
  • Add nofollow tag to links!
  • Work with RSS - use noindex for comments and for the whole feed!
  • When changing, permalink automatically redirects.
  • Add META indexing control tags (index, noindex, follow or nofollow, noodp, noydir, noarchive, nosnippet) to any post.

HeadSpace2
Despite the functional similarities with All-in-One-SEO, for many it may seem much more convenient. Moreover, All-In-One has not been updated for a long time, and HeadSpace2 offers much greater user convenience and wider SEO functionality. There are ample opportunities to import settings, even from the magnificent All-In-One-SEO.

wpSeo
wpSeo is another SEO plugin that allows you to work with titles (title) and meta data records. In contrast to the previous plug-ins, it has more functions, is more convenient, but is PAID!
  • License for 1 domain / blog with the author of the blog - $ 24.99
  • License without reference to the number of domains - blogs with indication of the author - $ 64.99
  • Unlimited license - $ 124.99

Robots meta
The Robots Meta plugin allows you to manage the indexing of each blog page by search engines by adding META tags. The plugin is very useful in countering indexing duplicate content.

Features of the plug-in Robots Meta:
  • Manage .htaccess and robots.txt from admin panel
  • Manage meta tags for all pages: search results, admin pages, author archives, authorization and registration pages, etc.

SEO Friendly Images
SEO Friendly Images plugin automatically adds to the title and alt images, where there are none. Very useful plugin for indexing images. What matters a lot, it makes the page correct in terms of W3C / xHTML standards.

Redirection
For a variety of reasons, it is sometimes necessary to transfer a page from one blog site to another. In this case, you risk losing the existing status of this page in the search results. Redirection eliminates this trouble with the help of the 301th redirect. In addition, the plugin collects statistics of 404 errors on your blog and much more ...

SEO Slugs
Removes unnecessary words from permalinks, leaving only weighted words for search engines.

2. Plugins for improved navigation and relink.


Cross linker
Plug-in for internal relinking: the specified words are automatically replaced with words with the necessary links, flexible setting. Plugin features:
  • Perform internal linking based on specified words or phrases (both in posts and in comments to them)
  • Assign the necessary attribute to each link, for example rel = "" or title = ""
  • Import links from a blogroll to link words to texts with these links.
  • Set a list of tags whose contents will be ignored during perelinkovka
  • Set a word limit for relinking in one post and more.

Related Posts
Displays links to "related entries." The plugin is very useful for both people and search engines, because organizes additional linking on the posts inside the blog.

Similar Posts
Similar Posts more powerful plugin than Related Posts, because contains many additional functions, the main of them is the possibility of linking records from only one rubric.

Simple Tags
Very popular plugin for working with tags. It helps to choose tags for the post, supports auto-completion of key phrases, mass editing, auto-creation of tags and much more.

Wp-cumulus
Tag cloud in the form of a spinning sphere. Uses Flash.

Tagnetic poetry
Another flash plugin for displaying tag clouds, unlike WP-Cumulus, is not animated. Tags look like inscriptions on plates. It looks interesting and does not distract attention by movement like WP-Cumulus, it is also more convenient when you have a large number of tags and a large sitebar for installing the plugin.

Breadcrumb navxt
Creates bread crumbs (distance covered for better site navigation). Very handy thing for visitors and robots (helps to properly distribute pr and improves indexing).

WP-PageNavi
Creates a convenient pagination navigation, instead of the links "the following entries and previous entries." The WP-PageNavi plugin is very helpful in navigating users and PS.

Sitemap Generator Plugin for Wordpress
Generator html site maps. The sitemap facilitates both navigation and indexing by search engines. You can customize the display of all posts and pages, only posts or only pages, you can turn off individual pages and specific sections of the blog for display on the site map, allows you to customize the number of posts displayed and much more.

Google XML Sitemaps
The Google XML Sitemaps plugin creates a sitemap in an XML-Sitemap format for search engines.

Plugin to insert in each post panels with links to add articles to various social services bookmarks. Plugin features:
Automatic generation of links to the main popular English and Russian social networks
Manage displayed links through admin panel
Automatic or manual insertion of the code in the template

Tweet this!
Automatically inserts a link into blog posts, making it easier to post a link to a post on Twitter. Any visitor who likes your post can easily add it to their Twitter profile.

LIci WP
Plug-in for cross-posting on the famous Russian blog service LiveInternet.ru.

LiveJournal Crossposter
Crossposting on LiveJournal (livejournal), the world's most famous blog site =)

4. Work with external links


WP-NoExternalLinks
The plugin allows you to replace all external links with internal ones on the fly: in posts, comments, comments profiles. It can also add attributes to links: noindex, nofollow, target = "_ blank". The plugin does not change anything in the database - it just makes a replacement on the output.

RC Link Redirector
The plugin replaces external links to internal ones, and also encrypts them (for example: https: //www.wp-info.ru/goto/X0FCFlhNHUcFQhVXXBQZVV===).

(J) ExR
Another wordpress plugin for closing external links. Compared with RC Link Redirector has more functionality. Allows you to enable / disable link encryption.
