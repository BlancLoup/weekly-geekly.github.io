Samsung does not want to abandon the Start button in Windows 8



One of the most frequently described problems in the new operating system from Microsoft is the use of the new interface, which until recently had the name " Metro " and which, with its unusual concept, was able to scare off many people accustomed to the "canonical" Windows view. Apparently, given this circumstance, Samsung want on new PCs, which will be sold with the new system, to officially install the S Launcher application, designed to imitate the buttons familiar to everyone.

Technically, S Launcher is a widget that can be placed anywhere on the system‚Äôs desktop and gives access to installed applications, folders and the search bar - as it is done in Windows 7. In addition, the application provides basic customization options for creating your own list of applications .

Such a move, according to the leadership of the South Korean company, should significantly increase the popularity of the new system among ordinary users. The reaction of Microsoft to such intentions is not yet known, because, in fact, we are talking about another pre-installed program, which can be found on new laptops. On the other hand, Microsoft refused to start up after a whole study involving telemetry of user actions, from which it follows that most users place shortcuts to access programs right on the desktop or on the taskbar, and rarely look into Start.
