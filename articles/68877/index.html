How to install (for those who know how to install UserJSs, you can skip this):
1) to the Opera
Just place the script in the folder with your UserJs. You can find out the path to the folder like this: Ctrl + F12 -> Advanced -> Content (Content) (in the left column) -> Customize Javascript (JavaScript Options ...). there below will be the path to the folder with UserJS-files. Move the script to this folder and that's it. He is ready to go! :)
2) On the FF. (please specify commentators)
Install the GreaseMonkey plugin.
Then - Tools -> GreaseMonkey -> New Script
In the name, enter habr atl. In the Namespace \ Namespace field, type "window" Click OK. Choose your favorite text editor :). Erase all that is in the window and paste the contents of the ATL (by opening an already saved script in another notebook window). Press Ctrl + S. Exit the editor. All is ready.
