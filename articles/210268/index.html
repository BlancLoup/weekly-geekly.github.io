KolibriOS Digest # 1: getting up to date

If you are interested in KolibriOS and its development, but you do not have the opportunity to follow the innovations on the forum (which is sometimes difficult, especially if you only recently started to be interested), then this article is for you. This is the first article in a series of monthly digests aimed at covering events in the OS community. Digests will be published on the last business day of the month. Since this is the first article, then, in order to build an information line, changes over a period of more than a month will be considered. I want to emphasize that especially major changes will continue to be covered in separate articles. Well, let's get started.

To begin with, changes at the system level (kernel, drivers, and libraries):


  • The implementation of a system-wide clipboard (support for copying text from one program to another at the kernel level). Perhaps this functionality is taken for granted in any operating system, but until recently it was absent in KolibriOS.
  • Improved work with IDE / SATA IDE drives and DMA support.
  • Improving graphics performance with Intel graphics cards.
  • CardBUS driver (for now it only determines some cards).
  • httplib: a library for working with the http protocol on Fasm.

Not without innovations and application software:


  • fasm : upgrade to stable version 1.71.16.
  • Fplay : overall speed boost; special assembly with vertical sync for devices with Intel graphics.
  • Tinypad : support system-wide clipboard; multiple fixes and optimizations.
  • TextEdit : support system-wide clipboard; conversion of cp 1251-> 866 and cp 866-> 1251 encoding added; bug fixes and minor improvements.
  • shell : start implementing system-wide clipboard support.
  • icon : new desktop, with support for dragging icons, more convenient adding icons, the ability to add your items to the right- click menu by editing ini.
  • Docky : new program - dock bar ( see screenshot below ).



  • End : autosave image before shutdown.
  • PCIDEV : added device subclasses according to changes in the PCI bus specification in recent years.
  • board : decrease CPU load when writing to file.
  • rdsave : minimizing the interface and simplifying the work with the program.
  • HTMLv : extended tag support; go to httplib; optimization and correction of bugs.
  • Eolite : recursive deletion of folders; context menu; dialogue "open with"; copy / cut files / folders between Eolite windows; Russian translation; minor fixes and improvements.



  • Liza : improved parsing of letters.
  • Landing : new arcade game.
    Details
    The essence of the game: to clear a place for landing bombs and dodge anti-aircraft fire, making a " barrel ". The plane is gradually reduced. You can not control the direction of flight, you can only throw bombs and make a "barrel". The level is lost if the plane crashed into the building or it ended in "health" (in the upper left corner). To reset the bombs, press ctrl, for the "barrel" - a space. During the "barrel" can not throw bombs. Bombs can be dropped only after a certain time interval.
    It is worth noting that this is the first native game under KolibriOS with sound effects.

Projects with a distant (sometimes foggy) perspective:


  • NetSurf : Porting a full web browser.



  • polarSSL : SSL library.
  • TextGame : a platform for simple text games.
  • kolibri-process : a new branch of KolibriOS, in which a new model of processes and threads is being developed, more similar to the model of large operating systems.

Other news:


  • Added Estonian localization (in addition to the already existing 4 languages - Russian, English, Spanish and Italian). Those who wish to translate Hummingbirds into your language we will always be happy.
  • KolibriN : preparation for the release of KolibriN9 . Distribution, created by one of our developers (@Leency), to reveal all the features of the OS. KolibriN is, in fact, the maximum assembly of KolibriOS.
  • Wiki section : update and extension section for developers; reorganize pages to facilitate navigation.
