Fast closing tabs in the assembly Vivaldi 1.0.321.3

Hello!

Last week, we released the first real beta version of the Vivaldi browser, but the development process, of course, is not over. On the contrary, we began to receive more feedback and error messages, so the release of weekly test builds is simply necessary. And we fully agree with this. So, today another assembly is ready, in which besides the fixes some new items have appeared.

Close tabs

In this assembly, we slightly changed the behavior of the tab close button. Now, when you click this button, the tabs will not be resized simultaneously, so you can close several tabs one by one without moving the mouse cursor.


This function also needs to be improved in terms of the speed of closing tabs and displaying page titles, so all suggestions and error messages are welcome. However, as with other browser components.

We also fixed several errors in working with extensions - you can find the details in the list of fixed errors at the end of the article.

Important note

This build belongs to the test branch, so it is not recommended to put it on top of the beta version.
