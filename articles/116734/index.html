Files Over Miles: P2P via browser


I constantly encounter the problem of transferring a couple of files, for this I have to use DropBox, exchangers and similar services. Most of the exchangers are overloaded, have limitations, a ton of advertising, low speed. Torrents are more intended for transferring large files, you must have a private torrent tracker, a client.

Files Over Miles is a reliable and easy way to transfer files over the Internet. After the user has selected the file to be sent, the application generates a link that must be sent to the recipient. The addressee follows the link and confirms its readiness. Convinced of successful communication between the sender and the recipient, the information is transmitted. File transfer is carried out directly from one user to another according to the P2P scheme, through the browser. During the exchange, the data is not stored on the intermediate web server and is transmitted via network channels in an encrypted form.

Features:
1. works from any modern web browser;
2. high data transfer rate (depends on UL / DL between peers);
3. encryption (128 bits AES);
4. NAT traversal using UDP Hole Punching.

Limitations:
1. You must have Adobe Flash Player 10 and above;
2. the transferred file must not be larger than the amount of available RAM;
3. The recipient can receive files no more than ¬Ω of the computer‚Äôs RAM;
4. does not work if UDP is blocked;
5. The browser must remain open in the process of acceptance and transmission.
