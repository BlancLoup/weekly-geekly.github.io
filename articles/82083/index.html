Stylish and powerful client on Silverlight 4

Today, the Tim Heuer blog (program manager for Silverlight) posted the announcement that the application for working with Facebook, which Scott Guthrie demonstrated on PDC09, has become publicly available to everyone.

I must say that the new client is very good! From the very first menu, the client pleases with its appearance:




Window for status update:



To upload photos to Facebook albums, you can easily drag and drop files into the window:



The image galleries themselves can be viewed in a convenient view that resembles the Zune Software application (in general, the entire client reminds him and this is very good).



Supported notifications:



As well as uploading images to Facebook directly from your camera:



Any video uploaded to Facebook can be viewed right there in the client:



I'd add from myself that this client works very quickly, faster than Fishbowl - the previously submitted client for Facebook on WPF.

The most amazing thing is that with all the possibilities that the client offers, it is built on pure Silverlight! The whole application is just a quality, very beautiful demonstration of what Silverlight 4 is capable of today.

The application is built on Silverlight 4 and to use it now you need to install the silverlight beta version, which can be found here: Windows version or Mac version .
