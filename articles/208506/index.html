Light Table Source Code Published



Light Table is a new interactive IDE that implements some ideas for improving and simplifying development tools. Among the main ideas are the automatic display of documentation on any function when selected with a mouse, an instant display of the result of calculations and a user-friendly interface.

Having collected the required amount on Kickstarter , American developer Chris Granger kept his word - implemented Python support and published the Light Table source code on Github.

The program has all the necessary infrastructure for writing and connecting plug-ins. Compared with the previous version, in Light Table 0.6.0 the stability, performance of work and the completed interface are significantly improved. In fact, this IDE can now actually be used in the work.

The author says that through the Behavior-Object-Tag engine plug-ins of almost any functionality are connected. As an example of a simple plug-in you can see the implementation of support for CSS .



Another important feature of the original video on Kickstarter , which is now being implemented, is the built-in search for documentation. It is enough to select a code fragment with the mouse - and the reference information appears on the right side of the screen.
