Flash Player 10.1 goes to mobile platforms

Today, Adobe opened the secret of Flash Player 10.1, which will work not only on a computer, but also in a mobile phone. The new level of cross-platform will allow developing applications for different systems simultaneously: smartphones, netbooks, phones, etc.
The developer version will be available before the end of this year for Windows Mobile, Palm webOS and desktop Windows, Macintosh and Linux. Versions for Google Android and Symbian OS are expected early next year.

Flash Player 10.1 allows you to create complex applications with complex content and high-quality video, regardless of platform. In addition, the player more actively uses the computing power of graphics systems, while reducing energy consumption and reducing resource consumption.

In turn, the mobile version of the flash player supports multitouch, gestures, accelerometers, changing the position of the screen. Media is now quietly broadcast over HTTP, and content can be protected using Adobe Flash Access 2.0

Nokia, Google, Motorola and other companies are connected to the project. Phones with 10.1 support are expected early next year.
