Firefox 4b1

The first beta version of Firefox 4 has been released.

Major changes

  • In the Windows version, tabs are now placed at the very top of the browser window;
  • on Windows Vista and Windows 7, the top menu has been replaced with the Firefox button;
  • new extension manager;
  • the page stop button and the page refresh button were merged into one in versions for Windows, Mac and Linux;
  • The bookmarks panel is replaced by the default bookmarks button (you can return if you wish);
  • protection against crashes in versions for Windows, Linux and Mac, if the crash is related to Adobe Flash, Apple Quicktime or Microsoft Silverlight plugins;
  • partial support for CSS transitions;
  • full WebGL support (disabled by default);
  • WebM support;
  • new HTML5 parser
And so on.
