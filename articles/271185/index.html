Upsource 2.5: smart email notification, new Review page design, improvements in IDE plugin and much more

We are glad to share with you great news - today we have released a new version of Upsource, our tool for code review and navigation through the repositories.

This time we completely rethought and redesigned the UI code review pages, based on the feedback we received from our users. A new review page should be much more convenient and faster, especially for reviews with a large number of changes.



We also made email notifications smarter. Upsource no longer sends a separate message to every event that requires your attention. Instead, notifications are collected in one email, so as not to litter your email client. In addition, we added the ability to respond to comments directly from the email client.

We did not ignore the IDE plugin, which is especially popular among our users. Now it can work through an HTTP proxy and is even closer to the web UI in terms of its available functionality.



In addition, now the integration with the IDE has become two-way - the web UI has the opportunity to open a review or just the code in the IDE.
