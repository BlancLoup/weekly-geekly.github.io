How browsers work: how modern web browsers work

Looking through one of the training videos of the Interface Development School of Yandex, I came across a link to the awesome work of the Israeli web programmer Tali Garciel ( Tali Garsiel ) " How browsers work " (How browsers work).

For several years, she tracked all the published information about the internal structure of browsers, studied the source code of WebKit and Gecko and, finally, put everything together. This is what Tali herself writes:
When IE was installed on 90% of computers, it was necessary to put up with the fact that it was a mysterious ‚Äúblack box‚Äù, but now that more than half of users choose open source browsers, it is time to figure out what is hidden inside them, in millions of lines of software C ++ code ...
Scrolling through, I was amazed - great work. The internal structure of browsers, parsing algorithms - everything is well illustrated, accessible and understandable. And without unnecessary details, pages 30-40. Just what you need. I decided - it must be translated. I dug a little more - it turned out that the translation is already 1.5 years old!

Perhaps for someone this is no longer news, but I hope it will be useful for those who have not heard (I did not find any references to it).
