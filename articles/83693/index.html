Rotation of geographic maps with CSS3 and jQuery

One of those pieces that I always wanted to do with network geographic maps is to rotate them: I'm used to this with real, material maps. And since the material cards today are becoming much smarter (for example, have you seen a card that can grow ?), It is time to be able to do this with network cards too.

Although Google supports the rotation of satellite and hybrid maps, the main ones cannot yet be rotated. That's why I took CSS3 transforms (wrapped up in a very useful jQuery plugin ), and voila! - rotation made possible:
