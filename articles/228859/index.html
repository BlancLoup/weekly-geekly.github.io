ThinkServer RD640 - Lenovo server, charged for work

Good afternoon dear friends.

Lenovo ThinkServer RD640 server is special, and stands out among its competitors, it has an individual architecture that provides, in my opinion, high performance on the cost of ownership. What makes it quite competitive in solutions for the corporate sector in its segment.



The main features of its architecture that are worth paying attention to:


Hardware component of ThinkServer RD640:
  • Supports up to two Intel Xeon E5-2600v2 FCLGA2011 processors (up to 12 cores per processor).
  • Supports 20 memory channels up to 320 Gb DDR3 (12 for 1 processor, and 8 for the second processor). Below we look at the pros and cons of this solution.
  • RAID support with Cashe SSD, LSI based.
  • Up to 16 2.5 ‚ÄùSFF (Small Form Factor) SAS, SSD, SATA drives.
  • Up to six PCIe expansion slots (2 PCIe x16 and 4 PCIe x8 FH / HL)
  • Supports two 800w power supplies each 80 Plus Gold with redundancy.


Engineers - designers have invested their whole soul in the development of this solution, the design turned out to be both external and internal, very laconic, without excesses. So far, no one has been hurt, but I would add a lid with a lock to protect from unnecessary hands.

As we see on the slide below, the layout of the external elements on the front of the server is quite familiar and intuitive.



The internal design of the server solution Lenovo ThinkServer RD640 is designed to maximize cooling during operation.



I also want to note that the baskets where the hard disks with the slides are inserted (highlighted in purple in the figure) are designed to absorb the vibration of the disks, which increases the service life of the disk system.
Thanks to engineering solutions, the cooling system meets high requirements, which undoubtedly makes it possible to place servers with high density and not be afraid of overheating. But if you are building a data center, then you still need external cooling for security reasons.



All elements of the server, as well as the active cooling unit (fans) are available for installation without additional tools, which makes it possible to quickly replace (supported replacement in working condition).



The processors and RAM in the server are distributed in one line, immediately behind the cooling unit. The Lenovo server runs on Intel E5-2600 v2 processors. Immediately I want to notice the feature of the solution, since the user does not always use the second processor and, all the more, they do not always clog up the server with memory, the engineers made a decision that not everyone understands. The number of RAM slots for the second processor was shortened from 12 to 8 places, while retaining all the basic principles of the channel layout structure. More details please read the picture below.




This solution has reduced the cost of owning a server without losing performance.
This is undoubtedly reflected in the best price per server compared to competitive offers on the market.
