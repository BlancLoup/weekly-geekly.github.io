Go-oo: Easier and faster OpenOffice with additional features



One of the main drawbacks of the open source product OpenOffice.org is that its applications run more slowly than conventional applications and even slower than their main competitor, Microsoft Office. If you are faced with a similar problem and are looking for a solution that would allow you to work with Microsoft Office-compatible documents, then try Go-oo (go-oo.org).

Go-oo is a branch of OpenOffice version 2.4 for Windows and Linux. It does not have some of the features of OpenOffice 3.0, but is easier and faster and includes a number of additional compatibility features that can be difficult to find in other products, even if you work mostly with OpenOffice. There is no need to delete OpenOffice in order to try Go-oo.



Go-oo contains embedded OpenXML filters and allows you to import Microsoft Works documents. You can import graphics, documents, and even complex macros from Excel to Go-oo Calc without any problems. Such macros work even better in Go-oo than in OpenOffice Calc. The standard applications in Go-oo are the same as in OpenOffice: Base, Calc, Draw, Impress, Math and Writer.

Users of the Linux version of OpenOffice can appreciate the additional features of Go-oo, such as improved multimedia performance. Those who are looking for additional Excel functionality will appreciate Solver (in the figure below, Solver performs optimization) and support for VBA macros. Also, Go-oo allows you to work with graphics created in Microsoft Visio.



In general, when you choose software, you look primarily at performance. Go-oo is easier, faster to run and faster (compared to OpenOffice). And it is just convenient to keep it close at hand, especially since the procedure for downloading and installing lasts about 5 minutes. Sometimes two heads are better than one.
