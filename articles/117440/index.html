Surtout - Ruby on Rails CMS

Surtout is a free open source CMS running on Ruby On Rails 3. The project is currently under development, but now you can watch a demo video on the project website:





Indeed, to some extent Ruby on Rails is already a great CMS in itself. Already written a bunch of plugins from which, as from the bricks, you can build anything. But when you create a website for a client that is far from IT, you cannot do without a beautiful, intuitive interface, no matter how cool. Given the frantic popularity of Ruby on Rails, it is surprising how few are worthy projects in this area. In my opinion, the only serious competitor now is RefineryCMS .

The main purpose of this post is to collect opinions about the development and, possibly, to find like-minded people. Also, there may be investors willing to support the project.
