Invite on Windows Store or how to get into Windows 8

Last month is full of news on Windows 8. Since the beginning of June, the Windows 8 Release Preview has been publicly available along with the first Russian applications , announcements of new devices have appeared, it has become known about the possibilities of upgrading from Windows XP, Windows Vista and Windows 7 to Windows 8 Pro.

Yesterday it was announced that the final version of Windows 8 (RTM) will be released very soon - in the first week of August.
Let me remind you that inside each device with Windows 8 there is a single application store - Windows Store. This is a single channel for delivering your applications to users of all desktops, laptops and tablets.
Thus, finding an application in the Windows Store provides direct access to the entire ecosystem of Windows users.
You can develop applications for Windows 8 using HTML5 / JavaScript, C # / XAML, or C ++ / DirectX.

Today, access to the Windows Store is open only by invite - only the most interesting and high-quality applications get into it. Of course, over time, it will be open to all comers. However, right now there is an opportunity to become one of the first applications at the very moment of maximum attention of users, journalists and device buyers.
