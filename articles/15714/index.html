Firefox Ultimate Optimizer

Today, for the sake of interest, I downloaded Firefox Ultimate Optimizer , developed by Spaniard Felipe Rafael Sarabia Caravantes (like his name). The author of the program promises that Firefox will eat less memory. He usually uses 150-200 MB for me with 30 tabs open. Before starting the program I checked for viruses, Nod32 did not find any viruses.
After launching this program, an icon appeared in the tray and Firefox began to use really less memory (the Firefox process itself is 15-35 MB and the optimizer process itself is about 12 MB). I do not know if this is a miracle or a hoax, please check and unsubscribe here about this thing.
The disadvantage of the program: the interface in Spanish, although in principle it is not needed because there are no settings and the interface itself.
