Media Resources module: cornerstone

Preparation of promotional materials is a very multi-step process, complex, associated with a bunch of all sorts of approvals, numerous adjustments and modifications of files. Headache is enough for everyone - both designers and customers. A lot of time is spent on routine - phoned, e-mail correspondence, sending back and forth file previews.

In order to get rid of all the annoying delays and make working with media content easy, pleasant and efficient, there are specialized media resource management systems (Media Asset Management, abbreviated as MAM systems).

We bring to your attention a story about such a system on the example of the BrandMaker Media Resources module. Moreover, it is one of the key modules of BrandMaker, around which all the others are built on. It provides centralized work with documents, images, multimedia data.



Any MAM system rests on three "whales":

1. Central storage of media objects of any format

The core of the MAM system is the central base of advertising materials, which serves as a storage place for media objects of any format and type, such as individual graphic images, brochures, booklets, banners, as well as video and audio files - everything that can be attributed to advertising resources.
Employees of the company can independently download the necessary advertising resources from the system via a local network or the Internet to use them in presentations or other advertising projects.
The client company itself can customize which advertising resources can be used under a certain license and for certain purposes, as well as which employees will have access to topics or promotional materials. Additionally, you can set the expiration date of the image or document.
All these tools provide the best protection against the illegal use of marketing materials, make it impossible to use uncoordinated or outdated versions, violation of corporate style requirements.

2. Processing the source material before loading.

This may include a convenient document preview function - when you hover the cursor over the file name, a preview is displayed with basic information about the document.

And the previews are created on the server, without affecting the productive capacity of the computer of the end user. Well, finally, I note two more useful features of the module.

First of all, the module does not delete earlier versions of images and documents when consistently reworking a file (for example, an advertising layout). So at any time, you can track the history of changes, go back to one of the previous stages.

Secondly, the search function is particularly convenient, acting on the principle of Google, when the search result is displayed after entering one or more keywords. The user can reduce the number of search results and make it more accurate, this will help him search by criteria: the amount of information, the format of data or images, as well as data access, etc. The functions of sequential search by several tags (‚Äúin the found‚Äù), or by several tags at the same time are implemented.

In addition, the data structure itself, individually developed for each client, helps to quickly navigate the information and simplifies the search for thematic and specialized material. Thus, the search for the necessary media objects is significantly accelerated.

3. Integration with other modules and the creation of a complete system

MRM-system BrandMaker covers all marketing processes: from planning and budgeting marketing projects, creating advertising materials, to generating reports and analyzing results.

"Media Resources", being a module for storing information, serves as a starting point for all activities in the framework of creating promotional materials. To create advertising objects, both for print and electronic media, MAM-system offers all the necessary advertising projects that correspond to the company's corporate style and are stored in the central multimedia content database.

Individualization of certain printed materials (advertisements, brochures or business materials) is carried out using the Web-to-Print module, and electronic materials for the Internet using the Web-to-Web module.

When creating a new advertising material, the Task Manager module will help you - the main module for creating and managing tasks and workflows in the field of marketing. At the end of the work materials are stored in the ‚ÄúMedia Resources‚Äù for later use.

The Marketing Planning module helps in general activities in the creation of advertising resources: planning, budgeting, information sharing and the use of the developments of previous advertising campaigns and projects.

Thus, being integrated into the MRM system, the BrandMaker Media Resources advertising storage facility provides a wide range of capabilities that go far beyond simple storage, management and preparation of media data.
