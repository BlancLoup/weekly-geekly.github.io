Apple fixed important OS X vulnerabilities

Apple released OS X 10.10.2, closing 54 vulnerabilities as part of the update APPLE-SA-2015-01-27-4. Most closed vulnerabilities allow attackers to execute arbitrary code in OS X, including system privileges. Apple also fixed a dangerous vulnerability called Thunderstrike , which compromised the security of OS X at the very early stages of the boot (OS X bootkit). Update APPLE-SA-2015-01-27-4 fixes multiple vulnerabilities in the OS, which were publicly disclosed by a team of experts from Google Project Zero.
