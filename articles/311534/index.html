Microsoft Ignite Digest

Good afternoon, I will try to explain the main news from this event. Focus on Azure, Azure Stack.


Go!

General news


  • Released to RTM Windows Server 2016 . Read more about innovations . Release mid-October. Docker Datacenter for Windows Server 2016 in Q4 2016 .
  • Released to RTM System Center 2016 . Read more about innovations . Release mid-October.
  • Changes in the certification branches .


miscellanea


  • Video from the event . Here are just technical (sort of) .
  • Details about Storage Spaces . Bonus: 2-node Storage Spaces Direct supported configuration.
  • SQL 2016 supports Windows Server 2016 and Storage Spaces Direct.
  • Updates for OneDrive .
  • OMS announcements . OMS Gateway and SCOM . Monitor VMWare . AppInsights Connector for OMS .
  • Details about the new in Hyper-V . More about "Large Scale VM Performance" .

Azure


  • Improved network performance in Azure by 33-50%. Fully implemented in 2016.
  • Released to Accelerated Networking. As far as I understand, this is just SR-IOV in Azure. This functionality will be available not only for virtual machines, but also for some services (Azure SQL, Azure Storage) when it is fully implemented, no user action is required. For virtual machines .



  • VNet Peering has been released. Read more about the service .


  • New SKU for ExpressRoute, 10Gbps, 99.95% SLA
  • New Active-Active VPN Gateway .
  • IPV6 support for Azure virtual machines.
  • Support for multiple VIP addresses on the ILB (internal load balancer), support for multiple IP addresses on a network interface card of a virtual machine, support for public IP for all network interface cards of a virtual machine (including via Load Balancer).
  • Azure DNS was published "in the light."
  • Improved security of the Application Gateway (end-to-end SSL encryption, user configurable SSL policies, routing rules) Public appraisal of Web Application Firewall, a service that is pre-configured to protect you \ your service from the most common types of attacks (according to the Open Web Application Security Project).
  • A new set of metrics for network monitoring , more details on the link (at the very end).
  • Graft Azure Monitor'a . Uses existing metrics and allows them to be viewed and further customized. Allows you to integrate with OMS \ Visual Studio Application Insights.
  • H-Series virtual machines . Aimed at high performance computing tasks.
  • L-Series virtual machines. Optimized to work with the story.
  • N-Series virtual machines. They have a graphic adapter from NVidia.
  • Azure Service Fabric for Windows Server has been released . Graft for Linux (news not from Ignite).
  • Azure Key Vault supports certificates from Digicert, Globalsign and WoSign.
  • Encryption Services are out.

Azure stack


Azure Stack TP2 released , in it ( more info ):

  • Architecture: Describing the many components and design of the Azure Stack .



  • Storage: Azure Queues, Premium Storage API, Deleted Storage Accounts Recovery
  • Network: User Defined Routes (similar functionality from Azure), VPN Gateway to communicate with Azure (and not only), iDNS (literally: a way to resolve DNS names of virtual machines inside the Azure Stack without configuring DNS)
  • Compute: Deallocate virtual machines (as in Azure, just turned off the virtual machine continues to spend your money, only the deallocated virtual machine is worthless), resizing disks, several network interfaces, reinstalling VM Extensions.
  • Portal: Regions for compute, network and storage appeared in Azure Stack. Preview of updates on the portal.
