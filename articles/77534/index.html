ESET begins beta testing of antivirus solutions for Mac OS X and Linux

Now, beta versions of ESET NOD32 for Mac OS X and Linux operating systems are now available to users.



The new products are based on the ESET NOD32 Antivirus kernel version 4, the architecture of which allowed it to be quickly optimized for Mac OS X and Linux. It is based on the ThreatSenseTM technology, which allows you to detect both known and new threats that have not yet been made to the signature database. The solutions use the latest ESET signature database, to which malware signatures for Mac OS X and Linux have been added. Thanks to technology Host Intrusion Prevention System (HIPS), ESET solutions protect against attempts of external influences that could adversely affect the security of Mac OS X and Linux.

ESET solutions for Mac OS X and Linux have significantly expanded scanning options ‚Äî the user can adjust the depth of viewing, file size and object scan time, get statistics on the scanning process of PC files, the number of detected threats, and real-time status reports also provides automatic scanning of removable media. The anti-virus database is updated depending on the user's settings - when the PC is restarted or immediately after the appearance of new signatures.



ESET NOD32 Antivirus for Mac OS X is designed for Mac OS X 10.5.x (Leopard) and Mac OS X 10.6.x (Snow Leopard) operating systems. ESET NOD32 Antivirus for Linux is compatible with RedHat, Debian, Ubuntu, Suse, Fedora and Mandriva distributions.
