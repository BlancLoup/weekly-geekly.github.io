New firmware - new phone

Yes, this time I allowed myself to call the topic that way. Two days ago, a new version of the firmware for the Nokia 5530 was released. I am amazed this time on the spot with the speed of my (what's hard to check) smart. All problems with input languages, kinetics, official firmware bugs (and such were) were solved. I note that in the subject I will consider a modification of the firmware called Simple Mod, which I considered earlier . Then version 1.0 was discussed. Now - meet version 4.0.
Under the cut list of changes, new instructions for the firmware, along the way, I will take screenshots, t. it's worth seeing. Less words, climb under the cat.


Well, perhaps I'll start with the list of changes, so that all the screenshots are clear. I will clean the list from trivial changes, it is very big. Who needs a full - write.

List of major changes


2.0

- Breaking protection
- Removed unnecessary programs from autoload
- Russian and English input languages
- Russian and English phone
- Delete standard browser bookmarks
- Changed the script executed at the first launch (ScriptInit)
- The protection from SIS transmission through the standard file manager has been removed.
3.0

- The player scans only E: \ Music
- Fixed screen rotation (configured just perfect) [!]
- Removed all help
- Accelerated application launch
- Application cache unlimited [!]
- The phone boots after turning on for a couple of seconds [!]
- Added Ukrainian input language [!]
- Version without kinetics in the menu [!]
4.0

- Standard themes - themes from Nokia N97 (if you want to leave the standard - do not change the * .fspx file to the modified one)
- New splash on / off
- Changed the font to Calibri from Windows 7 (not stupidly changed all the files for one, but replaced it wisely, so it won't be small)
- adjusted kinetic scrolling
- Returned voice commands
- Added touch control settings item
- Audio codecs from Nokia N97 (the quality is really better)
- Flicker without limit (if you have a missed call or message - the middle button of the phone will flash)
- In the QWERTY-keyboard instead of the button of hieroglyphs - the English layout (thanks xx3472)
- Java Runtime 2.1.18 is sewn in (now JAVA flies at all, put Opera Mini and you will not regret it ;-) there are no brakes)
- Added versions without Ukrainian input (by popular demand)
