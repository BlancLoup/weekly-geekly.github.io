TinyTwitter
www.tinytwitter.com
TinyTwitter has several versions: for WM (PPC and Smartphone), as well as two Java versions (MIDP 1.0 and MIDP 2.0), that is, theoretically, it can be used on all WM devices and on devices with java support.

The interface is not quite thought out - each tweet takes at least half a screen, large tweets - the whole screen. There, however, you can recommend it for reading - it is fast and bug-free, uploaded tweets in the last 100 hours.
