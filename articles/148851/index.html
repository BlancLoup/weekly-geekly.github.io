VKontakte messenger for Windows Phone, prize pool 2 000 000 rubles!

VKontakte is holding a contest to create an instant messenger for Windows Phone based on the existing API.
The start date of the competition is August 1, 2012, the deadline for accepting works is September 15, 2012 (21:00).
The prize fund is 2 000 000 rubles .

Contest group : http://vk.com/wp_contest

Conditions of the competition ( http://vk.com/page-41437534_43978998 ):
The task
We drew all the main screens and interface elements to create a messenger for smartphones based on Windows Phone using the VKontakte platform. A full description of the functionality is available on a separate page .

Your task
It is necessary to implement the application in the C # language on the basis of the VKontakte API in the volume described in the task. It is important to accurately reproduce the proposed aesthetics and achieve a high speed of the application: initial loading, opening of dialogs, sending messages and attachments, smoothness and reasonableness of all animations. Deviations from the proposed design and functionality are not allowed. Unstable and unfinished applications will not be allowed to participate in the competition.

The application must be compatible with all popular devices based on Windows Phone . All possible screen resolutions should be supported. An additional requirement for the task is the presence of Russian and English versions of the interface.

Publishing the application in the Windows Phone Marketplace to participate in the contest is optional. The developer has the right to choose any name for the application at its discretion. The application must be self-sufficient even for a user unfamiliar with VKontakte. The copyright of the application remains with the developer.

For ease of development, we have prepared all the necessary set of API methods on a separate page .

A set of graphic materials in PNG format , layouts of all screens and source graphic files in PSD format can be downloaded here: http://vk.cc/Ra6Tv

Reception of entries
By 9:00 pm on September 15, 2012, entries must be sent to contest@vk.com in the form of installation xap files and an archive with the source code of the application. The list of works of all participants and the results of the competition will be published in the Competition group and in the VKontakte API group.

The competition is open to residents of any country , not just Russia.

All participants are encouraged to join the Contest group to get relevant information and participate in discussions.
Information on developing on Windows Phone is available on MSDN in Russian and English .
