October 23 - Virtual Conference "All about Windows 8.1" for developers


On October 18, Windows 8.1 will become available to everyone, with a variety of new features for the user, a Start button and a huge amount of improvements for application developers. At this virtual event we will tell you what's new for the user in Windows 8.1 and how it is displayed on the new opportunities for the developer. Let's take a quick look at what's new in the WinRT API and dwell on the new features for developers of XAML / C # and HTML / JS applications.

The broadcast will be available from 10:00 MSK on October 23 right on the event page: www.microsoft.com/ru-ru/events/ep/allabout81/default.aspx

The speakers at the conference will be Konstantin Kichinsky and Stas Pavlov: www.microsoft.com/ru-ru/events/ep/allabout81/speakers.aspx .
Kostya ( kichik ) and Stas ( stasus ) may be familiar to you from speeches at numerous conferences, including the confession Windows Camp, Windows Phone Camp, DevCon, Design Camp, HTML Camp, TechEd Russia.

The conference will begin at 10:00 MSK c Konstantin Kichinsky's story about what opportunities Windows 8.1 provides for the user and developer , the conference will continue at 10:40 MSK with a review from Stas Pavlov of the available, updated and new WinRT API in Windows 8.1 . And the first part of the conference will end at 11:20 MSK with a talk by Konstantin Kichinsky on the design of applications for Windows 8.1 .

The following two and a half hours of the conference reports will be presented separately for XAML / C # and HTML / JS development:

12:00 MSK Application structure, standard templates, Hub template and processing of different views
12:30 MSK break
12:40 MSK Navigation and commands in the application: AppBar, NavBar, CommandBar and Flyout
13:10 MSK break
13:20 MSK ListView, GridVew and Data Linking
13:50 MSK break
14:00 MSK Work with contracts: Sharing, Search, Print. New search features: SearchButton and SearchBox

And at 14:40 MSK, Kostya and Stas will continue the story about advanced development capabilities. At the beginning, students will learn about live tiles, alerts and interaction with Windows Azure Mobile Services, and then at 15:20 MSK, you can learn more about the basics of debugging and optimizing applications for Windows 8.1 .

At 16:00 MSK Kostya and Stas will answer the most popular questions of developers.

Also throughout the day, we will monitor the live chat and Twitter chat via the # ruwin81jumpstart tag and answer questions during the breaks of the broadcast.

We are waiting for you at the virtual conference!

For convenience, you can add an event to the calendar: www.microsoft.com/ru-ru/events/ep/xmls/allabout81/cal.ics
