Tagnetic - tag cooler, plugin for jQuery

In response to a flash script from Ray Tanck implementing a tag cloud, stylized as a board with magnets , I wrote a jQuery plugin that creates a tag cloud, stylized as a fridge with magnets stuck to it :)

The plugin automatically replaces the cloud on the fridge, and if the user has the java script turned off, then the standard tag cloud will be shown to it.

See the script in the work here and here . If you wish, you can download the refrigerator tags and get acquainted with the documentation .

Tested work in FF 2 and 3, IE 6 and 7, Opera 9.60, Safari and Google Chrome. Unfortunately, in IE 6 due to the use of IE PNG Fix script noticeably slows down. Now we are working on the creation of new skins (I want to convert the refrigerator into a board or fence with ads-tags tagged on it) and on optimizing graphics.

The opinion of habrayuzer on the usefulness of this work, as well as possible glitches, is interesting.

Upd. With the filing nikel303 made an example in which magnets can be dragged.
