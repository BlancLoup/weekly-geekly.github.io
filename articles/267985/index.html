PBX 3CX for hosting - change the rules of the game

Since early September, a free cloud hosting platform has become available for download.

As it was before?


We have previously had a solution for cloud service, called 3CX Virtual PBX. The platform is installed on Windows Server 2012, and it doesn't matter if it is a virtual machine or a piece of hardware, and up to 25 independent IP PBXs could be deployed on it. It could be free demo versions or commercial licenses.



How will it be now?


3CX Cloud PBX Server is a new installation option for 3CX Phone System v14 that allows you to host up to 25 independent IP PBXs on a single physical server. All entities are separated from each other. Licenses can both temporary and lifelong. The platform itself is completely free.

How to set?


Downloading the distribution, starting with version 14, the installation package is one for the cloud version and for stand-alone:
http://www.3cx.com/phone-system/download-phone-system .

Software Requirements
1. Windows Server 2012 64-Bit. (IMPORTANT: Server core editions NOT SUPPORTED)
2. Microsoft .NET Framework 4.5.
3. Internet Information Services.
4. VMware ESX or Hyper V, ONLY 3CX Phone System should be installed on the server.

Minimum hardware requirements for 25 PBX
1. Intel Xeon CPU - E5505 2.00 GHz 2.00 GHz (6 - 8 cores) or higher.
2. Memory: 25GB or more.
3. HDD 80GB + place to record conversations
4. HDD performance: 80MB / s and more
5. Network: 1Gbit / 10Gbit

Also available is the option of deploying IP-PBX on its own cloud server on 3CX hosting, but more on that next time.
