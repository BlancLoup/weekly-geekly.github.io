Mobile integration with the new cloud PBX 3CX v14

3CX defines innovations in unified communications (UC) with support for PUSH technology for mobile clients, the ability to install PBXs in the cloud and an improved user interface.

3CX , the developer of a new generation of software PBX 3CX Phone System for Windows is the 14th version of its award-winning telecommunications solution. 3CX v14, using a single platform, can be placed both locally at the customer and in the public cloud. This new feature allows integrators to expand the range of services provided by including the service of cloud PBX.

New mobile clients for iOS, Android and Windows Phone


Clients rewritten from scratch for iOS and Android raise user mobility to a new level thanks to the integration of PUSH notifications and SIP tunneling technology. Users can use their smartphones instead of desk phones with the same ease that they are used to making regular mobile calls. PUSH notification of an incoming call or chat message activates the mobile client, which immediately accepts the call.

Nick Galea, 3CX CEO
Version 14 sets new technological standards in all directions, but especially in smartphone support. With integrated PUSH technology that guarantees PUSH messages to 3CX highest priority, 3CX Phone System offers unrivaled mobility in your pocket - wherever you are! Add to this the simplicity of setting and maintenance - and 3CX Phone System becomes the exclusive offer on the market.

Integrated Virtual PBX Hosting Features


We provide our partners with a new opportunity - hosting 3CX as a hosting / cloud PBX. 3CX Phone System v14 allows you to install up to 25 virtual PBXs on a single Windows server, significantly reducing the cost of installing new entities and simplifying their maintenance. 3CX offers a real virtual PBX, which differs from the usual multi-user systems in that each client has its own independent installation, including a set of services and personal data.

The possibility of a cloud installation appeared on the wave of increasing interest in cloud solutions for business. Studies predict that by 2018 the market turnover of cloud PBX and unified communications systems will be $ 18 billion. With the 3CX Phone System, integrators will be able to offer customers both the implementation of the system locally and cloud PBX / UC services.

Simple maintenance with additional features for administrators and support for new SIP trunks


For system administrators, 3CX offers improved administration features, such as backup and restore configuration on a schedule, new notifications of important events and an optimized management interface. It also offers a built-in fault tolerance system that requires minimal configuration. 3CX Phone System PRO can maintain a hot-swappable copy of the system. In the event of a primary system failure, the second node of the cluster is immediately put into operation. In addition, administrators will appreciate a number of new features, including scheduled e-mail call reports, voice mail storage and call recording quotas management, and many predefined templates for SIP operators, such as Broadvoice, AMcom, Deutsche Telekom and Time Warner Cable.

Improved built-in video conferencing


The corporate video conferencing system 3CX Webmeeting has also been significantly improved. In particular, the quality of the video has been improved thanks to the enhanced support of WebRTC. There are new features that are indispensable for webinars and online learning: built-in remote desktop management, the ability to host recorded webinars on the Youtube service and feedback surveys. Now you can transfer web conferencing management to another participant and download Power Point presentations in a special HTML format to improve quality and reduce bandwidth. Companies using 3CX Phone System can create WebMeeting web conferences for up to 10 simultaneous participants for free. Caring for the quality of the service provided, 3CX has deployed WebMeeting servers in data centers around the world to ensure the best possible video transmission.
