Microsoft has launched a beta version of the new service Live Folders

On June 26, public access to the beta version of the Live Folders service was opened. The service is a file storage with the ability to set different levels of access. Each new user is given 500 megabytes. Perhaps in the future, it will be possible to pay for additional space. The file is divided into several types, the maximum size of a single file is 50 megabytes.


Three access levels:
  • Personal
  • Shared
  • Public


Personal data is available only to you. Shared gives you the opportunity to grant rights not only to read, but also to write. Public gives access to data to any user. All files must be downloaded manually, the PC synchronization utility is not, but it is rumored that it will be.

Rumors about the new service have been around for a long time, including on the habr .
