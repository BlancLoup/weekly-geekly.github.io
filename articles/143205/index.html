Google has discovered the key technology Street View

Google has laid out the open source library C ++ of Ceres Solver , which simulates and analyzes the complex problems of nonlinear minimizing the sum of squares.

The method of least squares is used in various fields: from building complex curves in statistics to building 3D models from photographs in computer vision systems. In Street View, the system is used to link the expected position of the car (model) to real coordinates (data).



The blog Open Source at Google lists the main features of the library.
  • Simple, expressive API
  • Automatic differentiation
  • Reliable loss function
  • Local parameter setting
  • Jacobi interpreters and linear algebra, everything works in threads
  • QR decomposition of a dense matrix (uses the Eigen library)
  • Cholesky sparse matrix decomposition (uses SuiteSparse )
  • Specialized methods for 3D computer vision problems
  • Free license (new BSD)
  • Scaled from servers to mobile phones
