Govnokod.ru 2.0


Govnokod.ru is a small project dedicated to bad code.

About nine months have passed since the launch of the first version of the govnokod and today I am pleased to present you a new version, so to speak, ‚ÄúGovnokod‚Äù 2.0!




about the project


"Govnokod.ru" is a quote for programmers and about programmers.
How many times have you come across a code that either made you very angry, or made you smile?
When, once again, you find or write such a code, do not forget to share it with other people.

A bit of history


The idea to create a project was born suddenly and just as suddenly it was brought to life! The project was made fun for the sake of stretching their brains, which had become ossified at that moment. In a short time, the main functionality was implemented and the site came out approximately in this form:


The project somehow earned, the content began to increase, the first reviews began to appear. Together with the project we survived the move to a new server, survived even small hacker attacks (however, it was not always possible to do without victims :)).

All this dynamics captured me and did not let me get bored! I began to spend all my free time on this project, until one day my friend (and part-time designer) did not show his new sketches of govnokod design. I liked this design so much that I decided not to just redesign the site, but to completely redo the project.
The result can be seen below:


What has changed?


The concept has not changed. As before, some users publish various code fragments, while others vote and comment.

Here are just a few of the changes:
  • User accounts entered. Govnokod.ru supports OpenID technology, and you can use Gravatar as avatars.
  • Comments have become tree-like and a list of them can be obtained right on the page where the code is viewed: you no longer need to go into the detailed view to read or leave a comment.
  • The way the code is highlighted is now selected by the user. You can use either server backlighting or client-side highlighting with JS.
  • Reworked rating system.

Govnokod.ru, as before, works on the mzz framework and is not going to flow from it.
