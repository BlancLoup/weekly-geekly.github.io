DPI



Here we see the careful and precise DPI setting. I will try to briefly explain why it is needed at all for those who have never changed it and used what the mouse originally gives. If to be pedantic, then dpi - dots per inc, that is, the number of points per linear inch. Since these are not quite standard parameters for Russia, I will explain it easier. DPI Used to indicate resolution, or, if simpler, to indicate the speed at which the cursor moves across the screen. The higher the DPI, the faster the cursor moves accordingly.

It would seem that it could be easier, the more - the better. But no, not always, I tell you.

Low DPI is very useful for several things:
  • You are a designer, and you need to be very precise. Clearly draw a line or put an element of the composition
  • You play a game where you need to very clearly position the heroes and so on.

In general, this mode is useful in cases where you need to do something very clearly, accurately and painstakingly.

High DPI is useful:
  • You are a designer and you need to quickly paint over a large picture or apply some sharp strokes.
  • You play Action and you have to react very quickly.

In any case, high DPI is useful for fast and powerful things.


But you probably ask, why is there such a fine tuning here?

And I will answer you - this is because Razer cares about consumers and allows them to choose for themselves the ideal setting. That is why you can set exactly the DPI that will be convenient for you, not the manufacturer. This is very convenient, because you customize the mouse to suit you and it will be very convenient for you to use it. I have, for example, 2 settings - for Photoshop and games - these are 2500 and 5000. Also, you can configure several presets and switch between them.

Profiles



A very useful thing, especially if you put a profile switch on any button. In essence, a profile is your saved settings for individual applications or users. You can use them as you wish. For example, I created profiles for design, web surfing and programming. For Photoshop, 12 buttons are assigned to make settings, in Delphi they are certain constructions that are inserted into the code, but in Firefox, it is forward, backward, copy, print selected, etc. In general, it seemed to me very convenient.

Macros




Do I need to tell what macros are? I think no.

I want to say one thing: with this mouse they are very useful, because There are no very advanced settings for the buttons, you can give them virtually any action.

Trainers (trainers) or "navels"




This is also quite an important lotion to this mouse, but we will not dwell on it for a long time.

What it is? - these are just rubber flaps that can be glued to the numeric keypad of the mouse in order to better navigate them. It is quite convenient, because you immediately know which button is where. Razer offers several ways to apply them. But you can do as you like, even glue all the buttons. Conveniently, it is necessary, it helps to adapt, and the finger is soft on them.

Backlight




We can only enable or disable the highlighting of an element. It's simple. Sometimes it is necessary if someone is sleeping in the room, and the blue light interferes.
