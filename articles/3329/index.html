Intel captures web 2.0

Intel, a well-known manufacturer and trendsetter in the microprocessor industry, has created its Web 2.0 Development Kit. The Web 2.0 Technology Development Kit (TDK) is currently running on Windows XP and supports MSIE 6, MSIE 7 and Mozilla Firefox browsers. We were very surprised at the possibility of obtaining system information: about battery charging, network bandwidth, storage devices, location and even about the processor, including information about the processor cores. Considering that Intel itself suggests this TDK as an assistant for working with mobile devices (laptops and ultra mobile PC), it becomes clear the meaning of obtaining information about the platform and why this is one of the key features.
