Hotel module 3CX Hotel Phone System

In terms of communication, hotels have quite specific requirements that are not included in the standard set for corporate clients, mainly for various ‚Äúalarms‚Äù and a kind of management for telephones in guest rooms. 3CX has a separate solution for this - 3CX Hotel Module , today we will briefly review its main functions.



The hotel module 3CX Hotel Phone System performs the following functions:
  • Allows you to check in and check out guests.
  • Allows you to set the Do Not Disturb status for the guest number.
  • Allows you to block external calls to the guest number.
  • Allows you to schedule wake-up calls.




3CX Hotel Phone System integrates with many hotel management systems (Property Management Systems, PMS), such as: Micros Fidelio, roomMaster, Hilton, etc. With this solution, an employee at the reception can make the necessary telephone settings directly through the PMS interface or via the web. interface.

Opportunities


The hotel module 3CX Hotel Phone System consists of an external interface and a Windows service. Windows service runs in the background and provides integration with a hotel management system (PMS). With this architecture, most of the necessary functions are performed through the PMS application. Front Desk Phone Control Application is an easy-to-use Windows or web application that allows the front desk employee to perform the operations described below. This application can be used separately from the PMS system, and then it does not require integration at all. For small hotels this can be quite convenient.

Operations during guest registration


  • Assigning a name to an internal guest number
  • Unlocking the extension number, i.e. outgoing call permission
  • Delete all voicemails
  • Cancel Do Not Disturb settings

Operations at the guest's departure


  • Setting an empty name for the extension (you can see that the room is free)
  • Delete all voicemails
  • Block outgoing calls for extension
  • Clearing Do Not Disturb Settings


Do Not Disturb (Do NotDisturb, DND)


  • Setting the status to "None" (the same as in the interface)
  • All calls will be forwarded to voicemail.


Wake-up Calls


The system allows you to schedule call reminders. A call will be made to the internal number of the guest at the appointed time with a pre-recorded message - for example, a reminder of the request to wake up at the appointed time.



Billing


The system allows you to customize the format for reporting calls. This can be a CDR report for each call, or a text file with information about the call (one call in a text file), or a general text file with a list of all calls and information on them.

Integration options


The hotel module can be integrated with PMS systems in the following ways:

  • Through the intermediate application Lodging Link. Lodging link integrates with a variety of hotel systems and provides a transparent dock with the 3CX module for hotels. A list of supported hotel applications can be found here: www.comtrol.com/aboutus/partners/hospitality/pms
  • Through the "emulation" of the Mitel mode. In this case, the 3CX Hotel Module will emulate a Mitel PBX. The sales volume of Mitel PBX is quite large, so probably the PMS application already has an interface with Mitel.
  • Data exchange can be configured manually. In this case, the seller should provide the relevant documentation.


The hotel module 3CX Phone System comes in 2 versions, with the ability to integrate with and without PMS.
