Opera goes to the side of evil: Declared support for properties with the prefix -webkit-

Mobile Opera will soon receive support for the -webkit- prefix in CSS.

The changes will primarily affect Opera Mobile and can be seen in the experimental build of Opera Mobile Emulator.

Technical details will soon be discussed in a separate blog post on the Opera developers blog, but the developers noted that these are only aliases to the frequently used Webkit properties that Opera already supports. The prefixes will be affected by Opera Core, so this will soon be possible to see in both Desktop and Mobile builds.
