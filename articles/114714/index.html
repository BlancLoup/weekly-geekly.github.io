Stub for IE6 under Drupal 7

Since we no longer support IE6 by default in our projects, having made this support for customers paid - there was a need for our own stub.




The basis was taken HTML-version, which last winter was on Habr√©. We redraw the design and change the interface, with an emphasis on the fact that the stub does not prevent the user from seeing the site and he could continue to use it if he wanted to:

The module is written under Drupal 7:
arwm.ru/arwm_ie6bar.tar.gz

The JS code is minimized to speed up the stub loading. You can do localization - as an example in the archive there is a translation into English.
The code was written by vanya - you can get the version from it under Drupal 6 and the stand-alone version.
