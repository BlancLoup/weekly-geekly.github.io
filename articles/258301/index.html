50+ Best Bootstrap Add-ons



Thanks to the popularity of the CSS framework Bootstrap , a lot of different add-ons have been developed for it. Even now you can use Bootstrap for almost any task when developing and designing a website.

For the article I picked up the most useful additions "for all occasions".

Component Sets


Fuel ux




An excellent set of additional controls, which are often not enough when working with Bootstrap. It includes such modules as trees, patterns, datapicers, wizards, etc.

Website
Github

X-editable




I think this is an indispensable set of components for inline editing. It supports various versions of Bootstrap and works fine without it (on jQuery).

Website
Github

Jasny




A small set of additional items to Bootstrap. Included as a side menu, function buttons, input masks, etc.

Website
Github

Bootstrap Form Helper




Another interesting set of items. The premium version includes elements such as input for selecting currencies, countries, time zones, languages, etc.

Website
Github

Leapstrap




A set of elements to support the Leapstrap controller plugin. This is a rather unusual thing, so I include it in this list.

Website
Github

Individual components


jQuery Bootpag




An additional component, which is often lacking in the standard set of Bootstrap - Pagination. A special feature is the possibility of dynamic processing.

Website
Github

Tocify




Component to track page content. Very often this component is missing for creating modern websites.

Website
Github

Bootstrap Link Preview




Small and very easy to use component for previewing content by reference (like a Facebook preview).

Website
Github

Flippant.js




A small script to create "bilateral" blocks. Ideal for creating forms, portfolio cards, etc.

Website
Github

Bootstrap Tour




A great addition to Bootstrap, which allows you to create a step by step tour for new visitors to your site.

http://bootstraptour.com/
https://github.com/sorich87/bootstrap-tour

Bootstro.js




A variant of the previous tool. It is also designed to create a turn-based tour for your website or web application.

Website
Github

Yet Another MegaMenu (YAMM)




Component required by almost any modern site. Allows you to create the main navigation menu of almost any complexity.

Website
Github

Dropdown Menus Enhancement




Addition to the standard menu Bootstrap. Includes features such as menu positioning, display in the menu radio and checkbox-input and more.

Website
Github

Bootstrap Tree View




A simple solution to display a tree structure. This component supports standard Glyph-icons.

Github

GTreeTable




Another addition that allows you to create a tree structure. It features an abundance of functionality: Drag & Drop, editing, etc.

Website
Github

Bootstrap Star Rating




Component for easy rating creation, easy to customize. From experience I can say: very often you have to look for a similar component.

Website
Github

Gridmanager.js




Content editor based on the rows and columns of the Bootstrap framework. Not yet found how to put into practice. Laid out more, as an example of implementation.

Website
Github

Form Components


Bootstrap Tags




Component for easy tag creation. Very easy to use and customizable too. Works with all versions of Bootstrap.

Website
Github

Bootstrap Switch




Component to create neat switches. Suitable for any site design, besides all parameters are easily adjusted here: dimensions, animation, color, etc.

Website
Github

Bootstrap Maxlength




A good "limiter" for text input fields. Easy to use, sometimes just an indispensable addition.

Website
Github

Bootstrap Select




A very lightweight component that allows you to significantly expand the functionality of a regular list. Follow the link below to see all usage examples.

Website
Github

Chosen



A very simple script to significantly improve the combo box. Supports many options and variations for using this field.

Website
Github

Bootstrap Multiselect



Another option to expand the list box. This script handles events when opening / closing / selecting, etc. You never know what the client will ask ...

Website
Github

Bootstrap Validator




An indispensable component when working with forms in the Bootstrap framework. It saves a lot of time: in addition to checking, it contains ready-made design of form fields.

Website
Github

jqBootstrapValidation




Another component (option) for form validation on the Bootstrap framework. It contains a large number of options for creating validation of user input.

Website
Github

jQuery File Upload




A good component that allows you to significantly save time if you need to add a field to download files. Supports multiple server platforms.

Component for easy autocompletion creation. Very easy to use and lightweight (by weight) script. I think it is worth adding tools to the piggy bank.

Tag Manager




Ready script (component) to create tags. A lot of options and, accordingly, the possibilities of use.

Website
Github

Typeahead




Component for auto add-on with text box. Two scripts are used (for search and display), which can be used both together and separately.

Website
Github

Slider for Bootstrap




Another element that helps the user save a lot of time. Instead of entering data manually, simply drag the slider. Many options, fully styled.

Website

Tokenfield for bootstrap



Component for easy tag creation. Fully stylized, has many options. Supports both keyboard and clipboard input.

Website
Github

Label in place

Strength meter



Component to determine the complexity of the password entered. A simple solution for a common problem. Includes many options and ready-made design.

Website

Bootstrap File Input




Component for downloading files. Supports multi-boot and preview illustrations. Follow the link below to see examples of implementation.

Ladda UI for Bootstrap 3



Quite an interesting idea using the load indicators directly on the buttons.
