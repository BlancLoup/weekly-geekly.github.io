Briefly about the new: Samsung has developed the world's first 8-gigabit module of mobile DRAM-memory LPDDR4

Good afternoon, Habr!

Samsung Electronics has announced its success in developing the first industrial energy-efficient 8-gigabit mobile DRAM memory module complying with the LPDDR4 standard.

New memory is made on the basis of those. 20nm process class and carries one gigabyte per chip, which is the highest density for DRAM components on the market today.



Thanks to the new LVSTL (Low Voltage Swing Terminated Logic) I / O interface, the LPDDR4 chip provides data transfer speeds of 3200 Mbit / s per contact, which is twice as fast as previous generation LPDDR3 DRAM memory. In addition, the new interface LPDDR4 will provide 50% more performance than any samples of memory LPDDR3 or DDR3. With all this, the power consumption of the new memory has decreased by about 40% at a voltage of 1.1 volts.

Undoubtedly, Samsung will equip its premium models with a new generation of smartphones, tablets and ultra-thin laptops in 2014 with its new chip.
