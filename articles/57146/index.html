TV Control - TV program in the mobile

Our midlet (a program for the j2me platform) will allow you to have a fresh program on 220 channels in your mobile phone. There is a midlet for a long time, but only now, in our opinion, it has acquired a finished and comfortable look.



The plans (in the near future) is the implementation of support for the touchscreen, because many communicators have both a touchscreen and a pre-installed j2me emulator. And to finish the reminders, their midlet writes to the standard reminders in the phone (as long as it works on units).


Midlet features


  • You can download a complete list of available channels (with time on the server they can be added / removed) and select the channels you need
  • Download the TV program to all selected channels or to the current one, both with announcements and without them
  • see which TV shows are now going (as an option following it or 2 following)
  • compile a list of selected TV shows
  • TV search by word
