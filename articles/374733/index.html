Minecraft will be more secure for Windows users



Everyone knows that Minecraft requires a Java platform to launch, which is known for its vulnerabilities. However, this popular game will become more secure thanks to the new update.

This update for the Windows version of Minecraft installs a standalone Java version that only works during game play. Thus, you can remove the regular version of Java, without losing access to the game world.

Even if you have not installed the latest updates for your programs, the game will still work as usual. This update is not necessary for users, but it is worth considering if you use Java only to be able to play Minecraft.
