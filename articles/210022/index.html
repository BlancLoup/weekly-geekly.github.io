Server on ARM? Made in Russia!

All boards and all hardware except the processor are manufactured and designed in Russia!

Last fall, Rikor.IT addressed us with a proposal to test their servers. At first, we responded in a standard way - another super-wingman collector is trying to hit the price. But the more we talked with them, the more it was clear that this was an absolutely non-standard offer.
First, MicroCloud was offered. Secondly, ARM processors. Thirdly, they argued that all this is done in Arzamas in-house.





I must say that the server was easier than it seems and more than expected. The length is 800 mm exactly - but the small weight is due to the aviation aluminum from which the body is made.
A copy of us brought another engineering - sometimes there was manual work.

Attentive habrovchanin certainly notice the mini jack. This is not a sound - it is com-port. Included is a mini jack cable - usb. On a laptop, USB is defined as a COM port. A few words about ARM

Not everyone understands that ARM is not x86 and that it requires its own OS build, for example, ubuntu-13.10-server-armhf + omap4.img . Booting the system is in console mode and is controlled just through the very mini jakk (telnet). We tested the processors ourselves - in the next review we will post the test results.

Generally speaking, this ARM is very different from those that are in our smartphones - this is a server processor. It is worth noting that this is one of the first in the world of servers based on ARM. Yes - 32-bit processors, this is the main formal flaw - it slows down the release of Windows server on them. But by the end of the third quarter, marvel promises 64-bit processors for 8 cores (information from Rikorovtsy).

If someone wants to test such a server remotely or even take a chassis for himself to tests, write to LC.
