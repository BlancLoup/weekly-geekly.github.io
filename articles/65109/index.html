Expression Web 3 + SuperPreview Released

Finally, after the appearance of Blend 3 RC, Expression Web 3 Trial became available . All would be fine, but with Expression Web 3 comes the SuperPreview tool, which changes the way that markup is tested in browsers. It has already been written about this tool, but it was available only for Internet Explorer browsers. Now there is full support for Firefox (and the latest versions too).


In the screenshot you can see the work of the tool: habrahabr site for example. The markup is compared, Firefox is selected with the reference, and the IE8 rendering side shows the deviation of the selected markup element. You can compare with the downloaded source image, and not just the result of the browser.

Firefox and IE browsers are supported. Safari support will be added soon. Expression Web is a paid software trial for 60 days.
