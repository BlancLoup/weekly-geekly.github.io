Build 2013: the best reports on the Windows Azure platform



The largest conference of Microsoft Build 2013 was held , where a lot of announcements were made on the company's products, including new services of the Windows Azure cloud platform, including:

  • Commercial availability of cross-platform cloud backend for Windows Azure Mobile Services mobile applications and integration with Visual Studio 2013;
  • Commercial availability of the service and new features of Windows Azure Web Sites for creating scalable cloud websites;
  • the availability of a new service for automatically scaling web sites, cloud services and virtual machines;
  • availability of a new system notification service for developers and administrators about changes in the status of cloud services.

During several days of the conference, participants could visit dozens of technical reports. Now these reports are available for download and online viewing for everyone on the Channel9 site .
