Even more modern C ++

‚ÄúC ++ 11 feels like a new language.‚Äù - Bjarne Stroustrup

Not long ago, the Coat of Arms Sutter opened a new page on his website - Elements of Modern C ++ Style , where he describes the advantages of the new standard and how they will affect the code.

Standard C ++ 11 introduces many new features. Next, we consider those of them, thanks to which C ++ 11 can be considered a new language compared to C ++ 98, ie:
  • They change the style of your code, introducing new idioms. They will affect the architecture of the C ++ libraries. For example: smart pointers will become more common (both as function arguments, and as their return values), as well as functions that return large objects by value.
  • They will be used so often and intensively that you are likely to meet them in every second code example. For example: hardly any example of five or more lines of code will do without mentioning the auto keyword.

Feel free to use other features of C ++ 11. But first of all, pay attention to the ones listed below, since it is thanks to them that it is possible to write clean, safe and fast code in C ++ 11 ‚Äî as clean and safe as other modern languages.
