During the conference, reports were assessed by the audience, we suggest you also get acquainted with the top 10 reports that received the highest marks:

Reactive functional programming on F #: from analyzing twitter in real time to programming a Leap Motion sensor and mobile robots , Dmitry Soshnikov.
The report is devoted to the implementation of the concept of reactive programming in F #, which allows you to operate with ‚Äúlive‚Äù data streams with the same simplicity as you usually operate with data in memory. Starting with a short excursion into reactive programming, we then look at how this concept is used in examples to process twitter streams, three-dimensional data streams from Kinect and Leap Motion sensors, as well as to control mobile robots.

Developing secure Windows applications , Andrei Beshkov.
With each new version of the OS, Microsoft adds new security mechanisms such as DEP, SEHOP, ASLR. Attackers also do not stand still. They understand that it is becoming more and more difficult to attack the OS, so they focus their attacks on applications. You will learn how to create threat models and predict the next steps intruders, how to develop applications to reliably protect them and ensure the safety and security of your clients' privacy.

Programming of virtual reality applications: Windows Phone, two hydras and Oculus Rift from origami , Alexander Bragin.
The report will tell: How to collect virtual reality glasses for $ 10 based on Windows Phone. How to write the simplest virtual reality application on Unity. A little about Oculus Rift: How to wean from the outlet. How to connect to Unity. Features of avatar control in virtual reality, a brief overview of existing controllers. The basic principles of interface design for virtual reality applications. Demonstration and test drive of various virtual reality equipment for those who wish.

DirectX application development in Visual Studio 2013 , Dmitry Andreev.
If you use DirectX to create games or interactive applications for Windows and Windows Phone, it is very important to use efficient and convenient tools that help simplify your work. Visual Studio provides a rich set of such tools that allow you to create shaders, work with images and 3D models, diagnose DirectX applications. These tools were first introduced in Visual Studio 2012 and received additional features in Visual Studio 2013. This report will cover these tools and their capabilities.

Azure API Management: Managing and Publishing the Web API for Third-Party Developers and Partners , Maxim Kim.
API Management is a new service in Windows Azure that allows you to provide and publish your services and services as an API for external and internal use. The report will tell about new and interesting features of this service, based on the acquisition of Apiphany in 2013. In addition, in addition to the technical details, the report will cover the topic of providing API as a business.

Debugging and profiling Windows Store and Windows Phone applications using the latest Visual Studio tools , Maxim Goldin.
Come to this demonstration-packed report to learn first-hand how to use the latest tools in the Visual Studio 2013 diagnostic hub. You will learn how to make your XAML application faster, more responsive and efficient. How to make the most of the debugger features and the latest diagnostic tools for fixing errors. It will also show how using Visual Studio Diagnostic and Performance hub to study in detail the performance of your application, the level of energy consumption and to study how your application works with memory. The main approaches that will be demonstrated in the report can later be used to develop an application for both the Windows Store and Windows Phone Store.

Monetization and promotion of applications - our secrets for developers , Mikhail Chernomordikov.
You will learn the state and prospects of the market for applications on the Windows platform, learn the structure of the market - ‚Äúwho are your consultants.‚Äù Get acquainted with the programs and opportunities to promote your applications, learn about vivid examples of success, important trends, as well as the monetization mechanisms that the Microsoft Windows platform represents.
Under the hood: a scalable and high-loaded website of the Sochi 2014 Games on Microsoft Azure , Stanislav Kondratiev.
The Sochi 2014 Winter Olympic and Paralympic Games were undoubtedly the most important sporting event of 2014. The official website of the Games has become the most popular sports resource in the world during games. We will look behind the scenes and consider its distributed architecture, organically built on Microsoft Azure and able to cope with a sharply growing, unpredictable load. Come and learn about the interesting problems we have encountered, and how we designed our system in accordance with the high demands placed on this class of solutions.

How SQL Server stores data at the binary level , Mark Rasmussen.
Much of SQL Server works in a magical way. And you are right! But despite this magic, everything is actually understandable. This 500 level report will cover the internal storage format of SQL Server data - MDF files, how to access this data using no more than a hex editor. You will learn how SQL Server stores internal metadata about objects, how to search for data on disk, and after they are found, how to read them.

Deep immersion in the controls for Windows Phone 8.1 , Tim Hoer.
You will learn about the development of interfaces for the new mobile platform Windows Phone 8.1, the use of common controls for Windows and Windows Phone, new features and changes in the functionality of the controls you already know.
