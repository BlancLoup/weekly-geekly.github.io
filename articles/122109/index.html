HabraFav - access to selected

Personally, I often have the desire to find something in the habr in order to read / re-read.
Actually, this was the reason for the emergence of the idea to write an extension to chrome (in order of training), which will allow you to do this without leaving the cash register opening new tabs and not wasting time on a few reloads / redraws of the page.

In general, I do not like a long time to spread the thought on the tree, so meet - HabraFav .
From all hotelok implemented only:
  • Loading Favorites Tags
  • Search by tags
  • Download tag article headers

The extension does not require additional configuration and is ready to work immediately after installation. The configuration page is still used for testing purposes and does not configure anything.
What is planned to add:
  • New private message count
  • Quick access to private messages
  • Caching favorites for access without entering the Habr
  • Setting the default action (what to load when you click on the icon)
  • Manage favorites (for now only the deletion of records was invented)
  • + View favorites without tagging
  • + Last comments with links to articles

Waiting for suggestions for improvements.
The icon is scanned by a familiar designer. Everything seems to fit into the habr style, but if anyone has any ideas, I won't refuse.
