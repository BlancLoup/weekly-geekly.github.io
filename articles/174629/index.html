Chrome has been updated to version 26

In the version of Chrome 26 implemented many new features, as well as disabled some of the old ones, for security purposes or for improvement. Among the latest innovations was a significant improvement in the Spell Checker grammar checking module, which has been updated for all languages. Spell Checker's custom dictionary update is now synced between computers along with other information that Chrome synchronizes.



More information about the update is available under the cat.

In version 26 for Windows, there was an option to save labels on the desktop that correspond to different profiles. Several users who work on the same computer can now conveniently create each label corresponding to a specific set of settings, bookmarks, cache, surf history for their Google Chrome profile. A very handy feature for a family computer. Now you don‚Äôt need to shut down and log in with a different name, just launch another browser profile.



Chrome 26 for Mac and Linux is implemented asynchronous DNS resolver, which is already running in Chrome under Windows, speeding up the decryption of DNS records when requesting a URL.

Also, in Chrome 26, 11 vulnerabilities are closed, including two critical ones. Support for MathML has been removed from the browser for security purposes, and the support for the datetime data type has been removed.
