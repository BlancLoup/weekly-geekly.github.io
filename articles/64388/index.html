Java SDK 1.2.2, GWT 1.7 has been released

Full list of changes is available here .

- Finally, local dataviewer has appeared, it will be easier to write applications by an order of magnitude.
- The number of static files increased to 3000.
- Support for unindexed fields in ORM
- Fixed autogeneration of indexes and improved transaction support.
- Multiple minor fixes in the SDK and ORM.

For some reason, GWT 1.7 was released only in the google-plugin for Eclipse, the official site is still an old version. The main changes concern the support of Internet Explorer 8, Firefox 3.5 and Safari 4. Plus, some high-priority bugs have been fixed. In real life, the differences from 1.6.4 are small.

UPD : Version 1.7 posted on off.sayte, changes

Regarding the update of the plug-in for eclipse (compatible with 3.5), it is said that it will appear within 2 weeks.
