5 browser extensions for working with text in English



Hello! I continue to share my professional experience in the subject of work with English texts. Today we will talk about which browser extensions can be used to facilitate the creation of high-quality materials and learn to write better in English. Go!

Note : The text contains extensions for Google Chrome and Mozilla Firefox, because I only use these browsers. Not all mentioned products have a version for both browsers.

Linguix


This service is somewhat analogous to Grammarly - it also finds typos and grammatical errors in texts, but it costs almost two times less. There are also differences - for example, a database of content templates that can be used to more quickly write texts of various categories (for example, press releases or corporate news).



Of the features that like - the module for calculating statistics. The system analyzes the ‚Äúreadability‚Äù of the text and tells how complex it is. Useful to resist the temptation to make the text too bulky.



Another handy thing is the pastebin-editing mode in the web version, which works as a secret chat in the messenger. After writing and editing text, it is completely removed.

Writer


Mastkhev-application for all who work with texts in English using a browser. This extension allows you to open an extremely convenient, flexible and non-distracting editor in Chrome. It looks very cool:



The editor works Linguix - so you can immediately look for errors

From successful solutions - the extension automatically saves the text from time to time. You can set goals, easily get statistics on the number of characters in the text, etc. Despite the fact that the application has a separate page in the Chrome extension store, in fact it redirects to the web version of the service, and you can use it from any browser.

Browser version of Writer

Google dictionary


Dictionary from Google allows you to translate an unfamiliar word - it is useful when working with English-language materials to prepare the text. With this extension, you can effortlessly learn the meaning of any word that you came across on a web page. You just need to highlight the word, and a pop-up appears with the translation.



A handy thing - the program can say the words out loud, so you can pull up the pronunciation. It turns out that Google dictionary is useful not only when working directly on the text, but also as a language learning tool.

Chrome version

Dark reader


Let's be honest, many of us are constantly sitting at the computer until the night, including in total darkness. Some, for example I, can even work in such a format, including on texts.

Needless to say, most text editors are bright white pages with black letters on them. In the dark, this is an extremely serious load on the vision, so I use the DarkReader extension. It darkens the background on web pages, so it‚Äôs much more pleasant to work with them in the evening. For example, here‚Äôs how the Medium blog platform looks like with this extension:

Stayfocusd


Procrastination is one of the main enemies of all who have ever tried to work on texts. Write a particularly long text without being distracted by social networks, mail and news, is an extremely difficult task. The StayFocusd extension helps me solve it.



With it, you can prevent yourself from leaving a certain site for a while. In the end, if you need to write a blog post, then until you write, you will not go to facebook. With the beginning of the use of this extension, my productivity has increased markedly. Recommend.
