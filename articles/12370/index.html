Microsoft SharedView public beta released

Microsoft unveiled a beta version of the SharedView service, which allows employees to organize collaborative workspaces, edit and view documents using a local network or the Internet. Starting a new session is very simple - you need to select Start New Session and enter your Windows Live ID. Up to 15 people can join the session, and you can invite them to participate via email or Windows Live Messenger.
