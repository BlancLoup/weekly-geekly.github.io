Seafile updated to version 1.4

The Seafile project announced the release of version 1.4 of the server and clients to organize a Dropbox-like service on their servers. The new version contains the following improvements:

  • Generating a link to the directory to which you want to give access
  • The web interface now shows the modification time and there is a possibility of sorting by this parameter.
  • The library owner has the ability to set the size of the change storage history.
  • Ability to mark files on the directory page
  • Mobile client support
  • Optimized configuration options, improved https support

Clients have improved support for files larger than 4GB and containing European characters in the name. A more detailed list of improvements can be found here .

Also announced the release of the first stable version of the client for the Android system. The client requires a server version 1.4 or higher.

More details about Seafile and its capabilities can be found in my previous post .
