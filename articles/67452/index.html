Wikipedia Mobile for iPhone: useless client

The Wikimedia Foundation has released the first official application for the Apple iPhone. This is the Wikipedia Mobile program, a mobile client for viewing the encyclopedia. Distributed for free through the App Store ( iTunes link ).

True, the program is rather poor . It is not clear why install it, if it is just a window for accessing the mobile version of Wikipedia , which can be viewed with a regular browser. The only additional functionality of Wikipedia Mobile is saving the history of search queries on the site.



This is despite the fact that the App Store now has several programs for really convenient work with Wikipedia content. These are third-party programs: Wapedia ( iTunes link ), Wikiamo ( iTunes link ) and Wikipanion ( iTunes link ), and about 100 other programs. There is, for example, viewing the table of contents of articles, full-text search on the page, saving notes for later reading, etc.
