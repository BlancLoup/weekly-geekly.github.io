Version 0.9.4 Pineapple is released - the first Linux accounting platform.
The main changes in version 0.9.4 compared with the previous release:

* Changed requirements for the OpenOffice.org package used. Now OpenOffice version 2.0 or higher is required.
* Updated printed form of the invoice. The new form of the Invoice was approved by Decree of the Government of the Russian Federation of December 2, 2000 N 914 (as amended by the Government of the Russian Federation of May 11, 2006 N 283) The invoice is formed in the spreadsheet format * .ods
* Now you can select balances for several sections of the cumulative register. For example, through a serial call to SetFilter (dimName, dimValue) one or more times and call GetSaldo (date).
* In the Designer, it became possible to save to the file / download from the file individual objects of the business scheme, without data.
* Multi-currency support (RUR, USD, EUR) when converting a sum to a record using the MoneyToText () function
