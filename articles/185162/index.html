Try Windows Server 2012 R2, SQL Server 2014 and Visual Studio 2013 without installation!

In this article we will talk about the availability of new products Microsoft Windows Server 2012 R2, SQL Server 2014 and Visual Studio 2013 Ultimate in the Windows Azure Virtual Machines Gallery, where anyone can try new items without installing on their computer.



Windows Server 2012 R2


At the conference TechEd Europe 2013 , a new version of the Windows Server 2012 R2 server operating system was introduced. A preliminary version of this system is available today for download and testing.

The new revision of the operating system offers hundreds of innovations, some of them are presented on the slide. A huge number of innovations are presented in the updated Hyper-V hypervisor. There is also a new version of IIS 8.5 web server, innovations in which are aimed at increasing the manageability and scalability of web servers, including those for hosting.

In more detail about the new server you can find out on the specified links:

SQL Server 2014


Along with the release of Windows Server 2012 R2 at TechEd Europe 2013, a preliminary version of the data management system SQL Server 2014 was released.

The new SQL Server 2014 presents dozens of major innovations aimed at improving productivity, providing convenient work with Big Data and business intelligence, building hybrid cloud systems.

A key innovation in SQL Server 2014 is the in-memory technology set: the Hekaton project, which allows you to place tables in memory, updating ColumnStore mechanisms and business analytics with PowerPivot.

Visual Studio 2013


At Build 2013, the largest Microsoft conference for developers, a preliminary version of the application development environment was released - Visual Studio 2013.

As in other company updates, Visual Studio 2013 presents dozens of innovations in each of the components of the development environment. In particular, the .NET and ASP.NET development environments have been updated, the Windows 8 developer tools have been significantly updated with support for the new version of Windows 8.1. Many innovations have been received by code editors, testing tools, profiling and debugging code.

Try new Microsoft products using Windows Azure


The good news is that all of these new products, new features in the products can be tried personally without even installing them on your computer.

Today, Windows Server 2012 R2, SQL Server 2014, and Visual Studio 2013 Preview are available as virtual machine images in the Windows Azure Gallery.

Simply use your Windows Azure subscription or sign up for a free trial period and run new servers in Windows Azure virtual machines.



To do this, start creating a new virtual machine and select the server you need in the gallery: Windows Server 2012 R2, SQL Server 2014 or Visual Studio 2013.



Specify the virtual machine configuration, enter the administrator name and password.



Configure the virtual machine mode by specifying DNS and geographical location (data center) of the virtual machine.



The virtual machine will start to be created. Wait for the creation, it may take a couple of minutes.



After creation, you can go to the administration panel of the virtual machine and use the "Connect" function to connect to the virtual machine via RDP.



Having connected to the virtual machine you can use, study and test already installed Microsoft products, for example, Visual Studio 2013 Preview.
