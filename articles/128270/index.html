Microsoft has implemented an analogue of Synergy (control of several PCs with one mouse)

As part of the Microsoft Garage initiative, the Mouse without Borders (MWB) utility was developed, which allows you to control four computers with a single mouse and keyboard. The concept resembles the well-known open-source program Synergy , which has already been discussed at Habr√©.

Unlike Synergy, Microsoft development is easier to install and configure, and also has a number of additional features: for example, dragging files with a mouse from one desktop to another, simultaneous authorization on all computers or PrtScreen from any screen. The disadvantage compared to Synergy is obvious: MWB works only on one Windows platform, while Synergy supports Linux and Mac.

It should be added that Microsoft Garage is a kind of copy of the well-known Google initiative, within which employees can spend 20% of their time developing their own projects. Similarly, the program Mouse without Borders was created by Microsoft employees in their free time 1 and primarily for internal corporate use - almost all Microsoft employees in the office have more than one computer, at least a desktop and a laptop.

This program has been used by Microsoft employees for quite some time (the previous version was called Magic Mouse), but now it has been decided to open it to the general public.
