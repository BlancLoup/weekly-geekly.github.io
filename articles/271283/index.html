Trash in bookmarks and disabling side panels in the assembly Vivaldi 1.0.330.3

Hello!

We continue to delight you with new test builds. And in today's you will find some interesting new products, and also you will see corrected some number of errors that you have reported to us. And we have upgraded to the core of Chromium 47. Now everything in order.

Basket for bookmarks and notes

This function has long been familiar to many users, so we decided to add it - let it be. Actually, what else to say: a basket - it is a basket. Place for unnecessary things. So far, of course, its functionality is minimalist, but we'll think about it. And you can help us with your suggestions.

Turning off the side panels

Recently you can place your own web panels on the sidebar, and now we have added the ability to hide unused panels. To do this, use the context menu:


Also, if we are talking about web panels, you asked for access to the address of the web panel - we did. You can find the corresponding item in the context menu of the web panel.

Known Issues

  • When you turn on the bookmarks bar, it is empty. How to fix: open the bookmarks tab and specify the folder to be used in the panel.

Important note

As we already reported earlier, the beta version and snapshots are different branches, so they should be set separately. This is how Linux works. But, unfortunately, in Windows and Mac, the test build updates the beta version during manual installation. We are preparing a solution to this problem and apologize for the inconvenience. Auto-update test builds occur correctly, without affecting the beta version.
