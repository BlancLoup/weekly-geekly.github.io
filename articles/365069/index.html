Linknotize - desktop-online game for finding connections between events.



Linknotize is an unusual board game that includes both physical components and the use of the Internet. This is a game with maps, the purpose of which is to build a chain of links through Wikipedia links between two unrelated topics.

A player with the help of dice randomly receives two cards with themes. After that, within three minutes, he should, by opening the first topic, follow the links from Wikipedia to the second one. Points are awarded for the task execution time and for the number of connections - the smaller, the better.

The game reminds of the ‚Äúsix handshakes‚Äù theory, according to which any person can make a chain of acquaintances to any other person. Moreover, some argue that even two any topics can be linked to each other in no more than six intermediaries.

For example, how to tie Tom Brady (an American football player) and Bill Gates? Tom Brady -> NFL -> Seattle Seahawks -> Paul Allen -> Bill Gates. Another example: Abraham Lincoln -> Speech at Cooper Union College -> New York -> Broadway Theater.

Will you be able to do this quickly and using only the links from Wikipedia articles?
