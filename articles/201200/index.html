New . Getting started developing in HTML5 using JavaScript and CSS3
This course is a brief introduction to HTML5, CSS3 and JavaScript that will help students learn the basics of programming on HTML5 / CSS3 / JavaScript. The focus is on implementing software logic using HTML5 / CSS3 / JavaScript, defining and using variables, organizing cycles and branches, developing user interfaces, retrieving and validating user-entered data, saving them, and creating well-structured applications.

New . Getting started creating web applications with ASP.NET
This course is designed for experienced application developers who are going to develop applications for modern browsers using ASP.NET and Visual Studio 2012 . Three Microsoft lecturers with a wealth of experience working with ASP.NET talk in detail about developing web applications using ASP.NET 4.5 and ASP.NET MVC 4, which allow you to create websites that are conveniently viewed from mobile devices, social web applications and much another. This is a course for students with a level of knowledge from intermediate to advanced. It will help you prepare for the Microsoft 70-486 exam.

New . Getting Started with ALM in Visual Studio 2012
The target audience for this course is application developers interested in applying the ALM (Application Lifecycle Management Application Lifecycle Management) approach to improving development processes in Visual Studio 2012. The course is designed for experienced developers and meets the criteria for exam 70-498 for MCSD: ALM .

New . Advanced features of System Center 2012 R2
This course tells about additional, advanced features of System Center 2012 R2 in the field of building and managing infrastructure for cloud environments. We will consider the issues of reliability and resiliency of the components of System Center 2012 R2 to build a continuous model of System Center services.

New . Hyper-V in Windows Server 2012 R2
The course is a technical overview of the new features of Hyper-V in Windows Server 2012 R2. Discusses issues of scaling virtualization solutions, ensuring high availability of virtual machines and applications inside them, technologies for building modern data centers, ways to increase the flexibility of IT infrastructure. It is assumed that the audience is well acquainted with the capabilities of Hyper-V in Windows Server 2012
