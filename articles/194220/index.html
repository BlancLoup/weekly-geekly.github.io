Huawei W1 - budget smartphone on Windows Phone 8

The smartphone has only 4GB of permanent memory installed. And this is not enough. Working with microSD memory cards in WP8 is organized in a peculiar way. When you first start with a new memory card, the system will offer you to use it as a storage for photos, videos and music.


A feature of working with the card is the inability to install applications on it. Those. in fact, only about 1.77GB of memory is available to the user and nothing can be done about it. You can forget about many installed games.
By the way, applications and games can be installed not only from the market, but also from the SD card. To do this, copy the installation xap files to the SD card, launch the market and select the sd card as the installation source:

After that, the application will install and try to authenticate in the market.
Another unpleasant feature of WP8 is temporary files that are created in the "Other" folder. In the last WP8 update for Huawei W1, a memory management utility was added that can clear the cache and remove installation packages. But the size of the folder "Other" is not affected.

‚ñ∫ Software

If the theater begins with a hanger, then familiarity with the new phone usually begins with a firmware update. In our case, this is most likely a necessary measure, as there is no Russian language in the firmware.
The Huawei W1 firmware is a bit more complicated than the firmware on devices on Mediatek chips.
To install the firmware with the Russian language, we need:
1. Install the driver
2. Open the cover of the smartphone, remove the SIM card and memory card. Close the small round contact with the memory card slot on the metal cover of the SIM card slot.

and without releasing to press the power button. If everything is done correctly, then the smartphone should have a black screen. Now we connect the phone to the computer (contacts can be released). A device should appear in the device manager
3. Next install the QPST program. All programs must then be run as administrator. After installation, open the QPST Configuration, the phone should be automatically detected, minimize (do not close!) The program and open the eMMC Software Download program. Uncheck Program MMC Device. Then download the bootloader files and unzip them. In the eMMC Software Download program, select Load Build Contents and there specify the path to the contents.xml loader file and click Download. After that, the device manager should have the following device:

4. Next, remove the tick from the Program Boot Loaders and tick the Program MMC Device, select the Qualcomm MMC with the mouse and click Download. After that, Windows will offer to format the partition, click "Cancel". After that, disconnect the phone from the computer, pull out and re-insert the battery and press the power button. If you did everything right, then iWare uniBIOS FFU Application should appear on your phone.

, reconnect it to the computer and check the device manager, there should be the following device:
Qualcomm programs can be closed.
5. Download the flasher and firmware (Russian firmware GDR2). Open the flasher, enter the password ‚ÄúHuawei‚Äù, specify the path to the downloaded firmware and the configuration file WindowsPhoneDefaultsCfg.xml (located in the folder with the flasher), click Next and Scan && Download. The firmware process will begin.

and after that you can pull out the battery, insert the SIM card, SD card and turn on the phone
