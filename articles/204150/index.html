DotNetFiddle - JsFiddle for .Net developers

I am glad to present you our new product - free online sandbox for .Net. You can use it to test different ideas, test the code, and send pieces of code to developers around the world.

http://dotnetfiddle.net

How did the idea

Did you want to check some code, but after an hour of searching for something like StackOverflow, you decide to just forget about it. A lot of unnecessary actions ...

  1. Launch Visual Studio.
  2. Create a console application.
  3. Add the necessary reference.
  4. Write code.
  5. Run it.
  6. If there is a compilation error, then go back to step 4.
  7. If all is well, rejoice.


Solution - DotNetFiddle

DotNetFiddle leaves only one step:

  1. Write code


Feature List and Screenshots


With DotNetFiddle, you can use IntelliSense, send your code to other colleagues, and work with your colleagues online on this code. Pair programming in all its beauty.

C # and VB.Net

You can use C # and VB.Net.
Immediately upon loading the page, you will see HelloWorld in the selected language. But you can complicate it as you please. You can even write ‚ÄúHello Universe‚Äù.


Intellisence

Use Intellisense for properties, methods, class fields as soon as you write the magic character ".".


Live validation

Code validation is similar to Resharper in the process of writing the code itself.


Share

You can send the code in the following ways:
  • Copy to clipboard.
  • Send by letter.
  • Post to Twitter.
  • Post to Facebook.



Teamwork

You can use pair programming as it could be. From 5-10 developers at the same time. Or you can write code with your best friend, as it once was.


Mobile support

There is also support for iPhone and Android devices. The mobile version of DotNetFiddle is divided into 4 different parts with large buttons.
