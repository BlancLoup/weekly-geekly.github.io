NBStudio


Working in such popular Java IDEs as NetBeans, Eclipse, Intelliji IDEA, there is a seductive idea to use some of their functionality when developing code in Cach√©. Therefore, I decided to try to create an alternative to the studio, which includes the most popular features of modern IDEs, which are not available in Cach√© Studio. The first goal of the project is to repeat the functionality of the original studio.
Today implemented:

In the plans:
  • Work with projects: open, create, manage project content. (Currently in development).
  • Creating new classes, routines, etc. (Currently in development).

Sublimecache


This is a Brandon Horst project, posted on a githaba here . As the name implies, this is a SublimeText plugin for development with Cach√©. It requires an installed server part in Cach√© 2014.1 or higher, which can be found here . The server part is a REST application that accesses the necessary functions.
Currently available features:
  • Connection to several areas on different instances.
  • Loading and unloading code classes and routines.
  • The ability to open a class in the browser, for (CSP / ZEN / SOAP).
  • Import / Export in XML format.
  • Run a SQL query and see the result directly in Sublime.
