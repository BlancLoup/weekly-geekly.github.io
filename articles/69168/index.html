Startup Blog Rating System

I will try not to spread the thought on the tree.

The blogosphere now occupies a very important place in www. There are blog ratings in search engines and directories. But like any generalized rating, which is one of the many products of these services, it is rather primitive and, as a rule, reflects quite a few statistics, such as attendance.

I propose to discuss such a startup.

Create a separate project that will collect statistics in the blogosphere. And in general, who would specialize only in blogs. I go to the site and at my disposal a whole set of information in different perspectives and sections on the topic of blogging. What would I like as a user to see in such a startup?

1. Separation of blogs by subject. Each advertised blog, in principle, has a certain basic theme. Marketing, IT, advertising, literature, etc.

2. The possibility of ranking blogs on a geographical and linguistic basis. Not only Russian-speaking, but I may be interested in foreign blogs on a specific topic.

3. Ratings and reviews. That is, in addition to the banal attendance, you can organize a certain stat. a report, a compilation of several indicators that would objectively or objectively show the popularity of the resource.

4. Contact information in a separate line of bloggers, so that there is no need to search for contacts in each blog separately, if I need, for example, to communicate with hundreds of bloggers.

5. Yes, about the reviews. There is a WM Advisor system (it seems so it is called), which makes it possible to hang a rating for any site. Let the same opportunity be for blogs. Download the plugin and put the blog + or - and add a review. It's comfortable.

6. The ability to search for the desired blog by criteria: language, attendance, number of reviews, topics, etc.
