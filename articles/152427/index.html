Google Docs stops exporting to ‚Äúoutdated‚Äù .doc, .xls, .ppt formats



From October 1, 2012, Google Docs users will lose the ability to export documents to old Microsoft Office 1997-2003 formats (.doc, .xls, .ppt).

The Google Help section lists the remaining formats for export, quite enough, including the new Microsoft Office formats (.docx, .xlsx and .pptx), ODT and RTF for text, ODS and CSV for tables, SVG / PNG / JPEG for presentations, as well as PDF, TXT and HTML for any documents.

Users of Microsoft Office 1997-2003 formats (.doc, .xls, .ppt) are recommended to install the Compatibility Pack , with which you can open new formats (.docx, .xlsx and .pptx) in old versions of MS Office. Alternatively, you can use LibreOffice to convert.
