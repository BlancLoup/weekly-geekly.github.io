Creating epub books on Mac with Pages

Now you can create an e-book in ePub format directly on your Mac using the Pages application from the iWork office suite.

It became possible thanks to the fourth update for iWork package released today , number 9.0.4 , in which Apple included the function of exporting documents to the ePub format.



After installing the update via Software Update, launch Pages and open any of your documents from which you want to create an e-book in ePub format. Then just use the ‚ÄúFile‚Äù menu and select ‚ÄúExport‚Äù. In the window that opens, go to the ePub tab, fill in the required fields (Title, Author, Genre), specify whether to use the first page of the document as the cover when exporting, and save the finished ePub document.



Epub is the standard for iBooks in the iTunes Store and the preferred format for e-books on iPad, iPhone, and many other digital readers.
