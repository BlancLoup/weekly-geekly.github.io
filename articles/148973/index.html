Russian developers support program

The past year since the launch of Windows Phone in our country, Russian developers have created interesting and useful applications for users all over the world.
On August 1, we prepared the final version of the new Windows 8 RTM operating system, inside of which there is a single Windows Store application store.
Obviously, now is the right time for the authors of the best applications on Windows Phone to become the first developers in the global Windows Store, and we are ready to offer our help and support.

During August, we will launch a special Russian Developer Support Program. on windows phone.

Program conditions:
  • The program is valid from August 3 to August 31
  • The program is valid for authors of applications that have a rating of at least 4 out of 5 stars in the Russian or American Windows Phone Marketplace

What do we offer:
  • Individual Windows 8 application design consulting
  • Individual Windows 8 application development consulting
  • Help in preparing the application for publication in the Windows Store

If you want to participate in the program, please write to the address bestapps \ @ \ microsoft \ .com \ with a link to your application in the Windows Phone Marketplace.

Windows 8 and Windows Store offer a unique opportunity to access the world's largest ecosystem of Windows users. We very much hope that the best Russian developers will be able to become successful in this new technological race.
