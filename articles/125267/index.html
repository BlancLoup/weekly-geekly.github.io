Unlocked bootloader for Sensation and Evo 3d



On Friday night unrevoked and AlphaRev released S-Off (loader without locks) on Sensation, Evo 3D and Flyer.
This means that now you can install custom firmware on these devices and change the contents of any system partitions in any way.
At the moment, this solution is in beta test, you need to get the key on the download page. The key is based on the serial number of the device. As the author writes, this was done in order to be able to stop the program in case of mass failures.
On my HTC Sensation from MTS, everything worked perfectly.


1. Downloading home archive for your OS
2. Downloading driver for FastBoot
3. Install the driver. HTC Sync if worth should be turned off.
4. On the smartphone, turn on USB debugging mode
5. We connect smart to Big Brother
6. Run the program from the archive downloaded in the first step.
7. On the download page from the first step, generate a key using the data that appeared in the program window.
8. The resulting key is inserted into the program.
9. After several restarts of the unit, we are pleased with the received S-Off and do not forget to press "Y" to install the recovery.

This solution works on:
HTC Desire (bravo) 0.93.0001
HTC Desire CDMA (bravoc) 1.06.0000
HTC Wildfire (buzz) 1.01.0001
HTC Aria (liberty) 1.02.0000
HTC Incredible S (vivo) 1.09.0000 and 1.13.0000
HTC Droid Incredible 2 (vivow) 0.97.0000
HTC Desire S (saga) 0.98.0000 and 0.98.0002
HTC View (express) 1.09.0000 and 1.13.0000
HTC Flyer (flyer) 1.10.0000
HTC Sensation (pyramid) 1.17.0006, .0008, .0011 and .0012
HTC Evo 3D (shooter) 1.30.0000 and 1.40.0000

To get root rights, you need to flash SuperUser through the recovery .
Personally, I now look forward to the emergence of CyanogenMod 7 for Sensation. =)
