Microsoft introduced a new environment for web development Volta


The Microsoft Live Labs research division today unveiled its new development, the Volta software environment for creating distributed applications, as well as software running in a network environment.

The main highlight of the Volta environment is that with it you can create client-server applications in one project and one development cycle.

‚ÄúUnder normal conditions, programmers who create applications running in a client-server environment have to make decisions about which functions to transfer to the client application, and which ones to leave on the server. Often this has to be solved in the early stages of development, and if errors occur later, then many parts of the applications have to be rewritten. With Microsoft Volta, it will be possible to decide which functions to distribute between the client and the server much later, ‚Äùsaid Alex Delay, manager of Microsoft Live Labs.

In the future, Volta will become additional software for the Visual Studio 2008 environment. Volta has been written on Microsoft Intermediate Language (MSIL), which means that people familiar with Visual Studio languages, such as Visual Basic or C #, can work with Volta without any problems.

‚ÄúVolta allows you to write client-server code without separation, and only then use the annotations to determine which part will be executed,‚Äù explained DeLay.
