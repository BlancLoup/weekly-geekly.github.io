OllyDbg 2.01


The release OllyDbg 2.01 passed unnoticed and was not lit on Habr√©. Together with version 2, the author released the disassembler under the GPL v3 license. In late October, future support for x64 was announced.

Declared list of changes in the final version:

  • Help on 77 pages.
  • Support for language files for the interface.
  • Support for AVX instructions (AVX2 is missing and the upper 16 bytes of the YMM registers are not displayed)
  • Call stack window (as in version 1.10)
  • Handle window (as in version 1.10)
  • Mapping SEH VEH handlers. To decode VEH addresses of handlers, OllyDbg modifies NTDLL.RtlAddVectoredExceptionHandler (), so the process must be started from OllyDbg.
  • Multibyte encoding support in dump window.
  • .udl library, replacing objects from version 1.10.
  • Search for integers and floating numbers in the dump window.
  • Search for procedures (entry points)
  • Limited support for NTFS streams.
  • Dump the disk.
  • Breakpoints that use INT1, HLT, CLI, STI, or INSB instead of INT3.
  • Multiple views in one line, repeat support.
  • Dump arrays of structures.
  • Micro analyzers.
  • Accelerated search.
  • Assembly of declared data (DB xx, etc.)
  • Highlight in ‚Äúrun trace‚Äù.
  • Up to 2 ordinals per address.
  • Limited support for Win95 via Microsoft Layer for Unicode.
  • Analysis of more complex code sequences.
  • Show free memory.
  • Many fixes.


In the archive with OllyDbg 2 there is also a test application for displaying new features, demonstration of bug fixes for version 1.


The list of plug-ins for the second version which should be noted:

  • OD2-ExPlug - Adds many useful things that simplify the use of OllyDbg.
  • LuaJIT - Lua scripting in OllyDbg.
  • OllyDbg2-python - Python scripting in OllyDbg.
  • OllyExt - Anti-AntiDebug.
  • OllyDumpEx - Excellent process dumper.
  • OllyMigrate - Migration between debuggers.
  • Swordfish - Quickly set breakpoints through the menu.
  • RenameOD - Allows you to rename ollydbg and reigns plugins in order for them to work.
  • Multiline Ultimate Assembler - Convenient internal assembler to insert / copy code.
  • CleanUpEx - Cleaning the OllyDbg directory.
  • uberstealth - Anti-AntiDebug based on IdaStealth code.
