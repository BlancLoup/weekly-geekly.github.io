Yandex.Maps client update for iPhone and iPad

The update of the Yandex.Maps application for iOS has been released.
The version received the number 4.2 and is now available in the appstore .
From the worthwhile innovations worth mentioning:
1. Night mode - hello motorists, it is convenient to adjust the sunset time when it is worth reducing the display brightness.
2. A reworked mechanism for updating map fragments. Now only changed parts of the map are updated (traffic is saved).
3 Ability to save locations directly from the application and send it via sms, email or just copy.
4 Tag function. You can save the label on an interesting object and builds a route to it without tedious moving the flags.

Specially made resize tiles for the iPad, now instead of 64 fragments only 8 are needed, which greatly speeds up the loading of the map.

In general, lovers of bitten apple is recommended for download.

PS When this functionality will get a version for Android I do not know, but I think that soon.
UPD. For Android, there is a beta version with similar functionality. EternityMan , thanks for the post .
