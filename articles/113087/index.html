Combo avatars VKontakte

I have already published this post, but for unknown reasons , an hour later he suddenly moved to the drafts. Nevertheless, I will make a second attempt. If the post is moved to draft moderators, then please write the reason so that next time I will not repeat the same error.
Post about creating combo avatars in my application.


Most recently, a post about VkOMG was a service that uses new VKontakte features to create unusual avatars. Almost simultaneously with VkOMG, I launched my own VKontakte application , which used the same feature, but without the ability to create a combo avatar. The problem was that the pages of users can vary greatly and create some kind of universal stencil that would be suitable for everyone. It was necessary to customize the stencil for each specific page. A little thought, I solved this problem by loading a screenshot of the page into the application and fitting the stencil according to the screenshot.

After two days of work, the function of creating a combo avatar is completely ready and anyone can use it. Download the cut images in contact occurs automatically.

In the process of creating, I had to properly understand such things as Mask, Bitmap, Bitmap Data, ByteArray, Drag and Drop, loading data through Post requests and some others. Therefore, if someone has questions on Flex or API vkontakte - ask in the comments, I will try to answer.
