Twitter finally added search

Despite the simplicity of implementation, Twitter still lacked the ability to search for people. This shortcoming forced users to resort to the use of search services, and as a result, to leave the Twitter site (although I went to the Twitter site once to register, it is much easier to communicate using desktop applications).

Now the Twitter.com home page has a search form by name or other user information (location, biography, link to the site). For inexperienced users who really do not know how to use search services (believe me, there are such) - this is a great tool for finding new friends in this service.
