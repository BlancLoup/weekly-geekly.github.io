Introducing 3CX WebMeeting 7.1 Beta - Next Generation Video Conferencing

The 3CX team is pleased to present a novelty: 3CX WebMeeting 7.1, which uses improvements in WebRTC technology for high-resolution presentations and screen sharing. All this can significantly improve the efficiency of online meetings. To enhance the quality of online meetings, PowerPoint-HTML5 converter has been added to 3CX WebMeeting, which guarantees a clear display of your presentations for maximum effect. 3CX WebMeeting 7.1 also improved bandwidth usage and added an indicator that showed participants Internet connection problems that could affect the quality of an online meeting.
The 3CX WebMeeting 7.1 solution is available for free to 3CX Phone System 12.5 users (up to 10 participants) or you can download the local version - 3CX WebMeeting Server.

And now let's talk more about the key new features and improvements:

New features and improvements


  • The number of simultaneous participants in each 3CX hosted portal has been increased from 3 to 10.
  • Video bandwidth and screen sharing increased to 256 Kbps.
  • Added a connection quality indicator that informs the user about the current network usage.
  • The ability to pre-load presentations in HTML5 format to ensure a clear quality of the show, even using narrow Internet channels.
  • Import attendees from 3CXPhone clients for faster meeting organization.
  • Bandwidth management for high quality video.
