Reducing the podcast bit rate online

Regularly I see in the comments to the podcasts statements that the podcast is supposedly recorded at too high a bitrate and weighs a lot. It is inconvenient to swing on a slow connection, or expensive with paid traffic.
The authors of podcasts often do not want to reduce the bitrate, because this will degrade the sound quality they have been working on for so long.
I agree with the authors, if there is at least some opportunity to download and listen in the original quality, do it. But, if there is no possibility at all, online services can be called upon to help.
I want to tell you about the solution to the above problem.

For this we will use the site media-convert.com .
I took a podcast with a duration of 19:23, which was uploaded in mp3 format with a bitrate of 128kbps, while it weighed 17.8 mb.
We go to the site media-convert.com , set the URL and insert the path to the podcast mp3-file in the appropriate field.
In the ‚ÄúInput format‚Äù field, select ‚ÄúMPEG-1/2 Audio Layer 3 (.mp3)‚Äù. In the field ‚ÄúOutput format‚Äù the same appeared, if you need a different format, select the desired one.
Next, put the checkbox "I accept the terms" and click "Convert".
The site begins to load and process the file. This may take some time.
When the file is loaded, the site offers to set the bit rate and compression method. The minimum possible bitrate is 32kbps, and we set it up. I personally do not see any particular difference between CBR and VBR, therefore I leave CBR. Click OK.
Processing begins, which again takes some time.
Voila, we get at the output a file weighing 4.4 MB, click ‚ÄúDownload‚Äù and download it to your computer.
