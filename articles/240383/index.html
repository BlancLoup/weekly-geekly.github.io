Firefox 33 release

Firefox for desktops has become available for download. The main innovations: (and again) optimization of memory consumption, Off Main Thread for Windows, integration of OpenH264, various improvements in the search of the address bar, increased stability of session recovery, etc. List of all changes in Firefox 33 under the cut.


  • The mechanisms for storing JavaScript strings and processing string data were optimized , which reduced the memory consumption and increased the performance of string operations:



  • Added support for the H.264 video codec (in the sandbox) due to the use of the OpenH264 library open by Cisco ;
  • For Windows, Off Main Thread Compositing (OMTC) is enabled by default;
  • Improvements were made in the search operation in the address bar (for example, search tips were added on the about: home and about: newtab pages);
  • Improved session recovery stability ;
  • Added new backend for Content Security Policy;
  • Support connection to HTTP proxy using HTTPS;
  • Removed proprietary properties and window.crypto features;
  • The support of the debugging service JavaScript Debugger Service has been discontinued in favor of the Debugger API (the article on MDN is eng , rus );
  • Implemented DOMMatrix interface;
  • Implemented @ counter-style rule from the CSS3 Counter Styles specification;

Firefox Developer Tools
  • Added editor of cubic Bezier curves (video) ;
  • You can view DOM events in the Inspector;
  • A new sidebar is implemented, displaying a list of shortcuts for each @ media rule in the current style sheet;
  • Added the ability ( Paint Flashing ) to visually evaluate the elements being redrawn;
  • In the "Rules" section in the inspection mode, the ability to edit @keyframes rules has been added;
  • And also in the "Inspector" is now displayed highlighting CSS-transformations ;
