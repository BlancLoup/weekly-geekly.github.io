MS KB3004394 update causes virtualbox to stop working in virtualbox

After installing the latest set of updates from Microsoft, it was discovered that virtualbox had stopped running existing virtual machines. An attempt to start a newly created virtual machine ended with the same error.
It turned out that the update KB3004394 (December 2014 update for Windows Root Certificate Program in Windows) is to blame for the recommended ones. The only option to restore virtualbox to date is to remove the specified update. Other examples of incompatibility of updating with applications are also given. Actually for Windows 7 / Server 2008 r2
