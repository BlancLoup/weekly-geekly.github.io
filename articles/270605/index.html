Google will close Chrome web browser support for Windows XP and Vista

Google experts have announced the end of support for Chrome web browser for Windows XP & Vista, as well as for outdated versions of Apple OS X 10.6, 10.7 and 10.8 from April 2016. For these operating systems, the web browser will no longer be updated and vulnerabilities will not be updated for it. . Thus, users of outdated Windows XP will be under a double blow, they will not see updates not only for the OS, but also for the Chrome web browser if it is used.



This measure is another incentive for users to abandon outdated versions of the OS, which can no longer provide them with the proper level of security. Outdated versions of Windows do not contain modern built-in security features, and users of Windows Vista, which are still supported by Microsoft, cannot take advantage of modern versions of Internet Explorer like 10 and 11.
