Half-life 2 at extremely low settings

Hello.
Accidentally stumbled upon forums in which people set themselves the goal of running HL2 at the lowest possible settings, the teams listed there partially worked, and some did not. I was already tired of drinking with NG, so I took my license from the shelf, updated it to the latest version and decided to conduct experiments on myself.
Normal settings can not be so underestimate the quality.





Screenshots, videos and settings under the cut:




So, I imagine screenshots of the stage of image quality deterioration as commands are entered into the console:





















You need to run as follows:
D: \ Games \ Steam \ Steam.exe -applaunch 220 -console

Autoexec.cfg file contents,
located on the way D: \ Games \ Steam \ SteamApps \ jack7277 \ half-life 2 \ hl2 \ cfg :
con_enable 1
sv_cheats 1
mat_dxlevel 95 // Here you can set DX 6 = 60, but the quality will be more beautiful than 95, options 60, 70, 80, 90, 95
mat_picmip 4
r_WaterDrawReflection 0
r_WaterDrawRefraction 0
mat_wateroverlaysize 8
r_lod 3
mat_diffuse 1
mat_bumpmap 0
r_shadows 0
mat_fullbright 1
mat_forceaniso 0
mat_forcehardwaresync 0
mat_reducefillrate 0
mat_mipmaptextures 0
mat_filtertextures 0
mat_envmapsize 32
mat_envmaptgasize 8
mat_antialias 0

Happy New Year, rest while you can!



PS: To make barnacles visible, you need to do it in the r_lod 3 console
