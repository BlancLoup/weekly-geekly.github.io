AsteriskNotifier

The AsteriskNotifier program is a proprietary development of X telekom and is designed to manage your contact list with the ability to dial and identify incoming calls. It integrates with Asterisk IP PBX and you can call a person simply by clicking on the phone number. When an incoming call pops up the subscriber card with all its data.

What is AsteriskNotifier for?

AsteriskNotifier will save time manager for customer service. The manager will be able to talk to a lot of clients. During communication, the manager will be able to enter information about the subscriber in the directory, add new data, send a letter to the client.

The main features of the program

  • The contact list is synchronized between all clients on the network automatically.
  • Contacts in the directory are of 3 types: normal, personal and important. When a subscriber receives a call with the ‚Äúimportant‚Äù sign, a pop-up window is highlighted in red to attract attention. Contacts with the ‚Äúpersonal‚Äù sign are shown only to the subscriber who has installed this sign.
  • The program has no server part, it only stores data in the MySQL database, which is usually used by Asterisk.
  • Program settings containing passwords for access to the server are encrypted and are not stored in the clear with the subscriber.
  • When you call, you can see the current talk time.
  • It is possible to set up a synchronization database of contacts with your CRM.
  • Quick search through the list of contacts.
