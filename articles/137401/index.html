Opened repository with games for Ubuntu 10.04, 10.10, 11.04, 11.10

Yesterday, on the UALinux website, there was news that they had prepared a repository with games for Ubunt. The repository contains 250 games of different genres: you can install them with a button on the site , or you can add a repository and install packages.
The games themselves are mostly quite primitive: there are already a lot of things in the world that are much cooler than the uprisings of anthropomorphic rabbits or balls in the maze, but there are games from childhood ported from the console, analogs of the good old Worms, a few "race cards" and much more interesting. Well and, most importantly, that all games are available for free: connect the repository, set and play.
Several toys, by the way, are already in the distribution kit of the ubunt itself, but in the repository of the newer version.
In general, successful killing time.
