Autumn Postgres at Raiffeisenbank



On Monday, November 13, we invite you to the office of Raiffeisenbank, where the next PostgreSQL MeetUp will take place. The event is organized in collaboration with Postgres Professional.

We thought for a long time how to call our event, and decided to give it the romantic name ‚ÄúAutumn Postgres‚Äù, although, most likely, there will be little fall and maybe snow will fall.

Using PostgreSQL at Raiffeisenbank
Pavel Tsykin, Artem Krotov, Raiffeisenbank

As part of the report we will discuss the following aspects:

  • The need to use open source DBMS PostgreSQL
  • Problems of integrating PostgreSQL into the current infrastructure of the bank (backup / monitoring / information security requirements)
  • Example of migrating from commercial DBMS to PostgreSQL
  • Approaches to automatic deployment of PostgreSQL, embedding in the pipeline CI / CD


Postgres Pro for Developer
Ivan Frolkov, Postgres Professional

Talk about how an application developer can benefit from using Postgres Pro over vanilla PostgreSQL.


PostgreSQL Failover with Corosync / Pacemaker

Let's discuss practical recipes for using Corosync / Pacemaker in conjunction with PostgreSQL. Postgres Professional representatives will answer any questions from the audience.
