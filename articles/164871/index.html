AWS mobile management application

Hello!

Congratulations to Android users! From now on, the official application for managing the Amazon Web Services cloud from your device is available with the following functionality:
  • View EC2 server details and CloudWatch metrics
  • Filter and sort EC2 servers and CloudWatch metrics
  • Stop and overload EC2 servers
  • See alarms from CloudWatch
  • View charts from CloudWatch
  • View automatic actions for CloudWatch
  • Switch from CloudWatch alerts to EC2 servers that are affected.
  • Work in all regions
  • Switch between different users

Of course, AWS is going to add functionality to its application. Support for other popular platforms for the application has not been announced.
