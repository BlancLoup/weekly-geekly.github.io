API 2GIS news: geocoding, search enhancements and many little things



Two months ago, we spoke about the launch of the 2GIS Reference API. During this time we have received hundreds of applications for partnerships, and many of the projects have already been launched. First of all, these are city portals, but there are some rather unusual examples: from IPTV set-top boxes to SMS information services. In addition, the product itself has grown significantly in functional terms.

One of the most important new features is geocoding. This is an opportunity to search for geo-objects by name or coordinates. In addition to the actual search, it is possible to filter the results by the type of geo-objects, to limit the search to the radius or the number of results. In addition to geocoding, the functions of getting a list of projects and cities appeared.

Users familiar with other 2GIS products understand the importance of the rubricator when working with reference information. All the necessary features for this appeared in the API. The 2GIS rubricator is thousands of categories unique to each city. For example, only in Odessa there is a rubric ‚ÄúPoker clubs‚Äù, in Sochi - ‚ÄúGuest houses‚Äù, and in Vladivostok - ‚ÄúShip electrical equipment‚Äù.

For each of more than 1,127,000 POIs, our specialists indicate the time of work. Of course, this is very useful information that increases the value of the directory. If earlier API gave this data only in the profiles of organizations, now it became possible to use the filter by the time of work in search functions.

A lot of work has been done on the main search algorithms. The most important of the innovations are: taking into account the popularity of the organization and its remoteness from the center of the geoobject (specified manually, by the way); accounting for the type of geoobject for the selection of relevant results; the ability to search by landmarks; transliteration, correction of typos and errors in the "where" field.

Several new cities have been added to the API content: Nizhny Tagil , Belgorod , Tver , Orenburg , Yakutsk , Arkhangelsk and Bratsk . And in Moscow over the same time, more than 25 thousand new organizations have been added, their total number today is over 206,800.

A detailed description of the new (and old too) features can be found on the website api.2gis.ru.
