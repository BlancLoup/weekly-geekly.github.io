Alpha version of the Raspberry Pi came from the factory

Developers from the organization Raspberry Pi Foundation reported receiving from the factory the first batch of computers manufactured by their project , so now you can see photos of the board in high resolution.



This is an alpha version of motherboards, which differs from the final version: it is 20% larger in size, made in six layers instead of four, contains several expensive components that will be removed from the final release, as well as several test features that will also be removed.



But the developers do not refuse the final price. A full-sized mini-computer the size of a credit card, carrying a 700 MHz ARM11 processor, 128-256 MB of SDRAM, USB 2.0, composite and HDMI video outputs, as well as a general-purpose I / O port, will cost $ 25.

Checking the alpha version showed that Linux is working fine, all tests pass (see log )
