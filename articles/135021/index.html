CloudShare - a cloud service for creating servers with SharePoint and not only


There are various problems with installing SharePoint, and the process itself can take a long time. Some upload ready-made images as VHDs, and work with them.
But what if there is no place to install SharePoint or connect VHD? You can use the CloudShare service, which allows you to start using SharePoint in a few clicks.

Specs


What is included in your cloud (CloudShare ProPlus tariff plan)
  • The number of machines is from 1 to 6. Each machine and software on it are licensed. It may be
    • SharePoint 2010 Enterprise
    • SharePoint 2010 Information Worker
    • SharePoint Foundation 2010
    • SharePoint Foundation 2010 with Dev Tools
    • SharePoint 2007 Enterprise
    • CentOS 5 with MySQL
    • Windows 2003 R2 with Oracle 11g
    • Windows 2008 R2 Active Directory
    • etc. Full list of available machines can be found here .

  • Ability to configure the network between virtual machines
  • 10 GB RAM, 300 GB HDD and 10 virtual CPUs are allocated to all machines in total.
  • 24/7 support by email and forum

Who can it be useful?


The service can be useful to developers, IT professionals, sales managers, trainers, and anyone who needs to work with several environments, or don‚Äôt want to waste time on administering operating systems.

How can this be useful?


In fact, there is no need to spend time on administering operating systems (Fig. 1). It is also convenient that there is a snapshot mechanism that allows you to quickly create a copy of your environment.

Fig. 1. Administration panel in CloudShare
If necessary, you can provide access for external users (useful for various demonstrations and collaboration).
It is important that, if necessary, RAM increases automatically, but for a limited time.
Creating a new environment is simple, for this you need to go through a three-step process (Fig. 2)
1. Choose cars that will be surrounded

2. Specify a description of the environment

3. Run!

Thus, the service allows you to concentrate on work, forgetting about administration.

But what about the disadvantages?


The most significant drawback is that if you do not use your environment for a while, then its work is suspended. The reason is very simple - this tariff plan does not imply the use of the service as an Enterprise solution. For this case, there is an additional option ($ 599 per month) or the CloudShare Enterprise tariff plan .
