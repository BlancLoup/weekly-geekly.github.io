First contact


Usually sending a resume or call. Resume is better to write without beautiful, as many use automatic scanning systems. If the resume came up, the agent will call within 48 hours. After the first contact, the agent will disappear for a couple of days or forever. If the applicant was deemed worthy, the agent will go all the way these days to arrange an interview.
