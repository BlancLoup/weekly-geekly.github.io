Making the Google Notebook extension work in firefox 3.5

Here comes the long-awaited FireFox 3.5, after which many of your favorite extensions stopped working. Among them was Google Notepad (Google Notebook). I have met the news somewhere that the Google team has stopped supporting this project, but I haven‚Äôt found the link.

Fortunately, making it work is very simple. To do this, follow these simple steps:

1. Under M $ Vista (it should work under XP and 7) go to the % APPDATA folder % \ Moilla \ Firefox \ Profiles \ NEED_ PROFILE \ extensions \ notebook@google.com \
2. Open the file install.rdf
find the string <em: maxVersion> 3.0. * </ em: maxVersion>
change 3.0. * to 3.5. * and save the file
3. Copy the folder notebook@google.com to any place
4. Start FireFox and remove Google notebook from the list of extensions, close FireFox
5. Copy the folder notebook@google.com back (step 1)
6. Launch FireFox

UPD. Another option from hellt (but you have to be careful, incompatible extensions can cause problems):
0. about: config
1. right click, create -> logical
2. name extensions.checkCompatibility, value false

After launching the browser, you will again be able to use your favorite notepad. I have not detected any compatibility issues during operation.
