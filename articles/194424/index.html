This is how the editor in the new version of WordPress can look

For more than ten years, in order to edit the content of the site on WordPress, you have to enter the administration panel, but this should change soon. The WordPress development team set to work on the ability to create and edit content directly on the fly in your WordPress theme.





This approach has long been used in some online products, such as the Medium network and the Squarespace platform, but WordPress is much more difficult to implement this than it seems at first glance - after all, WordPress is well-known for its backward compatibility, and tens of thousands of themes have been written for WordPress.



For several years, the popular TinyMCE library, the most flexible and reliable visual editor for the web, has been used as an editor in WordPress. Rejecting it from WordPress is not planned, but on the contrary - the new version of TinyMCE 4.0 fits very well into current tasks and is not an obstacle to the implementation of your plans in the future.



The new editor is planned to support tags, categories, record formats. It is also planned to have full-fledged work with the WordPress media library, and of course, support for mobile devices and tablets.

The project is at the idea and prototype stage, which can already be installed from the plugins directory , and it is likely that such an editor will be included in the core WordPress in the future. Another similar project is the popular Front-end Editor plugin.
