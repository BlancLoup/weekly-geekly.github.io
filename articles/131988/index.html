GParted Live 0.10.0-3 Stable Release Released


Thanks to Steven Shiau, the GParted team proudly announced the release of the new stable GParted Live (0.10.0-3).

This release includes a GParted 0.10.0 application in which:
  • added support for resizing on Btrfs file system;
  • Removed outdated info icon;
  • the basis is GNU / Linux, the operating system is based on the Debian Sid repository (as of 2011/11/02);
  • Updated Fluxbox window manager to version 1.3.2.
