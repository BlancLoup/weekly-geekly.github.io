Skype stops supporting Skype Lite and Skype for Windows Mobile.

The latest changes in Skype‚Äôs mobile support policy are quiet and inconspicuous. Without high-profile press releases and news on the internal blog. So the other day, the page of the final Skype for Symbian appeared in the public domain, which until now had appeared exclusively as a trial beta version.

A modest line appeared on the Skype for mobile phone page today: Skype Lite and Skype for Windows Mobile are no longer supported . For more information, click here.

Unfortunately, the link to the detailed information is not presented any details, as the link leads to the page of the Skype To Go function.

However, I still managed to find an explanation for what is happening. According to the answer from the FAQ section of the site :

" Why can't I download Skype Lite or Skype for Windows Mobile?

Unfortunately, Skype Lite is the Skype version for mobile phone and Skype for Windows Mobile is no longer available for download from our website.

We decided to stop supporting Skype and Skype for Windows Mobile Lite because we want to offer our customers new improved products for mobile use, such as Skype for the iPhone, which has proven so popular and now available Skype for Symbian phones.

Our goal is to ensure efficient user experience by providing the ability to freely communicate with Skype-to-Skype and call ordinary phones at low rates as easily as it does on a desktop computer.
We felt that Skype Lite and Skype for Windows Mobile did not allow us to fully use all the features of Skype.

Can I use Skype Lite if it is already installed?
Yes. Users who have already installed Skype Lite can continue to use it throughout 2010.

Can I use Skype for Windows Mobile if it is already installed?
Yes. Users who have already installed Skype for Windows Mobile, of course, can continue to use it. "

Despite this, the product pages themselves are still available: Skype for Windows Mobile , Skype Lite .
