Microsoft may be collecting a developer version of the new Xbox



According to IGN, a new version of the Xbox gaming console is already being assembled at one of Flextronics' enterprises. This company collects the current version of Xbox, and most likely, it will also collect and game console new version. The assembly is not yet on a large scale, but only at one of the Flextronics enterprises in Austin, the Texas branch of the company.

According to an unnamed IGN source, Flextronics has created a new test group that works independently. This team tests the next generation Xbox software and hardware. The code name for the new console is Durango. Unfortunately, the fact that this device is already being tested is not a guarantee that Durango will soon appear in public.

Most likely, we are talking about the work of Flextronics on the developer version of the new version of the console. It is these next-generation versions of Xbox that will be sent out to game developers, the most famous companies, so that they understand what they will have to deal with in the future. Earlier, Microsoft has already announced that the new console will not be shown at the E3 event, which has been rumored for a long time.

Unfortunately, neither Microsoft nor Flextronics responded to an IGN request. So there is no official confirmation (actually, and refutations) of the above data. However, there is usually no smoke without fire, and probably a test version of the console is actually being developed.
