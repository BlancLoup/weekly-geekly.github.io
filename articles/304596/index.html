Introducing 3CX v15 Release Candidate

We are pleased to present the 3CX v15 Release Candidate! There are only a few days left until the release of our system. We remind you that we will be waiting for you and your questions at the v15 presentation on July 6 on the YouTube channel (5.00 pm Moscow time)!

The Release Candidate in 3CX Client for Windows has several new themes and the ability to set your own background. Also made several significant updates to speed up the creation of audio and video conferencing.



Major improvements


  • Added new domains to support the FQDN of your 3CX installations: 3cx.co.uk, 3cx.de, 3cx.nl, 3cx.fr, 3cx.es, 3cx.it, 3cx.ru, 3cx.in, 3cx.com.au and 3cx.co.nz
  • A number of improvements have been made in the 3CX Management Console system management interface.
  • 3CX Client for Windows development completed
  • Chat improvements

New features of 3CX Phone System v15, not included BETA version


  • Easy configuration of FXS gateways
  • Easy configuration of base stations and DECT terminals
  • System permissions at user level (extension)
  • Group editing of extension numbers
  • Ability to run VAD (Voice Application Designer) scripts
  • New version of 3CX SBC (Session Border Controller)
  • 3CX WebMeeting app for iOS

Clients 3CX for Mac and Windows will be automatically updated to work with 3CX v15. 3CX clients for Android and iOS v14 will continue to work with 3CX v15.
