5 social applications created on the basis of Code for America

Code for America is a platform for improving municipal services through ICT, as well as communication among government officials, programmers, and civil society activists. Main product: web applications that 1) allow citizens and officials to be connected more quickly and efficiently, 2) increase the transparency and responsibility of the authorities, 3) are publicly available, and therefore can be used by other cities and municipalities.

The consequence of the openness of the created applications is that not only US authorities can use such applications. Do you understand what is at stake? :) Everyone can use! We thought that a review of such open source projects would be helpful.

A little bit about how Code for America works

Code for America operation algorithm:
  1. Working with city administrations - identifying actual problems that can help solve web applications, publishing an application for creating an application;
  2. Competitive selection of the performer;
  3. The customer and the contractor are connected and must create a working application in 11 months;
  4. Throughout the process, Code for America specialists conduct training sessions, meetings, and advise both sides of the development process.


As a result, effective communication between the government and society has become a catalyst for the emergence of applications that can change the appearance of cities for the better. And, it is important to note, not only American.
