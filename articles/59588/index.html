Tag Slaps: The Contest for the Most Creative Microsoft Tag


Microsoft has developed a tool for customizing the design of its Microsoft Tag - you can insert a picture as a background, you can change the symbolism (originally triangles) - and announced an open competition for the best code design.

To participate in the contest, you should provide not only a beautifully designed code, but also a mobile site to which it should lead. Not only the aesthetic properties of the code will be evaluated, but also the marketing potential of the mobile site that is attached to it.

The works are put forward until June 12, then the jury selects the top 20 and from June 22 to June 26 an open vote will be held on the basis of which one winner will be determined. The winner will be famous on the main Microsoft Tag website, blog, various presentations and other media channels.
