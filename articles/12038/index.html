Fujitsu Custom LiberTouch Keyboard

Have you ever wondered why the keys on the keyboard are arranged in this order: QWERTY and further along the same lines? Engineers from Fujitsu obviously broke their heads over it. And as a result, a fully customizable LiberTouch keyboard was concocted. Rearrange the keys in the order you need only? Change the layout beyond recognition? No problem.


A special device is attached to the LiberTouch keyboard, with which you can remove and put any key on the right place. The manufacturer does not apply to the account of how the key permutation process is implemented from the technical side. But this is not our concern. The main thing is that LiberTouch allows you to put the buttons in the order you know. And this applies not only to the basic layout, but also to the function keys.
