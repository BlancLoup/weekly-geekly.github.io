Ubuntu 11.04 goes back to classic GNOME by default instead of Unity?

The message on the Ubuntu developer mailing list states that the current state of Unity will be brought up for discussion and, if necessary, the classic GNOME desktop can be left in Ubuntu 11.04 by default.
