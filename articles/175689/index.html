Corona SDK Starter - free version


Great news for developers for mobile platforms. The Corona SDK framework has got a free version that hardly differs from the Pro version in its recent understanding.

The Starter version will not only support microtransactions, analytics from Corona and access to daily builds.

The Pro version adds support for the plugin system, which partially solves the problem of the Corona SDK closeness, and also increases the price to $ 599 from next month.

Enterprise version gets its price tag of $ 999 and includes support for native code and create builds offline.

Now to start creating mobile applications quickly became even easier. The increase in the price of the Pro version, of course, is sad, but if you really make money with Corona, then $ 599 is still an acceptable amount.

Let me remind you that Corona SDK is a framework for creating 2D mobile applications and uses the Lua programming language.
By the way, soon the Corona SDK developers will introduce support for 2.5D graphics.

You can download from the Corona SDK Store page.

UPD: According to an interview with one of the employees:
  • current subscribers have been renewed for two months;
  • Indie subscriptions are automatically transferred to Pro status;
  • if you already have a Pro subscription or you purchase it before April 30, then two subsequent subscription renewals will cost $ 349 instead of $ 599.
