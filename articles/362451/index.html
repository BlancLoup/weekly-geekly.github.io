WEDG: personal cloud storage with reliable data protection



Protection of personal and corporate information is a problem that can be called critical. Users and companies store information on social networks, in cloud services, and there is no guarantee that this information does not fall into the wrong hands.

Personal information of users is generally very often attackers. A regular user simply does not know how to protect his information. And in order for such an ordinary user to get their hands on a reliable tool to protect their data, the Wedg project was founded. This is a personal cloud storage with reliable protection of personal data.



Wedg features



This device can be called a universal personal cloud, since Wedg allows you to store your files, set up e-mail accounts, create your own analog Dropbox (with encryption and passwords), store Bitcoin wallets, host sites. All this is equipped with an encryption system and a multi-level authentication system.



Wedg functions



Security and encryption. The system uses 512-bit encryption (XTS-AES), with the activation of the system at the first login to the created account. SSL is used for all browser sessions. All work is done using private encryption keys (GPG / OpenPGP) - a user without a key will not be able to access data;









File storage, data synchronization. With the help of Wedg, you can create a private analogue of Dropbox, with the ability to send links to access specific files. At the same time, the lifespan of such a link is specified, as well as the access password;
Application catalog. Third-party developers can create their own applications for Wedg, expanding and deepening the functionality of the system;
Email. Smart mail, address book and calendar for all users. Compatible with Microsoft Outlook, Apple Mail and Mozilla;
Streaming media files. On the basis of Wedg, you can create your own media server, broadcasting video and audio;
Placement of sites. Wedg can be used as personal hosting by hosting sites based on any CMS;
Cryptocurrencies. Wedg supports Bitcoin wallets and other cryptocurrencies;
Business analytics. On Wedg, you can put a system like CRM and work with it without having to pay a monthly fee every month or year.



Wedg specifications



The developers offer two versions of the system: on the ARM platform and the Intel Atom platform.



If something went wrong:

For various unpleasant incidents (hard drive failure, loss or theft of the device), the manufacturer has provided suitable solutions to the problem:

Backup data. You can upload the full archive to a special storage, to a drive in a NAS, external drive, Apple Time Machine or another Wedg device. The video below explains all this:



Device control



Work with your personal Wedg cloud using a mobile device on iOS 7/8, Android, Windows Phone 8 or BlackBerry platforms. Each device has its own application.



The developers tried to make the applications safe for work, for which they provided an SSL connection and
2-step authentication.



There are also desktop applications for Microsoft Windows and Apple OSX. The Linux application is being prepared for release, and will be ready by the time the first batch of devices is shipped.



Open source



The system is built on open source software. Developers are planning to further create a whole community, whose representatives will expand the number of applications for Wedg.

Price and order



The developers have already collected the necessary amount to launch the device into mass production. The first batch of WEDG will be shipped in March 2015.

Price ARM version - 238 US dollars (149 pounds).

Price Pro version - 350 dollars (219 pounds).
