Meet Compiz Fusion

The united community will now bear the proud name of Compiz Fusion. Details of the announcement in the mailing list .
But we have not only a new name, our developers have worked hard and we can show some improvements and changes.


The Zoom plugin was almost completely rewritten by David Reveman, which added new features like selection to it, followed by zooming or scaling.
The cube now has a highlight, thanks to Onestone for that. There is also a new application convolution mode (expose) that can work all over Cuba at the same time.

When you change the window will pop up an inscription with its size. There are some more useful and not-so-great enhancements that have been added to plug-ins.
And added a new animated effect Dodge.

Compiz now has two utilities for customization, so that users can easily adjust it to their needs.
