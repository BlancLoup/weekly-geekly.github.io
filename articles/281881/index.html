Tab closing options and other improvements in Vivaldi 1.1.453.6 assembly

Hello!

Almost two weeks have passed since the release of the first final version of the Vivaldi browser, and over the past few days we have received a huge amount of feedback from you. There were many positive ones among them, but we paid the most attention to error messages. Actually, we were engaged in the correction of these all the time, and soon we plan to prepare an update for the stable version to number 1.1 - you may notice this change in the number of today's build. However, as always, we not only corrected errors, but also expanded the functionality of the browser.

In the browser tab settings, some new options have appeared, now concerning the closure of tabs. You can choose which tab to activate when you close the current one: the next link, the next one according to the tab switching order specified in the settings, and also activate the tab to the left or right. So to say - for every taste.

Another improvement concerns the process of importing data from the Opera 12 browser. We added the ability to transfer Express Panel links to Vivaldi. After the import process is finished, the folder ‚ÄúImported Speed ‚Äã‚ÄãDial‚Äù will appear in the bookmarks, and one more Express panel with the same not yet translated name will be added on the start page. Unfortunately, importing the express panel sketches will not work - so you will have to create them again by updating the cells.

We also continued to work on improving the dark interface, and also fixed the error with the interception of the focus by the Vivaldi browser window. A complete list of changes you will find, as always, at the end of the article.

And now about the sad. We updated the browser core to version 50. Unfortunately, in this version of Chromium, support for Windows XP, Vista and MacOSX below 10.9 is disabled. For this reason, we have disabled the ability to upgrade Vivaldi to version 1.1 on these operating systems. Users of these operating systems will only be able to use Vivaldi 1.0. At the same time, we also disable NPAPI support - in Chromium, work with these plugins has become very unstable. Now all attention will be paid only to PPAPI plugins and HTML5 technologies. Read more about installing the PAPPI flash plugin here .
