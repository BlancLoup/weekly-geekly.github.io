Metrics for the blog: what and how to count?

Do you evaluate the success of your blog? I am sure that every blogger from time to time thinks about it. I will try to evaluate most of the possible blog metrics.

1. Blog Attendance

Standard metric that is used for any sites. You can use external counters (for example, li.ru ), or you can use log analyzers (I personally like awstats ), if your hosting allows it.

2. The number of subscribers

It can be measured with the help of counters (for example, feedburner , rss2mail ), and also with the help of log analyzers. Now reading blogs via RSS readers is extremely popular, so often the number of subscribers is greater than site traffic (for example, I have 2000+ subscribers and only 500 visitors to the blog). Based on the number of subscribers, ratings are built: for example, blograte.ru.

3. Average number of comments to blog entries

Of course, this parameter is indicative only for blogs, where comments are open (they are now open in almost all blogs), and there is also a moderation of comments (without moderation, comments will be 90% spam). I have an average of 12 comments on each blog post.

4. Technorati Rank

In fact, this is an indicator of how often a blog is mentioned in other blogs, i.e. popularity among bloggers. Technoratis not all blogs index, but very many, so the indicators are very truthful. Based on this, you can also build a rating, for example, like this .

5. Other

* mentions in blogs according to Yandex
* Referring pages by Google version
* referring pages by version of Yahu

Good luck in evaluating your blogs! With the desire on the basis of these parameters, you can come up with a formula for evaluating any blogs.
