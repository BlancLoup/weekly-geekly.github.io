New course on Windows Server 2012 available at MVA

In the run-up to the global availability of Windows Server 2012 , a new course Windows Server 2012 New Features has appeared on the MVA portal . Part 1. Virtualization, networks, storage . The course is a fairly detailed overview of the most interesting features of Windows Server 2012 related to Hyper-V, network infrastructure and storage organization.

All material is divided into three modules:
  • Module 1. Hyper-V. Comprehensive Virtualization Platform
  • Module 2. Network Infrastructure
  • Module 3. Storage and Availability

The information in these modules is interconnected, but we tried to ensure that, on the one hand, there were no intersections, on the other hand, you could listen to these parts in any sequence. Some technologies presented in the course have already been discussed on Habr√©, for example, Private VLAN , some you will see for the first time.

But anyway, you have quite a lot of technical information with a total duration of about 5 hours, 15 demonstrations of various technologies, including Network Virtualization, Storage Migration, SMB Multichannel, NIC Teaming, SMB Transparent Failover, Storage Spaces, Hyper-V Replica, DHCP Failover and etc.

The plans of the second part of the course, revealing the issues of security, management, remote access, use of a web platform. I hope the materials will be useful. Comments, suggestions, criticism can be sent directly to me.
