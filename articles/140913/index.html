GNOME 3.4

Today was the release of the remarkable GNOME 3.4 graphical environment.

Six months have passed since the release of the previous version. During this time, major changes have been made in GNOME applications, many bugs and flaws have been fixed.


GNOME 3.4 includes many new features. A new document search engine provides quick access to your data, both locally and online. GTK 3.4 adds smooth scrolling. The menus of many applications are transferred to the top panel, which frees up the workspace and allows you to get rid of the window title in full screen mode.

The new release also includes many changes regarding GNOME applications. The web browser, formerly known as Epiphany (now just Web), received a completely new interface and major performance improvements. New interface also received Documents and Contacts. All this - the result of work aimed at creating modern and functional applications GNOME 3.



The official GNOME website has a complete list of changes .

It should be noted that after the upgrade, the extensions may stop working, if the author has not yet updated them for the new version, and topics that need to add a few minor changes to support GTK 3.4.
