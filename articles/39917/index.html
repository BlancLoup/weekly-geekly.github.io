Plugin for viewing the Opera cache

What Opera users have been dreaming of for so long has happened!

A content (informational) plugin for Total Commander, the most popular file manager, has been released, allowing you to normally view information from the Opera 9.5x cache.
It looks like this:


Let me remind you that in Opera 9.5, files in the cache were stored without extensions, which caused confusion (and even abuse) among Opera users. With the same plugin, finding and ripping the desired file from the cache is as simple as possible.

Wrote a plugin comrade Kostya Vlasov, for which he thanks a lot.
Plugin on wincmd.ru.
