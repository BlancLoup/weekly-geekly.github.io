Other Drupal distributions

Here is a short list of other distros known (other than Acquia) to the community:

  • for users of the fifth version of Drupal, there is the Shvabra package from the Shvabrashvabr blog. But everything is a little more complicated. Site administrator named Romka offers to download and install 12 modules (Views, Organic Groups, CCK, Vote Up Down, Voting API, OG Alias, User Karma, Buddy List 2, Usernode, Tagadelic, Pathauto, Token, Global Redirect, Nodewords, Excerpt , Job queue) for version 5.x, the Aberdeen design theme, download the Shvabra installation profile ( direct link ), run the Drupal installer and so on ... The purpose of these manipulations is to create a full copy of the Shvabrashvabr website :)
  • for educational sites there is a distribution DrupalEd (for the version Drupal 5.6). Simple assembly, with Tagadelic and Pathauto. Here is the direct link .
  • for avid "members of the forum": vbDrupal ( direct link ) - this is Drupal version 5.1 + the popular "engine" for forums, vBulletin.
