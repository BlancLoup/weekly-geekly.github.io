Survey results for adding new features to the Vivaldi browser

Hello!

A week ago, we launched a vote for new features that, in the opinion of users, need to be added to the browser first. In total, more than 10 thousand people from around the world took part in the survey, and although we initially limited the number of available votes to three for each user, a serious struggle for leadership broke out between the functions. However, the first place at once and for a long time was taken by the most popular function - the ability to synchronize your data between several installed browsers. But for the second and third place was a fierce struggle between RSS, ‚Äúlazy tabs‚Äù and smooth scrolling. In the end, the latter pulled ahead and took second place. Below you can see how the votes are distributed throughout the wish list.


This list includes a wide variety of wishes related to different components or browser properties, so I decided to split the list into categories and see how the voices were distributed within each one. You can see the results below:


There are only three requests for performance, but by the number of votes this category is the undisputed leader.


In relation to the browser interface, there are mainly leading requests related to more informative work on the network, as well as opportunities to improve the usability of the numerous functions of the browser.

And this is not surprising: the browser is the most popular application that is used every day (and, sometimes, all day) by all network users, so it is logical that working with Vivaldi be comfortable and convenient.


Actually, the functions are what allows you to make a powerful and convenient application from a simple web page viewer that can provide the user with everything needed for all cases of network life.


At first glance it may seem strange that this category, which is responsible for the possibility of fine-tuning the browser to the user's habits, is so few. But in fact, all of the above functions are also directly or indirectly tools for customizing the browser to the requests of its owner.


And this is how the categories were distributed according to the average number of votes for each request. As you can see, first of all, users are concerned with the issue of performance and functionality, followed by the convenience of using the browser and the ability to customize its work and interface according to its preferences.
