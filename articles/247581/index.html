Best Photoshop plugins for web designer

Divine elemente



This is perhaps one of the best projects I have ever seen. Divine Elemente allows designers to create a WordPress theme without knowing any frontend / backend. Unfortunately, I did not have the opportunity to try out this plugin in action, as it exists only for Windows. For the work of designs you need to create them exactly with the preset templates. It looks almost like Twitter Bootstrap. The developers provided us with everything: SEO, semantic class names and easy to read code. Wonderful!

Layrs Control CS6, CC



"A collection of 7 scripts that simplifies tedious work." The plugin successfully copes with the removal of all disabled layer effects, unused layers, finds and removes similar files and folders, converts some objects and makes the design in the Flat style. By the way, the latter happens with the help of Skeuomorphism.it .

Cut & Slice CS6, CC



Wonderful plugin created for "cutting" templates. Allows you to export layers based on different devices and permissions (.xxdpi, .Idpi, .mdpi and .hdpi). The simplest syntax allows you to perform simple operations on layers and groups of layers: cropping, scaling, including buttons and more.

Render.ly CC


Render.ly does the same as Cut & Slice. It is also an extremely handy tool for converting. All your comments will be synchronized and available online.


CSS3Ps CS3 +



Awesome plugin that converts layers to CSS or SCSS / SASS with Compass. Dimensions, inner and outer shadows, gradients, radii, and more are supported. The only negative is that this beauty works with the server, and it takes about 30 seconds to get the result.

Composer CS5, CS6, CC



Composer allows you to edit parameters (style, position, visibility and placement relative to other objects) several layers at a time with a single click.

I also want to note an analogue - Transform Each 2.0 .

Guideguide



I will say simply: the best tool for building a grid.

There is also an equally useful open source Griddify :


WebFont CS5, CC



Today, many Internet projects use unusual fonts that do not work on all devices. This plugin gives you convenient access directly to Google Fonts directly from Photoshop. To use the plug-in, you will be asked to download a thirty-day demo version of Suitcase Fusion 5, but after the end of this period, the tool will still work.

Random User Generator



"Like Lorem Ipsum, but for humans." During the creation of a design, it is often necessary to create an image of a real person with viewing his profile in comments, posts and so on. Random User Generator saves you from searching for names and avatars. The tool works with the Randomuser.me API.

Blendmein



A cool plugin looking for thousands of options for you icons directly in Photoshop or Illustrator. All found material was created under the Creative Commons Attribution 3.0 Unported License.

Flat Icon CS5, CS6, CC



The plugin provides access to perhaps the largest (38000) collection of sets of flat-icons. The tool has more than 53,000 installations. The author is Flaticon.com .

Social Kit CS6, CC



Easily editable and customizable templates for social networking pages. All content is updated with each new product in the construction of similar sites.

Velositey


The best plugin for creating layouts.
