More cores, good and different! We meet new Xeon Phi



For more than two years, new multi-core Intel Xeon Phi coprocessors have not been introduced to the market. One would have thought that the company had lost interest in them, but this is not so, the proof of which is the newly announced new line 7200 (codename Knights Landing). There are 4 models in the line: 7290, 7250, 7230 and 7210. Compared with the previous generation, the number of cores has increased, now they are from 64 to 72 (up to 288 threads with hypertreaming). The process technology decreased to 14 nm. The operating frequency has slightly increased, while the power consumption of 215-245 W has dropped by almost 20%. L2 cache increased to 32-36 MB.

The new Xeon Phi received support for DDR4-2400 memory (except for the younger model, it uses DDR4-2133), the maximum memory capacity is 384 GB, the memory bus speed is 115/102 GB / s, the size of its own memory is 16 GB. Coprocessors can use the AVX-512 advanced instruction set, and a number of modern processor functions are available, such as AES-NI to improve encryption performance, Execute Disable Bit, and others. But hardware support for virtualization, unfortunately, is still not there. Host connection bus - PCI Express 3.0 x16.

New coprocessors will be available during this year, their cost - from $ 2500 for the younger model to $ 6200 for the top. Below you will find a summary plate of the main characteristics of Xeon Phi, for a detailed comparison table, see the Intel ARK .
