Fire.fm

Surprisingly, there are so many fans of Firefox and Last.fm, but nobody has yet written about the Fire.fm extension =) A wonderful thing that creates a toolbar that allows you to control the playback of tracks from Last.fm.
Take here - addons.mozilla.org/en-US/firefox/addon/7684 (sandbox, you need to login to deliver)
For some reason, only version 0.7 works for me - all later versions do not lose tracks, I am now chatting with developers on this topic. Whoever has the same problem as me, take version 0.7 from here - addons.mozilla.org/en-US/firefox/addons/versions/7684 , and in the comments write what you have for the system - then send all information for developers =)
