Consciousness transfer to another body: we clone the OS to a new drive


Contrary to popular myths, the operating system can be transferred from the hard drive to the SSD without any harmful effects without additional settings. And even without losing activation, as is the case with OEM versions of Windows after replacing the motherboard.

Of course, if the system partition has grown beyond the capacity of the purchased SSD, it would be good to remove large-sized junk and disable hibernation (it is much more sensible to use solid-state drives in sleep mode), but no more preparation procedures are required. Further, there are several options (as the decay decreases):
  1. A Microsoft- recommended way to migrate a system using Sysprep.
  2. With the free Clonezilla utility
  3. Using Acronis True Image or Paragon Migrate OS to SSD paid utilities
