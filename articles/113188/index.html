Microsoft made copy-paste available for Windows Phone developers

Microsoft Developer Relations Director Brandon Watson told PPCGeeks in the podcast last night that the company will release a new version of the Windows Phone Developer Tools to the registered developers of Windows Phone 7 today.

In addition to improving performance, the long-awaited copy-paste function is included in the update. It gives developers the opportunity to test their applications for compatibility. It is also a signal that a copy-paste will soon be available to end users.

During the podcast, Watson noted that Developer Tools will soon reach the milestone in downloads from the moment it was presented at the Mix conference last year. This is not small, considering that the number of registered developers of Windows Phone 7 is 24,000.

The release date of the update for all users is still unknown. Last week, Greg Sullivan, senior product manager for Windows Phone 7, spoke about the term "in the next few months."
