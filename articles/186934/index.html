BitTorrent Sync goes into beta


BitTorrent Sync Alpha was released in April, and today we are pleased to announce its transition to a new level - Beta.

BitTorrent Sync is a new product designed to synchronize data between computers with your friends, colleagues, as a team. It greatly facilitates the work within organizations, as it makes the most of the company's internal infrastructure. Sync was created to solve the real problems that we face every day when transferring files: restrictions on the size and speed, as well as threats to privacy and security that arise when using cloud technologies.



So, we are pleased to announce the release of BitTorrent Sync Beta. Since the release of the first version in April, we received a lot of feedback and know that Sync was used in universities, for transferring footage film, for backup, when writing blogs and managing projects. Thanks to the people who decided to try our application, we were able to improve and develop it. Now Sync includes two of the most requested features: support for mobile devices and archiving.

Mobile support


You asked - we did. :) BitTorrent Sync is now available for use on Android devices. You can synchronize folders selectively or completely, make backup of any folders, and also share files with friends and colleagues directly from your smartphone.
IOS users, we have not forgotten about you. Keep for updates.

We are especially proud of the function of fast file transfer between mobile devices. No need to create folders and enter secrets. Simply select the files, and Sync itself will create a virtual folder and generate a QR code (which contains the read only secret for the virtual folder). When you scan it on another device, Sync will immediately establish a direct connection and start transferring files. Directly, without limitation of size, at maximum speed between any number of devices.

Syncarchive


BitTorrent Sync Beta also includes basic versioning features. In each folder, a .SyncArchive directory is created, where all previous versions are added when files are changed, as well as files deleted on other devices. Now you will be able to recover data if they were changed or deleted by mistake.
