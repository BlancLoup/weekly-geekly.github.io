Minimum font size and default encoding in Vivaldi 1.0.344.5

Hello!

We continue to fulfill the wishes of users regarding the extension of the functionality of the Vivaldi browser and increasing its usability when working with the most diverse web pages in style and quality of execution. And today's news will help you to learn a little more comfort while working on the network.

Minimum font size

Here, in fact, no lengthy explanations are required: you set the minimum font size, which does not make you strain your eyes when reading texts on web pages, and you always forget about web designers who are trying to fit the maximum information per square centimeter of the monitor screen. Also, this feature will be useful for owners of very large monitors. To find this option, open the section Tools-> Settings-> Web pages-> Fonts and set the desired font size:


And in this section you can see the default encoding settings for web pages. Today, this is not the most important option, but sometimes it helps to read what web developers have diligently hidden, forgetting to specify the encoding for their pages.

Of course, the font settings will still expand, so we are waiting for your wishes and new ideas.
