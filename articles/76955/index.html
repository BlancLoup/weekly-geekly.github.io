Beagle will find the most hidden file on your Linux



If you are using Linux , then you are probably faced with a situation where you had to search through a bunch of folders and find a specific file. Today I will tell you about the small program Beagle , which will help you find even the most hidden file from your eyes.

This free program resembles Spotlight from Mac OS X or a standard search utility from Windows 7 . Beagle indexes files, IM correspondence, email, images, web pages and many other file types. Beagle performs its search for keywords and meta data files. For example the name of the MP3 track or the name of the sender of a letter. In addition, Beagle indexes retrievable devices (for example your Ipod).

Beagle works with both KDE and GNOME . Unfortunately for his work need a few third-party packages, such as Mono, sqlite ( full list ). You may not need to download anything from this list at all, since these components are already installed in the main Linux distributions ( Debian , Ubuntu , Fedora Core ).
