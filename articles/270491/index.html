Global Day of Code Retreat: free developer training

On November 14, Global Day of Code Retreat will take place at the cozy Attic of Rambler & Co's office.



What is Code Retreat?

Code Retreat is a one-day intensive training for developers that takes place in many countries in order to practice fundamental development skills: simple design, SOLID, TDD, pair programming. Also, this is an opportunity to communicate, share experiences, expand your horizons. The language you speak (write) does not matter.

The format of the event is as follows:
  • all day consists of five to six 45-minute sessions;
  • all programming goes in pairs;
  • each session pairs randomly shuffled;
  • at the end of each session, all written code is deleted;
  • each session solves the same problem (Conway's Game of Life / Game of Life), but with various additional restrictions.

Read more about Code Retreat here: http://gdcr.coderetreat.org/

Registration and driving directions here: https://rambler-digital-solutions.timepad.ru/event/260523/

Come, it will be interesting!
