New York Times prepares to release Chrome OS

The New York Times has released a new look for the site , specially prepared for Chrome - the operating system of Google.



Features Chrome version:
  • Easy interface for quick loading.
  • No extra elements, just text, informative pictures and, of course, advertising.
  • 10 different display templates - from ascetic text to colorful pictures.
  • Control using the keyboard, as in other Google products (Gmail or Reader).
  • Active use of the cache for offline reading.
Those who like this look of the newspaper can now use it in the latest versions of the Chrome and Safari browsers. For Firefox, there is a slightly simpler version ( darvecher , thanks for the tip).
