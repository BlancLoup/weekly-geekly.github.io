Now Microsoft accepts Bitcoins



The software giant Microsoft since December 2014 accepts Bitcoin for payment of cryptocurrency. So far, only US residents can use this feature. Direct purchases are not made - you first need to replenish your Microsoft account. Then these tools can be used when shopping in Windows stores, Windows Phone, Xbox Games, Xbox Music and Xbox Video.

To service payments, BitPay is used , which has proven itself to work with PayPal and Newegg. On the page with instructions for payment it is noted that payment using Bitcoin cannot be returned - this is one of the characteristic features of cryptocurrency. At one time you can replenish your account using Bitcoin for an amount not exceeding $ 100.

Bitcoin slowly but surely continues to penetrate into everyday life. PayPal is already accepting payments in Bitcoins (albeit, while in the US). The currency itself is still quite stable, since October 2014 the BTC exchange rate has been at around $ 350.
