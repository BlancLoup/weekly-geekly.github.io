Virtualization as a software distribution tool

I think most ITshnikov have heard of VirtualPC and VMware.

The extent to which this market is now at its peak has once again confirmed the departure of one of the top employees from MS to the ruling composition of VMware.
(I do not remember where I read, but it was this winter)

But recently I discovered that VMware Server is absolutely free!
VMware Server differs from VMware Workstation in that it is installed as a service and, when it is turned on, will automatically raise all virtual machines.

But VMware went further.
They began to offer ready-made virtual machines for free download.

For example, Kerio MailServer installed under Fedora Core 6, is a ready OAMP (OpenBSD + Apache + MySQL + PostgreSQL + PHP) server.

But most pleased with SourceForge Enterprise Edition .
Free for 15 users package, which includes the entire set of software for organizing the workflow of developers in which the project management system, documentation, Bugtracking, SVN, CVS are integrated.
Everything is already configured and installed on the virtual machine.
When you first start, you just need to change the root password and set the IP and domain name for the virtual machine in the dialog box.

There is a turnkey solution for Trac .

And the main advantages:
  • you can keep several virtualoks on the same server
    backup of a tuned system is reduced to copying virtual machine files
    Easy to transfer virtuals from one physical machine to another (if you are not a fan of reinstalling Linux :))
