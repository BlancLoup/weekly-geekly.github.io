Android update on smartphones 2011

In the very near future, an Android update will be available for phones of the 2011 model year: Xperia arc, Xperia arc S, Xperia mini, Xperia mini pro, Xperia PLAY, Xperia neo, Xperia neo V, Xperia pro, Xperia ray, Xperia active, as well as Sony Ericsson Live with Walkman.



Android update from version 2.3.3 to 2.3.4 for the countries of the northern region has been available since the end of September. Many Russian owners of Xperia arc and some other phones were also able to install this firmware for themselves, but the current release, however, is a significant event: firstly, this is a global update, officially including Russia, and secondly, since September it has been further tested and some changes have been made.

In short, with the new software (4.0.2.A.0.42) you will receive:
  • Updated version of Android 2.3.4
  • Swype text input (in my experience, it works much better than most programs from the Android market, allowing really intuitive input)
  • Panoramic 2D and 3D photography modes
  • The function of taking screenshots was added to the menu of the power button
  • Gtalk support with video chat feature for all front camera smartphones
  • Enhanced Facebook integration via Facebook inside Xperia version 2.0
  • 16x zoom when shooting video
  • DLNA standard support
  • The updated view of the folders on the desktop of the phone, which now contain the icons of four applications in which they are ‚Äúpackaged,‚Äù has become clearer where it is.
