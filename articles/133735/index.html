In the latest update of Xperia smartphones appeared support for WebGL

In the latest Android update for Xperia, Sony Ericsson added support for the WebGL graphics engine, thus becoming the first Android smartphone maker to support 3D movie rendering in the browser via javasctipt - good news for owners of Xperia 2011 model year smartphones.

A small demo video of WebGL using the example of Sony Ericsson Xperia arc:



Unlike other 3D web technologies, WebGL does not require special plug-ins; 3D graphics are generated directly in the browser by means of hardware-based graphics accelerators. WebGL is one of the HTML5 modules that allows you to write all the same things that you could write in flash before and explored as a potential platform for developing online games . All modern desktop browsers already support WebGL graphics playback, but in the mobile world, Sony Ericsson is among the pioneers, and the first among the manufacturers of mobile devices on Andorid.

One of the applications that allows you to test WebGL support, which I managed to find, is a generator of 3D models of GLMol molecules:



To learn how to develop applications using WebGL, we recommend visiting, for example, the Learning WebGL or the Khronos WebGL Tutorial . The technical side of the issue is covered in more detail in the Sony Ericsson developer blog .
