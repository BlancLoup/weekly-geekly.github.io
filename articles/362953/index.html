LinguaLeo
If you like to receive information on English-language sites, then there is no application for translation better than LinguaLeo. At the same time, you will not have to open the website of the service, or any other translator. All you need is to have an account in LinguaLeo and download the extension for your browser on a special page.

TranslateTab
LinguaLeo is good when you read articles, but if you translate texts or write your own in foreign languages, then TranslateTab will be of great help.



The application works on the basis of the Google Translate API and is located in the OS X system line. Its main function is to save the user from having to open the translator's website in the browser and make numerous switches between windows.

Dragondrop
It happened that you need to transfer a file from one folder to another or send it to Skype, and there is simply no space on the screen to open both windows. This inconvenience often occurs on laptops with a screen diagonal of 11 "and 13".
DragonDrop solves this problem by creating an intermediate measuring buffer on top of all windows. You put a file in it, switch to the desired application and "drag and drop" the file from the buffer.



This method works with folders and several files, and even allows you to save images from websites.

PuntoSwitcher
This utility is familiar to many. Many people dislike and criticize it, and yet it is one of a kind and provides really useful functionality. At first, the PuntoSwitcher can change the layout on the wrong words or not at all. The problem is solved by entering the word in the dictionary. To force a transfer or cancel it, you can use the option key. You can translate all the text by selecting it and also by pressing option.

Calcbar
Standard calculators in OS X are not always convenient. The one in Dashbord does not support brackets, the one in the standard set of applications, may seem cumbersome to many. If you need to do a couple of small calculations, then this utility is undoubtedly for you.



There are many analogues with more advanced functionality. However, CalcBar has two advantages: the application is free and has a clear, not cluttered interface.
