Ubuntu Home Server Edition

Inspired by the Windows Home Server , one of the members of the Ubuntu community, offered to implement something similar. His idea was well received, which led to the announcement of the project, which was called Ubuntu Home Server Edition .

As stated in the discussion on the forum , this project will be a good addition to the existing desktop and server implementations of Ububntu .

At the moment, the project is only in its infancy, an interested community is being created. All information about the project is available on its official website .

Project Goals


Create a system that allows people to administer their home network. It (the system) will allow you to save your music, movies and other data in one place, allow you to access this data from the Internet, and will also provide backup data, all of your computers.

This product is being implemented by the community, i.e. it has no official support. Therefore, he needs your knowledge, ideas and experience.
