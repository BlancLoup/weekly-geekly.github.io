nanoCAD 3.0: exit June 15, 2011


We were once called Musketeers in previous discussions. I liked it‚Ä¶
So: "One for all ..." "And all for me! ..".

We worked on the new version for half a year: we wrote new functions, collected the wishes of users on the forums, on social networks, in personal conversations. About 3 months ago, external beta testers started running in the product under actual design conditions. How much all we have corrected and redone. For example, the print function has been rewritten three times! And now we would like to tell Habra residents about the innovations that are awaiting users in the new version.


The development of functionality related to printing drawings was one of the main wishes of users. Let's start with it ...

As usual, users wanted everything at once and contradictory: ‚ÄúWe want both in Compass \ we want as in AutoCAD‚Äù, ‚ÄúWe need to be simple‚Äù, ‚ÄúMust satisfy everyone‚Äù. In particular, they asked for printing from the preview window, support for sheets of non-standard formats ... And, of course, a more convenient printing process as a whole.

In the process of painstaking work, a single print dialog appeared where you can configure all the parameters using a single window: printer / plotter settings, paper size and orientation, data output scale, print style. And, of course, remember the settings for their use in the future. As a result: the output of the drawings will now be much more convenient and easier - everything is at your fingertips, without any extra clicks, directly from the Print dialog (Fig. 1).


Fig. 1. Print Dialog

When specifying the print area, the screen displays the size and orientation of the sheet (blue frame), the border of the print taking into account the printer / plotter margins (blue dotted frame), and the physical dimensions of the sheet (yellow lines). Pay attention to this functionality - you have a question how to implement the layout of the drawing on sheets of a smaller format? In nanoCAD, this is easy to do!


Fig. 2. The visual location of the print area on a sheet (s) of paper

And, of course, we could not go past the Print button in the preview window: now the customized sheet is sent to print with one click (Fig. 3) - a small button, and how many broken copies.


Fig. 2. The visual location of the print area on a sheet (s) of paper

Batch printing will be convenient when working with large documents. If a set of drawings is created from one document, then by calling the Batch Print dialog (Fig. 4), you can create a task for printing the drawings: specify the list of output documents (1), specify the required number of copies (2) and send a batch print task (3 ).


Fig. 3. Dialogue Batch Printing

Do you want to see features that are not available, for example, in AutoCAD, but they were made at the request of our users? There are two of them in print: 1) the ability to print from the space of the model of two print areas simultaneously; 2) the ability to take into account the thickness of the lines, if the printed frame passes along the lines of objects.


Fig. 4. The ability to print from the space of the model of two print areas simultaneously


Fig. 5. The ability to take into account the thickness of the lines, if the printed frame passes along the lines of objects
