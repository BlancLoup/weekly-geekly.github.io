Introduced an updated collection of Metro 2033 and Metro: Last Light on Linux, Windows and new consoles

On May 23, Friday, Ukrainian 4A Games and German Deep Silver on the eve of the E3 exhibition announced the Metro Redux compilation, which will feature post-app-type first-person shooters Metro 2033 and Metro: Last Light with improved graphics, new features and all previously released updates. The set will be available in summer on Xbox One and PlayStation consoles, as well as for Windows users. A release for GNU / Linux and SteamOS is expected later this year. Pre-order can be issued now.



Metro 2033 Redux will be re-released on the latest version of the 4A Engine graphics engine used in Metro: Last Light, which will add to the game a dynamic weather change, increased detail and character animation, as well as improve the model of destruction, lighting complexity, effects of smoke, fire and other particles . In the re-release of Metro: Last Light Redux, new melee animations and all previously released add-ons will be added, adding 10 more hours of gameplay in a single game. And at the request of fans in the game, the ability to "Check inventory" and "Check hours."

Also, both games will get the "Survival" and "Spartan" modes with additional difficulties in the game (with clever opponents, long recharging and fewer resources) and a game with fewer obstacles, respectively. Advanced players will be returned the Ranger mode with the auxiliary interface disabled.

Since yesterday, you can pre-order the Metro 2033 Redux or Metro: Last Light Redux for 449 rubles, or place a pre-order for both games for 899 rubles in the digital distribution service Steam. In the same place, owners of previously released editions will receive a 50% discount on reissues, as well as a 25% discount on the set of both Redux games.
