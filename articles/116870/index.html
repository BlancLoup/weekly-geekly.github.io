New version of ASP.NET AJAX Control Toolkit released



On the official page of the AJAX Control Toolkit project there is a new version of this set of controls. AJAX Control Toolkit is an open source project that offers ASP.NET developers dozens of server-side controls working with ajax requests.

The new April version of the AJAX Control Toolkit (Version 50401) has the following changes:
  • fixed bugs and improved compatibility of controls with Internet Explorer 9;
  • The RoundedCorners and DropShadow extensions now take advantage of browsers that support the appropriate CSS3 properties for rounding and casting shadows on webpage elements;
  • Added new jQuery test project to better cover AJAX Control Toolkit with tests.
