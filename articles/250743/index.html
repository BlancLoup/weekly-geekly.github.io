Weekly assembly Vivaldi 1.0.105.7

Hello!

Today, the weekly Vivaldi browser build has been released under the number 1.0.105.7. In addition to the corrected bugs and the gradual improvement of the browser, in this assembly appeared some long-awaited user features.

First of all, in the test mode, a bookmark panel has been added, on which you can place your links, specifying the appropriate folder from the existing ones in the settings panel or creating your own. You can also change the placement of this panel by moving it to the bottom of the browser window. Naturally, this panel can be turned off completely if it is not needed :)

Also, this assembly is offered in the 32-bit version for Linux.

By popular demand, we added the ability to open bookmarks or websites from the Express Panel in the background by clicking the middle mouse button (or wheel).

Another good news for those who like to control the browser using the keyboard: we added the function of restoring the recently closed tab using keyboard commands (CTRL + Z and CTRL + SHIFT + T).

And finally, the list of languages ‚Äã‚Äãinto which the browser interface has been translated has been greatly expanded. Now the list of localizations has 22 languages, and the existing ones have been added:

It was possible to expand the list of localizations (and improve the quality of the translation of all previously available) thanks to the active help of voluntary translators from among the users of the Vivaldi browser, for which many thanks to them from all of us.
