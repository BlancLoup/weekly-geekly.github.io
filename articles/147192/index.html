Learn Github in 15 minutes

TryGit is a Codecademy - style interactive Github tutorial. The course consists of 25 mini-lessons that introduce the user to basic concepts and basic git commands, while the student himself enters the commands into the console, looks at the result and reads the accompanying inscriptions explaining what is happening.



This is how education in schools should be in the future!

Of course, the 15-minute introduction will not replace a full-fledged textbook. For training, you can recommend the free Pro Git book and the Getting Git video tutorial .
