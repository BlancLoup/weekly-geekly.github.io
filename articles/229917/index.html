We program for iPhone and iPad. 3rd ed



It is quite obvious that, despite the swift, full-scale onset of the Swift era, Objective C will not lose relevance for quite a long time. Therefore, to your attention we present another book of the series Head First: We program for iPhone and iPad. The name speaks for itself - this is a traditional series of detailed and interesting tutorial on how to program in Objective-C, design interfaces, create applications, work with the iPhone SDK tools (the development environment described in the book is Xcode5). All information is presented in a conversational, unobtrusive manner with a large number of illustrations and humor, which is typical of all Head First books. And don't be afraid of the classic facepalm on the cover - the story is really intelligible and fascinating.


The amount of documentation and articles on Swift language is growing, but the main book - Apple's official documentation is still in the process of translation. So this edition will also be useful for those who want to make up for gaps in programming for Apple products before switching to Swift, and to understand the organizational structure of this challenging and interesting platform.
