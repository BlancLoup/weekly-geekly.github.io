jquery-animateNumber - plugin for animating numbers

jquery-animateNumber is a jQuery plugin that animates numbers just like on stoloto.ru/rapido .

What can "out of the box"


  • simultaneous animation of numbers with different properties (like $ .animate);
  • animate numbers from smaller to larger and vice versa;
  • animation of numbers with automatic separation of digits (you can specify your own separator);
  • number animation with its animation pitch function.


Example 


<p>Fun level <span id="fun-level" style="color: red; font-size: 0px;">0 %</span>.</p> 

 //  $.animateNumber    , //   $.animate,    //  'number'  'numberStep' $('#fun-level').animateNumber( { number: 100, color: 'green', //  jquery.color 'font-size': '50px', easing: 'easeInQuad', //  jquery.easing numberStep: function(now, tween) { var floored_number = Math.floor(now), target = $(tween.elem); target.text(floored_number + ' %'); } }, 1800 );

Requires jQuery 1.8.0 or later for work.
