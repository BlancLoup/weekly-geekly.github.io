Yandex.XML learned to look for pictures

The Yandex.XML service, which allows displaying Yandex search results on its website in an arbitrary design, has been enhanced with a new feature. Now, along with searching for web pages and files in various formats, you can also search for images, as in Yandex.Images. In addition, in the results of a regular XML search, you can now display one of the most relevant images, as it happens in the search results on Yandex.

XML queries for image search are counted just like regular XML queries. The free limit , which was raised in the spring to 1000 requests per day, is defined as the sum of both types of XML requests.

In general, XML image search is similar to web search (see the documentation ). Of course, when searching for images there are specific functions, some of them are listed below.
