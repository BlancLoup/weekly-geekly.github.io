Details of cooperation NVIDIA and Opera Software

Earlier in the news already passed information about this event, but it was all, rather, a leak of information than an official statement. Today, finally, the press release of Opera Software has appeared, in which some interesting details of the new collaboration between browser developers from Norway and NVIDIA are announced.

So, as it was known before, the developers of NVIDIA decided to use Opera 9.5 Software Development Kit (SDK) in the development of software for their new chipsets. Now it turns out that, in general, it is not a matter of development tools. More precisely, in them too, but this is only the external side: the Opera SDK will be used to create a specialized optimized version of the Opera-9.5 browser, which will be pre-installed on mobile devices running Windows Mobile and Windows CE, developed using the new family of chipsets - NVIDIA Tegra .

As a result, users of such devices will receive a full-fledged Internet with improved support for JavaScript, vector graphics and video content. In other words, the collaboration between Opera Software and NVIDIA will lead to hardware acceleration when working with heavyweight Internet components. According to the plans of the developers, such devices should be on the shelves in 2009.
