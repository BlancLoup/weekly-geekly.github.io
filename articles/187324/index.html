New for web designer for June 2013

New useful things for web designers for June 2013. The rest of the collections are available on the tag " new for the web designer "

Instruments




Red Pen - the service allows you to easily get feedback on your work in PNG format: after downloading a picture, any user can leave their comments to the file.




Codifferous - a service for inspecting and improving your code



Bounce is a plugin that allows you to integrate the Dribble service into Photoshop, Illustrator, InDesign or Fireworks.



Colourcode is a tool for choosing a color scheme. Good competitor kuler'a from Adobe .



Pure - a set of lightweight adaptive CSS elements for the site, a good alternative to heavyweight frameworks.

Videry - constantly updated base of training and simply useful videos for web designers
