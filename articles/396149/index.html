Using the function "Restricting access on a schedule" on the equipment Draytek

Task: the company wants to establish a rule prohibiting its employees from accessing the Internet during working hours from 7:00 am to 6:00 pm. The rule is valid on all days of the week (from Monday to Friday).



Consider the installation of the schedule on the example of the Vigor 2110 Series
1. Log into WUI Vigor 2110 .
2. Set the System Maintenance Time >> Time and Date time standard.



3. Open Application >> Schedule.



4. Click Index 1. Set the time - 0 (12:00 o'clock at night), the length of time - 7 (hours). Then select Force On (Activate) in the Action Field.



5. Select Index 2. Set the time - 18 (18:00), Duration of time - 6 (hours). Then select Force On (Activate) in the Action Field.



6. Select Index 3. Set the time - 7 (7:00 am), the length of time - 11 (hours). Then select Force On (Activate) in the Action Field.



7. Now, open Internet Access >> PPPoE (Internet access >> PPPoE) (or another connection mode), enter Call Schedule Index 1 ~ 3).



8. Now the work schedule of the Internet is fully configured.
