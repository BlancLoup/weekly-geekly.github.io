Actually, in the sketch, comments seem to be sufficient, and the video shows everything, but in a few words about the timer operation: the Reset button drops the timer value and turns off the screen backlight. The ‚ÄúSetup‚Äù button for each press increases the timer value by 1 minute. You can not press it all the time, but just hold it down. Then the timer will increase every second for 1 minute. For better visibility, in the second line of the screen, the last character of each even second lights up "*", and each odd number is replaced with a space (the star goes out).
