Need wood for Vista? So take it!

The RadarSync site (in the English part of it) has a firewood section for Windows Vista coming out tomorrow. And, surprisingly for the English version of RadarSync, all the firewood can be downloaded for free. Probably, Microsoft, in terms of compensating for the money, has stomped up, since IXBT has made this service free.
So, if you suddenly want to buy Windows Vista, then you can not worry about drivers. Yes, they, in principle, and on the DVD with Windows were enough (it turns out that Microsoft sometimes has something to praise).
The site has drivers from the following manufacturers:
ATI, BENQ, C-Media, Creative, Crystal, Dell, ESS, Intel, Logitech, Marvell, NVidia, Philips, PVD, RealTek, SIS, SigmaTel, SMC, VIA, Marvell, Yamaha
