PHP 7.2.0 released



Yesterday, November 30, 2017 PHP development team announced the release of PHP 7.2.0
This is the second minor release in the PHP 7 series.

PHP 7.2.0 includes some improvements as well as new features.

  • Changes in array / object mapping, numeric keys
  • Changes in counting types
  • Object type declaration
  • HashContext as an object
  • Added algorithm Argon2 in the password hashing function
  • Improvements SSL / TLS Constant
  • Removed the Mcrypt extension https://wiki.php.net/rfc/mcrypt-viking-funeral
  • Added sodium extension
