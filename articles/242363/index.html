Mozilla released the first developer browser.

As previously reported , Mozilla has released the first browser, which was created specifically for developers. Despite the fact that the official press release on the release of new items will be available on November 10, you can try the new browser now.



New information about Firefox Developer Edition has become known:

  • Firefox Developer Edition will replace the already existing Firefox Aurora development channel. Thus, innovations from Firefox Nightly will fall into the assembly for developers. There will also be a six-week browser development cycle: Nightly - Developer Edition - Beta - Release. Thus, the developer will have 12 weeks before the innovation gets into the release;
  • The new browser will use a separate user profile, which will allow it to run simultaneously with Firefox;
  • Updated interface that accelerates access to developer tools. The default dark theme is included, but the browser supports both switching to a standard theme and installing third-party themes and extensions;
  • Experimental development tools. The new browser includes development tools that are still at an early stage of development. One of these was the Firefox Tools Adapter , which allows you to use Firefox development tools for debugging in other browsers (Safari on iOS, Chrome on Android). In addition, the composition of the new Firefox included a useful tool WebIDE , which has already been reported earlier.

More information about the innovations and Mozilla browser development plans can be found on November 10, 2014.
