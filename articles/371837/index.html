Top 10 HTML5 Games

In recent years, HTML5 and JavaScript are actively gaining popularity among developers of indie games. The reason for this choice is obvious - the potential of two applications in the production of games. HTML5 is now considered a serious gaming platform. Next, for example, 10 top games using HTML5.

1. A Grain of Truth



The game was developed by the brothers Rudowski (Poland). The essence of the game is simple - a young traveler in search of a magician. This point-and-click game (quest) is full of adventures, with very beautiful graphics that completely immerses gamers in the game world. The game is written using jQuery, which once again shows that this framework can be used not only for creating websites, but also for entertainment.

2. Save the Day



A simple adventure game in which you are a lifeguard. With the help of a helicopter you need to save the survivors for a certain period of time, picking them up on board the helicopter. As time passes (if not all poor people are saved), your rescue helicopter will explode.

The game was created on the HTML5 open source Turbulenz game engine. Turbulenz engines provide a wide range of functions and services for creating games (fault tolerance, asynchronous loading, scalability). In general - the engine for creating games that use web technology, rather than native code. Turbulenz is a social gaming game engine using HTML5, WebGL JavaScript. Its capabilities allow you to make exciting real-time 3D games with physics.

6. BrowserQuest



Multiplayer RPG game, BrowserQuest. To create the graphics of the game used the same - HTML5 canvas. Since the game was created in HTML5 / JavaScript, it is available through most browsers and platforms. The plot of the game: a young scout visits various dangerous places in search of adventure and thrill. Technologies such as web worker (defines the API for creating background scripts in web applications without blocking the game process itself), localStorage (saves the game history) are used.
