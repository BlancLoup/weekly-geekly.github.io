That the child is not lost

Amber Alert presented an interesting and useful device for caring parents called Amber Alert GPS 2G . This device allows you to keep track of the kids.


The main function of the device is a GPS beacon. Put it on the child (or threw it into the backpack) and be sure that the satellite will find it. Location can be found on the phone or track via the Internet. Nevertheless, in Amber Alert they pushed and fastened a couple more possibilities. For example, tracking the route of movement with notification of arrival at the end point or the creation of a "safe zone" - a specified perimeter, the output of the beacon for which will be equal to an alarm. The beacon itself is equipped with a button, clicking on which will mean that the child needs help.

Parents can set all the device settings using the phone.

Worth Amber Alert GPS 2G a lot - $ 380. In addition, you still need to buy a service package, the monthly cost of which ranges from $ 9.99 to $ 59.98. Calm is expensive.
