DEV Labs 2017. Online conference for .NET developers. March 4

We are opening a new conference season Soft Labs with a development event on the .NET platform. We have collected a good set of useful cases and want to share it with you on Saturday, March 4th.

The conference program includes 4 reports:

  • Performance optimization patterns. Sergey Teplyakov will talk about current trends, patterns and best practices for creating highly loaded managed applications.
  • Complex queries using IQueryable and Expression Trees. Ivan Koshelev using practical examples on the Entity Framework will show the differences and features of popular tools.
  • Chatbots development. Alexander Krakovetsky will share his experience in developing intelligent messenger bots using the Microsoft Bot Framework, Microsoft Azure, and other ML / NER systems.
  • Andrei Gordienkov will talk about the use and testing of Reactive Extension. As they say, not UI one!

The conference starts on March 4 at 11:00 Moscow time.
To participate in the conference you need to register . Participation is free.

In order to take part in any Soft Labs conference, it is enough to have a computer or mobile device connected to the Internet. No special software needs to be installed. Platform plugin will load in a few seconds.
