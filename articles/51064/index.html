Windows 7 Distribution Line

Microsoft announced information on the composition of various versions (SKU, Stock Keeping Unit), which will ship the Windows 7 operating system. The ruler is organized according to a simple principle: each following distribution kit is an improved copy of the previous one, that is, it has all its functionality, as well as additional features ( they are listed).

1. Windows 7 Starter (for netbooks): no more than three running applications.

2. Windows 7 Home Basic (only for developing countries, nowhere else to go on sale): added a live preview in the icons, improvements in the graphical interface, enhanced support for networks (sharing wireless connection and internet connection), Mobility Center.

3. Windows 7 Home Premium (for private users): Aero Glass interface and advanced navigation with windows, more supported media formats, improvements in Windows Media Center and media broadcasting, multitouch, handwriting recognition.

4. Windows 7 Professional (for small business): connectivity to a managed group with Domain Join, data protection and network backups, Encrypting File System, printing to different printers, depending on location (Location Aware Printing function).

5. Windows 7 Enterprise (for medium and large businesses) and Windows 7 Ultimate (only as a paid upgrade of the two previous SKUs): BitLocker data protection on internal and external drives, DirectAccess for transparent connection to corporate networks on Windows Server 2008 R2, support BranchCache in networks on Windows Server 2008 R2, a ban on the launch of certain applications using AppLocker firmware.

Microsoft considers two distributions to be the main ones (they are highlighted in bold in the list), and they will be the most actively promoted for most users.

Prices and terms of sales have not yet called.
