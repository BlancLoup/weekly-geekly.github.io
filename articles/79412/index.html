misultin - a lightweight web server on Erlang

This summer in the regiment of light web servers on the Erlang arrived. Roberto Ostinelli published his development - misultin . This is a library for creating lightweight high-speed HTTP servers. In this regard, the project resembles mochiweb , but the emphasis is even more on speed. According to the author's tests , misultin overtakes mochiweb by about 50-60% while working with 100-200 clients.
The functionality is still poorer, but there are basic things for creating services: sending simple answers, files, RESTful support, convenient sending of answers in parts (streaming). The author plans to further develop the project to add new features.

Colleagues, perhaps, who dealt with this library? What reviews?
