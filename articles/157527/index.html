OLinuXino A13: a miniature open-source computer for $ 60



The Bulgarian company Olimex recently introduced a mini-PC, which was called OLinuXino . At first glance, OLinuXino is very similar to its ‚Äúcolleague‚Äù, the Raspberry Pi. But the Bulgarian mini-PC has a faster processor, more ports, plus the whole project fully complies with the open-source standard, including both hardware and software.

This means that any developer can download the source code , and modify any driver, any program for this mini-PC. In addition, you can download source files for hardware, and if necessary (and the corresponding skills and capabilities) modify any part of this PC. A third-party developer based on OLinuXino can create your own device, and sell it.

At the moment there are two versions of OLinuXino. Both are priced at slightly below $ 60.

A13-OLinuXino includes Allwinner A13 ARM Cortex-A13 processor with 1 GHz core, Mali 400 graphics system, 512 MB of RAM, 3 USB ports, memory card slot, VGA output, audio output, UEXT connector for connecting other modules and GPIO connector for LCD, flash and other devices.

As the OS for this mini-PC, Android and Linux distributions will work.
