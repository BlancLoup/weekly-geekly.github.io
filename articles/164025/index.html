NVIDIA Nsight Tegra preview release

I did not notice the news about this event. And 2 months have passed since the release of the preview version.
For those who do not know what it is - a link to the introductory article.

Feature List:
  • Import existing projects into Visual Studio.
  • Manage native android projects as usual.
  • Building native code for Android projects using vs-android, ndk-build or make files.
  • Parallel compilation for both file and project.
  • Increase the performance of native (C / C ++) Android code by 20-30%.
  • Improved NEON support.
  • Link-time optimization (LTO).
  • Native support for multi-core Tegra CPU in GDB.
  • User-friendly debugging of Java and native C / C ++ code.
  • Using familiar debugging tools VS (Locals, Watches, Memory and Breakpoints windows)
  • Specialized debugging tools for the Android OS (such as logcat)


In order to download a preview, you must be registered with the Tegra Registered Developer Program. Registration there also requires manual appruva. Because post a direct link .
