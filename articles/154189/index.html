Published free e-books on Windows Azure technology from the Patterns & Practices team

Good news! The electronic books (EPUB, MOBI, PDF) on Windows Azure technology have been published in the TechNet Electronic Library ( E-Book Gallery for Microsoft Technologies ).

Application Migration to Windows Azure


Application Migration to Windows Azure, 2nd Edition
Reference \ Patterns & Practices Handbook

This is the first book in the patterns & practices series on the Windows Azure platform. The book examines various aspects (changes in architecture, cost, monitoring and diagnostics of the system, etc.) of the migration of a local application to the Microsoft cloud platform - Windows Azure. The book describes the migration by the example of the aExpense application, this is an ASP.NET application for approval of expense reports. First, the book will help to get a good idea of ‚Äã‚Äãthe issues that need to be considered when migrating an application to the cloud, and, secondly, the book provides answers to a number of practical tasks that are relevant not only when migrating an application, but simply when developing any cloud application.

Epub | MOBI | PDF

Application Development on Windows Azure Platform


Cloud Application Development on Platform 2nd Edition
Reference \ Patterns & Practices Handbook

This is the second book in the patterns & practices series on the Windows Azure platform. The book examines various aspects (dedicated or multi-tenant architecture, using CDN, unit testing applications and mock objects, data access layer, MapReduce algorithm, integration of LiveID and other providers with, etc.) developing cloud applications and services on the Windows platform Azure.

Epub | MOBI | PDF

Build hybrid applications on Windows Azure platform


Build hybrid applications in the cloud on the Windows Azure platform
Reference \ Patterns & Practices Handbook

This is the third book in the patterns & practices series on the Windows Azure platform. The book shows how you can use powerful infrastructure services that are part of Windows Azure to simplify development, integrate components of hybrid applications, including cloud and local components, as well as third-party systems: user authentication and authorization scheme, integrated application monitoring, messaging templates and the like. It also describes how to ensure maximum security, performance, scalability and availability. This guide focuses on architects, developers, and information technology (IT) professionals who design, build, or use applications and services that work in the cloud or interact with it.

Epub | MOBI | PDF

Creating resilient and resilient cloud applications (Enterprise Library for Windows Azure)


Creating resilient and resilient cloud applications: Enterprise Library Integration Package Developer's Guide for Windows Azure
Reference \ Patterns & Practices Handbook

This book demonstrates how you can use the Enterprise Library Integration Pack for Windows Azure in an existing Windows Azure application to improve the reliability, manageability, scalability, stability, and extensibility of the application. In particular, issues of automatic scaling in Windows Azure (Autoscaling Application Block), error handling of exceptions and minimization of their impact on system components (Transient Fault Handling Application Block), etc. are considered.

Epub | MOBI | PDF | raw materials

Migrating data-processing applications to Windows Azure


Migrating data-processing applications to Windows Azure
Reference \ Patterns & Practices Handbook

When transferring an application from a local platform to the cloud, several problems should be solved. This book describes recommendations for migrating local applications that are based on working with data. In particular, the book describes the issue of migration planning, for example, ways to reduce idle time during a transition; various migration considerations are considered: based on Windows Azure virtual machines (IaaS-based approach) or cloud services (PaaS-based approach); provides recommendations for migrating code from SQL Server to SQL Database and improving the performance of the SQL Database, as well as ensuring high availability of the solution.

Epub | MOBI | PDF | raw materials
