DIY alarm system

Everyone is concerned about their safety. You can ask the design and installation organization to make the project, and then on it installation. But it is uninteresting and stuck in the budget. So consider the alarm with their own hands. Will consider the fire alarm system (fire alarm) and access control system (ACS) in the apartment and house (cottage)

Design


In the beginning it is necessary to determine the project. The project should be, if it is a public place, to which our house does not apply. But the sensor circuit, where and where it is connected is desirable for future service.

Now we are determined with the tasks. The number of sensors and wires.

Fire Loops

There are address and simple. It is more convenient to use addressees, but the output is more expensive (usually a computer with the appropriate software is used), so we do simple.
To ensure protection against false positives, we put at least two chimneys in each room, except for the bathroom and the kitchen. There are no fire alarms in the bathroom, but flame detectors are made in the kitchen. Each detector controls the area of ‚Äã‚Äã5-10 square meters.

Consider a small example: if a room is 5 square meters, then there are two sensors, if 45 square meters, then there are 5 sensors evenly spaced around the room (the sensor has a range of about 3.5 meters). It does not place a pair of parking brake (for each exit separately). Handbrake - manual sensor, push the button and fire alarm is triggered. This is necessary if something caught fire, and the alarm did not work, especially if you have an agreement with firefighters. Press the button quickly, that call and call the address when you need to try to localize the fire and reduce losses.

Sensors work around the clock.

Wires

The power and control of the loop goes through two wires. For example, KSVV 2h0,5. But I recommend to make all the same KPSEng-FRLS 1x2x0.5 or KSEVEng-FRLS 2x0.5 (well, or another similar one marked FRLS), which differ in that the second is a twisted pair.

Cables FRLS, except for the certificate, are fire resistant. After all, if the fire started with cables, then the fire alarm may not work or it will go into fault mode. The cable is more expensive by 2-5 times, but they need not so much and this is done for several years (the warranty period is 5 years).

FRLS - a mandatory requirement for firefighters.

Connection

Sensors are connected in series, the connection diagram is in any instruction (both to the sensors themselves, and to the receiving devices).

Security Loops

We provide two circuits protection (SP required). The first is the perimeter of the house or apartment, this includes doors and windows, the second internal - motion sensors. This is convenient if you live in a house, then you can switch on the perimeter security sensors for the night and sleep better. It would be more correct to make loops through the rooms, so that it was clear in which room the alarm was triggered, but this way more loops are needed and this should be taken into account.

Wires

These sensors are considered active and they need 12V power. Four wires are used, such as KSVV 4x0.5 and the like. You can use UTP 2x2x0.51.

Connection

Sensors are connected in series.

Installation


Everything is simple, there should be no problems. The main thing is to follow some rules:
  • Wires away from power (220V).
  • Through the wall additionally in the cambric.
  • The instructions for the sensors have tips on placement and connection.
