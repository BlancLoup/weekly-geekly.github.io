Today ends the main phase of support for Windows 7

On the calendar of January 13, which means that the main support for Windows 7 is ending today . Now users of this operating system depend on extended support, which will end only after 5 years, on January 14, 2020.

According to Microsoft policy , this means that security patches will continue to appear until 2020, but now even for licensed users of Windows 7 technical support will be paid.

Five years is quite a long period of time. Most likely, by 2020, work on the next for Windows 10 OSes will come to the final stage. Or maybe Microsoft by then would abandon the practice of major releases.

It is planned that Windows 10, combining Windows Phone, PC and embedded versions of Windows operating systems will be released this year.

Today, more than half (55%) of Internet users use Windows 7 for their work. This is more than the Windows 8 and 8.1 - few people want to change from the "seven".

The popularity of Windows 7 provided a more stable and fast work than its predecessor, Windows Vista. Many users have decided to replace XP with Windows 7, although a few percent remain true to the operating system, the period of support for which finally expired last year.
