Serious Sam shooter developers have posted the source code of the game



In connection with the anniversary of the release of the first-person shooter Serious Sam, which will be held in March 2016, the game developers from the Croatian company Croteam decided to release the full source code of the game . The code was specially prepared for the study, providing detailed comments.

The version of the engine, now available to the public , is officially called Serious Engine v1.10. To release the source code was prepared by one of the programmers Croteam, Vyacheslav Nikitenko. He told the following about it:
From a historical point of view, the Serious Engine is important both for Croteam and for me personally. Before starting to work with source codes, I made several mods for the game - and it was a great tool for learning. And today it is even better!

Download the source code and head to the SeriousZone support forum , where there is both a lot of tutorials and a great community.
