Firefox 22 release

Firefox for desktops has become available for download. What's new in this version:

  • Firefox now uses the text scaling options specified for the display, which allows it to increase the text size on high-resolution displays (available only for Windows).
    To scale to fit your needs in about: config, find the layout.css.devPixelsPerPx parameter and set the value you need;
  • In OS X, the file upload indicator now appears on the Firefox icon;
  • A new section ‚ÄúServices‚Äù has been added to the add-ons manager to manage add-ons for social services;
  • Added support for H.264 video and AAC and MP3 audio for Windows Vista;
  • Integration of the optimizing OdinMonkey module with the implementation of support for Asm.js;
  • CSS3 Flexbox and WebRTC are enabled by default;
  • Web Notifications API support;
  • The ability to change the playback speed of audio and video content, access to which is organized through the means of HTML5;
  • Improved WebGL rendering performance - for this , the technique of asynchronous update of layers specified via canvas is used;
  • Reduced memory consumption and time spent on rendering images (bugs 1 and 2 ). Support for multi-stream image decoding is also provided ;
  • Text files are now displayed with word wrap;
  • To ensure the safety of users, the object | Components | now not available to access from web content;
  • HTML5 support for time and data ;
  • The Pointer Lock API can now be used in windowed mode;
  • An implementation of the clipboardData API for performing typical clipboard operations from a JavaScript web application, such as cut, copy, and paste;
  • Two new tools have been added to the Inspector (Ctrl + Shift + I): Font Inspector and Paint Flashing ;
  • The about: plugins now shows the full path to the installed plugin. Before that, to find out the path to the folder with the plugin, you had to switch the plugin.expose_full_path
  • High resolution faviconok support in HiDPI mode (if such mode is present in the operating system);
  • Fixed problem with slow scrolling on some high-resolution touchpads.
