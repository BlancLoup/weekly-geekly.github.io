Connect to the Jump Start online event March 15: "Infrastructure as an Azure Resource Manager-based service"
On March 15, 2016, we are holding an online event of the Jump Start series ‚Äú Infrastructure in the form of a service based on Azure Resource Manager ‚Äù. Get connected, start at 11:00 Moscow time .

We talked about using virtual machines and Microsoft Azure networks more than once. However, now a new mechanism is available for managing IT objects in the cloud - Azure Resource Manager (ARM). ARM allows you to work with the resources of a solution, for example, a distributed application, as with a group. You can create, update, or delete solution components in one consistent operation. A declarative template is used to describe and deploy the required resources, and for more effective management after deployment, ARM provides security, auditing, and tagging functions. Moreover, a similar approach is implemented in the Azure Stack, which allows IT professionals, using the same template, to deploy resources both in the cloud and on the basis of the local infrastructure.

Connect to another online event from the Jump Start series to learn how to use ARM to access various Microsoft Azure services, what are ARM templates, how you can use them to deploy the necessary resources in the cloud or locally.
