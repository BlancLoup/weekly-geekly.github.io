Server Expert

Server-Eskpert is a free software product of the company ADVANSERV , which allows you to assess the correspondence between the main server parameters and the requirements of server applications.
This product was created so that the client could pick up the basic characteristics of the server for the tasks of interest, while not having sufficient qualifications in the field of server solutions.




The performance calculation is based on the following characteristics:
  • CPU performance
  • the amount of RAM
  • volume and performance of the disk subsystem.

Processor performance is calculated from the processor model (architecture, frequency, number of cores) and their number. The amount of RAM is an absolute value and is taken into account directly for a specific task. The characteristics of the disk subsystem depend on the number, volume and speed of the disks. Of course, we should not forget about the fault tolerance and availability of the server, but we decided to omit these parameters, allowing the user to choose the level that suits him.

Depending on all these parameters, the client can accurately determine the model and configuration of the server. We included all main server roles in the task list, some of them are implemented on Linux OS, others on Microsoft Windows Server products. Also, the user can select several levels of server load, specifying the ‚ÄúLevel of difficulty‚Äù parameter.

All dependencies were built by our engineers with many years of experience in implementing various solutions and operating server systems. If you do not agree with certain configurations, we are always ready for dialogue and we will be happy with your help to finalize Server-Expert .

The convenience of Server-Expert also lies in the fact that a user may not be tied to a specific model, but choose a configuration, for example, from an existing server. To do this, we specifically left the processors of the previous generation and the old disk interface. That is, you can both select a server for a task, and select tasks for a real server, which in your case serves ‚Äúfaithfully‚Äù. The same Server-Expert will help you in estimating costs when increasing server requirements.

For example , is it necessary to completely change the server when the number of users doubles, or is it enough to deliver a second processor and add RAM?

By selecting server parameters, you can determine: which of them are most sensitive to the required task; which resources should be selected with a good margin, and which ones do not particularly affect performance with increasing load.

This development complements the line of software products created by ADVANSERV specialists, such as Server Configurator , Server-Audit , Storage Configurator . For convenience, we have integrated Server-Expert into the Server Configurator, and now the user can check the specific server model for compliance with current tasks.



Server Expert is constantly updated and polished. Recently, our specialists selected specific configurations of server models for each task, and for the three leading brands: HP , DELL , IBM .
