Published the full program of the conference WebProfessionals in Samara



The WebProfessionals conference will take place very soon - on April 3 in the city of Samara. Today we are pleased to announce that the full program with all the reports has been published on the event website. You can find the schedule at the following link .

WebProfessionals visitors are waiting for reports on the following topics:

  • Windows Server 8 and innovations in IIS 8 for web projects;
  • work with projects, frameworks, CMS on ASP.NET/PHP/Python/Ruby/Node.JS on IIS and Windows Azure;
  • new versions of Visual Studio web development tools and ASP.NET platform;
  • stories about practical examples of high-loaded solutions;
  • cloud platform Windows Azure;
  • Special report by Denis Kortunov (Parcsis / Turbomilk) on Metro UI style design.

Entry to the conference is free! Hurry up to register .

In addition, at the conference you will be able to talk with leading experts from Microsoft and invited experts from different companies: Abbyy, 1C-Bitrix, Parcsis / Turbomilk, Inln, and the Drupal community in Russia. A full list of speakers can be found on this page .

Special guest of the conference will be Pranav Rastodzhi - a developer from Redmond, USA, who participates in the development of Microsoft web platform tools. Pranav will make three presentations on the topics: ASP.NET 4.5, Visual Studio 11, WebMatrix 2, Windows Azure and Node.JS. One of the Pranava sessions will be presented in the form of live communication with the audience.
