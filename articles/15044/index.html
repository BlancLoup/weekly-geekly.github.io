Google maps for Symbian phones.

Google has finally released a google maps app for Symbian Series 60 3rd edition smartphones. Unlike the Java version, the Symbian application consumes less memory and runs faster. GPS support is also announced in the Nokia N95 (it looks like external bluetooth receivers must be supported).
Traffic jams should also appear, but unlike Yandex.Maps, this feature is not issued as the main one, and for Moscow it does not work yet.
It also favorably differs from Yandex.Map by the presence of the route construction feature, the lack of advertising (with Yandex.Maps, it is already painfully obsessive) and rather fast work.
