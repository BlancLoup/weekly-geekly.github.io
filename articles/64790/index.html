Habra-editor of comments dva.nol

Apparently, the first version of the editor was liked by habrovans, which positively influenced my motivation for writing the next version :)
So Habrahabr Comment Editor 2.0 , a lot of new and interesting!



The extension now works in firefox versions from 3.0 to 3.6a1pre , unfortunately earlier versions are not supported. The first thing you might have noticed is external changes. Say ‚Äúno!‚Äù To boring browser buttons :) The extension has got a configurator through which you can make some display / behavior settings. Further, the points tell you what for and how to use it all.

Tagging



By default, only three tags are put: <b> , <i> , <u>
You can add new ones (with the Add more / Add more button) or delete unnecessary ones (with the Delete button next to the tag to be deleted).

Open tag / Open tag - Close tag


The Open tag / Open tag field contains the code that will be inserted before the selected text fragment, the Close tag field / Close tag, respectively, after.
Suppose we need to add the tag <hh user = " text "> , then in the first field we add " <hh user =" "and in the second" "> ". You can not only remove / edit IMG tags and URLs .

Hotkey / Hotkey *


In this field you can assign a hot key to call the tag. Just put the cursor in the field and press the desired letter. If you need to clear the field, you can do this with the Delete or Backspace key.
Note It works only in Russian and English layouts.


Display / Display *


Here you can enter the name of the button for this tag (to display in the form).

* If the Display / Display field is empty, then the button will not be displayed in the comment form, but it will be available by hotkey, BUT if the Hotkey / Hotkey field is empty, then the tag will not be added.


Modifier / Modifier



Here you can assign a key, which in total with the Hotkey tag will give the desired tag (clumsily explained ...).

Fast quote / Fast quote



You can enable or disable quoting (described below what it is).
When ‚ÄúQuick Quote‚Äù is enabled, a ‚ÄúQuote‚Äù button appears next to each comment:

Clicking this button adds the entire text of the quoted comment, the user name and a link to the original comment in the comment field. It looks like this:

Sweetie : if we selected the text in the quoted comment and clicked the ‚ÄúQuote‚Äù button, then only the selected text is quoted.

Fast post citation (NEW from version 2.1.1)


Now you can quote the text of the post itself! :) To quote you need:
1. Press the Ctrl key.
2. Select the desired text fragment from the post.
3. Release Ctrl (after you release the mouse button)
4. The text is inserted into the form in the tags <blockquote>
Note This feature is available only when Quick Quotation is enabled.
