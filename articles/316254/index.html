  • RFC: User defined session serializer ‚Äî It is proposed to make it possible to register your own serializer for sessions.
  • RFC: Debugging PDO Prepared Statement Emulation v2 - The proposal to emulate the prepared expressions directly in PHP for the convenience of debugging successfully passed the vote, but caused criticism and the author updated the implementation.
  • RFC: Deprecations for PHP 7.2 - In addition to adding new features, outdated ones are removed step by step. The next issue proposes to declare obsolete:

    • __autoload
    • $ php_errormsg
    • create_function ()
    • mbstring.func_overload
    • (unset) cast
    • parse_str () without second argument
    • gmp_random ()
    • (binary) type casting and b "" literals
    • each ()
    • assert () with string argument
    • $ errcontext argument for error handler

  • RFC: Abolish Narrow Margins - If the RFC does not imply major changes, then 50% + 1 votes are enough for a vote to be taken. However, it is believed that such changes can lead to adverse results. Therefore, it is proposed for any voting to raise the threshold to 2/3 of the votes.
