Bootstrap toolkit for creating web applications



Today, Twitter announced the launch of a new toolkit Bootstrap for creating web applications using CSS. The set includes styles for typography, forms, buttons, tables, arrows, etc.

The creators say that the service was launched as a framework for creating frontend applications. The set was developed during the first Twitter hackweek.

The advantages of using Bootstrap in using Less (more about which you can read here ), which, according to tweeters, is a more flexible and powerful pre-processor than regular CSS.
