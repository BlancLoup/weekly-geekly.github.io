Microsoft's participation in the O'Reilly Open Source Convention (OSCON)



Next week, the O'Reilly Open Source Convention (OSCON) conference will be held on open source projects.

Microsoft will once again take part in the conference and present the following reports:
  • CoApp: Bringing the Open Source Package Management to Windows - a project for managing packages of opensource-programs in Windows;
  • Build Mission Critical Cloud Applications on Windows Azure Platform using Open Source Technologies - a report on the development of cloud applications based on opensource technologies;
  • Private Cloud Deployment with Microsoft and Novell: Driving Transformational Architecture in the Enterprise - a report on the results of cooperation between Novell and Microsoft in their desire to create a cloud product for enterprises based on Linux and Windows Server technologies
  • Reactive Extensions For JavaScript - a report on reactive extensions for JavaScript from their creator, Eric Mayer, the functional programming guru;
  • Advancing Interoperability, Patient Safety, and Efficiency with the Microsoft Connected Health Platform Open Toolkits - a report on an open set of tools for building applications in a medical environment.
Announcement of participation made in this blog .
