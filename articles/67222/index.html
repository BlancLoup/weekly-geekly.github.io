The best plugins for Textpattern

Many of the visitors (readers) of my blog know that I prefer Textpattern to other similar engines. It happened and now I just do not want to pick the insides of another CMS. Textpattern suits me perfectly.

For some reason, quite a little is written on Habr√© about this wonderful CMS. Of course, it is not as popular as the WordPress, but still deserves at least a little attention from the respected community :)

Much has been written about the features of this engine. I'm not ready to write a great and convincing article about Textpattern. Especially since I'm just a confident user :)

Well, for those who suddenly decided to use Textpattern for their own purposes (or are already actively using), I provide a list of the best, in my humble opinion, plug-ins for this engine.

I did not call this post "10 plug-ins for Textpattern without which it is impossible to live" or "11 best plug-ins for Textpattern". For two reasons.
  • This list will be updated. Confident in this 100%. In my blog for sure.
  • Each site requires its own plugins. And it is hardly possible to find such a site that would blindly push all the plugins from this page to itself. Given the first point, of course.

Yes, I know that many plug-ins have alternative, and maybe even superior in functionality. However, here I will collect only those that I use personally. Or going to use.
All plugins are arranged in alphabetical order.



glx_gravatar [ info ]


Recently, it has become fashionable to attach an image to your email address - an avatar. To make this avatar picture shown in your comments.
The plugin itself shows the picture corresponding to the e-mail of the commentator. So register on gravatar.com and many of your comments on different sites will be supplied with your picture.
So it‚Äôs really nicer ...
PS The plugin has a small bug: if you write your e-mail in the comments using capital letters, your picture will not be shown. Principle to fix this bug is not difficult, but it is better to write e-mail in small letters. For reliability.
UPD: the plug-in developer fixes this bug in summer, so update :)

hcg_templates [ info ]


The simplest theme installer (templates). Simply exports or imports all pages, forms, styles.
I use it only to bring up the same design on a local host or transfer it from a local server to the Internet.
Quick and convenient.

ob1_pagination [ info ]


By default, in Textpattern, the pagination is simple: previous entries, following entries. This is ineffective in terms of SEO (few pages) and usability (you have to click a lot to view the penultimate article with a large number of articles). This plugin allows you to embed page navigation as in Google - ticking :)
The plugin has a lot of settings and is on all my sites :)

ob1_title [ info ]


Needed by every SEO plugin. Allows you to set a unique title for the site pages (title tag). A lot of settings and clear help will make your addiction to this plugin inevitable :)

rah_sitemap [ info ]


Another SEO plugin that creates a sitemap (meaning sitemap - an XML list of all pages on the site). Sitemap becomes available at a specific address. And, most importantly, it can be fed to search engines for early and proper indexing.

rss_admin_show_adv_opts [ info ]


Small, but extremely convenient plugin. Allows you to open all the additional fields of the article when writing. And now you do not need to do extra clicks to manually change the date, write keywords or set any other parameters of the article.
I put on all my sites.

rss_superarchive [ info ]


Required if you need to group articles by date or other features. Very convenient and high-quality plugin.
Very good for creating a unique site map: a page on which links to all articles of the site are grouped.

smd_query [ info ]


This is generally a plugin that extends Textpattern to infinity. It simply allows you to directly write queries to the database tables and display them anywhere on the page. I already wrote about him separately: ‚ÄúExtending the capabilities of the textpattern using smd_query‚Äù .
If you need to do something on Textpattern, you know the basics of SQL and you are too lazy to look for special plugins - this is your plugin.

tru_tags [ info ]


An indispensable plugin for creating a tag cloud. Easy to install and configure. For storing tags uses keywords. Almost all txp-shniki that I know for the tag cloud use this particular plugin.

wet_quicklink [ info ]


Recently opened this useful plugin. It allows you to conveniently insert into the article links to other articles of your site.
Easy to install, easy to use and indispensable in search engine optimization.
Why link to your previous articles? I will write about this in more detail sometime ... :) Too much to talk about.

Thanks for attention. Everyone who uses Textpattern in their work, please add this list with your best plugins :)
