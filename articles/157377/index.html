DeathCraft: instruction



Units:
  • 0 SCV: collects resources, builds buildings, produced in the Command Center. Both minerals and gases are produced in Refinery.
  • 1 Vulture: enemy combat unit, laser fire, speed, produced at Barracks.
  • 2 Wraith: a flying unit of a player, does not attack, flies, high-speed, makes a landing Dinosaur, is made in Barracks.
  • 3 Car: a player who does not make a unit.
  • 5 Robot: player‚Äôs combat unit, scrambled eggs, slow, produced in Barracks.
  • 6 Dinosaur: player's combat unit, does not attack, few lives, with an explosion blows down all units within the blast radius.


Building:
  • 0 Command Center: produces SCV, here they are extracted resources.
  • 1 Refinery: location of resource extraction using SCV.
  • 2 Barracks: the place of construction of combat units.
  • 3 Gun: combat building of the enemy.


Mouse control: selection of units and buildings with the left click, the command to go and attack with the right click. There is no area selection, but there is a selection of all units of the selected type on the screen using Ctrl.
Keyboard control:
  • Arrows: move on the map
  • H: following the selected unit
  • P: pause
  • Delete: delete unit
  • C + U + F12: cheat code for creating a unit. Then you need to enter two digits on the numpad - the unit code and (optionally) two digits - the player number. For example, C + U + F12, 0500 will switch to the robot creation mode using a mouse. To deactivate, press C + U + F12 again.


Tips:
  • Protect Refinery! First of all, the enemy will demolish this building.
  • If you did not have time to build an army, and the enemy has already attacked, then dinosaurs dropped from an airplane will save you.
  • Build a lot, build quickly.
