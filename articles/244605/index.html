Php


  • RFC: PHP 7.0 timeline - Release Plan for PHP 7 approved by vote. Overall, in the middle of next year, it is planned to complete the development of new features, and in October 2015 it is expected to release PHP 7 itself. An alternative plan with the release of intermediate version 5.7 was also considered.
  • RFC: Remove PHP 4 Constructors - Offer to remove support for constructors in the style of PHP 4, that is, functions with a class name.
  • RFC: Static classes - It is proposed to add the ability to describe static classes.
  • RFC: Unicode Codepoint Escape Syntax - Offer to add the ability to set unicode characters as codes: echo "\u{202E}Reversed text"; .
