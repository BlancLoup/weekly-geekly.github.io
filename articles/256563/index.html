Weekly build Vivaldi 1.0.161.2

Hello!

And again breaking the usual schedule, when the weekly assembly comes out on Monday, we decided not to wait for the next week and show the assembly today. We think you will like this update, because we worked this week exclusively on the shortcomings and corrections that you reported us most actively. So - what's new in this build.

First of all - mouse gestures. Recently, this function has caused the greatest criticism, because actually did not work. We have corrected - now mouse gestures work both in Linux and Mac as well as for Windows users. In addition, the number of available mouse gestures even increased - we not only fixed, but also improved this function.

Further, another sore point is the native design of windows in various axes. We have begun work in this direction and in this assembly you can already try the results of our work. So far everything is reduced only to the native view of the application window, but this is only the beginning. Work in this direction will continue.

Another improvement was made for the tab grouping feature. We have added an indicator of the unread pages of the group, which will allow you to go directly to the newly opened page, rather than scroll through all the members of the group.

We also fixed one old problem that users of two monitors or monitors with HiDPI support constantly encountered. I'm talking about opening context menus in the wrong place ‚Äî on a nearby monitor or outside the application window. Now this problem is fixed.
