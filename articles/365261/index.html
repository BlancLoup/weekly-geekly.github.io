iPocket - Charging a smart device plus a wireless external memory with the possibility of sharing over WiFi



In modern phones, tablets and other gadgets, memory seems to be a lot, but it is always not enough. This is especially true for going on nature, on vacation - I want to make a million photos, record a lot of videos. But it ends either in the memory of the phone, or the energy in the battery. Sometimes these two troubles occur simultaneously.

The iPocket company offers the same-name device, which solves not two, but three problems at once.

What problems does the device solve and how does it work?


First , the problem of low memory. In one of the parts of the system (iPocket - 2-in-1 system) there is a slot for microSD memory cards, support up to 128 GB. All photos, videos, other data can be immediately recorded on this external drive. To use external memory, no wires are needed, access takes place via WiFi.



Secondly , the problem of discharging the battery. The second part of the system includes a 2500 mAh spare battery. This is enough for charging most of the phone batteries and a significant recharge of the tablet battery.

Thirdly , up to 8 users can simultaneously connect to the device via WiFi. You can give access to information to friends. And you can connect your phone, laptop, tablet, and what else you have there, and use the data in your small cloud. The maximum data transfer rate when working with such an access point is 54 Mbps.



Each part of the system has its own name. The first part is iPocket Cloud. Here is a wireless communication module and a slot for memory cards. The second part of the Power Stick - here we have a 2500 mAh battery and a USB port for connecting rechargeable devices.

An interesting solution with power / recharge. In normal mode, iPocket Cloud is powered by Power Stick. They are connected using USB. And this means that the ‚Äúcloud‚Äù can be connected to any other USB charging device or simply to the USB port of any device.
In turn, Power Stick can recharge any device for which there is a USB cable.

Comparison with analogues


According to the developers, the iPocket system compares favorably with analogs:
  • Most are 2-6 times larger;
  • Most have fixed non-removable memory;
  • Some require an extra drive;
  • similar devices have no charging function.

Working with the device is very simple, thanks to convenient software (for iOS, Android, Mac, PC). The software allows you to immediately view your files - both pictures and video.
