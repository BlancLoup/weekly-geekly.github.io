Microsoft Introduces Windows Azure Accelerator for Web Roles



Microsoft has announced the first release of Windows Azure accelerator for web roles . With this free tool, you can create a server farm to support websites running on Windows Azure running IIS.

Accelerator makes it easy to host a huge number of small websites in Windows Azure, as well as easily update and manage them. It also simplifies the management and large-scale projects hosted in Windows Azure.

Accelerator includes a project template for Visual Studio, which creates a web role to host one or more sites. By deploying this template only once, you can almost immediately start hosting sites in Windows Azure using Web Deploy. Placing a site in Windows Azure takes about 30 seconds. Thus, Windows Azure Accelerator for Web Roles allows you to combine a large number of websites deployed on Windows Azure into flexible hosting with a convenient management interface, thereby reducing the cost of supporting them and allowing you to easily manage traffic spikes even in large-scale projects. .

Windows Azure Accelerator also includes:
  • Documentation: it is included in the project template, as well as laid out on CodePlex, which simplifies the study and work with the accelerator.
  • Two modules that allow you to embed code in new or existing web roles. In addition, these modules can be used as the basis for developing a new solution.
  • Dependency Checker (Dependency Checker): the installer includes a new and improved verification program that allows you to make sure that the accelerator has all the necessary components for successful operation, for example ASP.Net MVC 3 or Windows Azure tools for Visual Studio 2010. Dependency Checker integrated into the Microsoft Web Platform Installer, which simplifies the deployment of sites.
Download Windows Azure Accelerator for web roles and more information at http://waawebroles.codeplex.com .
