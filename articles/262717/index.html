New build browser Vivaldi 1.0.219.51

Hello!

Weekly assembly smoothly turned into daily. Yesterday we also posted a new version . In fact, we do several internal assemblies on a daily basis, which is why it is not difficult for us to show you a new one every day, and you are not bored.

In today's version, we fixed a bug that caused the browser to crash when trying to log in to the popular Disqus service. True, some problems still remain - in particular, Vivaldi does not always manage to save data in the password manager when trying to log in to the account directly through the commenting form, therefore we recommend logging in directly to the service website and saving the password in Vivaldi. After that, you should have no problems commenting on sites that use Disqus.

We also corrected the localization again, removing some shortcomings.

That's all for now. Download the new assembly can be on the links below:

  • Windows 32-bit
  • Windows 64-bit (experimental)
  • Mac
  • Linux RPM 32-bit
  • Linux RPM 64-bit
  • Linux deb 32-bit
  • Linux DEB 64-bit

List of changes:
  • VB-7713 Updated translations: Again!
  • VB-7174 Crash when saving password in Disqus using popup login via a comment box
  • VB-6973 Settings Search not reachable if Native Window is set


That's all for today. We remind you that the weekly assemblies are test, so do not forget to backup important data. You can report any errors found at the same address .
