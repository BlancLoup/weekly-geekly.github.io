8 tools for online marketing planning

Galstuki.pro specialists have prepared a list of tools that need to be noted for small businesses in order to properly organize marketing planning in the network. Some of them are known to you, some will require some technical knowledge from you. However, believe me, the result is worth the effort.

1. Google Analytics
Website analytics is one of the most obvious and important tools for improving your business. But in our experience, not all companies use this. Another problem is that most small business analytics accounts are not configured correctly. There are a lot of configuration manuals on the Internet, we recommend that you familiarize yourself with them.

We use analytics tools when it is necessary to understand how successful the company‚Äôs business is online, or where indicators are wrong. When you need to set up your campaign goals to evaluate its effectiveness.

2. Google Website Adaptation Test
If the web site of your company has not yet been adapted for mobile devices, believe me, it's time to do it.



3. GTmetrix



GTmetrix - a tool to optimize the speed and efficiency of the site.
The speed of the corporate web site directly affects the degree of customer satisfaction with the quality of service, as well as the ranking of the site by search engines.

4. Google Drive
Google Drive allows you to create and share documents with any users, be it team members, partners, clients.
You can allow users to comment on documents and leave notes, as well as put labels in order to attract someone's specific attention. You can use the change tracking feature to see which users edited the document. In addition, with this application, you no longer have to send bulky files by mail. All you need is just to share a link to the document.

5. CoSchedule



CoSchedule is a great tool for project planning and management. In particular, it is suitable for organizing the work of a team of marketers (and even one blogger) and a team working on promotion in social networks.
As the name implies, the CoSchedule application allows you to organize the project management process in the schedule and calendar format. The application recalls upcoming publications and the timing of the execution of the work, reports on which of the publications have received great popularity.
For each individual project and publication, you can create separate boards, including deadlines for completing the task, lists of current tasks, a status bar that allows you to follow the progress of the tasks, comments on the project.
CoSchedule is perfect for blogging. On the board of a specific project, you can choose the categories of WordPress to publish, specify the author, and even follow the link to edit the publication in WordPress.
Currently, the application is supported by platforms such as Twitter, Facebook, Google+ and LinkedIn.

6. Trello



Trello is a free web application that allows companies to plan the promotion of content on social networks and track performance.

In our company, we actively use this service, as it is convenient to set and synchronize tasks among all employees. We recommend to try.

7. Basecamp



Basecamp is a web project management tool that allows you to create lists of current tasks, share files, chat with team members.
The program allows you to set deadlines for individual tasks and create checklists of questions, so that you can see which team member is responsible for which task. You can invite users for a one-on-one conversation, or have group meetings.

8. Evernote



Evernote is an application created like notebooks with which you can organize content. In each of the notes, you can add text, images, files and share it with other users.
The application allows you to turn notes into PDF files or instant presentations (for users with a premium account). For individual notes, you can also add reminders that are synchronized with the Google Calendar application. To make it easier to search for notes, you can add tags to the description.
The Evernote application has a chat feature that allows you to exchange instant messages in real time with other team members.
