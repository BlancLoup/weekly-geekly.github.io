Third Beta Candidate - Vivaldi 1.0.403.20


Hello!

Today, the assembly with minimal changes - only a few important fixes. The reason is simple: this is a kind of release candidate for the third beta, which will be released the other day. We are trying to fix some critical bugs for this release. It is also important to check whether any regressions have occurred since the release of the previous build number 1.0.403.17. If you find something like this - let us know so that we will correct it.

In this build, we fixed a bug with non-working HTML5 video on Windows systems with missing hardware acceleration.

Well, the accompanying news - we opened a gift shop and branded clothing Vivaldi . This is another way to help us, not even with money, but with the popularization of the Vivaldi browser.
