  • New features managed application.
  • New optimization tools for application developers.
  • New interface design.
  • Support for Oracle Database DBMS.
  • A mechanism for optimizing the generation of aggregated data for building analytical reports.
  • The mechanism of administrative management of the installation and launch of versions of "1C: Enterprise."
  • Failsafe 1C: Enterprise server cluster.
  • Dynamic load balancing of the 1C: Enterprise server cluster.
  • Optimization of the mechanism of work with the database.
