Opera Unite reopens the web



Cloud computing and web applications are no longer the same thing

June 16, Norway, Oslo - Opera Software today officially unveiled Opera Unite, a new technology that changes the perception of the Internet as a client-server model of information exchange. Opera Unite is able to make any computer not only a client device, but also a server. This allows users to connect with computers of friends and acquaintances directly and share information without the use of online services and applications.

With Opera Unite, sharing data between computers is as easy as working with web resources. This technology allows the end user to better control the use of his private data in the public domain. In addition, he can work remotely with information stored on his computer using any other device equipped with a modern web browser.

Developers need to know that Opera Unite services are based on the same open web standards as web sites. This greatly simplifies the creation of new applications based on Opera Unite - developing them is no more difficult than creating web pages.

Opera Unite technology is available in a special version of the Opera 10 browser, which can be downloaded for free at labs.opera.com .

Using Opera Unite
1. Download and open the browser assembly from Opera Unite from labs.opera.com
2. Click on the Opera Unite logo in the lower left corner of the browser window and enter the username and password that you use to log into other Opera resources, such as My Opera and Opera Link. If you do not have an account, Opera Unite will offer to create it.
3. Opera Unite is accessed from the sidebar of the browser, and new services can be downloaded from the unite.opera.com section .
4. Starting the service will create a direct link to the Opera Unite application, for example notebook.jondoe.operaunite.com/photo_sharing . In this link: ‚Äúnotebook‚Äù is the name of the local device with the contents (files), ‚Äújondoe‚Äù is the name of the user, ‚Äúphoto_sharing‚Äù is the name of the Opera Unite service to which you open access from the outside. By clicking on this link, friends and acquaintances will get access to the content that is open to them on the hard disk of your computer through a normal web browser.
5. To see which of your friends are using Opera Unite services on your My Opera Unite home page.

Opera Unite services work directly in the browser. Currently, the following services are available to users for free:

File Sharing: using this application, you can share other users with files on your computer. In this case, you do not need to pre-upload these files to specialized online services. Just select a folder on your local drive, access to which you want to provide. Opera Unite will generate a direct link to this folder. It can be sent to friends with whom you want to share files.
Web server (Web Server) : this service allows you to run a full-fledged website from a computer. Simply select the folder with the files of the website on the local disk and allow access to it from the outside. All content will remain on your computer, which will act simultaneously as a repository of information. Opera Unite automatically recognizes and launches index files, so your website will appear as it was designed.
Media Player : access to your music wherever you are and from any computer. To do this, select the folder with the music collection in MP3 format and allow access to it, and then use the direct link Opera Unite, opening it in any browser and starting to listen to the songs using the Opera Media Player.
Access to photo (Photo Sharing) : now you can share photos with friends and relatives directly from your computer without sending files to online services. To do this, simply select the folder with photos on the local disk of the computer, and the application will create a gallery of their thumbnails. Full-size photo is loaded by clicking on the thumbnail.
Living Room (The Lounge) : This service allows you to organize a chat with friends right on your computer. Your friends can join the conversation by clicking on a direct link, without having to register with any online services. Depending on the settings, you only need to tell them the password to enter the chat.
The Fridge : Attach notes to your friends' virtual refrigerators. Now you can exchange messages with them in safe mode and in real time. Do not forget to tell them a direct link to your "fridge".

Six services already available in Opera Unite, created to demonstrate the capabilities of the new technology. Web developers can unleash their imagination and create applications themselves, fully exploiting the potential of Opera Unite.

Why did we release Opera Unite?
Every day more and more data and more and more services for their processing and storage appear on the Internet. Cloud technologies allowed sharing information anytime, anywhere, with any user. However, this freedom of access is associated with the transfer of private data to third parties - web application servers. This significantly reduces the ability to control the use of personal files.

We hope that the "cloud" computing will continue to change our understanding of web applications for the better. But we also believe in another way to exchange private information - without losing control over it. It is this idea that formed the basis of the Opera Unite technology, and it is in it that we see the future of the Web.

‚ÄúToday we are discovering the real potential of the Internet for all users,‚Äù said Jon von Tetzchner, chief executive officer of Opera. - Technologies are developing in a spiral. Once the emergence of personal computers led to the distribution of computing previously focused on mainframes. In turn, Opera Unite is designed to make the "cloud" technology more democratic. A browser with an embedded web server will give developers the ability to quickly and easily create web applications, and end users will receive a flexible tool with a powerful access control system for private data and advanced information sharing functionality. We believe that Opera Unite is our most significant innovation ever, because this technology is changing the fundamental concepts of web space. "

What is Opera Unite?
Opera Unite is an interaction technology that allows one user to access private information, such as files, photos, music, placed on another user's computer. This technology uses a small web server hosted in Opera‚Äôs desktop browser that allows you to give access to data and services

Opera Unite works on all platforms that support Opera's bowser, including Windows, Unix and Mac.

In order to activate Opera Unite on his computer and provide access to its content, the user must use the Opera browser. But in order to get access to this content, any modern browser is enough.

Opera Unite services allow you to extend the capabilities of the browser - from the functionality of social networks and chat rooms to the ability to open files, music or photos for safe access to other users.
