The alpha version of the HyperVideo Actio Editor is available.

Good afternoon, dear habrasoobschestvu.

We are glad to present you the alpha-version of the Actio Editor hypervideo, which can be downloaded and used for free (attention, registration on the site is required), the delivery also includes a player for playing ready-made videos.

Hypervideo technology allows you to select objects in a video that the user can interact with. For example, when you hover or click on an object, text information, a link to the Internet, another video, or a sidebar with additional information may open.

Objects can be any area in a video clip. For simplicity, the markup has the ability to auto-track areas that are visually different from their surrounding background.

Objects selected using Actio Editor can include pop-up captions, descriptions, links to websites, links to other parts of the video, etc.

You can read more about what hypervideo is in my old article or see the technology in action (attention, you need an installed silverlight).

For Actio Editor to work properly, the following software must be installed:
  • OS Windows XP and above;
  • Microsoft .NET Framework 3.5 and higher (if not, then it will be downloaded from the Internet installed along with the editor);
  • Windows Media Player 10 and up.

Immediately I would like to note that the interface design is not final. This is a prototype that allows you to use the functionality of the editor, but by the time of release it will be redone.
