Yandex browser. First look

I installed a Yandex Browser. Just out of interest, although I did not feel any need to change Google Chrome as the main browser and Mozilla Firefox as an additional one. The first feeling is that Yandex.Browser works significantly slower than Chrome. Not catastrophically slower, no, but palpable.




And one more detail struck. When installing, Yandex Browser suggested closing Google Chrome in order to correctly copy all settings from it. I accepted this offer. As a result, not only the settings were copied, but all the passwords. Copying happened so correctly that when logging in from Yandex Browser to Gmail or Facebook mail, no password was required. Interesting technology, taking into account the conversations that here, they say, this or that IT-company does not issue user data. It turns out that it is worth making your assembly based on Chromium and everything that one company so carefully conceals will be in the hands of another.
