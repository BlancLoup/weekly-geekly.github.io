Release elementary OS "Freya"

Two years after the release of elementary OS Luna, we waited for the release of the next version, called Freya. I recall a distinctive feature of elementary OS is a graphical shell of its own design - Pantheon. Additionally, Freya is based on Ubuntu 14.04.


There are a lot of changes in the system; the full list can be found on the website elementary.io
OS Demo:

Record OS presentation:

After testing Freya, I can say that the system is more like another beta with a huge number of bugs. I am sure that the developers try and put a lot of effort, but I do not see it possible to use the current release as the main OS. We will wait for a more stable version.
