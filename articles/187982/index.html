Free Software Foundation begins fundraising for Replicant

The Free Software Foundation announced a collection of donations for the Replicant operating system, which is a completely free Android distribution. This is the first mobile OS that works without a single line of proprietary code.

Version 4.0.0004 of July 22 supports 10 different devices, but if developers manage to collect money, they will buy new devices, check and port the system on them. Support for each new device costs 400-600 dollars only to buy a device. Also, a little money is needed for the infrastructure and advertising of the project.

The FSF developers explain that Android itself is initially almost completely free, but in practice, manufacturers add proprietary system modules to it. For example, for communications with WiFi and Bluetooth chips. In addition, each device comes with a package of proprietary applications running on top of the operating system.

Replicant tries to provide all the same functionality, but only with the help of free software. All applications are installed from the F-Droid directory.
