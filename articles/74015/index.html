So, the concept of the necessary system was gradually formed. The team, represented by project manager Konstantin Kharitonov, chief programmer and architect Vladimir Barzionov, programmers Artyom Verbitsky (desktop client Cerebro) and Dmitry Samoylov (Mirada module) under the supervision of technical manager Ivan Ivanchik starts developing Cerebro - an integrated system for planning and managing projects. the fathers of the Cerebro system were films, ‚ÄúEurope-Asia‚Äù, directed by Ivan Dykhovichny and ‚ÄúIndigo‚Äù by Roman Prygunov.

About the system


In Spanish, Cerebro (correctly pronounced Cerebro) is the central nervous system of the brain. Supporting the vital activity of the project, the developed software complex performs the most important functions: resource management, ensuring the operational interaction of participants and the ability to quickly transfer files of any size. So, thanks to the built-in module Mirada (Spanish - view), the user has an image viewer that supports an extensive set of formats. It is a player of video files and audio recordings, and also contains a set of tools for reviewing, giving a unique opportunity, for example, to draw a brush over video files, create text and voice tags with reference to an image, visually mark comments on a single frame of a video, etc ... The publication of the review in Cerebro occurs automatically. The internal system Cargador (isp. - loader) is designed to transfer files without using services such as FTP-exchange, offers the possibility of automatic archiving, the option of using symbolic links to access local resources, performs independent search and download specified files from various sites on the network and much another.
