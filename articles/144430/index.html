Kinect for Windows SDK has learned to work with individuals



May 21 released version of Kinect for Windows SDK and Runtime 1.5 , where a number of new tools appeared:

  • Kinect Studio program for recording and then playing the user's movements.
  • Human Interface Guidelines (HIG) with examples of good interfaces.
  • Face Tracking SDK : real-time 3D mesh processing of a face: head position, eyebrow position, mouth shape, etc. So now your Windows program can respond to the user's facial expressions.

Many new code fragments in C ++ and C # have been added, the Basics section for beginners has appeared with examples in C ++, C # and Visual Basic.
