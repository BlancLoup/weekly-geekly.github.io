Replacing Task Manager


On something faster and more useful. Task Manager often opens long and tedious, especially when it is just really needed :)
Process killer is a miniature (about 30 kb) program that opens instantly and pops up on top of all windows even where the Task Manager cannot do this at all (in many hung full-screen games). In addition, the withdrawal process works absolutely instantly.

Author page: alex-home-pg.nm.ru
The program is free

Manage autoload and monitor system status


AnVir Task Manager
Very feature-rich program. I, probably, do not use even half of the possibilities, but this is what bribes me:
Tray icons that show CPU, memory and hard drive usage. Everything is customizable.
Startup Manager : any attempt of any program added to startup is fixed. You can allow or deny the change, just like in firewalls.
Postponed autoload . I have about 50 entries in my startup - and although most of them are miniature dlls, it still takes a long time to wait. Delayed autoloading comes to the rescue: loading in a minute (this value can be changed) of applications with an interval of 10 seconds (for resource-intensive applications. The interval can also be changed)
There are many other useful, though less demanded functions , such as a process manager with a detailed description of everything and everything, managing windows services and much more.

Homepage: www.anvir.net
Free for non-commentary use. About 1Mb.

Create virtual desktops


Altdesk
A very useful thing, especially if you do not have a widescreen monitor (and, as a result, 10 programs on the taskbar do not fit). Virtual tables are a great way out, besides, you can set everything up for work on one table, on the other - for surfing and so on.

Homepage: www.astonshell.ru/altdesk
By the way, these are the developers of the alternative Aston shell. The program is shareware (30 days trial), installed weighs about 5 MB.
