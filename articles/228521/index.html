A new version of the distribution has been released to create a pfSense 2.1.4 firewall.


On June 25, 2014, the release of the opensource distribution kit for creating firewalls and pfSense 2.1.4 network gateways was announced

The distribution uses the FreeBSD codebase and various software packages to get a user-friendly interface for various networking technologies and tasks.
Available for download architecture i386 and amd64, the size of 78 megabytes. It is possible to work with a USB flash drive - recording to the drive is minimal, it is possible to write logs to the RAM partition, with periodic unloading onto the media.
With the help of the distribution kit, you can build fault-tolerant systems - synchronization options are available between individual pfSense servers, various failover solutions, CARP, and so on.
