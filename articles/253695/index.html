1. HP Helion Presentation


The report will discuss HP cloud technologies, in particular, attention will be paid to such a solution as HP Helion , HP 's contribution to OpenStack, and how HP's cloud solutions can automatically solve many problems in configuring and debugging any IT infrastructure.


2. VDI VMware Solutions Video Review, Horizon 6 Update Review


Under the Horizon brand, VMware presents solutions for computing by computing: everything you need for client-server infrastructure jobs. The server part of these solutions is more known, the speaker will focus on the client side. It will be a question of architecture of the decision of Horizon. Also, the speaker will show and tell about the invisible, but very useful settings for administrators of virtual machines, which appeared in the latest version. There will also be a small demo showing how VMware products work in VDI solutions.


3. Overview of Extreme Networks network equipment updates


This video is about Extreme Networks networking technologies. As with the help of EN equipment, it is possible to build high-performance, stable and inexpensive modern networks. On the principles and approaches of EN when building a network infrastructure. About what updates in the line of network equipment, in particular, about the new module with 100-gigabit ports for BlackDiamond and other Extreme switches appeared. It will also show how, when a line is broken, the Extreme Networks based solution does not stop video transmission.


4. Demonstration of little-known features of Windows Server 2012R2, Office 365, Microsoft Azure


The speaker will talk about the capabilities of Microsoft cloud products, which for some reason are not always used by customers and integrators. For example, few people know that in Windows Server 2012R2 there is an opportunity to increase the size of the virtual disk without stopping the virtual machines, and the speaker will also broadcast a complex video conferencing system in a few minutes. We are talking about products such as: Windows Server 2012R2, Office 365, Microsoft Azure, there will also be a brief overview of what's new in these products.


5. Demonstration of the program ZyXEL Wireless Optimizer


The speaker will tell you about the work of the ZyXEL Wireless Optimizer utility for calculating access points coverage areas. The program takes into account many factors: the walls from which they are made, the removal of access points from each other, the height at which they are installed, the type of network: household, office, medical establishment, etc. As a result, you can see a picture of your wi-fi network in a graphic form. Also talked about technologies such as ZyMESH


6. Demonstration of Samsung's MagicIWB Interactive Solution for Business and Education


Touchscreen on the screen with a projector has long been known, but Samsung offers a more advanced solution. The speaker demonstrated the functionality of the software for interactive presentation. Such solutions are suitable for both business meetings and classroom training.


7. IBM SmartCloud Storage Access Video Demonstration


At the IBM booth, the IBM SmartCloud Storage Access software and hardware complex was presented. It is a simple, easy-to-deploy, self-service cloud storage system software that enables organizations to implement private cloud storage services. Thanks to this solution, most of the tasks of administering physical and virtual infrastructure can be automated.


8. EMC ViPR SRM - monitoring of the computing infrastructure of the customer, built on the equipment of various manufacturers


The Watch4net solution was demonstrated at the EMC booth . This software allows you to see the performance of networks, data centers and cloud infrastructures for the previous period, currently in real time and in the future when performing design work. The solution provides information on current, historical and projected performance indicators. Watch4net helps customers successfully cope with the most difficult tasks of managing both physical and virtual IT environments.



9. Dell Desktop Virtualization Solutions


The demo will feature Dell desktop virtualization solutions (DVS). They provide an opportunity to overcome traditional difficulties with the use of desktop virtualization. DVS provide simplified, scalable, better integrated virtualization. Simplified and secure access to virtual desktop resources is provided through Dell Wyse specialized thin, zero and cloud clients.
