Virtual machine for e-commerce

Choosing a CMS, as well as working with them, begins with a search for CMS reviews, after which
There is a process of visual acquaintance with the systems. Installing a CMS for an engineer
A programmer who has experience in this topic is a process that is clear, simple and
taking half an hour - maximum one hour of time.
If you do not have experience working with php \ mysql \ cms, then for you to start yourself
working with an online store will be quite difficult, so the following
the solution will simplify this task.


CatalogLoader engineers constantly have to work with dozens of popular
CMS, and for the convenience of their work, an image of a VirtualBox virtual machine was created with
pre-installed free CMS.

The created image of the virtual machine is best suited for independent
work with CMS systems. You can try to fill the store. And it is important that your
data will not be deleted if you do this on the demo pages themselves
CMS systems.

Debian 6 is used as the operating system. VHD file with CMS can be
download link:

http://static.mydataprovider.com/CatalogLoader_demo_shops.7z

The current version has installed free CMS.
