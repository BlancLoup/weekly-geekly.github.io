Distribution Alt Linux received a certificate FSTEC



Alt Linux has certified a new distribution. On April 18, 2011, Alt Linux SPT 6.0 distribution kit received the certificate of FSTEC No. 2317.

Unlike other certified versions (Alt Linux has been producing certified versions since 2002), Alt Linux SPT 6.0 can be used for all types of personal data protection systems, as well as state secrets.

Alt Linux SPT 6.0 has:
  • 4 class protection of computer equipment from unauthorized access
  • Level 3 controls the lack of undeclared capabilities.

Can be used to develop:
  • automated systems with security class of 1B inclusive
  • personal data protection systems for K1 inclusive.

Alt Linux SPT 6.0 is a unified distribution for servers and workstations with built-in software for protecting information.

The workstation is based on the GNOME environment, contains a standard set of applications (OpenOffice.org, Firefox, programs for working with graphics and multimedia and drivers for a wide range of equipment, including peripheral and much more).

The server combines all the features of a classic server and modern technologies for managing them through a convenient web interface. A key feature is the provision of a mandatory access control principle with SELinux. Also in the distribution includes a powerful backup system enterprise Bacula. Undoubtedly important for a certified system is compatibility with leading manufacturers of cryptographic tools. Authentication and update of workstations are performed centrally via a web interface.

Versions of the distribution software are not outdated, but not the latest, but it cannot be otherwise, since certification is not only an expensive process, but also quite lengthy (from 6 months to 1 year), and the final version of the image is delivered no later than than 4-5 months before the end of certification.

But the kernel version is the most current, currently used in all Enterprise versions of leading Linux distributions.

The certified package, in addition to the certified copy of the certificate, blank form, fstekovskoy stickers and disks (2 DVD for 32-bit and 64-bit) with unique numbers, includes a technical support coupon, which is needed primarily for free to receive regular certified updates that under the terms of the FSTEC, users are required to apply in their system.
