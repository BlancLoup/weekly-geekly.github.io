By default, the system will work on Unity 7.3, but there is information that you can later upgrade to Unity 8.

Now the locally integrated menus are used by default - LIM. They came to replace the global menu, which was present in previous versions.

The kernel used is the Ubuntu Linux Kernel 3.19.0, which is based on Linux 3.19.2.
