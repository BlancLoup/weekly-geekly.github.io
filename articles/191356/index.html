A step away from v12: 3CX Phone System 12 Release Candidate released

We are pleased to announce the release of the Release Candidate version for 3CX Phone System Version 12 (build 31763). Version 12 is the most functional and stable software PBX to date, with completely new multi-platform clients for Windows, Android and iPhone.



Compared to the Beta version, the ability to schedule conferences has been added to the v12 release candidate and the 3CXPhone for Windows client has been improved. In addition, the Postgres database version has been updated. Below is a brief description of the features:
