HTML / CSS / JS editor with IDE elements (SaaS version)

Just the other day, a completely redesigned template editor appeared in our website builder .
We tried to get as close as possible to the usual functions of ordinary editors.

Now you can edit several documents in one window, with switching through tabs (the pictures are clickable):



Tabs can be swapped, they change their size depending on their number and the width of the screen. The set and order of open tabs is saved and tied to the user, thus having logged in from any workplace, the set will remain the same.



Each file save is saved in the change history we wrote about earlier . Any two revisions can be graphically compared.



When you open the document, you can search for the file name by entering the typed characters:



Special attention is given to the autocomplete system (hints) for internal variables and django-tags. Typing {% or {{followed by Ctrl + space, the system will prompt for possible values.



Moreover, if there are parameters for tags, the system will prompt them, taking into account the content available on the site.



And of course, there is a full-screen mode, which when enabled along with the full-screen mode of the browser itself, makes the editor almost indistinguishable from the usual application, which is very convenient in many cases.

We plan to further develop the editor, as well as other functions that will allow webmasters and store managers to use the browser mainly in their daily work with the designer.
