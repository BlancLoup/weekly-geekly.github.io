Songbird, Winamp, iTunes and Media Player

The developers of Songbird, during the optimization of their player, conducted a comparative test of memory consumption by several popular programs for playing music files. Among them were Songbird, Winamp, iTunes and Media Player. Testing was conducted on Windows XP SP2 with 1GB of memory. 20,000 mp3 files were taken for testing.

Test results can be viewed under the cut.



The undoubted winner was Windows Media Player, which used, on average, two times less memory compared to all other programs. The main object of testing - Songbird version 1.1 began to consume memory almost 1.5 times less, compared with the previous version.
