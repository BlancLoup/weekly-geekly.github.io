Little utility for serious matters

Have you ever lost a flash drive? Did a lot of information go with her? Just recently, I went online to the free, unpretentious utility SyncToy 2.0 from Microsoft. Its functionality is also extremely clear and simple - we set folder A, we set folder B, we synchronize all the contents of folder A with folder B. Three modes are available:
  • Syncronize - changes folder A is transferred to folder B and vice versa.
  • Echo - all changes to folder A are transferred to folder B.
  • Contribute - Same as Echo, but deleted files in folder A are not deleted from folder B.

To minimize the consequences of losing the flash drive, I set up a daily synchronization of the flash drive and local folders of the work computer and laptop. Create a regular task in the "Task Scheduler" is only used as an executable file SyncToyCmd.exe with the -R key.

SyncToy 2.0 is available even for x64 systems. The utility has found its application on the server as a means of copying a WSUSContent folder to a backup server, and those familiar with WSUS (System Center Essentials) know that periodically you need to clean up unnecessary more patches and simply copying the WSUSContent folder will lead to its strong growth.
