Intel Parallel Studio XE 2013: optimizing performance in new ways


September 13, though not Friday, is still a great day to present the new version of the already well-known Intel Parallel Studio product to many, relating to the number ‚Äú13‚Äù - Parallel Studio XE 2013. In this message we will briefly tell you about the composition and functional Parallel Studio XE 2013, as well as its differences from previous versions.

Every developer wants to know how to make your application run faster. Intel Parallel Studio XE contains the latest versions of compilers and libraries, helpers in parallelization of applications, as well as means of comparison and analysis. It almost seamlessly integrates into Visual Studio and the GNU tool, providing the ability to optimize code not only for the latest IA architecture processors, but also for future models.

Intel Parallel Studio XE 2013 includes:
  • The set of development tools Intel Composer XE , consisting of:
    • Intel C, C ++ and Fortran compilers
    • Library of productive mathematical functions lntel MKL and Intel IPP
    • Efficient parallel programming tools for Intel Threading Building Blocks and Intel Cilk Plus
  • Intel Advisor XE Parallelization Helper
  • Intel VTune Amplifier XE Performance and Parallelization Profiler
  • Intel Inspector XE Memory and Parallelization Control Tool
  • Static Analysis Code Analysis Tool

Here is a list of innovations in Intel Parallel Studio XE 2013:
  • Support for new Intel Xeon processors and Intel Xeon Phi coprocessors
  • New Threading Assistant and Intel Advisor XE Tools
  • C ++ Performance Optimization Guide
  • Support standards Fortran (F2003, partly F2008) and C ++ 11
  • More thorough troubleshooting with Intel Inspector XE
  • Intel VTune Amplifier XE is simpler to use and provides additional profiling data.
  • Built-in compiler tool for determining the code that requests memory addresses outside of the specified range
