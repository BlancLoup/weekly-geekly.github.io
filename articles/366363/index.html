All hands on deck! Update on Build 10536

Another batch in Fast Ring for mobile Windows insiders has been released - 10536. Details under the cat.




A small background to the emergence of a new build
Insiders waited almost a month for a new build and managed to get Gabe on Twitter.
Last Thursday a vote was taken - the result of which was to be the release of build 10536.1003 last Friday.

The essence of the vote was that Gabe was interested in insiders, whether to release a new assembly today or wait for the bug fix with the Store until next week. Voting result: strw.pl/375839b3/results
As a result, this assembly should have been accompanied by a bug, which in about 28.8% of cases, led to the crash of the Store when trying to buy an app or make an in-game purchase, as a temporary solution offered to buy apps from a PC or the web version of the Store. But something went wrong, and on Friday, Gabe announced that the new build is being transferred to the next week due to the inability to upgrade from 10536.1003 to future builds.

The update was withdrawn at the last moment and according to Gabe - about 200 people managed to install it, if you are one of them, you will have to roll back to Windows Phone 8.1 to install the new build.
Immediately after the weekend, an internal testing of build 10536.1004 was announced, in which bugs were fixed with the store and updated to the next builds, this build was made available to insiders that night.

First about the pleasant. Innovations and fixes:


  • On Twitter, Gabe reported that there are many improvements to the Edge in this build.
  • Insider Hub - again present in the assembly.
  • The mobile access point has been restored.
  • Voice input has been updated to add speech recognition support for Japanese and English (India) languages.
  • Corrected the update - now you can upgrade to 10536 directly from 8.1, bypassing build 10166.
  • Fixed two-factor authentication when adding a Microsoft account on the phone.
  • Fixed idle Quiet hours.
  • The main screen is now loaded faster, without the eternal "Loading ..".
  • The clock on the lock screen now appears without delay.
  • Scaling maps fixed, now should work as intended.
  • Updated Photos app - folders are now shown, running faster, higher and stronger, as usual with Microsoft.
  • Single hand mode - now on all devices (Press and hold the Windows button so that the screen slides down halfway and you can reach the top elements on the screen). Previously it was only available on devices with a screen of 5 ".
  • Announced with the insider build 10527 for PC Feedback application Windows is now available in the mobile assembly.
