New builds of Ubuntu Touch are now laid out every day



Not so long ago on Habr√© there was a news that Canonical, perhaps, will refuse from the semi-annual releases of Ubuntu, in favor of more frequent "floating" (Rolling Release). As for the new mobile OS, Ubuntu Touch, its preview builds will be laid out (and laid out) every day. You can download the new version from this page . Now available version for 1-3 March inclusive.

As mentioned earlier, Ubuntu Touch is currently a two-part system. The first is the custom version of CyanogenMod 10.1 Android 4.2 ROM. The second part is the Ubuntu file system. The user actually accesses Ubuntu using chroot.

CM10.1 was chosen for the simple reason that this version of Android includes support for touchscreens, communication modules and other hardware elements of a large number of devices. Thus, Ubuntu Touch (it is worth recalling that the full name of the current builds is Ubuntu Touch Preview) works on an extensive range of mobile devices.

Each new build includes new features and, of course, fixed bugs of the previous release. Unfortunately, so far these builds are not stable, so using the Ubuntu Touch Preview as the main OS for your mobile device is unlikely to succeed. Get more information on installing this OS here .
