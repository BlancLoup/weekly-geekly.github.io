Firefox mobile

In early October, Mozilla executive director John Lilly announced that the mobile version of Firefox will be available for download in a few weeks. Today there was another case confirming the words of John and being another grain of information, the first screenshots of the browser were published. According to The Unwired , the screenshots depict Firefox Mobile, on a communicator running Windows Mobile Professional. The developers managed to run the Acid 3 test and get 88 points out of 100, which is not bad for a mobile browser. For comparison, my current Firefox 3.0.3 dials 71/100, and Google Chrome 0.2.149.30 79/100. Well, with access to the mobile market, Firefox will continue the confrontation with Opera, which is currently leading on mobile platforms.
