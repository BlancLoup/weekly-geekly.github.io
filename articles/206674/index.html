IP Phone Integration with Lync Server

Not a pleasant surprise when you deploy a Lync Server in a company can be a rather limited choice of IP phones that can work with Lync. And it‚Äôs quite easy to guess how the current situation affects the price of compatible equipment, a significant part of which is produced by one vendor and sold under several brands. The point here is not even in the laziness of other manufacturers of IP phones, but in the rather specific technological requirements of the Lync Server itself to the end client devices.

Today, one problem has become less. Introducing the Kinnex Media Gateway software product, which allows you to connect ordinary IP phones to Lync Server that are most suitable for existing tasks and budget.



Description

Kinnex Media Gateway, as the name implies, is a gateway that, on the one hand, interacts with the Lync Server as its full-fledged client, and on the other hand, as a data logger and data converter for IP phones. It is based on the Kinnex VoIP Engine media processing engine. By the way, on this engine we are preparing for the release of another product for the not yet officially released, but already gaining popularity technology.

Supported IP phone users have access to all basic corporate telephony features (calls to a number and SIP URI, hold, transfer, call forwarding). At the standby stage, support for advanced Lync features and additional functionality, which is missing in Lync itself, but can sometimes be very useful. It will work, naturally, only on IP phones that are connected through our gateway.

In addition to the main compatibility function, I would like to highlight the automation of managing a large fleet of phones, after all, Lync Server is a solution for a large and distributed organization. On the basis of Auto Provision technology and SIP PNP, Kinnex Media Gateway allows you to configure automatic user authorization and centralized updates. on ip phones. We constantly monitor and test software updates for each supported hardware manufacturer. Current firmware versions and configuration file templates are assembled into ready-to-install packages that are available on the product website.

Deployment

Kinnex Media Gateway is a software gateway, which means it can be installed both from one of the Lync Server roles and on a dedicated machine. The most suitable for joint installation will be the role of Standard Edition - in a compact deployment, SBS (registering the branch office server) and Mediation Server - if the role is moved to a separate server.



The process of installation and configuration is quite simple. In addition to the installation itself, you need to add users from the Lync Server and set pin codes for them. If you need a fully automated connection, including authorization, you will need to enter the MAC addresses of the equipment and associate it with user accounts. After that, to get started, just connect the phones to the network at the workplace.

Benefits

Using Kinnex Media Gateway provides a number of advantages that can be very significant for a company.
  1. The ability to create an optimal infrastructure based on the most appropriate equipment for the tasks.
  2. Optimization of the budget for the purchase of equipment.
  3. Universality of equipment that can be used not only to work with Lync.
  4. Easy installation and fleet management of IP phones.
  5. Affordable service by choosing the most suitable manufacturer.

What can be connected

Theoretically, any IP phone can connect to Lync via Kinnex Media Gateway, but in practice, the equipment has its own specific features in the interaction protocol that must be considered. Therefore, to ensure the stability of work, which in telephony is one of the key requirements, adaptation and testing of each model is necessary.

The list of compatible equipment will be constantly updated on this page as new models are integrated and tested. Moreover, we very carefully listen to our users, which means that everyone has the opportunity to leave a request for the integration of their favorite model or a whole range of your favorite manufacturer of IP-phones.
