Opera Devices SDK 9.7 presented at CES-2009



Opera Software at CES-2009 at Las Vegas unveiled a new version of Opera Devices SDK 9.7. This set of development tools erases even more the line between computers, mobile phones and various household devices with Internet access. The main innovations of Opera Devices SDK 9.7 include:

Opera Link - this technology allows you to synchronize bookmarks and Quick Start panel elements between computers, mobile phones, and now home appliances - TV / game consoles.

Opera Dragonfly - debugging JavaScript, inspecting CSS and DOM, finding and fixing errors in the code - all this is now available for consumer devices. Opera Dragonfly also allows remote debugging on home appliances directly from the desktop.

Opera Zoom hardware acceleration - this technology allows devices with limited resources to provide full scrolling and smooth scaling functions.

ACID3 Compliance - The new version of Opera Devices SDK 9.7 is fully tested for compliance with the current ACID3 web technologies.
