The book "Microsoft Visual C #. Detailed guide. 8th edition "

Hi, Habrozhiteli! We recently translated the book by John Sharp.

Who is this book for?


It is assumed that the reader of the book will be a developer who wants to learn the basics of C # programming using the Visual Studio 2015 environment and the .NET Framework version 4.6. After reading the book, you will get a complete picture of the C # language and will be able to use it to create adaptive and widely scalable applications capable of running under the Windows 10 operating system.

Who should not read this book?


This book is intended for developers who previously used the C # language, and is not designed for absolute beginners in programming. Therefore, its material is mainly based on the use of the C # language. This book was not intended as a detailed description of the many technologies available for building enterprise-level applications running on Windows, such as ADO.NET, ASP.NET, Windows Communication Foundation, or Windows Workflow Foundation.
