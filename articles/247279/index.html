Large collection of hashing functions on Github



Maciej Czyzewski collected on Github a collection of source codes for various hashing algorithms : for computing checksums, non-cryptographic and cryptographic.

In the repository you can find, for example, the implementation of CRC / MD5 / GOST 34.311-95 / SHA-3. Each hash function is represented by a source code in C and a makefile to build it. Algorithms are supposed to be used for learning purposes - in real projects it is recommended to use existing libraries (for example, Crypto ++ for C ++, BouncyCastle for Java, etc.), which are listed in the repository, for security purposes.

Active work continues on the repository, so in the future it is worth waiting for the collection to be replenished.
