Ungit is the easiest way to use Git

Good day, dear habrazhiteli. Literally just saw a terrific project on GitHub from FredrikNoren .



Ungit:


  • Clean and intuitive interface for Git (which is incredibly cool to master Git)
  • Works on any platform with Node.js installed and Git itself
  • Web oriented. Ability to run it in the cloud.
  • Github

Node.js version 0.10 or higher and npm version 1.3.1 or higher are required.
Installation 

npm install -g ungit
