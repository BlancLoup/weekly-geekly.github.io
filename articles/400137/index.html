Leaf Browser Alpha - browser with tree tabs


For fans of the Tree Style Tab extension (tree tabs) for Firefox, and for those who cannot live without it, but I would like to find a decent alternative to Firefox, the open source project Leaf Browser is dedicated to.
It is a browser based on Google Chrome, but with tree tabs.


The browser is implemented as an application for Google Chrome, and can be installed in a couple of clicks from the Chrome Web Store directly to your browser.


Unfortunately, this is only the alpha version so far and much has not been implemented, there are occasional glitches and crashes. So to use this browser on an ongoing basis is now almost impossible.
But now you can download and try, understand how fast and convenient it is.


At the moment, the project is being developed by a single developer, Sam Lanning, and its sources are available on GitHub .


Unfortunately, the author of the project lost a bit of enthusiasm in its development, but I think that this is due to the small popularity of the project. Although recently he seems to have changed his mind for the better.
Nevertheless, I really want the project to become more widespread and that there will be more developers and users interested in this project.


If you like this project, please make forks, send bug reports and write comments on the Chrome Web Store. Show the author that you are not indifferent to his creation.
