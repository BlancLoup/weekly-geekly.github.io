Lubuntu OpenStar Linux edition 10.04



Lubuntu OpenStar Linux edition 10.04 specially modified Linux operating system for weak computers in order to replace Windows XP in enterprises.

System features


The ease and speed of the system allow you to use all the advantages of modern programs on older computers.

System requirements:


Processor: 800 MHz x86;
RAM: 256 MiB;
Free disk space: 3 GB.

Low system requirements make it possible to use the Windows emulator wine programs on weak computers to run programs such as 1C Enterprise 7.7, 8.1, 8.2, KOMPAS-3D 10, Best Zvit, SAMO-Tour and many others.

Deployment rate:

  • Established Russian localization. All programs in the main menu are adapted for the Russian-speaking user;
  • The Public folder is accessible over the network for reading and writing;
  • All documents in OpenOffice are saved in Microsoft formats (doc, xls, ppt);
  • By default, three keyboard layouts are set: US, RU, and UA;
  • Added the label "Network Neighborhood" in PCManFM, by analogy with Windows XP;
  • Improved display of fonts compared to Lubuntu 10.04;
  • The interface is as close as possible to Windows XP, which allows you to accelerate learning and transition to Linux;
  • In the delivery are codecs, fonts, flash, as well as an extensive database of drivers for printers;
  • The system is binary compatible with Lubuntu 10.04, and therefore with Ubuntu 10.04;
  • Firefox has ad blocking and flash blocking plugins installed;
  • Added a simple program to open access to a folder over the network ( Standard -> Publish folder );
  • Installed all the necessary dependencies for installing the Wine @ Etersoft emulator.

Composition

  • Kernel 2.6.32-22;
  • OpenOffice.org 3.2.0;
  • PCManFM 0.9.5;
  • Firefox 3.6;
  • Chromium 5;
  • Opera 10.10;
  • ICQ / Mail-agent - qutIM;
  • Skype 2.1 Beta 2;
  • Email client - Thunderbird 3;
  • PDF viewer - Evince;
  • CHM viewer - CHMSee.
