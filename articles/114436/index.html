Newton Game Dynamics is open under the zlib license.



The cross-platform physics engine Newton Game Dynamics, popular among game developers, is now open and released under the free zlib license. This event happened on February 15th.

Talk on the topic of opening the source on the forum engine was enough, and many game developers have been waiting for this and such a moment has finally come.

Newton Game Dynamics enables fast and reliable simulation of the physical behavior of environmental objects and game physics. Includes scene management, collision detection and reaction to them, and also quite simple and easy to use.

Download the latest stable version is now possible not only on the official library website, but also in the corresponding section in the project on Google Code . Version 3.0 is under active development and is also available in the SVN project .

The engine is used in a large number of different projects , among which it is listed, for example, Penumbra Overture and several games for the iPhone. Available videos showing its capabilities. It can work both on GNU / Linux and MS Windows OS, and on Mac OS X and iOS.
