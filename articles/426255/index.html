Ii. In Vim, as in many other editors, there is a command mode that is invoked when you press the ':' key. The '!' has a dual meaning:

b) If it comes after the Vim command, then it will execute it without saving changes to the file. By the way, one bug (or, if you wish, a feature) is connected with this - if you did something with the text, then even if you return it to its original state, the editor will not close, if you do not save it or specify '!'.
