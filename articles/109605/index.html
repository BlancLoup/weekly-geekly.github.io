Update: Evernote 4.1 for Windows

Evernote 4 for Windows this morning received some nice improvements. If you're already using Evernote 4, check for updates. If you are still working with Evernote 3.5, you can download the fourth version here . Automatic update from version 3.5 to 4 will take place soon.

And now about the new features.

Notebook Sets


Organizing your notebooks has become easier. Sets allow you to group related notepads. For example, you can create ‚ÄúWork‚Äù and ‚ÄúPersonal‚Äù sets. The first can be folded all the notebooks dedicated to the project, and the second notebook with personal content.



Creating a set
To make a set, drag one notebook to another, or right-click on a notebook and add it to a new or existing set.

Search in sets
Let's say you are looking for something that is precisely located in a particular set, but which notebook you are not sure about. It's not a problem. You can search through all the notebooks included in the set by opening the advanced search and selecting the set of interest.



It is great in the sets that they will be visible on all Windows and Mac computers on which you have Evernote installed. The ‚ÄúWork‚Äù set can be kept open in the office and the ‚ÄúPersonal‚Äù set at home.

Create more notebooks


We increased the total number of synchronized notebooks that an account can hold from 100 to 250. This change will be useful for teachers who organize work with students through Evernote, for teams collaborating on projects using the appropriate Evernote features, and for all other people who use Evernote to organize their work.

Improved note information panel


The note information panel now has three possible states, so you can choose how much detail you want to see on the record.



Possible states (in the figure above, from left to right):
  1. panel closed;
  2. the panel is partially open, only labels and source address are shown;
  3. the panel is fully open, showing labels, source address, geolocation data, date of creation and update, etc.

Other improvements


  • Installing the application on the computer is now possible even without administrator rights.
  • Improved search in notes for task flags and records containing encrypted fragments.
  • Added support for Korean and Traditional Chinese in the interface.
