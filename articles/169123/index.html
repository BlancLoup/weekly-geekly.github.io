There is also the task of generating knowledge, but for now we will not consider it, with it, in general, scientists seem to cope as usual. Write us a program


I think every IT specialist has heard these words. Often this is an attempt to solve their problems at our expense, because if the program does not solve the problems, then this is a bad program, and if it does, then ‚Äúthat's how I came up with a good idea!‚Äù.
