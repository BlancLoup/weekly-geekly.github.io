Tools for Designers and Developers



I suggest to get acquainted with a small set of web development tools that can make your life easier. All services are free and easy to use, but (as usual) there is a minus - they are all in English, although when and who did it stop?)


Js feeddle



JS Fiddle is a handy tool for setting up and debugging a web application. The essence is simple, we write the application - we get the result. Everything is divided into four blocks - HTML, CSS, JavaScript and the result block itself. The most interesting thing is the ability to use one of several popular JavaScript Frameworks, such as Mootools, jQuery, Prototype, YUI, Glow Core, Dojo, Processing, ExtJs Core, Raphael or RightJS. But, if not one of the libraries does not suit you, we load ours (the link to the resource is indicated). To work with the service registration is not required, which is very pleasing.


Forrst



Founded by Kyle Bragger, the Forrst online community focuses on the niche of web designers and developers. The service has more than 20 thousand registered users. Here you can download site design , links , code for criticism or ask a question . An excellent resource for finding inspiration or answers to questions from people who know what they are saying. For registration you need an invite.


DB desgnr



These services will allow you to quickly and visually build a database architecture. Quite a powerful tool that supports MySQL, Postgres, SQLite, MSSql and Oracle. After creation of all tables we receive ready sql dump. The app itself is based on the Google App and if you have a Google account, you will not have to register.


Go mocking bird



Go mockingbird this interactive tool makes it easy to create, view, or share website or application layouts. It is possible to export your designs to PDF or PNG. Of the minuses - only paid registration will save projects. You can rate all features here.

I hope you found something interesting for yourself. Thanks for attention!
