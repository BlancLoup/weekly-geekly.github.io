Invisible Macintosh HD

Typical solutions did not help, or errors crashed, but I was lucky, however, and I got my way.

Solution to the problem:
1) Download Xcode (DeveloperTool), install
2) open the terminal, write
3) cd / Developer / Tools / click enter
4) further: ./GetFileInfo / we press Enter
5) if we see the line attributes: aVbstClinmedz, then enter
