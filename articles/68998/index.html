We leave in DOS, in normal, pure DOS

Sometimes you need to be loaded into DOS, for example, in order to run the hdd diagnostic system (like mhdd) or to see a 256 byte demo . But we don‚Äôt have to convulsively search through the attic in search of an old floppy disk and blow the disk drive, we don‚Äôt even need to remap the partitions to hdd for fat16, don‚Äôt even need to spoil the disc and look for an old school friend with a black and white monitor, 386 processor and a big beard.
Dos can be downloaded via memdisk.

1) Install the syslinux package
2) Find the memdisk file from this package (I had it in / usr / share / syslinux)
3) Copy memdisk to / boot
4) We take the floppy image with msdos (I can, already with mhdd and the puls demo )
5) Copy the image to / boot too
6) We supplement /boot/grub/menu.lst with this paragraph:
title MSDOS
root (hd0,0) # disk number to change to the desired
kernel / memdisk
initrd /Dos6.22.img
7) Reboot and nostalgic

UPD: I know that there are 9000 ways to load into a dus sidyuk, flash drives, zip, streamer, punched cards , but this all requires additional equipment and media. This method does not require anything other than the installed grub and the Internet.

UPD / 2: In this way, you can diagnose the screw on which the mhdd image itself is located.
