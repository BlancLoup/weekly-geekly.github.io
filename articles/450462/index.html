Russian translation of the course "Introduction to computer science with MakeCode for Minecraft"

Everyone, everyone, everyone who teaches computer science to children of 10 - 14 years old!

The link is a Russian translation of the course ‚ÄúIntroduction to Computer Science with MakeCode for Minecraft‚Äù .

According to the link, the course page will most likely be displayed in English and will not be able to switch to another language, but, like that gopher, there is still a Russian translation. It will be necessary to do so:

  1. go to the Minecode editor minecraft.makecode.com page
  2. switch to Russian there via settings (gear in upper right corner)
  3. go to minecraft.makecode.com/courses/csintro again

The course teaches the basics of programming using a block visual programming language like Scratch. Created programs run in the world of Minecraft. For geeks who want to grow up in JavaScript developers from a young age, a tab is available in the editor that allows you to do the same, but in JavaScript. Page editor .

Come in, look, maybe something can be used in the educational process.

I translate the course in my free time from architect work, and also work as a junior editor of what others are translating under the MakeCode project (the main editor is the spellchecker in my Chrome). If there is a desire to help children of Russia to learn the basics of programming - I am waiting for everyone on crowdin.com/project/kindscript - there is still a lot of work, for example, the documentation of the blocks has not been translated.
