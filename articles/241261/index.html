Microsoft Azure Cloud Hot Announcements: Bigger, Faster, and More Open

On October 20, Satya Nadella, CEO of Microsoft and Scott Guthrie, CVP Microsoft at the CloudDay event spoke about the company's cloud computing strategy and made a number of important announcements regarding the technologies and partnerships of the Microsoft Azure cloud platform at the event .





Among the announced news were the following:

  • New virtual machines of the G series - the largest in the market of public cloud services
  • New Premium Storage - high-performance, low-latency storage.
  • New Azure Marketplace service for finding and deploying solutions from startups, partners and corporate customers
  • New cool partners in the face of CoreOS and Cloudera
  • Launch of the new hybrid Microsoft Cloud Platform System solution - ‚Äúcloud in box‚Äù for your data center

In addition, during the presentations were revealed some data on the Microsoft cloud. On October 21, AzureConf 2014 online conference was held , during which several interesting announcements about Azure technologies were made.

About all the updates and announcements in more detail below.

New G-Series Virtual Machines


We are announcing a new G-series of virtual machines in Azure, which are characterized by increased performance, more memory and much larger SSD storage: up to 32 cores per VM, up to 450 GB of RAM, up to 6.5 TB of local SSD.

The G5 virtual machine is the largest in the market of public clouds in terms of memory and storage.

G-series virtual machines are based on the latest Intel Xeon E5 v3 processor family .



If you are interested in such facilities please follow the survey link .

New Premium Storage Storage


Azure Premium Storage is a new offering for organizing storage in the Azure cloud. Premium Storage is based on SSDs and is specifically designed to support high I / O load. With the help of a new type of storage, you can select the disk of the size you need, adjust its performance indicators, connect such disks to your VMs.

Azure Premium Storage offers up to 32 TB of storage with more than 50,000 IOPS for each VM and provides latency, unique to public clouds, in less than one millisecond for reading operations.

New storage capabilities and new virtual machines allowed us to certify not only Dynamics solutions, but also Exchange Server for deployment in the Azure cloud.

New Azure Marketplace Service


Azure Marketplace is a new service in which startups, independent developers and corporate clients can search and publish cloud solutions based on any technology. With just a few clicks, you can quickly find, buy and deploy a solution that interests you in the Azure cloud.

One of the first partners that were added to the Azure Marketplace was CoreOS and Cloudera solutions.

Access to the Azure Marketplace will be open on the new portal for the past week.

Partnership with CoreOS and Cloudera


Microsoft Azure supports the deployment of Linux-servers on the basis of both user-loaded distributions and images offered out of the box. You can find CentOS, Ubuntu, SUSE and openSUSE, Oracle Linux distributions in the Azure Virtual Machines Gallery.

Every fifth VM in Azure runs on Linux. Microsoft is announcing a partnership with the lightweight distribution of CoreOS .

In addition to the Linux distributions already proposed, developers will be available from the box an image of a CoreOS-based virtual machine that can be deployed in a few clicks in a cloud environment.



Another important partnership that was announced at the event was a partnership with Cloudera , the provider of Hadoop distribution for big data tasks.



A company spokesman, Mike Olson (Chief Strategy Officer, Cloudera) spoke at the event and spoke about the importance of the cloud integration of his decision, as well as the scenarios for the use of Claudera in the cloud and in the hybrid environment.





Mike demonstrated how it is possible to deploy a large solution of his company in Azure in a few clicks and then process the data in Microsoft analytics tools - Excel and Office 365 BI.

Microsoft Cloud Platform System


The big announcement of the event was the presented solution Microsoft Cloud Platform System , which is a pre-configured hardware and software complex, ready to connect to the data center to implement the Azure-based cloud platform.

The hardware solution partner is Dell. Each such complex (rack) corresponds to the following characteristics:

  • 512 cores on 32 servers (Intel Ivy Bridge, E5-2650v2 CPU)
  • 8 TB RAM, 256 GB per server
  • 282 TB of storage
  • 1360 Gb / s inside one complex
  • 560 Gb / s between such complexes
  • up to 60 Gb / s with external networks

Below you can find the image and architectural diagram of the entire Microsoft Cloud Platform System solution.



More details about the Microsoft Cloud Platform System will soon appear; detailed information will be presented as early as October at the TechEd, Europe conference in Barcelona (October 28-31).
