HPE MSA Entry Level Innovation Arrays: A More Available Flash Array And Rich Decision Integration

How does HPE invest in expanding the capabilities of MSA entry-level arrays with SSD support for companies with different budgets and storage needs?

More and more good news for small and medium business owners who are looking for an entry-level disk array - they are becoming more productive and easy to manage. HPE continues to develop MSA arrays so that you get more speed, data availability and mobility. I suggest taking a closer look at the innovations that will allow you to get much more from MSA.



A proven entry level array with a true business class ecosystem and new features.

It's no secret that the HPE MSA storage system is not a new player in the market. For nearly two decades, we have been improving and improving this product line. Last year, a new generation of MSA 1050, 2050 and 2052 was introduced with doubled performance at the same price as the previous generation. Today, with more than half a million units deployed, HPE MSA is the industry-leading FC SAN Array.

So what's new in MSA?

HPE Storage File Controller offers best-in-class services for block and file access to data. Take any HPE MSA 1050, 2050, or 2052 device and add the HPE Storage File Controller to create a single, effective file and block storage solution for working with structured and unstructured data. Thanks to advanced data services and flexible configurations, you can now optimize your SAN investment with HPE Storage File Controller by utilizing the unused capacity of your MSA array for shared Windows files. HPE Storage File Controller can be deployed in traditional or cluster configurations for high data availability. The HPE Storage File Controller perfectly complements file environments based primarily on Microsoft, especially in cases where file servers are outdated.

HPE and Zerto: hypervisor-based replication

HPE, together with the cloud solution provider Zerto, has joined forces to provide you with an unrivaled service that provides an almost synchronous, continuous availability solution that has mobility environments and the flexibility of multi-cloud technologies.

As part of data availability, we are talking about replicating MSA or other arrays to MSA to create a disaster-tolerant solution ‚Äî with very low RTO and RPO. This gives you the opportunity to use a cheaper array as a corporate repository used for disaster recovery, which, in turn, reduces overall costs. You can use HPE MSA and Zerto solutions to increase the productivity and mobility of the application work environment between two HPE MSA arrays and / or other HPE storage systems. Applications and workloads running on MSA can also be replicated to your primary data center or replicated to the cloud without risk. For more information on this solution, download HPE MSA and Zerto Hybrid Cloud-based Replication Technical Documentation.

MSA 1050 Platform: Performance Boot with New Firmware Update to Version v270

Thanks to the new update, the youngest model 1050 gets more than 30% performance gains on a random write and read operation. This is great news if you use SSD on your system. The new update is already available for both new and existing systems MSA 1050.

LDAP Support: Simplify User Authentication

We have enabled LDAP support in MSA 1050/2050/2052 models with the release of V270 firmware. Once user groups are configured on all MSA in your organization, users can be authenticated to any MSA through Active Directory. This allows you to centralize authentication rules and simplify access for users who have multiple accounts for multiple MSA systems.

Encrypted Drives: Deploying SSD / HDD Hybrid Solutions

MSA 2050 now supports encryption with self-encrypting spindle drives and local key management capabilities within the array. Not so long ago, support was added for two 800GB and 1.6TB solid-state drives. Accordingly, when you need to implement a solution that requires encryption, you can easily build it on HPE MSA.
