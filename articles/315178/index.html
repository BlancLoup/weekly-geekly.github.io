News and Releases


  • PHP 7.1.0 RC6 - The latest release candidate. The PHP team asks everyone if possible to connect to the test. If critical problems are not found, then at the end of November the final release of PHP 7.1 will be released.
  • PHP 5.6.28 , 7.0.13 - Updates to current branches with security fixes.
  • PrestaShop 1.7 - Popular E-commerce solution with new architecture on symfony components and PHP 7 support.
  • Symfony Moscow Meetup - Symfoniacs - # 10 - November 17, Moscow will host the tenth meeting of PHP / Symfony developers.

Instruments


  • krakjoe / ui - PHP 7 extension for developing cross-platform graphic applications based on libui . An example of the implementation of the game "snake" and video modem is available . There is documentation . In his post, the author writes that PHP 7 is not only suitable for the development of graphical applications, but also very fast and efficient.
  • dannylinden / aws-lambda-php - A tool for running PHP code on AWS Lambda.
  • EasyEngine / easyengine - A command line tool for conveniently managing WordPress sites on Nginx, PHP / HHVM, MySQL.
  • nategood / commando - A library for implementing console commands with a simple fluent interface.
  • jenssegers / imagehash - Perceptual hash for images - allows you to determine the degree of similarity of images.
  • simshaun / recurr - Library for working with the rules of repetition ( RRULE ), simplifies working with recurring calendar events.
  • bernardphp / bernard - A library for managing background (deferred) tasks with support for various backends: Predis / PhpRedis, Amazon SQS, Iron MQ, Doctrine DBAL, Pheanstalk, PhpAmqp / RabbitMQ.
  • woohoolabs / yin - Framework independent library for JSON implementation : API . An alternative implementation of neomerx / json-api .
  • aszone / avenger-sh - A tool to automatically search for potentially vulnerable sites.
  • akalongman / php-ip-tools - A tool for working with IPv4 / IPv6 addresses.
  • object-calisthenics / phpcs-calisthenics-rules - Object Calisthenics rules for CodeSniffer.
  • spatie / once - Convenient implementation of memoization for PHP.
  • Robo 1.0.0 - A tool for running tasks.
  • clue / php-socks-server - SOCKS proxy server implementation (SOCKS4, SOCKS4a, SOCKS5) based on React PHP.
