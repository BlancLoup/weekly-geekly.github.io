Announcement of laboratory work on the development, testing and management of the life cycle of software for Visual Studio 2015



We are happy to share with you the news about the availability of a new virtual machine containing the latest version of Visual Studio, TFS and all the necessary infrastructure to explore and evaluate all the capabilities of Microsoft development tools.

A virtual machine is accompanied by a set of new labs that will help evaluate the capabilities of development tools and try new scenarios and tools, as well as a dedicated blog that publishes the latest news and updates.

You can use the finished virtual machine using the instructions , or download and install Visual Studio 2015 on your own.

List of components required for laboratory work and contained in a virtual machine:

  • Microsoft Windows Server 2012 R2 Standard Evaluation
  • Microsoft Visual Studio Enterprise 2015
  • Microsoft Visual Studio Team Foundation Server 2015
  • Microsoft Office Professional Plus 2013 (Word, PowerPoint, Excel, Outlook)
  • Microsoft Visio Professional 2013
  • Microsoft Release Management for Visual Studio 2013 Update
  • Microsoft SQL Server 2012 Standard
  • Examples of user data to perform 24 laboratory work

Laboratory works in Russian:

  • Flexible portfolio planning and management with Team Foundation Server 2015 Microsoft Windows Server 2012 R2 Standard Evaluation
  • Continuous Delivery with Release Management for Visual Studio 2015
  • Creating and using manual tests in Microsoft Test Manager 2015
  • Visualize branching and merging in Visual Studio Team Foundation Server 2015
  • Creating the right application: generating Storyboards and collecting feedback using Visual Studio 2015
  • Team Foundation Server 2015 tools for interaction between development teams
  • Identifying problems in an industrial environment with IntelliTrace and Visual Studio 2015
  • Research Testing in Microsoft Test Manager 2015
  • Introduction to using Git with Team Foundation Server 2015
  • Overview of Coded UI tests in Visual Studio Enterprise 2015
  • Introduction to Team Foundation Build 2015 (vNext), Test Run Analysis and Machine Group
  • Manage test situations in Microsoft Test Manager 2015
  • Testing Web Performance and Load in Visual Studio Enterprise 2015
  • Increase Developer Productivity with Visual Studio Team Foundation Server 2015
  • Unit testing, code coverage and similar code analysis with Visual Studio 2015
  • Analysis and improvement of code quality with Visual Studio 2015 tools

Laboratory work has been updated, and new scenarios have been added, such as working with a new build subsystem, working with requirements using SmartOffice4TFS, etc.
