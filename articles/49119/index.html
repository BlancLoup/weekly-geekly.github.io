Watching user

The userfly service allows you to visually see what and how the user did on your page.
So far, all this is in development and apparently a lot more needs to be done, but the foundation can already be tried.

After registering, userfly generates a small javascript that needs to be inserted into the header pages. After that, all movements of users will be recorded, and you will soon be able to see how a small video, what visitors on your page do, what they are looking for, what functions are used more often, etc.
