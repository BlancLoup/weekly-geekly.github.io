Completing (?) The stories with the protection of Spore.

EA today announced changes to the DRM Spore system


Increase the limit of installations from 3 to 5.

Data from EA itself:

  • Total number of activated copies: 437 138
  • Activated on only 1 computer: 86%
  • Activated on 2 computers: 14%
  • Activated on 3 computers: 0.4%

Five installations of one game should be enough for almost everyone.

The account is now attached not to the copy, but to the installation . Each installation will have its own name (screen name), history of achievements, etc ...

In the event of disabling the authorization servers, Electronic Arts promised to release a patch that would remove DRM from the game.
