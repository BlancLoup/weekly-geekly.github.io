PaaS "all in one"

I want to draw attention of habrayusers to potentially very curious new service: DotCloud .

What is he curious about? About three years ago, the first more or less popular PaaSs appeared (Platform as a Service): Heroku and Google App Engine , each of which is focused on a specific technology stack (Ruby and Python, respectively). Since then, various PaaS'ov appeared quite a lot, but they did not have anything fundamentally new.



DotCloud is surprised by the number of supported technologies. The main page claims support for Python, Ruby, PHP, Node.js (!), Java, several databases, both SQL and NoSQL, and such things as RabbitMQ, memcached, Hadoop, etc.

I think many immediately have a question about how well it all works and how it is implemented. Unfortunately, I cannot answer the first question - there is no open registration yet, but you can leave a request for an invite. Invites come slowly due to the fact that, as we found out through the support, there are a lot of applications.

However, it is stated that several companies are already using DotCloud in production.

Information about the technology stack that DotCloud itself uses was found : the main part is written in Python (with Pylons and Django as a web framework), Ruby / Sinatra is also used; as PostgreSQL, Redis and Riak databases. The components of the system interact with ZeroMQ.

They also have an organization registered on github, but it is not clear which projects are actually used, and the latest update dates are not very recent.

Anyway, the service promises to be very interesting.
