Synology Announces the Official Release of DiskStation Manager 4.0

On the eve of International Women's Day, Synology announced the official release of DiskStation Manager 4.0 (DSM 4.0). For 50 days, during which beta testing was conducted, the Beta DSM 4.0 version was downloaded a record 90,000 times, an average of one download per minute. Synology is grateful to all users who took the most active part in testing.



‚Äú The DSM 4.0 release is a landmark in the history of Synology, testifying to the company's desire to provide its users with unique technologies, a simple personal cloud management concept and effective business solutions. And it's absolutely free , ‚Äùsaid Rosiel Lee, product manager for Synology Inc. With DSM 4.0 you get:

Easy-to-use cloud services - Cloud Station provides simple and automated file synchronization with the ability to track change history for data recovery. QuickConnect allows you to synchronize without having to configure port forwarding rules. Unlike licensed services, Cloud Station is growing along with user requests, while its use is absolutely free. Moreover, DSM 4.0 simplifies system monitoring to the maximum - thanks to sending push messages through Microsoft Windows Live Messenger, Skype and DS finder, the user always keeps up to date.

Reliable tools for business - Now users have the opportunity to integrate DiskStation into the VLAN environment for flexible creation of segmented networks on a single physical wire. DiskStation can also function as a DHCP server, distributing IP addresses to various devices. DSM 4.0 supports iSCSI LUN backup with a point-in-time backup technology, incremental backup and disk space recovery, which allows you to create snapshots and optimize task performance. Full VPN support gives Synology users the ability to create a low-cost private network for the secure exchange of information between individual nodes. In addition, DSM's intuitive interface allows you to quickly and easily edit Windows ACL settings.

Intelligent Management - DSM 4.0 implements a more efficient approach to installing and updating software, since all applications (Photo Station, Download Station, etc.) are presented as packages that can be added to the system as needed. Integration with HTML 5 and CSS 2D Transforms, Pilot View and Instant Preview - all this facilitates the transition between applications. Desktop widgets that display information about system resources simplify server administration even more.
Functional Video Surveillance Solutions - Surveillance Station now supports over 1000 IP cameras and accelerated video transmission. In addition, the deployment of a single fisheye camera (‚Äúfisheye‚Äù) allows you to cover the area at any angle without dead zones. Live-view Analytics provides real-time analytics through several channels and informs the user about the occurrence of abnormal situations.

Extended Mobile Support - Synology has set its sights on supporting Amazon Kindle Fire and developing the DS finder for Windows Phone 7. DS photo + and DS file are also available on Android tablets. Now, even more users have been able to use terabytes of information stored on the DiskStation, at any place and at any time.

Additional updates include application portals, proactive virus protection, hotspot support, improved configuration backup, Picasa Web Albums and Twitter support at Photo Station, etc.

Availability
Synology DSM 4.0 is free for users of DiskStation x08 series and later.
