New version of Evernote for Mac: beta testing, new features and many fixes

Let me introduce updated Evernote for Mac (version 1.5). This version includes a number of features and enhancements most frequently requested. Here is a list of the most notable of them.

Download New Evernote for Mac ¬ª

Beta testing



We invite the most courageous users to take part in beta testing of Evernote for Mac. By selecting the appropriate checkbox in the program settings, you will be able to receive updates on the latest beta versions. This means that you will have access to more frequent updates and new features earlier than the rest of the Evernote user community, but at the same time you will have to work with a less stable build. Once again, we do not offer this option by default, but if you try, you will greatly help us in improving Evernote.

Thank!

Toolbar for Safari
We recently discovered that our web clipper in Safari acts in an unsuitable friendly elephant manner - it removes other people's icons in the toolbar, then hides itself. In this update, we have improved the situation, although, judging by the reviews, some problems still remain. We will still work on it.

And finally ...

New features
  • Added support for viewing handwritten notes.
  • Improved search for texts in Chinese, Japanese and Korean.
  • Your inbox email address has been added to your account information window.
Fixed bugs
  • Improved import of large ENEX files.
  • Fixed an issue that sometimes resulted in a pink error message.
  • Fixed a bug that gave an empty note when combining notes.
  • Fixed problem with searching for notes by words surrounded by punctuation marks.
