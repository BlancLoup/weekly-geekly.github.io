New version of GNS3 emulator



Have you ever set up Cisco routers? Or maybe you encounter similar tasks every day because you are the person who is responsible for the stable and reliable operation of the data network in your organization? In this case, you probably had an idea how to simplify your life as much as possible and increase the efficiency of your work. Details below.

If you haven‚Äôt had many years of experience in setting up / designing networks, and the task has been set and there is nowhere to retreat, then the software specially designed for this case will come to the rescue. You have to agree, before you build a configuration on real hardware (which at best may simply not make money, and at worst it may turn out that you bought the wrong equipment) it would be much more efficient to assemble the same topology but in an emulator. And open source gives this opportunity. Thanks to such programs as dynamips, dynagen, GNS3, you have the opportunity to run "full-fledged" Cisco routers on an ordinary PC. And the emulated platforms are not limited to classic Cisco routers.
Now it has become more convenient to do. This month a new version of the GUI interface for emulating devices based on Cisco IOS - GNS3 0.7.3.
GNS3 is a graphical interface for such programs. Both dynamips and dynagen make it easy to emulate a whole range of Cisco devices:
1710,1720,1721,1750,1751,1760, 2610,2610XM, 2611.2611XM, 2620.2620XM, 2621.2621XM, 2650XM, 2651XM, 2691,
3620.3640.3660.3700.3725.3745, 7200 and many others.

List of changes in the new version:

+ It became more convenient to work with ATM / Frame Relay switches
+ PCAP support for Qemu based devices (ex. PIX, ASA)
+ New node type - EtherSwitch router with NM-16ESW module
+ Ability to configure AUX port to 7200
+ Autosave feature

Full list of changes is available at:
www.gns3.net/news

GNS3 - cross-platform software - will work in Windows, Linux, MacOS X. As stated above, it is free software, so GNS3 can be downloaded absolutely free. GNS3 integrates easily with Wireshark analyzers. You can also connect the emulated network and network interfaces on a real machine.
Supported types of images of the emulated hardware operating systems are IOS / IPS / PIX / ASA / JunOS.



! The GNS3 package as well as Dynamips / Dynagen does NOT include operating system images, such as Cisco IOS.
You need to have IOS / ASA / PIX images (have a contract, download from the purchased equipment, get it in another way) to use GNS3.
GNS3 is also indispensable for those who are preparing for the CCNA / CCNP exams (there have already been several articles on this topic, use the search if you are interested).
By the way, at the moment he is actively looking for volunteers for testing, as well as translators for translating documentation and software interface.
The main developers at the moment - Jeremy Grossmann, Xavier Alt.

It should be noted that according to the above characteristics, this piece of software has no equal.
