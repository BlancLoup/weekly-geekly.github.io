Rad Studio 2011 XE: New in Delphi, C ++ Builder, RadPHP and Delphi Prism

A very recently released new release of Embarcadero RAD Studio XE (2010). I suggest to get acquainted with the new features of this environment in comparison with the version 2010.



Delphi and C ++ Builder



Develop multi-tier applications with DataSnap

  • New wizards for creating server applications and client modules
  • Wizards for building DataSnap server applications using C ++ Builder
  • Encryption-Compressed Streams
  • Support for roles in the authentication system


Additional utilities

  • AQTime Standard Profiling Toolkit
  • A set of utilities for logging and advanced debugging CodeSite Express
  • A suite of Software IP Works components for developing Internet applications and working with IP family protocols
  • A utility for comparing and finding differences in files Beyond Compare Text Compare


Cloud computing

  • New components for working with basic Windows Azure storage systems (blobs, queues and tables are supported)
  • The ability to quickly develop software being developed on Amazon EC2 servers


Integrating Subversion Support into the IDE

  • Subversion has been integrated into the project manager (Project Manager) and the source code history manager (History Manager)
  • Basic repository operations are supported: import, update from server, commit and view logs.
  • Wizard to view the differences and results of the merger
  • Source code for working with Subversion tools through the Open Tools API is available as an open source project.


Tools to increase productivity and convenience

  • Code generation by the modeller is improved for getting better code.
  • Added the ability to generate sequence diagrams from methods in a Delphi application
  • Added code formatting settings by Ctrl-D
  • Expanded text search and IDE Insight menus for quick access to environment features
  • Added some hotkeys to move between changed lines of code.


Automate the build process

  • Added FinalBuilder automatic build management pack
  • Added Formatter.exe for automatic formatting of source codes.
  • Added AuditsCLI.exe to automatically start source code auditing and metrics calculation
  • Added GenDocsCLI.exe to automatically generate documentation


Language, compiler and library improvements

  • Updated VCL, RTL and STL libraries
  • Regular expressions support added
  • Extended TStrings and TDateTime classes functionality.
  • Extended support for RTTI in C ++ Builder, added new RValue links, movement semantics for std: string and some other VCL improvements for C ++
  • Added support for internal tabs in the OpenTools API
  • Improved compiler performance


ToolCloud integration

  • Added ability to work with Embarcadero ToolCloud


When you buy RAD Studio XE, you now get a license for all previous versions of Delphi and RAD Studio as a gift!

New to Delphi Prism XE



  • Subversion support
  • Cross-platform development on Windows and Mac OS X using MonoDevelop
  • .NET 4.0 and SilverLight support
  • The IDE is now based on the Visual Studio 2010 Shell with all the consequences
  • Support for parallel programming, functional programming (new if, case and for), BigInteger constants. Added new syntax for defining extension methods
  • The ability to import, copy and paste C # code with automatic conversion into Delphi Prism code
  • The ability to develop applications for the iPhone OS
  • Ability to obfuscate code directly from the environment
  • Raize CodeSite Logging System Integration


RadPHP


With the new RadPHP, you can develop and debug websites in PHP right from the environment. Built-in libraries allow you to create applications for FaceBook with the ability to expand functionality with third-party components. Developing data access applications has never been easier.
