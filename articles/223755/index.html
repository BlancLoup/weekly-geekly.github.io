DevConf 2014: Writing a functional, reliable and fast web application on Go


Dmitry Vyukov, developer, Google working on the core of the Go language will speak at DevConf 2014

Support Dmitry, so that the report is included in the program of the DevConf 2014 conference.

We offer you an interview with the author of the report.


- Tell a few words about yourself?
He graduated from Bauman Moscow State Technical University, since 2002 he has been working in the fields of web applications, billing, telecom, VoIP, social networks. Currently, Google is working on systems for dynamic code verification - AddressSanitizer (search for memory errors), ThreadSanitizer (search for multithreading errors); as well as support for parallelism and multi-cores in the Go language (parallel garbage collection, gorutin scheduler, channels, synchronization primitives, etc.). The winner of the Intel Threading Challenge 2010, one of the first in the world, was awarded the title Intel BlackBelt. He has a particular interest in the field of multithreading and multi-core, the author of dozens of innovative synchronization algorithms, developed the Relacy tool for their verification.
