The next version of the browser Vivaldi TP4 is ready

Hello!

As you already know, in recent days we have been actively working on the preparation of the next (by the way - the last version before the release of the full beta version) version of the Vivaldi TP4 browser. This is a technical prerelease, which is a kind of stage in the daily development process of the browser. A little more than two months have passed since the release of the previous version of TP3, and today we are pleased to show what has changed since that time.

A lot has changed. Several hundred bugs were fixed, the browser kernel version was updated, but beyond that we improved many functions and also added a significant number of new ones.

First of all, we have done a great job in expanding the possibilities of interface settings. Now you can change the color themes of both the application window itself and the start page (Express panel). You can even assign your own color or change the background image of the Express panel. Also, functions were added to scale the interface, change the location of the browser panels and many other options. It should be particularly noted that all these changes occurred according to the wishes of our users, that is, you.



Rumor has it that now Vivaldi browser has the most customizable interface of all browsers, and the number of available combinations of settings is more than 155 million! Generally speaking, the most noticeable changes since the release of TP3 are as follows:
  • Customizable start page (Express panel)
  • Interface scaling
  • Color schemes for the browser interface (in addition to the color tabs)

As a result of the increase in the number of available settings, the settings dialog was thoroughly revised. We have tried to distribute all the settings into categories in order to make it easier for you to navigate among them.



Another important direction of development has received increased attention from both users and from the actual developers. These are quick key combinations that allow you to perform popular actions using one or more keys. We have significantly expanded both the list of available commands and the possibility of self-tuning them. As a result, we even had to create a special cheat sheet so that you could remind yourself of combinations for this or that action.



A related topic is mouse gestures. Many have known them for many years, many have been actively using them. We have worked well on this function, making it more stable and intuitive. Also, considering that today many people prefer to work on laptops and use a touchpad instead of a mouse, we tried to make mouse gestures comfortable when working with a touchpad - while using the laptop keyboard instead of the right mouse button.

Also, a very popular tab switching function using the mouse wheel has been implemented. And many many others. Enumerating each option does not make sense - too much text for such an article. In short, here is a complete list of the most important changes:
  • Browser launch settings
  • Improved spatial navigation
  • Shortcut Keys (Cheat Sheet attached)
  • Mouse gestures (including switching tabs with the wheel and activation from the keyboard)
  • Improved installer (now it is allowed to install the browser without administrator privileges)
  • Redesigned settings
  • Partial extensions support
  • Adaptation for Windows 10
  • Customizable interface
  • Performance improvement
  • AutoUpdate for Mac and Win x64
  • Improved HiDPI support
  • Improved bookmarking experience
  • Support for forward and back mouse buttons on Linux
  • ... hundreds of bug fixes

Thank you all for your help in preparing this version, and special thanks to our Sopranos - voluntary testers who have made a huge contribution to debugging the Vivaldi browser. The next landmark release should be a full beta, in which we plan to show you a lot of interesting things.
