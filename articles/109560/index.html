Visual Studio 2010 SP1 Beta Pre-release Package Released

A preliminary version of the first set of updates for Visual Studio 2010 has been released. This update package has been available to MSDN subscribers since December 7, and will be publicly available on December 9. Details under the cut

Visual Studio 2010 SP1 Beta was developed based on feedback received through Microsoft Connect, polls and forums from users. We encourage you to install the package and share your opinion with us through the MSDN and Microsoft Connect forums .

The package includes the following changes:

1. Help Viewer. This tool is a simple application that provides new key features, including a new detailed table of contents and indexing articles. Read more in the article (eng.) Jeff Braaten (Jeff Braaten).

2. Improved platform support. Silverlight 4 development tools for Visual Studio 2010 are now part of the development environment along with Silverlight 3. In addition, highly specialized changes have been added, including support for the Direct2D API in the Microsoft Foundation Classes (MFC) specific to Windows 7, DirectWrite, and Windows Animation Technologies . We also added the following important features:
  • Unit testing for .NET 3.5
  • IntelliTrace F5 for 64-bit systems and SharePoint projects
  • Performance Wizard for Silverlight


3. Runtime switch for VB. This switch allows Visual Basic developers to create applications and libraries for platforms on which the runtime environment was not previously available.

You can get more information about Visual Studio 2010 at the following resources: Official website of Visual Studio 2010 , MSDN Forums , Download Center for MSDN Subscribers , Visual Studio 2010 Service Pack 1 Beta (MSDN downloads section, activated December 9, 2010)
