Easy installation and configuration of BURG



details under the cut

I think in detail about what is BURG is not worth it, but for those who do not look out of the tank - this is a graphical OS loader.
BURG manager is a program for easy installation of BURG itself and in addition with it its emulator and themes for it. As well as customization of its basic parameters (delay time, selection of the default Axis, download new themes, restore Grub, etc.).
Install Burg.

1. To do this, download and install BUC
2. Now download and install the Burg Manager itself.
3. Now install the BURG itself (Start-System Utilities-Burg Manager)
