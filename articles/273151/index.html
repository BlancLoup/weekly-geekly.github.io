Transfer tabs to another window in the new assembly Vivaldi 1.0.344.34

Hello!

We are getting closer to the release of the second beta version, and in addition to fixing the bug that led to the unauthorized closing of the browser, we added (or rather started the implementation) a new function - opening a tab in a new window. More precisely, the transfer of an already open tab into a separate window.

Transfer tabs to another window

In the context menu of the tab, we added a new item that allows you to open a tab either in a new window or in another existing one. The list of all open windows is also displayed in the context menu:


This is only the first implementation of the function, in the future we will improve it in terms of ease of use. For example, according to the requests of many users, we plan to add the ability to create a new window by dragging a tab outside the Vivaldi window.

That's all for now. A complete list of changes can be found at the end of the article.
