We test the modular printer "3D START" - part 2. Flat Print Setup



In the first part of the review, we introduced our readers to the general features of a 3D printer 3D START. It was found that print quality deserves special attention and explanation. In the second part, we will explain how to set up the printer so that printing is smooth. We connected the monitor and set up the printer with it.

The first part of the review . If you do not configure the printer, printing is not always normal. We will show how to configure the slicer and monitor.



Temperature setting


We set the initial and subsequent temperature and diameter of the thread.



Configuring the parameter "Same filling pattern"


Choose Rectilinear hatching, its% can be increased, the print will be longer, but the filling is more dense.

Types of supports


There are three types of support: Rectilinear, Rectilinear Grid, Honeycomb.

Types of filling


There are more fill types than support types. In addition, they may differ in density depending on the specified percentage of filling in the settings of the slicer.
