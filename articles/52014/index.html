Personal Wiki in your pocket

I want to introduce Habr to the community offline wiki application bLADE Wiki.




License Type: Free
Supported OS: Windows XP | 2003, Windows Mobile 5 | 6
Additional OS Requirements: Microsoft .NET Framework 2.0

This small application is a personalized wiki format database. Written in C # programming language, using the .NET 2 Framework, source code is not available, the author does not plan to share them. The system is constantly evolving since December 2006, that is, for more than two years. The intensity of updates can be judged by the blog - on average up to 10 new versions a month. Of course, with the passage of time, the new versions have become less, this can be explained by the completeness, the reasonableness of the product. He is almost perfect. We must pay tribute to the author, who listens with great enthusiasm to the requests of users and, within his means, adds functionality (for which he is honored and respected!).

A few words about the application itself.


In parallel, there are 2 versions - for PC and for mobile devices. They differ somewhat in appearance, but are fully compatible with each other (file synchronization using ActiveSync). More information can be found on the website of the author. I will describe the version for the PC, since I do not have devices on Windwos Mobile. The program has the ability to run without installation (Portable), which is very convenient for use on USB media - everything is always with you. It is intended, as well as the majority of Wiki, for maintaining the personal knowledge base, system of notes. Unlike tree systems, it has a flat namespace inside one database (there may be several databases). Each page is a file in the file system, the file extension is configured. Here I must mention some features. Since the program supports Unicode, each page with a Russian header generates a file in the file system with an unreadable view, like ‚Äú% D0% 9C% D0% BE% D0% B5.txt‚Äù is the coded word ‚ÄúMine‚Äù. Everything is in order in the application itself, everything is displayed correctly. English names in the file system are displayed correctly (readable). Since the project fell asleep, I still prefer the original version. Perhaps if there are people who are ready to wake the author, the projector has a future. I will be sincerely happy about it!
