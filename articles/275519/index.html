Icons in the bookmarks bar and the weight of everything in the assembly Vivaldi 1.0.375.3

Hello!

For various reasons, we did not have time to release the weekly build last week. Meanwhile, we have made quite a few changes to the browser, which we would like to show you and offer for testing. Immediately you have to disappoint Mac OSX users: your version in the current build is missing - at the very last moment a bug with auto-update appeared. We will try to fix it for the next build, which will probably be released this week.

From the obviously noticeable changes you can note the appearance of icons in the lists on the bookmarks bar:


There is still a lot of work to be done, including sorting options. But the first steps have already been taken.

We also added background images for the Express panel:


You can also briefly list the corrected errors that you reported on blogs and forums. These are previously not worked JS-addresses in the address bar, scaling the interface instead of the page when using Ctrl + Shift + Scrolling, constantly pop-up of the save password dialog, and much more. Also added new features like scrolling tabs with a wheel (when the cursor is over the tabs bar), global setting of the page scale, many new options in the settings, etc.

Full list of changes - at the end of the article.
