Opera Mini 5 beta released



Opera presents the fifth version of the popular mobile browser.

Oslo, Norway, September 16, 2009 - Opera Software today released a beta release of Opera Mini 5, a new version of the world's most popular mobile browser. Opera Mini "mobilizes" the Internet, making it accessible and easy to use for everyone and for everyone.

The fifth generation of Opera Mini browser develops the concept of full-fledged work on the Internet from any device promoted by Opera. The new interface with the Express panel for quick access to your favorite sites and tabs for convenient work with them, multiplied by an even higher speed of work, make Opera Mini the most convenient and fast web browser for phones. To make sure of this, it is enough to open the browser installed in the phone by default, type m.opera.com/next in the address bar and download the Opera Mini 5 beta version for free.

Five key browser features:
1. Elegant design. Opera Mini 5 offers a new intuitive interface.
2. Tabs. This desktop browser option is now available in the mobile version. Tabs allow you to work with multiple sites at the same time and simplify navigation through open pages.
3. Support for touch screens and keyboards. Opera Mini interface is adapted to work with websites using both the push-button keyboard and the touch screen of the phone.
4. Password Manager. Opera Mini is able to become a personal virtual data bank, saving passwords for entering social networks, an email inbox and secure applications.
5. Express panel. This is one of the most convenient ways to organize quick access to frequently used web resources. Using the Express panel, you can download your favorite sites with just one click.

You can get acquainted with new features and download Opera Mini 5 beta browser on the official website www.opera.com/mini/next

About Opera Software ASA
Opera Software ASA is an industry leader in the development of web browsers for desktops, mobile phones and other devices.
