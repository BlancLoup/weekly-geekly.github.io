We simplify web-maker's life in Windows a little

This post does not pretend to be a full-fledged article and is not advertising, but only a small lifehack, which can simplify the life of layout makers a little.

Probably every normal web-maker asked himself a question: ‚ÄúHow convenient is it to check the layout with a ready-made layout?‚Äù, Well, or something like that. Today I also asked this question.

The first thought that came to mind is ‚ÄúIt would be nice to quickly change the transparency of windows‚Äù. As far as I know, in Windows 7 there is no such function. A little googling I finally convinced. But the search did not stop there. Later, I came across a wonderful program called: See Through Windows .

With this small program, you can change the transparency of windows using a predetermined combination of hot keys. What, in fact, I was looking for.

The program interface is as follows:



From the screenshot it is not difficult to guess what the program can do:

  • Set the hot key combination to change the transparency of the window.
  • Set the transparency level. You can also see it here if you activate the checkbox with the Preview transparency value.
  • If you activate checkboxes under the Move window, using the shortcuts: Ctrl + Win + Up / Down and Ctrl + Win + Left / Right you can minimize / maximize the active window and move the window to another monitor, respectively.


Here, perhaps all that I wanted to share. I hope this utility will seem useful not only to me;)
