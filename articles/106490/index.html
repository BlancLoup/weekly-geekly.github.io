On Tuesday, Kaspersky Lab was forced to admit that hackers managed to hack kasperskyusa.com using a vulnerability in a third-party application. As a result, site visitors were automatically redirected to the malware page.

According to Kaspersky Lab, ‚ÄúThe website was a copy of the Windows XP Explorer window with a pop-up window that displays the scanning process on the local computer and offers to install a fake antivirus program. In general, the domain carried out this redirection for 3.5 hours. "
