How to quickly create a sheet in a cell using Photoshop and Excel

Somehow it was necessary to quickly make a sheet with cells, as in a school notebook.
Opened Photoshop and not a single thought. Could not figure out what tool from the standard set it to implement.
Oddly enough, Microsoft Excel came to the rescue.

And now more and with pictures.
Open Excel, create a new sheet and select the sheet or the range of cells you need:


Set the height of the cell:


Set the cell width:


Select the required number of cells (or your selected range) and copy to the buffer (Ctrl + C):


In Photoshop, create a new document (Ctrl + N).
The most interesting thing is that when you press Ctrl + N, Photoshop will create a new document based on the size selected in Excel.


Paste (Ctrl + V) and immediately see the result:


That's all so easy and fast. Thus, you can create a sheet of any size with a different cell size.
Yes, I agree, not the most elegant way, but quickly and accurately - as was important to me.
Hope you find it useful.
