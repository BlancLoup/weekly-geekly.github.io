Site owners can help in the search for missing people.



The overwhelming majority of people met with the standard response of the web server, when the URL to which the link points leads to nowhere and the standard 404th error is issued - ‚ÄúThe requested page was not found.‚Äù A number of European organizations, such as Child Focus, the European Federation for Missing and Sexually Exploited Children and some others, have taken a useful initiative for site owners to use the space on 404 pages with more benefit and place information about missing children on them.

To do this, the foundation NotFound.org created a special site where you can easily get a snippet of a frame for placement in error.html . The code will return information that will contain a photo of the missing child, name, age and telephone number, by which you can report possible information to the foundation dealing with the fate of the missing children.

Also, various hosting sites (especially free ones) are offered to automatically place such a code by agreement with customers. About 480 sites have already taken part in the initiative. It is important to note one problem - apparently, there is no correlation between the IP of the current position of the frame and Geo IP, and therefore the code returned will most likely be more useful for residents of European countries.
