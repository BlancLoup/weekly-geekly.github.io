TagScanner 5.0 build 514



For those music lovers who use iTunes + APPLE iPod on the farm, TagScanner is useful. The program works under Windows and supports Russian language. There is Help in Russian.

Program website www.xdlab.ru

In the assembly TagScanner 5.0 build 514 from 06/14/2007 fixed / added:

* Correct work with iTunes service comments in ID3v2 tags
* Fixed: Newly created scripts could be incorrectly saved for some operations.
* Fixed: Build 512 error - the comment field was not saved in ID3v2
* Fixed: Filename -> TAG: Incorrect work with Unicode names on some systems
* Fixed: It was not possible to clear the track number value for WindowsMedia files
* Translations: German, French

Information from the site:
