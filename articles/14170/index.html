In EVE Online it will be possible to play on the "alternative" OS

Icelandic company CCP Games announced that Linux and Mac OS X users will be able to play their multiplayer online role-playing game on EVE Online soon - the game is now available only on Windows.

Versions for "alternative" operating systems will appear thanks to an agreement between CCP Games and TransGaming, a company that port games to various platforms. Promised that the versions for Linux and Mac OS X will not yield to the performance of the Windows version. The ported versions should appear closer to the end of this year, although Linux users can play EVE Online even now, however, by resorting to using the Cedega emulator.

Currently, CCP Games is working on developing an update that allows players to communicate with each other as they leave their spacecraft, but the release date of this update is not known.
