News and Releases


  • Symfony 2.8.0 - Massive framework update: new LDAP components, PropertyInfo , form enhancements , an updated debugging panel , MicroKernelTrait treit for creating micro applications . Full review of changes in the beta1 announcement .
  • Symfony 3.0.0 - The release does not contain new features compared to 2.8. In fact, this is 2.8 without backward compatibility layers. Minimum supported version of PHP 5.5.9. A guide to transition from 2.x to 3.0 is available .
  • PHP 5.6.16 - List of changes .
  • Zend Framework 3 Update and Roadmap
  • Admin Interface for WordPress on JS - The company Automattic, which is behind the development of WordPress, announced Calypso - admin interface for wordpress.com, implemented on modern JavaScript-tools. Some sources mistakenly rushed to report that WordPress has been rewritten to Node.js.
