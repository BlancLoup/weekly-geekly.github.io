Yahoo announced the closure of Pipes on September 30

Starting August 30, 2015, Yahoo Pipes users will no longer be able to create new pipes. Until September 30, 2015, the service will operate in the ‚Äúread only‚Äù mode, and on September 30 it will be completely closed.

You can download the channels you created, specifying their Pipe-ID ( _id ) and the required format ( _out = json ) in the URL of the form: pipes.yahoo.com/pipes/pipe.info



For example,
pipes.yahoo.com/pipes/pipe.info?_id=ZKJobpaj3BGZOew9G8evXg&_out=json

Questions about this news can be asked at the Yahoo Developer Forum for Pipes developer forum.

Note translator -
Possible alternatives to Yahoo Pipes are, for example, on this page. I recommend to pay attention to Pipe2Py - translates the downloaded JSON into a script in Python, but you need your own server to use (free Google App Engine should come up).
