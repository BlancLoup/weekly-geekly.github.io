BitTorrent 6.0 Beta

The new version of the official client for the most popular Peer 2 Peer BitTorrent network. The sixth version is a merger of the corporate Bittorrent client with the recently acquired uTorrent . From the original Bittorrent program got experience in developing network protocols, and from uTorrent compact code and a wonderful interface. As a result, the program has become much easier, the ability to search for local peers, multiple simultaneous downloads, speed limiting, fast resuming downloads, RSS-downloader, etc. In fact, the same uTorrent.

This network in the shortest possible time became the most popular in the world, having displaced KaZaA from the first place of honor and displaced eDonkey2000. BitTorrent is fundamentally different from its counterparts - in order to download something, you need to have a special .torrent file, collections of which can be found on the web. The network works on a non-tricky principle: in parallel to how you take a file from someone, you give it to other users at the same time.

Many large developers (as a rule, open source products, such as Linux distributions) noticed BitTorrent and use this P2P network among other distribution channels of their products.
