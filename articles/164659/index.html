Ukrainian Land Registry uses OSM as default map

In Ukraine, opened access to the State Land Cadastre . Even an interactive map was written down : www.map.land.gov.ua/kadastrova-karta

Observe privacy and do not publish the names of the owners. You can see the cadastral number, if the cadastre indicates the privatization of the plot, then this fact and purpose of the plot is indicated. But nothing about the owner. And all the same interesting.

In particular, the Ukrainian optimizers are not alien to the use of open crowd-sourced map sources. By default, the map opens with a layer of maps from OpenStreetMaps .
