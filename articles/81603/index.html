Server racks from IKEA

Dutch computer scientists have found the cheapest option for server racks. These are the standard square tables LACK , which are sold for 5 euros in the furniture store IKEA. By happy coincidence, the distance between the legs is 19 "- exactly the size of a standard server unit. At the same time, the tables are perfectly attached to each other, like the Lego designer.



The LACKRack project was first introduced at the eth0 computer conference a few days ago.
