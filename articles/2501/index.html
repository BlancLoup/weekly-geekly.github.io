Venice Project uses Mozilla components

The free Venice Project P2P network, intended for broadcasting on the Internet, turns out to use Mozilla components. The Venice Project is based on the same Joltid Global Index peering technology as Skype and Kazaa, and was created by the founders of these resources, Niklas Zennstrom and Janus Friis.

Michael Arrington, the author of a note on TechCrunch , believes that the Venice Project could be an alternative to YouTube if viruses and other malicious software do not spread through the network.
