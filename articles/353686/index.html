"Closer to 5G": VMware helps telecom operators deploy new generation networks

At the Mobile World Congress 2018 conference, VMware representatives told that they intend to help providers and operators to create new generation 5G networks.

That will be involved for the decision of this task, we will tell further.


/ photo by John Leszczynski CC

Virtualize Network Services


The VMware vCloud NFV platform provides basic virtualized infrastructure for working with services and applications in a private and public cloud environment, as well as in hybrid Telco clouds. Hybrid Telco Cloud allows you to organize the operation of IoT-systems and basic network services (for example, the core of the wireless communication network architecture) and migrate them between different cloud environments.

In the future, VMware plans to expand the functionality of the architecture and implement the DPDK (Data Plane Development Kit). This is a set of transmitting level libraries, which speeds up packet processing in networks through a software framework for x86, ARM and PowerPC processors. It uses the Environment Abstraction Layer (EAL) abstraction layer, which provides a standard programming interface for hardware accelerators and operating system components.

In addition, VMware has teamed up with Intel to provide providers with the ability to configure 5G networks adapted to work with cloud environments. To do this, DPDK will receive an update that improves memory bandwidth when working with Intel Xeon processors.

Helps work with containers


Pivotal Container Service (PKS) solution is a solution for working with Kubernetes. Kubernetes support appeared on VMware vSphere , a solution for dynamically balancing the load on servers and storage and automatically restarting VMs when physical machines fail.


/ photo by Mark Finney CC

At the same time, PKS is constantly monitored for vulnerabilities noted in the CVE database. If a problem is detected, the team that monitors the security of Pivotal products immediately notifies the users about it - they have a separate page with reports on the website for this purpose. If there is a security risk, the system allows providers to update Kubernetes clusters without downtime using the PKS built-in Operations Manager tool. How to deploy Ops Manager solution on vSphere can be found in the link manual.

Simplify service development


VMware also talked about solutions that will simplify the development of service providers: accelerate their deployment, facilitate maintenance and offer customers new features. For this, the company has several products:

  • VeloCloud. This is a joint product of VMware and VeloCloud, which is a software-defined WAN-based network (SD-WAN) based on the cloud. It provides providers with the ability to manage client services remotely (as MSP ). VeloCloud's SD-WAN can be deployed on the operator‚Äôs equipment as a ready-made service, or as a virtualized network function on the client‚Äôs machines.
  • Digital Workspace. VMware recently updated Workspace ONE - a single Digital Workspace-platform for administering applications (details about the innovations and capabilities of the platform we wrote here ). Thanks to Workspace ONE, network operators will be able to administer and protect any user devices (including BYOD), as well as provide their customers with access to all applications (virtual, cloud, mobile, web applications, etc.) through a single registration portal. For example, it will help providers increase revenue from resale of software licenses or accompanying services.
  • Hybrid Cloud Extension. The tool will allow operators to transfer the VMware infrastructure to the public cloud, without requiring re-configuration, testing and modification of cloud tools. In addition, providers will be able to offer corporate customers access to large-scale clouds. The extension will enable users to create and run workloads related to IT or network functions in powerful virtual data centers and work with software-defined data center ( SDDC ) from VMware.
