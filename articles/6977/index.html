Speed ​​Dial for Firefox. We look fast.

The adoption of interesting and good features will never end :)
And for sure it will play into the hands of both browsers.
I think it is not worth spreading strongly about what it is and why it is needed at all: surely the majority of those who were interested already know / heard / saw / tried, and recently they mentioned it in a nearby charatopic . I will simply quote the main point:
Speed ​​Dial panel, which appears when you open a new tab and shows up to 9 thumbnails of the most frequently used sites. Hot keys Ctrl + 1..9 are also provided (for Mac OS + 1..9)

Of course, the development process is still progressing and the official version of the extension is not there yet, so you should not expect everything at once. For example, there is no search or drag and drop links between cells, but the basic concept is already laid.

And now some information specifically on the extension:
To display the “Speed ​​Dial” tab, use the “Speed ​​Dial” button (which can be added to the toolbar), or simply enter the “easy-to-remember” address “chrome: // speeddial / content” in the address bar. ;)

To add a site to the Speed ​​Dial, you need to use the “Set as Speed ​​Dial” item in the Bookmarks menu (or in the context menu of the tab (or in the page context menu)).

Speed ​​Dial is automatically loaded in new blank browser windows. In order for the same thing to happen in the new tabs, you will need to resort to the expansion settings (the benefit is not so much).

Once again, the official version has not yet been, so the work will not be "smooth." Those who are not afraid of difficulties or simply consider themselves to be an avid tester, thirsting for all the new - Good luck! ;)
Works with Firefox versions 2.0 - 2.0.0. *
