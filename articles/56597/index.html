Evernote for Mac OS 1.4.0: Russian version + pleasant things

Following the Windows client, Russification has reached our Mac OS application. A new version of the client for Mac OS with a fully Russified interface was released tonight. You can download it from here or through the built-in automatic updates Evernote.



The list of changes in version 1.4.0 is not limited to the support of the Russian language. We also tried to make Evernote work as fast as possible by working on usability and adding new keyboard shortcuts.

Firstly, it became possible to add selected text to notes by pressing CMD + CTRL + N (this combination can be independently redefined in the settings).

Secondly, the changes affected the work with the search. As a result, the new shortcut keys and the ‚ÄúSearch‚Äù menu section, where all relevant functions are now assembled.

Also, Evernote for Mac OS 1.4.0 added keyboard shortcuts for various formatting functions, made the Format menu more convenient, and fixed the notepad changing function from the note details panel.
