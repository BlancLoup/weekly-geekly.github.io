New devices on webOS from HP



Today (February 9, 2011) in San Francisco, HP introduced new devices on webOS: HP Veer, HP Pre 3 and HP TouchPad. And also, announced the release of webOS 3.0.


HP Veer :


OS : webOS 2.2;
Processor : Qualcomm MSM7230, 800Mhz;
Networks : HSDPA 7.2 / HSUPA 5.76; Quad-band GSM / EDGE: 850/900/1800/1900;
Wi-Fi 802.11b / g, Bluetooth, GPS (A-GPS);
Screen : 2.6 inches, multitouch, 320x400 px;
Keyboard : physical QWERTY;
Memory : 8 GB;
Camera : 5 Mpx;
Battery : 910 mAh, built-in;
Dimensions : 54.5x84.0x15.1 mm;
Weight : 103 g

HP Pre 3 :


OS : webOS 2.2;
Processor : Qualcomm MSM 8x55 (1.4 GHz);
Networks : HSDPA 7.2 / HSUPA 5.76; Quad-band GSM / EDGE: 850/900/1800/1900;
Wi-Fi 802.11b / g / n, Bluetooth, GPS (A-GPS);
Screen : 3.58 inches, multitouch, 480x800 px;
Keyboard : physical QWERTY;
Memory : 8 (or 16) GB; RAM: 512 MB;
Camera : 5 Mpx, HD, autofocus, second front;
Battery : 1230 mAh;
Dimensions : 64x111x16 mm;
Weight : 156 g

HP TouchPad :


OS : webOS 3.0;
Processor : Qualcomm 8060 dual-core Snapdragon (1.2GHz);
Networks : 3G; Wi-Fi 802.11b / g / n, Bluetooth, GPS (A-GPS);
Screen : 9.7 inches, multitouch, 1024x768 px;
Keyboard : virtual;
Memory : 16 (or 32) GB; RAM: 1 GB;
Camera : 1.3 MPx front;
Battery : 6300 mAh, built-in;
Dimensions : 240x190x13 mm;
Weight : 740 g.
