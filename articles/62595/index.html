Grammar Gravity

Quite often on Habrahabr, you can see comments that contain messages that this or that error crept into the article. Such comments do not relate directly to the topic that the article presents for public discussion, and it distracts the discussion participants from its fruitful conduct.

Such information needs to be sent personally to the author, but sometimes users find it simply too lazy to open their Habrapracht and type the necessary information there in order to notify the author about the error. What can help us?

The custom Grammar Gravity script will help you, without departing from the article and its discussion, to send the author an error message in two clicks. You seem to be writing a regular comment, but, in fact, you are sending a personal message to the author of the article! Beauty, is not it?



If it is not enough for you to simply inform the user about the errors, you want to write a full message to him, then you should install Grammar Gravity Parallel .



The extension was tested on Firefox 2 and 3, Opera 10, Google Chrome and Safari. It is possible that it will work on other browsers that do not fall into the above sample. To install the script in Firefox, you need Greasemonkey, in Opera you need to enable user scripts in the settings, in Google Chrome you need to add the options -enable-user-scripts to the options -user-scripts-dir = \ path \ to \ directory and put in the same directory script.

By sending an error message to the author with a personal message, you are doing a good deed! The world will remember you! Good luck to you!
