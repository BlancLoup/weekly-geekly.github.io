Piwik 1.7

A new version of Piwik has been released briefly:

Piwik is an open source web analytics program. He is the successor to the now phpMyVisites.

There is also a system of modules and plug-ins for best results and performance.

Changes:
  • New Web Analytics Report (Loyalty and Recency)
  • Better performance when tracking hundreds and thousands of websites
  • Improved PDF / HTML reports
  • Improved usability (panel change layouts, sections for indicating metrics on charts)
