Foreign GPS monitoring services



The article describes foreign GPS monitoring services. Selected services that have free demo access (in a number of existing services, demo access is provided only upon request). The analysis is carried out to determine and select the most important directions for the development of the VIALATM service.

All services are active (the latest updates or news were published no more than a month ago). All support a large line of personal and car trackers, as well as mobile applications, trackers of their own development. All services also provide the opportunity to work under their own brand (both within the framework of the manufacturer‚Äôs cloud infrastructures, and with the possibility of installation on their own servers / infrastructures).


GPSWOX.COM




Year of foundation: 2014


Main office: London, England Branches in the United States and the European Union


Representations in social networks: Youtube, Facebook, Google+, Twitter (very good presentation and educational videos in Youtube)


Mobile applications on platforms: IOS, Android, Windows Phone . Among them: applications to access the service, applications, trackers and applications that allow you to organize your own SMS gateways


Supports web client interface in a large number of languages


Useful features of the system


  • Sending commands to devices via SMS and GPRS channels
  • Creating templates for SMS and GPRS commands
  • Device grouping
  • Create and manage a list of vehicle drivers
  • Create and manage a list of events with binding them to specific types of devices. The event is configured depending on the set of message attributes that come from devices of a particular type.
  • Flexible configuration of notifications for geofences, speeding and device events
  • Create and manage routes
  • Creating and managing POIs
  • Access to the service via API
  • Branding mobile applications
  • Ability to customize web client design
  • Adding functionality to customer requests


GPS server




Year of foundation: 2010


Main office: Vilnius, Lithuania


Representation in social networks: Youtube, Facebook


Mobile applications on platforms: IOS, Android, Windows Phone, Blackberry . Among them: applications to access the service, applications, trackers and applications that allow you to organize your own SMS gateways


Supports web client interface in a large number of languages


Useful features of the system


  • Device grouping
  • Create and manage a list of vehicle drivers
  • Creation and management of the list of passengers
  • Creating and managing a trailer list
  • Flexible customization of notification templates
  • Flexible configuration of events (by time, taking into account the days of the week; routes, geofences)
  • Flexible report customization system (with the ability to send them on a schedule by email)
  • RFID and iButton support
  • Ability to work with galleries of photographs received from devices
  • Support for chat between users of the service


Geolink




Year of foundation: 2015


Main office: Vienna, Austria


Representation in social networks: Facebook, Google+, Twitter


Mobile applications on platforms: IOS, Android . Tracker applications


Supports the client web interface in English


Useful features of the system


  • Convenient display of trips (with the ability to filter the list by various objects, types of trips, ...)
  • Maintaining a list of expenses by objects (travel on toll roads, refueling, ...)
  • Ability to search for devices by IMEI, name
  • Visualization of data from the sensors in the form of graphs
  • Formation of automatic notifications created on the database coming from the device sensors
  • Access to the service via API
