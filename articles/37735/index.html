Downloading from Narod.Disk without captcha and without Yandex. Bar

As you know, to download a file from Narod.disk , you must either enter the captcha, or install Yandex. Bar. And if you don‚Äôt want Yandex.bar to callus eyes (or your favorite browser is opera)? Yes, it can be hidden, but why is it needed then, if it is used only for downloading files from Narod. Disk?
And you can do without it, while downloading files without captcha in any convenient browser that allows you to change the user agent.

Firefox Method
We type in the address bar about:config , in the Filter field we write:
general.useragent.extra.firefox
And we add to the value of the appeared parameter YB/3.5.1 (for me, for example, the result is "Firefox / 3.0.1 YB / 3.5.1"). Everything, now Disk thinks that we have installed Yandex. Bar

Opera way
In the Opera folder we find the opera6.ini file, open and look for the ISP section (if not, then create it). Find \ create in this section the parameter ID with the value YB/3.5.1 . You should have something similar:
[ISP]
ID=YB/3.5.1
We restart the opera and, if everything is done correctly, Yandex will be happy to inform you that we have the latest version of the bar installed

Method for other browsers
We find how to change the user-agent of the desired browser, and append either the YB/3.5.1 string to the end, or to the inside of the user-agent, just to avoid this:
My-browser/20.0 (YB/3.5.1) someone info
The Yandex. Bar identifier should not be surrounded by brackets (tested empirically). As a last resort, you can add some nonsense to it after the semicolon (";").
Examples of correct user agents:
Mozilla/5.0 (Windows; U; Windows NT 5.2; en-US; rv:1.9.0.1) Gecko/2008070208 Firefox/3.0.1 (ddd; YB/3.5.1; eee) eiireo werw 4242 FirePHP/0.1.1.1
Mozilla/5.0 (Windows; U; Windows NT 5.2; en-US; rv:1.9.0.1) Gecko/2008070208 Firefox/3.0.1 FirePHP/0.1.1.1 YB/3.5.1

If the described actions are true, then now you can download from the Disc without entering the code and without installing the Yandex bar. (Which, for example, in the opera is impossible for now)
