In the wake of the AzureCon conference - the main announcements

Only the AzureCon conference was held , and we are in a hurry to talk about what was announced there. Where use may not be very clear from the first time, notes were added, and links were added where there was already something to read in more detail and how to start using it.

In addition, XaocCPS has written an excellent review of other new products related to big data and analytics.



Under the cut:
* The most powerful virtual machines with GPUs in the public cloud
* Reserved Instances
* Platform for IoT solutions
* Cortana Analytics Suite

And much more. And, as usual, a lot about Open Source.

Let's start with the fact that, perhaps, long ago, users requested virtual machines with full-fledged GPUs. GPU, as is known, is used for heavy computing, incl. modeling, etc.



A new series of virtual machines in Azure under the letter N will soon become available to the general public. Inside these machines will be what is called the latest NVidia generation Tesla GPU, K80 , and the recently announced NVidia M60 . With NVIDIA GRID technology, you can deploy an application that actively uses graphics components, such as Autodesk, Esri, and others, in the cloud and work successfully with them. NVIDIA GRID 2.0, which supports the NVIDIA Quadro GPU, is twice the performance of its predecessor, and has support for Linux.

Besides the fact that onboard these VMs there will be powerful GPUs, several similar machines can be merged into a farm using super-fast RDMA .

Microsoft Azure Securi ty Center .
The Azure Security Center is a service that helps you detect and prevent security threats, and increase visibility and control over what happens to the security of your projects in Azure, including Advanced Threats, using Microsoft machine learning and experience. At the same time, it is not just a dashboard or portal, it is rather a living and constantly-evolving mechanism that allows you to react proactively and not only to threats, including various security tools (firewalls, antiviruses, etc.).

Azure Backup for GA Application
Backup was before, but now it is able to work at a more detailed level, helping you to reserve specific loads for you - for example, SQL, Hyper-V VM, SharePoint and Exchange.

Azure File Storage GA
Azure File is a service that allows you to connect Azure Storage using SMB 3.0. This makes the connection permanent and secure. Azure Files also has a RESTful API for performing management tasks.
Why: transfer file "ball" to the cloud, "sharing" data from the server, High Availability "ball", work from a local server with a cloud "ball".

Cortana Analytics Suite - already available
Cortana Analytics Suite is an umbrella term that combines several tools and services for building analytical solutions - information management, storage, analytics, visualization, and personal assistant Cortana. All of this can be integrated with each other in order to provide users with a personalized experience of communicating with big data.

Azure IoT Suite
Azure IoT Suite - an umbrella term that combines several tools and services designed to build IoT solutions - data collection from end devices (sensors, etc., Event Hubs and Stream Analytics), their storage (Azure Storage / ...), analytics , machine learning, visualization (PowerBI) and reaction to events with sending commands back to the end devices. There are more than 10 services in total, and you can already do remote monitoring, later predictive analytics (Azure Machine Learning) and management will be available.

In addition, if you are creating a solution, then you can use additional bonuses from interacting with Microsoft by certifying your solution in the Microsoft Azure Certified for IoT program - with further help in getting more from your solution.
Read more

Azure IoT Hub
A service designed to securely connect, deploy, update and manage end devices, capable of handling millions of different devices on different operating systems and protocols. It can be used both separately and as part of a project using the Azure IoT Suite.
Why: simplify the construction of IoT solutions; connection of devices and sensors; organization of sensor communication with the backend; safety regulation
Azure IoT Hub



Azure Container Service
As you know, we love Linux, Docker and many other useful projects. Azure Container Service is a service that combines Apache Mesos, Docker, Mesosphere Datacenter Operating System (DCOS) and the power of Azure to manage containers, both locally and in the cloud. Using this service, you can deploy a Mesos cluster to manage docked applications using Microsoft Azure Resource Manager. Will be available closer to the end of the year.



Azure Reserved Compute
Azure Reserved Compute is a special resource payment model in which you (from December) can pay 63% of the basic cost of a computing unit-virtual machine in Azure for 12 months at a discount of 63 months. This option is available only for those users who have entered into the Enterprise Agreement.

New DV2 Virtual Machines
DV2, a new VM series, comes out with 2.4 GHz Intel Xeon E5 v3 (Haswell) on board, and with the help of Intel Turbo Boost Technology 2.0 you can make 3.2 GHz out of it, which makes DV2 35% faster than the current VM D- series (since DV2 are based on D).
Why: for applications that require faster CPU, disk performance and more memory.

App Service Environment
App Service Environment released in GA with full SLA. ASE is a completely isolated and dedicated for you environment in which you can safely deploy and develop applications using scaling and various types of services - Web Apps, Mobile Apps, API Apps and Logic Apps.

Azure Mobile Engagement GA
Azure Mobile Engagement, formerly known as a Cappptain product, has been released to GA.
