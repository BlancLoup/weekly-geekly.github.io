Video tutorials running computational tasks in Python, R, C ++, Octave and Blender in FlyElephant

A couple of weeks ago, our team released the latest release of FlyElephant , a platform for scientists that provides ready computing infrastructure for making calculations, helps to find partners and work together on projects, as well as manage all data from one place.

The Azure cloud is currently used as a computing resource, and users can run computing tasks written using C ++ (with OpenMP support), R, Python, Octave, Scilab, Java, Julia, OpenFOAM, GROMACS, Blender on servers with a number of cores from 1 to 32 and RAM up to 448 GB.

Today we want to share video tutorials of launching tasks in FlyElephant. Under the cat you'll find videos on how to run computational tasks written using C ++, R, Python, Octave and render images using Blender, as well as a promotional code for getting free extra hours of your tasks.
