API Playground



Last week, Google officially unveiled another development in the framework of "20% of personal time" that went beyond the company. Development Engineer Ben Lisbakken created the Playground API , with which learning the code for Google Javascript API becomes easier than ever before.

The Playground API provides a simple and convenient interface for writing and testing javascript code for eight Google services using the APIs: Maps, Search, Feeds, Calendar, Visualization, Language, Blogger, Libraries and Earth. In addition, the new Playground API will soon become the default way to showcase JS code examples. Another good news is that the giant translated part of the source code under the Apache 2.0 license for use in the Playground.

At the moment, more than 170 javascript templates are available on the playground, ranging from simple lists, slideshows based on RSS media and ending with Google Earth KML files.

Playground is perfect for beginners who want to learn a little more about the Google API - in addition to clear examples of use in the Playground API, there is also support from experts.
