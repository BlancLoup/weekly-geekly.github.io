A new version of the autosave plugin for Movable Type has been released.

A new version of the AutoSave autosave plug-in by Patrick Benny for Movable Type was released yesterday.

Previous versions of MT AutoSave plug-in worked only with Internet Explorer and Mozilla Firefox, the new version adds support for the Safari browser. This means that now you can use almost any browser that can be launched under Windows, MacOS X and Linux.

The AutoSave plugin saves the text you typed in the blog every n-seconds. Thus, the unfinished message does not disappear completely after the server is disconnected or crashed and you have to add just a few words, although it depends on the configured autosave interval (initially it costs 5 minutes).
