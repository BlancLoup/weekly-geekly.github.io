Synology¬Æ Unveils DiskStation Manager 4.1 Beta



Synology has announced the release of a new beta version of the operating system DiskStation Manager (DSM) 4.1.

New OS offers users:
  • Advanced and reliable storage solutions for virtual infrastructure
  • Increase business and management efficiency by deploying services such as Cloud Station, Syslog Server and Mail Server
  • Creating an effective resource allocation system and applying high-performance solutions to share data that meets corporate standards
  • Enhanced multimedia features with new Video Station service, improved Photo Station, Audio Station and mobile applications
  • Enhance video surveillance security with updated Surveillance Station 6 software

DSM 4.1 was one of the first operating systems to support VMware vStorage API for Array Integration (VAAI) technology, which allows you to unload the Hypervisor by performing Full Copy, Zero Block, Atomic Test & Set (ATS) and Storage Reclamation functions on DiskStation network drives and RackStation. Using VAAI technology helps improve performance and optimize disk space in a vSphere 5.0 environment.



DSM 4.1 provides users with a significant increase in performance and advanced functionality. The Cloud Station service adds support for two shared folders at once, as well as the ability to receive and transfer files from mobile devices. Cloud Station now supports secured HTTPS communication and the use of proxy servers. Syslog Server has become more convenient thanks to the introduction of features such as quick browsing, monitoring by keywords and setting up a data archiving policy. The Mail Server application has become a full-featured email service with support for the SMTP protocol, additional email accounts, multiple domains, a spam filter, an answering machine, and the ability to track the number of messages and email traffic in real time.



DSM 4.1 offers efficient data sharing solutions. The Traffic Control and Speed ‚Äã‚ÄãLimit functions make it possible to prioritize, reserve and limit the use of resources for different applications, users and groups. Resources Monitor allows you to receive operational reports and statistics on the number of hits and data transfer rates. Support for SFTP and FXP protocols, as well as the ability to establish an Internet connection through a proxy server, provide more efficient and secure data transmission, in accordance with the requirements of corporate standards. Moreover, it provides for synchronization of shared folders stored on several DiskStation and RackStation devices, and combining them into a single data array.



DSM 4.1 provides enhanced multimedia features. On the page of the new Video Station service, a movie billboard and detailed information about the media content is automatically downloaded from the Internet. Using the TV Recorder module, you can watch TV programs online or record them on a schedule. The DS video mobile application allows you to play videos and change recording schedules on iOS devices. In Photo Station, user albums became available both in chronological order and linked to Google maps. Using filters by location and name, as well as automatic face recognition, will greatly facilitate the process of sorting photos. Using the updated Audio Station application, you can now display lyrics and create personal libraries for each user. Owners of iOS devices have become available to play music stored in the cache memory, even in the absence of an Internet connection. Mobile applications DS audio and DS photo + began to additionally support devices and based on Windows Phone. In turn, the updated DS download application allows users to manage download tasks.



Four sliders have been added to the advanced interface of Surveillance Station 6 for video surveillance, which correspond to different video playback channels, which makes it easier to return to certain periods of video recording. New features for tracking objects, such as: Tally Counter, Virtual Fence and No-idle Zone significantly extend the reliability of the video surveillance system when attempting to penetrate into the surveillance zone. Using the web API, developers can more easily integrate Surveillance Station with their own applications.
