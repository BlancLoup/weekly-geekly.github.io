point three. Hotkeys


One of the main advantages of Total IMHO are its hot keys. I will give an example
Some experienced users know them, but they may be unfamiliar to beginners:
Ctrl + Q - quick view in the next panel (which is noteworthy, if you have the IEView plug-in, the folders will open with the explorer, and the files with the programs with which they open by default)
Ctrl + Shift + F1 - open the object under the cursor in a new tab of the adjacent panel
Ctrl + Enter - copy the object name under the cursor to the command line
Ctrl + Home - go to the root partition of the disk
Ctrl + D bookmarks
Ctrl + Shift + F1 - view in thumbnail mode
Shift + F3 - launch of the object under the cursor (conveniently, when something is already selected, but you need to look :)
