New products in DreamSpark: Expression Studio 3, RDS 2008 R2, CCR & DSS Toolkit 2008 R2, XNA Game Studio 3.1

August is really a month of releases! The program DreamSpark lives and successfully develops, updated with new products.

A quick reference for those who hear about the program for the first time: DreamSpark is an initiative of Microsoft, in which any student, graduate student or student over 12 years of age can get Microsoft licensed software for development and design for free.

Today in DreamSpark 4 new products appeared:
  • Expression Studio 3
  • Robotics Developer Studio 2008 R2
  • CCR & DSS Toolkit 2008 R2
  • XNA Game Studio 3.1

CCR and DSS Toolkit 2008 R2


A set of .NET and Compact Framework libraries and utilities for developing loosely coupled and distributed applications.

The CCR part is aimed at supporting the development of applications for multi-core and parallel systems, facilitating the management of asynchronous operations, parallelization, control over parallel equipment and partial disruptions in operation.



DSS provides support for the development of distributed applications based on a composition of state-driven services, notification of events and weak binding through data isolation.
