Introducing Kerio Control 9



Today, Kerio Technologies has released Kerio Control 9.0, introducing features such as:

  • Shared definitions in MyKerio - allow you to create and manage groups of IP addresses, URL groups and time ranges directly in the MyKerio interface.
  • Protection against denial of service attacks - in Kerio Control 9.0, you can set various rules for incoming connections, user devices, and servers.
  • Two-step verification in MyKerio - increases security by requiring the authenticated user to provide an additional temporary password.
