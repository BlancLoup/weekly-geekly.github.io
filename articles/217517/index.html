Android Universal Image Loader



An interesting project created by our NOSTRA habrauser , this library will help you upload images asynchronously, it will also help you to cache, display and transform images. The developer tried to create an interesting and powerful project.

Chips:


  • Multi-threaded loading
  • Ability to configure to fit your desires (download, decode, caching, display options, and a lot of interesting)
  • The ability to download and cache images in the device memory and in the file system
  • To be informed about the progress of image loading.
  • Ability to customize each image display in its own way.
  • Widget support


Android Crop



This is a library which is accompanied by an Activity, and cuts an image as you please, based on AOSP source codes


Staggered gridview



A very interesting project that will help you create views like Pinterest, Etsy

Chips:


  • Setting the number of columns
  • Synchronized row position in any orientation
  • Adjustable cell indents
  • ListHeader & ListFooter support
  • Indentations that do not affect Header and Footer
  • Inherited from AbsListView



Flipview



Did everyone use FlipBoard? Like their paging effect, then this library is for you!
Only works with Android version Honeycomb and higher (API 11+)
A lightning effect has been added to the animation to make it more realistic.
