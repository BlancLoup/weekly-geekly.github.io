On the eve of Tizen Developer Summit Russia 2014

Hi Habr!
Today we have a lot of news. These are Tizen Developer Summit Russia in Moscow, and Tizen Hackathon right after it, and Tizen TV SDK.



But first things first.

So, registration continues for the first in Russia Tizen Developer Summit Russia , which will be held on July 10, 2014 at the Congress Park of the Ukraine Hotel in Moscow. The event includes a demonstration zone from Samsung and partners, as well as technical reports on Tizen, a master class on developing applications for Tizen, where you can learn how to develop applications for this platform, and many, many interesting things. Attention: the schedule has changed! Do not miss!

Registration is required! You can register by following this link . To register, use the promo code habr_invite .

Immediately after the Tizen Developer Summit Russia on July 11 and 12, 2014, the 10th anniversary Tizen Hackathon will take place in the Small October Digital October Hall in Moscow. In addition to prizes, honor and fame, it is also a great way to consolidate your knowledge gained earlier, for example, at master classes at the Tizen Developer Summit. Registration is also required! You can register by following the official website of the event . Test your skills in developing applications for the new platform!

And all this against the background of the fact that we are hosting the Russia Tizen App Challenge .
Everyone still has the opportunity to participate in the competition! Support is provided by us in our official group in VK .

There is one more news!

Most recently, the Tizen SDK beta for TV was released . Very little time has passed since the prototype Tizen TV was shown at the Tizen Developer Conference in San Francisco - and now the developer toolkit is available.

Key features Tizen TV SDK 1.0 Beta:

- Integrated Development Environment (IDE)
- Web Simulator
- Debugger
- Development of applications in HTML, JS, CSS
- Templates for Tizen TV apps
- Tools for testing TV applications on the computer
- ogg / ogv, MP4 support in web-simulator
- Debugger (Web Inspector)
- Code Inspection
- JS debugger
- Network and Resource Profiler

System requirements:
- Processor: Dual Core 1.5GHz / Single Core 3GHz or higher
- RAM: 2 GB or more
- Screen resolution: 1280 x 1024 or higher
- HDD: 5GB or more
- OS (32bit and 64bit):
  • Windows 7 (recommended), Windows XP Service Pack 2 or higher
  • Mac OS X: Intel-based hardware, OS versions 10.6 or higher.
  • Linux: tested on Ubuntu 12.x
