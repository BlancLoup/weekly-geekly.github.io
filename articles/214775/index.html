Free HTML5 Font Editor

More and more useful programs can be ported to open web technologies. Until now, there was no font editor among them. Now there is: Glyphr .



Professional font design software is a complex and expensive program. In contrast, the Glyphr interface is completely accessible to everyone, and the application itself is completely free.





Glyphr simplifies the creation of the font as much as possible, but at the same time the editor has all the necessary functions.

In the editor window, complex shapes are created, copying and pasting, rotation of elements, dragging and scaling, cubic Bezier curves with reference points specified in two- or three-dimensional space are supported.

Individual created forms can be saved as Linked Shape and used in a variety of letters (obdgpq). Changing the pattern changes the shape of all the characters.

You can try Glyphr on the example of this demo font , where all Latin lowercase letters have already been created. Start work from scratch or load a saved project - here .

The project is exported to a .txt file: do not forget to save at the end of the working session. However, the Glyphr editor can be copied as a single HTML file (224k) and work locally.
