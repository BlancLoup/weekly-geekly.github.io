Quareo hardware platform - intelligent TE connectivity



Continuing the review of connectivity solutions from TE Connectivity, we will look at the hardware Quareo platform with CPID technology today.

As already known, the intelligent system allows you to create SCS with automated control at the physical level. The entire system consists of ICM (Infrastructure Configuration Manager) software and one of two AMPTRAC or Quareo hardware platforms.

So what is Quareo?

This is a hardware platform that provides complete information about the physical level of the components of the SCS. The components of this system and the principle of operation are slightly different from AMPTRAC, which we considered earlier .

So, chips are installed in patch cords (Fig. 1), and patch panels have additional contacts for reading information from these chips (Fig. 2). Connection Point Identification (CPID) is provided by the use of microchips. These microchips contain information about the physical characteristics of the cord, including the serial number of the connector, type, class, length, etc., related to the connected patch cords. When connected, this information is read and registered in the ICM software database using the built-in panel scanner.

Fig. 2 Quareo Patch Panel

Quareo Series Network Panels



Quareo supports 24-port and 48-port panels. The patch panels of this system can be used as passive (standard), or with the help of a pre-installed controller, they can become part of Quareo (Fig. 3)

Network panels allow network administrators to display, manage, and audit connections at the physical level. This means that for the first time it became possible to apply the same control policies that were previously applied to higher layers of the OSI network model to the physical layer of the network.

And so, ICM software polls all network panels in real time and reads information from embedded microchips, after which it documents the data obtained in the database. In the future, these data can be used in other applications designed to monitor the entire network infrastructure of the enterprise or data center.



Fig.3 Installing the controller in the panel



Pic.4 Quareo patch panel controller

The CPU controller is designed to provide information transfer between the patch panel and the ICM software. It is also possible to provide power, programming, check the status of the panel.

This controller provides 10BaseT or 100BaseTX Ethernet technology, as well as power over Ethernet (PoE) connections and access to the browser's web interface. Quareo is controlled by a system connection system. The information is then sent to the ICM system software via the Ethernet controller port. Power is supplied either via PoE or auxiliary (AUX) input ports on the rear panel of the controller.

Quareo systems support multiple network chassis ports. The chassis can work as a passive chassis, or can become part of the Quareo system by inserting a controller into the chassis, controller, and connecting chassis of an Ethernet port controller. The chassis network is shown in Figure 5.

Summarizing all the above, let's summarize



Quareo with CPID technology provides a high level of awareness of the network infrastructure, including accurate documentation and reporting of all changes, thereby increasing security. It is also possible to ensure high availability of the network, at the same time reducing the level of operating costs associated with its service in all parts of the network, from data center to PC at the workplace.

Any change at the physical level is recorded in real time in the event log, allowing you to detect changes and localize possible problems in any area, from the data center to the PC in the workplace. Quareo allows you to increase the level of security and continuity of workflows, due to the built-in notification system, generating reports and instantly displaying information about outages, stops and unauthorized connections immediately when they occur.

According to statistics, more than 70% of all failures of the information network are associated with errors at the physical level. The new system, developed by TE Connectivity, with connection point identification technology, can significantly reduce the number of failures.
