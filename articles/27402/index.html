Habra Editor for those who write articles

Habra users and authors of articles I offer you my project Habra Editor , which allows you to look a little differently at writing articles for Habr. Habra Editor was created mainly for personal use. It implements the things that I lacked when writing articles.

What is


  • a spacious interface is not burdened with unnecessary blocks of text;
  • wide, almost full screen text entry fields;
  • convenient toolbar;
  • the main input field is significantly increased for better text navigation;
  • all Habrahabra tags are supported;
  • all tags work with selected text, now you can select a block of lines and make a list of it, like the one you are reading now;
  • convenient insertion of tags such as img, a, user, font, youtube;
  • the problem with habracut has been solved in the root; automatic insertion of this tag;
  • There is a preview that is still being finalized;
  • there is a panel for displaying the syntax highlighting service of poison.qsh.ru;
  • automatic copying of the received text to the clipboard;
  • transliteration;
  • character map;
  • saving drafts.

Browsers


Testing of Habr Editor was conducted on the following browsers:
  • Internet Explorer 7;
  • Opera 9.5;
  • Firefox 2.0.0.14;
  • Firefox 3.0.

What is planned


  • modify the preview;
  • make a panel with the display Habr;
  • improve browser support;
  • It is possible to implement the ability to save drafts.

I would be grateful to any comments, wishes, bug reports and criticism from all who are interested in such a project. Perhaps SuperGabr will allow you to write an article with great convenience, but for now I offer my version.
