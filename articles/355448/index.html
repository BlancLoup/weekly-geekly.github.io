Are you a goat, user?

In Android 4.2 (Jelly Bean), a new method i sUserAGoat () has appeared . The purpose of this method is rather vague, the documentation says ‚ÄúUsed to determine whether to make this call is subject to teleportations‚Äù - is the user who called this method an object for teleportation. The method should return true if the user is a goat. But if you look at the source of the method, you can find out that the method always returned false .

More precisely, it was until recently. With the advent of Android 5 Lollipop, the documentation has been amended - As of LOLLIPOP, you can now automatically identify goats recognition technology. - Starting with LOLLIPOP, this method can now automatically detect goats using advanced goat recognition technology. Curious programmers again got into the source and found the code
