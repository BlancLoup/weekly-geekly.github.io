Google introduces additional security mechanisms in the Android Market

Today, Google has announced that it is introducing additional means to protect users from malicious software. No, it's not about application moderation, but about antivirus for the application portal. This antivirus called Bouncer will scan all new applications, as well as the database of old applications for malicious code definitions of virus databases. In the event of a suspicion of danger, the application will run in the cloud infrastructure of Google to detect suspicious activity.
