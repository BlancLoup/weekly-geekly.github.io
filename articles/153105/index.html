Opera 12.10 beta - API for every taste



Today the first beta version of Opera 12.10 browser has been released. Usually we always try to make pleasant surprises for the users of our browsers, but today we have prepared many useful innovations for developers. We hope that the innovations will appeal to both the developers of Opera extensions and users.

Context menu API

Now, Opera's extension developers will be able to change browser context menus (also known to users as the right-click menu). Thanks to the new API, it became possible to add new functionality to the context menus, simplifying work with social networks, online translators and many other popular services.

API for taking screenshots

Taking screenshots is not always easy (does everyone know where the Print Screen button is located on the keyboard?), And it is even more difficult to share them with friends. Thanks to the use of the API for creating screenshots by extension developers, Opera users will get a simple and fast way to show friends what they see in their browser window.

Data loading API

This API allows Opera extensions to use their own CSS and JavaScript to improve or modify the functionality and design of web pages. For example, you can replace the Comic Sans family fonts with Helvetica, or use the advanced JavaScript features on all the web pages you view.

Standards again

This is not all news related to the API. Increasing the number of supported web standards, we added support for Fullscreen API and Page Visibility API, which will allow developers to better manage the display of page content and optimize the use of browser resources.

For developers using web sockets, we also have good news. Opera 12.10 includes support for the latest implementation of web sockets with enhanced functionality for developers and improved performance for end users. And this support is enabled by default.

SPDY

At the head of a long list of performance and speed improvements, we present the SPDY network standard as an add-on to the HTTP protocol, which promises a significant acceleration of web page load. In Opera 12.10, SPDY support is enabled by default and works with web pages that support this standard. For example, massive websites like Twitter or Gmail support the SPDY standard in addition to the traditional HTTP protocol.

Time for a change

In both worlds - Windows and Mac - this fall there have been tangible changes in operating systems. The new version of Opera 12.10 includes several innovations that make it possible to more fully utilize the new functions of these operating systems.

With the release of Mac OS X Mountain Lion, many new features have been added that are built directly into the system. Opera 12.10 for Mac supports innovations such as Notification Center, Retina Display and built-in social media publishing.

Soon, Windows 8 will become publicly available, and Opera 12.10 includes support for some of the features of this OS. In particular, we added touch control support for Windows 8 Classic mode, and as a bonus, similar support in Windows 7.
