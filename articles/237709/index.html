Return A. Kibkalo - free webinar on Storage Spaces - DSS based on Windows Server 2012R2

Alexey Kibkalo : In the recent past, a senior Microsoft architect responsible for technology and virtualization projects in the EMEA region (more than 80 countries). He designed, implemented and supported infrastructure projects to the largest organizations in Russia, Europe, Africa and the Middle East.

October 2, 2014 at 14:00 will hold a free webinar on the topic: Storage Spaces - DSS based on Windows Server 2012R2

This webinar considers the theory of building storage systems based on Windows Server as deeply as possible using Storage Spaces technology. In addition to the architecture, attention is paid to practical implementation tips, storage performance planning and methods for solving potential problems.

Using Storage Spaces technology, Scale-out File Server, and rapid deployment methods using System Center, deployment and maintenance tasks for storage systems are much easier and more accessible to administrators.
