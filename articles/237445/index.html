Appeared RTM release (for manufacturers) Ubuntu for smartphones



The first smartphones on Ubuntu should appear a little later this year, at a price of 200-400 US dollars. Developing a mobile version of Ubuntu took quite a lot of time, but anyone could follow this process (or even participate in it).

New releases of the mobile version of Ubuntu come out with an enviable regularity, and some users put such releases on their mobile devices, reporting on the normal (or problematic, options too) of a particular version of the distribution.

Now the Canonical team responsible for developing the mobile version of Ubuntu has announced the release of the RTM release (ready to manufacturer), which means that mobile Ubuntu is ready for use by mobile device manufacturers.

"Mobile Ubuntu" was significantly improved, with sharpening for devices with a touch screen and the corresponding features of the interface. Various notifications, home screen and mobile versions of applications are created from scratch.

This release fixes many shortcomings of previous versions, including support for Facebook chat notifications, interface improvements, and everything else.

You can download the release from here - Ubuntu-RTM / 14.09 . In addition, there are other versions of the release , not yet approved.
