Overview of the Russian office package open source OnlyOffice in InfoboxCloud

OnlyOffice is more than just an office suite in the browser. This is a multifunctional collaboration portal that includes document and project management. It allows you to plan work tasks and milestones, store and collaborate on corporate or personal documents, use social networking tools such as blogs and forums, and also communicate with team members through a corporate instant messaging program.



OnlyOffice is designed for small and medium-sized companies that are looking for an effective platform for managing business processes.

Package includes:
  • document management
  • online document editors supporting Word, Excel and Powerpoint formats
  • project management
  • gantt chart
  • CRM
  • mail aggregator



OnlyOffice is developed in Russia by CJSC New Communication Technologies (100% Russian company), all development and all management is located in Russia, in the city of Nizhny Novgorod. The company has a Latvian subsidiary Ascensio System SIA for sales in Europe and the USA. As part of the import substitution program, the product took the 1st place in the direction of "Office software".

Another good news is that the package has open source, and you can modify and modify the project if necessary.

OnlyOffice developers have SaaS solution, but it is hosted in foreign data centers. You can comply with the requirement to place personal data in Russia using the Moscow region of the InfoboxCloud cloud. In our solution, data will not be lost due to triple replication and backup of all user data.

The basic version of the package is free , including for corporate users. An Enterprise version will soon appear in which the following features will appear:
  • version control
  • reviewing
  • mail merge
  • desktop applications

At the same time, all existing functions of the free version will remain free.

In general, this solution will be as economical as possible, which is very important for a small business.

In this article, you can familiarize yourself with the capabilities of OnlyOffice and simply install the corporate portal into the InfoboxCloud cloud. If you have any questions, we will be happy to help .

How to update OnlyOffice?

Updating is very simple: 
 bash <(curl -s http://repository.sandbox.infoboxcloud.ru/scripts/onlyoffice/centos7/update.sh) domain.tld
where domain.tld is the name of the domain you want to use for OnlyOffice. If during the operation of the package you need to change the domain - just run the update script and specify a new domain. The data will not be damaged, since stored separately from application containers.
