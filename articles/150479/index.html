A detailed review of the tablet Ainol Novo 7 Fire


Ainol Novo 7 Fire is a tablet from a fairly well-known Chinese manufacturer of tablets Ainol. I had the opportunity to get acquainted with one of their first brainchild - Ainol Novo 7 Advanced, the configuration of which now causes only a smile - Cortex A8 1GHZ, 512MB RAM, screen 840x480 (not IPS, of course). Then there was the acquaintance with not quite successful budget tablet called Ainol Novo 7 Tornados. We can say that the previous generation in the form of models Tornados, Mars, Elf II and Aurora II was launched with great difficulty. The start date of sales was constantly postponed, and the first installments had many problems.
Let's see if Ainol was able to draw the appropriate conclusions and work on the bugs in the new Ainol Fire tablet.

The start of sales of Ainol Fire has also been postponed several times, and even now the issue with supplies is not entirely transparent. The second incident occurred with the name. First, the tablet was called Ainol Burning, then Ainol Flame, and finally settled on the name Ainol Fire.
There is an opinion that the ‚ÄúEnglish‚Äù version of the tablet will be called Ainol Novo 7 Flame. But how it will be different, except for the inscriptions in English on the box, at the moment it is not known.

But this is not very convenient, because You have to keep the tablet turned on so that you can interact with the buttons on the touchscreen. It is much more convenient to connect the gamepad from PS3 via Bluetooth.
In the feiyu_mod custom firmware, a driver is included to support PS3 gamepads.
To do this, you need to download the Sixaxis Controller and SixaxisPairTool.

Then:
1. Run the SixaxisPairTool on the computer and connect the controller to the computer using a USB cable. After installing the driver, the MAC address of the gamepad will be available in the format xx: xx: xx: xx: xx: xx
2. Run the Sixaxis Controller on the tablet, not forgetting to turn on Bluetooth. Click Start. The program will ask for root rights and at the bottom you will see the Local Bluetooth Address, also in the format xx: xx: xx: xx: xx: xx - enter it in the SixaxisPairTool on your computer,
Change Master field and click Update. Then disconnect the controller from the computer.
3. Turn on the controller with the PS button. If one light is on the controller, everything worked out! If you write Connection Failed - your device is not supported.
4. Now it remains only to click the Change Input Method in the program and select Sixaxis Controller.

Performance


It should be immediately noted that all the results of synthetic tests by themselves do not carry information. The point of testing is a comparison of indicators with other models, the results of which can be used to draw conclusions about performance.
Each benchmark will be accompanied by a brief description. In all tests of tests, a higher result means higher performance.

Amlogic 8726-MX can work in 3 modes: ‚ÄúNormal‚Äù, ‚ÄúPerformance‚Äù, ‚ÄúPower saving‚Äù.

Firmware update is very simple from the Recovery-menu. To enter the Recovery menu, you must simultaneously press the "Volume -" and "Power" buttons, wait for the robot to appear and release it.

Step-by-step instruction:
1. Download the firmware file (official firmware can sometimes consist of 2 parts - firmware without pre-installed Google applications and a patch with Google applications. Download both. In the custom Google firmware, applications are already embedded)
2. We reset the firmware (.zip file) to the microSD card and insert it into the tablet.
3. Turn on the tablet by simultaneously pressing the ‚ÄúVolume -‚Äù and ‚ÄúPower‚Äù buttons, wait for the robot to appear and release it.
If the firmware was in the root of the card - the firmware update will occur automatically. But it does not suit us if we want to install the official firmware and a separate patch with applications. Therefore, it is more convenient to copy the file with the firmware to a separate folder.


4. Select the apply update from sdcard item .

5. Select the firmware file and confirm the selection with the ‚ÄúPower‚Äù button. Attention! When updating the firmware, the internal memory is fully formatted.
6. If you install the official firmware, then repeat step 5 for the .zip file with the patch.
7. Select the reboot system now option and press the Power button.
After the reboot, the interface will be in Chinese, to change the language to Russian - select the ‚ÄúSettings‚Äù icon -> item with icon A -> first item -> scroll through the list of languages ‚Äã‚Äãand select the desired one.
