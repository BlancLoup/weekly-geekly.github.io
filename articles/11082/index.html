Update 10.4.10 for Mac OS X has been released

Today Apple has released another update for Mac OS X, available in versions for Intel and PowerPC Macs.

UPD : According to the TUAW blog, a new framework is being installed along with update 10.4.10, namely Yahoo Sync.
The framework is located in /System/Library/PrivateFrameworks and contains a built-in Yahoo! Sync application. It is not yet clear what functions this application performs, but it will probably be used to synchronize the Yahoo! and iCal.


The update includes the following changes / improvements:

Bluetooth
  • The problem was resolved when the Bluetooth headset could appear as an accessible device in the sound settings panel after it was removed from the list of Bluetooth devices.

Third-party
  • RAW support added for the following cameras: Panasonic DMC-LX1, Panasonic DMC-LX2, Leica M8, Leica D-LUX 2, Leica D-LUX 3, Fuji S5 Pro, Nikon D40x, and Canon EOS 1D Mk III.
  • Fixed a problem with DNG, when some images could be displayed darkened or distorted.
  • Improved compatibility of Mathematica 6 with 64-bit Macs.

USB
  • Improved reliability of use of the remote control after waking up from sleep.
  • Improved reliability of mounting external USB hard drives.
  • Solved the problem with the definition of TomTom GO 910 when connected via USB to Intel-Macs.

Other
  • Improved response to the display of the shutdown dialog when using the Control-Eject combination.
  • Solved the problem of dropping frames when copying video from a DV camera.
  • Includes previous security updates .
  • Fixed problem with calculating dates in some applications.
  • Fixed a problem with the rounding of decimal numbers in some applications.
  • An issue with Motion has been resolved when some textures could be damaged when the video memory is full.


To upgrade from a version lower than 10.4.9, use a combo update .
