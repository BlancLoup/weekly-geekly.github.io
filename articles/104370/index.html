ScrapeTorrent v. 3.0

The search aggregator for popular torrent sites ScrapeTorrent has been updated to version 3.0. The new version has a new design, increased the number of torrent indexed trackers, improved duplicate filtering, sorting by size and generating an RSS feed with the results.

Vertor
