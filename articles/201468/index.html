AngularJS 1.2.0: Timely Delivery

The long-awaited release of AngularJS 1.2.0 is ready. Pinch yourself is not a dream.

AngularJS 1.2.0 fixes many issues found in 1.2.0-RC3, and introduces several new features.

Since the last stable release, the most notable changes are:
  • Animation and transitions
  • Improved error messages (including minification errors)
  • Strict context screening
  • track by prevents duplication in ng-repeat
  • Using controller aliases in templates
  • Promise promise A + compliant
  • ngRoute rendered in a separate module


The team spent a lot of time combining, simplifying, improving the content and clarity of the documentation. Development is underway, so stay tuned for other improvements.
