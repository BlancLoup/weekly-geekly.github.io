Installing firefox 3.5 RC 2 or firefox 3.6 in ubuntu 9.04

At the moment, the latest available version of Fx 3.5 in the Ubuntov repositories is 3.5b4, and as we know, the third version of the candidate‚Äôs release has already been released. It's time to catch up with progress, let's get started!



Add the necessary repositories to sources.list, for this we write the following from the root:

echo 'deb ppa.launchpad.net/ubuntu-mozilla-daily/ppa/ubuntu jaunty main' >> /etc/apt/sources.list
echo 'deb-src ppa.launchpad.net/ubuntu-mozilla-daily/ppa/ubuntu jaunty main' >> /etc/apt/sources.list

Add the launchpad gpg key:

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 247510BE

Well, actually the final, install Fx 3.5 RC2:

sudo apt-get update && sudo apt-get install firefox-3.5

Or to install 3.6:

sudo apt-get update && sudo apt-get install firefox-3.6

To update the current version of Fx, just run:

sudo apt-get update && sudo apt-get upgrade
