ReactOS 0.4.6 is available for download.

Hello to all readers!

Practically at the same time as the unleashing of the third season of the Twin Peaks series we released the next release of the ReactOS operating system with number 0.4.6. The release is available for download right now, and there ‚Äôs no need to wait for October or November, as is the case with the iPhone X.



Download | Read the official news | View list of changes | TL; DR | Tests

In this release, you will find support for the distributed file system NFS , improved support for the UDF file system, renewed support for themes once again, and a significantly improved font engine.



It is also worth noting that the release contains the initial implementation of the Shim engine (Microsoft Windows Application Compatibility Infrastructure), used in Windows to ensure compatibility with applications built for older OS versions. By default, Shim is currently disabled and requires explicit activation in the registry.

Compared to the previous release, more than a million new unit tests have been added, evaluating compliance with the behavior of Windows. The total number of unit tests surpassed 14 million , of which 18419 (0.129%) failed

A total of 186 error reports, of which the oldest problem ( CORE-4107 , the inability to register Firefox as the default browser), remained uncorrected for 8 years, have been closed.
