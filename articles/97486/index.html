Nokia Remote Compiler

I would like to present a brief overview of the new experimental feature available in the recently released Nokia Qt SDK. This new feature allows Nokia to use build servers to build software.
This feature provides independence from the tulchein installed in the system; it may not be installed at all. Accordingly, in this way, you can easily build Symbian applications in Linux or MacOSi, and Maemo applications in Windows.
