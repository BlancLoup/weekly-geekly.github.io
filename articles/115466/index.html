Ekaterinburg: Microsoft Application Development Conference

Date: March 18, 2011
Venue: Yekaterinburg, Atrium Palace Hotel, ul. Kuibyshev, 44
Organizer: Microsoft in Russia
Event page: http://msdn.microsoft.com/ru-ru/gg622639
Participation conditions: free, pre-registration for the event

Microsoft in Russia invites you to take part in the conference on the development of business applications that will be held in Yekaterinburg on March 18, 2011. This event will be held in the city for the first time and will be one of the most significant events in the field of information technologies in the Ural region.

The conference is focused primarily on software technicians and architects. Participants will hear presentations by leading Microsoft, Oogway Software, and Smart Software, while Nikolai Pryanishnikov, President of Microsoft in Russia, will open the event. Using practical scenarios and demonstrations as an example, we will talk about solutions that are already available on the market and about future technologies.

The following topics will be covered at the conference:
  • Direction of development of the Microsoft platform and examples of the use of cloud computing to solve everyday IT problems.
  • Technology to quickly create desktop data entry applications.
  • The main components of the "framework" of a typical enterprise web application based on ASP.NET technology.
  • Using the new features of HTML5 Canvas and SVG technology for data visualization in web application applications.
  • Using Visio as a tool for dynamic visualization of business data and the creation of components to expand these capabilities.
  • Utilize SharePoint Workflow features to implement typical workflow schemes and extend this functionality.
  • Benefits of SQL Server Corporate Edition to increase the reliability, performance, and security of your business.
For more information about the event, please visit : http://msdn.microsoft.com/ru-ru/gg622639 . Pre-registration is required to attend.

The online broadcast of the conference is planned on the following resources: Hacker; Ixbt.com; It-eburg.com; Sources.ru; Oszone.net; Realcoding.net; MSDN Magazine; R2B.ru.
