Github limits the maximum file size to 100 megabytes.

Quietly and unnoticed on the github blog, there was a news about the introduction of a limit on the file size of 100 megabytes.
From June 24, the push will be changed and for large files it will do Reject for files larger than 100 megabytes and warning if the file is larger than 50 megabytes.
The size of the repository so far does not seem to limit:
help.github.com/articles/what-is-my-disk-quota
Still, a limit of 100 megabytes is too small.
