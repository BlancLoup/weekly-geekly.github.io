Icon

It so happens that for some site or interface or blog icons are needed. Where to get them? You can order from designers for money, or you can use icons from free sets. Everything is more or less clear with designers and money. Let's talk about free icon sets. For example, a website or blog has some special design and it is difficult to find suitable icons. Or the authors of the site want the icons to be exclusive. Or, suddenly, we need the same icons, but in different colors. (anything can happen)

We thought about all this for a long time and finally came up with such a project:



Iconza is a web application that gives access to a collection of free icons, but this collection is not easy. Icons in Icon can be repainted in any color. Plus, you can change the size of the icons. Thus, you can get a lot of "unique" icons.
