ONLYOFFICE: open like never before

Friends, let's be brief: in honor of our sixth birthday, we released the latest version of ONLYOFFICE in open source. From this day on, you'll be able to find on GitHub everything that we were asked about: a new version of editors with fast collaborative editing like in Google Docs and the latest version of a collaboration server (Community server).

Details of the release, what, where and under which license read on.




Document Editors


We post version 4.0 with the server-side code on Node.js (by the way, here we told about the transition in detail). Of the important, necessary features for all, there will be this:
  • Commenting and built-in chat . To express their witty remarks regarding the work of colleagues. Joke. A very useful tool in the joint creative process.
  • Fast co-editing in Google Docs . Because everyone wanted him. Read more about it in our previous article .
  • Review mode and change tracking . The scheme is simple: the received document for review checks it and suggests revisions, without editing the document itself. The author of the document looks at who, when and what he proposed, and smugly rejects the edits or, being ashamed, corrects random typos and delivers forgotten commas, muttering under his breath that well does not happen to anyone.
  • Versioned . The main tool paranoid attentive editor. Keep track of who made the changes and when. Go back to the previous version of the text. Suddenly she was better?
  • Text Art objects . Decoration, you know.
  • Named ranges , as well as the ability to set the formula language and regional parameters in the table editor. Also added support for sparklines for reading.
  • Create, delete and edit formatting styles . Format documents as quickly as possible. Format document fragments for speed.


In addition, we reworked autofilters and formatted tables in the table editor and sped up all editors when opening and converting.

New Community Server


Version 8.9.0 is now available in open source. Its most important innovations:
  • Mail and calendar integration , that is, the ability to exchange information about events between ONLYOFFICE Calendar and other calendars using mail as a communication channel. This means that you can send an invitation to any Internet user, knowing his email address. Seriously, whoever you want. And you can accept invitations sent from other calendars and reply to them.
  • New mail features . Address book for keeping personal contacts in a safe place, Answering machine for automatic replies to emails when you are on vacation, for example, and let everyone envy you. Well, the ability to create contact CRM directly from the chain of letters.


How to install it all?


In addition to standard installation methods (compilation of source codes, ready-made rpm and deb-packages, Docker images) ONLYOFFICE also offers a Docker script that allows you to lift containers with all the necessary components for ONLYOFFICE to work correctly.
