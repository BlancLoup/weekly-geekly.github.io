TVPRO - interactive media player with FullHD webcam



TVPRO is the first solution, including an interactive player, Full 1080p HD webcam, and an additional device. Ordinary TV it turns into "Smart", which can be used as a computer or mobile device.



TVPRO includes a quad-core processor that delivers fast performance, while Android OS provides access to over a million applications, movies, TV shows, music, and more through the Google Play Store. Along with these features, Internet viewing provides endless possibilities for entertainment, but the key feature of TVPRO is FullHD webcam.



The purpose of creating this device is simple: Add an interactive and convenient functionality of a computer and mobile device to the TV.



In addition to learning and studying in front of TVs with the best professionals, you can now communicate via Skype on the big screen in large HD resolution



Features TVPRO:





  • HD webcam has a wide 90-degree viewing angle. The most typical webcams have a viewing angle of 40 to 60 degrees, which usually covers one participant and is not full-length;
  • Dual noise reduction at the microphone provides improved sound quality, echo suppression and ambient sounds;
  • Built-in speakers;
  • Google Voice Search;
  • 1080p HD broadcasting;
  • There is a slot for microSD up to 32 GB;
  • Bluetoth allows you to connect headphones, keyboards, and other devices;
  • WiFi and LAN Ethernet connectivity;
  • Miracast, DLNA and XBMC support, which allows you to wirelessly send photos, videos and other files from compatible devices to TVPRO.




Specifications:





Two-way remote control, which has on one side a full keyboard layout and can be used as a mouse. This makes TVPRO a complete turnkey solution. This provides a complete and convenient control of the TV, as it would be when using a computer or mobile device.
