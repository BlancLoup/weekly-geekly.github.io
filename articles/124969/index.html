Is there life on the site?

Would you like to see who is reading this article with you now? And to be able to write them a message? If the answer to these questions is yes, then you should like the development of our company - the WhoisLive program. The program is essentially a plug-in for Firefox and Internet Explorer, which allows you to see who is currently with you on the same page and
chat with each other.

IE and Firefox are currently supported, but Chrome support will be soon.



Since the program works in a browser, most of it is written in Javascript. As a framework, Backbone.js was chosen, the one that MVC was written about and used on Habr√© more than once. Well, we also did not refuse to help jQuery and jQuery.UI. Unfortunately, WebSockets are not enabled by default in all browsers, but, judging by Wikipedia, they will soon be disabled in those that are enabled. So I had to use flash to communicate with the server. This is for the client side. The server part consists of a small web server, which was written specifically for the C ++ project and is rotated in the Amazon cloud.

Monetization plans are very simple - monetize. :) At the moment, everything is free and the plug-in does not contain any advertising, but ideas of monetization often slip through in my head. The first option is to give an opportunity for money to always be on top of the online list of the entire domain for the site owner. Thus, he can, for example, provide technical support. Option number two - contextual advertising, which varies depending on the text of the page on which you are looking.

I hope you enjoy using whoislive, if you have any questions or suggestions, please write, I will try to answer.

PS: Registration is available only through Facebook.
