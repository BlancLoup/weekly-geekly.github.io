VKontakte support appeared in Mobile Agent

Friends!

We have made a new Mobile Mail.Ru Agent (http://agent.mail.ru/) for Symbian. The main thing in it is the built-in support for instant messaging with users of the social network VKontakte. No need to specify server addresses and generally no additional settings - just enter your username and password to start instantly chatting with friends who are currently on the social networking site.



I note that in previous versions of Mobile Mail.Ru Agent there was already support for Jabber, which allowed working with instant messaging services GTalk, LiveJournal and Ya.Online.

What else is in the new Symbian Mobile Agent?
  • seriously redesigned interface embedded maps
  • A new geolocation service has appeared, which is much more accurate in locating people on devices without GPS support. If earlier only IP-address information was used for this, now data on Wi-Fi networks and cellular base stations closest to the user are also taken into account.
  • Now when the Agent publishes the photos on the blog, they automatically indicate the user's location at the moment when the picture was taken.
