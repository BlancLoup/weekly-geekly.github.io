Update: Apple Safari 3.0.3 Beta

Apple has released an updated beta version of the Safari 3 browser under Windows. Recommended update. Improved stability and security of the program.

Safari is an Apple Macintosh browser with a full set of features relying on this type of software. Very popular among users of the Apple platform. It has a very nice interface and a fairly fast page rendering, it starts almost instantly. As expected: tab-based interface; convenient bookmarks; blocking pop-ups; built-in search; RSS reader; auto-complete web forms; the ability to change the size of web forms; plugin support.
