Russian police buy software for large-scale surveillance of users of social networks



Today it was reported that the State Department of the Ministry of Internal Affairs for the Sverdlovsk Region announced the purchase of a software and analytical module for the already existing information system "Zeus". This software is used by the Sverdlovsk police. The competition should take place on August 8, the initial cost of the annual license is 1.85 million rubles.

Police posted a detailed technical task . It states that the new module should allow permanent monitoring of any user of social resources. The module should be able to track the addition of friends by this person, show notifications about the appearance of new posts, new user groups, photos, videos, and social circle.

The module for the Zeus system, if desired, the operator should be able to create a social diagram of the user's environment, with the establishment of links between specific people. The TOR states that the module requires the ability to analyze friends, relatives, acquaintances, general groups of a specific user.

In addition, the ability to search for images posted on social networks should be added. Search and ranking of photos should be carried out by time and keywords. The module will also display the location where the photo was taken.
