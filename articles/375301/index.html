Mozilla is working on a version of Firefox OS for wearable devices.



Firefox OS is a relatively new operating system, which is positioned by developers as a software platform for smartphones and tablets. Recently, this OS appeared on smart TV, as well as accessories for TVs.

After some time, a special version of Firefox OS will appear on wearable devices. Now Mozilla is actively working on adapting Firefox OS for smart watches, glasses and other devices.

Of course, Firefox OS is somewhat behind Google, whose Android Wear has already begun to travel the world. Nevertheless, Firefox OS will find its niche for wearable devices - a very wide range of devices produced by various manufacturers. And not everyone wants to pay manufacturers of ready-made software solutions, some companies are looking for cheaper alternatives.

The Mozilla solution is completely open, so that third-party developers will be able to adapt Firefox OS for various devices. At the same time, decisions from Google are also open, but the developers of Kindness Corporation post the source code of their decisions after the commercial version of a software version is released. The real development is conducted, so to speak, behind closed doors.

So far, the release date of Firefox OS for wearable devices is unknown.
