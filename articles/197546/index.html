Ghost: Just a blogging platform

Good day habravchane. I already mentioned Ghost in one of my digests :
As far as I know, this is the first CMS that will be developed by crowdfunding. The author of the project John O'Nolan collected already $ 400,000 on Kickstarter , which is 785% of the required amount. Ghost is an open project, a great design and a very user-friendly responsive interface.

Developers a few hours ago published a repository on GitHub ...

I recommend Habr's highly respected audience to look at the project even just because it was written about in Forbes, Wired, TechCrunch, Mashable, SmashingMagazine, Envato and many other popular Western media. As well as 2000 Starov for 10 hours.


To run Ghost, you need a gentleman's kit :
  • Node.js
  • SASS and Bourbon
  • Phantomjs 1.9 and Casperjs 1.1
  • Python
  • Grunt


To start, go to the project folder, type 
npm install --production 

 npm start

And your future blog appeared on 127.0.0.1:2368/ , and the admin panel on 127.0.0.1:2368/ghost/ :
