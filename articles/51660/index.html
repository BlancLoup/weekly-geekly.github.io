We collapsed your page in contact (And not only in it)

As it turned out, checking for input data, so actively crammed into newcomers of web programming, is not known to the developers of the beloved vkontakte.ru site. Armed with a firefox and a firebag, realizing that he does a person can somewhat diversify his page with various inaccessible goodies - such as the date of birth January 99, 9999 or multi-line statuses.



For example, change the date. All that needs to be done is using the Inspect mode on the profile data editing page to find the SELECT of the day and year of birth, and in the value attribute, change the number to whatever we choose. After saving, the data will be transferred to your page without any pain. Similarly with the status and any other elements of the type <input type = "text">. Selecting them and going into HTML editing mode, we change the INPUT to TEXTAREA in the Akuranto mode, after which you can safely write multi-line status. Similar things roll with their hometown, and many other elements.
