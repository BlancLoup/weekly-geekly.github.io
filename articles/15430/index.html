Comments on habrahabr.

As everyone knows, comments on Habrahabr have a tree structure, and loading a page with a large number of comments can take a long time (for those who have a narrow channel). It would be better and much more convenient if this structure was used to a greater extent. I explain:
Most of the comments from the post often consists of responses to the comments given to the post itself. If the post page initially displayed only comments, the data for it, and the left (right) of each comment was a button for displaying below (that is, as usual) answers to it, then it would be easier to read all this. The simplest example of such a system is in the file managers a tree-like display of folders and files.
