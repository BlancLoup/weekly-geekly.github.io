TheLastRipper - Last.fm ether grabber

TheLastRipper is a free application with which you can record streaming audio from Last.fm radio to mp3. In addition to downloading music tracks, the program allows you to download album art and record ID3v1 tags in mp3 files. The resulting mp3 can be sorted by tracks, artists or albums. There is also the ability to create playlists.

To operate TheLastRipper you need to have an account on Last.fm and its client. Grabber has versions for Linux, Windows and Mac OS X. However, to install the last two you will need to download Mono (a free implementation of .NET).
TheLastRipper developers recommend before using the program to familiarize themselves with the laws of their country. In Denmark (the birthplace of TheLastRipper), the radio recording can be used only for personal purposes.
