Linux applications on Windows? Easy!

Today, Ulteo announced the launch of a beta test of the new application, Ulteo Virtual Desktop .
Ulteo Virtual Desktop is a new system from Ulteo that allows you to run various Linux applications on Windows, including Firefox, Openoffice, Kopete and Konqueror.

Also, this system offers various services that are available in the Ulteo Application System. This includes automatic system updates, software updates, and the synchronization of documents and data over the Internet with Ulteo.

After installing Ulteo Virtual Desktop offers a special menu that allows you to run a variety of open-source applications.

What's important is that applications can access the Windows file system and have access to printers. Sound is also supported.

Application performance is the strength of this solution. This is not ordinary virtualization, but the use of a special Linux kernel to launch applications.
