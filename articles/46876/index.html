HP Jornada 690

Inspired by the articles about Toshiba Libretto 50ct, I decided to get something small, cheap, long-playing and with Linux. As a result, the choice fell on the HP Jornada 690:
  • Processor: Hitachi 7709 (SH3) 133 MHz
  • Screen: Color TFT (actually more like STN) 640 x 240 dots 64k colors
  • Batteries, power: Lithium-Ion battery, approximately 8 hours of work (the battery is old, so it takes 4 ... 5 hours)
  • Expansion slots: PC Card Type II (16 bit), CompactFlash Type I
  • Communications: IrDA, COM-port (no lace), modem

From left to right: Toshiba g900, SE k700i, HP Jornada 690

Installing Linux


On board the default WinCE 2.11, I played for one day and bought a 2 GB CF card to install Linux. Everything turned out to be extremely simple (http://jlime.com/):
  • put the card in the card reader,
  • broke into 3 sections: FAT16 (26mb), ext2 (1800 mb), swap (all that's left, about 150mb)
  • On FAT16 I loaded the bootloader, the config and zImage, the config was slightly corrected, changed root = / dev / hda2, to root = / dev / sda2, otherwise at boot time Kernel Panic
  • on ext2 section rewrote everything from the archive downloaded from jlime.com
  • insert the card into the device, go to the device in StorageCard2, run shlo.exe, agree with the license and the fact that all information will be deleted (and we did not create anything to delete something) and wait for everything to load

The choice will be given 2 graphic shell, choose iceWM and enjoy. There are already several programs onboard: AbiWord, GIMP1, several games, a file and archive manager, 2 browsers (Dillo and Konquerror), a mailer and several console applications.
