News and Releases


  • Lumen - a new PHP framework from the developer Laravel - Also available tutorials: Introduction to Lumen , Web application development on Lumen , API Development for Laravel application on Lumen .
  • Magento Critical Vulnerability - Allows an attacker to execute arbitrary PHP code on the server and gain full access to online store data.
    Interestingly, Magento does not include patches in the release , that is, installing the latest version of Magento, the user still needs to find and install all the patches. Meanwhile, the vulnerability is actively exploited .
