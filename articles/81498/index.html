iFlibusta - book archive on your iPhone

Today a new application for the iPhone / iPod Touch - iFlibusta.


The application is a client for working with the database of books on-line library flibusta.net


Main menu


Version, as you can see - 0.1, but already fully working.
In the main menu, you can select the sort by author or genre. After selecting the sort mode, the data is loaded from a 15 megabyte database - it takes about 40 seconds.


Search by author


The number of authors is amazing, without a search is not enough.
Having found the author you need, you will find yourself in the list of works available on the server Flibusta.


List of works


In the list of works, again, there is a search. As can be seen in the screenshot, only the genre is registered in the current database, so it will be difficult to search for collections of stories.
Having marked the works you need, it remains only to click on the ‚ÄúLoad‚Äù button and the books automatically get into the base of two popular readers : ShortBook and uBooks .


Settings


The settings in the program are scarce, there is only the possibility of on / off support for the above programs, and updating the catalog of books.

The result was a good program for fans to read. Bugs can be said not to be found, the only thing that upsets is the long opening time of catalogs and brakes during the search. We hope that this will be fixed in future versions of the program.
