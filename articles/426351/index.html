Microsoft Certification and Exam Changes

At the recently held Ignite conference, Microsoft introduced a new certification program. The company says that new certification based on roles / specializations will help you improve your skills, demonstrate your skills and knowledge to employers and discover new opportunities.


Announced six new specializations: Azure Administrator , Developer , Solutions Architect , DevOps Engineer, Microsoft 365 Modern Desktop Administrator and Enterprise Administrator. Exams and trainings for Azure Administrator certification are already available, and the next two exams, Azure Developer and Azure Solutions Architect, are in beta. The remaining three certifications from this list will be issued before the end of the year. ‚ÄúAnd this is just the beginning! We plan to announce more specializations in 2019. ‚Äùwrites Selina Winter on the Microsoft blog .




Also, in addition to the usual multiple-choice questions, they promise to add new ones based on active scenarios, drag-and-drop variations and situations more close to the workers.


Below are the preparatory courses for Azure Administrator certification.




Important! Many exams 70- XXX series will be considered obsolete from December 31, 2018 and are replaced by new series exams: AZ-10X, AZ-20X, AZ-30X, MS-10X.

If you plan to take Microsoft exams in the near future, then closely follow the information on the English exam pages. Perhaps your exam has already been replaced by a new one and it will be more practical to pass / get ready for new ones immediately than to take the old one.

All already certified Microsoft Azure specialists are given the opportunity to pass transient examinations worth $ 99. (Price varies by country / region and is reflected in the exam registration process.) They will be available until June 30, 2019 .


Transitional exams cover only the new material and the difference between current certification and the skills and concepts in the new certification.


If you pass one of the following exams (or pass before December 31, 2018, after this date they will be considered obsolete):


  • 70-533: Implementation of Microsoft Azure Infrastructure Solutions
  • 70-532: Developing Microsoft Azure Solutions
  • 70-535: Microsoft Azure Solution Architecture

You can pass the relevant transition exam (will be available until June 30, 2019):


  • AZ-102: Microsoft Azure Administrator. Transition exam
  • AZ-202: Microsoft Azure Developer. Transition exam (beta)
  • AZ-302: Microsoft Azure Solution Architect. Transition exam (beta)

Thus, there are two ways, two ways to get a new certificate.


1. Pass the outdated exam + transition and get a new certificate


70-533 + AZ-102 -> Microsoft Certified Specialist: Azure Administrator
70-532 + AZ-202 -> Microsoft Certified Specialist: Azure Developer
70-535 + AZ-302 -> Microsoft Certified Specialist: Azure Solution Architect


Or


2. Pass two new exams and get a new certificate


AZ-100 + AZ-101 -> Microsoft Certified Specialist: Azure Administrator
AZ-200 + AZ-201 -> Microsoft Certified Specialist: Azure Developer
AZ-300 + AZ-301 -> Microsoft Certified Specialist: Azure Solution Architect


I repeat once again that the first method will be available only until June 30, 2019.


If you are currently working on obtaining an MCSE certificate : Cloud platform and infrastructure , then you have time until December 31, 2018 . Starting January 1, 2019 , this certificate will be replaced by the new MCSE: Basic infrastructure, which includes MCSA Windows Server 2012 and MCSA Windows Server 2016 .




Examinations will be considered obsolete from December 31, 2018


  • 70-496: Administering Visual Studio Team Foundation Server
  • 70-497: Software Testing with Visual Studio
  • 70-498: Delivering Continuous Value with Visual Studio Application Lifecycle Management
  • 70-532: Developing Microsoft Azure Solutions (replaced by exams: AZ-200 and AZ-201 )
  • 70-533: Implementation of Microsoft Azure Infrastructure Solutions (replaced by exams: AZ-100 and AZ-101 )
  • 70-535: Microsoft Azure Solution Architecture (replaced by exams: AZ-300 and AZ-301 )

Examinations that will be considered obsolete from March 31, 2019


  • 70-346: Office 365 Identity and Requirements Management (replaced by exams: MS-100: Microsoft 365 and MS-101 identification and services. Microsoft 365 mobility and security, beta exams are scheduled for the end of 2018)
  • 70-347: Installing Office 365 Services (replaced by exams: MS-100: Identification and Microsoft 365 and MS-101 services. Microsoft Mobility and Security 365, beta exams are scheduled for late 2018)
  • 70-695: Deployment of Windows desktop and enterprise applications (replaced by exams: MD-100: Windows 10 and MD-101: Manage modern computers, beta exams are scheduled for the end of 2018)
  • 70-697: Setting up Windows devices (replaced by exams: MD-100: Windows 10 and MD-101: Manage modern computers, beta exams are scheduled for the end of 2018)
  • 70-698: Install and configure Windows 10 (replaced by exams: MD-100: Windows 10 and MD-101: Manage modern computers, beta exams are scheduled for the end of 2018)

Examinations will be considered obsolete from June 30, 2019


  • AZ-102: Microsoft Azure Administrator. Transition exam
  • AZ-202: Microsoft Azure Developer. Transition exam
  • AZ-302: Microsoft Azure Solution Architect. Transition exam

Examinations that will be considered obsolete from July 31, 2019


  • 74-343: Project Management with Microsoft Project 2013

Certifications that will be considered obsolete from December 31, 2018


  • MCSA: Linux on Azure Platform
  • MCSA: Cloud Platform

For all beta exams, a temporary discount of 80% applies if you register for the exam before October 26, 2018, and pass it until October 29, 2018.
