Windows XP mode



You can see that two versions of Microsoft Word are running.
Windows XP Mode (XPM) solves the issue of compatibility programs that are written for Windows XP. The user is provided with a virtual machine on Virtual PC with a fully licensed version of Windows XP SP3. This virtual machine integrates into Windows 7 in such a way that when installing applications in a virtual environment, the programs become available on the host.
