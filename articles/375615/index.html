What to do if you want to create such courses yourself?

To do this, there are programs for creating interactive lessons, such as Easiteach Next Generation , SMART Notebook and many others. Being built on the same basic principles as PowerPoint, these programs are sharpened for use with interactive equipment and allow you to add various elements to the presentation that you can interact with. Interactive presentations and lessons are just one of the ways to use an interactive projector with a computer. The only limitation is your imagination - any application with mouse or touch support can be used with an interactive projector.

Handwritten notes anywhere on the screen


If it was possible to pick up a marker and go to the image, then it is a sin not to write or draw something. Adding annotations over various materials is one of the most requested features of an interactive projector.

Add notes and save them inside the document allow, for example, applications MS Word, Excel and PowerPoint. The latter allows you to do this right during the slide show:


And if you need the ability to add notes on top of any material projected on the screen, this will allow you to make a free program Epson Easy Interactive Tools , created specifically for Epson interactive projectors. In the picture below you can see that we have written above all applications and are capturing the screen:


The fact is that, in addition to the possibility of adding handwritten notes, this program has a whiteboard mode that resembles a multi-page flipchart. You can not only draw on it, but also place there images, screenshots and graphic elements. The screen shot made in the previous illustration is sent to the whiteboard. You can add handwritten comments and return to working with the document at any time.


Thus, thanks to Easy Interactive Tools, we always have three useful functions at hand:

  • We work with documents right at the screen, using the interactive function as a mouse.
  • We can add entries over the screen.
  • At any time, you can switch to the ‚Äúwhiteboard‚Äù, placing everything you need during the discussion, or collecting the most important information there in the form of handwritten notes, fragments of documents, images, etc.

Firmware


Epson interactive projectors have a unique function: a light version of the Easy Interactive Tools program is built into the projector, which allows you to interact with the material on the screen without a computer! It is turned on in modern models with a single touch to the screen:


As in the case of the computer version, the embedded program has two modes: the mode of adding annotations over the contents of the screen and the ‚Äúwhiteboard‚Äù. These two possibilities are always at hand, even if a video player is connected to the projector, or a computer of one of the employees, on which no special software is installed.

Embedded Interactive Software Toolbar

It should be noted that there are models in the Epson line of interactive projectors that have even wider possibilities for working without a computer. Speech about Epson EB-1420Wi and Epson EB-1430Wi projectors , flagships of the line. For example, the whiteboard mode of these models allows you to embed images and multipage PDF documents from USB media and take a screen shot of the displayed source. And the contents of the slide can be saved to USB.

Included with these models is this control panel:


This panel allows you to control the projector wirelessly and / or act as a wired USB hub to connect Epson printers and MFPs to the projector, such as Epson Workforce. You can send slides to print, as well as scan documents directly to the whiteboard with one click! Another unique opportunity is sending the result of your work to an email address at any time if you have access to the Internet.

Network functionality


In addition to the above capabilities, the flagship models have the functionality necessary for conducting joint work sessions. You can combine up to 4 projectors simultaneously in geographically remote locations into a single work session. During these sessions, shared content is demonstrated, but all participants have access to interactive tools and the effect of each affects the overall image on the screen. This function works both in PC mode, where it is called ‚Äú Multi Location ‚Äù, and in stand-alone mode, where it is referred to as ‚Äú Whiteboard sharing ‚Äù.


In the first case, the computer connects via the network to several projectors using Epson EasyMP Network Projection , designed for this purpose, transmitting images to them, and the projectors, in turn, have access to mouse functions and, among other things, Epson Easy Interactive Tools capabilities. .

In the case of the Whiteboard sharing function, all projectors participating in the session work with the common space of the whiteboard. Moreover: through a normal web browser, any mobile device based on Android or iOS can be directly connected to the session!
