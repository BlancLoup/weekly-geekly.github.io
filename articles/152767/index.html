HP webOS officially went to open source. HP plans to continue developing the system. webOS ported to Galaxy Nexus

The mobile operating system based on the Linux kernel, HP webOS, was officially reborn today in Open webOS. The long cycle of code preparation ended today with the release of the source code of a stable version of the system. Free access enthusiasts and manufacturers around the world received 75 system components with a length of over 450 thousand lines of code. Despite a lot of problems with the mobile division of Hewlett-Packard, they were able to complete the release of the source code in time, which can be found in turnips on GitHub .

Compared to the Beta version released in late August, the following was added:
  • Support for the new version of the JavaScript framework Enyo (Enyo2)
  • Enabling the Mail, Calendar, and Contacts application code, and updating the browser
  • Fixed bugs and optimizations.

Many feared that the WebOS project, after opening the source codes, would throw HP into the dustbin of history, and were developing a Phoenix project to continue the development. But these concerns are dispelled by Hewlett-Packard engineers. In their ToDo list:
  • Migrating to Qt5 and WebKit2
  • Release updated audio / media source
  • Implement BlueZ stack
  • Complete ConnMan Connection Manager
  • Optimize Luna architecture

Developers are encouraged to join the discussion of future innovations and fixes on the Open webOS forum.
