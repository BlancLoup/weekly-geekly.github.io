Posterous.com: new service - Posterous Groups

The Posterous blog service, which combines the functions of the blog service itself and the mailing list service, yesterday announced the launch of a new type of resource, Posterous Groups. A few days of the group were available only to a small number of testers.

The main advantages of the group as a resource are all the same: the ability to send messages and comments there without registering for the service (by e-mail), the ability to make automatic translation of entries to many other blog services, bookmarks services, etc., convenient attachment processing first uploading media files somewhere to insert in the message, Posterous will do it himself). The group can be private (access only for members or use a password to access the group).

The main differences between a group and a blog: there may be several administrators in a group with all access and management rights; notifications of new publications can be received immediately or periodically, as a digest.

As before, you can link your own domain name to a group instead of using an address like mygroup.posterous.com, as well as using your own skin.
