Google Chrome got its own clipper



If you are using Google Chrome, then our new Evernote web clipper for this browser is what you need. Install Now >>

What can a web clipper


Our Chrome extension allows you to quickly save web pages to your Evernote account. You can forget about bookmarks or open tabs. If you see something interesting, just immediately copy it to your notes. Evernote saves text, links, images, etc. And later you can always read the saved page in Evernote on your computer or phone.

How does he work


The new web clipper works in the same way as a bookmarklet. It offers you two ways to copy:
  • Fragment selection and click : in this case, the clipper will save the selected part of the page and send it to Evernote.
  • Click without selection : this option will allow you to make a quick text note or add an open web page entirely - depending on your web clipper settings.

Compatibility with versions of Google Chrome


Currently, the web clipper, like other browser extensions, only supports Google Chrome beta for Windows. Install Google Chrome ¬ª

Other browsers


In conclusion, I would like to remind you that Evernote also provides web clippers for Internet Explorer, Firefox and Safari, which you can take on the corresponding page of our website .
