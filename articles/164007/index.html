Pre-New Year Announcement of Courses on the MVA Portal

On the eve of the long New Year and Christmas holidays, we published three new courses on the Microsoft Virtual Academy portal, two of them are dedicated to Windows Server 2012 , the third to System Center 2012 .

New features of Windows Server 2012. Part 2. Security, management, remote access, web platform


The course is a continuation of the first part and completes the technical review of Windows Server 2012. The four modules cover the following technologies:
  • Advanced access control with Dynamic Access Control (DAC).
  • Changes in Active Directory services, including VM-GenrationID for securely starting a domain controller in a virtual machine and cloning virtual controllers.
  • Some features of PowerShell 3.0 include: disconnected sessions, workflow, PowerShell Web Access, Integrated Scripting Environment (ISE) 3.0.
  • Remote Desktop Services (RDS) updates: VDI fast deployment script, User Profile Disk, RemoteFX over WAN.
  • New features DirectAccess and BranchCache.
  • Internet Information Services (IIS) 8.0 enhancements: support for NUMA, SNI, centralized SSL certificate storage, CPU throttling, Dynamic IP Restrictions, etc.

Migration to Windows Server 2012. Part 1


During migration, many nuances and pitfalls always pop up; in the first part of the course we tried to identify some of them:
  • A common approach to migration and basic tools, including the Windows Server Migration Tool (WSMT).
  • Migrate certain Windows Server services, namely: Active Directory, DNS, AD Certification Services, Microsoft Server Update Services (WSUS).

System Center 2012 Orchestrator Overview


This course talks about the Orchestrator component of System Center 2012, its capabilities, as well as the functional role in the System Center 2012 family, and the implementation of the private cloud using the System Center family.
