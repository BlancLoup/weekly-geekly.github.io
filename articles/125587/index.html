Requests that "break" the search engine Bing

Users report that, according to some requests, Bing gives something like a "blue screen of death." The result looks like this:



In the Google cache, you can find a lot of such unreadable Bing error messages. What it is?

The reason is not entirely clear. One of the versions is that this may be the result of Microsoft‚Äôs anti-hacking activity on blocking automatic queries. The fact is that the error occurred on various queries, but we managed to find the keyword that is present in almost all the buggy queries - [ phpBB ]. The same keyword appears in automatic requests from bots looking for forums with an unclosed phpBB vulnerability.

Google quite successfully filters automatic requests, so now bots use Bing.



However, an error due to the blocking of automatic requests is just a guess.

UPD. Additional verification shows that the error occurs if you make a request of the form www.bing.com/search?q=phpbb , without having Bing cookies . That is, the search engine checks cookies, and if they are missing, it gives this error.



Such is the strange protection from bots (if it is).
