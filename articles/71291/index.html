Midnight Commander 4.7.0-pre3 released

Today, after a month past from the release of the previous version, a remarkable file manager for Linux (and not only :)), the news was released about the release of mc 4.7.0-pre3.
In addition to correcting the found errors and adding new ones :) (and how without it?), The following important changes occurred in it:
  • There was a mechanism for changing the design (skins)
  • There was a mechanism for changing the predefined shortcut keys

Now you can assign almost any combination to replace existing ones, and make either look like VIM or FAR ...

List of major changes relative to version 4.7.0-pre2


General

  • GLib is used to process command line parameters instead of its own implementation.
  • implemented file highlighting by type and extension in panels
  • optimized display of the file name in the warning dialog when copying / transferring files
  • when you first start searching for files, the input fields are filled with values ‚Äã‚Äãfrom the history
  • now the option to support multiple encodings (--enable-charset) is enabled by default
  • implemented the mechanism of "skins" (frames, color schemes)
  • implemented key reassignment mechanism (panels, command line, editor, viewer)

Vfs

  • Fixed work with tar.xz files
  • Added recognition of lzma archives by extension

Editor

  • in the Save As file saving dialog you can now select the end of line type: Windows / UNIX / Mac (CR LF / LF / CR)
  • updated syntax highlighting for VerilogHDL, Shell script (shell scripts)
  • added syntax highlighting for .repo (yum), PKGBUILD, .install

Viewer

  • Fixed problem with displaying files with Windows and Mac ends of lines (CR LF / CR)

miscellanea

  • Support for environment variables ($ HOME, ~ username, etc.) has been added to the hotlist element.
  • added auto-completion support in the hotlist element
  • quick navigation through the lists using the number keys ("4" - selects the 5th element, etc.)

Error correction

  • fixed mc crash when parsing an incorrect color palette string
  • Fixed incorrect positioning of the panel encoding selection dialog when the panels are horizontal
  • eliminated 9 digit limit for displaying file exchanges in ranley
  • fixed line display in stickchars mode (option -a)
  • fixed mc crash on non-anonymous FTP login
  • Fixed mc behavior in the ‚ÄúScreen‚Äù terminal when pressing Ctrl-O
  • CP866 locale support (IBM866)
  • Improved glib version verification in the configure.ac configuration file (GLib 2.6 and higher required)
  • fixed mc crash when scrolling with the mouse wheel in the menu and history list
  • fixed text tagging with Meta + PgDn (Alt + PgDn)
  • fixed cursor behavior when using Ctrl + arrows in the case when the cursor is outside the end of the line
  • fixed autocompletion in the editor
  • Fixed Meta-d (Alt-d) behavior in file copy / move dialogs
  • fixed build issue with --with-gpm-mouse
