RubyMine 5.4 is ready for Rails 4

We are pleased to announce that the new version of RubyMine , an integrated development environment for Ruby on Rails, is available for download .

Despite the fact that only two months have passed since the release of RubyMine 5, we are ready to share new functionality and we hope that our users will be happy with the new features.

First of all, we, like all developers of the Rails community, are looking forward to the final release of the 4th version of the framework. So we did our best to have RubyMine 5.4 support Rails 4.

The following features are also added to RubyMine 5.4:

  • step-by-step debugging of the code directly on the device for RubyMotion-applications has become available;
  • Emmet supports dynamic abbreviations for speeding up HTML and CSS;
  • The CoffeeScript debugger can now be run in non-Rails applications using Node.js;
  • The IDE has a plug-in for REST Client ;
  • Nailgun support allows you to work more effectively with JRuby.

In addition, the dark scheme of the appearance of Darcula has been improved, support for remote interpreters and Ruby 2.0, etc. has been expanded. More information about RubyMine 5.4 is available on our website.

RubyMine 5 is available for free upgrade to all users whose licenses were obtained after April 10, 2012.
