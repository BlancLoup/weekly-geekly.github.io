Untethered jailbreak for iOS 4.3.1 released

Today, it can be considered that a double celebration happened - the other day 3 years after the release of PwnageTool for firmware 1.1.4 plus today, at 9 am Moscow time, dev-team guys released utilities for untethered jail latest firmware on i-devices. All devices on 4.3.1 are supported, including iPad 2, for which they are still looking for a vulnerability in the bootloader.
Also! Users of the ultrasn0w unlock should wait for a while with the Jail (explanations under the cat)

For non-bouncing: redsn0w 0.9.6rc9 :
  • OS X redsn0w
  • Windows redsn0w
Official torrent for PwnageTool
  • PwnageTool_4.3.dmg.6293151.TPB.torrent
The author of this jail was Stefan Esser (@i 0n1c on twitter). He has extensive experience in vulnerability research. Ironically, one of his first contributions to the work of the iCommunity was to improve security - last year he implemented ASLR for jailbroken iPhones.

Today's jail works on devices:
  • iPhone3GS
  • iPhone4 (GSM)
  • iPod touch 3G
  • iPod touch 4G
  • iPad1
  • AppleTV 2G (currently only with PwnageTool)
For ultrasn0w users:
This jail is not compatible with unlock, because you first need to fix conflicts when using 4.3.1 on the old frequencies supported by ultrasn0w. And one more thing: after the release of unlock, install it only with the help of IPSW custom firmware via PwnageTool, Sn0wbreeze or xpwn. Do not try to roll back to your firmware or upgrade to it, otherwise you will lose unlock.

For everyone else, redsn0w, which works on both Mac OS X and Windows, is perfect.
