Samsung launches EMMC Pro Class 1500 memory modules

Samsung again brings high technology to the masses. Recently, information about the start of production of embedded memory card eMMC (embedded Multimedia Memory Card) with a capacity of 16, 32 and 64 GB came from the camp of the international IT giant.

The company continues to actively increase the performance and ‚Äúcapacity‚Äù of its eMMC NAND Flash chips to meet the high demands of modern smartphones, tablets and other mobile devices. Currently, Samsung factories started mass production of eMMC Pro Class 1500 NAND modules.



The new series is designed for premium mobile devices. It will improve system performance and usability for a wide range of applications, including a web browser, 3D and HD video capture and playback, multitasking, augmented reality, the use of social networks and powerful games.

The latest generation of memory chips supports the Toggle DDR 2.0 interface and the highly demanded standard JEDEC eMMC 4.5, which standardizes more features to improve performance, efficiency, security and reliability, such as: SDR-200 interface (200MHz, maximum throughput - 200MB / s) , cache management, dynamic processing, file cleanup and power outage notification). The sequential read speed reaches 140 MB, and during recording the bandwidth can reach 50 MB / s. Samsung claims that the chips can perform 3,500 random I / O operations while reading and 1,500 IOPS while writing. This figure is already four times as much as previous generation eMMC.



Following the growing demand for thin solutions in hi-end mobile devices, Samsung now has the ability to create high-density memory - thanks to a 20-nanometer process. New eMMC chips can be stacked in two, four or eight dies with the largest capacity of 64 GB with a thickness of only 1.2 mm and a mass of 0.6 grams.

Considering the leading position of Samsung in the NAND Flash memory market, premium smartphones and tablets will carry these 64 GB of memory on board in their top of the range for another year or two.
