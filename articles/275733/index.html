Closing the tab with a double click and other improvements in the assembly Vivaldi 1.0.377.10

Hello!

The next weekly build seems to be an extraordinary one, but, first, we are again ready to offer a version for Mac OSX, and secondly, over the past three days we have made another 23 fixes to the browser code, which is probably good too.

So, what we have in this assembly.

We continue to add and edit browser settings. The number of available options is growing, which should please users who prefer full control over the browser. To be honest, I'm already starting to stray myself - what was added earlier, and what is just now. But it is better to say twice than to remain silent. In general, tab settings appeared inconspicuously in the browser, allowing you to close them with a double click. I remember that this function was asked for a long time. Well, now - now it is:


We also added the ability to create shortcut keys for tab duplication. Also you asked - we did.

There were also complaints about the inconvenient location of the sound indicator on the fixed tabs. We fixed it too.

News for users of the Belarusian locale. We fixed a bug due to which Belarusian bookmarks were not displayed by default in the browser. Check if everything is working fine now.

Well, and many more useful and necessary fixes.
