Building a semantic web using .NET

In addition to stories, how to implement support for OpenID , what is all this semantic web , add APML, FOAF, SIOC, XFN support to Blogengine.NET, he also took, and wrote a library with which you can add all these strange fashionable things to your project. True, the library is in an embryonic stage, far from everything has been implemented, but it is already useful as an educational tool.
