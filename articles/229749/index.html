Raspberry Developers Release Raspberry Pi Model B +

More than two years have passed since the first Raspberry Pi mini PC was shipped. Despite the fact that the developers create all new additions to this device and write new software, the model does not change, everything remains the same as it was two years ago.



But now everything has changed. Developers Raspberry Pi created a new device, called the Raspberry Pi Model B +. The official announcement has already been posted in the company's blog, and on some sites the device can be ordered. Under the cut - more about the update.



Here is a diagram of the new device:



New features (40 pins instead of 26, as in the previous model):



The characteristics of the new device include the following:

  • GPIO. As mentioned above, now 40 pins instead of 26, as in the previous model;
  • USB ports. Now there are 4 USB 2.0 ports, instead of 2 ports for the Model B;
  • Microsd. There used to be an SD card slot, now this slot has been replaced by a microSD version.
  • Low power consumption. The developers managed to reduce power consumption to a value of 0.5-1W.
  • Sound. Reduce noise, improve sound quality.
  • Improved form factor. The developers made the board more rational and convenient for work.




The processor (BCM2835), the amount of RAM (512 MB) and software remained the same.

In the RF, the Raspberry Pi Model B + is already also available for ordering.
