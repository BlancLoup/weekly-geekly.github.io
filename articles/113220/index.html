Actually today was a free minute, and this informer was born:



Want the same? Welcome under cat.

It works very simply: by crown it logs into your poison and extracts the amount on the account from the page, after which it can generate a picture, save the value to a file and issue it to the output.

Installation


1. Download the archive and unpack it somewhere.
2. Configure the script in config.php (login and password, font size, types of informers).
3. Configure the task in cron, which will execute the index.php file with the necessary frequency.

The script requires gd and curl in php.
