With a new Simpli! Simply Linux. Release 3

New Simply Linux has been released - an easy operating system for everyone. Despite the fact that this is the third release, which we collect on the Fifth Platform, innovations are quite significant.

Serious innovations


  • In addition to the installation images, there was a Live version for CD
  • kernel 2.6.32 (respectively, updated driver database)
  • the package database has been significantly updated (new versions of key programs, such as OpenOffice.org 3.2.1, Firefox 3.6.13, Gimp 2.6.11, Pidgin 2.7.7)
  • Wine emulator added to base package
  • Added support for Yota USB modems
  • added RPMInstall - a GUI program for installing RPM packages that are not included in the repository
  • New wallpaper from Vlad Gerasimov and New Year's wallpaper, specially prepared for Simply Linux by designer Anna Osmolovskaya.

Light innovations for serious matters



Together with Anya Osmolovskaya, a character was developed for Simply Linux - Simpli Fish. Despite the seeming childishness, his task is very serious - participation in various campaign events. For example, we began a series of comic strips featuring Simpli Fish calling for Linux. In the plans, also using the character as an informal logo.



First results


The Simply Linux project is one year old, you can try to take the first results. What to hide - criticism falls on us from all sides. Why another distribution, why this program, and not the one, make not a CD, but a DVD, change the installer, add this program, make simpli Linux on the gnome, on lxde, remove openoffice ... many different tips and suggestions that we try to digest and somehow systematize. Without criticism in any way, and we try to treat it adequately.

But the most important thing is that the project works in the direction in which we intended. We manage to attract new users, those who have never used Linux before, and in most cases did not even know about the presence of any alternative to proprietary software. There are many such users, they do not just try the alternative, but try and stay. We judge the audience of users of the distribution kit by letters that we receive in large quantities.
From the letters we learned that Simpli Linux is used everywhere: at home, in small firms, in schools, research institutes, schoolchildren, students, retirees. It is especially pleasant, of course, to receive letters of thanks and support.

I want to say a big thank you to all experienced and tried not one distribution kit and not for one year, users who support us. Perhaps not so much, since most of you have already established preferences, but nevertheless, we need an authoritative confirmation not of the futility of our efforts. That's how I wrapped :)

Happy New Year and Happy New Year!
