Counter-Strike for PS 3 will support mouse and keyboard

The gaming site Eurogamer reports that the version of Counter-Strike: Global Offensive, which is expected in I quarter. 2012, will support keyboard and mouse controls on the PlayStation 3.

Players in Counter-Strike for Xbox 360 will still be able to use only the game controller and will not get the opportunity to participate in a cross-platform multiplayer game with PS3 and PC / Mac players.

This is not the first such experiment. In 2007, the same feature was implemented in Unreal Tournament 3, but then the idea did not become widespread. By the way, they say that in UT3 tournaments, players with keyboards simply carried out opponents, the battle was unequal, so the current Xbox 360 discrimination is quite logical.
