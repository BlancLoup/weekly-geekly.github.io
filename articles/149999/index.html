Google introduced a new benchmark - Octane

Today, Google, in its blog of the project Chromium, introduced a new JavaScript benchmark - Octane . The essence of this new test is not only to assess the synthetic power of the engine, but also in combat conditions. For this, the engine performance is measured in the actual js-libraries, not counting the standard tests for Google V8 Benchmark, such as Richards or Delta Blue. Further, a brief description of the new tests and several measurements.


  • Box2DWeb - JavaScript port of a popular physical 2D engine used in some online toys.
  • Mandreel is a 3D engine this time, used to transfer games on Android and iOS to the Internet.
  • pdf.js is a JavaScript implementation of a PDF reader integrated into Firefox.
  • GB Emulator - Game Boy emulator .
  • CodeLoad - checks how quickly the JS engine is able to load frequently used libraries and start their execution. Developed as a derivative of Closure and jQuery projects.
