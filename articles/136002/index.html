The company Razer introduced the game tablet "Project Fiona"



CES has already brought many surprises, and among them are the Razer gaming tablet, which was named Project Fiona . This device, as can be seen from the announcement picture, is really sharpened by the games and players - for this, manufacturers even equipped the device with side-mounted manipulators. It is not clear how comfortable it will be to play in this way, but the idea is interesting.

It is also known that the tablet is running Windows, so that almost all new and old games can be run on this device. So far, the device has not hit the factory for mass production, but by the end of this year, according to the manufacturer, this will happen.
