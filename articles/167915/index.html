A new version of scrobbler for last.fm has been released, support for Raspberry Pi has been added



Last.fm today announced in a blog post that their desktop scrobbler has finally received an update.
Supported Platforms: Windows / Linux / OS X / Raspbian


I want to remind you what the old scrobler looked like:


And now about the changes:
- When activating the scrobbler, you no longer need to enter your login / password, authorization takes place using Oauth
- The radio service was tightly removed from the client
- After the first installation, the application will meet you like this with the first launch manager:





And a little more pleasant, the scrobbler is now available for the Raspberry Pi.

PS The last.fm musical social network has practically lost its meaning after giving up the radio service, but on the other hand, they decided not to close the project but develop further, perhaps this development will lead to something interesting.
