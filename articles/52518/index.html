The release of Ubuntu 9.10 received the code name "Karmic Koala"

Mark Shuttleworth presented the code name for the release of Ubuntu 9.10, which will be released in October 2009 and will be supported until April 2011. The release received the code name "Karmic Koala".

In addition, Mark clarified the main directions of development of the distribution. In the autumn release, they expect to improve the server version of the distribution kit in terms of working in cloud computing environments (cloud computing) like Amazon EC2 (renting a virtual machine provided as a service). The distribution kit will integrate the Eucalyptus system, which will allow creating a cloud-based infrastructure on its own cluster.


For the desktop version, they plan to continue working to reduce download time and improve the distribution for working on mobile Internet devices (MID) and ultra-portable computers (UMPC). For seamless download in graphical mode, without flickering when switching video modes, USplash is planned to be replaced by the Plymouth system and enable the ability to switch video modes at the kernel level (kernel mode-setting).

The final list of innovations will be presented after the completion of the Ubuntu Developer Summit, which will be held in late May.
