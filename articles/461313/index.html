Zimbra 8.8.15 LTS Released

Just the other day, the release of Zimbra Collaboration Suite 8.8.15, which was the latest version of Zimbra in the 8.8 branch, took place. The main feature of this release was that it received the status of LTS, that is, it will have a longer support period than other releases of Zimbra and the release of updates for it will stop only at the end of 2022. The release originally planned for July 1 was delayed by more than three weeks.



An important feature of the release is the improved compatibility of Zimbra with Ubuntu 18.04 LTS. However, this compatibility is still in beta testing due to a number of problems that arise when using Zimbra with this Linux distribution. The developers promise to ensure full compatibility of Zimbra 8.8.15 simultaneously with the release of the first patch for the Zimbra LTS release. That is why we recommend that those who want to run Zimbra on Ubuntu 18.04, upgrade to ZCS 8.8.15 only after the release of the first patch, which is expected in August, Those who run Zimbra on CentOS, RHEL and other Linux distributions can upgrade now. Recall that not so long ago we published information on how to safely update our Zimbra .

In general, Zimbra 8.8.15 includes all the innovations that appeared in the 8.8 branch. Among them are a hierarchical address book, the ability to assign a default calendar, as well as a password recovery function for users. However, there were some innovations. So, starting with version 8.8.15, a new engine will be used to filter user html code before output. If previously an engine based on Nekohtml was used for these purposes, which in some cases could cause a serious drop in performance, now Zimbra uses an engine based on OWASP, during which such drawdowns are eliminated.


The new version of Zimbra was codenamed James Prescott Joule in honor of the famous English scientist who became one of the founders of thermodynamics

In addition to Zimbra itself, the Zextras Backup module, which is part of the Zextras Suite add-on, has also been updated. It has several new features. In particular, now the Zimbra infrastructure administrator can independently adjust the compression level when compressing data in the Zimbra mail storage, as well as set the number of accounts and objects that can be simultaneously checked by the add-on. Learn more about Zextras Suite on the official website of the developer .

You can download Zimbra 8.8.15 now on the official Zimbra website . Those who wish can also compile a new version of Zimbra from the source code posted on GitHub .
