Kali Linux 2.0



More than two years have passed since the release of the Kali 1.0 distribution. Today came the 2.0 version of this distribution.

A brief digression: Kali linux is a distribution kit containing many utilities for conducting penetration testing - from analyzing the vulnerabilities of web applications to hacking networks and services and fixing them in the system. Previously, this distribution was known as Backtrack.

The update has affected, first of all, the appearance of the distribution and stability of work:

  • Basis: core 4.0, visual interface Gnome 3.14.
  • Distribution: Debian Jessie
  • Updates: support for rolling distribution (rolling release).
  • Relevance: monitoring and updating of utilities included in the distribution kit.
  • Workspace: new style, menu composition and user interface.



Like the previous version, the new distribution is multi-platform and supports the following ecosystems: x32, x64, ARM. There are both Live distributions and images of virtual machines, as well as ready-made installation kits.



In order to upgrade from Kali 1.0 to 2.0, you need to correct the source.list file and run dist-upgrade. 

cat << EOF > /etc/apt/sources.list deb http://http.kali.org/kali sana main non-free contrib deb http://security.kali.org/kali-security/ sana/updates main contrib non-free EOF apt-get update apt-get dist-upgrade #   reboot
