Twitter Bootstrap 3.0 Released



Today, on the day of the Twitter Bootstrap biennium , the developers introduced a new version 3.0 . Work on it has been going on for a long time - more than nine months, the framework has many innovations.

Twitter Bootstrap 3.0 website , direct download link .

List of changes


  • New simplified design. In the flat style, with such a design it will be easier to customize the standard framework theme .
  • Mobile Optimization: Bootstrap is now always responsive, for the convenience of displaying any site on small screens.
  • New customizer : more convenient and functional.
  • Improved block model: everywhere used box-sizing: border-box.
  • New grid system: added more features, four types of grid classes - phone, tablet, desktop and large desktop.
  • JavaScript plugins rewritten: many improvements, including in performance.
  • New icons Glyphicons: 40 new and all in the format of an icon font.
  • Improved navbar for more responsive site navigation.
  • Modal windows are optimized for viewing on mobile devices.
  • Added new components , removed obsolete:
  • Improved customization of elements - buttons, tables, forms, notifications, etc.
  • Removed support for Internet Explorer 7 and Firefox 3.6: Respond.js is required to work correctly in IE 8. More about browser support - in a new, improved documentation .


Some project statistics


  • 2700 commits from 319 members.
  • Changed 379 files, total 84000 edits.
  • More than 900 comments in pull request.
  • 20% reduced CSS - from 127kb to 97kb
  • Over 56,000 stars and 19,000 forks on GitHub.
  • More than 9,800 closed problems, this is about 13 per day from the day of the launch of the framework.
  • 15 million views of documentation for the last month.
  • 3 million downloads, 40% of which through a customizer.
