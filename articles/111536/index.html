A brief note on some of the third-generation Kindle features.


In this article I will try to begin the description of several interesting additional features of this wonderful reader. It will be about the options for Kindle 3g and Kindle Wi-fi of the third generation and some of the things not described in the manuals. Shared hotkeys



  • alt + shift + G - Take a screenshot
  • alt + G - Refresh screen (features of partial device screen redrawing)
  • shift + alt + m (On the start screen) - Start the Miner toy. After pressing G on the mine screen, another game is available: GoMoku.
  • alt + home - Open Kindle Store (need internet connection)
  • alt + buttons from the top row on the keyboard - Dial numbers from 1 to 9 and 0
  • Menu key - You can view the time by calling the menu. The start screen also displays information about the available free space.
  • Slider from the bottom (slide) - By holding it for 7 seconds in the extended position, you can turn off the Kindle. Hold 15 seconds, you can restart the reader. Reboot is also available from the menu (Home -> Menu -> Settings -> Menu).


Hot keys in read mode


  • alt + B - Set / delete bookmarks (Page-bookmark is indicated by a curved corner, bookmarks are available from the menu item View my notes & marks. Checked for .mobi, I suspect the same work for all text formats (except .pdf).
  • Joystick. Push up. - Displays the cursor at the bottom of the screen.
  • Joystick. Push down. - Displays the cursor UP screen.
  • Joystick. Pressing right. - The next chapter.
  • Joystick. Push left. - Previous chapter.


Text-to-speech


  • shift + Sym - Turn this thing on or off.
  • back - Disables the thing.
  • spacebar - Pause a piece.
  • By controlling the cursor, you can control the order of text playback out loud.


Music controls


  • alt + space - Enable / disable music.
  • alt + f - Next track (in alphabetical order).


More complicated things



Firmware updates


Firmware updates are available in automatic mode some time after the release (with Wi-fi on or 3G available), however, for impatient users or those who cannot access the Internet on the device, there is the following method:

  1. Go to www.amazon.com/gp/help/customer/display.html?nodeId=200324680
  2. Checking your version of Kindle on the settings page
  3. We carry out simple instructions from the site:
    1. download firmware for your version
    2. we connect the reader
    3. write the downloaded file to the root of Kindle
    4. disconnect the cable
    5. Home -> Menu -> Settings -> Menu -> Update Your Kindle
    6. waiting, enjoying the new firmware


View images


Kindle has a built-in image viewer. For what purposes it can be used, I do not know, perhaps lazy comic book lovers will like it. I advise the rest to convert to .pdf. I checked on .png and .jpg images. In order to enjoy this unearthly opportunity, you should:


  1. Connect device
  2. Create a directory named pictures in the root
  3. Inside the directory to create another one, with an arbitrary name (within reason!), Images placed directly in the pictures will not be displayed. You can create an arbitrary number of directories inside the pictures.
  4. Upload the required number of images.


Browser hotkeys


  • q - increase
  • w - reduce
  • e - reset zoom
  • c - real size
  • f - full screen
  • r - rotate


Book conversion


Amazon offers its service for converting books and sending them to Kindle via email , but this is not the most convenient way, in my opinion (but it is and should not be forgotten about it).


To convert books (as well as to store the library), you can use the wonderful Caliber program.
If you have difficulties with the development of Caliber, you can read, for example, this entry . Also, for converting doc -> mobi I used the service 2epub.com in conjunction with Caliber.

Additional dictionaries




Kindle comes with two Oxford dictionaries. However, nothing prevents you from installing your own dictionaries, for example, English-Russian! It is enough to find the dictionary in a suitable format, download it to the device (like a regular book in the Documents directory) and select it as the main one. Details in English and a list of ready-made dictionaries are in the topic on the-ebook.org (Pay attention to the NBARS)

Complicated things


These are already informal ways to make Kindle a more advanced device. Use them at your own risk.

Jailbreak (mandatory part, perform anyway!)


The first thing to start with is to install this wonderful hack. Despite the title, it is very easy to install it (and all subsequent ones), for this you need:

  1. Follow the link to mobileread.com
  2. Download the archive kindle-jailbreak-XXX.zip, at the time of writing this is kindle-jailbreak-0.4.N.zip
  3. Unzip, find update_XXX_install.bin file, c XXX? corresponding to your version of Kindle
  4. Connect Kindle to the computer, drop the file to the root of the device
  5. Unplug usb cable
  6. [HOME] -> [MENU]> Settings -> [MENU]> Update Your Kindle.

Your screensavers




With this hack, you will be able to replace the official screensavers with your own.
The installation procedure is similar to installing a jailbreak. It is enough to download the file kindle-ss-XXX.zip ( kindle-ss-0.18.N.zip at the time of writing the note) from the mobileread.com forum and repeat all the actions from the last paragraph.

After installing the hack, the linkss / screensavers directory will appear in the root of the device, into which you can upload any monochrome .png / .jpg images with a resolution of 600X800 . To find an image of this type is not difficult, for lazy I publish several links:
  • smile-back.blogspot.com/2008/07/kindle-screensavers.html and smile-back.blogspot.com/2009/05/kindle-screensavers-again.html are just funny images
  • kevinguyer.com/blog/?p=16 - some good photos of science fiction writers and scientists
  • community.livejournal.com/whatzitsgalore/5085.html - girl pictures, I liked
  • rolandog.com/archives/2010/12/29/kindle-screensavers - for imageboard lovers
  • www.gotmedieval.com/2010/12/your-kindles-medieval-manuscript-makeover-awaits.html - medieval manuscripts (looks very cool)



For those who are few, you can give simple advice to use Google search in pictures , for example.

If you are preparing Kindle as a gift, you can pour a few photos of this person, I think he will be pleased.

To get the effect, you need to restart the device. However, you can simply put an empty file named reboot in the linkss directory (or rename an already existing autoreboot file that is specifically there for this purpose) and the Kindle will reboot 10 seconds after disconnecting from the PC.

The screensavers will be applied cyclically in alphabetical order depending on the file name. To shuffle images after each reboot, just create an empty random file in the same linkss directory. After that, with each reboot, the images in the directory with screensavers will be NUMBER_IMAFAYL.
