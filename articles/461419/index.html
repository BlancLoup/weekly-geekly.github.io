Remind me - what is Medium?


Medium (Eng. Medium - ‚Äúintermediary‚Äù, original slogan - Don‚Äôt ask for your privacy. Take it back ; also in English the word medium means ‚Äúintermediate‚Äù) - a Russian decentralized Internet provider that provides I2P network access services at no cost basis.

Full name - Medium Internet Service Provider. Initially, the project was conceived as a Mesh network in the Kolomensky urban district .

It was established in April 2019 in the framework of creating an independent telecommunication environment by providing end users with access to I2P network resources using Wi-Fi wireless data technology.

Medium discusses the possibility of providing its users with LokiNet network resources


Jeff Becker , one of the leading developers of the I2Pd project and creator of the Loki Network, proposed using LokiNet as a secondary transport for the Medium network.

The Medium user community is currently discussing the possibility of LokiNet joining the Medium network. If all technical issues are resolved, LokiNet will be added as an additional transport for the Medium network.

Below are some of the visible advantages of LokiNet over I2P:

  1. Bidirectional tunnels - this allows for greater productivity and faster tunneling between participants
  2. More modern cryptographic algorithms
  3. chacha20 symmetric encryption (I2P uses AES ECB)
  4. blake2 for hashing (I2P uses SHA256)
  5. x25519 for key exchange, (I2P uses ElGamal)
  6. blake2 + x25519 + sntrup is used for internal network traffic (I2P uses ElGamal and AES)
