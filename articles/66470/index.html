FlightFiles - PHP-GTK File Manager

People criticizing the PHP-GTK module usually give two reasons:
  • PHP is not intended for desktop;
  • There are no serious programs on PHP-GTK.

The first dispute is useless, because it has already gotten into the subconscious. Yes, and no desire to breed holivar. But to refute the second one , the FlightFiles project was created, which I want to briefly describe today.

FlightFiles is a file manager written using PHP-GTK. So, it looks like a regular desktop program, for its work does not need a web server or browser, only the presence of a PHP interpreter. The project is still very young, only three months old, but I think that I managed to achieve a lot during this period of time. Already implemented most of the standard for such programs programs, in particular, navigation through the file system with the storage of the history of visiting directories, creating / deleting / copying / cutting files and folders, supports drag-and-drop files from one panel to another, also There is a built-in text editor and image viewer.
