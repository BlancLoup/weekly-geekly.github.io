ZXHN H118N firmware from Dom.ru without soldering and programmer

Hello!

Got out of the dusty closet. ZXHN H118N very needed from Dom.ru.

The problem is in his short firmware, which is tied to the provider dom.ru (ErTelecom), where you can enter only the PPPOE login and password for connecting to the Internet.

This functionality is enough for a housewife, but not for me.

Therefore, we will reflash this router!

The first difficulty is that you can not reflash it too. Dom.ru took care of this thoroughly. You can only on the firmware Dom.ru. But in the old firmware dom.ru, to which you can roll back (downgrade), it is possible to upload the original version of the firmware ZTE.

1. Enter in the address bar 192.168.0.1/html/menu.html then click any character - it will show the factory webcam.

2. Go to the page "Management"

3. ‚ÄúFirmware Upgrade‚Äù tab

4. First, flash the firmware from DOMRU - DOMRU__ZXHN + H118NV2.0.1d_E04_OV.img (Old dom.ru firmware with upgrade option).

5. After the firmware, the router receives IP - 192.168.0.1. But through the WEB face does not give. It is possible only through TELNET - User: root Pass: root (this is interesting to anyone).

6. By pressing the button resets, we translate it into the default state.

6.1. Perhaps the router will change the address to 192.168.1.1, as well as in the settings of your device connected to the router, you must manually register the details
IP 192.168.1.2
Mask 255.255.255.0
Gateway 192.168.1.1
Without this, at 192.168.1.1 I was transferred to page 404

7. Access to the WEB appears with user: admin pass: admin

8. Further, from point 4. Only flashing the original firmware taken from the archive - ZXHN + H118NV2.0.1d_E04_OV.img

9. WEB access with user: admin pass: admin

PS On the Internet, many forums are full of similar topics, and an oversupply of ways and information. This is a specific method that helped me, I hope it will help you.
