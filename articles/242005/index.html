Skype for Mac got support for OS X Yosemite

Good news for makovodov who at the same time do not forget to use good old Skype. After the release of the main update for the Mac earlier this month, Microsoft released an updated version of Skype, numbered 7.1, which provides full support for the new OS X Yosemite operating system. Also solved the problem with video calls and application departures in OS X Yosemite.


The motives are understandable - in a short time of accessibility for the general user, OS X Yosemite downloaded 13% of all Mac owners. Every day, thousands of new users are updated to OS X Yosemite.

In addition to supporting OS X Yosemite, the latest build contains a number of updates and improvements.

Among them:

Skype has become friendlier to users who wish, for example, to copy a link in a group chat. In addition, a problem that caused a crash during a video call on Yosemite, as well as a bug when sending contact information, has been fixed.



You can personally try Skype 7.1 for Mac by installing it from the official Skype website. Those who have already done this, argue that the overall performance has become much higher, and this also has a noticeable effect on the speed of the program.
