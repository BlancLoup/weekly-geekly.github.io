Virtual keyboard in new Kaspersky

The new version of the Kaspersky Internet Security 2009 software package will appear simple, but at the same time innovative (in antiviruses this has not yet been) a way to protect against keyloggers: the virtual keyboard.

A virtual keyboard is designed so that users enter their confidential data not from a standard keyboard, but from an on-screen keyboard. Thus, malware that intercepts keystrokes through a driver will remain out of work.

In addition to this "curiosity" in the new Kaspersky, it will be possible to limit the work with external media and the iSwift tool to reduce the time spent on checking files (checking checksums).

Internet Security 2009 and Anti-Virus 2009 should be available in July.
