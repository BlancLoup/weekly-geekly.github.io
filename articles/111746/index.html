In Moscow, again, a small change in dialing ...

I was encouraged to write a tiny note of three independent events:
- the wife swore that the work phone is not available
- A colleague was surprised that at home the phone suddenly stopped ringing
- in LiveJournal at a flea market I see an ad: ‚ÄúI will sell a wired telephone in defective mode.
It works only on incoming calls, calling him suddenly became impossible. Did not drop, did not drown. "


All three events occurred on January 10th. And it seemed to me that, as usual, very few people remember that dialing a number of landline phones in Moscow, since January 10, became the 10th digit!
* when calling within code 495, dial 8-495-xxx-xx-xx,
* when calling within code 499, dial 8-499-xxx-xx-xx,
* when calling inside code 498, dial 8-498---,
* when calling from code 495 to a number in code 499 or 498, dial 8-499-xxx-xx-xx or 8-498-xxx-xx-xx, respectively,
* when calling from code 499 to a number in code 495 or 498, dial 8-495-xxx-xx-xx or 8-498-xxx-xx-xx, respectively,
* when calling from code 498 to a number in code 495 or 499, dial 8-495-xxx-xx-xx or 8-499-xxx-xx-xx respectively.

In general, on the one hand, inconvenience for users, on the other hand, almost everywhere in the world, to call a landline phone, you need to dial the combination ‚Äúcountry code + city code + number‚Äù.

Yes, and of course - calls between geographic codes 495, 499 and 498 are local telephone connections and are payable as local calls, depending on the chosen tariff plan for local telephone communication.

The order of emergency call 01, 02, 03, 04, time service 100, telegram reception service 06, as well as call centers 09, 009, 007, 0669 did not change and remained the same ...
