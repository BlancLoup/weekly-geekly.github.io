Blogger + I'm Sync



A week ago, opened the registration service I'm Sync . I would like to talk about how I can use the I Sync publishing tools in my own blog, using the example of Blogger.

To add the publication components to the blog, I needed:
  • Register a blog for I'm Sync;
  • Add a publication component to a blog page using an HTML / JavaScript gadget;
  • Increase the width of the blog in the Design setting.
You can see the result yourself at http://iamsyncdemo.blogspot.com . Now I can add information about upcoming events, photos, videos, contacts, right on the blog.

In addition, I'm Sync allows you to give links to a specific event or photo, for example: http://iamsyncdemo.blogspot.com/#mmp=ViewPhotoPage&mmpid=143 .

A few words about who can use the service I'm Sync. First of all, these are owners of company sites and communities. The service can also be used in personal websites and blogs, for example, to publish events that interest you and your readers. You can add any component of the publication separately, for example, only events or only photos, or all together, as is done in the example.

By default, the publishing components use the CSS of your site. However, if you want to override the style, you can load your own CSS in the Appearance setting.

Similarly, you can use I'm Sync service in conjunction with other blogs ( Wordpress , Tumblr ), content management systems ( Drupal , uCoz ) and regular sites; wherever javascript can be embedded.
