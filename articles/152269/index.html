Google search ranking factors

The guys from webmasterworld.com decided to collect all the factors that Google teaches when ranking sites. The official representative stated (at that time) that there are more than 200 of them. While the list looks like this, it is possible that some items include several factors.

Domain

  1. Domain age
  2. Domain History
  3. Keywords in the domain name
  4. Subdomain or not
  5. Domain placement
  6. IP domain
  7. IP address / server placement
  8. Domain Sanctions History
  9. Keyword in subdomain name
  10. Domain registration with google local
  11. Validity of domain registration data
  12. Alexa Rank
  13. Compete.com Rank

Server

  1. Unavailability of the site
  2. Long response site
  3. Uptime server (not only the inaccessibility of the site and the server as a whole)

Domain owner

  1. History of domain master sanctions

Architecture

  1. HTML structure
  2. Using Header Tags
  3. URL type
  4. Using external CSS / JS files
  5. Keyword in folder name
  6. Keyword in URL parameters
  7. Keyword in page file extension
  8. Dynamic URLs (= URL type)
  9. Use SSL
  10. Using the Google Search Appliance

Content

  1. Keyword Density
  2. Keyword in the title tag
  3. Keyword in Meta Description (Google refused, I decided to leave for completeness of the list)
  4. Keyword (H, H ...)
  5. Keyword in the body of the page
  6. Novelty content
  7. Semantic information (phrase-based indexing and co-occurring phrase indicators)
  8. Taxonomic flag for the main category (transactional, informational, navigation)
  9. Taxonomic flag for novelty
  10. Taxonomic marketing niche flag
  11. Keyword closer to top Title
  12. Keyword closer to the beginning hh
  13. The position of keywords on the page ( computer help , as an example)
  14. Synonyms of keywords (related words) on a page / site
  15. alt image tag
  16. Keyword in the first words on the page
  17. URL keyword
  18. Keyword in file name
  19. Keyword Lists
  20. Keyword in HTML comments
  21. Content change history
  22. The ratio of code size to content
  23. HTML validation
  24. CSS validation
  25. Presence of advertising
  26. Adsense availability
  27. The presence of Google news feeds site
  28. Site feeds on Google Blog Search
  29. # Of subscribers
  30. The presence of images from the site in Google Images
  31. Using the Google Search Appliance
  32. Using Google's Hosted Web App
  33. Delicious data
  34. StumbleUpon data
  35. Twitter data
  36. LinkedIn data
  37. Facebook data
  38. Myspace data
  39. Address (Country / City) in the body of the page

Inbound links

  1. Site quality links
  2. Link Page Quality
  3. Site Age
  4. Page age

  6. Link Location (Basement, Navigation, Body)
