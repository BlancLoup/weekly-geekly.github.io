Music Widgets Overview

I picked up some of the most interesting, easy to use, as well as free music widgets that you can post on your blog, website, etc.

Since the embed code of widgets on Habr√© is not placed, the appearance will be represented by screenshots.

1. Mixwit.com - retro cassette


Rather stylish widget with extensive customization of appearance, as well as a powerful control panel. The most important advantage is the ability to place multiple tracks in one widget at once.

Disadvantages : the choice of songs comes from the database, which consists only of free distributed songs, that is, promotional songs. There are no Russian performers at all. Sometimes the songs may just disappear (if their authors decide to withdraw from the promo base). Registration is required to receive a code.

PS It is possible to download your track by pointing to the URL, but this is inconvenient, because you first have to upload the track somewhere to the site.

2. Muzicons.ru - bubble with smiles.


An easy-to-use widget - you can select an mp3 for downloading, an icon that is placed on the left, a bubble type (with text, or with a progress bar), and also a color. The code changes on the fly, it can be copied at any time. Several options for use - player, music message, status, and more.

Disadvantages: there is no control panel - you can not make it quieter or louder, return to the beginning of the track, and so on.

3. BooMP3.com - ultra compact


Very compact and fast widget - choose the file you need to upload, fill out a small form and immediately get the code.

Disadvantages: there is no possibility to customize the appearance. Problems with the placement of the code, and did not understand how to insert the code in Wordpress.

4. Songza.com - the global grabber


Very interesting service. This is a search among all possible services for the storage of audio and video, for example youtube. A very large base. Registration is not required.

Disadvantages: the songs can disappear, sometimes the music can go jerky, and so on. All this happens for the very same reason that Songza.com only searches among different services, and services can break, songs can be erased, and so on. There is no possibility to customize the appearance.

PS I use Songzu as a player. By the way Russian copy - mp3shki.ru :)

Bottom line: the widgets are actually much larger. Imeem.com did not consider, because this is a whole social network, and quickly share tracks using this site will not work. (without registration, activation, site development).
