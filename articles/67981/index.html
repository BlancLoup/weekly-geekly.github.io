Oomph 2.0 released


Updated Oomph - a set of tools for working with microformats and the introduction of microformats support to your pages. Oomph 2.0 was developed by the same MIX Online team, which created the Gestalt mechanism - the Python / Ruby embedding library in HTML ( about it in Habr√© ).

See what Oomph offers on this test page . It implements microformats hCard, hCalendar, hMedia Audio, hMedia Video, hMedia Image, which are aggregated in a specially added component, which is accessed via the icon in the upper left corner. The component has the ability to view address data, event information, see the location of the address on a map, view an image, and export calendar data and addresses to address books or user calendars.

What's new in the second version of Oomph:

* full implementation and support of microformat Value Class Pattern ;
* hMedia support;
* controls for the rapid integration of microformats in ASP.NET;
* improved performance;
* few bugs fixed and code refactored.

The Oomph toolkit is available with open source . You can download the installer with the tools and documentation from this link.
