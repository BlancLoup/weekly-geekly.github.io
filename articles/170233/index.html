Free Windows Phone / Windows 8 Developer School in St. Petersburg

Today starts the set in the fourth free school of e-Legion developers in St. Petersburg. From March 13 to April 10, in our office, a dozen students will learn to create mobile applications for Windows Phone and Windows 8 platforms.

To become a student of WP / W8 schools you must:

  1. Register on the site;
  2. Solve the test task ;
  3. Send the solved test task with the questionnaire using the form on the school page .

All this must be done before March 1 , do not delay with sending!
Detailed information about the program, teacher, schedule, etc. can be found at mbltdev.ru/school .
