Russian Drupal API Documentation

What the Bolsheviks have been talking about for so long has happened :)

So, I am glad to present you the site of the Russian Drupal API documentation - api.drupal.ru

The site is made on the basis of the English api.drupal.org, so all the paths with it coincide. Once from google to .org, you can go to the same page with us just by changing the domain to .ru

The site is only 3 days old, so my efforts translated only the Forms API Handbook for the fifth branch of Drupal .

But the main thing is that each of you will be able to participate in the translation . To do this, you just need to register on the site (there is no integration with Drupal.ru yet, but the work is underway), and to start the translation is very easy .
