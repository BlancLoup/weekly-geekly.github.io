Anyone who develops any applications, whether it is web or ordinary, knows that you often need a large variety of graphic elements ‚Äî icons for various buttons and other controls. The typical approach is yes, let's order all these icons to the designer, and they will be unique and inimitable. But there is another way - for example, to use a standard set of icons, for example, Silk Icon set , which contains excellent icons for almost all occasions, and they are made quite professionally and your visitors will never think again what this button means. .
