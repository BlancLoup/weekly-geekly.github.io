News and Releases


  • PHP 7.1.0 Alpha 1 - The first alpha version of the new PHP 7.1 branch is available for testing. The final release is scheduled for November. PHP 7.1: Upcoming Changes
  • Codeception 2.2 - The new version of the popular testing tool has added support for Gherkin syntax, the AngularJS module and other improvements.
  • Symfony 3.1.0
  • CleverStyle Framework 4 Release
  • Vulnerability in Wordpress Plugin Threaten Million Sites at Risk
  • PSR-SimpleCache - At the discussion stage, the proposal of the new PSR standard on a simpler caching interface. Sent by tkf .
  • PHP 5.5 support ends on July 10 - If you are still using PHP 5.5, then it's time to think about updating, because in less than a month the release of security bug fixes will be discontinued.


Php


  • RFC: Iterable - It is proposed to add an Iterable pseudotype, which would combine both regular arrays and objects that implement Traversable.
  • RFC: Closure from callable function - Offer accepted unanimously. In PHP 7.1, the function Closure::fromCallable(callable $callable) : Closure {...} will be added to convert the passed callable parameter to a full closure.


Instruments


  • asset-packagist.org - Alternative for fxp / composer-asset-plugin . Allows you to install Bower and NPM packages through Composer without plug-ins and Node.js. Sent by hiqsol .
  • consolidation-org / cgr - Replacing the standard composer global require . Sets each global dependency to a separate folder, eliminating the possibility of conflicts. Post in support.
  • paragonie / airship - a safety- focused CMS developed under the Paragon Initiative .
  • jgrossi / corcel - The tool allows you to use WordPress as an admin for an application, and for the front to use any framework to retrieve data using Eloquent ORM.
  • willdurand / Hateoas - Popular library for creating HATEOAS REST web services.
  • smalot / pdfparser - A library for retrieving information from PDF documents.
  • chrissimpkins / tweetledee - Easy access to data from Twitter.
  • maciejczyzewski / bottomline - A set of useful features for each day.
  • phanbook / phanbook - StackOverflow clone, forum and CMS in one platform. Implemented on the framework of Phalcon.
  • yohang / Finite - The state machine for PHP. It can be used, for example, to describe the states and transitions of objects in the system.
  • jderusse / composer-warmup - Plugin for Composer designed to warm up OpCache.
  • nextcloud / news is an open source RSS reader that also implements a RESTful developer interface.
  • behance / php-sniffs - Several additional rules for PHP_CodeSniffer.
  • Youshido / GraphQL - Implement GraphQL protocol in pure PHP.
