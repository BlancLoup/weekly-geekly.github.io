Notepad service Springpad is now offline



The service of notes Springpad has pleased all webmasters, and not only them, but also all who use this service, great news. More recently, the long-awaited support for offline mode. Unfortunately, there are a number of limitations, but nevertheless the news is simply excellent. This is what prevented me from moving from Evernote to Springpad.

In order to try out and use this new feature you need to use Google Chrome. For me, this is not a problem, this browser is used by default. You also need to install the Springpad app from Google Apps. After such simple manipulations, the offline access feature will be available. As the developers themselves say, this feature has become available thanks to HTML5. The size of the storage is not limited at all.

But unfortunately in a barrel of honey there is a fly in the ointment. Working with the Springpad service in offline mode is subject to a number of restrictions. Firstly, this is the lack of full-text search in all notes. Search works only by headline. For me, this is one of the critical moments, but in fact it is possible to work without it. Secondly, it is the impossibility of working with files offline. Those. You cannot send or download any file, including graphics. Finally, the Springpad shopping cart is not available in this mode. You can delete notes, but you can restore them or access them only by connecting to the Internet. These are not all limitations of offline, I just wrote out the most significant ones in my opinion. Also, the settings page, the sorting options, filter scans and quick links do not work. And yet a huge plus of offline access to notes covers all these shortcomings in the aggregate.

As the developers promise, their plans include the implementation of a similar feature in other browsers, so long you need to wait for all those who are not using Google Chrome yet. The focus of development right now is to make the Springpad a powerful note taking service as quickly as possible. And offline access using the Springpad App on Chrome is only the first step.
