Architectural opportunities


The ‚Äúcore‚Äù of the solution is a hardware-software complex with multiple network ports. It has three types of processors: for processing traffic (network processors), control and content processors. Managing processors distribute tasks between network and content processors, balance and generally perform system tasks. Large glands have several processors, so the tasks are fairly evenly parallelized. At the network processor level, the session is also distributed to the network coprocessors each time. All the "chips" such as email filtering, sandboxes and other things are on the content processors. If to use a piece of iron as usual NGFW, it is close to the declared synthetic results.
