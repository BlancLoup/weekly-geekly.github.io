A new record for data transfer speed has been set.

The International Commission of Physicists from Different Countries (USA, Brazil and Korea) set a new data transfer record - 17.7 Gbit / s. Ten pairs of servers on the one hand and eight on the other were provided with a communication line with a bandwidth of 10 Gbit / s.

Such a high data exchange rate was ensured by the Fast Data Transport Java application, which directs a large file stream through an open TCP socket in such a way that all this information looks like one huge file (the time spent on starting new downloads is saved ), reports PhysOrg.com .
