Transcend JetDrive Go 500S - a flash drive with Lightning and USB 3.1 connectors for Apple gadgets

If we take into account the rumors and photos published on the popular Chinese insider resource ww2.sinaimg, the manufacturer is thinking about solving this problem and intends to expand the memory capacity in the future generation of the iPhone 7 to 256 GB (in the older version of the iPhone 7 Plus). But as long as rumors remain rumors, and problems do not lose their relevance, it makes sense to consider an alternative option for expanding memory capabilities, which in any case will not be superfluous. Such an option for iPhone, iPad and iPod Touch owners may well be the Transcend JetDrive Go 500S with USB 3.1 and Lightning connectors. More information about the capabilities of this not-so-usual flash drive will be described below.

JetDrive Go app

The iOS application that comes with the drive allows you to take advantage of all the most important functions of the device. What do we get with the JetDrive Go app?



First, the client starts automatically when the drive is connected to an iPhone, iPad or iPod touch. If you are connecting for the first time, the system will offer you to install a compact universal application from the App Store.

The utility has built-in tools for viewing documents, photos and videos, playing audio files of many popular formats, including:

Images: JPG / JPEG / TIF / TIFF / BMP / BMPF / CIF / PNG / GIF / ICO / CUR / XBM
Video: MP4 / MOV / M4V / AVI / WMV / AVI / MKV / FLV / MPG / RMVB / TS / 3gp

On the main screen of the application there are four virtual buttons for navigating through key sections. In the left and upper right corner there are buttons for launching settings and a camera.

In the settings, you can set data protection options using a password and a Touch ID scanner, view statistics of used and free space, look at the subsection with service information.



Readable document formats: DOC / DOCX / XLS / XLSX / PPT / PPTX / PDF / TXT / KEYNOTE / NUMBERS / PAGES / htm / html / rtf.

One of the useful and important advantages of JetDrive Go is the convenience and speed of transferring files between the drive and the iOS device. The application will allow you to make backup copies of a photo shoot, pictures from the iCloud library and a list of contacts.



The camera, which is built into the application, has only two modes: for taking photos and shooting videos. Plus, it is possible to set the flash on options and activate the front camera. All the footage is automatically saved in a separate folder directly on the flash drive.
