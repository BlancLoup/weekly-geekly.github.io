NeoAxis 3D Game Engine updated to version 1.3



The NeoAxis Group Company announces the release of the NeoAxis Engine 3D update to version 1.3 . Many new features have been added to the new version of the engine, and the licensing terms of the engine version supplied with the full source code have been changed.



Major new features
  • Added Screen Space Ambient Occlusion effect. SSAO is a rendering technique used to effectively apply in real time the effect of Ambient Occlusion, widely used in computer graphics. This effect contains many parameters for fine tuning and helps to achieve the optimal balance of quality and performance.
  • Landscape enhancements based on elevation maps. The maximum size of the height map has been increased from 1024x1024 to 4096x4096 pixels (16 times). This was made possible thanks to a significant reduction in memory consumption. Added support for levels of detail, which made it possible to increase the speed of rendering. The implementation is based on the ‚ÄúContinuous Distance-Dependent Height for Detail‚Äù (CDLOD) technique.
  • Lens Flare Manager - a new object that simplifies the creation of lens flare effects for the camera.
  • Platformer Demo with 2D physics - a new map has been added to the NeoAxis SDK, which is a prototype of a platform game with support for 2D physics.
  • Pathfinding Demo - a new map has been added, demonstrating the work of the path search system based on the navigation mesh technology.
  • SceneBox is a new component of the user interface system of the engine that allows you to display the 3D scene as a separate control. The component allows you to place 3D models, effects in the resource editor or manage objects from code. An example of the use of such a component could be a garage with a choice of a car in a car simulator game, or drawing a 3D avatar of a character in a corner of the screen. To demonstrate the work, you must run the Game.exe file and in the main menu click the "Gui Test 2" button.
  • WinFormsMultiViewAppExample.exe is a demo window application based on WinForms. Demonstrates the ability to create applications with multiple 3D views.
  • In addition to the ability to import a variety of formats of 3D models, such as Collada, the ability to import models in FBX format has been added. Added the ability to import animated models with skeleton animation.
  • The tool for calculating Ambient Occlusion for individual models has been added to the resource editor.
  • The ability to export geometry to FBX or Collada (DAE) format has been added to the map editor.

Changes to the licensing terms


With the release of version 1.3, the licensing terms of the engine edition, which comes with the full source code ( Source License ), change.

The cost of the license depends on the number of developers and is divided into the following:
  • Personal license (for one developer) - $ 2895
  • Team license (2-5 developers) - $ 9800
  • Unlimited license (from six developers) - $ 29,500

Thus, to get full access to the full source code of the NeoAxis engine is now possible for only $ 2,895.

Licensing conditions:
  • Access to the full source code of the engine and its tools.
  • 4 years of free updates to new engine versions.
  • 1 year technical support.
  • No royalty.

A complete description of the conditions can be found on the licensing page .

NeoAxis as a platform for non-gaming projects



At the end of the news, we would like to emphasize the possibility of effectively using the NeoAxis engine for creating non-game projects for Windows and Mac. Due to the rich functionality, a powerful set of tools and modern technologies, the NeoAxis engine is a complete solution for the development of a wide range of applications in demand in industry, research, training and staff training, etc.

So, we have previously written about the successful experience of using the NeoAxis engine when creating a 3D crash simulator for the chemical industry, made by System Engineering. Developers from the University of Udine, Human-Computer Interaction Lab, Italy used the NeoAxis engine to create the EMSAVE project (Emergency Medical Services for the disAbled Virtual Environment) , an interactive simulator tutorial to provide medical care to people with disabilities. At the moment, the project is being tested in one of the training centers of the Italian hospital.
