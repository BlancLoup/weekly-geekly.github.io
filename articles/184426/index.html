Playstation 4 will work on a modified version of FreeBSD 9.0



The latest version of the PS4 emulator, game developers today, have confirmed that the new game console from Sony really works on Orbis OS and is downloaded from GRUB (the emulator came as part of the developer kit).

On the console, you can run the graphic or text mode; for the text, the console is immediately available, and when downloading it you can see many familiar files that reveal one of the secrets of PS4: Orbis OS is a modified version of FreeBSD 9.0. According to sources within the company, the reason for this is a more liberal BSD license and a switch from Cell to x86 (although, according to one version, CellOS in PS3 was also based on FreeBSD).
