Own server for Mozilla Weave in 4 steps.

Hello. For those who do not know what Mozilla Weave is, this is an add-on to Firefox that allows you to synchronize your bookmarks / passwords / entered addresses / cookies / open tabs between an unlimited number of computers using a certain remote server. Mozilla Weave came out as a development of Google Browser Sync , which is no longer available for download. But recently, my service has stopped working. According to preliminary data, the central server Weave was transferred to new equipment and some accounts were lost: (I used Weave from version 0.2.3 and was generally satisfied. But there was one huge drawback - the synchronization process took an inadmissibly long time. However, quite by chance I discovered on the network A wonderful Swedish service is mydisk.se . The service is something between a free hosting and a file sharing service and it also supports WebDAV - the protocol on which Weave works. So, what do you need to use? What is mydisk.se as your own Weave server?

If you already have Weave installed.
  1. Register on the service.
  2. Create a folder for Weave data - for example, sync
  3. In the Weave settings on the Advanced tab, specify the server https://mydisk.se/ username / folder name - for example https://mydisk.se/habrauser/sync/
  4. In the Weave logon window, enter the username and password on mydisk.se
The order of setting if you do not have Weave installed.
  1. Download Weave by the link in the topic.
  2. Restart FF.
  3. When you first start the Weave window appears, click Cancel.
  4. Go to the settings Weave, on the Advanced tab, enter the server as written above. Click Apply / Ok.
  5. Restart FF.
  6. At startup, the Weave window will reappear. Click Next-Set up another computer-Enter your data.
Is done. You now have a personal fast sync server.
Use on health :)
