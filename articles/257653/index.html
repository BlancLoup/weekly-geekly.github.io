Yii 2.0.4

New version of PHP framework Yii 2.0.4 has been released. Installation and upgrade instructions are available at http://www.yiiframework.com/download/ .

Version 2.0.4 is a patch release and contains about a hundred small improvements and fixes. It fixes the security issue CVE-2015-3397, which allowed XSS to be carried out for users of IE6 / IE7. If your application is designed to work with these browsers, update as quickly as possible.



Of the improvements can be noted:

  • New validator EachValidator , allowing you to check each element of the array.
  • A new method is yii\base\Model::attributeHints() , which allows you to customize prompts for input fields (in addition to their names).
  • indexBy() can be used with column() when yii\db\Query database via yii\db\Query . In this case, an array of scalar values ‚Äã‚Äãis returned, indexed by the specified field from the base or by a calculated index.


In addition to the above changes, there are others. A complete list can be found in the framework and individual extensions of the CHANGELOG.md .

You can follow the development of the framework by placing an asterisk on GitHub . Also, follow Twitter and join the Facebook group .

Thanks to everyone who is involved in the development of Yii .
