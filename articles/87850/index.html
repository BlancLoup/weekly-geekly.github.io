Sending notes to Evernote by e-mail has become more convenient




One of the ways to add information to your account is to send a note to the personal e-mail address of the Evernote user. Today we want to talk about improving this functionality, which allows you to specify a notebook and labels for the created note.

How it works

First, remember or find your Evernote email inbox. It looks like this: <username> .12345 @ m.evernote.com . Now try sending something to Evernote. In the subject line write the name of the note. And now the most important thing: the subject of the letter has the opportunity to specify the notebook to which you want to send a note, and the list of tags:
  • in order to specify a notebook, use the @ symbol, followed by the name of an existing notebook;
  • in order to set a label, use the # symbol, followed by the name of an existing label; You can specify multiple labels; in this case, each label must be preceded by a # character.

Features
  • This feature works only for existing notebooks and tags;

  • in the subject line you should always first specify the name of the note, and then add any notepads and tags;
  • This feature does not work for notebooks whose names contain the characters "@" or "#", and labels with the symbol "#" in the title.
