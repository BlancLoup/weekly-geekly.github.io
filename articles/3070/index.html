Miranda IM 0.6.6

The third version of one of the most popular messaging clients for ICQ, MSN, Yahoo, IRC and others has been released in the last week.

Download Miranda-IM 0.6.6:
1) For Windows 2000 / XP / Vista: miranda-im-v0.6.6-unicode.zip (1.1Mb)
2) For Windows 98 / Me: miranda-im-v0.6.6-ansi.zip (1.1Mb)

List of changes and fixes:
- 0.6.6 ( changelog )
1) Fixed error when transferring to the active window (drag'n'drop) a file with a name containing characters from a locale not set by default
2) Writing to the log of a large amount of information could cause the program to hang
3) Extracting events from the database sometimes failed
4) Sounds in windows with messages (message windows) sometimes were played incorrectly

0.6.5 ( changelog )
1) Fixed bug in update checker.

0.6.4 ( changelog )
1) Added a module for checking and reporting updates
2) Improvements in the dbTool utility, localization support for it
3) Fixed errors when loading avatars MSN

Miranda is strong in its simplicity and at the same time almost unlimited functionality. Immediately after installation, ICQ, MSN, Yahoo, AIM, IRC, Jabber are supported. Other protocols and additional features are provided by plugins. For some functions, there are several equally good plug-ins, I will only advise a few from myself that can simplify familiarity with the program for a start:
1) SmileyAdd , emoticon support
2) History ++ (2in1) , advanced viewing and management of message history
3) Russian Language Pack , Russian interface for the program itself and some plugins
4) Updater , check and automatic installation for your plugins
5) Scriver , modification of the standard message box with the support of tabs.
6) PopUp Plus , to display information in pop-up windows
7) Message Notify , setting up notification of incoming messages, including pop-up windows
8) MRA protocol , plugin for work on the Mail.Ru Agent protocol.

The interface of Miranda is quite simple, but it can be very differently designed using themes and additional plug-ins, which can be found in large numbers on addons.miranda-im.org
