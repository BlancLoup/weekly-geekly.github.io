Marmalade becomes free

Good Friday, Habr!

Tizen Association Partner Company - Marmalade Technologies Ltd. recently introduced a completely free license for its cross-platform SDK , which is currently the world leader and enables application and game developers from all over the world to implement creative ideas and commercial ideas without choosing between quality and performance on all mobile platforms.


Existing Marmalade customers will also benefit from a comprehensive innovation program: Tizen, BlackBerry, Windows Phone and Windows Store platforms are now available in each license; In addition, Windows and Mac desktops, Roku and some smart TV platforms are available for users of Indie, Plus and Pro licenses.

The free Marmalade license will allow developers to create applications and games for Tizen, iOS, Android, Windows Phone, Windows Store and BlackBerry using a single code base, and provide native performance for all devices. Just as with the well-known Marmalade C ++ SDK, all Marmalade developers have access to Marmalade Quick, Marmalade Web and Marmalade Juice - they give the opportunity to recompile Objective C IOS projects natively under Android.

In addition, the free Marmalade license is distinguished by a wide range of integrated extensions, including billing and IAP API for all major appstorov, as well as leading services - advertising, social, gaming, graphics, physics, analytics and database services.

Application performance is a critical issue for developers when considering cross-platform strategy, as shown by a recent study conducted by experts in the Research2Guidance application market. In their latest report, Cross Platform Tool Benchmarking 2013, Research2Guidance called Marmalade the best in performance among all cross-platform solutions, indicating that 88% of Marmalade users rated the performance of applications created using the Marmalade SDK as equal or better than native development.
