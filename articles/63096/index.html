Battlefield Heroes is now available to everyone.


The closed beta test of the casual free-to-play version of Battlefield has ended and now everyone can play it. The game is free, but those who want to buy new pants for their hero will have to fork out. Part of the game is performed in the browser (settings, etc.), and the gameplay itself is in full-screen mode. For the game you have to download about 300MB. Client is here , and registration is here.

PS Played only through IE or FF
PPS Habraludi, share the names in the game so that you can add each other as a friend :)

Attention!
→ If you have a BF Heroes website that requires you to activate an email, you need to activate it via this page . Sending activation does not work on BF Heroes website!
→ If you log into the game, but do not let you on the server - do not lose hope, try again (“Try again”). New players quite often have this problem.
→ If you are trying to join a friend, and gives out “error 3” - it means that there are no places in the friend’s team
