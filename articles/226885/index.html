ASUS PadFone E smartphone review

This niche is filled with the PadFone E kit, which includes a 4.7-inch smartphone and a full-size docking station. However, as is the case with other PadFone models, the smartphone can be purchased separately, which only plays into the hands of users. And today we bring to your attention the review of the smartphone ASUS PadFone E.

Software


The operating system uses Android 4.3 standard graphics shell ASUS. From the proprietary applications, the previously mentioned Splendid, SuperNote (a note taking program that recognizes handwriting input) and App Lock (allows you to set password protection to launch any application) are pre-installed.

The smartphone has a Qualcomm Snapdragon 400 processor (MSM8226) and an Adreno 305 graphics accelerator. Exactly the same set, including the operating frequency, as in PadFone mini 4.3. The amount of memory - 1 GB.
