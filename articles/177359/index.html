JQuery 2.0 release

A new version of jQuery 2.0, the most popular JavaScript library in the world, was announced today. In addition to abandoning support for IE below version 9, there is another important point: the library now consists of a number of modules , 12 of which are independent and among which, for example, Sizzle is a full-scale selector scripting engine

TowTruck - new service from Mozilla Labs

Thanks to the efforts of the guys from Mozilla, a new service developed in HTML5 / JavaScript and WebRTC and WebSocket technologies has appeared. The service allows for the joint viewing and development of sites in real time. The principle of operation is a bit like Google Docs, but in addition a text chat has been added and video and audio chat has been announced.
