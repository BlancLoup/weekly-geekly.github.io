Free Developer Package: Visual Studio Dev Essentials



Hello!

Microsoft's free development tools, contrary to popular belief, include not only the rich and powerful IDE Visual Studio 2015 Community , but also a number of other useful components and products. For example, Visual Studio Team Services has its free offer for teams of no more than 5 people. These and many other offers are now available to every developer using the Visual Studio Dev Essentials program.

The new program Visual Studio Dev Essentials , announced in November 2015, provides centralized access to the main free components for the developer at Visual Studio, and also offers various tools to ensure a full software development cycle, including tools and products from third-party companies, such as Parallels, Xamarin etc.

For several months, the Dev Essentials program also provides access to training materials such as Pluralsight, Xamarin University, etc. As well as the possibility of obtaining priority support on the forums.

Component overview


To access the program, you must have a Microsoft account (former Live ID). You can register it here .

After logging in to the portal Visual Studio Dev Essentials , at the very beginning of the page you can fix the proposals that you are most interested. By default, here are the tools offered by the Visual Studio family, free offers of the Microsoft Azure cloud platform, etc.

Let's take a quick look at each of the program components.

In addition to the Visual Studio 2015 Community and Visual Studio Team Services , here on the program portal, you can download Visual Studio Code , a cross-platform code editor available for Windows, OS X and Linux operating systems. Supports more than 40 languages, debugging Mono and Node.js, as well as many other additional features. Including the expansion of functionality using components from the Visual Studio Marketplace .

The offerable component of the Microsoft R Server version for developers will allow you to create powerful analytics solutions in the R language on Windows, Hadoop, Terradata and Linux.

The trial period of the Microsoft Azure cloud platform is one month, and the offer for Dev Essentials participants guarantees a monthly provision of $ 25 per year for using any services presented on the portal. The Azure App Service ‚Äî a service available in an Azure subscription includes work with web applications, mobile backends, etc. and provides some of the functionality for free.

Pluralsight's exclusive offer for participants in the Dev Essentials program includes 6-month access to training courses from a global platform that provides online training on various technologies. Another suggestion for those wishing to learn more knowledge and skills is Xamarin University Mobile Testing - training video courses and instructions for developing iOS and Android C # applications from Xamarin experts.

Instruments




Application Insights is a cross-platform component of the Microsoft Azure cloud platform, which allows the developer to get up-to-date information about the viability and performance of the services and applications being developed for free.

The Hockey App , useful to mobile developers, as the main service from Microsoft, to obtain timely information about the behavior of mobile and desktop applications at the stage of their beta testing, distribution of application test assemblies, collecting crash reports and feedback.

For those who want to get acquainted with the development of a universal platform for Windows 10 (UWP), the Dev Essentials program contains a ready environment for developing UWP applications in the format of an evaluation virtual machine for Hyper-V, VMWare, VirtualBox, Parallels.

Mac users can develop a 3-month subscription for Parallels Desktop for Mac Pro and a similar offer for Parallels Access to develop and test applications for any platform on their computer directly from the Dev Essentials personal portal page.

Power BI is useful to developers for convenient and clear presentation of data in any convenient place and on any type of devices.

The following Dev Essentials program suggestion reminds that you can create, edit and work with the team on project documents directly in the browser, using the capabilities of Office Online , without installing software on your computer, while being completely free. And save them, for example, placing in the OneDrive cloud storage.

Training




Visual Studio Dev Essentials contains many different offers for developers who want to go through to learn more about technologies and go through various training programs.

For example, a 3-month subscription to courses from WintellectNOW experts, including Microsoft developers, is available to program participants. Our favorite educational platform Microsoft Virtual Academy , containing a huge number of training courses from experts from Microsoft and partners, is also presented on the portal.

To keep abreast of the latest Microsoft technology news, you can also access MSDN Magazine and the ability to subscribe to MSDN Flash .

Support




As a participant in the Visual Studio Dev Essentials program, the developer will be given priority support in selected Microsoft forums. This advantage means that a Microsoft engineer will answer the question within 2 business days if, after the post was published, the community did not have time to find the answer to this question. More on the program is available $ 25 to use the service of virtual mentoring HackHands .

Conclusion


Visual Studio Dev Essentials - the main components of the developer Visual Studio - a completely new initiative and program of Microsoft, which is currently running in a preliminary version. The program still continues to evolve in order to provide developers with a maximum of useful and free offers of various services, tools and training materials.
