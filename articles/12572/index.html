Fortune 1000 Companies Web Server

Recently, readers of the Web 2.0 blog learned that Web 2.0 is built on Linux . The method of counting and sampling the Web 2.0 servers caused quite a heated discussion in the comments to the topic, but there is no doubt that such a trend exists.
Today I would like to bring to your attention a study conducted by Port 80 Software .
The purpose of the study: to find out which web servers use Fortune 1000 companies for their Internet sites. A standard HTTP response header from the company's server is being studied. Based on it, the web server that served the request is determined.
The study is conducted annually; therefore, in addition to current indicators, trends in this issue can be assessed.
