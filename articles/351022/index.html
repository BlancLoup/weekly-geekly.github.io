The program for network monitoring Network MACMonitor

More than a year has passed since my previous publication ‚ÄúMonitoring mac-addresses on the ports of Cisco network devices using the snmp protocol‚Äù . After the publication of the article, many began to wonder: do I plan to continue developing the program? To which I gave a positive response. And now a new version of Network MACMonitor is ready.

Network MACMonitor is a program designed to scan network devices (switches and routers) for the presence of connected end devices (computers, printers, ip phones, etc.). The program stores information about which port of the network device the end device is connected to, based on its mac address. The program also maps mac addresses to ip addresses, and ip addresses to host names. A mapping of a network device port with a patch cord, patch panel, network socket is supported.

Use of the program


The program will save time and solve problems in the following cases:

  1. No network connection on the end device, since the program provides information on where the end device was located when it was last available.
  2. Control that a particular device is constantly online. If it is unavailable, you can configure to receive an email alert.
  3. Loss of the end device. In the program you can see where and when it was last active. Moreover, you can configure to receive an email alert if it reappears on the network.

You can also:

  • search mac, arp, neighbor, vlan, interface, network device tables;
  • view the history of ip addresses, hostnames of a particular end device;
  • view the history of connections of end devices to the ports of network devices.

Features of the program


  1. The presence of the results table, which contains information about where and when there was one or another terminal device.


  2. The presence of a table of network devices. Contains devices such as switches and routers.


  3. The presence of a table of end devices. Contains devices such as computers, printers, ip phones, etc.


  4. Availability of a table of locations of network devices where you can specify the city, address, person in charge, phone number of the person in charge.


  5. The presence of groups of devices.


  6. Availability of the network interface switching table. Contains information about switching (network device, network interface, patch cord, patch panel, socket).


  7. The presence of mac, arp, neighbor, vlan tables, tables, interfaces, network devices.
  8. Export and import data in csv format.
  9. Authentication and user groups.
  10. Email alerts.

Future plans


The following versions of the program are planned:

  • implement the functionality that will be claimed by users of the program, therefore wishes are welcome;
  • add support for windows environment (definition of a user who works on a specific computer and combining this information with the switch network interface) - this item is in active development;
  • LDAP authentication.

All interested please participate in the testing program. Program site.
