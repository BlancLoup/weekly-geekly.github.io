New Cisco Security Courses

Recently, with the advent of new mobile devices and ways to connect to the network, we are seeing a change in user experience. More and more employees are demanding the ability to work with corporate resources, from anywhere and on any device. We receive mail on a mobile phone, work with a corporate portal while at the airport, connect to the desktop at home. Networks have lost their boundaries, gone beyond office buildings. Cisco has specifically developed a network architecture (concept) called Borderless Networks. Within the framework of this concept, it is supposed to provide connectivity to the corporate network at any time, anywhere, from any device, by any user. Naturally, ensuring the security of such connections is crucial.

Cisco has long been offering various network security solutions: specialized equipment, built-in features of routers and switches, software, and more. Latest products fully follow the concept of networks without borders. How to ensure the ability to safely connect to the corporate network from any device, from any place and at any time is considered in the new track of CCNP Security courses.

The track includes four courses that can be divided into the following areas:
  • Secure Access (SISAS)
  • Perimeter Protection (SENSS)
  • Mobile Security (SIMOS)
  • Threat Protection (SITCS)

Secure access. The course discusses the Cisco ISE product, which allows user authentication and authorization, device profiling, etc. The main topics are dedicated to the 802.1X standard and the EAP protocol. The goal of the course is to learn to provide connectivity from any device, at any time, from anywhere. But only in accordance with the company's security policy. For example, from a corporate laptop you can access any network resources, and from a mobile phone only Internet access.

Perimeter Protection. The methods for protecting the network from both external and internal threats are considered. Configure Firewall, protect network devices, NAT, filter bot-no traffic and much more. All these protective measures can be implemented not only on specialized equipment, such as ASA, but also on IOS-based routers and switches.

Protection of mobile solutions. The goal of the course is to learn how to securely transfer data from public networks using a VPN. The course covers both intersite connections and remote access. Studied technologies: IPSec VPN, FlexVPN, DMVPN, SSL VPN, AnyConnect VPN. These solutions are configured on both the ASA and IOS devices.

Threat Protection. The course will teach you how to reduce the risks to your IT infrastructure and applications using the capabilities of next-generation firewalls, intrusion prevention systems, email and web security systems. Products that are considered: ASA, Cisco Web Security Appliance, Cisco Cloud Web Security, Cisco Email Security Appliance, IPS.
