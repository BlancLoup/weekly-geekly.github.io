On4: Shop with built-in payment acceptance

The network has a new project Voice Commerce called On4 . This site offers any unprepared user to build and run a fully-functional online store from scratch. The video on the site describes 11 steps to create a store:
  • site template selection
  • category appearance selection
  • product type selection
  • selection of the type of detailed product description
  • basket type selection
  • selection of payment page type
  • site header selection
  • selection of themes
  • logo loading
  • selection of colors and font design
  • creating static pages
After this initial setup of stores, descriptions and images of products and categories are loaded.
Such a site will already be equipped with a built-in payment acceptance system and will be able to accept payments using Visa, MasterCard, American Express and VoicePay, a system also owned by Voice Commerce.
VoicePay technology works as follows: to make a purchase in an online store connected to VoicePay, the user clicks the corresponding icon, after which a call is received on his mobile phone, and the user can confirm or reject the transaction. This technology allows you to use On4 also for mobile sales. To register in the system, you must have a legal entity. After passing the verification of company information, a login and password is provided for accessing the system and creating a store.
