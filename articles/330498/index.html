Yii 1.1.19

The PHP framework team Yii has released version 1.1.19. You can get it either through Composer or as an archive from the page .


This version is a release of the Yii 1.1 branch, which has reached EOL and receives only security fixes and PHP 7 support.


Releases such as this allow you to upgrade PHP on servers with Yii 1.1 and, thus, upgrade to the supported PHP version of the command .


Yii 1.1.19 is compatible with PHP 7.1, security patches which will be released until December 1, 2019.


We recommend using Yii 2.0 for both new projects and new developments for old projects on Yii 1.1. How to do this is described in the official manual . Full transition to Yii 2.0, in most cases, means a complete rewriting of the code. Version sharing makes this transition a gradual and less tangible budget.


This version fixes incompatibilities with PHP 7 and improves security.


A complete list of changes is available at CHANGELOG .
For upgrade instructions, see UPGRADE .


Thanks to everyone who participates in the development of the framework .
