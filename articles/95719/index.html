How much to hang in grams?

Just a week after the preliminary announcement of the release of the SaaS version for WEBO Site SpeedUp, we are pleased to announce that it is already in beta.
What is SaaS in our understanding: it is an opportunity to pay only for the used features of the product and only for the time when they are used. All this is already in the new beta version of the product to automatically accelerate sites.

A little more about the concept



In WEBO Site SpeedUp version 1.0 configurations appeared, switching between them you could easily choose the required level of acceleration. In version 1.2, we went further and introduced flexible pricing for these configurations. Now you can turn on the maximum performance for the site at the moment of traffic flow, and when carrying out, for example, technical work, turn off optimization completely.

To implement such a flexible optimization of websites, WEBO Site SpeedUp introduces a balance, which is consumed over time according to the chosen configuration. Each configuration has a daily expense, which is formed due to the included paid options (all those that were available only in the full version of the product - now they are in the SaaS version).

For comparison: the full version of WEBO Site SpeedUp is now 1999 per year. If all its functionality is not needed, then you can enable minimum settings, for example, costing 12 points, or about 1.2 rubles per day. This will be only 36 rubles per month, or 432 rubles per year.

To replenish the balance, it will be enough to purchase the appropriate module (we plan to make it possible via SMS) and activate it for the product.

How to try the SaaS version


1. You need to download the latest package of WEBO Site SpeedUp (1.2.0)
code.google.com/p/web-optimizator/downloads/list
or update the current version to the latest beta (System Status - Update - Show Beta Information)
2. Then get a test key on the Personal data page in the product. For version 1.2.0, this activates SaaS.
3. Next, configure the product or use the current configuration.

Bugs and stuff


There are a number of flaws in the logic of calculating the cost of applying optimization (especially in the user interface: not all browsers are correctly tested). You can help find and fix errors by publishing their intelligible description in the running tracker:
code.google.com/p/web-optimizator/issues/list
At the end of testing, all beta users will be issued certificates for recharge (depending on the number of open tickets).

Transition from normal to SaaS version


If you have already purchased the full or light version of WEBO Site SpeedUp, but want to use the functionality of the SaaS version, then your current subscription can be renewed. To do this, send a request with your license key to sales@webo.name .

And some product news.


Recently there have been several major innovations:
1. Added delayed download for social widgets, accelerating the site according to Google Webmasters.
2. Added full support for the functionality of all known CDNs.
3. There is a native integration with CS-Cart, which still caches excellent pages on the server side.

What is expected:
1. WEBO CDN will be launched this month - all in the same SaaS version of the product, the cost of use - from 30 points (about 3 rubles) per day.
2. Add advanced server caching for WordPress and Joomla! (heaving user behavior and modularity of the site), we also plan this month.
