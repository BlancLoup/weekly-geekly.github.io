The announcement of the Cloud OS Summit conference is all about the Microsoft cloud. Participation is free, registration is open.

Dear developers, system administrators, heads of IT departments and groups of software developers! We are pleased to invite you to the Cloud OS * Summit conference, which will be held in Moscow on November 27, 2013 .



Cloud OS Summit is the largest cloud conference of Microsoft in Russia. The conference program will present the latest technologies designed to build modern data centers, integrate private and public clouds, involve all mobile devices in the company's work, efficiently manage data and their analysis in the enterprise and create modern business applications using the power of a public cloud. .

By participating in the event, you will learn all about the newest versions of Microsoft products - Windows Server 2012 R 2 , System Center 2012 R 2 , SQL Server 2014 , Visual Studio 2013, and Windows Azure services. Together with experts from Microsoft and its partners, you will discuss scenarios for applying the latest products to solve the problems of your business.

Participation in the conference is free! It is necessary to register . The number of places is limited, in a hurry to take your own.

The conference will feature a plenary report, which will open the event by covering all major topics and more than twenty reports divided into sections:

  • Modern data center - scenarios for building private clouds and automating, monitoring and managing applications, extensions to the public cloud, hybrid scripts;
  • Personal devices in the enterprise - the involvement of mobile devices in the work of the enterprise, the management of personal devices and the issue of security, access to the desktop from anywhere at any time;
  • Cloud platform for developing applications - building scripts for modern applications using the latest Windows Azure cloud services and Visual Studio tools, integrating cloud resources and applications with resources in the enterprise;
  • Cloud infrastructure as a service - application scenarios - scripts for building applications based on Windows Azure public cloud infrastructure services, developing and testing in the cloud, working with data, analytics and Big Data in the cloud, the latest SQL Server 2014 application scenarios.

* Cloud OS - cloud operating system

We will broadcast our cloud conference online, connect to us on November 27 at 10-00 MSK.

See you at the Cloud OS Summit conference !
