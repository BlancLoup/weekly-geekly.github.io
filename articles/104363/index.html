We invite authors to create an electronic journal on .NET

We invite independent experts and experts, partner students and other interested people to join the project to create an electronic journal dedicated to the development for the .NET platform.



Requirements for articles


Articles should contain the author's material, which has not been published anywhere else in the amount of not less than 5000 characters. Articles may contain illustrative material, code snippets, attached files.

Headings

  • Mobile - Windows Phone 7, Windows Mobile, etc.
  • Alternative - MonoTouch, Microsoft for PHP, Java, MongoDB, etc.
  • Deep Dive - CLR / DLR, in general, section for Pro
  • Design - Microsoft Expression, Web, prototyping
  • Web development - ASP.NET, MVC, Dynamic Data, Silverlight, JavaScript / jQuery, AJAX, etc.
  • Cloud - Azure, cloud technologies
  • Mainstream - General Development Issues for .NET
  • Administrator - Windows Server 2008, PowerShell
  • Security - ForeFront, BitLocker, etc.
  • How to / DIY / Q & A

  • Microsoft for Students

  • Book club
  • Calendar of events
  • Promotions, contests
