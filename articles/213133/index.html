Online conference for .NET developers. March 1

We invite you to take part in the first conference of the DEV Labs series this year, which will take place on Saturday, March 1 and will be devoted to the development on the .NET platform. As in the previous events of the series, we will try in an accessible form and in a short time to convey to the audience maximum of useful information. The emphasis in the formation of the program - practical skills acquired in solving non-trivial tasks.

Approved conference reports:

  • Garbage collection in .NET. Features of work, hidden problems and best practices
  • How to make a public API. Errors and difficulties in developing the application API
  • Creating a content management system in 45 minutes on ASP.NET MVC
  • Entity Framework Code First & Migrations. EF Code-First Data Access Technology
  • C # Tips and Tricks. Subtleties of behavior "advanced" features of the language C #


More information about the reports and speakers can be found on the conference website .

The format of the event is 1 day, 5 reports from 45 minutes to 1.5 hours for each performance. After each report, sessions are scheduled for questions. To become a member of the conference, you must register . Participation is free.

UPD: The conference starts at 11:00 Moscow time with an additional testing session for participants. The first report will begin at 12:00.
