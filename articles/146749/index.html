New version of HostCMS, 6 - what is it?

Many are familiar with HostCMS‚Äôs great content management system.

Benefits:



  • Ease of use - the interface of the administrative part is made, albeit a bit complicated, but everything is logical in principle and most of the requests are executed by AJAX. Accordingly, the speed of work with the interface is high.
  • Functionality - the basic version of Freebies contains great functionality. The module information systems allows you to make almost any functionality on the site. In essence, the modules that are added in the older (paid) editions of the system are simply convenient and ready for use implementation. But you can do everything on the editorial Freebies.
  • Multisite is a feature that in this version is probably not found in any CMS available for free.


Minuses:



  • Hosting requirements - the main Zend Optimizer, unfortunately with the PHP version => 5.3 problems arise, so you need to install PHP <5.3


Controversial:



  • Perhaps the controversial point can be considered the use of templates and transformations in the XSL system. The reason for the high threshold of entry, but after sitting and after picking up the use becomes more and more convenient and fast
.

Well, to the point



Soon there is a new version of HostCMS 6 and it eliminates the disadvantages.

While the version is available only for updates and is in the beta stage.

  • Developers refuse ZendOptimizer. Now the source code of the kernel and modules is available to us and we can change it. What naturally can not but rejoice.
  • Developed ORM - with the help of which it is now even more convenient to work with data.
  • To be honest, after looking at the source of the project, I was pleasantly surprised and saw the good old MVC. Very correct in my opinion the decision of the developers.
  • Already, you can upgrade to a new version with small dances with a tambourine. (dances are not long and in principle everything is solved simply)
  • In the administration panel, each new section now opens in a separate window that can be minimized. It is very convenient when you constantly switch from structure to typical dynamic pages, for example.


Conclusion



  • I advise everyone at least on the test server to update and see and rejoice for myself.
  • For those who have never set the system, it will be useful to install it.
  • We are waiting for the official release version.
