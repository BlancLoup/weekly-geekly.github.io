THL W3 - Catch up and overtake brands!


This Tuesday, we have on the table a top-end phone from the Chinese company THL, which according to its characteristics can compete with much more expensive branded phones. In the review - photos, tests, and of course the autopsy.
  • MT6575 processor running at 1GHz
  • 1GB RAM, and 2GB + 800MB user memory
  • 2 active SIM cards, one of which can work in 3G networks.
  • Screen IPS 4.5 ", with a capacitive sensor and a resolution of 1280x720 .
  • Gyroscope, proximity sensor, light sensor
  • Communications: GPS, Wi-Fi (n), BT
  • 2 cameras: rear 8MP , auto-focus, LED backlight and front 2MP
  • Android Ice Cream Sandwich 4.0.3
  • 2000mAh battery



About model THL W3 + - big brother

Already after I bought this phone, another phone began to appear on the virtual shelves (it is called differently, W3 +, W3s, W3 Dual) - all the same, but with the only difference - it is built on the 6577 processor, instead of the one used in this 6575 chip phone. Screen, memory, case and equipment are identical, firmware, however, do not fit.
Due to the fact that the new processor is compatible with the contacts (well, practically, the minimum modification of the board is still needed), Chinese developers can take the old board, fix a couple of tracks on it, installing a new processor and rebuilding the android, in fact, get a new phone.
Just like the I9220 from Haipai, the ‚Äúbig brother‚Äù appeared after a while - the X710D, and in THL W3 a new processor was added, receiving a new phone. Operating system firmware

How to flash:
1) Download the toolkit. Driver and firmware program
2) Download the firmware. Here is the official + rekoveri (you can not install it if you are not going to use it) , and here it is separately rekoveri . But the latest firmware is 0.4 , and the update is 0.5 .
3) Run the program (Flash_tool.exe) on behalf of the administrator, press the Scatter-loading button, select the txt file in the folder with the unpacked firmware (or just retrieve), which stores information about the addresses for which the firmware should be uploaded. Click Download. The phone is not connected!

2) Turn off the phone, remove-insert the battery, hold down the Volume-button, insert the cable. The system swears at the new device, we slip (if necessary - by force) the driver from the first paragraph. As soon as the driver is installed, the program will find the phone, flood the preloader with it (via com-port emulation, slowly - a red bar), and then launch it. Having started, the bootloader will get used to the phone, raise a full-fledged high-speed USB interface, to which the system, as always, will ask for the driver. Perhaps they will deliver them automatically, then skip the next item.
2.1) If the counter in the program counted, but did not find a new interface. Go to the device manager, see an unidentified device, install the same driver for it from the first item. Then, repeat paragraph 2.
3) If everything went well, the program will find the USB interface, then the flash driver will load (lilac stripe), which will already accept the firmware and stuff it into the phone‚Äôs memory (yellow stripe). After that, a window pops up with a green circle - that's it. We distort the battery, press the power button.
If you were flashing an official, this can be finished. But I don‚Äôt see the point in this, at first it is worthwhile to flash only recoveries, and then install firmware 0.4.
4) Copy the archives from item 1 to the memory card (ThL.W3.120714.6575.ICS.COM.RUS_v0.4.zip, ThL.W3.120714.6575.ICS.COM.RUS_v0.5.zip), go to Settings - About phone - Reboot Recorery.
