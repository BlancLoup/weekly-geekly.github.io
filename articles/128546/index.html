ProfitConf - conference of web developers in Novosibirsk on September 25-26

ProfitConf is a new conference about web development and web project management, which will be held in Novosibirsk on September 25 and 26 as part of Siberian Internet Weeks. And we want to make it damn good: we included excellent reports and several master classes into the program. But first things first.

  • "Modern approaches in layout: CSS3, HTML5, mobile platforms, responsive design, layout of mailing lists and other subtleties" - the second, revised version of the master class on layout from Yuri Artyukh from Evil Martians, which takes place not for the first time and not only Novosibirsk.
