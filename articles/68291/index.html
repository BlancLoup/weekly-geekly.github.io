Helios Javascript framework

Helios is an open-source framework and set of Javascript libraries for developing rich client web applications. The framework provides support for the modular structure, thus providing the ability to create applications in pure Javascript and at the same time use the usual include design to connect the required modules. A set of libraries in the kit provides the necessary API for developing the application. In particular, there is a handy Heliwidgets widget toolkit with support for customizable engines.

The link can be viewed on a demo application written using Helios & Heliwidgets:

home.gna.org/helios/helioscalc

Now the project is at an early stage of development and needs new members. If you are willing to join - write to heliosframework in the mail.

Source code for the framework and libraries is available under GPLv3 +

Project home page:
