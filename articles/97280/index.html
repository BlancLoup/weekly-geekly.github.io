Internet Explorer 9 Platform Preview 3

Today, the IE9 development team, following the promises made at the MIX10 conference, to publish updates for the new version of the browser, has posted the next update.

Download the IE9 Platform Preview on the IE9 TestDrive website .

In the new version, HTML5 audio, video and canvas tags (with hardware acceleration), support for the Web Open Font Format (WOFF) format, and a number of other improvements in speeding up and fixing bugs are now available.

New demo examples have also been added.
