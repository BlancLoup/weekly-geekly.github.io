DbOctopus - SQL editor with support for intellectual code completion

Somehow, in search of a free editor to work with SQL Server, I came across a good, from my point of view, DBOctopus program. The project was born not so long ago, so I did not expect to see something supernatural. But I must say, I was pleasantly surprised. Of course, there is still not enough, something is inconvenient, but among the advantages that are significant for me, I can mention the intelligent code completion mentioned in the topic. See how this is implemented in the program . I must say that even in SQL Server Management Studio 2005 from the well-known Microsoft, there is no such functionality. You can export sample results to textual formats such as xml, html, Excel, pdf and textual format. I was pleased with the convenient export to Excel. The program draws up the result very well in the final xls file with column headers.

Another plus is the ability to quickly filter query results without changing the SQL code. In the special field in the results window, you simply enter the filtering criteria and the program does the rest. Very comfortably. Here you can see how it looks.

The advantages include the efficiency in the responses of developers to letters with requests to improve the functionality of the program.

Among the major drawbacks of the program, it is possible to note the impossibility of full-fledged maintenance of SQL Server. Currently, DbOctopus will only work with tables, views, and procedures. For the administration will have to use a tool from Microsoft - SQL Server Management Studio.

As for the processing of requests and writing code in SQL, the program copes with the tasks set with a bang.

In addition, developers are currently distributing free licenses to work with the program in the event of:
1. If you will use the program for non-commercial purposes.
2. If you are a journalist and want to write an article or a note about the program.
3. If you want to make a presentation of the DbOctopus program to a group of 10 or more persons.
