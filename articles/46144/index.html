We work in DVD Studio Pro. Part I.

Not so long ago there was a need to create a multimedia DVD disc with a movie, menus, slideshow, etc. For certain reasons, this program was chosen. In this blog (I started in a friend, but this is my experience and copy-paste is used only for transferring ), I will try to tell you step by step how to get a decent result at the output. This opus does not pretend to anything - we will move from simple to complex.
Gradually - i.e. how time is enough%)

Briefly about the interface. Create a new project

Running DVD Studio Pro, we see 4 open windows: Assets, Project, Palette, Inspector.

Inspector window

First we need to decide on the format of the future disk, make the appropriate settings in the Inspector window
In the General tab, we set SD DVDs (that is, standard definition DVDs) - since this is the most common format at the moment. Also note the video standard PAL.
In the Disc / Volume tab, check Layer Options - Single, Number of sides - One, Disc Size - 12cm

In the Region / Copyright tab, you can not touch anything, I can only say that this tab is designed to restrict the viewing and protection of the (very conditional) rights of the manufacturer.

In the Advanced tab, you can also not touch anything yet.

Now it's better to save your project. I usually (regardless of the program in which I work) prepare in a certain way the place where the project will be located. For each type and type of files, I create folders in which everything is stored. This is done in order to make it clear that where it is and in general is order in the working place.

After saving the project, it is necessary to organize the workplace in the same way directly in the program.
Assets window

This window is used to import and manage the files that we will use. In this window there are several buttons - Import, New Folder, Remove. The meaning of these buttons seems to me to be clear. There is also the possibility of various sorts. Not all kinds of files can be imported. For example (it was a surprise for me) DVD Studio Pro does not work with .mpg - so we will work with .mov files.
Palette window

Palette - contains templates, styles, figures, there is a certain similarity of browsers for video and audio files, they are tied to the Music and Movies libraries, respectively. In this project we will not use templates and standard shapes - we will try to do everything ourselves. Music and video may not lie in the respective libraries (like for example, I don‚Äôt like iTunes and since the Movies library is organized. Project window
It's time to go to the last and, perhaps, the main window. This window contains the tabs Menu, Slideshow, Viewer, Graphical. For myself, I defined it as a project window - Project.

The Graphical tab contains the ‚Äúschematic diagram‚Äù of our disk. Initially, there is an icon under the default name Menu 1. Later you can replay everything, but for now let's rename the icon to MainMenu - that is, the main one. This icon has an icon in the form of a disk and an arrow, which indicates that the contents of this menu will be played first (First Play). It will be logical to conclude that there will be only one such element on any disk. The value of First Play can be put on any item, depending on the intended. As I said, Graphical is a scheme in which there are connections between elements that are subject to a certain logic. All links to the elements will be set through the Inspector window.
The Menu Editor tab is also one of the main elements of the program interface - this is where we will place the backgrounds and buttons.
The Slideshow tab is responsible for creating the slides accordingly - there will be a list of our photos.
