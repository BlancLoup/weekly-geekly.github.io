13 chips in Firefox (in Russian)

based on 13 chips in Firefox (in English)

You can get much more pleasure from using Firefox, knowing some of the chips.


1. To quickly search the page, click the slash (/) and start typing the word you are going to find (for example / word). The found word will be highlighted. To search for the next occurrence of a word, just press Ctrl + G

2. If you want to delete an address from the history of the address bar - open the drop-down menu of the address bar using the cursor keys, move the cursor to the desired address and press Shift + Delete.

3. Clear download history (Download history). This will make your download manager quicker. Tools | Options | Privacy

4. To view and save Firefox disk cache entries, simply type in the address bar: about: cache? Device = disk

5. In order to do the same procedure for the memory cache, type in the address bar: about: cache? Device = memory

6. To add a new download, simply drag the link into the download manager window.

7. If you accidentally deleted a bookmark and want to restore it - open the "bookmark manager" (Bookmarks> Manage Bookmarks) and use the key combination Ctrl + Z, or use: Edit> Cancel (Edit> Undo).

11. You can work offline (offline mode). Simply select File> Work Offline from the main menu. This means that you can view previously visited pages without having to connect to the network. This is truly a great opportunity that few people use.

12. You can add a page to your bookmarks simply by dragging the page icon from the address bar to the bookmarks folder. You can also drag the icon to the desktop, thus creating a shortcut for this page.
