How deep is the rabbit hole? CLRium # 5: Garbage Collector

The world is rushing forward, driven by progress and competition. You and I are unreal lucky: after all, the greatest minds work for us, creating truly serious mechanisms: compilers, IDE, databases. Make them so that we get real pleasure using them.


One of these mechanisms is the operation of the memory management subsystem of the .NET platform. And if before no one had the slightest idea how it works, now, when we have the source code, all the secrets are open. Now you can see absolutely all the details of the work of the platform's core: and this is really fascinating. For example, if earlier we thought that memory allocation is just to move the pointer, then now it became clear that everything is somewhat more complicated and interesting. This is a verified, excellent algorithm that leaves no questions: why it was done this way.
