fheroes2 - Open Source Engine Heroes of Might and Magic II


free heroes2 is a free version of the Heroes of Might and Magic II engine. Written using SDL by a Russian (seemingly) programmer. There are ports on Windows, Linux, WM, WinCE, Dingoo a320 , Cowon V5 ... well, almost everywhere. Anyone who wants to know more - under the cat

First you need to download the binary from the site if you have Windows or WinCE. If you are using Linux, then download the source from the same site or pick up the latest revision from SVN: there are no ready packages on the site. Compiling Linux versions did not cause any problems. But this is not enough: we still need the original files of the game Heroes of Might and Magic II: The Succession Wars (English version is required). They are easily searched in google. We throw the file heroes2.agg in the data folder, and the maps * .mp2 - respectively in the maps. In the fheroes2.cfg file, you can set the desired resolution, enable Unicode support (if you want to play in the Russian version ( help with translation)) and much more.



All is ready. We call friends to play with one computer or set up a network. I have not tried the game on the Internet yet, there is nobody with it, and even playing with one computer, hot seat, is more old-school)) Unfortunately, the campaign mode is still in development, maybe someday they will finish it. But you can play on a single map with AI.



If you wish, you can add music and sounds in the * .ogg format for better sound quality, but even so it is played very well.
