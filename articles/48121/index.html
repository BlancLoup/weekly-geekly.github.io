ATI R600 / 700 Video Cards Now Open Source

AMD has released the Open Source code for the ATI R600 and 700 3D accelerators, including models for the newest Radeon HD 4xxx line.

Among the published code:

1) a demo program that performs preprocessing of commands;
2) updated driver RadeonHD;
3) Updated Direct Rendering Manager.

At the same time, support for two-dimensional EXA acceleration for the latest ATI processors, as well as basic support for X-Video, has been made.

All the necessary documentation, AMD will publish in the coming weeks.
