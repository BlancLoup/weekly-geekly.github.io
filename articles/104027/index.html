Opera and Facebook Widget Contest



Opera Software company regularly holds various contests and Russian developers from time to time are among the winners, receiving well-deserved prizes and gifts. True, there is always a small, but rather annoying inconvenience: developers from other countries constantly make up pretty strong competition, often depriving our contestants of prizes. Today, I am pleased to announce the start of a new contest held jointly by Opera Software and Facebook, which guarantees that all prizes will go only to our people: as part of the first Facebook Facebook days recently held in Russia , Developer Garage has launched a competition to develop Opera widgets for this popular social network. network called ‚ÄúOperator Facebook‚Äù, and only programmers from Russia and neighboring countries can participate in it. Purpose of the competition
The main task we want to achieve with this joint event is the emergence of powerful and convenient opera widgets that allow Facebook users to discover new opportunities to use this social network, gain additional comfort, or even completely revise their style of work with this portal. Whether it will work out or not depends only on you: we do not limit the flight of developers' imagination and do not set additional conditions other than traditional and, in general, quite logical: widgets should not contain a code whose freedom of use is limited by copyright or patents at the same time, widgets must fully comply with existing web standards.

By the way, it should be recalled that opera widgets can be made universal - not only for a computer, but also for mobile devices.
