Firefox 4 may be released next week.



Mozilla Corporation is preparing to release the final version of the browser Firefox 4 next week. This move is caused by the appearance on the market of Internet Explorer 9, the frequent exits of updated versions of Google Chrome. Mozilla fears that if you delay the release of the final version of Firefox 4, you may lose some of the browser market (and after all, Firefox is slowly losing market percentages). In general, it is not yet known for sure whether Firefox 4 will see the light next week, but this is very likely. Note that the other day, Firefox 4 RC1 was released, which received warm testimonials from testers, which allows developers to bring the release date of the final version closer.

So far, the possible release date for Firefox 4 is March 22, but this date has not yet been approved. If any critical errors are not found in Firefox 4 RC1, it will become the final version. However, according to the developers, if critical vulnerabilities or errors in the code are found, RC2 will be released. This will be done "as quickly as possible."
