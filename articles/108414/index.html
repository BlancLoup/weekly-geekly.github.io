Dobroknizhka about HTML5 from Google

As long as there are wars on the Internet about HTML5, whether it will kill Flash or not, Google has released an online book, 20 Things I Learned About Browsers & The Web . And this is really a book!

The guys created a web application that resembles a children's book, you can browse it, leave bookmarks, and it is dedicated to web technologies.

The first thing to note is that the book was created using only HTML5 features, so each page of the book is an html page. For this reason, each of the pages can be shared in social services, and to share independently of the rest. And all for the same reason, all these pages are indexed.

Thus, this is probably the first real experience of creating a normal web reader. And if you add a touch screen here, then we get an excellent piece!
