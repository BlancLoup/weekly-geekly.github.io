BitTorrent Sync 2.0 has been released with a paid PRO package



BitTorrent Sync comes out of beta. Meet the new version 2.0 with the extended PRO package.

The service is offered in paid and free versions. The paid version of Sync 2.0 will cost 2499 rubles per year.

Pro version users can:
  • Manage folder permissions (read / read and write only)
  • Deny user access on subsequent folder updates
  • Assign other users as folder owners
  • Set up automatic data synchronization between connected devices
  • Add unlimited number of folders
  • Get fast technical support.

The pro version can be tried for free for 30 days.

The limitation of the free version is the synchronization of no more than 10 folders (for file size and transfer speed there are no restrictions).



A more detailed list of changes to version 2.0 can be found here .

Versions for Windows, Mac and Linux can already be said on the official site GetSync.com . Versions for mobile devices will be updated in the markets during the day.
