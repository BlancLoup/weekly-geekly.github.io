Orvibo AllOne: the universal remote control for home gadgets



All sorts of home appliances in any home are now quite a lot. Air conditioners, televisions, air purifiers and humidifiers, heaters ... This list can be continued indefinitely. For the sake of user convenience, most of these devices are equipped with a remote control - it is convenient, sitting at the TV, to regulate the temperature at home, on the spot.

Only now the consoles, with the acquisition of each new device, more and more, and in some apartments you can see 3-5 or even more consoles around the sofa in some apartments. They are lost, confused, batteries are sitting down somewhere, something is breaking. These are all small but annoying problems, and Orvibo has come up with an interesting way out.

Features Orvibo AllOne



This output is ... another device, a universal remote control, which eliminates the need to work with any other remote controls. The developers ‚Äútrained‚Äù the gadget by inserting into their memory control codes for thousands of home appliance models and electronic devices. Orvibo AllOne itself is controlled from a smartphone, with a pre-installed application.

Orvibo AllOne - a kind of hub that allows you to control all the equipment in the house, which has an IR receiver and remote control. The form of Orvibo AllOne, as seen in the photo, is round and 3 IR emitters are installed around the circumference, thanks to which the device sends signals in all directions, there are no dead zones here.

The device supports both Android and iOS devices.

How it works?



  • To start, you must perform the following steps:
  • Connect Orvibo AllOne to the smartphone via WiFi (respectively, you need to install the control application on the smartphone);
  • Perform pairing;
  • Pick up the IR code for all devices in the house;
  • Save settings to the "cloud", where they will be available at any time.
