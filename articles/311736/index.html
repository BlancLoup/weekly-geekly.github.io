Report from Moscow Python Meetup September 22



In late September, our office hosted the next Moscow Python Meetup. And today we want to share the materials of speeches, under the cut you are waiting for the presentation and video of three reports.

- Alexey Lobozov, Project Practice Group, leading consultant. Using Python to build network models

The report is devoted to the Predict module, which allows you to build network models for project management.





- Vasily Bolshakov, independent developer. What distinguishes a junior from a senior or as a pythonist do not have problems finding a job

The speaker shared his opinion on what you need to know, be able to and, most importantly, understand, in order to find a job, if you are a beginner Python-developer.





- Dmitry Shveenkov, Mail.Ru Group, head of the server development team. Features of the backend for mobile applications or Python Django UWSGI in production

The report is dedicated to the life of projects in production. It will be about a bunch of Python-Django-Uwsgi for backend mobile applications, setting up servers, measuring time and assessing the quality of the backend. Also, the speaker will talk about profiling Django, Memcached in Python.

Here and here you can read Dima's articles about asynchronous work with Tarantool in Python.
