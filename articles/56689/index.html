Jailbreak firmware 3.0 for iPhone

iPhone OS3.0 beta 2 ( 7A259g ) has already surrendered. We present you the jailbreak 2nd beta.



From the authors:
I will not discourage you from installing the firmware as it is done on other sites. I just want to warn that this firmware version is intended only to familiarize with the new version of the OS, but not for permanent use - as there are too many bugs in it.



Supported firmware:
7A238j (3.0 beta 1), 7A259g (3.0 beta 2)

Supported devices:
- iPod Touch 1G, iPhone 2G, official iPhone 3G. American 3G does not sew!

Installation:
1) Making a recovery (Shift + "Restore") for firmware 3.0 beta 2;
2) Run the patched QuickPWN;
3) Choose:
iPhone 2G - Jailbreak, Activate, (Logo)
iTouch - Jailbreak, Activate, (Logo)
iPhone 3G with reg. UUID - Jailbreak, (Logo)
iPhone 3G without registration. UUID - Jailbreak, Activate, (Logo) - will not catch the network


4) Follow the instructions;
5) Enjoy jailbreak OS3.0;

On the official 3G without a registered UUID developer, the network will disappear, but only before flashing to 2.2.1.

Files:
- firmware 3.0 beta 2 for iPhone 2G , iPhone 3G , iPodTouch 1G ;
- QuickPwn-030-2 ;
- A set of fixes ;
