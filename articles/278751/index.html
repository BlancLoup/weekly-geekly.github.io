Work with private repositories and other updates of the FlyElephant platform



The FlyElephant platform team has prepared a number of updates that allow you to work with private repositories, increase system security and improve work with tasks.

FlyElephant is a platform for scientists that provides ready-made computing infrastructure for carrying out calculations, helps to find partners and work together on projects, as well as manage all data from one place. FlyElephant automates routine tasks and allows you to focus on core research issues.

Azure cloud is used as a computing resource. Users can run computational tasks written with C ++ (with OpenMP support), R, Python, Octave, Scilab, Java, Julia, OpenFOAM, GROMACS, Blender on servers with 1 to 32 cores and 448 GB RAM .

Among the innovations are the following:

  • Work with private repositories.
    The opportunity to work with a private repository using the RSA key, which can be obtained in the settings of your account.

  • Improved procedure for running computational tasks.
    Now the program launch command can be specified directly in the task creation form. The required run.sh and build.sh files have been removed from the repository.

  • Added functionality for saving, stopping, deleting and duplicating tasks.

  • Improved logging system.
    On the details page of the task, in real time, you can see all the logs of the task.

  • Public API.
    A public API has been implemented that allows you to interact with the platform (manage tasks and data), as well as integrate FlyElephant with your existing solutions or build your new solutions based on the platform.

  • New payment system.
    Users now have two accounts: Computing Hours (CH) and General ($). The first account is a bonus, there is free time for calculations. The second is the main account, which can be credited with a credit card.

  • Referral program.
    By inviting their friends through a personal link, which is in the account settings, users can receive an additional 200 bonus hours.

  • SSL is implemented.

  • New documentation.
    The latest documentation is available at https://flyelephant.net/documentation/ .

We invite everyone to try the new functionality. If you are not registered with FlyElephant, register here . After registration, your account will have 500 hours; using the promo code 145124070132 and entering it in your Personal Account, you can recharge your account for another 300 hours.

We will be grateful for the feedback and suggestions for improving the platform, and we will also be happy to give you a 30-minute demo presentation on working with it. Book a time for a demo presentation here .
