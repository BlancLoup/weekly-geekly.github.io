New beta: Flash Builder 4, Flash Catalyst, Flex SDK 4

Somehow, behind the good news from Google, news about new Adobe products, especially Flash Catalyst, quietly passed through.

If someone does not know what Catalyst is - imagine how the site is being created at the moment, and we will describe it like this:

PSD -> HTML -> Server code

or even easier:

graphic design -> layout -> server code

Flash Catalyst is a product for flash makers that allows you to make the transition ‚Äúgraphic design -> layout‚Äù in a flash project.

The result of the work of Catalyst is fxg-code, which is transmitted to programmers for further work.
The product is still quite brake and eats a lot of memory, but there are surprisingly few bugs. I myself have been waiting for him for a long time, I liked working with him.

Flash Builder 4 and Flex SDK 4

The main feature of all - the new format FXG (1.0). This is a new XML format for working with graphics for the Flash Platform. It describes high-level graphics and text primitives.

The only disappointment is that the output swf file is very large, from 1 MB (compiled empty mxml took swf in 596 kilobytes), but maybe future browsers or flash-plugin itself will support FXG?
