Adobe has updated Flash Player

Adobe has released an update for Flash Player APSB15-28, closing 17 vulnerabilities in it. Most closed vulnerabilities are critical and can be used by attackers for remote code execution in a web browser, as well as organizing drive-by download attacks. The update does not address any 0day vulnerabilities in the player that are currently being used by attackers.



We recommend updating your Flash Player. Browsers such as Internet Explorer 11 on Windows 8.1 / Win10 and Google Chrome, as well as MS Edge update their Flash Player versions automatically. For IE, see the updated Security Advisory 2755801 . Check your version of Flash Player for relevance here .
