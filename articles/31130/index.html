Safari 3.0.2 beta

Download

Corrected the following:

- Fresh security updates.

For non-English versions of Windows:
- Safari unexpectedly collapsed in the management of bookmarks, or when entered into text fields.
- Lost bold and oblique text.
- Safari was not loaded if the user name in the system contained a non-English character.
- Glitch, when absolutely all the text disappeared, if the system has a large number of fonts, or when Safari was installed in a folder other than the default folder.
- Slow loading time if the system has a font whose creation time is specified in the future.
- Glitch, when sometimes on the pages instead of the Arial font a completely different font was displayed.
- Safari sometimes crashed when entering username and password during http authentication or when trying to process unsupported NTLM authentication
- Falling on buggy RSS feeds
- Hangs while updating RSS feeds
- Now you can install the Adobe Integrated Runtime (AIR) and Yahoo widgets.
- Safari fell while viewing Google Reader.
- Falling on systems with non-SSE processors, or with some specific video cards, or when a web server generated an empty field
Content-type in the HTTP header.

Personal impressions: Hooray! He started at my home! :) The truth was loaded for about a minute and ate more than 100 MB of memory. In the office, 3.0.0 starts instantly. (The difference in iron is not significant)
