TrueCrypt 6.1 has been released

In the new version:
  • Encrypt data discs without losing them. ( does not work in Windows XP Works in Windows Vista / 2008)
  • Support Tokens and Smart Cards
  • Disable and change the design of the bootloader. (When encrypting the system disk)
  • Pre-boot authorization can now be cached, which allows you to mount non-system partitions encrypted with TrueCrypt (apparently, during the boot process).
  • in versions for Linux and Mac OS X: the ability to mount a Windows system partition encrypted with TrueCrypt and mount a partition located on the Windows system disk that is encrypted with the Windows version of TrueCrypt.
