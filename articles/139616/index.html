New Three Features from Amazon Web Services

Only recently, amazon lowered prices on its instances on demand by 10% and on reserved instances by 30%, as a letter came in about new features that I lacked so much.

1) For 64-bit ami, there are instances with a small amount of memory available, which gives more maneuver in vertical scaling. Previously, the smallest instance with 620MB RAM could be used for 64-bit, and the next ascending instance was with almost 8GB of RAM. Now it is available as for 32-bit ami instance with 1.7GB of RAM.

2) There is a new type instances m1.medium with 2 EC2 Compute Units (ECUs) and 3.75GB RAM, now tacking in the lower price range has become easier.

3) Now you can finally log in to the Linux console from the amazon web panel.

This is just a miracle. Amazon did everything I missed in it.
