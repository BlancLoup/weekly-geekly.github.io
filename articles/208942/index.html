The distribution of the Nokia Black update for Lumia smartphones has already begun

Hi, Habr!

Just recently, the distribution of Lumia Black update for Nokia Lumia smartphones based on Windows Phone 8 was launched. The update included a number of new features and features, improvements in photo processing, new applications and innovations from Microsoft and Nokia.



In this post, we decided to sort out the list of innovations in more detail in this version of the firmware, so welcome under cat.

Folders / App Folder


This long-awaited feature (implemented as a separate application) is designed to help users organize and organize applications on the start screen. ‚ÄúFolders‚Äù, as the name implies, will be able to group several applications or separate settings on a single scalable tile, which is sure to be the way out for those users whose start screen is heavily overloaded with applications.



Screen Nokia Glance version 2.0


Now the Nokia Glance screen (screen of the locked smartphone in the standby mode) will be able to show not only the current time and battery level, but also notify the user about missed calls, unread messages and letters. The waiting screen will display the information that the user sets to display on the lock screen. And this, in addition to the above-mentioned notifications, can be alerts for Outlook, calendar, as well as some games and applications, such as Twitter, Facebook, WhatsApp.



Since Nokia Glance uses a low screen brightness mode, new features will allow users to stay up to date without significant battery charge.

Another, perhaps, not very significant, but a nice addition was the ability to set the color of the display information in the Nokia Glance night mode. The user is offered a choice of red, green or blue color graphics (previously, the default was red).

Bluetooth LE


Bluetooth LE support will optimize the performance of Lumia smartphones with wireless gadgets of the new generation, like the Adidas MiCoach sports sensor. Low power consumption in the new technology will get rid of the "headache" of the owners of these devices - the rapid discharge of devices due to the constantly turned on "bluetooth".

Nokia Camera / Nokia Refocus (Nokia Focus Change) / Nokia Storyteller (Nokia Stories) / Nokia Beamer (Nokia Projector)


We have already written about all these applications in detail earlier . It is worth noting that now they have received Russian names and are available for all devices of the Lumia line (Nokia Refocus and Nokia Projector are available for smartphones with 1 GB of RAM and above).


Nokia Projector application

An important point for the owners of Nokia Lumia 1020: the new update with the installation of the Nokia Camera application will also receive support for ‚Äúdigital negatives‚Äù (RAW in DNG format), which was previously implemented only in Lumia 1520. Due to this pleasant circumstance, mobile photo fans will have more post-processing capabilities.



New generation of image processing algorithms


In Lumia Black, we have improved image processing algorithms, including oversampling algorithms. As a result, we managed to achieve even greater improvement in the quality of photos, especially in the model Lumia 1020.

In general, on all devices we managed to achieve more natural, natural colors, reduce noise, leaving the sharpness and sharpness of images at a high level.

"Buns" from Microsoft


It is no secret that the Lumia Black update is based on the global GDR3 update for Windows Phone 8. Therefore, with the Lumia Black update, Nokia Lumia smartphones will come with several improvements from Microsoft. Among them, the safe mode ‚ÄúIn the car‚Äù, which will allow users to configure automatic disabling of all notifications, except for incoming calls and SMS with binding to a specific Bluetooth device.



In addition, with the update, users will also have the opportunity to set personal ringtones for each contact.



The functionality of the task manager will be somewhat expanded (users will be able to select and close applications directly from the manager menu), and in the settings there will be an option to block the screen rotation.



As we mentioned at the beginning of the post, the spread of Lumia Black has already begun. The first update will get smartphones Lumia 1020 and Lumia 925, but soon it will come to other models.
