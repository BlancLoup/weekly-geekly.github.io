Technical Reference


Point - a subscriber who has his own address, but does not have the right to vote on the Fido network.
A node is a network node, the main unit of the Fido user calculus.
A nodelist is a list of hosts that is both an administrative and technical (for routing mail) document.
Mailer is a program that provides data transfer between nodes or between a node and a point. In my case, binkd.
Tosser is a program that packs and unpacks the echo-conferencing data received through the mailer and scans them according to the necessary folders and files.
Tracker is a program that manages the transfer of personal mail. By simple - a tosser for netmail.
Netmail - personal mail.
Ehomail - ‚Äúpublic mail‚Äù, divided into sections (echo-conferencing).
