GNOME 2.29.0 Shell with a new notification system



GNOME Shell 2.29.0 is available for testing. According to the developers, the main feature of this version is the new notification system. The information panel at the bottom of the screen informs users, for example, about new connected devices or the track being played. The previous notification system is preserved and can be accessed through a special area in the lower right corner of the screen.



Another new feature allows you to set a global status that instant messaging clients and other applications can use.



In addition, the switching mechanism between active windows has been redesigned. Initial support for extensions has been added.





A complete list of changes can be found in the release announcement .

The developers emphasize that this version of GNOME Shell is not yet ready for corporate use.

The GNOME Shell source code is available for download from the project's FTP server.
Ubuntu users can also use the PPA repository .
