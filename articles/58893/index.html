Moonlight 2.0 preview 1 released

Moonlight 2.0 preview 1 became available for review and testing. Let me remind you that this is an open-source implementation of Silverlight technology for many platforms. In this issue:

* support for Silverlight 2.0 (not yet complete, but very close to full);
* Added controls for the original Silverlight, which are released under the MS-PL license (GPL-compatible);
* in addition to C #, you can write your code in dynamic languages ‚Äã‚Äãsuch as IronPython and IronRuby;

* DeepZoom support;
* support for some Silverlight 3.0 chips (which is still in beta status): out-of-browser, class WritableBitmap, support for SaveDialog, plugin mechanism for media conveyor (for writing your own codecs).
