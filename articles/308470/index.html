Linux is 25 years old


Linus Torvalds himself

Time is running very fast, and the Linux operating system is already a quarter of a century old. It would be more correct to say that the kernel of this operating system has turned 25 years old. The general history of the advent of Linux is probably familiar to all Habrahabr readers. On August 25, 1991, five months after starting work on his project, 21-year-old Linus Torvalds (then a student) spoke about creating a prototype of a completely new OS called Linux.

On September 17, 1991, the first public release of the Linux kernel took place. The kernel version at that time is 0.0.1 . Already then the number of lines of kernel code was 10 thousand. Its size was only 62 KB in compressed form. Now the core has many times more code lines - as many as 19 million. If the OS was developed by a commercial organization, the cost of such a project would be about a billion US dollars, or even more.

Linus Torvalds decided to create a kernel after working with the MINIX operating system. She did not suit the student limited license. As usual, Torvalds tried to blame plagiarism. Namely, in that he simply copied the code to a number of MINIX subsystems. But experts managed to prove that it is not. MINIX author Andrew Tanenbaum himself compared the code of his OS and Linux, and came to the conclusion that there are only a few irrelevant coincidences in the code that you can ignore. These matches are due to a number of POSIX and ANSI C requirements.

Interestingly, Linux might initially be called Freax ("free", "freak", and X (Unix)). This name was given to his project by Linus himself. But Ari Lemke (Ari Lemmke), who at the request of Linus laid out the kernel on his FTP server, called the directory with the kernel ‚Äúlinux‚Äù.   • 2.6.35 - August 2010, 13.4 million lines of code;

The kernel is developed by third-party developers. According to the Linux Foundation, since 2005, 13,500 specialists have participated in the development of the system. The average speed of work on the system is 7.8 patches per hour. In the development of the system, not only independent developers, but also many representatives of large technological corporations are taking part. Among others, we can mention Intel, Red Hat, Linaro, Samsung, SUSE, IBM, Renesas, Google, AMD, Taxas Instuments and ARM.
