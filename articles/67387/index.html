Typographer Evgeny Muravyev for TinyMCE

In the topic New version of the web-typographer of Studio Muravyov was presented a wonderful typographer.
The typographer processes texts not only according to classical laws (non-breaking spaces, correct quotes, hanging punctuation, etc.), but also according to the canons taken from Milchin A. ‚ÄúPublisher's and Author's Guide‚Äù. Fans of typography should appreciate this.

I used it in my store engine, and the simple and convenient plugin for TinyMCE became a by-product.


When you click on the button, no pop-up windows are displayed, the text is silently typed.

Download this typographer for TinyMCE
Online demo

How to tie it to TinyMCE


1. Copy the jaretypograph folder to tiny_mce / plugins


2. Connect the plugin when initializing Tinymce


Like that:
tinyMCE.init({
...
plugins : "jaretypograph",
...
});

3. Display the plugin button on the editor toolbar.


Like that:
tinyMCE.init({
...
theme_advanced_buttons1 : "jaretypograph",
...
});

Typographer works on PHP5 and UTF8.
