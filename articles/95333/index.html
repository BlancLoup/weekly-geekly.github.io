Autoscan a project in NetBeans 6.9

The NetBeans for PHP blog talks about the possibility of enabling and disabling project autoscan in NetBeans version 6.9.

The developers finally heard the requests of users complaining about the slowdown of work when scanning projects, and implemented the ability to disable the automatic mode.

Especially the slowdown was noticeable with a large number of folders in the project. NetBeans also rescanned projects when files were changed from the outside.

It was necessary to maintain the relevance of auto-completion and project navigation.



The ability to force a scan of changes is also implemented.
