Windows 7: work with virtual applications

Recently, there has been a tendency to move to / on cloud computing.


From Wiki: Cloud computing (cloud computing) is a distributed data processing technology in which computer resources and facilities are provided to the user as an Internet service.

One of the most difficult moments is the transfer of applications to a virtual environment. I want to tell about the solution from Microsoft - Application Virtualization for Desktops, it is included in the Microsoft Desktop Optimization Pack and very successfully "virtualizes" all the software you are used to.




The essence of this solution is simple: you install a server in your company on which virtual application packages are stored. These packages are created in a special way: an observer program is put on a clean system: Sequencer - this program oversees the installation of a program on this system and creates a virtual application package at the end. The main difficulty of the whole technology is to competently create this package. Not a few broken copies when trying to create them, but this is only a matter of experience. Then this package is placed on the above server, in its settings you can specify the AD groups to which the application will be available. And you can assign it to anyone with a Desktop Client installed on a Windows computer.



The scheme is working now and can be successfully applied in any company. After installation on a clean Desktop Client, you will immediately receive the entire set of software required for work.



The advantages of this approach:
+ No need to install software on a local computer;
+ no need to worry about updating it;
+ license responsibility is shifted to the server side;
+ If you store data in the cloud and use virtual applications, then your PC is no different from a regular terminal.

Disadvantages:
- such a mechanism is currently available only on the local network, due to the speed of the application download;
- not you decide what set of software you will be available;


I dare to suggest that this decision is a serious step towards the virtualization of everything and everyone. Perhaps in the future, all our PCs will simply be cross-platform terminals, to which certain applications will be delivered by subscription, you just need to bring it to the Internet.
