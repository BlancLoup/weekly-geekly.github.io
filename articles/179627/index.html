So, at the heart of any such storage medium is a keyboard for dialing a pincode. The devices are presented in various form factors: USB flash drive (datAshur, 4-32 Gb) and hard drives (250-3000 Gb).

datAshur




datAshur is a flash drive with a keyboard for entering a pin code. A very big plus of this device is that when working, no additional software is needed - Windows, Mac, Linux, Android and other operating systems that understand ordinary flash drives will easily accept this one.

The cryptography is based on 256-bit AES encryption on the fly, that is, all data that gets onto the USB flash drive is already protected and memory is soldered to another, ‚Äúordinary‚Äù USB flash drive will not be able to see the contents.

The flash drive weighs 25 grams, dimensions: 80x20x10 mm - a bit more than the average flash drive at the current moment, but the impact resistance and moisture resistance, in my opinion, are worth it.

A flash drive is available in 4 different volumes: 4, 8, 16 or 32 Gb. The read and write speed is at a very good level - up to 27 megabytes per second for reading, 27 - for writing.

Now, about the pin code. To activate the flash drive you need to dial a pin code (from 7 to 15 digits) on the keyboard and press the button with the key. If the code is entered correctly, the green LED lights up and you can connect to the computer. Otherwise, the red LED lights up and you can try entering the pin code again. After 10 incorrect attempts to enter the pin code, the data on the flash drive is erased permanently, and in this case only a complete reset of the device will help. The manufacturer provides 2 types of pin-code: administrator and user. Administrative PIN code is useful, for example, in case you want to give this flash drive to your employee, but, in which case, have access to the files.
After using the flash drive, simply unplug it from the computer and an automatic lock will occur, no additional actions need to be performed to block it. This is convenient, if a tax has come to you - you pull out the flash drive and you need not worry. Inside there is a built-in battery, which is automatically recharged when the USB flash drive is connected to a computer. The memory is non-volatile, if the battery sits down - you just need to plug in the flash drive without entering the pin-code for half an hour and the battery will charge again.

Such a device costs from 39 to 99 pounds (1900 - 4800 rubles), depending on the volume. The manufacturer provides a warranty of 3 years.
