Ciklum Java Subbotnik in Kiev, November 1

We are glad to please the Kiev audience of Java lovers with the upcoming Ciklum Java Saturday meeting, which will be held on November 1, from 11 am to 4:30 pm in the Kiev office of Siklum.

Ciklum Saturdays is a free event, open to everyone, where you can not only listen to interesting presentations and learn something new for yourself, but also chat in an informal circle of like-minded people.



As always, this time we have interesting speakers who will share their experiences with the participants:

1. Andrey Kravets
Topic: Is it time to be reactive, or ..?
Modern software requirements place fairly stringent restrictions on both advanced developers and the products they are working on. Reactive programming is exactly the approach that provides incredible flexibility, reliability, and at the same time ease of use in the process of creating complex, high-quality applications.
Is it time for everyone to make a choice, to be reactive or not?

2. Alexander Zakharov
Topic: Using EntityGraph in JPA 2.1
Entity Graphs is very important, but somewhat undervalued in JPA 2.1./Java EE7 and makes it possible to significantly improve performance compared to traditional lazy / eager. Proper use of entity graph gives a significant performance benefit to the application. The report will explain how to define entity graphs using annotations and programmatically.

3. Anton Gordiychuk
Topic: AngularJS - Advanced HTML for Web Applications
The report is primarily focused on developers who have not previously worked with AngularJS. In fact, it will be familiarity with the framework as well as its main components and capabilities. Target audience - developers who are interested in the latest trends in web development.

4. Alexander Zakharov
Topic: Domain Driven Design: Design by Model
When designing software, developers face the problem of choosing one or another architectural approach. In this report, the design is considered on the basis of the domain model, first considered in the articles of Fowler. Further development of this idea was made by Eric Evans in "Subject - Oriented Design". The report will discuss the main ideas of the subject-oriented approach and the organization of the project structure with practical tips.

5. Nikolay Papirny
Topic: Java memory model for mere mortals
Multi-threaded programming in java is a very difficult task, and the creators of the language made it easier to write. That is why they entered the JMM language specification. On the report you will learn about the features of the memory model in Java.
