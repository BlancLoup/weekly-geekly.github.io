YouTrack 2018.1 release: dependent task fields, personal localization and much more

Hello! We have released a new version of the project management system - YouTrack 2018.1 - and we hasten to share with you the innovations.




What's new in YouTrack 2018.1?

  • dependent task fields;
  • personal localization;
  • date-based lists;
  • custom widgets;
  • event Ribbon;
  • Markdown support;
  • export in Excel format;
  • Terms of use;
  • workflow API updates;
  • improve import functionality;
  • Mandatory email address verification.

Experimental functionality:


  • new task page;
  • checklists in Markdown.

Dependent task fields


Starting with YouTrack 2018.1, you can customize the display of task fields depending on the value of the task field. These conditions can be set for individual projects. For example, in your project there is a field - the type of the task (with the values ‚Äã‚Äãof the bug, support request, task) and the version. With this functionality, you can hide the ‚Äúversion‚Äù field for a task of the ‚Äúsupport request‚Äù type.




Personal localization


Previously, you could only use the localized version of YouTrack at the server level. In addition, localization included translation of the interface, as well as search queries, commands, and so on. It was not very convenient for many users, so now you can choose the interface language for yourself in the profile.




To ensure consistency, the language of search queries, commands, and custom fields is set globally. For example, if the server language is English, and your local language is Russian, then you can use the search and commands in English.

Date-based Slabs


Now on the Agile board, you can organize your slatelines by the ‚Äúdate‚Äù field. This functionality will help you keep track of the tasks you need to accomplish today, tomorrow, this week, and so on. If it is on, every midnight board will be updated and the cards will fall into the corresponding Christmas label. For example, the tasks that had to be done today will fall into our own slave yesterday.




Custom widgets


In addition to standard widgets, you can now add your own widgets, as well as use widgets created by other users on the dashboard. This functionality allows you to use content from external sources and display them in widgets right on your dashboard. Now each user can create and add a widget to the JetBrains plugin repository . After the widget passes verification, it will be added to the repository. Here you can learn how to write your own widget. Join YouTrack Community at Slack to share experiences.




event Ribbon


In the release of YouTrack 2017.2, we announced the event tape as experimental functionality. The event ribbon replaces the tab-based interface that was used when viewing and editing tasks. This functionality is now enabled by default on the Agile board, as well as in a new experimental mode of viewing and editing tasks, which I will discuss later.




Markdown support


Markdown support is now available by default in YouTrack for all users. You can switch between Markdown and YouTrack Wiki if you are used to the second.



Export Excel


We added the ability to export the task list in Excel format. Unlike CSV, exporting in Excel format saves data types for values ‚Äã‚Äãin custom fields.


Terms of use


In connection with the requirements for the protection of personal data for companies operating in EU member states, we have added functionality that allows you to add, save and edit the user agreement that users need to accept to enter YouTrack.




Improvements and updates


Workflow API Updates


In the new version of YouTrack, we have expanded the functionality of the workflow API:


  • added partial support for ES6 / JS2015;
  • Added new methods that allow you to determine when changes were applied without notifications, respond to deleting comments, and check whether Markdown is used to format a description of a task or comment.



Import functionality improvements


We have improved the functionality of importing projects from other trackers. We also added documentation for working with importing scripts using Python.


Required Email Verification


Now YouTrack administrators can enable the functionality that will require confirmation of user email to log in to YouTrack. This functionality enhances the security of your server, especially in cases where you allow self-registration or create new accounts using Mailbox integration.




Experimental functionality


In each release, we add experimental functionality so that you will be the first to try and evaluate our new products in the early stages of development. Do not forget about feedback, we really appreciate it! Recall that in order to use experimental functionality, the administrator must enable it on the Feature Configuration page via Global Settings.


New task page


We are pleased to present you an updated task viewing page. The new page supports Markdown, checklists, sorting comments in reverse chronological order, improved @mentions, improved field usability, a dark graphic theme, the ability to download all attachments to a zip archive. Comments, task history, time tracking, and VCS changes were merged into an event feed with the ability to quickly turn on and off various events.



If this functionality is enabled globally, just tick the ‚ÄúUse experimental version of the full page issue view by default‚Äù box in your profile to use the new default page.


Checkpoint Markdown


In markdown markup, you can create checklists. To fully use them, we recommend switching to a new version of the task page. In the old version, they are not fully supported.



We also made a number of other improvements - see the full list of completed tasks included in the release.

If you already use cloudy YouTrack, then your instance will be transferred to the new version on the calendar .
