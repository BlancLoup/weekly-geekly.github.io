Quite an interesting replacement for the standard terminal.



Enlightenment Workspace Developer Community has released a new version of its terminal emulator, Terminology. Details under the cut

In addition to the standard functions for a normal terminal, it has:
- viewing PDF, video, vector and bitmap files directly in the terminal window
- support tabs (not quite classic)
- setting appearance (wallpaper, color scheme, transparency, fonts, etc.)
- support for dragging text / links (a very relevant feature, at least for me)
- url, file paths, soap addresses will be highlighted as links
- inline content viewing by reference
- supports both X11 and Wayland
- adapted to the touch interface (apparently a tribute to samsung for financial support of the Enlightenment project)
- rendering as due to OpenGL so OpenGL-ES2 (not necessarily a requirement, just as an option)
- multibuffer
- supports most rvtx escape sequences and xterm + xterm extensions
- And much more‚Ä¶

Terminology is developed on the basis of the EFL toolkit and only the EFL and libc libraries are needed for it, while it uses minimal dependencies and supports the work of the fbcon framebuffer.

Some commands:

tyls - list of files and folders with thumbnails
typop - view the image in a small window, you can also just poke the mouse
tycat - also viewing images only directly in the terminal itself
tyq - Adds an image to the queue of a typop command.
tybg - changes the background to the supported multimedia content (pictures, video, gif-animation)
tyalpha - with on / off values ‚Äã‚Äãturns the background transparency on and off

Apparently there are other teams, but I have not yet found

Keyboard shortcuts:

Shift + PgUp - scroll up
Shift + PgDn - scroll down
Shift + Insert - Paste from clipboard (Ctrl + V / C)
Shift + Keypad-Plus - Increase font size
Shift + Keyboard-Minus - Reduce Font Size
Shift + Keypad-Multiply - reset the font size (default size is 10)
Ctrl + PgUp - Change the focus to the previous terminal window
Ctrl + PgDn - Change focus to the next terminal window
Ctrl + Shift + T - Creates a new terminal window.
Ctrl + Shift + Home - Move between terminal windows
Ctrl + Shift + PgUp - Splits a terminal horizontally
Ctrl + Shift + PgDn - Splits a terminal vertically
Alt + Home - Command Interpretation Mode
Alt + Return - Insert the first selection
Ctrl + Shift + C - Copy the current selection to the clipboard
Ctrl + Shift + V - Paste the current selection from the clipboard

Mouse control:

Right mouse button - setup menu
Middle mouse button - Paste clipboard selection
Left mouse button = selection
Scroll wheel - scroll up or down
Ctrl + Wheel = Increase / decrease font
