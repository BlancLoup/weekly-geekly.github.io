Day One, May 27


10:00 - The state of CSS / Sergey Popov (Liga A., HTML Academy)
The first report of the day will be about the lost technologies of the frontend, their application and support, so that we begin to use the full power of the current state of CSS.

11:00 - Promotion of open-source projects / Andrey Sitnik (Evil Martians)
The creator of the popular Autoprefixer, PostCSS, Browserlist and Nano ID will tell about their experience. 12:00 - Blameless environment: no one should write quality code / Nikita Sobolev (wemake.services)

13:00 - Cutting the monolith in Leroy Merlin / Pavel Yurkin (Leroy Merlin)
All large companies go through this stage. Stage when the business does not want the old way, and the monolith can not in a new way. And deal with this - simple developers. Switch to the backend and find out about one of the ways to solve this problem.

14:00 - Yandex Database: distributed requests in the clouds / Sergey Puchin (Yandex)
Consider the main points related to the execution of queries in Yandex Database (YDB) - a geo-distributed transactional database that allows you to perform declarative queries on data with low latency and strict consistency.

15:00 werf - our tool for CI / CD in Kubernetes / Dmitry Stolyarov, Timofey Kirillov, Alexey Igrychev (Fant)
Switch to DevOps and talk about the problems and challenges that everyone faces during a deployment in Kubernetes. By analyzing them, the speakers will show possible solutions and demonstrate how it is implemented in werf, an open source tool for DevOps engineers serving CI / CD in Kubernetes.

4:00 pm - 50 millions of deployments a year - The Story of DevOps Culture at Amazon (Tomasz Stachlewski (Amazon Web Services))
After we talk about the role of DevOps culture in the development of Amazon. We learn how and why Amazon moved from monoliths to creating microservices. 17:00 - New Adventures in Front-End, 2019 Edition / Vitaly Friedman (Smashing Magazine)
Let's return to the front end with a powerful report on everything you need to know about the frontend in 2019. Performance, JS, CSS, compilation, fonts, WebAssembly, grids and all.
