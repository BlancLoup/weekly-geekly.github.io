Overview of features and innovations MDM Zinc 4.0

What is Zinc?


Zinc (eng. Zinc, ukr. Zinc) - the chemical element Zn with an atomic mass of 65.38; Zinc is used for galvanizing steel products in order to prevent corrosion. Joke;)
But in our case, Zinc is a development environment, in the form of an add-on over Adobe ¬Æ Flash, designed to integrate flash applications and quickly develop desktop applications for Win, Mac OC and Linux. Zinc converts swf files into executable exe files. With Zinc, you can create your own screensavers, notebooks, FTP-clients, MP3-players. Zinc applications are easily transferred to computers running Windows, Mac OC and Linux.



MDM Zinc 4.0



So glad to introduce! In the arena, Zinc is version 4.0, and as developers of the cross-platform development environment, Simple, Flexible, Powerful and Recycled, as many as four times, the solution for creating desktop applications based on the SWF format from Adobe¬Æ Flash, praise. Zinc 4.0 compiles a SWF file into a powerful desktop application (creates a projector) for OS platforms such as Windows, Mac OSX or Linux, all within the same project.

Of the main advantages of the development environment is the creation of amazing commercial applications, screensavers, widgets, games, autorun CDs, DVDs, kiosks and more, which in turn will be available for both Windows and Mac OSX.
At the moment, Zinc 4.0 is the only X-Platform tool for rapid application development based on Adobe¬Æ Flash projects.

Innovations MDM Zinc 4.0


Ability to export a project to Xcode 4.0
Zinc projects can be easily exported to Xcode 4.0. As you know, Xcode 4, the new version of Apple's powerful integrated development environment for creating applications for Mac OS.

Zinc 4.0 can convert a compiled project into a valid Xcode 4.0 project, which can be further edited and signed for the Mac App Store (see below).

The ability to publish applications on the Mac App Store
The application compiled to Zinc 4.0 can now be converted to Xcode 4.0 projects as mentioned above, which are fully compatible with applications on the Mac App Store, which facilitates the easy distribution of such applications through the Mac App.

In turn, the Mac App Store has completely changed how Mac users can download and purchase software, integrating their Mac App purchases directly with existing Itunes accounts. Thanks to Zinc 4.0, you now have access to millions of potential users by submitting your Zinc 4.0 apps in the Mac App Store.

Zinc 4.0 Desktop Applications with HTML5
For the first time in zinc, implemented the ability to create desktop applications from more than just Flash. Zinc 4.0, easily builds a desktop application for Window and Mac OSX with full HTML5 support. HTML5 provides a wide range of open web technologies, including CSS3, HTML5 markup itself, and JavaScript, as well as libraries and frameworks such as jQuery.
Zinc 4.0 application can be compiled for remote or local use of HTML5 in the project. Using the power of CSS, Ajax, and the core HTML5 APIs, Zinc 4.0 users can create interactive desktop applications from existing and new web projects.

Asynchronous programming options
Zinc 4.0 is armed with the {mdm} Script 4.0 class - the fourth revision and improvement of the Zinc programming API. {mdm} Script 4.0 integrates directly with ActionScript versions 2.0 and 3.0, and includes asynchronous transfer of commands, including FileSystem, Database, Dialog and Network classes.

Asynchronous command execution allows the developer to initiate long cycles of commands without stopping other procedures. It should be noted that {mdm} Script 4.0 provides options for both synchronous and asynchronous commands, which is an unsurpassed flexibility for the developer.

Support for updated Flash Player 11
Zinc 4.0 has been updated and now supports the latest Flash Player 11 (October 2011 release). Support for Flash Player 11 is a serious and profitable update, since the latter is 1000 times faster in performance compared to Flash Player 10.

With full hardware accelerated rendering for 2D and 3D graphics, Flash developers can animate millions of objects with 60 frames per second. Stage3D support will be available in zinc assembly 4.0.1 (at the end of 2011).
Over 150 changes, fixes and bug fixes
All of the code for the new, fourth Zinc has been updated and redesigned to provide faster, more reliable, and more rational use for users and developers. Literally hundreds of updates, some tricks and changes were taken into account to optimize the source code at a more thorough level.
Zinc 4.0 is the most powerful, most sophisticated and promising tool for building X-platform desktop applications with Flash, Flex and HTML5.
