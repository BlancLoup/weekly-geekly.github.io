Procrastination pagoda


The widget is based on the script for viewing videos on Youtube: enter a search query, open the list, select the appropriate video, then watch the suggested related videos. The main task of the widget is to simplify this scenario, get rid of the need to open several tabs and use the ‚Äúback‚Äù button to work with the results of the initial search and view all the interesting objects suggested on the opened pages.

We simply must erect in its honor, if not a temple, then at least a pagoda.

The widget is predestined for endless surfing on objects of the same type: pictures, videos, goods. It is also suitable for a deep search, when there is little quality information on the topic of interest and it is necessary to sort through the mountains of informational garbage in order to select worthwhile resources.

If you click on any thumbnail, the resource opens in the viewing area. Viewed thumbnails are graphically highlighted. If it was a miniature from the top row, the row is shifted down, replacing the bottom row, and the search query forms a new row of objects. Under the main area can be located the history of viewed objects, in the form of a ruler of thumbnails.

In addition to a direct search query, a family of additional queries can be used to select resources, each of which has its own weight. The weight of the algorithm depends on the proportion of resources falling into the output from this algorithm. The weight itself may vary, depending on what resources the user chose to view. As additional requests, requests of similar objects to the resources previously selected by the user are used, and an algorithm that attempts to summarize the themes of the entire set of selectable resources and form a new request based on them.
