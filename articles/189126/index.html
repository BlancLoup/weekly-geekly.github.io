Bitmessage 0.3.5: What's New?



Bitmessage Crypto Messenger continues its active development, the new version brought the following features:
  • Russian language support
  • Fully anonymous decentralized chats
  • A huge number of bug fixes


Bitmessage - an application that allows you to exchange messages between two people or in chat mode, at the request of the user, he can identify himself (sending messages from his address) and not do it - while remaining anonymous.

The peculiarity of this messenger is such that the fact of communication between two users is almost impossible to prove, because the message from user A to B is not directly transmitted, but by sending it to all network participants, of course, only the correct user can read it.

Chats - Chans


Previously, a special address was used as a chat, all messages sent to it were sent to subscribers, the minus of this approach was that when you turn off the computer on which this address is registered, the chat ceased to exist.

Now, chats work on the principle of complete decentralization; to connect to a chat, you must know its address and password, you can send messages to the chat from your personal address, and anonymously , from the general one.

suggestions


The project forum began collecting ideas and suggestions for finalizing the client, the following proposals are now being discussed:
  • Inclusion of Bitmessage Addresses in DNS Binding of Bitmessage Addresses to a DNS Name
  • Bitmessage URL scheme Sending messages and subscribing to chat rooms by clicking on links like bitmessage:
  • Bitmessage Storage - Storing non-textual content on the Bitmessage network (email attachments)
