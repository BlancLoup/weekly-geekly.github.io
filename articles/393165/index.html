AAduino - Arduinka in AA battery form factor



An enthusiast of automated systems and an amateur platform Arduino from Sweden Johan Kanflo [Johan Kanflo] developed and published for free use the AAduino board construction scheme. This is a wireless Arduino clone in an unusual form factor that matches the size of a standard AA battery.

The board is specially designed for use in standard cases for 3 batteries, where it is inserted instead of the third battery. This is a wireless Arduino clone on an ATMega328p chip, with an auxiliary RFM69C chip, two DS18B20 temperature sensors and an indicator LED. The AAduino operates at 8 MHz to save power.

The board uses battery contacts from Keystone, while they are located on it so that the AAduino receives power from other batteries placed in a triple package. The layout, list of used materials, and the health checker are freely available on Github .



To create a miniature AA-format board, the author was inspired by another Arduino microclone, Tiny328 , from another enthusiast, Nathan Chantrell. Nathan, in turn, began miniaturization with the TinyTX Internet of Things sensor , and then decided that the whole arduinka would fit in with such a form factor.

Arduino is a brand of hardware and software for building simple automation systems and robotics for non-professionals. The software consists of a free software shell (IDE) for writing programs, compiling them, and programming hardware. The hardware is a set of assembled printed circuit boards, sold by both official manufacturers and third-party manufacturers. The completely open system architecture allows you to freely copy or supplement the Arduino product line.
