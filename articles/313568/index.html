Here is an example of one of the tasks:

  • develop a simple TCP echo server

Requirements:

  • runs on the IP address 0.0.0.0 and TCP port 2222
  • receives messages no longer than 1024 bytes and sends back to the client
  • closes connection when receiving a message with the text close

With this formulation, the task is simple: open a socket, make a bind, make a accept in a loop, read 1024 bytes each, and if the 'close' line arrives, close the connection.

In the next task we ask to change the server:

  • change your echo server so that it can work simultaneously with 10 clients
  • The transfer protocol is the same as in the previous task.

To solve this problem, you need to process requests in parallel. In this case, difficulties arise even with how to properly close the connection .
