Raspberry Pi Zero went on sale for $ 5: links to stores



The main purpose of creating the organization Raspberry Pi Foundation was to create a cheap small computer. In 2012, the A + model was released for $ 20 - the goal seems to have been achieved. But it turned out that this is not the limit. Today, the Raspberry Pi Foundation has introduced a minimalist model, the Raspberry Pi Zero, which costs only five dollars .

With a 1 GHz processor, the Zero model is also 40% faster than the B model.

Production of Raspberry Pi Zero is carried out at the plant in Wales (UK).

Processor : ARM11, Broadcomm BCM2835 at 1 GHz
Memory : RAM 512 MB
Drive : microSD
GPIO : 26/40 holes
USB connectors : USB On-the-Go (OTG), Micro USB
Video output : composite video via two pins on the board (labeled TV), HDMI via the mini HDMI connector (1080p60)
