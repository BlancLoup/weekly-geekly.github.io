Alignment of forces

  • Language must be predictable . Language is a carrier for expressing human ideas and executing them on a computer, therefore the human understanding of the programâ€™s correctness is critical.
  • Language must be complete . Similar things should be similar, different should be different. Knowing a part of the language should help in learning and understanding the rest.
  • The language must be brief . New languages â€‹â€‹exist to reduce the pattern inherent in old languages. (We all could program on machine codes.) The language should at the same time avoid introducing its own patterns.
  • Language must be reliable . Languages â€‹â€‹- tools for solving problems; the problems that they present in themselves should be minimal. Any strange moments cause embarrassment.
  • Language should be debugged . If something goes wrong, the programmer must fix it , and we need all the help we can get.
