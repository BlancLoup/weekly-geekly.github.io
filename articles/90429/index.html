Realtime xRTML - new html-like markup language


Hello!

It's about a new HTML-like markup language - xRTML , which allows you to edit the functions of the site in real time without using Ajax.

The developers of xRTML are part of the Portuguese IBT team (Internet Business Technologies), who, trying to create a new and accessible markup language that allows you to quickly make changes to the site in real time, came up with xRTML.

The main key features of xRTML are its ease of learning, the need for only one line of code, plug-ins for blogging systems such as WordPress, API for server languages, cross-browsing and cross-platform

How xRTML works


xRTML works with the current site as if on top of it. This is an agnostic server that starts working on a site after web pages are loaded from a web server. All this happens on Realtime servers, so Realtime does not depend on the current IT structure.

Realtime Framework is a set of tools created by developers for developers. The goal of which is to provide a means for developers to start using xRTML on their sites in real time today.
