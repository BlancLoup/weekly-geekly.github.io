Change the theme for Mac OS X Leopard

There is a program that allows you to change the theme in Mac OS X 10.5 "Leopard"! Previously, only users of the previous version of OS 10.4 "Tiger" had this opportunity.
Meet: free utility Magnifique !


I found a program that had already matured to version 2.1, in which many useful things appeared:
  • The ability to download themes from the developer site right in the program window
  • Dock design change (2D / 3D)
  • Preview topics
  • Etc.
