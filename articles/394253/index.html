Files transferred via Skype will be synchronized with all devices of the user.

Resource ghacks.net drew attention to the new function of Skype, which will appear in the next update of the desktop application. Now the files transferred by the user via Skype will be automatically synchronized with all devices that will be logged under this account. Simply put, sending a file to a working chat from a home computer, you can be sure that it will be available from a working laptop and from a mobile device. At the same time, there has not yet been an official announcement of a new feature in the Skype blog, however, the relevant help section has already been updated.

File synchronization will take into account the possible loss of connection: in this case, the transfer will resume automatically as soon as the Internet appears. However, there is a significant limitation on the amount of transferred files - the file size should not exceed 100 MB, and otherwise Microsoft recommends using its OneDrive cloud service.
