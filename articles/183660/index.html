Participants of the ToS; DR project have been analyzing the terms of user agreements on popular sites for exactly a year - and they make a clear note of several proposals, what is the essence of ToS and how it violates our rights. Squeeze contains key advantages and disadvantages of the service. Each ToS is rated. The cleared version of the user agreement is transmitted to the user through an extension in the browser.

Over the past year, 72 user agreements passed legal audits, dozens of others are on the way. The service has already collected ‚Ç¨ 20 thousand donations, extensions for all popular browsers have been released.



If you install the ToS; DR extension in the browser , then when you visit a site for which such reference information is ready, the corresponding icon appears in the address bar. Clicking on it displays help on the screen. Extensions released for Mozilla Firefox, Google Chrome, Opera, Safari and Internet Explorer.

ToS; DR is a non-commercial Open Source project, which started back in 2011, but the actual ToS analysis began in June 2012. The sources of all extensions and other modules lie on Github; there is an API for accessing information on services, each specific item and discussion topic on the forums. If you want to participate in the project - help is welcome .

Project organizers position it as an independent public organization. They receive money through donations and grants. Donations are accepted through Flattr .

Although the main part of the text analysis of ToS is carried out by the free software hacktivist Hugo Roy , but absolutely everything is open, dozens of people take part in the discussion on each item - you can join in and express your arguments.
