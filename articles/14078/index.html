ReactOS has been released

Yesterday a new alpha version of the free and free OS ReactOS 0.3.3 was released, compatible with the applications and drivers of Windows NT and Windows 2000.

In the new release, most of the main system core modules became compatible with NT5, which had a positive effect on stability and compatibility with Win32 applications. Thus, this version of ReactOS has become much more stable than previous releases, so now the OS can be installed not on virtual machines, but on quite real hardware. True, in ReactOS 0.3.3 there is still no support for USB, SATA and NTFS FS.
