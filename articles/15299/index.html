Outpost Firewall Pro 2008 Released


The new release of the popular firewall Outpost Firewall Pro. Among the many improvements in version 2008, the following are highlighted: full support for the Windows Vista operating system (32 and 64 bits); IPv6 support; local security improvements; advanced monitoring of critical system objects; enhanced protection from harmful sites; auto-learning mode; support fast user switching; improved interface; improved program performance.

Judging by polls, this is the most popular firewall (a program that protects against attacks from the Internet and local network) of all those present on the market. The huge popularity of the program is explained by a simple and intuitive interface, which novice users are especially pleased with. With regards to the quality of performance of direct duties, here, too, everything is in order. If you have sufficient qualifications - manual configuration, in the absence - automatic, which is fully functional. It is worth noting that there is support for modules that extend the functionality of the program. Included with the distribution there are several highly sought-after (for example, ad blocker and anti-spyware).

List of changes:

- Improved Local Security. The loading of unauthorized drivers is blocked and malicious code attempts to gain full access to the system, robust and reliable protection against unknown malicious programs trying to bypass the defense of your computer are prevented.
- Limited access to potentially malicious websites. The new web control component allows you to block potentially malicious sites, protecting users' systems from disclosing personal data and infections by hiding files. Filtering is based on the reputation of sites and occurs in the background, regardless of whether the Anti-Spyware is enabled or not. The list of blocked sites can be edited manually; It is also updated via automatic program update.
- Advanced monitoring of critical system objects. This feature allows more advanced users to assign allowed actions for subsequent control over vulnerable system objects and inappropriate program behavior.
- Full support for Vista and 64-bit platforms. Outpost‚Äôs completely redesigned architecture ensures that the product is now compatible with all versions of Windows since 2000, including Vista and all 64-bit versions of Windows.
- IPv6 support. Data transmitted over the new network connection protocol, IPv6, widely used in Windows Vista, is fully recognized, filtered and protected. The protocol is a development of its predecessor, IPv4, and is more scalable, and also presents a number of improvements in automation and security.
- Full support for user switching (Fast User Switching) and low privileged accounts. Outpost is now compatible with multi-user Windows environments and supports user switching and Remote Desktop connections. Under Vista, the product fully supports User Account Control security mechanisms.
- Effective web control. Extended ad blocking and protection against malicious sites ensure compatibility with even the most complex and sophisticated websites and provide even higher level of online security.
- Improved interface and ease of use. A streamlined Vista-style interface provides faster access to key program features.
- Auto-learning mode. When you first start, Outpost can start working in a special "learning mode" with the suppression of all notifications. While working in this mode, the program remembers the typical activity of user programs and connections monitored by the firewall. After the end of the training mode, the product returns to normal mode, but only requests action for new activity, significantly reducing the number of requests that require user response.
- A more intuitive scanning process for spyware. The scanner displays detailed information about the current process of scanning and removing malware, allowing the user to view the types of detected and deleted malicious objects in real time.
- High performance and low resource consumption. The computers protected by Outpost now load faster, work faster, scan faster and use less system resources, providing the most transparent automatic protection.
