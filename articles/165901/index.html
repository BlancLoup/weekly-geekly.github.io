Kiev - a free seminar "Proliant Gen8: new opportunities and technologies"



Target audience :
- specialists working with HP servers to learn about the new features of the Proliant Gen8 servers
- experienced users and beginners who want to choose a further direction of training in the use of servers.
Schedule :

1. Introduction (10.00-10.20).
Meet the listeners.
Schedule, form of training.
Definition of seminar objectives.

2. New features of Proliant Gen8 (10.20-11.20).
Overview of the school day.
Models and product lines.
CPU and memory.
Data storage.
Network interfaces
Software products.

Break (11.20-11.40)

3. Server management interface (iLO & OA) (11.40-12.40).
New iLO4 management interface.
Licensing policy.
Monitoring by the built-in agent (Agentless monitoring).
Additional collection of information (AHS).
Scripts and integration with mobile devices.
Basics of using the Onboard Administrator.
Video demo: AHS.

Lunch (12.40-13.40)

4. The initial boot order (ROM) (13.40-15.10).
New boot options.
Features "early video."
The database of error messages.
RBSU configuration utilities.
Video demo: Bootstrap, RBSU, ORCA, ACU.

Break (10.15-15.30)

5. Preparing to install the operating system (Intelligent Provisioning) (15.30-16.00).
Optimize initial configuration and installation of the operating system.
Drivers and control RAID controller.
Access to management interfaces (iLO) and control logs (AHS)
Package update (Intelligent Provisioning).
Video demo: Upgrading the Intelligent Provisioning Package.

6. Optimization of data center management (Intelligent Infrustructure) (16.00-16.30).
The main tasks: collecting information and using the collected data.
Installation of equipment: cabinets.
Power supply systems (PDU).
Monitoring equipment location.
Integration with Insight Control.

Break (16.30-16.50)

7. Automation of management (SPP, SIM, RDP) (16.50-17.40).
Overview of software products.
Service Pack for Proliant: integration of server support products.
Using SPP: firmware and driver updates, repositories.
Insight Control: corporate infrastructure management.
SIM as the core of the monitoring system.
Use RDP for automated installation of the operating system.
Video demonstration: Installing the operating system, using SPP.

8. Completion (17.40-18.00)
Questions and answers. Wishes
