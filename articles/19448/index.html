Zend Framework 1.5.0 Preview

Released almost release Zend Framework 1.5.0.

- New Zend_Form component with AJAX support for elements.
- New helpers for AJAX requests and alternative data upload formats.
- Helpers for Infocard, OpenID and LDAP.
- Support for complex types of searches in Lucene (fuzzy, date-range and wildcard).
- Support for Lucene 2.1 index file format.
- Helpers Partial, Placeholder, Action and Header for View.
- Zend_Layout.
- Support UTF-8 for PDF.
- New web services (Technorati, SlideShare and Remember the Milk).
