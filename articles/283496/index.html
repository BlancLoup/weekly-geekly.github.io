In the Opera browser appeared power saving mode


Battery life (hh: mm). Tests conducted by Opera on laptops Lenovo X250, Core i7-5600U, 16 GB of RAM and Dell XPS 13, 16 GB of RAM, under Windows 10 64bit in high performance mode

A new version of the Opera browser has been released on the dev-channel , in which the energy-saving mode is implemented. This is the first of the popular browsers with such a function (or the second, if you count Safari). Depending on the configuration of the computer and other factors, the browser prolongs the operating time of the laptop when browsing the web by several tens of percent.

Power saving mode must be enabled manually. When the laptop is disconnected from the power outlet, a battery icon appears near the address bar. Click the icon and move the slider.



The browser activates the power saving mode if the battery's power drops below 20%.

In this mode, CPU usage is significantly reduced due to several optimizations:

  • Reduced activity in background tabs
  • Better scheduling of JavaScript timers, which reduces the frequency of requests to the CPU
  • Auto pause unused plugins
  • Reduce frame rate to 30 frames / s
  • Adjust video playback settings, force use of hardware accelerated codecs
  • Stop animation in browser themes




To develop an energy-saving mode, tests were designed that simulate a realistic mode of using the browser. Optimization was tested in the telemetry framework - a great tool that shows the exact number of milliwatts used by the CPU.

The actual power consumption of a laptop is influenced by many other factors, including the brightness of the backlight, the distance to the WiFi access point, the presence of antivirus software, system updates, the number of running applications and much more.

In addition to the energy-saving mode, in the new Opera build, the VPN mode has been improved. By popular demand, the function of selectively enabling VPN for individual sites has been implemented.



Fixed a leak with a real IP address when establishing a WebRTC connection in VPN mode.
