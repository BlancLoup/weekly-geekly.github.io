Top 5 Things I Learned from IBM Software for System z for Dummies

Good afternoon, dear habra users!

Under the cut is a translation of the Mark Scapicchio note from the IBM System z blog, in which he shares his opinion on the book IBM Software for System Z for Dummies .



During the first few weeks after the release, the IBM Software for System z for Dummies free e-book was one of the most discussed topics among readers of our IBM Software blog and Twitter . As a result, it became the most popular theme of October .

A few days ago I was going to see what caused this noise, and I read the book IBM Software for System z for Dummies from cover to cover. It turned out that the 72 pages of the book contain detailed information about the available software for mainframes, which, on the one hand, is set out in simple language, and on the other hand, touches on issues that are not at all trivial for IT professionals.

I chose (which was not easy) 5 basic things that I learned from this book:

  • When in 1964 IBM unveiled the original System / 360 mainframe and promised customers that any code written for this machine would work unchanged on the following mainframes. This promise holds today.
  • The new zEnterprise EC12 can perform up to 75,000 MIPS (millions of instructions per second) and support more than 13,000 virtual machines on a server and more than 100,000 virtual machines in a cluster.
  • The introduction of cloud computing based on System z increases the speed of deploying applications and services on the x86 architecture up to 10 times, which is one of the key indicators for many customers.
  • New transaction accelerators IBM DB2 Analytics Accelerator for z / OS V3 , combining technology Netezza and System z, can accelerate analytical queries up to 2000 times. Yes, that's right, up to 2000 times. Clients report that if previously it took from 25 seconds to 90 minutes to process a single request, then this process takes only 8 seconds or even less with Accelerator DB2 Analytics .
