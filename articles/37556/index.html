Opera 9.52 Finale

Today announced the final release of Opera 9.52.

In addition to bug fixes and improved stability, the new version contains fixes to improve security, and therefore, developers recommend upgrading to all users.

In the list of changes:

- added some improvements in the design of the browser, the appearance is slightly changed;
- Fixed some problems when viewing the history of visits;
- corrections when working with encodings in the address bar;
- improved interaction with some web services (Gmail, aim.com, online.no);
- fixed small memory leaks when downloading BitTorrent files;
- added the option "Close tab" in the File menu;
- improved overall performance (viewing flash-videos has become much more stable);
Corrections in work with mail and news:
- fixed problems viewing mail if Opera is installed in the directory containing the # symbol in the title;
- restored the correct operation of the option "Mark all as read";
- fixed problems with updating signatures when switching user accounts (also sent bugreport about this).
