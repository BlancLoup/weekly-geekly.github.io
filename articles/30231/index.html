Inspirers


The basis of this idea was: evernote, for a fresh approach to storing content; todoist, for a convenient tree-like way of storing tasks, where each element can become a container for the same elements; oddly enough, outlook, for the ability to store everything in one place, and a bunch of sites and services about GTD orientation.

What happened?



The concept of a personal information manager was born, with a tree structure, but without a hard division by type of information.

At the first stage, 5 different types (text, image, bookmark, task, event) can be added to the tree and form any subordination structures.
