HP Software Application Performance Monitoring

When it comes to monitoring application performance, the following components of the HP solution should be highlighted:
- HP Real User Monitoring (HP RUM) ‚Äîmonitor control of real users transactions

HP RUM and HP BPM make it possible to assess application availability from the end user's point of view.

HP RUM parses network traffic, revealing real user transactions. This makes it possible to track user activity, the processing time of various transactions, to determine the relationship between user activity and business metrics.

HP BPM is an active monitoring tool, HP BPM performs synthetic user transactions that are indistinguishable from real ones for monitored systems.
