Django 1.8 released

According to the official blog Django, April 1, 2015 released a new version of Django - 1.8. This version will have long-term support (Long-Term Support) for at least another three years.

Of the interesting innovations -
  • native support for various template engines (API defined and Jinja2 built-in support added);
  • support for complex SQL expressions in ORM, including if - then - else expressions;
  • API is defined for Model._meta;
  • A new PostgreSQL-specific functionality has been added to contrib.postgres.

For release details, see release notes .

This release ends support for Django 1.6.
For Django 1.7, they will release security updates until the release of Django 1.9, scheduled for October 2015.
The previous long-term version, Django 1.4, will receive security updates before October 1, 2015.
