Unity lens for video search VKontakte



In unity there are very convenient lenses, there are a lot of videos in VKontakte, and flash in the new ubuntu turns all movies into an avatar. Therefore, I made a lens to search for videos in VK, opening them in totem.

The package can be delivered from ppa: 
add-apt-repository ppa:nvbn-rm/ppa apt-get update apt-get install vkvideo

After installation, you need to restart unity: 
 unity --replace

And pass authorization through the mechanism proposed in the lens.

The application is written in python, its source code is very simple and is available on github .

The lens was tested only at 12.04.
