Alternatives to parse.com

As you know, next January , the Parse service will be closed . Most developers are looking for an alternative to this service. In one of the guthub projects, there were quite a few alternatives, with links and descriptions. Before you is a translation of this page.

Part 1: universal services


Providers providing real-time information are marked with (*).

AnyPresence MBaaS (Mobile backend as a service). Simplifies the process of creating enterprise mobile applications running on different devices without compromising the flexibility of the tools available to the developer. Easy to use even for inexperienced users. Experienced developers get access to the source code of all components.

ApiOmat allows enterprise processes to work as mobile applications. Mobilizes complex business structures and processes by combining existing IT systems with mobile applications and devices. It can be installed directly in the data center of the company, which will allow employees to perform their work through applications regardless of their location.

Appcelerator Arrow (*) Consists of two main components, Builder and Cloud. Arrow Builder allows you to quickly create APIs, data models, and data connectors through a wizard or by writing code. Arrow Cloud is an elastic cloud service where APIs are executed and their usage is analyzed, plus two dozen of related services are provided (geo, photo storage, etc.) using ArrowDB

Appery.io Integrated MBaaS services - so that developers can concentrate on applications without being distracted by infrastructure.

Appmobi Customizable encryption, authentication and data storage in encrypted form, includes real-time updates, push services, and analytics.

Apps Panel (*) MBaaS from an independent software developer. It allows you to manage applications for the Internet of Things (content, marketing, analytics), provides a sophisticated web interface and API for interfacing with corporate information systems.

Appzio (*) offers a REST-style backend, push notifications, SMS sending. Content management system allows you to edit the logic and change the content through a web interface. Integration with Facebook, Google Analytics, Flurry and other APIs like REST. Offered code for iOS and Android

AppWooDoo (*) offers a simple and powerful real-time database and push notifications. Compatible with iOS and Android.

Apstrata is a powerful backend that works with different APIs: orchestration (JavaScript server-side scripting), No-SQL, sending messages (push notifications, email), integration with social networks (Facebook, Twitter, Google+, etc.), user management. Fast integration of business logic with REST API. Easy to use, plenty of documentation, support forums.

AWS Mobile Hub makes it easy to add and customize mobile application capabilities, including user authentication, data storage, backend logic, push notifications, content delivery and analytics. Easy access to testing on real devices and tracking your application usage.

Backand (*) powerful BaaS for AngularJS. Log in via social networks, push notifications, integration with Ionic and much more.

Backbeam (*) offers many services designed for web and mobile applications. Helps you work both during the development of the application and at the end of development.

Backendless is a system of connected cloud services with a REST API. User management, data storage, geolocation, streaming media content, subscription and distribution of messages, push notifications, custom business logic, analytics, code generation for mobile applications.

Buddy is a small, fast and flexible platform for collecting and processing data from the Internet of things. Offers a set of APIs specifically designed for mobile development - users, check-ins, push notifications and a metric.

Built.io (*) Winner of various contests, mBaaS, allows companies to create applications without having to develop a design and build a backend on their own. Built.io already has an industrial-level backend supporting applications for the web, mobile devices and the Internet of things. Applications can be created in days or weeks, not months.

CloudBoost.io (*) mBaaS for developing real-time applications, with the ability to simply enable search capabilities. This is something like Parse + Firebase + Algolia in one. Open platform, available on the docker resource and can be installed on any server or in the cloud. Can work as a managed service.

CloudKit is a framework that replaces such backend functions as databases, file storage, and user authentication. Provides a set of APIs for accessing iCloud. Allows you to create users associated with accounts in iCloud. Developers get access to the global database, the ability to save files to iCloud Drive, which users can later download using their devices. The work is organized as if everything happens to files stored locally.

CloudMine Allows you to instantly include push notifications, geolocation, social networks, SMS and fitness data in the application - all this is already available in finished form. Available on all platforms - iOS, Android, Microsoft, for the mobile web (HTML5), and cross-platform toolkits for creating UI.

Cosmic JS is a cloud-based content platform that makes it easy to add dynamic content to any website or application. Uses the API-first approach, allowing content to be available on any website or in an application and on any devices, regardless of programming language. This speeds up development cycles, content management and gives you more time to create great products.

FatFractal allows you to work with a schematic-free collection and creation of objects, and on this basis builds such functionality as local development, advanced query language, bidirectional search by column, declarative access rights, push notifications, code with triggers on events, end and on schedule, pluggable authentication, and more. Native SDK for iOS, Android / Java and HTML5 / JavaScript.

Firebase (*) Storing and synchronizing data with the Firebase NoSQL cloud database. Data is stored in JSON format, synchronized with clients in real time and available even when going offline. User authentication is available by email and password, via Facebook, Twitter, GitHub, Google, or integration with an existing authentication system.

Flybase (*) Clustered NoSQL allows you to store and synchronize data. They are stored in JSON format, accessible using a language similar to mongo, and synchronized with all clients in real time.

Kii Mobile backend allows you to create scalable and fast mobile applications for different OSes. Development time thanks to built-in tools is reduced from months to days.

Kinvey Backend as a service (BaaS) is very similar to Parse. Cloud storage, push notifications, custom business logic, a ready-made implementation of the Facebook Open Graph for applications that do not have websites.

Kumulos Service for mobile application development teams and agencies. Allows you to manage all clients and their applications from a single control panel. There are features for each stage of application development. In addition to SQL storage, there is a Hookup for connecting with other MySQL databases, KScript (for server-side Javascript), RESTful API, and client SDKs for iOS, Android, Windows, PHP.

Microsoft Azure Mobile App Service Service is similar to Parse in terms of data synchronization, user authentication, push notifications and cloud code. Allows you to quickly create cross-platform and native applications for iOS, Android, Windows and Mac.

MoBack (*) backend specializing in push notifications, data storage, REST API, analytics, iOS, user management, .Net, SDK, Android, Unity, Javascript and Custom.

PubNub (*) is a fast messaging system between any devices and platforms. Simple APIs and SDKs help you create and scale real-time applications on the PubNub Data Stream Network. You will need to focus on product development instead of creating and managing a system that serves data streams.

Reindex BaaS with GraphQL API. Data storage, authentication, access control, hooks for integration with third-party services and open source SDK.

Restdb.io BaaS, specializing in a powerful interface for creating database models and providing a JSON REST API.

ShepHertz App42 "multichannel" platform (mobile applications, social networks, games, TV, Internet of things, wearable gadgets, etc.) for creating applications and games. It operates in 9 countries, handles over 25 billion API calls and is used by thousands of independent developers, studios and enterprises. More than 25 mBaaS modules, push notifications, storage in the NoSQL database, cloud code, user management, file upload, analytics, backend API for games, etc.

Syncano's customizable BaaS reduces the need for backend code and intermediaries, so that developers can focus on user experience.

Telerik backend focusing on security and data encryption. As your business grows, you can be confident that this service scales appropriately to fit your needs.

ZetaPush (*) BaaS real-time, offering built-in authentication and other services: noSQL database, file storage, search, mail, SMS. There are SDK for mobile, web and internet of things.
