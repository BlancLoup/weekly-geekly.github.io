Google Documents List Data API released

Google has released a new API called Google Documents List Data API . In fact, the name is misleading, since the Google Documents List is only a file browser available on docs.google.com . Now there are stored text documents and spreadsheets, and soon there will be presentations (support for presentations should appear this month, it is possible that next Monday - provided that Google keeps to its schedule).

This new API can be used to upload documents to the server, as well as to get a list of available documents. The API also allows you to search through documents to retrieve individual documents. Google introduced code samples for Java and Python - a PHP sample was not presented.

Using this API, you can, for example, backup all your documents located on the local computer. You can also make a site with a different interface to access documents located on docs.google.com. In principle, you can make some mashup - it all depends on the imagination of the web developer.
