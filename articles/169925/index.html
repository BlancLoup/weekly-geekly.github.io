Online broadcast of Windows Azure Summit on AzureSummit.ru

Did you manage to register for the Windows Azure Summit conference on February 20? The event turned out to be so popular that we had to close the registration shortly after the announcement. If you do not have time, do not worry! We invite you to join the online broadcast of the event on the AzureSummit.ru website. The broadcast will begin on February 20 at 10:00 (Moscow time).

Recall that the key topics of the conference are:

  • Windows Azure as a development platform
  • latest platform updates, basic Windows Azure services
  • Scenarios for using Windows Azure
  • single backend for mobile solutions, transcoding and streaming video, a backend for gaming applications, creating extensions for SharePoint Online and SharePoint 2013, integration with corporate systems.
  • Business in the cloud
  • approaches to determining the cost of SaaS services and their pricing, methods and channels of promotion, compliance with the requirements of the law on personal data.
  • Cloud and Web
  • web development (HTML5, jQuery, SignalR, ASP.NET MVC 4) and hosting web projects in Windows Azure infrastructure

The conference is focused on:
  • IT executives who are already using cloud technologies for their tasks or just planning such a transition or integration;
  • Developers who want to learn Windows Azure and expand their knowledge in the technologies of creating applications, cloud services and WEB-projects;
  • Architects designing reliable cloud or hybrid solutions;
  • Directors and managers interested in innovative ways of business development
