iPhone client for Google Tasks

The extremely simple and minimalistic Google Tasks service is popular among those who do not need more (and sometimes interfering) functionality. It is conveniently integrated with Gmail and is very convenient to use it from the browser. But, to admit, the mobile interface is not so convenient and does not cause the feeling that this is something close and dear.

Introducing the iPhone client for Google Tasks.

Key features:
  • Google and Google Apps account support
  • Work in the absence of access to the Internet
  • Editing tasks without leaving the list
  • Changing the position of tasks in the hierarchy (drag for changing places, swipe for changing indent)
  • Secure data exchange with Google via an encrypted HTTPS channel
  • Ability to customize the view of the task list
  • Ability to select actions performed by single / double clicking on the task

The application is absolutely free, no hidden ads.
Download from the App Store.

UPD. Under version 3.1 poorly tested, there were some very annoying bugs. I promise to fix it.
