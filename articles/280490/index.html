Google Analytics in Telegram


Using Telegram? And even do it for work? Or maybe you are using Google Analytics for work? Then @MetricsBot is for you.


The simple Telegram bot I made, after authorization, allows you to show statistics from a project of your choice right in the Telegram chat. This is especially convenient if you add a bot to the chat with your work colleagues - in this case, someone alone requests the data, and all the others see the result, which can be immediately discussed.

Currently the bot supports the following commands:
/ users [today, yesterday, this week, last week, this month, last month, this year]
/ sessions [today, yesterday, this week, last week, this month, last month, this year]

It's funny that a bot can show its own data, since sends its usage statistics to Google Analytics. I think, is it worth it to screw the demo mode (so that instead of authorizing and choosing your project, you could watch the data of any demo project - for example, mine).
