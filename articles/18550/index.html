SW Counter - new counter

So, the time has come to talk about the new project SkyWander, which is almost ready for release (although it is better to call it open testing).
SW Counter is an invisible (hidden) counter.
The counter provides all the basic functions available in similar counters, but there is one more interesting feature - Views.

Representation
The function calls to make convenient samples from the statistics log by the URL of the page, so you can find out not only the statistics on the entire site, but also on its individual components, pages.

In the near future the meter will be available for public use. (for now, only closed testing), then the designers will make it all beautiful and more usable. Then we will add an API to display statistics on other sites or for any other needs of users, well, few people need anything =)
At the beginning of open testing, the meter will be available 100% free, but most likely with some limitations ...
But it will be later, but for now look at the screenshots (clickable), most of the links on the statistics pages are clickable and lead to a full log with the grouping of the selected data.
