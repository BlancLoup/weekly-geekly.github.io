Android 2.3


  • Improvements for programming gaming applications (GC, support for gyroscope, other sensors, sound API Khronos OpenSL ES, updated video drivers Open GL ES)
  • Open Media Standards Support (VP8 / WebM)
  • ACC / AMR support, sound effects and equalizer
  • Front camera support, very tight voip (sip) integration, but the SIP stack is configured by the device manufacturer
  • NFC support (we lean / just hold the phone via the NFC sensor, a window pops up with information - URL, text, contacts, etc. Almost like QR, only without starting the camera, etc.)
  • Improved energy saving (If some application sits for a long time in the background-e and loads the CPU, it will be closed if necessary. The display of who uses the battery is also improved)
  • Improved copy / paste (single-word selection, GUI)
  • Improved keyboard. Shift + letter = will give a number.
  • Download managment - management of downloaded files from the browser, email and other applications.
  • Refreshed and accelerated general GUI OS
  • Linux Kernel 2.6.35
