The digest of interesting news and materials from the world of PHP No. 48 (September 7 - 21, 2014)



We offer to your attention another selection with links to news and materials.

Enjoy reading!


News and Releases


  • New PhpStorm 8: developing together. - Supports Blade, Behat, WordPress, remote PHP interpreters, multi-allocations and more.
  • Laravel removed Issues on GitHub - From now on, only pull-requests are accepted to the repository, but not bug reports. At the same time, the development team of the framework introduced a special tool Liferaft , which automates the routine operations necessary to create the correct bug reports.
  • Updates on current PHP 5.4.33 and 5.5.17 branches


Php


  • RFC: Null Coalesce Operator - Has an early proposal for an implicit isset call in a ternary operator been transformed into a proposal to introduce a new operator ?? which will return the first operand if it exists and is not NULL , otherwise the second: $_GET['mykey'] ?? ""; $_GET['mykey'] ?? ""; Voting has already started.
  • RFC: loop + or control structure - Offer to add a block or for all existing loop constructs in PHP, which would be executed if the loop condition is false already at the first iteration.
  • RFC: Fix list () behavior inconsistency - A suggestion is to correct the behavior of list() when used with strings.
  • RFC: Remove alternative PHP tags - It is proposed to remove support for the <% and <script language=php> tags.
  • RFC: Switch default multiple - The proposal to ban the use of several default blocks in the switch construction was adopted unanimously.


Instruments


  • PHP-Parser 1.0.0 - The first stable release of the PHP parser for PHP. Versions> = 5.3 are supported including 5.6.
  • Rocketeer 2.0 - A quality tool for launching tasks and deployments updated to version two.
  • HTTP Mock - As the name implies, it allows you to get HTTP requests in your tests.
  • Minime \ Annotations - Simple work with annotations.
  • ClassPreloader - The tool allows you to optimize autoload by generating a single file with all classes.
  • PHP Cron Expression Parser - Allows you to parse CRON expressions, determine the date of the last and next launch.
  • Recoil - Multitasking with Corutin - React core, which implements generator-based asynchrony.
  • PHP Shell Framework - Framework for creating console applications.
  • Shell php module - An extension written in Zephir, and also intended for creating console programs.
  • Requests - A simple and convenient library for making HTTP requests.
  • humhub - Flexible social network on Yii 1.
  • epilog - Paint logs in the console.
  • Geocoder - An indispensable library for creating applications that process geo-data.
