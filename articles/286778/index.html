Microsoft is not going to kill Internet Explorer

Microsoft is currently working on a new browser, codenamed ‚ÄúProject Spartan‚Äù. This browser will work in Windows 10, the new operating system of the company, and many users of previous versions of the OS are interested in what will happen with Internet Explorer when a new version of the browser is released. Some media voiced the view that Microsoft is going to "kill" the brand Internet Explorer, in fact, it is not.

The developers of Windows 10 have confirmed that Internet Explorer will also be included in the distribution of the new OS. But now IE will be on the sidelines after Project Spartan. In turn, the new browser after the release will receive a different name.



A Microsoft spokesman told Business Insider in a conversation with journalists that "Project Spartan is a new generation browser created exclusively by Windows 10." At the same time, work with IE will continue.

Apparently, Microsoft will gradually move away from the IE brand, which for many users causes negative associations. At the same time, the corporation will attract the attention of users to the new project.

Project Spartan was first announced simultaneously with Windows 10. The browser has a new design and engine, also has integration with Cortana. The browser allows the user to work more efficiently with web pages through this integration: Cortana will be able to search for phone numbers, addresses and other useful information on the page.

Windows 10 with Project Spartan browser will be released at the end of the astronomical summer of 2015 (September 23) and will be released in 190 countries in 111 languages.
