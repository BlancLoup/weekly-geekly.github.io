Free translation close to the original .

New build.


Hello!
As you have already noticed, we are correcting tons of errors, so now we are posting a new build for you guys. Testing with!
It is noteworthy that we have something new for you - a modified page scale control (for win only, but soon will be for other platforms) :)

Do not forget to inform us about errors found through the bug-wizard . Look for regressions from version 10.10 - now it is of the highest priority for us.

Noticeable changes

  • Facebook is working again
  • A new page view control button has been added to the status bar, a scale control slider has been added (currently only for Windows)
  • Worked on tab management (MDI)
  • Correction themes, such as shine in the personal panel to make the text readable (Windows 7 / Vista)
  • Improved stability
ATTENTION: This version is under development: it contains the most recent changes, but other than that it may have critical problems, including crashes and data loss situations. In fact, it may not work at all.

Download

  • Windows MSI / Windows Classic
  • Mac (Intel) OS 10.5 or higher
  • Unix
