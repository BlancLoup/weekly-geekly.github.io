Pebble 2.0 and Pebble AppStore release

As the users were promised , during the presentation, a new version of the Pebble program was released. The current version of the branch 2.0, in addition to some updates, also supports the long-awaited novelty - Pebble AppStore


The developers say that there are already more than 1000 applications and hours. We'll see!


Let me remind you, a month ago, the developers introduced a new version of the clock (only the appearance has changed, only more memory has become made of iron) and told about the AppStore.

But the app store has earned just now.
iOS version updated last night. The application has changed the icon, and the interface has changed very seriously. After launching the updated application, it downloaded the new firmware, and updated the clock:


Then I checked all applications for compatibility with the new firmware. Although, to tell the truth, applications are not checked for any compatibility, they are all incompatible with the new firmware. The availability of the version of this application for the second firmware branch is checked, and if it is, then a new version is installed.


After that, the user is shown a short guide to the new functionality. The developers did not increase the number of available applications on the clock (although the memory in the clock with the new revision and in pebble steel allows), but made a virtual area called the app locker. You can drag-and-drop from the watch those applications that you sometimes need, but for storing which there is not enough space in the watch. Like, you need an application for golf once a month - drag it into the hours before the game, and then remove it back. For me, it would be enough just a list of "favorites" in the AppStore


Categories

Now in the Pebble AppStore in the Apps section there are 6 categories: Everyday (Daily), Utilities (Tools and Utilites), Notifications (Notifications), Remote Management (Remotes), Fitness (Fitness), Games (Games)


The Daily section contains applications that are used in everyday life: social networks, calendars, exchange rates, maps, schedules, news, weather, and so on:


From the Tools and Utilites you can download more specialized applications - level, calculators, timers, stopwatches, notepad (!) And the like:


What is in the category of notification, I think it is clear from the title. These are a variety of applications that report things. For example, about taking pills, about meetings from the calendar, about notifications from the phone.


Remotes contains an application that controls something. For example, Hue bulbs. Do not ask me the name of this application.
You can also control the camera on your phone, GoPro, music, torrent client (why?), XMBC (that's cool), smart home, voice recorder and a whole lot more.


Fitness - a section for applications related to sports lifestyle. Clients of fitness phone trackers, an application that every 10 minutes reminds you to do exercises, to score points in tennis, to choose a pose in sex (yes, that very thing ).


Well, Games, they are Games. Toys Labyrinths, tetris, rpg (!), Pseudo-3d rpg (!!), arkanoid, sudoku, chess, and a lot of other games. Interestingly, is there any rule like ‚Äúrule 65 - are there any games for any device‚Äù?


What's new in firmware 2.0?

  • Accelerometer Access
    Developers can make applications that you can interact with through gestures.
  • Save clock and application settings
    Previously, when exiting the application, the memory was cleared completely. Now you can save the data for use the next time you start. For example, make a game with saving levels.
  • Applications have access to battery charge data. Now you can make indicators in the clock, which will show the charge level of the clock battery.
  • Improve job stability and crash handling applications. Now the falling application does not reset the clock.


Where to download the new application?

You can download the new application now from the AppStore or from the Pebble for Developers Forum for Android (it is still in beta, so it is not yet available on Google Play).

Russian language support?

The official firmware is still not there. But on pebblebits.com/firmware , the latest version firmware (2.0.0 in the list) is already available, both with support for Russian characters in the font, and with full interface translation.
