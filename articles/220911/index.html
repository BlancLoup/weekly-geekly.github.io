Consoles games: like falling in love with video games turned into an interesting game dev

Hi, Habr!
Today we decided to share a few non-format stories for our blog. It will be about one developer, whose love for console video games in childhood gave rise to a rather unusual game project.



Christer Kaitila or McFunkypants, as his friends like to call, grew up at a time when the Atari and Tron prefixes prevailed. And, of course, the craze for video games has not bypassed him. Now, as a game developer with experience, he decided to pay tribute to the good old video games and game dev community ... Not so long ago, Chris introduced the open-source tools for creating games in the tower defense genre. ‚Äú Someone like me, but I always wanted to create video games. This desire came to me since the era of the Atari 2600. Era, in which I was lucky to be a child. Tower game starter kit


At the end of last year, Chris opened a new page in his developer biography. Working with Woot Studio, he released a rather non-standard project - Tower Game Starter Kit . It is a free cross-platform open source developer toolkit for creating your own tower defense-style games for Windows Phone, Windows 8, and HTML5-enabled browsers.



‚Äú We wanted to create something simple that could work equally well on tablets and smartphones, and most importantly, that it did not require high performance for rendering. ‚Äú When creating the Tower Game Starter Kit, we agreed with Woot Studio that it would be nice if the sample project was included, that is, MVP (product with minimal functionality) representing a fully functioning and certified project under Visual Studio, ready for compilation and placement in the App Store. "
