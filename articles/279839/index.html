A little about RetroShare


From Wikipedia:

RetroShare is a free cross-platform software for serverless email, instant messaging and files using an encrypted F2F (and optionally P2P) network built on GPG. It is not a darknet in the strict sense of the word, as peers can exchange certificates and IP addresses with their friends.


Strictly speaking, in the strict sense of the strict word, here it is, finally, the crown of anonymous communication in the world of distributed OpenSource! Thank you, oh mighty open source developers.
