MapFactor GPS Navigation - a new application for navigating maps Openstreetmap


In my first post for an invite, I made a comparison between programs for navigation using data from Openstreetmap. Not so long ago, a new application for working with OSM maps appeared in the Windows Phone application store - MapFactor GPS Navigation


As I understand it, this application has long existed on the Android platform, but I was not familiar with it.

Evaluation criteria are the same:
  1. Discretion and usability
  2. Map drawing speed
  3. Offline map features
  4. Route planning
  5. Design
  6. Subjective personal opinion

Discretion and usability - 4.5


When you open the program on the first screen there is a simple finger-oriented menu.

Unfortunately, the buttons on it cannot be dragged in places, and their arrangement is very strange, as I think. If these were tiles that you can drag, it would be great at all.
