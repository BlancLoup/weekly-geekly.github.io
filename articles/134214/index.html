Yandex - now search by default

Hello


Happy owners of smartphones with the Windows Phone operating system, I hasten to please! Today I connected my HTC Mozart to a laptop and Zune suggested upgrading from HTC. The fact that users of the Russified devices Yandex will soon become the default search has been written for a long time and in various blogs. So, rebooting the phone, I first pressed the hardware search button and was pleasantly surprised that it was not the Bing page that opened, but the Yandex page with the search query line, mail, news, train schedule, billboard, TV program and recommended applications from Windows for Windows Phone

Now you can confidently use all three hardware buttons in Windows Phone.
