A set of tools for building graphs, flowcharts and diagrams



There are many different services that make life easier for a web developer or designer, including creating different graphs, charts, flowcharts, etc. Below is a small selection.
(Beware, lots of images)

All pictures are clickable and lead to the appropriate resource.

Services


Amcharts



Amcharts is a set of flash-based charts for websites. It can extract data from CSV or XML files, it can also receive data generated dynamically using PHP, .NET, Java, Ruby on Rails, Perl, etc.

Anychart



Flash based visualization solution. There are not only web, but also desktop applications.

Autodesk



Offers tools for drawing electronic circuit diagrams, diagrams, various kinds of UI. Does not require any additional plug-ins - only a browser that works with JavaScript is required, is cross-browser. Supports export to PDF, JPG, PNG, and SVG.

Barchart



Used to build bar charts, in other words histograms.

Best4c



Best for Charts. Used to build various kinds of diagrams and diagrams.

Bubbl



Creates multi-colored schemes.

ChartDemo



Online tools for creating various graphs, supports labels and legend.

Chartgizmo



Allows you to create various graphics for websites, blogs, profiles, etc. It may be useful for visualizing scientific, financial or any other information.

CreateAGraph



Online service that supports 5 different graphs and charts of your choice.

DrawAnywhere



Flowcharts, various organizational, hierarchical graphs. Supports export to PDF.

Emprise



It supports various types of graphs, scaling along axes, auto-zoom, etc. Differs high interactivity.

Flowingdata



It works directly from the browser, it is recommended for visualization of various statistics.

Flowchart



Supports drawing a variety of objects (from lines and triangles to Bezier curves), exports to PDF or PNG.

Fooplot



A simple service that supports the construction of graphs and charts.

Gliffy



Excellent service for creating various diagrams, a very nice-looking interface and many features.

Mindomo




Allows you to create and edit communication diagrams.

Mindmeister



A very flexible tool for creating various link diagrams.

MGraph



A simple tool for creating diagrams.

Parto-chart



Online chart generator, supports seven initial tools for free.

Rich chart live



Free service, supports various charts, graphs, etc. Powered by Flash.

WebSequenceDiagrams



Service for creating schemes. A key feature is the description of schemes in a certain way of programming language.

Simpleseating



Excellent service for analyzing various information.

Smartchart



Service for various kinds of planning, also supports charts, is free.

Statistics Toolbox



Powerful tool for statistical modeling and analysis. Positioned to study statistics in practice.

Track-n-graph



A simple but pleasant-looking service for plotting graphs.

Libraries, Plugins, Developer Tools


Plotr



Easy framework for drawing simple graphs and charts. Does not use Flash, and does not support interactive elements.

Plotkit



Library for building graphs in JavaScript, supports HTML Canvas.

jqPlot



Open Source plugin for jQuery library. Supports all kinds of lines and types of graphs.

Google charts



A diverse set of tools for creating charts and graphs, including a JavaScript library.

Highcharts



JavaScript library for creating interactive graphs and charts.

Dygraphs



Another JavaScript library, Open Source.

Protovis



Uses JavaScript and SVG for native rendering.

jQuery Sparklines



A jQuery plugin that generates inline diagrams and graphics.

jpGraph



A powerful library that uses to display PHP.

Flot



jQuery graphing plugin. On the fly generates graphs of data sets.

JavaScript Diagram Builder



A simple library for javascript, allows you to draw graphics on web pages.

gRaphael



JavaScript library for embedding graphs and charts on a web page.

pChart




Totally free php framework.

Fusioncharts



Available for a variety of platforms. Integration with ASP, ASP.NET, PHP, JSP, ColdFusion, Ruby on Rails, etc. is possible. There are flash and HTML5 products.
