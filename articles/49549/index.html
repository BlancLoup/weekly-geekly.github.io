Viliv S5

The company Viliv announced a curious device Viliv S5 . This is a full-fledged computer format PDA. 4.8 "display, ~ 385g. True, only Windows XP is listed as an OS, but the fact that the older model Viliv S7 comes with XP and Linux also makes it possible to hope that Linux will run on S5. And Linux for S7 supports WiMax, there is hope that WiMax will work on S5 - and maybe its driver can be run under another Linux distribution installed on this PDA. I think S5 would be very interesting as a platform for a wearable computer. Sorry, the price and timing of this device is still unknown ...
