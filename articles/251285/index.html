Overview of Microsoft Azure platform updates for February

The news digest of the Microsoft Azure cloud platform contains key announcements and content for developers, IT professionals, and all professionals interested in cloud technologies and the Azure platform. Previous editions of the digest can always be found at the following link . From this digest in the useful content will be not only our content, but also interesting and good links that can be written and not Microsoft.

In today's digest, you can say, the HDInsight - Hadoop ball reigns in the cloud. However, mobile developers have some good news. And a lot about machine learning.

Yes, and in February, I turned 5 years from the commercial launch of the platform.



GA and thumbs


GA : Azure Machine Learning is a machine learning service. You can create models using Xbox and Bing algorithms, R / Python packages, or using your own R / Python code. Then this model is deployed on a web service that connects to any data source. And it all came out of the preview. What else is interesting - the emergence of Machine Learning Marketplace makes it possible to monetize it. More about the new very interesting changes .

GA : Apache Storm on Azure HDInsight - Apache Storm is a well-known OSS package that provides access to the functionality of handling a large number of events in real mode. Now its support has appeared in Visual Studio and Azure, which means that it works stably and if you have questions, you can contact technical support. And it is free.

GA : Azure HDInsight - cluster scaling - it was impossible to change the number of compute nodes after creating a cluster without recreating it. Now you can.

Preview : Azure HDInsight on Linux - HDInsight is Microsoft's implementation of Apache Hadoop in the cloud. As a service. That is, you can click and get a Hadoop cluster in a few minutes, calculate what you need, and delete it. In February, there was support for clusters on Linux.

Preview : Azure Mobile Engagement is a new service that works on the basis of data and provides real-time analytics on your users, segmentation, push notifications and more among all the devices connected to it. Very useful service for both developers and marketing.
