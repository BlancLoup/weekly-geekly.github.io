Alarm from computer and microphone

I want to bring to your attention a program that I wrote in an attempt to find another use for a GSM modem, although it can work without it. This program can have a lot of applications, you just need to turn on the fantasy. For example - the simplest version of the alarm, which reacts to a loud sound, GSM-nanny (by analogy with the radio-nanny).


The principle of operation is as follows - the user sets the signal level from the microphone, to which the program must respond, sets the time during which the excess should be observed, indicates the executable file, script, file (audio, video, etc.) that needs to be opened when fulfillment of conditions. You can also specify the GSM modem parameters, the text of the notification and the number to which you want to send a notification (optional). It starts monitoring, when conditions are met (a level was observed during a specified period of time) the program performs the specified actions (starts the file, sends a notification).



To use the program you need a microphone and, actually, a computer. If the recording level seems to be not high enough, you can play with the settings of the microvol, increase the level, add gain in the audio properties.
