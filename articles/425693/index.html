Search VPS 5 years! 70 hosters give discounts from 10 to 80%

Today, my project to search for virtual servers Search VPS is 5 years old. In my opinion it is quite difficult to determine when the birthday of a particular Internet project is, but I always consider this event on the day of the domain name registration. On October 9, 2013, I registered the domain name poiskvps.ru and on the same day the site worked on a virtual server from Digital Ocean.

The idea to create such a project arose quite unexpectedly: at some point I chose a virtual server from different hosters and decided that it would be convenient to choose a server, specifying some parameters that I needed.


My project is fundamentally different from competitors in that I do not have a rating system. From time to time I get questions from hosters how to improve my position in the rating, and every time I answer that there is no rating in the catalog and artificial intervention is impossible. All tariffs are sorted solely by numerical values ‚Äã‚Äã(price, disk volume, etc.), and any manipulations are excluded here. As a result, users get a completely unbiased picture, in contrast to the situation when those hosters who pay the project are shown first. In my opinion, this approach is very popular with the audience, so very often you can see the advice to use the UPU Search when someone is looking for a virtual server.
