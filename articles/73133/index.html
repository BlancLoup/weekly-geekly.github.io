Aladdin Presents New eToken Pro Anywhere Key

Speaking about the use of smart cards and electronic USB-keys, we are silent about the software that is necessary for the operating system to recognize the smart card or USB-key as a crypto storage. Consequently, their use is limited to those workstations on which special software (drivers) is installed, or to which these drivers can be installed (if of course there is where to install them).

Aladdin Knowlage System introduced to the market a technology, or rather a new version of eToken keys, which can be used without installing any software on a workstation running Windows.



New keys are called eToken Pro Anywhere. Despite the loud name, these keys can be used to authenticate certificates only on web resources. For example, to authenticate using this key in a VPN connection does not work. Although this product has just begun to develop, and in the future, I think, the key functionality will be significantly expanded.

Now a few words about how it works.

The eToken Pro Anywhere key contains a controller that emulates a USB CD-ROM. When eToken Pro Anywhere is connected from this CD-ROM, a mini-loader starts up that climbs to the Internet, to a pre-configured site, and via HTTP protocol, downloads a small library and settings file. This site is recorded in the key settings during its configuration.

After downloading these two files, the USB dongle controller turns off the CD-ROM and connects the dongle itself. However, in the system, this key is recognized as an HID-compatible device, which allows you to connect and use a key without administrative rights.

The downloaded library contains the functions for working with the key (read / write authentication information). The settings file contains a list of those sites that can authenticate using the certificates contained in the key.
