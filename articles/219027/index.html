April 13, Yii 2 becomes Beta

Less than a week ago, the Roadmap was developed by the development of the beloved by many Yii2 frameworks.

In the last few days, as well as right now, a huge number of changes are being poured into the main branch .

Here are some of the most interesting ones (compared to the current alpha, of course)

  • Removed `Application :: preload` in favor of` Application :: bootstrap`
    Not only the array index and the names of several classes have changed, but the module loading system has been decently reworked.
  • Added `HtmlResponseFormatter` and` JsonResponseFormatter`
    This is all preparation for the promised support from the RESTful API box.
  • A huge amount of documentation changes


It is unlikely that Yii2 will undergo major changes before the release, so if someone wanted to use it in his project, but naturally he was afraid of alpha, he had only two days to wait.
