Windows 8 is coming in 2012



The next version of the Windows operating system will appear in 2012, Ballmer said on Monday

Ballmer made this statement in Tokyo, speaking to an audience of software developers. In general, the release of Windows 8 was expected earlier, but yesterday the company officially confirmed for the first time that it would have to wait until 2012.

By the way, the fact that the system will be called Windows 8 is also officially mentioned for the first time.

The release date of Windows 8 from the launch date of the seven will be three years apart (Windows 7 was released in July 2009)

This year, the company hopes to sell 350 million copies of Windows 7.

The new version will have an updated user interface and support for ARM chips, which should give Microsoft a chance in the tablet market.
