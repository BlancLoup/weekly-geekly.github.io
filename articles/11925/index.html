DDR3 Memory Standard released

The new DDR3 standard, approved by the JEDEC Standard Organization for Memory Chips, provides significant performance gains and should reduce the power consumption of DDR3 compared to DDR1 and DDR2 memory modules.

The power supply voltage of DDR3 memory elements is 1.5V, instead of 1.8V for DDR1 and 2.5V for DDR2.
