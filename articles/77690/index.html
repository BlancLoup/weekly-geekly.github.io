ISPmanager Cluster

ISPsystem announces the release of a new ISPmanager Cluster software.
Cluster - the union of homogeneous elements into a group that functions and is considered as a whole.
ISPmanager Cluster is:
- Single entry point. Unlike the traditional ‚Äúmany servers - many control panels‚Äù approach, ISPmanager Cluster has a single entry point to the control panel. The ISPmanager Cluster web interface is built on the basis of ISPmanager Pro and has all the necessary functions for the provision of shared hosting services.
- Fault tolerance. Different types of services and tasks are distributed between different machines, and the failure of one of them does not affect the functioning of the system as a whole.
