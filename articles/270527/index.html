Official Firebird 3.0 Release Candidate 1 and Beta Documentation for Firebird 3.0

The official version of Firebird 3.0 Release Candidate 1 for Windows and Linux (as well as the source code) is available for download at www.firebirdsql.org .

A list of new features and capabilities can be found in the Firebird Release Notes (available as part of the Firebird installer archive).

For the Russian Firebird users, the Moscow Exchange and our company ( IBase / IBSurgeon ) have prepared a special, very pleasant bonus - the beta version of the documentation for the Firebird 3.0 language ( PDF ).
Full documentation is expected with the release of Firebird 3.0.
And yet - there will be a second release candidate, approximately in a month.
