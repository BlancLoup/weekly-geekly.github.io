Apple released iOS 8.1.1

The company released a set of updates for its products, including iOS 8.1.1 (APPLE-SA-2014-11-17-1), closing a number of vulnerabilities in mobile OS components with the wording " This release includes bug fixes, increased stability and performance improvements for iPad 2 and iPhone 4S ". In total, nine vulnerabilities in various components were closed, including the WebKit browser engine and the Lock Screen screen lock service. In addition, specially for iPad 2 & iPhone 4S, OS flaws were corrected, which led to a rapid discharge of the battery when using Wi-Fi, as well as problems with connecting the device via Bluetooth.
