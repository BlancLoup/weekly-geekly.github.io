TagConf 2015 - Fifth Annual IT Conference in Taganrog

Each year, Softline holds the largest regional conference for developers and testers in Taganrog - TagConf . This time the meeting will be organized together with us, the company Arcadia . We and our colleagues from Softline will present you 6 reports. The event will take place on July 11 - come, it will be interesting!



In the conference program:


  1. I have to test the mobile app!
    Artem Selekhov, Softline
    How to test mobile apps? What are the differences from testing web applications? Features of testing mobile applications. The report will be very interesting to those who have to test mobile applications, who have just begun to do this, as well as those who are interested in the advanced experience of other testers.

  2. Web application deployment management
    Anna Carpet (Arcadia)
    Deploying web applications often involves many operations at various levels, starting with updating the application itself and ending with working with the file system and ACL configuration. In the case of a combination of SaaS models and multi-instance architecture, the desire to automate this process grows in proportion to the number of application instances. As an example, we consider the automated deployment system we have implemented and support an integrated web product built on the Microsoft platform.

  3. Manage computers, tablets and mobile phones from Azure
    Sergey Bogulyan (Arcadia)
    It will be about the practical use of services and tools platform Azure, allowing you to easily scale, as a system as a whole, and individual modules. Architectural approaches will be considered when developing a centralized system for remote control of devices, implementation features and existing limitations for various platforms (Windows, iOS, Android).

  4. Features of industrial software development. Basic knowledge of a university graduate
    Roman Cudlay, Softline
    What is facing the developer who comes to the new company. A set of technologies that are useful not only in large companies, but also in amateur projects with several participants. Bad practice, from which not even some studios have left.

  5. TypeScript - a practical overview
    Alexander Shevnin, Arcadia
    TypeScript is a programming language introduced by Microsoft in 2012. Recently, this language has been developing rapidly and is beginning to receive support from other companies, and frameworks and libraries are written on it, as in the main language. The main features of TypeScript will be discussed, how they correlate with ES6, its advantages and disadvantages. All this will be illustrated with practical examples of SPA development using AngularJS.

  6. How not to ... play the project before it starts
    Anton Levin, Softline
    Recommendations and practices for presales. The report will cover the most important aspects of this stage of work on the project: documentation, interaction with the client, organization of the work of participants on the processes of presale.

To participate in the event you need to register . Participation is free.

The format of the conference provides an opportunity to participate in the discussion of issues of interest; the most active participants will receive valuable prizes.
