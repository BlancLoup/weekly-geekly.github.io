PlayStation Vita lets you share content with PC, PS3, PSP



Previously, gamers who had heard about the release of the new PlayStation Vita device wondered if it would be necessary to purchase the same game for this device that was already purchased for the PSP or PS3, again. There was no answer to this question, but now it has become clear that there‚Äôs no problem at all. In any case, the developers have released a new type of application for the exchange of content between all platforms available for some games. As mentioned above, these platforms include regular PC, PSP, PS3 and PlayStation Vita.

Information about this application is already available on the official PlayStation Vita website. In short, the program allows you to save and transfer content from PlayStation Vita to P, save downloaded games from PlayStation Store to PC or PS3 (you can also copy from platform to platform).

The exchange is possible not only for games, but also for other multimedia content, including music, video, pictures and more. Thus, all content can be stored on your home PC, for example, and you can play, listen and view content on Vita.

On the official website, you can get a different type of application, including players, image viewers, a browser, and so on. True, the interface so far only in Japanese. This, of course, is a big minus :)
