JetBrains - Android Development for Lazy People

Want to learn the development of Android, but too lazy to read big books with a bunch of theory? JetBrains prepared a course of 12 lessons for a quick start. Topics for lessons:

* Getting Started
* UI Designer
* Minimally Interactive
* Lifecycle
* More Activities
* List Views
* HTTP
* Storage
* Menus
* Dialogs
* Preferences
* Publishing

As you can see, there is everything you need to create a simple application. And yes, IntellijIdea is used as the ide. Perhaps for someone it will be a plus.
