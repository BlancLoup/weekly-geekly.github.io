Bulletin MS16-023: When a critical Windows security update affects more than just security

In recent days, Microsoft announced the release of new security bulletins, including MS16-023 . The goal of this security update is to address critical vulnerabilities in Internet Explorer that allow the execution of arbitrary code.

However, Microsoft decided that along with critical security updates, its users can also install something else ...


The thing is that having installed the critical update KB3139929 on their computers, the users of Windows 7 and Windows 8.1, in addition to eliminating vulnerabilities, will also find in the system tray an annoying suggestion to upgrade to Windows 10.

In search of explanations of what is happening, carefully read into the description of KB3139929 where we are notified that this is a security update that contains non-security edits .



So simple and not intricate, under the guise of benefactors, Microsoft took a new step to spread the imposition of both Windows 10 and the components of the system in which the user obviously did not need and received from there he did not expect at all.

In conclusion, I would like to note the fact that the update arrives even with the disconnection of the recommended ones and the receipt of only critical security updates. From this we can conclude that users of Windows 7 / 8.1 will now have to carefully read the descriptions for each update, literally study the newsletters with all dependencies in order to make sure that they will install only security patches and not anything else .
