Little experience with Windows 7

There was no problem and the build 7000 was installed.
Tested on AMD Athlon 64 3500+, ASUS A8N-SLI-Deluxe, 2Gb DDR and ATI 2900HD video.

All drivers picked up the first time except the built-in controller of the gigabit network, which in turn is installed from windows update.

There were many reviews and articles about what she is beautiful and all that. But I would like to touch upon some customization and compatibility issues.

The build is currently distributed only in 32bit mode in English. In general, not so much that hyper-unpleasantness, but still a factor when the question is about the car about the home working OSes.
