New view on Habra Editor

Not so long ago, I proposed my own version of the editor for Habr , I liked many of them and asked to continue.
We changed the design for this thanks to crackos , we do not regret karma :). Now he has become more caramelized. Actually the editor himself .

What are other news?


ColorPicker to select the color of the text, was made specifically for this project.

Now you can save not finished articles, so that you can add or edit it later.

Many bugs were fixed.
Added some more functions for working with text via hotkeys. Here is the complete list.
Hotkeys.

  • Tab - Insert a tab of 4 spaces
  • Shift + Tab - Remove tabs from 4 spaces
  • Shift + Enter - Insert html line break (<br />)
  • trl + B - Make the text bold
  • Ctrl + I - italic
  • Ctrl + U - Underlined
  • Ctrl + S - Save Article
  • Ctrl + O - Open saved article
  • Ctrl + D - Copy a string or selected text
  • Ctrl + L - Delete line
  • Ctrl + Shift + UP - Move line up
  • Ctrl + Shift + DOWN - Move Line Down
  • Alt + 1 - Level 1 heading
  • Alt + 2 - Level 2 Header
  • Alt + 3 - Level 3 heading
  • Alt + 4 - Level 4 Header
  • Alt + 5 - Level 5 Header
  • Alt + 6 - Level 6 Header
  • Alt + P - Creates a paragraph.
  • Alt + U - Insert List
  • Alt + O - Insert numbered list
  • Alt + L - Insert List Item
  • Alt + N - Insert preformatted text block
  • Alt + A - Insert Link
  • Alt + T - Insert Table
  • Alt + R - Table Row
  • Alt + D - Table Cell
  • Alt + H - Table Heading Cell
  • Alt + Q - insert a quote
  • Alt + C - Paste the code
  • Alt + Shift + H - Paste habracut
  • Alt + Shift + U - Paste habrauser
  • Alt + Shift + V - Insert Video

In general, many of the wishes were fulfilled, I hope he will help write well-designed and interesting articles. As always, leave all comments in the comments.
