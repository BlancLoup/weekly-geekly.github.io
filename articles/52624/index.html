Guide Bar for Adobe Fireworks

I have been using Adobe Fireworks for a long time and am as happy as the designer can be to the graphic editor. It is much more convenient in it than in Photoshop to edit the vector, it supports Photoshop Live Effects, and there are some goodies like random font smoothing. Of course, the master of the raster was and remains the master of the raster, but for directly preparing layouts, such as drawing buttons, menus, block alignment, making fonts to mind, Fw is nearly perfect. But he has one problem - no work with guides (Guides). They seem to be conceived, but only if created in Ps. We had to export the layout in Ps, insert guides, open them in Fw, and there already work with a calm soul until such time as new guides are needed. In general, not life, but torment. Nothing foreshadowed luck, but I came across a wonderful extension to Fw, suitable for both CS3 and CS4.

The extension is the Guides panel, implemented on the JavaScript API and easily merged with the rest of the panels in the right Fw column. It is delivered as a .mxp file, installed via Adobe Extension Manager, runs through the Window menu -> Guide and looks like this:



The panel provides complete editing tools for guides: paste, copy, delete, group. In general, now I will never leave my beloved Fireworks. :)
