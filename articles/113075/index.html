Virtual keyboard


The other day, I came across a good virtual keyboard. For some reason I have never met her anywhere before, although the product is decent. Our, Russian, even has a patent (ROSPATENT No. 20069611147 dated February 20, 2009).

Of the advantages: supports 70 input languages, IME input. Enter with the mouse, the current keyboard. The author meticulously worked through all languages, including input in Chinese, Japanese, and other languages.
The keyboard was tested in browsers:
  • Internet Explorer 6, 7, 8beta
  • Mozilla Firefox 1.5, 2.x, 3.0.x, 3.1
  • Opera 8.5, 9.x (there are some incompatibilities with keyboard drivers)
  • Safari 3.x (and Konqueror)
  • Google chrome

Also supported:
  • MS Keyboard Layout Creator to quickly create new layouts (there are about 160 layouts in the project).
  • Skins on CSS
  • Enter from right to left.
  • Lite install
  • Tinmyce
  • TinyMCE 3
  • Xinha
  • FCKEditor
