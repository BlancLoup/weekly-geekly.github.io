Auto-tuning IP phones in 3CX - just plug and play

We are constantly being approached with similar problems that arise due to an incorrect procedure for connecting extensions (3CX Extension) to 3CX Phone System. In this article, we will step by step examine the entire procedure for connecting extensions from beginning to end, to show you the only correct and recommended connection method, in which the occurrence of problems is reduced to a minimum.

All settings will be made automatically! You do not need to manually configure anything!

1. In the 3CX Management Console, go to Settings - General - Mail Server and configure the SMTP mail server and mailbox from which the system will send connection settings and other notifications to users.



2. In the Settings - Network - 3CX Tunnel section, specify the server interface on which the 3CX Tunnel service will work. If you have one network card installed, you do not need to change anything. The 3CX Tunnel serves to quickly and seamlessly connect remote 3CXPhone SIP softphones, as well as remote branches to the 3CX Phone System central system.



3. In the Settings - Phone Auto Setup section, correctly enter your external IP address, i.e. The address where the 3CX Phone System server is ‚Äúvisible‚Äù from the Internet. To find out for sure, on the 3CX server, open a browser and enter the address of the site that defines your IP, for example, http://www.whatsmyip.org/ . Immediately set the correct time zone for your hardware IP phones.



4. On the router, ‚Äúforward‚Äù to the 3CX server port 5090 TCP for 3CX Tunnel and port 5000 TCP to transmit information about status and presence . As mentioned, it is necessary to connect soft phones 3CXPhone. If you want to connect hardware IP phones directly to the server, also forward ports 5060 and 9000-9049 UDP, but this is not recommended . Theoretically, outside you can connect, pick up a password from the Extension, hackers and start calling at your expense.



5. In the Internal numbers section , create an extension number. In the Basic section, be sure to include the employee's e-mail address - it‚Äôs the personal number settings that will come to him!



In the 3CXPhone section, check the box for 3CX Tunnel for remote connection. Also, if you want to use 3CXPhone for Windows as a softphone, install the Built-in softphone (for details on the CTI / softphone modes, see the 3CX manual) .



6. Optionally, you can autoconfigure your desk phone. For details, see this autotuning guide .



7. After the extension is configured, select it with the mouse and click on top. Send Welcome Email . The user will be sent a letter with the settings.



8. Install 3CXPhone on your computer, but do not configure it manually! After receiving the email with the settings, just click on the attachment and the phone will configure itself.



9. Similarly, this is done on Android. Just install 3CXPhone for Android from Google Play and click on the attachment.



10. For 3CXPhone for iOS to work, it is necessary, along with the actual softphone, to install the 3CX Tunnel module from iTunes .



Following these simple recommendations immediately after installing 3CX Phone System will save you a lot of time and effort!
