Callback on Windows Mobile 6.5 (Windows phone)


Since December 2009, a new version of Moroko PDA is available . This is a convenient way to communicate using IP telephony from your WM-communicator or PDA on Windows Mobile 5-6.5.


The application is used for VoIP-accounts of IP-telephony operators "family" Betamax. Among them are 12voip.com, ActionVoip.com, BudgetSip.com, CallEasy.com, Dialnow.com, Freecall.com, Internetcalls.com, Intervoip.com, Jumblo.com, Justvoip.com, Lowratevoip.com, Netappel.fr, Nonoh.net, Poivy.com, Rynga.com, Sipdiscount.com, Siptraffic.com, SmartVoip.com, Smsdiscount.com, SMSListo.com, Sparvoip.de, Telbo.com, Voicetrading.com, Voipblast.com, Voipbuster. com, Voipbusterpro.com, Voipcheap.com, Voipdiscount.com, VoipGain.com, VoipRaider.com, Voipstunt.com, VoipWise.com, Voipzoom.com, Webcalldirect.com. Registration with one of these operators is required.

The functionality of the program includes:
  • call order,
  • balance check
  • SMS sending,
  • call history,
  • access to the notebook,
  • use of multiple accounts of VoIP-providers.

The new version added orientation for a set of fingers and the ability to replenish the account.

For operational control of the state of the account in the upper corner of the application window, a balance is displayed. In addition, during a call, the cost per minute is displayed.

The program design has undergone significant changes - the finger-oriented interface is activated, the menu has been shortened, the icons have been changed and enlarged, and the control has been transferred to the system buttons. Updated history format for recent calls and SMS-messages.
