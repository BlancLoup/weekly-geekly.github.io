Registration of applications in the Palm App Catalog e-commerce beta

Here it is! The platform opens for selling mobile applications for Palm WebOS (for Palm Pre devices) and you can already send your applications.

Those interested can already send applications for registration. The commission that Palm will take from the sale of applications is the same as that of all the others - 30%. Users will be charged one time for downloading the application ...

At the moment, it is planned that only the US will be able to use the system (which is not surprising, for devices, they are bound to an operator).

How to post an application


  • Send an email to catalog.ecommerce.beta@palm.com with the subject ‚ÄúCompany Name - Application Name‚Äù
  • In the body of the letter, indicate the user name in webOSDev, what type of application is paid or free, description of the application
  • Attach .ipk file with the attachment
  • Each application must be sent in a separate letter.

Application Acceptance Conditions


  • The application should be interesting to the end user or, at a minimum, useful.
  • The design of the application must comply with Palm UI Guidelines
  • The application must be designed specifically for webOS and cannot be accessed through a browser.
  • The application must use the capabilities of systems and devices (for example, an accelerometer or services)
  • The application must have imputed speed on devices
