Firefly online
The official game Firefly Online will be released in the summer of 2014 on PC, Mac, Android and iOS.
All these versions will be the same on all platforms, so it’s not worth fearing that your version will be completely different from others. From one account it will be possible to enter the game from any convenient device, for example, to continue the adventure started on the PC using a tablet while traveling to work.

The game should be a campaign for the passage, and a full online mode. In Firefly Online will be turn-based battles in which you can manage all the crew members. The battles are realized in 3D, it will be possible to use shelters and environments to win.




The release of the game is planned for 2014, the site of the project keepflying.com seems to be living and regularly issues news.
