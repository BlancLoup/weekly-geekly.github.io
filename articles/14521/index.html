Overview of the Linux model from fu_tu_ro_log :


We can say that this product from Asus will take an intermediate position between the PDA and laptops. Why? Now I will explain.

Firstly, this handsome (the working name "Terminal") is more functional than a normal PDA, but at the same time not as productive as a laptop. For those who do not need it, this performance. First of all, the Terminal will be positioned as the first computer for a child, as a computer for housewives, well, it may well become a suitable purchase for those people who want to bring an element of mobility into their lives in addition to the existing desktop computer. By the way, unlike the UMPC, this unit is not cut down while driving over bumps: there is not a single moving part inside, the hard disk is occupied by 4 GB of flash memory (which can be expanded with an SD card slot).





In addition, this, so to speak, mini-laptop will be an excellent tool in the learning process. There are various training programs installed on the Terminal (periodic table, virtual planetarium, Wikipedia), of course, they will need synchronization with the Internet. In the future, it will be possible to establish training programs in any sciences in which the child will be interested.





The system shell is Linux, inside, in addition to training programs, you can find the full set of programs for working in the Internet (including Skype and ICQ), the Open Office package (full compatibility with all Microsoft formats), the package for working with media (video, photo). manager).



Inside the Terminal, there are Intel Celeron 900 MHz processors (based on the Banias core), which, coupled with a two-cell battery, provides 1.5-2 hours of battery life (an option with a four-cell battery is also available). Next year, with the introduction of Low Power Intel Architecture, the Terminal (as well as umpc) will install new compactly packed, productive single-core processors, released using 45-nanometer technology, which will allow these devices to work on battery 1.5 times longer.

The terminal is also equipped with three USB ports, LAN- and Wi-Fi adapters, 256 (512) MB of RAM, a 7-inch (9 perspective) display, a modem, and a webcam.
