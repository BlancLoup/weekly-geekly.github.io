Expansion Opera: search open site 1.2

A new version of the Site Search extension has been published (not to be confused with Ctrl + F). The main difference is compatibility with Opera 15+.

On many sites it is not easy to find the necessary information due to the fact that the search on the site does not work or it works poorly. My extension solves this problem with the help of popular search engines (Google, Yandex, etc.).

Download and install from the Opera Extensions Catalog

Before first using the extension, you must restart the browser.

I foresee the question "Why hasn't compatibility with Opera 15 been so long?" Frankly, after the news about the incompatibility of the API Opera 12 and Opera 15, I said a lot of obscene words and scored on this extension. But users reassured me.
The converter of extensions from Opera 12 to Opera 15 has simplified the transition. But I still had to work with a file.

Fans of the FireFox browser can use a similar extension for their browser .
