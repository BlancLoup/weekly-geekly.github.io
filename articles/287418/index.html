Technorati: now not only blog search

Technorati's blog search engine today has significantly changed the information architecture and user interface. Dave Sifry made a brief review of the innovations on his blog . Changes, according to Sifri, were mainly made due to changes in the user base. According to him, every day more and more broadband Internet users began to use the service.

It also became a step away from blog search, although in recent times new search capabilities have been added. Apparently, Technorati realized that it‚Äôs hard enough for Google Blogsearch, or it‚Äôs just a strategy to get a more interesting market based on time-dependent material, or both.



The most noticeable transformation affected the main page, the design of which was completely changed. Technorati, also, completely got rid of the types of search: by keywords, by tags and by the directory of blogs, leaving only a general search form. All search query results are now displayed at - technorati.com/tag/your_query ( example ), and a single list that includes search results for blogs, videos, photos and audio materials. After receiving the search results, the user can use the tabs to view the results of interest. Users who want to use the search exclusively on blogs can use the address - search.technorati.com .
