MODX Revolution 2.2 RC1 released

MODX Revolution 2.2 has all chances to become the biggest release of the previously released. Developers have spent a tremendous amount of time improving the administrative interface to make work a pleasure, both aesthetically and in terms of performance.

This version is a candidate for release, so for now you should not expect safe updates on work sites without prior thorough testing. If you want to upgrade, it is strongly recommended that you test everything thoroughly before doing so.



Highlights of what's new in 2.2:

  • Static elements: you can store all elements (chunks, snippets, templates, etc.) in the file system, which makes it possible to edit them wherever you are (including Amazon S3 or Dropbox).



  • Custom panels: you can create your own welcome page, as well as add widgets to the toolbar.

  • Important interface improvements, including cleaner look and faster downloads.

  • Install and update from the command line for real geeks.
  • Significant improvements to the admin panel, giving developers the opportunity for simpler and deeper expansion.
  • Added the ability to create custom resource types, in particular, based on goals, such as blog posting, gallery, forum, events, etc.

  • The parser has been improved, more than 600 unit tests have been added, kernel optimization has been done.
  • More than 160 problems were solved, a lot of errors were fixed.
