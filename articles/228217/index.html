Apple fixes iOS 7 vulnerabilities

Apple released iOS 7.1.2, fixing over 40 vulnerabilities in iOS components such as CoreGraphics, Kernel, Lock Screen, Safari, etc. Users can now upgrade their iDevice (iPhone 4+, iPod Touch 5, iPad 2+) to latest iOS version. As usual, an update for devices can be obtained through the settings option in the OS itself, or through iTunes (when connected to a computer). Vulnerabilities fixed by the company can be used by attackers both for remote code execution via Safari and for accessing some data of a blocked device. One of the fixed vulnerabilities allows an attacker to disable the Find My iPhone feature without entering the Apple ID password on the unlocked device (i.e. Activation Lock bypass).
