Burning Disks to Unprivileged Users in Windows XP

Probably many faced with such a problem as the inability to burn a CD / DVD under the account of an unprivileged user.

Programs such as ImgBurn, Infrarecorder, Small CD writer and Nero simply ‚Äúdo not see‚Äù any drive.

Ahead Nero recommends using a separate utility called Nero BurningRights in this situation, but there is an easier way. which works with all programs for recording discs.

Simply open the Control Panel -> Administrative Tools -> Local Security Policy, or execute secpol.msc.
In the Local Security Settings window that opens, you need to go to Local Policies -> Security Settings -> Devices: only allow local users access to the CD-ROM drives -> Enable


After this simple operation, all free utilities for burning disks and Nero work perfectly under the user account, without installing additional programs such as Nero BurningRights.
