Opera Mobile Emulator 11.5

A new version of Opera Mobile Emulator has been released , which now corresponds to the current version of Opera Mobile 11.5. The new version has updated the list of devices to which you can simulate when you start Opera Mobile, as well as the launchpad itself has been updated.



Screenshots are clickable:



Currently, Opera Mobile Emulator offers the most convenient way to create and debug sites for mobile devices. You do not need to install bulky SDKs or buy a dozen devices, the emulator runs on Windows, Mac or Linux as a separate simple program.

Learn more about testing in Opera Mobile Emulator on Dev.Opera in the updated article " Opera Mobile Emulator for desktop ", in English.

For those who want to test their sites in Opera Mini, there is the Opera Mini Simulator , which runs as a Java applet directly on the site. If desired, you can run even Opera Mini 4.2 . Well, if you need to run some special version of Opera Mini or you want to do it right on your working machine, then there is MicroEmulator for this.
