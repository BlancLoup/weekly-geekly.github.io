kuler does flickr



Adobe kuler - a web application for creating and sharing color sets, got the opportunity to integrate with the Flickr service to create a color set based on images. To try, click the Create button on the left, then select From an Image . Following the button to upload your pictures, you will find a Flickr button that allows you to browse and search for your images or shared images. As in the screenshot .

Other innovations:
- View topics by time (for 7 days, last 30 days, all days), for more coverage of user content.
- The description of the color theme now includes the More Info mode in which you can select an avatar, and get a direct link to the theme, for posting on a blog, sending by e-mail or IM.
