The first neurocomputer interface on sale

The world's first commercially available neurocomputer interface was brought by Guger Technologies to CeBIT 2010. This is the first full-fledged device like this, because the Emotiv EPOC gaming device does not allow you to print texts, but only to jump and run in games.

The Intendix device is intended primarily for paralyzed hospital patients. The interface is ready for use after 10 minutes of configuration. After a workout, you can type texts from the on-screen keyboard at up to 1.25 characters per second by concentrating on rows and columns that flicker alternately (video under the screen). The cap for electroencephalography removes the signal from the cerebral cortex. This speed is a record for prototypes of such devices.

In addition to a set of characters, the Intendix software is able to give various signals at the request of the user, voice the text, send documents to print and e-mail.
