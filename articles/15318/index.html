Dreamweaver Alternative

I would like to discuss the possibility of replacing Adobe Dreamweaver with a free equivalent.
Here is what you need:
1) Highlighting PHP, HTML, CSS, JS.
2) code collapse for HTML and CSS, and preferably in php files too.
3) Search for text in the folder.
4) Full support for UTF-8, with the ability to convert non-UTF-8 files.
What is not needed (optional):
1) Visual editor.

As an option, Eclipse assemblies for PHP were primarily considered, but did not find solutions for 2,3,4
