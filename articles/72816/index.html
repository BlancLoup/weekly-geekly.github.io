Open images in the opera from Google search without an intermediate page.

I often use Google search through pictures and it makes me very angry that I have to open an intermediate page. Sighing, wrote user.js which solves this problem and retains the ability to open an intermediate page.

The only limitation of the script is that it works only after the page is fully loaded and may not work on the slow Internet, but then it is enough to stop the page loading.
To open the image directly, you now need to press on the image itself, and to open the intermediate page, click the link below the image size (by the way: as soon as the color of the domains changes, the script has worked and replaced <cite> with <a>).

The script must be downloaded and saved to the directory in which you store custom scripts: opera: config # UserPrefs | UserJavaScriptFile
