Virtualization today

Before the crisis, when introducing virtualization, the companies aimed not only to reduce IT costs, but also to optimize and increase the flexibility of IT infrastructure. Now, in unstable financial conditions, the main focus is on savings - the remaining parameters go into the background. What can virtualization offer?

There were three main areas: server virtualization, desktop virtualization, (desktop software maintenance and update) - and application virtualization, which reduces administration and software maintenance costs.
In Russia, virtualization projects were promoted by system integrators in almost all industries, with attention paid to the flexibility and resiliency of the infrastructure. At present, the situation has changed a little - the question arose: how to cut budgets while maintaining the same efficiency? Roughly I say, on what, in the current conditions can you save?

On the first plan came such savings, which are quite simple to measure: the number of switching equipment, reducing energy consumption ...
If virtualization is not used, you can start with free server virtualization. Such vendors as VMware, Citrix, Microsoft offer their products to resolve this issue.

Free solutions based on VMwera ESXi, Citrix XenServer will help reduce IT costs by up to 40 percent, and not spend money on purchasing new servers in the near future. This is a bit of help when you consider that IT budgets can be cut to a minimum.

Instead of ordering a server, a more powerful one is purchased, capable of performing much more tasks. In many cases, it is enough to increase the amount of RAM, since most of the tasks are not demanding on the processor resurases. It is also preferable to use shared storage for virtual machines.
If the company has a large infrastructure and there are funds for IT optimization, commercial virtualization solutions may be an option.
In the near future, proposals are expected from vendors for virtualizing a small IT infrastructure. For example, VMware has Acceleration Kits, allowing you to deploy 2-4 physical servers at a discount on products.

In the absence of funds for a full-scale virtualization project, you should think about hiring consultants "by the hour." It is not so difficult - the main thing is the right approach. Counseling can also help with employee training.

If previously a sysadmin managed 10 physical servers, then using virtualization, he will be able to serve up to 50 virtual servers with the same ratio for physical servers. This leads to the automation of the computing environment and is provided by commercial software, but people are always more expensive than software products.
For companies that have already implemented virtualization technologies, in difficult times of crisis, there is a direction in which to reduce costs. The goal is to get a resource-balanced, fault-tolerant data center with minimal costs. To do this, you need to increase user productivity (testing, Lab Manager products), reduce administration costs (virtual machine life cycle).
