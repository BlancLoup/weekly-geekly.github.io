For the help in eliminating vulnerabilities Chrome 14 corporation Google paid about 14 thousand dollars



The release of each new version of the Chrome web browser is going according to plan. This time, the 14th version of Google Chrome appeared on the web. Earlier it was written that Google promises to pay remuneration to independent developers who detect any browser vulnerabilities. For the detection of vulnerabilities in the new version of the browser was paid about 14 thousand US dollars. In addition, more than a dozen extensions of various kinds have been added that will be useful to web developers.

A total of 32 vulnerabilities were discovered and fixed (since the release of the latest stable version). The most important vulnerability eliminated was a vulnerability revealed by a programmer known by the nickname Miaubiz. This specialist received $ 3,500. Another 2337 dollars was received by the Russian expert Sergey Glazunov, who also revealed vulnerability with the category ‚Äúhigh danger‚Äù. The exact amount of payments to independent developers was $ 14,337.

As for innovations, there are many useful innovations in Chrome 14. This Web Audio API, a technology that extends the ability to work with sounds for developers. In addition, an important novelty is the Native Client. This technology allows you to execute C / C ++ code directly in the browser, similar to how modern web browsers work with JavaScript. Native Client was previously only an experimental feature, but now this is a formal innovation.

Much work has also been done on data protection, as well as on the adaptation of the browser under Mac OS X. As always, you can download the new version of the browser here.
