GitHub launched Developer Program

GitHub today announced the launch of the GitHub Developer Program , designed to better inform developers about ongoing changes and teamwork to improve services.



Members of the GitHub Developer Program will be notified of changes to the GitHub API, as well as the earliest access to new features. Program participants will also have the opportunity to request a developer license for GitHub Enterprise and post their tools for review on a new integration page , also launched today.
