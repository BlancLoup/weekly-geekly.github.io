Hosting Cafe

Today I want to introduce Hosting Cafe . This domain brand combines 6 different sites that are related to the hosting theme:

  • VDS.menu - search for virtual servers
  • SHARED.menu - search virtual hosting
  • DEDICATED.menu - search for dedicated servers (in development)
  • HTTPS.menu - search for SSL certificates
  • BACKUP.menu - search for backup storage space
  • LICENSE.menu - search for licenses on the control panel (in development)



At the moment, all projects are working, except DEDICATED.menu and LICENSE.menu. Information about VDS.menu and SHARED.menu has already been placed on Habr√©. Becoming the owner of these two sites, I decided to breathe life into them as much as possible. In addition to these two sites, the domain dedicated.menu was attached. After much deliberation, it was decided to combine these three domains under a single brand HOSTING.cafe - a cafe in which there are various hosting menus. In addition to the existing domains, HTTPS.menu , BACKUP.menu , LICENSE.menu were acquired.

Recently, quite a lot of work has been done on improving existing sites and developing new ones.

The search parameters interface has been improved - standard select boxes and checkboxes have disappeared, instead they are now using JavaScript counterparts, which are displayed equally in all browsers and fit the style of sites. If earlier the user had to choose a value from a drop-down list of three items, now this list has been replaced by a more convenient option with radio boxes. Thus, the number of clicks to get the desired result was significantly reduced.



Another improvement is the description of what data the hoster is requesting in order to be able to order a service from it. Recently it became not so easy to buy a service from a hoster - someone asks for a phone, someone asks for a copy of a passport, etc. However, it‚Äôs possible to find out what the hoster requires only at the very last stage of the service acquisition. Obviously, if at registration on the site of the host it would be indicated that the client would need to send a scan of the passport, they would be registered with less enthusiasm. From now on, a potential client will know what the hoster will require from him even before going to his website. Information is available on the host page or in the description of any tariff. The search for this parameter is not conducted, so that ‚Äúbad people‚Äù find it harder to find a hoster without protection against fraudsters.



Another improvement is related to the speed of finding the ideal tariff and hosting provider - now in order to ask the hosting company representative a question, it is not necessary to go to his website and look for a feedback form there, this can be done directly on the hosting page or the tariff description. You need to fill in the name and e-mail, ask your question, after which the hoster will answer you by e-mail. The function only works for hosters who have agreed to receive such letters.



In addition, some hosters have provided exclusive discount promo codes specifically for visitors Hosting Cafe.



On the page of the description of the tariffs of virtual servers, changes also occurred - earlier the list of operating systems was comma-separated, so its viewing was somewhat difficult. Now the list of OS is presented in the form of a separate block with icons of the operating systems themselves. If desired, this unit can be minimized and then upon further review of tariffs, it will remain in this position.



A form has appeared for sending an error message in the tariff description. You can only fill in the error description. The form works without reloading the page.



A block appeared with information on the date the tariff was added and the date of checking the current information.

BACKUP.menu




There is no need to describe why backup copies are needed, I‚Äôll just note that recently the service of providing storage for hosters has become increasingly popular, so it was decided to search for this service.
Parameters for search by which you can find the tariff:
  • Disk capacity
  • Amount of traffic
  • Data link speed
  • Price
  • Data exchange protocols
  • Country of server placement
  • IPv6 support
  • Payment Methods

The price ratio per gigabyte of space has been added to the standard sorting parameters, so you can find the cheapest place by limiting the top drive parameter.

Https.menu




SSL certificates have recently become more common, Google is increasing its position in issuing, scandals with data leaks or simply a desire to show care for users - one way or another the number of sites that use the https protocol will only increase. Now there are free certificates, but they have significant limitations. For example, the well-known StartSSl does not issue a certificate for commercial activities, therefore, for example, the owner of an online store will not be able to take advantage of this offer.

Search for SSL certificates is currently in beta, so errors or miscalculations are possible when designing this site.

Certificates are issued not by the hosters themselves, but by certification authorities, therefore the hoster acts only as an intermediary, providing the client with technical support and a more favorable price. Therefore, the choice of SSL certificates for many hosters is about the same, although there are slight deviations from the ‚Äúnorm‚Äù. Currently, only the most popular SSL certificates are used in the search, for which exact information on parameters has been collected (IDN support, subdomain support, etc.).

The user can search for SSL certificates by the following parameters:
  • Price
  • Brand - which certificate authority issues a certificate
  • Verification method: when issuing a certificate, either only ownership of the domain name can be verified, or the domain and the organization that owns this domain are being verified
  • IDN support - the ability to obtain a certificate for domains with non-English characters, for example, for domains in the zone. RF
  • SGC Support (Server-Gated Cryptography)
  • Support for subdomains (Wildard), when the certificate is issued not only for the domain itself, but also for all domains of the form * .domain.zone
  • Multiple domains - it is possible to get one certificate for several different domains
  • Extended Validation (EV, Extendet Validation) - certificates with extended validation, with a green bar in the browser
  • Payment method
