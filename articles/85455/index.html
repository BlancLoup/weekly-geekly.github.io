Registration for the RIT ++ 2010 conference started (April 12-14, Moscow)

Today, registration for the conference of professional web developers - RIT ++ 2010 has officially started. The conference will be held on April 12, 13 and 14 in Moscow at the InfoSpace conference center. The official website of the event is http://www.ritconf.ru/ .

RIT ++ 2010 will be an integrator conference, as there will be several thematic conferences within RIT ++: RootConf, AgileDays, ClientSide, MayPerl.


The conference will consist of 7 block conferences and more than 40 professional sections:
- system administration (RootConf): operating systems, peer-to-peer networks, virtualization, support, monitoring, deployment;
- databases : performance, scaling, replication, non-relational DBMS, MySQL, PostgreSQL, Oracle;
- technology of the future ;
- project management (AgileDays): Agile in web development, fixed price contracts: changing requirements at a fixed price, Agile Product Design, Scrum;
- quality : functional, load and integration testing, test management, QA, quality management, methodologies, place of testing in the company's business process, security testing;
- client programming (ClientSide): HTML5, CSS3, modern browsers, animation in browsers (Rich Media), Flex, SilverLight, ExtJS, applied JavaScript techniques, client optimization, server-side JavaScript, semantics, usability;
- server programming : MayPerl (Perl in large organizations, Perl 6, Perl in comparison with other programming languages), C / C ++, PHP, Ruby, Python, Java on the Web (Java for professionals in other fields, Java and finance, security on Java, frameworks, new trends, optimization of Java for high loads, management of project development in Java).

A round table of technical directors of Runet will be held within the framework of the project management conference.

The conference has a hash tag: #ritconf

For speakers

If you want to become a speaker, then by March 20 you need to send an application and abstract of your speech to Oleg Bunin by e-mail oleg.bunin@ontico.ru . More information for speakers is on the website .
