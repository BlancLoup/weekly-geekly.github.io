Minecraft in minecraft'e

I think no one needs to explain what Minecraft is . Simple at first glance, the game is fraught with great opportunities. Especially fans of the game succeeded in building various mechanisms. A calculator was assembled, and even a full-fledged programmable computer . Discussion of such machines is certainly accompanied by comments like:
Now you need to build a PC emulator in minecraft
And run minecraft in it
We are waiting when minecraft is implemented in minecraft

Congratulations, gentlemen, wait! In minecraft, minecraft itself was implemented.


The user of YouTubeCrafted has posted a video that shows a machine that allows you to play a simplified version of Minecraft. The project is called Mineception . The simplifications concern the following:
  • First of all, the game is two-dimensional.
  • There are only 8 types of blocks, 6 of them are available for installation by the player.
  • The size of the world is only 32x32 blocks.
  • The size of the playing field is even more modest, 9x9.
  • Simplified water physics, as in Minecraft Classic.
  • There are no redstone schemes, which means that it‚Äôs not possible to make minecraft-in minecraft-in minecraft.

But there is gravity, including a falling block (sand), spreading water and stairs. A player can walk, jump, place and remove blocks around himself, rob korovanov . Implemented scrolling of the playing field when a player moves.

Special mention deserves the implementation of the graphical interface of the game. The image is not displayed on the panel of light bulbs or torches, but on the map. The map in the world of minecraft is a special item that shows the terrain at a distance around the player. Changes to the landscape in real time are displayed on the map. The principle of the display is based on this.

The area of ‚Äã‚Äã8x8 blocks is projected onto the map as a single pixel, and the color of this pixel will depend on which blocks prevail in this area. If you fill an 8x8 pad with different types of blocks equally, then replacing just one additional block, you can change the color of this pad on the map. Replacing the blocks is done using pistons and a redstone scheme. A detailed description of the operation of such a display can be found here:
www.youtube.com/watch?v=h0e9fZ0242Y

The whole mechanism has huge dimensions, so you need to run it in multiplayer mode of the game, either by increasing the number of chunks that are simultaneously loaded into memory using the server settings, or by placing other players throughout the whole structure.
