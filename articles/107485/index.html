Announced Windows Small Business Server 2011


The official Server and Tools Business News Bytes blog has published the announcement of a new version of an integrated server solution for small businesses.

Windows Small Business Server 2011 (SBS) will be available in three options:
  • Windows Small Business Server 2011 Essentials , previously known under the codename Aurora, is the initial solution for a small company of up to 25 users. This option does not require a separate licensing (CAL) for each user. This server option will offer quick access to Office 365 cloud services, e-mail, collaboration services and CRM;
  • Windows Small Business Server 2011 Standard - previously known as Windows SBS ‚Äú7‚Äù. This option is designed to meet the requirements of the company up to 75 users and offers corporate quality technology. The structure includes the following functions: automatic server backup, email solution (Exchange), internal sites (SharePoint), remote access, file server. This server version requires licensing each user (CAL). This server option will be available in December 2010 (via OEM channels in February 2011), the server price will be $ 1096, the price of CAL - $ 72;
  • Windows Small Business Server 2011 Premium Add-on is an optional package that allows you to take advantage of Window Server 2008 R2 and SQL Server 2008 R2. This package can be enabled for both Essentials and Standard server options. Price SBS 2011 Premium Add-on will be $ 1,604, the cost of CAL - $ 92. This add-on will be available in December 2010.
