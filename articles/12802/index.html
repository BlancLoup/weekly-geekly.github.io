ExtJS 1.1 release released



This version includes the library itself, a simple WYSIWYG HTML editor, a new Ext.Ajax utility class, improved DateField and DatePicker components, as well as extensive documentation and fixes.
The most pleasant innovation is the independence of the library from third-party products and the minimum size of the basic library (36 KB).
