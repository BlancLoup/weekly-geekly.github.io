Retro Games Battle 2014

This morning started the game competition Retro Games Battle 2014 , in which new games written for the old ZX Spectrum platform compete.

There were no genre restrictions, the main requirement is the use of standard ZX Spectrum hardware up to 128k.
So, 3.5 MHz, 128 kilobytes. The main language is assembler z80.
