Broadcast Nokia Developer Days

Friends, the online broadcast of the second day of the conference for Nokia Developer Days mobile applications has just started. You can watch it on the site Nokia Developer Days .



Today, the reports will focus on the description of Nokia platforms and their capabilities, the design and development of applications for the Windows Phone platform, stories about the prospects for the S40 platform (including the recently announced Nokia Asha Touch smartphones) and the capabilities of tools for creating Java applications, thanks to in-app purchase api.

Our colleague from Microsoft, Konstantin Kichinsky, will talk about the concept of Metro UI and the design of applications for mobile platforms, as well as share the intricacies of navigation, examples of using system concepts and application integration with the Windows Phone platform.

With regard to the development for Windows Phone, in his report, Stas Pavlov from Microsoft will consider the advanced features of developing applications for Windows Phone 7.5: using the hardware capabilities of the platform, multitasking, and integration with the platform. In addition, the speaker promised to dwell on the advanced features of the Windows Phone 8 platform, and talk about Windows 8, since Windows Phone 8 has a common core with it.

Anyone who is interested in developing applications for the fast-growing market of Nokia Asha and Windows Phone smartphones, we highly recommend today's broadcast for viewing.
