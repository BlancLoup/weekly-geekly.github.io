Expression Web 4 Service Pack 1 with HTML5 and –°SS3 support released

The Expression Web 4 website design and development package received an update in the form of a service pack. This package, among other things, provides for the package new tools to support modern web standards: HTML5 and CSS3.

Service Pack 1 includes IntelliSense support for the full HMTL5 specification. In addition, some editors in the Expression Web 4 package have added support for CSS3 standard properties, for example, border-radius (screenshot below).



In addition, support for Internet Explorer 9 has been added to Expression Web 4 as a source of web page snapshots. A tool for simultaneous work with the markup of various browsers SuperPreview received a great update and support for rendering the browser Chrome and IE9. SuperPreview also added support for password-protected pages, which users have long requested.

Expression Web 4 SP1 has improved PHP support, added support for hundreds of PHP5 features, and improved some user interface features.

As trivialities: InetlliSense support was added for special characters and codes (left) and virtually all missing main browsers for the page preview function (right).
