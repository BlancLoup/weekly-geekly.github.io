Plugin to get comprehensive statistics about the current page

This is my first major programming experience. And I really hope the fruits of labor will be edible to someone.

Well, please love and favor: StatisFire is a lame plugin for quickly displaying a heap of various statistics.

Here is what the visible part of the popup looks like:



And if more. Clicking on the icon, you can find out:

Standard puzomerki:
  • PR
  • TCI

Check site availability in directories:
  • DMOZ
  • Yandex catalog (availability, as well as the assigned region and "source" of content)

Number of backlinks by:
  • Ya blog
  • Google
  • Yandex

Social references to:
  • Twitter
  • Facebook
  • Vk
  • Google plus
  • Delicious

Overall site traffic rating:
  • Alexa
  • Webomer
  • JAPP (for blogs)
  • Compete

From all sorts of things:
  • Errors from w3c-validator
  • The presence of the file / robots.txt
  • The presence of the site file /sitemap.txt

Learn something about the audience:
  • The number of unikas in the last 24 hours (liveinternet)
  • The number of unikas for today (liveinternet)
  • Number of friends per week (liveinternet)
  • Number of users per month (liveinternet, webomer)
  • Kernel size - regular site visitors (webomer)
  • Audience coverage (webomer)
  • The share of search engines in site traffic (webomer)
  • Gender distribution of audience (webomer)

Indexing page:
  • Is the current page in the Yandex index
  • Is the current page in Google index
  • Is the current page in the Bing index
  • Is the current page in index yahoo

Site Indexing:
  • Number of pages indexed by Google
  • Number of indexed pages on the site by Yandex
  • Number of indexed pages Bing site
  • The number of indexed pages of the site Yahu

Huiz:
  • Ipishnik
  • Where is the server
  • Server type
  • Domain Release Date
  • Domain Registration Date

Attendance schedules from:
  • Li.ru
  • Alexa
  • Compete

One of the goals was maximum customizability. So the display of any parameter is disabled.


I draw your attention to the fact that at the same time information can be displayed both for the current page and for the main domain as a whole.

The icon is also functional. Shows PageRank or TCI (customizable).

Future plans:
  • Much better looking
  • Cache the results
  • Highlight nofollow, noindex
  • Display statistics on the search result pages of all Google and Yandex
  • Show more, more, more statistics

I appeal to leave feedback:
  • What parameters do you want to see added?
  • From what other sites can you parse sensible statistics ?.
  • What is missing in the first place?


Well, I hope my first job was not done in vain, and for some it will come in handy.
