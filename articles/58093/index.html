A month has passed since the last release of the Desktop version of the MDC. Our path was long and thorny, we encountered complex and non-trivial tasks, solutions which were ultimately found ... The current release of the Desktop version of the MDC is called "Spring Aggravation."

(the picture is clickable, inside the flash drive with sound)
In this version we decided to lay the foundation for the work on MDC skinning. Even now, with some knowledge of html, you can create your own message window thanks to the HTML engine WebKit built into Qt . What has been done since the last release of the Flying Pepyka MDC :
  • Google Apps support;
  • Added settings window for each protocol;
Visually, it looks like this:
  • Fixed mismatch of ICQ statuses;
  • fixed errors when transferring lines in the chat window;
  • more stable transfer of contacts from the Inbox and Unlisted groups;
  • now the size of the area for entering a message in the chat window (the message window separator) is remembered;
  • another set of edits when working through a proxy;
  • regular contact sheet optimization;
  • bugfixes for deleting accounts from an MDC account;
  • edit popup positioning in Linux;
  • correction of reconnect errors when the network is lost (or changing ip);
  • You can now close the tab in the message window with the middle mouse button;
  • fixed bug with installing your avatar in jabber;
  • fixed bug with server history download;
  • now in the chat window displays information about the change of status of the interlocutor;
  • bug fixes of working with sound;
  • the chat window now displays the time when the message was received or sent;
  • MDC speaks Russian better and better;
  • MDC can now hide empty groups;
  • MDC now more correctly removes its child processes after shutdown;
  • if the contact is integrated, then when sending a message in the chat window you can see through which account the message will be sent;
