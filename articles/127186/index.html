Skype has launched an App Directory project.



This project is a catalog of applications designed to increase the functionality of Skype itself. Applications are created by third-party developers, and the software itself is fully compatible with the Skype service. Most of these applications have been available before, but now all this can be found in one place, in a centralized directory.

Now there are only 23 names in the Skype application catalog, but soon it is planned to supplement the catalog significantly. The creators of the catalog have added several categories, some of which are so far empty. So we can expect that a significant amount of additional applications will be added soon. The fact that the number of applications in the App Directory will increase, says the search directory, which, in fact, is not particularly needed for 23 titles.

Among other applications for Skype, you can separately mention the video conferencing recording application called VodBurner Video Call Recoder. There is also a useful application that allows you to organize a common virtual desktop, the application is called InnerPass Screen Sharing. Another application is called VuRoom, and it is intended for business users who need more functions in the video conferencing environment.

Some of the apps are completely free, others are worth a few dollars or more. When you click on the name of the application, its page is displayed with a detailed description of what this application does. Information about the system requirements of the application is also displayed. The Skype application catalog also allows a single user to rate applications, and leave comments.

It is reported that third-party developers can offer their own projects for this directory.
