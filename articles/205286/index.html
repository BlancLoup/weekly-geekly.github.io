The digest of interesting news and materials from the world of PHP No. 31 (November 25 - December 8, 2013)



We bring to your attention another collection with links to news and materials.

Enjoy reading!


News and Releases


  • Yii 2 alpha - The long-awaited alpha release of the second version of the popular framework, which by the way is now uses the features of PHP 5.4.
  • Symfony 2.4.0 - Planned release of the framework with many new features and full backward compatibility with symfony 2.3.
  • PHP_CodeSniffer 1.5.0 - The latest version of the system of automatic code verification for compliance with standards.
  • Slim 2.4.0 - Updated popular microfreemvork.
  • The PSR-4 standard was adopted - After a short delay, the standard was still adopted. As well as the PSR-0, the new standard concerns the autoload rules.


Instruments


  • Stringy - A library that implements many functions for manipulating strings and supporting multibyte strings.
  • Dukt Videos - A small library that abstracts the work with the video services Vimeo and Youtube.
  • Media Alchemyst - Need to extract audio from a video? Extract image from document? Generate an animated gif from video? This library will help to quickly solve this kind of problem.
  • Voodoo - Another PHP framework with good documentation.
  • Coupe - A simple web server implemented in pure PHP using coroutines from PHP 5.5. .
  • Parm - A nice PHP implementation of ActiveRecord with autogeneration of models according to the scheme, web interface and other features.
  • Rackem - Another attempt to transfer Rack from the Ruby world to PHP. This is already implemented in Stack .
  • get-in - Useful functions for working with nested arrays.
  • dbv.php - Versioning database, easy and convenient!
  • Wikitten - Small and fast wiki-like application, ideal for storing notes, snippets and ideas.
  • Miner - Simple and convenient Query Builder for PHP.
  • XHProf.io - A graphical interface for analyzing code profiling results obtained using XHProf.
  • Phabricator - Code management system from Facebook. Includes tools for viewing, reviewing and monitoring code, as well as many other features.
  • PHP Toolbox - Directory of PHP libraries. Clone Ruby Toolbox .
  • Awesome PHP - Another expandable list of excellent PHP libraries and frameworks.
