Task number 1 - "Nemean Echo"
In this task, the participants were provided with a so-called echo server, to which you can connect to a specific port and send data. In response, the server sends the same as sent to it. The flag lies on the server, and you need to get it, having only an IP address and port.


Task number 2 - "Soft and silky"
The story is commonplace - very important information flows into the network. Participants are given two files: the binary file itself, which generated the leaked traffic, and the pcap-dump of this traffic itself. The flag is in pcap-dump, but in encrypted form, and you need to reverse the binary file in order to decrypt the data and get the flag.

Almost all participants managed to establish celestial connections, the task became the leader in terrain. Baseline - reference to the telegram-bot.

The first thing that comes to mind: ‚ÄúIsn't there an OpenRedirect here?‚Äù.
