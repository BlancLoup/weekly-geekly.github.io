Stupid moblin

We have already written about the Eee Keyboard PC , a new conceptual wireless multimedia keyboard with a built-in microcomputer based on the Intel Atom processor . Eee Keyboard PC is about to be released as a finished product with the Windows XP operating system on board. However, it turns out all the new interesting details regarding the Eee Keyboard PC. It turns out that the keyboard also exists in the Moblin-version.



Moblin is Mob ile Lin ux - a project to develop a special version of the mobile OS Linux, which is almost entirely supervised by Intel employees. The goals of the Moblin platform are about the same as those of Google Android - to provide a whole class of devices with a standard operating system, with the only difference that if Android is an OS for phones, then Moblin is the basis for so-called MID (Mobile Internet Devices), mobile Internet devices, the idea of ‚Äã‚Äãwhich Intel has been promoting for several years, and which, after Intel, are now developing and constructing dozens of manufacturers.



Moblin has been successfully performing on a number of laptops and netbooks for over a year - for example, at the last Computex exhibition, a Foxconn laptop and an Inventech smartphone were demonstrated. Like all new devices, they are built on the Moblin 2 platform, which appeared literally this spring. The operating system created by Intel employees was specifically designed to work with Intel Atom processors and optimized so that it loads in just a few seconds and consumes less power than devices running on typical operating systems designed primarily for desktop computers. That is why there is a version of the ASUS Eee Keyboard PC running on the Intel Atom processor with the Moblin operating system.



In addition to the advanced power saving features, the keyboard version with Moblin on board has another significant advantage - Moblin is a very mobile operating system that loads in just a few seconds.
