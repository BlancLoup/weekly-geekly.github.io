Using different images on each monitor in Vista or XP

Do you use two monitors for work? And was there a desire to set the desktop image for each screen separately? I think that there was, but, trying to do this, a problem arose. The problem was that Vista did not allow to install different images, and put the same on both monitors. Then, in search of an answer to the question ‚ÄúHow to do this?‚Äù I found the following solution.

To solve this problem, you can use the DisplayFusion application.

DisplayFusion allows you to:
  • Use a different wallpaper for each monitor separately
  • Use images covering all monitors
  • Change images randomly using local files or downloaded from Flickr
  • Move the active window to another monitor ( using hotkeys )
  • Automatically attach windows to monitor edges
  • Flickr integration, image search and download
  • Control application with hotkeys


There are two versions of the application: paid and free. The difference between the paid version and the free version lies only in the ability to automatically change images (only noticed this when studying the functional).
