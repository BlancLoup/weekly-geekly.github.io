News and Releases


  • CakePHP 3.0.0 - More than 3 years have passed since the release of 2.0.0, and now the world has an updated version of the popular framework. Version back is incompatible, requires PHP at least 5.4.16, uses Composer. Many components have been updated, some of which are now also available as independent libraries.
  • PHP releases: 5.6.7, 5.5.23 and 5.4.39 - Fixed three vulnerabilities ( CVE-2015-0231 , CVE-2015-2305 , CVE-2015-2331 ) and fixed about 30 errors .


Php


  • Scalar Type Hints - The proposal for scalar type hinting in the version from Anthony Ferrar was accepted with a score of 108: 48. This was also facilitated by a conciliatory post from Zeev Suraski . The corresponding branch is already in PHP 7 , which means that we are waiting for the possibility of explicitly defining the scalar types int , float , string and bool for the arguments and values ‚Äã‚Äãof the functions: function foo(int $abc): int . And of course, a type- declare(strict_types=1) mode has been added: declare(strict_types=1) , in which a mismatch of the type of value being transferred or returned will result in an error. Support for type checking mode will be added, including for PHP extensions and built-in functions.
  • RFC: Anonymous Classes - Offer accepted, anonymous classes will be available already in PHP 7.
  • RFC: Generator Return Expressions - Support for return in generators is accepted.
  • RFC: Generator Delegation - Delegation of operations by generators to Traversable objects and arrays is accepted.
  • RFC: Reclassify E_STRICT notices - Remove the E_STRICT error class - accepted.
  • RFC: Easy User-land CSPRNG - Add a simple cryptographically strong pseudo-random number generator as functions random_bytes(int length) and random_int(int min, int max) - accepted.
  • PHP 7 Feature Freeze - Overview of already accepted and voting proposals. According to the schedule , proposals for new features for PHP 7 will no longer be accepted.
