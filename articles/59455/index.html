Preliminary system requirements for MS SharePoint Server 2010 announced

So far there is not a lot of news, but from the important one can mention these:
  • IE 6 support stops . He nevertheless approaches its end, one cannot but rejoice. IE 7 and 8, FF 3.0 will be supported, will work on better support for Safari 3.0
  • MS Office SharePoint Server (MOSS) will be entirely 64 bit , that is, the version for 32 bit processors will not be released at all. But this is old news.
  • Only a 64-bit version of SQL Server 2005/2008 can be used as a database.
The decision on the podzherku only 64-bit version of SQL Server is due to the fact that, based on tests and observations of live projects, it became clear that the 64-bit database affects the speed of MOSS and its scalability so much that 32-bit support decided not to insert. Even with MOSS 2007, everyone constantly advised to use the 64-bit version of SQL Server. Well, the reason for the iron companies to update, yes.
