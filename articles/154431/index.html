UPD: Firefox 16 distribution is temporarily stopped due to a security issue (bux fixed)

The developers of the Mozilla Foundation yesterday published a security bulletin , from which it follows that a problem was discovered in Firefox 16, which became available on Tuesday , using which you can get a list of visited sites. At this time, the download link from the Firefox site has been removed.

Vulnerability does not affect younger versions of the browser. The developers of the fund are already working on a fix - the release of the updated version of Firefox is expected within two days. For those who managed to upgrade to the vulnerable version, it is proposed to roll back to version 15.0.1 .

Firefox 16 is still available on Mozilla Russia (14.00 Moscow time). Cases of exploitation of vulnerability are not yet known.
