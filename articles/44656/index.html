keyla - keyboard layout switch for Windows

In order to avoid misunderstandings, I will immediately say that the keyla is not an automatic switch, like the Punto Switcher, and cannot switch the layout itself and is not even intended for this.

Once upon a time I had to use Punto Switcher to use the Caps Lock key to switch layouts. I did not like that for the sake of this it is necessary to install such a rather cumbersome program. So I decided to write something of my own with open source - keyla . Screenshots with a brief description are given on a separate page .



Opportunities


The program allows you to set a keyboard shortcut for cyclic switching layouts and keyboard shortcuts for switching to a specific layout. There is an opinion that with separate keys for each layout it is easier to live, especially when there are more than two layouts. Well, in addition to this, keyla has several settings. A list of all keyla features is provided on the main page.

Keyla displays the current layout with a tray icon. I am by nature a lazy person, so while only icons for Russian and English layouts are available. If you do not have enough icons, write in the comments the names of the language and I will add it. If you wish, you can do it yourself according to the manual, the link to which is easy to find on the main page.

Localization


Interface and installer keyla translated into Russian and English. Localization of the interface is done by editing resource files, which is convenient for the program developer, but translators are inconvenient in two aspects: first, editing resource files is simply less convenient than simple text files, as in Miranda; secondly, to add localization to the program, it must be recompiled.
