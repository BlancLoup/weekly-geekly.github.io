WordPress 3.3 ‚ÄúSonny‚Äù

The latest version of the wonderful WordPress blogging software version 3.3 ‚ÄúSonny‚Äù, named, as usual, in honor of jazz singer Sonny Stitt , is already available for download and update.

WordPress was downloaded more than 65 million times with the release of version 3.0, and in this, the third major iteration, qualitative changes were made in the field of user experience, navigation, downloading and importing data. Innovations can be estimated by viewing this video .


For users


Experienced users will appreciate the new drag-and-drop downloader, which automatically recognizes files, pop-up navigation menus, a new toolbar (upper horizontal menu), improvements during co-editing, and a new Tumblr importer. WordPress takes care of newbies, so version 3.3 contains significant changes in the help system, tooltips for new features that are added in each version, a welcome greeting for new ones. And of course, WordPress 3.3 is optimized for iPad and other tablet PCs.

For developers


A ton of changes for developers. For example, the new editor API , the new version of jQuery library 1.7.1 and the full version of the jQuery UI library, improvements in the system for adding help tabs, and with it the new API WP_Screen, flexible permanent links. The full list of changes is available on the Codex page and in Trac .

Tell them "thank you"


In the Participants tab on the new About WordPress page, the engine developers thank all contributors and release authors.
