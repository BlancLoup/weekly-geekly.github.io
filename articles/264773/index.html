Apple fixed a major vulnerability in OS X

Apple released APPLE-SA-2015-08-13-2 for OS X, which closes a number of important vulnerabilities. In particular, we are talking about the dangerous vulnerability CVE-2015-3760 in the dyld component, which relates to the operation of the component with the environment variable DYLD_PRINT_TO_FILE . The vulnerability was publicly disclosed by the well-known security researcher OS X Stefan Esser ( @ i0n1c ). Then an exploit for this vulnerability was discovered in-the-wild and was used to install adware on computers running OS X.
