Dell PowerEdge R820

Dell was the first to introduce a server that supports Xeon E5-4600 series processors, which we will explore today. But first, briefly about the new line of processors.
The Xeon E5-4600 series consists of 8 processor models, with frequencies from 2Ghz to 2.9Ghz, power from 90W to 130W, with 4 to 8 cores. The younger E5-4603 model contains 4 cores and does not support Turbo Boost, This technology is not supported by the younger 6-core E5-4607 model. The amount of cache memory of the third level they vary from 10MB to 20MB. Processors support DDR3 memory, as well as Hyper-Threading, AES and Trusted Execution technologies.

But now back to the culprit of our article - Dell PowerEdge R820 .




Specifications:

Form factor: 2U (rack mount)
Processor Connectors: 4
Memory: up to 1.5 TB (48 slots for DDR3 DIMM modules up to 1600 MHz)
Drive bays: 16 bays for 2.5 "SAS, SATA, or hot-swappable solid-state drives
PCIe slots:

Two x16 half length, full height connectors
One x8 half length, full height connector
Three x8 half length, half height connectors
One x8 half-length, full-height connector for a RAID controller

RAID controllers:

PERC H310
PERC H710
PERC H710P
PERC H810 (external)

Power supply: Platinum level PSU, 750W or 1100W

In the test copy provided by our partner - Dell, four 8-core Intel Xeon E5-4620 processors (in the end - 32 cores) and 128Gb of RAM are installed.



As a disk subsystem, we had 10k SAS, and 7.2k SAS, and SSD, and even a new-fangled PCIe-SSD (this disk subsystem is a separate topic of my future article, in which I will talk in detail about the Dell PCIe-SSD solution and performance different types of disk subsystems). The Dell R820 uses a chassis similar to the Dell R720 server, and therefore can install up to 16 SFF hard drives (with capacities up to 16 TB). 8 out of 16 slots can be used (as in our case) for PCIe-SSD flash drives with hot-swapping.



On the front panel there are 2 USB connectors, as well as monitor output and power button. There is also a ‚Äútongue‚Äù with information about the MAC address of the iDRAC, a slot for an SD card with iDRAC and DVD-RW, and an information and diagnostic display with control buttons.





The compactness of installing 4 processors in a 2U server is that the processors and memory are ‚Äúdivided‚Äù into 2 boards,
which are interconnected by the dual Intel QuickPath Interconnect (QPI), providing 8GT / sec speed, it also connects all processor sockets in one ring. The top board is conveniently removed and securely fastens into place with a fairly large handle with a bracket.



PowerEdge servers are the first in the world with an integrated Instant-On control system that supports instant access and quick setup with the touch of a button without using additional media. The lifecycle controller allows the administrator to prepare the server for the OS installation. All drivers for all OS supported by the server and the installation utilities are located on a special flash drive built into the server motherboard. This eliminates the need to use disks with drivers and utilities.

To increase performance, servers are equipped with Express Flash PCIe-SSD solid-state drives connected directly to memory and processor buses. They can be used either as a disk cache, for example, to store a paging file, or as a very fast disk for placing frequently requested data on it.

For servers of the 12th generation, the iDRAC control system (Dell Remote Access Controller) is supplied in a new, seventh, version. The iDRAC7 accesses the server console via a separate Ethernet connection, even when the OS is not running or there is no normal network connection. It provides in some cases the same functionality as KVM, but with additional options.
The iDRAC7 remote access control controller allows you to diagnose processors, memory, disks, etc. At the same time, diagnostic utilities are stored on a flash drive.



In general, these built-in functions also allow you to automatically install updates, manage changes, and reliably perform system diagnostics.
PCIe slots, a complete set of which can satisfy the needs of almost any consumer for various kinds of expansion cards.





The back of the server contains 2 USB ports, 4 network adapters, a COM port, a monitor output, an iDRAC adapter, and (as in our case) 2 power supplies that can be hot swapped.
Dell , as always, is pleasantly pleased with the build quality of its servers. Every time I have to work with Dell solutions, I get a lot of fun. But I want to "scold" the cooling system.



The DELL PE R820 server has 6 coolers installed (against 4 installed in PowerEdge R710), which behave quite noisy during the boot phase. And the load time, taking into account the number of expansion cards installed in our server, is quite large. But this is probably the only disadvantage of this server. Despite the noise during the boot phase, the Dell platform again shows low noise and heat generation during operation. Coolers, of course, hot-swappable.

Considering all the above, I would like to note that Dell is strengthening its position in the server market by actively competing with solutions from HP and IBM. And given the fact that this is the first platform for Xeon E5-4600 series processors, the model is extremely attractive for purchase, because in 2U format you can fit a reasonably productive system that solves many of your problems.
