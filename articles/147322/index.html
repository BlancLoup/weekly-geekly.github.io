The first prototype of the Raspberry Pi Model A

A miniature computer Raspberry Pi the size of a credit card on sale for several months. Devices scatter like hot cakes - by April, the Raspberry Pi Foundation organization took over 350,000 orders, which is understandable, given the exceptional cheapness of the computer. Its nominal price is only $ 35 ( plus VAT and delivery ).

It is easy to forget that the more expensive version of the Model B device initially went on sale, and from the very beginning they promised to release a simplified version of the Model A, another ten dollars cheaper. The developers did not forget about the promise made - yesterday, Eben Upton demonstrated the first prototype Model A for $ 25.




Compared to more expensive modifications, the Model A does not have an Ethernet connector and the corresponding network chip. The board has one USB connector instead of two. It was originally planned to reduce the amount of memory (128 MB instead of 256 MB), but in the end it was possible to optimize costs in such a way that the computer met the stated price even from 256 MB.

The good news is that the Model A has exactly the same form factor as the Model B, so the same package is suitable for it. The processor in the computer remains the same, so the Raspberry Pi, even in the $ 25 version, is capable of playing 1080p HD video, running 3D games, etc.

In the photos of Model A you can see two USB connectors, but in fact one of them is disabled, the test instance of Model A is made on the basis of the Model B.
