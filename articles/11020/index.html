Nokia Mobile Web Server is a web server that is always with you.


Nokia Mobile Web Server is a web server for S60 phones based on the port of the popular Apache web server.
Apache comes with pre-installed mods: mod_alias, mod_auth, mod_autoindex, mod_dav, mod_dav_fs, mod_dir, mod_log_config, mod_mime, mod_rewrite, mod_setenvif and (!) Mod_python.

Mod_python allows you to use not only Python scripts, but also Python Server Pages.
The ports are based on Apache 2.0.52 and mod_python 3.1.3.

Also included in the package are concept demos, showing that you are getting more than just saytik on your phone.
Can:
Remotely take pictures of the built-in camera.
Use your phone as a webcam.
Find other similar sites in the neighborhood.
Find the location of the mobile website (cellid).

You can also:
Send an SMS via the web by visiting the site; leave a blog entry or post a comment.
Get access to the phone book, a list of messages, a list of calls.
Pull off photos stored in the phone.
Share the / Apache folder on WebDAV for the ability to download and upload files.
