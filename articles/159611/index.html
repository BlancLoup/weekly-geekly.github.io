New version of NetWrix Active Directory Change Reporter with support of Windows Server 2012 and Exchange 2013 has been released


Without a doubt, the release of new versions of Windows Server 2012 and Exchange 2013 has not gone unnoticed by all of us.
We, in turn, try to keep pace with Microsoft in developing its solutions, and we are pleased to announce the release of a new version of NetWrix Active Directory Change Reporter 7.2 , which supports both Windows Server 2012 / Windows 8 and Exchange 2013 .

The update has affected both versions of the program. Now you can easily work with the free version of the program in Windows Server 2012 / Windows 8. Details under the cat.


What changes appeared in version 7.2?

  • Added support for domains based on Windows 2012.
