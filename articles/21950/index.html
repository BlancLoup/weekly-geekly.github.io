NVidia GeForce 9800 GX2 officially

Not so long ago, NVidia pleased gamers with the release of its new mid-range accelerator GeForce 9600 GT . And yesterday, finally, the high-performance graphics adapter GeForce 9800 GX2, the flagship product of the newest NVidia line of video cards, was officially introduced. This "monster" is built on two 65-nm D8P chips and will compete with the already released dual-processor accelerator Radeon HD 3870 X2 from AMD.

The new accelerator with the PCI Express 2.0 bus is equipped with GigaThread, PureVideo HD and Hybrid Power technologies, is fully compatible with DirectX 10.0 Shader Model 4.0 and is equipped with two Dual-Link DVI connectors.

GeForce 9800 GX2 boasts not only two processors, but also 1 GB of GDDR3 memory with a 256-bit interface. In this case, the memory bandwidth is 128 GB / s. The frequency formula of the accelerator looks like 600/1500/2000 MHz (cores / shader units / memory chips).

The recommended NVidia price for the GeForce 9800 GX is $ 600 ... Playing the latest PC games now is definitely expensive.
