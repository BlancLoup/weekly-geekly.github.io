Windows family turns 27 years old

Today, Windows marks a rather significant event - this family of operating systems is already 27 years old. The first version of this OS was released on November 21, 1985. Then the story of Microsoft has just begun. Here is a promotional video of those times where you can see young Steve Ballmer.



It is clear that the system requirements of Windows 1.0 were just ridiculous, by modern standards. For example, for normal operation of this operating system, 256 KB of RAM was sufficient. A distribution of Windows 1.0 was placed on only two floppy disks. True, already at that time, for normal operation, Windows required a graphics adapter. And yes, then Windows was not yet similar to Windows, so to speak. The familiar and recognizable interface elements were added to subsequent versions of the OS.

It is also worth noting that Windows 1.0 did not become particularly popular. And the second version also did not differ in demand. But Windows 3.0 and NT are already widely used thanks to new features and functions. But, of course, Windows 95 gained the most popularity, after which the victorious march of the Windows family began in offices, educational institutions and homes of users. There were some bad moments, but there are still more successful ones. According to research firm Net Applications, about 70% of computers worldwide are running Windows.
