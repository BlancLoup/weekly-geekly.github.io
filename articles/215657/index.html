We program in Python

This book is the perfect guide for beginners to learn Python . The manual, written by an experienced developer and teacher, will teach the fundamental principles of programming on the example of creating simple games. You will acquire the necessary skills for developing applications in Python and learn how to use them in real practice. For better assimilation of the material in the book are many examples of software code. At the end of each chapter you will find a draft of a full-fledged game, illustrating the key ideas of the topic set out, as well as a brief summary of the material covered and the tasks for the self-test. After reading this book, you will fully familiarize yourself with the language of Python, learn the basic principles of programming and will be ready to transfer them to the ground of other languages, for the study of which you will undertake. Learn how to program in Python playfully!
