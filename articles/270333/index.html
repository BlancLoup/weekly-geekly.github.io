Unofficial Firebird 3.0 Release Candidate 1 Installer


A few days ago, the source codes of the first release candidate version of Firebird 3.0 were recorded in the repository of the Firebird Project. Usually it takes several weeks from the moment of fixing the source code to the release of the official installer, so we are releasing an unofficial installer at IBase.ru ,
with which you can install and try Firebird 3 RC1 right now:

http://firebird3.com/en/download-firebird-3/

We remind you that this version of Firebird 3 is still not intended for industrial use, but all the main functions in it are already fixed, so it's time to test its work with your databases and applications.

If you want to install Firebird 3 as a second copy, watch the video where the installation process of FB3 is shown as a different port .
