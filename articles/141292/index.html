Direct Attached Storage CFI-B4043JDGG Review

I recently learned about an interesting class of DAS devices (Direct Attached
Storage).



This is a NAS that does not have its own operating system.
system on board, but allows you to create a hardware raid array of 2-8
drives. I admit that if before that I did not have NASa, then I would
doubted that such a device could be claimed at all.
It is not clear how it is better than a regular external hard drive. BUT
it turned out that in certain circumstances DAS is exactly what
need to.



Before that, I wrote a review of Synology DiskStation DS211, very worthy of the NAS
with a bunch of features and bells and whistles, but if you have a full-fledged home
file, web or something else there is a server that you yourself
admin (like me), as if a NAS is not needed. After all, he sharpened "under all
works out of the box "- and put some kind of software on it - the whole
problem. Therefore, DiscStation was presented to iBear , and I began to settle down with
DAS CFI-B4043JDGG, because without knowing it, I already had its prototype
used:



As you can see in the picture - DAS is just what I really needed.
Firstly, I could finally collect hard not fit into the systemic
drives in a civil box, and secondly, instead of a pile of wires
use one USB-3 or eSATA. Data transfer rate while
can reach 300 megabytes per second.



I have such a pretty standard picture:



I have already said that DAS is essentially a hardware raid. Configuration
raid done just by 5 points:



As you can see, it is made using three switches. Possible
configurations:
Branded by CFI:
  • Clean - all disks are independent of each other;
  • Large - all disks are combined into one large virtual disk;
  • Clone - data is cloned on each installed disk, while
    storage size is equal to the size of the smallest disk.


And of course the classics:
R0, R1 / 10, R3 and R5.
Honestly, I still could not understand why I could use
Large instead of R0, and Clone is generally some kind of strange mode, if the disk
two, then R1 is the most it, and if 4, then here already R5 asks.

Clean mode is useful if the DAS is used as a hard drive box.

After selecting the mode, you need to hold the reset for five seconds, turn on the power, and the device
will configure the raid according to the position of the switches. In this case, all
information from hard drives will be erased. Then you need to mount
device like a disk in the operating system and enjoy. Quickly,
clear and no hemorrhoids.

Hardware and software



  • The main raid functionality provides raid controller JMicron JMB394 or JMS539;
  • At most, you can install 4 3.5 "disks;
  • There are no hardware limitations on the disk size, but you need to coordinate it.
    with the OS to which the DAS will be connected;
  • Cooling discs produced using 80mm fan on the rear panel;
  • On the front is the hard drive compartment cover;
  • Compatible with Windows, Linux, OS X.
