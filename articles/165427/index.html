Chrome Beta is available on Google Play



The Google team has released the first beta version of the Chrome browser for Android in the Google Play store. Chrome Beta has version 25.0.1364.8 and a number of distinctive features.



You can try new features (and bugs!) Faster than they appear in the base version of Google Chrome. At the same time, feedback is maintained with the developer. Some new features may be slightly unfinished, but the developers promise to update the Beta version to eliminate these inaccuracies.

Chrome for Android has the advantage of speed, security, and improvements over other platforms.
For example, in version 25.0.1364.8 , Octane's performance improved by approximately 25-30%. This update contains interesting HTML5 properties for developers such as CSS Filters. This is only one step out of many in the way of improvements in the development of the mobile Internet.



You can install Chrome Beta along with the current version of Chrome for Android. Chrome Beta will request additional permissions when using Chrome Sync on first launch.
play.google.com/store/apps/details?id=com.chrome.beta
