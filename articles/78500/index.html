SeeMap.ru (or SmotKartu.ru) - a project that allows you to find the best routes between two points on the map in the city of St. Petersburg.



Highlights:

1. Search routes with a selection of all vehicles passing by. The search radius of the first route from the starting point depends on the distance between two points. This is very convenient if, for example, a route passes a kilometer away from you, which takes you to the end.
2. Select a specific route and display it on the map.
3. View all routes in a given area (up to 1000 meters)
A handy thing if you want to find out where you can generally get to from a given point or find out which routes pass near the subway or cafe.
4. It is possible to create your own links on the site of the form: seemap.ru/idxxx, for example
