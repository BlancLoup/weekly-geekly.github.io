SMS Gateway



Expansion of the functionality of the portal www.aucoins.ru

Hardware


Separately, a computer with a minimum configuration was purchased for about 7,000 rubles. The only focus was on minimizing the noise level: embedded video, a quiet fan to the processor and in the power supply, as the computer works at night.

This was followed by buying a phone (budget 1000r). The main requirement is the possibility of charging from usb. From the first time did not work. Motorola V3 refused to send long sms (more than 150 characters). It turned out to be the trouble of all motorol. Then SonyEricsson K500 was purchased and a usb data cable with the ability to charge the phone from the usb port.

Software


First, the choice fell on ozeki . Of the benefits can be noted great functionality. Among the shortcomings: an inconvenient interface, besides, I never managed to set up a full-fledged sending of Russian-language SMS from scripts.

The search for the program continued, and further choice fell on the nowsms . The program with a simple intuitive interface. On the developer's site there are examples of sending from php script. Everything is simple and clear, you can make friends with any programming language.
Now it only remains for the provider to connect the service with a static ip-address and everything is ready for a commercial launch.

After setting up the program nowsms and installing modem driver for the phone, everything worked like a clock.
Nowsms works as a service and provides access via HTTP protocol.
