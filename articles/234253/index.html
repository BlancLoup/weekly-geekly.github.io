Take part in the Games Jam 2014 competition

Dear friends! On August 9, the Games Jam 2014 competition started, and you can still take part in it, and you have a chance to get a double prize as there is a nomination from Microsoft within the competition. As part of the competition, experts from Microsoft will help developers with useful tips and master classes. On September 5, Microsoft conducts an online webinar where Microsoft experts will tell you how to create a successful game for the Windows / Windows Phone platform. For this contest we have created a special page where you can find out the details.


You can develop a game for Windows Phone and Windows 8.1 using a large number of technologies and tools, many of which are corss-platform. If you have already created a game for other platforms, for example, using Unity3d or HTML5 / JavaScript, you should definitely port it to the Windows Phone and Windows 8.1 platform, thus you will increase your chances of winning the prize. And these are three Xbox One for the first three places and three Nokia Lumia 930 for three special nominations.

Already, you can find out how to create and transfer gaming applications to our platform on a special page of the competition . In addition, you can ask your questions related to technology in the comments to this post.

Within the framework of the contest, on September 5, at 10 am, there will also be an online webinar, which you can register for now . As part of the webinar, we will talk about how to register in the Windows application store, what steps need to be done to create an application package (using the example of a game based on Unity3d and HTML5 technologies), and how to publish this application. In addition, during the webinar, you will be told about how you can use Microsoft Azure technology as part of your game project.

C refereeing criteria and competition rules you can find at http://special.habrahabr.ru/microsoft/kri/#designer-games .

We wish you good luck and victory in the competition!
