Kali Linux 2017.1



The long-awaited release of Kali Linux 2017.1!

Kali Linux 2017.1 rolling-release, which contains many interesting updates and features: updated packages, updated kernel, which provides better hardware support, updated tools and innovations.

Kali Linux is a distribution kit that contains many utilities for conducting penetration testing - from analyzing web application vulnerabilities to hacking networks and services and consolidating into the system. Previously, this distribution was known as Backtrack.

What's new?


Support for wireless cards RTL8812AU


Distribution developers have received many requests to include drivers for wireless chipsets RTL8812AU. These drivers are not part of the standard Linux kernel and have been modified to allow packet injection in the 802.11 AC standard.

Optimized CUDA GPU Cracking support


Installing pre-installed graphics drivers has always been a source of frustration in Kali. In the new distribution, this flaw has been eliminated: you can now use NVIDIA GPUs natively when using Hashcat and Pyrit.
