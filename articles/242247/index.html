The first browser specifically for developers: soon

We at Mozilla understand that developers are the foundation of the web, so we are actively promoting standards and continue to develop great tools to make it easier for you to create great web content and applications.

The web development process typically uses countless different tools that often do not work well together. This means that you have to switch between different tools, platforms and browsers, which slows down the process and reduces productivity.

So we decided to give free rein to our group that creates tools for Firefox developers to make the entire browser entirely. And see what happens.

Now we are ready to show a glimpse of the first browser, created specifically for developers.
