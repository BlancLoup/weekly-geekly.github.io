GPU Global Illumination

Well, gentlemen, V-Ray / Mental Ray lovers ... CryTek decided to make fun ... If you used to have one frame with technology like "global illumination" (light reflection), you needed hours or days, now it can appear on your screens ... as fast 120 fps! What we saw only on pre-rendered things, levels, games with baked lighting from Illiminate Labs (Mirror's Edge) - will now be real-time:

CryEngine 3 - Global Illumination with Light Propagation Volumes

(Watch this, of course, in HQ and Full Screen)

At about 1:00 AM, Manhattan Apartment will begin, showing real-time illumination changes and dynamic Global Illumination.

Of course, for the time being this is only done for CryEngine 3, but the point is that modern motherboards allow such calculations to be done in real time, which means that in a short time it will be possible to forget about the days spent on rendering one frame.

And from them - Crytek never ceases to impress - massive illumination at 120 fps.

CryEngine 3 - Massive Lighting with Radiance Volumes


However, a small digging deep into the Internet shows that, in fact, the developers of V-Ray are not sleeping and are already presenting their own GPU version, which can already make rough sketches at 3 fps and speed up the process at 20x!
