YouTrack 2017.4 Release: Time Evaluation Report, Markdown Support, and more

Hello! We recently released a new version of the project management system - YouTrack 2017.4 - and we hasten to share with you the innovations.




What's new in YouTrack 2017.4?


  • Time evaluation report.
  • Sort by relevance in search results.
  • Customizable date and time fields.
  • The npm package to support JavaScript workflows.
  • Localization in Japanese.

What else?


  • Text indexing for task fields.
  • Support for additional languages ‚Äã‚Äãin the text index.
  • Shared Mailbox support in Microsoft Exchange and Office 365.
  • Ability to disable email notifications.
  • Markdown support as part of experimental functionality.

Time evaluation report


We have added a new report that allows you to track time estimates in one or more projects. In the new report, tasks can be grouped by project, executor, subsystem, and sprint.





Sort by relevance in text search results


Now, when text searching by task, results are sorted by relevance. This means that tasks that contain more matching words in the title, description, or comments are displayed above in the list of results.


This sorting is applied to the default search results, unless the search query contains explicit sorting attributes (for example, sort by: updated) or the expressions "or".





Customizable date and time fields


We added support for custom fields in which dates are stored in a "date and time" format. This means that it is now possible to distinguish between fields in which only the date is stored, and fields for which time is of the essence. This innovation allows you to keep track of time with greater accuracy.





Npm package to support JavaScript workflows


We are constantly improving the process of working with workflow. In this release, we added the ability to manage YouTrack workflows from our own development environment. This feature is available in any editor that supports javascript. To start using this functionality, install Node.js in your environment. For more information, see our documentation .





Localization in Japanese


YouTrack now supports Japanese. Companies communicating in Japanese can set it as the default language in their YouTrack.


Text indexing for task fields


The text index now includes all values ‚Äã‚Äãstored in all task fields. This means that the text search finds matching words in any attributes, not just in the title, description and comments.


Moreover, you can not specify the attribute in which you want to search for the value - just type in the search field!





Support for additional languages ‚Äã‚Äãin the text index


Now YouTrack supports text indexing for new languages ‚Äã‚Äã- Chinese, Japanese, Korean, Turkish and Polish. In order to use this functionality, enable text indexing in the required language, and then when searching for tasks all word forms of the words from the search query will be taken into account.


Shared Mailbox support in Microsoft Exchange and Office 365


We have added the ability to connect and receive mail from a shared mailbox on Microsoft Exchange server. Such mailboxes are often often used to organize technical support by several engineers from one external mailing address, for example, support@company.com. If you use Microsoft Exchange or Office 365, then YouTrack can now be integrated with them to automate the work of technical support and organize its interaction with developers.





Ability to disable email notifications


If you no longer want to follow the updates of specific tasks, you can now unsubscribe from them. To unsubscribe, click the link at the bottom of the letter from YouTrack. A list of disabled alerts from tasks is displayed on the Notifications tab of your user profile. You can subscribe to tasks at any time.





Markdown support


In YouTrack 2017.4, we added markdown support as part of the experimental functionality. For text formatting in task descriptions, comments, and tasks, you can use the Markdown syntax. You can switch between the Markdown syntax and the wiki to choose a format that is more familiar to you. Markdown is enabled by default, if you do not want to use this functionality, turn it off on the Experimental features page.




We also made a number of other improvements - see the full list of completed tasks included in the release.


We will be glad if you try YouTrack 2017.4 and share your impressions. You can download a free version for 10 users or register in the cloud to use the trial version for 30 days.


If you are already using cloud YouTrack, then your instance will be transferred to the new version according to the calendar .
