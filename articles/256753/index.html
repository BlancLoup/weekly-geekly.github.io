Vivaldi 1.0.162.2 TP3 Candidate

Hello!

Today is not quite the usual build. Firstly, it is extraordinary, and secondly, it is a preliminary release of the next Technical Preview at number 3. In other words, the TP3 release promised a few days ago and postponed for technical reasons finally reached the finish line.
