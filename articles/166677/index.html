New Instance EC2 Type - High Memory Cluster Eight Extra Large Instance

Hello!

Right after High-Storage Eight Extra Large, Amazon Web Services announced a new type of instance for In-Memory computing ‚Äî High Memory Cluster Eight Extra Large Instance.

So, the technical characteristics of the new type of instance:
  • two eight-core Intel E5-2670 processor with a frequency of 2.6 GHz with support for Intel Turbo Boost and NUMA
  • 244 GB of RAM (!)
  • two 120 GB SSD hard drive
  • 10 GB network for Cluster Placement Groups0
  • HVM virtualization only
  • only EBS-backed AMI

In general, why might such a monster be needed? Usually, such servers are needed for applications that are memory hungry or in-memory cache systems. In total, we get a thirty-two-core system (hyper-cracking of sixteen cores), based on 88 ECU (EC2 Compute Units).

These machines are available from $ 3.50 for Linux and $ 3.831 for Windows per hour. With an annual reservation, we get a price of ~ $ 1063 per month. As for me, this is a very good price for such capacities. Your thoughts on this?
