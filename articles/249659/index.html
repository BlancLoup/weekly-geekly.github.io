Weekly build Vivaldi 1.0.94.2

Hello!

As we promised - today we offer you the first weekly assembly of the browser Vivaldi 1.0.94.2. I remind you that this is a test unstable build, in which we fixed a part of the errors that you reported to us after the previous version, but it is also possible that we added new bugs and even made a certain number of regressions. Yes, the process of debugging software is thorny and unpredictable - you have to be ready for any surprises.

Meanwhile, we paid a lot of attention to correcting errors in the browser code, and below you can see this list.

So, what's improved and fixed in this weekly build.

Major changes:

Improved browser performance on some operating systems;

Added the key combination Ctrl + Enter for automatic addition to the address bar www and .com;

Added the ability to view the forward / backward history of browsing pages on a tab, opens with a long click on the button (in the process of improvement);

Added the ability to open recently closed windows;

Fixed a browser launch error if the program window was closed with the settings window open;

Fixed error processing of single-keyboard shortcuts when typing text in forms;

Fixed a bug in the Linux version, leading to the opening of a black window;
