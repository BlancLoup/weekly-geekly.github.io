IceCash 1.3. Linux workplace cashier on php, with the driver Shtrih-M



Hello. Once a long time ago I wrote an article about the first IceCash project. Actually, this project is not lost anywhere, it lives to itself in my organization and is gradually improving. It works at about forty box offices. Not a masterpiece, of course, but fulfills its function.

Namely, the program manages the fiscal apparatus, interacting with the cashier through a local site. You can connect to this site remotely and also work by controlling the local fiscal. For all the time, various jambs were identified and eliminated and new ones were written :) Well, everything is as usual. The project has its own server part. All transactions are collected in one database, scripts for the exchange of server information with cash registers work. A local report was added, from which you can make copies of the check. Also, the system can now work with bonus cards (associated with the bonus server).


Since everything that was written was incomplete, even very chaotic, it was impossible to convey something completed to the interested persons. And yet I had to make an installation disk, which was supposed to remove a lot of garbage from my memory in the form of instructions for installing the cash register. I decided to put this disc on the Internet. Perhaps someone come in handy.



Briefly about the system itself : Linux Debian 6 (kernel 3.2) icewm, apache2, mysql. The fiscal control daemon is written in python. Web muzzle on php. Driver Shtrih-M taken from Dmitry Shamov project
The installation disk can be downloaded from here iso or from here torrent . In the installation it is possible to install the installer itself on a USB flash drive, so it is more convenient. It is possible to connect to the intallator via RDP. The system itself takes about 2GB of disk space. The installer will split the disk into a swap of 1 GB and 7 GB root.
