3CX v16 Update 1, 3CX app for iOS Beta and the new version of 3CX Call Flow Designer

We present a review of recent 3CX. There will be many interesting things - do not switch!

3CX v16 Update 1


We recently released another 3CX v16 Update 1. The update includes new chat features and an updated communication widget for your 3CX Live Chat & Talk site. Also in Update 1 there is a new service Call Flow Service, which adds a call management scripting interface to the PBX. The script engine works together with the Call Flow Designer development environment and allows you to create call processing scripts of any complexity.

Updated Web Client Chat




The updated chat now allows you to flexibly manage your conversations. In addition, it interacts well with the 3CX Live Chat & Talk communication widget.

  • The site visitor can start a chat with the 3CX Operator Queue. This creates a chat group that includes all queue operators and this visitor.
  • Further, the Queue operator can switch the group chat to itself and continue personal communication with the visitor.
  • Also, the operator can transfer the chat to another operator or a regular PBX user, if the need arises.
  • To unload the web client interface, selected conversations can be transferred to the archive (however, they are not deleted).
  • Different types of chats (from the site, group, etc.) now have different icons for easy identification.
  • Now you can quickly send a letter to the site visitor by clicking on the e-mail in the chat window.

More features are described in the Chat and Video Guide .

Updated 3CX Live Chat & Talk widget




The updated widget 3CX Live Chat & Talk offers additional interface customization and enhanced integration with sites made on CMS WordPress, as well as on other technologies.

  • Set chat window icon - you can set a suitable image for the chat window title. This could be, for example, your company logo.
  • Installing the operator icon - you can also set the chat operator icon, for example, his photo.
  • Placement of the widget - the ‚ÄúPosition‚Äù parameter determines the placement of the widget on web pages ‚Äî bottom right (by default) or bottom left.
  • The view in the mobile browser is a small but important addition. Now when entering the site from a mobile device, the chat window is displayed minimized by default.
  • Chat pop-up window - in 3CX v16 Update 1, the 3CX Live Chat & Talk widget window ‚Äúpops up‚Äù in a separate window, allowing the visitor to freely navigate the site, but at the same time contact the chat at any time.

Call Flow Service Script Interface


In 3CX v16 Update 1 a new scripting interface Call Flow Apps Service has appeared. It supports voice applications 3CX new standard. However, existing applications can be converted or improved in the new version of Call Flow Designer (see below). The architecture of the Call Flow Apps Service is today finalized. The application server runs on 3CX for Debian / Raspbian Linux and Windows.

New 3CX app for iOS beta


We have not updated our branded 3CX application for iOS for a long time. Some users even complained about the incorrect operation of file transfer. But in the next update, all problems are fixed! This time the emphasis is on the possibilities of integrated chat. Now chat in a mobile application is almost as good as chat in the 3CX web client.



The application has the opportunity to create group chats and assign them names. Also added chat archiving. To archive a conversation, swipe your finger to the left (in the same way, you can restore the conversation from the archive).

Also in the application appeared chat with site visitors through the communication widget 3CX Live Chat & Talk. Here is how it is now implemented:

  • Swipe to the left to open additional options: Pick yourself, Translate, Finish and Delete.
  • Chat icons from a site are different from regular chat icons so you can easily distinguish them.
  • Push messages from the site display the visitor‚Äôs name and message content.
  • Chats routed to the Operator Queue include the Queue name - for your convenience.

New release 3CX Call Flow Designer


Just this week, we released a new version of the 3CX CFD voice application development environment. It features new C # components, a refined user interface, improved error handling, and auto-update of the development environment. New CFD is needed to create voice applications of a new type for 3CX v16 Update 1 and higher.


Updated Development Environment (IDE) CFD offers additional tools for developers:

  • New components Execute C # File and Code. They replace the outdated component ‚ÄúLaunch External Script‚Äù. Components can run files with C # code or embedded code fragments directly from CFD applications.
  • The new component ‚ÄúSet extension status‚Äù implements a simple installation of extension parameters from the CFD application.
  • Improved error handling. The new expression editor (Expression Editor) additionally checks the values, identifying errors at the compilation stage.

In addition to improvements related to the development process, we added several features that enhance the usability of the application itself:

  • Auto update application. CFD now automatically checks the availability of new versions and installs updates as soon as they are released.
  • A new menu item ‚ÄúSave Project As‚Äù allows you to save a CFD project with a new name or in a different location.
  • New context menu "Open Audio Folder" for components that use audio files. It opens Explorer for easy viewing of the folder with the project audio files.
  • Convenient display of disabled components. They are now grayed out to differ from active CFD components.

The new CFD release involves the use of 3CX V16 Update 1. Download the CFD and install it according to the Call Flow Designer Installation Guide .
