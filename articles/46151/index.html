AppleJack - Mac OS X troubleshooter

AppleJack is the perfect free troubleshooting application for your Mac OS X. It will be an excellent solution even if you cannot download the system GUI interface or you do not have an Mac OS X installation disc at hand.



It is worth considering that AppleJack runs only in Single User Mode (Single user mode).

To use AppleJack, restart your Mac in single user mode, hold Command + S on startup, and then type applejack , applejack auto or applejack auto restart (restart your Mac automatically at the end of the process).
