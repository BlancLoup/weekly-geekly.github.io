When a picture is more eloquent than 1024 words - prototyping with MockingBird

For writing TK, I wondered about finding a simple and convenient tool for creating prototypes of web pages. I wanted something free and online, some alternative to Axure. This service was found, and I bring it to your attention. So, meet www.gomockingbird.com - a service that allows you to create very beautiful prototypes easily and conveniently.
In addition to the presentation of the service itself, this post also wanted to draw public attention to the useful practice of prototyping.

MockingBird is based on Cappuccino and looks quite like a desktop application with the ability to export rendered prototypes to PNG and PDF.

Features more



MockingBird is made for prototyping sites and web applications, so it already has all the most used interface elements.

Create as many pages as you need, link them, rearrange them easily and conveniently.

The system is completely online, so you do not need to install or download something. You, like your customers, can access your prototypes via the Internet from anywhere.

Easy and convenient resizing of objects also changes the size of the text inside them, which allows you to create prototypes faster.

You can create neat and understandable prototypes that are not cluttered with unnecessary details.

Convenience of use is achieved due to many pleasant things, such as, for example, alignment with the grid.
