Moscow number in code 495 for free


The telephone symbol of Moscow - code 495 has ceased to be a luxury. Connect and use the Moscow number in the prestigious code is now free.
Technically, the toll-free number is fully functional and unlimited.
A natural question arises: if money is not paid, what is required of the client? The requirement is only one - the number must be used.

Connection of numbers takes place via SIP, with each number 10 incoming lines are provided with the possibility of free expansion. Numbers support all popular codecs (G711 u / a, G729, Speex, GSM), fax reception by T38 and G711, DTMF (rfc2833). There is a possibility of SIP-redirection (in the format number @ ip) and normal redirection (to any number).

What conditions?


As already mentioned, the condition for the free work of the room is its use, which is determined by the number of incoming minutes per month. For clarity, we present in the form of infographics:


As shown in the table, for completely free use of a room, it is enough to take 700 incoming minutes per month or 22 minutes per day (incoming calls themselves are naturally free too). For any corporate user, this is an absolutely insignificant figure.

How to connect?


To connect the free number 495 you need:
  1. select a direct Moscow number and register on the site
  2. replenish the account for 290 rubles (fireproof balance - in case there are no monthly calls to the number)
  3. turn on the number and receive calls (to connect the number you need to download a scan of a passport copy)

The number will start working immediately after confirming the documents (no more than 2 hours on a working day).
Attention: the number of free numbers is limited.
