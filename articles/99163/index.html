openSUSE 11.3 "Teal"



Today a new version of the openSUSE distribution has been released.

In 11.3 on DVD and in official repositories, in addition to KDE 4.4.4, GNOME 2.30.1 and XFCE 4.6.1, the lightweight LXDE 0.5.5 environment is also included.

For lovers of thrills, grub2 and the upstart initialization system are included in the official repositories; in addition, the installer provides the ability to install on btrfs.

Significant changes in the graphics subsystem:
  • Sax2 went into oblivion - a utility for configuring the X server, since 11.3 uses X.org 7.5 and xserver 1.8, which can independently choose the appropriate driver for the video card.
  • For systems with Intel, Nvidia and AMD / ATi video cards, the KMS mode is enabled by default.
  • The radeonhd driver has been replaced by radeon.
  • For Nvidia video cards, the Nouveau driver is used by default.


Long-awaited changes in package management system:
  • Zypper now has the ‚Äìclean-deps key that allows you to remove all its dependencies that are not related to other packages with the package
  • Zypper now warns about changing the manufacturer when updating (repository).


For KDE users, Ktorrent compiled with DHT support is prepared; a package installation mechanism is available on request.
for installing packages with codecs or debug packages, the Oxygen-molecule theme for better integration of GTK applications.

GNOME users will be able to try GNOME 3.0 Preview, Transmission with DHT enabled and a significantly improved GTK version of YaST Software Management.

The following innovations may be of interest to administrators:
  • The conntrack system has been added, which includes a kernel module and configuration utilities.
  • MariaDB and MySQL Cluster packages have been added to the official repositories.
  • Xen 4.0, KVM 0.12.3 with patches from 0.12.5 are suggested as virtualization tools in 11.3. as well as the new VirtualBox-ose 3.2.6.


LiveCD images are hybrid, so you can write to a flash drive with the command dd.
