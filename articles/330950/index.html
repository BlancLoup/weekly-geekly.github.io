Online conference DEV Labs JavaScript. June 24

At one of the recent conferences, we promised our participants to no longer mix the subject of Java and JavaScript within the framework of one event and to organize a separate JS conference. Promised - execute. June 24th holivarim exclusively on the subject of JavaScript.

In order not to take away a whole summer holiday from you, there are not many speeches in the program, but everything is strictly in business:

  • Vladimir Sonkin will talk about reactive programming using RxJS and examples of using this library in Angular 4;
  • Ivan Alyakskin, by the example of an application that works with a graph database, shows the advantages of the React + Relay + GraphQL bundle;
  • Orkhan Gasimov will compare the main features of Angular and React to find out which one is better in a given situation.

The conference starts on June 24 at 11:00 Moscow time.

In order to take part in the conference and get access to video reports, you need to register . Participation is free.

To participate in Soft Labs conferences, it is enough to have a computer or mobile device connected to the Internet. No special software needs to be installed. Platform plugin will load in a few seconds.
