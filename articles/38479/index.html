Recover Lost Data

Hello!
This topic is an appeal to the habrasoobshchestvu for help.
When you turn on the laptop with a chkdsk flash drive stuck in it with a usb flash drive, there are cases with folders with long names. I realized this after a similar experience: I created a folder with a long name, put the files in there for half a gig and started the check. The result - the folder is gone, but there is no more free space on the flash drive.
Help please recover data!

UPD:
Here's what they wrote to me on Google. Replies:

(c:\) "r" (recucle restore). A0344f.tmp ( /)

but, alas, there is no such thing, perhaps because there was less free space on the C drive than the volume of lost folders.

UPD2:
Checking for deleted files does not help. CHKDSK hides them in some other way.

Decision:


The problem was solved with the help of the EasyRecovery Pro package.
Everything turned out to be extremely simple:
1. Run the program
2. In the right menu, select Data Recovery
3. Advanced Recovery
4. Select the disk on which we want to recover data.
5. Poke Next and go to sleep :)
After scanning, a list of folders and files appears, which consists of deleted and damaged files. All deleted files are displayed as well, including the nesting of folders, but all the damaged ones are in the Lost Files folder, there we select everything we need and restore. The process of restoring files is rather long, so if there are more files than 1 gigabyte there, then it is better to set up for the night.
