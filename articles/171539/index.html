Plugin to extract data from sites

Colleagues developers have released a plugin that can extract and analyze poorly structured information. The project is focused on a wide range of users, in particular, will be useful to owners of online stores and their customers. What it is and how it works - in the video:



The service is called Convextra . It works as follows: when the plug-in is launched, the program code is called that analyzes the information on the page and navigates through the pages of the site. To go to the plugin uses a special search for buttons to navigate the site or the user can independently specify this button. Within a few seconds, the plugin collects all the information that is on the page in its own table. Further from the table, the user can display the data in a CSV format file, specifying which data should be displayed. Daily use is carried out in the most trivial way: the user drags the plug-in button onto the bookmarks bar and brings it up on the store page. Within a few seconds, the data will automatically be presented in a table, it remains only to save them to a file.

The plugin does not work flawlessly yet, the development is somewhere between alpha and beta stages. So if on some site it did not start, then please do not throw rotten tomatoes: the purpose of the post is not advertising, but feedback from potential users. First of all, the plugin was designed for owners of online stores. Data analysis on the market is very important and one of the most difficult jobs for the store, the plugin helps to collect all the data in a table, from where it will be easier to analyze. Also, the development will be useful to people who want to choose a product or store where it can be bought, while saving part of the budget. Therefore, we will be grateful for the feedback and suggestions - what functions are needed, how would you use this thing.
