Django Dash 2012


This Saturday at 00:00 UTC, the opening of Django Dash 2012 registration will take place: this is one of the most interesting and promising competitions for Django programmers.

It hosts the 48 hour competition in writing a high-quality and useful service / application for the Django framework. This international annual competition will be held for the 5th time.



The rules from last year, in my opinion, have not changed:
  1. Be sure to use the Django framework
  2. No code before the competition, just ideas and notes on paper
  3. 48 hours for implementation
  4. Team of maximum 3 people
  5. Using Git or Mercurial for development
  6. The project must be open source
  7. The implemented project should be installed using pip or buildout
