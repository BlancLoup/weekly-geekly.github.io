  • PHP 7.1.0 - The long-awaited major release of PHP delivered strictly on schedule. Thanks to all contributors and release managers for their work! The new version of the interpreter has improved performance and added a number of new features:

    • The ability to pass null as an argument with the specified type
    • Return value of type void
    • The pseudotype Iterable , which combined both regular arrays and objects that implement Traversable
    • Access modifiers for class constants
    • List () improvements: short syntax and the ability to specify keys
    • Ability to catch several types of exceptions in a single catch block
