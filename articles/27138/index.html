toodoo supported OpenID

Moscow, June 10, 2008. The social network toodoo.ru announced support for OpenID.

OpenID is an open decentralized single sign-on system that allows you to use one login and password on a large number of sites.

Large websites themselves act as OpenID providers. Members of the LiveJournal.com and LiveInternet.Ru blog services can link to their accounts when working with sites that support OpenID.

In June 2008, the list of Russian providers of OpenID was replenished with the social network of toodoo website users (http://toodoo.ru). When working on external sites, users were able to refer to their toodoo profile. In the near future, OpenID authentication will be implemented inside toodoo.ru itself.
