Yellowsn0w - promised soft unlock for iPhone 3G

Well, today the Dev-Team, as promised (and also showed ), put unlock for 3G in Cydia and now locked phones can be used with any operator.
This means, for example, you can now safely buy inexpensive iPhones from the United States from intermediaries and use them anywhere.



Actually, who needs it, everything can be read in the release .
I will highlight the main thing:
  • It only works on modem firmware 02.28.00 which is actually included in update 2.2
  • The program itself is a startup daemon.
  • Available in Cydia and Installer (if someone else uses it ...):

  1. Cydia source: apt9.yellowsn0w.com
  2. Installer repo: i.yellowsn0w.com
