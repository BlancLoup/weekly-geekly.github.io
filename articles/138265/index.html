Document Foundation announces release of office suite LibreOffice 3.5

The Document Foundation has announced the release of a new version of the office suite LibreOffice 3.5. In this version, many innovations have been implemented to improve its use.

So, it is noted that the office suite LibreOffice 3.5 received a number of optimizations in the source code. In addition, it introduced a mechanism for automatic online check for updates. Thus, the user will be notified when a new version of the package. In addition, a number of improvements have been reported in individual applications of the LibreOffice 3.5 package:




Writer
New built-in grammar checker for some languages;
improved typographic features;
interactive statistics calculation window, which updates data in real time;
a number of changes in the user interface.

Impress / Draw
improved tools for importing Smart Art forms and objects from PPT / PPTX
added tools to embed multimedia and color surfaces in ODF documents;
implemented a new display switch for the presentation console;
improved charts;
Added import filter from Microsoft Visio.

Calc
support for up to 10 thousand sheets;
introduced a new multiline input field;
added new features in accordance with the ODF OpenFormula specification;
improved performance when importing files from other office suites;
added the ability to multiple selections in autofilters;
implemented an unlimited number of rules for conditional formatting.

Base
integrated own PostgreSQL driver.
You can download LibreOffice 3.5 office suite from the official project site.
