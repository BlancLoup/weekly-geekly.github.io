APIshnik Yandex.Maps

In the last post, I unobtrusively announced an event, which we ingeniously called ‚ÄúAPIshnik Yandex.Maps‚Äù , during which you can listen to reports on the basic concepts of the second version of the Maps API, communicate in an informal setting with the developers, eat cookies. The API will be held in Moscow on April 26, i.e. next thursday.

I publish now the program of the concert :

  • Alexander Chupakhin, the head of the group of the development of visual components of the API, will tell about the architecture of the Yandex.Map 2.0 API. You will learn what parts the API consists of, how they work and interact with each other.
  • As in API 2.0, you can customize the map and control its behavior, will tell Alexander Zinchuk, a Yandex.Maps API developer. You will learn how to add, delete and move controls on the map, what types of geoobjects you can use, how to configure them.
  • Anton Korzunov, Yandex.Maps API developer, will tell you about graphics capabilities in API 2.0. You will learn what new technologies are supported in API 2.0, how and in what cases it is better to use them.
  • Marina Stepanova, developer of Yandex.Maps API, will tell you about how to place multiple points on the map and how to use the new tool in the API.
  • And finally, I personally will tell you how to show your own map or any other image on the site, as well as how to create tiles and how to display them using the API.
