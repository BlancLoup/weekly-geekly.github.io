Introducing the final release of 3CX Call Flow Designer


This week we released the final version of the 3CX Call Flow Designer voice application development environment.

The application has been in the preliminary testing stage for quite a long time, and during this period has received numerous reviews and comments from the developers. Based on these reviews, we added several ready-to-use specialized components. Previously, these components had to be created manually. The process of creating such components is discussed in previous articles of this blog. Now you can use them immediately, without additional programming.



New components


  • Date Time Conditional - performs various branches in the voice application, depending on the date and time, and for certain DID lines. It does not require the use of C # expressions.
  • Authentication - queries subscriber ID and PIN, processes basic input errors and checks entered values ‚Äã‚Äãin an external data source (database).
  • Credit Card - requests a credit card number, expiration date, CVV code, handles major input errors and checks entered values ‚Äã‚Äãin an external data source (database).
  • Web Service REST - for working with REST web requests using various authentication methods.
  • 3CX Get Extension Status - gets the current status (profile) of the 3CX extension number (including possible ‚Äúoverlays‚Äù of the profile policy), and also checks whether the extension number (user) is busy or free.
  • 3CX Get Queue Extensions - gets a list of extensions (operators) connected to the selected Call Queue, as well as their status: connected / not connected.
  • New session variable session.transferingExtension - transmits the Extension number from which the call was redirected to the voice application (if such a number exists). Used to transfer a call back to an extension number after the application terminates.
  • New functions for working with long numeric arguments: SUM_LONG, MULTIPLY_LONG, DIVIDE_LONG, NEGATIVE_LONG, ABS_LONG.
  • New GET_LIST_ITEM_COUNT and GET_LIST_ITEM functions for working with lists returned by the new 3CX Get Queue Extensions component.

Free 3CX training in the Republic of Belarus


This week we will conduct wonderful training courses on 3CX Phone System Minsk, Republic of Belarus! Courses will be held on Wednesday-Thursday, September 27-28, 2017 at the Hotel Belarus. Training is conducted in Russian.



Courses are completely free for any 3CX partner. Note that similar courses on VoIP-systems of other vendors cost from a few hundred dollars per participant, and we also have a delicious lunch!

The training fully examines the capabilities of the system, as well as prepares for passing certification exams . 3CX certification is required to obtain Preferred Partner 3CX status and place on the 3CX website in the Where to Buy section.

3CX courses consist of two parts: basic and advanced. Among others are considered:

  • initial installation of PBX and 3CX clients
  • firewall setup
  • system update and recovery
  • extensions, remote connections
  • VoIP security

Registration information and detailed schedule:

  • Basic training: Wednesday, September 27, 2017, Minsk, Belarus
  • Advanced training: Thursday, September 28, 2017, Minsk, Belarus

We recommend not to delay registration, since limited number of seats.
See you in Minsk!
