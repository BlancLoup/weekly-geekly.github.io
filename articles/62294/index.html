Opera - linux speed

It has long been haunted by the speed of rendering pages in Linux (and the work as a whole). I mainly use Linux, and when I need to write a double-layer disc from the CloneCD image, then Windows.

I decided to run V8 Benchmark Suite - version 4 on Ubuntu Linux 9.04 and Windows XP SP3 (IE7) in the VirtualBox 2.1.4 virtual machine (which runs from under the same Ubuntu Linux). All Linux / Windows application applications have been disabled. The default desktop environment. Intel video 945, driver 2.7.99.901, default settings X.org. Compiz is disabled.

Windows browser version:
Opera Version: 9.64, Build: 10487
Windows result - Score: 138

Linux browser version:
Opera Version: 9.64, Build: 2480
Linux result - Score: 79.4

The same machine, Firefox 3.0.11 browser:
Windows result - Score: 134
Linux result - Score: 68.8

On native Opera under FreeBSD 6.4 the same cockroaches. I do not pretend to be the reference measurements, but the performance in Linux is almost twice as low. And it provided that Windows is started from under VirtualBox.

UPD : Tested Opera 10:
Windows result - Score: 135
Linux result - Score: 95
