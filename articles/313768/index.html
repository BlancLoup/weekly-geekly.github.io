Adobe has released an emergency update for Flash Player.

Adobe has released an emergency update APSB16-36 , which fixes the critical vulnerability CVE-2016-7855 type Remote Code Execution (RCE). Vulnerability is at the stage of active exploitation by hackers. As in the case of other vulnerabilities of Flash Player, CVE-2016-7855 allows attackers to remotely execute code in a web browser using a specially created Flash object.



The vulnerability itself was discovered by a group of Google antivirus analysts and was observed in use in directed cyber attacks on users of Windows 7, 8.1 and 10. The above table shows the current versions of Flash Player, in which the vulnerability was closed. We recommend that users update their version of the web browser to the current one. Updates are available for Windows, OS X, Linux and Chrome OS platforms.
