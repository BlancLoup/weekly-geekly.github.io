Windows SSH support by Microsoft

This is all initiated by the PowerShell development team, which is periodically actively kicked on this topic. There will be both an integrated PowerShell ssh client and an SSH server under Windows. In addition, the interaction between the windows-machines for remote control via PowerShell is also planned through ssh. It is planned to implement this business on the basis of the OpenSSH project, to which the necessary patches will be sent.

Before that, there were attempts to implement an SSH client in PowerShell V1 / V2, but they were never presented to the public.
