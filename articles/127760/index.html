Virtualized ATMs or ATMs without a heart are not wrong

Diebold introduced a prototype of the world's first virtual ATM, using virtualization technology. Using this technology will improve the reliability and security of the ATM, as well as simplify the management system of self-service channels. This is stated in the press release of the company.

Diebold's innovative solution was presented at the VMworld 2011 exhibition and conference in Las Vegas on August 29th. Virtual ATM developed in conjunction with the company VMware.

If previously for 20 years the basis of ATMs were computers built into them, then thanks to the virtualization of the self-service system, the ATM acquires the ability to work without an embedded computer and is attached to a single computing center. Thus, the physical components of a single server carry out all the computational operations of many virtual ATMs.



Consolidation allows you to create a single secure data center and increases the level of control and security of IP addresses. Virtualization provides a more efficient server operation and unification of ATM management, which increases operational efficiency, provides faster error recovery and creates conditions for accelerated software updates and service distribution. Simply put, virtualization leads to lower maintenance costs and increases the period of productive operation of ATMs.

In addition to virtualization, the Diebold demo terminal, presented at the VMworld 2011 exhibition and conference, is equipped with a biometric identification device for enhanced security and NFC technology (contactless technology) for greater user convenience.
