Now Multilingual!

"He (Rincewind) could shout" Help "in fourteen languages, and beg for mercy on forty-two."
Eric Terry Pratchett

Not so long ago (about four months ago), the guys from MediaStudio offered me to make a Wordpress plugin that would allow to blog or create websites in several languages.
The result is this - WP Multilingual . To the requested functionality just to blog in several languages, I added some more (I think all the same useful features), briefly summarizing what WP Multilingual can do:
  • Manage Posts and Pages in multiple languages
  • Customize the Wordpress localization for the admin part (this is useful if you have a blog running several people and they are all different nationalities) in their own way. To whom is Ukrainian, who is Russian, to whom English is not changed.
  • Customize Wordpress localization for themes - the dependence of the theme localization on the selected language is introduced
  • Manage category / tag / blog link translation

The program interface is localized in Ukrainian, Russian, Greek (thanks to the design bureau ‚Äúmake‚Äù and Irina in particular), French and Italian.

While there are several ideas for development - one of them is proposed by Mark Gosh and consists of adding the possibility of autotranslation using for example the Google Translate service - which will be implemented as there is free time.
The plug-in's temporary page is at WP Multilingual .

I will be glad to hear practical comments and suggestions.
