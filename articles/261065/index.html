3CX WebMeeting 8 web conferencing solution is already available in Australia and Oceania

The beta version of our new product 3CX WebMeeting 8 is already available to customers in Australia and New Zealand!

3CX WebMeeting 8 adds new features that make it an excellent platform for web and video conferencing based on WebRTC technology.

The following new features have been implemented:
  • Ability to record a web conference and download it in mp4 format.
  • Remote control of computers on the Windows platform (requires installation of the Google Chrome extension).
  • Significantly improved video quality and optimized Internet channel.
  • Now you can conduct surveys of participants.
  • Ability to transfer web conferencing to another participant.
  • The possibility of anonymous participation in the conference.
  • Choosing a venue for a web conference - you choose the MCU closest to you for the best video quality.

Web conferencing location (MCU)


The server we run supports MCU video mixer located in Melbourne, Australia. This layout greatly improves the quality of the video for users from Australia and New Zealand. If you plan to use 3CX WebMeeting in the European Union or the United States, select the location of the web conference (Create Meeting> Options) in the European Union or the United States, respectively. You can also set the default web conferencing location in the system administration interface. If you leave the default location in Melbourne, there may be a deterioration in video quality for users in the European Union and the USA. Servers in the European Union and the United States will be transferred to Webmeeting v8 in the next few weeks.

Web conferencing is now possible without 3CX Phone System


The good news is that 3CX WebMeeting is now available offline from the 3CX Phone System.

Three versions are offered:
  • 3CX WebMeeting is a standalone version. Meetings are organized through a web portal.
  • 3CX WebMeeting for 3CX Phone System - version integrated with 3CX Phone System. Meetings are organized from 3CXPhone.
  • 3CX WebMeeting Server is a local version of 3CX WebMeeting.
