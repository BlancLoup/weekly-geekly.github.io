HideMyAss Proxifier

Hey.
I want to talk about my small but useful extension for Chrome.
Surely many are in a situation where the provider or administrators are blocking some resources.

Not so long ago, I was in such a situation: administrators at work blocked access to YouTube, which gave impetus to the creation of extensions. The solution we all know is TOR or numerous web proxies. Naturally, the second option in this case is much more convenient. But attempts to find a proxy, allowing video streams to pass through themselves, were not crowned with success. And then I learned about HideMyAss.com , which can correctly display videos directly from the YouTube site. But the video inserted into the pages of other sites, while still view using this service can not. Making it possible to view the embed video was the original purpose of the expansion. Later I implemented some more useful features for convenient work with HideMyAss.

What is now


  • the ability to anonymously view and download virtually any (except inserts via 'iframe') YouTube video embedded in web pages. At the same time, the player is replaced with a block with three icons that provide the ability to view and download videos via HideMyAss, as well as through a standard YouTube player (just as it would have looked without the use of an extension);


  • quick anonymous opening of any page via HideMyAss. To do this, simply click on the extension icon and enter the desired url.
  • quick anonymous opening of the current tab in two clicks. Useful in cases when the page you have opened is not loaded for any reason.


  • Anonymous opening of any link on the page through the context menu.
