Torrent stream for ubuntu

Open testing of Torrent Stream software on Linux / Ubuntu OS has begun!


The world's best software application for streaming / online playback of video and audio content via torrents, now available for Ubuntu users

Detailed information about the Torrent Stream System and the software being released under this project can be found on the website torrentstream.org

Everyone who is already familiar with the features and capabilities of Torrent Stream and wants to take part in
testing or just wants to know what products are currently available
for Linux / Ubuntu, you are welcome under cat.

The test builds for Torrent Stream for Ubuntu include:

TS Engine - The main software product that provides all the functions and capabilities of the Torrent Stream technology (multimedia streaming control engine)

TS Engine is the basic core of the Torrent Stream platform and can be used, through the API ( TS Engine API ), by any developers to create their own products, integrating the functionality and capabilities of Torrent Stream into them.

TS Player - Multimedia player (modified and improved version of the VLC player), with the ability to play video and audio online, via torrent files, without waiting for content to load.
Comment: Now you can run and play torrents in the desktop player as normal video and audio files, without using different torrent clients and without waiting for downloads. Also, if the user knows the content ID, then it can also be used as a source, specifying it in the corresponding field when specifying the source (Content ID, is essentially an analogue of magnet links, only within the TS system).

The inclusion of the desktop version of the player in the assembly is due to the fact that it is necessary to use the advanced settings of the browser-based web player (running on the basis of TS P2P Multimedia Plug-), since most settings of TS Player are synchronized with it. For example, if you want to use a video accelerator or reassign hotkeys for a web player, the user will need to go into the TS Player settings, set these settings and restart the player page (or better web browser), and the next time you start it, all the new settings will be work and in the web player.

TS P2P Multimedia Plug-in - Browser Plugin (built on the basis of the VLC Plug-in)
Allows you to use the world's most functional web player on all sites using Torrent Stream technology, as well as on popular video hosting sites such as You Tube, etc., and with full support for all content being played with the bittorrent protocol.

Comment: Probably for those who installed and used Flash exclusively for watching videos, this will be a very good alternative, given that in addition also bittorrent will act as an accelerator, ensuring the stability of the stream and the maximum download speed for a custom Internet channel.
