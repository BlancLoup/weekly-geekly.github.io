Erlang plugin for IntelliJ IDEA version 0.4

Erlang plugin for IntelliJ IDEA version 0.4 has been released today - it turns IDEA into a fully-functional IDE for Erlang for free.

Innovations

  • Integration with Rebar including running Eunit tests through Rebar
  • Erlang type support: navigation, autocompletion, rename, etc.
  • External ( Shift + F1 ) and local ( Ctrl + Q ) documentation for OTP functions and modules
  • Tip options fukntsii ( Ctrl + P )
  • Go to symbol ( Ctrl + Alt + Shift + N )
  • Improved autocompletion for BIFs (includes BIFs for OTP modules, for example, lists: member / 2 )
  • Darcula dark scheme support and syntax highlighting enhancement
  • And much more, see the full list of changes.

Since the release of version 0.1 (since July 2012), the changes are even greater. If someone tried the very first build and was not satisfied, now is the time to try again.
