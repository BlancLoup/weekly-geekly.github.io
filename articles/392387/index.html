Today , new versions of Yandex.Maps mobile applications for iOS and Android have been released.

The most important innovation is that the maps of all regions have noticeably decreased in size. For example, the offline map of Moscow now occupies only 144 MB instead of 1.9 GB, the offline map of St. Petersburg and the region - 231 MB instead of 3.2 GB.

Reducing the size is achieved due to the fact that Yandex has finally switched from raster maps to vector maps, so now it‚Äôs not necessary to save several variants of the same map in different scales.

In addition, the new version of the maps implemented offline search - something that users have long asked for. On maps without an internet connection, you can search for addresses and organizations: cafes, shops, ATMs, pharmacies, gas stations, and so on. The cards will show the transport schedule, the address of the organization and provide details: opening hours and telephone. For hotels the approximate cost of rooms is indicated, for restaurants - the average bill.



The developers write that they added several more functions: first, changing the angle of the map (you need to run two fingers across the screen), second, constant display of street names in a visible place, even if traffic jams are on or a route is laid.
