4 ways to free download daily and step by step information from US stock exchanges

In the original, there were 6 ways, but the author of the translation could not force some of them to work, and therefore does not see any meaning to mention.

A few months ago, I made a post about where to find retrospective data on US markets at the time of closure, and listed 10 sites that provide this information for free. ( 10 ways to download historical stock quotes for free ).

Step-by-step and daily information is also in the free network access. Today I will talk about four places where you can download historical data. Some sites are popular, some you've probably never heard of.

Let's start with the most famous


Google finance



In Google Finance, data is available freely for some markets. A full list can be found here .
Data can be obtained for a set of frequencies, the lower limit is the minute. Since 2011, the Google Finance API is not recommended for use, but continues to be available. (details here )

Link format

www.google.com/finance/getprices?i=[PERIOD]&p=[DAYS]d&f=d,o,h,l,c,v&df=cpct&q=[TICKER]
[PERIOD]: Period of data in seconds;
[DAYS]: The requested amount of data in days;
[TICKER]: Exchange code.

Example

http://www.google.com/finance/getprices?i=60&p=10d&f=d,o,h,l,c,v&df=cpct&q=IBM

Yahoo finance



Like Google Finance, Yahoo allows you to download data from several exchanges. List of supported here .

Link format

chartapi.finance.yahoo.com/instrument/1.0[TICKER]/chartdata;type=quote;range=[DAYS]d/csv

[TICKER]: Exchange Code;
[DAYS]: The requested amount of data in days.

Example

http://chartapi.finance.yahoo.com/instrument/1.0/GOOG/chartdata;type=quote;range=1d/csv

Stooq



There are two periods to choose from: hourly and five-minute. The big advantage is the ability to download all the data in one compressed file.

Link to select the date

http://stooq.com/db/

If you do not need more accuracy, this site may be the best option.

Finam



Finam is a Russian website, providing data on stocks, futures, ETF and Forex exchanges. The main advantage is that you can download several months of step-by-step data. The problem is a small number of shares.

Download link

http://www.finam.ru/analysis/profile041CA00007/default.asp
