Extension for Firefox :: Habra comment editor

I am glad to present to your attention my first extension for Firefox - Habrahabr Comment Editor.





Requirements


Firefox: 3.0 - 3.5

Description


The extension adds edit buttons to the comment input field.
Able to frame the selected text in the comment field in the following tags (support for "hot" keys implemented):
Button Hotkey Result
B ctrl + b <strong> text </ strong>
I ctrl + i <i> text </ i>
U ctrl + u <u> text </ u>
Quote ctrl + Q <blockquote> text </ blockquote>

And also you can insert images ( IMG button, hot key ctrl + P ) and links ( URL button, hot key ctrl + L ) with just one click! Just today for 9.99!

You can download the whole thing from here or directly from here .

Future plans

  • Add more tags
  • Make a settings menu through which you can customize the display of only the buttons you need
  • Add parsing emoticons (for all comments)
  • And maybe implement something like a WYSIWYG editor

Of course, I will be glad to hear additions and criticism in the comments :)

UPD. At the request of habrovchan added tag <blockquote> . The implementation is so far only in the form of another button Quote (and hot key ctrl + Q ). The updated version is available at the links above. The update process is the same as the installation process - the old version will simply be replaced with a new one.
