Window shop in Vivaldi 1.13.997.3


Hello!

Many days have passed since the previous build, and you quite rightly await explanations. Do not worry: everything is normal, nothing force majeure has happened - there is a normal development process, which is never smooth. And in this case, two circumstances came together at the same time, which you can learn about under the cut.

So, the first circumstance is the update of the Chromium core to version 62.0.3202.63. We had to restore quite a lot after this ‚Äúimprovement‚Äù. The second circumstance was also the Chromium update, but the future. From the plans of the developers of this kernel, we learned that some functions that we used in our work will soon be disabled. Thus, we had to urgently rewrite, in particular, the system of working with application windows, which would allow us to avoid future problems, and also, in some cases, would increase the speed of the browser. You can tell us about all the changes that you notice in the process of opening windows or working with them - we hope that we have been able to significantly improve this component of the browser.

Also, due to big changes in the browser code, we have disabled the auto-update option for now - you can do it manually by installing a new version of the assembly over the previous one, but be prepared for unexpected browser errors. In general, we recommend installing this version separately for now.

Well, now - about the new features that also exist.
