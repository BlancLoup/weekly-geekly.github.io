Several useful services. Continuation

Continuing the topic of the previous post Several useful services , I want to share new links.

Content export
  • rss-script.ru is a service for inserting RSS feeds into a site. For his work, you only need to generate a script on the site and paste it yourself. There are several settings that allow you to change the look of the content.
  • twitterfeed.com allows you to post RSS feeds to Twitter, Facebook and LinkedIn. A fairly simple service, there are no additional settings, but there is a click counter.
  • ifttt.com is a multifunctional service that allows you to link various accounts ( description )

Sandboxes
You can add dabblet.com , jsbin.com (added by Jekyll ) to the previously mentioned cssdesk.com and jsfiddle.net . There is also a sandbox for SQL - www.sqlfiddle.com (added by volinrok ).

Sharing text fragments / code
In order to share a piece of text or code, you can use the well-known pastebin.com , and you can use its alternatives:
  • gist.github.com
  • pastecode.com
  • tinypaste.com
  • paste.ubuntu.com
I like Gist on gitHub the most. Firstly, all text fragments are attached to your account; secondly, you can create several linked Gists; and thirdly, there is the possibility of commenting.

There is also a service ideone.com :
Ideone is more than Pastebin; This is an online compiler and debugging tool that allows you to compile and run code written in more than 40 programming languages.

Code highlighting
  • sc.me (12 languages, result in HTML or CSS)
  • Syntax Highlighter (more than 80 languages, no copyright, preview, code generation in HTML format) (added by Derailed )

Communication with site visitors
  • Hypercomments - a new look at the comments (the ability to comment on a separate piece of text)

For familiar comments, you can use the widgets Cackle , IntenseDebate , Disqus .

Social authorization
In addition to the well-known Loginza and uLogin, there are other widgets for authorization through popular web services.
  • LoginRradius
  • JUMP (Janrain User Management Platform)

Time management
  • www.toggl.com
  • www.tahometer.com
  • www.timedoctor.com (only paid account)

Testing the site load
Both services are paid.

  • www.neotys.com
