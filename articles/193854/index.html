3CX Voice Application Designer 12 - a new version of the voice application editor has been released

The release of the 12th version of 3CX Voice Application Designer.



The new version of the editor works with the phone 3CX Phone System 12. A distinctive feature of 3CX VAD is that it allows you to create applications by drag & drop - in other words, applications are created from ready-made components, and the user does not need special programming knowledge.
