Skype 5.5 decrypted for reverse engineering


On March 25, the decrypted Skype v. Binary file was published. 5.5. Now it is possible to run Skype in the debugger, which could not be done earlier, since the original version was strongly obfuscated and had protection against debugging. The goal of the project is to create an open source Skype.

A message about this appeared in the blog of Efim Bushmanov, who deciphered the Skype protocol version 1.x / 3.x / 4.x a year ago and published a working version of the program.

Post in his blog:
skype-open-source.blogspot.com/2012/03/skype55-deobfuscated-released.html

Download link (the archive contains the Skype binary file and the IDA database file): github.com/downloads/skypeopensource/skypeopensource/skype55.zip

Interesting discussion on this topic: news.ycombinator.com/item?id=3753155
