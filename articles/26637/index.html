Local Area Network via Television Cable

The company D-Link has released an interesting device Coax Ethernet Adapter Kit, with which the local network can be created using a conventional coaxial television cable. And without interference for its main functions.

In fact, the Coax Ethernet Adapter Kit is two adapters with an Ethernet port and a connector for connecting a television cable. One of the adapters is connected to the router, the second is connected to a PC or any other device with an Ethernet port.

The cable broadcast signal does not affect the product from D-Link, since it uses the range of 800-1500 MHz for operation.

The cost of the Coax Ethernet Adapter Kit is $ 200. This set of adapters will be available in the middle of summer.
