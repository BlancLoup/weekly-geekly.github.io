Compiz 0.9.0 release


Compiz 0.9.0 release is released today - a composite window manager for the X Window System. Allows you to customize the appearance of windows and effects for them.
Almost a year has passed since the last stable version.

Major changes in the latest version:
  • Completely rewrote the code in c ++ (was C).
  • Render was split into 2 plugins: OpenGL and Composite
  • Reworked plugin that controlled animations.
  • New API for developers (all plugins rewritten)

From the main innovations:
  • Plug-in for automatic change of desktop wallpaper
  • Plug-in similar in functionality to Aero Snap (This already exists in KWin)
  • Replacing automake with cmake
