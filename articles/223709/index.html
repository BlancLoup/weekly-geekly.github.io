Briefly about the service:

This is such a teammate for teamwork. Moreover, it positively differs from any skype and vibers with wonderful features, such as:

The team members also did not immediately understand why this is cool, but now it has become clear to everyone that this is more convenient.
Each team member receives notification only of those messages that are addressed specifically to him. At the same time, archives of messages are available in all channels and can always be re-read.

For example, we can discuss with the Android-programmer in the chat about the Android-client any questions, during the discussion of which there is a question to the developer of the server part. At the same time, I made a separate chat, which I called #meetingminutes, where I briefly write everything that we agreed upon as a result of such discussions. Thus, we do not need to re-read the message history to remember what decisions were made.
