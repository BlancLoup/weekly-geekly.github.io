Meet Lingvo for Android

Despite the fact that I generally sympathize with Android (having a Symbian phone at the same time), I understood one of its significant shortcomings compared to iOS, with which it is usually compared. There is no Lingvo for it! But, if you read the name of the topic, then you know that now you can safely take an Android phone *. So, after the versions for Windows , Mac OS X , iOS , Symbian, Windows Mobile and even e-books , we present to your attention Lingvo for Android .

For the most determined there is this

And for those who need to think first, please under the cat.

There is, it appears, the application for $ 2.99. Its main task is a convenient display of dictionaries purchased specifically for it. At first there are no dictionaries, then we download them (there are dictionaries in Lingvo , which can be downloaded for free ) or we buy them.



The application generates language directions from them and generates a vocabulary in a direction selected by the user. There is, of course, a search string that briskly responds to the input of characters by scrolling the vocabulary to the desired word and guesses the desires of the user who allowed the occult word.



Having found the right word, the user can see how it is translated, see examples of usage and look at the table with the forms of this word.



All this is to some extent customized, where do without it.
