Smallest pc

CompuLab has released the fit-PC2 , which is currently the smallest PC. Its dimensions are only 115x101x27 mm. This kid is running OS Linux (Ubuntu Linux 8.04). However, models with preinstalled Windows XP are also available.


fit-PC2 is built on the Intel Menlow platform. This PC has an Atom Z530 processor, 1 GB of RAM, a 160-gigabyte hard drive, a gigabit Ethernet card, Wi-Fi, six USB 2.0 ports, and a miniSD flash card slot. There is also a mini PCI Express slot. Optionally, fit-PC2 can also be supplied with an SSD drive.


Coolers in fit-PC2 is not provided. Accordingly, he does not really make noise. In addition, the novelty consumes only 6 watts.

The cost of fit-PC2 with a 1.1 GHz Z510 processor without a pre-installed OS and a hard disk is $ 245. With the same processor, Ubuntu 8.04 and a 160-gigabyte SATA HDD, this PC costs $ 300 already. The most expensive equipment - a Z530 1.6 GHz processor, Windows XP and a 160 GB HDD - will cost $ 400.
