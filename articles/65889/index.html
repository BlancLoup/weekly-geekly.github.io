Timer To Light



Designer Jasper Hou came up with this modern version of the hourglass for the kitchen. Timer chip To Light - LED lights. While the spring is turning the timer - the LEDs are lit, as soon as the time runs out - extinguished. And, judging by the different brightness in the pictures, there is a dependence of the saturation of color and time on which the timer is set.

Hmm ... if you think about it, this thing can be used not only during cooking. Several timers will fit for the birthday party, and to give a signal ...
