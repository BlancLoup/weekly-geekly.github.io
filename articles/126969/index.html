ATP "Pravo.ru" for iPad: more convenient, cooler and tastier

In mid-December, we presented a mobile version of ATP Pravo.ru for iPad owners, which allows working with federal and regional legal documents. Since then, the application has been installed by 64,800 users, as a result of which the app's even got into the top free apps in the AppStore.



We thank everyone who appreciated our efforts. Motivated by such demand, we prepared an update of the ATP application Pravo.ru for the iPad. Now it is still more convenient, cooler and tastier!

And still free for download in the AppStore



What's new?



Document update




In the new version of the application "Pravod.ru" for iPad 3 types of updates are available. The user can update the entire database by selecting the appropriate option on the main page. Or you can separately update the category of documents, for example, federal legislation. Also implemented update one selected document.


Document search




The search mechanism remains the same. On the main page, you can specify the details of the document or the search word. Now, if an application is present in the output of the results, the system specifies the ‚Äúparent‚Äù document, that is, the law / decree that this application enacts.



The documents viewed are marked in the list so that the user does not open the same thing several times. This feature works within one session, with the new search condition, the documents will again be highlighted as unread. If the database does not contain the required document, the user can send a request to add it. Feedback works quickly and the document will be available soon.



Work with document




The new version has corrected the display of sections of the document and added information about revisions. So now all the chapters and sections are displayed, and in the header of the document all the Federal Laws that have made changes to the document being viewed are indicated.

The text of the documents appeared links to related regulations. In other words, the user can go to the mentioned PPA in the text under study and see what they refer to.
