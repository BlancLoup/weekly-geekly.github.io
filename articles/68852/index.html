Debian GNU / Linux 5.0.3 released

On September 5, 2009, the third stable release of Debian GNU / Linux codenamed "Lenny" was released, which traditionally includes many fixes, more about which can be found on this page.
It should be noted that updates affect important packages such as apache2 , dhcp3 , bind9 , mysql-dfsg and others. ISO images will get on the mirrors in the coming days, while you can upgrade with apt now.
I recommend system administrators to use the debian-security-announce mailing list, so you will always be up to date with the latest updates.
