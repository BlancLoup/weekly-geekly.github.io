Ext JS 4 released

Good news - today a new version of the wonderful framework has been released.
I can not post a topic link, so I decided to share this way.
Read the overview and download the library here: http://www.sencha.com/products/extjs/
Among the innovations are a sandbox for running multiple versions of Ext JS in one application, a reduced size of scripts (from 100 kb) required for work, improvements in the appearance of graphs and forms, layouts, new features for grids, updated and significantly extended documentation. The changes also affected the class system and data management, and very significantly.
Also, new examples have already been added, which can be viewed in the corresponding section .
