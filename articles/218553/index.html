Jump Start on April 8th. Data Center Virtualization with Windows Server 2012 R2 and System Center 2012 R2. Configure virtualization hosts

Dear Colleagues!

We bring to your attention a series of online events dedicated to data center virtualization using Windows Server 2012 R2 and System Center 2012 R2 . As part of these online events, the key Windows Server and System Center technologies used to deploy, manage, and maintain the physical and virtual data center infrastructure are discussed in detail. If you are using Hyper-V in Windows Server 2008 / R2, you will see a huge amount of changes that have occurred in the Microsoft hypervisor with the release of Windows Server 2012 / R2. If your organization uses VMware, you can evaluate the capabilities of the latest version of Hyper-V, as well as compare the technologies of Windows Server and System Center with VMware vCenter and vCloud Suite solutions.

Jump Start April 8: Data Center Virtualization with Windows Server 2012 R2 and System Center 2012 R2. Configure virtualization hosts

In the first online event, after a brief overview of the Microsoft virtualization platform, we will focus on configuring virtualization hosts: from operating system deployment, to connecting and configuring existing storage systems, building storage systems based on Windows Server 2012 / R2, and fine-tuning network components using Virtual Machine Manager.

There will be many demonstrations, connect on April 8 from 12:00 Moscow time. The player with the broadcast will be available on the specified page. Registration is not required.

Waiting for you!
