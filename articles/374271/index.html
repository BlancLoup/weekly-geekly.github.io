Wanhao D7 Photopolymer 3D Printer Update: Version 1.5



Wanhao continues to refine the budget photopolymer printer Duplicator 7 or D7. The current version - Wanhao Duplicator 7 1.5 is different from the previous reworked mechanics.


Previously, in Duplicator 7, in versions up to 1.4 inclusive, the carriage of the vertical axis, to which the platform is attached, was moved with the help of a trapezoidal screw along a profile guide. The new version uses two spaced cylindrical guides, and the carriage is now made of aluminum instead of sheet steel.




This solution is designed to minimize the transverse oscillations of the carriage, which adversely affected the print quality.
