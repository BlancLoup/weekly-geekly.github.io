HD Photo for Photoshop plugin released

Microsoft has released the first version of Photoshop's HD Photo plugin, with which any HD Photo format (.wdp or .hdp) can be opened, edited and re-saved in this graphical editor. The old beta version of the plug-in expires on December 31, so release 1.0 is required for installation.

The HD Photo standard used to be known as Windows Media Photo, and later received another name JPEG XR, under which Microsoft is trying to promote the new format as an international standard through the Joint Photographic Expert Group (JPEG).

The format of the new generation HD Photo was created specifically for saving digital photos, it supports up to 32 bits per pixel and has the characteristics of RAW formats of digital cameras. This standard should replace the outdated JPEG and RAW formats.

Plugin 1.0 for Adobe Photoshop CS2 and CS3 can be downloaded from the official website: for Windows and for Mac OS X.
