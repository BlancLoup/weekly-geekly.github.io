A small review of Ubuntu MID Edition 8.04

I present to you the free translation of the article about the new version of Ubuntu for Internet tablets.

2 months after the release of Ubuntu 8.04 and a month after the first presentation of Ubuntu Notebook Remix , Ubuntu MID Edition 8.04 is finally released. This is a new product from the Canonical family of Ubuntu, designed specifically for Internet tablets (MID). Ubuntu MID Edition is a slightly truncated version of Ubuntu 8.04 for desktop machines, most of the software packages are specially redesigned for best performance on the Intel Atom processor and on mobile peripherals. This was made possible by the Moblin project.



A certain number of packages have been changed, deleted, added specifically for maximum performance on Internet tablets or UMPC. For example, one of the new programs was a web browser on the Gecko engine with the ability to scale and specially designed for small screens. It is called ‚Äú MID Browser ‚Äù.



The package includes: Pidgin for instant messaging, Cheese for working with a webcam, Claws Mail , as an email client and Moblin Media for playing video and sound. The package also includes games such as Neverball and Frozen Bubble . For reading books on the Internet tablet, Ubuntu MID offers FBReader in one of the standard packages. Also in the kit comes a lot of small but important applications, for example, a clock, a calculator and an address book.




Although the interface looks a little different, Ubuntu MID still uses NetworkManager for network connections, and XServer 1.4.1 prerelease and Linux 2.6.24 manage the system from the inside.



Ubuntu MID uses much of the Moblin project, for example, parts of the user interface, browser, multimedia system, and applets. All this software is undoubtedly optimized for the Intel Atom and Menlow platforms. Several Menlow-based UMPCs were presented at CES in January of this year.



Canonical provides various Ubuntu MID Edition 8.04 versions for different purposes. For example, if you want to simply try this system in action without having an Internet tablet, then you can download a special option for installation using KVM (Kernel-based Virtual Machine). The company also provides special builds for McCaslin- and Menlow-platforms. Canonical used the Samsung O1U, based on the McCaslin platform during development, and Menlow support came later. However, the presented images are intended, rather, not for end users, but for manufacturers and developers. Apparently, significant changes will be required for the system to operate on an unsupported Menlow device. Nothing has been announced to support the recently announced NVIDIA Tegra or VIA Nano . The KVM version of Ubuntu MID 8.04 is 471 megabytes.



Ubuntu MID Edition 8.04 can be downloaded from Canonical servers .

From myself I would add that I was interested in, that when creating a distribution kit, they pushed away from the desktop version, thanks to which, I think there should not be big problems with the number of programs, as with the same Nokia Maemo . However, the interface, the visual theme of Nokia seems at times more thoughtful, on the Ubuntu MID screenshots above, the interface looks childish and alyapov. Separately, I roll my nose from the fonts used. So, the initiative is good, but much will have to erode, IMHO.
