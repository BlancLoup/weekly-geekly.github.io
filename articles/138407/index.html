Oracle announced a significant increase in performance of MySQL Cluster

Oracle is trying to regain lost karma. The latest release of MySQL Cluster 7.2 GA was released under the GPL, and the 8-node MySQL cluster processes 1.05 billion queries per minute (17.6 million per second).

The developers also claim a 70-fold increase in performance on complex queries (JOIN), in fact, in the 7.2 back end, the Memcached module is integrated and the native Memcached API is implemented.

Thanks to Memcached, now MySQL Cluster 7.2 GA can also receive requests through the NoSQL C ++ NDB API.
