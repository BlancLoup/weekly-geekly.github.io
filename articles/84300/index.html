Maemo + Moblin = MeeGo. What will be?

Nokia and Intel announced the creation of a joint project of the open mobile platform MeeGo , based on maemo and Moblin, respectively. So far there are few details (in press releases, as always, mostly loud words), but it is already clear that this is a new attempt to make a single system that could work on mobile (and not only) hardware of different types - from communicators and internet tablets to netbooks and televisions. From the technical details of the architecture, I would point out using Qt as the main GUI library (which is expected, for Nokia); at the same time, it promises that Gtk will be supported for compatibility with a wider range of applications. In general, the battle for a unified mobile environment continues: Apple and Google will not be bored.
