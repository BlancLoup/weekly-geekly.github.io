Linux Mint 9 "Isadora" released

The final release of Linux Mint 9 "Isadora", one of the most user-friendly Linux distributions based on Ubuntu "Lucid Lynx" 10.04 LTS , was released today. As usual, the Mint distribution preserves compatibility with the original Ubuntu, while adding additional functionality: a modified set of software, its own utilities mintTools (mintMenu, mintInstall, mintUpdate, mintBackup), a modified theme, and much more. Like Ubuntu 10.04, this release is based on the 2.6.32 kernel, and the desktop environment is GNOME 2.30.

Here are some new releases:
  • New program manager (Software Manager) with community reviews, the ability to install and uninstall packages in the background, interface improvements
  • New backup tool
  • Improvements in the menu: editable menu items, transparency, saving a list of favorite programs
  • Desktop settings: instant application of changes, additional options
  • Improve appearance and usability
  • System improvements: added installer from under Windows, Usb-creator, local repository and Gnome-ppp, new commands for Apt
  • Changes in the project: the new site, distributions based on KDE, Xfce, LXDE and Fluxbox become official, etc.

Details can be read both in the original and in Russian .
