Windows Mobile 6.1 officially announced

At CTIA Wireless 2008, Microsoft officially announced Windows Mobile 6.1. The new OS includes improved navigation and management, an increased level of protection of confidential business information, as well as a number of opportunities to save user time. The new built-in Internet Explorer Mobile allows you to browse web pages and video in full-screen mode, and also supports the standards H.264, Adobe Flash and Microsoft Silverlight. The first smartphones on this OS are expected to go on sale by the end of this year.
