Another build Vivaldi 1.0.303.37 - Beta Candidate

Hello!

We present to your attention the next prebet assembly, in which we fixed some critical errors.

Sketches of pages and work with a disk

After the release of the previous build, there were many complaints about the browser‚Äôs too active work with the hard disk. As it turned out, the problem was in the process of creating page thumbnails: the browser created new copies of the thumbnails each time the Express Panel page was accessed. Now this bug is fixed.

In order to completely get rid of the traces of this bug on your computers, you can follow these steps:

  1. Close browser
  2. Login to browser user profile
  3. Delete Top Sites File

Another way is to cleanly reinstall the browser, deleting the profile and switching to a new build (if you do not have critical data saved in the browser).

We also fixed a mistake when working with Facebook, added a few fixes to the Mac version and did a good deed for Finnish users by adding Finnish to the list of localizations.
