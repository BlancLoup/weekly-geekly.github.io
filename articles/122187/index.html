Software


Those who make software for gaming devices Razer - well done. Well, almost always great.



Key setting. If everything should be very clear with the mouse (setting up the main keys and scrolling), then with the keyboard, hoo-dah! "Zabindit" you can absolutely any combination of any key. You can hang macros, change profile, run the necessary programs. Believe me, everything is very convenient.



Customize profiles. I do not see the need to explain something here. Under each application / game you will have your own profile, with a specific setting. Profile changes - all settings change (even the color of the backlight). The profiles still cannot switch automatically (for example, when starting wow.exe, the software tightens the World of Warcraft profile, and refuses to switch back when exiting), confirmed by Razer. A huge stone in the garden for developers!



Macros. Expansion for those who like to automate part of the action. You can record any key combination by setting the required delay between clicks in fractions of a second. Any macro can be hung on the "bandage". Personally, I do not use. At all.

By the way, on the Razer Anansi there is an opportunity to record On-The-Fly macros. To do this, the engineers introduced a special keyboard shortcut: Fn + Alt (M).



Backlight settings. Both devices are capable of displaying a rich color scheme. But the mouse does not want to display some very dark shades, it goes out immediately. In addition, sleep and brightness settings are available wirelessly for the mouse.



And of course, rich sensitivity settings (general or X / Y separately). Change the level of DPI (dots per inch) "on the fly" and the acceleration of the mouse cursor when moving.
