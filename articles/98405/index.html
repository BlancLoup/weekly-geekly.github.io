Install the official update of Android 2.1 on the problematic HTC Hero

Most recently, on June 29, the HTC Hero finally received an OTA-update to Android 2.1. According to agreements with Google, all HTC phones are updated only by air (technology Firmware Over The Air), and RUU-packages no longer apply.

Some owners of the HTC Hero have not been able to upgrade to Android 2.1, faced with a problem that they managed to call the ‚ÄúBermuda Triangle‚Äù. In fact, the phone could not exit recovery mode and resume the installation (the problem and the recovery process for Hero was described here ). The problem is whether it is modified, gray or other phones can only guess.

Thanks to the operational work of the resource Shipped Roms, a relatively honest way of recovery has appeared. If your Hero failed to upgrade using the OTA update:
  • Reboot your phone and go into bootloader mode (Power + Back / Volume Down).
  • Select the Fastboot mode and connect the phone to the computer.
  • Download the RUU package via the link and run the update.
If the update via USB failed, the phone can be updated using an SD card, for this:
  • According to the instructions , unzip heroimg.zip from the new firmware version.
  • Copy the file to the root of the SD card and turn off the phone.
  • Start your phone by pressing the Power + Back keys.
  • Then click Volume Down and follow the instructions.

Have a nice update!
