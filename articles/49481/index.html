Conversion of built-in types


Objects of one type can be converted to objects of another type implicitly or explicitly. Implicit conversions happen automatically, the compiler does it for you. Explicit conversions are performed when you ‚Äúcast‚Äù a value to another type. Implicit conversions also ensure that data is not lost. For example, you can implicitly cast from short (2 bytes) to int (4 bytes).
Again a small example. Let we have a bucket and a cup. =) If we pour water from a cup into a bucket, then we get an ‚Äúimplicit conversion‚Äù, and if we pour it from a bucket into a cup, then only ‚Äúexplicitly‚Äù. But how is it possible? We just have all the rest of the water will pour out.

If you do the inverse transform, then of course you may lose the information. If the value in int is greater than 32.767, it will be truncated during conversion. The compiler will not perform an implicit conversion from int to short:
