BitTorrent Dropbox Alternative


BitTorrent Inc. announced a new application that allows users to synchronize folders using the bittorent protocol with encryption. The program is free, has no restrictions on storage volumes and can be used as backup storage or a shared folder. BitTorrent Sync will be especially useful for those groups of users who need to share large files over the Internet.

Bittorrent is a powerful, fast, but at the same time simple tool for sharing files between multiple devices. At least two web giants, Twitter and Facebook , which successfully use this technology to exchange between their servers, can confirm this to you.



Now secure synchronization or backup using bittorrent has become available to the end user. BitTorrent Sync has very similar functionality to popular cloud storage providers, such as Dropbox and Skydrive, except that this system is not essentially cloudy. A huge plus of this system is that third parties do not have access to your data.



BitTorrent Inc. invites everyone to try out the application, but emphasizes that it is still in the alpha testing stage and therefore errors are not excluded.

‚ÄúThis program fits perfectly into our common goal of making the Internet better with peer-to-peer technologies,‚Äù the company comments on this release, hoping to get as many feedback and suggestions on how to improve the application from users.

There are many options for the successful use of the program, for example, the user can use one folder for backup storage on all their devices, this will be fast and fault tolerant. People working together on large files, for example, video monitors will like both speed and unlimited volume.

Setting up the application is simple and takes no more than a minute of time. Install the program, select the folder for synchronization and generate a key, that's all! The private key will be needed to connect other devices with which synchronization is required.

More information about this and other BitTorrent products can be found at the new BitTorrent Labs website.
