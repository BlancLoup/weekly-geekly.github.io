DEV Labs 2017. Online conference for Java and Web developers. March 25

March 25 will be the 25th online conference of the Soft Labs project. The theme is Java development. Old problems and new solutions, the pros and cons of the latest tools, the analysis of common mistakes and many living examples - all within the framework of an eight-hour online session.

In honor of a small jubilee, they collected a rich program:

  • Right during the conference, within one hour, together with Denis Prokopyuk, we will write a chat using TCP \ IP and Socket;
  • Vladimir Sonkin will talk about the features of the TypeScript language and show examples of its use in the development of web applications based on Angular 2;
  • Mikhail Sokolov will show how using Hystrix to make an application less susceptible to cascading errors;
  • Together with Evgeny Merkuriev, we will try to deal with the main problems that confront the designer of the modern JSON-based API;
  • From the report of Vyacheslav Lapin, we learn how to write in Java as in Scala, as well as speculate about new trends in functional programming.

The conference starts on March 25 at 11:00 Moscow time.

To participate in the conference you need to register . Participation is free.

In order to take part in any Soft Labs conference, it is enough to have a computer or mobile device connected to the Internet. No special software needs to be installed. Platform plugin will load in a few seconds.
