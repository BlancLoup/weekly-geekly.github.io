FEZ Hydra debug board based on the .NET Gadgeteer Open-source project

On Habr√© already mentioned the previous development of the company FEZ Spider Kit . GHI Electronics has continued to develop in the direction of developing. NET Gadgeteer devices . And on November 8 a new addition to the FEZ Hydra designer was introduced. FEZ Hydra uses an ARM9 processor with a clock speed of 240Mhz. With 16 MB of SDRAM RAM, and 4 MB of flash-type flash memory. As the developer indicates, it is possible to start the Linux OS on the device, since the system is completely 100% open.





Specifications:
  • 14 .NET Gadgeteer compatible socket
  • ATMEL 240Mhz ARM9 processor
  • 16 MB of RAM
  • 4 MB ROM
  • Rich graphics support
  • Support TFT touch display
  • FAT16 / FAT32
  • Full TCP / IP stack support: SSL, HTTP, TCP, UDP, DHCP
  • USB devices
  • Support 4-bit SD memory cards with no limit on volume
  • Analog input / output
  • Pwm
  • Tires I2C, SPI
  • UART interface
  • Real time clock
  • Dimensions 3.42 "x 2.44" (8.7 cm x 6.2 cm)
  • Consumption during operation is 130 mA
  • Temperature condition: from 0 to 70 ¬∞ C
  • Complied with the directive RoHS (Restriction of Hazardous Substances)

A feature of this board is that the firmware (firmware) was compiled by the open-source GCC compiler . Thanks to this, the global community can easily modify the code and develop the platform by the developer.
The design of the board was carried out in the EAGLE software (Easily Applicable Graphical Layout Editor), which is used to develop a cost-effective printed circuit board (printed circuit board). AGLE is usually used to develop open source devices like Arduino.
One of the main advantages of FEZ Hydra is its low cost. The price of the board is $ 79.95, with a starter kit, the cost is $ 99.

Currently, FEZ Hydra is searching for partners to develop the project. If you have experience with the NETMF Porting Kit, please contact GHI Electronics . Pre-orders will start in December 2011. All materials on the board layout and source code will also be available in December 2011. For more information about FEZ Hydra, please visit http://www.ghielectronics.com/catalog/product/328 .
