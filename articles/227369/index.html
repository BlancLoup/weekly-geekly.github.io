The first Evernote beta for Mac 5.6 is available for installation.



For six months, our team worked on the complete processing of Evernote for Mac in all aspects of the application. We set ourselves the task of creating a faster and more stable version of Evernote for Mac. The new version also turned out to be more economical in terms of energy consumption.

The work is not finished yet, but we are already ready to offer you to try out our beta version and help with the detection of errors in the operation of the application. Those interested can install the application at the link at the end of the post.

What's new


In the new version, much has been improved or appeared for the first time. Below we describe in more detail about all the innovations.

Speed


  • Synchronization began to work three times faster. Especially significant improvements in work will be noted by users with many common notebooks and Evernote Business subscribers.
  • New notes are synced instantly, so you will immediately access them on other devices.
  • Notes can be shared without waiting for the end of synchronization.
  • The time for starting and closing the application has been significantly reduced.
  • The new key combination allows you to quickly go to the list of notes (CMD + |). Tab allows you to quickly switch between search, list of notes, title and text field of the current note.
  • Significantly reduced power consumption in the background and in the application mode.


New opportunities


  • Added table editing features: change borders, adjust background color and border style.
  • The image size can now be resized right in the notes editor. Just click on the image and pull in the right direction for the lower right corner.
  • Search results are now sorted by relevance.
  • Now on the list of notes are notebooks in which you recently worked.
  • Check boxes lists in the editor of notebooks has become more convenient.
  • Now authorization in Evernote is saved by default.
  • Improved application stability
  • We fixed more than 400 errors, including more than 100 causes of emergency shutdown.
  • Our crash notification system has been completely redone.


Known issue


  • Spotlight search is currently unavailable.


Please note: this is a beta version.


Evernote for Mac 5.6 is currently in beta status. We recommend weighing all the pros and cons before installation, so that you can work comfortably with an early version of the application. Currently, there is a small likelihood of problems in work and, possibly, even data loss. Some features are not yet available, but they will appear very soon.
