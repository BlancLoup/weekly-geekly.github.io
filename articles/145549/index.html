Visual Studio Express 2012 for Windows Desktop

According to community pressure, Microsoft still announced the release of the free Visual Studio 2012 Express for Windows Desktop program, which will return the possibility of free development of desktop applications for Windows 8.

Two weeks ago, the company shocked developers with the news that the development of desktop applications will require an upgrade to Visual Studio 2012 Professional for $ 400, since the free Visual Studio Express package will come out only in four options: for Metro applications, web applications, Azure applications and Windows Phone applications , that is, it will cover all areas, except for desktop applications.

Of course, many were unhappy with this Microsoft solution. Fortunately, the company listened to the community, admitted a mistake and changed plans.

Apparently, in Express Desktop 2012 it will be possible to create applications for Vista, Windows 7 and Windows 8. ‚ÄúVisual Studio Express 2012 for Windows Desktop will provide a simple from start to finish way of developing desktop applications and will support all versions of Windows that are supported by Visual Studio 2012, - says the official blog . - With the new edition of Express, developers will be able to use C ++, C # and Visual Basic to create desktop and console applications. Developers will also have access to new features implemented in the entire Express line in Visual Studio 2012, such as the latest compilers and tools, embedded unit testing and the ability to coordinate development with small teams through Team Explorer and TFS Express. ‚Äù

Visual Studio Express 2012 for Windows Desktop is scheduled for release in the fall of 2012.
