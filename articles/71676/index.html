A bug in Chrome or how Yandex broke Google

Found a strange bug in Google Chrome.
It was succeeded to reproduce only on Yandex. Dictionaries:
  1. Launch Fresh Chrome
  2. Go to slovari.yandex.ru
  3. Without clicking in the dictionaries, we try to go to any URL (for example, google.ru )
  4. Chrome is either trying to search in dictionaries, or silently falls
If before entering another URL, click anywhere on the page (remove the focus from the dictionary search line), the problem will not be reproduced.
Noticed in the latest version of Chrome 3.0.195.25 (build 27847) and earlier.
When falling, you can see the advantage of tabs as separate processes, since only one falls.
