SQL Server Audit with NetWrix SQL Server Change Reporter: What's New in Version 2.5


Our program for auditing SQL servers - NetWrix SQL Server Change Reporter - received an update to version 2.5. The new version provides administrators with enhanced SQL Server auditing and simplifies control over changes in databases.

The program audits changes made to the configuration, databases, access rights and other objects of SQL Server and creates reports on them. The tool reports changes related to server instances, users, roles, logins, schema changes, and many other objects.

NetWrix SQL Server Change Reporter 2.5 offers the following features:
  • Audit of the database contents at the record, row and column levels, including the current and previous value;
  • The reports contain information about the computer from which the change was made, which helps to determine the source even more accurately, for example, in a situation where several changes were made under one account;
  • Audit changes to SQL Jobs, Steps and Job Scheduling;
  • Audit of rights changes to tables, views, stored procedures and functions;
  • Ability to receive reports in compressed form as an attachment to e-mail messages in CSV format.
