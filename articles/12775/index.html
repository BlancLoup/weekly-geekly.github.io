Yandex.Direct added the ability to limit the display of ads by IP-addresses of users.

To limit the display of ads to certain users, on Yandex.Direct it is possible to prohibit the display of ads on the user's IP address.

On the campaign settings page, you can add up to 25 IP addresses to the list of prohibited advertisements for displaying. In this case, users from the specified IP addresses will not see ads from this ad campaign.

Each computer with Internet access necessarily has an IP address consisting of 4 numbers from 0 to 255, separated by dots (for example, 123.34.0.1 or 200.190.34.120), with:

* in case of modem connection with the provider, the IP address is issued dynamically and may change the next time the same user connects to the Internet;
* Companies and providers often organize access to the Internet through a proxy server and then all computers accessing the Internet through this proxy server have the same IP address.

In both cases, a large number of unrelated users may be behind the same IP address. Therefore, it is not recommended to disable these types of IP addresses, as this may lead to a decrease in the number of potential customers.
