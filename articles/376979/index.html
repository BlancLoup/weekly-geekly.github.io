Microsoft makes it legal to download ISO images of Windows 7

Microsoft has opened a new section on the site - Microsoft Software Recovery , where official Windows 7 images are published for download. They are laid out for users who have a broken computer and lost the original OS version, or need to reinstall the system or create Windows 7 virtual machines.

The site also published step-by-step instructions for burning a DVD and a bootable flash drive.

To download, you must specify a valid product key (25 characters). Then you should choose the interface language - and then the link to the ISO file (from 2 GB to 3.5 GB) appears.

If you have lost your 25-digit code, but you are using a legal copy of Windows, you can restore it using a utility like Magical Jelly Bean.



Recall that from Microsoft you can legally and for free also download a preliminary version of Windows 10 Technical Preview .

There is also a way to legally download the latest Windows 8.1 image without a subscription .



In addition, for MSDN and TechNet subscribers, the company periodically uploads ISO images with preliminary versions of the system. But for this you need a paid subscription, which costs about $ 1000 per year.

Naturally, a wide variety of versions of Windows are in torrents.

MS used to post ISO images of other versions of Windows (for creating virtual machines) to free access, but now, apparently, it has stopped this practice.
