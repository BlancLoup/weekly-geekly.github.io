Kohana 3.3.0 released

Intro


I wanted to read about the changes in the new release of the wonderful framework Kohana and did not find any mention of this. Correcting the situation.

Body


Yesterday, October 23, 2012, a new release of the Kohana framework v.3.3.0 was released

The list of changes is so extensive that immediately lost the desire to list them all in this post.
All in brief, with reference to the original sources.

A total of 94 corrections were made.

Changes affected the following modules:
  • Auth (1 patch);
  • Cache (2 fixes);
  • Database (6 fixes);
  • Kernel (52 fixes);
  • Minion (1 patch);
  • ORM (19 corrections);
  • UnitTest (3 fixes);
  • Userguide (10 fixes).
