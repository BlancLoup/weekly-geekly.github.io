Evernote for Mac has become even more beautiful


The new version of Evernote for Mac has been significantly improved user interface, and also made some not so noticeable improvements and fixes.

Download the latest version of Evernote for Mac

Improved note sketches
We designed Evernote for Mac as an assistant for visual memory, and maximum perception of note thumbnails was one of the main tasks. In the latest version we have completely reworked the list of sketches, made it more likeable and easy to use. Now the shape of the sketch reflects the contents of the note, which helps their visual perception and memorization. In addition, indicator icons on top of thumbnails indicate the presence of attached files. These innovations are available both in thumbnail display mode and in mixed display mode.



Among other notable innovations:
  • changes in the left navigation bar;
  • improved text alignment in the request description panel;
  • optimized arrangement of thumbnails and text in a mixed display mode;
  • New appearance of the indicator of the remaining space for notes;

Also, the new version fixed a sufficient number of errors and added support for audio files in AMR format (for both free and premium accounts). If you already have Evernote for Mac installed, simply start checking for updates and download the new version, or download it directly from this link .

Evernote users for Windows, do not despair! Big changes are coming soon for you too.
