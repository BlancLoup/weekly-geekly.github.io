Security Essentials is not so bad

Oddly enough, but the first attempt by Microsoft to release antivirus software was quite successful. The independent laboratory AV-Test.org checked the free Security Essentials antivirus on test virus databases and got quite a good result .

Base WoldList of 3732 of the latest viruses and threats: the result is 100% in manual and active mode.

The total base of AV-Test.org of 545,034 viruses, Trojan worms, and so on: the result is 98.44%.

On a special database of spyware and adware, the program showed a result of 90.95%.

True, the software from Microsoft proved to be an absolute zero in terms of detecting dynamically modified viruses. It seems that he simply does not know how to look for such code.

In terms of scanning speed, Security Essentials showed an average result comparable to other antiviruses.
