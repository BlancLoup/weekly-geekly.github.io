Application Statistics in the Windows Store

As the author of applications for Windows 8, I am interested to know the detailed statistics of the store - how many applications are published, which categories are most popular, the price of applications, etc. Microsoft this information is not published and there is no API for getting it programmatically. Therefore, I wrote a small utility that analyzes individual application pages published on apps.microsoft.com. Below I want to give some of this analysis, I hope other developers it will be just as interesting and useful.

Number of published applications


Currently, there are about 86 thousand applications in the Windows Store. Some of them are published only for certain regions, so the number of applications available to the user varies from country to country.



Subsequent data obtained from the US Windows Store.

Application Categories




At the beginning of the year, the Education category ranked first in the number of published applications, but gradually the games took the first place and steadily increased their lead.

Application price




Almost 95% of published applications are either free or have a price of less than 2 dollars. As the number of applications grows, this picture practically does not change.

Application quality




As in the case of the price of the application, the quality assessments of published applications have changed little over the past six months, even though the number of applications has almost doubled. However, the number of applications having at least some assessment has decreased by 10%.
