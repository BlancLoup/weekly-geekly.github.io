New features of Photoshop and Flash CS5



The Everything about Adobe Creative Suite 5 blog has new descriptions of Flash CS5 and Photoshop CS5 graphic editors, which are scheduled for April 2010.

Adobe Flash CS5 expects a lot of innovations:

- updated text engine TLF, expanding the possibilities of working with text, and adding new attributes to the text, special characters and text containers;
- introduction of the API for the Text Layout Framework (TLF);
- updated font embedding;
- removal of Carbon elements for Mac versions that are not supported in Mac OS X 10.6 Snow Leopard;
- reorganized interface that saves the user from endless cascading dialogues and menus;
- the Info panel has been updated, now the X / Y and W / H coordinates will be updated in real time, and also when you hover over the object its RGBA value is displayed;
- updated color-mixer, now it is close to the standard color-mixer Adobe, and also allows you to create gradients at the bottom of the panel;
- New XML-based .FLA format, presented in the form of XML-based information about the document and all related objects in a single file;
- improved work with Cue Points (key points) for interactive Flash-video, now the main control of Cue Points is transferred to the Properties panel
- Preview Flash-video during creation, using the new FLVPlayback component for real-time preview;
- improvements to the Compiler Error panel; visibility of the error log has been improved with icons, advanced navigation options using the keyboard and shortcuts and additional information on the panel (a divided number of errors and warnings, for example);

Read more (in English) here .

For Adobe Photoshop CS5 , there is no such detailed list of innovations, but there is a video showing new editing and retouching features:



This short video demonstrates advanced image editing techniques made possible by a new algorithm that was designed to quickly find the best matches between all parts of two different image areas. The technology was jointly developed by Adobe, Princeton University and Washington University to speed up retouching and restore image detail. Apparently, this technology will be implemented in the next version of Adobe Photoshop.

You can read more about this technology (in English) in this material .

You can follow the rest of the news about the development of the new Adobe editors package on an unofficial blog (organized by Adobe developers, apparently) "Everything about Adobe Creative Suite 5" .
