10 useful tools for working with VMware

This article is a translation, see the original here - www.vladan.fr/top-10-free-tools-to-manage-your-virtual-infrastructure

Every system administrator who is constantly working with a virtual environment has its own tools for VMware infrastructure.

In this review, we will look at 10 excellent tools for virtual environments, among which there are both widely known and new ones.

1. Putty. One of the most popular utilities for working with Unix \ Linux servers. In the context of virtualization, it is used to connect to ESX \ ESXi servers using the SSH protocol. The utility is available as a ‚ÄúPortable‚Äù version. You can download the link: portableapps.com/apps/internet/putty_portable

2. VMware Converter. Free utility that allows you to convert a real system into a virtual machine. In addition to the ability to convert physical servers to virtual, the utility also recognizes Acronis and Symantec Ghost images. You can download it on the VMware website: downloads.vmware.com/d/info/datacenter_downloads/vmware_vcenter_converter_standalone/4_0

3. Trilead VM Explorer - a program to manage backup and recovery in a virtual environment, makes it easy to connect to the ESX infrastructure and manage VMDK files. Supports up to 5 ESX servers. Download here: www.trilead.com

4. Veeam FastSCP - thanks to this small utility it is possible to copy virtual machine files over SSH protocol. The utility allows you to connect to multiple VMware ESX and ESXi servers and transfer files between them from one interface. The utility is free, available at the link: www.veeam.com/vmware-esxi-fastscp.html

5. BGInfo is a small but very useful utility that displays on the desktop key information about the current server to which you are connected. In the presence of a large number of sessions, this functionality allows not to get lost in them. You can get here: technet.microsoft.com/en-us/sysinternals/bb897557.aspx

6. Daemon Tools is a well-known program with which you can mount ISO files in the form of CD / DVD discs. Allows you to emulate up to 4 devices, which makes it possible to work with several images simultaneously. Download the program here: www.disc-tools.com/request?p=4f261f3d8ccec3edc1d8ecc623740c59/daemon4304-lite.exe

7. ImgBurn - It has similar functionality with Daemon Tools, but allows you to create ISO images from a set of files or directories. Download the program here: www.imgburn.com

8. mRemote is a free program, thanks to which it is possible to organize work with many different protocols from one interface. The following protocols are supported:
- RDP (Remote Desktop)
- VNC (Virtual Network Computing)
- ICA (Independent Computing Architecture)
- SSH (Secure Shell)
- Telnet (TELecommunication NETwork)
- HTTP / S (Hypertext Transfer Protocol)
- Rlogin (Rlogin)
- RAW
Download here: www.mremote.org/wiki/Downloads.ashx

9. OpsCheck is a program that helps to verify the correctness of system settings for VMWare VMotion support. The program analyzes configurations from ESX servers and provides recommendations for proper configuration. Download the program on Tripwire website: www.tripwire.com

10. RVTools is a utility that displays information about the processor, memory, disks, network adapters, virtual machine snapshots, VMWARE tools and ESX \ ESXi storage. Download the utility here - www.robware.net
