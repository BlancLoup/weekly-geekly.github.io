Cloud for developer companies, release one: Azure Marketplace - a store of services and solutions for every taste

Our next column of author articles is called ‚ÄúCloud for development companies‚Äù. In it, we will acquaint you with interesting scenarios for the use of cloud technologies and Microsoft Azure, Office 365, and other platforms to maximize the benefits of professional software development.


The first article will tell you about the Azure Marketplace store, which, with the help of Microsoft, provides developers and IT professionals with a convenient way to deploy ready-made certified solutions and access to the cloud services of independent developers who expand the capabilities of the cloud platform. In addition, the store offers developers from a number of countries (the list will expand) a new sales channel for their solutions in 100+ countries of the world.

Azure Marketplace


The Azure Marketplace is a catalog of solutions, services, images of environments of varying complexity, ready for quick automatic deployment on the Azure platform.

The main idea of ‚Äã‚Äãthe Azure Marketplace is to give an independent developer a sales channel for their solutions around the world, and Azure customers are an easy way to deploy functional solutions in their own infrastructure.

Marketplace as a functional extension of Azure


Today in the Azure Marketplace you can find dozens of services offered by independent developers that extend the functionality of the Azure cloud. Solutions available in the store are divided into categories: virtual machines, developer services, API Apps, connectors for Azure Active Directory, web applications, data services, Docker containers, Microsoft Dynamics solutions. Consider them in order.

Virtual machines

The ‚Äú Virtual Machines ‚Äù section aggregates all solutions built on the basis of Linux and Windows virtual machines, both by the open source software community and third-party commercial software developers and Microsoft. In mid-July 2015, the store presented more than 650 different solutions, among them: Windows Server 2012 R2, Cloudera Enterprise, Jelastic, DataStax, Veeam Cloud Connect, SAP HANA Developer Edition, NGINX, CoreOS, Kaspersky Labs solutions, IBM DB2 and many others.


Developers and IT professionals can, in a few clicks, purchase and deploy any of the solutions in their Azure infrastructure environment in the form of one or a whole set of virtual machines. The store greatly facilitates the deployment of solutions, offering everything you need for quick manual or, most importantly, automatic deployment of solutions, both on the basis of a single machine, and the most complex configurations of entire clusters of machines, combined into virtual networks and configured for high availability.

Developer Services

The services section for developers offers new and existing PaaS services that are previously available in the Azure Store, offering a variety of features for IaaS or PaaS solutions based on Azure cloud.


As of mid-July 2015, more than 50 services from independent developers and Microsoft were presented in the services catalog, including: ABBYY OCR SDK, Auth0, Bing Maps API, ClearDB MySQL, Face APIs, IronMQ, Memcached Cloud, New Relic, MongoLab, Nodejitsu Redis, SendGrid, RavenDB and others.

API Apps

API Apps are ready-made API applications created by Microsoft for use by developers in developing Azure Logic App applications built on the basis of workflows (more about Logic Apps).


Among the API Apps presented are Oracle DB Connector, Twitter Connector, Slack Connector, Office 365 and OneDrive Connector and others.

Connectors for Azure Active Directory

The Azure Active Directory service allows you to combine cloud applications and applications located in the enterprise network under a single system of authentication, authorization and security.


A distinctive feature of Azure AD is its integration with external SaaS services and applications, such as Google Apps, Dropbox, Saleforce, Office365, and others. Thanks to the hundreds of ready-made connectors available through the Azure Marketplace, users of Azure AD can easily access third-party services using one account.

Web applications

Azure Marketplace offers dozens of ready-made templates for popular CMS and web applications such as Drupal, Wordpress, Django, CakePHP, Flask, Joomla, Express, ASP.NET projects and many others.


A distinctive feature of web application templates is that they are designed to be deployed in the Azure Web App environment ‚Äî in the form of services rather than virtual machines, thereby simplifying and speeding up their deployment, maintenance and cost reduction.

Data services

By integrating Azure Marketplace with a large number of data services and data sources, developers can easily access public data on demographics, maps, search queries, recommendations, geography, translation, and others.


Data from the Azure Marketplace can be easily obtained and integrated into your own solutions.

Docker Containers

One of the most recent additions to the Azure Marketplace has been the containerized applications section, ready for deployment in the Azure cloud environment.


Among them, at the moment are presented: Redis, Nginx, MySQL, Jenkins, Memcached, Elasticsearch, RabbitMQ and other popular tools. All containers from this section are created using Docker technology and can be quickly deployed to Azure.

Microsoft Dynamics Solutions


Another section of the Azure Marketplace is the Microsoft Dynamics Solutions section, which is a set of ready-to-use virtual machine-based configurations of the Dynamics family for quick deployment on the Microsoft Azure platform.

Quick start with Azure Marketplace

Azure Marketplace as a sales channel


We looked at the Azure Marketplace as a powerful tool that makes it easy for developers and IT professionals to deploy solutions and organize environments based on different solutions in the Azure cloud.

On the other hand, Azure Marketplace is a powerful sales channel for independent software developers. Today dozens of new solutions from companies from many countries are added to the store every month. These solutions receive a variety of marketing support from Microsoft and additional promotion to the target audience. An example of such a promotion is the Microsoft Partner Apps channel.

With its presence in more than a hundred markets around the world, the Azure Markertplace channel is a great way for any software developer to enter the global scene and start selling themselves anywhere in the world without the hassle of legal and tax issues.

Azure Marketplace in Russia

Today, developers from two dozen countries can publish solutions in Azure Marketplce. While Russia is not included in the list of such countries and the publication for Russian developers is limited. However, several Russian companies with a global presence have already successfully published their solutions in the Azure Marketplace. Among them: Kaspersky Labs , Veeam , Jelastic and NGINX .



The possibility of publishing solutions from Russia will be added in the near future. In the meantime, Russian developers and IT professionals can freely use the Azure Marketplace and third-party solutions to create new interesting projects based on the Azure cloud.

Conclusion


We looked at the opportunities and advantages of the Azure Marketplace store for developers and IT professionals who are interested in new features and independent software vendors who want to get a new sales channel for their solution, enter new markets, get support and more. promotion.

Azure Marketplace is a new tool, a platform, a channel from Microsoft, its functionality will constantly increase, and its geographical presence will expand. Today, the store has more than 3,300 different-purpose products from hundreds of independent developers and open source communities.
