The hot summer has come to an end, and a no less hot autumn has come. But despite the heat and the desire to relax, the community continued to work on KolibriOS, and there is something to show. I warn you immediately, under the cut a lot of pictures, who are not afraid Welcome.

System-wide changes (kernel, drivers, libraries):


  • proportional scalable system font
  • increased speed and stability of IDE drives, and SATA in IDE mode.
  • Ability to work with any of the installed network cards (previously used the first detected network card)
  • console.obj : scroll with mouse.
  • kmenu.obj: a library with the implementation of the main and context menu.



Changes in application software:


  • Eolite : change of attributes of both a single file / directory and a group of selected ones; custom font size; progress bar in copy dialog; the work of shotcards on layouts other than English; two panel mode; remembering the size and position of the window; multiple fixes and optimizations.


  • WebView : use of new system fonts; improvements and bug fixes.
  • MouseCfg : check and configure mouse parameters.
    Spoiler header
    The program allows you to check the performance of the mouse, adjust the speed and delay of the cursor, and also is a shell for the console programs that allow you to make the edges of the screen through (the cursor reaches one edge, jumps to the opposite) and also emulates the mouse using the NumPad key.



  • TmpDisk : displays the size of the created discs.


  • TextEdit : toolbar change and full menu


  • TinyPad : bug fix when working with clipboard.
  • Calypte : text file reader.
    Spoiler header
    Current functionality:
    - opening text documents encoded in DOS
    - navigation using the PgDn, PgUp, Down, Up, Home, End keys
    - adaptation of the output to the width of the window

    As you can see, the functionality is not rich, but very soon we should expect a zagladka, a custom font, and many other useful buns.



  • Calc : use an enlarged font to display numbers.
  • VNC Client:; support for ZRLE / TRLE, 32bpp, keyboard; bug fixes and refactoring






  • End (C-- version) : new design.


  • MTDBG : reduced redrawing of the window at program start.
  • TicTacToe : a new logic game, the feature is that it is written in Oberon07


  • Maze : new puzzle game, the feature is that it is written on Oberon07


  • ALMAZ : new arcade game, analog Lode Runner
