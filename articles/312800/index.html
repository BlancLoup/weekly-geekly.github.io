Introducing 3CX V15 SP2 with Debian Linux 8 support

The 3CX V15 SP2 includes many fixes and performance improvements. In particular, the assignment of user access rights to the 3CX interface and group editing of extension numbers has been implemented. The utility of adjustment of automatic telephone exchange 3CX Configuration Tool for convenience of the administrator received the web interface!


But the main innovation is the release of the Linux version of the system. 3CX V15 now works on Debian Linux 8! The Linux version has almost the same features as the Windows version. The remaining discrepancies will be resolved in the coming months.


Why did we release the Linux version? First of all, because it is easier to place in the cloud. Now you can install 3CX on most Linux VPS servers using Openstack technology. The largest international data centers, such as OVH , offer such servers for less than $ 4 per month! Right now you can try the system for free on Google Cloud . The Linux version can work as a virtual machine, or on ‚Äúbare hardware‚Äù, implementing a ‚Äúturnkey‚Äù communication server. By the way, an ISO image will soon appear to simplify installation.


Attention : Clear the browser cache and convert the system audio files. Incorrectly recorded audio files that are played in the Call Queues and IVR menu will not play.


Preparation of audio files for correct playback in 3CX V15 SP2


Unfortunately, many 3CX installations use improperly prepared WAV files that have incorrect or unsupported audio format.


In 3CX Service Pack 2, the audio file should have the following format:


  • File Format: WAV
  • Channel: Mono
  • Bitrate: 8 kHz
  • Discretization: 16 bit

Before upgrading to V15 SP2, be sure to convert your audio files. We have developed a utility that will automatically convert all audio files of your installation to the correct format.


  1. Download 3CXConverter Utility
  2. Save and unzip it to the 3CX server.
  3. Run ‚Äú3CXConvertPrompts.exe‚Äù
  4. The command line will display a message stating the following services: 3CX Media Server, IVR Server and Queue Manager Service.
  5. After stopping services, press Enter to start converting. All voice files of the system will be automatically converted, after which the message Conversion completed appears . The stopped services can be restarted . Press any key to exit the utility.
  6. Start the services: 3CX Media Server, IVR Server and Queue Manager Service.

After converting the files, upgrade the system to SP2. Of course, you can convert files after upgrading to SP2.
