Meet: DrupalSN - a social network of drupal-developers

Hello friends and colleagues!

We are glad to inform you that we have finally launched the long-awaited project DrupalSN - a social network for drupal developers.

DrupalSN is a community of developers who make websites on Drupal CMS / CMF and are looking to share solutions and recipes for creating websites on this wonderful system.

The project consists of sections and pages:
  • topics that are divided into blogs (groups) ;
  • quick question-answer system ;
  • section " Work ", which has vacancies and resumes ;
  • screencasts

The site includes a rating system that is awarded for both topics and comments.

For convenience, included:
  • internal correspondence between users;
  • system of connections between users, such as friends ;
  • bookmark system;
  • posting on Facebook and Twitter;
  • invites (invitations);
  • post removal from publication by its author.

Currently, registration on the site is available only by invitation. If you need an invite, leave a request and mail, I will send.

For information about the rules, frequently asked questions about the project, you can see the FAQ page.

In addition to the lack of opportunities to write topics and participate in discussions, unauthorized users have hidden personal data in vacancies, resumes and profiles.

The project is similar to Habrahabr , so that no one has any doubt that there is nothing impossible when working with Drupal - everything depends on imagination and developer skills.

At present, the development stage of the project can be called the initial stage, but the project is developing and will be developing. We have a lot of ideas, as well as fantasies.
