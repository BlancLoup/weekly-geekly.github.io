CMS Simpla has connected PaySto payment module

Gaining popularity in RuNet CMS Simpla has replenished with another payment module. The PaySto onlineMerchant interface has been added to the store management system.

Paysto module (onlineMerchant) is included in Simpla by default, starting with the next version of Simpla 2.1.1. To install the new module in the current version 2.1.0, just download the archive from the Simpla website, then copy the unpacked Paysto folder via FTP into the / paynment folder.


At the final stage, it remains to create a new payment method by choosing the PaySto module for it. This module must be made active and enabled for the desired delivery methods. An example of the settings can be found in the archive called example.png. Demo storefront is available at demo.simplacms.ru.

Recall online merchant is an automatic payment processing solution that is easily integrated with any online store or online service. Thus, the seller gets the opportunity to accept payment using almost all available methods: electronic money (WebMoney, Yandex.Money, RBK.Money, etc.), bank transfers and cards (VISA, MasterCard, American Express), as well as using money transfer systems transfers (CONTACT, UNISTREAM) or through terminals (QIWI).

Note that integration with the Simpla store management system for PaySto is the first experience of cooperation with CMS. In the near future, the number of such partners PaySto will increase.
