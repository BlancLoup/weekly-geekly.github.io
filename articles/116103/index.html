9 months ago we already showed you our benchmark for flash drives ( one , two , three ). Here we will try to summarize the work of the site usbflashspeed.com .


Data


  1. Produced 6200 tests with the publication on the site.
  2. Tested 4800 devices with different serial numbers.
  3. About 150 people visit the site daily, 40% of them are returning users.
  4. Despite the English-language interface and the absence of direct competitors in the western market, 75% of users know Russian.
  5. People do not want to enter the names of their devices.
  6. The link to our site was found in the November issue of Hacker magazine.
  7. The project was not advertised or promoted except for announcements on Habr√©.
  8. Advertising on the site brings about 300 rubles per month.
  9. For nine months, we made almost no changes to the benchmark and project site.
  10. The site is written in django, the protocol used is thrift, the interface is on .net 2.0, and the testing procedure is in C.
  11. No Linux / MacOS versions, benchmark works only under Microsoft Windows.

Conclusion


Whatever the project is relevant and necessary, by itself it will not take off. With all the interest in the project from the point of view of users, the project is not gaining even 1000 unique traffic per day.
In connection with the fact number 5, the database contains a huge number of unsigned devices. There was an attempt to implement data mining algorithms, but unfortunately for the data already collected it is almost impossible to do this, for this we are releasing a new version of the site and the client collecting more information about the device.

In the new version


Changes in the testing utility :
  • Added speed chart.
  • Displays the VID and PID of the device. (USB Vid and Pid - manufacturer code and product code for the USB device used).
  • The restriction on testing various drives has been removed, now even hard drives can be tested ( not recommended ). The results of testing anything other than a flash drive will not fall into the top shared on the site.

Changes on the project website :
  • The site now displays the speed range for each block size.
  • The site now displays the file system of the tested device.
  • There was a filter on the size of the drive.
