10 million hits a day with WordPress on the server for $ 15

English developer Ewan Leith has published step-by-step instructions on how to raise a virtual microserver on Amazon, Linode or another cloud hosting service that can run a WordPress blog and maintain 10 million hits per day (the report is compiled using Blitz.io), while cost only fifteen dollars a month.

The instruction describes how to consistently install Ubuntu 11.10 (Oneiric), MySQL, PHP with PHP FPM, APC and the MySQL module, Nginx with the configuration for WordPress , and of WordPress itself. After that, the server is still rather slow, but everything changes the installation of W3 Total Cache and Varnish, which together with Nginx are capable of creating a real miracle .

Even a person who is not a technical specialist is able to perform the actions from the instruction.



Disclaimer: Evan Leith is the founder of NutmegData , which helps companies migrate to modern platforms.
