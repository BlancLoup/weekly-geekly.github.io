The first screenshots of the new Spartan web browser from Microsoft

Previously, the media, with reference to Microsoft, published information about the new Spartan web browser, which will become part of Windows 10. Spartan does not belong to Internet Explorer 12, but will become a completely new browser and will be supplied as part of the new OS. This browser will be more "lightweight" and will resemble the competitors of IE - Mozilla Firefox and Google Chrome. Under the ZDNet data, Windows 10 will ship with both browsers, i.e. Internet Explorer and Spartan.



One of the main innovations that Spartan will receive is the handwritten notes mechanism for web pages (touch screen). These notes can be shared with other users through the existing OneDrive cloud engine and perform co-editing. The second innovation is the integration of Spartan with voice assistant Cortana . It will be used for booking hotel rooms, as well as for other information, such as flight information.

Another feature of the browser will be a mechanism for grouping open browser tabs. For example, a user will be able to group tabs according to a certain principle, to separate web pages used for work purposes from those used for entertainment and web surfing.

Spartan will become a browser that will be supported on all types of devices, i.e., PCs and tablets running Windows, as well as mobile devices running Windows Phone.

The Chinese edition cnbeta published the first screenshots of this browser.
