Codify: native development environment for iPad

The Codify editor, the first development environment for the iPad, with a native interface and touchscreen support, has appeared in the App Store. Very convenient: click on the characteristics of the object with your finger - and you see a pop-up window with a list of options. But the most important thing is that at any time written code can be run to check the operation of the accelerometer and multitouch on the go. No more torment with emulators!



Codify is built in the Lua language , it is a simple and elegant programming language that is well suited for creating games for iPad.

Thanks to the latest changes in the rules for publishing applications on the App Store, it has become possible to upload such tools there. Perhaps this is only the first swallow, and in the future there will be convenient programs for other scripting programming languages ‚Äã‚Äãwith other libraries.

True, the numerous restrictions that Apple imposes still make it very difficult to work in Codify. For example, applications are not allowed to download code from external sources, so everything in Codify must be written manually. Similar problems with the export code (you have to do copy-paste manually, because normal export is prohibited).

The cost of the Codify editor for iPad is $ 7.99 .
