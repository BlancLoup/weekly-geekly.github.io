The final version of Microsoft SQL Server 2014 has been released

Today, a new version of Microsoft SQL Server has become available.


From the post on the official SQL Server blog
Today, Microsoft has released SQL Server 2014, the core product of our cloud-based data processing platform.
SQL Server 2014 delivers record-breaking performance with new memory technologies. That will help customers speed up their business and implement new work scenarios.

In addition, SQL Server 2014 allows you to use new hybrid cloud solutions. This will take advantage of new cloud computing benefits. For example, in scenarios such as backup to the cloud and disaster recovery of a locally installed SQL Server.

SQL Server 2014 offers the best in business intelligence industry by integrating with familiar tools such as Excel and Power BI for Office 365.

We also released the final version of SQL Server Backup for Microsoft Azure Tool. This is a free tool that allows clients to backup from older versions of SQL Server to Azure Storage.

Try SQL Server 2014 today.

Download and try SQL Server 2014 on your own hardware, or within a few minutes, start working with the version deployed in the cloud.
