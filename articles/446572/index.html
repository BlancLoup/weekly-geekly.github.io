Editor.js is an excellent editor saving source code in JSON format.

CodeX are cool guys who are based at ITMO University. On the basis of this university, students and graduates create interesting and useful projects: Hawk.so , Capella , AR-tester and others.

One project really helped our business. Editor Editor.js saving source code in JSON.



The main advantage of the editor is the storage and presentation of data in JSON format. Those who have both the web version of the product and the mobile application will especially understand. Content managers love to use a WYSIWYG editor for writing articles, but after that we get ‚Äúspaghetti‚Äù:



Spaghetti just killed the whole layout and integrating them into mobile apps was just pain . I had to write a separate article for the web version and the mobile application. It is long and hard to maintain.

A compromise is the JSON format. It is simple, has a strict structure and flexible. In the admin panel, add the Editor.js editor, write the content and save it as JSON. According to the API, we send JSON to the mobile application and create it further as it should, and we convert it to HTML via Editor.js to the web version of JSON. And all happiness.
