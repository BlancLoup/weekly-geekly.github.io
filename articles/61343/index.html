Amarok 2.1 release

The news is written by a Unicorn user, but he cannot publish it due to negative karma.

After long months of development, the release of the popular player Amarok 2.1 was released.


Amarok 2.1 has undergone many improvements since the last version. First, it is a redesigned new interface. Now you can hide player panels, define a playlist template, (which fields, which font and in what order to display).

Default


Changed playlist

With the help of Amarok URL, you can save links to specific resources and send them to friends and acquaintances. You can bookmark not only the composition, but also set the position in the composition itself, it is very convenient if you are listening to a large file, for example, an audio book. Amarok will automatically highlight the last listened song.
Context View Applets have lost their many pages, they now have the appearance of a vertical column, which you can optionally collapse to tray.

Context view

The new version supports technology Replay Gain to automatically level the volume in different audio formats. Added a plugin to work with Last.FM without creating an account.

Last.fm

Among the main novelties of Amarok 2.1 are a new, fully customizable playlist with a drag-and-drop-capable editor; cue sheet support like Amarok 1.4 (playing .cue files); Ampache 3.4.4 and 3.5.x support; you can delete tracks from the local collection; Playlist Browser displays playlists from various sources; in the Context View a toolbar for managing applets has been added; fixed a large number of errors found.
