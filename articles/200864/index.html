3D Encryption Software Released

Owners of 3D printers are preparing for confrontation with the owners. Everything leads to the fact that the owners of rights to various 3D-models (for example, Mickey Mouse figures) will prevent the publication of these models in the public domain. In addition, law enforcement agencies may interfere with the free distribution of other prohibited items, such as weapons.

London programmer Matthew Plummer-Fernandez (Matthew Plummer-Fernandez) believes that the problem can be solved with the help of cryptography. He developed the Disarming Corruptor program that encrypts the STL file, so that you can get the original 3D model only if you have a password. In this case, protected files can be published in publicly accessible directories like Thingiverse .


Original pistol model (left) and encrypted model (right)

You can freely publish the encrypted model, and distribute the key among trusted people.

The program allows you to manually draw arbitrary shapes in the fake model and set a seven-digit code that corresponds to the transformation parameters (yes, the protection is not very reliable, but this can be fixed).



So far, the author has released only a free utility under OS X , is currently working on versions for Linux (32, 64) and Windows (32, 64).
