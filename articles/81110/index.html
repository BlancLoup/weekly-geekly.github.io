Network access to nokia 5800 via WiFi / Internet

Not so long ago, I read an imwode habraiser article on accessing a computer disk via WiFi / Internet using the built-in Symbian client. And all would be okay, but in our nokia 5800 our dear Finnish friends did not bother to add this client by default. But I didn‚Äôt become discouraged about this and went surfing in search of the solution I needed. As it turned out, to no avail.
Currently, for symbian smartphones, a whole series of programs is available for wireless (WiFi, 3G, GPRS, ...) access between the phone and the computer from Telexy Networks, Inc. . The programs have full touchscreen support, have a user-friendly interface and many other goodies.

SymVPN - PPTP VPN client for S60 (3rd and 5th edition). The settings are simple. Set the name of the connection, select the access point (WiFi or GPRS), specify the server name, enter the login and password. It works perfectly through WiFi, but it was not possible to connect via the PPSOS network, it may be wise to do something with the routes. The program creates an access point that is not visible in the list of access points from the settings menu, but is visible when you select an access point directly from programs that require Internet. You can use in conjunction with Mail for Exchange.



SymSync - A program to synchronize the specified folder of the smartphone with a folder on the local computer. Allows you to assign a synchronization interval and set multiple profiles at the same time, there is also the ability to work in the background. For example, we set up the Images folder and once a week the smart merges everything that we managed to photograph on the home computer.



SymNAS - The program allows you to organize access from a computer to the files of the phone, in fact, we get a shared network folder. Settings are also extremely simple. Specify the folder to which we want to give access (full or read only), register the working group and the access point. You can contact the phone either by IP address or by name.



SymRDP - The program is a client implementation of RDP (Remote Desktop Protocol) on the Symbian platform. Designed to provide remote access from an S60 5th phone to computers supporting the RDP protocol:
- Microsoft Windows NT / 2000/2003/2008;
- Microsoft Windows XP Professional;
- Microsoft Windows Vista (Ultimate, Business, Enterprise);
- Microsoft Windows 7 (Professional, Enterprise).
Actually here it is better to see and touch once, personally, I was just delighted with the first connection)



From the settings there is nothing special, we specify the IP address of the remote machine, the access point (here we can use a VPN), the name / password, in general, as usual. Yes, and do not forget to open remote access on the local machine of course.
The screen resolution is set equal to the resolution of the phone (640x360 in the case of nokia 5800), or 1280x720, at which scrollbars appear (in the screenshot there are no). It is quite convenient to control the stylus, although sometimes the response time makes you want something better. Transfer of video and audio, as I understand it, is not supported.
And finally, SymNC - The program combines all of the above, plus a convenient file manager with support for networking.
