Windows hotkeys

Everyone in life probably has come / may have a moment when the mouse ‚Äúdies‚Äù. Many start to get lost in such cases, not knowing the elementary combinations of hot keys. If you enter their number or want to replenish your knowledge with new combinations, please! The combinations were tested in Windows Vista, but should not differ much from other operating systems.

PrintScreen - Take a snapshot of the entire screen.
Backspace - go to previous page

Alt + D - Menu Access
Alt + Enter - Opens the properties of the selected object.
Alt + F4 - Closes the current window.
Alt + Space - Active Window Control Menu
Alt + Printscreen - Copy a screenshot of the current active window (application) to the clipboard.

Ctrl + X - Cut object
Ctrl + V - Paste from clipboard
Ctrl + C - Copy to clipboard
Ctrl + W - Close the current window
Ctrl + P - Prints the current document.
Ctrl + N - Open a new window
Ctrl + A - Select All
Ctrl + Space - Select the current object
Ctrl + F - Call the "Search file window in the current folder" (also see F3)
Ctrl + O - Call the file open dialog
Ctrl + Escape - Displays the start menu (also see Win)
Ctrl + S - Save Document
Ctrl + R - Refresh Document
Ctrl + Z - Undo the last action.

Alt + Tab - Switch between running applications

Tab - Move the focus on the elements. (also see F6)
Enter - alternative to double click
Win - Display Start Menu

Arrows - move through the conductor tree

F1 - Help call
F2 - Rename object
F3 - Call "Search file window in the current folder"
F4 - A list of addresses is opened.
F5 - Updates the current window
F6 - Moving focus on elements.

Win + Tab - In a special way switches tasks (Works when Aero is on)
Win + R - Run window
Win + F - File Search window
Win + E - Opens Explorer
Win + M, Win + D - Collapse all open windows
