The secret arrow in the address bar and other surprises in the assembly Vivaldi 1.0.291.18

And again the build, in which we mainly worked on fixing bugs on the way to the beta version, which looms on the horizon. We continued to improve the rendering of sketches in the Express panel, we also paid special attention to the notes - we fixed several unpleasant bugs in this part of the browser. We also did some work to improve the list of entered addresses in the address bar, but more on that below and in more detail.

History of entered addresses

For those who prefer to work with a mouse, we have added a secret arrow to the address bar. It is not like it, but it appears when the focus is moved to the address bar - for example, when opening a new tab. Clicking this arrow will open the list of all the addresses entered earlier:


The function is still damp, and I, for example, could not remove any of the addresses from the list, but we do not stop at what has been achieved and we will finish this function further.

Note

Starting from this build, we are trying to find the best ways to work with search services. This is so that users are happy, and we got something for bread and butter. Therefore, in the new build, the default search is Bing. But I will immediately calm those who use Russian, Ukrainian or Belarusian versions: Yandex is still the default search in these builds. And there will be no dramatic changes in the near future.

Yes, and a collection of localizations was added. This time, lucky connoisseurs of the great and powerful Albanian language. Very welcome comments and suggestions.
