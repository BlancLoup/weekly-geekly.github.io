jQuery UI Bootstrap Theme

On projects I use Twitter Bootstrap and jQuery UI for prototyping interfaces of various page elements, including forms.

The other day it took to supplement the form with one field. The most suitable format was chosen, in the form of a set of switches (input element, type radio).

In Twitter Bootstrap, a set of switches is presented as a set of buttons and does not work directly with the input element, which implies writing additional code to handle events. The jQuery UI library has a solution to a similar problem , and all event handling has already been implemented.

Stopped at the jQuery UI library. It remains to resolve the issue with the appearance. The prototype is a prototype, but did not want to break the slender appearance, different from Twitter Bootstrap and jQuery UI.

The jQuery UI Bootstrap project came to the rescue, the essence of which was to transfer the visual execution of Twitter Bootstrap widgets to jQuery UI widgets. For the most part, the project is a jQuery UI-compatible theme.
