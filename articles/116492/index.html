Manage PHPShop online store can now be directly in the interface 1C

We are glad to present a new handler for remote site management via 1C , created for more efficient management of the PHPShop store.


A new handler for remote control of an online store through 1C allows you to organize the work of a content manager directly from his workplace in 1C.

The program creates a separate menu in 1C to manage the online store and allows you to remotely edit all sections of the site, including the product catalog, add news and new pages, manage polls, text blocks and menus.

The entire content of the online store, as well as in the administrative panel, is presented in the menu of the processor in the form of a compact tree-like catalog of pages, the contents of each of which can be changed using the usual WYSIWYG editor . In addition, the file manager for managing hosting files is now also located in the 1C interface. All data is stored on the server, and 1C is used as a shell for editing them. Thus, everything you need to work with an online store is now collected in one window.

The program supports the delineation of user rights , which means that the owner of an online store will be able to effectively distribute duties among his staff, entrusting, for example, one staff with the management of the catalog, and the other adding news and articles.

Handler for remote content management via 1C is free and is available for versions of the online store Enterprise and Enterprise Pro 1C, as well as the most popular versions of 1C 8.2 and 8.1. The program is delivered as active processing (epf), runs from a file and does not affect the standard 1C configuration . Suitable for any configuration, like any other application from PHPShop for 1C.
