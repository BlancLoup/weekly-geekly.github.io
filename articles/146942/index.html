VLC beta for Android released



Today, the beta version of the well-known VLC universal free free player has become available in the Play Store , which just recently celebrated one billion downloads.

A preliminary alpha version of VLC for Android appeared in September last year. Then the player, according to the developers, worked extremely unstable. At the moment, the development has reached a new stage (although it is stated that "It is intended for power users and hackers") and the officially submitted version supports all file formats that the desktop version of VLC, audio and video libraries with full search for them work with. , supports streaming playback.

Interestingly, it was previously unofficially announced that Android versions will be supported starting from 2.3. Obviously, these plans have been changed by developers, and now VLC for Android supports a wider range of devices - it can be used on devices with Android 2.1 and higher.

Since the program is in beta, developers are asked to tolerate possible problems and ask for help in testing. If you have an unnecessary Android device, then you can send it to this address so that this model can be tested officially.
