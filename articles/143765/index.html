Customize your google

After writing this post, I was interested in the search options that you can use for convenient work.
Having asked this question, I did not find more or less complete information in runet.
But! It turns out Google has long been for everyone, everything showed .



Let's start simple - www.google.com/search?q=
This is the simplest request, in which all parameters are disabled or parameters that are stored in your cookies are used.

Further I will tell about the most interesting and used parameters. If you want to know more, go here .

Although this is called a search parameter, I will still use the word marker.
So, for the test, you can insert any of the described markers at the end of the line (if your line is the same as a bit higher).

Country Search Marker


&cr=countryRU
By specifying this marker, we will get results from the specified country.
List of abbreviations used.
Asus example in Finland


UPDATE (thanks to Soutlan for the bug)

Result language marker


&lr=lang_uk
Search results will be in the specified language.
As I said in the comments, I was let down by the specs, not all languages are indicated there , and this misled me.
In the end, I picked open the Google Advanced Search.
And I chose all the parameters responsible for the result language.
Here is the complete list. 100%
Example

Google Interface Language Token


&hl=ru
Search results will be in the specified language.
List of abbreviations used.
Example

Marker to automatically filter results


&filter=0
If I'm not mistaken (checked by spear method)
0 - repeating pages show an example , more
1 - repeating pages are grouped or omitted example , more
The default value is 1.

Safe Search Status Token


&safe=high
Show or hide dangerous content. I think the most expedient is to set such a marker by default to the child profile.
Has 3 meanings
off - example
medium
high - an example
In this query, it makes no sense to use medium.

Time search marker


&as_qdr=y
Has 4 meanings
d - day - example
w - week - example
m - month - example
y - year - example
Shows the results that appeared in the selected time period.

Site Search Marker


&as_sitesearch=habrahabr.ru
Will find information on the selected site
Example

File Type Search Marker


&as_filetype=DOC
Shows the results in the selected file format. It is convenient to search for specs, datasheets and more.
Matter:
pdf, ps, xls, ppt, doc, wks, wps, wdb, rtf, swf, txt
Example
Very interesting example

Content Type Search Marker


&tbm=vid - video - example
&tbm=nws - news - example
&tbm=isch - pictures - example
&tbm=shop - shopping - example
&tbm=bks - books - an example
&tbm=plcs addresses - example
&tbm=blg - blogs - an example
&tbm=dsc - discussions and discussions - example
&tbm=app - applications - example
