Announcement of Sync.NET mitap # 3



Friends, on Friday, November 4, we invite everyone to attend the meeting of the Kharkov .NET community. You are waiting for interesting reports, a block of questions and answers, as well as traditional coffee with cookies.

Speakers and Topics



Vladimir Chmuzh, a report titled " ASP.NET Core JavaScript Services: ASP.NET Integration with Modern Frontend Framework and Tools "

Vladimir will talk about the features of a set of tools called JavaScriptServices, offering an infrastructure for SPA developers based on ASP.NET Core. Will be discussed:

  • current trends in web development;
  • executing Node.js modules in a .NET application;
  • integration with webpack and hot module replacement;
  • isomorphic applications, server-side prerendering, in applications on Angular 2, React and ASP.NET Core.

In his report, Ivan will tell you what can happen if you avoid using SQL as much as possible: what difficulties you will have to face and what opportunities will be lost:

  • pitfalls of using ORM on the example of the Entity Framework;
  • use of SQL queries and ORM;
  • The benefits of using SQL.
