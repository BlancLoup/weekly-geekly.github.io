Autoinstaller LiveStreet - as a gift to hosters

In June 2010, LiveStreet and COMTET published the APS package in the public repository apsstandard.org.
Today, KOMTET provides an opportunity for all hosting companies to download the updated version of LiveStreet 0.5.1 autoinstaller .

Since the publication of the first version of the LiveStreet APS Package, up to 6500 copies have been downloaded so far. Drupal, WordPress and Joomla with their 70-80K downloads are still far away, but this is not so little.

What is the APS-standard and APS-package, we have already told in the previous article. In short, this is a way to create a distribution kit, which allows a developer to more efficiently distribute their web applications (in particular, through the service providers RTCOMM and SoftCloud). Hosting providers that use Parallels products - Plesk, Small Business Panel and others, as well as other hosting control panels with APS support - are able to provide hundreds of web application names to Clients. And end users get rid of the need to download distributions of CMS (forum or online store) via FTP, create a database and generally understand how it works. All the work on installing applications for them makes auto-installer in the hosting control panel. This is convenient not only for novice website builders, but also for experienced developers to quickly develop websites. In addition to the actual web application deployment, the package can provide the ability to select plugins, installation language, themes / templates, and so on - as we do for Joomla autoinstallers, WordPress, and even Django.

In the near future, it is planned to prepare an APS package and publish it on apsstandard.org for the PunBB forum. And then ... we can prepare an APS package for your web application - and publish it in a public repository. To enable thousands of hosting providers around the world to provide your product to their customers. We will do it disinterestedly, that is for nothing, of course - if your project is non-commercial and open source.
This can be an online store, forum, content management system - in fact, in general, any web service. The main thing is that it works and is needed by people.
