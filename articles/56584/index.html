Version 1.0.4

Congratulations to all with the release of the next version of the wonderful framework Yii.

As always, the release is marked by the addition of a dozen or two new features and the correction of errors found in the last month.

Added by:
  • eAccelerator cache driver
  • cache driver for Zend Data
  • events beforeControllerAction and afterControllerAction
  • CExistValidator to check for data in the table
  • the ability to work with captcha, which is created by other controllers
  • Several new features when working with ActiveRecord:
    • working with tables without a primary key
    • A new type of communication - STAT, which allows you to receive statistical information about dependent objects
    • working with SQL Server
  • the ability to use TextArea (and not just TextField) in CAutoComplete
  • varyByExpression in COutputCache, which allows you to create different versions of the cache, not only for different routes (controller + action), get-parameters or sessions, but also for calculated values

As well as many other minor and major fixes and additions.

Read more about all the innovations in the changelog and on the forum in the announcement topic .
