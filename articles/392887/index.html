Windows operating system BSOD will receive a QR code for each type of problem

The blue screen of death OS Windows will become much more informative


More recently, BSOD looked terribly strict

For many users of the Windows operating system (different versions), BSOD is a nuisance, due to which the information and unsaved information already entered into the document is lost, and the computer has to be restarted. It is clear that the "blue screen of death" is only a consequence of the failure of the operating system or equipment. BSOD usually provides the problem code and a brief description of what happened (in some cases), but this information does not help much to non-specialists.

But the situation may soon change. Microsoft, as far as can be judged, is going to make it easier for users to identify the problem that caused the system to crash. To this end, when the BSOD appears, a QR code will appear on the screen, with which you can get detailed information about the cause of the failure. In addition, there will be a link to the page with a description of the problem. This was reported by one of the users of Reddit. In the continuation, there is a BSOD screenshot with a QR code (namely, a screenshot, the Windows version with the update was launched on the virtual machine).



As far as can be judged, a new type of BSOD appeared on the Insider preview Windows 10. Some users note that the QR code does not always appear, but there is a link to the page with descriptions of the problem codes. So far it is unclear whether this innovation will be launched for all users, or is it just one of the experiments.
