LibreOffice 3.3

The Document Foundation has released LibreOffice 3.3, the first stable release of this free office suite.

Among the new changes: added the ability to import and work with SVG-files, easier way of formatting the header pages and their numbering in Writer, improved navigation tool in Writer, improved ergonomics in Calc when managing sheets and cells, added Microsoft Works and Lotus Word Pro import filters .

In addition, the package includes many extensions, for example, PDF import, presentation console, improved report builder.

LibreOffice also includes all the new changes in OpenOffice 3.3: inserting standard PDF fonts into PDF, new font Liberation Narrow, increased document protection in Writer and Calc, new settings for importing CVS files into Calc, and much more.

A detailed list of new features is available on this page: libreoffice.org/download/new-features-and-fixes .

Users who have added a PPA should receive an update soon. Download the first stable release on this page: libreoffice.org/download .
