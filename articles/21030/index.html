Antivirus check

The best, but somewhat dumb, way to test the quality of antivirus software installed in the system is to enter some God-forgotten garbage dump or XXX resource. But this, so to speak, stress testing. And you can test the performance of your antivirus with the help of a test "code".

Just create a plain text file (.txt), copy a special test line into it.

X5O! P% @ AP [4 \ PZX54 (P ^) 7CC) 7} $ EICAR-STANDARD-ANTIVIRUS-TEST-FILE! $ H + H *

and click "Save".

If the anti-virus is working in the monitoring state, it should immediately display a message about placing the file in quarantine (or deletion, depending on the settings). If monitoring is disabled, you can scan the file. The result should be the same.
I note, ESET Smart Security with the test handled. What are your results?
