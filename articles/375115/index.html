How to make your The Pirate Bay with games and orgies: instructions

While the whole world watches with bated breath the fate of the most popular tracker The Pirate Bay (will return? Will not return? How will everything turn out?) ‚ÄúThe time for single trackers has passed. It's time for everyone to create their own. ‚Äù

And most importantly, it is very simple. It is necessary only:

1) download the source tracker

2) choose a hosting and upload the source there

3) install scripts

Advanced programmers can help develop a project on Github .

Detailed instructions for installing scripts


Even beginners can do it. You only need to find hosting with PHP from 5.3 and Apache version 2. The script will work using the isoHunt database.

Advanced webmasters can install their own torrent database. To do this, they will need sphinx 2.1 and higher, and mysql version 5.

After you have found a hosting and set up a good domain name for it, you need to upload the tracker's source to the server (and on some hosting you don‚Äôt even need to unpack them from the zip, the hosting does it yourself). The apache web server will be configured by default, its config lies in the source code in conf / example.htaccess. If you want, you can configure nginx - its config is in /conf/example.nginx.conf.

Those who make their base can read the instructions for configuring sphinx and create a database in MySQL from the dump /src/protected/data/schema.mysql.sql
Database dump with data can be downloaded from isoHunt .

After that, the remaining steps can be done through the web assistant, which you will see in the browser when you open your new server by its name. All - the tracker is ready!
