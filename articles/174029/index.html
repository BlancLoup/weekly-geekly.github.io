Accordion menu on pure CSS

Good day, dear habrazhiteli. To date, most of the harmonica menu creation techniques include the use of the :target pseudo-class. Problem of use :target is the inability to close any block in our menu, only the accordion will ‚Äúrotate‚Äù.



Example 1: drop-down list (checkbox)

Example 2: collapsing list (radio)


Therefore, I would like to show you an example of implementing the Accordion menu in pure CSS using the pseudo-class :checked . With this method you will be able to control the opening / closing of blocks. The menu will be of two types: fully pop-up and the display of one block when the others collapse. checkbox and radio will be used accordingly.


HTML markup


The menu itself will be in ac-container . checkbox , input ID.
<section class="ac-container"> <div> <input id="ac-1" name="accordion-1" type="checkbox" checked /> <label for="ac-1"> </label> <article> <p> ... </p> </article> </div> <div> <input id="ac-2" name="accordion-1" type="checkbox" /> <label for="ac-2"></label> <article> <p> ...</p> </article> </div> <div> <input id="ac-3" name="accordion-1" type="checkbox" /> <label for="ac-3"></label> <article> <p> ...</p> </article> </div> <div> <input id="ac-4" name="accordion-1" type="checkbox" /> <label for="ac-4"></label> <article> <p> ...</p> </article> </div> </section>
