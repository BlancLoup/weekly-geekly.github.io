The easiest way to start an encrypted chat


The idea for the service, as often happens, was born out of an unmet need. I often someone takes off sensitive information, such as FTP access, through an unsecured channel. If it is still possible to explain to people with whom you communicate often what encryption is, and what kind of software you need to install for it, then it is a thankless task to impose your paranoia on random interlocutors.

So Otr.to was born - a browser - based p2p chat with OTR encryption. To start a chat, just go to the main page, copy the generated link and drop it to the other person. Chat starts as soon as he opens it.

The project is open source, everything is done in Javascript, so that anyone can easily make sure that the chat is really p2p and the messages do not go to the server.

Due to numerous requests from workers, Otr.to has become overgrown with additional services.

Self destruct encrypted message . It may be necessary when you need to send something to a person who is not currently online, or just too lazy to start a chat. The message is encrypted on the client side of Javascript using the AES256 algorithm. The recipient decrypts it already in his browser. You can throw off the link and key in different channels.



Similarly, encrypted file transfer works. The file is encrypted in the browser before downloading. So even the one who has access to the server cannot read the contents of the file, since Keys on the server do not go away.

I want to note that Otr.to is not another Skype killer - it is not even a messenger, just a tool that allows you to make communication safer without giving up the usual means of communication.
