Rosetta Project



Within the framework of the Rosetta project, which is launched to create materials that facilitate the migration of developers from Flash to Silverlight, the Flash to Silver Guide Guide will be actively replenished.

Now Flash to Silverlight Guide consists of three parts:

* Fundamentals and Tools - talks about basic concepts and development tools
* MovieClip To UserControl - talks about the migration from MovieClip to Fash to UserControl in Silverlight
* Flex Components to Silverlight Controls - a large section on how to match Flex components to Silverlight.

In the next update of Flash to Silverlight Guide, we plan to add information on fonts and graphics, animation, code writing and working outside the browser.
