We test Panasonic 3D Image Sensor D-imager

Not so long ago (even before Kinekt appeared on the market), a 3D image sensor Panasonic from Panasonic came into our hands for a week. The sensor is designed for control systems that are built on the recognition of gestures .



By the principle of operation, this device is similar to Kinekt, but with a lower resolution for image output (160x120 px) and drivers only under Windows XP.


For a couple of days, our programmer Dmitry Sukhorukov wrote for him the simplest image grabber + socket server to push this image in Flash, and added D-imager support in CCV (http://ccv.nuigroup.com/) to try to use D -imager as a multi-touch input device.

On the remaining days, Flash clients were written, which can be seen in this video. It is a pity that the device was taken so quickly, we just started to get a taste!
