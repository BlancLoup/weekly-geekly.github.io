Windows Developer Program for IoT - now with Windows

Good day,

Today is important news. :) On April 2, on // build /, we announced a completely new and all-iron program - the Windows Developer Program for IoT (Internet Of Things). In the first days after the announcement, we received a huge number of applications, including from those users who already had on board Intel Galileo, which this program is dedicated to. The only issue that was not resolved until yesterday was the issue of firmware with Windows - and our colleagues from Intel responded very quickly, releasing the UEFI firmware. On Github program published instructions.




This release only supports Intel Galileo Gen 1 so far, but work on implementing Gen 2 support is already in full swing, and the following Windows images will be available from the Connect website. Also, it can and should be used by users already building magic devices with Galileo in order to be with the latest update.

In the Microsoft blog, we will gradually reveal this new topic for ourselves, talk very closely about how to use Galileo with Microsoft Azure cloud and, of course, Visual Studio.

PS You can buy Galileo, for example, on Amazon .
