Useful materials for mobile developer # 45 (March 17-23)

This week, interesting news rained down from cornucopia - GDC announced the fifth version of Unity, Unreal and CryEngine announced the transition to a subscription distribution model and a percentage of sales, Google introduced a platform for wearable Android Wear devices, Tizen SDK for Wearable came out, Flurry described what genres is best to monetize on Android.



CryEngine becomes available as Engine-as-a-Service

Starting from May of this year, indie developers will be able to access CryEngine for $ 9.99 / EUR per month without any royalties.

Unreal Engine 4 is now available for everyone.

Epic Games announced today that anyone can access Unreal Engine 4 for only $ 19 per month + 5% of the cost of each copy of a product made on the basis of UE4.

Announcement and pre-order Unity 3D 5

The developers of the Unity 3D game engine have announced a new version number 5. A lot of new features are promised.

Google announced SDK for wearable devices

Now it is possible to write programs for watches. Moreover, besides the already filled up rectangular clocks, round ones appeared!
