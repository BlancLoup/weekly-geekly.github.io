New Acquisition Options and Free Visual Studio 2015 Offers



In the previous article, we introduced you to the main announcements of the largest event about the development tools of Microsoft. Now let's talk more about the new features of the acquisition of Visual Studio.

As it became known on November 19, Microsoft created new proposals and options for the acquisition and use of tools, both for large companies and for small teams and individual developers.

Acquisition


To begin, consider the changes affecting the paid version of Visual Studio: Pro and Enterprise.

Now you have two options available for purchasing a Visual Studio license using a standard and cloud subscription.



The differences are that Visual Studio's cloud subscription is a way to purchase a Visual Studio license through the Visual Studio Marketplace. To purchase a tool in this way, you need to have an Azure subscription that will be billed. Subscription in the status of the trial period for these purposes will not work. Just as they are not suitable for payment and funds for the use of the cloud platform, issued by the programs BizSpark, MSDN, etc.



Another qualification is MSDN subscription access. If you purchase a Visua Studio cloud subscription for a year, then you get a product with all the benefits of MSDN. Accordingly, when buying a product for a month, MSDN is not provided.

What is the Visual Studio Marketplace? A new online platform that provides a catalog of extensions for all Microsoft development tools, and now the possibility of acquiring Visual Studio.

Learn more about Visual Studio offers by reference .

Free features for developers


For more than a year, developers have been able to use the free and fully functional version of the Visual Studio Community. And at the Connect conference this year, a new developer program for developers, Visual Studio Dev Essentials , was announced - the main components for the developer at Visual Studio.

This program is in the development stage, but it already offers a number of tools to ensure a full software development cycle, including tools and products from third-party companies, such as Parallels, Xamarin, etc. Cloud capacities will soon be available to support a variety of Dev / Test scenarios, renewable monthly.

For several months, the Dev Essentials program provides access to training materials, such as Pluralsight, Xamarin University, etc. As well as the possibility of obtaining priority support on the forums.

Access to all free offers for developers on the program Visual Studio Dev Essentials can be obtained on the member profile page.
