Sony Memory Stick XC

Sony has published the final specifications of the new high-capacity memory card format. The first information about the development of a new format, now known as Memory Stick XC, appeared in January at CES 2009. This format uses the new exFAT file system, which allows you to create on its basis memory cards with a giant capacity of up to 2 TB.





The specifications for the Memory Stick XC format include the creation of at least four different types of memory cards, including Memory Stick XC Duo, Memory Stick XC Micro, Memory Stick XC-HG Duo and Memory Stick XC-HG Micro. Memory Stick XC Duo and XC-HG Duo cards will be 20 x 31 x 1.6 mm in size, while Memory Stick XC Micro and XC-HG Micro will be 15 x 12.5 x 1.2 mm. All these types of memory cards will support MagicGate content protection technology. At the same time, the maximum and minimum data transfer speeds should reach 480 and 40 Mbit / s for high-speed Memory Stick XC-HG Duo and XC-HG Micro cards, and for the two remaining types, the same figures can be 160 and 15 Mbit / s, respectively.

New Memory Stick XCs use the same base slot as existing Memory Stick solutions. It is expected that Sony will find a use for the new standard of memory cards in digital photos and video cameras, as well as other portable devices. However, so far neither the Japanese corporation itself, nor SanDisk, which is also interested in this standard, does not report the timing of the appearance of Memory Stick XC memory cards and devices supporting them.
