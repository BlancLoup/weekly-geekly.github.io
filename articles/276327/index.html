Linux kernel support 2.6.32 LTS ends in February

Linux kernel 2.6.32.70 LTS is available for download now - this is the latest update for the 2.6.32 branch. In principle, the 2.6.32 branch is a kind of Methuselah - the period of its support was the largest among all the other branches. Update 2.6.32.70 LTS was submitted on January 29, 2016, and along with the update was posted information about the termination of support for the branch.

True, the developers of this version of the kernel say that there may be more updates before the end of support - but only if a problem is found that needs to be fixed.

The changes in 2.6.32.70 LTS affect 59 files, with 487 appendices and 187 deletions. Among other changes, it is worth highlighting the improved support for x86, s390, PA-RISC, and MIPS architectures, as well as EXT4, NFS, FUSE, and sysv file systems. Updated ISDN, SPI, SCSI, USB drivers, network drivers, network patch fixes added.

Those users who have not yet switched to the new kernel must decide whether to upgrade to 2.6.32.70 or start the transition to a newer version of the kernel, for example, 4.4 LTS. Sources 2.6.32.70 LTS can be downloaded from kernel.org or Softpedia .
