Make a UTP Connector Tester

I already wrote in my blog about the board. The rover is a universal board for various handicrafts. This is the Open Source Hardware Project. Previously, with the help of this board, we made mostly toys: they were different machines (for example, controlled by IR or moving out of a maze ), or an Easter printer for coloring eggs, or the video game Tennis and many others.

Now we have made a new project (using the Mars Rover card), which, I hope, can be really useful for the local network administrators. We made a simple tester connector UTP installed on the cable.


The principle of operation of the device is quite simple, and it is described on the website . There is also the source of the project.

The device has 8 LEDs and a connector for UTP cable. I inserted the cable into the device - if all 8 LEDs light up, it means that the cable is compressed correctly and all the contacts are there. If there is a bad contact with any pair or there is none at all or somewhere in the short circuit, then the corresponding pair of device LEDs will not light up.

The device only tests the connector and the nearest meter and a half of the cable. The state of the cable down the line does not affect the measurement results.
