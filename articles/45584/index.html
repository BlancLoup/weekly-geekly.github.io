Amarok 2 RC1 came out and is in the repositories.

Actually, this is not news, in Kubuntovsky RSS it was already on Monday, but still ... I think many will be interested.

In general, everything is cool, you can update. The second version of the full power of KDE4 technology and already sweetie. While the truth is not enough plasmoids, but oh well, let's wait ...

Update as follows :

Add a rep line to /etc/apt/sources.list:
deb http://ppa.launchpad.net/kubuntu-members-kde4/ubuntu intrepid main
Update cache apta:
sudo apt-get update
We put the second Amarok, its installation will replace the old one from KDE3:
sudo apt-get install amarok-kde4
