Canonical launches new Ubuntu 13.04 Raring Ringtail builds every day.



Ubuntu developers have announced plans to upload updated Ubuntu 13.04 Raring Ringtail builds (pre-release) every day. The final release is expected on April 25, 2013, but interested in this version, users can see every day how the system is changing. The fact is that Canonical decided to change the standard scheme for announcing new releases, and now there will be no alpha builds of the updated distribution.

But every day really laid out new and new versions of the system, available for the test. In addition, every two weeks will be laid out also distributions with a cumulative update, called "cadence" releases.

The first daily cadence builds of Ubuntu 13.04 are not too different from version 12.10, but this is not surprising, since only a few weeks have passed since the release of this version. Now, as you can see, Ubuntu 13.04 is in a very early development stage, and anyone interested in this can test the system by sending a feedback to the developers.

According to Mark Shatllworth , now developers have set goals such as increasing the battery life of laptop computers with Ubuntu, as well as optimizing the system so that Ubuntu can work normally on a wide range of devices, including smartphones, tablets and TV.
