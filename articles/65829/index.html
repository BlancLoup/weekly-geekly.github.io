Opened by Seadragon.com - hosting gigapixel images

The Live Labs team announced yesterday the discovery of the seadragon.com resource, which allows hosting gigapixel images for free (in fact, the size is unlimited). For example, the image below shows an image size of 73 gigapixels !

The work of the resource is simple, you specify a link to any image available on the web and get the opportunity to transfer a link to the deepzoom-page of this image with all the features inherent in this technology: smooth zoom and viewing super-large images. In addition, you can embed a deepzoom element with your image into your blog or resource.

All this works in any browser and on any system, thanks to the technology Seadragon Ajax, which does not require Silverlight and has the same functions. In addition, the Seadragon client is available for iPhone , which can be downloaded on the AppStore. The resource works in a cloud on Windows Azure.
