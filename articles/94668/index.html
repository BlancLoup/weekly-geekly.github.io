Widget with announcements of events to insert on the website or blog


A few months ago, I wrote about the catalog of IT events that I have been doing recently. The response was positive and constructive. Unfortunately, in the intervening time, it was possible to realize far from everything that was planned, and moreover, not everything that was offered in reviews and by mail. Nevertheless, we are not standing still and today I want to talk about widgets for posting on third-party sites.

The idea, among others, was proposed in the responses to the story about the site. At the moment, it was decided to implement only the basic functionality: the choice of groups of events and cities for displaying in the widget, as well as setting the block parameters - width and height (all other display parameters are inherited from the block in which the widget is located). In addition, it is possible to send the generated widget code and a link for further management of the widget to an e-mail.

Due to the fact that the entire site is based on Django, a simple base class was created to implement the basic functionality of the web widget, like the Feed class used to generate news feeds. I could not find a ready-made solution, so if someone has a similar problem - write, I will be happy to share the code.

Here you can try how, in fact, everything works. If you place the widget somewhere at home, please skip the link - just for statistics. And, of course, I will be glad to feedback and suggestions.

PS For our site, widgets are PR, and attract visitors. For those who post them on their website, this is primarily an opportunity to expand the information content of the site. But in order for the benefit to be mutual, soon we plan to launch an affiliate program, under which we will deduct %% for the valuable actions of visitors.
