Release 3CX v15.5 Update 3 with new features and important changes.


The other day we released 3CX v15.5 Update 3 update and at the same time introduced new licenses - 3CX Standard from 4 to 16 simultaneous calls (approximately, organization up to 50 employees). These licenses are absolutely free for all users of the former free 3CX edition, which was called PBX Edition.

New Standard free license offers unlimited SIP trunks (external lines) and extensions (users), 25 simultaneous web conferencing participants and free hosting on Google Cloud! All current PBX Edition licenses (with an active subscription to updates, ie, not expired) will be automatically converted into annual licenses Standard, 16 simultaneous calls.

The number of simultaneous participants of the built-in web conferencing system 3CX WebMeeting has also been increased: up to 25 participants in the Standard, up to 100 participants in the PRO, and up to 250 in the Enterprise.

We are once again starting to officially support the Grandstream phones (so far in Beta status): GXP16XX , GXP17XX and GXP21XX . Implemented the following functions:

  • Autotune on the local network, via the SBC utility and via the RPS server (PnP and manual configuration)
  • Certificates Let's Encrypt
  • UaCSTA protocol supported
  • Automatic dialing from application (CTI technology)
  • 3CX Certified Firmware Updates

Answers on questions


I am having difficulty downloading the update

  1. Go to Parameters - All Parameters and find the parameter DOWNLOADS_URL
  2. Replace its value with https://downloads-global.3cx.com/downloads/v155 and click OK
  3. In the management interface, go to the Updates section and update the system.

What does ‚Äúsupport‚Äù of Grandstream phones mean?

Support means that in addition to basic SIP communications, 3CX supports advanced device management:

  • The BLF-GXP1610 / 15/20/25 indicators do not have them, on the others they are centrally configured from the 3CX interface.
  • Voice codec priority control - centrally from 3CX interface
  • Language on the phone display - centrally set in the 3CX interface
  • Time zone, date and time format - centrally from 3CX interface
  • Password access to the phone's web interface - centrally generated and set in the phone - excellent security guaranteed!
  • The default call transfer method (followed, unaccompanied) is centrally set in the 3CX interface
  • Ringtone - centrally set in the 3CX interface
  • The behavior of the LED power indicator, display backlight, screensaver timeout - centrally set in the 3CX interface
  • LLDP support and VLAN configuration - centrally set in 3CX interface
  • Your company's logo on the phone display is centrally loaded, but only in the GXP17XX and GXP21XX series.
  • Supports UDP / TCP / TLS signaling and SRTP voice encryption

Perhaps in the future, however, it depends not only on 3CX, but also on representatives of the company Grandstream. At the moment, it is impossible to promise anything.

What does "out-of-box TLS support" mean?

This means that when using the FQDN from 3CX, a security certificate will already be installed on the system. You just need to manually configure the phones for SIP communication over TLS. However, if you are ready to wait for Update 4, the automatic work on TLS will already be implemented in the IP phone auto configuration templates (at the moment we expect the phone vendors to be ready for the firmware).



What are the important limitations of the Standard version?

The FQDN of the 3CX server name in the Standard version is issued only by 3CX, i.e. own domains are not allowed. However, if at the time of updating to Update 3, the user has already used his own FQDN, he will continue to use it. But as soon as the user unplugs this FQDN from his license key, in the future he will be able to use only the FQDN from 3CX.

SMTP server for sending e-mail notifications only from 3CX. However, if at the time of the update to Update 3, the user has already used his own SMTP, he will continue to use it. But as soon as the user changes his SMTP to SMTP 3CX, he will receive a corresponding warning, and then he will not be able to go back.

Therefore, if you need maximum flexibility and independence, use the Pro edition.

What happens to my current commercial license Standard?

Absolutely nothing, all the conditions will remain the same. In addition to the FQDN and SMTP server (and only if you decide to change these services to services from 3CX, as mentioned above).

If your subscription to updates is still active (for free licenses it is active the first year after the key is activated), your PBX Edition license will ‚Äútransform‚Äù into an annual license of 16 Standard.
