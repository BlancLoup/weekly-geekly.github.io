Features of installation and update 3CX v15

After the release of the new version of 3CX Phone System, we received many questions related to installing, testing or updating existing systems. We have prepared a review article in which we explain how best to act in a given situation.

In 3CX v15, we significantly simplified the installation process and increased server security. 3CX can now create and maintain the FQDN of your server, as well as generate and update a free trusted SSL security certificate.
However, in order for the installation process not to cause difficulties, it is necessary to take into account a number of features of the new version:

  • Testing or Demonstrating v15
  • Restore backup
  • Translation test version v15 to commercial
  • Upgrade an existing installation to v15
  • Successful update rules

Consider these features in more detail.

The basic principle: if you decide to use the FQDN name created for you by 3CX, the license key of your v15 installation is tied to your chosen FQDN server.

Testing or Demonstrating v15


Fill out the form with a valid e-mail address, and you will receive a free 3CX PBX Edition license key. If you want to know about the features of the free edition of 3CX, see the comparison of editions .
For the initial 3CX evaluation, use only the free 3CX PBX Edition key . Do not use your affiliate NFR or purchased commercial key!
During installation, you will be prompted to specify a subdomain that will form the FQDN of the 3CX server name. Choose a test subdomain. Do not specify your main subdomain, which you will use in your work in the future! Use your main subdomain only with a commercial key after testing is completed. For example, if your company is called abclimited , use the abclimitedtest subdomain for tests. Now, if you purchase a commercial key, you can use your main subdomain abclimited with it .
However, there is one exception - if you plan to buy a license after testing, you can upgrade your existing free key by saving the domain name. For example, you started testing on a free key, specifying the main subdomain abclimited . After testing, you decide to purchase the system. Then you just need to upgrade the free key to the commercial one. The subdomain will remain the same.

Restore backup


If you installed 3CX v15 with a license key, do not restore the configuration from a backup with a different key! The exception, as mentioned above, is that you upgrade the free key to a commercial one.
If you restore a v15 backup with a different key, you will encounter problems with the FQDN name and video conferencing operation. Changing the license key or SSL certificate requires a complete reinstallation of the 3CX server.

Translation of test version to commercial


If you have tested the system and decided to switch to the commercial version, then the most correct way is to purchase an upgrade of the free key 3CX PBX Edition.

  • The end user of the system must tell the 3CX partner their free upgrade key. At the same time, as was said, the FQDN of the 3CX server can be left unchanged.
  • A 3CX partner should order a new product key only if the end user has not previously tested and does not have a free key. When ordering a commercial license, the partner should clarify whether the user does not have a free product key and upgrade this key.

Upgrade an existing installation to v15


If you want to upgrade your existing previous version to 3CX v15, use your existing license key.

  • If you use your own domain name, use your own trusted SSL certificate (as it was in v14). You can import it during installation. Important: 3CX v15 works only with trusted, not self-signed certificates!
  • If you want to use the FQDN name and SSL certificate from 3CX, specify your main subdomain, for example, company name ( abclimited ). The rest of the domain name (3cx.eu, 3cx.ru, etc.) you can choose from the options offered. Your FQDN of the type abclimited.3cx.eu will be assigned to you and maintained for you, provided that you renew your subscription to 3CX Maintenance updates.

Successful update rules


Before upgrading 3CX to v15, first of all, make sure that you have an active subscription to updates. After that, proceed as follows:

  1. Install all available Windows updates. This may take some time and slow down the transition to the new version of 3CX, so it is advisable to upgrade the system in advance.
  2. Back up the current version.
  3. Uninstall the current version of 3CX.
  4. Download and run the 3CX v15 installation file.
  5. The 3CX PBX Configuration tool wizard will appear, prompting you to specify the location of the backup. Specify the path and file name of the backup. The backup contains the license key of the system, which will be used to activate v15.
  6. You will then be asked to specify the FQDN of the server. If you use your own FQDN, you will be prompted to import your own trusted SSL certificate. If you decide to use the FQDN from 3CX, use your main working subdomain. Your license key will be linked to the selected FQDN.
