Changing content improves search engine rankings.

The fact that Google loves more updated websites, we know so. Jerry Cullen decided to find out exactly how.

He created two practically identical culinary sites, with the same saturation with keywords, the same external links and so on. The difference was only in a small text block, which on one of the sites was filled with news in the topic. The static site received 55 visitors in two months, the site with a dynamic text block received 122 visitors from Google.



Experiment Report (PDF)

Of course, Jerry did this experiment for a reason - he promotes the NewsBox service, which actually showed dynamic content on the second site. News is selected by editors manually, but it costs $ 99 a month and is not suitable for everyone.

What should everyone else do? If you dig thoroughly - any website has resources. Let's say a blog post may show at the end a automatically generated list of similar articles, sorted randomly . Thus, people and search engines will be something interesting to read. If there is no such plug-in, you should write it.

The same idea will work on any other site with a large amount of textual content, say on a site with articles or on a forum.
