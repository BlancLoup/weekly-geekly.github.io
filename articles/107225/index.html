Banshee will be the default media player in Ubuntu 11.04

Banshee ( banshee.fm ) will be the default media player in Ubuntu 11.04 Natty Narwhal.

This decision was made during the session of the Default Package Selection at the Ubuntu Developer Summit.

UPNP support, easy installation of codecs, the Ubuntu One Music store, and support for playlists in the sound indicator are issues to be addressed by the project.

By tradition, Ubuntu, Banshee in the menu will be called "Banshee Music Player".
