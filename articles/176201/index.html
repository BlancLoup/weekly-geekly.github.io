Monosnap updated to version two

A few months ago, we wrote about our Monosnap application (see Personal Screenshot Software ) and received hundreds of helpful suggestions and tips.
Since then, the application has continued to evolve, and, opening the company's blog, I would like to tell you about the second version of the program.


So, if you were not previously familiar with Monosnap, I‚Äôll briefly explain - this is an application that helps you create, edit and share screenshots over the network. After the previous review, the application received the function of recording video from the screen, broadcasting the desktop, downloading files and working with the webcam.

Let's look at each function in more detail.

Work with files


When developing, we often exchange files within the team ‚Äî be it beta versions of the application, statistics reports, or document scans. At the same time, we often ran into restrictions. In Skype, transfer is only possible when both people are online, Dropbox requires that the file remain on the hard drive in the same place, and looking at the Droplr and Cloud App, we realized that we can provide a more complete solution.

So, to share the file via Monosnap, you need to drag the object to the application icon (or dropzone in Windows) and a link to the page where you can download it will appear in the clipboard. So far, the service is in beta testing and there are no restrictions on the place in our cloud.



Broadcasting


Starting from version 1.5.0, we, initially as an experiment, added the ability to conduct live broadcasts of the desktop. The first implementation took us two weeks, and statistics showed that the function is in demand.

The broadcast can be viewed on our website. Playback is done via web sockets using HTML5 and works in any modern browser, both on a computer and on smartphones. In the latest version, this mode has additional tools - so, during the broadcast, you can put an arrow or circle the discussed fragment with a pen.

Screencast recording


When developing, we often had situations when a screenshot was not enough to explain the problem. Therefore, in recent versions, we have added the ability to shoot and share video clips.



You can simultaneously broadcast live and record video to later upload it to the network. The application allows you to upload videos to both the Monosnap server and Youtube.


In addition to global changes, the editor also received new functions - each tool has several new modes, including filled rectangles and ovals, a translucent ‚Äúmarker‚Äù for text selection and others. From the last review it also became possible to upload images and files to the ftp, sftp and webdav server. In the next release, Amazon S3 will be added to this list.
