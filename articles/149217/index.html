Winners of the contest Ubuntu App Showdown have become known


Today , information was published about the applications that took the first three places in the competition Ubuntu App Showdown ( competition announcement on Habr√© ). They are:

Lightread is a lightweight client for Google Reader with support for offline work and integration with Unity tools.


Fogger is a tool for creating desktop applications from a web application running in a sandbox. Special JS API allows the application to interact with the system.
