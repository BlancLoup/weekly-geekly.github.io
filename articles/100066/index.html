FreeBSD 8.1-RELEASE Available

Now release 8.1 has been officially announced and published release notes.

Of the most interesting, not counting the bug fixes and new drivers:
  • serious ipfw update (now ipfw3)
  • ZFS update and new ZFS kernel bootloader
  • pfil may contain separate filters for VIMAGE jails
  • added framework (Highly Avalable STorage) for a fault-tolerant disk system over TCP with a master-slave scheme
  • A new module GEOM_SCHED has been added to GEOM - the ability to select an I / O manager for individual disks

Summary of changes: www.freebsd.org/releases/8.1R/relnotes.html
Detailed description of the changes: www.freebsd.org/releases/8.1R/relnotes-detailed.html
