SDK and Frameworks

iOS

For iOS developers, the latest version of our SDK is available for download. Thanks to simple instructions, you can easily import the SDK into your project, and classes and functions are already available for you, with which you can perform almost all operations with the API.

Also, you can find documentation , quite familiar methods describing all classes and functions in the SDK.

Windows phone

As in the case of iOS, a development framework for Windows Phone is available for developers. The SDK is very easy to add to the studio and now you are ready to create your own application. SDK documentation is available here .

Android & Web / Facebook

At this stage, the frameworks for these platforms are in development and, I hope, in the near future you will be able to see and use them in your projects. Samples are now available to help mobile developers; for example, you can integrate your application with QuickBlox .

Samples

Samples in the QuickBlox methodology are open source programs available for download and modification to the QuickBlox development community. We provide 2 types of samples:
  • Super Samples
  • Simple samples

As the name implies, super samples are separate and ready-to-use programs. You change the data (authorization keys), change the design, add your functionality and your application is ready for release. Yes, everything is so simple! Yes! We offer you open source large applications that are ready to be used.

Simplsems are small pieces of code that describe the operation of any QuickBlox functionality. For example, username or message in the chat. Everything is simple and clear enough for developers.

Samples can be found here and they are presented for the following platforms:
  • iOS
  • Android
  • Web / Facebook
  • Windows phone
  • Phonegap

The samples considered the following functions:
  • Chat on the map
  • Determination of coordinates
  • User authorization
  • List users
  • Sending - receiving push notifications
