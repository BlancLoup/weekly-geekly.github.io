ReSharper 7.1 released



We have been working tirelessly for the release of the next version of ReSharper. And now - it happened! The official, just released version 7.1 can be downloaded here .

There are many reasons to upgrade to version 7.1.

  • Since release of version 7.0, more than 300 defects and performance issues have been fixed ! Especially noteworthy are the improvements when working with SharePoint and ASP.NET MVC projects, as well as with resource files.

  • Improved code formatting : the most requested improvements have been implemented in this important area. So, ReSharper 7.1 can format comments from XML documentation and method call chains. We recently wrote about this in a blog devoted to .NET-products JetBrains.

  • It is now easier to export the results of the inspection of the ReSharper code and distribute them among team members . In addition, you can now copy the results of the inspection code to convert them into work items. For more details, see the blog post ‚ÄúEasy sharing of ReSharper inspection results‚Äù .

  • Windows Phone 8 support : if you already use the Windows Phone 8 SDK for developing applications, ReSharper 7.1 will help you do it faster and better. How exactly? Read the recent blog post on Windows Phone 8 support .

  • VB.NET support has been improved and now includes Extract Class refactoring, new quick fixes and improved IntelliSense.

  • XAML support has been significantly expanded , including IntelliSense, character input, naming style control, and code generation. Our XAML support specialist recently told all that he knew about it .

  • Support for the implementation of the INotifyPropertyChanged interface covers more usage scenarios that are relevant to developers of Windows Forms, WPF and Silverlight applications.

  • And other improvements , including fixes in a number of refactorings, improved support for Entity Framework 5, ASP.NET, ASP.NET MVC, and JavaScript.

You can upgrade to ReSharper 7.1 for free if you have a license for version 7, or a license for version 6, purchased not earlier than June 1, 2012.

If you use ReSharper plugins, do not worry - their compatibility with version 7.1 is being tested. Most of them will be available for download in the next few days.

Download ReSharper 7.1 and tell us what you like and what not!
