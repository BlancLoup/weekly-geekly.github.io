Google stops supporting browser IE 9

Google has reminded Internet Explorer users of the current rules, according to which Google Apps guarantees correct operation only in the latest browsers. In particular, the latest version of Chrome is supported, as well as the last two versions of Firefox, Internet Explorer and Safari.

In connection with the recent official release of Windows 8.1 and the Internet Explorer 11 browser, the third from the end version of Internet Explorer remains, ie IE 9. Although IE11 was released three weeks ago, Google only now officially stops testing all of its services for IE9. Thus, more than half of Internet Explorer users (taking into account those using IE8 and IE9) remain in a suspended state, as they may encounter incorrect web applications at any time when logging in to Gmail or another Google site.

In the coming days, all IE9 users will see a notification when they visit Google sites to stop supporting their browser.
