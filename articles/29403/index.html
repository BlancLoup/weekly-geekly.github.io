Loading covers from Last.fm in Amarok

I bring here my music collection in order.
I use Amarok as a music player, which I recommend to all Linux users. Among other goodies, this player has the ability to download album art from Amazon.com. And everything is great if you listen to only popular ‚ÄúInternational‚Äù music, but if there are Russian artists in your collection, like for example with me, then Amazon will hardly help you.
Here, actually, in search of a solution to this problem, I came across a Cover Fetcher script.
This is the script for Amarok, which is installed via Tools-> Script Manager.
The script allows you to download covers from Discogs.com and more useful to Last.fm, by clicking on any track in the playlist and selecting Fetch Cover-> For Currently Playing Track.

That's all. Ordered collection to you =)
