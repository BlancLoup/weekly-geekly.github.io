First PyPy3 beta released with Python 3 support

The PyPy developers presented the first beta of the upcoming PyPy3 release. This is the first release to support Python 3 (3.2.3).

You can download the link . There are several known problems in beta, including a serious performance regression (for example, for the time being they write that execution of the same program can be 37 times slower than it was in PyPy 2.0.2). Problems promise to fix to the final release.

X86 versions available for Linux 32/64, Mac OS X 64 and Windows 32; ARM supported with Linux 32 - ARMv6 (Raspberry Pi) and ARMv7 (Chromebook).

Source: PyPy Status Blog
