HttpTorrents - download torrents through a browser



Downloading files with torrents is very convenient, but what if your friend, a little computer savant, needs to download something with a torrent?

httpTorrents is a new service that allows you to download popular torrent files and then download them from the service via the HTTP protocol.

Not all files are immediately available for download via HTTP, and if the file you need is not found on the server, it is automatically placed in the list of future downloads. Now this service is integrated with KickassTorrents . Perhaps in the future the service will be integrated with other torrent trackers that allow you to view the torrent hash.

How to use the service:



Enter the torrent hash and find it.

Choose from the list what you want to download.

Download files from the server.
