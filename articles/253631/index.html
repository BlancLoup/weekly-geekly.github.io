10 relatively unknown facts about K-Meleon

  1. I have already written this, but it is not a sin to repeat: K-Meleon has existed for 15 years, since 2000, being one of the oldest browsers retaining its relevance and continuity up to our days.
  2. K-Meleon downloaded from the official page on sourceforge more than 7.6 million times.
  3. The most popular K-Meleon ... in Russia! Approximately 20% of all downloads were made by users of our country.
  4. From the very beginning, K-Meleon got into the well-known anti-monopoly panel Browser Choice , where it remained the only browser of the ‚Äúsecond cohort‚Äù, kept on the list throughout the term of the mandatory choice of the browser in Windows (completed in 2014).
  5. K-Meleon repeatedly turned out to be the easiest browser in terms of memory consumption among current browsers and is such today for all current versions of Windows (including XP), except for Windows 8, where the data is still inconsistent.
  6. K-Meleon today is the most customizable browser, where the user can not only change the browser's subtle settings, but also completely customize the menu, toolbars and the look of the browser. Part of the procedures involves configuration files, which have not the most obvious structure and syntax for a beginner, while some can be done by simply copying the necessary files to the right places.
  7. The browser has its own macro language that can not only manage the internal commands of the program, but also integrate third-party javascript code, as well as provide full interaction with the file system and the OS registry for writing and reading, which allows you to connect virtually any external programs to the browser From K-Meleon, you can make an almost complete shell of the OS).
  8. The browser usually had a single main developer, the first of which was Christian Thibault, after which the project went several times into other hands until it got to the French programmer Dorian Boissonnade, who is still working on it so far as a hobby time.
  9. K-Meleon in the new version 75 has become partially compatible with Firefox extensions.
