AeroSQL - MySQL database management

AeroSQL is an open source ( GPLv3 ) web based tool for managing databases on a MySQL server using PHP and ExtJS 2.2.1 . The key feature of this product is that it has an intuitive interface for accessing data in database tables. Sample results are displayed as tables. Tables are editable: you can edit cells and save changes directly to the server. Screenshots and video under the cut.

In this version, the query browser has become more functional: administrative tools (backup and recovery), table management (creation / deletion), etc.

PS I have not got accustomed, phpMyAdmin and Sypex Dumper are closer to my soul.
