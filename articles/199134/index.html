Firefox 25 release

Firefox for desktops has become available for download. What's new in this version:


  • The search string (Ctrl + F) is now shown for each tab separately (the previous input is saved for each tab separately - a single buffer for all tabs is no longer used);
  • If Firefox has not been launched for several months, a form is displayed to the user with a proposal to transfer work history and settings from another browser;
  • When reset, Firefox now saves and transports windows and tabs you have opened;
  • Ability to set the iframe content in the current block (inline), without downloading an additional file. Content is indicated through the new attribute iframe srcdoc = "" ;
  • Web Audio API support ;
  • Support for CSS3 background-attachment: local ;
  • Implemented a series of new functions from the ECMAScript 6 specification ( 1 , 2 , 3 , 4 , 5 );
  • Fixed an issue where when opening a new tab page thumbnails are shown empty or there is no content in them.
