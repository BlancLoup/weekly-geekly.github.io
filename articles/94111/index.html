Instructions on firmware


  1. Download the firmware here for 5530 , here for 5800 and here for the N97 install it on your computer
  2. Modified files that need to be selected and downloaded here for 5530 , here for 5800 and here for N97 are moved to the folder where the firmware is installed (by default C: \ Program Files \ Nokia \ Phoenix \ Products \ RM-504), agreeing to replace files
  3. Here we download the program for the JAF firmware (there are instructions and reviews).
  4. We connect the phone in PC Suite mode
  5. After installation, run the JAF program from the desktop, ignoring the message BOX DRIVER NOT INSTALLED , it does not hurt us.
  6. Go to the BB5 tab, click the Use INI checkbox, after which our window will pop out, where we select our phone model. Ok, uncheck the CRT 308, set to: Manual Flash, Dead USB, Factory Set.
  7. Turn off the phone without disconnecting. We press in the Flash program, we agree with the error and by briefly pressing the Power button on the phone, we turn it on. Then the program will do everything for you. After a few minutes, you will receive a new phone.

We do not forget to make backups of flash drives, contacts and necessary programs. Thanks for attention.
