Kolobok in SMF

By default, the smiles in the SMF , to put it mildly, are not very beautiful, so we decided to replace them with something more decent. And I wanted one of the best sets of smiles Kolobok . A lengthy search for a positive result did not work, so it was decided to adapt the kit for this engine personally.



You can install as follows:

  1. Download a set of smiles for dark or light themes.
  2. Go to Administration - Modification Packs - Install Packages and download the archive.
  3. After downloading, select Install and install the package.
  4. Now go to Administration - Smilies and icons - Installed smiles and click the button Install new smilies.
  5. We come up with a name for the set, specify the KolobokLight or KolobokDark folder (depending on the package you chose) and, if necessary, make the set used by default by ticking the appropriate box.
