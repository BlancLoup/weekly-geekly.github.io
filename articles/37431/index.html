Firebug lite

I look for the public passed unnoticed by such an event as updating Firebug-Lite to a qualitatively new level. Now we have a similar FF FireBug tool in Opere, Safari and rejoice in IE.
To connect it is enough to put on your page.
<script type='text/javascript' src='http://getfirebug.com/releases/lite/1.2/firebug-lite-compressed.js'></script>
Or download on offsite and use a local copy. Of course, compared to the FF plugin, it undermines, but the main tools are available - Javascript console, DOM tree browser, HTML tree browser, CSS viewer and runner, Script viewer.
