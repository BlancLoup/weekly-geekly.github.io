Well, Centos 5.9 was released

This is an amazing and quick release, less than six months later, as it was after the release of the next version of rhel (only 10 days passed).
The length of the centos releases is probably the main pain of the community in recent years, partly the problem was solved with the help of a continuous repository, but there were only security-fix

From the interesting I added:

1) Added Java 7 and ant 1.7 (packages ant17, java-1.7.0-openjdk). Javista servers with centos 5 rejoice.
2) Now you can use rsyslog version 5 (rsyslog5 package).
3) Surprisingly, the redhat himself guessed to add native support for storing postfix users in mysql. Many years ago, we had to compile postfix handles for this, and then centos added the postfix-mysql package to the repo, and now it turns out that it is not needed.
4) Added drivers for hyper-v, probably performance happiness for those who have centos vps on hyper-v.
5) Switched to samba 3.6, you can use smb2. In the config, specify max protocol = SMB2.

Congratulations to your favorite distribution with a minor release.
