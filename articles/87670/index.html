Long-awaited Windows Phone 7!


The first 15-20 minutes of the Windows Phone presentation repeated what we saw at the WMC conference in Barcelona a month ago - an introduction to the platform, interface concept, sections.

It was finally confirmed, the applications on Windows Phone 7 are FULLY Silverlight 3 applications, not ‚ÄúLite‚Äù and not some subset, but completely the same. Plus, additional features like accelerometer support. For high-quality games, the XNA framework, familiar to Windows and XBox360 developers, is offered.

The fun began with the demonstration of new things. The first such thing was the AP content reader application. The application allows you to read news, comment, view images and videos.





The application running receives notifications of hot news.



There is no screenshot, but the application is very cool monetized: at some point, an animation appears and an advertisement that is not interfering with navigation is shown. This time Ford. This is a demonstration of how you can make money on free applications for Windows Phone 7.

After the mobile version of AP, they showed the second application - it turned out to be a personal diary with the ability to record, attach pictures and all that stuff. A simple but very nice application.



Next new feature is Windows Phone. This time related to images. Windows Phone 7 has the ability to use third-party image editing providers! That is, you write a certain plugin and Windows Phone consistently uses it to perform some actions with the image of the user. The user thus gets the opportunity to expand his phone to the level of a mega-editor.

Below: a list of providers, the Colorizer provider is selected, which leaves in the photo in color only those parts that the user specified. The image can be scaled using multitouch and naturally draw directly on it at any scale.
