BlackBerry 10+ Free Marmalade SDK



Marmalade is a popular tool in the field of cross-platform development, together with Blackberry announced the launch of a new offer (well forgotten old ), thanks to which there is a chance to get a free license Marmalade SDK (Indie $ 499) for a year and become one of the exclusive Blackberry devices Dev Alpha to test your applications.

What is needed for this?


1. Register on the Marmalade website.
2. Get Blackberry Vendor ID, you can do it here .
3. Download a temporary license . (Trial 30 days).
4. Submit the app to BlackBerry World for Blackberry 10, created using your existing Marmalade SDK or temporary license in draft mode or for final review.

What can I get?


1. If you upload the application to BlackBerry World in drafts, and it passes the test that was really developed with the help of Marmalade, you can get the Marmalade BlackBerry Indie License - a six-month license that allows you to use the Marmalade SDK for development only for Blackberry (Playbook OS and BB 10 ).
2. If the application is poured for consideration, and it passes the test for sale, you can get Marmalade Indie License - a one-year license that will allow you to develop applications for a wide range of mobile platforms including Blackberry, iOS and Android.
3. The first 300 developers who upload applications on BlackBerry World for review or drafts will receive a Blackberry Dev Alpha device for testing.

For those who have already purchased the Marmalade Indie License , there is a chance to extend it from the end date, if your application passes the test and will be available for sale on BlackBerry World.

Attention, only those who have not participated in previous Marmalade-Blackberry promotions can take part in the offer.

Offer valid from May to September 2013.

Marmalade at a glance


Marmalade is a cross-platform tool that allows developers to expand the scope of applications for a large number of devices. Powerful SDK allows you to apply one source code to function on different operating systems, thereby increasing the number of potential users of different platforms, saving the time and effort of developers, which gives you the freedom to create amazing applications and games.

Marmalade SDK provides maximum performance. Working on (C ++), RAD (Lua) or HTML5, Marmalade allows you to quickly deploy and test games / applications on iOS, Android, BlackBerry, Roku, Windows Phone, Windows and Mac desktop PCs, as well as on certain Smart Tvs models (shortly time).
