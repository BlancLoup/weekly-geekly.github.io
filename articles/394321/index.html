Windows system requirements raised for the first time since 2009

The minimum amount of RAM increased from 1 to 2 GB



Design a new start menu

The July update of Windows 10 Anniversary Update will bring many innovations, including more advertising in the Start menu and the mandatory activation of the TPM 2.0 crypto processor. It is included in the minimum configuration of the computer under Windows 10 Mobile and Windows 10 on desktop PCs. For Windows Server 2016, this dubious option will remain optional.

For many users, it‚Äôs more important not so much the loss of control over their computer because of TPM, as the increase in the system requirements for Windows 10, because this happens for the first time since 2009. The minimum amount of RAM with Windows 10 upgrade has been increased from 1 to 2 GB .

Many still remember the problem of the dramatically increased system requirements of Windows Vista in 2006: a processor with a minimum clock frequency of 1 GHz, 1 GB of RAM, DirectX graphics, 40 GB of disk space plus 16 GB of free space was required!

It was an extremely sharp increase that many could not afford. Since then, Windows system requirements have remained almost unchanged. In 2009, with the release of Windows 7 in the 32-bit version, the disk space requirement was only slightly increased from 15 to 16 GB, and in the 64-bit version, the minimum RAM was increased from 1 to 2 GB.

Windows 8 and Windows 10 (at first) were also satisfied with the old system requirements of the sample of 2009, but after updating the Anniversary Update for the 32-bit version, the amount of RAM increased to the same level as in the 64-bit version, that is, 2 GB.

However, the minimum system requirements are mandatory only for official PC assemblers. In fact, the system will run and run even on computers with less RAM.

In addition to RAM, the requirements for the allowed screen size for the mobile version of Windows 10 Mobile have changed. Previously, it was allowed to install only on devices with a screen up to 7.9 ", now the bar has been raised to 9". On devices with a diagonal of 8 ", only Windows 10 was allowed to be installed, now the bar was lowered to 7", giving OEM assemblers a choice.
