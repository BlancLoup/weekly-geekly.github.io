Video clip


Following the results of shock rendering, it was decided to make a video.



Unlike last time , to create this video, it was necessary to shovel a much smaller amount of information. It turned out that one dump from the GIS-Lab area (RU-ORE-140122) and ten daily OSM diffs were sufficient .
The same processing scheme was used:
  • Filtering and sorting data homemade program
  • Homemade render in SVG
  • Convert to BMP via ImageMagick
  • Build video in avidemux

Additionally were used
  • Maperitive to create a starting frame (state of the map at noon on January 22)
  • ImageMagick for fade in / out

An option was considered to render all frames in Maperitive or equivalent, but there is no highlighting of fresh changes, so the appearance of houses and other minor changes would be almost imperceptible. In addition, it would be necessary to pull out all versions of all relations from the database.
