Multitouch Mac OS X where? Already here!


This development, oddly enough, was published not from the walls of Apple , but it perfectly reflects what can be expected from the corporation in the near future. Especially considering the fact that only one guy, Christian Moore , made it work on a simple MacBook .

Its development - the Lux framework, is an open source program and allows you to work in Mac OS X Leopard using a multitouch controller. In truth, as Christian himself says, this framework is cross-platform and can even work in a browser, allowing mixes of various user interfaces.

Gizmodo has a brief interview with Christian about how this thing works and what to expect from it.

Gizmodo : Tell us more about Lux. This is a framework that anyone can use, right? Under any platform?
Christian : Well, yes, this is an open source framework I worked on to experiment with the user interface. This project does not focus on any one platform. The video shows the work of AS3 under the flash version 9 on Mac OS, but the framework can be imported under any OS.

G: Why flash?
K: Because you can quickly make a prototype on it. Anyway - the whole program is divided into several parts. One C ++ application tracks hand movements and commands flash or WPF (here we mean the Windows Presentation Foundation ‚Äî the graphics subsystem included in the .NET framework 3.0), or another C ++ application or ... anything. Multitouch can be activated in any environment, for example Cocoa.

G: So you can take this framework and make multitouch applications work on Mac OS X or Windows?
K: Yes. We have an Xcode developer for photo and video applications - all this will be available soon, as well as the touch-tracking application itself. But using Flash technology, as in this video, was the fastest way to get the framework to work.

G: How many people were involved in the development:
K: I wrote the core of the system, and four people from the NUI group helped create the demo video.

G: What computer is it all running on? It works damn fast.
K: Just a MacBook.

G: And the multitouch screen?
K: I use ~ box from naturalui. I know that this is a technology of the ghetto level, but I developed most of Lux on this. And it works great.
