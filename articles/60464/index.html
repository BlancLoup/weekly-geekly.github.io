Arora 0.7 - Webkit browser

A new version of the promising browser arora written in Qt4 using the Webkit engine has been released.



One of the most interesting innovations is the support of Open Search with the ability to add various search engines (previously only search from Google was)

Also added the ability to enable ClickToFlash mode. When this mode is enabled, the flash plugin is not loaded automatically, and instead of flash content, a button is displayed, which you need to click to view the content.

In version 0.6 there were problems with the disk cache; after solving them, the ability to adjust the size of the disk cache was introduced.
