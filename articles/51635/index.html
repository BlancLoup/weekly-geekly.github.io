SDK 1.1, release 1

After many owners of android-based devices began to receive an update notification (which calls itself Android 1.1), an official message appeared that a new and fresh SDK 1.1, release 1 is now available for all Android developers.

Despite the fact that the system will work fine with the old versions of the SDK (1.0_r1 and 1.0_r2), who will not want to try the new API features ? The accompanying text of the RTFM message will not succeed in finding major changes, but according to Google, all the existing ones will be useful (for example, a new emulator for testing written applications).
