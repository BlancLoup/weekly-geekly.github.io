Introducing 3CX V15 SP5 Beta and New Subscription Annual Licenses

3CX V15 SP5 Beta


We are approaching the release of a new 3CX V15 SP5 service pack! In the meantime, we offer our most active users to test SP5 beta build. We are looking forward to the reviews, because we want to take into account all your wishes! What does the V15 SP5 include?

New features 3CX V15 SP5


  • Ability to configure the pre-installed SMTP mail server (Google, Office 365, Exchange)
  • Updated invitation letter for user. It contains, among other things, a link to the new 3CX web client, which will be available in V15 SP6.
  • Added the ability to automatically update the 3CX server
  • Added Debian OS security updates
  • Fixed errors in counting available server updates.
  • Notifications about the availability of new firmware for IP phones appear only if there are such phones on your network.
  • Source identification identification bugs fixed and the ability to double check Source ID added
  • PUSH notifications for chat messages in 3CX mobile clients for iOS and Android - the functionality is added from the server side, and the updated clients are already available in the respective app stores
  • The ability to upload backups to Google Drive. You can also restore the system from Google Drive and download the backup file from Google Drive.
  • The report User Activity has been added, which shows the most loaded periods during a day, month or year.
  • Support for the latest firmware versions of Yealink phones
  • Getting the number of extensions connected to the system through the API.
  • New feature for hotels - Wake Up. For this, a special IVR service is implemented in the system.
  • Fixed fax service. For some SIP trunks, the fax server sent a 400 Bad Request response due to the T38FAXMAXIFP media attribute in the SDP T38 header.


Beta testing


First get a free test key . After that, download and run the beta version of 3CX for the selected platform:

  • 3CX for Windows
  • 3CX for Debian is installed with this guide . In the /etc/apt/sources.list.d/3cxpbx.list path, specify downloads.3cx.com/downloads/3cxpbxbeta / . After this, run apt-get update and apt-get install 3cxpbx . The whole installation script looks like this: 

wget -O- http://downloads.3cx.com/downloads/3cxpbx/public.key | apt-key add - echo "deb http://downloads.3cx.com/downloads/3cxpbxbeta/ /" | tee /etc/apt/sources.list.d/3cxpbx.list apt-get update apt-get install 3cxpbx

Do not install this version in a working environment. We do not guarantee that it can be updated before the final release. Portability of backups is also not guaranteed. We are waiting for your feedback in the comments to this article or on the technical support forum .


New 3CX Annual Subscription Licenses


Our product is constantly evolving, and our relationships with customers and partners are also developing. Since February 2017, new prices for 3CX licenses have entered into force, reflecting exchange rate fluctuations. In addition, a new type of license has appeared - an annual license by subscription. An annual subscription license is offered for all editions and 3CX containers. The license is active for 12 months and is subject to renewal at the price of the original purchase. Prices start from :

  • 149 Euro - Standard Edition
  • 189 Euro - Professional Edition
  • 298 Euro - Enterprise Edition

You can change the perpetual license for annual and vice versa, however the exact price of the change will be offered to you individually in the 3CX Sales Department.

At the moment, all users of expired 3CX licenses (i.e., licenses with an expired subscription to updates) can replace them for free with an annual 3CX V15 license of the same edition and capacity!

The changes also affected the licensing of the WebMeeting videoconferencing system. A separate license for WebMeeting is no longer available. Now it is included in the single license (annual and unlimited) on the PBX as follows:

  • Standard Edition - 10 simultaneous video conference participants
  • Edition Professional - 50 simultaneous video conference participants
  • Enterprise Edition - 100 simultaneous video conferencing participants

All questions related to new 3CX licenses can be checked with your partner or in the 3CX Sales Department.
