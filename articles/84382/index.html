Apache subversion

The project of the popular version control system Subversion is transferred to Apache Incubator and will subsequently become a top-level project of the Apache Software Foundation.

Press release dated November 4, 2009.

The real address of the project: subversion.apache.org . Some materials are still at the old address ( subversion.tigris.org ), but will gradually be transferred.

The current stable version 1.6.9 (1.6.7 and 1.6.8 are not publicly released) is released as Apache Subversion. It fixed a number of bugs, added support for Berkeley DB 4.8, working copy format of development version 1.7, improved svn + ssh: // operation under Windows.

You can get binaries here: http://subversion.apache.org/packages.html
I prefer to work with version control systems in the console (except for viewing diffs), so in Windows I use the SlikSVN build - only console utilities, there is a x64 version.
