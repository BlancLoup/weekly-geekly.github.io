Thunderbird 2.0 released

One of the most popular OperSource email clients, Mozilla Thunderbird was updated to version 2.0 at the end of July. Several "tasty" things have been added. In particular, anti-encryption has been improved, work with extensions and themes has been formalized, and the functionality of a pop-up notification of new letters has been expanded.


The official announcement is located at this address . Some things were already present in version 1.6 (with which I was remarkably updated), some are completely new. I'll tell you in order that pleased me personally:

  1. Keep up to date with the latest posts . A pretty useful feature (for me) of Thunderbird, shows in the pop-up window some information about new messages. In the previous version, it simply showed their number, in version 2.0 it was possible to display the sender, the subject and part of the message itself in this notification.
  2. Protection from fraudulent emails (phishing) . Very pleased (with natural anti-spam and just filters) warning about possible phishing messages. A couple of times, this warning saved us from following false links.
  3. Add-ons Manager for Extensions and Topics . In the previous version, in order to establish a new extension or theme, it was necessary to make a number of "squats" (download the file separately, and then only find the option in the mail client, in which you need to point to this local file). Now all the work with extensions and themes has been moved to a separate module (like in Firefox), which made the work very easy.


Well, familiar to Mozilla is the automatic update to the latest version (I hope it will function the same as in Firefox, because the previous version of Thunderbird did not want to upgrade to 2.0).
