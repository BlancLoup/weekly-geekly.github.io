New product released - SQL Management Studio for Oracle!

EMS Database Management Solutions is pleased to announce the release of a new product. SQL Management Studio for Oracle is a complete solution for administering and developing databases. With a range of components that are designed to solve critical database management tasks, SQL Studio provides you with indispensable tools for administering databases and managing their objects, migrating, comparing and retrieving databases, as well as importing, exporting and comparing data. SQL Studio combines all of these tools into a single, powerful and convenient working environment that can work around the clock. Whether you are an application or database developer, administrator, or analyst, SQL Studio for Oracle provides you with all the tools you need to make your work more productive than ever!

How does EMS SQL Management Studio for Oracle differ from a set of separate SQL tools?
  1. New EMS SQL Management Studio provides the user with a number of very useful additional features:
    • Easy start of utilities and services
    • Store all templates in one repository
    • A single list of data sources for all services and utilities
    • Creating complex tasks and their automated execution
    • Ability to run external applications in tasks
    • Storing logs of all tasks in a single database
    • Create custom notifications; sending them in various ways
    • The ability to set a single visual style for all applications
    • Quick access to technical support
  2. Unlike packages that were simply a set of utilities sold by the kit, EMS SQL Management Studio is a full-fledged and independent product with a single installation distribution, a single interface for registering all its components and a single system for updating the functional modules included in the studio.
  3. Finally, the cost of ownership of SQL Studio is significantly less than the cost of owning a set of products. The cost of SQL Studio already includes one year of subscription to Software Maintenance, which allows the owner of SQL Studio to receive any updates for free throughout the year. Subsequent renewal of the subscription will cost only 20 - 30% of the cost of SQL Studio per year, while the owner of the set must separately purchase each update of the product included in it for 50% of its retail price.


The main advantages of SQL Studio:
  • Quick launch of tasks gives you the opportunity to focus only on the result.
  • A single repository for storing templates makes it much easier to work with utilities.
  • Customizable tasks save significant time
  • Scheduler allows you to automate daily tasks
  • Various notifications provide an opportunity to control the process of performing tasks.
  • All logs are stored in one place, which makes it easy to follow the execution of operations
  • The ability to set a single visual style for all applications


You can download EMS SQL Management Studio for Oracle at: http://www.sqlmanager.net/products/studio/oracle/download
The evaluation version is fully functional, and you can freely test it for 30 days.

We hope you enjoy working with our products!
