Simple search extensions for Firefox

Faced with the need to search on the page several words at once. I had a list of 500 Wikipedia articles from which I had to choose certain words containing at the beginning. You can have it on a forum or blog page where you look for nicknames of several people at once in order to quickly jump to their comments. Or the main page Habra, where you are looking for specific tags. In general, the need for this function is small, but for those who need it - its presence in the browser can make life easier and save time.

With the search for several words at the same time the SearchWP extension does a good job. It does not work with the Firefox search panel (Cntrl + F), but with the panel for search engines (search plug-ins).



Enter words there, click with the mouse somewhere outside the panel (Enter does not click - it will start the search in the network as usual) - the words are framed, become buttons. To search for a specific word - click on it. After installing the add-on, a backlight button will also appear to the right of the search field, activating which we highlight the searched words in the text.



Expansion settings look extremely simple. It is possible to select the minimum number of characters for the word and the ability to collapse the list of words in the drop-down menu. At the same time mugs will appear in the search field, by clicking on which we will call this list.



What is missing:
  • I would like more colors to highlight (there are 5 of them, after the fifth word they are repeated)
  • I would like to be able to simultaneously search for independent words and phrases
  • I would like to be able to go in order for all the selected words at once, and not for one of them.


When using this add-on, you had to constantly use the drop-down list, since there were a lot of words, and constantly having a search field on the split-screen (even though the address field is next to it) is not very convenient. In this case, and in many others, the Searchbar Autosizer extension will help you . It allows you to customize the search field behavior.

You can set up add-ons either using the setup wizard (on the first screen, choose what you need, and then you can set the required field size in various situations) or using the advanced settings screen, where you can immediately tick all the boxes and set all dimensions in numerical form . The last menu is shown below. Separate the items separately is not necessary, everything is clear and well illustrated in the settings window itself.



And finally, the simplest addon Context Search , which adds a list of search engines to the context menu. This allows you to not copy the search word in the search field. Simply select it and select what to search in the context menu.
