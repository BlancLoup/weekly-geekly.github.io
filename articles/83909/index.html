Add your own search engines

What I like about Google Chrome is the ability to work very conveniently with different search engines: I entered a keyword, a space and typed a query. However, this function is used by 100%, not all.

Today I want to tell you how to add your "search engines" (this, in fact, can be not only search engines) and thereby increase the performance of the Internet.



Where is this all set up? See the picture:



1) Open the Chrome settings dialog, tab "Basic". Click the "Manage" button.
2) In the dialogue "Search engines" click "Add ...".
3) In the dialog that appears, fill in the parameters of the new search engine.

Options:

Name: The prompt that will be displayed in the address bar. Chrome adds the ‚ÄúSearch‚Äù prefix to it, so for beauty I bow the name so that it answers the question ‚ÄúSearch for what ?‚Äù Or ‚ÄúSearch where ?‚Äù.

Keyword: a set of characters that will be the search prefix, i.e. which you have to type in the line and then press the spacebar to activate this search engine. It should be mostly short and / or well-remembered combination of letters, abbreviation.

URL: The search engine address. In fact, it can be any address, i.e. its template, in which you can substitute some variable part (indicated by the parameter %s in the URL).
