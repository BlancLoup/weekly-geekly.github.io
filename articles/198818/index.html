Windows Azure Backup cloud backup service is available for commercial use.



The Windows Azure cloud backup service has gone out of the preview stage and is available for public use.

Windows Azure Backup is a cloud-based solution for Windows Server that allows you to back up and restore files and folders to the cloud, providing built-in protection against data loss. The service offers IT administrators and developers the ability to back up and protect critical data and an easy way to recover it from anywhere and without the cost of hardware.

Windows Azure Backup contains the following key features:

Simplified configuration and management . Windows Azure Backup service integrates with the familiar Windows Server Backup utility in Windows Server and with Data Protection Manager in System Center and Windows Server Essentials in order to offer a single backup path to disk or cloud.

Incremental block level backups . The Windows Azure Backup Agent performs incremental backups based on changes in the file and changes at the block level and copies only those blocks that have been changed, which reduces traffic and the need for storage. Different versions of backups efficiently use storage space, storing only changes between blocks.

Data compression, encryption and throttling . The Windows Azure Backup Agent provides data encryption and compression for transfer to the Windows Azure storage. As a result, only encrypted backup data can be stored in the Windows Azure cloud. The encryption key is not available to the Windows Azure Backup service in the cloud and as a result, the data cannot be decrypted and not decrypted in the cloud service. Service users can also configure the function and parameters of throttling and indicate to the service how it should dispose of the network bandwidth during backup or restore.

The integrity of the data is verified in the cloud . In addition to secure backups, backup data is automatically checked for integrity. As a result, any data corruption that might have occurred during data transmission over the network is automatically detected and corrected.

Configuring policies to save data in the cloud . The Windows Azure Backup service allows you to use policies to configure a period of time after which backups should be deleted. It helps to comply with corporate standards and save money on backup storage.

To familiarize yourself with the work of the Windows Azure Backup backup service, you can read the following articles:

  • Manual: Restoring Files and Folders with Windows Azure Backup Agent
