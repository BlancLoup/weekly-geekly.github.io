JavaScript and CSS Compilation

In this post, I decided to share with the habrasoobshchestvom its selection of reference books and training resources for web development, mainly in javascript. It will be useful to those who have recently begun to engage in frontend-development and want to have on hand a selection of convenient reference materials for work.


Learning from scratch


  • htmlacademy - learning the basics of html and css
  • webref - a good explanation of css properties
  • sorax - css basics tutorial series
  • w3schools is a web-based learning resource.
  • tutorialrepublic - similarly
  • tutorialspoint - similar

Need something to remember or find in the documentation


  • css-tricks - examples of using css properties
  • bootstrap - cheat sheet
  • jquery - cheat sheet
  • angular - crib
  • devdocs - convenient documentation search for web developer
  • overapi - a handy list of functions of different api
  • MDN - an extensive resource for javascript development

Need to find a library or plugin


  • bower (rating) - a list of popular frontend libraries
  • awesome-javascript - a useful javascript collection on github
  • awesome-nodejs - the same for nodejs
