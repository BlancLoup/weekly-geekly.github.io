PHP Digest number 117 - the latest news, materials and tools (September 10 - 24, 2017)



Fresh selection with links to news and materials. In the release: PHP 7.2.0 RC 2, about the future of HHVM, suggestions from PHP Internals, compilation of PHP chats, video from conferences and meetings, and much more.
Enjoy reading!



News and Releases


  • On the future of HHVM - Not so long ago, many projects have refused to support HHVM. Now the HVM team has announced that in the long run it does not plan to strive for full support for PHP 7. Instead, the guys from Facebook will focus on Hack. Nevertheless, in the near future we plan to fix compatibility issues with popular tools like Composer and PHPUnit.
  • PHP 7.2.0 RC 2 - The second release candidate is scheduled . The next release is expected on September 28th. You can read about branch changes here and here . Test with a prepared Docker image .
  • Sylius v1.0.0 - Major release of the popular e-commerce platform based on symfony.


Php internals


  • [RFC] RFC Workflow & Voting - It is proposed to regulate the RFC process, in particular, when voting, raise the threshold for acceptance of changes to 2/3. Criteria are also indicated for those who can vote.
  • [RFC] Friendship Class - A second attempt to implement the concept of friendly classes. A friendly class has access to the private and protected fields of the class in which it is declared friendly.
  • [RFC] Fiber - An interesting addition to PHP generators to simplify asynchronous code.
  • Pre-draft PipeOp v2 - Internals discusses a draft proposal for a pipe operator. The original proposal was criticized because of the use of the $$ placeholder and now a simpler version has been proposed: 

     $x = "hello" |> 'strtoupper' |> function($x) { return $x . " world"; }; // $x === "HELLO world"
