Cocosino

Cocosino is an IDE written in JavaScript and designed to facilitate the development of games on Cocos2d-x and Cocos2d-html5 and porting them to iOS, HTML5, Windows, Mac OS and Linux.
Cocosino developers promise to provide users with detailed documentation, examples and training materials, debugger code, and most importantly the ability to develop games for iOS, HTML5, Windows, Mac OS and Linux, and then publish them for smartphones, tablets, web and desktop computers with one click.


Download and evaluate the alpha version of IDE here .
At the moment, this project is raising money on Kickstarter .
For the collected $ 25,000, they promise to release a stable version.
By collecting $ 35,000, they will expand the IDE features by:
  • cocos2d-x and cocos2d-HTML5 project templates
  • embedded lightweight web browser for HTML5-cocos2d
  • source code navigator, autocompletions and tooltips using Esprima
  • skins for desktop and web emulators
  • textbooks and examples

For $ 45,000 a JavaScript debugger will be integrated.

Sponsors can get everything from a license for the full version for $ 35, which will then cost all 80, and ending with its own SDK module (for example, iCloud, GameCenter, InMobi, inneractive, etc.).
