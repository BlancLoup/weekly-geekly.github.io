Veeam Backup Cloud Edition has been released with support for Azure, Amazon, Google, HP Cloud and several other cloud platforms.

Veeam Backup Cloud Edition is a backup product for virtual environments VMware vSphere and Microsoft Hyper-V , which allows you to back up to any of the currently leading cloud platforms that support OpenStack , such as Windows Azure, Amazon S3, Amazon Glacier, Google , Rackspace, HP Cloud and several others (15 platforms in total). In such a scenario, the cloud can act as an alternative to tape and branchless backup storage, since the average annual resiliency of a cloud storage, such as Amazon Glacier, is 99.999999999%.

The architecture of the backup process in the cloud includes two links (see figure). First, a regular backup to a locally located repository is performed, and then a backup repository is partially or completely copied automatically and on schedule to the cloud. When transferring and storing, backup copies are archived, deduplicated and encrypted using an administrator-defined password (various algorithms up to AES256 are supported), thus avoiding the information security risks associated with trusting the information security of the cloud provider.

The administrator can set up email alerts about the status of backup copy jobs to the cloud. It is possible to set a limit on the consumption of cloud resources by a product, expressed in gigabytes or in dollars. You can configure the level of consumption of Internet channel resources - both at the current time, and according to the schedule. This allows, for example, to adjust consumption at 10% in company working hours and at 100% at other times. The product is easy to configure in terms of working with the selected cloud platform and does not require any specific technical knowledge from the user - thus. the user can choose a cloud platform based on their economic interests, without going into the technical features of integration with it.

Veeam Backup Cloud Edition comes in two versions (Stadard and Enterprise) and is sold by annual subscription, which allows customers to reduce the cost of capital investments, and flexibly manage their operating costs by paying exactly for the amount of data stored in the cloud that they need. moment, having the ability to increase or decrease transaction costs as needed.
