Tiny Thief - a cool mobile game made using Flash + Autodesk Scaleform Mobile SDK



In the beginning I wanted to write a translation of a review article from Autodesk about the TIny Thief game , but in the process of translation I understood that the article was a little more than completely aimed at advertising the Autodesk Scaleform Mobile SDK , and not at disclosing useful information on game development.

But this does not change the essence. Gentlemen, I am very pleased that a tool has appeared which, apparently, allows porting Flash to mobile platforms in a convenient way and gives good performance. I, as a Flash developer, are very, very happy (since, unfortunately, Adobe, in my opinion, did not manage to create a good tool for porting Flash to iOS and Android).

Brief specifications:
1) In the list of stated platforms for porting: iOS, Android, Win Phone 8 / RT (although judging by the comments on the forums, Win-support has not yet been implemented)
2) Allows you to use vector graphics created in Flash, without additional transfer to a raster.
3) Apparently, it keeps good performance.
4) Allows you to load / unload .swf files ‚Äúon the fly‚Äù, which makes it possible to control the load on the RAM.
