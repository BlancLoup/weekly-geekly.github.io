Release of the 11th version of Asterisk



The final day of the 2012 Astricon conference, October 25, announced the release of the 11th version of Asterisk, which has the status of LTS, support is promised until October 2017. The version is available for download from www.asterisk.org/downloads , so far only in the source code.

Of the most notable innovations:
  • WebRTC support (calls from the browser, natively)
  • logging the call ID in each row of all log files
  • new jingle / google talk driver (chan_motif)
  • support custom codecs, generating / parsing SDP parameters

The complete list is wiki.asterisk.org/wiki/display/AST/New+in+11


Video with a demonstration of a call to Asterisk IVR from a browser, watch from 4:23 :



As for codecs, I mean support for such constructions in SDP: 

a=rtpmap:108 AMR/8000 a=fmtp:108 mode-set=0,2,5,7;mode-change-neighbor=1;mode-change-period=2 a=rtpmap:102 AMR/8000 a=fmtp:102 mode-set=0,2,5,7

By the way, AMR NB codec is not included in Asterisk, but there are patches that were adapted to the 11th version using the new features for SDP parsing.
