Windows Mobile Marketplace available for WM 6 and 6.1


Launched with the new version of Windows Mobile 6.5, the online store of the Windows Mobile Marketplace software was initially aimed at supporting the WM 6.5 platform. About the support of early versions of WM 6 and 6.1 appeared and now finally in the Marketplace there was an opportunity to work with them.

If you have a Windows Mobile 6 or higher smartphone, then from it you can go to mp.windowsphone.com and you can install a small client that allows you to navigate the store. At the time of launch of the store, about 250 applications were registered in it, today, a month later, there are already more than 800 applications (for different countries the number varies).

It is a pity, but so far only free applications are available for Russian accounts.
