Install and configure 3CX Webmeeting Server

Hello!

In this article we want to talk about installing and configuring the server version of 3CX Webmeeting - a convenient solution for video conferencing based on WebRTC technology. WebMeeting does not require access to the Internet and can be used solely as an internal corporate service with simultaneous connection to the conference from 2 to 250 participants.

Installation


The installation itself does not require much effort. The main thing is to have an external domain name, well, or until a name that is resolved inside the local network, provided that the conference is only internal.


Conference Setup



To schedule or create a conference, you only need to set its name and select the client type - WebRTC (Chrome browser) or your own stand-alone client from 3CX.

Advanced mode allows you to customize the video format and bandwidth
  • From 128kb / s to 2 mb / s
  • From QCIF to 1080p
  • From 5 to 30 frames per second


Participants feature settings


  • Sharing from 128kb / s to 2 mb / s

You can restrict users via communication channels in advance, or you can do this during the conference.



Information on resource loading is available in the Monitor section for the administrator.



Download address book for invitations to the conference from a text file.

Preloading documents for general viewing.
Applications and utilities for users.

To create a conference


Set a topic and click the " Create a conference" button.



You can add participants or send, then the link will be sent by email or send an invitation in any convenient way.



If the client is already installed, click " Is the conference client already installed?" and the application will start automatically.
