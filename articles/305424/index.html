Techno Breakfast "Effective work with Percona Server for MySQL on high-load web cluster projects"



1. We at Bitrix love to share knowledge.
2. We love using Percona Server instead of standard MySQL.
3. Sometimes we have business breakfasts.

Combining all these facts together, we decided to hold a techno breakfast . :) Special guest - Peter Zaitsev, director of Percona.

Anyone can participate absolutely free of charge online - watch, ask questions. Register and mark in your calendar on July 13th. :)

Details - under the cut.

The key theme of the event is the nuances of using MySQL / Percona in high-load projects.

What are we going to talk about:

- Maximum performance and fault tolerance of the database.
- Features of Percona solutions to optimize the performance of your database.
- Using master-master replication to build a geo-distributed fault-tolerant web cluster.
- Features of the Percona Server in the cloud infrastructure.
- How to work with thousands of databases on one server.
- Advanced monitoring of typical and unusual characteristics of the database.
- Practical application of tools Percona Toolkit.

Separately, we will talk about the 1C-Bitrix solutions migration program from MSSQL and ORACLE Database to MySQL.

If you already have questions and topics that you would like to discuss, write in the comments.

July 13. 10:00 Sign up and participate !
