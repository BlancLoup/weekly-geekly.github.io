Backup hosting - new service from Infobox

The hosting company Infobox launches the new Backup it service - to create and store backup copies of various data online. The service is intended both for home use ( Home line of plans ), and for corporate clients ( Pro line ).

The solution offered by Infobox is focused mainly on the needs of business customers, for which the priorities are high reliability and uninterrupted service, greater security, greater flexibility and ease of implementation. The new product meets all these characteristics.

Technically, the Backup it solution meets all modern requirements for information systems of high availability and reliability and is built on a cluster running Windows Server 2008 R2.

Backup it is easy to use. Backup it is installed on the computer once, after which the user creates one or several tasks (copy creation schedules) and an encryption key. Then you can forget about the existence of the program. At the right time, the program will automatically record any changes in the files and send them to the server. After completing each task, the user receives an email notification of the changes made.
Distributions for Windows, Linux and Mac are available to our customers.
In the event of loss or damage to the computer, you can install the Backup it client on a new computer and perform the above actions.
If data has been lost, Backup it can easily recover it. To do this, just select the necessary documents and click on the ‚ÄúRecover Now‚Äù button. The selected files are copied from the server in the data center to your computer.

Also, the saved data is accessible from any computer via a web browser using the online control panel.

Security is the main issue of concern to customers most when choosing a "cloud" services. Everyone is afraid of access to their information at a time when the data is on the servers of the artist. In this regard, the Backup it service is suitable even for the most biased skeptics.
The fact is that all backup copies of data are encrypted using Advanced Encryption Standard (AES) using 128-bit or 256-bit encryption technology. The encryption key used in the program is not reversible, is set by the end user and is known only to him, so no one else can open the data in backup copies on the Infobox servers. When restoring data, backups are transferred back to the user's computer and decrypted on the client side.
All communication between the computer and the server takes place through a secure SSL channel and through a secure SOAP / XML interface. All this ensures reliable transmission and storage of data.

Backup is performed by incremental method. The expediency of this backup method is based on the fact that the bulk of the data is static and does not change. Studies have shown that only about 5% of the data changes daily.
We took this into account, and Backup it, after creating the first full copy of the information, further analyzes and compares the data since the last copy was created and sends only changes to the server. This allows you to reduce the amount of traffic, and most importantly reduce the time to create backups.

In addition, Backup it uses strong data compression technology, which reduces the size of backup copies of data even more. Compression, depending on the type of data, averages 50-60% of the original size. This reduces the size of the required disk space and speeds up the download of backups even with a relatively slow connection to the Internet (for example, ADSL).

For users who are not demanding of security and functionality, there are free and shareware products on the market, such as Amazon S3, Dropbox, Skydrive, SugarSync, etc. They are fairly simple to use, mostly offering initial free plans with limited space, but in fact offer simple file storage on cloud hosting. The advantage of Backup it over similar services is, first of all, security and the ability to conveniently work with business data.

The most popular counterpart is the overseas Amazon S3. Its main advantage is cheapness, although on the other hand the price of Backup it is quite comparable. For example, the cost of a gigabyte at the Home 50 tariff is 7 rubles or 0.23 cents, which, taking into account the unlimited traffic included in the tariff, is not so much different from 15 cents with the free traffic of Amazon S3 solution. The disadvantages of the latter (in terms of business use) include the impossibility of backing up standard things for many organizations, such as databases, mail servers and some other obviously business-oriented applications.

If we talk about Dropbox, then, by and large, the function of Dropbox is to simply synchronize files that are in the same folder with the server, located in the cloud. The main and fundamental difference between Backup It and Dropbox is the greater flexibility of the first. Using Backup It, the user has absolute freedom in what, how and when to backup.
