SAFe Program Consultant (SPC) certification class in Moscow, November 19-22

On November 19-22 , 2014, Ciklum together with Incremental Gains and Scaled Agile Academy invites you to take part, in the first in the CIS, SAFe Program Consultant (SPC) certification class .
The certification course program includes such modules as: Leading SAFe , Implementing SAFe and an exam for obtaining an SPC certificate.
A number of global companies such as ING, TomTom, T-Mobile, Intel, BMC Software, Nokia Corporation, John Deere, Tripwire, Tendril Inc., Mitchell International, SEI, Nordstrom, Nokia Siemens Networks, F-Secure Corporation and have long been interested. and successfully implement SAFe.


The two-day certification course will be interesting and useful for top managers, project and program managers, project management consultants and corporate Agile trainers.
The course is conducted by experienced trainers Michael Stump, SAFe Program and Portfolio Consultant and Timofey Evgrashin - SAFe Program Consultant, Agile Trainer, Approved ICAgile Trainer.
