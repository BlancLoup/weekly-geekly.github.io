The current version of the program is marked as EA15.1. External Alpha.
What do we already know how?
  • Synchronize directories on p2p directly. It was possible to achieve a speed of 40 mbps over wi-fi.
  • Synchronize directories exclusively with your typewriters. No one steals our files.
  • Synchronize directories on the server developers. The limit is 1 GB per directory (as aerofs, they are called library)
  • Select libraries to synchronize for each individual machine. Control only locally (that is, you cannot tell the second PC to stop synchronizing files through the aerofs client from the first PC)
  • Able to draw a beautiful plate with a list of files that are now swinging / distributed. Also there is information from where and where the file is being sent and the speed is indicated.
  • Each client in your account can have a custom User name and Computer Name.
  • Send invites to mail


Yes, the functionality is absolutely scanty. But the most important and necessary f-tion is. Files really sync.   • Simlinks do not sync
  • Integration with FM. Well, more precisely from the program's menu, you can open the library in FM ... But no icons on folders and other goodies DropBox
  • The program icon in the tray does not change its appearance. We don‚Äôt know that files are not synchronized.
