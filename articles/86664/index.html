Opera 10.50 (under Windows) and how to make friends with it with Silverlight

Recently, after updating the Opera browser, my sister asked me to figure out why a strange picture appeared instead of the usual player.


I went to the installation page ... but I saw the inscription Microsoft Silverlight not supported by this browser , and the table of supported browsers (see below), among which Opera was not. But the output was still. Instructions under habrakat.



As you can see, Opera is not listed. But it was not there, because Browser it is unofficially supported !!!

First, install the plugin itself: Silverlight 3 .

After installation, launch the browser and in the address bar write opera: config # UserPrefs | PluginPath . We fall into the settings editor. In the Plugin Path we add the following address (the addresses are separated by the symbol ';'):

For installed Silverlight 3
  • Windows x86: C: \ Program Files \ Microsoft Silverlight \ 3.0.50106.0 \
  • Windows x64: C: \ Program Files (x86) \ Microsoft Silverlight \ 3.0.50106.0 \

* For installed Silverlight 4
  • Windows x86: C: \ Program Files \ Microsoft Silverlight \ 4.0.41108.0 \
  • Windows x64: C: \ Program Files (x86) \ Microsoft Silverlight \ 4.0.41108.0 \

Then click save, restart the browser. All is ready!!!

UPD : according to remarks @gefix - added to the list of addresses for the installed Silverlight 4
