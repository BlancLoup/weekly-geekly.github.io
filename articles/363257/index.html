Android 5.0: Xperia AOSP source codes and executables available

Software developers and enthusiasts will be pleased to know that we have published source codes, preliminary versions of executable files, and firmware build instructions for a number of Z series devices as part of the Android Open Source Project. Thanks to this, you can already start working on your own firmware based on Android 5.0 Lollipop.





You can get acquainted with the necessary materials in the corresponding section Developer World . You will also need to download executable files, executable files and instructions . You can install the firmware on smartphones Xperia Z1, Z1 Compact, Z2, Z3 and Z3 Compact with an unlocked bootloader (we have already written about the easiest way to unlock). You can upload the results of your work to our repository on GitHub.

It should be noted that the current firmware version is not intended for everyday use and has some flaws, for example, the camera and the radio module do not work.
