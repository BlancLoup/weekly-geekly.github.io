Wi-Fi microSD Adapter: add a Wi-Fi module to the camera



Well, or any other device, where there is a slot for SD memory cards. The fact is that the other day Monoprice announced a Wi-Fi microSD Adapter. This is an adapter for microSD with Wi-Fi module.

Now there are already cameras with a Wi-Fi module, but most models are still not equipped with such a module. There is another option such as Eyefi, but the price of such a solution can be quite high. A device from Monoprice costs less than $ 35.

The adapter supports work with microSD cards with a capacity of up to 32 GB. If not enough, you can simply replace the card. And in the case of the same Eyefi, the user is limited by the size of such a card, with no options.

In order to work with a Wi-Fi microSD Adapter , you need to insert a microSD card into the adapter, and insert the adapter into the camera slot (or other device). The adapter starts working as an access point. Then on the laptop / tablet we find the desired Wi-Fi network with the appropriate name, and join by entering the password from the manual to the adapter.



Access to files on the map is carried out through a web interface that allows you to view photos, download images one by one, or a whole folder at once in the archive. The range of the access point is about 15 meters. Data transfer is not the fastest, only 1 MB (as far as you can understand, just a megabyte) per second with a file size of 130 megabytes.



The web interface also allows you to view images in JPEG format, video files and images of other formats can not be viewed in the browser, but you can download them on a PC / tablet. In general, a fairly convenient gadget, which may well be useful to any camera owner and / or user of a different type of device.
