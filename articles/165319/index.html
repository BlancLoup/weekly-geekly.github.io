The first certified Android prefix for downloading torrents



BBK and other companies have been making TV extensions for a long time, through which you can transfer files from computer to TV, receive IPTV and download torrents. The latter feature is especially helpful. The prefix remains on for the night and downloads the latest series of films, according to a given list. After downloading, all these films are immediately available for viewing on TV.

The download function for torrents was usually positioned as a secondary, additional, but today BBK launched the BitTorrent Box for Android, in which this function is the main one. This is the world's first certified TV set-top box with the word BitTorrent in the title. Let's see how the right holders will react to such ‚Äúarrogance‚Äù.

The 4 GB built-in flash memory is not enough for a long time, the IP1501 model goes without HDD, but it supports the connection of an internal or external disk. In this case, the prefix can work as a video recorder, recording TV shows on a schedule, pausing them and rewinding commercials on TV.

Alternatively, movies can be saved to a flash card or USB flash drive, which are now available in sizes up to 1 terabyte .

The device connects to the router via Ethernet, and after downloading distributes content to all surrounding devices via WiFi 802.11 b / g / n.
