New NemoMobile image with UI MeeGo Harmattan

On April 26, a new version of the NemoMobile software platform became available. This project, along with SailfishOS, is being developed by Jolla with the participation of the Developer Community. Like SailfishOS, the first device on which Jolla promised to present on May 20, NemoMobile is based on the source code of the Mer project , which in turn is an intermediate branch between MeeGo Harmattan (Maemo6) and Tizen.


Unlike Maemo, NemoMobile can be installed on devices with different architecture. For example, recently NemoMobile has been ported to Pandaboard . Another example: Nemo on CuBox . So, it turns out a software platform for a wide range of devices. As for the latest firmware version, its demonstration on the Nokia N900 device can be viewed in this review .

It is noteworthy that the UI NemoMobile is almost completely consistent with the UI MeeGo Harmattan. So it is possible that Nemo will at some point become the final global update for the N9, an update that will continue to develop as a free mobile platform.
