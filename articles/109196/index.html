Released Google Reader app for Android

Finally, the long-awaited official Google Reader app for Android has been released.

It supports all the basic functions - the number of unread entries, friends, distribution, likes.



Also from the interesting:
  • support for multiple accounts;
  • synchronization settings;
  • full support for subscription features;
  • Search.
More obscure features:
  • navigation with the volume change key: if you turn it on in the settings, you can navigate through the recordings using the volume change keys;
  • a long press on a folder or subscription brings up a context menu that allows you to rename, unsubscribe and change folders;
  • If you press the menu key on a separate entry, you can use the send function that integrates with other phone applications.

PS Declared support for all androids, starting with Donat, but for some reason the application is not on my Waldfaer on Ekler. Write, please, in the comments, who managed to install.
