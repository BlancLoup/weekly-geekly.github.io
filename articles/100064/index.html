Youtube offers new embedded HTML5 friendly codes.

A blog about a new way of embedding videos on external sites, based on iframe, appeared on the Youtube blog.
The main advantage of this method is the choice between Flash and HTML5. The choice depends on the settings of a particular user on the site Youtube . New embedded codes are still being finalized, but the developers hope that sites will start to implement them now so that the improvement process will be faster.
Up to this point, a video from Youtube could be embedded on its pages only as a Flash player. As you know, not all mobile devices support Flash. With the introduction of a new method of embedding, video from Youtube will be even more accessible.

New codes look like this:

< iframe class ="youtube-player" type ="text/html" width ="640" height ="385" src ="http://www.youtube.com/embed/video_id" frameborder ="0" >
