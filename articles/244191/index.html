Adobe fixed critical flash player vulnerability

Adobe has released an unscheduled update APSB14-26 for its Flash Player, which complements it with special protection mechanisms against exploits of the CVE-2014-8439 vulnerability. This vulnerability can be used by attackers to remotely execute code on a vulnerable system. The Flash Player update is available for users of Windows, Linux and OS X. According to the well-known French security manager kafeine , CVE-2014-8439 is already at the stage of active operation.



Below is a message to install an updated version of Internet Explorer on Windows 8.1.



We recommend updating your Flash Player as soon as possible. Browsers such as Internet Explorer 10 & 11 on Windows 8 / 8.1 and Google Chrome update their Flash Player versions automatically.
