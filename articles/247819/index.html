New Release Candidate for 3CX Phone System 12.5 (+ demo key!)

The release of the new version of 3CX Phone System 12.5 is almost ready! Introducing the second version of the release candidate. Support for Fanvil X3 IP phones has been added to RC2, templates for Yealink devices have been updated, and some changes have been made to the IP address configuration process.

Added support for Fanvil X3



Added Plug and Play setting for new budget IP phone Favil X3. With the help of auto-tuning, you can connect the device to the 3CX Phone System in a few minutes. There is full support for CTI and remote configuration.

New in v12.5 RC2:


  • When installing 3CX Phone System, you can now specify the internal and external IP address in case of no DNS.
  • The Yealink SIP-T41 / T42 and SIP-T46 / T48 templates are now merged into one file.
