Wordpress 4.0 Review "Benny"

A release of Wordpress 4.0 , called ‚ÄúBenny‚Äù in honor of jazz clarinetist and conductor, ‚ÄúKing of Swing‚Äù Benny Goodman, became available yesterday. In the first few hours after the release, Wordpress 4 was downloaded more than 200 thousand times. The third version of Wordpress was downloaded over 300 million times and successfully competes with Joomla for the title of the most popular CMS in the world.



Despite the fact that 4.0 is just a number between 3.9 and 4.1, the developers have invested a bit more in this release. The new version makes the process of writing posts and managing CMS even more convenient thanks to more than 100 new features and improvements in working with media files and the editor.

You can download the new version from the official website, update Wordpress from the control panel or install it in 1 click on the cloud hosting Infobox Jelastic (for this we packed Wordpress 4 into Jelastic according to the JPS standard immediately after the release of this CMS).

Under the review of the main changes in the new version of Wordpress 4.0.

New view of the media grid




The media library is now displayed as a grid of images by default. When you click on an element, you will see a preview of the image and additional information, which you can simply edit and quickly move on to the next image.



Using multiple selection, you can select multiple images at once and delete if necessary.



Improved editor




Writing text in Wordpress editor has become even more convenient. The visual editor is now expanding to fit all content, instead of using a scrollbar. When editing, now displayed fixed toolbars when scrolling. Thus, the necessary tools are always at hand.



oEmbed preview


Wordpress 4.0 added support for oEmbed previews for TED talks, Mixcloud, CollegeHumor, Issuu, Polldaddy, as well as support for YouTube playlists. The visual editor now displays previews for media files added via oEmbed links.

It looks like copying the link from the browser line of the open video on Youtube and pasting the link directly into the Wordpress editor. A video will be inserted that will be played in the post and directly in the editor.



Updated process for finding and installing plugins


Wordpress 4.0 makes it easier to find a plugin from the admin panel. The plugin installer now displays them as cards in the grid. Extension authors can now add their own icons that will be displayed in the installer.



Clicking on the "details" link you can see the description, ratings, reviews and information about the compatibility of the extension.



Improved language support


If you install Wordpress 4 yourself, you can select a language during the installation.



You can also choose the installation language directly from the main section of the Wordpress 4 control panel settings.
