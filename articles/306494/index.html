Community of experts, collaborating on projects and other updates of the FlyElephant platform



The FlyElephant team is pleased to announce the release of the FlyElephant 2.0 platform, which includes the following updates: internal community of experts, joint work on projects, public tasks, Docker and Jupyter support, new data storage and work with HPC clusters.

FlyElephant is a platform for data researchers, engineers, and scientists that provides off-the-shelf computing infrastructure for high-performance computing and rendering, helps find partners and work together on projects, and manage all resources from one place. The platform consists of 3 main components:
  • Compute. Quick access to the computing cluster in the cloud with the necessary software or HPC cluster, as well as automation of the calculations.
  • Collaborate. Teamwork on projects and a community of experts where you can find partners to solve a complex task or get expert advice.
  • Manage. Manage licenses, software, computing resources, patterns, algorithms, data and results in one place.

Among the innovations we note the following:
  • Internal Expert Community
    We are starting the development of an internal community of experts. Now you can publish your profile in the catalog, add colleagues to friends, share tasks and results, and also receive bonus computing hours for activity within the community.


  • Project Collaboration
    You can create public and private projects, invite colleagues to them and share the results of calculations within projects. Anyone can join the public project and contribute to his decision.


  • Public tasks
    Users can publish and share their tasks and results. All public user tasks are available on his profile page.


  • Docker support
    We implemented support for Docker containers. Now users can run their calculations even faster using programs from the catalog, and in the future it will be possible to launch their own Docker container inside the platform.


  • Jupyter support
    We implemented Jupyter support for interactive work with code.


  • VNC support
    For remote access to applications that require a desktop, we have implemented VNC support.


  • New data warehouse
    We have implemented a new storage based on our ownCloud. Now users can easily synchronize data between all their workstations and repositories.


  • Work with HPC clusters
    We implemented Slurm support, which allows you to directly send tasks from the FlyElephant to the HPC cluster. The first HPC cluster has already been added to our directory.


  • Registration through social networks
    Registration has become even easier and faster.


  • Slack community
    Join the FlyElephant user community: slack.flyelephant.net and our HPC community: hpc.community
