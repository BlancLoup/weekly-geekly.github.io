3CX Phone System V15 SP3 released and integrated with Insightly CRM

3CX Phone System V15 SP3


As promised, shortly after the release of 3CX SP2 and the ISO image of 3CX Phone System for Debian Linux, we are releasing V15 SP3. The main change is the opening of the G729 codec both on the server and on 3CX clients. From now on, G729 will be available on the server for all simultaneous calls. And soon we will release an update for 3CX clients, adding the G729 codec to them!


Note: If you go to the Licenses section of the 3CX interface, you will still see half the number of G729 channels. Do not pay attention, because the actual number of channels of the G729 is equal to the number of simultaneous calls .


Adding extended support for the G729 codec solves two important tasks:


  • Allows you to more efficiently host 3CX on cloud hosting. G729 has a bandwidth of 8 kbps, which is about 8 times less than the standard G711 codec. Thus, if the traffic on the cloud server is paid, you will get a serious savings. Also, the codec has good resistance to packet latency, which is very important for remote cloud hosting.

  • Saves traffic and improves voice quality on mobile devices. This is especially true when making calls over 3G networks.
