Kinect for Windows SDK beta


Microsoft has long promised to give everyone the official opportunity to create applications under Kinect. I must say that thanks to a number of opensource projects, this was possible before, but this state of affairs did not give all the possibilities, and from a licensing point of view it was unacceptable. Now everyone can officially develop software that uses the full power of Kinekt to its fullest.

List of SDK features:
  • C ++, C #, or Visual Basic Development
  • Low-level access to sensor data (camera, depth sensor, microphone array)
  • Skeleton tracking for one or two people per frame
  • Advanced work with sound (noise and echo cancellation, speaker definition, integration with speech recognition engine)
