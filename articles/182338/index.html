Intel will introduce Tizen for developers at DevConf 2013 - June 14

Tizen is an open source software platform supported by leading mobile operators, device manufacturers and chipset suppliers.

Tizen provides a reliable and flexible environment for HTML5-based application developers. Using the incredible features of HTML5 and multiplatform support, it is quickly gaining popularity as a development environment for mobile applications and services. The Tizen SDK development kit and application interface allow developers to use HTML5 and related web technologies to create applications that work in segments of various devices.
