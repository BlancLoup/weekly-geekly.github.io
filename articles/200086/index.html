Corona SDK will support Windows Phone 8 and Windows Store


Corona Labs , the creators of the cross-platform solution for the development of mobile applications, announced the start of cooperation with Microsoft. Windows Phone 8 support will be implemented in the first quarter of 2014 and later Windows Store.
