It remains to flash ramdisk into memory (I used burntool under xp) and - voila, the boot phase. Yes, I stitched instead of recovery, so to start Linux you have to hold down the keys at startup.

Configure autologin on the first console

In /etc/init/tty1.conf we replace the last line with exec /bin/login -f Yor_Login < /dev/tty1 > /dev/tty1 2>&1 then in your ~ / .bashrc you can write commands that will start with automatic login
