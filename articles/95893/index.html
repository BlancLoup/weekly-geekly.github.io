GNU / Linux Audio Player - DeaDBeeF

Today I found for myself a very nice player DeaDBeeF. But what to say, after Audacious and Exaile is just a fairy tale, not a player. And so, what we have:

DeaDBeeF is an audio player for GNU / Linux systems. Distributed under the General Public License version 2.

mp3, ogg vorbis, flac, ape, wv, wav, m4a, mpc, cd audio (and many others);
utf8, cp1251 support;
cuesheet support (.cue files);
does not depend on GNOME, KDE;
trey;
drag and drop;
hotkeys;
playlists;
18-band EQ (supports foobar2000 presets);
radio;
plugins;
and much more

Install :
1) Lucid, Karmic.
sudo add-apt-repository ppa: alexey-smirnov / deadbeef
sudo apt-get update
sudo apt-get install deadbeef
2) Other distributions
