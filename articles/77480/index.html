Mozilla Test Pilot: New Test


For the "test pilots," a new task - " A Week in the Life of a Browser ". This study will collect information about bookmarks (including the number, number and depth of folders in the "favorites"), the number of downloads and extensions, as well as the browser time. The test will run until December 14 and will be repeated every 60 days.
There is also a new survey ( Pilot Background Survey ) in which you are asked to answer eight questions (how long have you been using the Fire Fox, are you using other browsers).
