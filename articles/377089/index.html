Signal 2.0 - Free Encrypted Messages and Calls



Developers from Open Whisper Systems on March 2 in their blog announced a new version of the iOS-application Signal 2.0 . This program is designed for secure communication. It supports encrypted calls, text messaging, pictures and video. All the functionality of the application is available for free. The application does not use standard SMS and MMS channels.

Not only iOS users can communicate with each other - Signal is compatible with TextSecure android applications (text messages) and RedPhone (calls). These applications are also being developed at Open Whisper Systems. Signal works on the push scheme, so the exchange of messages takes place in asynchronous mode. Encryption keys are stored only on the end devices themselves, and the source of the programs is open and uploaded to GitHub . The program also blocks attempts by third-party applications to make a screenshot during its work. This makes Signal one of the safest systems for private communication.
