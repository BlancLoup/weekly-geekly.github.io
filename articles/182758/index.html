Windows 8 Technology Conference

On June 18, 2013, a technology conference on Windows 8 will be held in Moscow.



This is the first conference in Russia, in which Windows 8 is considered from the point of view of an IT specialist. It will discuss both the components of the OS itself and the scenarios for the use of various services that use a bunch of Windows Server 2012 + Windows 8. Leading Microsoft experts will cover such important topics as:
  • tools and ways to deploy Windows 8 across the enterprise;
  • Remote access to Windows 8 and OS usage scenarios by mobile workers;
  • features of installing a new type of applications (WinRT-applications) in the corporate environment;
  • Windows 8 security and system recovery tools.


A detailed program of the reports can be viewed below. As the event takes place in the IT Camp format, in addition to theoretical reports, you can attend laboratory classes where you can perform practical tasks in a modular environment.

Pre- registration is required for participation in the conference. Participation in the conference is free!

To register

If you do not have the opportunity to personally attend the event, you can watch the on-line broadcast of the event, which will be available on the day of the event on TechNet.
