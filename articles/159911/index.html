New version of NetWrix Change Reporter Suite provides reliable control over IT infrastructure



Last week, we already wrote about the release of a new version of our solution for auditing Active Directory - NetWrix Active Directory Change Reporter 7.2 . But in addition to this event, we had another important release - the NetWrix Change Reporter Suite received an update to version 3.2. The solution received a module update for Active Directory auditing and a new module for consolidating data from event logs for Windows and syslog.



What does version 3.2 give?
  • Installing the software package has become even easier - version 3.2 adds automatic installation and configuration of SQL Server 2012 Express Edition with Advances Services (SQL Server 2008 for Windows Server 2008, Windows Server 2003 or lower)
  • Active Directory Audit Module - NetWrix Active Directory Change Reporter - received support for Windows Server 2012 / Windows 8 and Exchange 2013. Now Active Directory auditing is possible on all supported Microsoft operating systems
  • Updated modules for auditing Exchange Server (Exchange Change Reporter now supports Exchange 2013), Group Policy (Group Policy Change Reporter works with Windows Server 2012 / Windows 8) and recovering AD objects
  • Added module for managing NetWrix Event Log Manager event logs . It is used to consolidate and archive event logs in Windows and syslog, receive alerts about individual events. With the addition of this module to NetWrix Change Reporter Suite 3.2, more than 30 new reports on changes in IT infrastructure have become available.
  • The SQL Server Change Reporter 2.5 module supports auditing the configuration and contents of SQL Server 2012 databases.
  • The entire software package can now be installed on Windows Server 2012 and Windows 8 , ensuring stable and reliable operation in the most modern infrastructure. All products included in the package now support work on Windows Server 2012 domains
  • Also received an update to the NetWrix Management Console , which provides easier management of all components included in the solution NetWrix Change Reporter Suite
