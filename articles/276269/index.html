Tor Browser 5.5 released



The Tor Browser developers have released an updated version of their software, the current version is 5.5. As before, the program is designed for users who would like to ensure the anonymity and security of their personal data on the web. Developers have updated many features, plus added several security updates. As you know, a feature of this browser is the redirection of traffic through the Tor network. The basis is Firefox. As a result, determining the user's IP is almost impossible.

Addition to the browser HTTPS Everywhere allows you to encrypt traffic on any sites that support this feature. Also in the bundle, the user receives the addition of NoScript, so that the user is not exposed to the threat of carrying out attacks using JavaScript.

In the new version of the browser, the user gets protection against attacks on user identification, using brute-force fonts on the victim's PC. The developers also implemented isolation of the Shared Workers handlers, as well as improved protection against attacks on keyboard input patterns. The following components have been updated:
  • Firefox 38.6.0esr;
  • libevent 2.0.22-stable;
  • NoScript 2.9.0.2;
  • Torbutton 1.9.4.3;
  • Tor Launcher 0.2.7.8.


Now we are working on the release of the version of Tor Browser 6.0 and Tor Browser 6.0-hardened, the main one for which is the Firefox 38 ESR.
