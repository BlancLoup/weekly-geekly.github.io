LibreOffice RC3

The Document Foundation announced the release of the third release candidate (RC3) LibreOffice 3.3.

The final release of LibreOffice 3.3 will be released soon.

The new release candidate has made several fixes for Ubuntu 10.10. Other changes include translation and branding updates, bug fixes and overall performance improvements.

A complete list of changes can be read here .

Users who have added a PPA should receive an update soon. Download the release candidate for Windows, Linux or Mac OS X on this page: libreoffice.org/download .
