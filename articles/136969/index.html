Improvements to working with PDF in Evernote for Windows

Today's Evernote update for Windows has received several enhancements related to PDF. If you store files of this type in Evernote, then this update should please you, since the work of Evernote with PDF has improved noticeably in almost all aspects.

Download Evernote for Windows ¬ª

New functionality opens up a number of previously inaccessible features for notes with PDF. Search results inside the PDF are now highlighted in color, you can easily drag and drop files from Evernote, you can even copy text. Also, moving the cursor to the area with PDF, you will see a new toolbar. And now more about these features.

Search inside PDF
If you have a scanned document or digital PDF in Evernote, then the search results found in its text will be highlighted right inside the file. Clicking the right and left arrows on the toolbar for PDF you can move between the results found.




Drag and Drop PDF from Evernote
In addition to navigating through the search results, the new PDF toolbar also allows you to drag and drop files of this type from Evernote to other applications, to the desktop or somewhere else. To do this, click and hold the PDF icon on the right side of the panel, and then drag the file to the desired location.



Copy and Paste
If your PDF file contains highlighted text (as well as text recognized by our servers in images), you can select and copy this text right inside Evernote and paste it into other applications.

PDF and Evernote Premium Account


One of the features available to Evernote Premium Subscribers is automatic text recognition in images inside PDF files. Now, when search results in PDF are highlighted in color, this function becomes even more useful.
