We look into the clouds

The other day, we have a new product line, into which cloud protection has been deeply integrated as never before. This technology is both us and it is called Kaspersky Security Network.

Of course, cloud protection alone is not a panacea, because then it would be enough for a malicious program to simply turn off the Internet in order to deprive the user of anti-virus protection. That is why in the new line we stopped at the hybrid protection, which flexibly uses the advantages of several detection technologies at once.
