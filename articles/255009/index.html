What is a slicer?

Slicer (from the English. Slice- cut) is an auxiliary software that converts the 3d model into machine code (as a rule it is G-code). This code contains the commands for the 3d printer control board (or CNC machine). The temperature of the heating element of the extruder, the feed rate of the plastic thread, the speed of movement, the speed of the cooling fan and most importantly all the coordinates for building a physical copy of the 3d model and its auxiliary elements.

In the software for managing a 3d-printer - Repetier, it is possible to use one of the three slicers: Slic3r, Skeinforge, Cura. Each of them has its own algorithms for converting models. I use the first one.
