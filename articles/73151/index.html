Psystar announced "Rebel EFI" allowing you to install Mac OS X on ordinary PC

Psystar today announced the release of Rebel EFI - a new software product that allows you to install any modern operating system, including Mac OS "Snow Leopard", on a number of PCs.

Psystar Corporation begins selling software to run OS X Snow Leopard on a standard Intel-compatible hardware. The ‚ÄúRebel EFI‚Äù suite available for download from Psystar.com provides users with the ability to install any operating system on their computers. Today Rebel EFI supports machines with Intel Core 2 Duo, Core 2 Quad, i7 and Xeon Nehalem processors. Users can now simply download new software from Psystar.com, burn it to a CD and follow the instructions when downloading it.




The product is available as a limited demo version. It allows you to install Mac OS X on your computers with reduced support for hardware and imposes a time limit of two hours, unlike the full version. Regular price for the full version is $ 89.99. But now there is a discount and the product can be purchased for $ 49.99. Psystar also declares that it will consider the suggestions of users to include in the composition of ‚ÄúRebel EFI‚Äù hardware that currently does not fall within the system requirements of the program.



The program also supports the released release of Windows 7. Through the Darwin Universal Boot Loader, available in the full version of the program, the multi-boot mode of the following operating systems is possible: Windows 7, XP, Vista, various Linux and OS X distributions of Snow Leopard.

According to the developer, Rebel EFI supports a wide range of hardware. Moreover, during installation, the program automatically detects hardware and, if necessary, downloads new or updates obsolete components and drivers.

Apple and Psystar have long been embroiled in lawsuits about the fact that selling Macs with Mac OS pre-installed violated the operating system license by selling Mac OS clones. What will happen this time and how they will react to Cupertino for such a bold move can only guess.
