Mobile HTML5

Today we have a new book - HTML5. Mobile application development .



The book is available in print and electronic format. When you purchase a virtual book, you will receive two formats - PDF and ePub. During the week there is a 10% discount on sections of HTML and XHTML. Web Markup Languages and O'Reilly's Bestsellers . Coupon code - e46e73d .

Some information about the book.

The author of the book Estelle Vale is a client interface developer who has been programming standardized and user-friendly websites since 1999. She maintains a technical blog with a multi-million user audience, and also lectures around the world on CSS3, HTML5, JavaScript, client code performance, and mobile web development.

Annotation:



Create stunning websites and applications for any mobile (and desktop) platform. To do this, you just need to add HTML5 and CSS3 technologies to your web development toolkit. Armed with this practical book, you will learn how to develop web applications that not only work well on iOS, Android, Blackberry and Windows Phone, but are also very convenient to use.

The book contains many examples of code and markup. It will help you master the work with a variety of HTML5 features - in particular, with new web forms, scalable vector graphics (SVG), canvas (Canvas), localStorage and other related APIs. In addition, CSS3 style sheets will be discussed in detail in this book. You will learn to develop applications that work equally well on huge monitors as well as on tiny screens.

  • Get familiar with the HTML elements, syntax and semantics of this language.
  • Learn how to create web forms that are more user friendly, but contain relatively little JavaScript code.
  • Explore HTML5 media APIs for graphics, video, and audio
  • Provide offline work of your applications - AppCache, localStorage and other APIs will help you with this.
  • Learn everything you need about CSS3 selectors and syntax.
  • Understand the CSS3 features: multiple background images, gradients, border images, transitions, transformations, and animation
  • Make your web applications more convenient, responsive and accessible.
  • Design for high performance, user experience and functional reliability on all platforms.
