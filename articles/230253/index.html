An animator from Russia is working on creating a 3D version of the intro to Futurama.



Animator Alexei Zakharov, apparently, is a fan of the show Futurama, because he has a lot of work tied to the series. These are renders of a spaceship from a cartoon, detailed drawings of scenes from the series.

Now the animator is working on creating a 3D version of the intro for the series, with some of the work already done. This is the very beginning of the introduction, done very, very carefully.

According to the author of the work, he used 3ds Max, Nuke, Photoshop, After Effects to create the video. Now only the first part of the introduction is ready, let's hope others will follow.
