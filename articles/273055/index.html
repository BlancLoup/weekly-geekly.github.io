Open-source implementation of domestic cryptoGOST

At the weekend I decided to look for open-source implementations of domestic cryptographic standards. First of all, they were interested in new ones: Stribog hash function (GOST R 34.11-2012), Grasshopper (GOST R 34.12-2015) and EDS (GOST R 34.10-2012 or 2001 (without 512-bit)). The old GOST 28147-89 was not specifically sought, since to find its implementation there were no problems for a long time.
So let's see what happened. Immediately I warn you that I did not check the correctness of implementations.

Grasshopper


  • Official implementation of TC 26
  • The implementation of Markku Saarinen (not optimized)
  • The implementation of Roman Oleynikov (not optimized)
  • Implementation optimized for 8/16/32/64/128 bit architectures
  • Implementation of Eric Filiol (not optimized)
  • A KeePass plugin that implements Grasshopper
  • JS implementation of the WebCrypto project
  • Assembly implementation (UPD by @omegicus)
  • C ++ implementation
  • Implementation in Python by @NeverWalkAloner
  • Implementing on Go by @ stargrave2
  • Another Go implementation
  • Python implementation on @ stargrave2
  • Implementation on C99 with SSE2 optimization by @aaprelev


Stribog


  • The famous implementation of Degtyarev
  • Calculation of Stribog hash sums on Qt (based on Degtyarevskaya)
  • Implementing Kazimirov (C and sage)
  • Assembler by @omegicus (not optimized)
  • Implementation in libgcrypt
  • Java implementation
  • Pure C
  • Implementation in Gostcrypt - fork Truecrypt
  • PHP implementation of Kolesnikov (development of Degtyarevskaya)
  • Cryptocurrency SibCoin on Stribog (Cryptocurrency on Stribog, Karl!)
  • JS implementation of the WebCrypto project
  • Python implementation
  • C ++ implementation
  • LibreSSL
  • Go implementation
  • Optimized by @ Ivan_83 + HMAC
  • Openssl engine


Signature

  • Python implementation
