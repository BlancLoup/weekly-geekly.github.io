Firefox 1.5 latest update

Mozilla has released a patch covering several holes in Firefox 1.5.x and Firefox 2.x browsers. In particular, one critical vulnerability is closed and another dangerous hole associated with the use of the addEventListener function in malicious scripts that steal passwords. In addition, some minor holes and irritating behavior of the browser when filling forms are fixed.

Release numbers increased to 1.5.0.12 and 2.0.0.4, respectively. It is noteworthy that for the Firefox 1.5.x family this will be the last release. This browser support, which was released in November 2005, is scheduled to stop, and users are encouraged to switch to Firefox 2.x. In the next few weeks, Firefox 1.5.x users will receive an automatic update message asking them to install Firefox 2.0.0.4.
