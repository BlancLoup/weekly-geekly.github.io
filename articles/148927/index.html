Visual Studio 2012 will be available on September 12



Immediately following the announcement of the RTM versions of Windows 8, Windows RT and Windows Server 2012, Microsoft added Visual Studio 2012 to a number of completed major projects.

According to a published entry in the official Microsoft blog yesterday, final builds of Visual Studio 2012 and .NET 4.5 were assembled. In fact, for MSDN subscribers, products will be available from August 15; Together with them, developers will receive a free account in the Windows Store for a period of one year.

Formally, the public announcement of Visual Studio 2012 Microsoft plans to implement at the online event visualstudiolaunch.com on September 12, which to some extent should be associated with the figure 2012 in the name of the system.

Some time ago it became known that free Visual Studio Express 2012 would only support the creation of Metro applications, which caused a negative reaction from the developer community. Microsoft listened to dissatisfied voices and changed the release policy - the ability to create ordinary desktop applications for Windows was added to the version of Visual Studio Express 2012, which was officially announced on the developer's blog on June 8.
