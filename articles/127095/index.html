Fake LJ comments from famous bloggers



In LiveJournal found a way to use html-code in the comments to create fake comments from well-known bloggers to mislead readers. The method is based on a sophisticated imitation of the presence of a comment due to a special layout of the source code of the comment.
As a result, one comment is visually perceived as two comments, one of which is phishing.



eugenyshultz :
Given the high social and political significance of LiveJournal, I consider it necessary to warn about the presence of an elementary method that allows you to create a comment in any journal that looks no different from the present. You can expose by viewing the source code of the page in HTML, but it is time consuming and obviously the overwhelming number of users will not do this.

Today, I deliberately - as a demonstration of possibilities, I left comments in the most popular journals, in which HTML code was inserted, which externally imitated another comment. From a technical point of view, I left a comment, which is mine, but EXTERNAL it looks like a comment from another user.
