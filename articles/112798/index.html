An updated version of the Web Camps Training Kit has been released.



An updated version of the package of educational materials developed to support Web Camps conferences, which Microsoft regularly conducts on web development, has been released.

This kit contains training materials on the latest web technologies that Microsoft is developing or promoting. Among them: ASP.NET MVC 3, Internet Explorer and HTML5 / CSS3 / JavaScript web standards, open Odata protocol, jQuery library, web applications and CMS, new WebMatrix application development kit.

The Web Camps Training Kit contains presentations, demos and labs with a full step-by-step description. In addition, the developers will find a free book and other materials on the WebMatrix development environment.

Summary Web Camps Training Kit:

ASP.NET MVC 3
  • ASP.NET MVC 3 Fundamentals
  • ASP.NET MVC 3 Razor
  • ASP.NET MVC 3 Models and Data Access
  • ASP.NET MVC 3 Helpers, Forms and Validation
  • Validation, Data Annotations, Client side validation
IE9 and HTML5
  • Building A Fast Web Experience
  • IE9 Dev Overview
  • Pinned Sites in IE9
Odata
  • Open Data for the Open Web
  • Session 1 - OData Concepts
  • Session 2 - Building OData Services
  • Session 3 - Consuming OData Services
jQuery
  • Session 1 - jQuery Fundamentals
  • Session 2 - Ajax
  • Session 3 - Templating and jQuery UI
Webapps
  • Umbraco CMS
  • Kentico CMS
  • DotNetNuke CMS
Webmatrix
  • Learning Web Matrix (3 pieces)
  • WebMatrix eBook
  • WebMatrix Whitepaper
  • Creating web pages using WebMatrix: A 6 Part Tutorial for Students
