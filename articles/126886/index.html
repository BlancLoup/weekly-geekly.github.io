GIMP 2.7.3 with single-window mode released



Unstable version 2.7.3 of GIMP graphic editor released.

In this version, finally, you can see the implementation (although not yet fully completed) of the long-awaited single-window mode. Single-window mode is not enabled by default, it can be enabled via the menu: Windows ‚Üí Single-window mode.

In addition, this version provides support for layer groups and session management, improved text editing on canvas.

A complete list of changes from version 2.7.2 can be found at developer.gimp.org/NEWS .

The release of the next stable version of GIMP at number 2.8 is scheduled for the end of this year.

The source code of GIMP 2.7.3 can be downloaded from the official GIMP website . Ubuntu users can use unofficial PPA ppa: matthaeus123 / mrw-gimp-svn .
