Microsoft has released a P2P client for Windows 7

Microsoft has released in open access the beta version of the program Windows Live ID Sign-in Assistant 6.5 , which allows you to directly copy files from computer to computer. A user with a Windows Live ID account indicates in the settings for which other accounts he is allowed to open his files, and those become available on the local network or via the Internet.

Thus, you can easily share folders, for example, with fresh movies for friends, colleagues or family members.
