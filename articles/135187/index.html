Dropbox for Android 2.0!

Dropbox has updated its Android app to version 2.0! The design has been completely redesigned and added several new features!

Several new features :

Favorites: Want to know for sure that the file is always in the phone? Label will help you to quickly access the file offline.
Bulk upload: No need to upload files one at a time. Select multiple photos and videos to be uploaded in the background.
Rename files: files and folders can be renamed.
Ice Cream Sandwich support: Optimized for the newest Android devices.
Access to actions 1 by pressing: All files and folders of actions are now accessible with one click.
Many others: Improved gallery view, export (and load from) local data storage, etc.
