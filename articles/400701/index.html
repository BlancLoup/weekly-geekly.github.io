D-Wave published qbsolv tool for programming quantum computers on Github



D-Wave, a developer of quantum computing systems and programming tools, published the source code for his project qbsolv on Github , as reported on the company's official website .

D-Wave quantum computing systems have been tested by NASA, Google, and some laboratories since 2013. Translation of the main tool - software capable of creating code for quantum systems built on the principle of using qubits - in open source, will significantly accelerate the development of this field. In any case, they think so in D-Wave.

The main difficulty in using quantum computing systems is that the standard von Neumann logic does not apply to them. A qubit - a quantum digit used to store information, has two states at once. Using only 0 or 1 in it, as it happens for current classical systems based on the use of binary code, is impossible.

For this reason, software that can program quantum systems is almost the same, if not more valuable for progress, than quantum computers themselves.

Qbsolv is used for solving optimization problems that have wide practical application. With this software, it is possible to process large amounts of data, automatically breaking them up into small segments, which are processed in parallel by individual segments from the qubits of the D-Wave quantum processor, and then combined into one common solution.

As the company claims, with the growth of the power of quantum computing systems, the task segments suitable for processing by a quantum computer will also increase, which will lead to a significant increase in the final efficiency. At the same time, according to the developers, the qbsolv package allows you to increase the performance of existing D-Wave computing systems up to twenty times.

In addition to qbsolv, there is another tool for programming quantum systems - the qmasm package, the source code of which is published on Github by Scott Pachin from the Los Alamos National Laboratory. It is also applicable to work with computers D-Wave. Now the laboratory staff switched to using qbsolv.
