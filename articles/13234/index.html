Movable Type 4 released

A year later, the company Six Apart has released Movable Type 4 . Like previous versions of MT, the new version is distributed under both commercial and non-commercial licenses. The new OpenSource version , distributed under the GPL license, includes the main Movable Type functionality.

Significant new features (some of which are not available in the GPL version) are support for user registration and the ability to comment using OpenID. With the help of the new package, you can implement the social network out of the box, allowing users to create their own profiles, as well as evaluate others. Also, this product allows administrators to make commentators authors.
