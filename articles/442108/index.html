Guide for developers to create AI applications

Create your first smart bot with Microsoft AI


Artificial intelligence accelerates digital transformation in every industry, including manufacturing, retail, finance, healthcare, and many others. At this level, each industry will be able to use AI to improve their work. In this e-book, Anand Raman and Wee Hiong Tok from Microsoft show developers a comprehensive development map for creating their first artificial intelligence application.


Using Conference Buddy as an example, you will learn the key components needed to develop an intelligent chat bot that helps conference participants interact with presenters. This e-book is a brief introduction to tools, infrastructure and services on the Microsoft AI platform and teaches you how to create powerful intelligent applications.


  • Understand how a bunch of clouds, data and AI allow organizations to create intelligent systems.
  • Explore the tools, infrastructure, and services available as part of the Microsoft AI Platform for developing AI applications.
  • Teach the Conference Buddy app new artificial intelligence skills using the previously created artificial intelligence capabilities such as vision, translation, and speech.
  • Learn about the Open Neural Network Exchange.
