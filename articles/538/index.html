C Parallels Windy put faster on the Mac!

Parallels has announced the release of a new version of Parallels Desktop, the host package for creating virtual machines under Mac OS X. Now the program boasts a shocking feature - installing Windows on a Mac with its help is faster than a typical installation of Windows on a PC.

This effect was achieved through the new Parallels Installation Assistant wizard, which fully automates the creation of a virtual machine and the installation of Windows. All that user Q needs to do is select a version (XP or Vista), enter his or her name and company name, as well as an activation key. An optimal virtual machine will be created for the system. The Installation Assistant wizard also supports installing other versions of Windows, as well as FreeBSD, Solaris, OS / 2, eComStation, MS-DOS, and any Linux build. In each case, the user can select the express mode or independently create a virtual machine to your taste.
In addition, the program has a special network compatibility mode Shared Networking. With it, you can connect multiple virtual machines to the Internet from a single IP address. For connection you can use any real connection - from a modem to a cable network.

The new version of Parallels Desktop is a build version numbered 1970. Package owners can update it through the standard Check for Updates feature in the Help menu. For new users, the program will cost $ 79.99. The package also has a free trial that runs for 15 days.

According to MacDailyNews and www.maccenter.ru
