SWUSE - creating a design (pre-) beta

Some time passed and a new version of the cutting design for CMS SWUSE appeared . This time everything has become more professional, but not yet ideal. I will not tell what it is, in the last post I wrote about it, so I‚Äôll tell you what's new and what has been updated from the old one.

Work with PSD and layers

In the past (alpha) version, only single-layer simple images could be cut into pieces, which reduced the possibility of the layout designer. In the new version, it is possible to download PSD (photoshop) files, which means that now the template can consist of several layers, layers can be applied, positioned, etc. Now you can change the background of the selected area when you hover the mouse (convenient for any buttons). Layers can be selected, hidden, saved in different formats (jpg, gif, png + alpha).

CSS editing
This time, the CSS can be edited together with the template, this is quite logical, although previously the CSS was edited separately from the template, and the design change killed the CSS file.

Today (September 30) all this is still a pre-beta, because there are several known bugs, some of which are not written.
