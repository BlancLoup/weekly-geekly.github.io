Ubuntu, Kubuntu and Xubuntu 11.10 operating systems OEM in the assembly from UALinux

Today on the UALinux website there was news that they presented their Ubuntu, Kubuntu and Xubuntu 11.10, targeted at OEM equipment suppliers (Original Equipment Manufacturer), which presupposes the delivery of equipment with pre-installed distributions, as well as free download and installation. by users.
The main differences from the base distributions are that these assemblies include everything you need in order to immediately start working with the system without having to download something from the Internet, or install it additionally.


The main differences from the base distributions are as follows:

- Localization packages are included, so the system immediately after installation supports Russian, Ukrainian and English.
- Installed all the necessary tools to work with multimedia: audio, video codecs, Flash plug-in, etc.
- Archiver supports most existing types of archives
- OpenJava is installed in the system
- Batch database is updated as of December.

Like the Ubuntu base line, systems are presented for several target audiences (Unity or Gnome 3 (Gnome-Shell and Gnome-Classic) in Ubuntu, or KDE in Kubuntu), as well as for weak systems, which are now in abundance in the state. Institutions, a variant on Xfce in the Xubuntu distribution is provided.
