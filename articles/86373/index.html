Just in case, let me remind you what a linked list is. A linked list is a data structure, which is a set of elements, each of which consists of two parts: the actual data and the pointer to the next element. This is one of the simplest data structures, it is used everywhere, for example, to implement a stack, a queue, a hash, lists with gaps, etc.

The patent describes one of the varieties of the list, namely the doubly linked list . The elements of a doubly linked list contain two pointers: the next and previous elements. Thanks to this list can be viewed both forward and backward.
