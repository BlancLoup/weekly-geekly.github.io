Base GeoIP - countries and cities, September 2013

An updated version of the Russian-language database of countries and cities has been released.

The database is distributed in two versions: " Countries and cities " (13Mb, after installing ~ 74Mb) and " Only countries " (2Mb, after installing ~ 9Mb). Also in the archives is a small example of using the database in php.

The accuracy of determining the country ~ 99%, the accuracy of determining the cities on average ~ 95%, the accuracy of determining the cities of Ukraine ~ 50%. A lot or a little depends on the tasks that you are going to solve with the help of this base.

Test demo - determine your country and city by IP-address

Work demo - definition of your city in the form of air ticket search

To install the database, I recommend using programs like mysqldumper or via ssh to execute

Update maxmind database of cities takes 10-15 minutes.

In the directory of countries there are islands that are not independent states, but significantly removed from their metropolitan areas.

In the directory of cities errors are possible (including duplicates).

Theoretically, European countries should be more accurately determined by the first option, but there was no practical test.

Ideas to improve and improve the quality of the base are welcome.
