Smart goals



In mid-December, it became possible to use Smart Targets in Google Anaytics.
They allow you to identify visits with the best metrics that you can use as conversions to optimize your AdWords campaigns. Each visit is evaluated, and Google Analytics identifies ‚Äúbest visits‚Äù using machine learning technologies. To assess the quality of the visit, many factors are used, for example, the duration of the session, the depth of viewing, information about the location, device, browser.
This function is appropriate to use for sites that do not have explicit target actions, as well as to optimize campaigns in AdWords.
To start using Smart Targets, you need the following: Google Analytics and Google AdWords accounts must be linked, at least 1000 clicks on ads from a linked AdWords account must be recorded in the last 30 days in the Google Analytics view.
How to use this feature: Create a goal in Google Analytics



By activating Smart Targets in Analytics, we can import them into AdWords and use them to optimize bids in ad campaigns.
