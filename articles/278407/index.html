The key to JetBrains products for all students on Stepic.org

This spring, JetBrains gives the keys to its products to all active students of mass open online programming courses on Stepic.org !




We at Stepic.org, of course, believe that it is great to study at any time of the year, but in the spring of '16 it will be especially pleasant to do this!

TL; DR


We will send an email with an activation code for any development environment from JetBrains IDE to everyone who studies at the open online courses Stepic.org. Important: The code is intended for educational use only.

How to get the code?


Start or continue to take any course on programming on the platform Stepic.org, solve problems, write the code, and the activation code will come to you in the mail within a day. Additionally, we recommend checking that you have received email notifications ( in the profile settings ), otherwise the code will not reach you.

About JetBrains and code activation


JetBrains produces professional software development tools that support the most advanced technologies, frameworks and programming languages. You can download any JetBrains product at https://www.jetbrains.com/products.html , you will need to use an activation code when you first start it. You can use multiple JetBrains products with this code at the same time.

How long will the license be valid?


The license is valid until May 15, 2016. If this time is not enough for you, just continue to study in Stepic.org courses and you will receive a new activation code.

Last but not least


We are grateful to JetBrains for providing an opportunity for our students, as well as for the support and commitment to education!

Have a good training and develop with pleasure,
Team Stepic.org.
