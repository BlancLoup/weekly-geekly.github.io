New free translation tools

If you use an online service, the main problem is that the service must either be placed as favorites, or constantly typed, or kept minimized. Routine gestures with the mouse. But there is something that can make your life easier if you still do not agree to purchase an electronic dictionary. This is an online translation module.

What is this all about? And to the fact that recently on the www.lingvo.ru/tools posted modules for quick access to the translation from the browser. There are tools for two main browsers - Internet Explorer and Mozilla Firefox, and a standard Vista gadget.

If you are using Internet Exploreer, you can embed the Lingvo Online button in the Google toolbar. The translation will open in the same window. The word you selected with the right mouse button is translated.



The solution for Firefox looks more interesting - this is the line for entering the word on the toolbar, and you can choose the translation direction in the line itself.

It looks like this:



The word must be typed or copied into the translation field. Translation opens in a new tab.

From the same module with one click you can get to the forum Lingvo.

Perhaps, there is only one thing to say: Hurray!
