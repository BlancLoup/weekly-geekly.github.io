Daemon Tools Pro v4.10.0218

Some time ago, a new version of the famous virtual CD-ROM Daemon Tools was born. The novelty supports up to 34 virtual disks, a new customizable interface, an improved reading mechanism, extended online help, a Russian interface and an SPTD driver version 1.50.

If the third version of Daemon Tools was just a good virtual CD-ROM, with almost no interface, by the fourth program lost its free, got additional features and became something like a completely commercial program Alcohol 120% . Now, in order to use Daemon Tools Pro, you must register on www.disc-soft.com and install the WhenU Save Adware module, which shows ads as a payment for using the program. There is also a paid version of Advanced, which among other things is able to create CD images on its own. However, one advantage from Daemon Tools does not take away - with the help of this program you can use backup images of even those disks that are protected using Safedisc, Securom, Starforce and other copy protection systems.

Download Daemon Tools:
- daemon410-x86.exe , 1.8Mb
- daemon410-x64.exe 2Mb, for 64-bit Windows
- 1049.dll (library for adding Russian interface)
