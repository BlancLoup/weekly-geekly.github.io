Why Telegram


Usually, a separate application is immediately released for such projects, but Telegram as a platform has a number of attractive advantages.

  • By launching a bot, not a separate application, you open the doors, behind which there is more than one hundred millionth (judging by the growth rates mentioned by the Durovs - the 200 millionth soon) army of messenger users. And convincing a user to try a chat bot is much easier than putting up a whole new application.
  • Telegram is a multi-platform instant messenger, so with the help of a bot you get access to all platforms for one code. This is not the same as developing separate applications for PC, Mac and mobile systems.
  • And, as you can see, including from the screenshots above, Telegram now provides the most advanced API for creating bots, far ahead of WhatsApp and Viber.

The creators of Callcoin deliberately tried to make a kind of model bot, using the API's capabilities to the maximum, including integration with the user's notebook, ubiquitous emoji, be they wrong, and, of course, a multi-level menu. Of the new features of the API, only HTML5, which was presented by Telegram in October, when Callcoin was already launched, remained untouched.
