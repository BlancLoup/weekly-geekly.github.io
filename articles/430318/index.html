LVEE - the most informal and soulful IT party

About event


The annual LVEE 2018 conference (Linux Vacation / Eastern Europe), organized by the Belarusian Linux user group in 2005, concluded near Minsk. The event was attended by over a hundred people from 7 countries, and the program included 23 reports, lightnings, a workshop, a round table, a live demo, and a traditional evening program of the informal part with barbecues and free discussions.







The reports covered a wide range of topics - from the introduction and integration of free software to low-level features of the implementation of system libraries and packages.

Alexander Klyga and Vlad Sharpio (Belarus, Minsk) gave reports on data storage systems, Vadim Zhukov (Russia, Moscow) and Alexander Zaitsev (Belarus, Minsk) on modern approaches in developing and debugging C-applications. Vladislav Zhalnerovich (Belarus, Minsk) read an introductory note about the low-level structure of libraries in the ELF format. Roman Garanin (Belarus, Brest) spoke about the work of DevOps for QA using the example of a Java Enterprise project. Alexey Cheusov gave a visually vivid report in which he described his experience in post-processing photographs in Darktable.

Alexey Khlebnikov (Norway, Oslo) shared how to optimize the execution of a program for Android applications using the advanced obfuscation method of constants.



Several reports dealt with the implementation of open source software: Anastasia Markina and Alexander Grishkin (Russia, Moscow) shared their experience of implementing in the VGIK branch, Dmitry Stepanov (Belarus, Minsk) - the experience of transferring the company's infrastructure from Windows to a heterogeneous architecture based on Linux, Andrey Romanyuk (Belarus, Minsk ) told how open source software changes the enterprise sector. The report by Anton Novik (Belarus, Minsk) was about the development of the architecture of the Game-stream platform based on OpenSource components.

What give


LVEE, like any other event, tries to attract attention not only with cool content, but also with unusual merch.

For each event a special branded production is made: for T-shirts - green T-shirts, sometimes with individual branding, for speakers and sponsors - the same, but black, for all other participants - white branded T-shirts. In addition, round magnets with the LVEE logo, pints, circles, collections, and stickers are put into the participants' packages.
