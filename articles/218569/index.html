Recently, developers Raspberry Pi introduced a new product. This device / board is based on the Raspberry Pi, only smaller in size, with basic configuration. The new project is called Raspberry Pi Compute Module. The size of this device is the same as the size of the RAM bar for a laptop. On the "bar" was only the processor, ROM and RAM.

What is all this done for? In order for third-party developers to create mini-PCs based on a central module, with the functionality of such a PC is higher than that of the Raspberry Pi. For example, you can create your tablet, netbook or board, with specific functionality.

To demonstrate the capabilities of the Raspberry Pi Compute Module, the authors of the device released the Compute Module IO Board, an expansion card, with an HDMI / USB connector, a wireless module, and other necessary modules.



As soon as the Raspberry Pi Compute Module connects to the expansion card, the system can work like a normal PC with good functionality.

As for the configuration of the Raspberry Pi Compute Module, the developers left it the same as the Malinki itself, with a Broadcom BCM2835 ARM11 700 MHz processor, 512 MB of RAM, 4 GB of flash memory.

The price of the device, if you buy in bulk in 100 pieces - $ 30. If you buy at retail, so to speak, then more.
