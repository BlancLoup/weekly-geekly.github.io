Seagate Technology announced a 4 TB hard drive



The volume of storage media is constantly increasing, every month we hear something about a new capacious flash drive, or about a new SSD drive. Now there is news that Seagate has released a new model of external hard drive, which has a capacity of 4 terabytes. According to company representatives, the new drive is not intended for corporate needs, but for small businesses whose representatives cannot afford to purchase expensive data storage systems.

The new model is called GoFlex Desk, and according to the developers, this hard disk can be used as a portable storage of information - its dimensions allow you to take a drive on the road. The mass of the novelty is 1.08 kilograms, and the size of the disk is 158 * 124 * 44 mm. In general, a kilogram is quite a lot, but if necessary, you can take it with you on the road.

GoFlex Desk is compatible with USB 2.0, USB 3.0 and FireWire 800 standards. There is no wireless communication module. This model, like all other members of the GoFlex model range, is equipped with specialized software for creating encrypted data archives.



The external drive comes with the NTFS file system, but it is clear that the user can change the file system if they wish. The device is also compatible with a special adapter GoFlex Home, which allows you to turn it into a network drive. The cost of the disk is not so great, and is $ 229. You can buy on Amazon , however, now this product is "Temporarily out of stock".
