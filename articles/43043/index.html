FF plugin: hot keys for search engines

Once again, poking the Google results into the output of Google, I caught myself thinking that it would be more convenient for me to open pages from the issue using the number keys, i.e. ‚Äú1‚Äù is the first page, ‚Äú2‚Äù is the second, etc.

The search for existing solutions is not 100% satisfying, so I decided to write such a small plugin myself. I want to present what happened in your court. Maybe someone will come in handy :).

Supported search engines:


  • Google
  • Yahoo
  • Yandex


Using


To work with the plugin, you need to be on the search engine page with search results, press one of the following shortcuts.
Key combinations:
  • ALT + (0..9) - opens the url from the search results that are in the specified position in the output. The focus is set on the open page.

  • ALT + CTRL + (0..9) - opens in the background pages that are from 1st to the specified position in the issue.

* In both cases, the countdown is from one. "0" - opens the 10th link in the issue.
