Introducing the updated Webmaster Tools API

Webmaster Level: Advanced


All summer, the Webmaster Tools team has been working on updating our API . Compatible with other Google APIs, it simplifies authorization for applications and web services, and also allows you to work with the basic functions of Webmaster Tools.

Users with experience with other Google APIs can easily upgrade to the new Webmaster Tools API. We have already prepared its variants for Python , Java and OACurl .

Using this API, you can:
  • add and remove sites from the account, as well as organize them (now there can be up to 500 sites in the account);
  • add and delete Sitemaps, and make lists of them;
  • receive statistics for warnings, errors, etc. on specific Sitemaps;
  • request lists of all crawl errors on your site;
  • save examples of various scan errors;
  • Mark individual scan errors as fixed (this will not affect their processing, but will simplify your user interface).

Tell us about the tasks you solve with the Google API. If you have questions about how to use the API or you want to talk about your project, leave comments below, write to our webmaster community or Help Forum .
