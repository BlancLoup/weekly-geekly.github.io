Google Analytics Reporting Suite

Google Analytics Reporting Suite is an application that allows you to view Google Analytics statistics. All that could be seen by logging into Google Analytics under your account is now available from the application on the desktop.

This application was created using Adobe AIR. You can install it from here.

Just click Install Now under the screenshot. After installation, you need to start the application (an icon will appear on the desktop) and create a new profile where you will need to enter your account information in Google Analytics.

Only 4 fields:

* Profile Name (Name for the profile being created)
* Analytics account login (Google Analytics account name)
* Account password (Google Analytics account password)
* Description (Description of your profile. Optional)
