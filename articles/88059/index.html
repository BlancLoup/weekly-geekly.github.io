Full automatic, image library and free Windows Server!

Progress does not stand still and Parking.ru moves with it.
We have had dramatic changes in the Hyper-V facility mechanism, thanks to our Parking Cube technology, virtual machines are created in fully automatic mode .



For convenience, we have created a library of pre-configured images of operating systems that are connected when creating a virtual machine. The image can include both the operating system itself and various application software.

Choose one or another image depending on your tasks. Among the images are versions on Windows Server Web 2003 and Windows Web Server 2008 with Zend, PHP, FastCGI, etc. installed. The list of images will be constantly updated with new builds for different typical usage models.

You can also connect Windows Server 2003/2008 images of Web, Standart, Enterprise editions, and Linux / Unix fans can use Debian, Fedora 10, FreeBSD 6.4, Gentoo, OpenSUSE 11, Ubuntu 8.10, or connect your own image.

Another pleasant news is that now the cost of our Hyper-V tariffs includes the cost of a license for server versions of Microsoft Windows 2003/2008, i.e. OS you get for free !
This means that now you can install any version of Windows Server 2003/2008 up to Enterprise without overpaying for licenses.
