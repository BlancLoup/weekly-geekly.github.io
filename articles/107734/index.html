Evernote for Android: Meet version 2.0

We are very enthusiastic about the growth of the Android platform, the possibilities offered by the new OS and the many new excellent phones, tablet computers and netbooks.

But most of all, of course, we are excited about the version of Evernote 2.0 for Android , which is already available in the Android Market. This is the biggest leap in functionality that we have ever done in the same version on any of the platforms. As usual, Evernote for Android allows you to write notes, take snapshots, record audio and get easy access to existing data in Evernote, but now the app does all these things better, and here's why:

New interface

New home screen
You should notice a simpler and more convenient home screen that allows you to create new notes, gives quick access to existing notes and displays the synchronization status.


New search interface
We have placed the search icon at the top of each screen so that you can start your search from anywhere. The search screen allows you to add new search filters, view existing ones, access saved queries, and search for notes made near your current location. Just type your query in the text box and hit enter to start a keyword search. You will notice that during the search, we display the appropriate tags, notepads, saved queries, as well as the search history, so that the query is easy to repeat.

New views, new options
In the note view, you will notice a small tab at the bottom of the screen. Click on it to see other notes in the current search results. The list can be scrolled with a finger, and to quickly switch to a note, just click on it.

To view detailed information about a note, just click on its title. And through the menu, you can access the location data, open the original web page in the browser and switch to other viewing modes.

Simplified Sorting and Navigation
We have simplified navigation through notes, tags, and notebooks. Now you can group your notes by notepad, location or creation month and use the scroll bar to quickly switch between groups. Sorting notes occurs almost instantly and works even without connecting to the server. If you press and hold your finger on any note in the list, a menu of quick access to functions such as editing, deleting, receiving detailed information about the note, sending it by e-mail, viewing on the map, assigning notes to the notes will open.

We also added a convenient toolbar to the interface that allows you to easily select saved queries, tags and notepads.

Productivity increase

In Evernote 2.0, navigation, browsing and searching for notes is much faster. This was made possible by loading all the data about your notes (tags, names of notebooks, saved queries, note headers and thumbnails) during the first launch of the application. And thanks to the corresponding features of Android, we periodically start synchronization in the background so that you always have fresh data.

In addition, each time you create or view a note, it will be saved on your device for quick access to it, even without an Internet connection. This feature is available for owners of both free and premium accounts.

Please note that the initial synchronization may take some time, so we recommend putting the phone on charge and using a Wi-Fi connection.

Offline notebooks for premium users (what we were asked for most)

Our premium users will also have the opportunity to select notebooks, access to which is needed without an Internet connection (see the ‚ÄúOffline Notebooks‚Äù option in the application settings menu). Evernote will download all the notes contained in these notebooks, including any attached images and files, and save them to the SD card.



Easier note creation and support for multiple attachments

We have improved the process of creating a note, allowing you to add several images, PDF and audio files to it at once. In the mode of creating and editing notes, you can now use the buttons in the toolbar to add images and other files, add tags and select a notepad.

Widgets

Google search widget searches in Evernote

Evernote widget

Home Screen Labels

One of our favorite new features of the Android operating system are shortcuts. With Evernote 2.0, you can put a shortcut on your home screen to any note or search query. This is a great solution in cases where you want to quickly go to your to-do list or a frequently used selection of notes right from the Andorid home screen. You can create a shortcut through the corresponding menu item in the modes of viewing an individual note or a list of notes.

Other improvements
  • Audio recording in parallel with text entry: voice recording is now available directly on the new note creation screen, so you can type text and record audio in parallel.
  • Attachments and more: when viewing notes, you can open attachments (for example, PDF) and view handwritten notes created in Evernote for Windows.
  • Placement options: does your device run out of free memory? Evernote 2.0 allows you to place the application and all its data on the SD card.
  • Data import: using the ‚ÄúShare‚Äù menu in applications that support this feature, you can send content to Evernote and create a new note from it.
  • Buying a premium account: you can now subscribe to an Evernote premium subscription right from the application settings menu.
