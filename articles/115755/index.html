SiteStatus.ru - check the status of the site

A year ago, my best friend [ Dzuba ] wrote a site availability check service.
So he works all year. Does not bother anyone. Not monetized.


Often, when the site is down, you ask questions like:
- What's the matter?
- Maybe my ISP has a problem?
- Or on the hosting server?
- Maybe the site has recently been transferred, and DNS records have been cached?

Now it is easy to find out the truth in an accessible form!

Service SiteStatus.ru can:
- Identify problems in the configuration of DNS records that impede access to the site.
- Identify problems in the HTTP server configuration that impede access to the site.
The test results are displayed in a brief verbal form. If necessary, you can see more detailed technical information.

Service supported:
- Domains of the second, third and more levels, including distributed across multiple servers.
- Check HTTP availability for the specified address of the site page, and not for the main page.
- HTTP redirects with the function of quickly moving to the redirect address.

This year I used the service more than 20 times. I think it will be useful to you.
