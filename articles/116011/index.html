Mozilla Releases Firefox 4 Release Candidate for Android and Maemo

In addition to the final release of Firefox 4 for the desktop , Mozilla has released the release candidate Firefox 4 for Android and Maemo.

Since the release of the latest beta, developers have increased the speed of page scrolling, improved the performance of the Firefox Sync extension, and improved the interface in general.

Firefox 4 RC is available in the Android Market and for download for Maemo . If no significant problems are found, Mozilla will release the final release of mobile Firefox soon.
