Qt Charts 1.3.0 released

A new version of Qt Chart has been released, providing new charts (Polar and Boxplot) and some API improvements. The package, as usual, is available in the Customer portal .

Polar chart allows you to present data in a pie chart. Polar chart supports line, spline, area and scatter series and all axis types they support.

From this version, Qt Quick 2 is now supported, allowing you to embed graphics in applications created with Qt Quick Controls. However, note that Qt Charts still uses the QtGraphicsView framework even with Qt Quick 2. Therefore, chart performance is usually better with Qt Quick 1 than with Qt Quick 2.

Qt Charts 1.3.0 also contains several minor improvements and fixes that are listed in the README file. For an overview of the functionality of the Charts, visit the Charts product page .

The release of Charts 1.3.0 is available through the Customer portal . If you do not have a Qt commercial license, you can try Charts by downloading the 30-day free version .
