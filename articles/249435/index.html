How to download Microsoft Office 16 from the Microsoft website

The other day I told you how to get Windows 10 Preview through Windows Update for your Windows 7 / 8.1. The article caused unexpectedly stormy interest and more than a hundred comments, which led me to the idea that it might be interesting for the public to see the Microsoft Office 2016 Technical Preview. The other day, the January version became available on Connect. It is interesting that usually access to preliminary versions of Office is extremely limited, and now - without announcements and advertisements, anyone can download the assembly. Different languages ‚Äã‚Äãare available, including Russian. It is also possible to regularly upload new builds. There is, however, one ‚Äúbut‚Äù - the product comes in a Click2Run format. I described this in detail in the article ‚ÄúHow to download the latest Office from the Microsoft website without any App-V‚Äù . Let's see what we need in order to download the pre-assembly Office.

Attention: parallel installation to Office 2013 is not supported (this is probably the C2R version, but this is exactly what is stated in the description), you will need to perform an upgrade from 2013 to 16 or install it in a virtual machine.

If you're still here, the first thing we need is to register with Microsoft Connect. To do this, go to connect.microsoft.com using your Microsoft ID . If you have never been to Connect before, you will need to create a profile. I do not fill in any fields without giving extra information. It will be obligatory to indicate the country (Russia works perfectly) and the distribution format (it does not matter if you indicate that you are not going to subscribe to it).



Connect after registration will greet you with a virgin empty page:



Go to the Directory, and select the link "View Connect products currently not accepting feedback."




There we will see the cherished program Microsoft Office Office Consumer Technical Preview. We select Join, we agree with the agreement, we enter a name and that e-mail that was used for Microsoft ID. Voila, we have downloads!



Now we need to load the Office 16 Deployment Tool from the v1.06.1.zip file:



Extract the archive, there is the required bootloader: C: \ v1.06.1 \ Files \ O16 \ setup.exe
Further I recommend you to read my previous article ‚ÄúHow to download the latest Office from the Microsoft website without any App-V‚Äù . From this point on, all actions are equivalent to that, here I will briefly describe. I create configuration.xml with a list of Retail products in both languages ‚Äã‚Äã(attention, Professional Plus Retail is now available): 

<Configuration> <Add SourcePath="C:\OfficeC2R" OfficeClientEdition="32"> <Product ID="ProplusRetail"> <Language ID="en-us" /> <Language ID="ru-ru" /> </Product> <Product ID="VisioProRetail"> <Language ID="en-us" /> <Language ID="ru-ru" /> </Product> <Product ID="ProjectProRetail"> <Language ID="en-us" /> <Language ID="ru-ru" /> </Product> </Add> </Configuration>

I execute the command setup.exe / download Configuration.xml and download the package. At the time of this writing, version 16.0.3629.1008 is available, two days ago the version was 16.0.3629.1006, and about three weeks ago I downloaded 16.0.3327.1020:



Now it remains to install it by running the command setup.exe / configure Configuration.xml



In the names of icons and titles, the product still calls itself Office 2013 (in the Start Menu, on the right, I rendered the icons):



At the same time, if you launch any product, you will see from the inside that its version of Office 16:


Office will need to be activated the way you activate the Retail version of Office 2013. If you have an Office 365 subscription, then when preparing the Configuration.xml file, you do not need to specify ProplusRetail, but O365ProPlusRetail, otherwise if you try to activate with your subscription, you will be prompted to download the appropriate version:



I installed the O365ProPlusRetail version and successfully activated it:
