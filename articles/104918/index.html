Windows Phone 7 does not support tethering



Only today on the portal ferra.ru there was news that the new mobile OS from Microsoft Windows Phone 7 will support tethering, that is, devices with this OS can be used as a hot spot to connect other devices to the Internet via the phone via the cellular network. However, Ferra wrote the news on the original source of September 23. In fact, the original source still claimed that there would be a tethering in Windows Phone 7, and this OS would thus compete for Android and Palm. But as it turned out, everything can change in a day. Today, CNET has received official information from Microsoft that there will be no tethering in this mobile OS.

A Microsoft source also declined to say whether support for tethering is planned in future versions of this mobile operating system. In addition, mobile devices based on Windows Phone 7 for the CDMA mobile communications protocol will not appear until next year.

Generally speaking, not everyone needs tethering, of course, but a large number of people use it. For example, I myself connect to the Network via CDMA communicator HTC Mogul with WinMo 6.5 during trips to the dacha - however, I had to install additional software called WiFi Router, which can connect you to the Internet via cellular in many ways.

But if the new OS has the tethering function disabled, then mobile devices with this mobile system will obviously lose some of the potential users. Due to the fact that the tethering function in the iPhone was made paid, somehow a big scandal arose - at the same time a free application was released, sort of like a flashlight, which made it possible to use the tethering function completely free. The application was soon removed, so I still had to pay. AT & T, for example, added the tethering function, it only takes $ 20 a month for it - the function was added just after the elimination of the unlimited packages of this mobile operator.

It is worth noting that the same OS supports multitasking, but it is not as multitasking (almost pun intended, yeah), like other types of OS. In general, the product seems to be a little damp, and nothing can be done about it. Of course, it is likely that in the future there will be new versions of the OS with various additions, but the first version is not enough.
