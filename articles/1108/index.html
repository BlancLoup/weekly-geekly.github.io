Google talk talk with vista

Google has made the necessary changes to its instant messaging client, making it compatible with Windows Vista, which is expected to be released soon.

Major changes were made to the security system in Google Talk. In earlier versions, by default, the program was installed for all users of the computer. The new version installs the default client only for the current user, although this can be changed during the installation process.
