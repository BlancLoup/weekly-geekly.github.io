The final schedule of Community DevCamp (Moscow, December 12)

As I already wrote , we have prepared for you Community DevCamp - an event for developers from developers. Finally, we decided on the final grid of reports, if you planned to approach a certain report - see the updated schedule.

The event will take place this week, on Saturday. The main speakers are recognized community experts who will talk about how they see, use or plan to use the latest innovations for developers on .NET - .NET Native, Roslyn, cross-platform development on ASP.NET, Docker containers, Azure Service Fabric F #, C # 7.
