Return support for Windows XP in the new assembly Vivaldi 1.0.334.3

Hello!

Today there are few changes, mostly corrections of errors and minor flaws. However, some small changes should be mentioned separately.

In the basket of recently closed tabs, a cleanup option has appeared. In fact, this option was in the earliest versions of Vivaldi, but was then removed for various reasons. Now we have returned this popular feature back.

Another small innovation is included in the browser settings. Open Settings-> Web Pages . A new option is called "Follow the links using Tab". By enabling this option, you can navigate to the newly opened web pages through the embedded links using the Tab key. In general - a trifle, but sometimes very convenient.

For Windows XP users, the good news is that we made the appropriate corrections to the code so that you can use the Vivaldi browser again on this good old operating system.

We also fixed a small bug with incorrect display of tab headers.
