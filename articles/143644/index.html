Idea


Search the browser means not only on the open page of the site, but throughout the site.

How should this work?


  1. The person presses Ctrl + F and enters the search word;
  2. The browser passes the word to the site;
  3. The site uses internal mechanisms * searches for what the user needs and sends the xml file generated according to certain rules to the browser;
  4. The data are displayed in the usual form for a person.

* instead of internal mechanisms, a search for Google, Yandex or another search engine can be used.

Small explanation


I think everyone paid attention to the fact that almost every site requires a search, and each resource copes with this task on its own.
Developers puzzled over the problems of search mechanisms, display, and much more.

I propose to shift this unbearable burden from the shoulders of website developers to the shoulders of browser developers and search giants.
