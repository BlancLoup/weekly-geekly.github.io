Upsource 3.5 is released with numerous changes

Hi, Habr! Last week we released a new version of our tool for code review and navigation through the repositories, Upsource . Let's see what changes we have included in this release.

Updated project page


We rethought the product home page and made it more convenient for those companies that have a dozen projects, and for those who have hundreds of projects.

In addition, the homepage as a whole has become more informative - now you can immediately see not only the projects added to your bookmarks, but also the projects with the most activity, recently created projects and projects in which you participate.



User profiles


Upsource 3.5 allows your team to know more about each other. The name of each user is now a link to his profile, in which you can see in which projects this colleague is involved, in which code review participates and what changes he makes.

Your own profile gives you quick access to similar records of your activity.



Inter-project search


We redesigned the search field and added it to the main page. Now you can search for files in all projects and even find what has already been deleted. Upsource also helps you find other users and projects. In addition, the Upsource search is now context sensitive. This means that if you are viewing a project in a particular revision, the search will be performed in the context of that revision.



More flexible process automation settings


In the previous release, we added the ability to partially automate the code review process. This time we made the settings more flexible by adding the ability to add multiple templates and exceptions. For convenience, we also updated the UI of this functionality.



New analytics


Upsource 3.5 uncovers another side of your projects. Now you can visually see which social connections are formed in your team with the help of code review - who views whose changes and which members of your team never looked at each other‚Äôs changes.



And much more


As always, we have added a lot of new functionality in integration with the IDE. In addition, the integrated IntelliJ IDEA has been updated in Upsource 3.5, which means that all new code inspections will be available to you. But this is not all. PHP developers will be happy to know that Upsource now supports Composer and external dependencies.

You can learn about the rest of the Upsource features on the product page or by signing up for a free webinar .
