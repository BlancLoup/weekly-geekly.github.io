MegaphoneModem + in Mandriva Linux

Repeatedly I saw and read how people set up modems that are visible as a USB flash drive + modem in Linux, saw scary scripts, a bunch of packages, someone even reassembled the kernel. But here I became the happy forced owner of the Huawei e160g modem EDGE, which includes the Megaphone Modem Plus kit.

Configured on Mandriva Linux 2009.0, it will be different in other distributions, as there is no drakconnect everywhere.

I thought it would be scary, in fact it turned out to be as simple as possible. So, the instruction (copy-paste from a personal blog ):
  1. preparation for setup - you should install 2 ppp and comgt packages in advance (in the superuser mode # urpmi comgt ppp)
  2. we insert the modem itself into the USB port, right-click on the network applet (2 monitors (blue squares) with a green check mark or a red cross next to the watch), select "Configure network" and rush (after each item, click Next or Finish):
    • select GRPS / EDGE / 3G in the list
    • In the list, select our modem (it will be there only if you have not installed many more of these) with the name approximately ppp0: HUAWEI Technology HUAWEI modem
    • enter your PIN code
    • choose your network (if not, go on)
    • Megafon did not have the list of providers, so select " Not in the list - edit manually "
    • we enter: the name of the access point is " internet " (without quotes), the login with the password is unclear what, I did not enter anything at all
    • put the necessary checkboxes (at your discretion)
    • choose what we want to connect right now (option " Yes ")
    • we wait
    • enjoy the internet (not a dedicated line, of course, but you can still live)


UPD (2009-07-02) : one o'clock (MSK) - I put a 2.2 GB movie on download, the speed fluctuates around 20 KB / s (+ -4 KB / s), the program promises to download the movie in about 1 day and 5 hours . Traffic is not taken into account anywhere (money is not withdrawn from the account, 100 free MBs also do not decrease). In general, it is not bad if there are no underwater stones - for giving / traveling / lack of dedicated line - the most it.
