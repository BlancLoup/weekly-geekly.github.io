For information users Opera 10.51


With this message, on behalf of the company, I want to apologize to all users of Opera 10.51 for Windows, who have encountered various problems after installing this version, released two days ago. The development of a complex software product, in which hundreds of people are involved, is inevitably connected with the human factor - from time to time an unnoticed error of one of the employees can turn into a rather unpleasant surprise. It happened this time too: by mistake, as the final release of Opera 10.51, an internal test build was posted on the server, which was not intended for this purpose at all. As a result, many users after upgrading or installing a clean version of this version are faced with problems that are missing in the official version prepared by the developers of Opera Software.

In more detail, I can say that the almost identical name of the installation file (the difference was only in one digit) of the actual and test builds became the culprit. Yesterday morning, Opera 10.51 installation files were replaced on the company‚Äôs server, so everyone who updated after 11:00 AM Moscow time on March 23 can sleep peacefully - you have the version you need. Those who have upgraded to the specified time are recommended to reinstall by downloading the correct version from Opera Software servers. Unfortunately, you will need to manually reinstall the browser.

How to recognize the wrong version:
In the erroneous version, the browser identification string contains the word Yandex (you can see this parameter through the Help menu -> About Opera).
The error occurred with the International version, the Russian version was originally correct.
