Say2go



The technology of instant messaging in recent years has not changed significantly. AOL IM today is not very different from its version in '97. Thanks to the discovery of VoIP and the increase in the number of webcams, Skype appeared, which really changed something in the communications and IM market. But two things remain the same: voice communication is conducted in real time, and text - in near-real, according to TechCrunch.

Say2GO is a new messaging system in near real time, which can combine both types of communication. This technology is more like voice mail than IM. The product, which is currently very limited, allows the user to send and receive voice messages that are sent in the original and text (the program recognizes the voice) types. The client uses the Microsoft Speech API for recognition, and therefore it is available only to users of the Windows operating system. In addition, you will have to install some additional libraries; without them, the program becomes just ordinary IM with a neat and clean interface.

MSN Messenger has the same practical features, called MS Voice Clips, which appeared in 2005. They allow the user to send voice messages to friends for up to 15 seconds. Say2GO, on the other hand, makes it possible to repeat the recording again, cancel it or put it in a queue in order to send a message later.

It seems that Microsoft is seriously interested in this technology, because many experts note that voice recognition is up to the mark - the future of mobile platforms.
