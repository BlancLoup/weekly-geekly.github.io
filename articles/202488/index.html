Good day, dear sokhabrovtsy!

As i understand it

How to propose to limit

There is a registry of banned sites of ROSKOMNADZOR and a list of extremist materials of the Justice Ministry , which it is proposed to use when restricting content.

Problems

It is impossible to get a complete list of resources of the unified registry of ROSCOMNADZOR (not counting not always up-to-date and incomprehensible to what extent a legal resource ). You can only check a specific site for "unsuitability" by typing it into the query string of the registry site. Since filtering in my educational institution is carried out using the squid + dansguardian bundle, then:
- I cannot add all sites to the blacklist, because this list cannot be obtained
- I also cannot fully use the list of prohibited phrases for filtering, since there is no officially approved list
- In addition, by https my filter also does not know how to work, so I have to ban some resources (social networks, for example) over ip, using IPFW

After writing the article, I found a way to block https-urls with "hands on the battered" (recall that dansguardian did not block them). It is necessary to specify a proxy server in users' browsers (even if it is transparent) and port 8080 . I did this with a GPO.
