TiddlyWiki or your little wikipedia ...

‚ÄúTiddlyWiki is a wiki engine and a wiki concept that the entire wiki site is a single HTML page, the interactivity of which is provided by scripts.‚Äù

So this real wikipedia says about this wonderful flight of thought of programmers. And what is it really? This is what many, if not all, dream of - a local, small, compact and transportable wiki knowledge base, independent of the WEB server and server languages ‚Äã‚Äãwith the ability to synchronize with a distant, distant server. Personally for myself, I discovered it just yesterday, when I was too lazy to dig into web programming and create a website on the Web.



First, let's take a look at the capabilities of the engine:

* Full AJAX, so to speak, or rather its client part - everything works without a single reboot
* The entire engine fits in a single HTML file.
* You can create, edit and delete any records with some kind of wiki-like markup that will be saved in the same file
* Plugin support
* Ability to auto-update the main engine code
* Search by posts
* Assignment of labels for each entry
* Fully customizable interface. by custom CSS and standard HTML template changes
* Ability to synchronize with other files and files on the server
* Import from other files and servers
* ...

In general, it can do a lot, but most importantly, it is comfortable enough and perhaps worthy of attention. The only drawback I encountered is the lack of a Russian translation. However, this problem is solved simply by writing the appropriate plug-in, which I did - I added my slightly clumsy translation, which can be downloaded here .

Now how to get and how to start using TiddlyWiki. To do this, go to the site www.tiddlywiki.com and download the latest version of the engine, now it is 2.4.1. In principle, this is all - it remains only to open the file itself, specify your name, change the headers and start adding your records. But I advise you to do a little differently. There is a site tiddlyspot.com which is nothing more than a free hosting for TiddlyWiki.

By registering on this site, you have at your disposal a subdomain and a TiddlyWiki page ready for use. It remains only to save it on the local disk, click on the ‚Äúdownload‚Äù button on the right and open the saved page in the browser, enter your password on tiddlyspot.com (hidden TspotOptions entry), install the translation of the main part of the engine as desired (how to install translations and other plugins described here ) and that's it! Now, after any edits, you should just click on the ‚Äúupload‚Äù button and be happy =))
