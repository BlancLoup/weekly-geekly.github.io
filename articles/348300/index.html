Auto dial: a good way to enhance customer base

In a highly competitive customer base is one of the core values ‚Äã‚Äãof the company. The auto redial feature available in Zeon IP PBX allows you to activate customers with minimal cost and, with the right approach, make additional sales.


When collecting customer contacts, retailers usually ask for a phone number, as well as an email address. It is very easy to send news to all customers, but, as they say, ‚Äúwho reads them?‚Äù. Make dialing more efficient, but at first glance - much more difficult: You need to go through the customer base, select the necessary contacts, dial their numbers. However, it is possible without it - if you use the function of automatic dialing. And today we will understand how this works.

When to use auto redial


In order to use automatic dialing, in general, you do not need a special occasion. This may be a one-time action, a sale or some other event. Moreover, a personal offer is ideal for a call, which can be ‚Äúone for all‚Äù or created individually within the CRM system.

You can also call customers on the basis of the completion of the order or delivery. For example, this can be done the next day following the results of the previous day. But in any case, dialing manually is associated immediately with a whole set of problems. If the operator himself dials numbers, the process is not effective, labor productivity decreases, and the quality assessment turns out to be problematic.

Auto Dial with IP PBX


Automatic dialing is much easier if you are working with an IP PBX, such as Zeon. When installing such a solution, operators and other employees are initially connected to a programmable PBX, and their SIP phones allow not only to dial a number, but also to connect subscribers using a predetermined algorithm.

Let's look at how auto dialing works with Zeon IP PBX.


To start the process, you need to go to the IP-PBX web interface and create a 'Queue' for auto-dialing. Connection of operators occurs directly in the browser, and you can always see which of them is available for work.

When the redial queue has already been created, you can add a "Customer List" to it. It should be in CSV format with the name, phone and, possibly, notes. This list can be exported from CRM, provided by a partner or transferred to the delivery department. The essence of the note is that the operator sees it on the screen of his computer when he picks up the phone to talk to the customer.

The 'autodial task' can be organized in various ways. You can choose which of the previously downloaded call lists to use. Which operators to use, as well as determine parameters such as the number of lines involved. You can stop the task at any time, and then the calls will stop until it is convenient to continue the call. For example, you can stop a task at 17 o'clock to continue at 10 am.

When the operator picks up the phone, a customer card will be shown at his workplace, which contains the phone number, name and note. If this is not about a ‚Äúcold‚Äù base, but about clients with whom you or your partners have already worked, the ‚Äúnote‚Äù field can be a good help for additional cross-sales.

After pressing the ‚Äúsave and go to the next call‚Äù button, the system automatically dials the next client and immediately connects it to the released operator.

Use auto redial for your business needs.


The auto redial feature is available to all Zeon IP PBX users. If you use the integration with the CRM system, employees will also be able to make notes on the CRM card, recording the results of dialing. The advantage of this method is that you can call up customer bases without attracting outside contractors, but using the free resources of your employees.

In sum with the available telephony from iptelefon.su, auto redial allows you to expand your customer base using new data and contacts, as well as activate passive clients. Using the open CSV format makes it possible to start calls on any list of numbers - all entries from the list fall into the Contacts section and can later be used for a new call.
