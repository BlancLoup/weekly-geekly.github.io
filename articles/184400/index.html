Notepad ++ has a spelling highlighting error



In a wonderful editor finally appeared ( Upd. Since May 5) highlighting errors in the spell checker! For me, this is a very welcome innovation, which was very much waiting and hoping. At one time I even wanted to jump to other editors because of this problem. In Notepad ++, it was possible to set up a spell check up to this point, but there were no highlighting of words with errors, which diminished the benefit of spelling checking.

Further I will tell how to configure check of Russian and English.

1. Install the version not lower than 6.3.3: http://notepad-plus-plus.org/download/ .



2. Spelling of the English language is now already enabled by default.



3. Now set up the Russian language. There are two ways to do this: through Hunspell or through Aspell. Let's go the second way.

4. Go to aspell.net/win32 and download the full version of Aspell and the Russian dictionary (download and install English at the same time):




5. Open Notepad ++ and go to the settings of the new DSpellCheck plugin:



6. There we choose the type of Aspell spell checker:



7. Select Multiple Languages :



8. And there we select all the dictionaries that we need to check:



9. Everything! Now we have a complete spell checker in Notepad ++:



And when you hover over the underlined word, a red button will appear below, when clicked, the correction options are displayed. You can also ignore words or add to the dictionary. In general, everything is great!

Upd. They say that it is necessary to install the English dictionary all the same when installing ASpell, otherwise there are a lot of problems because of this.
