How the number of installations depends on the weight of the application


Fiksu calculated statistics on conversions to installs depending on the size of the applications. To begin with, it is important to emphasize that the statistics take into account only gaming applications and only motivated advertising - Fiksu themselves placed this information in a footnote at the end of the report. Without an understanding of this fact, a false picture may emerge. So, this is data on the effectiveness of conversions to installations when using motivated ads.

In general, the statistics do not show anything new - the more the application weighs, the fewer conversions into installations. The highest efficiency in applications weighing 0-20mb is 87.94%. But even for applications larger than 100mb, the conversion is not much different - 76.31%. It is significant that when using motivated advertising, the size of the application is not such an important factor. Most likely, the more actively the technologies develop, the less this difference will be felt - if the user can quickly and painlessly install both small-sized applications and applications of 100+ mb, the difference in the number of installations will be smoothed out.
