Tooltips in Power BI - a new feature for Desktop

In the middle of March, another Power BI Desktop update was released. Microsoft added a couple of interesting features, including the ability for users to create tooltips themselves - Tooltips.

In this post I will show a simple example of working with this functionality.

Activate the tooltips feature in Power BI Desktop


The tooltip function is in the preview stage and needs to be turned on / off manually.

This means that this feature is not yet fully integrated into the Power BI platform, hints will not work when viewing reports on mobile applications, when publishing on the Internet, just pop-up hints are not supported for custom visual elements.
