Yandex.Navigator learned to work offline



Today, a new version of the Yandex.Navigator application has been released, the main and long-awaited difference is the addition of the ability to load a map of a selected city into the cache and use it without access to the Internet.

Downloading a map is almost the same as in the Yandex.Maps application, you select the region you need and the download of the given map cache starts in a compressed form, then unpacking occurs (because of this, the maps can take up more space on the device than shown when downloading ) .

In addition, with this update, Yandex made vector maps for St. Petersburg and Ukraine (at this time, vector maps are already used for the Moscow region) , in addition, vector maps for Moscow and Moscow region have been updated.

Than vector maps are better than usual?
Perhaps how briefly this can be explained: the cache of maps for Moscow and the Moscow region in vector space takes only 261.1 MB against the usual Yandex maps 2.9 GB (Moscow 801 MB + MO 2.1 GB)
