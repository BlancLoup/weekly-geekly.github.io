Published code of the game Commander Keen



The source code for the Commander Keen game , one of the first video games for the IBM PC / XT personal computer with MS-DOS operating system, is published on Github. The series was notable for its innovative use of graphics, including the use of the advanced EGA format and the first use in the world of image shifting adaptive tile updating , which John Carmack invented.

This is one of the first games id Software and John Carmack. The original version was released in December 1990.

A version of Commander Keen: Keen Dreams was released on Github, which was released as early as 1991 and was being finalized until 1993, respectively, had different version numbers for different platforms and distribution channels.

The code is published under the GNU GPLv2 + license. By the way, this game in 2013 was ported to Android.

True, the free license does not apply to game data (apparently, graphics and music), which must be separately licensed.

The code is written in the Borland C ++ 2.0 environment, but should compile normally in versions up to 3.1.
