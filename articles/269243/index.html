Microsoft fixed bugs in Windows drivers

Microsoft has released a set of updates for Windows, which close a number of bugs in the drivers and system files of the OS. Update KB3078405 eliminates bugs in the Tunnel.sys driver on Windows 8.1, which led to sudden BSOD errors with codes 0x0000004A and 0x0000009F. The computer could fall into the blue screen of death in various situations, including when using the connected removable USB-drive and when switching between different power modes.



Update KB3095649 disables the Touch Hardware Quality Assurance (THQA) certification for Win32k.sys on Windows 8.1. Another update KB3095701 eliminates the bugs in the Tpm.sys driver, which did not allow the system running Windows Server 2012 R2 to recognize some TPM 2.0 devices.
