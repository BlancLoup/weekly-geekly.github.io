Photoshop vs. Fireworks: compression

Surprisingly, based on the results of my recent research, there are quite a few Fireworks users. Personally, I never used it. Usually, I use Photoshop for daily work. But, did you know that Fireworks is better than compressing images in Photoshop? I'm not a programmer, I can't explain why Fireworks compresses better. But I can prove it by showing a series of experiments that I conducted.

PNG compression


For the test, I used an uncompressed image with transparency. Export Photoshop image to PNG-24 with transparency and get a file about 352 kb in size. Using the same picture I exported it to PNG-32 using Fireworks. File size is about 332 kb. Visually, both images look the same (at least for me). However, the file size of the exported Fireworks is about 20 kb smaller.



Gif compression


Then we export the image to GIF, 256 colors, adaptive, with no dither.
Results: Fireworks = 88kb, Photoshop = 92kB. That is 4 kb less. In addition, the images are not the same. I liked more the image exported by Fireworks because it has more details.



JPG compression


Now I will try to compress JPG. I exported the image to Photoshop with 60% quality. File size was about 80 kb. Then I exported the same image to Fireworks with 80% quality and received a file of 72 KB in size. Exported images look the same, but Fireworks is 8KB ahead of Photoshop.



Note: Fireworks and Photoshop have different compression technologies. 80% compression in Fireworks is similar to 60% compression in Photoshop.

What is special?


Does it matter ? These are just a few kilobytes ... These kilobytes have a big impact on your traffic and download speed.

Let's give an example. With Fireworks, I can reduce the size of the logo on the site by 20kb. With 16,000 visits per day, we have the following traffic. 20 Kb x 16 000 = 320, 000 kb (we forget that in megabyte 1024 kb - comment of the translator). It is as much as 320 megabytes per day! And here I am talking only about the logo. Could you imagine how much traffic I can save if I compress all the graphics on the site using Fireworks? Most importantly, I can make my website load faster!



Conclusion


If you have Photoshop and Fireworks, you can try it yourself. You will be surprised at the results. From now on, I'm going to use Fireworks to export web graphics, especially graphic templates.
