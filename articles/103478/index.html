Google Wave give the people in finished form

The Google Wave development team decided to launch an open source project based on Google Wave, with the exception of the 200k lines of code that are already open (see waveprotocol .). The new project is a more clearly arranged solution, in fact, a ready-made system.

The project is called Wave in a Box and includes the following:

* a package of applications, including server and clients with support for structured real-time communications modeled on the Google Wave system;
* fast and full-featured wave-panel for the web client with full support for trade chats;
* stable data storage and search implementation for the server (done on third-party code for MongoDB implementation)
* enhanced client-server protocols;
* API support for gadgets, robots and data;
* support for importing wave data from wave.google.com ;
* the ability to combine different Wave in a Box instances, with some additional configuration.

The project will not have the full functionality of Google Wave, but third-party developers and companies are able to easily raise their own wave-servers and keep the "waves" on their own equipment.

The Google Wave team believes that this is a great project from a technical point of view. They received a tremendous response from the technical community and seriously hoped to create a new generation of communication and collaboration tools. They are proud to have inspired many independent developers to create a huge number of related projects.

Developers from Google say that now the future of Google Wave already depends on the Wave community. The technical details of the plan will be discussed in more detail at the Wave Protocol Forum . In Russian - in the groups.google.com/group/google-wave-using group.
