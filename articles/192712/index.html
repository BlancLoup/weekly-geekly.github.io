Another version of the top panel for Habr + button on the Google Chrome panel



Hello, Habr!

In recent times, posts about finishing or modifying Habr often appear. I saw enough and decided to do something of my own. The options were good, but I wanted to make the appearance as close as possible to Habravsky + slightly load the interface + button.

As a result, DenimHabr 0.1 appeared. If interested, welcome.

Based on 2 extensions, this is: HabraRed Designer 0.1.5.1 from KirAmp ( many thanks!) And HabrInfo from some an9227 (it was not possible to find the author, there is no email address, there is no author on GitHub either).

Socket

The panel repeats HabraDesign almost completely though there are several cosmetic edits.
  • I decided to remove the most important links from the dropout, I left there only the exit button
  • Added button with "blob", which leads to habrahabr.ru/feed
  • Visual changes (buttons, fonts, colors)


Button and popup window

The button itself remained untouched, but I shoveled the popup pretty hard. I did not like that too much text information, did not like the round avatar, well, in principle, the popup seemed overloaded.

Eventually:


  • Replaced the withdrawal of autarky, removed the border it.p. Left only neat tile.
  • Removed ALL text info from footer
  • Added links to main pages in the form of ikonochek
  • Replaced the output of karma and rating in the Habra-style


Instead of the total:

The code is still quite a curve, 2 extensions live separately and much can be cut or optimized. This topic was created in order to understand whether it is worth further developing this initiative for the people of Habr or limit themselves to their own needs? If the first, then I would like to hear the wishes / comments on the appearance and functionality.
