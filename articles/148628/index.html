The Ouya game console becomes more functional, with the addition of support for the OnLive service.



Probably, many have already heard about the Ouya game console, which has the size of a Rubik's cube, and is based on the Android OS. Not so long ago, Habr√© published news that this project raised $ 5.5 million on Kickstarter, which is the exception rather than the rule for a hardwired project. The developers got even more than planned, now they are going to add new features to Ouya.

So, previously it was assumed that this game console in functionality will not exceed the tablet or smartphone on Android OS. Of course, it is not very correct to compare a smartphone and a game console, but nonetheless. Due to the good configuration, the device can provide comfortable leisure for fans of "mobile" games. Well, for example, Shadowgun and Angry Birds.

Now this console will offer gamers hundreds of other games available through OnLive. At the moment, this service offers games from more than 80 game studios. In addition, the game started on Ouya can be continued on a regular PC. As you know, the service offers 30 free minutes for the game, to be sure that the chosen game suits you completely. In the future, if you want to continue to play, the right to play will have to buy.

In general, the Ouya game console now offers not only mini-games, mobile games (including various emulators of other game consoles), but also full-fledged computer games. It may well be that this functionality of this gaming console will attract many gamers to it, who will also like the compactness of this console.

By the way, for Ouya now released and a great game controller (pictured in the announcement). Its cost will be about $ 30. The controller + console kit costs $ 99.
