Preview of the first updates of the Microsoft Edge web platform



The other day we brought our Microsoft Edge Dev developer portal out of beta status, and now we are redirecting all traffic from status.modern.ie to a new page with platform status , which we significantly accelerated and visually cleaned up.

Today we are happy to tell you the first details about the upcoming wave of updating the functionality of the Microsoft Edge web platform. Let's start with a story about what new features are currently in development for EdgeHTML.
