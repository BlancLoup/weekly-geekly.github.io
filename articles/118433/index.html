Competition for application developers with guaranteed prizes-smartphones

Not so long ago, BlackBerry conducted a campaign on which developers of the PlayBook application received this tablet as a gift. Now RIM has announced the start of the competition for developers of applications for smartphones and tablet PlayBook. The total prize fund is $ 3 million. They promise guaranteed free devices, which is a little lower.


The competition takes place in three categories:
  • Best Flash / AIR Mobile App for BlackBerry PlayBook
  • The most innovative BlackBerry WebWorks app for the PlayBook and BlackBerry 6
  • The most "addictive" social application using the BBM Social Platform

Most recently, on April 28, acceptance of applications in the first nomination (Flash / AIR Mobile) began. Accepting applications will last until August 12. Other nominations start later (the second - May 9, about the third is still unknown).

Winners will be determined at the BlackBerry Developer Conference (DEVCON), which will be held October 18-20, 2011 in San Francisco.

In addition to pretty good prizes for the first and third places, there are also guaranteed prizes for the first 100 applications in each nomination: a BlackBerry smartphone and one week of application rotation in the AppWorld carousel!

Applying for a competition consists of two steps:
  1. Register as a developer in Vendor Portal , if the account is not there yet
  2. Filling out an application

It should be noted that the application form requires the indication of the application ID, which can be viewed by clicking the product edit button in Manage Products. Therefore, if you are going to send a new application to the competition, you must first create a product (without release, that is, without the application file itself) in Vendor Portal, and then fill out an application. You can also send already existing applications to the competition, you can also make several releases of the same application (the last one will be considered). Also, if your application is paid and requires you to enter a login-password, you will need to prepare 10 valid pairs of login-passwords for testing it by the judges.
