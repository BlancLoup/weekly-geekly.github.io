3CX Phone System Pro Edition has been released.

The advanced functionality of the 3CX Phone System Pro Edition puts the program on a par with traditional PBXs. Among the most important features of the new edition are the following:

  • Advanced call center functions (monitoring of missed calls, convenient control panel, where you can see in real time the number of people in the queue and call statistics, etc.);
  • Automatic conversion of the phone number into the name of the subscriber using integration with Microsoft Exchange;
  • Ability to track the status of presence in several offices, as well as detailed reporting of calls;




Caller ID with Microsoft Exchange Integration


3CX Phone System Pro has Microsoft Exchange Server integration: contact requests are made directly through the Exchange Server using the 3CX phonebook, allowing employees to always be aware of who is calling them. The program uses only one database: the phone book 3CX has automatic synchronization with the Microsoft Exchange server - this helps save a lot of time.

Attendance tracking


3CX Phone System Pro will allow users to track the presence of their colleagues in other offices using 3CX Phone clients for Windows, Android and iPhone. With a few mouse clicks, you can make calls, create conferences, send instant messages, leave voice mail and transfer calls to colleagues in other offices.

Extended reporting


3CX Phone System Pro has more powerful reporting tools by providing detailed data. The reports display key information about queues, customer waiting times, statistics on the number of calls that have been called, the distribution of calls, missed calls and other indicators that allow you to adjust the work.
