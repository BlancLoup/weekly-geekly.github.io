Dedicated Internet backbone can connect BRICS countries, Africa and the USA



True, this will happen only if a positive decision is made on this project. The cost of the same project for laying a 34,000-kilometer underwater Internet highway is, at the most approximate estimate, one and a half billion dollars. A dedicated highway, in the case of this project, should connect the BRICS countries (Brazil, Russia, India, China, South Africa) with the United States.

The project investors include several large companies, including Google. The project is being developed by the i3 Africa group, which performed well in developing and implementing similar projects on the African continent. This project is one of the largest in its field: as mentioned above, the length of the highway will be 34 thousand kilometers.

If the project is approved, then the main part of the project will be completed by the end of 2014. In order to implement the project, its authors and initiators need to agree with all investors. It is worth recalling that the first information about this project appeared last year. Since then, plans have changed a bit.
