Bitmessage - program for P2P exchange of encrypted messages

Most recently, a beta version of the new program for P2P exchange of encrypted messages between two users or a group of users. The release of the beta version of Bitmessage took place on March 21st. It uses strong cryptography that reliably protects subscribers from listening at the level of the Internet service provider (SORM) or on the server, as in the case of Skype and others who pass traffic through themselves.



The further description with source codes under a cat.

The cryptography system practically exactly copies the scheme used in the P2P system Bitcoin. More details can be found in the technical description (pdf, in English).

The program is still fresh, and developers are turning to the expert community with a request to conduct an independent audit of the source codes, as is usually required for cryptographic software.

The source code of the utility is also available ( code on github ).
