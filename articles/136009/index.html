StopSocial - extension to block social content (Opera, Chrome) [UPD]

Do you use the buttons "Like", "Share"? And social networks in general? Many habra users will probably answer "no."

I use social buttons and widgets extremely rarely. If I want to share with someone a link, it will not be difficult for me to open the necessary site in the next tab and copy the link there. Nevertheless, recently more and more sites began to use 5-6 "Share" buttons from different social networks. The ubiquitous Share and Follow with the obligatory indication of the number of copies is not only annoying, but in some cases also slows down the page loading. In addition, very often the colorful buttons of various shapes and sizes do not fit into the design of the site on which they are located. Not every site uses social buttons, adapted to the design.

In the catalogs of extensions of popular browsers there are only a couple of extensions that have the ability to block social content - this is AntiSocial for Chrome and Adblock with the function of blocking Facebook buttons. AntiSocial did not work because of the lack of support for Russian-language social networks (and I was looking for an extension for Opera), and with Adblock it‚Äôs clear that Russian sites are not the only ones that use Facebook.

So a very simple StopSocial extension appeared , supporting the most popular social services, such as VKontakte, Facebook, Google +1, Mail.ru, etc. The full list can be found on the page with the extension.

Technical information


Uses tools that are typical for extensions of this type: Opera URL Filter API and CSS selectors that can be disabled for a specific site if something extra is blocked.
