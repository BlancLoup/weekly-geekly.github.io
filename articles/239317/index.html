Windows 95 on Samsung Gear Live smartwatch



Preview image is not a photoshop, despite the weirdness of this image. The fact is that one of the users of Reddit was able to launch the Windows 95 operating system (on Habr√©, there are probably users who have never seen or worked with this OS) on the modern Samsung Gear Live smart watch.

Of course, the functionality of Windows 95 on these watches is limited, since no one especially aimed at adjusting the OS to hardware and device capabilities. Nevertheless, the fact of launching the operating system, which is already about 20 years old, is very interesting on modern watches.

The launch was made possible thanks to the fact that the watch itself runs on the Android Wear operating system, and Android itself allows you to run the ADOSBOX emulator. It was in the ADOSBOX environment that Windows was launched.
