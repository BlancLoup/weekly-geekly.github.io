[Peter, Announcement] Meeting JUG.ru with Dmitry Alexandrov - Nashorn: a lightweight JavaScript engine on the JVM



On Wednesday, September 14, at 20:00, a meeting of JUG.ru with Dmitry Alexandrov , an expert on Java EE, Spring and JavaScript, will take place in the office of SEMrush. Theme of the meeting: Nashorn is a lightweight JavaScript engine on the JVM .

In this session, we will introduce the Nashorn project - a lightweight JavaScript engine on the JVM. We will go from simple to complex: we will learn how and why Java and JavaScript can coexist. Let's see how to invoke javascript from java and vice versa. Let's learn how to work in the jjs console, or how to write (and, most importantly, debug) OS scripts that are independent of the OS if there is a JDK. Let's see how you can use JavaScript libraries in Java projects. Consider how you can give dynamism to our programs using JS scripting.

Further on with specific examples, let's move on to the complex - let's take a look at the avatar.js project for writing web applications in the node.js style. Let's see how avatar.js integrates with Java application servers. For example, consider how a stack of Java + EE + JPA + Nashorn + Avatar.js + AngularJS technologies gets on.


About speaker


Dmitry Alexandrov is a lead programmer / architect at T-Systems, part-time co-leader of the Bulgarian Java User Group. In addition to the work reads lectures at conferences. Conference organizer jPrime - the largest Java conference in the Balkans.
