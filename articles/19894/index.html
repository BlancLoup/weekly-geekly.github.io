Official documentation for Windows Vista SP1 is available for download.


In anticipation of the imminent release of Windows Vista SP1, Microsoft has released official SP1 documentation. It contains a full description of changes in SP1, as well as recommendations for installing and configuring the product. Microsoft places great emphasis on the fact that it‚Äôs very important to read the documents before installing SP1.

Microsoft's publication of documentation is always timed to a quick product release, SP1 was no exception. On February 4, Microsoft announced the completion of Windows Vista Service Pack 1, but users were disappointed because the company announced that this update would not be available, at least , for another six weeks. It is planned that Windows Vista SP1 will appear on Windows Update in the middle of March. But this update will not be distributed automatically. Only a month later, in mid-April, this feature will be enabled.

But it is quite possible that these deadlines are not final. According to a Microsoft representative, testers found problems when using drivers for some devices. The company refers to the fact that the developers did not follow the recommendations when creating the drivers and now the users who installed Windows Vista Service Pack 1 may have problems with them. Over the next month, Microsoft says, the company will try to fix all possible driver problems.
