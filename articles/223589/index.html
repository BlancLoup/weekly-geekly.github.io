Tronsmart Vega S89 - media player with 8-core video accelerator Mali 450


Replacing the media players with the Rockchip RK3188 comes a new generation on the Amlogic S802 chip, which is based on a 4 core Cortex-A9 processor with a maximum frequency of up to 2 GHz and an eight core Mali 450 video accelerator.
Today we have one of the first media players on this chip on the review - Tronsmart Vega S89.

HDMI: 480i, 480p, 576i, 576p, 720p50, 720p60, 1080p24 (starting with firmware 106k4), 1080i50, 1080p50, 1080i60, 1080p60, 2160p24, 2160p25, 2160p30, 2160pSMPTE
VGA CVBS: 480i, 576i
Amlogic S802 quad-core processor with Cortex A9 architecture
2 GB of RAM
Android 4.4.2 operating system
16 GB user memory + microSD card slot
Connectivity: Wi-Fi (802.11 b / g / n 2.4GHz / 5GHz), Ethernet 10 / 100Mbps, Bluetooth 4.0

The Trosnsmart S89 is positioned as a device that supports 4K image output, also known as QFHD or UltraHD. The resolution of this format is 3840 x 2160 pixels.

There are several modifications of this player: S89 Standard, S89 Elite and S89-H.

‚ñ∫ Firmware

The first thing after receiving the device immediately update the firmware to the most current. At the moment, this is a modified firmware Finless 1.8. From significant updates compared with previous versions can be identified:
- Supports 1080P 24hz image output.
- Built-in drivers for XBOX and Playstation gamepads

Warning: flashing deletes all data from the internal memory! If you have important information - do not forget to make a backup.
Firmware installs from microSD card
Download the latest firmware from the Freaktab website, at the time of writing this article - this is firmware version 1.8
You need to unpack the archive and copy the resulting zip-files to a microSD card formatted in FAT32.
Now you need to turn on our prefix S89. Before switching on, make sure that the OTG cable is not connected, otherwise it will start downloading to the flashing mode - and we do not need it.

You can update the firmware in three ways:
Method 1:
1) By running S89 with the inserted memory card, go to "Applications" and run the application "Update"
2) Click the ‚ÄúSelect‚Äù button and select the file s82_s89_finless18.zip
3) Click the Update button.
4) The system should reboot and automatically install the firmware.
If after installation the S89 rebooted by itself, then everything is fine and the firmware is installed.
If Recovery is booted, continue the installation:
5) Use the mouse to select ‚ÄúApply updated from EXT‚Äù
6) Select the SD card and then the file s82_s89_finless18.zip
7) The firmware will be installed and you will go to the main menu Recovery
8) Select Reboot Now and after rebooting the firmware will complete the installation.

Method 2:
This method can be used if your S89 does not load or Method 1 does not work. To flash S89 in this way, you need to connect a mouse to navigate to Recovery.
1) Make sure your OTG cable is not connected.
2) Copy the file s82_s89_finless18.zip to the microSD card and paste it into S89
3) Disconnect power from S89
4) Reconnect the power, but do not press the power button
5) There is a Reset button at the back of the AV-out, you need to hold it with a paper clip
6) While holding the Reset button, press the power button on S89
7) Hold the Reset button is necessary before the Tronsmart logo appears, after which it can be released
8) The recovery menu should load. If not loaded - repeat steps 3-7 again.
9) Use the mouse to select "Apply updated from EXT"
10) Select the SD card and then the file s82_s89_finless18.zip
11) The firmware will be installed and you will go to the Recovery main menu.
12) Select Reboot Now and after rebooting the firmware will complete the installation.

Method 3:

‚ñ∫ Management

The package includes a control panel, but it is rather primitive for full-fledged work in the Android OS. The movement of the cursor is carried out by holding the direction of the arrows, which is not very convenient. Alternative control options are the so-called Air mouse, in which there is a built-in gyroscope, you control the cursors using hand movements in the air.
At the moment I have such a set of remote controls for control:

1 - iPazzport. In my opinion - this is the most convenient mouse-keyboard. Easy to hold in one hand to move the cursor and convenient to type text on the keyboard by expanding it horizontally.
2 - UKB-500 RF. Keyboard with touchpad. To control the cursor using the touchpad is not very convenient, and the keyboard itself is rather cumbersome.
3 - Q6. Large remote, in which the keyboard is located under the flip cover. Conveniently enough, if you just need a good remote control. Of the additional features - built-in microphone and earpiece.

If you do not want to buy an additional device to control your media player - you can simply connect a mouse and keyboard via USB. Another budget management option is the ability to control from another device on the Android OS. To do this, you must install the RemoteIME application on your tablet or phone, enable the remote control option in the S89 settings, launch the RemoteIME application, connect to the device found:


The application is quite simple, the cursor on the screen repeats your movements that you do on the touchscreen.

You can switch the operating mode of the slides, call the keyboard, or open additional controls to start the videos.

Despite its simplicity, it is quite convenient to use this method.
