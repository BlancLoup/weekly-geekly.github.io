DSM 5.0 Beta is a new OS version for Synology NAS. Part 1 - For home, for family

Good afternoon, dear users!

A little more than a month ago, Synology introduced the beta version of the updated OS for network drives - DSM 5.0.
This update opens up new possibilities for using Synology NAS for home and corporate users. In this article, I will try to describe in detail all the innovations affecting the home user of Synology network storage.


You can not ignore the brand new web interface of the operating system. First, let's remember what the predecessor looked like - DSM 4.x






This is what DSM 5.0 looks like.






Personally, the new interface reminds me of the symbiosis of iOS 7 and Windows 8. It looks bright and fresh!
The main features of the new interface are:
Ease of use with touch screens due to the increase in the size of the elements and the full-screen main menu (3rd screenshot)
Adapted to 4K resolution
Using widgets on the main screen (setting the location and size)

Updated QuickConnect.

Now you can access Synology from anywhere in the world! And for this, it is not at all necessary to get a white ip, forward ports and reconfigure the network. It is enough to register an account in MyDS Synology Center (which will be offered when you start the updated DSM for the first time). As a result, we get QuickConnect ID for login from Synology mobile applications and the address of the form quickconnect.to/username to log in to the web interface.


Cloud Station - the organization of the global cloud.

Now thanks to the update, two-way synchronization of two Synology servers in real time is possible. This most likely refers to users of geographically separated organizations, but one should not miss such a demanded function.
There are also pleasant surprises for the home user - by popular demand, the workers have finally added support for DropBox and GoogleDrive.

DS Video for Samsung SMART TV

Limitations of the DLNA multimedia protocol do not always allow you to create a beautiful film collection with covers, filled with information about the genre, actors, director, etc. And here is the first application from Synology for SMART TV! One more step towards the future.


The release of the full version of DSM 5.0 is scheduled for March 14 (date subject to change)
