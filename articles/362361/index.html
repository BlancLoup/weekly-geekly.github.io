Google Inc. introduced Google Fit: service and application for monitoring user health



Google has announced a large number of interesting new products over the past few months. One of them is Google Fit's health monitoring application, the corporation reported on this application earlier at Google‚Äôs I / O conference. Now this application has become available to users.

Already, you can download Google Fit for free from the Google Play application catalog. It is worth noting that the application is compatible with Android 4.0 and newer versions of this OS.

Google Fit is not just an application, but a whole platform, a cloud service that monitors a person‚Äôs activity and some characteristics of his body. The application is compatible with a number of tablets and smartphones, but it is best to work with Google Fit using fitness trackers, smart watches and other specialized devices.

Google Fit features



The application can be used to track a person's activity when practicing various sports. At the same time, Google Fit automatically detects running, walking and cycling.



During the workout, the application keeps track of the distance traveled, the number of steps taken, calories burned and other information. Additional features can be obtained by connecting specialized fitness gadgets (officially announced support for devices from Nike, HTC, LG, RunKeeper, Withings, Motorola, Noom, Runtastic and Polar).

The user has the ability to add goals (for example, to go at least 2 kilometers per day), after which the application will monitor the implementation of the plan. In order for the results to be assessed, the user is provided with a graphical visualization of the information received by Google Fit.



You can view your results and an overall assessment of the activity both in the mobile application and using the Google Fit web version .

In addition, Google Fit is compatible with third-party applications: Strava, Runtastic or Runkeeper. Data from these applications can be transferred to Google Fit, where information will be analyzed, and then provided to the user in the form of informative graphs, charts and activity logs.



Now, solutions similar to Google Fit are being offered by many companies, including Apple and Samsung.
