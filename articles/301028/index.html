Good afternoon, survived the next move!

3 months after the last release and 2 weeks after the start of the public test , we present you ReactOS 0.4. one

Download.
The official press release.
List of changes. Tests of programs.

What awaits you?


0. Patch for problems related to the activation context.
1. Cumulative semi-annual update of the UniATA ATA device driver.
2. Debut support file system btrfs.
3. Fixes of several serious regressions affecting the operation of PowerPoint Viewer, Word Viewer, FastStone Image Viewer.
5. Network balloon support Virtual Box and VMWare.
6. USB stack fixes, now the number of drops due to it decrease.
7. New, improved non-boring icons in the explorer.
8. Other bug fixes in the amount of more than 140 pieces.

Do you want to try out ReactOS? We recommend starting with a live distribution. If you do not have an optical drive, then use the methods of PXE-boot or RAM-USB



Can I patch ReactOS under KDE4?


Can. But, unfortunately, the bug fix as the corresponding bug report appeared suddenly and therefore did not have time to get into the release.



Now on trunk builds, you can install and run KDE4 without special dances with a tambourine.

PS Friends, project volunteers have written a comprehensive instruction on the preparation of bug reports. Know the more quality bug reports you send, the earlier version 0.4.2 will be released. For the release of a minor release, we will need about 150 fixes (26 are already ready).
