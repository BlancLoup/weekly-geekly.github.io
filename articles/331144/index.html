The list of domains in the zone ru / su / tatar / rf / kids is publicly available due to incorrect DNS settings

Starting from June 6, part of the DNS serving Russian domain zones is allowed to transfer (AXFR), which allows you to get a complete list of domain names for .ru , .su , .tatar , .rf and .children domain names.

The AXFR request is available on the following servers:

  • a.dns.ripn.net
  • b.dns.ripn.net
  • d.dns.ripn.net

To get all the domains of a zone, you can use the dig utility from BIND:

The zone .ru contains 5.1% of all world domains, second only to the first in popularity zone .com .

This feature of the DNS was discovered mandatoryprogrammer (Matthew Bryant) , which has long been engaged in tracking changes in all domain zones with automated tools. The .ru zone is so big that its software fell from a lack of RAM. Last year, he discovered the possibility of obtaining all the domains of the North Korean .kp zone using the same method.

Zone dumps compressed with xz can be downloaded via Bittorrent and I2P Bittorrent , a torrent file or via the following Magnet link:
