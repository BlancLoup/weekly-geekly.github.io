Getting user email via Vkontakte API

Good day to all!


Perhaps the title of my topic, already beaten beyond recognition, and even, can cause sarcastic smiles on the faces of readers. But, nevertheless, I will try.

Although doing web development, and programming in general, not so long ago, I still had to face the API of social networks, such as Facebook and Vkontakte. Robot with "likes" and other data was a great success. But here, I got into a deadlock: how do I link users to a shared account (the problem was following Vkontakte, unlike my older brother, I did not want to give the user email). After reading a lot of forums, various FAQs, I came to the conclusion that this is still impossible. They say that social network developers do not want to give out such confidential information about their users. I had to accept it. The task was solved in another way, but in the shower a small insult lurked.
