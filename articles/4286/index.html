Microsoft has released an offline blog editor.

The free Windows Live Writer program (now in beta) makes it possible to evaluate the design of the posting before publishing it online. The program does not even require knowledge of HTML.

For many users, especially on the dial-up, it is much more convenient to write their texts offline. Here you can calmly and carefully think about them, weigh each word, evaluate the external design. When everything is ready, it‚Äôs enough to go online for a few seconds and click the ‚ÄúPublish‚Äù button. This is how the new Windows Live Writer blog editor from Microsoft works.

According to the developers, the program will help bloggers avoid mistakes, typos and numerous corrections. In addition, the program allows you to feel calm and not be afraid that the text will be lost due to server failure. Autosave works here, so the text will remain on the computer anyway.

Before publication, the program shows how the posting will look exactly, including font size and font style, headings, font color, background images, paragraph spacing, indents, and the appearance of quotes ( screenshot ). An experienced blogger can switch the program to another view mode, for example, with the source code displayed.

Live Writer is capable of updating blogs on the Windows Live Spaces brand site, and is also compatible with many other blogger platforms, including Blogger, Livejournal, TypePad, and WordPress. The editor supports RSD (Really Simple Discoverability), Metaweblog API and Movable Type API. To integrate with the online platform, you must initially specify your login / password and page address, after which the blog will be updated with one click of a button.

Convenient photo editing tools are built into the program interface: for scaling, text wrapping settings and adding graphic effects ( screenshot ). There is a function to access the Windows Live Local map service, so a map or satellite photograph of any region of the planet is quickly located and inserted into the text of the posting ( screenshot ). This feature is especially useful, for example, to show the place where you are at the moment, or, say, to illustrate a message about a political conflict.

Interestingly, third-party modules can be connected to Windows Live Writer, which will allow the program to edit and publish other types of content, such as photos from photo services, video and audio files, and much more. For the development of this kind of additional modules, the Windows Live Writer SDK developer package is used. The program has already published the first plugins .

Windows Live Writer is not the first blog editing program. Earlier this year, Apple released a similar program iWeb , but it is almost tightly tied to Apple‚Äôs own blogging service.
