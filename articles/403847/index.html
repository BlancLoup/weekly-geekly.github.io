Total Commander: search in the current default folder

Total Commander has a search problem. By default, it is produced in all subdirectories. For example, you have a Downloads folder in which a bunch of subdirectories and files. You know that the file you need is in the folder itself, but the TC will by default also search in all subdirectories, which takes time. I noticed that I was mostly looking for something only in the selected folder. This means that it is more efficient to do so by default. The problem is that in TC there is no such possibility. But there is a crutch!
