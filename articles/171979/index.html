Passing a password on Samsung devices with Android OS

Recently, a vulnerability was discovered in Samsung devices with the Android operating system, the vulnerability allows you to bypass the screen lock and temporarily gain access to the system without entering a password (pincode). A similar vulnerability was found in Apple devices based on iOS.

The method below works on Samsung Galaxy Note II for Android 4.1.2 and on Galaxy S Plus for Android 2.3.6 (has not been tested on other devices).

The method is described under the cut.
Sequence of actions to bypass protection:

  • Activate the screen.
  • Set up an Emergency Call.
  • Press the "ICE" key at the bottom left.
  • Hold the physical Hold button for a few seconds, then release.
  • The home screen will appear for a while.
  • While the home screen is visible, you need to run some application or widget.


The attack has a limited scope of use. An application or widget immediately goes into the background, although if the program must perform an action at system startup, it will execute it.
