Technology Wonders - Automatic Overclocking DDR4 from Kingston

Hi, Giktayms! The situation in which to overclock the memory has to poke into BIOS and the stars do not always converge in the best way. Not every user is an overclock guru, but almost every user wants more for the same money (and so not only in IT). Kingston has already successfully implemented the technology of Plug and Play for DDR3 RAM, and now it is the turn of the flagship direction of DDR4. A key feature of these memory modules is the automatic tuning of frequencies stitched into the XMP profile, without the need to tune the BIOS. In other words, the system itself will set the maximum possible specification (frequency and timings) of the RAM without user intervention. Under the cut a little more details for those who, despite the growth of courses, are interested in new products in the hi-end segment.



HyperX FURY DDR4 is an inexpensive and high-performance solution for upgrading systems based on 6-core and 8-core Intel processors. This memory is suitable for those who want to quickly edit video, create 3D-content and play the most modern computer games. The novelty uses all the advantages of DDR4 memory energy with a voltage of 1.2 V, has a low-profile heat sink with an asymmetrical FURY design and a black PCB that will favorably emphasize the appearance of modern PCs.

What makes HyperX Plug-and-Play automatic overclocking? To reveal all the secrets will not work, yet this is a unique development, which is not the competitors. But we can say with confidence that close cooperation with motherboard manufacturers has led to the desired results. The BIOS of the board, when initializing the RAM, is recognized by the flashed XMP profile and independently adjusts the frequency, while other memory modules need to be tuned through the BIOS and this does not always work out correctly.



In addition, the HyperX brand expanded the Predator DDR4 high-speed memory family by introducing kits with a total capacity of 32 GB and 64 GB. They are addressed to users who need maximum performance. HyperX Predator DDR4 memory supports up to 3000 MHz and is available with various capacity options.
