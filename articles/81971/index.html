The basis for the modem was chosen firmware from the bourgeois provider Globe Telecom , posted on the official website of Globe as an update for the same ZTE MF626.
The firmware was made on Windows XP Pro SP3.
Firmware is recommended to carry out on a stationary computer using a USB port located directly on the motherboard. Using USB hubs, extension cords, etc. It is not recommended, moreover, it is desirable to disconnect from the computer all USB devices, including a USB mouse. SIM must be removed.

1. Downloading ( mirror depositfiles ) MF626UpdateTool_GLOBE_PHV1.2.exe from the official website of Globe Telecom.

2. Insert the ZTE MF627 into a free USB port, wait for the installation of MTS Connect Manager and initialization of the device in the system.


3. Disable MTS Connect Manager via Tray - Menu - Exit.


4. Run MF626UpdateTool_GLOBE_PHV1.2.exe , click Download . The firmware process will begin. Do not touch anything, waiting for the completion of the process. It took me 25 minutes to go through all the stages. During the flashing process, the MTS Connect Manager will be run periodically, no problem, these are the consequences of multiple device reboots.






Now we have two options for further work with the device.
Option number 1 - no longer requires any action on your part, except for setting up Globe Visibility Connection Manager , but it does not provide voice communication.
Option number 2 , proposed by SerVik , provides us with the possibility of voice communication and correction of errors in the transmission of data in online games, seen on the firmware for ZTE MF626. The work of third-party software in this version of the firmware is also supported.

Configuring Globe Visibility Connection Manager for example Megaphone:

5.2 Register the " Connection Name ", " Dial Number ", " APN ", click " OK ", then " Set as default ".

" Connection Name " - here you can enter an arbitrary name, in this case - Megaphone.
" Dial Number " - * 99 *** 1 # or * 99 #
" APN " - This is an important point, each operator has his own:
Megaphone: internet name and password is not required, leave blank.
Beeline: internet.beeline.ru name: beeline password: beeline

This information is available on the help page of your operator.

5.3 Next, go to the "Internet" tab, select your profile, click "Connect".




Globe Visibility Connection Manager allows you to send / receive SMS, view / edit SIM contact list. By the way, MTS Connect Manager is deprived of all this, strangely enough, because it is hardware provided for in the modem, and software developers from MTS could not be unaware of it.
