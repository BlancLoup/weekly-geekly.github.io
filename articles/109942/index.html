JS LOIC: one-click attack

As you know, to conduct DDoS-attacks on sites Mastercard, Visa, Paypal and so on. Many members of the Anonymous group use the LOIC program (Low Orbit Ion Cannon), written in C # and originally intended for stress testing sites. The same program used to be used for DDoS attacks on the Church of Scientology sites and the RIAA website (in October 2010)

A remarkable fact: from December 8 to 10, 2010, the client version of this program was downloaded about 30,000 times, while the total number of downloads for the entire time of its existence is approximately 50,000. Most downloads in recent days come from the US (10,000 downloads), but An unexpected interest in stress testing sites was also shown by users from the UK (3200), Germany, France, Holland, Spain and Russia (each of these countries has from 1000 to 2000 downloads).

The server version of LOIC was downloaded about 33 thousand times.

With regard to the online tool "revenge" JS LOIC , then there is the number of users can not be counted.
