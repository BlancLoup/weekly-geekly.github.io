A collection of developer tools

Hello! I have always been interested in finding out what my fellow developers use. It's cool to find an unfamiliar icon on the screenshot in the article or in the screencast of the next report.


No less interesting is what sources help developers get up-to-date information (RSS feeds, podcasts, favorite books)


I decided to dedicate a small home project to this topic and create an open collection with developer tools. This is a GitHub repository where anyone can add their own set of useful programs. From the repository, I generate the site and RSS.


To add your toolkit you need to describe it in a file with a simple microformat. It looks like this: 


name: Nikolay Garmash occupation: Software Engineer > WebStorm Couple of installed plugins‚Ä¶
