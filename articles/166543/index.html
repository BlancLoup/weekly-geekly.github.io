Release of fuse-exfat 1.0.0

On January 19, 2013, the first version of the fuse-exfat software was released.

The application is an extension for FUSE and allows you to mount drives formatted in the ex -ATF file system. In fact, the ExFAT (Extended File Allocation Table) is an extended version of FAT32, with which the limit of one file size of 4 GB has been removed. This driver supports both reading and writing. Among user drives, this file system has not yet gained popularity, but it may soon be actively promoted to the market in connection with licensing agreements signed between Microsoft and major hardware manufacturers. The cameras of some manufacturers already use this FS. For example, in a Canon 60D camera, large-capacity SDXC memory cards are already formatted in ExFAT.

Among the major changes include:
  • Improved stability
  • Performance is higher
  • If the volume is unmounted, while there are open files, the application now does not crash.
  • Big_writes mode appeared
  • MacOS version available

Software is distributed freely under the GNU GPL license.
