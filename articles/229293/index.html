Chrome Remote Desktop extension now supported in Ubuntu and Debian



Google announced that the plugin for Chrome, which allows users to access other computers through a browser, now works in Ubuntu and Debian. The plugin itself was released back in 2012 and is already supported on Windows and Mac OS, an Android application has also recently been released .

Download the extension can be in the Chrome Web Store, and instructions found in Google Help. The setup process in Linux is not as simple as in other systems: first, the user needs to manually create a separate session of the virtual desktop. Possible problems are described on the Google forum , where you can ask questions.
