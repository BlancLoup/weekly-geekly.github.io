New netprofile tool in Mandriva 2010.0

One of the innovations in the future version of Mandriva Linux 2010.0 will be modified work with network profiles. One of the developers (Evgeny Dodonov, Russian only by relatives) posted a screencast on his blog, showing his future work (carefully, pictures!).

So let's think about simplifying simple things. Suppose you use a laptop at home and everything is fine. In turn, each time you come to work, you must constantly change the settings of your network, proxy and firewall. Of course, you can do it manually, or maybe even be able to write a script, but everything will be much easier with the new network profile tool.

The key task of this post is to illustrate how to solve this problem with the new netprofile and the updated drakx-net application.

The first thing you need to do is launch the draknetprofile application and create a new network profile



When you create a new profile, give it a name, for example work:



After that, a new profile will be created and automatically activated:



Now, when using other drakx-net applications, they will be able to detect that you have several network profiles and show you tips on which profile you are currently using. For example, you can configure the firewall profile for:



your network connection:



proxy server settings:



and other network settings:



When you come home, you can simply activate the default profile again with a simple click on the network applet icon:

net_applet switching profiles
