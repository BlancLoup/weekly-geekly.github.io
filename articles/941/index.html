The color of the point on the Enzol map corresponds to the Popularity Rank of the page, and the points themselves are ordered from left to right and from top to bottom in increasing order of the number of hops (hops) of the corresponding web pages. Pages explicitly specified in the search engine config get a hops value of 0, pages offered for indexing via a web form or were found in one of the Internet directories get a hops value equal to 1. All other pages get the value when they first enter the search engine database. hops 1 more than the page where the link to this page was found. In this sorting, the smoothed map looks like this:
