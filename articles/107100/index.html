The first releases of the PDC10 conference



Despite the fact that the PDC10 conference will only begin in a few hours ( live broadcast at 20:00 Moscow time), the first releases of the instruments dedicated to this event have already begun to appear.

Below is a list of released updated tools with download links:

- WCF Data Services (Oct 2010 CTP 1) - tools for building web services based on the open OData protocol in .Net 4.0 and Silverlight 4.0 projects;
- Microsoft Sync Framework 4.0 October 2010 CTP - a new version of the framework that allows you to build applications with synchronization support. In the new release, Odata support has been announced, Windows Phone 7 and SQL Azure synchronization support, Diagnostic Dashboard diagnostic tool has been added, examples have been updated, an example of using the framework on iPhone has been added;
- Windows Azure AppFabric SDK V1.0 - October Update - a new version of the toolkit for building applications that interact with the cloud services of Windows Azure AppFabric Access Control and Windows Azure AppFabric Service Bus;
- WCF RIA Services V1.0 SP1 Beta for Silverlight 4 is a new version of tools for building distributed applications based on Silverlight. Details about the release can be found in this blog post;
- WCF RIA Services Toolkit October 2010 - a set of additional tools for use with WCF RIA Services.

We look forward to the start of the PDC10, during which we expect high-profile announcements, new technologies and a host of interesting reports that will be available online in real time.
