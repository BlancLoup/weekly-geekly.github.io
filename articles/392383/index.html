How to easily crack unencrypted Android, or why you should still encrypt


This note was born later on trying to explain the reason why I am now encrypting my Android smartphone.
The degree of obviousness and simplicity of the method of obtaining information (including the data section) from an unencrypted device blocked by any type of ScreenLocker ªa, extremely surprised me.
This method has been tried on my LG L9, but can be repeated on many other models. For example, on virtually all Lenovo devices, this can be accurately repeated with even smaller gestures.

Required:
-Anroid smartphone and its modelname
-Ability to use search engines
-Memory card
-USB-lace
-PC

So let's go.

The essence of the method is extremely simple and depends on two factors: the encryption of the device and the ability to install custom Recovery. Encrypted devices cannot be so ‚Äúhacked‚Äù.
I think many have already guessed what the essence.
The fact is that on most devices there is a fastboot mode, through which you can easily install custom recovery, in my case TWRP, whose functionality often includes folder mounting management, a simple file manager and a terminal emulator. Thus, we can easily copy the contents (including private) from the internal memory to an external memory card.

I warn you that the instructions are extremely generalized, because On some devices, the method may differ from what I have done, but the essence is the same.

Algorithm:

0. Having learned the device model, we search and download for it an image (.img) of custom recovery.

1. From the same Internet we will learn about how to install this image on the device. In most cases, a method for entering fastboot mode is described. PS: fastboot is not only the Android device mode, but also the utility that we need. When you search, see for yourself.

2. We fall, according to the instructions on the Internet, in fastboot.

3 
fastboot flash recovery recovery.img

For Lenovo devices, the SP Flash Tool utility, they do not need any fastboot.

4. According to the instructions for recovery, we get into it, mount all folders, using the file manager, copy the contents of the directories we need to an external memory card. It is also possible to delete some key files, without which any password will work. For information, contact the search engine.

5. Profit.

And the blocked bootloader is not always a hindrance. In my case, I just found a recovery, which already works.
