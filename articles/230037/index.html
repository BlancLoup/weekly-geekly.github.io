TOX - Protected Messenger got to Alpha version (For all platforms)



Recently, a very important event in the field of secure communications and instant messengers - the Tox project.
He set himself the task of replacing Skype and now he got to the Alpha version.

I want to draw your attention, the alpha version means that more than 90% of the functions work on all OS simultaneously:
- Audio calls
- Video calls
- File transfer
- TCP Relay
- DNS Name discovery
- Group chats

Tox developers decided to choose the best client from the set and call it official, this was uTox ( https://github.com/notsecure/uTox )



You can download it from the official site of the project - wiki.tox.im/download.html

See all other clients here wiki.tox.im/Unofficial_binaries
The Android version is here wiki.tox.im/Antox (pre-alpha)

DNS Discovery


But besides the release of the alpha version, another event happened - a standard was approved and implemented which allows not to memorize long TOX ID - DNS Discovery
Thus, it is possible to register your address your_name@domain.net, i.e., by analogy with the mail.

Technical specifications are described here github.com/Tox/Tox-STS/blob/master/STS.md

And you can try this service here - utox.org / toxme.se

Already, Tox works fine through several firewalls and can do a lot, and this is just alpha ...
