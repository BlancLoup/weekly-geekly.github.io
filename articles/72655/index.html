Blackra1n released for jailbreaking iphone / ipod touch (windows + mac os)

The principle of breaking the firmware is quite simple, you just need to download blackra1n for your operating system, start up aytyuns, update the firmware and then close the aytyuns and run blackra1n, then the program will do everything itself, it takes about 30 seconds to crack the firmware.
In the light of the new firmware update to 3.1.2, we need a new installd patch (to install games downloaded NOT from the AppStore)
1. For a start, start Cydia itself. She herself will offer you this if something is not installed in her.
2. Open Cydia> Manage (in the bottom menu)> Sources> Edit (in the upper right corner)> Add (in the upper left corner)
3. A window will open in which we enter the cydia.hackulo.us repository address
4. After adding the Hackulo.us repository (it will appear in your list of repositories) click the Search icon (in the lower right corner of the screen) and in the search bar we write AppSync
5. The search engine gives us versions of AppSync that are currently available.
6. For firmware 3.0, install AppSync3.0, for firmware 3.1.1 or 3.1.2, install AppSync3.1 (tested for functionality)
7. We put any free application through iTunes (we need an account and an authorized computer),
Then download the games in the .ipa format and open them in iTunes, synchronize the player and play!
