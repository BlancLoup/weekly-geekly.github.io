Opera 12 beta - try new fish



Today, Opera Software has released the first beta version of Opera 12 , codenamed "Wahoo", the latest development in the long history of desktop web browsers. Opera 12 is the fastest and most convenient compared to its predecessors, and also presents new features for testing by both developers and end users, before the release of the final version of the browser. And by tradition, with the change of the serial number, new items are coming. How good they are - to evaluate you. So, what can we offer today to the fish dish ...

Topics

Themes provide you with an easy way to change the look and feel of Opera according to your taste and mood. Just select the appropriate topic - your browser will instantly change. Try different themes here .

It is also very easy to create new themes. If you want to try your hand at this exciting business, find out how to do it here .

Separate processes for plugins

Plugins are the main cause of unstable browser operation, so we separate them into separate processes. This allows you to keep Opera browser working even if the plugin stopped working correctly.

What if web pages look like book pages automatically formatted to fit a computer screen? What if it only takes a few lines of code? Security knows no limits

The new version of Opera has been improved security icon, so that users can more easily determine the degree of security of visited websites. You can visually determine if the website uses information about your location or if it needs an enabled webcam.

Extension extensions

The Opera extensions platform has become even more functional with the advent of the Windows API and Tabs API, which allows extensions to manage windows and tabs in the browser.

Among other innovations:
- 64-bit version for Mac and Windows
- Support Do Not Track function
- Support for CSS3 Animations properties
- Support HTML5 Drag and Drop
- CORS Support - Cross-Origin Resource Sharing
- Supports WebGL and optional hardware acceleration

And, of course, a lot of corrections that can not but rejoice. True, there are some changes that can cause discomfort among some users. The fact is that we decided, finally, to decide on and choose one priority in the large family of our browser add-ons. There were no surprises: Opera extensions won by a large margin. Therefore, in the future we will focus on them. But for the sake of this task, unfortunately, you will have to abandon several less popular add-ons. Starting with Opera 12, widgets, voice features and Opera Unite will be removed from the browser. Users of these add-ons will be able to use them until the end of 2012, then support for add-ons will be discontinued. Part of the functionality of these add-ons will go into extensions.
