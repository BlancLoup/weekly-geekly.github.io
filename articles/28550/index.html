Ubuntu 8.04.1 update released

Two months after the release of Ubuntu 8.04, the first update of the distribution was released - 8.04.1, in which more than 200 packages were updated. Release 8.04.1 is a kind of reference point and differs from Ubuntu 8.04 only by updating the installation images, with the inclusion of the accumulated fixes of serious errors and security problems. The update is also available for other Ubuntu products - Kubuntu, Kubuntu KDE4 remix, Gobuntu, Xubuntu, UbuntuStudio and Mythbuntu.

Some of the most important fixes are:
* The composition includes the final version of Firefox 3.0 (Ubuntu 8.04 release included the fifth beta version). GNOME updated to release 2.22.2;
* LANMAN authentication is enabled in the samba client, without it, problems of connection with some NAS and old Windows servers were observed;
* Fixed problems in the FUSE interface to the GNOME VFS;
* Simplified procedure for selecting the time zone on the installation LiveCD;
* The Hardware Drivers application no longer claims that the proprietary nVidia driver is active, although in fact it is not even installed;
