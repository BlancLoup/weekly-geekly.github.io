AWS Command Line Interface - all in one

Hello!

Do you use command line utilities to manage your AWS servers and services? How tired of updating and downloading them from different places. And the heterogeneity of the teams is also not happy ...

AWS officially released a consolidated CLI, which will later include all available services. And now you can manage
  • Amazon Elastic Compute Cloud (Amazon EC2)
  • Elastic load balancing
  • Auto scaling
  • AWS CloudFormation
  • AWS Elastic Beanstalk
  • Amazon Simple Notification Service (Amazon SNS)
  • Amazon Simple Queue Service (Amazon SQS)
  • Amazon Relational Database Service (Amazon RDS)
  • AWS Identity and Access Management (IAM)
  • AWS Security Token Service (STS)
  • Amazon CloudWatch
  • Amazon Simple Email Service (Amazon SES)

Utilities are available on the Github repository. You can also install them via pip or easy_install. The package name is awscli.
