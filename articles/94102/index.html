Skype introduced the Skype Manager business service.


Skype Internet telephony service today presented its next business development. The new centralized web console will allow organizations to deploy and manage Skype services in their local networks on employees' work computers. A new solution Skype Manager replaces the system Skype Business Control Panel.

Previously, Skype Business Control Panel was positioned by developers as a universal tool that simplifies the deployment of Skype customers at the lowest cost and highest efficiency among medium and large businesses. Now using Skype Manager, commercial Skype users can create Skype accounts on a large scale, manage them through the central panel, enable or disable certain functions, create voice mailboxes, organize commercial subscriptions, and carry out other operations.

Also, the new development supports the integrated payment system Skype Credit, through which you can buy Skype paid services. In addition to this, administrators can track the use of Skype by employees in real time (by department, by group, individually and by company as a whole).



Below is a list of all the benefits that are promised to users of Skype Manager:
  • Account creation : centralized account creation for all employees, including automatic integration of resume files into the program.
  • Balance management : automatic top-up and conversion between accounts of employees from different countries, as well as the overall management of all accounts.
  • Special options : voice mail, call transfer and many other useful functions in the office.
  • Analytics : real-time reports on the use of Skype and the cost of services.
Well, as for a regular custom Skype client, the use of chat and file transfer options between employees is suggested.

In addition, with the help of Skype Manager you can quickly set up Skype for SIP . This solution allows you to integrate the existing PBX and Skype capabilities. Companies with TDM-based PBXs or Key Systems can use Skype for SIP to use interfaces that are manufactured by external suppliers and certified to work with Skype.
