Beta testing ABBYY Lingvo Dictionaries for Android


In the comments, you often ask us when the Lingvo dictionary for Android comes out. We tell: we are actively working on this application and we expect to release it in December. When Lingvo for iOS came out, you helped us improve the program by suggesting many useful things in the comments. We are very grateful to you for this and would like to take into account your wishes when developing a version for Android. Therefore, in October, we plan to conduct an open beta test of ABBYY Lingvo Dictionaries for Android. If you want to take part in it, write an email and a name in a personal, the first 100 people will be able to test the program.
System requirements: Android 2.2 and higher.

Now a little about the beta version:

1. It will contain a dictionary for testing, as well as allow you to download and install other dictionaries from an external server.

2. Morphological support. If the program did not find the exact word you wrote in the input line, it will first look for its normalized forms, and if there is no such form, it will suggest similar words.

3. The ability to view grammatical forms of words, etc.

The program will work without an Internet connection (you will need it only when you want to download new dictionaries or sound to them).
