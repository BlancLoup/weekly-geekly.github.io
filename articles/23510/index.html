1. Let's start with the Transmission. This is an independent development based on its library, it has builds under Mac OS X, and Linux itself on GTK +.

Transmission is written in C. It can work in daemon mode, which is exactly what I needed;). You can control it with the help of the transmission-remote utility that comes with it. The capabilities of the daemon are pretty simple:
  • add / remove torrent
  • setting speed limits
  • torrent management, start / stop
  • check data
  • change port
  • change proxy settings

For him, I found only one web snout called Clutch . To install it, you need PHP 5 installed, with support for sockets. The installation is quite simple, I threw it into the server folder, changed the rights to the Transmission socket, went to the server and voila, it all worked.
