Label colors added to Gmail


Google promised to quickly add new features to the new version of Gmail. The last innovation is the color labels: you can now assign each color its own color, which will help to identify messages more easily. Not all accounts yet have this feature, but, obviously, it will appear to all in the very near future.



It has also become easier to search for messages by labels and remove a label from a message without resorting to the drop-down menu.


Interestingly, similar functionality was previously available as a Greasemonkey script.
