New options for opening and closing tabs in the assembly Vivaldi 1.0.385.5

Hello!

In today's assembly, we invite you to experiment with tabs, especially with the process of opening and closing them. It is no secret that many people prefer to work with a variety of tabs and we decided to simplify a bit and make it more convenient (we hope) to work with these tabs. So what we offer today.

Tab placement when opening

Starting from today's assembly, all links opened from the active tab will ‚Äúpile up‚Äù together. For example, if you searched for some information on a page and want to open several results for comparison, you can open the necessary links from the active page using Ctrl + Click (open in the background page), while all opened links will be placed on adjacent tabs to the right of the active.

At the same time, opening a new tab using Ctrl + T or using a plus on the tab bar will work in standard mode - the new tab will open to the right of all open tabs. Try to work in this style, if it does not suit you - you can disable it in the settings by opening Tools-> Settings-> Tabs-> Position of the new tab and selecting Always at the end .

Behavior when closing tabs

Closing tabs is a logical step after opening them, so we made changes to the default behavior and in this function of the browser. For example, if you close the active tab, you will go to the rightmost ‚Äúassociated‚Äù with it (for example, the one that was opened from the tab to be closed earlier). If there are no ‚Äúrelated‚Äù tabs, you will find yourself on the previous active tab. The purpose of this behavior of the browser is the ability to step through the previously opened tabs from one site.

For this function there is also the possibility of shutdown. To do this, go to Tools-> Settings-> Tabs-> When Closing a Tab and select the Activate Previous In Activation Order item (the translation is not ready yet, it will be later).

In general, try, test, and tell us your impressions and wishes. We will make the necessary changes to these functions if necessary.
