Theme for night reading Habrahabr

Good day, habrazhiteli!
For a long time, I was worried about the problem of night reading habrahabr: I was preventing my wife from sleeping, and my eyes quickly got tired in the dark to read the white site. I made several attempts to create a nightly dark theme for habrahabr and it performed its function, but it was not high enough to put it in public access - after all, the site layout was very conservative and didn‚Äôt respond well to a neat restyling.

With the release of the new design of habrahabr, the layout of the site has become more progressive and simple, and I decided that this is the right moment to implement a long time ago.



In the process, the extempl user connected , and for two we had a completely usable nightly theme, the source code of which was published on Github , and the theme itself was published on userstyles.org and periodically updated to the latest version from the repository.

The main feature of the theme can be considered the use of trendy css-filters to reduce the contrast of images and videos. The original image is displayed when you hover over it with the mouse cursor. It is comfortable and beautiful. Now and in the afternoon I use this topic!

The theme can be installed in most current browsers using the Stylish extension. To install using Stylish, you just need to go to the page above on userstyles.org and click the big attractive button.

I invite to use. If you have found bugs that are bothering you or have any ideas for improvement, I‚Äôm waiting for your pull-requests on Github!
