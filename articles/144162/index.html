Team Foundation Server Express


Team Foundation Server 11 is coming soon. This is a new major release that includes a lot of new things both for developers and for project managers, testers and analysts.
In Team Foundation Server 11, a new version is introduced called Team Foundation Server Express.

This publication will be told exactly about the free edition of Express. The publication is intended for novice developers and those who have not worked with Team Foundation Server.

This version includes features such as:
  • Source Code Management
  • Work Item Tracking
  • Assembly automation
  • Agile Taskboard
  • and much more ...

The most important thing is that this version is absolutely free for individuals and for groups of developers up to 5 people. TFS Express provides an easy way to get started and explore Team Foundation Server. You can install it on your own server.
The express release is essentially the same as the standard Team Foundation Server, except that the installation process is simplified for novice users and is now incredibly quick and easy.

Limitations Express Version


Team Foundation Server Express has the following limitations:
  • no integration with SharePoint,
  • no reporting
  • no more than 5 users supported
  • only supports SQL Server Express Edition (which is installed automatically if it is not already installed)
  • no multiserver configuration
  • There is an Agile Taskboard but no planning and management feedback is provided.
  • does not support TFS Proxy


Of course, your team can grow and you may need more opportunities. You can add more users simply by purchasing Client Access Licenses (CAL) for additional users. If you want more TFS features, you can also upgrade to a full TFS license without losing data.

In addition, integration with Team Foundation Server, including the express version, was added to the products of the Visual Studio Express line.

Install Team Foundation Server Express


The installation of TFS Express is very simple.
You can install it in almost any system. TFS Express does not require a configured domain. The installer also installs SQL Server Express Edition if necessary.


After installing the required components, you will be familiar with the process of updating the server configuration.


After the installation is completed, you can use the administration console for more flexible configuration.


TFS Express also provides web access to the following address: http: // <tfs-server>: 8080 / tfs
