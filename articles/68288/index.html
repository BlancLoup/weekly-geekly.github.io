Quick Config - program for quick change of the specified settings

Becoming the owner of a laptop felt all the delights of mobility. Work has become easier, there is no need to carry projects on a flash drive, set up the environment on a home and a working machine one-on-one. Now you can take with you work on vacation.

But at the same time there was a new need - to change the network connection settings, proxy settings in browsers, fumble folders, close balls depending on the connection location and current tasks.

Having tried a lot of ready-made programs, I decided to write my own, which will fully satisfy my current and future requests.

This is how Quick Config appeared.

The current version of the program suggests changing the following settings:



Network
- Network connections TCP / IP parameters (IP address, mask, gateway, DNS server address), change of connection status
- Changing the name of the computer, inclusion in the domain or workgroup
- Connecting, disconnecting network drives, assigning a drive letter, connecting using user credentials
- Permission, prohibition to local resources, separation of access rights
- Changing the contents of the hosts file
- Change network routes, add permanent routes

System
- Change the sound volume level, change the current sound circuit
- The purpose of the default printer
- Start, stop, restart selected system services

Browser
- Internet Explorer change of properties of connection, home page
- Fire Fox change connection properties, home page

Desktop
- Change the screen resolution, color quality

Programs
- Purpose of the programs used by default: Internet browser, E-Mail client used by default
- Run programs after applying all configuration profile settings

To get started, you need to create configuration profiles. In the properties of the profile, you must specify the name of the profile, include the necessary configuration sections and specify their settings. You can assign a combination of hot keys, when clicked, which profile will be activated. Also, to speed up access to the profile, you can create a shortcut for the profile on the desktop.



The program can apply the settings even when running from an account with limited rights. To do this, the program uses the system service QCService. The service is installed optionally, if there is no need for it, then you can refuse to install it during the program installation process.

Quick Config supports themes, support for themes became possible thanks to the HTMLayout library. Each topic is a regular html page. Ready-made themes can be found in the Skins folder.

Program Status : Freeware (i.e. free)
Operating system : Windows 2000, XP, Vista, 7
