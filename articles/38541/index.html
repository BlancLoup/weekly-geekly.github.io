IE8 Beta 2 Causes Issues With Windows XP SP3

With the release of the second beta version of Internet Explorer 8, Microsoft warned users of Windows XP Service Pack 3 that in some circumstances they would not be able to uninstall the browser or the service pack.

This applies to those users who downloaded and installed IE8 Beta 1 before updating Windows XP SP3. If such users want to upgrade to IE8 Beta 2, they will no longer be able to uninstall the latest browser version and service pack, they will only be able to install later releases of Internet Explorer 8 over the installed one. If you fall into the number of such users, you will receive a notification when installing the second beta version of the browser.

For those users who want to work with IE8 Beta 2, but want to have the ability to uninstall, Microsoft recommends the following steps: first uninstall Windows XP SP3, then IE8 Beta 1, then install Windows XP SP3, and then - IE8 Beta 2.
