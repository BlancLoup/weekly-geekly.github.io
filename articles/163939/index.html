GitHub: statistics for 2012

Good day!

It is unlikely that you can now find an IT person who has not heard about the Git version control system and the popular GitHub hosting. Many actively use it both in production and for storage of personal work and bicycles. Most recently, statistics for 2012 was published on their blog.

Now on a typical day on GitHub, the following happens:
  • 10k new users are registered
  • 140 Gb of data is filled
  • 25k repositories and 7k pull requests are created
  • 10k users create their first repository




The number of users has increased dramatically: now their number is estimated at 2.8 million, an increase in 2012 was 133%. Interestingly, the total number of repositories has increased even more: by 171% (to 4.6 million).



Top 10 most visited GitHub countries:

  1. USA
  2. Germany
  3. Great Britain
  4. China
  5. Japan
  6. France
  7. India
  8. Canada
  9. Russia
  10. Brazil


Top 10 repositories that users have marked by clicking on the asterisk:

  1. Font-Awesome is a symbolic font for Twitter Bootstrap
  2. textmate - text editor for Mac OS X

  4. saasbook / hw3_rottenpotatoes - a platform for creating SaaS-services based on Ruby on Rails

  6. jkbr / httpie is an analogue of the curl utility

  8. twitter / bower - a package manager for web created by Twitter
  9. Kicksend / mailcheck - Javascript library for validating email input forms
  10. Prince-of-Persia-Apple-II - the original code of the game Prince of Persia


Top 10 projects by number of developers:
