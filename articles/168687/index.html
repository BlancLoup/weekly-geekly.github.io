LibreOffice 4.0.0

Today, February 7, The Document Foundation in their blog announced the release of the fourth version of the popular office suite LibreOffice.
Among the major changes and improvements can be identified:
  • Improved work with DOCX and RTF.
  • Integration with multiple document management systems (Alfresco, Nuxeo, OpenText, etc.)
  • Improved performance in Calc and Writer, especially when working with large files
  • Presented an application for remote control of a presentation from Android devices, which works as a remote control.
  • Some interface improvements. Implemented support for themes from Firefox and improved integration with the global Unity menu.
  • Different headers and footers on the first page of Writer documents, without having to create a separate page style.
