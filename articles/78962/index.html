Porting applications for Mac OS X (WineSkin)



For over a year, our team has been engaged in the transfer of games and applications to the Mac OS, 99% of the requested applications have been ported. For this purpose, special utilities and wrappers have been developed, which are available in open source. But I don‚Äôt have time to tell all at once. Perhaps I'll start with Wine Skin .

What it is?
WineSkin is just a shell made for easy port creation of Windows applications for Mac OS X. It can contain a Windows program inside and also run it. And run it easily by simply double-clicking the file, just like a regular Mac. App. The final App is done using WineSkin, Xquartz and all user configurations (in most cases).

Currently available Wineskin RC5 for Mac OS X 10.4, 10.5 and 10.6 (Russian language is supported)
The program includes:
  • Wine - Stock Wine version 1.1.34 X11
  • Xquartz 2.5.0 (x.org server 1.7.2)


Built-in utilities:
  • Installer - installing an application in Wine Skin without Windows
  • ScreenOptions - window mode options
  • WineskinConfig ‚Äî Wine setup (Figure 1)
  • Winetricks - scripts


picture 1

  1. Set Windows Executable - the choice of running the application inside WineSkin
  2. Set Windows Executable flags - additional parameters for running the application, for example, -exec Korich.cfg
  3. Set Icon - ICNS format icon (max 512x512)
  4. Set Menubar Name - the name of the application on the main menu
  5. Set Version Number - application version
  6. Run Winecfg.exe - setting the internal system
  7. Run Regedit.exe - launch registry editor for fine tuning
  8. Create a Custom EXE Launcher - for additional applications if additional utilities are installed with the application
  9. Remake wineprefix for new Wine version - removes debris from the C drive inside the wrapper
  10. Kill all Wine and Wineskin processes - closes the config and exits all the wine processes


Example:
Well, I'll start with the most beloved game Heroes III;)

1. insert the game disc and run the Installer and see

Everything is done in Mac OS X!
2. Further, according to the installer instructions, install the game where you need it, I left the default path
3. launch ScreenOptions and choose Full Screen and resolution for the monitor, for example 800x600
4. Run the WineskinConfig and set the executable file (Set Windows Executable). a window opens with the path "C: \", go to the folder where the game is installed and select HEROES3.EXE
5. Add an icon and set the display name to menubar (Heroes 3)

6. Done!
