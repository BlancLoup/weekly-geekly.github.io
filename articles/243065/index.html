.NET Framework is coming to Open Source and on * nix


Main

  • Reference Source for .NET 4.6 is relicensed under MIT;
  • In the future, the framework will be open source and shipped in parts via NuGet, you can, with the application, deliver your build, which will be isolated from everything else;
  • Development moves to GitHub ;
  • Runtime sources will open soon, including RyuJit and garbage collection;
  • For all this happiness, official support for the Nixes is planned.
