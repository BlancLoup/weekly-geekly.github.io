Microsoft will share information about Windows 10 updates

Previously, we have already pointed out the almost complete lack of information on the part of Microsoft about the useful properties of updates for Windows 10 (not security). This became clear at the time of the release of the first major update for Windows 10 in early August last year. Unlike users of previous versions of the OS, especially Windows 7 and Windows 8 / 8.1, users of the new OS remained completely unaware of what this or that update was intended for, since the company, in fact, refused to notify users and specialists about it. , for which such information is very important.



Today, on the day of the next batch of updates, Microsoft decided to correct this situation by submitting a special section on its website called Windows 10 update history . This portal will contain information about both security and non-security updates released for Windows 10 in chronological order.

We provide you with information about how you need it. deliver through Windows Update. You will see a summary of the important product developments. New updates are released.

Microsoft indicates that the updates will be delivered to the two supported branches of Windows 10: originally released with build number 10240 and with the November major update build number 10586 (aka 1511).

Judging by the published data for today's update package, besides security updates, the bugs in the update installation mechanism, authentication, and full OS installation have been fixed. Edge web browser has fixed the cache for caching user-visited URLs in a confidential InPrivate browsing mode. The bugs were also fixed in components such as the Windows UX and taskbar (taskbar), as well as the Internet Explorer 11 web browser.

For complete information, see windows.microsoft.com/en-us/windows-10/update-history-windows-10
