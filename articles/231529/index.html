cool-old-term - good old terminal

Someone with the nickname Swordfish90 is developing a cool-old-term terminal emulator with an interface reminiscent of old CRT monitors. The author admits that he dreamed of implementing such an application from the time when he first played in Fallout III. Swordfish90 tries to make an old-school, but at the same time customizable and lightweight terminal emulator. The project is being developed using Qt5 and QtQuick, development is being done on Github .
