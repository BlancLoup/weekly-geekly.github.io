Triggit: WYSIWYG editor for any sites

Startup Triggit has released an interesting web program (browser toolbar) that allows you to edit the web page where its script is registered. The Blogger and Typepad blogs are integrated into the script at the touch of a button, and on other platforms you need to manually register it. After that, the user can add links, Flickr photos and YouTube videos to his blog through the Triggit pop-up window in the browser, where content is viewed and selected. This is the easiest visual web page editor you can think of.



At the moment, Triggit is in the closed beta stage, but invites are already being heard by the hundreds.
