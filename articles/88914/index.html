In June, Google launched Sputnik , a utility designed to test browser compatibility with JavaScript. The utility includes about 5000 tests. Recently, the utility has become easier for any user, since it has become available through the browser. The test results are very interesting.

In particular, the results are interesting because Opera and Safari browsers have overtaken the ‚Äútest‚Äù hosts browser - Chrome. As you can see in the screenshot below - Opera is the undoubted leader with only 78 errors (the closer to the center, the fewer errors). Tests were run on computers running Windows with the latest versions of each browser. Testing on a computer with MacOS showed similar results, except for IE, of course.

Since JavaScript is most focused on performance (the SunSpider test is also aimed at), Sputnik is interesting because it focuses on compatibility and correct processing, which is more like the Acid3 test, which tests for compatibility with web standards. In the Acid3 Chrome, Safari and Opera tests fully comply with the standards, Firefox 3.6 does not lag behind - 94 out of 100. IE8, by the way, is the worst. He has only 20 out of 100. IE9 does better - 55 out of 100.
