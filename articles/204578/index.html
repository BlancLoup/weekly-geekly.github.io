Yii 2 alpha

The first alpha version of Yii 2 has been released. You can download it as an archive from the official site . It also contains installation instructions through Composer.

Yii 2 is a fully reworked version of one of the most popular PHP frameworks. The new version inherits the general spirit of the first version, namely the desire for simplicity, speed and extensibility. In Yii 2, many of the features of PHP 5.4 apply. The framework follows the best practices of modern web development.



A full description of the possibilities can be found in the full manual, and especially in the section with differences from version 1.1 .

The release of alpha is an important event in the development of Yii 2 and means that the code has reached a certain degree of stability. If you are just starting to learn Yii or your project is not limited to a rigid time frame, you can try the new version. Do not use Yii 2 in combat projects as the framework team can still make significant changes without any prior notice.

Within two years, a huge amount of code was received from the community . Thank you for making this release possible.
