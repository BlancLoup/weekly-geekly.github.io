Pro Git translation completed

What could be the best gift for a day of knowledge for a programmer? Of course, a useful book;) Therefore, the team of translators ‚ÄúPro Git‚Äù jumped up and completed the translation of the book into Russian.

Pro Git is a fairly extensive educational book about Git by Scott Shakon, an active participant in the development of Git and GitHub projects. The author examines various aspects of working with Git in the text, starting with the installation of the program and the basic principles of decentralized version control systems, and ending with considering the internal structure of Git and creating valid objects in the Git database with my own hands. Despite the rather extensive material and touching on quite specific topics, the book is written in a fairly simple language, contains a lot of examples and illustrations, and therefore should be understandable to beginners, just starting to get acquainted with version control systems.

The latest version of the translation of the book is available in pdf , epub , mobi .
An online version of the translation is also available, but unfortunately it has not been updated since May.
