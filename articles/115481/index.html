Lost in static

I already wrote about one great game , and thanks to Habra it was ported to androyd, rewritten in javascript and flash.

In continuation, I remembered the wonderful Lost in static :

Lost in the Static is a strange platform game that uses the human perception system to create a whole visual world with the help of one simple static picture, the separate parts of which move in different directions and at different speeds.
