iBotnet is the fair name of the first botnet for Mac Os X

The first botnet was found, the computers of which are running the Apple Mac OS X operating system, according to eWeek, citing data from the computer security company Symantec.
This network of zombie computers is called iBotnet. Its formation began a couple of months ago.

IBotnet includes computers infected with OSX.iServices trojan. It is distributed under the guise of a program that removes protection from Photoshop CS4 and iWork'09.

Symantec estimates that there are currently several thousand computers in iBotnet. They have already been used for DDoS attacks. It is possible that iBotnet will be involved in sending spam.

OSX.iServices Trojan was discovered in January 2009. According to the company Intego, which develops antivirus programs for Apple computers, along with pirated iWork'09 and Photoshop CS4, this trojan hit 20,000 computers running Mac OS X.
