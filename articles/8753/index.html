Firefox users are more serious about security than other browsers.

The latest statistics provided by Secunia shows that users of Mozilla Firefox browser are more likely to surf the web with the latest security updates installed than Internet Explorer and Opera users.

Secunia scanned 4.9 million software on users' computers with the help of its free Software Inspector tool and found that for 1.4 million of them, no patches were installed that cover critical vulnerabilities.

Comparing Firefox, Internet Explorer and Opera, Secunia found that Firefox 2 was the least vulnerable browser ‚Äî only 5.19% of browser copies did not install security updates. As for Internet Explorer, updates were not installed on 9.61% of copies of IE6 and 5.4% of copies of IE7. Users of the Norwegian browser in 11.96% of cases do not update their browser - with this indicator Opera, or rather its users, become outsiders.

Since Software Inspector is designed only for MS Windows, statistics on browsers such as Safari or Konqueror are not available to us, alas.
