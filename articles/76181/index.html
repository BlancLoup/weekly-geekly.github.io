Core i9 - fast, but not so

The Polish site PCLab.pl was the first to test the six-core Core i9 Gulftown processor.



The test sample was made using 32nm technology and had a frequency of 3.07 GHz, however, to obtain more "honest" results, the frequency was artificially lowered to 2.8 GHz. Core i7, Core i5, Core 2 Quad and AMD Phenom II were chosen as rivals.

In short, i9 justified its flagship title and did not justify the possible purchase costs. I want to say that the CPU was really faster than all its opponents (in the synthetic test, Sandra simply broke everyone), but in many applications and games in particular, this advantage was a few percent.

A striking example of this is the test results for FarCry 2
