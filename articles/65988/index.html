What is MODx?


MODx is a free content management system and framework for creating open source web applications under the GPL license. The system is incredibly flexible and easy to learn!

From myself I can say that we have been waiting for this release for a long time. Glitches with Russian have finally disappeared. After installation, you do not need to fix anything with a file to give it work. Well, also a new convenient front end editor. And a lot of delicious little things.

The Evolution version is the completion of the 0.9.6.x branch, and now the work on the MODx 2.0.0 Revolution is in full swing (you can download it too).

1.0 !? Haha, children!


I want to warn you against one delusion. Apparently, the point is that MODx developers were not at all embarrassed by the magic of numbers and for a very long time they made stable versions with numbers less than one for quite a while (0.9.6.3, for example).

But at one point, they decided to make version 0.9.7, rewriting the source code again. After half a year after the start of work, they decided that this amount of work deserves version number 2.0 and the name Revolution (absolutely right? By the way!).

And in the 0.9.6 branch, they decided to release the final version 1.0 Evolution, the appearance of which happened on the last day of July.
