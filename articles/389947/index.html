Fitbit Alta: new Fitbit tracker with interchangeable accessories and motivating features



The company Fitbit, a manufacturer of fitness trackers of the same name, has released a new device - Fitbit Alta. This is a stylish tracker with interchangeable accessories and with an extended set of features that encourage the user to lead an active lifestyle.

The body of the gadget is now made of stainless steel (silver and gold hues are available), with the addition of several lining options in various colors, created from different materials. These materials include leather. As planned by the manufacturer, such a tracker can be changed, modified, depending on where the user is going.



If for a jog - and the usual body, and if the meeting, in the theater, etc. - You can add a leather insert or insert from other materials.



Other features of the bracelet include the usual - counting steps, mileage, calorie counter and sleep quality detector, and additional. For example, a bracelet tells the user when to move. This allows you to be active all day and not stay up too long at your desk.

Plus, it is possible to set a common goal for the day (walk 10,000 steps, for example), and a mini goal for an hour. And the bracelet will remind you if a person needs to complete the planned 200 steps per hour. If necessary, all this can be turned off - well, for example, if a person is now at a business meeting or conference.



Another feature, SmartTrack tracks all types of user activity throughout the day. In this case, the system helps to make a plan for the week ahead. Among the types of activity that the bracelet recognizes are walking, cycling, running, aerobics, dancing and weightlifting (training on simulators).

The user can see the information on the OLED-display (it is small, if you need a larger screen, then this is another model - Fitbit Surge ). More detailed data is transmitted to the smartphone, to the appropriate application. If desired, the Bluetooth Smart Wireless Bracelet interface helps you make phone calls, send text messages, and view plan items for the day.

The smarter model is the Fitbit Blaze smart watch, which is about to enter the Russian market.

The battery is capable of maintaining the device for 5 days. The software is under Android, iOS, Windows.
