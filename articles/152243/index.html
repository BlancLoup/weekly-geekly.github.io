Free dictionaries for Android

A small review of dictionary skins for Android. Dictionaries on the phone are great. And here‚Äôs what to watch them ...

ColorDict


This program is quite popular. It has not a lot of settings, a very simple interface and it performs its functions well.

You can use dictionaries from stardicict, as well as search for articles by headlines in Wikipedia and wordnet.
Play audio files with pronunciation of Colordict can not, but this is not the biggest drawback.
When searching, autocompletion works and the query history is saved.

Full-text and morphological search in dictionary entries is not implemented. This is a significant drawback for many languages.

All installed dictionaries can be disconnected and connected as needed, but this is very inconvenient to do, especially if there are several dictionaries and they are for different languages. Each dictionary has to be turned on / off individually, multiple selection of dictionary list items and a group of dictionaries are not implemented.
In addition, for each dictionary, you can set the color of the frame that will frame the dictionary entries.


The settings allow you to change the font size and display of dictionary entries. There are very few settings, and they are understandable almost at a glance.


The display of entries is quite satisfactory, but some stardict dictionaries are shown with a large number of whitespace,
so reading such dictionaries on the small screen of the phone is difficult.
Many e-book readers are integrated with ColorDict.
Conclusion ColorDict has a very simple interface and is quite convenient, and easy to use and, if the number of necessary dictionaries is small, then it copes with its task.

Fora dictionary


The following program "understands" significantly more formats, namely:
StarDict, XDXF, DSL, DICTD and Plain (TSV) dictionary formats, in addition, she can search for articles in Wikipedia.

In addition to the standard search, you can try another Wildcard search and Fuzzy search. Morphological search is implemented only for the English language.

Auto-completion also works and query history is saved.

The great advantage of the program is the presence of profiles ‚Äî several dictionaries can be combined into one profile (group) and switch between them to search by a group of dictionaries.


The procedure for adding dictionaries is rather inconvenient, but after dictionaries are sorted by profiles, life is greatly simplified. When connecting dictionaries, indexing proceeds. Well, very slowly ...
When trying to connect the English-Russian Muller dictionary in stardict format, the program failed in an exceptional situation, and a couple more dictionaries also could not be connected.

The program has a fairly large number of settings, including font settings, colors, etc., most of which are not immediately clear, but over time, some of them are very useful.
Dictionary entries are displayed quite compactly, and to quickly navigate through dictionary entries (if there are several of them as a result of the search), you can enable the sidebar-panel for quick navigation.

As well as ColorDict for each dictionary, you can set the color, which will be displayed dictionary entries.
Conclusion Fora Dictionary supports all popular formats of dictionaries, allows you to work with a large number of dictionaries, almost omnivorous, but not quite stable, and with a rather complex interface and a difficult character.
