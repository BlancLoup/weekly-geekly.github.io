BlackBerry 10 Native SDK beta released

As planned, today launched the BlackBerry World 2012 and the BlackBerry 10 Jam . At the conference, developers will be available devices BlackBerry 10 alpha. I am very pleased that the developers will not only get the devices, but also the new toolkit, including the BlackBerry 10 Native SDK, which includes the following improvements ( relative to the PlayBook NDK 2.0 ):

- Push Management API is the foundation of the BlackBerry platform. These interfaces allow developers to create push applications.

- Battery Monitoring and LED Control gives applications access to the hardware of the device. Battery level monitoring allows applications to read the battery level, so that they can inform users of potential data loss if the battery level is too low. The app may also receive a low battery warning.

- Payment API facilitates the integration of payment processing mechanisms in applications. Applications.

- Scoreloop SDK 2.0 , which provides interfaces for working with user profiles and global high score tables, achievements, and competition with other players. Game developers, in particular, can improve their game by adding new interesting features for social interaction.
