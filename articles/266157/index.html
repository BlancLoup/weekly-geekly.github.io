System-wide changes (kernel, drivers, libraries):


  • proportional scalable system font
  • increased speed and stability of IDE drives, and SATA in IDE mode.
  • Ability to work with any of the installed network cards (previously used the first detected network card)
  • console.obj : scroll with mouse.
  • kmenu.obj: a library with the implementation of the main and context menu.



Changes in application software:


  • Eolite : change of attributes of both a single file / directory and a group of selected ones; custom font size; progress bar in copy dialog; the work of shortcuts on layouts other than English; two panel mode; remembering the size and position of the window; multiple fixes and optimizations.


  • WebView : use of new system fonts; improved support for some tags; program optimization and bug fixes.
  • MouseCfg : check and configure mouse parameters.
    Spoiler header
    The program allows you to check the performance of the mouse, adjust the speed and delay of the cursor, and is also a shell for working with madmouse programs (allows you to make the edges of the screen pass-through, that is, the cursor reaches one edge, jumps to the opposite one) and mousemul (also emulates a mouse using NumPad keys) that do not have their own interface.



  • TmpDisk : displays the size of the created discs.


  • TextEdit : toolbar change and full menu


  • TinyPad : bug fix when working with clipboard.
  • Calypte : text file reader.
    Spoiler header
    Current functionality:
    - opening text documents encoded in DOS
    - navigation using the PgDn, PgUp, Down, Up, Home, End keys
    - adaptation of the output to the width of the window

    As you can see, the functionality is not rich, but very soon you should expect bookmarks, a custom font and many other useful buns.



  • Calc : use an enlarged font to display numbers.
  • VNC Client:; support for TRLE, 32bpp, keyboard; bug fixes and refactoring






  • End (C-- version) : new design.


  • MTDBG : reduced redrawing of the window when the program starts.
  • TicTacToe : a new logical game, the feature is that it is written in Oberon07


  • Maze : new puzzle game, the feature is that it is written on Oberon07


  • ALMAZ : new arcade game, analog Lode Runner
