Windows 8: Ribbon and Metro UI to the masses

This weekend, the network has penetrated the screenshots gallery of Windows 8, which is in deep development. Judging by them, the corporation goes to all the experiments with the Ribbon - this interface strip first appeared in Microsoft Office 2007 and at first caused a lot of controversy. In Windows 7, this GUI element has already appeared in standard programs - Wordpad and Paint. Now it was the turn of the holy of holies of the system, namely, the Explorer. Using Ribbon allows interface designers to increase the number of new features offered by Explorer without hiding them in a pile of menus and submenus. So, Long Zheng, a fan of Microsoft from Australia, noticed that in the new version of Windows, out-of-the-box, there will be synchronization and web sharing options that fit perfectly into the stated trend: Windows + Cloud, because Most likely the new chips use the full power of Windws Live and Live SkyDrive.

Under the cat you will find a gallery of leaked screenshots and a bonus - a new login screen made in Metro UI style (Windows Phone 7), the media notes that despite the date on the screenshot (April 1), many sources confirm that it exists.
