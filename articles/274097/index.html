When I saw for the first time that the material in Cycles not only shines itself, but also illuminates everything, I thought that the lamps as a class of objects are no longer needed. I was wrong.

Note the OpenEXR format
It allows you to save an image of 32 bits (usually 8 bits). With the help of Photoshop, you can return overexposed details. Open the rendered frame in Photoshop and translate it into 8 bits. When translating a dialog will appear in which you can adjust the brightness.



We cannot see at the same time the whole range of brightness that nature offers us. But the full range of brightness can be saved and corrected after the render.

Camera switching
  • 0 (zero) (numeric keypad) - view from active camera
  • Ctrl + 0 (zero) (numeric keypad) - view from the selected camera. Makes this selected camera main, i.e. active. You can also watch from objects and light sources.


Switch cameras in animation
  • In the working window we create cameras and place them.
  • In the timeline window, click M (Marker) and add markers where we want to switch cameras.
  • We select the camera. Select a marker. In the timeline window, press Ctrl + B (Bind Camera to Markers) . We do this with every camera.

How to set up automatic rendering of the scene on multiple machines?
Customize the scene:
  • Assign the path to the folder rendered frames. It must be relative ie dancing from the scene folder, not from the root of the disk.
  • Checkbox Overwrite uncheck - prohibit replacing already created frames.
  • Checkbox Placeholders put - when the frame is taken into processing, a fake file is created simulating the created frame.




  • Scene copy on each machine is not necessary. Open network access to the folder with the scene.
  • Blender cannot open network folders with its own dialog. Open the scene through the operating system explorer and render.
  • It happens for various external reasons that some frames are not calculated. Usually it is easy to sort these frames in Explorer by size in megabytes. Defective frames have a small size. You just have to throw them out and re-render the render, the machines themselves will fill the resulting gaps.
