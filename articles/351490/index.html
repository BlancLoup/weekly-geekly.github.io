KOMPAS-3D v17 Home allows you to:

  • create models;
  • create drawings;

History of the project KOMPAS-3D Home


Themselves used for their projects


The product becomes quality when you want to work in it yourself. Even the very first versions of KOMPAS were used both by ASCON employees and users for personal purposes: to arrange furniture, to calculate the number of tiles, the wallpaper footage, to make a project of a country house, etc. Users took for this purpose trimmed LT versions and pirated products. In 2011, we decided to conduct an experiment and released a version without reducing any functionality (with all our libraries and applications) at a price affordable to any user. None of the world's manufacturers of professional three-dimensional CAD systems did this at the time. The product found its user. Like the product and our staff.

Description of the appearance and capabilities of KOMPAS-3D Home


Interface






  • The seventeenth KOMPAS-3D Home, like the whole KOMPAS family, switched to a new interface. The interface is reminiscent of the Ribbon, but operates on a different principle. The key feature is that the toolbox contains all the commands necessary for current activities.
  • The entire interface is completely in Russian and supports Russian standards. That, actually, is not surprising for a Russian company.
  • Rather familiar arrangement of menu items. There are, of course, small changes, but you quickly get used to them.

In the new version, the way of interaction with the mouse is unified ‚Äî in the drawing and in the model, the middle button (or wheel) is responsible for moving, and the right button is responsible for rotating the model.

What is a group of teams?




The same type of team combined into groups. Inside the groups, the parameters of the command are saved to the maximum. Commands within a group can be switched at the top of the Options panel or selected from the drop-down panel that appears if you hold down the left mouse button on the main command in the toolbar.

Dashboard Sketch


Most of the constructions in KOMPAS-3D begin with the construction of a sketch - a flat contour, which is then given volume by means of formative operations.


The Sketch panel contains the most frequently used tools for flat contouring.
By default, the panel consists of one command line with three lines. The remaining commands are in the ‚Äúbasement‚Äù, which can be opened by clicking on the line with the name of the panel. If you are uncomfortable opening the ‚Äúfooter‚Äù every time, you can open the context menu and select Show all commands on the panel .




The figure shows a view of the folded and unfolded panel.



Sketch toolbar commands;
In this panel we see the tools:

Autoline

This tool is needed to build a line that is drawn with dots. These points can be attached to other objects or set in any order. You need Autoline, for example, to build a contour of the figure. If you squeeze it in the future, you get a blank of the desired shape.

Circle

This element is a group of commands designed to build circles in different ways.

Rectangle

This element is a group of commands designed to build rectangles and polygons in various ways.

Auto Ax

The command is designed to simplify the creation of various types of centerlines.

Auxiliary straight

A group of teams serves to create various auxiliary lines necessary for approximate constructions, and also helps inexperienced users to build basic geometry, since replaces a ruler or tire on the screen.

To project an object

The team creates in the sketch a projection on the plane of geometry from a three-dimensional model or another sketch, which is parametrically connected with the source of the projection.

After activating any of these commands, the system starts the process of placing the sketch on the plane, and then goes into sketch mode. Sketch mode is a normal drawing on a plane. In sketch mode, a set of sketch tools opens - accordingly, additional commands appear.



The process of placing the sketch on the plane

End description of the Sketch panel.



Toolbar Body elements


The commands of this panel are used to transform a flat contour into a three-dimensional model or ‚Äúsolid body‚Äù, as well as to modify the resulting model in various ways.



Commands panel body elements
Extrusion Element

This element is a group of commands designed to build shaping elements. Extrusion is necessary in order to give the specified contour a volume by extruding it in the indicated forward direction. Rotation adds volume by rotating the contour around the axis of symmetry. The element along the path ( earlier this command was called Kinematic operation ) adds volume to the contour along the path specified by the user. The element on sections creates a solid body from several specified flat contours.

Extrude cut

This element is a group of commands designed to cut from the body the same elements that are added to the previous group of commands.

Rounding

This element is a group of commands for building.

Hole simple

This element is a group of commands that allow you to make holes in the workpieces by giving them diameter and depth. Also here is the chamfer.

Boolean operation

This function is needed to subtract or remove the volume of a particular body from another object. It may be necessary, for example, when we need to cut the shape of an asterisk in a rectangle or make holes for bolts.

Thicken

This command adds a layer of material to a face or surface.

Section

This command cuts off part of the body using a sketch, plane or surface.

To scale

This command proportionally changes the size of the body or the whole part.

Stiffener

A stiffener is needed to give strength to any structure, especially in those places where it is impractical to put a full-fledged wall.

Shell

The shell leaves a thin layer of material from the body, deposited on the outer or inner surface left over from the body.

Change position

Creates a new solid, surface or part by changing the position of an existing object.

Add workpiece detail ...

With this function, you can add any part that you previously saved to our project.

Bias

This command tilts the existing face with respect to the other face.

End of body parts panel description.

Short modeling lessons


Here are three small construction lessons on the main types of modeling. Lessons do not intersect with the ABC, so it will be a good addition to them.

Printed hanger






The process of modeling a solid model
  1. First create a new document Detail.
  2. Let's right-click on the root element of the tree and select ‚ÄúModel properties‚Äù in the context menu.
  3. Enter the name of the model "Hook". Choose any color you like, for example, Lime.
  4. Open the list of materials, choose any plastic. Click OK in the ‚ÄúMaterial Density‚Äù window and Create in the Parameters panel.
  5. Click Save As to save the item.
  6. Select the folder in which to save the model and click Save.
  7. We hold down the left mouse button on the ‚ÄúRectangle‚Äù command group, in the opened panel, select ‚ÄúPolygon‚Äù.
  8. Specify the XY plane.
  9. Build a polygon from the origin. Set the number of vertices - 6 pieces, diameter 50 mm. Specify the angle - 0 degrees.
  10. Without completing the previous command, run the Extrusion command.
  11. Enter the distance - 5 mm, angle - 10 degrees and, if necessary, change the direction of the angle. Not completing the previous command, click the button Sketch.
  12. Specify the ZY plane.
  13. Clamp the left mouse button on the ‚ÄúArc‚Äù command group, in the opened panel, select ‚ÄúArc, tangent to the curve‚Äù.
  14. Specify a segment for tangency and arbitrarily specify the starting point of the arc.
  15. Enter the radius of the arc 10, if necessary, change the direction of the arc and arbitrarily indicate the end point of the arc.
  16. If the end of the arc is detached from the segment, run the command Point on the curve. As a curve, we indicate the segment to which the arc is tangent, and as the point, the lowest point of the arc.
  17. Run the Avtorazmer command.
  18. Put down the radial size, set the horizontal dimensions from the origin to the beginning and end of the arc. Exit sketch mode.
  19. Clamp the left mouse button on the ‚ÄúOffset plane‚Äù command group, in the opened panel select ‚ÄúPlane through a point perpendicular to the edge‚Äù.
  20. Specify the arc, then the starting point of the arc - created a plane. Without interrupting the command, we again specify the arc and the end point of the arc - the second plane was created.
  21. Without interrupting the previous command, run the Circle command.
  22. Specify the top plane.
  23. Specify the origin and enter the diameter of 4 mm. Exit sketch mode.
  24. Run the Circle command again.
  25. Specify the bottom plane.
  26. Specify the origin and enter the diameter of 8 mm. Then we hold down the left mouse button on the ‚ÄúExtrusion Element‚Äù command group, in the opened panel, select ‚ÄúElement by sections‚Äù.
  27. We indicate the constructed circles as sections, and the arc as the center line.
  28. Without interrupting the previous command, run the "Rectangle" command.
  29. Select the ZY plane.
  30. We build an arbitrary rectangle from the origin to the side of the hook.
  31. Run the command autosize and set the size of the rectangle 10 and 13 mm.
  32. Change the line style at the vertical segment, leaving the origin.
  33. Run the Round command.
  34. Set a radius of 4 mm and indicate the segments at the angle opposite to the origin.
  35. Clamp the left mouse button on the ‚ÄúExtrusion Element‚Äù command group, in the opened panel, select the ‚ÄúRotation Element‚Äù.
  36. We hold down the left mouse button on the ‚ÄúHole simple‚Äù command group, in the opened panel, select ‚ÄúHole with countersink‚Äù.
  37. Set the diameter of the hole - 4 mm, diameter countersink - 6 mm. We open section placement. Specify a distance of 1 - 13 mm, a distance of 2 - 6 mm.
  38. We hold down the left mouse button on the ‚ÄúArray on grid‚Äù command group, in the opened panel we select ‚ÄúMirror array‚Äù.
  39. Specify the hole and the plane ZY.
  40. Run the Round command.
  41. Set the radius - 2 mm. Specify the faces that need to be rounded. Since the operation is the last, click the Create button.
  42. We are preparing a model for 3D printing. Click the "Save As ..." button.
  43. Select the file type Stl.
  44. Click the triangle to the right of the Save button and select "Save with parameters ...".
  45. Specify the maximum angular deviation - 3 degrees and the maximum length of the edge - 5 mm. Maximum linear deviation is disabled.
  46. The detail is ready for 3D printing. It remains to transfer it to the printer slicer and get the g-code.

Complex case




Surface modeling process
Suppose you have developed a similar concept design of the case. You need to make it a reality.


Using solid modeling, this is difficult to do because the form is complex.
At best, it will be possible to get something like this, or it will take too much effort.


We use to obtain the desired shape surface modeling.

To begin with we will break the initial image into form-building elements.


  1. Create a part.
  2. Create a sketch in the ZY plane. Run the Conic Curve command.
  3. We build an arbitrary conical curve, set the coefficient to 0.5.
  4. Activate the mode of displaying degrees of freedom.
  5. Run the autosize command and set the dimensions as in the picture.
  6. Run the Alignment command, specify the lowest point of the conic curve and the origin, select the horizontal line to align the points horizontally.
  7. We build a vertical auto-axis from the origin of coordinates. In the image, the size is tucked down so as not to interfere with construction.
  8. Run the command Mirror reflection. Specify the conical curve. Click Create.
  9. Specify the centerline. Then run the command line.
  10. We build a segment between the lower points of the conical curves. Exit sketch mode.
  11. Create a new sketch in XZ.
  12. Again, we build a conical curve (coefficient 0.75) and dimension the dimensions and dependencies as in the figure. Exit sketch mode.
  13. Run the "Project Object" command.
  14. Specify the XY plane.
  15. We project into the sketch the lower point of the conical curve constructed earlier. To see the curve, it is best to expand the model using the right mouse button.
  16. Run the command "Spline on points". We specify 5 points so that the average coincides with the projected point. Run the Align command.
  17. We specify in pairs unbound points and set verticality between them. Run the Avtorazmer command.
  18. We put down the dimensions, as in the figure. Exit sketch mode.
  19. Create a new sketch in the ZY plane. Run the command "Spline Poles". We build an arbitrary spline of 7 poles. Run the command autosize.
  20. We put down the dimensions, as in the figure. Exit sketch mode.
  21. The main shaping sketches are built - you can start building surfaces.
  22. Switch to the set of "frame and surface." Launch the Extrusion Surface command.
  23. Open the transparent tree. Select the first sketch and set the distance to 180 mm. Create an operation.
  24. Run the command Surface trajectory.
  25. We specify the third sketch as a section, and the second as a trajectory. Select the method of building the "maintain the angle of inclination." Create an operation.
  26. Run the Extrusion Surface command again. Specify the fourth sketch and a distance of 200 mm. Run the command Trim the surface.
  27. Specify the extrusion surface in the tree. Switch to a segmented object - specify the kinematic surface. If necessary, change the direction so that the part forming the surface we need remains. Create an operation.
  28. Select the kinematic surface and the first sketch as a cutting object. If necessary, change the direction so that the part forming the surface we need remains. Run the command Surface sewing.
  29. We indicate the parts of the kinematic surface and the extrusion surface remaining after truncation. Run the command Equidistant surface.
  30. Specify the stitching of surfaces in the tree. When creating equidistants, edge effects inevitably appear. They need to be cut off. Run the offset plane to create a truncation.
  31. We build the displaced plane at a distance of 2 mm in the direction of the surfaces. Run the command Trim the surface.
  32. Specify the stitching of surfaces and the displaced plane as a clipping object. If necessary, change the direction so that the part forming the surface we need remains. Create an operation.
  33. Specify the equidistant surface and the displaced plane as a clipping object. If necessary, change the direction so that the part forming the surface we need remains. Create an operation.
  34. Specify "Extrusion Surface: 2" and the offset plane as a cutting object. If necessary, change the direction so that the part forming the surface we need remains. Run the Split Surface command.
  35. We specify in the tree the merging of surfaces and in the ‚ÄúExtrusion Surface: 2‚Äù model window as a clipping object. Create an operation.
  36. We specify in the tree the equidistant of the surface and in the model window "Surface extrusion: 2" as a clipping object. Run the command Trim the surface.
  37. We indicate in the model window ‚ÄúExtrusion surface: 2‚Äù and in the tree the merging of surfaces as a clipping object. If necessary, change direction, to remain the inner part. Create an operation.
  38. We indicate in the model window the remaining part of the extrusion surface and in the tree the surface equidistant as a clipping object. If necessary, change the direction so that there is a part between the stitching and equidistant. Run the command "Delete Faces".
  39. We indicate three inner faces (equidistants) on the top and four outer faces (stitches) on the bottom. Run the patch command.
  40. Specify the surface edges to form a patch outline. Run the command Surface sewing.
  41. We indicate in the tree the surface stitching, the surface equidistant, ‚ÄúExtrusion surface: 2‚Äù and the patch. Create an operation.
  42. Switch to the "Solid Modeling" set. Run the Round command. Set the radius of 11 mm, specify the edges, as in the figure. Create an operation.
  43. Set a radius of 6 mm, specify the edges, as in the figure. Create an operation.
  44. Switch to the method with a constant chord. Set a radius of 3 mm, specify two edges, as in the figure, the rest are allocated automatically, because the rounding is extended along the tangent edges. Run the shell command.
  45. Specify a flat face (the team can remove only flat faces). Set the shell thickness of 3 mm (shell thickness can not be more than the smallest radius in the model). Create an operation.
  46. It turned out such a body. It remains to supplement it with fasteners for internal structures and the rear.

Leaf model modeling process
It is not interesting to create a banal box from a sheet - everyone will figure it out anyway, and there are a lot of video lessons. Let's try to build from the sheet an analogue of the body, which was made by surface modeling in the last lesson. Completely repeat it will not work, but at least the basic form.

  1. Create a part
  2. For convenience, we will include the specialization sheet metal. Choose File - Specialization - Sheet Detail.
  3. The basic set has changed to "Sheet modeling".
  4. Run the autoline line command. Create a sketch in the ZX plane. We build the first point just above the origin.
  5. We build the second point at the origin, and the third is slightly to the right of the origin.
  6. Switch to arc. We build an arbitrary arc to the level of the middle of the first segment.
  7. Switch back to the segment. We build a vertical segment to the horizontal level of the beginning point of the construction.
  8. Launch the ‚ÄúAlignment‚Äù command. Align horizontally the start and end points of the autoline.
  9. Align horizontally the middle of the first segment and the second point of the arc.
  10. Run the command "Touch" Specify the arc and the last segment.
  11. Run the command "Avtorazmer." We build the dimensions as in the picture. Run the command Sheet body.
  12. We build a sheet body with a thickness of 2 mm, a radius of 4 mm, at a distance of 200 mm. Run the command "Fold".
  13. Specify the top edge of the side face. Switch the width setting type to ‚ÄúTwo indents‚Äù. Set the left margin to 4 mm. Establish a method for constructing the bend extension ‚ÄúTo the object‚Äù We specify as an object one of the vertices of the vertical edge. Set the radius of 4 mm. Create a new sketch.
  14. Specify the bend face as the plane to build the sketch.
  15. Run the command "Project an object" and project the external vertical and arcuate bend edges.
  16. We finish the sketch as in the picture. Run the command "Plate".
  17. In the settings you do not need to change anything. Run the command "Closing the corners."
  18. Specify the fold. Set the closure method "Closing with overlapping." Run the command "Fold".
  19. Specify the inner edge of the vertical face of the curved part of the sheet part. Set an angle of 180 degrees, and a radius of 0.001 mm, a length of 10 mm. This bend is needed for stiffness, it also allows you to remove the sharp edge. We press the button "Create".
  20. Specify the inner edge of the opposite vertical edge. Switch the width setting type to ‚ÄúTwo indents‚Äù. Set the left margin to 3 mm so that there is no intersection with the side fold. The remaining parameters do not change. Run the command "Sketch".
  21. Create a sketch on the side of the bend.
  22. We build a horizontal segment with dimensions as shown. Run the command "Burtik".
  23. Set the height of the collar 2 mm, radius - 2 mm and radius of the base of the collar - 2 mm. The resulting shoulder can be used to connect the case with the lid, it will also give bending stiffness. Run the command "Sketch".
  24. Specify the vertical edge of the curved part.
  25. We build three equal horizontal segments with dimensions as in the figure. Run the "Blinds" command.
  26. Set the height of the blinds 6 mm, width - 16 mm and the radius of the base of the blinds - 2 mm. Blinds are needed if we need air flow into the housing. Run the "Mirror Array" command.
  27. The type of objects in the array is indicated - ‚ÄúBodies or surfaces‚Äù, we specify the sheet body, and the Plane ZX is specified as the plane of symmetry. Create an operation, then exit the command (click the red cross), because we need to switch to the scanning mode.
  28. Run the "Expand" mode.
  29. Specify the horizontal part of the sheet body.
  30. A phantom of a sweep appeared. We press to create.
  31. Run the "Create a drawing by model" command.
  32. Select the orientation of the ‚ÄúSweep‚Äù model, set the scale to 1: 1, activate the ‚ÄúSweep‚Äù option, open the ‚ÄúLines‚Äù section and turn on the display of fold lines. , .
