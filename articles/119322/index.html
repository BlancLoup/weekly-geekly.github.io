Globalscale Technologies announced a developer kit based on the silent D2Plug minicomputer



A shelf of miniature low-power computers has arrived. In early May, the company Globalscale Technologies has posted the announcement of the release of the D2Plug developer kit.
D2Plug is a logical continuation of the line of similar products of the company: DreamPlug, Guruplug, SheevaPlug. Devices of this series are distinguished by low power consumption, relatively high performance (PC-like), small dimensions, low noise level (though not in all models), the presence of many different interfaces, as well as the software used (for example, based on Linux).

D2Plug is positioned as a single software and hardware platform for computing, graphics processing and multimedia.
The characteristics claimed support of displays with a resolution of up to 1080p (1920 x 1080). The board integrates high-speed network interfaces, I / O interfaces using USB interfaces. OSS (OSS) is used as software, - Ubuntu Linux 10.04 is initially flashed.
High reliability is achieved by the absence of moving parts. The same equipment can be used both as a server and as a workstation.
