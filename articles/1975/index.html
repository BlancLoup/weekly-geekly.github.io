Technorati supports Open ID

Technorati's blog search engine has been enhanced with Open ID support . This technology allows you to use a single URL identifier to confirm your identity on various Internet resources. That is, each person receives a name (and with it a reputation) once and for all, so that it can easily translate their reputation between sites.

Open ID pass-through authentication provides many benefits to its owner. For example, this technology is supported by the LiveJournal blog platform, Vox, Movable Type, and Wordpress, as well as Wikitravel and Zooomr.

It should be noted that the largest companies, such as Microsoft, Oracle and Sun, take part in the discussions around Open ID.
