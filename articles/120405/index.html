API: added new and deleted old

Google has done a spring cleanup (Spring cleaning for some of our APIs).

Unfortunately, on December 1, 2011, 11 APIs will be deleted, and until this date the number of requests will be limited:
  • Blog Search API .
  • Books Data API , Books JavaScript API ( new API , delete only the old).
  • Image Search API .
  • News Search API .
  • Patent Search API .
  • Safe Browsing API (remove only the old v1).
  • Translate API .
  • Transliterate API .
  • Video Search API .
  • Virtual Keyboard API .
7 APIs are deprecated, the date of removal is not defined:
  • Code Search API
  • Diacritize API
  • Feedburner APIs
  • Finance API
  • Power Meter API
  • Sidewiki api
  • Wave API
Added 7 new APIs:
  • Discovery service
  • Tasks API
  • Books API
  • Pagespeed Online API
  • Places API (now generally available)
  • Prediction API (now generally available)
  • Fusion Tables API (additional feeds)
