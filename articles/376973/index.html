Google is going to build a "glass utopia" in Mountain View



As previously reported, Google plans to build a new campus in Mountain View . Now the details of the project proposed for implementation have become known: the corporation is not going to modernize the existing buildings, instead a whole series of new buildings will be created. Danish architect Bjarke Ingels from Bjarke Ingels Group and Thomas Heatherwick from Heatherwick Studio are involved in the construction. The architects plan to build a series of modular glass buildings, the configuration of which can be changed in a matter of hours. Those who managed to get acquainted with the project, describe it as a "glass utopia."

Outwardly, all this will be similar to greenhouses enlarged many times, where the position of ceilings, walls and floors can be changed in a few hours, thanks to the provided structure features: automatic block cranes and robots. At present, according to Google representatives, this plan cannot be fully implemented, so there will be some static objects in the buildings, like staircases and rooms for recreation. However, all other design elements can be modified. The indoor microclimate will be controlled, so there will be no hot in new buildings.
