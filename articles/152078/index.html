Passbook earned in Russia

Passbook - organization and storage of air tickets, passes to events, tickets to the cinema and concerts, bonus cards, coupon codes, gift certificates and so on. The objectives of Passbook is not only to collect coupons, but also to promptly notify the user about everything related to coupons.



A few days ago in Russia earned a Passbook. If you go to Passbook and click on the App Store at the bottom, Apps for Passbok (RU) will open with 4 applications.

Apple provides developers with a complete set of APIs for creating Passbook cards. Given that many global companies are interested in introducing a new technology, many instructions for implementing the project can be found on the web. Apple divides Passbook cards into four types: boarding passes, tickets, store cards, and coupons. Each of these categories contains a strictly defined structure, but there is a possibility of customization. Cards should be well recognizable for users, so it is possible to change the background, use corporate colors and logo logos.

In fact, there is not much work for developers when creating cards for a Passbook. Each of them is actually an archive with the extension ".pkpass", which contains graphic and JSON files (text format for storing and transmitting data in a structured form), familiar to every iOS developer. In fact, the creation of Passbook cards is not a difficult task, provided that the company has a streamlined system for generating serial numbers.

The only difficulty the company may encounter when creating Passbook cards is the choice of a bar code format. Currently, only QR, Aztec and PDF417 are supported, since standard 1-D codes are poorly read from the device screen.
