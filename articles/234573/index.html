3. Windows Phone 8.1 Developer Preview - open Lumia 1520 in a new way!


Installing the Windows Phone 8.1 Developer Preview update on the Lumia 1520 is very simple:
1. Register as a developer at appstudio.windows.com/en-us?stay=1
2. Download the application Preview for Developers from the Windows Market
3. Login to it.
4. Restart the phone and download everything that appeared in the update section.
5. Rejoice!
