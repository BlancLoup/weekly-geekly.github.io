What's new in YouTrack 2018.4

Hello!

On the eve of the New Year, we decided to please you with another release of YouTrack.

In this post I will tell you what improvements are included in YouTrack 2018.4.



Project Templates


Any project in YouTrack can now be used as a template for new projects. After you mark an existing project as a template, it will appear among the available types on the project creation page.



Cloning project


Now you can clone any project. The new project inherits the parameters of the original, including the project team, access rights settings, custom fields, time tracking, workflows, integrations (except VCS) and Agile boards. This functionality is especially convenient in cases when you need to create several similar projects.



Time accounting report


The new report shows how users allocated their time spent on tasks in the form of a calendar table. Results for each user and task can be viewed daily, weekly, or monthly.



Aggregation options for reports

Time reporting improvements


New parameters in the report settings by time allow you to group work items by the main task, display comments on work items and estimate the total elapsed time for the main task.



LDAP group synchronization


Now you can synchronize groups from an LDAP directory with groups in YT. This configuration is available for Active Directory, LDAP, and Open LDAP authentication modules.

Update GitHub Authentication Module


The new parameter allows you to prevent users who are not in the specified GitHub organizations from entering.

Experimental functionality


Single Word Search Optimization


If this functionality is enabled, text search on one condition is preferred to words that exactly match the specified word form. If you enclose such a query in quotes, you will only receive exact matches with the specified word form.
