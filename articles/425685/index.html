Part of the Minecraft game code was transferred to Open Source by Microsoft


Illustration opensource.com

Part of the Minecraft: Java Edition source code was legally published on GitHub under a free license by Microsoft.

The code was opened under the MIT license and is the Brigadier and DataFixerUpper Java libraries. They allow you to parse, send and process user commands, as well as process data for new versions of the game.

Previously, the main way to see this code for enthusiasts was ‚Äúdecompiling‚Äù - converting bytecode back to human-readable Java code. This was enough for many, but, of course, the ‚Äúsource texts‚Äù obtained in this way were somewhat harder for perception than the original sources.

In the future, it is planned to ‚Äúrelease‚Äù other universal Minecraft subsystems, which may be useful for developing other games and simplifying the work of mod-makers. Probably, at the next stage, the source texts of the completely rewritten Blaze3D rendering engine, planned for release in Minecraft version 1.14, will be opened.

Brigadier



Brigadier is a tool for parsing and dispatching commands. According to the developers, the library uses user input data in Minecraft, and turns them into a function that the game will perform. From the user's point of view, the component looks like a standard console with command syntax highlighting and auto-completion of input.

DataFixerUpper



DataFixerUpper includes tools for incremental assembly, merging and optimization of data conversion operations necessary to transfer existing game data to new versions of Minecraft.
