A selection of useful materials on Azure

We have accumulated a lot of free materials on working with the cloud, so we decided to share with you the latest ones. Under the cat you will find a description of 6 courses and 12 books / guides on learning to work with Azure.



Free Azure Courses


In this section, we will tell you about Russian-language online courses on working with the Azure cloud of various levels of complexity. Please note that on the website you can find 5 additional courses on this topic.

Azure Basics
This is the first course of the series. You will learn about the basic concepts and tools of Azure, learn to work with the classic and new portals, Azure Powershell, Azure CLI. Course presenter Corey Hines will demonstrate basic Azure services, such as virtual machines, virtual networks, storage, Azure Active Directory, and application services. Take a course ‚Üí


Microsoft Azure virtual machines
Virtual machines are the foundation of Microsoft Azure, they support IaaS implementation on platforms. It is important that any type of virtual machine can be deployed in an environment from both the Azure portal and Azure PowerShell. Corey Hines, the founder and architect of Learn on Demand, will guide you through the full life cycle of a virtual machine. Take a course ‚Üí


Microsoft Azure Virtual Networks
Virtual network management, addressing and VPN connections setup are the main tasks of this course. Cory Heins will explain how to manage network traffic and load balancing using Azure DNS, Load Balancer, Azure Traffic Manager and Application Gateway. Take a course ‚Üí


Microsoft Azure Storage Service
Storage is an integral part of any Azure service. The course covers storage for structured and unstructured data (BLOB objects), as well as various services that use Azure storage. It also focuses on daily management tasks such as backup, site recovery and storage analytics. Take a course ‚Üí


Identity in Microsoft Azure
One of the foundations of any infrastructure is the Active Directory Identity Service. This course deals with Azure Active Directory, the integration of Azure Active Directory with on-site Active Directory. Module 3 will reveal the secrets of expanding on-site Active Directory capabilities through Azure AD Premium. Take a course ‚Üí


Azure Workload Automation
Having mastered the skills of manually managing Azure services, you can move on to learning automation. The development of ARM templates, automatic runbook actions, and automatic configuration of virtual machines will be demonstrated by Corey Hines. Take a course ‚Üí


Azure Databases
You will learn about the different types of database services available in Azure, as well as tools for preparing, monitoring and managing these services. The course focuses on the SQL Azure Database, although other database services, such as DocumentDB, are also covered. Take a course ‚Üí
