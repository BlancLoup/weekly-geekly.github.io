New course in web development for beginners in Russian MSDN



An updated web development course for beginners using ASP.NET and the free WebMatrix development environment has been published in Russian MSDN . The new course consists of 16 articles in Russian, which will give enough knowledge to create their first sites and gain web development skills.

Course content:
  • Chapter 1. Getting Started with WebMatrix and ASP.NET Web Pages
  • Chapter 2. Using ASP.NET Web Page Code
  • Chapter 3. More complex example on ASP.NET
  • Chapter 4. Introduction to the Razor syntax. Programming Recommendations
  • Chapter 5. Introduction to Razor syntax, server code, and ASP.NET
  • Chapter 6. Introduction to the Razor syntax. Error processing
  • Chapter 7. Working with Forms
  • Chapter 8. Working with Data
  • Chapter 9. Data Display
  • Chapter 10. Image Handling
  • Chapter 11. Work with video
  • Chapter 12. Caching and Performance Optimization
  • Chapter 13. Traffic and attendance analysis
  • Chapter 14. Webpage Optimization for Search Engines
  • Chapter 15. Using Package Manager
  • Part 16. Developing ASP.NET Webpages and Razor in Visual Studio
In addition, for PHP and ASP.NET developers in Russian MSDN published additional materials on the professional use of WebMatrix including to work with popular CMS:
  • Quick installation and publishing to a WordPress project server using WebMatrix
  • Quick installation and publication on the server of a CRM project based on SugarCRM
  • Quick installation and publishing to a server using a Drupal-based WebMatrix project
  • Quick installation and publishing to the server project based on Joomla using WebMatrix
  • Video: Rapid Community Site Creation with WebMatrix and .NET Forge CMS
  • Video: Quickly download, install, configure, and publish WordPress using WebMatrix
  • Video: Quick installation and configuration of a SugarCRM-based project using WebMatrix
  • Video: quick installation and setup of a Drupal-based project using WebMatrix
  • Page Debugger for WebMatrix
  • ASP.NET API Quick Reference
  • Using VisualBasic on ASP.NET Web Pages
  • WebMatrix: working with mobile clients definition package from 51Degrees.mobi
