It is not a secret to anyone that in the modern world mobile technologies have completely turned the idea of ‚Äã‚Äãpeople about where and how to work. High-performance wireless networks have long ceased to be a luxury for companies and become a necessity. Despite all this, often the wireless network stands apart from the wired campus network with its own equipment, management, its own settings and policies.

It's time to change the approach and simplify the work, deployment and administration of wireless networks!

The HP Unified Wired and Wireless Network solution , part of the HP FlexNetwork architecture , is intended to help .
First of all, this solution includes a new line of equipment, namely, wireless access controllers / switches.

Let's start with the HP 830 Unified Wired-WLAN Switch :


The HP 830 Unified Wired-WLAN Switch series combines a wireless access controller and a gigabit PoE + switch.

This implementation is convenient for the organization of wireless networks of small and medium scales, as it allows you to make an access network using a single device. Such a network will be scalable, since the number of access points supported by the controller can be extended with licenses to 60 access points to the controller.

As a result, we get:
‚Ä¢ Unified management and policies for LAN and WLAN users;
‚Ä¢ Unified management with IMC;
‚Ä¢ Built-in portal authentication server and IEEE 802.1X;
‚Ä¢ Switches with 8 and 24 GbE PoE + ports;
‚Ä¢ Wi-Fi Clear Connect technology for optimizing wireless networks.

Learn more about the HP 830 Unified Wired-WLAN Switch .

For larger solutions, a more powerful HP 850 Unified Wired-WLAN Appliance controller, which has 8 Gigabit combo ports and 2 10G interfaces, is suitable.


This device provides support for up to 512 access points to the controller with a centralized bandwidth of 10G. A distinctive feature of HP 850 controllers and older is the support of 1 + 1 redundancy controllers, with the detection time of the main controller failing in terms of milliseconds. The presence of IEEE 802.1X hot-backup functionality, which synchronizes the 802.1X state along with information about the connected clients between the controllers, allows you to save client connections when the primary controller fails. Also, do not forget about the possibility of redundant power supplies.

For more information, see the HP 850 Unified Wired-WLAN Appliance .

The next in line is an even more powerful HP 870 Unified Wired-WLAN Appliance controller, which has 40G bandwidth and provides 1536 access points.


The device has 12 copper and 12 SFP gigabit interfaces, as well as 4 SFP + interfaces. The controller also features redundant power supplies and IEEE 802.1X hot-backup technology, which makes it possible to ensure uninterrupted operation of all access points as a central controller of a wireless LAN for large companies and large-scale implementations.

Read more here HP 870 Unified Wired-WLAN Appliance .

For large solutions with HP wired network, the controller is also implemented as a service card HP 10500/7500 20G Unified Wired-WLAN Module .


As the name implies, this module is suitable for the chassis switches of the HP 10500 and HP 7500 lines. The bandwidth of processing the traffic when deploying the centralized architecture of this module is 20G. One module provides support for up to 1024 access points. Increasing the number of such modules in the chassis, you can get 11264 access points supported by a single chassis. Support for the ability to backup such modules, as well as 802.1X hot-backup functionality.

In the most detailed way in HP 10500/7500 20G Unified Wired-WLAN Module .
