“Simple Business”, version 1.5.6.0 - cases for employees and many other convenient functions for the effective management of the organization

The company “Simple. Ru” presents to its customers a new release of the program under the number 1.5.6.0 ( www.prostoy.ru ). Version 1.5.6.0 has been expanded with new functionalities that allow users to efficiently organize working time records, work with e-mail and phone calls, save and restore organization data.
Now in the program "Simple business" appeared function "Automatic time tracking."
The administrator of the program or the owner of the organization can control the time for the assignments to be completed by employees. You can activate the function in the organization window by setting the “Maintain automatic case management” flag on the “Organization” tab.



When the automatic time tracking option is turned on, employees who do not perform any tasks will be regularly informed about the need to take on any task. With an interval of 10 minutes, a free employee will receive a message stating that he has no active business.



You can create a new case on the My Affairs tab of the main program window (using the Create command in the context menu of the bookmark). You can also turn any comment from a task into a business by selecting the “Convert to business” command from the task's context menu.



Starting to perform the assigned case, the employee must start the case for execution. This procedure can be performed on the “My affairs” tab, by calling the context menu of the case and selecting the “Run” command in it. A similar function is performed by the launch button, located to the left of the case name (in the form of the “play” button). Time to do things is counted in minutes.



Working in the “Simple Business” program, an employee has the opportunity to assign affairs not only to himself, but also to other employees of the organization. To perform this operation, it is enough to select the employee of the organization responsible for its execution in the case properties window.



During the execution of the case by an employee in the contacts tree, as well as in the chat window, a record is displayed about what business he is currently doing.



View the status of the execution of affairs by employees available to the owner / administrator of the organization. Also, this right can be used by employees who have the appropriate privileges set in the properties of the organization on the “Access” tab. Other employees can see the status of only those users who are in the group "My friends".



The functions of creating and executing cases allow you to quickly receive information about the employment of employees and maximize the work of the organization’s staff.

In the new version of the program “Simple Business”, changes were made to the procedure for finding users - now in the “Search” window on the “People” tab for each record, the date of registration of each user in the system is displayed.



New features added to the contact window. The “Mail” tab has been added to it, where you can see the records of mail correspondence with a contact. Another innovation in the add to task menu (“+” button): the ability to add bookmarks with task names appeared by selecting the Add Task function from the dropdown menu. On the resulting tab will be placed all the comments to the selected task. If necessary, the employee can use the “Comment” and “Open Task” functions located in the left pane of the contact window.



Also in the contact window, it became possible to make a call or send an SMS - the corresponding functions are located in the menu on the left pane of the contact window.



Another new feature of version 1.5.6.0 of the “Simple Business” system is the ability to recover deleted tasks. Any task created in the program is assigned a unique ID (identification number). In the left part of the "Restore Tasks" window appeared the function "Restore by ID". To restore a specific task, simply enter its identifier in the field and click the “OK” button.



Use the updated version of the “Simple Business” system ( www.prostoy.ru ), and you will be able to appreciate the advantages and convenience of using new program features.
