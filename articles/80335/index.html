Plastic Logic Introduces Que E-Reader New Ebook

On Thursday, Plastic Logic showed the final version of its Que E-Reader.
The device, as expected, is a full-screen e-book with a touchscreen and an interface designed to quickly navigate a document or book.
An innovation is the smooth surface of the device, free from buttons. Instead, the device is fully controlled via a touch interface using gestures.

Plastic Logic will sell a model with 4Gb of memory, Wi-Fi and a 11.6-inch display for $ 649, and the version with double memory and 3G in addition to Wi-Fi will sell for $ 799. The start of sales is scheduled for April.
