Google opened source code of Chrome OS



Google has opened the source code for Chrome OS. Let me remind you that the presentation of the system took place last night , and the OS is intended primarily for use on mobile devices that are constantly connected to the Internet - netbooks, for example. The core of Chrome OS is the Chrome browser, and the system itself can run only web applications.

Chrome OS chip - speed. The OS loads in a maximum of seven seconds and is supposedly faster than the latest version of the Chrome browser under Windows or Mac OS. At the same time, most of the OS data is stored on servers on the web. The device itself stores only temporary files that are used to speed up the system.
