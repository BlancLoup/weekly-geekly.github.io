Github for Android



The Github team is happy to announce the release of the Github mobile app for Android, which is now available in the Google Play catalog and in the corresponding repository (applications for new features and bug reports are accepted there).

The Android program displays status updates (issues), gists, and news feeds from all friends and repositories. View the source from the phone does not work.
