Windows Store application directory officially opened with 7873 applications



Yes, the Windows Store application directory is officially launched. It opened almost simultaneously with the announcement of Windows 8. It is worth noting that the developers were concerned about the localization of the application catalog in different languages, and translated the Windows Store into 109 languages.

In the Windows Store already at the opening added a lot of applications. In general, the head of Microsoft Windows was right when he said that there would be more applications in this directory when opening than in any other directory at the launch stage.

As already mentioned in the title, 7873 applications for Windows 8 were placed on Windows Store. At the same time, 88% of these applications are free. Other applications include Skype for Windows 8, Netflix and, of course, Angry Birds. 3367 developers worked on these applications.

This directory is available only from the Windows 8 environment, which is logical, since all applications in the directory are designed specifically for this operating system. Microsoft is positioning its catalog as a download center for a large number of Windows 8 devices (from desktop PCs to tablets).

This application catalog has already been seen by many users and developers who set themselves preliminary releases of Windows 8. For example, users who tried Windows 8 RTM release could also try some of the applications from the catalog that were available at that time.

Like other similar directories, Windows Store has its own security system, which allows you to check the downloadable application for malicious software or other similar surprises. There is a Windows Store and quite tough rules that are worth familiarizing each developer.

When purchasing (or simply downloading) applications from this directory, the user has the right to install them on several PCs (up to 5). You can evaluate the application and leave comments for them. Applications can be sorted by category, price or rating.
