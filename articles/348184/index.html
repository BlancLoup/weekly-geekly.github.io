Wine 3.0 and many goodies

On Habr√©, the third version of Wine, an open Win32 API implementation, passed unnoticed. It‚Äôs hard to find another open source project, with the exception of the kernel, which would mean so much to users of Linux, MacOS, FreeBSD and other POSIX-compatible OSs. Every success of the Wine developers leads to the fact that many new games and applications are available for Linux users.



In this regard, Wine 3.0 clearly did not disappoint: support for Direct3D 10 and 11, the Android graphics driver, the flow of Direct3D commands, the AES cryptographic standard, and a new level of support for Direct Write and Direct2D are the main results. Following the results of more than 6000 made changes, many interesting things happened, including the transition of the default version from Windows XP to Windows 7.


  • 4586 applications with higher platinum status. Work without complaints.
  • 3918 applications with gold status. With minimal settings and external DLLs work without complaints.

Even before the release of the previous significant version, the Wine developers announced the transition to a new release schedule. Instead of 1.xy / 2.xy, versions 2.0, 3.0, 4.0, etc. are released once a year.


Direct3d


The new version implements a significant part of the properties of Direct3D 10 and 11.


  • Computational and mosaic or tessellation shaders.
  • Stream output.
  • Request format capabilities.
  • Unordered browsing counters.
  • Structurally defined buffers and byte addressing.
  • Production levels mip-map.
  • Instructions and interpolation modifiers for shaders 4th and 5th models.
  • Cubic texture.
  • Typeless resources.
  • Multi-level processing of 3-dimensional textures and texture arrays.
  • Conservative inference and bias in favor of depth.
  • AppendStructuredBuffer , ConsumeStructuredBuffer .

Support for the main OpenGL contexts in Direct3D has been further developed and is now being used in systems with Intel and AMD video cards. The database of recognizable graphic cards has been expanded.


Graphics


  • Direct2D now supports drawing contours and calculating the boundaries of geometric objects, support for brushes with linear and radial gradients. Added compatibility with GDI .
  • The list of extensions OpenGL updated to version 4.6.
  • The GLU system library (OpenGL Utility Library) is only required when using the Nurbs handler, and the rest of the GLU functions GLU now implemented inside Wine.
  • New GdiPlus : processing of graphical operations and reproduction of most types of WMF (Windows metafile) specific to GdiPus .
  • WindowsCodecs supports encoding of image formats, including the palette.

On Reddit there is a fixed case of installing Photoshop CC 2018 64bit on Linux, using Wine.


Core


  • By default, the program uses Windows 7.
  • The full-fledged semantics of named pipes, including channels with messaging mode, has been released.
  • Added support for PIE (Position Independent Executables) for Wine executables and launching external binaries.
  • Devices with serial and parallel ports are automatically created. Ports can be redefined via HKLM\Software\Wine\Ports .
  • The secure search DLL is enabled by default. It can be disabled via the HKLM\System\CurrentControlSet\Control\Session Manager key by setting the SafeDllSearchMode variable to 0.
  • The mode of safe search for processes is turned off by default. It can be enabled via the HKLM\System\CurrentControlSet\Control\Session Manager key by setting the SafeProcessSearchMode variable to 1.
  • Improved asynchronous I / O performance.
  • 64-bit platforms were able to allocate areas of virtual memory of arbitrarily large size.
  • Memory write operations that simultaneously use the same file I / O buffers work correctly.

User interface


  • For new monitors with high DPI, there are new and better mouse cursors, Shell Explorer , RichEdit and all major dialogs.
  • Screen DPI can be redefined via the value of the LogPixels parameter in the HKEY_CURRENT_USER\Control Panel\Desktop branch.
  • Desktop mode can support higher resolutions with various aspects of aspect ratio.
  • The TWAIN library has gained support for the scan source selection dialog.
  • Device-independent bitmaps and WMF structures can be stored in OLE Data Cache .
  • TaskDialog , Internal User Interface in MSI , drawing themes through double buffering.
  • Fonts are compatible with the innovations of FreeType version 2.8.1.
  • Tables of characters translated into Unicode standard 10.0.0
  • Contextual replacement of glyphs in Uniscribe, Nepali and Bangla (India) keyboard layouts.

Android


  • Wine can now be collected as a regular APK package for the Android mobile platform.
  • Written full graphics driver. Due to the limitations of the Android API window interface, only full-screen operation is supported.
  • Full audio driver.
  • Implemented support for OpenGL ES API .

Desktop Integration


  • In the .desktop files, the StartupWMClass parameter is StartupWMClass to bind to the Windows executable file.
  • The Xfixes library keeps Xfixes of changes in the clipboard made by X11 applications.
  • The RichEdit component can insert WMF .
  • Added protocol Program Manager DDE .
  • The HID discovery service of PnP devices is enabled by default.
  • Version 4 of the system tray notification protocol is supported on macOS.

Networks and the Internet


  • Web Services adds support for TCP/UDP receivers, as well as support for the .NET Message Format Protocol and the .NET Binary Format . Implemented support for receiving messages in asynchronous mode.
  • HTML: reworked event handler code, added support for many new standard APIs, the browser has learned to understand MHTML .
  • .NET better supports embedded HTML documents.
  • WinHTTP correctly interprets cookie attributes.

Cryptography


  • Added support for Rijndael, aka AES - symmetric block encryption algorithm.
  • The cryptographic hash functions are implemented in-house, instead of using GNU TLS.
  • The additional programs Gecko and Mono are checked using the SHA56 hashing SHA56 .
  • Microsoft's root certificate of 2011 is included in the list of well-known Microsoft certificates.

D3DX 9


  • New and better support for prescalers.
  • ID3DXEffectStateManager is supported - a user interface that allows the user to set the state of the device from the effect.
  • An effect pool is supported for sharing parameters.

Directwrite


  • Truncation signs of the closing line are supported in both character and text modes.
  • Improved line feed mode.
  • Support for uniform and proportional line spacing.
  • Inclined and bold playback in bitmap mode.
  • In applications, support for loading font files into RAM is available.
  • The cache is safe for multiple threads.
  • Overhanging metrics for labeling implemented.

ARM platform


  • ABI floating-point defaults to softfp for compatibility with Windows executables.
  • Added support for preloading, as well as the use of debugging relay mode.

And other comrades


In this extensive list of new features far from being exhausted. But there is no need to go into further details, then the salad of the remaining interesting innovations.


  • The wineinstall utility wineinstall deprecated and has been removed.
  • RegEdit always shows 64-bit registry views on 64-bit prefixes.
  • There was a folder ProgramData .
  • ODBC got support for installing the SQL driver.
  • Kerberos Authentication Package uses the krb5 library.

Third Wine in Numbers


The release consists of 5,170,395 lines of code in 7,373 files. For all the time, 121,944 updates were made by 1,480 programmers.



From the second version, 6,026 updates occurred, in which 448,982 were added and 216,298 lines of code were removed.



Wine 3.1


While I was writing about Wine 3.0, we quickly rolled out a more recent version 3.1. Here are the main features of the latest stable version.


  • Kerberos authentication support.
  • The ability to redirect the Window class to Common Controls 6 .
  • For X11, ARGB visualization support has been implemented.
  • DOSBox is now required to run DOS applications.
