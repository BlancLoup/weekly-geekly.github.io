http: // (Any_word) .jpg.to

The jpg.to website has begun its work, with the help of which it is convenient to receive illustrations on the fly on the keyword. Specify the word in the URL in any language - and get the desired picture. Pictures are taken from Google Image Search, in the URL, you can use the Cyrillic alphabet. For example, kittens.jpg.to .

This can be useful for fast posting pictures in chat / email, as well as for automatic illustration of the content. You can make a script that will load the pictures on the context of the page or on the text while the user is typing. Jpg.to is a minimalist alternative to the Google API , which is scheduled for closure.

The author promises in the near future to add additional functionality, including the parsing of image search terms:
  • Size: keyword.jpg.to/large, keyword.jpg.to/medium
  • color range: keyword.jpg.to/red, keyword.jpg.to/green, keyword.jpg.to/white
  • type of image photo / clipart: keyword.jpg.to/photo, keyword.jpg.to/clipart
  • choose a random image (keyword.jpg.to/random.jpg) or by number in the search results (keyword.jpg.to/301.jpg)
  • information about the author and the ability to remove pictures from the issue at the request of the author
