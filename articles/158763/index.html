ImageCMS 4.0b Release - New Interface and Functional Flexibility

For the last six weeks, our team has completely devoted to translating the wishes of ImageCMS users. All these wishes boil down to the fact that managing an online store becomes a concise, flexible and functional process. The 4.0b ImageCMS and ImageCMS Shop release is our implementation of these solutions.
This version is key to the development of the system, since one correct vector has been chosen for the subsequent development of new features, modules and templates, and the setting of a flexible update system.

The adaptability of the design of the administrative part has become for us one of the main projections of development - today we cannot ignore this issue in the context of usability of the interface. The ability to work with the administrative part of the site or shop based on mobile devices is becoming increasingly popular for the modern person.

Another direction actively worked out by our team in this release is SEO goals. Correct, structured HTML code helps to optimize the process of promoting your website and online store.

Let's go over the key features of the new version of the system.


1. Bootstrap & Jquery


To build the administrative part of the interfaces used Bootstrap - a framework from Twitter . This tool allows you to quickly and conveniently create templates for modules of the ImageCMS system. In addition, the process of adding and customizing existing solutions is as easy as creating your own solutions. The framework supports the features of HTML5, CSS3.
In previous releases, up to version 4.0b, we used mootools and mochaui to build the UI. This greatly limited us in terms of flexibility and scaling. Changing the underlying principles of the mochaui user interface library was quite a laborious task.
Bootstrap is a powerful designer tool that will help you quickly respond to your wishes and comments on further improving the interface.
Along with Bootstrap, there is also support for the Jquery library. We write less - do more! Create your own modules, make adjustments to the existing interface using Twitter Bootstrap and Jquery.


2. Changes in the interface


  • added more autocomplete fields and convenient filters to all sites, the search in which could be difficult due to the large amount of content.
  • worked through the product list view - now there is a quick access to the filter (in the previous version it was not available at all). Make the necessary changes even faster, set statuses and adjust prices directly from the list. Manipulations are also available for many products - just tick and apply an action from the top menu.

3. Orders


We have added tools that make the process of order management even easier. Apply filters by date, status, customer, product and order amount. Also use information tools such as a quick review of the contents of the order, and make changes to orders already completed.


4. CSV-import of goods


The new product database approach for ImageCMS Shop is via CSV. Now we suggest using one of three slots for storing files.

Files are checked and stored in the backups folder. The time of instantiation is indicated next to each of them - you can return to each file at any time if necessary.
Based on the selected slot, a string of fields is formed for unloading to the storefront. From the indicated slot, the first line is read, on the basis of which it is now possible to visually establish correspondences. Simply select from the list of available values ‚Äã‚Äãand start the import.
