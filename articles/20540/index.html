Open ATI 6.8.0 driver release released

After a long test, a new stable release of X.Org drivers for ATI video cards - xf86-video-ati 6.8.0 has been released.

Major innovations:
>> Translation of the driver to use the new libpciaccess X.Org library for accessing PCI devices;
>> Support for radeon r5xx, rs6xx, r6xx chips using AtomBIOS parser;
>> The operation of the output in the PAL mode for cards with TV-out has been adjusted;
>> Initial support for hardware acceleration (EXA Render Accel) for ATI R3x / 4x video cards. Currently, only acceleration of moving and rotating objects is implemented.
>> Improved driver interaction with video card BIOS.
