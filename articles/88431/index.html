Opera 10.51 for Windows


Today released an updated version of the browser Opera 10.51 for Windows. In this release, the vulnerability found a little earlier in version 10.50 is closed, and a large number of errors have been fixed - thanks to users for actively sending messages about found bugs. Full list of changes can be found here .

In addition, the optimization of the Carakan engine was carried out, due to which it was possible to increase the processing speed of JavaScript by another 20% compared to the browser version number 10.50. I can also inform you that the auto-update system has been re-enabled, which will allow users of the previous version to carry out this procedure in the usual way. You can also download Opera 10.51 for Windows from the official website . Versions for Mac and Linux will be released later.
