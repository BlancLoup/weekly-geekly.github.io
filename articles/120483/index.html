Optimize with Optimus

This post is dedicated to the Optimus application, implemented in the latest models of the Qosmio lineup, which will soon appear in Russia.

The application uses software and hardware innovations that instantly and continuously provide high performance and long battery life in automatic mode.
The principle of operation of Optimus is quite simple.
Most laptops are equipped with the function of switching between the integrated graphics processor, which is used when working with heavy graphics applications, and the discrete processor, which is turned on when working with lighter programs - this saves battery power. Optimus technology performs this function automatically, instantly switching between the discrete NVIDIA GPU, if you need high performance for watching videos and running 3D games, and a low-power embedded processor if the user has accessed office applications, the Internet, or mail.



Optimus is ideal for gamers, because when you start the game automatically turns on the graphics processor. Users can watch movies in HD-quality, use the Internet or run the most demanding 3D games - at the same time, performance will always be at its best, and the battery will work for a long time.
