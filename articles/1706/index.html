W3C fixed typos in XML and prepares new standards

The W3C has published updated versions of official XML 1.0 / 1.1 specifications , as well as Namespaces in XML 1.0 / 1.1 . In the new editions of the most important network standards, all known misprints were corrected, and the text was carefully edited from a grammatical point of view, especially in those places that allowed double interpretation.

W3C experts are carefully working on refining and extending XML standards, which are extremely important for the development of the web. By the end of this year, the consortium is also planning to publish recommendations for the XML Query 1.0 standard and XSLT 2.0. The structure of the XML Schema, which is actively used in SOAP-based web services, is currently being revised. A new Efficient XML Interchange Working Group has also been established to develop standards that are designed to increase the efficiency of storing, transmitting and processing XML documents, as well as to describe additional XML features that will emerge in the future, such as streaming.
