KDE 4.3 preview

Soon the release of the next version - KDE 4.3 I decided to make a small review of this product.
To start a little video:


Main innovations:
- Updated graphic effects



- PolicyKit integration. PolicyKit is a set of development tools for controlling system privileges in Unix-like operating systems.
PolicyKit-KDE includes an authorization manager and an authentication client, they look like this:



- Integration of geolocation services, the location will be determined either by a built-in GPS-receiver, or by IP.
- Updated runner

- Many updates in Plasma

- Improved hardware integration with Solid and Phonn libraries

And smaller:
- Old system settings mode returned

- New system tray

- Absolutely completely changeable appearance, a combination of modern hardware capabilities, Oxygen graphic themes and an updated Plasma shell


Google widgets

- Improved Qt development platform using C ++, Python, Ruby and other programming languages
- Many minor changes in applications and services, many bug fixes
Improved Kopete and Skype integration

Updated Dophin


As a result, I would like to say that this version of KDE looks pretty decent.
