Invites to Team Foundation Service Preview


In the framework of the recent Meet Windows Azure events, at which the new buns of the Microsoft cloud platform were presented, people really asked for invitations for Team Foundation Service . And Brian Harry in his blog hurried to please everyone.

Actually, reusable invite code - bharry-invite . Any restrictions on its duration and number of activations are not specified.

To use the invite code, you should follow the link tfspreview.com and click Create Account. Naturally, you will need a Windows Live ID.

UPD: Wow . The limit of the invite code has already been exhausted.
UPD 2012-06-13: TFS Preview - registration is open for everyone.
