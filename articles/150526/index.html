Google launches new ad format for Google Maps



In order to attract developers to Google Maps, Google is trying to use different "gingerbread". Now Google representatives have announced their intention to launch a new advertising format that will allow developers to earn money on Google Maps. Advertisements of the new system will be similar to ordinary adsense ads (contextual link blocks), but it will all work differently.

According to the developers, the ads will show a large number of topics that are relevant to the content of the page. When a user clicks on one of these topics / headings, he will go not to the advertiser's site, as in the case of AdSense, but to another page with relevant advertisements. This page is similar to the parking page of the domain.



In addition, the corporation allows third-party developers on Google Maps to change the appearance of advertisements to match the style of the card itself (colors, links, text, borders). This will allow you to set the relevant color scheme that will correspond to the page on which the advertisement is displayed.



The corporation has taken this step in order to attract additional developers, and more successfully compete with the systems OpenStreetMap and Bing Maps, which are gaining popularity.
