Atari 2600 data collection for reinforcement training


Website atarigrandchallenge.com , designed to collect data

For my master's work, I want to collect a collection of replays of games for the Atari 2600, played by people. In the future, I will use it to train bots and lay out all the collected data in open access so that everyone can use them for their research.

For this purpose, the site created atarigrandchallenge.com . In fact, the site is a wrapper for the JavaScript-emulator Javatari, which for each frame of the game records all keystrokes, as well as points in the game, number of lives and other information about the game (for example, is the frame the last frame of the current game).

Based on this information, the sequence of frames of the original game will be restored. The bot will learn the game based only on the pixels of the screen and keystrokes.

If you want to help research, just go to the site and play a couple of games. To play you need a keyboard. The emulator does not work on mobile devices.
