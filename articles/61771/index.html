NeverMap "Russia - the birthplace of elephants"

This rubber stamp in the title is not accidental.
I want to introduce you to the geo-service NeverMap for working with notes Evernote , whose symbol is an elephant. And for Russia, the service is special in that it is built on the Yandex.Maps engine, and shows Russian cities as well as possible.

You can read in the Evernote company's blog about the geographical capabilities of the service.
Our service is an excellent addition and assistant for organizing and working with notes using recorded geo-information.

Come and try it!
In short, you can search and look at the map notes:
  • Made in a specific place, for example, all notes from Italy
  • Found for key phrases, for example, all picnic spots
  • Found on key phrases and recorded in a specific place, for example, all cafes in Paris
  • From a specific notebook, for example, all business proposals
  • From saved search query

It is interesting to note that less than a week passed from the arrival of the idea to the implementation and launch of the project. ;)
This shows the power of open protocols, data access through the API and the effectiveness of modern development tools.
From the technical details:
  • ASP.NET MVC 1.0 Platform
  • IIS 7.0 server
  • Great library for working with OAuth - DotNetOpenAuth
  • Map display and geo-coding - Yandex.Maps
  • Initial determination of your location via Google Gears (W3C GeoLocation implementation)
  • Evernote API is based on the Thrift framework (came out of facebook), allowing you to create services and clients for them, regardless of the programming language
