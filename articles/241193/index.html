Founder of the Raspberry Pi Foundation has demonstrated the display for the "raspberry"



Eben Upton, founder of the Raspberry Pi Foundation, has recently demonstrated a new piece of hardware for the raspberry. This is a touchscreen display that will complement the ecosystem of accessories and additional elements for the Raspberry Pi.

According to Upton, now on the basis of the Raspberry Pi you can create even more devices than before. The very first thing that comes to mind is a tablet. True, the diagonal of the display is not so great - only 7 inches, but quite a lot of tablet PCs with such a diagonal are now on the market. The plates are supposed to be called Pi-Pads.

Since the display was created specifically for the Raspberry Pi, it connects very easily, without additional gestures.
