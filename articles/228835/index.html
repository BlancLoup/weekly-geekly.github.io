Google has published a preview version of the source code Android L for the Nexus line



Not so long ago, Google introduced the beta version of Android L for developers. The release of this OS is scheduled for autumn, and, according to some information, there are really a lot of innovations in it.

This is an updated interface, a Material design, and a new notification system, plus the optimization of the energy consumption of a device with an Android L. In general, there is a lot of new innovation.

Recently, the corporation posted in the repository a preview version of the original when Android L for Nexus 4 and Nexus 5 smartphones, plus Nexus 7 tablets (2012 and 2013 release, with support for Wi-Fi, 3G and LTE) and Nexus 10.

Unfortunately, neither the kernel source nor the binary packages exist yet, there are updates for some GPLs. So custom firmware based on the laid out code to create, until the full release of the source code, will not work. Nevertheless, all this is quite useful for developers, it is still possible to get acquainted with some details.

In addition, you can expect the imminent appearance of the new version of Android OS for the above devices. Well, the source code is available here:

Nexus 4 - https://android.googlesource.com/device/lge/mako/+/l-preview
Nexus 5 - https://android.googlesource.com/device/lge/hammerhead/+/l-preview

Nexus 7 (2012 GSM) - https://android.googlesource.com/device/asus/tilapia/+/l-preview
Nexus 7 (2013 Wi-Fi) - https://android.googlesource.com/device/asus/flo/+/l-preview
