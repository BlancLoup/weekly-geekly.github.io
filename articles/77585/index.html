Red Hat introduced the release of the platform MRG 1.2

Red Hat introduced a new release of the MRG 1.2 platform (Messaging, Real Time and Grid) to create solutions for tasks with guaranteed runtime, organize messaging between business applications and create a distributed computing environment.

The new version adds support for the deployment of Grid-environments based on the KVM virtualization system integrated in Red Hat Enterprise Linux 5.4. Another set of changes is related to optimizing the performance of messaging organization code and realtime extensions. For example, testing showed the ability to process on one system more than 1.5 million messages per second, with guaranteed responsiveness for 40 microseconds. In addition, the package includes a new utility rteval, which helps to track hardware problems leading to a decrease in responsiveness (latency).

The MRG infrastructure consists of basic components:
  • "MRG Messaging" - for the organization of fast and reliable messaging between business applications or web-services. The solution is based on the open protocol AMQP;

  • "MRG Grid" - the creation of a grid environment for distributed solutions of diverse problems. The development is based on the open source Condor project. Network creation based on local and remote machines, virtual and cloud-based environments is supported, based on the use of processor idle time on ordinary workstations;
  • Means for centralized and unified configuration management of a set of machines;
  • High Performance Computing (HPC) - the organization of high-performance computing, with full support for the development of parallel algorithms in Java, JMS,. NET, C ++, Ruby and Python.
