1. Summary of the proposal
This document formulates ideas for the development of the DVR, which differs from most commercially available devices in that it has an external (remote) control - a special radio unit with buttons for mounting on the steering wheel, as well as the ability to control using a smartphone. The DVR is a device that records audio-visual information on the built-in memory or external media, with the ability to remotely control the recording using a smartphone.

The mode of broadcasting to the Network allows you to transfer images from the geo-coordinate from the front camera of the device to the web server with a preset delay (2-5 minutes).
