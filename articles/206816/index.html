AWS announced new instances. Now up to 32 cores, 244GB RAM, 8x800 GB SSD

Just received the announcement that instances of the new generation Amazon EC2 High I / O have become available on AWS. These types of instances are based on the new generation of Intel Ivy Bridge processors. Each virtual CPU (vCPU) corresponds to one hardware execution thread (hyperthread) of the Intel Xeon E5-2670 v2 processor (Ivy Bridge).

Here is a sign:

Instance type vCPUs ECU Rating Memory (GiB) Instance Storage SSD (GB) Linux On-Demand Price * ($ / hr)
i2.xlarge four 14 30.5 1x800 $ 0.85
i2.2xlarge eight 27 61 2x800 $ 1.71
i2.4xlarge sixteen 53 122 4x800 $ 3.41
i2.8xlarge 32 104 244 8x800 $ 6.82


These instance types provide the largest number of IOPs (I / O operations per second) compared to other existing types. For example, i2.8xlarge gives more than 365,000 random read operations of 4 kilobyte blocks and more than 315,000 random write operations of 4 kilobyte blocks.

AWS recommends I2 type instances for high-performance NoSQL DBMSs such as Cassandra and MongoDB.
