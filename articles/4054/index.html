Word 2007 will be able to create and edit blogs

The MS Office team was pleased with the good news. In the new version of the text editor Word 2007, the function of exporting text messages with pictures directly to the blog will appear. This was announced by Joe Friend (Joe Friend), the lead programmer of the Word development team.

As you know, MS Word could export files to HTML before. But the resulting code was, to put it mildly, not entirely correct. To put it bluntly, this code was just awful. It contained hundreds of completely unnecessary tags, style sheets and other elements, from which the professional web developer had hair on his head looking on end, sometimes.

A leading Microsoft programmer assures that all these problems are a thing of the past. The developers managed to optimize the HTML export function and now it works almost perfectly. In particular, the last post on his blog, Joe Friend, published directly from the beta version of Word 2007. To see how much better HTML export works, you can view the source code of a web page. In principle, it is noticeable that it is generated automatically, but, in general, the code is rather clean and logically structured, it correctly uses style sheets. This is not the same abracadar, which issued the previous version of Word.

The new version of the text editor will have built-in support for popular blogging services such as MSN Spaces, SharePoint 2007, Blogger, and Community Server (the latter is used on blogs.msdn.com). In addition, it will be possible to automatically export text to any site that supports FTP upload and program metaheblog APIs for illustrations. Publication of messages is carried out with one click of a button, and after successful completion of the download process, a message appears about it (see screenshot ). A special toolbar with two tabs has been developed for formatting blog entries.

Editing posts in a professional text editor may be more convenient than in the web interface. There are many familiar features in Word, such as automatic grammar checker and autochange, because of the lack of which annoying typos often occur.

Publishing blogs directly from a text editor will also help to popularize the blogging itself. It can be expected that there will be thousands of newbies updating their diaries with the help of the Word, as it was with the home pages a few years ago. To publish your own blog, just select the corresponding item in the text editor menu.
