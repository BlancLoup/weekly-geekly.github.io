Adobe updated Adobe AIR to version 24 with support for a new target platform.

The company Adobe released the next scheduled update of its product Adobe AIR. The key feature of this update is that a new (for this development tool) target platform has appeared - Apple tvOS .

I recall that Adobe AIR is a cross-platform (Android, iOS, tvOS, Windows, OSX) development tool that uses an enhanced version of Flash Runtime and is designed for standalone-free work without plug-ins. As a programming language used ActionScript, which runs inside a virtual machine.

Today, quite a few well-known gaming applications run on Adobe AIR and are available for download in various digital stores (Apple AppStore, Google Play, Steam, Gog, etc.).
