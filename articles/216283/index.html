Google announced SDK for wearable devices

Today, Android Wear Developer Developer Preview has appeared on the Android developer blog ( link ).

There is also a new section http://developer.android.com/wear/ .
Now it is possible to write programs for watches. Moreover, besides the already filled up rectangular clocks, round ones appeared!
Can be seen on video.


To start testing a new SDK, you need to register. In response, you will receive an email with links to the examples and to the program on Google Play, which must be installed on the phone with version 4.3 and higher. Unfortunately, I do not have such a phone, but the program is not installed on the Nexus 7 tablet.
In addition, you can download the image for the emulator. Emulators are also two - for rectangular and round watches.
To look at this miracle, set the emulator for round hours.


Since I do not have a device to communicate with the emulator, there is little use for it. Have to run to the store for a new phone.
On the site to which I have already given a link at the top, you can find additional information and videos from developers with code examples.
This step has been long awaited. We will follow the further development of the project.
By the way, if you recently installed ADT 22.6 and you have problems installing and editing old emulators, then the good news for you is update 22.6.1. We swing and rejoice.
