Windows Azure Summit


I draw your attention to the fact that on February 20 in Moscow a large Windows Azure Summit conference will be held, dedicated to the Windows Azure platform, both from a technical and a business side.

Key topics of the conference:

  • Windows Azure as a development platform: latest platform updates, basic Windows Azure services.
  • Scenarios for using Windows Azure: a single backend for mobile solutions, transcoding and streaming video, a backend for gaming applications, creating extensions for SharePoint Online and SharePoint 2013, integration with corporate systems.
  • Business in the cloud: approaches to determining the cost of SaaS services and their pricing, methods and channels of promotion, management of the quality level agreement (SLA), compliance with the requirements of the law on personal data.
  • Cloud and Web: Web development (HTML5, jQuery, SignalR, ASP.NET MVC 4) and hosting web projects in Windows Azure infrastructure.

We are pleased to announce that all the conference reports will be broadcast online on our updated portal AzureHub.ru ! We ask you to connect to the broadcast on February 20 at 10-00 in order to watch the opening of the conference and subsequent reports.

Visit the Windows Azure Development Center, see you at the conference!
