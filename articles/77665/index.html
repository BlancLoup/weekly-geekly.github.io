Adobe Illustrator - danger!

According to reports from Adobe (bulletin No. CVE-20094195), a vulnerability has been found in Adobe Illustrator CS4 and Adobe Illustrator CS3, leading to a buffer overflow and the execution of arbitrary code. The vulnerability is related to the processing of EPS-files, in the light of which I recommend, such files obtained from unreliable sources should not be opened until this error is corrected. Adobe plans to release a patch update no later than January 8, 2010, which will not hide a satisfied long time (MONTH!) For a security update.
