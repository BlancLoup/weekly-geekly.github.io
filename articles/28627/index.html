Radio-T Notifier

Hello,

For the convenience of tracking new releases of the Radio-t podcast, I created a small extension for the Firefox browser.

The principle of operation is simple - it hangs in the browser's status bar, shows the latest version of the podcast; if there is a new release, it shows a pop-up alert.
In the context menu, you can view information about the podcast and download it.
If you have a wheel on your mouse - feel free to rotate it on the extension icon and you can see all the podcasts in RSS.

As always, adequate comments and suggestions are welcome. If you can - only adequate.

Download radio-t-notifier .
To install, simply drag the downloaded extension to the Firefox version 3 window.

Good luck.
