Analog HTC clock / weather widget in windows

Many owners (and not only) of HTC devices are familiar with a widget that displays hours and weather. Windows mobile, Android, and now the widget is ported as a program, hanging in the tray, and on Windows Vista / 7.

This thing is called uncomplicated - HTC Home for Windows.
The only installation requirement is the availability of these most Windows Vista or Windows 7, as well as Microsoft .NET Framework 3.5 or higher.

Now there are 2 versions. The first is stable and recommended for download. The second is in alpha and includes additional features, such as a 5-day weather forecast.
