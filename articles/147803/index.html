XBMC: Android Home Cinema



Finally, the best cross-platform media center XBMC ( official announcement ) has been ported to Android. This is an ideal program for organizing DIY home theater: open source, open API, extensions in Python. For example, you can write a script for downloading torrents and foldering or an extension for playing online torrents . There are hundreds of add-ons for different purposes, which people exchange in forums .

The XBMC feature set for Android is the same as for other platforms. This is not a remote control, not a client, but a full-featured media center.

So far, .apk is not officially advertised, everyone is asked to test the code. But judging by the first reviews, XBMC works quite well, so you can start making set-top boxes, digital set-top boxes from older Android smartphones.

Github source code

To install XBMC on a smartphone, even a jailbreak is not required, the media center is installed as a normal application.
