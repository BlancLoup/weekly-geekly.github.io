Express gate


For this wonderful feature, I once did a separate post where I told everything in detail. But since we started talking about the hidden partition on the drive in ASUS netbooks and laptops, then we must not mention such a thing as ASUS Express Gate . For this function, there is always a separate button on the device, but many do not know about it, because they press it only when Windows is loaded, when the button has a different function.



What is ASUS Express Gate for? Well, in general, applications can be found fully. There is a file manager (with access to the disk file system), a photo viewer and a music player. There was a place for a full-fledged browser (Firefox with support for Flash), an IM client (with support for Google Talk, ICQ, MSN, AIM, QQ, Gadu-Gadu and Yahoo networks), as well as Skype. Super Hybrid Engine


As I said above, there is a separate button on the case of netbooks and laptops for the ASUS Express Gate function - as a rule, it looks exactly like the power button. When Windows boots up, this button acquires another function called the ASUS Super Hybrid Engine .

Quite a useful feature, the main task of which is to balance between performance and battery life. Pressing the magic key will switch the operating modes of the laptop - from Battery Saving to Super Performance. The result is an increase in productivity by 10-15%, or a gain in operating time. Both that, and that can be useful, therefore far from the socket there is a sense to master operation of the button, switching between the modes one in one pressing.
