20 years ago came Internet Explorer

The Microsoft Edge browser development team in its twitter account congratulated the predecessor browser with a farewell message with the appropriate video:


Internet Explorer 1 was released on August 16, 1995 along with the OEM release of Windows 95. Like other products of the company, the browser appeared due to the fact that Spyglass had purchased the license for its Spyglass Mosaic browser for $ 8 million. The first version of Internet Explorer existed for a relatively short time and only a month and a half later Internet Explorer 1.5 for Windows NT was released. It is believed that the screenshot below represents the IE 1 interface with the open site:



There is a screenshot of the start page of IE 1:



2 years before this, the first browser for Windows with a graphical user interface Mosaic, which was created by Mark Andrisen at the National Center for Supercomputing Applications, NCSA, appeared. Mosaic quickly gained popularity (over the year after its appearance it was downloaded more than 2 million times), in 1994 it received a new name Netscape and on its basis the business company Netsacpe Communications was organized. Microsoft responded to the actions of competitors quickly and, as mentioned above, released Internet Explorer 1 in 1995. The browser war broke out, in which Microsoft won, essentially incorporating the functionality of Internet Explorer into Windows and making the choice of browser unallocated.

As many already know, in Windows 10 a new browser Edge has appeared, which should replace Internet Explorer, but so far it comes in parallel with it. Its main innovations, well visible to the end user, are Cortana voice assistant, the ability to take notes on web pages and the planned support for extensions designed for other browsers.

According to the analyst firm StatCounter, Google Chrome is currently the leader among browsers in the global market, and Internet Explorer is behind it with a significant lag.
