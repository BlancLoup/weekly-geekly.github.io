SVG-Edit - draw online, draw offline



We meet the new version (2.4) of the open vector editor SVG-edit.



This is a really fast, web-based vector editor using only JS, HTML, CSS and SVG.

It can be used directly online, without any information, without installing any extra plug-ins, and it works in all modern browsers.

What words, try !



Chips


  • Free-hand drawing
  • Line tool
  • Rect / square tool
  • Ellipse / Circle tool
  • Text tool
  • Color palette / picker
  • Save drawing to SVG
  • Select / drag / resize elements
  • Multiselect elements
  • Undo / Redo
  • Rotate tool
  • Polygon / Polyline Editing
  • Linear gradient picking
  • View and Edit SVG Source
  • Clone tool
  • Align tool


In the new version:

  • Raster Images
  • Group / Ungroup
  • Zoom
  • Layers
  • Curved paths
  • UI Localization
  • Wireframe mode
  • Change Background
  • Draggable dialogs
  • Resizable UI (SVG icons)
  • Convert Shapes to Path


And the most interesting, it can be used offline.
There is even an extension for FF and a widget for Opera.
