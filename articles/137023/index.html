Source codes of cross-platform framework Enyo 1.0 and 2.0

HP has begun to fulfill the promise and put open parts of webOS in open source. The first gift took place tonight - the Enyo JavaScript framework under the Apache 2.0 license. Now this is a cross-platform framework.

Enyo is great for creating lightweight and fast applications: the Enyo kernel weighs only 13 KB. Enyo has a Playground where you can write any code and see how it works.

Download Enyo 2.0
View code on github

By the way, the first application made on Enyo has already appeared in the Android Market. See also sample apps on the Enyo website.

The operating system Open WebOS 1.0 promise to release in September 2012.
