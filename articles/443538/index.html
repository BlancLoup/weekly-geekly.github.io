Data Saver for Chrome on Android is now available for HTTPS

Google helps Android users work faster in Chrome with an improved Data Saver mode, which, it is claimed, can reduce data usage by up to 90% and reduce web page load times even on slow networks. However, so far it has only worked with HTTP sites.

The search giant announced that Data Saver for Chrome on Android is now applied to HTTPS pages, so you can get a safe and fast page view even if you have a poor connection.



It's also easier for users to see if Chrome optimizes the page you are trying to view using the ‚ÄúLite‚Äù icon that appears in front of the URL. It is also possible to view the original page if the user so desires by clicking on the lightweight page indicator.



According to Google, this feature only works when Chrome detects that a user is viewing content through a 2G network, or when the estimated download time may take more than five seconds, depending on the network status and the capabilities of the device. The company added that Data Saver will be automatically disabled for users who often choose the original version of the page.

Google also tries not to cross the confidentiality boundary with this feature, stating that only the URL is provided to the company, leaving cookies, login information and personalized page content secret.
