Apple closes 16 (sixteen) security holes in Safari 4

Apple has released a service update, Safari 4.0.5, which aims to eliminate the huge number of security holes in this browser.

Since the previous update in November, 16 (sixteen) vulnerabilities of the most diverse nature have been discovered and closed in Safari: from compromising user data to remote execution of arbitrary code.

The previous service update covered significantly fewer holes: six in total.

You can get information on closed holes on the official page .
