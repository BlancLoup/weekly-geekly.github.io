Krita 2.9 Animation Edition Beta: the first beta version of the animation and engine for working with 10k images


Six months ago, the Krita project launched a campaign on Kickstarter to raise funds to create a new version of the graphical editor. Among the main goals for the new version (Krita 3.0) were the ability to create animated videos and work with 10k images. And today, the first beta version of the promised functionality was published!

What is the new version? For user Crete almost did not change. This is all the same editor, with the same wide range of brushes. Only three new panels were added: Animation , Timeline and Onion Skins . These three panels allow you to fully control the animation process: control the frame order, playback speed and onion layers. In addition to the animated interface, a new ‚ÄúInstant Preview‚Äù mode (past working title ‚ÄúLevel of Detail Mode‚Äù) has been added. This mode allows you to draw on huge images "as if" in real time.

The changes seem insignificant, but they have a downside. Krita 2.9 Animation Edition has a practically new, redesigned core, which allows you to have several copies of the same image and work with them in parallel. In the mode of working with animation, Crete uses these copies for efficient storage and rendering of individual frames, and in the ‚ÄúInstant Preview‚Äù mode, along with the image of Crete, it stores its reduced copy, the scale of which depends on the current zoom. When drawing Crete, it first calculates its reduced copy, shows it to the user, and then when it becomes inactive (to think, or choose a new cool brush), Crete will already begin to calculate the original image in the background. More information about this process can be found in this article on Habr√© .

Here it is worth noting that this is the first public version of the ‚Äúanimated‚Äù Kryta, and some things in it may not yet be implemented. For example, there is no saving animation in the usual compressed video formats. Instead, Crete allows you to export frames into a sequence of ordinary images, which can then be reduced to the desired format in any composer. Of course, this restriction is temporary.

So, Krita 2.9 Animation Edition Beta, welcome download!
