Microsoft has adapted virtual reality for visually impaired people.



Microsoft experts have created a set of 14 tools that allow people with visual impairments to better perceive virtual reality.

Microsoft has introduced a set of 14 software functions for VR devices that simplify the perception of the virtual world for the visually impaired. Solutions will help users detect obstacles in a timely manner and better navigate in space in scenes with low contrast.


This article is in our blog.


SeeingVR functionality can be found in this video .




With SeeingVR, you can, for example, add a description of objects, as well as highlight guide contours and change the color of objects. In this case, nine tools can work with any types of virtual reality helmets, and the other five must be integrated into a VR application using software packages used in the Unity game engine.


SeeingVR was tested on 11 people with visual impairment. A team of researchers noted that using SeeingVR, they began performing normal tasks much faster. Earlier, Microsoft also developed the Seeing AI application, which recognizes and voices objects that fall into the lens of a smartphone for visually impaired people.
