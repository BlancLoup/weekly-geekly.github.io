Social network for writers

Recently I was looking for a book on lib.ru. I came across the section "Samizdat". It turns out that there are a lot of writing people, which became for me a kind of revelation. I shared this thought with my comrades. It turned out that there is no less popular resource - stihi.ru . After that, the idea was born to make a social network for people writing.

I'll say right away - this is still an idea. And if someone wants to join - I will be glad.

The standard feature set includes the following features:
  • Add your own works (prose, poems, script, etc.)
  • Rating of works
  • User Rating
  • Writing reviews of works
  • Connecting broadcast rss feeds from personal blogs of authors
  • Internal mail
  • Short messages in a profile (as in tweeter)
  • Draft discussion. Users can leave a margin in the margins.
  • Export data to popular formats
  • Personal board (dashboard). On the board, you can leave notes to yourself. Perhaps in the form of sticky leaves. Like implementation in tink.ru
  • Blanks: a description of the characters, the meeting place, etc. (http://www.webook.com/911writersblock)

But the main chip should be a text editor. It should change depending on what type of work (story, poem, etc.) the user writes.

Main features of the editor:
  • Work in full window mode (full screen)
  • Marks in the fields. Marking may be attached to a paragraph.
  • Change history (as per wiki)
  • Autosave
  • View mode: by chapters, by pages, in one canvas
  • Automatic compilation of table of contents
  • Publishing texts via email and SMS. If a piece of text or a picture falls into a general draft. If the code of the work is indicated, then on the fields of this work
  • Spellchecking
  • Bookmarks in the text
  • Customize color schemes
  • Turn on rulers

If the piece is prose:
  • Quick editing of the heroes of the work. It is assumed that there is a directory of actors. For each, you can add a name, description and images. When the author inserts the name of the hero into the text (as a macro), the name is associated with this reference book. Those. From any place you can click on the name and see the description of the hero. When you insert a name it should bow

If the work is a poem:
  • Selection of rhymes in the context menu

If the work is a script:
  • The screen is divided into 2 parts. On the left are the names of the characters. Right - replicas
