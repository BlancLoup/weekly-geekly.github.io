CyanogenMod 10.2 released



A new release of the famous alternative build Android, developed by an independent community - CyanogenMod 10.2! Version 10.2 is based on Android 4.3.

Privacy guard

In addition to updating the code base, a new feature was added to CyanogenMod 10.2 - Privacy Guard. It allows you to organize the launch of selective applications in incognito mode, implemented by analogy with the private browsing mode in browsers. That is, when accessing the address book, message database, calendar, page opening history, the application will always return empty lists. This feature is enabled in the advanced application startup settings section.
