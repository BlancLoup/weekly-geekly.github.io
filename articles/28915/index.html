Announcement of DirectX 11 in July

Microsoft will introduce the new DirectX 11 at the Gamefest 2008 event, which will be held July 22-23 in Seattle, USA. The release of the new API is scheduled for early 2009.

The main novelties of DirectX 11 - support for tessellation, multi-threaded rendering and shaders version 5.0.

You can find out more about what the Redmond team has pushed into its set of API functions also at Nvision, an event from NVIDIA, which will be held in late August.

The first games on the new API, we may see in 2010.
