GTDlite - GTD Mobile

I want to submit my development on GTD - gtdlite.appspot.com .

Before that, I actively used Remember The Milk , but it was very difficult: the lack of projects (I tried to use GreaseMonkey squeaks for this, but not conveniently) and poor mobile support (I have noki n810).

I tried other web services for GTD, but it didn‚Äôt happen.

The last straw that prompted the start of work was the announcement of the jqtouch library, as well as the desire to use Google AppEngine somewhere.


The current version allows you to:
- add any information to MISCELLANEOUS
- sort it: the following actions, calendar (actions for date), deferred, etc.
- manipulate actions

While this is only the alpha version, in the future it is planned:
- Finalization of the basic functionality (your wishes are welcome)
- Integration with Google Calendar in the work with the calendar
- Integration with Evernote (section "Information")
- Local data storage for working offline (currently only offline HTML caching is used)

The resource works best on webkit browsers, but firefox 3.5 also shows quite tolerable. On the desktop I recommend trying to work in sidebar mode (sidebar).
