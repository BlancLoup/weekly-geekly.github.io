Device for cloning USB drives

The company Kanguru Solutions has released a device for simultaneous recording of the same information immediately to 9 or 24 (depending on the model) USB drive. Called Kanguru USB Duplicator, this product will speed up the replication of data written to USB flash drives, such as, for example, presentation materials, software distributions, etc.

Kanguru USB Duplicator is compatible with USB 1.1 and 2.0 drives, even works with external HDDs and supports Windows, DOS, Linux and Mac OS file systems. In addition, the device is able to record data in several modes, including automatic.

Information about the cost of USB Duplicator, as well as, however, any other details about the device on the manufacturer's website yet.
