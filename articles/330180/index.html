Dell EMC Cloud for Microsoft Azure key features

About a month ago, Dell EMC introduced the Dell EMC Cloud for Microsoft Azure Stack, a platform that will optimize the deployment of hybrid cloud solutions based on Microsoft Azure Stack. And while it is not yet available for order, we will briefly describe its main advantages and what it will give to customers.


Dell EMC Cloud for Microsoft Azure Stack is a hybrid cloud platform for providing infrastructure (IaaS) and platform (PaaS) as services. It supports Azure functionality, which is available both in the cloud and in the local data center. With it, you can safely receive, create and share application services in the Azure and Azure Stack. This is available for both traditional and cloud applications.

The platform is developed on the basis of Dell EMC's proprietary converged infrastructure with advanced encryption and backup features. Complement its convenient and functional application development tools from Microsoft. This provides high performance, fault tolerance and an adequate level of security when using the platform.

The Dell EMC Cloud for Microsoft Azure Stack enables enterprises to automate the delivery of IT services for all applications in the Azure ecosystem. This happens through the self-service portal, while maintaining operational control of the infrastructure. For example, a choice of additional parameters is available for data protection and encryption, backup and recovery of application data is provided. These and many features and capabilities enable customers to confidently switch to a hybrid cloud model.

System owners can connect Azure‚Äôs power to their own data center and work with them on their own firewall. These capacities will be designed, tested and installed using the Dell EMC. They will also be serviced and maintained through this platform by the supplier‚Äôs specialists. This reduces the complexity of implementing a hybrid cloud and speeds up this process. At the same time, Dell EMC takes over the management of the life cycle of the entire hybrid cloud platform, which frees up the company's resources, saves money and concentrates on the company's tasks.

When you deploy a platform, Dell EMC experts provide support at every stage, from planning. First, customers receive recommendations regarding the optimization and expansion of hybrid cloud platforms. This includes the development and configuration of services, systems and identification and access systems, as well as the expansion of monitoring and accounting systems for the Azure Stack. All this will help you quickly and painlessly integrate the Dell EMC Cloud for Microsoft Azure Stack into the IT infrastructure of any enterprise.

In short, users get the following features:

- accelerated delivery of traditional and cloud applications;
- Increased efficiency of the provision of IT services and accelerated the creation and deployment of business applications;
- encrypted virtual machines that protect against data theft in any part of the cloud;
- Automatic delivery of private and public cloud resources;
- backup and restore created for virtual environments and clouds;
- obtaining resources by authorized users through the self-service catalog.

The new platform will be available to order in the II quarter of 2017.
