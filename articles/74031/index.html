SilverX - convert flash to silverlight projects



I am glad to announce a fairly new project that can make life easier for silverlight developers and greatly simplify the process of migrating flash movies to silverlight.

The SilverX site has the following project description:

‚ÄúThe SilverX project is designed for silverlight developers and designers who migrate their flash content to silverlight. SilverX can be used in the following scenarios:
  • export vector graphics from CorelDraw to a SWF file and then convert to silverlight XAML;
  • conversion of vector graphics embedded in the SWF file into content ready for use in silverlight applications;
  • extract vector graphics embedded in the SWF file;
  • the transformation of flash-animations in silverlight-rollers;
  • transformation of flash-banners into ready-made slilverlight-banners.
".

It is necessary to add that the content is not just transformed and converted from flash to silverlight. Additionally, a project is created that is ready to work with it in Visual Studio 2008 SP1 and Expression Blend 3. That allows you to make changes to the content immediately after conversion in convenient tools for working with silverlight.

SilverX is actively developed and is free for personal use.
I am glad that the author of the project seems to be a Russian-speaking developer named Roman Petrov.
