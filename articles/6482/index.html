In this article, we would like to highlight, perhaps, the main function of the Voice SMS system - the possibility of anonymous receiving messages.

Voice SMS allows you to replace typing dictation messages. You simply indicate the 10-digit number (3 digits of the code + 7 digits of the number) of the addressee and dictate the message. The recipient of the message will receive an SMS, which will allow him to listen to the message. You can send a message in this way to any mobile subscriber.



An important feature of the system is that when sending a message, instead of the number of the subscriber, you can specify the code corresponding to its number. Naturally, you can send a message by specifying the code corresponding to your phone number. And if so, then you can freely publish this code as a means of prompt communication with you. The sender of the message will not know your real number. Your number will not be visible even when replying to received messages.

Where does this code come from? There are at least two ways:

1. Call from your phone to the service number 0942 * and enter the service code 8883. That is, in response to the IVR request for the service code, you must enter 8883 #.

The IVR will tell you the code corresponding to your phone number. If you have forgotten the code, you can always call back and listen to it again.

2. When you register on our service sites (for example, Mobile Notifications ), you can see the code for your number after authorization.

A description of the publication of a contact link is provided in the description of the voice SMS service (see Examples of Use ). This link is just an instruction on how to send a Voice SMS using a specific code.

For example, the phone code of the developers of this service: 24. That is, you can send a message directly to the author of this article, following the instructions for this link:

Send message
  • 1. Call 0942
  • 2. Enter service code 8882
  • 3. Enter the subscriber code 24
  • 4. Dictate the message
