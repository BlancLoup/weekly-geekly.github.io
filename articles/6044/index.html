Koala will remember everything

IBM laboratories are developing a wiki-like plug-in to the Firefox browser called Koala . Its purpose is to automate repeated operations on the web. For example, a plugin can facilitate the tedious work of filling in various forms.

Koala technology remembers user actions, creating a pattern that should minimize the repetition time through the same chain of actions. Ready-made scripts Koala can be published on the Internet.

Since there is a possibility that Koala will have scenes created with malicious intent, it is important for developers to solve a number of security issues. The technology was demonstrated by IBM developers at the ETech conference.
