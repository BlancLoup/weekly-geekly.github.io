Beeline automatically adds a toolbar and changes the design of sites.





Not so long ago I wrote a topic about Beeline and their automatic addition of a toolbar with mail.ru search . Then a java-script file was added to the html code, which in turn added a plate with the mail.ru referral search. Two months passed and Beeline decided to do everything differently.

Yesterday, while browsing websites via the mobile Internet, I saw that an updated toolbar from Beeline was being displayed on them. It was unclear how to turn it off: there was no close button, it was only possible to remove it to the top, but still it was visible on the screen. I had to accept an agreement to sell my organs, and only after that the magic button ‚ÄúDelete toolbar forever‚Äù appeared.

I thought that this was all over, but as it turned out - I was wrong.

Looking through the web pages, I saw that the buttons on the websites began to look different: they became larger and yellow. Very surprised, but did not attach importance. However, I was even more surprised when on my own websites the buttons also changed appearance.

Immediately remembered Beeline and began to look for the cause. The reason, of course, turned out to be in the communications operator, and this time a different code is added to each webpage. Before the closing html tag, the Beeline adds the following script: tb.beeline.ru/Scripts/App/AnchorCompiled.min.js

You can download it only from the mobile Internet network Beeline.
The file weighs 152 kilobytes and is jQuery version v1.10.2 and additional code from programmers beeline.
