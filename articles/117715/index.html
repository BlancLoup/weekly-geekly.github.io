Solar system in 3D



The site solarsystemscope.com posted an interactive map of the solar system.
Interesting entertainment, with a good set of functions.

Go to the page and play with the solar system



Opens the animation of the project logo.


Further, if nothing is touched, the available species are cyclically demonstrated.
Namely: the system of planets; Planet; Telescope (constellations, as they are visible from Earth).






To start using the map, you need to click on it with the mouse, after which the map will switch to the Planets System display mode. The control is intuitive - we scale the mouse wheel and rotate the map, dragging it with the cursor. In addition, you can scale using the scale on the right.


On the left are the elements of the interface settings.
Just screenshots, no comments.










Some menu items have sub-items.
This, for example, adjusts the linearity of the radii of the orbits and the displayed dimensions of the planets.


Below is the player. It starts the animation of the planets, and also allows you to quickly "rewind" their position forward or backward. If you know the exact date of any event (on a cosmic scale, of course), then you can manually set the date and time and admire the parade of the planets . By default, the position of the planets corresponds to the current date.


In the upper right corner is a block of buttons that allow you to select the interface language (English, Slovenian), hide the interface elements and switch to full-screen mode.


What practical benefits can be derived from this resource?
  • You can find out the distances from one planet to another.
  • You can see the relative position of the planets and constellations at a particular time.
  • You can search for a star (constellation) by name and clearly see where it will be at the appointed time (the search function is currently at the development stage) .
  • You can just kill half an hour before the end of the working day.


That's all. For those interested, again duplicate the link to the page of the project .
