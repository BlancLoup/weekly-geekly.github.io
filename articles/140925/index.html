The fastest processor module

This year, Variscite introduced the world to the new VAR-SOM-OM44 processor module, which, according to the manufacturer, is the fastest, most powerful and ‚Äúrichly‚Äù equipped module on the market at the moment, combining all this power with a very compact size (38.5 √ó 67.8 √ó 3.7 mm), low power consumption and reasonable price (for such performance).

The module is based on the OMAP4460 microprocessor from Texas Instruments, with a dual-core Dual-Core Cortex-A9 1.2-1.5 GHz with symmetric multiprocessing. With this performance, the standard power consumption is from 1 to 3 W (depending on configuration).

The maximum configuration of the module includes 1 GB of dual-channel LPDDR2 memory, and the integrated graphics accelerator Imagination PowerVR SGX540 supports simultaneous connection of several displays.
The module "copes" with two-dimensional and three-dimensional graphics, decoding and encoding high-definition video in Full HD 1080P format, including stereoscopic video.

The module is also equipped with a Gigabit Ethernet network interface, USB 2.0 ports (two hosts, OTG support) and 3 UART ports (up to 3.6 Mbps). Wi-Fi 802.11 b / g / n and Bluetooth V2.1 + EDR are supported using the TI WiLink 6.0 single-core controller.
There are inputs and outputs of the audio subsystem (line input, headphone output, microphone and S / PDIF Out), an interface for connecting the camera.



The module works with support for Linux kernel version 3.0, as well as with support for OpenEmbedded to create RootFS / SDK. The Linux kernel supports all embedded peripherals. There is also support for Android (GingerBread and ICS). X-Loader and U-Boot are used as boot loaders. More details can be found here .
