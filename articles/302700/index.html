Migrate users from Asterisk (FreePBX) to 3CX Phone System

The main issues facing the system engineer when migrating from one system to another are the laboriousness of migration, downtime and possible data loss. If you are faced with the task of migrating from an existing Asterisk system (FreePBX) to 3CX Phone System, you can do it quickly and without losing user information - in a few simple steps! For this purpose, a specially created user export module from Asterisk PBX (FreePBX) is used.

Installing the user export module from Asterisk


1. Log into the FreePBX administration interface and in the upper left corner go to Admin> Module Administration> Upload modules .



2. In the Type drop-down list, select Download (From Web).

3. In the Download Remote Module field, enter the download address of the module http://downloads.3cx.com/downloads/ext23.tar.gz and click the Download (From Web) button .



4. After the module is loaded, a message appears confirming successful loading. Click on local module administration to proceed to the module installation.



5. Go to the Application section and select the FreePBX to 3CX Converter module. Expand the module settings and click the Install button. At the bottom of the page with a list of modules, click Process .



6. Click Confirm. FreePBX to 3CX Converter will be installed. In the upper right corner of the interface, click Apply Config (red).



Export users from Asterisk


1. In the top pane, click on the Applications button, and then on the 3CX Extension Export module. You export users to a CSV file, which will then be imported into 3CX Phone System.



2. In the Select target 3CX Platform Version drop-down list, select 3CX Phone System Version 14 .

3. If your Asterisk PBX has extension numbers of different lengths, set the Toggle Checkbox to map extension numbers to option so that the export module will bring all the numbers to the same length.

4. If you want to export only extensions that correspond to the selected number length (for example, 3 digits format ), set the Save that fits!



Import users in 3CX


In the 3CX Phone System management interface, go to the Extensions section and click the Import button from the csv file . Specify the exported CSV file and follow the system instructions.
