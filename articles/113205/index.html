8 useful services for web developer and designer

Under the cut - a description of eight services that can significantly facilitate the life of a web developer, coder or designer.


1. Wordmark.it




Service for the selection of fonts Wordmark.it - allows you to see the spelling of a given phrase with all the fonts installed on your computer. Smartly loads, convenient filters. Sorry, there is no way to set the background colors and letters.

2. Spritebox




Spritebox is designed to ‚Äúcut‚Äù pictures into sprites. Upload a picture, select the necessary areas, a couple of clicks - and you have ready css-styles for sprites.
CSS sprites are a representation of a large number of small images, by creating one large image, which significantly saves download time and server resources (the request goes to just one image, instead of all).

3. CSS3 Generator




CSS3 Generator will help you to use CSS3 features wisely: show a list of available CSS3 styles, generate a cross-browser code for them, let you know which browsers support or do not support these styles.

4. ProCSSor




ProCSSor is a tool for checking / formatting css-styles. Upload your styles.css, choose formatting settings and get a beautiful valid css-code at the output.

5. Super Conversion Button




Super Conversion Button is a tool for quickly creating button designs. Simply choose styles, colors, lettering text, etc.

6. 0 to 255




0 to 255 - a simple and convenient service for selecting shades of colors. I think the screenshot is clear.

7. Frame Box




Frame Box - a service for creating the so-called "wireframes" - block diagrams of tables. A very useful thing at the design stage of marking up new sites or any web interfaces. Created schemes can be saved, as well as share links to them.

8. jsFiddle




The jsFiddle service is a sandbox for web developers. The working part of the window is divided into 4 parts: HTML, CSS, JavaScipt and the result. In the menu on the left, select the js framework, the libraries for it, the HTML specification, etc. In the corresponding parts of the workspace we write the code - and immediately see the finished result. Projects can be saved and shared.
