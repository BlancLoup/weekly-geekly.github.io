Calculate Linux 10.4 released

Purpose


Calculate Linux is a family of distributions based on the Gentoo GNU / Linux project and fully compatible with it. Calculate Linux is divided into three areas, Calculate Linux Desktop (CLD, CLDG, CLDX) , Calculate Directory Server (CDS) and Calculate Linux Scratch (CLS) .

Features

  • Ready client-server solution for enterprises.
  • 100% compatible with Gentoo.
  • Support for an interactive build of the system - making changes to the distribution kit.
  • Supported languages: English, Spanish, Italian, German, Polish, Portuguese, Russian, Ukrainian and French.
  • Installation on HDD, USB-HDD with file system is supported: ext4, ext3, ext2, reiserfs, xfs and jfs, or on USB-Flash with FAT32.

Major changes

  • Calculate Linux includes the new Calculate Linux Desktop distribution with the GNOME Desktop (CLDG).
  • Added utility 'cl-passwd', for changing user password on authorization server.
  • Reduced hard disk space.
  • Reduced time to transfer roaming profile data from a remote server.
  • Added the ability to manage updates for user application settings through the file ~ / .calculate / ini.env.
  • Parallel installation of packages is enabled by default.
  • Extended support for HP, Epson and other printer models.

additionally in CLD
  • The desktop view has been changed to display the ~ / Desktop directory.
  • OpenOffice uses the 'Oxygen' style, adds support for the KDE menu for opening and saving files.

additionally in CLDX
  • Optimized to run on weak PCs.
  • Added progressbar when logging into a domain user session.
  • Changed the menu structure by analogy with KDE and Gnome versions.
  • Added file-sharing client network Direct Connect LinuxDC ++.
  • Added RSS (Atom) -agregator Liferea.
  • Pidgin adds support for Facebook, hotkeys.
  • The xclear plugin has been added to Firefox.
  • Fixed keyboard layout in tty1 after starting a graphical session.

additionally in CDS
  • Added support for executing CLD user commands on the server through the profile file ~ / .calculate / desktop.env. Supported profile packing in the archive, changing the user password in the LDAP server.
