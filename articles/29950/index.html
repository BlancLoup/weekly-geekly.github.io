Disk burning utilities

Almost all Windows users know about the Nero Burning Rom program, but you can burn discs without a single one. I tried to look at alternative utilities that exist in Russian and that have been updated in the last couple of months. Two of them are completely free.

ImgBurn ( www.imgburn.com , free)
It has the most ascetic interface and small size - only 1925Kb. Russian interface downloads separately from the site of developers.



Able to burn CDs and DVDs, allows you to set all the traditional settings - speed, closing the session and so on. In addition, there are specific settings for certain types of drives, which, however, during the testing did not have time to come in handy.

You can create ISO-images and burn discs from them, but the program does not support the creation of audio and video discs, does not produce DVD authoring.

Conclusion : a convenient, small and fast utility that is useful to those who often burn discs with different types of data.

Ashampoo Burning Studio 8 (www.ashampoo.com, $ 49.99)
Size 31.55MB, Russian interface included



A large program, reminiscent of Nero in size, functionality and, unfortunately, at a high price - about a thousand rubles. The functional set is impressive in its scope - recording of any kind of discs, DVD authoring, recording of audio CDs, BluRay. Supports copying discs, as well as grabbing AudioCD.

It is possible to create your own menus for discs with screenshots and splitting into episodes. The set includes a program for drawing covers.

Conclusion : The program clearly claims to be Nero among professional users. It is necessary for those who often and a lot are engaged in recording various kinds of discs and use all the functions. The most expensive program of our review.

ChiliBurner ( www.movavi.com , $ 29 or 999 rubles per package)
Size 14MB or 50MB (VideoSuite package), Russian interface only when downloading a package.



The utility is available in seven languages, but it is downloaded and sold separately in English only. In Russian, you need to download the entire VideoSuite package. True, the price of the package in Russia is approximately equal to the price of one utility for foreigners.

Extremely simple interface in which you can choose the type of disk - data, audio, video, action - recording or copying and make additional settings - PAL or NTSC, screen aspect ratio, quality and type of disk.

Not only standard CD and DVD discs are supported, but also such not too common formats as Super-VideoCD. You can create your own menu for the disc from a large number of proposed animated or static templates.

CDBurner XP ( www.cdburnerxp.se , free)
Size 2914Kb, Russian interface included.



Despite being free, the program has a nice interface and an impressive number of supported functions - support for CD, DVD and BluRay discs is declared. The user can copy discs, burn them from images (the utility for converting images from Nero to ISO formats is built-in) and create your own audio CDs, setting pauses between tracks. Unfortunately, the program does not know how to do DVD-authoring.

Conclusion : If you do not need to burn DVDs, then an excellent choice. Free program with user-friendly interface. Recalls earlier versions of the same Nero.
