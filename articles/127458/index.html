Faviconist: Create a beautiful favicon with HTML5


Each site must have a favicon. If you have not yet made an icon for your site - make it, now it's even easier!
Faviconist is a new favicon generator by Michael Mahemoff.

You no longer need Photoshop to create a simple icon. As Mahemoff writes in his blog : Faviconist is for 95% of the people who are satisfied with the micro-design of their icons, the rest will still have to resort to Photoshop.



Even if you do not need to create a new icon, the site should be well studied to see what you can do with the HTML5 Canvas tag and client processing. In fact, the only thing done in the server part is the conversion of the .PNG file to the .ICO file. The service also uses Node.js, Kaffeine, jQuery and Google Fonts.
