Debian GNU / Linux 5.0.4 released

On January 30, 2010, the fourth update of the stable release of Debian GNU / Linux codenamed "Lenny" was released, which traditionally includes a lot of fixes, more about which can be found on this page.
I would like to highlight the most significant updates - DSA-1915 for the 2.6 kernel (privilege elevation associated with the default value of the mmap_min_addr parameter), DSA-1948 for ntp (DoS) and # 563245 ticket in the buglist regarding the false-positive triggering of the FH_DATE_PAST__20XX rule for spamassinin (spamassinin (false) for the rule FH_DATE_PAST_20XX for spamassinin (false-positive for the spamassinin) has the number 3.2.5-7 and 3.2.5-2 + lenny1.1 ~ volatile1).
As always, ISO images will fall on the mirrors in the coming days, but you can upgrade with apt now.
I recommend system administrators to use the debian-security-announce mailing list, so you will always be up to date with the latest updates.
