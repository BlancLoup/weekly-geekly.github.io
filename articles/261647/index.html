Apple released iOS 8.4

Apple released iOS 8.4 ( APPLE-SA-2015-06-30-1 iOS 8.4 ), fixing a number of serious security vulnerabilities in its proprietary mobile OS (relevant for devices such as iPhone 4s +, iPod touch 5 and iPad 2+). The company has eliminated a known vulnerability that made it possible to remotely reboot the device using a specially crafted text message.



Apple also fixed a number of vulnerabilities, similar to the well-known Masque vulnerability , which we wrote about in more detail here . The previous Masque vulnerability allowed one iOS application to be overwritten by another, with the new application gaining access to the previous data. New vulnerabilities also relate to the incorrect operation of iOS with the bundle ID of already installed applications, which allows attackers in one way or another to gain access to the data of the installed application.
