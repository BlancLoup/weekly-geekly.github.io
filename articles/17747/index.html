Release Candidate SP3 for Windows XP

Whatever Microsoft itself says about the need to migrate to Vista, people's love for Windows XP does not lose its mighty power. Yes, and the corporation promised to support XP for a few more years, so that from the release of the next set of patches can not go anywhere. In particular, from Service Pack 3, which recently acquired the status of a candidate for releases and was officially posted on the Microsoft website. Weighing SP3 RC 336 megabytes. However, for some privileged testers, the output of the SP3 RC is not new, because a certain circle of users managed to familiarize themselves with the service pack a few weeks earlier. Now the set of fixes is available to everyone without exception.

The final version of Service Pack 3 for Windows XP will be released only in the first half of next year. This will be the latest service pack for the world's most popular operating system.
