Dedicated Enterprise Server

Dear readers, we invite you to familiarize yourself with the new product of our company - Dedicated X16, which is the first to open a line of dedicated servers from ColoBridge.


The equipment is located on the technical platform in the SAS 70 Type II -certified Databurg data center in Frankfurt am Main, Germany. All ColoBridge server rooms are equipped with redundant uninterrupted power supply, air conditioning, gas fire extinguishing and protection against unauthorized access. Externally, the site is connected to the networks of Internet operators of the first tier Level 3 and GlobalCrossing, as well as to the largest traffic exchange points: MSK-IX (Moscow); DE-CIX (Frankfurt am Main); AMS-IX (Amsterdam); Equinix (Ashburn, United States); LINX (London). The internal network infrastructure is built on gigabit interfaces.



One of the main features of the Dedicated X16 is a hardware platform that allows multiple physical servers to be placed in a single package. A distinctive feature of this solution is the hot-plug support for both hard drives and server modules. Due to this, when a hardware problem occurs, the module is quickly replaced with a backup one, and server idle time is reduced to a minimum. Localization of the problem component (processor, motherboard, etc.) is already performed outside the client's server. The housing is equipped with two power supplies, each of which can serve all the modules, thus providing redundancy of electricity supply to the servers.

Standard server equipment:


  • Processor : server quad, Intel Xeon L3426 with Hyper-Threading support (8 virtual cores).
  • RAM : 16 GB DD3 ECC Registered (expansion up to 32 GB is possible).
  • Hard drives : a total of up to 6 media can be connected to the server (two media are standard). Connectivity options: 2 x Intel X-25M G3 80 GB SSD, 2 x SATA Seagate Constellation.2 500 GB or 1 x SSD and 1 x SATA. The discs are designed for corporate use.
  • Network : 2 x 1 Gbps backup network connection.
  • KVM over IP : power management, console, virtual media connectivity, hardware monitoring.
  • Backup : 100 GB on external file storage.

Additionally, VPN connection services based on Cisco equipment and blocking network access to the server via SMS are available. Dedicated X16 thanks to modern hardware supports any type of virtualization and almost all current operating systems. If necessary, ColoBridge technical staff will advise you on any issues and help you deploy the project on the server.

If you are interested in a server of a different configuration, just email us and we will create an individual commercial offer for you.
