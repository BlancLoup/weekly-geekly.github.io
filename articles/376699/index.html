It contains the Arduino microcontroller, a basic set of components, a breadboard, a book with lessons (programs, installation diagram). A soldering iron is not required - to install the elements, the provided prototype board is sufficient.

3D printing



3D printing is certainly one of the most ‚Äúgeeky‚Äù topics - but it‚Äôs not affordable for everyone to buy a 3D printer. But there are portable 3D printers - namely, 3D pens.

The principle of its operation is identical to the "big brother" - it contains plastic in the form of a thread, and at the tip - an extruder that heats the plastic, and while cooling - a figure is created. In the 3d pen, everything is done safely and the heating elements are protected from direct contact.

You can see this pen in Moscow Hackspace, in the 3d printing lab - the Funtastique One 3D pen with an OLED display . This pen is able to work with PLA plastic - which is odorless and environmentally friendly.
