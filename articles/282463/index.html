The first update of the stable branch Vivaldi 1.1


Hello!

From the day of the release of the first stable version of the Vivaldi browser, it turned out that it is not so stable. More precisely, you sent us many messages about errors found, as well as suggestions for improving the browser. In fact, everything happens as planned: it is naive to expect that the final release will not contain errors and will suit all users with the existing functionality. Therefore, we have been actively working with your messages in the past weeks, and today we are ready to show some results by updating the stable version of Vivaldi to number 1.1.

Working with tabs

The largest number of requests received for tabs. Users simply overwhelmed us with requests to expand the possibilities of working with many open pages. In general, this is understandable: tabs are the most actively used element of the browser. So, what we managed to do by today.

First of all, we have expanded the number of options available for managing tab groups. For example, by selecting several tabs, in the context menu you will find several new items that will allow you to perform various actions on these tabs:


Also in the browser settings you will find several new options concerning the opening and closing of tabs, including - and combined into groups.

Bookmarks

There are also a lot of requests here, but not all of us managed to implement them for today's release. At the moment we have added the ability to open several bookmarks at the same time - just select them in the list using the Ctrl or Shift key and then press Enter.

In general, there will be quite a few changes with bookmarks soon, so, as they say, do not switch :-)

Other

We also had a little time to work with the function of importing data from Opera 12 - we added the ability to import Express panels. Naturally, we cannot transfer the real opera Express panel, so that in Vivaldi it will appear in its usual form. When importing transferred links placed in a separate folder. Well, you can either transfer these links to the new Express panel, or use the folder with imported links as the Express panel.

Another small innovation is the ability to disable the drop-down list of the address bar.

And, of course, fixes bugs. This is our constant priority. In the Linux version, it became possible to change the interface language through the browser settings, fixed work with the Pocket extension, fixed the page update instead of searching from the address bar, and much more. Full list of changes you can read on this page .

That's all. Do not forget to install the new version of the browser Vivaldi:


And also do not forget to report errors found .
