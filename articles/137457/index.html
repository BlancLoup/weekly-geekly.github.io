The gov.uk platform will be completely open. Source codes published on GitHub

UK government creates fully open e-government platform gov.uk
The gov.uk platform is designed to replace the current DirectGov system.

The platform architecture was created taking into account its work in the cloud service, and AWS was chosen as the provider of cloud infrastructure. That is, the system will work in a public data center, and will be observed.
all data protection requirements.

The system is written in Ruby, its code is fully open and published on GitHub.

At the moment, 53 repositories have been created on GitHub with code for various subsystems of gov.uk and 21 employees are engaged in their support. The code is laid out under the MIT license, which allows it to be used in virtually any type of project (commercial or public).

Ubuntu is used as an operating system for EC2 servers, as a MySQL and MongoDB database.
The mail service provider selected are Amazon SES and Gmail, DNS Provider Dyn.com

In my opinion, this is a completely different level of openness, to which our government is still very far away.
In fact, anyone can download and study the source codes of the system, as any company has the opportunity to place their services in the same data center where gov.uk is located

For comparison, you can consider the latest news about the election observation system. It says that employees engaged in the implementation of the system are prohibited from discussing with the journalists any technical details.
