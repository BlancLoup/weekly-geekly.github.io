Ubuntu user day



Custom days are a collection of courses aimed at teaching beginners how to work with Ubuntu. This is a series of online courses (IRC) where you can:
  • Learn how to install Ubuntu
  • Learn how to find the right program.
  • Learn how to get help.
  • Learn how to join our community.
  • Learn the basics of working with Ubuntu
