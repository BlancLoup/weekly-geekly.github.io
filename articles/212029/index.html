The public alpha version of the decentralized Tox messenger has been released.



More recently, the public version of Tox Messenger has become available.
(Now the download button on the site is finally active)

Let me remind you that Tox is a decentralized messenger that is created for the future replacement of Skype, the main objectives of the project:
- Fully open source
- Decentralized work
- Absence of bookmarks and wiretaps
- Lack of advertising




The current version of the messenger can:


- Send messages
- participate in chat rooms
- Supports Windows / OS X platforms (already compiled version)
- Bulb message routing ( What's this? Git )

What will be able to Tox in the future?


- Voice calls
- Video calls
- File transfer (Implementation in GUI, the kernel is already able)
