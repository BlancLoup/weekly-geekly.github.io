Instructions: delete personal data from iPhone

How do you like this situation: after the release of the new iPhone 3G, you sell the old device (after making restore in iTunes to delete all the data), and after some time, your personal photos, data from correspondence, passwords to accounts and etc.


It turns out that the following problem exists: when a new firmware is restored via iTunes, the user's personal data is not overwritten and it will not be difficult for anyone with the necessary software to restore them.

The story is known when an Oregon state police officer restored an iPhone file with letters, etc., belonging to the previous owner (I remind you that refurbished is a so-called refurbished technique, that is, the user delivers a defective laptop / PDA / phone , it is repaired and sold at a discount)

Strictly speaking, there is no absolutely adequate solution yet - there is only an opportunity to minimize such a development of events:

  2. On the ‚ÄúInformation‚Äù tab, we remove the checkboxes from ‚ÄúSync contacts ...‚Äù, the same for calendars, mail, bookmarks.
  3. On the tabs "Photo", "Podcasts", "Video" - do the same. Although interesting podcasts can be left))
  4. Create 3 large playlists with music (or video) - at least the same as the capacity of your iPhone? better a little more.
  5. On the Music tab, select the first of the playlists to sync. Make sure the ‚ÄúCapacity‚Äù indicator at the bottom of the screen is 100% full.
  6. Press the "Sync." Button, then select the next playlist and sync again, then the third one.


This technique can obviously help wipe traces of data. Although the data is stored separately from the music / video, but the size of logical drives on the iPhone is dynamically changing, so there is a great chance that all data marked for deletion will be erased.
