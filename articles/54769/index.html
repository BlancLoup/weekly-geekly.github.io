ReadySET is a free set of software project internal documentation templates.

There is such a project ReadySET , it contains an excellent structured and streamlined collection of templates for internal project documentation. All templates can be viewed on the site, downloaded by the archive or exported via Subversion. All templates are well documented and provide a ready infrastructure for project documentation.

Almost all aspects are covered: project planning, vision, requirements gathering, development, system design and architecture, implementation, testing, installation, technical support and maintenance. Templates are created in very clear XHTML / CSS format.

Although all templates are documented, they are fairly compact and require some basic knowledge of development processes. There is also a PRO-version of the project, templates have been converted into it for use in MS Office formats, and several additional documents have been added.

The resource is extremely useful. Everything is in English. BSD License .
