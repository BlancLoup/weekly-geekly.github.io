Radio tags and beacons will not allow to forget or lose things. Medgadgets selection



Modern technologies allow us not to forget things and prevent their loss. Radio tags and beacons can be attached to a phone, tablet, or even a pet's collar. After that, you will always know where that or another thing is. Well, or where the pet ran. In the continuation - a large selection of tags-Anti-theft, devices that allow you to protect your luggage while traveling and remember what you need to take on the road.

Anti-theft Stick N Find





Stickers of this type provide a permanent signal, thanks to which the Radar Screen application (iOS, Android) shows the location of things with beacons installed. If things are not yet in the range of the label, the application can be configured to be notified when the label appears. Such a function can be used, for example, at the airport, to detect your luggage in the general "carousel".

The beacon can be triggered when leaving the contact zone with the phone. This feature can be used to control pets by sticking Stick N Find on the collar. If the animal has gone beyond the safe zone, a buzzer sounds and a message appears on the phone display.

  • Connection Type: Bluetooth 4.0;
  • Signal type: constant, radio signal, sound, light;
  • Working distance: about 30 meters;
  • Fastening: magnesium, or mechanical, in the form of a medallion or pendant;
  • Display on the map: the Radar Screen application shows only the remaining distance to the Stick N Find beacon, but not the direction;
  • Work offline: about a year, CR2016 battery; After the battery can be easily replaced;
  • Application: Radar Screen (Adnroid, iOS);
  • Price: 2990 rubles;
  • Features: the sticker has a buzzer and a light bulb, so you can find a thing with an attached beacon even in complete darkness.


GOkey: charger, cable, search device, USB flash drive





This gadget is not a sticker, a multifunction device. GOkey can be used as a data cable and phone charger. There is also a memory module that allows you to constantly carry with you important information.

In addition, there is a built-in beacon. The beacon works in conjunction with a mobile application (iOS, Android). The application on the mobile device shows the distance to the beacon. The latter should be used as a key fob. If the application shows that the key fob is already close, then by pressing the button you can turn on the beep on GOkey, and it will immediately become clear in which direction you need to continue searching.

In addition, by pressing the button on GOkey, you can turn on the sound on the phone - this is how reverse search works. Now, neither the keys nor the phone will not be lost in the apartment or office.

  • Connection Type: Bluetooth 4.0;
  • Signal type: permanent, radio signal, sound;
  • Working distance: about 30 meters;
  • Fastening: mechanical, key chain;
  • Display on the map: the application shows only the remaining distance to GOkey (signal strength in percentage terms) but not the direction;
  • Work offline: a few days, the battery;
  • Application: Adnroid, iOS;
  • Price: 3800 rubles;
  • Features: the sticker has a speaker, you can find a thing by sound.


Anti-theft Tile





A waterproof beacon tag that helps prevent the loss of the thing on which the beacon is attached. Mechanical attachment, the label can be hung on any thing: a guitar, a wallet, a telephone, whatever.

The tag works in conjunction with a smartphone app. If the label goes out of sight of the phone, the latter gives a signal. The application on your smartphone will allow you to find the label anytime and anywhere.

  • Connection type: Bluetooth 4.0 (2.4GHz ~ 2.4835GHz);
  • Signal type: radio signal;
  • Working distance: about 45 meters;
  • Fastening: mechanical, key chain;
  • Map display: the app shows the last location of the tag;
  • Work offline: year, built-in battery. When the battery is discharged, the manufacturer changes the label for a new one for free;
  • Application: Tile app (iOS);
  • Price: 2000 rubles (but the beacon is discontinued, and it is no longer for sale);
  • Features: one application can work with 10 beacons.

  • The safe zone mode - in this mode, the device works, if attached to the collar of the animal, or to the child's clothing. If there is an exit beyond the safe zone, a signal is triggered, a notification arrives on the smartphone;
  • The motion sensor allows you to determine even the slightest movement of a thing that should be fixed. It is useful if you want to insure against theft;

Chipolo anti-theft





An easy to use and practical key ring tag. If you lose the thing to which the key chain is attached, the mobile phone application will show where the tag was last detected. When you click on the button in the application, the label starts to emit beeps.

Also, with the help of a label, you can search for a paired smartphone. To do this, you need to shake the label, and the phone will beep.

Chipolo can be used as a means of tracking pets. The application will always tell you if the animal has moved away from the owner for a distance greater than 60 meters.

The device determines the ambient temperature by transmitting this data to the application on a mobile phone.

  • Connection Type: Bluetooth 4.0;
  • Signal type: radio signal, sound notification;
  • Working distance: about 60 meters;
  • Fastening: mechanical, key chain;
  • Display on the map: the application shows the last location of the label, the distance to the label, also displays the boundaries of the safe zone;
  • Work offline: half a year, replaceable battery CR2025;
  • Application: Chipolo (iOS, Android);
  • Price: 1400 rubles;
  • Features: to search for a smartphone you need to shake the label. Also, the label is equipped with a temperature sensor, and transmits temperature information to the smartphone. Thus, the label can also be used as a thermometer / weather station.
