Ruby Lessons for Kids



Among the teachers there was an opinion that Ruby is the most suitable language for learning programming. The first PL in life should be as easy to use and, of course, object-oriented. It should not have any ambiguous concepts or strange features of the syntax. In general, Ruby fits perfectly, the guys from The Hybrid Group decided and released the KidsRuby app ( code on github ) - Ruby lessons for children.


The application can be downloaded under Windows, OSX and even in the form of a ready-made assembly of Ubuntu-based KidsRuby OS. See examples of exercises here .

It is clear that none of the adults is not recognized in this, but KidsRuby can be recommended to novice programmers.
