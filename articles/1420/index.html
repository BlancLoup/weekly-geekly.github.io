Printing from the web is inconvenient, but it will change.

Guide Lexmark International , to explain to Americans what they need new software for this company, ordered a special study, which showed that Americans are increasingly using the Internet as the main source of information, but to display this information on physical media (print texts and pictures, to example) they are uncomfortable.

But despite this, people both pressed the ‚ÄúStart‚Äù printer button and click. 79% of Americans print out a description of the trip itinerary before they set off, while only 19% prefer to stop and ask for directions.

Web documents interviewed print out, mainly for reasons of practicality. The reasons that encourage you to make paper copies: the need for step-by-step instructions of various kinds or in the description of the routes (55%), the fear of losing an electronic copy on the Internet (36%) or some personal notes (46%).

Among the reasons why printing information from the Internet is inconvenient, respondents noted:

- Prints unnecessary ads;
- Cut off the words or images;
- Prints unnecessary images;
- Images are difficult to print separately;
- Useless paper and toner.

To facilitate printing, Lexmark has developed a special software, the Lexmark Web Toolbar , which allows printing only what the user needs. For example, using the toolbar, you can print only text, excluding ads and unnecessary images, print the entire page without cutting off the edges or text, print a black and white version of the page or photos from borderless websites.
