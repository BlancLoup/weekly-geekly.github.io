Requests to Yandex live

Hello!
Some time ago I came across an interesting thing: http://export.yandex.ru/last/last20x.xml - recent requests to the search engine Yandex.

I was sitting, updating it, probably for an hour, this very thing is interesting, to see what people are asking for Yandex.

I remember, there was a post ‚ÄúRequests to Yandex - Live‚Äù , they showed a map with requests, but for some reason the link is not working now.

I remembered that I had once seen a page where requests for google.ru were displayed with tiles, and decided to make my own, only about Yandex, with all that it implies.



At first he did it for himself, but many liked this thing, decided to submit it for a general review. Hope not to fall .

How it works: we open the page - we see the last requests, according to the Ajax, the tiles for new requests are randomly updated. Since there are requests in different languages, for example, in Turkish, screwed Yandex translator https://tech.yandex.ru/translate/ . He tries to translate everything that is not written in Russian (small print below).

For beauty, you can change the color of the tiles to red, blue or green, the default is multi-colored.

Caching, of course, is only small, 2 seconds. And I noticed that Yandex gives 403 with frequent requests.

You can see this creation here: http://nikitchenko.ru/zaprosy-k-yandex-online/
