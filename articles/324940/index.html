A selection of useful tools with Product Hunt for March

Everyone is familiar with the Product Hunt resource, how many new projects - innovative and banal, useful and stupid - appear there every day. For those who do not always have enough time to sort this stream, we offer a small selection of interesting finds from several categories (CSS, Performance, Messaging Systems, API) that you have chosen for yourself over the past few weeks.


CSS


Animista - an extensive collection of ready-made CSS animations from Ana Travas. On the developer's site, each of them can be assessed in action, applying to various design elements (background, text, button, image, and so on), setting the direction and selecting the necessary settings (duration, delay, intermittent movement). You can add the most suitable options to your favorites, download, and also generate code for them.

Waffle grid is a simple and compact Flexbox grid system from Lucas Gruwez. Allows you to easily adapt the layout for a number of devices, easily configured according to the needs of the user.

Dima Blover and Alexander Isora offer their craft mates a collection of 180 linear gradients that can be used as a web page background. Currently available on the site are: previews, PSD code and PNG versions for downloading each individual sample, as well as packages with a full set of gradients for Sketch and Photoshop.

Performance


Product Roadmap Builder is a roadmap creation tool developed by the ProdPad team. Keeping feedback, backkids, specifications, brainstorming results, the application gives your team the opportunity to not only properly comprehend and streamline the workflow, but in addition to effectively visualize the resulting scheme - deliver information to colleagues and impress customers at the same time. The system integrates with other popular project management tools (Trello, JIRA, GitHub).

Wildfire extension allows you to record any sequence of actions on a web page (both online and offline) and create simulations on its base with the possibility of further editing. When recording or playing back a session, a backlog is automatically created. A good option for developers who use automation tools (for example, iMacros, Ghost Inspector), or testers working with Selenium and the like.

Loom Workflow is a powerful tool for visual communication with colleagues or clients. Integrate the extension with the necessary services (Gmail, Intercom, Trello, GitHub, Basecamp, JIRA, Google Docs, Hacker News, InVision App, Dropbox, Product Hunt, GitLab and others) to record video showing a particular process, right from the screen and quickly share it with the other person

Messaging systems


Parlo is a platform for creating text and voice chatbots on web pages and in applications. It can be used as a ready-made solution or customized at the discretion of the developer. Integrates with a variety of popular corporate systems. The project is currently in beta test.

SendBird helps embed messaging or chat into your application. The developer has the ability to serve 100,000 viewers, create separate chats for audience segments, apply moderation tools, including automatic protection against spam and obscene language. The tool can be useful in the development of entertainment or information services, playing video in real time, multiplayer games, social platforms.

A nice innovation from the company Slack : with the help of App Unfurls you can personally create a preview for any links that you share with the interlocutor. This works both for internal content stored on your company's domains, and for external links. In addition to the actual links, the preview can include one or several action buttons - interactivity will help the team save time.

API


The new Zapier command line interface provides those who write applications on Node JS with instant integration with the developer platform directly from the command line. Exiting to the ecosystem of more than 750 tools opens up more opportunities to write, test and deploy products: connect the API to request information, share data and lead joint projects, simplify the process of automation and get feedback from the community.

Perspective - a tool that is designed to maintain a healthy atmosphere in online communities and make discussions more productive. Analyzing the text of the comment as the user enters it in the field, the service determines and displays the approximate level of "toxicity" before sending. Such an algorithm reduces the risk of abusive comments, providing a feedback to the user, and simplifies the moderation process.

Jsnoxious offers a collection of API endpoints that you can use in the architecture of your application, as well as the ability to create your own directly from the browser.
