Announced and available to developers the 3rd update of Windows Phone 8

Yesterday the third update for Windows Phone 8 was announced . And what is most interesting is that this update became immediately available to developers using the Windows Phone Preview for Developers program.


What's in Windows Phone 8 Update 3?

Support for high resolution large screens
Future devices on Windows Phone with five- and six-inch screens and 1080p resolutions will be able to display up to 6 small live tiles in a row on the start screen, instead of four.


New powerful processor support
The update adds support for the Qualcomm Snapdragon 800 4-core processor.

Driving mode
Allows you to configure the mode of answering text and voice calls, system warnings while driving, if the phone is connected to a specific Bluetooth device - the head unit on the car or any other speakerphone device.


Improved access to the general Internet
Now you can simply connect your phone and Windows 8.1 device via Bluetooth, choose the name of the network - and you can use the Internet through the phone.

And much more ...
Improved performance, with noticeable eyes - I saw how my phone began to work faster after receiving this update on the program Windows Phone Preview for Developers .

In addition, with this update, you can use your own calls for instant messages, letters, voice mail and reminders. You can also assign your own call to a text message from a specific contact, so that you can understand who is writing to you without having to look at the phone.

Added a setting that allows you to disable auto-rotate the screen. Reading lying on the couch has become even more convenient!

Advanced phone storage management.

Ability to close the application displayed in the list of switching between applications.

The ability to connect to WiFi during the first launch of the phone.

Improved Bluetooth.

And this update is available to all developers right now!

Along with the announcement of the 3rd update for Windows Phone, as we promised earlier, we are launching a program that allows any Windows Phone developer to access this update so that in a few months it will start to spread to users' phones. .

Who can participate?
One of the three conditions must be met:
or phone is unlocked - http://msdn.microsoft.com/en-us/library/windowsphone/develop/ff769508(v=vs.105).aspx#BKMK_UsingtheWindowsPhoneDeveloperRegistrationTool ;
or you are a registered Windows Phone Store developer - http://msdn.microsoft.com/en-us/library/windowsphone/help/jj206717(v=vs.105).aspx ;
or you are a registered Windows Phone App Studio developer - apps.windowsstore.com .

Given that any Microsoft Account owner can unlock one phone, this is an offer for all enthusiasts who are willing to try the update at their own peril and risk. Because You may lose the manufacturer warranty.

Detailed upgrade instructions are available on the following page: dev.windowsphone.com/en-us/featured/update3

Important notes on 3rd Developer Update
The third update does not have a new and updated API, so there will be no SDK update . There will be an optional update with new images for the emulator. Most developers do not need to do anything with their application to work on phones with the third update.

There are some changes you should pay attention to:
1) The memory available for background audio playback on phones with> = 1GB of memory will increase to 25MB.
2) Viewport Internet Explorer and WebBrowser control has changed for all phone permissions.
