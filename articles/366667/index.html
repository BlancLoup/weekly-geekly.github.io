Single board baby with two wireless interfaces. Acquaintance



Supported by the crowdfunding stage, Onion Omega arrived and today we met. This is a tiny, 2 by 4 centimeters, single-board computer based on Atheros AR9331 (MIPS) running OpenWRT. The loader works Das U-Boot. As a result, we have cheap and compact 400 MHz, 16 MB flash and 64 MB DDR2. Next under the cut

In addition to the interesting size: there are two WiFi interfaces on the SoC selected by the developers - one through wifisetup is easily configured to work with the desired external network, and the second by default serves for local access to the device. Actually, the idea is that the board will become the basis for a WiFi-controlled robot, which in turn will also have access to the network. For the emergence of the same USB ports, if you need such a project based on this board, you will have to connect the Dock

After the first start, it is worth starting oupdate to get the latest firmware version. Then it all depends on what you want to get from the baby. The batch manager in OpenWRT is called opkg, with the upgrade key it gets a list of possible distributions, lists from the list, and installs the necessary ones via install. In my case, for example, you need opkg install python-base. To be continued
