Parallels acquires application developer for remote access 2X Software

Parallels, a company specializing in virtualization and automation, has acquired 2X Software, a developer of applications for remote access to desktops. Team 2X will join Parallels.



Founded in 1999 by Sergey Belousov, Parallels is the global leader in cross-platform virtualization and automation solutions. 2X Software develops software for accessing remote desktops, applications, and devices.

Applications based on Malta in 2004 include 2X Software - 2X Mobile Device Management and 2X Remote Application Server. 2X MDM is an application that allows you to monitor and manage corporate and personal mobile devices of company employees. 2X RAS is an application delivery server and Windows remote client for any operating system, including mobile OS. The main competitors of 2X are XenDesktop and XenApp from Citrix.
