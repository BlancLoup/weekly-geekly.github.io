Google will show a netbook based on Chrome OS

Engadget reports that the event will take place tomorrow (December 7) in San Francisco, USA. The event itself is dedicated to the cloud operating system Chrome OS.

It is assumed that the netbook will be released in a limited edition of 65 thousand units and will be designed for testing by third-party software developers and the OS itself.

About the complete set of the netbook until almost nothing is known. Only the processor is known - Intel Atom - the most popular on the netbook market.


In addition, according to TechCrunch, Google will very soon be launching its Chrome Web / App Store app store, as evidenced by the recent release of the Chrome 8 browser, which already supports these Web apps.
