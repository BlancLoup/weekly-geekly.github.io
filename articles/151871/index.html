Opera Mini Smartphone - More than a Smartphone



What is the difference between a smartphone and a regular phone? Yes, it is more powerful, more productive, more functional, larger in size ... But in reality the prefix "smart" means something else: such devices allow us to make our work on the network more convenient and complete due to the ability to install applications that simplify and, sometimes, optimize familiar desktop work with your favorite services, cutting off all unnecessary and leaving only what is really important. This is great, but can this feature of smartphones be further improved? We thought it was possible.

Introducing the new version of Opera Mini 7.5 for Android-based smartphones. Traditionally, the new version contains many fixes and improvements, but still the main thing, which is why the number ‚Äúgrew up‚Äù immediately by a tangible figure - we added a new function that fundamentally changes both the style of work with Opera Mini and the functionality of the browser. Now it will be possible not to complicate life by finding and installing additional applications - it is quite possible that the functionality you are looking for already exists in your browser! The new function is called the Start page.

In fact, users of regular phones using the Java version of Opera Mini 7 are already familiar with this feature, but now it's time for the owners of more advanced devices to rejoice, so let's briefly get acquainted with the features of the Start Page. It contains several sections, the most interesting of which are:

Social networks

Here you will receive real-time updates of events occurring in the social networks you use. Connect your accounts on Vkontakte, Facebook or Twitter to learn about new events in one place without having to visit several web pages.

news

Here comes news from a variety of resources that you often visit. If your favorite websites have RSS news feeds, the browser will automatically connect them to this section and collect the latest news from the sources of interest to you, all in one place.

Recommended links

This section contains a categorized list of resources that you may find useful. This list is formed automatically depending on the country in which you use your smartphone - we tried to collect all popular local resources here.

Here, in brief, that's all. Yes, it remains to add that the classic Express panel also does not disappear anywhere - it is still available next to the Start page. Moreover, if you do not need new features and you want to use only the Express panel, you can easily disable the Start page in your browser settings.

As usual, you can download the new version of Opera Mini 7.5 for Android via the link m.opera.com or from Google Play.
