Internet Exporer Mobile 6 emulator released

Yesterday, developers from Microsoft announced the release of emulators and full documentation for IE Mobile 6. This is the next generation browser, which will soon be included in the Windows Mobile operating system and, accordingly, will appear on all new Windows Mobile devices.

In IE Mobile 6, normal full-screen rendering appeared, templates for correct display of text blocks were updated, script support (Jscript 5.7 from Internet Explorer 8) and AJAX was improved, Adobe Flash Lite 3.1 plug-in for flash display was added, touchscreen support and gesture recognition were added, zooming as well as many other usability improvements.



If you want to see how your site will look like in a new browser, you can download an emulator right now.
