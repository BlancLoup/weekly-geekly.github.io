FusionCharts free


FusionCharts Free is a free open source component for drawing graphs (for desktop and web applications).



Opportunities:
* 22 popular chart types - Column, Line, Pie, Bar, Area, Stacked, Candlestick and Funnel Chart, which look the same in all browsers
* animation and interactive
* works with PHP, ASP.NET, JSP, ColdFusion, Python, RoR, HTML and even with PowerPoint presentations
* interaction with any databases
* AJAX support

Website: www.fusioncharts.com/free

Demo: www.fusioncharts.com/free/demos/Blueprint

License: double: MIT (X11) and the GNU GPL

Additional features / goodies are offered for money.

Flare


Flare - ActionScript library (Flex) for visualizing graphs and charts, animation, tree maps, etc.



Website: flare.prefuse.org

Demo: flare.prefuse.org/demo

License: BSD

Aflax


Aflax is a library that allows developers to use Flash features in JavaScript.



Opportunities:
* animated charts
* integration with Flash code
* support network, video, camera

Website: wiki.github.com/aptana/aflax

Apparently open source.

Open flash chart


Open Flash Chart is a free library that allows you to generate charts and graphs from data obtained from php, perl, python, java, asp, asp.net applications or text files.



Supports:
* Data Lines Chart
* Bar Chart
* 3D Bar Chart
* Glass Bar Chart
* Fade Bar Chart
* Sketch Bars Chart
* Area Chart
* Bars + Lines Chart
* Pie chart
* Scatter Chart
* Candle chart

Website: teethgrinder.co.uk/open-flash-chart

XML / SWF Charts


XML / SWF Charts is a tool for creating interactive graphs and charts.



Support:
- Line charts
- Column charts
- Stacked column charts
- Floating column charts
- 3D column charts
- Stacked 3D column charts
- Parallel 3D column charts
- pie charts
- 3D Pie charts
- Bar charts
- Stacked bar charts
- Floating bar charts
- Area charts charts
- Stacked area charts
- Candlestick charts
- Scatter charts
- Polar charts
- Mixed charts
- Composite charts
- Joined charts

Website: www.maani.us/xml_charts/index.php

License: free, single, bulk

Axiis


Axiis is an open source framework based on Degrafa, Flex & ActionScript 3.0.



Website: www.axiis.org

Built on the Degrafa graphics framework and Adobe Flex 3.

Kap Lab Visualization Components For Flex (AS3)


Kap Lab offers 5 free components for Flex.



Components:

Column & Bar chart demo: www.amcharts.com/column
