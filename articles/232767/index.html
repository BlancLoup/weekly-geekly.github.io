Grafana - a convenient dashboard for metrics



Graphana is the first really good dashboard for displaying metrics.

Dear colleagues, I want to share with you such an interesting find as the open-source project Graphana

Graphana is designed to display all sorts of cyclic metrics. In addition to the predefined system metrics (CPU, IO, etc ..), you can configure any set of arbitrary metrics - online, profit, etc. Graphana has two built-in themes - both look very good. You can create any number of random dashboards.

Demo version: play.grafana.org/#/dashboard/file/default.json

A video that briefly shows the capabilities of this software: www.youtube.com/watch?v=OUvJamHeMpw

GitHub Link: github.com/grafana/grafana

This project is developing a Swedish guy named Torkel.

I admit honestly, having tried all the available dashboards and having found Graphana, I also believe that he will succeed.

All good performance on dashboards.
