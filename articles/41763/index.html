What is it all about?


CommentsFx is a script that adds new features to the Habrahabr comment pages. CommentsFx is, in fact, part of HabraFx, but I also decided to allow other users to use this extension to bypass the HabraFx installation (some users do not need all the features of HabraFx). HabraFx users should not worry: the extension will be updated very soon and will include the CommentsFx functionality.

How to install?


To install you will need Greasemonkey. I think there should be no problems here, but if they arise, write in the comments.

What can?


Here is a list of skills of the current version of the extension:
  1. Sort branches comment by date;
  2. Display only new comments (more precisely, branches with new comments);
  3. Hiding comments with some strange things, incomprehensible to the ordinary person;
  4. Linear display of comments (sometimes this kind of viewing is very useful);
  5. Management of individual user comments: highlighting the comments of specific users, hiding comments of their own;
  6. Status memory: you do not have to re-configure the extension each time you open comments, as CommentsFx remembers your choice and honors it;
  7. Robbery of trains and caravans .

I think it's not worth telling for a long time - try everything yourself.
