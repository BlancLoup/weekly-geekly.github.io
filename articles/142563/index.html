Release Midnight Commander 4.8.3

After the release of version 4.8.2, in which the virtual file system subsystem (VFS) was radically reworked, it was decided to release a release that would be devoted solely to error correction, and which would later become the basis for a stable branch.



In other words, no new features just bugfixing:
  • fixed search and replace problems
  • fixed found errors in the editor
  • multiple problems with file operations


As always, thank you very much to everyone who participated in the testing and to those who send patches.
