Announcement of the Picaso Designer X PRO 3D printer



We are pleased to announce the long-awaited announcement and the launch of pre-orders for the Picaso Designer X PRO 3D printer.

X Pro is a 3D printer with a new core, new mechanics and electronics, as well as a number of automatic chips that will allow solving 3D problems even faster and more accurately. There are no analogues on the speed and accuracy of two material prints on the market now!

The novelty was presented by Picaso 3D on Form Next 2016 under the brand name for the European market - ODIN. We decided to see for ourselves the quality of this 3D printer and got acquainted with it at the exhibition.







Specifications:

  • Material used for 3D printing: PLA, ABS, PLA Flexible, PVA, PC, Hips, Nylon, Laywood, FilaFlex, Filamentarno (filament diameter 1.75mm);
  • Printing area: 200 x 200 x 210 mm;
  • Body Material: Aluminum (composite);
  • Guides: XY: rail (steel), Z: cylindrical (steel)
  • Layer thickness: from 0.05 to 0.25 mm (Layer thickness is adjusted by the printer software settings);
  • Positioning accuracy: XY: 11 microns; Z: 1.25 microns;
  • Availability of heated platform: Yes;
  • Printing platform: Aluminum - glass;
  • Number of print heads: 2;
  • Automatic table calibration: Yes;
  • Nozzle diameter: 0.3 mm;
  • Maximum extruder operating temperature: 380C;
  • Maximum working temperature of the working platform: 140C;
  • Noise level: 55 dBA;
  • Management board: based on the ARM CORTEX M4 core 32-bit STM processor;
  • Interfaces: USB, Ethernet, USB Flash [included];
  • Intellectual support system: Picaso 3D Polygon X software supplied with the printer automatically creates support where it is needed to ensure the strength (stability) of the part during printing;
  • Compatibility with software: Windows XP, Windows 7, Windows 8;
  • Print speed: up to 30 cm3 / hour;

NEW! Automated printing control system:

Plastic control system.
Each blockage of the nozzle is a waste of time and material, which means your money.
Thanks to this system, the printer can detect blockage nozzles and automatically clean the print head. In case automatic cleaning is impossible, the device pauses and waits for user intervention.

First layer printing control system
Print quality of the first layer, in many respects, determines the success and quality of printing of your model. That is why this system is so important. Now, when printing the first layer, the printer itself will determine the necessary parameters based on the profile of the materials installed in the printer.

Profile system
The profiles installed in the printer is a new and incredibly convenient system that allows you not to prepare a new task for different materials, colors and different plastic manufacturers. The printer will do everything for you! When loading plastic, you only need to select the correct material profile and further, Designer X PRO will set all the necessary parameters.

Alert system
Monitor each stage of the printer, the alert system will help you with this. Designer X PRO uses visual alerts and a mailing list so that at any time you will know exactly what state your device is in.

Serviceability

  • Calibration table level 50 seconds.
  • Automatic height calibration between nozzles for 5 seconds.
  • Calibration offset nozzles 5.5 minutes. (print template and set parameters)

Updated Extruder Jet Switch
The updated extruder Jet Switch allows you to create objects of unsurpassed quality due to the complete shutdown of the second material, without reducing the working temperature. Fully redesigned extruder (updated mechanical part) in the new version has a more stable print quality and increased productivity, as well as resistance to blockages. Together with the supply valve, this ensures the fastest possible switching between the two materials and an increase in print quality. Picaso 3D Designer X PRO provides the fastest and most quality printing with two materials.
