10 special-purpose Linux distributions

The beauty of Linux is that you can make a distribution for any requests. It takes only time and desire. Many popular Linux distributions are the result of the work of individual enthusiasts or groups dissatisfied with the existing set of options. Here are some of the special-purpose Linux distributions.

1. Parted Magic OS : distribution for disk recovery and partitioning, which supports more than ten file systems and includes various utilities for ‚Äútransfer, copy, create, delete, expand and compress disk partitions. Cloning disks and the implementation of a full backup. Testing drives for the possibility of failure in the future. Search for bad sectors. Testing computer performance. Safe full disk cleaning without the possibility of recovery. Access to non-bootable HDD to recover important data. "



2. SystemRescueCd is similar to Parted Magic OS, but goes a couple of steps further and helps restore the system after a crash. It also contains utilities for recovering deleted files, resetting the Windows administrator password, compressing and backing up an existing system, etc.



3. BackTrack Linux : distribution for pentesting, that is, testing computer and network security using different toolkits: information gathering, vulnerability assessment, exploit usage, privilege escalation, access maintenance, reverse engineering, RFID tools, load testing, forensic expertise, reporting tools.



4. Ubuntu Studio : a distribution based on Ubuntu, but aimed at creative activities. Comes with popular Linux-based applications for editing audio, video, photos, graphics and publishing.



5. Mythbuntu : a video recording system, a Ubuntu-based digital video recorder using mythTV on the front end . Can be used as an alternative to Tivo and similar digital set-top boxes. Mythbuntu is similar to the ArchLinux-based LinHES distribution.



6. Scientific Linux is based on Red Hat Enterprise Linux with the participation of Fermilab and CERN. As the name implies, it is intended for scientists.



7. Tails : A Debian based distribution comes with Tor anonymizer, encryption and security software for all the paranoids among us.



8. LinuxCNC : software package for controlling various mechanisms, such as milling machines, lathes, plasma cutters, robots, hexapods.



9. Edubuntu : Ubuntu-based distribution, but created by free education movement activists in schools around the world.



10. VortexBox : a distribution based on Fedora can be used as a music server or a jukebox. It can automatically convert CDs to FLAC and MP3 formats, add ID3 tags, download covers for discs. After that, Vortexbox distributes media files to network-connected players, such as Logitech Squeezebox, Sonos and Linn. Music can be broadcast on systems under Windows or Mac OSX.



More Ubuntu Christian Edition and Ubuntu Satanic Edition deserve special mention for its uniqueness.

It should be noted how many distributions are made based on Ubuntu, which itself is based on Debian, the reason for which could be the ease of creating your own distributions using Ubuntu Builder . This is not a complete list. There are hundreds, if not thousands of different Linux distributions.
