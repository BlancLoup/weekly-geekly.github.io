Error in software interrupted flight F-22

On February 11, 12 F-22 Raptor fighters that flew from Hawaii to Japan were forced to go back because of a program error that occurred when crossing the date line.
Rumors that the delay in the arrival of fighters in Japan was caused by a problem with the software went before. Yesterday morning, CNN television reported that all fighters had completely lost all navigation and communication tools when crossing the date line. After reporting this, the planes turned around and using the tanker as the lead returned to Hawaii. It is worth noting that if there were no tanker with fighters and there was bad weather, the consequences could have been worse.
