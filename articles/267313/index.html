Free site callback widget

Habrahabr has repeatedly raised the topic of testing, comparing and describing callback widgets. In comments to many articles there is a common thought - expensive. As a result: some refuse to use such widgets, while others write the CallBack widget themselves to pay for calls at IP-telephony rates.
The project Zadarma offers no more time and money to write widgets on your own. The CallBack widget from Zadarma works for free, and calls to customers are made at your current IP telephony rate (yes, with free minutes, yes, by the second, yes, you can send a call to the manager to your asterisk).

What is a callback widget?


If someone did not come across this name, then they most likely came across a similar window on one of the sites:

The main purpose of such a window, of course, is to increase conversion, or in simple words - to turn more site visitors into buyers. The system determines non-standard behavior and shows the visitor a widget with an offer to connect it with the sales manager in 20 seconds.
Now about the features of the CallBack widget from Zadarma:

Key features and capabilities:


They tried to make their CallBack widget simple but functional.
Of the main functions:
  • the ability to choose between the display of the widget on behavioral factors and a simple "button on the site";
  • setting widget display criteria;
  • definition of working hours;
  • setting all parameters and texts of the displayed window;
  • integration with all Zadarma services.
On the last point we will stop more detailed. Integration with Zadarma services allows you to not only make calls on your ip-telephony rates, receive both widget statistics, and calls cost statistics. A call ‚Äúto the manager‚Äù can be routed to the phone, to the SIP number in the system (and connected to it, for example, your asterisk), to a free PBX .
Directing a call to a free cloud PBX will allow you to receive a number of nice bonuses:
  • You can send each widget to your own PBX call processing scenario, and in it set a list of managers in the right order with the required timeouts (they can receive calls both via the Internet and their phones);
  • Naturally, in the PBX are available: call recording, voice mail, conditional call forwarding, etc .;
  • working with API: some API functions can be used for the widget without a PBX (for example, statistics), but with the PBX, the API functions become more interesting (for example, an http-notification of an incoming call is available).
In addition, it may simply be convenient to simultaneously use a Callback, an incoming phone number, a virtual PBX and a SIM card with a single account, general settings and a single tariff plan, all covering. It turns out saving not only money, but also time.

Create your widget in 3 minutes


To connect the widget, just register on the site and create a new widget in your account.
  1. We are registered on the site .
  2. Create a widget and set the basic settings.
  3. Copy the code on your site.
From the required settings, you only need to choose: where the call to the manager will go (you can set his phone number, or select a SIP number, internal PBX number, PBX script). Also, if you already have a number from Zadarma, then the call to the customer can go with the CallerID of this number.
