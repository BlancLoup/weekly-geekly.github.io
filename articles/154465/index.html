Smooth Web Fonts with CSS3

June 7, EA Games launched a new version of their site. The interface is pretty good, but web fonts look like they are ugly.

And, it seems, I found a solution that allows you to smooth the font with the help of the text-shadow CSS3 properties . It works perfectly on Windows XP (if ClearType fonts are displayed or the standard method) and on older operating systems if the browser is modern.



View this trick .
