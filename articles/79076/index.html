Porting Applications for Mac OS X (Console Emulators)


In the last topic, I told about one of the examples of how to port a Windows application under Mac OS X, but why not run the console game ...

To do this, you need zeroPSX Hybrid, zeroDS, DeSmuME, zeroN64, Shark

What is zeroPSX?
zeroPSX is an ePSXe shell made for running PSX games for Mac OS X.

ZeroPSX Hybrid v2.0 for Mac OS X 10.4, 10.5 and 10.6 is currently available (Russian is supported)
The program includes:
  • CrosOver Chromium 8.0
  • ePSXe 1.6
  • ePSXe 1.7

How to start
1. Download zeroPSX Hybrid v2.0 , unpack and mount the image, then transfer the zeroPSX Hybrid to / Applicaitons
2. Go inside the application and run the "Emulation Engine Selector", in which we select the desired emulator.
3. launch zeroPSX Hybrid and in the file menu select the PSX game type of the download and the image itself or the executable file
4. Done!

What is zeroDS?
zeroDS is a shell for NO $ GMB made for running DC games for Mac OS X.

ZeroDS v1.0 is currently available for Mac OS X 10.4, 10.5 and 10.6 (Russian is supported)
The program includes:
  • CrosOver Chromium 8.0
  • NO $ GMB 2.5

How to start
1. Download zeroDS v1.0 , unpack and mount the image, then transfer the zeroDS to / Applicaitons
2. start zeroDS and in the opened dialog box select DC game
3. Done!
Ps. there is also a naive DeSmuME application for running DC games

What is zeroN64?
zeroN64 is a Project64 shell made for running Nintendo 64 games for Mac OS X.

ZeroN64 v1.0 for Mac OS X 10.4, 10.5 and 10.6 is currently available (Russian is not supported)
The program includes:
  • CrosOver Chromium 8.0
  • Project64 v1.6

How to start
1. Download zeroN64 v1.0 , unpack and mount the image, then transfer the zeroN64 into / Applicaitons
2. launch zeroN64 and select the Nintendo 64 games ROM from the file menu
3. Done!
Hint : you can select the window type, go to the inside of the application for this and launch the ‚ÄúWindow Mode Selector‚Äù in which we select the desired type, there are also links to folders for games and saves ..

What is a Shark?
Shark is a Dolphin shell made for running Nintendo GameCube, Wii, and Triforce games for Mac OS X.

Shark v1.0 is currently available for Mac OS X 10.4, 10.5 and 10.6 (Russian is supported)
The program includes:
  • CrosOver Chromium 8.0
  • Dolphin SVN R4479M

How to start
1. Download Shark v1.0 , unpack and mount the image, then transfer the zeroDS to / Applicaitons
2. Run Shark and select intendo GameCube, Wii or Triforce game from the file menu
3. Done!
Hint : you can select a window type, go to the inside of the application for this and launch the Window Mode Selector, in which we select the desired type, there are also links to folders for games and saves.

All the wrappers and utilities for emulators are constantly being improved for more stable operation and are open for free download.
