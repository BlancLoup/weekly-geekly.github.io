Updated Windows VPS from Infobox with triple data replication and Enterprise SSD caching. We test disk performance

Today, fully updated Windows VPS from Infobox has become available. We transferred virtual servers to a cloud platform, which significantly improved performance and reliability.

Now Windows VPS is available not only in Russia, but also in Europe . This allows you to create highly available solutions and select the most suitable region for storing information and the speed of its delivery. Available regions: Moscow and Amsterdam.

For new Windows VPS, Enterprise SSD ‚Äì caching on PCI ‚Äì Express SSD and SSD DataCenter levels is used, which significantly increases the speed of the virtual server.

It provides triple replication of all data in a distributed, fault-tolerant storage system. Simply put, all virtual server data is stored on at least 3 physical servers at the same time, which allows you not to worry about the safety of information.



In this article, we will test the performance of the disk subsystem of the updated Windows VPS.

Disk subsystem performance


In Infobox, there is no separation for a slow hard drive and fast temporary storage SSD, on which data is erased upon reboot (as is common in many clouds). The entire disk subsystem of the new Windows VPS from the Infobox is replicated and Enterprise SSD caching is used for it. Thus, our fast drives can and should be used for data storage. No need to choose between speed and reliability of data storage.

According to the results of the PassMark PerformanceTest 8.0 test, the speed of the Windows VPS drive is at the level of dedicated SSD-drives and is especially high for random access to data, which is very important for working with databases.



According to IOMeter, for a test with 50% random read and 50% random write in 4kb blocks, Windows VPS allows you to get more than 7000 IOPS.



When sequential reading in blocks of 256 kb each, Windows VPS allows to receive more than 3500 IOPS at a read speed of more than 918 mb / s.



When sequential recording in blocks of 256 kb each, Windows VPS allows you to receive more than 1600 IOPS at a write speed of more than 440 mb / s.



Now we measure the speed of the disk subsystem using the Microsoft SQLIO test. Test results: more than 14,000 IOPS.



Conclusion


Using the Windows VPS from Infobox does not need to choose between performance and reliability.
