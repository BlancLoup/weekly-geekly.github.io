VNC Roulette. Srsly?



Many VNC servers do not require a password to be set, which is why Internet-based client or server machines with open access using this protocol appear on the Internet. Having scanned the entire Internet, it is not always clear what to do with them all. Why not make them a roulette with elements of a social network?

srsly.de - VNC Roulette. Update the page, get every time a new server with a screenshot. You can connect to it through a web client or through a regular VNC, put likes, report or even add your VNC servers to the database!

For scanning, zmap was used, which scanned ports 5900-5910, then the python script tried to connect to the server and take a screenshot. The web part is written in Go.

Have fun.
