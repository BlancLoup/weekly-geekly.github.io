Softphone 3CXPhone for Mac OS released

We are pleased to announce the release of the Alpha-version of 3CXPhone for Mac Os. The softphone is designed to work with 3CX Phone System 12 on Mac.

3CXPhone for Mac OS is an addition to the product line, where you already have 3CXPhone for Windows, Android and iPhone.

The new 3CXPhone Mac OS supports multiple lines and allows you to receive up to five calls at the same time, you can automatically set up your account by simply clicking on the attachment 3CX welcome email, call transfer, presence status.

New 3CXPhone for Mac Alpha

  • G722 Broadband Codec
  • Audio Routing for Sound
  • Auto Audio Detection
  • Echo cancellation
  • Microphone / speaker / ringer volume control
  • Auto Answer
  • Receive multiple calls simultaneously
  • Groups in the Presence Status section
  • Dialing from conversation recording
  • Chat with subscribers 3CX IP PBX
  • Call History
  • View Active Calls
  • View Queues
  • See the full list of changes here.
