3CX Softphone got the BLF panel, import from Google and ...

Yesterday they released a new updated softphone for 3CX v14.



The main thing is that the new softphone fully supports Windows 10, i.e. Now the owners of Windows Phone can sleep peacefully.
The second - added BLF-panel with 28 buttons, plus you can add more on additional pages.
Third - now search for contacts is available as in a mobile phone. You need to start typing a name or number and then the softphone will prompt you the options. This works both for a call and for transferring a call (Russian is also fully supported). You can watch it live on video.



Fourth - added the ability to return to the previous status by timer. He left for lunch, set Lack status for 1 hour, in an hour he will return to normal.


And finally, the last - the opportunity to import contacts from Google and Office 365
