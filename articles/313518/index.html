7 more services for monitoring sites

In the comments to the previous article, there were such a number of sites providing monitoring services, which was enough for one more post. I thank for the feedback, without which there would be no this review.


Another seven new monitoring services in this article.

Today we look at the following tools:

  1. newrelic.com
  2. nodeping.com
  3. ping-admin.ru
  4. pingdom.com
  5. statuscake.com
  6. trafficprotector.com
  7. uptrends.com

newrelic.com


For familiarization with the services are provided 14 days of free use.



Most of the functions are designed to monitor the operation of applications (Ruby, PHP, Java, .NET, Python, Node.js, Go) or servers (Ubuntu, Debian, RedHat, CentOS, SmartOS, Windows), which requires installation of the client. You can run Ping, HTTP / HTTPS checking with the ability to search for keywords in the response.



Notifications are available by email and / or mobile phone, you can use Webhook, set up delivery to Hipchat, Campfire, PagerDuty.

nodeping.com


15 days of free use are available for all tariffs.



Check suggested:

  • HTTP / HTTPS (GET, POST, PUT, HEAD, TRACE, CONNECT methods);
  • the content of the HTTP request response;
  • FTP;
  • DNS (A, AAAA, CNAME, MX, NS, PTR, SOA, TXT records);
  • IMAP, POP3, SMTP;
  • MySQL work;
  • Ping accessibility;
  • availability of a given port and more.



Notifications come to email, mobile phone via SMS or Push messages, Twitter, starting with Business tariff, Webhookes and messages in PagerDuty are configured.

The base rate allows you to run 5 monitors updated every minute, add an unlimited number of contacts for notifications for $ 8 per month.



ping-admin.ru


In the past, one of the most well-known Russian-language monitoring services was unfairly overlooked. After registration, the account is $ 1, which is enough for a certain number of checks.



Availability is checked in the following ways:

  • HTTP / HTTPS (HEAD, GET, POST methods);
  • FTP (with authorization and without it);
  • the performance of MySQL and PostgreSQL databases;
  • SMTP, POP3, IMAP;
  • DNS (A, CNAME, MX, SOA), it is possible to check the presence of a domain or IP address in blacklists;
  • Ping;
  • Telnet.



For notifications, email, Telegram, ICQ, Skype, Jabber, phone numbers for SMS or calls, HTTP GET request with a specific key are used.

There are no tariffs, the minimum cost per check is $ 0.00015. Affiliate program is valid.



pingdom.com


Upon registration, free access is given for 14 days, after which one should choose one of the ready-made tariffs. For registration, you must specify a credit card number.



Availability is checked using the HTTP / HTTPS protocol (GET, POST, HEAD methods), the Ping utility, the presence of DNS records, the performance of the selected ports, SMTP, POP3, IMAP mail protocols. Provides the ability to collect statistics on the actions of users on the site.

Notifications can be sent to email, mobile phone by SMS, to PagerDuty, Hipchat, Slack, and there are also apps for Android and iOS.

On the starting tariff plan for $ 14.95 per month, it is possible to run 10 checks at one-minute intervals and one extended check, which includes checking transactions (for example, registering, searching or adding to cart) every 10 minutes and checking page loading speed once every half hour . This also includes tracking user actions for 100,000 single-site views and 50 SMS.



statuscake.com


A free rate is provided with the ability to create an unlimited number of monitors to check every 5 minutes from random locations, imposing significant restrictions on the functionality.



You can check HTTP, DNS records, port availability, SSH, SMTP, Ping, page loading speed, domain expiration and SSL certificate, server health.



Notifications are sent to e-mail, mobile phone as SMS, given by the HTTP request, integration with Hipchat, OpsGenie, PagerDuty, Slack, PublicTweet, Pushbullet, Pushover, VictorOps is possible.

To remove restrictions and increase the frequency of inspections, one of the paid rates is selected.



trafficprotector.com


A three-day test period is included with no functionality limitations.



The availability of the site is checked, the server and traffic are monitored, and it is possible to check the address for presence on the blacklists.

For $ 19.99 per month, you can check 5 addresses at intervals of 5 minutes and send ten SMS.



uptrends.com


You can try the features of this service within 30 days.



Check availability is possible via HTTP / HTTPS (HEAD, GET, POST methods), page loading speed, DNS records.



Notifications come by e-mail or SMS.

At the initial tariff, you will be able to launch 10 monitors with a check interval from 1 to 60 minutes, receive 60 SMS alerts.
