Self-Destructing Notes

Recently MrKable has shared a very useful in my opinion service of notes https://privnote.com/

When you create a note, you get a link to it, which you can transfer to a person. And when reading the note, it self-destructs :), of which the recipient is obviously warned. That is, if the contents need to be saved - then it is necessary to copy it before closing the window.

Plus it is possible to send a notification to the specified mail about reading the notes.

It is an interesting service :)
