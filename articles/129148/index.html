Workshop on the design of applications for Windows Phone



On October 8, at the User Experience conference, we will organize a master class on design applications for the Windows Phone 7 platform .



During this master class will be told about the principles of design and opportunities laid down
into the Windows Phone platform. We consider the features that need to be considered when designing mobile applications, and approaches that are used in the development of top-end applications. Walk through the subtle points and model errors that should be avoided.

Main themes

  • Metro-design, ideology and basic concepts
  • Features of the Windows Phone platform from the point of view of design and UX
  • Innovations Mango-updates that are important for the design of applications
  • Using Expression Blend for interface development
  • Classic mistakes in designing UI and UX applications for WP7

Participation is free. Registration is required . Limited number of seats. Moscow.
