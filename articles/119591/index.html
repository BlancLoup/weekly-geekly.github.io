Tool for offline ordering in the online store

PHPShop online stores open up new online trading opportunities. For example, one of our latest developments was the interactive price list , which is an MS Excel file automatically filled in based on the store catalog, with an integrated virtual basket.

The file is placed in the online store, the buyer can download it and place an order directly from the Excel file. In fact, the buyer can familiarize himself with the whole assortment of the online store and decide on the order without having access to the Internet. The opportunity will please wholesale customers who will get rid of the need to look for the right products on the pages of the online store catalog.

To add to the basket it is now enough to put the quantity of goods in the price column, after which a virtual basket is formed with a transition to the store site, where the order is already being processed. Visually working with the interactive price list is shown in a small video presentation:



Interactive price list allows the buyer:


  1. Perform quick directory navigation using bookmarks with directory names
  2. when you hover on the column with the name of the product and if there is an image, view it from the site
  3. when clicking on a product, go to the full description of the product on the website
  4. add products to the cart in any quantity;
  5. create an order basket;
  6. go to the site with a ready order for registration.


Personalization



The owner of an online store can personalize the price list, setting the output settings for pictures and price columns, the number of goods, the size of pictures, units of measurement, as well as select a specific color scheme.

The price list file (.exe) and the macro are digitally signed, protecting them from being blocked by antivirus.

In addition, for PHPShop stores with 1C: Enterprise synchronization, there is an interactive price list that fills up on the basis of the nomenclature. Both price lists, together with the utility Shop2D, form a single software package for off-line orders.
