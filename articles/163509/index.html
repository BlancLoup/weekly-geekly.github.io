A reader of quote books that saves you time and traffic



Special uniqueness:


  • Picture support (xkcd, bash, something else)
  • Quotes are updated via a dedicated update server that:
    • much faster than parsing html / xml on device
    • saves traffic by compressing and not reloading what is already loaded
    • ALL quotations are updated with one request, 2-3 seconds - and you already have 200 new quotes
  • All quotes form a single tape, but if you wish, you can read any quote book separately.
  • Integration with twitter (well, like without it) + all urls are pressed through bit.ly, so more useful text will fit into one tweet.
  • The reader is initially charged with 512 quotations, so even in the wilderness without the Internet there will be something to read.


Initially I decided to write my reader because:


  1. tired of waiting when already one well-known reader reads all its sources
  2. at the same time, it also closes everything with the modal loading window, although it would be possible to let the user read what is, while the new one is loading
  3. in case of a hangup of the network / site / xs of which - the whole program hung, completely
  4. Some quotations (sramu.net) in RSS and on the main quotation are not given entirely, but the half with the link "read more" - I still wanted to read such quotes also normally and without unnecessary clicks
  5. Well, comics are also interesting to be


Some technical facts


  • It took half a year to write
  • I read quote books myself every day
  • AppStore rejected a similar my application, bastard
  • backend: nginx, ruby, Sinatra, memcached
  • Hosting: Amazon EC2 small instance
  • IDE: VIM, rake
