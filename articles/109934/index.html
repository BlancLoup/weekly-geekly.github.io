New version of open source e-commerce engine has been released nopCommerce 1.90

nopCommerce, judging by the number of CMS downloads from the web application gallery on the Microsoft website , is one of the most popular .NET engines, about which there is almost no information on Habr√© (except for one post and several mentions). In this post a little more detailed information about what it is.

nopCommerce is a free ASP.NET open source e-commerce engine. The system began to be developed by several Russian programmers from Yaroslavl, who released the first release about two years ago. Since then, the system has been downloaded more than 200 thousand times, and the development team has increased to seven people and has become international (Russia, USA, UK, New Zealand and Denmark).
Today, more than 10 thousand sites operate on the basis of nopCommerce. Some interesting examples of sites are on the main site of system developers: www.nopcommerce.com/showcaseliveshops.aspx

System functionality

The system contains the basic functionality necessary to launch an online store: product management and order processing, discounts and coupons, reports and mailings, support for various payment modules and online services for calculating delivery, access control system (ACL). A complete list of implemented functionality can be viewed on the page on the main site of the system .

Installation in a few clicks

On December 7, a new version of nopCommerce 1.90 was released. You can install and try the system very simply - for this you can use the Web Platform Installer tool - version 1.90 is available only in the third version of WebPI. On the information page , version 1.60 is still available. This page will be updated only after the New Year holidays.
System requirements nopCommerce available on the main site .

Future

The system is actively developing: in the next version, scheduled for next spring, the system will be rewritten under MCV 3 using the Razor presentation engine, as well as the creation of a truly extensible architecture when adding new modules will be possible simply by copying the assembly.
In addition to developer-oriented changes, support for mobile devices, support for multiple stores (multi-store), and paid registration are planned.

If you want to take part in the project as a developer or tester, nopCommerce developers will be glad to receive your help .

Official site of the project: www.nopcommerce.com
