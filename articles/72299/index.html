Nimble staring for Nautilus

In his endless search to find a quick peep of graphic files, I came across the wonderful program Gloobus , which will certainly please the happy owners of netbooks with Linux on board. Immediately make a reservation, everything is sharpened under the file manager Nautilus, but no one bothers to screw this miracle to anything.



ATTENTION: For the correct binding of Gloobus with Nautilus, you need a patched Nautilus (it is installed automatically according to the instructions below), but there is one BUT! He slightly modified the interface, apparently to minimize the space eaten on the screens of netbooks.
See the picture at the end of the article.

I was pleasantly surprised both by the speed of work (relevant for netbooks) and omnivorous - the developers were not limited to just graphic files. Gloobus "digests" XCF, PSD, MPG, AVI, MP3, PDF, DOC, ODF, XLS, PPT, ODS, CBR, CBZ, ICNS, TXT, C ++, C, JPG, PNG, SVG, FOLDERS, and some more formats .

Two views are available: plain (fast) and coverflow (covers, as in Apple applications), and only normal ones are available from the native repository.



Installation

Add a key

sudo apt-key adv - keyserver keyserver.ubuntu.com --recv-keys 0624A220
sudo gedit /etc/apt/sources.list

in a text editor, insert at the end of the line:

deb http://ppa.launchpad.net/tualatrix/gloobus/ubuntu jaunty main
deb-src http://ppa.launchpad.net/tualatrix/gloobus/ubuntu jaunty main

Save, close. We update the list of sources.

sudo apt-get update

Install Gloobus

sudo apt-get install gloobus-preview

Update the system to get patched Nautilus

sudo apt-get upgrade

And restart Nautilus

nautilus -q
nautilus &

For ubuntuvodov have the ability to install directly from Ubuntu Tweak



Further instructions on how to install Ubuntu Tweak, who is not installed. Plunge into the console:

sudo apt-key adv --recv-keys - keyserver keyserver.ubuntu.com FE85409EEAB40ECCB65740816AF0E1940624A220
sudo gedit /etc/apt/sources.list

and in a text editor we insert two lines at the end depending on the version:

Ubuntu 9.10 Karmic:

deb http://ppa.launchpad.net/tualatrix/ppa/ubuntu karmic main
deb-src http://ppa.launchpad.net/tualatrix/ppa/ubuntu karmic main

Ubuntu 9.04 Jaunty:

deb http://ppa.launchpad.net/tualatrix/ppa/ubuntu jaunty main
deb-src http://ppa.launchpad.net/tualatrix/ppa/ubuntu jaunty main

Ubuntu 8.10 Intrepid:

deb http://ppa.launchpad.net/tualatrix/ppa/ubuntu intrepid main
deb-src http://ppa.launchpad.net/tualatrix/ppa/ubuntu intrepid main

Ubuntu 8.04, or earlier:

deb http://ppa.launchpad.net/tualatrix/ppa/ubuntu hardy main
deb-src http://ppa.launchpad.net/tualatrix/ppa/ubuntu hardy main

Save, close, continue in the console:

sudo apt-get update
sudo apt-get install ubuntu-tweak

After that, Ubuntu Tweak will appear in Applications> System Utilities. After launching, we go to Applications> Third-party repositories and turn on the Gloobus item, follow the prompts to install. Ubuntu Tweak will do the rest for you.



And restart Nautilus

nautilus -q
nautilus &
