New version of NetWrix Active Directory Change Reporter 7.1 has been released.

If you work with Active Directory, then surely you have come across our audit program - NetWrix Active Directory Change Reporter . This program tracks changes to AD, generates automatic reports, stores audit data for a long time, and allows you to quickly and losslessly recover deleted or changed AD objects. If you are interested in the functionality of the program, you can read the review .
On Monday, we announced the release of a new version 7.1 , which was created taking into account the requirements imposed by users all over the world. First of all, when creating a new version, we focused on the stable operation of the program in large organizations.

What's new in version 7.1?


  • Improved support for Microsoft Exchange Server 2010
    NetWrix Active Directory Change Reporter is tightly integrated with our other program (NetWrix Exchange Change Reporter), which allows you to monitor changes that occur on Exchange servers. In version 7.1, changes were made that allow you to more fully and clearly capture the changes in Exchange 2010.
  • Improved support for large AD environments
    Stable work of the program in large organizations with an extensive Active Directory structure is another task that our developers faced.
    It is worth noting that these improvements are also included in the free version of the program.
  • Integration with NetWrix Non-owner Mailbox Access Reporter for Exchange
    Now, directly from the program, you can get information about access to other people's mailboxes - this is the responsibility of the NetWrix Non-owner Mailbox Access Reporter for Exchange program, which also recently received a significant functionality update. This ensures complete control of the AD + Exchange bundle.
  • Display the user's primary group in static reports (Snapshot Reports)
    The problem that we wrote in one of our previous posts has been resolved. (We are building a report on the membership of users in AD groups habrahabr.ru/company/netwrix/blog/148472 )
  • Asian language support in report filters
  • New reports on changes in AD
  • Ability to set the period for storing audit data in a SQL database
And of course, fixes and other improvements to the new version!

Both versions of the program (free and full) are available for download. You can try the full version for 20 days.
