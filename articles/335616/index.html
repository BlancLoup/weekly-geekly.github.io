Graphic Designers for the Mind and Soul: 20 New Lessons

Even the most experienced and knowledgeable designers never stop learning, tutorials will tell you how to use new technologies, functions, tools and methods. Perhaps the below listed techniques can save you time and energy, improve your skills and abilities. And remember: repetition is the mother of learning.



Here are some graphic design lessons that have come online this year. The list is divided into tutorials on Illustrator, Photoshop, InDesign and for other programs.

Adobe Illustrator


Teaching materials for Illustrator. Despite growing competition with applications such as Affinity Designer and Sketch, Adobe Illustrator is still a tool - the number one choice among graphic designers and studios around the world. Here are six of the best tutorials / lessons for Illustrator that were seen this year.

1. How to create a geometric pattern




Here, briefly, Chris Spooner from Spoon Graphics demonstrates how to create a geometric pattern in a vector. You can either follow the step by step guide to create the same pattern, or turn on the fantasy and create your own work.

2. Master class. Dynamic gradient



In the lesson from the Russian artist Alexandra Zutto there are 16 simple steps that will demonstrate with an example how to create something magical from simple forms. The only thing you need is patience. Soft gradients can add depth and beauty to your vector graphics.

How to optimize and export SVG image




By following this guide, your vector images will look perfect on websites. In this tutorial for Illustrator, Ivaylo Gerchev explains how to properly prepare SVG illustrations in Adobe Illustrator.

4. How to create waves from digital particles




To create this amazing effect, use the Blend tool in Illustrator. In this tutorial, Chris Spooner explains how to create fantastic abstract wave graphics using the Blend tool. Initial elements and effect are created in Illustrator, you can improve the image by working with it in Photoshop.

5. How to create graphic shapes




You can learn the basics of designing graphic shapes with expert Ben O'Brien.
In this 10-step lesson, illustrator Ben O'Brien will guide you through the process of creating graphic shapes that you can fit into the scene effortlessly. Careful attention is paid to body language, physical proportions and the positioning of various parts of the body.

6. How to create an easily editable retro text style




Create editable but appealing text using the Appearance panel in Illustrator.
Chris Spooner shows how to use fills and strokes in the Appearance panel to create a permanently defined graphic text that is editable. As a result of the lesson - a set of fashionable, "3D" letters in retro style.

Adobe Photoshop


Adobe Photoshop is omnipresent, recently it has become a verb, a noun, and also a concept meaning any edited image. Something like Xerox) A popular software for design professionals, below the last six and best Photoshop lessons available on the Internet.

7. Fast selection in Photoshop




Luke O'Neill explains three ways to select objects in Photoshop. It's no secret that a large volume of everyday work in a design studio focuses on cut-out and masked skills, which is inherent in every designer. This lesson describes the three main methods of selection, as well as how to use masks to create cool effects.

8. How to make a GIF




Monika Zagrobelna in this tutorial for beginners will show you how to create a GIF. An animated GIF image in Photoshop is amazingly simple.

9. How to create a repeating pattern




The process of creating a repeating pattern in Photoshop. In this tutorial, Christopher Fin demonstrated how to break an image into squares in Photoshop, and then use it to create a repeating pattern.

10. How to quickly create a two-tone text effect in Adobe Photoshop




At the moment, two-tone text is one of the main trends in design. This tutorial in five steps shows how to use textures, a gradient map, adjustment layers, and filters to create a Duotone text effect.

11. How to create the illusion of an endless picture




The concept of ‚Äúpicture in a picture‚Äù is an optical illusion, the creation of which in this lesson is demonstrated by Melody Nieves. Creating an effect does not require something complicated, just simple methods for manipulating photos.

12. How to create a text-title movie in vintage style in Photoshop




In this lesson, Chris Spooner demonstrates how to create titles for films in the style of old black and white films from the 30s-40s.

Indesign


Below are 4 lessons for InDesign, which appeared in 2017.

13. How to create a cover for a book using InDesign




Laura Hawk in a lesson from 11 steps shows how to create the perfect cover for a book in the Adobe InDesign program.

14. Mixed Ink Swatch in Indesign




A lesson from Joe Gulliver, Senior Editor, in which she explains how to create ink samples and color groups using Mixed Ink Swatch in Adobe InDesign. It allows you to mix pantones among themselves and with process colors, forming new color swatches.

15. How to design a typographic poster using InDesign




Want to create a poster in InDesign? The lesson by Joe Gulliver shows how to create a typographic poster using a grid of documents.

16. Some subtleties in the process of printing covers. Indesign





If you really want to complete your printing project, you need to explore various ways of finishing your printed products. In this tutorial, Joe Gulliver explains the processes she uses when creating covers in InDesign (varnishes, ink blocking sheets, embossing and hot stamping).

17. How to add video to interactive PDF files in InDesign




Yes, you can add videos to PDFs and this lesson from Pariah Burke demonstrates how to do it.

Lessons for other programs


Below are links to useful lessons in other programs (not for Photoshop, Illustrator or InDesign).

18. How to create a retro logo using Affinity Designer




How to make a logo using a popular alternative to Illustrator - Affinity Designer.

In the lesson, Simon Middleweek explains how to use gradients and effects to create a logo in the 80s style.

19. How to create your own font




In this tutorial, Julia Sokolova talks about creating his first font, Bananito, from beginning to end. From drawing the initial design, transferring the drawing to a computer and turning it into a font using software - FontLab or Glyphs.

20. Create AR-style graphics with After Effects




After Effects contains several powerful tools that will help you create augmented reality graphics (AR), in this lesson Laura Hawk explains how to do this.
