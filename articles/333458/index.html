PHP Digest number 112 - the latest news, materials and tools (June 26 - July 16, 2017)



Fresh selection with links to news and materials. In release: PHP 7.2.0 Alpha 3, many new tools, Machine Learning for PHP, tutorials on asynchronous PHP, and much more.
Enjoy reading!



News and Releases


  • PHP 7.2.0 Alpha 3 - The last alpha in the loop. The first beta release is scheduled for August 20, which means no new features are expected. You can try PHP 7.2 with the help of a prepared Docker image or by collecting it from the sources according to the instructions .
  • PHP 7.1.7 , PHP 7.0.21 , PHP 5.6.31
  • PHP TestFest 2017 - As part of the PHP TestFest initiative, PHP user groups around the world will write tests for PHP itself. Fresh screencasts on building and writing tests for PHP.
  • Superjob PHP-meetup - July 20, Moscow.
  • Symfoniacs # 12 (07/27/2017) - Moscow meeting of PHP / Symfony developers visiting Skyeng.


Php


  • RFC: Allow loading extensions by name - A small improvement adopted in PHP 7.2 that will allow you to load extensions by name, making the configuration cross-platform.


Instruments


  • paragonie / chronicle - Microservice based on the Slim Framework, which allows users to save information to the public unchangeable append-only journal a la blockchain. More in the post . Chronicle possible use example: Packagist meta data publication .
  • tideways / php_garbage_stats - An extension for accessing the statistics of the garbage collector in PHP. More in the post .
  • tacnoman / dephpugger - A tool for debugging PHP code from the command line without an IDE. Read more about using in the post .
  • serializededitor.com - A visual editor for serialized data in PHP. Post from the authors in support.
  • edsonmedina / php_testability - Static analyzer for testing for testability.
  • spatie / browsershot - Library for creating screenshots of web pages based on Headless Chrome.
  • ambionics / phpggc - A tool for creating strings to attack unsafe deserialization in PHP projects. An example of using to attack on Piwik in a post .
  • cweiske / jsonmapper - Hydrator for converting nested JSON structures into PHP objects.
  • kreta / Kreta - Project management tool implemented in the form of microservices on the principles of DDD.
  • phpsocialnetwork / phpfastcache - A powerful component for caching with support for many drivers.
  • queue-interop / queue-interop - A proposal for a standard for working with message queues.
  • moeinrahimi / monolog-telegram - Handler for Monolog, which will send logs to Telegram.
  • analogueorm / analogue - Data Mapper ORM.
  • daveearley / Email-Validation-Tool - Library for PHP 7 that implements maximum validation of email addresses
  • fightbulc / moment.php - Library for working with date and time, internationalization support, inspired by moment.js.
