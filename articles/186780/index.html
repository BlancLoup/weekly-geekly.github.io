Meet Intel Integrated Performance Primitives 8.0

Many software developers are well-known library Intel Integrated Performance Primitives (Intel IPP) - a set of functions for optimized data processing and multimedia. By actively using the features of the micro-architecture of processors, in particular, the possibility of vectorization, the functions of Intel IPP significantly increase the speed of the application; besides, application development is accelerating directly, because the programmer has ready-made ‚Äúbricks‚Äù at hand and there is no need to reinvent the wheel. Intel IPP implements hundreds of widely used functions for image transformations, statistics, computer vision and a dozen more areas, all of which are optimized, debugged, and are not burdened with royalties.
And now the functionality of the product has expanded further - a new version of Intel IPP 8.0 has been released .

Intel Integrated Performance Primitives 8 is significantly different from version 7 - no less than Windows 8 from Windows 7, only, we hope, for the better.
  • Now IPP supports Windows 8 new UI (formerly known as "METRO") through static single-threaded libraries.
  • Support for OSX * has been added to the IPP package (previously available only as part of OSX * Composer)
  • Optimization for Haswell microarchitecture has been added to many functions.
  • Additional optimization of image processing functions using SSSE3 and higher.
  • The main distribution includes examples of the use of IPP, previously supplied separately. Added new examples.
  • Expanded documentation for use with Android.
  • And finally, IPP becomes heterogeneous, that is, it gets the opportunity to run on Intel Graphics! The corresponding version of IPP is available in the ‚ÄúPreview‚Äù mode, and in two versions: as an asynchronous C / C ++ library and as an OpenCL-based library.
