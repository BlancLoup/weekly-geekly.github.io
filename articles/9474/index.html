Ext JS 1.1 dev release

For the day, the famous javascript Ext JS framework library has been updated to version 1.1. Despite the imminent appearance of version 2.0 (I really want to hope that it will be soon) Jack & Co. continues to work on the branch of the first version.
ExtJS version 1.1 (beta1) can be downloaded from the library website: extjs.com/download

From the "improvements" you can highlight the appearance of the convenient widget "HtmlEditor", the correction of many errors and most importantly: now the library can be used in the stand-alone mode, i.e. without using third-party libraries (which, however, does not interfere with continuing to use them)



Sources: extjs.com and extjs.ru
