Firefox 31 release

July 22, the release of the stable version of Firefox 31 for desktops and mobile devices based on Android.

Major changes:
- A new system to block the download of files containing malicious software using the Google Safe Browsing API ;
- for the Windows platform, built-in handlers ogg and pdf have been added (can be redefined);
- eliminated a number of critical vulnerabilities ;
- when you open a new tab now displays a search field (by default, Google, you can change);

- Added new features ECMAScript 6;
- Web Video Text Tracks support is now enabled by default;
- improved developer tools;
- support of variables in css blocks with the prefix "-" (previously proposed "var-");
- Added support for the Prefer: Safe http header to support the parental control feature.

Changes for developers .
