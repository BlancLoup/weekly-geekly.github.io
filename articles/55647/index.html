Opera browser - already with geolocation support



This year promises to be very fruitful for the appearance in the Opera browser of new functions that have never before been presented in such applications. Most recently, Turbo technology was announced , and today the Norwegian company announced another innovation - integrated support for geolocation services. Naturally, such a complex technology could not appear from scratch - Skyhook Wireless, one of the leaders in this area, as well as the developer of the original hybrid positioning using all available technologies - Wi-Fi, GPS and cellular networks - acted as a partner.

It is no secret that positioning technologies are not something special today - there are a lot of projects that use geolocation for their services. Meanwhile, the new operatic development has one important difference: the browser has built-in support for the new standard developed by the international committee of W3C, which is the Geolocation API program interface. This technology allows absolutely any service to start using geolocation technologies, adding only a few lines of JavaScript code to its web page. When you enter this page using the Opera browser, the user will be notified of the presence of an integrated positioning system, and after agreeing to use this feature, the system will be ready for operation. As a result, this may be the display of cafes, shops and other places of interest that are of interest to the moment, closest to the user.

Another plus of cooperation with Skyhook Wireless is that the software developed in it can work on any platform. True, this test version of Opera is designed so far only for the Windows XP operating system, but in the future geolocation will be available to all Opera users.
