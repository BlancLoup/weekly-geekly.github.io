The source code of the Doom III engine is available under a free license.

John D. Carmack, one of the founders and lead developer of id Software, announced on their Twitter availability of the source code for the game Doom III.



The company for a long time could not publish the source code of the engine in connection with the use in it of the patented shadow calculation algorithm from Creative Labs , the publication of which could have been an offense. Now, this part of the code has been rewritten by lawyers from Zenimax, the parent company of id Software, they have no complaints about it.

The source code is published on GitHub and on the official FTP of the company under the GPL v3 license (the version chosen by the same lawyers Zenimax; Carmack did not figure out why). Game data is still proprietary and distributed for a fee.

Doom III is the latest game of the iconic Doom shooter line developed by iD Software since 1993. For the game, a specially created 3D engine id Tech 4, which is quite relevant in our time, despite the fact that Doom III was released back in 2004. By tradition, the source code of each new engine of this company is released under a free license, with the right of commercial licensing, after the release of a new technology that remains closed and commercial until the next one.

John Carmack is the non-replaceable developer of all key id Software technologies, including the id Tech engine of various versions.
