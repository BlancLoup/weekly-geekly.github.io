DeepMind has opened free access to a virtual machine learning environment



Recently, representatives of the DeepMind division (now part of the Alphabet holding) reported on providing free access to developers to the source code of the DeepMind Lab platform. This is a Quake III based machine learning service that is designed for artificial intelligence training. Namely - learn how to solve problems in three-dimensional space without human intervention. The platform is based on the Quake III Arena game engine.

But the open three-dimensional world that can be changed represents a much more promising environment for learning AI than the flat world of graphically simple toys for Atari. AI in the three-dimensional world has clear tasks that are consistently changed in such a way that the experience gained in solving each previous task turns out to be useful for AI in the course of solving the next one.

The advantage of a three-dimensional environment is that it can be used to train computer systems to respond to various problems that a robot can expect in the real world.
