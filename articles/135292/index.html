In order to show how easily and quickly you can develop your own configuration based on the Client Base program, we created and posted a video in which our manager, who does not know programming languages, develops a business automation system for a credit broker for him.

For example, let's take a simplified technical task, for example, the following:
We are engaged in the selection of optimal loans for individuals. We need:
  1. receive applications from the site (they must immediately get into the system and transferred to work)
  2. draw up new applications in yellow, green in work, red overdue
  3. make reminders on requests
  4. applications must be distributed among responsible managers (everyone can edit only their own applications)
  5. automatically carry out deductions to determine the best loan in each application
  6. keep track of customers and the history of working with them
  7. send commercial proposals for each application in PDF format
  8. keep records of all actions of employees in the system
  9. send SMS when changing application status
