Briefly about the PhysX SDK


The PhysX SDK routine software allows game developers to avoid writing their own program code to handle complex physical interactions in modern computer games. On July 20, 2005, Sony licensed the PhysX SDK for use in its PlayStation 3 seventh generation gaming console. The PhysX SDK is being actively offered for sale (licensing) to third-party companies, currently more than one hundred games are used.

The PhysX Physics SDK engine consists of three main components for physics processing:

- processing of solids;
- fabric treatment;
- processing of liquids;

On February 13, 2008, NVIDIA acquired Ageia, making the PhysX SDK the property of NVIDIA. PhysX SDK support has been integrated into the CUDA structure, for which there are already many Linux drivers. Thus, the need for a dedicated physical processor PhysX disappeared. PhysX SDK support will be available for all Nvidia video cards starting with the 8xxx series. The PhysX Physics SDK engine is now known as the NVIDIA PhysX SDK.

In March 2008, Nvidia announced that it would make the PhysX SDK an open standard accessible to everyone.
