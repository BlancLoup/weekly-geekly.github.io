For finding a vulnerability in Adobe Flash Player, a reward of $ 100,000 has been announced

Adobe added isolated heap to Flash. This month we pay $ 100K (with sandbox) and $ 65K (without sandbox) per #exploit bypassing this mitigation

- Zerodium (@Zerodium) January 5, 2016


The company Zerodium announced a prize of $ 100,000 to those who find the zero-day vulnerability in the new version of Adobe Flash Player. The latest version of the player, released in December, introduced protection through heap isolation. This method should enhance the security of the program.

In December, Adobe announced the release of a new version of Flash Player, in which the memory manager was completely rewritten. Isolating these processes in memory makes it difficult to hack a player. The ideas and part of the code for this were contributed by security experts working on a Google project called Project Zero .

Adobe Flash Player is famous for its bad reputation as one of the least secure programs. In 2015, in terms of the number of vulnerabilities, it was second only to Apple OS X and iOS OSes. The once very popular and virtually the only means to use interactive web animation, the flash player is now losing ground to more modern means - for example, the new HTML5 standard.

Zerodium periodically buys information about software vulnerabilities from various security specialists and resells them to government agencies.


Price list of the company for vulnerabilities in various software products

This, however, is not the biggest reward for zero-day vulnerability. The same company Zerodium last year announced a reward of one million dollars for remote hacking iOS 9 - and one team of hackers even won this prize .
