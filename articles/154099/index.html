New for web designer in October 2012

I continue the September selection of new useful pieces for web designers.

Services and tools


Cut & Slice me is a free Photoshop plugin that helps you quickly prepare design elements for various types of devices.



Webplatform.org is a huge documentation base on all web technologies from Apple, Adobe, Facebook, Google, HP, Microsoft, Mozilla, Nokia, and Opera.



Mobile Mozaic - a gallery of excellent application interfaces for mobile devices.



Ribbon Builder is a service for quickly creating beautiful stripes.



Viewport Resizer is a bookmarklet tool for quick testing of responsive design.



Endless Icons - a site where quality icons appear every day, designed in the same style.



World Cloud Generator is a service for creating a beautiful cloud from keywords.



Responsive Sketchsheets - ZURB PDF templates for rapid prototyping of responsive design on paper.



Simply Testable is a website validation service with Twitter Bootstrap interface. Checks a lot of pages.



Plugins


jBar - plugin to create a simple and beautiful horizontal bar.



Pikaday - plugin to select the date.



Sly is a flexible and simple plugin for creating convenient image scrolling.



Spin.js is a useful plugin for creating a bootable "twist" of the desired size and shape.



SEP Tooltipster - jquery plugin that helps to quickly create beautiful tooltips.
