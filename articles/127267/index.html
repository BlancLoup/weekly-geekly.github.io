How to install fonts in e-books Alphabet n516 and Alphabet n618

Often, users complain that, say, there is only one font on our e-books, although they do not know that any font can be customized. We decided to fill this gap, as people not only suffer with inconvenient and inappropriate fonts, but also do not read the instructions :)

In addition to the many built-in fonts that are available in ABC readers (Tahoma, Arial, PT Sans, Droid, Courier, etc.), you can easily install new reading fonts that you like.

The sequence of installation of new fonts in the e-book alphabet ABC n516

1. Put the .ttf format fonts in the created .fonts folder on the SD card (in our video example, this is the font verdana.ttf)
2. Unplug the card from the PC and reload the book using the Reset button on the back of the book.
3.When you select a text file that you want to read in the new format, press the "M" button, which will be accessible by the "Settings" menu. Click on the ‚ÄúFormat and Style‚Äù button, then select ‚ÄúFont‚Äù and in the list find the font you have installed verdana.
4. After confirming with the OK button, you can read the book in a convenient font.
ATTENTION:
If you cannot save a folder with a dot (you are using a PC with MS Windows installed), then use the following advice:
Click the "Start" button -> "Run ..." (Run) -> run "cmd" command (enter cmd in the line) -> execute "md : .fonts" command (type md in the command line e: .fonts, where "e" is the name of the removable disk of your memory card).

The sequence of installation of new fonts in the alphabet ABC n618:

1. New fonts must be placed in the root directory on the SD card in the .ttf format (in our video example, this is the font cambria.ttf)
2. After this, the font will be available from the ‚ÄúSettings‚Äù section, the ‚ÄúFonts‚Äù tab.
3. You can translate the new font installed by us into the list used when reading with the ‚ÄúAdd‚Äù button.
4. Go to the desktop of the book and go to the "Files" menu. When reading books of text formats Fb2, rtf, doc, html, chm, you can use the new text font by pressing "Menu" and selecting "fonts". In the drop-down list of fonts you will find a new font cambria and using the OK button you can apply it for reading.

ATTENTION:
Save the font file on the memory card if you need it later.
After deleting the file from the card to use it in the future, you will need to load the font again.
