Microsoft acquires Xamarin



Friends, great news caught up with us right in the middle of the work week!

Microsoft has officially announced the acquisition of Xamarin - the largest platform for developing mobile applications.

Xamarin allows developers to use all the advantages and capabilities of the .NET platform for creating mobile applications using the C # language, and provides complete sets of native APIs to use the capabilities of each of the mobile platforms for which the application is created.

We have been working with Xamarin for a long time and jointly managed to implement integrations with Visual Studio, Microsoft Azure, Office 365 and Enterprise Mobility Suite, in order to provide developers with a full cycle for building native cross-platform applications.

Visual Studio and Xamarin together are a powerful combination of technologies that allow developers to create mobile applications in C # and provide the user with a fully native experience of mobile applications on leading platforms - iOS, Android and Windows.

And by tradition, more information you can find out at the end of March at the largest conference // build from Microsoft.
