We listen to Podcasts and redirect them to stream.


Recently there was an opportunity to listen to Podcasts, but it was clearly not a bright feature after the release, and little was mentioned about it, but you should definitely pay attention to it:
CTRL + P, Show settings (All). Playlist - Search Services - Podcasts , and add a podcast feed.
After that, just go to
- Media - Services Discovery - Podcasts (For loading added tape)
- Media - Services Discovery - Configure Podcasts (To edit and add new tapes.)



If this is not enough for you, and you want to redirect one incoming podcast stream to another computer, or broadcast to a local network, it‚Äôs enough to set up the podcast broadcast to the network (I will give an example on ‚Äúhttp‚Äù, although for this purpose we recommend ‚ÄúUDP‚Äù. just listen, and ‚Äúhttp‚Äù was your remote management interface)

Making ScreenCast With VLC


In order to shoot video from your screen, it is enough to do the following:
1. CTRL + C, - Capture Mode (Desktop)
2. Choose the number of frames with which we will do the shooting. Next two times Next .
3. In the section N ew Destination there are a lot of opportunities where you can shove what we are going to take off the screen.
- Save locally to file;
- Broadcast to the network ( It is possible to use for training remote employees or friends );
and you can also broadcast on different protocols and save to a local file at the same time, if you have both the power and the channel to allow it.
4. Next, select / edit codecs and click " Stream ".
