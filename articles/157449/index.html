Microsoft is going to dismiss Live Messenger in favor of Skype



Microsoft is going to retire its Windows Live Messenger client in favor of Skype. The Verge has learned from several sources that the Windows Live Messenger service will be disabled and integrated into Skype in the coming months.

Over the past few months, Microsoft has gradually moved Skype to the Live Messenger server capacity. About 80 percent of all messages sent via Skype are processed by the Messenger backend.


According to sources, the company will soon announce the disconnection of Windows Live Messenger, possibly already this week.
A more profitable move, compared with the global failure of Windows Live Messenger, is to link your Microsoft account with Skype logins and passwords. Skype users version 6.0 for Mac and Windows already offer the ability to log in with a Microsoft or Facebook account.

The Verge turned to Microsoft for comments, maybe something will become clear.
