Plug - home cloud storage



Today, cloud services are very popular, and a home cloud storage project, the Plug , has raised more than $ 600,000 for kickstarter (the project collected just $ 5,000 in just five days). The cheapest places for $ 59 ended, but remained at $ 79.

It is necessary to connect the home Internet and external USB-drive to the device, as well as install a special application on all connected devices.



There are several advantages over Dropbox:

  • Storage is in your home, which is much safer than trusting your content to some service.
  • For the same reason, the data transfer rate is much higher if you are at home.
  • Unlike cloud services, files are not synchronized, but stored in one place.
  • The cost is lower, the volume depends only on the capacity of your USB drive.




You can connect multiple USB disks to the device.



The software is not much different from cloud service applications. You can make backups, share files, etc.



The rest can be read on the project page on Kickstarter

Technical details:

  • Linux embedded
  • x86 compatible processor
  • Ethernet port 10/100 Mbps
  • USB2 port
  • Supported file systems: NTFS, HFS +, Ext3 / Ext4, FAT32
  • Supports up to 8 USB drives
  • Dimensions 70mm x 33mm x 25mm
  • Weight 200 g


It is also interesting to see the 3D models of the device: one and two . Start delivering the device promise in December.
