"Google Now" on all versions of Android

Some semblance of one of the distinguishing features of Android Jelly Bean - Google Now - is available on "outdated" devices with Android 2. *.

Google Now is a personalized search service that gives the user information that interests him, depending on a number of conditions, such as the current time or his coordinates - search results, local weather in the form of beautiful cards. If you add a bot with the address guru@googlelabs.com to the Google Talk contact list, it will be possible to give it text commands and receive a response from the Google Now service.

It is clear that all communication takes place in text mode, so the user will not see any decorations, only English commands are supported:

  • Score {team} : Let's search for the score
  • Weather {city} : Tells you
  • Define {word} : Dictionary
  • Translate {word} : Translator
  • Web {query} : Google!
