Regardless of which platform you are developing your mobile application (Windows Phone, Windows 8, iOS or Android), the best backend for it will be the Windows Azure cloud platform.

We invite you to a two-hour webinar on July 31 at 14:00 , during which we will tell you about:
  • how to use ready-made cloud infrastructure for storing data and sending push notifications;
