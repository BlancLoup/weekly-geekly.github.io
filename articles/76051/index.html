NAS i-Stor iS607 Short Review

There is such a network disk array. And I decided to share this joy with the community. Little review.




So let's go


Technical Specifications:


  • HDD: 2x3.5 '' SATA (I or II)
  • RAID: 0.1, JBOD, Spanning
  • Chipset: Storm SL3516
  • Supported protocols: CIFS / SMB, NFS, FTP
  • Host Interface and Data Transfer Rate: Gigabit Ethernet
  • Management: Web-interface and telnet (by default), ssh (from the Optware package)
  • Size: 14 (width) x24.5 (depth) x8.3 (height)
  • Weight: 1.1kg
  • Arm-Processor: FA526id (wb) rev 1 (v4l)
  • RAM: 128Mb
