Monkey Do - an unusual task manager

We present you the most beautiful task lists with unique functionality. This is not just an advertising slogan designed to attract the user's attention. Just look at the main screen of the application - simple, neat, nothing superfluous and at the same time, everything you need.

Getting Started



The main idea of ‚Äã‚Äãthe application is to preserve the atmosphere of notes on paper, what we use every day. White pages, black letters, deliberately careless strikethrough. In fact, a pack of sheets in our pocket.

The first thing I would like to mention is the synchronization. We can keep lists on different devices using the iPad, for example, in the office, and the iPhone in the car. Tasks will be automatically synchronized and stored in the Cloud.
The application has the ability to highlight important tasks by increasing the font size. Everything is on paper: we want to select a task from the general list - we make it larger. If visual gradation is not enough for us, and we want a sorted list, just shake the device, and all tasks will line up in descending order.



If the task is completed, it can zachcherkunt. Moreover, unlike other programs where a visually strikethrough is of a formal nature, in this application you will be like a pencil (or a marker, if you choose) to gradually cross out the task. You will feel as if you are scribbling a task on a real sheet.

Completed ... and not completed tasks can be removed from the list altogether. To do this, we will select the task and on the screen of the detailed description of the task we will run with scissors along the line of the cut sheet - we will hear the characteristic sound of the sheet being cut with scissors. We will cut off the sheet with the task :) - we will return to the task list screen.



Unusually implemented task list filtering. When we create a list, we select the category to which we want to assign this list. Moreover, categories are quite abstract entities - stylish pictograms, the value of which we choose ourselves. In the future, just by these icons filtering is carried out - choosing the category we need, we display only its lists on the screen, thus narrowing the circle of searching for the tasks we need.



What no scheduler can do without is reminders. If the task is long-term and we do not want to forget about it, go to the detailed description of the task, you can set the deadline time. A progress wheel will appear in the task list, which will show how soon the time limit will expire, and when the time limit has expired, the application will notify us that it is time to finish the task.
