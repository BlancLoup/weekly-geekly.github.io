VTB24 and PayPal: now incompatible!

Today, PayPal refused to accept payment from my MasterCard from VTB24. As it turned out, not only me and not only today.
Recently, the VTB24 payment system requires the transfer of a CVC (CVV) code for each transaction, which, of course, was not reported to anyone beforehand.
I do not know when PayPal will reconfigure its system for these changes and I doubt that it will reconfigure it at all.
This is such security at the expense of the convenience of their own clients.

PS I wonder if there is any information about which payment systems transmit this very CVC code and work with VTB24? Bank employees could not name them.
