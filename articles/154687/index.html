A selection of useful for fans of Twitter Bootstrap

In the collection of services, plugins, themes and other usefulness, facilitating the work with Twitter Bootstrap.

Stylization


BootSwatchr - fast styling Twitter Bootstrap.



StyleBootstrap.info is a similar service. Another analogue is the Bootstrap Generator .



Beautiful Buttons for Twitter Bootstrappers - a generator of beautiful buttons.



Twitter Bootstrap Button Generator is a similar service.



Lavish - creating a color scheme for Twitter Bootstrap based on any image.



Jetstrap is an online interface creator on Bootstrap (added by MechanisM ).

Topics


Bootswatch are free themes, there is also a Wrapbootstrap with premium themes.



Wordpress Bootstrap is a free WP theme. A few more: BootstrapWP , Roots , Bootstrap , Bootstrap for Wordpress , StrapPress , Alien Ship .



jQuery Mobile Boostrap is a mobile theme.



Buns


Twitter Bootstrap Photoshop Template PSD - UI for Photoshop.



Bootstrap PSD - similar to the previous one.



Font Awesome is a special icon font.



FB Bootstrap - Facebook-style Twitter Bootstrap. There is also a Kickstarter , Google and Windows 8 style.



Additions


Colorpicker and Datepicker for Twitter Bootstrap.



Date Range Picker - choose a date like in Google Analytics (added by Nikita Shulipa).

jQuery File Upload - file downloader (added by Nikita Shulipa).

Bootstrap Image Gallery - photo gallery.



WYSIWYG editor for Twitter Bootstrap.



BootBox.js is a small js-library for creating beautiful pop-up windows.



Notifications - neat and simple notifications (added by MechanisM ).

Bootstrap toggle buttons - cool switch buttons (added by MechanisM ).

Bootstrap Carousel Animate - js-folback carousel animation for browsers that do not support CSS3 transitions (added by tigerforce ).

Editable - quick editing of content on the page.

BootstrapX - plugin that extends the built-in popover functionality (added alphard ).

Twitter bootstrap UI as a module for Drupal.

MooTools for Twitter Bootstrap.

Bootbox for conveniently connecting Rails 3 (added by Bonch )

miscellanea


Translation of the official website into Russian (added by tigerforce ).

Built With Bootstrap - gallery of sites on Twitter Bootstrap.

Yii Bootstrap is a handy bundle of the Yii framework and Twitter Bootstrap.
