Mandriva Linux 2008 made it even easier to install and configure

10082186.1e5382e0edfd270762e2ae401dc045bb.1192602777.bf9efbe4a1ff87602155ff5233177efa


Press release:
Mandriva announces the release of Mandriva Linux - Mandriva Linux 2008. In this version, much has changed and improved, now Mandriva (thanks to the new base system) supports even more new equipment and is even easier to install and configure, and also includes a number of utilities intended for users who are just starting their familiarity with Linux.




Among the interesting new products in the distribution of Mandriva Linux 2008:

Desktop enhancements:

Compiz Fusion's newest 3D desktop, the result of the collaboration between the developers of Compiz and Beryl. New effects when switching between virtual desktops, grouping and window management will not leave anyone indifferent. The latest achievements of the open source world are kernel 2.6.22, KDE 3.5.7 as well as experimental versions of KDE 4, GNOME 2.20, X.org 7.3, RandR1.2, OpenOffice.org 2.3 and Mozilla Firefox 2.0.0.6.

Mandriva's proprietary utilities to simplify work:

A tool for migrating documents and settings from Windows: a new utility for migrating from Microsoft Windows to Mandriva Linux. The transfugdrake utility, which you already know, has been greatly enhanced to provide easier and faster migration for novice users of Mandriva Linux and users who want to use both systems at the same time.

New optimized control center.

Significantly improved utility for installing rpmdrake applications, which makes the installation of new software in Linux simple and very fast.
The centralized Draknetcenter system for setting up and managing the network has replaced the group of individual applications that were in previous releases of Mandriva Linux.
The new network configuration system includes support for WPA-EAP authentication (or WPA Enterprise) and additional security features.

Hardware Support:

The hardware detection subsystem, with modular IDE drivers and support for a large number of new devices - video cards, sound cards, Wi-Fi chips.

Multimedia:

The Mandriva 2008 Powerpack includes a convenient LinDVD product for watching DVDs. TransGaming Cedega is included in Powerpack and gives you the opportunity to play Linux in the latest popular video games like Battlefield, World of Warcraft, Madden and others.
Mandriva PowerPack also now includes Fluendo modules for correct and legal playback of Windows Media, AC3 and MPEG formats.
A free version of Mandriva Linux 2008 is available for download at ftp.linuxcenter.ru
