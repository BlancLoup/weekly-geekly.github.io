Firefox 2: Safer, Faster, Better

For the third year in a row, autumn has been marked by the release of a new version of the browser , which was able to put up competition for the brainchild of Microsoft. In Firefox 2, you will not find unique innovations. The developers decided to go by improving the existing capabilities and correcting old errors. During the development of FF2, more than 2000 bugs were fixed, some of them occurred over several years.

Most of the new features of FF2 are taken from popular extensions for previous versions. Let us dwell on them in more detail.

Tabs
The use of tabs has been significantly improved. Each tab has its own close button. And with a large number of open tabs, scroll buttons appear. Also, you can view a list of all open tabs.

Sessions
FF2 learned to save sessions and tabs on its own. Users can no longer worry if they accidentally close the necessary page. You can open a previously closed tab from the special menu. In addition, after restarting the browser after installing the extension, the user will be prompted to restore the session, i.e. Restore all tabs open before installing the extension. The same will happen in case of improper closing of the browser. By changing the configuration file, you can achieve the same behavior after each opening of FF2.

RSS feed
Improved and subscribe to the RSS-feeds. Along with the usual Live Bookmarks, it became possible to choose an alternative service for reading RSS. It can be either a program installed on a computer, or one of three web services ( Bloglines , My Yahoo!, Google Reader ).

Built-in search
The new version of the browser allows you to choose the built-in search engines and arrange them in a convenient order. In addition, by entering the first letters of the request for Google or Yahoo! the user will see the list of suggestions based on the popularity of the requests.

Spellchecking
Now in each text field on the web page, you can turn on spell checking. Localizations of FF2 initially include a dictionary corresponding to a given language, and users can download additional ones.

Naturally, the new version appeared in a new look. The interface retains recognizable features, but it has become more modern and harmonious. Fashionable design elements such as gradients, transparency, etc. are widely used.

We should also note the improved performance of Firefox 2 with memory. The so-called memory leaks are well known to all users of previous versions. But it seems that the developers of the new version have successfully coped with this problem. During active use of beta versions, the size of the consumed memory did not exceed 80 MB.
Among other improvements - built-in authentication sites, the use of JavaScript 1.7, "live" headers (Live Titles) and much more.

Summing up we can say that the new version was a success! It is superior to version 1.5 in all respects, and in comparison with 1.0 it looks just revolutionary. It should be noted that the developers of almost all popular extensions prepared in advance for the release of the new version and updated their products.
