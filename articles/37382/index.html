Competitor Analysis
I use the analysis mainly using three tools:
Tulbar Netpromoter http://netpromoter.ru/pagepromoterbar/ . With it you can quickly get the primary information about the site: PR, Tits, robots.txt, the number of links on the page, etc. Knows how to count all this for search results.
A big plus is absolutely free.

CS Yazzle program ( http://yazzle.ru ). The program is paid, but it pays for its $ 130. (I bought cheaper, apparently, has risen in price).
The main service is to check competitor backlinks. You can get the parameters of referring sites and calculate the approximate budget. There are far more small services like pr checking on Google data centers, analyzing links on a page, etc. Of the more unusual, I note the check on the indexing and parameters of the links that you bought on sape.ru.

Program Site-Auditor ( http://www.site-auditor.ru ) Makes an express analysis of the site and shows the visibility of the site in search engines for selected queries.

I would recommend another tool:
Tulbar Seoquake ( http://ff.seoquake.com/index.php?sln=ru ) Very powerful free analysis tool. Shows and compares all that is possible. I rarely use it, as I got used to Netpromoter and I just don‚Äôt need most of the Seoquake information.

Site position monitoring
I use two services. http://www.siteposition.ru/ - for operational verification of the site on some request. Great search depth. There is a paid version for continuous monitoring, but it is not as convenient as the second service I use - http://od.jdev.ru/ . The service is also paid, but more convenient and cheaper than siteposition.ru.
You score a project, keywords, and then you get daily results on an email. Naturally, there are modules for building reports.

Where to buy links?
Of course, on the exchanges, plus something can be purchased in manual mode.

Link Exchange
I use mostly two:
http://Sape.ru - a lot of sites for different tastes.
http://Xap.ru - I use it mainly for low-frequency queries and as an addition to sape.
These exchanges are the most popular with webmasters, there are most sites.

When there is a shortage of the above two, you can look at http://www.mainlink.ru/ . Also quite a big stock exchange. If this is not enough, you should visit http://clx.ru . I used both as an alternative, but somehow did not work out, apparently too used to the user-friendly interface sape.

For those who do not want to learn the new interface, and use the familiar Seipovskiy, you should look at http://www.propage.ru . The interface is very similar. But there are not so many sites like sape and xap.
These exchanges are enough for you to purchase links.

If you need to buy links from a narrow topic resource, you can manually search for such a site on the forums http://forum.searchengines.ru/ and http://mastertalk.ru .
