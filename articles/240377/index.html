Vulnerability of zero day in Microsoft Windows (the patch will be available today)

We notify our customers of a critical vulnerability in Microsoft Windows CVE-2014-4114 (codename "Sandworm"). Fixing the vulnerability will require installing a patch from Windows Update and restarting your server. An update fixing the problem is expected to be available in Windows Update today on Tuesday, October 14, 2014.



Until the patch is released by Microsoft, no additional action is required. After the update appears in Windows Update, you will need to install it and restart the server. The update should fix the security issue. We report an important update in advance so that customers can plan for installing updates and rebooting servers. Infobox will not restart your servers automatically.

Taking care of your safety.
