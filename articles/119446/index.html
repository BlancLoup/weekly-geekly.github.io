Seagate has released a wireless hard drive with battery



Seagate has recently delighted fans of the wireless lifestyle with the release of a new device: a GoFlex Satellite mobile hard drive with a WiFi module. Among other advantages, this hard drive is equipped with a battery, so that the device can really be called mobile. By the way, the WiFi module works with b / g / n, plus the device is equipped with USB 3.0 interface, which allows you to quickly transfer data from the disk to any device and vice versa. Some details are in the sequel.

So, the company also released an application for iOS, which allows you to facilitate the task of interacting with the device for owners of mobile devices. An Android application will also be released soon. All the rest will need to use the web interface for this device, which is also not particularly difficult. By the way, up to three devices can be connected to the GoFlex Satellite simultaneously, which is not bad.

However, there will still be problems with connections. The fact is that if you work with a mobile device, then you can connect to either the hard drive, or, for example, to the access point. At the same time working with the hard drive and with other wireless devices, connecting them with your mobile device, will not work, since the hard drive can work only with the ‚Äúdirect‚Äù type of connection. Seagate, apparently, one of the main advantages of the device considers the function of video broadcast on mobile devices (so far for iOS, as mentioned above).

On one battery charge, the hard disk is able to work up to five hours in the video broadcast mode, while in the standby mode, the device has 25 hours. The capacity of the hard drive is 500 gigabytes, which is not so much, but for a device of this class should be enough. Home sales GoFlex Satellite is expected in July this year, and the recommended price - 199.99 (oh, those nine) US dollars.
