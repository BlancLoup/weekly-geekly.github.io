The first mobile application that I had a chance to announce in RuNet was Filmoteka film cataloger for iPhone and Android . The program is designed only for the Russian-speaking user and is distributed free of charge as a mobile client of the Windows Movie Catalogizer - All My Movies. The application synchronizes with the cataloger on the desktop computer, allowing the film lover to have a catalog of their films always at hand on the home computer and on an iPhone or Android smartphone.





Cataloger finds information about films in the largest Russian-language film base Kinopoisk.ru , which allows you to find data on almost any film or actor. The user can view the description of the movie and transfer it to the movie catalog on a mobile device. In the future, the list of movies can be viewed without connecting to the Internet, which is especially convenient, for example, in a store, so as not to accidentally buy the same film twice.
