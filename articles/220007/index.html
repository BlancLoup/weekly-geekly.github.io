2. First launch


Prior to this case, I already had a zm-ve200, which burned when I tried to solder the usb-connector. There is data and _ISO folder on the hard disk where disk images are stored.
After turning on the hard drive, it was determined immediately and the _ISO folder too, on the previous version of this device there were dances with firmware, since He did not see 500GB drives.
On the numeric keypad there are arrows. Selecting the desired iso image and pressing the number 5, we get a usb cd-rom with the inserted disc. You can change the image on the fly. This is a very convenient feature in the era of ultrabooks and servers without a cd drive.
At the very top of the case is a white LED that blinks as data is exchanged with hdd.
Keyboard Touchscreen. A film is applied on the entire front surface.
The device display shows:
  • Image name
  • Lock - you can or can not write to hdd
  • Device operation mode: cd disk (O), hdd (H), cd and hdd (D)
  • Iso connection indicator
  • Connection Mode: USB2.0 or USB3.0

On keyboard:
  • Numbers from 0 to 9.
  • Menu and Enter buttons


3. Menu


There are 6 items in the menu:
  • Mode select. In this section, you can select the device operation mode (Dual, HDD, ODD)
  • Encryprion. Here the password is set when the device is connected to the PC. Be careful, because All data is encrypted when it is encrypted and canceled!
  • Settings. Device settings: display brightness in active mode, brightness in standby mode, transition time to standby mode.
  • Information. SMART - temperature and health (I have 34 and good), HDD Model, HDD Serial, Firmware Version, USB Speed, USB Input Voltage.
  • USB Connect. In this section, there is Refresh - to simulate reconnection of the wire and Safe Removal - to safely disconnect the device.
  • Advanced. Here you can set the write protection on the hdd, as well as ‚Äúisman‚Äù iso.

4. Encryption


First, select the device menu Encryprion. They agree to the loss of data and set the password. After a while, the device will reboot and prompt you for a password. 10 attempts are given.
Then 1st Partishion: 0 appears on the screen. The disc is not formatted. You need to format the disk and you can transfer files.
After encryption, the data transfer rate dropped from 29 to 25 MB per second over usb2.0.
