Twishort - enhanced features for Twitter

Sometimes it happens that, no matter how hard you try, it‚Äôs impossible to contain a 140-character thought. Sometimes it happens that I want to talk about something, but I have to throw out the details in order to invest in the restriction, and after that it seems that it was better not to tell. Sometimes it happens that you want to quote great words, but you have to cut them, and they no longer sound. A lot of things happen - and it was for such cases that we created Twishort , a service that allows us to extend the posting capabilities on Twitter, and the basis of the project is the ability to write tweets longer than 140 characters.

In order to start using Twisort, you do not need to register - it is enough to have a Twitter account. All you need to do is click the login button and allow access to our service.

By default, text is sent to Twitter, clipped to the allowed number of characters, and a link to the full text on Tvishort. It should be noted that if your body text is less than 140 characters, then the link to it will not be added. In addition, you have the opportunity to independently edit the text, which will go to Twitter.

We also provided a set of characters that are often used, but not accessible from the keyboard (in any case, not easily accessible) - for example, a dash, a copyright sign, quotation marks, and many others. Now it is very easy to use them. In addition, for example, a three-dot allows you to save two characters :)

Having implemented this rather simple functional, we thought about why not to tape our long tweets. And they did. And besides, the tape is provided for each user, and it is convenient.

Your tweets can be edited (for obvious reasons, only the full text is edited) and deleted (the post on Twitter is also deleted). It is possible to respond to other people's tweets - exactly as on Twitter.

Well and, perhaps, the main thing is the tweet page with the full text. To save characters, we tried to make the link to it as short as possible: it looks like http://twishort.com/aaaaa . At this address, anyone can read your full text.

PS Honestly, at first we wanted to start the service next year. But then they thought, and decided that, probably, many people would like to share their New Year's impressions via Twitter - and 140 characters are not enough for good impressions. Therefore, we are launching the project now and wish everyone a happy New Year! :)
