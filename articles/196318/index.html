Registration is open for SQL Saturday November 30 in Moscow

SQL Saturday is a free one-day training on Microsoft SQL Server technologies, which is conducted in various countries by the international community of professional DBAs and developers under the auspices of the Professional Association for SQL Server (PASS). Participation in the event is a good opportunity not only to get acquainted with technological innovations, but also to communicate with colleagues and experts in this field. Traditionally in Russia, speakers are active participants of the Russian SQL Server User Group .
The list of reports is still being made up , however preliminary registration is already open on the page of the event .
