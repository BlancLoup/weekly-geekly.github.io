Vivaldi Browser - the first test version.



Hello!

From the very first day of the Vivaldi.net site, users were tormented by vague suspicions: another web site resembling a social network does not make sense, Vivaldi obviously conceived something, and most likely it will be a browser ... Well, your expectations were not deceived: we really worked all this time to create a browser Vivaldi. Firstly, it was logical after the actual death of the good old Opera, and secondly, we still can do nothing except browsers.

In general, the hour X has come - meet the new browser Vivaldi.



This is the first test version that we decided to submit to the users, so naturally there are still a lot of bugs and shortcomings. But we hope that our users will not remain indifferent and will be able to help us get rid of them as soon as possible. We will also need your help in the process of creating the browser itself - it is the opinion of Vivaldi users that will be taken into account when adding new functions to the browser. However, we already have something to show.

I will not dwell in detail on those functions that have already become familiar to browser users - like the Express Panel (although now you can create your own thematic folders, including nested ones) or sidebar. I think you can experiment and test these functions in your own work. I note only a few interesting in my opinion. For starters - Quick commands.



This dialog box opens by pressing F2 and offers you a quick way to work with the browser. You can search through open tabs, perform various actions in one click, navigate through tabs, navigate to browser settings, and more. Try it yourself - you will like it.
