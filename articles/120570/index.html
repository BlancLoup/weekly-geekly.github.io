Visual demonstration of machine learning algorithms



PhD from the Federal Polytechnic School of Lausanne, Basilio Noris, has created a wonderful program that is perfect for demonstrating some of the tasks that solve machine learning algorithms (classification, clustering, regression - by various methods). In one program collected libraries, algorithms and code fragments that were found. Unlike Matlab, here the GUI works quickly interactively, so it turns out very nicely.

Distribution:
MLDemos 0.3.2 for Windows (minimum requirements: XP SP3)
MLDemos 0.3.2 for Mac (minimum requirements: Snow Leopard)
MLDemos 0.1.3 for Linux 32bit (deb) (build for: Ubuntu 10.04)
