iCab Mobile



The Safari browser built into Aypad is certainly gorgeous, but if you cannot live without the usual tabs, then iCab Mobile will be a good alternative. Clicking on the "+" button in the upper right corner of the application allows you to open a new tab. The application also boasts other features that Safari lacked so much: the ability to search by page, the button that hides the browser panel so that the content of the page fills the entire screen of the device.

This is a fast and cute application, even though it‚Äôs disgusting and you‚Äôd never want to keep it on the main panel.

LogMeIn Ignition



The iPad is not yet a self-contained device, and from time to time you may need to use your main computer while on the road. Say, for example, you want to start downloading a new movie so that it will be downloaded before you arrive, or send an important document to someone that you left on the hard drive of your home computer by email. LogMeIn Ignition is a great application for remote desktop access that allows you to control your computer from afar with ipad.

Installation is quick and simple: create an account on LogMeIn.com, install the application on your computer and log into your system using ipad. You will have access to basic actions on the remote desktop - moving the mouse, keyboard input is a bit inconvenient using the touchscreen, but nevertheless allows you to perform the necessary actions.
Price: $ 30, Download


Simplenote



The built-in notepad application looks great, but is outdated: you cannot view your notes except on the iPad (of course you can send them to yourself by e-mail, but this is at least inconvenient). While we are waiting for Apple to integrate wireless online synchronization into the device, Simplenote is a great alternative. This is a really very simple application: click on the "+" button to leave a note that will be compiled online when you are done. After that, you can immediately view the same note on an iPhone running SimpleNote, or via a web browser on the site simplenoteapp.com. With this application is absolutely free.
Price: $ 0, Download


Air display



Air Display is a handy application that allows Mac users to use Aypad as an optional external display. After you download applications to iPad and Mac, combine them into one Wi-Fi network and voila - your iPad will become another screen for your computer. This is very convenient if you want to select a small Aipad display for viewing feeds on Twitter or perhaps you will transfer an instant messaging program to it while you are doing your main work on the main display.
Price: $ 10, Download


The guardian eyewitness



The Guardian (British daily newspaper, approx. Translator) found another interesting use for Aipad's beautiful display using this application. Every day, The Guardian Eyewitness shows new photos from a high-resolution paper newspaper. Apart from the fact that the photos are magnificent in themselves, a description and professional advice from the photographer is attached to them, about how a particular picture was taken.
Price: $ 0, Download
