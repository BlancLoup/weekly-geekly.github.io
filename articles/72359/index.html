Opera 10.10 beta - now with Unite



Today the first beta version of Opera browser has been released under number 10.10, which includes a new technology - Opera Unite. Official information in English can be read in the latest press release ( www.opera.com/browser/next ), but in Habr√© today the official will be a little less. This introductory part is completed and you can go to the actual technology Opera Unite. Those who have already tried this unusual development in the alpha versions of the browser do not need a special introduction, so I will briefly describe what Unite is for those who first hear this name.

In a nutshell, Opera Unite is a web server built into the browser. With all that it implies. I mean the benefits. This can be visualized as follows: if now, when communicating via the Internet, we usually use a computer-server-computer chain, now, with Opera Unite technology, it is possible to switch to using a direct computer-to-computer connection. Of course, it‚Äôs still not possible to do without servers ‚Äî after all, they manage all the data transmitted through the world wide web, but servers running as file storages or online services can easily be pressed. The data is stored on your computer, the server is built into the browser, and all you need is to download and run the application corresponding to the tasks, right in the browser.

At the moment, in the list of available Unite-applications, you can count two dozen interesting developments - from a simple means of ‚Äúsharing‚Äù files to highly specialized applications intended for a ‚Äúdifficult‚Äù user. Unite applications are created on the basis of simple and clear technologies - HTML, CSS and JavaScript, densely seasoned with web standards. This allows you to open great prospects for third-party developers, and to further simplify the task - there is a fairly extensive set of documentation on Unite technology ( dev.opera.com/articles/unite ). For ordinary users, you can recommend to try the following list of applications:

Photo sharing
Allows you to open photo galleries for your friends, right on your computer, without uploading pictures to the network.
Messenger
Simple chat client for communication.
File sharing
Allows you to open access to your locally stored files (similar to Photo Sharing).
Web server
A very simple application that allows you to run a full-fledged website in a few clicks right on your computer.
Media player
A player that allows your friends to listen to music from your collection.
Fridge
A fun toy for sharing short sticky text notes.

The entire list of available Unite-applications can be found here unite.opera.com/applications

To try Opera Unite at work, you need to do a few simple steps:
1. Download the Opera 10.10 beta browser from www.opera.com/browser/next to your computer
2. Launch Opera Unite by clicking on the Opera Unite icon in the lower left corner of the browser and enter the account information (Opera ID), which is used to log into other Opera services, such as My Opera and Opera Link. If you do not have an account, Opera Unite will offer to create it.
3. Now everything is ready to use Opera Unite. Access to the default Opera Unite applications is provided from a special browser panel.
