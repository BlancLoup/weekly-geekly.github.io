Mozilla introduces Webmaker project and Thimble editor

Mozilla today announced the launch of the Mozilla Webmaker project and Thimble editor.



Thimble (English Thimble) is a new tool that allows everyone with incredible ease to create and publish their own web pages and other projects in minutes.
Thimble simplifies development. This is an intuitive visual editor that allows you to create and edit HTML right in your browser, view and edit your work, and then publish and share fully functional web pages with a single click. Thimble opens up barriers for novice users who have begun to learn coding, and includes a number of start-up projects and templates to help virtually anyone get started quickly.

Webmaker includes projects that help users learn what can be done with Thumble, as well as other tools Webmaker'a. There are already some projects that will help you to do everything - from setting up your blog template, creating interactive video, creating fantastic 3D web pages - including projects from partners such as Tumblr, the London Zoo, the New York Public Library and many others.

Like many other Mozilla projects, Webmaker is also written in python / django and the source code of the project can be found on github .
