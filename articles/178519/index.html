Lightworks Video Editor for Linux - Official Release (Beta)



Great news for all habravchan who loves Linux and edit video! The Lightworks video editor for Linux has appeared in public access! At the moment, the program has the status of a beta version, but according to reviews it works quite stably.

The application is already available for download: lwks.com/betas-linux .

The developers noted the following:
  • Only 64bit systems are supported.
  • Wanted ATI or Nvidia Graphics Cards with Latest Drivers
  • Officially, only Ubuntu distributions (including 13.04 ) and Linux Mint are supported.
  • Not all tools work.
  • Registration is required for installation (free)
  • After registration you will receive a PRO version for 7 days.
