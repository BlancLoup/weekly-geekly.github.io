Many Windows "desktop managers" work according to a similar principle - they hide all windows (almost through SW_HIDE) that are not related to the current desktop (there are no such desktops as such, there are more window sets). As a result, any program opens windows and message boxes on the first desktop, not on the current one, plus a number of problems appear if the program itself hides its windows (for example, when hiding in the tray).
