BitTorrent Sync P2P file synchronization service launched for everyone



After three months of testing , BitTorrent launched BitTorrent Sync to synchronize files and backup using BitTorrent between arbitrary devices.

BitTorrent Sync is more secure than Dropbox and other cloud services, because it does not store files in the cloud, and no company or outsider can access your files. Instead, the user chooses which devices to back up to. All traffic between BitTorrent Sync nodes is encrypted with a 256-bit AES key, which is created on the basis of a random string unique to each folder. The key is generated using the dev / random (Mac, Linux) and the Crypto API (Windows).



Synchronization is possible between devices on different operating systems. The main thing is to specify the same secret key.



Theoretically, the key can be shared with a large community of people, organizing large joint P2P-storage. For example, BitTorrent has opened a public repository with the key RWC4YD3S723QRVHHHIZWJXPTQMO6GKEQR for testing purposes.
