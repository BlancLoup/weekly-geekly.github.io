SSD for serious people

Working moments - check out the new RamSan 620.



Not so long ago, Cooler-Online wrote about this disk array as about some kind of prohibitive miracle. 4 months have passed, and I managed to try it live.

And this is another connection at half the speed of the channel!

Manufacturer Specifications:
* 2-5 TB SLC Flash storage
* 250,000 IOPS random sustained throughput
* 3 GB / s random sustained throughput

For comparison, a regular hard disk is capable of delivering up to 300 IOPS (I / O operations per second). Everyone knows about linear speed - a maximum of 100-120 Mb / s.

Who is interested - I ask the manufacturer's website . There are much more extreme options. But even this company spent some astronomical six-figure sum, of course, not in rubles.

We look forward to when we can try the database server at full load. Unless of course, with such equipment we can reach this limit ...
