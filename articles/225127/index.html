Digest KolibriOS # 5: we are with you again

System-wide changes:


  • FTDI driver;
  • httplib.obj : fix bugs;
  • box_lib.obj : support for [Ctrl] + [End], [Ctrl] + [A] combinations in the t_edit component;
  • msgbox.obj : select option by [Tab];
  • newlib : updated to 2.1.0;


Changes in application software:


  • ThreadInfo : a small utility that displays information about the process when you right-click on a desired process in the process manager;
  • MouseState : a small utility to check the functionality of some mice;
  • Calc + : redesigned design; hide / show the keyboard by clicking on the button in the header (the function is available only in the experimental assembly); calculation history; added operator | (module); removed auto calculation;


  • CPU : start ThreadInfo when you click right on the process;
  • FTPc : added RDIR command; Notify user if connection failed;
  • IRCc : message output when changing the nickname; nickname output when entering the / nick command without a parameter;
  • Setup : fixes for the Belgian layout;
  • Palitra : setting noisy background;
    Hidden text



  • TextEdit : fix horizontal scroll when pasting text more than 100 lines;


Other changes:


  • Added games to LiveCD assembly: QUAKE, DOOM, Jump'n'Bump ; emulators: gameboy, scummvm, zsnes ;
  • Experimental build with a new skin subsystem ;
