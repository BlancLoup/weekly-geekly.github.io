QUIDAM 2.3: creating 3D characters

Hello!

Great news for those working with 3D or amateurs. This is a new version of the program for creating 3D characters QUIDAM.



The program contains a large library of three-dimensional characters, as well as individual elements and accessories, from which you can make models on the principle of identikit. Having collected the character from the available elements, you can correct the facial features using modeling tools. In addition, you can snap characters. At the last stage, the finished model can be exported to formats that are supported by 3ds Max, Cinema4D, Maya, Lightwave, Carrara, XSI, Truespace, and others.

The latest version has added a library of available content, improved QDM import and export capabilities, improved selection tools accuracy, improved Bump channel support for all visualization engines, updated plugins for exporting files from other applications, improved texture work. Source of

The program is not free, $ 270. You can download the demo from here . Works with all versions of Windows and Mac OS X.
