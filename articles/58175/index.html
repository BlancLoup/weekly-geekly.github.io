Has your project already reached the main page of Yandex?

Some have already heard that Yandex has launched a program to support sites with regional content . But the beauty of innovation is that thanks to it, not only regional, but in general any projects can place their block widget on the main page of the search engine:





The developer makes a widget, adds it to the system and receives a link code, through which users can add it to their Yandex main page:

+ Widget? Ok!



When the number of people using the widget exceeds a certain threshold (unfortunately, I forgot: either 500 or 1000), it will go to the Yandex widget catalog. Plus, if it satisfies the conditions of the regional program , the link to the widget (after passing moderation) can be placed directly on the main page of the search engine.

If you already itch to try, here is the link to the documentation section: http://wdgt.yandex.ru/
