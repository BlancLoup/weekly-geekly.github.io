How to download a large map from Google maps

But recently I started planning my trip to Spain by car. It is very convenient to engage in sim obscenity using Google meps . But Google Maps has one major drawback - the work done (trip route) cannot be saved in good resolution with a picture. Rummaging through the internet, I found a wonderful way to save Google maps with or without routed routes. I‚Äôm in a hurry to share this technology with the habrasoobshchestvom.



So, in the beginning there will be preparation.
For this action we need:
1.) Fire Fox
2.) A plugin for a fiery fox called screengrab. It can be freely found by name on the page Edon .
3.) The usual vindosovsky notebook (or any other text editor).

After installing the screengrab and opening the fiery fox, let's proceed.

Step One :
Open Google Map and compose a route (optional, you can simply open the map in the right place).

Step two :
We are bringing the map to the desired resolution (I, for example, needed to see secondary roads). And set the map approximately to the center of the desired area (For example, if you want a city map, then you need to scroll the city center).

Step Three :
Click on the "Link" (located in the upper right corner of the map). And copy the text from the field "Paste the HTML code to use on the site."


Step Four :
Save this text in a map.html file.

Step five, very responsible :
Change the width and height of the map in pixels to the ones you need (for example, width = "2500" height = "5000").
The main thing here is not to be greedy =)

Step Six :
We open this file in the same fiery fox and wait until the whole map is downloaded.

Last step :
When the whole map is loaded, scroll the screengrab button and select save-> complete Page / Frame, thereby indicating to it that it is necessary to save not the visible area, but the entire loaded frame.


It's all.
I hope this life hack is useful to someone other than me.
