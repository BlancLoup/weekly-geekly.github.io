Robo ‚Äì operating system is now on Android



Two years ago, it announced intentions to create a special operating system for robots . It was created and has already been successfully applied, supplemented with new and new "goodies" .

Yesterday, a group of scientists from the Technical University of Munich announced the porting of this OS to Android.


What is ROS?


ROS consists of a kernel to which various nodes can connect. Nodes can be distributed. A typical example of use: sensors and actuators are connected to the core. The signals from the sensors are processed by the core. When a certain event occurs, a signal is given to the actuator.

What does this have to do with Android?


Until now, ROS could only be used in Linux. Now, any android ‚Äì smartphone can act as a mobile sensor, generating signals for the core. To connect to the core, on the phone you need to run the script and enter the ip address of the computer with the core ROS. There is also an alternative option - read the QR code, which contains the ip address. For example, there is a room, the light in which is controlled through ROS. Before entering the room there is a QR code. The user can read this code through a mobile phone, connect to the core and control the light in the room. Video link shows the entire system in action.

Communication of the phone with the core is two-way. Thus, the sensors available on the phone (camera, accelerometer, gyroscope, display) are available to the operator at the computer with the core ROS.
