Firmware 5530 without "frills"

In my last topic on Modified Firmware , many requests were left and a comment was made on improving the firmware. Unfortunately, I didn‚Äôt manage to figure out the firmware change myself (rarely at home), but today I discovered a new firmware version for the Nokia 5530, codenamed Simple Mod v.1.0 by Wayfer. By the way, it is Wayfer that is developing a program for self-changing firmware for 5530, the output of which is not far off (although it was promised before). Less words - look under the cat.

With the development of the program, some problems have arisen, the program is being rewritten, therefore, we meet the firmware, which, I hope, will suit more people.

Simple Mod is a standard firmware with improved performance with virtually no visual changes.



Changes in this version:
  • Breaking protection (Russian RomPatcher; patches: Install Server, Open 4 All, Mega Bass, Charger Notes OFF, Record Tone OFF, Reset On Power Long; at startup Install Server and Open 4 All)

  • Quick effects themes
  • Russian language of the default smartphone
  • Help files removed

  • Delete standard browser bookmarks

  • Increased disk cache size
  • Changed HEAP size and application launch delay
  • Changed folder location with browser cache to E: \
  • Corrected kinetics
  • Updating the media library of the player from E: \ Music
  • Update gallery from specified folders only
  • Hiding the icons of extra programs from the menu
  • The path to save files received via Bluetooh in E: \ Bluetooth

  • Decrease photo compression ratio

Instructions on firmware


  1. Download the firmware here , install it on your computer
  2. Modified files that were downloaded earlier (link above) are moved to the folder where the firmware is installed (by default, C: \ Program Files \ Nokia \ Phoenix \ Products \ RM-504), agreeing to replace the files
  3. Here we download the program for the JAF firmware (there are instructions and reviews).
  4. We connect the phone in PC Suite mode
  5. After installation, run the JAF program from the desktop, ignoring the message BOX DRIVER NOT INSTALLED , it does not hurt us.
  6. Go to the BB5 tab, click the Use INI checkbox, after which our window will pop out, where we select our phone model. Ok, uncheck the CRT 308, set to: Manual Flash, Dead USB, Factory Set.
  7. Turn off the phone, do not disconnect. We press in the Flash program, we agree with the error and by briefly pressing the Power button on the phone, we turn it on. Then the program will do everything for you. After a few minutes, you will receive a new phone.

And again, do not forget to make backups of flash drives, contacts and necessary programs.
