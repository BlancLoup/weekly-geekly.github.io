Ford has released an SDK for its cars.



Ford invites mobile application developers and all car owners to hack their cars. Especially for this purpose, an open platform OpenXC has been created , which provides access to Ford‚Äôs internal sensors via the standard OBD-II interface.

Ford has released an Arduino-compatible microcontroller that connects to the OBD-II port (the so-called CAN translator), and a Java library that runs on almost any Android device.

Thanks to the open SDK, Ford cars become part of the Arduino radio designer. You can add buttons, knobs, indicators, additional displays and control panels to cars - just enough imagination.
