About: about page in Firefox

What is sometimes observed in the address bar of a blank page ‚Äúabout: blank‚Äù, for Mozilla, became the philosophy of access to browser resources. Few people know about this philosophy, except enlightened monks, for whom the address bar is less sacred and more instrumental than for the rest. But it is difficult to remember all the variety of properties of this line. Open the window to enlightenment helps mantra "about: about". Now, at the time of version 37, it opens 34 options for continuations of links after the colon. There are undocumented.

Jokers rub their hands: you can call the most natural puzzling pages with minimal effort. But there are a number of useful pages. The purpose of the list below is to highlight the share of useful pages for an advanced browser user.

There are pages in this list that add various add-ons if they are installed in the browser. For example, about: scriptish, about: stylish-edit, about: treestyletab-group. (Previously, it was about: speeddial, but now for some reason it stopped this practice without changing its useful functions.)

This is even written in another holy book: https://developer.mozilla.org/en-US/docs/about:_protocol or more generally, for Mozilla products - kb.mozillazine.org/About_protocol_links .



Not all of its pages are equally valuable. We describe the most important of them. (It makes sense to open all of them in browsers on the Mozilla engine - Firefox, Seamonkey, K-Meleon, by copying the fat dough into the address bar.)

about: about - a list of the ‚Äúofficially‚Äù valid pages of this pseudo-protocol. Although, for the advanced user, only some of these links are important, listed here.
about: addons - an important page for managing add-ons (for those who use them; there is a call to it from the menu - Tools - Add-ons or Ctrl-Shift-A)
about: cache - current general data about browser caches (how much memory it takes)
about: config - important - browser settings and manual management
about: customizing - call the settings of the browser GUI (as per the View menu - Toolbars - Customize ...)
about: memory - detailed statistics on occupied memory (by processes)
about: newtab - Speeddial in Firefox-ovsky (9 links with partial pages).
about: permissions - Access and rights to previously visited sites and the ability to change them.
about: plugins - plugins (which are not addons)
about: preferences - Surprisingly, these are the settings for Firefox in a slightly different full-page format. Missing only the tab Options --Tabs for some reason.
about: support is a very useful dialog about important browser settings. Here the easiest way to get a list of addons with versions. Although the name is not invoking, therefore the real chance to get to it is to remember the mantra " about: about "
about: downloads - the list of downloads (Ctrl-J (in Win)), but in a tab, not in a special window
