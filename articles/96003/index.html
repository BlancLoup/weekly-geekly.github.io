If anyone is interested in how to transfer their domains with the status of Private Person from mchost.ru write, tell.

ADD at the request of IMA:

How to withdraw from mchost.ru domain with the status of Private Person for 75 rubles.

So, to transfer a domain, you need to get the EPP-code from the domain and change the status from Private Person to normal, so that your email address is specified on the whois page of the domain.

1) We write to domains@mchost.ru request for a topic send an EPP code from my domain.
If you already had this code, then note that the codes tend to become obsolete, so make sure that you have the current code.
2) We write a letter to the support with a request to register an email in the domain description. If you have registered then go to step 6. Otherwise, as in my case, you will be asked to bind the domain to the hosting (on mchost.ru) and create a questionnaire there, then make a request for assigning the profile data to the domain.
3) We buy the cheapest hosting for 1 month. I got 75 rubles.
4) We bind a domain to this hosting and create a form in which we indicate your email
5) We write a request to the support with a request to assign personal data to the domain
6) Go to webnames.ru and start the domain transfer process. Everything is simple there.
