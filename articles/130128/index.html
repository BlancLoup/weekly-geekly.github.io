Web browser in NeoAxis Engine - Integration of Awesomium and NeoAxis

NeoAxis Group, the developers of the domestic game engine NeoAxis , report on the successful integration of the Awesomium library. Awesomium - a framework based on the Chromium / WebKit engine, which can now be connected to NeoAxis as an extension. Using Awesomium, developers can embed web pages that are online in their applications, whether they are games or anything else - thus realizing unique and original gameplay features or game design elements. In fact, this is a full-featured web browser right inside the game engine.

;

Details under the cut.

The implementation of Awesomium for NeoAxis includes simple GUI controls and support for the in-game 3D interface provided by the NeoAxis features. The extension is available for Windows and Mac as compiled binary files, including the source code of the integration.

Opportunities :
  • Compiled binaries for NeoAxis 1.0
  • Simple to use GUI controls
  • Support for in-game 3D GUI
  • Windows and Mac support
  • Awesomium 1.6.2
  • Integration source code available

The integration of NeoAxis and Awesomium opens up new possibilities in the development of games, the implementation of innovative approaches and ideas in game design. One of the features of NeoAxis - support for the in-game 3D interface - can be effectively utilized by the capabilities of Awesomium. For example, game developers can now interactively display online web content or local HTML files on virtual monitors or terminals.

The pages embedded with the framework retain their full functionality, similar to that provided by conventional browsers like Firefox or Internet Explorer - users can visit any pages, click and follow links, enter URLs manually, and also enter text and submit forms.

The field of practical application of such opportunities is very wide - from web statistics browsers and achievements embedded in the game, high score tables and clan pages, walkthrough guides or references, to registration forms in MMO games fully integrated into the game world. The only obvious condition is that the user has an active Internet connection and the availability of the necessary pages and services.

To summarize, the integration of NeoAxis and Awesomium opens up new possibilities in game development. Pleasant terms of licensing the framework makes it affordable for almost everyone - it has never been so easy to add something new and original to your game project.

The use of the framework is free for non-commercial projects and independent studios, whose total income for the previous year was less than $ 100,000.
  • For more details on the Awesomium framework and licensing terms, please visit http://awesomium.com .
  • Learn more about the NeoAxis Engine game engine on its official website: http://neoaxis.com/ru .
  • Expansion and demo level are available on the Awesomium page on the NeoAxis website: http://www.neoaxis.com/ru/addons/extensions/awesomium
