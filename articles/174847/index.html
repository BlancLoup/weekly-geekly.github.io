Software

Android 4.1.2 was installed out of the box, which was immediately updated to Android 4.2.1 in the following way:
1. Download archive
2. Unpack and copy 6 files to the SD card
3. Clip the Reset button, turn on the device. The blue diode will flash 3 times, the clip can be released. The firmware will be updated automatically.
The firmware from Android 4.2.1 is in beta testing, with respect to 4.1.2 the following features have been added / corrected:
1. Fixed error in decoding DTS sound
2. Hardware video decoding in XBMC
The manufacturer claims support for USB Bluetooth dongles and USB webcams. Unfortunately, I did not have a webcam, but a simple Bluetooth dongle did not want to work.

When we first turn on, we are greeted by the wizard settings, which will offer to select the system language, adjust the image frame, configure Wifi and LAN connections.

Video

The list of output modes is present in both 720p and 1080p. But in fact, only 720p mode works.


AV output also works very peculiar. Taking the media player out of the box, I connected it to the old TV, but was immediately disappointed - only black and white stripes appeared on the screen. The solution to this problem is the following: connecting the device via HDMI, you need to select 480p mode. After that, connecting the media player to the TV via the AV output will work correctly.
