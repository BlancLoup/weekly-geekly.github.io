Another testing of 5 browsers from extremetech.com

Performance tests and compatibility with the standards of web browsers Firefox 3.04, Google Chrome beta, Internet Explorer 7 (7.0.6001.18000), Opera 9.6 and Safari 3.2 (526.26.13) were conducted. Test results:

* JavaScript engine performance (using the V8 Benchmark Suite test suite created by Google Chrome JavaScript engine developers): as you might expect, Google Chrome passed its own set of tests the best. Opera, Firefox and Safary lagged behind significantly (10 times), but showed close results. Incidentally, when measured with the SunSpider test suite, Chrome was slightly ahead of Firefox, Safari and Opera, but it turned out to be slower than the TraceMonkey engine, on the basis of which the release of Firefox 3.1 will be built.
* The performance of using the SVG and Canvas tags for drawing vector images: Firefox won with a very small margin from Safari and Chrome, Opera was 30% behind, IE went the distance because it could not perform the test;
* Acid3 pass test for compatibility with web standards: Opera completed 85 tasks, Chrome - 79, Safari - 75, Firefox - 71, IE - could not complete the test;
* Performance of the Adobe Flash plug-in : the miracle did not happen, the results are about the same, 4% faster than Opera, IE and Safari turned out to be Chrome, Firefox fell behind by 4%;

Bottom line: Google Chrome won in 3 tests, Mozilla Firefox - in 2, Opera - in 1. It should be noted that in some tests, the winner's result differed quite a bit from the results of the catch-up.
