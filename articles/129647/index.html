JavaOne 2011

JavaOne 2011 started today in San Francisco. The online broadcast is available at http://www.oracle.com/javaone/live/index.html , there are English subtitles.



In total, more than 400 performances are scheduled, divided into 7 tracks:


- The Java Frontier - questions not directly related to Java
- Java ME, Mobile, Embedded, and Devices - phones and embedded devices
- Enterprise Service Architectures and the Cloud - Java application in large enterprise level systems
- Emerging Languages, Tools, and Techniques - use different languages ‚Äã‚Äã(for example Scala, Ruby) and technologies (for example HTML5) in the Java platform
- Java SE, Client Side Technologies, and Rich User Experiences - JavaFX and Swing
- Core Java Platform - Java core issues (language development, virtual machine optimization, etc.)
- Java EE Web Profile and Platform Technologies - Java to Web
