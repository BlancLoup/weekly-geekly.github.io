Windows Vista: six vulnerabilities per month

Six serious vulnerabilities were discovered in the Windows Vista operating system by security experts and simple hackers during the month of its existence in the market, writes the Monkey Bites blog.

In particular, an error has been described in Internet Explorer 7, allowing attackers to covertly install malicious programs on the user's computer when they visit infected sites. In addition, it was reported about the possibility of disabling Microsoft Exchange Server by sending an infected email message.

Finally, quite recently, a Russian programmer published information about visibility found by him in several Microsoft operating systems, including Windows Vista. The rim, which was missed by the developers, allows to expand the user's rights of the OS.

All these gloomy events are completed with a message from Trend Micro, whose employees found on the Japanese electronic bulletin board an offer to sell information about a new vulnerability in Windows Vista for a measly $ 50,000.
