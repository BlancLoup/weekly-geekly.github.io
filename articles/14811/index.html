ASUS has released a motherboard with embedded Linux

ASUSTek has launched the P5E3 Deluxe motherboard, which is distinguished by the presence of Linux firmware that complements the standard BIOS (recalls the IBM PC with the BASIC interpreter in ROM).

The motherboard uses a specialized Linux distribution kit Splashtop Desktop , developed by DeviceVM.

Splashtop supports network access via ethernet or wifi, it is possible to update Linux firmware. Of the applications, you can note the presence of a web browser SplashTop Browser (an abbreviated version of Firefox 2.0) and a Skype VoIP client. But the main thing is the download speed, SplashTop Desktop loads in just 5 seconds.

Not without negative aspects: the price of the device is $ 360; Firefox and standard applications cannot write local files; no possibility to mount hard disk partitions; the missing terminal emulator is missing.

Officially, the site of the project Splashtop Desktop will be presented to the public on October 10, on the same day the details of the distribution of the project will be known.
