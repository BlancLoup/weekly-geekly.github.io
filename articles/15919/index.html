GoGo.ru accepts reports about updates from photo hosting sites.

Mail.Ru today announced the launch of GoGo.Ru XML-syndication with domestic photo hosting sites. Simply put, photoresources will now generate an XML file with information about new uploaded images every day and send it to Gogo.ru for indexing.

The developers of the search engine are planning with the help of the innovation ‚Äúguaranteed to index all the information on each added image, which increases the completeness of the search. In addition, this approach makes it possible to increase the frequency of indexing partner databases by about 30 times. " That is, if earlier the search robot indexed each photohosting about once a month, now the data in the Gogo database is updated every day. Based on the data received from photo hosting sites via XML, Gogo.ru plans to launch new photo search functionals. The blessing to transmit for each picture will be a maximum of data - from the size to the link to the personal page of the author.

Now nine photo-hosting sites of Runet participate in the project, including Photo@Mail.Ru, LiveInternet.ru, 500px.com, Flamber.Ru, 35Photo.ru, PhotoForum.ru, Keep4U.Ru, AutoWP.ru, and Fota.Mota. ru.

This is not the first innovation introduced in the image search engine. Gogo.ru also knows how to automatically recognize photos and separate them from other pictures.
