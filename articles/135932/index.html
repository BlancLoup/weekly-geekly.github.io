ASUS Tablet on NVIDIA Tegra 3 for $ 249

Today, NVIDIA and ASUS held a joint presentation at CES 2012 in Las Vegas, which presented the budget tablet ASUS M370T, built on the basis of SoC NVIDIA Tegra 3 (Kal-El Project). The main feature of the tablet, in addition to the hardware platform and Android 4.0, is the low price set at $ 249, which looks very competitive against the Amazon Kindle Fire. Characteristics of the device are as follows:
CPU : ARM Cortex-A9 4-core processor + companion core
RAM : 1 GB
GPU : NVIDIA GeForce ULP
Storage : 16/32 GB
Screen : 7 "IPS-display with a resolution of 800x1280
Camera : Eight-megapixel camera with the ability to shoot 1080p video.
Wireless interfaces : Bluetooth 2.1 + EDR, GPS, Wi-Fi
Sensors : Gyroscope, proximity sensor
Battery : 4400 mAh


Whether HDMI is available, a slot for memory cards and micro USB / USB is still unknown. Deliveries of the tablet will begin in the second quarter of 2012, tentatively in April-May.
