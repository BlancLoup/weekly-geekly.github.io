Personal search engine using Live Search Macros

One of the new services that work within the Live search system from Microsoft is Live Search Macros , which allows you to customize your own search engine.

Live users are given the opportunity to create and configure a macro search and subsequently use it to search for information only on those sites that interest them. The created macro can be left only for own use or included in the catalog .

Being interested, for example, breeding mice at home, the user can enter a list of sites that he considers authoritative sources in this matter. Subsequently, by going to the page of your macro, you can find the addresses of stores that sell mouse food, using information only from these sites.

To create a macro, it is proposed to use simple or advanced mode. In simple mode, the user will be enough to enter a list of necessary sites and determine its name. In advanced mode, using the commands, you can set a lot of additional parameters: for example, the created macro will always look for
- only all the words of the search query (in the usual search engine you need to enclose the phrase in quotes - ‚Äúsale of feed for mice‚Äù),
- pages only in a certain language (language: RU raising mice),
- only in selected RSS feeds (feed: http: //webplanet.ru/rss, a sensation in the mouse market), etc.

Any of the proposed in the macro catalog, users can add to a personalized Live page. To use the service, you must have an account in Live (MSN): anonymous users can only test macros, but not save them.
