k3s - small but certified Kubernetes by Rancher Labs



Requirements:

  • Linux 3.10+
  • 512 MB of ram per server
  • 75 MB of ram per node
  • 200 MB of disk space
  • x86_64, ARMv7, ARM64

Rancher Labs has released a new product for Edge, IoT, CI with ARM support: k3s

There is no support for Legacy / Alpha, there is a replacement for etcd3 by sqlite by default (you can enable etcd3).

Instead of Docker, containerd is used: no libnetwork, swarm, drivers for Docker storage.
To reduce memory consumption, the Kubelet, kubeproxy and flannel agent are combined into one process.

The installer is simple, certificates are generated automatically.
