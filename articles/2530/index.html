Secunia Software Inspector will find "holes" in your computer

You can check which patches are missing on the local computer using the online tool from Secunia - Software Inspector .

With java, this utility checks whether the latest security updates are installed in Windows and other Microsoft products, as well as whether versions of programs such as Firefox, Opera, Skype, Adobe Reader, QuickTime, iTunes, Macromedia Flash are installed on the local computer Player, Sun's Java JRE, Winamp and others .

Secunia Software Inspector can perform both standard and advanced thorough system inspection; all folders will be checked, not just standard Program Files. To use the utility, you must have a java-machine and one of the browsers - Internet Explorer (6.x or newer), Opera 9.x or Firefox (1.5.x or newer). Check for the presence of "holes" can only the latest versions of Windows - 2000, XP and 2003.
