I would like to introduce the next service to the community, made for myself, but suddenly someone else will come in handy.

O'breed - let it be so called.

The bottom line : every day we review a certain set of blogs for updates and interesting articles. But I would really like to go only one page and immediately see what interesting articles appeared in your favorite blogs. Also, I want to see what records have already been read.

To solve these problems, we made the O'blocking .

The main page shows all the blogs that we consider interesting and periodically review them. Each blog shows the last 5 entries; when you hover the cursor over the line with the title - shows the description of the record, limited in length.

The headings are links, so we assigned the control of the read articles to the browser (a: visited will help us).

All blogs have tags and there is a tag cloud that should make it easier to see blogs of interest that are filtered by topic. In the header of the site there is a search string in which you can enter several filter tags. When searching, a bookmark is automatically created for this set for later use.

If a blog is not interesting, you can hide it so that it does not show (if you don‚Äôt want to)

The robot periodically walks through the RSS feeds of blogs and checks for new blog entries.
