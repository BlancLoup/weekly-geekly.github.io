Ubuntu 11.10 build from NNLUG

By tradition, for the next Linux InstallFest Nizhny Novgorod User Group Linux is preparing to build a new version of Ubuntu "convenient" for installation.

It is no secret that every user immediately after installation downloads system Russification packages, sets of audio, video codecs, flash-player, fonts for compatibility with documents from other platforms and so on and so forth.

To facilitate these tasks and save the Internet channel both on Linux InstallFest and when installing the system at home, we did an assembly that included all of these and some other things.

Download the Ubuntu 11.10 build from NNLUG here:
Yandex: http://narod.ru/disk/31588522001/Ubuntu-11.10-desktop-i386-NNLUG.iso.html
Torrent Ubuntu.ru: http://s1.ubuntu.ru:6969/

List of changes made in Ubuntu 11.10 NNLUG Remix:

  • Added localization packages for both Gnome and KDE applications. Immediately after downloading, all applications will "talk to you" in Russian
  • Added Gnome 3 Classic packages to enable switching to a classic two-pane interface, similar to Gnome 2
  • Added all the necessary audio-video codecs , flash-player, all kinds of fonts
  • A set of OpenClipart images has been added to LibreOffice .
  • Added the most necessary applications: GIMP, Clementine, Cheese (in our very humble opinion)
  • Added Nautilus plug-ins for mass resizing and rotating photos (very useful!)
  • Changed tzdata package settings to Moscow time
  • Fixed problem with Russian file names in zip archives
  • Cached binary driver packages for Nvidia video cards
  • Added document templates . Now you can create various types of documents by right-clicking!

After InstallFest, a couple of flaws were discovered, which were fixed in the new build:
  • Fixed problem with printing documents from LibreOffice (the problem is present in the original ubuntu)
  • Added 4 localization packages that our assembly asked to install after installing the system on the hard drive (mostly English help pages)
