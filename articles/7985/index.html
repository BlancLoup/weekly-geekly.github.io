RELAY - file manager in web 2.0 format

As far as I know, file managers for sites have not been discussed yet. Taking this opportunity, I want to introduce you to a file manager called RELAY, this script was written in PHP under MySQL, and PERL was used to dynamically display the boot process.

Script features:
  • AJAX Drag & Drop
  • Preview of pictures
  • Preview PDF Documents
  • Work with directories (create, delete, move, rename)
  • Virtual directories
  • Add users
  • Search
  • Basket
  • Sending files via email
  • Download files as archive
