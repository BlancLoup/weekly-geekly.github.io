Alternative DropCopy

The author of the topic needed a convenient way to share files over a network on computers under Windows. He solved this task by installing an FTP server on each of the machines and a script on AutoHotKey that allows you to send files by throwing a script icon on the desktop.

The method is interesting, but it seems to me that it was possible to make everything easier.
How would I do it, you can find out under the cut.


And I would not have done without an FTP server, only now it is enough to install it on one machine, and not on each one.

We get on the server an account for each exchange participant and a folder for his files.
Access to your folder should allow you to take files, in someone else's - upload.

On the participants' desktop, add the file sharing folder (you may have better ideas for the name) in which we create shortcuts:

Pick up files (to log in to the FTP server under your account)
explorer.exe log:pass@serverIP/2Pasha/

Send Misha (for uploading to an FTP server in the Misha folder)
explorer.exe log:pass@serverIP/2Misha/

Send to Masha (for uploading to FTP server in Masha's folder)
explorer.exe log:pass@serverIP/2Masha/

Send ... (that's who you have to send, for those users and do shortcuts)

Or open your own directory on the server and pick up "mail" :)
