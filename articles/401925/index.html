Presented by Pi Zero W



A new version of the Rapberry Pi Zero single-board computer is introduced - Pi Zero W with wi-fi and bluetooth

The antenna is part of the PCB:



New official case for payment. The top cover is available in several versions - with a cutout under the GPIO cable, a hole for the camera module in the middle, and fully closed.



Some specifications:

  1. Dimensions: 65mm x 30mm
  2. Weight: 9 grams
  3. Processor: BCM2835 with VC4
  4. CPU default frequency: 1GHz with L2 cache overclock
  5. Memory: 512MB RAM
  6. Wireless module: BCM43438 chip for WiFi / BT
  7. WiFi: 802.11 b / g / n
  8. Bluetooth: 4.0 (full UART with flow control)
