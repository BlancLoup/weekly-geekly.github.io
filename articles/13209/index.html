Only 25% of downloaded Firefox users use it.

Journalists found a statistics page on the Mozilla website . It shows what percentage of Firefox downloaded the distribution of the browser actually use it. It turns out that this percentage is quite small: only 50% install the browser and launch it at least once. Among them, only 50% become active users. In other words, only 25% of Firefox who downloaded the browser actually use it.

The number of downloads recently hit all records, including through the help of Google, which began to sponsor this process (partners who will force the user to download Firefox, receive a denyuzhku from Google ). But now it is clear that 75% of downloads are wasted - this is too much. Frankly, this is a threatening statistic. Even Steve Jobs himself recently said that he sees no future for Firefox . Like, in the future there will be only two browsers: IE and Safari.

To improve statistics, Mozilla developed a special 12-point plan to simplify the installation process and further work with the browser. The task is to increase the percentage of growth in the Firefox user base from the current 3.7% per month to 10% per month.
