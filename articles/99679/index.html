HP Virtual Showcase Server

We built, built and, finally, built!

The website hp.ru/vitrina has opened an online storefront for selling models of standard HP ProLiant entry-level server architecture. To order a server through the Storefront, you just have to follow 3 simple steps: choose the server model, its configuration and the HP partner nearest to you.

After that, the order information is sent by email to your HP partner, and he must contact you within one business day and start the server delivery process. The website showcases the ruble prices for the end user, including VAT. HP partners involved in sales through Storefront will deliver servers at a cost that does not exceed this price.



At the initial stage, four lines of HP ProLiant 100-series servers ‚Äî a single-processor tower ML110G6, a two-processor tower ML150 G6 and two rack-mount servers: a single-processor DL120 G6 and a two-processor DL160 G6 ‚Äî are exposed in the Internet showcase.

As you know, HP ProLiant entry-level 100th level servers are designed specifically for small businesses, which still have the role of a file server and print server as a regular desktop PC or a self-assembled server. Thus, for the first time in their lives, many online storefront visitors place an order for a server, and to help them, HP tried to simplify the purchase process as much as possible. The customer immediately sees the configuration, the price of a suitable model and its availability in stock, without the need to look for a profitable offer among numerous HP partners.

For each model, brief descriptions of use cases are placed on the internet storefront , and the proposed configurations include all server components necessary for most models. In addition, any server can be ordered bundled with the operating system Microsoft Windows Server 2008 R2 or Microsoft Windows Server 2008 R2 Foundation, with free technical support for the first 90 days.
