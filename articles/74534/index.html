Remote device access

Good day to all! I want to talk about RDA (remote device access) - a technology from Nokia that will allow remotely
take advantage of some of her devices. This can be useful both for developers and just for those who want to try the next novelty.
All who are interested please under the cat.

How to start working with the service:


First you need to register on the forum nokia .
Then we go to apu.ndhub.net we enter, if our browser meets the requirements, we should see this page:



Then we select the device we need, we reserve it for the time we need (the maximum reservation time is 4 hours, the maximum time to use the phone per day is 8 hours) and press the start button. A launcher will be downloaded to your computer, which will check the computer for compliance with the requirements for working with the service, and then load the RDA program itself and the missing items. When the download is complete, the program will start and you will see something like this window:



Everything can work!

It is also possible that the device with which you want to work may not be available at this time, in this case it can be reserved for a later time. To do this, go to the reservations page:


At the top of the page is a timeline. The blocks marked in red are already reserved time, the gray is free time, the selected blocks will be marked green. One such block in time is equal to 15 minutes. Red highlighted devices that are currently unavailable. You can not mark the blocks closely adjacent to someone else's session. To reserve a session, tick the number of blocks you need.

Now more about the technical part.

To work we need:


  • Browser with support for JavaScript ‚Ñ¢;
  • Java Runtime Environment 5.0 or newer;
  • The Internet:).


Key features:


  • Install and launch applications on the device;
  • File exchange between the device and the user's computer;
  • Reboot the device;
  • Changing screen orientation on devices that support it;
  • Change the color depth, resolution of the image transmitted from the device;
  • Receiving debug information from the device.


Available devices:


Devices with S60 3rd Edition and newer, and Maemo 5 and newer are available. On devices installed European version of the firmware. Firmware
regularly updated.

  • Outgoing calls, messages are blocked;
  • WLAN and Elisa internet points are available;
  • GPS can not be tested.
