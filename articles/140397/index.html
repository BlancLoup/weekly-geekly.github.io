When studying this and other scripts, I had a question: if it can be Java-Script, then why not C #? And it turned out that it could. When installing ANSYS MS Visual Studio itself finds links to all the libraries of ANSYS, but they, at least in version 11, are heavily ‚Äútrimmed‚Äú. Now, by calling the SendCommand method, it will be possible to send ANSYS APDL commands from our external application, performing all the necessary actions. This method emulates the input of commands directly to the ANSYS command line, but many commands need confirmation.
