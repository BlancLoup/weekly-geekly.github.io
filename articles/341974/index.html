Caesar3 all the same open



Some time ago, approximately since 2008, I had a hobby project to recreate the fairly famous late 20th century city building driver Caesar III ¬©, while working on the game a number of articles were written on the mechanics of the game and its internal structure ( for example, this one ).

A little later on the fans site of the game I met another developer who was also keen on recovery, and although our approaches to restoring the game were completely different, this did not prevent the exchange of information. It was very interesting to study the code of the original (through IDA, simultaneously restoring some of the source code) mechanics and, after reworking it, use it in my game.

About four years later, the project turned out to be more or less lively with the levels transferred from the progenitor, but already described in json-configs, with a self-written GUI bolted by the js engine, with minor changes in the graphic design. And it turned out to be a completely rethought dynamics of the city, leaving a very similar behavior to the original. Actually so it was conceived.
