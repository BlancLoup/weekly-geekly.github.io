cPxStack - Nginx / PHP-FPM on the cPanel server, without Apache!

The plugin allows the user to switch Cpanel web server to Nginx and PHP-FPM, completely shutting down Apache.



Not only proxy-pass on Apache, but implements the full stack of Nginx + PHP-FPM without Apache!


cpXstack for server administrator provides the following benefits:



  • Providing an alternative web server and PHP stack than the one that cPanel supports in a standard installation.
  • The cpXstack WHM module allows the administrator to limit the maximum number of PHP processes per user, allowing him to scale the web hosting plans.
  • Nginx can provide more predictable performance under high loads, thereby allowing administrators to reduce server load and server scalability.
  • Nginx consumes less RAM when compared to Apache HTTPD, thereby helping administrators efficiently use the server.
  • PHP-FPM has more features than other FastCGI implementations and is used in many installations.
  • Provides multiple PHP versions for end users.
  • Provides clean installation and removal - the plugin does not make significant changes to your standard Cpanel configuration and once deleted, returns the system to the same state as before with Apache.


cpXstack for the user provides the following benefits:



  • Choosing to use Nginx + PHP-FPM instead of Apache / PHP
  • The user can add additional configurations for Drupal, Joomla, Wordpress, etc. directly editing the config in cpanel.
  • Better user protection against DDOS attacks due to design features.
  • Select PHP version.
