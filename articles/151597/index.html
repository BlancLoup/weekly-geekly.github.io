The new version of Adobe Photoshop will not work in Windows XP



Graphic editors reported that Adobe Photoshop CS6 (13.0) will be the latest version that runs on Windows XP (recall that it does not work with Vista). All subsequent versions will no longer support Windows XP. So, if you still use this OS as your working system, you should think about changing the operating system. In addition, the features available to Creative Cloud subscribers will also not work with Windows XP.

The developers decided to take this step, since the new software and hardware can significantly improve the performance of the graphic editor, as well as add some interesting features. Eliminating support for Windows XP allows you to optimize the image editor by adding features such as 3D, Blur Gallery and Lighting Effect that are not available to users of Windows XP.

The development team has expressed hope (well, that remains) that users of Adobe Photoshop will understand and accept this step and this decision. In general, it's time to think about switching to a new operating system, if you are going to work with new versions of the graphic editor from Adobe.
