Ubuntu Maverick added drivers for Apple Magic Trackpad!


Earlier, it was reported that Canonical developed full support for multi-finger gestures and multi-touch devices.

Now Chase Douglas announced that the Apple Magic Trackpad support has been added to the existing Magic Mouse kernel drivers, and the corresponding changes were accepted in Ubuntu and upstream into the HID tree being developed by Jiri Kosina Linus Torvalds.

This functionality will be included in Ubuntu Maverick 10.10 and, as Chace hopes, in the 2.6.37 kernel.

This driver still has one more problem: the protocol used by the device is unique, and because of this, working with the HID Layer causes many difficulties. Ubuntu uses workarounds to solve these problems, but Michael Poole has a patch that should solve the problem once and for all. Chace plans to properly test this patch, after which he, with a positive outcome, will also be sent to the upstream.
