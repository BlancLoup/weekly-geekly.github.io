Last month, we started writing a review of open source projects at http://fireforge.net/ and thanks for the good reviews. This month there is a new charge of energy and an incentive to continue sharing your code with others. In January, 5 new projects and 10 news items were added.

New projects:
  • sharingan - Client and server for MMORPG games
  • imagemanager - Image viewer with user-friendly interface
  • RTS game studio - A program for creating games - strategies.
  • Railroad Timetable - Suburban electric train timetable aggregator
  • PFirewall - personal firewall for Windows (-like) systems
