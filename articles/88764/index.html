Announced new release plan for Ruby 1.9.2

According to the new plan , the release will take place around mid-August.

More details:
  • March 31 will be frozen list of features Ruby 1.9.2
  • On April 30, the code will be frozen, all features not implemented until this time will not be included in the release
  • May 31 - release 1.9.2-preview2: features that are not completed and stable at this point will not be included in the release
  • June 30 - output 1.9.2-rc: all bugs found in the previous stages should be fixed
  • July 31 - release 1.9.2-p0: for two weeks they will collect bug reports and finish the release before releasing it

Well, let's hope that this time the developers will cope with the deadlines and we will get a stable release of the 1.9.x branch until autumn :-)

PS If someone wants to express wishes for the release of 1.9.2, then hurry to leave a ticket in the Ruby Issue Tracking System
