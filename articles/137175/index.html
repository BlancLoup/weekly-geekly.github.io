Casio CFX-9850G. Color graphing calculator, 1996

After writing a review of the Casio CSF-5750 color e-book, I was carried further and I got a graphical calculator from the same company, with a color display using the same technology. Tram-pa-pa, in front of you is a graphical calculator Casio CFX-9850G .




We also call such calculators engineering or programmable.

Their capabilities include scientific computing, graphing and programming, statistical and matrix operations.

The technical characteristics of this model, taken from the English Wikipedia : three-color (RGB) display 128 √ó 64 pixels, memory size 32 kb, weight - 190 grams, the model itself was introduced in 1996.

Dimensions approximately: 19.7 mm 83 mm 176 mm

There is a possibility of pairing to exchange data both with each other and with a computer (SB-87 cable is required).

The programming language of both series is BASIC. It should be noted that the implementation of this language in these models of calculators allows you to write programs suitable for automating complex calculations. By its capabilities in calculations, CASIO BASIC far exceeds the computer version, in particular, due to embedded matrix operators, including: addition, subtraction, multiplication, determinant calculation, transposition, inversion, exponentiation, assignment of all elements to the same values, creation of unit matrices , permutation of rows, addition of rows, dot product of rows, insertion and deletion of rows and columns, combining matrices, transferring data from columns to lists (one-dimensional arrays), etc. Source
