Collecting VK posts for Samsung Gear

Having bought samsung gear s2 for a long time, I could not get enough of it. Everything is good, everything works. The stream of notifications coming to my phone began to excite and distract me much less. One look at the wrist, I can immediately find out what this time notifies my phone.

However, one nuance still ruined my life. Namely, only the last message from the VK messenger was displayed in the notifications. If messages come one after another, but also from different people, you had to constantly look at the clock so as not to miss anything.

I found the solution to the problem in the MacroDroid application. It allows you to assign actions for the occurrence of any events in the phone. There are a lot of events to which you can assign actions in the application, and there are a lot of actions themselves. For my task I had to do the following steps:

1. In the pop-up curtain of MacroDroid settings, go to the "Variables" menu.


2. Add a variable by selecting the type "String". I called her "led away."


In the course of adding macros and actions, we agree with all the actions requested by the application, for example, giving it access to reading notifications.

3. Add the ‚ÄúCleanup‚Äù macro.

This macro is needed to clear the read notifications and reset the variable.

Trigger Notification cleared. Any notification text from the MacroDroid application.
Actions MacroDroid variable. We select this action and make the value of the "lead" variable empty. Clear notifications from the VKontakte application.
Limitations . None.


Save the resulting macro and move on.

3. Add the macro "VK".
Trigger Received a notification with any text from the VKontakte application.
Actions :

  • Disable the cleaning macro.
  • MacroDroid variable. In the value string, we write [v = led] \ n [not_title]: [notification].

    Values ‚Äã‚Äãin square brackets can not be written manually, but choose in the list to the right of the value line, where [v = lead] is var ‚Äúlead‚Äù, [not_title] is the notification header, and [notification] is the notification text.
  • Notification. We add a notification, we invent any name, I selected ‚ÄúVK‚Äù, the notification value is made [v = led] (again, you can select from the list to the right of the value string). Be sure to check "Overwrite existing notification".
  • Activate the cleaning macro.

Restrictions : screen off. This is to ensure that you do not receive notifications from this application while you are using the phone.


Save the macro and you're done.

If you receive new notifications from VKontakte, they will be written to the variable and displayed all at once in the notification from the MacroDroid application. When clearing MacroDroid notifications on both the clock and the phone, the variable will be reset and the message collection will start from scratch.
