Captain Slopek reports: ReactOS 0.4.4 has been released

Good afternoon, Habr!

With a slight lag behind reality, we announce that the release of ReactOS version 0.4.4 took place last Friday. We successfully maintain a schedule for the release of new versions every three months.

Download | Read the official news | View list of changes | TL; DR | Tests



What is worth noting?


  • Version 0.4.4 was the first in which the possibility of printing documents . So far, only the text and only through the LPT port. But this is only for now.
  • It was also once again (compared to the previous release 0.4.3) the support of network sockets was improved, which provided the possibility of network interaction of an even greater number of programs.
  • The embedded RDP client has added support for SSL connections.
  • Provided the correct stop of services, as well as the correct transfer of information about system variable environments to service processes.
  • Further improvements in the work of shared folders in virtual machines. Completely got rid of hacks in this component.


What we did not have time?


  • We are ashamed, but we didn‚Äôt have time to import the GSoC 2016 groundwork prior to release 0.4.4, but we already did this in nightly builds. So, in version 0.4.5 there will be guaranteed a lot of interesting things.
  • We were also unable to meet the improvements of the USB stack on time , but their import into the main development branch is expected soon.


What are we working on right now?


  • Improved support for themes.
  • Correction of the display of Cyrillic fonts.
  • Modernization of the font engine for a better and more complete display of Asian fonts and symbols.
  • Securing the installation and launch of Microsoft Office 2010 on ReactOS
