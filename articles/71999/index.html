Multiple jQuery plugins that you might be able to use

qTip





qTip is a free jQuery plugin for creating tooltips. This cross-browser plugin has ample opportunities for customization and contains many functions, such as: rounded corners without using images, speech bubbles (speech image as in comics), as well as effects - fade (gradual disappearance of the image), slide and the ability to create your own effects. This plugin already includes five ready-made themes, but you can create your own themes without much effort. In addition, you can create dialog boxes, change the background color of the tooltip and the place where it is displayed.

Download | Demo


Easy tooltip





Easy Tooltip, as the name says, is very easy to use. It does not have such features for setting as qTip, but contains many useful functions - managing the position of the prompt, assigning different id different css properties, which allows you to change the design of the prompt at its own discretion.

Download | Demo

Star rating





Star Rating is a plugin for creating a rating system based on a set of radio boxes. The plugin is very easy to use, since all you need to do is add the star class to the radio box. It includes several interesting features, for example, an incomplete rating (say 4 and a half), a default rating, a disabled rating (for viewing only), as well as an indication of the displayed number of stars.

Download | Demo

Star Rater





Star Rater is very similar to Star Rating, but does not have a cancel button. In this plugin, you can any size and number of stars.

Download | Demo

Uploadify





Uploadify is an upload plugin that allows users to upload files from their computers to the server. Very simple and undemanding to use. To work it needs flash. Includes downloading multiple files, setting up downloads, processing file queues, errors, processes, and onComplete events. One of the best jQuery plugins of this kind.

You can also read about this plugin here.

Download | Demo

jqUploader





jqUploader is another upload plugin that combines html input fields and a flash based widget for downloads, errors, processes and onComplete events. One of the best jQuery plugins of this kind. It has all the functions that Uploadify has, as well as some of the functions that Uploadify does not have. Displays download progress and progress bar view and in percent, allows you to set a file size limit, limit download permissions.

Download | Demo

Thickbox





ThickBox is a lightweight, crossbaruzer jQuery lightbox plugin. It can display images, embedded content, ifame `s, flash, AJAX, pdf. You can customize the size, transitions and captions for pictures and galleries. The modal window always remains centered on the screen, even when scrolling the page. The window closes when you press Esc or the close button. Very cool and easy to use plugin.

Download | Demo

Fancybox





Cool lightbox plugin with a beautiful design that can enhance the look of your site. Allows you to display pictures, html, video, and more. Content loads very quickly and you can add transitions like in Thickbox. You can select options such as scale, image size, padding, or change styles via css.

Download | Demo
