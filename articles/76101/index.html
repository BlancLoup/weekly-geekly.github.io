Project management on one page (table from the book of the same name by KA Campbell)

Just read the book indicated in the title of the topic. She offers a very interesting and convenient scheme for project management. In fact, this is a somewhat complicated Gantt chart .

However, I assumed that the table already created in Excel would be convenient for many users. The book itself has an 80-90% value in this scheme. But, I think, without primary sources, it is quite possible to cope with the study of this diagram, especially since on the official website of the publisher it is presented as:

  • 12 steps to complete (by example)
  • Form for your project
  • Example from the book itself

Advantages of this chart:
  1. All stages of a project are indicated on a timeline.
  2. Each stage, in turn, may have a separate diagram.
  3. Correlation of stages (tasks) with individual goals of the project
  4. Ranged for each stage
  5. There are fields for listing expenditures, current findings and forecasts
  6. All this is placed on one page.
  7. This is not only a visual presentation for the project manager, but also a ready (if regularly filled out) report to the boss


In general, quite a useful tool. Here is his example .
