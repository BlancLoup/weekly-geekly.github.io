Silverlight 3 officially released in Russia

Microsoft Silverlight 3 interactive web application platform has been officially updated to the final version . In the official Russian-language press release, you can find out the details, as well as the impressions of Russian companies from its use - RTR, Rutube, Actio, the Mariinsky Theater and the Library of the President.

Silverlight 3 itself and application updates for developers are already available:
  • Silverlight 3 RTW
  • Silverlight 3 RTW Dev Runtime
  • Silverlight 3 Tools for Visual Studio 2008 SP1 (includes Silverlight SDK)
  • Silverlight 3 SDK
  • Expression Blend 3 + SketchFlow RC
  • Deep zoom composer
  • Silverlight toolkit
  • .NET RIA Services July 2009 Preview
Silverlight 3 includes many new features, including support for off-browser work, 3D transformations, hardware acceleration, shader effects, many ready-made controls, adaptive broadcasting using Smooth Streaming technology, and much more.

In addition to Silverlight 3, there is a release candidate SketchFlow (and Blend 3 as a whole), which allows you to work with application interface sketches, gradually turning them into real design. The final version of Expression 3 will be released within 30 days.

All Russian-language materials on the capabilities of Silverlight 3 and Expression 3 were conveniently gathered by the authors of the Silverlitera developer community. By the way, I recommend to subscribe to the RSS feed of their Bulletin to keep abreast of Russian-language materials on Silverlight.

The international launch of Silverlight 3 and Expression 3 will take place today at 18:00 Moscow time on the site www.seethelight.com .
