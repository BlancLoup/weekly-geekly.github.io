Mod for Minecraft, adding microprocessor



RedPower 2 PR5 is now available .

The new mod allows you to add a microprocessor to Minecraft (6502).

The system consists of three cubes (the processor itself, a monitor and a hard disk) connected together with ribbon cables.

This thing can interact with other blocks in Minecraft.
