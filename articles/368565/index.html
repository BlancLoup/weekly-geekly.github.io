An application from Disney Research transforms 2D images into 3D models.



A division of Disney, called Disney Research, presented a new, interesting project. Namely, software, a tool that can turn an ordinary two-dimensional drawing (for children, including) into a 3D model. The application runs on Android and iOS, and it is built on the basis of Unity, a platform commonly used to create mobile games.

In order to activate the picture, you just need to point at the camera of the phone running the application. Then the program will do everything itself, it remains only to enjoy what they see. At the same time, as you can see, the original "texture" and color scheme remains the same, the author. This application can be used even if you just painted the picture in the children's coloring book.



Somehow, the application is even able to model the reverse side of the object, which for obvious reasons is not visible in the figure, and often the author himself has no idea about it. The announcement posted just such a picture, where the application modeled the back of a humanoid elephant, dressed in pants and shirt. According to the developers, the program copies the pixels from the visible side, and uses this information to generate the remaining parts.

Perhaps the developers will further improve their project, and make it possible to use the generated characters in any interactive game. Unfortunately, so far you can neither buy nor download for a sample. There is only an opportunity to explore the project site here or watch the video, and be surprised at how interesting everything is organized.
