Small (???) Youtube design changes

Good evening, Habr.
Just went to Youtube and was surprised to see that it now looks a bit different:
  • Video description has moved - it used to be on the right, now it is under video
  • The asterisks have disappeared - now a simple system has replaced them
  • Extended view statistics are now available directly from the video viewing page.
  • When you hover over almost any element of the interface (button, comment), a context menu appears


As I understood, these changes should have made the interface more minimalistic, but so far the whole effect for me personally is erased by pop-up explanations to the button
