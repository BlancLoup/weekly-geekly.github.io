LVMOO LM028N - a mobile charger with a Wifi router


Mobile chargers, which include a Wifi router and the ability to connect a 3G modem or flash drives, began to appear on the market. LVMOO LM028N differs from other similar devices in the presence of a slot for microSD memory cards.

Specifications


Protocol support: IRRR 802.11 b / g / n; IEEE 802.3, IEEE 802.3u, IEEE 802.3x
Wifi connection speed: 150Mbps
Security Wifi: filtering by MAC address, 64/128 bit WEP encryption, WPA / WPA2, WPA-PSK / WPA2-PSCK.
10 / 100Mbps WAN / LAN port
2 USB ports, one of which can be used to connect a 3G modem or a flash drive
SAMBA, FTP, DLNA servers
DOS attack protection, built-in firewall
Input voltage: 5V / 1A
Output voltage: 5V / 1.5A
Built-in powerbank capacity: 9600mAh

The functioning of the indicators at the beginning does not seem quite logical.
When charging external devices, the light is always on 100%. When the charge is exhausted, the device turns off.
You can check the charge level by holding the Charge button for a few seconds. A triple blink of 25% indicates a low charge. If the charge is sufficient, then a queue of light bulbs will light up: 25% and 50%, 25%, 50% and 75% or all 4 at once if the charge is full.
When the router is connected to the network, the bulbs from 25% to 100% will light up in turn.

Access files from Android


You can access files stored on a microSD card or USB drive and connected to a router via a SAMBA server.
To do this, in the settings of the router, you need to activate the SAMBA server (Applications section) and add a new user (System-> SAMBA section)
You need to install SAMBA-client on your smartphone. For example AndSMB
In the settings, add the IP address of the router and specify the login password for the connection.


To connect via FTP, you can use the AndFTP utility. You also need to activate the FTP server in the settings of the router and add a user.


To access via DLNA, you need to install a DLNA client. For example - Skifta . By activating the DLNA-Access on the Wifi-router you can get quick access to the media files stored on the connected drives:

Advantages and disadvantages


SAMBA, FTP, DLNA servers
MicroSD card slot
A printed manual describing all the possibilities.
The actual capacity of the powerbank does not match the stated

LVMOO LM028N is a very good solution for both home use and travel. Even despite the discrepancy in the claimed and real capacity of the built-in battery, they can at least 3 times charge a regular smartphone with a 2000 mAh battery.
A built-in slot for memory cards will help organize convenient storage for files accessible via SAMBA, FTP or DLNA.
