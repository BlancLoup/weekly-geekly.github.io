Dropbox - new version and what awaits us in the near future

Two weeks ago, a new stable release Dropbox 0.6.507 was released .

Changes that can be expected in the next 2-3 months


In the client:
- LAN P2P (finally!) (I.e., dropboxes on the same network will exchange blocks of information between themselves, rather than requesting them from the server; this is what happens at their Dropbox office)
- selective synchronization (i.e., the ability to exclude certain paths from synchronization to certain computers)
- improved performance when manipulating a large number of files
- improved work with memory (both in synchronization mode and idle mode)
- Snow Leopard support
- A new plugin for Nautilus, which does not start for each user, new icons and ... Linux CLI (!)

Online:
- improved web downloader
- improved conductor (multiple manipulations)
- improvements in feeds
- new user guide with the ability to download

Mobile / API:
- iPhone app
- non -iPhone mobile site
- alpha Dropbox API
