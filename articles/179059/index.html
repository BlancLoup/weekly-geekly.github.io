BitTorrent announced a new format of torrent files: Bundle, with a request for action before the jump



It's hard to describe the new torrent file format introduced by BitTorrent, Inc. This is generally a new principle of working with torrents, because the BitTorrent Bundle allows the right holder - the distribution author to request an action before downloading the file. For example, you can request payment, or the provision of personal data, or registration, or something else. The new format, according to the developers, will allow the company to "get away from the fire" accusations of aiding the pirates.

A new type of torrent file format is not presented by itself. A good demonstration of the development is the torrent release of the Bundle format from DJ Kaskade. Distributing a DJ while jumping by a fan / music lover will request a rocker e-mail, and in return provide exclusive material. This is a high-resolution trailer from a DJ performance, plus a remix, plus some other materials. In the new distribution, about half of the materials do not require any action from the music lover. The other half, as mentioned above, becomes available for viewing / listening only after the e-mail is provided by the file sharing amateur.
