Start service I'm Sync



We are finally pleased to announce that the registration on the I Sync service is now available for everyone. To use the publishing tools, you must register your site on our service.

Using the components of the publication, you can add to your site information about upcoming events, indicate them on the map, upload photos, videos and contacts.

For those who hear about the service I Sync for the first time, we recommend that you familiarize yourself with what is written here and here .

Publishing components are embedded in any site using JavaScript:



You need to add this code to the page of your site where you would like the corresponding component to be displayed. You can use the service in conjunction with any CMS system, as well as in blogs that support JavaScript.

You can see how the publication components look like on the demo site.

For those who would like to use I'm Sync publishing tools, but do not have their own site, the service allows you to create a site on our server. At the same time, you can select its address, for example, demo.iamsync.ru or specify your own domain.

To avoid habra effect and other possible negative points, the first time activation will be carried out in manual mode. This means that after registration, you will have to wait for the message about the activation of the service for your site. This can take from several minutes to a maximum of several days.

Also, the existing limitation is that so far only one component of the publication can be embedded on one page.
