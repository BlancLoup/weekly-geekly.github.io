Bookmarks Tab extension

If you use bookmarks in Firefox, allow me to bring to your attention the extension Bookmarks Tab . All it does is display a list of your bookmarks in the middle of the browser page when you open a new tab. It looks like this:



In my opinion, this is more convenient than using the menu. (Just in case, add that I am the author of this extension.)

Firefox version: 2.0 and higher
Size: 7Kb

To install

Additionally:
  • UPD: In the process of navigating through folders, use the right-click or the Esc key to return to the previous page.
  • UPD: In the settings you can set a different start page (Tools> Add-ons> Bookmarks Tab> Settings)
  • You can change bookmark icons with the Favicon Picker 2 extension (Firefox 2 only).
  • You can change the look of the Bookmarks Tab page with the Stylish extension. For example, here is the black style .
