VulnHub Parsing HackDay: Albania

We continue analysis a lab with VulnHub . This time we will deal with HackDay: Albania . The image has the following description, which shows that this is a task with HackDay Albania's 2016 CTF

The level is beginner to intermediate.
It uses DHCP.
Note: VMware users may have issues with the network interface. We recommend (for once!) Using Virtualbox.
