The Channel9 portal is available to record the joint webinar Microsoft and Veeam Software " What's new in Windows Server 2016 ", which we held on March 30.




The webinar is a technical overview of some of the most interesting features of Windows Server 2016 Technical Preview 4 . If you have not had time to get acquainted with the new server, the webinar will help you to get information on major technological innovations and improvements in a compressed form within an hour or so.

The following topics are covered:
  • Hyper-V changes, including Shielded VM, PowerShell Direct, Failover Clustering
  • Windows Containers and Hyper-V Containers
  • Windows Server Based Storage
    • Storage space direct
    • QoS for Storage
    • Storage replica
  • Nano server

And the mass of additional information on TP4 , not included in the webinar, on TechNet.
