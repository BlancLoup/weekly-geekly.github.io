Intel launches first SSD with 3D Xpoint technology

Intel began selling its first device based on 3D Xpoint technology, the Intel Optane SSD DC P4800X. The target audience of the device are the owners of data centers, and so far only the very first model of the 375 GB motherboard is available to customers - 1.5-terabyte devices and other form factors will appear on the market in the second half of 2017.

/ photo Remko van Dokkum CC

Fast enough


This type of memory is not built on transistors, but on elements with a change in phase state. The Intel Optane SSD DC P4800X is designed as an expansion card and uses four PCIe 3.0 interface lines. Information is exchanged between the drive and the server using the NVMe protocol. The system is based on 128-gigabit 3D XPoint microcircuits manufactured according to 20-nanometer technology standards.

According to Intel, the DC P4800X can provide up to eight times faster reads compared to the DC P3700 flash drive launched in 2014. The sequential read and write speed is about 2 GB / s, and the performance when processing random 4 KB blocks is 500‚Äì550 thousand IOPS. The device latency is 10 ¬µs.

Also, the Intel Optane P4800X drive has a long service life. According to the manufacturer, the memory is designed for 30 rewriting cycles per day (DPWD / Drive Writes Per Day), while the sum of rewritable bytes (TBW / PBW) is 12.3 PB.

Together with a new type of memory, Intel announced Intel Memory Drive technology, which allows the operating system to recognize the Optane SSD DC 4800X as RAM, and not as a drive. This gives the system the ability to access large amounts of memory than its architectural features allow. Intel states that this feature will allow you to store large data sets used for machine learning and simulation, in low-latency memory for better performance.
