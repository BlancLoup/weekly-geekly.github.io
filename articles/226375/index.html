OpenXcom 1.0 released - free version of X-COM



There are old games that can be called legendary. Decades later, they have a large audience of enthusiastic players. Among such legends - Doom, "Civilization". This also includes the step-by-step strategy of UFO: Enemy Unknown (X-COM).

Since the initial release of X-COM in 1994, more than a dozen ‚Äúclones‚Äù or ‚Äúspiritual successors‚Äù to this game have been created. And now the community has completed work on OpenXcom 1.0 - the first major release of the free version of OpenXcom. In the free version, a higher resolution is supported, new features are added and many bugs of the original X-COM are fixed.

The release of OpenXcom 1.0 is timed to the 20th anniversary of X-COM. The development of a free clone has also been going on for a long time: since version 0.9, 5,224 commits have been made, 606 tickets have been closed. The work took more than a year, and three and a half years have passed since the release of version 0.1! So this is a really big release.

The list of changes made in 1.0 cm. Here . Here are some of them:

  • Rudimentary mods support in configuration files (ruleset).
  • Support for several new languages.
  • Localization of the installer.
  • New names of soldiers from different countries.
  • The new battle mode loads the available cards from individual files, so now they do not need to be embedded in the main game module.
  • Fully reworked Options menu, settings can be changed as the game progresses

This is only a small part of the innovations.

Download OpenXcom 1.0 for Windows, Mac OS X, openSUSE, ArchLinux, Gentoo, OpenPandora, Android.

OpenXcom 1.0 source code : a cross-platform game can be compiled for any platform. It is written in C ++ using the SDL library and published under the GPL license. True, to install OpenXcom 1.0, you must first install the original X-COM.
