All important announcements Tech-ed North America 2011

At this point in the United States ended a major conference of Microsoft Tech-ed North America 2011 during which Microsoft made several important announcements. The plenary report of the first day of the conference, during which many announcements were presented, is available at this address .



Visual Studio and TFS

Some results were announced since the release of Visual Studio 2010, which has become the best-selling studio in the entire history of the product.



Some innovations were presented in the next-generation Visual Studio. As one of them, a new tool Storyboard Assistant was introduced, designed for rapid prototyping of interfaces using a familiar business application - PowerPoint. The new add-on for PowerPoint will allow you to quickly sketch out the interfaces for applications, even to an unprepared person.



Innovations will come in Team Foundation Server. An updated web interface for TFS was demonstrated. The new interface not only has a new design, but also offers drag-and-drop support with real-time changes on the server.



Among the innovations in the TFS web interface, a new Task Boards tool has been announced, which will allow better tracking of tasks.



The new web interface will allow you to use touch panels to manipulate the elements of Task Boards using your hands.

The mechanism of the work of the Team Navigator tool was demonstrated. It will allow working with tasks in the development team and will offer such a function as saving the current state of the working environment (open projects, files, panels, tabs) and subsequent restoration of the environment. This useful innovation comes in handy when you need to stop working on one new task in order to make quick adjustments to another old one.

A closer integration of Visual Studio with unit testing tools was announced. Out of the box will be supported by Nunit, xunit and MS Test with the ability to expand the list.



The site http://www.microsoft.com/visualstudio/en-us/roadmap provides detailed information about the future of Visual Studio's next generation in terms of development lifecycle management tools. A detailed description of the innovations is also available in the blog Jason Zander.

A new System Center Connector CTP tool was introduced that allows integrating into the TFS environment the ability to handle incidents received from System Center Operations Manager administrators. An administrator who handles an incident logged into System Center can easily escalate the problem to the development department, who will receive the incident as a TFS item with numerous useful debugging information.

On the Channel 9 website a video has been published demonstrating new features of Visual Studio for C ++ developers.

The Java SDK for Team Foundation Server has been released , which will allow TFS to be expanded on demand using Java components.

Windows Azure

An updated version of the Windows Azure Toolkit for Windows Phone 7 v1.2 toolkit was released for the conference, which you can read about here .



The Microsoft website published the results of a study on cost savings using private and public cloud technologies (right figure).

A new version of Windows Azure AppFabric SDK V2.0 CTP (May) was announced and available for download, you can read more here . In addition, the conference announced plans to release the next version of the Windows Azure AppFabric SDK in June, which should include new tools for building multi-tier applications:
  • AppFabric Developer Tools - improvements in the Visual Studio toolkit for the visualization, design and construction of cloud applications;
  • AppFabric Application Manager - will provide new opportunities for the automatic deployment, monitoring and management of AppFabric-applications;
  • set of .NET extensions Composition Model.

New features of the Azure SQL Cloud Database have been announced and will be available later:
  • SQL Azure Management REST API - a web API for managing SQL Azure servers;
  • support for creating multiple SQL Azure servers in a single subscription;
  • new version of JDBC Driver for SQL Azure;
  • DAC Framework 1.1 ;
  • integration of import / export functions in the management portal to facilitate the transfer of databases from servers to the cloud;
  • Improvements in the ability to manage databases on the portal, including the Houston tool, as well as a new service for managing SQL Azure databases through the Odata protocol.

Hyper-v



It was announced that HSBC , the largest banking and financial company, will deploy one hundred thousand virtualized VDI-based desktops based on private cloud technologies and Microsoft's Hyper-V based on Windows 7. This will be the largest implementation of virtualization technologies in history. In addition, all SQL Server and SharePoint servers within the company will also be virtualized based on Hyper-V.

It was announced that NetApp and Cisco became Microsoft partners in the Hyper-V Cloud Fast Track program.

During the conference, it was announced that Microsoft officially supports CentOS as an operating system for virtualization on the Hyper-V platform. The official application and more information can be found here . This article provides a detailed description in Russian.

System center



It was announced that the upcoming new generation of System Center 2012 will fully support the management of devices based on iOS (iPhone, iPad), Android and Symbian.

The System Center Orchestrator tool (previously known as Opalis) was demonstrated, which will automate IT tasks using workflows. The beta version of System Center Orchestrator will be released already in June 2011.

A new version of the client system of protection against malware and attacks was released for the conference. Forefront Endpoint Protection 2010 will improve protection capabilities and will be integrated with System Center 2012. The beta version of the product is already available for download and testing.

Other

During the conference , the new Windows Identity Foundation (WIF) v3.5 Extension for SAML 2.0 Protocol toolkit was released , which is designed to build .NET applications for enterprises and government agencies that require the use of the SAML 2.0 authentication protocol. A preliminary version of the new tools is available today .

It was announced that a new version of Microsoft Assessment and Planning (MAP) Toolkit 6.0 will be released on May 31, 2011 . MAP Toolkit is a tool that simplifies the migration of existing applications to a virtualized environment or cloud.

It was announced that the first service packs for Office 2010 and SharePoint 2010 will be released at the end of June. Packages will be aimed at improving the stability, security and speed of applications. Details and a list of some changes can be found on the official website .
