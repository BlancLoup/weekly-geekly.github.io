Thunderbird 3.1 RC2 released



The second preview version of the next release of the Thunderbird email client has been released. The version, known under the code name Lanikai and created on the basis of the platform Gecko 1.9.2. One of the main goals of the release is to identify possible problems that may arise due to the change of the platform. The new version introduced some improvements in IMAP, fixed bugs with smart folders, message filters and attachment management, improved the design, improved memory handling, improved the stability of the program, added new features:

  • Accelerate search results:
    The indexing of messages and the delivery of search results to users is accelerated
  • Quick Filter Panel:
    The new quick filter panel allows you to filter by search queries, tags, star messages, contacts in the address book, new emails and attachments.
  • New Migration Assistant:
    New Migration Assistant gives Thunderbird 2 users a way to choose between using the new features of Thunderbird 3.1 and preserving the usual features and settings.
  • Saved Files Manager:
    The new manager of the saved files displays all the files you downloaded from the mail to your computer.
  • Account Creation Wizard:
    To make it easier to configure Thunderbird, hundreds of mail provider settings have been added. The Account Creation Wizard queries the mail settings database of popular email providers, so you only need to enter your name, email address and password to set up new email accounts.
  • Improved stability, reduced memory consumption, improved work with passwords

The development of Thunderbird is a separate organization of Mozilla Messaging, which was created as a division of the Mozilla Foundation. For more complete information about fixed bugs, refer to the full list of fixed bugs .
