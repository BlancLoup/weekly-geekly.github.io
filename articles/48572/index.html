Forum Tree Engine

There is an idea to release a tree forum engine based on our development. I would like to understand - do you need such a forum? What would you like to add in this engine? Inside the description of features and a link to the demo.


Of the features of the engine :
- treelike
- quick sending of messages, quick creation of topics
- quick registration, quick login
- advanced personal account with the ability to create in your profile your galleries

What else will be done:
New - view all new messages since last visit
Subscribe to topics
