Comments on the July Application Excellence Labs and an invitation to the August


That was a busy week of the July Windows 8 Application Excellence Labs . As the program continues to follow many requests, I want to make a few observations on the results of observations of the Application Excellence Labs that will help you plan to participate in the program, to better understand what you can receive and what is expected of you.

So let's start with what Application Excellence Labs are not :
  • this is not a hackathon, a development school or a bootkamp - we expect the application to be at least half ready;
  • this is not a formal procedure for obtaining a token for registration in the Windows Store: come, sat, got a token - the application must comply with the requirements and guidelines.

What are Application Excellence Labs :
  • This is an opportunity to get a token for registration in Windows Store
  • this is an expert consultation;
  • This is an application report.
  • these are the recommendations of the expert to finalize the application;
  • This is an opportunity to get to the "short" test session at the following Application Excellence Labs, if you have not passed the current.


So, if you failed to get to the July Application Excellence Labs - we are pleased to announce that it begins with the set for August, which will begin on August 20 . Detailed information on the format, conduct and is contained in the post habrahabr.ru/company/microsoft/blog/146543 .

How to get to the August Application Excellence Labs?
Application Excellence Labs is a very, very compact format with a very limited number of participants. Therefore, among those who want to get to the laboratory work, an additional selection is carried out:

  • The authors of the best applications will receive an invitation to the Application Excellence Labs.
