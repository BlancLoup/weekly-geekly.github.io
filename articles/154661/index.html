DriverPack Solution Update Configurator

New feature "Configurator updates DriverPack Solution", allows you to save time and traffic on updates.

Main functions:
The ability to select only those components that are needed for you;
Allows you to receive updates directly from the program;
Maximum download speed almost everything goes through Torrent;
If downloading via Torrent is blocked by the provider, the Configurator will automatically switch to HTTP download;
If any of the files are already in your build of the DriverPack Solution, then it will not be downloaded again.

Thanks to the Configurator, now updates will be released much more often! Now he is in Beta-testing mode, errors may pop up.
