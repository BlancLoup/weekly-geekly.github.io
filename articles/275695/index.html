Skype began to hide the default IP addresses



Microsoft finally solved the problem with IP addresses in its Skype messenger. Four years ago, it was known that a vulnerability in a VoIP messenger allowed an attacker to obtain the IP addresses of users in the service. Despite the fact that Microsoft relatively recently provided an opportunity for users to hide their IP, only now this option has become default .

The hardest thing was for gamers and streamers (and even more so for gamers-streamers). Opponents of the player could always calculate his IP, and carry out a DDoS attack, throwing the gamer out of the system. Initially, the attacker needed a Skype ID, after which this identifier was resolved to IP using the service.
