The open Tarantool database from Mail.ru is certified and placed in the Azure Marketplace

Dear colleagues, I have great news for you!

The open NoSQL-database Tarantool , which is developed by Mail.ru, was certified for Azure cloud platform and published in the Azure Marketplace for ease and speed of deployment.

Tarantool is a high-performance Lua-based application server integrated with NoSQL DBMS. The database is developed as an open project under the BSD license by Mail.ru and the developer community.

The largest case for using Tarantool is Mail.ru itself, which, with the help of this database, provides 30 million users and 500 million emails per day. Other Tarantool users are Badoo, Avito, Qiwi, Wallarm, and so on.

You can learn more about Tarantool on the official website http://tarantool.org/ and the Facebook page at https://www.facebook.com/TarantoolDatabase/ .

Tarantool & Azure Marketplace


We are pleased to announce that the Tarantool solution is now certified and available to Azure users from the Azure Marketplace. You can find a description of the solution on a separate page in the store .
We are very pleased to present the Tarantool database to the developer community on the Microsoft Azure cloud platform. Azure Marketplace will simplify Tarantool deployment in a cloud environment and reduce the entry threshold for developers who need a reliable, open source solution for storing and processing data.

Tarantool is a database and cache in one bottle. It combines the advantages of traditional databases ‚Äî secure storage, transactions, secondary indexes, server query language ‚Äî and at the same time it works with lightning-fast cache speeds, allowing you to achieve query execution time in a fraction of a millisecond and throughput of millions of transactions per second.

The Tarantool solution hosted in the Azure Marketplace is offered free of charge, the Azure user only pays for the computing power on which he will deploy the database. Let me remind you that in-minute pricing works in Azure.
