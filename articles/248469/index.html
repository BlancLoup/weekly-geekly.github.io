MUK Training Center invites everyone to attend the course "ENA 15.3: Switch Operation, Configuration & Management (ENA15.6-200 / 4)", which will be held from January 27 to 30, 2015. The course provides the necessary knowledge for the ENA certification exam.

This course consists of lectures and corresponding laboratory workshops. The course is aimed at obtaining the basic knowledge necessary for the operation of telecommunications networks built on the equipment of the Extreme Networks company.
The course covers switch management via the command line, as well as examples of using tools such as ScreenPlay, CLI Proxy and Ridgeline 3.0.
The course provides the necessary knowledge for the ENA certification exam. The course is based on Extreme XOS 15.3.

The course will be useful for staff responsible for setting up, managing, maintaining and supporting networks built using Extreme Networks equipment.
