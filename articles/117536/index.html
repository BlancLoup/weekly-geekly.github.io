Evernote web interface now allows you to attach files to your notes.

Just a couple of weeks ago, we launched a redesigned Evernote web client with a new interface, new sharing features, and speed improvements. At the time of launch, some of the features available in the old interface, we did not have time to transfer to the new one. And today we are pleased to announce that in the new web interface one of the most important functions is again available - attachment of files to notes.

Two ways to attach files in the Evernote web interface



Now it‚Äôs easier to add files to your notes in the web client. You have two options.

Paperclip icon
Click on the paperclip icon in the notes toolbar. Then, in the dialog box, select the desired file on your computer. You can attach multiple files to one note.



Drag and drop
If you need a way to quickly, just grab the file on the desktop or in a folder with the cursor or drag it to the note title bar. You can drag and drop several files at once or do this in turn - you can attach up to ten files in total.



Why do you need it


Memories have a different shape. If you are going on a trip, your Evernote account may contain memos, itineraries, hotel reservations, photos, and copied web fragments. If you are working on a project with your team, you can store notes on meetings, Powerpoint presentations, audio recordings and snapshots of notes on the board. If everything is stored in one place, life becomes much easier.

Premium feature: file synchronization


Users of free accounts can attach texts, images, audio recordings and PDF files of up to 25 MB to their notes. Premium subscribers can add files of absolutely any type to notes of up to 50 megabytes in size. Also, premium account holders can download up to 1 gigabyte of notes per month, so you can safely add many files to your notes. Read more about the premium subscription here .

Work on an updated web client is in full swing. This is just the beginning.
