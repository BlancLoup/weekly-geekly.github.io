Free DivX Pro for Mac / Win

Today, DivX is conducting a promotional campaign in which it distributes free licenses for the DivX Pro Kit for Mac and Windows. In order to receive a free activation code for the product, the user must enter his email address on this page of the DivX website, after which the company will send the code to this address.

The usual price of a DivX Pro kit is $ 19.99. The DivX Pro software allows you to play and create videos in the popular format of the same name, while the free version is intended only for playing content. DivX Pro for Mac includes codec, DivX Converter, DivX Player and DivX Web Player.

DivX Pro for Mac requires Mac OS X 10.3.9 and QuickTime 7 to work.

The offer is valid for a day ...
