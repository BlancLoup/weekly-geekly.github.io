Extension to highlight code

Comrade Egloo wanted to share his experience with the world, but in view of the minus karma he asked me. So below is his post:

Often you have to view the source in the repositories located on such hosting projects, such as Google Code, Freepository, etc.

The disadvantage is that often files from the same svn on similar services are issued in the form of text / plain. In some cases, it deprives the source code of familiar readability: it is much more convenient to look at any piece of code with syntax highlighting.


Having rummaged on the Internet, I did not find such a solution under chrome, therefore, having devoted the evening to the study of chromoplagging, I wrote a small extension that realizes this possibility.

I would be glad if it would be useful to anyone else.

Components Used:
  • Alex Gorbatchev syntax highlighting engine
  • JQuery (could have been without, but so used ...)
  • Icon from Oliver Scholtz and co.
