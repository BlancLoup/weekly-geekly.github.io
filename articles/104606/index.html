Button Evernote site memory for Habr

There is not always time to read an interesting article on Habr√© and I put them as a bookmark. But it happened that going from the bookmarks to the page received a message that the topic was moved to drafts, or the author was blocked. There is a good option for this - Evernote, but under the linux version there is no nevernote.sourceforge.net ( andy_minsk thanks for the link), and the clipper for firefox blocks all other firefox windows during operation (maybe this is just my problem), besides the whole page with extra blocks is saved, or it is necessary to select what needs to be saved. Therefore, with the advent of the site memory button , there was a desire to fasten it to the habr. The result was a user script . The script adds the evernote button to the information panel at the bottom of the post.

When saving via this button, only the post itself is saved, a tag for the note used by the author of the post is also suggested, and the name of the blog in which the post was published is suggested as a notebook.

It works in Firefox Greasemonkey and Opera, in Safari GreaseKit was not checked, it was not possible to get to work in Google Chrome.
UPD: Updated script now saves comments and 2 options
1) bit.ly/9a9e3f all styles of comments are saved, but it slows down

2) bit.ly/9fJEwo simplified view of comments, does not slow down
