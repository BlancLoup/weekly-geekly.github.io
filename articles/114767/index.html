Browser as a condition of anonymity

It is no secret that IP is not the only information that allows you to identify the user. And even if you use a variety of anonymizers, there is a chance of leakage of your real IP and indirect data, allowing you to learn more about you than you would like. The browser is an important link in the chain called "anonymity."

Let's start, perhaps, with a review of existing solutions, then we will gradually move on to how to do everything yourself. Well, in the end, of course, company news.

Browsar


The first "lucky", which I discovered on the Internet, was Browsar. The browser can be downloaded for free, and the installation file takes up only 210 kilobytes, which even before downloading led to bad thoughts about the presence of a trick. So it happened.

pros

  • The developer declares that the history of visited pages, Cookies, temporary Flash and Java data is erased without the possibility of recovery after finishing work with the browser.

Minuses

  • After downloading it turned out that this is not a browser at all, but an odd job, in fact, being just another shell over Internet Explorer.
  • It turned out that the program is not much more anonymous than the same regular Internet Explorer, and does not add any new features, except for the "anti-boss" button and pop-up blocking, which, by the way, is already present in all normal browsers.
  • It's clear that there can be no open source speech.
  • It works only under Windows.

Operator YAPO


Quite a good build of the Opera browser. It can be seen that with the soul.

pros

  • Based on the relatively new build of Opera - 10.52.
  • Portable, installed on a flash drive or hard drive.
  • By default, it erases all session data after exiting.
  • You can configure it so that after you exit the desired data is saved.

Minuses

  • The data that you still want to save is not stored in a cryptocontainer. The minus is irrelevant in most cases for home use or when using special programs that encrypt data.
  • Opera, and, accordingly, the assembly is distributed without open source codes. Hello paranoid!
  • It works only under Windows.
I would like to note that there are certain problems and security holes in the browser when using a proxy or Tor, for example, de-anonymization when entering a link like nntp: // or through a special web page that uses the fact that addresses like http: // number in some cases, they can be interpreted as a decimal representation of the IP address and the connection occurs bypassing the proxy settings.

When using VPN, fortunately, such problems do not arise.
