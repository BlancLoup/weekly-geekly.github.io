New version 14.2 of DevExpress components for VCL has been released.

With the release of version 14.2, the package of our products for VCL has traditionally been updated and expanded, replenished with both completely new products and numerous modifications of existing ones. So, in this release we are ready to please our users with the following new components.

The new component Gauge Control - the scale manager - is intended for visual display of data using elements that simulate dial gauges, linear scales, as well as digital sectional displays. By embedding visual elements into each other, you can create information panels and combination devices that simulate the dashboards of real devices and installations.




We also released a preview of the version of the Rich Edit Control component, which allows you to create a rich text editor. Despite the fact that while our Rich Edit Control is just a trial version, it is possible to create, edit and save texts in RTF format using a set of formatting commands.



From a programmatic point of view, these commands are implemented as Action objects. By binding them to existing interface elements, such as the Ribbon menu, toolbars and buttons, you can give your application the functionality of a powerful word processor.

In addition to these multifunctional products, in version 14.2, we are pleased to present to our users small, but interesting components of Camera Control and Toggle Switch Control.

The Camera Control component allows you to capture and display both a video stream and individual images from any connected camera - whether it be built-in cameras of a laptop / tablet or web cameras connected to a PC.



Toggle Switch Control is a slider component designed as a replacement for the traditional checkbox in a touch (touch) environment.



Each new release for VCL is not only an extension of our product line, but also the development of components already familiar to users. We will list the most iconic improvements in version 14.2.

Our Grid Control component has got a search panel that allows end users to easily find the information presented in it. The Map Control component now supports a number of Bing Maps services, including geocoding and navigation / routing, as well as requests to Bing Maps at your current location.



The Spreadsheet Control component now supports loading, unloading and editing print settings from / to spreadsheet files, as well as working with data files in formulas.



Layout Control now has the ability to create floating groups, which makes it possible to implement a simple analogue of docking in your application.



We also expanded our library of icons (Icon Library) with new icons in the style of Microsoft Office 2013. Now our library has about 2000 icons for free use in the application interface.
