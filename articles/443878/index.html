DEV Labs 2019. Python for solving non-trivial tasks. Online MAP

Dear Colleagues!

We invite you to participate in the DEV Labs series online meeting, which will be held on March 20 and will be devoted to the Python language. We will talk about the use of Python in the design of smart home systems and consider examples of solving problems of recognizing human actions in video.

Read more about the reports :

"Working with GPIO Raspberry PI in Python". Speaker - Pavel Tsytovich.

Raspberry Pi can be a central hub of smart home systems. In a smart home, we want to interact with various devices. Using GPIO allows you to connect many different devices to the Raspberri PI. In this report, we will look at the review of Python libraries for working with GPIO, as well as the simplest example of how to use GPIO in practice.


"Python in the field of machine learning and computer vision." Speaker - Pavel Zhdanov.

Consider the use of Python in the field of machine learning and computer vision, get acquainted with Keras and OpenCV on the example of the problem of recognizing human actions from video clips.

The event starts on March 20 at 17:00 Moscow time.

To connect to the online session and to quickly access the videos after the event, you need to register . Participation is free.
