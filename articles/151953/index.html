Open webOS ported to Raspberry Pi microcomputer

The developer under the name aaa801 ported an early version of Open webOS to work on Raspberry Pi . This computer is based on the system on a chip (SoC) Broadcom BCM2835 with a frequency of 700 MHz (ARM11 ARM1176JZF-S) and costs only $ 35.



Of course, this is not the final version (the appearance of the graphical interface in the Open webOS version for ARM devices will have to wait some more time) until the system is successfully loaded on the microcomputer.

Raspberry Pi is a motherboard with a processor, RAM, USB ports, an HDMI port and an Ethernet connector. So, if you connect a keyboard, mouse and monitor, you can use it as a desktop PC :). At the same time, Open webOS has a user interface optimized for touch displays, so the question of its use on a computer remains open.
