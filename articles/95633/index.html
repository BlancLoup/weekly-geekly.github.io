WebM license is now GPL compliant

Goolge has released a new license for WebM video format.

Prior to this, according to the previous text of the license, if someone made a complaint about violations of his patents allegedly present in the codec, he was deprived of the right to use both the patented algorithms and the code itself. This condition made the license incompatible with GPLv3 and GPLv2.

Now the license is divided into two parts. The first is a code license and is a copy of a BSD license. And the second is a license for patented algorithms, which gives a gratuitous, indefinite right to use algorithms, provided that the user does not declare that the codec violates its patents. In this case, only the license for the patented algorithms is canceled, the code license remains in force.
