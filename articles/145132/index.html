You can run Android apps on ASUS computers

The company ASUS, which is called as the main vendor for the release of the tablet from Google, has entered into an agreement with the well-known BlueStacks project, according to which, this decision to launch Android applications on Windows will be pre-installed on new ASUS computers.

As a pleasant surprise, customers can use new ZenBook, ASUS Eee and desktop computers for six months to use even paid Android applications, including popular games, for free.

There will also be available a special service for mutual synchronization of the Android environment on the computer with the user's mobile devices, so that the book can be continued to be read on the smartphone from the point where it was closed on the computer or continue the saved game on the tablet.

Previously, ASUS has released a series of teaser videos, in which it hinted that the dominion of Android on mobile devices comes to an end in the light of the release of Windows 8.
