Registration for a free seminar for web developers is open!

We invite you to attend the seminar " Microsoft Technologies for Web Developers ". Also known as WebCamp, the seminar will take place at the Krylatsky Hills Business Center, ul. Krylatskaya, 17, bldg. 1, Moscow Russia.

The event is intended primarily for: novice web developers; developers mastering the Microsoft web platform; developers using the Microsoft web platform, but not familiar with the capabilities of the latest versions of the components of the web technologies stack (IIS, ASP.NET, SQL Server).

Seminar program:

11: 30-12: 00 - Registration
12: 00-13: 00 - WebMatrix - a tool for quickly creating websites, Gaidar Magdanurov, expert, Microsoft. The report will tell and show how using the free and easy-to-use WebMatrix tool you can quickly create, test and publish websites.
13: 00-13: 30 - Break
13: 30-14: 00 - Embed HTML5 today, Konstantin Kichinsky, Microsoft expert. The report will answer questions that have been troubling developers lately: What is HTML5? Why do you need HTML5, when and how to implement it? How to ensure compatibility between different browsers? How to make new items work in old browsers?
14: 00-14: 30 - Break
14: 30-15: 30 - Using SharePoint to develop corporate portals, Yuri Gugnin, Technical Director, ADV Platform Features. Benefits. Scope and examples of application.

Registration for the seminar is required.

When registering, please indicate the name, surname, place of your work or study, position. The absence of these data may cause a denial of registration.

Hurry up! The number of seats at the event is limited.
