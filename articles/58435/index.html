Opera Desktop, the progenitor of modern Opera browsers, celebrates its 15th anniversary



Oslo, Norway - April 28, 2009

Fifteen years ago, two scientists from the company's research laboratory, now known as Telenor, were eagerly preparing for a new project. They were going to create their own web browser. The first lines of code subsequently turned into Opera, a browser that set and continues to set standards for functionality and speed for the entire browser market.

The Opera browser for desktop computers has become the first step to creating a single cross-platform kernel. It is capable of working on many hardware, and today Opera offers Internet access through a wide range of devices. Televisions, set-top boxes, media players, mobile phones, game consoles, cars and, of course, computers - all of these devices make Opera browsers available to hundreds of millions of users around the world.

15 years of browser innovation

User needs are the main and constant reference point of all Opera innovations since the creation of the first browser. Opera Desktop has offered new features to make the Internet more productive and useful.
The first release of Opera contained the basis for using tabs that allow you to load and view several web pages at the same time in a single browser window.

In 2001, the company Opera proposed the mouse control function - Mouse Gestures, which radically improved the navigation on the World Wide Web. About Opera Software ASA

The company Opera Software ASA has opened up new possibilities for browsing the web on PCs, mobile phones and other devices. The cross-platform browser technology developed by Opera is known for its functionality, standards compliance and small size. At the same time, it provides users with faster, safer and more dynamic access to the Internet. The main office of Opera Software is located in Oslo, Norway, with offices around the world. The company is listed on the Oslo Stock Exchange under the exchange code OPERA. More information is available at www.opera.com .
