Overview of IP Phones for Lync Server

Frankly, I can‚Äôt say anything so bad about the phones that have passed through our hands. Despite the very noticeable variation in price, in the basic functionality, all models behave approximately the same, including the sound quality. There are differences and bugs in the implementation of the protocols, but we smooth them, so for comparison, from the user's point of view, only the appearance and usability remain.
