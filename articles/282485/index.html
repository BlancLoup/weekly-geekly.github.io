New Firefox 46.0 is available for free on FTP Mozilla

Less than a day ago, Mozilla posted on its FTP-server 46.0 version of its main browser Firefox. You can install the browser on the main popular platforms: Win x86-x64 , Linux x86-x64 and mac x86-x64 . While on the official site for downloading is available only 45 version of the product.

This patch is called one of the smallest, since it did not bring serious UI innovations. The main work of the Firefox team was done on what is under the hood of the browser.

For example, GTK3 integration for GNU / Linux platforms was implemented, the JavaScript JIT (Just In Time) compiler security was improved, and CDM (Content Decryption Module) support was added as a backup for decoding unencrypted H.264 and AAC streams.

Among other innovations, improved stability and performance of API WebRTC , document.elementsFromPoint support for HTML5, and HKDF support for API Web Crypto .

Full original changelog:
Just In Time (JIT) Compiler
Use the Decryption module and the decode unencrypted H.264 and AAC media where available
GTK3 integration (GNU / Linux only)

Changed

WebRTC fixes to improve performance and stability

Developer

Display dominator trees in Memory tool
Allocation and garbage collection pause profiling in the performance panel
Launch responsive mode from the Media Editor sidebar

HTML5

Added HKDF support for Web Crypto API
Added support for document.elementsFromPoint

You can download the Firefox 46.0 build from the Mozilla FTP server .
