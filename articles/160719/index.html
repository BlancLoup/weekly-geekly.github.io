Winulator: launch of the 90s games on Android OS



Now for Adnroid there are a large number of applications that allow you to run on this mobile OS applications for DOS, games for Sega / Dendy, and more. Soon the number of such projects will be supplemented by another project, the Winulator . This software allows you to run the games of the 90s on smartphones and tablets with Android OS. So far there is no release , but the project, according to the developer (he is one), is almost ready to launch, and there is a video showing us all the advantages of the Winulator.

It is worth noting that the Winulator is designed to run games since Windows 95/98 on the above platform. Here is a video, and below is some more information about the project.



The winulator is not an emulator for Android, in any case, at this stage of development, this is not the case. In fact, this application is desktop, for Windows, and it is intended to convert the game code to get the game to work on Android. Now the launch of the game on Android consists of two stages. The first is the game conversion on the desktop, and the second is the transfer of the resulting files to the device with Adnroid, with the launch already on the "non-native" platform.

Soon, the developer promises to make an Android application that will do all this on the fly, i.e. There should be an emulator. After an indefinite time winulator will appear in the Google Play Store.

Work Winulator has already been tested on the Samsung Galaxy and Asus Transformer Eeepad TFT-101 (all shown in the video above). Tested game Caesar III .
