Ubuntu 10.10 RC is available for download!


The release candidate for Ubuntu 10.10 Maverick Meerkat is available for download.

Changes from 10.04:
- Slightly redesigned design
- New wallpaper;)
- New Font
- The sound applet has been reworked.
- New installer
- From now on, Ubuntu Software Center installs .deb files.
- Ability to purchase applications through the Software Center
- Support multitouch.
- All applications have been updated, some have been replaced by others.
- Fixed bugs.
