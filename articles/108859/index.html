The full program of the conference MageConf & ZFConf Ukraine is available.



Only 1 day left before the start of the joint conference MageConf & ZFConf Ukraine , which will be held in Kiev tomorrow, November 27. Therefore, it is useful to recall that a detailed program of the event has been available for about a week.

The number of streams has been expanded, now it's a 4-in-1 hurricane conference in general:
  • Magento - reports for developers for ecommerce system Magento
  • Zend Framework - reports for developers using the Zend framework
  • PHP - a stream of interesting information for all PHP developers
  • Mix - okolotematicheskie reports on open source, project management, entering the mobile market and similar things

The program is really interesting, talks about the technologies for which the future. Therefore, the speakers and listeners come from different cities and countries.

The conference is designed for software developers and Project Management specialists from Ukraine, Russia and nearby CIS countries. Participation in it is free, which separately pleases. And, although the number of participants has already exceeded 800 people (!), It is not too late to join this event, watch the program and manage to register before the end of the day.

Good Saturday to all of us :)
