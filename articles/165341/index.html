Samsung Galaxy Nexus with Ubuntu Phone OS is shown at CES 2013



The news that Ubuntu has released its own mobile operating system was published on Habr√© recently. Many have wondered (and still are wondering) what the system looks like and whether it is convenient to work with it. In order to dispel rumors at least a little, and show those interested in the real capabilities of the system, Mark Shuttleworth, the founder of Canonical, showed Ubuntu Phone OS at CES 2013 using Samsung Galaxy Nexus as a working device.

At the end of February, the developers promise to put in open access the firmware images for Samsung Galaxy Nexus, so that any user of this device will be able to try out Ubuntu Phone OS at work. So far, the system is not fully ready, but what already exists looks pretty good.

As mentioned earlier, the system can work not only on powerful mobile devices, but also on relatively weak, low-end, devices. This system can be used on smartphones, on tablets, and on TVs.
