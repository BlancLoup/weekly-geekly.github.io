Wordpress client for Symbian in Russian

Do you have a wordpress blog? Now it can be controlled using your mobile phone (Nokia running Symbian).
The program was written by Marcelo Barros, here is the official blog: http://wordmobi.wordpress.com/
Here is the official version of the program on google.code: http://code.google.com/p/wordmobi/
Distribution - under the terms of the GNU GPL v3. The principle of the application - communication with the blog using the XML-RPC interface provided by Wordpress. The program itself is written in Python, its source code can be found on the same google.code .

Instructions for installing the English version:
  1. Install Symbian Python 1.9.6 runtime
  2. Install the English version of the application: Wordmobi 0.9.1
  3. Customize your blog (Settings - Writing - Enable the WordPress, Movable Type, MetaWeblog and Blogger XML-RPC publishing protocols)
  4. Start WordMobi - and go :)

Since the author of the program published on Google code the Russian version for testing (made by me), here is the instruction for the Russian language:
  1. Install Symbian Python 1.9.6 runtime
  2. Install the English version of the application, including translation into Russian : Wordmobi 0.9.1 - en - preview
  3. When you first start the program, switch it to Russian: Options - Settings - Language - Russian
  4. Enjoy :) If in the process of working with the program there are places to be curvature of the translation - let me know, please: I will fix it :)

Known issues:
  1. Statistics not translated: there is no such option in the current version; In the next version, the author promises to add support for localizing statistics.
  2. Buttons Options and Exit in the interface remained in English
  3. The list is subject to addition as irregularities are detected ..
