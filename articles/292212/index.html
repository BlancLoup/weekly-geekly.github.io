The Yandex.Money service turns into a universal payment system

A new version of the Yandex.Money mobile application for iOS has been launched. Now, to pay for services through Yandex.Money, it is not necessary to register an electronic wallet - a bank card will suffice. Thus, the service gradually turns into a payment system.

The new application, like its previous version, allows you to pay for cellular communication, Internet, utility services and other services, according to the website of the service.

If desired, the user can create an electronic wallet. Then he will be able to get discounts or order a Yandex.Money bank card.

Yandex.Money applications with the ability to pay without registration were previously released for Android and Windows Phone .
