Windows Server 2012 Essentials released

A Russian Windows Server 2012 Essentials image appeared on MSDN. This server is a continuation of the SBS line, but you cannot call it a full-fledged successor. Migration is carried out in most cases painlessly, as in previous versions. According to the developers, the product has incorporated parts of the home server, storage server essentials and many other things. Very much emphasis on cloud services.

Brief TTX.


Supports no more than 25 users and 50 devices. However, most functions will work up to 75 users.

Simplified transition to Windows Server 2012 Standard.

For Exchange, there are now three options: local, standalone (On-Premises), and cloud, if clients have Office 365 accounting.

Backups: there are classic options, from new products - File History, stores changes, can roll back.

There is a built-in integration with Office 365 and Online Backup Service.

Add-ons for SBS 2011 and WHS 2011 should work fine. Positioned as a feature.

Everything, as expected, 64x-bit. Naturally, he is friends with windows 8, metro interface, WP7 and other new products.

So, now there is a new SBS based on windows server 2012 with office 365 and metro interface.



We look forward to the release of 64-bit tablets.
