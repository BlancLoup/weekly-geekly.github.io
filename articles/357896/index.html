Arduino controller programming


Once we have decided on the components for the weather station, we will start programming the controller. For the Arduino firmware, the Arduino IDE was used. Also used libraries from Adafruit.

Before going to the sketch, consider the functionality:

  • The readings are taken from the sensors every 10 seconds and only those indicators are updated on the screen that have been changed from the previous measurement.
  • Implemented data transfer via COM port
