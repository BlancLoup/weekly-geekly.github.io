GitHub adds two-factor authentication



I think many have been waiting for this moment. GitHub added another security layer. Finally, two-factor authentication is added.

There are two options for obtaining a code for authentication - SMS message with code or authentication using the application on your smartphone or tablet computer ( Google Authenticator (Android / iPhone / BlackBerry), Duo Mobile (Android / iPhone) or Authenticator (Windows Phone 7)). It is exactly the same options that Dropbox offers us.

The first option, unfortunately, is now apparently unavailable in Russia. There is no Russia in the list of countries with your number.

The second option works great. I tried and liked it, while there is no SMS you can use. I used Duo Mobile, each time I request a key in the application, a new login code is generated.

GitHub also prepared a script in case of losing your device. These are downloadable recovery codes.

Read more in the announcement in the blog and in the help section on the site:
github.com/blog/1614-two-factor-authentication
help.github.com/articles/about-two-factor-authentication

UPD: now there is Russia in the list of available countries.
