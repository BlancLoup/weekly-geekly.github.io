ECS LIVA: a productive mini PC with Windows 8.1



The other day there was official information that the ECS LIVA mini-PC should be on sale soon. The size of the device is really very small - 4.6 ‚Ä≥ x 2.8 ‚Ä≥ x 2.2 ‚Ä≥, but the device is quite productive, given its dimensions.

ECS LIVA can work with various operating systems, including various Windows versions, Ubuntu (or other Linux distributions). Until now, this PC was sold without an operating system for about $ 165. Now, if you pay extra, you can get a system with already installed Windows 8.1.

The developers also provided for the availability of USB 3.0, USB 2.0, HDMI VGA ports, as well as gigabit Ethernet. There is a microUSB to which the network adapter is connected.
