Listen to audio books on Nokia 5800

Audio books are one of the most popular ways to use my phone. Because There is still no third-party software that allows you to conveniently listen to audio books in mp3 format (there are no Scheherazads for the touch interface), you have to use the built-in player. True Nokia interferes with all the music files in one pile, which clearly does not add convenience. Running across book chapters while listening to the entire library is very awkward. In addition, a very important requirement for playing audio books - remembering the position. After all, reading books without bookmarks is not at all handy. Unfortunately, for normal music, only the position of the last played file is remembered. And if you accidentally or specifically switch to another file, you will have to try again to find the desired position in the book.

Fortunately, for some time now, the ‚ÄúPodcasts‚Äù section has appeared in the standard player. Files from this section do not mix with the rest of the music. In addition, for podcasts, the position in the file is stored individually for each file. So, you can use the podcast player for our purpose. To do this, simply upload your book files to the Podcasts / Sounds folder on the memory card and update the player‚Äôs library. The chapters section will appear in the podcast section. You can safely switch from the book to any other compositions. Upon return, the book will be played from the same place.

It is also convenient to glue books into one large file. Then it does not clutter up the podcast library. I do this in Linux with mp3wrap and ffmpeg. The only drawback is that starting from a certain volume, the phone copes with rewinding such a file with a big creak. So, my race to the middle of a 700 megabyte book takes up to 20 seconds. In this case, it is better to beat apart. But in general, this feature greatly adds convenience.
