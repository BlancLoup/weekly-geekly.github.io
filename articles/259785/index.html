New free courses of virtual academy Microsoft Virtual Academy, May 2015



In this review, we‚Äôll talk about the free courses of the Microsoft MVA Virtual Academy , which will be useful both for professional software developers and IT pro, and for beginners. Some courses offer free preparation for official certification exams. Please note that the video player on the site for a number of courses that are created in English contains the ability to include Russian subtitles.

The absolute hit of this month is Dmitry Soshnikov‚Äôs ‚ÄúHow to teach a child to program‚Äù course .

This overview course covers a variety of technologies that can help you to entice your child with computers and programming. Without pretending to methodological completeness and an exhaustive story about technology, this course will give you the right links and ideas, thanks to which you can spend many hours with the child at the computer, studying and creating with it.



New! Using Docker Containers in Azure IaaS Cloud

Igor Chubin, an instructor in UNIX / Linux-based systems, discusses the design of the Docker container management system and its use in IaaS-type cloud systems. You will learn about various virtualization, emulation and containerization systems, such as KVM, Xen, OpenVZ, LXC and others.

The practical part of the module shows how to place an application in a container or a number of interconnected Docker containers, and how these containers are managed: create, delete, start, stop and restart, and also link containers together.

New! Certification. MCSA 8.1 Preparation Course

This course is aimed at students who seek to obtain the status of Windows 8.1 MCSA and prefer self-training and self-study. The course deals with the necessary skills and areas of knowledge that you should pay attention to when preparing for exams 70-687, 70-688 or 70-689 and 70-692 if you plan to update the upcoming status versions from past client operating systems. The first module in the course is an overview, and modules 2 through 6 tell about the necessary skills and give examples and demonstrations of their application. This course is not positioned as the only resource for preparing for exams, but serves as a starting point in the process of self-preparation.

New! The course from Alexander Belotserskovsky "Backup in the cloud Microsoft Azure"

Alexander Belotserskovsky, an expert on strategic technologies from Microsoft, will introduce you to the main backup options available on the Microsoft Azure cloud platform. After all, redundancy is the most important aspect of the functioning of a system of any complexity, and the cloud provides opportunities to simplify its provision.

New! Getting Started with Microsoft Azure Active Directory (in English)

Do you want to learn about Active Directory and the cloud? Here you will find everything you need. Active Directory is the core of the Microsoft identity management proposal. Active Directory is used in 93% of large organizations and has already reached the cloud through Microsoft Azure Active Directory. Anthony Bartolo and Todd Lamothe will show you all the capabilities of Azure Active Directory for managing identities in the cloud.

New! Hosting 1C on Microsoft Azure cloud platform

The course is dedicated to the placement of 1C applications on the Microsoft Azure cloud platform. A course on practical examples shows the full cycle of 1C deployment in the cloud - starting from the selection and configuration of a virtual machine in the cloud and ending with the installation and configuration of the 1C application itself.

New! Corporate Mobile Device Management

Modern mobile devices have various sizes and characteristics, work under control of different operating systems, support various security technologies. Is it possible in this case to somehow manage them? What tools for this exist?

In the course, you will learn how using Microsoft Intune and System Center you can manage various mobile devices, including devices with Windows, Windows Phone, iOS and Android. We will look at the benefits these tools give to mobile employees and IT administrators.

New! Managing systems on Microsoft Azure using Chef (in English)

In this course, specialists Steve Murawski and Damien Caro will show you the capabilities of Chef, review its components, their use and demonstrate how it works with Microsoft Azure. Watch some hands-on demos and learn how to use Chef to configure virtual machines (VMs) and infrastructure in Azure.

New! Getting started with the required PowerShell Status (DSC) service (in English)

View a detailed and helpful demo of DSC by Jeffrey Snover, Distinguished Microsoft Engineer and PowerShell Inventor, and Windows PowerShell MVP Specialist Jason Helmick. Using DSC as a management platform, you can control the deployment, maintenance and configuration of your environment today and get ready for new cloud-based scenarios. In addition, you will learn how to create custom resources for your own applications.

New! The course by George Hajiyev "Designing Microsoft Azure Solutions"

This course is aimed at students who seek to obtain the status of Microsoft Specialist in the design of Microsoft Azure solutions. The course tells about the necessary skills and knowledge for the exam 70-534. The first module in the course is an overview, and modules 2 through 6 tell about the necessary skills and give examples and demonstrations of their application. This course is not positioned as the only resource for preparing for exams, but serves as a starting point in the process of self-preparation.

New! Hyper-V Basics in Windows Server 2012 R2

The course "Basics of Hyper-V in Windows Server 2012 R2" is an introduction to the Hyper-V service, with which you can deploy and manage a virtual environment on the server. We will look at what Hyper-V is like, create a virtual machine, talk about the networking features, and how to save the state of the virtual machine using control points, export and import virtual machines.

New! Monitoring: from server to application. System Center Operations Manager 2012 R2

In the course, you will get acquainted with System Center Operations Manager 2012 R2 and get answers to these and other questions. We will discuss the architecture, capabilities, and scenarios of using Operations Manager to proactively monitor the various components of your IT infrastructure.

New! Detailed review of Microsoft Azure IaaS, introductory course (in English)

Learn how to integrate Windows Azure virtual machines into your infrastructure. If you are an experienced IT professional, but only recently started working with Windows Azure virtual machines, you will appreciate these real-world examples using the Windows Azure infrastructure as a service (IaaS).

New! Work with Linux in the Azure cloud

This video course contains all the information you need to work with Linux virtual machines in the Azure cloud for those who are not familiar with the Microsoft cloud. The course is advanced, goes beyond basic knowledge and contains a large number of demonstrations and practical recommendations.

Hit! Developing modern C # applications

The main objective of the course is to show the power of the modern Microsoft .NET platform and managed languages, in particular, the C # programming language (although we also briefly show how beautiful F # can be). In the overview form, we talk about how C # can be used to create universal applications on the Windows platform, cloud services, as well as devices for the ‚ÄúInternet of things‚Äù and mobile robots.

Hit! Express immersion in JavaScript application development

If you are developing in JavaScript, this course will help you understand what awaits you in the near future. We have tried to give a brief overview of new trends and modern possibilities in developing applications for JavaScript: whether they are complex web solutions, Windows applications or cross-platform applications, or games. We will also look into the future a bit and look at the possibilities of the next version of the JavaScript standard - ECMAScript 6 and touch on new areas in which JS can be applied today.

Hit! Love2Code: Programming for girls and not only

There is an opinion that programming is the lot of stern men with a mathematical education, uncommunicative and unfriendly. This course we are trying to dispel this myth and show that programming is an exciting creative activity that is accessible to everyone! Moreover, the main goal of our course is to love programming, and were able to solve practical problems with it.

The course is built around the creation of mobile applications - using simple applications as an example, we tell about important programming concepts in simple and accessible form: variables, cycles, functions ... And every time we learn a new concept, we add something new to our application.

Hit! Basics of developing websites and web applications

This course, designed for beginners, gives a complete overview of web programming technologies: from the basics of the Internet and HTML / CSS markup languages, to web programming on the client (Javascript) and on the server (C #, ASP.NET). At the same time, we do not go into any of the technologies in sufficient detail, leaving the reader with an opportunity to study in more detail the sections he liked during the individual courses.

The course is based on using WebMatrix, a handy web page editing tool. Some familiarity with the basics of C # programming will be useful for getting acquainted with the course, but this is not mandatory.
