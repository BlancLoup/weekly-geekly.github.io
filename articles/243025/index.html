New on VPS Search: Notepad Domains and Notepad Servers

Today I want to talk about two new features that have appeared on the VPS Search for the last month: a notebook of servers and a notebook of domains . I transferred the notepad of domains from the closed project 2ns.info.

Notepad domains




Domain names are now actually the basis of the Internet - without them, the existence of any site is impossible. As the statistics show, most of the existing domains are not used by their current owners: someone buys domains for future projects, and someone buys them for resale at a price that is tens or hundreds of times more than the initial one. I myself have a certain number of domains that I bought, but have not yet managed to use them for any project.
When there are only a few domains, it is easy to keep track of them, but as the number of domains grows, it can be quite difficult to keep track of when which domain you need to renew and control which domains are available. You can compile a label in excel, but it will be rather problematic to update the information about the end date of registration of each domain.

Especially for such cases, I wrote ‚Äú Domain Notepad ‚Äù - a service that will help you compile a list of all domains and will remind you of when which domain should be renewed.

From a technical point of view, everything looks simple: the user adds the domain, and the system automatically parses the whois of this domain and determines the registration date, the expiration date of the registration and the registrar of this domain. Also comes the definition of DNS servers and TCI. Once every few days data is updated, so if the domain is renewed, the data about it will be updated automatically as soon as possible.

You can also subscribe to a monthly newsletter about expiring domains: the system will send information about domains whose registration deadline ends this month. In addition, there are ideas to make the tracking system for SSL certificates and send information about the fact that it is time to renew the SSL certificate along with the information on the termination of domain registration.



From additional ‚Äúbuns‚Äù I can select a domain calendar - it allows you to visually imagine which domain will expire in the coming year.

Notepad Servers




The idea behind Notepad Servers is the same as the Notepad Domains, only the data object is slightly different. However, there is a slight difference: everything is easier with domains, and you can automatically find out the end date of registration. With servers, unfortunately, this will not work.

Therefore, it was decided to simply make a convenient form for storing data about servers.
To add a server, you need to specify one of IP addresses (IPv4 or IPv6) and server type (virtual or dedicated), all other fields are optional: name, hostname, list of additional IP addresses, server location, hoster, description, link to control panel and price.

Obi features are completely free for users, no fee for use is planned. They are created solely for the convenience of users.
