Onyx Boox i63SL Kepler - reader based on Android

Most recently, the company MakTsentr, which is the exclusive distributor of ONYX International products in Russia, presented a model from a new series of devices for reading electronic books, named after the great astronomer. The main difference of the new model is that it works on the Android operating system.



There are very few e-books on the operating system from Google. But they are slowly gaining popularity. The model from this small series of devices - Onyx Boox i63SL Kepler - came into our hands. In addition to the presence of the OS, the novelty also offers a number of other interesting features, which we will discuss in the article.

The device is based on a 1 GHz processor that allows you to instantly process complex documents, and runs on Android OS 2.3.1, which is open for installing additional programs. Onyx Boox i63SL has 512 MB of RAM, 4 GB of flash-memory and a slot for memory cards of up to 32 GB, which can store up to 35,000 books in the library. The reader is also equipped with a 6-inch E Ink Pearl HD screen with a 12: 1 contrast ratio. Of course, there is the backlight function MOON Light. And there is a sufficiently capacious battery of 1600 mAh, which will allow to use the device for a large amount of time.

‚ñå DISPLAY


Onyx Boox i63SL Reader Kepler uses E-Ink Pearl HD with a 12: 1 contrast ratio and a resolution of 758 x 1024 px. Using this resolution eliminates problems with any font size. The screen does not support touch input. All controls are implemented using buttons and joystick. There is an anti-reflective coating.



Image quality is excellent in contrast and in general impression when reading. The repainting interval can be changed in the settings.



The device is equipped with a function of continuously adjustable screen backlighting MOON Light. You can turn it on by holding the joystick or the Back key, or in the menu, and the brightness is adjusted in the status bar. MOON Light uses a reflective light distribution method. The light guide layer is located above the display. The light goes on this layer, is reflected in the side of the screen (from the viewer), absorbed by the dark elements of the image and reflected from the light, then enters the eye. Feels like light in this case is similar to light from an external source. This light is much more comfortable for perception.



In general, the display is good, but the presence of the backlight makes it even better.

‚ñå SHELL AND APPLICATIONS


On Onyx Boox i63SL Kepler is installed the operating system Android 2.3.1, but, unfortunately, a heavily cropped version without the Wi-Fi module. To control the joystick system, which is adapted to the touch input, not immediately get used to. The device has its own OnyxLauncher launcher, which displays the last open book on the main screen and the rest of the files just below are sorted by opening time. The dock bar contains the Library, Application, Backlight, and Settings icons. And on top of the status bar, the time and the current date are standardly reflected.

The Library section displays your books on the drive. You can customize the sorting through the context menu, which comes off clicking on a special button in the upper corner. Books can be sorted (by name, type, size, creation date, title, author, publisher).
I will add that by pointing the joystick to the file you need, via the context menu, you can perform operations with this file, for example, delete it or rename it. In addition, a search function is provided.



In the settings, you can select the system language, set the date, system font, select programs to open certain formats (a good function), adjust the full redraw interval, auto power off and turn off the screen. Immediately adjust the backlight brightness and application management.



You can also view information about the device, learn about the memory and clear it (if required), configure the side buttons (which I already mentioned in the review).



One of the features of the reader is that the QuickDic dictionary is used here, you can select a word while reading, search the dictionary, get a translation. Naturally, you can use the dictionary and just like that, here you also need to use a virtual keyboard to enter.



The file manager is very similar to the library, only operating with regular files. Nothing special. When you select a file, you can select through which program to open it and use this application by default.
