Twitter + dynamic IP

Today, when discussing problems with dynamic IP addresses and a home server, it was decided that not only dynamic DNS can be used to determine the IP address, but also Twitter. The API is open, you can not only record the status (ie, IP), but also read it. A few hours of work with your hands and you can get a lightweight automatic dynamic DNS by putting the send code on the crown. Or write a daemon / service to monitor changes in external IP. And you can extend the functionality and transfer some useful information or server status.
