Pre-order available on Google Chromebook Pixel. Screen 2560x1700, 1.8GHz Core i5, touchscreen, optional LTE


Today was announced Google Chromebook new generation. This device is interesting both from the technical and software side. Pixel is knocked out of the budget Chromebook line. The most impressive "iron" innovation is a great multitouch screen with a resolution of 2560x1700 (!). From software innovations, this is perhaps an integrated office suite Quick Office, which will be pre-installed. Unfortunately, the editing mode is activated only after 3 months. At the moment the tool is not ready. It is worth noting that Quick Office will run natively, unlike many other chromeOS web applications. Interesting is also the function of instantly uploading photos in the social. Google+ network. Something similar is in android, but on the Chromebook it will be photos in the original resolution. Images will be loaded immediately after the user inserts a memory card. The laptop is already available for pre-order. Cost $ 1299 for the Wi-Fi version and $ 1,449 for LTE. LTE version will have 100 Mb / month of traffic for 2 years for free. Of the additional free-options worth noting 1TB cloud storage Google Drive for 3 years.
The characteristics of this device are:
  • Screen 12.85 inches, aspect ratio 3: 2
  • Gorilla Glass multi-touch screen
  • 2560x1700 resolution, 4.3 million pixels, 239 PPI. Brightness 400 cd / m2
  • Intel i5 Processor (Dual Core, 1.8GHz)
  • 32/64 GB Solid State Drive
  • 1TB cloud storage Google Drive for 3 years
  • Viewing angle of 178 degrees
  • Aluminum housing
  • Weight 1.51 kg
  • 4GB DDR3 RAM
  • Intel HD 4000 Integrated Video Card
  • 2 USB ports
  • Card reader (MMC / SD)
  • Battery life: up to 5 hours
  • Bluetooth 3.0
  • Hd webcam
  • Mini display port
  • OS: ChromeOS
