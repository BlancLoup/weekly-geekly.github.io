Yota is testing Mobile WiMAX Release 2.0 technology

Yota, together with its partner, Samsung Electronics, has begun testing the next version of Mobile WiMAX technology - Release 2.0 (IEEE 802.16m standard).
The 802.16m standard is a 2.0 release of Mobile WiMAX mobile broadband technology. It is considered the most likely candidate among all the communication technologies of the so-called IMT-Advanced generation (advanced IMT-2000) for the status of a new standard for commercial use. The data transfer rate in Mobile WiMAX Release 2.0 networks is more than 4 times higher than in networks using the current Mobile WiMAX technology (802.16e). This is achieved using the Multi-User Multiple Input Multiple User Output (MIMO) solution, which improves the efficiency of using the frequency range. 802.16m is expected to have better performance than the 3GPP LTE standard.

Mobile WiMAX Release 2.0 provides full backward compatibility with Mobile WiMAX Release 1.0 (802.16e) solutions. Due to this, the current Mobile WiMAX operators can easily switch to the 802.16m technology, updating a part of the modules on the motherboards and software of their base stations. The current Mobile WiMAX Release 1.0 subscriber devices are also fully compatible with base stations that will be developed in the future.

As members of the board of directors of the WiMAX Forum standardizing organization and the leaders of the Mobile WiMAX industry (in the development of technology and commercial operation of networks based on it), Yota and Samsung play an important role in creating the Mobile WiMAX Release 2.0 technological specifications and promoting it worldwide.
