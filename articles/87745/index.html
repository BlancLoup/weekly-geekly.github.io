Vote for Yii support in NetBeans 6.9

On the NetBeans website, you can vote to enable support for the wonderful Yii PHP framework in version 6.9.
The fight broke out between the following frameworks:
1) CakePHP (149 votes)
2) CodeIgniter (56 votes)
3) Yii (52 votes)
4) Kohana (10 votes)

Let me remind you that in the previous version 6.8, the same vote was used for the symfony framework
UPD: Yii +2 voices, Kohana +2 voices.
