Review of video / audio flash-players

In this topic, I will try to consecrate the pros and cons of free video / audio flash-players that I could find in Habr√©.


Flowplayer


Supported media formats: FLV, SWF, MP3, MP4, H.264 video, JPG and PNG
Demo: http://flowplayer.org/demos/index.html
Pros:
  1. Playing both video formats and audio.
  2. In the audio there is an opportunity to insert a picture as a screensaver.
  3. A huge number of plugins
  4. Plugin to send links by mail (simple mailto :)
  5. Plugin to generate code to insert the player in your blog.
  6. Publication plugin in popular bookmarks / social networks, including Twitter and Facebook.
  7. Flexible and customizable interface.
  8. Full screen mode.

Minuses:
  1. I did not find the possibility of playing video in high quality, although perhaps this is solved by writing a plugin.
  2. Only English interface.

Uppod


Supported media formats: FLV, F4V, MP4, MOV, M4A, MP4V, 3GP and 3G2, MP3, H.264
Demo: http://uppod.info/
Old player version: flv-mp3.com/ru
Pros:
  1. It has a Russian interface and documentation.
  2. Playing video, audio formats.
  3. It is possible to generate code to insert into the blog.
  4. Flexible customizable interface in the visual editor.
  5. Full screen mode.

Minuses:
  1. Need to register on the site.
  2. The ability to play video in high quality is possible in the paid PRO version.

Cinema HD Player FS


Supported media formats: H.264 MOV, MP4, FLV and other formats supported by Flash
Demo: http://sonettic-cinema.com/products/players/free/cinema-free.htm#samples
Pluses:
  1. Plain.
  2. It is possible to generate code to insert into the blog.
  3. Full screen mode.
  4. Flexible customizable interface.

Minuses:
  1. English documentation.
  2. Few examples.
  3. Advertising during a pause.
  4. There is no possibility to play media in high quality.


From TheShock user:

Jw player


Supported media formats: FLV, H.264 / MPEG-4, MP3 & YouTube Videos
Demo: http://www.longtailvideo.com/players/jw-flv-player/#jwplayer
Pluses:
  1. Playing both video formats and audio.
  2. Plain.
  3. A lot of plugins, including plugins to insert code into your blog and buttons to go to high-quality media viewing
  4. For each plug-in there is detailed documentation.
  5. Full screen mode.

Minuses:
  1. English documentation
