How did you manage to make friends with the TP-LINK router and mobile devices

But the Nokia 5800 smartphone with Symbian 9.4 on board and the AirPad X10 tablet with Android 2.3.1 have never crashed while working over WiFi with WPA or WPA2 security. Until yesterday, when I had to install TP-LINK TL-WR741N router. Making this miracle work in routing mode with auto-connection via L2TP was possible without problems. WiFi connection was tested with WPA / WPA2 PSK security mode and AES encryption, for which a wireless connection was created on a netbook with Lubuntu 11.10, which rose from half a ping and in a few seconds the browser displayed the home page - Google.
