Video tutorials sync online store PHPShop with 1C 8.2

A number of new video tutorials on synchronization of the PHPShop online store with 1C 8.2 have been prepared. The lessons cover such issues as unloading the item, loading orders, editing the order, downloading the interactive price list and others.

PHPShop synchronization with 1C 8.2 has been updated to version 3.3.0, new features have been added to it:
  • Order composition editing
  • Order status editing
  • Automatic removal of goods marked in 1C for deletion
  • Synchronization of only goods with changed data
  • Trade Management for Belarus ed. 3.0
  • Jewelry Trading House, ed. 10.4

Editing Orders




If you work in 1C, and you need to change the quantity of goods in the order, or edit the order status, you do not need to do this in the admin part of the online store - in the PHPShop handler you can edit orders directly from 1C 8.2.
