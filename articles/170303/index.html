Bitcoin 0.8.0 released (Important changes)



The release of Bitcoind and Bitcoin-Qt version 0.8.0 took place yesterday

Bitcoin is a digital, decentralized cryptocurrency that is designed to conduct secure transactions that can not be canceled, it is not subject to banks or countries.

List of changes:
- Transition to a new database - LevelDB (previously used by Berkeley DB)
(The change of database was due to a decrease in the load on the I / O system)
- Executable files signed by the Bitcoin Foundation key (This is required for the latest version of OS X and Windows 8)
- Reduced CPU usage
- Implemented multi-thread processing, so the more cores the CPU has the faster the wallet synchronization
- Added the ability to send only significant transactions to some Bitcoin clients.
(Thus, a client in the Bitcoin network can work without pumping out the full history of the transactions performed, but only those that he needs to work. This is necessary for the mobile clients of the Bitcoin network and the desktop client - Multibit)

After updating the client, the database used will be changed, it can last from 10 minutes to several hours, depending on your CPU.
