Upgrade from Windows 7 / 8.1 to Windows 10 TP via Windows Update

According to the announcement, Windows 10 will be offered free of charge as an update on all Windows 7 and Windows 8.1 systems during the first year after the public release. How exactly? Through Windows Update. You need to understand that this is an upgrade offer, not a license. That is, the license for Windows 7 will allow you to automatically update the system within a year. Serial number, image, box or sticker will not be offered. It also promised that those who upgrade now on Windows 10 Technical Preview will then receive through Windows Update a version of Consumer Preview, Release Preview, and then the final version. With the release of the January Windows 10 Technical Preview (10.0.9926), a small utility for Windows 7 and Windows 8.1 also appeared, after installing which, Windows 10 comes to your PC as an update. That is what I will talk about below.

For those who want to try Windows 10 Technical Preview right now, I will be non-original, this is a preliminary version, some of the functionality does not work, problems with drivers and third-party software may arise. We continue. If you want to upgrade your Windows 7 to Windows 10, download the special Prepare utility from the Microsoft website ; for Windows 8.1, there is a similar preparation utility .
