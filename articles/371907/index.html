The new Firefox 45 removed tab grouping. How to fix



On March 8, an upgrade to Firefox 45 ( release notes ) took place on a stable Firefox channel. At the same time, the beta version was updated to 46, the Developer Edition - to 47, Nightly - to 48, Firefox ESR - to 45.0.

Major changes
  1. The release of the long-term release of Firefox ESR 45.0. It disables the services Firefox Hello and Service Workers. Users have 12 weeks to upgrade from ESR 38.x, after which support for 38.x will be discontinued.
  2. Removed tab grouping function . All groups moved to bookmarks.
  3. Extension signature verification is enabled by default. The function can be disabled (the xpinstall.signatures.required line in about:config ), but the ability to disable it will be removed in Firefox 46.

The most noticeable change is the disappearance of the convenient tab grouping function. Alternatively, one of the third-party extensions is offered.

  1. Onetab
  2. Tree Style Tab
  3. Simplified Tab Groups
  4. Tab Groups

The expansion of Tab Groups is the closest to the original Firefox functionality. Mozilla itself recommends using it.

Of the other innovations - sharing tabs with a list of contacts via chat Firefox Hello, a new button "Cloud tabs" to display tabs from synchronized devices.



When searching or typing the address, the synchronized tabs are now shown in the drop-down tooltip.

In the about:config settings, a new network.dns.blockDotOnion option has appeared for blocking DNS nodes in a hidden .onion network. The default value is true .
