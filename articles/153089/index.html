AMD has made its Android store ... for Windows

This is probably funny, but AMD also opens its own Android-application store ... true for Windows. And you can already download it.

How is this possible? No, AMD did not start mass porting of Android applications to Windows, instead, the company uses BlueStacks technology, an Android emulator under Windows that allows you to run applications. Currently, BlueStacks is optimized to work with only a few dozen applications, so the perfect result is not yet guaranteed.



However, AMD in the framework of the AMD AppZone project offers not just a branded BlueStacks player, it made changes to it to support Radeon cards and APU (Accelerated Processing Units). This should significantly improve the speed and visual component of the applications supported by BlueStacks.

AppZone allows you to search and buy applications right in the program, it also integrates Google Play, Amazon Appstore, 1Mobile Market and GetJar. AMD plans to pre-install it on computers; however, it is not yet clear when this will happen.

Official download link: http://cdn.bluestacks.com/public/bstkappsonpc/amdapps/Downloads/bluestacks_appplayer_AMD_Installer.exe
