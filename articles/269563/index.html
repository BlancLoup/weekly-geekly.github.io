Vivaldi Beta RC

Hello!

Today is not quite the usual build. We decided to call her a candidate for the title of the beta version. We did this because the amount of work done and the quality of the current version of the browser, we hope, are sufficient for the release of a full beta version. If no critical flaws are found - this build is re-qualified for beta, if during testing the developers or users of the Vivaldi browser will come up with some very unpleasant errors - we will make the necessary changes to the code before announcing the full beta version.

For many months now we have been working with you on a new browser. As they say, you and we are already waiting. Much remains to be done, but much has already been achieved by today. It's time to take the next step, and we hope that what we are doing together will be in demand for so many people who are looking for a fast, convenient and functional browser.
