Now GDS 5 communicates with the user in 29 languages, including Russian

Today, Google has released localized versions of its local search engine Google Desktop Search 5 , available immediately in 29 languages, including Russian .

Seven weeks after the release of the English version of GDS5, Google translated the application into 29 languages, including French, Spanish, German, Danish, Japanese, Russian, Chinese, and Hindi (the application was translated into this language for the first time).

Google Desktop 5 is a local search engine for Windows Vista, XP, or 2000, which allows you to search for files on your hard disk using the interface known to us from the Internet version of the Google search service. Like a search engine on the Internet, GDS can display a preview of the contents or documents without running the corresponding applications.
