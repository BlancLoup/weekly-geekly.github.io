How to download Windows 8.1 from the Microsoft website without any subscriptions?

With the release of Windows 8.1 to Microsoft, the view on the infallibility of its OS has apparently changed. Now the corporation understands that users sometimes need to reinstall the OS. And how to reinstall the OS on a PC or laptop with a pre-installed system, if no disc was attached, and the dinosaur DVD is already extinct in modern laptops (and servers)? Microsoft now offers to use the tool for creating installation media for Windows 8.1 . You can download the Media Creation Tool (the direct link is valid at the time of writing. If the utility is updated in the near future, you can download it from the page from the previous link). By running the utility on a client computer with access to the Internet (the utility does not work on my server OS, but on Windows 7/8 / 8.1 / 10 it is perfectly executed), you can select the desired OS edition, language and bit depth, and, depending on the speed of your connection with The Internet and the speed of the hard drive, in 10-15 minutes to get the desired image with a fresh version of Windows 8.1 with integrated updates.



When choosing a Russian version, Windows 8.1, Windows 8.1 Professional and Windows 8.1 Single Language are available (‚Äúone language version‚Äù, which OEM manufacturers like to install on inexpensive laptops). For English, Windows 8.1 N and Windows 8.1 Professional N are also available - without a media player, a version for the European Union. The Russian language pack is perfectly installed on them - if you need the Russian version without the Microsoft media player.
A complete list of all downloaded images (in ESD format) can be seen on the link to the Windows Store site .
You can immediately create a bootable USB flash drive, or download an ISO image from the OS.



The download process took me about 10 minutes on a WiFi connection, checking the image for about five more minutes on my SSD drive.



Unfortunately, the resulting image will not have the correct disk label, and it will be called ESD-ISO regardless of the build. However, installing the OS, you will understand that you have a fairly current version, simply by checking for updates. As I said, at the moment through the Media Creation Tool R4 assemblies are loaded with updates for August 2014. I expect that the utility will soon begin to download R5 images already.

By the way, only in this format is available ARM version of Windows RT 8.1 with updates. An ESD image can be downloaded directly from Microsoft servers , converted to ISO using the ESDtoISO utility, integrated drivers for your Windows RT tablet to be able to make a clean installation. Attention, for Surface RT and Surface 2 it will be more correct to download a ready recovery image from the Microsoft website .

And yet, by installing the Retail version of Windows 8.1 Professional, you can always change its serial number to the client KMS from the TechNet pages and activate the OS with your corporate KMS server, or its equivalent.
