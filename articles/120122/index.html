DepthJS: Kinect Web Navigation Framework

Web navigation is only one of the applications of the DepthJS framework, which is a complete environment for interaction between a web page and a game controller. Kinect can now be used in games and other web applications, and for this you do not need to learn any language other than Javascript.

As you know, the Kinect controller consists of an infrared projector and a photosensor, which allows it to receive a three-dimensional image, recognize gestures and movements of the human body. Now all this can be used in web user interfaces. For example, to authorize a user, recognizing a unique sequence of movements of the arms and legs.



DepthJS is published under the AGPL license ( source code ).
