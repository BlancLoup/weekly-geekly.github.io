Today at 10:00 (MSK), see the online broadcast "Developing applications with common code for the Windows platform"

November 8 at 10:00 (MSK), join the virtual event dedicated to the development of applications with common code for the Windows platform. The modern Windows platform includes a whole range of client devices: tablets and desktops running Windows 8, Windows Phone smartphones, and an Xbox game console. Under the cut you will find a player with live broadcast.


This event is dedicated to the effective development of applications for different client devices, the discussion of relevant development technologies (XAML + C #, DirectX and C ++, XNA / Monogame, HTML5-JS) and various techniques to maximize the use of common code. For example, HTML + JavaScript allows you to develop a very wide range of applications that will work not only in Windows 8 / Windows Phone, but also in the browser and on other platforms. Using XAML + C # in conjunction with the competent design patterns (MVVM) also allows you to select the maximum amount of common code when developing for Windows 8 and Windows Phone. During the event, you will learn about these and some other approaches for developing applications with common code for various Microsoft platforms.

Connect on November 8 at 10:00 (MSK) to the online broadcast .
