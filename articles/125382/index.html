Android Player Review for Blackberry Playbook



Installation


  1. Enable Developer Mode
  2. Download DDPb Installer 1.0.7.0 (installation errors can be skipped), if you have experience developing under Blackberry, then you can install it immediately via blackberry-deploy.
  3. After we run and specify the file sys.android.bar
  4. Profit


Installing android applications


  1. Using DDPb Installer 1.0.7.0 or via adb installl
  2. Via DDPb, select the apk file and upload it, via adb, you must connect to the adb connect IP device (you need to make sure that Android Player is in developer mode), then adb install path_to.apk
  3. Then you can run the installer through Android itself, using File Manager
