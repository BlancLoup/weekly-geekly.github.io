Attach files to comments

For a long time, you can attach pictures to the descriptions of problems and merge requests. Now we have expanded this feature, and it gets support for the files:

  • Microsoft Word ( .docx )
  • Microsoft Powerpoint ( .pptx )
  • Microsoft Excel ( .xlsx )
  • Text ( .txt )
  • PDF documents ( .pdf )


Just drag and drop the files into the comment frame - and they will appear in your comment.
