Convenient online dictionary

Hi habr.

I wrote a very simple web service that I had long dreamed of. I wrote it for myself, but if it is convenient for someone else to use it, I will only be happy.

Now I will explain the essence of this web service. As a student, I often have to translate incomprehensible words from English to Russian or from Russian to English. For this, I always used the online dictionary www.abbyyonline.ru , because it translates well. But unfortunately it was not always convenient, because in order to translate a single word I had to perform too many actions: open a new tab in the browser, write the site name abbyyonline.ru, wait for the site to load, click on the input field for the word which I want to translate, write this word, click inter. And so it was necessary to do sometimes several times a day. As a result, I got tired of it and I came up with the service eng2.ru. Now if we say I want to translate the word "hello" - I write in the browser " eng2.ru/ hi " and immediately the page with the translation opens. Thus, the number of unnecessary actions is significantly reduced. You can also write " eng2.ru/word " and translate the word vice versa from English to Russian.
