Windows 7 RC is available for download with MSDN, MSDNAA and TechNet subscriptions.

Starting today, Windows 7 RC has become available for download with MSDN and TechNet subscriptions, as well as with MSDN Academic Alliance !



For students, graduate students and teachers, this means the following: if an MSDN AA subscription is registered at your faculty and you can download the installation image via the Internet, then you can contact the program administrator to provide you with the Windows 7 RC installation image.

Also, in about a week, Windows 7 RC will be available through the ELMS system (if this system is not used in your university, ask the administrator of the MSDNAA program to configure it).
