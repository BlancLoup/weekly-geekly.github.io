R-keeper


The oldest player in the automation of the restaurant business. The first version of the R-Keeper program was created in 1992, at the moment the system is installed in 37 thousand restaurants in different countries of the world.

The system includes various modules (front office for direct work in a restaurant and back office for controlling processes and costs) and applications. Despite the popularity, some users complain about a rather complex and non-intuitive interface.



As for the support service, the possibility of support via chat (which would be convenient) is not provided, no one answered a few phone calls.

Installation type : system is installed locally

The cost of the license : from 77,000 to 180,000 rubles.

iiko


The project, created by co-founder of ABBYY, David Yan, ranks second in popularity in Russia. The system is characterized by the presence of a large number of different modules that are connected to a special server and work through it. There is the functionality of the front and back office.



A large number of modules that are connected for a fee, is quite difficult to administer, which leads to an increased likelihood of failures.

There is no support with the help of chat, telephone communication in 24/7 mode is quite operational, but connected to operators who are not aware of all the technical details of the system.

Installation Type : Local

Cost : the basic functionality of the program costs 23990 rubles for the server, 19990 for iikoFront (automation of the cash station) and 11990 for iikoOffice (automation of the warehouse management)

The system is also modular - there are applications for managers of the hall, waiters who synchronize data with each other (the waiter accepted the order, the cook saw which dish to cook, the manager receives information about the cooking time, etc.)



According to statistics of the founder of the service, the program is used by 3,500 institutions in the CIS countries. A fairly flexible system, for example, it allows you to change various data even in hindsight - in the ‚Äúreal world‚Äù such a need should arise quite often.
