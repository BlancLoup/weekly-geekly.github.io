Microsoft introduced a new generation of Microsoft Dynamics CRM 2011



The public beta version of the new generation of customer relationship management (CRM) Microsoft Dynamics CRM 2011 has been released today. New features in Microsoft Dynamics CRM 2011 include the following new features:
  • ribbon-interface in all versions of the product;
  • native support for integration with SharePoint Services 3.0;
  • Odata support;
  • LINQ API in SDK.
The preliminary version of the product is available only for the following languages: English US (1033), German (1031), French (1036), Hebrew (1037), Italian (1040), Japanese (1041), Spanish (3082), Chinese (2052)

The beta version can be installed on the operating system Windows XP or later.

Along with the release of the public test version of the new product, Microsoft launches a special promotional website www.crm2011beta.com , where information materials and videos are available with an introduction to the new product features (9 pieces).

Not only the desktop version of Dynamics CRM 2011 , but also the online version of Microsoft Dynamics CRM Online has been released .
