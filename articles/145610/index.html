Emacs 24.1 Released

A new version of the working environment (and a good editor) Emacs 24.1 has been released today.

Among the new features:

  • New package system (Mx list-packages), which makes it easier to download and install extensions. The standard repository resides on the GNU server and is maintained by the Emacs developers.
  • Language support with right-left writing (for example, Arabic or Hebrew).
  • Support for lexical scoping in Emacs Lisp. In the near future, this will add multithreading to Emacs.
  • Improved system of themes (Mx customize-themes) - customize fonts, colors, etc. should be simplified.
  • Improved and universal code completion system for a variety of packages and modes.
  • GnuTLS library support (for embedded TLS / SSL encryption), GTK + 3, ImageMagick, SELinux, and Libxml2.


A full list of improvements is available in the etc/NEWS file.
The new version can already be downloaded on the GNU FTP project or on one of its mirrors .

UPD. Already available version for Windows .
