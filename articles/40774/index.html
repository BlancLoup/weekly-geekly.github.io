Windows Mobile 7 is delayed

Perhaps this news will please someone, but the operating system for mobile devices Windows Mobile 7 will not be released as scheduled. Microsoft has already announced this through its partner channels. Officially, there is no information, but Microsoft partners expected that they would receive a new OS in January-February, but now the dates have shifted to the second half of next year.

The new OS will be very different from the current version 6.1, there will appear gesture recognition (perhaps even through a digital camera), voice input using Tellme technology and much more. Some innovations will appear before the release of WM7, including porting the IE6 engine to mobile phones, making it possible to handle Ajax and Flash.

Windows Mobile 7 is a very important development for Microsoft, because with this system the company is trying to gain a foothold in the market of mobile devices and resist the threat of Apple (Ballmer assured that Windows Mobile will surpass Apple in 2008, but what happens next?). Therefore, the delay in the release for Microsoft can be very painful. Maybe in a year Microsoft will be thinking about how to push the Google Android platform out of the market.
