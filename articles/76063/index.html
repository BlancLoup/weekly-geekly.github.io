Fishbowl - opensource client from Microsoft



Microsoft has released Fishbowl , a rich-functional opensource client for Facebook. Fishbowl is a WPF client application that gives you quick and easy access to all the features of Facebook. The source code is distributed under the free MSPL license , you can download it from the official project page on Codeplex.



The Fishbowl application can work in two modes: standard and minimized. In the second case, a small window appears on the screen with the latest status updates of your friends.



There is a close integration with Windows 7 Aero: display updates, quick access to functions through JumpLists, Aero Preview.



Fishbowl allows you to significantly simplify the process of uploading photos to Facebook by simply dragging photos onto a special window.
