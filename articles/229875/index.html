New VDI service from SFCLOUD set

Our range of cloud services has been supplemented by the rental of virtual desktops or VDI (Virtual Desktop Infrastructure).
When purchasing a service, a customer is provided with a certain number of virtual workplaces with an installed Microsoft Windows OS, with the parameters specified, depending on the selected tariff. Virtual desktops are hosted on the SAFEDATA server platform in their own data center and are virtual machines running on the VMware Horizon View desktops virtualization platform.

When using the VDI service from SAFEDATA, the resources of the workstation and the OS resources belong to only one user, in contrast to the competing solutions available on the market for providing a remote desktop using traditional terminal servers.
