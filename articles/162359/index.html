Research options

Objects of study: vacancies
The cities studied: Moscow, St. Petersburg, Novosibirsk, Yekaterinburg and Nizhny Novgorod.

Research methodology

We collect data on salaries indicated in vacancies / resumes on the main Job portals and then put them in our job directory according to the name of the vacancy / resume. If the title of a vacancy / resume is too general or inaccurate (for example: ‚ÄúTester‚Äù or ‚ÄúTest Engineer‚Äù), then after studying the description of a vacancy / resume we select it for a specific position on the basis of an expert assessment. 90 percentile - statistical maximum

Testing Engineer (manual)


A test engineer is able to write test cases and test plans, usually testing with a black box.

Similar to the Testing Engineer, plus more work experience. Perhaps the ability to read the code - testing the "white box".
