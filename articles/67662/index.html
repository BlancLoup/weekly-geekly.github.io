Aegis Padlock Hard Drive: A New Level of Security

The company Apricorn has released an interesting device - Aegis Padlock portable hard drive with hardware data encryption. To access the disk, the user will have to enter the correct PIN-code using the alphanumeric keypad on the front panel. Actually, the main thing here is the absence of the need to use encryption software.


Technical characteristics of Aegis Padlock: spindle rotation speed of 5400 revolutions per minute, average search time 12 ms, buffer size 8 MB. The drive supports the ability to set 10 user and 1 administrator password. It connects to a PC and is powered by USB 2.0. Compatible with Windows, Mac OS X and Linux.

Apricorn has Aegis Padlock 250, 320 and 500 GB hard drives for $ 120, $ 130 and $ 160.
