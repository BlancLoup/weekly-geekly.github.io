Ubuntu 11.04 Natty Narwhal Alpha 3 released

Exactly a month ago , the second one appeared , and today the third one - and the last one - alpha Ubuntu 11.04 Natty Narwhal has been released.

The main changes are the same as in the second alpha: including the updated Dash, indicator improvements and, as usual, updates to a number of programs.

The release of the first beta version of Ubuntu 11.04 is scheduled for late March, and the second for mid-April. Natty will not have a release candidate because of the Easter holidays.

New Dash


The interface of the transition menu and applications in Dash has been redesigned, search filtering has been improved, the opportunity to switch to full screen mode has been added, and other visual improvements have been made to the overall design



Hot keys for applications


In the application launch menu, it became possible to assign hot keys for these applications. Now you can press Alt and the corresponding number or letter to launch the desired application.



Panel decoration


The top panel now matches the selected GTK theme, and the transparency can be adjusted for it.

Core


The third alpha includes the kernel 2.6.38-5.32, which is based on the latest release candidate 2.6.38-rc6. This is a major update from 2.6.35 in Maverick and a significant update from 2.6.38-rc2 included in the second alpha. The release is expected to be based on 2.6.38.

This kernel update brings little new functionality, but fixes a lot of bugs. Major updates for Intel, ATI Radeon and Nvidia graphics.

Installation


If you want to start using third alpha, then upgrade instructions from 10.10 and links to the system itself are available here: ubuntu.com/testing/natty/alpha3 .
