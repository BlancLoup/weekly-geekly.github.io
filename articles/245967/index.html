Live webcast IT Pro Community Day December 17

On December 17, 2014, the Moscow Office of Microsoft Russia will host a conference for IT specialists and IT managers - IT Pro community day . During the event, leading IT technical specialists will talk about the main server and cloud products of Microsoft. In each of the three tracks, experts have prepared interesting and deep technological content that will be useful to specialists with different skill levels. For those who can not personally attend the event, there will be an online broadcast, a link to which is located under the cat.





IT Pro community day is a unique opportunity to talk with the best technical experts from Microsoft Russia, who will share their unique experience. The conference program is devoted to the following products: Windows Server, System Center, SQL Server, Microsoft Azure, Intune & EMS.

You will find an interesting program that will be presented in three tracks.

Server and infrastructure management


As part of this track will be presentations on topics:
  • Microsoft System Center Operations Manager (SCOM)
  • Building Storage Infrastructure
  • Migration Scenarios from Windows Server 2003

Database


This track contains reports on topics:
  • Microsoft Cloud Platform: Azure and EMS
  • Windows intune
  • Cloud technology
  • SQL Server

Corporate mobility and cloud


By visiting this track, you will learn what Microsoft has to look at corporate mobility management solutions, what is the relationship between corporate mobility and clouds.

The full program is presented on the event website .
