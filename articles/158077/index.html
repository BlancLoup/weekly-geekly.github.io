Google Chrome develops application launcher



Apparently, the browser of the search giant will get even closer integration with the user's system - in one of the future versions of Chrome will appear a special luncher of installed applications Chromium Apps Launcher, which will be available with the browser closed. As the basis for the appearance of the lunch is taken from the same Chrome OS.

At the moment, App Launcher is, apparently, deeply experimental, because it is available only in the Canary Chrome builds, in which the latest browser features are being tested, but even there it can be turned on only with a special setting. To do this, you need to enter chrome: // flags in the address bar and enable Show Chromium Apps Launcher .

After restarting the browser, the corresponding icon appears on the taskbar, which launches the launcher - in this way, it will be possible to avoid a few clicks in order to launch Gmail or Google Docs.
