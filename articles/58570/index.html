1 Return the Quick Launch Panel to its rightful place:
  • Right-click on the TaskBar => Toolbars => New ToolBar
  • In the address line of the "Folder selection dialog" window, enter the following value:

    %userprofile%\AppData\Roaming\Microsoft\Internet Explorer\Quick Launch
  • For the time being, disable Lock Taskbar (Lock Taskbar), right-click on the appeared separator. Make sure that the Show Title checkbox is unchecked, and that the Small Icons value is checked.
  • Use the delimiter and slide out the QuickLaunch as much as you need. After these operations, do not forget to ‚ÄúLock Taskbar‚Äù again (Lock Taskbar)

7 Shortcut Keys
Win + Home - work with one active window, minimize all inactive ones.
Win + T - walk on task bar
Shift + Click or middle-click on the application icon on the taskbar, will open a new application window, and not show the already open
Win + P - opens the projecting dialog, one or two clicks you can transfer the image to a second monitor or projector
Win + Left Arrow and Win + Right Arrow - join the window to the border of the screen
Win + Up Arrow and Win + Down Arrow - maximize-restore / minimize window
Win + Shift + Up Arrow and Win + Shift + Down Arrow - maximizes-restores / minimizes the window vertically, but not full screen.
Win + Shift + Left / Right - (for owners of multi-monitor systems) transfers the window to the left / right monitor
Ctrl + Shift + Click on the application to immediately call it with administrator rights.
Win + Space - Looking at the desktop, time

I will remind old not less useful combinations

Win + D - minimize all windows, open the desktop. Repeated actions return windows to their former place.
Win + L - Close the desktop, especially effective when the user account has a password.
Win + R - Call Start / Run Dialog
Ctrl + Alt + Delete in Win7 returns you to a screen like a welcome screen with a choice of some operations including Task Manager (Task Manager). If you want to immediately open Task Manager, try the following Ctrl + Shift + Esc .

8 Assign a shortcut key for the Snipping tool
Snipping tool is a utility for taking screenshots of a selected area.
Start menu => Accesories => Snipping Tool => Right click => Properties
In Properties, in the Shortcut Key field we try to type any key combination. Note that we are immediately given the first two keys Ctrl + Alt + , the choice of the third key is yours, as you prefer. For example, I have Ctrl + Alt + S. The same operation can be performed with Sticky Notes , also specifying a key combination for its quick recall.
