Microsoft has introduced Windows Defender Security Center



The other day, Microsoft introduced the Windows Defender Security Center (free for Windows 10). According to their statements - now it is the safest version of Windows from the previously presented. Translation is taken from the source .

The security threat is one of the most pressing problems that can be encountered using modern technology. And that is why Microsoft decided to release a new version of Windows 10 - the safest ever created. With comprehensive built-in features, Windows 10 provides reliable protection against viruses and malware, Windows Hello * performs more secure authentication using a smile or a fingerprint for authorization, and the security system is regularly updated via Windows Update. Windows Hello requires special hardware, including a fingerprint scanner, an IR sensor, or other biometric sensors and devices.

But you do not need to be a security expert to protect your device and understand which security features to use.

In Windows 10 Creators Update, Microsoft introduces an innovation called Windows Defender Security Center, which makes it easier to view, monitor the level of protection and better understand the security features that are already used in Windows 10.

Windows Security Center includes five principles that make it possible to monitor the security of a device and its state, including security on the Internet:

Virus & threat protection - is a new look at anti-virus protection, whether it is free of charge bundled with Windows 10 Windows Defender Antivirus or antivirus software from other manufacturers. For example, if you scanned the system using Windows Defender Antivirus, and as a result received a list of threats, you will be able to scan them using alternative antivirus software.

Device performance & health provides information about the latest Windows update, installed drivers, etc. Additionally, it is possible to ‚Äúrefresh‚Äù the system by resetting all settings and starting working with clean Windows. This feature will save your personal files and some Windows settings, but will remove most applications.

Firewall & network protection provides information about network connections and active Windows Firewall settings, including informational links to help troubleshoot problems.

App & browser control allows you to adjust SmartScreen settings in applications and browsers, helping you to be more aware of the security on the Internet, warning you about possible malicious sites, downloads, unknown applications and files from the Internet.

Family options allows you to easily connect to the desired settings online. This feature will give you the ability to access Parental Control, setting the viewing time, including reports on the activities of your children on the Internet, as well as information on purchases of games and applications. You can also see the safety and status of the devices of all your family members.

The main purpose of Windows Defender Security Center is to inform you and make security easier and more understandable. In this case, it is important that your device is constantly protected. And if your subscription or paid antivirus software expires on your license, Windows Defender Antivirus will protect your device by default until you do something. After all, it is important to make sure that it is you who controls your PC.
