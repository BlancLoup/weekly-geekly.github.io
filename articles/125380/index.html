The Android 2.3 update on HTC Desire (for developers) can already be downloaded, but ...



You should really want to upgrade because the update is installed with a number of restrictions. The released update is generally recommended for installation only for advanced users of HTC Desire or for developers who want to experiment with the new firmware. That's because HTC engineers decided to get rid of memory problems by deleting Android apps and Android add-ons that were typical of previous firmware versions during installation. Only the naked ROM upgrade utility (RUU) is offered for download.

So, when installing the update, all messages, email addresses, contacts and other personal information will be erased. In addition, all changes made by the user earlier will be erased. Only the contents of the SD card will not be affected And so, everything below is erased:

- all customizations, including applications installed by the telecom operator who sold the phone to the user;

- all wallpapers, except those that were set by default;

- some HTC applications;

- official Facebook app.

According to representatives of HTC, all these changes, which are made by the installed update, are made in order to remove memory problems, more precisely, with its lack. As you can see, the changes are not particularly terrible, so who really wants to install this update, can try, after creating a backup for their data.

By the way, judging by the comments of "satisfied" users, all of them are hardly too happy about the appearance of this update. Despite the fact that many expected him, he brought no joy.
