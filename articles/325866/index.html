101 free services for the designer

We present you a selection of free services for the designer. This is the first option, in the future we plan to expand, adding new sections and tools.



Fonts


  • Google Fonts is a favorite library of fonts. Allows you to choose and download different fonts for all occasions.
  • 1001 Free Fonts is a font library. It has easy navigation - the fonts are sorted both alphabetically and by type. There is a section of Russian fonts.
  • Fontstruct is an online font designer. A very simple tool will allow you to create your own font. You can draw not only with standard squares, but also with other figures - dots, rounded corners, etc. For a beginner, an ideal tool to try and see typography in action.
  • Font Squirrel is one of the largest font libraries. In addition, you can download a picture and get the names of the fonts used on it. There is a font shop, and even a clothing store with logos.
  • Type light is a full-featured font editor. Allows you to create, edit and convert OpenType, TrueType and PostScript fonts. Need to download and install. The free version has limited functionality compared to the free one. But to create a simple font is enough.
  • What Font is - an online service for selecting fonts for the site. Download the font file, specify the url of the site and see how the downloaded font will look on the site. Convenient service.
  • 1001 Fonts - font library (more than 15,000). Easy navigation - the font can be selected by size, type, theme, design, etc. The menu allows you to customize the output, which makes it possible to see each font in different options: size, style, etc.
  • Font Flame is an online service for selecting a pair of compatible fonts. It works very simply - on the main page a leaflet with two inscriptions made in different fonts is offered. I liked it - click "Love", no - "Hate". Then the selected can be viewed again. Clicking on the font name opens Google Fonts.
  • Typecast - font selection for the prototype. You can see how the font will look on any device. It is necessary to designate different types of text - headings, quotes, etc. You can select a headset, size, style, set indents. You will see your future page in the fonts you have installed. A big plus is the choice of layout (one column or several, different block arrangements).
  • Wordmark - allows you to choose the font installed on the user's computer for text, logo or title. When you enter text in the field shows it with different fonts. You can choose the size, case, display in black and white or vice versa. Favorite options are published in Tritter or on Facebook - at the user's choice. You can send by email.

  • FoodiesFeed - food photo. Images are categorized. You can choose the best, exclusive, icons, mokapi, etc.
  • Free Refe Photos - a huge list of images "Real life". Mainly landscape and nature photography. Inconvenient navigation, all photos on the main page, appear when scrolling.
  • Little Visuals - high-quality photos of landscapes and buildings. You can download the archive the best photos posted for the week. Photos are marked with hashtags, but not categorized.
  • Gratisography is a huge library of free photos. All photos are divided into six topics: people, grimaces, cities, objects, nature and animals. You can search by words.

Drains graphics


  • Brusheezy - free brushes, patterns, textures and more for Photoshop. 12 categories, each of which subcategories. The choice is huge.
  • Iconfinder - icons, both paid and free. You can download in formats ICO, PNG, ICNS, SVG.
  • Brushez - a large collection of brushes for photoshop. Besides them, there are also vector primitives, forms, patterns, styles. Each section is divided into subsections. For example, for brushes more than 20 categories.
  • Vecteezy - Vector graphics library, including images and illustrations. A large number of categories and a huge - graphics.
  • Iconbird is a Russian-language icon search service. You can enter a topic and get a set of icons, and you can browse by category.
  • Icons8 is a Russian-language service offering more than 42,000 icons on various topics.

Infographics


  • Vizualize is an online service for creating infographics "About Me". In essence, it creates a graphic summary.
  • Canva is a simple tool for creating any design. Allows you to arrange a presentation, poster, writing in social networks, books, letters, materials and much more. Everything is done on the basis of templates, the text changes. But there are a lot of templates, there are plenty to choose from.
  • Easelly - creating infographics based on templates. Built-in online editor allows you to change the design, place additional items, change the text.
  • Infogram is an online service for creating infographics and interactive charts. Not all features are available, you will have to buy an account.
  • Visage is a simple tool for creating any kind of visual content (charts, infographics, blog images, etc.). Many templates, simple editor. Some features are paid.
  • Gliffy is an online charting service. The editor operates on the principle of Microsoft Visio.
  • Visme is an online presentation and infographic tool. More than 1000 templates. The editor allows you to add text (headings, lists, quotes and regular), animated text and widgets. In a free account you can download as a jpg file.
  • Piktochart - creating high-quality graphics - presentations, infographics or printed material. PNG, JPG of different sizes is unloaded.
  • Drawio is a convenient online charting service. You can save to the dropbox and similar services, you can to disk.
  • Venngage - simple creation of infographics, posters, reports, materials for social networks, etc. In the free version gives only a link to the publication. You have to pay for downloading.
  • Cacoo - online service for drawing diagrams and site maps. Perhaps the best in features. A large number of tools and templates. You can export to PNG, PDF, SVG, PPT, etc.
  • Creately is an online chart drawing service. Available for publication only. Downloading paid.
  • Lucidchart - an online service for creating diagrams and drawings. It is possible to import a Microsoft Visio chart. Exported to PDF, PNG and JPG. There is a paid functionality.

Colors


  • Color Hunt - selection of color combinations. You can pick up to four colors. There are ready-made solutions for selection.
  • TinEye - analyzes the site and displays a list of resources that use the picture from the site.
  • ColorZilla - allows you to create colors and gradients. Displays css properties for them.
  • Adobe Color CC - creating color schemes.
  • Colicious - press the space bar and generate a new color.
  • Colorscheme - selection of color schemes. There is a version in English and in Russian. Allows you to choose colors to the site. It has fine tuning, shows the color code. Ultimately allows you to view the page in the selected color scheme. The biggest plus is viewing the gamut in terms of color perception deviation. Also allows you to choose different color models. Saves to palette Gimp, CSS, PNG, etc. It has a color converter to translate from one color model to another.
  • Paletton is a tool for creating color combinations. No different from Colorscheme.
  • Pictaculous - analyzes image palette PNG, JPG or GIF. Download the image and get a set of all the colors that are present in it.
  • Hex Color Tool is a color selection tool. The sliders adjust the color, you can choose the brightness. Then the selected color is saved. You can pick a palette.
  • (Un) clrd is a browser plugin that removes all colors and leaves the site in black and white.
  • BrandColors is the largest collection of official brand color codes. Choose a brand name and download the names of its colors, including in the css-file.
  • Colortyper - selection of colors from an unusual angle. The screen shows multicolored symbols that intersect. The result of the intersection is a new color. Looks interesting.

Inspiration


  • Dribbble - The largest community of designers. Work, community, communication - here you can find anything.
  • Freebbble - templates and designs from Dribbble. Something you can download for free, buy something. You can find templates for CMS, designers, icons, mockups, custom templates and much more.
  • Muzli - Daily design inspiration. Resource for designers with examples of the most unusual work.
  • Awwwards - the best designs of designers. Organized as a community where you can vote site. The site of the day, month, year, etc. is selected. A large number of design, video, and developments.
  • Design You Trust is a design community that shares the latest trends, news, portfolio, designs, and creative announcements.
  • Really Good Emails - a huge collection of well-designed emails. Choose a template and get its html code, css and js. You can change the text, pictures, etc.
  • Fubiz - publications from the field of design and creativity. You can find photos, design, graphics, video, etc. Everything is divided into categories.
  • Designspiration is a collection of Pinterest-style designs. A huge number of ideas and examples - typography, logos, letters and much more.
  • Behance - a collection of portfolios from professionals. You can leave a job or resume.
  • Mobile-patterns - a library of iOS and Android screenshots. Everything is divided into sections. You can find a lot of ideas.
  • One Page Love is a gallery of the best web page designs. You can buy templates and themes. There are free.
  • Uispace - a resource for designers. You can download a free psd, sketch, fonts, mockups, etc.

Graphic editor


  • DAZ 3D - 3D modeling. Register and download the application.
  • Gimp is a free analogue of photoshop. It allows you to customize the modular grid, create guides, work with layers, add effects, etc. It is a good free replacement for photoshop. Is extensible - you can download additional plugins.
  • Blender - 3D modeling, animation, rendering, sound, video. Allows you to create games. A fast-paced editor, very popular.
  • Aviary is an online photo editing service. Many features, including the "red eye effect", resizing, etc.
  • Pixlr - online photo editor and graphics editor separately. The latter looks like a photoshop. Saves to disk.
  • BeFunky is an online photo editing service. Effects, frames, graphics, retouching and more. There is a Russian interface.
  • Inkscape is a full-featured open source vector graphics editor. It supports the capabilities of the SVG format - work with shapes, outlines, text, vectorization of raster graphics, etc. In addition to processing vector images, it allows you to create a site template and cut it - a modular grid is present. Gives you the ability to draw backgrounds, banners, vector logos, etc.
  • IconEdit2 is a very simple icon drawing program. It has filters. Allows you to export an image for Android and Apple. You can adjust the color depth, rotate and adjust the icon. Supports translucency, multiple images in a single ICO file and anti-aliasing in all drawing tools.
  • Falco GIF Animator is a program for creating time-lapse animation and animated icons. On the resource you can download a lot of different programs.
  • Krita is a professional open source desktop editor. Available for different operating systems.
  • Boxy-SVG is a vector graphics editor similar to Adobe Illustrator, Sketch, and Inkscape.
  • Photopos is a multifunctional program with the functions of a photo editor, a graphic editor and an application for creating computer graphics.
  • 5Dfly is a free photo batch processing application. Able to create a slide show under PowerPoint. Separately, you can download the converter to PDF.
  • PaintStar is a free image editor. The interface is very similar to Paint, but more features. Layers, for example.
  • PhotoScape is a simple photo editor. It has the functions of adjusting brightness and color, white balance, backlight correction, frames, balloons, mosaic mode, adding text, drawing images, cropping, filters, removing red eye, coloring, brush, clone-mark, effect brush, etc.
  • Pablo is an online editor that creates images for social networks. You can upload an image, view it for different devices, impose an effect and and text.
  • Internet Marketing Ninjas - online service for quick cropping images. Very handy if you need to do it quickly.
  • Meme Generator - memes generator. You can select a picture, create a caption or generate it automatically. Not downloaded, only published in social networks.

Logo creation software


  • Free Logo Maker is an online editor for creating simple logos. Allows you to draw from scratch. Free saves in png.
  • DesignMantic is a service that generates a logo. In the editor, you can change by adding a new form or text.
  • Logotype Maker is a service that generates a logo based on a template. Then you can change in the editor. Saves in JPG and PNG formats.
  • Jeta Logo Designer - allows you to quickly create a colorful logo. Very easy to handle. It has a set of more than 10,000 shapes and icons. The possibilities of styling are quite large. Objects, like text, can be transformed and rotated, add smoothing effects, glow, stroke, shadows, etc. There are about 200 styles for figures. Manual drawing is not provided, but you can upload pictures. Has its own logo templates.
  • Flaming Text is an online service for logo design. Based on templates, but has a built-in editor. Saves in PNG format.
  • Logo Ease - a program for creating logos. The logo can be created on the basis of the template, edit.
  • Online Logo Maker - online logo creation service. Simple editor, easy to download. Free option only at 300px.

Prototyping


  • Napkin is a desktop application for quickly creating mockups for iPhone.
  • HTML5 UP - responsive and responsive HTML5 templates.
  • Inkbrush - Online tool for designing email letter templates. It is combined with postal services.
  • Marvelapp is an online service for creating an interactive prototype. You can create prototypes for different devices. Allows you to load ready-made layouts by simply dragging them into the specified area. Use mockup for pictures. Load, select areas, customize transitions. Integrated with DropBox and GoogleDisk, Sketch.
  • MockPlus is a program for creating prototypes on all platforms. Download and create. Available for all devices.
  • InVision is a collaboration application for a prototype project. Tool for companies of any size. Development of a software product with general access - all stages can be seen and controlled by both the developers and the management of the company.
  • Pencil is a desktop tool for creating prototypes and interfaces. It has a large set of tools, including a chart.
  • iPlotz - creating managed site frameworks or applications. For templates, you can get the HTML code. There is an online version, there is a desktop version.
  • MockFlow - creating user interface models. Free access is limited to one user per project.
  • Pictofon - quick creation of a mocap for any device. Select the layout, load the image and get the mocap in PNG format.
  • Placeit - Free templates and mockups for all types of devices. Choose a layout (the view angle depends on it), specify the site name and see how it will look.
  • Pixli is a web design platform. There is a free account in which you can create a mocap, a prototype and, on its basis, a site immediately.

In our collection we collected 101 design tools. Not all services are absolutely free, some have paid accounts. However, free versions can also be used to get the result.
