Calculate Linux 5 years!

Five years ago, the domain calculate-linux.ru was registered and an archive was posted with the first version of the Calculate Linux distribution. The distribution kit could be installed only by an experienced Gentoo user, since it was distributed in the form of a tar archive of Stage4 image (913 MB). Console installer along with themes and settings could be downloaded only by a separate archive (17 MB). In addition, Calculate used its original update method by deploying a new version of the system to a free hard disk partition. The first two years the distribution came out on the first day of each month.

The first versions of Calculate Linux had a KDE 3 desktop, designed in their corporate identity, which did not change during the whole time.



Fig. 1. Appearance of the desktop of the first versions of Calculate Linux

The main task of opening the project was to share their work. We all store information in files, notebooks or bookmarks. Each has its own. The site allowed to put together all the necessary documentation and share it with others.

For a long time, the distribution did not have a graphical installer, all configuration was done exclusively by console utilities. Many things took root months or years later. These include the use of binary package repositories, porting to Git, support for several distributions. Gradual development made it possible to eliminate most of the errors and not destroy the project.

Now, for the anniversary of the project, preparing Calculate Linux 12 update is being prepared.



Fig. 2. Appearance of the desktop Calculate Linux Desktop 12 KDE

A distinctive feature of the distribution will be the use of new Calculate 3 utilities (so far only installation and configuration of the system), which took almost 2 years to develop from idea to implementation. Calculate utilities are used everywhere - when building a system, installing, configuring packages, setting up a user session, etc. Traditionally, utilities work from the console, neglecting the GUI. A feature of the third version of the utilities will be the separation of the client-server. The client will work both from the console and through the GUI, displaying server information. In this case, the server can work on another machine. Currently, console and graphical (Qt4) clients are implemented. In the future, as needed, clients will be written under the Web and Android / iPhone.
