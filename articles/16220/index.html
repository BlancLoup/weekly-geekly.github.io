Events before the release of the site!

1. Check the full compliance of the laid out layout, the original layout of the designer (digressions, navigation, colors, whether it is in its place text content, etc.).
2. Check availability 4O4 page.
3. Check all pictures (quality, size, prescription alt, see how pictures open in new windows).
4. Check the layout in (Opera, IE, Mozilla), at screen resolutions 800x600, 1024x768, 1280x1024.
5. Check the spelling on the site.
6. Check whether tittle is registered on the site.
7. Check the convenience of navigation, moving from one page to another (understanding of which page of the site the user is in; whether the active section is underlined, if a color change occurred during the transition to a section, or if another effect occurred)
8. Check for favicona.
9. If the site is with flash elements, check whether it will load if the user does not have flash.
10. Check the availability of a password in the content management system, enter all data in the FTP access table.
11. Test the operation of the content management system, test all its functions.
12. Place the site on the selected client hosting. To inform the project manager about the successful placement of the project in the Internet.
13. Prepare a disk with graphic layouts of the site, flash-layouts, software, logins and passwords to the site.
