Skype Gateway 3CX Gateway for Skype 2.0

The Russian company ICE Partners, the official distributor of 3CX Ltd, announced that the final version of the 3CX Gateway for Skype 2.0 program was released, which allows you to transfer calls from Skype to landlines and mobile phones. This is what was missing from all Russian IP telephony operators.

A bit from myself:
Three years ago, when I worked as a programmer at Infosvyaz, customers asked a question: can the ‚Äú Universal City Number ‚Äù allow transferring calls to Skype or from Skype. Then it seemed to us that there was no solution. Skype is a private protocol and proprietary software code. Having rummaged in the sources available to me, I answered with regret - it is almost impossible.
Last year I bought a laptop and when I first started the computer asked me for permission for the application to use the access to the Skype client. On the Internet, I found that the new version of Skype has a software interface so that other applications can transmit voice and video through it. I decided for myself that the appearance of services of VoIP operators to work with Skype was not long to wait.

Throughout the summer of 2009, a Skype gateway for 3CX was beta tested. According to its results, since the beginning of September, the final version has become available - 3CX Gateway for Skype 2.0 Final.

3CX Gateway for Skype is a software gateway for 3CX Phone System, developed by 3CX Ltd, which allows you to receive and send calls to the Skype network from any extension.

Now you can place the Skype Call Me button on your web site, or inform your partners about your corporate Skype account - all Skype calls will be processed as normal incoming calls! They can be recorded, transferred to mobile, queued, etc.
At his internal number, the user can enable redirection to your personal Skype account. In this case, it became possible to use communicators based on Windows Mobile OS or iPhone. It is enough to install on them the mobile version of Skype and receive office calls on these devices!

3CX Gateway for Skype is installed on the same server as the 3CX Phone System. This makes it possible to launch several Skype applications on one computer for organizing a multichannel line.

3CX Gateway for Skype features:
- support for incoming and outgoing Skype-calls;
- the ability to specify the main Skype-account, which will receive calls. Then they will switch to subsidiary accounts, and the main one will always be free to receive new calls;
- Support for the latest version of Skype;
- add Skype accounts via 3CX interface;
- indication of the balance on Skype accounts;
- call logging.
