Installing Java applications on SAMSUNG U800 Soul b

In light of the craze for android, iPhones and other sophisticated devices, they ignore simpler devices, believing in vain that they are good for nothing, but what should people who do not want (cannot) part with the existing devices do? Exit - finish yourself!

At one time, was acquired by SAMSUNG U800 Soul b. It was chosen as an alternative to the Nokia 6500 (the ability to work in 3G networks, 1Gb of memory on board, a convenient slot for a memory card, and last but not least, design).

As the phone never failed, as the modem for UTEL - 1-1,5 Mbps is stable, the camera is so-so, of course, but the biggest disappointment is the complete lack of access to the default file system. ICQ and Opera managed to put through bitpim and shamanism with switching USB settings.

But I also wanted to use the phone as a book reader, and I wanted to load books as easily and quickly as possible. And the solution was found.



All actions performed on Windows 7. (For XP, you need to update the windows media player)

In the phone menu, select:

We connect the phone to the computer via USB. In the phone menu "Select USB mode" select item: "Synchronize Windows Media"

After that, in the explorer in the section "Portable devices" we see our phone: SGH-U800.


We go, we see two folders: Card and Phone.

Open the phone. Create a folder Games.

Copy the folder containing the application in JAVA to the Games folder, there should be 2 files in the folder: with the extension .jar and .jad. (Each application must have its own folder.)

The next time we connect to the PC, we can see the GAMES folder and all factory games and our installed applications.


To delete an application, first delete the application menu (where the games are), then delete the game folder directly from the phone (we find another folder - applications, all the installed games are visible there), first remove the memory card from the phone, update the Java DB (see . above).

No matter what they say about the lack of access to the file system in Samsung, I will say that this phone has access to the file system, not the entire system, but still ... If there is no filesystem folder on the memory card, we also create programs that can work with FS choose mmc_filesystem. In this folder we can create subfolders, etc.



After gaining access to the file system of your phone, I installed a reader for books in FB2 format. Now you can not bother with the input codes (update database JAVA). Just throw books in the mmc_filesystem folder or subfolder.
The reader is called TequilaCat BookReader. Files and detailed instructions are on the site developer TequilaCat
