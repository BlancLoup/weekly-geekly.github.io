Attempt to reincarnation and stuffing in Open-Source Yandex.Following

Hello, habrasoobschestvo! Many have noticed that Yandex has closed the subscription service. Reading the comments, I had the idea to create an analogue. Raise it to the API from Yandex, and then transfer it, so to speak, to self-contained power, replacing the API with your own (do not worry, I will not do it as imperceptibly as the Great Yellow did).

Development went. At the moment, it is already possible to see the list of feeds, get all the posts from them, see the tags that were assigned to these posts. There is also a counter of unread messages, as well as a counter of new messages since the last activity.

At the moment, the service is in a state of deep alpha, even, so to speak, under-alars. Post on Habr√© was written in order to attract public interest in this project, as well as get supporters who will help develop this service. Also would not hurt the server capacity.

So welcome to YaGrabber !
