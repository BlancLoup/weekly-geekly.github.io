The del.icio.us buttons in FF3

With great pleasure, I switched to Firefox 3 - a lot of nice improvements in terms of convenience.
The only problem that this version of FF has now is incompatibility with plugins written for earlier versions and not the desire of some developers to write updates to their crafts for beta releases.

One of these capricious plug-ins was the plugin from the social bookmarking service del.icio.us. The support service said so: until the release comes out, you will not see the new plug-in.

Well then.
