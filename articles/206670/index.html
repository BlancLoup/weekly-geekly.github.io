Add attachment to your resume

It is no secret that issues of a successful career concern many students. And although there is a shortage of specialists in the IT labor market, many still think about how to find the best job. Especially for you, we have come up with a competition , by participating in which you acquire useful skills and you can get an internship in one of the best IT-companies!



By the way, what do you think, how much time does an HR employee spend on viewing one resume?

It turns out that it takes about 10 seconds to view a resume! What then do employers pay attention to? First of all, these are concrete achievements (preferably documented):
  • wins contests ( Imagine Cup )
  • received certificates
  • made projects (portfolio)

Let us dwell on the last paragraph - projects. A good project is a complete project that can be shown. From this point of view, the ideal project is a mobile application placed in any application store (Windows Store, Apple Store, etc.) - confirmation of the project made in this case is simply a link to the application, where you can download it if you want look.

Especially for those who are thinking about their careers in the future, together with ITmozg , we developed a project in which you can get prizes for developing a Windows 8 / Windows Phone application. The project website has detailed conditions and links to learning resources that will help you learn how to create applications, as well as some ideas that can be taken as a basis. It's time to spend a little time on vacation in self-education, and create an application that you can be proud of later.

The authors of the best applications will be invited to undergo training in large IT companies as a developer on the Microsoft platform. Other winners will receive nice prizes (books, hard drives or additional space in the Skydrive cloud storage), while all participants will be able to specify a link to the created application in their resume that will help you find a good job!
