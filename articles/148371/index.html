Windows 8 Summer School



All of you have already heard about the new operating system from Microsoft - Windows 8 . Some, even, probably installed and tried to write applications for it.

Windows 8 is Windows reinvented. It works on ARM and x86 / x64 computers: tablets, slates, ultrabooks and desktop computers.

Continuing the tradition of the summer school on Metro-design , we decided to hold a summer school for developers on Windows 8 .

UPD: You could get a letter about the "transfer of the metro-school on July 13-14." This is an erroneous letter. Our event is not a metro school and it is not transferred (for July it is not possible), it is not canceled, and it will be held on August 5 in Gorky Park (Neskuchny Garden).

Short description


Anyone who is interested to get acquainted with the basics of developing under Windows 8, learn about the possibility of monetization of applications on this platform, find or come with a team of like-minded people and start developing the application, having the opportunity to ask a question to community experts and Microsoft, we invite Boot Camp to Windows 8.

A bit about the format of the event


The format of the event: "managed hackathon" with lectures.

You need to come with laptops with Windows 8 installed and development tools .

It is better to come up with a ready-made idea, a team, but you can come up with something and find like-minded people on the spot.
