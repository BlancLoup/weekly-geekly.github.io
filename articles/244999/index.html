Kotlin for Android, or an easy way to stop programming in Java

At our conference of mobile developers # MBLTDev , which took place at the end of October in Moscow, Ilya Ryzhenkov, a product specialist at JetBrains, told us about the Android programming language Kotlin .

Kotlin was developed by JetBrains and publicly introduced in 2011. In 2012, the source code was opened.


Presentation

Ilya told about the history of creating a language, the advantages of programming in Kotlin, the differences between Kotlin and Java, and what can be done on it. He also showed some code samples on Kotlin for the Android platform.
