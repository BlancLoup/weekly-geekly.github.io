X-Jumper: New Platformer Puzzle Game


X-Jumper is a very complex, but fascinating indie platformer from domestic developer Platinum Arcade. The game has a number of unique game elements. See the trailer inside the news to get acquainted with the gameplay more clearly.

The game currently has 12 levels, dynamic gameplay, there is a map editor.

The essence of the game


In the game you need to collect all the crystals / coins / apples for speed and get to the exit.

The uniqueness of the gameplay is achieved through the use of springy floor and special portals. By combining them, you can pass the level in hundreds of ways.

Similar games


The game will surely please fans of games such as N , Portal: The flash version and Elasto Mania .
