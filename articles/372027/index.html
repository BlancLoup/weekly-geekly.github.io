Microsoft launched bots in Skype


Several bots from Microsoft in the new version of Skype

Today at the Build 2016 conference, Microsoft announced the release of a new version of Skype with support for bots for Windows, Android, iPhone and iPad, as well as the opening of the SDK and API for Skype Bot Platform.

You can add a bot to your Skype, Slack or Telegram contacts from the bot directory .

Search bots works in the new version of Skype for iOS and Android.



To access the Skype Bot Platform, you need to register with Skype Bot Developer Preview. In the same place, registration is open for participation in the bot creation hackathon, which will be held in May.

In the future, it will be possible to communicate with the bot in a natural language via audio and video calls (Skype Video Bots). Thus, the "artificial intelligence" will receive a visual form - a 3D model that will communicate with the person.



Microsoft believes that automating actions through bots and voice communication will change the usual ways of making purchases, games, ordering food, setting up meetings, reserving tickets, managing a smart home, and much more.

In addition to Skype bots, Microsoft introduced the Microsoft Bot Framework to program chatbots in Slack, Office 365, web applications, etc. Probably, the chatbot Tay worked on this framework until it was turned off for racist statements and drug taking .
