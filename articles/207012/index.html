Embree 2.1 has been released.


A new version of Embree 2.1 ray tracing kernels developed in Intel is available for download. Embree is designed for graphics developers who want to increase the performance of their programs using Intel cores that are optimized for rendering photographic quality images on the latest Intel processors that support SSE, AVX, AVX2 and Xeon Phi vector instructions. For maximum performance, it is recommended to use Embree via API. Embree is open source software and is licensed under the Apache 2.0 license.

On KDPV - rendering of an automobile block headlight with use of Embree, the author - Martin Lubich .

What's new in Embree 2.1?
  • New API with significant focus on dynamic scene support
  • Many optimizations for AVX2 (Intel Haswell architecture)
  • Automatic selection of executable code for SSE, AVX and AVX2
  • Support for user-defined geometry
  • New and improved BVH builders

Download Embree 2.1 for Windows, Linux and Mac or in source codes.
