Final build of Windows 1809 revoked

Due to known problems with deleting files from a profile when updating, Microsoft decided to recall the final build of Windows 10 October 2018 Update (1809)

Since October 6, the assembly is not distributed through the Windows Update service. If you have downloaded the ISO image yourself, do not install it and wait for the new build.

The current builds of Windows Server 2019 and Windows 10 Enterprise (including LTSC and IOT) are also affected by this problem.
