HTML5 and CSS3. Site development for any browsers and devices



Annotation: today more than ever the problem of responsive web design is acute. More and more tablet computers, smartphones and even televisions are used to access the Internet. Web developers need to take into account the huge variety of screen sizes, as well as take into account the characteristics of the corresponding user interaction. Responsive web design allows you to best display the content of sites on the screens of devices used for viewing. At the same time, web pages will look good on displays not only of modern devices, but also of those that will appear in the near future. Start to develop sites in accordance with the new methodology of responsive web design, so that they will be beautifully displayed on screens of any size. Read this book, incidentally creating and improving responsive web designs using HTML5 and CSS3. You will learn how to put into practice new technologies and methodologies designed to become the tools of the future for web developers of client applications.

HTML5. For professionals



Annotation: we are on the verge of a revolution in web development and programming for mobile devices, and at the heart of all these changes is the HTML5 format. This book will help you gain the necessary knowledge about this standard and master the complex topics included in the HTML5 specifications. You will learn how to organize your documents with HTML5, how to style them using CSS3 and how to work with the most advanced JavaScript API. This publication is not an introduction to HTML5, but a full-fledged training course that will teach you to create modern websites and web applications using HTML5. Each chapter is devoted to a specific key topic of HTML5, and also contains complex HTML5, CSS3, and JavaScript issues. The book contains many examples of software code, so you can easily assimilate and apply knowledge for each tag, style and function included in the HTML5 specification.

HTML5. Programming recipes



Annotation: this book, which is a collection of recipes and ready-made solutions, will allow you to gain practical experience with the basic elements of HTML5. The publication covers a wide range of issues: from semantic markup, web forms and multimedia elements to geolocation technologies and JavaScript API. Each recipe discussed in the book includes a task, sample code, and a detailed description of the solution. The book is ideal for web programmers of primary and secondary levels who want to quickly master the practical techniques of using HTML5 in web development.
