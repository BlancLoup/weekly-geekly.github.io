Safari 5 released

Apple sent a press release on the release of Safari 5. The press release quoted Philip Schiller as saying that Safari is used on more than 200 million devices worldwide, and his free WebKit engine on more than 500 million devices.

Major changes include the emergence of the Safari Reader feature, improved performance, the addition of Bing to the number of available search providers, and improved support for HTML5.

Safari Reader is said to facilitate the process of reading web pages through special text organization:



JavaScript in Safari 5 runs 30% faster than Safari 4, 3% faster than Chrome 5, and almost twice as fast as Firefox 3.6. Safari 5 loads web pages faster by improving caching and getting DNS records.

Safari 5 adds support for many HTML5 features: full screen playback, closed captions for HTML5 video, HTML5 Geolocation, HTML5 sectioning elements, HTML5 draggable attribute, HTML5 forms validation, HTML5 Ruby, HTML5 AJAX History, EventSource and WebSocket.

The new free Safari Developer Program allows developers to customize Safari 5 with extensions based on HTML5, CSS3 and JavaScript. Extension Builder makes it easy to develop and install extensions.

Safari 5, as usual, is available for both Mac OS X and Windows. Safari 5 for Mac OS X requires Mac OS X Leopard 10.5.8, Mac OS X Snow Leopard 10.6.2 or higher, and Safari 5 for Windows requires Windows XP SP2, Windows Vista or Windows 7.
