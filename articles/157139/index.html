AWS: More IOPS for EBS

Good day!


Another good news from Amazon Web Services has become known today: now each EBS disk provides 2000 IOPS performance (twice the previous 1000 IOPS limit).



If the disk performance in 2000 IOPS is not enough for you, then EBS disks can be combined into a RAID array, thereby obtaining the required number of I / O operations.
