Updates on vds.menu

Hi, Habr!

Since the announcement of the vds.menu project, only two weeks have passed, but there are already news that I would like to tell: during this time, a number of changes have been made that improve the quality of the search for virtual servers.




First, the opportunity to search by price for 3, 6 or 12 months. Very often, hosters make discounts for long-term payments, so now you can specify the period for which you want to pay and the minimum and maximum prices. At the same time, the total price for the specified period will be displayed in the tariff information.




As a result of this, another improvement appeared: now you can move the mouse to the cost information and see the price when paying for a month, 3 months, six months or a year.



Due to the unstable currency exchange rate, foreign hosters have already ceased to be so attractive in price and many are looking for a similar replacement, for example, with a fixed price in rubles. Especially for this was added the ability to search for the currency of the host.



If you move the mouse to the type of virtualization and see a list of operating systems that the hoster offers to install.

Search by processor can now be done not only by the total frequency, but also set limits on the number of cores:



It was possible to specify restrictions on the channel speed and select additional criteria for the search: the presence or absence of a Windows license included in the price, the presence or absence of an ISPmanager license, the presence or absence of IPv6 addresses, the type of server activation (manual or automatic) and the minimum payment period (1 month or more than one month).



I hope that these improvements will help find the best virtual server.
Also, I remind you that in order to add tariffs to the site, the hoster needs to fill out this table and send it and the logo on a transparent background to info@vds.menu
