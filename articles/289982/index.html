Tips for those who want to do something

Of the main tips for someone who wants to make an application for the first time - get ready thoroughly! The more and more detailed you think of everything yourself, the easier and faster developers will be to implement your idea.

  • Start now! "You will always regret that you started too late, even if you start doing it on time."

  • High-quality technical project preparation. There will be a good TK - it will be cheaper development and less problems for programmers. The more detailed and detailed, the better for everyone.

  • Marketing plan ahead. Budget for advertising too. My case is more the exception than the rule.

  • Feedback is an important and indispensable factor. Answer all letters. I regularly write users with requests, wishes, advice and complaints. Not a single letter, not a single comment in social networks remains unanswered.

  • You start not when you create an application, but when it enters the Appstore. Here I will not surprise anyone, you yourself know it.

  • Make the application and run it in the basic version as soon as possible. Perfectionists in such a dynamic business will be the hardest. Updates you can then add the desired functionality.

  • Think of the benefits for people that the application will perform. Think about the money later.
