How does Windows work?
After installing the operating system, you have 30 days - the trial period. During this period, you can use the operating system with full functionality. After this period, the system will remind you of activation by a pop-up window, an inscription on your desktop, and a black screen. Microsoft provides a legal way to reset activation. This reset can be performed 3 times. To perform activation reset, we need to open a console (command line) with administrator rights and execute any of the following commands:

slmgr / rearm
sysprep / generalize

At the moment it is the most painless way to live without activation. The system will not be different from the original installed and will live forever. 1. Activation with the Key Management Service
I will describe the technology itself. There is a server that is activated by a special key. After that, it becomes able to activate other operating systems that access it with a request. The request can be either in automatic mode (internal network with DHCP, DNS), or manually (by entering several commands). The technology is intended for corporate users. Please note that no software is required to install.

Pirates have expanded the idea with the server and provided 2 options to choose from:

A) A dedicated server that you run in a virtual machine and activate the operating systems you need. Convenient for those who do not want to rely on others, for the local network of a small enterprise. A computer is required. Able to run a virtual machine from time to time (additional 512 megabytes of RAM)

B) "Chinese server". Several activation servers are available on the network. To activate, you will need to register the IP address of one of these computers and periodically monitor whether this server is alive. It should be noted that the disadvantage of this activation method is that it is necessary to have an Internet connection.

After the activation is completed, it does not work for life, it is periodically checked and updated. The maximum activation period is 180 days. This means that you need to connect to the Internet more often than in 210 days (activation + 30 days period).

2. BIOS firmware
The following 2 activation methods are based on the principle used by OEM manufacturers of computers (Sony, Dell, Lenovo and others). 3-step technology. To complete the activation, you must comply with 3 points:

A) The presence of a key to activate
B) Availability of the manufacturer‚Äôs certificate in the operating system
C) The presence of SLIC tables in the BIOS.

Manufacturers prepare in a special way the bios of the computer, stitching data into it, which the operating system then receives. This allows the system to verify that it is running on genuine hardware and to perform activation. This data, flashed into BIOS, is called SLIC.

History: For whist table version 2.0 was required. The new generation of operating systems requires version 2.1.

The next step, the manufacturers prepare a special image of the operating system into which the manufacturer's certificate and other garbage are implanted (a bunch of very necessary trial software, drivers). Usually this disk is applied with a laptop and users should install the OS only from it.

Pirates modify the BIOS for your motherboard (by adding the SLIC table), you flash it and get an indistinguishable from a legal computer. Next, you need to install the certificate (using a modified distribution or activators) and activate the system.

On the same board and several other resources there are ready bios. You can ask more experienced colleagues to prepare a bios for you.

+ 100% reliability of the method.
+ Activation of life, you can not be afraid to put updates.
- Not all BIOSes and computers are supported.
- BIOS flashing operation is dangerous
