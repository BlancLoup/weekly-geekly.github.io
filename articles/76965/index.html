Nokia's wise company and language switching on Nokia E52

Honestly, I have never loved the S60 platform because of its clumsiness (in any case, according to my inner feelings). But such a horror, as in the E52, I did not expect even from Nokia.



It is given: a very beautiful E52 phone, working under S60 3rd Edition FP2, in which Russian is specified in the settings as the first input language and English as the second one. Task: write a message in Russian using a couple of English words.

It would seem, what could be easier? Using many years of different brands of phones, I got used to the fact that the language is switched either with one button (Samsung, Sony Ericsson), or from the context menu (Motorola). What was my surprise when I learned that there is no way to switch to another language right in the input process. More precisely, there is one, but extremely perverted: it is necessary to climb into the menu Options -> Input Options -> Writing Language and change the first and second language places there!



For the sake of justice, I should note that when T9 is on, the phone will try to guess by itself which language you want to type in. But this is not an option, because when you try to enter a more or less difficult word, the smart system constantly makes mistakes and offers options in the wrong language.
