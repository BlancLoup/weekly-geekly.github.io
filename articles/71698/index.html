VMware Announces Fusion 3.0 for Mac Virtualization System



VMware Company announced the product VMware Fusion 3 and announced the release date of the new version of the popular virtualization software.

According to the manufacturer, the delivery of a package that allows you to run Windows, Linux and other operating systems on the Intel Mac hardware platform, will begin on October 27. A solid list of improvements implemented in the new version includes extensive support for the new operating systems Snow Leopard and Windows 7 and a wide range of modern graphics technologies, as well as the new Migration Assistant tool, which makes it extremely easy to transfer the system to a virtual machine.



In total, users are invited to more than fifty improved and new features. According to the manufacturer, the version of VMware Fusion 3 is optimized for running on systems running Snow Leopard, and the redesigned virtualization engine ensures stable and productive operation of the application on computers equipped with 64-bit processors. Fans of computer games will appreciate the expanded support for OpenGL 2.1, as well as DirectX 9.0c and third version shaders. In addition, the new version of Fusion for the first time supports Aero interfaces in Windows 7 and Windows Vista.
The Fusion Unity technology, which allows displaying windows of Windows applications along with the windows of Mac OS X programs, has been significantly improved. For example, users can switch between Windows applications using keyboard commands, and Windows application icons in the Dock interface allow you to view a list of open windows.

VMware also offers customers to evaluate the new Migration Assistant tool and the innovative method of transferring physical systems to virtual machines. The user simply inserts the CD into the appropriate drive on the Windows-based computer, runs the installer, gets the four-digit code generated by the system and enters it on the Mac system. All remaining operations will be performed automatically without user intervention.
