Overview of additions to popular CMS for the organization of microblogging service



In a recent review, I described the best standalone solutions for microblogging. Now we will see what popular CMS offers on this account :-)



Drupal microblog


Website: drupal.org/project/microblog


Users can update their status, subscribe and respond to the status of other users. The module also allows you to show the so-called live site, compiled from the latest status updates.
Unfortunately, the module currently does not provide an API for updating or for obtaining statuses from other sites.

Drupal facebookstatus


Site: drupal.org/project/facebook_status
Demo fbss.icecreamyou.com

This module allows you to simulate such a function of Facebook and Twitter as a ‚Äúwall‚Äù: each user notes what he feels at the moment or what he is doing. There is also tag support (#, @) and the tet-a-tet function, similar to Facebook. It is necessary to note the presence of an open API.


Joomla microblog


extensions.joomla.org/extensions/news-production/microblogging/8202

Implements most of the functions of microblogging. Of the features it should be noted the use of the default Joomla editor for formatting messages.

TWPtter


wordpress.org/extend/plugins/twptter


TWPtter is a microblogging plugin for Wordpress. It allows you to publish short posts (also known as twps). They can not be more than 150 characters long, but allow you to include links to other sites, articles and blog posts. We are pleased with the support of Wordpress tags and search for twipses by tags. Posting by email is also possible. The plugin is perfectly customizable: it is possible to specify the page title for twips, the number of twips per page, their sorting, etc.

Wordpress microblogger


wordpress.org/extend/plugins/microblogger

Creates a sidebar on which the latest statuses are displayed. It is worth noting the flexible configuration of the appearance, through a special UI and the generation of RSS-feed from the microblog.

LiveStreet's Microblogging Module


Demo site: extblog.ru
Examples of use: magov.net/micro
In the modules catalog: livestreet.ru/addons/82


Adding tweets occurs on ajax without reloading the page, as well as commenting them, which is very convenient. There is a mechanism for hashtags, the ability to attach videos and images, automatically shorten URLs, vote for tweets, and more. In addition, it is planned to integrate with the Twitter API and pingator, which allows getting into the catalog of blogs.yandex.ru/services .
Tweets are displayed for each user, friends, and also in the form of a ‚Äúmicroether‚Äù of the site.

Of all the solutions found, this module, unfortunately, is the only one that is paid. Perhaps this is his most significant drawback.
