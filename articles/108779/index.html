Ready solutions on ASP.NET for creating websites

Often, the ASP.NET platform claims that there are no ready-made solutions based on which corporate websites, forums, blogs, etc. can be worth. etc. In this small post I list the most popular web applications written in ASP.NET, which, in addition, can be installed using the Web Platform Installer (WebPI) .

Title
Features
Official site
WebPI
Blogs

BlogEngine.NET
A large number of themes and extensions have been created for BlogEngine.NET, it is possible to create your own components. Supports SQL Compact and XML data storage. www.dotnetblogengine.net
Installation
dasBlog
Does not require the use of DBMS, data is stored in XML files.
www.dasblog.info
Installation
Subtext
An easy-to-use blog engine with beautiful architecture. One of the creators is Phil Haack, a member of the ASP.NET MVC development team.
www.subtextproject.com
Installation
Atomsite
Convenient blogging platform, similar to the popular Wordpress engine.
www.atomsite.net
Installation
Content Management Systems (CMS)

DotNetNuke
One of the most popular CMS on .NET. For DotNetNuke there is a huge variety of modules and extensions.
www.dotnetnuke.com
Installation
Umbraco
The most dynamic open source CMS on .NET. Designed for creating websites of varying complexity - from promotional sites to large portals.
www.umbraco.org
Installation
mojoPortal
Supports a large number of MS SQL 2005/2008 / SqlAzure DBMS, SQL CE, MySql, PostgreSql, SQLite, or Firbird Sql. Extends through the UserControl mechanism.
www.sourcetreesolutions.com
Installation
Kentico
Supports advanced functionality for creating social networks.
www.kentico.com
Installation
Kooboo
CMS is developed on the MVC Framework.
www.kooboo.com
N2 CMS
A project with good documentation in which there are cross-references to more detailed information on the ASP.NET platform in MSDN. May be a good help when learning ASP.NET.
www.n2cms.com
Installation
Orchard
A project created by developers from the ASP.NET team. A good example of extensible architecture.
www.orchardproject.net
Instruction
Composite C1
Multifunctional CMS. Convenient content management interface.
www.composite.net
Instruction
Bitrix.NET
.NET version of the most popular commercial CMS in Russia.
www.1c-bitrix.ru
Axcms
CMS for creating large portals.
www.axcms.net
E-commerce

nopCommerce
One of the most popular e-commerce solutions on .NET. Developed by guys from Yaroslavl.
www.nopcommerce.com
Installation
Kartris
Online store engine written in VB.NET. Convenient user interface for the site.
www.cactusoft.com
Installation
Forums

YetAnotherForum.NET
The most versatile open source forum on ASP.NET, a simple system for applying topics, the ability to integrate into an existing ASP.NET site.
www.yetanotherforum.net
Installation
Galleries

Gallery server pro
The engine for publishing galleries of images, audio and video. Supports Silverlight and Flash.
www.galleryserverpro.com
Installation
Kigg
News Gallery.
kigg.codeplex.com
Wiki engines

ScrewTurn Wiki
Wiki engine with plugin and theme support. The ability to use a SQL Server database or file storage.
www.threeplicate.com
Installation

I invite you to help in the addition of this list. In addition to adding to this list, I will recommend to colleagues to add these applications also to the web application gallery used by the Web Platform Installer . And also I invite you to talk about your own experience using these engines.
