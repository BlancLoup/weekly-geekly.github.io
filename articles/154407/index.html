Gmail Show M. Later

Some time ago, I made one very handy thing for Gmail



In short, this google script allows you to create shortcuts (folders) in your mailbox, putting the letter in which it will lie until the required period passes, and then it moves again to the inbox and becomes unreadable. I have not had such a thing for a long time, because Often there are letters to which you need to respond, but at the moment there is no opportunity to respond to them.

There are more ancient options in the gallery of scripts, Gmail Snooze for example, but they allow you to work only with daily intervals and are designed in such a way that the letter added at 23:00, at 01:00 will again appear in the inbox. Therefore, I first modified and then completely rewrote. In my implementation, the letters live a more correct life.


The trial period lasted two weeks. This is only the latest stable version, there were other implementations before that, I tried using Cache instead of ScriptDB, but he too often lost keys. With this version there were no problems, a couple of times, it was that the script was hanging, but I scored on this case, because I could not repeat the error.

Regarding the volume of data. Each google user is allocated 50mb under ScriptDB, but it can be more if you have a paid account.
