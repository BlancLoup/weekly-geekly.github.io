Tools for exploring and migrating to Windows Phone 7 for iPhone developers

Windows Phone 7 Interoperability is a resource that was launched by Microsoft to support developers of third-party mobile platforms who want to familiarize themselves with the development of Windows Phone 7 and be able to quickly port their application code to WP7

The other day in the official blog devoted to the development for Windows Phone 7 , the extension of the resource Windows Phone 7 Interoperability and the availability of a large set of new tools was announced .

A new tool that was announced is the API Mapping from iPhone iOS to Windows Phone 7 .



This tool is a convenient directory of conformance API iOS and Windows Phone 7, with which iPhone developers can quickly find similar functions for Windows Phone 7. The tool offers examples of using C # code and a link to the corresponding reference documentation on certain functions on MSDN.

In addition, the Windows Phone 7 Guide for iPhone Application Developers has been published, which consists of 8 chapters and offers iPhone developers an easy way to get started with developing for Windows Phone 7. The guide contains the following chapters:
  • Chapter 1: WP7 Platform introduced to iPhone application developers
  • Chapter 2: User Interface Guidelines
  • Chapter 3: Developer and designer tools introduced to iPhone application developers
  • Chapter 4: C # programming introduced to Objective-C programmers
  • Chapter 5: Image Format Considerations for WP7
  • Chapter 6: Application Lifecycle Differences Between WP7 and the iPhone
  • Chapter 7: iPhone to WP7 Application Preference Migration
  • Chapter 8: Introduction to WP7 Notifications for iPhone Developers
This manual can be read online or downloaded electronically (90 pages) in .docx or .pdf formats.

Another new proposal of the site concerns a series of videos under the general title Developer Stories: Experience developing for Windows Phone 7 , in which developers of different companies share their experience in developing applications for the Windows Phone 7 platform. Videos are available from the home page of the site http: //windowsphone.interoperabilitybridges. com / .
