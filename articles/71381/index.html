moc - console two-pane client-server music player



In my first post on Habr√©, I want to tell you about a wonderful music player that I use every day
$ mocp - player launch (server and client)

The player is controlled by pressing the letter keys of the keyboard:

Q (Shift + q) - closing the server and moc client
q - closing the client moc, the server at the same time, will continue to play

moc - Two-pane player.

Left panel - file manager
Right panel - current playlist

Tab - Navigation between panels

You can play files from both the file manager and the playlist.

And now let's deal with playlists.

a - Add file to play in playlist
A (Shift + a) - recursive addition to the playlist, those all music files in the folder will be added to the playlist

moc - has the most convenient playlist editor
music files added to the playlist are played in the order they are in

u - move the file higher in the playlist
j - move the file below in the playlist
d - delete file from playlist

V (Shift + v) - save playlist
C (Ctrl + c) - clear playlist
moc is very comfortable and functional

ENTER - start barrel organ
n - play the next track
b - play the previous track

> (Shift +.) Make the volume more
<(Shift +,) make the volume less

p - pause in playback (remove pause - press p again)
s - stop the player (start playback - ENTER)

moc has playback modes

S (Shift + s) - turns on the Shuffle mode - plays the playlist in random order
R (Shift + r) - turns on Repeat mode - playlist cyclical playback

h - help for moc commands

enjoy listening;)

Project site - http://moc.daper.net/
