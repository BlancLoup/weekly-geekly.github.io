Hyper CMS. Now we are with Joomla!



Thanks to our partners from Joom.ru, the HyperCMS special tariff line from Parking.ru has been supplemented with another high-performance solution based on the popular CMS Joomla!

Now you can use the special Russian-language build of Joomla 1.5.15 and create any sites, without limitation, using the flexible capabilities of this CMS system, all on the basis of a powerful Hyper-V virtual machine.

The composition of the virtual machine includes pre-configured and load-tested Windows Web Server 2008 R2 (x64) + IIS 7.5 + PHP5 (+ Zend Server) + MySQL5 and pre-installed and configured CMS Joomla! 1.5.15 RE (Russian Edition) assembly Joom.ru.
