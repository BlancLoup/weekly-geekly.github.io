XNeur and gXNeur version 0.9.4 released!

X Neural Switcher (XNeur)


This program is an automatic keyboard switch. Those. when typing, if it is typed in the wrong language, xneur will automatically (or upon request) switch the language to the desired one.

The principle of operation of the program is similar to Punto Switcher.

XNeur and gXNeur version 0.9.4 released!

List of changes:
  • Transliteration is given to the ISO 9 standard (GOST 7.79-2000).
  • Added the ability to switch the language when typing, and not after the space.
  • Added new actions, which are hung sounds and OSD.
  • Added the ability to commit layouts. If the layout is fixed, then there will be no automatic switching to it. Accordingly, if a fixed layout is currently enabled, switching from it will not occur.
  • Fixed work with accents (√¢, ≈±, √´ and the like). Thanks to Polish users for bug reports and testing.
  • Added Bulgarian, Czech, Greek, Estonian, Armenian, Lithuanian, Latvian, Polish, Spanish and Uzbek. In total, the program now supports eighteen (18) languages.
  • Improved gxneur, added all sorts of amenities. Of course, gxneur supports all the features of xneur.
  • Dictionaries, starting with this version, are entered in the input language, not in English code.
  • Developed new flags 24x24. They are conveniently used with gxneur in KDE environment. You can get here xneur.ru/browser/icons/bobs . It is necessary to put in / usr / local / share / gxneur / pixmaps.
  • Multiple improvements and bug fixes.


Remember, be sure to delete the old user dictionaries in ~ / .xneur / ru, ... be, ... uk, etc. These are files named dict.

I draw your attention, ALWAYS put aspell package for your language (usually, aspell-ru, aspell-uk, etc.) - the quality will improve significantly.

I also inform you that kxneur did not die completely. There are developers who make a new front-end Qxneur, which will be written using only Qt, without binding to KDE.

Download archives with programs can be in the "Downloads".

Deb

For Debian / Ubuntu distributions, packages may be included in the official distribution repositories, but it is recommended to install the latest version. To do this, add one of the following lines to the list of connected repositories (depending on the distribution used): 
  •   deb http://gq.net.ru/debian stable xneur 
  •   deb http://gq.net.ru/debian testing xneur 
  •   deb http://gq.net.ru/ubuntu feisty xneur 
  •   deb http://gq.net.ru/ubuntu gutsy xneur 
  •   deb http://gq.net.ru/ubuntu hardy xneur 
  •   deb http://gq.net.ru/ubuntu intrepid xneur

Download packages directly from this link.
