Home sales UC PBX 3CX Phone System v15

Expanded feature set and simplified administration allow you to deploy the system on a corporate server or in a private cloud

Introducing the new version of UC PBX 3CX v15, which greatly simplifies administration and has new features, additional security, modern interface and simple integration with third-party applications.

The first PBX with the ideology of Zero Admin


In v15, the routine administration of the system is reduced to almost zero. Most tasks are automated, and this allows you to abandon the unnecessary costs of third-party cloud services. You retain complete control over the system, wherever it is located - on the server or in the cloud. Security, backup and recovery, fault tolerance and update management - now 3CX automatically does it all.

Layered Security


The 3CX multi-level security is perfectly designed:

  • Automatic generation and management of SSL certificates
  • HTTPS phone auto setup
  • Connection of system clients and administrator via SSL protocol
  • A + rating by SSL lab
  • SRTP voice traffic encryption
  • Full traffic encryption when connected via the 3CX and SBC tunnel
  • Enhanced web server security
  • Detection and automatic blocking of attempts to use hacker SIP utilities

It can be said that today 3CX sets the standards for corporate PBX security.

New interface


The new system management interface and the appearance of 3CX customers have become even more convenient. The 3CX client interface is in line with the trends of modern UC clients, while maintaining the functionality of a softphone. Added a lot of elements, including the ability to install your own logo.



Extensive integration of external applications


Integration with external systems has been greatly expanded - many applications are supported out of the box, and new applications can be connected via a new script (script) interface. This interface allows integration with any CRM, call center, or outbound dialing system. We have simplified the installation of integration modules - now they are downloaded and installed centrally from the PBX server console. 3CX currently supports: Office 365, Google Contacts , Microsoft Dynamics , Exact , Sugarcrm , Salesforce , act! , Zendesk and Freshdesk. Expect new integrations soon!

  • Centralized installation and update via 3CX management interface
  • Simple scripting language
  • Easy adaptation of existing or creation of new integration with any CRM system
  • Creating outbound call scripts

Open standards integrated web conferences


3CX includes an open standards based video conferencing system. For its work does not require licensing, administration and installation of any software.

  • New technology of a single media stream to minimize traffic
  • New Android Client
  • Click2Meet Personal Link (click for conference)
  • The ability to install your own logo of the organization

Works on minimal resources


3CX v15 is optimized to run on miniPC - fanless PCs on SSDs, costing up to $ 100 (including the Windows license). The ideal solution for small companies with up to 32 simultaneous calls that do not plan to purchase a server.

New edition of FREE


In v15, we again present the free edition of our system! 3CX FREE Edition offers 8 simultaneous calls, 5 simultaneous web conference participants and 1 external SIP connection (trunk or gateway). Included and free updates for 1 year! This is an ideal solution for small companies that can later upgrade to a fully functional commercial version.

On Monday, July 11, we started selling 3CX Phone System v15 Release!
