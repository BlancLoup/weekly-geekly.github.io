BitTorrent Bleep: protected instant messenger from BitTorrent



As public authorities pay more and more attention to the interaction of ordinary people, many companies and individual developers have started developing secure instant messengers. The BitTorrent team, even last year, announced its intention to create such an application.

And now the BitTorrent developers have presented a new application, a protected instant messenger (so far only alpha release), called BitTorrent Bleep.

According to the developers, Bleep encrypts all user messages, all the way from user to user. In theory, only the sender and recipient of the message can read them. The Bleep team says that this type of communication on the Web should become commonplace in the ‚Äúpost-Snowden era‚Äù (Edward Snowden, remember?).

In principle, there are quite a lot of instant messengers who encrypt user messages (in fact, the vast majority of IM do this kind of work), but Bleep is one of the first, or the first application that does not leave metadata on the central server. Almost all instant messengers leave the metadata on a central server. At the same time, metadata is often stored in unencrypted form, which leaves you able to find out who communicates with whom, at what time, and how many messages are contained in a regular session.



As for Bleep, when a user communicates between them, a direct connection is established, and messages are sent directly to each other, bypassing the central server. The Bleep network is largely based on the BitTorrent protocol, with some additions that further enhance the security of communication participants. The developers claim that the Bleep uses the same distributed BitTorrent network, and any information is encrypted. This allows you to leave information about the location of the participants of communication closed to outsiders, to collect metadata with information about the participants of communication "is almost impossible."

Bleep is now available for Windows 7 and 8, Mac, Android. The iOS version is in development, and will not be released until the final release of the messenger.

In addition to text communication, with the help of Bleep, you can also make voice calls. One has only to remind you that the current version of the messenger is alpha prerelease, so you should not expect perfect work. You can get a new messenger on the BitTorrent site.
