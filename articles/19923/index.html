What kind of beast is this? Meet FMS3!

In continuation of the news that it was released that the long-awaited FMS3 was released, I decided to tell what it is all about.

The product has just been released in two versions: Flash Media Interactive Server (FMIS) and Flash Media Streaming Serve r (FMSS) . Flash Media Server 3 is a new generation of software from Adobe for streaming video and real-time communications. Flash Player, Adobe Media Player, Adobe AIR, and Flash Lite 3 (on mobile devices) can connect to Flash Media Server 3 streams.
  • Flash Media Interactive Server (FMIS) is a new FMS configuration that has all the features of Flash Media Server 2 Professional. However, it supports distributed processes that support more connections and improved caching systems. This server supports the new C ++ framework, as well as server-side ActionScript, necessary for the development of its own solutions and multi-channel communication services.
  • Flash Media Streaming Server (FMSS) is a new offering from FMS. Inexpensive server with restrictions only for live and video-on-demand (VOD) (video on demand) applications. It cannot be used for multichannel applications, such as video chat, server playlist or stream recording to server. Flash Media Streaming Server does not support the new plug-in architecture and server-side ActionScript API.
  • Flash Media Development Server (FMDS) is a free application designed to enable you to evaluate and test ALL PRODUCT CAPABILITIES. It does not have a term of use as in regular trial products and contains all the functionality contained in Flash Media Interactive Server. However, the number of simultaneous connections is limited to 10 users.

A little about the cost. Everything is simple and good. The server has dropped significantly. For starters, you can buy Flash Media Streaming Server 3 (MSRP US $ 995). And later you can buy Flash Media Interactive Server 3 (MSRP US $ 4,500, update US $ 1995).
