Visual Studio 2015 will allow you to create software for more platforms



The next version of Visual Studio 2015 will run on more platforms than ever before.

A few days ago, the official names of the following versions of the integrated development environment and framework were announced - this is Visual Studio 2015 and .NET 2015. Microsoft‚Äôs plans to turn Visual Studio into a truly cross-platform tool became known.

In addition to existing support for Xamarin and HTML5 / Cordova applications, Visual Studio 2015 will also include support for C ++, the Clang compiler, and the LLVM infrastructure. Thus, in the long run, it will be possible to write applications for Android and iOS in Visual Studio!

Moreover, Microsoft decided to make a more productive Android emulator instead of the regular one, which is distinguished by extremely slow work. The new emulator will be included in Visual Studio 2015 and, eventually, will be launched even separately from this IDE. For example, it can be used with Eclipse.

Testers also point to Java, ANT, SQLite, and WebSocket4web support.



Other new features in Visual Studio 2015 include automatic generation of unit tests, as well as cross-platform support in Visual Studio Online. Now there it will be possible to develop iOS and OS X applications on "poppies".

A limited preview of the updated Visual Studio Online will roll out within a few weeks, and a preliminary version of Visual Studio 2015 can be downloaded now .
