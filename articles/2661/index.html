Mail.Ru Agent for mobile phones

Mail.Ru made a novelty today - it introduced the special version of the Agent online pager intended for mobile phones. The version has the status of "beta", and the company encourages it to actively test.

It is reported that the mobile "Agent" for Java 1.0 contains all the basic functions of the "stationary" program, including the ability to search for contacts and view profiles. Free SMS sending application also provides. You can communicate with users of all versions of the Internet pager

According to the developer, the mobile version of Agent is compatible with most mobile phones that support J2ME applications (Java 2 Micro Edition). Installation on the smartphone is also possible - the program supports touch screen and stylus.
