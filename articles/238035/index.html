Apple recalls iOS 8.0.1

Apple did not have time to release the first update for iOS 8 - 8.0.1, which fixes the first vulnerabilities and bugs in various services of the new version of the mobile OS, as it had to urgently recall this update. iOS 8.0.1 contains fixes for the Safari browser, the AirDrop service, directly features of calls and third-party on-screen keyboards. However, when installing this update on a device, it may experience communication failures, as well as problems with the Touch ID functionality. A problem with Touch ID can leave a user with a locked device. Also, the user will remain without cellular communication.

IOS 8.0.1 update. We are actively investigating these reports. In the meantime we have pulled back the iOS 8.0.1 update.
