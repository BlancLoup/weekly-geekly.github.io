Then I give a brief description of the products mentioned.

Windows Web Server

Designed to work in online scenarios. It can be included in the customer's domain, which allows employees to authenticate themselves on the site and work with it. Does not require additional user licenses for employees and external authenticated users.

It can also be used as a separate web server without connecting to Active Directory. In this case, users can be authenticated using Forms Authentication, as I described in the previous article .

In Russia, as part of a promotional campaign , distributed free of charge without restrictions in terms of its commercial use.

Product page .

Windows server

If the customer already has a Windows Server - you can use it. For a site that does not need to authenticate external users and visitors to the site will be anonymous - do not need to buy anything. If the number of authenticated users is known and it is small, for example, 15 partners, you will need to purchase 15 Windows CALs. If you need an unlimited number of authenticated users, you can purchase the External Connector.

Product page .

Windows SharePoint Services

Essentially, this is the SharePoint software platform. It is included with Windows Server and is not additionally licensed.

Introduction to developing on SharePoint .

Microsoft Office SharePoint Server

Microsoft Office SharePoint Server is a commercial product that implements the tasks of a corporate portal, built on the basis of Windows SharePoint Services. Its leading position in the market gives confidence to customers and partners that the solution built will continue to grow successfully.

If you want to implement an extranet script, i.e. you need to bring some of the processes out to interact with partners, then you can purchase the required number of licenses for partners for $ 100.

If for a large organization, for some reason, you need to provide access to the portal of an unlimited number of users (authenticated and anonymous), then you can buy MOSS for Internet Sites, which removes restrictions on the number and provides the ability to use the functionality of both the standard and the advanced version portal.

Note that there is no External Connector for Microsoft Office SharePoint Server, although it is often mentioned in conversations.

Comparison of Microsoft Office SharePoint Server products .

Windows Internal Database

With Windows SharePoint Services, in the case of a default installation, the Windows Internal Database is deployed. This is a full-fledged SQL server with one limitation - it is intended only for use by the product that installed it. Those. this server cannot be joined from the outside; accordingly, it cannot be controlled. There is no limit on the size of the base. This edition of the server is free.

Wikipedia product page

SQL Express

The free version of a full-featured SQL server, which is limited to the size of the database in 4Gb. It can be installed on the Windows Web Server. If the web application is not planned to be used as a document exchange solution and, basically, web forms, lists, forums, Wiki sites, etc. will be used, it will not be possible to quickly reach the restriction in most cases.
