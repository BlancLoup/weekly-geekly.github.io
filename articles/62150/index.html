Zend Studio 7.0

Zend Studio 7.0 Early Access (EA) is a preview of new features and enhancements planned for the seventh version of the IDE based on the Eclipse Galileo. This is by no means the final option, but a very good way to familiarize yourself with the upcoming innovations. Your feedback and error messages will be taken into account by the developers in the final release.


PHP 5.3 support



Support for the latest PHP features such as namespaces and lambda functions:
  • Code Assist and PHP 5.3 syntax highlighting
  • Namespace Schemes
  • Built-in PHP executables and debugging features

Advanced Source Code Editing



Improved OOP Support and Advanced Editing
  • Redesigned indicators for fast navigation between methods.
  • Camel-Case Special Support
  • Type Hierarchy view for structured class hierarchy *
  • New features in autocompletion code
  • Built-in refactoring to quickly rename an item
  • Modular Application Development Tools
  • Turn a block of code into a function or variable using the Extract Variable / Method
  • Semantic analysis and autofix

New application development



You can start developing in minutes
  • Fast and painless installation and environment setup
  • Automatic discovery and configuration of Zend Server
  • Simple application deployment on a local server for debugging

Benefits of developing with Zend Server



Tight integration with Zend Server for super fast debugging
  • Easy debugging
  • View a list of problems detected by Zend Server from within Zend Studio
  • Fast and accurate reproduction of problems detected by Zend Server

Rapid Application Development with Zend Framework



Integration with Zend Tool and code generation for rapid application development
  • Simple creation of Zend Framework elements
  • Arbitrary project structure
  • Updated sample project on Zend Framework
  • Code generation using the Zend Tool

Increased productivity


  • Reduced memory consumption
  • Using the Eclipse Indexing and Caching Architecture
  • Removed unnecessary dependencies
