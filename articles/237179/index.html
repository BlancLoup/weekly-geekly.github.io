Input - a new font for programming

Font Bureau has developed a new Input font family, the most important of which in this case is Input Mono for us. Cyrillic is present. For personal use fonts are free.



For fonts, several styles are available - Sans, Serif and Mono, several saturation options - Thin, Light, Regular, Medium, Bold, Black, widths - normal, Condensed, Narrow and Compressed. Line spacing also changes.

You can choose the options for the outline of some characters - a, g, i, l, 0 and *. You can choose either on the download page, or by modifying the font files in the Python program in the archive.
