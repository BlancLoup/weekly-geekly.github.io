ARM announced mbed v3.0


In early October, at the ARM TechCon 2014 conference, ARM announced a new generation of mbed v3.0. Developments are underway on the new software - mbed OS and mbed Device Server.

The mbed Device Server will be able to process IoT (Internet of Things) data in the cloud through a standard REST API, similar to a web server.
mbed OS will perform the role of a full-fledged operating system for Cortex-M microcontrollers, including: C ++ framework, protection management (TLS / DTLS), power management, connection to wireless networks (Bluetooth Smart, 2G, 3G, LTE, CDMA and Wi-Fi).





Also under development on its own SDK. Developers will be given the opportunity to create projects and conduct development in the desktop version of the IDE. In addition, it is planned to create its own library manager and software components similar to the popular NPM.



The first alpha version is scheduled for December 2014, and release for October 2015.
