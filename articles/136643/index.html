IBM launched a beta version of its cloud office suite IBM Docs


Apparently, Google Docs and Office 365 will soon receive another follower and a powerful competitor. IBM Lotus Live cloud platform has received a new opportunity to create and edit documents online as part of the new IBM Docs project. The seriousness of IBM‚Äôs intentions in this field is indicated by the fact that the company intends to implement the transformation and rebranding of Lotus Live, which will receive the name IBM Smart Cloud for Social Business in the future; the latter, therefore, will be focused primarily on the corporate sector.

Like competitors, IBM Docs offers standard document creation, editing, spreadsheet, and presentation features; In addition, you can upload existing files, share them with colleagues, track changes to them, organize them into folders and add to Favorites. Naturally, in addition to the common office file formats from Microsoft, OpenDocument-format files are also supported. There are also social functions that allow discussing certain nuances that arise in the process of working on a file.

As a distinguishing feature from competitors in IBM Docs is the ability to create assignments (LotusLive Activities) and tracking their performance in a team. The bottom line is that the manager can identify certain areas for the document, the creation or editing of which can be entrusted to individual participants, and these tasks and their implementation status are displayed in a dedicated panel that allows you to monitor them and adjust the teamwork process.
