OpenID has launched the mobile authenticator CallVerifID

MyOpenID has launched an additional ability to authenticate users using the phone. The new authentication method is called CallVerifID and in the near future can replace the currently used tokens, smart cards and certificates. CallVerifID uses PhoneFactor , a recognized leader in phone security and authentication.



The basic principle of the CallVerifID service is similar to the work of any OpenID provider and can work with any OpenID providers. The security level of myOpenID is therefore high, as it always starts in SSL secure mode.


To start the service, just on myOpenID you need to select CallVerifID as the authentication provider. For authentication, any telephone (mobile or landline) is used as the second authentication factor. The user, as usual, enters the username and password of myOpenID, then, almost instantly, a callback comes in, after which you simply need to press # to check the authenticity on your phone. All done, the authentication process is complete. CallVerifID is currently available for free to myOpenID users as an additional service, and there is also an option for business clients.

Last month, OpenID also launched the ability to simplify logging into various services using the id selector , without having to remember the OpenID username.
