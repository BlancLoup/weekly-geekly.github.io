Php internals


  • [RFC] FFI - Foreign Function Interface - The concept from Dmitry Stogov is now in the form of a full-fledged proposal. The implementation will enable the use of functions and data structures from C-modules directly. As an example, you can look at the PHP binding for TensorFlow . Great opportunities, however, open up a wide scope for potential problems and vulnerabilities, so it is proposed to allow FFI by default only in CLI scripts and preloaded files.
  • [RFC] Covariant Returns and Contravariant Parameters - It is proposed to make the types more consistent, in particular, to allow ‚Äúnarrowing‚Äù the return type in the heirs, and ‚Äúexpanding‚Äù the type of parameters.
