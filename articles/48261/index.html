Habrapodarki. Santa Claus hasn't arrived

On December 19, Thematic Media wrote in their blog that, in connection with the New Year, they would gladly present a six-month Startup subscription plan to young Internet companies.
Further, in the comments, Denis Kryuchkov clarified the requirements (in short and roughly speaking - you need to have a public beta of the current project at an early stage).

Here comes the New Year, it is strange that there is no news about the start-ups that have received a subscription. (If there are any, please write in comments, I wonder :))

The fact is that I sent an application to TM for my startup watermark.org.ua , on the 29th I wrote about the project itself (I was still waiting for the update of name-servers so that the project would work on a new domain), I received no answer.
On the 30th I wrote to TM again, gave a link to my web service, which then still worked on another domain (datahider.org.ua), did not receive a reply again.

The web service is intended for recording invisible information inside images and reading it from there. Packing information into an image takes place according to the principles of steganography, the color of each pixel varies slightly according to a certain algorithm (so that it is not noticeable to the human eye), as a result, each pixel carries additional information bits that can be obtained using the algorithm of reverse packing information into an image. The simplest example is the even / odd color value of a pixel. If the number is even - the current bit of the hidden information is 1, otherwise 0. While the web service is very raw. It is planned to create a plug-in for photoshop, save images to lossy formats (jpg, gif), now - only png, the ability to hide inside images not only text, but also other images / files of arbitrary format, the use of algorithms that are resistant to modification of the carrier image (now, if you cut a piece of an image of sufficient size, where the information is hidden and save it in graphical format without loss of quality - it can be restored by the web service).
