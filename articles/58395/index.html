Double Commander release 0.4.0 beta released

About the program:


Double Commander is a cross-platform open source file manager, with two panels. The goal of this project is to create a file manager of Total Commander with a similar functionality and compatible with its plugins.

Here are some of the main characteristics of Double Commander:
Unicode support
All operations are performed in the background.
Group rename tool
Tab Support
Customizable speakers
Built-in text editor (F4) with syntax highlighting
Integrated file viewer (F3) for viewing files in hex, binary or text format
Working with archives is the same as with subdirectories. You can easily copy files to and from the archive. The following archive types are supported: ZIP, TAR GZ, TGZ, as well as BZ2, RPM, CPIO, DEB, RAR.
Advanced file search, including text search in any files.
Customizable toolbar with buttons to launch external programs or internal menu commands.
Total Commander WCX, WDX and WLX plugins support
Logging file operations
