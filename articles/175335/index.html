IntelliJ IDEA 12.1 with JavaFX 2 support

Hi, Habr!

Traditionally every spring we celebrate with the release of a fresh update for IntelliJ IDEA with fixes and usability improvements. This year is no exception! We are pleased to announce that IntelliJ IDEA 12.1 is finally available for download .



As you know, at the end of last year, IntelliJ IDEA acquired Java 8 support, Play Framework 2, the UI designer for Android, more than twice the accelerated compiler, new tools for working with databases and, of course, the dark color theme of Darcula. In IntelliJ IDEA 12.1, most of these features have been improved, taking into account the comments we received during this time, and many of the problems found have been fixed.


Javafx 2
In IntellIJ IDEA 12.1, there is finally more than extended support for JavaFX 2, the new Rich Client Platform for Java. Support includes support for editing FXML markup, custom CSS files, autocompletion, search and navigation, refactoring, packaging tools and integration with SceneBuilder.

Darcula and Retina
I must say that we were very surprised to receive so many positive reviews about the new dark color scheme of Darcula. Many users consider Darcula the most important innovation of IntelliJ IDEA 12. Considering this, we tried to modify the dark scheme to make it even better.

Other innovations in IntelliJ IDEA 12.1:

  • Fullscreen mode for Windows
  • Support Spring Frameworks 3.2 and Play Framework 2.1 (Ultimate)
  • Groovy 2.1 support (I include new anotations, compilations, etc.)
  • Improved Scala support (support for new compiler )
  • Improved integration with Gradle (support for gradle wrapper, task, etc.)
  • Adobe Gaming SDK (Ultimate) support
  • CoffeeScript and TypeScript debugger via Source Maps (Ultimate)
  • Improved Sass support (custom function definition, autocompletion, rename refactorings, nested properties, and more ...) (Ultimate)


To learn about all the improvements and download IntelliJ IDEA, visit our official website .
