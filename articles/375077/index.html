MeArm: do-it-yourself robot manipulator

MeArm is a pocket version of an industrial manipulator. This is an opsource project - all design data and program files are available for free download under a Creative Commons license, so that you can edit and learn from them.



MeArm is designed to make robotics even more accessible. All development, which collect money using Kickstarter , will be placed on the network under a Creative Commons license, which allows you to use, learn from their example, edit them and create your own devices.

At the moment, crowdfunding brought 13,649 pounds out of the required 5,000.

Manipulator MeArm in the process.



Aruino-compatible MeBrain controller. After connecting the controller to the MeArm manipulator, you can control it, as well as record movements.
