Windows 7 XPM Preview (XP Mode)



I decided to keep a heading of articles devoted to the review of opportunities, as well as their use in the new OS from Melkomyagkih, Windows 7 .

Today I want to draw your attention to the technology / application that will be implemented in the new system. According to small-scale ones, it will be announced with the official implementation of Windows 7 RC 7100, (May 5).

So, just over a month ago, Microsoft announced its new technology, which will be announced with the release of Windows 7 RC. For two days now there have been reports on this technology. It is called Windows 7 XPM . This abbreviation stands for Windows 7 XP Mode . (previously: Virtual Windows XP or Virtual XP, VXP).

XPM is built within the next generation Microsoft product line, Virtual PC 7 , which requires processor-dependent virtualization support (Intel, AMD) to exist and act on the victim computer, as well as Hyper-V, server virtualization platform from Microsoft. But with all this, XPM is not Hyper-V for the client, but a solution for the local PC

XP Mode consists of the Virtual ‚ÄìPC virtual environment, and a fully licensed copy of Windows XP SP3. Will be available, for free, for owners of the following systems: Windows 7 Professional, Enterprise, and Ultimate.

The XPM package will not be attached to the base system upon purchase, but will be considered as a third-party update. And it will be available for download from Microsoft.

XP Mode works almost as well as Virtual ‚ÄìPC, but naturally with differences. Let's look at these differences.
  1. You do not need to launch a separate XPM application window (that is, it works behind the scenes)
  2. Once installed, the program in XP Mode will be immediately imported into the local system (Windows 7). The icon of this application will be placed in the Start Menu.
  3. XMP can be controlled via Active Directory and Group Policies


Ie, with this technology, the user of Windows 7 will be able to run applications of outdated, but not forgotten OS and Windows XP. Such applications as IE 6, Office 2003.

Fig .: Office 2003 running in XP Mode, Office 2007 in Windows 7


If Microsoft used to say that Windows 7 has full compatibility with Windows Vista, now they can declare that they have almost full compatibility with Windows XP, or almost 100% compatibility with all Windows applications currently running.
