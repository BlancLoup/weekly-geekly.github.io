Android 3.2 platform introduced and SDK tools updated

The Android development team unveiled the Android 3.2 platform, an additional release that adds several new features for users and developers. The new platform includes API changes, and the API level is upgraded to 13.

Some of the highlights of Android 3.2:
  • Optimized for more tablets.
  • Fixed-sized applications can be scaled to display on larger devices.
  • On devices that support a removable SD card, users can download multimedia files directly from the SD card to applications that use them.
  • Expanded screen support API.
