Change phone and e-mail in the data of the certificate

Continuing a series of useful tips. In the information below, there is nothing new and sensational, but I will again, if someone post will be useful.

The phone and e-mail specified in the certificate are not always needed, but sometimes without them in any way: for example, when you receive the activation code and the unlock code IP, confirmation of transactions via SMS ...
How to change the data specified in the certificate?

No problem ... if you have not lost access to the keeper. Then everything is simple - log in to the certificate control panel and change it (click on the "change" link next to the phone number or email address.


There is a nuance - in the process of changing the phone, the verification code comes twice: to the new phone number, and then to the old one. But what if the old number is already unavailable?
If at registration a control question-answer was indicated , then instead of the code, you can enter this very answer to the control question.

If at registration the control question-answer was not indicated , it is possible to restore the old phone number from the mobile operator. To do this, it is enough to call in with the passport to the nearest office of the operator and spend five to ten minutes there.
You can not restore the number? We'll have to write an application for changing the number, notarize it and send it to Moscow. It is, of course, longer, but also real.

E-mail is changing in the same way. If there is no access to the old email , but during the registration a control question-answer was indicated, enter it instead of the code.
If the security question was not specified , you can get the code by SMS or request the call of the autoinformer.
Another option is to restore access to the old email.
Or - get a verification code in the mail by clicking the appropriate link. Attention - the letter will be sent to the address specified in your certificate.
Did not work out? then without options, you need to write a statement about changing the number, notarize it and send it to Moscow.
As you can see, no mysticism, no tyranny - everything is clear and, in general, feasible.
