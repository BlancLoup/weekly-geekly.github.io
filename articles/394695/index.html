Photoshop CC learned how to fix beautifully littered horizon



Adobe has announced a feature that many amateur photographers have been waiting for. At last they can, with the touch of a button, ‚Äúmake beautifully‚Äù, that is, correct the littered horizon. Adobe says this is the most requested feature that users have asked to add many times.

We are talking about the new tool Content-Aware Crop - turning photos with automatic filling of voids with the surrounding background. The new tool will appear with the next update in Photoshop CC and is a direct development of the well-known Content-Aware Fill feature (content-based fill).

Usually when you rotate photos, you have to trim the edges, so the picture is reduced in size. Now she miraculously keeps the same size.



The demo video shows how the Content-Aware Crop tool works. The program analyzes the pixels in the areas near the corners of the image - and fills the empty space with the background, which imperceptibly merges with the surrounding pixels.

The function is great not only to align the horizon, but also to change the aspect ratio in the photo: it adds content to the side, top or bottom.

In June, a major upgrade of Photoshop CC is planned, and Content-Aware Crop is not the only new feature that Adobe has prepared.
