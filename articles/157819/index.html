Ratchet - rapid prototyping of mobile applications in HTML, CSS and JS



Ratchet is a new framework that allows you to quickly create prototypes of applications for iOS. In fact, is an analogue of Twitter Bootstrap for mobile devices.

Project Website | Ratchet on GitHub | Discussions in Google Groups | Twitter

The framework has components of most interface elements of iOS applications: buttons, switches, lists, forms, sliders, etc.



Included in the distribution of Push.js combines all the pages of the prototype using AJAX, which allows you to make the prototype very similar to a real iOS application.
