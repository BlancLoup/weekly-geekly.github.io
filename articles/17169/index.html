Google Chart API

A few days ago, Google released a new service - Google Chart API .

Service like all ingenious is simple: now all that is needed for plotting is to form a URL.

To date, several types of diagrams are supported:


All necessary (for the first time) options are supported:
  • headline
  • legend
  • color set
  • fill color charts, backgrounds and surfaces
  • different types of axes
  • grid
  • vertical and horizontal markers
  • different types of lines
  • portable labels for pie charts

A great way to download your own server;)
