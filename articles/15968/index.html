Fedora 8

On November 8th at 6 pm MSK, as it was planned, the 8th release of one of the leading Linux distributions Fedora was released .

Among the features of this release, I would like to mention the inclusion of a completely free JDK based on OpenJDK and IcedTea, the installation of a new sound server (Pulse Audio) compatible with ESD and a new configuration utility firewall.

From system-wide packages, attention should be paid to glibc 2.7 with support for password hashing using the SHA256 and SHA512 algorithms.

Main packages:
Kernel: 2.6.23.1
X.org server: 1.3
GNOME: 2.20.1
KDE: 3.5.8
OpenOffice.org: 2.3.0
GIMP: 2.4.0

You can download a new Fedor from one of the mirrors, one recommended way is to download using BitTorrent in order not to overload the project mirrors. All links (including .torrent) are available from the official download page .
