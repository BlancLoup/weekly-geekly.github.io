Opera Extension: Radio

Half a year ago, when the extensions for the Opera browser appeared, I thought to make an extension that would allow you to listen to the radio while surfing the Internet.



But at that time there were more important tasks and my idea was postponed for a while.
And now, rummaging through the Opera extensions catalog in the hope of finding my idea implemented by another developer - was very surprised that no one had yet done a very simple, as it turned out, extension.

I present to you the expansion of Radio : there is no advertising, there is only one button that includes on and off radio.

You can select the radio in the Extension Settings:



You can install the extension from here: https://addons.opera.com/addons/extensions/details/radio-2/

PS
I think the extension will be useful for anyone who likes to listen to music.
Waiting for proposals to improve the expansion, adding new radio stations.

UPD
The audio stream plays a flash drive that is built into the extension.
Audio stream format: audio / mpeg
