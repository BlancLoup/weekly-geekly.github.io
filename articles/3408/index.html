NSIS 2.24

Nullsoft Scriptable Install System, as the name of one of the best systems for creating installation files under Windows is deciphered. Initially, the development of NSIS involved the company Nullsoft , which used the application to create the installation files of its own music player Winamp . Later, the NSIS source code was completely open and ported as a project to Sourceforge . The list of programs for creating installation files for which NSIS is impressive is enough to mention that it contains Mozilla Firefox.

Download NSIS 2.24: nsis-2.24-setup.exe , 1.41Mb

The new version resolves problems with rebooting when reinstalling / deleting libraries, reduces the likelihood of error messages in quiet mode, makes it easier for users to use the program without administrative rights, fixes minor errors ( full list ).
