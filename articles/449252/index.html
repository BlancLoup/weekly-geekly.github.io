The second zombie project in my analysis today is the Latvian startup ‚ÄúRoamer‚Äù (roamerapp.com/ru).


04/21/2019 on the server in Germany an open MongoDB database of the Roamer mobile application was discovered.



The base, with a size of 207 MB, is publicly available from 11/24/2018 (according to Shodan)!


By all external signs (not working technical support email address, broken links to the Google Play store, copyright on the 2016 website, etc.) - the application has long been abandoned.



At one time, almost all thematic media wrote about this startup:


  • VC: ‚Äú Latvian startup Roamer is a roaming killer ‚Äù

The ‚Äúkiller‚Äù seems to have killed itself, but even the dead continue to disclose the personal data of their users ...


Judging by the analysis of information in the database, many users continue to use this mobile application. In a few hours of observation, 94 new entries appeared. And for the period from 03/27/2019 to 04/04/2019, 66 new users registered in the application.


In open access there are logs (more than 100 thousand records) of the application with such information as:


  • user phone
  • call history access tokens (available on links like: api3.roamerapp.com/call/history/1553XXXXXX )
  • call history (numbers, incoming or outgoing call, call cost, duration, call time)
  • mobile operator
  • User IPs
  • user's phone model and mobile OS version on it (for example, iPhone 7 12.1.4 )
  • user email address
  • user account balance and currency
  • user's country
  • current location (country) of user
  • promotional codes
  • and much more.
