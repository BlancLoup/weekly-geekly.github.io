Logitech has released a keyboard for multiple devices.



Logitech has announced the release of the Logitech Bluetooth Multi-Device Keyboard K480 Bluetooth Keyboard, which can simultaneously work with multiple devices ‚Äî you can simultaneously connect three devices (a computer, a tablet, and a smartphone) to it. To switch between devices using a special wheel on the keyboard.

The company has already released a description in Russian:



The new keyboard allows you to type text on devices with different operating systems. The keyboard can work on two AAA-batteries, which should be enough for at least two years of active use. Made a new device in two versions - black and white. Sales should start in September 2014. In Europe and the US, the K480 keyboard will cost $ 50, the price of the device in Russia is still unknown.
