Abracadabra decoding
  • VSS - Virtual Switching System, Cisco 4500/6500/6800 switch stack technology
  • Access switches - switches to which we connect terminal equipment (for example, user PCs)
  • LAKPI aggregated channels (= LACP) - physical channels combined into one logical one; Link Aggregation Control Protocol (LACP) can be used to match which channels to merge into one logical channel.
  • Without loops - the loop implies that there are several paths between the switches (in the form of a ring) on ‚Äã‚Äãthe data link layer network; in this case, there is excess / parasitic traffic (due to duplication of packets)
  • STP ‚Äî Spanning Tree Protocol, a protocol that allows you to disable redundant connections between switches to prevent loops
  • Garda (= guards) - various technologies within STP, which allow to achieve greater stability of the network. For example, BPDU guard disables the switch port if it sees STP packets on it. Typically, BPDU guard is configured on the port where the terminal equipment is to be connected.
  • Vlans (VLAN) - Virtual LAN, virtual networks
  • Rut - route traffic between subnets
  • Spill over VTP (VTP) - in order not to manually configure the virtual networks on all switches in the network, the VLAN Trunk Protocol (VTP) may be launched, which will automatically distribute this information throughout the network
  • ACLs - Access Control Lists, access lists that define the packet transfer rules through the hardware interface
  • We raise the dynamics - the dynamic traffic routing protocol is configured
  • Summarized routes - instead of several specific routes, one common route is transmitted, which includes them
  • Flapping - the process of periodically switching routes, leading to a constant recalculation of the routing table
  • Fast network convergence - time to switch from one route to another, in case of any failures in the network
  • Port-security is a security feature for switch routing. For example, allowing to remember the MAC address of the device. If a device with a different MAC address is connected to the port, the port is disabled
  • DAI - Dynamic ARP Inspection, ARP packets are inspected to prevent IP-MAC compliance substitution.
  • Storm - transmission over the network of a large number of packets (for example, due to the presence of a loop), leading to its unstable operation
  • Confirmation of the policy of the control plan - setting up policies for handling various types of traffic (ICMP, ARP, etc.) addressed directly to the device in order to prevent a denial of service situation
  • NTP (NTP) - Network Time Protocol, a protocol that allows each device to verify its built-in clock with some dedicated device
  • Merge syslog messages ‚Äî transfer informational messages about the status of equipment to one central server
  • SNMP - Simple Network Management, a protocol that allows you to collect various kinds of information from devices
  • Water - periodically poll equipment to remove statistics from it via the SNMP protocol
  • We will set on - we will configure monitoring system for removal of statistics on SNMP plow from a certain device
