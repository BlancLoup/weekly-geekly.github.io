Microsoft helps Eclipse


At the Eclipse Summit in Germany, the summit announced that Microsoft, together with partners Tasktop and Soyatec, is releasing a number of tools and updates for Eclipse:

* Eclipse will support Windows 7 and all of its new interface features ( learn more );
* Eclipse Standard Widget Toolkit (SWT) will receive an extension for new features that are presented in Windows 7 and Windows Server 2008 R2;
* open source tools available Windows Azure Tools for Eclipse for PHP-developers;
* open source tools are available Windows Azure Tools for Eclipse for Java-developers;
* Eclipse Tools plugin for Silverlight is available.

According to a vote (THE OPEN SOURCE DEVELOPER REPORT 2009 ECLIPSE COMMUNITY SURVEY MAY 2009 pdf ), conducted among Eclipse users, more than 64% of users use Windows in their work.
