Resharper 7 released



The release version of ReSharper 7.0 was announced today.

Among the innovations:
  • Full support for Visual Studio 2012
  • New templates and tools for development under Win RT
  • New refactoring tools
  • Extended and improved code generation tools
  • INotifyPropertyChanged support (automatic substitution of property names)
  • Deleting unused references throughout the solution
  • Enhanced web development tools, including improved JavaScript support and basic support for ASP.NET 4.5 and ASP.NET MVC 4


For a detailed and complete list and review of the changes, see the JetBrains website and also in their blog:
http://www.jetbrains.com/resharper/whatsnew/?dotnetblog
http://blogs.jetbrains.com/dotnet/tag/resharper-7/
