Firefox 3.6 Beta 5 is available for download


This update contains more than a hundred bug fixes compared to the previous beta. 70% of addons have announced support for Firefox 3.6, but if your favorite addon has been updated, you should install the Add-on Compatibility Reporter , which should speed up the solution to your problem.


New features:
* HTML5 File API support
* Changed the way of integrating third-party software with Firefox to increase stability.
* Ability to run scripts asynchronously , will speed up page loading time.
* Users can now change the look and feel of their browser with one click, with built-in support for Personas .
* Firefox 3.6 will warn users about outdated plugins for increased security.
* Theora-video, can now be displayed in full screen , and also supports poster frames .
* WOFF font support.
* Improved JavaScript performance, overall browser responsiveness and launch time.
* Support for new CSS, DOM and HTML5.

Also, it became known that almost all the components of the Electrolysis project were added to the experimental development branch. Already you can download a special test build of Firefox. And in about: config, changing the value of the variable dom.ipc.plugins.enabled to true, activate Electrolysis.
