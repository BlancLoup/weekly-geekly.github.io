AdBlock Plus for Android (root is not needed)



Version 1.0 is available for download in the market (need Android 2.1 and later). The application successfully cuts ads displayed both on the web and in applications.

This should please users, but developers offering free applications with a small advertisement are unlikely.

Setting up the application is not very difficult, almost everything can be immediately configured when you first start.

There are 3 main modes of operation:

1) You have a routine device. It will be possible to filter traffic when connected to WiFi, and to the mobile network.

2) There is no rue (3.1 and higher). It means that there is no possibility to filter mobile traffic either, only WiFi.

3) There is no rue (3.0 and below). The application must be configured manually as a proxy server. If your device basically does not support the installation of a proxy, then the application can not download.

Yes, you can still download the apk itself.
