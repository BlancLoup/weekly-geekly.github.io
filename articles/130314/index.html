Google+ has instant search and support # hashtags

The Google+ team has introduced two new updates to their social network - Instant search and improved support for hashtags.

# 1 Instant Search


Have you searched on search.twitter.com? As soon as you enter a keyword and click search, it shows a list of tweets filtered by this keyword. If you stay on this page for a while, you will notice that a message about new tweets will appear on top. As soon as you click on it, you can read these new tweets and this is constantly repeated. So why is this happening? Twitter monitors all tweets for a specific keyword and shows in real time - this is called real-time search.
You thought you wouldn't see such a useful feature in Google +? Do not worry, your wishes have been heard and Google + represents an instant search today. (It turns on gradually, so if it does not work for you, you may have to wait a day or two). Once this feature is enabled on you, you can start your search, as usual, using the Google+ search box. You will remain on the search page and if new messages are added to Google+ with the same keyword, you will see a link to view new messages at the top. As soon as you click on this message, you will be shown these new messages and so on, it will be automatically updated.

# 2 Hashtag support (Hashtag turns into a link automatically)


The hashtag (#word) is an interesting and easy way to add additional metadata to any part of the content. For example, you can subscribe with the hashtag #OccupyWallStreet and talk about the history of protest on Wall Street). With this simple method, all content that contains similar hashtag can be easily combined ...
And so on, if you are writing a post or want to share some news, then add # hashtag and Google+ will automatically translate it into a link. As soon as someone reads a message, he will see your hashtag as a link and he can simply click on it and read all messages with a specific hashtag.
