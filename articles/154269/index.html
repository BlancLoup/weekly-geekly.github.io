30 years with QNX: QNX demo diskette

You just have to see it. But first of all, keep in mind, this QNX demo diskette appeared in the late 1990s, and the web browser was released back in 1999, and not at all in 2012. I think you can forgive the browser for error messages when browsing modern websites.

You will be amazed! All that is shown here is the operating system itself, the graphical window subsystem, the browser, the TCP / IP stack, the file manager, games, etc. - placed on a single boot floppy size of only 1.44 MB. No CD or hard drive. And, as you can see, the demo can even load and run programs (even graphics drivers!), All on the fly. I would say that is cool!



Did you know? The image of the demonstration diskette was downloaded more than 1,000,000 (million!) Times. It was the first truly successful marketing campaign for QNX Software Systems. The goal of the diskette was to show the developers of highly reliable and embedded systems, what kind of performance and features of QNX can fit in such limited resources.
