The hosting version of BlackBerry Enterprise Server Express is available in Russia!

Hosting version of BlackBerry Enterprise Server Express in Russia


Today we would like to talk about what BlackBerry Enterprise Server Express is , for which RIM created it in its time, and what prospects await it in Russia.
The product history of the BlackBerry Enterprise Server began a long time ago, and today we can say with confidence that the period of adolescence and youth of BES is somehow overcome, the latest release (5.0.3) demonstrates the best stability and the richest functionality in the entire history of the product.

A little excursion into the history


The first version of the BlackBerry Enterprise Server appeared back in 1999 and worked on Exchange Server 5.5. It seems that not very many modern administrators found Exchange Server 5.5, what can we say about the BlackBerry Enterprise Server , which did not appear in Russia right away.
In November 2004, a branch of the 4th version appeared, which is still used by many people and where it is used (of course, with subsequent service packs). It became the first version of the BlackBerry Enterprise Server , which the author of this text substantively acquainted with and even certified on it in 2007 :)

What do BlackBerry users especially appreciate in the BlackBerry Enterprise Server?


Send a message from your smartphone, immediately watch it in the Sent Items folder in Outlook and be sure that the message will be delivered to the recipient instantly? Of course, this is great, but, by the way, this functionality appeared in BES v1, released in 1999 :)
Since then, the ecosystem around the BlackBerry solution has changed dramatically, grown, enriched with functionality, has become much more stable and more integrated with the mail standard for Enterprise users - Microsoft Exchange Server .
Now the actual server branch is 5, the service pack that recently reached the third. Integration with Microsoft products has become even closer - there is support for Enterprise messaging services, which in general is fully consistent with the stated trend for the convergence of RIM and Microsoft .
Having conquered the Enterprise market in North America and the countries of the ‚Äúgolden billion‚Äù in February 2010, the corporation made an unexpected move for many, releasing a free version of its product, without cutting off the functionality as radically as all vendors standardize, differentiating free from paid versions.

The idea, as it became clear after the smoke of doubts and incomprehensibility of the step cleared, was very simple - to allow the BlackBerry solution to be used to a wider segment of potential users who do not need functions inherent only in the BlackBerry Enterprise Server.

What functions, if you "sort everything out on the shelves," of the BlackBerry Enterprise Server available in the full version, does Express support?


In fact, it is much easier to list what it does not support. The list of unsupported includes premium options, such as:
Enterprise Instant Messaging Clients (OCS and Lync), Enterprise Social Networking Solutions, Chalk Pushcast Software and BlackBerry Mobile Voice System, most of the IT policies (the full version supports more than 500 policies, Express - 35), native support for high availability and monitoring.
The list of BlackBerry Enterprise Server Express IT policies is described in detail in the administrative guide .
It must be said that for the vast majority of medium and small businesses, these functions are not often used, so their presence or absence is not the main criterion for choosing BlackBerry . The main criterion is the BlackBerry brand itself, carrying brand quality and brand services.
