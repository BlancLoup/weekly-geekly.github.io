Making a beautiful flash gallery in Picasa


Most recently, I needed to make a selection of several of my photos. It should have been portraits. Without thinking twice, I decided that it was necessary to realize all this in a flash, because it's simple, beautiful, and I think almost everyone has a flash player.

The first thought was to download some gallery where you can attach flickr or picas. But suddenly I came across one good site. There were laid out the templates of galleries and their demonstrations. After downloading the template I liked, to my surprise, I did not find any readme ... I had to search the site. It turned out that these templates need to be inserted into the picas program ... And then in the program interface to create a ready-made gallery.

And now everything in more detail.

1) First you need to install the very Picas. You can download it on the official website picasa.google.com


2) Next, go to www.paulvanroekel.nl/picasa There we select and download the desired template.


3) Extract the contents of the archive to \ Program Files \ Google \ Picasa \ web \ templates \ Each template must be placed in a separate folder.


4) Create a folder with ready-made images that we want to add to the gallery.
