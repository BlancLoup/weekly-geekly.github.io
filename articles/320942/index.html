GitHub introduced tags for repositories

Topics offer completely new opportunities to discover similar repositories. Topics are tags for creating more thematic connections between GitHub repositories. They allow you to view projects by type, technology and more.



Click on the topic that interests you and see the repositories associated with it. Add topics to your repository to help other users find them easier.



When adding a topic to an open repository, possible options for tags will be offered. These proposed options are the result of machine learning and natural language processing of the repository contents. We are only at the very beginning of the path and if you reject those topics that do not suit you (note lane.: Next to the tag there is a button indicating the reason why it is not suitable) , then this will help us train our model to produce more meaningful results.

Our documentation will show you how topics work today, but then it will be much more interesting. Topics will evolve as we learn from you and better understand the role of GitHub in the discovery and discovery of projects. Can't wait to see how you start using it!
