Release close ...

The calendar winter has come in the northern hemisphere of the Earth, and we hasten to rejoice with the news of all those interested in our project: the release cycle of the epic version of ReactOS 0.4 has been launched!

November 22, 2015, Amine Khaldi, with the number 70,000, created the branch for release preparation 0.4. This means that now and until the release itself, all changes that fall into the trunk (the main development branch) must be deliberately transferred to the release branch if the need arises. This technique allows you to avoid unexpected regressions and random bugs, but is not sufficient - you also need targeted testing. Therefore, we invite everyone to take part in pre-release testing of the programs in ReactOS. Installation and boot for testing should be taken here , it is proposed to use builds with the number 69999 (first of all) or newer.

If you want your program (which you created or just use) or your device starts working in 0.4, it's time to send a bug report . We also accept patches with joy, as, for example, in the history of DBMS Linter.
