Microsoft Office Groove 2007 is free and almost free

Office Groove 2007 is a desktop application for data synchronization, collaboration, and convenient file sharing between users and computers without using email and online services. Details about the program tells this demo .
The program is really wonderful, but, like most Microsoft products, is paid. However, 2 methods were immediately found that make it possible to use it completely legally with minimal cost.

From the site www.trygroove.com you can download a 60-day full-featured demo version, and during the trial period set up all the necessary workspaces with any number of users and computers. After 60 days, the Groove functionality will become limited, in this mode you will not be able to create new areas and add users. But at the same time, all existing connections will continue to work as if nothing had happened, therefore data exchange will continue in the standard mode.

Corporate customers can also take advantage of this feature and take advantage of a 99% discount when purchasing a full version. I think this is a great offer! Although the discount is for some reason not very advertised, since this is written only on the Microsoft partner site: https://partner.microsoft.com/rus/salesmarketingsection/offerspromos/40088713
