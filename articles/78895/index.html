Testing MySQL: ZFS vs UFS

I had some time ago the need to create instant backups of the mysql database. The desire existed for a long time, but somehow it still seemed to be living normally with replication and backup with a slave. But the cases are different, and the ability to take an instant snapshot from the master server's file system can make life much easier. I realized that I needed a snapshot . And where there is a snapshot , ZFS appears semi-automatically on the horizon.
