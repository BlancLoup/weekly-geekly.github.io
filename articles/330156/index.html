The release of the beta version of multi-line TAPI driver for Terminal Services and the announcement of the presentation 3CX v15.5

This week we will talk about the new multi-line TAPI driver, which is used to integrate with many CRM systems, and invite you to the online presentation of the 3CX v15.5 release.

Multiline TAPI Driver 3CX for Windows Terminal Services


Due to numerous requests from partners and customers, we have released Multi Line TAPI Driver for TAPI-compatible CRM systems installed on a Windows terminal server.

The Multi Line TAPI driver requires the latest 3CX client for Windows (to be included in the 3CX v15.5 distribution) and the 3CX Pro edition.

Installing a 3CX Multi-Line TAPI Driver


  1. Download the distribution and run the driver installation.



  2. During installation, a window will appear in which you should enter extensions and user names. In it, you can add up to 5 users (lines). The remaining lines are added after installation via the configuration file C: \ ProgramData \ 3CXMultiLineTapi \ 3CXTAPI.ini.

  3. After installing the 3CX Multiline TAPI driver, download and install the latest 3CX Windows client .

  4. Run the client. This completes the installation.
