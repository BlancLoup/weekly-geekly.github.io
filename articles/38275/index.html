Made a version for Opera, because it was not difficult =) Thanks for the hint yolk .
You can download the version for the carmo-poor or the carmo-rich .

To install, put the file (any one) in the directory in which you store UserScript for Opera.

If you do not have such a directory - create it in a secluded place, put the file there. Then go to Tools -> (or Opera-> to Mac) -> Preferences-> Advanced-> Content-> JavaScript Options ...-> User JavaScript files-> specify the directory with the script.

The main thing is that the file should be called * .user.js, otherwise it will not work.

Tested on Opera 9.52 IntelMac version.
