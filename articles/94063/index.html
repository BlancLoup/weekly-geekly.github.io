How to honestly earn big discounts

Everyone loves a freebie! That is why our next startup is dedicated exclusively to discounts on everything that you want, and reluctant to spend money on it .

In search of a suitable name for our new project in the vastness of vanet, it was decided to name Startup Coupons.

So what is this project for the end user?

The pattern of user interaction with the site is approximately as follows:

  • Make a purchase with a good discount and set aside savings for MacBook \ iPhone \ Hummer (underline the necessary);

Restrictions
