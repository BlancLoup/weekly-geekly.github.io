On LG G Watch smart watches installed Windows 7



The characteristics of many smart watches are now better than those of PCs that were considered quite good several years ago. At the same time, manufacturers of smart-watches do not stop at what has been accomplished, releasing more and more new models, with heart rate monitors, GPS, slots for SIM-cards and other "buns".

The configuration of some devices allows you to install not Android Wear, but, for example, Windows XP or Windows 95 . But the YouTube user with the nickname Hacking Jules didn‚Äôt think this was enough, and he decided to install Windows 7. In the end, everything worked out, although the "lite" version of the Seven was running. Loading it on this device takes about three hours.

After booting from the installed OS, you can interact using the touchscreen clock display. Clearly, such a work with the operating system cannot be called full-fledged, but the installation itself was done for fun.



The other day there was also a guide for installing this OS on a smart watch with Android Wear. A prerequisite for a successful installation is the presence of a device with Android Wear and activated ADB ( Android Debug Bridge ). Also requires a PC with installed smart watch ADB-drivers, files and knowledge of basic DOS and ADB commands.
