Microsoft TechDay 2015 Live Broadcast



Good morning!

We invite you to join the Microsoft TechDay 2015 webcast today at 10:00.

What is Microsoft TechDay?

We held the first such event in June 2014 ago and received a lot of positive feedback. We hope it will become a good tradition, and we invite you on June 10 to the next Microsoft TechDay - an event that is intended for the technical community and which is created by the technical community. And therefore you will find interesting and deep technological content.

Who speaks at Microsoft TechDay?

The main speakers of the event are independent experts and Microsoft MVP in various fields.

What will Microsoft TechDay talk about?

This time, the focus will be on Windows Server vNext. We will discuss the most interesting news of the next version of Windows Server, from the new features of Hyper-V to containers and Nano Server. The hottest news about the server from the recent Build and Microsoft Ignite conferences you can listen to and see, offline or online, in one day.

10:00 - 11:00 What's new in Hyper-V in Windows Server vNext, Mikhail Komarov
