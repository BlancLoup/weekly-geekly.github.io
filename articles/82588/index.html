I often began to ask about what kind of literature you can read on this subject. To help everyone at once, I decided to write this post. =)
Under the cut you can see - what literature I use in the process of their work.



Horowitz P., Hill U. The Art of Circuit Engineering



Pretty famous book. Unlike conventional textbooks, everything is written in simple language. It contains a large number of typical schemes, as well as non-operable schemes are given so that it is clear how not to do it. :) The book is universally recognized as the "bible of electronics". In general, I recommend to everyone.

Evstifeev A.V. AVR microcontrollers of Tiny and Mega families from ATMEL



In fact, this is the transfer of datasheets to the main 8-bit AVR microcontrollers. In my opinion, more convenient than reading everything from the screen. In the book, everything is detailed and translated. You can quickly find what you need for a particular MK.

Barry Wilkinson. Fundamentals of digital circuit design



This book can be used as a desktop reference for all types of logic circuits. Given combinational circuits, triggers, counters. In general, a lot of useful things. :)

Herbert Shieldt. C / C ++ Programmer's Reference



Since, when using microcontrollers, I have to deal not only with circuitry, but also with programming, I use this reference book. The author, I think, is already quite famous. =)
