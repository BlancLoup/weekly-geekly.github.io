BitTorrent is testing a new advertising system.



Despite the fact that many BitTorrent are associated with illegal file sharing, the eponymous client, in the first place, is just a file sharing tool. And who and how to use this tool is another question. Now the company is trying to show itself as a supporter of legal file sharing (which, in general, is true). To this end, BitTorrent developers are testing a new advertising system that allows all sorts of singers and musicians to make money on their works.

This project is being implemented in collaboration with Fame House (this company is engaged in marketing of digital products), and one of the clients of this company. So far, this client is the only musician for the entire program of legal file sharing with royalties. The model is as follows: works by musicians, film makers and other products of this kind are distributed free of charge.

When downloading one of these free files, the BitTorrent user sees an ad from one of the advertisers participating in the new file sharing program. Announcement offers download any software. If the user downloads such software, money is withdrawn from the advertiser's account, and part of the funds are transferred to the author of the work.

According to BitTorrent management, this project is one of many that will be tested in the future, in a few months. It may well be that one or several projects that successfully passed the testing program will be implemented in practice in the near future.
