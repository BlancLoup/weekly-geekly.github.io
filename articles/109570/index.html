Cr-48 Chrome Notebook Beta Testing Program

Google launched the Cr-48 Chrome notebook beta program today.



Characteristics of the device, as described by Google:
  • display 12.1 "
  • full size keyboard
  • clickable touchpad larger size
  • 3G support
  • Wi-Fi 802.11n
  • 8+ hours of work
  • 8+ days standby
  • Webcam
  • SSD disk
Register in the testing program and get a copy of the device, US residents can here . The number of netbooks is limited, registration of testers ends on December 21, 2010.















We add that netbooks with Chrome OS manufactured by Acer and Samsung will be available only in mid-2011, so the current beta testing program is the only way to get netbooks with Chrome OS in the next six months.

A few days ago, Engadget published a frame from the presentation of the company Inventec , which is preparing to release such netbooks (perhaps commissioned by Google). The photo shows that the standard layout is slightly modified.



The CapsLock button disappeared altogether, and the upper register mode can be enabled via the function key and Shift. Added buttons for browser navigation and full-screen mode. There is no Delete button.

There is another exclusive snapshot of the not yet officially presented Acer netbook with Chrome OS.
