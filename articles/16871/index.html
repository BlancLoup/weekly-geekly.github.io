Vixy.net Converter App

One of the most convenient services for converting on the fly video from YouTube Vixy.net got an application for PC. Saving video is available in the following formats:
  • Avi for Windows (DivX + MP3)
  • MOV for Mac (MPEG4 + MP3)
  • MP4 for iPod / PSP (MPEG4 + AAC)
  • 3GP for Mobile (MPEG4 + AAC)
  • MP3 (audio only)

The beta version (1.7 MB) has the same functionality as the online service, but it saves time and allows you to avoid Vixy overloads, which recently prevented you from using the service. The beta will work until the end of 2007, after which the shareware release without ads will be released.
