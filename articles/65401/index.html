New site using Google services

Yesterday we opened a new site in public access - Parket Premium ( http://www.parketpremium.ru ). The site was entirely made using Google Apps, except that in addition to automatic domain registration (zone. Ru) is not supported.
Conventional tools for editing and various formatting are everywhere. Here it is worth paying attention to many pleasant trifles - all of what may be needed for the website of a small company is available, namely:
  1. Feedback forms and questionnaires (it is also possible in a few minutes to create any other form with automatic saving of the results in spreadsheets). This is done based on google docs for apps. I also liked the possibility of elementary embedding of the form in the body of the page in any place.
  2. Large selection of standard themes for a site with very large background settings / sizes / fonts.
  3. Integration with Google Analytics and Webmasters. There is no possibility of automatic integration with Yandex.Webmaster, so you had to confirm it with the help of the file upload (Site Manager - Applications - Add) yandex_xxxxxxxxxxxxxx.txt to the root of the site. Please note that an empty file does not load. But unfortunately, Yandex doesn‚Äôt react to the file with the redirect and does not confirm the site.
  4. Automatic generation of the site tree and the ability to customize the menu.
  5. Convenient layouts for organizing the structure of each individual page (1,2,3 columns with / without header / footer)


Total: fast, powerful tool for creating websites. There are problems and pitfalls - but almost all of them are currently solved.

Do not judge the site strictly - this is its first version. Constructive criticism is welcome.
