2 projects are looking for friends

There are two ideas of startups, constructive criticism is needed. Projects are offered as part of the motiVenture 2009 competition.

Yet Another App Finder
Posted by: vakkaeo

The idea is to create a Windows application for the selection of software. A well-known analogue is the Linux App Finder. The selection of software is carried out on the basis of the user's answers in a kind of questionnaire in which you can specify a set of necessary functions.

Features:

- Rating system
- Direct download links
- Easy user-friendly interface
- Russian and English versions
- Ability to search not only software, but also web-apps

One more PM
Posted by: Dmarck

There are a lot of project management systems, but everyone has some flaws and an idea to write a simple but functional online project-manager with notification via the instant messenger and the cross-platform client.

Both projects need implementation ideas and volunteers. Contact is better with the authors directly.
