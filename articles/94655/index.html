External links in Google Wave without an intermediate Redirect Notice page

Intermediate page when opening sites - evil in its purest form.
I immediately began to look for a solution to the problem through UserScript, and was surprised to find that there is no ready-made script yet. The thought crept in that there might be another solution ...

But while this thought is maturing, I just made a script (just two), just in case:

Google Wave External Links URL Fixer and Google Wave Auto-Redirect
(I had to use an abbreviation, because Habr does not tolerate the word script in the URL)

The second is needed for a guarantee, since the first is based on a timer, and in order not to process all the links each time, the condition was added: ‚Äúif the number of links has changed since the last pass. Because of this optimization, it is rare, but there is a situation of switching between waves with the same number of links, and accordingly the wave is not processed. In such a situation, the second script does an automatic redirect right after opening the intermediate page.
