Igelle DSV 1.0.0



Most recently - a few days ago - version 1.0.0 of the Linux Igelle distribution was released. This is a distribution kit in the full sense of the word, and not some colored Ubuntu. Here you can find your own Esther window manager, Sjapp package management system, and even a browser - not FireFox.


Igelle comes as a live-cd with the ability to install on a hard disk. It loads very quickly, which surprised me. And it does not require any special knowledge and skills - it just loads, that's all. I decided to install it on a hard disk.


He asks where to install.


After that, it formats partitions and copies to them. Everything happens surprisingly fast.




After the reboot, Igelle asks you to specify the hostname, the time zone and create a user account. Another reboot, and everything, you can work.


Quite accidentally, a panel with quick launch icons appeared leaving the bottom of the screen: Home folder, application list, system settings, documentation, some Task Manager, video player, eerie-looking Osmo organizer, calculator, image viewer, web browser and gEdit.


Task Manager is a cross between a list of processes of the same name from Windows and top. You can change the status and priority of processes, you can kill.


Web browser of own production. It is called everywhere just WebBrowser 2.28.1, compiled on the basis of Apple WebKit, and I was able to recognize Epiphany in it.


Igelle's own repository is pretty poor. But there is everything you need. OpenOffice, FireFox, ThunderBird, PHP, RapidSVN, Pidgin, and a whole lot more.

In addition, there is a tool called Sjapp. It can perform the role of both a package manager (sjapp install pre-downloaded-package.sjapp) and a tool for compiling any of the source codes, and in a rather tricky way.

The sjappc compiler takes as input a pre-prepared SysJob Script file (* .sjs), which, in the simplest case, looks like this: 
  geany: sjapp {
       setself "url" "http://download.geany.org/geany-0.18.tar.bz2"
 }
I took an example from the Sjapp documentation. I.e
sjappc geany.sjs
And you have the geany editor installed. Well, must be installed.

The distribution is fast and lightweight, containing everything you need. Most is for a laptop. Released with kernel 2.6.32.
