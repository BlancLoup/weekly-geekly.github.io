~ / Templates folder and right-click context menu


There is such a possibility: in the menu of the right button in the nautilus create a new document. But not many people know that instead of a simple empty document, you can create pre-designed, written, in general, something from the templates. So if you need to constantly use the same thing, be it a Word document, HTML, PHP, XML and also God knows what, welcome :)
