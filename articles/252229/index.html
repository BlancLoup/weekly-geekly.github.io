Vladimir Ivanov, Oracle - Deep immersion in invokedynamic

This announcement is dedicated to those who have been crying all winter, that there has been no hardcore on JUG.ru for a long time.

On Wednesday, March 11, a meeting of JUG.ru with Vladimir Ivanov , an engineer of the HotSpot JVM team, will take place at the Petersburg office of Oracle.

The topic of the meeting is the invokedynamic instruction and its implementation in JDK / JVM.



In Java 7, a new instruction appeared at the bytecode level - invokedynamic. The programmable binding instruction allowed dynamic languages ‚Äã‚Äãon the Java platform to reach new heights of performance. JRuby has been actively using invokedynamic since Java 7, and Java 8 has a JavaScript runtime based entirely on the new API.

The focus of the report will be on the internal implementation of the java.lang.invoke API and the support of the new instruction in the JVM. Particular attention will be paid to the interaction of JVM and Java-level implementation.

Participation is free, registration HERE .
