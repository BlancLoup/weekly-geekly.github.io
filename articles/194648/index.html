The traffic compression feature has been added to Google Chrome for iOS

Few know that Google has a service similar to Opera Turbo: a proxy server that compresses the data sent to the browser in order to reduce potentially expensive mobile traffic. Support for this feature in Chrome for Android appeared a few months ago, and now the time has come for Chrome for iOS. Google claims that SSL traffic and traffic in incognito mode is transmitted directly, without compression.

A new service allows you to open sites that are blocked by Roskomnadzor, which is especially useful on iOS, where installing programs to bypass locks can be difficult.
