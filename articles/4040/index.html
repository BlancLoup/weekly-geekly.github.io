Ten things that are only possible in Opera

Everyone knows that Opera is a sophisticated and functional browser, in which new, very convenient features constantly appear. For example, about ten years ago, tabs (tabs) first appeared there, now becoming the de facto standard for all browsers. And there are many such examples. Here is a list of some unique features of the Opera browser.

Some of these functions can be repeated using extensions in Firefox or in other browsers, but it was in Opera that they appeared for the first time.

1. Duplication of tobacco. You came to a web page, and you want to read it later, but now you need to return to the previous page. What can I do in another browser? Copy the URL, open a new tab, download this page there. In the Opera browser, everything is much simpler: just press the "Duplicate" button. An exact copy of the tab opens along with its entire history (previous pages, etc.).

2. Go to the address (Go to URL). You see the URL in the text of the page, but it is not highlighted as a hyperlink. You want to copy it and put it in the address bar, but in Opera browser you can simply select the text and select the ‚ÄúGo to URL‚Äù command by the right button.

3. Reboot every 30 seconds. A very useful feature for monitoring your favorite web pages. Eliminates the need to constantly press F5.

4. Fit to the width of the window. If the webmaster decided to make a wide web page with a horizontal scroll bar, then you will not be annoyed by the fact that Opera has a fit function that fits the width of the window.

5. Backward scrolling (Rewind). This feature allows you to quickly go back in the history of surfing, without examining the list of visited pages.

6. Names for collections of sites. Favorite links in the "Favorites" can be combined into collections, give them short names (for example, news), and then open them with a command from the address bar. The word "news", pressing Enter - and each site opens on its tab.

7. Intellectual arrangement and closing of tabs, taking into account the order in which they opened. For example, in Firefox you work with tab A, open from it tab B, which is located last in the list of tabs. If you close tab B, then the penultimate tab C will remain on the screen, which has nothing to do with tab A. The Opera browser is more intelligent.

8. Instant return. The transition to the previous page is instantaneous - it is loaded from the cache.

9. Scaling pages. A unique feature that applies to the entire page at once (pictures, text, formatting), keeping the ratio of individual elements.

10. Recovery after a failure. When you later download Opera, it will restore all the tabs that were on the screen just before the crash (the corresponding plug-in for Firefox is called Crash Recovery ).

Of course, this is not a complete list. The Opera browser is also able to obey voice commands , mouse feints (for example, holding the right mouse button and clicking the left button, you can give a command to load the previous page), it allows you to change hot keys for various commands, and also can create real presentations without using Microsoft PowerPoint.
