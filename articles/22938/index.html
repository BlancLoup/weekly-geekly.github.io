Pwnage for iPhone and iPod Touch

Dev Team - a team of experienced iPhone hackers that has proposed many methods of jailbreaking - has released a utility called Pwnage.

This program is a panacea for hacking Apple's phone. Using it, you can install absolutely any firmware on your phone, which makes the hardware architecture of the iPhone and iPod Touch truly open. This means that the opportunity to run on the phone and modified versions of both software and operating system. Of course, in this way, you can also run code that initially did not allow Apple to run. For example, there is a chance to install Flash-enabled Safari on your phone. Theoretically, with this program on the iPhone, you can even put Linux.

Unlike the usual jailbreak, which gives the right to write applications to the phone, Pwnage changes the file system, allowing you to perform feats that its installed version is not capable of. This is accomplished by circumventing the in-built code certification verification. When changing the firmware, the iPhone checks if the Apple firmware is signed. Pwnage causes the iPhone to ignore such a check when loading a new firmware.

Of the bad, the creators of the program do not guarantee that the hacking will take place 100% successfully, but they promise that the worst thing that can happen is the removal of software (which can be restored by standard means).

At the moment, there is only a version of the program for OS X 10.4 and higher. Nevertheless, the Dev Team is working on a version for Windows and promising its speedy release. The program itself works for versions of the original firmware 1.1.4 or 1.2 beta.
