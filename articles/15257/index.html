A couple of tweaks for sidebar

Personally, I do not use the sidebar, but I know that many simply cannot live without it.
Undoubtedly, this is a very convenient thing, but the pursuit of perfection is an endless thing, so I suggest you try out the following tricks to remove a couple of restrictions.

In order to use them, you will need to edit the userChrome.css file (which is usually located in the chrome folder in the profile folder; if you do not have it there, then go to the chrome folder and edit userChrome-example.css, renaming it to userChrome .css ), adding the following lines to its end:
  • To move the sidebar from the left to the right:
    #browser { -moz-box-direction: reverse;}
  • To increase the sidebar width:
    #sidebar {max-width: none !important; }

Restart Firefox!
