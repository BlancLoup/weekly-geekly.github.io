Wolfram Alpha can determine which type of aircraft is flying over you.



And not only. This service will also be able to tell which company the aircraft belongs to, and where exactly it was headed. In order to find out all this, you only need to know your coordinates, and fill in the search box with a query like ‚Äúflights overhead‚Äù. This can be done both in the desktop browser and in the mobile device browser (if geolocation is enabled, then everything is just perfect). The search results are surprising, they can be seen in the screenshot below.

As already mentioned, you get a complete list of aircraft that are currently in close proximity to you (also indicated aircraft with a slight deviation from the specified coordinates). A map of the sky is also shown, indicating the location of the aircraft next to you.



For complete information on a specific flight, you can click on the plane you are interested in. This displays the full information on the flight, including the duration of the flight, a complete flight map, time spent in the air and so on. In the future, the developers will include such a function as eavesdropping conversations of passengers of the flight and watching them.

Yes, there is one thing - on the desktop browsers the sky map is displayed (the function is called Sky Map), but for some reason in mobile browsers it is not. But with the help of the item "Angle" the problem is solved in the mobile browser. Interestingly, the Wolfram Alpha application is available for both iPhone and Android phones. But using the Wolfram Alpha application on mobile devices to track flights does not make sense, it works for itself and just request in a mobile browser.
