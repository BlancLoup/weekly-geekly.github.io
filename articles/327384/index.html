Second webinar on SAP Cloud Platform: a detailed overview of services

Updated: webinar video available.

  • how to tie together different software applications at the process and data level - create a single business process consisting of business services using the Integration Service
  • how to "bring" and use your own ready-made program code to run on the SCP platform (using Cloud Foundry)
  • how to turn your software service into a marketable product using API Management
  • gamification service that allows you to turn a boring business application into an exciting adventure
  • Webide
