Wiki-OS in Chrome


Album: Wiki-Os

I was impressed with recent information about Wiki-OS and wanted to test it.
Upset that does not run in Chrome.

In IE and FF, everything is fine. I found recommendations on the forum how to stitch Wiki-OS to Chrome (en) - here the guy writes how to adapt Chrome (just download the archive and unpack it in C: \ Documents and Settings \ Username \ Local Settings \ Application Data \ Google \ Chrome \ Application directories for XP).

For Vista, I managed to unpack it in C: \ Users \ Valery \ AppData \ Local \ Google \ Chrome \ Application \

Important!
The archive has one problem - the plds4.dll file is missing (a message about this appears in Chrome when loading the initial page www.wiki-os.org ).

I found it in C: \ Program Files \ Mozilla Firefox and just copied next to the other libraries in C: \ Users \ Valery \ AppData \ Local \ Google \ Chrome \ Application \
Everything worked right away.

Comparative feelings - Chrome works the fastest, and the full screen (F11) gives a complete perception of the environment as a mixture of simplified VS C # and VBA interfaces.
In general, the impressions are good. Probably, this is one of the best MS solutions after years of treading around WinAPI.

I hope that interaction with other clients (Linux, MacOS) is possible in the future.

UPD: Thanks beloks - here is the missing file: www.abelok.hut.ru/plds4.dll.zip
