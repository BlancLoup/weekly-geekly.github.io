Firefox introduces shared account and updates sync system.


Few people know, but for several weeks now, Mozilla has updated the synchronization function of the browser. It was decided to remove the inconvenient secret synchronization codes and passwords to recover lost keys.

Firefox Sync is a service that allows users to transfer all their history, passwords, bookmarks, open tabs, and even add-ons between different devices that have Firefox installed for both the desktop version and Android.

To date, the new sync works only in the Fx version of Aurora and above. It is also possible to use only one synchronization option. Therefore, to create a synchronization profile, you must first disable the attached devices.

For those who prefer the old version of synchronization, it will continue to work, but it will not be possible to connect new devices.

The introduction of a single account is connected with the desire of Mozilla to create a single ecosystem between Firefox for Desktop, Firefox for Android, Firefox OS, Where's My Fox , and the application store Firefox Marketplace .
