Unity 4.3



The long-awaited release of Unity 4.3 took place , in which new tools for working with 2D were added, deeper integration of Mecanim was implemented, and many improvements were included in the Unity IDE.

Innovations


Another dimension
New features for working with 2D:
  • Automatic sprite connection
  • 2D animation window
  • Easy 2D and 3D combination
  • Automatic generation of animation from sprites
  • Fully integrated physical 2D system with rigidbodies, joints and customizable colliders

Mecanim
Mecanim got its own powerful animation tool.
Also, optimization was made in the work of Mecanim.

Improved scripting
Added latest version of MonoDevelop (4.0.1).



New features for Windows applications

In this release, Unity made happy developers for Windows Phone 8 and the Windows Store.
Added support for the trial API, which allows you to make applications based on try-before-you-purchase (try-before-buy).

Version control system

VCS Perforce was integrated into Unity 4.2. Now added Plastic SCM. Like Perforce, Plastic SCM works well with large binary files.

And much more...
