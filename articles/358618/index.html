DevConf 2018 is already on Friday, May 18

This Friday, May 18, will be held a professional conference of web developers DevConf 2018. We invite you to participate. Very briefly about the program:

DevConf :: BackEnd : Aggregative thinking, rewriting a project from Drupal 8 to Yii 2, Yii 2.1, PHP and frameworks, Service Oriented Architecture, sending notifications, sample profiling, open tracing, Business Intelligence.

DevConf :: FrontEnd : HOC in React, WebRTC: video calls from the browser, Headless Chrome, architecture of a large project legacy, JSON API, Framework for custom design, Test it, Node.js and a free LuaInMoscow meeting.

DevConf :: Storage : MySQL / MariaDB / Percona Server: what's new, logical replication in PostgreSQL, SphinxSearch, ClickHouse, ProxySQL for MySQL fault tolerance, overview report on various data compression mechanisms from Andrey Aksenov.

DevConf :: Mobile : Localization and internationalization of a mobile application, Mobile Application Security Verification Standard, Features of developing enterprise applications, WebRTC in mobile applications, developing cross-platform applications using Qt5.

DevConf :: DevOps & Management : Deploy systems in VM, Nomad and Kubernetes, Static code analysis (PVS-Studio), Building a service mesh, Getting started on the project testing, pay in IT, from Scrum to Kanban, protecting intellectual property in an IT company, tools of the head of the development team and the difference between professional programmers and amateurs, OpenStack mitap.

For Habr's readers registration is discounted .
