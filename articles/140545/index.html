What to expect from WordPress 3.4

A couple of weeks later, the beta WordPress 3.4 is released, and in the middle of April its official release is expected, which will receive some interesting innovations. I propose to consider some of them.



Theme Customizer





One of the most interesting pieces in the new WP is Theme Customizer. It will allow a new approach to the design of templates, especially for default ones. The customization mode is quite easy to use and in real time allows you to add an image to the header of the site, change the background and link colors, menus, title, slogan, and other components. And the best part is that all these changes can be observed visually, without leaving the admin panel.

Multisite enhancements


Probably, many appreciated the multisite mode, which appeared in the third version of WordPress. The new release will improve the work with subdirectories. In addition, WordPress seeks to improve search by users and sites by adding auto-completion. Search users on a specific site will also be available. So we expect both these and other, more minor improvements.

Support for HTML tags in image headers




One of the most requested properties that many have been waiting for is the ability to add HTML tags to the headers directly in the admin fields. Now you do not need to make extra gestures, correcting templates to add a design element or link.

More XML RPC


XML-RPC is the thing that allows apps like Android for WordPress to work with your site. This protocol allows third-party applications to post content, manage comments, etc. not logging in admin. WordPress panel. New methods allow you to manage a custom post type, taxonomy and custom blogs on a multisite. Perhaps these improvements will add additional functionality to your favorite Weblog client.

Improved international support


There will also be implemented several important properties in the field of localization and internationalization. Add comments to editable post




You will be able to manage comments directly in the Edit Post mode, including the ability to add new comments.

Other improvements


  • Dozens of bug fixes
  • Acceleration of work along with decrease in consumption of memory
  • Endless scroll when searching for topics
  • A completely new API for topics that affect their overall functionality.
  • Clicking on the admin bar will scroll the screen to the top of the site (mega-convenient thing)
  • Links widget will be able to sort and filter links.

Twenty Twelve Theme



The long-awaited Twenty Twelve theme will not be included in WordPress 3.4, despite the fact that its development is in process. Most likely the new theme will come with a WordPress 3.5 version, which will be available at the end of October of this year.
Its development has been temporarily postponed to github .

Support download favicon

Another expected feature is the work with favicon and its setting right out of the box. There is a small chance that this will be implemented in WordPress 3.4, but most likely you will have to wait for version 3.5.
