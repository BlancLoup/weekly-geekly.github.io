Nimbus Screenshot can now send screenshots directly to Slack

We are pleased to announce that Nimbus Screenshot has learned how to send screenshots directly to Slack. Recently, the popularity of this corporate messenger has gained a truly cosmic scale, and requests to add its support have become more and more. We couldn‚Äôt tolerate further, and in the new version of the application we finally added support for Slack, and we tried to do it in the most convenient form.



Download Nimbus Screenshot for Google Chrome - https://chrome.google.com/webstore/detail/nimbus-screenshot/bpconcjcammlapcogcnnelfmaeghhagj?hl=en-US



Working on the integrat, we tried to make it so that sending a screenshot takes as little clicks and time as possible. For this, a separate panel was drawn with the choice of a channel or user, which does not close until a direct instruction is given to hide it. Thus, after creating the screenshot, it is enough to select the channel / user and click on Send (or Ctrl + Enter). Naturally, before this you can write a comment to the screenshot.

At the moment, integration works only with Chrome. Firefox users will need to wait a bit (literally a couple of weeks).
