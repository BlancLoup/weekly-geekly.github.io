Ubuntu 11.04 Natty Narwhal beta2 released - the latest version before the release

Quietly came the second beta of the popular operating system. It is noteworthy that this time there will be no Release Candidate, which means that this is the latest version before the release.
There are not many changes, but visually they are noticeable.

I'll be brief, this is:
  • Added and fixed Unity, now the panel disappears smoothly, pleasing the eye. Overall improved search and stability. You can also search for alternative places, such as Twitter.
  • Finally, they paid attention to the options and settings of Unity. Now you can choose when the panel should appear - when you hover in the upper corner or when you move the cursor to the left edge of the screen.
  • Appearance - updated icons and standard themes Radiance and Ambiance. The scrollbar is now darker / lighter (which is not always relevant due to the next change)
  • Overlay Scrollbars - a beautiful, but unusual update is now installed by default. Some windows are equipped with a thin scroll bar:

    When the mouse approaches, the slider becomes visible. It works only in applications that use Gnome / GTK +, so, for example, in Firefox this is not the case. But there is in the standard programs. Some programs (for example, a terminal) need patches.
