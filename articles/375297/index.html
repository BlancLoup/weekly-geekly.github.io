Nimbus Note for Windows - now with support for tabs

For a start, I would like to congratulate the respected habravchan on the coming new year, in which we wish the fulfillment of all your desires.

Well, we are glad to present you a new version of Nimbus Note for Windows, in which a number of important changes have occurred, such as: adding a system of tabs, Format Painter for the editor and much more.



Download Nimbus Note for Windows - http://nimbus.everhelper.me/nimbus-note-windows.php

Tabs

Probably the most popular feature requested by our users is the ability to simultaneously work with several notes, between which you could quickly switch, and also copy / paste the necessary information. Frankly, it was not easy to do, given that initially the program was not designed for such work. There were several ways to implement such an opportunity, but we decided to make tabs that I think are familiar to everyone from browsers, and they are also present in many note programs (Notepad + or OneNote for example). By default, all new notes now open in new tabs. If you need to open an existing note in the new tab, then just make a simple ALT + click on the tab. Double clicking on the tabs panel also creates a new note (we took this from the good old Opera :)). If you do not need tabs or interfere, then you can always disable them in the settings.



Format Painter (the same format on the model)

Most of this feature is familiar to Microsoft Word, where it makes life easier when working with notes with less complex text formatting. In Nimbus Note, ‚Äúformat by pattern‚Äù can be useful, both with ordinary notes and in notes received from a clipper. The function works as standard - select the text with the necessary formatting and click on the Format Painter button, then select the desired piece of text with the mouse, to which you need to apply the formatting and voila! If you need to apply formatting to several pieces, then you need to use the keyboard shortcut Ctrl + Format Painter, and you can exit this mode via ESC.



Other improvements

Among other innovations, it is worth noting the addition of sorting folders and tags by date and name, as well as the overall optimization of the program, which favorably affected the speed of work (and not slow before that). Adding tabs required changing a considerable part of the code. Of course, we checked the work of the program very carefully, but there could still be bugs. Therefore, in case of any problems, we hope for your understanding and ask to immediately write us in case of problems found.
