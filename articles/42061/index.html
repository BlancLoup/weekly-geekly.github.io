Public Jabber services (as related services)

Below is a list of public Jabber-services, as related or additional services to other services.
New: jabber-service at GMX.com, saving conversation history in fastmail.fm.
Also: I can not connect to the Jabber-service Nextmail.ru, I am waiting for an explanation from the technical support.

  • LiveJournal (LiveJournal, LiveJournal)
    To connect to the Jabber-service (LJ Talk) is required: an account in LJ, of any type.
    Connection options:
    JID: username in @ livejournal.com domain (eg, user @ livejournal.com)
    Server: livejournal.com, port 5222.
    Use traditional SSL if available, plaintext authentication with encrypted connection.
    Warning : unencrypted connection - use with caution.
    Note : offline messages in LJ Talk do not reach.
  • Gmail.com , an email service, is part of Google Apps.
    To connect, you need: a mailbox in gmail .com or in a domain that uses ‚ÄúGoogle applications for domains‚Äù.
    Connection options:
    JID: mailing address (eg, user gmail .com)
    Server: talk.google.com, port 5223.
    Use traditional SSL, plain text authentication with an encrypted connection.
    Note : Gmail.com saves your message history. It is very convenient for keeping archives - you can connect transports to other IM and all negotiations will be saved in the Chats folder.
  • Ya.ru (blogs, photo galleries, chat, etc. - a social network from Yandex).
    To connect to the service (it's Ya. Online) you need: a mailbox on yandex.ru (ya.ru, narod.ru).
    Connection options:
    JID: mailing address in the domain @ ya.ru (eg, user @ ya.ru)
    Server: xmpp.yandex.ru, port 5223.
    Use traditional SSL, plain text authentication with an encrypted connection.
    Note : the communication history is saved, available when using the Web interface on the * .ya.ru pages at http: // username .ya.ru / webchat_history.xml (use the corresponding username instead of username).
  • Fastmail.fm is a free and commercial mail service with many useful features.
    To connect you need: a mailbox on the fastmail.fm service, of any type, in any domain.
    Connection options:
    JID: full mailbox name (eg user@fastmail.fm)
    Server: chat.messagingengine.com, port 5222
    Use on-demand encryption (STARTTLS), plaintext authentication with an encrypted connection.
    Note : the service offers the service of saving the call history (option, disabled by default).
  • GMX.com , free mail service, also - file storage, organizer, etc.
    To connect you need: a free mailbox on the service gmx.com.
    Connection options:
    JID: full mailbox name (eg user@gmx.com)
    Server: gmx.com, port 5222
    Use on-demand encryption (STARTTLS), plaintext authentication with an encrypted connection.
    Note : for users of the service, transports are provided in ICQ, Yahoo IM.
  • Gizmo Project , Internet telephony service, voice communication.
    To connect you need: an account in the Gizmo Project (created for free).
    Connection options:
    JID: username in the domain @ chat.gizmoproject.com (eg user @ chat.gizmoproject.com)
    Server: chat.gizmoproject.com, port 5222
    Use on-demand encryption (STARTTLS), plaintext authentication with an encrypted connection.
    Note : transports are provided for users of the service in ICQ, Yahoo IM, AIM, MySpaceIM, GG, IRC, MSN.
