Tag Cloud Alternative

Like an example of a tag tree or a pyramid? If several tags are connected hierarchically, then such a representation would be more useful than a tag cloud by which you can judge only the popularity of something, but not a hierarchical position. At the same time, the tag tree can be formed from a multidimensional array in cycles or using recursion. In terms of graphic representation, everything is in our hands.
