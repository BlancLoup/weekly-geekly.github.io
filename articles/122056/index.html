New mouse and HDD allow you to feel the "weight" and file size



Quite interesting devices developed by German students. Devices are a feedback mouse and a hard disk whose size increases or decreases depending on the amount of data stored on the storage medium. The mouse allows you to feel how big is the size of the file that you are dragging. The larger the file or folder size, the more the mouse resists movement. In addition, the mouse still shakes, depending on how often the file is used.

Unusual, yes, and not very practical. But it is interesting - I have not heard of such concepts. I remember, however, a terrible kind of charging, which moved when connecting any device, this charge also makes an impression.

Both the mouse and the hard disk are two components of a single project, which the authors named DataBot. Do you think their development will have the right to life, or will it remain a concept? I liked the hard drive more than the mouse, because I remember how annoying the arbitrary behavior of the mouse was when the wheel was dirty (when ball mice were still running).
