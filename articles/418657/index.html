Available mixed reality: how to create a virtual space

Environment Setup (VR)


To run Speaker Placement in virtual reality (that is, in closed or immersive headsets), we need to create a virtual room and set up the environment so that we can teleport inside the room.

Switch to standard panel layout [optional step]


If you did not work in Unity before, we recommend using the standard Unity panel layout to perform the following actions. A description of the actions is given based on this original location.

To switch to the default layout of panels in Unity, select the command Window -> Layouts -> Default (Window -> Location -> Standard).

After that, the panels will be arranged as shown below. Pay attention to the different panel names in Unity - we will refer to them in our instructions.



Creating a virtual room


Let's add to the scene the room we prepared earlier.

In the Unity Project panel, find the RoomPrefab.prefab room prefab in the Assets -> Prefabs folder (Resources -> Prefabs) and drag it to the Hierarchy panel. The prefab must be dragged to the empty part of the Hierarchy panel to prevent the room from being added to any other object as a child.

Since adding a prefab to the hierarchy adds it to the scene, our room should now be displayed in the Scene panel.

Try searching the room using the keyboard shortcuts and mouse buttons below, but do not move nearby objects. If you accidentally move or move scene objects, simply remove the RoomPrefab prefab from the hierarchy and add it again.

  • To rotate the scene, press the Alt key and move the mouse in the desired direction.
  • Use the arrow keys (right, left, up, down) to move around the scene.
  • To move to the object and place it in the center of the scene, select the object and press Shift + F.

Additional options and ways to move around the scene can be found in the Unity documentation. Navigating the scene .

Space Type Setup


In the MixedRealityCameraParent game object added in the previous section, there is a Boundary object that we need to configure in order to be able to navigate through our virtual room. First, we set the floor.

1. In the Hierarchy panel, expand the RoomPrefab prefab to see all of its children.

2. In the Hierarchy panel, expand the MixedRealityCameraParent element and select (or click) the Boundary object. After that, the Boundary object will be selected in the Hierarchy panel, and its properties will be displayed in the Inspector panel.

3. Leaving the Boundary object selected (try not to click any other item in the hierarchy panel), scroll down the hierarchy pane, find the Floor object and drag it to the Floor Quad property of the Boundary Manager section (Inspector panel) Inspector).

Note. If you do not want to drag and drop (for example, you do not have a mouse to select an object precisely), you can set properties using the scopes button to the right of the Floor Quad property in the Inspector panel only when selecting a target Select objects that are on the Scene tab, and not on the Assets tab.

Since we will work with the standing-scale configuration, we will change the type of space to stationary.

4. While the Boundary object is still selected, set the Opaque Tracking Space Type parameter to Stationary.



For details on the various configurations of Mixed Reality applications, see the documentation for details.

5. Last modified: select MixedRealityCameraParent in the Hierarchy panel and change the value of Transform -> Position -> Y to 0.8. The camera will be located at the level of human eyes.



Study virtual room


Now we can move to our virtual room.

If you have a headset, here are a few recommendations on what to look for when exploring the scene.

  • The headset will track the movements of your head and, by moving your head, you will see different parts of the room. Tracking is performed in such a way that you will feel as if you really are in this room.
  • Walk around the room (for example, step aside) and watch the headset track your position.

If you are using a simulator, follow the instructions in these documents for setting up the simulator and controlling it using the keyboard, mouse, or Xbox gamepad.

But enough expectations! Click the play icon in the Unity editor, put on the headset and explore the room.
