Facebook has released an application for iPad



After postponement and a lot of speculation, Facebook finally released its iPad app. Surprisingly, it looks almost identical to the version that leaked back in July - with a convenient two-pane interface and great similarity with the version for the iPhone.

We probably will never know for sure why the application has not been released for so long, but there are no new Facebook features, including a timeline . However, it has many other key features, including chat, full-screen games and news feed notifications.

So we can say goodbye to all sorts of third-party applications like Friendly Plus. The official app is already available on the App Store.

But that's not all - Facebook has also added features such as bookmarks for applications, a new query dialog with notifications from applications, and Facebook Credits support for internal payments to all of its iOS applications, including the new iPad application.

Facebook assures that applications should be expected for other platforms, including Android.
