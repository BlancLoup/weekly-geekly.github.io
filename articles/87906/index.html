Spring updates Yota

Yota has updated the programs that ensure the operation of devices in the mobile WiMAX network.
A new release of YotaAccess 1.5.1 is available for download - a version for laptops with an integrated Intel WiMAX module and a new software version for the ASUS Mobile WiMAX / Wi-Fi Center router.
The new version of YotaAccess 1.5.1 for laptops with a built-in WiMAX-module Intel Echo Peak aka Intel 5150/5350 will appear in auto-update format from March 18. The updated version improves the reliability and stability of laptops with both the previous version of the WiMAX / Wi-Fi module and the new Intel Kilmer Peak aka Intel 6250 module, first introduced in Sony VAIO models. Improved integration with Wi-Fi is provided by the new version of Intel Wi-Fi 13.1.1, which is also contained in the update.
