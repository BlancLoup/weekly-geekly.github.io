Large-scale update of the mobile version of 2GIS

The iOS version is already available, and for Android it will appear in a few days. By the way, if only the absence of 2GIS for this OS stops you from buying a device on Windows Phone, then you can start choosing a device. ;)

First of all, we answer the question: have 2GIS been made for iPad?



Yes. Right now you can go to the App Store and install an update, in which the long-awaited adaptation for iPad displays appeared. The rest is all the same familiar application that has acquired several new features.

For example, search by phone number. In the search field, you can enter not only the address or name of the organization, but also its phone number. Very useful in situations where you need to determine where you got the call.



Another significant innovation is the appearance of 3D-models of remarkable buildings. MSU, the cruiser "Aurora" and other attractions in your pocket (or bag). Previously, 3D buildings were already available at 2GIS Online , running on API and PC versions .



Of the other nice nuances - light refreshment design. The new main screen, made in the 2GIS corporate style now being updated.
In general, install 2GIS from the Apple App Store right now. The Android version will appear on Google Play a little later. A little later (but very soon), the Windows Phone version will be released.
