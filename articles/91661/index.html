Alt Linux 5.0 Ark. Do you need something else?

Despite the fact that the boxed version was released only recently, the software has already gained popularity, since the free, full-featured version has long been available for download to FTP.

Alt Linux 5.0 Ark is a set of two distributions, Server and Workstation.



In addition to standard functions, such as virtualization, web-based management, and a rich set of applications in a workstation, Alt Linux 5.0 Ark has two important features.

Centralized domain authentication

All users' data is stored on the server, so any user of the system can log in to any of the machines on the network, as well as store data on the server. It is also possible to lead the user centrally.

Powerful backup system for both server and workstation



Alt Linux 5.0 Ark has a backup feature based on Bacula. Using the graphical interface, you can not only assign a partition for which you want to make a backup, but also set the periodicity of the backup. Example: every day, from 12 am to 1 am, the server copies all data to a neighboring disk, or important user data is saved to a personal flash drive in a workstation.

What else?

Thanks to a special module, you can conveniently and quickly configure various filters, such as anti-spam filters, set the size limit for incoming emails. An LDAP user receives a mailbox automatically when registering with the domain.



The system has the ability to centrally update all computers included in the domain. To start the update procedure, you just need to select the machine architecture and the repository from which the update should be made.

Well, a few words about virtualization. If the iron allows, in Alt Linux 5.0 Ark this functionality can be used. Virtual machines are needed for sharing physical resources. At the same time, each virtual machine does not depend on the others, and in the case of its hacking or virus attack, the other virtual machines remain intact. Virtual machines can be: create, adopt (attach ready), pack (for transfer). Alt Linux Server 5.0 Ark supports both kvm virtual machines (also with Windows) and OpenVZ.

Work station



The workstation is based on the GNOME environment and contains a typical set of useful programs needed for office work and leisure. Like all Alt Linux 5.0 products, the Workstation is characterized by complete Russification, integration into the Server, and a convenient interface for managing and configuring the system.



The boxed version, like the electronic version, contains 4 systems: Server and Workstation for two architectures. A significant advantage of the boxed version, the availability of technical support for 1 year, a license agreement and a paper book.
