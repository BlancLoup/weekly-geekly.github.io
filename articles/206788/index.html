Programming for Android. For professionals

As I promised today, the book " Programming for Android. For Professionals " was finally published.
What is interesting about this edition?

This book will teach you all the subtleties of developing applications for mobile devices running on the popular Android operating system. Based on the training course of the famous educational IT project Big Nerd Ranch, this edition contains key Android programming concepts, an explanation of the API, and many code examples and exercises. In the process of working with a book, the reader will independently develop eight Android applications of varying complexity and subject matter, including a client for downloading photos from the Flickr database, an application that simulates a television remote control, and a geolocation service that tracks the user's movement around the world and displays them on the map . All educational applications have been designed to demonstrate important concepts and techniques of Android programming and give experience in their practical application.

The book is sold in paper and in PDF format.
