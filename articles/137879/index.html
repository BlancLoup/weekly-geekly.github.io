Not a single SaaS ...

TeamLab is mainly associated with users of the SaaS solution, while in fact it is not the only available way to deploy portals. If SaaS is, first of all, ease of use, then open source solution, available almost from the very beginning of the project, is a higher degree of independence of you from us, the product developers. Most portal owners prefer SaaS, and that is why all updates primarily concern this version. At the same time, this does not mean that we forget and drive into open-source fans. Just today we want to tell you about updates in the open-source version of TeamLab.

First of all, calendar and CRM were added to 6.0, which many users have been asking for since they were released in SaaS. Thus, all service modules, with the exception of the online document editor, are now available for self-installation. In addition, an installation package is available on sourceforge that allows you to deploy TeamLab on Windows servers.




In other words, the TeamLab deployment process has now been reduced to a download and launch of the installation, which will configure IIS for you, create the database and correct the configs.

What happens during the installation

Installation TeamLabServer.exe

- if there is no Microsoft .NET Framework 3.5 SP1 or MySQL ODBC Connector 5.1 on the machine, it downloads and installs these components;

- copies to the installation directory the necessary files for the service and the site;

- creates the TeamLabSvc system service for chat and full-text search;

- on the MySQL server it forms the necessary database, and in IIS it creates the Teamlab portal, while all the bindings are configured by default on it.

System requirements for installing TeamLab on a local server

  • Processor: 1.7GHz or higher
  • RAM: 1GB or higher
  • Free disk space: at least 100Mb
  • Operating system: Microsoft Windows not lower than XP SP3, recommended by Microsoft Windows Server 2003 and Microsoft Windows Server 2008
  • Microsoft .NET Framework 3.5 SP1
  • Microsoft IIS version not lower than 5.1
  • MySql Server version 5.1 or higher
  • Browsers: IE8 and higher, Firefox 6 and higher, Opera 10 and higher, Safari 5 and higher, Chrome 10 and higher


More detailed information on installing TeamLab on a local server can be found in the user manual .

And information on the transition to the 6th version of open source here .
