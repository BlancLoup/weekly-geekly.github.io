April 3, Samara. WebProfessionals - Microsoft web conferencing. the entrance is free



Dear web developers, programmers, project managers and people who are just passionate about programming, on behalf of Microsoft, I am pleased to invite you to the new fully technological WebProfessionals conference dedicated to web development technologies.

The conference will be held on April 3 in the city of Samara , in the Holiday Inn hotel, where several hundred participants will gather to listen to Microsoft's leading experts and representatives of the developer community, Abbyy, 1C-Beatrix, Parcsis / Turbomilk, Inln, and the Drupal community. The special guest will be a representative of the developers of the Microsoft web platform from Redmond.

During the whole day, 14 technical reports on the following topics will be read in two tracks:
  • IIS web server : efficient work with the server, description of innovations in the new version, placement of ASP.NET, PHP, Ruby, Node.JS, Python and so on projects;
  • Content Management Systems (CMS): systems work on the Microsoft and PHP web platform, efficiency and productivity, migration to the cloud environment;
  • Web application development tools: Effective web application development in the newest version of Visual Studio 11, load testing issues, new development tools and functions for easy development of HTML / CSS / JavaScript code;
  • ASP.NET and frameworks : a detailed description of the innovations in the latest versions of the ASP.NET platform, Web Forms frameworks and ASP.NET MVC, the free WebMatrix development environment;
  • cloud technologies and Windows Azure platform : an introduction to the Windows Azure cloud platform, hosting and migrating existing projects both on ASP.NET and Node.JS / PHP / Java / Ruby and so on, hosting Drupal and .NET content management systems in the cloud Forge CMS.
Participation in the conference is free! Sign up now .
