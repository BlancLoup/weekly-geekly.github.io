foobar2000 v1.1 (beta)

The first beta version of the wonderful foobar2000 audio player has been released.

List of major changes:
  • New ways to install components, as well as their auto-update from the official site.
  • Plug-ins are no longer installed in the player folder, but in the profile folder (% appdata%), which allows each user to use a different set of components.
  • New interface for audio formats conversion plugin, Converter.
  • The new look of most of the Preferences pages and other windows in accordance with the Windows User Experience Interaction Guidelines .
  • Improved compliance with the Matroska container standard, as well as the ID3v2 tag standard.


Now the plugins that are in the official repository can be updated directly from foobar2000. In addition, the plugins downloaded from the repository can be installed from the Preferences via the Install button, or by dragging a zip archive to the Components page:



To avoid prompting UAC from the user when updating or installing plugins, they were moved to the profile folder. The installer of the new version will offer to do this during the installation. However, the components that come with the official installer are placed in the folder of the program itself and are not updated through the player (highlighted in gray).
