EiskaltDC ++. First release number 2.0

Most recently, the first release of the new DC ++ client for Linux was EiskaltDC ++ 2.0. The project is being developed by our compatriots and is the direct heir of EiskaltDC. Immediately, version 2.0 was released due to the transition to the dc ++ core (EiskaltDC was based on dclib), the code was completely rewritten, the new interface is based on Qt4 and looks as close as possible to the original DC ++ clients. As a result, EiskaltDC ++ has become the front-end on Qt4 for the dc ++ core.


Main innovations:


  • Migrating to the dc ++ core (v0.75) and respectively supporting ADC
  • Full-featured chat (parsing of magnets, links, smiles, search in chat, filtering in the list of users, chat commands)
  • Full-featured search engine, file browser (highlighting already shared files, etc.)
  • Grouping gear in the connection manager
  • Separate Download Manager
  • Lists sent / downloaded
  • Selected users (autoslot, etc.), hubs
  • Text (both by Qt and through the default system notification manager) and sound notifications
  • Support for application themes, icons and smiley packs
  • The ability not to share files by mask
  • UPnP support
  • Tag substitution for select hubs
  • Forced indication of external IP for select hubs
  • Auto-update external IP via DynDNS for select hubs
  • Search spy
  • IP filter
  • Spell Checker with Aspell

Source codes and collected packages, as well as basic information can be found on the project page .
