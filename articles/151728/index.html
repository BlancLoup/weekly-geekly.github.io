ShareXMod - fumbles anything and everything



After several hours of searching and testing about 15 programs for sharing screenshots and files in Windows, I think I found the most functional, convenient, and plus freeware & open-source solution in my opinion . I thought that I would feel like a big egoist if I didn‚Äôt share it with someone else. So, welcome!

ShareXMod is an application that allows you to share files and screenshots (+ editing / annotation) in a transparent and convenient way using more than 20 different third-party services. Do not be intimidated by the large window in the picture, all the functionality of the application is implemented in the background, in a state that is minimized.

First of all, I ‚Äôll make a reservation that ShareXMod is an alternative branch of the ShareX open-source implementation . As far as I understand, their main difference is that ShareXMod supports editing / annotating screenshots.

The main functions of the program:

  • File sharing via: Dropbox.com, Rapidshare.com, Sendspace.com, Minus.com, Box.com, Ge.tt, Custom uploader, FTP server, Shared folder, Email
  • Automatic copying to the clipboard of the shared file
  • Screenshot of the entire screen / specific monitor / window / selected area / specific shape
  • Auto-resize snapshot
  • Editing a picture (arrows, texts, effects, turns, etc.)
  • Copy snapshot to clipboard
  • Save snapshot to file on disk
  • Sharing screenshots through: imageshack.us, tinypic.com, imgur.com, flickr.com, photobucket.com, picasaweb.google.com, uploadscreenshot.com, twitpic.com, twitsnaps.com, yfrog.com, imm.io, File uploader
  • Automatically copy to shared clipboard
  • Reduction of links through: goo.gl, bit.ly, j.mp, is.gd, tinyurl.com, turl.ca
  • Configure hotkeys
  • Flexible configuration of automatic actions
  • Command Line Run Options

The application is very flexible. Some settings are not in the most obvious places, but if you want everything you can find and understand. In particular, for me, the most relevant turned out to be: setting the option to turn off the display of a request for actions with a snapshot (see screen ) and enable automatic editing of the picture (see screen ).

I hope the application will seem to you no less interesting than me.
