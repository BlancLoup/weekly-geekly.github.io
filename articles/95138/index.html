Nigma added priority to official sites

Now, if a user is looking for a company or a person who has an official website, this is the site that Nigma will show in the first place.

If there is a Russian version of the site , then, first of all, the user will receive it.

For this purpose, an algorithm was created that collects information from Wikipedia and other sources, which forms a database of official sites for millions of queries. According to the developers , the CTR of the issue page with the block of official sites is one and a half times higher than without them.
