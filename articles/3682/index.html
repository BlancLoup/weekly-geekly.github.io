StarOffice for educational institutions for free

As part of developing its education programs, Sun Microsystem offers free StarOffice licenses for educational institutions.
It is necessary to remind that StarOffice is a package currently developing on the basis of OpenOffice, but having more fonts, dictionaries and its own database. Like OpenOffice, StarOffice is available under GNU / Linux, Solaris, and Windows. The default file format in it is the same as in OpenOffice - OpenDocument (OpenDocument is an international standard approved by OASIS)
