SWAY.ru presents interactive maps of Russian cities

The SWAY project is an online directory of organizations and enterprises in different cities, with reference to the map. The project is conceived as an open, socially significant and socially-oriented Internet resource, addressed to a wide circle of Internet users. SWAY is implemented and will be constantly improved as a cartographic Internet resource covering various cities and offering users a variety of options for the most advanced technological solutions for interactive mapping. After the site leaves the beta mode (active testing), it is intended to significantly expand the site‚Äôs functionality with new features for users.

Already on the site are presented the cities: Belgorod, Stary Oskol, Gubkin. Coverage area will be expanded regularly. Soon you will be able to see the cities: Bryansk, Lipetsk, Voronezh, Kursk, Oryol, Nizhny Novgorod, St. Petersburg and others. The interface for working with maps is implemented using ADOBE FLASH technology and is an interactive environment with the following features:
- smooth zooming of the map without reloading the page;

- the original directory makes it easy to find an organization by activity;
