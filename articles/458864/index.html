Competition developers bot for TamTam



Bot API appeared in the TamTam messenger recently, 3 months ago. We conducted a closed beta test, in which 86 developers created 330 bots. Together with beta testers, we made hundreds of edits, released two updates to the Bot API, and one for documentation .

We decided to support the developers and together with Mail.ru Cloud Solutions we are holding a bot contest for TamTam. The prize fund is a total of 200,000 rubles worth of coupons for the use of the Mail.ru Cloud Solutions cloud infrastructure and any services based on this platform. Each participant in the competition, in addition to the winners, will also receive a promotional code for 5,000 rubles to use the Mail.ru Cloud Solutions services. Registration of participants is open until July 21, 2019 inclusive.

To take part in the competition, you need:

  • develop a bot for TamTam;
  • write a competitive bot and register your bot.

4 winners will receive coupons of 50,000 rubles each for 1 year of use of the Mail.ru Cloud Solutions cloud infrastructure. The winners will be selected and announced by the TamTam team on the official Bot API TamTam channel on July 30, 2019.

Documentation and client libraries:

  • Description Bot API ;
  • OpenAPI JSON schema ;
  • Official Java Client ;
  • Javascript ;
  • Kotlin DSL ;
  • Python .
