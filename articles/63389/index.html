Tags in Gmail: drag and drop and more

Gmail has updated the interface again. This time, the changes have affected the labels.

We will be informed about this immediately upon entry:


Tags are now in the folder list and collapse into the drop-down menu:

Which tags to display and which to hide can be customized as you wish.

The most important thing is drag-n-drop, now you can mark the letter by dragging it with the mouse to the appropriate label, it is very convenient:


Changes unfortunately are not yet available in Google services for domains.
UPD: Habraludi write that the new version is not available for everyone in the standard Gmail, and for some it works even in domain services. Apparently someone will have to wait.

More details about the changes can be found in the official blog Gmeyl (Eng.) .
