Microsoft is developing an adaptive shell for Windows 10

Journalists of Windows Central claim that Composable Shell will be an adaptive shell that can adapt to the gadget on which Windows 10 is running.

Now Microsoft uses various shells for Windows 10 on PC, mobile OS and Xbox system. Composable Shell removes the need to use its own shell for each of these types of devices. The OS will be unified and will be able to work, in addition to the mentioned devices, on other systems.

The upcoming Microsoft phone will have a foldable screen, and Composable Shell will show its capabilities. So, in the folded state, Windows 10 will work in mobile mode. When you open the screen, Windows 10 with Composable Shell will switch to desktop mode. Composable Shell - a shell divided into separate modules - components. The system can transfer data between individual components, if necessary.

The shell will be truly universal. Windows 10 Mobile, for example, will be able to work on a desktop PC, but the desktop version of this operating system will work without problems on a mobile device. Windows 10 will be used to work on devices like Hololens and Xbox.
