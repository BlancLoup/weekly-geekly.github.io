Git for Windows 2.5.0 released


The first official version of Git for Windows has been released. Unlike msysGit, the installer now contains only Git (previously, there was everything needed to build Git), now there is also an installer for the 64-bit version.



You can download from GitHub .

A brief description of previous versions that did not come out under Windows:
  • Version 2.5: git worktree
  • Version 2.4: atomic pushes
  • Version 2.3: push-to-deploy
  • Version 2.2: minor improvements to various commands
  • Version 2.1: improved console output, improved performance
  • Version 2.0: some default behavior changes for git push, git add and git svn
