Firefox OS 1.3 has been released: improvements in graphics, audio, dual-SIM support



Mozilla continues to work on improving its mobile operating system, Firefox OS. This time, Firefox OS 1.3 was released (this is a release for Mozilla partners), and soon ZTE Open C smartphone with this operating system will appear . By the way, the price of a smartphone is only $ 100.

As indicated in the title, the OS underwent significant improvements, including the addition of dual-SIM support, improvements to the camera module, optimization of work with video / audio streams.

Key updates / enhancements are:

  • Add or improve support for WebGL, asm.js, and WebAudio;
  • Support dual-SIM;
  • Improved camera software, includes constant autofocus and flash on supported devices;
  • Gallery allows you to sort photos by date, and display information for each file;
  • Taking control of the player to tray;
  • Ability to work FM-radio module with built-in speakers, without headphones;
  • Sort applications by type (games, music, social, etc.);
  • Support for email notifications, as well as support for POP3 accounts;
  • Support transferring a group of files at once via Bluetooth.


In addition, NFC support has been introduced, but now this is more an option for developers than for regular users.
