Laptop for 99 dollars!



While various vendors and organizations are only trying to make children from developing countries happy with laptops of $ 100 or $ 200, the little-known company JoinTech is already offering to buy their laptop for only 99 conventional units. Of course, the JL7100 cannot be called a full-fledged computer, but for most tasks assigned to devices of this class, it is quite suitable. New running Windows CE 5.0 on a 400 MHz ARM processor from Samsung. The volumes of the built-in and RAM memory are extremely modest (only 64 megabytes in both cases), but the bundle will include a 4GB flash drive, due to which you can later increase the available memory capacity. The undoubted advantages include 3 USB ports, a 7-inch VGA screen (800 x 480) and a fairly decent set of pre-installed programs. Moreover, binary compatibility with applications from Windows Mobile is expected (due to the use of the same processor architecture). Such a laptop weighs only 680 grams, while it is equipped with a full keyboard and touchpad, so you can take it with you wherever you go.

More information is available on the manufacturer's website: www.jointech.com.hk/jl7100.html
