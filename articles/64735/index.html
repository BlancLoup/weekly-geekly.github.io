First mobile botnet detected

The first full-featured botnet was detected on the mobile Internet - special malware infects the users' phones and is able to steal their data and send spam.

Computer security specialists from Trend Micro announced that they had discovered the first botnet, including mobile devices. In their opinion, this fact will significantly affect the development of network threats for mobile users.

Trend Micro employees found malware for Symbian systems called SYMBOS_YXES.B. It infects users' mobile devices when they visit various websites.

This software is able to collect confidential data of the owner of the phone and send this information to attackers. In addition, SYMBOS_YXES.B can send spam SMS from an infected phone to all numbers in the user's contacts.
