Windows Store crossed the mark of 500,000 applications


Microsoft has reported on the new achievement of the Windows Store and Windows Phone Store - the number of available applications has stepped over 500 thousand.
To be more precise, the number of applications in the Windows Store reached 150 thousand in March, and in his mobile colleague - 300 thousand in August.

It will be appropriate to compare this data with the number of active applications in the stores of major competitors. Thus, in Google Play, the number of applications in August of this year reached 1.3 million, and the App Store reached this figure in September.

Microsoft is under serious pressure and is trying in every way to expand the market of its platforms, so the release of a branded phone is not far off. For this, Microsoft donated even the Nokia trademark, as it changed the loyalty program for its products as a whole - for example, Microsoft started to release its products for Android Gear, and the latest update for Office Mobile on iOS made it more functional than on Windows Phone itself.

If we talk about the total share of mobile devices on Windows, then according to Kantar Worldpanel, Microsoft is experiencing hard times: having reached a peak of 10% in 2013, the share of Windows dropped to 8.1% in 2014. The only exceptions are some countries - for example, in Italy, the share of Windows has surpassed even iOS and reached 15% in September.

Oculus SDK for mobile platforms


Oculus Mobile SDK is now available at the Oculus Developer Center. This SDK will allow you to develop mobile applications for the Samsung Gear VR Innovator Edition virtual reality helmet that is compatible with the Samsung Galaxy Note 4.



Pre-order can be done today. The cost will vary from $ 199 to $ 249. Deliveries of a helmet from Samsung will begin at the end of this year.
