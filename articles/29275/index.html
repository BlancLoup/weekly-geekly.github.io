What is mini fring ? This is a fring client that can work on mobile phones supporting the J2ME (Java) platform. Those. This version of mini fring is supported by more than a hundred phone models. In particular, it works on models from Sony Ericsson and Nokia (S40 platform editions 2, 3 and 4).

This version of mini fring allows you to chat using text chat with most of the services supported by fring.
