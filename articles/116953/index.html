MonoDroid 1.0 released


This is the first stable release. Among the main features:
  • development on .NET and C # for Android tablets and smartphones
  • .NET-bindings for Android API
  • application publishing in the Android Market
  • Visual Studio 2010 plugin (MonoDevelop support will appear in future versions)
  • Mono 2.10

More information can be found on mono-android.net
Link to trial version: mono-android.net/DownloadTrial
