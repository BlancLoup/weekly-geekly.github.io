Linux Mint 11 Katya released

Today released the release of Linux Mint 11 "Katya".
Release Candidate has already been discussed on Habr√©.
For those who have RC installed, it is enough to install all updates of the 1st and 2nd level.

The distribution is based on Ubuntu 11.04, kernel 2.6.38. Compared to the previous version, OpenOffice has been replaced by LibreOffice, Rythmbox by Banshee. Details about the innovations version can be read here and here . Yes, Gnome 2.32, not 3.0, as planned once.

I use it with RC, everything is stable and pretty good. From the observed bugs:
- the application manager shows that OpenOffice is installed, it does not find LibreOffice
- Banshee refused to play mp3 files. There are no problems in Rhytmbox and other players.
- X.org refused to boot when an additional power cable was removed from the nVidia video card. Windows 7 in this case only reports a power shortage.
- In MintMenu, sometimes the selection of a menu item is not removed when you remove the cursor.
