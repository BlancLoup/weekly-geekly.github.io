Embarcadero Announces Purchase of Raize Components

Friends!

Embarcadero announced the purchase of Raize Components and at the same time release two new products:
Konopka Signature VCL Controls - solution for designing Windows application interfaces;
CodeSite is a logging solution for Delphi, C ++ Builder, and Visual Studio developers.



Konopka Signature VCL Controls is a solution that allows you to quickly design interfaces for Windows applications that meet modern requirements. Using a set that contains more than 200 components for Windows, which will help make the application comfortable and attractive. A large variety of features of the property editor simplifies the design and development of interfaces for Windows applications.


Components fully support VCL styles, which also allows you to create modern applications for Windows 10.


CodeSite is a logging or logging system that allows you to maintain a convenient and, most importantly, the most informative log of the application.
It is known that the CodeSite tool with limited functionality has long been part of the standard delivery of RAD Studio, and this functionality was quite enough for many people to conduct full-fledged logging of their applications.


However, before each developer sooner or later the question arises, how to track the work of your application? And the developer can go expensive invention of the "bike" or use a ready-made professional solution specifically designed for such tasks.

CodeSite enables the developer to better understand the mechanism for executing his code, both during the development phase of the application and at the work of the application with end users. CodeSite allows logging with saving to a special file that can be easily sent to the developer, and he will open it with the help of a special tool Live Viewer.


My personal experience shows that those programmers who got acquainted with CodeSite and put it into practice never return to their previous tools, which is what you want!
