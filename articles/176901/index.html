Trainings Nokia Developers Workshop in Russia

We invite you to take part in the Nokia Developers Workshop in Russia training, which will be held April 22-24, 2013 at Petrozavodsk State University.
During the event, two professional trainings are planned for developers of mobile platforms Windows Phone 8 and Series 40 Asha.




Training for developers on Windows Phone 8.
In this lesson, students will become familiar with the specifics of developing programs for the latest models based on Windows Phone 8, such as the Nokia Lumia 920.
The program includes several brief reviews on current programming topics and on the Windows Phone API, followed by practical exercises in application development. A prerequisite is that participants have practical experience in object-oriented programming.
The purpose of the training is to prepare participants for the independent development of applications and services for the Windows Phone platform.

Training for Java ME developers for Asha.
The event will showcase the latest user interface APIs, as well as the latest tools and APIs for geolocation services, maps, sensors and monetization.
The goal is to share with the Java ME programmers the latest updates for Nokia Asha series devices that are significant to the developer.

Participation in the event is free, the number of seats is limited, so do not forget to register for the event at the link: www.fruct.org/nokia13

Requirements:
Participants are expected to be familiar with the principles of object-oriented programming. Preferably experience in the development of C # / XAML.

Windows Phone 8 Training: The only tool you need is the WP8 SDK and Internet access. Download the SDK by following the link dev.windowsphone.com/en-us/downloadsdk .
The WP8 SDK requires a workstation running Windows 8. If this is not possible, you can use the WP7.X SDK on a computer running Windows 7.

Java ME training for Asha: Training that does not take much time and does not involve large amounts of application programming. All necessary materials will be offered to participants on a USB-drive.
