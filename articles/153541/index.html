Briefly about the new: Samsung introduced a new series of solid-state media SSD 840 Series

Last week, Samsung Electronics announced the launch of a new line of high-performance solid-state drives designed for use in PCs. The new series with support for the SATA 3.0 interface will be delivered in two options: basic for general consumer purposes and an extended PRO option for advanced users.


SSD 840 is the first consumer solid-state drive built on TLC flash memory (unlike MLC, this memory uses three bits per cell, not two, which allows for increased storage density).



The premium version of the Samsung SSD 840 PRO is still built on the MLC flash memory. It is more suitable for use in corporate environments and servers.



The SSD package also includes updated software, including the new SSD Magician + migration utility, which allows you to clone the operating system without any extra effort. The entire new SSD line is created on the 21-nm process technology.



The innovation of Samsung solid-state drives lies in the proprietary MDX controller (by the way, Samsung is currently the only memory manufacturer developing all its elements on its own). The new controller provides high performance, reliability and durability of SSD-NIS. The premium 840 PRO provides 100.000 I / O operations per second (IOPS) for random read and 90.000 IOPS for random write.



The new SSD 840 series will hit the market in mid-October.
