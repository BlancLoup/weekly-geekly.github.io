Clementine-player 1.0.0 released

On December 27, a new version of the free cross-platform media player and music collection manager, Clementine, was released . Clementine was created in 2010 as an Amarok 1.4 port on Qt4.

Major changes:
  • Added support for Spotify;
  • Grooveshark support added;
  • Added radio stations Digitally Imported (di.fm) and Sky.fm;
  • Global search function for personal music collection, radio stations, Spotify and Grooveshark;
  • Added support for Audio CD;
  • Amazon has been added as an alternative source for covers;
  • Added Nyanalyzer Cat (analyzer);
  • Many bugs fixed.
