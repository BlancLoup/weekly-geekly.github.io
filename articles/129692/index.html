Introducing the Menutka project

One of the first projects presented by Glavstart at the recent Investment Session was the Menutka project.

Menutka is a service with the help of which people search for specific dishes from restaurants menu according to various parameters, including ratings and reviews of other people.

The main feature of the application is that Menutka can recommend dishes based on the tastes of the user. The system adjusts to the tastes of a particular user using the collaborative filtering algorithm:
