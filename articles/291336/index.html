What solution do we offer


If the visitor needs assistance in finding a product or service, the operator can demonstrate them directly on the client‚Äôs monitor using the Teleportor . It is enough to inform the operator of the code indicated on the page.



It looks like the TeamViewer remote computer management system, only there is no need to install software. It is safe - the user's browser cannot be controlled unless it has enabled the service.

Service can:

  • show the page the operator is looking at
  • show the page the client is looking at
  • move the visitor to the selected part of the page
  • you can select the text and it will be allocated to the client

The visitor cannot be transferred to another site: you can move only within the domain name for which the Teleportator code is set. So we excluded the possibility of intercepting visitors by sorting codes and redirecting to competitors' sites.

Read more about the process


During a telephone conversation, an employee of the company asks the site visitor to click on the Teleportor button and dictate a four-digit access code.


How it works on the site Enter

The operator translates the dictated code into the control panel.

Everything, now the Krible server connected two sessions in the general connection. The operator can request the address of the page that the client is currently viewing. One click - and the Teleportor transmits the URL from the client to the operator (makes a redirect in the operator's browser).

Now the operator sees the same page as the client. The dialogue is becoming more substantive.

And vice versa - the operator can show the necessary pages on the client‚Äôs screen.
One click, the URL is transferred in the other direction.

The operator can highlight the selected text with a marker. To do this, select a section of text with the mouse and click on the "Send" button. The service will display the desired page on the client‚Äôs screen, scroll to the required site and tint the selected text. It is very convenient to discuss the points of the contract or the pepper bulk price list.

This method of communication transmits a minimum of data, provides full cross-platform compatibility, does not require the installation of additional plug-ins and libraries.

How else can a teleporter be used?
