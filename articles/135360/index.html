New version of MotivateClock. We did what you asked for!



New Year's Eve is always filled with festive bustle, children's laughter and the smell of mandarin. This is the time when you believe that any miracle will happen. Suffice it to be stronger and want to close your eyes with all your might.

Exactly three months ago , on September 27, we announced to the Habrasoobschestvu about the start of our project. Most of the comments and suggestions then we heard about the need to implement site-by-site time spent in various browsers.

Today, December 27, we will present a new version of the program Motivate Clock 1.1.147. I do not know for whom this is a great gift for the New Year tree - for you or for ourselves. We did what seemed difficult enough and managed to do it on time.

Official website of the program: motivateclock.ru

Link to download MotivateClock 1.1.147: motivateclock.ru/download/motivateclock.exe



The most important functionality of the new version is the recording of time spent on websites (supported browsers: hrome, FireFox, Opera, Internet Explorer).



It is also worth noting that the manual management of any project is now available, support for Java applications has been added, and soundtracks have been implemented for pop-up windows informing about the duration of work and rest.



We could not go past the requests to translate the interface of the program into English. Now you can choose the language version in the program settings.



If there is a demand for any European or Asian language versions - write, we will try to implement promptly :)

I am sincerely glad that we are entering the New Year with an updated program, created largely on your comments, suggestions and suggestions. Implementation was made possible thanks to a team of professionals with whom I was fortunate enough to work. Work does not stop for a second. That's what I managed to photograph while the designer turned away :)
