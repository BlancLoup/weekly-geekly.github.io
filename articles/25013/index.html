Add cells to SpeedDial, and other tuning

I know that many of you dreamed about it! It is done! :)




How is this done?

- close the Opera
(if you don‚Äôt know where your profile folder is located, first look at the path to it via Help > About > Opera directory )
- in your profile folder, find and open the speeddial.ini file
- after the comment header, paste the following code, then save the file:
[Size]
Rows=4
Columns=5
- start the Opera! ;)

Naturally, instead of 4 and 5 you can enter any desired number, but, as the discoverer of this hidden feature writes , the Dutchman Remco Lanting, if there are too many cells in SpeedDial, the Opera can be very slow, so know the measure :)

UPD: As it turned out, this feature works exclusively in version 9.5 beta 2.
Also, Comrade Shpankov noticed that ...
when synchronizing with Opera Mini, only the first 9 tabs will be presented on the mobile phone, synchronization with another desktop version will take place without problems, but, of course, you must first correct the speeddial.ini in the second copy of the browser.
UPD2: The number of cells can also be reduced. Useful in two cases:
- if you use not all cells, and do not like graphic garbage
- if you have a widescreen monitor: you can make yourself 4 √ó 2, for example :)
