New server configurations based on Intel Xeon E5

Intel has just officially launched the sale of E5 processors, as we launched the server rental service with these processors.

As the first configurations, we chose the following processors: E5-2603 , E5-2620 and E5-2630

Inside, there is a Supermicro X9DRi-F motherboard.

What can you expect from these processors? The main thing is speed, as in new processors there are more cores. The higher the exchange rate with memory, higher cache, optimized queues, pipelines and other hidden from the eyes of the nuances that distinguish the new and fast processor from the old and slow.

Based on the wishes of our customers, we added more memory to the new configurations. It turned out a great platform with maximum performance.

And in the top configuration, SSD disks were also added, which allow to get the maximum performance of the disk subsystem. According to the specifications, these SSDs give 10k IOPS for writing and 20k IOPS for reading.
