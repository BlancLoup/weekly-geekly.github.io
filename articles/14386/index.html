CMS "Energine" - public exit

In the huge family of Open Source CMS replenishment. Energine's previously closed software product is now available to everyone. This development by ColoCall has so far been used to create and maintain company sites, as well as for other commercial projects.

Energine is a powerful framework that helps you create websites quickly and effortlessly, but at the same time get an excellent result. The system compares favorably with the rest of it like a clear hierarchy of classes and ease of extensibility. Since the program is based on the component principle, you do not have to change the kernel to add fundamentally new functionality. All that is needed is to simply create, following the example of already existing components, a new component - and work only with it. Also, you do not need to worry about how the system interacts with databases, or processes certain data obtained - you simply use ready-made methods and constructs, redefining them as necessary.
Energine is written in PHP5. MySQL5 is used as a DBMS. The system uses XML + XSLT technology, which allows flexible management and a clear separation of the visual presentation (:-)) and the logical part.
