35 fresh and useful jQuery plugins

Fast and powerful jQuery can help designers and developers create great interactive websites that will be attractive and compatible with most browsers. Your site will be both interesting and entertaining. Navigation, galleries and slideshows are the components that can shine on your site.

This article contains 35 useful and fresh jQuery plugins focused on navigation, galleries, slideshows, calendars, tabs, etc., which will reduce the time and effort required to increase the number of visitors to your site.

jQuery navigation plugins



Leopard Desktop with jQuery using jqDock

jQuery adds to your site a great many wonderful features. He can do a lot, starting with animation and ending with AJAX functions. This jQuery plugin that uses jqDock can assist you in creating a complete analog widget panel like in Leopard OS. It conveniently hides many gadgets or widgets for which you do not have enough space.

jQuery gallery and slideshow plugins



jQuery.popeye 1.0

Big release of a completely rewritten jQuery.popeye. This is an image gallery plugin that was created as an alternative to the popular JavaScript lightbox.



Pikachoose jQuery Image Gallery ( Demo )

Pikachoose is a lightweight jQuery plugin that provides an easy way to present images with slideshow options and navigation buttons. Pikachoose is designed so that it can be easily installed and configured.



Supersized - jQuery Fullscreen Plugin / Slideshow Plugin ( Demo )

The jQuery Supersized plugin can resize the background image while keeping the aspect ratio. Also, the plugin can change the background images in the form of a slideshow using transitions and preloading. The navigation menu allows you to pause or go back and forth.



Easy Slider 1.5 - The easiest jQuery plugin for image and content slide

Easy Slider 1.5 jQuery plugin has many features, such as automatic slide, long slide, ‚Äúgo to first‚Äù, ‚Äúgo to last‚Äù buttons, hide control buttons, vertical slide, multiple slides on one page, etc.



jQuery lightbox for images, video files, YouTube, iFramees from Stephane Caron

jQuery prettyPhoto plugin supports not only images. It also supports video files, Flash, YouTube and iFrame. This is a full-fledged media lightbox with useful APIs that can be launched from anywhere, even from a USB drive.



Spacegallery - jQuery plugin

Spacegallery is a JavaScript image gallery with jQuery plugin. After setting the parameters, the images are animated as a perspective. When you click on the first image, it disappears smooth animation and is replaced by the next image. Despite the fact that the plugin is created with one type of navigation, this is a good gallery when you do not have too many images to display.



jquery-rotate - Google Code

This library is an extension of jQuery for rotating images directly in JavaScript. The plugin is quite useful if you need to organize thumbnails of images without synchronizing with images through the server.



Galleriffic - jQuery plugin for visualizing high-speed photo galleries

Galleriffic is a high-performance plugin for working with a large number of photos. Features include preloading images after the page has been loaded, navigation through page-by-page thumbnails, support for bookmarked URL images, a slideshow with optional auto updating of the URL of bookmarks, and support for multiple galleries on one page.



JQuery Cycle Plugin

The jQuery Cycle plugin is a lightweight slideshow plugin. It supports pause when hovering over the image, automatic stop, forward / backward navigation, and many other effects. It also supports, but does not require the Metadata and Easing plugins.



jQuery Infinite Carousel

jQuery Infinite Carousel is a spinning carousel that works in the same way as a carousel on an Apple Mac's ad page. This guide will tell you about all the properties of this effect and teach you how to create it.



Fancybox

FancyBox is a tool for displaying images, HTML content and multimedia. Created using jQuery library.



jQuery Flash plugin

This jQuery plugin is designed to embed Flash movies. Pages are progressively expanded when using Flash and JavaScript and, conversely, with simple (X) HTML. Inspired by SWFObject, UFO and sIFR but fully created for jQuery, this plugin does not require any other scripts, which means less frills, small file size and fast loading.



JQuery Plugin MopBox 2.2

MopBox is a drag and drop container that can contain images, Flash movies, video clips, etc. If it contains more than one item, then navigation is automatically shown. Size can be changed.



Jquery media plugin

jQuery Media Plugin supports converting standard layout to display content. It can be used to embed almost any media content, including Flash, Quicktime, Windows Media Player, Real Player, MP3, Silverlight, PDF, etc.

jQuery tabbing plugins



tabSwitch

tabSwitch is a jQuery tab plugin. With this plugin you can create a tabulation system with less code usage. At the moment you can choose from 7 different effects with 2 output styles.



Autotab: jQuery auto-tab and filter plugin

Autotab is a jQuery plugin for auto-tabbing and filtering text fields in a form. As soon as the maximum number of characters in a text field is reached, the cursor automatically moves to the next defined field. In the same way, if you delete the contents of a field by pressing the Backspace key, the cursor will move to the previous field. The purpose of this plugin is to create auto-tabulation among a variety of text fields.
