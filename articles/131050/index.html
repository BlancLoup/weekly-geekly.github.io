Instructions for setting up startup for Linux and Windows insider

Photos of the long-suffering RZ12


Windows
1) Download technet.microsoft.com/en-us/sysinternals/bb963905.aspx
2) Run and enable autologin on behalf of the user.
3) Add the required applications to the standard autoload (Start-> autoload).

Linux
a) Method inaccurate, but widely used:
/etc/rc.local
All commands in this file will be run after rebooting the system.

b) Neat way:
If the service is installed correctly, then chkconfig --add $ service_name will add an alias to launch at the boot levels /etc/rcX.d
Naturally, before that you should install chkconfig itself. In the Debian family, unlike RHEL, it must be installed: apt-get install chkconfig.
