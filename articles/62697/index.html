Joomla 1.6 - Alpha version


Post to Google Groups
Download Joomla 1.6

Joomla 1.6 demo

We have worked a lot over the past few weeks on Joomla 1.6, and I can say with satisfaction that we have made progress in that time, but there is much to be done. We have made significant progress on the ACL (access control level) system, the nested categories, and also made a large number of other improvements.


In the new ACL system all the way. Now you can create new user groups, assign user groups, assign access levels to groups, and assign content (that is, make content available only to specific user groups), view user access to access levels. We continue to work on the details of the user interface and business logic for access levels. The transition to MooTools 1.2 is complete and we have started cleaning the core JavaScript code to make it more compact and efficient. The menu system and manager menu have been completely rewritten and now, due to a number of important improvements, have become more powerful, efficient, and easier to set up.

We redid a number of other components, including com_content, com_users, and com_weblinks to make them easier to maintain, configure, and extend. We cut a lot of outdated code and media from the kernel, which increased the size of the distribution by 30% or so. We also made a number of improvements to the performance of the plug-in system and the menu system, which will noticeably affect major websites.

So, in the light of all of this, we are releasing an alpha version to give people the opportunity to see what we are working on and look at Joomla 1.6. It is important to understand that this is still the alpha version, not the final product. There are many shortcomings, something damp. We know that. Many functions with bugs, so do not panic if something is wrong, as you might expect. As I said earlier, we still have a lot to do.

In the end, I would like to thank Hannes, Gergo, Ercan, Louis, Andrew, and Sam for their hard work.

Enjoy

About Joomla.
CMS Joomla is one of the most popular content management systems in the world, made in php using MySQL database. The version of Joomla is divided into lines (1.0; 1.5; 1.6). Each new line is very different functionally from the previous one (the code is very much rewritten). The work of extensions of the previous line in the transition to the new line is possible in the so-called inheritance mode (legacy), which is included in the Joomla administration panel. Inside the version line, they are not very different from each other (as a rule, patches are released to eliminate critical vulnerabilities, as well as bug fixes).
