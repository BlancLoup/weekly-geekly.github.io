Opera 11.00 Release Candidate 1

Thanks to testing conducted both within Opera and the community, we now have a very good release candidate Opera 11.0.

In fact, we have now reached the stage when Opera 11.0 is mature enough for the final release.

Over the past year, our improved development process has allowed us to work on a large number of things at the same time, and at the same time maintain a stable main branch. Therefore, when released builds were unstable, it was usually the result of adding experimental features added separately to create builds (in a team, these builds are informally called Frankenstein Builds or Frankenbuilds ). After proper testing, new features and fixes are added back to the main branch. This is the reason why it is impossible to judge the quality of the final release based on previous builds.

At the moment, there are still a few not very serious errors, but we want to make sure that there is nothing obviously interfering with the release. We are especially interested in hearing about major issues affecting tabs, extensions, and the new mailbox or Windows installer. And although our improved development process helps prevent regressions in the main branch, we still want to know about all the problems that could potentially hinder the release.

So if someone finds any new major bugs in this release candidate - let us know!

Note: since this is a release candidate, the default installation folder will be the last stable installation folder.
