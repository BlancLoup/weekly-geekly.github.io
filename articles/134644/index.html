2GIS API. New organizations every day

Since December 2011, the API 2GIS product has been updated daily. For example, if an organization has opened on Monday, data on it may already appear on Tuesday in our directory. This applies to all online products based on API: Flamp , 2GIS Online version and partner projects. Therefore, now you can find out the address of a new restaurant or write a review about it literally the next day after opening, when information about this will be recorded by our specialists.



Previously, API 2GIS was updated monthly on the 1st day. But with such a schedule, companies that opened, for example, on the 2nd or 3rd day, got into the directory only next month. Now all changes are made every day, which ensures a constant freshness of information. For example, this week a finishing materials shop ‚ÄúAtelier of Comfort‚Äù and a beer restaurant ‚ÄúBeerman & Pizza‚Äù opened in Novosibirsk, and after a few days, information about these institutions appeared in the online version of 2GIS. Restaurant visitors, by the way, have already managed to leave their first reviews on the Flamp.

To begin with, the daily update was introduced only in Novosibirsk, but in the very near future all other cities will be connected - first of all those in which there is the Flamp. If you become aware of a new organization that is not yet in the directory, please let us know via the feedback form . Together we can update the data even faster.

API 2GIS is a company product that allows any Internet projects and applications to receive all 2GIS reference information through software interfaces. The API was launched in August 2011; for 4 months of work, applications for connection were submitted by more than 600 companies; among the already launched projects are Independent City Sites ( ngs.ru ), Ekaterinburg On-Line ( e1.ru ), U-mama.ru , Electronic City ( cn.ru ) and others. The reference data contains detailed information on more than 140 cities, including more than 850,000 companies and over 1,220,000 POIs. In addition to reference data, it is planned to launch API maps, transport, reviews and other information in the near future.
