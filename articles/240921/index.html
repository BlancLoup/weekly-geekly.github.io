Verve 2: an electronic designer of smart devices for beginners + the opportunity to learn the basics of electronics



Modern technologies have made great strides forward, and many people would like to learn the basics of electronics, but they consider such training very difficult, if at all possible, for the uninitiated person. On the other hand, the same technologies allow you to create universal electronic kits, with which you can assemble your own smart devices, with connection to the Network, without knowledge of electronics.

The Verx 2 project from Inxus is an opportunity to gain knowledge of the basics of electronics, in the process of creating your own smart device (this could be a remote control system, a home security system, and everything else).

Appearance





The main board is enclosed in a green plastic case, on the sides of which there are connectors for connecting additional sensors. In addition, there is a USB connector for connecting to a PC and a connector for connecting the power supply.

The functionality of the kit is fully disclosed when additional sensors are connected. There are 9 of them, each designed to work / react to a certain factor, including sound, pressure, electric current, magnetic field, movement, light, and rotation of the handle.



In addition, there is a tenth type of sensor, which is called a DIY-sensor. This is a miniature electronic node, the functions of which can be configured by users already with any knowledge of electronics. The sensor includes 0.1 "GND, signal, and 3.3V pins.

The developers of Verve 2 presented four types of sets, from basic to advanced:


1. Basic kit with Verve 2 board, connecting cable, lighting sensor, flash drive with drivers, mini-USB cable.


2. Set with a Verve 2 card, a connecting cable, 7 sensors (all except the sound sensor and a force sensor), a flash drive with drivers, a mini-USB cable.


3. Complete set with all sensors.


4. Double full set
