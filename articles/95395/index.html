Lingvo Mobile: Present and Future

Perhaps for some this is not news, but ABBYY has finally supported Symbian 9.4 and Windows Mobile 6.5.3 in its Lingvo x3 Mobile product. To be honest finger-oriented product interface has not yet become, but all serious errors have been corrected, and it is quite possible to use it.

To the traditional for Lingvo boxed versions with several translation directions and dozens of dictionaries, a cloud of distributions has been added, each of which contains a software shell and exactly one dictionary of general vocabulary. For example, in ABBYY Lingvo x3 Mobile En-Ru, this is the English-Russian dictionary LingvoUniversal. Single-version versions are much smaller than a box (190 rubles at the moment), placed both on the box and on top of each other.

I think they will still be asked, so I will immediately answer:
  1. Yes, they released an update for a long time. But now all the worst is over.
  2. For Android and Windows Phone, so far (say, this year) Lingvo will not.
  3. On the iPhone, on the contrary, everything will be fine.

Konstantin Tarachev,
Head of Mobile Linguistic Applications Development Group
