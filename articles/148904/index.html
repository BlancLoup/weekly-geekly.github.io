The development of Windows 8 has been completed. The system has moved to the RTM stage

Today in a blog of the Windows 8 development team, Brandon LeBlanc said that the development of a new Windows has been completed. Starting today, key OEM partners will begin receiving final system assemblies for building devices.


An access calendar has also been published for subscribers of various Microsoft projects:

  • August 15, all MSDN and TechNet subscribers will be able to access the assembly of the "golden code"
  • On August 16, all Microsoft Software Assurance customers will be able to access Windows 8 Enterprise through the Volume License Service Center , as well as all Microsoft Partner Network members will have access to all versions of Windows 8
  • August 20 Microsoft Action Pack Providers will get access to Windows 8

If you are a member of other Microsoft programs, then expect more information about the availability of Windows 8 images. For regular users, there is still a Release Preview .
On August 15, all developers are expected at the Developer Center , where all the necessary development tools will be available.
The final version of the system will be officially released on October 26, 2012, and on this day, users will be able to access the Windows Upgrade Offer.
