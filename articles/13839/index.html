4-terabyte data storage from LaCie

LaCie introduced Biggest Quadra, a 4-terabyte storage designed for data backup. The novelty supports the ability to create RAID-arrays of levels 0, 0 + 1 and 5.


To connect to a computer in the Biggest Quadra used interfaces FireWire 800, FireWire 400, USB 2.0 and eSATA. The device supports hot-swappable drives, it also has an automatic temperature control function. Included is a backup utility for EMC Retrospect Express (PC and Mac versions).

The first, 2-terabyte device from the line will appear in the current month, and will cost $ 1099. 3- and 4-terabyte models (cost, respectively, $ 1,599 and $ 2,199) will be released in October.
