Version 1.1 beta

Everything is going according to plan, and the Yii framework team proudly presents us with beta version 1.1.

The version contains corrections of several minor bugs found in the previous alpha. Also made a few small improvements.
  • The CSort class has been slightly reworked, and now it is possible to do sorting using complex expressions and composite fields .
    This change affects the upgrade procedure, since the attributes property has changed somewhat.
  • Added support for changing the primary key of an AR object using the save () method.
  • Added support for database table prefixes .

Along with the changes in the framework itself, there is also noticeable progress in the development of Zii (the official library of extensions for Yii). The integration with jQuery UI (JUI) was completed and a full set of widgets was made to facilitate the use of JUI plugins.

See changelog as usual for a complete list of changes.

About Upgrade


This version should not be used to complete the work of applications. Since this is still beta and some features may undergo major changes in future releases.

And if your websites with Yii work fine inside, you are not recommended to switch from 1.0 to 1.1 yet. However, if you still want to upgrade (or at least prepare for it), the instructions are prepared for you:
