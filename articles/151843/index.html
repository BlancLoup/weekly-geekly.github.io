Cloud php hosting in one click!

We are pleased to announce that the Jelastic platform is becoming a ‚Äúpolyglot‚Äù - in the fourth quarter of 2012 it is planned to launch PHP clouds. The unique feature of Jelastic PHP service is vertical scaling: you no longer need to overpay for large servers or suffer from a lack of resources. Due to this feature with increasing load, the application automatically receives the necessary resources, with a decrease, respectively - the opposite. This is really a new approach to cloud hosting: the platform is very simple and easy to use, so even beginners can easily deploy their applications, at the same time the supported functionality is very wide, so the platform will appeal to even the most advanced PHP developers.

You get huge benefits throughout the entire life cycle of the application (deployment / testing / production): after the application is loaded into Jelastic, you can test, scale and deploy it to production without any changes to the code, you can clone environments data, for troubleshooting or modification, update the application with zero downtime and determine a different level of access to the environment for each member of your team.




It is also worth highlighting such advantages as:
  • Horizontal scaling: with a few clicks of the mouse, you can set the maximum resource consumption limit for your application;
  • Once again, ease of use: no additional installations or configurations are required, as a wide range of software stacks are supported;
  • Versatility: you can easily deploy any PHP project (supported formats .tar, .tar.gz, .zip and .bzip2);
  • NGINX and Apache application server with all key extensions, if necessary, you can add any module;
  • Support for various SQL and NoSQL databases (MySQL, MariaDB, PostgreSQL, MongoDB and CouchDB) with the possibility of remote access;
  • Full access to all configuration files;
  • Ability to select a version of PHP;
  • Ability to deploy / update directly from git / svn repositories;
  • Plus built-in memcached node for efficient memory use.

I remind you that the free beta Jelastic PHP will be launched in the fourth quarter of 2012, and its commercial launch is scheduled for the first quarter of 2013. Thanks to the feedback of habravchan, we made the functionality of the Java platform exactly the way users wanted it to be. And the purpose of launching pre-beta, first of all, is the need to collect feedback on real problems that I would like to solve, to receive requests for important functionality for developers. So we invite everyone to participate in the beta testing. To do this, simply leave the email address on the page php.jelastic.com , and you will receive a notification about the launch of the platform.
