IRC channel and Jabber conference room

IRC.
A channel of the festival Chaos Constructions Antuque 2007 has been created in the RusNet network.

The channel is already working and is supposed to be the main means of communication for visitors during the festival (and some time after it).

Details:

channel name : # cc07
network : RusNet
Connection server : irc.run.net or any of those listed on the official RusNet site .

If you are using a UTF-8 client, you should connect to the utf.rusnet.org.ru server on port 7770. For the rest of the encodings, read the MOTD server carefully.

Jabber
The jabber room chaos_constructions was raised to the linux.spb.org server. Her full name is chaos_constructions@conference.linux.spb.org

You can join the room using any public jabber account.

The room is temporary, after the festival, if this service is popular enough, a full-fledged jabber server will be raised (maybe even with public registration - it‚Äôs not quite clear what we will have with iron for this).

The festival will run an internal jabber server. The parameters of which will be announced later. It will be possible to connect to it only from within the local network of the festival and only using the nickname that you have to register on it. These nicknames after the festival will not be transferred to the external server.

If you have problems connecting to the test room - immediately describe them in this topic - it is very important that we solve all the problems before the festival.
