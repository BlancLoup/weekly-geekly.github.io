Management and display

The main control element is the trackball, the piece is really comfortable, besides it is absolutely indifferent to the thickness of the gloves on your hands. On the sides of the trackball - large buttons, covered with silver, which over time can and after. The keyboard is full-size QWERTY, very convenient, from the additional keys - Alt , Shift and Sym (bol). From the left end is one customizable button, on the right - 2 to adjust the volume. Above - a power management button (long press) and a backlight (short presses - off , max . Auto ) and a button to go to Standby mode (the keyboard is locked, which helps when using non-original covers and holsters) - IMHO useless the presence of a good case. Right from the conversational speaker - light indicator, the modes are as follows:
  • red (call, message, calendar event)
  • blue (working bluetooth connection)
  • yellow (low battery)
  • green (indication of network coverage, in a simple way - constant blinking, disconnected).

Interface

We redefine with the help of themes, of which there are 4 types according to the set of icons on the screen in the standby mode:
  • Zen (customizable icon set, determined by the order in the first line of the main menu icons)
  • Today (lists of recent messages and calendar events are displayed)
  • Icons (in essence, without standby, displaying the main menu)
  • L (predefined dial on the left and bottom edges of the screen)
All of this varies in third-party themes. I use the standard Dimensions Zen, because no matter what topics I put, they did not change the top of the screen in the main menu - I don‚Äôt know whose bug it is, but that‚Äôs what‚Äôs right with me. The icons in the main menu are customizable, it can be hidden these and placement in folders. The menu button brings up the menu itself, the Back button - to go up a level or close an application (less often - to hide, depends on the application), the end button - to exit to the standby mode. The call button works according to its destination from everywhere (well, except for the Skype client, iSkoot , in which it also initiates a voip call to the contact selected in the list). When you try to enter from the keyboard - the search in the address book works, so to make a call you just need to start typing the name of a person from the standby mode or from the main menu. If the first character entered is a digit, a number input is available in addition. This behavior is customizable, in the alternative case, you can hang a quick application call on the keyboard.
