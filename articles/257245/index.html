Digest KolibriOS # 8: the gifts of spring

Spring is coming to an end, preparing to hand over its powers to summer. Well, we want to tell you about what was done in KolibriOS during this period.

System-wide changes (kernel, drivers, libraries):


  • Fixed a bug that caused the system to freeze when using the network.
  • 86duino GPIO driver
  • Truetype : acceleration of some functions.


Changes in application software:


  • FLI Player : work optimization.
  • uPDF : mouse wheel scroll; improved keyboard management (earned up, down, pg up / pg down, home, end buttons); implemented toolbar and the ability to go to the entered page; fix bugs.

  • Eolite : gesture management; displaying properties of selected items; the possibility of using an enlarged font.
  • WebView : download manager; 2x font increase; fix bugs.
    Hidden text



  • TTF Viewer : a viewer of TTF fonts.
  • TmpDisk : display the size of the created disks.
  • Dungeons : keyboard control fix.


Miscellanea:


  • Updated and added new icons.


In the process:


  • Work on improving IDE and SATA IDE drives (forum thread).
  • Write support in NTFS driver.
  • ECAD (CAD) for the design of printed circuit boards and electrical circuit diagrams
