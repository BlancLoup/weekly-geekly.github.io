New beta Yandex. Browser 14.2: improved download manager and view office documents

Today a new beta version of Yandex Browser is released , in which several improvements have appeared at once. In particular, the download manager has been significantly updated - now he knows how to recognize major errors and try to restore the download process, if possible. And also, the browser itself is now able to show the main office formats.



Full list of innovations:
  • New Download Manager
  • Office format support
  • Catalog of improvements for Yandex Browser
  • FastDNS
  • Blocking sites with SMS fraud


New Download Manager

We have already told that Yandex. Browser 13.12 learned to recognize errors that may occur during the download of files. This allowed for a more flexible approach to this process and to restore interrupted downloads if this is possible.

This version has changed the interface for working with downloads. Now there is no obsessive panel that pops up at the bottom of the screen and closes through the cross. The boot process is now displayed to the right of the Smart line. There is also a button for quick access to downloads.



To access recently uploaded files, there is now no need to open a full list page in the tab. It is enough to use the new section "Downloads" in the scoreboard.

Office documents support

Quite often, users download office documents in the browser just to extract some data from them, and then the file itself is either deleted or replenishes the collection of forgotten documents in a standard download folder.



Now Yandex.Browser for Windows can display not only pdf documents, but also files with the extensions .doc, .docx, .rtf, .ppt, .pptx (we plan to expand this list). In this case, your document will not be transferred to a third-party service - all work goes locally.

Improvement Catalog

Improvements are a new catalog in Yandex. Browser, which contains the extensions we recommend, switches for standard functions (for example, Turbo), as well as third-party extensions installed by the user.

From the variety of extensions can be identified several of the most popular categories, for example, blockers of intrusive and animated advertising. Now, any user can simply connect them by selecting from those that are recommended in the Yandex Browser. Recommended extensions are downloaded from our server, and any updates for them are monitored.



In the catalog of improvements, you can find not only recommended extensions or user-installed extensions, but also switches for Turbo mode and synchronization.

Improvements are our first step in providing the user with an easy and secure way to connect additional browser features.

FastDNS

For all users of this beta version, we are launching FastDNS technology, which should help the browser quickly obtain the IP addresses needed for navigation.

It works as follows: when opening a page, the browser sends a request to the FastDNS server, and in response it receives not only the address of the requested host, but also the addresses of some of the hosts referenced on the page. Information about links on the page is taken from a special extract of the search index. This response is cached in the browser and used for further navigation.

How is this different from the DNS prefetch that works in Chromium? The DNS prefetch is launched after the browser has received the body of the page, and sends on request to each host found. No more than 3 simultaneous requests are allowed. DNS prefetch continues to be used if there is no address for the host in the response from our server.

Normal local DNS queries also continue to be used in parallel with FastDNS in some cases. For example, if there is no data in the cache. We also understand that sometimes the answers of our server and the local one may not coincide (corporate segment, black lists). In this case, the local response takes precedence.

FastDNS will be tested in the beta version of Yandex. Browser. In a stable release of 14.2 full transition is not expected yet.

Blocking sites with SMS fraud

In May last year, a special panel appeared in Yandex. Browser that warned the user about the transition to a site suspected of SMS fraud. Unfortunately, most users ignore this warning and continue to enter their phone numbers. Therefore, starting from 14.2, such sites will be blocked. You can still download the site, but this will require consciously making your choice.

  • Updated morphology dictionaries
  • New design options, scoreboard
  • In the Smart beta line for Windows, you can see experimental buttons for copying the URL to the clipboard and for sharing on social networks.


Work on the innovations described continues. Not everyone can enter the stable release browser.

We remind you that the beta version is installed instead of stable and uses the same profile.

You can download the Yandex. Browser beta at browser.yandex.ru/beta .
