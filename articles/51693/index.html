Briefly about the main

Since the IPTV blog has opened, I think it is worth telling at first what it is, which I will try to do. Immediately I warn you that I will try to describe everything on the fingers in an accessible language :)

So what is IPTV?


As many have already guessed IPTV is television over IP. But the very concept of IPTV is more extensive, and includes a much larger number of services, but in order :)

How is broadcasting going? Broadcasting TB channels occurs using multicast. As already described in the next topic , multicast is a special form of broadcasting using IGMP. The channels themselves are usually broadcast via either the UDP protocol or RTP . One channel occupies an average bandwidth of 4-5 Mb / s. Now about the most interesting - there is an erroneous opinion that if one channel occupies a band of ~ 5 Mb / s, then if a subscriber has 100 channels, then he needs a band of 500 Mb / s - this is not so, multicast roughly eats exactly as much as it looks currently a subscriber. If one channel is currently being watched, then ~ 5 Mb / s will be allocated to it. Thus, the provider can broadcast a theoretically unlimited number of channels, in contrast to cable and terrestrial TV.

What does the IPTV complex include?



First of all, IPTV includes software called Middleware - distributed software, which usually includes a content management system, billing, statistics, and subscriber portal.

The second, but no less important component is the head station, which performs the task of broadcasting a TV signal to an IP network. The head station receives a signal from a satellite, ether or cable, if necessary, then decodes, decrypts and encapsulates the mpeg stream into IP packets.

What is the difference between IPTV and classic TV? This is primarily interactivity and services.

Interactive Services



I will list the most delicious of them.
  • EPG ( Electronic Program Guide - electronic TV guide ). In fact, the usual TV program.
  • VOD ( Video on Demand ) - video on demand.
  • PVR ( Personal Video Recorder - personal video recorder that allows you to record TV programs).
  • Pause Live TV - allows you to pause the channel.
  • information services such as exchange rates, weather, horoscope, jokes, etc.


Subscriber device



To use IPTV services on a TV set, an STB ( set-top-box ) set-top box is used that has an Ethernet input and outputs on a TV set. Controlled by the console from the remote control.

STB is essentially a miniature computer most often running Linux.
The user interface is implemented in different ways. This is usually HTML / Javascript with display via the browser built into the console, or Qt. Each of them has its advantages and disadvantages.

Total


In practice, IPTV services can be an alternative to home storage, dvd / bd player / media center providing at least similar services without additional headaches :)
