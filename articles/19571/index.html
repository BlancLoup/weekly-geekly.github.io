The creator of the curtailed "Vista" became a celebrity

Croatian fifth-year student Dino Nuhagiƒá (Dino Nuhagiƒá) wrote a program that reduces the size of the operating system Windows Vista to 1.4 GB. As you know, the full installation of "Vista" takes about 4 GB, and Microsoft recommends allocating 15 GB, mainly for storing completely useless files, including unnecessary drivers, language packs, help files, screen savers. Now you can safely get rid of all this.

The program called vLite allows you to choose which components of Vista to keep and which components to remove. After that, the program burns DVD with a truncated set of files. Fans of Nukhadzic compressed the installer to 515 MB, although after installation all the same 1.4 GB remain.

vLite is the second such program that Dino Nuhadzic wrote. Four years ago, he created a similar nLite utility to cut down Windows 2000 and Windows XP. According to him, that program was much more difficult to do, because the installation process in old versions of Windows was carried out by file, and the system gave an error if some file was missing from the distribution. Now there is no such problem. By the way, the old nLite program is still four to five times more popular than vLite, judging by the attendance of websites (programs have different websites).

Microsoft has already issued an official warning that in the reduced versions of "Vista" there may be problems with installing updates and operating system stability.

Dino Nuhadzic says that Microsoft recruiters have already contacted him regarding future employment, but he did not give his consent.
