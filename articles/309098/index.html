[St. Petersburg, Announcement] CodeFreeze meeting with PHP developer Dmitry Stogov about the internal structure of the PHP virtual machine

On Wednesday, September 7 , at 8:00 pm, the St. Petersburg office of JetBrains will meet with Dmitry Stogov , PHP compiler developer, Zend Technologies employee. The topic of the meeting is the internal structure of the PHP virtual machine and, in particular, the latest changes in PHP 7.



Participation, as always, is free. Registration - by reference . Limited number of seats.


About the report


The PHP virtual machine has various internal changes, but the most interesting are raising performance from version to version. Dmitry will tell about them, having paid attention to the last changes implemented in PHP 7 and brought two-fold improvement, and to the new ideas implemented in still not released versions.


The report will be of interest to all interested in the development of interpreted programming languages.


About speaker


Dmitry Stogov - Principal Engineer at Zend Technologies. One of the leaders in the development of Open Source PHP, the leading PHP developer, the leader of the PHPNG project who laid the foundation for PHP 7, is the author of several PHP extensions.


https://github.com/dstogov
