Redoing WinXP Home in Professional

Not so long ago I had to enter Windows Home Edition into the domain, but due to the curtailment of this system, it was not possible to do this using the usual Windows tools, and I came across an article on remaking Home into Professional. Redoing is to edit the registry ... now in order.

1. Open Registry Editor (Start-> Run-> regedit)

2. Go by key HKEY_LOCAL_MACHINE-> SYSTEM-> ControlsSet (002,003, etc.) Control-> Product Options
3. Remove the parameter ProductSuite
4. Create a Brand DWORD Type Parameter
5. Close the Registry Editor and reboot.
6. When loading, press the F8 key (to display the menu for selecting the type of loading)
7. Download "Last successful configuration"
8. We work in Microsoft Windows Professional (Even the inscriptions have changed)

The above procedure allows you to enter the machine into the domain, share resources and assign rights to them.

Some functions (services, programs, etc.) will remain at the level of the old system, because they are absent by default.

Ps. by analogy, you can convert Win2000 Pro to Win2000 Server
