Trillian for Android is available for public.

Less than a month after the announcement of Trillian for Android, as the first beta version became available to all comers.
You can familiarize yourself with the features of the IM client and, of course, you can download it here .


A short list of features of the messenger:
  • Support for Facebook Chat, Windows Live, Yahoo !, AIM, ICQ, Google Talk, Jabber / XMPP and MySpaceIM
  • Synchronization of accounts, contacts, avatars, statuses and other things (message history?) With Trillian versions for other platforms
  • Integration into the Android notification system
  • Sort contacts, support groups, metacontacts and more
  • The interface is specifically designed for the Android OS
  • Support for devices running Android 1.6 and up

PS Trillian for Android only makes its first steps, so that serious errors and shortcomings are likely. If you are determined to report a found bug, go here: bugs.ceruleanstudios.com (username and password: beta)
