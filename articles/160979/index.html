Mini PC Raspberry Pi Model A for $ 25 has already been put into production



Raspberry Pi Foundation since the release of its mini-computer Raspberry Pi Model B has already sold hundreds of thousands of such devices. The price of this model is 35 US dollars. But if you remember, the company promised to provide mini-computers for $ 25, and it was about the Raspberry Pi Model A that the habrasoobschestvuy heard for the first time. Now the company fulfills the promise, and the Raspberry Pi Model A is put into production. In other words, at the very beginning of next year, everyone will be able to get this model (the first devices have already left the assembly line).

This is how it looks (photo taken by developers). Of course, this model lacks some modules and functions (for example, there is no Ethernet port, only one USB port, and only 256 MB of memory). There are positive moments. For example, an ARM 11 processor with a 700 MHz core clock frequency can be overclocked to 1 GHz without loss of warranty. In addition, to the USB port, any user who needs the Web can connect a WiFi module.

If you remember, the Raspberry Pi Foundation originally conceived its computer as a way to provide British students with cheap PCs. And in fact they provided, only now the Raspberry Pi computers have become extremely popular not only among students, and not only in the UK.
