askdev.ru - life after habraeffekta


I would like to tell you what happened to the site after the recent publication here on habrahabr.
  • What's new?
  • What is habraeffekt (everyone has it, but I will tell about ours)



The publication about our project caused a heated discussion, in total more than 200 comments were left, many of which carried useful feedback on the site‚Äôs work. There were good and bad reviews, but for us they are all excellent because they help to make askdev better.

What has been done in these six weeks?


  • The site design in general remained the same, but the types of questions and answers were changed to avoid confusion and improve the perception of information.
  • Gravatar support has been added, after which most of the users of the site have avatars.
  • Removed the gradient in the preview of the question and the "incomprehensible" background in the website's header.
  • Added sorting on the tags page by alphabet and popularity
  • Arrows of voting for questions and answers have become brighter, so as not to "burden" the left side of the site.
  • Account and profile is now one page where you can see your questions / answers and edit personal information.
  • The profile has a section where you can view all your comments.
  • Fixed bugs in checking the username and when changing the password (were not critical and did not interfere with the work of the site)
  • Changed URLs for questions, added URL question URL (search speed in the database by ID is of course higher than the long line)
  • Reworked tag cloud, excludes all tags used only once, and the build algorithm is now logarithmic
  • Updated user list page
  • Layout is fixed, now there is no horizontal scrolling on 1024x768
  • Now you can not vote if there is no minimum rating (10 points)
  • RSS now displays the full text of the question (previously abbreviated)
  • Added widget of choice of tags of interest / ignored
  • Now you can edit your answers and questions at any time, because the change history is now recorded and available for viewing. In the future, it is possible that users with high ratings will have the opportunity to edit other people's answers / questions as on stackoverflow, but up to this point you need to go further. The Discussions tab has been added to the new profile where the feed of answers and comments is posted to the user
    Small link to send a question to twitter


    Of course, this is not the whole list of changes, because it makes no sense to list all the details. Most of the changes are already small, but on the whole they gave quite a good result.
