Messenger embedded in your site

In this post I want to talk about a new product that I used, but I am not its creator. At one point, I had the need to develop an instant messenger, like the Facebook messenger, for one of my sites. The task was not trivial at all. I did not reinvent the "bicycle" (the illness of many project managers and developers) and found a ready-made solution Imsgr.ru , with which you can install the messenger on any site.



All you need to do to install the messenger on your site is:

1. Go to the messenger site
2. Get PHP code
3. Paste it on the site

The messenger will be displayed directly inside your site (not in a separate window, but inside) in the lower right corner. It can also be minimized and expanded. Here's what it looks like:



In order for users to learn about the new "feature" of your site, it is enough to place a special code that will display an envelope picture with a link. Clicking on it opens a window of correspondence with this user (and it is automatically added to the contact list if it is not there yet).

The design of the messenger is neutral. The messenger is still in the alpha stage and is distributed for free, so you can download the js file and change the design.
