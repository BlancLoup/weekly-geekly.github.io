The new application for Windows and Mac establishes a permanent connection to the Kerio Connect server, allows displaying notifications about new letters, chats and events, and can be used as the default email client.

Kerio Connect 9.2 also allows iOS users to set up an account in the popular advanced email application Spark from Readdle using the built-in Kerio Connect profile, thereby helping users focus on important emails and spend less time sorting less important emails.

The new release introduces integration with MyKerio, a centralized management tool that simplifies the administration of several deployed Kerio services, including Kerio Control, Kerio Operator, and Kerio Connect.

After the release of the MyKerio app for the iPhone and Apple Watch at the beginning of the year, Kerio Connect 9.2 also introduces the MyKerio app for Android, providing real-time monitoring and notifications for all connected devices.
