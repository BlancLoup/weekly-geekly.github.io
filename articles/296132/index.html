Checklist of 68 points for website promotion in TOP10 Yandex

In order for Yandex to show your site among the first, it is necessary for it to respond to the user's request as completely as possible. And it will be half the success. The remaining half is convenience, the technical component of the site and its simplicity. Below I gave a checklist of 68 points, which was developed specifically for the PTYSH task management system, in conjunction with the DFAKTOR company. And now I share it with you. The painstaking execution of each item from this checklist will bring virtually any site to the top of the search results. But it is necessary to work as it should. Let's start with the smallest.



Site preparation
  1. Make a backup of the site
  2. Install the Yandex Metric site (http://metrica.yandex.ru)
  3. Add a site to Yandex Webmaster (http://webmaster.yandex.ru)
  4. Add the physical address of the organization in Yandex Webmaster
  5. Specify a region in Yandex Webmaster
  6. Specify the activities of Yandex Webmaster
  7. Add a logo and view of a building from the street (http://sprav.yandex.ru/org/add/)
  8. Check the address and phone for uniqueness in Yandex search (http://yandex.ru)
  9. Check the address and phone for uniqueness in Google search (http://google.com)
  10. Check website for affiliates
  11. Check all site texts for uniqueness.
  12. Set the register of the site name in Yandex Webmaster (http://webmaster.yandex.ru)
  13. In Yandex.Webmaster, specify the main mirror of the site with www or without www
  14. Define the main site mirror and configure .htaccess (301 redirect from www / without www)
  15. Check availability of pages link-dumps on the site if available, remove
  16. Check website for viruses
  17. Configure the CNC. The address of the pages should reflect the structure of the site (http://dfaktor.ru/prodvigenie_saytov/portfolio/)
  18. After setting up the CNC, set up and make 301 redirects from old pages to new ones
  19. URLs on the site should be short
  20. Make the Contacts page use the schema.org markup
  21. Make a block with contacts through on all pages of the site to use the layout schema.org in the most prominent place in the header of the site.
  22. Phones of the company must be larger than the address, the city code must be present. Specify working hours and days off
  23. All texts should be checked for spelling, stylistic, punctuation errors in the text (http://webmaster.yandex.ru/spellcheck.xml?checkurl=)
  24. Add an organization to Google My Business (https://www.google.com/local/business/)
  25. Add a site to Google.Webmaster (https://www.google.com/webmasters/tools/home?hl=ru)

Work with queries
  1. If possible, replace the domain name (with the entry of a key request, such as zakazat-prodvigenie-sayta.ru, etc.)
  2. Split requests into subject groups
  3. Analyze competitors by content: number of text, number of requests, presence of subject pictures on pages, Lists of listings, video on a page being promoted, what is in title, keywords and description, which buttons, which information blocks, what texts are about.
  4. Select for each request the most appropriate page for promotion, if there is no page, create one.
  5. Based on the data obtained, create an ‚ÄúIdeal‚Äù page for each request. (Title, H1, keywords, description, texts, cleaning of unnecessary code and unnecessary characters, adding buttons, information blocks, videos, etc.)
  6. Put yourself in the place of the visitor, check the usability of the site being promoted and the completeness of the information.

Website promotion
  1. When promoting complex queries, make links from the main page to the advanced one or from the menu, at least a footer. (requests that are not promoted to the main page)
  2. Create sitemap.xml
  3. Sitemap.xml add to Yandex Webmaster (http://webmaster.yandex.ru)
  4. Create / Edit robots.txt Be sure to take into account the features of the CMS, specify the main mirror and siemap.xml
  5. Make an HTML sitemap
  6. Check for pages with 404 error if necessary create
  7. Make all internal links relative (reduce the number of code)
  8. Make favicon.ico
  9. Close external links and counters from indexing
  10. If possible, remove ratings and counters, leave only one. (e.g. Yandex Metrica or Google Analytics)
  11. Place on the site the legal name and details of the company INN, OGRN, etc.
  12. Make a Yandex map and an interactive location map (http://api.yandex.ru/maps/tools/constructor/)
  13. Make a page "Reviews"
  14. Make the page "About the Company": from what year? What have been achieved? Photo certificate and certificates
  15. Create an object on a folk map (http://n.maps.yandex.ru/)
  16. Register a site in Yell.ru
  17. Register site in spr.ru
  18. Register site in 2gis.ru
  19. Register a site in zoon.ru
  20. Check and correct gross errors on the site by the Validator (http://validator.w3.org/)
  21. In the template page with useful information and articles add the button "Share" social plugin "Like" (http://api.yandex.ru/share/ vk.com/developers.php)
  22. Add a tweet button to your site if appropriate (https://about.twitter.com/resources/buttons)
  23. If it is advisable to install the Comments plugin (http://vk.com/developers.php)
  24. Check server response code for 404 page and promoted (http://mainspy.ru/otvet_servera)
  25. Make a FAQ section for frequently asked questions or by analogy with competitors
  26. Analyze competitors' sites for services, usability, info-blocks, menu items, presentation of information and other obvious positive aspects of sites. Consider how you can increase user confidence in the site, ease of selection, range, design.
  27. Create an official Twitter page if not yet (https://twitter.com/signup)
  28. Create an official Vkontakte page, if not yet (http://vk.com)
  29. Create an official Facebook page if not yet (http://www.facebook.com/pages/create.php?ref_type=sitefooter)
  30. Add links to official pages in social networks to the main site
  31. Add links to social pages in Yandex. Reference (http://sprav.yandex.ru/org/add/)
  32. Analysis and elimination of errors of indexed pages in Yandex (http://webmaster.yandex.ru)
  33. Check the site for the absence of duplicate pages in the index
  34. Analysis of the generation of urls (url pages promoted should strictly reflect the structure, without a set of characters and numbers)
  35. Analyze the behavior of users on the site, to understand the reasons that may not suit them.
  36. Check requests for which there are "failures", find out the cause and fix
  37. Prepare articles that will help answer questions related to the activities of the site (How to choose wallpaper? How to glue the wallpaper? Wallpaper from which material is better to choose?)
  38. Make the site lively, constantly posting useful thematic articles, news of the field of activity and other useful materials.


The main task in promoting the site is to understand how to improve it for the user and do it. Cheating the search engine is becoming increasingly difficult, investing in improving the properties of the site is much more efficient. Now the main tool is time and thoroughness of analysis.

There are no more secrets left in SEO. To bring the site to the top you do not need to take a tambourine, buy links and cheat behavioral factors. The mechanism of website promotion in theory has become obvious and simple. The only difficulty is that, in practice, the work on the removal of the site in the TOP 10 began to take much more time and effort. An SEO specialist is no longer one who has sacred knowledge and is supposedly on a short foot with Yandex, but a person who can analyze well and thoughtfully.

I really hope that the article was useful and many will take it to the note, and maybe part of it, just fill up their checklists.
