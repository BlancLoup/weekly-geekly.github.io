1. Unity for Windows Phone.


In the framework of the Build conference, Microsoft and Unity will continue to cooperate in the game direction . In addition to Windows and Windows Phone, Unity will also support Xbox development in the future.

Currently, with the release of Unity 4.2, the final support of applications for Windows 8 and Windows Phone has been added. The addon is available in both the free version of Unity and free of charge as an addition to the pro version. On the Unity website, you can find a gallery of applications for Microsoft platforms using the game engine: unity3d.com/pages/windows/gallery

The companies also launched a joint application contest with a total prize pool of $ 100,000: unity3d.com/contest/windows

Extras: on Channel 9, you can find an overview report Introducing Unity for Windows Phone Development .

2. Update examples and toolkit


The second important practical news for those who learn from examples is the update of examples in our code gallery. An overview of the June updates can be found here: blogs.windows.com/windows_phone/b/wpdev/archive/2013/06/11/windows-phone-8-samples-update.aspx .

In July, we added some more examples: Free App with Paid Products (using embedded payments), Environment Recipes (collecting information about the environment and the device), Passing Data between Pages (methods of transferring data between pages) and Periodic Background Agents for Tiles and Toasts ( using periodic background tasks). Details and links here: blogs.windows.com/windows_phone/b/wpdev/archive/2013/07/17/windows-phone-8-samples-july-update.aspx .

There was also an update to the Windows Phone Toolkit , in which localization support was significantly updated, an example of using gestures without using XNA assemblies was added, a new Speech Text Box control appeared, as well as minor bug fixes.

3. Nokia Imaging SDK


In July, along with the launch of Lumia 1020, Nokia released a special SDK for working with images on Windows Phone 8. By the way, the SDK supports all Windows Phone 8 phones.



The SDK is a WinPRT library with which you can work from both C # or VB and C ++ code. The library includes more than 50 filters, effects and enhancements for applications, including auto-enhancement, brightness control, RGB levels, HUE, etc. You can also combine them to create your own. Additionally, you can use the library to crop, scale, and rotate images. Another interesting feature is the partial decoding of JPEG images, which allows you to quickly get previews, the imposition of effects, turns, etc. for large images.

Download the SDK and learn more at Nokia: developer.nokia.com/Develop/Windows_Phone/nokia_apis/imaging.xhtml

4. AWD SDK


Finally, quite the latest news: Amazon announced support for Windows Phone and Windows Store applications in its Amazon Web Services SDK, however, so far in the form of a Developer Preview. You can download here: aws.amazon.com/sdkfornet
