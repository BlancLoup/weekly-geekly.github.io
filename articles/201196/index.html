Have fun together: Single sign-up to the Windows app stores and Windows Phone


We are pleased to announce that since November 6, 2013, a single registration for developers has been implemented for Windows and Windows Phone application stores! The cost of a single registration has become significantly lower. Now all developers with an account in the Windows Phone store can publish applications in the Windows store, and, of course, all developers with an account in the Windows store can publish apps in the Windows Phone store!

How will this all work?

Registered in the Windows store developers can now publish applications in the Windows Phone store, using the same Microsoft Account, without the need for additional payment.

Developed in the Windows Phone Store, developers can now publish apps to the Windows Store using the same Microsoft Account, without the need for additional payment.

New developers can register or update existing registrations using the same Microsoft Account. When registering, developers will enter registration information only once and pay only once - $ 19 for individual developers and $ 99 for companies . For individual developers, the cost of registration is significantly lower, and for companies just lower. After registration, developers will be able to publish apps in both the Windows app store and the Windows Phone app store.

Developers already registered and in the Windows app store and Windows Phone with the same Microsoft Account will receive an email code this month that can be used to update the current registration when it is required.

Student registration continues to be provided through the DreamSpark program (and does not require payment)

As before, different publishing and management centers should be used for publishing and managing applications in different stores.
