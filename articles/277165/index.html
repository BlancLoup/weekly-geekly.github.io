3CX integration with Office 365

Special 3CX plugin allows you to connect 3CXPhone (this is a softphone) with contacts from Office. He can show the caller's information in the Office 365 card, save the new contact if the number is not found and keep statistics on the calls. Outgoing calls for Office 365 can be made using a hot key combination. Simply select the number and press the desired buttons, the number will be dialed through 3CX Phone automatically.




System requirements


  • Any commercial version of 3CX Phone System v14
  • Installed and running 3CXPhone for Windows
  • Office 365 account

Install the plugin for Office 365


Download the distribution here:
http://downloads.3cx.com/downloads/3CXPlugins.exe
Select Microsoft Office 365 and follow the instructions.



After the installation is complete, run the 3CX Plugins Manager to configure the plug-in.

Customize Office 365 Plugin




The Office 365 plugin settings are available in the General section. Here you can configure exactly when to show the contact card (when calling, after answering, or not showing at all), you can limit the length of the Caller ID to which the application will respond (for example, do not open for internal three-digit numbers).



If you need to register the fact of a call with notes to a contact, enable Enable Journaling . Logging will add a new line to the contact notes every time after the end of the conversation. Also, you can create a new contact if Caller ID is not found, to do this, select the appropriate option.

In the Behavior section, you can select the number of compared digits in the number. This is necessary for cases when the same number can be presented in a different form depending on the service provider or format (domestic, international). The comparison is the last X digits of CallerID with the last X digits of the number in contact.

Office 365 setup


Here you need to fill in the authentication information. Enter your email and password.



The Contact Lookup section allows you to customize the display of information about an incoming call in the default browser or any other.

Incoming calls



When an incoming call and a match from the contact list, the card will automatically open in the browser.



If the Enable Journaling option is enabled, after the end of the conversation, information about the call will be added to the notes.
If the number is not found and the option Create new contacts if not found is enabled, a new contact will be created.



Calls from Office 365


For outgoing calls from Office 365, you need to use a hot key combination. This feature is already built into 3CXPhone for Windows . And the Office 365 plug-in itself does not add any functionality.

Simply select the number with the mouse, and press a combination of buttons, the number will automatically be copied to the 3CX Phone and dialed, thus removing all unnecessary garbage (spaces, dashes, plus, etc.).
