2.) Urti Search Engine



The aim of the project is to create a service that allows you to simplify the search procedure as much as possible throughout the Internet and on a specific resource in the document. And give the found information in the form of an exact answer to the request.

Example 1. A user enters a query: ‚ÄúA thin mobile phone is needed, lightweight, with a 9 inch screen, with built-in GPS‚Äù, the system selects suitable phones with all the properties: specifications, reviews, prices, availability in stores, and so on.
