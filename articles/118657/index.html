Ciklum .NET Saturday - in Kiev this Saturday

May 14, traditionally, the company Ciklum opens the door for the next work day.

Ciklum invites .NET application developers to this meeting. Subbotnik entry is free for registered participants



iklum Saturdays are events created especially for developers to get acquainted with leading experts in the field of software development and with each other.
