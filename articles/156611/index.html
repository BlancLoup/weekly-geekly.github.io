Valve: Linux is preferable for games than Windows 8



Drew Bliss (Valve) said in a speech at the Ubuntu Developer Summit that Linux is a more acceptable platform for gaming than Windows 8.



Some key points of the presentation:

- Native Steam works well on Ubuntu.
- Collaboration with Canonical was very productive.
- Linux has everything you need - good OpenGL, OpenAL, PulseAudio. Well, on the Linux side a large community.
- For Linux, new Source games will be available soon.
- For game publishers there is a copy protection feature.

I remind you of beta testing Steam for Linux.
