Software Freedom Day / Saratov

We invite everyone to a holiday of free software!


Date: September 19, 2009
Venue: Saratov, SSU them. N.G. Chernyshevsky, the 12th academic building (Volskaya 10a, intersection with Beloglinskaya).
Duration: 10: 00-18: 00
Our website: sfd.sgu.ru

Software Freedom Day is a worldwide free and open source software festival held annually on the third Saturday of September on the initiative of the non-profit organization Software Freedom International. Volunteer teams around the world organize local Software Freedom Day celebrations to attract public attention.

Program


10:00 Meeting guests. Exhibition handout materials.
10:30 Introduction. A story about the history of Software Freedom Day. Communication with participants.
11:30 Break. Preparation for presentations and master classes.
I-th block of reports and master classes.
Topics for presentations:
12:00 - Practical experience Solving software licensing issues. (Jacob Popov)
12:30 - Python - an overview of the main features. (Yu.A. Blinkov)
13:00 - Organization of parallel computing using open source software. (V.A. Podchukayev)
13:30 - Russian-speaking community and LoCo team Ubuntu. (Dmitry Agafonov)
Themes of the master classes:
12:00 - Bluetooth + Linux + Phone. (Ekaterina Polyakova)
13:00 - Building GUI applications using PyQt4. (Yu.A. Blinkov)
14:00 - Break.
II block of reports and master classes.
Topics for presentations:
15:00 - Preparation of system administrators of Linux-systems. (V.M. Solovyov)
15:30 - Rapid development of web applications using Django. (Leonid Shvechikov)
16:00 - LiveCD - unlimited possibilities. (Ilya Zakharenkov)
16:30 - OpenSIPS - a key component of SIP-based VoIP solutions. (Ivan Khvatov)
Themes of the master classes:
15:00 - TrueCrypt. Secure data storage.
16:00 - Rapid development of web applications using Django.
17:00 Closing of the festival. Distribution of materials.

Throughout the festival, an install-fest will be held in the lobby, where everyone can install Linux OS or free software for Windows OS on their laptops.

ATTENTION! In case of installing Linux OS, take care in advance about saving important information from the system partition of the hard disk of your laptop !!!
