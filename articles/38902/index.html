MDC: beta release of a new multi-protocol messenger

Today I will talk about MDC - the new project of NetStream . MDC is a multi-protocol messaging client with support for Windows, Linux, MacOS X and ICQ, Mail.Agent, Jabber, AOL protocols (we will gradually expand the list of protocols).


The first launch starts with the creation of a personal profile, where the program will put all the basic settings and message history.

After logging in, the contact list window you are accustomed to opens. You can change your status by clicking on the colored vertical strip. Here you can change your nickname, which will later be displayed in the message window.
Also, in the upper part of the contact list window, we selected the 3 most requested options: search and add a contact, display contacts by groups / without groups and display offline statuses:


Overview of MDC Benefits


  • Storing the history of correspondence on the server . No need to think about where you were: at home, at work or with friends, you can always find the necessary information, having access to the Internet. I would like to immediately note that this ‚Äúopportunity‚Äù can always be turned off in the program settings.
  • Portability Today, MDC beta is available only for Microsoft Windows (32-bit / 2000 / XP / Vista, 32-bit). In the coming weeks, we will release a version for Linux (32/64) and Mac OS X (Intel). After a month and a half, the web version of MDC (lite-client version) will be available.
  • Support basic messaging protocols . Now MDC supports protocols such as ICQ, AOL, Mail.Agent, Jabber (plus Gtalk and Ya.Online). In the near future, a beta version of MDC will be released with support for the MSN protocol. Of course, with each new release we will try to add the protocols most requested by users.
  • Merging contacts (metacontacts) . This is another useful feature that allows you to combine the contacts of the same person from different messaging systems, which allows you at least not to think about what protocol your interlocutor uses.

  • Auto update . MDC allows you to automatically update the program. You do not have to download and install the client every time to take advantage of new features, improvements and corrections of MDC. And in the case of a change in any protocol, you simply won't notice it.


Other MDC Features


  • the ability to save settings on the server;
  • use of advanced statuses and status pictures;

  • notifications (prints to you, etc.);
  • search and add contacts;
  • full and abbreviated type of contact sheet;

FAQ


Assuming a large number of questions from Habraludey, we will try to answer the most pressing questions.
  1. And why is the version only for Windows?
    MDC is a portable program, after additional internal testing (in about 2-3 weeks) we will release a version for Linux & Mac OS X.
  2. This is not open source! They will make it paid, as soon as the user base is typed!
    Yes, today MDC is distributed only in compiled form (without source code). Perhaps we will open MDC fully or partially in open-source. Well, and certainly the program will never be paid.
  3. The central server collects user passwords!
    MDC does have a central server where the user history, program settings, etc. are stored. It provides the ability to move the user between computers without losing information. but
    The server does not receive passwords to your contacts, only the list of your accounts and their settings are saved there. In the next release we will make it possible to protect additional information on the server with a password (today chatback is used for authorization).
