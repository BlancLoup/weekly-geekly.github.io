Server sales are growing thanks to the web

Server sales grew by 8% in I quarter. 2008 compared with the same period last year. In three months, almost 2.3 million servers worth $ 13.6 billion were sold worldwide.

This market is clearly correlated with the state of the Internet economy. The most successful web projects are now actively building and renting entire data centers, and the number of servers in them amounts to tens of thousands. The largest software makers, like Microsoft, consider building an extensive network of data centers around the world one of their main tasks. This is not surprising, because all the software is slowly but surely migrating to the web platform. According to the vice-president of the analytical company Gartner, the new data centers for web applications are the main generator of market growth.

On the general rise, only the RISC-Itanium Unix server sector looks sad, which decreased over the period by 8% in the number of servers (although it grew by 4% in financial terms).

The first two places, as before, are shared by Hewlett-Packard and IBM. At this time, HP burst forward, overtaking a competitor by 0.7%. However, almost everyone else is also gaining momentum. Among manufacturers, a decrease in revenue was noted only for Sun
