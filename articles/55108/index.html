Image upload plugin for TinyMCE

A few days ago I wrote about the release of a new version of my plugin for TinyMCE, many have already managed to test it, for which many thanks to them, all critical bugs have been fixed.


Unlike analogs, the plug-in interface is quite pleasant:



There are 2 download modes: normal and using flash



List of main features


  • Download up to 5 files at a time, without using Flash Player
  • Multiboot using Flash Player
  • Convenient folder navigation, creating / deleting folders
  • Displaying information about the file (in the bottom panel)
  • File name change
  • Summary information for selected files
  • Automatic insertion of links to the original image (if it is more than the maximum allowed)


And, of course, the most interesting is a demo .
You can view information about used libraries and scripts, as well as download the plugin on the project page .
