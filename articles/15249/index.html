New version of web browser for developers has been released.

Today, a message has been received from the Land of the Rising Sun about the release of a new version of the free web browser Sunrise, aimed primarily at web application developers and web editors.


Sunrise is built on the WebKit engine used by Apple in its Safari , and its main highlights are the quick resizing of the window for the main screen resolutions (640, 800, 1024), translucent windows of the web page and / or source code, exporting the page in the format PDF. And although most of these functions can be easily found in ordinary browsers, here they are very conveniently integrated into the interface.

Of the other features of Sunrise: graphical bookmarks, quick copying of the page address, the ability to save site icons, upload files to the specified address.

The latest version of the browser has eliminated various minor bugs, including those that cause memory leaks.

System requirements: Mac OS X 10.3 and higher.
Terms of distribution: freeware.
