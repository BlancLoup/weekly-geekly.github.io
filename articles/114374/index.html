PHP training package on Windows and SQL Server Training Kit

Microsoft has updated the training package for PHP developers who host their sites on Microsoft IIS and SQL Server.

This package contains the following laboratory work:
  • IIS 7.5 Access Control Features for PHP;
  • Updated: Using IIS 7.5 Media Features in a PHP Application (contains a new exercise on the Smooth Streaming technology);
  • Troubleshooting PHP;
  • Migrating PHP Applications to IIS 7.5;
  • Introduction to Using SQL Server with PHP;
  • Using SQL Server Full-text Search and FILESTREAM Storage with PHP;
  • New: Getting Started with SQL Server Migration Assistant for MySQL;
and the following demos: Spatial Demo and SQL Server Reporting Services Demo.
