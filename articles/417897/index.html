News and Releases


  • PHP 7.3.0 alpha 4 - The last alpha in the loop. The first beta release is scheduled for August 2. To the list of new features will be added the recently accepted proposal to introduce new functions array_key_first() / array_key_last() for working with arrays: 

     $array = ['a' => 1, 'b' => 2, 'c' => 3]; $firstKey = array_key_first($array); // 'a' $lastKey = array_key_last($array); // 'c'

    The voting phase passed the proposal of Deprecations for PHP 7.3 , in which several features are declared obsolete. Also, case-insensitive constants will be declared obsolete in PHP 7.3.

    As for the proposal on typed properties , it was decided to postpone it until the next version, which is likely to become PHP 8.0.
