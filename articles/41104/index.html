Pay attention


Often it is necessary to draw the user's attention to any part of the interface.

For example, if there is an error in entering data into the form. Usually webmasters use color highlighting or text errors, and usually this helps, but there are particularly slow-witted users who will not find their mistake and just simply leave your website. And there are many such situations.

I want to show you a technique that allows you to pay attention to the user, in my opinion, with great accuracy.

This technique works on the basis of the jQuery javascript library, and how not pity it will not work in a browser with javascript disabled.

Selection of the desired element occurs by dimming the rest of the content (and, if desired, blinking), that is, the user will see only the desired element.



This script, as the developers assure, works in all browsers. I would also like to note that this solution is very thoughtful, I especially liked the support of scrolling to the right place on the page.

I advise you to see examples of this script at its office. site.
Visit the office. plugin website for jQuery "seekAttention"
