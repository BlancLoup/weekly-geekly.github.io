Correo: email client for Mac based on Thunderbird code with native interface

Nick Kreeger , one of the contributors to the Camino project, recently released another version of his brainchild - an email client based on the Thunderbird code with a native Mac OS X interface called Correo (which means mail in Spanish). So far, the mailer has only the most basic features, but the author plans to develop an API for writing plug-ins, tabs, address book support and other improvements.
