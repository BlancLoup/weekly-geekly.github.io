View Photoshop brushes

When a collection of brushes grows to several gigabytes, it becomes difficult to understand. Googling, found a program to view brushes abrViewer.NET . Having rummaged in it, found several flaws that I decided to fix. Unfortunately, the author did not respond to the letter, so it was not possible to place his release on sourceforge.net.

List of changes:
  • Added anti-aliasing thumbnails
  • Saving settings (size of thumbnails, colors, background alternation)
  • Save all paths so that each time you do not specify the directory or path to the file
  • Fixed a bug with empty space at the bottom of exported PNGs
  • Automatic substitution of the exported PNG name (from the name of the brush file)
