Firefox 2.0 has been released codenamed Bon Echo

The first beta of Firefox 2.0, codenamed Bon Echo, is uploaded to official FTP for free access. Here are direct links to distributions for Windows (5.4 MB), Linux (8.9 MB) and Mac (17 MB). Localized versions of Firefox 2.0 Beta 1 in Russian are available from this page .

The grammar checker in the Firefox browser is done in much the same way as in the MS Word text editor. Incorrectly written words are underlined by a red wavy line. When you click on such a word with the right mouse button, a context menu appears where you are prompted to add the word to the dictionary or replace it with another word ( screenshot ).

In the new version of the browser, extensions and themes are combined under the general concept of ‚Äú add-ons ‚Äù ( add-ons ). Thus, it is now easier to manage and update them ( screenshot ).

As the developers announced in advance, the most popular extensions that best recommended themselves will be built into the second version of Firefox. A typical example is the function of restoring tabs after a browser crash. Now, if the computer or browser is frozen, then after the restart, you can open all web pages that were opened in the browser at the moment of ‚Äúfreezing‚Äù ( screenshot ). Recently closed tabs are also listed in the History - Recently Closed Tabs menu, or you can simply right-click on any tab and select the "Undo Close Tab" command from the context menu ( screenshot ).

In the second version of the browser it has become much easier to manage the list of search engines that are available through the search bar. Now they can be easily edited: interchange, delete and add new ones ( screenshot ). Directly from the search panel is now available auto-complete feature, similar to Google Suggest ( screenshot ).

Another new feature - the anti-phishing filter, which was previously integrated into the Google Toolbar, is now integrated directly into the browser ( screenshot ). By default, Firefox checks the local list of dangerous sites without contacting the Google server, but you can change the settings ( screenshot ).

Now it's easier to subscribe to feeds. When you click on the orange icon in the address bar, you can now automatically add a feed to a third-party RSS reader: Outlook, MyYahoo, Bloglines, or any other ( screenshot ). To change the default RSS reader, you need to edit the browser settings ( screenshot ).

To test Bon Echo in fact does not necessarily give up the current stable version of Firefox. The beta version of Firefox 2.0 copies itself to another directory and does not attempt to become the default browser.
