NB! I tested it only on Ubuntu 16.04, but in theory, after a light modification, the file should work.

Install it is quite simple:

1. Method one, for lovers of deb package:

Downloading the deb package I built from here:
github.com/dargiri/xshot/tree/master/dist/debs (at the time of writing this article is xshot_0.1.deb ).

Then either double click (open the GUI with the package installation manager) or install everything from under the command line: 

sudo dpkg -i xshot_0.1.deb #   -   ,     : sudo apt-get -f install


2. The second way, for fans of the command line (or just those who are not based on Debian (not Ubuntu, not Mint, etc.) system. 
 sudo apt-get update \ && sudo apt-get --yes install xclip jq curl \ && curl https://raw.githubusercontent.com/dargiri/xshot/master/xshot > xshot \ && chmod +x xshot \ && sudo mv xshot /usr/local/bin/
