What is Greasemonkey and how will it improve your internet life?

Here is how Wikipedia writes about Greasemonkey: Greasemonkey is a Mozilla Firefox extension that allows you to add custom JavaScript to any page recorded in the format of this extension.
But do not be afraid of the perceived complexity, knowledge of JavaScript simple user is completely unnecessary. By putting the Greasemonkey plugin on Firefox, you will discover such an incredible amount of new improvements for online services that this plugin will remain on your PC for a very long time.
