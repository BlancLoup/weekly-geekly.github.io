Canon has released a 120-megapixel CMOS sensor

Canon has developed a digital camera sensor that has a physical resolution of 13,280 x 9,184 pixels. That is, a full-length HD video can theoretically be shot using only 1/60 of its area.

The sensor has the APS-H format, that is, the same format as in professional digital cameras EOS-1D, but one can hardly expect the appearance of a commercial version of these cameras with a new matrix. For example, Canon released the same 50-megapixel sensor back in 2007, but has not yet installed it in the EOS-1D (there is a 16 megapixel CMOS). Apparently, new developments are used in some other equipment or are generally made solely for experimental purposes.
