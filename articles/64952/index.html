Spelling checker for both Russian and English in OS X

The spell checker in OS X has one major drawback. There is no way to check the spelling of words in a mixed text. It is possible to check either the spelling of English words (Russian will not be checked), or the spelling of Russian words (English words will be underlined as words with errors). In the service settings there is an opportunity to select a multilanguage check, but for some reason the Russian language is not included in this group of dictionaries.
This topic describes how to enable spell checking in mixed texts.

Step-by-step instruction:
  1. you need to download and install the cocoAspell program, which is an adaptation of the good old aspell for mac os x;
  2. the next step is to download the dictionary with Russian and English words;
  3. put the unpacked directory with the dictionary in ~ / Library / Application Support / cocoAspell;
  4. select the Spelling item in the System Preferences application, select the Russian dictionary in the list of dictionaries (in my opinion, change the name for this dictionary, for example, RuEn to easily find it in the list of dictionaries) and close System Preferences;
  5. To check the result of the above actions, you can run TextEdit, select the ‚ÄúRuEn (Aspell)‚Äù dictionary in the spellcheck settings and try to write with errors in Russian and English.
