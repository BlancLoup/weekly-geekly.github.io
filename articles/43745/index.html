Nimbuzz - Skype, SIP and IM on any mobile

The Nimbuzz project, officially launched by the eponymous Dutch company in May 2008, is today one of the most actively growing mobile VoIP (mVoIP) and IM applications. At the Mobile 2.0 Europe Summit, Nimbuzz received an award in the category ‚ÄúBest startup in the mobile industry‚Äù. In October of this year, Nimbuzz introduced the Russian-language version of its mobile client.



Nimbuzz Mobile is positioned as an all-in-one solution, and allows you to use the services of the most popular VoIP and IM networks, including: ICQ, Skype, GoogleTalk, Jabber, MSN, Yahoo and AIM on almost any mobile device. Already, the list of devices supporting Nimbuzz includes more than 1000 phones, PDAs and smartphones running on platforms: Symbian, Java and Windows Mobile.

Nimbuzz Mobile users can make and receive voice calls on Skype, MSN, GoogleTalk and Nimbuzz networks, or use the services of any SIP operator. Using SIP and Skype calls to landline and mobile numbers to any subscriber from the phone's phonebook can significantly save money when communicating. Nimbuzz is a full-fledged mobile VoIP (mVoIP) client, as it uses the existing Internet connection to transmit voice traffic.

For work Nimbuzz Mobile can use any available connection to the Internet: Wi-Fi, 3G or GPRS / EDGE.

Nimbuzz also supports the transfer of chat messages, the transfer of files and pictures, the presence function (presence) and Buzz (Alert).

In addition to the mobile client, Nimbuzz allows you to use the application for a personal computer or use a unique web interface. The desktop version of Nimbuzz - Nimbuzz PC - is now available for Windows XP and Vista. Versions for MAC and Linux are in development. To work with a web-based version of Nimbuzz, it will be enough to launch a browser with support for the Adobe Flash player.

Another interesting feature of Nimbuzz is Nimbuzz Widgets . By installing Widgets on the pages of your personal site or blog, you will allow site visitors to call your phone or VoIP account at any time, send a text message and file.

All Nimbuzz applications have a very high quality of work, an intuitive and user-friendly interface.
