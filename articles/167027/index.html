A few words about how the timeline is made:
The online service timeline.verite.co is a tool for creating timelines, written in JavaScript (anyone can fork it on GitHub). Allows you to create lines using data from popular social resources Twitter, Flickr, Google Maps, YouTube, Vimeo, Dailymotion, Wikipedia, SoundCloud. Data is received in JSON and from the Google Docs table. In my example, I just filled in the Google-Spreadsheet template . Allows you to use HTML tags in cells. It is possible to look at it yourself.


Based on the above information, the life of Android can be divided into three generations:
  • First generation. From early versions to the release of "Eclair". Features: low demand, cheap iron, extremely high fragmentation of devices. About 100 thousand activations per day.
  • Second generation From Froyo to the first release of ICS. Features: a growing number of users, more productive devices, higher functionality. About 500 thousand activations per day.
  • Third, the current generation. From ‚ÄúICS‚Äù to an unknown border. Features: mainstream, excellent hardware, excellent user interface (compared to earlier OS releases - as noted by ninja ). About 1.5 million activations per day.
