IBM has released a new generation of storage - Storwize v5000 Gen2



Last month, Storwize v5000 Gen2, a new generation storage system from IBM, was announced. Now the range of systems includes 3 main models, these are v5010, v5020, v5030. Each of the models differs from each other in hardware characteristics and the presence or absence of certain functions. However, all models now support the ability to install FC 16Gb daughter cards, there is also the possibility of equipping any of the models with a SAS 12G host port (this is not possible with the V5000).

It should immediately be said that all three systems provide the basic functionality of IBM Spectrum Virtualize Software for Storwize V50x0 V7.6.1. Plus, the user can get several additional options. In more detail about characteristics of all three models - in continuation.
