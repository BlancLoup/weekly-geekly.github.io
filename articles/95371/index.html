An example of logging in to two accounts on the same site at the same time using Private Tab Opera

Many of us have several accounts on the same site. If you entered under one name, usually you will not be able to enter under another, because cookies are common to all tabs.
In Opera version 10.5 and higher , Private Tab appeared, with which cookies are not common, so you can access the site from two accounts at the same time. To call Private Tab, do the following: right click on the tab> New Private Tab


Other ways to do the same:
  • Install Opera in another directory
  • Create a new user profile for Opera.
