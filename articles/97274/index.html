New features and high performance

Completely redesigned and optimized for mobile devices, Flash Player 10.1 provides new ways to interact with support for mobile input models. Accelerometer support allows users to view Flash content in landscape and portrait modes. With smart zoom, users can comfortably scale content in full screen. Optimization of work for almost all major mobile devices makes efficient use of the processor and battery.

The new feature of "smart" rendering ensures that the content in Flash only works when it becomes visible on the screen, which provides even more reduction in the load on the processor and reduces power consumption. In sleep mode, Flash Player automatically slows down when the device enters splash screen mode. Advanced memory management (when it is not enough) will allow the player to efficiently process non-optimized content that consumes an excessive amount of resources, as well as automatically reduce the use of RAM by up to 50 percent. Flash Player automatically pauses when events such as incoming calls or browser transitions to other applications occur. As soon as users switch back to the browser, Flash Player will show where it left off.
