Ubuntu 10.04.1 LTS service update released



On August 17, Canonical released a service update for the Ubuntu 10.04.1 LTS "Lucid Lynx" operating system.
This release aims to fix all sorts of vulnerabilities and bugs.

The new version is available in all versions, for a home computer, for servers and netbooks.
Kubuntu , Edubuntu has also been updated.

Support for the desktop version of Ubuntu 10.04 will be available until April 2013, while the server version of the OS will be supported until April 2015.

Closed a huge number of vulnerabilities and fixed many bugs.
A huge list of fixes can be found on the wiki on the official Ubuntu site.

Download the updated distribution can be on the official site - releases.ubuntu.com
