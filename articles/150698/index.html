Release Ubuntu 12.04.1

Without fanfare and dust on August 26, 2012, 4 months after the release of the distribution, an update of Ubuntu 04/12/1 LTS was released.



The following products have been updated:

  • Ubuntu 04/12/1 (LTS)
  • Ubuntu Server 04/12/1 (LTS)
  • Ubuntu Core (LTS)
  • Edubuntu 04/12/1 (LTS)
  • Kubuntu 04/12/1 (LTS)
  • Mythbuntu 04/12/1 (LTS)
  • Ubuntu Studio 04/12/1 (LTS)
  • Xubuntu 04/12/1 (LTS)


Description of the changes:

Ubuntu 04/12/1 includes updates with the implementation of support for new equipment, eliminated errors in the installer and bootloader, as well as updated packages to eliminate vulnerabilities and errors in them.

It makes sense to install this assembly only to those who decide to reinstall the system; already installed systems will receive all changes present in Ubuntu 12.04.1 through the standard update installation system.

Support for the release of updates and security fixes for the server and desktop editions of Ubuntu 12.04 LTS will last until April 2017. The first notification will be displayed to users of the LTS branch 10.04 with a proposal to upgrade to version 12.04, since support for the desktop edition of Ubuntu 10.04 LTS will end in April 2013 (the server version 10.04 will be supported until April 2015).
