The official release of Kaspersky Anti-Virus 7

The official release of Kaspersky Anti-Virus 7 took place. As we were promised, the 7th version was released on the 7th, 7th month, 07th year. The build number is like the last technical release I wrote about a week ago, that is 124.

In the 7th version, the interface has been updated, a new heuristic analyzer has been added, a security wizard has been introduced to fix configuration problems, a news agent has been built in to get the latest news from the Lab, keylogger detection has been improved, protection against new types of leak tests has been added, an advanced search has been implemented quick installation program.

Kaspersky Anti-Virus is a product designed to protect personal computers from viruses and malware. The total functionality of the program includes: file, email and web antiviruses. If you need comprehensive protection for your computer and a simple antivirus for you, then it is recommended to pay attention to the version of Internet Security, which adds to the standard functionality of Kaspersky Antivirus: firewall, spam protection, anti-spyware protection and parental control system.
