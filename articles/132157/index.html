Olympiad in Java programming for students


Oracle company together with the organization It-Planet invites students of universities and colleges to take part in the International Java Programming Olympiad - Oracle Java Olympic. Competitions are held in Ukraine, Russia, Belarus and Kazakhstan and consist of three stages. For participation, registration of the university or college in the system is desirable, but personal participation is also possible. Registration for participation in the Olympiad takes place until November 30, 2011 at http://world-it-planet.org/ . To register on the first page, select "Oracle Java Olympic" and your region of residence.

The first stage is qualifying. Held remotely. After registration, participants receive a link to take the test and within 45 minutes answer 40 questions on Java technology. The test can be taken at any time from December 1 to December 15 inclusive. Each participant has only one attempt to pass the test.

Second phase
The winners of the qualifying stages in their university are invited to participate in a full-time regional round, during which participants receive 7 tasks, 4 hours of time and a computer with an operating system, JDK7, NetBeans 7.0.1 and Java API documentation. Participation in the competition is personal.

The third stage is the final one. It is held in a format similar to the second stage. The main difference is more complex tasks (and rivals of course).

Details and a detailed description of the competition, see the website of the Olympiad.
