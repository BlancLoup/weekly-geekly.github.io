An overview of specialized ways to circumvent locks on the Internet

  • Usability: 3/5

  • Selective proxying: Yes

Description :

settings page.

Does not help against Full DPI.

FriGate (Chromium-based, Firefox)

  • Usability: 4/5
  • Speed: 5/5
  • Traffic Encryption: No
  • Anonymity: Yes
  • All traffic proxying: No
  • Selective proxying: Yes

Description :
Addon uses its own list of blocked sites. You can also
make lists with your own sites that you want to proxy.
Addon is able to determine whether you have blocked a specific site or not, and
enabled only if it is locked but available
option "always proxy".
You can use your own proxy, not a proxy service.
AntiDPI mode is declared to bypass provider DPI, however
I did not earn it.
The Firefox version can proxy all sites, unlike the Chrome version.

Pros :
Uses a list of blocked sites updated by administrators.
Able to determine whether the site is blocked, and only in the case of blocking
use proxy.

Cons :
In the list of blocked sites a small number of addresses
