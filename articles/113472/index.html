Evernote update for Android

We are pleased to present you with the Evernote update for Android (version 2.6), which includes a number of fantastic new features and internal improvements. You can download the new version right now from the Android Market ¬ª

New note display mode


We always try to find a balance between the density of information placement and the features of the form factor of a particular device. If the content is very much, it is difficult to perceive. Too little - and there is emptiness. In this version, we present a new ‚ÄúMixed‚Äù display mode. It was created with the expectation of a more dynamic presentation of information about the note and more rational use of space on the screen.




If the note contains an image, we will show you the title of the note, the first few lines of text and a thumbnail of the image. If this is a text note, then instead of a thumbnail, we use the entire width of the screen to fit as much text as possible. All this is necessary so that you can get maximum information at a glance. In many cases, you do not even have to open a note - you will receive all the necessary data from the text displayed in the list.

Other viewing modes have not gone away - you can switch between them in the display settings menu.

Notebook Sets


Recently, notebook sets have been added to Evernote for Windows and Mac, a feature that allows you to visually group notepads. Now you can view and interact with these sets created on your computer.

Tip: if you have special notebooks with which you work most frequently on the phone, for example, travel notes or shopping lists, you can combine them into a set called ‚ÄúMobile notepads‚Äù - this will simplify the selection of such notes.



Editing notes with formatting


Now you can edit any note in your account, be it a text note, a fragment of a web page, a photo or a voice recording. You will no longer see annoying pop-ups ‚ÄúAdd text to the end of a note or create a copy of a note without formatting?‚Äù. This feature allows you to save the design of any item that you decide to change. For example, if you edit a bulleted list, the new line that you start will also begin with a bullet; if you are editing a bold text section, all added content will follow the corresponding style.

A note on this feature: Evernote for Android now allows you to edit notes with formatting, but you cannot yet create and copy new styles. We are working on it.
