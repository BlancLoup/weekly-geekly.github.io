Change application icons

Today I had a task to change the icon for one of the programs. But let's take this topic a little wider. Mac OS X has traditionally been very good icons. This applies to both applications and the system as a whole. But sometimes situations arise when the icon either bothers or, in the case of third-party programs, does not suit at all. These are only the most likely causes. In fact, you can invent a lot of them. Speaking in advance, it was the second time that led me to write the article. On this from the introductory move to the case.

Apple in Mac OS X has its established format of icons, which wears the extension icns . It supports the following sizes 16 √ó 16, 32 √ó 32, 48 √ó 48, 128 √ó 128, 256 √ó 256 and 512 √ó 512. But you must admit, it‚Äôs not so easy to search for a ready-made icon of the format we need as to find a suitable graphic image. And as a result, the question of conversion arises.

The most common web service for these tasks is called iConvert . It will help make the icon not only for Mac OS X, but also other popular operating systems. In order to use it, you need to find or make a graphic image and point the way to it through the navigation key on the site. Upon completion of the conversion, it will provide us with icons for Mac OS X, Windows and Linux.



You can search for image sources in Google Images or in Icon Finder, my favorite service.

Lovers of local software to meet the same needs, you can put the application Img2icns . It is distributed in two versions - free and pro. The first one does not know how to save conversion history, export icons for iPhone and favicon , convert icons into graphic formats. Yes, we do not need it. Drag the image into the program body and select the Icns position.



By the way, the program can immediately create an icon for the folder. Some will certainly be useful.

Now that we have a ready file of the required format, we need to substitute it for the current application icon. Make it very easy. Select the file or folder where you need to replace and navigate through the context menu in Get Info (you can simply press Command + I). Then, simply drag the icns file onto the icon in the upper left corner, as shown in the screenshot below.



The result will be visible both in the file manager and in the Dock.
