AWS: New AWS Asia Pacific Region (Sydney)

Hello!

Great news! Amazon Web Services opens up a new region in Sydney , Australia !

The region codenamed ap-southeast-2 has just recently appeared on the menu, but the Internet has already stirred up! Now Aussies and New Zealanders will be able to deploy their applications and deploy huge server infrastructures right on their continent! As usual for redundancy, the region has 2 zones and allows you to create fault-tolerant applications and architectures.



Almost all Amazon services are available in Sydney:
  • Amazon Elastic Compute Cloud (EC2)
  • Amazon CloudWatch
  • Amazon Virtual Private Cloud (VPC)
  • Amazon Simple Storage Service (S3)
  • Amazon Elastic Block Store (EBS)
  • Auto scaling
  • Amazon Simple Queue Service (SQS)
  • Amazon Simple Notification Service (SNS)
  • Elastic load balancing
  • AWS Support
  • Amazon Elastic MapReduce
  • VM Import / Export
  • Amazon SimpleDB
  • Amazon Relational Database Service (RDS)
  • AWS CloudFormation
  • AWS Storage Gateway
  • Amazon DynamoDB
  • AWS Elastic Beanstalk



As in all regions outside the United States, the cost of using AWS resources in Sydney is quite serious, but many are already preparing relocation plans.

By the way, among other things, Amazon CloudFront (CDN) also has a new Edge Location in Sydney!
