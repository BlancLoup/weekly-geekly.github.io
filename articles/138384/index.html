Google is preparing a built-in password generator for Chrome


Obscured stories about how, it would seem, quite respectable people and companies use as a password for access to mail and other service combinations of numbers, names of wives, girlfriends, dog names and other easily calculated combinations, with new releases of Chrome, hopefully , will no longer be associated with this particular browser.

Google is preparing an additional tool to increase the security of its users - a password generator built into Chrome that will work according to the expected pattern: the user visits the page, and if you need to enter a password in one of the form fields, a special icon will appear next to it, clicking on which will cause the tool to generate a secure, strong password. If the user agrees to use the proposed password, the generator will immediately place it in the input field and enter it into the standard password manager for saving so that it will not have to be entered next time.
