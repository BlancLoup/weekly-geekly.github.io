Microsoft introduces FrontPage replacement

On December 5, the first public presentation of Microsoft Expression will be held - a software package for web designers that will replace FrontPage.

The application package for creating websites includes Expression Web - a basic program for creating web pages, as well as a Graphic Designer (working with vector and raster graphics) and an Interactive Designer (for developing dynamic pages and web programming). The developers declared full support for XHTML, XML, CSS, AJAX, etc.

The Expression package, like many new Microsoft products, is designed to work with Windows Vista and Internet Explorer 7.
