Tips in the search box in Opera for Google and others

As you may already know, Opera in version 10.60 has begun to support search hints. However, while they are immediately shown only for Bing and Wikipedia. However, it is quite easy to make friends with Google Opera (and some others).




Ready solution:

  • We make a copy of Google search (I personally could not get to suggest an existing search). To do this: go to Settings-> Search, go to Google editing, copy the contents of the Address field, cancel; Create a new search by inserting the copied text into the Address field.
    Alternatively, just go to google.com and create a new search (right-click on the input field, Create Search). UPD: the second option is undesirable because it does not save the Opera partner's partner identifier when searching.
  • We are closing the Opera. We wait until it disappears from the process (I personally close for quite a long time).
  • Go to the folder of the Opera profile (if you do not know where it is, first look at the config: opera:config#Opera%20Directory ).
  • Save search.ini (just in case) and open it for editing. UPD: use only a good UTF-8 editor, such as Notepad ++. Do not use windows notepad!
  • We are looking for a Google search created in it (it will be at the end). At the end of the section of this search we add the lines:
    Suggest Protocol=JSON
    Suggest URL=http://suggestqueries.google.com/complete/search?json&q={SearchTerm}
  • UPD: If you duplicate the search, be sure to check the block of the created search for the presence of Encoding = koi8-r
    If found, replace with Encoding = utf-8
  • We save, run the Opera, try and enjoy.
  • Delete the old google search if it is no longer needed.

The options in the tooltip will be only 3, it can not be increased (for now).

UPD: for the benefit of Opera, please save partner identifiers in search links (for example, for Google this is the component sourceid = opera). Simply put, creating new searches, take the search URL from the existing (standard).

Addresses for hints on other searches (taken from the opera forum , I tested only Youtube): 

  Youtube: http://www.google.com/complete/search?hl=ru&ds=yt&output=firefox&q={SearchTerm} 
 Google Images: http://www.google.com/complete/search?hl=en&ds=i&output=firefox&q={SearchTerm} 
 Wikipedia (rus): http://ru.wikipedia.org/w/api.php?action=opensearch&search={SearchTerm}&format=json 
 Yandex: http://suggest.yandex.ru/suggest-sl?&part={SearchTerm} 
 Yandeks.Slovari: http://suggest-slovari.yandex.ru/suggest-sl?&part={SearchTerm} 
 Yandex.Market: http://suggest.market.yandex.ru/suggest-market?part={SearchTerm} 
 Yandex.video: http://sitesuggest.yandex.ru/suggest-ya.cgi?site=video&part={SearchTerm} 
 NIGMA: http://cplt2.nigma.ru/complete/query_help.php?suggest=true&q={SearchTerm} 
 Bing: http://api.bing.net/osjson.aspx?Query={SearchTerm}&FORM=OPERAS&Market=ru-ru
 Yandeks.Kartinki: http://suggest-images.yandex.ru/suggest-img?&part={SearchTerm}
