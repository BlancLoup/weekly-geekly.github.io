Update for Windows 7 (KB971033)

Update method: Important

This update for Windows Activation Technology detects exploits in the activation mechanism and unauthorized modification of the main Windows system files. These exploits attempt to circumvent the usual procedure for activating Windows and are sometimes included in unlicensed copies of Windows.
