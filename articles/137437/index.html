OGS Mahjong Solitaire 0.9, one step away from the finish line

Watching the reaction to previous topics on OGS Mahjong, I came to the conclusion that every next post about this project is interesting to the community less and less. However, I decided what to write about the appearance of the new version of the game is still worth it, because I hope that the changes in the new version are more interesting than the past ones.

There are many changes in the new version. Not all of them can be seen with the naked eye, but we have tried to make this new version worthy of a release candidate. And, as it seems to me, we succeeded.

So, in version 0.9, OGS Mahjong solitaire received:
  • A completely new environment for the playing field (thanks to the model of the room thanks to Anton Chernov).
  • Support for shaders in the materials of the chips and the scene.
  • The possibility of separate selection of the environment and topics chips.
  • Support for kmahjongg-1.1 layouts with non-standard sizes.
  • New, more convenient and intuitive game settings interface.
  • The ability to dynamically switch the language, as well as the French localization (for the translation, thanks to Thierry Delaunay).
  • Installer for Windows version
  • Layout editor with kmahjongg-1.1 format support
  • Three new tracks in the archive with additional music (one from Kevin MacLeod and two from JewelBeat.com).
  • Well, many bug fixes.

We wanted to support MacOS, but due to a number of circumstances, it did not work out. Let's try to make it to version 1.0.
