A set of beta testers for virtual hosting



TEL Hosting announces a set of virtual hosting beta testers. In the near future, we plan to launch hosting in commercial operation, but first we would like to debug the whole process.

Each participant will be given access for 6 months to participate in beta testing. Applications are accepted until August 31, 2011. All this time, beta testers will be able to use our services for free on the selected tariff plan. Testers get all the rights of ordinary users.

-preparation of a monthly (or more often) free-form report on problems found during work
-the introduction of proposals to improve the functioning of the hosting
