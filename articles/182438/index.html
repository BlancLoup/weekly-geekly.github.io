Customize standard IOS icons without JailBreak

Hello,

Today I would like to tell you how to change the standard application icons without jailbreak using the IOS Configuration Utility and Web Clips.



Everyone knows that IOS supports the ability to add shortcuts to web pages on the desktop, and also that most standard applications have their own protocol for launching using Safari. enter. Thus, you can create an HTML page with a custom icon that will redirect to the sms: // address and open the standard Messages application. The disadvantage of this approach is that to open an application using a modified icon, a constantly working internet connection is required. But there is a way to avoid the need for an internet connection. Who loves non-standard design and experiments, I ask for a habrak.



To implement our plan, we need the iPhone Configuration Utility and our own modified icons.

Step 1



Open the iPhone Configuration Utility and navigate to the Configuration Profiles tab.



Step 2



Scroll down and select the Web Clips tab.



Step 3



Add icons and create short links. A list of available links for standard applications can be found here.



Step 4



We sign and export the profile to the desktop or send to the device



Step 5



Install a profile using iTunes, Safari or Mail and enjoy the icons :)
