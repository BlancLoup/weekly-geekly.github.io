Reporting for Virtual Environments with Veeam Reporter

Today reporting tools are a necessary tool for any virtual environment, regardless of its size and scope. In this review, we will look at the development of Veeam, the product of Veeam Reporter, which helps a system administrator to assess current workload, track changes and plan the use of virtual environment resources.

Veeam Reporter collects performance, event, and configuration data through the vSphere API, without the use of agents. The collected data covers 25 types of objects, 600 different properties and more than 150 events. Reporting is created in SQL Server (SSRS) format and is accessible via the web (see Fig. 1); it can also be presented in Visio, Excel, Word, PDF.


Figure 1 - Report on the use of disk space

The virtual environment is very dynamic and constantly changing. Changes are initiated either by the VMware system itself, or by one of the administrators, and occur at various points. All this is tracked in Reporter. As a result, you can create reports on the current state of the virtual environment, as well as on any point in time in the past. The reporter records not only what change was made, but also ‚Äúwho, where, when, and how‚Äù made it. Thanks to this data, it is possible to improve workflows with changes and investigate them if they are problematic (see Fig. 2).


Figure 2 - Tracking Changes in a Virtual Environment

In addition, Veeam Reporter allows you to analyze trends (trends), thanks to which you can determine the average and maximum load or identify problem areas. Reporter also helps to ‚Äústay ahead of trend‚Äù by predicting when the level of use of processor, memory and data storage resources reaches threshold values. Given the information about the configuration of the virtual environment and its use in the past and present, Reporter provides recommendations on the allocation and acquisition of resources. You can also check how resource usage changes when you add or remove hosts or virtual machines.
Veeam Reporter has a free version - Veeam Reporter Free Edition, the functionality of which has a list of ready-made sets of reports, including capacity planning. The free version is not limited in time of use. More information about this solution can be found on the manufacturer's website (including in Russian) - www.veeam.com
