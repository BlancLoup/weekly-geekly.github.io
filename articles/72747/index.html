Official denial of rumors about the mysterious touchscreen phone Nokia N920 with NokiaPort.de.

My job.

A few months ago, I was entertained by creating a new touchscreen phone that would satisfy my personal needs.
It occurred to me when I came across photos of the official release of the phone, which were published a long time ago. I was not sure that it was a presentation from Nokia, Symbian, Maemo or anything else. This is a kind of clipart, taken as a template.

Honestly, I could not decide which interface to use for this phone. So, using the search engine, I found photos of the Nokia N800 Internet Tablet. This is a fairly modern phone, although it works on an outdated Maemo 4. A welcome phone, but with an outdated user interface !? I didn't care ... some of you noticed a trick here.

Start

The story began with a simple message on my twitter in early September.
A simple message: "Nokia is similar to the IPhone, [Link to image]." (9th of September)
That's all!

For several weeks, the message remained unread, and I thought that nothing would change. But I was wrong ...
