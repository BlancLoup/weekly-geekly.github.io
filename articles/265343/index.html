Five reasons to try the new release of Intel Media Server Studio 2015 R6

Introducing Intel Media Server Studio 2015 - a set of tools for developing, debugging and evaluating media applications and enterprise-level solutions based on Intel processors.
If you are engaged in just such solutions, then now is the time to try Intel MSS, because just recently its latest update was released. Under the cut - a list of major improvements that have appeared in the new version.


1. Support for fifth-generation processors (Broadwell). To speed up the processing of media files, the new release of Media Server Studio supports the fifth generation of Intel Core processors and the Intel Xeon E3-128x v4 processor family with integrated Intel Iris Pro graphics solutions. The flagship processor for desktop systems Intel Core i7-5775-C allows you to increase the processing speed of media files by 35% . And thanks to Xeon processors, video encoding is 1.4 times faster. This turns out to be a significant advantage when working with demanding graphics applications.

2. Free Community Edition . The free version of Community Edition includes all the features of Essentials Edition: Media SDK, executable modules, multimedia and graphics drivers, OpenCL Code Builder, etc. However, it does not cover Intel Premier Support (direct support from Intel technicians).
  • See the details on the product description page.
  • Download the free version.
  • Read the documentation , release notes, and other manuals; Refer to the forum and frequently asked questions for help.

3. HEVC. The version of Media Server Studio Professional Edition includes a software implementation of HEVC, thanks to which 4K video can be processed on Xeon E5 processors in real time. Compared with previous releases, it was possible to significantly increase productivity (on average by about 15%). For the HEVC format, software and hardware implementations (with GPU-based acceleration) are now provided for 4: 2: 2 subsampling with a 10-bit color depth. It was possible to significantly improve various methods of controlling the bitrate, for example, CBR, VBR and AVBR. For HEVC software and hardware accelerated encoder based on a graphics processor, proactive bit rate control and optimized 1: N LA methods are supported. The hardware accelerated HEVC implementation has become available for more targeted usage modes. (See the article about using HEVC )

4. Version Metrics Monitor and Intel VTuneTM Amplifier for Linux OS. To better understand the distribution of GP workloads when working with media data, use a GP kernel level driver in Linux, with which you can monitor the necessary work parameters. Thus, you can monitor the status of certain hardware modules of GP:
  • rendering engine (executive modules);
  • multiformat CODEC (MFX);
  • video quality module;
  • module for working with bliter objects.

Implemented support for Intel VTune Amplifier for Linux. VTune Amplifier is a great tool for setting parameters and optimizing performance; with it, you can monitor resource-intensive areas, streaming processing, locks and delays, OpenCL, bandwidth, and much more.

5. Virtualization support on E5. Use the software implementation of Media Server Studio in a virtual environment E5 - KVM + Xen, Xenserver in Linux. This will allow you to run a virtual image of the operating system, work with several operating systems or simultaneously perform multiple transcoding processes.

This article does not list all the benefits of the new release of Media Server Studio 2015 R6; See the release notes for details.
