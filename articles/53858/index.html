Alpha version qutIM 0.2 is waiting for you

After months of hard development, problems with the server and other difficulties encountered, the developers of the cross-platform IM client qutIM are pleased to announce the release of version 0.2-alpha.



A little bit about what version 0.2

First of all, it is worth noting that this is a completely new kernel, that is, the code of the new kutim is written from scratch and has no relation to version 0.1.1.

Should I upgrade to version 0.2?
It is worth it, since the new version is more powerful, more flexible and has more features than the old one.
Briefly about the features of the new version.

Appearance of the program


  • Smile pack support from Kopete
  • Styles support for contact list
  • Support for changing the form for chat windows (changing the location of buttons, adding / removing unnecessary, etc.)
  • Support for Adium and Copet styles for chat logs
  • Styling support for pop-up notifications
  • Adium Status Pack Support
  • Icon pack support for the entire application
  • Translation support
  • Style support for the entire application
  • Support for frames for contact list

New features of the program


  • Profile support (the ability to create your so-called container with individual settings, program style and a set of accounts)
  • Extended contact list features
    • ability to display status text directly in the contact list under the contact's nickname
    • display avatar contacts
    • ability to swap groups
    • drag and drop contacts from group to group
    • selection of a contact by typing letters from the keyboard
    • and much more )
  • Enhanced chat window features and work optimization
    • context menu of the contact from the chat window
    • support for forms made in Qt Designer
    • more optimized use of RAM in tabbed mode than before
  • and much more

But the most important introduction is extensibility with the help of third-party plug-ins.

At the moment there are the following protocol plugins:

--- ICQ
  • x status support
  • file transfer
  • privacy statuses
  • support for private lists, lists of visibility, invisibility
  • user search
  • etc.


--- Jabber / LiveJournal / Yandex / GTalk
  • File transfer
  • Moods
  • Activity
  • Ad-hoc
  • Search services / conferences
  • Transport support
  • Search users
  • Conference invitations


---Mail.Ru
  • Receiving / sending messages
  • Outgoing / incoming input notifications
  • Message Delivery Confirmation
  • Sending / processing authorization messages
  • RTF message support (text only)
  • Offline support
  • Receive alerts for new email inbox
  • Login to the mailbox from the client (without authorization on the site)
  • Adding / Editing / Deleting Contacts
  • Search for contacts
  • Contact Information
  • Avatars
  • The ability to send SMS messages


--- IRC
  • conference
  • private chat rooms
  • basic commands (/ me, / nick, / join, etc.)
  • CTCP teams
  • and etc.


At the moment, the addition of the functionality is frozen and the trapping and correction of critical errors begins. About the errors found with a detailed description of writing on the track program trac.qutim.org
