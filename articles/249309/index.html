YouTube hotkeys

All of us (well, almost) use the YouTube service, but not everyone knows that the process of watching videos on the above service can be compared in convenience with watching videos on a local media player (for example, Media Player Classic), if you use hot keys.

Going to the controls with the mouse is not always convenient, I really like the hot keys and, besides, I did not find such a publication on Habr√©, so I decided to start small and devote my first publication to compiling a detailed list of YouTube hot keys.

Playback control


Space or k : pause / play
> : increase playback speed
< : decrease playback speed

f (fullscreen): go to fullscreen mode and vice versa (you can also exit fullscreen mode using the Esc key )
