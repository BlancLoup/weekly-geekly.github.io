Opera Mobile: one interface for all mobile versions of Opera browsers



December 10, 2009 - Oslo, Norway

- Opera presents a new cross-platform interface shell.
- Opera Mobile 10 browser is now available for mobile operators and mobile phone manufacturers on major platforms, including Symbian / S60, Windows Mobile, Android and BREW

The company Opera Software today released Opera Mobile 10 browser for mobile operators and mobile phone manufacturers. This version is available for Symbian / S60, Windows Mobile, Android and BREW and includes a new cross-platform interface shell. As part of a general program for unifying the appearance and user benefits of all Opera browsers, this shell will allow operators and phone manufacturers to offer their users the same opportunities to work on the Internet, regardless of the brand and platform of mobile devices.

The building blocks of the new interface
The cross-platform shell is a superstructure above the browser core. This gives it the opportunity to work on all platforms within Opera Mobile and Opera Mini and allows you to create dynamic user interfaces. Thus, mobile operators and phone manufacturers can deploy a single browser interface on all phones distributed by them with minimal integration, saving both time and money on implementation. End users receive a single and familiar tool for accessing the Internet, which increases loyalty to the operator and phone manufacturer.

Opera Mobile 10 is a browser for smartphones and powerful mobile phones, installed on more than 135 million devices worldwide. The browser has all the features to make working in the mobile web easier and easier. The set of functions includes tabs, Express panel of quick access to your favorite resources and Password Manager. The current release of the version for operators and device manufacturers on the most popular platforms opens up access to Opera browsers for even more users.
