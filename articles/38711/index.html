Comparison of WIPmania and MaxMind GeoIP databases

In connection with the alternative that appeared to the base, MaxMind decided to compare how much these bases differ. It is the differences that are compared, not the correctness / incorrectness of the definition.

Comparison

1) The size of the bases. MaxMind base is about twice as large as WIPmania (100k vs 50k)
2) The number of ip-addresses entering into all ranges is practically the same here and there.
3) Address ranges are 88% identical. (That is, start / end ip-addresses in the database of MaxMind and WIPmania)
4) Of these 88%, only 662 (1.2%) of the subnets reassigned the country.
5) Next, the number of ip-addresses entering into all subnets for the country.

Compare how much different the WipMania data from MaxMind is for uneven ranges.
Algorithm such:
a) we define the country by the initial and final ip in the WipMania database by the MaxMind database.
b) we get data like start_ip, end_ip, wipmania_cc, maxmind_start_cc, maxmind_end_cc
c) select all subnets where maxmind_start_cc! = wipmania_cc OR maxmind_end_cc! = wipmania_cc
The result is 2633 entries.
e) besides, they were undecided on the basis of GeoIP 1394
The difference is about 7%.

Findings:
1) Indeed, other techniques have been used to determine GeoIP.
