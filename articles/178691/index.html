New version of WebMatrix 3: integration with the cloud, TFS, Git, remote access to sites


A new version of the free code editor and integrated application development tool WebMatrix 3 has been released. Among the features of WebMatrix 3 are the following:

  • code support for HTML / CSS / JS, ASP.NET, PHP, Node.js projects;
  • built-in SQL Server and MySQL database editor;
  • support by editors Jade, EJS, LESS, CoffeeScript;
  • gallery of popular open source CMS templates: Joomla, Drupal, DotNetNuke, Orchard CMS, WordPress, phpBB and dozens of others ;
  • ready start projects, including Node.js with Express and Socket.io;
  • support for auto-completion and intellisense for HTML / JS / CSS / jQuery, ASP.NET, PHP, Node.js;
  • mobile development : emulators for mobile web development, mobile site templates;
  • site analysis tools for SEO, error, performance;
  • system and gallery extensions from the developer community;
  • support for the gallery of NPM modules for Node.js and the Nuget repository for ASP.NET;
  • built-in integration with Git and TFS version control systems;
  • integration with the Windows Azure cloud, publishing PHP, Node.js and ASP.NET projects to the cloud, remote access to sites;
  • remote access to sites and publishing via FTP, WebDeploy, access to cloud locations.

WebMatrix 3 is available for free download on the official website http://webmatrix.net/ .

A quick review of the third version is available in this video on Channel9. For a more detailed description of the features of the editor, refer to this article and this announcement of the very first version. Below you will find information about innovations in the third version of the development tool.

Deeper integration with the Windows Azure cloud


Previously, WebMatrix offered the ability to quickly publish PHP, Node.js, and ASP.NET sites using publish profiles downloaded from the administration portal. Today, the development environment can be integrated into your Windows Azure subscription in order to further simplify the site publishing process.



After connecting your own Windows Azure account, you will be able to create new websites in the cloud or choose which of the existing sites to publish the project opened in WebMatrix.



Now the publication of the new site does not require working with the Windows Azure administration panel. Below you can find a video that describes the process in detail:

  • Using WebMatrix to Publish a site to Windows Azure

Versioning with Git and TFS


One of the most anticipated features of WebMatrix was support for version control systems. In the third version, it appeared for Git and TFS systems, as well as public services TFService, GitHub, CodePlex. Support for version control is implemented through extensions, so soon we can expect the emergence of support for other systems.



Tools for working with version control systems, such as Git, support all the necessary functions for commits, brunches, simultaneous work, continuous publishing to the cloud, and so on.



Remote code editing


The new feature of WebMatrix 3 will allow you to open remotely sites located in the cloud or accessible via FTP. This will allow on the fly to make small edits to sites hosted on the web, using convenient code editing tools in WebMatrix.



All files are edited remotely, but if necessary, you can download the entire site entirely to your local machine.
