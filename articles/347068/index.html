Beta test section for beta testers Habra and Hiktayms

At the end of last year, we summarized the results , talked about the highest-rated publications, counted users and shared the results. And so, when the car of the new working year began to accelerate and gain full speed, we are ready to present several useful changes.



Beta Test Page


Some of our releases in the previous year were, to put it mildly, not perfect. In some cases, immediately after the announcement, we received messages from users about errors found (for which we are very grateful to them). Of course, we promptly made corrections, but this situation in itself did not suit us.

We rethought the situation and found a way to fix it. We decided to connect the collective mind to our own efforts to test innovations. Moreover, if earlier this opportunity was only for some users with a secret cookie, now we decided to make it available for everyone, because often some scripts need to be tested with different roles: with negative karma, with positive, in readonly mode, and so on.


In the user menu (top right), each one has a link to the beta section - in it you can switch the site to beta mode. On the page that opens, there is a list of ready-for-test features that can be turned on and off. Under each feature - comments, where you can report an error found or leave a wish.


And the first two updates are ready for testing:

Updated profile settings


We have completed the redesign of the main sections of the site (tape, publication page, comments, profile, etc.), but still some pages remain. One of them is the user profile settings page, where we also made a facelift, removed extra fields and extra projects in the contact information section, rewrote everything to Vue.js, etc. Try it.

Updated Subscription Page


To read only what is interesting on the site, each registered user can set up a personalized tape . Then on the main page of the site in the " By subscription " section there will be only publications from those hubs to which you are subscribed.

We have remade the ribbon settings page - now all hubs are displayed there, with avatars and short descriptions, as well as companies.



To quickly find out about the innovations available for testing, enable email notifications on the page that opens.
