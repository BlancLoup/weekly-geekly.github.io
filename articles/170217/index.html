Best Android Mobile Development Apps



I am sure that not only I, but most of those present here feel without mobile devices as without hands. Fortunately, we live in a time when mobile development is becoming the norm, and more and more truly user-friendly applications are entering the market, with which you can not just fix the line of code, but also fully continue working on your project.

Under the cat, I will give a list of applications for Android devices that will allow you to effectively develop from your mobile device from anywhere in the world.

DroidEdit
If you are looking for the perfect text editor source code for your tablet or phone, it is difficult to advise anything better than DroidEdit. Syntax highlighting of languages, including C, C ++, HTML, CSS, JS, Ruby, Python and many others.

The cost of 60 rubles.
Free trial available

Programmer Keyboard
Write a line of code on a mobile device is oh how difficult. PHP programmers must evaluate this keyboard, with auto-completion of all PHP commands, as well as undo and redo functions. It also has the Ctrl, Alt, and arrow keys!

After installation, you need to add this keyboard through the settings of your device.

Application for free

kWS Web Server
Easy and fast (and free) web server for your phone. You can host web sites, and you can use to access via FTP. Uses HTTP 1.0. Many other interesting features. Recommend.

There is a PRO version for 80 rubles.

gAnalytics
Applications for analytics attendance on the market abound, but my favorite is gAnalytic, a great interface, speed and usability.

Price 150 rubles.

AndFTP
Gorgeous master-hev FTP client in Russian, with the ability to access the file system of your phone. Rescued me more than once!

Price 160 rubles.

AIDE
This application should appeal to developers of mobile applications for Android. It is compatible with the Eclipse project and makes it easy to transfer the projects of your applications from a work computer to a phone. It is also compatible with Dropbox.

Application for free

View Web Source
Love to dig someone else's source code to see "how is it done with them"? That way. We drive in the URL of the site of interest and see what's under it under the hood.

The application is free.

HC-16C Programmer's Calculator
Excellent emulator known to many calculator. Programmable, in addition to division / addition / multiplication / subtraction, can use many useful functions useful to programmers.

The cost of 202 rubles.
