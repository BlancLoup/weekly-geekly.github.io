Release Ubuntu 9.10 "Karmic Koala"

We all waited for this, and now, in exact accordance with the plans, a new version of Ubuntu, 9.10, code-named Karmic Koala, was released in the final version.

Distributions are available for download on the official website . Also, all Ubunt cards, such as Kubuntu , versions for netbooks ( Netbook Remix ) and servers ( Server Edition ), were updated simultaneously.

Suppose this version is not LTS (the next Long Term Support is waiting for us in April of next year), it contains a lot of tasty. The full list (it was written for beta or RC, but refers to the release in full) can be found both in Russian on ubuntu.ru or on Habr√© , and in English .

Major changes:
  • Optimization of the boot process, the transition to the use of Upstart
  • Software Center ( Software Center )
  • GNOME 2.28 (with new wallpaper, icons, and Empathy instead of Pidgin as the default IM
  • Ubuntu One Sharing Service
  • Linux Kernel 2.6.31-11.3
  • Default ext4 file system (new installations only)
  • Switch to GRUB2 (for new installations only)
  • Waiver of HAL mechanism in favor of devicekit
