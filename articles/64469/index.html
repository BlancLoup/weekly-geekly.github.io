So, about how you still get it. In the iTunes App Store there is an application that is called Prowl - Growl Client (the link will open iTunes) and it allows you to connect with the famous Growl . Growl is a simple program with only one function - to notify. She notifies about everything you ask for: completed downloads, new IM messages, fresh tweets, incoming mail, etc.
