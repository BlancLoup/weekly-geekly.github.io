How to configure EDGE (mobile Internet) on iOS 5 beta



This advice is written based on the post In iOS beta 7, the EDGE settings have disappeared . They themselves will not appear there, but in order to appear, we will do a small "operation" (completely legal with respect to Apple, since it is performed by means developed by Apple for such cases)

So let's get started:

First of all, go here and download the iPhone Configuration Utility 3.3 for Mac OS X

Install , run and connect your iPhone cable

In the left pane, select your iPhone


At the same time in the right pane on the Configuration Profiles tab we will see the entire list of installed additional customization profiles for your iPhone. Usually you should not have any additional profiles.

Create a new profile



Let's start filling out the minimum necessary information for the operation of our mobile Internet.
On the General tab, fill in:
  • Name: Macovod
  • Identifier: com.macovod.profile


No additional actions are needed to save. Once you have left the input field, the information is saved.

Go to the Advanced tab, click on the Configure button and fill in:
  • Access Point Name (APN): we drive in the name of the access point of your mobile operator
  • Fill in the remaining fields as needed (usually filling in other fields is not required)





After filling, we have actually created a new profile for our iPhone, only the last step remains.

Install Profile on iPhone


Click on the name of our iPhone in the left panel, section DEVICES. In this case, in the right section, in the Configuration Profiles section, select the profile that we just created and click the Install button .

All that remains to be done is to confirm the installation of the new profile on the iPhone itself, and if the mobile Internet did not start immediately, restart the iPhone.
