UTorrent adds support for Android and iOS devices

In the latest version of the client uTorrent ( 3.1 alpha 25671 ) there is a function of copying video to a mobile device. Now you can drag and drop the downloaded files onto the Apple player / smartphone / tablet (via iTunes), Android (directly), PS3 or XBox prefix (via USB) directly from the client. The function of automatically converting video into a mobile device format is still announced, but does not work. Perhaps they will connect codecs in one of the following uTorrent builds.



Devices connected to the computer are automatically recognized (USB) and appear in the new menu. This menu can be hidden if annoying.



Of course, support for mobile devices is clearly not done for geeks, who copy files in a file manager, and convert to VirtualDub. This function is for a mass audience, for those who still consider uTorrent to be a complex program and cannot find files after the end of the download.
