A simple Dropbox based blog service.

It started, however, in the early beta stage (the creator writes ‚Äúsuper-beta‚Äù), the service http://scriptogr.am , the main purpose of which is the ability to quickly create a blog (so far without anything extra like advertising), which will be hosting be a dropbox account. It's pretty simple:
  • It is required to allow the service to access the account on Dropbox, in which it will accurately create its scriptogram folder, and the posts folder in it - there will be stored markdown files proper, which are future blog entries.
  • Then in the admin you need to somehow call the blog and thereby obtain the address http://scriptogr.am/username ; right after you can start a blog with an extremely simple text editor that offers a service. The editor allows you to create a markdown file by placing it in a dropbox account or editing an existing one.
  • The next step is synchronization between DropBox and scriptogr.am (the button for this is hard to find), after which a new blog will be available at http://scriptogr.am/username . It is perfectly acceptable to load pre-prepared md files and synchronize them in the same way.
Demanding users can edit CSS styles or the HTML code of a blog, which, nevertheless, looks pretty good by default - this is what the official blog of the project looks like.
