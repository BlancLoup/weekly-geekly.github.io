Searchme continues to invent

Since the last post about the visual search engine Search me , she has new features of media retrieval and collections ( Stacks ). Video search is not a new thing (although it continues to be invented ), but the possibility of creating galleries with a flick of a mouse is interesting.

Prior to this, the features of the search engine were only in the visual representation of Cover Flow (it seems that they added a selection of words to the screenshot in the frame) and the category assumption, due to the lack of which exceptions should be entered in the usual search engine.
And yes, AdBlock still blocks Searchme, but only screenshots of sites, so you need to add the exception "@@ searchme.com".

Stacks


Stack - collection / folder of links to found pages, videos and images that are displayed in the form of the same gallery (but in the order of their addition, which can then be changed through the field in the corner of the slide). With the collections, as it turned out, it is very convenient to work - take a picture and drag it to a corner, or by clicking on the add icon to the selection . You can add manually by typing a link with a title and description - it turns out a good tool.

The selection can be inserted as an embed object into a blog, and even the video will be played. For pictures it turns out a simple slideshow. Or you can just give a link - Veronica Belmont is cool :) And with a permanent link I can change the content.
Examples from the developers can be found on the page descriptions of collections . Collections are stored in the same way as in Picamatic , without registrations. Simplicity is a guarantee of health, only if the browser flies, then without a reference to the selection it will not be returned.

Media search

So far, the search for videos is done on YouTube, and the pictures are on Flcikr, but expansion is a matter of time. In the search for video clips are played one by one ( thus Searchme can be used as an online media player), but if it is not necessary, then autoplay can be disabled in the settings.
Search in pictures would not prevent the display in three lines, but to search is already so convenient, and more pleasant than through Google, although it is inferior in functionality.

Conclusion

As a user, I want Google / Yahoo / Yandex to have another competitor who has priorities in new features, an interface, thanks to which the accuracy of algorithms is no longer so important - it becomes easier for a person to navigate the search engine output, and SEO-wound sites easier to notice. But until Searchme comes into action, it is not as close as it is, on the Internet as there were no results, it is not (even Odnoklassniki does not find it), and many functions are not yet completed, although they deserve attention. But for myself, I have already found a use - a replacement for the PicLens add-on, if you just want to see something.
