UA ATMS. ATMs of Ukraine



A very convenient UA ATMS application appeared in the Appstore. The application aims to help users find the nearest ATM, which is not very simple in an unfamiliar area or city. There is also a search function of ATMs of a particular bank, which allows you to save on commission.
The current version of the application is designed for use in Ukraine.
