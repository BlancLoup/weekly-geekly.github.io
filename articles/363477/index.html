Linux with "material" design

A group of enthusiasts began work on the Quantum OS project - an operating system on the Linux kernel, but with an interface in the style of ‚Äúmaterial‚Äù design from Google .

This is how the experimental design that the developers are striving for looks about (this is a concept, it will differ from the final version):



The graphical shell will be developed primarily on Qt 5 and QML on top of the QtCompositor API.

The application framework will be made using the QML UI toolkit, which embodies the ‚Äúmaterial‚Äù design and is created entirely from scratch.

In order to fully implement Material Design and its unique features in applications, it is advisable to port existing applications to a new framework (this will be easy to do). But, in principle, they promise the Qt / Gt theme for old programs.

Like Qt itself, the UI framework will be cross-platform, so applications written for it will work on OS X, Windows or other Linux distributions.

In addition, it is planned to implement support for web applications.



The basic operating system will first be made on the basis of Arch or Ubuntu (most likely, the first).
