Hello to all Habrovchanam! Yes, for a long time we did not remind of ourselves, it‚Äôs time to correct this situation. Not so long ago, our new smartphone OPPO Clover entered the Russian market. Work with 2 sim cards.
One feature of these models is the work with two sim cards. The shell itself determines if both cards are installed and, depending on this, changes its interface settings and notification lines.

You can also separately configure the functions of a voice call, sending messages and selecting a sim card for data transfer.

In our smartphones there is one radio module, but in the case of a conversation on one SIM card, a notification will come about a call to another, at this point you can take a call by putting the previous caller on hold. This function depends on the connected services of both numbers, it is desirable that both have the ability to hold the call.



There is no need to choose which SIM card will be responsible for the Internet, switching is performed in a matter of seconds, without restarting the smartphone, and both slots work with the WCDMA frequency. The interface for working with messages also allows you to simply select from which SIM card to send and monitor messages. If the second sim card is used only for the Internet, then you can simply disable the interface for working with two SIM cards, in this case the interface of the dialer and SMS will change, but the smartphone will continue to work through both SIM cards.




Not forgotten, and the ability to select a ringtone for a call and receive notifications for each of the sim.

Launcher.
The standard launcher was created in order to bring all the icons to one visual style. Depending on the theme, an ornament is created around a non-standard icon. As a result, we have an individual recognition of each application and a single view of the desktop as a whole.

The last desktop is a skeletic morphic music widget that occupies the entire screen. When you connect headphones, if you are on the desktop, the activity will switch to it. In addition to the music window, you can install up to 9 desktops. Controlling the location and selection of the main table is done with a standard pinch gesture. Also added the ability to quickly move between windows when you click on the lower display points of the active desktop.


The application menu has three tabs: everything, history and downloads, as well as a search button by name.

On the desktop, when you swipe up, the settings menu leaves, allowing you to install widgets, select wallpapers, set the page turning effect, and also quickly change the theme. Design options can be downloaded through the pre-installed application ‚ÄúNearMe Themes‚Äù.


By pushing the notification line you can access the main toggle switches for working with a smartphone: wi-fi, bluetooth, etc. Long hold on the buttons in many cases will provide an opportunity to go directly to the settings of these parameters. Of course, there is the ability to select and sort the desired parameters to achieve greater comfort when working with a smartphone.
The notification string itself can be called even from the full-screen application, it is enough to make the swipe down from the top in the middle and it will appear.



In addition to this, with a long hold on the ‚Äúhome‚Äù button, the usual menu of the latest applications appears, from which you can clear all applications or launch Google Now. If an application is very important for you, you can mark it with a lock so that when clearing all items this element is not unloaded from memory.


We turn to the settings , we have divided them into categories, for convenient access to the most important parameters.
In the window of "general" settings, we observe the standard for android control parameters. It is possible to quickly turn on flight mode, work with wi-fi, bluetooth, mobile network. In the call settings, you can determine if you need the two SIM cards at all when dialing a number and SMS, while both cards remain working. You can also set up a blacklist, call forwarding and the ability to hide the number of the outgoing call. All these settings are available for both sim cards separately.


In the memory section, you can change the installation location of applications from internal memory to the card and vice versa.

There is a separate timer on and off the device, with the ability to set two cycles per day.

In the Battery parameters, you can see the charge level, as well as enable the display of the percentage value next to the clock in the notification line.

In the ‚ÄúGestures and movements‚Äù section, there are options for using sensors and sensors in a smartphone, which you can personally select and combine, making working with a smartphone more comfortable. The ‚ÄúSound‚Äù category allows you to set beeps and volume separately for calls and notifications. You can also customize the soundtrack when working with a smartphone: turn on, tap on the screen, keys.

The ‚ÄúScreen‚Äù section includes not only the brightness, auto-rotate and sleep mode settings. In the custom settings option, you can choose the font style, its size, change the lock screen or launcher, if you have installed any Apex or MiHome.



The ‚ÄúSecurity‚Äù section is also highlighted separately. In addition to storing account data, there is a separate notification manager that allows you to control which applications will give their information to the notification line.

You can configure and services that will run automatically when you turn on the smartphone.

Let's run through the pre-installed software.
- Dialer, allows you to pause the interlocutor during a conversation, as well as include a call recording. It supports T9 mode, work with groups and has the ability to edit SMS for a quick answer to a call.

- The message, in addition to the previously described features, allows the received message to be displayed on top of all windows pop-up, as well as locally save SMS.

- The browser uses the standard Google Chrome browser.
- Gallery. Slightly reworked standard application with the ability to select a directory, as well as hide unnecessary folders.

- Video player understands containers AVI, MP4, FLV. Certain codecs allow you to play video up to FullHD at the hardware level. - Built-in file manager. Convenient assistant in finding information. There is a categorization by type of content. As well as built-in FTP and DLNA servers for easy transfer of content without wires within the same network.

- Recorder, in addition to the possibility of direct recording, you can set a timer for automatic recording.

- Lomo Camera. Effects for the camera.



- NearMe Sync - service for synchronizing phone numbers and SMS.
