We use the tablet as a second monitor to develop



To develop in the style of livecoding need a second monitor. And if it is not?

COLT in its delivery includes a web-server that launches the tested html page by the network address. What leads to the simple idea that if you have an ipad (or another tablet), you can use it to design as a second monitor.

How to setup.

Open the editor with the project and run COLT.




Click on the "green lightning" and open the page in the browser.




We take the tablet, make sure that it is in the same network and enter the address with which our page opened in the browser.



We change the code on the computer and instantly see the result on the tablet.



In order to simplify the process of opening a project, we are now developing a mobile application that will show a list of all applications open on the local network running through COLT. It remains only to click on the link.
