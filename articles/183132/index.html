Quiet and quiet - FreeBSD 8.4 release


Besides bug fixes, there are quite a few changes. From the most interesting it should be noted:

  • Upgrading ZFS and geom subsystems towards asynchronous operation and using TRIM on SSD
  • The ipfw syntax has been expanded, which now allows using the names of interfaces, IPv6 addresses, port numbers and ID jails in tables
  • ng_netflow is now able to export NetFlow data of the 9th version, with the help of export9 hook, it is also possible to export data simultaneously in both the 5th and 9th versions of NetFlow.
  • Improved balancing in the sched_ule scheduler, which on systems with SMT (Symmetric MultiThreading) or HyperThreading (as it is called on Intel processors), shows an increase in performance up to 10-15%.




Also a set of third-party software from the basic environment has been updated:
  • AWK 20121220.
  • ISC BIND 9.8.4-P2.
  • BZIP2 1.0.6.
  • TENEX C shell (tcsh) 6.18.01.
  • LESS (less) v451.
  • libexpat 2.1.0.
  • netcat included with OpenBSD 5.2.
  • OpenSSH 6.1.
  • OpenSSL 0.9.8y.
  • sendmail 8.14.7.
  • The timezone tzdata2012j
  • XZ 5.0.4.
