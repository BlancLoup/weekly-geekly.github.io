IKey Army Keyboard



We have already become accustomed to all kinds of communicators and laptops for the military, but the guys from the iKey company offer an armband keyboard for the military combat control system WC2.

The gadget was created by order of SpecOps Systems, compatible with any compact electronics that has a USB port and uses the principle of SMS-messages. That is, after pressing the key, you need to select the desired character from the list.

There are 17 such keys. Keyboard dimensions are 65x70x5.6 mm. As an option to the keyboard LED backlight is offered.
