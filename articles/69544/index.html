Haiku R1 / Alpha 1

The Haiku project is proud to introduce Haiku R1 / Alpha 1, the first official release of an open source personal-oriented OS. The project aims to create a fast, efficient and easy to use and learn, but at the same time powerful operating system.


The goal of this release is to provide a stable version of the OS being developed in order to expand the audience for more thorough testing and debugging.

Errors found in Alpha 1 should be sent to the Haiku error tracking system.
