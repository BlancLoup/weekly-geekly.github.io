Ringmark: Mobile Browser Test Suite

Facebook and Bocoup have released Ringmark mobile browser test suite (the source code promises to open and donate to W3C over the coming weeks).



Ringmark conducts a quick browser test to support various specifications and provides a detailed report. Sets of specifications are combined into rings (rings), where each next ring means a more difficult level. The next test level does not start if the previous ring is not 100% full. In the first ring there are 83 tests, in the second - more than 300.

Over time, the developers plan to make more rings, in addition to the current three.
