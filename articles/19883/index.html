Surprise: opera: config in Opera Mini!

Yes it is. In connection with the recent upgrade of the server components of Opera Mini browser, the owners of the fourth version of the browser have the opportunity to make some corrections to the work of the server part for their phone. To gain access to these settings, it is enough to enter the address already familiar to the desktop version - opera: config. In the list of options available to change:

- Disable the display of empty squares in place of disabled images
- Disable text fit on page width
- Displays the Feeds icon at the top of the page.
- Enable display mode only headers Feeds
- Disable tracking mode for multiple feeds and display as a list
- Disable recognition of phone numbers on the page
- Change the minimum number of digits in identifiable telephone numbers
