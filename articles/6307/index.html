GTalk Online


First, when Google released Google Talk, the GTalk desktop application appeared, then integration with GMail took place. Some time later, a gadget appeared that added GTalk to the home page.

Now it is possible to launch Google Talk as a web application (a la ICQ2Go). To do this, you need to go to the Google Talk page and click on the button .

The usual desktop version will appear in the new window (only chat windows will be grouped by tabs) executed on a flash, so you will first have to install an Adobe Flash plugin for your browser.
