First mouse optimized for MMOG

New mouse production Razer (model Krait ) is optimized for online multiplayer games (MMOG) and real-time strategy (RTS). The developers emphasize that this is an important difference, since game mice are usually optimized for first-person shooters (FPS).

In appearance, the new mouse looks quite normal. The symmetrical shape is suitable for right-handers and left-handers, the scroll wheel is also a button at the same time. It seems to be nothing special. Nevertheless, experts are inclined to accept the novelty favorably , because the manufacturer, Razer, has a good reputation among gamers.
