HP Wireless Networks

Today, corporate networks are increasingly used by various wireless devices, from laptops, tablets and smartphones.

Under these conditions, there are questions about how to most easily and cost-effectively solve the problem of accessing corporate data from mobile WiFi devices. One solution is offered by HP - HP MSM access points. And the HP IMC network management software can centrally manage both wireless and wired network segments.

The HP MSM wireless access point series includes models with one (IEEE 802.11n / a / b / g) and two radio modules (802.11 n / a / b / g and a / b / g). The MSM-460 and MSM-466 models use 3x3 MIMO wireless data transmission, which allows them to take the place of the most productive devices on the market. Access points can be designed for both indoor and outdoor use with IP67 / NEMA 4X weather protection (MSM-466R). Access points can be equipped with built-in or removable external antennas. MSM access points require one Ethernet cable, which can also be powered using Power over Ethernet (PoE) technology. PoE IEEE 802.3af and IEEE 802.3at standards are supported. ( Read .)

Together with access points, HP MSM controllers are used to centralize control, automatically coordinate access point settings and provide user mobility. The controller architecture provides the use of 802.11a / b / g access points and the transition to 802.11n without the need to install new controllers by transferring authorized traffic bypassing the controller. Account profiles based on roles and identification information, as well as VSC policies with QoS, authentication, encryption, VLAN settings are supported. The controllers can be used in networks from 10 access points (MSM-710) to 800 (MSM-760/765) and up to 2000 (Mobility Traffic Manager). Internal and external authentication, authorization, and accounting (AAA) servers are supported. Controllers have a built-in state-based firewall and can assign VLANs to each user. ( Read .)


HP Planner software is used to simplify the design of HP wireless networks.

This software has all the characteristics of access points and typical building materials, which allows you to pre-plan a wireless network without measurements at the equipment location.

The HP MSM415 Wireless Security Scanner and the HP RF Manager System provide wireless security through IDS / IPS features that prevent IEEE 802.11a / b / g and 802.11n wireless security threats in 24 x 7 mode. ( Read. )

MSM access points work in conjunction with the AeroScout location software. Details can be found here , keyword RTLS.

We recommend using separate spectrum analyzers to troubleshoot problems in the WiFi network.
