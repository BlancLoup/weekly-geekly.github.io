Suggest.io: for people with queries



Live Search Suggest.io continues to improve.

Major updates:
  • 1. Due to the increase in the number of users, small changes were made in the registration / indexation of newly connected sites. Now the indexing begins after installing the code in the site page templates. Indexing progress can be observed from the personal account.
  • 2. A personal account system has been introduced - now you can customize the look of the search items, customize the search query output, including pictures in the issue results.

In the future, a large amount of additional functionality will appear in your personal account.



Personal Area.



Authorization is needed for those who want to change the default settings of the system and continue to use all the emerging opportunities.

Currently supported authorization through Mozilla Europe.



In your account, you can add all your sites to further customize your Suggest.io search.

To work with the settings, you must pass verification. After adding a domain, the system will offer to download a small .txt file - it should be uploaded to the root directory of your website. After Suggest.io checks for the file, you can proceed to the settings.



Personalized settings.



All so expected ability to customize the look of the search.
Design settings search elements occur directly on your site, after clicking on the link from the admin.



What can be customized:

- Template window search results. While there are not so many templates, but their number will be constantly replenished.



- Colors. Almost any. The palette is generated based on the colors used on the site.



- The size and position of the search window on the page.



For our foreign clients - quick setting of the language of search windows.

Settings search results.


Now in the search results you can customize the display of images from the pages of your site.
The function is configured from the Suggest.io settings panel.



To link illustrations to pages, you need to add a tag to the pages containing the main image address for the page.

Please note that it is necessary to replace the content of the content attribute with real addresses of images.
The tag format satisfies the Open Graph protocol specifications, and may already be used on your site to interact with other search and syndication systems. After adding a new tag to the site templates, you must return to the administer panel Suggest.io and enable the issuance of pictures, then start re-indexing the site.

An alternative way is to add the attribute class="sio-search-image" for tags that indicate the main page class="sio-search-image" .



The behavior of search results in the presence of illustrations is configured here. For example, do not quote text from a page if a picture is shown.

Changes in the search engine and indexer


Fixed:
  • Partial or complete loss of search functions on the site when nodes fall out in the Suggest.io cluster.
  • Invalid crawl site for some complex structures in robots.txt. Rewritten robots.txt parser.
  • Incorrect deduplication of pages with the active use of long repetitive textual structures (hidden through display: none).
  • Long delay when adding a new site to Suggest.io.


Additionally:

  • Improved handling of unclosed php-tags on rendered pages.
  • Automatic correction of incorrect links like " habr.rf / page " "/% 0% C1 / something", i.e. when incomplete percent / punicode coding of links is performed.
  • Accelerated search, improved stability and corrected hundreds of less significant problems.


For now.
Suggest.io team continues to search for solutions to improve the service. Keep for updates.
