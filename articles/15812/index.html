E-ten introduced the Glofiish X600: the budget version of the older brother

The company E-TEN Information Systems introduced the X600 - a budget version of the older brother, X500 communicator. Despite the fact that the index of the new model is higher, it is not a continuation of the X500 and opens up a new class of devices for E-TEN - budget communicators. The relatively low cost (which, unfortunately, is not mentioned) is not the only feature of the X600. According to official data, this is the smallest communicator Glofiish. Its dimensions are only 107 x 58 x 14.7 mm. For comparison, the dimensions of the X500 are 113 x 60 x 16 mm. The weight of the new 136 g.




The communicator is designed to work in GSM / GPRS / EDGE mobile networks and is equipped with a 2.8-inch touchscreen display with a resolution of 240 x 320 pixels. Wireless interfaces include Wi-Fi 802.11b / g and Bluetooth 2.0, as well as an integrated satellite navigation module based on the SiRF Star III chipset. Like other solutions Glofiish new communicator has a rich set of pre-installed proprietary software. There are utilities for speed dialing and launching applications, a world time function and a weather forecast.

The device is based on a Samsung processor with a clock speed of 400 MHz running Windows Mobile 6 Professional. The amount of RAM is 64 MB, constant - 128 MB, for expansion of which a microSD slot is provided.
