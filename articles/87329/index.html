OpenSource project is looking for developers

Hello, Habr! I want to introduce you to my non-commercial project and find like-minded / co-developers.

Motivational screenshot





about the project


Ajenti is a toolkit for managing Linux servers, aimed at simplicity and stability.
The project provides a platform for quickly writing plug-ins to manage various server software, almost a full-fledged UI-toolkit (on AJAX) for management via a web interface.
In addition, all already written plugins support (and new ones must support) the validity and accuracy of the corresponding server configs.
Also now a simple automatic backup manager is written.

Who are you looking for?


I am looking for people willing to help code to support a variety of server software. In the near future - writing plugins for
  • dhcpd
  • samba
  • apache and maybe nginx
  • basic operations with MySQL (database management and tables, simple queries with output of results, phpMyAdmin, in general, have long been written)


Designers and layout designers are also wanted to help with the controls / layout of the interface, because I am a programmer (:

If you are ready to participate in the project, then it is advisable:
a) know python
b) understand the software for the plugin for which you are going to take
