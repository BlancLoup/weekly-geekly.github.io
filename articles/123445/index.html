Craftsman added another display to the Nexus One



Apparently, the jack of all trades to a master named Brian Jepson really lacked an additional display, and he decided to attach another small screen to his Nexus One. What for? According to Brian, so that you can view messages received during the game. All this looks pretty monstrous, but if you really want something, you should definitely try to do it, right? There are a few technical details under Habrakat and links to software sources that allow all this pile of parts to work.

So, in the course of the work were used such parts and gadgets:

- Arduino Duemilanove with ATmega328

- SparkFun USB Host Shield

- LCD display

- Google Nexus One

In addition to the hardware, the craftsman wrote his own software, which allowed all this hardware to work. For writing software needed Android Open Accessory Protocol. Under the link below, you can find the source software written by Brian.



According to him, the program "hangs" in the invisible mode, and starts to work automatically when you connect the second display. Everything starts working automatically, without user intervention. When additional equipment is turned off, the program goes back to invisible mode.
