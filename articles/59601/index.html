Zend Framework 1.8.1 released


Download Zend Framework 1.8.1
Full zip (43.0 MB) | tar.gz (38.1 MB)
Minimal zip (4.5 MB) | tar.gz (2.8 MB)

Changes:
- Over 68 bugs fixed
- Dojo 1.3 instead of 1.2 (released by mistake)
- Zend_Loader :: registerAutoload () now calls Zend_Loader_Autoloader, setting the fallbackAutoloader flag to true.
- Fixed bugs and finalized Zend_Application
- Zend_Tool taught to generate modules.
- Strong object typing added to Zend_Amf

A complete list of fixes can be found in the bug tracker .
