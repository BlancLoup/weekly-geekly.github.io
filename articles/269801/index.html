Beta is getting closer - Vivaldi 1.0.303.40

Hello!

New build with new fixes - everything is also on the way to a full beta version.

Working with tabs

The main correction is that now you can again flip through the list of tabs, if there are a lot of them and they are all left or right. We also corrected the button for closing tabs - now it does not overlap with the scroll bar.

For those who prefer to place tabs below or above, there is also news. The close button is now hidden on inactive tabs, and on the active one it is placed over the site icon. In general, we tried to avoid unintentional closing of the tab as much as possible while maintaining the convenience of the functionality :-)

Import in Windows version

It's no secret that many of you are importing data from Opera 12. We have added the ability to import data from Opera 12 installed in portable mode (USB-mode). You can also import data from any version of Opera based on the Chromium core.

That's all for now. Download assembly can be on the links below:

  • Windows 32-bit
  • Windows 64-bit (experimental)
  • Mac
  • Linux RPM 32-bit
  • Linux RPM 64-bit
  • Linux deb 32-bit
  • Linux DEB 64-bit

Full list of changes:

  • VB-10159 Close button not hidden when tabs are tiny, tiny, tiny
  • VB-10224 Poor handling of many tabs on the left side
  • VB-10261
  • VB-10296 Update installer background image
  • VB-10273 Import from Opera 12 Windows standalone (USB-mode) fails

We remind you that the weekly assemblies are test, so do not forget to make backup copies of important data. You can report any errors found at the same address .
