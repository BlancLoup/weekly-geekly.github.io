Stream codes will enumerate all web content.

The channel switching system is called Radio Handi Stream Codes , or simply ‚Äústream codes‚Äù. Thanks to them, you can make it so that any multimedia content from the Internet (audio and video streams, podcasts, video archives) will be convenient to listen / view from almost any portable device.

Technology works like this . Each of the stream codes is a set of 4 to 16 digits. Thus, almost any name can be easily coded into a digital code (HABR = 4227). When a user dials one of these codes on a device that supports stream-code technology, the following happens: the device accesses the stream-code directory (similar to the DNS on the Internet) to get information which web address (or several addresses) corresponds to This combination of numbers (in the catalog information is stored at the address of the form guide.radiohandi.com/4227 ). Immediately after this content begins to be broadcast on the device.

It is assumed that the stream code directory will be stored in various formats: as a web service, for round-the-clock processing of incoming requests, as downloadable XML and CSV files, and also as an ENUM DNS table.

The system of stream codes is ideal for mobile and even landline phones, regardless of the age of the device (the main thing is that tone dialing is supported). For example, in this way it is very convenient to listen to podcasts. It is enough to make a special gate that will download podcast files from the Internet. This gate needs to register numbers in the fixed network of MGTS, in cellular networks MTS, Beeline, etc. Then the user from any phone calls the number within the network (by city number, for example), dials a stream code - and a podcast immediately starts broadcasting to him.

This, of course, is a very primitive example, but the developers of the technology of stream codes think much more widely. In their opinion, the support of this technology can be embedded into devices at the program level, as now the WAP protocol is implemented in most mobile phones. Moreover, stream codes can be implemented not only in cell phones, but also in TVs, handheld computers and any other devices that can connect to the Internet. Truly Napoleonic plan. We add that the creators of the technology distribute it under two licenses at the same time: a commercial license for paid services and a free license for free ones. The owners of the stream-code catalog are going to make money on paid inclusion (from $ 250 per month).
