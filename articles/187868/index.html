IE11 preview for Windows 7, updates for modern.ie, 25% discount for Parallels for Mac

Today, a preliminary version of Internet Explorer 11 for Windows 7 was released. Previously, it was only available for Windows 8.1 users. The new browser will provide high performance, greater speed of loading pages, and support for the latest standards required for the new generation of websites. In addition, developers will be able to take advantage of completely redesigned tools F12.



Download and try the IE11 Developer Preview today.

With IE11, developers can use professional-quality video on sites, hardware-accelerated 2D and 3D web technologies, including WebGL.

From today, all the benefits of IE11 are available to more than 700 million users of Windows 7. In addition, today we present an updated developer tool - www.modern.ie with free virtual machines and an online testing service for websites on all platforms.



Even faster and smoother internet


Speed ‚Äã‚Äãis one of the most important qualities of a browser. IE11 code is optimized to make sites appear as fast as possible, and web services run at the speed of applications installed on the computer.

IE11 is the world's first browser that decodes JPG images in real time using a computer's graphics processor. Due to this, the pages load faster, use less RAM - this helps to reduce power consumption and extend the battery life of mobile devices. IE11 is also the world's first browser that uses a graphics processor to render text. Text and images are the basis of the Internet, which means these two technologies will help you quickly display almost every page you visit.

In order to load pages faster, the browser must be able to optimally choose the order of loading of the site elements. IE11 is the first browser to use the W3C Resource Priorities standard. This means that developers can specify which page elements should be loaded first. Support for the SPDY protocol, the basis of the HTTP 2.0 specification being developed, also speeds up the loading of some existing sites.



You can yourself evaluate the speed of IE11 with the help of new demonstrations on www.ietestdrive.com . Pay attention to Levitation and Lawnmark : they allow you to estimate the speed of working with 2D and 3D graphics. The markup display can be assessed using LiteBrite as an example, combining the use of HTML5, CSS, and JavaScript.

High performance javascript


The JavaScript engine ‚ÄúChakra‚Äù has undergone significant improvements. It has become even faster, while maintaining compatibility with previous versions and a high level of security. IE11 also supports the latest ECMAScript 6 standard features described: let, const, Map, Set, and WeakMap, as well as __proto__ for compatibility.

The JIT compiler now supports new optimization methods, including polymorphic caching (polymorphic cashing) of properties and embedding of functions, which means that more code passes through JIT and less time is spent on calculations. Engineers working on Internet Explorer 11 focus on scenarios for working with existing sites. Nevertheless, IE11 in JavaScript tests, such as WebKit SunSpider, continues to significantly outpace competitors.



In addition, IE11 uses the ECMAScript Internationalization API (version 1.0).

3D hardware acceleration


Internet Explorer 11 supports WebGL, a new technology that will allow web developers to create websites with complex 2D and 3D graphics that are displayed using a computer's graphics processor. The WebGL technology, which was previously used only for experiments, is becoming an important development tool, allowing you to create entirely new types of interfaces.

For example, recently in partnership with GlacierWorks , an interactive website was created telling about the effects of climate change on the glaciers of the Himalayas, and about the effects of melting glaciers on the ecosystem. Today we have updated this site by creating an interactive 3D map of the Himalayas using WebGL. This is a great example of how a 3D interface can interact with photo and video content that already exists on the site. The site works on any device: for example, if you went to www.glacierworks.org from a tablet on Windows 8, you can appreciate another great IE feature: CSS device orientation, the ability to navigate a mountain range simply by taking the tablet in hand.



IE11 enables WebGL to work on any device with Windows 7 or 8, and can use GPU resources without compromising the security or reliability of the browser.

Professional quality video


Internet Explorer 11 allows you to use HTML5 video without additional plug-ins. IE11 supports closed captioning standards ( TTML and SDP ), so the videos you post on the site can be of the highest quality. IE11 also supports the HTML Full Screen API, which makes it easy to deploy HTML5 video to full screen while watching. IE11 can download videos without caching files on the hard disk, which helps to extend the battery life of devices.

New F12 Developer Tools


IE11 includes fully updated F12 developer tools to help optimize sites. IE team engineers worked on them together with the developers of Visual Studio. With the new F12 tools, developers will be able to quickly obtain the necessary data to help optimize performance, diagnose errors, and get complete information on how IE displays the site.

New F12 tools include:
  • UI debugging and memory usage diagnostic tool
  • Live DOM Explorer and CSS inspection that updates information at the same time as the page, giving you the opportunity to evaluate the impact of dynamic content on markup and styles.
  • JavaScript debugging - a tool that does not require page refresh




It is also important that the F12 tools are in many ways similar to the usual Visual Studio tools. Try the new F12 tools in the interactive review F12 Adventure .

Everything that is already working will continue to work.


Internet Explorer 11 is fully compatible with existing sites. It appeared API that will allow developers to make websites even better. Also, some APIs have been changed and removed so that the websites that work now continue to be displayed without errors in new versions of the browser. IE11 supports more HTML5 and CSS properties, such as enhanced 2D Canvas (anti-aliasing, padding, dashed lines), CSS border image, Device fixed positioning, Pointer Events and CSS Flexbox, mutation observers, and more .

IE11 includes many other changes. For example, the built-in text editor has been significantly improved: It will be easier for developers to create applications that use text editors due to the ability to insert images from the buffer, undo actions, etc.
A full overview of the new features in IE11 can be found here . And the IE team is waiting for your suggestions and wishes in Connect .

However, keep in mind that IE11 has a substantially updated User Agent String.
