IE6: Standoff

All web developers know what IE6 is. Some fight courageously, returning to the place of the incomprehensible where the fallen divs, driving the margines and swam padding, and at the same time fighting the PNGA and dodging the enclosed CSS-classes. Others (more cunning / lazy / principled) simply put a bolt on the layout for IE6, in the hope of forcing the user to update the browser / feel like sucker / leave the site.

There are also IE6 users who, for various reasons, are forced to use this terrible weapon of the Inquisition against themselves.

No, no one argues, at one time IE6 was not so bad. However, everything flows and changes, and in the XXI century it is time to put this rarity in a jar of formalin and remove it from sight.


jReject Plug-in




This jQuery plug-in, designed to notify the user that it is time to think about changing the browser. The notification may appear depending on the browser, the specified version, platform or engine. Appearance and list of browsers are fully customizable.

jReject Plug-in

Getbrowsers.com




Designed for those who have already decided to change the donkey to something more worthwhile, but still do not know what. The site offers direct links to the latest versions of 5 popular browsers: Google Chrome, Mozilla Firefox, Microsoft IE, Opera, Apple Safari. The developers did not neglect the browser from Microsoft to please those who are already used to it, or for some reason cannot / do not want to change it.

There is also a section for testers and fans with links to unstable versions (Dev, Alpha, Beta, RC), and information about updates is published via RSS and Twitter.

Soon the site will display a gallery of banners stimulating to change the browser.
