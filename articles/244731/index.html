GPS monitoring for personal use (part 2. Systems overview)

In this part of the article I will consider two Russian and one Belarusian system.

The choice was made according to the criteria: applicable for personal use, free of charge, and can work with tracking applications on smartphones (for each system, an application on smartphones with Android OS, created as part of a system developer company, is considered).

All systems provide access through browsers of mobile devices (not just opening the same page as browsers for notebook / desktop, but with changing the screen layout for small resolutions).

All considered systems have their own (developed within the companies) telematic servers. For all, it is possible to install them on the servers of customers. To clarify the specific conditions it is necessary to contact the company-developer. The important questions are: what resources will be required for this, how easy the system is to maintain and how the update process is organized.

There are much more systems on the market, but the ones selected are certainly worth considering when choosing tools for personal monitoring.

Perhaps I missed some important points from the point of view of system developers or users. Hopefully they will appear in the comments.


WHERE MY MY http://www.gdemoi.ru/ (Russia)




The company has been working in the telematics market since 2005.

Applications for personal monitoring: child minding; help for the elderly; care for novice drivers; active rest, tourism, sport.

The range of devices that are connected to the system is quite wide and also includes devices of its own production (which makes it possible to effectively integrate them into the system).

In addition to GPS / GLONASS monitoring in the system, it is possible to work with GSM cameras, which in my opinion is also interesting in the field of personal use.

Large corporate clients are also offered an option when the software distribution kit is installed on the organization‚Äôs server.

Restrictions for free fare : up to 2 objects; history for 1 day; 3 maps of Russia and the world. The free tariff plan is available only for trackers (for mobile applications it has become unavailable).

Useful features of the system


  • Mapping: 3 providers (out of 10); traffic traffic (Yandex, ProGorod, Google).
  • Search for a location on the map at. Geocoder can be selected (Yandex, ProGorod, Google, OpenStreetMap).
  • Geofences: circle, polygon, route. The route is laid instantly. At the same time, the corridor, specified at its creation (along the radius), is displayed.
  • Geofences: convenient editing, deletion and display of marked geofences on the map.
  • The ability to display a mini-map.
  • Custom layers can be added to the map (in KML format).
  • The list of objects is easily displayed / hidden on the screen. By selecting an object in the list, you can see trips and events on it for a specific time interval. For each object in the list displays its status icon.
  • When you click on an event selected from the list, the location of this event is displayed on the map.
  • Icons of objects on the map vary depending on the state.
  • If the information panel is open (at the bottom of the screen), then when you click on the name of an object on the map (or an object in the list), it displays detailed information about the state of the object.
  • Object dashboard consists of widgets. If all the widgets do not fit on the screen width, you can scroll left and right, switch places (dragging with the mouse) and hide.
  • You can simultaneously display on the map multiple tracks from the same or different objects. For each track, speed, mileage and travel time are available on the map.
  • A well-developed alert rule system. Notification rules can be created for different types of events: start / end of the trip; entrance / exit from geofence; over speed; control deviations from the route; alarm button; low battery; shutdown or loss of communication with the object and others. The rules set event parameters (permissible speed, battery charge, ...), text and channel (SMS, email) alerts, time schedule (by hours and days of the week) and geofences where alerts should be generated for these events.
  • Directly from the application you can contact technical support.

The application for smartphones "NAVIXY Android GPS Tracker"


Key features:
  • Setting the time and distance after which the location record is formed.
  • Gprs traffic saving mode. Setting the data transfer mode: continuously or at regular intervals.
  • Ability to configure the address of the server to which the transfer.
  • Battery saving mode: setting the gps receiver on interval.
  • Configuring the application widget: show the current state of the application or start / stop tracking when clicked.
  • Display technical information - the number of points not sent.


GPSHome http://www.gpshome.ru/ (Russia)




The monitoring system has been developed by GlobalSat LLC specialists since 2007.

Applications for personal monitoring: monitoring and ensuring the safety of children, the elderly, relatives and friends.

In addition to GlobalSat devices, dozens of different GLONASS / GPS trackers from other manufacturers can be connected to the system, including the popular models Falcom Stepp, Enfora, Xexun, Teltonika and others.

A professional GS Monitoring Online platform is offered for organizing your own monitoring service. GS Monitoring Online uses dozens of successful companies in Russia and foreign countries that offer GLONASS / GPS monitoring services under their own trademarks.

Restrictions for free fare : up to 3 objects; history for 1 day; no control of visiting geofences; There are no operational notifications about events via email or via SMS.

Useful features of the system


  • Mapping: Yandex, Google, OpenStreetMap; traffic traffic (Yandex, Google).
  • Geofences: polygon.
  • Geofences: easy editing, deleting, changing color, displaying marked geofences on the map.
  • A handy tool to add your own POI (Landmarks). Good set of icons. This is a kind of round geofence, with a given radius and an icon in the center.
  • The list of objects is easily displayed / hidden on the screen. By selecting an object from the list, you can quickly look at its history, trips and events for a specific time interval. For each object in the list, the battery charge and its status icon are displayed.
  • When you click on an event selected from the list, the location of this event is displayed on the map.
  • When you click on an object in the list (or on an object icon on the map), it is centered on the map and an information window opens for it. Right in it, you can choose a short or detailed information display mode.
  • For each track, speed, mileage and travel time are available on the map. Tracks can be exported to KML, GPX.
  • An advanced alert rule system is available. Alert rules can be created for different types of events: SOS; control of entry and exit from geozones; battery charge control; speeding control and others. The rules set event parameters (permissible speed, battery charge, ...) text and channel (SMS, email) alerts.
  • For the account, you can create additional users (operators). They can track objects and receive reports, but do not have administrative access to the personal account.

GPS-Trace Orange http://gps-trace.com/ (Belarus)




Founded: 2007 (but if we take into account that this is a product of Gurtam, we can assume that the system has been developed since 2002).

A simple and convenient system for personal monitoring: control and safety of children and loved ones. Allows you to record and share events with friends (including through social networks).

The company does not supply its devices, but the list of devices with which the system can operate far exceeds all existing systems.

Gurtam has developed a new Wialon Local service product ‚Äî a GPS and GLONASS monitoring system that can be installed on customers' servers.

Restrictions : in one account, you can connect up to 5 objects of monitoring (one car and four personal trackers); story for 1 month; maximum of 20 geozones per account.

Useful features of the system


  • Cartography: Yandex, OpenStreetMap, Gurtam Maps, Virtual Earth; road traffic (Yandex).
  • Search for a location on the map at.
  • Geofences: circle, polygon, polyline.
  • Geofences: easy editing, deleting, changing colors.
  • Button for quick transition to the maximum scale of the map.
  • The list of objects is displayed in the lower line of the screen. Left personal trackers, right - car.
  • When you double-click on the name of the object (in the lower line of the screen) in the lower part of the screen, the object information panel appears.
  • In the information panel, you can edit the parameters of the object. You can change the icon, device type, imei and sim card. Here the object can be removed from the personal account.
  • In the object panel, you can customize the track display modes and request a report by tracks. A handy tool for playing the track. For each track, when you hover the cursor over its points, the address, time and speed are displayed.
  • In the object panel, you can set up notifications about events on speeding, geofences, communication with the device and an alarm button. Notifications are sent to the specified email.
  • At the bottom of the screen you can show / hide the panel with the current message flow from objects. It also records the user's operations for editing information and parameters of objects.
  • Each object can be fixed (follow the object). At the same time, when messages arrive, the map will be scaled so that the object is visible.
  • A button is available to move to the application display in full screen.
