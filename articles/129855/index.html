MonoDevelop 2.8 release - integration with Xcode 4, support for iOS 5 and more



A new version of the MonoDevelop 2.8 development environment has been released for .Net, Mono, MonoTouch platforms.



The greatest number of improvements concerns the development for iOS devices. Here they are:
  • Integration with Xcode 4
  • IOS 5 support (including Storyboards)
  • Integration with Xcode UI Designer. Changes to the visual presentation editor are automatically detected by the environment.
  • Updated set of examples for iOS developers. Updated documentation. Added new project templates
  • Integration with TestFlight iOS application testing service
  • New plist-file application settings (Application Settings)


Other improvements:
  • New dependency editor (references) as in Visual Studio
  • New style editor code highlighting
  • Improved open document navigator
