WEB OS: 40+ network operating systems

Strange as it may sound, it turns out that there are quite a few operating systems whose task is to transfer your regular desktop applications to online. That is, having access to the Internet and a browser, you get other applications on the Internet using an operating system, which may be fundamentally different from the one in which you work. Some of these systems are familiar desktop, while others (start pages) collect information from the entire network. The following are more than 40 Web OS, worthy of your attention.

Remote operating systems




AstraNOS - with the mind of Windows 98. Or OS X. Or the evening sky. And perhaps all together :-). You now have a pretty good AstraNOS application.

cmyOS - free webtop, based on eyeOS.

Desktoptwo - you get not only 1 GB of space, but also a fully functional set of OpenOffice.org. No, not the basic online editor, which has simple formatting options. The complete OpenOffice.org 2.0 package from SUN, converted into a Java applet.

GCOE X - Good webOS with a powerful terminal and iPhone support.

eyeOS - Excellent Webtop on eyeOS operating system.

Glide is an online operating system with interesting features for BlackBerry, Palm, Windows Mobile, Symbian and iPhone users.

G.ho.st - 3 GB of space, ftp access, support for Zoho Office. Surely you will like this.

goowy is an excellent Webtop with the provision of a personal e-mail address (@ goowy.com), messaging services, 1 GB of space (via Box.net), and many others.

The original message contains references and descriptions of more than 30 WebOS: standalone, desktops, start pages, etc.
