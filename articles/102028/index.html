Free Windows Phone 7 Jump Start Course

There is less time left before the launch of Windows Phone 7 and more and more materials for developers appear in the public domain.
Let me remind you that Russian developers can register in the Windows Marketplace now and from the first day of launch they can download their applications (both paid and free) for phone buyers from other countries. Development tools are completely free .
Today all the materials of the Windows Phone 7 Jump Start course are available - these are 12 hour lessons that completely cover the technical aspects of developing applications for a new platform:
  • Introduction
  • Building a Silverlight Application, Part 1
  • Building a Silverlight Application, Part 2
  • Building Games for the Windows Phone 7 Platform
  • Building XNA Games for the Windows Phone 7 Platform, Part 1
  • Building XNA Games for the Windows Phone 7 Platform, Part 2
  • Advanced Application Development, Part 1
  • Advanced Application Development, Part 2
  • Advanced Application Development, Part 3
  • Marketing Your Windows Phone 7 Application
  • Working with media
  • Final Silverlight Topics and Wrap-Up
