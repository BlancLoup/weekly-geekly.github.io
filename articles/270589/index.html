A selection of free tools for developers

Today we bring to your attention an adapted selection of tools (including cloud) for developers who allow you to create truly high-quality projects. Here are presented only SaaS, PaaS and IaaS services that provide free packages for infrastructure software developers.

/ Photo Informedmag CC

Source Repositories


  • bitbucket.org - free private repositories (GIT and Mercurial), which can work with up to five people;
  • chiselapp.com - unlimited free public and private Fossil repositories;
  • github.com - unlimited free public repositories;
  • about.gitlab.com - unlimited free public and private GIT repositories. The number of participants is also unlimited;
  • hub.jazz.net - unlimited free public repositories. Three free private repositories;
  • visualstudio.com - unlimited free private repositories (GIT and TFS), which can be used by up to five people at a time;
  • assembla.com - unlimited free repositories;
  • www.fogcreek.com/kiln - unlimited number of public and private repositories (a hybrid of GIT and Mercurial) for two users;
  • www.plasticscm.com - free repositories for individual developers, open source projects (OSS) and non-profit organizations;
  • Cloud Source Repositories are free private GIT repositories deployed on the Google Cloud cloud platform. The service supports synchronization with the GitHub and Bitbucket repositories. Storage size up to 500 MB.
