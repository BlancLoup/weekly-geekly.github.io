Release Ubuntu 11.10 (Oneiric Ocelot)

A brief illustrated overview of the improvements to the Ubuntu 11.10 user interface.

Unity and Unity 2D


The standard GNOME shell is not included in the list of default programs. In the list of session types, Unity and Unity 2D are available for low-power devices with weak graphics cards.

Desktop




Ubuntu button moved to the Launcher panel. When opening a large number of applications, part of their Launcher icons are folded into an accordion.



The indicator panel has also changed, additional indicator settings have been added, work with power management has been improved.



The Alt + Tab window switch has become more convenient and functional. Another harmonica, but horizontally. Switching between windows is carried out by the keys "left" and "right"



Now the windows of one application are grouped, you can open the group with the "down" key.

Dash




Dash interface has been completely redesigned, blur effect added







Control center




According to the developers, with the new control center, setting up the system will become simpler and clearer.

Ubuntu Application Center




Ubuntu Application Center is ported to GTK 3. The new version of the program is even more integrated into the system, it has become faster and much more convenient. The rating system and application reviews have changed.

Applications


  • Evolution mail client changed to Thunderbird 7.
  • The utility for creating backups Deja Dup is now built in as the default program.
  • The Synaptic package manager has been removed from the standard installation.
  • Gwibber has become much faster and more attractive.
