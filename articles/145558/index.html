Mozilla Shumway - open source SWF to HTML5 converter

Under the Mozilla account on github, the development of the open source project Shumway has begun - this is a technological experiment in which explores the possibilities of rendering SWF without using native code, that is, using HTML5 and JavaScript.

Mozilla hopes that such a tool will still be created and will be used everywhere as a universal open standard for parsing and rendering Flash-animation. Integration with Firefox is possible if the experiment is successful.

SWF Inspector

Shumway Demo Page

This is not the first attempt to write a Flash-player on HTML5. For example, for almost a year, such a Swiffy converter has been developed at Google, although Google has not yet opened the source code, although it promised to do it. In addition, Swiffy uses features of SVG, which were initially supported only by browsers on the Webkit engine, and only SWF 8 and ActionScript 2.0 are supported.
