AWS: EBS Image Copies

Hello!

From now on, you can copy EBS snapshots to other regions . This is an excellent and convenient feature that allows you to clone your services and data in different regions using native methods!

In the menu snapshots appeared option Copy Snapshot:


In this menu, you can select the region where you want to take this snapshot.


Selecting another region, click Yes, Copy. And we see in the selected region:


The eight-gigabyte snapshot in the example was flooded from US-EAST-1 to EU-WEST-1 (S. Virginia, USA - Ireland) in 3 minutes . Speeds are generally good .

Also in the latest version of EC2 API TOOLS 1.6.6.0, the ec2-copy-snapshot function appeared. Using it is also easy to automate this process.

Copying EBS images will greatly simplify the DR procedure in another AWS region. For me, the AWS solution architect, this function is like manna from heaven. Thanks, AWS!

UPD: I forgot to note that the cost of transportation is considered to be normal traffic. As always, the price includes only outgoing at the price of the region. Incoming - for free.
