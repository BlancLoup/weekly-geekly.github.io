Hyperboria: Internet 2.0



Hyperboria is the Open Source implementation of the Mesh networks project, an analogue of the well-known and desired Netsukuku project, which, unfortunately, has died.

In other words, Hyperboria is a self-organizing decentralized network, which itself builds routes between nodes.
Or, to put it simply, this is what the Internet should have been, free, uncensored, fast and automatically scalable.


A short introduction to the Mesh Network, for those who are not familiar with them




What is it ?


This is a peer-to-peer network that operates in two modes:
1) Through the Internet creating an overlay network (which works ten times faster than Tor and I2P)
2) Between routers directly.

How it works ?


As I already mentioned, the network operates in two modes, but the basic principles of operation are the same for both modes.

To access the network, you need to install software that acts as a router - CJDNS

During the installation, your public and private keys will be created, a tunnel device will be created for which iPv6 address will be generated.

Then, to connect to the network, you must:
1) When connecting via a regular Internet, find an already existing network node and find out its address and key
2) When you connect a router router - everything happens independently.

After connecting, you will get access to the entire network, your traffic will be encrypted, it will not be possible to decrypt or censorize it.

Network goal


Create small islands of network freedom on the basis of a router-router and unite them together via Internet channels.

Due to the fact that the network does not have access to the normal Internet, users submitting their channel will not be afraid to get a subpoena for breaking the bank through its router.

In this way, the network will be able to grow and cover new spaces.

IPv6


On the network, you generate an IPv6 address that belongs to the private part of the IPv6 address, which means there will be no collision between the real IPv6.

In addition, any application will be able to work on the Hyperboria network if it supports work through IPv6.
After installing the necessary software, the traffic is automatically redirected to this network, which allows you to avoid additional infusion of programs.

Anonymity & Privacy


Hyperboria network is private - your traffic can not be decrypted by anyone except the node to which it should be delivered.
The network is NOT anonymous, the purpose of the creation was not to make an I2P / Tor clone, but to make a new Internet, so using the trace you can find out the chain of nodes and find out your IPv4 address, it is obvious that when connecting a router-router type, this problem will disappear and the network will become same anonymous.

Traffic routing


Traffic routing is provided using a system similar to Kademilia DHT, more precisely, the route catalog is constantly updated due to the fact that the network configuration can change, thus the network maintains an optimal load through all nodes and selects the shortest path for traffic.

DNS


Currently, a system is being developed that could replace the DNS, making it decentralized, options for using Namecoin or similar services are being considered, now domain name conversions are performed using the existing DNS service of the regular Internet (prescribing AAAA addresses)
