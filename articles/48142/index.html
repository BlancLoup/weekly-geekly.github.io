Delete the send feedback line in Windows 7

In all the beta of Windows 7 at the moment there is an opportunity to write to the developers about the errors found.

Next, I will describe how to get rid of it, at least in Build 7000

We need to change the registry a bit.
So, we start regedit
Go along the way

HKEY_CURRENT_USER \ Control Panel \ Desktop

Double click on FeedbackToolEnabled and change the data line value data to 0


Now you just need to login again with your user.


And it is better to use this opportunity for the intended purpose and to let the developers know about their mistakes and the possibilities of improving the system.
