The fidoip software package, starting from version 1.0.5, has added support for hypertext Fidonet

On December 30, Linux.org.Ru posted the news that the cross-platform package of Fidonet programs, called fidoip , was recently included GoldED-NSF , a special version of the popular GoldED + fidmail editor, which allows you to go to FGHI URLs as hyperlinks. This means that hypertext Fidonet support has been added to fidoip.
