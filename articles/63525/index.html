Testing and Deployment Tools

Having compatibility issues with your applications? To do this, there are tools that will help solve these problems.

Tool kits and examples:


Microsoft Deployment Toolkit (MDT) 2008 Update 1 (571 KB - 20.7 MB)
Microsoft Toolkit (MDT) 2008 is a tool for distributing (installing) OC desktop and server applications.

Microsoft Application Compatibility Toolkit 5.5 (404 KB - 20.6 MB)
The Microsoft Application Toolkit (ACT) contains the necessary tools and documentation to assess and mitigate application compatibility issues before deploying to Windows 7, Windows Vista, Windows Update, Internet Explorer.

Microsoft Assessment and Planning Toolkit (27 KB - 219.3 MB)
The Microsoft Assessment and Planning Toolkit is a tool that helps you assess the current IT infrastructure for various migration methods.

App-V is a tool for resolving compatibility issues.
