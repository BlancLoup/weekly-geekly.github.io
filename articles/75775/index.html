Knoppix 6.2 release

A new release of the popular LiveCD / LiveDVD distribution KNOPPIX. Version 6.2 contains a completely updated set of software synchronized from the Debian branches 'Lenny', 'Testing' and 'Unstable'.

Some features of Knoppix 6.2:
- The new version uses Linux kernel 2.6.31.6, LXDE 0.4.2, X.Org 7.4, Compiz 0.8.2, OpenOffice.org 3.1.1, Firefox 3.5.5, Thunderbird 2.0.0.22, Gimp 2.6.7, Mplayer- svn20090405, Pidgin 2.4.3, VirtualBox 3.0.10.
- A complete reworking of the Microknoppix boot system was made, the possibility of parallel execution of various initialization stages was added, which led to a significant acceleration of the load.
- Ensured a high level of compatibility with Debian GNU / Linux.
- In addition to the standard LiveCD version of the lightweight graphical environment LXDE, the ability to use GNOME and KDE has been added to LiveDVD.
