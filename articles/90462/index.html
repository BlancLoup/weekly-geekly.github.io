Open source platform for building applications and package manager for Windows

Gaerett Serack, a Microsoft employee, published a note in his blog that he would work with the new coapp.org project (The Common Opensource Application Publishing Platform), which would combine:
  • infrastructure to facilitate the development and assembly of Open Source applications for Windows
  • package manager for Windows
CoApp will not be a port of any existing unix-infrastructure, but will fully use Windows-based technologies - WinSxS, MSI, Windows Error Reporting.

The message on slashdot also says that he will dedicate all his working time to this, but I did not read this out on his blog, although he is present in the comments and does not protest.

It is interesting that this will all turn out - while the project is at an early stage.
