Twitter Friendly Links Plugin

Most recently, I started developing plugins for WordPress. Three days ago I released my newest - Twitter Friendly Links called. In general, this is something like TinyURL (tr.im, etc.) only within its own domain (blog). For example:

kovshenin.com/wordpress/plugins/twitter-friendly-links

turns into

kovshenin.com/712

Very useful where there is a limit on the number of characters, such as Twitter. Sobsno therefrom and the name follows. WordPress.org Plugin Page: Twitter Friendly Links

Full description and discussion on my site (alas, in English): Twitter Friendly Links
