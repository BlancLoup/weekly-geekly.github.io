New Windows 10 and other fantastic news from Microsoft



A few hours ago, the broadcast of the Windows 10: The Next Chapter event ended, where Microsoft presented a lot of different new products, some of which, more recently, we could only present in the movies and on the pages of science fiction novels.

Everything that you read below is not some distant future, but very, very near even.

The main announcements of this event were related not only to the software, there was enough iron news and news for gamers:
  • New Windows 10 Consumer Preview
  • New Windows 10 for mobile devices
  • New browser
  • Xbox Live client and games from Steam for all devices on Windows
  • Surface Hub with 84-inch screen
  • Holographic glasses augmented reality HoloLens

Everything in order.

New Windows 10 Consumer Preview


Windows 10, the new operating system that Microsoft introduced in September , has now received many nice changes and, very soon, will be available to users participating in the Windows Insider program for trial use.

1.7 million people have already connected to this program and began to use the new operating system on their computers.

Completely different devices, such as tablets, laptops, personal computers, even XBox, can now work on Windows 10.

After the release of the operating system (dates are not announced yet), everyone will be provided with a free upgrade from Windows 7, Windows 8 and Windows 8.1 to Windows 10 for the first year .

The developers did a good job to enable users to use the same operating system on different devices, form factors and using different input methods.

The operating system will implement both windowed mode familiar in Windows 7 and a mode optimized for pressing, similar to Windows 8. The Start Panel will also adapt to the mode of using the device.



Voice control Cortana now comes in personal computers. On the taskbar, Cortana has its place where useful notifications are displayed. You can communicate with Cortana, which, over time, learns about your interests and preferences, becoming a personal assistant. Cortana can already quite a lot. Including, find and run applications.



New Windows 10 for mobile devices


The operating system for phones and mobile technologies also do not stand still.
Microsoft is preparing a new operating system Windows 10 and mobile devices.



The developers have significantly improved the Notification Center, integrated Skype into the message screen, worked on the convenience of text typing and much more.



In parallel with Windows 10, a huge range of universal applications is developed, with which you can work on computers, tablets and smartphones. These applications should cover the basic needs of users.

First of all, this is an office suite. The full-fledged tools for formatting and editing documents Word, Excel, PowerPoint and of course Outlook will be built into the mobile operating system.



In addition, there will be applications not only for work, for example:
  • universal application Photos with collections, autocorrection, synchronization with OneDrive;
  • A new music application that syncs between devices, and music can be stored on OneDrive;
  • new beautiful contacts and much more.




New browser


Today, a new browser, codenamed Spartan and a fast rendering engine, was shown.

The browser has received a new interface that fits into the overall design of Windows 10. One of the key functions is the ability to write notes on top of web pages. With the stylus or finger, you can mark and sign the desired object or leave a comment.



The new browser will have a reading mode that allows you to read articles without ads and unnecessary elements, and Cortana itself will tell you the way to the restaurant, which is open in the browser.

Xbox Live client and games from Steam for all devices on Windows


For gamers and just addicted, Microsoft announced the built-in Xbox Live client for all devices on Windows 10.



In one application you can see all the games that belong to the user, his activity and information associated with them. It is important that games from Steam, which are used by many people, come here.



Want to play and you have Windows 10? Just press Win + G. The Xbox application opens.

You can record game clips right on the PC. Even if the game is from Steam. Then add an entry to your profile.

Two players can play the same game simultaneously on different devices. The new Windows will use DirectX 12, which will improve the performance of games.

Surface Hub with 84-inch screen


Surface Hub - a new device for business. It can be convenient for meetings or discussions of some work tasks.

You can write and draw directly on the device, display content from any other device, using it as a large touchpad.

In addition, you can connect to the screen remotely, use Skype and any other applications.



For technical specifications, not much is known yet: 4K resolution, 84 inches, built-in microphones, speakers, camera.

Holographic glasses augmented reality HoloLens


Welcome to the world of holograms with Windows 10!
Microsoft introduced the HoloLens device, glasses that allow you to dive into augmented reality.



By the way, for HoloLens do not need a phone or computer. They are themselves a computer. In these glasses you can work not only with special programs, but also with any universal Windows applications.

HoloStudio allows you to create your own holograms, your 3D-objects, even print them, in the presence of a 3D-printer.
The environment supports the management of voice commands, you can create your own objects, immersing yourself in a virtual environment. All this is controlled by simple and natural gestures.

Frames from a sci-fi movie? Nooo, this is just Microsoft's HoloLens!


Good news for developers, the Holographic API promises to be included in Windows 10, that is, on all devices with Windows 10 there will be a Holographic API.

When to expect?


Windows 10 Consumer Preview will be available on Windows Insider next week!

New Windows 10 for mobile devices will appear a little later, in February.
