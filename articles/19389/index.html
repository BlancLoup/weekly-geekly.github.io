SQL Server 2008 is delayed

Microsoft has announced that the release dates of SQL Server 2008 are being moved to Q3. 2008, because the development of the DBMS unexpectedly strongly delayed. It was originally planned to implement a joint release of Windows Server 2008, Visual Studio 2008 and SQL Server 2008 at the end of February, but now it is clear that programmers do not have time.

Instead of the final release on February 27, 2008, a full-featured preview version will be released. Then in the second quarter. 2008 release candidate will appear, and only in the III quarter. 2008 - the official version.

SQL Server 2008 is a highly anticipated product. The number of downloads of technological preview has already exceeded 100 thousand. The list of new functions includes improved scalability, a resource reservation system for specific users / Resource Governor processes, support for spatial data type (spatial), compression modules, libraries for distributing privileges for databases and individual tables, support for structured and semi-structured data, including digital formats for images, sounds, video, and other types of multimedia.

According to market analysts, the revenues of the Microsoft database division are growing faster than the revenues of similar divisions from competing companies Oracle and IBM. And in general, this division is becoming one of the most profitable and fastest growing at Microsoft.
