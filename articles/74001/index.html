Socionics



I have long been fond of various typologies: socionics, psychosis and others (there are a lot of them in general).
Socionics is a teaching about the exchange of information between people. It involves the division of all people into 16 types or so-called. TIMs (Types of Informational Metabolism), each of which is described by its own model of information processing. In turn, all information is divided into 4 types (intuition, sensation, logic and ethics), which are called aspects. Engine



In April-May of this year, I decided to write a habra-like, blog-social engine. In early June, he was ready, but all summer I was busy first with the session, and then with my work, that I had temporarily forgotten about the engine.

Briefly list the main features:
  • Communities moderating communities
  • User Blogs
  • Friends, private messages
  • Favorites
  • Profile and Site Settings
  • Guestbook for each user
  • Karmapodobnaya rating system
  • OpenID support


The engine itself is not much different from a technical point of view (although in the spring I thought I was doing something brilliant :-) :-) :-) :-)). Classic MVC (custom samopisny framework). No ORM is used (which is a huge minus). The client uses jQuery with its many plugins.
