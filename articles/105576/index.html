KlavaDict


KlavaDict is a web application for memorizing foreign words. At the moment there are three dictionaries: English-Russian, German-Russian and Esperanto-Russian. The principle of work is that the Russian or foreign word is displayed and 5 variants of translation, among them there is only one correct one, which must be chosen. In the event of an error, the correct version will be highlighted, and the word will become more common until it is permanently fixed in the memory. One lesson lasting 5-10 minutes (100 words), although you can stop at any time, it will not affect the result of training. I guarantee that after the first lesson, learn a few new words, unless of course you already know all the words from the dictionary.

dict.klava.org

The numbering of points is done so that you can select options using the numeric keypad.
When you click on the title, you can return to the beginning, to the dictionary selection screen, although in this case the statistics will be reset.
The application is optimized for small screens, it can be used on the iPhone / iPod Touch (maybe on other platforms, I do not have the opportunity to test now).
