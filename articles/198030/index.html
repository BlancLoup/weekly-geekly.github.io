Yandex launches PaaS Cocaine

Yandex released PaaS (Platform-as-a-Service) based on its own Cocaine product, an open source system that allows you to create analogs of Google AppEngine or Heroku.
Currently, C, C ++, Python, and Perl and JavaScript are supported.

Cocaine uses LXC - Linux Containers, a lightweight virtual machine similarity, using the Docker wrapper written in Go.

To create your own cloud, you just need to run cocained on one or more machines. Clients that connect to cocained are called Cocaine Dealer, and have a FastCGI interface that can be easily screwed to any web server.
All applications in the cloud run independently of each other.

Cocaine is already used in:
  • Yandex Browser
  • Yandex.Passport
  • Yandex internal infrastructure
