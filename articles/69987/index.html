Another reason to start using Chrome

Remember "Dandy" (aka NES - Nintendo Entertainment System)? This is an eight-bit gaming console released in 1985. Among other things, it was she who introduced us to legendary games like Super Mario Bros, Tetris and The Legend of Zelda.

What does all this have to do with Google Chrome? It's simple. Ben Firshman ported vNES (NES emulator) by rewriting it in JavaScript, and named it JSNES. It works great in Firefox 3.5 and Safari 4, but it works many times faster in Chrome.

In order to start playing, simply open this address in the browser , select the ROM with the game you need.

Besides the fact that it will help you to pass the time, this example shows that all the conversations around the optimization of JavaScript operation in Chrome were not empty chatter ...
