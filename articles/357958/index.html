Thermogram Raspberry Pi 3 Model B

Bad news for those who are going to buy a new Raspberry Pi 3 .

Gareth Halfacree , author of the Raspberry Pi user manual The User Guide has published thermograms for various Raspberry Pi models , including the new Raspberry Pi 3 Model B on the BCM2837 SoC.

It turned out that the Raspberry Pi 3 Model B with a 100% CPU load (without using the GPU) warms up to 101.3¬∫C for five minutes!

At such temperatures, mini-computers are unlikely to be able to work normally in a case without active cooling. Well, at least the holes for ventilation must be left in the case.
