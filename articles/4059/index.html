Yahoo Livesearch: live search

The developers from Yahoo announced the introduction of the Livesearch technology (‚Äúlive‚Äù search) on the AlltheWeb search site, which updates the search results on the screen during the search query input process.

Like Google Suggest , Livesearch technology performs the function of auto-filling the search bar in real time. That is, when you type a search query with the introduction of each new character on the screen, the drop-down list is updated with options for search queries.

Unlike similar technologies, Livesearch not only lists the most likely options for auto-completion, but also displays search results in the right-most column for the most likely option. Theoretically, this allows you to significantly speed up the search on the Internet, because there is no need to type the text of the search query to the end and press the Enter key. In practice, the speed of appearance of search results on the screen is not too high yet, even if you set the minimum number of results on a page (10), and if you select the maximum value (100), the search will work quite slowly. You need to enter the query text rather slowly to see the search results on the screen. However, this can be called minor shortcomings, which may well be corrected in the future. The system is already trying to analyze how fast the user is typing. Accordingly, the page is not updated every time a new character is introduced, but less often.

Advanced settings allow you to make search results appear in a new window. By the way, contextual advertising is updated along with normal search results in real time.

Livesearch is currently in beta testing. The program does not work in the Opera browser (apparently, it will be fixed soon), but it works fine in Firefox and IE.

On the AlltheWeb search site, Yahoo is testing a variety of innovative technologies. For example, it was here that for the first time on the Internet, the calculator function from the search line was implemented. The Livesearch technology originates from the already well-known Instant Search technology.
