FireFTP: Firefox extension

In our understanding, an FTP client can only be an independent program. But the developers of the Firefox browser platform decided that the program for working with a remote server could be part of a popular browser and created an ftp client called FireFTP, which is installed as a Firefox extension.

The program weighs only 133 KV, is installed in seconds, and after rebooting the browser is available in the "tools" tab.
