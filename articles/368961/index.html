Google tests black links in search results



Google search can greatly change its appearance. This will happen if the results of A / B testing that is going right now prove successful. Since May 8, Google is conducting an experiment , replacing for some users the search results from the usual blue links to black ones.

A / B testing assumes that some of the users for the same search queries get results with blue links, and some with black links. Then the effectiveness of the issue is compared, that is, how often users navigate through advertisements and search results.
