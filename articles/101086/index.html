Tab Grouping in Opera

Recently there has been a lot of talk about grouping tabs in browsers as a very convenient way to manage tabs, but did you know that in Opera this possibility existed for many years?

This method may not be completely obvious, but the Windows panel in the browser, which shows open windows with their tabs, can be used to create groups of tabs. The panel is not enabled by default, so you will have to add it either in the settings, or by clicking on the plus sign in the panel and choosing ‚Äúwindows‚Äù.

Therefore, now remember: "Window = group".

Now new windows will create original tab groups, each of which will be called according to the active tab in the window. Tabs can be dragged between groups / windows, to create a new window, simply pull the tab out of the window. In this case, "Quick Search" will help you navigate among all the tabs.

Opera can also remember all open windows on exit. To do this, close the browser through the menu item "Exit", and not a cross or Alt + F4. Also, the browser remembers closed windows as well as tabs.
