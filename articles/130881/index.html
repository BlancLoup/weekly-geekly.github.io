Gmail updated again



It seems that Google is now very productive period - there are new products, the old products are being improved, there are new features and design elements. The mail service is no exception, its quite significant update is now being planned, and despite the fact that the new Gmail has not yet been submitted, you can already see how it will look after making changes. The fact is that the Gmail developers have posted a video of the updated Gmail on the official YouTube channel of Google.

According to the developers, Gmail has undergone significant changes. The service interface has been updated, many things have been revised, many things have been added, so that users should be happy. First, Gmail has become "rubber", now the system interface will automatically adjust to any window size. Secondly, users will be able to resize many interface elements, including the size of the chat window. Gmail themes now include high quality images. "Branches" of messages will receive pictures for profiles, plus their design will be greatly changed. The search has become even more user friendly.

The developers claim that the current service update is the most significant since the appearance of the mailer from Google. The Gmail team has tried to make the service more modern and also more user friendly. Judging by the video, all this is really quite convenient.
