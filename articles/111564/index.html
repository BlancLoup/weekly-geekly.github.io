GECK or World Village Creation Kit

GECK (Compact Eden Generator, Group of Eden Structural Components, Building Paradise Town). A lot of game hours spent in Fallout 2 to find this coveted bag. More hours spent in thinking about the implementation of such a project in reality.
And there is a similar project in reality, although in size it is somewhat larger than the case. The objectives of the project called The Global Village Construction Set to develop a set of inexpensive and open tools for building settlements.



Of course, he is a set of 40 industrial machines that allow you to create a small civilization with modern comfortable conditions. The set resembles Lego, a life-size designer, including interchangeable motors, parts, and power units. Currently 8 out of 40 machines are created. All documentation is posted on the openfarmtech.org wiki.
