Video review tablet Asus VivoTab RT



Asus VivoTab RT (TF600T) - a new tablet from Asus, powered by the Nvidia Tegra 3 processor running the Windows RT operating system (Windows 8 version for tablet computers based on ARM architecture). With its design and layout, the device continues the traditions established in the Eee Pad Transformer lineup. In addition to the tablet itself, a keyboard dock with a built-in battery is attached, thanks to which the tablet can be used as a full-fledged netbook.

VivoTab RT comes with two gigabytes of RAM and 32 or 64 GB of internal memory for data storage. In addition, Asus provides three years of free access to its WebStorage network storage of 32 GB. From wireless communication, only Bluetooth 4.0 and WiFi (802.11 b / g / n) are available for this model.

Unfortunately, Windows RT does not support standard applications for personal computers (x86, x64), the installation of new software is possible only from the built-in Windows application store, which does not yet please the user with a huge range of programs. The advantages of the system include the presence of a full-fledged MS Office 2013 and good compatibility with peripheral equipment, such as printers or external hard drives.
