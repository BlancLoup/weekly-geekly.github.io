One-button video hosting - Vimple

What is it?

Vimple is a video hosting service that allows you to upload an unlimited number of videos for later insertion of the player code on a web page (website / blog).



No site / blog? Vimple provides an opportunity to send a personal link to your friends via e-mail / skype / ICQ / VKontakte / G +, etc., following which they will watch the video directly on the site vimple.ru

What are the limitations?

Video format - any of the existing
Duration - any
The file size (and this is the only thing Vimple will limit you to) - up to 1GB.

What's new?
