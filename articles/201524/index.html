Release of Tizen platform version 2.2.1



On the official website of Tizen a message appeared from the Technical Steering Group that a new version of Tizen was released - 2.2.1.

The release of the new version of the Tizen platform means the publication of a new Tizen SDK and new images of the operating system for devices for developers.

Tizen 2.2.1 contains new features in the personal data management system and in the basic components. Improved user interface performance. Development bugs fixed and new API functions added.

The following innovations stand out especially:
  • An item responsible for personal data has been added to the settings menu.
  • Applications can be installed on the SD card.
  • The firmware update tool supports the USB.org and Mac OS X debugging system .


More detailed details about this version of Tizen are available in the release notes and the API change list .

It is worth using the new Tizen SDK when preparing your applications for the Tizen Store , the Tizen App Challenge contest, which was extended to December 8th, and the VKontakte client contest.

The Lubuntu 13.10 boot image will be updated to use the latest version of the Tizen SDK. We also planned new trainings and hackathons to develop for Tizen, which will be reported a little later.

Follow Tizen news in our hub , Twitter and VKontakte group .
