PHP Digest number 63 - interesting news, materials and tools (May 11 - 31, 2015)



We offer to your attention another selection with links to news and materials.

Enjoy reading!

News and Releases


  • Accepted PSR-7 HTTP message interfaces - The PHP-FIG group adopted the PSP-7 standard governing HTTP operations. The standard is already applied in Guzzle , Symfony , and the implementation is also available in the form of zend-diactoros .
  • PHP 7 alpha 1 will be released on June 11 - Release PHP 7 managers have announced the first releases of the new version of the interpreter. In the near future, we expect alpha releases, and the final release is planned for November of this year. A list of all the RFCs included in PHP 7 is currently available .
  • Symfony 2.7.0 LTS Release
  • Updates to current PHP 5.6.9, 5.5.25 and 5.4.41 branches - A number of security bugs have been fixed in the releases.
  • Elastica 2.0.0 - The most popular PHP client Elasticsearch has changed the major version. And immediately Elastica 2.1.0 .
  • Release Deployer 3.0 - Fresh release, many improvements, parallel deploy. Sent by elfet .
  • Guzzle 6.0.0 - The most popular HTTP client has been updated. The new version uses the standard PSR-7, involved its own implementation of promises, as well as many other improvements.
  • Phalcon 2.0.2 release
  • DevConf 2015 - June 19 in Moscow will host the next annual conference for web developers. Among the speeches in the PHP section , a report from Dmitry Stogov on PHP 7 is planned.


Php


  • RFC: Throwable Interface - Along with the implementation of exceptions, the PHP kernel also adopted a new exception hierarchy. In this case, the BaseException class, base for all types of exceptions, was introduced. Now it is proposed to get rid of the base class, and instead introduce and use the Throwable interface.
