123D Design by Autodesk



Autodesk has released a free 3D modeling program called 123D Design . With it, users can quickly and effortlessly create 3D-models.

Christian Pramuk (Autodesk Manager):

- Unlike heavy programs for modeling, 123D Design is more understandable and intuitive. Users who have only basic skills in this field will be able to work with him. Or do not have such skills at all.

The program allows you to create 3D-objects using a set of basic forms and their modifications.



There are also a number of ready-made objects for editing (for example, a robot).

The application is available for Windows, Mac and iOS (iPad) and is compatible with products such as 123D Sculpt, 123D Catch, 123D Make.
