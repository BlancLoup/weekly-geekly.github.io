3CX Phone System v14: version for technical specialists

We are releasing a new release of 3CX Phone System! Our development team did a great job and presents an innovative solution, ready to be installed in the cloud, with new and improved features - 3CX Phone System v14: a version for technicians.
The new version of 3CX Phone System 14 will especially appeal to cloud service providers. The previous 3CX Cloud Server system has been replaced with a unified version, which allows you to install 3CX Phone System v14 both as a normal local server and as a virtual PBX. At the same time, it is possible to install up to 25 independent virtual PBXs on one physical server. The creation and management of virtual PBXs is carried out from the 3CX partner portal or through an API. Compared to other cloud platforms, 3CX offers much more extensive functionality, the ability to connect your own SIP trunks and full user control over the system parameters.

Briefly about new features


  • Ability to install both local and virtual PBX
  • New client 3CXPhone for Android
  • New 3CXPhone for iPhone client with integrated 3CX tunnel (waiting for publication on the Appstore)
  • Significantly faster response to PUSH notifications from all mobile clients
  • Significantly reduced battery consumption by all mobile customers
  • Built-in server failover mechanism
  • Automatic backup and system restore on schedule
  • Automatic deletion of voice mail messages and conversations with a certain period of limitation
  • Voice mail and call recordings are saved in an economical format.
  • New reporting system with a scheduler and sending reports by e-mail
  • Office365 Integration
  • Many new pre-installed SIP providers
