Introducing the Nokia Asha Platform

Most recently, we announced the Nokia Asha Developer Bootcamp event . And now, in connection with the release of Nokia Asha 501, built on the basis of the Nokia Asha platform, we want to introduce you to the new, broad capabilities that it provides. The platform will be especially interesting for developers, as it will allow them to enter the promising market of inexpensive smartphones. So, we plan to sell about a hundred million new generation smartphones Asha in the next two years.




Many popular applications are already available or are under development for the Nokia Asha Platform: CNN, eBuddy, ESPN, Facebook, Foursquare, Line, LinkedIn, Nimbuzz, Pictelligent, The Weather Channel, Twitter, WeChat, WhatsApp, World of Red Bull, and games from Electronic Arts, Gameloft, Indiagames, Namco-Bandai and Reliance Games. More apps will be available in the coming months.

Now is the time to start writing applications for an updated platform - we have already prepared convenient tools for development, you also have ample opportunities for selling and advertising, including the new Nokia In-App Payment tool.

Key features of the platform:


1. Standardization
Increased attention is paid to the combination of software and hardware. This gives developers confidence that if they use any hardware component in their application, it will be available on all Asha smartphones.

2. Update
Smartphone users are counting on regular software updates that add new features and functions to their devices. The Nokia Asha platform is built with these expectations in mind and the updates for it are currently under development.

3. Ease of use
Attractive user interface optimized for 3-inch QVGA displays. It uses scrolling gestures to switch between applications and the home screen, as well as to navigate the home screen itself. Applications can use the entire screen, hiding the status lines and menus until they are triggered by a special gesture.

4. New advanced applications
Creating full-featured applications using Java is now easier. Advanced features such as 2D and 3D graphics, the use of accelerometers and location information, allow you to create high-quality applications, such as those expected by smartphone users.

5. Web applications
When using web applications, content is downloaded to the device, which allows up to 90% less data to be consumed. The user interface becomes interactive: in contrast to the browser-based proxy system, Nokia Asha web applications can update it locally on the smartphone.

The Nokia Asha software platform allows you to develop applications using two technologies: Java ME and web applications. Java is ideal for creating applications and games that provide rich functionality and high performance requirements, and web technologies can be used for applications focused on delivering online content in an attractive, traffic-saving way.

New Nokia Asha SDK and Nokia Asha Web Application Tools

The Nokia Asha Software Development Kit 1.0 supports the development, testing, packaging, and deployment of Java applications for the Nokia Asha platform.

The APIs in this SDK include:
- Application localization tools (JSR238).
- Nokia alerts.
- Select file.
- Image scaling.
- Network status.
- Phone settings.
- Contacts.

Learn more and download Asha SDK 1.0 here .

Asha web application tools include a web development environment (WDE) and an integrated development environment (IDE), which can be used to create and edit Nokia Asha web applications. The kit also includes an easy-to-use Web Inspector for debugging and inspecting items.

Downloading tools for creating web applications is available here .

Nokia In-App Payment

This new tool will simplify the sale of content within applications. It gives users the ability to make purchases easily and safely, as well as provides transparency of payments. Nokia In-App Payment will also be available for existing Asha and Series 40 devices (starting with the 6th platform version and above, with the exception of the Nokia C1-01 and C1-02 models). In contrast to the existing payment tool, Nokia In-App Payment is characterized by the possibility of one-click payment, payment without registration and backward compatibility with the Series 40 platform. The public beta version will be available in the coming weeks, but for now you can sign up for testing .

Nokia Store

Applications for the new Asha platform are very easy to publish using Nokia Store . Since its inception, it has provided a total of more than 10 billion downloads.
