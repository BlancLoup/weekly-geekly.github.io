Search service for Russian books for Kindle


After I got the Kindle 3, there was the problem of filling it with books in all of us understandable Russian. The book itself supports Cyrillic from the box, but the Russian layout is not provided for it. A great opportunity to use the browser built into the book partially suffers due to the lack of a Russian layout.

Of course, it is possible to use firmware hacks to localize the device, but this is not always convenient, since when updating the firmware these hacks will have to be rolled back, in addition, we may lose the warranty. It is also not convenient to connect the book via USB with a wire all the same, because after all, why is there WiFi and 3G in it? A way out of this situation can be a Russified on-screen keyboard.

For one evening, a suitable on-screen keyboard written in JavaScript was found and optimized for the Kindle screen. The domain name kin3lib.ru is registered for our project. The service transfers the search query typed on the virtual keyboard to the Flibusta library, where you can safely download the book into its native mobi format. In the future, you can implement the choice of libraries on which to search, but, unfortunately, I found only one that allows you to download books in the format of mobi. If anyone knows more such libraries write to me.

I will be pleased to hear the opinions of habravchan on such a service, as well as who and how solves this problem.
