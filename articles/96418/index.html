WineGame 0.1.0

I am pleased to announce the release of the first more or less stable version 0.1.0. your project Winegame.

Previous post about rc4 version

In brief Winegame is a FOSS utility for easy installation of win32 applications (mostly games) in wine. This is an analogue of Playonlinux , if it tells you something.


Compared with the previous one, many bugs were fixed in this version, such as:

- The interface does not freeze when starting the Wine process
- Fixed several bugs leading to the fall of the program.
- Fixes in the support of games on multiple disks (although everything is still unstable there)
- Cancel the download distribution Wine

Also, now all the necessary files (packages) are downloaded from the Internet, thanks to the adnull user for providing hosting.
