Universal virus

According to the Kaspersky Lab , a virus has appeared that can work both in Windows and Linux. It has a double name - Virus.Linux.Bi.a / Virus.Win32.Bi.a . The code is written in assembler and is able to infect binary files of the ELF and PE format. According to representatives of the laboratory, the virus is still harmless and is only a conceptual model, confirming the possibility of universal "contagion", but less harmless variations may appear soon. However, it should be noted that the ability of a virus to infect both Windows and Linux machines does not mean that the damage will be the same. Observance of the simplest security rules (do not work as root, do not run unknown executable files, keep an active anti-virus daemon in system processes, etc.) will allow Linux users to more easily transfer infection.
