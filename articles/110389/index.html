GWT 2.1.1 released

Released in October, the release of GWT 2.1 caused a wide resonance among the developers. Many comments and suggestions were received, some of which were implemented and included in GWT 2.1.1 .
Simultaneously updated Google Plugin for Eclipse and GWT Designer .
Download links:
  • Google Plugin for Eclipse: code.google.com/eclipse/docs/download.html
  • GWT Designer: code.google.com/webtoolkit/tools/download-gwtdesigner.html

Detailed information can be found in the official GWT 2.1.1 report is now available , and below are the main updates.


GWT SDK


The RequestFactory component added in GWT 2.1 attracted much community attention. Based on the wishes received, the following functionality was added:
  • Service layer (service layer), which includes support for non-static service objects
  • Support for a value object
  • Calling multiple methods for a single request

Google Plugin for Eclipse


  • Improved UiBinder error reporting in SpringSource Tool Suite (STS)
  • Optimized IDE behavior: removed unused Java builders, latest STS release fixes for AspectJ

GWT Designer

  • Improved support for UiBinder , including annotations:
    • @UiField (provided)
    • @UiFactory

  • WebKit rendering for 32-bit Windows (used for IE)
