The final release of the browser Vivaldi 1.2


Hello!

Today we present the next update of the stable branch of the Vivaldi browser - now at number 1.2. By tradition, minor versions contain mostly corrections of errors found, but we do not forget to add new functions, as well as expand the capabilities of existing ones. This time the list of the most noticeable changes is not very big, but it deserves your attention:

  • Customizable mouse gestures
  • Setting a custom start page when creating a tab
  • Closing multiple selected tabs
  • Key combinations for opening an editable address from the address bar

In addition, we have done a lot in terms of expanding the capabilities of settings for existing functions, but allocating them to a separate item in the list of new products is not entirely correct, so we‚Äôll briefly discuss them in the meantime, let's look at noticeable innovations in more detail.

Customizable mouse gestures

This is one of the most requested features from almost the very first day of the browser‚Äôs existence. We tried to make it as convenient and intuitive as possible, but just in case we prepared small video instructions:


In general, after watching this video, you are unlikely to have problems with the development of a new function. Try, experiment - you can always return to the default settings.

Custom Start Page

The request for this function appeared relatively recently, but very quickly gained popularity among users. In fact, it is quite convenient to open a certain page when creating a new tab, so we decided to take a break from fixing bugs and add this functionality to the browser:


We have not had time to add the possibility of specifying any internal browser page as a start page, but in the future such an opportunity will surely appear.

Closing multiple selected tabs

Working with tabs is probably the most important function when working in a browser. At least - the most popular. Therefore, expanding the possibilities of working with tabs, especially in terms of managing them, is one of the most important tasks for us. However, you yourself have already noticed that updates in this area occur quite quickly. This time we have added the function of group closing tabs. Select several tabs and close them through the context menu. Or close all the others:


We will work on the translation, but how do you like the idea itself? In our opinion - quite worthy of your attention. In addition, look at the context menu in the picture above: its content changes depending on the number of selected tabs and on whether any tabs are selected or not. In general, you already understood the direction of thought: we are trying to ensure that the context menu contains only relevant, currently available options, and all temporarily inaccessible must be hidden from the user. Conveniently.

Address Bar Shortcut Keys

This function will not be needed by everyone, but at the same time, those who find it useful will be able to appreciate it. What is the novelty? It works like this. For example, you have a certain page open, but you need to simultaneously open another from the same site. In order not to use the menu of the website, you can simply correct the address directly in the address bar, and then use the key combination Alt + Enter. As a result, the desired page will open in a new tab. If you use the Alt + Shift + Enter key combination, the page you are looking for will open in the background. For web developers, the feature can become quite popular.

Expanded Accessibility

The Internet today is used by a huge number of users, there are among them people with poor eyesight. We are sure that such problems should not deprive a person of access to information on the network, so in the new version of the Vivaldi browser, you can set an individual scale for each tab to view the pages without straining your eyes. We also increased the scaling limits: from a minimum of 20% to a maximum of 500%.


And much more...

This is only a small part of the changes from the last stable version of the browser. We also worked a little on the design of the bookmark button, changed the default document icons, expanded the number of localizations, and ... Yes! Now you can enable the option to open bookmarks in one click from both the sidebar and the bookmark manager. You can view the detailed list of changes on this page . By the way, with the growing number of users of the Vivaldi browser, extension developers are beginning to take this into account in their work - recently Vivaldi support was added to the 1Password extension, popular with MacOSX users, and Vivaldi is also supported by another popular service - Lastpass , where support is implemented through the extension.
