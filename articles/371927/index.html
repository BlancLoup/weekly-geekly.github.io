Microsoft no longer works with bitcoin



In 2014, Microsoft solemnly announced the start of work with bitcoin cryptocurrency. This ‚Äúcurrency‚Äù users were able to pay for purchases in the Windows 10 Store and Windows 10 Mobile Store. Now almost two years later, the corporation has stopped working with bitcoin.

Microsoft did not make a special announcement, but a page has already appeared in the Windows Store FAQ , which says that bitcoins are no longer accepted. If a cryptocurrency currency remains on the user's balance, you can still make a payment. But it is impossible to replenish the bitcoin balance.



At the same time , the page on how to replenish funds in a Microsoft user account using cryptocurrency has not been deleted yet . But, probably, the removal of this page is just a matter of time.

Why is Microsoft no longer working with bitcoin? There may be several answers. The first - the cryptocurrency did not become mainstream, which was the original calculation of the corporation. The second is technical problems of bitcoin, because of which even some transactions have ceased to take place, or they go through, but with a significant delay.

Most likely, the number of bitcoin transactions in the Windows 10 store was very small, and Microsoft considered it inappropriate to support cryptocurrency.
