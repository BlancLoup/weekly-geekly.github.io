‚ñ∫ About radiation certification

This phone is the first Chinese who fell into my hands, certified by the level of permissible radiation - SAR . Certification conducted by the German office Phoenix TestLab (which is strange, in China, and its many). Results are valid for the entire line of Zopo 9xx - 900, 900S, 950X, 950H. They use the same antenna, case and chipset. Here you can read the full report, but I will focus on the results.

As can be seen from the tables, the phone develops its maximum power during transmission in the GSM-900 standard - 0.593W / kg.
For GSM-1800, this figure is already lower - 0.328W / kg.
From the WiFi transmitter we get 0.363W / kg, and the work in WCDMA mode is 0.484W / kg.
The maximum figure, I remind you, is 2W / kg for Europe and 1.6W / kg for the USA.
Thus, all the fears that the radiation of Chinese phones is slowly roasting the brains of its owner turned out to be groundless.


‚ñ∫ Getting Root and firmware

The firmware process:
1) Download the toolkit. Driver , and flasher , scatter file and recovery , root
2) Root_Superuser_3.1.3_Busybox_1.20.2.zip is copied to the phone's memory card, Recovery + root.zip is unpacked into a folder on the computer.
3) Run the program (Flash_tool.exe) on behalf of the administrator, press the Scatter-loading button, select the txt file with the unpacked firmware, which stores information about the addresses for which the firmware should be uploaded. Click Download. The phone is not connected!

2) Turn off the phone, remove-insert the battery, hold down the volume +, insert the cable. The system swears at the new device, we slip (if necessary - by force) the driver from the first paragraph. As soon as the driver is installed, the program will find the phone, flood the preloader with it (via com port emulation, slowly a red stripe), and launch it. Having started, the bootloader will get comfortable in the phone, will raise a full-fledged high-speed USB interface, to which the system, as always, will ask for the driver. Perhaps they will deliver them automatically, then skip the next item.
2.1) If the meter in the program counted, but did not find a new interface. Go to the device manager, see an unidentified device, install the same driver for it from the first item. Repeat point 2.
3) If everything went well, the program will find the USB interface, then the flash driver will load (lilac stripe), which will already accept the firmware and stuff it into the phone‚Äôs memory (yellow stripe). After that, a window will pop up with a green circle - that's it. We distort the battery, press the power button. To download to the recovery - volume + and power button at the same time.
4) Boot into Recovery, select ‚Äúbackup and restore‚Äù, ‚ÄúBackup‚Äù, ‚ÄúYES‚Äù. After that, the image of the current firmware will be saved on the memory card, which can always be restored via the menu item ‚Äúrestore‚Äù. This is just in case something goes wrong.
