Vivaldi Beta 2 browser release


Hello!

Few days remain until the new year and we are in a hurry to make a good gift to all our users - the second beta version of the Vivaldi browser. Since the release of the first beta, exactly one and a half months have passed and we have not lost this time in vain. In addition to fixing a lot of errors and shortcomings that you reported to us (at the end of the article a complete list - more than 130 fixes and improvements), we did not cease to add new features. We have already talked about them in each announcement of the weekly test assembly, but let's briefly go through all the innovations - we will make a peculiar report on the work done.

So, since the release of the first beta, we have added the following functions to the browser:

Beta 2 (release date 12/17/2015)
Fast closing tabs
Basket in bookmarks and notes
Turning off the side panels
Tab Mute (STFU)
HTML5 notifications
Minimum font size
Default encoding settings
Move tabs between windows

Fast closing tabs

When you click the close button of a tab, the tabs are not resized simultaneously, so without moving the mouse cursor, you can close several tabs one by one.

Minimum font size and default encoding settings

The first stage of the implementation of settings for fonts is only in terms of minimum size, plus the installation of the default encoding. Everything is in one section of Settings Tools-> Settings-> Web Pages-> Fonts :



Move tabs between windows

In the context menu of the tab, a new item has been added, allowing you to open a tab either in a new window or in another existing one. The list of all open windows is also displayed in the context menu:



Well, perhaps that's all for today. It remains only to congratulate everyone on the upcoming holidays and wish a lot of new impressions and unexpected surprises in the new year - perhaps one of those positive surprises will be the final version of the Vivaldi browser as the main browser :-)
