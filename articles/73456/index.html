Windows 7 hotkeys

I bring to your attention a list of hotkeys, the use of which makes working with Windows 7 even more convenient (in brackets - an alternative way to do the same):

Window management


  • Win + Home : Minimize / maximize all windows except active. ("Shake" the window)
  • Win + Space : Make all windows transparent. (Hold the cursor over the small button in the lower right corner of the screen.)
  • Win + Up : Maximize the active window to full screen. (Drag the window up until it stops, or double-click on the upper frame)
  • Win + Down : Minimize the active window or restore the size if it is maximized.
  • Shift + Win + Up : Maximize window (Drag the bottom of the window to the taskbar)
  • Win + Left / Right : Take a window left / right half of the monitor, or restore the size (Drag the window left / right until it stops)


Taskbar


  • Win + number : Run the program under this number or switch to it
  • Shift + Win + number : Start a new copy of the program with this number (click the wheel on the icon)
  • Ctrl + Win + number : Switch to the next program window under this number (click while holding Ctrl )
  • Alt + Win + numeral : Open the jumplist of the program under this number
  • Win + T (+ Shift ): Switch between all open windows with thumbnails above the taskbar in the direct (reverse) order
  • Win + B : Focus on Tray
  • Ctrl + Shift + Click : Run the program as Administrator
  • Shift + Right-click : Show window / group menu

Explorer (and Desktop)


  • Ctrl + Shift + N : Create a new folder
  • Alt + Up : Climb up a folder
  • Alt + P : Enable File Browser
  • Shift + Right-Click (by file): Adds the "Copy Path" item to the context menu and many additional items in the "Submit" submenu
  • Shift + Right-Click (by folder): Adds an item to the context menu that allows you to open a command line from this folder.

Other


  • Win + P : Open the presentation mode settings menu (second monitor - projector)
  • Win + ( + / - ): enlarge / shrink the image (causes a magnifier)
  • Win + G : Switch between running gadgets
