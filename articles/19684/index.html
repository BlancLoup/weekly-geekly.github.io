New parameter for waki - offset relative to the cursor

Now a new parameter has appeared in the waki HTML code: offset relative to the mouse cursor:

waki_cursor_padding_left = -30; // offset of the waki window relative to the mouse cursor horizontally
waki_cursor_padding_top = -60; // offset of the waki window relative to the mouse cursor vertically


Those who did not have these parameters before - you need to register them (you can copy it from here as it is)
