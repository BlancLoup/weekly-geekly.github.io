Synergy - a platform for creating online stores

In essence, Synergy is a cumulative Open Source extension of Spree , focused on the creation of online stores in Russia.
The project's mission is to provide all the necessary functionality specific to Russian online stores, as well as to simplify the integration of recommended extensions.

In turn, Spree is a full-featured e-commerce platform running on the Ruby on Rails 3 framework. It is designed to simplify the development of online stores by incorporating all the basic functionality needed by developers.

After installation, Spree provides a ready-to-use store that can be used out of the box. But, most importantly, it is also a powerful tool for developers designed to create sophisticated web-based e-commerce applications. Notable features of Spree are the presence of a RESTful API for integration with external applications, support for connecting third-party search engines such as Solr and Sphinx, as well as the ability to work on cloud hosting.
Retaining all the benefits of the most recent versions of Spree, Synergy adds the following features:
  • creating static pages
  • WYSIWYG editors support
  • integration with Robokassa
  • integration with zopim.com (online consultant)
  • export to Yandex.Market
  • import from Yandex.Market (piece by piece)
  • Google.Sitemap dynamic generation
  • address book for registered users
  • contextual help in the administration panel
  • 404-error page replacement, built-in store design
  • and, of course, somewhat more complete support for the Russian language

The current version of Synergy (0.50.0) supports work on top of Spree 0.50.2 and Spree 0.60.x
