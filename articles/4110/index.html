Adobe Flex 2 - Ajax Companion and Alternative

At the same time as Flash Player 9, Adobe released the Flex 2 web development environment. The initial version of the Flex SDK is now distributed for free, making this platform a potential competitor or companion for Ajax.

With Flex 2, you can create web applications using Flash and / or Ajax. The Flex 2 environment is specifically designed to create websites with an interactive interface and the use of JavaScript scripts.

In fact, in the past, quite a few companies announced the release of handy tools for creating interactive websites, but as a result, a new ‚Äútext editor‚Äù with a built-in collection of scripts was born. In contrast, Flex 2 is a step in the right direction. The Flex Builder 2 IDE system is based on Eclipse, a very popular free development.

In addition, Flex analyzes syntax and supports autocomplete, has a built-in compiler and debugger that works in real time. All this is very convenient and saves time in the process of working on the code.

Adobe has created a gallery with samples of sites created on the Flex platform, and has also opened a section for developers . Even earlier, popular web services made using Flash, for example, the new version of Yahoo Maps , Google Finance , Zillow and many others, began to appear on the Internet.

The Flex platform is distributed under a new pricing scheme, but the main package for the Flex SDK developer has now become free. According to some experts , Flex has every chance of becoming a real hit, because it allows you to create truly wonderful, beautiful and functional web applications without having special knowledge of ActionScript and JavaScript.

According to Adobe, about 5,000 developers around the world are now using Flex, but the company expects them to grow to 1 million by the end of the year.
