  • [RFC] New custom object serialization mechanism - Offer accepted. In PHP 7.4, a new object serialization mechanism has been implemented and two magic methods have been added:

  • [RFC] JIT - JIT for PHP supports Windows and ZTS. The latter means that you can use in multi-threaded applications. The proposal has already started a vote, judging by the course of which, it is likely to be accepted for PHP 8.0, but the idea of ‚Äã‚Äãincluding in PHP 7.4 will not receive sufficient support.
    The topic also includes a post with benchmarks of the Redis server for PHP clue / php-redis-server , where the author received an increase of ~ 4-23% on the version with JIT.
  • [RFC] Arrow Functions 2.0 - Third attempt to implement short lambda in PHP. This time the syntax is suggested. 
     $y = 1; $fn1 = fn($x) => $x + $y;

    In this case, the variables from the external scopa are automatically assigned by value and do not use use ().
    Internals also discusses alternate syntax options: 
     ($x) ==> $x + $y; 

     \($x) => $x + $y;
  • [RFC] Generator comprehensions - The concept is framed in a complete proposal for the implementation of list inclusion (list comprehension). Compared with the concept, as in short lambdas, there is no need to use use() :
    Hidden text 
     $y = 3; $gen = [for $list as $x if $x % 2 yield $x * $y];
