Kotlin is now open source

Today, JetBrains has announced the availability of the source code of the Kotlin programming language , which is intended for development on Java and JavaScript platforms. In addition, access was provided to compiler builds, Kotlin standard libraries, build tools (integration with Ant, Gradle, Maven), and a plugin for Intellij Idea. Earlier, the Web demo version of Kotlin became available.
