Total Commander 7.5 public beta 1

Today, after a lengthy (and, I would say, podzatyanuschegosya) period of private testing begins open beta testing of the new version of Total Commander.
Let me remind you about the most notable innovations of the upcoming version:

Unicode support

The main disadvantage of TC is the lack of full support for files with Unicode characters on the way. Partially, the problem was fixed due to the Windows compatibility functions, but now Unicode is natively implemented:

Why was such an important change not implemented before? Because to rewrite the application that has been developing for the past 15 years is not a pound for you to eat. But now - cheers.
Unicode is also supported for the plug-in API, developers already have access to new documentation describing the updated functions.

FTP improvements

New features for organizing FTP connections, secure password encryption, bug fixes:

New navigation features

Now you can use the breadcrumb - navigation bar in the style of Windows Vista Explorer (containing a drop-down menu with a list of child directories).

Also improved are the individual directory trees that appeared in the previous version, and much more, for example, the toolbar:


Alas, version 7.5, apparently, will not include the previously declared functionality for burning discs. The reason is that a suitable version of the burning engine developed by a third-party company is still not ready. Perhaps the missing functionality will appear in one of the following minor TC updates.

Why, then, has everything been tested for so long, because initially they were waiting for beta in January-February, or even earlier?
This is the principle of development - to clean up all the bugs to the last. Until the testers found errors in the new functionality, the beta version had to be postponed. A total of 17 closed versions came out, thus appearing is the eighteenth.

Naturally, using beta versions is at your own risk, since even the most thorough testing does not exclude serious errors.
