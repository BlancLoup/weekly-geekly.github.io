Adobe Flash Player for MacOS X. It is finished!

It is surprising how information about this event has not yet appeared on Habr√©.

As you know, Flash Player for MacOS X had a problem with heating the processor, since all calculations were made not through a video card, but directly through a percent. And finally it happened.

For testing took the flash game TetrisFriends Arena (www.tetrisfriends.com):
Before the new version, the processor was heated to 80 degrees Celsius.
After installing the new version of Flash Player, the results are as follows:
Safari: 58-60 degrees;
Firefox: 58-62 degrees;
Opera: 58-62 degrees;
Google Chrome: all the same 80 degrees (Chrome has an integrated flash player, so these are the results).
Comment from LinZ : By the way, Chrome has also been updated and now there is the same version as the link.

When viewing HD video, the temperature also does not rise, and is within the normal range.

fpdownload.macromedia.com/get/flashplayer/current/install_flash_player_osx_ub.dmg
