Internet sample 1985-1990 (simulator)



Want to see what the internet was like (arpanet / usenet) before the advent of WWW and browsers? Welcome to the Telehack website, a simulator of the ancient Network, with support for 25,000 virtual hosts and BBS stations that were available at that time, with real files from those years. Of course, surfing the Web is done using telnet: using the telnet telehack.com command or via the web interface.

In addition to virtual hosts, there are also virtual users in the simulator, which could then often be found on the Web. The FINGER host command displays a list of users from each host.

The simulator allows you to even "crack" the ancient hosts, as it was a quarter of a century ago. For instructions, see the readme file .
