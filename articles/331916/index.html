At the stage of removal of the initial parameters of the site using the service, we usually fix ( Fig. 1 ):

  • Number of links (URL) - this characteristic is taken as a reference point and further allows you to evaluate the progress in building up the reference mass in the promotion process ( Fig. 1, position 3 );

  • The ratio of donor domains to the number of backlinks - there are no clear recommendations on the maximum allowable value of this parameter. Since it is not recommended to receive many references from one donor, this ratio should ideally strive for a unit ( Fig. 1, Pos. 4: Pos. 3 ).

  • The rating of the Ahrefs domain and the main pages of the site should grow in the process of promotion, besides, the rating of more significant pages for promotion should be higher than that of others ( Fig. 1, pos. 1-2 ).




Fig. 1 - Site parameters recorded during the audit (for example, interflora.co.uk)

This data already provides interesting information for the SEO specialist:

  • whether the site has links in general or will have to build up a profile from almost zero;

  • whether external links to the pages of the site are distributed in proportion to their importance for the client‚Äôs business goals and / or promotion (for this you need to first clarify and / or formulate these goals).

When selecting the semantic core, Ahrefs allows you to see which keywords the site already shows in the search, displaying statistics on the frequency of requests (replacing the Google Adwords keyword scheduler, which now shows not specific values, but the range of the number of requests), extradition, as well as a number of other useful parameters (Fig. 11).


Fig. 11 - Keywords for which the site is shown in the TOP-100 of the search results (for example, seicosecurity.com)

In addition to standard snippets in search results, other search results are returned for the site on this request - SERP features (video, contextual advertising, an image, an extended regular snippet, etc.). For example, in fig. 12 you can see that for a number of keywords the site is shown in the search for pictures, product ads, advanced snippets, etc.


Fig. 12 - Results of the site display in search results for the query ‚Äúmyers pump parts‚Äù (on the example of interflora.co.uk)

LIFEHACK!
If the site is already shown for a number of keywords in the search results, then at the initial stage it is better to choose for promotion those requests that are not yet in TOP-10 (TOP-3, TOP-5), but already in TOP-20 (TOP-30 TOP-50). If the data provided by Ahrefs for the checked resource is not enough, then you can check the semantic core of competitors, with higher positions in the search results, or similar sites. With high probability, from their semantic core, it will be possible to select relevant queries, in sufficient volume to start promotion, without using other tools.

When selecting search queries to a site, an effective tool is Keywords Explorer , which allows you to select query options. To do this, enter the basic requests (or download them in a file) that describe the site, set the promotion region and select from the proposed options (Fig. 13).
