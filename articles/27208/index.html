ImgLikeOpera Plugin in Firefox 3 RC

The man on the Firefox plug-in site found an easy way to get the popular ImgLikeOpera plugin to work in the third Firefox RC.

1. Save the plugin to yourself and open it as an archive.
2. Open the file install.rdf
3. Find the string <em: maxVersion> 2.0.0. * </ Em: maxVersion> and change the number. I have become <em: maxVersion> 3.5.0. * </ Em: maxVersion>
4. Find the updateURL function just above the line and delete it.
5. Close and save.
6. In Firefox, select File -> Open File and install the plugin.
