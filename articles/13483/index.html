Mail.Ru Agent 4.10 will save from spam

The new version of mail.ru agent , released on August 23, is enriched with new functionality designed to combat spam in the form of instant messaging.

Messages received from unauthorized users can be blocked in the new version while simultaneously stored in the Questionable folder so that later you can sort spam from messages of real users who for some reason did not pass the mandatory authorization procedure.
