Elementary OS Luna: when already?

Question: when will Elementary OS Luna be released? The answer is the classic ‚ÄúWhen it's ready‚Äù.


Elementary OS does not have a clear release cycle. Other systems, such as Ubuntu, decided to release the system at a specific time each year. But we decided that the release will be when we achieve all the goals and do not leave tangible bugs in the system.
Consider: luna is actively developing. When you finish reading this article, it may already become obsolete. You can see a depleting list of our tasks on the launchpad .

The foundation


Branding

Luna is based on Ubuntu. It makes our work easier, because a lot of it has already been done by Canonical and the Ubuntu community. But sometimes we accidentally inherit their brand.

Granite

Our framework needs to solve a few small issues - including for more efficient use of system resources and better support for internationalization.

Download and login



Before you get to the desktop, you must go through the process of downloading and logging into the system (KO - comment of the translator) . Need a good theme for Plymouth (system boot) and login (LightDM). The design for Plymouth is drawn, but you still need to write it. But for LightDM there are only a few minor fixes left.

Desktop




Design


Elementary GTK has changed a lot since Elementary OS Jupiter. Already better, but we have a few more ideas. We are planning to release another major release before the release of Luna.

The wallpaper has already been officially accepted .

WingPanel


WingPanel hasn't changed much, although there are still a few bugs related to multiple monitors and hotkeys.

Slingshot


Slingshot is an application for running other applications.

Plank


Our Plank Dock is designed based on a Docky with several modifications for Elementary. Now we have to test Drag & Drop and fix a few bugs.

Switchboard


Switchboard is an application for setting up the system and the most important part of it.

Applications


Files


Files is the Elementary OS file manager (approx. Translator - it‚Äôs also called marlin) that is developing well. Having posted it just 3 months ago, we received over a hundred error messages and several suggestions.

Terminal


The terminal has not changed much. There are several errors associated with shortcuts.

Scratch


Scratch is a text editor that has already acquired the status of a stable (recently there was a release). There are a few mistakes . The next version is likely to appear soon.

Maya


(comment of translator-ko. - Mayan calendar?)
Maya is also one of the new applications that first appeared in Luna. It needs to be more thoroughly tested, but we are still leaving it in this form, improving support for internationalization.

Noise


Noise is the most important part of Luna. In Jupiter, it did not seem to us that at least one of the players fit in well with the Elementary OS. Creating Noise, we solved this problem in the root. But while we are doing it, he has a lot of mistakes that need to be corrected before release.

Midori


Although the release of Midori‚Äôs development is independent of Elementary OS (like many other browsers, they release a new release every few months), there are a few things that we would like to see in Midori in Luna.

Public beta


After all these questions are closed, we can release the first beta release. This will be a much broader testing phase, and it will be closer to release. If the beta test goes well and all problems are resolved, we will release the release candidate and finally the release.
