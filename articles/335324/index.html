Vivaldi 1.11 - the desire for comfort


Hello!

Summer time, but the work goes on as usual - we present a new stable version of the browser Vivaldi 1.11. There are some changes in it due to summer holidays, but you will find something interesting under the cut.

Reading Mode Settings


Reading mode has become a popular feature among users who prefer to get rid of distracting attention and, at times, rather annoying elements of web pages when reading articles on the web. But just to delete everything around the article is not enough - it would be nice to be able to customize the display of text on the screen so that it is comfortable to read. For these purposes, we added the settings for the reading mode. First of all, these are text display settings:


But besides this, we took into account that many users have different preferences in the color design of the pages, so we added the ability to change themes for the reading mode:


All settings are available directly on the page when the read mode is on.

Turning off gifs


Another, sometimes very disturbing thing - animated pictures. Especially - advertising banners, which are often thickly littered with web pages. We have added an option that allows you to control the display of such images. If you wish, you can allow continuous animation playback, only once, or disable animation altogether.


Other amenities


Each of us, like the artists, has his own style of work: someone loves wide, sweeping movements with the mouse, while others prefer miniaturism and work with a minimum of movements. That is why we added the ability to adjust the sensitivity of the mouse when performing gestures. You can choose the length of the stroke, with which the browser will determine whether this movement is a gesture or the usual movement of the cursor.

Well, of course, you noticed a new application icon. We changed its design. We hope you enjoy it.


That's all for now. As always, the list of all changes and corrections contains a lot of interesting things. If you did not find the problem that was disturbing you in the list of corrected ones, do not hesitate to tell us about it through the appropriate form of sending error messages .

You can download a new version of the browser from the official download page .
