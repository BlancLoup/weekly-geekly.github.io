KDE4.1 RC1 released

The first release of a KDE4.1 candidate has been released - a version that should change the not very stable KDE4.0.x line

The final release is expected on the 29th of July, 6.5 months after the release of the KDE4.0.0 version.

New applications in KDE4.1:
  • Dragon Player - Multimedia Player
  • KDE-PIM module with some functionality from Akonadi
  • Skanlite Scanner Applications
  • Step - physical simulator
  • Games - Kdiamond, Kollision, Kubrick, KsirK, KBreakout
  • Many innovations in Plasma
