Qt SDK 1.1 release

Today we announce the release of Qt SDK 1.1. It was a long way to release, which many of you have been waiting for. If you do not go into details, here are the download links:
  • Microsoft Windows - Online Installer (15MB) - Complete Installation File (1.5GB)
  • Linux 32bit - Online installer (23MB) - Complete installation file (687MB)
  • Linux 64bit - Online installer (23MB) - Complete installation file (690MB)
  • Mac OS-X - Online Installer (12MB) - Complete Installation File (680MB)
You can also get it on the Nokia Forum pages . Also note that if you have previous versions of the Qt SDK installed, you will be notified of updates and will be able to download all changes using the maintenance tool.

Rather than talking about minor changes between the release candidate and the final version, let me dwell on what includes Qt SDK 1.1 and what it includes significant improvements over the Nokia Qt SDK 1.0.
  • This is the first version of the Qt SDK that allows you to create applications on Qt 4.7, including applications on Qt Quick, and publish them on the Ovi Store,.
  • Qt SDK 1.1 also integrates various earlier versions of the SDK. Thus, it is aimed at both the desktop and mobile developers. It will help you experience new platforms for your application and easily transfer it.
  • Qt Mobility 1.1 is an important step for us in the mobile application market. In combination with Qt Quick, it will allow you to create really powerful and rich applications.
  • Qt Creator 2.1 provides you with a development environment with all the tools and features you need to create applications.
  • Qt Simulator 1.1 has received many updates to support the latest versions of Qt and Qt Mobility, as well as simulated conditions for additional usage scenarios. Added additional device models.
  • With Symbian Complementary Package, you can install native APIs that will be used in your development environment for Symbian ^ 3.
  • Remote Compiler allows you to compile the project for all platforms supported by Nokia, as well as for Linux and Mac.
