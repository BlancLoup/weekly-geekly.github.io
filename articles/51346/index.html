Opportunities



Syntax highlighting

  • Intelligent Syntax Highlighter - automatically switch between PHP, HTML, CSS and JS highlighting directly during coding.
  • PHP 4 and up
  • SQL (MySQL, MSSQL 2000, MSSQL 7, Ingres, Interbase 6, Oracle, Sybase)
  • HTML / XHTML
  • CSS (versions 1 and 2.1)
  • Javascript
  • Vbscript
  • Java
  • C #
  • Perl
  • Python
  • Ruby
  • Smarty

Php

  • PHP 5 support (where without)
  • Code Explorer for PHP (includes, classes, interfaces, properties, functions, constants and variables)
  • PHP code hints as you type.
  • Work with any PHP frameworks
  • PHP server variables
  • Powerful PHP Code Beautifier with tons of options
  • phpDocumentor
  • Commenting and comment removal with one click
  • Go to the location of the variable, function, class, constant

Syntax check

  • PHP, HTML, and CSS syntax checking as you type
  • PHP syntax check using PHP interpreter

Editor

  • Highlighting (un) closed brackets / tags
  • Automatic indents
  • Auto adjustments
  • Unlimited undo / redo
  • Bookmarks (1-9)

  • Drag-n-drop support

Projects and Files

  • Transition to declaration in project files in filtering by classes, interfaces, functions, variables and constants
