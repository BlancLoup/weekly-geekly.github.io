New improved Evernote add-on for Firefox

Evernote in Firefox has become a more powerful, versatile and easy-to-use tool. Restart your browser to get the latest version of the add-on (4.0.0) or install it from the Mozilla website .

New options and settings


Now you have the opportunity to customize the Evernote add-on. To do this, open the corresponding add-ons menu in the browser and click the Settings button.

One of the basic parameters allows you to choose whether to use the Evernote desktop client (standard behavior of previous versions of the add-on) when copying web pages, or notes should always be added to the web client (i.e., use Firefox add-on in web clipper mode) . Using the web client is convenient for those who want to immediately add notes without editing (perhaps by adding a label or selecting a notepad) and continue working on the Internet. If you prefer to edit the created notes, then you would prefer to use the desktop version.

Please note that if, if you have an Evernote client installed on your computer, you prefer to save notes in a web client, notes will appear in the desktop client only after synchronization.

Built-in note search



When using an add-on in the web clipper mode, a search for all notes is now available in its pop-up window. To do this, click on the green tab at the bottom of the window. This can be useful to quickly find a specific note without opening the web interface or Evernote client on the computer.

As long as you enter a query in the search box, tags and notebooks with the appropriate names, as well as saved queries, if any in your account, will be automatically suggested as prompts. This allows you to access the necessary notes much easier and faster. Of course, you can ignore all these tips and just enter any query you like.
