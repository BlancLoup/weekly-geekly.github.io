Piwik 2.0 was released



Great news for webmasters and site owners. Yesterday, the release of web analytics Piwik released under version 2.0, a free alternative to Google Analytics.

What's new is waiting for us:

  • Rebranding and new website design Piwik.org
  • New cute default theme - Morpheus
  • Piwik Marketplace launched for plugins and themes
  • View and install plugins / themes directly from Piwik now
  • New, fully redesigned, mobile apps
  • Fixed over 171 bug
  • Piwik 2.0 is translated by volunteers into 53 languages

Demo can be found here .

Like Piwik itself, mobile apps are free and open. Available in more than 50 languages.
