Kirill Sheinkman will send Americans to the cloud

Cloud computing is one of the most fashionable technologies these days. Microsoft Azure and Amazon EC2 platforms contribute to the emergence of a whole layer of related businesses. One example is the American startup Elastra , founded two years ago by Cyril Sheinkman. Born in Leningrad, Kirill emigrated to the USA in the late 80s and received a serious technical and business education there. After four years of work as a sales consultant at Oracle, he went into the field of creating startups. Elastra is his third project and seems to be the most promising. In addition, this is another one of a number of technological startups created by people from Russia.

In the photo - Kirill Sheinkman (profiles on flickr , facebook , linkedin ).

The company has just released a second version of the Elastra Cloud Server, with which you can quickly configure and prepare any application for the cloud. After processing in Elastra Cloud Server, you can either give the application to Azure or EC2, or deploy it on your own corporate cloud.

As you know, ECB runs the JBoss Application Server, Apache Web Server, Apache Tomcat Java Servlet software package and MySQL database. So, Elastra adds Oracle DBMS and Oracle WebLogic application server to this package.

Elastra Cloud Server 2.0 is the first corporate version of the server. It integrates with many enterprise systems, including IBM Tivoli, HP OpenView, BMC Patrol, CA Spectrum, Zenoss and Splunk. It can work with virtual machines.

The basic version of Elastra Cloud Server is free , but it only works on EC2 on open source applications.

The commercial version of Elastra Cloud Server 2.0 costs from $ 30,000 to infinity, depending on the scale of the system. According to Sheinkman, the average order size is around $ 80,000.
