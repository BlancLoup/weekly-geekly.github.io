Up to 288 GB of RAM on a single motherboard - is it possible?

At the Shanghai IDF, Intel demonstrated systems with Nehalem processors. One of these systems is based on the Tylersburg chipset and can be equipped with two processors, each of which is provided with six memory slots of the DDR3-1333 standard. In combination with the MetaRAM technology, up to 192 GB of RAM can be placed on this board ...

About MetaRAM technology:
MetaRAM is a technology of the same name by an American company that offers a cheaper way to increase server RAM than a traditional one - using expensive memory modules (example: 16 GB memory module, if I am not mistaken, costs more than $ 2000 dollars) with high-density chips. The essence of this method lies in the use of microcircuits installed directly on the module, which are a kind of bridge between the memory controller and the memory module of the MetaSDRAM R-DIMM type. According to the employees, their technology allows to increase the amount of RAM in a workstation or server up to 4 times, while maintaining substantial funds. I also note that for arrays of hundreds of gigabytes of RAM, which this technology allows to create, the latency rate is at an acceptable level, for example, for DDR3-type MetaSDRAM, the CAS value is between 6-8 ns.

Returning to the Nehalem-based system, it is worth saying that, shown in the photo above, the stand can be equipped with 192 GB of RAM, provided that two processors are used.
