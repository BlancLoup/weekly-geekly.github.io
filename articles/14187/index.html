PocketPC: The latest version of the ICQ client mChat

Software developer mGSLab has announced the release of a fresh version of the mChat program. Recall that this is a free ICQ-client interface in several languages, including Russian. The program also allows you to communicate with users of networks using the Jabber and Mail.ru Agent protocols. The new version of mChat 2.3.0.0 announced the following changes:
- Support for ICQ server lists visibility, invisibility, ignoring.
- New x-statuses (and very lousy icons ala QIP), added detection of several ICQ clients.
- Search by history, uploading the entire contact history.
- Added new interface languages.
- Avatars in Jabber.
- Other bug fixes and minor improvements (such as new interface icons).
The program is compatible with handheld computers and communicators running the operating system Pocket PC 2002, Windows Mobile 2003 and above. To work, you need the installed library. NET Compact Framework 2.0.
