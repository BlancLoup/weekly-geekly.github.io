DNS will work faster

For the first time in the last three years, a new open source version of the BIND server has been released - this is the open and most common implementation of the DNS server, which provides the conversion of the DNS name to an IP address and vice versa.

The new version of BIND 9.4 works much more efficiently than the previous version 9.3, which was released in April 2004. The main reason for increasing productivity is the implementation of support for DLZ (dynamically loaded zones) and embedded cache. In addition, there is an additional check for memory leaks.
