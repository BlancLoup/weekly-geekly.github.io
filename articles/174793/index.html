Widget for displaying custom data on Android

Universal Widget allows you to place on your Android desktop information about any of your systems, described in JSON format. For example, statistics of visits to your site, information about orders in the online store, CPU usage on the server, etc.



Three simple steps to put information on the desktop:

1) Create a script - data provider in JSON format
2) Install Universal Widget
3) In the settings of the widget, specify the address of the script data provider, as well as the update interval

JSON example:

{
"title" : "Universal widget" ,
"type" : "list" ,
"date" : "03/18/13 5:31:41 PM" ,
"data" : [
{ "name" : "Row one" , "value" : "String" , "color" : "red" } ,
{ "name" : "Row two" , "value" : "1200" , "color" : "# FFA500" } ,
{ "name" : "Row three" , "value" : "50%" , "color" : "yellow" } ,
{ "name" : "Row 4" , "value" : "1`230.00 $" , "color" : "green" } ,
{ "name" : "Row 5" , "value" : "19403" } ,
... ]
}
