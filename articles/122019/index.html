Google Docs developers share future plans

Five employees from the Google Docs team conducted a question and answer session on Reddit and told them what they are working on at the moment. Summarizing the results of the interview, we can note several projects that will be completed shortly.

Significantly improved application for Android. The Android application was one of the most popular questions, and the developers assured that it expects "a lot and a lot of improvements." For example, editing documents without launching a browser.

Work offline when internet connections are lost. Initially, this feature was implemented through Google Gears, but now it is being redesigned for HTML5 standards, including using AppCache, File API and IndexDB. The final version will be presented this summer.

Printing through the cloud to a connected printer will be implemented on all platforms (today only under Chrome OS).

Support for embedded Youtube video and other embedded content is in the field of view of developers and may appear in the near future.

The best version of the Cloud Connect plugin for synchronizing Google Docs with MS Office programs, the plugin allows you to work with documents from the Google cloud in the familiar Word, Excel or PowerPoint interface (in the screenshot).



The preview of the .txt files that were removed last year will be back.
