The first photos, specifications and description - under the cut.

First, a few words about the positioning of the models, as defined by Sony Ericsson:
  • Xperia Neo is a multimedia smartphone that combines the best of Android with Sony entertainment technologies.
  • Xperia Pro is a business smartphone with intuitive and fast typing and messaging, and the ability to edit documents
  • Xperia PLAY is a powerful smartphone that provides PlayStation gaming features on the latest Android platform.
All three phones - on the latest Android 2.3 Gingerbread.

Xperia pro and Xperia neo


Sony Ericsson Xperia Pro and Xperia Neo - like twins, differing only in form factor and positioning - if neo is more an entertainment smartphone, then pro is the first big Sony Ericsson with a sliding keyboard since Xperia X2 (now the pro title in the Xperia line is still model Xperia mini, but it is smaller).
