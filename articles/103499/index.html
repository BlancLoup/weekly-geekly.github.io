10.62 RC

It's time to update the stable branch, and today a new build 10.62 has been released. The following changes are relative to release 10.61.

Since this is a release candidate, developers are asked to let them know if you see regressions from 10.61, and remind you that the missing fixes from 10.70 are not regressions.

It is also worth noting that the build numbers for different platforms are different here. Starting at 10.70, single build numbers will continue to be used. And since this is a 10.6x branch, there is still a build for PowerPC.

Download

  • Mac (Universal) / Mac (Intel-only)
  • Linux / FreeBSD
