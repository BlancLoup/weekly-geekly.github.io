Domestic processor "Elbrus-4C" will soon be launched into mass production

The Russian company MCST, whose specialization is the development of universal microprocessors, microcontrollers and control computing systems, announced the successful completion of tests of the Elbrus-4C processor (Elbrus-2S), which is ready for mass production.





The Elbrus-4C is a 64-bit universal microprocessor, which is by far the most high-performance microprocessor in the company's portfolio. It is manufactured using 65 nanometer technology, contains 4 cores (operating at a frequency of 800 MHz) and supports 3 DDR3-1600 memory channels. It is assumed that the novelty will be used in server hardware and systems where you need to ensure autonomy and secrecy.

The base operating system for the processor is OS Elbrus , built on the basis of the Linux kernel version 2.6.33.

For the first time "Elbrus-4C" was presented to the public in March 2014 at the exhibition "New Electronics - 2014".
