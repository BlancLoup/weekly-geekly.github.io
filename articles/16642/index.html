Color scheme and gamma

The article will be useful to all owners of pages on the Internet - this is both personal sites and blogs (diaries). Written to help cope with wild combinations of colors that accompany almost every second resource. The secret will not be that the correct selection of colors for the same site plays a key role in perception.

In the article:

In the first part:

  • Introductory information.
  • Table of base colors and RGB color values.

In the second part:

  • Overview of sites mixers on the selection of the right combinations.
  • An overview of a very useful program in this matter is ColorPic.

Part 2:

In the last post ( Part 1 (!!! not Habrahabr !!!) ) I wrote about the colors, the table of basic colors and so on. To understand this is not easy. And it is difficult to choose the right colors right away. To perform these actions, there is a whole series of website mixers, which I am going to tell you now. These services work on the simple principle: you choose one color that you want to see in your work, and the service selects the rest.

kuler.adobe.com

This service works like this. Choose and customize one color, others adjust accordingly. There is a section with sorted color sets rated by popularity rating. And also with new palettes. http://kuler.adobe.com/

colorvision.name

This service works the same algorithm. Choose and customize. It is possible to export to Photoshop and Illustrator. http://colorvision.name/

colourlovers.com

Another mixer. The same ratings, many options for ready-made variations of color. Pretty comfortable. http://www.colourlovers.com/

colr.org

According to another principle, colr.org works. He uses in the work of photography. Load the photo you want and see colors from it. There is support for downloading from Flickr photo hosting. At work, he is at first not very clear, but it is possible to figure it out and gain benefit from the times. http://www.colr.org/

colorblender.com

Another representative mixers. In principle, performs all the same functions. Able to save settings to a file for Photoshop and Illustrator. http://www.colorblender.com/

There are representatives of desktop mixers.

One of these representatives is the ColorSchemer Studio OSX program. Color can be set as a code, or simply select from the palette in the program. Created so-called schemes can be saved for later use in the same photoshop. http://colorschemer.com/osx_info.php

And lastly - ColorPic

A program that, for a long time, has helped a lot. ColorPic - Utility for taking the color of any point on the screen, saving it in the palette and / or placing the color code on the clipboard. ColorPic also includes a color mixer that allows you to analyze and change the selected color using a honeycomb, rainbow, tint, or adjustment using individual sliders. In addition, ColorPic allows you to choose the closest color that matches the web palette, and also use your own color palette. 10 tips for beginners. 1. Capture color To capture the color from the screen, you need to move the mouse pointer to the desired point and press "Ctrl" and "G" on the keyboard. The selected color will appear in the Chip (cell). To remove the color from the cell simply press "Ctrl" and "G" again. 2. Accurate capture color Use the arrows on the keyboard to push the mouse pointer one pixel. 3. Choosing Chips Use the mouse or use the function keys at the top of the keyboard to select the color of the chips in your palette. 4. Small screen problems Use the blue up and down toggle buttons on each of the five sections to save screen space. You can also resize the ColorPic window from the bottom right. 5. Increase Change the magnification factor of the slider above the magnifier area. 6. Smooth Color Swatches Use the dot size slider over the zoom area to select a single, 3x3 or 5x5 pixels dot size for a smoother color sample. 7. Websafe Colors If the selected color is not websafe, click "WebSnap" to change the color to the nearest safe value. 8. Specify the exact color. Using the keyboard, you can enter the exact value in any of the textboxes, including Hue, Saturation and Val textboxes. 9. Using color mixers When using any color mixers and sliders, you can use the arrow keys on the keyboard and the mouse to push the color in one step. 10. Use colors in another application.
