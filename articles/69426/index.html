Proposal for improving the login interface to Habr

Good day.

Today, only for the 10th time we managed to correctly enter the captcha and it turned out that the password was entered incorrectly, so I had to enter the captcha a couple more times :)
I think the difficulties with the introduction of a new captcha arise not only from me. So the proposal, since the captcha has become many times more difficult and more resistant to the selection of artificial intelligence, to offer to enter it to the user only once.
For example, if the user entered the image correctly, and in the Password or Login field was wrong, the captcha will no longer display.
