First start


The laptop has already been pre-installed Windows 8 Pro. Therefore, when you first turn on, you will be asked to set the primary settings: select a language, country, keyboard layout, computer name, wireless network, download updates, and so on. The user account in the OS is created on the basis of email, the password must consist of at least 8 characters: numbers and letters of different registers. After thinking for a while, you will get to the Windows 8 desktop. We will not describe in detail the operation of this operating system, since this is beyond the scope of this review.
