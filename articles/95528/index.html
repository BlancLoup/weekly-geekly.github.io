Do not rush to upgrade your HTC Hero to Android 2.1

Apparently, HTC still fulfilled the Legend sales plan and got generous with the release of the long-awaited update for Hero. The update is already available for the Chinese version of the hero - G3 (what we already managed to mention today in Habr√©), so the European one is just around the corner. This news is full of all developer forums of custom firmware, offering to test the official Android 2.1 firmware on your hero right now, if only English (Chinese or Chinese) is enough.

This update is important primarily because it again makes Hero a modern phone, which is almost as good as the more expensive Legend. So if the aluminum housing is not important for you, then you can buy Hero in excellent condition almost two times cheaper, which might interest, for example, developers looking for a workhorse to test Android applications and various experiments with the platform.

So we are waiting for the update for a few days, but do not rush to update your phone if you use custom firmware. At the developer forum of one of the most high-quality and stable firmware VillainROM , it was reported that it is very likely that the official update will install a new bootloader, which will make it impossible to install the custom root firmware on Hero later. Speaking more simply, the bug, due to which it is possible to install the root firmware on the phone, will most likely be fixed. Therefore, it is recommended to wait for the unofficial root-build (even on the basis of the Chinese version, several versions are already available to choose from) and install it.

There is another reason why you should not miss the opportunity to install custom firmware - most likely 2.1 will be the last update that Hero will receive. So take care of leaving yourself a loophole if you still would like to see 2.2 Froyo on your hero.
