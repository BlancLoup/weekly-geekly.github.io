Microsoft is preparing a major update for Windows RT

Long before the announcement of the release date of Windows 10, it became obvious to users that Microsoft is not going to develop a special edition of Windows 8 for the ARM architecture - Windows RT, as it does in the case of Windows 8 or Windows Phone 8, which will receive updates to Windows 10 Unlike the simple version of Windows 8, for RT there are significant security restrictions that are similar to iOS: only applications with a digital signature are allowed to run, applications can be installed only from the Windows Store, all applications run in the AppContainer sandbox.



Microsoft has already indicated that Windows RT 8.1 will not receive an upgrade to Windows 10. This mobile OS has been used on various tablets, which are getting smaller and there is no plan for any mass release of devices on it. At the same time, Microsoft is going to support this OS and release security and other updates for it. One of the major updates awaits Windows RT at the same time as Windows 10 is released. Its details have not yet been disclosed.
