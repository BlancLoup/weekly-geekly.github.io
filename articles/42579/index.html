Inquisitor no longer hangs Safari


If you use the Inquisitor search plugin for the Safari browser, you most likely faced a problem the other day, when searching through this plugin, the browser stopped loading web pages. A temporary solution to the problem was the reconfiguration of the Inquisitor, which boils down to replacing the main search engine from Google to Yahoo! However, today the developer of the plugin has released an update that fixes the error. It is personally unclear to me what this bug was connected with, but somehow I recommend downloading Inquisitor 3.2 v58 .
