Flashphone ..? Flaphone!

As we promised our users, flashphone.ru has been updated. We changed the name, changed the design and moved to the .com zone. Now we have become flaphone.com . This is not a complete list of innovations, but first a little history about the project.



Our project began over a year ago with the desire to create something new, both for the Internet community and for the VoIP community. There was a thought that it would be great if users would not need to install additional software and configure it to access IP-telephony services. It was Adobe's Flash technology that turned out to be the most suitable for this purpose, and we used it. This is how the flashphone.ru service appeared, which, during its existence, attracted 100,000 users (the figure is certainly not huge, but we are proud of it and want to increase the number of zeros at the end :)).

Flaphone is a web service that is a softphone (softphone), compatible with SIP (Session Initiation Protocol) and working directly in an Internet browser, without installing special software. All you need is access to the Internet, speakers and a microphone. Flaphone users are free to call each other, to SIP phones, as well as to any phone numbers by connecting an account of any SIP operator via the control panel.
