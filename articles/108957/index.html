Modified firmware ports from Nokia C6 to 5530/5800

I would like to continue my own business, so to speak, and continue the business of timo4ey . In addition to the usual ported firmware from C6 for 5530/5800 phones, I would like to tell about their modifications, which I used to do for regular firmwares. As you have already been told , the PNHT team ported, in their opinion, the firmware, which is better than the standard one installed on the 5530/5800 phones. On the pros / cons of the firmware, you can argue a lot, because they have a place to be.


In order not to deprive anyone, in the topic I will consider two versions of Firmware from C6, ported to 5530/5800 (hereinafter - the "port"). One - for 5530, the second, of course, for 5800.

I'll start with my beloved, with 5530 . I warn you right away: I choose the best firmware in my personal opinion.
For 5530, I chose Fusion mod version 1.3 by Roks, it seemed to me prettier, and the latter is new.

List of major changes


  • Phone languages: Russian, English (Russian seems to be the default)
  • Changed menu hierarchy, two menu options
  • Extra programs from autoload disabled
  • Custom themes effects, added screen rotation effect
  • Increased disk cache size, cache stored on E: /
  • The player updates music only from E: / Music
  • Gallery does not scan system folders
  • Improved photo quality (reduced compression)
  • Russian language added to the dictionary
  • Performance is configured.
  • Disable camera caching in RAM.
  • Removed protection from BT and MMS Sis transmissions.
  • Sewed: Correspondence, Autoinstaller, Btswith, Bright Light, Timer Pro.
  • Increased speaker volume during a conversation. raised to 10
  • Improved video quality.
  • Reduced application launch delay.
  • Improved kinetics
  • Now many applications will immediately go to the folders that should go
  • File manager displays real folder names
  • In the application reading sms added Russian voice Olga
  • "Slide. for unblock "on the unlock slider changed to" Unlock "
  • Unlock now both in landscape and portrait mode (without auto-rotate)
  • Fast turnaround (very fast)
  • Search works
  • After switching on, only widgets will remain on the desktop (link 1 and clock)
  • And other minor changes ...

Instructions on firmware


0. Install the necessary firmware for 5530 or 5800 .
Start JAF (via the Start JAF shortcut, then select ‚ÄúRandom PKEY Thomas SN‚Äù and press ‚ÄúSTART‚Äù).
1. Go to the BB5 tab.
2. Put a tick: Dead USB, Factory Set, make sure that there is a tick in Normal Mode and remove the tick from the CRT308.
3. Open through the explorer folder where the firmware files are located. We find the file RM-504_31.0.005_prd.core.C00 and, as if we want to move to another folder, drag it to the JAF panel, where the green circles are drawn. A menu will pop up, select ‚ÄúMCU file‚Äù.
4. Open the folder with the modified file (if it was in the archive - extract) and also drag it to the same panel. The menu will pop up, select the PPM file. (Usually, the file is called RM-504_31.0.005_prd.rofs2.V06, but there may be different posts in the name, especially ReBuilt (which means it is reassembled).
5. The MCU and PPM bulbs should come on. Now we connect the switched off Nokia 5530 phone to USB (directly to the motherboard, save all data in all possible ways before flashing) and press the FLASH button.
6. A confirmation question will pop up - agree, the firmware process will start and JAF will ask you to briefly press the phone's power button (1-2 seconds).
7. If everything is successful - the firmware process will continue, there may be different inscriptions and images on the phone screen, do not pay attention ;-)
Before flashing, delete folders from flash maps: sys, system, resource, private!
