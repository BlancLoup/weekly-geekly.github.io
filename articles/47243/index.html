Qt Creator - Beta Issue

On December 18, Qt Software released the Beta version of Qt Creator. Qt Creator is a new cross-platform development environment for building applications based on Qt libraries. IDE source codes are available under a GPL license through a public repository.

Beta release is a minor release for many who already use the first pre-release. It only helps to catch all existing errors, thanks to ordinary users who will act as testers. For more detailed information, videos using IDE and to download, visit this page here.

The final release is expected in early 2009 .

From myself: I felt a little Beta version and did not find any differences from the previous version. Nevertheless, I would like to encourage as many people as possible to test this development environment. Apparently, she will soon be able to become a serious candidate for use if you are a programmer using Qt + C ++.
