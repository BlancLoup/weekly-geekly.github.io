The Linux Piter conference program has been published.

The Linux Piter 2015 conference is dedicated to free software and is targeted at IT engineers using GNU / Linux and related technologies to develop their own ready-made solutions and products.

  • Swarm: Distributed storage for Ethereum, the Turing-complete blockchain / Daniel Nagy (ePoint Systems Ltd.)
  • DPDK - Intel RDB (Intel)
  • NTRDMA v0.1 - An Open Source Driver for PCIe NTB and DMA / Allen Hubbe (EMC)
  • Live migration of containers: pros, cons, pitfalls. / Pavel Emelyanov (OpenVZ)
  • Debugging the performance of MySQL / Sveta Smirnova (Percona)
  • Linux tuning to improve PostgreSQL performance / Ilya Kosmodemyansky (PostgreSQL-Consulting LLC)
