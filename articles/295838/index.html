Desmos - online graphing calculator



Have you ever needed to quickly draw a graph of a function, and you don‚Äôt have a favorite desktop program at hand? I was saved more than once by a free online graphing calculator from Desmos . Multilingual interface, incl. with the support of the Russian language.


Desmos is an online service that allows you to create graphs using a function formula. The function itself fits into the left column, and the graph is automatically plotted on the right side. The service will be useful to those who need to quickly and simply build a graph of a function, for whom the construction of graphs of functions causes difficulties or to those who need to check the correctness of plotting graphs with the least cost.



In addition to the fact that Desmos Calculator can perform all the functions of conventional graphing calculators, it also has several additional features that conventional graphing calculators do not have.

What can be done in DC:
  • draw functions;
  • create animated images by binding objects to functions with parameters;
  • create dynamic visibility;
  • quickly create screenshots with formulas and functions.


Desmos Calculator can build the following graphs:
  • Constant function
  • X versus y
  • Inequalities
  • Charts in the polar coordinate system
  • Piecewise functions
  • Point
  • Point group
  • Moving point
  • Functions with parameter
  • Complex functions


When plotting, you can use the following functions:
  • Power, exponential and logarithmic functions
  • Trigonometric functions
  • Inverse trigonometric functions
  • Hyperbolic functions
  • Statistical and probability functions
  • Other features




In the library of ready-made applications you can find a lot of dynamic applets for functions of different types (from linear to trigonometric) with built-in "sliders."

Authorization in the service allows you to save created applets and share them in the form of a link, embedded code or image.

Detailed information on the construction of graphs of functions can be found in the sources published in the basement of the article.

Quick schedule for publishing a feature graph created in Desmos:


  1. Go to https://www.desmos.com/calculator
  2. Register or log in to your account (upper right corner)
  3. Build a graph of the desired function
  4. Save the schedule
  5. Post a link to your schedule or save as a picture
