Last.fm hotkeys


Today we will improve many well-known Last.fm program.
For those who do not know, this program keeps track of what you are listening to and, in consequence, picks up the music that you should potentially like, a kind of personal radio station. Thanks to her, I added more than a dozen new unknown tracks to my collection. The program is distributed free of charge and is available at http://www.last.fm .
However, this program has a huge minus, which will soon be a hundred years old, but the developers stubbornly ignore all requests to correct the flaw. And this minus is the lack of global hotkeys . Those. if you are listening to music and playing a game in parallel, in order to switch to the next track you need to maximize the program window and click on the ‚ÄúSkip‚Äù button. It's pathetic!
So today we will correct this deficiency. For this we need:
  • Last.fm program itself
  • AutoHotkey program

After you have installed and configured your Last.fm account, as well as after installing AutoHotkey, open the Last.fm settings and change the language to English. This is due to the fact that the developers apparently did not even suspect that the hot keys in Russian might not work.
Now download this script and run it.
As a result, we have 5 global hotkeys:
  • alt + shift + PageDown - next track
  • alt + shift + End - mute
  • alt + shift + L - add a song to your favorite
  • alt + shift + Up - increase the volume
  • alt + shift + Down - decrease the volume

To ensure that hotkeys always work, we add this script to autoload.
If desired, the assigned keys can be changed to those that are convenient for you. The script programming language AutoHotKey is quite easy to learn + detailed help is attached to the program.
