New content management system. NET Forge CMS 6.0 with free edition



.NET Forge CMS is a new site management system for web developers who know and love .NET technologies. The new product includes a free edition for community sites and a commercial edition with an online store integrated with 1C. .NET Forge CMS is a professional platform for .NET-developers with well-developed ORM-tools, MVC-architecture, productive and proven on large projects.

‚ÄúWe welcome the emergence of the .NET Forge CMS product, which includes a convenient Hermitage interface, which has proven itself well in several thousand web studios. It is especially nice to see that the free edition of the popular commercial system is now available to the ASP.NET community. Given the growing interest in the ASP.NET platform over the past year, the .NET Forge CMS appeared just in time to satisfy this interest, ‚Äùcommented Gaidar Magdanurov, head of web technologies at Microsoft Russia.

.NET Forge CMS 6.0 includes two editions: Community and Business. Free Community edition allows you to quickly create modern community sites, blogs, personal sites and other web projects, saving on the platform. Business edition is a commercial system for creating online stores, high load sites, for which scaling, fault tolerance and security are critical.

The Hermitage interface is integrated into .NET Forge CMS 6.0, multisite technology is implemented, managed caching, a SKU system in the online store, and a new licensing policy is applied.

In .NET Forge CMS, the concept of the Hermitage interface is implemented, which has been used for a year in 1C-Bitrix: Site Management. The Hermitage has proven itself in the work, combining an improved management interface and a ‚Äúpackage‚Äù of recommendations for web developers, following which the developer creates fast, safe, convenient and easily manageable web projects.



In .NET Forge CMS 6.0, multisite technology was introduced to create multiple sites on a single copy with different domain names. All sites operate within the same application. Multisite also works on shared hosting.

Managed caching technology (Cache Dependencies), implemented in .NET Forge CMS 6.0, is used to automatically update data immediately after changing it. This technology allows you to display changes on the site, without waiting for the update cache, which is produced by the system at specified periods of time. Managed caching is one of the most important technological components of user experience with the site.

The product has built-in support for working with SKU (Stock Keeping Unit) in the online store. Now you can add different price offers to one product depending on the properties or characteristics (for example, shirt color, clothing size, car equipment).
