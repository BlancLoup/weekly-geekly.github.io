Livkoding immediately for multiple devices



Today we made a live demonstration of the HTML page on several devices and browsers at once.
The demonstration involved three laptops (safari, firefox, chrome), one TV (Samsung with Smart Hub), one iPad 3, and iPhone 4 and the Apple iPhone 5C page.

With the help of COLT , we edited the html-page immediately on all the above devices, without rebooting. It is worth noting that the reloading of such a page on the TV takes about 5 seconds, and in our test it happened almost instantly.



What you need:
1. All devices must be running on the same local network.
2. Run the COLT example and click the start session button - bottom right.
3. Verify that the session has started (the lightning is green) and enter the same address on the devices.
4. Open HTML in any text editor and start modifying the text. On all devices, changes will occur instantaneously and simultaneously. In addition to HTML changes, you can experiment with CSS and javascript.

Read about COLT at http://codeorchestra.com !
