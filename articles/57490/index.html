Ardentryst is a free platformer with role-playing game elements.

Recently released version 1.7 of the game Ardentryst . This is an arcade-scroller like Castlevania, in which a character not only runs, jumps, fights and collects various useful items, but also gains experience points, develops and becomes stronger level by level, is engaged in trade, communicates with other characters. And it all revolves on a two-dimensional engine with rich graphics and pleasant music in the background.



What's new in this version:
* Bug fixes.
* Small changes in the economy of the game.
* Data is now licensed by Creative Commons 3.0 (the engine itself is under GPL3 + ).

Before starting the game, make sure that the pygame package is installed on the system.
