Opera Mobile 10.1 for Android



Today, users of devices running Android received a long-awaited gift - a public release of the first beta version of Opera Mobile 10.1 browser. It's no secret that users had an ambivalent attitude to the Opera Mini used earlier on Android devices: if the small version still had enough of handheld devices, then on all sorts of tablets and other novelties of the unpredictable market of computer products a small Java client (and Opera Mini is exactly that) looked very unconvincing. But this is now in the past - now you can use Opera Mobile with all the resulting bonuses, such as:


- Smooth zooming with two fingers (pinch-to-zoom);
- Full browsing of pages with JavaScript (Presto kernel is used, the same as in Opera desktop browsers);
- Visual tab switching;
- Sync bookmarks using Opera Link;
- Ability to install Opera Mobile as the default browser;
- Support for positioning technology (Geolocation);
- Support for scalable vector graphics (SVG);
- Opera Turbo mode as desired / necessary;

And much more. In other words, now instead of Opera Mini Probe, you can install a full-fledged full-size browser. All Android versions from 1.6 and above are supported, you can download the browser from both the Android Market and the official site by going from your device to m.opera.com/next, or by scanning the image below.
