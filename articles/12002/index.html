Well, do you make your web 2.0 with Microsoft PopFly?

Today I will tell you about a new Microsoft project called PopFly. Already now you can take part in testing this product by requesting an invitation at www.popfly.ms - most likely in a few days you will receive an invitation to participate in testing this product.

Microsoft PopFly is a special system that will allow everyone to create interesting and interactive pages without having any web programming skills. You can create something new or combine data from various sources (mashups) on your page such as Flickr, Yahoo, Virtual Earth or your own blog.

So, you received an invitation and the first thing you have to do is fill out a short questionnaire, which only NickName is a required field, thanks to the Microsoft team.



After that, you will see the main PopFly screen, which allows you to create a web page, create a mashup (see above), or join a community.



One of the main advantages of this system is that the user does not need any additional knowledge, such as HTML, DHTML or Java, to create their own page. Choosing the option ‚ÄúCreate a Web Page‚Äù you will see an interactive and convenient web page editor:
