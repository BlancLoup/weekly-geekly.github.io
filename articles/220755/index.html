Unreal Engine 4.1 - support for XBox One, PlayStation 4, Linux, SteamOS and more



Epic Games announced the release of a new version of the engine - 4.1

All * UE4 subscribers have access to the XBox One and Playstation 4 platforms .
* all subscribers who are registered developers of XBox / Playstation.
At the moment, only source code with XBox One and Playstation 4 support is available, so the build will have to be compiled independently.
In the near future, it is planned to release a ready-made binary version and no code compilation is required.

Also added test support for Linux and SteamOS.

Thus, Unreal Engine version 4.1 supports the following platforms:

  • Windows;
  • Mac OS X;
  • Linux;
  • SteamOS;
  • Xbox One;
  • PlayStation 4;
  • iOS;
  • Android;
  • web (HTML5).


More information about the innovations of version 4.1 can be found on the official website .
