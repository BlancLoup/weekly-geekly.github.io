The first beta version of the Lightspark flash player is presented.

After a year of development , the first beta version of the free flash player Lightspark is presented . It is based on the use of a JIT compiler that translates ActionScript code into x86 instructions. Based on LLVM technology. The source code is written in C ++ and is distributed under the GPL v3 license.

Key Features

  • The use of OpenGL in the formation of output of geometric objects and video (for work requires a video card with shader support). It is possible to connect filters and effects that change video parameters, thanks to the use of textured output using OpenGL.
  • ActionScript 3.0 support, first introduced in Adobe Flash 9 (in Gnash, full support for ActionScript 3.0 has not yet been implemented).
  • Plugin for Mozilla.

There is a PPA repository.
