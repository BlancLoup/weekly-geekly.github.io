Roughly speaking, by answer, I mean return $ var of a certain method or function. Here it is necessary to make a reservation that the MVC design pattern is used , where the module is, of course, a model. Further, some common words from how MVC works in my CMS.

The task of the main controller is to call the module method (the modules are represented by classes) and return its response to the view. Thanks to this method, we get rid of the need to write code for processing templates in modules, thereby removing the binding of modules to a specific template engine and, moreover, we can use several template engines simultaneously.
