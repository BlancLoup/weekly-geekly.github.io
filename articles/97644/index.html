Synchronet: New service for synchronization and data exchange

Sinhro Systems has launched a new online service for synchronization and data exchange - www.synchronet.ru . The project presentation was held at Sviaz Expocomm 2010.

The service is designed to solve the problem of data synchronization for owners of mobile devices and computers. As you know, users are often faced with a situation in which the necessary data right now is stored on a device that is currently not at hand. Drivers, wires, etc. are needed to synchronize most mobile devices with computers. Things are much worse if data or files need to be transferred from one device to another, especially if these devices are not equipped with wireless communication, or are out of the scope of these tools.

SynchroNet can:
  • Synchronize files, contacts, events, other data - between the vast majority of mobile devices and computers.
  • Provide access (for reading or writing) to the stored data by e-mail, for users, not even registered in the system.
  • Provide public access to files stored in the system.


Virtually any modern device can be connected to the service. As part of the Synchronet project, Sinhro Systems developed client applications for the main mobile platforms, such as Android, Java 2 ME, which are available for free download and installation over the Internet. For the convenience of users, for each client application, you can get an individual link via SMS. Using this link, the user does not need to further configure the application after installation. The service also supports third-party client applications if they are based on the SyncML protocol. Most modern devices have pre-installed applications of this type.

It distinguishes us from other services of this type:
  • Using the SyncML protocol to implement synchronization, which makes it possible to use our service with a large number of mobile devices.
  • The ability to synchronize files.
  • Social functions.
  • Interface implemented in Flex.

At the moment the service is available in beta testing. We are waiting for your feedback.
