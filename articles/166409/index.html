#MongoUA - The first meeting of the MongoDB developer community in Ukraine (Kiev January 23)

On January 23, 2013 , the first in Ukraine MongoDB User Group will be created. We want more people to learn and love a more flexible, scalable, document-oriented MongoDB database.

Why is this necessary? We will create a warm and lamp environment for intimate technical conversations, each time there will be new faces both among speakers and participants, we will prepare really interesting presentations on MongoDB. In short, regular events that can and should be wasted time.

The first meeting was organized with the support of the internal movement for the free exchange of knowledge of Ciklum Practice Leaders, but this is only the beginning.

UPD: a live webcast of reports on the community site http://mongoua.tk/
Questions to speakers can be asked with the hash tag #MongoUA



What is it this time? The first meeting will set a good tone for the MongoDB User Group event style.
18:30 - 19:20 CRUD operations in MongoDB , Alexander Gavrilenko, Playtech
The report focuses on basic document operations in MongoDB. Great opportunity to get to know MongoDB.
19:20 - 20:00 Aggregation framework , Sergey Zvinsky, Ciklum
This framework is a declarative way to work with such data. No more missing SQL :)
20:00 - 20:30 MongoDB for Location Based Services , Vitaly Domnikov, Ciklum
MongoDB is simply created for applications working with geo-data. Do you want to find the nearest ATMs or a warm, lamp cafe ?! Then you will learn how to implement it.
20:30 - 21:00 Questions to the speakers and free communication. Beer, cookies.
Any questions about MongoDB? We will answer them with pleasure. Do you have an interesting experience with MongoDB? With pleasure we listen!

Want to be part of the MongoDB movement in Ukraine? Join us

Time: January 23, collection from 18:00, ending at 21.00
Location: SkyPoint 20th Floor, Ciklum Office, ul. Amosova 12
