Introducing 3CX V16 with a communication widget for the site

Last week we presented the release of 3CX v16 and the communication widget 3CX Live Chat & Talk, which can work with any web site, and not just on CMS Wordpress.

3CX v16 allows customers to quickly connect with your company by offering powerful and efficient call handling technologies - call center with call distribution by operator qualification, web service quality monitoring service (SLA) and improved call recording management.

In addition to the contact center, the new 3CX received improved performance, new security technologies, improved administration tools, chat, video conferencing, and integration with Office 365. This is the first communication system in the industry that successfully runs Raspberry Pi.

Communication widget 3CX Live Chat & Talk



A new communication widget 3CX Live Chat & Talk allows a site visitor to start a chat, make a call or video call to your sales department with one click. This is the easiest, ‚Äúdirect‚Äù and completely free way of communication, which, moreover, is conveniently ‚Äúconducted‚Äù by your employees. The uniqueness of the 3CX approach is that a chat can be transferred to a voice call at any time ‚Äî without a separate phone call, when the client can reach the perfect other person. The 3CX Live Chat & Talk widget is delivered as a WordPress plugin and as a set of scripts for any CMS. You can arrange the widget in the design of the site or in accordance with the expectations of your visitors. For example, you can set your own greeting message, the names of managers answering questions, allow or block video calls, etc.

Installing the widget is very simple and consists of 3 steps:

  1. Set the widget parameters in the 3CX control interface to configure the communication channel between the PBX and the site.
  2. Download the widget files and set the parameters and colors that match the design of your site.
  3. Copy the CSS into the HTML content of the site.

Updated integrated call center


3CX v16 comes with a thoroughly revised call center module, which is included in the Pro and Enterprise editions :

  • Distribution of incoming calls, depending on the qualifications of the operator.
  • Server REST API integration with most popular CRM systems, including 1C: Enterprise , Bitrix24 , amoCRM , call fixing in the client card.
  • REST API integration with MS SQL Server, MySQL, PostgreSQL databases .
  • Monitoring the quality of service (SLA) in a separate window of the Operator Panel.
  • Updated operator performance reports.
  • Improved call recording management:

Corporate chat server and WebRTC softphone




3CX v16 allows you to communicate effectively thanks to modern business communications:
  • Browser softphone on WebRTC technology allows you to call directly from Chrome and Firefox browsers. It can also manage your desktop IP phone or 3CX mobile softphone, expanding their functionality.
  • Full integration with Office 365 , including the synchronization of contacts and user statuses. The integration works on the server side of the PBX. All Office 365 subscriptions are supported.
  • Improved corporate chat - added transfer of images, files and emoji.
  • Support for dealers (dialers) of some CRM, in particular, Salesforce, allows you to call directly from the interface of the CRM system.

New 3CX WebMeeting Video Conferencing Features


The free WebMeeting web conferencing service in 3CX v16 also offers several new features that both users and administrators will appreciate:
  • Voice call from a regular phone to a web conference.
  • Improved video quality due to dynamic bandwidth change.
  • An international network of video conferencing servers on the infrastructure of Amazon and Google for high stability of the service.
  • Demonstration of the screen of your PC to the conference without installing additional plug-ins.
  • Support for popular and affordable Logitech Conference Room video conferencing suites.

Recall that the main advantage of 3CX WebMeeting is that video conferencing and webinars are completely free and without studying a third-party online conference system.

New features for PBX administrators




3CX v16 offers a number of interesting new products for integrators and system administrators. All of them are aimed at reducing the "headache", i.e. decrease in labor costs for diagnostics and routine maintenance.

  • 3CX Instance Manager (Multi Instance Manager) - manage and monitor multiple 3CX PBX servers from a single administration portal.
  • Synchronize 3CX extension numbers and Office 365 users - manage PBX users from a single point.
  • Advanced Security:
  • Installing 3CX on a Raspberry Pi for systems with up to 8 simultaneous calls.
  • Significant reduction in processor and memory requirements for installation on low-cost VPS servers.
  • RTCP protocol statistics for monitoring VoIP quality.
  • Copy user - create a new 3CX extension based on an existing one.

Test license 3CX Pro, free license for 8 S and price reduction by 40%


The new 3CX v16 price list lowered the cost of the Standard edition by 40% and to 20% on the PRO and Enterprise editions! Annual mid-sized licenses have also been added that will allow you to start with the acquisition of a small system and smoothly expand it as your business grows.

The free edition of 3CX has been expanded to 8 simultaneous calls and will remain free forever! Please note that when downloading a new system from the 3CX website, you will receive a fully functional version of 3CX Pro, which will last 40 days. Then it will switch to 3CX Standard and remain free.

  • Revision 3CX Standard for 8 simultaneous calls - free forever
  • Additional license sizes: 24, 48, 96 and 192 S
  • Expansion of the license - payment only for the difference in cost, without additional conditions
  • 3CX Standard has excluded Call Queues, call recording, call reports, inter-office trunks, and CRM / Office 365 integration.

V16 development plans


In the video presentation, the head of 3CX told about the nearest development plans of the system. Some functions will appear in v16 Update 1 within a month, some closer to summer. Please note that plans may change, so just take note of them.

  • Enhanced integration with SQL databases.
  • Improved corporate chat - archiving, translating, intercepting messages
  • New programming environment PBX Call Flow Designer .
  • New 3CX SBC, allowing you to manage remote phones from the 3CX interface.
  • Improved work with DNS for compatibility with SIP-operators.
  • Simplified configuration of failover cluster PBX.
  • REST API for making outgoing calls via 3CX.
  • Interactive, customizable call center panel.

  • In the cloud - get a license and use the PBX Express service to run a free PBX in your Google cloud account
  • On the local server: download for Windows, Linux or Raspberry Pi
