What is new for PHPStorm / Webstorm 3.x?


A list of TODO developers JetBrains regarding the development of the third branch of WebStorm / PHPStorm:



ATTENTION! Roadmap is constantly updated, the current version is here . Translation is already outdated.

PhpStorm & Web Storm 3.0 (Q4, 2011)


  • PHP (PhpStorm only!)
    • UML diagrams support (when updating the code, the diagram will be updated and vice versa)
    • Improved PHP code formatter
    • Support profiling with xDebug and zend
    • Improved PHPDoc Metadata Support: Call Chains autoload
    • Zend Framework MVC support: views generators
    • Improved Twig Support

  • FTP
    • Fine tune resources to exclude from synchronization

  • XSLT debugger
  • Support node.js (including debugging)
  • Javascript
    • JsTestDriver support (for QUnit and Jasmine testing frameworks)
    • CoffeeScript support

  • Other
    • Improved UI design: More compact placement of elements in all dialogs, improved drawing of window styles in modern OS, Full screen support in OS X Lion.


PhpStorm & Web Storm 3.x (Q ?, 2012)


  • PHP (PhpStorm only!)
    • PHP 5.4 support
    • PHAR archives support
    • Symfony Framework MVC support : views generators
    • Navigation in dependent files for the Zend Framework and the Symfony Framework

  • CSS
    • LESS compilation support
    • SASS 3 Compile Support
