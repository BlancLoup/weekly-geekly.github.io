Opera Mini 5 beta 2 - now also for Windows Mobile



At the same time, the opera developers were preparing to release a new version of Opera 10.50 for desktops, the mobile development department also did not sit back and today, without a special pump, rolled out the next version of Opera Mini 5 beta 2, this time for happy owners of smartphones running Windows Mobile 5.x and 6.x versions. It would seem - why is it necessary? After all, for such a platform, a more functional version of Opera Mobile is intended, and the popularity of this forever outdated platform from the Redmond giant leaves much to be desired. In fact, there are several reasons why we did it.

First of all, however trite it may sound, the users themselves asked us to do this. Well, they are not satisfied with their regular browser, what can you do. However, if you wish, you can also install the version of Opera Mobile 10 beta 3 , which will be able to do everything the same as the built-in IE, only much better (plus the Turbo accelerator, of course). But for a number of reasons, many prefer the lightweight Opera Mini, even if it even somewhat limits the work with the loaded JavaScript pages. Well, just asking - why not do it. Moreover, we have never refused to expand the number of supported devices, and this is the second.

Yes, we live in the real world, where motorists are measured by "horses" under the hood, and software manufacturers - by market shares. Therefore, even with all the successes of Opera Mini in the market of small and medium-sized mobile telephony, we are not opposed to try your hand on devices running the newfangled Android, stake out a place on the iPhone or send an old man IE on a deserved rest to Windows mobile platforms.

It is worth noting that Opera Mini for Windows mobile platforms is now fully integrated into the new system environment and does not require complex gestures to install additional software - the browser code has been rewritten and this version is no longer the same Java application as the version for regular mobile phones. At the same time, everything is preserved both functional, including traffic compression, and external attributes of the familiar Opera Mini 5 interface.

In general, what is there to argue long - install and use, saving time and money.

Download Opera Mini 5 beta 2 for Windows Mobile browser in two ways:
On the computer , with the subsequent transfer and installation on the smartphone;
Go to your device .

And do not forget to send messages about errors found - after all this is beta.
