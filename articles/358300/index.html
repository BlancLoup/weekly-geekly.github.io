Event: AI / Machine Learning OpenHack
Date: June 4-6, 2018
Location: Moscow, Microsoft Office

What is OpenHack?


OpenHack is an event where developers are invited to solve a series of tasks using a set of specific technologies. Participants work in small teams, solving challenging tasks, getting to know and communicating with colleagues and industry experts.

What is OpenHack machine learning?


This OpenHack is focused on solving computer vision problems. Tasks include working with data, cognitive services, Azure Machine Learning Workbench, and other machine learning solutions using standard frameworks (TensorFlow, CNTK). We are planning a series of speeches by technical experts, as well as consulting customers and partners on current issues and issues.

Where and when will OpenHack take place on machine learning?


The event will be held at the Microsoft office at: Krylatsky Hills Business Park, ul. Krylatskaya, 17 p.1, in Moscow from 4 to 6 June from 10:00 to 18:00. The main language of OpenHack is English, some experts are Russian-speaking.
