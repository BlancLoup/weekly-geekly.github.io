Firefox add-ons that lead to memory leaks.

Adaptation of an article with Cybernet
One of the main reasons Firefox has reached 400 million downloads is its customizability, achieved through add-ons.
Of course, add-ons give the browser unique and useful features, but at the same time can cause unwanted problems.

There is already an add-on that successfully helps users and add-ons to developers of memory leaks in add-ons. Unfortunately, not all problems in some add-ons are solved, which leads to failures, memory leaks, and increased CPU load.

To find the problematic add-ons, the author of the article turned to the Mozilla knowledge base and their bug tracker .
Below is a small list of problem additions with possible fixes.
Note: it is not necessary that you have the problems described, so if you do not have problems with any addition, it makes no sense to delete it.

Adblock
Problem: browser crash while trying to use Google Chat.
Solution: none. It is recommended to use AdBlock Plus instead.

All-in-One Gestures
Problem: can crash the browser, configures with the built-in spell checker for Linux and OS X.
Solution: absent at the moment, it is recommended to remove the add-on if you are using linux or OS X.

Fasterfox
Problem: may affect access to sites, and problems may continue even after the removal of the add-on.
Solution: roll back to the default settings or create a new Firefox profile.

Firebug
Problem: Causes CPU overload.
Solution: disable the add-on when it is not used, or create a separate profile to use this add-on.

ForecastFox
Problem: memory leaks when using satellite images.
Solution: remove add-ons, or use an older version without satellite images.

Google Browser Sync
Problem: excessive CPU load when working with GIF-animation.
Solution: not available at the moment, the removal of the add-on is recommended.

Google Web Accelerator
Problem: browser crash on shutdown.
Solution: not available at the moment, the removal of the add-on is recommended.

IE Tab
Problem: memory leaks.
Solution: remove add-on, use IE View Lite as an alternative.

Loki
Problem: memory leaks.
Solution: not available at the moment, the removal of the add-on is recommended.

Media Player Connectivity
Problem: Causes Unresponsive Script Warnings .
Solution: not available at the moment, the removal of the add-on is recommended.

Mouseless Browsing
Problem: CPU overload.
Solution: not available at the moment, the removal of the add-on is recommended.

SessionSaver
Problem: memory leaks.
Solution: use instead of Session Manager .

Tabbrowser Extensions (TBE)
Problem: reduces the performance of Firefox, conflicts with other add-ons, can lead to a complete inoperability of the browser.
Solution: remove add-ons, use Tab Tab Plus instead.

Firefox 3 users should be careful when using ForecastFox and FoxClocks add-ons, both add-ons lead to memory leaks.

I also advise you to refer to additional Mozilla resources - tips on reducing memory usage ( Reducing Firefox Memory Usage ), as well as a list of Problem Extensions .
