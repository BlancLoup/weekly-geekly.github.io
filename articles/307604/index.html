CodeRush for Roslyn 16.1.6 - Official Release

Today released the first release of CodeRush for Roslyn (hereinafter CRR).


Now you can buy a license CRR. There are 2 purchase options:

  • for $ 49 includes free updates throughout the year, technical support through forums .
  • for $ 249 includes everything that is available in the $ 49 version + CodeRush Classic with support for older versions of Visual Studio and full technical support.
After a year, you can either renew the license or continue working without upgrades.

As with the preview versions, we will continue to work on the 45-day release cycle. Those. for the year CRR 8 times will receive updates with new functionality.

CRR works with both Visual Studio 2015 and Visual Studio Next - Visual Studio "15". Using the Roslyn API guarantees support for all new language features that Visual Studio understands.

You can download it in Visual Studio Gallery . Free trial period of 30 days.
