New version of Mono 2.8 released

A new version of the free implementation of the .NET Mono 2.8 platform has been released . This version is an important update that introduces significant improvements:
  • C # 4.0 support;
  • New Generative Garbage Collector, a high performance garbage collector (below);
  • new components:
    • ASP.NET 4.0;
    • Parallel Framework and PLINQ;
    • System.XAML;
    • System.Dynamic;
    • MEF;
    • MVC 2;
    • System.Data.Services.Client (OData);
    • WCF Routing;
    • CodeContracts;
    • System.Numerics
    • LINQ to SQL;
    • DLR is included in the package.
  • the second version of the embedding API;
  • OpenBSD support.
