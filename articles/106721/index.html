For developers on PS3, a normal EA web browser is available.

As you know, the Web browser built into the Sony Playstation 3 is hopelessly outdated and does not meet the demands of the current market.

So far, Sony has not done anything to change this situation, but now the developers of applications for the Sony Playstation 3 have a choice.

EA has released a new product within the Skate 3 project - EA WebKit , which is built on Win32, Win64, Playstation 3 and Xbox 360 platforms.
