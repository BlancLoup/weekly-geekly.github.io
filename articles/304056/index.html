Testing 3CX Phone System V15 BETA

Today we are pleased to present 3CX Phone System v15 BETA. At the same time we inform you that the final version is planned for July 6!

In this release, a new Windows client with an improved interface. He received a modern design, characteristic of the best applications of the United Communications. The main focus is on the interface indicating the presence of users. Only one color scheme of the application is included in the BETA version, but in the release several more interesting styles will be added.



Updated web conferencing


In V15, video conferencing is included in the core of the system. The conference interface has become more intuitive and implemented on mobile devices! We have released a special Android application that allows you to participate in video conferencing right from your smartphone. Now we can talk about the complete mobility of all 3CX services!

Android client works on Android 4.4 and higher, and allows video conferences of up to 4 participants.

Improvements and fixes


  • Added call history converter from version 12.5 to version 15
  • All new updates are now displayed in the system interface and downloaded via HTTPS
  • Improved 3CX Tunnel security, improved socket management, and improved audio stream quality.

New features of 3CX Phone System v15, not included BETA version


  • Easy configuration of FXS gateways
  • Easy configuration of base stations and DECT terminals
  • System permissions at user level (extension)
  • Group editing of extension numbers
  • Ability to run VAD (Voice Application Designer) scripts
  • New version of 3CX SBC (Session Border Controller)
  • 3CX WebMeeting app for iOS

Clients 3CX for Mac and Windows will be automatically updated to work with 3CX v15. 3CX clients for Android and iOS v14 will continue to work with 3CX v15.
