Ubuntu 10.04 alpha 3

The third and last alpha ubuntu 10.04 was released, the first beta is due on March 18.

Brief description of the changes:

- Music Store support has been added.
- open-source Nouveau driver for users with a video card from nvidia is included in the kernel and is available by default
- load / shutdown time is further reduced.
- Added a new featured feature category to the Ubuntu Application Center.
- improved notification system
- the default application is available to work with social networks Gwibber
- fixed all recent bugs, added security updates
