DOOM Classic and Wolfenstein 3D in source



id Software is a company most likely known to all visitors of Habrahabr, because they are behind such monsters as: Dangerous Dave, Commander Keen, Wolfenstein, Doom, Heretic, HeXen, etc. Speaking at QuakeCon some time ago, John Carmack promised to release the Doom 3 source codes, but so far this has not happened.

But today ZeniMax is a company. which now owns id Software (as well as Bethesda), released updates for Doom Classic and Wolfenstein 3D for iPhone, adding Retina Display support and versatility (now they can be played on both iPhone and iPad), as well as the same as is customary in updates, correcting a number of errors.

Along with this update, the company opens access to the current source codes of these games, so now you can make something of your own, based on their developments, or learn from the best game developers of the best how to make such games.
