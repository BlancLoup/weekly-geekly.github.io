Today I propose to overcome one very popular puzzle open source method. The task is trivial and has had many solutions. But for me personally, the question is still open. Namely: the web-based file manager ...



eXtplorer - using Ajax (ExtJS, ku-ku!) and PHP, our test is doing fine ...

eXtplorer is a web file manager. You can use it for:

* View directories and files on the server
* Editing, copying, moving, deleting files,
* Search, upload and download files,
* Create and extract archives,
* Create new files and directories
* Changes to many other files ...

You can even use eXtplorer to login to an FTP server (for example, net2ftp) and work as you would through an FTP client.

What are its advantages over other solutions?

Habitual drag & drop, a familiar tree explorer of folders, working with archives and editing files with code highlighting!

I was very happy when I stumbled upon it. Open source, easy to use and above all functional. In general, one of the candidates for leaders on this issue! Yuse!
