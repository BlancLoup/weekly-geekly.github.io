The system of social recommendations for recreation


And so VeniVidi launched a recommendation system. The algorithm is based on ratings that users submit. Now you can rate any country, city, region or hotel by 17 parameters. For assessments do not need to register and I tried to make them as unobtrusive as possible.

In the same form, where you put the estimates, the opinion of other users is displayed. You can see the ratings of others and set your own for example here: Thailand

The search interface is here: PLACES . In essence, these are tezhe runners, but now you set your preferences. For example, you want to find the most democratic beaches. Expose the sliders "Beaches" and "Cheapness" to the maximum, and are looking for. In accordance with the estimates of these users for these parameters you will select a country or city.

I think the approach is clear. For those who will search - do not wait now for very relevant results, the rating base should be accumulated. In most places, it is still zero, and it should take some time.
