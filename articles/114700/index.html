Game Boy games on an oscilloscope



Remember that post about porting Doom to an engineering calculator? So, a certain fan of games on the Game Boy was able to adapt this portable console to play on an oscilloscope. Of course, this is not porting the game, but all the same - the system seems quite complicated, but the author did it all. On a video of just one minute, we can evaluate the work of the boom, who was able to create a single gaming system from the oscilloscope and the Game Boy.

In the original source, the link to which is indicated below, there is a detailed wiring diagram, so those who wish to repeat this feat can already begin work. I wonder how soon Kinect will be attached to the oscilloscope, which has become extremely popular among geeks? I note that it is quite difficult to play on such a system, but the author does not seem to want to go through all the levels of Zelda on an oscilloscope, all of which is demonstrated as an example of a combination of what seems completely incompatible.

By the way, some visible violations in the display of the game (for example, periodic blackout) - this is the result of shooting the gameplay on the camera, according to the author, the image is fairly clean, so you can even play. Of course, the pleasure here is not so much from the game itself, but from the fact of successfully connecting the Game Boy to the oscilloscope. The author named his development NintendOscope.

At the very end of the source, there is a link to the source code of the software used to make it all work normally.
