Conference "Windows Azure Cloud Platform for Mobile Applications"



Developing mobile apps for Windows 8? Windows Phone? ios? Android? Then you may be interested in the Windows Azure Mobile Applications conference, which will be held on September 18, 2012 at the Digital October center in Moscow. Independent experts and Microsoft experts will talk about the Windows Azure cloud platform, using which you can get a ready, pre-configured, reliable and scalable backend for your application.

At the event, you will learn how to use Windows Azure for solving problems of storing relational and non-relational data, authenticating users using different providers (Live ID, Facebook, Google, etc.) and sending push notifications for Windows Push Notification Service (WNS) , Microsoft Push Notification Service (MPNS), Apple Push Notification Service (APNS), Cloud To Device Messaging (C2DM). And of course, on all technical reports there will be demonstrations for each of the devices.

Participation in the event is free, registration is required .
