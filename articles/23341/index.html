Windows 7 concept

We can‚Äôt touch Windows 7 with our hands, but we already know what the new OS will be based on. The so-called ‚Äúpillars of windows 7‚Äù - the concept - are described by the aeroxp.org browser. Below I will give a squeeze of five notes with AeroXP, which outlined the main ideas, the embodiment of which we will see in Windows 7.

I have expressed my opinion in italics.

  1. Data security
    Data must be protected and once again protected. And so on two times. AeroXP does not report anything new, but it assumes that the new security scheme will pay special attention to external media.
    Responsiveness.
    In Vista, frequently used applications are preloaded by the system into memory in order to reduce the time of the first program launch. Windows 7 extends this idea and transfers it not only to media applications, Internet surfing, but also to the whole system.
    I dare to assume that access to any frequently used function or file will be faster.
    Improved convenience when working with tablet PCs.
    Apparently, MS intends to gradually move away from the well-established tandem mouse + keyboard. M + K scheme will be expanded to use gestures and touches. In this light, Microsoft Surface is gaining more weight.
    Wireless connection.
    Simplified network deployment. Increased reliability. More convenient connection - both manually and automatically.
    Network profiles.
    It is supposed to minimize user actions when moving from network to network. Ideally, the system itself will determine whether you are at home or at work, and then download the appropriate hardware profile. The development of existing technology.
    Power Management.
    Windows will be optimized to use minimal mobile hardware as efficiently as possible.
    It sounds good, but does not cause optimism.


    1. Updates.
      Windows Update in Vista already downloads updates not only for the system, but also the necessary drivers. In Windows 7, they promise to do everything that has long been implemented in Linux - centralized updating of all applications, systems and drivers in one place.
      In my opinion, a very useful innovation. I want to believe that we will see him.
      Easy upgrade.
      If earlier the user did not always understand which of the PC components he needs to replace and why, then in the new OS Microsoft is going to make the upgrade a more transparent and simple process.
      Windows Online.
      Online Knowledge Base, download resources and even assistance in choosing the right machine for a specific user. In general, the system will differ little from the existing one.
      Community.
      The user will be able to upload information about his system for viewing by the community. Windows Experts will help resolve current problems and answer questions.
      For family.
      Parental controls introduced in Vista will be extended. The possibility of introducing blacklists for adult sites is being considered.
      Children, of course, do not care. But an adult full list of "bad" sites will be very useful.
      Embedded applications.
      WordPad, calculator, Paint and other built-in tools that have not changed for ten years, will be improved.
      Personally, I hope that, for example, Paint.NET will be the replacement for Paint.




      1. ‚ÄúThis desktop is made for me!‚Äù
        The thing that I use must fit me. In the case of Windows 7, the workspace can be completely redone for yourself.
        Cultural features.
        This refers to cultural differences between countries. The user can not only set the language and time zone, but also select the so-called ‚Äúmarket theme‚Äù, which will reflect the cultural characteristics of a particular country.
        The author suggests that this will accelerate the expansion of Windows on computers of the third world.
        Access data from anywhere.
        The development of remote access. Working with documents on a remote computer will look as if you are working with them locally. The data transfer will be encrypted. The idea of ‚Äã‚ÄãOffline Files will also be developed.
        The author suggests that remote access to home resources will be tightly integrated with Windows Live.
        Movable personal profile.
        Another feature that has tight integration with Live ID. IE bookmarks, site passwords, perhaps even a whole user profile will be moved and accessible from anywhere.
        The idea is wonderful, but it is not very clear how the security situation will be.
        Home Networking.
        Simplified home network deployment and connectivity. Some resources will be automatically shared, for example, printers. Work data will remain confidential.




        1. HiFi graphics.
          It is supposed to be optimized for high resolution displays.
          Smart House.
          Wireless connection between devices and access to media data on them. MS already has Windows Media Center Extenders, as well as Windows Media Player Sharing. Also, the new scheme will include mobile devices.
          Here, I somewhat freely translated Home Media Streaming, recalling the project ‚ÄúSmart Home‚Äù by CNews.
          Improved media playback and display.
          Nothing new here. Apparently, the Media Library scheme will be expanded.
          TV.
          There will appear mechanisms for access to any type of television: digital, cable, satellite or conventional (in our understanding). Perhaps the emergence of premium services.
          I look forward to adding the word "three-dimensional" to the list of supported TV.
          Sound.
          The underlying Windows Vista stack will be reworked.


          1. Data Recovery and Diagnostic and Health.
            Shadow copies, easy backups, reliability monitor, NAP - everything that was rich in Vista. Significant changes from the current situation is not planned. It may be easier to use the old features.
            Less UAC.
            Many actions will no longer require administrative access, for example, a change of DPI. As a result, the UAC window will appear much less frequently. In a corporate environment, this means less administrative effort.
            Migrating from Vista to Windows 7 will be much easier than migrating from Windows XP to Vista. The browser is not sure about the normal migration of applications.
            Reduced TCO, increased reliability.
            Administrators will be able to deploy Windows 7 faster, thereby saving the company money. At the same time, the network built on the new OS will be more reliable.
            Devices will just work.
            It is assumed that all devices will work without additional effort from the user.
            Beautiful, but almost unattainable.
            Quick install.
            The Windows 7 development team set out to achieve a complete installation in ten minutes with one reboot.
            Reducing IT costs.
            Improved group policies, expanded PowerShell. This will save the command line. For system administrators, a set of cmdlets will appear in the OS bundle - simple system configuration scripts.
            I note that to Windows Server 2008, such cmdlets already go a separate disk.
            Improved security of corporate data.
            Thanks to an improved audit, you can control which applications were launched and which data was accessed. It is possible that you can even see the list of functions used in the application.


            Push.
            I think the concept of Windows 7 was not much different from the concept of Windows Vista. Most of the ideas did not see their embodiment. Nevertheless, it is interesting to me to find out how the future OS looks at least on paper.
            Some points seem interesting to me. For example, centralized Update or moved bookmarks. I hope that these functions will actually be implemented.
            In some points, I frankly do not believe. For example, in the absence of problems with drivers or efficient use of hardware resources.
            Integration with Live ID. Why not? Just to ensure proper security.
