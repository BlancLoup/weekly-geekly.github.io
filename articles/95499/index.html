Desktop client-server applications for PHP - SiteFusion

SiteFusion - Open Source platform for developing client-server applications in PHP . As a shell for applications uses XUL .



XUL is actively used to write extensions for the Mozilla Firefox browser.
As with HTML, XUL allows you to describe interfaces using a markup language, define the appearance of an application using CSS, and define application behavior using JavaScript. But unlike HTML, using XUL , you can create dynamics in the user interface much faster and more conveniently.



The application runs on the server generating JavaScript commands, but it looks and behaves like a normal desktop, with full access to the file system. Server settings are minimal, connects to a live web server. Suitable for creating real-time applications.



Project page : www.sitefusion.org
