Service monitoring the status of services in the cloud and alert administrators

In this article we will talk about Windows Azure Alerts - a new service of alerts and notifications about the state of Windows Azure cloud services for developers and administrators.



Windows Azure Alerts Notification Service


Windows Azure Alerts is a new cloud platform service designed for administering and monitoring computing cloud resources deployed in Windows Azure.

Windows Azure developers and administrators can easily set up monitoring rules for the metrics of their websites , mobile services, cloud services or virtual machines and receive alerts when their status changes.

Windows Azure Alerts features




In the preview phase, the Windows Azure Alerts service offers the ability to configure up to ten metric monitoring rules.

Windows Azure users can customize the monitoring of various performance indicators for websites , mobile services, cloud services and virtual machines. For example: CPU consumption, hard disk I / O operations, incoming and outgoing traffic values ‚Äã‚Äãand others.

When a configured rule is triggered, that is, when one of the parameters reaches a specified target value, Windows Azure Alerts service can send notifications to email addresses of account administrators, co-administrators, or another arbitrary address.

For each given rule, a dashboard is available that allows you to track changes in monitored metrics.

Work with Windows Azure Alerts




Let's take a look at working with Windows Azure Alerts. Go to the "Settings" section of the administration panel. Using the "Add Rule" button you can add new monitoring rules.



To create a new rule, set a name and description, as well as select the required subscription, set the type and name of the service and other parameters.



In the next step, specify the metric for monitoring and the conditions that will be set for this metric. For example, for CPU consumption, specify a percentage, above which the rule will trigger.

Specify settings for email alerts and display of monitoring results in the administration panel.



By creating a rule, you can go to the results of monitoring the service to track changes in parameters. Now, if the control values ‚Äã‚Äãare exceeded, the Windows Azure Alerts service will send you email notifications.
