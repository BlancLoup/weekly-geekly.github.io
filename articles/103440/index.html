CMS Plone 4 - Better, Faster

The final release of Plone 4 (Python, Zope Application Server) has been published. The appearance of the updated content management system went unnoticed at Habr√©, but in vain. Plone 4 has changed significantly, and for the better.
According to CMS data, Plone 4 is approximately twice as fast as Plone 3.3.6, 3.3 times faster than WordPress 2.9.1, 3.7 times faster than Drupal 7 alpha 1 and 4.2 times faster than Joomla 1.5.15.


The results of the tests are impressive.

For testing, a 2GHz MacBook with 2GB RAM was used. Plone 4.0 was installed using buildout, MAMP 1.7.2 was used to run PHP-CMS. The default installation of each product was carried out, without additional modules / plug-ins. The default content and CMS settings were also used. The system did not install caching systems. Performance was measured on the home page for an anonymous user.

New in Plone 4

Major changes in Plone 4
  • Significant performance improvement;
  • New default theme;
  • Improved search and indexing systems;
  • Dashboards to view user contribution estimates;
  • Significant changes in the processing of large files and media files;
  • New, faster implementation of folders;
  • More powerful user and group management;
  • The framework for generating forms on jQuery Tools;
  • Simplified the process of installing and running a Plone site;
  • The CMS upgrade process is lightweight, including with Plone 3;
  • Reduced memory requirements;
  • Updated infrastructure.

More information: http://plone.org/4 and video demo (in English).

About Plone

Plone 4 can be downloaded for free at plone.org , the project code is open. Installers are available for most common operating systems: Windows, Mac, Linux, and various UNIX systems.

Plone distribution takes 2% among all open source projects. 340 people and more than 300 companies in 57 countries, including Russia, participate in the development of the Plone 4 core. The project has been actively developed since 2001, CMS Plone has been translated into more than 40 languages, including Russian.

Plone has the best safety record. You can read about Zope and Plone security in the article Six top reasons for using Zope. Zope is very secure ( Russian translation ).

In 2009, Plone won first place in the Open Source CMS Award among non-PHP projects. Since the opening of the nomination in 2007, Plone has confidently entered the top three CMS and already ranked first in 2008. Most likely, the first place Plone will take in the ongoing Open Source Award .

You can learn more about Plone at plone.org and ploner.ru (the most actively developing Plone-developers project in Russia).
General information about CMS is presented in Wikipedia .
We have already begun to transfer our site to Plone 4 (there is a detailed instruction ), if there are interesting migration features, we will publish it.
