CUBRID Bug Bash Contest Results

So, it's time to take stock of our event , which started last month. In the rules, we asked users to use the latest versions of tools and Web applications CUBRID and within a month to report possible bugs in our tracker . In this way, users could get to know CUBRID products better and also help us improve their quality. As a result, we got a rather large list of bugs related to the interface, certain functions, help desk, and a couple of very important errors. We have already fixed some of them, and new versions are expected very soon.

Registered bugs


A total of 55 bugs were registered with the tracker tagged bug-bash . Among them, 33 bugs were accepted, of which 17 are already fixed. In the table below you can see the classification of all the bugs by component, as well as the estimated release date of the new version.
