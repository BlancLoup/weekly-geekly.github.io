Transmission torrent client is finally released under Windows

One of the most popular torrent clients Transmission later many years after all came out under Windows. "Windows users eventually got a good BitTorrent client," writes TechCrunch, hinting at the degradation of uTorrent in recent versions.

Transmission 2.92 distributions for 32-bit and 64-bit Windows have already appeared in the official depository (at the very bottom), along with the standard versions for Linux and OS X, although the release has not been announced officially yet.

When installing along with a standard Qt client, it is suggested to install the Transmission daemon, command line utilities and a web interface to control the client through a browser or remotely from another computer.

In terms of functionality, Transmission is not inferior to uTorrent. For example, here, too, there is a scheduler with a speed limit on the days of the week and on the clock. There is a black list to block the connection of specified nodes. You can configure the client to track certain folders for the emergence of new torrent files, set encryption rules, add / remove trackers for specific torrents.



It should be noted that in addition to the official version, there is an unofficial client Transmission-QT .

The Transmission interface for Windows is extremely simple. And most importantly, there is no advertising.
