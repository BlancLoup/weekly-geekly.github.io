YouTrack 2019.2: system-wide banner, improvements to the page with a list of tasks, new search options and more

Hello!

Today I will tell you what changes were included in the release YouTrack 2019.2!



Workflows "State machine", depending on the type of task


In this release, we added the ability to create workflows in which the rules depend on the value of another field. Thus, for the state of tasks of different types, you can set different restrictions within the same project. Learn more about this from the documentation .




Universal two-factor authentication


Now, having set up two-factor authentication for the YouTrack account, you can bind to it a device that is compatible with the FIDO2 standard. This standard provides support for a variety of security keys and biometric technologies.

System wide banner


Starting from this version, you can add a global banner that will be displayed on all YouTrack pages. To show the banner, enable this option on the global settings page and add your text there. Please note that the banner will be visible to all users of your server, including guests.
You can use this functionality to notify users about system events, such as performance issues, scheduled maintenance, or other updates.



Updates on the task list page


We continue to update the experimental version of the task list page. Several changes have been added to the latest version:

  • saving search queries: now you can save a search query by clicking on the ‚Äúbookmarks‚Äù icon in the search query window.
  • display of the number of tasks: the number of tasks corresponding to the current request is now displayed at the top of the list along with controls on the toolbar.
  • setting the visibility of the fields: we have changed the control that allows you to choose which fields will be displayed in the task list.




As always, your feedback on the task list page is very helpful in improving it, so we would be very grateful if you share them with us.

Mandatory two-factor authentication


In the group settings, you can now request that members include two-factor authentication in their accounts. This increases the security of the YouTrack server.



Improvements in time reports


Time tracking has become even easier thanks to a number of changes in the display of these time reports.

  • A new option to display the types of work shows the totals for each type of work in a separate column.
  • A new parameter in the report settings allows you to display values ‚Äã‚Äãfor custom fields if the ‚Äúper issue‚Äù parameter is enabled in the report. This parameter also determines which fields are included in the dataset, which can then be downloaded in CSV and Excel formats.
  • A new option in the report allows you to display the remaining (unspent) and total elapsed time.



Parameters for future dates in the search query


We added predefined values ‚Äã‚Äãto the search query for future date ranges. Now you can find tasks that are scheduled for tomorrow, next week or next month.



Password Support for Bitbucket Integration Applications


Now YouTrack can be connected to the Bitbucket repository with the username and password of the application. This allows you to customize integrations with Bitbucket accounts for which two-factor authentication is used. It also means that you no longer have to use your personal password in the integration settings.



As we announced some time ago, starting with YouTrack 2019.2, we officially stop supporting Internet Explorer 11. YouTrack will still continue to work in any versions of this browser, but we no longer guarantee stable performance and do not correct errors that can occur only in this browser. For optimal performance, we recommend switching to a supported browser .
