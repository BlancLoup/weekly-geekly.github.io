Useful resources for interface designers in Axure RP



Quite a long time ago there were no articles on Habr√© about such a convenient and effective tool for rapid prototyping as Axure RP.
I decided to post my selection of bookmarks on Akshur in the form of a consolidated list of resources.


Libraries and .rp files with interfaces

acleandesign.com/2009/04/better-defaults-for-axure-a-widget-library
acleandesign.com/2008/09/axure-design-pattern-library-v2
acleandesign.com/2009/09/social-widget-library-for-axure - sets of elements from Loren from ACleanDesign.com;

axure.com/widgetlibraries#Widget_Libraries - official libraries and widgets;

axureland.com - site collector of free widgets and libraries;

userfocus.co.uk/resources/axure_grids.html - 960 Grid Markup for Axure;

axlib.uxtechniques.com is one of the richest libraries;

konigi.com/store/product/wireframe-icons-axure - simple everyday icons from konigi.com for Axure;

archive.org/details/AriFeldmanArisOSXAxureRPWidgetLibrary is a library of MacOsX elements, the detailing is not the best, but is suitable for rapid prototyping right in front of the client;

kreativr.de/ipad-axure-widget-library - a huge number of elements for the design of iPad applications;

paulsizemore.com/axure-iphone-widgets-and-library - elements for prototyping iPhone applications;

totalwireframe.com - paid libraries, there was a hope that the site would develop into something worthwhile, however, apart from a small set of libraries, nothing new appeared;

viminteractive.com/blog/post/axure_pro_social_media_icons_widget_library - social media icons;

webbizdesigns.com/axure-widgets/advertising-web-ads-axure-widget-library-file - a very useful library with various types of banners;

conetrees.com/2009/06/downloads/download-axure-clear-input-field-value-on-focus-prototype-widget-library - an input widget with the implemented field cleaning function with focus;

Axure Network Resources

axure.com/howto - for beginners, a very detailed description of the program from the authors themselves;

analyst.by/tools/1049-axure - good translations of individual Axure articles;

ru-axurerp.livejournal.com/862.html - an excellent interpretation of the Actions panel;

uxsux.com/post/3938302984/speed-prototyping-mit-axure - high-speed prototyping of the site in Axure, there are many useful points, especially for newcomers to Axure;

usabilityprofessionals.ru/UsabilityBulletin-26.aspx?EntryID=787 - usability bulletin about Axure, a lot of useful information when considering the transition to prototyping in this tool;

Indirectly useful resources for Axure

subtlepatterns.com - backgrounds for your prototypes;

patterntap.com - patterns for interface designers;

usability.ru/sources.htm#UI - great books on usability and everything connected with it;

I would be very grateful if the comments will indicate the quality resources that were unfairly deprived of my attention.
