Instapaper is available for Android, $ 3



This application is actively used by those who like to postpone reading for later - saved the site page using Instapaper, and then in his spare time he opened and read the necessary material, without banners and decorative excesses. You can save a lot of interesting things for later reading offline.

Previously, the application was available only to owners of apple devices, now there is a version for Android, thanks to the guys from Mobelux .


It is an application 101 rubles, here's a page on the market.

Functions

- font size and font itself - to choose;
- night reading mode;
- synchronization between your devices the position at which you finished reading the previous time;
- convenient folders for storing articles;
- native sharing;
- rotation lock.

Requires Android 2.2 and up.
