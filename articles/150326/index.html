Firefox 15 release

As usual, the release of the stable version of Firefox appears before the official announcement, which will take place on August 28 (Tuesday) .
What's new in this version:

  • There was another optimization of memory consumption in the framework of the MemShrink project. This time efforts were directed at reducing the memory consumption of add-ons installed in the browser.
  • Added built-in document viewer in PDF format - PDF.js. The viewer supports scaling and page thumbnail navigation functions, provides loading of TrueType and Type 1 fonts, can generate graphs and charts, supports work with compressed objects.
    By default, PDF.js in version 15 is disabled. For those who want to test this function, in about: config, find the pdfjs.disabled parameter and switch to false ;
  • Improved Firefox startup speed in Windows;
  • Implementing Opus audio codec support;
  • Added support for the third version of the SPDY protocol.
    By default, support for this protocol in version 15 is disabled. To enable it, go to about: config, find the network.http.spdy.enabled.v3 parameter and switch it to true ;
  • A full JavaScript debugger is integrated into the developer‚Äôs tools (Ctrl + Shift + S);
  • Added tool "Adaptive design" (Ctrl + Shift + M) for browsing web pages on different types of devices with different screen resolutions;
  • In ‚ÄúInspector‚Äù (Ctrl + Shift + I) a new view mode has been added to view the layout of elements on the page;
  • Improved work with WebGL: implemented using compressed textures to improve performance;
  • GStreamer multimedia framework support as a backend for playing HTML5 audio / video elements;
  • The audio and video elements support the played attribute, and the source element supports the media ;
  • As part of the Performance API, support has been added for a high-precision event timer, which allows to find out the time to within one thousandth of a second.
