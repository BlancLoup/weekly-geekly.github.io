Introducing the New 3CX Call Flow Designer and 3CX CRM Template Generator

New 3CX Call Flow Designer with Visual Expression Editor




3CX adheres to the principle - our products must be simple and clear. And so we once again updated the 3CX Call Flow Designer (CFD) voice application development environment. The new version has a modern user interface (new icons) and a visual editor, an expression editor used to create scripts.



The new version of CFD offers the following features:

  • Visual expression creation. You can drag and drop the built-in functions, variables, and constants from the Expression Elements section. You can intuitively create complex expressions without opening multiple editor windows.
  • Quick feature selection. Built-in functions are now grouped into convenient categories from which you can quickly select the desired function.
  • Dynamic inclusion of constants. Available constants are dynamically included in the Expression Elements panel when appropriate components are added to the CFD workspace.

New 3CX CRM Template Generator for connecting your CRM


We also introduced a new version of the template generation utility for connecting your own CRM systems to 3CX. In the new version of CRM Template Generator, an XML Template Creation Wizard has appeared, which will guide you through all the stages until you get ready-made and working integration.



To get started, check out the RESTful API documentation for your CRM. Then run the Template Generation Wizard.

  • Follow the prompts of the Wizard, indicating the appropriate parameters from the documentation for your CRM API.
  • Creating an XML template requires basic programming skills and support for the RESTful API CRM system.
  • When the wizard works, each section of the template is correctly formed with the inclusion of all the required parameters in it.
