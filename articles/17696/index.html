Firefox VoIP Extension - SIP Phone in Browser

The Italian company Abbeynet presented another interesting development - an extension for the Mozilla Firefox browser, which allows using the browser as a full-fledged SIP client. After installing the extension, a small panel will appear in the browser, using which you can make VoIP calls all over the world, at the rates of any SIP operator.

To configure the Firefox VoIP Extension, you only need to specify the SIP server, username and password.
