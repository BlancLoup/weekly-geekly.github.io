Yii 1.1.13 RC

Sometimes, even in the release versions, errors creep in, which are detected immediately after the release, so we decided to try to release several prerelease versions.

After RC and up to a stable version (GA), only bug fixes for RC will be accepted.

So, the preliminary release of Yii version 1.1.13 is posted. You can pick up, as usual, from the framework download page:

www.yiiframework.com/download/



This time a lot of work has been done: more than 120 improvements and fixes. Many thanks to everyone who takes an active part in the development on github: resurtm, SonkoDmitry, klimov-paul, creocoder, DaSourcerer and many others.

Since this is not a final release, but a preliminary one, you should not use it on combat servers. A stable release will be released approximately within a few weeks, after fixing all the errors found, but for now we will be very grateful if you check your projects with preliminary release, albeit locally, and inform us about errors. Thank you in advance!

Also, thanks for your bug reports in the yiiframework.ru documentation through Orphus. Finally, an automatic update of all documentation is set straight from the repository, so that it will always be the most recent on this site.
