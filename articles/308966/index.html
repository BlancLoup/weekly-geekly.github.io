New release of Kali Linux 2016.2



A little more than a year after the release of Kali Linux 2.0, Offensive Security (as promised on Black Hat and Def Con) released a new distribution called Kali Linux 2016.2. Let me remind you that current distributions are released in the form of a so-called. rolling release.

A brief digression: Kali Linux is a distribution kit containing many utilities for conducting penetration testing - from analyzing the vulnerabilities of web applications to hacking networks and services and consolidating them in the system. Previously, this distribution was known as Backtrack.

What's new?


First of all, this is native support for multiple desktop environments. Many did not like the heavy Gnome, which was used by default in the previous release, so you had to put something easier, for example XFCE. Now you can download pre-built builds with the most preferred environment: KDE, Mate, LXDE or Enlightenment version.



Also, the package update mechanism has undergone changes - in the case of a ‚Äúlong release‚Äù the update process took quite a long time. In order to have the latest versions of the distribution (with an updated set of utilities) Offensive Security decided to publish new releases weekly and distribute them using their mirrors. If you have the opportunity to place a mirror with high bandwidth and thereby help the project - for you there is a special page .

Bugfixing


New relevant tools for Kali have been added to the current build, a number of various bugs have been fixed, and the operating system has been improved. For example, adding HTTPS support to BusyBox now allows you to install a pre-distribution via SSL.



The outdated Kali Sana repositories were also moved - they are now in the EOL stage and placed in old.kali.org.
