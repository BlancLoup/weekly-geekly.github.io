Foresight Mobile Edition 1.0 - replenishment in mobile Linux

Today, Foresight Mobile Edition 1.0 has been released - Linux for netbooks or ultra portable PCs, which contains: an automatic update system, a revolutionary Conary package manager , a GNOME desktop environment optimized for working with low screen resolutions and a selection of necessary software.




This miracle happens from a famous project from Canonical, which I wrote about earlier . Emphasis was placed on the speed of access to applications and documents. To return to the desktop from the menu, simply click on the GNOME logo in the upper left corner of the top panel.

Foresight Mobile Edition includes the following applications:
  • F-Spot Photo Manager
  • Firefox web browser
  • Pidgin Instant Messenger
  • Banshee Media Player
  • Gnome movie player


As well as applications and system utilities included in GNOME.

Also for the Mobile Edition users, the repositories of the standard Linux desktop distribution Foresight GNOME edition are available. To find and add an application, click Add / Remove Software in the Administration menu on the left side of the screen.

Foresight Mobile Edition has been tested on Asus eeePC and Intel Classmate.

Some bugs are known on the Asus eeePC: the problem with turning off the power from the applications and the problem with the wireless network.

Download this miracle here: http://www.foresightlinux.org/mobile.html
