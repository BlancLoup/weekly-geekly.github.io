Germany has developed requirements for home routers

The sharp increase in the participants of our channel in Telegram (@router_os) showed that the problem is extremely acute.

In Germany, a document was issued with recommendations and minimum standards that the router should meet for end users.

The document contains 22 pages. Recommendations that should be applied to new routers include the default settings requirements, including:

  1. Hard firewall settings
  2. Prevent remote control
  3. WI-FI password is only for WPA2 and should not allow it to "count"
  4. Automatic firmware update.
