Comprehensive Registry Tweak for Windows 7 + Bonus: Migrate User Data

Quick Windows Setup

Bonus: Moving user data.

By default, user data on computers running Windows 7 is stored in the C: \ Users \% username% folder, which is inconvenient. Inside the user's folder there are several sub-folders, which for some time, for ‚Äúsloppy‚Äù users, ‚Äúexpand‚Äù considerably in volume. These folders are: Desktop, Downloads, Documents, Music, Images, Videos. I suggest immediately after reinstalling Windows to transfer these folders to the D: \ drive. Of course, you can do it manually - in the properties of each folder, change its location. And you can use my batch file, which will "carefully" copy all the contents to a new location and change the necessary registry settings.
