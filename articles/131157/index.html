Opera Mini 4.4 Final Release



Today came the new final version of Opera Mini 4.4 for the J2ME platform. In terms of its functionality, this release is close to what is planned for Opera Mini 6.5 (Android versions have already been released recently and are being prepared for other platforms), but at the same time, this version can work even on the weakest phones that 6.5 can have difficulties due to inadequate hardware.

From noticeable changes:
- fixes in the authorization process when using Opera Link
- improved stability and performance

In addition, this version contains changes that allow mobile operators to expand the possibilities of unlimited Internet access for Opera Mini users.

As always, you can download the new version from the official download page directly to your phone or computer .
