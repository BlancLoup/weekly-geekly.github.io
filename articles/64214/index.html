One of the best twitter clients goes to the web

Some time ago, on Habr√©, there was already a review of bDule - powerful, beautiful and, in my personal taste, the best twitter client for windows. Since that review, bDule has become even better and has got new features, so the latest release that happened the other day added functionality with the ability to search simultaneously not only on twitter, but also on FriendFeed and OneRiot , plus a list of the most popular twitter trends appeared.

But the main announcement of the company Sobees, which develops bDule, was the announcement of the resource www.sobees.com, which is an almost complete online copy of the desktop version of bDule. Look at the screenshot:



The online version is written in Silverlight 3, which was released the other day. It is not much different from the desktop version and looks exactly the same. In the presence of a customizable interface, support for multiple accounts at once, powerful search. But the main feature of the project is the support of out of browser, which allows you to disconnect the web application from the browser and make it desktop with shortcuts on the desktop and subsequent launch without a browser. So the twitter client stepping from the desktop to the web is able to step back to the desktop.

Anyone with a platform that allows you to run silverlight 3 applications, I strongly recommend trying the new Twitter web client. He may well become your main twitter client.
