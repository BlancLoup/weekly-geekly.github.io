PhpStorm 5 Released - Even More Smart Useful Pieces



Continuing to focus on the analysis and quality of the code, we finally got to the frameworks. Work on the IDE is already underway, and right now we offer to try:

  • New type analyzer - fewer incorrect warnings without additional annotations
  • More warnings about runtime errors right in the editor
  • More highlighting of unused code and constructions
  • MVC framework entity bar - while Symfony 2 and yii, the rest as ready
  • Transparent PHAR support
  • Support for callbacks in literals (call_user_func, etc) - including usage lookup and refactoring
  • Serious improvements to the database console - for all databases - refactoring, export, procedures
  • Code formatting according to PSR1 / 2, Symfony, Drupal standards
  • LiveEdit - editing PHP / HTML / CSS / JS can be seen in the browser in real time without reloading the page
  • Seriously improved support for CSS / SASS / SCSS / LESS - analysis, addition, formatting
  • Jade template support
