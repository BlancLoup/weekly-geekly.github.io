Sony is going to release a laptop with Chrome OS



Sony is going to soon release a laptop that will work based on Google's Chrome OS. It will be a netbook rather than a laptop, with a display diagonal of 11.6 inches, and probably with the Nvidia Tegra 250 T25 platform.

Like most modern netbooks, this device will be equipped with WiFi and Bluetooth wireless communication modules, USB and HDMI ports. The new laptop / netbook will be equipped with an SD memory card slot, which can be used as an additional drive.

By the way, the platform Nvidia Tegra 250 T25 assumes the presence of a dual-core processor ARM Cortex A9 1.2 GHz. In addition, it is known about 2 GB of RAM. The netbook will not install a hard drive, but probably an SSD-drive with a capacity of 16 GB. So the device should be powerful enough, fast, and hopefully functional. The new model received the working title of Sony Vaio VCC111.

Unfortunately, when the device should go on sale, and how much it will cost is unknown. However, the application to the FCC was filed by Sony back in September last year, so we can hope for a netbook to be available soon.
