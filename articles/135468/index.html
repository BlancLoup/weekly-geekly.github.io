Firefox Sync will support the synchronization of add-ons (extensions and themes) between computers

As you know, Firefox Sync is a tool that allows two (or more) Mozilla Firefox browsers on several computers (or even cell phones) to have the same set of bookmarks, passwords, settings, visited pages and open tabs. This set is replenished synchronously: for example, you will add a bookmark on one computer - and it immediately appears on the other.

Each item from this list can be disabled (for example, who does not want to transfer saved passwords from a personal computer to a worker, he completely disables the synchronization of saved passwords). Extensions can add their own elements for synchronization: for example, Adblock Plus can share inconsistent filters.

The Firefox developers thought and thought, and in December of this (2011) they added synchronization support and extensions to Firefox Sync (see bug 534956 ); Considering the schedule , then its appearance in the release will take place in the spring of 2012.

Details on how the new feature works are outlined in MozillaWiki . Initially, it will only affect extensions located on addons.mozilla.org, and will not be enabled by default for current users of Firefox Sync, in order not to be terrified by their possible drastic changes in the appearance and functionality of the browser after the update.
