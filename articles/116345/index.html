Monitoring uptime 50 popular APIs

The startup WatchMouse, which supports the operation of the API-Status monitoring service, has published uptime statistics for the 50 most popular APIs for the last month (from February 16 to March 17).

The efficiency of the API is checked every five minutes from 56 geographically distributed WatchMouse servers with a simple query that checks the validity of the result.



API popularity was calculated by the number of mash- ups in the web services catalog ProgrammableWeb .
