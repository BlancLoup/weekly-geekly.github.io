Malware presence detected in the GNOME-Look directory.

In the deb-package with the screensaver waterfall, distributed through the well-known GNOME-GNOME-Look application directory, the presence of a hidden malicious insert was fixed. This insert can:
  • Update yourself
  • Participate in DDoS attacks
  • Receive remote tasks

To remove a trojan, enter the following command:
sudo rm -f /usr/bin/Auto.bash /usr/bin/run.bash /etc/profile.d/gnome.sh index.php run.bash && sudo dpkg -r app5552
