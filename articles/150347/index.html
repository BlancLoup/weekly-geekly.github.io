Cackle commenting system - new features

Cackle is a commenting system for creating an active community from the audience of your site. Below is a brief description of the main features and benefits of the system, and under the cat a story about the new features.

Main:

  • Authorization without registration through the 18 most popular social networks, among them Vkontakte, Google+, Facebook, Twitter, LinkedIn, Mail.ru, Yandex and others
  • Crossposting in social networks: Vkontakte, My World, Facebook, Twitter
  • Anonymous commenting, Ban users or IP addresses, Filter obscene words, Comment Rating
  • Widget recent comments and the number of comments
  • WordPress, Blogger, Joomla, PrestaShop, LiveStreet, MaxSite Plugins
  • Image recognition png, jpg, gif, video from Youtube, Vimeo, Vkontakte, RuTube, Google presentations and Slideshare
  • Moderation through the widget on your site, without logging in to the admin panel


What's new?

We made a new design. On the main page you can find a demonstration of the widget, a description of the main features, popular clients, see the latest topics and comments, as well as leave a question or suggestion through the feedback form.

The main domain cackle.ru moved to the domain of the international level cackle.me with the English locale by default, the Russian locale is available at ru.cackle.me.

In the admin panel there is a button by which all comments from the selected user are deleted.

Added authorization via LinkedIn.

The plugin for WordPress saves Cackle comments to the database of your server, due to which they can be indexed in search engines.

When creating a comment containing links to images (png, jpg, gif), videos (Youtube, Vimeo, Vkontakte, RuTube) or presentations (Google, Slideshare), players will be added from the bottom.



Updated the FAQ section.

There are three paid accounts - Start, Pro, VIP.

Cackle start

This account is suitable for bloggers and small online stores. Key features:

Moderation of comments right in the widget on your site

If you are the administrator or moderator of the widget, then you can see all the comments (rejected, deleted, spam) and you can administer them through the dialog that occurs by clicking on the link ‚Äúmoderate‚Äù for each comment.


Header editor

An editor is available in the admin panel, on the ‚ÄúHeaders‚Äù tab, in which you can change any text or widget title, for example: 'on behalf of', 'Leave your comment', 'Comment' and all the rest.


Missing link

Due to numerous requests from customers in a paid Start account, there is no link to cackle.me.

Cackle pro

An account for more advanced IT users, includes two main features:

Crossposting comments to any external service

In the admin panel, in the API tab, you can register any address of the service to which all the parameters of the comment (message, author, time, url, etc.) will be sent when creating it or changing its status.

Style editor

Allows you to change the color, background, font, size, css3 effects and the location of some elements of the widget.

Cackle VIP

The most expensive account for sites with a large load (more than 1 million users per day). The main advantage of this tariff is that your commenting system will be installed on a separate dedicated server and will not depend on the performance of the main servers.

Very soon a white list of users whose comments are published without pre-moderation will appear in the admin panel.
Very soon we will filter comments on statuses - expect, approved, rejected, spam.
Very soon, we will make a comment cross-threading in several social networks at once and add a Help section.
