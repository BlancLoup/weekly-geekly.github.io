Logitech Announces G25 Edge Keyboard Coming Soon

Logitech announced the imminent release of the G25 Edge keyboard, each key of which is a miniature LED display and can be programmed as convenient for the user: with the G25 Edge, you can set not only the character values, but also highlight the keys in any color.


The keyboard is positioned as a convenient device for work and play. In addition to the 113 main keys, the G25 Edge also has nine additional G-keys, which you can program in 27 more actions. In addition, the gadget is equipped with a horizontal scroll wheel, with which you can control the functions of five additional keys, programmed at your discretion.

When designing the keys used new technology ePaper, which, according to the manufacturer, is not subject to wear. The minimum resolution produced by the display key is at least 64 x 64 pixels. Highlight the buttons can be any color from the palette, numbering 512 shades. G25 Edge, which supports Windows Vista, will be offered in black, white and silver cases. It should appear on sale in the near future, however, the price of the novelty has not yet been announced.

It is likely that it will be about 1.5 thousand dollars, because about the same is the keyboard Maximus Optimus , released by the Studio Artemy Lebedev in March of this year.
