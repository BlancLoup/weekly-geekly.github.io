New Intel SDK for OpenCL Applications 2014

Intel SDK for OpenCL Applications 2014 is a unified development environment for OpenCL API with new debugging features.
New in the latest version:
  • Unified SDK with support for Intel processors, Intel Xeon Phi coprocessors, and Intel graphics;
  • Enhanced support for Intel Atom processors with Intel HD Graphics and PowerVR G6400;
  • remote development for OpenCL for Android;
  • debugging at the API level;
  • The new OpenCL executable (version 14.1) for Intel and Intel Xeon Phi processors.

New functionality in the testing phase:
  • step-by-step debugging of the OpenCL core on Intel Graphics;
  • generation and use of SPIR 1.2 code;
  • recommendations in the kernel code for optimal code.

The new product combines all the functionality previously available in versions 2014 Beta, SDK XE 2013, and SDK 2013. It is recommended that everyone using this software update both the development tool and the drivers and executable files associated with it.
