TechDays.ru: 200 free online reports or why we did it

About a week ago, Habr√© already told that we launched a new version of the site TechDays.ru . I would like to explain in more detail why we are doing this project, how it can help you, and also to highlight some controversial points noticed by the community. Before going to the main part of the message I want to note that all content is ABSOLUTELY FREE and will remain so.
The whole post is divided into parts about the content, design and functionality, so you can safely flip down to the fragment you are interested in. Perhaps I'll start with the most important thing, with the content

Content :
- We have added more than 80 new reports telling about Application Development and IT Infrastructure Management
- We have created the Ready solutions section, in which we offer ready-made manuals and tools for solving work tasks, which you can use in your activity the next day.
- Added reports on Windows 7 (in the near future we expect replenishment)

Continued under the cut:

Design + usability :
When I talked about the launch of the first version of the project in September 2008, I received quite a lot of negative feedback from you. I admit, many of them were fair, design and usability were clearly limping. That is why one of the main tasks when launching an updated version of the site we gave design. So, what we did:
- Made the design modern and beautifully displayed by all browsers
- On the page of the report they gave the central place to the player (see the screenshot below)
- Grouped all materials related to the report, next to the player. Now you can easily find the file to download to your iPhone :)
- Added a tag cloud to simplify navigation (and there is a tag cloud for all audiences, as well as separate ones for application development and infrastructure management)
- Made explanatory pictures for those who do not have Silverlight installed
- Added filters to categories to simplify the search for the desired content.


Functional :
The question of functionality was also not the last in the list of works that were carried out before launch. What is new?
- One of the main innovations: the ability to insert a video of any report to your blog or website. To do this, you need to register on the site and go to the page of the report of interest. To the right of the player under the voting block for the report you will find the link Code for insertion into the blog, clicking on which you will receive the code on the iframe, which later can be placed on the website / blog
- We made readable URLs. Hooray :) You no longer need to memorize unrealistic links, now they look like this: http://www.techdays.ru/videos/1237.html
- The Selected Reports page has appeared. After registration, you will be able to add interesting reports for you on the site and then quickly access them.
- In each category, the opportunity to subscribe to the RSS feed updates. In general, now almost any update in any category / report can be subscribed through the RCC

Frequently asked questions . I will try to give answers to them, because Often these things cause difficulties when working with the site:
- To view all the videos you need the Microsoft Silverlight player, you can download it here . (free). It is installed on MACs :) For installation on Linux, FreeBSD, see this page .
- If you do not want to install Silverlight, then just download the files you are interested in: WMV, Podcast (mp3), video for iPhone / iPod (m4v)
- Silverlight works on the latest versions of the Opera browser
- Live.ID and Open.ID are only authorization systems, that is, when you first enter the site, you will be asked to fill in a SHORT registration form (minimum fields). In the future, you can easily log in to Live.ID and Open.ID and forget about the login / password on the site
