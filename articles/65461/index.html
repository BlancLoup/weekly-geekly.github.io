Unified communications in action

My friends and partners ITooLabs are working on ITooLabs Unified Communications
Server. In spite of the fact that the decision is more likely to be a camera solution, this does not prevent us from making services for ordinary users.

RZ.ru is the first service based on ITooLabs UCS. Now this is more likely a demo beta for invites, but in the future, I hope will become an independent service.

What is on rz.ru :

  • Mail: WebMail, POP3, IMAP4 and other joys such as push e-mail (AirSync).
    For iPhone, specify as Exchange, for Nokia there is a Mail for program
    Exchange.
  • Work with contacts on the server. Synchronization with the phone (AirSync). For
    happy owners of Apple computers - there is an ITooLabs iSync Agent .
  • Synchronization of calendars and tasks (iCal, AirSync). Web version of calendars
    not yet.
  • Integration with MS Outlook via MAPI. Contacts, tasks, meetings,
    delegation of rights and group work. Download plugin .
  • Instant Messenger. The server supports XMPP and SIP / SIMPLE protocols,
    therefore, you can connect with any SIP or Jabber client. For Windows there is
    own development - ITooLabs Communicator
  • Telephony. SIP protocol is available in the demo version. You can call
    both within the server and other SIP subscribers. Call Screening / Call
    Presentation, Voice Mail, IVR and other chips.
    Connect SIP phones, softphones and other equipment.
  • In the commercial version, work with urban
    Numbers, Advanced Call Management, Voice Traffic Routing
    and other things specific to IP PBX.


Because extensive documentation in Russian in public access so far
is absent, the offer is designed, rather, for advanced users.
