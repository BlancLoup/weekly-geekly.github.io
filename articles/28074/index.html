Safari, Mac OS X 10.5.3: Working with WebMoney Light

Because of the large number of questions from friends, I decided to write this post.

I think many people noticed that after upgrading to 10.5.3 - safari stopped working with webmoney light.
This is due to a tighter policy with certificates.
This is corrected by adding the following addresses to the allowed addresses for the WM certificate:
light.webmoney.ru/login.aspx
light.webmoney.ru/LoginCert.aspx
login.wmtransfer.com/wmauth.aspx

More details here: support.apple.com/kb/HT1679?viewlocale=en_US
