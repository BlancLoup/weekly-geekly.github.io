iPADrom - S01E05

iPADrom is a rubric dedicated to iPad app video review. In each issue I will try to tell interestingly about three applications.



Photoshop for iPad is a simplified version of a popular graphic editor. In this case, the application is intended solely for image editing. No drawing mode or use brushes, only the most necessary.
Home Budget - an application for doing home bookkeeping. A convenient and beautiful shell allows you to keep all your costs and income in a convenient and understandable form. Using this program for a long period you will be available all kinds of reports and analytics.
eBay for iPad is the official client for working with the popular online auction site eBay.com. In this application, all the most necessary tools are available for purchases, rates, search and selection of the required lot.
