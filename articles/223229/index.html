Typograph Muravyova published on GitHub



We have been asked for a very long time to publish the GitHabe typographer to enable us to develop the product even faster. Since we do not create modules for different systems, but we do only the typographer code, then it is time to publish the typographer on the GitHab and give everyone the opportunity to make forks and openly develop the product to fit their needs.

Wellcome: https://github.com/emuravjev/mdash



Taking this opportunity, I want to tell you that this is version 3.3 already - corrected and supplemented. Since the publication of the first version of PCP, we have made a lot of changes (you can see them on the site ) and released a version for Python. On GitHub we published only the main product code, but besides it, we have autotests and a compiler code for Python. All this will also be published in the foreseeable future.

The next version is likely to be for Ruby and will include several new chips. We still continue to develop the service and are very pleased that already dozens of sites use our work. I would be grateful for comments and new suggestions.
