Taming file sharing P2P networks. DC (Direct Connect). Part 2

Not much time has passed since the first publication of the MediaDC program on this site, but we can already say that the program is of interest.

In short, the program is a tool for viewing files (movies, music, pictures) using DC (Direct Connect) protocol.
Thus, it is possible, without downloading, to evaluate the content and avoid frustration from the lost time of downloading an interesting (in poor quality, etc) movie.

The program is designed to erase the line between local files and files available in DC, in the likeness of the usual (SMB, Netbios balls).
A more detailed description can be found in my first post Taming File-Sharing P2P Networks. DC (Direct Connect) .
