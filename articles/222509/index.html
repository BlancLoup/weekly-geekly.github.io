New to the Cisco UCS Server Family



Simultaneously with the release of a new line of Intel Xeon E7 v2 processors, Cisco has developed three new servers based on them: Cisco UCS C460 M4 Rack Server (4-socket, 4U rack server), B460 M4 Blade Server (4-socket, full-size blade server) and B260 M4 Blade Server (2-socket, full-length blade server). The new servers are the flagship line of Cisco and are designed to solve complex corporate problems that require working with large amounts of memory.

In the new models, as in all Cisco UCS servers, the UCS Management solution and the SingleConnect technology are initially implemented. Using Cisco UCS Manager, an administrator can programmatically model the desired server (determine the service profile) and then create an instance of this server. In other words, server hardware is abstracted and modeled by software. The process of configuring and deploying new servers based on existing ones takes only a few minutes. Using the UCS Manager management system and service profiles, you can move the workload to a more powerful server. To do this, simply create an instance of the service profile of the previous server on the new server. This is a simple transition to a server of greater power - and this is the essence of the new concept of vertical scaling. At the same time, from the point of view of the operating system and the hypervisor, the new server is no different from the old one, except that it has more resources at its disposal.

In addition to the processor Intel Xeon E7 v2, the new UCS-servers have other advantages that can be described as: horizontal scaling, horizontal scaling, plus vertical scaling.
Horizontal scaling. The characteristics of the UCS B260 M4 Blade Server are horizontally scaled to the following values:
- I / O rate 160 Gb / s,
- 1.5 TB memory with the ability to grow up to 3.0 TB,
- 2 expansion slots and built-in network adapter mLOM,
- two Intel Xeon E7 v2 15-core processors operating at a maximum rated power of 155 W,
- An innovative design (upgrading the Cisco UCS B260 M4 server to double the processing power, memory, and I / O throughput is done by simply adding another UCS B260 M4 module and a scalable UCS M4 Scalable Connector).
