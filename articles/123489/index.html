DataRetriever - a service for collecting and analyzing data from online media and social networks

We present to your attention our product - DataRetriever - a service designed to monitor open sources (Internet media, social networks, blogs, forums, multimedia services) and analyze the data obtained.



What does all this mean?

It's simple :) With the help of our service, you can find mentions in the media and social networks - companies, products, people, etc., build graphs of the number of publications, analyze sources, generate reports, export data to MS Office.

Basic settings include working with the following sources:
1. All media available on the Internet and available for processing.
search engines
2. Large blog sites: Livejournal, LiveInternet, Blogspot, Wordpress, etc.
3. Twitter
4. Social networks: Facebook, Vkontakte
5. Flickr, Youtube

And the list can be supplemented and changed without restrictions. Tracked
both texts, and graphic materials and video.

How it works?

User workflow is close to working with a search engine: you need
set a list of keywords or phrases, and, after some time (necessary for
monitoring), the first results appear. Next, the system continues
monitoring sources for specified keywords offline, adding to the list of results:


Information retrieval can be done in multiple languages. System is
self-learning: if the monitoring process reveals new sources of information, they are automatically remembered, the initial list of sources is expanded, and the source is automatically removed from the list if the user deletes keywords associated with this source.

The received information can be processed. For this, the user is provided with a set of tools for analyzing the number of publications over time, identifying the most active sources, determining the tone of publications, compiling reports and exporting them to MS Office format, as well as tools for teamwork (chat).

Here is a graph of the frequency of references to a particular topic:


Technically, the system is based on Drupal 6; The prototype was one of his little-known assemblies of a similar purpose (Tattler) - we taught her to work with the Russian language, corrected multiple errors and omissions, and added some new features.

Who needs it?

This is a tool for a PR specialist, marketer, SMO-optimizer, and press analyst. The presence of a free version and a workable demo distinguishes our product from competitors.

System options

DataRetriever exists as an Internet service ( demo version ), and as a server application distributed under a GPL version 3 license (we will soon post it on the site). Both options are provided with user manual.

What our product does not do, but will soon do

While there is no determination of the location of the authors, we hope it will appear in the next version. We also plan to provide the system with a semantic analyzer, which will allow us to more accurately find references and screen out garbage.
