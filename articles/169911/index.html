1C Sync Setup Wizard for PHPShop Pro 1C

The correct operation of the information exchange mechanisms of programs based on 1C: Enterprise with the PHPShop site depends on how correctly the synchronization settings are made in the exchange processing. However, it is the implementation of the initial settings that most often cause users the greatest difficulty.

As one of the solutions in providing technical assistance to its users, PHPShop included a configuration assistant in the 1C exchange processing, which allows you to consistently fill in and check the settings for connections to the site and ftp server, uploading goods to the site and loading back orders into 1C.


Running assistant


The settings assistant is automatically called when you first start processing on the user's computer. After filling in and checking all the entered data, the user will be prompted to save the changes and the work of the assistant will be completed. However, if necessary, the assistant can be opened at any time. To do this, go to the processing settings and select the item "Open Assistant".
