Meet the new address bar, mouse gestures and updated mail with extensions


On the eve of the release of the beta version of Opera 11, we are pleased to present you today two new features and several significant improvements, in particular, when working with mail.

Safer address bar


The browser address bar is an extremely important feature. This is your key to navigation, search and security on the web. For Opera 11, we want to bring the address bar to a new level, making it less noticeable and helping you focus on the most important tasks. To begin with, we note that the new address bar should protect the majority of users who think that they are on Google, while in fact they went to google.com.spoof.co, as well as for those users who cannot ‚Äúparse ¬ªA string with the URL, without making a conscious effort. Opera‚Äôs tech-savvy users can, and can, do it, but for everyone else, the contents of the address bar are a real "garbage". Although, on the server side, this can be decided differently, our solution is an attempt to highlight in the link only the most important part, which should show the users exactly where they are.

Features of Opera's new address bar

  • All web pages received an icon in the left part of the address bar. The icon replaces the names of the protocols, such as HTTP, HTTPS and opera:, which are now displayed only when the mouse cursor is placed in the address bar.
  • Clicking on the icon provides a quick overview of the more comprehensive security information found on the current web page.
