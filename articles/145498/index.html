Facebook has opened the App Center application directory



Social network number 1 in the world introduced a new application catalog App Center , which was announced around mid-May this year. This directory, according to the developers, will help social network users to find the necessary web applications, or mobile applications. It is clear that everything that is placed in the directory is related to Facebook itself.

On the main App Center page, you can see both the applications already selected by the user, and those that the developers of the social network liked (similar to the ‚Äúeditorial selection‚Äù on Google Play). Provided and the breakdown of applications by category, which facilitates the search for what the user needs at the moment. If you wish, you can hide all web applications, leaving only mobile, and vice versa.

The catalog itself is organized somewhat differently than analogues from other companies. As mentioned above, the catalog only makes it easier to find the application you need, but you can‚Äôt buy anything directly in the catalog (the purchase is made on Facebook, but all the contributions are left to the developers). By the way, if a user enters the App Center from a mobile device, the directory automatically detects the OS of the mobile device, and shows applications for that particular device.

In App Center there is a rating, a description of programs, screenshots, and everything else that users may need. Now the catalog has only a few hundred applications, but it is planned to expand the range to about 4.5 thousand.

Currently, the App Center is available only for Americans, however, in a few days or weeks it will be available to users from other regions.
