The first presentation of Windows 7 will be held in October



Some of us like Microsoft products, some don't. But many of us are waiting for the new Windows 7 operating system, which will replace Windows Vista in every way criticized. And, most recently, Microsoft has opened an official blog dedicated to the development process of this operating system.

While there is little interesting information, as the developers keep the details about the project in secret, however, one of the main Windows developers, Stephen Sinofsky, claims that in October of this year Microsoft will hold the first presentation of this operating system. It will be held on October 27 at the Microsoft Professional Developers Conference (PDC).

Also, but at the Windows Hardware Engineering Conference (WinHEC), the first technical information regarding Windows 7 will be made public. Experts say that Microsoft will not disclose all the details about the system until it reaches a certain progress in development.

The release of Windows 7 is scheduled for January 2010.
