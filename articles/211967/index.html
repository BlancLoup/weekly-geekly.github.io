Nimbus Note for Windows - Note Management Software

Hello! We are pleased to inform you that we have released a version of Nimbus Note for Windows, which will allow you to conduct full-fledged work with notes on your computer, as well as synchronize notes with other devices. We already have applications for browsers, mobile devices and a web part, but we rightly thought that it was impossible to call this type of service full-fledged, without a normal desktop program, because, what would not be said, but working on the desktop with notes is much more convenient, than on any mobile device. Work on the program was carried out for almost a year and its results can be seen under the cat.

Download Nimbus Note - http://nimbus.everhelper.me/download.php





Functional editor

Our editor can have almost all functions that rely on any decent text editor, ranging from bullets and superscripts to superscripts to creating and editing tables.



Import Microsoft Word documents and txt files

Nimbus Note allows you to import Word documents and TXT files and work with them as with a regular note, that is, change text, insert pictures, etc.



High speed

The application for notes should be fast and this is an immutable rule. The program can have a fantastically beautiful interface, but if it hangs while working, then it is unlikely anyone would be interested. We have worked a lot to optimize the program and are rightly proud of the result. The program was tested on accounts, where there were thousands of notes, and the size of the database exceeded gigabytes, but this was not felt at all at speed.

Smart Sync

Why wait until all the notes are downloaded, if only a specific one is needed? That's right, no need. The program has a phased synchronization, when first snippets of notes are downloaded, and only then their bodies with attached files and you can always select the desired note and download it out of turn. And there are situations when, in principle, you do not need to download all the notes, even with pictures and attached files. For such cases, we added a ‚Äúfast‚Äù sync, in which only the snippets of the notes, as well as their bodies, are downloaded, and the pictures and attached files are downloaded on request.

Syntax highlighting

A function (while in beta status), which we think Habr users will be interested in :) I think it‚Äôs not worth explaining its meaning, and so everything is clear on the screen. You just need to select the desired piece of text and select Format as code block. So far not many designs are supported, but over time we will improve.



Task list for each note

We did not work with tasks quite normally, but it was we who generally separated the tasks from the note. For tasks we have a separate panel, where you can create as many tasks as you need.



Search and Search Items

Search is conducted throughout the database, but you can also search in separate folders. There are no tips yet, we hope to push them in the next version. But there are Search Items, which allow you to keep a search history and at any time return to the desired results.
The notes themselves have Search & Replace, which performs the same functions as the good old Notepad.



... and many other features

In addition to the above, there are many other pleasant features: full-fledged work with proxy servers, the ability to select the font for the list of folders and notes (who do not like default Tahoma), creating screenshots, support for hot keys, etc.

This is the first version of the program and far from all the functions that were conceived (and are constantly thinking) implemented, but we hope that you will like everything.
