Linux Mint 9 LXDE Released

Linux Mint 9 LXDE final release released. The news of this is published on the distribution site. Download the distribution here .
The system is based on Lubuntu 10.04 (in fact, being its counterpart), kernel 2.6.32, Openbox 3.4.10, LXSession 0.4.3, and Xorg 1.7.5. Details are described here .

The system is suitable for weak computers. In my opinion, LXDE is similar to XFCE, but somewhat faster and subjectively more comfortable. In general, on a laptop on which licensed Vista moves slowly, this release has a chance to register.
Unfortunately, the release goes without a Windows installer (mint4win), but with the help of the Popular Universal USB Installer there were no problems with creating a bootable USB flash drive (I chose Mint 9). During the installation process and a couple of hours of work, including the installation of all the necessary software, problems are also not noticed. So far, only the inability of the file manager to connect via SSH has upset you, which makes it worse than the gnome Nautilus.
