Kaspersky Security Network

Kaspersky Labs Kaspersky Internet Security 2009 and Kaspersky Anti-Virus 2009 have introduced new functionality in collecting new statistics on the type of threats, their source and developing a neutralization method.

Using Kaspersky Security Network involves sending the following information to Kaspersky Lab:

* A unique identifier assigned to your computer. This identifier describes the hardware parameters of your computer and does not contain any personal information .
* Information about threats detected by application components. The composition of the information depends on the type of threat detected.
* System information: operating system versions, installed service packs, downloadable services and drivers, browser and mail client versions, browser extensions, the version number of the installed Kaspersky Lab application.

The Kaspersky Security Network also collects advanced statistics, which include information about:

- executable files and signed applications downloaded to your computer,
- applications running on your computer.
- sending of statistical information occurs at the end of the application update.

Kaspersky Labs ensures that no personal data is accumulated and sent within the Kaspersky Security Network.

Since July 2008, LK publishes all its monthly announcements about viral activity using this service. By the way, information for September is already available on the website: www.kaspersky.ru/news?id=207732811
