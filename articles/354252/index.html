Vivaldi 1.15 - nice and comfortable



Hello!

Despite the race, which we got in connection with the massive updates of the Chromium code, we still tried to deal not only with catching and correcting regressions, but also adding new functions. Or, at least, the improvement of existing ones. So let's talk about it.

So, what we have to offer today:

  • Window background
  • Bookmarks menu
  • Remembering choices when taking screenshots
  • Access to the address bar from full screen mode

Now the same, but a little more.

Window background


We continue to add features that allow you to make the browser as individual as possible. And today it is the background image of the window. You can experiment with the default picture or choose your own. But remember that too many contrasting details will interfere with web browsing. Bookmarks menu


This function is rather long and persistently requested by users of the old version of Opera. And so it happened - we did it. Of course, you always want to quickly, but here, as they say, not everything depends on us.

There is nothing special to tell here - the function is simple and clear to use. But for pro forma, here is a screenshot:

Remembering choices when taking screenshots


Here you can do without words. After you take a screen shot in Vivaldi, the browser remembers the options you have selected, and even after restarting the browser will be able to offer the same set when trying to take a screen shot. In general, it is quite convenient, especially when you need to do a lot of similar screenshots.

Access to the address bar from full screen mode


And here you can also explain everything without pictures. For those who prefer to work in full-screen mode, it is now possible to open the address bar or panels using shortcut keys. By the way, the feature is also actively requested by users.

We also fixed the sound work when working with HTML5, so if someone had a problem, check how fully this bug is fixed.

That's all for now. You can download the new version on the official website , and see the list of changes here .
