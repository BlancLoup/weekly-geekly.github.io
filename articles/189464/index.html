Real Pixels - Dropbox gallery for designers that do not scale images.

We create applications for iOS and Android and are faced with the problem of viewing designs on Android devices with different resolutions - if you want to check how your screen looks in 100% quality on Android, the image should be precisely cut to the screen resolution that will be viewed . Otherwise, the standard viewer (or Dropbox application) will scale the image to fit the screen and the image will look blurry.

Real Pixels allows you to view images from Dropbox in real size, without scaling. First of all, this app will be useful for Android application designers and web designers who need to evaluate the quality of their work on Android devices ( link to Google Play ).



Key features - viewing graphics at 100% scale on any Android device, changing the background when viewing an image (useful for icons), integration with a Dropbox account for easy file uploading.
