Google code search updated

Google code search updated. Added new features that improve the search and navigation code. Main innovations:
  1. displaying the structure of code written in Java, C, C ++, C #, Python, JavaScript and Pascal;
  2. now you can click the include or import mouse in Java, C, C ++ or Python code and go straight to the imported file;
  3. Two new search operators have been added, class: and function:, which allow applying regular expressions only to the names of classes and functions, respectively.
