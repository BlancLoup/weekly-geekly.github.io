Mouse manager extension

Hi Habr!
I really liked the opportunity in Opera browser to switch between tabs with the mouse wheel, previously holding down the right mouse button. Unfortunately I did not manage to find such a plugin; therefore, as you probably guessed, I decided to write my own.


Everything is simple to disgrace. In the settings, you set the sensitivity of the wheel and attach the action of the plugin to the scrolling wheel. The following actions are now available:
  • new inset
  • new window
  • new incognito window
  • refresh tab
  • previous tab
  • next tab
  • pin / unpin tab
  • close tab
  • top of page
  • down the page
  • Homepage
  • back page
  • page forward
  • source
  • extension page
  • settings page
  • bookmarks page
  • history page
  • download page

Why so many actions themselves? Everything is very simple, two events (wheel up and down) in my opinion are few, therefore I want to add some more events in the near future:
  • mouse gestures: left, right, down and up
  • left click
  • double click of the left mouse button

Install the plugin here

Of the minuses, the plugin does not work in some cases:
- if the cursor is over the flash
- on service pages
- on chrome.google.com

I would like to receive comments and suggestions for improving the plugin. Thank!
