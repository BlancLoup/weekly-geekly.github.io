Livejournal has Jabber IM client embedded

Throughout the summer, Livejournal developers worked on integrating an IM pager based on the Jabber protocol into this system. And today they presented the result of their work : this is the LJ Talk program, which was created in collaboration with the Gizmo Project . Access to the instant messaging server is open for everyone who has an account in LJ, with the same username and password.

After logging in to the Jabber server, the program automatically imports a list of contacts from LJ friends. The function of updating the blog is supported by messages that are sent to a special robot through an IM pager. There is a voice communication function.

As is known, the Jabber protocol provides for work through open servers to which various client programs can connect. In addition to LJ Talk, other famous Jabber networks are Google Talk, Gizmo Project, etc. Accordingly, you can connect to the LJ Talk server using any of these programs, and vice versa.
