Opera Max, Opera Coast for iPhone and more


We have a lot of news for you!

So much that I had to find the most hipster photo for the cover to get attention. We launched two new products: Opera Max for Android and Opera Coast version for iPhone , updated Opera Mobile for Android, released Opera 12.17 security update and launched a new version of Dev.Opera . Let's get everything in order.

Opera Max for Android




We have launched in Russia and the CIS countries a new product Opera Max for Android , which allows you to compress traffic not only in the browser, but for all applications on your device. To do this, a VPN connection is raised at the system level, through which all unencrypted connections go, which saves up to 50% of traffic.

Opera Coast for iPhone and iPad




The Opera Coast browser for iOS, which was sensational last year, now also comes out for the iPhone. We managed to optimize it for phones and not lose the minimalism that users liked so much. As they say, no buttons - no problem.



New Mobile Opera for Android


In this version, most of the problems with drawing the interface have gone, if you have noticed any of them. And:

  • Remade Recommendations, now the latest news can be conveniently scrolled
  • Fixed problems in the password manager that prevented passwords on many sites.
  • Improved interface redrawing when the keyboard appears or the screen rotates.
  • The download sensor will no longer pop up during updates on the same page.
  • Added an option to copy the link in the context menu.
  • Chromium updated to version 34
  • Improved stability and usability

Security Update Opera 12.17 for Windows


In the wake of the recent vulnerability Heartbleed (read more about it in the blog of our security experts ), we released an update 12.17 for the old Opera on Presto. The update was released only for Windows, versions for Mac and Linux were not affected by the vulnerability.

Restart the Dev.Opera site


The site Dev.Opera with articles and documentation on web technologies and our products, was partially based on the My Opera service and we worked all winter to restart it on a new platform - and, as you see, they restarted.



The site now not only works on the most fashionable technologies, like Markdown, Jekyll, Sass and Grunt, but also lies on GitHub and is ready for your pull-ups: translations, typos, new articles (for which, by the way, we pay ). The site is not yet fully prepared and there are not enough articles, but we will continue to develop and supplement it - subscribe to the committees, if interested.
