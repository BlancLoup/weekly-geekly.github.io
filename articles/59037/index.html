As you can see, a certain x.54.152.100 sent 28k packets in the stream.

There is a nuance: for the command to display flows, the ip flow ingress command must be present in the interface configuration
For example:

interface GigabitEthernet0/0
no ip address
ip flow ingress
duplex auto
speed auto
media-type rj45
negotiation auto
