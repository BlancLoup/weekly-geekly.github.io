The Game of Java: Java Conference in Kiev, October 14-15, 2016

Autumn Kiev for two days will become the center of gravity for Java specialists - on October 14-15, JavaDay 2016 Java-conference will be held here, which is organized by the Ukrainian community of developers Java User Group Ukraine .




JavaDay sixth - how it will be


Last year's 5th anniversary conference pleased the participants with 5 tracks with interesting sessions from 40 speakers . This year, the event will repeat the success of JavaDay 2015 , which ranked second among the IT events in Ukraine . The conference especially the event will please those who want to get as many insights as possible.
