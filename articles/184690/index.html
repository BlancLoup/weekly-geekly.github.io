Updated hubs in Russian TechNet

TechNet has updated six major hubs on Windows Server 2012 , Windows 8 , System Center 2012 , Windows Azure , the data platform, and productivity issues. Links to hubs can be seen right on the TechNet main page.

The TechNet portal contains a wealth of technical information on numerous Microsoft products - from documentation to forums and links to blogs. Not always especially for a newbie just to understand this variety. Therefore, we decided to slightly simplify the ‚Äúentrance‚Äù to this diversity by creating so-called hubs in six main products / areas.



Each hub groups information according to a certain principle and simplifies navigation through the remaining sections of the TechNet portal. The first four mentioned hubs have the same structure with three main subsections:
  1. Learn where learning materials are collected, mostly from the Microsoft Virtual Academy portal.
  2. Expand , with information on options and features of the installation.
  3. Use , with technical application scenarios.

The last two hubs have a slightly different content due to their specificity. In the "Data Platform" you will find three main sections: databases, business analytics, databases in the cloud. Thus, we tried to bring together all the technologies and solutions for working with data.

The Hub "Productivity" reflects the office direction and accumulates information on Office 365, Exchange, SharePoint, Lync and Project.

We hope that this option of presenting information will help you navigate easier and start using this or that technology faster. And of course, we will update the contents of the hubs with the release of Windows Server 2012 R2, System Center 2012 R2, Windows 8.1.
