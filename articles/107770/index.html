Results of the first day of the TechEd conference



Microsoft TechEd conference started in Berlin yesterday. This is the annual largest Microsoft event in Europe dedicated to server technologies and enterprise-level software products.

Cloud computing and new initiatives from Microsoft and its partners: Dell, Fujitsu, Hitachi, HP, IBM and NEC, were the main topics of the conference and keynote address. Together with the start of the conference, Microsoft is launching a new company, Cloud Power.

Below are the main announcements of the first day made during the key report.

Hyper-V Cloud Announcement




Microsoft is announcing a new initiative called Hyper-V Cloud , which is a set of recommendations, programs and guidelines to simplify and standardize private clouds.

Hyper-V Cloud will allow companies to build their own cloud solutions based on a unified infrastructure of Windows Server 2008 R2 and hardware partners of Microsoft. The conference announced that the following major companies supported the Hyper-V Cloud: Dell , Fujitsu, Hitachi , HP , IBM and NEC . These companies, which together constitute more than 80% of the market, will offer their customers solutions based on the Hyper-V Cloud Fast Track specification.

Hyper-V Cloud Fast Track

Hyper-V Cloud Fast Track is a predefined, proven configuration for implementing a private cloud for companies that are aware of the usefulness and convenience of cloud computing, but would like to have complete control over hardware for security reasons. Hyper-V Cloud Fast Track unifies the parameters of computing power, data storage, network resources, virtualization and management systems. Any company can build its solution based on the Hyper-V Cloud Fast Track using the various hardware components offered by Microsoft partners.

HP Cloud Foundation for Hyper-V

In addition to the Hyper-V Cloud Fast Track initiative, HP and Microsoft announced the joint development of the HP Cloud Foundation for Hyper-V . The HP Cloud Foundation for Hyper-V is a basic architecture combining the HP BladeSystem Matrix and Microsoft System Center with Windows Server 2008 R2 Hyper-V.

This development is part of the HP Hyper-V Cloud Fast Track initiative.

Hyper-V Cloud Service Provider Program

Microsoft presented information about the distribution of the partnership program Hyper-V Cloud Service Provider Program, according to which more than 70 companies around the world offer infrastructure as a service based on virtualization technologies from Microsoft. Among them are the following companies: Korean Internet Data Center ; Fasthosts (UK, US); Agarik (France); Hostway Corporation (US, UK, Netherlands, Germany, France, Belgium, Romania).

Hyper-V Cloud Accelerate and Hyper-V Cloud Deployment Guides

Microsoft announced a set of tools and guides based on the experience of Microsoft Consulting Services (MCS) for customers who are planning to build their own private clouds. New materials and services will be used by MCS and qualified partners from the Microsoft Partner Network.

Announcement of System Center Virtual Machine Manager Self Service Portal 2.0


In addition to the important announcements of the new cloud initiative, the keynote was announced the release of the final version of System Center Virtual Server Manager Self Service Portal 2.0 - a platform that allows providers to simplify the management of virtual servers for their customers.

Forefront Endpoint Protection 2010 Release Candidate Announcement


In addition, a candidate for the release of Forefront Endpoint Protection 2010 , a client security tool built on the basis of System Center Configuration Manager 2007, was announced . The final version of this product will be released at the end of 2010.
