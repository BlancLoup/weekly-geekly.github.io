Laptops now almost completely allow to refuse the bulky boxes collecting dust and occupying space.

What's inside?


Inside this laptop is a novelty. AMD A6 processor.


Quote:
The AMD A-Series A6-3400M (codename Llano) is a quad-core processor for entry-level laptops. It belongs to the AMD Fusion family of hybrid processors (APUs). On a single chip with four x86 processor cores, the integrated high-performance video card Radeon HD 6520G is placed. The processor runs at 1.4 GHz, however, thanks to the Turbo Core, this frequency can increase to 2.3 GHz.

The memory controller integrated into the A6-3400M supports dual-channel DDR3L-1333 memory and PCIe 2.0 interfaces.


For my part, I would compare this processor with an Intel Core i3 processor of the 2011 family, or even an Intel Core i5, but with the same performance, AMD will be significantly cheaper than quad-core Intel Core i5, or the same price with Intel Core i3 (2 cores).
