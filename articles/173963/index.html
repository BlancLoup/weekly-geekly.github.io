UserScript to improve Google Reader style feedly

A convenient Readly script has appeared in the UserScripts.org catalog, which optimizes the Feedly RSS aggregator interface for fast reading of hundreds and thousands of feeds.

The script stretches the lines across the entire width of the screen and compresses the markup, filling the maximum space.



Readly also selects categories with unread messages, reduces CAPITAL letters in category headings, marks read messages with a gray background, always displays favicons for websites, uses the Arial font, activates Shift + A to indicate all messages read.

The script is tested in Firefox (via Greasemonkey) and in Chrome (via Tampermonkey). To work, you need to choose in Feedly Modern Gray theme and type Titles.
