Evernote is now on the Nokia N900



We have another great example of integration based on the Evernote API. Nokia has released the Evernote Sharing Plug-in app for the N900 smartphones. To add notes do not need to run any program separately. This plug-in, developed by Nokia, adds Evernote to the device‚Äôs built-in functionality and allows users to send camera and photo gallery pictures, audio notes and other content to Evernote.



How the plugin works

Immediately after a quick installation, Evernote is embedded in a number of standard device applications, and users can transfer various types of information to their Evernote from the regular send data menu. You can give a name to the created notes and assign tags, as well as view the records created on the phone or in Evernote clients for other devices, through the N900 browser-optimized web interface www.evernote.com/mobile .



Availability

The Evernote Sharing Plug-in is available for free at the Nokia Ovi Store . Please note that the plugin is designed for the Maemo platform.

Integration

As mentioned above, to create this plugin that works well with the recently introduced Maemo platform, Nokia used the interface for Evernote API developers. And more than a thousand developers now use this interface to integrate Evernote into their solutions: from applications for the iPhone and web services to pre-installed software for various devices. We are also pleased that among them are Russian developers, whose projects we are watching with interest.
