Haiku R1 Alpha 4 released

The Haiku team is pleased to announce that our long-awaited release of Alpha 4 is now available for download. A year and four months have passed since Alpha 3 saw the light and the Haiku project has progressed significantly during this time. The main task of the new alpha release is to provide a stable platform for testing and developing programs. For this, the Haiku distribution kit includes a rich set of utilities for developers.



This release contains many improvements on all fronts, including:

Bug Fixes - More than 1000 bugs have been fixed since Alpha 3 was released.
Debugger - New native debugger. Now KDL generates QR codes
File System - BFS is faster. Improved support for NTFS and Blu-ray discs
Hardware - Improved USB OHCI driver support and CPU detection
Localization - Fixed many translations in various languages. New Layout Switch
Multimedia - 10-band graphic equalizer and VST-loader
Network and Internet - Improved network card drivers. New PCNet driver. Basic IPv6 support
OpenGL Kit - Mesa updated to version 7.8.2 for gcc2 and 8.1.0-devel for gcc4
System - Improved operation of virtual memory settings and the creation of swap files
Video - Most Radeon HD video chips and Intel Extreme chipsets are supported.
Wireless Networks - Supports WPA / WPA2. Corrected wireless card drivers

To delve into the details of improvements and fixes, visit the Release Notes page .

You can get Haiku R1 Alpha 4 by free download (USB, CD, virtual images) or by buying a CD.
