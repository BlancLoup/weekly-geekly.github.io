The world's smallest Linux server

Digi International presented a device called Digi Connect ME 9210, which is a miniature computer running Digi Embedded Linux OS. According to the company, Digi Connect ME 9210 is the world's smallest embedded Linux server. It is argued that the new product will allow OEMs to add a secure Ethernet connection using Linux to devices with a limited volume.


Digi Embedded Linux is based on Linux 2.6.26. In addition, the assembly includes a component Digi Package Manager, which allows you to download software updates directly through the development environment Digi ESP.

The Digi Connect ME 9210 module configuration includes the NS9210 processor (ARM9 core), running at 75 MHz, 8 MB of RAM, 2 or 4 MB of flash memory. The list of peripheral interfaces includes 10/100 Ethernet (10 / 100Base-T), serial port, SPI, I2C, GPIO, CAN, 1-wire and Flexible Interface Modules (FIM). Using FIM, you can adapt the device to specific application conditions.

Digi Connect ME 9210 has a built-in NIST certified AES accelerator for secure network connectivity, 10 times superior in performance pure software solutions. The server supports power-over-Ethernet (PoE), which allows you to create devices that do not need an external power source. A set of tools under the general name Digi Dynamic Power Control allows you to reduce power consumption. The size of the miniature Linux server is 36.7 x 19.05 x 18.67 mm.

The modules and developer kit Digi Connect ME 9210 JumpStart Kits for Linux will be on sale in December at a price of $ 48 and $ 399, respectively (the price of the module is indicated per piece in a batch of 1000 pieces).
