The new version of SkyDrive: convenient file sharing, HTML5-loading, etc.

SkyDrive cloud hosting has finally grown to a normal appearance and at least in terms of functionality is able to compete with Dropbox and Google Apps. At least, in this form it is no longer a shame to recommend it to friends. This morning, the developers rolled out a new version of SkyDrive and reported on the work done.



Firstly, the interface for uploading files to the hosting has been fundamentally reworked. If earlier it was designed for file sharing between Windows Live users, now there is no discrimination. A couple of clicks of the mouse - and you send the link to any email address, you can immediately publish to Facebook. Or just get a URL that you can pass to the addressee (s).



Appeared normal context menus in photo hosting.



Many other interface improvements have been made. For example, now you can change the properties (private / public) not only for the whole folder, but also for individual files in it. After all, before working on documents in the ‚Äúprivate‚Äù folder, you could not just give access to this document to your colleague. It was necessary to first copy the document to another folder.

Another example is to create or rename a folder right now in this window, instantly, simply by clicking on the folder name or by pressing the F2 hot key.





There was a possibility of mass upload, download and move files, before it was only one by one.



And the most important thing is that the browser does not load files and photos in the outdated Silverlight, but through the normal HTML5 File API, which is supported by all modern browsers, including Chrome, Firefox, Safari and IE10. The download is in the background and is very convenient.



Also, there is support for PDF, RAW and other formats.
