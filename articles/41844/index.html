Spytrack

SpyTrack‚Äôs unique service, developed by iTrack ( http://www.itrack.ru ), allows real-time tracking of visitors on your site.

According to the developers, there are no analogues to the Internet service either in Russia or abroad.

With SpyTrack, you can quickly and conveniently get answers to many questions that arise during the development of sites and interfaces:
  1. Do the design elements perform the functions assigned to them?
  2. Do important blocks in the interface attract attention?
  3. Do users experience difficulties with navigation?
  4. How much time does an untrained user need to deal with the interface?
  5. Why an order in an online store causes difficulties for users and many other issues one way or another connected with user interaction with the site or interface
Also, with the help of the service, it is possible to evaluate the effectiveness of banner or contextual advertising - information is stored about which site the user came from, if the transition occurred from a search engine, then keywords are saved.

The system provides an opportunity to visually assess the usability of a website and is an essential element for conducting research on the usability of websites and web interfaces.

Currently SpyTrack is launched in alpha testing mode, the installation of the system on the site is possible by invitation, which can be obtained here: www.itrack.ru/portfolio/websoft/websoft_92.html

View system in action - spytrackdemo.itrack.ru/spytrack
