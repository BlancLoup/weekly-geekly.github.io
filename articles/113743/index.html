Google has released an application for creating Android avatars



Google has released a new Androidify application that allows you to create an avatar for yourself and your friends in the Android style, similar to the user. By default, the user sees in the program the standard avatar of Android - the green man. However, almost everything lends itself to editing - height, size, clothing, hairstyle, accessories, colors.

Thus, the user can create his own virtual avatar in the Android style and directly from the application upload to VKontakte, Facebook, Twitter, send by e-mail, save to a memory card, etc.



The most important thing is that the avatar can be installed on the contacts of friends in the address book, after which it will be displayed on the call.



Now you can distinguish friends visually with an incoming call and without downloading their photos, you just need to create a similar Android man, especially this is quite an exciting activity - you need to deftly select a distinctive feature from a friend, such as a beard or glasses.
