VimpelCom will release Ogo communicators



The operator VimpelCom began working with IXI Mobile , he is going to present its Ogo solution for small and medium-sized corporate subscribers. Ogo is a communicator running the IXI-Connect OS operating system, plus IXI Mobile provides the ability to provide additional mobile services for subscribers. Communicator (there are several variations of configuration) supports Push email technology, allows you to work with ICQ-messengers, RSS-feeds, as well as view graphics, Word, PDF and PowerPoint documents. This device is made in the form of a clamshell with a horizontally oriented display and a QWERTY-keyboard. Which of the models will be presented by the operator has not yet been reported, but there are both very simple and more advanced options. For example, the older model uses a QVGA display with 65K colors, 32 MB RAM and 64 MB ROM, a microSD card slot, Bluetooth 2.0 interface, GSM / GPRS / EDGE (850/900/1800/1900 MHz) and Wi -Fi (802.11 b / g).
According to the companies, VimpelCom will provide Push Email to corporate subscribers, as well as support for ICQ and RSS services. The release dates are not yet known. It is expected that when servicing the contract, the communicators themselves will cost only about $ 80.
