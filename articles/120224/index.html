PHPShop Micro Edition - a tool for effective website promotion

We continue to develop the project of free content management system PHPShop.CMS Free. At this time, we are in a hurry to submit PHPShop Micro Edition - a universal tool for quick editing of site content. The program is designed for small sites where information is rarely updated and time saving is required for changing and installing it. For example, for business cards sites.

Installation and Filling


To install PHPShop Micro Edition, you do not need to run the installer, simply copy the files to FTP.

Filling a site with PHPShop Micro Edition is reduced to simply filling a template with content, bypassing the administrative panel. Visual and HTML editors are invoked with one click for a specific part of the template (news, articles, descriptions).

To create the name.ru/test.html page, you need to create the test.html file and put it in the pageHTML folder. The test.html file should contain content that will be displayed in the overall design of the site.

SEO


Each page has a personalized title , keyword and description . For this, tags in the text of the file h1, desc and key are used.



Differentiation of access rights


The program provides for the separation of user access, which means that the owner of the site, for example, may instruct the secretary to add news, and give the SEO specialist access to the files needed for promotion. By the way, Micro Edition is useful for SEO-specialists who create a network of sites for promoting third-party servants, advertising or reference exchanges.

PHPShop Micro Edition does not require a MySQL database and works on connecting HTML files from the special pageHTML folder, which means it is suitable for free hosting sites. The low weight of the program (800kb in total) is also a significant advantage.

Sharing


PHPShop Micro Edition was created for sites based on PHPShop.CMS Free and, like the content management system itself, is a free product. In addition, owners of PHPShop online stores can use the Catalog module and display products from the online stores of the Enterprise and Pro 1C versions on the pages of their website using an affiliate program .

Free Design Templates


The appearance of the site can be customized by selecting one of the 120 free PHPShop.CMS Free design templates . Open source allows you to set your design or make the necessary changes in functionality.
