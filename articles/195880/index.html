Google has released an HTML5 editor for web design.

Google has put Google Web Designer (beta) into free access: an HTML code editor with a built-in graphic editor for creating ad banners.



The editor is initially focused on responsive design, so that websites look good on screens of any size.

Work is supported in graphical mode and in a text editor code. You can even create 3D graphic effects without typing a single line of CSS3.



Creating banners is one of the key functions of the editor. Templates are supported with all standard AdMob and DoubleClick sizes.



In the "advanced" mode (Advanced mode) it is convenient to make more complex animated banners, with different effects.



The program works under Windows 7 or 8, Max OX 10.7 and higher versions.
