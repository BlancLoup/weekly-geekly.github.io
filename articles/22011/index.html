Jared Spool. Hidden Power of Web 2.0
Web 2.0 is not a thing in itself, but a set of approaches that accelerate development. Elements such as API (Application Programming Interface), RSS, folksonomy and social networks, suddenly gave developers a new way to solve complex problems extremely effectively.

Jacob Nielsen. Web 2.0 can be dangerous ... for your wallet
