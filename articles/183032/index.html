Attention geeks: At the main bottom of DevConf, Intel will introduce Tizen for developers!
Tizen is an open source software platform supported by leading mobile operators, device manufacturers and chipset suppliers.

Intel employees will conduct a Developer Lab (training - 4 hours) to create applications based on web technologies. The Tizen Developer Lab is an in-depth, hands-on demonstration of great tools and APIs available today to developers building applications for the Tizen system. It includes a review of Tizen SDK features (currently available for Mac, Wndows, and Ubuntu), a Web API overview, tips and tricks on porting, and development components.
