Microsoft advises to refrain from installing the update MS14-045

Microsoft recommends not installing the MS14-045 update (KB2982791), which was released this week as part of patch tuesday. After installing this update, the user's computer may be inoperable, in particular, the operation of Windows may occur with BSOD or "blue screen of death", which will not allow the computer to start normally. Users who have installed this update and are faced with a problem, the company offers instructions on how to fix it. Anyway, the update was removed from the update server and the link to it was also removed from the Microsoft website.



Update MS14-045 (Vulnerabilities in Drivers Could Allow Elevation of Privilege) fixes three vulnerabilities such as Elevation of Privelege in system drivers of all supported versions of Windows. It is about fixing the drivers of the Windows subsystem (win32k.sys) and DirectX (Dxgkrnl.sys). Users with an installed update may have permanent BSODs, besides this, Windows may interfere with normal work with fonts (incorrect display of them).
