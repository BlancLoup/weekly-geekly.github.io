Information is the main management tool

Before each there was a situation when you need to track some data in real time, whether it be weather, the euro rate or other information. In this case, would help simple Windows Sidebar, included in the delivery of Windows Vista / 7 for these purposes? The answer is simple - although Windows Sidebar has a lot of gadgets, but sometimes there are not those that could solve the problem, for example, monitoring the state of a computer, or a remote server based on Windows. This review will discuss how to make life easier for yourself and others with the help of a free and at the same time powerful program - Samurize ( http://www.samurize.com ). In itself, the program reminds everyone the beloved Conky - a system monitor common among Linux users.


Samurize application allows using the form designer to create a presentation of information as you like, while consuming a minimum of resources. Creating a panel is reduced to the fact that in the configuration wizard you only need to place the necessary elements, after specifying its type.



In order to display more detailed information, the program has support for plugins and VBS scripts, which in turn greatly increase the functionality of the program. From the currently playing track in Winamp to displaying the temperature of the graphics processor, via plug-ins. On the program's website, in the downloads section, there is a huge amount of all sorts of buns, from skins and icons to ready-made program configuration. If for some reason it turns out that there is no plug-in you need on the site, the program has its own SDK available in Delphi / C ++.

What does this all mean? Want to own information and be aware of everything that happens in your computer, and even in the world - then Samurize certainly can help you with this not an easy task.
