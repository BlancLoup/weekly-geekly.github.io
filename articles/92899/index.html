X-Moto 0.5.3 released

X-Moto is a two-dimensional game in which a player driving a motorcycle collects various items, after which he needs to find a way out of the level. The difficulty of the game lies in the fact that the levels are crammed with obstacles, to overcome which you need skill. X-Moto is very similar to Elastomania .

Version 0.5.3 contains a number of fixes, graphics improvements, checkpoints, a new theme music, a set of maps, etc. have appeared. You can find the full list of changes here

The game is free and available for free download for Windows , Linux and Freebsd platforms ( Mac OS X is on the way).
