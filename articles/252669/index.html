Sync Opera bookmarks between iOS, Android and computer



Today we have a parade of updates: Opera 28 for computers, Opera 28 for Android and Opera Mini 10 for iOS. Something new is on each of the platforms, but all of them are united by the main thing: uniform bookmarks that can be added, deleted, edited and these changes are synchronized on all devices. We talked about synchronization for the first time in October 2014 and then regularly shared updates. And today we can say with confidence: synchronization is ready and ready on all devices.

Enable synchronization is simple: enter Opera login and password in the settings (if you already had a My Opera account or Opera Link, it will work) - or get yourself a login if you haven't had time yet:



What is new in browsers?



Opera 28 for Windows, Mac and Linux
  • Bookmark Sync
  • Updated engine Chromium 41
  • Best integration with OS X Yosemite
  • New embedded themes


Opera 28 for Android
  • Bookmark Sync
  • Updated engine Chromium 41
  • Texture compression (the browser now takes up less video memory)
  • New exit dialog and application icon


Opera Mini 10 for iOS
  • Bookmark Sync
  • Hints previously entered domains in the address bar
  • Many minor fixes and improvements.

If you are a web developer and you wonder what's new in Opera 28 on Android and computers upgrading to Chromium 41, then we talked about this in more detail on Dev.Opera . In short: a lot of tasty from ES6 and not only.
