Media Player and Internet Explorer in Windows - unfair competition?

In connection with the next investigation into the violation of antitrust laws, I would like to discuss whether the presence of Media Player and Internet Explorer in Windows is an unfair (and illegal) method of competition. Let me remind you that the presence of the Media Player in Windows was recognized by the European Commission as illegal; as a result, Microsoft released a special version of its OS without this player and paid a large fine.

What should Microsoft do?


  • Release the OS without player and browser
    Make OS components replaceable by third-party programs.
    Same as previous item + build package manager into OS (like Synaptic for Debian and Ubuntu) to automatically install third-party programs
