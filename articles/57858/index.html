RME HDSPe AIO - new standards in ProAudio

RME brings to your attention HDSPe AIO - the most versatile PC audio interface in the world. This sound card is a universal solution to almost any musical task.



HDSPe AIO is an advanced interface with the latest analog and digital converters (192 kHz). The signal-to-noise ratio is more than 112 dB. For the first time, all inputs and outputs can function simultaneously, including SPDIF (phono) and AES / EBU (XLR). Using an affordable expansion card for HDSPe AIO, additional high-quality analog inputs and outputs can be added.
Of course, the TotalMix multipurpose mixer remained in the new interface, and the sensational SteadyClock synchronization technology, known for its remarkable maximum noise reduction capabilities when working with external signals; HDSPe AIO also supports the optional TCO standard for timecode synchronization (LTC / video). As a result, we get a professional sound card that is unique in its capabilities.

The advanced PCI Express core takes full advantage of its new format, which results in superior multi-channel audio with the lowest latency. With the help of reliable flash-update technology from RME, enhancements, additions and updates to hardware software can be easily installed at any time.

Audio Interface Specifications:

PCI Express interface;
S / N ratio> 112 dB;
1 x Stereo analog input / output (192 kHz);
1 x ADAT digital input / output (192 kHz);
1 x digital SPDIF input / output (192 kHz);
1 x digital AES / EBU input / output (192 kHz);
1 x stereo headphone output (Separate DAC);
1 MIDI in / out with 16 channels of high-speed MIDI;
TotalMix: 760-channel mixer with an internal resolution of 42 bits.

Software included:

DIGICheck: Spectral Analyzer, Professional Level Meter, Vector Audio Scope, etc.
Drivers: Windows 2000 / XP / Vista / 64, Mac OS X Intel.

In the basic configuration, the HDSPe AIO comes with analog RCA / phono cables (headphones: TRS jack). That is why all the analog inputs / outputs are unbalanced. Analog XLR cable is available as an add-on (headphones: Neutrik TRS jack). With this cable you can achieve a balanced analog signal.

HDSPe AIO is a unique SteadyClock sync technology combined with features such as maximum noise reduction or sample rate control.
