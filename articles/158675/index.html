Unity3D 4.0 release

A letter has just arrived in the mail saying that Unity3D 4 has been released.


Unity is a cross-platform tool for developing 2D and 3D games.
Innovations:
  • New animation system Mecanim
  • Real-time shadows for all platforms
  • DirectX 11 support
  • Particle system added collision functional with the world
  • Added two new platforms: Adobe Flash and Linux
  • Cross Platform Dynamic Fonts


Official news with a detailed description of the changes.

List of supported platforms:
  • iOS
  • Android
  • Wii
  • Xbox 360
  • Playstation 3
  • Windows
  • Linux
