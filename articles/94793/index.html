Mail@Mail.Ru opens API



Friends!
We are glad to inform you of great news - the largest mail in RuNet has opened its API. It has become part of the unified API of the Mail.Ru portal, thanks to which independent developers have already gained access to the rich functionality of the social network Moi Mir@Mail.Ru.

What is the new API?

At this stage, two classes of functions are available to developers: the first allows you to work with the received letter, the second - with the created one. Lists of relevant applications will be available to users directly on the pages of reading and writing messages. And now let's deal with all this in order.

one.
The application launched by the user on the reading page accesses the text of the letter, as well as other fields, such as subject, sender's address / name, etc. This gives developers the ability to add additional functionality that can assist the user in working with the received message. For example, it can be an application that allows, without leaving the mail interface, to play videos or view photos, links to which are contained in the letter.

2
In the interface of writing a letter, applications can help the user with its formation - for example, to offer an advanced HTML editor, a designer of cards or a tool for editing photos and creating a slide show.

3
But that is not all! Applications can insert special ‚Äútags‚Äù into letters with an indication of the recommended application for viewing this letter. This feature can be useful to owners of existing services or sites (for example, online stores) who send informational and notification letters to their audience and want to make them more useful. When receiving a letter with such a label, the addressee of the message will see a recommendation to launch a specific application to view it.

And of course - everything is done only with the consent of the user.
