Twitter now automatically shortens links

The long-awaited automatic link shortening feature has appeared in the Twitter web interface. Now you can easily add links of any length to tweets - they will be reduced to 19 characters.

Adding long links now looks like this:

After entering the 13 characters URL, a message appears stating that the link will be shortened.



This link leads to the address of the domain t.co, but will look like a shortened version of the original address.



The t.co Twitter service is available for use only on Twitter itself. Of course, if you like some kind of third-party link shortening service, then you can continue to use it.
