Mono 2.10 released - F # included

Mono 2.10 released. Now Mono includes F #, see here for more details.

Major changes:
  • Google Native Client Support
  • New profiler engine
  • Faster socket stack
  • Improved Parallel Framework
  • Exact stack scans and many SGen performance improvements .
  • MonoTouch / Monodroid Unified Runtime Support
  • Cecil / Light
  • New C # Compiler backend (you can use your own mscorlib)
  • The VB compiler can now compile into both profiles: 2.0 and 4.0.
  • Supports ASP.NET MVC3 , Razor and new WebPages.
  • New database API WebMatrix.Data
  • Mono for OSX enhancements
  • F # and IronRuby
