Video surveillance was organized without difficulty, since Hikvision HiWatch DS-n201 is perfectly made cameras, each such camera can work independently only needs storage, but for centralization you need a server and for a software (CCTV) server. Therefore, delve into the program for video surveillance Xeoma.

Xeoma automatically detects your cameras and starts working immediately after launch. Xeoma supports a large number of modules and brands of USB and IP cameras, Wi-Fi, H.264, MJPEG, MPEG4 and PTZ PTZ cameras and is compatible with all versions of Mac OS and mobile OS.

Xeoma features:
  • Full remote access to the system. Users can control their system from anywhere in the world, adjust settings, view archives, connect / delete cameras, view the view from them in real time via a web browser or mobile devices.
  • Detection of movement in a given sector and activation of customized reactions using a special detector. Protection against false positives allows you to not miss important events.
  • Diverse reactions to user-defined events. The system can generate e-mail notifications with attached photos / videos, upload data to an FTP server, perform automatic unfolding from a minimized state using the Popup window module, start playing a sound file (conditional alarm, siren and alarm notify responsible employees).
  • Support for analog, wireless and wired cameras, IP-and USB-cameras, integrated web-cameras, support for PTZ-camera functions (rotate, tilt, zoom).
  • No need to install and have administrator rights. Initial setup is not required. The solution starts working immediately after downloading and running, with default settings that can be changed later.
  • Uninterrupted work on Windows, Linux and Mac platforms. The program is available in Russian, English, Spanish, German, Dutch and Danish - the list is constantly expanding.
  • Broadcast over the Internet. Users can add a video stream or dynamically updated screenshots to their blog or site, as well as view MJPEG-video / JPEG-photo in a browser using a mobile device.
    Modes of operation Xeoma


In the free Xeoma mode, you can use 2 sources of 3 modules each, and in Trial mode you can use an unlimited number of sources and modules, but the settings will be reset every 4 hours and when the program is closed, and the Archive has a limit of 1 hour.

The Xeoma video surveillance software is based on the Client-Server architecture and consists of 2 parts - the Xeoma Server and the Xeoma Client.
The server processes cameras and other signal sources and generally does all the work, while the Client is used to view and communicate with the server.

Licenses are purchased only on the server side. No matter how many client parts will be connected to the server, only a server license is needed. Installation and launch of the Xeoma program can work immediately after the download and does not require installation.

When you first start Xeoma scans the network in search of available cameras. All found cameras will be added to the Main window, with the exception of cameras that require authorization - such cameras will be listed in the list in the "+" menu in the bottom panel.

Also, the creators of Xeoma provide service "Repeater"

This is a service that controls Xeoma, even if the computer is not accessible from the Internet, i.e. is located on a local network, has a local IP address or is connected via the mobile Internet. With it, you can access your cameras and the archive, you can publish any of your files on the network.

The Xeoma web server allows you to view cameras in real time by connecting this module to the camera, you can view the cameras remotely via a browser (provided you have a real static IP address) from any device.
