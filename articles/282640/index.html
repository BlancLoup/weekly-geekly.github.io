Xamarin SDK in Open Source and more at Xamarin Evolve 2016



An incredible number of events happened this year and undoubtedly the addition of Xamarin to Microsoft is one of the brightest. It seemed that after the announcements at the Build 2016 conference , there was absolutely nothing to surprise developers of cross-platform applications and fans of the Xamarin framework, however, today at the Evolve 2016 conference, the Xamarin team succeeded.

Xamarin SDK, as promised, has officially become an open source project today, and everyone has the opportunity to make their own efforts to develop this technology. Open bindings API for iOS, Android and Mac, command line tools, Xamarin.Forms, etc. Details by reference: open.xamarin.com

At Evolve 2016, Visual Studio becomes an even more complete mobile application development environment with a demonstration of a remote iOS Simulator that allows you to interact with your iOS applications in Visual Studio. Simulator, not without the participation of Mac, runs on a machine with Windows, supports the capabilities of touch screen and multi-touch. The iOS USB remoting tool was also announced, which allows you to deploy and debug applications from Visual Studio to an iPad or iPhone connected to your Windows machine. Details and instructions on the link.

The Test Recorder plugin for Visual Studio will allow you to generate test scripts in Visual Studio. Test Recorder will automatically generate scripts from a device or simulator that can be run on thousands of devices in the Xamarin Test Cloud.

New features Xamarin Studio for Mac and Xamarin.Forms - Data Pages and Themes will allow with a few lines of code to connect the application to the data source and create a nice user interface, respectively. A new tool Forms Previewer in real time displays the creation of a UI with Xamarin.Forms, written in XAML. To access these features, you need to install a preview version of Xamarin.Forms for Xamarin Studio on Mac.



Finally, the solution demonstrated at the opening of Evolve 2016 clearly confirmed that the Microsoft platform today is a complete solution for creating and organizing lifecycle management processes for mobile applications , thanks to the implemented integration between Visual Studio Team Services , HockeyApp and Xamarin Test Cloud .
