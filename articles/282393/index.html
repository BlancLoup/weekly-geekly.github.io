GitLab in Russia

The company GitLab, the developer of one of the most popular web-based GIT repository management and software development management systems, has entered into a partnership agreement with Softmart.



GitLab is available to users in 2 versions: Community Edition (CE), Enterprise Edition (EE). Each edition can be used either locally on its network (self-hosted), or on the gitlab.com hosting. And on a free hosting gillab.com EE edition is deployed, but without administration features. An important feature of GitLab hosting is the lack of restrictions on the number of connected users.

The CE edition is free and therefore the most popular - it allows you to integrate GitLab with build systems, organize the code review process, use the built-in defect and task management module.

Revision for serious EE professionals is primarily interested in its additional functionality, which is not available in other editions. For example, GIT HOOKS, allows you to analyze the version description, find the macro commands provided in them and open them. This is how integration with Jira is implemented, if someone needs to have feedback in an external task management system.

Learn more about the versions on the GitLab website .
