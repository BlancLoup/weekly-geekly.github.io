EasyPrint and idle printers

Unfortunately, not all printers print via redirection in windows server 2008 easy print. In order for them to print they will have to install their driver on the server. Then you need to run the gpedit.msc command. In the opened snap select:
Computer configurations->
Windows components->
Remote Desktop Services- >
Remote Desktop Session Host->
Printer redirection

Here izment property use remote easy printet driver first to the state Disable.

Then the server will first try to print using the printer driver and only if it does not find it will contact the easy print driver.
