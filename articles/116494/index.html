WingBee - mobile ip telephony for Android


The popularity of the Android operating system is gaining momentum. E1tele.com also did not stand aside from this business. We present you the application - Wingbee . This is an application for phones that allows you to make uncomfortable VoIP-telephony simple and understandable.

When installed, the application exports the list of all contacts of the address book and SIM card to WingBee contacts, after which the subscriber can dial up to each of them freely, add and delete contacts, view calls made, use the short number entry system. E1tele.com allows you to create a full- featured notebook for short numbers containing up to 1000 contacts.

WingBee does not change the phone settings. This is a "smart program", with which you can choose the cheapest way to call. With the help of WingBee, you can choose how to make a call: by the callback technology (callback) or by the callin technology (direct call). Here are our tariffs for callback and callin in rubles.

Call ordering is possible in several ways: through a data channel (GPRS, Wi-Fi) or via a mobile network (SMS, call-in number). Installing the application does not require possession of special skills, but just in case, you can read instructions on how to do this. You can download in the Android Market on Google
Android
