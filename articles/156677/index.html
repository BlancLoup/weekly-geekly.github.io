Chrome Remote Desktop out of beta



About a year ago, Google announced an interesting Chrome Remote Desktop extension , which was launched in beta version (actually, like most Google services). This extension allows you to remotely access another computer through the Chrome browser, regardless of the operating system (Windows, Linux, Mac, Chromebook). Of course, this is an extension for the Chrome browser, and not a universal application. Now the developers have finalized the extension, making it more functional and withdrawing from beta.

Among other new features added by developers, it is worth noting audio in real time (for Windows). This feature can be useful if, for example, you prefer to listen to music from your home collection, from your home PC or laptop. Also added a tool that allows you to use copy / paste with the remote on the local machines and back.

In his blog, the developers did not fail to mention the new Samsung Chromebook. It is indicated that Chrome Remote Desktop allows you to turn this device into a full-fledged workstation, while working on a fairly portable device.

Now developers are actively developing Chrome Remote Desktop, and, according to them, we can soon expect new features for this extension.

Yes, and who has not yet installed this extension, you can try it out at work: Chrome Remote Desktop .
