Skype for Business - Cloud Connector Edition (CCE) released



A few days ago, Microsoft released Cloud Connector Edition (CCE) for Skype for Business Online, which is a hybrid solution for integrating corporate PBXs with Office 365. CCE is a set of components that allows organizations to connect to the PSTN, organize a SIP trunk from your Office 365 (Skype for Business Online). This solution is useful for countries that do not have the ability to use telephone service directly in Office 365 as part of an E5 corporate telephony plan, currently only available in the United States.


So, what is CCE and what are the features of its licensing?

What is Skype for Business CCE?


Skype for Business Cloud Connector Edition (CCE) is a hybrid solution that consists of many virtual machines that implement local communication with Cloud PBX.
By deploying the minimal Skype Business Server topology in a virtual environment (whether cloud or local), users of the organization can send / receive calls from fixed and mobile phones using any existing voice infrastructure, for example, through an enterprise PBX or a SIP trunk VOIP provider.
CCE includes a package of four virtual machines that must be deployed and configured using the installation wizard. List of virtual machines:

  1. Domain controller
  2. Central Management Server (CMS)
  3. Mediation server
  4. Edge server

However, there is no dependence on the existing infrastructure of AD, at the deployment stage, it creates its own forest. CCE virtual machines are recommended to be placed in the DMZ. Companies that need high availability can deploy 2 CCE packages (2 x 4 virtual machines, i.e. 8 virtual machines in total).

Licensing


You do not need to purchase licenses for virtual servers included in Cloud Connector Edition (CCE). Licenses are required for Office 365 users using a hybrid configuration; they require the purchase of an E5 corporate plan.
