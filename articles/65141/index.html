ThrowCatch.me


ThrowCatch.me is not only a website of answers and questions for programmers, but also a knowledge base similar to wikipedia and formed by the community. The concept of the site incorporated a combination of several popular Internet systems: Digg, wikipedia, forums and blogs.
The goal of the project is to create a place where not only one could ask a question, but also to assemble a unique knowledge base in Russian.
I will write about what is, and then I will write why.

And that's what came out of it


Briefly about the most important features.


  • Personal tape (and ignored tags)
  • Tag Filtering
  • Easy interface
  • Easy registration
  • Commenting questions and answers
  • Gravatar support
  • Custom icons
  • Question and Answer History
  • Filters and sorting questions, tags, users
  • Search users and tags
  • Community self regulation
  • Best answer
  • Custom business cards
  • API interface
  • Response Alerts
  • Multifunctional profile


Why, if there is an itquest and askdev?


Look with your own eyes and draw your own conclusions about the pros and cons.
The site began to be created in early May due to the lack of decent sites of this kind in RuNet. Then I already knew about itquest and askdev. Their main disadvantage is too extensive subject and insufficient functionality.

Used technologies


The engine is written in python using the django framework . On the client side, prototypejs is used. For version control, use GIT. Deployment of new versions occurs immediately after the git push script using nginx + apache + wsgi, so the server is always the freshest version (after testing) and all the shortcomings are corrected very quickly.
