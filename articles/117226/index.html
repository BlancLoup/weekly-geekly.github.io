We invite you to test virtual machines based on KVM


Dear boring people from the Internet, colleagues! I am glad to offer you for beta testing virtual servers (VDS) based on KVM !


Now I will not dwell on the differences between KVM and Xen, KVM and VMware, it will all be later. I have written about a dozen articles on how to configure KVM, how to modify images of virtual machines and much more - I plan to publish all this in the near future.

From the tasty things that we have now implemented:
  • In the presence of resources, all the power of one core is allocated.
  • VNC access
  • In an individual order it is possible to install any operating system from the provided ISO image.

From what will be in the near and not very perspective:
  • FreeBSD, there are certain performance problems with it, but it won't stop the real red-eyed!
  • Windows, well, where do without it.
  • Access to server console via SSH.
  • VDS images with pre-installed panels.

I note that virtualki work in HVM, and system administrators can build the kernel as they need.

It is assumed that 1 month is given for testing. According to the results of testing, we very much hope to receive feedback, recommendations, features and other feedback from you. However, unfortunately, the number of places on the test server is limited and, as they say, who had time, that and sneakers.

Well, the main thing! Sign up for the test - here . The decision on the availability of testing is made within this day.
