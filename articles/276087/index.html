Oracle stops Java plugin support for web browsers

Oracle has announced an imminent change that can be perceived with enthusiasm by many users. It is about stopping the release of the Java plug-in for web browsers, starting with the upcoming 9th version of this software. The plugin will be removed from software packages such as Oracle JDK and JRE, as well as Java SE. In turn, authors of applets and plug-in content are encouraged to switch to Java Web Start technology .



It should be noted that such popular web browsers as Google Chrome and Microsoft Edge have already abandoned the use of the NPAPI interface, which is necessary for the operation of the Java player. The same step is provided for Mozilla Firefox this year. The plugin for playing Java content was repeatedly criticized by security researchers for the presence of a large number of vulnerabilities that were previously used by attackers to remotely install malware.

It is up to you to take a look at the plugin. Web Start technology. Oracle plans to deprecate the Java browser plugin in the JDK.

The company has published a special document covering the mechanism for moving away from the development of applets www.oracle.com/technetwork/java/javase/migratingfromapplets-2872444.pdf .
