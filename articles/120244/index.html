Firefox 6 integrates a Workspace extension for previewing scripts

After the fourth version, Firefox has switched to a new accelerated release cycle, so that new versions of the browser will now be released much faster than before. The fifth Firefox has already reached the beta stage, and in May the alpha version of Firefox 6 appeared on the Gecko 6.0 engine. The final release is expected in the III quarter. 2011, a preliminary list of innovations that are important for web developers has already been compiled.



Interesting in Firefox 6 will be the new Scratchpad feature for conveniently testing Javascript fragments right in the browser ( Workspace extension).

It will be enough to press F4 in the browser - and the built-in editor opens.



There you can insert a ready-made script or experiment yourself. Then press Ctrl-E - and the script is launched in a separate tab. It should be very convenient, especially for learning Javascript.

In addition, a new web console will appear in Firefox 6 and the Web Developer menu will be added to access these tools (as seen in the first screenshot).

The sixth version of the browser is expected to provide support for the WebSockets protocol up to version 07, to implement support for new CSS, DOM, and other functions. The list is constantly updated on this page .
