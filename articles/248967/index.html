HP 3PAR StoreServ Webinar


HP Corporate Solutions Department invites you to a new series of webinars , starting on February 5th.
The first webinar of the series is dedicated to HP 3PAR StoreServ. At the seminar, an overview of disk arrays will be presented, a stand for Peer Motion federated data storage will be demonstrated, as well as HP 3PAR Peer Persistence Fault Tolerant Solution technology.

The 3PAR Peer Persistence technology allows building fault-tolerant solutions in the Windows or WMware environment, where in case of failure of one of the arrays, switching between sites is fully automatic and transparent for applications. The volume is replicated between arrays and fault-tolerant paths are presented to it, presented to the host. Disk arrays located at different sites are interconnected by replication implemented using 3PAR Remote Copy.

3PAR Peer Persistence provides servers with uniform access to two storage systems at data center sites, while each logical drive (LUN) of each storage system 3Par has a synchronous copy on another site.

This copy is presented to the VMware or Windows environment passively to the main LUN, since it has the same WWN identifier (unique device identifier in SAN networks). In the event of a failure of the main LUN, its backup remains visible to the hosts.

At the same time, switching to a backup, for specific volumes, array occurs without the use of third-party mechanisms, which makes it possible to transfer virtual machines (applications) to another site without stopping them and at the same time does not complicate infrastructure management.



3PAR Peer Motion technology is designed for efficient and easy data migration between different storage systems, without service downtime.
