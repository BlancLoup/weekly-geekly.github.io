Familiarity with the fourth version. Part 2

Today we will talk about the new version of the ESET Remote Administrator (ERA) application. Since the application is an integral part of our corporate solutions ( ESET NOD32 Business Edition and ESET NOD32 Smart Security Business Edition ), we decided that it would be appropriate to talk about ERA in line with the fourth generation of products.

ESET Remote Administrator (ERA) is a scalable tool for centralized deployment and administration of enterprise ESET solutions in network environments of varying complexity. Using ESET Remote Administrator, you can remotely install and uninstall ESET software products, monitor the operation of anti-virus software, create ‚Äúmirrors‚Äù of ESET NOD32 product updates inside the network, which leads to a significant reduction in external Internet traffic.

With the help of the ESET Remote Administrator component, you can centrally deploy, monitor, and administer a security software solution within a corporate network or from anywhere in the world. Of particular importance is this factor for companies distributed geographically, as well as for system administrators, preferring the remote form of work or being in traveling.

Working with the new version of the ESET Remote Administrator application has become more convenient. Added features: ‚Äúnotification manager‚Äù and ‚Äúlicense manager‚Äù, synchronization with Active Directory, support for popular databases.

During installation, the administrator can select the database that is more convenient for him. Attention! After installation, changing the type of database will not work.



Using the ‚Äúlicense manager‚Äù, you can manage all user license files within one ERA server, automatically launch a new license via the ERA console after the previous license expires.



The list of events "notification manager" is configured arbitrarily, depending on specific tasks. For example, you can receive a notification about the expiration of the license of ESET products, about the task performance, about an error on the server, etc. The administrator can either create the notification rules himself or use the already prepared ones.



Thanks to the synchronization of the "notification manager" with Active Directory, there is no need to create user groups manually. Notifications can be displayed in the logs, come to an email inbox or arrive as alerts.



Improved user settings management system. Endpoint configuration management policies have become more flexible, data replication and the hierarchical structure of ERA servers are supported. Implemented control of client configuration changes. Each time a client logs in, the configuration specified by the administrator is automatically applied.



The SysInspector module, a special utility for diagnosing systems on the Windows NT family of platforms, is integrated into ESET Remote Administrator, which allows the administrator to receive real-time reports on the system status: running processes, registry keys, files, services, drivers, network connections, possible malfunctions.



Now it is possible to work with the ESET Remote Administrator console in read-only mode, which allows users with limited rights to give ERA access (for example, to monitor the state of the system without changing the settings). The functions to start scanning, applying configurations, etc., from read-only mode are not available.
