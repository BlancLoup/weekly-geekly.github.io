Chrome beats all the speed of processing scripts

Google has laid out a set of tests that allows you to evaluate the performance of JavaScript in various browsers. Although the tests are written mostly by Google experts themselves, the source code is available and they seem to show the real picture. Judging by the results, none of the existing browsers can compare with Google Chrome in terms of JavaScript processing speed.

Independently compare your browser with Chrome can be on this page (the result is shown as a percentage of the performance of Chrome). The table shows the results of five tests, each of which tested a specialized set of scripts.
