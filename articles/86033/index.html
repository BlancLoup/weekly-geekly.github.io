Project Gustav: Immersive Digital Painting

The Gustav project is a prototype of a realistic drawing system that allows artists to work with the screen like a real painting. The computational power of modern GPUs helps to achieve interactivity and realism, using the capabilities of multi-touch and tablet input to the fullest, as well as the latest natural algorithms for media modeling and imitation of a brush. The Gustav project is a great example of how Microsoft is working hard to research the new and enchanting technologies.

The video shows the previous version of the program interface, but everything else works the same.



about the project


In fact, drawing on a computer is completely different and has nothing to do with painting with the same brush. Painting is a rather complicated phenomenon, a 3D brush consisting of thousands of individual deformed hairs interacting with paint on canvas creating complex strokes. Not so long ago, at home, there was not enough computing power to simulate realistic drawing. The Gustav project is designed to increase the capabilities of a PC, to provide a natural user interface for drawing to professionals and people who have drawing as a hobby.
