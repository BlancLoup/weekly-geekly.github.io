IE7 overtakes other browsers in RIA tests
I would like to note that this article may upset users of browsers. However, if you trust the results of the new tool, when it comes to web applications (from the English Rich Internet Applications - RIAs), IE7 shows better results than Firefox and Opera in Windows XP.

Moreover, IE7 in XP is ahead of Safari in MacOS X 10.5. In tests created by developer Sean Christmann, the latest browsers were used (that is, neither IE8, FF3, nor Opera 9.5 took part in the test).

Christmann developed a test application, later called GUIMark, which tracks the number of frames per second (from the English. Frames Per Second - FPS) when rendered by RIA technologies browsers. The list of these technologies includes: DHTML, Flex, Java, Silverlight 1 and Silverlight 2. GUIMark evaluates common types of web application UI elements (vector redraw, alpha transparenc, text reflow, bitmap motio, 9 scale slicing).
