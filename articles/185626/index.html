Prototype: Hadoop: The Definitive Guide
Publisher: O'Reilly Media; Third Edition Edition (May 26, 2012)

Apache Hadoop is an open source framework that implements the computational paradigm known as MapReduce, which enabled Google to build its empire. This book will show you how to use the full power of Hadoop to create reliable, scalable, distributed systems and handle giant datasets. Programmers will find analysis techniques here, administrators will learn how to install and run Hadoop clusters.
