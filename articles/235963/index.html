The digest of interesting news and materials from the world of PHP No. 47 (August 24 - September 7, 2014)



We offer to your attention another selection with links to news and materials.

Enjoy reading!


News and Releases


  • PHP 5.6.0 - The long-awaited release of a new version of the interpreter. There are many new features on board: scalar expressions in constants , functions with variable number of arguments , exponentiation operator ** , import of functions and constants from namespaces , built-in phpdbg debugger , operator overload for GMP objects and for other extensions . By tradition, there is a guide to the transition to the new version . PHP 5.6 screencast in 10 minutes .
    Core-developer Stas Malyshev summed up the release of 5.4 and tried to predict the future 5.6 .
  • WordPress 4.0 release Benny - The fourth version of the most popular CMS has become available. The release includes many improvements and several new features, although nothing revolutionary.
  • HHVM 3.3 - First Release with Long Term Support (LTS)
  • Engine Yard sponsors Composer - A $ 15k grant has been announced for a year to support Composer as represented by one of the authors, Nils Adermann . Remembering the statement by another author Composer Jordi Boggiano about the development of a commercial Toran Proxy to support Composer, there is reason to believe that Composer is waiting for a significant development.


Php


  • RFC: Implicit isset () in the ternary operator - Instead of isset($_GET['mykey']) ? $_GET['mykey'] : "" isset($_GET['mykey']) ? $_GET['mykey'] : "" you can just $_GET['mykey'] ?: "" .
  • RFC: Abstract syntax tree - The proposal was adopted unanimously: 47 - for and none against!
  • RFC: Closure :: call - It is also accepted to add a call method for anonymous functions with the possibility of passing the object this .


Instruments


  • Optional - A library designed to reduce checks on null in code. Detailed post describing the motivation of the author .
  • Checker - Set for self-testing of the class and its individual methods.
  • CodeLobster PHP Edition - Alternative IDE for PHP development.
  • Broadway - A tool for implementing the principles of CQRS and Event Sourcing in PHP. Post in support .
  • Cuzzle - The library allows you to save Guzzle queries as cURL commands for further debugging or logging.
  • Versionscan - A tool for checking the current PHP installation for known vulnerabilities.
  • Toggle - The library allows you to organize the principle of disconnected features in a PHP application. Post in support .
  • Amp - Asynchronous multithreading in PHP.
  • php-wise - Symfony Config based configuration library.
  • PHP Reflect - Library allows you to explore classes, interfaces, functions, and more. Unlike the standard Reflection API, it does not directly require loading the code, since it is parsed using a PHP lexer.
  • CLImate - The best PHP helper in CLI. Color and formatted command line output.
  • UnifiedArchive - Library for unified access to archives of various formats. Also in the form of a command line utility - Archive .
  • FOSHttpCache - The tool allows you to integrate a PHP application with a cache proxy a la Varnish.
