Rostelecom is introducing a new type of Internet traffic filtering. Not DPI

In the third quarter of 2014, Rostelecom will finally introduce a page-by-page filtering system for Internet traffic by URL.

Until now, this largest and mainline telecom operator restricted access to Internet resources by IP address. And this often happened not only for the users of this provider in Russia, but also for the clients of those telecom operators in the states adjacent to Russia (such as Armenia, Kazakhstan, Uzbekistan, Moldova, Belarus, etc.) who received Internet traffic from Russia on the highway.



As reported by "Vedomosti":
In the test mode, a new type of filtering can be earned at the end of May and this is not about the Deep Packet Inspection (DPI) system - the operator will not install it, does not see ways to monetize.

The representative of Rostelecom, Valery Kostarev, confirmed: DPI is not economically viable, and as for the new filtering system, Kostarev hopes that it will be able to start blocking resources by URL in the third quarter of 2014.

DPI can not only filter sites, but also detect and block viruses, as well as filter out information that does not meet specified criteria. Providers use DPI not so much for filtering as for blocking applications like BitTorrent. The solution that Rostelecom is preparing to establish is focused only on filtering prohibited resources, knows the analyst of the Russian Association of Electronic Communications Irina Levova. It works like this: if the address of the page that the user enters is blacklisted, then his request is sent to the router. This device recognizes the URL to be blocked and denies access to it. And access to the allowed pages on the same IP address remains open.

So what's the new kind of filtering online space?
