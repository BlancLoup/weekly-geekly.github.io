Yandex.Maps are now able to build routes by public transport in Kiev

Yandex.Maps have learned to build driving directions in Kiev by public transport.

Now the service helps to plan trips not only for drivers, but also for pedestrians.



The service will tell you what will get faster now. For example, in the morning it can be one route, and in the evening it can be another. More than half of Kyiv residents use public transport - 52%, so the new Internet service has a new future. So far, the new Internet service from Yandex.Maps has been launched in beta and only for Kiev, but in the future other large cities of Ukraine will be covered.
