PHPShop Editor supports multisite support.

The EasyControl toolkit has been updated: the Template Editor Editor already allows you to create several site directories, and synchronize each one.

A multisite feature has appeared in the Template Editor Editor: now, the user can save the required number of connections and work with each of them.

Instruction


To get started, from the list of installed programs, you need to run the Editor utility or immediately the Synchronization Wizard - Synchronizer Conf.



Then, from the Editor, click the button to call the Sync Wizard.



Create the necessary connections to the sites you work with. Now you can choose from the list with which site you need to synchronize.



Synchronization allows you to select the necessary files for updating: only images, code files, templates, or to perform a full synchronization.



Synchronization wizard also available from IDE





Anyone can:
Download PHPShop EasyControl and test the Sync Wizard.
