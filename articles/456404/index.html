Looper - Plugin for Sketch

Looper generates patterns . With the help of Looper, I make backgrounds. The pattern is generated by repeated duplication of the workpiece with changes from copy to copy. The blank can be any shape or combination thereof.



Look at the short video how it works with the plugin.

Looper is a plugin for Sketch . If you do not have a paid version, do not worry - Sketch is available for free for 30 days without any functionality restrictions. But Looper is completely free :



Installation


Download the plugin and open the installation file. Sketch will do everything by itself. Create a new project, Looper will appear in the Plugins menu. The shortcut key CMD + L runs the plugin. If you are in the XVIII century, then you can run from the panel.

Instruments


We make the workpiece. Create a new Artboard , add a background. I will set a gradient with colors # 5581F1 - # 1153FC:



Press the O button and draw a small circle. For users of the XVIII century button is available on the panel. If you are from XXI - accustom to shortcuts. Remove the fill and set the white color of the stroke. Change the shape: double-click on the shape to switch to edit mode. Changes in shape will make the result more effective. My preparation looks like this:



Call with CMD + L plugin. Already remember the combination? A window will open with settings for generating a pattern:



Consider the options:

  • Duplication count is responsible for the number of duplicate items. Good result in the range of 10-30 pieces.
  • Scale increases the size of the last elements. You can scale by X and Y or on each axis separately. You can set the scale in pixels or percent.
  • Move moves objects vertically, horizontally, diagonally and randomly. The distance is set in pixels.
  • Form a grid creates a grid. Set indents and layout for elements. I rarely use this tool.
  • Rotate manually. Auto mode works weird. If you move in a direction, use linear mode. If you want symmetry from the center - use a sinusoidal.
  • Cool tool Update opacity - changes the transparency. I expose a linear from 0 to 100 to create volume. Sometimes a random mode can come in handy.


Customization


Working with Looper is to combine settings. Draw an object without a fill, with a stroke and no sharp corners. I use the oval, changing the shape each time:



Select the workpiece and call the Looper control panel. For Duplication count install 25 pcs. Scale on two axes with an increase of 5px. For Move - horizontal mode and an offset of 10 pixels. To add life, install a linear Rotate with an angle of 7 (range from 1 to 359). For the volume, I'll set the transparency from 0 to 100.

The panel will look like this:


All changes are immediately displayed on Artboard . To see the intermediate result, the settings window will have to be moved. Apply the settings:



For the workpiece, you can use a gradient. Add it using the previous scheme:



Looper works with any objects . You can use several shapes:



The more complex the form, the more time is needed to set up. Avoid clusters of copies and minced flowers . If the result does not look very, simplify the workpiece.
