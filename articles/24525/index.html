Adobe launches a revolutionary project.

Adobe today announced the Open Screen Project. The project aims to develop Internet technologies created by Adobe. As part of the project, Adobe opens up access to Flash technology:
* removes all restrictions on the use of specifications for formats SWF, FLV, F4V;
* opens the protocol specifications Flash Cast and AMF;
* removes license fees, which will make the next versions of the Flash-player and AIR for devices free;
* publishes API for porting Flash Player to devices.

The project is supported by industry giants: ARM, Chunghwa Telecom, Cisco, Intel, LG Electronics Inc., Marvell, Motorola, Nokia, NTT DoCoMo, Qualcomm, Samsung Electronics Co., Sony Ericsson, Toshiba and Verizon Wireless. Content monsters also join the project - MTV, BBC, NBC.
