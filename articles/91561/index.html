Wall Street Corporations will report via a python script

Instead of publishing mandatory reporting on paper or PDF, in the future all public companies whose shares are traded on open exchanges will submit all corporate information through a special Python script connected to the EDGAR system. Thus, corporate reporting will become more transparent, and important figures will be transferred to automatic trading systems in real time.

More details about the SEC proposals are in this document (667 pages, the description of the script on pages 205 and 210). This document contains a number of SEC proposals for exchange trading reform to make it more open and honest.
