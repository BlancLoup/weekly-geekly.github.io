Rumors of the futility of ReactOS were greatly exaggerated

Or a comprehensive .NET alternative



Officially announced the launch of the project Wine-Mono , within which a package with the implementation of the .NET runtime and class libraries for Wine was formed on the basis of Mono and other components. Wine-Mono is designed only to work in conjunction with the implementation of the .NET Runtime Execution Engine embedded in Wine (mscoree.dll). On the practical side, Wine-Mono allows you to organize the launch of .Net-applications without installing the official .NET framework from Microsoft in the Wine environment.

I convey to everyone who has buried ReactOS due to the lack of support for the relevant libraries, heartfelt greetings. Soon your argument will cease to be an argument.
