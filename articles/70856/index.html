DIY or Insolent spammers

A few days ago, several friends hacked into VKontakte accounts and sent a picture from their faces, where it was written about a supposedly new hole on VKontakte and offered to visit the site pro-vk.ru . Curiosity got the better of it and I decided to see what is there :) This address redirects us to megavk.blogspot.com . There is only one single post in which the user himself (!) Is asked to open the hosts file and enter there:

210.51.166.249 vkontakte.ru
210.51.166.249 www.vkontakte.ru

True, there is something wrong with them, because according to that IP address, the inscription Apache is functioning normally .

This is just funny :)

PS I already reported where to go.

UPD: Smart people suggested that if you make an http request that contains the address of vkontakte.ru to this address, then everything works.
