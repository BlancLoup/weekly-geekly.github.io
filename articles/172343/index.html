Keepers: Review of Credit Card Storage Applications

Despite the fact that mobile acquiring terminals such as Square or 2can allow you to accept payment cards simply by signing on the screen of a smartphone or tablet, our life is still literally crammed with all sorts of account numbers, passwords, PIN codes and other confidential information that we need remember and track. Passwords for frequently visited websites, credit cards, bank accounts, passwords for social networks or specialized groups, driver's license number, passport and insurance are just a small number of examples of information that sometimes we need.



How often do you hear that ‚Äúfor security reasons‚Äù you should not use the same password or pin code for all accounts? And also, that for greater security, the password should contain a combination of letters and numbers. Also, the password, ideally, should consist of more than eight characters. Different register. And, to finally secure yourself, use at least one special character. Are they serious? .. Multiply the number of options by the number of your accounts and you quickly realize that it is not possible to keep in memory all these multi-character combinations. Writing passwords on a piece of paper to better remember or recall on occasion, taking a hint from under the keyboard, is altogether counterproductive.
For the average man in the street to remember this amount of information can be a doable task, but it is very boring. Fortunately, there are applications for the iPhone and iPad, created to help people safely store and organize access to all sources of sensitive data so that the information is available then when it is necessary, but it is reliably protected and encrypted from unauthorized access.

Applications for secure storage of information on iOS devices


The AppStore is full of security related applications. At first glance it may even seem that they are generally the same, as they implement a simple procedure. But not all of them are equally useful and each offers a unique set of features that make usage more specialized. In the end, we need to know what to pay for.

Top apps


1Password, 579 rubles


If you have ever forgotten your password, then you know how unpleasant and annoying the recovery procedure is (especially when it comes to the PIN code to a bank card). With the application 1Password, you will need to remember only one password. This password will open access to the application in which all other passwords will be stored. This application can also generate a strong, unique password for each site. It has a built-in browser and saves other types of personal data, such as credit card, passport and bank account numbers. The fact that 1Password is a universal application and is synchronized with a computer makes it one of the most loved by users.


Wallet, 329 rubles.


Wallet is an application that stores and encrypts your passwords, credit card numbers, serial numbers, and so on. All information is synchronized securely via Dropbox or another WebDev server. For the price of just $ 10, the Wallet is an excellent replacement for 1Pwordword at a lower price. Even the Mac version costs less than a similar solution from 1Password, which makes Wallet an even more useful and attractive solution.


mSecure, 329 rubles


256-bit encryption and a whole set of tools for conveniently saving passwords and account numbers. In addition, the proprietary technology to protect cloud data, independent of the reliability of the ‚Äúcloud‚Äù itself, backup by sending to e-mail, optional data destruction when attempting to hack. The highest rating in the AppStore.


oneSafe, 199 rub


This nice universal app uses iCloud for syncing. Retaining the same functionality as 1Password, OneSafe and mSecure, this program uses all the features of iOS to give the user a convenient and functional interface, coupled with intuitive navigation. iCloud works great, which makes this app a winner in this review. oneSafe is probably one of the best apps of its kind for iOS.


Worth paying attention


DataVault Password Manager, 329 rubles.


DataVault is a worthy replacement for 1Password. The application provides for easy migration by importing data from 1Password, eWallet, SplashID, mSecure, IE, Firefox, Roboform, and other applications through Windows or Mac software. The application allows you to synchronize data via MobileMe, WebDAV, local Wi-Fi and file sharing. It can also store various data formats, including passwords, credit cards, frequent flyer cards, bank accounts, and much more. This program has a pretty nice interface, but not as beautiful and elegant as 1Password.


eWallet, 329 rubles


The application performs all the same functions as the previous ones, but does it in a spartan and utilitarian way. If you are looking for true beauty, then this is not your choice. If you are a supporter of a practical approach, then this is a great application. Includes 256-bit encryption, a keeper of all passwords, synchronization via Wi-Fi, provided you are on the same network, and an automatic password generator.


AllYourPasswords, 33 rub


This application best describes the feedback of one of the users in the AppStore: ‚ÄúGreat application, what you need, on the subject, without whistles :)‚Äù The functionality is standard, low price.



How to protect the contents of the wallet using these applications?


Once you have installed one of these programs, start by walking through the contents of your wallet. Enter in the application account numbers and additional information about each debit and credit card, as well as all ID cards and club cards. For each cell, enter the support telephone number in case of loss or theft.
Subsequently, if you ever lose your wallet or steal it, you will have a complete list of missing items with phone numbers in order to block the cards and order new ones. You will not spend a lot of time looking for numbers of all organizations that were recorded on maps. Simple but effective solution. What to do if your phone was stolen? - do not worry, because at least you have a wallet and a backup copy of the data on the network.
