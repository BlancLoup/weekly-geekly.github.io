Microsoft Tag

VS


At CES 2009, Microsoft demonstrated the new Microsoft Tag technology.
If you think that this is just a clone of QR-code technology - you are mistaken.

differences:
  • The tag does not almost store information, it contains a unique id on which through MS-ovsky server you get more detailed information (link, picture, text, video, pointer to the map, etc.). On the one hand, minus, on the other hand, you can find out how many people viewed your tag
  • the use of color and triangles makes it possible to store more information on the same physical size
  • The reader programs that are available (http://gettag.mobi) for Windows Mobile, J2ME, iPhone (sic! Is already the second application), Blackberry, and Symbian S60, if a contact (business card) is coded in the tag, can immediately add it to adressbook device
  • Microsoft Tag is based on High Capacity Color Barcodes (HCCBs) technology that was invented at Microsoft Research.
  • Anyone can make a tag quickly and for free using tag constructor
