Changes in the Yandex ranking algorithm

Tonight, July 3, 2007, the ranking algorithm will be updated on a search in one of the Yandex data centers. Within several days on the same query, the results for different users may differ markedly, this is normal. As soon as the same search works in all data centers, the difference will become insignificant.

Alexander Sadovsky: We hope that in a number of queries, an improvement in the relevance of the search will be noticeable to the naked eye. If you still find requests, where there is a clear deterioration, please write to us with an indication of the request, the disappeared sites and an explanation why you consider them more relevant than sites that remain: feedback.yandex.ru/?from=webmaster

To find out which ranking algorithm fulfilled your request, go after receiving the search results to the page www.yandex.ru/cgi-bin/hostname , which shows the name of the server that served you. If it looks like sfront ??. Yandex.ru or ludi.yandex.ru, then you got a lucky ticket and the opportunity to test a new search. If the name looks like sfront2 - ???. Yandex.ru, then a lucky ticket will drop you the other day.
