The August update of Windows Phone Toolkit is out!



On Thursday, Microsoft released the August update of the Silverlight Toolkit for Windows Phone. An updated version of this library is available for download here .


List of changes:
  • Smooth new LongListSelector designed for Windows Phone 7.1 OS
  • MultiselectList
  • LockablePivot
  • Hubtile
  • ExpanderView
  • LocalizedResources
  • Datetimeconverters
  • DateTimePickers
  • HeaderedItemsControl
  • Phonetextbox
  • Improved Transitions performance
  • Improved ContextMenu performance
  • ListPicker supports multiselect
  • Various bug fixes

Items included in the toolkit:
  • Autocompletebox
  • Contextmenu
  • Datepicker
  • Datetimeconverters
  • Datepicker
  • ExpanderView
  • GestureService / GestureListener
  • HeaderedItemsControl
  • Hubtile
  • Listpicker
  • LocalizedResources
  • LockablePivot
  • LongListSelector
  • MultiselectList
  • Page transitions
  • PerformanceProgressBar
  • Phonetextbox
  • TiltEffect
  • Timepicker
  • Toggleswitch
  • Wrappanel
