Multitasking and Silverlight in Windows Phone 7 confirmed

The network has leaked documents relating to the development of a platform for Windows Phone OS 7. Among other things, these materials confirm what was rumored: first, Windows Phone OS 7 supports multitasking, and secondly, this is the most pleasant , development of applications for Windows Phone 7 will be conducted using Silverlight technology.



Besides:
- it was confirmed that it would be possible to write XNA-applications for the platform;
- ordinary developers will write managed code, but OEMs will be able to write unmanaged code and an additional API;
- the vector interface is built on XAML (this obviously results from Silverlight), it can be developed in Blend and Visual Studio 2010 (apparently with a special SDK);
- push-notifications for applications with the ability to activate a closed application after the arrival of the event.

We are waiting for the MIX10 conference in March for official details.
