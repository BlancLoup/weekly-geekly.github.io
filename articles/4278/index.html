GoDaddy opens podcasting platform

GoDaddy, one of the world's largest domain registrars, has opened a service for hosting and managing podcasts. Platform services cost from $ 5 per month, and each client is given a discount on the purchase of a domain name.

The Quick Podcast service is intended primarily for individual podcasters and small companies. The interface is as simple and clear as possible. First you need to register your RSS feed ( screenshot ), after which you can upload to the server the first file indicating its genre, related tags, titles, graphic images and a brief description of the file ( screenshot ). As soon as a podcaster uploads an audio file, the system automatically generates a new message in the RSS feed and exports it to all major podcast directories, including iTunes, Yahoo, iPodder, etc.

Each user can quickly make an individual web page from the ready-made templates for his podcast, which by default has a built-in MP3 player so that you can listen to a podcast excerpt directly on the site without downloading the entire file.

Files are stored on the GoDaddy server for an arbitrarily long time, while there is a special interface for managing the media archive. The podcaster also gets access to detailed statistics with the number of downloads of audio files, listeners' IP addresses, incoming URL URLs and other statistics ( screenshot ).

Of the additional features - support audio format AAC, which is used in portable iPods on the Apple platform, as well as in the online store Apple iTunes.
