Ubuntu 7.10 course

On January 16, the release of the textbook Ubuntu 7.10 ( Ubuntu 7.10 Desktop Course ) was announced. The course is quite voluminous: in order to fully master 10 lessons, it will take 2 days. In addition to learning, it is possible to use the book as a reference book for finding answers to emerging questions.

There are versions for student and tutor (two files of 72.5 megabytes, 398 and 414 pages, PDF). The course for the teacher contains instructions and answers to the questions of the exercises formulated in the course for the student.

The ten sections of the course include: introduction to Ubuntu, studying Ubuntu's desktop, using the Internet, using OpenOffice, Ubuntu and games, customizing the desktop and applications, working with graphics and photos, music and video, Ubuntu help system and support, partitioning and download.
