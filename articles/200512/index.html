Football Manager 2014 released and available under Linux



FM14 was the first game in this series available on Linux. In this game there are a number of innovations. Players will be able to use cloud storage, which will allow access to their saves from any computer anywhere in the world. Steam Workshop, which means that managers will be able to easily create and publish game-related materials, such as picture / emblem sets, new and modified competitions, as well as unique challenges - this will be helped by the new Test Editor. Improved match engine. Revision tactics. More realistic transfers and contracts. Improved interaction with the board. Improved interaction with players, staff and media. News system. Improved user interface. The evolution of classic mode Football Manager.
