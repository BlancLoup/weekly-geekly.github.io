About CyberFilter


CyberFilter is a cloud solution that allows service providers to filter out prohibited resources by URL, domain or IP address. Compliance with the requirements of 149-FZ, 187-FZ (‚Äúanti-piracy law‚Äù), court decisions without losing subscribers and implementing expensive DPI solutions.
CyberFilter allows filtering (blocking) of individual pages of sites, as well as entire sites. Filtering can be done by URL, domain / host or IP address. Ease of integration allows you to start using CyberFilter quickly, with minimal changes in the configuration of the equipment of the telecom operator. CyberFilter features:

  • formation of the RPZ zone and route table, according to the registers of prohibited sites
  • automatic loading and processing of general registers (UAIS, etc.)
  • private lists of prohibited sites (to fulfill the decision of the courts on blocking sites)
  • filtering by URL, domain, IP
  • view and search in the registers of prohibited sites
  • creating users and assigning roles to them (for example, for technical support staff)
  • viewing system usage statistics
