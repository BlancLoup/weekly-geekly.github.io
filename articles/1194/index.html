Opera browser on a multimedia chip

3DLabs successfully ported Opera browser on a multimedia chip.

Media-centric SoC (system-on-chip) DMS-02 uses Opera 9 For Devices and this means that it can launch Opera widgets, allowing developers and manufacturers to create small programs to run on a chip. In other words, this chip can be used with any chip devices that have a screen.

Thus, instead of creating on-screen menus and other interface elements in traditional machine code, you can write them in the classic set of HTML, CSS, and JavaScript as Opera widgets.

Based on OperaWatch and MyOpera .
