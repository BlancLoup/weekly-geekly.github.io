Copy Cyrillic links in Firefox

Probably every Firefox user faced the problem of copying and pasting links containing Cyrillic (or, for example, hieroglyphs).

Suppose I want to give a link to the main page of Wikipedia; if I select the address and copy it to the clipboard, then when pasted it will look like this: ru.wikipedia.org/wiki/%D0%97%D0%B0%D0%B3%D0%BB%D0%B0%D0% B2% D0% BD% D0% B0% D1% 8F_% D1% 81% D1% 82% D1% 80% D0% B0% D0% BD% D0% B8% D1% 86% D0% B0

How to keep readability of the link, I will tell further.

  1. Drag-and-drop - drag and drop. Click on the address bar with the left mouse button (the address should completely stand out) and, holding the same button, drag the link to the right place for insertion.
    Disadvantages: works only with the address bar (the link from the phrase ‚ÄúWelcome to Wikipedia ‚Äù after dragging will be in hexadecimal representation).
  2. Incomplete copying.

    Disadvantages: the same as in the first paragraph.
  3. Edit about: config. I learned about this method just by typing in Yandex the request ‚Äúcopying a link with Cyrillic‚Äù ( 1st link ).
    We type in the address bar about: config and change the network.standard-url.escape-utf8 true to false. Restarting the browser is not required.
    Disadvantages: the same as in the first paragraph.

More information about URL coding can be found in Wikipedia .

It should be noted that in the Opera link is inserted readable with any of the three above-mentioned manipulations. Almost did not experiment with other browsers.
