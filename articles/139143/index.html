Game Survivor (Commodore 64) on html, css and javascript

Quite high-quality emulation of the game on HTML + CSS + JavaScript, html blocks are used for drawing, not canvas.

Imagine that the court is now the year 1982 and you are playing an arcade game. It was a favorite pastime among teenagers of that time. Your task is to destroy everything you can :) Use arrows to control, shift or ctrl to fire, and space to mine. The goal of the game is to destroy the base, while not getting caught by the enemies and not being killed by the walls.

Survivor was originally written for the Atari 2600 in 1982 and was ported to the Commodore 64 a year later. This browser version was written by Scott Schiller in homage to the Commodore 64 and for another demonstration of how HTML + CSS + JavaScript can be used to emulate games and computers that were running more than 30 years ago.
