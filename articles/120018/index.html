Transfer contacts to wp7



A few days ago I became the proud owner of an excellent device HTC 7 Mozart. Naturally with windows phone 7 on board. I used to use the good old Qtek S200 with Windows mobile 6.1.

The first problem that arose is how to transfer contacts ?! Contacts in my old device 273 pieces. To interrupt manually is difficult and lazy. Microsoft does not offer any cross-platform programs (I did not find it) - only complex and sophisticated methods, and even then - those who tried to do something according to the recommendations of MS know that even following precise points - you can get confused even more ...

All third-party proposed methods did not suit me. Because it describes the process of synchronization through ActiveSync and Outlook Express. And I have Win7 installed, so there are no such programs for it.


So. Way out.

1) Go to the small-scale website and download / install the Windows Mobile Device Center 6.1 - this is the equivalent of ActiveSync for Vista / Seven.

2) We synchronize the old device on wm6.1 with the help of a normal cord - the settings there are simple, we accept all the recommended parameters. We choose what needs to be synchronized (in my case - it was just ‚ÄúContacts‚Äù - I took off all the other checkboxes).

NOTE: Contacts will be synchronized with Microsoft Office Outlook - respectively, you must have this pack installed. If not, first put it, and then continue to synchronize the contents of the old device.

3) Windows Live Mail has already been installed in win7 - we are looking for options using these words in the Start menu. Run the program, log in using @ live.com login, which is tied to your Windows Phone. Further, from the bottom left of the program, select "Contacts".

4) Click ‚ÄúFile - Import - Address Book of the current Windows user‚Äù. All contacts added. Remember that the computer must be online (cut to an Internet). Otherwise, all actions are useless. Well, either, I suppose, the contacts will be updated with the connection of the program to the Internet.

5) We chop up the Internet on the phone, waiting for a bit of time: the contacts are synchronized independently in the background, without any notification. If it will be long - wait, do not panic and look for other ways to import. Everything is already uploaded to the Live server (if it‚Äôs not 3 pt) and is loaded into the phone ... Just slow, as it was in my case.
