Skype 4.3 for Linux released



As Microsoft promised in late May , a new version of Skype for Linux has been released. This is a fairly large update:
  • Updated UI
  • Cloud chats (sending messages offline, adequate synchronization between devices)
  • Improvements in sending files when using multiple devices simultaneously
  • Improvements for people with visual impairment
  • PulseAudio 3.0 and 4.0 support
  • A large number of bug fixes

However, there is a very controversial change: there is no longer support for ALSA. Offer to use PulseAudio.
It is not yet clear whether video chat support has appeared or not.
