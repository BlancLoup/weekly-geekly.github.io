Developed new NPAPI extensions for plugins

As you know, any browser can be improved by expanding its functionality through plug-ins. But until recently, often plug-ins were created individually for each browser (with rare exceptions), which did not allow the use of the same plug-in in all browsers. But now, it seems, the situation will change dramatically: a friendly team of developers, which includes programmers from Opera Software, Mozilla Foundation, Adobe, Apple and Sun Microsystems, jointly wrote new extensions NPAPI, which will allow you to create cross-browser plug-ins. Now it only remains to wait for really reliable and convenient plug-ins created by talented developers using new extensions to appear.
