Mozilla will stop supporting Firefox 1.5 in April

Yesterday, Mozilla announced it would soon stop supporting Firefox 1.5.x. This information appeared in the annotation to yesterday's update for Firefox 1.5 and 2.0, which says that for the old version of the update will be released only until April 24, 2007. After this date, Firefox 1.5.x will no longer be supported by the development team. So if you want to receive updates for your browser, you will have to go to Firefox 2.0.

First of all, the lack of support for Firefox 1.5 will become a problem for corporate IT departments, as companies make a massive transition to new software difficult. It is precisely because of this that Microsoft continues to support Internet Explorer 6, despite the fact that it has moved to a better seventh version a relatively long time ago.
