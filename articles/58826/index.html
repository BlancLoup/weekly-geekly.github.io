Russia, University Tour 2009

But that's not all. For the first time in the framework of the Russian university tour will be held:

WEB PROGRAMMING COMPETITION

The competition is held at each university, so that students from all visited cities (not even from the host university) can count on the prizes (and they will). The rules are the same for all universities and are very simple.

Contest Rules

The theme of the competition - "Widgets for education"

The task is to create an opera widget, the functionality of which will allow to somehow use it in the process of learning in a university. It can be anything from a regular RSS aggregator that reports university portal news to a sophisticated web application integrated into a distance learning system. You are limited only by your imagination, professional level and store of creativity. In this case, even unfinished works, which are only a prototype and a description of the idea, will be evaluated. Naturally, this should be a really cool idea;)

Primary requirements:

- Widgets must be created in compliance with existing web standards;
- Widgets should work on all versions of Opera 9.6x and higher (for Windows, GNU / Linux, Mac OSX), ideally - on Opera Mobile;

Dates of the competition

Start - has already begun;)
The deadline for accepting applications from applicants is May 11, 2009 at 24:00 Moscow time.
