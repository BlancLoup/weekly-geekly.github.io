Php


  • PHP 7.3 has implemented a significant garbage collector improvement.
  • [RFC] is_countable - A function has been added to PHP 7.3 to check for enumeration.
    Before: 
    if (is_array($foo) || $foo instanceof Countable) { // $foo is countable }

    After: 
     if (is_countable($foo)) { // $foo is countable }
