Open data as a basis

I deal with such a phenomenon in Russia as open data and lead the project OpenGovData.ru ( http://www.opengovdata.ru/ ). This is a non-commercial project in the form of a catalog of links to publicly available government data on the Internet, as well as an initiative to translate this data into machine-readable form.

What is open data?

Open data is a way of presenting publicly available information in machine-readable form. In the form in which developers can download them into databases, analyze and present in a much more visual and understandable form than how it is done in state systems.

Why do you need it?
Open data is what is essential for a large number of commercial projects, as well as projects aimed at the public good.
For example, without crime data in machine-readable form, it is impossible to put them on a map and show it clearly as they do in the project Crime Mapping - http://www.crimemapping.com/ .

Without data on the state budget that can be processed, it is impossible to show them clearly as it is done in the Where Does My Money Go? .

There are many other examples of how open data is the foundation. They help developers create something that state bodies cannot create due to their inability or impossibility.

Open APIs
The project OpenGovData.ru is actively developing and in recent initiatives it has an API that allows using some data arrays in practice.

In particular, there was an API for accessing data from the central bank of Russia - you can read more about this in my blog at http://ivbeg.livejournal.com/371247.html , as well as the API for obtaining information about the administrative-territorial division of Russia http: / /ivbeg.livejournal.com/370987.html .

These APIs can be freely used for any non-commercial and commercial purposes.

It is also not the latest API, but there will be others. In particular, I want to do the same thing that the guys from Sunlight Labs and OpenCongress did - collect data about our legislators and submit them in a convenient form for work.


Suggestions and Suggestions

I collect suggestions and suggestions for the development of the project. It has existed for more than two years, but just as soon as I got to announce it on HabrHabr and I hope for criticism and suggestions on how to make it as useful as possible for citizens.
