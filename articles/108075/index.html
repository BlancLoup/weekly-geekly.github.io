Available special build Ubuntu 10.04 for teachers

Today, the Ubuntu 10.04 teacher assembly has become available to the general public.

The assembly was prepared by the Nizhny Novgorod LUG with the support of the NRTK and is Ubuntu 10.04 with full Russian localization and a set of additional programs corresponding to the CPPS.

Below is a list of changes regarding the original distribution.

Added:
  • Russification of all applications
  • aptitude system
  • GIMP raster graphics editor
  • VLC video player
  • audio and video codecs
  • Xsane image scanning application
  • YAGF text recognition system
  • file and folder access protocol support
  • a set of applications for education by analogy with School Linux Junior 5.0.1
  • FreePascal IDE launch button
  • phun application
  • methodical materials on PSPO
  • batch image processing scripts in nautilus
  • master document templates
  • galleries in OpenOffice.org
  • HP LaserJet printer support
  • i-Sensys Canon support
  • support for MFU scanners from HP devices
  • 3G modem support

Corrected:
  • problem with keychains and passwords
  • texture problem in Stellarium

Other changes:
  • default desktop background
  • keyboard layout options
  • desktop icon alignment
