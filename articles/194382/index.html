Red Pitaya - 5 tools in 1 for $ 400

Just a day left before the end of the campaign on Kickstarter . So you can still have time to order a multifunctional measuring device that includes:
  • 2 channel oscilloscope
  • 2 channel spectrum analyzer with 60 MHz bandwidth
  • 2 channel arbitrary waveform generator
  • 2 channel frequency response analyzer with 60 MHz bandwidth
  • programmable PID controller



By the way, because of the success of the campaign, which has already collected more than $ 200,000, the amount of memory in the device has been increased from 1 GB to 4 GB DDR3, and the network adapter from 100 Mbit to 1 Gbit.

Red Pitaya Ecosystem


But Red Pitaya is not just a multifunctional board. It includes a whole ecosystem that simplifies conducting experiments, learning, developing and distributing applications.

Red Pitaya Ecosystem includes:
  • Red Pitaya - a high-performance tool for $ 400
  • Bazaar is a free market where open source applications are available in one click for direct use. The initial set includes: oscilloscope, spectrum analyzer, arbitrary waveform generator, frequency response analyzer PID controller, which can be accessed from any web browser from a tablet or laptop, and this market is growing!
  • Backyard is an organized repository containing relevant open source code and tools needed for application development.
  • Spark Center is a place to share new ideas, publish sketches or messages, and create new applications and hardware expansion modules together.
