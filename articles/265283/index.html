Microsoft added telemetry collection tool in Windows 7/8 / 8.1

Today on MS Windows 7 and 8.1 updates KB3080149 and KB3075249 arrived . Telemetry information is transmitted to the following servers:
vortex-win.data.microsoft.com
settings-win.data.microsoft.com

Transmission is via TCP on port 443 using SSL.
For paranoid people who are concerned about the unauthorized transfer of Microsoft information (hereinafter possibly to third parties), I suggest adding servers to the router's exclusion list, and following lines to the hosts file ( % windir% \ System32 \ drivers \ etc \ hosts ):
127.0.0.1 vortex-win.data.microsoft.com
127.0.0.1 settings-win.data.microsoft.com
