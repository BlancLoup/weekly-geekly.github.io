6 Gigabit SAS for 6th Generation HP ProLiant

This summer, HP began shipping a full package of components for use on HP ProLiant 6 Gb / s Serial Attached SCSI (SAS) servers, including 2.5-inch 7200 rpm hard drives and 15k and 10k rpm high-speed drives, RAID- Smart Array controllers and StorageWorks D2000 external disk shelves.



The introduction of SAS 6 Gb / s provides a doubling of the speed of data exchange with the server disk subsystem compared to the SAS 3 Gb / s, which is especially important for such applications with intensive input / output and large data volumes such as databases and transaction processing systems, server virtualization , work with streaming video and data backup.

Also in the new version of SAS, additional mechanisms for eliminating errors in data transfer are implemented and the flexibility of configuration and scaling of the disk subsystem serving several servers has been improved. Also, compared with SAS 3Gb / s, the maximum cable length has increased from six to ten meters, which makes it possible to combine disk arrays of several racks into a single system, and it has become possible to use miniature Mini SAS connectors designed primarily for 2.5-inch connectivity. winchesters.

The 6 Gbps SAS drives are already available for the rack-mounted models of the HP ProLiant DL380, DL370, DL360, DL180 and DL160 servers, HP ProLiant BL460c server blades and HP ProLiant ML370 tower servers. HP StorageWorks D2000 disk shelf with 6 Gb / s SAS drives is also available.

SAS 6 Gb / s is backward compatible with the SAS 3 Gb / s interface, which has been widely used in servers and storage systems since 2007, as well as SATA 1.5 and 3 Gb / s, so new drives and RAID controllers can be installed in servers the previous generation HP ProLiant G5, however, in this case, the capacity of the server disk subsystem will remain at 3 Gbit / s, so you should use the latest HP ProLiant sixth generation to realize all the advantages of six-gigabit SAS.
