Stability, Recycled Fonts and Geolocation for Linux

This version fixes some more bugs that have affected many people. International fonts have also been improved (although note the following list of known issues), and Wifi geolocation is now available on Linux.

If you have problems with geolocation on Linux, please provide information about the driver of your wireless adapter. You can get it using the command:

lspci | grep Ethernet

Note that FreeBSD is still not using Wifi geolocation, but only less accurate than the IP address of the computer.

Known issues:

International fonts do not work correctly on Linux and FreeBSD.
Geolocation on FreeBSD uses IP.

Warning: This is an unstable version. It is intended for testing and may contain bugs and shortcomings.
