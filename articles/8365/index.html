The first device for reading books on electronic paper, which appeared in my collection, was the LBook www.lbook.com.ua . As it turned out, a very nice device with its own undeniable advantages, still not accessible to competitors. At first glance, there should be no differences in the Lbook screen itself and competitors using the same type of screen, but it turned out to be not so simple.


The first and most pleasant feature of the Lbook is low power consumption and the ability to read Russian texts in 1251 encoding made from any material with the help of the simple Save as txt function without any problems. As I filled up my Lbook with books, I realized that a significant advantage over the same Sony Reader is the clear structure of folders that you can create with only three levels (for example, 1. Fantasy, 2. Stanislav Lem 3. "Solaris") which is most likely due to the type of file system used by the device, but this turned out to be quite enough to conveniently use a library of almost 5 thousand books. That is how much can fit in a simple text format on the SD card 2Gb, the maximum possible to use Lbook. Larger size cards are not supported.
