Google publishes Chrome source code for iOS



Google has published the source code of its browser Google Chrome for iOS, according to the official blog Chromium.

‚ÄúHistorically, the Google Chrome code for iOS was separated from the rest of the Chromium project due to the added complexity of implementing the platform. After several years of thorough refactoring, the entire code was reunited with the Chrome project and moved to the Open source repository, ‚Äùthe blog says.

Google Chrome developers have spent a lot of time and effort refactoring Chrome code for iOS. As a result, now any user will be able to compile the Chromium version for iOS as well as for any other platform.

Due to the limitations of the iOS platform, all browsers had to be built using WebKit rendering. For Chrome, this meant both WebKit support and Blink for other platforms. This created additional difficulties for the team and the developers tried to avoid it when placing the sources in the Chromium database.

The development of Chromium began in 2008. On the basis of this open project, the company's main browser, Google Chrome, has been built. In addition to Google engineers, companies such as NVIDIA, Opera Software and Yandex are taking part in the development of browsers on this platform. In addition, the forces involved in the open source community are involved.
