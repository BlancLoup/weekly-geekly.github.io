Citrix XenServer Version 7.1 Hypervisor New Release Released

The XenServer 7.1 version introduces several unique features compared to version 7.0. At the moment, due to the tight integration with Citrix XenApp / XenDesktop, XenServer is the best hypervisor for working with Citrix corporate mobility applications and the only one with full service support for 10 years. The main innovations of XenServer version 7.1 you will find under the cut.



First of all, XenServer 7.1 started supporting patching (.iso format) interactively, which allows updating the hypervisor without the need to disable or transfer virtual machines. When the patch is loaded, the system allows you to determine how much it requires changes related to the reboot, and if it is not required, the patch is installed without interrupting the processes. This feature significantly reduces the negative impact associated with downtime in infrastructure maintenance. Also in XenServer 7.1, batch updates were supported. In addition, in the new version of the Citrix hypervisor, the transfer of Linux or VMware vSphere virtual machines takes place without damage or changes to the files.

For the first time, the XenServer platform is offered as a long-term support version (LTSR) and today is the only hypervisor with full service support for 10 years. The life cycle period of XenServer 7.1 is fully consistent with the life cycle time of XenApp / XenDeskop, which simplifies maintenance and consolidation of the infrastructure.

It should also be noted that important changes have occurred in terms of licensing - now any edition of XenApp / XenDesktop includes the right to use Citrix XenServer with functionality similar to the Enterprise edition.

XenServer 7.1 supports Provisioning Services technologies for application and desktop virtualization. The PVS-Accelerator technology, included in the hypervisor, is a host-level cache memory, which reduces the network bandwidth requirements by up to 98% for each XenServer host, which increases the speed of loading virtual machines. The new simplified PVS architecture makes it possible to reduce the number of PVS servers required to simultaneously load the same number of workstations.


Caching with PVS-Accelerator in XenServer 7.1 (per 50-100 VM)

Citrix and Nutanix recently announced XenServer support for the Nutanix Enterprise Cloud HCI systems, which allows the Nutanix user to get free access to advanced XenServer features without having to use vTax for Citrix XenApp or XenDesktop scripts. These features include PVS integration, Direct Inspect API technology, online patch installation, and NVIDIA GRID vGPU technology (which Nutanix Acropolis does not support). To manage XenServer, the Nutanix Prism web-based web console can be used.


Nutanix High-Level Architecture Diagram

XenServer is strategically important for Citrix and provides the foundation for innovation of all Citrix software products, including the PVS-Accelerator, helping to implement large partner projects with HPE, Nutanix, etc.
