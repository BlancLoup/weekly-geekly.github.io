OpenSSL 1.0.0 release with GOST support

After 11 years of development, OpenSSL has released the first official version of this open source cryptographic package for working with SSL / TLS. See the official report .

OpenSSL allows you to create RSA, DH, DSA keys and X.509 certificates, sign them, generate CSR and CRT. It also has the ability to encrypt data and test SSL / TLS connections. The new version fixes many flaws in 0.9.8n, as well as adds new features (for example, support for BeOS and Russian encryption algorithms, which are fixed in GOST). See the full list of changes here .

The OpenSSL package is based on the SSLeay technology written by Eric Young (Eric A. Young) and Tim Hudson (Tim Hudson), who unofficially finished working on it in December 1998, after which they switched to RSA Security.
