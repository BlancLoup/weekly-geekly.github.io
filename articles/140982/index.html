The best applications - free promotion!

Nokia and Microsoft announce the program to support the best applications for Windows Phone.

Till May, 20th there is a reception of applications applications for Windows Phone from developers then the expert jury will choose 7 best of them. The winners will receive Nokia Lumia 800 smartphones, as well as free promotion of their applications during the month.

Terms of participation in the program


  • The program is open to developers from Russia and the CIS countries who have sent an application for participation before May 20;
  • The application must be published in the Russian Windows Phone Marketplace ;
  • The application should use the features of the Windows Phone platform, such as Fast Application Switching fast switching and Live Live Tiles icons.
