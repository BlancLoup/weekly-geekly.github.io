So, materials:

- paper, scissors, pencils
- plastic container - serves as a memory storage

Game mechanic:

Animals address the programmer with various problems that he solves by manipulating memory. Viewers help the programmer.

What it looks like, example:

The action takes place in the form of a theater. The little mammoth (from the famous cartoon) addresses the problem - the elephant refuses to recognize him. The programmer implants the elephant with an artificial memory of the mammoth - the programmer along with the elephant is removed behind the scenes, from where the programmer returns without an elephant, but with a container.
