Remote Desktop (RDP) in Windows 7

Remote Desktop Client in Windows 7 has received new features and improvements. The most interesting thing is to increase support for desktops, and full support for the effect of Aero Glass.

(IP for example shown)
The following are the main innovations of the remote desktop client in Windows 7

* Support for Windows Aero 7
* Support for Direct 2D and Direct 3D 10.1 applications
* Multi-monitor support
* Improved remote desktop client kernel
* Support multimedia accessories
* Media Foundation support
* DirectShow support
* Support low audio latency
* Bidirectional audio support

To watch a demo (recorded during the MVP 2009 summit) of all functions, including support for multiple monitors, support for DirectX 2D ((duration 35 min) TV episode of Brian Madden ).

I did a remote desktop client check in the latest Windows 7 build 7201 and tried a few things. The remote desktop client is significantly improved compared to its previous versions.
