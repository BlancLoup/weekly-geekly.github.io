How to get your web screenshots service - version 1.1

At the end of last year, for the purpose of supporting users of Windows programs, I needed to organize a Web service of screenshots with a client program that allows you to select a rectangular screen for placement. Third-party services like Gyazo ceased to meet due to the excessive amount of advertising and the inability to control measures to ensure reliability like backup using their own resources, so their own screenshots service and client program were written.

Approximately at the same time I wrote a post in the Sandbox (there was not yet an invite) - it is possible that it still lies there.
Since the release, small shoals and flaws have surfaced in the client program, such as the lack of support for multiple monitors. There was not enough time to figure out and finish these features for a long time, but the other day it appeared. So today is already present version 1.1. As usual, the program is laid out for free use and self-deployment along with the source code of the client program in Delphi 2009 (in theory, it should be compiled in earlier versions starting from 7 and later), server-side PHP-script and instructions for deployment. To deploy you need a Web server or hosting with PHP 5, a link to which is written in the client settings file, after which this file is imported into the Windows registry.
All information on the service and the archive for download are available on the page http://www.lubezniy.ru/soft/vlss/ . Naturally, criticism and bug reports are available (here).
