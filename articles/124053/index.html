MySketch: Google Docs (Drawing) and Skype in one person

When searching for a common solution, it is sometimes difficult to express a thought in words, it is much easier to depict it in a drawing ...



The easiest way to describe the essence of My-Sketch.com is already used in the title. It is simply a convenient tool for collective drawing.

It allows 2 people to draw various sketches (for example, site schemes, logo basics, etc.) in real-time mode with the possibility of text / voice chat and then save the received sketches.

MySketch is useful for designers, interface designers and many other people who are at a distance from each other.
