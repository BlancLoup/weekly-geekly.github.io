Moscow, September 12, Windows Camp - come, it will be interesting!


At the end of July, we announced the date and topics of the annual Windows Camp conference . The conference will be held on September 12 in Moscow in Digital October and we will talk about the development for Windows 8.1, Windows Phone and Windows Azure on it. And today we are pleased to announce that the conference schedule has become available with the names of reports and speakers.

The conference will be opened by Alexander Lozhechkin, Director of Strategic Technologies at Microsoft in Russia. After opening the conference is divided into three rooms. In the first, there will be two sessions in which partners will share their thoughts on how to successfully conduct business with the Microsoft client and cloud platform. The business session on the client platform will be moderated by Mikhail Chernomordikov, the head of the group of experts, and the session on the client platform will be conducted by Dmitry Marchenko, the head of the department for work with partners.

At the same time, in two other halls it will be possible to find out about the new opportunities for the developer in Windows 8.1 from Konstantin Kichinsky and the new Windows Azure services from Vladimir Yunev, experts in strategic technologies. The story about Windows 8.1 will be continued by Alena Gilevskaya with her report on application design, in parallel you can learn about new features in Visual Studio 2013 for application developer from Dmitry Andreev.

Conference reports will continue until 7 pm. You can learn about new features for XAML / C # and HTML / JS developer in Windows 8.1 and how you can organize a test environment using Windows Azure.

First-hand, you will learn about the opportunities offered by Nokia Imagine SDK, the gMaps developers will talk about the possibilities of cross-platform development with Monogame and cocos2d. Web developers are not forgotten either - they can learn how using Windows Azure they can create scalable websites and services.

And this is not all. New in Windows 8.1 and little-known features of WinRT, including the use of desktop applications in the report of the expert on strategic technologies Stas Pavlov, the use of shared code in Windows Phone and Windows 8 solutions from Dmitry Soshnikov, as well as authentication in your solutions using Windows Azure from Andrei Moore .

The final chord will be reports on the promotion and monetization of Windows Phone and Windows 8 applications, new in C / C ++ DirectX development in Windows 8.1 and the use of a ready-made cloud backend in mobile applications.

It seems to me that we managed to collect an interesting program and everyone will be able to find reports for themselves.

Due to numerous requests, we continue to accept applications for the conference. The advantage in consideration are applications from developers who are already developing for Windows Phone, Windows 8 and Windows Azure platforms.

If you are not able to join us offline, do not worry - we plan to broadcast and record conference reports, although, in my opinion, there is little that can replace live communication with speakers and fellow participants.
