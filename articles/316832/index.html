Demo cloud access to Ultimate 2C

A registered user after ~ 20 minutes of waiting for the generation of a private cloud will get access to a deployed installation of the Ultimate 2C entry-level enterprise management system.

In the created cloud will be deployed:

  • Ultimate Solid Universal IEM Platform
  • Ultimate 2C base configuration
  • development and customization tools
  • demo data (goods, customers, suppliers and other counterparties).
  • web component of an online store based on CMS UMBRACO

Interface - English (with the ability to switch to Russian in the settings).

The demo cloud is working in test mode on limited capacity, please treat the possible queues with understanding.

Distributions for self-installation available at any time for download without queuing
