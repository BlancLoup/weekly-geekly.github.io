Ext Designer released


The final build of the WYSIWYG interface editor for the popular JavaScript framework ExtJS has been released .

The editor is paid and costs $ 229
Let me remind you that ExtJs is a framework designed to create a RIA (rich internet application), which involves the use of various, rather complex user interface elements. And there are plenty of such elements in ExtJS. Setting all the parameters is troublesome.
Ext Designer is here for visual customization. It also allows you to configure data stores (DataStore) and bind them to the interface elements
