Chrome 4.0 Stable


Just now, a Google developer blog reported that the 4th stable version of the browser has been released.

Download online installer (currently only Windows - version)

Version 4.0 contains:

  • Extensions (perhaps the main innovation)
  • Bookmark Sync
  • Advanced Developer Tools
  • HTML5: Notifications, Web Databases, Web Sockets, Ruby Support
  • V8 performance improvement
  • Skia Graphics Library Performance Improvement
  • Complete ACID3 test passing due to newly added support for downloadable fonts.
  • Even safer (many fixes + new security mechanisms)
