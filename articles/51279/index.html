JS Charts - a generator of graphs and schemes.


JS Charts is a free JavaScript generator of charts and diagrams that requires a minimum of programming knowledge. With JS Charts, drawing diagrams is simple and easy. It does not require any additional plug-ins and server modules. Just plug in the scripts, prepare the data for the schema in XML or a JavaScript array and your table is ready.

JS Charts allows you to create charts based on various templates, such as columns, pie charts or simple lines. And it is compatible with most web browsers.

Opportunities:
* You do not need to write code
