Each site owner - own search!

The company "Search technology" has created a search engine Flexum. The developers say that any site owner can manually create their own search engine and install it on their resource. This can be a search by a group of sites devoted to a certain topic, a search by Internet resources of a region or by a separate site.
The creation process takes place by manually entering the portals into the database. The resulting search engine webmaster can install for free on its resource.

Flexum engine can install an ordinary webmaster simply by copying a special program code to the site. Developers plan that in winter the search engine will pick up the necessary momentum. Experts are interested in a new product, but they doubt the interest of consumers. Reports the newspaper "Kommersant".
