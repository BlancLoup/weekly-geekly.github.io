Three releases of Ruby on Rails in three days

6 months after the last release, Ruby on Rails has been updated to version 2.3.6. The changes are mainly related to bug fixes, adding a few new features, and preparing a springboard for switching to Rails 3.0. A complete list of changes can be found at the link Ruby on Rails 2.3.6

The next day came out Ruby on Rails 2.3.7 , which fixed the HTML security bugs copied from Rails 3.0, but added a bug related to the rails_xss plugin. This bug made vulnerable applications that did not use this plugin.

Well, the last at the moment is Ruby on Rails 2.3.8 , in which everything works as it should.
