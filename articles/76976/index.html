15 good mobile website examples

We selected 15 examples of valid mobile web sites. The number of users of mobile devices exceeds the number of users connecting from their computers, and it is clear that the future is mobility.

Surprisingly, there are very few sites on the web that have mobile versions. Although some large sites have such versions, but this is not the norm - most sites today are not designed for mobile devices.
The design of mobile sites varies from very simple text to rich with graphics. Today, most mobile devices can work with high-speed Internet, correctly handle HTML, JavaScript and sites with AJAX-technologies.
We expect the rapid development of mobile websites, and after him the quality of the sites and their interactivity will increase.
To date, there is a tendency to create uncomplicated mobile websites for mobile phones, and more saturated for the iPhone. I hope this trend will also change very soon, as even older phone models can perfectly display HTML, and their number many times exceeds the iPhone - this begs the question: why not create mobile sites that meet the requirements of most devices.

What makes a good mobile web site a really good mobile web site?
1) Compatibility.
The site should normally display in the most common screens. The most popular today is the size 240x320. Sizes larger, like 480x320 (iPhone), are becoming more and more popular as new models of devices appear with larger displays and higher resolutions. If you are developing a site for mobile devices, make sure that the site is correctly displayed on the screens of the most popular sizes.
The site must support xhtml. Try to avoid using javascript, but if you have to - do not use it in critical elements, such as navigation and links.
2) Content must be optimized.
Pictures should be reduced in size, both in size and in kilobytes, in order for them to load quickly and fit most popular screen sizes.
3) Usability
The site should be convenient to use. Many mobile sites use this kind of navigation, which makes website orientation a nightmare.
4) Good design.
Design is very important. A good mobile design should not only look good, but should not seem cramped. Navigation should be easy, and the site should load quickly.
