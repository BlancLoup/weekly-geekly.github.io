Edumandriva 2009 Spring LXDE One CD - final release

The final release of Edumandriva One 2009 LXDE has been released. This is a Live CD designed to work without installation on the hard disk, as well as an installation system, as well as installation on Flash charms.

In this version:

+ Updated software up to date: Scratch, Kumir, FoxItReader
+ Fixed bug with non-assembling programs from Geany under FreeBasic.
+ In Geany, the default FreeBasic syntax is configured for compatibility with QBasic, and Haskell with Hugs98.
+ Fixed bug with Qt4 libraries (different versions)
+ Added plugins to AbiWord
+ Fixed a bug with the desktop in LXDE
+ The distribution kit is correctly installed on the hard disk.
+ Localizations: English, Russian, Italian, Ukrainian, Belarusian, German, Tatar and French (system).

Full disc:

File managers
* Midnight-Commander (mc) - FAR analogue for the console

Archivers
* Rar format support
* 7zip format support
* File-roller

Office applications
* AbiWord is a text-sensitive processor from the GnomeOffice package. By functionality close to MS Word 97
* Gnumeric ‚Äî spreadsheet-undemanding spreadsheets - similar to Excel 97
* FoxitReader - fast PDF viewer

Raster and vector editors
* GIMP - raster graphics editor, the functionality is not inferior to AdobePhotohop
* InkScape - Powerful vector editor in SVG + uniconverter format for working with CorelDraw files
* TuxPaint - A simple graphic editor for children and adults. + A set of image stamps.

Audio and video

WEB / INTERNET

Programming

* FreePascal - Pascal and Object Pascal compiler - Geany is recommended as IDE
* Geany - A simple IDE for FreePascal, C, C ++, etc.
* Scratch - Programming for kids based on Logo
* Hugs98 - functional programming language (simplified Haskell)

Keyboard simulators
