Publishing tools

Finally, the moment has come when we can share the news about the imminent launch of the project I'm Sync .

While our team is completing preparations for launch, we have the opportunity to talk a little about what the project is and how it can be useful for you. Especially for this, we have created our blog . In it, we will post information about the project and share news from the development front.

So what is I'm sync? First of all, this is a new approach to publishing information.

As part of the project, we have developed special components that can be embedded into any site using JavaScript. Using these components, users will be able to upload various information to the site, for example, information about upcoming events, photos, etc. By adding a publishing component to your site, you will get convenient tools for publishing information.

The principle of operation of the components is similar to the Google Maps service, with the only difference that the components of the publication can be used not only to display information, but also to upload it to the site.

Even if your site is written only in HTML and is located on a free hosting service, you can easily upload information to it using the components of the publication.

Later we will tell you more about what information you can download, as well as show how it will look. If you have any questions, please ask them in the comments or send to support@iamsync.com.

In addition, we are now available on Twitter , so that you can receive news about the project and participate in the discussion in real time.
