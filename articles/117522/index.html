Fennec (Firefox) Aurora and Nightly

Following the Mozilla Foundation-initiated Firefox transition to four-channel development with the release of Firefox 5 alpha 2 and Firefox 6 alpha 1 , the Fennec browser (codename for the versions of Firefox Mobile in development) was brought up. Thus, Fennec Aurora and Fennec Nightly became available as successors of Firefox Mobile 4, and Fennec Beta will appear in the future.



Currently, the Fennec Aurora branch contains the code for Firefox Mobile 5, and Fennec Nightly for Firefox Mobile 6, and by the time Firefox 5 is ready to switch to the Beta channel (that is, after a few weeks), Firefox will come to the Nightly place Mobile 7.

Test versions of Firefox Mobile for Android on ARMv7 (ARMv6 is no longer supported) (in English):
Download .apk Fennec 5.0 alpha 2 Aurora (11.8 MB)
Download .apk Fennec 6.0 alpha 1 Nightly (13.8 MB)
