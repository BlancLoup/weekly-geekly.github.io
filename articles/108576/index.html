Remote control of the camera

Greetings, habrasoobschestvo! I want to talk about an interesting way to wirelessly control the camera. I myself constantly work in the studio so that the captured images are immediately transferred from the camera to the laptop via USB, which makes my method even more convenient.

What you need: a mobile phone with Bluetooth and Java and a laptop connected to the camera via USB.

It all started with the fact that I, in my Flying F003 Chinese phone, found a built-in program for controlling a computer via Bluetooth. In principle, such programs are for any phones with Java support. In many programs, there is support for assigning phone keys, which further simplifies the work.

And then everything is simple. We start a remote control program on the computer that comes with the camera (I have Canon Utilites with their Camera remote shooting), connect the phone and start using it.

It is clear that the above method is far from universal. But for work in the studio or at home, when shooting with a tripod, is quite acceptable, and most importantly, a budget option. I repeat that it is especially convenient to use this when photos are transferred directly to a computer, since there is an opportunity to immediately watch the result and make corrections without touching the camera itself.
