Microsoft released API for Windows Live Spaces

With the release of the Windows Live Spaces API for developers, including commercial companies, Microsoft will allow them to create different mashups and deal with millions of unique users per month for free. If developers manage to attract a larger audience, then they will have to either pay Microsoft 25 cents per user per year, or conclude a partnership report on the company's advertising. Microsoft will also provide an API for working with photos and user contacts (with their permission) to their Windows Live Spaces social network.

Obviously, Microsoft is trying to increase traffic to their web projects. Microsoft is making a number of its products more and more open, providing more opportunities and in many ways competing with companies based on Internet technologies like MySpace, which is not so open. With the release of its software interface, Microsoft also makes its service suitable for business solutions.

Just in Windows Live Spaces, several new features have been added for their mobile services. New pages of the latest updates and comments allow you to see recent changes in blogs, photos, as well as find out what your friends are doing.

Microsoft is expected to make a few more statements about Windows Live Spaces at the upcoming Mix07 conference next week. We will keep you in the center of events ...
