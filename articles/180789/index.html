Google Code does not allow file placement.

Google warns that it is forced to close the file hosting service on Google Code due to the ‚Äúincreasing number of cases of misuse of the service‚Äù and to ensure the ‚Äúprotection and security‚Äù of users.

New rules come into effect immediately for new projects. Post files for new projects on Google Code is now impossible. Old files are safe, and Google has no plans to delete them in the foreseeable future.

If the old project wants to place new files, then it can only do this until January 14, 2014, after which the possibility of publishing the files will also be closed for old projects.

Google recommends moving the files to some alternative hosting, at least on Google Drive . For more on this, see the FAQ . Unfortunately, no more than 15 GB of files can be placed on Google Drive for free and download statistics are not kept.

The company does not explain what is the "misuse" of the service, but you can guess. Free fast hosting with unlimited space for any type of file is a pretty useful thing.

Google Code is a service for publishing source codes, with support for Subversion, Git and Mercurial version control systems. According to independent statistics , approximately 300,000 open source projects are hosted on Google Code.
