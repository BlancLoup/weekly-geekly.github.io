Export contacts from Facebook to Google+

A couple of days after the announcement of Google+, the first extension for Chrome has already appeared, which automates the export of contacts from Facebook to Google+. For four days, the expansion of Facebook Friend Exporter was installed by more than 17 thousand people.

Source code fb-exporter on github .

With the advent of Google+, the war for the social graph , which Google and Facebook have long fought, is resumed. As you know, Facebook does not allow exporting a list of your friends to mailing addresses and prohibits the automatic collection of this data (paragraph 3.2 ToS ). The previous Chrome extension with the same functionality was blocked on Facebook, and then removed from the Google Code site and from github. Let's see what Facebook will do this time (UPD: 05.07 the application is blocked).
