Browser extensions - search for information on any site

On many sites it is difficult to find the necessary information due to the fact that the search on the site does not work or it works poorly.
My browser extensions solve this problem - you can search for information on any site using popular search engines (Yandex, Google, etc.).



The idea is simple - use special search engine operators to search for information on the current site (opened in the active window / tab).

I started developing these extensions back in 2010. The first was FireFox . Then Opera was added to it. The last to be created is an extension for Google Chrome / Yandex Browser.

Install from the add-ons directory FireFox .

Install from the Opera extensions directory (including for ‚Äúclassic‚Äù Opera 12).

Install from the Chrome online store (including for Yandex Browser).

There are two possible uses:
  1. Entering a search query in the search form (for FireFox, the standard search form and Cntrl + Enter, for all the others, the pop-up search form and Enter)
  2. Search through the context menu. Select a text fragment on the page -> right click -> "Find ... on this site"


After entering the text, the search engine opens with a list of pages of the current (open) site, on which there is a search text.

Extensions are configured to work in search engines Yandex, Google, Bing, Yahoo and Baidu. You can select a search engine in the search form or on the Expansion Options page. You can also turn off Google Instant and open a new window.

Answers to frequently asked questions:
  1. No, this is not Ctrl + F. Ctrl + F - search by current page. Extensions - search the entire site.
