Release MeeGo v1.0

Today was the release of the open platform MeeGo sponsored by Nokia and Intel.

This release contains the stable kernel MeeGo (MeeGo Core Foundation) and the user interface for netbooks (MeeGo Netbook UX). A user interface release for handheld devices (Handheld UX) will be released in October, and its development will begin in June. Also was released MeeGo SDK, which, so far, is focused on developing applications for netbooks.



MeeGo Netbook User Experience includes:
  • Rich user interface for netbooks, built using the latest open source technologies
  • Direct access to the calendar, tasks, recently used files and social network updates via the home screen
  • Aggregation of social networking content, which allows you to see your activity in them, as well as communicate with friends and update status.
  • Integrated Google Chrome or Chromium for fast and rich surfing. In the future, releases for handheld devices will support Mozilla Fennec
  • Easy to use applications for mail, calendar and media player
  • High performance optimization
  • Languages: Japanese, Korean, Chinese, Swedish, Polish, Finnish, Italian, Brazilian Portuguese, French, German, Spanish, Russian, English, British English

The MeeGo v1.0 Core Software Platform includes:
  • Linux kernel 2.6.33
  • DeviceKit and udev for interacting with devices
  • Modern 2D / 3D graphic stacks (Cairo, OpenGL / OpenGL ES)
  • Connman Connection Manager, Ofono, BlueZ telephony stack
  • Qt 4.6
  • Universal Plug and Play (gUPnP)
  • Media frameworks (GStreamer)
  • BTRFS is used as the default file system.

Further releases will be released every 6 months, that is, MeeGo v1.1 will be released in October 2010.
