A proposal to unite all observers

Hello!
I hope it is not a secret to anyone that on March 4 we have a presidential election. If you suddenly forgot, I will remind you now.

We have prepared for this moment and launched a system for monitoring the situation in the elections in real time.

The system consists of two parts:



1. Website - let's call it a platform for the focal point. Now it is launched on demo.digitalsoftware.ru . Here come the data from the observers (from the mobile application).

2. The mobile application is what the observer has and, accordingly, is the source of data for the information system.

The site collects statistics from observers and in real-time displays data at any level of the hierarchy - from the Precinct Election Commission (lowest level) to Russia (upper level), allows you to send messages to observers (both to a specific PEC and to all of Russia / districts / areas / areas / cities - depending on the current level of the hierarchy, which is the coordinator on the site).



Mobile app features


1. The application allows you to fix the violation with the immediate sending of photos / audio / video to the server.

2. The application allows you to keep a record of the number of ballots in the ballot box (the +1 button) with periodic data sending to the server - this way we can plot the activity of voters for the entire voting day and detect possible discrepancies with the official totals.

3. The application allows you to take a picture of the final protocol and then send it to the server.

4. The application contains a memo for observers

5. The application allows you to receive notifications from the server during the voting day.

Initially, we began to work with RosVyborami , but later came the idea to unite all. Also thanks to the Gracon project for the correct hierarchy of PECs.

I propose to unite all observers, give them our application and monitor the statistics in real time in the coordination center.

Below are the application screens:

1. Home at the time of receiving a new message



2. Home without new posts



3. Report a violation



4. The result of the counting of votes



5. Messages



Supported application platforms: iphone, android, symbian.
