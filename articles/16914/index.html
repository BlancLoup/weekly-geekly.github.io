So, exactly two months have passed, since it was possible to get the final version of Flash Lite 3 (for this you had to send a request ). Today, Flash Lite 3.0 Developer Edition is available for download at Adobe Labs .

Flash Lite 3.0 Developer Edition is a standalone mobile player in ‚Äúpre-release‚Äù status, and is more intended for developers who can already test applications on the Developer Edition until Flash Lite 3 is put into commercial operation. And where does the name Developer Edition come from ?
Interesting thing. Flash Lite 3 Developer Edition does not bind to the serial number of the device. So there will be no problems with the IMEI number :) Yes, and at startup, it will be said that this is the Developer Edition.

When to wait for the final release ?
It is not known, as they say, as soon as it is immediately. And these are not my words, but Adobe words.

What devices can Flash Lite 3 Developer Edition test on ?
Officially, Flash Lite 3 Developer Edition can be installed on Nokia N95. Theoretically and technically, it should work on any Nokia S60 3-series phone or higher. These are: 3250, 5500 Sport, 5700, 6110 Navigator, 6120 Classic, 6121 Classic, 6290, E50, E51, E60, E61, E61i, E62, E65, E70, E90, N71, N73, N73 Music Edition, N75, N76 , N77, N80, N80 IE, N81, N81 8GB, N82, N91, N91 8GB, N92, N93, N93i, N95, N95 US 3G, N95 8GB.

And just in case, how to install Flash Lite 3 Developer Edition on the phone. It's simple :

1. Download Flash Lite 3 Developer Edition
2. Launch Nokia PC Suite
3. Click on the Install Applications icon
4. Select the SIS file and download it to your phone.
5. Restart the phone
6. In the phone menu, select the Applications / Applications item and find Flash Lite (it's easy to find using the familiar Flash icon)
7. And that's it. :) There are questions, ask them on the official forum .
