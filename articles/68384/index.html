Racios - Dream, plan, realize!

www.Racios.ru is my new project. Dedicated to time management. It implements a fresh look at planning, and also offers interesting applications that allow you to simplify work with a large flow of information (Projects, Task Box, Regular Affairs, Diary). The project is still in the beta version, it is now actively searching for a full-time programmer to bring to mind and start implementing an ambitious plan to introduce a new interesting functionality!

The application "Regular affairs" is useful for those who have many repetitive tasks. The purpose of the application is to help track the time spent on a specific task (for example, you decided to read the book, and forgot about it for a week ... and with this application, with daily ticking of the tasks you took the time, you can find out tasks)



Another application "Diary". He has 2 tasks: 1st - to show that time is running out (the past days become faded, let's understand that they cannot be returned. And 2nd task - ala memoir - to write down the most interesting, memorable events on the past day)



The project chip is drag'n'drop between the tables - so you can transfer tasks between different planning levels, for example, assign a weekly task to the day.



Daily mailing arrives with a review of tasks for today / week / month.
