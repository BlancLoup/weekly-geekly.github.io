Telegram detected another successful attack

Not so long ago, we wrote about a successful series of attacks on Telegram users in Iran. Using the technique of intercepting SMS messages with the activation code of a new device, the attackers were able not only to de-anonymize users of the service, but also, in some cases, to access their correspondence.
