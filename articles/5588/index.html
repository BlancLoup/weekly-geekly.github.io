Greasemonkey: a cloud of popular Habrategov when adding habratopik

Taking the idea from del.icio.us, I wrote a Greasemonkey script, which on the add-on page of Habratopik makes ajax request to the popular tags page (http://www.habrahabr.ru/tag/) and loads the list of popular tags for blogs under the input field tags for your new topic. Click on the tag: it is added to the tag input field, click again, deleted.

To install the script, you need to go to the url:
userscripts.org/scripts/show/8038

And press the black button Install Script at the top right.

ps tags to this post are added using the cloud :-) With the exception of the tags cloud tag and greasemonkey, which are not among the popular ones.

update :
Greasemonkey is a plugin for Firefox. First you need to install it:
www.greasespot.net

Then you can put scripts that change the contents of the pages using javascript. Many useful scripts can be found here:
userscripts.org
