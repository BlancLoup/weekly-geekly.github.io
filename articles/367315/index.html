KoBoJet - keyboard with projector



Most all-in-one computer systems need to be connected to the display for work. The Taiwanese company Sho U decided to create a system for which a display is not needed. True, this is not a full mini-PC, but a keyboard with a built-in projector. In order to use such a system, you need an Intel PC Compute Stick mini PC.

A pico projector embedded in the keyboard projects the image onto any vertical surface (for example, on the opposite wall).

The size of the projected image is 21-33 inches, the brightness of the projector is 150 lumens. Image resolution is 1920 * 1080 pixels. In addition to the pico projector, the system configuration includes 3 W speakers, WiFi, Bluetooth, USB ports, HDMI, Ethernet. Due to the presence of HDMI, it is possible to connect a full-fledged display if the projector is not suitable for some reason.

For the convenience of the user, you can also connect a wired / wireless mouse, and we get a full-fledged PC. Unfortunately, the cost of the keyboard KoBoJet is still unknown. In the sale of the device comes in 2016.

Such a keyboard can be useful in many cases, for example, on the road. Of course, a laptop is more convenient than a keyboard, to which you also need to connect a mini-PC, but you can save money here - especially if you take some Chinese mini-PC to connect. True, it remains to wait for the announcement of the price of KoBoJet.
