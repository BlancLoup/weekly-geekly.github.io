2GIS for iPhone

What happened was what we were so often asked about, what we had been waiting for and what we worked so hard on. Meet: 2GIS mobile version for iPhone . We promised that it will appear before the end of the year - it appeared. The greatest mobile omission is finally fixed, and the favorite directory is now available on all major platforms, including iOS. The application is already in the Apple App Store - hurry to download and share feedback. What's new and why so long - read on.





What's new? Almost all. We could not just take and adapt the old mobile version for the iPhone. It was necessary to develop it separately. New design, new interface, new 3D maps. We tried to make everything as beautiful, stylish and functional as possible - in general, especially for the iPhone. The map now displays 3D buildings that can be viewed from all angles. The size of the map file that needs to be downloaded to work with a particular city has significantly decreased (in the case of Novosibirsk, more than 2 times).



Why so long? Of course, the wait was stretched - some even doubted that the iPhone version would be released altogether. But given the volume of tasks that we had to perform, it took not so much time. Development for the iPhone began in late February 2011. Much has been written from scratch, as the schemes of the previous mobile version for the iPhone were simply unacceptable. For example, the interface was completely rewritten (organization cards, icons, screen views, design, code structure). In general, we tried to take into account everything that is already in the current mobile version, as well as provide the ability to add other useful features in the future. It's safe to say that the iPhone version is a new, significantly reworked application.



What's next? We will certainly modify the original version, both in terms of map display, and in terms of functionality. We have big plans for 2GIS for iPhone, and we are already working on their implementation. Of course, the new design and 3D map will extend to the other versions that were released earlier: we recall, this is 2GIS for Android, Symbian OS and Windows Mobile.

The current version is beta, which will soon become better and better. At this stage, your comments are very important to us, so please feel free to share ideas, comments and suggestions.
