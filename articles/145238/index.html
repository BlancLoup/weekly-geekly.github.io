Google Analytics adds a feature for analyzing the size of a user's browser window.



No wonder they say that Google Analytics is one of the best tools for monitoring and analyzing site traffic. In general, in this package there is already quite a large amount of functions, but the other day the developers added another one. Now it is possible to analyze the compliance of your site with the size of the browser window of visitors. This feature is the result of a combination of two other functions: tracking the display resolution of visitors + browser size tool from Google Labs.

As usual, testing of the new functionality is currently being conducted, so it will work fully with all of this in a week or two (some users of Google Analytics already have the function, others have not yet).

Anyway, you should check the ‚ÄúContent‚Äù section in your Google Analytics account, plus ‚ÄúIn-Page Analytics‚Äù. And in this section there should be (or appear later) the Browser Size menu item.



According to the developers, now the new opportunity is more relevant than ever, as more and more people come to the Network via mobile devices, with displays of various sizes. At the same time, site content (including contextual advertising) is not always distributed in the browser window in the best way. In some cases, when working with certain devices, your (or any other) site is not particularly attractive - especially if the user has to scroll the page both vertically and horizontally to get the desired content.
