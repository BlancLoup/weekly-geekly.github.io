Synology has released the official version of DiskStation Manager (DSM) 4.2


A brilliant solution for storing large amounts of data and transferring multimedia streams.

DiskStation Manager 4.2 offers:

Secure storage. The LUN Snapshot and Clone functions allow you to receive up to 256 snapshots from each LUN, without affecting system performance and taking up minimal storage space. Synology's fault tolerance system (SHA) is supported by a large number of DiskStation and RackStation models, and, thanks to the hot-swap drive function, ensures uninterrupted system operation for small and medium-sized businesses. A two-stage verification system, including a given password and a one-time six-digit code, increases the level of security.

Ease of management. RADIUS server enhances the protection of the Wi-Fi network and provides convenient centralized access to the system.

DNS Server allows users to host and centrally manage various domains. The Disk Usage Report feature clearly demonstrates the dynamics of disk space usage, providing the administrator with all the necessary information for developing storage strategies. And with the Task Scheduler feature, which is used to centrally distribute any tasks according to a user-defined plan, system management becomes even easier. In addition, Synology NAS supports TFTP and PXE environments, making it possible to download client PCs or servers remotely.

Advanced cloud architecture. LDAP and AD support makes it easy to manage an almost unlimited number of user accounts. Amazon Glacier BackupBeta provides cost-effective data storage and backup. Data management becomes even more efficient, increasing the level of performance when transferring data to Mac devices by 52%. And File Station, thanks to the support of hot keys and the ability to display objects in the form of lists / icons, simplifies data management.

New multimedia features. Video Station with support for DVB-T tuner and HDHomeRun allows you to play and record several channels simultaneously. Also, in this service it is possible to transfer multimedia content to AirPlay and DLNA devices, using mobile devices to provide remote access.

Audio Station has a stylish, user-friendly interface with an updated mini-player and supports Bluetooth A2DP technology, providing high-quality audio content to any point in the room.

Photo Station and Photo Station Uploader provide new features for dragging items and simplify the transfer of photos over the network as much as possible.

Full data control. The DS cloud application allows you to synchronize files between desktops and devices on iOS / Android operating systems using the SSL security protocol. The user can set the transfer rules, the size and type of files, as well as set up synchronization using a Wi-Fi connection. The DS video application makes it easy to continue watching videos on any mobile device from the point where you left off. The DS photo + application provides the ability to automatically back up and transfer photos in real time from iOS / Android devices to Photo Station. In addition, in DS cloud, DS audio and DS photo + applications, there is now a QuickConnect function for remote access to files without the need for additional port forwarding settings.
