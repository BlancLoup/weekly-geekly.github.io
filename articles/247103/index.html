How to get sounds from top iOS apps



  • Turn MacBook on
  • Run iTunes application
  • In the Search field - drive in Angry Birds
  • Select Angry Birds from the list and download to your computer.
  • Go to the My Applications tab
  • On the Angry Birds icon, click the right mouse button and select Open in Finder
  • We leave iTunes in the Finder and copy the Angry Birds 1.6.0.ipa file to another location.
  • Rename the Angry Birds 1.6.0.ipa file to the Angry Birds 1.6.0.zip file
  • Unpack the Angry Birds 1.6.0.zip file
  • We go in the unpacked archive in the Payload folder
  • In the folder is the application AngryBirdsClassicLight.app - click the right mouse button and select Show Package Contents
  • Go to the data / audio / sfx directory
