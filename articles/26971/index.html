QIP PDA Build 2000

The program is designed for devices running Windows Mobile 5 and 6 with a touchscreen. The program provides VGA and qVGA display modes.



This program is not a development of the previous version 1020 for this type of device, but was written from scratch taking into account the experience gained by the author in writing previous versions and new realities. The program has become multiprotocol. At this stage, ICQ (almost full) and Jabber protocols (receiving / sending text messages and working with a list of contacts) are implemented.

Before installing, it is recommended to remove the old version of the QIP PDA, since New installed in the same folder! Transfer of settings and history is not built in yet.

In the near future it is planned to actively develop the program taking into account the wishes of the participants of our forum.

Despite the fact that this is the first (test) release of the program, according to the author, the program already favorably differs from the previous version with its stability and functionality. In addition to the basic features of the IM client, the program joined the QIP.RU products with full skins support! Now each user can customize the appearance of the program for themselves, as well as share a good design with other users. The list of contacts can now be displayed in several columns. There is support for vibration, animated emoticons and pop-up message generation system.
