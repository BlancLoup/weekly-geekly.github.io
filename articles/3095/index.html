Touchscreen Phone with Samsung QWERTY Keyboard

Samsung Electronics has introduced a new model in its series of thin phones. At first glance, the Ultra Smart F700 is quite similar to the already familiar slider communicators with side-mounted QWERTY keyboards. With closer scrutiny, it becomes clear that it has more similarities with Apple's iconic iPhone, although there is still a sliding QWERTY. This communicator or phone, while it‚Äôs not too clear to which class of devices it should be attributed, is equipped with a 2.78-inch touchscreen display and has almost no buttons when folded. In this work is clearly not based on Windows Mobile.



Among the advantages of the model is the use of VibeTonz technology, which provides the sensation of pressing a button when pressing a touchscreen display. The device supports high-speed data transmission standard HSDPA. In addition, it is equipped with a 5-megapixel camera with autofocus. Samsung is going to demonstrate the Ultra Smart F700 at the 3GSM Congress.


Specifications:
- GSM / EDGE (900/1800/1900 MHz), UMTS / HSDPA (2100 MHz)
- 5 megapixel camera
- 2.78-inch touchscreen TFT-display with a resolution of 240x440 pixels
- Bluetooth, USB
- slot for microSD memory cards
- QWERTY keyboard
- Media player
- HTML browser, Email, Java, WAP 2.0
- Document Viewer
- Dimensions - 104x50x16,4 mm
