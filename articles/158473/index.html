Opera extension: search open site 1.1

On November 6, the final version of Opera 12.10 was released. One of the innovations is the Context Menu API, a long-awaited work with the context menu. As a result, a new version of the Site Search extension (not to be confused with Ctrl + F).

On many sites it is not easy to find the necessary information due to the fact that the search on the site does not work or it works poorly. My extension solves this problem with the help of popular search engines (Google, Yandex, etc.).

Download and install from the Opera extension directory

Before first using the extension, you must restart the browser.

There are two uses:

1. Enter the query in the search form.

After entering the text in a new tab, a search engine will open with a list of pages of the current (open) site, on which there is a search text.


2. Search from the context menu.


The extension is configured to work in search engines Google, Yandex, Bing and Baidu. To select a search engine, go to the Tools menu -> Extensions -> Manage Extensions (or press Ctrl + Shift + E) -> InSite Search Extension Settings. If you wish, you can open the search results in the current window and disable Google Instant.

Fans of the FireFox browser can use a similar extension for their browser .

Updated
A new version (1.1.1) has been released. The bug with encoding Russian words in the context menu has been fixed.
