DipTrace. Innovations

Dear friends, I am glad to inform you that DipTrace continues to be updated and improved.

The picture is taken from the official site (c) www.diptrace.com .

So improvements (information is also taken from the official source):

Beta-Version 2.3.5 (January 14, 2014)

- Redesigned user interface, the program has moved to a new development platform.
- Unicode support in all program elements.
- Updated library and component display system (library panel removed, component panel updated)
- Updated Stamp Editor.
- Improvements to the panel "Snap to the body." This panel is now identical in Circuit Design and Component Editor.
- Updated polygonal pads window ("pads").
- Changed import dialog boxes of third-party schemes and lists of connections, remembering used cases.
- Recycling of the properties panel (in the construction area) in the Component Editor and the Body Editor.
- Zoom / pan the scroll wheel in the preview pane in the print / import / export dialog boxes.
- Updated interface of other dialog boxes in the program.
- Import STEP and IGES 3D enclosure models.
- Export 3D board model in STEP.
- Algorithm copper fill improved (3x-5x fold increase in speed).
- Correct change of the side for groups of objects in the Editor of Cards.
