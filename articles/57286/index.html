Evernote + Twitter = Instant Memory



We have acquired a new way to enter information in Evernote and hope that for many, especially fans of microblogging, this process will become even faster and more convenient. As it probably already became clear, now you can add notes to Evernote using any Twitter client.

The most pleasant thing in our opinion is that Twitter clients are on many different platforms, including Linux, Symbian, JavaME, so memorizing your thoughts, friends' messages and other different things will become even easier, regardless of device and platform. .

All you need is to link your accounts in services.

How it works?
  1. We created a special Twitter account, myEN, which acts as an intermediary between Twitter and Evernote. Go to http://twitter.com/myen and ask friends to this virtual user by clicking ‚ÄúFollow‚Äù.
  2. The user myEN will immediately reciprocate and send a personal message (Direct Message) to Twitter, where there will be a special link:
  3. On it you need to go to the Evernote website, if necessary, enter the login / password to log into the account and confirm the account binding. This preparatory work ends.

Now, to create notes, you can simply send private messages to user myEN through any Twitter client. The message will immediately appear in Evernote, in your default notebook. You can also archive and public tweets by simply adding the @myen line anywhere in the message.


We also taught Evernote how to process images posted on twitpic.com that are referenced in your tweets. Since many Twitter clients support autoloading photos to this resource, this feature should be useful. It works like this. If the message has a link to this photo, Evernote when creating a note will insert a small copy of the image there with a link to the original.
