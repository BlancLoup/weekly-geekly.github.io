The first day of the PDC09 conference. What happened?

a little about Internet Explorer 9, C-code on Windows Azure, data mining, MVC 2 Beta and Martians



The first day of the PDC conference from Microsoft took place - one of the most significant events of the company in the year when new technologies are shown, products are released and plans are announced.

As always, this time it was not without interesting things about which I would like to tell. I‚Äôll say right away that I‚Äôm telling information about the time keynote, so the data may differ slightly from the real ones or they need to be clarified.


Windows Azure


Windows Azure was the main pivot of the day. The platform will be launched in January and will remain free until February. Azure will support PHP, Java, MySQL and of course the entire .NET and SQL Server stack as Sql Azure. A representative from WordPress spoke and confirmed WordPress support for Azure. But the strangest demonstration of the possibility of a new cloud was writing code in pure C language, which can be published later in Azure. C ++ is also supported. In my opinion, this is somewhere beyond good and evil, but it perfectly shows the capabilities of the platform. Of course, the code must be led to some requirements, but in general they do not interfere much.

It was announced that for the Azure platform, Microsoft has already built 6 large data centers around the world and will build more.

PinPoint and Dallas


Microsoft launches two new projects PinPoint and Dallas . The first is an online store of services, web applications and data for many corporate and not-so-many companies. The second is the technology that allows you to exchange data sets using open protocols: sell, buy, subscribe. Together - as I understand it, this is a kind of ordered Atom-based store where you can buy a data pool, for example, NASA and process it as soon as you want. We will see what comes out of this, but for now the idea looks interesting - the data store.

Be a martian


On the basis of Azure and Dallas, the space agency NASA has launched an entertaining project " Be a Martian ", which is designed to popularize science and tell about Mars.

Internet Explorer 9


About IE9 will tell today. Yesterday did not say anything, only dry spread throughout the Internet. However, at keynote it was stated that IE would be the best browser for Windows :-) that, for sure, just marketing.

Silverlight


We are all waiting for some news about the next version of SL4. And neither should appear. Today will be the famous Scott Guthrie, so we are waiting. Yesterday, in the context of SL, it was announced that Seesmic (makes popular twitter clients) migrates all its applications from Air to Silverlight. Trifle, but nice.

AppFabric


A new edition of the Windows Server 2008 add-on over IIS - AppFabric was announced. It is designed to create high-speed computing servers for hosting the Windows Workflow Foundation and WCF services. The server will contain a combination of new technologies for hosting and caching Dublin and Velocity.

Windows Identity Foundation


The former Geneva project has reached the final stage of RTM and is ready to exit. It will be released for VS 2008/2010 with a GoLive license. If you don‚Äôt know what the framework is, I recommend reading it . This is a new platform for creating trust relationships between different environments.

ASP.NET MVC 2 Beta


At the end of the keynote was announced the release of ASP.NET MVC 2 Beta, which anyone can download now . The source of the project with the addition of Futures. New in this version of the mass. Wait for a separate note.

My impressions


For many, the first day seemed boring. The thing is that here all the rules of Windows Azure and those who are not interested in this topic - missed. Wait for Scott Guthrie today and tweet with the hashtag # pdc09, I think that will be about to write.
