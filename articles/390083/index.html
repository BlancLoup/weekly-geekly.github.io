[Announcement] Kingston increases the maximum capacity for DataTraveler Micro 3.1 USB drive and SDXC memory card

DataTraveler Micro 3.1 is one of the most compact USB drives in the Kingston lineup. In addition to ultra-small dimensions, the advantages of the device include a metal case and high speed operation.

Specifications:


  • Capacity: 16, 32, 64 or 128 GB
  • Speed: USB 3.1 Gen 1 (USB 3.0)
    • 16 GB - 100 MB / s read, 10 MB / s write
    • 32-128 GB - 100 MB / s read, 15 MB / s write
  • Warranty: 5 year warranty and free technical support
  • Compatible OS: Windows 10, Windows 8.1, Windows 8, Windows 7, Windows Vista, Mac OS X v.10.8.x +, Linux v.2.6.x +, Chrome OS


Kingston microSDHC / SDXC UHS-I U3 memory cards support read speeds up to 90 MB / s and write speeds up to 80 MB / s and fully comply with the requirements of UHS-I Class U3. They are suitable for shooting video in Full HD (1080p), Ultra HD (2160p), 3D and 4K formats with the help of high-performance portable devices - GoPro smartphones and cameras. SDCA3 memory cards provide fast loading and editing of files, especially when used with USB 3.0 card readers. Steadily high recording speed allows you to shoot HD-video without delays and conduct continuous photography. A full-length adapter to SD allows you to use these memory cards with digital SLR cameras, video cameras and other digital devices.
