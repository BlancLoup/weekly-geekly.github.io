All English in one flash application

Hi, Habrayuzer.

This is unbelievable, but the fact is that almost all the grammar and words of the English language that an ordinary person might need were able to fit into one small application on a flash in order to make learning a language more mobile and simple.

Learn English in your free time when you have internet or a phone at hand - in the subway, in traffic, before going to bed - just do the exercises and move forward.

In the application, you just need to select the topic that you would like to practice - prepositions, purchases, times. Computer words and Computer words 2 tests are highly recommended to all IT professionals.

Each test consists of ten questions. If something does not work, run the test again. If something remains unclear - click the Notes button next to each question.

Good luck in learning English!
