NetBeans IDE 7.2 Beta Release



Today, all fans of IDE NetBeans were pleased with the new release, containing quite a lot of interesting and very useful innovations.

While the new version of IDE is in Beta state (the release version is promised in June), but, as the developers themselves say, why wait, if you can take advantage of new features today ?

The most significant changes include support for various PHP frameworks, improved support for Groovy, integration of Scene Builder into JavaFX, as well as many other improvements in Java EE, Maven, C / C + + and the NetBeans platform. Let's name the most interesting of them.

Java

  • FindBugs support
  • Library Support for TestNG Testing
  • New refactoring and tips
  • Various improvements for Maven


Java ee

  • Oracle Public Cloud and Amazon Elastic Beanstalk support
  • More editor tips for EJB and CDI
  • Improved and updated support for JSF components


Javafx

  • Scene Builder Integration
  • Improved support for FXML and JavaFX CSS


Php

  • Support Symfony2, Doctrine2 and ApiGen frameworks
  • Remote FTP sync
  • Improved PHP editor: support for annotations, editing Neon and Apache configuration files and much more.


Groovy and grails

  • Groovy 1.8.6 support
  • Significantly improved work with the editor: tips, code completion and much more.


C / C ++

  • C ++ 11 standard support
  • Improved work with GDB


Besides:

  • Subversion 1.7 support
  • Improved project search functionality
  • 64-bit version of launcher for Windows


Also, the developers promised a significant performance improvement, an even smarter editor, and, of course, they fixed bugs that were discovered.
