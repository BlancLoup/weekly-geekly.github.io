New version of Kinect for Windows SDK, new website and blog for developers released



By the anniversary of the sensor Kinect, Microsoft has prepared new items not only for ordinary users, but also for developers. New resources were announced today: http://www.kinectforwindows.org/ and the official Kinect for Windows Blog . You can read the first blog entry at the following link .

But the biggest gift for developers is the new version of the Kinect for Windows SDK Beta 2 development tools. This version reports on improving the speed and accuracy of skeleton tracking, x64 support, multi-core processors, and turning off and on the sensor without consequences. In addition, a new API is presented for working with several sensors simultaneously.

Download the new version of Kinect for Windows SDK at the following link http://www.kinectforwindows.org/download/ .

In addition, the official Facebook page and twitter account are available at http://twitter.com/#!/KinectWindows .

And on Channel9 you can find a video dedicated to these news.
