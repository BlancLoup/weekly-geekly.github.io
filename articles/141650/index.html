Website promotion, code optimization

Website promotion on the Internet, an integral event after the creation of a web site. These events are made in order to attract visitors to your site. There are many methods of website promotion. This article will describe how to properly optimize the internal pages of the site for the proper and successful promotion of your resource on the Internet.


In a previous article about the impact of site loading speed, I described how the site loading speed affects page ranking by search engines. In order to successfully promote, it is important to minimize the weight of the pages of the site. that is, the reduction of the code page, as the main page of the site, and internal.


Code optimization

All search engines first of all pay attention to the amount of content on the page and the amount of code. That is, the ratio of the size of the content (article) to the size of the code of the entire page.

There is no specific correlation, it may exist, but nobody knows it, but only one knows less code on a page, and the more unique text on a page is, the higher is the probability that your page will be higher in search engine results than your competitor‚Äôs page.

One of the factors ranking Google search engine, is the speed of loading pages, the speed of loading a site or page, the better. The probability of hitting the top increases.
Reducing the weight of the page, removing unnecessary scripts, removing extra html terms on the page, you will take the first step to successful promotion.

Meta - site page tags

After optimizing the page code, adding unique content to it, you need to correctly specify the meta tags for each page of the site. These tags must be different. These tags must be present on the page. With the help of these tags, search engines find the necessary pages and determine which pages to issue after the search query.

Key Meta Tags
  • Title - page title;
  • Description - page description;
  • Keywords - keywords for the page.


By correctly specifying the meta tags you tell the search engine information about your site, or about a particular page of your resource. This is a very important point in SEO optimization and website promotion in search engines.

H1-h6 headers

Tags h1-h6 define the importance of the text that is located after the title. With the help of them you can select individual sections of the text. The most important heading is the H1 heading; it is considered to be the first level heading, respectively, and the priorities are the largest.

They allocate the title of the article on the page. The remaining tags are distributed in order and have lower priorities. The main tags are usually the tags H1, H2, H3.

In conclusion, we can conclude that to promote the site you need to perform several simple activities.

Website promotion. Code optimization:
  • Code reduction
  • Adding meta tags to the page
  • H1-H6 Header Writing
  • Writing attributes title and alt to images
  • Rewind site pages (relink)
  • Validation code verification
