July free cookies for designers and developers



In this issue, a selection of the most interesting free cookies. Almost every day there are many interesting applications, resources and libraries that optimize the work of designers and developers. In this issue, I collected the most interesting products from Western communities that came out in July:

Sketch: Supernova Studio, Qwikly, Sketch Material
Web apps: Paper Sizes, Grabient, Canva colors, TypeHero, Abstract
Developers stuff: MDB Angular GMD kit
Designers stuff: Memphis patterns, Travelisto UI kit, Flow font, Lists.design, WayFX logos


By the way, if you use Figma , I recommend paying attention to our ready-made design systems . They help freelancers complete more orders per month, programmers can create beautiful applications on their own, and the Timlids run sprints faster using ready-made design systems for teamwork.

And if you have a serious project, our team is ready to deploy a design system within the organization based on our developments and tailor it to specific tasks using Figma. Web / desktop, and any mobile. We are also familiar with React / React Native. Write to T: @kamushken




Supernova Studio


https://supernova.studio/

Sketch plugin that allows you to convert your design directly to Xcode and boldly transfer all resources to developers. Android is also supported. If they are telling the truth, then this thing has been waiting for a long time and it has enormous potential.





Qwikly


https://www.getqwikly.com/

Qwikly is an attempt to add properties to the prototyping tool for Sketch. Allows you to test your design immediately on your smartphone, there are several standard transition effects between screens. Rapid testing of ideas has not harmed anyone.





Sketch material


https://websiddu.github.io/sketch-material/

If you need to build a UI for the Google Material guides immediately in the Sketch, this plugin is for you. Tables, cards, buttons, etc. There are all components from the GMD set made in accordance with the specifications.





Paper sizes


http://papersizes.io/

Do you work with printing? All international formats and standards will be useful to you: sheet, envelope, business card, poster, etc. All this is presented in the form of an interesting UI wrapper with a sauce of modern typography and style.





Angular Bootstrap with Material Design


https://mdbootstrap.com/angular/

Another UI framework for Angular. Made with high quality. According to the developer, it contains: 400+ material elements, 600+ icons, 74 CSS animations and in general absolutely no jQuery.





Grabient


https://www.grabient.com/

Against the backdrop of a universal boom: the next online gradient constructor! More advanced than the previous ones: you can change the angle, add your own colors, adjust the transparency. Export to Sketch? Of course.





Canva colors


https://www.canva.com/colors/

Are you starting a new design project and there are absolutely no ideas on colors? Then you are on the ‚ÄúGospel‚Äù. Ready palettes (very much), their applicability, examples and even a little psychology in the form of a detailed description.





Type hero


https://typehero.now.sh/

Some unofficial extension of Google Fonts functionality. You can work with colors, test different palettes. It could have been done better, lacking the ability to assemble font pairs and too much space not being used.





Abstract


https://www.goabstract.com/

Abstract is positioned as git for designers, but so far is only friends with Sketch. They promise that the rest is on the way. You can use the web version or download under macOS. Started just recently, the interest of the audience increased. He blows some success from this product, although personally for my practice it is not needed yet.





10 Memphis Style Patterns


https://pixelbuddha.net/freebie/10-memphis-style-patterns-free

This is a collection of free memphis style backgrounds. If you love collecting likes on Dribbble, this is for you. Now there is almost no interest in a shot without triangles, circles and squares. And against trends you will not trample ...





Travelisto UI Kit For Sketch


https://dribbble.com/shots/3624797-Travelisto-UI-Kit-For-Sketch

Free UI KIT for a travel theme sketch. Vobschem something without any alterations under any other topic. The concept is based on a trend vertical scrolling in the form of concise cards. Thank you, Robert Anitei.





Flow font


http://danross.co/flow/

Flow is a free font for rapid prototyping. Modern alternative to fish texts. Available for download in three styles. You drive in abracadabra, put breaks with a space. Conveniently!





Lists


https://www.lists.design/

But if you make a hi-fi prototype, then you can‚Äôt do without real texts. And the project Lists comes to the rescue: addresses, surnames, cities, positions, etc ... Absolutely any type of texts in the form of a list. JSON export is available.





WayFX logos


https://wayfunction.com/logos/

If you are launching a startup and there is no time to invent a logo, WayFx hurries to help by offering various modern emblems for free (including for commercial use). Available for download formats SVG and AI.





UX Flow


https://free.lstore.graphics/uxflow/

UX Flow is a free prototype UI kit. Adapted to the Sketch, it offers a universal set of standard screens with the ability to quickly customize and build a flow of ways.





Git point


https://gitpoint.co/

Free, simple, modern and stylish Git-client for iOS. The developers claim that their product is endowed with all the necessary features and even more.





Sound Kit (by Facebook)


http://facebook.design/soundkit

And for dessert, a free Facebook sound kit for your mobile apps. The main thing - do not overdo it with the implementation of sounds in your product, so as not to cause irritation among users.
