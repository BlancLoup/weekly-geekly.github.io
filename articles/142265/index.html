Windows 8 editions became known

Microsoft in the Windows blog officially called the list of editions of the new operating system: Windows 8 , Windows 8 Pro , Windows RT and Windows 8 Enterprise .

Compared to previous versions of Windows, the number of revisions has decreased significantly.

For normal user systems, desktops, laptops and tablets based on x86 processors, a regular version of Windows 8 will be released.

The pro version will receive enterprise-level functions, namely encryption, virtualization technologies, the ability to enable PCs to the domain and manage them using group policies. Windows 8 Pro will become the only editions with the multimedia package of Windows Media Center.

Windows 8 Enterprise will include all the features of Windows 8 Pro, plus IT organization features that include PC management and deployment, advanced security policies, virtualization, new mobility scenarios (?), And much more.

Windows RT (formerly known as Windows 8 ARM) is the Windows edition for tablet and other computers based on ARM processors. It will include versions of Word, Excel, PowerPoint and OneNote, optimized for the touch screen (package known as Office 15). Also, the operating system will include a special version of Internet Explorer 10, but the usual Windows Media Player is missing. It is worth noting that Windows RT will not be compatible with existing Windows programs. Windows RT cannot be installed by itself - it will be possible to get it only with a new computer as a pre-installed system at the factory.

All editions will have the Metro interface and support the Windows Store application store.
