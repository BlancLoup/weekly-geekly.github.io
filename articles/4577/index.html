Google strikes back. Personal search engine with Google CSE

A week ago , Microsoft announced an update to the Live Search Macros service , allowing everyone to create their own search engine. Today, Google introduced new features as part of the Co-op (beta) service , namely Google Custom Search Engine .

Creating your own search engine in Google CSE begins with defining a list of sites that the user considers to be authoritative sources of information in a particular area. The main difference from the Microsoft service is that Google CSE allows you to configure two modes of operation with this list: the created search engine can search only these sites or simply give them preference.

You can connect other users to work on changing the created mechanism, or allow anyone to change the settings. The home page of the search engine, in addition to the search string, will include the creator profile and a list of specified sites. It is possible to customize the display of search results - colors and logo. Owners of network resources can get the html-code of the search form.

The search results pages will display ad units that can generate revenue for the creator of the search engine: to do this, just enter your Google Adsense login name in the settings. If the user is not set up to earn or does not have an account in Adsense, he can disable the display of advertising.

There are only three drawbacks yet:
- the Cyrillic alphabet cannot be used in the name and description of the search engine: if you edit the settings later, the data entered in Cyrillic will be distorted;
- when creating a search engine you can not set additional commands, as it allows Live Search Macros in the advanced mode;
