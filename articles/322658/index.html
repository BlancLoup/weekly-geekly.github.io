Foreign GPS monitoring services (part 2)



The article continues the description of foreign GPS monitoring services. As in the previous article, selected services that were updated or have published news no more than a month ago. All services support a large line of personal and car trackers, all have their own mobile applications. All provide the opportunity to work under their own brand (both within the cloud infrastructures of the manufacturer, and with the ability to install on their own servers / infrastructures).

There is no demo access in the TrackMe service, but you can quickly register and, using their mobile tracker application, get acquainted with the capabilities of the service. The Navixy service seems to be of Russian origin, but since I have never touched it in previous articles and it has many offices in America and Europe, I included it in this article.


As was correctly noted in the commentary to the previous article, this is not an analysis as such, but a statement of factual information. This work is carried out with the aim of further developing the free IOT & GPS service VIALATM . In my opinion, the collected information can be useful for developers and users of GPS monitoring systems, so I found it possible to share the collected data on Habr√©.


Tracking platform




Year of foundation: 2015 .


Main office: London, England . Branches: Greece, Lebanon, Belgium, Peru .


Representation in social networks: Youtube, Facebook, Google+, Twitter .


Mobile applications on platforms: Android, Windows Phone . Among them: employee monitoring applications, tracker applications and Live Chat applications (integrated with the platform).


Supports web client interface in a large number of languages .


Useful features of the system:


  • Auto coloring of vehicle icons depending on the state (connection break, traffic, parking, ...).
  • Control of fuel consumption by mileage (in this case, you can set various parameters for the summer and winter periods).
  • Create custom object attributes.
  • Definition of notifications on service vehicles (depending on mileage, engine hours, ...).
  • Connecting sensors. Very handy setup panel, includes the ability to set the calibration table.
  • Configuring the installation of states of motion / parking and the validity of determining the location of both GPS information and data taken from the sensors.
  • Setting up events, creating templates for event notifications.
  • RFID and iButton support.
  • Convenient setting of reports with the possibility of sending them by email on a schedule.
  • Ability to send commands to devices via GPRS / SMS. Includes setting of sending commands on a schedule and / or depending on the state of the object's parameters.


Navixy




Year of foundation: 2009 .


Representative offices : USA, Mexico, England, France, Germany, Russia .


Representations in social networks: Twitter, Facebook, LinkedIn .


Mobile applications on platforms: IOS, Android . Among them: applications to access the service, applications, trackers.


Supports web client interface in a large number of languages .


Useful features of the system:


  • Displays on the map of several tracks, including tracks for different objects.
  • Auto sorting of the list of objects by various criteria.
  • Convenient work with tracks and events on a time scale.
  • A large set of reports, with the ability to customize them. Setting the schedule for generating reports.
  • Receive reports directly in the application window, with the display of tabular data about events and graphic information. At the same time, you can quickly switch and display data for different objects.
  • Task formation and task list management. Tasks are attached to both employees and vehicles.
  • Work with routes, including the formation of tasks for the routes.
  • Control of driving style (speeding violations, sudden braking, ...).
  • Ability to add your layers to the map and import data from KML files into them.
  • Convenient work with geofences, including the ability to import geofences from KML files.


Trackme




Year of foundation: 2015 .


Main office: Sri Lanka .


Representations in social networks: Facebook, Google+, Twitter .


Mobile applications on the platform: Android . Tracker application.


Supports the client web interface in English .


Useful features of the system:


  • The ability to import tracks for devices entered into the system.
  • Simple and clear report template editor.
  • Ability to work with a mini card widget.
  • A simple geofence editor with the ability to set their color and snapping to objects.
  • Identification of emergency situations with objects (falling) and immediate transfer of data to the support service (by user-defined numbers).
  • The ability to generate push and email notifications about event objects.
  • REST API access to the service.
