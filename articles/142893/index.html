New navigator on code examples Microsoft All-In-One Code Framework, MSDN, Windows 8 and much more



I already told habr about the free centralized code library: Microsoft All-In-One Code Framework .

Briefly recall what it is: the goal of the Microsoft All-In-One Code Framework project is to provide code examples for common tasks for all Microsoft development technologies. At the same time, the project team does not invent these examples from the air. In order to determine which examples are needed, developer forums, social media and calls to Microsoft support are reviewed. Based on all these queries, for the most common ones, code samples are published to the library.

Such a library is virtually useless without the possibility of easy navigation and searching through code examples. And the library developers immediately provided this opportunity as a separate application and add-on to Visual Studio . The application and add-on provided a convenient way to work with examples from the All-In-One Code Framework, but there are many other code examples on Microsoft sites that are not included in the All-In-One Code Framework and which could be conveniently accessed from inside. one program. I think that I was not the only one who came up with such an idea.

A couple of weeks ago, a new version of the navigator with code examples was released, which will allow access to more than 3,500 examples, including examples from MSDN Code Gallery and official Windows 8 examples.
