Opera Devices SDK 9.6 - Mobile Abundance



Today, a press release from Opera Software announced that a new development platform, Opera Devices SDK 9.6, will be presented at the IBC exhibition opening in Amsterdam. The new platform will allow mobile devices to add a lot of useful and convenient features. In particular, Opera Zoom is one of the most visible "chips" available with the help of new development tools, which allows smooth and fast scaling or scrolling of pages on mobile devices, using hardware acceleration. Other new features include Opera Devices SDK 9.6:

OBML (Demo) Support - Opera Binary Markup Language (OBML) is the very technology that allows Opera Mini to compress to indecently small sizes all content coming to the phone. In other words, in the near future, many mobile devices will be able to save money and time of their owners by downloading Internet content in the same way as Opera Mini does.

Protection against fake websites - naturally, since we are planning to open a wide road to a full-fledged Web for mobile devices - we should think about security.

Universal Plug and Play support (UPnP) (Demo) - this feature is implemented using a special operatic widget that acts as a kind of remote content management on a wide variety of devices.



By the way, you can watch the above-mentioned technologies at work using a specially prepared video ( watch | download 53 MB).

If someone accidentally happened to be in Amsterdam - you can see the novelty and live - Opera Software is located in Hall N3, stand C11.
