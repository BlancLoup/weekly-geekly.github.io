Synology Introduces New DiskStation DS413j

Full-fledged budget personal "cloud" running DSM 4.1

Synology Inc has announced the release of a 4-disk NAS DiskStation DS413j. The new product is an economical and reliable solution for storing and sharing files in a small office or at home. DS413j is running the latest OS DiskStationManager (DSM) 4.1, which provides high protection of files and provides ample opportunities for working with multimedia.

Featuring high power savings, the DS413j consumes only 31.56 watts in operating mode and 7.68 watts in sleep mode, which makes it an ideal alternative to traditional file servers and servers for offline loading. Thanks to the highly customizable automatic on / off modes on a schedule and the transition to sleep modes, as well as the ability to remotely turn on / off the device (wake-on-LAN), you can save even more electricity. The special noise-absorbing housing design reduces the noise level to just 18.9 dB, which is of great importance for using the device in small rooms.

Key Features DS413j
  • 1.6 GHz processor
  • 512 MB DDRIII RAM
  • Gigabit LAN port
  • 2 USB 2.0 ports
  • Support 4TB HDD
  • Read speed of 108.19 MB / s, write speed of 53.29 MB / s
  • Can connect up to 5 IP cameras

For more information, visit http://www.synology.su/products/75

Recommended retail price - 17 160 rubles.
