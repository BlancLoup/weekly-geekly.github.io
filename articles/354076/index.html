Live demos of HPE Synergy Composable Infrastructure and HPE SimpliVity Hyper-Converged System

We offer to start the working week from exploring the new generation of IT infrastructure: HPE Synergy composable infrastructure and HPE SimpliVity hyper-convergent system. On Tuesday, April 24th, we will host our HPE Tech-Talk event, during which we will show live demonstrations. Learn more about the demo content and register below.


1. HPO Synergy demo - implementation of composable infrastructure. Calculation and storage

Demonstration of graphical user interface management software:

  • HPE OneView - c-Class server blades management;
  • HPE Synergy Composer - Synergy Management;
  • HPE Global Dashboard - a single data center management console.

2. Demo "HPE Synergy: Network Infrastructure and Management"

Demonstrate automatic deployment of HPE Synergy compute modules:

  • Configuring hardware parameters using scripts;
  • Deploy the hypervisor using HPE Image Streamer;
  • Deploy containers to install applications.

3. Live demonstration of HPE SimpliVity

In the demo session, we will show how an administrator of a virtual environment, without any additional training, can master the management of HPE SimpliVity as a storage and backup system, and several remote sites at once. We will address the main functions of HPE SimpliVity, as well as the stability of the solution to hardware and software failures and recoverability after failures.
