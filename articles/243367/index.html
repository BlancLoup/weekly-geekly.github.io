Enjoy reading!


News and Releases


  • About symfony 3.0
  • PHP releases: 5.4.35 , 5.5.19 and 5.6.3 - Updates on current branches, including fixes vulnerabilities in fileinfo.
  • Facebook has opened the osquery monitoring system code - The tool allows you to track the status of the operating system by executing SQL queries to a virtual database. PHP shell available.
  • Hack Transpiler - Also Facebook has announced the release of the h2tp tool, which allows you to translate the Hack code back to traditional PHP.
  • Graph of supported versions of PHP - New PHP.net page with a visual representation of the periods of support for existing versions of the interpreter.
  • Symfony Marketplace - Catalog of symfony and ecosystem related services and products.


Php


  • RFC: Return Type Declarations - Vote Canceled - Voting on such an expected proposal was canceled because a bug was found that cannot be fixed in the current implementation. Read more about the bug here .
  • RFC: Additional Usage for the Splat Operator - It is proposed to use the splat operator to implement array_merge :
    $arr1 = ['d' => 4, 'e' => 5, 'f' => 6]; $arr2 = ['a' => 1, 'b' => 2, 'c' => 3, ...$arr1]; .
  • RFC: Filtered unserialize () - Offer to extend the unserialize() function to prevent security issues. More in the post of the author .
  • RFC: Standardized PHP Http Interface - It is proposed to add the HttpMessageReceive and HttpMessageSend interfaces to the kernel, as well as the HttpRequest and HttpResponse classes for working with HTTP requests.
  • RFC: Default constructors - It is proposed to implement the concept of default constructors.
