What's new in Fedora 23 Workstation



New release of Fedora 23 Workstation released. It is a reliable, convenient and powerful operating system aimed at home users, amateurs, students, and software developers. Fedora 23 Workstation includes the latest GNOME 3.18 release of the GNOME community. This GNOME release includes updates to the file manager, new calendar, and Todo applications. Fedora 23 Workstation is the first release of Fedora, including LibreOffice 5.



Files


The file manager, also known as Nautilus, now shows a progress bar when large files are copied or moved. The button on the top panel allows you to see the progress immediately. Searching and renaming files in the file manager is now also faster and easier.

Support for Google Drive content is now also improved. If you have a connected Google account in the Control Center, you will see your Google Drive content in Files, with the Drive icon on the side. Check out the GNOME 3.18 release notes for more information on File improvements in the Fedora 23 workstation.



LibreOffice 5


Fedora 23 Workstation comes with LibreOffice 5 , the newest version of the widely used office suite. It includes LibreOffice Writer for creating test documents, LibreOffice Calc for spreadsheets and LibreOffice Impress for presentations. LibreOffice 5 includes many new features and improvements, such as:

  • Preview styles in the sidebar
  • Built-in image cropping
  • Calc Histogram Interface
  • Timestamp protocol support in PDF export
  • Improved import and export of various file types
  • Improved support for HiDPI

New Calendar


The new application provides easy editing and use of your calendars. It includes viewing important dates for a month or a year, as well as a simple search. Calendars can be stored locally on your Fedora Workstation computer, or you can add and remove calendars from sources such as Google Calendar and OwnCloud.

The new calendar integrates with your online Fedora Workstation account. When you add online accounts in the Control Center, the application displays calendars from the added accounts.



Builder IDE


Builder is also significantly better in Fedora 23 Workstation. It introduces new and improved features, such as:
  • Integrated terminals and system monitors
  • More accurate auto-complete code
  • Autocomplete Code for Python
  • Index to search for documentation
  • Building target lists
  • The list of characters to search in the current document
  • ‚Ä¶ and other




Firmware Updates


The Linux Vendor Firmware Service is now integrated into the Application Manager. The application will tell you if firmware updates are available for your computer, for example, BIOS updates. You can install them directly from the Application Manager.



Wayland


Wayland support in Fedora 23 Workstation has also been improved. It is still available as a preview using the GNOME Wayland session when you log in. After selecting a user account to log in, select the gear icon to select a Wayland session. The Workstation team is currently planning to make Wayland the default server on Fedora 24.

Getting and Installing Fedora 23 Workstation


To install Fedora Workstation on a new machine, download the live image, copy it to a USB drive , and then boot from it. To upgrade to a previous version of Fedora Workstation, check out our tutorial in Fedora Magazine.
