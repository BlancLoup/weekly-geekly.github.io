The beta release of the distribution package Elementary OS 0.4 Loki

Hello!
It so happened that I actively follow the development of the Elementary OS and periodically as I release releases I publish news about it. This time, the Elementary OS team pleased us with the beta version of the new release ‚Äúelementary OS 0.4 Loki‚Äù based on Ubuntu 16.04 LTS.

The developers promise us a bunch of new features and bugs for users and developers, but they warn you - beta is beta and should not be used in any production. The OS (or the distribution kit?) Is under active development and will gradually become overgrown with new and new features.
What's new we have prepared:

  1. Ubuntu 16.04 LTS Base Repository (Gtk 3.18, Vala 0.32, Linux 4.4)
  2. Using GtkPopoverMenu - native beautiful scrolbars
  3. Improved notification and indicator panel
  4. Refusing to use unsafe repositories (by default, there are no utilities for working with deb * packages and adding third-party repositories) and switching to a model of containerized isolated applications.
  5. New beautiful icons and styles for GTK!

Try, write to the bugtracker and develop free software! Link for download https://elementary.io/en/developer
