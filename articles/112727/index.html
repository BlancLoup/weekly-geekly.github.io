BlogEngine.NET 2.0 blogging engine

In early January, the second release of the popular engine for creating open source blogs on ASP.NET BlogEngine.NET 2.0 was published. The package for installing BlogEngine.NET 2.0 is also available on the Microsoft website's web application gala , which allows you to install it in just a few mouse clicks.

The open source engine BlogEngine.NET is designed with two basic principles in mind - ease of use and ease of functionality enhancement. At the moment is one of the most powerful and easy to use platforms for creating blogs.

Major changes in version 2.0
  • Using .NET 3.5
  • Using Captcha Recaptcha
  • TypePadFilter comment spam filter
  • The ability to use the comment system Disqus
  • SQL CE 4.0 support (database is stored in App_Data, no database server is needed)
  • Recycle Bin mechanism for deleted posts and comments
  • JQuery library is automatically connected to all pages.
  • HTML 5 video support
  • Built-in library for syntax highlighting Syntax Highlighter
  • Role Based User Rights Management System
  • Improved JS / CSS compression system
  • Support for the latest version of the tinyMCE editor
For those who update the engine from version 1.6 to 2.0, upgrade instructions (ENG) are available.

The project site has published several videos on installing and configuring the engine , for those who want to do it themselves. Those who want to install Divzhok can automatically use the Web Platfrom Installer .
