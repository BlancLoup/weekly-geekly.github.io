SAP Cloud Platform Webinar Library: from Internet of Things services to machine learning and UX

We have prepared a selection of 11 SAP Cloud Platform webinars covering a wide range of topics: from a general overview of the platform to interface functions and services, security issues, services for the Internet of things, machine learning and much, much more.

1. Overview webinar on the SAP Cloud Platform

The first webinar is devoted to an overview of the SAP Cloud Platform. Why did SAP create its cloud platform? The vision of the platform from the side of SAP. Which SAP Cloud Platform applications can be used. What is the use of SCP for SAP customers? What services are available on the SAP Cloud Platform.



2. Components for working with UX in the SAP Cloud Platform

Webinar on those components that are available for working with user interfaces in the SAP Cloud Platform. SAP Fiori 2.0 - features a new user interface. Differences between SAP Fiori and SAP UI5.


3. Information Security of SAP Cloud Solutions

General approaches to ensuring information security of cloud solutions:

  • SAP Standards
  • Additional Information

Features of cloud security in Russia:
  • Certification
  • Certification


Overview webinar on how the security of the SAP Cloud Platform is built.

4. Collaboration Services in the SAP Cloud Platform

Overview of services available for collaboration in the SAP Cloud Platform (involvement of employees and customers, communication between employees and customers, the sharing of any information):

  • Gamefication
  • SAP Document Center
  • SAP Jam


5. Internet of Things and SAP Cloud Platform

Services for the Internet of things in the SAP Cloud Platform, application scenarios. How IoT services work on the SCP: from connecting devices to establishing business processes.


6. Mobile services in SAP Cloud Platform

Overview of mobile services on the SAP Cloud Platform:

  • Mobile service for development and operations
  • Mobile service for SAP Fiori
  • Mobile service for managing applications and devices


7. Integration Services in the SAP Cloud Platform

Webinar on which integration services are available in the SAP Cloud Platform. Overview of the services "Integration of processes" and "Integration and transformation of data."


8. Analytical services on the SAP Cloud Platform

Overview of analytical services of the SAP Cloud Platform:

  • Predictive analytics (Predictive service)
  • SAP Analytics Cloud
  • Smart business service
  • Streaming analytics


9. Data & Storage services on SAP Cloud Platform

Manage data management and storage services on the SAP Cloud Platform. Overview of available services:

  • Services related to relational databases: SAP HANA, SAP ASE, PostgreSQL
  • Services related to the storage of information NoSQL: MongoDB, Redis, Object Store
  • Special service for working with documents


10. Dev & Ops services on SAP Cloud Platform

What SAP offers on the SAP Cloud Platform from the point of view of Dev & Ops: Pipeline, which tools to use.

What services exist in the SAP Cloud Platform to automate Dev & Ops processes.


11. Machine learning in SAP Cloud Platform

SAP Leonardo Machine Learning Webinar. Machine learning in SAP products. An example of using SAP c machine learning products on real projects. SAP Predictive Analytics product, its features and internal services. A set of SAP Leonardo Machine Learning Foundation services: image, video, ticket recognition in the system. SAP API. SAP plans to launch and develop new services using machine learning.
