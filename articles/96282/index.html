Interactive boards again


I was lucky last week to take part in a very interesting event called the CIO Congress "White Nights". The event was very rich and interesting, as always, but now it is a little about something else. In addition to all the traditional attributes of conferences, there was an interactive whiteboard TeamBoard. So I had a chance to poke my hands in the enemy miracle of technology myself ...

All the work of the system is described in one word - logical. Everything is thought out from the point of view of logic, which sometimes conflicts with intuitiveness. But even this does not spoil the impression of work.

So, everything looks more than solid:

A large board, with a very bright short-focus projector (if you don‚Äôt know the details, not everyone understands where the picture comes from), with a set of buttons at the bottom of the screen and with the possibility of adjusting the board height (electrolift). The projector is positioned so that it practically does not give shadows when a person works at the blackboard. The screen taps are tracked using resistive technology. Buttons allow you to choose the color of the marker. Ta-dam! When you select this tool, the following happens: the screen shot becomes the next slide in the program attached to this complex. It is logical, but you have to get used to it. First knocks.

The first thing that almost all IT professionals did was checking what would happen if they used not one hand, but two. The answer turned out to be logical, but not obvious - the system calculated the average position for the cursor.


The signal comes from the podium speaker, which is a laptop and touch screen. It is understood that the laptop is closed and everything is controlled through the board or screen with the same success. The tribune is very comfortable, but it is much more pleasant to poke a finger into the board.

If you use standard programs for the presentation of presentations, it is difficult to fully use the advantages of the board, it takes practice to understand which application is being used at a particular point in time. If you use native software, you can enjoy all the possibilities of managing slides, drawing on them and other trifles.

Seen cons:
  • The touch screen on the podium is not very sensitive.
  • It is inconvenient to click off the slides in the presentation back, if you do not use special devices or laptop keyboard.
  • In order to more fully use all the properties of the board, the presentation slides need to be overtaken into the program that comes with the board, respectively, the benefits of the presentation programs are lost.


Nevertheless, this device significantly adds life to the communication of the speaker with the audience. When the board was used, there was a feeling of closer communication, the speaker seemed to become like one of the participants, it was easier for him to ask questions and it was easier to discuss.

Probably it is for these feelings that people are ready to pay.
If you know about such opportunities in advance, you can significantly diversify the presentation formats. In general, I really liked it.
