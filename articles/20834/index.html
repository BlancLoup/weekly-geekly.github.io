Talkonaut 4.0 - Mobile VoIP for Symbian S60 platform

The development team of the famous Russian mobile VoIP application Talkonaut has announced the release of a new version of the client - Talkonaut 4.0.

The main difference of the fourth version is the transfer of the client from J2ME to Symbian S60, which significantly reduced the consumption of RAM, increase productivity and add a number of previously inaccessible features such as VoIP via GPRS, access to the phone file system and interaction with other phone applications.



The new implementation of the VoIP stack will allow users of the Talkonaut application to make voice calls via a mobile phone's Internet connection, such as via GPRS, EDGE, 3G or WIFI (recall that previous versions for the J2ME platform used only callback technology to make calls).

Thanks to using the Speex narrowband codec suite, as well as the Jingle Audio open protocol implementation as an extension to XMPP (the same protocol is used in the Google Talk application), Talkonaut offers a very powerful combination of IM chat, Presence and VoIP calls to a wide range of users.

- Talkonaut 4.0 allows you to make free VoIP calls to Google Talk, MSN, Yahoo, AIM and ICQ users, to SIP phones, as well as to other users of the Talkonaut application.

- Talkonaut 4.0 allows you to receive free calls from SIP phone users, Google Talk, MSN, Yahoo, AIM and ICQ clients.

- Talkonaut 4.0 allows you to make cheap VoIP calls at the rates of the operator GTalk2VoIP to any mobile or landline number in the world.

- Talkonaut 4.0 allows you to use an unlimited number of user-defined SIP accounts to make free (or very cheap) calls through any SIP providers. The special ‚ÄúDialing Plan‚Äù feature allows the flexibility to choose the right provider for a given set of directions.

- Talkonaut 4.0 to contain all the functionality previously available on the J2ME platform in previous versions, inherits the same graphical user interface and menu structure.

- Talkonaut 4.0 operates on a wide range of Nokia phones based on Symbian S60:
