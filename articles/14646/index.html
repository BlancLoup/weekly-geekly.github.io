EBay Desktop public beta released

The project, previously known as San Dimas, is now available for download to everyone for free.

eBay Desktop, no matter how trite it sounds, is eBay on the desktop (who would have thought, right?), that is, it allows you to search for lots on the site right from the program interface, view them, bid, read and write messages . In general, eBay Desktop offers what you need to conduct trading on eBay, doing without a web browser. But the program looks more attractive due to the fact that its functionality is not limited to the functionality of the site. So, eBay Desktop shows notifications in case of interrupting the price or in the case of completion of lots, and also allows you to track the appearance of new lots.


Download eBay Desktop for Mac OS X and Windows XP / Vista on the special eBay page .
