Boxee Media Center Beta Released

Recently, beta-testing of the Cross-box Boxee media player / center has begun. From the alpha version, released in June 2008, this version differs by a new interface, the developers have implemented agreements on integration with TV.com, blip.tv, IGN. Also, it became possible to set bookmarks on the media you like. Also, Boxee supports online media services such as YouTube, Flickr, Last.fm, Picasa Web, and others. Integration with social networks: Twitter, Facebook. By the way, a torrent client is integrated into Boxee.
Developed by Boxee in C ++, and plugins in Python. The basis of the source code under the GNU GPL license.



The list of supported formats is very large:
Video: MPEG-1, MPEG-2, MPEG-4 (SP and ASP), MPEG-4 AVC (H.264), Huffyuv, Indeo, MJPEG, RealVideo, QuickTime, Sorenson, WMV, Cinepak.
Audio: AIFF, WAV / WAVE, MP2, MP3, AAC, AACplus, AC3, DTS, ALAC, AMR, FLAC, Monkey's Audio (APE), RealAudio, SHN, WavPack, MPC / Musepack / Mpeg +, Speex, Vorbis, WMA.
Graphics: BMP, JPEG, GIF, PNG, TIFF, MNG, ICO, PCX and Targa / TGA.
Subtitles: AQTitle, ASS / SSA, CC, JACOsub, MicroDVD, MPsub, OGM, PJS, RT, SMI, SRT, SUB, VOBsub, VPlayer.
Containers: AVI, MPG / MPEG, WMV, ASF, FLV, MKV, MOV, MP4, M4A, AAC, NUT, Ogg, OGM, RealMedia (RAM / RM / RV / RA / RMVB), 3GP, VIVO, PVA, NUV , NSV, NSA, FLI, FLC, DVR-MS.
