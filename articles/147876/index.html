RubyMine 4.5 release

JetBrains is pleased to announce the release of RubyMine 4.5, a new version of intelligent integrated development environment for Ruby on Rails, which can be downloaded from our website.

We sincerely believe that to create a good product, you first need to listen to the developer community. In this version of RubyMine, we tried to implement as much of the new functionality as possible, taking into account the comments you left and the votes cast.

The following features have appeared in RubyMine 4.5:
  • code formatting and analysis for the Slim and Sass languages;
  • MacRuby syntax support;
  • Deploying Rails Applications with Capistrano
  • Embedded Rails Engines development
  • code analysis, launching and debugging for applications using Sinatra;
  • integration with Mongoid / MongoDB;
  • Ruby version management using pik and rbenv;
  • debugging code in the Rails console and multiprocess debugging applications;
  • new refactoring methods for Sass, CoffeeScript and HAML, as well as renaming Rails models.

You can get more information about the changes on the What's New page.

RubyMine 4.5 is available for free upgrade to all users whose licenses were obtained after July 18, 2011.

In addition, from today, a new type of license - Academic license, which will allow students and teachers to buy IDE at a reasonable price. More information is available here.

For anyone who would like to experience RubyMine 4.5 in action, a 30-day full-featured trial is available.
