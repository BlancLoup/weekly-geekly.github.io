Explore Minecraft: Dungeon Keeper Ancestors

I think almost everyone has heard of Minecraft. But perhaps not everyone has seen one of the great games from which Notch took his ideas - I say Dungeon Keeper. I'm not going to retell Wikipedia about what a game is, for it seems to me that it‚Äôs better to see once than to hear or read 100 times;). If the video above is a little interesting for you, you can buy the game , ask your friends for a disc or try to find an image of Dungeon Keeper Gold on the Internet (the most complete version of the game). Since the game was released in 1997 under Windows and under DOS, there are 2 startup options: under Windows in compatibility mode with Windows 95 or DOSBox. In general, I would not recommend trying to run it natively under Windows, although there are, let's say, KeeperFX , which should fix some compatibility issues.

Make sure your disk image contains music.


Interestingly, the original CD is the so-called Mixed Mode CD , in which the first track is recorded data (ie, the game itself), and all the remaining tracks are musical accompaniment. If you try to make an ISO image from such a CD, you can either get only the first track, or several .iso-files (the first link is for the query "iso mixed mode cd" ). In any case, this format will not suit you for a disk image (if you want to play with music): it will be very problematic to mount several .iso files as one logical partition. Therefore, for example, a disk image should be in the format .cue , which is devoid of the disadvantage described above and allows you to make images for Mixed Mode CD.

Mount .cue in DOSBox


If you launch Dungeon Keeper via DOSBox, and you have a .cue image, then I have good news for you: DOSBox supports mounting such files. To do this, type the following command after launch (or put it in the [autoexec] section of the configuration file):
imgmount d /path/to/image/DKGOLD.cue -t cdrom

Game setup


After you have installed the game, the keeper.exe and deeper.exe executable files should appear in the C: \ KEEPER folder (only if you have installed the Deeper Dungeons add-on that is included in Dungeon Keeper Gold) - they both require an inserted disc D: \ in order to start the game. In principle, the usual Dungeon Keeper does not check the installation directory too carefully, so you can copy the missing files from the disk and specify the path to the installation in KEEPER.CFG as ‚ÄúC: \ KEEPER \‚Äù. But then you lose the musical design, and the music in the game is very good.

If you set the game in DOSBox, then by default you will have the reduced resolution mode enabled. The game has 2 resolutions - high (by default in windows-version) and low (dos-version). You can switch between them by pressing Alt + R directly during the game. I also recommend that you spend some time setting DOSBox so that it performs aspect-correction, since the game is designed for 640x400 mode, but with a 4: 3 aspect ratio (most older CRT monitors support this mode). Otherwise, the game will not look like the authors assumed :).

If you want to play Deeper Dungeons (and even for some fragments of Dungeon Keeper this is relevant), then you may notice that scrolling is tied to the frequency of the processor, and therefore some levels can not be chosen. This is where DOSBox comes to the rescue: you can lower the speed of the game to the desired level during the selection ( DOSBox keys ), and then put it back to the maximum so that the game goes more smoothly.
