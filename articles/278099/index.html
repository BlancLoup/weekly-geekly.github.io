Microsoft acquires Xamarin

Microsoft has officially announced the acquisition of Xamarin - the largest platform for developing mobile applications. Xamarin allows developers to use all the advantages and capabilities of the .NET platform for creating mobile applications using the C # language, and provides complete sets of native APIs to use the capabilities of each of the mobile platforms for which the application is created.
