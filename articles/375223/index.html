The biggest Rubik's cube in the world: it takes 7.5 hours to solve the puzzle



If you know how to add a Rubik's cube 3 * 3, and do it well, then it's time to start reaching new heights. This can be done using the new Rubik's cube, in the format 17 * 17 * 17. In this puzzle 6.69 * 10 ^ 1054 possible combinations, so there will be time to play.

By the way, despite the large number of variations of this puzzle, the 3 * 3 * 3 format still remains the most popular. However, professionals and fans of the cube work with other formats. And one of the pros has already learned how to fold the largest Rubik's cube in the world.

According to Kenneth Brandon, Rubik's cube pro, there is nothing so terribly complicated here, working with the format 17 * 17 * 17 is similar to working with the format 7 * 7 * 7 or even 5 * 5 * 5. If you can solve such die formats, then there will be no problems with 17 * 17 * 17. True, it will take a very long time to add a puzzle.



The best option is to work from the center to the edges; this is the most logical and quick option in this case. Once the center is complex, you can proceed to the edge elements.
