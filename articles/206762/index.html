A beta version of the product for the visual creation of Windows 8.1 applications Microsoft Project Siena



A beta version of the Microsoft Project Siena product (codename) was released yesterday. The product is designed to create Windows 8.1 applications in visual mode.

Now you do not need to have special programming skills in order to realize your ideas and create powerful applications.

How does Siena work


Creating applications in Siena is no more difficult than working with documents in office products. Place controls on pages, customize their appearance and behavior.



Use data sources, be it databases, web services in the cloud, REST, RSS, SharePoint lists, or just xml or Excel files.



If you need to implement the logic in the application, you can use functions similar to the functions that are in Microsoft Excel.

An overview video and many other useful information can be found on the product site .

Technology and Development


Siena applications are HTML5 and JavaScript applications that are published, installed and managed like any other Windows 8 applications.

Developers can open Siena applications in Visual Studio and, if necessary, expand using programming tools.

Using


Installing Microsoft Project Siena is available from the Windows Store now.
