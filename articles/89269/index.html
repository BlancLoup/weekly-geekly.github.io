AMD Opteron 6000: 12 server cores

AMD has introduced the new Opteron 6000 series processors. This family includes 8-core and 12-core and chips designed for use in high-performance servers.

The line includes five 12-core processors. They vary in frequency (from 1.7 to 2.3 GHz), but all are manufactured using 45nm technology, have an integrated DDR3-1333 memory controller, 12 MB third-level cache memory and 512 kB L2 cache per core. TDP new products from 65 to 105 W, respectively. To work the chips need a socket Socket G34.
