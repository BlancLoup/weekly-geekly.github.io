Synchronize PC (Windows XP) with PDA (Windows Mobile 5) via Bluetooth.

With time on our desktops, piles of wires from various devices accumulate and it‚Äôs difficult to work without tangling in the wires. Wireless technology comes to the rescue. If the infrared connection has a number of inconveniences for the user (such as: small radius of action, modest speed), then the connection via bluetooth does not have these drawbacks. Therefore, to synchronize our devices we will use them.

So, we have a desktop computer with the Windows operating system and a pocket computer with Windows Mobile version 5 (we consider this version as the most popular).

Sync we will address book, calendar, emails, to-do list, notes and favorites.

We take into account that Microsoft ActiveSync is already installed on the PC and the Bluetooth adapter is connected. In my case, the D-Link DBT-122 was used as a bluetooth adapter.

Getting started.

It is necessary to configure the software on the PC. First, open the ‚ÄúAdvanced Settings‚Äù of the bluetooth connection manager. In the "Local Services" tab we find the button "Add COM-service". In the window that appears, you need to specify the name of the connection, specify whether the service will start automatically (I recommend to put a check mark) and whether a secure connection will be used and select the used COM port, the number of which you need to remember; On the "Alerts" tab, you are prompted to select a device connection notification. You can choose ‚ÄúVisual Alert‚Äù in the form of a pop-up window, which will indicate which device and which service has connected to; and an audio notification in which the specified audio file will be played at the time of connection (you can select only the wav file format).
After that, we open Microsoft ActiveSync and continue the path to a wireless future. In the "File" menu, select "Connection Settings". It is obligatory to set a tick on the item ‚ÄúAllow connection to the following objects‚Äù and a window just below you need to specify the COM port that we selected earlier through which the connection will be made.
Now take in the hands of the CCP. Launch iPAQ Wireless and go to the ‚ÄúDispatcher‚Äù section. By clicking "Create" we launch the Connection Wizard. Gently touch the ‚ÄúActiveSync via Bluetooth‚Äù item with the stylus. Carefully read what is written in the pages that appear and check if we have correctly made the PC pre-settings. After that, we are asked to choose a device with which we will synchronize (identify your PC by name). And here we see through what ports on the PC we can establish a connection. We are invited to choose a serial port. Choose the one we gave the name at the first stage and click ‚ÄúNext‚Äù. It remains only to create a shortcut to quickly establish a connection between devices. It is also required to establish the use of a secure connection if this mode was selected at the first stage. Click "Finish" and enjoy the disappearance of one wire from the desktop.
For the synchronization itself, you need to run iPAQ Wireless on your PDA. Open the "Manager" and hold the stylus on the synchronization point to select "Connect". After that, it remains to enter the same PIN-codes on the PC and PDA. I recommend allowing this device (PDA) to get access to the service on the PC in order not to answer this question each time before synchronization.

Now you need to activate the bluetooth on the PC and start Microsotf ActiveSync to synchronize. Then turn on the PDA and perform those actions that were performed during the first synchronization.
