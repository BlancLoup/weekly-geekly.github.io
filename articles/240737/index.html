How Odesk Freelancers Merge

Odesk is a popular platform that gives freelancers the opportunity to find a job, and employers to find a contractor for a percentage of the deal. The site actively ensures that users do not enter into agreements to bypass the platform. If the system determines that the user merges the transaction outside the platform, then you can get under the account review and get a ban for violating the user agreement.

At the same time, the site itself discloses information on which you can contact the user in Skype (for example). And already using this contact to offer cooperation.

Disclosed information - user login. To establish it, it is enough to choose a category in which you need to find a user (for example, Web Developers & Consultants) or search by keywords.

Next, select the desired freelancer and see the HTML code in the img tag for the user's photo, or rather, the src parameter - in the parameter value (Users: userlogin: PortraitUrl) and the user login is located.

Given that the login can be quite unique in nature, you can with high probability make contact with the user in Skype.

Also, due to the periodicity of leakage of information about passwords from various services, the disclosure of information about a login can carry a potential threat of unauthorized access to your account.
