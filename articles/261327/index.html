News and Releases


  • PHP 7.0.0 Alpha 2 - The second alpha release includes a modified exception hierarchy with the new Throwable interface . The next release is expected on July 9 and will be the first beta, which means that new features will not be added.
  • Google App Engine for PHP came out of beta - PHP support in App Engine has been available in test mode for more than two years , and finally it was announced about the general accessibility of the platform.
  • The PHP 5.5 support period ends. Two years after the release of the first version of PHP 5.5, 26 updates were released. On July 9, PHP 5.5.27 is expected to be released, which will be the last regular update of this thread. After that, only security bug fixes will be issued during the year.
  • The PHP-FIG group proposed a new standard for the Interface - The introductory vote on this proposal was successful and the standard moved to the draft stage.
  • The long-awaited release of Zephir 0.7 Beta - Sent by ovrweb .
  • The implementation of Hack language support in PhpStorm has been postponed.
  • PHPPackages.org - New resource where php packages are ranked by popularity.
