What version of java are you using?

Two years ago, I did a survey "Do you use Java 7?" . It looks like it's time to repeat it and expand it a bit. Especially considering the fact that Java 8 is not far away: the first Release Candidate was released on February 4.

Under the cut a variety of polls and collections of predictions. For example, it would be interesting to see in a year what percentage of people who thought that they would switch to Java 8 would actually do it.

Please, if you do not use java at all , press "abstain", so as not to spoil the statistics.

The result of the last survey
At the time of the survey with the release of Java 7, 6 months and 17 days have passed. The current version was 7u3.
