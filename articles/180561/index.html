Customizable buttons in Yandex. Mail

In the Yandex.Mail panel, a new, customizable button has appeared: in the series of familiar and understandable actions for working with letters, it makes it possible to add controls with actions that you need.



We were asked to make an opportunity:
  • remove the letters from the inbox to the archive;
  • shift letters to any folder and put a label with one click;
  • forward the letter to the pre-specified address;
  • respond to letters with templates.

And we did. All this is now possible with a new button.

Creating this opportunity, we were guided by the preferences and expectations of the audience, which uses mail as a professional communication tool.



In the three-pane interface, we plan to add this new feature in the near future.

But this is only the beginning. We perfectly understand that each person works with letters in his own way, therefore we are ready to expand the list of actions at your will. If among the first set there is no expected and usual case for you, please write about it in the comment to the post.

Focusing on the number of votes per comment, we will be able to build your ideas on popularity, and in the same order we will try to implement them.
