Introduction


Mobile hosting control and management programs are not a substitute for the usual browser interface on the computer screen, but provide the hosting administrator with the opportunity to know in time and react quickly in case of problems on the hosting or with its websites. This article describes the program for mobile devices, focused on work with hosting, running cPanel.

HostU



HostU is not for hosting users, but for a hosting provider or resellers and provides mobile access to WHM. The program allows you to manage domains, tariff packages, DNS - server and restart the web server services.
