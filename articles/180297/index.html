New solution for remote management of smartphones: 3CX Mobile Device Manager (Android + iOS)

Last week, 3CX introduced the 3CX Mobile Device Manager: an inexpensive and simple solution for managing smartphones on Android and iOS.

The functionality of the software turned out to be quite advanced:
- Remote lock or delete data

If your smartphone is lost or stolen altogether, you can remotely block it and / or erase all content, including data on an SD card, via the 3CX Mobile Device Manager. Protection against unauthorized access is possible without an internet connection. A plus, You can tighten the conditions for using a password and set a limit on the minimum number of characters in the password, as well as change the time-delay to automatically block the device if the password is entered incorrectly.
‚ÄîDelete unwanted applications and bulk install

In the admin panel, you can view all the applications that are installed on the smartphone, and remove any unnecessary applications (which kill employee time or take too much traffic). It is also possible to install programs on all connected equipment without using Appstore or Google Play.
‚ÄîRemote administration


With 3CX MDM, you can set new passwords for Wi-Fi or change the security settings immediately on all smartphones on the network. At the same time, employees will receive a notification of any changes in the system settings via email.
Search and track devices and monitor employee movement

3CX Mobile Device Manager collects all the information about the location of the connected devices and stores detailed statistics on where the smartphones / tablets were located and their owners. All locations are shown along with their full address on Google Maps.
‚ÄîRemoved corporate email setup
3CX plans to soon introduce the popular Aqua Mail email client into 3CX Mobile Device Manager. It allows all administrators to remotely set up corporate mail, gaining complete control over working mail without affecting personal correspondence. Thus, a company can delete all corporate correspondence, including file attachments that are stored on a mobile device, after an employee leaves the company.

The trial version of 3CX Mobile Device Manager allows you to connect 5 smartphones for free, and you can download it on the official website: www.MobileDeviceManager.com .
