Multifox for Firefox: go to the same site under different logins



I was tormented for a long time by the question of how you can log in to the same site, but under different logins. The simplest example: there are two mailboxes on the same mail service, one worker, the other personal. Naturally there is a desire to check them regularly. But at the same time, it is inconvenient to leave one and go to the other many times a day. If you sit most of the time under one login, then you can not see what happens to the second. I had to get out using different browsers (for Firefox, you can use the extension IE Tab). But this method still has limitations (the number of browsers is limited) and besides, it is not convenient because there are different features everywhere. Even for Firefox, it is possible to use different profiles, but this method is also far from perfect.

Today I finally got this situation and I decided to take it up in more detail. As a result, I came across a wonderful extension Multifox , which allows you to do exactly what I was missing. At the same time, I was slightly surprised that I had never heard anything about him before, although as it turned out, the extension even won the Extend Firefox 3.5 competition.

At the moment there are two current versions of Multifox - 1.3.2 and 2 beta 3, both of which are suitable for browser versions from 5 to 8a. The main difference - the first version allows you to go under different logins in different windows, and the second version, including in different tabs.

Using the plugin is quite convenient. After it is installed and the first authorization on the site, a button with the user name appears in the address bar.


Further, if we need another user, you need to click on this button and select the Add Account menu item. The page will refresh and you will be able to log in again.


As a result, we will have two tabs with different logins on the magic button. In this case, in all tabs with the site we need, we will be able to switch between active users, and the content will be updated accordingly.
