New for web designer for the first half of December

New useful things for web designers for the first half of December 2012. Last month: November . In all such topics I will add the tag " new for the web designer "

Services and tools


Weavly is a free online service for conveniently creating video clips from video, audio and gif-pictures.



Best Web Fonts is a very convenient service for viewing web fonts, including using various effects.



Ai -> Canvas - a service for exporting images from Adobe Illustrator to HTML5 Canvas.



Method Draw is a simple but functional online tool for creating SVG graphics.



Cedvel is an online service for creating your own meshes.



Artery is a simple way to sell any digital goods through a Dropbox account.



POP is an interesting iPhone application that can translate interface prototypes, drawn on paper, into a convenient digital form.



34Grid - new grid for responsive design, worthy of attention.



jQuery plugins


Calendario is an impressive calendar.



Fullscreen pageflip layout - a plugin with which you can scroll pages like book pages.



Stylization drop down lists from Codrops.



Spectragram is a plugin that can create feed from Instagram photos via API.



Crumble is a plugin that uses grumble.js to create a site tour in the form of hints.
