Loudtalks Lite for Windows Mobile - turn your device into a walkie-talkie



Loudtalks Lite is a free application for Windows and Windows Mobile that allows you to communicate in push-to-talk mode, that is, as on a walkie-talkie. The project was released in the fall of 2007, but it became really interesting now that we released a version for mobile devices.

Everything is very simple


  1. Install the program on your favorite device
  2. Run, create an account
  3. Add friends
  4. We communicate as a walkie-talkie via WiFi, 3G, EDGE or GPRS


What is

  1. Message History
  2. Sound notifications
  3. Send messages to a group of up to 10 people
  4. Windows PC version
  5. Support for all versions of Windows Mobile since Windows Mobile 2003
  6. Interface with which you can work with your fingers, stylus or "iron" buttons
  7. Talk traffic ~ 60 kilobytes per minute
  8. The traffic in the mode of doing nothing with a contact list of 10 people ~ 3 kilobytes per minute (theoretically, it can be reduced approximately 5 times, but the hands have not reached yet)

What not

  1. Supports Japper, SIP, ICQ and Mailruagent
  2. VoIP phone
  3. Save messages on the server
  4. Advertisements
