Turn off auto-run ads in the browser

Hello. I want to tell you about the problem that arose in my daily work on a PC and how I found its solution.

Here is how it was. I downloaded the program and, either I didn‚Äôt get enough sleep, or something else, but instead of the necessary software, I downloaded the good old version that installs a lot of very cool programs. Without hesitation, launched it. When Windows asked me if I really trusted this source, even then I realized that I had downloaded something wrong (the name of the developer was not at all what I expected). I click "cancel" with the thoughts that I had time to notice a trick. But it was not there. I did not have time to interrupt the operation, as I saw Amigo on my desktop, and Search the Internet, and so on.

It seems to be not difficult, I went to the Task Manager, killed all unnecessary processes, uninstalled all these wonderful programs and calmed down. However, after about half an hour, a tab with advertising of a casino in Chrome opened by itself. Shut down An hour later another one.

I checked all the extensions in Chrome - nothing new, everything was as it was. Downloaded CureIt, checked. Antivirus found some problems, fixed it, but the problem did not disappear. In the process, too, nothing suspicious. I usually do not turn off the computer for the night, and in the morning I saw about 15 open tabs with ads.

The problem was as follows. Opened the Windows Task Scheduler and found a task there.

The task, in turn, opened Chrome with the parameter andyounnews.org/thronesm:
