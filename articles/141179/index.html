Two cursors in Google Chrome



Today, on an empty tab in Chrome, I had this line: New! Browse the web with twice the mice. Try Chrome Multitask Mode.

The link is a video in which Glen Murphy, the designer of Google Chrome, talks about the fact that we only use our surfing capabilities by 50% because it does it with one hand.


Thanks to Chrome Multitask Mode, it became possible to work simultaneously with two mice, for example, simultaneously with a friend at the same computer. Click on the Try Multitask Mode button and you will get a second cursor.

PS If you leave the tab open for a while, and then return, the cursors begin to multiply.
