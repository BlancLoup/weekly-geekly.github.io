New Mobile Agent for iPhone and iPod Touch supports multitasking in iOS 4 and ... iPad!

Friends, colleagues and just apple fans!

We have released a new version of Mobile Agent for iPhone and iPod Touch. The main innovation - support multitasking iOS 4.



Multitasking support in iOS 4 allows you to work simultaneously with several applications without closing them, as well as receive messages, even if the Agent "goes into the background." In particular, implement multitasking in iOS 4 and increase the speed of the application allowed the transition to SDK 4.

IPad support is sure to please the owners of popular tablets.

Other features of the updated version:

High resolution graphics support for Retina displays.

Support power management - if the device is connected to external power, it does not go into sleep mode - the screen does not go off, the user remains online and continues to receive messages. Although, it can be dangerous, most likely, at this moment you are driving. But we hope that you will be careful :)

Separate settings for different types of notifications - previously it was possible either to turn on or turn off all notifications, now we have made separate settings for notifications about an IM message, incoming letter or microblog entry.

Interface localization - the English interface of the application will please those for whom it is more familiar than the Russian-speaking :)

You can download Mobile Agent for iPhone and iPod Touch, as always, on the App Store ( http://itunes.apple.com/us/app/mail-ru-agent/id335315530 )

Follow the news of the Mobile Agent - a lot of interesting things are planned for the next year!
