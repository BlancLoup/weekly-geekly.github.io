GitLab: open source version of Github

Want to pick up a Github clone on your own server with private repositories behind a corporate firewall? Now you can do it thanks to the open source project GitLab . It is a good alternative for the corporate version of Github worth up to $ 5000 per year.

Compared to Gitorious , the GitLab system has a nice interface and is much easier to install.

On October 13, version 1.0 was released, a week later they promise to roll out 1.1, and then new releases of GitLab 1.2, 1.3, etc. will come out every month.

You can explore the demo version of GitLab here by entering the email address test@test.com and password 123456. Now there are four projects, including Diaspora (an open source alternative to the social network Facebook).



The GitLab project itself is hosted on github . The author indicated Dmitry Zaporozhets ( dzaporozhets ).
