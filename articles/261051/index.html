May Python Meetup: machine learning and where to put the source

Hello! We want to share with you the recordings of speeches from the previous Python Meetup. This time we discussed the utility of saving source code with Grigory Petrov and features of machine learning with Andrem Grinenko.





Where to put the source / Grigori Petrov

Grigory told how a typical project grows from several dozen lines of code into a multi-megabyte monster with a hundred dependencies, and what to do about it so that it does not hurt painfully. He told the public about the crazy evolutionary theory of software development, gave many examples from life, and as a result showed the development of large projects from an unexpected angle.





Machine learning and the national economy / Andrey Grinenko

Great search and speech recognition, credit scoring and advertising display - more and more processes and technologies are optimized using modern data analysis. Andrei spoke about the basic principles of machine learning and cited cases of its use in real life.
