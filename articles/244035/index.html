Overview of Microsoft Azure platform updates for October-November

The news digest of the Microsoft Azure cloud platform contains key announcements and content for developers, IT professionals, and all professionals interested in cloud technologies and the Azure platform. Previous editions of the digest can always be found at the following link .



Announcements


Now you can test websites without creating an Azure account! - Without specifying a credit card, without creating an Azure account. All you need is a Microsoft Account.

GA and thumbs


Preview: Azure Batch - a service for running task packages without setting up a cluster and task scheduler.

Preview: Stream Analytics is an engine for processing information about events and receiving useful data in real time. There is integration with Event Hubs, which makes it possible to collect millions of events from millions of devices and process it right there.

Preview: Data Factory - a service for managing data sources - structured, unstructured, etc. You can connect SQL Server, Azure Database, Tables, Blobs, local SQL Server and create a workflow from this that will process data using Apache Hive and Pig or proprietary algorithm in C #. Video about the service.

Preview: Elastic Scale - a feature that makes it easy to scale an application data layer in the cloud. Based on sharding practices.

Preview: Apache Storm for Azure HDInsight - now you can handle real-time events with a platform for analytics. In HDInsight, you can also use the Hortonworks Data Platform 2.2 (for migrating data from a local Hadoop to an HDInsight).

GA : WebJobs is the functionality with which you can run background tasks when using the WebSites service ‚Äî you can download almost any executable algorithm and run it according to a trigger or on a schedule. GA also released WebJobs SDK, a framework for interacting with WebJobs.

GA : Event Hubs is a service with which you can collect millions of events from millions of devices. Conceptually speaking, a pub / sub queue that scales and stretches according to needs. Here you can listen to what this service is and how to use it.

GA : Public IP Addresses for Virtual Machines and Cloud Services

Azure SDK 2.5 - a large number of updates in the new version.

SSL v 3 on Azure WebSites is disabled . We read in more detail.

OpenStack Swift Object Storage for Windows Azure Pack - service from Microsoft partner - Outworx. Gives support for blobs in a private cloud running the Azure Pack.

VM Readiness Assessment tool is a utility for automatic inventory of local infrastructure and creating a report based on the inventory of what and how to migrate to Azure. Extremely useful.
