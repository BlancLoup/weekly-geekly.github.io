10 useful sites with 2D resources for games



With the development of HTML5 and mobile platforms, 2D games are back in fashion and attract the attention of even major publishers. Thanks to tools like Unity, and online resources with gaming assets, it has become much easier to develop these games. Below is a list of the top 10 sites with 2D resources for games - both paid and free.

* Sites are listed in random order *

1. Unity Asset Store (paid)

The largest site with paid gaming assets. It is not necessary to use the Unity engine for work - to get access to resources on the Asset Store, you just need to download its free version. Downloaded files will be saved to the Unity project folder, from where they can be imported into any 2D game engine.

It is worth noting that Unity in the hands of a skilled developer has enormous potential, as evidenced by the large number of successful 2D games (which only Rovio‚Äôs Bad Piggies is worth). And the Unity Asset Store offers the widest selection of 2D assets.



2. GameDev Market (paid)

GameDev Market is a relatively new site, conveniently organized by content categories (like Super Game Asset, which will be discussed below). Here you will find both 2D and 3D resources, including UI, character sprites, icons, and environments. Artists exhibit assets for sale, and the range is growing every day.



3. Graphicriver - Game Assets (paid)

Graphicriver is one of the most popular template image resources. In connection with the development of casual mobile games, there appeared a separate category with 2D assets. Before placing on the site all the content is internally checked, and after publication, any user can leave their assessment and review. Total: this is a reliable, actively developing site under the close moderation of Envato.



4. Scirra Store (paid)

Scirra is the developer company of Construct 2, a popular HTML5 2D editor, who recently had its own store of assets. Here you can find audio, graphic resources and even ready-made game templates for Construct 2. However, any other 2D editor can be used for work.



5. Game Art Partners (paid)

On Game Art Partners, you can buy a variety of cartoon assets for 2D platformers, including animated characters, monsters, weapons, visual effects, and sets of interface elements.



6. Super Game Asset (Paid)

If you are creating an RPG or game with isometric graphics, this site is for you. Here you will find the highest quality assets: colorful RPG icons (perhaps the best available online), 2D sprites, animated character sprites, and huge isometric maps. Most importantly, most of the presented resources are made in the same style.



7. Open Game Art (free)

A comprehensive resource for open source game developers, Open Game Art can be called the largest collection of free-asset gaming assets. All materials ‚Äî from sprites to icons ‚Äî can be downloaded under GNU or Creative Commons licenses. This is a great site for beginners, but many assets are different in visual style, so they will have to be carefully selected.



8. Kenney Game Assets (free)

Another great site with more than 20 thousand assets, including UI elements and various sprites for 2D platformers. Most of the resources are presented in vector graphics and are suitable for any device, regardless of the screen resolution. Assets can be downloaded separately (for free) or in one set (for a modest fee of $ 9).



9. Game-Icons.net (free)

Game-icons.net is the best site with free icons, of which there are as many as 2000. Despite the fact that all the icons are in black and white, they are quite versatile and can be used to denote actions, spells, skills, items, etc. P. Nice bonus - vector format.



10. Reiner's Tilesets (free)

The best site with free tiles. Here you can find sprites of animals, plants, buildings, weapons, visual effects and almost any objects for RPG with isometric graphics. Assets style resembles Diablo II. This is an excellent resource for novice developers who want to test the game engine or start creating their own game.
